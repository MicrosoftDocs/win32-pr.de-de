# <a name="firewall-dynamic-keywords-structures"></a>Dynamische Firewallschlüsselwörterstrukturen

Dynamische Firewallschlüsselwörter enthalten die folgenden Strukturen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | Beschreibung |
|-|-|
| [**FW_DYNAMIC_KEYWORD_ADDRESS_DATA0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address_data0) | Enthält die Daten, die an den Client zurückgegeben werden, wenn die **Enumerations-APIs** aufgerufen werden. |
| [**FW_DYNAMIC_KEYWORD_ADDRESS0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address0) | Ermöglicht dem Client das Erstellen einer dynamischen Schlüsselwortadresse, die eine Liste von IP-Adressen enthält. |
