# <a name="firewall-dynamic-keywords-enumerations"></a>Enumerationen dynamischer Firewallschlüsselwörter

Dynamische Firewallschlüsselwörter enthalten die folgenden Enumerationen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [**FW_DYNAMIC_KEYWORD_ADDRESS_ENUM_FLAGS**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_address_enum_flags) | Definiert Konstanten, die die Art(en) von Objekten angeben, die in einen Enumerationsvorgang eingeschlossen werden sollen. |
| [**FW_DYNAMIC_KEYWORD_ADDRESS_FLAGS**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_address_flags) | Definiert Konstanten, die angeben, wie IP-Adressen aufgelöst werden sollen. |
| [**FW_DYNAMIC_KEYWORD_ORIGIN_TYPE**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_origin_type) | Definiert Konstanten, die den Ursprung des dynamischen Schlüsselwortadressobjekts in einem [**FW_DYNAMIC_KEYWORD_ADDRESS_DATA0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address_data0)angeben. |
