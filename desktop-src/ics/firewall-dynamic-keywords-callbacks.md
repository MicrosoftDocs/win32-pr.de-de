Dynamische Firewallschlüsselwörter enthalten die folgenden Rückrufe.

## <a name="in-this-section"></a>In diesem Abschnitt

 (fwpmu.h) ()

| Thema | BESCHREIBUNG |
|-|-|
| [**FWPM_DYNAMIC_KEYWORD_CALLBACK0**](/windows/win32/api/fwpmu/nc-fwpmu-fwpm_dynamic_keyword_callback0) | Eine von Ihnen implementierte Rückruffunktion, die mit Benachrichtigungen zu Änderungen an dynamischen Schlüsselwortadressenobjekten ([FW_DYNAMIC_KEYWORD_ADDRESS0](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address0)) aufgerufen wird. |
