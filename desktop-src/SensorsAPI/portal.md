## <a name="purpose"></a><span data-ttu-id="7bf1e-104">Zweck</span><span class="sxs-lookup"><span data-stu-id="7bf1e-104">Purpose</span></span>

<span data-ttu-id="7bf1e-105">Windows 7 umfasst systemeigene Unterstützung für Sensoren, bei denen es sich um Geräte handelt, die physische Phänomene wie Temperatur oder Standort messen können.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-105">Windows 7 includes native support for sensors, which are devices that can measure physical phenomena such as temperature or location.</span></span> <span data-ttu-id="7bf1e-106">In dieser Dokumentation wird die Sensor-API beschrieben, die es Anwendungen ermöglicht, Daten von Sensoren auf standardisierte Weise zu erhalten und zu verwenden.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-106">This documentation describes the Sensor API, which enables applications to get and use data from sensors in a standardized way.</span></span>

<span data-ttu-id="7bf1e-107">Als Menschen verlassen wir uns auf unsere Sinne, um uns Informationen zur Welt rund um die Welt bereitzustellen.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-107">As humans, we rely on our senses to provide us with information about the world around us.</span></span> <span data-ttu-id="7bf1e-108">Wenn wir Computer erstellen, die einige unserer Aufgaben Unternehmen, fügen wir Sensor Mechanismen hinzu, damit die Computer angemessen auf veränderliche Bedingungen reagieren können.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-108">When we create machines to take on some of our work, we add sensor mechanisms so the machines can respond appropriately to changing conditions.</span></span>

<span data-ttu-id="7bf1e-109">Auto Mobil Engines verwenden z. b. normalerweise eine Vielzahl von Sensoren.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-109">For example, automobile engines typically use a variety of sensors.</span></span> <span data-ttu-id="7bf1e-110">Diese Sensoren werden von einem integrierten Computer überwacht, der kontinuierlich Einstellungen anpasst, wie z. b. Engine-zeitliche Steuerung, um die Leistung und Effizienz zu maximieren.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-110">These sensors are monitored by an onboard computer that continuously adjusts settings, such as engine timing, to maximize power and efficiency.</span></span> <span data-ttu-id="7bf1e-111">Ein Fernsehgerät kann einen Umgebungslichtsensor verwenden, um die Helligkeit des Bilds an geänderte Raumbedingungen anzupassen.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-111">A television may use an ambient light sensor to adjust the brightness of the picture to match changing room conditions.</span></span> <span data-ttu-id="7bf1e-112">Sogar etwas so einfaches wie eine Türklingel Schaltfläche fungiert als rudimentärer Sensor zum Erkennen einer menschlichen Präsenz an der Tür.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-112">Even something as simple as a doorbell button acts as a rudimentary sensor to detect a human presence at the door.</span></span>

<span data-ttu-id="7bf1e-113">Obwohl die rein mechanische Türglocke ihren Zweck erfüllt, werden die von komplexen Sensoren bereitgestellten Informationen weitaus leistungsfähiger, wenn Sie mit Software kombiniert werden.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-113">While the purely mechanical doorbell fulfills its purpose, the information provided by complex sensors becomes far more powerful when it is combined with software.</span></span> <span data-ttu-id="7bf1e-114">Moderne Sensoren können sehr schnell und in einer Vielzahl von Formaten viele Daten bereitstellen, sodass Software einen natürlichen Mechanismus zur Erstellung von Sensordaten bereitstellt.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-114">Modern sensors can provide a lot of data very quickly, and in a variety of formats, so software provides a natural mechanism for making sense of sensor data.</span></span>

<span data-ttu-id="7bf1e-115">Softwareentwickler können heutzutage Programme schreiben, die Sensoren verwenden, aber eine fehlende Standardisierung macht das Programmieren für Sensoren zu einer schwierigen Aufgabe.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-115">Today, software developers can write programs that use sensors, but a lack of standardization makes programming for sensors an arduous task.</span></span> <span data-ttu-id="7bf1e-116">Nachdem ein Sensor basiertes Programm abgeschlossen ist, ist es in der Regel immer von einem bestimmten Hardwaretyp abhängig.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-116">After a sensor-based program is completed, it is usually forever dependent on a particular type of hardware.</span></span> <span data-ttu-id="7bf1e-117">Die Verwendung einer oder mehrerer vertikaler Lösungen zum Aktivieren der Bereitstellung eines softwarebasierten Systems hat die Integration von Sensoren in Computer Hardware eingeschränkt. bisher gab es bei Windows-basierten Computern keine Ausnahme.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-117">Using one or more vertical solutions to enable deployment of a software-based system has limited the integration of sensors with computer hardware and, until now, Windows-based computers have been no exception.</span></span>

<span data-ttu-id="7bf1e-118">Windows 7 bietet systemeigene Unterstützung für Sensoren, erweitert durch eine neue Entwicklungsplattform zum Arbeiten mit Sensoren, einschließlich Standort Sensoren wie GPS-Geräten.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-118">Windows 7 includes native support for sensors, expanded by a new development platform for working with sensors, including location sensors, such as GPS devices.</span></span> <span data-ttu-id="7bf1e-119">Die Plattform für Windows-Sensoren und-Orte bietet Geräteherstellern standardmäßig die Möglichkeit, Sensor Geräte für Softwareentwickler und-Consumer verfügbar zu machen, während Entwicklern eine standardisierte Anwendungsprogrammierschnittstelle (Application Programming Interface, API) für die Arbeit mit Sensoren und Sensor Daten zur Verfügung steht.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-119">The Windows Sensor and Location platform provides a standard way for device manufacturers to expose sensor devices to software developers and consumers, while providing developers with a standardized application programming interface (API) for working with sensors and sensor data.</span></span>

<span data-ttu-id="7bf1e-120">Sensoren sind Geräte oder Mechanismen, mit denen physische Phänomene gemessen, beschreibende Daten bereitgestellt oder Informationen über den Status eines physischen Objekts oder einer Umgebung bereitgestellt werden können.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-120">Sensors are devices or mechanisms that can measure physical phenomena, provide descriptive data, or provide information about the state of a physical object or environment.</span></span> <span data-ttu-id="7bf1e-121">Computer können integrierte Sensoren, Sensoren, die über Kabel-oder drahtlos Verbindungen verbunden sind, oder Sensoren nutzen, die Daten über ein Netzwerk oder das Internet bereitstellen.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-121">Computers can make use of built-in sensors, sensors that are connected through wired or wireless connections, or sensors that provide data through a network or the Internet.</span></span>

<span data-ttu-id="7bf1e-122">Die Sensor-API bietet eine Standardmethode zum programmgesteuerten Zugriff auf Daten, die von Sensoren bereitgestellt werden.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-122">The Sensor API provides a standard way to programmatically access data that sensors provide.</span></span> <span data-ttu-id="7bf1e-123">Die Sensor-API standardisiert Folgendes:</span><span class="sxs-lookup"><span data-stu-id="7bf1e-123">The Sensor API standardizes:</span></span>

-   <span data-ttu-id="7bf1e-124">Sensor Kategorien,-Typen und-Eigenschaften.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-124">Sensor categories, types, and properties.</span></span>
-   <span data-ttu-id="7bf1e-125">Datenformate für Standard Sensortypen.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-125">Data formats for standard sensor types.</span></span>
-   <span data-ttu-id="7bf1e-126">COM-Schnittstellen zum Arbeiten mit Sensoren und Auflistungen von Sensoren.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-126">COM interfaces for working with sensors and collections of sensors.</span></span>
-   <span data-ttu-id="7bf1e-127">Ereignis Mechanismen für den asynchronen Empfang von Sensordaten.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-127">Event mechanisms for asynchronously receiving sensor data.</span></span>

<span data-ttu-id="7bf1e-128">Die Sensor-API ermöglicht außerdem das Definieren von benutzerdefinierten Sensor Kategorien,-Typen,-Eigenschaften,-Datenformaten und-Ereignissen.</span><span class="sxs-lookup"><span data-stu-id="7bf1e-128">The Sensor API also enables you to define custom sensor categories, types, properties, data formats, and events.</span></span>
