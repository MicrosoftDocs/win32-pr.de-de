# <a name="the-sensor-collection-object"></a>Das Sensor Auflistungs Objekt

Das Sensor Sammlungsobjekt enthält eine Gruppe von Sensor Objekten.

Manchmal empfiehlt es sich, mit Sensoren als Gruppe zu arbeiten. Beispielsweise können Sie den Sensor-Manager verwenden, um alle verfügbaren Bewegungssensoren zu ermitteln. Das sensorauflistungs-Objekt stellt Methoden zum Arbeiten mit einer Gruppe von Sensor Objekten bereit. Sie können die Schnittstelle " [**isencollection**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorcollection) " verwenden, um Methoden aufzurufen, die Sensoren in der Auflistung zählen, hinzufügen, entfernen und abrufen.
