# <a name="com-interfaces-sensor-api"></a>COM-Schnittstellen (Sensor-API)

Die Sensor-API macht die folgenden COM-Schnittstellen verfügbar.



| Schnittstelle                                              | BESCHREIBUNG                                                                                                      |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**Ilocationberechtigungen**](/windows/desktop/api/sensorsapi/nn-sensorsapi-ilocationpermissions)   | Gibt den Status der Systemeinstellung an, mit der Benutzer die Speicherort Einstellungen ändern können.                         |
| [**Ilogicalsensormanager**](/previous-versions/windows/desktop/legacy/dd318934(v=vs.85)) | Stellt Methoden bereit, die von logischen Sensor Entwicklern zum Verwalten logischer Sensorverbindungen verwendet werden.                            |
| [**ISensor**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensor)                             | Stellt einen Sensor dar.                                                                                             |
| [**Isencollection**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorcollection)         | Stellt eine Auflistung von Sensoren dar.                                                                              |
| [**Isensordatareport**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensordatareport)         | Stellt einen Sensordaten Bericht dar.                                                                                 |
| [**Isenevents**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorevents)                 | Die Rückruf Schnittstelle zum Empfangen von Sensor Ereignissen.                                                              |
| [**Isenmanager**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensormanager)               | Stellt Methoden zum Ermitteln und Abrufen verfügbarer Sensoren und eine Methode zum Anfordern von Sensor-Manager-Ereignissen bereit. |
| [**Isenmanagerevents**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensormanagerevents)   | Die Rückruf Schnittstelle zum Empfangen von Sensor-Manager-Ereignissen.                                                      |
