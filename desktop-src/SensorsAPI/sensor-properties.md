# <a name="sensor-properties"></a>Sensoreigenschaften

Die Sensor- und Location-Plattform definiert Konstanten, die Eigenschaften für Sensoren identifizieren. Sensorhersteller können auch eigene Eigenschaften definieren.

Die Plattform definiert die folgenden **PROPERTYKEY-Werte** für Sensoreigenschaften. Diese Eigenschaften sind schreibgeschützt, sofern nicht anders angegeben.

Jede plattformdefinierte Sensoreigenschaft **PROPERTYKEY** basiert auf einer gemeinsamen **GUID** mit dem Namen SENSOR \_ PROPERTY COMMON \_ \_ GUID:

{7F8383EC-D3EC-495C-A8CF-B8BBE85C2920}.

> [!IMPORTANT]
> Verwenden Sie diesen Basiswert nicht, um Ihre eigenen Eigenschaftsschlüssel zu definieren.

 

Werte für Eigenschaften, die als Lese-/Schreibzugriff festgelegt sind, können von der Clientanwendung angegeben werden. Werte für Eigenschaften, die als statisch festgelegt sind, dürfen sich im Laufe der Zeit nicht ändern. Als erforderlich gekennzeichnete Eigenschaften müssen vom Sensor unterstützt werden.
