# <a name="sensor-properties"></a>Sensor Eigenschaften

Die Sensor-und die Location-Plattform definieren Konstanten, die Eigenschaften für Sensoren identifizieren. Sensor Hersteller können auch Ihre eigenen Eigenschaften definieren.

Die Plattform definiert die folgenden **PropertyKey** -Werte für Sensoreigenschaften. Diese Eigenschaften sind schreibgeschützt, sofern nicht anders angegeben.

Jede Platt Form definierte Sensor Eigenschaft **PropertyKey** basiert auf einer allgemeinen **GUID** mit dem Namen "Sensor \_ Property \_ Common \_ GUID":

{7f8383ec-d3ec-495c-a8cf-b8bbe85c2920}.

> [!IMPORTANT]
> Verwenden Sie diesen Basiswert nicht, um eigene Eigenschafts Schlüssel zu definieren.

 

Werte für als Lese-/Schreibzugriff bezeichnete Eigenschaften können von der Client Anwendung angegeben werden. Werte für die als statisch bezeichneten Eigenschaften dürfen sich im Laufe der Zeit nicht ändern. Als erforderlich festgelegte Eigenschaften müssen vom Sensor unterstützt werden.
