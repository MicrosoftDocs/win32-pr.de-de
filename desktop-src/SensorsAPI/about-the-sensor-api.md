# <a name="about-the-sensor-api"></a>Informationen zur Sensor-API

Die Sensor-API stellt die Funktionalität über einen Satz von COM-Schnittstellen bereit. Die Sensor-API macht einige dieser Schnittstellen verfügbar, was bedeutet, dass Sie Methoden für diese Schnittstellen abrufen können. Andere Schnittstellen werden durch die API definiert, sodass Sie die Methoden implementieren können, die die Sensor-API benötigt, um Ereignis Benachrichtigungen für Ihr Programm bereitzustellen.
