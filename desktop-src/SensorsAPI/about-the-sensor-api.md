# <a name="about-the-sensor-api"></a>Informationen zur Sensor-API

Die Sensor-API stellt ihre Funktionalität über eine Reihe von COM-Schnittstellen bereit. Die Sensor-API macht einige dieser Schnittstellen verfügbar, was bedeutet, dass Sie Methoden für sie aufrufen können. Andere Schnittstellen werden von der API definiert, sodass Sie die Methoden implementieren können, die die Sensor-API benötigt, um Ereignisbenachrichtigungen an Ihr Programm zu senden.
