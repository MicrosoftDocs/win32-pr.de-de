# <a name="sensor-api-glossary"></a>Sensor-API-Glossar

<dl> <dt>

<span id="winsensors.sensors_glossary_ambient_light_sensor"></span><span id="WINSENSORS.SENSORS_GLOSSARY_AMBIENT_LIGHT_SENSOR"></span>**Umgebungslichtsensor**
</dt> <dd>

Ein Sensor, der den aktuellen beleuchtungsgrad erkennt.

</dd> <dt>

<span id="winsensors.sensors_glossary_geodetic_reference_system"></span><span id="WINSENSORS.SENSORS_GLOSSARY_GEODETIC_REFERENCE_SYSTEM"></span>**Geodätisches Referenzsystem**
</dt> <dd>

Die Kombination aus einem Ellipsoid, der die Größe und Form der Erde angibt, und ein Basispunkt, von dem aus der breiten-und Längengrad aller anderen Punkte referenziert wird.

</dd> <dt>

<span id="winsensors.sensors_glossary_light-aware_user_interface__ui_"></span><span id="WINSENSORS.SENSORS_GLOSSARY_LIGHT-AWARE_USER_INTERFACE__UI_"></span>**helle Benutzeroberfläche (UI)**
</dt> <dd>

Ein Programm, das Umgebungslichtsensor-Daten verwendet, um die Benutzeroberfläche und Inhalte für bestimmte Beleuchtungsbedingungen zu optimieren.

</dd> <dt>

<span id="winsensors.sensors_glossary_logical_id"></span><span id="WINSENSORS.SENSORS_GLOSSARY_LOGICAL_ID"></span>**logische ID**
</dt> <dd>

Ein Globally Unique Identifier (GUID), der einen bestimmten logischen Sensor identifiziert. Eine logische ID ist immer erforderlich, wenn Sie eine Verbindung mit einem logischen Sensor herstellen.

</dd> <dt>

<span id="winsensors.sensors_glossary_logical_sensor"></span><span id="WINSENSORS.SENSORS_GLOSSARY_LOGICAL_SENSOR"></span>**logischer Sensor**
</dt> <dd>

Ein Sensor, der Daten ohne Verwendung eines Hardware Geräts bereitstellt. Ein logischer Sensor könnte z. b. Daten über den aktuellen Speicherort des Benutzers bereitstellen, indem er einen Dienst verwendet, der eine IP-Adresse in einer Tabelle nach sucht.

</dd> <dt>

<span id="winsensors.sensors_glossary_lux_value"></span><span id="WINSENSORS.SENSORS_GLOSSARY_LUX_VALUE"></span>**Lux-Wert**
</dt> <dd>

Die Menge an Licht (in dünn) pro Quadratmeter, die eine Oberfläche beleuchtet. Umgebungslichtsensoren stellen Daten in Lux-Werten bereit.

</dd> <dt>

<span id="winsensors.sensors_glossary_reference_ellipsoid"></span><span id="WINSENSORS.SENSORS_GLOSSARY_REFERENCE_ELLIPSOID"></span>**Verweis-Ellipsoid**
</dt> <dd>

Eine Ellipsoid, die einem geodätischen Verweis System zugeordnet ist, dessen Oberfläche der Größe und Form der Erde entspricht.

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR"></span>**k**
</dt> <dd>

Hardware oder Software, mit der externe Ereignisse oder Umgebungsbedingungen erkannt werden können. Beispielsweise ist ein GPS-Empfänger (Global Positioning System) ein Sensor, der den aktuellen physischen Speicherort eines Benutzers bestimmen kann.

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_category"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_CATEGORY"></span>**Sensor Kategorie**
</dt> <dd>

Eine umfassende Klasse von Sensorgeräten. Beispielsweise gehört ein Sensor in einem Videospiel Controller, der die Position und Bewegung der Hand eines Players erkennt, zur Kategorie Ausrichtung der Sensoren.

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_collection"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_COLLECTION"></span>**Sensor Sammlung**
</dt> <dd>

Mehrere Sensoren, die als Gruppe eingesetzt werden können. Eine Sensor Auflistung könnte beispielsweise alle verfügbaren Bewegungssensoren enthalten, und es könnte eine-Methode aufgerufen werden, die diese Auflistung abruft.

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_data_report"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_DATA_REPORT"></span>**Sensordaten Bericht**
</dt> <dd>

Ein Bericht, der Daten Feldwerte enthält, die von einem Sensor generiert werden.

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_device"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_DEVICE"></span>**Sensorgerät**
</dt> <dd>

Ein Hardware Gerät, mit dem externe Ereignisse oder Umgebungsbedingungen erkannt werden können. Ein einzelnes Hardware Gerät kann mehrere Sensoren enthalten.

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_manager"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_MANAGER"></span>**Sensor-Manager**
</dt> <dd>

Ein Programmier Objekt, das eine Liste der verfügbaren Sensoren verwaltet und Zugriff darauf bietet.

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_type"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_TYPE"></span>**Sensortyp**
</dt> <dd>

Eine bestimmte Art von Sensor, z. b. ein dreidimensionaler Beschleunigungsmesser.

</dd> </dl>
