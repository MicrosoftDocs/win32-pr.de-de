<span data-ttu-id="0b4a7-104">Das NTFS-Dateisystem enthält eine Datei mit dem Namen *Master File Table* oder MFT.</span><span class="sxs-lookup"><span data-stu-id="0b4a7-104">The NTFS file system contains a file called the *master file table*, or MFT.</span></span> <span data-ttu-id="0b4a7-105">Es gibt mindestens einen Eintrag in der MFT für jede Datei auf einem NTFS-Dateisystem Volume, einschließlich der MFT selbst.</span><span class="sxs-lookup"><span data-stu-id="0b4a7-105">There is at least one entry in the MFT for every file on an NTFS file system volume, including the MFT itself.</span></span> <span data-ttu-id="0b4a7-106">Alle Informationen zu einer Datei, einschließlich Größe, Zeit-und Datumsstempel, Berechtigungen und Dateninhalt, werden entweder in MFT-Einträgen oder außerhalb des durch MFT-Einträge beschriebenen MFT gespeichert.</span><span class="sxs-lookup"><span data-stu-id="0b4a7-106">All information about a file, including its size, time and date stamps, permissions, and data content, is stored either in MFT entries, or in space outside the MFT that is described by MFT entries.</span></span>

<span data-ttu-id="0b4a7-107">Wenn Dateien einem NTFS-Dateisystem Volume hinzugefügt werden, werden der MFT weitere Einträge hinzugefügt, und die Größe des MFT vergrößert sich.</span><span class="sxs-lookup"><span data-stu-id="0b4a7-107">As files are added to an NTFS file system volume, more entries are added to the MFT and the MFT increases in size.</span></span> <span data-ttu-id="0b4a7-108">Wenn Dateien aus einem NTFS-Dateisystem Volume gelöscht werden, werden die MFT-Einträge als frei markiert und können wieder verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="0b4a7-108">When files are deleted from an NTFS file system volume, their MFT entries are marked as free and may be reused.</span></span> <span data-ttu-id="0b4a7-109">Der Speicherplatz, der für diese Einträge zugewiesen wurde, wird jedoch nicht neu zugeordnet, und die Größe des MFT wird nicht verringert.</span><span class="sxs-lookup"><span data-stu-id="0b4a7-109">However, disk space that has been allocated for these entries is not reallocated, and the size of the MFT does not decrease.</span></span>

<span data-ttu-id="0b4a7-110">Das NTFS-Dateisystem reserviert Speicherplatz für die MFT, um die MFT so zusammenhängend wie möglich zu halten, wenn Sie wächst.</span><span class="sxs-lookup"><span data-stu-id="0b4a7-110">The NTFS file system reserves space for the MFT to keep the MFT as contiguous as possible as it grows.</span></span> <span data-ttu-id="0b4a7-111">Der vom NTFS-Dateisystem für die MFT in jedem Volume reservierte Speicherplatz wird als MFT-Zone bezeichnet.</span><span class="sxs-lookup"><span data-stu-id="0b4a7-111">The space reserved by the NTFS file system for the MFT in each volume is called the MFT zone.</span></span> <span data-ttu-id="0b4a7-112">Der Speicherplatz für Dateien und Verzeichnisse wird ebenfalls von diesem Bereich aus zugewiesen, jedoch erst, nachdem der gesamte volumespeicherplatz außerhalb der MFT-Zone zugeordnet wurde.</span><span class="sxs-lookup"><span data-stu-id="0b4a7-112">Space for file and directories are also allocated from this space, but only after all of the volume space outside of the MFT zone has been allocated.</span></span>

<span data-ttu-id="0b4a7-113">Abhängig von der durchschnittlichen Dateigröße und anderen Variablen kann die reservierte MFT-Zone oder der nicht reservierte Speicherplatz auf dem Datenträger zuerst class="sxs-lookup"><span data-stu-id="0b4a7-114">Volumes with a small number of relatively large files will allocate the unreserved space first, while volumes with a large number of relatively small <span data-ttu-id="0b4a7-116">Wenn der nicht reservierte Speicherplatz vollständig zugeordnet ist, wird der Speicherplatz für Benutzer Dateien und Verzeichnisse von der MFT-Zone
