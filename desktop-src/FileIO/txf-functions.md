# <a name="txf-functions"></a>TxF-Funktionen

Transaktionale NTFS (TxF) bietet die folgenden Funktionen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Funktion                                                                      | BESCHREIBUNG                                                                                                                  |
|-------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [**Txflogkreatefilereadcontext**](/windows/desktop/api/TxfW32/nf-txfw32-txflogcreatefilereadcontext)<br/> | Erstellt einen Kontext, der zum Lesen von Replikations Datensätzen verwendet wird.<br/>                                                         |
| [**Txflogdestroyread Context**](/windows/desktop/api/TxfW32/nf-txfw32-txflogdestroyreadcontext)<br/>       | Schließt einen von der [**txflogkreatefilereadcontext**](/windows/desktop/api/TxfW32/nf-txfw32-txflogcreatefilereadcontext) -Funktion erstellten Lese Kontext.<br/> |
| [**Txflogreadrecords**](/windows/desktop/api/TxfW32/nf-txfw32-txflogreadrecords)<br/>                     | Liest die Redo-Einträge aus dem Protokoll.<br/>                                                                              |
