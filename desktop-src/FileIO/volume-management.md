# <a name="volume-management"></a>Volumeverwaltung

Die höchste Organisationsebene im Dateisystem ist das *Volume*. Ein Dateisystem befindet sich auf einem Volume. Ein Volume enthält mindestens eine *Partition*, bei der es sich um eine logische Division eines physischen Datenträgers handelt (Weitere Informationen finden Sie unter Datenträger [Geräte und Partitionen](disk-devices-and-partitions.md)). Ein Volume, das Daten enthält, die auf einer Partition vorhanden sind, wird als *einfaches Volume* bezeichnet, und ein Volume, das Daten enthält, die auf mehr als einer Partition vorhanden sind, wird als *Volume mit mehreren Partitionen* bezeichnet.
