# <a name="file-access-rights-constants"></a>Dateizugriffsrechte-Konstanten

Die gültigen Zugriffsrechte für Dateien und Verzeichnisse umfassen die Standardzugriffsrechte **DELETE,** **READ \_ CONTROL,** **WRITE \_ DAC,** **WRITE \_ OWNER** [und](/windows/desktop/SecAuthZ/standard-access-rights) **SYNCHRONIZE.** In der folgenden Tabelle sind die Zugriffsrechte aufgeführt, die für Dateien und Verzeichnisse spezifisch sind.



| Konstante/Wert                                                                                                                                                                                                                                             | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                        |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="FILE_ADD_FILE"></span><span id="file_add_file"></span><dl> <dt>**FILE \_ ADD \_ FILE**</dt> <dt>2</dt> </dl>                                      | Für ein Verzeichnis das Recht, eine Datei im Verzeichnis zu erstellen.<br/>                                                                                                                                                                                                                                                                                                           |
| <span id="FILE_ADD_SUBDIRECTORY"></span><span id="file_add_subdirectory"></span><dl> <dt>**FILE \_ ADD \_ SUBDIRECTORY**</dt> <dt>4</dt> </dl>              | Für ein Verzeichnis das Recht, ein Unterverzeichnis zu erstellen.<br/>                                                                                                                                                                                                                                                                                                                    |
| <span id="FILE_ALL_ACCESS"></span><span id="file_all_access"></span><dl> <dt>**FILE \_ ALLE \_ ZUGRIFFE**</dt><dt></dt> </dl>                                 | Alle möglichen Zugriffsrechte für eine Datei.<br/>                                                                                                                                                                                                                                                                                                                                  |
| <span id="FILE_APPEND_DATA"></span><span id="file_append_data"></span><dl> <dt>**FILE \_ APPEND \_ DATA**</dt> <dt>4</dt> </dl>                             | Bei einem Dateiobjekt das Recht, Daten an die Datei anfügen. (Bei lokalen Dateien überschreiben Schreibvorgänge keine vorhandenen Daten, wenn dieses Flag ohne **FILE WRITE DATA angegeben \_ \_ wird.)** Für ein Verzeichnisobjekt das Recht, ein Unterverzeichnis zu erstellen (**FILE \_ ADD \_ SUBDIRECTORY**).<br/>                                                                                        |
| <span id="FILE_CREATE_PIPE_INSTANCE"></span><span id="file_create_pipe_instance"></span><dl> <dt>**FILE \_ CREATE \_ PIPE \_ INSTANCE**</dt> <dt>4</dt> </dl> | Für eine Named Pipe ist dies das Recht, eine Pipe zu erstellen.<br/>                                                                                                                                                                                                                                                                                                                           |
| <span id="FILE_DELETE_CHILD"></span><span id="file_delete_child"></span><dl> <dt>**FILE \_ DELETE \_ CHILD**</dt> <dt>64 (0x40)</dt> </dl>                  | Für ein Verzeichnis das Recht, ein Verzeichnis und alle dateien, die es enthält, einschließlich schreibgeschützter Dateien, zu löschen.<br/>                                                                                                                                                                                                                                                              |
| <span id="FILE_EXECUTE"></span><span id="file_execute"></span><dl> <dt>**FILE \_ EXECUTE**</dt> <dt>32 (0x20)</dt> </dl>                                  | Bei einer nativen Codedatei das Recht, die Datei auszuführen. Dieses Zugriffsrecht für Skripts kann je nach Skriptinterpreter dazu führen, dass das Skript ausführbar ist.<br/>                                                                                                                                                                                                   |
| <span id="FILE_LIST_DIRECTORY"></span><span id="file_list_directory"></span><dl> <dt>**FILE \_ LIST \_ DIRECTORY**</dt> <dt>1</dt> </dl>                    | Für ein Verzeichnis das Recht, den Inhalt des Verzeichnisses auflisten.<br/>                                                                                                                                                                                                                                                                                                       |
| <span id="FILE_READ_ATTRIBUTES"></span><span id="file_read_attributes"></span><dl> <dt>**FILE \_ LESEN \_ DER ATTRIBUTE**</dt> <dt>128 (0x80)</dt> </dl>        | Das Recht zum Lesen von Dateiattributen.<br/>                                                                                                                                                                                                                                                                                                                                      |
| <span id="FILE_READ_DATA"></span><span id="file_read_data"></span><dl> <dt>**FILE \_ LESEN \_ VON DATEN**</dt> <dt>1</dt> </dl>                                   | Bei einem Dateiobjekt das Recht, die entsprechenden Dateidaten zu lesen. Bei einem Verzeichnisobjekt das Recht, die entsprechenden Verzeichnisdaten zu lesen.<br/>                                                                                                                                                                                                                           |
| <span id="FILE_READ_EA"></span><span id="file_read_ea"></span><dl> <dt>**FILE \_ LESEN \_ VON EA**</dt> <dt>8</dt> </dl>                                         | Das Recht, erweiterte Dateiattribute zu lesen.<br/>                                                                                                                                                                                                                                                                                                                             |
| <span id="FILE_TRAVERSE"></span><span id="file_traverse"></span><dl> <dt>**FILE \_ TRAVERSE**</dt> <dt>32 (0x20)</dt> </dl>                               | Bei einem Verzeichnis das Recht, das Verzeichnis zu durchlaufen. Standardmäßig wird Benutzern die Berechtigung **BYPASS \_ TRAVERSE \_ CHECKING** [zugewiesen,](/windows/desktop/SecAuthZ/privileges)die das **ZUGRIFFsrecht FILE \_ TRAVERSE** [ignoriert.](/windows/desktop/SecAuthZ/access-rights-and-access-masks) Weitere Informationen finden Sie in den Anmerkungen unter [Dateisicherheit](file-security-and-access-rights.md) und Zugriffsrechte.<br/> |
| <span id="FILE_WRITE_ATTRIBUTES"></span><span id="file_write_attributes"></span><dl> <dt>**FILE \_ WRITE \_ ATTRIBUTES**</dt> <dt>256 (0x100)</dt> </dl>    | Das Recht zum Schreiben von Dateiattributen.<br/>                                                                                                                                                                                                                                                                                                                                     |
| <span id="FILE_WRITE_DATA"></span><span id="file_write_data"></span><dl> <dt>**FILE \_ SCHREIBEN \_ VON DATEN**</dt> <dt>2</dt> </dl>                                | Bei einem Dateiobjekt das Recht, Daten in die Datei zu schreiben. Bei einem Verzeichnisobjekt das Recht, eine Datei im Verzeichnis zu erstellen (**FILE \_ ADD \_ FILE**).<br/>                                                                                                                                                                                                                      |
| <span id="FILE_WRITE_EA"></span><span id="file_write_ea"></span><dl> <dt>**FILE \_ WRITE \_ EA**</dt> <dt>16 (0x10)</dt> </dl>                              | Das Recht, erweiterte Dateiattribute zu schreiben.<br/>                                                                                                                                                                                                                                                                                                                            |
| <span id="STANDARD_RIGHTS_READ"></span><span id="standard_rights_read"></span><dl> <dt>**STANDARD \_ RECHTE \_ LESEN**</dt><dt></dt> </dl>                  | Schließt **READ \_ CONTROL ein.** Dies ist das Recht, die Informationen in der Sicherheitsbeschreibung des Datei- oder Verzeichnisobjekts zu lesen. Dies schließt die Informationen in der SACL nicht ein.<br/>                                                                                                                                                                                        |
| <span id="STANDARD_RIGHTS_WRITE"></span><span id="standard_rights_write"></span><dl> <dt>**STANDARD \_ RIGHTS \_ WRITE**</dt><dt></dt> </dl>               | Identisch mit **STANDARD \_ RIGHTS \_ READ**.<br/>                                                                                                                                                                                                                                                                                                                                     |
