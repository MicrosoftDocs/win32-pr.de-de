# <a name="directory-management-macros"></a>Verzeichnis Verwaltungs Makros

Die folgenden Makros werden mit Analyse Punkten verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt



| Makro                                                                     | Beschreibung                                                                                                                             |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**Isanalyse-tagmicrosoft**](/windows/desktop/api/Winnt/nf-winnt-isreparsetagmicrosoft)<br/>         | Bestimmt, ob ein Analyse Punkt-Tag einen Microsoft-Analyse Punkt angibt.<br/>                                                  |
| [**Isreparsetagnamesurrogate**](/windows/desktop/api/Winnt/nf-winnt-isreparsetagnamesurrogate)<br/> | Bestimmt, ob der zugeordnete Analyse Punkt eines Tags ein Ersatz für eine andere benannte Entität (z. b. ein eingebundener Ordner) ist.<br/> |
