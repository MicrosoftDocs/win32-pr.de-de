# <a name="directory-management-macros"></a>Verzeichnisverwaltungsmakros

Die folgenden Makros werden mit Aufbereitungspunkten verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt



| Makro                                                                     | Beschreibung                                                                                                                             |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**IsReparseTagMicrosoft**](/windows/desktop/api/Winnt/nf-winnt-isreparsetagmicrosoft)<br/>         | Bestimmt, ob ein Analysepunkttag einen Microsoft-Analysepunkt angibt.<br/>                                                  |
| [**IsReparseTagNameSurrogate**](/windows/desktop/api/Winnt/nf-winnt-isreparsetagnamesurrogate)<br/> | Bestimmt, ob der zugeordnete Analysepunkt eines Tags ein Ersatzzeichen für eine andere benannte Entität (z. B. ein bereitgestellter Ordner) ist.<br/> |
