# <a name="retrieving-file-type-information"></a>Abrufen von Dateityp Informationen

Die [**getfiletype**](/windows/desktop/api/FileAPI/nf-fileapi-getfiletype) -Funktion Ruft den Dateityp ab: Datenträger, Zeichen (z. b. eine Konsole), Pipe oder unbekannt. Die [**getbinarytype**](/windows/desktop/api/WinBase/nf-winbase-getbinarytypea) -Funktion bestimmt, ob es sich um eine ausführbare Datei handelt. ist dies der Fall, ist dies der Typ der ausführbaren Datei. Eine Liste der unterstützten ausführbaren Typen finden Sie unter **getbinarytype** .
