# <a name="retrieving-file-type-information"></a>Abrufen von Dateitypinformationen

Die [**GetFileType-Funktion**](/windows/desktop/api/FileAPI/nf-fileapi-getfiletype) ruft den Typ einer Datei ab: Datenträger, Zeichen (z. B. Konsole), Pipe oder unbekannt. Die [**GetBinaryType-Funktion**](/windows/desktop/api/WinBase/nf-winbase-getbinarytypea) bestimmt, ob eine Datei ausführbar ist, und falls ja, den Typ der ausführbaren Datei. Eine Liste der unterstützten ausführbaren Typen finden Sie unter **GetBinaryType.**
