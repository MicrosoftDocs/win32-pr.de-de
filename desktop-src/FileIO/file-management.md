# <a name="file-management-local-file-systems"></a>Dateiverwaltung (lokale Dateisysteme)

Ein *Datei Objekt* bietet eine Darstellung einer Ressource (entweder ein physisches Gerät oder eine Ressource auf einem physischen Gerät), die vom e/a-System verwaltet werden kann. Wie bei anderen Objekten ermöglichen Sie die Freigabe der Ressource, Sie haben Namen, Sie sind durch objektbasierte Sicherheit geschützt und unterstützen die Synchronisierung. Das e/a-System ermöglicht außerdem das Lesen von oder das Schreiben in die Ressource.
