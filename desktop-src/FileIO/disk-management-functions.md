# <a name="disk-management-functions"></a>Datenträgerverwaltungsfunktionen

Die folgenden Funktionen werden in der Datenträgerverwaltung verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt



| Funktion                                                    | BESCHREIBUNG                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetDiskFreeSpace**](/windows/desktop/api/FileAPI/nf-fileapi-getdiskfreespacea)<br/>     | Ruft Informationen zum angegebenen Datenträger ab, einschließlich des freien Speicherplatzes auf dem Datenträger.<br/>                                                                                                                                                              |
| [**Getdiskfreespaceex**](/windows/desktop/api/FileAPI/nf-fileapi-getdiskfreespaceexa)<br/> | Ruft Informationen über die Menge des verfügbaren Speicherplatzes auf einem Datenträgervolumen ab. Dies ist die Gesamtmenge des Speicherplatzes, die Gesamtmenge des freien Speicherplatzes und die Gesamtmenge des freien Speicherplatzes, der dem Benutzer zur Verfügung steht, der dem aufrufenden Thread zugeordnet ist.<br/> |



 

Andere Funktionen, die in der Datenträgerverwaltung verwendet werden.



| Funktion                         | BESCHREIBUNG                                                                                                                                                                                                                        |
|----------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateFile**](/windows/desktop/api/FileAPI/nf-fileapi-createfilea) | Erstellt oder öffnet eine Datei oder ein E/A-Gerät. Die am häufigsten verwendeten E/A-Geräte sind: Datei, Dateistream, Verzeichnis, physischer Datenträger, Volume, Konsolenpuffer, Bandlaufwerk, Kommunikationsressource, Mailslot und Pipe.<br/> |
| [**DeleteFile**](/windows/desktop/api/FileAPI/nf-fileapi-deletefilea) | Löscht eine vorhandene Datei.<br/>                                                                                                                                                                                               |
