# <a name="disk-management-functions"></a>Funktionen der Datenträgerverwaltung

Die folgenden Funktionen werden in der Datenträgerverwaltung verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt



| Funktion                                                    | BESCHREIBUNG                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetDiskFreeSpace**](/windows/desktop/api/FileAPI/nf-fileapi-getdiskfreespacea)<br/>     | Ruft Informationen zum angegebenen Datenträger ab, einschließlich der Menge an freiem Speicherplatz auf dem Datenträger.<br/>                                                                                                                                                              |
| [**GetDiskFreeSpaceEx**](/windows/desktop/api/FileAPI/nf-fileapi-getdiskfreespaceexa)<br/> | Ruft Informationen über den verfügbaren Speicherplatz auf einem Datenträger Volume ab, d. h. die Gesamtmenge des Speicherplatzes, die Gesamtmenge an freiem Speicherplatz und die Gesamtmenge des verfügbaren freien Speicherplatzes für den Benutzer, der dem aufrufenden Thread zugeordnet ist.<br/> |



 

Andere Funktionen, die in der Datenträgerverwaltung verwendet werden.



| Funktion                         | BESCHREIBUNG                                                                                                                                                                                                                        |
|----------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateFile**](/windows/desktop/api/FileAPI/nf-fileapi-createfilea) | Erstellt oder öffnet ein Datei-oder e/a-Gerät. Die am häufigsten verwendeten e/a-Geräte lauten wie folgt: Datei, Dateistream, Verzeichnis, physischer Datenträger, Volume, Konsolen Puffer, Bandlaufwerk, Kommunikations Ressource, Mailslot und Pipe.<br/> |
| [**DeleteFile**](/windows/desktop/api/FileAPI/nf-fileapi-deletefilea) | Löscht eine vorhandene Datei.<br/>                                                                                                                                                                                               |
