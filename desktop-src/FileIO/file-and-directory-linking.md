# <a name="file-and-directory-linking"></a>Datei- und Verzeichnisverknüpfung

Das NTFS-Dateisystem bietet die Möglichkeit, eine Systemdarstellung einer Datei oder eines Verzeichnisses an einem Speicherort in der Verzeichnisstruktur zu erstellen, die sich von dem Datei- oder Verzeichnisobjekt unterscheidet, mit dem verknüpft wird. Dieser Prozess wird als Verknüpfen bezeichnet. Es gibt zwei Arten von Links, die im NTFS-Dateisystem unterstützt werden: [harte Links und Verbindungen.](hard-links-and-junctions.md)

Das NTFS-Dateisystem stellt auch den Dienst für die [Nachverfolgung verteilter Links](distributed-link-tracking-and-object-identifiers.md)bereit, der Links automatisch nachverfolgt, während sie verschoben werden.
