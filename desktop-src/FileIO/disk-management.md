# <a name="disk-management"></a>Datenträgerverwaltung

Eine *Festplatte ist ein* starrer Datenträger auf einem Computer, der relativ schnellen Zugriff auf große Datenmengen bietet. Dies ist der Speichertyp, der am häufigsten mit der Windows. Das System unterstützt auch Wechselmedien.
