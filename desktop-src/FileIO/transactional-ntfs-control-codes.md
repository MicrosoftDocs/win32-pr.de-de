# <a name="txf-control-codes"></a>TxF-Steuerelementcodes

\[Microsoft empfiehlt Entwicklern dringend, alternative Mittel zu verwenden, um die Anforderungen Ihrer Anwendung zu erfüllen. Viele Szenarien, für die TxF entwickelt wurde, können durch einfachere und leichter verfügbare Techniken erreicht werden. Darüber hinaus ist TxF in zukünftigen Versionen von Microsoft Windows möglicherweise nicht verfügbar. Weitere Informationen und Alternativen zu TxF finden Sie unter [Alternativen zur Verwendung von Transaktional NTFS](deprecation-of-txf.md).\]

Transaktionales NTFS (TxF) stellt die folgenden Steuerungscodes bereit.

## <a name="in-this-section"></a>In diesem Abschnitt



| Steuerungscode                                                                                                 | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                             |
|--------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FSCTL \_ TXFS \_ CREATE \_ MINIVERSION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_create_miniversion)<br/>                         | Erstellt eine neue [Miniversion](glossary.md) für die angegebene Datei. <br/> Mit Miniversionen können Sie während einer Transaktion auf eine Momentaufnahme der Datei verweisen. Miniversionen werden verworfen, wenn für eine Transaktion ein Commit oder rollback ausgeführt wird.<br/>                                                                                                                                                                      |
| [**FSCTL \_ TXFS \_ – ABRUFEN VON \_ \_ METADATENINFORMATIONEN**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_get_metadata_info)<br/>                          | Ruft TxF-Metadaten (Transacted NTFS) für eine Datei und die **GUID** der Transaktion ab, die die angegebene Datei gesperrt hat (wenn die Datei gesperrt ist). <br/>                                                                                                                                                                                                                                                                         |
| [**FSCTL \_ TXFS \_ GET \_ TRANSACTED \_ VERSION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_get_transacted_version)<br/>                | Gibt eine [**TXFS \_ GET \_ TRANSACTED \_ VERSION-Struktur**](/windows/desktop/api/WinIoCtl/ns-winioctl-txfs_get_transacted_version) zurück. Die -Struktur identifiziert die zuletzt ausgeführte Version der angegebenen Datei, die Versionsnummer des Handles. <br/>                                                                                                                                                                                                            |
| [**FSCTL \_ TXFS \_ LIST \_ TRANSACTION \_ LOCKED \_ FILES**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_list_transaction_locked_files)<br/> | Gibt eine Liste aller Dateien zurück, die derzeit von der angegebenen Transaktion gesperrt sind. Wenn der Rückgabewert **ERROR \_ MORE \_ DATA** lautet, wird die Länge des Puffers zurückgegeben, der zum Zeitpunkt dieses Aufrufs zum Speichern der vollständigen Liste der Dateien erforderlich ist.<br/>                                                                                                                                                                                           |
| [**FSCTL \_ TXFS \_ LIST \_ TRANSACTIONS**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_list_transactions)<br/>                           | Gibt eine Liste aller Transaktionen zurück, die derzeit am angegebenen Ressourcen-Manager beteiligt sind.<br/>                                                                                                                                                                                                                                                                                                                                 |
| [**FSCTL \_ TXFS \_ MODIFY \_ RM**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_modify_rm)<br/>                                           | Legt die Protokollmodus- und Protokollparameterinformationen für einen sekundären Ressourcen-Manager (RM) fest.<br/>                                                                                                                                                                                                                                                                                                                                       |
| [**FSCTL \_ TXFS \_ QUERY \_ RM \_ INFORMATION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_query_rm_information)<br/>                    | Ruft Informationen für einen Ressourcen-Manager (RM) ab.<br/>                                                                                                                                                                                                                                                                                                                                                                           |
| [**FSCTL \_ TXFS \_ – LESEN VON \_ \_ SICHERUNGSINFORMATIONEN**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_read_backup_information)<br/>              | Gibt TxF-spezifische Transaktionsinformationen (Transactional NTFS) für die angegebene Datei zurück.<br/>                                                                                                                                                                                                                                                                                                                                                |
| [**FSCTL \_ TXFS \_ SAVEPOINT \_ INFORMATION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_savepoint_information)<br/>                   | Der [**FSCTL \_ TXFS \_ SAVEPOINT \_ INFORMATION-Steuerungscode**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_savepoint_information) steuert das Festlegen, Löschen und Rollback zum angegebenen Sicherungspunkt.<br/> Rufen Sie zum Ausführen dieses Vorgangs die [**DeviceIoControl-Funktion**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) mit den folgenden Parametern auf.<br/>                                                                                                                 |
| [**FSCTL \_ TXFS \_ TRANSACTION \_ ACTIVE**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_transaction_active)<br/>                         | Gibt einen booleschen Wert zurück, der angibt, ob beim Erstellen der Momentaufnahme Transaktionen auf dem zugeordneten Volume aktiv waren. Dieser Aufruf ist nur für schreibgeschützte Momentaufnahmevolumes gültig.<br/>                                                                                                                                                                                                                                   |
| [**FSCTL \_ TXFS \_ WRITE \_ BACKUP \_ INFORMATION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_write_backup_information)<br/>            | Schreibt TxF-spezifische Informationen (Transactional NTFS) in eine angegebene Datei. Der **Puffermember** der [**TXFS \_ WRITE BACKUP \_ \_ INFORMATION-Struktur**](/windows/desktop/api/WinIoCtl/ns-winioctl-txfs_write_backup_information) muss der **Puffermember** der [**TXFS \_ READ BACKUP INFORMATION \_ \_ \_ OUT-Struktur**](/windows/desktop/api/WinIoCtl/ns-winioctl-txfs_read_backup_information_out) sein, die von [**FSCTL \_ TXFS \_ READ BACKUP \_ \_ INFORMATION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_read_backup_information)zurückgegeben wird.<br/> |
