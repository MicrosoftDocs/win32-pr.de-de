# <a name="disk-management-control-codes"></a>Steuerungscodes für die Datenträgerverwaltung

In der folgenden Tabelle sind die Steuercodes aufgeführt, die in der Datenträgerverwaltung verwendet werden.

|-----------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_IOCTL-DATENTRÄGER \_ SIND VOLUMES \_ \_ BEREIT**](ioctl-disk-are-volumes-ready.md)<br/>                       | Wartet, bis alle Volumes auf dem angegebenen Datenträger einsatzbereit sind.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                               |

Um zusätzliche Parameter zur Verfügung zu stellen, verwenden Sie [**stattdessen IOCTL \_ DISK FORMAT TRACKS \_ \_ \_ EX.**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_format_tracks_ex)<br/>                                                                                                                                                                                                                                                                                                                                          |

| [**IOCTL \_ DISK \_ GET \_ DRIVE \_ GEOMETRY \_ EX**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_get_drive_geometry_ex)<br/>              | Ruft erweiterte Informationen zur Geometrie des physischen Datenträgers ab: Typ, Anzahl von Zylindern, Spuren pro Zylinder, Sektoren pro Spur und Bytes pro Sektor.<br/>                                                                                                                                                                                                                                                                                                                                                                    |

| [**IOCTL \_ DISK \_ GET \_ PARTITION \_ INFO \_ EX**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_get_partition_info_ex)<br/>              | Ruft erweiterte Informationen zum Typ, zur Größe und zur Art einer Datenträgerpartition ab.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                           |

| [**LEISTUNG DES \_ \_ IOCTL-DATENTRÄGERS**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_performance)<br/>                                     | Aktiviert Leistungsindikatoren, die Datenträgerleistungsinformationen bereitstellen.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**IOCTL \_ DISK \_ PERFORMANCE \_ OFF**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_performance_off)<br/>                            | Deaktiviert die Leistungsindikatoren, die Datenträgerleistungsinformationen bereitstellen.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**IOCTL \_ DISK \_ REASSIGN \_ BLOCKS**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_reassign_blocks)<br/>                            | Leitet das Datenträgergerät an, einen oder mehrere Blöcke dem Ersatzblockpool zu zuordnen.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**IOCTL \_ DISK \_ REASSIGN \_ BLOCKS \_ EX**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_reassign_blocks_ex)<br/>                     | Leitet das Datenträgergerät an, einen oder mehrere Blöcke dem Ersatzblockpool zu zuordnen.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                     |

| [**IOCTL \_ DISK \_ SET \_ CACHE \_ INFORMATION**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_set_cache_information)<br/>               | Legt die Datenträgerkonfigurationsdaten fest.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |

| [**IOCTL \_ DISK \_ SET \_ DISK \_ ATTRIBUTES**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_set_disk_attributes)<br/>                   | Legt die Attribute des angegebenen Datenträgergeräts fest.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**IOCTL \_ DISK \_ SET \_ DRIVE \_ LAYOUT**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_set_drive_layout)<br/>                         | Partitioniert einen Datenträger gemäß angabe des Laufwerklayouts und der Partitionsinformationsdaten.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**IOCTL \_ DISK \_ SET \_ DRIVE \_ LAYOUT \_ EX**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_set_drive_layout_ex)<br/>                  | Partitioniert einen Datenträger gemäß dem angegebenen Laufwerklayout und partitionsinformationsdaten.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**IOCTL \_ DISK \_ SET \_ PARTITION \_ INFO**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_set_partition_info)<br/>                     | Legt Partitionsinformationen für die angegebene Datenträgerpartition fest.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**IOCTL \_ DISK \_ SET \_ PARTITION \_ INFO \_ EX**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_set_partition_info_ex)<br/>              | Legt Partitionsinformationen für die angegebene Datenträgerpartition fest, einschließlich Layoutinformationen für AT- und EFI-Partitionen (Extensible Firmware Interface).<br/>                                                                                                                                                                                                                                                                                                                                                                           |

| [**IOCTL \_ DISK \_ VERIFY**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_verify)<br/>                                               | Überprüft den angegebenen Wert auf einem Festplattendatenträger.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**IOCTL \_ STORAGE \_ DEVICE \_ POWER \_ CAP**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_device_power_cap)<br/>                   | Windows können diesen Steuerungscode verwenden, um einen maximalen Betriebsverbrauch für ein Speichergerät anzugeben. Das Betriebssystem wird es am besten tun, das Gerät in einen Energiezustand zu überwechseln, der den angegebenen Höchstwert nicht überschreitet. Dies hängt jedoch davon ab, was das Gerät unterstützt. Der tatsächliche Höchstwert kann kleiner oder größer als der gewünschte Höchstwert sein.<br/> Um diesen Vorgang durchzuführen, rufen Sie die [**Funktion DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) mit den folgenden Parametern auf.<br/>                        |
| [**IOCTL \_ STORAGE \_ FIRMWARE \_ ACTIVATE**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_firmware_activate)<br/>                  | Windows-Anwendungen können diesen Steuerungscode verwenden, um ein Firmwareimage auf einem angegebenen Gerät zu aktivieren.<br/> Um diesen Vorgang durchzuführen, rufen Sie die [**Funktion DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) mit den folgenden Parametern auf.<br/>                                                                                                                                                                                                                                                                                          |
| [**HERUNTERLADEN DER \_ \_ IOCTL-SPEICHERFIRMWARE \_**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_firmware_download)<br/>                  | Windows können diesen Steuerungscode verwenden, um ein Firmwareimage auf das Zielgerät herunterzuladen, aber nicht zu aktivieren. Wenn das herunterzuladende Image größer als die maximale Datenübertragungsgröße des Controllers ist, muss diese IOCTL mehrmals aufgerufen werden, bis das gesamte Image heruntergeladen wird.<br/> Um diesen Vorgang durchzuführen, rufen Sie die [**Funktion DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) mit den folgenden Parametern auf.<br/>                                                                                     |
| [**\_IOCTL-SPEICHERFIRMWARE \_ \_ – \_ GET-INFORMATIONEN**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_firmware_get_info)<br/>                 | Windows können diesen Steuerungscode verwenden, um das Speichergerät nach detaillierten Firmwareinformationen zu fragen. Ein erfolgreicher Aufruf gibt Informationen über Firmwarerevisionen, den Aktivitätsstatus sowie Lese-/Schreibattribute für jeden Slot zurück. Die menge der zurückgegebenen Daten variiert je nach Speicherprotokoll.<br/> Um diesen Vorgang durchzuführen, rufen Sie die [**Funktion DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) mit den folgenden Parametern auf.<br/>                                                                         |
| [**\_IOCTL-SPEICHERPROTOKOLLBEFEHL \_ \_**](/windows/desktop/api/winioctl/ni-winioctl-ioctl_storage_protocol_command)<br/>                    | Windows können diesen Steuerungscode verwenden, um Eigenschaften eines Speichergeräts oder Adapters zurück zu geben. Die Anforderung gibt die Art der abzurufenden Informationen an, z. B. Abfragedaten für ein Gerät oder Funktionen und Einschränkungen eines Adapters. <br/>                                                                                                                                                                                                                                                                               |
| [**\_IOCTL-SPEICHERABFRAGEEIGENSCHAFT \_ \_**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_storage_query_property)<br/>                        | Windows können diesen Steuerungscode verwenden, um die Eigenschaften eines Speichergeräts oder Adapters zurück zu geben. Die Anforderung gibt die Art der abzurufenden Informationen an, z. B. die Abfragedaten für ein Gerät oder die Funktionen und Einschränkungen eines Adapters. [**IOCTL \_ STORAGE \_ QUERY \_ PROPERTY**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_storage_query_property) kann auch verwendet werden, um zu bestimmen, ob der Porttreiber eine bestimmte Eigenschaft unterstützt oder welche Felder im Eigenschaftendeskriptor mit einer nachfolgenden Änderungseigenschaftsanforderung geändert werden können.<br/> |
| [**IOCTL \_ STORAGE \_ SET \_ TEMPERATURE \_ THRESHOLD**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_set_temperature_threshold)<br/> | Windows Anwendungen können diesen Steuerungscode verwenden, um den Temperaturschwellenwert eines Geräts (wenn es vom Gerät unterstützt wird) fest zu legen. <br/>                                                                                                                                                                                                                                                                                                                                                                                              |
