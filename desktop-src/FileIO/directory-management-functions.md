# <a name="directory-management-functions"></a>Verzeichnisverwaltungsfunktionen

Die folgenden Funktionen werden in der Verzeichnis Verwaltung verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt



| Funktion                                                                      | BESCHREIBUNG                                                                                                                                                               |
|-------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateDirectory**](/windows/desktop/api/FileAPI/nf-fileapi-createdirectorya)<br/>                         | Erstellt ein neues Verzeichnis.<br/>                                                                                                                                       |
| [**"Kreatedirectoriyex"**](/windows/desktop/api/WinBase/nf-winbase-createdirectoryexa)<br/>                     | Erstellt ein neues Verzeichnis mit den Attributen eines angegebenen Vorlagen Verzeichnisses.<br/>                                                                                 |
| [**"Kreatedirectoriytransagiert"**](/windows/desktop/api/WinBase/nf-winbase-createdirectorytransacteda)<br/>     | Erstellt ein neues Verzeichnis als transaktionalen Vorgang mit den Attributen eines angegebenen Vorlagen Verzeichnisses.<br/>                                                      |
| [**Findclosechangenotifizierung**](/windows/desktop/api/FileAPI/nf-fileapi-findclosechangenotification)<br/> | Beendet die Überwachung des Änderungs Benachrichtigungs Handles.<br/>                                                                                                                   |
| [**FindFirstChangeNotification**](/windows/desktop/api/FileAPI/nf-fileapi-findfirstchangenotificationa)<br/> | Erstellt ein Änderungs Benachrichtigungs Handle und richtet anfängliche Filterbedingungen für die Änderungs Benachrichtigung ein.<br/>                                                                |
| [**FindNextChangeNotification**](/windows/desktop/api/FileAPI/nf-fileapi-findnextchangenotification)<br/>   | Fordert an, dass das Betriebssystem ein Änderungs Benachrichtigungs Handle signalisiert, wenn es das nächste Mal eine entsprechende Änderung erkennt.<br/>                                         |
| [**GetCurrentDirectory**](/windows/desktop/api/WinBase/nf-winbase-getcurrentdirectory)<br/>                 | Ruft das aktuelle Verzeichnis für den aktuellen Prozess ab.<br/>                                                                                                       |
| [**"Read directoriychangesexw"**](/windows/desktop/api/WinBase/nf-winbase-readdirectorychangesexw)<br/>         | Ruft Informationen ab, die die Änderungen im angegebenen Verzeichnis beschreiben. Diese können erweiterte Informationen enthalten, wenn dieser Informationstyp angegeben ist.<br/> |
| [**"Read directoriychangesw"**](/windows/desktop/api/WinBase/nf-winbase-readdirectorychangesw)<br/>             | Ruft Informationen ab, die die Änderungen im angegebenen Verzeichnis beschreiben.<br/>                                                                               |
| [**RemoveDirectory**](/windows/desktop/api/FileAPI/nf-fileapi-removedirectorya)<br/>                         | Löscht ein vorhandenes leeres Verzeichnis.<br/>                                                                                                                           |
| [**Removedirectoriytransagiert**](/windows/desktop/api/WinBase/nf-winbase-removedirectorytransacteda)<br/>     | Löscht ein vorhandenes leeres Verzeichnis als transaktiven Vorgang.<br/>                                                                                                 |
| [**SetCurrentDirectory**](/windows/desktop/api/WinBase/nf-winbase-setcurrentdirectory)<br/>                 | Ändert das aktuelle Verzeichnis für den aktuellen Prozess.<br/>                                                                                                         |
