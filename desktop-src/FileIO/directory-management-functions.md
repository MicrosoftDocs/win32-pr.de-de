# <a name="directory-management-functions"></a>Verzeichnisverwaltungsfunktionen

Die folgenden Funktionen werden in der Verzeichnisverwaltung verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt



| Funktion                                                                      | Beschreibung                                                                                                                                                               |
|-------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateDirectory**](/windows/desktop/api/FileAPI/nf-fileapi-createdirectorya)<br/>                         | Erstellt ein neues Verzeichnis.<br/>                                                                                                                                       |
| [**CreateDirectoryEx**](/windows/desktop/api/WinBase/nf-winbase-createdirectoryexa)<br/>                     | Erstellt ein neues Verzeichnis mit den Attributen eines angegebenen Vorlagenverzeichnisses.<br/>                                                                                 |
| [**CreateDirectoryTransacted**](/windows/desktop/api/WinBase/nf-winbase-createdirectorytransacteda)<br/>     | Erstellt ein neues Verzeichnis als transaktiven Vorgang mit den Attributen eines angegebenen Vorlagenverzeichnisses.<br/>                                                      |
| [**FindCloseChangeNotification**](/windows/desktop/api/FileAPI/nf-fileapi-findclosechangenotification)<br/> | Beendet die Überwachung des Änderungsbenachrichtigungshandle.<br/>                                                                                                                   |
| [**FindFirstChangeNotification**](/windows/desktop/api/FileAPI/nf-fileapi-findfirstchangenotificationa)<br/> | Erstellt ein Änderungsbenachrichtigungshandle und richtet anfängliche Filterbedingungen für Änderungsbenachrichtigungen ein.<br/>                                                                |
| [**FindNextChangeNotification**](/windows/desktop/api/FileAPI/nf-fileapi-findnextchangenotification)<br/>   | Fordert an, dass das Betriebssystem bei der nächsten Erkennung einer entsprechenden Änderung eine Änderungsbenachrichtigung signalisiert.<br/>                                         |
| [**GetCurrentDirectory**](/windows/desktop/api/WinBase/nf-winbase-getcurrentdirectory)<br/>                 | Ruft das aktuelle Verzeichnis für den aktuellen Prozess ab.<br/>                                                                                                       |
| [**ReadDirectoryChangesExW**](/windows/desktop/api/WinBase/nf-winbase-readdirectorychangesexw)<br/>         | Ruft Informationen ab, die die Änderungen innerhalb des angegebenen Verzeichnisses beschreiben, die erweiterte Informationen enthalten können, wenn dieser Informationstyp angegeben ist.<br/> |
| [**ReadDirectoryChangesW**](/windows/desktop/api/WinBase/nf-winbase-readdirectorychangesw)<br/>             | Ruft Informationen ab, die die Änderungen im angegebenen Verzeichnis beschreiben.<br/>                                                                               |
| [**RemoveDirectory**](/windows/desktop/api/FileAPI/nf-fileapi-removedirectorya)<br/>                         | Löscht ein vorhandenes leeres Verzeichnis.<br/>                                                                                                                           |
| [**RemoveDirectoryTransacted**](/windows/desktop/api/WinBase/nf-winbase-removedirectorytransacteda)<br/>     | Löscht ein vorhandenes leeres Verzeichnis als transaktiven Vorgang.<br/>                                                                                                 |
| [**SetCurrentDirectory**](/windows/desktop/api/WinBase/nf-winbase-setcurrentdirectory)<br/>                 | Ändert das aktuelle Verzeichnis für den aktuellen Prozess.<br/>                                                                                                         |
