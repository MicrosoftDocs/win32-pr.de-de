# <a name="systemmonitorreset-method"></a>Systemmonitor:: Reset-Methode

Entfernt alle [**-**](counteritem.md) Ansichts Objekte aus dem **indikatorensammlungs Objekt** und löscht die Diagramm-und Berichts Ansichten.

## <a name="syntax"></a>Syntax


```VB
Sub Reset()
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.
