# <a name="system-monitor"></a>Systemmonitor

## <a name="purpose"></a>Zweck

Systemmonitor (SYSMON) ist die Anwendungsprogrammierschnittstelle (Application Programming Interface, API), die Sie zum Konfigurieren des Microsoft System Monitor ActiveX-Steuerelements verwenden. Mit dem Systemmonitor-Steuerelement können Sie Echtzeit- und zuvor protokollierte Leistungsindikatordaten wie Arbeitsspeicher-, Datenträger- und Prozessorleistungsindikatordaten anzeigen.

## <a name="where-applicable"></a>Anwendungsbereich

Verwenden Sie die SYSMON-API, wenn Sie das Systemmonitor-Steuerelement einem ActiveX-fähigen Container hinzufügen, z. B. einem Microsoft Word-Dokument oder einer webseite, die von Internet Explorer angezeigt wird, und sie dessen Standardverhalten ändern möchten. Beispielsweise können Sie einen Leistungsmonitor erstellen, der nur einen vordefinierten Satz von Systemindikatoren oder vordefinierte Anzeigeeinstellungen anzeigt.

## <a name="developer-audience"></a>Entwicklergruppe

SYSMON ist für die Verwendung durch C-, C++-, Visual Basic- und Skriptentwickler konzipiert.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

SYSMON ist ab Microsoft Windows 2000 verfügbar.

Informationen zu Laufzeitanforderungen für ein bestimmtes Programmierelement finden Sie im Abschnitt Anforderungen der Referenzseite für dieses Element.
