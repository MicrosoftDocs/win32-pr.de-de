# <a name="system-monitor"></a>Systemmonitor

## <a name="purpose"></a>Zweck

System Monitor (Sysmon) ist die Anwendungsprogrammierschnittstelle (Application Programming Interface, API), die Sie zum Konfigurieren des Microsoft System Monitor-ActiveX-Steuer Elements verwenden. Mit dem System Monitor-Steuerelement können Sie Echtzeitdaten und zuvor protokollierte Leistungsindikatoren anzeigen, wie z. b. Arbeitsspeicher, Datenträger und Prozessor Leistungsindikatoren.

## <a name="where-applicable"></a>Anwendungsbereich

Verwenden Sie die sysmon-API, wenn Sie das System Monitor-Steuerelement einem beliebigen ActiveX-fähigen Container hinzufügen, z. b. einem Microsoft Word-Dokument oder einer Webseite, die von Internet Explorer angezeigt wird, und Sie das Standardverhalten ändern möchten. Beispielsweise können Sie einen Leistungs Monitor erstellen, der nur einen vordefinierten Satz von System Indikatoren oder mit vordefinierten Anzeigeeinstellungen anzeigt.

## <a name="developer-audience"></a>Entwicklergruppe

Sysmon ist für die Verwendung durch C-, C++-, Visual Basic-und Skriptentwickler konzipiert.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Sysmon ist ab Microsoft Windows 2000 verfügbar.

Informationen zu den Lauf Zeitanforderungen für ein bestimmtes Programmier Element finden Sie im Abschnitt "Anforderungen" der Referenzseite für dieses Element.
