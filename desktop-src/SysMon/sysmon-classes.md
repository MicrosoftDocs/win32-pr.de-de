# <a name="sysmon-classes"></a>Sysmon-Klassen

Sie können die folgenden Klassen verwenden, um das System Monitor-Steuerelement (Sysmon) zu konfigurieren.



| Klassenname                             | Beschreibung                                                                                                  |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**Ratteritem**](counteritem.md)     | Stellt Eigenschaften Informationen für die Objekt Eigenschaft bereit.                                                                  |
| [**Counters**](counters.md)           | Stellt die Methoden zum Verwalten einer Auflistung von " [**count**](counteritem.md) "-Objekten bereit.                |
| [**Logfileitem**](logfileitem.md)     | Stellt eine Eigenschaft zum Abrufen des Pfads einer einzelnen Protokolldatei bereit, auf die vom System Monitor-Steuerelement zugegriffen wird.     |
| [**LogFiles**](logfiles.md)           | Stellt die Methoden und Eigenschaften für die Verwaltung einer Auflistung von [**logfileitem**](logfileitem.md) -Objekten bereit. |
| [**System Monitor**](systemmonitor.md) | Stellt die Methoden und Eigenschaften bereit, mit denen das System Monitor-Steuerelement konfiguriert wird.                            |
