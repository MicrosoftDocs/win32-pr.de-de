# <a name="systemmonitorshowhorizontalgrid-property"></a>Systemmonitor. showhorizontalgrid (Eigenschaft)

Ruft einen Wert ab, der bestimmt, ob horizontale Rasterlinien im Diagramm angezeigt werden, oder legt ihn fest.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
Property ShowHorizontalGrid As Boolean
```



## <a name="property-value"></a>Eigenschaftswert

True gibt an, dass horizontale Rasterlinien im Diagramm angezeigt werden. andernfalls false. Der Standardwert für diese Eigenschaft ist False.
