# <a name="systemmonitorshowscalelabels-property"></a>Systemmonitor. showscalelabels (Eigenschaft)

Ruft einen Wert ab, der bestimmt, ob die Skalierungs Bezeichnungen auf der vertikalen Achse des Diagramms angezeigt werden, oder legt ihn fest.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
Property ShowScaleLabels As Boolean
```



## <a name="property-value"></a>Eigenschaftswert

True gibt an, dass die Skalierungs Bezeichnungen auf der vertikalen Achse des Diagramms angezeigt werden. andernfalls false. Der Standardwert dieser Eigenschaft ist „TRUE“.
