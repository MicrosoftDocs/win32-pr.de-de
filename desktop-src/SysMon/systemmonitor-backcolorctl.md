# <a name="systemmonitorbackcolorctl-property"></a>Systemmonitor. backcolorctl-Eigenschaft

Ruft die Hintergrundfarbe des Steuer Elements ab oder legt diese fest.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
Property BackColorCtl As stdole.OLE_COLOR
```



## <a name="property-value"></a>Eigenschaftswert

Hintergrundfarbe des Steuer Elements.
