# <a name="logfileitem-object"></a>Logfileitem-Objekt

Verwenden Sie diese Klasse, um den Pfad einer Protokolldatei abzurufen, auf die vom System Monitor-Steuerelement zugegriffen wird.

Rufen Sie zum Abrufen dieses Objekts [**Logfiles. Item**](systemmonitor-logfiles-item.md)auf.

## <a name="members"></a>Member

Das **logfileitem** -Objekt verfügt über diese Typen von Membern:

-   [Eigenschaften](#properties)

### <a name="properties"></a>Eigenschaften

Das **logfileitem** -Objekt verfügt über diese Eigenschaften.



| Eigenschaft                                    | BESCHREIBUNG                                    |
|:--------------------------------------------|:-----------------------------------------------|
| [**ADS**](logfileitem-path.md)<br/> | Ruft den Pfad der Protokolldatei ab.<br/> |
