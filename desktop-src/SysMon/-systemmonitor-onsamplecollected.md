# <a name="systemmonitoronsamplecollected-event"></a>Systemmonitor. onsamplegesammelte-Ereignis

Benachrichtigt Sie, wenn Beispiel Werte für die Leistungsindikatoren erfasst wurden.

## <a name="syntax"></a>Syntax


```VB
SystemMonitor.OnSampleCollected()
```



## <a name="parameters"></a>Parameter

Dieses Ereignis weist keine Parameter auf.

## <a name="return-value"></a>Rückgabewert

Dieses Ereignis gibt keinen Wert zurück.
