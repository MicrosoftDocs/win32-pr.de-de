# <a name="logfilescount-property"></a>Logfiles. Count (Eigenschaft)

Anzahl der [**logfileitem**](logfileitem.md) -Instanzen in der Auflistung.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
Property Count As Long
```



## <a name="property-value"></a>Eigenschaftswert

Anzahl der Protokolldateien in der Sammlung.
