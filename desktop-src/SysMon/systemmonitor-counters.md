# <a name="systemmonitorcounters-property"></a>Systemmonitor. Counters (Eigenschaft)

Ruft die **Zähler** ab, die eine Auflistung von " [**count**](counteritem.md) "-Objekten enthalten.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
Property Counters As ICounters
```



## <a name="property-value"></a>Eigenschaftswert

Ein **indikatorenobjekt, das eine Auflistung** von " [**count**](counteritem.md) "-Objekten enthält.

## <a name="remarks"></a>Bemerkungen

Dies ist die Standard Eigenschaft des [**Systemmonitor**](systemmonitor.md) -Objekts.
