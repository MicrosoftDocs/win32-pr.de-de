# <a name="systemmonitorshowtoolbar-property"></a>Systemmonitor. Show Toolbar (Eigenschaft)

Ruft einen Wert ab, der bestimmt, ob die Symbolleiste auf dem Steuerelement angezeigt wird, oder legt ihn fest.

## <a name="syntax"></a>Syntax


```VB
Property ShowToolbar As Boolean
```



## <a name="property-value"></a>Eigenschaftswert

True gibt an, dass die Symbolleiste auf dem Steuerelement angezeigt wird. andernfalls false. Der Standardwert dieser Eigenschaft ist „TRUE“.

## <a name="remarks"></a>Bemerkungen

Die Symbolleiste ermöglicht dem Benutzer das Hinzufügen oder Entfernen von Leistungsindikatoren, das Auswählen des Diagramms zum Anzeigen der Indikator Daten und das Ändern anderer Eigenschaften des Steuer Elements.
