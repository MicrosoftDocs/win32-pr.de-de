# <a name="systemmonitortimebarcolor-property"></a>SystemMonitor.TimeBarColor-Eigenschaft

Ruft die Farbe der Zeitleiste ab oder legt sie fest. Der Zeitbalken ist der vertikale Balken, der sich über das Diagrammfenster bewegt, um den Durchlauf jedes Samplingintervalls in der Liniendiagrammansicht anzugeben.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
Property TimeBarColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>Eigenschaftswert

Farbe der Zeitleiste.
