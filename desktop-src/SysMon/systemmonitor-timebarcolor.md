# <a name="systemmonitortimebarcolor-property"></a>Systemmonitor. timebarcolor (Eigenschaft)

Ruft die Farbe der Zeitleiste ab oder legt Sie fest. Die Zeitleiste ist der senkrechte Strich, der sich über das Diagramm Fenster bewegt, um den Durchlauf der einzelnen Samplingintervalle in der Liniendiagramm Ansicht anzugeben.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
Property TimeBarColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>Eigenschaftswert

Farbe der Zeitleiste.
