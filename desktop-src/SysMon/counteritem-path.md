# <a name="counteritempath-property"></a>CounterItem.Path-Eigenschaft

Ruft den Pfad des Leistungsindikators ab.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
Property Path As String
```



## <a name="property-value"></a>Eigenschaftswert

Pfad des Leistungsindikators, z. B. "Processor( \_ Total) \\ % Processor Time".
