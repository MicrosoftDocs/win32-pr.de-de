# <a name="systemmonitorgridcolor-property"></a>Systemmonitor. GridColor-Eigenschaft

Ruft die Farbe der im Diagramm verwendeten Rasterlinien ab oder legt Sie fest.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
Property GridColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>Eigenschaftswert

Farbe der horizontalen und vertikalen Rasterlinien, die in einem Diagramm verwendet werden, wenn die Eigenschaften [**showhorizontalgrid**](systemmonitor-showhorizontalgrid.md) oder [**showverticalgrid**](systemmonitor-showverticalgrid.md) ausgewählt sind.
