|---|---|
| [**IRadialControllerConfigurationInterop**](/windows/win32/api/radialcontrollerinterop/nn-radialcontrollerinterop-iradialcontrollerconfigurationinterop)<br/> | Ermöglicht die Interoperabilität mit einem [**RadialControllerConfiguration-Objekt**](/uwp/api/Windows.UI.Input.RadialControllerConfiguration) (Universal Windows Platform, UWP) und bietet Zugriff auf **RadialControllerConfiguration-Member** zum Anpassen eines [**RadialController-Menüs.**](/uwp/api/Windows.UI.Input.RadialController)<br/> |
