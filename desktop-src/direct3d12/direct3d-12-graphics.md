# <a name="direct3d-12-graphics"></a>Direct3D 12-Grafiken

Dieses Programmier Handbuch enthält Informationen zur Verwendung der programmierbaren Direct3D 12-Pipeline zum Erstellen einer angepassten Grafik-Engine.

Die Direct3D 12-Header und-Bibliotheken sind Teil des Windows 10-SDK. Für die Verwendung von Direct3D 12 ist kein separater Download oder eine separate Installation erforderlich.

| Thema | Beschreibung |
|-|-|
| [Direct3D 12-Programmieranleitung](directx-12-programming-guide.md) | Direct3D 12 bietet eine API und Plattform, mit deren Hilfe Apps die Grafik-und Berechnungsfunktionen von PCs nutzen können, die mit mindestens einem Direct3D 12-kompatiblen GPUs ausgestattet sind. |
| [Direct3D 12-Referenz](direct3d-12-reference.md) | In diesem Abschnitt werden APIs für die Direct3D 12-basierte Grafik Programmierung behandelt. |
| [Direct3D 12 Glossar](directx-12-glossary.md) | Diese Begriffe sind von Direct3D 12 unterschiedlichen. |
