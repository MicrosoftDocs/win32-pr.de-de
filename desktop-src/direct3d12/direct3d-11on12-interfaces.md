# <a name="11on12-interfaces"></a>11on12-Schnittstellen
Diese folgenden Schnittstellen werden in d3d11on12.h deklariert.

## <a name="in-this-section"></a>In diesem Abschnitt
| Thema | BESCHREIBUNG |
|-|-|
| [**ID3D11On12Device**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device) | Verarbeitet das Erstellen, Umschließen und Freigeben von D3D11-Ressourcen für Direct3D 11on12. |
| [**ID3D11On12Device1**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device1) | Ermöglicht eine bessere Interoperabilität mit einer Komponente, die möglicherweise an ein Direct3D 11-Gerät übergeben wird, aber stattdessen Direct3D 12 nutzen möchte. |
| [**ID3D11On12Device2**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device2) | Hier können Sie Ressourcen, die über die Direct3D 11-APIs erstellt wurden, in Direct3D 12 verwenden. Diese Schnittstelle erweitert [ID3D11On12Device1](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device1). |
