# <a name="d3d12_gpu_virtual_address"></a>\_Virtuelle GPU \_ - \_ Adresse D3D12

Eine D3D12_GPU_VIRTUAL_ADDRESS die eine virtuelle Adresse im GPU-Speicher darstellt. Der-Datentyp wird in d3d12. h wie folgt definiert:


```
typedef UINT64 D3D12_GPU_VIRTUAL_ADDRESS;
```
