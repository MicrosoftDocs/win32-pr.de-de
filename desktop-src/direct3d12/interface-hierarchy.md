# <a name="interface-hierarchy"></a>Schnittstellenhierarchie

Das Diagramm zeigt die Hierarchie der Schnittstellen Vererbung.

In den meisten apps werden nur die Schnittstellen am Anfang der Hierarchie verwendet, und die geerbten Schnittstellen werden nicht direkt verwendet. [**ID3D12Pageable**](/windows/win32/api/d3d12/nn-d3d12-id3d12pageable), [**ID3D12DeviceChild**](/windows/desktop/api/d3d12/nn-d3d12-id3d12devicechild), [**ID3D12Object**](/windows/desktop/api/d3d12/nn-d3d12-id3d12object) und [**ID3D12CommandList**](/windows/desktop/api/d3d12/nn-d3d12-id3d12commandlist).

Neue Schnittstellen werden in einem grünen Feld angezeigt.
