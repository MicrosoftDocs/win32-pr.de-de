# <a name="helper-interfaces-for-direct3d-12"></a>Hilfsschnittstellen für Direct3D 12

Diese Hilfsschnittstellen helfen insbesondere bei der Verarbeitung von Unterressourcen und werden in `d3dx12.h` deklariert.

`d3dx12.h` ist getrennt von den Direct3D 12-Headern verfügbar. Sie können `d3dx12.h` von [der D3D12-Hilfsbibliothek](https://github.com/microsoft/DirectX-Headers/blob/main/include/directx/d3dx12.h)herunterladen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | Beschreibung |
|-|-|
| [**ID3DX12PipelineParserCallbacks**](id3dx12pipelineparsercallbacks.md) | Eine Schnittstelle, die eine Auflistung von Parser- und Fehlerrückrufen darstellt, die von der [**D3DX12parsePipelineStream-Hilfsfunktion**](d3dx12parsepipelinestream.md) verwendet werden. |
