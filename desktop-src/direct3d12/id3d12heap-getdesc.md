# <a name="id3d12heapgetdesc-method"></a>ID3D12Heap::GetDesc-Methode

Ruft die Heapbeschreibung ab.

## <a name="syntax"></a>Syntax


```C++
D3D12_HEAP_DESC GetDesc();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **D3D12 \_ HEAP \_ DESC**](/windows/desktop/api/d3d12/ns-d3d12-d3d12_heap_desc)**

Gibt die [**D3D12 \_ HEAP \_ DESC-Struktur**](/windows/desktop/api/d3d12/ns-d3d12-d3d12_heap_desc) zurück, die den Heap beschreibt.
