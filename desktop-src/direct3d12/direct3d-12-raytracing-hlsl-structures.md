# <a name="raytracing-hlsl-structures-direct3d-12"></a>Raytracing HLSL-Strukturen (Direct3D 12)

Die folgenden HLSL-Strukturen unterstützen die Direct3D 12-Raytracing-Pipeline.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                                       | BESCHREIBUNG                                                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Struktur von Aufrufparametern**](call-parameter-structure.md)<br/>                              | Eine benutzerdefinierte Struktur, die als ein INOUT-Argument für einen callshader-Aufruf bereitgestellt wird, und als INOUT-Parameter für den Aufruf baren Shader.<br/>                                                                                                                                                                                                                                              |
| [**Struktur von Schnittmengenattributen**](intersection-attributes.md)<br/>                              | Eine benutzerdefinierte Struktur, die als ein INOUT-Argument in einem [**traceray**](traceray-function.md) -Befehl bereitgestellt wird, und als INOUT-Parameter in den shadertypen, die möglicherweise auf die Ray-Nutzlast zugreifen.<br/>                                                                                                                                                                                                                                              |
| [**Struktur von Ray-Nutzlast**](ray-payload.md)<br/>                              | Eine benutzerdefinierte Struktur, die als ein INOUT-Argument in einem [**traceray**](traceray-function.md) -Befehl bereitgestellt wird, und als INOUT-Parameter in den shadertypen, die möglicherweise auf die Ray-Nutzlast zugreifen.<br/>                                                                                                                                                                                                                                              |
| [**RayDesc-Struktur**](raydesc.md)<br/>                              | Flags, die an die [**traceray**](traceray-function.md) -Funktion übergeben werden, um Transparenz, Vererbung und frühe Ausgabeverhalten zu überschreiben.<br/>                                                                                                                                                                                                                                              |
