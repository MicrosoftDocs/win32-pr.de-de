# <a name="dwm-structures"></a>DWM-Strukturen

Dieser Abschnitt enthält Informationen zu den DWM-Strukturen (Desktopfenster-Manager).

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                     | BESCHREIBUNG                                                                                                                                               |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DWM \_ BLURBEHIND**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_blurbehind)<br/>                      | Gibt DWM-BlurBehind-Eigenschaften an. Wird von der [**DwmEnableBlurBehindWindow-Funktion**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmenableblurbehindwindow) verwendet.<br/>                     |
| [**DWM \_ PRESENT \_ PARAMETERS**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_present_parameters)<br/>     | Gibt DWM-Videoframeparameter für die Framekomposition an. Wird von der [**DwmSetPresentParameters-Funktion**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmsetpresentparameters) verwendet.<br/>   |
| [**EIGENSCHAFTEN \_ VON DWM-MINIATURANSICHTEN \_**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_thumbnail_properties)<br/> | Gibt DWM-Miniaturansichtseigenschaften an. Wird von der [**DwmUpdateThumbnailProperties-Funktion**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmupdatethumbnailproperties) verwendet.<br/>                 |
| [**\_ \_ DWM-ZEITSTEUERUNGSINFORMATIONEN**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_timing_info)<br/>                   | Gibt Informationen zur Zeitlichen Steuerung der DWM-Komposition an. Wird von der [**DwmGetCompositionTimingInfo-Funktion**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmgetcompositiontiminginfo) verwendet.<br/>         |
| [**MilMatrix3x2D**](/windows/desktop/api/Dwmapi/ns-dwmapi-milmatrix3x2d)<br/>                         | Gibt eine 3x2-Matrix an, die eine Transformation beschreibt. <br/>                                                                                            |
| [**VERHÄLTNIS OHNE VORZEICHEN \_**](/windows/desktop/api/Dwmapi/ns-dwmapi-unsigned_ratio)<br/>                      | Definiert einen datentyp, der von den DWM-APIs verwendet wird. Es stellt ein generisches Verhältnis dar und wird für verschiedene Zwecke und Einheiten verwendet, auch innerhalb einer einzelnen API.<br/> |
