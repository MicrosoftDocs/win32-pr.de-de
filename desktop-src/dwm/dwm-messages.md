# <a name="dwm-messages"></a>DWM-Nachrichten

Dieser Abschnitt enthält Informationen zu den Desktopfenster-Manager -Nachrichten (DWM).

## <a name="in-this-section"></a>In diesem Abschnitt




| Thema | BESCHREIBUNG | 
|-------|-------------|
| <a href="wm-dwmcolorizationcolorchanged.md"><strong>WM_DWMCOLORIZATIONCOLORCHANGED</strong></a><br /> | Informiert alle Fenster der obersten Ebene darüber, dass sich die Farbgebungsfarbe geändert hat.<br /> | 
| <a href="wm-dwmcompositionchanged.md"><strong>WM_DWMCOMPOSITIONCHANGED</strong></a><br /> | Informiert alle Fenster der obersten Ebene darüber, dass die DWM-Komposition aktiviert oder deaktiviert wurde. <br /><blockquote>[!Note]<br />Ab dem Windows 8 ist die DWM-Komposition immer aktiviert, sodass diese Nachricht unabhängig von Änderungen des Videomodus nicht gesendet wird.</blockquote><br /> | 
| <a href="wm-dwmncrenderingchanged.md"><strong>WM_DWMNCRENDERINGCHANGED</strong></a><br /> | Wird gesendet, wenn sich die Richtlinie zum Rendern von Nicht-Clientbereich geändert hat.<br /> | 
| <a href="wm-dwmsendiconiclivepreviewbitmap.md"><strong>WM_DWMSENDICONICLIVEPREVIEWBITMAP</strong></a><br /> | Weist ein Fenster an, eine statische Bitmap zur Verwendung als <em></em>Livevorschau <em>(auch</em> als Vorschau anzeigen) dieses Fensters zu verwenden.<br /> | 
| <a href="wm-dwmsendiconicthumbnail.md"><strong>WM_DWMSENDICONICTHUMBNAIL</strong></a><br /> | Weist ein Fenster an, eine statische Bitmap zur Verwendung als Miniaturansicht dieses Fensters zur Verfügung zu stellen.<br /> | 
| <a href="wm-dwmwindowmaximizedchange.md"><strong>WM_DWMWINDOWMAXIMIZEDCHANGE</strong></a><br /> | Wird gesendet, wenn ein dwm-zusammengestelltes Fenster maximiert ist.<br /> |
