# <a name="dwm-enable-composition-constants"></a>DWM– Aktivieren von Kompositionskonstanten

Flags, die von der [**DwmEnableComposition-Funktion**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmenablecomposition)  verwendet werden, um den Zustand der DWM-Komposition (Desktopfenster-Manager) zu ändern.

<dl> <dt>

<span id="DWM_EC_DISABLECOMPOSITION"></span><span id="dwm_ec_disablecomposition"></span>**DWM \_ EC \_ DISABLECOMPOSITION**
</dt> <dd> <dl> <dt>

0
</dt> <dt>



Deaktivieren Sie die Komposition.


</dt> </dl> </dd> <dt>

<span id="DWM_EC_ENABLECOMPOSITION"></span><span id="dwm_ec_enablecomposition"></span>**DWM \_ EC \_ ENABLECOMPOSITION**
</dt> <dd> <dl> <dt>

1
</dt> <dt>



Aktivieren Sie die Komposition.


</dt> </dl> </dd> </dl>
