# <a name="systemmusicalbumartistsortoverride"></a>System. Music. albumartistsorbuverride

Dieser optionale Zeichen folgen Wert ermöglicht das Überschreiben der standardmäßigen Sortierreihenfolge von System. Music. albumartist. Dies ist sehr wichtig für eine ordnungsgemäße Sortierung von Musikdateien in Japanisch, die phonetisch (die vom Benutzer erwartete Reihenfolge) ohne dieses Feld nicht vollständig sortiert werden kann. Sie kann auch zum Anpassen von Sortierungen in nicht ostasiatischen Szenarios verwendet werden, z. b. wenn der Benutzer das Entfernen von Artikeln zu Sortier Zwecken ermöglicht.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Music.AlbumArtistSortOverride
   shellPKey = PKEY_Music_AlbumArtistSortOverride
   formatID = F1FDB4AF-F78C-466C-BB05-56E92DB0B8EC
   propID = 103
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
