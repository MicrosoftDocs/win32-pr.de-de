# <a name="systemphotoflashmanufacturer"></a>System. Photo. Flash Hersteller

Eine Zeichenfolge, die den Hersteller des Flash-Bilds angibt, der zum Bild verwendet wird. Kann leer sein oder nicht vorhanden sein.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FlashManufacturer
   shellPKey = PKEY_Photo_FlashManufacturer
   formatID = AABAF6C9-E0C5-4719-8585-57B103E584FE
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
