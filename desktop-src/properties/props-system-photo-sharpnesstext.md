# <a name="systemphotosharpnesstext"></a>System. Photo. sharpnesstext

Die benutzerfreundliche Form von [System. Photo. Schärding](./props-system-photo-sharpness.md). Es ist nicht vorgesehen, Programm gesteuert analysiert zu werden.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.SharpnessText
   shellPKey = PKEY_Photo_SharpnessText
   formatID = 51EC3F47-DD50-421D-8769-334F50424B1E
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
