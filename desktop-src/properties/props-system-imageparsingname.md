# <a name="systemimageparsingname"></a>System. imageparsingname

Der Name der Bildverarbeitung.

```
propertyDescription
   name = System.ImageParsingName
   shellPKey = PKEY_ImageParsingName
   formatID = D7750EE0-C6A4-48EC-B53E-B87B52E6D073
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Multivalue String
```
