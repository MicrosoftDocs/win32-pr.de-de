# <a name="systemmusicalbumtitlesortoverride"></a>System. Music. albumtitlesorbuverride

Mit diesem optionalen Zeichen folgen Wert kann die standardmäßige Sortierreihenfolge von System. Music. Album überschrieben werden. Dies ist sehr wichtig für die ordnungsgemäße Sortierung von Musikdateien in Japanisch, die nicht nach phonetisch (der vom Benutzer erwarteten Reihenfolge) ohne dieses Feld sortiert werden kann. Sie kann auch zum Anpassen von Sortierungen in nicht ostasiatischen Szenarios verwendet werden, z. b. wenn der Benutzer das Entfernen von Artikeln zu Sortier Zwecken ermöglicht.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Music.AlbumTitleSortOverride
   shellPKey = PKEY_Music_AlbumTitleSortOverride
   formatID = 13EB7FFC-EC89-4346-B19D-CCC6F1784223
   propID = 101
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
