# <a name="systemmediaseasonnumber"></a>System. Media. seasonnumber

Eine 1-basierte, monoton inkrementierte Zahl, die der Saison entspricht, in der die Show erstmals präsentiert wurde.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Media.SeasonNumber
   shellPKey = PKEY_Media_SeasonNumber
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 101
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
