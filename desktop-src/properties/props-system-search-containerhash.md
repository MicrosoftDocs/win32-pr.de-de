# <a name="systemsearchcontainerhash"></a>System. search. containerhash

Der Hashcode, der zum Identifizieren der zu löschenden Anhänge basierend auf einer gemeinsamen Container-URL verwendet wird.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.ContainerHash
   shellPKey = PKEY_Search_ContainerHash
   formatID = BCEEE283-35DF-4D53-826A-F36A3EEFC6BE
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
