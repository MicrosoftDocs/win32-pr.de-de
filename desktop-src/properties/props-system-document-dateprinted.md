# <a name="systemdocumentdateprinted"></a>System.DocUmschlag. Dateprint

Gibt das Datum und die Uhrzeit des letzten Druckens des Dokuments an. Der Legacy Name ist "DOCLASTPRINT".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Document.DatePrinted
   shellPKey = PKEY_Document_DatePrinted
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 11
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Document.DatePrinted
   shellPKey = PKEY_Document_DatePrinted
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 11
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
