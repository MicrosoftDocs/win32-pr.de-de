# <a name="systemdevicesnotworkingproperly"></a>System. Devices. notworkingrichtig

Wenn Variant \_ true ist, funktioniert das Gerät nicht ordnungsgemäß.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.NotWorkingProperly
   shellPKey = PKEY_Devices_NotWorkingProperly
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 83
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enum
            name = NotWorkingProperly
            value = #TRUE#
            text = Needs troubleshooting
            defineToken = ISNOTWORKINGPROPERLY_NOTWORKINGPROPERLY
```

## <a name="windows-7"></a>Windows 7

```
propertyDescription
   name = System.Devices.NotWorkingProperly
   shellPKey = PKEY_Devices_IsNotWorkingProperly
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 83
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enum
            name = NotWorkingProperly
            value = #TRUE#
            text = Needs troubleshooting
            defineToken = ISNOTWORKINGPROPERLY_NOTWORKINGPROPERLY
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
