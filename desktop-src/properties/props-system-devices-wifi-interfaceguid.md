# <a name="systemdeviceswifiinterfaceguid"></a>System. Devices. WiFi. interfaceguid

GUID der Wi-Fi-Schnittstelle

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.Devices.WiFi.InterfaceGuid
   shellPKey = PKEY_Devices_WiFi_InterfaceGuid
   formatID = EF1167EB-CBFC-4341-A568-A7C91A68982C
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

Pkey-Werte werden in "propkey. h" definiert.
