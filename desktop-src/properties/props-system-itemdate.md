# <a name="systemitemdate"></a>System. itemdate

Das primäre relevante Datum für ein Element. Im Fall von Fotos wird diese Eigenschaft z. b. [System. Photo. DateTaken](./props-system-photo-datetaken.md)zugeordnet.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.ItemDate
   shellPKey = PKEY_ItemDate
   formatID = F7DB74B4-4287-4103-AFBA-F1B13DCD75CF
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.ItemDate
   shellPKey = PKEY_ItemDate
   formatID = F7DB74B4-4287-4103-AFBA-F1B13DCD75CF
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
