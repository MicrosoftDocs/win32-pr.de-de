# <a name="systemcomputerdecoratedfreespace"></a>System. Computer. decoratedfreespace

Der freie Speicherplatz des Computers und der gesamte Speicherplatz als "% s frei von% s".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Computer.DecoratedFreeSpace
   shellPKey = PKEY_Computer_DecoratedFreeSpace
   formatID = 9B174B35-40FF-11D2-A27E-00C04FC30871
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue UInt64
      IsInnate = true
```

Pkey-Werte werden in "propkey. h" definiert.
