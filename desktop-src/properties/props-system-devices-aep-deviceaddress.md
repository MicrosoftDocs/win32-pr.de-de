# <a name="systemdevicesaepdeviceaddress"></a>System.Devices.Aep.DeviceAddress

Adresse basierend auf dem Protokoll des Geräte Zuordnungs Endpunkts. IP-Adresse für ein IP-Gerät, Bluetooth-Adresse für Bluetooth-Gerät usw.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.Devices.Aep.DeviceAddress
   shellPKey = PKEY_Devices_Aep_DeviceAddress
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
