# <a name="systemlastwriterpackagefamilyname"></a>System. lastbeschreiterpackagefamilyname

Markierung des App-Containers. Der Paket Familienname der letzten APP, die den Inhalt der Datei bearbeiten soll.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.LastWriterPackageFamilyName
   shellPKey = PKEY_LastWriterPackageFamilyName
   formatID = 502CFEAB-47EB-459C-B960-E6D8728F7701
   propID = 101
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
