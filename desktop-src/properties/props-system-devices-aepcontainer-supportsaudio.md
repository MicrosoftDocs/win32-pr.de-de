# <a name="systemdevicesaepcontainersupportsaudio"></a>System.Devices.AepContainer.SupportsAudio

Gibt an, ob der Geräte Zuordnungs Endpunkt-Container Audioumwandlung unterstützt

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.Devices.AepContainer.SupportsAudio
   shellPKey = PKEY_Devices_AepContainer_SupportsAudio
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
