# <a name="systemsearchentryid"></a>System.Search.EntryID

Die Eintrags-ID für ein Element innerhalb eines bestimmten Katalogs im Windows Suchindex.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.EntryID
   shellPKey = PKEY_Search_EntryID
   formatID = 49691C90-7E17-101A-A91C-08002B2ECDA9
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
      IsInnate = true
```

## <a name="remarks"></a>Hinweise

[System.Search.EntryID]() wird in der SQL verwendet, die generiert wird, um den Index abzufragen. Dieser Wert wird im Laufe der Zeit nicht als eindeutig betrachtet, da er möglicherweise wiederverwendet wird.

PKEY-Werte werden in Propkey.h definiert.
