# <a name="systemphotocontrasttext"></a>System. Photo. kontra Text

Die benutzerfreundliche Form von System. Photo. contrast. Er soll nicht Programm gesteuert analysiert werden.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.ContrastText
   shellPKey = PKEY_Photo_ContrastText
   formatID = 59DDE9F2-5253-40EA-9A8B-479E96C6249A
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```
