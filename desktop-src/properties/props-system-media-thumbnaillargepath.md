# <a name="systemmediathumbnaillargepath"></a>System. Media. thumbnaillargepath

Dateisystempfad zur großen Miniatur Ansichts Darstellung des Medien Elements.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.Media.ThumbnailLargePath
   shellPKey = PKEY_Media_ThumbnailLargePath
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 47
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
