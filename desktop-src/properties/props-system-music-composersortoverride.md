# <a name="systemmusiccomposersortoverride"></a>System. Music. composersorbuverride

Mit diesem optionalen Zeichen folgen Wert kann die standardmäßige Sortierreihenfolge von System. Music. Composer überschrieben werden. Dies ist sehr wichtig für die ordnungsgemäße Sortierung von Musikdateien in Japanisch, die nicht nach phonetisch (der vom Benutzer erwarteten Reihenfolge) ohne dieses Feld sortiert werden kann. Sie kann auch zum Anpassen von Sortierungen in nicht ostasiatischen Szenarios verwendet werden, z. b. wenn der Benutzer das Entfernen von Artikeln zu Sortier Zwecken ermöglicht.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Music.ComposerSortOverride
   shellPKey = PKEY_Music_ComposerSortOverride
   formatID = 00BC20A3-BD48-4085-872C-A88D77F5097E
   propID = 105
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
