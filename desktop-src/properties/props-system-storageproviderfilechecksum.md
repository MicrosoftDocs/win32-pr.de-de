# <a name="systemstorageproviderfilechecksum"></a>System. storageproviderfilechecksum

Die Prüfsumme, die vom Speicher Anbieter für die Datei berechnet wird. Dateien mit demselben Prüfsummen Wert haben denselben Inhalt.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.StorageProviderFileChecksum
   shellPKey = PKEY_StorageProviderFileChecksum
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
      IsInnate = false
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
