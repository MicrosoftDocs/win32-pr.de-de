# <a name="systemdeviceinterfaceproximitysupportsnfc"></a>System. System. Gerätename. near. supportsnfc

Gibt an, ob das Gerät NFC Communications (ndef) unterstützt.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.DeviceInterface.Proximity.SupportsNfc
   shellPKey = PKEY_DeviceInterface_Proximity_SupportsNfc
   formatID = FB3842CD-9E2A-4F83-8FCC-4B0761139AE9
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
