# <a name="systemdevicesaepbluetoothleisconnectable"></a>System. Devices. aep. Bluetooth. Le. isconnectable

Gibt an, ob das Bluetooth Le-Gerät zurzeit eine Verbindungs fähige Ankündigung sendet.

## <a name="windows-10-version-1703"></a>Windows 10, Version 1703

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Le.IsConnectable
   shellPKey = PKEY_Devices_Aep_Bluetooth_Le_IsConnectable
   formatID = 995EF0B0-7EB3-4A8B-B9CE-068BB3F4AF69
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
