# <a name="systemimagecompressiontext"></a>System. Image. compressiontext

Die benutzerfreundliche Form von System. Image. Compression. Es ist nicht vorgesehen, Programm gesteuert analysiert zu werden.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Image.CompressionText
   shellPKey = PKEY_Image_CompressionText
   formatID = 3F08E66F-2F44-4BB9-A682-AC35D2562322
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

Pkey-Werte werden in "propkey. h" definiert.
