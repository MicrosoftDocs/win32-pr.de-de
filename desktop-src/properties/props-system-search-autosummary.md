# <a name="systemsearchautosummary"></a>System. search. autosummary

Eine automatisierte Suchsystem Zusammenfassung des voll Text Inhalts eines Dokuments, das in der Suchergebnis Ansicht im **Such Explorer** angezeigt wird. Verwenden Sie diese Eigenschaft nur, wenn Sie die Werte in einer Anwendung verwenden, nicht bei der Angabe von Werten für einen Eigenschafts Handler.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.AutoSummary
   shellPKey = PKEY_Search_AutoSummary
   formatID = 560C36C0-503A-11CF-BAA1-00004C752A9A
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
