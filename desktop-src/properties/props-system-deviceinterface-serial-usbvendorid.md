# <a name="systemdeviceinterfaceserialusbvendorid"></a>System. de viceinterface. Serial. US-VendorID

USB-Anbieter-ID des seriellen Geräts.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.DeviceInterface.Serial.UsbVendorId
   shellPKey = PKEY_DeviceInterface_Serial_UsbVendorId
   formatID = 4C6BF15C-4C03-4AAC-91F5-64C0F852BCF4
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
