# <a name="systemdevicesdevobjecttype"></a>System.Devices.DevObjectType

DevQuery-Geräteobjekttyp

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.Devices.DevObjectType
   shellPKey = PKEY_Devices_DevObjectType
   formatID = 13673F42-A3D6-49F6-B4DA-AE46E0C5237C
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>Hinweise

PKEY-Werte werden in Propkey.h definiert.
