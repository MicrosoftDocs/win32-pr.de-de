# <a name="systemparentalratingsorganization"></a>System. parametrialratingsorganization

Der Name der Organisation, deren Bewertungssystem für " [System. parentalrating](./props-system-parentalrating.md)" verwendet wird.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ParentalRatingsOrganization
   shellPKey = PKEY_ParentalRatingsOrganization
   formatID = A7FE0840-1344-46F0-8D37-52ED712A4BF9
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
