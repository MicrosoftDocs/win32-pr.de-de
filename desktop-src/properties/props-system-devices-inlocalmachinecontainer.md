# <a name="systemdevicesinlocalmachinecontainer"></a>System. Devices. inlocalmachinecontainer

Das Gerät befindet sich im Container "lokaler Computer".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.InLocalMachineContainer
   shellPKey = PKEY_Devices_InLocalMachineContainer
   formatID = 8C7ED206-3F8A-4827-B3AB-AE9E1FAEFC6C
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
