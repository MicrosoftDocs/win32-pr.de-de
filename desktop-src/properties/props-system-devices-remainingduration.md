# <a name="systemdevicesremainingduration"></a>System. Devices. restingduration

Die verbleibende Wiedergabezeit auf dem Gerät in 100-ns-Einheiten.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.RemainingDuration
   shellPKey = PKEY_Devices_RemainingDuration
   formatID = 3633DE59-6825-4381-A49B-9F6BA13A1471
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
