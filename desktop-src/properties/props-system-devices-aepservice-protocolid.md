# <a name="systemdevicesaepserviceprotocolid"></a>System.Devices.AepService.ProtocolId

Identität des Protokolls, über das dieser Geräte Zuordnungs Endpunkt-Dienst ermittelt wurde

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.Devices.AepService.ProtocolId
   shellPKey = PKEY_Devices_AepService_ProtocolId
   formatID = C9C141A9-1B4C-4F17-A9D1-F298538CADB8
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
