# <a name="systemcreatoropenwithuioptions"></a>System. kreatoropenwithuioptions

Stellt Optionen bereit, die das Verhalten von UI-Steuerelementen beeinflussen, mit denen die Datei mit der in "System. kreatorappid" angegebenen app gestartet wird.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.CreatorOpenWithUIOptions
   shellPKey = PKEY_CreatorOpenWithUIOptions
   formatID = C2EA046E-033C-4E91-BD5B-D4942F6BBE49
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = false
      EnumeratedList
         UseValueForDefault = True
         enum
            name = Hidden
            value = 0
            text = Hidden
            defineToken = CREATOROPENWITHUIOPTION_HIDDEN
         enum
            name = Visible
            value = 1
            text = Visible
            defineToken = CREATOROPENWITHUIOPTION_VISIBLE
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
