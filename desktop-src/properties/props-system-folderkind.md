# <a name="systemfolderkind"></a>System. folderkind

Diese Eigenschaft stellt die Inhaltstypen dar, die in diesem Ordner gespeichert sind, der vom Speicher Anbieter angegeben wird. Jeder Ordnertyp muss einer der bekannten Werte sein, die von System. Kind definitionSystem angegeben werden. folderkind ist eine schreibgeschützte Eigenschaft, die nur vom Speicher Anbieter aktualisiert werden soll.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1

```
propertyDescription
   name = System.FolderKind
   shellPKey = PKEY_FolderKind
   formatID = FCEFF153-E839-4CF3-A9E7-EA22832094B8
   propID = 101
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
