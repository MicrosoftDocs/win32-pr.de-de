# <a name="systemsearchrank"></a>System.Search.Rank

Relevanzrang der Zeile mit einem Bereich zwischen 0 und 1000. Größere Zahlen bedeuten bessere Übereinstimmungen. Nur Abfragezeit; nicht im Suchschema definiert. Diese Eigenschaft ist abrufbar, aber nicht durchsuchbar.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.Rank
   shellPKey = PKEY_Search_Rank
   formatID = 49691C90-7E17-101A-A91C-08002B2ECDA9
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
      IsInnate = true
```

## <a name="remarks"></a>Hinweise

PKEY-Werte werden in Propkey.h definiert.
