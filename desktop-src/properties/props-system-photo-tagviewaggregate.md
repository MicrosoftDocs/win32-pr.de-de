# <a name="systemphototagviewaggregate"></a>System. Photo. tagviewaggregate

Eine schreibgeschützte Aggregation von tagähnlichen Eigenschaften zur Verwendung beim Aufbau von Sichten.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Photo.TagViewAggregate
   shellPKey = PKEY_Photo_TagViewAggregate
   formatID = B812F15D-C2D8-4BBF-BACD-79744346113F
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
