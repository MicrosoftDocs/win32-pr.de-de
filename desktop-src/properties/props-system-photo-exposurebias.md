# <a name="systemphotoexposurebias"></a>System. Photo. exposurebias

Die Menge des auf dem Foto verwendeten Expositions-Bias, wie von der Kamera gelesen. Diese Eigenschaft wird aus [System. Photo. exposurebiasnumerator](./props-system-photo-exposurebiasnumerator.md) und [System. Photo. exposurebiasnenner](./props-system-photo-exposurebiasdenominator.md)berechnet. Die Einheit ist der Apex-Wert. Normalerweise liegt der Bereich zwischen-99,99 und 99,99.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.ExposureBias
   shellPKey = PKEY_Photo_ExposureBias
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 37380
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
