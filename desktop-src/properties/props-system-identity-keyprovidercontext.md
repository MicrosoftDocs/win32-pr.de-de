# <a name="systemidentitykeyprovidercontext"></a>System. Identity. keyprovidercontext

Identitätsschlüssel-Anbieter Kontext

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Identity.KeyProviderContext
   shellPKey = PKEY_Identity_KeyProviderContext
   formatID = A26F4AFC-7346-4299-BE47-EB1AE613139F
   propID = 17
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

Pkey-Werte werden in "propkey. h" definiert.
