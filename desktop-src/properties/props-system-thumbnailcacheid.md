# <a name="systemthumbnailcacheid"></a>System. thumbnailcacheid

Ein eindeutiger Wert, der als Schlüssel zum Zwischenspeichern von Miniaturansichten dient. Der Wert ändert sich, wenn sich der Name, das Volume oder die Daten ändern, die von einem Element geändert werden.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ThumbnailCacheId
   shellPKey = PKEY_ThumbnailCacheId
   formatID = 446D16B1-8DAD-4870-A748-402EA43D788C
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt64
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
