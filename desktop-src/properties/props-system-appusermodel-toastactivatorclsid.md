# <a name="systemappusermodeltoastactivatorclsid"></a>System. appusermodel. $ astactivatorclsid

Wird zum CoCreate einer inotificationactivationcallback-Schnittstelle verwendet, um Informationen über Popup Aktivierungen zu benachrichtigen.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.AppUserModel.ToastActivatorCLSID
   shellPKey = PKEY_AppUserModel_ToastActivatorCLSID
   formatID = 9F4C2855-9F79-4B39-A8D0-E1D42DE1D5F3
   propID = 26
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = false
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
