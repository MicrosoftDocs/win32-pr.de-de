Die benutzerfreundliche Form von [System. Photo. meteringmode](./props-system-photo-meteringmode.md). Es ist nicht vorgesehen, Programm gesteuert analysiert zu werden.

```
propertyDescription
   name = System.Photo.MeteringModeText
   shellPKey = PKEY_Photo_MeteringModeText
   formatID = F628FD8C-7BA8-465A-A65B-C5AA79263A9E
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```
