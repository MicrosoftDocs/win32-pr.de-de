# <a name="systemdevicesaepcontainersupportsnetworking"></a>System. Devices. aepcontainer. supportsnetworking

Gibt an, ob der Geräte Zuordnungs Endpunkt-Container supportsnetworking unterstützt.

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10, Version 1703, Windows 10, Version 1607

```
propertyDescription
   name = System.Devices.AepContainer.SupportsNetworking
   shellPKey = PKEY_Devices_AepContainer_SupportsNetworking
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
