# <a name="systemdevicesaepbluetoothcodmajor"></a>System. Devices. aep. Bluetooth. COD. Major

Bluetooth-Klasse des Geräte hauptcodes.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Cod.Major
   shellPKey = PKEY_Devices_Aep_Bluetooth_Cod_Major
   formatID = 5FBD34CD-561A-412E-BA98-478A6B0FEF1D
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
