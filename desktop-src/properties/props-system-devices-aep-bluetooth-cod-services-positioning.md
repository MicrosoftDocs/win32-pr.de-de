# <a name="systemdevicesaepbluetoothcodservicespositioning"></a>System. Devices. aep. Bluetooth. COD. Services. Positionierung

Bluetooth-Klasse der Positionierung von Geräte Diensten.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Cod.Services.Positioning
   shellPKey = PKEY_Devices_Aep_Bluetooth_Cod_Services_Positioning
   formatID = 5FBD34CD-561A-412E-BA98-478A6B0FEF1D
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```
