# <a name="systemsourcepackagefamilyname"></a>System. sourcepackagefamilyname

Der Paket Familienname der APP, von der die Speicher Element Instanz stammt.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.SourcePackageFamilyName
   shellPKey = PKEY_SourcePackageFamilyName
   formatID = FFAE9DB7-1C8D-43FF-818C-84403AA3732D
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
