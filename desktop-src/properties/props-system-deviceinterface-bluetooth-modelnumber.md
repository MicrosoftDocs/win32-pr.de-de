# <a name="systemdeviceinterfacebluetoothmodelnumber"></a>System. Geräte-ID. Bluetooth. modelnumber

```
propertyDescription
   name = System.DeviceInterface.Bluetooth.ModelNumber
   shellPKey = PKEY_DeviceInterface_Bluetooth_ModelNumber
   formatID = 2BD67D8B-8BEB-48D5-87E0-6CDA3428040A
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
```
