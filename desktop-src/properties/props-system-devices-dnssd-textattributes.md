# <a name="systemdevicesdnssdtextattributes"></a>System.Devices.Dnssd.TextAttributes

Textdaten, die der Dienstinstanz zugeordnet sind. Jede Zeichenfolge ist in der Regel ein Schlüssel-Wert-Paar, getrennt durch "=".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.Devices.Dnssd.TextAttributes
   shellPKey = PKEY_Devices_Dnssd_TextAttributes
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Hinweise

PKEY-Werte werden in Propkey.h definiert.
