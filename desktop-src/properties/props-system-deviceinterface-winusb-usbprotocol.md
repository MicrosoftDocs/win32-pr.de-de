# <a name="systemdeviceinterfacewinusbusbprotocol"></a>System.DeviceInterface.WinUsb.UsbProtocol

Protokollwert aus dem ersten USB-Schnittstellendeskriptor des USB-Geräts.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.WinUsb.UsbProtocol
   shellPKey = PKEY_DeviceInterface_WinUsb_UsbProtocol
   formatID = 95E127B5-79CC-4E83-9C9E-8422187B3E0E
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Byte
```

## <a name="remarks"></a>Hinweise

PKEY-Werte werden in Propkey.h definiert.
