# <a name="systemislocationsupported"></a>System. islocationsupported

Gibt an, ob ein Speicherort zu dem Zeitpunkt, zu dem er der Bibliothek hinzugefügt wurde, lokal oder Remote indiziert wurde.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.IsLocationSupported
   shellPKey = PKEY_IsLocationSupported
   formatID = 5D76B67F-9B3D-44BB-B6AE-25DA4F638A67
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
