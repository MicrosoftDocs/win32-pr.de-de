# <a name="systemphotodigitalzoom"></a>System.Photo.DigitalZoom

Das digitale Zoomverhältnis beim Erstellen des Bilds. Lesen Sie aus der Kamera in den EXIF-Informationen (Exchangeable Image File) der Datei. Diese Eigenschaft wird aus [System.Photo.DigitalZoomNumerator](./props-system-photo-digitalzoomnumerator.md) und [System.Photo.DigitalZoomDenominator berechnet.](./props-system-photo-digitalzoomdenominator.md) Wenn der Zähler des aufgezeichneten Werts 0 ist, wurde kein digitaler Zoom verwendet.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.DigitalZoom
   shellPKey = PKEY_Photo_DigitalZoom
   formatID = F85BF840-A925-4BC2-B0C4-8E36B598679E
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Hinweise

PKEY-Werte werden in Propkey.h definiert.
