# <a name="systemphotodigitalzoom"></a>System. Photo. Digitalzoom

Das digitale Zoomverhältnis, wenn das Bild gedreht wurde. Lesen von der Kamera in der Datei mit den Informationen zur austauschbaren Image Datei (EXIF). Diese Eigenschaft wird aus [System. Photo. digitalzoomzähler](./props-system-photo-digitalzoomnumerator.md) und [System. Photo. digitalzoomnenner](./props-system-photo-digitalzoomdenominator.md)berechnet. Wenn der Zähler des aufgezeichneten Werts 0 ist, wurde kein digitaler Zoom verwendet.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.DigitalZoom
   shellPKey = PKEY_Photo_DigitalZoom
   formatID = F85BF840-A925-4BC2-B0C4-8E36B598679E
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
