# <a name="systemdevicesisdefault"></a>System.Devices.IsDefault

Wenn diese Eigenschaft den Wert true hat, ist das Gerät das Standardgerät.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.IsDefault
   shellPKey = PKEY_Devices_IsDefault
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 86
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enum
            name = Default
            value = #TRUE#
            text = Default
            defineToken = ISDEFAULTDEVICE_DEFAULTDEVICE
         enum
            name = NotDefault
            value = #FALSE#
            text = Not Default
            defineToken = ISDEFAULTDEVICE_NOTDEFAULTDEVICE
```
