# <a name="systemsearchcontents"></a>System. search. Content

Der Inhalt des Elements. Diese Eigenschaft ist nur für Abfrage Einschränkungen vorgesehen. Sie kann nicht in einem Abfrageergebnis abgerufen werden. Der Anzeige Name für den Indizierungs Dienst lautet "Content".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.Contents
   shellPKey = PKEY_Search_Contents
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 19
   SearchInfo
      InInvertedIndex = true
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
