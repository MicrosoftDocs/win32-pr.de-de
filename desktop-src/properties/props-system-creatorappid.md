# <a name="systemcreatorappid"></a>System. kreatorappid

Die AppID der Anwendung, die diese Datei erstellt hat.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.CreatorAppId
   shellPKey = PKEY_CreatorAppId
   formatID = C2EA046E-033C-4E91-BD5B-D4942F6BBE49
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = false
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
