Wenn true, ändern Sie die Metadaten dieser Datei nicht. Wird nach Benutzer festgelegt.

```
propertyDescription
   name = System.Media.UserNoAutoInfo
   shellPKey = PKEY_Media_UserNoAutoInfo
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 41
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
```
