# <a name="systemstorageproviderfileidentifier"></a>System. storageproviderfileidentifier

Der Bezeichner des Speicher Anbieters für diese Datei.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1

```
propertyDescription
   name = System.StorageProviderFileIdentifier
   shellPKey = PKEY_StorageProviderFileIdentifier
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
      IsInnate = false
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
