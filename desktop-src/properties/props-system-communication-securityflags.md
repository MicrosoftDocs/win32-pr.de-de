# <a name="systemcommunicationsecurityflags"></a>System. Communication. SecurityFlags

Mit dem Element verknüpfte sicherheitsflags, die angeben, ob das Element verschlüsselt, signiert oder DRM aktiviert ist.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Communication.SecurityFlags
   shellPKey = PKEY_Communication_SecurityFlags
   formatID = 8619A4B6-9F4D-4429-8C0F-B996CA59E335
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
