# <a name="systemphotomaxaperture"></a>System. Photo. maxaperture

Die maximale Öffnung des Bilds, wie von der Kamera aufgezeichnet, entnommen aus den Informationen zur austauschbaren Bilddatei (EXIF). Diese Eigenschaft wird aus [System. Photo. maxaperturenumerator](./props-system-photo-maxaperturenumerator.md) und [System. Photo. maxaperturenenner](./props-system-photo-maxaperturedenominator.md)berechnet.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.MaxAperture
   shellPKey = PKEY_Photo_MaxAperture
   formatID = 08F6D7C2-E3F2-44FC-AF1E-5AA5C81A2D3E
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
