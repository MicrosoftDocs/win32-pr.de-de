# <a name="systemdevicesaepisconnected"></a>System.Devices.Aep.IsConnected

Gibt an, ob das Gerät zurzeit mit dem System verbunden ist oder nicht.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.Devices.Aep.IsConnected
   shellPKey = PKEY_Devices_Aep_IsConnected
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
