# <a name="systemdevicesaudiodevicemicrophonesignaltonoiseratioindb"></a>System. Devices. Audiodevice. Mikrofon. signaltonoiseratioindb

Signal an Rauschverhältnis Informationen in der Datenbank für ein Mikrofon Gerät.

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10, Version 1703, Windows 10, Version 1607

```
propertyDescription
   name = System.Devices.AudioDevice.Microphone.SignalToNoiseRatioInDb
   shellPKey = PKEY_Devices_AudioDevice_Microphone_SignalToNoiseRatioInDb
   formatID = 8943B373-388C-4395-B557-BC6DBAFFAFDB
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
