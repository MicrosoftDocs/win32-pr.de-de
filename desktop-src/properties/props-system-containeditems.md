# <a name="systemcontaineditems"></a>System. containeditems

Eine Liste mit dem Inhaltstyp im Element. Dieser Wert wird als Vektor Array von GUIDs dargestellt, wobei jede GUID einen bestimmten Typ (z. b. URLs oder Anhänge) darstellt.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ContainedItems
   shellPKey = PKEY_ContainedItems
   formatID = 28636AA6-953D-11D2-B5D6-00C04FD918D0
   propID = 29
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue Guid
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
