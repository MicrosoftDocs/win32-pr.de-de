# <a name="systemtitlesortoverride"></a>System. titlesorbuverride

Dieser optionale Zeichen folgen Wert ermöglicht das Überschreiben der standardmäßigen Sortierreihenfolge von System. Title. Dies ist sehr wichtig für die ordnungsgemäße Sortierung von Musikdateien in Japanisch, die nach dem Feld nicht vollständig (die vom Benutzer erwartete Reihenfolge) ohne dieses Feld sortiert werden kann. Sie kann auch zum Anpassen von Sortierungen in nicht ostasiatischen Szenarios verwendet werden, z. b. wenn der Benutzer das Entfernen von Artikeln zu Sortier Zwecken ermöglicht.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.TitleSortOverride
   shellPKey = PKEY_TitleSortOverride
   formatID = F0F7984D-222E-4AD2-82AB-1DD8EA40E57E
   propID = 300
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
