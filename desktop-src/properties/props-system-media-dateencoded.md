# <a name="systemmediadateencoded"></a>System. Media. datecodiert

Stellt das Datum und die Uhrzeit der Codierung der Datei dar. Der DateTime-Wert ist UTC (im doc, nicht im Dateisystem).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Media.DateEncoded
   shellPKey = PKEY_Media_DateEncoded
   formatID = 2E4B640D-5019-46D8-8881-55414CC5CAA0
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Media.DateEncoded
   shellPKey = PKEY_Media_DateEncoded
   formatID = 2E4B640D-5019-46D8-8881-55414CC5CAA0
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
