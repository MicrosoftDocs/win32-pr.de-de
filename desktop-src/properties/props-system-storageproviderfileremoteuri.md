# <a name="systemstorageproviderfileremoteuri"></a>System. storageproviderfileremoteuri

Der Remote-URI des Speicher Anbieters für diese Datei.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1

```
propertyDescription
   name = System.StorageProviderFileRemoteUri
   shellPKey = PKEY_StorageProviderFileRemoteUri
   formatID = FCEFF153-E839-4CF3-A9E7-EA22832094B8
   propID = 112
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
