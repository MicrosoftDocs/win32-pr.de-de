# <a name="systemdevicesaepcontainerispaired"></a>System.Devices.AepContainer.IsPaired

Gibt an, ob einer der untergeordneten Geräte Zuordnungs Endpunkte mit dem System gekoppelt ist.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.Devices.AepContainer.IsPaired
   shellPKey = PKEY_Devices_AepContainer_IsPaired
   formatID = 0BBA1EDE-7566-4F47-90EC-25FC567CED2A
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
