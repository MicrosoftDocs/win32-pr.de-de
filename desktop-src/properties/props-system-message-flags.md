# <a name="systemmessageflags"></a>System.Message.Flags

Flags, die E-Mail-Nachrichten zugeordnet sind (z. B. zur Identifizierung, dass ein Lesebeleg aussteht). Die hier von Outlook gespeicherten Werte werden für PR \_ MESSAGE \_ FLAGS definiert.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Message.Flags
   shellPKey = PKEY_Message_Flags
   formatID = A82D9EE7-CA67-4312-965E-226BCEA85023
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
      IsInnate = false
```

## <a name="remarks"></a>Hinweise

PKEY-Werte werden in Propkey.h definiert.
