# <a name="systempropgroupmusic"></a>System. propgroup. Music

Das Eigenschaften Gruppen Trennzeichen, das in Eigenschafts Listen zum Trennen von Musikdateien von anderen Typen verwendet wird. Verwenden Sie diese Eigenschaft nicht zum erhalten oder Festlegen von Werten. Sie ist nur als Marker vorgesehen.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropGroup.Music
   shellPKey = PKEY_PropGroup_Music
   formatID = 68DD6094-7216-40F1-A029-43FE7127043F
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Null
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
