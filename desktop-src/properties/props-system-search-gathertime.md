# <a name="systemsearchgathertime"></a>System. search. gathertime

Der DateTime-Wert, der die Zeit angibt, zu der der Windows Search-gathererprozess die Eigenschaften dieses Dokuments zuletzt an die Windows Search-Gatherer-Plug-ins gesendet hat

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Search.GatherTime
   shellPKey = PKEY_Search_GatherTime
   formatID = 0B63E350-9CCC-11D0-BCDB-00805FCCCE04
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Search.GatherTime
   shellPKey = PKEY_Search_GatherTime
   formatID = 0B63E350-9CCC-11D0-BCDB-00805FCCCE04
   propID = 8
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
