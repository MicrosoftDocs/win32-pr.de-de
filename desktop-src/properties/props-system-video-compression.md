# <a name="systemvideocompression"></a>System. Video. Compression

Gibt das Video Komprimierungs Format an.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Video.Compression
   shellPKey = PKEY_Video_Compression
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 10
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Der Wert dieser Eigenschaft ist eine Zeichen folgen Darstellung einer GUID in kanonischer Form: "{xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx}". Der GUID gibt das Videoformat an. Mögliche Werte finden Sie [unter Video Untertyp-GUIDs](../medfound/video-subtype-guids.md).

Pkey-Werte werden in "propkey. h" definiert.
