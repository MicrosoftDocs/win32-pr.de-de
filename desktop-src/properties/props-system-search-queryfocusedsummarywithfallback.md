# <a name="systemsearchqueryfocusedsummarywithfallback"></a>System. search. queryfocusedsummarywithfallback

Die Zusammenfassung des Dokuments in der Abfrage. Wenn keine verfügbar ist, wird die automatische Zusammenfassung zurückgegeben. Diese Eigenschaft wird vom Suchsystem automatisch erstellt. Verwenden Sie diese Eigenschaft nur, wenn Sie die Werte in der Anwendung verwenden, nicht für die Bereitstellung von Werten für einen Eigenschaften Handler.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Search.QueryFocusedSummaryWithFallback
   shellPKey = PKEY_Search_QueryFocusedSummaryWithFallback
   formatID = 560C36C0-503A-11CF-BAA1-00004C752A9A
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
