# <a name="systemdevicesbatterypluschargingtext"></a>System.Devices.Batterypluschargingtext

Die verbleibende Akku Lebensdauer des Geräts und der Ladezustand des Geräts als Zeichenfolge.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Devices.BatteryPlusChargingText
   shellPKey = PKEY_Devices_BatteryPlusChargingText
   formatID = 49CD1F76-5626-4B17-A4E8-18B4AA1A2213
   propID = 23
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
