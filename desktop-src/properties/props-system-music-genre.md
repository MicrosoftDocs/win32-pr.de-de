# <a name="systemmusicgenre"></a>System. Music. Genre

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Music.Genre
   shellPKey = PKEY_Music_Genre
   formatID = 56A3372E-CE9C-11D2-9F0E-006097C686F6
   propID = 11
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.

Beachten Sie, dass einige Dateiformate, wie z. b. eine MP3-Datei, nur ein einzelnes Genre unterstützen. Andere, wie z. b. WMA-Dateien, unterstützen mehrere Genres. Wenn Sie mehrere Genres für ein Single-Genre-Dateiformat angeben, wird nur das erste Genre in der Liste erkannt.
