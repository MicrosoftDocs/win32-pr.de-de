# <a name="systemappusermodelisdestlistseparator"></a>System. appusermodel. isdestlistseparator

Fügt ein Trennzeichen in den **Aufgaben** Abschnitt einer Sprung Liste ein. Legen Sie diese Eigenschaft für ein [**IShellLink**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ishelllinka) -Objekt fest, und übergeben Sie es an [**ICustomDestinationList:: AddUserTasks**](/windows/win32/api/shobjidl_core/nf-shobjidl_core-icustomdestinationlist-addusertasks).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.AppUserModel.IsDestListSeparator
   shellPKey = PKEY_AppUserModel_IsDestListSeparator
   formatID = 9F4C2855-9F79-4B39-A8D0-E1D42DE1D5F3
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = false
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
