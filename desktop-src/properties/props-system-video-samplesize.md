# <a name="systemvideosamplesize"></a>System. Video. samplesize

Gibt die Stichprobengröße für den Videostream in Bits an. "Samplesize".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Video.SampleSize
   shellPKey = PKEY_Video_SampleSize
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Video.SampleSize
   shellPKey = PKEY_Video_SampleSize
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 9
   SearchInfo
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
