# <a name="systemcalendarshowtimeastext"></a>System. Calendar. showtimeastext

Die benutzerfreundliche Form von [System. Calendar. showtimeas](./props-system-calendar-showtimeas.md). Dieser Wert sollte nicht Programm gesteuert analysiert werden.

```
propertyDescription
   name = System.Calendar.ShowTimeAsText
   shellPKey = PKEY_Calendar_ShowTimeAsText
   formatID = 53DA57CF-62C0-45C4-81DE-7610BCEFD7F5
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```
