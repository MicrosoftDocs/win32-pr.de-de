# <a name="systemproplistnonpersonal"></a>System. proplist. nonpersonal

Die Liste der Eigenschaften, die als "nicht persönlich" angesehen werden. Das System lässt diese Eigenschaften unverändert, wenn Sie dazu umgeleitet werden, alle nicht persönlichen Eigenschaften aus einer bestimmten Datei zu entfernen. Registrieren Sie sich unter dem regvalue-Wert von "nonprivate".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.NonPersonal
   shellPKey = PKEY_PropList_NonPersonal
   formatID = 49D1091F-082E-493F-B23F-D2308AA9668C
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
