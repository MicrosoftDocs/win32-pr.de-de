# <a name="systemphotosaturationtext"></a>System. Photo. sätationtext

Die benutzerfreundliche Form von [System. Photo. Sättigung](./props-system-photo-saturation.md). Es ist nicht vorgesehen, Programm gesteuert analysiert zu werden.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.SaturationText
   shellPKey = PKEY_Photo_SaturationText
   formatID = 61478C08-B600-4A84-BBE4-E99C45F0A072
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
