# <a name="systemimagedimensions"></a>System.Image.Dimensions

Die Bilddimensionen im Zeichenfolgenformat als `horizontal pixels x vertical pixels` . Beispiel: `3080x2100`.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Image.Dimensions
   shellPKey = PKEY_Image_Dimensions
   formatID = 6444048F-4C8B-11D1-8B70-080036B11A03
   propID = 13
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Hinweise

PKEY-Werte werden in Propkey.h definiert.
