# <a name="systemproplistfileoperationprompt"></a>System. proplist. fileoperationprompt

Die Liste der Eigenschaften, die im Bestätigungs Dialogfeld des Datei Vorgangs angezeigt werden sollen. Eigenschaften mit leeren Werten werden nicht angezeigt. Wenn diese Liste nicht angegeben wird, wird stattdessen die infotip-Eigenschaften Liste verwendet. Registrieren Sie sich unter dem regvalue-Wert von "fileoperationprompt".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.FileOperationPrompt
   shellPKey = PKEY_PropList_FileOperationPrompt
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
