# <a name="systemdeviceinterfacewinusbusbsubclass"></a>System. devicabterface. winusb. Start-Unterklasse

Der Wert der Unterklasse aus dem ersten USB-Schnittstellen Deskriptor des USB-Geräts.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.WinUsb.UsbSubClass
   shellPKey = PKEY_DeviceInterface_WinUsb_UsbSubClass
   formatID = 95E127B5-79CC-4E83-9C9E-8422187B3E0E
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Byte
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
