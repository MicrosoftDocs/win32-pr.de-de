# <a name="systemphotoexposureindex"></a>System. Photo. exposureindex

Gibt den auf der Kamera oder dem Eingabegerät zum Zeitpunkt der Erstellung des Fotos ausgewählten Expositions Index an. Berechnet aus dem pkey \_ \_ -Foto exposu. xnumerator und pkey \_ Photo \_ exposuentrxnenner.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.ExposureIndex
   shellPKey = PKEY_Photo_ExposureIndex
   formatID = 967B5AF8-995A-46ED-9E11-35B3C5B9782D
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
