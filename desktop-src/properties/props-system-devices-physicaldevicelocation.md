# <a name="systemdevicesphysicaldevicelocation"></a>System.Devices.PhysicalDeviceLocation

ACPI \_ PLD-Daten für das Gerät.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.PhysicalDeviceLocation
   shellPKey = PKEY_Devices_PhysicalDeviceLocation
   formatID = 540B947E-8B40-45BC-A8A2-6A0B894CBDA2
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
      IsInnate = true
```

PKEY-Werte werden in Propkey.h definiert.
