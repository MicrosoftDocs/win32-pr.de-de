# <a name="systemdeviceswinphone8cameraflags"></a>System. Devices. WinPhone8CameraFlags

Flags für ein WP8 Kamera-Gerät.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.Devices.WinPhone8CameraFlags
   shellPKey = PKEY_Devices_WinPhone8CameraFlags
   formatID = B7B4D61C-5A64-4187-A52E-B1539F359099
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
