# <a name="systemdevicesnotificationsnewmessage"></a>System. Devices. Benachrichtigungen. newMessage

Benachrichtigung über neue Geräte Nachricht.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.Notifications.NewMessage
   shellPKey = PKEY_Devices_Notifications_NewMessage
   formatID = 2BE9260A-2012-4742-A555-F41B638B7DCB
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Byte
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enumRange
            name = NewMessageNotification
            minValue = 0
            setValue = 0
            text = You have a new text message.
```
