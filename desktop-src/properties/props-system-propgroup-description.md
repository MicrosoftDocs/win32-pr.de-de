# <a name="systempropgroupdescription"></a>System. propgroup. Description

Das Eigenschaften Gruppen Trennzeichen, das in Eigenschafts Listen zum Trennen von Beschreibungen von anderen Typen verwendet wird. Verwenden Sie diese Eigenschaft nicht zum erhalten oder Festlegen von Werten. Diese Eigenschaft ist nur als Marker vorgesehen.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropGroup.Description
   shellPKey = PKEY_PropGroup_Description
   formatID = 8969B275-9475-4E00-A887-FF93B8B41E44
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Null
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
