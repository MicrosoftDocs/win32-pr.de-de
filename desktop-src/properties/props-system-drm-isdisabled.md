# <a name="systemdrmisdisabled"></a>System.DRM.IsDisabled

Gibt an, ob die Mediendatei vom DRM deaktiviert wurde.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.DRM.IsDisabled
   shellPKey = PKEY_DRM_IsDisabled
   formatID = AEAC19E4-89AE-4508-B9B7-BB867ABEE2ED
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
```

## <a name="remarks"></a>Hinweise

PKEY-Werte werden in Propkey.h definiert.
