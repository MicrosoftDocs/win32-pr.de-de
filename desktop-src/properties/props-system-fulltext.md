# <a name="systemfulltext"></a>System. Fulltext

Diese Eigenschaft wird verwendet, um Suchbegriffe anzugeben, die so weit wie möglich über alle gültigen Eigenschaften der zu durchsuchenden Datenquelle (n) angewendet werden sollen. Er sollte nicht aus einer Datenquelle ausgegeben werden.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.FullText
   shellPKey = PKEY_FullText
   formatID = 1E3EE840-BC2B-476C-8237-2ACD1A839B22
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
