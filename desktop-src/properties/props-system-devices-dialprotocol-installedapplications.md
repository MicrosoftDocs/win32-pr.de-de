# <a name="systemdevicesdialprotocolinstalledapplications"></a>System.Devices.DialProtocol.InstalledApplications

Liste der Anwendungen, die das DIAL-Protokoll auf dem Device Association-Endpunkt unterstützen

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.Devices.DialProtocol.InstalledApplications
   shellPKey = PKEY_Devices_DialProtocol_InstalledApplications
   formatID = 6845CC72-1B71-48C3-AF86-B09171A19B14
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Hinweise

PKEY-Werte werden in Propkey.h definiert.
