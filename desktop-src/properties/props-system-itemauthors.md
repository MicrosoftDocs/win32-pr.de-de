# <a name="systemitemauthors"></a>System.ItemAuthors

Generische Liste von Autoren, die einem Element zugeordnet sind. Der Name des Interpreten für eine Musikspur ist beispielsweise der Elementautor.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ItemAuthors
   shellPKey = PKEY_ItemAuthors
   formatID = D0A04F0A-462A-48A4-BB2F-3706E88DBD7D
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Hinweise

PKEY-Werte werden in Propkey.h definiert.
