# <a name="systemsensitivitytext"></a>System. sensitivitytext

Die benutzerfreundliche Form von [System. Sensitivität](./props-system-sensitivity.md). Dieser Wert sollte nicht Programm gesteuert analysiert werden.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.SensitivityText
   shellPKey = PKEY_SensitivityText
   formatID = D0C7F054-3F72-4725-8527-129A577CB269
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
