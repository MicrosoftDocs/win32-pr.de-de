# <a name="systemdateaccessed"></a>System. datezugriff

Gibt den Zeitpunkt an, zu dem das Element zuletzt aufgerufen wurde. Der Anzeige Name für den Indizierungs Dienst lautet "Zugriff".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.DateAccessed
   shellPKey = PKEY_DateAccessed
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 16
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.DateAccessed
   shellPKey = PKEY_DateAccessed
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 16
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
