# <a name="systemdevicesdnssdttl"></a>System. Devices. dnssd. TTL

SRV-Daten Satz für die Gültigkeitsdauer. Wird normalerweise nicht verwendet.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.Devices.Dnssd.Ttl
   shellPKey = PKEY_Devices_Dnssd_Ttl
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
