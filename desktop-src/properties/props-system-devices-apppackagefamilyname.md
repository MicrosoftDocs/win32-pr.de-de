# <a name="systemdevicesapppackagefamilyname"></a>System. Devices. apppackagefamilyname

Der Paket Familienname, der als App für dieses Gerät registriert ist.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.AppPackageFamilyName
   shellPKey = PKEY_Devices_AppPackageFamilyName
   formatID = 51236583-0C4A-4FE8-B81F-166AEC13F510
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
