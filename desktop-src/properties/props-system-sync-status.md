# <a name="systemsyncstatus"></a>System. Sync. Status

Der Status der System Synchronisierung.

```
propertyDescription
   name = System.Sync.Status
   shellPKey = PKEY_Sync_Status
   formatID = 7BD5533E-AF15-44DB-B8C8-BD6624E1D032
```
