# <a name="systemdevicesdnssdfullname"></a>System. Devices. dnssd. FullName

Vervollständigen Sie den Namen der DNS-SD-Dienst Instanz, einschließlich Instanz, Dienst und Domäne.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, Version 1703, Windows 10, Version 1607, Windows 10, Version 1511, Windows 10, Version 1507

```
propertyDescription
   name = System.Devices.Dnssd.FullName
   shellPKey = PKEY_Devices_Dnssd_FullName
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Bemerkungen

Pkey-Werte werden in "propkey. h" definiert.
