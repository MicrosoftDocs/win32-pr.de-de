# <a name="installing-the-peer-infrastructure"></a>Installieren der Peer Infrastruktur

Die Windows-Peer-Netzwerkinfrastruktur ist eine Entwicklerplattform zum Erstellen von Peer Anwendungen für Computer, auf denen Windows XP mit Service Pack 1 (SP1) mit dem erweiterten Netzwerk Paket, Windows XP mit Service Pack 2 (SP2), Windows Vista und Windows Server 2008 ausgeführt wird.

Computer, auf denen Windows Vista oder Windows Server 2008 ausgeführt wird, verfügen bereits über eine Windows-Peer Netzwerkinfrastruktur. Für Computer, auf denen Windows XP mit SP2 ausgeführt wird, sind die folgenden Schritte erforderlich, um Windows-Peer Netzwerke zu installieren:

-   Klicken Sie auf **Start**, klicken Sie auf **Systemsteuerung**, und klicken Sie dann auf **Programme hinzufügen oder entfernen**.
-   Klicken Sie auf **Windows-Komponenten hinzufügen/entfernen**.
-   Klicken Sie unter **Komponenten** auf **Netzwerkdienste** (aktivieren Sie das Kontrollkästchen jedoch nicht), und klicken Sie dann auf **Details**.
-   Aktivieren Sie das Kontrollkästchen **Peer** , und klicken Sie dann auf **OK**.
-   Klicken Sie auf **weiter**, und befolgen Sie dann die Anweisungen im Assistenten.

Für Computer, auf denen Windows XP mit SP1 ausgeführt wird, ist die Installation des [erweiterten Netzwerk Pakets](https://www.microsoft.com/downloads/details.aspx?FamilyID=e88cc382-8ce6-4739-97c0-1a52a6f005e4)(kostenloser Download) erforderlich, um Peer Netzwerke zu aktivieren.
