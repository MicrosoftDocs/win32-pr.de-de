# <a name="why-use-people-near-me"></a>Gründe für die Verwendung von Personen in meiner Umgebung

Im folgenden finden Sie einige Beispiele für Szenarien, die [Personen in meiner](about-people-near-me.md) Umgebung (PNM) ermöglichen:

-   Initiieren Sie während einer Besprechung einen Chat mit einer Person in einem Konferenzraum. Verwenden Sie eine Chatanwendung, um andere Benutzer im Raum dynamisch zu ermitteln, einen bestimmten Benutzer auszuwählen und eine Chatsitzung zu starten, um zusammenzuarbeiten oder Informationen auszutauschen.
-   Die PNM-API ermöglicht die Freigabe von Dokumenten für alle Personen in Ihrem Konferenzraum. Obwohl USB-Flash Laufwerke (Universal Serial Bus, UFDs) das Übertragen von Dokumenten zwischen Computern in einer geschlossenen Einstellung (z. b. eine Besprechung) sehr einfach machen, nimmt es Zeit und Mühe, das UFD an jeden Computer anzuschließen und das Dokument in einen lokalen Ordner zu kopieren. Bei Anwendungen, die über PNM verfügen, können Sie die Benutzer in dem Raum auswählen, mit dem Sie ein Dokument freigeben oder senden möchten.
-   Streamen Sie die Informationen auf dem Desktop eines Computers für die Ressourcen Freigabe an den Computer eines Kollegen.
-   Starten Sie ein Spiel mit einer Person während einer Schicht an einem Flughafen Terminal, das drahtlose Konnektivität unterstützt. Die PNM-API ermöglicht es einer Anwendung, Ihr Interesse an einem Spiel zu ankündigen, andere Personen am Flughafen zu ermitteln, die auch am Spiel spielen möchten, und eine Spiele Sitzung zu starten.
-   Ankündigen Sie Interessen an einer Branchenkonferenz, und entdecken Sie die anderen Personen mit drahtloser Konnektivität. Nach der Ermittlung können Benutzer mit dem Austauschen von Ideen chatten oder nicht ordnungsgemäße Besprechungen einrichten.
