# <a name="why-use-people-near-me"></a>Gründe für die Verwendung von Personen in meiner Umgebung

Einige Beispiele für Szenarien, die [Personen in meiner Umgebung](about-people-near-me.md) (PNM) ermöglicht, sind die folgenden:

-   Initiieren Des Chats mit einer Person in einem Konferenzraum während einer Besprechung. Verwenden Sie eine Chatanwendung, um andere Benutzer im Raum dynamisch zu ermitteln, einen bestimmten Benutzer auszuwählen und eine Chatsitzung zum Zusammenarbeiten oder Austauschen von Informationen zu starten.
-   Die PNM-API ermöglicht die Gemeinsame Nutzung von Dokumenten für alle Benutzer in Ihrem Konferenzraum. Obwohl USB-Speichersticks (Universal Serial Bus) die Übertragung von Dokumenten zwischen Computern in einer geschlossenen Einstellung wie einer Besprechung sehr einfach machen, dauert es dennoch Zeit und Aufwand, die UFD an jeden Computer anzufügen und das Dokument in einen lokalen Ordner zu kopieren. Mit PNM-fähigen Anwendungen können Sie die Benutzer in dem Raum auswählen, für den Sie ein Dokument freigeben oder senden möchten.
-   Streamen Sie die Informationen auf dem Desktop eines Computers zur Ressourcenfreigabe an den Computer eines Kollegen.
-   Starten Sie ein Spiel mit einer Person während eines Layouts an einem Flughafenterminal, das drahtlose Konnektivität unterstützt. Mit der PNM-API kann eine Anwendung Ihr Interesse am Spielen eines Spiels ankündigen, andere am Flughafen entdecken, die auch an der Wiedergabe des Spiels interessiert sind, und eine Spielesitzung starten.
-   Bewerben Sie Ihre Interessen auf einer Branchenkonferenz, und entdecken Sie die Interessen anderer Mithilfe von Drahtloskonnektivität. Nach der Ermittlung können Benutzer chatten oder impromptu-Besprechungen einrichten, um Ideen auszutauschen.
