# <a name="graphing-api-callbacks"></a>Graphing-API-Rückrufe

Die Graphing-API verwendet die folgenden Rückrufe:



| Rückruf                                                          | Beschreibung                                                                                                                                                                                                                  |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [*PFNPEER \_ \_ FREE-SICHERHEITSDATEN \_*](/windows/desktop/api/P2P/nc-p2p-pfnpeer_free_security_data) | Gibt die Funktion an, die die Peerdiagramminfrastruktur aufruft, um von [*PFNPEER \_ SECURE \_ RECORD- und PFNPEER*](/windows/desktop/api/P2P/nc-p2p-pfnpeer_secure_record) [*VALIDATE \_ \_ RECORD-Rückrufen*](/windows/desktop/api/P2P/nc-p2p-pfnpeer_validate_record) zurückgegebene Daten freizubehalten. |
| [*PFNPEER \_ SECURE \_ RECORD*](/windows/desktop/api/P2P/nc-p2p-pfnpeer_secure_record)            | Gibt die Funktion an, die die Peerdiagramminfrastruktur aufruft, um Datensätze zu sichern.                                                                                                                                        |
| [*PFNPEER \_ VALIDATE \_ RECORD*](/windows/desktop/api/P2P/nc-p2p-pfnpeer_validate_record)        | Gibt die Funktion an, die die Peerdiagramminfrastruktur aufruft, um Datensätze zu überprüfen.                                                                                                                                      |
