# <a name="pnrp-structures"></a>PNRP-Strukturen

Die PNRP-Namespaceanbieter-API verwendet die folgenden Strukturen:



| Struktur                                                             | BESCHREIBUNG                                                                                                                                                         |
|-----------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PEER \_ PNRP ENDPOINT INFO (PEER-PNRP-ENDPUNKTINFORMATIONEN) \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_pnrp_endpoint_info)         | Enthält die IP-Adressen und Daten, die einem Peerendpunkt zugeordnet sind.                                                                                                 |
| [**PEER \_ PNRP \_ CLOUD \_ INFO**](/windows/desktop/api/P2P/ns-p2p-peer_pnrp_cloud_info)               | Enthält Informationen zu einer PNRP-Cloud (Peer Name Resolution Protocol).                                                                                            |
| [**PEER \_ PNRP REGISTRATION INFO (PEER PNRP-REGISTRIERUNGSINFORMATIONEN) \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_pnrp_registration_info) | Enthält die Informationen, die von einer Peeridentität bei der Registrierung bei einer PNRP-Cloud bereitgestellt werden.                                                                           |
| [PNRP und BLOB](pnrp-and-blob.md)                                    | Übergibt Daten bei Aufrufen mehrerer Funktionen an die [**WSAQUERYSET-Struktur.**](winsock-nsp-reference-links.md)                                                  |
| [PNRP und WSAQUERYSET](pnrp-and-wsaqueryset.md)                      | Erleichtert das Auflösen von Namen und die Enumeration von Namen und Clouds.                                                                                         |
| [**\_ \_ PNRP-CLOUD-ID**](/windows/desktop/api/Pnrpdef/ns-pnrpdef-pnrp_cloud_id)                              | Enthält die Werte, die eine Netzwerkcloud definieren.                                                                                                                    |
| [**PNRPCLOUDINFO**](/windows/desktop/api/Pnrpns/ns-pnrpns-pnrpcloudinfo)                                | Zeigt auf den **lpBlob-Member** der [**WSAQUERYSET-Struktur.**](winsock-nsp-reference-links.md)                                                            |
| [**PNRPINFO \_ V1**](/windows/desktop/api/Pnrpns/ns-pnrpns-pnrpinfo_v1)                                      | Zeigt auf den **lpBlob-Member** der [**WSAQUERYSET-Struktur**](winsock-nsp-reference-links.md)                                                             |
| [**PNRPINFO \_ V2**](/previous-versions/windows/desktop/legacy/aa371671(v=vs.85))                                   | Zeigt auf den **lpBlob-Member** der [**WSAQUERYSET-Struktur**](winsock-nsp-reference-links.md) und bietet Unterstützung für anwendungsspezifische nicht transparente Daten. |
