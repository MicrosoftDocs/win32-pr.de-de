# <a name="collaboration-api-functions"></a>Funktionen der Zusammenarbeits-API

Die Peerzusammenarbeitsinfrastruktur unterstützt die folgenden Funktionen.



| Funktion                                                                                       | Beschreibung                                                                                                                                                                                     |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PeerCollabAddContact**](/windows/desktop/api/P2P/nf-p2p-peercollabaddcontact)                                           | Fügt der Kontaktliste eines Peers einen Kontakt hinzu.                                                                                                                                                   |
| [**PeerCollabAsyncInviteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabasyncinvitecontact)                           | Sendet eine Einladung an einen vertrauenswürdigen Peerkontakt, um der Peerzusammenarbeitsaktivität des Absenders über eine gesicherte Verbindung beizusein.                                                                       |
| [**PeerCollabAsyncInviteEndpoint**](/windows/desktop/api/P2P/nf-p2p-peercollabasyncinviteendpoint)                         | Sendet eine Einladung an einen angegebenen Peerendpunkt, um der Peerzusammenarbeitsaktivität des Absenders beizusein. Die Verfügbarkeit der Antwort auf die Einladung wird durch ein asynchrones Ereignis aktualisiert. |
| [**PeerCollabCancelInvitation**](/windows/desktop/api/P2P/nf-p2p-peercollabcancelinvitation)                               | Bricht eine Einladung ab, die zuvor vom Aufrufer an einen Kontakt gesendet wurde.                                                                                                                               |
| [**PeerCollabCloseHandle**](/windows/desktop/api/P2P/nf-p2p-peercollabclosehandle)                                         | Schließt das Handle für eine Einladung zur Peerzusammenarbeitsaktivität.                                                                                                                                  |
| [**PeerCollabDeleteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabdeletecontact)                                     | Löscht einen Kontakt aus dem aktuellen Peer.                                                                                                                                                        |
| [**PeerCollabDeleteEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabdeleteendpointdata)                           | Löscht die Peerendpunktdaten auf dem aufrufenden Peerknoten, der den angegebenen Endpunktdaten entspricht.                                                                                                |
| [**PeerCollabDeleteObject**](/windows/desktop/api/P2P/nf-p2p-peercollabdeleteobject)                                       | Löscht ein Peerobjekt aus dem aufrufenden Endpunkt.                                                                                                                                                |
| [**PeerCollabEnumApplications**](/windows/desktop/api/P2P/nf-p2p-peercollabenumapplications)                               | Gibt das Handle an eine Enumeration zurück, die die Funktionen enthält, die für die Endpunkte eines bestimmten Peers registriert sind.                                                                                |
| [**PeerCollabEnumApplicationRegistrationInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabenumapplicationregistrationinfo) | Ruft das Enumerationshand handle ab, das zum Abrufen von Peeranwendungsinformationen verwendet wird.                                                                                                                   |
| [**PeerCollabEnumContacts**](/windows/desktop/api/P2P/nf-p2p-peercollabenumcontacts)                                       | Gibt ein Handle für eine aufzählte Gruppe zurück, die alle Netzwerkkontakte für die Peerzusammenarbeit enthält, die derzeit auf dem aufrufenden Peer verfügbar sind.                                                     |
| [**PeerCollabEnumEndpoints**](/windows/desktop/api/P2P/nf-p2p-peercollabenumendpoints)                                     | Gibt das Handle an eine Enumeration zurück, die die Endpunkte enthält, die einem bestimmten Peerkontakt zugeordnet sind.                                                                                       |
| [**PeerCollabEnumObjects**](/windows/desktop/api/P2P/nf-p2p-peercollabenumobjects)                                         | Gibt das Handle an eine Enumeration zurück, die die Peerobjekte enthält, die dem Endpunkt eines bestimmten Peers zugeordnet sind.                                                                                 |
| [**PeerCollabEnumPeopleNearMe**](/windows/desktop/api/P2P/nf-p2p-peercollabenumpeoplenearme)                               | Gibt ein Handle für eine aufzählte Gruppe zurück, die alle Endpunkte des Peerzusammenarbeitsnetzwerks "Personen in meiner Nähe" enthält, die derzeit im Subnetz des aufrufenden Peers verfügbar sind.                     |
| [**PeerCollabExportContact**](/windows/desktop/api/P2P/nf-p2p-peercollabexportcontact)                                     | Exportiert die einem Peernamen zugeordneten Kontaktdaten in einen XML-Datenzeichenfolgenpuffer des Kontakts.                                                                                                       |
| [**PeerCollabGetAppLaunchInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetapplaunchinfo)                               | Erhält die Startinformationen der Peeranwendung, einschließlich des Kontaktnamens, des Peerendpunkts und der Einladungsanforderung.                                                                     |
| [**PeerCollabGetApplicationRegistrationInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetapplicationregistrationinfo)   | Erhält spezifische Anwendungsregistrierungsinformationen.                                                                                                                                          |
| [**PeerCollabGetContact**](/windows/desktop/api/P2P/nf-p2p-peercollabgetcontact)                                           | Erhält die Informationen für einen bestimmten Peerkontakt unter Gewinnung des Peernamens des Kontakts.                                                                                                         |
| [**PeerCollabGetEndpointName**](/windows/desktop/api/P2P/nf-p2p-peercollabgetendpointname)                                 | Ruft den Namen des aktuellen Endpunkts des aufrufenden Peers ab, der zuvor durch einen Aufruf von PeerCollabSetEndpointName festgelegt wurde.                                                                           |
| [**PeerCollabGetEventData**](/windows/desktop/api/P2P/nf-p2p-peercollabgeteventdata)                                       | Erhält die Daten, die einem Peerzusammenarbeitsereignis zugeordnet sind, das auf dem Peer ausgelöst wird.                                                                                                                 |
| [**PeerCollabGetInvitationResponse**](/windows/desktop/api/P2P/nf-p2p-peercollabgetinvitationresponse)                     | Erhält die Antwort von einem Peer, der zuvor zur Teilnahme an einer Peerzusammenarbeitsaktivität eingeladen wurde.                                                                                                        |
| [**PeerCollabGetPresenceInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetpresenceinfo)                                 | Ruft die Anwesenheitsinformationen für den Endpunkt ab, der einem bestimmten Kontakt zugeordnet ist.                                                                                                         |
| [**PeerCollabGetSigninOptions**](/windows/desktop/api/P2P/nf-p2p-peercollabgetsigninoptions)                               | Hier werden die aktuellen Optionen für das Netzwerk für die Peerzusammenarbeitsnetzwerk-Präsenz des peer angemeldeten Peers erhalten.                                                                                                               |
| [**PeerCollabInviteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabinvitecontact)                                     | Sendet eine Einladung zum Beitreten zu einer Peerzusammenarbeitsaktivität an einen vertrauenswürdigen Kontakt. Dieser Aufruf ist synchron und erhält bei Erfolg eine Antwort vom Kontakt.                               |
| [**PeerCollabInviteEndpoint**](/windows/desktop/api/P2P/nf-p2p-peercollabinviteendpoint)                                   | Sendet eine Einladung an einen angegebenen Peerendpunkt, um der Peerzusammenarbeitsaktivität des Absenders beizusein. Dieser Aufruf ist synchron und erhält bei Erfolg eine Antwort vom Peerendpunkt.      |
| [**PeerCollabParseContact**](/windows/desktop/api/P2P/nf-p2p-peercollabparsecontact)                                       | Analysiert einen Unicode-Zeichenfolgenpuffer, der XML-Kontaktdaten enthält, in eine [**PEER \_ CONTACT-Datenstruktur.**](/windows/desktop/api/P2P/ns-p2p-peer_contact)                                                                         |
| [**PeerCollabQueryContactData**](/windows/desktop/api/P2P/nf-p2p-peercollabquerycontactdata)                               | Ruft die Kontaktinformationen für den angegebenen Peerendpunkt ab.                                                                                                                               |
| [**PeerCollabRefreshEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabrefreshendpointdata)                         | Aktualisiert den aufrufenden Peerknoten mit neuen Endpunktdaten.                                                                                                                                           |
| [**PeerCollabRegisterApplication**](/windows/desktop/api/P2P/nf-p2p-peercollabregisterapplication)                         | Registriert eine Anwendung beim lokalen Computer, damit sie in einer Peerzusammenarbeitsaktivität gestartet werden kann.                                                                                   |
| [**PeerCollabRegisterEvent**](/windows/desktop/api/P2P/nf-p2p-peercollabregisterevent)                                     | Registriert eine Anwendung bei der Peerzusammenarbeitsinfrastruktur, um Rückrufe für bestimmte Peerzusammenarbeitsereignisse zu empfangen.                                                                |
| [**PeerCollabSetEndpointName**](/windows/desktop/api/P2P/nf-p2p-peercollabsetendpointname)                                 | Legt den Namen des aktuellen Endpunkts fest, der von der Peeranwendung verwendet wird.                                                                                                                             |
| [**PeerCollabSetObject**](/windows/desktop/api/P2P/nf-p2p-peercollabsetobject)                                             | Erstellt oder aktualisiert ein Peerdatenobjekt, das in einem Peerzusammenarbeitsnetzwerk verwendet wird.                                                                                                                     |
| [**PeerCollabSetPresenceInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabsetpresenceinfo)                                 | Aktualisiert die Anwesenheitsinformationen des Aufrufers für alle Kontakte, die ihn beobachten.                                                                                                                          |
| [**PeerCollabSignIn**](/windows/desktop/api/P2P/nf-p2p-peercollabsignin)                                                   | Signiert den Peer bei einem gehosteten Internet (serverloses Vorhandensein) oder Subnetz ("Personen in meiner Umgebung").                                                          |
| [**PeerCollabSignOut**](/windows/desktop/api/P2P/nf-p2p-peercollabsignout)                                                 | Signiert einen Peer von einem bestimmten Typ eines Peerzusammenarbeitsnetzwerk-Anwesenheitsanbieters.                                                                                                            |
| [**PeerCollabShutdown**](/windows/desktop/api/P2P/nf-p2p-peercollabshutdown)                                               | Fährt die Peerzusammenarbeitsinfrastruktur herunter und gibt alle zugeordneten Ressourcen frei.                                                                                                 |
| [**PeerCollabStartup**](/windows/desktop/api/P2P/nf-p2p-peercollabstartup)                                                 | Initialisiert die Peerzusammenarbeitsinfrastruktur.                                                                                                                                              |
| [**PeerCollabSubscribeEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabsubscribeendpointdata)                     | Erstellt ein Abonnement für einen verfügbaren Endpunkt.                                                                                                                                                |
| [**PeerCollabUnregisterApplication**](/windows/desktop/api/P2P/nf-p2p-peercollabunregisterapplication)                     | Aufheben der Registrierung der spezifischen Anwendungen eines Peers auf dem lokalen Computer.                                                                                                                        |
| [**PeerCollabUnregisterEvent**](/windows/desktop/api/P2P/nf-p2p-peercollabunregisterevent)                                 | Aufheben der Registrierung einer Anwendung bei Benachrichtigungen zu bestimmten Peerzusammenarbeitsereignissen.                                                                                                          |
| [**PeerCollabUnsubscribeEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabunsubscribeendpointdata)                 | Entfernt ein Abonnement für einen Endpunkt, der mit PeerCollabSubscribeEndpointData erstellt wurde.                                                                                                             |
| [**PeerCollabUpdateContact**](/windows/desktop/api/P2P/nf-p2p-peercollabupdatecontact)                                     | Aktualisiert einen Peer, der an einem Peerzusammenarbeitsnetzwerk beteiligt ist, mit neuen Informationen zu einem Peerkontakt.                                                                                            |
