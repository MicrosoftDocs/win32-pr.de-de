# <a name="graphing-api-structures"></a>Grafisch rentieren von API-Strukturen

Die graphende-API verwendet die folgenden Strukturen:



| Struktur                                                                 | BESCHREIBUNG                                                                                                   |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**\_ \_ \_ Änderungs Daten für Peer Ereignis Knoten \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_node_change_data)    | Enthält einen Zeiger auf die Daten, wenn ein Änderungs Ereignis für ein **Peer \_ Graph- \_ Ereignis \_ Knoten \_** ausgelöst wird.                  |
| [**\_ \_ Synchronisierte Daten des Peer Ereignisses \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_synchronized_data)   | Gibt den Typ des Datensatzes an, der synchronisiert wurde.                                                      |
| [**Peer \_ Graph- \_ Ereignis \_ Daten**](/windows/desktop/api/P2P/ns-p2p-peer_graph_event_data)                 | Enthält Daten, die einem Peer Ereignis zugeordnet sind.                                                                   |
| [**Peer \_ Graph- \_ Ereignis \_ Registrierung**](/windows/desktop/api/P2P/ns-p2p-peer_graph_event_registration) | Gibt an, für welche Peer Ereignisse eine Anwendung Benachrichtigungen benötigt.                                        |
| [**Peer \_ Graph- \_ Eigenschaften**](/windows/desktop/api/P2P/ns-p2p-peer_graph_properties)                  | Enthält Daten über die Richtlinie eines Peer Diagramms, einer ID, eines Bereichs und anderer Informationen.                             |
| [**Peer \_ Knoten \_ Informationen**](/windows/desktop/api/P2P/ns-p2p-peer_node_info)                                | Enthält Informationen, die spezifisch für einen bestimmten Knoten in einem Peer Diagramm sind.                                   |
| [**Peer \_ Sicherheits \_ Schnittstelle**](/windows/desktop/api/P2P/ns-p2p-peer_security_interface)              | Gibt die Sicherheits Schnittstellen an, die Aufrufe von Peer-graphingapis verwenden, um Datensätze zu validieren, zu sichern und freizugeben. |
