# <a name="windows-vista-components-for-people-near-me"></a>Windows Vista-Komponenten für Personen in meiner Umgebung

[Personen in meiner Umgebung](about-people-near-me.md) (PNM) verwendet die folgenden Komponenten von Windows Vista:

### <a name="web-service-discovery"></a>Webdienstsuche

Die Komponente Webdienstermittlung ermöglicht die Veröffentlichung und Ermittlung von Diensten im lokalen Subnetz. In PNM ist ein Benutzer ein Dienst, der im lokalen Subnetz angekündigt wird. Weitere Informationen zur Webdienstermittlung finden Sie unter [XML-Webdienstermittlung.](/previous-versions/dotnet/netframework-4.0/fxx6cfx2(v=vs.100))

### <a name="schannel"></a>Schannel

Die Schannel-Komponente ermöglicht eine verschlüsselte Kommunikation zwischen zwei Endpunkten.

### <a name="windows-sockets"></a>Windows-Sockets

Die komponente Windows Sockets wird von Anwendungen für die Kommunikation über TCP/IP verwendet.

### <a name="next-generation-tcpip-stack"></a>TCP/IP-Stapel der nächsten Generation

Neuer TCP/IP-Stapel in Windows Vista, in dem IPv6 integriert und standardmäßig aktiviert ist. Weitere Informationen finden Sie unter [TCP/IP-Stapel](https://www.microsoft.com/technet/itsolutions/network/tcpip/default.mspx)der nächsten Generation in Windows Vista.
