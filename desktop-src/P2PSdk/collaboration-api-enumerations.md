# <a name="collaboration-api-enumerations"></a>Zusammenarbeits-API-Enumerationen

Die folgenden Enumerationen unterstützen die Infrastrukturfunktionen der Peerzusammenarbeit.



| Enumeration                                                                         | Beschreibung                                                                                                                                                                                                                                                                   |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_REGISTRIERUNGSTYP DER PEERANWENDUNG \_ \_**](/windows/desktop/api/P2P/ne-p2p-peer_application_registration_type) | Definiert den Satz von Peeranwendungsregistrierungsflags.                                                                                                                                                                                                                       |
| [**\_ \_ PEERÄNDERUNGSTYP**](/windows/desktop/api/P2P/ne-p2p-peer_change_type)                                      | Definiert den Satz von Änderungen, die für ein Peerobjekt, einen Endpunkt oder eine Anwendung ausgeführt werden können.                                                                                                                                                                                  |
| [**PEER \_ \_ SIGNIN-FLAGS**](/windows/desktop/api/P2P/ne-p2p-peer_signin_flags)                                    | Definiert die Veröffentlichungsverhalten von Peers, die verfügbar sind, wenn sich der Peer bei einem Peerzusammenarbeitsnetzwerk anmeldet.                                                                                                                                                      |
| [**PEER \_ \_ WATCH-BERECHTIGUNG**](/windows/desktop/api/P2P/ne-p2p-peer_watch_permission)                            | Definiert den Berechtigungssatz, der die Präsenz eines Kontakts bei einem Beobachtenden Peer darstellt.                                                                                                                                                                                        |
| [**PEER \_ \_ COLLAB-EREIGNISTYP \_**](/windows/desktop/api/P2P/ne-p2p-peer_collab_event_type)                         | Definiert den Satz von Ereignissen, die von der Netzwerkereignisinfrastruktur der Peerzusammenarbeit auf einem Peer ausgelöst werden können.                                                                                                                                                                |
| [**\_ \_ PEER-ANWESENHEITSSTATUS**](/windows/desktop/api/P2P/ne-p2p-peer_presence_status)                              | Definiert den Satz möglicher Anwesenheitsstatuseinstellungen, die einem Peer zur Verfügung stehen, der an einem Peerzusammenarbeitsnetzwerk teilnimmt. Diese Einstellungen können von einem Peerzusammenarbeitsnetzwerkendpunkt festgelegt werden, um die aktuelle Beteiligungsebene des Peerkontakts für seine Watcher anzugeben. |
| [**ANTWORTTYP DER \_ PEER-EINLADUNG \_ \_**](/windows/desktop/api/P2P/ne-p2p-peer_invitation_response_type)           | Definiert den Satz von Antworten, die auf eine Einladung zum Beitreten zu einer Peerzusammenarbeitsaktivität empfangen werden.                                                                                                                                                                                 |
| [**\_ \_ PEERVERÖFFENTLICHUNGSBEREICH**](/windows/desktop/api/P2P/ne-p2p-peer_publication_scope)                          | Definiert den Satz von Bereichen für die Veröffentlichung von Peerobjekten oder -daten.                                                                                                                                                                                                        |
