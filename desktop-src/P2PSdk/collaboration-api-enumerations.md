# <a name="collaboration-api-enumerations"></a>Zusammenarbeits-API-Enumerationen

Die folgenden Enumerationen unterstützen die Funktionen der Peerzusammenarbeitsinfrastruktur.



| Enumeration                                                                         | Beschreibung                                                                                                                                                                                                                                                                   |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ \_ PEERANWENDUNGSREGISTRIERUNGSTYP \_**](/windows/desktop/api/P2P/ne-p2p-peer_application_registration_type) | Definiert den Satz von Peeranwendungsregistrierungsflags.                                                                                                                                                                                                                       |
| [**\_ \_ PEERÄNDERUNGSTYP**](/windows/desktop/api/P2P/ne-p2p-peer_change_type)                                      | Definiert den Satz von Änderungen, die für ein Peerobjekt, einen Endpunkt oder eine Anwendung ausgeführt werden können.                                                                                                                                                                                  |
| [**PEER \_ \_ SIGNIN-FLAGS**](/windows/desktop/api/P2P/ne-p2p-peer_signin_flags)                                    | Definiert den Satz von Peer-Presence-Veröffentlichungsverhalten, das verfügbar ist, wenn sich der Peer bei einem Peerzusammenarbeitsnetzwerk ankn.)                                                                                                                                                      |
| [**PEER \_ \_ WATCH-BERECHTIGUNG**](/windows/desktop/api/P2P/ne-p2p-peer_watch_permission)                            | Definiert den Berechtigungssatz, der das Vorhandensein eines Kontakts für einen Beobachtenden Peer repräsentiert.                                                                                                                                                                                        |
| [**PEER \_ \_ COLLAB-EREIGNISTYP \_**](/windows/desktop/api/P2P/ne-p2p-peer_collab_event_type)                         | Definiert den Satz von Ereignissen, die von der Netzwerkereignisinfrastruktur für die Peerzusammenarbeit auf einem Peer ausgelöst werden können.                                                                                                                                                                |
| [**PEER \_ PRESENCE \_ STATUS**](/windows/desktop/api/P2P/ne-p2p-peer_presence_status)                              | Definiert den Satz möglicher Einstellungen für den Anwesenheitsstatus, die für einen Peer verfügbar sind, der an einem Peerzusammenarbeitsnetzwerk beteiligt ist. Diese Einstellungen können von einem Peerzusammenarbeits-Netzwerkendpunkt festgelegt werden, um die aktuelle Beteiligungsebene des Peerkontakts für seine Watcher anzugeben. |
| [**ANTWORTTYP \_ DER \_ \_ PEEREINLADUNG**](/windows/desktop/api/P2P/ne-p2p-peer_invitation_response_type)           | Definiert den Satz von Antworten, die auf eine Einladung zum Beitreten zu einer Peerzusammenarbeitsaktivität empfangen werden.                                                                                                                                                                                 |
| [**\_ \_ PEERVERÖFFENTLICHUNGSBEREICH**](/windows/desktop/api/P2P/ne-p2p-peer_publication_scope)                          | Definiert den Satz von Geltungsbereichen für die Veröffentlichung von Peerobjekten oder Daten.                                                                                                                                                                                                        |
