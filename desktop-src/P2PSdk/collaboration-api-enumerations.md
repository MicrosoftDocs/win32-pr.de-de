# <a name="collaboration-api-enumerations"></a>Zusammenarbeits-API-Enumerationen

Die folgenden Enumerationen unterstützen die Infrastrukturfunktionen der Peer Kollaboration.



| Enumeration                                                                         | Beschreibung                                                                                                                                                                                                                                                                   |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Peer \_ Anwendungs \_ \_ Registrierungstyp**](/windows/desktop/api/P2P/ne-p2p-peer_application_registration_type) | Definiert den Satz von Registrierungs Flags für Peer Anwendungen.                                                                                                                                                                                                                       |
| [**Peer \_ \_ Änderungstyp**](/windows/desktop/api/P2P/ne-p2p-peer_change_type)                                      | Definiert den Satz von Änderungen, der für ein Peer Objekt, einen Endpunkt oder eine Anwendung ausgeführt werden kann.                                                                                                                                                                                  |
| [**Peer \_ SignIn- \_ Flags**](/windows/desktop/api/P2P/ne-p2p-peer_signin_flags)                                    | Definiert die Gruppe von Peer Anwesenheits-Veröffentlichungs Verhalten, die verfügbar sind, wenn sich der Peer bei einem Peer-Zusammenarbeits Netzwerk anmeldet.                                                                                                                                                      |
| [**Peer \_ Überwachungs \_ Berechtigung**](/windows/desktop/api/P2P/ne-p2p-peer_watch_permission)                            | Definiert den Berechtigungs Satz, der die Anwesenheit eines Kontakts zu einem beobachtenden Peer darstellt.                                                                                                                                                                                        |
| [**\_peercollab- \_ \_ Ereignistyp**](/windows/desktop/api/P2P/ne-p2p-peer_collab_event_type)                         | Definiert den Satz von Ereignissen, die von der Peer Zusammenarbeits Netzwerk-Ereignis Infrastruktur für einen Peer ausgelöst werden können.                                                                                                                                                                |
| [**Peer \_ Anwesenheits \_ Status**](/windows/desktop/api/P2P/ne-p2p-peer_presence_status)                              | Definiert den Satz möglicher Anwesenheitsstatus Einstellungen, die für einen Peer verfügbar sind, der an einem Peer Zusammenarbeits Netzwerk teilnimmt. Diese Einstellungen können von einem Peer-Zusammenarbeits Netzwerk-Endpunkt festgelegt werden, um die aktuelle Ebene der Teilnahme an den Watchern des Peer Kontakts anzugeben. |
| [**\_peereinladungs- \_ \_ Antworttyp**](/windows/desktop/api/P2P/ne-p2p-peer_invitation_response_type)           | Definiert den Satz von Antworten, die für eine Einladung empfangen werden, um einer Peer Kollaborations Aktivität beizutreten.                                                                                                                                                                                 |
| [**Peer \_ Veröffentlichungs \_ Bereich**](/windows/desktop/api/P2P/ne-p2p-peer_publication_scope)                          | Definiert den Satz von Bereichen für die Veröffentlichung von Peer Objekten oder-Daten.                                                                                                                                                                                                        |
