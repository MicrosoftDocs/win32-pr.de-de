# <a name="common-functions-peer-to-peer"></a>Allgemeine Funktionen (Peer-zu-Peer)

Die Peerinfrastruktur verwendet die folgenden allgemeinen Funktionen.



| Funktion                                         | BESCHREIBUNG                                                                                                                          |
|--------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| [**PeerEndEnumeration**](/windows/desktop/api/P2P/nf-p2p-peerendenumeration) | Gibt eine Enumeration frei, z. B. einen Datensatz oder eine Memberenumeration, und gibt die Der -Enumeration zugeordneten Ressourcen frei. |
| [**PeerFreeData**](/windows/desktop/api/P2P/nf-p2p-peerfreedata)             | Gibt einen Datenblock frei und gibt ihn an den Speicherpool zurück.                                                                       |
| [**PeerGetItemCount**](/windows/desktop/api/P2P/nf-p2p-peergetitemcount)     | Gibt die Anzahl der Elemente in einer Peerenumeration zurück.                                                                                  |
| [**PeerGetNextItem**](/windows/desktop/api/P2P/nf-p2p-peergetnextitem)       | Gibt eine bestimmte Anzahl von Elementen aus einer Peerenumeration zurück.                                                                          |
