# <a name="grouping-api-structures"></a>Gruppieren von API-Strukturen

Die Gruppierungs-API verwendet die folgenden Strukturen.



| Struktur                                                                  | BESCHREIBUNG                                                                                                   |
|----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**Informationen zu Peer Anmelde \_ \_ Informationen**](/windows/desktop/api/P2P/ns-p2p-peer_credential_info)                     | Definiert Informationen, die zum Abrufen und Ausstellen der Sicherheits Anmelde Informationen eines Peers verwendet werden.                                   |
| [**Änderungs Daten für Peer- \_ Ereignis \_ Mitglied \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_member_change_data) | Enthält Daten, die eine Änderung des Status eines Peer Gruppenmitglieds beschreiben.                                   |
| [**Peer \_ Gruppen- \_ Ereignis \_ Daten**](/windows/win32/api/p2p/ns-p2p-peer_group_event_data-r1)                  | Enthält Informationen zu einem bestimmten Ereignis der Peer Gruppe.                                     |
| [**Peer \_ Gruppen- \_ Ereignis \_ Registrierung**](/windows/desktop/api/P2P/ns-p2p-peer_group_event_registration)  | Definiert das jeweilige Peer Gruppen Ereignis, für das ein Mitglied registriert werden kann.                                            |
| [**Peer \_ Gruppen \_ Eigenschaften**](/windows/desktop/api/P2P/ns-p2p-peer_group_properties)                   | Enthält Daten über die Mitgliedschafts Richtlinie einer Peer Gruppe.                                                    |
| [**Peer \_ Einladungs \_ Informationen**](/windows/desktop/api/P2P/ns-p2p-peer_invitation_info)                     | Definiert Informationen über eine Einladung zum beitreten zu einer Peer Gruppe. Einladungen werden als Unicode-Zeichen folgen dargestellt. |
| [**\_peermember**](/windows/desktop/api/P2P/ns-p2p-peer_member)                                        | Enthält Informationen, in denen ein Mitglied einer Peer Gruppe beschrieben wird.                                                 |
