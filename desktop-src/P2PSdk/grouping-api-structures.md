# <a name="grouping-api-structures"></a>Gruppieren von API-Strukturen

Die Gruppierungs-API verwendet die folgenden Strukturen.



| Struktur                                                                  | BESCHREIBUNG                                                                                                   |
|----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**\_PEERANMELDEINFORMATIONEN \_**](/windows/desktop/api/P2P/ns-p2p-peer_credential_info)                     | Definiert Informationen, die zum Abrufen und Ausstellen der Sicherheitsanmeldeinformationen eines Peers verwendet werden.                                   |
| [**ÄNDERUNGSDATEN VON \_ \_ PEEREREIGNISSEN \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_member_change_data) | Enthält Daten, die eine Änderung des Status eines Peergruppenmitglieds beschreiben.                                   |
| [**\_ \_ \_ PEERGRUPPENEREIGNISDATEN**](/windows/win32/api/p2p/ns-p2p-peer_group_event_data-r1)                  | Enthält Informationen zu einem bestimmten Peergruppenereignis, das aufgetreten ist.                                     |
| [**\_ \_ \_ PEERGRUPPENEREIGNISREGISTRIERUNG**](/windows/desktop/api/P2P/ns-p2p-peer_group_event_registration)  | Definiert das bestimmte Peergruppenereignis, für das sich ein Mitglied registrieren kann.                                            |
| [**\_ \_ PEERGRUPPENEIGENSCHAFTEN**](/windows/desktop/api/P2P/ns-p2p-peer_group_properties)                   | Enthält Daten zur Mitgliedschaftsrichtlinie einer Peergruppe.                                                    |
| [**\_PEER-EINLADUNGSINFORMATIONEN \_**](/windows/desktop/api/P2P/ns-p2p-peer_invitation_info)                     | Definiert Informationen zu einer Einladung zum Beitreten zu einer Peergruppe. Einladungen werden als Unicode-Zeichenfolgen dargestellt. |
| [**\_PEERMITGLIED**](/windows/desktop/api/P2P/ns-p2p-peer_member)                                        | Enthält Informationen, die ein Mitglied einer Peergruppe beschreiben.                                                 |
