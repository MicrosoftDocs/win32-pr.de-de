# <a name="grouping-api-enumerations"></a>Gruppieren von API-Enumerationen

Die Gruppierungs-API verwendet die folgenden Enumerationen:



| Enumeration                                                                        | Beschreibung                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Peer \_ Gruppen- \_ Authentifizierungs \_ Schema**](/windows/desktop/api/P2P/ne-p2p-peer_group_authentication_scheme)    | Definiert den Satz möglicher Authentifizierungs Schemas, die verwendet werden können, um Peers zu authentifizieren, die einer Peer Gruppe beitreten.                                                                                                                                           |
| [**Peer \_ Gruppen \_ - \_ Ereignistyp**](/windows/desktop/api/P2P/ne-p2p-peer_group_event_type)                          | Enthält die spezifischen Peer Ereignis Typen, die in einer Peer Gruppe auftreten können.                                                                                                                                                                                |
| [**Anmelde \_ \_ \_ informationsflags für Peer Gruppen Probleme \_**](/windows/win32/api/p2p/ne-p2p-peer_group_issue_credential_flags) | Gibt an, ob Benutzer Anmelde Informationen in einer Gruppe gespeichert werden.                                                                                                                                                                                                  |
| [**Peer \_ Gruppen \_ - \_ Eigenschaftenflags**](/windows/desktop/api/P2P/ne-p2p-peer_group_property_flags)                  | Gibt die Peer Gruppen-Mitgliedschafts Einstellungen an.                                                                                                                                                                                                                 |
| [**Peer \_ Gruppen \_ Status**](/windows/desktop/api/P2P/ne-p2p-peer_group_status)                                   | Gibt an, ob für die Peer Gruppe Verbindungen vorhanden sind.                                                                                                                                                                                          |
| [**\_ \_ Änderungstyp für Peer Member \_**](/windows/desktop/api/P2P/ne-p2p-peer_member_change_type)                      | Definiert den Satz möglicher Peer Gruppen Mitgliedschafts-und Anwesenheits Zustände für einen Peer.                                                                                                                                                                         |
| [**\_peermember- \_ Flags**](/windows/desktop/api/P2P/ne-p2p-peer_member_flags)                                   | Ermöglicht einer Anwendung, anzugeben, ob alle Member oder nur vorhandene Member aufgelistet werden sollen, wenn die [**peergroupummembers**](/windows/desktop/api/P2P/nf-p2p-peergroupenummembers) -Funktion aufgerufen wird, oder, um anzugeben, ob ein Member innerhalb der Peer Gruppe vorhanden ist. |
