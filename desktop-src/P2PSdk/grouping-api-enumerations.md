# <a name="grouping-api-enumerations"></a>Gruppieren von API-Enumerationen

Die Gruppierungs-API verwendet die folgenden Enumerationen:



| Enumeration                                                                        | Beschreibung                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ \_ \_ PEERGRUPPENAUTHENTIFIZIERUNGSSCHEMA**](/windows/desktop/api/P2P/ne-p2p-peer_group_authentication_scheme)    | Definiert den Satz möglicher Authentifizierungsschemas, die zum Authentifizieren von Peers verwendet werden können, die einer Peergruppe beitreten.                                                                                                                                           |
| [**\_ \_ \_ PEERGRUPPENEREIGNISTYP**](/windows/desktop/api/P2P/ne-p2p-peer_group_event_type)                          | Enthält die spezifischen Peerereignistypen, die innerhalb einer Peergruppe auftreten können.                                                                                                                                                                                |
| [**FLAGS FÜR ANMELDEINFORMATIONEN FÜR \_ \_ \_ \_ PEERGRUPPENPROBLEM**](/windows/win32/api/p2p/ne-p2p-peer_group_issue_credential_flags) | Gibt an, ob Benutzeranmeldeinformationen in einer Gruppe gespeichert werden.                                                                                                                                                                                                  |
| [**\_ \_ \_ PEERGRUPPENEIGENSCHAFTENFLAGS**](/windows/desktop/api/P2P/ne-p2p-peer_group_property_flags)                  | Gibt Einstellungen für die Peergruppenmitgliedschaft an.                                                                                                                                                                                                                 |
| [**\_ \_ PEERGRUPPENSTATUS**](/windows/desktop/api/P2P/ne-p2p-peer_group_status)                                   | Gibt an, ob für die Peergruppe Verbindungen vorhanden sind.                                                                                                                                                                                          |
| [**\_ \_ PEERMEMBER-ÄNDERUNGSTYP \_**](/windows/desktop/api/P2P/ne-p2p-peer_member_change_type)                      | Definiert den Satz möglicher Peergruppenmitgliedschaften und Anwesenheitszustände für einen Peer.                                                                                                                                                                         |
| [**\_ \_ PEERMEMBERFLAGS**](/windows/desktop/api/P2P/ne-p2p-peer_member_flags)                                   | Ermöglicht einer Anwendung anzugeben, ob alle oder nur vorhandenen Member aufzählt werden sollen, wenn die [**PeerGroupEnumMembers-Funktion**](/windows/desktop/api/P2P/nf-p2p-peergroupenummembers) aufgerufen wird, oder ob ein Mitglied innerhalb der Peergruppe vorhanden ist. |
