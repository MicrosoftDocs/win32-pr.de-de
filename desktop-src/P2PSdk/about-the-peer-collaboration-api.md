# <a name="about-the-peer-collaboration-api"></a>Informationen zur Peerzusammenarbeits-API

Die Peerzusammenarbeitsinfrastruktur bietet ein netzwerkbasiertes Peerframework für gemeinsame serverlose Aktivitäten, z. B. Netzwerkspiel-Match-Schaltung, Konferenzaktivitäten und andere interaktive Aktivitäten mit mehreren Teilnehmern.

Die Peerzusammenarbeitsinfrastruktur stellt APIs zur Verfügung, die den Prozess vereinfachen, mit dem Anwendungen peer presence ohne Server nachverfolgen, Einladungen an Teilnehmer senden, Peers im gleichen Subnetz oder LAN entdecken und Kontakte verwalten können.

Darüber hinaus ermöglicht diese Infrastruktur Peers auch, andere Peers im gleichen Subnetz oder lokalen Netzwerk zu finden, und ermöglicht die Interaktion zwischen diesen Peers.

Serverloses Vorhandensein ist ein Feature der Peerzusammenarbeitsinfrastruktur, das für alle Windows Vista-Anwendungen verfügbar ist. Jede Anwendung, die die serverlose Anwesenheitskomponente der Zusammenarbeitsinfrastruktur verwendet, kann die Präsenz des Benutzers für jede Anwendung, die sie beobachtet, ordnungsgemäß anzeigen. Die Benutzer können auch detaillierte Informationen zu ihrem Vorhandensein veröffentlichen und Datenobjekte veröffentlichen, die beim Peer registriert sind.
