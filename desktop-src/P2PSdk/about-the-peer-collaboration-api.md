# <a name="about-the-peer-collaboration-api"></a>Informationen zur Peer Kollaborations-API

Die Infrastruktur für die Peer Zusammenarbeit bietet ein Peer-netzwerkbasiertes Framework für die Zusammenarbeit von Server losen Aktivitäten, z. b. Netzwerkspiel Matchmaking, Konferenzen und andere interaktive Aktivitäten mit mehreren Teilnehmern.

Die Infrastruktur für die Peer Zusammenarbeit bietet APIs, die den Prozess vereinfachen, mit dem Anwendungen Peer Präsenz ohne Server nachverfolgen, Einladungen an Teilnehmer senden, Peers im gleichen Subnetz oder LAN ermitteln und Kontakte verwalten können.

Außerdem ermöglicht diese Infrastruktur es Peers auch, andere Peers im gleichen Subnetz oder lokalen Netzwerk zu ermitteln, und ermöglicht die Interaktion zwischen diesen Peers.

Server lose Anwesenheit ist ein Feature der Infrastruktur für die Peer Zusammenarbeit, das für alle Windows Vista-Anwendungen verfügbar ist. Jede Anwendung, die die Server lose Anwesenheits Komponente der Zusammenarbeits Infrastruktur verwendet, kann die Anwesenheit des Benutzers für jede Anwendung, die Sie überwacht, ordnungsgemäß anzeigen. Die Benutzer können auch ausführliche Informationen über Ihre Anwesenheit veröffentlichen und Datenobjekte veröffentlichen, die beim Peer registriert sind.
