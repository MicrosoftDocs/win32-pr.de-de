# <a name="common-constants"></a>Allgemeine Konstanten

Die Peerinfrastruktur verwendet die folgenden allgemeinen Konstanten.



| Konstante                                                      | BESCHREIBUNG                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------|
| [**\_ \_ PEERVERBINDUNGSFLAGS**](/windows/desktop/api/P2P/ne-p2p-peer_connection_flags)      | Gibt die Verbindungstypen an, über die ein Peer verfügen kann.      |
| [**\_ \_ PEERVERBINDUNGSSTATUS**](/windows/desktop/api/P2P/ne-p2p-peer_connection_status)    | Gibt den Status einer direkten Peer- oder Nachbarverbindung an. |
| [**\_ \_ \_ PEEREINTRAGSÄNDERUNGSTYP**](/windows/desktop/api/P2P/ne-p2p-peer_record_change_type) | Gibt die Änderungen an, die an einem Datensatz vorgenommen werden können.             |
| [**\_ \_ PEEREINTRAGSFLAGS**](/windows/desktop/api/P2P/ne-p2p-peer_record_flags)              | Gibt einen Satz von Flags für Peerdatensatzverhalten an.           |
