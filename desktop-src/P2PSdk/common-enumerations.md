# <a name="common-constants"></a>Allgemeine Konstanten

Die Peer Infrastruktur verwendet die folgenden allgemeinen Konstanten.



| Konstante                                                      | BESCHREIBUNG                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------|
| [**Peer \_ - \_ verbindungsflags**](/windows/desktop/api/P2P/ne-p2p-peer_connection_flags)      | Gibt die Verbindungstypen an, die ein Peer aufweisen kann.      |
| [**Peer \_ Verbindungs \_ Status**](/windows/desktop/api/P2P/ne-p2p-peer_connection_status)    | Gibt den Status einer direkten Peer-oder Nachbar Verbindung an. |
| [**\_Änderungstyp des Peer Datensatzes \_ \_**](/windows/desktop/api/P2P/ne-p2p-peer_record_change_type) | Gibt die Änderungen an, die an einem Datensatz auftreten können.             |
| [**Peer \_ Daten Satz- \_ Flags**](/windows/desktop/api/P2P/ne-p2p-peer_record_flags)              | Gibt eine Gruppe von Flags für das Verhalten von Peer Datensätzen an.           |
