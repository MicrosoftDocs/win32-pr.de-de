# <a name="peer-distribution"></a>Peer Verteilung

Die Peer Distribution-API, die das BranchCache-Feature in Windows 7 und Windows Server 2008 R2 unterstützt, bietet eine Reihe von Plattform-APIs, mit denen sich die Reaktionsfähigkeit zentralisierter Anwendungen beim Zugriff über Remote Niederlassungen erhöhen kann, sodass Benutzer in diesen Büros die Erfahrung haben, in Ihrem lokalen Netzwerk zu arbeiten. Die Funktionen dieses Satzes von APIs können auch zur Reduzierung der gesamten WAN-Auslastung (Wide Area Network) beitragen.

Zum Implementieren dieser Features ist keine bereits vorhandene Infrastruktur erforderlich. Verbesserungen an der Leistung von Remote Netzwerken werden einfach durch Bereitstellen von Windows 7 auf Client Computern, Bereitstellen von Windows Server 2008 R2 auf Server Computern und Aktivieren der Verzweigungs Cache-Funktion erreicht.

Die Peer Distribution-APIs können nahtlos zusammen mit Netzwerk Sicherheitstechnologien verwendet werden, einschließlich SSL, SMB-Signaturen und End-to-End-IPSec. Mit dieser API können Sie die Auslastung der Netzwerkbandbreite verringern und die Anwendungsleistung verbessern, auch wenn der Inhalt verschlüsselt ist.
