# <a name="peer-distribution"></a>Peerverteilung

Die Peerverteilungs-API, die das Branchcachefeature in Windows 7 und Windows Server 2008 R2 unterstützt, bietet eine Reihe von Plattform-APIs, mit denen die Netzwerkschnelligkeit zentralisierter Anwendungen erhöht werden kann, wenn von Remotebüros aus darauf zugegriffen wird, damit Benutzer in diesen Niederlassungen in Ihrem lokalen Netzwerk arbeiten können. Die von dieser Gruppe von APIs gebotenen Funktionen können auch dazu beitragen, die allgemeine WAN-Auslastung (Wide Area Network) zu reduzieren.

Für die Implementierung dieser Features ist keine bereits vorhandene Infrastruktur erforderlich. Verbesserungen an der Leistung von Remotenetzwerken werden einfach durch die Bereitstellung von Windows 7 auf Clientcomputern, die Bereitstellung von Windows Server 2008 R2 auf Servercomputern und das Aktivieren des Branchcachefeatures erreicht.

Die Peerverteilungs-APIs funktionieren nahtlos zusammen mit Netzwerksicherheitstechnologien, einschließlich SSL, SMB-Signatur und End-to-End-IPsec. Sie können diese API verwenden, um die Auslastung der Netzwerkbandbreite zu reduzieren und die Anwendungsleistung zu verbessern, auch wenn der Inhalt verschlüsselt ist.
