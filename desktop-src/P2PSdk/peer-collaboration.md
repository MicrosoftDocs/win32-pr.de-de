# <a name="peer-collaboration"></a>Peerzusammenarbeit

Die Peerzusammenarbeitsinfrastruktur bietet eine Reihe von Anwendungsprogrammierschnittstellen, die in Verbindung mit der Peerinfrastruktur verwendet werden können, um gemeinsame serverlose Netzwerkanwendungen zu ermöglichen, z. B. Instant Messaging und Multiplayerspiel-Matchgestaltung.
