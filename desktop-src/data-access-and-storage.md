# <a name="data-access-and-storage"></a>Datenzugriff und-Speicherung

Windows verfügt über APIs, Komponenten und Dienste, die Ihre Desktop-Apps im Datenzugriff und-Speicher unterstützen. Sie bieten:

-   Datei-und Dateisystem Verwaltung.
-   Datenbankzugriff.
-   Unterstützung für die Übertragung, Synchronisierung und Replikation von Daten.
-   Zugriff auf XML-, Paket-und Protokolldateien.
-   Bild-Mastering.
-   Unterstützung der Sicherung.
