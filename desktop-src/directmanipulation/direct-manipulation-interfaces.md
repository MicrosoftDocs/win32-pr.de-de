# <a name="direct-manipulation-interfaces"></a>Schnittstellen für die direkte Bearbeitung

Die in diesem Abschnitt enthaltenen Themen enthalten die Referenzspezifikationen für [Schnittstellen für die direkte Bearbeitung.](direct-manipulation-portal.md)

> [!Note]  
> Stellen Sie beim Implementieren [eines Objekts der direkten Manipulation](direct-manipulation-portal.md) sicher, dass die [**IUnknown-Implementierung**](/windows/win32/api/unknwn/nn-unknwn-iunknown) Multithreading durch threadsichere Verweiszählung unterstützt. Weitere Informationen finden Sie unter [**InterlockedIncrement**](/windows/win32/api/winnt/nf-winnt-interlockedincrement) und [**InterlockedDecrement**](/windows/win32/api/winnt/nf-winnt-interlockeddecrement).

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema                                                                                                       | BESCHREIBUNG                                                                                                                                                                                                                                                                  |
|-------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IDirectManipulationAutoScrollBehavior**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationautoscrollbehavior)<br/>           | Stellt das Animationsverhalten für automatisches Scrollen von Inhalten dar, wenn er sich der Begrenzung einer bestimmten Achse oder Achsen nähert.<br/>                                                                                                                                                   |
| [**IDirectManipulationCompositor**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationcompositor)<br/>                           | Stellt ein Compositorobjekt dar, das bearbeiteten Inhalt einer Zeichenoberfläche zuordnt, z. B. [**Canvas**](/uwp/api/Windows.UI.Xaml.Controls.Canvas).<br/> |
| [**IDirectManipulationCompositor2**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationcompositor2)<br/>                         | Stellt ein Compositorobjekt dar, das bearbeiteten Inhalt zeichenoberflächen über mehrere Prozesse hinweg zuordnt.<br/>                                                                                                                                               |
| [**IDirectManipulationContent**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationcontent)<br/>                                 | Kapselt Inhalt in einem Viewport, wobei der Inhalt eine visuelle Oberfläche darstellt, die im Viewport abgeschnitten ist.<br/>                                                                                                                                                    |
| [**IDirectManipulationDeferContactService**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationdefercontactservice)<br/>         | Stellt einen Dienst zum Verwalten von Zuordnungen zwischen einem Kontakt und einem Viewport dar.<br/>                                                                                                                                                                                  |
| [**IDirectManipulationDragDropBehavior**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationdragdropbehavior)<br/>               | Stellt Verhaltensweisen für Drag & Drop-Interaktionen dar, die durch Querschiebe- oder Gedrückthalten-Gesten ausgelöst werden. <br/>                                                                                                                                              |
| [**IDirectManipulationDragDropEventHandler**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationdragdropeventhandler)<br/>       | Definiert Methoden zum Behandeln von Drag & Drop-Verhaltensereignissen.<br/>                                                                                                                                                                                                              |
| [**IDirectManipulationFrameInfoProvider**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationframeinfoprovider)<br/>             | Stellt ein Zeiterhaltungsobjekt dar, das die Latenz der von der Anwendung verwendeten Kompositionsinfrastruktur misst und diese Daten [für die direkte Bearbeitung zur Verfügung stellt.](direct-manipulation-portal.md) <br/>                                                            |
| [**IDirectManipulationInteractionEventHandler**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationinteractioneventhandler)<br/> | Definiert Methoden, um Interaktionen zu behandeln, wenn sie erkannt werden.<br/>                                                                                                                                                                                                    |
| [**IDirectManipulationManager**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager)<br/>                                 | Bietet Zugriff auf alle Features und APIs [der direkten](direct-manipulation-portal.md) Bearbeitung, die für die Clientanwendung verfügbar sind.<br/>                                                                                                                           |
| [**IDirectManipulationManager2**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager2)<br/>                               | Erweitert die [**IDirectManipulationManager-Schnittstelle,**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager) die [](direct-manipulation-portal.md) Zugriff auf alle Features und APIs der direkten Bearbeitung bietet, die für die Clientanwendung verfügbar sind. <br/>                              |
| [**IDirectManipulationManager3**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager3)<br/>                               | Erweitert die [**IDirectManipulationManager2-Schnittstelle,**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager2) die [](direct-manipulation-portal.md) Zugriff auf alle Features und APIs der direkten Bearbeitung bietet, die für die Clientanwendung verfügbar sind. <br/>                            |
| [**IDirectManipulationPrimaryContent**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationprimarycontent)<br/>                   | Kapselt den primären Inhalt in einem Viewport.<br/>                                                                                                                                                                                                               |
| [**IDirectManipulationUpdateHandler**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationupdatehandler)<br/>                     | Definiert Methoden zum Behandeln von Manipulationsaktualisierungsereignissen.<br/>                                                                                                                                                                                                          |
| [**IDirectManipulationUpdateManager**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationupdatemanager)<br/>                     | Verwaltet, wie Compositorupdates an direct [Manipulation gesendet werden.](direct-manipulation-portal.md)<br/>                                                                                                                                                                 |
| [**IDirectManipulationViewport**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationviewport)<br/>                               | Definiert einen Bereich innerhalb eines Fensters (als Viewport bezeichnet), der Eingaben von Benutzerinteraktionen empfangen und verarbeiten kann. <br/>                                                                                                                                   |
| [**IDirectManipulationViewport2**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationviewport2)<br/>                             | Ermöglicht die Verwaltung von Verhaltensweisen in einem Viewport. Ein Verhalten wirkt sich auf die Funktionalität eines bestimmten Teils des Workflows für die direkte [Bearbeitung](direct-manipulation-portal.md) aus. <br/>                                                                                 |
| [**IDirectManipulationViewportEventHandler**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationviewporteventhandler)<br/>       | Definiert Methoden zum Behandeln von Status- und Aktualisierungsereignissen für den Viewport.<br/>                                                                                                                                                                                           |
