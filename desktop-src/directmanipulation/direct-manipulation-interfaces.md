# <a name="direct-manipulation-interfaces"></a>Schnittstellen für direkte Manipulation

Die Themen in diesem Abschnitt enthalten die Referenz Spezifikationen für [direkte Manipulations](direct-manipulation-portal.md) Schnittstellen.

> [!Note]  
> Stellen Sie beim Implementieren eines [direkten Manipulations](direct-manipulation-portal.md) Objekts sicher, dass die [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) -Implementierung das Multithreading durch Thread sichere Verweis Zählung unterstützt. Weitere Informationen finden Sie unter [**interlockedinkrement**](/windows/win32/api/winnt/nf-winnt-interlockedincrement) und [**InterlockedDecrement**](/windows/win32/api/winnt/nf-winnt-interlockeddecrement).
