# <a name="span-idwwan_profile_v2element_displayprovidernamespandisplayprovidername"></a><span id="WWAN_profile_v2.element_DisplayProviderName"></span>Display Provider Name

Das [**displayprovidername**](element-displayprovidername.md) -Element ist ein optionaler [**providernametype**](./schema-providernametype-simpletype.md) , der den Namen der Netzwerkverbindung enthält, der im Windows-Verbindungs-Manager angezeigt werden soll. Dieser Name wird nur angezeigt, wenn sich der Abonnent in einem Heimnetzwerk befindet und kein Roaming erfolgt. Der Name des roamingnetzwerks wird basierend auf Informationen vom mobilen Breitband Gerät angezeigt.

## <a name="element-hierarchy"></a>Elementhierarchie

**<DisplayProviderName>**

## <a name="syntax"></a>Syntax

``` syntax
<DisplayProviderName>

  providerNameType

</DisplayProviderName>
```
