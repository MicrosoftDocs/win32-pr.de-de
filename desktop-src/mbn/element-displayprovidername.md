# <a name="span-idwwan_profile_v2element_displayprovidernamespandisplayprovidername"></a><span id="WWAN_profile_v2.element_DisplayProviderName"></span>DisplayProviderName

Das [**DisplayProviderName-Element**](element-displayprovidername.md) ist ein optionaler [**providerNameType,**](./schema-providernametype-simpletype.md) der den Netzwerkverbindungsnamen enthält, der im Windows Verbindungs-Manager angezeigt werden soll. Dieser Name wird nur angezeigt, wenn sich der Abonnent in einem Heimnetzwerk befindet und nicht roamingt. Der Roamingnetzwerkname wird basierend auf Denkinformationen des mobilen Breitbandgeräts angezeigt.

## <a name="element-hierarchy"></a>Elementhierarchie

**&lt;DisplayProviderName&gt;**

## <a name="syntax"></a>Syntax

``` syntax
<DisplayProviderName>

  providerNameType

</DisplayProviderName>
```
