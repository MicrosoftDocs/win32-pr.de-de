# <a name="span-idwwan_profile_v2element_displayprovidernamespandisplayprovidername"></a><span id="WWAN_profile_v2.element_DisplayProviderName"></span>DisplayProviderName

Das [**DisplayProviderName-Element**](element-displayprovidername.md) ist ein optionaler [**ProviderNameType,**](./schema-providernametype-simpletype.md) der den Netzwerkverbindungsnamen enthält, der in der Datei Windows Verbindungs-Manager. Dieser Name wird nur angezeigt, wenn sich der Abonnent in einem Heimnetzwerk und nicht im Roaming befindet. Der Name des Roamingnetzwerks wird basierend auf Informationen des mobilen Breitbandgeräts angezeigt.

## <a name="element-hierarchy"></a>Elementhierarchie

**&lt;DisplayProviderName&gt;**

## <a name="syntax"></a>Syntax

``` syntax
<DisplayProviderName>

  providerNameType

</DisplayProviderName>
```
