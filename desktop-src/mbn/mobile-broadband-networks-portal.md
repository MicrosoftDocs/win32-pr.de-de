# <a name="mobile-broadband"></a>Mobiles Breitband

## <a name="purpose"></a>Zweck

Die Mobile Breitband-API wird verwendet, um Konnektivität mit Mobilfunknetzen zu implementieren. Anwendungen sollten nicht direkt mit solchen mobilen Breitbandgeräten kommunizieren. Stattdessen müssen sie die mobile Breitband-API verwenden.

## <a name="developer-audience"></a>Entwicklergruppe

Die Mobile Breitband-API ist für C++-Entwickler konzipiert.

Es gibt auch eine Schnittstelle für .NET-Entwickler, die C \# verwenden. Weitere Informationen finden Sie in diesem [Whitepaper.](https://www.microsoft.com/whdc/connect/wireless/MB_ManagedCode.mspx)

Die Mobile Breitband-API kann von Drittanbieteranwendungen verwendet werden, die mobile Breitbandschnittstellen steuern und verwalten müssen.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Die Mobile Breitbandunterstützung beginnt mit Windows 7.
