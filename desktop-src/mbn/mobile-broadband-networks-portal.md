# <a name="mobile-broadband"></a>Mobiles Breitband

## <a name="purpose"></a>Zweck

Die mobile Breitband-API wird zum Implementieren der Konnektivität mit Mobilfunknetzen verwendet. Anwendungen sollten nicht direkt mit solchen mobilen Breitbandgeräten kommunizieren. Stattdessen müssen Sie die mobile Breitband-API verwenden.

## <a name="developer-audience"></a>Entwicklergruppe

Die mobile Breitband-API ist für C++-Entwickler konzipiert.

Es gibt auch eine Schnittstelle für .NET-Entwickler, die C verwenden \# . Weitere Informationen finden Sie in diesem [Whitepaper](https://www.microsoft.com/whdc/connect/wireless/MB_ManagedCode.mspx).

Die mobile Breitband-API kann von Drittanbieter Anwendungen verwendet werden, die mobile Breitband Schnittstellen Steuern und verwalten müssen.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Die Unterstützung mobiler Breitbanddienste beginnt mit Windows 7.
