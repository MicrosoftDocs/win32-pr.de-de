# <a name="description-mbnprofile-element"></a>Description (MBNProfile)-Element

Das **Description (MBNProfile)-Element** enthält eine Beschreibung des Profils.

Es handelt sich um eine Zeichenfolge mit einer Länge von bis zu 255 Zeichen.

``` syntax
<xs:element name="Description"
    type="nameType"
 />
```

Das **Description-Element** wird durch das [**MBNProfile-Element**](schema-mbnprofile-element.md) definiert.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|-------------------------------------|---------------------------------------------------|
| Unterstützte Mindestversion (Client)<br/> | Windows 7 \[ Desktop-Apps \| UWP-Apps\]<br/> |
| Unterstützte Mindestversion (Server)<br/> | Nicht unterstützt<br/>                         |
