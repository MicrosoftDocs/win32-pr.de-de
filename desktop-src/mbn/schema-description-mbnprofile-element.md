# <a name="description-mbnprofile-element"></a>Description-Element (mbnprofile)

Das **Description-Element (mbnprofile)** enthält eine Beschreibung des Profils.

Es handelt sich um eine Zeichenfolge mit einer Länge von bis zu 255 Zeichen.

``` syntax
<xs:element name="Description"
    type="nameType"
 />
```

Das **Description** -Element wird durch das [**mbnprofile**](schema-mbnprofile-element.md) -Element definiert.
