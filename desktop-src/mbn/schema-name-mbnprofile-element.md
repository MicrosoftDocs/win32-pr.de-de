# <a name="name-mbnprofile-element"></a>Name (mbnprofile)-Element

Das **Name-Element (mbnprofile)** identifiziert den Namen des Profils.

Es handelt sich um eine Zeichenfolge mit einer Länge von bis zu 255 Zeichen.

Das-Element ist erforderlich.

``` syntax
<xs:element name="Name"
    type="nameType"
 />
```

Das **Name** -Element wird durch das [**mbnprofile**](schema-mbnprofile-element.md) -Element definiert.

## <a name="requirements"></a>Anforderungen

|-------------------------------------|---------------------------------------------------|
| Unterstützte Mindestversion (Client)<br/> | Windows 7 \[ -Desktop-Apps \| UWP-apps\]<br/> |
| Unterstützte Mindestversion (Server)<br/> | Nicht unterstützt<br/>                         |
