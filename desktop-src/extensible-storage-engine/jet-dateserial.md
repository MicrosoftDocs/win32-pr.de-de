# <a name="jet_dateserial"></a>JET_DATESERIAL


_**Gilt für:** Windows | Windows Server_

## <a name="jet_dateserial"></a>JET_DATESERIAL

Der **JET_DATESERIAL** Datentyp stellt ein Datum in Bruchteilen seit dem Jahr 100 dar.

```cpp
    typedef double JET_DATESERIAL;
```

### <a name="data-types"></a>Datentypen

JET_DATESERIAL

Enthält eine Gleitkommazahl mit doppelter Genauigkeit (8 Byte). Dies entspricht dem [C++-DATE-Typ.](https://msdn.microsoft.com/library/82ab7w69(VS.71).aspx)
