# <a name="jet_dateserial"></a>JET_DATESERIAL


_**Gilt für:** Windows | Windows Server_

## <a name="jet_dateserial"></a>JET_DATESERIAL

Der **JET_DATESERIAL** -Datentyp stellt ein Datum in Bruchteilen der Tage seit dem Jahr 100 dar.

```cpp
    typedef double JET_DATESERIAL;
```

### <a name="data-types"></a>Datentypen

JET_DATESERIAL

Enthält eine Gleit Komma Zahl mit doppelter Genauigkeit (8 Byte). Dies entspricht dem [Date](https://msdn.microsoft.com/library/82ab7w69(VS.71).aspx) -Typ C++.
