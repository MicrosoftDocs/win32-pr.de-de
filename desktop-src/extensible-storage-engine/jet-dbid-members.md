# <a name="jet_dbid-members"></a>JET_DBID-Member

Einschließen geschützter Member  
Einschließen geerbter Member  

Ein JET_DBID enthält das Handle für die Datenbank. Ein Datenbankhandle wird verwendet, um das Schema einer Datenbank zu verwalten. Sie kann auch zum Verwalten der Tabellen innerhalb dieser Datenbank verwendet werden.

Der [JET_DBID-Typ](./jet-dbid-structure.md) macht die folgenden Member verfügbar.
