# <a name="jet_objtyp"></a>JET_OBJTYP


_**Gilt für:** Windows | Windows Server_

## <a name="jet_objtyp"></a>JET_OBJTYP

Die **JET_OBJTYP** Gruppe von Konstanten stellt den Typ eines Datenbankobjekts dar. Derzeit werden nur Tabellen unterstützt.


| <p>Konstante/Wert</p> | <p>BESCHREIBUNG</p> | 
|-----------------------|--------------------|
| <p>JET_objtypNil<br />0</p> | <p>Stellt alle Objekttypen dar. Derzeit werden nur Tabellen unterstützt.</p> | 
| <p>JET_objtypTable<br />1</p> | <p>Stellt eine Tabelle dar.</p> |
