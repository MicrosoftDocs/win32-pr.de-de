# <a name="apitrymoveprevious-method"></a>API. trymuveprevious-Methode

Versuchen Sie, zum vorherigen Datensatz in der Tabelle zu wechseln. Wenn kein vorheriger Datensatz vorhanden ist, wird false zurückgegeben. Wenn ein anderer Fehler auftritt, wird eine Ausnahme ausgelöst.
