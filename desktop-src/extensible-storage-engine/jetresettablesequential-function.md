# <a name="jetresettablesequential-function"></a>Jetresettablesequential-Funktion


_**Gilt für:** Windows | Windows Server_

## <a name="jetresettablesequential-function"></a>Jetresettablesequential-Funktion

Die **jetresettablesequential** -Funktion benachrichtigt die Datenbank-Engine, dass die Anwendung den gesamten aktuellen Index, der einen bestimmten Cursor enthält, nicht mehr scannt. Dieser Rückruf kehrt eine von [jetsettablesequential](./jetsettablesequential-function.md)gesendete Benachrichtigung um.

**Windows XP:**   **jetresettablesequential** wird in Windows XP eingeführt.

```cpp
    JET_ERR JET_API JetResetTableSequential(
      __in          JET_SESID sesid,
      __in          JET_TABLEID tableid,
      __in          JET_GRBIT grbit
    );
```

### <a name="parameters"></a>Parameter

*-sid*

Die Sitzung, die für diesen-Befehl verwendet werden soll.

*TableID*

Der Cursor, der für diesen-Befehl verwendet werden soll.

*grbit*

Für die zukünftige Verwendung reserviert.

### <a name="return-value"></a>Rückgabewert

Bei Erfolg ist der aktuelle Index des Cursors nicht mehr für einen sequenziellen Scan des gesamten Indexes optimiert. Es erfolgt keine Änderung des Daten Bank Status.

Bei einem Fehler tritt keine Änderung an der Konfiguration des Cursors auf. Es erfolgt keine Änderung des Daten Bank Status.

#### <a name="remarks"></a>Bemerkungen

Es ist sicher, diesen Vorgang für einen Cursor durchführen zu lassen, der zuvor nicht durch einen-Befehl von [jetsettablesequenziell](./jetsettablesequential-function.md)konfiguriert wurde.
