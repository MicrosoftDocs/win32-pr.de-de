# <a name="apijetgetrecordposition-method"></a>API. jetgetrecordposition-Methode

Gibt die Bruch Position des aktuellen Datensatzes im aktuellen Index in Form einer [JET_RECPOS](./jet-recpos-class.md) Struktur zurück. Siehe auch [jetgotoposition (JET_SESID, JET_TABLEID, JET_RECPOS)](./api.jetgotoposition-method.md).

**Namespace:**  [Microsoft. ISAM. ESENT. Interop](./microsoft.isam.esent.interop-namespace.md)  
**Assembly:**  Microsoft. ISAM. ESENT. Interop (in Microsoft.Isam.Esent.Interop.dll)

    sesid As JET_SESID, _

#### <a name="parameters"></a>Parameter

  - -sid  
    Typ: [Microsoft.ISAM.ESENT.Interop.JET_SESID](./jet-sesid-structure.md)  
    
    Die zu verwendende Sitzung.

<!-- end list -->

  - TableID  
    Typ: [Microsoft.ISAM.ESENT.Interop.JET_TABLEID](./jet-tableid-structure.md)  
    
    Der Cursor, der auf dem Datensatz positioniert ist.

<!-- end list -->

  - recpos  
    Typ: [Microsoft.ISAM.ESENT.Interop.JET_RECPOS](./jet-recpos-class.md)  
    
    Gibt die ungefähre Position des Datensatzes zurück.
