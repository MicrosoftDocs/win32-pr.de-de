# <a name="apijetresettablesequential-method"></a>API. jetresettablesequential-Methode

Benachrichtigt die Datenbank-Engine, dass die Anwendung nicht mehr den gesamten Index scannt, auf dem sich der Cursor befindet. Dieser Rückruf kehrt eine Benachrichtigung um, die von [jetsettablesequential (JET_SESID, JET_TABLEID, settablesequentialgrbit)](./api.jetsettablesequential-method.md)gesendet wird.

**Namespace:**  [Microsoft. ISAM. ESENT. Interop](./microsoft.isam.esent.interop-namespace.md)
