# <a name="jet_pstr"></a>JET_PSTR


_**Gilt für:** Windows | Windows Server_

## <a name="jet_pstr"></a>JET_PSTR

Der JET_PSTR Datentyp enthält eine AUF NULL terminte ASCII-Zeichenfolge (char \* ).

**Windows Vista: JET_PSTR** wird in Windows Vista eingeführt.

```cpp
    typedef __nullterminated char *  JET_PSTR;
```

### <a name="data-types"></a>Datentypen

JET_PSTR

NULL-terminiert, ASCII-Zeichenfolge (char \* ).
