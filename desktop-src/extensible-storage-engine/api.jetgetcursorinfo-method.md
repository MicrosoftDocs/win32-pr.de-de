# <a name="apijetgetcursorinfo-method"></a>API. jetgetcurrsorinfo-Methode

Bestimmen, ob ein Update des aktuellen Datensatzes eines Cursors zu einem Schreib Konflikt führt, basierend auf dem aktuellen Update Status des Datensatzes. Es ist möglich, dass ein Schreib Konflikt letztendlich zurückgegeben wird, auch wenn jetgetcurrsorinfo erfolgreich zurückgegeben wurde. Da eine andere Sitzung den Datensatz aktualisieren kann, bevor die aktuelle Sitzung denselben Datensatz aktualisieren kann.
