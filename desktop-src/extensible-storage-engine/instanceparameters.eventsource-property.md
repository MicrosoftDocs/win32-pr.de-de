# <a name="instanceparameterseventsource-property"></a>InstanceParameters.EventSource -Eigenschaft

Ruft eine anwendungsspezifische Zeichenfolge ab, die allen Ereignisprotokollmeldungen hinzugefügt wird, die von der Datenbank-Engine ausgegeben werden, oder legt diese fest. Dies ermöglicht eine einfache Korrelation von Ereignisprotokollmeldungen mit der Quellanwendung. Standardmäßig wird der Name der ausführbaren Hostanwendung verwendet.

**Namespace:**  [Microsoft.Isam.Esent.Interop](./microsoft.isam.esent.interop-namespace.md)
