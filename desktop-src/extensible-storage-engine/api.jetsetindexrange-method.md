# <a name="apijetsetindexrange-method"></a>Api.JetSetIndexRange-Methode

Begrenzt vorübergehend den Satz von Indexeinträgen, die der Cursor mit [JetMove(JET_SESID, JET_TABLEID, Int32, MoveGrbit)](./api.jetmove-method-jet-sesid-jet-tableid-int32-movegrbit-.md) durchgehen kann, auf diejenigen, die vom aktuellen Indexeintrag beginnen und am Indexeintrag enden, der mit den Suchkriterien übereinstimmt, die vom Suchschlüssel in diesem Cursor und den angegebenen gebundenen Kriterien angegeben werden. Ein Suchschlüssel muss zuvor mit [JetMakeKey(JET_SESID, JET_TABLEID, , \[ \] Int32, MakeKeyGrbit)](./api.jetmakekey-method.md)erstellt worden sein. Siehe auch [TrySetIndexRange(JET_SESID, JET_TABLEID, SetIndexRangeGrbit).](./api.trysetindexrange-method.md)
