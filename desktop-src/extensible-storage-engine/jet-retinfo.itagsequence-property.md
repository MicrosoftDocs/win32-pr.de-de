# <a name="jet_retinfoitagsequence-property"></a>JET_RETINFO. itagsequence (Eigenschaft)

Ruft die Sequenznummer des Werts in einer mehrwertigen Spalte ab oder legt Sie fest. Das Array von Werten ist 1-basiert. Der erste Wert ist Sequenz 1, nicht 0. Wenn die Daten Satz Spalte nur über einen Wert verfügt, sollte 1 als itagsequence übergeben werden.
