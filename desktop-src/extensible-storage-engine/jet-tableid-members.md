# <a name="jet_tableid-members"></a>JET_TABLEID-Member

Einschließen geschützter Member  
Einschließen geerbter Member  

Ein JET_TABLEID enthält ein Handle für den Datenbankcursor, das für einen Aufruf der JET-API verwendet werden soll. Ein Cursor kann nur mit der Sitzung verwendet werden, die zum Öffnen dieses Cursors verwendet wurde.

Der [JET_TABLEID-Typ](./jet-tableid-structure.md) macht die folgenden Member verfügbar.
