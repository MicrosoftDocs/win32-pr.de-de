# <a name="jet_pcstr"></a>JET_PCSTR


_**Gilt für:** Windows | Windows Server_

## <a name="jet_pcstr"></a>JET_PCSTR

Der **JET_PCSTR** -Datentyp enthält eine NULL-terminierte, Konstante **ASCII** -Zeichenfolge (Char \* ).

**Windows Vista: JET_PCSTR** wird in Windows Vista eingeführt.

```cpp
    typedef __nullterminated const char *  JET_PCSTR;
```

### <a name="data-types"></a>Datentypen

JET_PCSTR

NULL-terminierte, Konstante ASCII-Zeichenfolge (Char \* ).
