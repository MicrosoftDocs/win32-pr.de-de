# <a name="understanding-markup-compiler-messages"></a>Grundlegendes zu Markupcompilermeldungen

Der Windows Menübandframework-Markupcompiler UI Command Compiler (UICC.exe) überprüft das Menübandmarkup sowohl anhand des Menübandschemas als auch anhand zusätzlicher Regeln, die vom Menübandframework definiert werden.

-   [Validierungsmeldungen](#validation-messages)
    -   [Überprüfungsfehler](#understanding-markup-compiler-messages)
    -   [Validierungswarnungen](#validation-warnings)
    -   [Information Messages](#information-messages)
-   [Zugehörige Themen](#related-topics)

## <a name="validation-messages"></a>Validierungsmeldungen

Die folgenden Validierungsmeldungen werden während der Kompilierung generiert. Weitere [Informationen finden Sie unter Kompilieren von Menübandmarkup.](windowsribbon-intentcl.md)

> [!Note]  
> Die `/W{0\1\2}` Befehlszeilenoption kann angegeben werden, um Ereignismeldungen basierend auf dem Schweregrad zu filtern.

 

### <a name="validation-errors"></a>Überprüfungsfehler

**Fehlermeldungen** weisen darauf hin, dass Probleme im Menübandmarkup schwerwiegend genug sind, damit der Compiler den Kompilierungsprozess abbrechen kann. Diese Probleme müssen behoben werden, damit die Kompilierung erfolgreich ist.
