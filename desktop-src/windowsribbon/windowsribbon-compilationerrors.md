# <a name="understanding-markup-compiler-messages"></a>Grundlegendes zu Markup Compiler-Meldungen

Der Markup Compiler des Windows-Menüband-Frameworks (Menüband), der UI-Befehls Compiler (UICC.exe), überprüft das Menüband-Markup sowohl mit dem Menüband-Schema als auch mit einem zusätzlichen Regelsatz, der durch das Menüband

## <a name="validation-messages"></a>Validierungs Meldungen

Die folgenden Validierungs Meldungen werden während der Kompilierung generiert. Weitere Informationen finden Sie unter Kompilieren von Menü [Band Markup](windowsribbon-intentcl.md).

> [!Note]  
> Die `/W{0\1\2}` Befehlszeilenoption kann angegeben werden, um Ereignismeldungen basierend auf dem Schweregrad zu filtern.

 

### <a name="validation-errors"></a>Überprüfungsfehler

**Fehler** Ereignismeldungen weisen auf Probleme im Menüband-Markup hin, das schwer genug ist, damit der Compiler den Kompilierungs Vorgang abbricht. Diese Probleme müssen behoben werden, damit die Kompilierung erfolgreich ausgeführt werden konnte.
