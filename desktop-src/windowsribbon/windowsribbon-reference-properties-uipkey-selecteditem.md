# <a name="ui_pkey_selecteditem"></a>UI \_ PKEY \_ SelectedItem

Identifiziert die \_ PKEY \_ SelectedItem-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_SelectedItem
   shellPKey = UI_PKEY_SelectedItem
   formatID = 00000104-7363-696e-8441798acf5aebb7
   propID = 104
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ SelectedItem wird von einer Anwendung verwendet, um das ausgewählte Element in einem Katalogsteuerelement abzufragen.

Der -Eigenschaftswert ist der Index des ausgewählten Elements in einer Auflistung.
