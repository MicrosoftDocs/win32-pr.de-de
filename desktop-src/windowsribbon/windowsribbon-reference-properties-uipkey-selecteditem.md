# <a name="ui_pkey_selecteditem"></a>UI \_ pkey \_ SelectedItem

Bezeichnet die Benutzeroberflächen- \_ pkey- \_ SelectedItem-Eigenschaft.

```
propertyDescription
   name = UI_PKEY_SelectedItem
   shellPKey = UI_PKEY_SelectedItem
   formatID = 00000104-7363-696e-8441798acf5aebb7
   propID = 104
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Bemerkungen

UI \_ pkey \_ SelectedItem wird von einer Anwendung verwendet, um das ausgewählte Element in einem Katalog-Steuerelement abzufragen.

Der Eigenschafts Wert ist der Index des ausgewählten Elements in einer Auflistung.
