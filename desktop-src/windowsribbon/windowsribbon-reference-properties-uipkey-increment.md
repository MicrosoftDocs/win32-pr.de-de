# <a name="ui_pkey_increment"></a>\_PKEY-Inkrement der Benutzeroberfläche \_

Identifiziert die \_ PKEY \_ Increment-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_Increment
   shellPKey = UI_PKEY_Increment
   formatID = 00000205-7363-696e-8441798acf5aebb7
   propID = 205
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Hinweise

Das \_ PKEY-Inkrement der Benutzeroberfläche wird von einer Anwendung \_ verwendet, um abzufragen, wie stark der Wert im Bearbeitungsfeld des [Spinner-Steuerelements](windowsribbon-controls-spinner.md) erhöht oder verringert wird, wenn auf die Schaltfläche zum Erhöhen oder Verringern geklickt wird.
