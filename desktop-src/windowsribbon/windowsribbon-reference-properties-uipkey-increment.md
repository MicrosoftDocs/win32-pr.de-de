# <a name="ui_pkey_increment"></a>UI- \_ pkey- \_ Inkrement

Bezeichnet die \_ pkey- \_ Inkrement-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_Increment
   shellPKey = UI_PKEY_Increment
   formatID = 00000205-7363-696e-8441798acf5aebb7
   propID = 205
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Bemerkungen

Die UI- \_ pkey- \_ Inkrement wird von einer Anwendung verwendet, um abzufragen, wie stark der Wert im Bearbeitungsfeld des [Spinner](windowsribbon-controls-spinner.md) -Steuer Elements vergrößert oder verringert wird, wenn auf die Schaltfläche Inkrement oder Dekrement geklickt wird.
