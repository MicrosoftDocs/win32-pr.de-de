# <a name="ui_pkey_minvalue"></a>UI- \_ pkey- \_ MinValue

Identifiziert die Benutzeroberflächen- \_ pkey- \_ MinValue-Eigenschaft.

```
propertyDescription
   name = UI_PKEY_MinValue
   shellPKey = UI_PKEY_MinValue
   formatID = 00000204-7363-696e-8441798acf5aebb7
   propID = 204
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Bemerkungen

Der UI- \_ pkey- \_ MinValue wird von einer Anwendung verwendet, um den minimalen Wert des [Spinner](windowsribbon-controls-spinner.md) -Steuer Elements abzufragen.
