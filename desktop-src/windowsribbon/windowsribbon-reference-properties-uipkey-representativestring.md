# <a name="ui_pkey_representativestring"></a>UI \_ PKEY \_ RepresentativeString

Identifiziert die \_ PKEY \_ RepresentativeString-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_RepresentativeString
   shellPKey = UI_PKEY_RepresentativeString
   formatID = 00000208-7363-696e-8441798acf5aebb7
   propID = 208
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ RepresentativeString wird von einer Anwendung verwendet, um die Länge des Bearbeitungsfelds für ein [Spinner-Steuerelement](windowsribbon-controls-spinner.md) abzufragen.

> [!Note]  
> Der Wert von UI \_ PKEY \_ RepresentativeString wird vom Framework verwendet, um die Breite des [Spinner-Steuerelements](windowsribbon-controls-spinner.md) zu berechnen, und wird nie angezeigt.
