# <a name="ui_pkey_representativestring"></a>UI \_ pkey \_ representativestring

Identifiziert die Benutzeroberflächen- \_ pkey \_ representativestring-Eigenschaft.

```
propertyDescription
   name = UI_PKEY_RepresentativeString
   shellPKey = UI_PKEY_RepresentativeString
   formatID = 00000208-7363-696e-8441798acf5aebb7
   propID = 208
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Bemerkungen

Der Benutzeroberflächen- \_ pkey \_ representativestring wird von einer Anwendung verwendet, um die Länge des Bearbeitungs Felds für ein [Spinner](windowsribbon-controls-spinner.md) -Steuerelement abzufragen.

> [!Note]  
> Der Wert von UI \_ pkey \_ representativestring wird vom Framework verwendet, um die Breite des [Spinner](windowsribbon-controls-spinner.md) -Steuer Elements zu berechnen, und wird nie angezeigt.
