# <a name="ui_pkey_maxvalue"></a>MaxValue der UI- \_ pkey \_

Bezeichnet die Eigenschaft "MaxValue" der UI- \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_MaxValue
   shellPKey = UI_PKEY_MaxValue
   formatID = 00000203-7363-696e-8441798acf5aebb7
   propID = 203
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Bemerkungen

Der Benutzeroberflächen- \_ pkey- \_ MaxValue wird von einer Anwendung verwendet, um den maximalen Wert des [Spinner](windowsribbon-controls-spinner.md) -Steuer Elements abzufragen.
