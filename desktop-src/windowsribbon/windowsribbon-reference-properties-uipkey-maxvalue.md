# <a name="ui_pkey_maxvalue"></a>UI \_ PKEY \_ MaxValue

Identifiziert die \_ PKEY \_ MaxValue-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_MaxValue
   shellPKey = UI_PKEY_MaxValue
   formatID = 00000203-7363-696e-8441798acf5aebb7
   propID = 203
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ MaxValue wird von einer Anwendung verwendet, um den maximalen Wert des [Spinner-Steuerelements](windowsribbon-controls-spinner.md) abzufragen.
