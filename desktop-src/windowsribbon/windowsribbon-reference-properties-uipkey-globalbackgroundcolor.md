# <a name="ui_pkey_globalbackgroundcolor"></a>UI \_ pkey \_ globalbackgroundcolor

Identifiziert die UI \_ pkey \_ globalbackgroundcolor-Eigenschaft.

```
propertyDescription
   name = UI_PKEY_GlobalBackgroundColor
   shellPKey = UI_PKEY_GlobalBackgroundColor
   formatID = 00002000-7363-696e-8441798acf5aebb7
   propID = 2000
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Bemerkungen

UI \_ pkey \_ globalbackgroundcolor wird von einer Anwendung verwendet, um die globale Hintergrundfarbe für eine angepasste Multifunktionsleiste abzufragen.

**Eingeführt in Windows 8:** [UI \_ pkey \_ globalhighlightcolor](windowsribbon-reference-properties-uipkey-globalhighlightcolor.md) kann nicht unabhängig von UI \_ pkey \_ globalbackgroundcolor festgelegt werden.
