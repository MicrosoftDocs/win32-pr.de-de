# <a name="ui_pkey_standardcolorstooltips"></a>UI \_ PKEY \_ StandardColorsTooltips

Identifiziert die \_ PKEY \_ StandardColorsTooltips-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_StandardColorsTooltips
   shellPKey = UI_PKEY_StandardColorsTooltips
   formatID = 00000412-7363-696e-8441798acf5aebb7
   propID = 412
   typeInfo
      type = VT_VECTOR | VT_LPWSTR
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ StandardColorsTooltips wird von einer Anwendung verwendet, um die QuickInfos zur Farbwatch eines [**DropDownColorPicker abzufragen.**](windowsribbon-element-dropdowncolorpicker.md)

Der Eigenschaftswert ist ein Array von Zeichenfolgenwerten.

Jeder Zeichenfolgenwert entspricht der QuickInfo für eine Farbuhr in einem [**DropDownColorPicker,**](windowsribbon-element-dropdowncolorpicker.md) unabhängig vom für das **ColorTemplate-Attribut** angegebenen Wert.
