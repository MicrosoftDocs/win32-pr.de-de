# <a name="ui_pkey_decimalplaces"></a>UI \_ pkey \_ DecimalPlaces

Identifiziert die UI \_ pkey \_ DecimalPlaces-Eigenschaft.

```
propertyDescription
   name = UI_PKEY_DecimalPlaces
   shellPKey = UI_PKEY_DecimalPlaces
   formatID = 00000206-7363-696e-8441798acf5aebb7
   propID = 206
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Bemerkungen

UI \_ pkey \_ DecimalPlaces wird von einer Anwendung verwendet, um die Anzahl der Ziffern abzufragen, die nach dem Dezimaltrennzeichen im Bearbeitungsfeld des [Spinner](windowsribbon-controls-spinner.md) -Steuer Elements angezeigt werden.
