# <a name="ui_pkey_decimalplaces"></a>UI \_ PKEY \_ DecimalPlaces

Identifiziert die \_ PKEY \_ DecimalPlaces-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_DecimalPlaces
   shellPKey = UI_PKEY_DecimalPlaces
   formatID = 00000206-7363-696e-8441798acf5aebb7
   propID = 206
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Hinweise

Ui PKEY DecimalPlaces wird von einer Anwendung verwendet, um die Anzahl der Ziffern nach dem Dezimaltrennzeichen im Bearbeitungsfeld des \_ \_ [Spinner-Steuerelements abfragt.](windowsribbon-controls-spinner.md)
