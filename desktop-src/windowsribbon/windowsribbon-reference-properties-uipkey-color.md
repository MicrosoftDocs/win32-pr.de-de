# <a name="ui_pkey_color"></a>UI- \_ pkey- \_ Farbe

Bezeichnet die Eigenschaft der Eigenschaft "pkey" der UI \_ \_ .

```
propertyDescription
   name = UI_PKEY_Color
   shellPKey = UI_PKEY_Color
   formatID = 00000400-7363-696e-8441798acf5aebb7
   propID = 400
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Bemerkungen

Die UI- \_ pkey- \_ Farbe wird von einer Anwendung verwendet, um den Farbwert des [**dropdowncolorpicker**](windowsribbon-element-dropdowncolorpicker.md) -Steuer Elements abzufragen.

Der Eigenschafts Wert ist ein [COLORREF](../gdi/colorref.md) -Wert.

Das höchst wertige Byte des [COLORREF](../gdi/colorref.md) -Werts wird ignoriert.
