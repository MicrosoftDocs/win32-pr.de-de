# <a name="others"></a>Sonstige

Referenzdokumentation zu zusätzlichen Markupelementen im Windows Menübandframework.



| Thema                                                                              | Inhalte                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Anwendung**](windowsribbon-element-application.md)                           | Stellt das Element der obersten Ebene in der Markupspezifikation des Menübandframeworks dar.<br/>                                                                                                                      |
| [**Application.Commands**](windowsribbon-element-application-commands.md)         | Stellt einen Container für alle Befehle dar, die von der Anwendung definiert werden.<br/>                                                                                                                    |
| [**Application.Views**](windowsribbon-element-application-views.md)               | Stellt einen Container für jede Menübandframeworkansicht dar, insbesondere [**das Menüband**](windowsribbon-element-ribbon.md) und [**den ContextPopup.**](windowsribbon-element-contextpopup.md)<br/>          |
| [**ColumnBreak**](windowsribbon-element-columnbreak.md)                           | Stellt ein vertikales Trennzeichen (sichtbar oder ausgeblendet) in benutzerdefinierten [**SizeDefinition-Layoutvorlagen**](windowsribbon-element-sizedefinition.md) dar.<br/>                                                      |
| [**ControlGroup**](windowsribbon-element-controlgroup.md)                         | Stellt eine Gruppe von Steuerelementen in einer [**SizeDefinition-Layoutvorlage**](windowsribbon-element-sizedefinition.md) dar.<br/>                                                                                 |
| [**ControlNameDefinition**](windowsribbon-element-controlnamedefinition.md)       | Stellt einen Namen eines Steuerelements in einer benutzerdefinierten [**SizeDefinition-Layoutvorlage**](windowsribbon-element-sizedefinition.md) dar.<br/>                                                                          |
| [**ControlNameMap**](windowsribbon-element-controlnamemap.md)                     | Stellt einen Container für Steuerelementnamen in einer benutzerdefinierten [**SizeDefinition-Layoutvorlage**](windowsribbon-element-sizedefinition.md) dar.<br/>                                                                |
| [**ControlSizeDefinition**](windowsribbon-element-controlsizedefinition.md)       | Stellt den Layoutstil einer Gruppe von Steuerelementen in einer benutzerdefinierten Vorlage dar.<br/>                                                                                                                            |
| [**FlowMenuLayout**](windowsribbon-element-flowmenulayout.md)                     | Stellt ein horizontales Layout mit Zeilenumbrüchen für Elemente in einem Katalog dar.<br/>                                                                                                                             |
| [**GroupSizeDefinition**](windowsribbon-element-groupsizedefinition.md)           | Stellt eine Layoutgröße für eine Gruppe von Steuerelementen in einer benutzerdefinierten Vorlage dar.<br/>                                                                                                                              |
| [**Zeile**](windowsribbon-element-row.md)                                           | Stellt eine Zeile von Steuerelementen in einer benutzerdefinierten SizeDefinition-Layoutvorlage dar.<br/>                                                                                                                            |
| [**Skalieren**](windowsribbon-element-scale.md)                                       | Stellt die Größe und Layoutpräferenz einer [**Gruppe**](windowsribbon-element-group.md) von Steuerelementen über ein Paar {**Group**, [**SizeDefinition**](windowsribbon-element-sizedefinition.md)} dar.<br/> |
| [**ScalingPolicy**](windowsribbon-element-scalingpolicy.md)                       | Stellt einen Container für skalierungsspezifikationen dar.<br/>                                                                                                                                                  |
| [**ScalingPolicy.IdealSizes**](windowsribbon-element-scalingpolicy-idealsizes.md) | Stellt einen Container mit Skalierungsspezifikationen für die bevorzugte [**SizeDefinition-Vorlage**](windowsribbon-element-sizedefinition.md) basierend auf der Menübandgröße dar.<br/>                                   |
| [**SizeDefinition**](windowsribbon-element-sizedefinition.md)                     | Stellt eine benutzerdefinierte Layoutvorlage von Menüband-Steuerelementen dar. <br/>                                                                                                                                            |
| [**VerticalMenuLayout**](windowsribbon-element-verticalmenulayout.md)             | Stellt ein vertikales Layout für Elemente in einem Katalog dar.<br/>                                                                                                                                                |
