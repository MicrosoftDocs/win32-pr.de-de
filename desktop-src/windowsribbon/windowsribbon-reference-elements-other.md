# <a name="others"></a>Sonstige

Referenz Dokumentation für zusätzliche Markup Elemente im Windows-Menü Band Framework.



| Thema                                                                              | Inhalte                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Application**](windowsribbon-element-application.md)                           | Stellt das Element der obersten Ebene in der Markup Spezifikation des Multifunktionsleisten-Frameworks dar.<br/>                                                                                                                      |
| [**Application. Commands**](windowsribbon-element-application-commands.md)         | Stellt einen Container für alle Befehle dar, die von der Anwendung definiert werden.<br/>                                                                                                                    |
| [**Application. views**](windowsribbon-element-application-views.md)               | Stellt einen Container für jede Menüband-frameworkansicht dar, insbesondere das [**Menüband**](windowsribbon-element-ribbon.md) und das [**contextpopup**](windowsribbon-element-contextpopup.md).<br/>          |
| [**Columnbreak**](windowsribbon-element-columnbreak.md)                           | Stellt ein vertikales Trennzeichen (sichtbar oder ausgeblendet) in benutzerdefinierten [**sizedefinition**](windowsribbon-element-sizedefinition.md) -Layoutvorlagen dar.<br/>                                                      |
| [**Controlgroup**](windowsribbon-element-controlgroup.md)                         | Stellt eine Gruppe von Steuerelementen in einer [**sizedefinition**](windowsribbon-element-sizedefinition.md) -Layoutvorlage dar.<br/>                                                                                 |
| [**Controlnamedefinition**](windowsribbon-element-controlnamedefinition.md)       | Stellt den Namen eines Steuer Elements in einer benutzerdefinierten [**sizedefinition**](windowsribbon-element-sizedefinition.md) -Layoutvorlage dar.<br/>                                                                          |
| [**Controlnamemap**](windowsribbon-element-controlnamemap.md)                     | Stellt einen Container für Steuerelement Namen in einer benutzerdefinierten [**sizedefinition**](windowsribbon-element-sizedefinition.md) -Layoutvorlage dar.<br/>                                                                |
| [**Controlsizedefinition**](windowsribbon-element-controlsizedefinition.md)       | Stellt den Layoutstil einer Gruppe von Steuerelementen in einer benutzerdefinierten Vorlage dar.<br/>                                                                                                                            |
| [**Flowmenulayout**](windowsribbon-element-flowmenulayout.md)                     | Stellt ein horizontales Layout mit Zeilenumbrüchen für Elemente in einem Katalog dar.<br/>                                                                                                                             |
| [**Groupsizedefinition**](windowsribbon-element-groupsizedefinition.md)           | Stellt eine Layoutgröße für eine Gruppe von Steuerelementen in einer benutzerdefinierten Vorlage dar.<br/>                                                                                                                              |
| [**Zeile**](windowsribbon-element-row.md)                                           | Stellt eine Zeile von Steuerelementen in einer benutzerdefinierten sizedefinition-Layoutvorlage dar.<br/>                                                                                                                            |
| [**Migen**](windowsribbon-element-scale.md)                                       | Stellt die Größe und die Layouteinstellung einer [**Gruppe**](windowsribbon-element-group.md) von Steuerelementen durch ein {**Group**, [**sizedefinition**](windowsribbon-element-sizedefinition.md)}-Paar dar.<br/> |
| [**ScalingPolicy**](windowsribbon-element-scalingpolicy.md)                       | Stellt einen Container für die Skalierung von Spezifikationen dar.<br/>                                                                                                                                                  |
| [**Scalingpolicy. ideal sizes**](windowsribbon-element-scalingpolicy-idealsizes.md) | Stellt einen Container mit Skalierungs Spezifikationen für die bevorzugte [**sizedefinition**](windowsribbon-element-sizedefinition.md) -Vorlage basierend auf der Menü Band Größe dar.<br/>                                   |
| [**Sizedefinition**](windowsribbon-element-sizedefinition.md)                     | Stellt eine benutzerdefinierte Layoutvorlage von Menü Band Steuerelementen dar. <br/>                                                                                                                                            |
| [**Verticalmenulayout**](windowsribbon-element-verticalmenulayout.md)             | Stellt ein vertikales Layout für Elemente in einem Katalog dar.<br/>                                                                                                                                                |
