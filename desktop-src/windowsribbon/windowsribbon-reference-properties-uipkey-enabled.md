# <a name="ui_pkey_enabled"></a>UI- \_ pkey \_ aktiviert

Bezeichnet die \_ Eigenschaft "aktivierte UI-pkey" \_ .

```
propertyDescription
   name = UI_PKEY_Enabled
   shellPKey = UI_PKEY_Enabled
   formatID = 00000004-7363-696e-8441798acf5aebb7
   propID = 1
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Bemerkungen

\_ \_ Der aktivierte UI-pkey wird von einer Anwendung verwendet, um abzufragen, ob ein Steuerelement aktiviert ist und auf die Benutzerinteraktion reagieren kann.

> [!Note]  
> Der Wert wird nicht automatisch auf 0 festgelegt, wenn der Anwendungsmodus geändert wird und die Sichtbarkeit eines Befehls deaktiviert ist.
