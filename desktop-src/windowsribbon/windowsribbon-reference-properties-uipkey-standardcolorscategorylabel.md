# <a name="ui_pkey_standardcolorscategorylabel"></a>UI \_ PKEY \_ StandardColorsCategoryLabel

Identifiziert die \_ PKEY \_ StandardColorsCategoryLabel-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_StandardColorsCategoryLabel
   shellPKey = UI_PKEY_StandardColorsCategoryLabel
   formatID = 00000404-7363-696e-8441798acf5aebb7
   propID = 404
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ StandardColorsCategoryLabel wird von einer Anwendung verwendet, um den Wert der Bezeichnung abzufragen, die der Kategorie **Standardfarben** eines [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md)zugeordnet ist.

Ui \_ PKEY \_ StandardColorsCategoryLabel ist nur für [**dropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) gültig, wobei `ThemeColors` als Wert des **ColorTemplate-Attributs** angegeben wird (dies ist die einzige Vorlage, die bezeichnete Kategorien enthält).

Der folgende Screenshot zeigt einen `ThemeColors` [**DropDownColorPicker.**](windowsribbon-element-dropdowncolorpicker.md)
