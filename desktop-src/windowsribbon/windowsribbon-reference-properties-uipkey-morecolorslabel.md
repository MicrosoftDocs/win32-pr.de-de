# <a name="ui_pkey_morecolorslabel"></a>UI \_ PKEY \_ MoreColorsLabel

Identifiziert die \_ PKEY \_ MoreColorsLabel-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_MoreColorsLabel
   shellPKey = UI_PKEY_MoreColorsLabel
   formatID = 00000408-7363-696e-8441798acf5aebb7
   propID = 408
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Hinweise

Ui PKEY MoreColorsLabel wird von einer Anwendung verwendet, um den Wert der Bezeichnung abfragt, die der Schaltfläche Weitere Farben eines \_ \_ [**DropDownColorPicker zugeordnet ist.**](windowsribbon-element-dropdowncolorpicker.md) 

Ui PKEY MoreColorsLabel ist nur für dropDownColorPicker gültig, wobei oder als Wert des \_ \_ [](windowsribbon-element-dropdowncolorpicker.md) `ThemeColors` `StandardColors` **ColorTemplate-Attributs angegeben** wird.
