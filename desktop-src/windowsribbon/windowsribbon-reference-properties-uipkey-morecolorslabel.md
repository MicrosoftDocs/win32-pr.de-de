# <a name="ui_pkey_morecolorslabel"></a>UI \_ pkey \_ morecolorslabel

Identifiziert die Benutzeroberflächen- \_ pkey- \_ Eigenschaft "morecolorslabel".

```
propertyDescription
   name = UI_PKEY_MoreColorsLabel
   shellPKey = UI_PKEY_MoreColorsLabel
   formatID = 00000408-7363-696e-8441798acf5aebb7
   propID = 408
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Bemerkungen

"UI \_ pkey \_ morecolorslabel" wird von einer Anwendung verwendet, um den Wert der Bezeichnung abzufragen, die mit der Schaltfläche " **Weitere Farben** " eines [**dropdowncolorpicker**](windowsribbon-element-dropdowncolorpicker.md)verknüpft ist.

\_Die UI pkey \_ morecolorslabel ist nur für eine [**dropdowncolorpicker**](windowsribbon-element-dropdowncolorpicker.md) gültig, bei der `ThemeColors` oder `StandardColors` als Wert des **colortemplate** -Attributs angegeben wird.
