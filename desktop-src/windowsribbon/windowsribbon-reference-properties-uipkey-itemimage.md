# <a name="ui_pkey_itemimage"></a>UI \_ pkey \_ itemImage

Identifiziert die Benutzeroberflächen- \_ pkey \_ itemImage-Eigenschaft.

```
propertyDescription
   name = UI_PKEY_ItemImage
   shellPKey = UI_PKEY_ItemImage
   formatID = 00000106-7363-696e-8441798acf5aebb7
   propID = 106
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Bemerkungen

UI \_ pkey \_ itemImage wird von einer Anwendung verwendet, um das Bild abzufragen, das einem Element oder Befehl in einem Katalog-Steuerelement zugeordnet ist.

Der Eigenschafts Wert ist ein [**iuiimage**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage) -Objekt.
