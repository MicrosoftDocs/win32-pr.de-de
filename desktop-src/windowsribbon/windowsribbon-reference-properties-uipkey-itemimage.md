# <a name="ui_pkey_itemimage"></a>UI \_ PKEY \_ ItemImage

Identifiziert die \_ PKEY \_ ItemImage-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_ItemImage
   shellPKey = UI_PKEY_ItemImage
   formatID = 00000106-7363-696e-8441798acf5aebb7
   propID = 106
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ ItemImage wird von einer Anwendung verwendet, um das Image abzufragen, das einem Element oder Befehl in einem Katalogsteuerelement zugeordnet ist.

Der Eigenschaftswert ist ein [**IUIImage-Objekt.**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage)
