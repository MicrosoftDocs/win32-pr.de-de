# <a name="commands-and-resources"></a>Befehle und Ressourcen

Referenzdokumentation zu Befehls- und Ressourcenmarkupelementen im Windows Menübandframework.



| Thema                                                                                            | Inhalte                                                                                                                |
|--------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**Befehl**](windowsribbon-element-command.md)                                                 | Stellt eine Befehlsdefinition dar.<br/>                                                                             |
| [**Command.Comment**](windowsribbon-element-command-comment.md)                                 | Stellt einen Kommentar oder eine Anmerkung für einen Befehl dar.<br/>                                                          |
| [**Command.Id**](windowsribbon-element-command-id.md)                                           | Stellt eine eindeutige ID für einen Befehl dar.<br/>                                                                        |
| [**Command.Keytip**](windowsribbon-element-command-keytip.md)                                   | Stellt die Keytip für ein -Steuerelement dar.<br/>                                                                         |
| [**Command.LabelDescription**](windowsribbon-element-command-labeldescription.md)               | Stellt eine Bezeichnungsbeschreibung dar.<br/>                                                                              |
| [**Command.LabelTitle**](windowsribbon-element-command-labeltitle.md)                           | Stellt einen Bezeichnungstitel dar.<br/>                                                                                    |
| [**Command.LargeHighContrastImages**](windowsribbon-element-command-largehighcontrastimages.md) | Stellt einen Container von Images dar. in diesem Fall große Bilder für die Verwendung mit Systemeinstellungen mit hohem Kontrast.<br/>     |
| [**Command.LargeImages**](windowsribbon-element-command-largeimages.md)                         | Stellt einen Container von Images dar. in diesem Fall große Bilder.<br/>                                                |
| [**Command.Name**](windowsribbon-element-command-name.md)                                       | Stellt den Namen eines Befehls dar.<br/>                                                                            |
| [**Command.SmallHighContrastImages**](windowsribbon-element-command-smallhighcontrastimages.md) | Stellt einen Container von Images dar. in diesem Fall kleine Bilder für die Verwendung mit Systemeinstellungen mit hohem Kontrast.<br/>     |
| [**Command.SmallImages**](windowsribbon-element-command-smallimages.md)                         | Stellt einen Container von Images dar. in diesem Fall kleine Bilder.<br/>                                                |
| [**Command.Symbol**](windowsribbon-element-command-symbol.md)                                   | Stellt den Namen eines [**Befehls dar,**](windowsribbon-element-command.md) auf den extern verwiesen werden kann.<br/> |
| [**Command.TooltipDescription**](windowsribbon-element-command-tooltipdescription.md)           | Stellt eine QuickInfo-Beschreibung dar.<br/>                                                                            |
| [**Command.TooltipTitle**](windowsribbon-element-command-tooltiptitle.md)                       | Stellt einen QuickInfo-Titel dar.<br/>                                                                                  |
| [**Image**](windowsribbon-element-image.md)                                                     | Stellt ein Bild dar.<br/>                                                                                         |
| [**Image.Source**](windowsribbon-element-image-source.md)                                       | Stellt den Verzeichnispfad eines Bilds dar.<br/>                                                                   |
| [**String**](windowsribbon-element-string.md)                                                   | Stellt eine Zeichenfolgenressource dar.<br/>                                                                                |
| [**String.Content**](windowsribbon-element-string-content.md)                                   | Stellt den Inhalt einer Zeichenfolgenressource dar.<br/>                                                                 |
| [**String.Id**](windowsribbon-element-string-id.md)                                             | Stellt die eindeutige ID einer Zeichenfolgenressource dar.<br/>                                                               |
| [**String.Symbol**](windowsribbon-element-string-symbol.md)                                     | Stellt den Namen einer Zeichenfolgenressource dar.<br/>                                                                    |
