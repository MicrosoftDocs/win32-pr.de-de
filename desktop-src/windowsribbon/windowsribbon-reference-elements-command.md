# <a name="commands-and-resources"></a>Befehle und Ressourcen

Referenz Dokumentation für Befehls-und Ressourcen Markup Elemente im Windows-Menü Band Framework.



| Thema                                                                                            | Inhalte                                                                                                                |
|--------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**Get-Help**](windowsribbon-element-command.md)                                                 | Stellt eine Befehls Definition dar.<br/>                                                                             |
| [**Command. Comment**](windowsribbon-element-command-comment.md)                                 | Stellt einen Kommentar oder eine Anmerkung für einen Befehl dar.<br/>                                                          |
| [**Command.Id**](windowsribbon-element-command-id.md)                                           | Stellt eine eindeutige ID für einen Befehl dar.<br/>                                                                        |
| [**Command. KeyTip**](windowsribbon-element-command-keytip.md)                                   | Stellt den KeyTip für ein-Steuerelement dar.<br/>                                                                         |
| [**Command. labeldescription**](windowsribbon-element-command-labeldescription.md)               | Stellt eine Bezeichnungs Beschreibung dar.<br/>                                                                              |
| [**Command. labeltitle**](windowsribbon-element-command-labeltitle.md)                           | Stellt einen Bezeichnungs Titel dar.<br/>                                                                                    |
| [**Command. largehighkontra stimages**](windowsribbon-element-command-largehighcontrastimages.md) | Stellt einen Container von Bildern dar. in diesem Fall sind große Bilder für die Verwendung mit den Systemeinstellungen mit hohem Kontrast.<br/>     |
| [**Command. largeimages**](windowsribbon-element-command-largeimages.md)                         | Stellt einen Container von Bildern dar. in diesem Fall große Bilder.<br/>                                                |
| [**Command.Name**](windowsribbon-element-command-name.md)                                       | Stellt den Namen eines Befehls dar.<br/>                                                                            |
| [**Command. smallhighkontra stimages**](windowsribbon-element-command-smallhighcontrastimages.md) | Stellt einen Container von Bildern dar. in diesem Fall werden kleine Bilder für die Verwendung mit den Systemeinstellungen mit hohem Kontrast verwendet.<br/>     |
| [**Command. smallimages**](windowsribbon-element-command-smallimages.md)                         | Stellt einen Container von Bildern dar. in diesem Fall kleine Bilder.<br/>                                                |
| [**Command. Symbol**](windowsribbon-element-command-symbol.md)                                   | Stellt den Namen eines [**Befehls**](windowsribbon-element-command.md) dar, auf den extern verwiesen werden kann.<br/> |
| [**Command. tooltipdescription**](windowsribbon-element-command-tooltipdescription.md)           | Stellt eine QuickInfo-Beschreibung dar.<br/>                                                                            |
| [**Command. ToolTipTitle**](windowsribbon-element-command-tooltiptitle.md)                       | Stellt einen QuickInfo-Titel dar.<br/>                                                                                  |
| [**Image**](windowsribbon-element-image.md)                                                     | Stellt ein Bild dar.<br/>                                                                                         |
| [**Image. Source**](windowsribbon-element-image-source.md)                                       | Stellt den Verzeichnispfad eines Bilds dar.<br/>                                                                   |
| [**Schnür**](windowsribbon-element-string.md)                                                   | Stellt eine Zeichen folgen Ressource dar.<br/>                                                                                |
| [**String. Content**](windowsribbon-element-string-content.md)                                   | Stellt den Inhalt einer Zeichen folgen Ressource dar.<br/>                                                                 |
| [**String.Id**](windowsribbon-element-string-id.md)                                             | Stellt die eindeutige ID einer Zeichen folgen Ressource dar.<br/>                                                               |
| [**String. Symbol**](windowsribbon-element-string-symbol.md)                                     | Stellt den Namen einer Zeichen folgen Ressource dar.<br/>                                                                    |
