# <a name="ui_pkey_contextavailable"></a>\_Ui PKEY ContextAvailable (Benutzeroberflächen-PKEY-Kontext \_ verfügbar)

Identifiziert die \_ PKEY \_ ContextAvailable-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_ContextAvailable
   shellPKey = UI_PKEY_ContextAvailable
   formatID = 00001100-7363-696e-8441798acf5aebb7
   propID = 1100
   typeInfo
      type = UI_CONTEXTAVAILABILITY
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ ContextAvailable wird von einer Anwendung verwendet, um die Verfügbarkeit eines [Registerkartengruppen-Steuerelements](windowsribbon-controls-tabgroup.md) basierend auf dem aktuellen Anwendungskontext abzufragen.
