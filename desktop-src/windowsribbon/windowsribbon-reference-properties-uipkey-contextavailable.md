# <a name="ui_pkey_contextavailable"></a>UI \_ pkey \_ contextavailable

Identifiziert die UI \_ pkey \_ contextavailable-Eigenschaft.

```
propertyDescription
   name = UI_PKEY_ContextAvailable
   shellPKey = UI_PKEY_ContextAvailable
   formatID = 00001100-7363-696e-8441798acf5aebb7
   propID = 1100
   typeInfo
      type = UI_CONTEXTAVAILABILITY
```

## <a name="remarks"></a>Bemerkungen

UI \_ pkey \_ contextavailable wird von einer Anwendung verwendet, um die Verfügbarkeit eines [Register](windowsribbon-controls-tabgroup.md) Karten-Steuer Elements auf Grundlage des aktuellen Anwendungs Kontexts abzufragen.
