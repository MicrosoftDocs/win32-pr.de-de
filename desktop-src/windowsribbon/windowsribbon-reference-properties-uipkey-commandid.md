# <a name="ui_pkey_commandid"></a>UI- \_ pkey- \_ CommandID

Gibt die Benutzeroberflächen- \_ pkey- \_ CommandID-Eigenschaft an.

```
propertyDescription
   name = UI_PKEY_CommandId
   shellPKey = UI_PKEY_CommandId
   formatID = 00000100-7363-696e-8441798acf5aebb7
   propID = 100
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Bemerkungen

UI \_ pkey \_ CommandID wird von einer Anwendung verwendet, um die IDs der Befehle in einem Katalog-Steuerelement abzufragen.

Der Eigenschafts Wert ist die ID eines Befehls.
