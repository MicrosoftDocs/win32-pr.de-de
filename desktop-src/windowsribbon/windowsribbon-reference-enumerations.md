# <a name="constants-and-enumerations"></a>Konstanten und Enumerationen

Referenzdokumentation für die Windows Menüband-Frameworkkonstanten und -enumerationen.

### <a name="constants-and-enumerations"></a>Konstanten und Enumerationen




| Konstante | BESCHREIBUNG | 
|----------|-------------|
| <a href="/windows/desktop/windowsribbon/windowsribbon-ui-all-commands"><strong>UI_ALL_COMMANDS</strong></a><br /> | Gibt eine Konstante an, die die Auflistung von Befehlen identifiziert, die in der Markupressourcendatei deklariert sind.<br /> | 
| <a href="/windows/desktop/windowsribbon/windowsribbon-ui-collection-invalidindex"><strong>UI_COLLECTION_INVALIDINDEX</strong></a><br /> | Gibt eine Konstante an, die einen ungültigen Index in einer Auflistung identifiziert.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_collectionchange"><strong>UI_COLLECTIONCHANGE</strong></a><br /> | Gibt Werte an, die die Typen von Änderungen identifizieren, die an einer Auflistung vorgenommen werden können.<br /> | 
| <a href="/previous-versions/windows/desktop/legacy/hh448736(v=vs.85)"><strong>UI_COMMANDCATEGORY</strong></a><br /> | Gibt Werte an, die angeben, ob ein Befehl zur Kompilierzeit oder zur Laufzeit definiert wurde.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_commandtype"><strong>UI_COMMANDTYPE</strong></a><br /> | Gibt Werte an, die den Befehlstyp identifizieren, der einem Menübandsteuerelement zugeordnet ist.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_contextavailability"><strong>UI_CONTEXTAVAILABILITY</strong></a><br /> | Gibt Werte an, die die Verfügbarkeit einer <a href="windowsribbon-element-ribbon-contextualtabs.md"><strong>kontextbezogenen Registerkarte</strong></a>identifizieren.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_controldock"><strong>UI_CONTROLDOCK</strong></a><br /> | Gibt Werte an, die den Andockzustand der Schnellzugriffssymbolleiste (Quick Access Toolbar, QAT) identifizieren. <br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_eventlocation"><strong>UI_EVENTLOCATION</strong></a><br /> | Identifiziert die Speicherorte, an denen ereignisse, die einem Menübandsteuerelement zugeordnet sind, stammen können.<br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_eventtype"><strong>UI_EVENTTYPE</strong></a><br /> | Identifiziert die Typen von Ereignissen, die einem <a href="windowsribbon-element-ribbon.md"><strong>Menüband</strong></a>zugeordnet sind.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_executionverb"><strong>UI_EXECUTIONVERB</strong></a><br /> | Gibt Werte an, die die Ausführungs-IDs identifizieren, die Aktionen zugeordnet sind, die ein Benutzer für einen <a href="windowsribbon-element-command.md"><strong>Befehl</strong></a>initiieren kann. <br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontdeltasize"><strong>UI_FONTDELTASIZE</strong></a><br /> | Gibt Werte an, die angeben, ob der Schriftgrad einer hervorgehobenen Textläufe erhöht oder dekrementiert werden soll.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontproperties"><strong>UI_FONTPROPERTIES</strong></a><br /> | Gibt Werte an, die den Schriftarteigenschaftszustand eines <a href="windowsribbon-element-fontcontrol.md"><strong>FontControl-Objekts</strong></a>identifizieren, <strong>z. B. Durchgestrichen.</strong><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontunderline"><strong>UI_FONTUNDERLINE</strong></a><br /> | Gibt Werte an, die den Unterstreichungszustand eines <a href="windowsribbon-element-fontcontrol.md"><strong>FontControl</strong></a>-Werts identifizieren.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontverticalposition"><strong>UI_FONTVERTICALPOSITION</strong></a><br /> | Gibt Werte an, die den vertikalen Ausrichtungszustand eines <a href="windowsribbon-element-fontcontrol.md"><strong>FontControl</strong></a>identifizieren.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_invalidations"><strong>UI_INVALIDATIONS</strong></a><br /> | Gibt Werte an, die den Aspekt eines zu invalidierenden Befehls identifizieren.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_ownership"><strong>UI_OWNERSHIP</strong></a><br /> | Gibt Werte an, die die Besitzbedingungen identifizieren, unter denen ein Bild vom Menübandframework erstellt wird.<br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_swatchcolormode"><strong>UI_SWATCHCOLORMODE</strong></a><br /> | Gibt an, ob eine Farbuhr im normalen oder monofarbenen Modus vorliegt.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_swatchcolortype"><strong>UI_SWATCHCOLORTYPE</strong></a><br /> | Gibt die Werte an, die angeben, wie eine Farbwatch in einem <a href="windowsribbon-element-dropdowncolorpicker.md"><strong>DropDownColorPicker</strong></a> oder einer <a href="windowsribbon-element-fontcontrol.md"><strong>FontControl-Farbauswahl</strong></a> <strong>(Textfarbe</strong> oder <strong>Textmarkierung</strong>) ausgefüllt wird.<br /><blockquote>[!Note]<br />Dies sind nur Empfehlungen. Die Anwendung kann diesen Werten jeden Fülltyp zuordnen.</blockquote><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_viewtype"><strong>UI_VIEWTYPE</strong></a><br /> | Gibt Werte an, die die Menübandframeworkansicht identifizieren.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_viewverb"><strong>UI_VIEWVERB</strong></a><br /> | Gibt Werte an, die den Typ der Aktion identifizieren, die in einer Menübandframeworkansicht abgeschlossen werden soll.<br /> |
