# <a name="windows-ribbon-framework-glossary"></a>Glossar für Windows-Menü Band Framework

## <a name="g"></a>G

<dl> <dt>

**Galerie**
</dt> <dd>

Umfassende, anpassbare Listenfelder, die verwendet werden können, um Elemente nach Kategorie zu organisieren, Sie in flexiblen Spalten-und Zeilen basierten Layouts anzuzeigen und diese mit Bildern und Text darzustellen. Abhängig vom Typ des Katalogs wird auch die Live Vorschau unterstützt.

</dd> </dl>

## <a name="m"></a>M

<dl> <dt>

**MiniToolbar**
</dt> <dd>

Eine Reihe von Menü Band Befehlen, die auf den aktuellen kontextbezogen sind und in einem rechteckigen Container angezeigt werden, der in der Nähe des Zeigers angezeigt wird.

</dd> </dl>

## <a name="q"></a>Q

<dl> <dt>

**Symbolleiste für den schnell Zugriff (QAT)**
</dt> <dd>

Eine kleine anpassbare Symbolleiste auf der Menüband-Benutzeroberfläche, mit der ein Klick auf Befehle ermöglicht wird, die vom Benutzer ausgewählt wurden.

</dd> </dl>

## <a name="r"></a>R

<dl> <dt>

**Menü Band Markup**
</dt> <dd>

Standard mäßiges XML-basiertes deklaratives Markup, das die Befehle, die visuellen Ressourcen, das Steuerelement Layout und das Skalierungs Verhalten der Menüband-Benutzeroberfläche angibt.

</dd> </dl>

## <a name="t"></a>T

<dl> <dt>

**Vorlage**
</dt> <dd>

Ein Satz von layouthinweisen, die die Größe und die Organisation der Menüband-Steuerelemente basierend auf der Menü Band Größe angeben.

</dd> </dl>

## <a name="v"></a>V

<dl> <dt>

**Ansicht**
</dt> <dd>

Ein Container der obersten Ebene, der die Benutzeroberflächen-Steuerelemente der Windows-Multifunktionsleiste

</dd> </dl>

## <a name="w"></a>W

<dl> <dt>

**Windows-Menü Band Framework (Menüband)**
</dt> <dd>

Ein umfangreiches Befehls Präsentationssystem, das Hauptfunktionen einer Anwendung über eine Befehlsleiste am oberen Rand eines Anwendungsfensters verfügbar macht.

</dd> </dl>
