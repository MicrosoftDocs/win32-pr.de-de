# <a name="ui_pkey_formatstring"></a>UI \_ PKEY \_ FormatString

Identifiziert die \_ PKEY \_ FormatString-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_FormatString
   shellPKey = UI_PKEY_FormatString
   formatID = 00000207-7363-696e-8441798acf5aebb7
   propID = 207
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ FormatString wird von einer Anwendung verwendet, um das Format (oder die Maßeinheit) für den Wert des [Spinner-Steuerelements](windowsribbon-controls-spinner.md) abzufragen.
