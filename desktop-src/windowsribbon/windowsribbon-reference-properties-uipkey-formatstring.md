# <a name="ui_pkey_formatstring"></a>UI \_ pkey- \_ Format Zeichenfolge

Bezeichnet die Benutzeroberflächen- \_ pkey \_ FormatString-Eigenschaft.

```
propertyDescription
   name = UI_PKEY_FormatString
   shellPKey = UI_PKEY_FormatString
   formatID = 00000207-7363-696e-8441798acf5aebb7
   propID = 207
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Bemerkungen

Die UI- \_ pkey- \_ Format Zeichenfolge wird von einer Anwendung verwendet, um das Format (oder Maßeinheit) für den Wert des [Spinner](windowsribbon-controls-spinner.md) -Steuer Elements abzufragen.
