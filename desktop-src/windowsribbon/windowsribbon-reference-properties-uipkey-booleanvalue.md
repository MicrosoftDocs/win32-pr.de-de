# <a name="ui_pkey_booleanvalue"></a>UI \_ PKEY \_ BooleanValue

Identifiziert die \_ PKEY \_ BooleanValue-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_BooleanValue
   shellPKey = UI_PKEY_BooleanValue
   formatID = 00000200-7363-696e-8441798acf5aebb7
   propID = 200
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ BooleanValue wird von einer Anwendung verwendet, um den Umschaltzustand eines [Kontrollkästchens,](windowsribbon-controls-checkbox.md)der [Umschaltfläche](windowsribbon-controls-togglebutton.md)und des Schaltflächensteuerelements einer [**SplitButtonGallery**](windowsribbon-element-splitbuttongallery.md)abzufragen.
