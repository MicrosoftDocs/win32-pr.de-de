# <a name="ui_pkey_booleanvalue"></a>UI \_ pkey \_ BooleanValue

Gibt die \_ Eigenschaft "BooleanValue" der UI-pkey an \_ .

```
propertyDescription
   name = UI_PKEY_BooleanValue
   shellPKey = UI_PKEY_BooleanValue
   formatID = 00000200-7363-696e-8441798acf5aebb7
   propID = 200
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Bemerkungen

UI \_ pkey \_ BooleanValue wird von einer Anwendung verwendet, um den UMSCHALT Zustand eines [Kontrollkästchens](windowsribbon-controls-checkbox.md), eine [UMSCHALT Fläche](windowsribbon-controls-togglebutton.md)und das Schaltflächen-Steuerelement eines [**splitbuttongallery**](windowsribbon-element-splitbuttongallery.md)abzufragen.
