# <a name="ui_pkey_minimized"></a>UI- \_ pkey \_ minimiert

Gibt die \_ minimierte Eigenschaft "UI pkey" an \_ .

```
propertyDescription
   name = UI_PKEY_Minimized
   shellPKey = UI_PKEY_Minimized
   formatID = 00001001-7363-696e-8441798acf5aebb7
   propID = 1001
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Bemerkungen

Der Benutzeroberflächen- \_ pkey \_ wird von einer Anwendung verwendet, um den reduzierten oder erweiterten Zustand der Menüband-Benutzeroberfläche abzufragen.



| Anzeige Zustand | Eigenschafts Schlüsselwert |
|---------------|--------------------|
| Expanded      | **false**          |
| Reduziert     | **true**           |



 

Wenn sich die Multifunktionsleisten-Benutzeroberfläche in einem minimierten Zustand befindet, bleibt die Menüband-Registerkarte sichtbar und voll funktionsfähig.
