# <a name="ui_pkey_pinned"></a>UI- \_ pkey \_ fixiert

Identifiziert die angeheftete Eigenschaft des UI- \_ pkeys \_ .

```
propertyDescription
   name = UI_PKEY_Pinned
   shellPKey = UI_PKEY_Pinned
   formatID = 00000351-7363-696e-8441798acf5aebb7
   propID = 351
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Bemerkungen

Der angeheftete UI- \_ pkey \_ wird von einer Anwendung verwendet, um abzufragen, ob ein Element im [Anwendungsmenü](windowsribbon-controls-applicationmenu.md) über Anwendungs Instanzen hinweg "fixiert" oder dauerhaft ist. Beispielsweise kann auf ein Element in der Liste der zuletzt verwendeten Elemente (MRU) zugegriffen werden, und die Liste der MRU-Elemente wird erst gelöscht, wenn es "nicht fixiert" ist.
