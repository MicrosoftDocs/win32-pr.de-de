# <a name="ui_pkey_fontproperties_foregroundcolor"></a>UI \_ PKEY \_ FontProperties \_ ForegroundColor

Identifiziert die \_ PKEY \_ FontProperties \_ ForegroundColor-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_FontProperties_ForegroundColor
   shellPKey = UI_PKEY_FontProperties_ForegroundColor
   formatID = 00000308-7363-696e-8441798acf5aebb7
   propID = 308
   typeInfo
      type = COLORREF
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ FontProperties \_ ForegroundColor wird von einer Anwendung in Verbindung mit [der Benutzeroberfläche \_ PKEY \_ FontProperties \_ ForegroundColorType](windowsribbon-reference-properties-uipkey-fontproperties-foregroundcolortype.md)verwendet, um Einstellungen des **Textfarbkatalogs** abzufragen.

Der Standardwert ist `0x00000000`.
