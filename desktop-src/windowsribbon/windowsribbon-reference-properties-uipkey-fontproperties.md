# <a name="ui_pkey_fontproperties"></a>\_ \_ PKEY-Schriftarteigenschaften der Benutzeroberfläche

Identifiziert die \_ PKEY \_ FontProperties-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_FontProperties
   shellPKey = UI_PKEY_FontProperties
   formatID = 00000300-7363-696e-8441798acf5aebb7
   propID = 300
   typeInfo
      type = IPropertyStore
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY FontProperties ist ein Eigenschaftenspeicher, der von einer Anwendung zum Abfragen \_ aller Schriftarteigenschaften verwendet wird.
