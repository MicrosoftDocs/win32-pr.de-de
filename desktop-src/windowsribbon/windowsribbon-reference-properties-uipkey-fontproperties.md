# <a name="ui_pkey_fontproperties"></a>UI- \_ pkey \_ fontproperties

Bezeichnet die Eigenschaft "UI \_ pkey \_ fontproperties".

```
propertyDescription
   name = UI_PKEY_FontProperties
   shellPKey = UI_PKEY_FontProperties
   formatID = 00000300-7363-696e-8441798acf5aebb7
   propID = 300
   typeInfo
      type = IPropertyStore
```

## <a name="remarks"></a>Bemerkungen

UI \_ pkey \_ fontproperties ist ein Eigenschafts Speicher, der von einer Anwendung verwendet wird, um alle Schriftart Eigenschaften abzufragen.
