# <a name="ui_pkey_nocolorlabel"></a>UI \_ PKEY \_ NoColorLabel

Identifiziert die \_ PKEY \_ NoColorLabel-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_NoColorLabel
   shellPKey = UI_PKEY_NoColorLabel
   formatID = 00000407-7363-696e-8441798acf5aebb7
   propID = 407
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ NoColorLabel wird von einer Anwendung verwendet, um den Wert der Bezeichnung abzufragen, die der Schaltfläche **Keine Farbe** eines [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md)zugeordnet ist.

Der -Eigenschaftswert ist eine Zeichenfolge, die die Bezeichnung darstellt, die der Schaltfläche **No color** eines [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md)zugeordnet ist.

UI \_ PKEY \_ NoColorLabel ist für alle **ColorTemplate-Attributwerte** gültig.
