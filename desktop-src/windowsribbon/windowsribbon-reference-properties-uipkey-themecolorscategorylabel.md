# <a name="ui_pkey_themecolorscategorylabel"></a>UI \_ PKEY \_ ThemeColorsCategoryLabel

Identifiziert die \_ PKEY \_ ThemeColorsCategoryLabel-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_ThemeColorsCategoryLabel
   shellPKey = UI_PKEY_ThemeColorsCategoryLabel
   formatID = 00000403-7363-696e-8441798acf5aebb7
   propID = 403
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ ThemeColorsCategoryLabel wird von einer Anwendung verwendet, um den Wert der Bezeichnung abzufragen, die der Kategorie **Designfarben** eines [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md)zugeordnet ist.

UI \_ PKEY \_ ThemeColorsCategoryLabel ist nur für [**dropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) gültig, wobei `ThemeColors` als Wert des **ColorTemplate-Attributs** angegeben wird (dies ist die einzige Vorlage, die bezeichnete Kategorien enthält).

Der folgende Screenshot zeigt einen `ThemeColors` [**DropDownColorPicker.**](windowsribbon-element-dropdowncolorpicker.md)
