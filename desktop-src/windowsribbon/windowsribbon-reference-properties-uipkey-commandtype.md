# <a name="ui_pkey_commandtype"></a>\_PKEY-Befehlstyp der \_ Benutzeroberfläche

Identifiziert die \_ PKEY \_ CommandType-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_CommandType
   shellPKey = UI_PKEY_CommandType
   formatID = 00000105-7363-696e-8441798acf5aebb7
   propID = 105
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ CommandType wird von einer Anwendung verwendet, um die Befehlstypen in einem Katalogsteuerelemente abfragt.

Der Eigenschaftswert ist aus der [**\_ COMMANDTYPE-Enumeration der Benutzeroberfläche.**](/windows/desktop/api/uiribbon/ne-uiribbon-ui_commandtype)
