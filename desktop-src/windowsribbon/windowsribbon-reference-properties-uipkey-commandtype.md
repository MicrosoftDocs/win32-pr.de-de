# <a name="ui_pkey_commandtype"></a>UI \_ pkey \_ CommandType

Bezeichnet die \_ Eigenschaft "pkey CommandType" der UI \_ .

```
propertyDescription
   name = UI_PKEY_CommandType
   shellPKey = UI_PKEY_CommandType
   formatID = 00000105-7363-696e-8441798acf5aebb7
   propID = 105
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Bemerkungen

Der UI- \_ pkey \_ CommandType wird von einer Anwendung verwendet, um die Befehls Typen in einem Katalog-Steuerelement abzufragen.

Der Eigenschafts Wert ist aus der [**UI \_ CommandType**](/windows/desktop/api/uiribbon/ne-uiribbon-ui_commandtype) -Enumeration.
