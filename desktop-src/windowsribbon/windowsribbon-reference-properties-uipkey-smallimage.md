# <a name="ui_pkey_smallimage"></a>UI \_ pkey \_ smallImage

Identifiziert die Benutzeroberflächen- \_ pkey- \_ Eigenschaft smallImage.

```
propertyDescription
   name = UI_PKEY_SmallImage
   shellPKey = UI_PKEY_SmallImage
   formatID = 00000009-7363-696e-8441798acf5aebb7
   propID = 9
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Bemerkungen

"UI \_ pkey \_ smallImage" wird von einer Anwendung verwendet, um das mit einem Menüband-Steuerelement verknüpfte kleine Bild abzufragen.

Der Eigenschafts Wert ist ein [**iuiimage**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage) -Objekt.
