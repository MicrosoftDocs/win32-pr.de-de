# <a name="ui_pkey_smallimage"></a>UI \_ PKEY \_ SmallImage

Identifiziert die \_ PKEY \_ SmallImage-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_SmallImage
   shellPKey = UI_PKEY_SmallImage
   formatID = 00000009-7363-696e-8441798acf5aebb7
   propID = 9
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ SmallImage wird von einer Anwendung verwendet, um das kleine Bild abzufragen, das einem Menübandsteuerelement zugeordnet ist.

Der Eigenschaftswert ist ein [**IUIImage-Objekt.**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage)
