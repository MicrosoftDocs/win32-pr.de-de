# <a name="ui_pkey_stringvalue"></a>UI \_ PKEY \_ StringValue

Identifiziert die \_ PKEY \_ StringValue-Eigenschaft der Benutzeroberfläche.

```
propertyDescription
   name = UI_PKEY_StringValue
   shellPKey = UI_PKEY_StringValue
   formatID = 00000202-7363-696e-8441798acf5aebb7
   propID = 202
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Hinweise

Ui \_ PKEY \_ StringValue wird von einer Anwendung verwendet, um den Wert im Bearbeitungsfeld des [Kombinationsfeld-Steuerelements](windowsribbon-controls-combobox.md) abzufragen.
