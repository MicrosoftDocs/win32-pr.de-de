# <a name="ui_pkey_decimalvalue"></a>UI \_ pkey \_ DecimalValue

Identifiziert die Benutzeroberflächen- \_ pkey \_ DecimalValue-Eigenschaft.

```
propertyDescription
   name = UI_PKEY_DecimalValue
   shellPKey = UI_PKEY_DecimalValue
   formatID = 00000201-7363-696e-8441798acf5aebb7
   propID = 201
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Bemerkungen

UI \_ pkey \_ DecimalValue wird von einer Anwendung verwendet, um den Wert im Bearbeitungsfeld des [Spinner](windowsribbon-controls-spinner.md) -Steuer Elements abzufragen.
