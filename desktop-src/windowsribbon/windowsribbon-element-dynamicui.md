# <a name="dynamicui-element"></a>Dynamicui-Element

Stellt einen Container für Menü Band Markup dar. Dieser Container und sein Inhalt werden zur Laufzeit deklariert.

## <a name="usage"></a>Verbrauch

``` syntax
<DynamicUI>
  child elements
</DynamicUI>
```

## <a name="attributes"></a>Attribute

Es gibt keine Attribute.

## <a name="child-elements"></a>Untergeordnete Elemente



| Element                                                               | BESCHREIBUNG                                     |
|-----------------------------------------------------------------------|-------------------------------------------------|
| [**Contextpopup**](windowsribbon-element-contextpopup.md)<br/> | Muss genau einmal auftreten <br/> <br/> |
| [**Registerkarte**](windowsribbon-element-tab.md)<br/>                   | Muss genau einmal auftreten <br/> <br/> |
| [**TabGroup**](windowsribbon-element-tabgroup.md)<br/>         | Muss genau einmal auftreten <br/> <br/> |



## <a name="parent-elements"></a>Übergeordnete Elemente

Es gibt keine übergeordneten Elemente.

## <a name="element-information"></a>Elementinformationen



|                                     |                                                                         |
|-------------------------------------|-------------------------------------------------------------------------|
| Unterstützte Mindestversion (System)<br/> | Windows 7, Windows Vista mit SP2 und Platt Form Update für Windows Vista |
| Kann leer bleiben                        | Nein                                                                      |
