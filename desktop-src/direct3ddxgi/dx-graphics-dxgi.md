# <a name="dxgi"></a><span data-ttu-id="0eb84-103">DXGI</span><span class="sxs-lookup"><span data-stu-id="0eb84-103">DXGI</span></span>

<span data-ttu-id="0eb84-104">Microsoft DirectX Graphic Infrastructure (DXGI) verarbeitet das Auflisten von Grafikadaptern, das Auflisten von Anzeigemodi, das Auswählen von Pufferformaten, das Freigeben von Ressourcen zwischen Prozessen (z. B. zwischen Anwendungen und dem Desktopfenster-Manager (DWM)) und die Darstellung gerenderte Frames in einem Fenster oder Monitor für die Anzeige.</span><span class="sxs-lookup"><span data-stu-id="0eb84-104">Microsoft DirectX Graphics Infrastructure (DXGI) handles enumerating graphics adapters, enumerating display modes, selecting buffer formats, sharing resources between processes (such as, between applications and the Desktop Window Manager (DWM)), and presenting rendered frames to a window or monitor for display.</span></span>

<span data-ttu-id="0eb84-105">DXGI wird von Direct3D 10, Direct3D 11 und Direct3D 12 verwendet.</span><span class="sxs-lookup"><span data-stu-id="0eb84-105">DXGI is used by Direct3D 10, Direct3D 11 and Direct3D 12.</span></span>

<span data-ttu-id="0eb84-106">Obwohl die meisten Grafikprogrammierungen mit Direct3D durchgeführt werden, können Sie DXGI verwenden, um Frames für ein Fenster, einen Monitor oder eine andere Grafikkomponente für die spätere Komposition und Anzeige darzustellen.</span><span class="sxs-lookup"><span data-stu-id="0eb84-106">Though most graphics programming is done using Direct3D, you can use DXGI to present frames to a window, monitor, or other graphics component for eventual composition and display.</span></span> <span data-ttu-id="0eb84-107">Sie können dxgi auch verwenden, um den Inhalt eines Monitors zu lesen.</span><span class="sxs-lookup"><span data-stu-id="0eb84-107">You can also use DXGI to read the contents on a monitor.</span></span>
