# <a name="dxgi"></a>DXGI

Microsoft DirectX Graphics Infrastructure (DXGI) übernimmt das Auflisten von Grafikadaptern, das Auflisten der Anzeigemodi, das Auswählen von Puffer Formaten, das Freigeben von Ressourcen zwischen Prozessen (z. b. zwischen Anwendungen und der Desktopfenster-Manager (DWM)) sowie das darstellen von gerenderten Frames für ein Fenster oder einen Monitor zur Anzeige.

DXGI wird von Direct3D 10, Direct3D 11 und Direct3D 12 verwendet.

Obwohl die meisten Grafik Programmierung mit Direct3D durchgeführt wird, können Sie DXGI verwenden, um einem Fenster, einem Monitor oder einer anderen Grafik Komponente Frames für die spätere Komposition und Anzeige darzustellen. Sie können DXGI auch zum Lesen der Inhalte auf einem Monitor verwenden.
