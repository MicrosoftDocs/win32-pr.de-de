# <a name="memory-performance-information"></a>Arbeitsspeicherleistungsinformationen

Arbeitsspeicherleistungsinformationen sind vom Arbeitsspeicher-Manager über die [Systemleistungsindikatoren](../perfctrs/performance-counters-portal.md) und über Funktionen wie [**GetPerformanceInfo,**](/windows/win32/api/psapi/nf-psapi-getperformanceinfo) [**GetProcessMemoryInfo**](/windows/win32/api/psapi/nf-psapi-getprocessmemoryinfo)und [**GlobalMemoryStatusEx verfügbar.**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-globalmemorystatusex) Anwendungen wie Windows Task-Manager, Zuverlässigkeit und Leistungsmonitor und das Tool [Prozess-Explorer](/sysinternals/downloads/process-explorer) verwenden Leistungsindikatoren, um Speicherinformationen für das System und für einzelne Prozesse anzuzeigen.

In diesem Thema werden leistungsindikatoren den von Speicherleistungsfunktionen und der -Funktion zurückgegebenen Daten Windows Task-Manager:

-   [Systemspeicherleistungsinformationen](#system-memory-performance-information)
-   [Informationen zur Prozessspeicherleistung](#process-memory-performance-information)
-   [Zugehörige Themen](#related-topics)
