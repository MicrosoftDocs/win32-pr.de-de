## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                    | BESCHREIBUNG                                                                                                                                                                         |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IDENTITÄTSRICHTLINIE FÜR \_ ENCLAVE-VERSIEGELUNG \_ \_**](/windows/desktop/api/ntenclv/ne-ntenclv-enclave_sealing_identity_policy)<br/> | Definiert Werte, die angeben, wie eine andere Enclave mit der Enclave verknüpft werden muss, die [**EnclaveSealData**](/windows/desktop/api/winenclaveapi/nf-winenclaveapi-enclavesealdata) für die Enclave aufruft, um die Daten zu entsiegeln.<br/> |
