# <a name="memory-management-memory-management"></a>Speicherverwaltung (Speicherverwaltung)

Der Speicher-Manager implementiert den virtuellen Arbeitsspeicher, stellt einen Kernsatz von Diensten bereit, wie z. b. im Speicher abgebildete Dateien, kopiearbeitsspeicher, Unterstützung für großen Speicher und zugrunde liegende Unterstützung für den Cache-Manager.
