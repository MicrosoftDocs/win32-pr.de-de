# <a name="memory-management-memory-management"></a>Speicherverwaltung (Speicherverwaltung)

Der Arbeitsspeicher-Manager implementiert virtuellen Arbeitsspeicher, stellt einen Kernsatz von Diensten bereit, z. B. speicherabbildete Dateien, Speicher für Kopiervorgang beim Schreiben, Unterstützung für großen Arbeitsspeicher und zugrunde liegende Unterstützung für den Cache-Manager.
