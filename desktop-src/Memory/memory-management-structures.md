# <a name="memory-management-structures"></a>Speicher Verwaltungsstrukturen

Die folgenden Strukturen werden mit der Speicherverwaltung verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [**CFG \_ - \_ \_ aufrufzielinformationen**](-cfg-call-target-info.md) | Stellt Informationen über callziele für den Ablauf Steuerungs Schutz (Control Flow Guard, cfg) dar. |
| [**Heap \_ Optimieren von \_ Ressourcen \_ Informationen**](/windows/desktop/api/winnt/ns-winnt-heap_optimize_resources_information) | Gibt Flags für einen heapoptimizeresources-Vorgang an, der mit [**HeapSetInformation**](/windows/desktop/api/HeapApi/nf-heapapi-heapsetinformation)initiiert wird. |
| [**Anforderungen an die Speicher \_ Adresse \_**](/windows/desktop/api/winnt/ns-winnt-mem_address_requirements) | Gibt eine niedrigste und höchste Basisadresse sowie eine Ausrichtung als Teil eines erweiterten Parameters an eine Funktion an, die den virtuellen Arbeitsspeicher verwaltet. |
| [**\_Erweiterter Arbeitsspeicher \_ Parameter**](/windows/desktop/api/winnt/ns-winnt-mem_extended_parameter) | Stellt einen erweiterten Parameter für eine Funktion dar, die den virtuellen Arbeitsspeicher verwaltet. |
| [**\_grundlegende \_ Informationen zum Speicher**](/windows/desktop/api/winnt/ns-winnt-memory_basic_information) | Enthält Informationen zu einem Bereich von Seiten im virtuellen Adressraum eines Prozesses. |
| [**MEMORYSTATUS**](/windows/desktop/api/WinBase/ns-winbase-memorystatus) | Enthält Informationen zum aktuellen Zustand von physischem und virtuellem Arbeitsspeicher. |
| [**MEMORYSTATUS usex**](/windows/desktop/api/sysinfoapi/ns-sysinfoapi-memorystatusex) | Enthält Informationen zum aktuellen Zustand von physischem und virtuellem Arbeitsspeicher, einschließlich erweiterter Arbeitsspeicher. |
| [**Prozess \_ Heap \_ Eintrag**](/windows/desktop/api/minwinbase/ns-minwinbase-process_heap_entry) | Enthält Informationen zu einem Heap Element. |
| [**Win32- \_ Speicher \_ Bereichs \_ Eintrag**](/windows/desktop/api/memoryapi/ns-memoryapi-win32_memory_range_entry) | Gibt einen Bereich von Arbeitsspeicher an. |
| [**Win32- \_ Speicher \_ Regions \_ Informationen**](/windows/desktop/api/memoryapi/ns-memoryapi-win32_memory_region_information) | Enthält Informationen zu einem Speicherbereich. |
| [**Atlthunkdata \_ t**](/previous-versions/windows/desktop/legacy/mt805050(v=vs.85)) | Eine nicht transparente Datenstruktur, die einen ATL-Thunk darstellt. |
