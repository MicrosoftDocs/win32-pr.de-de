# <a name="trusted-execution"></a>Vertrauenswürdige Ausführung

Enclaves werden verwendet, um vertrauenswürdige Ausführungsumgebungen zu erstellen. Eine Enclave ist eine isolierte Code- und Datenregion innerhalb des Adressraums für eine Anwendung. Nur Code, der innerhalb der Enclave ausgeführt wird, kann auf Daten innerhalb derselben Enclave zugreifen.
