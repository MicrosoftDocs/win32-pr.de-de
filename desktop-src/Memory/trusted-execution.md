# <a name="trusted-execution"></a>Vertrauenswürdige Ausführung

Enklaven werden verwendet, um vertrauenswürdige Ausführungs Umgebungen zu erstellen. Eine Enclave ist ein isolierter Code Bereich und Daten innerhalb des Adressraums für eine Anwendung. Nur Code, der innerhalb der Enclave ausgeführt wird, kann auf Daten innerhalb desselben Enclave zugreifen.
