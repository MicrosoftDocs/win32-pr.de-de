## <a name="in-this-section"></a>In diesem Abschnitt

| Thema                                                                 | BESCHREIBUNG                                                                 |
|-----------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**Heap \_ Informations \_ Klasse**](/windows/desktop/api/winnt/ne-winnt-heap_information_class) | Gibt die Klasse der Heap Informationen an, die festgelegt oder abgerufen werden sollen. |
| [**\_erweiterter \_ Arbeits \_ Parametertyp**](/windows/desktop/api/WinNT/ne-winnt-mem_extended_parameter_type) | Definiert Werte für erweiterte Parameter, die für die Datei Zuordnung in einem Adressraum verwendet werden. |
