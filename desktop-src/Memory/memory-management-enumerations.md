# <a name="memory-management-enumerations"></a>Speicherverwaltungsenumerationen

Die folgenden Enumerationen werden bei der Speicherverwaltung verwendet:

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema                                                                 | Beschreibung                                                                 |
|-----------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**HEAP \_ \_ INFORMATION-KLASSE**](/windows/desktop/api/winnt/ne-winnt-heap_information_class) | Gibt die Klasse der Heapinformationen an, die festgelegt oder abgerufen werden sollen. |
| [**MEM \_ EXTENDED \_ PARAMETER \_ TYPE**](/windows/desktop/api/WinNT/ne-winnt-mem_extended_parameter_type) | Definiert Werte für erweiterte Parameter, die für die Dateizuordnung in einen Adressraum verwendet werden. |
