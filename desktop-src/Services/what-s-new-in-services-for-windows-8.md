# <a name="whats-new-in-services-for-windows-8"></a>Neuerungen in Diensten für Windows 8

Windows 8 und Windows Server 2012 umfassen die folgenden neuen Funktionen für Dienste.

## <a name="new-service-functions"></a>Neue Dienstfunktionen



| Funktion                                                                            | BESCHREIBUNG                                                                    |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**QueryServiceDynamicInformation**](/windows/desktop/api/Winsvc/nf-winsvc-queryservicedynamicinformation)<br/> | Ruft dynamische Informationen im Zusammenhang mit dem aktuellen Dienststart ab.<br/> |

| API-Element                                                                                     | BESCHREIBUNG                                                                                                                                                   |
|-------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**HandlerEx**](/windows/desktop/api/WinSvc/nc-winsvc-lphandler_function_ex)<br/>                                                       | **DAS \_ DIENSTSTEUERELEMENT \_ USERMODEREBOOT** wurde hinzugefügt.<br/>                                                                                                        |
| [**\_DIENSTSTATUS**](/windows/desktop/api/Winsvc/ns-winsvc-service_status)<br/>                                        | **SERVICE \_ ACCEPT \_ USERMODEREBOOT** wurde hinzugefügt.<br/>                                                                                                         |
| [**\_ \_ DIENSTTRIGGERSPEZIFISCHES \_ \_ DATENELEMENT**](/windows/desktop/api/winsvc/ns-winsvc-service_trigger_specific_data_item)<br/> | SERVICE **\_ \_ \_ \_ TRIGGER-DATENTYPEBENE,** **SERVICE \_ \_ \_ TRIGGER-DATENTYPSCHLÜSSELWORT \_ \_ ANY** und **\_ DIENSTTRIGGER-DATENTYPSCHLÜSSELWORT \_ \_ \_ \_ ALLE** hinzugefügt.<br/> |
