# <a name="windows-networking-callbacks"></a>Windows-Netzwerk Rückrufe

Die folgenden Funktionsprototypen werden in Windows-Netzwerken verwendet.



| Funktionsprototyp                                 | BESCHREIBUNG                                                                                                           |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**Pfngetprofilepath**](/previous-versions/windows/desktop/legacy/aa385358(v=vs.85))     | Prototyp für eine Funktion, die einen Benutzerprofil Pfad abruft.                                                            |
| [**Pfnprocesspolicies**](/previous-versions/windows/desktop/legacy/aa385360(v=vs.85))   | Prototyp für eine Funktion, die die einem bestimmten Windows-Prozess zugeordneten Benutzerkonto-Richtlinien Informationen zurückgibt. |
| [**Pfnversöversleprofile**](/previous-versions/windows/desktop/legacy/aa385364(v=vs.85)) | Prototyp einer Funktion, die ein lokales Profil mit dem entsprechenden globalen Profil gleicht.                        |
