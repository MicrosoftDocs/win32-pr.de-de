# <a name="windows-networking-callbacks"></a>Windows Netzwerkrückrufe

Die folgenden Funktionsprototypen werden in Windows Networking verwendet.



| Funktionsprototyp                                 | Beschreibung                                                                                                           |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**PFNGETPROFILEPATH**](/previous-versions/windows/desktop/legacy/aa385358(v=vs.85))     | Prototyp für eine Funktion, die einen Benutzerprofilpfad erhält.                                                            |
| [**PFNPROCESSPOLICIES**](/previous-versions/windows/desktop/legacy/aa385360(v=vs.85))   | Prototyp für eine Funktion, die die Benutzerkontorichtlinieninformationen zurückgibt, die einem bestimmten Windows Prozess zugeordnet sind. |
| [**PFNRECONCILEPROFILE**](/previous-versions/windows/desktop/legacy/aa385364(v=vs.85)) | Prototyp einer Funktion, die ein lokales Profil mit dem entsprechenden globalen Profil abgleicht.                        |
