<dl> <dt>

<span id="_security_virtual_store_gly"></span><span id="_SECURITY_VIRTUAL_STORE_GLY"></span>**virtueller Speicher**
</dt> <dd>

Ein Zertifikat Speicher, der aus einer Sammlung anderer Zertifikat Speicher besteht. Eine einzelne Such-oder Enumerationsfunktion in einem virtuellen Speicher führt alle Filialen durch, die Mitglieder der Auflistung der virtuellen Filialen sind. Jede Addition eines virtuellen Stores fügt einem der Mitglieder dieses Stores ein Element hinzu.

</dd> <dt>

<span id="_security_virtualization_gly"></span><span id="_SECURITY_VIRTUALIZATION_GLY"></span>**Virtualization**
</dt> <dd>

Ein Feature der Benutzerkontensteuerung (User Account Control, UAC), das Datei-und Registrierungs Vorgänge pro Computer für virtuelle, benutzerspezifische Datei-und Registrierungs Speicherorte anstelle der tatsächlichen computerspezifischen Standorte zulässt.

</dd> </dl>
