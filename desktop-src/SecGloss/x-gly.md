<span id="_security_x.400_gly"></span><span id="_SECURITY_X.400_GLY"></span>**X.400**

</dd> <dt>

<span id="_security_x.500_gly"></span><span id="_SECURITY_X.500_GLY"></span>**X.500**

</dd> <dt>

<span id="_security_x.509_gly"></span><span id="_SECURITY_X.509_GLY"></span>**X.509**
</dt> <dd>

Ein international anerkannter Zertifikatstandard zur Definition erforderlicher Bestandteile.

</dd> <dt>

<span id="_security_x509_asn_encoding_gly"></span><span id="_SECURITY_X509_ASN_ENCODING_GLY"></span>**\_X509-ASN-CODIERUNG \_**
</dt> <dd>

Codierungstyp, der die Zertifikatcodierung angibt. Zertifikatcodierungstypen werden im Niedrigwertwort eines **DWORD** gespeichert (Wert ist: 0x00000001). Dieser Codierungstyp ist funktionell identisch mit CRYPT \_ ASN \_ ENCODING.

</dd> <dt>

<span id="_security_xor_gly"></span><span id="_SECURITY_XOR_GLY"></span>**Xor**
</dt> <dd>

Exklusives **ODER**. Ein boolescher Vorgang, der nur true ergibt, wenn einer seiner Operanden true und der andere false ist. Wenn beide Operanden gleich sind (entweder TRUE oder FALSE), gibt der Vorgang false zurück.

</dd> </dl>
