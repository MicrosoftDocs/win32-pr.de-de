# <a name="u-security-glossary"></a>U (Sicherheits Glossar)

<dl> <dt>

<span id="_security_ucs_gly"></span><span id="_SECURITY_UCS_GLY"></span>**UCS**
</dt> <dd>

Siehe *universellen Zeichensatz*.

</dd> <dt>

<span id="_security_unicode_gly"></span><span id="_SECURITY_UNICODE_GLY"></span>**Unicode-**
</dt> <dd>

Ein weltweit verwendende Zeichen Codierungsstandard, der es ermöglicht, dass mehr Informationen in jeder Zeichenfolge enthalten sind, indem 16-Bit-Zeichen folgen anstelle der standardmäßigen 8-Bit-Zeichen folgen definiert werden. Unicode ermöglicht den universellen Datenaustausch und verbessert die mehrsprachige Textverarbeitung. Unicode-Zeichen folgen werden auch als *Breite* Zeichen folgen bezeichnet.

</dd> <dt>

<span id="_security_universal_character_set_gly"></span><span id="_SECURITY_UNIVERSAL_CHARACTER_SET_GLY"></span>**Universeller Zeichensatz**
</dt> <dd>

UCS Ein 16-Bit-Zeichensatz.

</dd> <dt>

<span id="_security_universal_well_known_security_identifier_gly"></span><span id="_SECURITY_UNIVERSAL_WELL_KNOWN_SECURITY_IDENTIFIER_GLY"></span>**universelle, bekannte Sicherheits-ID**
</dt> <dd>

Eine [*Sicherheits*](s-gly.md) -ID (SID), die auf allen Systemen bekannt ist, die das Sicherheitsmodell auf der Grundlage von authentifizierten Benutzern und der freigegebenen Zugriffs Steuerung einhalten.

</dd> <dt>

<span id="_security_upn_gly"></span><span id="_SECURITY_UPN_GLY"></span>**UPN**
</dt> <dd>

Siehe *Benutzer Prinzipal Name*.

</dd> <dt>

<span id="_security_user_gly"></span><span id="_SECURITY_USER_GLY"></span>**Bedienungs**
</dt> <dd>

Ein Benutzer eines Systems wird durch eine Anmeldeprozedur identifiziert, mit der ein Prozess eingerichtet wird, mit dem Anwendungen ausführen und auf andere sicherheitsrelevante Objekte zugreifen können. Ein Beispiel für eine Anmeldeprozedur ist die Verwendung einer Smartcard, möglicherweise in Verbindung mit einem Kennwort oder einer persönlichen Identifikationsnummer (PIN).

</dd> <dt>

<span id="_security_user_interface_smart_card_gly"></span><span id="_SECURITY_USER_INTERFACE_SMART_CARD_GLY"></span>**Benutzeroberfläche (Smartcard)**
</dt> <dd>

Ein gängiges Dialogfeld, mit dem der Benutzer eine Verbindung mit einer Smartcard herstellen und in einer Anwendung verwenden kann. Mithilfe des Dialog Felds kann der Benutzer eine bestimmte Karte angeben oder nach der zu öffnenden Smartcard suchen.

</dd> <dt>

<span id="_security_user_principal_name_gly"></span><span id="_SECURITY_USER_PRINCIPAL_NAME_GLY"></span>**Benutzer Prinzipal Name**
</dt> <dd>

UPN Einen Benutzerkonto Namen (manchmal auch als Benutzer Anmelde Name bezeichnet) und einen Domänen Namen, der die Domäne identifiziert, in der sich das Benutzerkonto befindet. Dies ist das Standardverfahren für die Anmeldung an einer Windows-Domäne. Das Format ist: someone@example.com (wie bei einer e-Mail-Adresse).

</dd> </dl>
