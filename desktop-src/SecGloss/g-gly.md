<dl> <dt>

<span id="_security_gdi_gly"></span><span id="_SECURITY_GDI_GLY"></span>**GDI**
</dt> <dd>

Siehe *Graphics Device Interface*.

</dd> <dt>

<span id="_security_gina_gly"></span><span id="_SECURITY_GINA_GLY"></span>**Gina**
</dt> <dd>

Eine grafische ID und eine Dynamic Link Library (dll) für die Authentifizierung. Die Gina ist eine ersetzbare dll-Komponente, die von der ausführbaren [*Winlogon*](w-gly.md) -Datei geladen wird. Die Gina implementiert die Authentifizierungs Richtlinie des interaktiven Anmelde Modells und soll alle Benutzerinteraktionen mit Identifikation und Authentifizierung durchführen.

</dd> <dt>

<span id="_security_graphics_device_interface_gly"></span><span id="_SECURITY_GRAPHICS_DEVICE_INTERFACE_GLY"></span>**Graphics Device Interface**
</dt> <dd>

Das ausführbare Programm, das grafische Funktionsaufrufe von einer Windows-basierten Anwendung verarbeitet und diese Aufrufe an den entsprechenden Gerätetreiber übergibt, der die Hardware spezifischen Funktionen ausführt, die eine Ausgabe generieren.

</dd> </dl>
