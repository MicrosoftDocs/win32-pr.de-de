<dl> <dt>

<span id="_security_object_identifier_gly"></span><span id="_SECURITY_OBJECT_IDENTIFIER_GLY"></span>**Objekt Bezeichner**
</dt> <dd>

OID Eine Zahl, die eine Objektklasse oder ein Attribut eindeutig identifiziert. Ein Objekt Bezeichner wird als gepunktete Dezimal Zeichenfolge dargestellt, z. b. 1.2.3.4. Objekt Bezeichner sind in einer globalen Hierarchie angeordnet. Nationale Registrierungsstellen geben Stammobjektbezeichner an Einzelpersonen oder Organisationen aus, die die Hierarchie unter ihrem Stammobjektbezeichner verwalten.

Siehe auch [*Abstract Syntax Notation 1*](a-gly.md), [*Attribute*](a-gly.md).

</dd> <dt>

<span id="_security_ocsp_gly"></span><span id="_SECURITY_OCSP_GLY"></span>**OCSP**
</dt> <dd>

Siehe *Online Certificate Status-Protokoll*.

</dd> <dt>

<span id="_security_ofb_gly"></span><span id="_SECURITY_OFB_GLY"></span>**OFB**
</dt> <dd>

Siehe *Ausgabe Feedback*.

</dd> <dt>

<span id="_security_oid_gly"></span><span id="_SECURITY_OID_GLY"></span>**OID**
</dt> <dd>

Siehe *Objekt Bezeichner*.

</dd> <dt>

<span id="_security_online_certificate_status_protocol_gly"></span><span id="_SECURITY_ONLINE_CERTIFICATE_STATUS_PROTOCOL_GLY"></span>**Online Certificate Status-Protokoll**
</dt> <dd>

OCSP Ein Internet Protokoll, das zum Abrufen des Sperr Status eines digitalen X. 509-Zertifikats verwendet wird.

</dd> <dt>

<span id="_security_opaque_blob_gly"></span><span id="_SECURITY_OPAQUE_BLOB_GLY"></span>**undurchsichtiges BLOB**
</dt> <dd>

Eine Sequenz von Bytes, die zum Speichern von Sitzungs Schlüsseln verwendet werden. Nicht transparente BLOB enthalten das Basis Schlüsselmaterial und alle aktuellen Zustandsinformationen. Dies schließt Informationen wie den Salt-Wert, den Initialisierungs Vektor und die Schlüssel Tabelle ein. Das Format von nicht transparenten blobbden ist nicht veröffentlicht. Jeder CSP-Anbieter bestimmt sein eigenes BLOB-Format, das die Verschlüsselung der nicht transparenten BLOBs mit einer Art symmetrischen Schlüsselbein haltet.

</dd> <dt>

<span id="_security_outer_content_gly"></span><span id="_SECURITY_OUTER_CONTENT_GLY"></span>**äußerer Inhalt**
</dt> <dd>

Die Verbesserungen für einige gekapselte Daten. Dieser Begriff wird hauptsächlich bei der Erörterung von erweiterten Daten (innerer Inhalt) in PKCS \# 7-Nachrichten verwendet.

</dd> <dt>

<span id="_security_output_feedback_gly"></span><span id="_SECURITY_OUTPUT_FEEDBACK_GLY"></span>**Ausgabe Feedback**
</dt> <dd>

OFB Ein Blockchiffre Modus, der ein Feedback verwendet, das dem CFB-Modus (Verschlüsselungs Feedback) ähnelt. Der einzige Unterschied zwischen den beiden Modi besteht darin, wie das UMSCHALT Register ausgefüllt ist.

Siehe auch [*Chiffre Feedback*](c-gly.md).

</dd> </dl>
