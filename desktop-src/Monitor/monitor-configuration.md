# <a name="monitor-configuration-monitor-configuration"></a>Monitor Konfiguration (Monitor Konfiguration)

Mit den Konfigurationsfunktionen von Monitor können Sie Informationen von einem Monitor erhalten und die Einstellungen eines Monitors ändern. Sie können diese Funktionen für Folgendes verwenden:

-   Ändern Sie die Geometry-Einstellungen eines Monitors, z. b. die Breite und Höhe der Anzeige.
-   Ändern Sie die Bildeinstellungen, wie z. b. Helligkeit und Kontrast.
-   Setzen Sie die Einstellungen eines Monitors auf die Werkseinstellungen zurück.
-   Hiermit wird der Monitor deaktiviert.

Intern verwenden die Monitor Konfigurationsfunktionen die Befehlsschnittstelle zum Anzeigen von Datenkanälen (DDC/CI), um Befehle an den Monitor zu senden.

Die Überwachungs Konfigurationsfunktionen sind in den folgenden Abschnitten dokumentiert:

-   [Informationen zur Monitor Konfiguration](about-monitor-configuration.md)
-   [Verwenden der Monitor Konfiguration](using-monitor-configuration.md)
-   [Monitor Konfigurations Referenz](monitor-configuration-reference.md)
