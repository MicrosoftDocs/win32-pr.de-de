# <a name="monitor-configuration-monitor-configuration"></a>Monitorkonfiguration (Monitorkonfiguration)

Sie können die Monitorkonfigurationsfunktionen verwenden, um Informationen von einem Monitor zu erhalten und die Einstellungen eines Monitors zu ändern. Sie können diese Funktionen für verwenden:

-   Ändern Sie die Geometrieeinstellungen eines Monitors, z. B. die Breite und Höhe der Anzeige.
-   Ändern Sie die Bildeinstellungen, z. B. Helligkeit und Kontrast.
-   Setzen Sie die Einstellungen eines Monitors auf die Werkseinstellungen zurück.
-   Entfernt den Monitor.

Intern verwenden die Monitorkonfigurationsfunktionen die Display Data Channel Command Interface (DDC/CI), um Befehle an den Monitor zu senden.

Die Konfigurationsfunktionen des Monitors sind in den folgenden Abschnitten dokumentiert:

-   [Informationen zur Monitorkonfiguration](about-monitor-configuration.md)
-   [Verwenden der Monitorkonfiguration](using-monitor-configuration.md)
-   [Referenz zur Monitorkonfiguration](monitor-configuration-reference.md)
