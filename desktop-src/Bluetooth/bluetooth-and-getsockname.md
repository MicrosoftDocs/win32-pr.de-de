# <a name="bluetooth-and-getsockname"></a>Bluetooth und getsockname

Bluetooth Ruft mithilfe der Funktion [**getsockname**](/windows/desktop/api/winsock/nf-winsock-getsockname) die Server Geräteadresse und die Portnummer ab, die einem Socket durch einen vorherigen Aufrufen der [**Bind**](/windows/desktop/api/winsock/nf-winsock-bind) -Funktion zugeordnet sind.

Weitere Informationen finden Sie unter [Windows Sockets](/windows/desktop/WinSock/windows-sockets-start-page-2).
