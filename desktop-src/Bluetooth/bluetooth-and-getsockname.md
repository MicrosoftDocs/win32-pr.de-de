# <a name="bluetooth-and-getsockname"></a>Bluetooth und getsockname

Bluetooth ruft mit der [**getsockname-Funktion**](/windows/desktop/api/winsock/nf-winsock-getsockname) die Servergeräteadresse und portnummer ab, die einem Socket durch einen vorherigen Aufruf der [**bind-Funktion zugeordnet**](/windows/desktop/api/winsock/nf-winsock-bind) wurden.

Weitere Informationen finden Sie unter [Windows Sockets](/windows/desktop/WinSock/windows-sockets-start-page-2).
