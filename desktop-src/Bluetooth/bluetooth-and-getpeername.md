# <a name="bluetooth-and-getpeername"></a>Bluetooth und getpeername

Die [**getpeername-Funktion**](/windows/desktop/api/winsock/nf-winsock-getpeername) wird verwendet, um die Bluetooth Adresse des Peers Bluetooth Geräts abzurufen. Die **getpeername-Funktion** ist nur für verbundene Bluetooth Sockets gültig.

Weitere Informationen finden Sie unter [Windows Sockets.](/windows/desktop/WinSock/windows-sockets-start-page-2)
