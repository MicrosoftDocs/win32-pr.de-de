# <a name="bluetooth-and-getpeername"></a>Bluetooth und getPeer Name

Die [**getpeername**](/windows/desktop/api/winsock/nf-winsock-getpeername) -Funktion wird zum Abrufen der Bluetooth-Adresse des Peer-Bluetooth-Geräts verwendet. Die **getPeer Name** -Funktion ist nur für verbundene Bluetooth-Sockets gültig.

Weitere Informationen finden Sie unter [Windows Sockets](/windows/desktop/WinSock/windows-sockets-start-page-2).
