# <a name="about-the-volume-shadow-copy-service"></a>Informationen zum Volumeschattenkopie-Dienst

Der Volumeschattenkopie-Dienst (VSS) erfasst und kopiert stabile Images zur Sicherung auf laufenden Systemen, insbesondere Servern, ohne dass die Leistung und Stabilität der von Ihnen bereitgestellten Dienste übermäßig beeinträchtigt werden.

Die VSS-Lösung ermöglicht Entwicklern das Erstellen von Diensten ([*Writer*](vssgloss-w.md)), die durch die Sicherungs Anwendung eines beliebigen Anbieters mithilfe von VSS ([*anfordernde Personen*](vssgloss-r.md)) effektiv gesichert werden können.
