# <a name="volume-shadow-copy-glossary"></a>Glossar der Volumeschattenkopie

Die Volumeschattenkopie-Dienst (VSS)-API verwendet eine eigene Sammlung von Begriffen. Viele dieser Begriffe sind Entwicklern vertraut, verfügen jedoch über neue oder geänderte Definitionen in der VSS-API-Umgebung. Überprüfen Sie die folgenden Definitionen, um die VSS-API-Implementierung dieser Begriffe zu verstehen.
