# <a name="working-with-the-writer-metadata-document"></a>Arbeiten mit dem Writer-Metadatendokument

Jeder Writer erzeugt ein Writer-Metadatendokument. Die Sicherungs Anwendung verwendet dieses Dokument, um Informationen zu diesem Writer, zu den Daten, die er besitzt, und zum Wiederherstellen dieser Daten zu erhalten. Nachdem der Writer ihn erstellt hat, ist das Writer-Metadatendokument ein Schreib geschütztes Dokument für die Sicherungs Anwendung.

Um das Writer-Metadatendokument verwenden zu können, müssen Sie Folgendes verstehen:

-   [Lebenszyklus von Writer-Metadatendokumenten](writer-metadata-document-life-cycle.md)
-   [Inhalt des Writer-Metadatendokuments](writer-metadata-document-contents.md)
