# <a name="working-with-the-writer-metadata-document"></a>Arbeiten mit dem Writer-Metadatendokument

Jeder Writer erzeugt ein Writer-Metadatendokument. Die Sicherungsanwendung verwendet dieses Dokument, um Informationen zu diesem Writer, zu den Daten, die er besitzt, und zum Wiederherstellen dieser Daten abzurufen. Sobald der Writer es erstellt hat, ist das Writer Metadata Document ein schreibgeschütztes Dokument für die Sicherungsanwendung.

Um das Writer Metadata Document verwenden zu können, müssen Sie Folgendes verstehen:

-   [Lebenszyklus von Writermetadatendokumenten](writer-metadata-document-life-cycle.md)
-   [Inhalt des Writer-Metadatendokuments](writer-metadata-document-contents.md)
