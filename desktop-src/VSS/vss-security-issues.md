# <a name="vss-security-issues"></a>VSS-Sicherheitsprobleme

Die VSS-Infrastruktur basiert intern auf COM. Daher erfordert die Kommunikation von Writern und Anforderern COM-basierte Sicherheit. Com-Sicherheit bietet Mechanismen und Richtlinien, die ein VSS-Entwickler berücksichtigen sollte:

-   [Sicherheitsüberlegungen für Writer](security-considerations-for-writers.md)
-   [Sicherheitsüberlegungen für Anforderer](security-considerations-for-requestors.md)
