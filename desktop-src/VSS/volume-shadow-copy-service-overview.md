# <a name="volume-shadow-copy-service-overview"></a>Volumeschattenkopie-Dienst Übersicht

Der Volumeschattenkopie-Dienst (VSS) ist ein Satz von COM-APIs, die ein Framework implementieren, um Volumesicherungen zu ermöglichen, während Anwendungen auf einem System weiterhin auf die Volumes schreiben.

VSS bietet eine konsistente Schnittstelle, die die Koordination zwischen Benutzeranwendungen ermöglicht, die Daten auf dem Datenträger [*aktualisieren*](vssgloss-w.md)(Writer) und anwendungen sichern [*(Anfordernde*](vssgloss-r.md)).
