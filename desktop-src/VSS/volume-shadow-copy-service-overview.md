# <a name="volume-shadow-copy-service-overview"></a>Übersicht über Volumeschattenkopie-Dienst

Beim Volumeschattenkopie-Dienst (VSS) handelt es sich um einen Satz von COM-APIs, mit denen ein Framework implementiert wird, mit dem Volumesicherungen durchgeführt werden können, während Anwendungen auf einem System weiterhin auf die Volumes schreiben.

VSS stellt eine konsistente Schnittstelle bereit, die die Koordination zwischen Benutzer Anwendungen ermöglicht, die Daten auf einem Datenträger ([*Writer*](vssgloss-w.md)) aktualisieren, und Anwendungen, die [*Anwendungen (*](vssgloss-r.md)Anforderer) sichern.
