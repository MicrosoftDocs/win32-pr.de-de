# <a name="swbemobjectpathisclass-property"></a>SWbemObjectPath.IsClass-Eigenschaft

Die **IsClass-Eigenschaft** des [**SWbemObjectPath-Objekts**](swbemobjectpath.md) ist ein boolescher Wert, der angibt, ob dieser Pfad eine Klasse darstellt. Diese Eigenschaft ist schreibgeschützt.

Eine Erläuterung dieser Syntax finden Sie unter [Dokumentkonventionen für die Skripterstellungs-API.](document-conventions-for-the-scripting-api.md)

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
SWbemObjectPath.IsClass As Boolean
```
