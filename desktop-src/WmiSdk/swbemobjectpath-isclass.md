# <a name="swbemobjectpathisclass-property"></a>Errbemubjectpath. IsClass-Eigenschaft

Die **IsClass** -Eigenschaft des [**errbemobjectpath**](swbemobjectpath.md) -Objekts ist ein boolescher Wert, der angibt, ob dieser Pfad eine Klasse darstellt. Diese Eigenschaft ist schreibgeschützt.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


