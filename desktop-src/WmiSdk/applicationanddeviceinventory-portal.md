# <a name="application-inventory-classes"></a>Anwendungsinventurklassen

## <a name="purpose"></a>Zweck

Die WMI-Klassen für anwendungsinventur ermöglichen die Ermittlung der installierten Win32-Anwendungen und Windows Speichern von Anwendungen auf einem Windows System.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                  | Beschreibung                                                                                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Win32 \_ InstalledWin32Program**](win32-installedwin32program.md)<br/>         | Die [**Win32 \_ InstalledWin32Program-Klasse**](win32-installedwin32program.md) stellt eine installierte Win32-Anwendung dar.<br/>                                                                                                                            |
| [**Win32 \_ InstalledStoreProgram**](win32-installedstoreprogram.md)<br/>         | Die [**Win32 \_ InstalledStoreProgram-Klasse**](win32-installedstoreprogram.md) stellt eine installierte Windows Store-Anwendung dar.<br/>                                                                                                                    |
| [**Win32 \_ InstalledProgramFramework**](win32-installedprogramframework.md)<br/> | Die [**Win32 \_ InstalledProgramFramework-Klasse**](win32-installedprogramframework.md) stellt ein Technologieframework dar, mit dem eine [**Win32 \_ InstalledWin32Program-Instanz**](win32-installedwin32program.md) zur Laufzeit kompiliert oder verwendet wird.<br/> |



 

## <a name="developer-audience"></a>Entwicklergruppe

Die WMI-Klassen für die Anwendungsinventur sind für die Verwendung durch Entwickler von Enterprise-Verwaltungslösungen und lokalen oder Remoteverwaltungstools konzipiert, die eine Inventur von Anwendungen auf einem System generieren müssen.
