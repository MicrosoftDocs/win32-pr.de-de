# <a name="swbemmethodinparameters-property"></a>' Taubemmethod. InParameters '-Eigenschaft

Die **InParameters** -Eigenschaft des-Objekts von " [**errbemmethod**](swbemmethod.md) " ist ein " [**errbemubject**](swbemobject.md) "-Objekt, dessen Eigenschaften die Eingabeparameter für diese Methode definieren. Diese Eigenschaft ist schreibgeschützt. Beachten Sie, dass alle an diesem Objekt vorgenommenen Änderungen nicht in der zugrunde liegenden Methoden Definition widergespiegelt werden.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Diese Eigenschaft ist schreibgeschützt.

```VB
SWbemMethod.InParameters As Object
```
