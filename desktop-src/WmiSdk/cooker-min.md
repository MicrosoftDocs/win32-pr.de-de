# <a name="cooker_min"></a>Koch \_ Min.

Die \_ Formel für den Typ des Wrapper-min-Zählers identifiziert den kleinsten Wert aus einer Reihe von Beobachtungen einer Eigenschaft in einer [**Win32- \_ perfrawdata**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) -Instanz. Dieser berechnete Minimalwert wird in eine Eigenschaft in einer [**Win32- \_ perfformatteddata**](/windows/desktop/CIMWin32Prov/win32-perfformatteddata) -Instanz eingefügt. Dieser Indikatortyp wird nur innerhalb von WMI definiert und steht nicht für die Technologien zur Leistungsüberwachung zur Verfügung, wie z. b. [Leistungsindikatoren der Version 6,0](/windows/desktop/PerfCtrs/performance-counters-portal).

Weitere Informationen zur Formel für den Counter-Typ finden Sie unter [counter Types (gegen Typen](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10))).

Weitere Informationen zu Hochleistungs Anbietern und zur Skripterstellung für WMI finden Sie unter [WMI-Leistungsdaten Typen](wmi-performance-counter-types.md).
