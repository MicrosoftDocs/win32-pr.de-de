# <a name="msft-classes"></a>MSFT-Klassen

MSFT-Klassen sind spezifisch für die Windows-Implementierung des [*Common Information Model*](gloss-c.md) (CIM).

Objekte können in verschiedene Hauptkategorien unterteilt werden:

-   [Richtlinien Anbieter Klassen](policy-provider-classes.md)
-   [WMI-Fehler Behebungs Klassen](wmi-troubleshooting-classes.md)

Die folgenden MSFT-Klassen werden in "System. mof" definiert, aber nicht von WMI implementiert.

-   MSFT \_ wmicoreevent
-   MSFT \_ wmicorelogoffevent
-   MSFT \_ wmicorelogonevent
-   MSFT \_ wmicoreobject
-   MSFT \_ wmicorestatus
-   MSFT \_ wmicoreuser
-   MSFT \_ wmitaskfailure
