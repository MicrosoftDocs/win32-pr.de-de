# <a name="programming-with-the-system-registry-provider"></a>Programmieren mit dem System Registrierungs Anbieter

In den folgenden Themen wird beschrieben, wie Anwendungen den System Registrierungs Anbieter verwenden können, um Daten für die von Ihnen erstellten Klassen bereitzustellen.



| Thema                                                                                                                      | BESCHREIBUNG                                                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Definieren von Klassen für den System Registrierungs Anbieter](defining-classes-for-the-system-registry-provider.md)                 | Für den Zugriff auf die Systemregistrierung müssen Sie Windows-Klassen erstellen, die Registrierungsinformationen im WMI-Repository (Windows-Verwaltungsinstrumentation) speichern.                                                                     |
| [Verwenden des System Registrierungs Anbieters als Eigenschaften Anbieter](using-the-system-registry-provider-as-a-property-provider.md) | Der System Registrierungs Anbieter erfordert, dass Sie mehrere Qualifizierer zum Anzeigen von Registrierungs Eigenschaften verwenden.                                                                                                                             |
| [Beschreiben einer Ressource für die Registrierung](describing-a-resource-for-the-registry.md)                                       | Sie können auf Daten in der Registrierung zugreifen, in denen eine System Ressource beschrieben wird. Um die Informationen zu ändern, benötigt der System Registrierungs Anbieter eine bestimmte Syntax.                                                                      |
| [Zugreifen auf einen unbenannten Registrierungs Wert](accessing-an-unnamed-registry-value.md)                                             | Um auf die meisten Registrierungs Werte zuzugreifen, erstellen Sie eine Klasse, um die Werte in WMI zu empfangen, oder Sie können bestimmte Registrierungs Eigenschaften überprüfen. Der System Registrierungs Anbieter benötigt jedoch eine spezielle Syntax, um unbenannte Registrierungs Werte anzuzeigen. |
