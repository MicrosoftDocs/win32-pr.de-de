# <a name="mapping-active-directory-to-wmi"></a>Zuordnen von Active Directory zu WMI

Der Verzeichnisdienstanbieter spiegelt Klassen und Instanzen aus Active Directory im LDAP-Namespace (Lightweight Directory Access Protocol) von WMI. Aufgrund grundlegender Unterschiede in den beiden Umgebungen können Sie die Active Directory-Klassen und -Instanzen nicht einfach umbenennen und hoffen, ordnungsgemäß auf diese Klassen und Instanzen in WMI zugreifen zu können. Stattdessen folgt der Verzeichnisdienstanbieter Benennungsregeln, um die Beziehungen zwischen den Active Directory-Klassen und -Instanzen zu erhalten.

Die folgenden Themen enthalten Informationen zum Zuordnen von Active Directory-Klassen und -Instanzen:

-   [Zuordnen von Active Directory-Klassen](mapping-active-directory-classes.md)
-   [Zuordnen von Active Directory-Instanzen](mapping-active-directory-instances.md)

> [!Note]  
> Weitere Informationen zur Unterstützung und Installation dieser Komponente unter einem bestimmten Betriebssystem finden Sie unter [Betriebssystemverfügbarkeit von WMI-Komponenten.](operating-system-availability-of-wmi-components.md)
