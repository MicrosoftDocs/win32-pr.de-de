# <a name="mapping-active-directory-to-wmi"></a>Zuordnung von Active Directory zu WMI

Der Verzeichnisdienst Anbieter spiegelt Klassen und Instanzen von Active Directory in den WMI Lightweight Directory Access Protocol (LDAP)-Namespace. Aufgrund grundlegender Unterschiede in den beiden Umgebungen können Sie nicht einfach die Active Directory Klassen und Instanzen umbenennen und hoffen, ordnungsgemäß auf diese Klassen und Instanzen in WMI zuzugreifen. Stattdessen befolgt der Verzeichnisdienst Anbieter Benennungs Regeln, um die Beziehungen beizubehalten, die zwischen den Active Directory-Klassen und-Instanzen bestehen.

Die folgenden Themen enthalten Informationen zum Mapping von Active Directory Klassen und Instanzen:

-   [Zuordnung von Active Directory Klassen](mapping-active-directory-classes.md)
-   [Zuordnung von Active Directory Instanzen](mapping-active-directory-instances.md)

> [!Note]  
> Weitere Informationen zur Unterstützung und Installation dieser Komponente auf einem bestimmten Betriebssystem finden Sie unter [Betriebssystem Verfügbarkeit von WMI-Komponenten](operating-system-availability-of-wmi-components.md).
