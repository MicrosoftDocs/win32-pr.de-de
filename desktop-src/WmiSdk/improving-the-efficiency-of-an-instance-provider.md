# <a name="improving-the-efficiency-of-an-instance-provider"></a>Verbessern der Effizienz eines Instanzanbieters

Ein WMI-Hochleistungsanbieter erhöht erheblich die Geschwindigkeit, mit der ein WMI-Client Informationen von einem WMI-Instanzanbieter abrufen kann. Die Hauptänderung besteht darin, dass ein Hochleistungsanbieter als In-Process-Server für die Clientanwendung oder WMI ausgeführt wird. Indem Sie den Anbieter im Clientprozess platzieren, können Sie die Interaktion zwischen den beiden beschleunigen.

Weitere Informationen dazu, wie Sie Ihren Instanzanbieter zu einem Hochleistungsanbieter machen, finden Sie unter [Making an Instance Provider into a High-Performance Provider](making-an-instance-provider-into-a-high-performance-provider.md).
