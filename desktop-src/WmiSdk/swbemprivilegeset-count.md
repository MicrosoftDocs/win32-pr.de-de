# <a name="swbemprivilegesetcount-property"></a>Taubemprivilegeset. Count-Eigenschaft

Die **count** -Eigenschaft des " [**Swap Privilege**](swbemprivilege.md) "-Objekts enthält die Anzahl der Elemente in einer " [**taubemprivilegeset**](swbemprivilegeset.md) "-Auflistung. Diese Eigenschaft ist schreibgeschützt.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
SWbemPrivilegeSet.Count As Integer
```
