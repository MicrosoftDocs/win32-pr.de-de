# <a name="trap-type-macro"></a>TRAP-TYPE-Makro

Aus Einheitlichkeitsgehilfen wird jede Instanz des TRAP-TYPE-Makros zunächst einer Instanz des [NOTIFICATION-TYPE-Makros](notification-type-macro.md) mithilfe der in [RFC 1908](https://www.ietf.org/rfc/rfc1908.txt)definierten Zuordnung zugeordnet.

> [!Note]  
> Weitere Informationen zum Installieren des Anbieters finden Sie unter [Einrichten der WMI-SNMP-Umgebung.](setting-up-the-wmi-snmp-environment.md)
