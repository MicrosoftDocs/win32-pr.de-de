# <a name="trap-type-macro"></a>Trap-Type-Makro

Bei der Einheitlichkeit wird jede Instanz des Trap-Type-Makros zunächst mithilfe der in [RFC 1908](https://www.ietf.org/rfc/rfc1908.txt)definierten Zuordnung einer Instanz des Typs " [Notification-Type](notification-type-macro.md) " zugeordnet.

> [!Note]  
> Weitere Informationen zum Installieren des Anbieters finden Sie unter [Einrichten der WMI-SNMP-Umgebung](setting-up-the-wmi-snmp-environment.md).
