# <a name="retrieving-amended-classes"></a>Abrufen von geänderten Klassen

Sie können eine geänderte Klasse abrufen und ändern, indem Sie ein Gebiets Schema angeben und ein Flag festlegen, das angibt, dass lokalisierte Informationen angezeigt werden sollen. Die Prozedur zum Abrufen lokalisierter Klassendefinitionen hängt davon ab, wie Sie eine Verbindung mit WMI herstellen. Weitere Informationen finden Sie unter [Abrufen von geänderten Klassen mithilfe der Skript-API für WMI](retrieving-amended-classes-using-the-scripting-api-for-wmi.md) und [Abrufen von geänderten Klassen mithilfe der com-API für WMI](retrieving-amended-classes-using-the-com-api-for-wmi.md).
