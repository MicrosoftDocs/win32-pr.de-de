# <a name="retrieving-amended-classes"></a>Abrufen geänderter Klassen

Sie können eine geänderte Klasse abrufen und ändern, indem Sie ein -Locale angeben und ein Flag festlegen, das angibt, dass Sie lokalisierte Informationen wünschen. Das Verfahren zum Abrufen lokalisierter Klassendefinitionen hängt davon ab, wie Sie eine Verbindung mit WMI herstellen. Weitere Informationen finden Sie unter Abrufen geänderter Klassen mithilfe der [Skript-API](retrieving-amended-classes-using-the-scripting-api-for-wmi.md) für WMI und Abrufen geänderter Klassen mithilfe der [COM-API für WMI.](retrieving-amended-classes-using-the-com-api-for-wmi.md)
