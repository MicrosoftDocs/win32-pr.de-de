# <a name="swbempropertysetcount-property"></a>Taubempropertyset. Count-Eigenschaft

Verwenden Sie die **count** Automation-Eigenschaft des Objekts " [**Swap PropertySet**](swbempropertyset.md) ", um zu bestimmen, wie viele Elemente in der Sammlung " **Swap PropertySet** " vorliegen. Diese Eigenschaft ist schreibgeschützt.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
SWbemPropertySet.Count As Integer
```
