# <a name="swbemqualifierisamended-property"></a>' Swap. isgeänderten '-Eigenschaft

Die **is-** Eigenschaft des " [**Swap Qualifier**](swbemqualifier.md) "-Objekts ist ein boolescher Wert, der angibt, ob dieser Qualifizierer mit einem Zusammenarbeits Vorgang lokalisiert wurde.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
SWbemQualifier.IsAmended As Boolean
```
