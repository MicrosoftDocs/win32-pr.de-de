# <a name="swbemnamedvalue-object"></a>SWbemNamedValue-Objekt

Das **SWbemNamedValue-Objekt** stellt einen einzelnen benannten Wert dar, der zum [**SWbemNamedValueSet-Auflistungsobjekt**](swbemnamedvalueset.md) gehört. Dieses Objekt kann nicht durch den [**VBScript-CreateObject-Aufruf**](creating-an-object-using-vbscript.md) erstellt werden.

## <a name="members"></a>Member

Das **SWbemNamedValue-Objekt** verfügt über diese Typen von Membern:

-   [Eigenschaften](#properties)

### <a name="properties"></a>Eigenschaften

Das **SWbemNamedValue-Objekt** verfügt über diese Eigenschaften.



| Eigenschaft                                          | Zugriffstyp           | Beschreibung                                                                                    |
|:--------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------|
| [**Name**](swbemnamedvalue-name.md)<br/>   | Schreibgeschützt<br/>  | Name des **SWbemNamedValue-Elements.**<br/>                                               |
| [**Wert**](swbemnamedvalue-value.md)<br/> | Lesen/Schreiben<br/> | Wert des **SWbemNamedValue-Elements.** Dies ist die Standardeigenschaft dieses -Objekts.<br/> |
