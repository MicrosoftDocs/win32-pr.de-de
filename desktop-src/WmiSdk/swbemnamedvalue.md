# <a name="swbemnamedvalue-object"></a>Taubemnamedvalue-Objekt

Das Objekt " **Swap-NamedValue** " stellt einen einzelnen benannten Wert dar, der zum Objekt " [**Swap namedvalueset**](swbemnamedvalueset.md) " gehört. Dieses Objekt kann nicht durch den VBScript-Befehl "up- [**Object**](creating-an-object-using-vbscript.md) " erstellt werden.

## <a name="members"></a>Member

Das Objekt " **taubemnamedvalue** " enthält diese Typen von Membern:

-   [Eigenschaften](#properties)

### <a name="properties"></a>Eigenschaften

Das Objekt " **taubemnamedvalue** " verfügt über diese Eigenschaften.



| Eigenschaft                                          | Zugriffstyp           | BESCHREIBUNG                                                                                    |
|:--------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------|
| [**Name**](swbemnamedvalue-name.md)<br/>   | Schreibgeschützt<br/>  | Der Name des Elements " **Swap Name Value** ".<br/>                                               |
| [**Wert**](swbemnamedvalue-value.md)<br/> | Lesen/Schreiben<br/> | Der Wert des Elements " **Swap NamedValue** ". Dies ist die Standard Eigenschaft dieses Objekts.<br/> |
