# <a name="swbemobjectpathdisplayname-property"></a>Errbemubjectpath. Display Name (Eigenschaft)

Die **DisplayName** -Eigenschaft des-Objekts von " [**errbevibjectpath**](swbemobjectpath.md) " ist eine Zeichenfolge, die den Pfad in einem Formular enthält, das als Moniker-Anzeige Name verwendet werden kann.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
SWbemObjectPath.DisplayName As String
```
