# <a name="whats-new-in-wmi"></a>Neues in WMI

## <a name="new-security-feature-in-windows-7"></a>Neues Sicherheits Feature in Windows 7

Im folgenden finden Sie die neue WMI-Sicherheitsfunktion (Windows-Verwaltungsinstrumentation), die in Windows 7 verfügbar ist.

<dl> <dt>

<span id="Controlling_provider_security"></span><span id="controlling_provider_security"></span><span id="CONTROLLING_PROVIDER_SECURITY"></span>Steuern der Anbieter Sicherheit
</dt> <dd>

Änderungen zur Erhöhung der Sicherheit des WMI-freigegebenen Anbieter Host Prozesses (wmiprvse.exe). Durch diese Änderungen werden drei neue Gruppenrichtlinien und zwei laufende Modi für den freigegebenen WMI-Host eingeführt, die als sicherer und kompatibler Modus bezeichnet werden. Weitere Informationen finden Sie unter [Registrierungsschlüssel zum Steuern der Anbieter Sicherheit](registry-keys-for-controlling-provider-security-.md).

</dd> </dl>

## <a name="new-or-updated-features-in-windows-8"></a>Neue oder aktualisierte Features in Windows 8

Das wichtigste Update für Windows-Verwaltungsinstrumentation war das vollständige Release der WMI-Version der nächsten Generation, die als Windows-Verwaltungsinfrastruktur (Mi) bezeichnet wird. Mi ist vollständig kompatibel mit früheren Versionen von WMI und bietet eine Reihe von Features und Vorteilen, mit denen das Entwerfen und entwickeln von Anbietern und Clients einfacher als je zuvor ist. Weitere Informationen zu Mi finden Sie unter [What es New in Mi?](/previous-versions/windows/desktop/wmi_v2/what-s-new-in-mi)

Weitere Informationen zu den neuen Features für WMI in früheren Betriebssystemversionen finden Sie unter Neuigkeiten [in Windows 7](what-s-new-in-windows-7.md)und [Neuerungen in Windows Vista](what-s-new-in-windows-vista.md).
