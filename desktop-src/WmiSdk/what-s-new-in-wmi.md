# <a name="whats-new-in-wmi"></a>Neues in WMI

## <a name="new-security-feature-in-windows-7"></a>Neues Sicherheitsfeature in Windows 7

Im Folgenden wird die neue WMI-Sicherheitsfunktion (Windows Management Instrumentation) aufgeführt, die in Version 7 Windows ist.

<dl> <dt>

<span id="Controlling_provider_security"></span><span id="controlling_provider_security"></span><span id="CONTROLLING_PROVIDER_SECURITY"></span>Steuern der Anbietersicherheit
</dt> <dd>

Änderungen zur Verbesserung der Sicherheit des WMI-Hostprozesses (shared provider host process, wmiprvse.exe). Diese Änderungen führen drei neue Gruppenrichtlinien und zwei Ausführungsmodi für den freigegebenen WMI-Host ein, die als sicherer und kompatibler Modus bezeichnet werden. Weitere Informationen finden Sie unter [Registrierungsschlüssel zum Steuern der Anbietersicherheit.](registry-keys-for-controlling-provider-security-.md)

</dd> </dl>

## <a name="new-or-updated-features-in-windows-8"></a>Neue oder aktualisierte Features in Windows 8

Das Hauptupdate für Windows Verwaltungsinstrumentation war das vollständige Release der WMI-Version der nächsten Generation, die als Windows Management Infrastructure (MI) bekannt ist. MI ist vollständig kompatibel mit früheren Versionen von WMI und bietet eine Vielzahl von Features und Vorteilen, die das Entwerfen und Entwickeln von Anbietern und Clients einfacher denn je machen. Weitere Informationen zu mi finden Sie unter [What es New in MI?](/previous-versions/windows/desktop/wmi_v2/what-s-new-in-mi)

Weitere Informationen zu neuen Features für WMI in früheren Betriebssystemversionen finden Sie unter Neues [in Windows 7](what-s-new-in-windows-7.md)und Neues [in Windows Vista.](what-s-new-in-windows-vista.md)
