# <a name="swbempropertyname-property"></a>SWbemProperty.Name-Eigenschaft

Die **Name** -Eigenschaft des-Objekts von " [**Swap Property**](swbemproperty.md) " gibt den Namen dieser WMI-Eigenschaft zurück. Diese Eigenschaft ist schreibgeschützt.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
SWbemProperty.Name As String
```
