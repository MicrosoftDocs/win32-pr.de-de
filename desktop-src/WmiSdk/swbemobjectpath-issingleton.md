# <a name="swbemobjectpathissingleton-property"></a>Errbemubjectpath. issingleton (Eigenschaft)

Die **issingleton** -Eigenschaft des [**errbemobjectpath**](swbemobjectpath.md) -Objekts ist ein boolescher Wert, der angibt, ob dieser Pfad eine Singleton-Instanz darstellt. Eine Singleton-Instanz ist eine Instanz einer Klasse, die nie mehr als eine Instanz aufweisen kann. Diese Eigenschaft ist schreibgeschützt.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
SWbemObjectPath.IsSingleton As Boolean
```
