# <a name="swbemobjectpathissingleton-property"></a>SWbemObjectPath.IsSingleton-Eigenschaft

Die **IsSingleton-Eigenschaft** des [**SWbemObjectPath-Objekts**](swbemobjectpath.md) ist ein boolescher Wert, der angibt, ob dieser Pfad eine Singletoninstanz darstellt. Eine Singletoninstanz ist eine Instanz einer Klasse, die nie mehr als eine Instanz haben darf. Diese Eigenschaft ist schreibgeschützt.

Eine Erläuterung dieser Syntax finden Sie unter [Dokumentkonventionen für die Skripterstellungs-API.](document-conventions-for-the-scripting-api.md)

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
SWbemObjectPath.IsSingleton As Boolean
```
