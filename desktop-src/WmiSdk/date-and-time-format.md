# <a name="date-and-time-format"></a>Datums-und Uhrzeit Format

WMI verwendet die Datums-und Uhrzeit Formate, die von der Spezifikation für die verteilte Verwaltung von Task Force ([DMTF.org](https://www.dmtf.org/)) Common Information Model (CIM) definiert werden.

Das **CIM- \_ DateTime** -Format wird in der [MOF-Datei (Microsoft Managed Object Format)](managed-object-format--mof-.md) durch den [DateTime](datetime.md) -MOF-Datentyp implementiert. Die Datums-und Uhrzeit Formate können auch ein Zeitintervall Ausdrücken.

Weitere Informationen zu WMI-Formaten finden Sie unter [CIM \_ DateTime](cim-datetime.md) -und [Interval-Format](interval-format.md).

Weitere Informationen zum Wechseln in das und aus dem WMI-DateTime-Format finden Sie unter " [**austauschen von DateTime**](swbemdatetime.md) " und im technet scriptcenter [-Artikel über die Zeit (Oh und auch über Datumsangaben)](https://www.microsoft.com/technet/technetmag/issues/2006/07/ScriptingGuy/default.aspx).
