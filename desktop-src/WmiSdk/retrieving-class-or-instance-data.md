# <a name="retrieving-wmi-class-or-instance-data"></a>Abrufen von WMI-Klassen-oder-Instanzdaten

Die Hauptdaten Objekte in WMI sind Klassen und Instanzen. In der folgenden Tabelle werden die verschiedenen Techniken aufgelistet, die zum Abrufen von Daten aus einer WMI-Klasse oder-Instanz verfügbar sind.



| Thema                                                                   | BESCHREIBUNG                                                                       |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| [Abrufen einer WMI-Klasse](retrieving-a-class.md)                        | Abrufen von Informationen zu den allgemeinen Parametern eines Objekts.                    |
| [Abrufen einer WMI-Instanz](retrieving-an-instance.md)                 | Abrufen von Informationen, die ein eindeutiges, einzelnes Objekt in Ihrem Unternehmen beschreiben. |
| [Abrufen eines Teils einer WMI-Instanz](retrieving-part-of-an-instance.md) | Abrufen eines oder mehrerer Teile einer Instanz, jedoch nicht der gesamten-Instanz.         |
| [Aufrufen eines WMI-Qualifizierers](accessing-a-qualifier.md)                  | Zugreifen auf Weitere Informationen in einem Qualifizierer.                              |



 

Weitere Informationen finden Sie unter Bearbeiten von [Klassen-und Instanzinformationen](manipulating-class-and-instance-information.md).
