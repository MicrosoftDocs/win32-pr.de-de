# <a name="cim-classes-wmi"></a>CIM-Klassen (WMI)

Die CIM-Klassen (Distributed Management Task Force) Common Information Model (Distributed Management Task Force) sind die übergeordneten Klassen, auf denen WMI-Klassen erstellt werden. In der folgenden Tabelle sind die Kategorien von CIM-Klassen aufgeführt.

> [!IMPORTANT]
> Die CIM-Klassen (Distributed Management Task Force) (DMTF (Distributed Management Task Force) (Common Information Model) sind die übergeordneten Klassen, auf denen WMI-Klassen erstellt werden. WMI unterstützt derzeit nur die [CIM 2.x-Versionsschemas.](https://dmtf.org/standards/cim/schemas) Weitere Informationen zu CIM finden Sie unter [Common Information Model](common-information-model.md).
