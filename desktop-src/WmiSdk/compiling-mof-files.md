# <a name="compiling-mof-files"></a>Kompilieren von MOF-Dateien

Der Managed Object Format (MOF)-Compiler ( [**Mofcomp.exe**](mofcomp.md)) analysiert eine Datei, die MOF-Anweisungen enthält, und fügt die Klassen und Klassen Instanzen, die in der Datei definiert sind, dem WMI-Repository hinzu.

In der folgenden Tabelle sind die Themen aufgeführt, in denen die Verwendung des MOF-Compilers beschrieben wird.

| [Ausführen des MOF-Compilers für eine Datei](running-the-mof-compiler-on-a-file.md)                       | Kompilieren von MOF-Code über die Befehlszeile |
| [Kompilieren von MOF-Code mit Floating-Point Werten](compiling-mof-code-with-floating-point-values.md) | Verarbeiten von Gleit Komma Werten in MOF-Code    |
| [Behandeln von Fehlern mit dem MOF-Compiler](handling-errors-with-the-mof-compiler.md)                 | Umgang mit teilweise kompilierten MOF-Dateien     |
