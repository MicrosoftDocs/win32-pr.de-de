# <a name="swbemdatetimeisinterval-property"></a>' Taubemdatetime. isinterval '-Eigenschaft

Die **isinterval** -Eigenschaft des " [**Swap DateTime**](swbemdatetime.md) "-Objekts ist ein boolescher Wert, der angibt, ob ein Feld in einem CIM-DateTime-Wert als Intervall interpretiert werden soll.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
SWbemDateTime.IsInterval As boolean
```



## <a name="property-value"></a>Eigenschaftswert

## <a name="examples"></a>Beispiele

Beispiele für die Verwendung des Objekts " [**slibemdatetime**](swbemdatetime.md) " zum Konvertieren von CIM-DateTime-Werten in das bzw. aus dem **DateTime** -oder **VT \_ Date** -Format finden Sie unter [WMI-Tasks: Datums-und Uhrzeitangaben](wmi-tasks--dates-and-times.md). Eine Beschreibung des CIM-DateTime-Formats finden Sie unter [Datums-und Uhrzeit Format](date-and-time-format.md).
