# <a name="accessing-other-operating-system-features-with-wmi"></a>Zugreifen auf andere Betriebs System Funktionen mit WMI

WMI verfügt über Erweiterungen oder Anbieter, mit denen Sie auf andere Betriebssystemfunktionen wie Active Directory, ODBC und SNMP zugreifen können. Sie können auch auf Daten von Gerätetreibern zugreifen.

Die WMI-Erweiterungen sind nicht in allen Betriebssystemen verfügbar. Weitere Informationen finden Sie unter [Betriebs System Verfügbarkeit von WMI-Komponenten](operating-system-availability-of-wmi-components.md).

In den folgenden Themen finden Sie Informationen zum Zugriff auf andere Betriebssystem Features:

-   [Zugreifen auf Daten von Gerätetreibern](accessing-data-from-device-drivers.md)
-   [Zugreifen auf Active Directory](accessing-active-directory.md)
-   [Zugreifen auf SNMP-Geräte](accessing-snmp-devices.md)
