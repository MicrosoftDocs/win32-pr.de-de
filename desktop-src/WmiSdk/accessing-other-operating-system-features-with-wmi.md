# <a name="accessing-other-operating-system-features-with-wmi"></a>Zugreifen auf andere Betriebssystemfeatures mit WMI

WMI verfügt über Erweiterungen oder Anbieter, mit denen Sie auf andere Betriebssystemfeatures wie Active Directory, ODBC und SNMP zugreifen können. Sie können auch über Gerätetreiber auf Daten zugreifen.

Die WMI-Erweiterungen sind nicht in allen Betriebssystemen verfügbar. Weitere Informationen finden Sie unter [Betriebssystemverfügbarkeit von WMI-Komponenten.](operating-system-availability-of-wmi-components.md)

Die folgenden Themen enthalten Informationen zum Zugriff auf andere Betriebssystemfeatures:

-   [Zugreifen auf Daten über Gerätetreiber](accessing-data-from-device-drivers.md)
-   [Zugreifen auf Active Directory](accessing-active-directory.md)
-   [Zugreifen auf SNMP-Geräte](accessing-snmp-devices.md)
