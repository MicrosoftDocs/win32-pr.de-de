# <a name="void"></a>Leere

VOID ist ein Datentyp, der das Fehlen eines Werts darstellt. Verwenden Sie VOID in MOF nur, um eine Methode anzugeben, die keinen Rückgabewert hat.

``` syntax
void Method2 ([in] sint32 InParam);
```
