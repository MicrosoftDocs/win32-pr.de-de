# <a name="cooker_range"></a>COOKER \_ RANGE

Die Formel für den COOKER RANGE-Indikatortyp zeigt den Unterschied zwischen den Min- und Max-Werten für einen Satz roher Beobachtungen einer Eigenschaft in einer \_ [**Win32 \_ PerfRawData-Instanz.**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) [](cooker-min.md) [](cooker-max.md) Dieser Indikatortyp wird nur in WMI definiert und ist nicht für die Leistungsüberwachungstechnologien verfügbar, z. B. Leistungsindikatoren [Version 6.0.](/windows/desktop/PerfCtrs/performance-counters-portal)

Weitere Informationen zur Zählertypformel finden Sie unter [Indikatortypen](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Weitere Informationen zu Leistungsstarken Anbietern und Skripts finden Sie unter [WMI-Leistungsindikatortypen.](wmi-performance-counter-types.md)
