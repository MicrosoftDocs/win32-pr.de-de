# <a name="cooker_range"></a>Koch \_ Bereich

Die \_ Formel für den Typ des hashbereichzählers zeigt [](cooker-min.md) den Unterschied zwischen den minimalen und [maximalen](cooker-max.md) Werten für einen Satz von unformatierten Beobachtungen einer Eigenschaft in einer [**Win32 \_ perfrawdata**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) -Instanz. Dieser Indikatortyp wird nur innerhalb von WMI definiert und steht nicht für die Technologien zur Leistungsüberwachung zur Verfügung, wie z. b. [Leistungsindikatoren der Version 6,0](/windows/desktop/PerfCtrs/performance-counters-portal).

Weitere Informationen zur Formel für den Counter-Typ finden Sie unter [counter Types (gegen Typen](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10))).

Weitere Informationen zu Hochleistungs Anbietern und zur Skripterstellung finden Sie unter [WMI-Leistungsdaten Typen](wmi-performance-counter-types.md).
