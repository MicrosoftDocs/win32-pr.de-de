# <a name="managed-reference-for-wmi-windows-powershell-command-classes"></a>Verwaltete Referenz für WMI-Windows PowerShell-Befehls Klassen

Windows PowerShell implementiert Windows-Verwaltungsinstrumentation (WMI)-Funktionalität durch eine Reihe von Cmdlets. Sie können diese Cmdlets verwenden, um die End-to-End-Aufgaben abzuschließen, die zum Verwalten von lokalen Computern und Remote Computern notwendig sind.

Weitere Informationen finden Sie in der [Windows PowerShell](https://msdn.microsoft.com/library/dd835506(v=vs.85).aspx) .

## <a name="wmi-powershell-classes"></a>WMI-PowerShell-Klassen

**Namespace**: Microsoft. PowerShell. Commands

**Assembly**: Microsoft. PowerShell. Commands. Management

**Dll**: Microsoft.PowerShell.Commands.Management.dll

Diese WMI-Befehls Klassen werden von Windows PowerShell implementiert. Diese Klassen sind nur aus Gründen der Vollständigkeit in diesem Software Development Kit (SDK) enthalten. Die Member dieser Klassen können nicht direkt verwendet werden und sollten nicht zum Ableiten anderer Klassen verwendet werden.



| Klasse                   | BESCHREIBUNG                                                                                                                                                                                                                                 |
|-------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Getwmiobjectcommand     | Ruft Instanzen von WMI-Klassen oder Informationen zu den verfügbaren Klassen ab.<br/> Beispiele und ausführliche Informationen zu den Parametern finden Sie im Cmdlet " [Get-WMIObject](/previous-versions//dd315295(v=technet.10)) ".<br/> |
| Invokewmimethod         | Ruft WMI-Methoden auf.<br/> Beispiele und ausführliche Informationen zu den Parametern finden Sie im Cmdlet " [Aufruf-WmiMethod](/previous-versions//dd315300(v=technet.10)) ".<br/>                                                     |
| Registerwmieventcommand | Abonniert ein WMI-Ereignis.<br/> Beispiele und ausführliche Informationen zu den Parametern finden Sie im Cmdlet " [Register-wmievent](/previous-versions//dd315242(v=technet.10)) ".<br/>                                            |
| Removewmiobject         | Löscht eine Instanz einer vorhandenen WMI-Klasse.<br/> Beispiele und ausführliche Informationen zu den Parametern finden Sie im Cmdlet [Remove-WMIObject](/previous-versions//dd347605(v=technet.10)) .<br/>                          |
| -Instanz          | Erstellt oder aktualisiert eine Instanz einer vorhandenen WMI-Klasse.<br/> Beispiele und ausführliche Informationen zu den Parametern finden Sie im Cmdlet " [Set-wmiinstance](/previous-versions//dd315391(v=technet.10)) ".<br/>                |
