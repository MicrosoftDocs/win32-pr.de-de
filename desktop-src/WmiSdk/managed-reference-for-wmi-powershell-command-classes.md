# <a name="managed-reference-for-wmi-windows-powershell-command-classes"></a>Verwaltete Referenz für WMI Windows PowerShell Befehlsklassen

Windows PowerShell implementiert Windows WMI-Funktionalität (Management Instrumentation, Verwaltungsinstrumentation) über eine Reihe von Cmdlets. Sie können diese Cmdlets verwenden, um die End-to-End-Aufgaben auszuführen, die zum Verwalten von lokalen und Remotecomputern erforderlich sind.

Weitere [Windows PowerShell](https://msdn.microsoft.com/library/dd835506(v=vs.85).aspx) finden Sie unter .

## <a name="wmi-powershell-classes"></a>WMI-PowerShell-Klassen

**Namespace:** Microsoft.PowerShell.Commands

**Assembly**: Microsoft.PowerShell.Commands.Management

**DLL:** Microsoft.PowerShell.Commands.Management.dll

Diese WMI-Befehlsklassen werden von Windows PowerShell. Diese Klassen sind nur der Vollständigkeit halber in diesem Software Development Kit (SDK) enthalten. Die Member dieser Klassen können nicht direkt verwendet werden und sollten auch nicht zum Ableiten anderer Klassen verwendet werden.



| Klasse                   | BESCHREIBUNG                                                                                                                                                                                                                                 |
|-------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| GetWmiObjectCommand     | Ruft Instanzen von WMI-Klassen oder Informationen zu den verfügbaren Klassen ab.<br/> Beispiele und ausführliche Informationen zu den Parametern finden Sie im [Cmdlet Get-WmiObject.](/previous-versions//dd315295(v=technet.10))<br/> |
| InvokeWmiMethod         | Ruft WMI-Methoden auf.<br/> Beispiele und ausführliche Informationen zu den Parametern finden Sie im [Invoke-WmiMethod-Cmdlet.](/previous-versions//dd315300(v=technet.10))<br/>                                                     |
| RegisterWmiEventCommand | Abonniert ein WMI-Ereignis.<br/> Beispiele und ausführliche Informationen zu den Parametern finden Sie im [Register-WmiEvent-Cmdlet.](/previous-versions//dd315242(v=technet.10))<br/>                                            |
| RemoveWmiObject         | Löscht eine Instanz einer vorhandenen WMI-Klasse.<br/> Beispiele und ausführliche Informationen zu den Parametern finden Sie im [Remove-WmiObject-Cmdlet.](/previous-versions//dd347605(v=technet.10))<br/>                          |
| SetWmiInstance          | Erstellt oder aktualisiert eine Instanz einer vorhandenen WMI-Klasse.<br/> Beispiele und ausführliche Informationen zu den Parametern finden Sie im Cmdlet [Set-WmiInstance.](/previous-versions//dd315391(v=technet.10))<br/>                |
