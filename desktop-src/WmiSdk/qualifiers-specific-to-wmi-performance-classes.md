# <a name="qualifiers-specific-to-wmi-performance-classes"></a>Für WMI-Leistungsklassen spezifische Qualifizierer

Rohleistungs Indikatoren-Klassen, die von [**Win32 \_ perfrawdata**](/windows/desktop/CIMWin32Prov/win32-perfrawdata)abgeleitet werden, enthalten nicht berechnete Werte, die als Eingabe für [**Win32 \_ perfformatteddata**](/windows/desktop/CIMWin32Prov/win32-perfformatteddata) -Klassen und System Monitor Berechnungen dienen.

Leistungsindikator Qualifizierer geben Informationen über das Leistungs Objekt und die Leistungsindikatoren für das Objekt an. Leistungs Objekte werden WMI- [Leistungsindikator Klassen](/windows/desktop/CIMWin32Prov/performance-counter-classes) zugeordnet, und Leistungsindikatoren werden durch die Eigenschaften der WMI-Klasse dargestellt.

-   [Klassen Qualifizierer für Leistungs Leistungsdaten Klassen](class-qualifiers-for-performance-counter-classes.md)
-   [Eigenschafts Qualifizierer für formatierte Leistungs Objektklassen](property-qualifiers-for-performance-counter-classes.md)
