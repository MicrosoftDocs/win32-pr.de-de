# <a name="swbemprivilegedisplayname-property"></a>Taubemprivilege. Display Name (Eigenschaft)

Die **DisplayName** -Eigenschaft eines ' [**taubemprivilege**](swbemprivilege.md) '-Objekts ist eine Zeichenfolge, die eine Beschreibung eines ' **taubemprivilege** '-Objekts anzeigt. Beispielsweise hat die Berechtigung, die steuert, ob ein Benutzer die Shutdown-Methode eines Objekts ausführen kann, die Zeichenfolge "System beenden" in der Eigenschaft " **Swap-Privileg. Display Name** ". Diese Eigenschaft ist schreibgeschützt.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Diese Eigenschaft ist schreibgeschützt.

```VB
SWbemPrivilege.DisplayName As 
```
