# <a name="swbemobjectpathserver-property"></a>SWbemObjectPath.Server (Eigenschaft)

Die **Server-Eigenschaft** des [**SWbemObjectPath-Objekts**](swbemobjectpath.md) enthält den Namen des Servers, der Teil des Objektpfads ist.

Eine Erläuterung dieser Syntax finden Sie unter [Dokumentkonventionen für die Skripterstellungs-API.](document-conventions-for-the-scripting-api.md)

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
SWbemObjectPath.Server As String
```
