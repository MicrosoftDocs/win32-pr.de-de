# <a name="swbemobjectpathserver-property"></a>Errbemubjectpath. Server-Eigenschaft

Die **Server** -Eigenschaft des " [**errbemubjectpath**](swbemobjectpath.md) "-Objekts enthält den Namen des Servers, der Teil des Objekt Pfads ist.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
SWbemObjectPath.Server As String
```
