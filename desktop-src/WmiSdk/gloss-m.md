# <a name="m-wmi"></a>M (WMI)

[a](gloss-a.md) B [C](gloss-c.md) [D](gloss-d.md) [E](gloss-e.md) [F](gloss-f.md) G [H](gloss-h.md) [I](gloss-i.md) J [K](gloss-k.md) [L](gloss-l.md) M [N](gloss-n.md) [O](gloss-o.md) [P](gloss-p.md) [Q](gloss-q.md) [R](gloss-r.md) [S](gloss-s.md) [T](gloss-t.md) U V [W](gloss-w.md) X Y Z

<dl> <dt>

<span id="wmi.gloss_managed_object"></span><span id="WMI.GLOSS_MANAGED_OBJECT"></span>**verwaltetes Objekt**
</dt> <dd>

Eine Hardware-oder Softwaresystem Komponente, die als Instanz einer WMI-Klasse dargestellt wird.

</dd> <dt>

<span id="wmi.gloss_managed_object_format"></span><span id="WMI.GLOSS_MANAGED_OBJECT_FORMAT"></span>**Managed Object Format (MOF)**
</dt> <dd>

Eine auf IDL (Interface Definition Language) basierende Sprache, die Verwaltungsinformationen beschreibt.

</dd> <dt>

<span id="wmi.gloss_management_information_base"></span><span id="WMI.GLOSS_MANAGEMENT_INFORMATION_BASE"></span>**Management Information Base (MIB)**
</dt> <dd>

Eine Datenbank, in der eine Auflistung von Darstellungen von Netzwerk Objekten gespeichert wird.

</dd> <dt>

<span id="wmi.gloss_metamodel"></span><span id="WMI.GLOSS_METAMODEL"></span>**MetaModel**
</dt> <dd>

Eine CIM-Komponente, die die Entitäten und Beziehungen beschreibt, die *verwaltete Objekte* darstellen. Beispielsweise sind Klassen, Instanzen und Zuordnungen im MetaModel enthalten.

</dd> <dt>

<span id="wmi.gloss_metaschema"></span><span id="WMI.GLOSS_METASCHEMA"></span>**metaschema**
</dt> <dd>

Das [*Schema*](gloss-s.md) des *MetaModel*.

</dd> <dt>

<span id="wmi.gloss_method_provider"></span><span id="WMI.GLOSS_METHOD_PROVIDER"></span>**Methoden Anbieter**
</dt> <dd>

Ein Anbieter, der den WMI-Zugriff auf die Methoden einer Klasse ermöglicht. Siehe auch [Schreiben eines Methoden Anbieters](writing-a-method-provider.md).

</dd> <dt>

<span id="wmi.gloss_mib"></span><span id="WMI.GLOSS_MIB"></span>**MIB**
</dt> <dd>

Weitere Informationen finden Sie unter *Management Information Base (MIB)*.

</dd> <dt>

<span id="wmi.gloss_mof"></span><span id="WMI.GLOSS_MOF"></span>**MOF**
</dt> <dd>

Weitere Informationen finden Sie *unter Managed Object Format (MOF)*.

</dd> <dt>

<span id="wmi.gloss_mof_file"></span><span id="WMI.GLOSS_MOF_FILE"></span>**MOF-Datei**
</dt> <dd>

Eine Textdatei, die Definitionen von Klassen und Instanzen enthält, die mit der *Managed Object Format (MOF)* -Sprache angegeben werden.

</dd> </dl>
