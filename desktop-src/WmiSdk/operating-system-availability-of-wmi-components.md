# <a name="operating-system-availability-of-wmi-components"></a>Betriebs System Verfügbarkeit von WMI-Komponenten

WMI wird standardmäßig auf allen Windows-Desktop-und-Serverplattformen installiert. Bestimmte WMI-Anbieter sind abhängig von der Betriebssystemversion und-Konfiguration möglicherweise aber nicht installiert. Der SNMP-Anbieter ist beispielsweise standardmäßig nicht aktiviert, und der Windows Installer Anbieter (MSI-Anbieter) wird nicht standardmäßig auf 64-Bit-Betriebssystemen installiert. Weitere Informationen zur Verfügbarkeit eines bestimmten Anbieters finden Sie unter [WMI-Anbieter](/windows/desktop/SrvNodes/wmi-mi-omi-providers).
