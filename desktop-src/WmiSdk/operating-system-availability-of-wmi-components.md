# <a name="operating-system-availability-of-wmi-components"></a>Betriebssystemverfügbarkeit von WMI-Komponenten

WMI wird standardmäßig auf allen Windows Desktop- und Serverplattformen installiert. Je nach Betriebssystemversion und -konfiguration können jedoch bestimmte WMI-Anbieter installiert oder nicht installiert werden. Beispielsweise ist der SNMP-Anbieter nicht standardmäßig aktiviert, und der Windows Installer-Anbieter (MSI-Anbieter) ist auf 64-Bit-Betriebssystemen nicht standardmäßig installiert. Weitere Informationen zur Verfügbarkeit eines bestimmten Anbieters finden Sie unter [WMI-Anbieter.](/windows/desktop/SrvNodes/wmi-mi-omi-providers)
