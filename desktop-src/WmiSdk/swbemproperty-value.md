# <a name="swbempropertyvalue-property"></a>Taubemproperty. Value-Eigenschaft

Die **value** -Eigenschaft des " [**Swap Property**](swbemproperty.md) "-Objekts definiert den Variant-Wert der WMI-Eigenschaft. Dies ist die standardmäßige Automatisierungs Eigenschaft dieses Objekts.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
SWbemProperty.Value As Variant
```
