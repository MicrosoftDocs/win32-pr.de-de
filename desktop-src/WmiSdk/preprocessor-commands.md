# <a name="preprocessor-commands"></a>Präprozessorbefehle

Ein Präprozessorbefehl ist eine Codezeile, die in einer MOF-Datei platziert wird und steuert, wie der MOF-Compiler Code kompiliert. Es gibt zwei Präprozessorbefehle, die beim Kompilieren von MOF-Dateien verwendet werden:

-   [\#darunter](-include.md)
-   [\#pragma](-pragma.md)
