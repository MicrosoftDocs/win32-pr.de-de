# <a name="swbemrefreshableitemobject-property"></a>' Swap. Object '-Eigenschaft

Die " **errbemerfrischendes ableitem. Object** "-Eigenschaft stellt [**eine einzelne,**](swbemobject.md) aktualisierte Metadateninstanz dar. Das Element ist in einem " [**Swap**](swbemrefresher.md) -Aktualisierungs"-Objekt enthalten.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
SWbemRefreshableItem.Object As Object
```



## <a name="property-value"></a>Eigenschaftswert

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist **null** , es sei denn, der Wert für " [**Swap. isset**](swbemrefreshableitem-isset.md) " ist **falsch**.
