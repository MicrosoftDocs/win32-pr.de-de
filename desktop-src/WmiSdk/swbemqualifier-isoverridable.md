# <a name="swbemqualifierisoverridable-property"></a>SWbemQualifier.IsOverridable-Eigenschaft

Die **IsOverridable-Eigenschaft** des [**SWbemQualifier-Objekts**](swbemqualifier.md) ist ein boolescher Wert, der angibt, ob dieser Qualifizierer überschrieben werden kann, wenn er weitergegeben wird.

Eine Erläuterung dieser Syntax finden Sie unter [Dokumentkonventionen für die Skripterstellungs-API.](document-conventions-for-the-scripting-api.md)

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
SWbemQualifier.IsOverridable As Boolean
```
