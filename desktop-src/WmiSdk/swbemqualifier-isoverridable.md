# <a name="swbemqualifierisoverridable-property"></a>"Taubemqualifizierer. isoverridable"-Eigenschaft

Die **isoverridable** -Eigenschaft des [**austaubemqualifier**](swbemqualifier.md) -Objekts ist ein boolescher Wert, der angibt, ob dieser Qualifizierer bei der Weitergabe überschrieben werden kann.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
SWbemQualifier.IsOverridable As Boolean
```
