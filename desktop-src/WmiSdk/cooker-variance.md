# <a name="cooker_variance"></a>COOKER \_ VARIANCE

Die \_ Cooker VARIANCE-Indikatortypformel bietet Variabilität, die verwendet, um Diens für einen Satz roher Beobachtungen einer Eigenschaft in einer [**Win32 \_ PerfRawData-Instanz**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) zu charakterisieren. Die Varianz wird berechnet, indem die Summe des Quadrats der Abweichung vom Mittelwert jedes Zählers durch die Anzahl der Indikatoren dividiert wird. Anders ausgedrückt: Die Varianz ist der Durchschnitt der quadratischen Abweichungen vom Mittelwert für jeden Indikator. Dieser Indikatortyp wird nur innerhalb von WMI definiert und ist nicht für die Leistungsüberwachungstechnologien verfügbar, z. B. [Leistungsindikatoren, Version 6.0.](/windows/desktop/PerfCtrs/performance-counters-portal)

Weitere Informationen zur Indikatortypformel finden Sie unter [Indikatortypen.](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10))

Weitere Informationen zu Hochleistungsanbietern und Skripts finden Sie unter [WMI-Leistungsindikatortypen.](wmi-performance-counter-types.md)
