# <a name="cooker_variance"></a>Koch \_ Varianz

Die \_ Formel für den Typ des kochvarianz Zählers bietet Variabilität, mit der die Dispersion für eine Reihe von unformatierten Beobachtungen einer Eigenschaft in einer [**Win32 \_ perfrawdata**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) -Instanz charakterisiert wird. Die Varianz wird berechnet, indem die Summe des Quadrats der Abweichung vom Mittelwert der einzelnen Zähler um die Anzahl der Zähler dividiert wird. Das heißt, die Varianz ist der Durchschnitt der quadratischen Abweichungen vom Mittelwert für jeden Zählers. Dieser Indikatortyp wird nur innerhalb von WMI definiert und steht nicht für die Technologien zur Leistungsüberwachung zur Verfügung, wie z. b. [Leistungsindikatoren der Version 6,0](/windows/desktop/PerfCtrs/performance-counters-portal).

Weitere Informationen zur Formel für den Counter-Typ finden Sie unter [counter Types (gegen Typen](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10))).

Weitere Informationen zu Hochleistungs Anbietern und zur Skripterstellung finden Sie unter [WMI-Leistungsdaten Typen](wmi-performance-counter-types.md).
