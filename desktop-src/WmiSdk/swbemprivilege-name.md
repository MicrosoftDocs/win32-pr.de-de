# <a name="swbemprivilegename-property"></a>SWbemPrivilege.Name-Eigenschaft

Die **Name** -Eigenschaft eines " [**taubemprivilege**](swbemprivilege.md) "-Objekts ist eine Zeichenfolge, die eine Berechtigung eindeutig beschreibt. Beispielsweise hat die Berechtigung, die steuert, ob ein Benutzer die Shutdown-Methode eines Objekts ausführen kann, die Zeichenfolge "SeShutdownPrivilege" in der **SWbemPrivilege.Name** -Eigenschaft. Diese Eigenschaft ist schreibgeschützt.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Diese Eigenschaft ist schreibgeschützt.

```VB
SWbemPrivilege.Name As 
```
