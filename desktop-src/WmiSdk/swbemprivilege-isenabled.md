# <a name="swbemprivilegeisenabled-property"></a>SWbemPrivilege.IsEnabled-Eigenschaft

Die **IsEnabled-Eigenschaft** eines [**SWbemPrivilege-Objekts**](swbemprivilege.md) ist ein boolescher Wert, den Sie zum Aktivieren oder Deaktivieren dieser Berechtigung verwenden. Weitere Informationen zu den Konsequenzen der Aktivierung bestimmter Berechtigungen finden Sie unter [**Ausführen mit speziellen Berechtigungen.**](swbemsecurity-privileges.md)

Eine Erläuterung dieser Syntax finden Sie unter [Dokumentkonventionen für die Skripterstellungs-API.](document-conventions-for-the-scripting-api.md)

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
SWbemPrivilege.IsEnabled As Boolean
```
