# <a name="swbemprivilegeisenabled-property"></a>Taubemprivilege. isaktivierte Eigenschaft

Die **isenable** -Eigenschaft eines ' [**taubemprivilege**](swbemprivilege.md) '-Objekts ist ein boolescher Wert, den Sie verwenden, um diese Berechtigung zu aktivieren oder zu deaktivieren. Weitere Informationen zu den Auswirkungen der Aktivierung bestimmter Berechtigungen finden Sie unter [**Ausführen mit besonderen Berechtigungen**](swbemsecurity-privileges.md).

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
SWbemPrivilege.IsEnabled As Boolean
```
