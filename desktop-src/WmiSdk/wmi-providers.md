# <a name="wmi-providers"></a>WMI-Anbieter

In der folgenden Tabelle sind die WMI-Anbieter des Betriebssystems aufgeführt. Andere Anbieter und deren Klassen können auf Ihrem Computer vorhanden sein, die von Microsoft-Produkten oder Drittanbietersoftware installiert werden. Weitere Informationen zum Schreiben ihres eigenen Anbieters finden Sie unter [Verwenden von WMI.](using-wmi.md)

Die WMI-Anbieter senden Fehler- und Warnmeldungen an das Ereignisprotokoll. Weitere Informationen finden Sie unter [**WMI-Anbietermeldungen.**](wmi-event-messages.md)
