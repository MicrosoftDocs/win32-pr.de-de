# <a name="wmi-providers"></a>WMI-Anbieter

In der folgenden Tabelle werden die WMI-Anbieter des Betriebssystems aufgelistet. Andere Anbieter und Ihre Klassen sind möglicherweise auf dem Computer vorhanden, die von Microsoft-Produkten oder Drittanbieter Software installiert werden. Weitere Informationen zum Schreiben eines eigenen Anbieters finden Sie unter [Verwenden von WMI](using-wmi.md).

Die WMI-Anbieter senden Fehler-und Warnmeldungen an das Ereignisprotokoll. Weitere Informationen finden Sie unter [**WMI-Anbieter Nachrichten**](wmi-event-messages.md).
