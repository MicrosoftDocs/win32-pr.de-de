# <a name="swbemservicessecurity_-property"></a>' Swap Services. Security '- \_ Eigenschaft

Die **Security \_** -Eigenschaft des " [**Swap Services**](swbemservices.md) "-Objekts wird verwendet, um die Sicherheitseinstellungen für ein " **taubemservices** "-Objekt zu erhalten oder festzulegen. Diese Eigenschaft ist ein [**Swap Security**](swbemsecurity.md) -Objekt.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
SWbemServices.Security_ As Object
```
