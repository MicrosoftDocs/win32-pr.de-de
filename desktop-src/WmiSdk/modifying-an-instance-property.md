# <a name="modifying-an-instance-property"></a>Ändern einer Instanzeigenschaft

Nachdem Sie eine Instanz einer Klasse erstellt oder gefunden haben, möchten Sie die Instanz wahrscheinlich ändern. Beispielsweise möchten Sie möglicherweise den Namen des System namens ändern, der in der lokalen Instanz der Win32- [**\_ Computersystem**](/windows/desktop/CIMWin32Prov/win32-computersystem) -Klasse gespeichert ist. Die gängigste Methode zum Aktualisieren einer-Instanz besteht darin, die gesamte-Instanz zu aktualisieren. Weitere Informationen finden Sie unter [Aktualisieren einer gesamten Instanz](updating-an-entire-instance.md). Die zweite Option besteht darin, nur einen Teil der-Instanz zu aktualisieren. Weitere Informationen finden Sie unter [Aktualisieren eines Teils einer Instanz](updating-part-of-an-instance.md).

Weitere Informationen zum Arbeiten mit Klassen finden Sie unter Bearbeiten von [Klassen-und Instanzinformationen](manipulating-class-and-instance-information.md).
