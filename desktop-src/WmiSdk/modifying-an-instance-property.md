# <a name="modifying-an-instance-property"></a>Ändern einer Instanzeigenschaft

Nachdem Sie eine Instanz einer Klasse erstellt oder gefunden haben, möchten Sie die Instanz wahrscheinlich ändern. Beispielsweise können Sie den Namen des Systemnamens ändern, der in der lokalen Instanz der [**Win32 \_ ComputerSystem-Klasse gespeichert**](/windows/desktop/CIMWin32Prov/win32-computersystem) ist. Die gängigste Methode zum Aktualisieren einer Instanz ist das Aktualisieren der gesamten Instanz. Weitere Informationen finden Sie unter [Aktualisieren einer gesamten Instanz.](updating-an-entire-instance.md) Die zweite Option besteht im Aktualisieren nur eines Teils der -Instanz. Weitere Informationen finden Sie unter [Aktualisieren eines Teils einer Instanz.](updating-part-of-an-instance.md)

Weitere Informationen zum Arbeiten mit Klassen finden Sie unter [Manipulating Class and Instance Information](manipulating-class-and-instance-information.md).
