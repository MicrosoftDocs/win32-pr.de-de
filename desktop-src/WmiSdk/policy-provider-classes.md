# <a name="policy-provider-classes"></a>Richtlinien Anbieter Klassen

Der WMI-Richtlinien Anbieter stellt Erweiterungen für die Gruppenrichtlinie durch Klassen bereit und ermöglicht Verfeinerungen in der Anwendung der Richtlinie. Die Richtlinie wird mit dem WMI Query Language (WQL) festgelegt und befindet sich im Active Directory für eine einfache Bereitstellung.

In der folgenden Tabelle werden die Klassen für den WMI-Richtlinien Anbieter aufgelistet.



| Klasse                                               | BESCHREIBUNG                                                       |
|-----------------------------------------------------|-------------------------------------------------------------------|
| [**MSFT- \_ Anbieter**](/previous-versions/windows/desktop/wmisystemprov/msft-providers) | Macht eine Konfiguration für Anbieter Instanzen verfügbar.             |
| [**MSFT- \_ Regel**](/previous-versions/windows/desktop/policmanprov/msft-rule)                | Definiert eine einzelne Regel innerhalb eines Verwaltungsbereichs (SOM).         |
| [**MSFT \_ somfilter**](/previous-versions/windows/desktop/policmanprov/msft-somfilter)      | Stellt eine Liste von Regeln bereit, die auf einem Zielcomputer ausgewertet werden. |
