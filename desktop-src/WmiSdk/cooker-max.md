# <a name="cooker_max"></a>COOKER \_ MAX

Die Formel für den COOKER MAX-Indikatortyp identifiziert den größten Wert aus einer Reihe von Beobachtungen einer Eigenschaft in einer \_ [**Win32 \_ PerfRawData-Instanz.**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) Dieser berechnete Höchstwert wird in einer Eigenschaft in einer [**Win32 \_ PerfFormattedData-Instanz**](/windows/desktop/CIMWin32Prov/win32-perfformatteddata) platziert. Dieser Indikatortyp wird nur in WMI definiert und ist nicht für die Leistungsüberwachungstechnologien verfügbar, z. B. Leistungsindikatoren [Version 6.0.](/windows/desktop/PerfCtrs/performance-counters-portal)

Weitere Informationen zur Zählertypformel finden Sie unter [Indikatortypen](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Weitere Informationen zu Leistungsstarken Anbietern und Skripts finden Sie unter [WMI-Leistungsindikatortypen.](wmi-performance-counter-types.md)
