# <a name="the-performance-monitoring-provider"></a>Der Leistungsüberwachungsanbieter

Daten aus Leistungsbibliotheken sind als [WMI-Leistungsindikatorklassen](/windows/desktop/CIMWin32Prov/performance-counter-classes) über den [WMIPerfClass-Anbieter](wmiperfclass-provider.md) und den [WMIPerfInst-Anbieter](wmiperfinst-provider.md)verfügbar.
