# <a name="datetime"></a>DATETIME

Der DATETIME-Datentyp ist eine \_ VT-BSTR-Zeichenfolge fester Länge, die entweder ein bestimmtes Datum und eine bestimmte Uhrzeit oder ein bestimmtes Intervall darstellt. Weitere Informationen zu DATETIME als Datum finden Sie unter [Datums- und Uhrzeitformat.](date-and-time-format.md) Weitere Informationen zu DATETIME als Intervall finden Sie unter [Interval Format](interval-format.md).
