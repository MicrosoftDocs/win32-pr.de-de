# <a name="creating-an-instance"></a>Erstellen einer Instanz

Eine Instanz ist ein Fall einer Klasse, die ein bestimmtes Objekt in Ihrem Unternehmen beschreibt. Wenn Sie z. b. über zwei CD-ROM-Laufwerke in einer Desktop Plattform verfügen, können Sie zwei Instanzen der [**Win32 \_ cdromdrive**](/windows/desktop/CIMWin32Prov/win32-cdromdrive) -Klasse verwenden, um beide zu beschreiben. Da eine Instanz eine echte Eigenschaft Ihres Unternehmens darstellt, werden höchstwahrscheinlich mehr Instanzen als Klassen erstellt. Jede Klasse, von der eine Instanz erstellt wird, verfügt über eine oder mehrere Schlüsseleigenschaften. Eine Instanz mit mehr als 256 Schlüsseln kann nicht erstellt werden.

Weitere Informationen zum Erstellen von-Instanzen finden Sie in den folgenden Themen:

-   [Erstellen und Deklarieren einer Instanz mithilfe von C++](creating-and-declaring-an-instance-using-c-.md)
-   [Erstellen einer Instanz mit MOF](creating-an-instance-using-mof.md)
-   [Erstellen eines Objekts mithilfe von VBScript](creating-an-object-using-vbscript.md)
