# <a name="creating-an-instance"></a>Erstellen einer Instanz

Eine -Instanz ist ein Fall einer Klasse, die ein bestimmtes Objekt in Ihrem Unternehmen beschreibt. Wenn Sie z. B. über zwei CD-ROM-Laufwerke auf einer Desktopplattform verfügen, können Sie zwei Instanzen der [**Win32 \_ C CLOUDDrive-Klasse**](/windows/desktop/CIMWin32Prov/win32-cdromdrive) verwenden, um beide zu beschreiben. Da eine Instanz eine echte Eigenschaft Ihres Unternehmens darstellt, erstellen Sie wahrscheinlich mehr Instanzen als Klassen. Jede Klasse, aus der eine Instanz erstellt wird, verfügt über mindestens eine Schlüsseleigenschaften. Sie können keine Instanz mit mehr als 256 Schlüsseln erstellen.

Weitere Informationen zum Erstellen von Instanzen finden Sie in den folgenden Themen:

-   [Erstellen und Deklarieren einer Instanz mit C++](creating-and-declaring-an-instance-using-c-.md)
-   [Erstellen einer Instanz mit MOF](creating-an-instance-using-mof.md)
-   [Erstellen eines Objekts mit VBScript](creating-an-object-using-vbscript.md)
