# <a name="swbemnamedvaluename-property"></a>SWbemNamedValue.Name-Eigenschaft

Die **Name** -Eigenschaft des " [**Swap**](swbemnamedvalue.md) Name"-Objekts ist eine eindeutige Zeichenfolge ohne Beachtung der Groß-/Kleinschreibung, die beim erhalten eines " **Swap NamedValue** "-Elements verwendet wird. Diese Eigenschaft ist schreibgeschützt.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
SWbemNamedValue.Name As String
```
