# <a name="cooker_average"></a>COOKER \_ AVERAGE

Die \_ Cooker Average-Indikatortypformel stellt den statistischen Mittelpunkt der Daten für einen Indikator bereit. Die Formel für diesen Indikatortyp summiert wiederholte Beobachtungen einer Eigenschaft in einer [**Win32 \_ PerfRawData-Instanz**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) und dividiert die Summe durch die Anzahl der Beobachtungen. Dieser Indikatortyp wird nur innerhalb von WMI definiert und ist nicht für die Leistungsüberwachungstechnologien verfügbar, z. B. [Leistungsindikatoren, Version 6.0.](/windows/desktop/PerfCtrs/performance-counters-portal)

Weitere Informationen zur Indikatortypformel finden Sie unter [Indikatortypen.](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10))

Weitere Informationen zu WMI-Hochleistungsanbietern und Skripts finden Sie unter [WMI-Leistungsindikatortypen.](wmi-performance-counter-types.md)
