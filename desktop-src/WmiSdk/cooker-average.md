# <a name="cooker_average"></a>\_kochmittel Wert

Die Formel für den Mittelwert des Mittelwerts \_ für den kochtyp gibt den statistischen Mittelpunkt der Daten eines Zählers an. Die Formel für diesen zähtertyp fasst die wiederholten Beobachtungen einer Eigenschaft in einer [**Win32 \_ perfrawdata**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) -Instanz zusammen und dividiert die Summe durch die Anzahl von Beobachtungen. Dieser Indikatortyp wird nur innerhalb von WMI definiert und steht nicht für die Technologien zur Leistungsüberwachung zur Verfügung, wie z. b. [Leistungsindikatoren der Version 6,0](/windows/desktop/PerfCtrs/performance-counters-portal).

Weitere Informationen zur Formel für den Counter-Typ finden Sie unter [counter Types (gegen Typen](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10))).

Weitere Informationen zu Hochleistungs Anbietern und zur Skripterstellung für WMI finden Sie unter [WMI-Leistungsdaten Typen](wmi-performance-counter-types.md).
