# <a name="cim-schema-compatibility"></a>CIM-Schema Kompatibilität

Eine Schlüsselkomponente der WMI-Infrastruktur (Windows-Verwaltungsinstrumentation) ist ein objektorientiertes Modell der verwaltbaren Entitäten in einem System. Das Modell entspricht einem Standard, der vom Desktop Management Task Force ([DMTF](https://www.dmtf.org/standards/wsman)) verwaltet wird, und wird als Common Information Model (CIM) bezeichnet. Das CIM-Schema stellt einen einzelnen Daten Beschreibungs Mechanismus für beliebige Daten bereit, die Sie bereitstellen.

Ab Windows 7 ist WMI mit der CIM-Schema Version 2.17.1 () kompatibel [https://www.dmtf.org/standards/cim/cim\_schema\_v2171/](https://www.dmtf.org/standards/cim/cim_schema_v2171) . Benutzer können jetzt ein DMTF-definiertes Schema auf einem Windows-Computer kompilieren.

## <a name="benefits-of-cim-schema-version-2171-compatibility"></a>Vorteile der CIM-Schema Version 2.17.1 Kompatibilität

-   Wenn ein Benutzer CIM-Schema Version 2.17.1 oder DMTF-MOF-Dateien herunterlädt, kann er auf dem Windows-Zielcomputer erfolgreich kompiliert werden.
-   Die CIM-Schema Version 2.17.1 zusätzliche Unterstützung für BIOS, Drucker, Standard Meldungen, Betriebssystem Status, moderne Energie Verwaltungs Paradigmen, Virtualisierung und Sicherheit.
-   CIM-Schema Version 2.1.7.1 bietet zusätzliche Unterstützung für Profile. Weitere Informationen finden Sie unter [Cross Namespace Association Traversal](cross-namespace-association-traversal.md) .
