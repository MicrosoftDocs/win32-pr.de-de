# <a name="cim-schema-compatibility"></a>CIM-Schemakompatibilität

Eine wichtige Komponente der WMI-Infrastruktur (Windows Management Instrumentation) ist ein objektorientiertes Modell der verwaltbaren Entitäten in einem System. Das Modell entspricht einem Standard, der von der Desktop Management Task Force[(DMTF)](https://www.dmtf.org/standards/wsman)verwaltet wird und als Common Information Model (CIM) bekannt ist. Das CIM-Schema bietet einen einzelnen Datenbeschreibungsmechanismus für alle daten, die sie bereitstellen.

Ab Version Windows 7 ist WMI mit der CIM-Schemaversion 2.17.1 ( ) [https://www.dmtf.org/standards/cim/cim\_schema\_v2171/](https://www.dmtf.org/standards/cim/cim_schema_v2171) kompatibel. Benutzer können jetzt ein DMTF-definiertes Schema auf einem Windows kompilieren.

## <a name="benefits-of-cim-schema-version-2171-compatibility"></a>Vorteile der Kompatibilität mit CIM-Schema, Version 2.17.1

-   Wenn ein Benutzer CIM-Schema,Version 2.17.1 oder DMTF MOF-Dateien herunterlädt, kann er erfolgreich auf dem Zielcomputer Windows werden.
-   Die CIM-Schemaversion 2.17.1 hat Unterstützung für BIOS, Drucker, Standardmeldungen, Betriebssystemstatus, moderne Paradigmen für die Energieverwaltung, Virtualisierung und Sicherheit hinzugefügt.
-   Cim Schema Version 2.1.7.1 bietet zusätzliche Profilunterstützung. Weitere Informationen finden Sie unter [Namespaceübergreifender Zuordnungsdurchlauf.](cross-namespace-association-traversal.md)
