# <a name="__securityrelatedclass-class"></a>\_\_Securityrelatedclass-Klasse

Die abstrakte System Klasse **\_ \_ securityrelatedclass** dient als übergeordnete Klasse für alle Arten von Sicherheitsklassen. Es werden nie Instanzen dieser Klasse erstellt, und alle Sicherheitsklassen müssen von dieser Basisklasse abgeleitet werden.

Die folgende Syntax wird durch MOF-Code (Managed Object Format) vereinfacht und schließt alle geerbten Eigenschaften ein. Eigenschaften werden in alphabetischer Reihenfolge und nicht in der MOF-Reihenfolge aufgelistet.

## <a name="syntax"></a>Syntax

``` syntax
[abstract]
class __SecurityRelatedClass
{
};
```

## <a name="members"></a>Member

Die **\_ \_ securityrelatedclass** -Klasse definiert keine Member.
