CIM \_ Studio und andere WMI-Verwaltungs Tools können von [WMI-Verwaltungs Tools](https://www.microsoft.com/downloads/details.aspx?FamilyId=6430F853-1120-48DB-8CC5-F2ABDC3ED314)heruntergeladen werden.

Die WMI-Tools umfassen Folgendes:

-   WMI CIM Studio

    Das CIM Studio-Tool ermöglicht Ihnen die Arbeit mit WMI-Klassen und-Namespaces im CIM-Repository. Es ist auch ein wichtiges Tool für die Entwicklung von WMI-Anbietern, da es bei der anfänglichen Erstellung von WMI-Klassen im CIM-Repository hilfreich ist.

    WMI CIM Studio bietet folgende Möglichkeiten:

    -   Stellen Sie eine Verbindung mit einem ausgewählten System her, und Durchsuchen Sie das CIM-Repository in einem beliebigen Namespace
    -   Suchen Sie nach Klassen anhand ihres Namens, ihrer Beschreibungen oder Eigenschaftsnamen.
    -   Überprüfen Sie die Eigenschaften, Methoden und Zuordnungen, die mit einer bestimmten Klasse verknüpft sind.
    -   Weitere Informationen finden Sie in den verfügbaren Instanzen für eine bestimmte Klasse.
    -   Führen Sie Abfragen in der WQL-Sprache aus.
    -   Generiert eine MOF-Datei, die auf den ausgewählten klassenbasiert.
    -   Kompilieren Sie eine MOF-Datei, die Sie laden möchten, in das CIM-Repository.

-   WMI-Objektkatalog

    Mit der Objektkatalog können Sie die WMI-Objekte anzeigen, die auf Ihrem Computer vorhanden sind. Der Stamm der Hierarchie ist die Instanz des [**Win32 \_**](/windows/desktop/CIMWin32Prov/win32-computersystem) -Computer Systems, das das Computersystem darstellt. Mithilfe der Objektkatalog können Sie Eigenschaftswerte und Qualifizierer bearbeiten und Methoden ausführen.

-   WMI-Ereignis Registrierung

    Das Ereignis Registrierungs Tool ermöglicht Ihnen das Erstellen, anzeigen und Ändern von Ereignisconsumern, Filtern und Timern für einen angegebenen Namespace. Es ist ein Tool für Entwickler und Systemadministratoren.

-   WMI-Ereignisanzeige

    Sie können den Ereignisanzeige verwenden, um das Vorkommen von WMI-Ereignissen anzuzeigen, die Sie in der WMI-Ereignis Registrierung konfiguriert haben.

Sie können PowerShell RC1 und die Dokumentation unter herunterladen [https://www.microsoft.com/technet/scriptcenter/topics/msh/download.mspx](https://www.microsoft.com/technet/scriptcenter/topics/msh/download.mspx) .
