# <a name="swbemqualifierpropagatestosubclass-property"></a>' Taubemqualifier. propagatestosubclass '-Eigenschaft

Die **propagatestosubclass** -Eigenschaft des [**austaubemqualifier**](swbemqualifier.md) -Objekts ist ein boolescher Wert, der angibt, ob dieser Qualifizierer an eine Unterklasse weitergegeben werden kann.

Eine Erläuterung dieser Syntax finden Sie unter [Dokument Konventionen für die Skript-API](document-conventions-for-the-scripting-api.md).

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
SWbemQualifier.PropagatesToSubClass
```
