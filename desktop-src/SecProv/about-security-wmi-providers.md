# <a name="about-security-wmi-providers"></a>Informationen zu WMI-Sicherheitsanbietern

Die WMI-Anbieter für Sicherheit ermöglichen Anwendungen die Interaktion mit dem Trusted Platform Module (TPM) und BitLocker-Laufwerkverschlüsselung (BDE) über das Unified Management Framework von Windows-Verwaltungsinstrumentation (WMI).



| `Section`                                                                  | BESCHREIBUNG                                                                                                                    |
|--------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [WMI-Anbieter Referenz für Sicherheit](security-wmi-providers-reference.md) | API-Dokumentation für Win32-Klassen " [**\_ verschlüsseltablevolume**](win32-encryptablevolume.md) " und " [**Win32 \_ TPM**](win32-tpm.md) ". |



 

## <a name="remarks"></a>Bemerkungen

Managed Object Format-Dateien (MOF) enthalten die Definitionen für Windows-Verwaltungsinstrumentation (WMI)-Klassen. MOF-Dateien werden nicht als Teil des Windows SDK installiert. Sie werden auf dem Server installiert, wenn Sie die zugehörige Rolle mithilfe der Server-Manager hinzufügen. Weitere Informationen zu MOF-Dateien finden Sie unter [Managed Object Format (MOF)](../wmisdk/managed-object-format--mof-.md).
