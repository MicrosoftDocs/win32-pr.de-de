# <a name="win32_tpmisready-method"></a>Win32 \_ TPM:: isReady-Methode

Gibt an, ob das TPM einsatzbereit ist.

Diese Methode ist nur für lokale Administratoren zugänglich.

## <a name="syntax"></a>Syntax


```C++
uint32 IsReady(
  [out] BOOL IsReady
);
```



## <a name="parameters"></a>Parameter

<dl> <dt>

*IsReady* \[ vorgenommen\]
</dt> <dd>

Legen Sie diese Einstellung auf " **true** " fest, wenn TPM und System vollständig für die TPM-Verwendung bereitgestellt werden.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Alle TPM-Fehler sowie Fehler, die für die [TPM-Basisdienste](../tbs/tbs-return-codes.md) spezifisch sind, können zurückgegeben werden.

Allgemeine Rückgabecodes sind unten aufgeführt.



| Rückgabecode/-wert                                                                                                                                 | BESCHREIBUNG                           |
|---------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**S \_ OK**</dt> <dt>0 (0x0)</dt> </dl> | Die Methode war erfolgreich.<br/> |
