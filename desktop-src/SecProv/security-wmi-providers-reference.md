| [Trusted Platform Module Provider](trusted-platform-module-provider.md)       | API-Dokumentation zum Verwalten des Trusted Platform Module (TPM) mithilfe der [**Win32 \_ Tpm-Klasse.**](win32-tpm.md)                            |
