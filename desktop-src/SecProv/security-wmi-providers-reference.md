# <a name="security-wmi-providers-reference"></a>WMI-Anbieter Referenz für Sicherheit

WMI-Referenzseiten für Sicherheit enthalten ausführliche Beschreibungen des WMI-Anbieters für verschlüsselbare Volumes und des TPM-WMI-Anbieters (Trusted Platform Module).

WMI-Anbieter für Sicherheit enthalten die folgenden Anbieter.



| `Section`                                                                        | BESCHREIBUNG                                                                                                                                  |
|--------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------|
| [Anbieter der BitLocker-Laufwerkverschlüsselung](bitlocker-drive-encryption-provider.md) | API-Dokumentation zum Verwalten von BitLocker-Laufwerkverschlüsselung (BDE) mithilfe der Win32-Klasse " [**\_ verschlüsseltablevolume**](win32-encryptablevolume.md) ". |
| [Trusted Platform Module Anbieter](trusted-platform-module-provider.md)       | API-Dokumentation zum Verwalten des Trusted Platform Module (TPM) mithilfe der [**Win32- \_ TPM**](win32-tpm.md) -Klasse.                            |
