# <a name="win32_programgrouporitem-class"></a>Win32 \_ ProgramGroupOrItem-Klasse

Die abstrakte [WMI-Klasse](../wmisdk/retrieving-a-class.md) **\_ Win32 ProgramGroupOrItem** stellt eine logische Gruppierung von Programmen im Menü **\\ Startprogramme des Benutzers** dar. Sie enthält Programmgruppen und Programmgruppenelemente.

Die folgende Syntax wurde aus MOF-Code (Managed Object Format, verwaltetes Objektformat) vereinfacht und enthält alle geerbten Eigenschaften. Eigenschaften und Methoden sind in alphabetischer Reihenfolge und nicht in MOF-Reihenfolge.

## <a name="syntax"></a>Syntax

``` syntax
[Abstract, UUID("{86E30E86-7DB2-11d2-90CB-0060081A46FD}"), AMENDMENT]
class Win32_ProgramGroupOrItem : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
};
```
