# <a name="cim_extracapacitygroup-class"></a>CIM \_ extracapacitygroup-Klasse

Die **CIM \_ extracapacitygroup** -Klasse wird von einer Redundanz Gruppe abgeleitet, die angibt, dass die aggregierten Elemente mehr Kapazität oder mehr Kapazität aufweisen, als erforderlich sind. Ein Beispiel für diese Art von Redundanz ist die Installation von N + 1 Stromversorgungen oder Lüfter in einem System.

> [!IMPORTANT]
> Die Klassen der DMTF-CIM (Common Information Model) sind die übergeordneten Klassen, auf denen WMI-Klassen erstellt werden. WMI unterstützt zurzeit nur die [CIM 2. x-Versions Schemas](https://dmtf.org/standards/cim/schemas).

 

Die folgende Syntax wird durch MOF-Code (Managed Object Format) vereinfacht und schließt alle geerbten Eigenschaften ein. Eigenschaften werden in alphabetischer Reihenfolge und nicht in der MOF-Reihenfolge aufgelistet.

## <a name="syntax"></a>Syntax

``` syntax
[Abstract, UUID("{570ED2E8-E3D1-11d2-8601-0000F8102E5F}"), AMENDMENT]
class CIM_ExtraCapacityGroup : CIM_RedundancyGroup
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
  string   CreationClassName;
  uint16   RedundancyStatus;
  uint32   MinNumberNeeded;
};
```
