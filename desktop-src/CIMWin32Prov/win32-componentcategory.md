# <a name="win32_componentcategory-class"></a>Win32 \_ ComponentCategory-Klasse

Die [WMI-Klasse](/windows/desktop/WmiSdk/retrieving-a-class) **Win32 \_ ComponentCategory** stellt eine Komponentenkategorie dar. Komponentenkategorien sind Gruppen von COM-Klassen (Component Object Model), für die eine definierte Funktionalität festgelegt ist, die von ihnen gemeinsam genutzt wird. Ein Client, der diese Schnittstellen verwendet, fragt die Registrierung nach dem Kategorietitel und dem eindeutigen Bezeichner **namens CategoryID** ab, der aus einem global eindeutigen Bezeichner (GLOBALLY UNIQUE Identifier, GUID) erstellt wird.

Die folgende Syntax wurde aus MOF-Code (Managed Object Format, verwaltetes Objektformat) vereinfacht und enthält alle geerbten Eigenschaften. Eigenschaften werden in alphabetischer Reihenfolge und nicht in MOF-Reihenfolge aufgeführt.

## <a name="syntax"></a>Syntax

``` syntax
[Dynamic, Provider("CIMWin32"), UUID("{0F73ED5A-8ED9-11d2-B340-00105A1F8569}"), AMENDMENT]
class Win32_ComponentCategory : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Status;
  string   CategoryId;
  string   Name;
};
```
