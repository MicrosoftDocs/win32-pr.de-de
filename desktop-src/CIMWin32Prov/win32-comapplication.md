# <a name="win32_comapplication-class"></a>Win32 \_ COMApplication-Klasse

Die abstrakte [WMI-Klasse](/windows/desktop/WmiSdk/retrieving-a-class) **Win32 \_ COMApplication** stellt eine com-Anwendung (Component Object Model) dar. In diesem Kontext ist eine COM-Anwendung eine logische Gruppierung von COM-Klassen.

Die folgende Syntax wurde aus MOF-Code (Managed Object Format, verwaltetes Objektformat) vereinfacht und enthält alle geerbten Eigenschaften. Eigenschaften werden in alphabetischer Reihenfolge und nicht in MOF-Reihenfolge aufgeführt.

## <a name="syntax"></a>Syntax

``` syntax
[abstract, UUID("{0F73ED4F-8ED9-11d2-B340-00105A1F8569}"), AMENDMENT]
class Win32_COMApplication : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
};
```
