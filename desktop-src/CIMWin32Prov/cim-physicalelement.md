# <a name="cim_physicalelement-class"></a>CIM \_ PhysicalElement-Klasse

Die **CIM \_ PhysicalElement-Unterklassen** definieren jede Komponente eines Systems, die über eine eigene physische Identität verfügt. Instanzen dieser Klasse können in Form von Bezeichnungen definiert werden, die physisch an das Objekt angefügt werden können. Alle Prozesse, Dateien und logischen Geräte werden nicht als physische Elemente betrachtet. Beispielsweise ist es nicht möglich, eine Bezeichnung an ein Modem anfügen. Es ist nur möglich, eine Bezeichnung an die Karte anfügen, die das Modem implementiert. Dieselbe Karte könnte auch einen LAN-Adapter implementieren.

Konkrete verwaltete Systemelemente (in der Regel Hardwareelemente) verfügen über einen physischen Speicher. Ein verwaltetes Systemelement ist nicht notwendigerweise eine diskrete Komponente. Beispielsweise ist es möglich, dass eine einzelne Karte (ein Physischer Elementtyp) mehr als ein logisches Gerät hosten kann. Die Karte wird durch ein einzelnes physisches Element dargestellt, das mehreren logischen Geräten zugeordnet ist.

> [!IMPORTANT]
> Die CIM-Klassen (Distributed Management Task Force) (DMTF) (Common Information Model) sind die übergeordneten Klassen, auf denen WMI-Klassen erstellt werden. WMI unterstützt derzeit nur die [CIM 2.x-Versionsschemas.](https://dmtf.org/standards/cim/schemas)

 

Die folgende Syntax wird durch Managed Object Format (MOF)-Code vereinfacht und enthält alle geerbten Eigenschaften. Eigenschaften werden in alphabetischer Reihenfolge und nicht in MOF-Reihenfolge aufgeführt.

## <a name="syntax"></a>Syntax

``` syntax
[Abstract, UUID("{FAF76B61-798C-11D2-AAD1-006008C78BC7}"), AMENDMENT]
class CIM_PhysicalElement : CIM_ManagedSystemElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
  string   CreationClassName;
  string   Manufacturer;
  string   Model;
  string   OtherIdentifyingInfo;
  string   PartNumber;
  boolean  PoweredOn;
  string   SerialNumber;
  string   SKU;
  string   Tag;
  string   Version;
};
```
