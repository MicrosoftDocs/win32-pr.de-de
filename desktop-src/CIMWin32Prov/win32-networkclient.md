# <a name="win32_networkclient-class"></a>Win32- \_ networkclient-Klasse

Die **Win32 \_ Network Client** - [WMI-Klasse](../wmisdk/retrieving-a-class.md) stellt einen Netzwerkclient auf einem Windows-System dar. Jedes Computersystem im Netzwerk mit einer Client Beziehung zum System ist ein Nachfolger (oder Mitglied) dieser Klasse.

Die folgende Syntax wurde aus MOF-Code (Managed Object Format, verwaltetes Objektformat) vereinfacht und enthält alle geerbten Eigenschaften. Eigenschaften und Methoden sind in alphabetischer Reihenfolge, nicht in der MOF-Reihenfolge.

## <a name="syntax"></a>Syntax

``` syntax
[Dynamic, Provider("CIMWin32"), UUID("{8502C4D5-5FBB-11D2-AAC1-006008C78BC7}"), AMENDMENT]
class Win32_NetworkClient : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Status;
  string   Manufacturer;
  string   Name;
};
```
