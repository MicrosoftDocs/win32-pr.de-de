# <a name="cim_replacementset-class"></a>CIM \_ replacementset-Klasse

Die **CIM \_ replacementset** -Klasse aggregiert physische Elemente, die zusammen ersetzt werden müssen. Wenn Sie z. b. eine Speicherkarte austauschen, können die Komponenten Speicher-Chips auch entfernt und ersetzt werden. Oder diese Zuordnung kann verwendet werden, um einen Satz von Speicherchips zu ersetzen oder zu aktualisieren.

> [!IMPORTANT]
> Die Klassen der DMTF-CIM (Common Information Model) sind die übergeordneten Klassen, auf denen WMI-Klassen erstellt werden. WMI unterstützt zurzeit nur die [CIM 2. x-Versions Schemas](https://dmtf.org/standards/cim/schemas).

 

Die folgende Syntax wird durch MOF-Code (Managed Object Format) vereinfacht und schließt alle geerbten Eigenschaften ein. Eigenschaften werden in alphabetischer Reihenfolge und nicht in der MOF-Reihenfolge aufgelistet.

## <a name="syntax"></a>Syntax

``` syntax
[Abstract, UUID("{FAF76B6C-798C-11D2-AAD1-006008C78BC7}"), AMENDMENT]
class CIM_ReplacementSet
{
  string Description;
  string Name;
};
```
