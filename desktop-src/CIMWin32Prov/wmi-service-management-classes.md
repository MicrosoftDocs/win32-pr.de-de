# <a name="wmi-service-management-classes"></a>WMI-Dienst Verwaltungs Klassen

WMI-Dienst Verwaltungs Klassen werden verwendet, um den WMI-Dienst selbst und nicht das Computersystem oder Unternehmensnetzwerk zu verwalten. Das Verwalten von WMI umfasst sowohl das Konfigurieren des WMI-dienbes als auch das Verwalten von WMI

Die Kategorie WMI-Dienst Verwaltung enthält die folgenden Unterkategorien von Klassen:

-   [WMI-Konfigurations Klassen](#wmi-configuration-classes)
-   [WMI-Verwaltungs Klassen](#wmi-management-classes)

## <a name="wmi-configuration-classes"></a>WMI-Konfigurations Klassen

Die WMI-Konfigurations Klasse leitet Methoden ab, die den WMI-Dienst konfigurieren.

Im folgenden finden Sie eine Tabelle der WMI-Konfigurations Klassen.



| Klasse                                                             | BESCHREIBUNG                                                                     |
|-------------------------------------------------------------------|---------------------------------------------------------------------------------|
| [**Win32 \_ methodparameterclass**](win32-methodparameterclass.md) | Abstrakte Basisklasse, die von dieser Klasse abgeleitete Methoden Parameter implementiert. |



 

## <a name="wmi-management-classes"></a>WMI-Verwaltungs Klassen

Die WMI-Verwaltungs Klassen stellen Betriebsparameter für den WMI-Dienst dar.

Im folgenden finden Sie eine Tabelle der WMI-Verwaltungs Klassen.



| Klasse                                                       | BESCHREIBUNG                                                                                   |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**Win32- \_ wmisetting**](win32-wmisetting.md)               | Enthält die Betriebsparameter für den WMI-Dienst.                                      |
| [**Win32- \_ wmielementsetting**](win32-wmielementsetting.md) | Zuordnung zwischen einem Dienst, der im Windows-System ausgeführt wird, und den WMI-Einstellungen, die verwendet werden können. |
