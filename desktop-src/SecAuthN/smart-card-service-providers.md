# <a name="smart-card-service-providers"></a>Smartcarddienstanbieter

Smart Card Service Providers (SCSP) ermöglichen den Zugriff auf [*Smartcardfunktionen*](../secgloss/s-gly.md) . Sie können den Zugriff auf eine einzelne Funktion ermöglichen, wie z. b. die vom Smartcard-SDK bereitgestellten Basis Dienstanbieter, oder Sie können Zugriff auf verschiedene Funktionen bereitstellen, um eine komplexere Aufgabe zu erledigen.

Dienstanbieter ermöglichen den Zugriff über COM-Schnittstellen. Das Smartcard SDK stellt die COM-Schnittstellen bereit, die von den eigenen Basis Dienstanbietern verwendet werden, sowie mehrere Anwendungsschnittstellen, die beim Entwickeln von benutzerdefinierten Dienstanbietern verwendet werden können.
