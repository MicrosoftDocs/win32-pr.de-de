# <a name="smart-card-database-management-functions"></a>Verwaltungsfunktionen für Smartcard-Datenbanken

Die folgenden Funktionen verwalten die [*Smartcard-Datenbank*](../secgloss/s-gly.md)und aktualisieren die Datenbank mithilfe eines angegebenen [*Resource Manager-Kontexts*](../secgloss/r-gly.md).

> [!Note]  
> Die Datenbanksicherheit wird durch das Platzieren von Zugriffsbeschränkungen für die Datenbank gewahrt, anstatt dem [*Smartcardsubsystem*](../secgloss/s-gly.md)Sicherheitsmechanismen hinzuzufügen.

 



| Thema                                                            | BESCHREIBUNG                                                                                                                                                             |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Scardadressaderto Group**](/windows/desktop/api/Winscard/nf-winscard-scardaddreadertogroupa)           | Hinzufügen eines [*Readers*](../secgloss/r-gly.md) zu einer [*Lesergruppe*](../secgloss/r-gly.md) |
| [**Scardforgetcardtype**](/windows/desktop/api/Winscard/nf-winscard-scardforgetcardtypea)               | Entfernen Sie eine Smartcard aus dem System.                                                                                                                                    |
| [**Scardforgetreader**](/windows/desktop/api/Winscard/nf-winscard-scardforgetreadera)                   | Entfernen Sie einen Reader aus dem System.                                                                                                                                        |
| [**Scardforgetreadergroup**](/windows/desktop/api/Winscard/nf-winscard-scardforgetreadergroupa)         | Entfernen Sie eine Lesergruppe aus dem System.                                                                                                                                  |
| [**Scardintroducecardtype**](/windows/desktop/api/Winscard/nf-winscard-scardintroducecardtypea)         | Stellen Sie eine neue Karte für das System bereit.                                                                                                                                     |
| [**Scardintroducereader**](/windows/desktop/api/Winscard/nf-winscard-scardintroducereadera)             | Stellen Sie dem System einen neuen Reader vor.                                                                                                                                   |
| [**Scardintroducereadergroup**](/windows/desktop/api/Winscard/nf-winscard-scardintroducereadergroupa)   | Stellen Sie dem System eine neue Lesergruppe vor.                                                                                                                             |
| [**Scardremuvereaderfromgroup**](/windows/desktop/api/Winscard/nf-winscard-scardremovereaderfromgroupa) | Entfernen eines Readers aus einer Lesergruppe                                                                                                                                    |
