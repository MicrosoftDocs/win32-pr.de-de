# <a name="smart-card-database-management-functions"></a>Verwaltungsfunktionen für Smartcard-Datenbanken

Die folgenden Funktionen verwalten die [*Smartcard-Datenbank*](../secgloss/s-gly.md)und aktualisieren die Datenbank mithilfe eines angegebenen [*Ressourcen-Manager-Kontexts.*](../secgloss/r-gly.md)

> [!Note]  
> Die Datenbanksicherheit wird beibehalten, indem Zugriffseinschränkungen für die Datenbank vorgenommen werden, anstatt dem [*Smartcardsubsystem*](../secgloss/s-gly.md)Sicherheitsmechanismen hinzuzufügen.

 



| Thema                                                            | BESCHREIBUNG                                                                                                                                                             |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SCardAddReaderToGroup**](/windows/desktop/api/Winscard/nf-winscard-scardaddreadertogroupa)           | Fügen Sie einer [*Readergruppe*](../secgloss/r-gly.md)einen [*Reader*](../secgloss/r-gly.md) hinzu. |
| [**SCardForgetCardType**](/windows/desktop/api/Winscard/nf-winscard-scardforgetcardtypea)               | Entfernen Sie eine Smartcard aus dem System.                                                                                                                                    |
| [**SCardForgetReader**](/windows/desktop/api/Winscard/nf-winscard-scardforgetreadera)                   | Entfernen Sie einen Reader aus dem System.                                                                                                                                        |
| [**SCardForgetReaderGroup**](/windows/desktop/api/Winscard/nf-winscard-scardforgetreadergroupa)         | Entfernen Sie eine Lesergruppe aus dem System.                                                                                                                                  |
| [**SCardIntroduceCardType**](/windows/desktop/api/Winscard/nf-winscard-scardintroducecardtypea)         | Führen Sie eine neue Karte in das System ein.                                                                                                                                     |
| [**SCardIntroduceReader**](/windows/desktop/api/Winscard/nf-winscard-scardintroducereadera)             | Führen Sie einen neuen Reader in das System ein.                                                                                                                                   |
| [**SCardIntroduceReaderGroup**](/windows/desktop/api/Winscard/nf-winscard-scardintroducereadergroupa)   | Führen Sie eine neue Lesergruppe in das System ein.                                                                                                                             |
| [**SCardRemoveReaderFromGroup**](/windows/desktop/api/Winscard/nf-winscard-scardremovereaderfromgroupa) | Entfernen eines Readers aus einer Lesergruppe.                                                                                                                                    |
