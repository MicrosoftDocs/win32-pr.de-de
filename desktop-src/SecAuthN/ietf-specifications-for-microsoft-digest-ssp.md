# <a name="ietf-specifications-for-microsoft-digest-ssp"></a>IETF-Spezifikationen für Microsoft Digest SSP

HTTP-Authentifizierung: Die Standard- und Digestzugriffsauthentifizierung (RFC 2617) ist unter [https://www.ietf.org/rfc/rfc2617.txt](https://www.ietf.org/rfc/rfc2617.txt) verfügbar.

Die Verwendung von Digest als SASL-Mechanismus Internetentwurf ist unter [https://www.ietf.org/rfc/rfc2831.txt](https://www.ietf.org/rfc/rfc2831.txt) verfügbar.

Weitere Informationen zu SASL finden Sie auch in der Simple Authentication and Security Layer-Spezifikation (RFC 2222) unter [https://www.ietf.org/rfc/rfc2222.txt](https://www.ietf.org/rfc/rfc2222.txt) .
