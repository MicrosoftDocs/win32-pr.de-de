# <a name="primary-service-provider"></a>Primärer Dienstanbieter

Jede [*Smartcard,*](../secgloss/s-gly.md) die dem System bekannt ist, verfügt möglicherweise über eine registrierte COM-Schnittstelle, die als primärer Dienstanbieter aufgeführt ist. Wenn Sie über COM auf eine *Smartcard* zugreifen, stellt dieser [*primäre Dienstanbieter*](../secgloss/p-gly.md) die Steuerungsschnittstellen für die Karte bereit. Dadurch können Kartendienste für eine Vielzahl von Programmierumgebungen verfügbar gemacht werden, einschließlich Java und dem Microsoft Visual Basic-Entwicklungssystem.
