# <a name="primary-service-provider"></a>Primärer Dienstanbieter

Jede [*Smartcard*](../secgloss/s-gly.md) , die dem System bekannt ist, verfügt über eine registrierte COM-Schnittstelle, die als primärer Dienstanbieter aufgeführt ist. Wenn Sie über com auf eine *Smartcard* zugreifen, stellt dieser [*primäre Dienstanbieter*](../secgloss/p-gly.md) die Steuerungs Schnittstellen für die Karte bereit. Dies ermöglicht das verfügbar machen von Kartendiensten für eine Vielzahl von Programmierumgebungen, einschließlich Java und das Microsoft Visual Basic-Entwicklungssystem.
