# <a name="quality-of-protection-and-ciphers"></a>Qualität von Schutz und Verschlüsselungen

Die Qualität des Schutzes bestimmt die Sicherheitsstufe, die beim Senden und Empfangen von Nachrichten verwendet wird. Microsoft Digest wählt eine Ebene basierend auf den Anforderungen von Client und Server aus.

Wenn Digest als SASL-Mechanismus verwendet wird, können Anwendungen angeben, dass Nachrichten mithilfe der Verschlüsselung vertraulich gehalten werden. Der Server gibt die Verschlüsselungen an, die er für verschlüsselte Nachrichten unterstützt, und der Client wählt eine Verschlüsselung aus dieser Liste aus.

In den folgenden Themen werden diese Sicherheitsmechanismen ausführlich erläutert:

-   [Quality of Protection](quality-of-protection.md)
-   [Ciphers](ciphers.md)
