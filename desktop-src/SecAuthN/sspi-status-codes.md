# <a name="sspi-status-codes"></a>SSPI-Statuscodes

Die folgenden Statuscodes werden in SSPI-Anwendungen verwendet und in Winerror.h definiert.



| Statuscode                                        | Bedeutung                                                                                                                                                                                                                                                                                                                                           |
|----------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| SEC \_ E \_ ALGORITHM \_ MISMATCH<br/>             | Client und Server können nicht kommunizieren, da sie keinen gemeinsamen Algorithmus besitzen.<br/>                                                                                                                                                                                                                                               |
| SEC \_ E \_ BAD \_ BINDINGS<br/>                   | Die vom Client bereitgestellten SSPI-Kanalbindungen sind falsch.<br/>                                                                                                                                                                                                                                                                        |
| SEC \_ E \_ BAD \_ PKGID<br/>                      | Der angeforderte Paketbezeichner ist nicht vorhanden.<br/>                                                                                                                                                                                                                                                                                       |
| SEC \_ E \_ BUFFER \_ TOO \_ SMALL<br/>              | Die für die Funktion bereitgestellten Puffer sind nicht groß genug, um die Informationen zu enthalten.<br/>                                                                                                                                                                                                                                                  |
| SEC \_ E CANNOT INSTALL (S.E KANN NICHT INSTALLIERT \_ \_ WERDEN)<br/>                 | Das [*Sicherheitspaket*](../secgloss/s-gly.md) kann nicht erfolgreich initialisiert werden und sollte nicht installiert werden.<br/>                                                                                                                                                                     |
| SEC \_ E \_ CANNOT \_ PACK<br/>                    | Das Paket kann den Kontext nicht packen.<br/>                                                                                                                                                                                                                                                                                             |
| SEC \_ E \_ CERT \_ EXPIRED<br/>                   | Das empfangene Zertifikat ist abgelaufen.<br/>                                                                                                                                                                                                                                                                                                  |
| SEC \_ E \_ CERT \_ UNKNOWN<br/>                   | Unbekannter Fehler beim Verarbeiten des Zertifikats.<br/>                                                                                                                                                                                                                                                                            |

| SEC \_ E \_ CONTEXT \_ EXPIRED<br/>                | Die Anwendung bezieht sich auf einen Kontext, der bereits geschlossen wurde. | SEC \_ E CROSSREALM DELEGATION FAILURE (S. E \_ CROSSREALM-DELEGIERUNGSFEHLER) \_ \_<br/> | Der Server hat versucht, eine Kerberos-eingeschränkte Delegierungsanforderung für ein Ziel außerhalb des Bereichs des Servers zu senden.<br/>                                                                                                                                                                                                                        |
| SEC \_ E \_ CRYPTO \_ SYSTEM \_ INVALID<br/>         | Die Kryptografiesystem- oder Prüfsummenfunktion ist ungültig, da eine erforderliche Funktion nicht verfügbar ist.<br/>                                                                                                                                                                                                                                 |
| SEC \_ E \_ DECRYPT \_ FAILURE<br/>                | Die angegebenen Daten konnten nicht entschlüsselt werden.<br/>                                                                                                                                                                                                                                                                                             |

| SEC \_ E \_ DOWNGRADE \_ ERKANNT<br/>             | Das System hat einen möglichen Versuch erkannt, die Sicherheit zu gefährden. Stellen Sie sicher, dass der authentifizierte Server kontaktiert werden kann.<br/>                                                                                                                                                                                                         |
| SEC \_ E \_ ENCRYPT \_ FAILURE<br/>                | Die angegebenen Daten konnten nicht verschlüsselt werden.<br/>                                                                                                                                                                                                                                                                                             |
| SEC \_ E \_ ILLEGAL \_ MESSAGE<br/>                | Die empfangene Nachricht war unerwartet oder schlecht formatiert.<br/>                                                                                                                                                                                                                                                                                |
| SEC \_ E \_ UNVOLLSTÄNDIGE \_ ANMELDEINFORMATIONEN<br/>         | Die angegebenen Anmeldeinformationen waren nicht vollständig und konnten nicht überprüft werden. | SEC \_ E \_ UNVOLLSTÄNDIGE \_ NACHRICHT<br/>             | Die angegebene Meldung war unvollständig. Die Signatur wurde nicht überprüft.<br/>                                                                                                                                                                                                                                                                   |
| \_S.E \_ NICHT GENÜGEND \_ ARBEITSSPEICHER<br/>            | Zum Abschließen der Anforderung ist nicht genügend Arbeitsspeicher verfügbar.<br/>                                                                                                                                                                                                                                                                                |
