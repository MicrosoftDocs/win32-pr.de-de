# <a name="sspi-status-codes"></a>SSPI-Statuscodes

| \_ \_ UNGÜLTIGE \_ BINDUNGEN IN SEC E<br/>                   | Die vom Client bereitgestellten SSPI-Kanalbindungen sind falsch.<br/>                                                                                                                                                                                                                                                                        |
| SEC \_ E \_ BAD \_ PKGID<br/>                      | Der angeforderte Paketbezeichner ist nicht vorhanden.<br/>                                                                                                                                                                                                                                                                                       |
| SEC \_ E \_ BUFFER \_ TOO \_ SMALL<br/>              | Die für die Funktion bereitgestellten Puffer sind nicht groß genug, um die Informationen zu enthalten.<br/>                                                                                                                                                                                                                                                  |

| SEC \_ E \_ CANNOT \_ PACK<br/>                    | Das Paket kann den Kontext nicht packen.<br/>                                                                                                                                                                                                                                                                                             |
