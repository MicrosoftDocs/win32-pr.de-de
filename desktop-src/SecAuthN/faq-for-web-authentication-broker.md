# <a name="faq-for-web-authentication-broker"></a>Häufig gestellte Fragen zum Webauthentifizierungs Broker

Häufig gestellte Fragen zum Webauthentifizierungs Broker.



| Frage                                                                                                    | Antwort                                                                                                                                                                                                            |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Wie kann ich sicherstellen, dass meine https://-Seite mit dem Webauthentifizierungs Broker funktioniert?                         | Laden Sie die Seite in Windows Internet Explorer, bevor Sie Sie mit dem Webauthentifizierungs Broker ausprobieren. Wenn die Webseite ohne Fehler geladen wird, wird Sie auch vom Webauthentifizierungs Broker ordnungsgemäß gerendert. |
| Wenn ein Fehler auftritt, werden die Fehlercodes dem Benutzer angezeigt?                                         | Während dem Benutzer eine Fehlerseite angezeigt wird, wird der zugrunde liegende Fehlercode nicht angezeigt. Sie wird nur an die APP zurückgegeben. Alternativ können Sie auch die Betriebs Protokolle verwenden.                                    |
| Wie kann ich weitere Details zum aufgetretenen Fehler finden?                                                       | Ausführliche Informationen finden Sie in den Betriebs Protokollen.                                                                                                                                                                             |
| Gibt der Single Sign-on (SSO)-app-Container seine Cookies mit Internet Explorer oder anderen Browsern gemeinsam? | Nein.                                                                                                                                                                                                               |
