# <a name="user-name-functions"></a>Benutzernamen Funktionen

Zu den Funktionen von Benutzernamen gehört eine einzelne Funktion, [**npgetuser**](/windows/desktop/api/Npapi/nf-npapi-npgetuser). Diese Funktion ruft entweder den aktuellen Standardbenutzer Namen oder den Benutzernamen ab, der zum Herstellen der Netzwerkverbindung verwendet wird.
