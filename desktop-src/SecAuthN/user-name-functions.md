# <a name="user-name-functions"></a>Benutzernamenfunktionen

Benutzernamenfunktionen enthalten eine einzelne Funktion, [**NPGetUser.**](/windows/desktop/api/Npapi/nf-npapi-npgetuser) Diese Funktion ruft entweder den aktuellen Standardbenutzernamen oder den Benutzernamen ab, der zum Herstellen der Netzwerkverbindung verwendet wird.
