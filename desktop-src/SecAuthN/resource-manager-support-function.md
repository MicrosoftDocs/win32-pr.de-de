# <a name="resource-manager-support-function"></a>Resource Manager Supportfunktion

Die folgende Funktion gibt durch die Verwendung des SCARD AUTOALLOCATE-Längenkennzeichens zugewiesenen Arbeitsspeicher frei, was die Verwendung der anderen \_ [*Resource Manager-Funktionen*](../secgloss/r-gly.md) vereinfacht.



| Thema                                      | BESCHREIBUNG                                                     |
|--------------------------------------------|-----------------------------------------------------------------|
| [**SCardFreeMemory**](/windows/desktop/api/Winscard/nf-winscard-scardfreememory) | Geben Sie speicher frei, der mithilfe von SCARD \_ AUTOALLOCATE zurückgegeben wird. |
