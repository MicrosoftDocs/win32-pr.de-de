# <a name="winlogon"></a>Windows-Anmeldung

[*Winlogon*](../secgloss/w-gly.md) verarbeitet Schnittstellenfunktionen, die unabhängig von der Authentifizierungsrichtlinie sind. Sie erstellt die Desktops für die Fensterstation, implementiert Time out-Vorgänge und stellt eine Reihe von Unterstützungsfunktionen für die [*GINA*](../secgloss/g-gly.md)bereit.
