# <a name="authentication-reference"></a>Authentifizierungsreferenz

Authentifizierungsreferenzseiten enthalten ausführliche Beschreibungen der Microsoft-Authentifizierungsfunktionen, Schnittstellen, Objekte, Strukturen und anderen Programmierelemente. Diese Themen enthalten Informationen zu den Programmierelementen, die verwendet werden, um die Identität von Benutzern und Diensten zu erstellen. Nachdem eine Identität eingerichtet wurde, kann sie verwendet werden, um den Zugriff auf einen einzelnen Computer oder auf die Ressourcen in einem Computernetzwerk zu versuchen.

Referenzseiten sind in die folgenden Gruppen unterteilt.



| `Section`                                                          | BESCHREIBUNG                                                                          |
|------------------------------------------------------------------|--------------------------------------------------------------------------------------|
| [Authentifizierungskonst constants (Authentifizierungs](authentication-constants.md)         | Konstanten, die von Authentifizierungsprogrammierelementen verwendet werden.<br/>                    |
| [Authentifizierungsdatentypen](authentication-data-types.md)       | Datentypen, die von Authentifizierungsprogrammierelementen verwendet werden.<br/>                   |
| [Authentifizierungsenumeration](authentication-enumerations.md)   | Enumerationen, die von Authentifizierungsprogrammierungselementen verwendet werden.<br/>                 |
| [Authentifizierungsfunktionen](authentication-functions.md)         | Funktionen, die bei der Einrichtung und Überprüfung der Benutzeridentität verwendet werden.<br/>  |
| [Authentifizierungsschnittstellen](authentication-interfaces.md)       | Schnittstellen, die bei der Einrichtung und Überprüfung der Benutzeridentität verwendet werden.<br/> |
| [Authentifizierungsregistrierungsschlüssel](authentication-registry-keys.md) | Registrierungsschlüssel, die von Authentifizierungsprogrammierelementen verwendet werden.<br/>                |
| [Authentifizierungs-Rückgabewerte](authentication-return-values.md) | Gibt von Authentifizierungsprogrammierelementen verwendete Werte zurück.<br/>                |
| [Authentifizierungsstrukturen](authentication-structures.md)       | Strukturen, die mit Authentifizierungsfunktionen, Schnittstellen und Objekten verwendet werden.<br/>   |
