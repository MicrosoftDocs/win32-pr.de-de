# <a name="authentication-reference"></a>Authentifizierungs Referenz

Authentifizierungs Referenzseiten enthalten ausführliche Beschreibungen der Microsoft-Authentifizierungsfunktionen,-Schnittstellen,-Objekte,-Strukturen und anderer Programmier Elemente. Diese Themen enthalten Informationen zu den Programmier Elementen, die zum Einrichten der Identität von Benutzern und Diensten verwendet werden. Nachdem eine Identität eingerichtet wurde, kann Sie verwendet werden, um den Zugriff auf einen einzelnen Computer oder auf die Ressourcen in einem Computernetzwerk zu versuchen.

Verweis Seiten sind in die folgenden Gruppen unterteilt.

| [Authentifizierungs Enumerationen](authentication-enumerations.md)   | Enumerationen, die von Authentifizierungs Programmier Elementen verwendet werden.<br/>                 |
| [Authentifizierungsfunktionen](authentication-functions.md)         | Funktionen, die mit der Einrichtung und Überprüfung der Benutzeridentität verwendet werden.<br/>  |
| [Authentifizierungs Schnittstellen](authentication-interfaces.md)       | Schnittstellen, die bei der Einrichtung und Überprüfung der Benutzeridentität verwendet werden.<br/> |
| [Registrierungsschlüssel für die Authentifizierung](authentication-registry-keys.md) | Registrierungsschlüssel, die von Authentifizierungs Programmierungs Elementen verwendet werden.<br/>                |
