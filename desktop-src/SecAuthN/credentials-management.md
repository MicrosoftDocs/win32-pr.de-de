# <a name="credentials-management"></a>Verwaltung von Anmelde Informationen

Entwickler, die für Windows schreiben, können die Anmelde Informations Verwaltungs-API verwenden, einschließlich der Benutzeroberflächen Funktionen zur Verwaltung von Anmelde Informationen, um Anmelde Informationen wie Benutzernamen und Kenn Wörter abzurufen und zu verwalten. Diese Funktionen fordern die Verwendung von Windows-Kontoinformationen anstelle der Anmelde Informationen an, die bei der Anmeldung eingerichtet wurden. Solche Anforderungen treten in der Regel auf, wenn die Anmelde Informationen nicht über Berechtigungen verfügen, die für die Anwendung erforderlich sind.

Die Benutzeroberflächen Funktionen für die Anmelde Informationsverwaltung stellen Schnittstellen mit der Darstellung der Windows-Benutzeroberfläche bereit. Diese Funktionen umfassen anpassbare Optionen, die dem Anmelde Informationsspeicher des Benutzers Benutzerinformationen hinzufügen.

Die folgenden Themen enthalten weitere Informationen zur Verwaltungs-API für Anmelde Informationen:

-   [Arten von Anmelde Informationen](kinds-of-credentials.md)
-   [Benutzernamen Formate](user-name-formats.md)

Informationen zu Funktionen in der Anmeldeinformationsverwaltungs-API finden Sie unter [Authentifizierungs Referenz](authentication-reference.md).
