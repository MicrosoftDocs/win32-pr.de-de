# <a name="credentials-management"></a>Verwaltung von Anmeldeinformationen

Entwickler, die für Windows schreiben, können die Anmeldeinformationen Verwaltungs-API einschließlich Der Anmeldeinformationsverwaltung Benutzeroberfläche (UI)-Funktionen verwenden, um Anmeldeinformationen wie Benutzernamen und Kennwörter abzurufen und zu verwalten. Diese Funktionen fordern Windows Kontoinformationen an, die anstelle der Anmeldeinformationen verwendet werden sollen, die bei der Anmeldung eingerichtet wurden. Solche Anforderungen treten in der Regel auf, wenn die Anmeldeinformationen nicht über Berechtigungen verfügen, die von der Anwendung benötigt werden.

Die Benutzeroberflächenfunktionen der Anmeldeinformationsverwaltung stellen Schnittstellen mit der Darstellung der Windows Benutzeroberfläche bereit. Diese Funktionen enthalten anpassbare Optionen, die dem Anmeldeinformationsspeicher des Benutzers Benutzerinformationen hinzufügen.

Die folgenden Themen enthalten weitere Informationen zum Verwaltungs-API Anmeldeinformationen:

-   [Arten von Anmeldeinformationen](kinds-of-credentials.md)
-   [Benutzernamenformate](user-name-formats.md)

Informationen zu Funktionen im Verwaltungs-API für Anmeldeinformationen finden Sie unter [Authentifizierungsreferenz.](authentication-reference.md)
