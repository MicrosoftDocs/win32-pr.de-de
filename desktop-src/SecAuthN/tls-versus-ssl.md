# <a name="tls-vs-ssl"></a>TLS im Vergleich zu SSL

TLS ist ein Standard, der eng mit SSL 3.0 verknüpft ist und manchmal als "SSL 3.1" bezeichnet wird. TLS ersetzt SSL 2.0 und sollte in der Neuen Entwicklung verwendet werden. Ab Windows 10 Version 1607 und Windows Server 2016 wurde SSL 2.0 entfernt und wird nicht mehr unterstützt.

Anwendungen, die ein hohes Maß an Interoperabilität erfordern, sollten SSL 3.0 und TLS unterstützen. Aufgrund der Ähnlichkeiten zwischen diesen beiden Protokollen sind SSL-Details nicht in dieser Dokumentation enthalten, es sei denn, sie unterscheiden sich von TLS. Folgendes stammt aus [RFC 2246:](https://www.ietf.org/rfc/rfc2246.txt)

"Die Unterschiede zwischen diesem Protokoll und SSL 3.0 sind nicht drastisch, aber sie sind so signifikant, dass TLS 1.0 und SSL 3.0 nicht zusammenarbeiten (obwohl TLS 1.0 einen Mechanismus enthält, mit dem eine TLS-Implementierung auf SSL 3.0 zurückgreifen kann)."
