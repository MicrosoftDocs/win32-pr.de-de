# <a name="administrative-functions"></a>Administrative Funktionen

Mit den folgenden Verwaltungsfunktionen kann ein Netzwerkanbieter netzwerkspezifische Aktionen ausführen, um netzwerkspezifische Verzeichnisse anzuzeigen und zu bearbeiten, d. h. Verzeichnisse, die nicht Windows zugeordnet sind.



| Funktion                                         | Beschreibung                                                                                                                                                   |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**NPGetDirectoryType**](/windows/desktop/api/Npapi/nf-npapi-npgetdirectorytype) | Wird vom Datei-Manager aufgerufen, um den Typ eines Netzwerkverzeichnisses zu bestimmen.                                                                                          |
| [**NPDirectoryNotify**](/windows/desktop/api/Npapi/nf-npapi-npdirectorynotify)   | Wird vom Datei-Manager aufgerufen, um den Netzwerkanbieter über Verzeichnisvorgänge zu benachrichtigen. Diese Funktion kann verwendet werden, um ein spezielles Verhalten für bestimmte Verzeichnisse durchzuführen. |
