# <a name="administrative-functions"></a>Verwaltungsfunktionen

Mit den folgenden administrativen Funktionen kann ein Netzwerkanbieter Netzwerk spezifische Verzeichnisse zum Anzeigen und Bearbeiten von Netzwerk spezifischen Verzeichnissen verwenden, d. h. Verzeichnisse, die nicht-Windows-Protokollen zugeordnet sind.



| Funktion                                         | BESCHREIBUNG                                                                                                                                                   |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Npgetdirectoriytype**](/windows/desktop/api/Npapi/nf-npapi-npgetdirectorytype) | Wird vom Datei-Manager aufgerufen, um den Typ eines Netzwerk Verzeichnisses zu bestimmen.                                                                                          |
| [**Npdirectoriynotify**](/windows/desktop/api/Npapi/nf-npapi-npdirectorynotify)   | Wird vom Datei-Manager aufgerufen, um den Netzwerkanbieter von Verzeichnis Vorgängen zu benachrichtigen. Diese Funktion kann verwendet werden, um ein spezielles Verhalten für bestimmte Verzeichnisse auszuführen. |
