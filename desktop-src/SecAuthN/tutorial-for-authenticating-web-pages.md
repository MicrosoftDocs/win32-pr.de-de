# <a name="tutorial-for-authenticating-web-pages"></a>Tutorial zum Authentifizieren von Webseiten

In diesem Tutorial werden die verschiedenen Aspekte für Webentwickler beim Entwerfen von Seiten für den Webautorisierungsfluss für Windows 8. In diesem Abschnitt wird ein zweiseitiger Autorisierungsablauf veranschaulicht.

**Ziel:** Das Ziel des Beispiels besteht nicht im Präskriptivisieren des Layouts oder visuellen Designs, auf das jeder Anbieter sicherlich einen einzigartigen Blickpunkt haben wird, sondern auf einige Bereiche, in die es sich zu investieren lohnt, um eine erstklassige Microsoft-Designstil-Erfahrung für Windows 8.

## <a name="prerequisites"></a>Voraussetzungen

Um Webauthentifizierungsbroker-APIs verwenden zu können, benötigen Sie:

-   Windows 8 (nur x86 oder amd64)
-   Microsoft-Internetinformationsdienste (IIS) sollte über den Systemsteuerung unter "Programme und Features" installiert werden und die Option "Windows-Features aktivieren oder deaktivieren" verwenden.

**Gesamtzeit bis zum Abschluss:** 2 Minuten.
