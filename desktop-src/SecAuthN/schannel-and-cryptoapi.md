# <a name="schannel-and-cryptoapi"></a>SChannel und CryptoAPI

SChannel verwendet [CryptoAPI](../seccrypto/cryptography-essentials.md) für kryptografische Vorgänge, wie z. b. das Speichern von [*öffentlichen und privaten Schlüsseln*](../secgloss/p-gly.md).

Die folgenden Themen enthalten ausführliche Informationen zur Verwendung von CryptoAPI durch Schannel.



| Thema                                                                   | BESCHREIBUNG                                                                                                          |
|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [Zertifikatspeicher](certificate-stores.md)<br/>                 | Client-und Server Zertifikate müssen in einem Zertifikat Speicher gespeichert werden.<br/>                                |
| [Private Schlüssel CryptoAPI 2.0](cryptoapi-2-0-private-keys.md)<br/> | SChannel-Anmelde Informationen werden intern als [**CERT- \_ Kontext**](/windows/win32/api/wincrypt/ns-wincrypt-cert_context) Strukturen dargestellt.<br/> |
