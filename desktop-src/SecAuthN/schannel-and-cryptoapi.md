# <a name="schannel-and-cryptoapi"></a>Schannel und CryptoAPI

Schannel verwendet [CryptoAPI](../seccrypto/cryptography-essentials.md) für kryptografische Vorgänge wie das Speichern [*von öffentlichen/privaten Schlüsseln.*](../secgloss/p-gly.md)

Die folgenden Themen enthalten ausführliche Informationen dazu, wie Schannel CryptoAPI nutzt.



| Thema                                                                   | BESCHREIBUNG                                                                                                          |
|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [Zertifikatspeicher](certificate-stores.md)<br/>                 | Sowohl Client- als auch Serverzertifikate müssen in einem Zertifikatspeicher gespeichert werden.<br/>                                |
| [CryptoAPI 2.0 Private Schlüssel](cryptoapi-2-0-private-keys.md)<br/> | Schannel-Anmeldeinformationen werden intern als [**CERT \_ CONTEXT-Strukturen**](/windows/win32/api/wincrypt/ns-wincrypt-cert_context) dargestellt.<br/> |
