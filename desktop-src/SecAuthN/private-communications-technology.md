# <a name="private-communications-technology"></a>Private Kommunikationstechnologie

Die private Kommunikationstechnologie (PCT) wurde durch [*Secure Sockets Layer*](../secgloss/s-gly.md) 3,0 und das Transport Layer Security- [Protokoll](transport-layer-security-protocol.md)abgelöst.

SChannel unterstützt PCT 1,0 nur aus Gründen der Abwärtskompatibilität. Er sollte nicht für die neue Entwicklung verwendet werden. PCT unterstützt die meisten Verschlüsselungs Sammlungen, die von SSL 2,0 unterstützt werden.
