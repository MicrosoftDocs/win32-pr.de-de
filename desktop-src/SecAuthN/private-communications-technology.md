# <a name="private-communications-technology"></a>Private Kommunikationstechnologie

Private Communications Technology (PCT) wurde durch [*Secure Sockets Layer*](../secgloss/s-gly.md) 3.0 und das [Transport Layer Security Protocol](transport-layer-security-protocol.md)ersetzt.

Schannel unterstützt PCT 1.0 nur aus Gründen der Abwärtskompatibilität. sie sollte nicht für die Neuentwicklung verwendet werden. PCT unterstützt die meisten Verschlüsselungssammlungen, die von SSL 2.0 unterstützt werden.
