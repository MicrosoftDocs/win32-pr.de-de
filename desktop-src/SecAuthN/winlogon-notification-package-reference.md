# <a name="winlogon-notification-package-reference"></a>Referenz zum Winlogon-Benachrichtigungs Paket

In den folgenden Themen werden Referenzinformationen zu [*Winlogon*](../secgloss/w-gly.md) -Benachrichtigungs Paketen bereitgestellt:

-   [Registrierungseinträge](registry-entries.md) listet die Schlüsselwerte und-Einstellungen auf, die in der Systemregistrierung eingegeben werden müssen, um ein Benachrichtigungs Paket und seine Ereignishandlerfunktionen zu registrieren.
-   Der [Prototyp der Ereignishandlerfunktion](event-handler-function-prototype.md) erläutert den Prototyp für Ereignishandlerfunktionen. Jede Ereignishandlerfunktion in Ihrem Benachrichtigungs Paket muss mit diesem Prototyp identisch sein.
