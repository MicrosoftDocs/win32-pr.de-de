# <a name="winlogon-notification-package-reference"></a>Referenz zum Winlogon-Benachrichtigungspaket

Die folgenden Themen enthalten Referenzinformationen zu [*Winlogon-Benachrichtigungspaketen:*](../secgloss/w-gly.md)

-   [Registrierungseinträge](registry-entries.md) listet die Schlüsselwerte und Einstellungen auf, die in die Systemregistrierung eingegeben werden müssen, um ein Benachrichtigungspaket und seine Ereignishandlerfunktionen zu registrieren.
-   [Der Prototyp der Ereignishandlerfunktion](event-handler-function-prototype.md) enthält Details zum Prototyp für Ereignishandlerfunktionen. Jede Ereignishandlerfunktion in Ihrem Benachrichtigungspaket muss mit diesem Prototyp übereinstimmen.
