# <a name="sample-sspi-code"></a>SSPI-Beispiel Code

Dieser Abschnitt enthält zwei komplette Beispiele:

-   [Verwenden von SSPI mit einem Windows Sockets-Client](using-sspi-with-a-windows-sockets-client.md)
-   [Verwenden von SSPI mit einem Windows Sockets-Server](using-sspi-with-a-windows-sockets-server.md)

Die Client-und Server Beispiele sind so konzipiert, dass Sie zusammenarbeiten. In beiden Beispielen wird die Header Datei Security. h verwendet, die in der [Header Datei für SSPI-Client-und-Server Beispiele](header-file-for-sspi-client-and-server.md)zu finden ist.

Ein Beispiel für die Initialisierung eines Arrays von Sicherheits Puffern finden Sie unter [Beispiel Code für secbuffer und secbufferdesc](secbuffer-and-secbufferdesc-example-code.md).

Weitere Informationen zum Signieren oder Verschlüsseln von Daten, nachdem eine sichere Verbindung hergestellt wurde, finden [Sie unter Verwenden von Nachrichten Unterstützung](using-message-support.md).
