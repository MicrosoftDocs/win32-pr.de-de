# <a name="microsoft-digest-ssp"></a>SSP Microsoft Digest

Microsoft Digest ist eine [*Security Support Provider*](../secgloss/s-gly.md) (SSP), die das Digest-Zugriffsprotokoll implementiert, ein schlankes Authentifizierungsprotokoll für Parteien, die an der Kommunikation mit http (Hypertext Transfer Protocol) oder SASL (Simple Authentication Security Layer) beteiligt sind.

Microsoft Digest bietet einen einfachen Challenge Response-Mechanismus zum Authentifizieren von Clients. Dieser SSP ist für die Verwendung durch Client/Server-Anwendungen mithilfe der http-oder SASL-basierten Kommunikation vorgesehen.

> [!IMPORTANT]
> Die Digest-Authentifizierung wird nicht Gesamtstruktur übergreifend unterstützt.
