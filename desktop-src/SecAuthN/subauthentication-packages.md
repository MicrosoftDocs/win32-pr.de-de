# <a name="subauthentication-packages"></a>Unter Authentifizierungs Pakete

Die mit Windows bereitgestellten [*Authentifizierungs Pakete*](../secgloss/a-gly.md) unterstützen die Anpassung mithilfe von [*unter Authentifizierungs Paketen*](../secgloss/s-gly.md). Ein unter Authentifizierungs Paket ist eine DLL, die einen Teil der Authentifizierungs-und Validierungs Kriterien, die vom Haupt Authentifizierungs Paket verwendet werden, ergänzt oder ersetzt. Beispielsweise kann ein bestimmter Server ein unter Authentifizierungs Paket bereitstellen, das das Kennwort eines Benutzers mit einem anderen Algorithmus überprüft oder Arbeitsstations Einschränkungen in einem anderen Format angibt.

Weitere Informationen zu unter Authentifizierungs Paketen finden Sie im subauth-Beispiel, das mit dem Platform Software Development Kit (SDK) installiert wurde.
