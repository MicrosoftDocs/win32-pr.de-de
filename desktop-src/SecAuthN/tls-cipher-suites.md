# <a name="tls-cipher-suites"></a>TLS-Verschlüsselungs Sammlungen

SChannel unterstützt die folgenden Verschlüsselungs Sammlungen für TLS 1,0 in Windows Server 2003 und Windows XP. Die Suites werden in der Standard Reihenfolge aufgelistet, in der Sie ausgewählt werden. Informationen zu Verschlüsselungs Sammlungen für Windows Server 2008 und Windows Vista finden Sie unter Verschlüsselungs Sammlungen [in Schannel](cipher-suites-in-schannel.md).

-   TLS- \_ RSA \_ mit \_ RC4 \_ 128 \_ MD5
-   TLS- \_ RSA \_ mit \_ RC4 \_ 128 \_ SHA
-   TLS \_ RSA \_ mit \_ 3DES \_ Ede \_ CBC \_ SHA
-   TLS \_ dhe \_ DSS \_ mit \_ 3DES \_ Ede \_ CBC \_ SHA
-   TLS- \_ RSA \_ mit \_ des \_ CBC \_ SHA
-   TLS \_ dhe \_ DSS \_ mit \_ des \_ CBC \_ SHA
-   TLS- \_ RSA- \_ EXPORT1024 \_ mit \_ RC4 \_ 56 \_ SHA
-   TLS- \_ RSA- \_ EXPORT1024 \_ mit \_ des \_ CBC \_ SHA
-   TLS \_ dhe \_ DSS \_ EXPORT1024 \_ mit \_ des \_ CBC \_ SHA
-   TLS- \_ RSA- \_ Export \_ mit \_ RC4 \_ 40 \_ MD5
-   TLS- \_ RSA- \_ Export \_ mit \_ RC2 \_ CBC \_ 40 \_ MD5
-   TLS- \_ RSA \_ mit \_ NULL \_ MD5
-   TLS- \_ RSA \_ mit NULL- \_ \_ SHA
