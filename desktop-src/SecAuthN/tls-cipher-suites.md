# <a name="tls-cipher-suites"></a>TLS-Verschlüsselungssammlungen

Schannel unterstützt die folgenden Verschlüsselungssammlungen für TLS 1.0 in Windows Server 2003 und Windows XP. Die Sammlungen werden in der Standardreihenfolge aufgeführt, in der sie ausgewählt werden. Informationen zu Verschlüsselungssammlungen für Windows Server 2008 und Windows Vista finden Sie unter [Verschlüsselungssammlungen in Schannel.](cipher-suites-in-schannel.md)

-   TLS \_ RSA \_ MIT \_ RC4 \_ 128 \_ MD5
-   TLS \_ RSA \_ MIT \_ RC4 \_ 128 \_ SHA
-   TLS \_ RSA \_ MIT \_ 3DES \_ EDE \_ CBC \_ SHA
-   TLS \_ DHE \_ DSS \_ MIT \_ 3DES \_ EDE \_ CBC \_ SHA
-   TLS \_ RSA MIT DES \_ \_ \_ CBC \_ SHA
-   TLS \_ DHE \_ DSS \_ MIT DES \_ \_ CBC \_ SHA
-   TLS \_ RSA \_ EXPORT1024 \_ MIT \_ RC4 \_ 56 \_ SHA
-   TLS \_ RSA \_ EXPORT1024 \_ MIT DES \_ \_ CBC \_ SHA
-   TLS \_ DHE \_ DSS \_ EXPORT1024 \_ MIT DES \_ \_ CBC \_ SHA
-   TLS \_ RSA \_ EXPORT \_ WITH \_ RC4 \_ 40 \_ MD5
-   TLS \_ RSA \_ EXPORT \_ WITH \_ RC2 \_ CBC \_ 40 \_ MD5
-   TLS \_ RSA MIT NULL \_ \_ \_ MD5
-   TLS \_ RSA MIT NULL \_ \_ \_ SHA
