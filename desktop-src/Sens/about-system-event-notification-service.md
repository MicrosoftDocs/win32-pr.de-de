# <a name="about-system-event-notification-service"></a>Informationen zum System Ereignis Benachrichtigungsdienst

Anwendungen, die für die Verwendung durch mobile Benutzer entwickelt wurden, benötigen einen eindeutigen Satz von Konnektivitätsfunktionen und-Benachrichtigungen, und in der Vergangenheit waren individuelle Anwendungen erforderlich, um diese Features intern zu implementieren. Der System Event Notification Service (Sens) stellt diese Funktionen jetzt im Betriebs System bereit und erstellt eine einheitliche Konnektivität und Benachrichtigungs Schnittstelle für Anwendungen. Mit dem Synchronisierungs-Manager und dem Client seitigen Caching wird die Infrastruktur für die vollständige Unterstützung mobiler Computing bereitgestellt.
