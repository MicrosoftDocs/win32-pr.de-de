# <a name="i-menus-and-other-resources"></a>I (Menüs und andere Ressourcen)

[a](a.md) [B](b.md) [C](c.md) D [E](e.md) [F](f.md) G H I J K L M [N](n.md) [O](o.md) P Q [R](r.md) S [T](t.md) [U](u.md) [V](v.md) [W](w.md) X Y Z

<dl> <dt>

<span id="tools.i_1_gly"></span><span id="TOOLS.I_1_GLY"></span>**Ungültige Zugriffstaste**
</dt> <dd>

Ein *Ereignis* Parameter in der [**Accelerators**](accelerators-resource.md) -Anweisung wurde nicht erkannt oder ist mehr als zwei Zeichen lang.

</dd> <dt>

<span id="tools.i_2_gly"></span><span id="TOOLS.I_2_GLY"></span>**Ungültiger Accelerator-Typ (ASCII oder VIRTKEY)**
</dt> <dd>

Der *Typparameter* in der [**Accelerators**](accelerators-resource.md) -Anweisung muss entweder den **ASCII** -oder den **VIRTKEY** -Wert enthalten.

</dd> <dt>

<span id="tools.i_3_gly"></span><span id="TOOLS.I_3_GLY"></span>**Ungültiges Steuerzeichen**
</dt> <dd>

Ein Steuerzeichen in der [**Accelerators**](accelerators-resource.md) -Anweisung ist ungültig. Ein gültiges Steuerzeichen besteht aus einem Caretzeichen (^), gefolgt von einem einzelnen Buchstaben.

</dd> <dt>

<span id="tools.i_4_gly"></span><span id="TOOLS.I_4_GLY"></span>**Ungültiger Steuerungstyp**
</dt> <dd>

Die Control-Anweisung in [**einer Dialog**](dialog-resource.md) -Anweisung muss eine der folgenden sein [**: AUTO3STATE**](auto3state-control.md), [**AUTOCHECKBOX**](autocheckbox-control.md), [**AUTORADIOBUTTON**](autoradiobutton-control.md), [**CheckBox**](checkbox-control.md), [**ComboBox**](combobox-control.md), [**Control**](control-control.md), [**CTEXT**](ctext-control.md), [**DEFPUSHBUTTON**](defpushbutton-control.md), [**EDITTEXT**](edittext-control.md), [**GroupBox**](groupbox-control.md), [**Symbol**](icon-control.md), [**ListBox**](listbox-control.md), [**LTEXT**](ltext-control.md), [**PUSHBOX**](pushbox-control.md), [**PUSHBUTTON**](pushbutton-control.md), [**RadioButton**](radiobutton-control.md), [**rtext**](rtext-control.md), [**ScrollBar**](scrollbar-control.md), [**STATE3**](state3-control.md).

</dd> <dt>

<span id="tools.i_5_gly"></span><span id="TOOLS.I_5_GLY"></span>**Ungültige Direktive in der vorverarbeiteten RC-Datei.**
</dt> <dd>

Der angegebene Dateiname enthält ein eingebettetes Zeilen Trennzeichen.

</dd> <dt>

<span id="tools.i_6_gly"></span><span id="TOOLS.I_6_GLY"></span>**Ungültig. EXE-Datei**
</dt> <dd>

Die ausführbare Datei (exe-Datei) ist ungültig. Stellen Sie sicher, dass der Linker ihn ordnungsgemäß erstellt hat und dass die Datei vorhanden ist.

</dd> <dt>

<span id="tools.i_7_gly"></span><span id="TOOLS.I_7_GLY"></span>**Ungültiger Switch, Option**
</dt> <dd>

Eine verwendete Option war ungültig. Wenn Sie eine Liste der Befehlszeilenoptionen verwenden, verwenden Sie die **RC/?** ausführen.

</dd> <dt>

<span id="tools.i_8_gly"></span><span id="TOOLS.I_8_GLY"></span>**Ungültiger Switch, zu viele D-Schalter**
</dt> <dd>

Es wurden zu viele **d-** Optionen angegeben. Wenn Sie eine große Anzahl von Symbolen definieren möchten, verwenden Sie die [**\# include**](-include.md) -Direktive, um eine Datei mit Definitionen einzuschließen.

</dd> <dt>

<span id="tools.i_9_gly"></span><span id="TOOLS.I_9_GLY"></span>**Ungültiger Typ**
</dt> <dd>

Der Ressourcentyp gehörte nicht zu den Typen, die in der Includedatei definiert sind.

</dd> <dt>

<span id="tools.i_10_gly"></span><span id="TOOLS.I_10_GLY"></span>**Ungültige Verwendung. Verwenden Sie RC-? Hilfe**
</dt> <dd>

Stellen Sie sicher, dass mindestens ein Dateiname vorhanden ist, mit dem Sie arbeiten können. Wenn Sie eine Liste der Befehlszeilenoptionen verwenden, verwenden Sie die **RC-?** ausführen.

</dd> <dt>

<span id="tools.i_11_gly"></span><span id="TOOLS.I_11_GLY"></span>**E/a-Fehler beim Lesen der Datei.**
</dt> <dd>

Fehler beim Lesen. Da es sich hierbei um eine generische Routine handelt, wird kein spezifischer Dateiname angegeben.

</dd> <dt>

<span id="tools.i_12_gly"></span><span id="TOOLS.I_12_GLY"></span>**E/a-Fehler beim Suchen in der Datei**
</dt> <dd>

Fehler beim Suchen in der Datei. Da es sich hierbei um eine generische Routine handelt, wird kein spezifischer Dateiname angegeben.

</dd> <dt>

<span id="tools.i_13_gly"></span><span id="TOOLS.I_13_GLY"></span>**E/a-Fehler beim Schreiben der Datei.**
</dt> <dd>

Fehler beim Schreiben. Da es sich hierbei um eine generische Routine handelt, wird kein spezifischer Dateiname angegeben.

</dd> </dl>
