# <a name="i-menus-and-other-resources"></a>I (Menüs und andere Ressourcen)

<dl> <dt>

<span id="tools.i_1_gly"></span><span id="TOOLS.I_1_GLY"></span>**Ungültiger Zugriffstastenbeschle**
</dt> <dd>

Ein *Ereignisparameter* in der [**ACCELERATORS-Anweisung**](accelerators-resource.md) wurde nicht erkannt oder war mehr als zwei Zeichen lang.

</dd> <dt>

<span id="tools.i_2_gly"></span><span id="TOOLS.I_2_GLY"></span>**Ungültiger Zugriffstastentyp (ASCII oder VIRTKEY)**
</dt> <dd>

Der *Typparameter* in der [**ACCELERATORS-Anweisung**](accelerators-resource.md) muss entweder den **ASCII- oder** **VIRTKEY-Wert** enthalten.

</dd> <dt>

<span id="tools.i_3_gly"></span><span id="TOOLS.I_3_GLY"></span>**Ungültiges Steuerzeichen**
</dt> <dd>

Ein Steuerzeichen in der [**ACCELERATORS-Anweisung**](accelerators-resource.md) ist ungültig. Ein gültiges Steuerzeichen besteht aus einem Caretzeichen (^), gefolgt von einem einzelnen Buchstaben.

</dd> <dt>

<span id="tools.i_4_gly"></span><span id="TOOLS.I_4_GLY"></span>**Ungültiger Steuerelementtyp**
</dt> <dd>

Die Control-Anweisung in einer [**DIALOG-Anweisung**](dialog-resource.md) muss eine der folgenden Sein: [**AUTO3STATE**](auto3state-control.md), [**AUTOCHECKBOX**](autocheckbox-control.md), [**AUTOGRAMMBUTTON**](autoradiobutton-control.md), [**CHECKBOX**](checkbox-control.md), [**COMBOBOX**](combobox-control.md), [**CONTROL**](control-control.md), [**CTEXT**](ctext-control.md), [**DEFPUSHBUTTON**](defpushbutton-control.md), [**EDITTEXT**](edittext-control.md), [**GROUPBOX**](groupbox-control.md), [**ICON**](icon-control.md), [**LISTBOX**](listbox-control.md), [**LTEXT**](ltext-control.md), PUSHBOX , [**PUSHBUTTON**](pushbutton-control.md), [**RADIOBUTTON**](radiobutton-control.md), [**RTEXT**](rtext-control.md), [**SCROLLBAR**](scrollbar-control.md), [**STATE3**](state3-control.md). [](pushbox-control.md)

</dd> <dt>

<span id="tools.i_5_gly"></span><span id="TOOLS.I_5_GLY"></span>**Ungültige Direktive in vorverarbeiteter RC-Datei**
</dt> <dd>

Der angegebene Dateiname verfügt über ein eingebettetes Neue-Zeichen.

</dd> <dt>

<span id="tools.i_6_gly"></span><span id="TOOLS.I_6_GLY"></span>**Ungültige .EXE Datei**
</dt> <dd>

Die ausführbare Datei (.exe) ist ungültig. Stellen Sie sicher, dass der Linker ihn ordnungsgemäß erstellt hat und dass die Datei vorhanden ist.

</dd> <dt>

<span id="tools.i_7_gly"></span><span id="TOOLS.I_7_GLY"></span>**Ungültiger Schalter,Option**
</dt> <dd>

Eine verwendete Option war ungültig. Eine Liste der Befehlszeilenoptionen finden Sie unter **rc /?** ausführen.

</dd> <dt>

<span id="tools.i_8_gly"></span><span id="TOOLS.I_8_GLY"></span>**Ungültiger Schalter, zu viele -D-Schalter**
</dt> <dd>

Es wurden **zu viele -d-Optionen** angegeben. Um eine große Anzahl von Symbolen zu definieren, verwenden Sie die [**\# include-Direktive,**](-include.md) um eine Datei mit Definitionen einderherzustellen.

</dd> <dt>

<span id="tools.i_9_gly"></span><span id="TOOLS.I_9_GLY"></span>**Ungültiger Typ**
</dt> <dd>

Der Ressourcentyp gehört nicht zu den typen, die in der Includedatei definiert sind.

</dd> <dt>

<span id="tools.i_10_gly"></span><span id="TOOLS.I_10_GLY"></span>**Ungültige Verwendung. Verwenden Sie RC - ? hilfe**
</dt> <dd>

Stellen Sie sicher, dass Sie über mindestens einen Dateinamen verfügen, mit dem Sie arbeiten können. Eine Liste der Befehlszeilenoptionen finden Sie unter **rc -?** ausführen.

</dd> <dt>

<span id="tools.i_11_gly"></span><span id="TOOLS.I_11_GLY"></span>**E/A-Fehler beim Lesen der Datei**
</dt> <dd>

Fehler beim Lesen. Da es sich um eine generische Routine handelt, wird kein bestimmter Dateiname angegeben.

</dd> <dt>

<span id="tools.i_12_gly"></span><span id="TOOLS.I_12_GLY"></span>**E/A-Fehler beim Suchen in einer Datei**
</dt> <dd>

Fehler beim Suchen in der Datei. Da es sich um eine generische Routine handelt, wird kein bestimmter Dateiname angegeben.

</dd> <dt>

<span id="tools.i_13_gly"></span><span id="TOOLS.I_13_GLY"></span>**E/A-Fehler beim Schreiben der Datei**
</dt> <dd>

Fehler beim Schreiben. Da es sich um eine generische Routine handelt, wird kein bestimmter Dateiname angegeben.

</dd> </dl>
