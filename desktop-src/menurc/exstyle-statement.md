# <a name="exstyle-statement"></a>ExStyle-Anweisung

Definiert erweiterte Fenster Stile für ein Dialogfeld. In einer Ressourcendefinition wird die **ExStyle** -Anweisung mit den optionalen Anweisungen vor dem Anfang des Texts der Ressourcendefinition platziert.

``` syntax
EXSTYLE extended-style
```

<dl> <dt>

<span id="extended-style"></span><span id="EXTENDED-STYLE"></span>*Erweiterter Stil*
</dt> <dd>

Erweiterte Fenster Stil für das Dialogfeld oder Steuerelement. Eine Liste erweiterter Fenster Stile finden Sie unter [**Erweiterte Fenster Stile**](/windows/desktop/winmsg/extended-window-styles).

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

Für Steuerelemente werden erweiterte Stile nach der *Style* -Option in der Control Resource-Definition-Anweisung angegeben. Weitere Informationen finden Sie in der Resource-Definition-Anweisung für das jeweilige Steuerelement.
