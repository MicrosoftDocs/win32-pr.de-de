# <a name="preprocessor-operators"></a>Präprozessoroperatoren

RC unterstützt die Verwendung der standardmäßigen C-Vorverarbeitungsoperatoren in Makrodefinitionen. Diese Operatoren werden in der folgenden Tabelle beschrieben.



| Operator | Beschreibung                                                 |
|----------|-------------------------------------------------------------|
| \#       | Schließt das Argument in Anführungszeichen ein.                            |
| \#@      | Schließt das Argument in einfache Anführungszeichen ein.                     |
| \#\#     | Verkettet Token, die als Argumente verwendet werden, um andere Token zu bilden. |
