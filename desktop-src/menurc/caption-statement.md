# <a name="caption-statement"></a>Caption-Anweisung

Definiert den Titel für ein Dialogfeld. Der Titel wird in der Beschriftungs Leiste des Felds angezeigt (sofern vorhanden).

Die Standard Beschriftung ist leer.

``` syntax
CAPTION "captiontext"
```

<dl> <dt>

<span id="captiontext"></span><span id="CAPTIONTEXT"></span>*CaptionText*
</dt> <dd>

Eine Zeichenfolge, die in doppelten Anführungszeichen (") eingeschlossen ist.

</dd> </dl>

## <a name="examples"></a>Beispiele

Im folgenden Beispiel wird die Verwendung der **Caption** -Anweisung veranschaulicht:

``` syntax
CAPTION "Error!" 
```
