# <a name="displaying-progress-messages"></a>Anzeigen von Statusmeldungen

Standardmäßig wird RC kompiliert, ohne Statusmeldungen anzuzeigen. Sie können jedoch angeben, dass RC Statusmeldungen anzeigen soll. Verwenden Sie hierzu die Option **/v.**

Im folgenden Beispiel meldet RC seinen Fortschritt, während die Ressourcendefinitionsdatei Sample.rc kompiliert und die kompilierte Ressourcendatei Sample.res erstellt wird:

**rc /v sample.rc**
