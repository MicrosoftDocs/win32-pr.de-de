<dl> <dt>

<span id="tools.w_1_gly"></span><span id="TOOLS.W_1_GLY"></span>**Warnung: ASCII-Zeichen entspricht nicht dem Code des virtuellen Schlüssels**
</dt> <dd>

In der [**Accelerators**](accelerators-resource.md) -Anweisung ist ein ungültiger Code für den virtuellen Schlüssel vorhanden. Die ASCII-Werte für einige Zeichen, z. b. \* , ^ oder &, entsprechen nicht den virtuellen Schlüsselcodes für die entsprechenden Schlüssel. Im Fall des Sternchen ( \* ) entspricht der Code des virtuellen Schlüssels dem ASCII-Wert für 8, dem numerischen Zeichen für den gleichen Schlüssel.

</dd> <dt>

<span id="tools.w_2_gly"></span><span id="TOOLS.W_2_GLY"></span>**Warnung: die Verschiebung oder das Steuerelement wird ohne VIRTKEY verwendet.**
</dt> <dd>

Die Optionen **alt**, **UMSCHALT** und [**Steuer**](control-control.md) Element gelten nur für virtuelle Schlüssel in der [**Accelerators**](accelerators-resource.md) -Anweisung. Stellen Sie sicher, dass die Option **VIRTKEY** mit einer dieser anderen Optionen verwendet wird.

</dd> <dt>

<span id="tools.w_3_gly"></span><span id="TOOLS.W_3_GLY"></span>**Schreiben von resourcetypeimagelang: languagesize: size**
</dt> <dd>

(V) der Microsoft Windows-Ressourcen Compiler (RC) schreibt die angegebene Ressource. Der *Typ* ist der Ressourcentyp. Dabei kann es sich um eine Zahl oder eine Zeichenfolge handeln. Das *Bild* kann eine Zahl oder eine Zeichenfolge sein. Die *Sprache* ist die nationale Sprache der Ressource. Die *Größe* entspricht der Dezimal Größe der Ressource in Bytes.

</dd> </dl>
