# <a name="rc-diagnostic-messages"></a>RC-Diagnosemeldungen

In diesem Abschnitt werden von RC erzeugte Diagnosemeldungen beschrieben. Viele dieser Meldungen werden angezeigt, wenn RC Ressourcen nicht ordnungsgemäß kompilieren kann. Die Beschreibungen verdeutlichen die Ursache der einzelnen Nachrichten.

Ein Groß-V in Klammern (V) am Anfang einer Meldungsbeschreibung gibt an, dass die Meldung nur angezeigt wird, wenn RC mit der **Option /v** (ausführlich) ausgeführt wird. Diese Meldungen sind im Allgemeinen informativ und weisen nicht notwendigerweise auf Fehler hin.

Die Nachrichten sind in den folgenden Themen in alphabetischer Reihenfolge aufgeführt:

[A](a.md) [B](b.md) [C](c.md) [E](e.md) [F](f.md) [I](i.md) [N](n.md) [O](o.md) [R](r.md) [T](t.md) [U](u.md) [V](v.md) [W](w.md)
