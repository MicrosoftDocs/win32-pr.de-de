# <a name="rc-diagnostic-messages"></a>RC-Diagnosemeldungen

In diesem Abschnitt werden Diagnosemeldungen beschrieben, die von RC generiert werden. Viele dieser Meldungen werden angezeigt, wenn RC Ressourcen nicht ordnungsgemäß kompilieren kann. Die Beschreibungen verdeutlichen die Ursache der einzelnen Nachrichten.

Ein Capital v in Klammern (v) am Anfang einer Nachrichten Beschreibung gibt an, dass die Meldung nur angezeigt wird, wenn RC mit der Option **/v** (verbose) ausgeführt wird. Diese Nachrichten sind im allgemeinen informativ und weisen nicht unbedingt auf Fehler hin.

Die Nachrichten werden in alphabetischer Reihenfolge in den folgenden Themen aufgeführt:
