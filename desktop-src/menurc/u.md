# <a name="u-menus-and-other-resources"></a>U (Menüs und andere Ressourcen)

[a](a.md) [B](b.md) [C](c.md) D [E](e.md) [F](f.md) G H [I](i.md) J K L M [N](n.md) [O](o.md) P Q [R](r.md) S [T](t.md) U [V](v.md) [W](w.md) X Y Z

<dl> <dt>

<span id="tools.u_1_gly"></span><span id="TOOLS.U_1_GLY"></span>**Fehler beim Generieren von "Ziel".**
</dt> <dd>

Der Microsoft Windows-Ressourcen Compiler (RC) konnte die Zieldatei nicht erstellen. Stellen Sie sicher, dass genügend Speicherplatz vorhanden ist.

</dd> <dt>

<span id="tools.u_2_gly"></span><span id="TOOLS.U_2_GLY"></span>**Unausgeglichene Klammern**
</dt> <dd>

Stellen Sie sicher, dass Sie jede öffnende Klammer in der [**Dialog**](dialog-resource.md) Feld Anweisung geschlossen haben.

</dd> <dt>

<span id="tools.u_3_gly"></span><span id="TOOLS.U_3_GLY"></span>**Unerwarteter Wert in RCDATA.**
</dt> <dd>

Die Werte für den *RAW-Data-* Parameter in der [**RCDATA**](rcdata-resource.md) -Anweisung müssen ganze Zahlen oder Zeichen folgen sein, die durch Kommas getrennt sind. Stellen Sie sicher, dass Sie kein Komma oder Anführungszeichen um eine Zeichenfolge auslassen.

</dd> <dt>

<span id="tools.u_4_gly"></span><span id="TOOLS.U_4_GLY"></span>**Unerwarteter Wert in Wertdaten.**
</dt> <dd>

Eine-Anweisung enthielt Informationen mit einem Format oder einer anderen Größe als der erwartete Wert für diesen Parameter.

</dd> <dt>

<span id="tools.u_5_gly"></span><span id="TOOLS.U_5_GLY"></span>**Unbekanntes DIB-Header Format**
</dt> <dd>

Der geräteunabhängige Bitmap-Header (DIB) ist keine [**BITMAPCOREHEADER**](/windows/win32/api/wingdi/ns-wingdi-bitmapcoreheader) -oder [**BITMAPINFOHEADER**](/previous-versions//dd183376(v=vs.85)) -Struktur.

</dd> <dt>

<span id="tools.u_6_gly"></span><span id="TOOLS.U_6_GLY"></span>**Unbekannter Menü Untertyp.**
</dt> <dd>

Der *Item-Definitions* -Parameter der [**Menu**](menu-resource.md) -Anweisung kann nur [**MenuItem**](menuitem-statement.md) -und [**Popup**](popup-resource.md) -Anweisungen enthalten.

</dd> <dt>

<span id="tools.u_7_gly"></span><span id="TOOLS.U_7_GLY"></span>**Unbekanntes VERSIONINFO-Feld. Anfang oder Komma erwartet**
</dt> <dd>

Das Format der Informationen, die auf eine [**VERSIONINFO**](versioninfo-resource.md) -Anweisung folgen, ist falsch.

</dd> </dl>
