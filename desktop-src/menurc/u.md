# <a name="u-menus-and-other-resources"></a>U (Menüs und andere Ressourcen)

[A](a.md) [B](b.md) [C](c.md) D [E](e.md) [F](f.md) G H [I](i.md) J K L M [N](n.md) [O](o.md) P Q [R](r.md) S [T](t.md) U [V](v.md) [W](w.md) X Y Z

<dl> <dt>

<span id="tools.u_1_gly"></span><span id="TOOLS.U_1_GLY"></span>**Createdestination kann nicht erstellt werden**
</dt> <dd>

Der Microsoft Windows Resource Compiler (RC) konnte die Zieldatei nicht erstellen. Stellen Sie sicher, dass genügend Speicherplatz vorhanden ist.

</dd> <dt>

<span id="tools.u_2_gly"></span><span id="TOOLS.U_2_GLY"></span>**Unausgewogene Klammern**
</dt> <dd>

Stellen Sie sicher, dass Sie alle öffnenden Klammern in der [**DIALOG-Anweisung**](dialog-resource.md) geschlossen haben.

</dd> <dt>

<span id="tools.u_3_gly"></span><span id="TOOLS.U_3_GLY"></span>**Unerwarteter Wert in RCData**
</dt> <dd>

Die Werte für den *Raw-Data-Parameter* in der [**RCDATA-Anweisung**](rcdata-resource.md) müssen ganze Zahlen oder Zeichenfolgen sein, die durch Kommas getrennt sind. Stellen Sie sicher, dass Sie kein Komma oder Anführungszeichen um eine Zeichenfolge weglassen.

</dd> <dt>

<span id="tools.u_4_gly"></span><span id="TOOLS.U_4_GLY"></span>**Unerwarteter Wert in Wertdaten**
</dt> <dd>

Eine -Anweisung enthielt Informationen mit einem anderen Format oder einer anderen Größe als der erwartete Wert für diesen Parameter.

</dd> <dt>

<span id="tools.u_5_gly"></span><span id="TOOLS.U_5_GLY"></span>**Unbekanntes DIB-Headerformat**
</dt> <dd>

Der DIB-Header (Device-Independent Bitmap) ist keine [**BITMAPCOREHEADER-**](/windows/win32/api/wingdi/ns-wingdi-bitmapcoreheader) oder [**BITMAPINFOHEADER-Struktur.**](/previous-versions//dd183376(v=vs.85))

</dd> <dt>

<span id="tools.u_6_gly"></span><span id="TOOLS.U_6_GLY"></span>**Unbekannter Menüuntertyp**
</dt> <dd>

Der *Elementdefinitionsparameter* der [**MENU-Anweisung**](menu-resource.md) darf nur [**MENUITEM-**](menuitem-statement.md) und [**POPUP-Anweisungen**](popup-resource.md) enthalten.

</dd> <dt>

<span id="tools.u_7_gly"></span><span id="TOOLS.U_7_GLY"></span>**Nicht erkanntes VERSIONSINFO-Feld; BEGIN oder Komma erwartet**
</dt> <dd>

Das Format der Informationen nach einer [**VERSIONINFO-Anweisung**](versioninfo-resource.md) ist falsch.

</dd> </dl>
