# <a name="menu-statement"></a>Menü Anweisung

Definiert das Menü für ein Dialogfeld. Wenn keine-Anweisung angegeben wird, enthält das Dialogfeld kein Menü.

``` syntax
MENU menuname
```

<dl> <dt>

<span id="menuname"></span><span id="MENUNAME"></span>*MENUNAME darf*
</dt> <dd>

Der Name oder Bezeichner des Menüs, das verwendet werden soll.

</dd> </dl>

## <a name="examples"></a>Beispiele

Im folgenden Beispiel wird die Verwendung der **Menü** Dialog-Anweisung veranschaulicht:

``` syntax
MENU errmenu
```
