<span id="tools.t_1_gly"></span><span id="TOOLS.T_1_GLY"></span>**Textzeichenfolge oder Ordnungszahl in Control erwartet**
</dt> <dd>

Der *Textparameter* einer [**CONTROL-Anweisung**](control-control.md) in der [**DIALOG-Anweisung**](dialog-resource.md) muss entweder eine Textzeichenfolge oder ein Ordinalverweis auf den erwarteten Steuerelementtyp sein. Wenn Sie eine Ordnungszahl verwenden, stellen Sie sicher, dass Sie über eine [**\# define-Anweisung**](-define.md) für das Steuerelement verfügen.
