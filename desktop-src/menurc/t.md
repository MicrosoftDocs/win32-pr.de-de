<span id="tools.t_1_gly"></span><span id="TOOLS.T_1_GLY"></span>**Text Zeichenfolge oder Ordnungszahl im Steuerelement erwartet**
</dt> <dd>

Der *Text* -Parameter einer [**Control**](control-control.md) -Anweisung in der [**Dialog**](dialog-resource.md) -Anweisung muss entweder eine Text Zeichenfolge oder ein Ordinalverweis auf den erwarteten Typ des Steuer Elements sein. Wenn Sie eine Ordinalzahl verwenden, stellen Sie sicher, dass Sie über eine [**\# define**](-define.md) -Anweisung für das Steuerelement verfügen.
