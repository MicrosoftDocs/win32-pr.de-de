<span id="tools.c_1_gly"></span><span id="TOOLS.C_1_GLY"></span>**Zeichenfolgenkonstanten können nicht erneut verwendet werden**
</dt> <dd>

Sie verwenden denselben Wert zweimal in einer [**STRINGTABLE-Anweisung.**](stringtable-resource.md) Stellen Sie sicher, dass Sie keine überlappenden Dezimal- und Hexadezimalwerte gemischt haben.

</dd> <dt>

<span id="tools.c_2_gly"></span><span id="TOOLS.C_2_GLY"></span>**Zeichen außerhalb von Bereich \[ A steuern? Z\]**
</dt> <dd>

Ein Steuerelementzeichen in der [**ACCELERATORS-Anweisung**](accelerators-resource.md) ist ungültig. Das Zeichen nach dem Caretzeichen (^) muss im Bereich A bis Z liegen.

</dd> <dt>

<span id="tools.c_3_gly"></span><span id="TOOLS.C_3_GLY"></span>**Openin-file-name konnte nicht geöffnet werden.**
</dt> <dd>

Der Microsoft Windows Resource Compiler (RC) konnte die angegebene Datei nicht öffnen. Stellen Sie sicher, dass die Datei vorhanden ist und Sie den Dateinamen richtig eingegeben haben.

</dd> <dt>

<span id="tools.c_4_gly"></span><span id="TOOLS.C_4_GLY"></span>**OpenResource-name konnte nicht geöffnet werden**
</dt> <dd>

</dd> <dt>
