# <a name="menus-and-other-resources"></a>Menüs und weitere Ressourcen

Eine *Ressource* sind binäre Daten, die Sie der ausführbaren Datei einer Windows-basierten Anwendung hinzufügen können. Eine Ressource kann entweder "Standard" oder "definiert" sein. Die Daten in einer *Standardressource* beschreiben Symbol, Cursor, Menü, Dialogfeld, Bitmap, erweiterte Metadatei, Schriftart, Zugriffstasten Tabelle, Nachrichten Tabelleneintrag, Zeichen folgen Tabelleneintrag oder Versionsinformationen. Eine von der *Anwendung definierte Ressource*, auch als *benutzerdefinierte Ressource* bezeichnet, enthält alle Daten, die von einer bestimmten Anwendung benötigt werden.
