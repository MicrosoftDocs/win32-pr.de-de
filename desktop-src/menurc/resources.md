# <a name="menus-and-other-resources"></a>Menüs und weitere Ressourcen

Eine *Ressource* sind Binärdaten, die Sie der ausführbaren Datei einer Windows Anwendung hinzufügen können. Eine Ressource kann entweder standard oder definiert sein. Die Daten  in einer Standardressource beschreiben ein Symbol, einen Cursor, ein Menü, ein Dialogfeld, eine Bitmap, eine erweiterte Metadatei, eine Schriftart, eine Zugriffstastentabelle, einen Meldungstabelleneintrag, einen Zeichenfolgentabelleneintrag oder Versionsinformationen. Eine *anwendungsdefinierte Ressource*, auch als benutzerdefinierte *Ressource bezeichnet,* enthält alle Daten, die von einer bestimmten Anwendung benötigt werden.
