# <a name="resource-compiler"></a>Ressourcencompiler

Beim Microsoft Windows-Ressourcen Compiler (RC) handelt es sich um ein Tool, mit dem Windows-basierte Anwendungen aufgebaut werden. In dieser Übersicht wird beschrieben, wie eine Ressourcen Definitionsdatei (Skript) erstellt wird, wie die Ressourcen der Anwendung kompiliert werden und wie kompilierte Ressourcen zu Ihrer Anwendung hinzugefügt werden.

Dieses Tool ist in Visual Studio und im Microsoft Windows Software Development Kit (SDK) verfügbar.

In den folgenden Themen werden die Ressourcen Datei Syntax und die RC-Befehlszeile beschrieben:

-   [Informationen zu Ressourcen Dateien](about-resource-files.md)
-   [Verwenden des Ressourcencompilers (RC-Befehlszeile)](using-rc-the-rc-command-line-.md)
