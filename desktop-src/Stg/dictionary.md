# <a name="dictionary"></a>Wörterbuch

[**IPropertyStorage:: Beschreib tepropertynames**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-writepropertynames) wird mithilfe des [Wörterbuchs der Eigenschaft "ID 0](/windows/desktop/Stg/reserved-property-identifiers)" implementiert. Auf die eigen schafts-ID 0 kann nicht über [**IPropertyStorage:: Read Multiple**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-readmultiple) oder [**IPropertyStorage:: Write-Multiple**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-writemultiple)zugegriffen werden.
