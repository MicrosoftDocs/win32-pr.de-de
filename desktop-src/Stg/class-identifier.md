# <a name="class-identifier"></a>Klassen Bezeichner

Der Klassen Bezeichner (CLSID) ist das einer Klasse, die den programmgesteuerten Zugriff auf die Eigenschaftswerte anzeigen und/oder bereitstellen kann. Wenn keine solche Klasse vorhanden ist, wird empfohlen, dass Sie diesen Wert genauso wie die fmtid festlegen (siehe unten). Alternativ können Sie diesen Wert auf alle Nullen festlegen. die Verwendung von fmtid ermöglicht in Zukunft jedoch mehr Flexibilität.
