# <a name="class-identifier"></a>Klassenbezeichner

Der Klassenbezeichner (CLSID) ist der einer Klasse, die programmgesteuerten Zugriff auf die Eigenschaftswerte anzeigen und/oder bereitstellen kann. Wenn keine solche Klasse vorhanden ist, wird empfohlen, diesen Wert auf denselben Wert wie fmtid (siehe unten) zu setzen. Alternativ können Sie diesen Wert auf alle Nullen festlegen. die Verwendung von FMTID ermöglicht jedoch in Zukunft mehr Flexibilität.
