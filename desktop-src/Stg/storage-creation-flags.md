# <a name="storage-creation-flags"></a>Storage Erstellungsflags

Storage Erstellungsflags geben an, was COM tun soll, wenn ein vorhandenes Speicher-, Stream- oder Lockbyteobjekt den gleichen Namen wie ein neues Speicher- oder Streamobjekt hat, das Sie erstellen. Standardmäßig wird eine Fehlermeldung zurückgegeben, und das neue Objekt wird nicht erstellt. Sie können nur eines dieser Flags in einem bestimmten Erstellungsaufruf verwenden.
