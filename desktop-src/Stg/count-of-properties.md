# <a name="count-of-properties"></a>Anzahl von Eigenschaften

Dieser **DWORD** -Datentyp gibt die Anzahl der Eigenschaftswerte im-Abschnitt an. Ein Eigenschaften Satz kann eine beliebige Anzahl von Eigenschafts Werten enthalten. Leser müssen in der Lage sein, den Fall zu behandeln, in dem keine Eigenschaften vorhanden sind.
