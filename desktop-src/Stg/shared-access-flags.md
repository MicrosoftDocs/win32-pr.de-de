# <a name="shared-access-flags"></a>Shared Access Flags

Die Shared Access-Flags geben den Zugriffsgrad an, den andere Anwendungen auf ein Objekt haben können, das Ihre Anwendung öffnet. Sie können den Lesezugriff verweigern, den Schreibzugriff verweigern oder den zugriff verweigern. Sie können auch explizit angeben, dass kein Zugriffstyp verweigert wird.
