# <a name="shared-access-flags"></a>Shared Access-Flags

Die Shared Access-Flags geben den Grad des Zugriffs an, den andere Anwendungen auf ein Objekt haben können, das von der Anwendung geöffnet wird. Sie können Lesezugriff verweigern, Schreibzugriff verweigern oder den gesamten Zugriff verweigern. Sie können auch explizit angeben, dass kein Zugriffstyp verweigert werden soll.
