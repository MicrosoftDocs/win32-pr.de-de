# <a name="temporary-creation-flags"></a>Temporäre erstellungsflags

Das Flag für die temporäre Erstellung gibt an, dass die zugrunde liegende Datei automatisch zerstört werden soll, wenn das Stamm Speicher Objekt freigegeben wird. Diese Funktion ist besonders nützlich für das Erstellen temporärer Dateien.
