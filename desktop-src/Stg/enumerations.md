# <a name="enumerations-windows-update-agent"></a>Enumerationen (Windows Update-Agent)

Enumerationen in strukturiertem Speicher sind Datentypen, die aus einer Sequenz von benannten Werten in einer bestimmten Reihenfolge bestehen. Ein enumerierter Datentyp ermöglicht, dass ein Member einer Klasse von Objekten in natürlicher Sprache beschrieben wird. Enumerationen sind besonders nützlich in dem Kontext, in dem alle möglichen Werte einer Variablen bekannt sind und der Ausdruck dieser Variablen in der Programmiersprache in natürlicher Sprache ausgedrückt werden kann.

Die strukturierten Storage-Enumerationen werden in den folgenden Abschnitten beschrieben.

-   [**LOCKTYPE**](/windows/win32/api/objidl/ne-objidl-locktype)
-   [**STATFLAG**](/windows/win32/api/wtypes/ne-wtypes-statflag)
-   [**Stgc**](/windows/win32/api/wtypes/ne-wtypes-stgc)
-   [**Stgfmt**](/previous-versions/windows/desktop/legacy/aa380330(v=vs.85))
-   [**Stgmove**](/windows/win32/api/wtypes/ne-wtypes-stgmove)
-   [**STGTY**](/windows/win32/api/objidl/ne-objidl-stgty)
-   [**Stream \_ Seek**](/windows/win32/api/objidl/ne-objidl-stream_seek)
