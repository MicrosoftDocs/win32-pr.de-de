# <a name="access-permission-flags"></a>Zugriffsberechtigungsflags

Zugriffsberechtigungsflags geben den Zugriffstyp an, den eine Clientanwendung auf ein geöffnetes Objekt hat: Lesen, Schreiben oder Lesen/Schreiben. Mit der Leseberechtigung kann eine Anwendung den Inhalt eines Streams lesen, aber keine Änderungen in den Stream schreiben. Mit der Schreibberechtigung kann eine Anwendung eine Funktion aufrufen, die Änderungen im Speicher eines Objekts committ, aber nicht den Inhalt des Streams liest. Mit der Lese-/Schreibberechtigung kann eine Anwendung entweder die Datenströme des Objekts lesen oder in den Speicher des Objekts schreiben.
