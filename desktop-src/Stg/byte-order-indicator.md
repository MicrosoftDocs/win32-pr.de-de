# <a name="byte-order-indicator"></a>Byte-Order Indikator

Der Byte Reihenfolge-Indikator ist ein **Word** -Datentyp und sollte immer den Wert 0xFFFE enthalten. Dies entspricht dem Unicode-Byte Reihenfolge Indikator. Dieser Wert wird immer in der Intel-Byte-Reihenfolge geschrieben und in der Datei oder im Stream als 0xFE, 0xFF angezeigt.
