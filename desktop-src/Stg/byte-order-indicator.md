# <a name="byte-order-indicator"></a>Byte-Order Indikator

Der Byteordnungsindikator ist ein **WORD-Datentyp** und sollte immer den Wert 0xFFFE. Dies ist identisch mit dem Unicode-Byte-Order-Indikator. Dieser Wert wird immer in Intel-Byte-Reihenfolge geschrieben und daher in der Datei oder im Stream als 0xFE, 0xFF.
