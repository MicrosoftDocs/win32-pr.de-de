# <a name="constants-structured-storage"></a>Konstanten (strukturierter Speicher)

Konstanten sind in strukturiertem Speicher eine Sequenz von benannten Werten. Konstanten ermöglichen das Beschreiben von Objekten in natürlicher Sprache.

Die strukturierten Speicher Konstanten werden in den folgenden Abschnitten beschrieben:

-   [Propsetflag-Konstanten](propsetflag-constants.md)
-   [STGM-Konstanten](stgm-constants.md)
