# <a name="structured-storage-client-sample-stoclien"></a>Beispiel für strukturierte Speicher Clients (stoclien)

Im **stoclien** -Beispiel wird eine einfache Zeichen Anwendung eingeführt. Diese Anwendung ermöglicht dem Benutzer, eine Maus oder ein Tablet-Gerät zu verwenden, um das frei Form Zeichen im Client Fenster zu verwenden. Die Farbe und Breite der elektronischen frei Hand Eingaben können ausgewählt werden, und die Zeichnungen können in Dateien gespeichert werden.

Wenn Sie das **stoclien** -Beispiel entwickeln und ausführen, finden Sie die notwendigen Informationen, um die strukturierte Speicher Client Struktur, ihre Verwendungsmöglichkeiten und Funktionen zu verstehen.
