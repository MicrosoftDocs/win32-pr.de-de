# <a name="allocating-format-identifiers"></a>Zuordnen von Format Bezeichnerzeichen

Fmtids werden auf die gleiche Weise wie com-CLSIDs und Schnittstellen Bezeichner erstellt und dargestellt. Verwenden Sie zum Erstellen einer eindeutigen fmtid das UUIDGEN.EXE Programm, das in der Win32-®-API enthalten ist.
