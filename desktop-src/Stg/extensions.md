# <a name="extensions"></a>Erweiterungen

Eigenschafts Satz Erweiterungen, wie im ursprünglichen com-Eigenschaften Satz Format definiert, wurden entfernt und werden nicht unterstützt, mit Ausnahme [der in den Eigenschaften Sätzen documentsummaryinformation und UserDefined definierten Eigenschaften](the-documentsummaryinformation-and-userdefined-property-sets.md).
