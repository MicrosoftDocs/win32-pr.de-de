# <a name="reserved"></a>Reserviert

Der Wert des **DWORD** -Datentyps ist für die zukünftige Verwendung reserviert. Writer von Eigenschafts Sätzen sollten diesen Wert auf 1 festlegen. Readern von Eigenschafts Sätzen sollten sicherstellen, dass dieser Wert mindestens 1 beträgt. Eine Ausnahme von dieser Richtlinie besteht darin, dass der Wert für [die documentsummaryinformation und die benutzerdefinierten Eigenschaften Sätze](the-documentsummaryinformation-and-userdefined-property-sets.md)2 sein kann.
