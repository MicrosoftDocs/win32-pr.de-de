# <a name="structures-structured-storage"></a>Strukturen (strukturierter Speicher)

Strukturen sind primär Sammlungen von Datenelementen oder Membern, die ein Organisationsschema zum Interpretieren von Daten oder zum Ausführen von Vorgängen vereinfachen. Die strukturierten Speicherstrukturen werden in den folgenden Abschnitten beschrieben:

-   [**PROPSPEC**](/windows/win32/api/propidlbase/ns-propidlbase-propspec)
-   [**PROPVARIANT**](/windows/win32/api/propidlbase/ns-propidlbase-propvariant)
-   [**Remsnb**](/windows/win32/api/objidl/ns-objidl-remsnb)
-   [**SNB**](snb.md)
-   [**Status-setstg**](/windows/win32/api/propidlbase/nn-propidlbase-ienumstatpropsetstg)
-   [**Status-g**](/windows/win32/api/propidlbase/nn-propidlbase-ienumstatpropstg)
-   [**Status g**](/windows/win32/api/objidl/ns-objidl-statstg)
-   [**Stgoptions**](/windows/win32/api/coml2api/ns-coml2api-stgoptions)
-   [**Storagelayout**](/windows/win32/api/objidl/ns-objidl-storagelayout)
