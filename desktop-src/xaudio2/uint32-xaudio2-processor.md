## <a name="remarks"></a>Hinweise

**XAUDIO2 \_ PROCESSOR** ist als 32-Bit-Wert definiert.

<div class="alert"><b>Hinweis:</b>  Wenn beim Aufrufen von <a href="/windows/desktop/api/xaudio2/nf-xaudio2-xaudio2create">XAudio2Create</a>mehrere Prozessorbits festgelegt werden, erstellt das System einen separaten Arbeitsthread für jeden Prozessor.</div>



```
typedef UINT32 XAUDIO2_PROCESSOR;
```
