# <a name="x3daudio"></a>X3DAudio

X3DAudio ist eine API, die in Verbindung mit [XAudio2](programming-guide.md) verwendet wird, um die Illusion eines Sounds zu erstellen, der von einem Punkt im 3D-Raum stammt.
