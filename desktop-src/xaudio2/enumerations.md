# <a name="xaudio2-enumerations"></a>XAudio2-Enumerationen

Dieser Abschnitt enthält Informationen zu Enumerationen, die von der Microsoft XAudio2-API bereitgestellt werden.



| Enumeration                                      | Beschreibung                                                                       |
|--------------------------------------------------|-----------------------------------------------------------------------------------|
| [\_XAUDIO2-FILTERTYP \_](/windows/desktop/api/xaudio2/ne-xaudio2-xaudio2_filter_type) | Gibt an, ob es sich bei einem Filter um einen Niedrigen Durchgang, bandpass, high pass oder notch handelt.<br/> |
| [\_XAPO-PUFFERFLAGS \_](/windows/desktop/api/xapo/ne-xapo-xapo_buffer_flags)     | Beschreibt den Inhalt eines Streampuffers.<br/>                                 |
