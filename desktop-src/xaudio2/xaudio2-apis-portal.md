# <a name="xaudio2-apis"></a>XAudio2-APIs

## <a name="purpose"></a>Zweck

XAudio2 ist eine Low-Level-Audio-API, die Signalverarbeitung und Mischungsgrundlage für die Entwicklung leistungsstarker Audio-Engines für Spiele bereitstellt.

## <a name="developer-audience"></a>Entwicklergruppe

XAudio2-APIs sind für die Verwendung durch Spieleentwickler konzipiert, die die Leistung in ihren Spielen verbessern möchten.
