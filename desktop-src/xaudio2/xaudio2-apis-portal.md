# <a name="xaudio2-apis"></a>XAudio2-APIs

## <a name="purpose"></a>Zweck

XAudio2 ist eine audioapi auf niedriger Ebene, die Signalverarbeitung und Vermischung der Grundlage für die Entwicklung Hochleistungs-audioengines für Spiele bereitstellt.
