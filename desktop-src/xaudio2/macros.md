# <a name="xaudio2-macros"></a>XAudio2-Makros

Dieser Abschnitt enthält Informationen zu Makros, die von der Microsoft XAudio2-API bereitgestellt werden.



| Makro                          | Beschreibung                                                                                                                                               |
|--------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XAPOAlloc**](/windows/desktop/api/XAPO/nf-xapo-xapoalloc) | Speicherzuordnungsmakro, das von IXAPO-Methoden verwendet wird, die Strukturen beliebiger Größe zuordnen müssen, die anschließend an die Anwendung zurückgegeben werden.<br/> |
| [**XAPOFree**](/windows/desktop/api/XAPO/nf-xapo-xapofree)   | Makro, das verwendet wird, um mit dem XAPOAlloc-Makro zugeordneten Arbeitsspeicher frei zu geben.<br/>                                                                                  |
