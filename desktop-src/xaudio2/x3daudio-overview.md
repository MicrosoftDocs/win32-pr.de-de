## <a name="listeners-and-emitters"></a><span data-ttu-id="5f601-106">Listener und Emitter</span><span class="sxs-lookup"><span data-stu-id="5f601-106">Listeners and Emitters</span></span>

<span data-ttu-id="5f601-121">Neben der Position können Listener und Emitter auch Velocity enthalten.</span><span class="sxs-lookup"><span data-stu-id="5f601-121">In addition to position, listeners and emitters can include velocity.</span></span> <span data-ttu-id="5f601-122">Anders als bei einem 3D-Renderingmodul verwendet X3DAudio nur Velocity zum Berechnen von Doppler Effekten (es wird nicht zum Berechnen der Position verwendet).</span><span class="sxs-lookup"><span data-stu-id="5f601-122">Unlike a 3D rendering engine, X3DAudio only uses velocity to calculate Doppler effects (it is not used to calculate position).</span></span>
