# <a name="vds-reference"></a>VDS-Referenz

\[Ab Windows 8 und Windows Server 2012 wird die COM-Schnittstelle des [virtuellen Festplatten Dienstanbieter](virtual-disk-service-portal.md) durch die [Windows-Speicherverwaltungs-API](/previous-versions/windows/desktop/stormgmt/windows-storage-management-api-portal)ersetzt.\]

In diesem Abschnitt wird die Anwendungsprogrammierschnittstelle für den virtuellen Datenträger Dienst (Virtual Disk Service, VDS) beschrieben.

Anwendungsentwickler verwenden die Schnittstellen, Strukturen, Enumerationen und symbolischen Konstanten in dieser API, um Speicher – von Datenträgern zu Speicherarrays abzufragen und zu konfigurieren. Eine ausführliche Betrachtung der Beziehung zwischen VDS-Typen finden Sie unter [VDS-Objektmodell](vds-object-model.md).

Speicherhersteller implementieren viele der Schnittstellen, die in diesem Abschnitt definiert sind.
