## <a name="overview"></a><span data-ttu-id="9bf1b-107">Übersicht</span><span class="sxs-lookup"><span data-stu-id="9bf1b-107">Overview</span></span>

<span data-ttu-id="9bf1b-108">Eine *Netzwerkschnittstelle* ist der Punkt, an dem zwei Teile der Netzwerkausrüstung oder Protokoll Ebenen eine Verbindung herstellen.</span><span class="sxs-lookup"><span data-stu-id="9bf1b-108">A *network interface* is the point where two pieces of network equipment or protocol layers connect.</span></span> <span data-ttu-id="9bf1b-109">Dies wird in der Regel durch eine physische Netzwerkschnittstellenkarte (NIC) für die Verbindung zwischen einem Computer und einem privaten oder öffentlichen Netzwerk repräsentiert.</span><span class="sxs-lookup"><span data-stu-id="9bf1b-109">Typically, this is represented by a physical Network Interface Card (NIC) for connection between a computer and a private or public network.</span></span> <span data-ttu-id="9bf1b-110">Es kann jedoch auch die Form einer reinen Softwarekomponente wie der Loopback Schnittstelle ( `127.0.0.1` für IPv4 oder `::1` für IPv6) annehmen.</span><span class="sxs-lookup"><span data-stu-id="9bf1b-110">However, it can also take the form of a software-only component such as the loopback interface (`127.0.0.1` for IPv4 or `::1` for IPv6).</span></span>

<span data-ttu-id="9bf1b-111">Netzwerkschnittstellen werden von der Internet Engineering Task Force (IETF) in [RFC 2863](https://tools.ietf.org/html/rfc2863) definiert und sind nicht für die Definition durch Windows gedacht.</span><span class="sxs-lookup"><span data-stu-id="9bf1b-111">Network interfaces are defined by the Internet Engineering Task Force (IETF) in [RFC 2863](https://tools.ietf.org/html/rfc2863) and are not meant to be defined by Windows.</span></span> <span data-ttu-id="9bf1b-112">Ausführliche Fragen zur Bedeutung von Netzwerkschnittstellen bezeichgern, wie z. b. **ifindex**, finden Sie in den IETF-Definitionen.</span><span class="sxs-lookup"><span data-stu-id="9bf1b-112">For detailed questions about the meaning of network interface identifiers such as **ifIndex**, see the IETF's definitions of them.</span></span> <span data-ttu-id="9bf1b-113">Im weiteren Verlauf dieses Themas werden die Windows-spezifischen Implementierungsdetails erläutert.</span><span class="sxs-lookup"><span data-stu-id="9bf1b-113">The rest of this topic discusses Windows-specific implementation details.</span></span>
