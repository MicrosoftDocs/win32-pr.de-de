# <a name="windows-color-system"></a>Windows Color System (WCS)

## <a name="purpose"></a>Zweck

Farbverwaltungsschemas werden in Microsoft Windows Betriebssystemen implementiert, um das Rendering von Farbinhalten in allen Formen der digitalen Kommunikation zu verbessern.

Das Farbverwaltungsschema, das ab Windows 2000, Windows XP und Windows Server 2003 verwendet wird, wird als Image Color Management (ICM) 2.0 bezeichnet. Das Farbverwaltungsschema, das ab Windows Vista verwendet wird, wird Windows Color System (WCS) 1.0 genannt. Das WCS 1.0-Farbverwaltungsschema ist eine Obermenge von ICM 2.0-APIs und -Funktionen.

## <a name="where-applicable"></a>Anwendungsbereich

ICM können in allen Anwendungen unter Windows Betriebssystemen 2000 und höher verwendet werden. WCS kann in allen Anwendungen unter Windows Vista und späteren Betriebssystemen verwendet werden.

## <a name="developer-audience"></a>Entwicklergruppe

Die WCS-API ist für die Verwendung durch C/C++-Programmierer konzipiert. Kenntnisse der Windows grafischen Benutzeroberfläche, der nachrichtengesteuerten Architektur und kenntnisse der Farbverwaltungskonzepte sind erforderlich.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Anwendungen, die die ICM-API verwenden, erfordern Windows 2000 oder höher. Anwendungen, die die WCS-API verwenden, erfordern Windows Vista oder höher. Spezifische Laufzeitinformationen zu einer Funktion finden Sie im Abschnitt Anforderungen der Referenzseite für diese Funktion.
