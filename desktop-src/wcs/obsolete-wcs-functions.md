# <a name="obsolete-wcs-functions"></a>Veraltete WCS-Funktionen

In diesem Abschnitt werden Funktionen beschrieben, die in WCS 1.0 veraltet sind. Sie werden derzeit aus Gründen der Abwärtskompatibilität beibehalten. Sie werden jedoch in späteren Releases von WCS aus der API entfernt.



|     Funktion                               |      Beschreibung                             |
|--------------------------------------------|----------------------------------------------|
| [**UpdateICMRegKey**](/windows/desktop/api/Wingdi/nf-wingdi-updateicmregkeya) | *Veraltet:* verwaltet Farbprofile und CMMs. |
