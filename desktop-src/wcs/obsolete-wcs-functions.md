# <a name="obsolete-wcs-functions"></a>Veraltete WCS-Funktionen

In diesem Abschnitt werden Funktionen beschrieben, die in WCS 1,0 veraltet sind. Sie werden zurzeit aus Gründen der Abwärtskompatibilität beibehalten. Sie werden jedoch in späteren Versionen von WCS aus der API entfernt.



|                                            |                                              |
|--------------------------------------------|----------------------------------------------|
| [**Updateicmregkey**](/windows/desktop/api/Wingdi/nf-wingdi-updateicmregkeya) | *Veraltet*: dient zum Verwalten von Farbprofilen und CMMS. |
