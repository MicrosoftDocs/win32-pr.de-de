# <a name="additive-primary-colors"></a>Additive Primärfarben

Die additive Methode der Farbmischung basiert auf der Annahme, dass Sie mit Schwarz beginnen. Das heißt, wenn keine anderen Farben vorhanden sind (das Bild ist schwarz), und Sie Rot hinzufügen, wird das Bild rot angezeigt. Wenn Sie dann Blau hinzufügen, wird das Bild magenta angezeigt.

Die drei [additiven Primärfarben](p.md) sind Rot, Grün und Blau. In der folgenden Abbildung wird die Additive Primärfarbmischung veranschaulicht.
