# <a name="a"></a>Ein

## <a name="additive-primary-colors"></a>Additive Primärfarben

Farben, die schwarz hinzugefügt werden können, um eine neue Farbe zu mischen. Die additiven Primärfarben sind Rot, Grün und Blau.
