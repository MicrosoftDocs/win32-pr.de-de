# <a name="d"></a>D

## <a name="dac"></a>DAC

Digital-zu-Analog-Konverter.

## <a name="device-link-profile"></a>Gerätelinkprofil

Eine Datei, die eine Verkettung von Farbkonvertierungen enthält, die häufig verwendet werden.

## <a name="device-profile"></a>Geräteprofil

Eine Datei, die Informationen zur Beschreibung der Farbfunktionen eines Geräts enthält. MIT DEVICE-Geräteprofilen werden Farben aus der Gamut eines Geräts in den PCS konvertiert. Sie enthält auch die Informationen, die für die Konvertierung von Farben vom PCS in die Gamut des Geräts erforderlich sind. WCS XML-Geräteprofile stellen die Messdaten und algorithmischen Verweise für die Konvertierung zwischen dem nativen Gerätebereich und CIEXYZ bereit.

## <a name="dmp"></a>Dmp

Das Profilformat für Farbgerätemodelle ist ein XML-basiertes Dateiformat, das Messungen enthält, die erforderlich sind, um die Beziehung zwischen farblichen Werten des nativen Gerätefarbraums und CIEXYZ-Werten zu beschreiben.
