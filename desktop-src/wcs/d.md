# <a name="d"></a>D

## <a name="dac"></a>DAC

Digital zu analogen Konverter.

## <a name="device-link-profile"></a>Geräte Verknüpfungs Profil

Eine Datei, die eine Verkettung von Farb Konvertierungen enthält, die häufig verwendet werden.

## <a name="device-profile"></a>Geräte Profil

Eine Datei, die Informationen darüber enthält, wie die Farb Überschriften eines Geräts gekennzeichnet werden. Die Geräteprofile von ICC konvertieren Farben von einem Gerät in die PCs. Sie enthält außerdem die Informationen, die für die Konvertierung von Farben von den PCs in den Gerätebestand benötigt werden. WCS-XML-Geräteprofile stellen die Messdaten und algorithmische Verweise für die Konvertierung zwischen dem systemeigenen Gerätebereich und CIEXYZ bereit.

## <a name="dmp"></a>DMP

Das Farb Gerätemodell-Profil Format ist ein XML-basiertes Dateiformat, das Messungen enthält, die zum Beschreiben der Beziehung zwischen systemeigenen Farbraum Werten für Geräte und CIEXYZ-Werten erforderlich sind.
