# <a name="t"></a>T

## <a name="tint"></a>Tönungs

Ein Farbton einer Farbe wird erstellt, indem ein Farbton mit weiß gemischt wird.

## <a name="tone"></a>Tone

Ein farbiger Farbton wird durch die Mischung des Farbton mit grau erreicht.
