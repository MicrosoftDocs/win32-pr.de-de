# <a name="p"></a>P

## <a name="primary-colors"></a>Primärfarben

Die drei wesentlichen Farben in einem Farbraum, die gemischt werden können, um alle anderen Farben zu erzeugen.

## <a name="profile-connection-space-pcs"></a>Profilverbindungsbereich (PCS)

Ein geräteunabhängiger Farbraum, der für Farbkonvertierungen und Farbübereinstimmungen verwendet wird.
