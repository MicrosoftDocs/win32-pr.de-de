# <a name="p"></a>P

## <a name="primary-colors"></a>Primärfarben

Die drei wesentlichen Farben in einem Farb Raum, die gemischt werden können, um alle anderen Farben zu entwickeln.

## <a name="profile-connection-space-pcs"></a>Profil Verbindungsbereich (PCs)

Ein Geräte unabhängiger Farbraum, der für Farb Konvertierungen und Farb Übereinstimmungen verwendet wird.
