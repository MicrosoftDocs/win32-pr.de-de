# <a name="subtractive-primary-colors"></a>Subtraktive Primärfarben

Die subtraktive Methode der Farbmischung basiert auf der Annahme, dass Sie mit weiß beginnen. Alle Farben sind in gleicher Menge vorhanden. Wenn Sie Zyan und gelb von White subtrahieren, ist das resultierende Bild Magenta. In einem grünen Bild führt das Subtrahieren von Zyan dazu, dass das Bild in Gelb geändert wird.

Die subtraktiven [Primärfarben](p.md) sind Cyan, gelbe und Magenta. In der folgenden Abbildung wird die subtraktive primäre Farbmischung veranschaulicht.
