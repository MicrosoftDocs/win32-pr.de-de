# <a name="subtractive-primary-colors"></a>Subtraktive Primärfarben

Die subtrahtive Methode der Farbmischung basiert auf der Annahme, dass Sie mit Weiß beginnen. Alle Farben sind in gleichen Mengen vorhanden. Wenn Sie Cyan und Gelb von Weiß subtrahieren, ist das resultierende Bild Magenta. In einem grünen Bild führt das Subtrahieren von Cyan dazu, dass das Bild in Gelb geändert wird.

Die subtrahtiven [Primärfarben](p.md) sind "cyan", "yellow" und "magenta". In der folgenden Abbildung wird das subtrahtive Mischen primärer Farben veranschaulicht.
