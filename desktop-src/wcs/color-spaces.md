# <a name="color-spaces"></a>Farbräume

Das menschliche Auge ist häufig in der Lage, viel mehr Farben zu erkennen, als digitale Geräte reproduzieren können. Wenn Sie z. b. eine leere, weiße Seite des Papiers betrachten, werden Sie wahrscheinlich mindestens 100 verschiedene weiß-Schattierungen erkennen. Eine weiße Wand kann problemlos 1500-Schattierungen weiß.

Qualitativ hochwertige digitale Kameras, Scanner und andere Abbild Erfassungsgeräte können auch Hunderttausende oder sogar Millionen von Farben erkennen. Da so viele erkennbare Farben vorhanden sind, haben Abbild Fachleute Modelle zum Angeben von Farben erfunden. Diese Modelle werden als *Farbbereiche* bezeichnet.

Der Grund dafür, dass diese Modelle als Farbbereiche bezeichnet werden, besteht darin, dass die meisten von Ihnen in einem 2D-, 3D-oder 4-d-Koordinatensystem ähnlich einem kartesischen Koordinatensystem zugeordnet werden können. Daher können Farben als Zusammensetzung aus Koordinaten in einem 2D-, 3D-oder 4-d-Raum bezeichnet werden. Die Farbkomponenten in einem Farbraum werden auch als *Farbkanäle* bezeichnet.

Einige Farbbereiche sollen unabhängig von Geräten sein, die zum Entwickeln von Farbbildern verwendet werden. Einige sind sehr Geräte abhängig. In den folgenden Abschnitten werden sowohl Geräte abhängige als auch geräteunabhängige Farbbereiche behandelt:

-   [HSV-Farbräume](hsv-color-spaces.md)
-   [HLS-Farbräume](hls-color-spaces.md)
-   [CMY- und CMYK-Farbraum](cmy-and-cmyk-color-spaces.md)
-   [Geräteabhängige Farbräume](device-dependent-color-spaces.md)
-   [Geräteunabhängige Farbräume](device-independent-color-spaces.md)
