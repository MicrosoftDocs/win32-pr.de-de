# <a name="describing-color"></a>Beschreiben von Farbe

Die Farbe kann in Form von Komponenten beschrieben werden. Die Komponenten, die am häufigsten von Abbild Erstellungs Experten verwendet werden, sind *Hue*, *Chroma* und *Helligkeit*. Ein Farbton ist das, was normalerweise als Farbe angesehen wird. Der Farbton einer Farbe korrigiert seinen Platz im sichtbaren Lichtspektrum. Der Chroma einer Farbe ist, wie "Pure" oder "Strong" eine Farbe ist. Neutral grau bedeutet, dass keine Sättigung vorliegt. Die Helligkeit einer Farbe bezieht sich auf die Intensität des Lichts, das von einem Bild reflektiert oder übertragen wird.

Die Begriffe " *Tönungs*", " *Tone*" und " *Farbton* " werden auch in der Farb Imaging-Literatur gut verwendet. Ein Farbton einer Farbe wird durch die Mischung aus Farbton und weiß abgerufen. Ein farbiger Farbton wird erstellt, indem ein Farbton mit grau gemischt wird. Ein Farbton einer Farbe wird durch Hinzufügen von schwarz zum Farbton erzielt.
