# <a name="describing-color"></a>Beschreiben von Farbe

Die Farbe kann in Form von Komponenten beschrieben werden. Die Komponenten, die am häufigsten von Bildverarbeitungsexperten verwendet werden, sind *Farbton,* *Färbung* und *Helligkeit.* Ein Farbton wird normalerweise als Farbe bezeichnet. Der Farbton einer Farbe behebt seine Position im sichtbaren Spektrum des Lichts. Der Farbton ist, wie "rein" oder "stark" eine Farbe ist. Neutrales Grau wird als Nullsättigung bezeichnet. Die Helligkeit einer Farbe bezieht sich auf die Intensität des Lichts, das von einem Bild reflektiert oder übertragen wird.

Die Begriffe *"Tönung",* *"Ton"* und *"Schattierung"* werden auch in der Farbbildbearbeitung gut verwendet. Eine Tönung einer Farbe wird durch Mischen des Farbtons mit Weiß erzielt. Ein Farbton wird erstellt, indem ein Farbton mit Grau kombiniert wird. Ein Farbton wird durch Hinzufügen von Schwarz zu seinem Farbton erstellt.
