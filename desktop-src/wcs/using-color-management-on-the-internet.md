# <a name="using-color-management-on-the-internet"></a>Verwenden von Farbverwaltung im Internet

Farbbilder und Grafiken stellen eine zunehmend wichtige Möglichkeit dar, Informationen für Benutzer im Internet zu kommunizieren. Farb Konvertierungs Informationen können in einige Dateiformate eingebettet werden. Die derzeit verwendeten Einbettungs Techniken sind nicht kompakt. Überlegungen zur Internet Bandbreite machen diese häufig nicht praktikabel. Zu den Internet Überlegungen zählen die folgenden Themen:

-   [sRGB: ein Standard Farbraum](srgb--a-standard-color-space.md)
-   [WCS 1,0-Standardwerte für Eingabe Farbraum und Ausgabeprofil](srgb--a-standard-color-space.md)
-   [sRGB und eingebettete Profile](srgb--a-standard-color-space.md)
