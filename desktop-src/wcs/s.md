# <a name="s"></a>E

## <a name="saturation-chroma"></a>Sättigung (Chroma)

Im Allgemeinen ist Chroma die graue Menge, die in einen Farbton gemischt ist. NULL Chroma erzeugt einen grauen Farbton. Full Chroma erzeugt einen reinen Farbton.

## <a name="shade"></a>tig

Ein Farbton einer Farbe wird durch Hinzufügen von schwarz zum Farbton erzielt.

## <a name="soft-proofing"></a>weiche Absicherung

Anzeigen oder durch sehen eines Bilds in einer Videoanzeige.

## <a name="srgb"></a>sRGB

Ein internationaler Standard-RGB-Farbraum, der von Microsoft vorgeschlagen wird und Hewlett-Packard der Geräte unabhängig ist. Dieser Standard wurde als formaler internationaler Standard durch die Multimedia-Systeme und die "equipmentcolor"-Messung und managementpart 2-1: Color managementdefault RGB-Farbraum genehmigt. Er ist direkt von der IEC unter verfügbar https://www.iec.ch/ .

## <a name="subtractive-primary-colors"></a>Subtraktive Primärfarben

Farben, die von weiß subtrahiert werden können, um eine neue Farbe zu mischen. Die subtraktiven Primärfarben sind Cyan, gelbe und Magenta.
