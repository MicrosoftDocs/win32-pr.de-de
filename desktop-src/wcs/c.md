# <a name="c"></a>C

## <a name="camp"></a>Camping

Das Modell für die Farbdarstellung des Modell Profils ist ein XML-basiertes Dateiformat, das Anzeige Bedingungen enthält, die erforderlich sind, um die Beziehung zwischen den DMP-CIEXYZ-Werten und den ciejab-Werten zu beschreiben.

## <a name="cmyk"></a>CMYK

Der Farbraum Cyan, Magenta, gelb und schwarz. Sie wird häufig auf Druckern implementiert.

## <a name="colorant"></a>Colorant

Etwas, vor allem ein Pigment oder eine Handschrift, das den Farbton von etwas anderem Farben oder ändert.

## <a name="color-channel"></a>Farbkanal

Eine Komponente eines Farbraum. Beispielsweise verfügt ein RGB-Farbraum über rote, grüne und blaue Farbkanäle.

## <a name="color-conversion"></a>Farbkonvertierung

Der Vorgang zum Umrechnen von Farben von einem Farb Raum in einen anderen.

## <a name="color-management-module-cmm"></a>Farb Verwaltungsmodul (CMM)

Ein Codemodul, das Geräteprofile zum Durchführen der Farbkonvertierung und der Farbzuordnung verwendet.

## <a name="color-mapping"></a>Farbzuordnung

Siehe Farb Übereinstimmung.

## <a name="color-matching"></a>Farb Übereinstimmung

Übereinstimmung mit einer konvertierten Farbe mit ihrer visuellen Farbe im Ziel Farbraum.

## <a name="color-model"></a>Farbmodell

Siehe Farbraum.

## <a name="color-space"></a>Farbraum

Eine Zuordnung von Farbkomponenten zu einem geometrischen Koordinatensystem in drei Dimensionen.
