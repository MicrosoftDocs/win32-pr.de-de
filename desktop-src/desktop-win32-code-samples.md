Siehe auch [Codebeispiele.](https://developer.microsoft.com/windows/samples/)

## <a name="find-a-sample-for-the-api-youre-interested-in"></a>Hier finden Sie ein Beispiel für die API, an der Sie interessiert sind.

Sie können Microsoft Visual Studio verwenden, um den Quellcode eines gesamten Repositorys zu durchsuchen, um festzustellen, ob die Verwendung einer bestimmten Windows-API veranschaulicht wird.

* Klonen Sie eines der in den folgenden Abschnitten aufgeführten Repositorys (oder laden Sie die ZIP-Datei herunter) in Ihr lokales Dateisystem.
* Suchen **Sie** dann in Dateien in Visual Studio. Legen **Sie Suchen in** auf den Ordner fest, in den Sie geklont oder heruntergeladen haben, und suchen Sie nach dem Namen der API, an der Sie interessiert sind. Das Überprüfen der **Übereinstimmungsfall** liefert die besten Ergebnisse.
* Achten Sie darauf, das Kontrollkästchen **Ganzes Wort** abgleichen zu überprüfen, wenn die API in verschiedenen Formen aufgerufen werden &mdash; kann, z. B. **CreateMutex,** **CreateMutexA** und **CreateMutexW.** Suchen Sie in diesem Fall nach CreateMutex with **Match whole word** unchecked ( **CreateMutex** with Match whole word unchecked).

> [!NOTE]
> Sie können [Visual Studio](https://visualstudio.microsoft.com/downloads/) ohne Kosten installieren. Die Community-Edition ist für Schüler/Studenten, Open-Source-Mitwirkende und Einzelpersonen kostenlos. Natürlich können Sie dasselbe mit jedem Beispiel-Repository tun.
