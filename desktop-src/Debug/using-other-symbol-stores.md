# <a name="using-other-symbol-stores"></a>Verwenden anderer Symbol Speicher

Es ist möglich, ein eigenes Programm zur Symbol Speicher Erstellung zu schreiben, anstatt SymStore zu verwenden.

Da SymStore-Transaktionen alle im CSV-Format Textdateien protokolliert werden, können Sie alle vorhandenen SymStore-Protokolldateien für die Verwendung in Ihrem eigenen Datenbankprogramm nutzen.

Wenn Sie beabsichtigen, das SymSrv-Programm zu verwenden, das mit Debuggingtools für Windows bereitgestellt wird, empfiehlt es sich, auch SymStore zu verwenden. Updates für diese beiden Programme werden immer zusammen freigegeben, sodass Ihre Versionen immer abgeglichen werden.
