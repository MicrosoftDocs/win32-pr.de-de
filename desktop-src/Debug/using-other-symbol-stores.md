# <a name="using-other-symbol-stores"></a>Verwenden anderer Symbolspeicher

Es ist möglich, ein eigenes Symbolspeicher-Erstellungsprogramm zu schreiben, anstatt SymStore zu verwenden.

Da SymStore-Transaktionen alle in Textdateien im CSV-Format protokolliert werden, können Sie alle vorhandenen SymStore-Protokolldateien für die Verwendung in Ihrem eigenen Datenbankprogramm nutzen.

Wenn Sie das SymSrv-Programm verwenden möchten, das mit Debugtools für Windows bereitgestellt wird, empfiehlt es sich, auch SymStore zu verwenden. Updates für diese beiden Programme werden immer zusammen veröffentlicht, und daher werden ihre Versionen immer übereinstimmen.
