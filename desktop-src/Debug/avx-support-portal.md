# <a name="intel-avx"></a>Intel AVX

## <a name="purpose"></a>Zweck

Bei Intel Advanced Vector Extensions (AVX) handelt es sich um eine 256-Bit-SIMD-Gleit Komma Vektor Erweiterung der Intel-Architektur. Sie enthält Erweiterungen für Anweisungs-und Register Sätze.

Microsoft hat einige API-Erweiterungen entwickelt, wie z. b. xstate-Funktionen, mit denen Anwendungen auf Informationen und den Status erweiterter Prozessor Funktionen zugreifen und diese bearbeiten können (einschließlich Intel AVX).

## <a name="developer-audience"></a>Entwicklergruppe

Intel AVX ist für Anwendungen konzipiert, die stark Gleit Komma-Rechen intensiv sind und vektorisiert werden können. Beispiele für Anwendungen sind Audioverarbeitung und Audiocodecs, Bild-und Videobearbeitungsanwendungen, Analyse-und Modellierungssoftware für Finanzdienstleistungen sowie Fertigungs-und Engineering-Software.
