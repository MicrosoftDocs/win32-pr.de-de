# <a name="about-basic-debugging"></a>Informationen zum grundlegenden Debugging

Die [Debuggingfunktionen](debugging-functions.md) können verwendet werden, um einen einfachen, ereignisgesteuerten Debugger zu erstellen. *Ereignisgesteuert* bedeutet, dass der Debugger jedes Mal benachrichtigt wird, wenn bestimmte Ereignisse im debuggten Prozess auftreten. Mithilfe der Benachrichtigung kann der Debugger geeignete Aktionen als Reaktion auf die Ereignisse durchführen.

Eine Übersicht über die debuggingbegriffe finden Sie unter [Debugging-Terminologie](debugging-terminology.md).

Die [Debugunterstützung aus Prozess-, Thread-und Ausnahme Funktionen](debug-support-from-process-thread-and-exception-functions.md) beschreibt die debuggingspezifischen Funktionen bestimmter Prozess-, Thread-und Ausnahme Behandlungs Funktionen.

Beim [Erstellen eines einfachen Debuggers](creating-a-basic-debugger.md) wird die Verwendung der grundlegenden Debugfunktionen zum Erstellen eines einfachen Debuggers beschrieben. Mit diesen Funktionen kann eine Anwendung auf Debugereignisse warten, Haltepunkt Ausnahmen auslösen, die Ausführungs Steuerung an den Debugger übertragen usw.

[Debugereignisse](debugging-events.md) beschreiben den Debug-Ereignis Mechanismus. Das Debuggen von Ereignissen bewirkt, dass das System den Debugger benachrichtigt.
