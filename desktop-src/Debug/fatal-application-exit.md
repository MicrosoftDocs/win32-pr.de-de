# <a name="fatal-application-exit"></a>Schwerwiegender Anwendungs Exit

Die Funktion **fatalappexit** zeigt ein Meldungs Feld an und beendet die Anwendung, wenn der Benutzer das Meldungs Feld schließt. Diese Funktion sollte nur als letzte Möglichkeit verwendet werden, da der Arbeitsspeicher oder die Dateien, die sich im Besitz der Anwendung befinden, möglicherweise nicht freigegeben werden.
