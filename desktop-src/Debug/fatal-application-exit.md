# <a name="fatal-application-exit"></a>Schwerwiegendes Beenden der Anwendung

Die **FatalAppExit-Funktion** zeigt ein Meldungsfeld an und beendet die Anwendung, wenn der Benutzer das Meldungsfeld schließt. Diese Funktion sollte nur als letztes Mittel verwendet werden, da sie möglicherweise nicht den Arbeitsspeicher oder die Dateien frei gibt, die sich im Besitz der Anwendung befinden.
