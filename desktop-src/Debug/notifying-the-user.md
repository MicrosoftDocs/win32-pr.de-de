# <a name="notifying-the-user"></a>Benachrichtigen des Benutzers

Um den Benutzer zu benachrichtigen, dass ein Fehler aufgetreten ist, führen viele Anwendungen einfach einen Sound aus, indem Sie die [**MessageBeep**](/windows/desktop/api/WinUser/nf-winuser-messagebeep) -Funktion verwenden oder das Fenster mithilfe der Funktion "Flash [**Window**](/windows/desktop/api/Winuser/nf-winuser-flashwindow) " oder "Flash [**windowex**](/windows/desktop/api/Winuser/nf-winuser-flashwindowex) " blinken. Eine Anwendung kann diese Funktionen auch verwenden, um auf einen Fehler aufmerksam zu machen und dann ein Meldungs Feld oder eine Fehlermeldung mit Details zum Fehler anzuzeigen.
