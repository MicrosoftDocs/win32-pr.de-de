# <a name="creating-a-basic-debugger"></a>Erstellen eines einfachen Debuggers

Beim [Debuggen eines laufenden Prozesses](debugging-a-running-process.md) wird das Anfügen des Debuggers an einen laufenden Prozess beschrieben Beim [Schreiben der Hauptschleife des Debuggers](writing-the-debugger-s-main-loop.md) werden die Funktionen beschrieben, die in der Haupt Ausführungs Schleife des Debuggers verwendet werden.

Obwohl die meisten Debuggingfunktionen verwendet werden, um einen Debugger zu erstellen, sind mehrere Funktionen für die Verwendung im debuggten Prozess konzipiert. Weitere Informationen finden Sie unter [kommunizieren mit dem Debugger](communicating-with-the-debugger.md).
