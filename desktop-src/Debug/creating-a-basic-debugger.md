# <a name="creating-a-basic-debugger"></a>Erstellen eines einfachen Debuggers

[Debuggen eines ausgeführten Prozesses](debugging-a-running-process.md) beschreibt das Anfügen des Debuggers an einen ausgeführten Prozess. [Beim Schreiben der Hauptschleife des Debuggers](writing-the-debugger-s-main-loop.md) werden die Funktionen beschrieben, die in der Hauptschleife der Ausführung des Debuggers verwendet werden.

Obwohl die meisten Debugfunktionen zum Erstellen eines Debuggers verwendet werden, sind mehrere Funktionen für die Verwendung in dem Prozess konzipiert, der debuggt wird. Weitere Informationen finden Sie unter [Kommunikation mit dem Debugger](communicating-with-the-debugger.md).
