# <a name="debug-help-library"></a>Hilfebibliothek zum Debuggen

In dieser Übersicht wird der Von der Debughilfebibliothek DbgHelp bereitgestellte Funktionssatz beschrieben. Sie enthält eine Reihe von Debugunterstützungsroutinen, mit denen Sie mit ausführbaren Bildern im portierbaren ausführbaren Format (Portable Executable, PE) arbeiten können.

Die DbgHelp-Dokumentation sieht wie folgt aus:

- [Informationen zu DbgHelp](about-dbghelp.md)
- [Verwenden von DbgHelp](using-dbghelp.md)
- [DbgHelp-Referenz](dbghelp-reference.md)

Um die neueste Version von DbgHelp.dll zu erhalten, wechseln Sie zu [https://developer.microsoft.com/windows/downloads/windows-10-sdk](https://developer.microsoft.com/windows/downloads/windows-10-sdk) Debugtools für Windows, und laden Sie sie herunter.

Um eine Beschreibung des PE-Formats zu erhalten, laden Sie die Spezifikation von folgendem Speicherort herunter: [https://www.microsoft.com/whdc/system/platform/firmware/PECOFF.mspx](https://www.microsoft.com/whdc/system/platform/firmware/PECOFF.mspx) .

Informationen zum Durchsuchen von Informationen in PDB-Dateien finden Sie im [Debug Interface Access SDK](/visualstudio/debugger/debug-interface-access/debug-interface-access-sdk?view=vs-2015).
