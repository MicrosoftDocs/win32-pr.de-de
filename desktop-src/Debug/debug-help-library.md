# <a name="debug-help-library"></a>Hilfe Bibliothek Debuggen

In dieser Übersicht wird der von der Debug-Hilfe-Bibliothek (dbghelp) bereitgestellte Funktions Satz beschrieben. Sie enthält eine Reihe von Debug-Unterstützungs Routinen, mit denen Sie mit ausführbaren Images im PE-Format (portable ausführbare Datei) arbeiten können.

Die dbghelp-Dokumentation lautet wie folgt:

- [Informationen zu dbghelp](about-dbghelp.md)
- [Verwenden von dbghelp](using-dbghelp.md)
- [Dbghelp-Referenz](dbghelp-reference.md)

Um die neueste Version von DbgHelp.dll zu erhalten, navigieren [https://developer.microsoft.com/windows/downloads/windows-10-sdk](https://developer.microsoft.com/windows/downloads/windows-10-sdk) Sie zu, und laden Sie die Debugtools für Windows herunter.

Eine Beschreibung des PE-Formats erhalten Sie, wenn Sie die Spezifikation von folgendem Speicherort herunterladen: [https://www.microsoft.com/whdc/system/platform/firmware/PECOFF.mspx](https://www.microsoft.com/whdc/system/platform/firmware/PECOFF.mspx) .

Informationen zum Durchsuchen von Informationen in PDB-Dateien finden Sie in der [Debug Interface Access SDK](/visualstudio/debugger/debug-interface-access/debug-interface-access-sdk?view=vs-2015).
