# <a name="avx-functions"></a>AVX-Funktionen

Im folgenden sind die Intel AVX-Funktionen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                   | BESCHREIBUNG                                                                                                                    |
|-------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [**Copycontext**](/windows/desktop/api/WinBase/nf-winbase-copycontext)<br/>                           | Kopiert eine Quell Kontext Struktur (einschließlich eines beliebigen xstate) in eine initialisierte Ziel Kontext Struktur.<br/>         |
| [**Getenabledxstatefeatures**](/windows/desktop/api/WinBase/nf-winbase-getenabledxstatefeatures)<br/> | Ruft eine Maske von aktivierten xstate-Funktionen auf x86-oder x64-Prozessoren ab.<br/>                                                    |
| [**Getxstatefeaturesmask**](/windows/desktop/api/WinBase/nf-winbase-getxstatefeaturesmask)<br/>       | Gibt die Maske von xstate-Funktionen zurück, die innerhalb einer [**Kontext**](/windows/desktop/api/WinNT/ns-winnt-wow64_context) Struktur festgelegt sind.<br/>                        |
| [**Initializecontext**](/windows/desktop/api/WinBase/nf-winbase-initializecontext)<br/>               | Initialisiert eine [**Kontext**](/windows/desktop/api/WinNT/ns-winnt-arm64_nt_context) Struktur in einem Puffer mit der erforderlichen Größe und Ausrichtung.<br/>       |
| [**Loerexstatefeature**](/windows/desktop/api/WinBase/nf-winbase-locatexstatefeature)<br/>           | Ruft einen Zeiger auf den Prozessor Zustand für eine xstate-Funktion innerhalb einer [**Kontext**](/windows/desktop/api/WinNT/ns-winnt-arm64_nt_context) Struktur ab.<br/> |
| [**Setxstatefeaturesmask**](/windows/desktop/api/WinBase/nf-winbase-setxstatefeaturesmask)<br/>       | Legt die Maske der in einer [**Kontext**](/windows/desktop/api/WinNT/ns-winnt-arm64_nt_context) Struktur festgelegten xstate-Funktionen fest.<br/>                             |
