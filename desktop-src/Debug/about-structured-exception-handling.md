# <a name="about-structured-exception-handling"></a>Informationen zur strukturierten Ausnahmebehandlung

Die strukturierten Mechanismen zur Ausnahmebehandlung und Beendigungsbehandlung sind integrale Bestandteile des Systems. sie ermöglichen es dem System, stabil zu sein. Sie können diese Mechanismen verwenden, um konsistent stabile und zuverlässige Anwendungen zu erstellen.

Die strukturierte Ausnahmebehandlung wird in erster Linie über die Compilerunterstützung verfügbar gemacht. Beispielsweise unterstützt der Microsoft C/C++-Optimierungscompiler das **\_ \_ schlüsselwort try,** das einen geschützten Codetext identifiziert, das **\_ \_ Schlüsselwort except,** das einen Ausnahmehandler identifiziert, und das **\_ \_ finally-Schlüsselwort,** das einen Beendigungshandler identifiziert. In dieser Übersicht werden zwar spezifische Beispiele für den Microsoft C/C++-Compiler verwendet, aber auch andere Compiler bieten diese Unterstützung.
