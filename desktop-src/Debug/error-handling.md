# <a name="error-handling-error-handling"></a>Fehlerbehandlung (Fehlerbehandlung)

Gut geschriebene Anwendungen enthalten Fehler Behandlungs Code, mit dem Sie von unerwarteten Fehlern ordnungsgemäß wieder hergestellt werden können. Wenn ein Fehler auftritt, muss die Anwendung möglicherweise einen Benutzereingriff anfordern, oder Sie kann selbst wieder hergestellt werden. In Extremfällen kann die Anwendung den Benutzer abmelden oder das System Herunterfahren.

-   [Informationen zur Fehlerbehandlung](about-error-handling.md)
-   [Verwenden der Fehlerbehandlung](using-error-handling.md)
-   [Fehler Behandlungs Referenz](error-handling-reference.md)

Weitere Informationen zur Ausnahmebehandlung finden Sie unter [strukturierte Ausnahmebehandlung](structured-exception-handling.md).
