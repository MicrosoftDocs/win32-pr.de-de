# <a name="error-handling-error-handling"></a>Fehlerbehandlung (Fehlerbehandlung)

Gut geschriebene Anwendungen enthalten Code zur Fehlerbehandlung, mit dem sie nach unerwarteten Fehlern ordnungsgemäß wiederhergestellt werden können. Wenn ein Fehler auftritt, muss die Anwendung möglicherweise einen Benutzereingriff anfordern, oder sie kann eigenständig wiederhergestellt werden. In Extremfällen kann die Anwendung den Benutzer abmelden oder das System herunterfahren.

-   [Informationen zur Fehlerbehandlung](about-error-handling.md)
-   [Verwenden der Fehlerbehandlung](using-error-handling.md)
-   [Fehlerbehandlungsreferenz](error-handling-reference.md)

Informationen zur Ausnahmebehandlung finden Sie unter [Strukturierte Ausnahmebehandlung.](structured-exception-handling.md)
