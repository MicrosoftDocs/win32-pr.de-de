# <a name="symbol-handling"></a>Symbol Behandlung

Mit den Funktionen des Symbol Handlers können Anwendungen einfach und portabel auf die symbolischen Debuginformationen eines ausführbaren Bilds zugreifen. Diese Funktionen werden ausschließlich verwendet, um den Zugriff auf symbolische Informationen sicherzustellen. Dies ist erforderlich, da diese Funktionen die Anwendung vom Symbol Format isolieren.
