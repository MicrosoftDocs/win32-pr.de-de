# <a name="symbol-handling"></a>Symbolbehandlung

Die Symbolhandlerfunktionen ermöglichen Anwendungen einfachen und portierbaren Zugriff auf die symbolischen Debuginformationen eines ausführbaren Images. Diese Funktionen werden ausschließlich verwendet, um den Zugriff auf symbolische Informationen sicherzustellen. Dies ist erforderlich, da diese Funktionen die Anwendung vom Symbolformat isolieren.
