# <a name="compression-api"></a>Komprimierungs-API

## <a name="purpose"></a>Zweck

Die Komprimierungs-API macht die Komprimierungs Algorithmen Windows mszip, Xpress, Xpress \_ Huff und lzms verfügbar. Dadurch können Entwickler von Windows-Anwendungen Versionen und Dienste verwalten und die verfügbar gemachten Komprimierungs Algorithmen erweitern.

## <a name="developer-audience"></a>Entwicklergruppe

Die Komprimierungs-API ist für die Verwendung durch professionelle C/C++-Entwickler von Windows-Anwendungen konzipiert. Die Komprimierungs-API macht die von Windows verwendeten verlustfreien Komprimierungs Algorithmen für eine öffentliche Schnittstelle und eine API im Benutzermodus verfügbar. Die Komprimierungs-API ist die empfohlene Methode für Windows-Entwickler, die Komprimierungs Algorithmen zu steuern. Diese Funktion ist 64-Bit auf 64-Bit-Windows und 32-Bit auf 32-Bit-Windows.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Die Komprimierungs-API ist ab Windows 8 oder Windows Server 2012 verfügbar.
