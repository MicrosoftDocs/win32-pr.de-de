# <a name="effect-compiler-tool"></a>Effect-Compiler Tool

FXC (fxc.exe) ist ein Offline Tool zum Kompilieren von [HLSL](/windows/desktop/direct3dhlsl/dx-graphics-hlsl) -Shadern für alle Versionen von Direct3D. Das Tool befindet sich unter:

(*SDK*-Stamm) \\ Hilfsprogramme \\ bin \\ x86\\
