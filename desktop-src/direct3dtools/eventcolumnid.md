## <a name="constants"></a>Konstanten

<span id="EventColumnID_EVENTTYPE"></span><span id="eventcolumnid_eventtype"></span><span id="EVENTCOLUMNID_EVENTTYPE"></span>**Eventcolumnid- \_ eventType**  
Eine ID, die der Ereignistyp Spalte entspricht.

<span id="EventColumnID_EID"></span><span id="eventcolumnid_eid"></span><span id="EVENTCOLUMNID_EID"></span>**Eventcolumnid- \_ Eid**  
Eine ID, die der Ereignis-ID-Spalte entspricht.

<span id="EventColumnID_PARENTEID"></span><span id="eventcolumnid_parenteid"></span><span id="EVENTCOLUMNID_PARENTEID"></span>**Eventcolumnid \_ -Parser**  
Eine ID, die der übergeordneten Ereignis-ID-Spalte entspricht.

<span id="EventColumnID_EVENTNAME"></span><span id="eventcolumnid_eventname"></span><span id="EVENTCOLUMNID_EVENTNAME"></span>**Eventcolumnid \_ EventName**  
Eine ID, die der Ereignis Namensspalte entspricht.

<span id="EventColumnID_FRAME"></span><span id="eventcolumnid_frame"></span><span id="EVENTCOLUMNID_FRAME"></span>**Eventcolumnid- \_ Frame**  
Eine ID, die der Frame-Spalte entspricht.

<span id="EventColumnID_EVENTSTRING"></span><span id="eventcolumnid_eventstring"></span><span id="EVENTCOLUMNID_EVENTSTRING"></span>**Eventcolumnid- \_ Ereignis Zeichenfolge**  
Eine ID, die der vollständigen Zeichenfolge entspricht, die für das Ereignis analysiert werden soll.

<span id="EventColumnID_THISPTR"></span><span id="eventcolumnid_thisptr"></span><span id="EVENTCOLUMNID_THISPTR"></span>**Eventcolumnid \_ thisptr**  
Eine ID, die dem Ressourcen Handle für den this-Zeiger entspricht.

<span id="EventColumnID_STARTMOMENTHANDLE"></span><span id="eventcolumnid_startmomenthandle"></span><span id="EVENTCOLUMNID_STARTMOMENTHANDLE"></span>**Eventcolumnid \_ startmomenthandle**

<span id="EventColumnID_InfoQueueMessages"></span><span id="eventcolumnid_infoqueuemessages"></span><span id="EVENTCOLUMNID_INFOQUEUEMESSAGES"></span>**Eventcolumnid \_ infoqueuemess**  
Eine ID, die jeder Nachricht entspricht, die von dieser in den SDK-Schichten generiert wurde.

<span id="EventColumnID_ThreadId"></span><span id="eventcolumnid_threadid"></span><span id="EVENTCOLUMNID_THREADID"></span>**Eventcolumnid \_ ThreadId**  
Eine ID, die der Thread-ID entspricht.
