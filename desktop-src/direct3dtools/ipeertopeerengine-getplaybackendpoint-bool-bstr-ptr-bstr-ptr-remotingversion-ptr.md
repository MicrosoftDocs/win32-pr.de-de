# <a name="span-idvspixengineipeertopeerengine_getplaybackendpoint_bool_bstr_ptr_bstr_ptr_remotingversion_ptrspanipeertopeerenginegetplaybackendpoint-method"></a><span id="vspixengine.ipeertopeerengine_getplaybackendpoint_bool_bstr_ptr_bstr_ptr_remotingversion_ptr"></span>Ietertopeerengine:: getplaybackendpoint-Methode

Ruft die Endpunkt Adresse einer Remote-Engine ab.

## <a name="syntax"></a>Syntax


```C++
HRESULT GetPlaybackEndpoint(
   BOOL              bUseAuthentication,
   BSTR *            pEndpoint,
   BSTR *            sessionKey,
   RemotingVersion * pRemoteVersion
);
```

## <a name="parameters"></a>Parameter

*busauthentifizierung*   
true, wenn die Remote-Engine die-Authentifizierung verwendet. andernfalls false.

*pdpoint*   
Bei Rückgabe eine com-Zeichenfolge, die die Endpunkt Adresse des Remote Wiedergabe Computers enthält.

*SessionKey*   
Bei Rückgabe eine com-Zeichenfolge, die den für die Verschlüsselung verwendeten Sitzungsschlüssel enthält.

*premoteversion*   
Bei der Rückgabe die Version der Remote-Engine.

## <a name="return-value"></a>Rückgabewert

Wenn diese Methode erfolgreich ausgeführt wird, gibt Sie **S \_ OK** zurück. Andernfalls wird ein **HRESULT** -Fehlercode zurückgegeben.
