# <a name="configuration-constants"></a>Konfigurations Konstanten

Dieser Abschnitt enthält die Referenz Spezifikationen für [**SystemParametersInfo**](/windows/win32/api/winuser/nf-winuser-systemparametersinfoa) -Konstanten, die sich auf [Konfigurations](configuration.md) Systemattribute beziehen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                             | BESCHREIBUNG                                                                                                                                                            |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Kontakt Visualisierung**](contact-visualization.md)<br/> | Die folgenden Konstanten werden von Anwendungen oder Benutzeroberflächen-Frameworks verwendet, um zu ermitteln, wie das UI-Feedback verarbeitet wird, wenn ein Eingabe Kontakt erkannt wird.<br/>               |
| [**Gesten Visualisierung**](gesture-visualization.md)<br/> | Die folgenden Konstanten werden von Anwendungen oder Benutzeroberflächen-Frameworks verwendet, um zu ermitteln, wie das UI-Feedback verarbeitet wird, wenn eine der aufgelisteten Gesten erkannt wird.<br/>     |
| [**Stift Visualisierung**](pen-visualization.md)<br/>         | Die folgenden Konstanten werden von Anwendungen oder Benutzeroberflächen-Frameworks verwendet, um zu ermitteln, wie das UI-Feedback verarbeitet wird, wenn eine der aufgelisteten Stift Gesten erkannt wird.<br/> |
