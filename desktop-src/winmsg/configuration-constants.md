# <a name="configuration-constants"></a>Konfigurationskonstanten

Dieser Abschnitt enthält die Referenzspezifikationen für [**SystemParametersInfo-Konstanten**](/windows/win32/api/winuser/nf-winuser-systemparametersinfoa) im Zusammenhang mit [Configuration-Systemattributen.](configuration.md)

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                             | Beschreibung                                                                                                                                                            |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Kontaktvisualisierung**](contact-visualization.md)<br/> | Die folgenden Konstanten werden von Anwendungen oder Benutzeroberflächenframeworks verwendet, um zu identifizieren, wie Benutzeroberflächenfeedback verarbeitet wird, wenn ein Eingabekontakt erkannt wird.<br/>               |
| [**Gestenvisualisierung**](gesture-visualization.md)<br/> | Die folgenden Konstanten werden von Anwendungen oder Benutzeroberflächenframeworks verwendet, um zu identifizieren, wie Benutzeroberflächenfeedback verarbeitet wird, wenn eine der aufgelisteten Gesten erkannt wird.<br/>     |
| [**Stiftvisualisierung**](pen-visualization.md)<br/>         | Die folgenden Konstanten werden von Anwendungen oder Benutzeroberflächenframeworks verwendet, um zu identifizieren, wie Benutzeroberflächenfeedback verarbeitet wird, wenn eine der aufgelisteten Stiftgesten erkannt wird.<br/> |
