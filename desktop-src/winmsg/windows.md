# <a name="windows-windows-and-messages"></a>Windows (Fenster und Meldungen)

In einer grafischen Windows-basierten Anwendung handelt es sich bei einem Fenster um einen rechteckigen Bereich des Bildschirms, in dem die Anwendung die Ausgabe anzeigt und Eingaben vom Benutzer empfängt. Daher besteht eine der ersten Aufgaben einer grafischen Windows-basierten Anwendung darin, ein Fenster zu erstellen.

Ein Fenster teilt den Bildschirm mit anderen Fenstern, einschließlich derjenigen aus anderen Anwendungen. Nur ein Fenster zu einem Zeitpunkt kann Eingaben vom Benutzer empfangen. Der Benutzer kann die Maus, die Tastatur oder ein anderes Eingabegerät verwenden, um mit diesem Fenster und der Anwendung, die es besitzt, zu interagieren.
