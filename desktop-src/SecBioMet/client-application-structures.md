# <a name="client-application-structures"></a>Clientanwendungsstrukturen

Die folgenden Strukturen werden für die Clientanwendungsentwicklung von der Windows Biometric Framework-API unterstützt.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                        | BESCHREIBUNG                                                                                                                     |
|----------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**WINBIO \_ ANTI \_ SPOOF \_ POLICY**](winbio-anti-spoof-policy.md)<br/>                   | Stellt die Antispoofingrichtlinie für einen Benutzer dar.<br/>                                                                       |
| [**WINBIO \_ \_ ASYNC-ERGEBNIS**](/windows/desktop/api/Winbio/ns-winbio-winbio_async_result)<br/>                              | Enthält die Ergebnisse eines asynchronen Vorgangs.<br/>                                                                   |
| [**WINBIO \_ BDB \_ ANSI \_ 381 \_ HEADER**](winbio-bdb-ansi-381-header.md)<br/>              | Gibt Informationen zu einer Reihe von Fingerabdruck- oder Handflächenstichproben an.<br/>                                                 |
| [**WINBIO \_ BDB \_ ANSI \_ 381 \_ RECORD**](winbio-bdb-ansi-381-record.md)<br/>              | Enthält Informationen zu einem einzelnen Fingerabdruck- oder Handflächenbeispiel eines Endbenutzers.<br/>                                     |
| [**WINBIO \_ BIR**](winbio-bir.md)<br/>                                                 | Stellt einen biometrischen Informationsdatensatz (BIR) dar.<br/>                                                                     |
| [**WINBIO \_ BIR \_ DATA**](winbio-bir-data.md)<br/>                                      | Gibt die Größe in Bytes und den Offset eines Blocks biometrischer Informationen an.<br/>                                    |
| [**WINBIO \_ \_ BIR-HEADER**](winbio-bir-header.md)<br/>                                  | Enthält den Header eines biometrischen Informationsdatensatzes (BIR).<br/>                                                         |
| [**\_WINBIO-BSP-SCHEMA \_**](winbio-bsp-schema.md)<br/>                                  | Beschreibt die Funktionen eines biometrischen Dienstanbieters.<br/>                                                          |
| [**INFORMATIONEN \_ ZUR ERWEITERTEN \_ WINBIO-ENGINE \_**](winbio-extended-engine-info.md)<br/>             | Enthält Informationen zu den Funktionen und Registrierungsanforderungen des Engine-Adapters für eine biometrische Einheit.<br/>  |
| [**WINBIO \_ – ERWEITERTER \_ \_ REGISTRIERUNGSSTATUS**](winbio-extended-enrollment-status.md)<br/> | Enthält zusätzliche Informationen zum Status einer registrierung, die gerade ausgeführt wird.<br/>                               |
| [**WINBIO \_ – ERWEITERTE \_ \_ SENSORINFORMATIONEN**](winbio-extended-sensor-info.md)<br/>             | Enthält Informationen zu den Funktionen und Registrierungsanforderungen des Sensoradapters für eine biometrische Einheit.<br/>  |
| [**WINBIO \_ EXTENDED \_ STORAGE \_ INFO**](winbio-extended-storage-info.md)<br/>           | Enthält Informationen zu den Funktionen und Registrierungsanforderungen des Speicheradapters für eine biometrische Einheit.<br/> |
| [**\_WINBIO-EREIGNIS**](winbio-event.md)<br/>                                             | Enthält Statusinformationen, die an die Rückrufroutine gesendet werden, wenn ein Ereignishinweis ausgelöst wird.<br/>                             |
| [**WINBIO \_ IDENTITY**](winbio-identity.md)<br/>                                       | Enthält einen identifizierenden Wert, der einer biometrischen Vorlage zugeordnet ist.<br/>                                                  |
| [**WINBIO \_ PRESENCE**](winbio-presence.md)<br/>                                       | Enthält Informationen über das Vorhandensein einer Person, deren Vorhandensein überwacht wird.<br/>                          |
| [**REGISTRIERTES \_ \_ WINBIO-FORMAT**](winbio-registered-format.md)<br/>                    | Gibt ein registriertes Datenformat als Besitzer-/Formatpaar an.<br/>                                                          |
| [**\_WINBIO-SPEICHERSCHEMA \_**](winbio-storage-schema.md)<br/>                          | Beschreibt die Funktionen eines biometrischen Speicheradapters.<br/>                                                           |
| [**\_WINBIO-EINHEITENSCHEMA \_**](winbio-unit-schema.md)<br/>                                | Beschreibt die Funktionen einer biometrischen Einheit.<br/>                                                                      |
| [**\_WINBIO-VERSION**](winbio-version.md)<br/>                                         | Enthält die Softwareversionsnummer einer Komponente eines biometrischen Dienstanbieters.<br/>                                      |
