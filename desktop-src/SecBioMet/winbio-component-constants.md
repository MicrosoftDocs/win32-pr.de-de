# <a name="winbio_component-constants"></a>WINBIO \_ COMPONENT-Konstanten

Die folgenden Konstanten können beim Aufrufen von [**WinBioControlUnit**](/windows/desktop/api/Winbio/nf-winbio-winbiocontrolunit) oder [**WinBioControlUnitPrivileged**](/windows/desktop/api/Winbio/nf-winbio-winbiocontrolunitprivileged) verwendet werden, um den Typ des verwendeten Adapters anzugeben.



| Konstante                                                                                                                                                                                        | BESCHREIBUNG                             |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------|
| <span id="WINBIO_COMPONENT_SENSOR"></span><span id="winbio_component_sensor"></span><dl> <dt>**\_WINBIO-KOMPONENTENSENSOR \_**</dt> </dl>    | Gibt einen Sensoradapter an.<br/>  |
| <span id="WINBIO_COMPONENT_ENGINE"></span><span id="winbio_component_engine"></span><dl> <dt>**\_ \_ WINBIO-KOMPONENTEN-ENGINE**</dt> </dl>    | Gibt einen Engine-Adapter an.<br/> |
| <span id="WINBIO_COMPONENT_STORAGE"></span><span id="winbio_component_storage"></span><dl> <dt>**\_WINBIO-KOMPONENTENSPEICHER \_**</dt> </dl> | Gibt einen Speicheradapter an.<br/> |
