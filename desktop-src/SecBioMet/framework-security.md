# <a name="framework-security"></a>Framework-Sicherheit

Der Windows-biometrische Dienst ermöglicht es nicht privilegierten Client Anwendungen, biometrische Vorgänge auszuführen, ohne Ihnen direkten Zugriff auf biometrische Beispiele oder Hardware zu bieten. Der Dienst erstellt eine Sicherheitsgrenze zwischen den Clients und den biometrischen Komponenten, wie in der folgenden Abbildung dargestellt.
