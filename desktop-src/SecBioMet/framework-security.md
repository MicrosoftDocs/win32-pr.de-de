# <a name="framework-security"></a>Frameworksicherheit

Der Windows Biometric Service ermöglicht nicht privilegierten Clientanwendungen die Durchführung biometrischer Vorgänge, ohne ihnen direkten Zugriff auf biometrische Stichproben oder Hardware zu bieten. Der Dienst erstellt eine Sicherheitsgrenze zwischen den Clients und den biometrischen Komponenten, wie in der folgenden Abbildung dargestellt.
