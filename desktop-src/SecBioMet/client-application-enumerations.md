# <a name="client-application-enumerations"></a>Clientanwendungsenumerationen

Die folgenden Enumerationstypen werden von der Windows Biometric Framework-API unterstützt.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                      | Beschreibung                                                                                                                                       |
|--------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WINBIO \_ ANTI \_ SPOOF \_ POLICY \_ ACTION**](winbio-anti-spoof-policy-action.md)<br/>  | Gibt die Arten von Aktionen an, die Sie für die Antispoofingrichtlinie eines Benutzers ausführen. <br/>                                                        |
| [**WINBIO \_ ASYNC \_ \_ NOTIFICATION-METHODE**](/windows/desktop/api/Winbio/ne-winbio-winbio_async_notification_method)<br/> | Definiert Konstanten, die angeben, wie Abschlussbenachrichtigungen für asynchrone Vorgänge an die Clientanwendung übermittelt werden sollen.<br/> |
| [**WINBIO \_ CREDENTIAL \_ FORMAT**](winbio-credential-format.md)<br/>                  | Definiert Flags, die verwendet werden können, um das Format der Endbenutzeranmeldeinformationen anzugeben.<br/>                                                              |
| [**WINBIO \_ CREDENTIAL \_ STATE**](winbio-credential-state.md)<br/>                    | Definiert Werte, die angeben, ob den biometrischen Daten für einen Endbenutzer Anmeldeinformationen zugeordnet wurden.<br/>                          |
| [**\_WINBIO-ANMELDEINFORMATIONSTYP \_**](winbio-credential-type.md)<br/>                      | Definiert Flags, die zum Filtern nach dem Anmeldeinformationstyp verwendet werden können.<br/>                                                                       |
| [**\_WINBIO-RICHTLINIENQUELLE \_**](winbio-policy-source.md)<br/>                          | Listet die möglichen Quellen von Richtlinieninformationen für die Erkennung von Spoofing für biometrische Faktoren auf.<br/>                                  |
