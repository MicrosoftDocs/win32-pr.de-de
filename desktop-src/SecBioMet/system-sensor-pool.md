<span data-ttu-id="034d6-113">Eine biometrische Einheit kann in den System Pool eingeschlossen werden, wenn dies der Fall sein kann:</span><span class="sxs-lookup"><span data-stu-id="034d6-113">A biometric unit can be included in the system pool if it can be:</span></span>
