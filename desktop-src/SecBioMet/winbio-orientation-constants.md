# <a name="winbio_orientation-constants"></a>WINBIO \_ ORIENTATION-Konstanten

Die folgenden Konstanten geben die möglichen Kameraausrichtungen an, die die Sensorkomponente als obligatorisch angibt.



| Konstante/Wert                                                                                                                                                                                                                                                           | BESCHREIBUNG                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------|
| <span id="WINBIO_ORIENTATION_UNSPECIFIED"></span><span id="winbio_orientation_unspecified"></span><dl> <dt>**WINBIO \_ AUSRICHTUNG \_ NICHT ANGEGEBEN**</dt> <dt>0</dt> </dl> | Eine obligatorische Ausrichtung für die Kamera ist nicht angegeben.<br/> |
| <span id="WINBIO_ORIENTATION_LANDSCAPE"></span><span id="winbio_orientation_landscape"></span><dl> <dt>**WINBIO \_ \_AUSRICHTUNGSLANDSCHAFT**</dt> <dt>1</dt> </dl>       | Die Querformatausrichtung ist für die Kamera erforderlich.<br/>    |
| <span id="WINBIO_ORIENTATION_PORTRAIT"></span><span id="winbio_orientation_portrait"></span><dl> <dt>**WINBIO \_ \_AUSRICHTUNGS-HOCHFORMAT**</dt> <dt>2</dt> </dl>          | Die Hochformatausrichtung ist für die Kamera erforderlich.<br/>     |
| <span id="WINBIO_ORIENTATION_ANY"></span><span id="winbio_orientation_any"></span><dl> <dt>**WINBIO \_ ORIENTATION \_ ANY**</dt> <dt>3</dt> </dl>                         | Jede Ausrichtung ist für die Kamera zulässig.<br/>             |
