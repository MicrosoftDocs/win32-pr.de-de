# <a name="winbio_orientation-constants"></a>Winbio- \_ Orientierungs Konstanten

Die folgenden Konstanten geben die möglichen Kamera Ausrichtungen an, die von der sensorkomponente als obligatorisch angegeben werden.



| Konstante/Wert                                                                                                                                                                                                                                                           | BESCHREIBUNG                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------|
| <span id="WINBIO_ORIENTATION_UNSPECIFIED"></span><span id="winbio_orientation_unspecified"></span><dl> <dt>**Winbio \_ Ausrichtung \_ nicht angegeben**</dt> <dt>0</dt> </dl> | Es wurde keine obligatorische Ausrichtung für die Kamera angegeben.<br/> |
| <span id="WINBIO_ORIENTATION_LANDSCAPE"></span><span id="winbio_orientation_landscape"></span><dl> <dt>**Winbio \_ Orientierungs \_ Landschaft**</dt> <dt>1</dt> </dl>       | Die quer Ausrichtung ist für die Kamera erforderlich.<br/>    |
| <span id="WINBIO_ORIENTATION_PORTRAIT"></span><span id="winbio_orientation_portrait"></span><dl> <dt>**Winbio \_ Orientation \_**</dt> -Hochformat <dt>2</dt> </dl>          | Die Hochformat Ausrichtung ist für die Kamera erforderlich.<br/>     |
| <span id="WINBIO_ORIENTATION_ANY"></span><span id="winbio_orientation_any"></span><dl> <dt>**Winbio \_ Ausrichtung \_ alle**</dt> <dt>3</dt> </dl>                         | Jede Ausrichtung ist für die Kamera zulässig.<br/>             |
