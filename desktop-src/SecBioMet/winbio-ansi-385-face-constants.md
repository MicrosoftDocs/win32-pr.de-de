# <a name="winbio_ansi_385_face-constants"></a>Winbio \_ ANSI \_ 385- \_ Gesichts Konstanten

Die folgenden Konstanten sind **\_ \_ subtypwerte von winbio-Biometrie** , die verwendet werden können, um die beiden Typen von frontal Bild Bildern anzugeben, wie in ANSI-INCITS 385-2004: "Gesichts Erkennungs Format für Datenaustausch" definiert: vollständige Auflösung und niedrige Auflösung. In der Praxis verwendet das biometrische Framework nur voll auflösende Bilder für die Gesichtserkennung.



| Konstante/Wert                                                                                                                                                                                                                                                                          | BESCHREIBUNG                                                |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------|
| <span id="WINBIO_ANSI_385_FACE_TYPE_UNKNOWN"></span><span id="winbio_ansi_385_face_type_unknown"></span><dl> <dt>**Winbio \_ ANSI \_ 385- \_ Gesichts \_ Typen \_ unbekannt**</dt> <dt>0</dt> </dl>    | Der Bildtyp für frontal Gesichter ist unbekannt.<br/>         |
| <span id="WINBIO_ANSI_385_FACE_FRONTAL_FULL"></span><span id="winbio_ansi_385_face_frontal_full"></span><dl> <dt>**Winbio \_ ANSI \_ 385 \_ Gesicht \_ frontal \_ voll**</dt> <dt>1</dt> </dl>    | Der Bildtyp für frontal Gesichter ist vollständig aufgelöst.<br/> |
| <span id="WINBIO_ANSI_385_FACE_FRONTAL_TOKEN"></span><span id="winbio_ansi_385_face_frontal_token"></span><dl> <dt>**Winbio \_ ANSI \_ 385 \_ Gesicht \_ frontaler \_ Token**</dt> <dt>2</dt> </dl> | Der Bildtyp für frontal Gesichter ist eine niedrige Auflösung.<br/>  |
