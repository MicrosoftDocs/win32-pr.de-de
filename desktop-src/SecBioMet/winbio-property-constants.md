# <a name="winbio_property-constants"></a>WINBIO \_ PROPERTY-Konstanten

Die folgenden Konstanten sind **WINBIO \_ PROPERTY \_ ID-Werte,** mit denen die eigenschaft angegeben werden kann, die im *PropertyId-Parameter* der [**WinBioGetProperty-Funktion abgefragt werden**](/windows/desktop/api/Winbio/nf-winbio-winbiogetproperty) soll. Einige Eigenschaften können auch mit dem gleichen Parameter in der [**WinBioSetProperty-Funktion festgelegt**](/windows/desktop/api/winbio/nf-winbio-winbiosetproperty) werden. Die Ergebnisse der Abfrage werden zurückgegeben oder im *PropertyBuffer-Parameter* dieser Funktionen angegeben, und Typ und Größe des Ergebnisses variieren je nachdem, welche Eigenschaft abgefragt wird.
