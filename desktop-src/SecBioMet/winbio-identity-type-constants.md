# <a name="winbio_identity_type-constants"></a>WINBIO \_ IDENTITY \_ TYPE-Konstanten

Die folgenden **WINBIO \_ IDENTITY \_ TYPE-Konstanten** können verwendet werden, um das Format der Identitätsinformationen anzugeben, die in der [**WINBIO \_ IDENTITY-Struktur**](winbio-identity.md) enthalten sind.



| Konstante                                                                                                                                                                                      | BESCHREIBUNG                                               |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------|
| <span id="WINBIO_ID_TYPE_NULL"></span><span id="winbio_id_type_null"></span><dl> <dt>**\_WINBIO-ID-TYP \_ \_ NULL**</dt> </dl>             | Der Vorlage ist keine ID zugeordnet. <br/>            |
| <span id="WINBIO_ID_TYPE_WILDCARD"></span><span id="winbio_id_type_wildcard"></span><dl> <dt>**WINBIO \_ ID \_ TYPE \_ WILDCARD**</dt> </dl> | Die Struktur entspricht allen Vorlagenidentitäten.<br/> |
| <span id="WINBIO_ID_TYPE_GUID"></span><span id="winbio_id_type_guid"></span><dl> <dt>**\_ \_ WINBIO-ID-TYP-GUID \_**</dt> </dl>             | Eine GUID identifiziert die Vorlage.<br/>                |
| <span id="WINBIO_ID_TYPE_SID"></span><span id="winbio_id_type_sid"></span><dl> <dt>**\_ \_ WINBIO-ID-TYP-SID \_**</dt> </dl>                | Eine Konto-SID identifiziert die Vorlage.<br/>        |
