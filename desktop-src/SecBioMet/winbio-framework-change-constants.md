# <a name="winbio_framework_change-constants"></a>WINBIO \_ FRAMEWORK \_ CHANGE-Konstanten

Die folgenden Konstanten können in der [**WinBioAsyncMonitorFrameworkChanges-Funktion**](/windows/desktop/api/Winbio/nf-winbio-winbioasyncmonitorframeworkchanges) verwendet werden, um den Änderungstyp zu identifizieren, der im Framework aufgetreten ist.

<dl> <dt>

<span id="WINBIO_FRAMEWORK_CHANGE_UNIT"></span><span id="winbio_framework_change_unit"></span>**WINBIO \_ FRAMEWORK \_ CHANGE \_ UNIT**
</dt> <dd> <dl> <dt>

0x00000001
</dt> <dt>



Eine biometrische Einheit wurde an den Computer angefügt oder vom Computer getrennt.


</dt> </dl> </dd> </dl>
