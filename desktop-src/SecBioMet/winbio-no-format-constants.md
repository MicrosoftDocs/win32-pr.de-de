# <a name="winbio_no_format-constants"></a>WINBIO \_ NO \_ FORMAT-Konstanten

Wenn die Member einer [**WINBIO \_ REGISTERED \_ FORMAT-Struktur**](winbio-registered-format.md) nicht 0 (null) sein dürfen, können Sie die folgenden Konstanten verwenden, um nach Fehlern zu überprüfen.



| Konstante/Wert                                                                                                                                                                                                                                                                   | BESCHREIBUNG                                                                                          |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="WINBIO_NO_FORMAT_OWNER_AVAILABLE"></span><span id="winbio_no_format_owner_available"></span><dl> <dt>**WINBIO \_ KEIN \_ \_ FORMATBESITZER \_ VERFÜGBAR**</dt> <dt>0</dt> </dl> | Es ist kein IBIA-Besitzerwert (International Biometric Industry Association) verfügbar.<br/> |
| <span id="WINBIO_NO_FORMAT_TYPE_AVAILABLE"></span><span id="winbio_no_format_type_available"></span><dl> <dt>**WINBIO \_ KEIN \_ \_ FORMATTYP \_ VERFÜGBAR**</dt> <dt>0</dt> </dl>    | Es ist kein vom Besitzer zugewiesener Formattyp verfügbar.<br/>                                               |
