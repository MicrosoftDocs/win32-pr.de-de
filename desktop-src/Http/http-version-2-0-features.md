# <a name="http-version-20-features"></a>HTTP-Version 2,0-Features

Das folgende Feature wird für die HTTP-Server Version 2,0-API unterstützt.

-   Konfigurationsobjekte für Server Sitzung und URL-Gruppe
-   Konfigurations Eigenschaften, die für die Konfigurationsobjekte verfügbar gemacht werden
-   Erweiterte Verwaltung und Konfiguration von Anforderungs Warteschlangen
-   Vollständiger Kernel Modus-SSL
-   Kernel Modus-Antwort Cache
-   Server seitige Authentifizierung
-   Objekt Bereichs bezogene Versionsverwaltung
-   Server seitige Protokollierung
-   Ordnungsgemäßes Herunterfahren der Anforderungs Warteschlange
-   Bedarfs Start in der Anforderungs Warteschlange
-   Portfreigabe
-   Ereignis Ablauf Verfolgung für Windows-Unterstützung
-   Leistungsindikatoren
-   Unterstützung für internationalisierte Domänen Namen (IDN)
-   NET Shell-Konfigurations Hilfsprogramm
