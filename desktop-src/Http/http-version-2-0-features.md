# <a name="http-version-20-features"></a>Features der HTTP-Version 2.0

Das folgende Feature wird für die HTTP Server-API Version 2.0 unterstützt.

-   Konfigurationsobjekte für Serversitzung und URL-Gruppe
-   Konfigurationseigenschaften, die für die Konfigurationsobjekte verfügbar gemacht werden
-   Erweiterte Verwaltung und Konfiguration von Anforderungswarteschlangen
-   Ssl im vollständigen Kernelmodus
-   Kernelmodus-Antwortcache
-   Serverseitige Authentifizierung
-   Objektbereichsversionsierung
-   Serverseitige Protokollierung
-   Ordnungsgemäß herunterfahren der Anforderungswarteschlange
-   Bedarfsstart in der Anforderungswarteschlange
-   Portfreigabe
-   Ereignisablaufverfolgung für Windows Unterstützung
-   Leistungsindikatoren
-   Unterstützung für internationalisierte Domänennamen (INTERNATIONALIZED Domain Name, IDN)
-   Hilfsprogramm für die Net Shell-Konfiguration
