# <a name="deleting-a-reservation"></a>Löschen einer Reservierung

Der Vorgang zum Entfernen einer Reservierung beginnt mit einer Zugriffsüberprüfung, die mit der oben beschriebenen identisch ist. Die Benutzer-ID des aufrufenden Prozesses muss über Zugriffsrechte für den *übergeordneten Knoten* des zu löschenden Reservierungseintrags verfügen.
