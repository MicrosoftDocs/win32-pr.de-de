# <a name="whats-new-for-windows-server-2003-sp1"></a>Neuerungen bei Windows Server 2003 SP1

Die folgenden Features sind für Windows Server 2003 mit Service Pack 1 (SP1) verfügbar:

-   [Kernelmodus-SSL-Unterstützung.](kernel-mode-ssl.md)
    -   Windows Server 2003 mit SP1 fügt die Option zum Ausführen von SSL-Vorgängen im Kernelmodus hinzu, um die Leistung zu verbessern.
-   Unterstützung für die Ausführung von 32-Bit-Anwendungen unter einem 64-Bit-Betriebssystem (WOW64).
-   [Verbesserungen der HTTP-Fehlerprotokollierung.](error-logging-in-windows-server-2003-sp1.md)
    -   Unterstützung für W3C-Stilheader.
    -   Zusätzliche Protokollierungsfelder verfügbar.
    -   Datums- und Uhrzeitbasierter Protokolldateirollover.
-   [HTTP-Analyse-Ausnahmen.](parsing-exceptions.md)
    -   Unterstützt Aus Gründen der Abwärtskompatibilität Ausnahmen von HTTP/1.1.
-   [Verbesserungen des HTTP-Parsers.](parser-enhancements.md)
