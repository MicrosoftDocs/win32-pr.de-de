# <a name="whats-new-for-windows-server-2003-sp1"></a>Neuerungen in Windows Server 2003 SP1

Die folgenden Features sind für Windows Server 2003 mit Service Pack 1 (SP1) verfügbar:

-   [SSL-Unterstützung im Kernel Modus.](kernel-mode-ssl.md)
    -   Windows Server 2003 mit SP1 fügt die Option zum Ausführen von SSL-Vorgängen im Kernel Modus hinzu, um die Leistung zu verbessern.
-   Unterstützung für die Ausführung von 32-Bit-Anwendungen unter 64 Bit-Betriebssystem (WOW64).
-   [Erweiterungen der HTTP-Fehler Protokollierung.](error-logging-in-windows-server-2003-sp1.md)
    -   Unterstützung für W3C-Stil Header.
    -   Zusätzliche Protokollierungs Felder verfügbar.
    -   Datums-und Uhrzeit basierter Rollover der Protokolldatei.
-   [HTTP-Element Ausnahmen.](parsing-exceptions.md)
    -   Unterstützt Ausnahmen für HTTP/1.1 aus Gründen der Abwärtskompatibilität.
-   [Erweiterungen für den HTTP-Parser.](parser-enhancements.md)
