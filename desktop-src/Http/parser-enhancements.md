# <a name="parser-enhancements"></a>Parsererweiterungen

In Windows Server 2003 mit Service Pack 1 (SP1) lässt die HTTP-Server-API die folgenden Anforderungen von HTTP-Clients zu.

-   Anforderungen, die einen einzelnen Zeilenvorschub (Single Line Feed, LF) als Zeilenabschlusszeichen verwenden.
-   Anforderungen, die linearen Leerraum (LWS) zwischen der HTTP-Anforderungszeile und dem Anfang von Headern enthalten.

Diese Verhaltensweisen sind standardmäßig aktiviert und werden nicht durch Registrierungseinstellungen gesteuert.
