# <a name="netsh-commands-for-http"></a>Netsh-Befehle für HTTP

Sie können die Netsh-Befehle für den HTTP-Kontext zum Abfragen und Konfigurieren HTTP.sys Einstellungen und Parametern verwenden.

Sie können diese Befehle an der Eingabeaufforderung im Windows Vista-Betriebssystem oder an der Eingabeaufforderung für den **netsh-HTTP-Kontext** ausführen. Damit diese Befehle an der Eingabeaufforderung in Windows Vista funktionieren, müssen Sie **netsh http** eingeben, bevor Sie Befehle und Parameter eingeben, wie sie in der folgenden Syntax angezeigt werden. Geben Sie *CommandName***/?** ein, wobei *CommandName* der Name des Befehls ist, um hilfe für einen Befehl an der Eingabeaufforderung ein.

Informationen zur Befehlssyntax finden Sie in den Befehlen:

-   [**add iplisten**](add-iplisten.md)
-   [**add sslcert**](add-sslcert.md)
-   [**add timeout**](add-timeout.md)
-   [**add urlacl**](add-urlacl.md)
-   [**delete cache**](delete-cache.md)
-   [**delete iplisten**](delete-iplisten.md)
-   [**delete sslcert**](delete-sslcert.md)
-   [**delete timeout**](delete-timeout.md)
-   [**delete urlacl**](delete-urlacl.md)
-   [**flush logbuffer**](flush-logbuffer.md)
-   [**show cachestate**](show-cachestate.md)
-   [**show iplisten**](show-iplisten.md)
-   [**show servicestate**](show-servicestate.md)
-   [**show sslcert**](show-sslcert.md)
-   [**show timeout**](show-timeout.md)
-   [**show urlacl**](show-urlacl.md)
