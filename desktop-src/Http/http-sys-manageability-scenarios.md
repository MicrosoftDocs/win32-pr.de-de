# <a name="httpsys-manageability-scenarios"></a>HTTP.sys verwaltbarkeitsszenarios

Die folgenden Szenarien enthalten Beispiele dafür, wie diese Tools IT-Experten bei der Diagnose und Behebung von Problemen im Zusammenhang mit der Komponente der HTTP-Server-API (HTTP.sys) helfen können. Die in diesen Beispielen verwendeten Tools sind in Windows Vista und Windows Server 2008 verfügbar.

-   [Szenario 1: Beispiel für http-Timeout mit etw-Ablauf Verfolgung und Netsh-Befehlen](scenario-1--http-timeout-example-using-etw-tracing-and-netsh-commands.md)
-   [Szenario 2: parserbeispiel mit etw-Ablauf Verfolgung](scenario-2--parser-example-using-etw-tracing.md)
-   [Szenario 3: Leistungsindikatoren](scenario-3--performance-counters.md)

Die Szenarien in diesem Dokument sind Beispiele für die Verwendung der Verwaltbarkeitstools der HTTP-Server-API-Komponente. Diese Beispiele sollen keine bestimmten Konfigurationsänderungen vorschlagen.
