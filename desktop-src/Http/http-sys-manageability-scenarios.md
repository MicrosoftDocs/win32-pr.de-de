# <a name="httpsys-manageability-scenarios"></a>HTTP.sys Verwaltbarkeitsszenarien

Die folgenden Szenarien enthalten Beispiele dafür, wie diese Tools IT-Experten beim Diagnostizieren und Beheben von Problemen im Zusammenhang mit der HTTP-Server-API -Komponente (HTTP.sys) unterstützen können. Die in diesen Beispielen verwendeten Tools sind in Windows Vista und Windows Server 2008 verfügbar.

-   [Szenario 1: HTTP-Timeoutbeispiel mit ETW-Ablaufverfolgung und Netsh-Befehlen](scenario-1--http-timeout-example-using-etw-tracing-and-netsh-commands.md)
-   [Szenario 2: Parserbeispiel mit ETW-Ablaufverfolgung](scenario-2--parser-example-using-etw-tracing.md)
-   [Szenario 3: Leistungsindikatoren](scenario-3--performance-counters.md)

Die Szenarien in diesem Dokument sind Beispiele für die Verwendung der Verwaltungstools der HTTP-Server-API-Komponente. Diese Beispiele sollen keine spezifischen Konfigurationsänderungen vorschlagen.
