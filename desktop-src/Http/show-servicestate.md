# <a name="show-servicestate"></a>show servicestate

Zeigt eine Momentaufnahme des http-Dienes an.

``` syntax
show servicestate [[view=]{session|requestq}] [[verbose=]{yes|no}]
 
```

## <a name="parameters"></a>Parameter

<dl> <dt>

<span id="__view___session_requestq__"></span><span id="__VIEW___SESSION_REQUESTQ__"></span>**\[\[View = \] {Session \| requestq}\]**
</dt> <dd>

Anzeigen einer Momentaufnahme des http-Dienststatus basierend auf Server Sitzung oder Anforderungs Warteschlangen.

</dd> <dt>

<span id="__verbose___yes_no__"></span><span id="__VERBOSE___YES_NO__"></span>**\[\[Verbose = \] {Yes \| No}\]**
</dt> <dd>

Ausführliche Informationen zu den Eigenschafts Informationen anzeigen.

</dd> </dl>

## <a name="examples"></a>Beispiele

**SERVICESTATE ausgeführt wird View = Session anzeigen**

**SERVICESTATE ausgeführt wird View = requestq Verbose = Yes anzeigen**
