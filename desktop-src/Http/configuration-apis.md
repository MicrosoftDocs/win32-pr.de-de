# <a name="configuration-apis"></a>Konfiguration-APIs

Die HTTP-Server-APIs stellen Funktionen bereit, mit denen Systemadministratoren Konfigurationsparameter festlegen, Abfragen und löschen können. Die Konfigurations-APIs steuern die Einstellungen für alle Anwendungen, die die HTTP-Server-API auf dem System verwenden.

Die unten aufgeführten Abschnitte enthalten Informationen zu Konfigurationseinstellungen.

> [!Note]  
> All dies sind optionale Konfigurationsschritte, die basierend auf den System-und/oder Anwendungsanforderungen angewendet werden müssen.

 

[IP-Abhörliste](ip-listen-list.md)   – Beschreibt das Konfigurieren von http für den Betrieb über bestimmte IP-Schnittstellen.

[SSL-Zertifikate](ssl-certificates.md)   – erläutert die Verfahren zum Konfigurieren von Zertifikaten mit der HTTP-Server-API.

[Namespace Reservierungen, Registrierungen und Routing](namespace-reservations-registrations-and-routing.md)   – Beschreibt den Prozess der Reservierung von Teilen des Namespace für alle Anwendungen auf dem Computer mithilfe der HTTP-Server-API. Der Prozess zum Registrieren und Weiterleiten von Anforderungen an bestimmte Anwendungen wird ebenfalls beschrieben.

[HttpCfg.exe](httpcfg-exe.md)   – ein Beispielprogramm, das die Konfigurations-APIs verwendet. Eine kompilierte Version dieses Tools wird auch mit dem Betriebssystem der Support Tools geliefert, zusammen mit einer Hilfedatei, in der dokumentiert wird, wie Sie verwendet wird.
