# <a name="show-iplisten"></a>show iplisten

Listet alle IP-Adressen in der IP-Abhörliste auf. Die IP-Abhörliste wird verwendet, um die Liste der Adressen zu erweitern, an die der HTTP-Dienst gebunden ist. „0.0.0.0“ bedeutet eine beliebige IPv4-Adresse und „::“ bedeutet eine beliebige IPv6-Adresse.

``` syntax
show iplisten
 
```
