# <a name="httpcfgexe"></a>HttpCfg.exe

Mit diesem Tool können Sie die HTTP-Server-API konfigurieren, indem es die [HTTP-Konfigurations-APIs](configuration-apis.md) für die folgenden Parameter umschließt:

-   IP-Listenliste
-   SSL-Zertifikate
-   Namespacereservierungen

HttpCfg.exe enthält das Betriebssystem ab Windows Server 2003 als binäre ausführbare Datei sowie eine Hilfedatei, die die Verwendung beschreibt. Sie kann vom Betriebssystemverteilungs-Datenträger als Supporttool installiert werden.

HttpCfg.exe ist auch als Download verfügbar. weitere Informationen finden Sie [unter Windows XP SP2-Supporttools.](https://www.microsoft.com/downloads/details.aspx?familyid=49AE8576-9BB9-4126-9761-BA8011FABF38)

**Windows Vista und Windows Server 2008:** HttpCfg.exe ist veraltet und wird in netsh.exe durch ähnliche Funktionen ersetzt.
