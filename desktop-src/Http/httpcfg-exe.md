# <a name="httpcfgexe"></a>HttpCfg.exe

Mit diesem Tool können Sie die HTTP-Server-API konfigurieren, indem Sie die HTTP- [Konfigurations-APIs](configuration-apis.md) für die folgenden Parameter Umpacken:

-   IP-Abhörliste
-   SSL-Zertifikate
-   Namespace Reservierungen

HttpCfg.exe wird mit dem Betriebssystem gestartet, beginnend mit Windows Server 2003 als binäre ausführbare Datei, sowie eine Hilfedatei, in der die Verwendung beschrieben wird. Sie kann auf dem Betriebssystem-Verteilungs Datenträger als Support Tool installiert werden.

HttpCfg.exe steht auch als Download zur Verfügung. Weitere Informationen finden Sie [unter Support Tools für Windows XP SP2](https://www.microsoft.com/downloads/details.aspx?familyid=49AE8576-9BB9-4126-9761-BA8011FABF38).

**Windows Vista und Windows Server 2008:** HttpCfg.exe ist veraltet und wird durch eine ähnliche Funktionalität in netsh.exe ersetzt.
