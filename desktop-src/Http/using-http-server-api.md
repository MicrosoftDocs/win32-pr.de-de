# <a name="using-http-server-api"></a>Verwenden der HTTP-Server-API

Die Quelldateien des Anwendungsprogramms enthalten die http. h-Header Datei für den Zugriff auf Funktionsprototypen und Strukturdefinitionen für die HTTP-Server-API. Entwickler können die Bibliothek Datei "Httpapi. lib" verwenden, um Anwendungen zu erstellen, die die HTTP-Server-API verwenden. Zur Laufzeit verknüpfen Anwendungen mit dem Httpapi.dll.
