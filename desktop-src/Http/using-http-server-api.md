# <a name="using-http-server-api"></a>Verwenden der HTTP-Server-API

Quelldateien des Anwendungsprogramms enthalten die Http.h-Headerdatei, um auf Funktionsprototypen und Strukturdefinitionen für die HTTP-Server-API zuzugreifen. Entwickler können die Bibliotheksdatei Httpapi.lib verwenden, um Anwendungen zu erstellen, die die HTTP-Server-API verwenden. Zur Laufzeit sind Anwendungen mit dem Httpapi.dll.
