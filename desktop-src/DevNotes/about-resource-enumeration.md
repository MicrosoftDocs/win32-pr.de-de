# <a name="about-resource-enumeration"></a>Informationen zur Ressourcenenumeration

Die Ressourcenenumeration bietet eine genaue und genaue Ansicht der Instrumentierungsaktivität, die bei der Ausführung einer Anwendung stattfindet. Die Instrumentierungsinformationen können im Rahmen des Debugprozesses abstrahiert und als eine Reihe prozessspezifischer Elemente kategorisiert werden.

Tools, z. B. Debugger, benötigen häufig Zugriff auf Instrumentierungsinformationen, stellen aber möglicherweise fest, dass sie nicht verfügbar sind. Um dieses Problem zu beheben, extrahiert die [**VerifierEnumerateResource-Funktion**](/windows/desktop/api/Avrfsdk/nf-avrfsdk-verifierenumerateresource) diese Art von Informationen aus Anwendungen, um bessere Kontextinformationen für das zu debuggende Problem bereitzustellen.
