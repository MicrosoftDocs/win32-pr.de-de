# <a name="about-resource-enumeration"></a>Informationen zur ressourcenenumeration

Die ressourcenenumeration bietet eine genaue Ansicht der Instrumentations Aktivität, die stattfindet, wenn eine Anwendung ausgeführt wird. Die Instrumentierungs Informationen können als Teil des Debugprozesses als Satz Prozess spezifischer Elemente abstrahiert und kategorisiert werden.

Tools, wie z. b. Debugger, benötigen häufig Zugriff auf Instrumentations Informationen, sind aber möglicherweise nicht verfügbar. Um dieses Problem zu beheben, extrahiert die [**verifierenumerateresource**](/windows/desktop/api/Avrfsdk/nf-avrfsdk-verifierenumerateresource) -Funktion diese Art von Informationen aus Anwendungen, um bessere Kontextinformationen für das Problem bereitzustellen, das gedebuggt wird.
