# <a name="_alldiv-routine"></a>\_alldiv-Routine

Dividiert zwei **LONGLONG-Ganzzahlen.** Um beispielsweise zwei int64-Werte zu teilen, generiert der Compiler möglicherweise einen Aufruf von \_ alldiv Routine.

## <a name="remarks"></a>Hinweise

\_alldiv Routine ist eine Hilfsroutine für den C-Compiler. Ob der Compiler \_ alldiv Routine aufruft, hängt vollständig vom Optimierungssatz ab.
