# <a name="_alldiv-routine"></a>\_alldiv-Routine

Dividiert zwei **Longlong** -Ganzzahlen. Um z. b. zwei Int64-Werte aufzuteilen, generiert der Compiler möglicherweise einen Aufrufen der \_ alldiv-Routine.

## <a name="remarks"></a>Bemerkungen

\_alldiv Routine ist eine Hilfsroutine für den C-Compiler. Ob der Compiler die \_ alldiv-Routine aufruft, ist vollständig vom Optimierungs Satz abhängig.
