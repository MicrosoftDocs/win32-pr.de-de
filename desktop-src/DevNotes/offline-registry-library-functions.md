# <a name="offline-registry-library-functions"></a>Funktionen der Offline-Registrierungs Bibliothek

Im folgenden werden die Funktionen der Offline-Registrierungs Bibliothek angezeigt.



| Funktion                                       | BESCHREIBUNG                                                                                                         |
|------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**Orclosehive**](orclosehive.md)             | Schließt die angegebene Offline Registrierungs Struktur und gibt für die Hive zugeordnete Arbeitsspeicher frei.                                 |
| [**Orclosekey**](orclosekey.md)               | Schließt ein Handle für den angegebenen Registrierungsschlüssel in einer Offline Registrierungs Struktur.                                          |
| [**Orkreatehive**](orcreatehive.md)           | Erstellt eine Offline Registrierungs Struktur, die einen einzelnen leeren Stamm Schlüssel enthält.                                             |
| [**Orkreatekey**](orcreatekey.md)             | Erstellt den angegebenen Registrierungsschlüssel in einer Offline Registrierungs Struktur. Wenn der Schlüssel bereits vorhanden ist, wird er von der Funktion geöffnet.   |
| [**Ordeletekey**](ordeletekey.md)             | Löscht einen Unterschlüssel und seine Werte aus einer Offline Registrierungs Struktur.                                                      |
| [**Ordeletevalue**](ordeletevalue.md)         | Entfernt einen Wert aus dem angegebenen Registrierungsschlüssel in einer Offline Registrierungs Struktur.                                        |
| [**Orenkey**](orenumkey.md)                 | Listet die Unterschlüssel des angegebenen geöffneten Registrierungsschlüssels in einer Offline Registrierungs Struktur auf.                              |
| [**Orenvalue**](orenumvalue.md)             | Listet die Werte für den angegebenen geöffneten Registrierungsschlüssel in einer Offline Registrierungs Struktur auf.                              |
| [**Orgetkeysecurity**](orgetkeysecurity.md)   | Ruft eine Kopie der Sicherheits Beschreibung ab, die den angegebenen geöffneten Registrierungsschlüssel in einer Offline Registrierungs Struktur schützt. |
| [**Orgetvalue**](orgetvalue.md)               | Ruft den Typ und die Daten für den angegebenen Registrierungs Wert in einer Offline Registrierungs Struktur ab.                           |
| [**Orgetversion**](orgetversion.md)           | Ruft die Version der Offline Registrierungs Bibliothek ab.                                                              |
| [**Orgetvirtualflags**](orgetvirtualflags.md) | Ruft die virtuellen Flags für den angegebenen geöffneten Registrierungsschlüssel in einer Offline Registrierungs Struktur ab.                         |
| [**Oropenhive**](oropenhive.md)               | Lädt die angegebene Hive-Datei in den Arbeitsspeicher und überprüft die Hive.                                                   |
| [**Oropenkey**](oropenkey.md)                 | Öffnet den angegebenen Registrierungsschlüssel in einer Offline Registrierungs Struktur.                                                       |
| [**Orqueryinfokey**](orqueryinfokey.md)       | Ruft Informationen über den angegebenen Registrierungsschlüssel in einer Offline Registrierungs Struktur ab.                                 |
| [**Orsavehive**](orsavehive.md)               | Schreibt die angegebene Offline Registrierungs Struktur in eine Datei.                                                               |
| [**Orsetkeysecurity**](orsetkeysecurity.md)   | Legt die Sicherheit eines geöffneten Registrierungsschlüssels in einer Offline Registrierungs Struktur fest.                                              |
| [**Orsetvalue**](orsetvalue.md)               | Legt die Daten für den Wert eines angegebenen Registrierungsschlüssels in einer Offline Registrierungs Struktur fest.                                |
| [**Orsetvirtualflags**](orsetvirtualflags.md) | Legt virtualisierungsflags für den angegebenen geöffneten Registrierungsschlüssel in einer Offline Registrierungs Struktur fest.                           |
