# <a name="offline-registry-library-functions"></a>Funktionen der Offlineregistrierungsbibliothek

Im Folgenden finden Sie die Funktionen der Offlineregistrierungsbibliothek.



| Funktion                                       | Beschreibung                                                                                                         |
|------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**ORCloseHive**](orclosehive.md)             | Schließt die angegebene Offlineregistrierungsstruktur und gibt den für die Struktur zugeordneten Arbeitsspeicher frei.                                 |
| [**ORCloseKey**](orclosekey.md)               | Schließt ein Handle für den angegebenen Registrierungsschlüssel in einer Offlineregistrierungsstruktur.                                          |
| [**ORCreateHive**](orcreatehive.md)           | Erstellt eine Offlineregistrierungsstruktur, die einen einzelnen leeren Stammschlüssel enthält.                                             |
| [**ORCreateKey**](orcreatekey.md)             | Erstellt den angegebenen Registrierungsschlüssel in einer Offlineregistrierungsstruktur. Wenn der Schlüssel bereits vorhanden ist, öffnet die Funktion ihn.   |
| [**ORDeleteKey**](ordeletekey.md)             | Löscht einen Unterschlüssel und seine Werte aus einer Offlineregistrierungsstruktur.                                                      |
| [**ORDeleteValue**](ordeletevalue.md)         | Entfernt einen Wert aus dem angegebenen Registrierungsschlüssel in einer Offlineregistrierungsstruktur.                                        |
| [**OREnumKey**](orenumkey.md)                 | Enumeriert die Unterschlüssel des angegebenen offenen Registrierungsschlüssels in einer Offlineregistrierungsstruktur.                              |
| [**OREnumValue**](orenumvalue.md)             | Enumeriert die Werte für den angegebenen offenen Registrierungsschlüssel in einer Offlineregistrierungsstruktur.                              |
| [**ORGetKeySecurity**](orgetkeysecurity.md)   | Ruft eine Kopie der Sicherheitsbeschreibung ab, die den angegebenen offenen Registrierungsschlüssel in einer Offlineregistrierungsstruktur schützt. |
| [**ORGetValue**](orgetvalue.md)               | Ruft den Typ und die Daten für den angegebenen Registrierungswert in einer Offlineregistrierungsstruktur ab.                           |
| [**ORGetVersion**](orgetversion.md)           | Ruft die Version der Offlineregistrierungsbibliothek ab.                                                              |
| [**ORGetVirtualFlags**](orgetvirtualflags.md) | Ruft die virtuellen Flags für den angegebenen offenen Registrierungsschlüssel in einer Offlineregistrierungsstruktur ab.                         |
| [**OROpenHive**](oropenhive.md)               | Lädt die angegebene Hive-Datei in den Arbeitsspeicher und überprüft die Struktur.                                                   |
| [**OROpenKey**](oropenkey.md)                 | Öffnet den angegebenen Registrierungsschlüssel in einer Offlineregistrierungsstruktur.                                                       |
| [**ORQueryInfoKey**](orqueryinfokey.md)       | Ruft Informationen zum angegebenen Registrierungsschlüssel in einer Offlineregistrierungsstruktur ab.                                 |
| [**ORSaveHive**](orsavehive.md)               | Schreibt die angegebene Offlineregistrierungsstruktur in eine Datei.                                                               |
| [**ORSetKeySecurity**](orsetkeysecurity.md)   | Legt die Sicherheit eines geöffneten Registrierungsschlüssels in einer Offlineregistrierungsstruktur fest.                                              |
| [**ORSetValue**](orsetvalue.md)               | Legt die Daten für den Wert eines angegebenen Registrierungsschlüssels in einer Offlineregistrierungsstruktur fest.                                |
| [**ORSetVirtualFlags**](orsetvirtualflags.md) | Legt Virtualisierungsflags für den angegebenen offenen Registrierungsschlüssel in einer Offlineregistrierungsstruktur fest.                           |
