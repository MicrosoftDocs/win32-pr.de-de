# <a name="patchapi"></a>PatchAPI

In diesem Abschnitt werden mehrere Funktionen beschrieben, die zur PatchAPI-Schnittstelle gehören. PatchAPI ist Teil der [Delta Komprimierungs-API](/previous-versions/bb417345(v=msdn.10)), die für die programmgesteuerte Erstellung von Delta-Deltas (ähnlich wie Patches oder komprimierte diffs) zwischen Dateien entworfen wurde.
