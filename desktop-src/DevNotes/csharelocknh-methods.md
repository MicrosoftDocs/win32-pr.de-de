# <a name="csharelocknh-methods"></a>Csharelocknh-Methoden

Eine Gruppe von Methoden, die zum Bearbeiten von Sperren verwendet wird.

## <a name="methods"></a>Methoden

Die folgenden Methoden werden von Rwnh.dll exportiert.



| Methode                                                                   | BESCHREIBUNG                                                     |
|--------------------------------------------------------------------------|-----------------------------------------------------------------|
| [**Exclusivelock**](csharelocknh--exclusivelock.md)                     | Erhält eine Lese-/Schreibsperre.                                  |
| [**Exclusivetopartial**](csharelocknh--exclusivetopartial.md)           | Ändert den Zustand.                                              |
| [**Exclusiveunlock**](csharelocknh--exclusiveunlock.md)                 | Gibt eine Sperre frei.                                                |
| [**Firstpartialto Exclusive**](csharelocknh--firstpartialtoexclusive.md) | Wird verwendet, um eine partielle Sperre in eine exklusive Sperre umzuwandeln.         |
| [**Partitionier.**](csharelocknh--partiallock.md)                         | Verhindert, dass mehr als ein Thread eine Sperre erhält. |
| [**Partialunlock**](csharelocknh--partialunlock.md)                     | Gibt eine partielle Sperre frei.                                        |
| [**Sharelock**](csharelocknh--sharelock.md)                             | Erhält eine Sperre für den freigegebenen Modus.                                 |
| [**Share Unlock**](csharelocknh--shareunlock.md)                         | Gibt eine Sperre aus dem freigegebenen Modus frei.                               |
| [**Sharedtopartial**](csharelocknh--sharedtopartial.md)                 | Ruft eine partielle Sperre ab.                                         |
| [**Tryexclusivelock**](csharelocknh--tryexclusivelock.md)               | Erhält exklusiv eine Sperre.                                     |
