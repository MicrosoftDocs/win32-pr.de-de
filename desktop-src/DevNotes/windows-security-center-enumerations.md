# <a name="windows-security-center-enumerations"></a>Windows-Sicherheit Center-Enumerationen

Die folgenden Enumerationen werden mit Windows-Sicherheit Center verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                | Beschreibung                                                                                                      |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**WSC \_ SECURITY \_ PRODUCT \_ STATE**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_product_state)<br/>       | Definiert den aktuellen Status des Sicherheitsprodukts, das für Windows-Sicherheit Center verfügbar gemacht wird. <br/> |
| [**\_WSC-SICHERHEITSANBIETER \_**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider)<br/>                  | Definiert alle Dienste, die von Windows-Sicherheit Center (WSC) überwacht werden.<br/>                         |
| [**INTEGRITÄT DES \_ \_ WSC-SICHERHEITSANBIETERS \_**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider_health)<br/>   | Definiert die möglichen Zustände für jeden Dienst, der von Windows-Sicherheit Center (WSC) überwacht wird.<br/>               |
| [**\_ \_ WSC-SICHERHEITSSIGNATURSTATUS \_**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_signature_status)<br/> | Meldet den aktuellen Versionsstatus des Sicherheitsprodukts an Windows-Sicherheit Center. <br/>               |
