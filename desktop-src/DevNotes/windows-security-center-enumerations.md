# <a name="windows-security-center-enumerations"></a>Windows Security Center-Enumerationen

Die folgenden Enumerationen werden mit Windows-Security Center verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                | BESCHREIBUNG                                                                                                      |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**WSC- \_ Sicherheits \_ Produkt \_ Status**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_product_state)<br/>       | Definiert den aktuellen Status des Sicherheits Produkts, das für Windows Security Center verfügbar gemacht wird. <br/> |
| [**WSC- \_ Sicherheits \_ Anbieter**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider)<br/>                  | Definiert alle Dienste, die von Windows Security Center (WSC) überwacht werden.<br/>                         |
| [**Integrität des WSC- \_ Sicherheits \_ Anbieters \_**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider_health)<br/>   | Definiert die möglichen Zustände für jeden von Windows Security Center (WSC) überwachten Dienst.<br/>               |
| [**Status der WSC- \_ Sicherheits \_ Signatur \_**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_signature_status)<br/> | Meldet den aktuellen Versions Status des Sicherheits Produkts an Windows Security Center. <br/>               |
