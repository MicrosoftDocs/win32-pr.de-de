# <a name="windows-security-center"></a>Windows-Security Center

Windows Security Center (WSC) ist ein umfassendes Bericht Erstellungs Tool, mit dem Benutzer eine Schutz Sicherheitsebene um Ihre Computersysteme einrichten und verwalten können. Sobald eine Sicherheitsschicht eingerichtet ist, ist die Windows-Security Center unauffällig, da Sie den Integritäts Zustand des Computers überwacht. Wenn jedoch Sicherheitsrisiken bestehen, bietet WSC Warnungen und eine ausführliche Anleitung, um dem Benutzer zu helfen, einen sicheren Zustand zu erreichen, der dem Endbenutzer über das Aktions Center angezeigt wird.

Damit Sicherheitslösungen von Drittanbietern (Antivirus, Antischadsoftware oder Antispyware) mit Windows kompatibel sind und der Status für das Aktions Center erfolgreich gemeldet werden kann, müssen Sie sich bei Security Center registrieren und alle nachfolgenden Statusänderungen mithilfe privater APIs für die Kommunikation mit dem WSC melden. WSC kommuniziert diese Updates wiederum an das Aktions Center, wo Sie schließlich dem Endbenutzer angezeigt werden.
