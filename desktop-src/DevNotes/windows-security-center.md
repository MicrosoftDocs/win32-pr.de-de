<span data-ttu-id="11dab-104">Windows Security Center (WSC) ist ein umfassendes Bericht Erstellungs Tool, mit dem Benutzer eine Schutz Sicherheitsebene um Ihre Computersysteme einrichten und verwalten können.</span><span class="sxs-lookup"><span data-stu-id="11dab-104">Windows Security Center (WSC) is a comprehensive reporting tool that helps users establish and maintain a protective security layer around their computer systems.</span></span> <span data-ttu-id="11dab-105">Sobald eine Sicherheitsschicht eingerichtet ist, ist die Windows-Security Center unauffällig, da Sie den Integritäts Zustand des Computers überwacht.</span><span class="sxs-lookup"><span data-stu-id="11dab-105">Once a security layer is established, Windows Security Center is inconspicuous as it monitors the computer’s health state.</span></span> <span data-ttu-id="11dab-106">Wenn jedoch Sicherheitsrisiken bestehen, bietet WSC Warnungen und eine ausführliche Anleitung, um dem Benutzer zu helfen, einen sicheren Zustand zu erreichen, der dem Endbenutzer über das Aktions Center angezeigt wird.</span><span class="sxs-lookup"><span data-stu-id="11dab-106">However, if vulnerabilities exist, WSC provides alerts and prescriptive guidance to assist the user in achieving a secure state which is surfaced to the end user through the Action Center.</span></span>
