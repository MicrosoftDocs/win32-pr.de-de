# <a name="windows-security-center"></a>Windows-Sicherheit Center

Windows-Sicherheit Center (WSC) ist ein umfassendes Berichterstellungstool, mit dem Benutzer eine schützende Sicherheitsebene um ihre Computersysteme einrichten und verwalten können. Sobald eine Sicherheitsschicht eingerichtet ist, Windows-Sicherheit Center unauffällig, da der Integritätsstatus des Computers überwacht wird. Wenn jedoch Sicherheitsrisiken vorhanden sind, bietet WSC Warnungen und anleitungen, um den Benutzer beim Erreichen eines sicheren Zustands zu unterstützen, der dem Endbenutzer über das Action Center angezeigt wird.

Damit Sicherheitslösungen von Drittanbietern (Antivirensoftware, Antischadsoftware oder Antispyware) mit Windows kompatibel sind und den Status erfolgreich an das Action Center melden, müssen sie sich bei Security Center registrieren und alle nachfolgenden Statusänderungen mithilfe privater APIs für die Kommunikation mit WSC melden. WSC kommuniziert diese Updates wiederum an das Action Center, wo sie schließlich dem Endbenutzer angezeigt werden.
