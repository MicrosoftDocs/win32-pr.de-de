# <a name="sapcreateinstance-method"></a>SapCreateInstance-Methode

Erstellt einen Kommunikationskanal mit dem Treiber für den sicheren Audiopfad.

## <a name="syntax"></a>Syntax


```C++
MSSAPDLLObject* SapCreateInstance();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Zeiger auf ein **MSSAPDLLObject-Objekt.**
