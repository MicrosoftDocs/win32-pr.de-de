# <a name="sndvol32exe"></a>SndVol32.exe

Das SndVol32.exe Programm steuert die Volumeeinstellungen für das System. Um dieses volumesteuerelement anzuzeigen, wechseln Sie zur Systemsteuerung, und starten Sie Sounds und Audiogeräte. Wechseln Sie zu **Geräte Volume** , und klicken Sie auf **erweitert**.

Um dieses Programm mit Befehlszeilen Argumenten auszuführen, wechseln Sie zum **Startmenü** , klicken Sie auf **Ausführen**, und geben Sie den folgenden Befehl ein.

**SndVol32.exe \[ -R \| -D** *\] n * ** **

<dl> <dt>

<span id="-R"></span><span id="-r"></span>-R
</dt> <dd>

Startet die Anwendung im Daten Satz Modus.

</dd> <dt>

<span id="-D_n"></span><span id="-d_n"></span><span id="-D_N"></span>-D *n*
</dt> <dd>

Gibt einen Geräte Bezeichner an.

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

Dieses Programm ist das Native audiovolume und das in Windows enthaltene mischungssteuerelement. Sie kann nicht von Microsoft heruntergeladen werden. Das Programm wird in der Regel im folgenden Verzeichnis installiert: C: \\ Windows \\ system32.

Um diese Datei auf Ihrem System zu finden, öffnen Sie Windows-Explorer, klicken Sie auf die Schaltfläche **Suchen** , und wählen Sie **alle Dateien und Ordner** aus. Geben Sie den Dateinamen (SndVol32.exe) in die erste verfügbare Zeile ein, und klicken Sie auf die Schaltfläche **Suchen** .

Wenn Sie weitere Informationen zur Verwendung dieses Steuer Elements benötigen, starten Sie das Programm, und wählen Sie **Hilfe** aus. Beachten Sie, dass es keine Möglichkeit gibt, Programm gesteuert auf die Funktionalität dieses Programms zuzugreifen.
