# <a name="tag-types"></a>TAG-Typen

Beschreibt den Datentyp des Werts, der einem TAG zugeordnet ist.



| Konstante/Wert                                                                                                                                                                                                                            | Beschreibung                                           |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| <span id="TAG_TYPE_NULL"></span><span id="tag_type_null"></span><dl> <dt>**TAG \_ TYPE \_ NULL**</dt> <dt>0x1000</dt> </dl>                | Dem TAG ist kein Wert zugeordnet.<br/> |
| <span id="TAG_TYPE_BYTE"></span><span id="tag_type_byte"></span><dl> <dt>**TAG \_ TYPE \_ BYTE**</dt> <dt>0x2000</dt> </dl>                | Ein **BYTE-Wert.**<br/>                          |
| <span id="TAG_TYPE_WORD"></span><span id="tag_type_word"></span><dl> <dt>**TAG \_ TYPE \_ WORD**</dt> <dt>0x3000</dt> </dl>                | Ein **WORD-Wert.**<br/>                          |
| <span id="TAG_TYPE_DWORD"></span><span id="tag_type_dword"></span><dl> <dt>**TAG \_ TYPE \_ DWORD**</dt> <dt>0x4000</dt> </dl>             | Ein **DWORD-Wert.**<br/>                         |
| <span id="TAG_TYPE_QWORD"></span><span id="tag_type_qword"></span><dl> <dt>**TAG \_ TYPE \_ QWORD**</dt> <dt>0x5000</dt> </dl>             | Ein **ULONGLONG-Wert.**<br/>                     |
| <span id="TAG_TYPE_STRINGREF"></span><span id="tag_type_stringref"></span><dl> <dt>**TAG \_ TYPE \_ STRINGREF**</dt> <dt>0x6000</dt> </dl> | Ein tokenisierter Zeichenfolgenwert.<br/>                  |
| <span id="TAG_TYPE_LIST"></span><span id="tag_type_list"></span><dl> <dt>**TAG \_ TYPE \_ LIST**</dt> <dt>0x7000</dt> </dl>                | Eine Liste von TAG-Werten.<br/>                      |
| <span id="TAG_TYPE_STRING"></span><span id="tag_type_string"></span><dl> <dt>**TAG \_ TYPE \_ STRING**</dt> <dt>0x8000</dt> </dl>          | Ein Zeichenfolgenwert.<br/>                            |
| <span id="TAG_TYPE_BINARY"></span><span id="tag_type_binary"></span><dl> <dt>**TAG \_ TYPE \_ BINARY**</dt> <dt>0x9000</dt> </dl>          | Ein binärer Wert.<br/>                            |
