# <a name="windows-security-center-interfaces"></a>Windows Security Center-Schnittstellen

Die folgenden Schnittstellen sind für die Verwendung mit Windows-Security Center verfügbar.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                 | BESCHREIBUNG                                                                                                                      |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**Iwscproduct**](/windows/desktop/api/Iwscapi/nn-iwscapi-iwscproduct)<br/>         | Stellt Methoden bereit, mit denen Produktinformationen für einen einzelnen Anbieter zum interagieren mit Windows-Security Center erhalten werden.<br/> |
| [**Iwscproductlist**](/windows/desktop/api/iwscapi/nn-iwscapi-iwscproductlist)<br/> | Stellt Methoden zum Sammeln von Produktinformationen für den ausgewählten Typ von Anbietern bereit, die auf dem Computer installiert sind.<br/>         |
