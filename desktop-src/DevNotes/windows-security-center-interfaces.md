# <a name="windows-security-center-interfaces"></a>Windows-Sicherheit Mittlere Schnittstellen

Die folgenden Schnittstellen sind für die Verwendung mit Windows-Sicherheit Center verfügbar.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                 | Beschreibung                                                                                                                      |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**IWscProduct**](/windows/desktop/api/Iwscapi/nn-iwscapi-iwscproduct)<br/>         | Stellt Methoden zum Abrufen von Produktinformationen für einen einzelnen Anbieter bereit, um mit Windows-Sicherheit Center zu interagieren.<br/> |
| [**IWSCProductList**](/windows/desktop/api/iwscapi/nn-iwscapi-iwscproductlist)<br/> | Stellt Methoden zum Sammeln von Produktinformationen für den ausgewählten Typ der auf dem Computer installierten Anbieter bereit.<br/>         |
