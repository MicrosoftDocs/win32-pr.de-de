# <a name="_getwindowtext-function"></a>\_GetWindowText-Funktion

\[Diese Funktion ist ein Wrapper für die **GetWindowText-Funktion.** Diese Funktion kann in Zukunft geändert oder nicht mehr verfügbar sein. Anwendungen sollten **GetWindowText** direkt aufrufen.\]

Ruft den Text aus der Titelleiste des angegebenen Fensters ab. Siehe [**GetWindowText**](/windows/win32/api/winuser/nf-winuser-getwindowtexta).
