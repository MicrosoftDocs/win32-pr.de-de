# <a name="_loadcursor-function"></a>\_LoadCursor-Funktion

\[Diese Funktion ist ein Wrapper für die **LoadCursor-Funktion.** Diese Funktion kann in Zukunft geändert oder nicht mehr verfügbar sein. Anwendungen sollten **LoadCursor** direkt aufrufen.\]
