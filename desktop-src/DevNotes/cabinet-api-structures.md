# <a name="cabinet-api-structures"></a>Cab-API-Strukturen

In diesem Abschnitt werden die folgenden Strukturen der Cabinet-API erläutert:



| Struktur                                              | Beschreibung                                                                                     |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**CABINETDLLVERSIONINFO**](cabinetdllversioninfo.md) | Enthält die Versionsinformationen für Cabinet.dll.<br/>                                    |
| [**CCAB**](/windows/desktop/api/Fci/ns-fci-ccab)                                   | Enthält Informationen zum Schränk.<br/>                                                        |
| [**Erf**](/windows/win32/api/fdi_fci_types/ns-fdi_fci_types-erf)                                     | Gibt Fehlerinformationen von FCI/FDI zurück. Der Aufrufer sollte diese Struktur nicht ändern.<br/> |
| [**FDICABINETINFO**](/windows/desktop/api/Fdi/ns-fdi-fdicabinetinfo)               | Enthält Details zu einer bestimmten Cab-Datei.<br/>                                    |
| [**FDINOTIFICATION**](/windows/desktop/api/Fdi/ns-fdi-fdinotification)             | Geben Sie Informationen für [**FNFDINOTIFY**](/windows/desktop/api/fdi/nf-fdi-fnfdinotify)an.<br/>                           |
| [**Sitzung**](session.md)                             | Enthält Informationen zur aktuellen Sitzung.<br/>                                      |
