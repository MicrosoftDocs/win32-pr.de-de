# <a name="cabinet-api-structures"></a>CAB-API-Strukturen

In diesem Abschnitt werden die folgenden CAB-API-Strukturen ausführlich erläutert:



| Struktur                                              | BESCHREIBUNG                                                                                     |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**Cabinetdllversioninfo**](cabinetdllversioninfo.md) | Enthält die Versionsinformationen für Cabinet.dll.<br/>                                    |
| [**CCAB**](/windows/desktop/api/Fci/ns-fci-ccab)                                   | Enthält CAB-Informationen.<br/>                                                        |
| [**ERF**](/windows/win32/api/fdi_fci_types/ns-fdi_fci_types-erf)                                     | Gibt Fehlerinformationen von FCI/FDI zurück. Der Aufrufer sollte diese Struktur nicht ändern.<br/> |
| [**"F"**](/windows/desktop/api/Fdi/ns-fdi-fdicabinetinfo)               | Enthält Details zu einer bestimmten CAB-Datei.<br/>                                    |
| [**Abgleichen von Informationen**](/windows/desktop/api/Fdi/ns-fdi-fdinotification)             | Stellen Sie die Informationen für [**fnfdinotify**](/windows/desktop/api/fdi/nf-fdi-fnfdinotify)bereit.<br/>                           |
| [**Sitzung**](session.md)                             | Enthält Informationen über die aktuelle Sitzung.<br/>                                      |
