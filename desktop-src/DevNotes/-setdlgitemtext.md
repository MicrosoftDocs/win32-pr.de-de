# <a name="_setdlgitemtext-function"></a>\_SetDlgItemText-Funktion

\[Diese Funktion ist ein Wrapper über die Funktion **SetDlgItemText** . Diese Funktion kann in Zukunft geändert oder nicht mehr verfügbar sein. Anwendungen sollten **SetDlgItemText** direkt aufrufen.\]

Legt den Titel oder Text eines-Steuer Elements in einem Dialogfeld fest. Weitere Informationen finden Sie unter [**SetDlgItemText**](/windows/win32/api/winuser/nf-winuser-setdlgitemtexta).
