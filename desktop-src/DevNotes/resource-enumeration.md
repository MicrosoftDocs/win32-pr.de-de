# <a name="resource-enumeration"></a>Ressourcenenumeration

Entwickler können die ressourcenenumeration als Teil der Instrumentierung verwenden, die bestimmte Arten von Informationen zu Ihren Anwendungen bereitstellt.

Microsoft baut Instrumentation in das Windows-Betriebssystem auf, aber diese Features sind so eng an das Betriebssystem gebunden, dass Sie für Anwendungsentwickler nur eingeschränkt eingesetzt werden können. Microsoft stellt eine Ressourcen Aufzählungs-API (Application Programming Interface) zur Verfügung, die mit den folgenden Instrumentierungs Ressourcen verwendet werden kann: Heap Zuweisungen und Ablauf Verfolgungs Informationen.

In den folgenden Themen wird beschrieben, wie Sie diese API in Ihren Anwendungen verwenden:

-   [Informationen zur ressourcenenumeration](about-resource-enumeration.md)
-   [Ressourcenenumerationsverweis](resource-enumeration-reference.md)
