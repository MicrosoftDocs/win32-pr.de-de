# <a name="resource-enumeration"></a>Ressourcenenumeration

Entwickler können die Ressourcenenumeration als Teil der Instrumentierung verwenden, die bestimmte Arten von Informationen zu ihren Anwendungen bietet.

Microsoft integriert die Instrumentierung in Windows Betriebssystem, aber diese Features sind so eng an das Betriebssystem gebunden, dass sie für Anwendungsentwickler nur eingeschränkt verwendet werden können. Microsoft stellt eine Api (Application Programming Interface) für Ressourcenenumeration zur Verfügung, die mit den folgenden Instrumentierungsressourcen verwendet werden kann: Heapzuordnungen und Verarbeiten von Ablaufverfolgungsinformationen.

In den folgenden Themen wird beschrieben, wie Sie diese API in Ihren Anwendungen verwenden:

-   [Informationen zur Ressourcenenumeration](about-resource-enumeration.md)
-   [Referenz zur Ressourcenenumeration](resource-enumeration-reference.md)
