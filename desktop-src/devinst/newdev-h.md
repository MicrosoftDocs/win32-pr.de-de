# <a name="newdevh"></a>Newdev. h

Dieser Abschnitt enthält Referenz Themen für den newdev. h-Header.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                     | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Diinstalldevice**](/windows/desktop/api/Newdev/nf-newdev-diinstalldevice)<br/>                                     | Die Funktion " **diinstalldevice** " installiert einen angegebenen Treiber, der im [Treiber Speicher](/windows-hardware/drivers/install/driver-store) auf einem angegebenen Gerät vorinstalliert ist, das im System vorhanden ist.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**Diinstalldriver**](/windows/desktop/api/Newdev/nf-newdev-diinstalldrivera)<br/>                                     | Die Funktion " **diinstalldriver** " führt eine Vorinstallation eines Treibers im [Treiber Speicher](/windows-hardware/drivers/install/driver-store) durch und installiert dann den Treiber auf Geräten, die in dem vom Treiber unterstützten System vorhanden sind.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**Dirollbackdriver**](/windows/desktop/api/Newdev/nf-newdev-dirollbackdriver)<br/>                                   | Die Funktion **dirollbackdriver** führt einen Rollback für den Treiber aus, der auf einem angegebenen Gerät installiert ist.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**Dishowupdatedevice**](/windows/desktop/api/Newdev/nf-newdev-dishowupdatedevice)<br/>                               | Die Funktion " **dishowupdatedevice** " zeigt den Hardware Update-Assistenten für ein bestimmtes Gerät an.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**Diuninstalldevice**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice)<br/>                                 | Mit der Funktion " [**diuninstalldevice**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice) " wird ein Gerät deinstalliert, und der zugehörige Geräteknoten ([devnode](/windows-hardware/drivers/)) wird aus dem System entfernt. Dies unterscheidet sich von der Verwendung von [**setupdicallclassinstaller**](/windows/desktop/api/Setupapi/nf-setupapi-setupdicallclassinstaller) mit dem [**DIF- \_ Entfernungs**](/windows-hardware/drivers/install/dif-remove) Code, da versucht wird, den Geräteknoten zusätzlich zu den untergeordneten Devnodes zu deinstallieren, die zum Zeitpunkt des Aufrufes vorhanden sind.<br/> Vor Windows 8 werden alle untergeordneten Geräte, die zum Zeitpunkt des Aufrufes nicht vorhanden sind, nicht deinstalliert. Ab Windows 8 werden alle untergeordneten Geräte, die zum Zeitpunkt des Aufrufes nicht vorhanden sind, jedoch deinstalliert.<br/> |
| [**UpdateDriverForPlugAndPlayDevices**](/windows/desktop/api/Newdev/nf-newdev-updatedriverforplugandplaydevicesa)<br/> | Wenn eine INF-Datei und eine [Hardware-ID](/windows-hardware/drivers/install/hardware-ids)vorliegen, werden von der Funktion **UpdateDriverForPlugAndPlayDevices** aktualisierte Treiber für Geräte installiert, die der Hardware-ID entsprechen. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
