# <a name="newdevh"></a>Newdev.h

Dieser Abschnitt enthält Referenzthemen für den Newdev.h-Header.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                     | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DiInstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diinstalldevice)<br/>                                     | Die **DiInstallDevice-Funktion** installiert einen angegebenen Treiber, [](/windows-hardware/drivers/install/driver-store) der im Treiberspeicher vorinstalliert ist, auf einem angegebenen Gerät, das im System vorhanden ist.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**DiInstallDriver**](/windows/desktop/api/Newdev/nf-newdev-diinstalldrivera)<br/>                                     | Mit **der DiInstallDriver-Funktion** wird [](/windows-hardware/drivers/install/driver-store) ein Treiber im Treiberspeicher vorinstalliert und dann auf Geräten installiert, die im vom Treiber unterstützten System vorhanden sind.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**DiRollbackDriver**](/windows/desktop/api/Newdev/nf-newdev-dirollbackdriver)<br/>                                   | Die **DiRollbackDriver-Funktion** führt ein Rollback des Treibers aus, der auf einem angegebenen Gerät installiert ist.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**DiShowUpdateDevice**](/windows/desktop/api/Newdev/nf-newdev-dishowupdatedevice)<br/>                               | Die **DiShowUpdateDevice-Funktion** zeigt den Hardwareupdate-Assistenten für ein angegebenes Gerät an.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**DiUninstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice)<br/>                                 | Die [**DiUninstallDevice-Funktion**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice) deinstalliert ein Gerät und entfernt seinen Geräteknoten ([devnode](/windows-hardware/drivers/)) aus dem System. Dies unterscheidet sich von der Verwendung von [**SetupDiCallClassInstaller**](/windows/desktop/api/Setupapi/nf-setupapi-setupdicallclassinstaller) mit dem [**DIF \_ REMOVE-Code,**](/windows-hardware/drivers/install/dif-remove) da zusätzlich zu den untergeordneten Devnodes, die zum Zeitpunkt des Aufrufs vorhanden sind, versucht wird, den Geräteknoten zu deinstallieren.<br/> Vor Windows 8 untergeordnete Geräte, die zum Zeitpunkt des Aufrufs nicht vorhanden sind, nicht deinstalliert. Ab Windows 8 werden jedoch alle untergeordneten Geräte deinstalliert, die zum Zeitpunkt des Aufrufs nicht vorhanden sind.<br/> |
| [**UpdateDriverForPlugAndPlayDevices**](/windows/desktop/api/Newdev/nf-newdev-updatedriverforplugandplaydevicesa)<br/> | Bei einer INF-Datei und einer [Hardware-ID](/windows-hardware/drivers/install/hardware-ids)installiert die **UpdateDriverForPlugAndPlayDevices-Funktion** aktualisierte Treiber für Geräte, die mit der Hardware-ID übereinstimmen. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
