# <a name="programming-guide-xinput-game-controller-apis"></a>Programmier Handbuch (xinput-Spiel Controller-APIs)

Diese Anleitung enthält Informationen zur Verwendung der xinput-API für die Interaktion mit dem Xbox-Controller, wenn eine Verbindung mit einem Windows-PC besteht.

- " [Getting Started with xinput](getting-started-with-xinput.md) ": Dieser Abschnitt enthält eine Beschreibung von XInput und beschreibt die ersten Schritte bei der Verwendung von xinput in Ihrer Anwendung.
- [Xinput-Versionen](xinput-versions.md) : Dieser Abschnitt enthält Informationen zu den Versionen von xinput.
- [XInput und DirectInput](xinput-and-directinput.md) : Dieser Abschnitt enthält Informationen zu den Unterschieden zwischen XInput und DirectInput und dazu, wie Sie beide nebeneinander verwenden können.
- [DirectInput-und xusb-Geräte](directinput-and-xusb-devices.md) : enthält Informationen zu den Geräte Zuordnungen der Xbox Common Controller Class (xusb).
- [XInput-und Controller-Untertypen](xinput-and-controller-subtypes.md) : Listet die in xinput verfügbaren Controller Untertypen auf.

Weitere Informationen zu bestimmten API-Methoden finden Sie in der [Programmier Referenz](programming-reference.md).
