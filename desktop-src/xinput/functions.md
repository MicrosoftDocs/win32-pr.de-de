# <a name="functions-programming-reference"></a>Funktionen (Programmier Referenz)

Verfügbare xinput-Funktionen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema                                                                                 | BESCHREIBUNG                                                                                                                                             |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Xinputenable**](/windows/desktop/api/XInput/nf-xinput-xinputenable)<br/>                                       | Legt den Berichterstattungs Status von xinput fest.<br/>                                                                                                          |
| [**Xinputgetaudiodeaktiviert**](/windows/desktop/api/XInput/nf-xinput-xinputgetaudiodeviceids)<br/>                 | Ruft das Sound Rendering und audioerfassungs-Audiogeräte-IDs ab, die dem mit dem angegebenen Controller verbundenen Headset zugeordnet sind.<br/> |
| [**Xinputgetbatteryinformation**](/windows/desktop/api/XInput/nf-xinput-xinputgetbatteryinformation)<br/>         | Ruft den Akku-und den Ladestatus eines drahtlosen Controllers ab.<br/>                                                                       |
| [**Xinputgetfunktionalitäten**](/windows/desktop/api/XInput/nf-xinput-xinputgetcapabilities)<br/>                     | Ruft die Funktionen und Features eines verbundenen Controllers ab.<br/>                                                                           |
| [**Xinputgetdsoundaudiodeviceguids**](/windows/desktop/api/XInput/nf-xinput-xinputgetdsoundaudiodeviceguids)<br/> | Ruft die GUIDs für Sound Rendering und audioerfassungs Geräte ab, die dem mit dem angegebenen Controller verbundenen Headset zugeordnet sind.<br/>          |
| [**Xinputgetkeystroke**](/windows/desktop/api/XInput/nf-xinput-xinputgetkeystroke)<br/>                           | Ruft ein Gamepad-Eingabe Ereignis ab.<br/>                                                                                                             |
| [**Xinputgetstate**](/windows/desktop/api/XInput/nf-xinput-xinputgetstate)<br/>                                   | Ruft den aktuellen Zustand des angegebenen Controllers ab.<br/>                                                                                     |
| [**Xinputsetstate**](/windows/desktop/api/XInput/nf-xinput-xinputsetstate)<br/>                                   | Sendet Daten an einen verbundenen Controller. Diese Funktion wird verwendet, um die Vibrationsfunktion eines Controllers zu aktivieren.<br/>                              |
