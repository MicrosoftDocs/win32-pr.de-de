# <a name="functions-programming-reference"></a>Funktionen (Programmierreferenz)

Verfügbare XInput-Funktionen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema                                                                                 | BESCHREIBUNG                                                                                                                                             |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XInputEnable**](/windows/desktop/api/XInput/nf-xinput-xinputenable)<br/>                                       | Legt den Berichtszustand von XInput fest.<br/>                                                                                                          |
| [**XInputGetAudioDeviceIds**](/windows/desktop/api/XInput/nf-xinput-xinputgetaudiodeviceids)<br/>                 | Ruft die Audiogeräte-IDs für Soundrendering und Audioaufnahme ab, die dem Headset zugeordnet sind, das mit dem angegebenen Controller verbunden ist.<br/> |
| [**XInputGetBatteryInformation**](/windows/desktop/api/XInput/nf-xinput-xinputgetbatteryinformation)<br/>         | Ruft den Akkutyp und den Ladestatus eines Funkcontrollers ab.<br/>                                                                       |
| [**XInputGetCapabilities**](/windows/desktop/api/XInput/nf-xinput-xinputgetcapabilities)<br/>                     | Ruft die Funktionen und Features eines verbundenen Controllers ab.<br/>                                                                           |
| [**XInputGetDSoundAudioDeviceGuids**](/windows/desktop/api/XInput/nf-xinput-xinputgetdsoundaudiodeviceguids)<br/> | Ruft die Geräte-GUIDs für sound rendering und sound capture ab, die dem Headset zugeordnet sind, das mit dem angegebenen Controller verbunden ist.<br/>          |
| [**XInputGetKeystroke**](/windows/desktop/api/XInput/nf-xinput-xinputgetkeystroke)<br/>                           | Ruft ein Gamepad-Eingabeereignis ab.<br/>                                                                                                             |
| [**XInputGetState**](/windows/desktop/api/XInput/nf-xinput-xinputgetstate)<br/>                                   | Ruft den aktuellen Zustand des angegebenen Controllers ab.<br/>                                                                                     |
| [**XInputSetState**](/windows/desktop/api/XInput/nf-xinput-xinputsetstate)<br/>                                   | Sendet Daten an einen verbundenen Controller. Diese Funktion wird verwendet, um die Schwingungsfunktion eines Controllers zu aktivieren.<br/>                              |
