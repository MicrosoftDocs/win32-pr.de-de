# <a name="xinput-game-controller-apis"></a>Xinput-Spiel Controller-APIs

## <a name="purpose"></a>Zweck

Die xinput Game Controller-API ermöglicht Anwendungen das Empfangen von Eingaben vom Xbox-Controller für Windows.

## <a name="developer-audience"></a>Entwicklergruppe

Xinput-Spiele Controller-APIs sind für die Verwendung durch Entwickler vorgesehen, die den Xbox-Controller für Ihre Windows-Anwendungen verwenden möchten.
