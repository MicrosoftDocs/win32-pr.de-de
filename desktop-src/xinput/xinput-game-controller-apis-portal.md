# <a name="xinput-game-controller-apis"></a>XInput Game Controller-APIs

## <a name="purpose"></a>Zweck

Mit der XInput Game Controller-API können Anwendungen Eingaben vom Xbox Controller für Windows empfangen.

XInput Game Controller-APIs sind für Entwickler konzipiert, die den Xbox Controller für ihre Windows-Anwendungen verwenden möchten.
