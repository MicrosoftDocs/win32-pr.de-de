# <a name="hyper-v-wmi-classes"></a>Hyper-V-WMI-Klassen

Im Folgenden sind die Hyper-V-WMI-Klassen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                 | Beschreibung                                                                                                                                                                                                                                                                                 |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [CIM-Klassen](cim-classes.md)<br/>                                             | Die CIM-Klassen sind Basisklassen, von denen alle WMI-Objekte abgeleitet werden.<br/>                                                                                                                                                                                                               |
| [BIOS-Klassen](bios-classes.md)<br/>                                           | Das virtuelle BIOS ist ein Softwareimage, das in den RAM geladen wird, um einige der grundlegenden Aspekte eines Computersystems zu konfigurieren und zu starten. Es gibt ein BIOS-Element pro Computersystem, und dieses Element kann nicht ersetzt oder entfernt werden.<br/>                                                 |
| [Eingabeklassen](input-classes.md)<br/>                                         | Die Benutzereingabegeräte werden durch diese Klassen dargestellt. Einem virtuellen Computer ist immer eine Instanz jeder Klasse zugeordnet.<br/>                                                                                                                                             |
| [Integration Services-Klassen](integration-services-classes.md)<br/>           | Integrationsdienste sind Softwaredienste, die auf dem Gastbetriebssystem auf einem untergeordneten virtuellen Computer und als Teil des Virtualisierungsstapels im Verwaltungsbetriebssystem ausgeführt werden, um ein gewisses Maß an Integration in das Verwaltungsbetriebssystem bereitzustellen.<br/> |
| [Speicherklassen](memory-classes.md)<br/>                                       | Die Speicherklassen modellieren den Arbeitsspeicher für einen virtuellen Computer.<br/>                                                                                                                                                                                                                       |
| [Prozessorklassen](processor-classes.md)<br/>                                 | Die Prozessorklassen modellieren einen Prozessor für einen virtuellen Computer.<br/>                                                                                                                                                                                                                   |
| [Profilregistrierungsklassen](profile-registration-classes.md)<br/>           | Virtualisierungs-WMI-Klassen im Zusammenhang mit der Profilregistrierung.<br/>                                                                                                                                                                                                                      |
| [Remotedesktop Klassen](remotefx-classes.md)<br/>                             | Die Remotedesktop-Virtualisierungs-WMI-Klassen werden verwendet, um Remotedesktop und RemoteFX Ressourcen und virtuelle Computer zu verwalten.<br/>                                                                                                                                                     |
| [Ressourcenverwaltungsklassen](resource-management-classes.md)<br/>             | Virtualisierungs-WMI-Klassen im Zusammenhang mit der Ressourcenverwaltung.<br/>                                                                                                                                                                                                                       |
| [Klassen für serielle Geräte](serial-devices-classes.md)<br/>                       | Die seriellen Geräte in einem virtuellen Computer bestehen aus seriellen Controllern und seriellen Anschlüssen. Jeder virtuelle Computer verfügt über genau einen seriellen Controller, und jeder serielle Controller verfügt über genau zwei serielle Anschlüsse.<br/>                                                                         |
| [Speicherklassen](storage-classes.md)<br/>                                     | Die Speicherobjekte sind in drei Typen unterteilt: Controller, Laufwerke und Medien.<br/>                                                                                                                                                                                              |
| [Videoklassen](video-classes.md)<br/>                                         | Alle virtuellen Computer spiegeln das Vorhandensein eines emulierten S3-Videocontrollers und eines beschleunigten synthetischen Videocontrollers wider.<br/>                                                                                                                                                     |
| [Virtuelle Systemklassen](virtual-system-classes.md)<br/>                       | Virtualisierungs-WMI-Klassen im Zusammenhang mit dem virtuellen Computer.<br/>                                                                                                                                                                                                                       |
| [Verwaltungsklassen für virtuelle Systeme](virtual-system-management-classes.md)<br/> | Virtualisierungs-WMI-Klassen im Zusammenhang mit der Verwaltung virtueller Computer.<br/>                                                                                                                                                                                                                |
