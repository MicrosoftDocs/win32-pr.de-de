# <a name="hyper-v-wmi-classes"></a>Hyper-V-WMI-Klassen

Im folgenden finden Sie die Hyper-V-WMI-Klassen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                 | BESCHREIBUNG                                                                                                                                                                                                                                                                                 |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [CIM-Klassen](cim-classes.md)<br/>                                             | Die CIM-Klassen sind Basisklassen, von denen alle WMI-Objekte abgeleitet werden.<br/>                                                                                                                                                                                                               |
| [BIOS-Klassen](bios-classes.md)<br/>                                           | Das virtuelle BIOS ist ein Software Abbild, das in den RAM geladen wird, um einige der grundlegenden Aspekte von zu konfigurieren und ein Computersystem zu starten. Es gibt ein BIOS-Element pro Computersystem, und dieses Element kann nicht ersetzt oder entfernt werden.<br/>                                                 |
| [Eingabe Klassen](input-classes.md)<br/>                                         | Die Benutzereingabe Geräte werden durch diese Klassen dargestellt. Ein virtueller Computer verfügt immer über eine Instanz jeder zugeordneten Klasse.<br/>                                                                                                                                             |
| [Integration Services-Klassen](integration-services-classes.md)<br/>           | Integration Services sind Software Dienste, die auf dem Gast Betriebssystem innerhalb eines untergeordneten virtuellen Computers ausgeführt werden, und als Teil des Virtualisierungsstapels im Verwaltungs Betriebssystem, um ein gewisses Maß an Integration mit dem Verwaltungs Betriebssystem zu ermöglichen.<br/> |
| [Speicher Klassen](memory-classes.md)<br/>                                       | Die Speicher Klassen modellieren den Arbeitsspeicher für eine virtuelle Maschine.<br/>                                                                                                                                                                                                                       |
| [Prozessor Klassen](processor-classes.md)<br/>                                 | Die Prozessor Klassen modellieren einen Prozessor für eine virtuelle Maschine.<br/>                                                                                                                                                                                                                   |
| [Profil Registrierungs Klassen](profile-registration-classes.md)<br/>           | WMI-virtualisierungsklassen im Zusammenhang mit der Profil Registrierung.<br/>                                                                                                                                                                                                                      |
| [Remotedesktop-Klassen](remotefx-classes.md)<br/>                             | Die WMI-Klassen für die Remotedesktop-Virtualisierung werden verwendet, um Remotedesktop-und remotefx-Ressourcen und virtuelle Computer zu verwalten.<br/>                                                                                                                                                     |
| [Ressourcenverwaltungsklassen](resource-management-classes.md)<br/>             | WMI-virtualisierungsklassen im Zusammenhang mit der Ressourcenverwaltung.<br/>                                                                                                                                                                                                                       |
| [Serielle Geräteklassen](serial-devices-classes.md)<br/>                       | Die seriellen Geräte auf einem virtuellen Computer bestehen aus seriellen Controllern und seriellen Anschlüssen. Jeder virtuelle Computer verfügt über genau einen seriellen Controller, und jeder serielle Controller verfügt über genau zwei serielle Anschlüsse.<br/>                                                                         |
| [Speicherklassen](storage-classes.md)<br/>                                     | Die Speicher Objekte sind in drei Typen unterteilt: Controller, Laufwerke und Medien.<br/>                                                                                                                                                                                              |
| [Video Klassen](video-classes.md)<br/>                                         | Alle virtuellen Computer spiegeln das vorhanden sein eines emulierten S3-Video Controllers und eines beschleunigten, synthetischen Video Controllers wider.<br/>                                                                                                                                                     |
| [Klassen des virtuellen Systems](virtual-system-classes.md)<br/>                       | WMI-virtualisierungsklassen, die mit dem virtuellen Computer verknüpft sind.<br/>                                                                                                                                                                                                                       |
| [Verwaltungs Klassen für virtuelle Systeme](virtual-system-management-classes.md)<br/> | WMI-virtualisierungsklassen im Zusammenhang mit der Verwaltung virtueller Computer.<br/>                                                                                                                                                                                                                |
