# <a name="hyper-v-application-health-monitoring-interfaces"></a>Hyper-V-Anwendungsüberwachungsschnittstellen

Die Hyper-V-Anwendungsüberwachungs-API definiert die folgenden COM-Schnittstellen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                         | Beschreibung                                                                                                                                                            |
|-------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IVmApplicationHealthMonitor**](ivmapplicationhealthmonitor.md)<br/> | Meldet den Integritätsstatus einer Anwendung, die auf einem virtuellen Computer ausgeführt wird, an die Hyper-V-Integrationskomponenten, die auf demselben virtuellen Computer ausgeführt werden.<br/> |
