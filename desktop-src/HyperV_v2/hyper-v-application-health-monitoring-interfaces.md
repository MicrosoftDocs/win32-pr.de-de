## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                         | BESCHREIBUNG                                                                                                                                                            |
|-------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Ivmapplicationhealthmonitor**](ivmapplicationhealthmonitor.md)<br/> | Meldet den Integritäts Status einer Anwendung, die auf einem virtuellen Computer ausgeführt wird, an die Hyper-V-Integrations Komponenten, die auf demselben virtuellen Computer ausgeführt werden.<br/> |
