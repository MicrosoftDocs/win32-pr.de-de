# <a name="processor-classes"></a>Prozessorklassen

Die Prozessorklassen modellieren einen Prozessor für einen virtuellen Computer.

Im Folgenden finden Sie Virtualisierungs-WMI-Klassen, die mit dem Prozessor verknüpft sind.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                      | Beschreibung                                                                               |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**Msvm-Prozessor \_**](msvm-processor.md)<br/>                       | Stellt den virtuellen Prozessor in einem virtuellen Computer dar.<br/>                         |
| [**Msvm \_ ProcessorPool**](msvm-processorpool.md)<br/>               | Aggregiert die Prozessorressourcen, die einem virtuellen Computer zugeordnet werden können.<br/> |
| [**Msvm \_ ProcessorSettingData**](msvm-processorsettingdata.md)<br/> | Stellt die Einstellungen des virtuellen Prozessors für einen virtuellen Computer dar.<br/>               |
