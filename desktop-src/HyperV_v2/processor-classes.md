# <a name="processor-classes"></a>Prozessor Klassen

Die Prozessor Klassen modellieren einen Prozessor für eine virtuelle Maschine.

Im folgenden finden Sie virtualisierungswmi-Klassen, die mit dem Prozessor verknüpft sind.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                      | BESCHREIBUNG                                                                               |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**MSVM- \_ Prozessor**](msvm-processor.md)<br/>                       | Stellt den virtuellen Prozessor in einem virtuellen Computer dar.<br/>                         |
| [**MSVM \_ processorpool**](msvm-processorpool.md)<br/>               | Aggregiert die Prozessorressourcen, die einem virtuellen Computer zugeordnet werden können.<br/> |
| [**MSVM \_ processorsettingdata**](msvm-processorsettingdata.md)<br/> | Stellt die Einstellungen des virtuellen Prozessors für eine virtuelle Maschine dar.<br/>               |
