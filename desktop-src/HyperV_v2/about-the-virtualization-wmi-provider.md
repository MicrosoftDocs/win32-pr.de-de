# <a name="about-the-hyper-v-wmi-provider"></a>Informationen zum Hyper-V-WMI-Anbieter

Mit dem WMI-Anbieter für Hyper-V können Entwickler und Skripter schnell benutzerdefinierte Tools, Hilfsprogramme und Erweiterungen für die Virtualisierungsplattform erstellen. Die WMI-Schnittstellen können alle Aspekte der Hyper-V-Dienste verwalten.

## <a name="about-microsoft-windows-server-2008-r2-hyper-v"></a>Informationen zu Microsoft Windows Server 2008 R2 Hyper-V

Mit Microsoft Windows Server 2008 R2 Hyper-V können Systemadministratoren separate Hardwareserver auf einem einzelnen Server konsolidieren, auf dem Microsoft Windows Server 2008 R2 als Hostbetriebssystem ausgeführt wird.

Jeder der gehosteten virtuellen Computer wird in einer eigenen separaten und isolierten virtuellen Umgebung ausgeführt. Dies ermöglicht es dem Administrator, eine große Anzahl von virtuellen Computern einfach zu verwalten, bereitzustellen und zu warten, während gleichzeitig die Notwendigkeit reduziert wird, mehrere Hardwarelösungen auszuführen, um verschiedene Produkte und Betriebssysteme zu verwenden.

Ein weiterer Vorteil der Umgebung des virtuellen Computers ist die Test- und Supportumgebung. Neue Server- und Systemkonfigurationen können parallel zum vorhandenen System bereitgestellt und getestet werden, ohne dass während der Rolloutphase kostspielige Ausfallzeiten erforderlich sind. Jeder virtuelle Computer kann so eingerichtet werden, dass er unterschiedliche Konfigurationen verwendet, während er auf einer Standardplattform ausgeführt wird, um bessere Testergebnisse zu erzielen. Mit Unterstützung für frühere Betriebssysteme kann ältere Hardware, Systeme und Anwendungen unterstützt und getestet werden. Mithilfe der Momentaufnahmeunterstützung können Sie eine Momentaufnahme des Zustands eines virtuellen Computers erstellen, damit Sie bei Bedarf zu einem früheren Ereignis zurückkehren können.

Hyper-V stellt eine umfassende Schnittstelle zur Verfügung, die es dem Benutzer ermöglicht, die Umgebung des virtuellen Computers zu überwachen und zu steuern. Der Großteil von Hyper-V kann mithilfe des WMI-Anbieters gesteuert werden, der eine einfache, aber leistungsstarke Anpassung der virtuellen Computer ermöglicht.
