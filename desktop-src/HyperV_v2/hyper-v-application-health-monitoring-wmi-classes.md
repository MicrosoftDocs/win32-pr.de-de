# <a name="hyper-v-application-health-monitoring-wmi-classes"></a>WMI-Klassen für die Hyper-V-Anwendungs Integritäts Überwachung

Die Hyper-V-anwendungsintegritäts-Überwachungs-API definiert die folgenden WMI-Klassen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                        | BESCHREIBUNG                                                                                                                                                                 |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ heartbeatcomponent**](msvm-heartbeatcomponent.md)<br/>                       | Stellt den Status des Takt diensdienstankts dar, der für die Überwachung des Zustands einer virtuellen Maschine zuständig ist, indem ein Takt in regelmäßigen Abständen gemeldet wird.<br/> |
| [**MSVM \_ heartbeatcomponentsettingdata**](msvm-heartbeatcomponentsettingdata.md)<br/> | Stellt den konfigurierten Status des Takt dienstankts dar.<br/>                                                                                                        |
