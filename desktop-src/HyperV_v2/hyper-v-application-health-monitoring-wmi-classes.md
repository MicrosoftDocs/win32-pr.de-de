# <a name="hyper-v-application-health-monitoring-wmi-classes"></a>WMI-Klassen für die Hyper-V-Anwendungsüberwachung

Die Hyper-V-Anwendungsüberwachungs-API definiert die folgenden WMI-Klassen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                        | Beschreibung                                                                                                                                                                 |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ HeartbeatComponent**](msvm-heartbeatcomponent.md)<br/>                       | Stellt den Zustand des Heartbeatdiensts dar, der für die Überwachung des Zustands eines virtuellen Computers verantwortlich ist, indem in regelmäßigen Abständen ein Heartbeat gemeldet wird.<br/> |
| [**Msvm \_ HeartbeatComponentSettingData**](msvm-heartbeatcomponentsettingdata.md)<br/> | Stellt den konfigurierten Zustand des Taktdiensts dar.<br/>                                                                                                        |
