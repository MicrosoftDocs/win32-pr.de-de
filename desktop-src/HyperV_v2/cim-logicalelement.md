# <a name="cim_logicalelement-class-hyper-v-management"></a>CIM_LogicalElement-Klasse (Hyper-V-Verwaltung)

**CIM \_ LogicalElement** ist eine Basisklasse für alle Komponenten eines Systems, die abstrakte System Komponenten, z. b. Dateien, Prozesse und logische Geräte, darstellen.

## <a name="syntax"></a>Syntax

``` syntax
[Abstract, Version("2.6.0"), UMLPackagePath("CIM::Core::CoreElements"), AMENDMENT]
class CIM_LogicalElement : CIM_ManagedSystemElement
{
};
```

## <a name="members"></a>Member

Die **CIM \_ LogicalElement** -Klasse definiert keine Member.
