# <a name="msvm_diskettedrive-class"></a>MSVM \_ diskettedrive-Klasse

Stellt ein Diskettenlaufwerk innerhalb des virtuellen Computers dar. Ein Diskettenlaufwerk kann mit einer Datei aufgefüllt werden, die Disketten Medien darstellt, oder das Laufwerk kann leer sein. Physische Medien werden nicht unterstützt. Es ist genau ein Diskettenlaufwerk pro Disketten Controller vorhanden, und es kann nicht entfernt werden.

Die folgende Syntax wird Managed Object Format (MOF)-Code vereinfacht und enthält alle geerbten Eigenschaften.

## <a name="syntax"></a>Syntax

``` syntax

  uint16   OperationalStatus[] = { 2 };

  uint16   RequestedState = 12;

  uint16   TransitioningToState;

  string   DeviceID = "Microsoft:GUID\device-specific-data";
  boolean  PowerManagementSupported;

  uint16   Availability = 6;
```
