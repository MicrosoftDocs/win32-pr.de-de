# <a name="resource-management-classes"></a>Ressourcenverwaltungsklassen

Im Folgenden finden Sie Virtualisierungs-WMI-Klassen im Zusammenhang mit der Ressourcenverwaltung.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                                        | BESCHREIBUNG                                                                                                                                                      |
|--------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ AbstractResourcePoolSettingData**](msvm-abstractresourcepoolsettingdata.md)<br/>             | Stellt die Einstellungen einer [**Msvm \_ ResourcePool-Instanz**](msvm-resourcepool.md) dar, die nicht zuordnungsbezogen sind.<br/>                                |
| [**Msvm \_ AllocationCapabilities**](msvm-allocationcapabilities.md)<br/>                               | Definiert die Mittel, mit denen ein Client den gültigen Bereich von Standardeinstellungen für eine virtuelle Ressource finden kann.<br/>                                          |
| [**\_Msvm-ElementAllocatedFromPool**](msvm-elementallocatedfrompool.md)<br/>                           | Ordnet eine Instanz einer zugeordneten Ressource dem Ressourcenpool zu, aus dem sie zugeordnet wurde.<br/>                                                   |
| [**Msvm \_ ElementCapabilities**](msvm-elementcapabilities.md)<br/>                                     | Stellt die Zuordnung zwischen verwalteten Elementen und ihren Funktionen dar.<br/>                                                                           |
| [**Msvm \_ FeatureSettingData**](msvm-featuresettingdata.md)<br/>                                       | Eine abstrakte Klasse, die Einstellungen für ein bestimmtes Feature eines Systems oder einer Komponente darstellt.<br/>                                                           |
| [**Msvm \_ FeatureSettingsDefineCapabilities**](msvm-featuresettingsdefinecapabilities.md)<br/>         | Stellt eine Verknüpfung zwischen der Funktionsinstanz des Ethernet-Switches und den minimalen, maximalen, inkrementellen und Standardeinstellungen für eine Ressource bereit.<br/> |
| [**Msvm \_ ResourceAllocationFromPool**](msvm-resourceallocationfrompool.md)<br/>                       | Ordnet eine Instanz einer Ressourcenzuordnung dem Ressourcenpool zu, aus dem sie zugeordnet wird.<br/>                                                    |
| [**Msvm \_ ResourceAllocationSettingData**](msvm-resourceallocationsettingdata.md)<br/>                 | Stellt den aktuellen und aufgezeichneten Zuordnungszustände einer virtuellen Ressource dar.<br/>                                                                          |
| [**Msvm \_ ResourcePool**](msvm-resourcepool.md)<br/>                                                   | Beschreibt einen Typ der virtuellen Ressource, die für die Verwendung auf virtuellen Computern verfügbar ist.<br/>                                                                           |
| [**Msvm \_ ResourcePoolConfigurationCapabilities**](msvm-resourcepoolconfigurationcapabilities.md)<br/> | Beschreibt die Funktionen der zugeordneten [**Msvm \_ ResourcePoolConfigurationService-Klasse.**](msvm-resourcepoolconfigurationservice.md)<br/>           |
| [**Msvm \_ ResourcePoolConfigurationService**](msvm-resourcepoolconfigurationservice.md)<br/>           | Stellt die aktive Verwaltung von Ressourcenpools zur Verfügung.<br/>                                                                                                     |
| [**Msvm \_ ResourcePoolSettingData**](msvm-resourcepoolsettingdata.md)<br/>                             | Stellt die Einstellungen einer [**Msvm \_ ResourcePool-Instanz**](msvm-resourcepool.md) dar, die nicht zuordnungsbezogen sind.<br/>                                |
| [**\_Msvm-EinstellungenDefineCapabilities**](msvm-settingsdefinecapabilities.md)<br/>                       | Stellt eine Verknüpfung zwischen der Capabilities-Instanz und den minimalen, maximalen, inkrementellen und Standardeinstellungen für eine Ressource bereit.<br/>                         |
