# <a name="resource-management-classes"></a>Ressourcenverwaltungsklassen

Im folgenden finden Sie WMI-Klassen, die sich auf die Ressourcenverwaltung beziehen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                                        | BESCHREIBUNG                                                                                                                                                      |
|--------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ abstractresourcepoolsettingdata**](msvm-abstractresourcepoolsettingdata.md)<br/>             | Stellt die Einstellungen einer [**MSVM- \_ resourcepool**](msvm-resourcepool.md) -Instanz dar, die nicht zugeordnet sind.<br/>                                |
| [**MSVM- \_ Zuweisung**](msvm-allocationcapabilities.md)<br/>                               | Definiert die Möglichkeiten, mit denen ein Client den gültigen Bereich der Standardeinstellungen für eine virtuelle Ressource ermitteln kann.<br/>                                          |
| [**MSVM \_ Element Zuordnung**](msvm-elementallocatedfrompool.md)<br/>                           | Ordnet dem Ressourcenpool, aus dem Sie zugewiesen wurde, eine Instanz einer zugeordneten Ressource zu.<br/>                                                   |
| [**MSVM- \_ Element Funktionen**](msvm-elementcapabilities.md)<br/>                                     | Stellt die Zuordnung zwischen verwalteten Elementen und ihren Funktionen dar.<br/>                                                                           |
| [**MSVM \_ featuresettingdata**](msvm-featuresettingdata.md)<br/>                                       | Eine abstrakte Klasse, die Einstellungen für eine bestimmte Funktion eines Systems oder einer Komponente darstellt.<br/>                                                           |
| [**MSVM \_ featuresettingsdefinecapabilitäten**](msvm-featuresettingsdefinecapabilities.md)<br/>         | Stellt eine Verknüpfung zwischen der Instanz der Ethernet-Switch-Funktions Funktionen und den minimalen, maximalen, inkrementellen und Standardeinstellungen für eine Ressource bereit.<br/> |
| [**MSVM \_ resourcezucationfrompool**](msvm-resourceallocationfrompool.md)<br/>                       | Ordnet dem Ressourcenpool, aus dem Sie zugeordnet ist, eine Instanz einer Ressourcenzuweisung zu.<br/>                                                    |
| [**MSVM \_ resourcezusettcationsettingdata**](msvm-resourceallocationsettingdata.md)<br/>                 | Stellt die aktuellen und aufgezeichneten Zuordnungs Zustände einer virtuellen Ressource dar.<br/>                                                                          |
| [**MSVM- \_ resourcepool**](msvm-resourcepool.md)<br/>                                                   | Beschreibt einen virtuellen Ressourcentyp, der für die Verwendung auf virtuellen Computern verfügbar ist.<br/>                                                                           |
| [**MSVM \_ resourcepoolconfiguration-Funktionen**](msvm-resourcepoolconfigurationcapabilities.md)<br/> | Beschreibt die Funktionen der zugeordneten [**MSVM \_ resourcepoolconfigurationservice**](msvm-resourcepoolconfigurationservice.md) -Klasse.<br/>           |
| [**MSVM \_ resourcepoolconfigurationservice**](msvm-resourcepoolconfigurationservice.md)<br/>           | Ermöglicht die aktive Verwaltung von Ressourcenpools.<br/>                                                                                                     |
| [**MSVM \_ resourcepoolsettingdata**](msvm-resourcepoolsettingdata.md)<br/>                             | Stellt die Einstellungen einer [**MSVM- \_ resourcepool**](msvm-resourcepool.md) -Instanz dar, die nicht zugeordnet sind.<br/>                                |
| [**MSVM \_ settingsdefinecapabilitäten**](msvm-settingsdefinecapabilities.md)<br/>                       | Stellt eine Verknüpfung zwischen der Funktionen-Instanz und den minimalen, maximalen, inkrementellen und Standardeinstellungen für eine Ressource bereit.<br/>                         |
