# <a name="revertfailover-method-of-the-msvm_replicationservice-class"></a>Revertfailover-Methode der MSVM \_ replicationservice-Klasse

Stellt das aktuelle Failover für einen virtuellen Computer wieder her, indem der aktuelle failoverdatenträger verworfen wird. Nach diesem Vorgang kann [**initiatefailover**](initiatefailover-msvm-replicationservice.md) erneut aufgerufen werden.

## <a name="syntax"></a>Syntax


```mof
uint32 RevertFailover(
  [in]  CIM_ComputerSystem REF ComputerSystem,
  [out] CIM_ConcreteJob    REF Job
);
```



## <a name="parameters"></a>Parameter

<dl> <dt>

*Computersystem* \[ in\]
</dt> <dd>

Ein Verweis auf eine [**CIM \_ Computersystem**](/windows/desktop/CIMWin32Prov/cim-computersystem) -Instanz, die den virtuellen Computer darstellt, für den das Failover wieder hergestellt werden soll.

</dd> <dt>

*Auftrag* \[ vorgenommen\]
</dt> <dd>

Wenn der Vorgang asynchron ausgeführt wird, gibt diese Methode 4096 zurück, und dieser Parameter enthält einen Verweis auf ein Objekt, das von [**CIM \_ concretejob**](/previous-versions//cc136808(v=vs.85))abgeleitet wird.

</dd> </dl>
