# <a name="virtual-system-management-service"></a>Verwaltungsdienst für virtuelle Systeme

Das Profil des virtuellen Systems beschreibt die Objekte, aus denen ein virtueller Computer besteht. Zu diesen Objekten gehören das Basissystem, die Geräte, aus denen das System besteht, die Einstellungen für das System und seine Geräte und der Verwaltungsdienst, der Vorgänge auf dem System ausführt.
