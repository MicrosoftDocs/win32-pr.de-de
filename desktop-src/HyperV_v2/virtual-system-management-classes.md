# <a name="virtual-system-management-classes"></a>Verwaltungs Klassen für virtuelle Systeme

Im folgenden finden Sie WMI-Klassen für die Verwaltung virtueller Computer.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                                                | BESCHREIBUNG                                                                                                                                                                                                                                                          |
|----------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM- \_ affectedjobelements**](msvm-affectedjobelement.md)<br/>                                               | Stellt eine Zuordnung zwischen einem Auftrag und dem verwalteten Element dar, das von seiner Ausführung betroffen sein kann.<br/>                                                                                                                                                    |
| [**MSVM- \_ concretejob**](msvm-concretejob.md)<br/>                                                             | Stellt eine Arbeitseinheit dar und wird verwendet, um den Fortschritt von asynchronen Vorgängen zu verfolgen.<br/>                                                                                                                                                                   |
| [**MSVM \_ diskmergesettingdata**](msvm-diskmergesettingdata.md)<br/>                                           | Stellt den Konfigurations Status der Datenträger zusammenlaufseinstellungen für einen virtuellen Computer dar.<br/>                                                                                                                                                                      |
| [**MSVM \_ Element SettingData**](msvm-elementsettingdata.md)<br/>                                               | Ordnet einen verwalteten-Element seinen Konfigurationsdaten zu.<br/>                                                                                                                                                                                                 |
| [**MSVM- \_ Fehler**](msvm-error.md)<br/>                                                                         | Enthält Informationen über den Schweregrad, die Ursache, Empfohlene Aktionen und andere Daten, die sich auf den Ausfall eines CIM-Vorgangs beziehen.<br/>                                                                                                                            |
| [**MSVM- \_ hubabhängigkeit**](msvm-hosteddependency.md)<br/>                                                   | Ordnet dem Computersystem Objekt, das das physische Host System darstellt, eine Instanz eines virtuellen Computers zu.<br/>                                                                                                                                       |
| [**MSVM- \_ hubdienst**](msvm-hostedservice.md)<br/>                                                         | Ordnet dem hostingcomputersystem einen Dienst zu.<br/>                                                                                                                                                                                                    |
| [**MSVM \_ lastappliedsnapshot**](msvm-lastappliedsnapshot.md)<br/>                                             | Stellt eine Zuordnung zwischen einem virtuellen System und den Einstellungsdaten der Momentaufnahme dar, die zuletzt auf das virtuelle System angewendet wurde.<br/>                                                                                                             |
| [**MSVM \_ owningjobelements**](msvm-owningjobelement.md)<br/>                                                   | Stellt eine Zuordnung zwischen einem Auftrag und dem verwalteten Element dar, das für die Erstellung des Auftrags zuständig ist.<br/>                                                                                                                                                  |
| [**MSVM \_ serviceaffectselements**](msvm-serviceaffectselement.md)<br/>                                         | Ordnet dem Verwaltungsdienst eine Instanz des virtuellen Computers zu, der den Zustand steuert.<br/>                                                                                                                                                                |
| [**MSVM \_ servicingsettings**](msvm-servicingsettings.md)<br/>                                                 | Enthält Einstellungen, die während des Wartungs Vorgangs verwendet werden.<br/>                                                                                                                                                                                                       |
| [**MSVM \_ settingsdefinestate**](msvm-settingsdefinestate.md)<br/>                                             | Ordnet einen virtuellen Computer und seine Geräte Instanzen von [**CIM \_ SettingData**](/previous-versions//cc136911(v=vs.85)) zu, die die aktuellen Einstellungen darstellen, die für diese Objekte gelten.<br/>                                                                               |
| [**MSVM \_ systemexportsettingdata**](msvm-systemexportsettingdata.md)<br/>                                     | Ordnet einen virtuellen Computer und seine Export Einstellungsdaten zu.<br/>                                                                                                                                                                                                 |
| [**MSVM \_ virtualsystemexportsettingdata**](msvm-virtualsystemexportsettingdata.md)<br/>                       | Bietet zusätzliche Informationen, die mit der [**exportsystemdefinition**](exportsystemdefinition-msvm-virtualsystemmanagementservice.md) -Methode der [**MSVM \_ virtualsystemmanagementservice**](msvm-virtualsystemmanagementservice.md) -Klasse verwendet werden können.<br/> |
| [**MSVM \_ virtualsystemmanagementfunktionen**](msvm-virtualsystemmanagementcapabilities.md)<br/>             | Beschreibt die Funktionen des zugeordneten [**MSVM \_ virtualsystemmanagementservice**](msvm-virtualsystemmanagementservice.md).<br/>                                                                                                                         |
| [**MSVM \_ virtualsystemmanagementservice**](msvm-virtualsystemmanagementservice.md)<br/>                       | Stellt den virtualisierungdienst dar, der auf einem einzelnen Host System vorhanden ist.<br/>                                                                                                                                                                                    |
| [**MSVM \_ virtualsystemmanagementservicesettingdata**](msvm-virtualsystemmanagementservicesettingdata.md)<br/> | Stellt die Einstellungen für den virtualisierungdienst dar, der auf einem einzelnen Host System vorhanden ist.<br/>                                                                                                                                                                   |
| [**MSVM \_ virtualsystemsnapshotservice**](msvm-virtualsystemsnapshotservice.md)<br/>                           | Dienst zum Erstellen, anwenden und zerstören von Momentaufnahmen von virtuellen Computern.<br/>                                                                                                                                                                                      |
