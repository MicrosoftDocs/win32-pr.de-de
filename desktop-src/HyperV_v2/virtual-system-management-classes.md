# <a name="virtual-system-management-classes"></a>Verwaltungsklassen für virtuelle Systeme

Im Folgenden sind Virtualisierungs-WMI-Klassen im Zusammenhang mit der Verwaltung virtueller Computer.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                                                | Beschreibung                                                                                                                                                                                                                                                          |
|----------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ AffectedJobElement**](msvm-affectedjobelement.md)<br/>                                               | Stellt eine Zuordnung zwischen einem Auftrag und dem verwalteten Element dar, die möglicherweise von seiner Ausführung betroffen ist.<br/>                                                                                                                                                    |
| [**Msvm \_ ConcreteJob**](msvm-concretejob.md)<br/>                                                             | Stellt eine Arbeitseinheit dar und wird verwendet, um den Fortschritt asynchroner Vorgänge nachzuverfolgen.<br/>                                                                                                                                                                   |
| [**Msvm \_ DiskMergeSettingData**](msvm-diskmergesettingdata.md)<br/>                                           | Stellt den Konfigurationsstatus der Datenträger-Mergeeinstellungen für einen virtuellen Computer dar.<br/>                                                                                                                                                                      |
| [**Msvm \_ ElementSettingData**](msvm-elementsettingdata.md)<br/>                                               | Ordnet ein verwaltetes Element seinen Konfigurationsdaten zu.<br/>                                                                                                                                                                                                 |
| [**\_Msvm-Fehler**](msvm-error.md)<br/>                                                                         | Enthält Informationen zu Schweregrad, Ursache, empfohlenen Aktionen und anderen Daten im Zusammenhang mit dem Ausfall eines CIM-Vorgangs.<br/>                                                                                                                            |
| [**Gehostete \_ Msvm-Abhängigkeit**](msvm-hosteddependency.md)<br/>                                                   | Ordnet dem Computersystemobjekt, das das physische Hostsystem darstellt, eine VM-Instanz zu.<br/>                                                                                                                                       |
| [**Msvm \_ HostedService**](msvm-hostedservice.md)<br/>                                                         | Ordnet dem Hostcomputersystem einen Dienst zu.<br/>                                                                                                                                                                                                    |
| [**Msvm \_ LastAppliedSnapshot**](msvm-lastappliedsnapshot.md)<br/>                                             | Stellt eine Zuordnung zwischen einem virtuellen System und den Einstellungsdaten der Momentaufnahme dar, die zuletzt auf das virtuelle System angewendet wurde.<br/>                                                                                                             |
| [**Msvm \_ OwningJobElement**](msvm-owningjobelement.md)<br/>                                                   | Stellt eine Zuordnung zwischen einem Auftrag und dem verwalteten Element dar, das für die Erstellung des Auftrags verantwortlich ist.<br/>                                                                                                                                                  |
| [**Msvm \_ ServiceAffectsElement**](msvm-serviceaffectselement.md)<br/>                                         | Ordnet dem Verwaltungsdienst, der den Zustand steuert, eine VM-Instanz zu.<br/>                                                                                                                                                                |
| [**Msvm \_ ServicingSettings**](msvm-servicingsettings.md)<br/>                                                 | Enthält Einstellungen, die bei Wartungsvorgängen verwendet werden.<br/>                                                                                                                                                                                                       |
| [**Msvm \_ SettingsDefineState**](msvm-settingsdefinestate.md)<br/>                                             | Ordnet einen virtuellen Computer und seine Geräte Instanzen von [**CIM \_ SettingData**](/previous-versions//cc136911(v=vs.85)) zu, die die aktuellen Einstellungen darstellen, die für diese Objekte gelten.<br/>                                                                               |
| [**Msvm \_ SystemExportSettingData**](msvm-systemexportsettingdata.md)<br/>                                     | Ordnet einen virtuellen Computer und die zugehörigen Exporteinstellungsdaten zu.<br/>                                                                                                                                                                                                 |
| [**Msvm \_ VirtualSystemExportSettingData**](msvm-virtualsystemexportsettingdata.md)<br/>                       | Stellt zusätzliche Informationen bereit, die mit der [**ExportSystemDefinition-Methode**](exportsystemdefinition-msvm-virtualsystemmanagementservice.md) der [**Msvm \_ VirtualSystemManagementService-Klasse**](msvm-virtualsystemmanagementservice.md) verwendet werden sollen.<br/> |
| [**Msvm \_ VirtualSystemManagementCapabilities**](msvm-virtualsystemmanagementcapabilities.md)<br/>             | Beschreibt die Funktionen des zugeordneten [**Msvm \_ VirtualSystemManagementService**](msvm-virtualsystemmanagementservice.md).<br/>                                                                                                                         |
| [**Msvm \_ VirtualSystemManagementService**](msvm-virtualsystemmanagementservice.md)<br/>                       | Stellt den Virtualisierungsdienst dar, der auf einem einzelnen Hostsystem vorhanden ist.<br/>                                                                                                                                                                                    |
| [**Msvm \_ VirtualSystemManagementServiceSettingData**](msvm-virtualsystemmanagementservicesettingdata.md)<br/> | Stellt die Einstellungen für den Virtualisierungsdienst dar, der auf einem einzelnen Hostsystem vorhanden ist.<br/>                                                                                                                                                                   |
| [**Msvm \_ VirtualSystemSnapshotService**](msvm-virtualsystemsnapshotservice.md)<br/>                           | Dienst zum Erstellen, Anwenden und Zerstören von Momentaufnahmen virtueller Computer.<br/>                                                                                                                                                                                      |
