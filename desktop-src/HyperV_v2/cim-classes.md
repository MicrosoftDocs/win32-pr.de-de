# <a name="cim-classes-hyper-v-management"></a>CIM-Klassen (Hyper-V-Verwaltung)

Die CIM-Klassen sind Basisklassen, von denen alle WMI-Objekte abgeleitet werden. Jeder Anbieter kann möglicherweise eine etwas andere Version der CIM-Klassen verwenden: Daher enthalten die meisten v2-WMI-Anbieter eine eigene Dokumentation der CIM-Klasse.

Die folgenden CIM-Klassen befinden sich im \\ Namespace "root Virtualization \\ V2".
