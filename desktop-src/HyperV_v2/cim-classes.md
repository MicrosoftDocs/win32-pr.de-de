# <a name="cim-classes-hyper-v-management"></a>CIM-Klassen (Hyper-V-Verwaltung)

Die CIM-Klassen sind Basisklassen, von denen alle WMI-Objekte abgeleitet werden. Jeder Anbieter kann möglicherweise eine etwas andere Version der CIM-Klassen verwenden. daher enthalten die meisten v2-WMI-Anbieter eine eigene CIM-Klassendokumentation.

Die folgenden CIM-Klassen befinden sich im \\ Root Virtualization \\ v2-Namespace.
