# <a name="hyper-v-networking-api-reference"></a>Referenz zur Hyper-V-Netzwerk-API

Die Netzwerkarchitektur für die Virtualisierung modelliert die physische Netzwerkarchitektur. Es werden Standardnetzwerkobjekte wie Switches, Switchports und Netzwerkadapter verwendet.

Im Folgenden sind Virtualisierungs-WMI-Klassen im Zusammenhang mit Netzwerken.
