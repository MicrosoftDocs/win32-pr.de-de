# <a name="hyper-v-virtual-fibre-channel-api-reference"></a>Virtuelle Hyper-V-Fibre Channel-API-Referenz

Die virtuelle Hyper-V-Fibre Channel-API definiert die folgenden Programmier Elemente.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                    | BESCHREIBUNG                                                                                                    |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ externalfcport**](msvm-externalfcport.md)<br/>                           | Stellt einen externen Fibre Channel Anschluss dar.<br/>                                                          |
| [**MSVM-Dienst für die \_ Verbindungs Herstellung**](msvm-fcactiveconnection.md)<br/>                   | Verbindet einen Switchport mit dem Fibre Channel-Endpunkt, mit dem der Port verbunden ist.<br/>                |
| [**MSVM \_ fcdebug-Implementierung**](msvm-fcdevicesapimplementation.md)<br/>     | Stellt eine Zuordnung zwischen einem Dienst Zugriffspunkt und dem logischen Gerät dar, von dem es implementiert wird.<br/> |
| [**MSVM \_ fcendpoint**](msvm-fcendpoint.md)<br/>                                   | Stellt den logischen Verbindungspunkt für einen Fibre Channel-Port dar.<br/>                                   |
| [**MSVM \_ fcportzugewiesene Daten**](msvm-fcportallocationsettingdata.md)<br/> | Stellt den konfigurierten Status eines synthetischen Fibre Channel Ports oder eines Fibre Channel Switchports dar.<br/>   |
| [**MSVM \_ fcswitchport**](msvm-fcswitchport.md)<br/>                               | Stellt einen Port auf dem virtuellen Fibre Channel Switch dar.<br/>                                              |
| [**MSVM \_ syntheticfcport**](msvm-syntheticfcport.md)<br/>                         | Stellt einen synthetischen Fibre Channel Anschluss dar.<br/>                                                          |
| [**MSVM \_ syntheticfcportsettingdata**](msvm-syntheticfcportsettingdata.md)<br/>   | Stellt den konfigurierten Status eines synthetischen Fibre Channel Ports dar.<br/>                                  |
| [**MSVM \_ virtualfcswitch**](msvm-virtualfcswitch.md)<br/>                         | Stellt einen virtuellen Fibre Channel Switch dar.<br/>                                                          |
| [**MSVM \_ virtualfcswitchsettingdata**](msvm-virtualfcswitchsettingdata.md)<br/>   | Stellt die Konfiguration eines virtuellen Fibre Channel Schalters dar.<br/>                                     |
