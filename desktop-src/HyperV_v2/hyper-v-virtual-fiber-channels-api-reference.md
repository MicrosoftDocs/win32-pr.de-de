# <a name="hyper-v-virtual-fibre-channel-api-reference"></a>Referenz zur virtuellen Hyper-V-Fibre Channel-API

Die virtuelle Hyper-V-Fibre Channel-API definiert die folgenden Programmierelemente.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                    | Beschreibung                                                                                                    |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ ExternalFcPort**](msvm-externalfcport.md)<br/>                           | Stellt einen externen Fibre Channel Port dar.<br/>                                                          |
| [**Msvm \_ FcActiveConnection**](msvm-fcactiveconnection.md)<br/>                   | Verbindet einen Switchport mit dem Fibre Channel Endpunkt, mit dem der Port verbunden ist.<br/>                |
| [**Msvm \_ FcDeviceSAPImplementation**](msvm-fcdevicesapimplementation.md)<br/>     | Stellt eine Zuordnung zwischen einem Dienstzugriffspunkt und dem logischen Gerät dar, das ihn implementiert.<br/> |
| [**Msvm \_ FcEndpoint**](msvm-fcendpoint.md)<br/>                                   | Stellt den logischen Verbindungspunkt für einen Fibre Channel Port dar.<br/>                                   |
| [**Msvm \_ FcPortAllocationSettingData**](msvm-fcportallocationsettingdata.md)<br/> | Stellt den konfigurierten Zustand eines synthetischen Fibre Channel Ports oder eines Fibre Channel Switchports dar.<br/>   |
| [**Msvm \_ FcSwitchPort**](msvm-fcswitchport.md)<br/>                               | Stellt einen Port auf dem virtuellen Fibre Channel Switch dar.<br/>                                              |
| [**Msvm \_ SyntheticFcPort**](msvm-syntheticfcport.md)<br/>                         | Stellt einen synthetischen Fibre Channel Port dar.<br/>                                                          |
| [**Msvm \_ SyntheticFcPortSettingData**](msvm-syntheticfcportsettingdata.md)<br/>   | Stellt den konfigurierten Zustand eines synthetischen Fibre Channel Ports dar.<br/>                                  |
| [**Msvm \_ VirtualFcSwitch**](msvm-virtualfcswitch.md)<br/>                         | Stellt einen virtuellen Fibre Channel Switch dar.<br/>                                                          |
| [**Msvm \_ VirtualFcSwitchSettingData**](msvm-virtualfcswitchsettingdata.md)<br/>   | Stellt die Konfiguration eines virtuellen Fibre Channel Switchs dar.<br/>                                     |
