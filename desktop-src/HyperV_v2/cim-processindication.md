# <a name="cim_processindication-class"></a>CIM \_ ProcessIndication-Klasse

**CIM \_ ProcessIndication** ist eine abstrakte Oberklasse für spezialisierte Indikatorklassen, die bestimmte Änderungen und Warnungen adressieren, die von Anbietern und Instrumentierung veröffentlicht werden.

## <a name="syntax"></a>Syntax

``` syntax
[Indication, Version("2.6.0"), UMLPackagePath("CIM::Event"), AMENDMENT]
class CIM_ProcessIndication : CIM_Indication
{
};
```

## <a name="members"></a>Member

Die **CIM \_ ProcessIndication-Klasse** definiert keine Member.
