# <a name="cim_processindication-class"></a>CIM \_ processindiationklasse

**CIM \_ Processindikation** ist eine abstrakte Superklasse für spezialisierte Indikator Klassen, die bestimmte Änderungen und Warnungen adressieren, die von Anbietern und Instrumentation veröffentlicht werden.

## <a name="syntax"></a>Syntax

``` syntax
[Indication, Version("2.6.0"), UMLPackagePath("CIM::Event"), AMENDMENT]
class CIM_ProcessIndication : CIM_Indication
{
};
```

## <a name="members"></a>Member

Die **CIM \_ processindiation** -Klasse definiert keine Member.
