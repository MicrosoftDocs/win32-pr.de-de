# <a name="msvm_persistentmemorycontroller-class"></a>MSVM \_ persistentmemorycontroller-Klasse

Stellt einen virtuellen permanenten Speichercontroller dar.

Die folgende Syntax wurde aus MOF-Code (Managed Object Format, verwaltetes Objektformat) vereinfacht und enthält alle geerbten Eigenschaften.

## <a name="syntax"></a>Syntax

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_PersistentMemoryController : CIM_Controller
{
};
```

## <a name="members"></a>Member

Die **MSVM \_ persistentmemorycontroller** -Klasse definiert keine Member.
