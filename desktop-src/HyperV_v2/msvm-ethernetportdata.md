# <a name="msvm_ethernetportdata-class"></a>Msvm \_ EthernetPortData-Klasse

Eine abstrakte Klasse, die Portlaufzeitdaten darstellt, die von einer Ethernet-Switcherweiterung gesammelt werden. Alle Ethernet-Portdatenklassen müssen von dieser abstrakten Klasse ableiten.

Die folgende Syntax wird Managed Object Format (MOF)-Code vereinfacht und enthält alle geerbten Eigenschaften.

## <a name="syntax"></a>Syntax

``` syntax
[Abstract, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_EthernetPortData : CIM_ManagedElement
{
  string InstanceID;
  string Caption;
  string Description;
  string ElementName;
  string SystemCreationClassName;
  string SystemName;
  string DeviceCreationClassName = "Msvm_EthernetSwitchPort";
  string DeviceID;
  string CreationClassName;
  string Name;
};
```
