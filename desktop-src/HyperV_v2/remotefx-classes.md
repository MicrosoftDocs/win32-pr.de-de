# <a name="remote-desktop-classes"></a>Remotedesktop Klassen

Die Remotedesktop-Virtualisierungs-WMI-Klassen werden verwendet, um Remotedesktop und RemoteFX Ressourcen und virtuelle Computer zu verwalten.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                                            | Beschreibung                                                                                                                                                  |
|------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ InteractiveSessionACE**](msvm-interactivesessionace.md)<br/>                                     | Stellt einen *Zugriffssteuerungseintrag (ACE)* dar, der den Zugriff auf die interaktive Sitzung eines virtuellen Computers bestimmt.<br/>                                |
| [**Msvm \_ Physical3dGraphicsProcessor**](msvm-physical3dgraphicsprocessor.md)<br/>                         | Beschreibt die physische 3D-Grafikverarbeitungseinheit (GPU).<br/>                                                                                        |
| [**Msvm \_ PhysicalGPUInfo**](msvm-physicalgpuinfo.md)<br/>                                                 | Enthält Informationen zu einer RemoteFX physischen Grafikverarbeitungseinheit (GPU).<br/>                                                                    |
| [**Msvm \_ RdvComponent**](msvm-rdvcomponent.md)<br/>                                                       | Stellt den Zustand der RDV-Komponente dar, die für die Bereitstellung eines Transports für das übergeordnete Element zum Gast zu Konfigurationszwecken zuständig ist.<br/> |
| [**Msvm \_ RdvComponentSettingData**](msvm-rdvcomponentsettingdata.md)<br/>                                 | Stellt den konfigurierten Zustand der RDV-Komponente (Remotedesktop Virtualization) dar. Der Standardstatus ist Aktiviert.<br/>                               |
| [**Msvm \_ Synth3dVideoPool**](msvm-synth3dvideopool.md)<br/>                                               | Enthält Informationen zu den synthetischen 3D-Grafikprozessoren (GPUs), die auf dem Hostsystem verfügbar sind.<br/>                                 |
| [**Msvm \_ Synthetic3DDisplayController**](msvm-synthetic3ddisplaycontroller.md)<br/>                       | Stellt den synthetischen 3D-Anzeigecontroller dar, der einem virtuellen Computer zugewiesen ist.<br/>                                                            |
| [**Msvm \_ Synthetic3DDisplayControllerSettingData**](msvm-synthetic3ddisplaycontrollersettingdata.md)<br/> | Stellt Einstellungen für einen synthetischen 3D-Anzeigecontroller für einen virtuellen Computer dar.<br/>                                                                 |
| [**Msvm \_ Synthetic3DService**](msvm-synthetic3dservice.md)<br/>                                           | Beschreibt den synthetischen 3D-GPU-Dienst.<br/>                                                                                                          |
| [**Msvm \_ Synthetic3DServiceSettingData**](msvm-synthetic3dservicesettingdata.md)<br/>                     | Stellt die Einstellungen für den synthetischen 3D-Dienst dar, der auf einem einzelnen Hostsystem vorhanden ist.<br/>                                                            |
| [**Msvm \_ TerminalService**](msvm-terminalservice.md)<br/>                                                 | Verwaltet alle Remoteterminalverbindungen mit einem bestimmten Host.<br/>                                                                                     |
| [**Msvm \_ TerminalServiceSettingData**](msvm-terminalservicesettingdata.md)<br/>                           | Stellt die Einstellungen für die Terminaldienste des virtuellen Computers auf einem Host dar.<br/>                                                                     |
