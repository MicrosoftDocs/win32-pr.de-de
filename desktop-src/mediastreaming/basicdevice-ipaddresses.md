# <a name="basicdeviceipaddresses-property"></a>Basicdevice. IPADRESSEN (Eigenschaft)

Ruft einen Vektor von IP-Adressen ab.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```C++
HRESULT get_IpAddresses(
  [out] IVector< HSTRING > **value
);
```



## <a name="property-value"></a>Eigenschaftswert

Eine Aufzähl Bare Auflistung von Zeigern auf IP-Adressen.
