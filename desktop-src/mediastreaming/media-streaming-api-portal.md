# <a name="windows-media-streaming-api"></a>Windows Media Streaming-API

## <a name="purpose"></a>Zweck

Die Media Streaming-API modelliert die Daten von Digital Media-Renderer (DMR) und Digital Media Server (DMS) gemäß den DLNA-Richtlinien für eine einfache programmgesteuerte Verwendung, sodass Entwickler Anwendungen mit Funktionen für Digital Media Controller (DMC), Digital Media Player (DMP) und Push Controller (+ PU +) gemäß den DLNA-Richtlinien erstellen können.

## <a name="developer-audience"></a>Entwicklergruppe

Media Streaming API ist für die Verwendung durch C++-Entwickler konzipiert, die Anwendungen für digitale Medien erstellen, die mit DLNA-Geräten interagieren.
