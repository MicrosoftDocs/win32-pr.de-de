# <a name="streamselector-class"></a>Streamselector-Klasse

Implementiert die [**istreamselector Statics**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-istreamselectorstatics) -Schnittstelle und ermöglicht die Auswahl eines Streams.

**Streamselector** verfügt über folgende Typen von Membern:

-   [Methoden](#methods)

### <a name="methods"></a>Methoden

Die **streamselector** -Klasse verfügt über diese Methoden.



| Methode                                                                      | BESCHREIBUNG                                                  |
|:----------------------------------------------------------------------------|:-------------------------------------------------------------|
| [**Getstreampropertiesasync**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85)) | Ruft die Eigenschaften des Streams asynchron ab.<br/> |
| [**Selectbeststreamasync**](/previous-versions/windows/desktop/legacy/hh829002(v=vs.85))       | Abfragen asynchron für den besten Stream.<br/>       |
