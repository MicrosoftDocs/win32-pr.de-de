# <a name="basicdeviceuniquedevicename-property"></a>Basicdevice. uniquedevicename-Eigenschaft

Ruft den eindeutigen Gerätenamen (User Name, udn) des Geräts ab.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```C++
HRESULT get_UniqueDeviceName(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Eigenschaftswert

Ein Zeiger auf den Modell-udn des Geräts.
