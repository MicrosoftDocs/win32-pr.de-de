# <a name="media-streaming-enumerations"></a>Medienstreamingenumerationen

Die [Media Streaming-API](media-streaming-api-portal.md) stellt die folgenden Enumerationen bereit.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                   | BESCHREIBUNG                                                                               |
|---------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**ConnectionStatus**](connectionstatus.md)<br/> | Stellt den Status des Geräts im Netzwerk als zuletzt angezeigt dar.<br/>                |
| [**Gerätetypen**](devicetypes.md)<br/>           | Beschreibt die DLNA-Gerätetypen, die von der Media Streaming-API unterstützt werden.<br/> |
| [**Transportstate**](transportstate.md)<br/>     | Definiert die verfügbaren Transport Zustände gemäß den UPnP-Richtlinien.<br/>      |
| [**Transportstatus**](transportstatus.md)<br/>   | Definiert den verfügbaren Transport Status gemäß den UPnP-Richtlinien.<br/>      |
