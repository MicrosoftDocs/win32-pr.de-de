# <a name="media-streaming-enumerations"></a>Medienstreamingenumerationen

Die [Medienstreaming-API](media-streaming-api-portal.md) stellt die folgenden Enumerationen bereit.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                   | BESCHREIBUNG                                                                               |
|---------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**ConnectionStatus**](connectionstatus.md)<br/> | Stellt den Zustand des Geräts im Netzwerk wie zuletzt gesehen dar.<br/>                |
| [**DeviceTypes**](devicetypes.md)<br/>           | Beschreibt die DLNA-Gerätetypen, die von der Medienstreaming-API unterstützt werden.<br/> |
| [**TransportState**](transportstate.md)<br/>     | Definiert die verfügbaren Transportzustände gemäß den UPnP-Richtlinien.<br/>      |
| [**TransportStatus**](transportstatus.md)<br/>   | Definiert den verfügbaren Transportstatus gemäß den UPnP-Richtlinien.<br/>      |
