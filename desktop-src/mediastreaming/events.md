# <a name="events-media-streaming-api"></a>Ereignisse (Media Streaming-API)

Die [Media Streaming-API](media-streaming-api-portal.md) generiert die folgenden Ereignisse.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                           | BESCHREIBUNG                                                                                                                                              |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Connectionstatuschangi-Ereignis**](connectionstatuschanged.md)<br/>     | Tritt auf, wenn sich der Netzwerk Verbindungsstatus des Geräts ändert.<br/>                                                                                   |
| [**Devicearrival-Ereignis**](devicearrival.md)<br/>                         | Tritt auf, wenn der Liste der Geräte, die von der [**cacheddevices**](idevicecontroller-cacheddevices.md) -Methode zurückgegeben werden, ein neues Gerät hinzugefügt wird.<br/>     |
| [**Deviceabflug-Ereignis**](devicedeparture.md)<br/>                     | Tritt auf, wenn ein neues Gerät aus der Liste der Geräte entfernt wird, die von der [**cacheddevices**](idevicecontroller-cacheddevices.md) -Methode zurückgegeben werden.<br/> |
| [**Renderingparametersupdate-Ereignis**](renderingparametersupdate.md)<br/> | Tritt auf, wenn ein beliebiger Satz von renderingsteuerungsparametern auf der DMR aktualisiert wird.<br/>                                                          |
| [**Transportparametersupdate-Ereignis**](transportparametersupdate.md)<br/> | Tritt auf, wenn eine beliebige Anzahl von Transport Parametern auf der DMR aktualisiert wird.<br/>                                                                  |
