# <a name="events-media-streaming-api"></a>Ereignisse (Medienstreaming-API)

Die [Medienstreaming-API](media-streaming-api-portal.md) generiert die folgenden Ereignisse.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                           | BESCHREIBUNG                                                                                                                                              |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ConnectionStatusChanged-Ereignis**](connectionstatuschanged.md)<br/>     | Tritt ein, wenn sich der Netzwerkverbindungsstatus des Geräts ändert.<br/>                                                                                   |
| [**DeviceAvaval-Ereignis**](devicearrival.md)<br/>                         | Tritt ein, wenn ein neues Gerät der Liste der Geräte hinzugefügt wird, die von der [**CachedDevices-Methode**](idevicecontroller-cacheddevices.md) zurückgegeben werden.<br/>     |
| [**DeviceDeparture-Ereignis**](devicedeparture.md)<br/>                     | Tritt ein, wenn ein neues Gerät aus der Liste der Geräte entfernt wird, die von der [**CachedDevices-Methode**](idevicecontroller-cacheddevices.md) zurückgegeben werden.<br/> |
| [**RenderingParametersUpdate-Ereignis**](renderingparametersupdate.md)<br/> | Tritt ein, wenn einer der Renderingsteuerelementparameter für die DMR aktualisiert wird.<br/>                                                          |
| [**TransportParametersUpdate-Ereignis**](transportparametersupdate.md)<br/> | Tritt ein, wenn einer der Transportparameter für die DMR aktualisiert wird.<br/>                                                                  |
