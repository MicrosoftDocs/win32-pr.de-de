# <a name="media-streaming-interfaces"></a>Medien Streaming-Schnittstellen

Die [Media Streaming-API](media-streaming-api-portal.md) stellt die folgenden Schnittstellen bereit.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                 | BESCHREIBUNG                                                                                                                                                                                                                                           |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Iactivebasicdevice**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-iactivebasicdevice)<br/>                           | Stellt ein aktives [**ibasicdevice**](ibasicdevice.md) -Element dar, das einem UPnP-Gerät zugeordnet ist. <br/>                                                                                                                                          |
| [**Iactivebasicdevicestatik**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-iactivebasicdevicestatics)<br/>             | Stellt statische Methoden zum Erstellen von [**iactivebasicdevice**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-iactivebasicdevice) -Objekten bereit. <br/>                                                                                                                                            |
| [**Ibasicdevice**](ibasicdevice.md)<br/>                                       | Kapselt die Methoden und Ereignisse, die erforderlich sind, um ein DLNA-Gerät zu modellieren.<br/>                                                                                                                                                                         |
| [**Idevicecontroller**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-idevicecontroller)<br/>                             | Kapselt die Methoden und Ereignisse, die erforderlich sind, um eine Liste der zwischengespeicherten Digital Media-Renderer (DMRs) und/oder Digital Media-Server (DMSS) abzurufen oder um die DMRs und/oder DMSS, die sich derzeit im Netzwerk befinden, asynchron zu suchen.<br/>              |
| [**Ideviceicon**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-ideviceicon)<br/>                                         | Kapselt die erforderlichen Methoden zum Bereitstellen von Informationen zum Symbol eines DLNA-Geräts.<br/>                                                                                                                                                    |
| [**Idevicepair**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-idevicepair)<br/>                                         | Stellt ein paar von [**activebasicdevice**](/previous-versions/windows/desktop/legacy/dn385755(v=vs.85)) -Objekten dar, das aus einem Renderer und einem Server besteht.<br/>                                                                                                                 |
| [**Imediarenderer**](imediarenderer.md)<br/>                                   | Kapselt die Methoden und Ereignisse, die erforderlich sind, um ein DLNA Digital Media-Renderer-Gerät (DMR) darzustellen.<br/>                                                                                                                                        |
| [**Imediarendereraktioninformation**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-imediarendereractioninformation)<br/> | Kapselt die Methoden, die erforderlich sind, um Informationen darüber bereitzustellen, welche Methoden aktuell für die DMR aufgerufen werden können.<br/>                                                                                                                             |
| [**Imediarendererfactory**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-imediarendererfactory)<br/>                     | Kapselt die Methoden, die erforderlich sind, um asynchron eine neue Instanz eines Objekts zu erstellen, das die [**imediarenderer**](imediarenderer.md) -Schnittstelle implementiert.<br/>                                                                               |
| [**Istreamselector Statics**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-istreamselectorstatics)<br/>                   | Kapselt die Methoden, die erforderlich sind, um einen Datenstrom asynchron auszuwählen.<br/>                                                                                                                                                                         |
| [**Itransportparameters**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-itransportparameters)<br/>                       | Kapselt die Methoden, die zum Bereitstellen von Informationen zu den aktuellen transportbezogenen Einstellungen des DMR erforderlich sind. Diese Einstellungen umfassen den aktuellen Transport Status und Informationen darüber, welche Methoden derzeit auf der DMR aufgerufen werden können.<br/> |
