# <a name="delegates"></a>Delegaten

Die [Media Streaming-API](media-streaming-api-portal.md) stellt die folgenden Ereignishandlerfunktionen bereit.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                   | BESCHREIBUNG                                                                                                                                                                                                            |
|-----------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ConnectionStatusHandler**](/previous-versions/windows/desktop/legacy/hh828836(v=vs.85))<br/>                   | Stellt die Funktion dar, die das [**ConnectionStatusChanged-Ereignis**](connectionstatuschanged.md) behandelt, das den Client über Änderungen im Netzwerkverbindungsstatus des Geräts benachrichtigt.<br/>               |
| [**DeviceControllerHandler**](/previous-versions/windows/desktop/legacy/hh828843(v=vs.85))<br/>       | Stellt die Funktion dar, die die [**Ereignisse DeviceArrarr und**](devicearrival.md) [**DeviceDeparture**](devicedeparture.md) behandelt, die den Client über Änderungen in der Liste der zwischengespeicherten Geräte benachrichtigen.<br/> |
| [**RenderingParametersUpdateHandler**](/previous-versions/windows/desktop/legacy/hh828994(v=vs.85))<br/> | Stellt die Funktion dar, die das [**RenderingParametersUpdate-Ereignis**](renderingparametersupdate.md) behandelt, das den Client über ein Update auf einen der Renderingsteuersteuerparameter der DMR benachrichtigt.<br/>  |
| [**TransportParametersUpdateHandler**](/previous-versions/windows/desktop/legacy/hh829007(v=vs.85))<br/> | Stellt die Funktion dar, die das [**TransportParametersUpdate-Ereignis**](transportparametersupdate.md) behandelt, das den Client über ein Update für einen der Transportparameter der DMR benachrichtigt.<br/>          |
