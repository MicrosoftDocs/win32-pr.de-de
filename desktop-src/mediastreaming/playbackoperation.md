# <a name="playbackoperation-class"></a>PlaybackOperation-Klasse

Registriert einen Ereignishandler, der aufgerufen wird, wenn ein asynchroner Vorgang, der von einer der [**MediaRenderer-Wiedergabemethoden**](mediarenderer.md) gestartet wurde, abgeschlossen wird, und stellt eine Methode bereit, die die Ergebnisse des Vorgangs zurückgibt.

**PlaybackOperation** verfügt über diese Typen von Membern:

-   [Methoden](#methods)
-   [Eigenschaften](#properties)

### <a name="methods"></a>Methoden

Die **PlaybackOperation-Klasse** verfügt über diese Methoden.



| Methode                                             | BESCHREIBUNG                                                                                                                                |
|:---------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetResults**](playbackoperation-getresults.md) | Gibt die Ergebnisse eines asynchronen Vorgangs zurück, der von einer der [**MediaRenderer-Wiedergabemethoden**](mediarenderer.md) gestartet wurde.<br/> |



 

### <a name="properties"></a>Eigenschaften

Die **PlaybackOperation-Klasse** verfügt über diese Eigenschaften.



| Eigenschaft                                                    | Zugriffstyp           | BESCHREIBUNG                                                                                                                                                                           |
|:------------------------------------------------------------|:----------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Abgeschlossen**](playbackoperation-completed.md)<br/> | Lesen/Schreiben<br/> | Ruft einen Ereignishandler ab, der aufgerufen wird, wenn der von einer der [**MediaRenderer-Wiedergabemethoden**](mediarenderer.md) gestartete asynchrone Vorgang abgeschlossen wird, oder legt diesen fest. <br/> |
