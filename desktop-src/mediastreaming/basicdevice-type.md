# <a name="basicdevicetype-property"></a>Basicdevice. Type (Eigenschaft)

Ruft einen Enumerationswert ab, der den Gerätetyp des DLNA-Geräts angibt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```C++
HRESULT get_Type(
  [out] DeviceTypes *value
);
```



## <a name="property-value"></a>Eigenschaftswert

Ein Zeiger auf einen Wert vom Typ " [**de vicetypes**](devicetypes.md) ".
