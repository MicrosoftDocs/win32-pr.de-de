# <a name="getmuteoperation-class"></a>Getmuteoperation-Klasse

Registriert einen Ereignishandler, der aufgerufen wird, wenn der von [**getmuteasync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync) gestartete asynchrone Vorgang abgeschlossen ist, und stellt eine Methode bereit, die die Ergebnisse des Vorgangs zurückgibt.

**Getmuteoperation** verfügt über folgende Typen von Membern:

-   [Methoden](#methods)
-   [Eigenschaften](#properties)

### <a name="methods"></a>Methoden

Die **getmuteoperation** -Klasse verfügt über diese Methoden.



| Methode                                            | BESCHREIBUNG                                                                                                                   |
|:--------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------|
| [**GetResults**](getmuteoperation-getresults.md) | Gibt die Ergebnisse des asynchronen Vorgangs zurück, der von [**getmuteasync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync)gestartet wurde. <br/> |



 

### <a name="properties"></a>Eigenschaften

Die **getmuteoperation** -Klasse verfügt über diese Eigenschaften.



| Eigenschaft                                                   | Zugriffstyp           | BESCHREIBUNG                                                                                                                                                            |
|:-----------------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Abgeschlossen**](getmuteoperation-completed.md)<br/> | Lesen/Schreiben<br/> | Ruft einen Ereignishandler ab, der aufgerufen wird, wenn der von [**getmuteasync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync) gestartete asynchrone Vorgang abgeschlossen ist, oder legt diesen fest. <br/> |
