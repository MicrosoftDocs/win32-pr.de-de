# <a name="basicdevicemanufacturername-property"></a>Basicdevice. ManufacturerName-Eigenschaft

Ruft den Herstellernamen des Geräts ab.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```C++
HRESULT get_ManufacturerName(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Eigenschaftswert

Ein Zeiger auf den Herstellernamen des Geräts.
