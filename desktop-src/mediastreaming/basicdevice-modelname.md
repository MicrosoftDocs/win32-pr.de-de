# <a name="basicdevicemodelname-property"></a>Basicdevice. modelname (Eigenschaft)

Ruft den Modellnamen des Geräts ab.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```C++
HRESULT get_ModelName(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Eigenschaftswert

Ein Zeiger auf den Modellnamen des Geräts.
