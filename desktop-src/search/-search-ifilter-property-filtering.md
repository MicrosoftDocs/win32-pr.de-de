### <a name="property-size-limitations"></a><span data-ttu-id="9e2d5-131">Einschränkungen der Eigenschafts Größe</span><span class="sxs-lookup"><span data-stu-id="9e2d5-131">Property Size Limitations</span></span>

- <span data-ttu-id="9e2d5-133">Die maximale Größe der Daten, die von Windows Search pro Datei akzeptiert werden.</span><span class="sxs-lookup"><span data-stu-id="9e2d5-133">The maximum size of data that Windows Search accepts per file.</span></span>
- <span data-ttu-id="9e2d5-134">Die maximale Größe pro Eigenschaft, wie in der Eigenschafts Beschreibungsdatei definiert.</span><span class="sxs-lookup"><span data-stu-id="9e2d5-134">The maximum size per property as defined in the property description file.</span></span>

<span data-ttu-id="9e2d5-135">Derzeit verwendet Windows Search die definierte Eigenschaften Größe nicht, wenn die Menge der Daten berechnet wird, die Sie von einem Element akzeptiert.</span><span class="sxs-lookup"><span data-stu-id="9e2d5-135">Currently, Windows Search does not use the defined property size when calculating the amount of data it accepts from an item.</span></span> <span data-ttu-id="9e2d5-136">Stattdessen ist der Grenzwert, den Windows Search verwendet, das Produkt der Dateigröße und der `MaxGrowFactor` (Dateigröße N \* MaxGrowFactor), die aus der Registrierung gelesen werden.</span><span class="sxs-lookup"><span data-stu-id="9e2d5-136">Instead, the limit Windows Search uses is the product of the size of the file and the `MaxGrowFactor` (file size N \* MaxGrowFactor) read from the registry.</span></span> <span data-ttu-id="9e2d5-137">Der Standardwert `MaxGrowFactor` ist vier.</span><span class="sxs-lookup"><span data-stu-id="9e2d5-137">The default `MaxGrowFactor` is four.</span></span>

```
   SOFTWARE
      Microsoft

            MaxGrowFactor
```

<span data-ttu-id="9e2d5-138">Wenn Ihr Dateityp tendenziell klein ist, aber größere Eigenschaften hat, akzeptiert Windows Search möglicherweise nicht alle Eigenschaften Daten, die Sie ausgeben möchten.</span><span class="sxs-lookup"><span data-stu-id="9e2d5-138">Consequently, if your file type tends to be small in total size but have larger properties, Windows Search may not accept all the property data you want to emit.</span></span> <span data-ttu-id="9e2d5-139">Sie können jedoch die `MaxGrowFactor` an Ihre Anforderungen anpassen.</span><span class="sxs-lookup"><span data-stu-id="9e2d5-139">However, you can increase the `MaxGrowFactor` to suit your needs.</span></span>
