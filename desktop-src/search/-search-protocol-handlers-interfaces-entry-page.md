# <a name="protocol-handlers-interfaces"></a>Schnittstellen für Protokollhandler

Dieser Windows Search-Referenzabschnitt besteht aus Protokollhandlerschnittstellen.


## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                   | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IItemPropertyBag**](iitempropertybag.md)<br/>                                 | Definiert Methoden zum Abrufen von Informationen über die Eigenschaften eines Suchelements. Diese Schnittstelle wird nur auf Windows XP und Windows Server 2003 unterstützt und sollte nicht mehr verwendet werden.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**IOpLockStatus**](/windows/desktop/api/Searchapi/nn-searchapi-ioplockstatus)<br/>                               | Stellt Methoden zum Überprüfen der von Microsoft Windows Desktop Search (WDS) verwendeten opportunistischen Sperre für Elemente während der Indizierung bereit. Wenn ein anderer Prozess die Datei auf inkompatible Weise sperrt, verliert WDS seine Sperre und lässt zu, dass der andere Prozess über die Datei verfügen kann. Dieser Mechanismus ermöglicht die Ausführung von WDS im Hintergrund. Daher muss WDS seine Sperren überprüfen, um sicherzustellen, dass kein anderer Prozess Vorrang hat, während WDS das Element indiziert. <br/> Ein [**IUrlAccessor-Objekt**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) eines Drittanbieters kann diese Schnittstelle implementieren, wenn der zugrunde liegende Datenspeicher einen Mechanismus zum Nachverfolgen des gleichzeitigen Zugriffs auf Elemente bereitstellt. Wenn diese Schnittstelle von **IUrlAccessor** verfügbar gemacht wird, überprüft WDS den [**IOpLockStatus**](/windows/desktop/api/Searchapi/nn-searchapi-ioplockstatus) beim Indizieren von Elementen aus diesem Speicher. <br/> |
| [**IProtocolHandlerServices**](/previous-versions/windows/desktop/legacy/bb231447(v=vs.85))<br/>         |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IProtocolHandlerSite**](/windows/desktop/api/Searchapi/nn-searchapi-iprotocolhandlersite)<br/>                 | Stellt Methoden für das [**IUrlAccessor-Objekt**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) eines Protokollhandlers bereit, um den Filterdaemon nach dem entsprechenden Filter für das URL-Element abzufragen. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**ISearchProtocol**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocol)<br/>                           | Stellt Methoden zum Aufrufen, Initialisieren und Verwalten von [**IUrlAccessor-Objekten**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) bereit. Methoden in dieser Schnittstelle werden vom Protokollhost aufgerufen, wenn URLs vom Gatherer verarbeitet werden. <br/> Der Protokollhandler implementiert das Protokoll für den Zugriff auf eine Inhaltsquelle im systemeigenen Format. Verwenden Sie diese Schnittstelle, um einen benutzerdefinierten Protokollhandler zu implementieren, um die Datenquellen zu erweitern, die indiziert werden können. <br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**ISearchProtocol2**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocol2)<br/>                         | Stellt Methoden zum Aufrufen, Initialisieren und Verwalten von [**IUrlAccessor-Objekten**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) bereit. Methoden in dieser Schnittstelle werden vom Protokollhost aufgerufen, wenn URLs vom Gatherer verarbeitet werden. <br/> Der Protokollhandler implementiert das Protokoll für den Zugriff auf eine Inhaltsquelle im systemeigenen Format. Verwenden Sie diese Schnittstelle, um einen benutzerdefinierten Protokollhandler zu implementieren, um die Datenquellen zu erweitern, die indiziert werden können. <br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**ISearchProtocolThreadContext**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocolthreadcontext)<br/> | Mit dieser optionalen Schnittstelle kann der Protokollhandler eine Aktion für den Thread ausführen, der für die Filterung im Protokollhost verwendet wird. Wenn der Protokollhost gestartet wird, initialisiert er zunächst alle Protokollhandler und erstellt dann die Filterthreads. Die Methoden auf dieser Schnittstelle ermöglichen Protokollhandlern die Verwaltung ihrer Ressourcen, die von einem Filterthread verwendet werden.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**ISearchProtocolUI**](-search-isearchprotocolui.md)<br/>                       | Stellt eine Methode zum Aufrufen von [**ISearchItem-Objekten**](-search-isearchitem.md) bereit. Methoden in dieser Schnittstelle werden vom Protokollhost aufgerufen, wenn URLs vom Gatherer verarbeitet werden. Der Protokollhandler implementiert das Protokoll für den Zugriff auf eine Inhaltsquelle im systemeigenen Format, und diese Schnittstelle implementiert einen benutzerdefinierten Protokollhandler, um die Datenquellen zu erweitern, die indiziert werden können.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor)<br/>                                 | Stellt Methoden zum Verarbeiten eines einzelnen Elements in einer Inhaltsquelle bereit, dessen URL vom Gatherer für den Filterhost bereitgestellt wird. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IUrlAccessor2**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor2)<br/>                               | Erweitert die Funktionalität der [**IUrlAccessor-Schnittstelle.**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IUrlAccessor3**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor3)<br/>                               | Erweitert die Funktionalität der [**IUrlAccessor2-Schnittstelle**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor2) um die [**IUrlAccessor3::GetImpersonationSidBlobs-Methode,**](/windows/desktop/api/Searchapi/nf-searchapi-iurlaccessor3-getimpersonationsidblobs) um Benutzersicherheits-IDs (SIDs) für eine angegebene URL zu identifizieren.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**IUrlAccessor4**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor4)<br/>                               | Erweitert die Funktionalität der [**IUrlAccessor3-Schnittstelle**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor3) um die [**IUrlAccessor4::ShouldIndexItemContent-Methode,**](/windows/desktop/api/Searchapi/nf-searchapi-iurlaccessor4-shouldindexitemcontent) die angibt, ob der Inhalt des Elements indiziert werden soll. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
