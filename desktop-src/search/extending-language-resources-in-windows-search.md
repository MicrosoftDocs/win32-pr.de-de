# <a name="extending-language-resources"></a>Erweitern von Sprachressourcen

Windows Die Suche verwendet Sprachressourcen wie Wörter wörterbrechende Wörter und Wortstammzeilen, um Text im nativen Locale während der Indexerstellung und Abfrageverarbeitung zu unterbricht. Microsoft stellt Wörter wörter- und wortstammende Wörter für mehrere Sprachen zur Verfügung. In diesem Abschnitt wird beschrieben, wie Sie benutzerdefinierte Wörterschnitte und Wortstammstammnoten für Sprachen und Lokalien implementieren und verwenden, die über die von Microsoft bereitgestellten Hinausgehen hinausgehen.

-   [Grundlegendes zu Sprachressourcenkomponenten](understanding-language-resource-components.md)
-   [Implementieren einer Wörterbruch- und Wortstammnotung](implementing-a-word-breaker-and-stemmer.md)
-   [Linguistische und Unicode-Überlegungen](linguistic-and-unicode-considerations.md)
-   [Problembehandlung für Sprachressourcen und bewährte Methoden](troubleshooting-language-resources.md)
