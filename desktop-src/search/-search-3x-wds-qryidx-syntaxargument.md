# <a name="syntax-argument-windows-search"></a>Syntax Argument (Windows-Suche)

Das- `syntax` Argument ermöglicht es Ihnen, die Syntax der Abfrage zu identifizieren. Zulässige Werte sind entweder eine natürliche Abfrage Syntax (NQS) oder eine erweiterte Abfrage Syntax (AQS). AQS ist die Standardeinstellung und wird immer als analysiert und unterstützt angenommen.

Dieses Thema ist wie folgt organisiert:

-   [Beispiel](#example)
-   [Zugehörige Themen](#related-topics)

## <a name="example"></a>Beispiel


```
 search-ms:query=from:paolo&syntax:AQS
```
