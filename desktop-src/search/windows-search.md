# <a name="windows-search"></a>Windows-Suche

Windows Search ist eine Plattform für die Desktop Suche, die über sofortige Suchfunktionen für die meisten gängigen Datei-und Datentypen verfügt, z. b. e-Mail, Kontakte, Kalendertermine, Dokumente, Fotos, Multimedia und andere Formate, die von Entwicklern von Drittanbietern erweitert werden können. Diese Funktionen ermöglichen es Benutzern, die zunehmende Menge von Daten zu finden, zu verwalten und zu organisieren, die in Heim-und Unternehmensumgebungen üblich sind.
