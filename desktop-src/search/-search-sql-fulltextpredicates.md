# <a name="full-text-predicates"></a>Full-Text Prädikate

Die Microsoft Windows Search-Abfragesprache unterstützt zwei Prädikate für die Volltextsuche. Das CONTAINS-Prädikat führt Vergleiche für Spalten aus, die Text enthalten. Die CONTAINS-Klausel kann basierend auf der Nähe der Suchbegriffe einen Abgleich für einzelne Wörter oder Ausdrücke durchführen. Im Vergleich dazu wird das FREETEXT-Prädikat auf die Bedeutung der Suchbegriffe für Textspalten abgestimmt.

Dieser Abschnitt enthält die folgenden Themen:

-   [CONTAINS-Prädikat](-search-sql-contains.md)
-   [FREETEXT-Prädikat](-search-sql-freetext.md)
