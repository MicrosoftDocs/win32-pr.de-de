# <a name="full-text-predicates"></a>Full-Text Prädikate

Die Microsoft Windows Search-Abfragesprache unterstützt zwei Prädikate der Volltextsuche. Das enthält-Prädikat führt Vergleiche bei Spalten aus, die Text enthalten. Die enthält-Klausel kann basierend auf der Nähe der Suchbegriffe Übereinstimmungen mit einzelnen Wörtern oder Ausdrücken ausführen. Im Vergleich dazu wird das frei Text Prädikat entsprechend der Bedeutung der Such Ausdrücke für Textspalten optimiert.

Dieser Abschnitt enthält die folgenden Themen:

-   [Enthält Prädikat](-search-sql-contains.md)
-   [Frei Text Prädikat](-search-sql-freetext.md)
