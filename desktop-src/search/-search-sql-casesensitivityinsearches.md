# <a name="case-sensitivity-in-searches"></a>Groß-/Kleinschreibung in suchen

Bei Microsoft Windows Search-Abfragen wird Groß-/Kleinschreibung nicht beachtet. Dies schließt Abfrage Schlüsselwörter ein (Select ist identisch mit SELECT, SELECT und SELECT) und Suchbegriffe. Wenn ein Bezeichner oder Literale Unicode-Zeichen verwendet, die die Semantik für die Groß-/Kleinschreibung aufweisen, betrachtet die Suchengine alle Fälle als äquivalent.
