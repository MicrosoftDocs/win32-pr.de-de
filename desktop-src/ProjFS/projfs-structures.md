# <a name="projfs-structures"></a>ProjFS-Strukturen

Die folgenden Strukturen werden in projectedfslib.h deklariert.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | Beschreibung |
|-|-|
| [**PRJ_CALLBACK_DATA**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_callback_data) | Definiert die Standardinformationen, die für jeden Vorgangsrückruf an einen Anbieter übergeben werden. |
| [**PRJ_CALLBACKS**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_callbacks) | Ein Satz von Zeigern auf den Ort, an dem der Anbieter seine Implementierungen der Rückrufroutinen speichert. |
| [**PRJ_COMPLETE_COMMAND_EXTENDED_PARAMETERS**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_complete_command_extended_parameters) | Gibt parameter an, die zum Abschließen bestimmter Rückrufe erforderlich sind. |
| [**PRJ_FILE_BASIC_INFO**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_file_basic_info) | Grundlegende Informationen zu einem Element. |
| [**PRJ_NOTIFICATION_MAPPING**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_notification_mapping) | Beschreibt eine Benachrichtigungszuordnung, bei der es sich um eine Kopplung zwischen einem Verzeichnis (als "Benachrichtigungsstamm" bezeichnet) und einer Reihe von Benachrichtigungen handelt, die als Bitmaske ausgedrückt werden. |
| [**PRJ_NOTIFICATION_PARAMETERS**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_notification_parameters) | Zusätzliche Parameter für Benachrichtigungen. |
| [**PRJ_PLACEHOLDER_INFO**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_placeholder_info) | Ein Puffer mit Metadaten für die Platzhalterdatei oder das Platzhalterverzeichnis. |
| [**PRJ_PLACEHOLDER_VERSION_INFO**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_placeholder_version_info) | Informationen, die den Inhalt einer Platzhalterdatei eindeutig identifizieren. |
| [**PRJ_STARTVIRTUALIZING_OPTIONS**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_startvirtualizing_options) | Optionen, die beim Starten einer Virtualisierungsinstanz zur Verfügung zu stellen sind. |
| [**PRJ_VIRTUALIZATION_INSTANCE_INFO**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_virtualization_instance_info) | Informationen zu einer Virtualisierungsinstanz. |
