# <a name="projfs-structures"></a>Projfs-Strukturen

Die folgenden Strukturen sind in projectedfslib. h deklariert.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [**PRJ_CALLBACK_DATA**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_callback_data) | Definiert die Standardinformationen, die an einen Anbieter für jeden Vorgangs Rückruf übermittelt werden. |
| [**PRJ_CALLBACKS**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_callbacks) | Ein Satz von Zeigern, auf die der Anbieter seine Implementierungen der Rückruf Routinen speichert. |
| [**PRJ_COMPLETE_COMMAND_EXTENDED_PARAMETERS**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_complete_command_extended_parameters) | Gibt Parameter an, die erforderlich sind, um bestimmte Rückrufe abzuschließen. |
| [**PRJ_FILE_BASIC_INFO**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_file_basic_info) | Grundlegende Informationen zu einem Element. |
| [**PRJ_NOTIFICATION_MAPPING**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_notification_mapping) | Beschreibt eine Benachrichtigungs Zuordnung, bei der es sich um eine Kopplung zwischen einem Verzeichnis (als "Benachrichtigungs Stamm" bezeichnet) und einem Satz von Benachrichtigungen handelt, ausgedrückt als Bitmaske. |
| [**PRJ_NOTIFICATION_PARAMETERS**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_notification_parameters) | Zusätzliche Parameter für Benachrichtigungen. |
| [**PRJ_PLACEHOLDER_INFO**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_placeholder_info) | Ein metadatenpuffer für die Platzhalter Datei bzw. das Verzeichnis. |
| [**PRJ_PLACEHOLDER_VERSION_INFO**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_placeholder_version_info) | Informationen, die den Inhalt einer Platzhalter Datei eindeutig identifizieren. |
| [**PRJ_STARTVIRTUALIZING_OPTIONS**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_startvirtualizing_options) | Optionen, die beim Starten einer virtualisierungsinstanz bereitgestellt werden sollen. |
| [**PRJ_VIRTUALIZATION_INSTANCE_INFO**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_virtualization_instance_info) | Informationen zu einer virtualisierungsinstanz. |
