# <a name="provider-overview"></a><span data-ttu-id="caf04-103">Übersicht über den Anbieter</span><span class="sxs-lookup"><span data-stu-id="caf04-103">Provider Overview</span></span>

## <a name="data-projection"></a><span data-ttu-id="caf04-107">Datenprojektion</span><span class="sxs-lookup"><span data-stu-id="caf04-107">Data Projection</span></span>

<span data-ttu-id="caf04-108">Der Teil des Dateisystems, dem der Anbieter gehört, dessen Daten projiziert werden, befindet sich in einem Verzeichnis namens "virtualisierungsroot".</span><span class="sxs-lookup"><span data-stu-id="caf04-108">The part of the file system that the provider owns, where its data is projected, is rooted in a directory called the “virtualization root”.</span></span>  <span data-ttu-id="caf04-109">Wenn der Anbieter seine Daten projizieren möchte, startet er eine "virtualisierungsinstanz", bei der es sich um ein Objekt handelt, das die class="sxs-lookup"><span data-stu-id="caf04-109">When the provider wants to start projecting its data it starts a "virtualization instance", which is an object that manages communication between the virtualisierungsstamms sind, die nicht lokal vom Benutzer erstellt wurden, werden vom Anbieter über die projfs-API materialisiert.</span><span class="sxs-lookup"><span data-stu-id="caf04-110">Any data-ttu-id="caf04-111">Diese Elemente werden als virtuelle Dateien und Verzeichnisse gestartet, was bedeutet, dass Sie nicht auf dem lokalen Speichergerät des Benutzers vorhanden sind, sondern in die enumerationsergebnisse von projfs eingefügt werden.</span><span class="sxs-lookup"><span data-stu-id="caf04-111">These items start off as virtual files and directories, meaning that they do not exist nachfolgenden Zugriff zwischengespeichert wird.</span><span class="sxs-lookup"><span data-stu-id="caf04-112">As the items are opened and read, ProjFS invokes callbacks implemented by the provider to löschen, um die neue Ansicht des Datenspeicher widerzuspiegeln.</span><span class="sxs-lookup"><span data-stu-id="caf04-113">If the user's view of the backing data store needs to change, for instance
