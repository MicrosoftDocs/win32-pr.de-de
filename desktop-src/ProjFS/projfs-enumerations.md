# <a name="projfs-enumerations"></a>Projfs-Enumerationen

Die folgenden Enumerationen werden in projectedfslib. h deklariert.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [**PRJ_CALLBACK_DATA_FLAGS**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_callback_data_flags) | Flags, die Steuern, was in der-Enumeration zurückgegeben wird. |
| [**PRJ_COMPLETE_COMMAND_TYPE**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_complete_command_type) | Gibt Befehls Typen an. |
| [**PRJ_FILE_STATE**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_file_state) | Der Zustand eines Elements. |
| [**PRJ_NOTIFICATION**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_notification) | Ein Benachrichtigungs Wert, der beim Senden der Benachrichtigung in einem Rückruf angegeben wird. |
| [**PRJ_NOTIFY_TYPES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_notify_types) | Benachrichtigungs Typen, die eine Änderung der Datei oder des Ordners beschreiben. |
| [**PRJ_PLACEHOLDER_ID**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_placeholder_id) | Definiert die Länge eines Platzhalter Bezeichners. |
| [**PRJ_STARTVIRTUALIZING_FLAGS**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_startvirtualizing_flags) | Flags, die beim Starten einer virtualisierungsinstanz bereitgestellt werden sollen. |
| [**PRJ_UPDATE_FAILURE_CAUSES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_update_failure_causes) | Beschreibungen für den Grund, warum ein Update fehlgeschlagen ist. |
| [**PRJ_UPDATE_TYPES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_update_types) | Flags, die angeben, ob Updates für den Zustand einer Datei oder eines Verzeichnisses auf dem Datenträger zulässig sind. |
