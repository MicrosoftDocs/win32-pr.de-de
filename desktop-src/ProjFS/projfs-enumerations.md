# <a name="projfs-enumerations"></a>ProjFS-Enumerationen

Die folgenden Enumerationen werden in projectedfslib.h deklariert.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [**PRJ_CALLBACK_DATA_FLAGS**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_callback_data_flags) | Flags, die steuern, was in der -Enumeration zurückgegeben wird. |
| [**PRJ_COMPLETE_COMMAND_TYPE**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_complete_command_type) | Gibt Befehlstypen an. |
| [**PRJ_FILE_STATE**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_file_state) | Der Zustand eines Elements. |
| [**PRJ_NOTIFICATION**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_notification) | Ein Benachrichtigungswert, der beim Senden der Benachrichtigung in einem Rückruf angegeben wird. |
| [**PRJ_NOTIFY_TYPES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_notify_types) | Typen von Benachrichtigungen, die eine Änderung an der Datei oder dem Ordner beschreiben. |
| [**PRJ_PLACEHOLDER_ID**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_placeholder_id) | Definiert die Länge eines Platzhalterbezeichners. |
| [**PRJ_STARTVIRTUALIZING_FLAGS**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_startvirtualizing_flags) | Flags, die beim Starten einer Virtualisierungsinstanz zur Verfügung stellen. |
| [**PRJ_UPDATE_FAILURE_CAUSES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_update_failure_causes) | Beschreibungen der Ursache, warum ein Update fehlgeschlagen ist. |
| [**PRJ_UPDATE_TYPES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_update_types) | Flags, um anzugeben, ob Updates zulässig sind, wenn der Status einer Datei oder eines Verzeichnisses auf dem Datenträger angegeben wird. |
