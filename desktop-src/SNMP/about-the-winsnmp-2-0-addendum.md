# <a name="about-the-winsnmp-20-addendum"></a>Informationen zum WinSNMP 2.0-Nachtrag

Das WinSNMP-Nachtrag der Version 2.0 ist ein Update der WinSNMP-Manager-API-Spezifikation, Version 1.1a. (Derzeit definiert keine Spezifikation Version 2.0 der WinSNMP-API.) Der Nachtrag wurde gemeinsam mit der Zusammenarbeit, Eingabe und Unterstützung von mehreren Unternehmen, Vereinigungen und Einzelpersonen entwickelt.

Die WinSNMP 1.1a-Spezifikation definiert die WinSNMP Manager-Anwendungsprogrammierschnittstelle für die Entwicklung von SNMP-basierten Netzwerkverwaltungsanwendungen. Im WinSNMP-Nachtrag der Version 2.0 wird die Schnittstelle als WinSNMP-API umbenannt. Dies liegt daran, dass WinSNMP 2.0 sowohl Agent- als auch Manager-SNMP-Vorgänge unterstützt. Sie sollten mit beiden Dokumenten vertraut sein, wenn Sie in der WinSNMP-Umgebung programmieren.

Die Microsoft WinSNMP-Implementierung ist mit WinSNMP 2.0 kompatibel. Sie unterstützt alle Funktionen und Vorgänge, die sowohl in der WinSNMP 1.1a-Spezifikation als auch im WinSNMP 2.0-Nachtrag definiert sind.
