# <a name="about-the-winsnmp-20-addendum"></a>Informationen zum WinSNMP 2,0-Nachtrag

Der Nachtrag von WinSNMP, Version 2,0, ist ein Update der WinSNMP-Manager-API-Spezifikation, Version 1.1 a. (Zu diesem Zeitpunkt definiert keine Spezifikation Version 2,0 der WinSNMP-API.) Der Nachtrag wurde gemeinsam mit der Zusammenarbeit, der Eingabe und der Unterstützung von mehreren Unternehmen, Zuordnungen und Einzelpersonen entwickelt.

Mit der Spezifikation WinSNMP 1.1 a wird die Anwendungsprogrammierschnittstelle WinSNMP-Manager zum Entwickeln von SNMP-basierten Netzwerk Verwaltungs Anwendungen definiert. Im Nachtrag von WinSNMP, Version 2,0, wird die-Schnittstelle als WinSNMP-API umbenannt. Dies liegt daran, dass WinSNMP 2,0 sowohl Agent-als auch Manager-SNMP-Vorgänge unterstützt. Wenn Sie in der WinSNMP-Umgebung programmieren, sollten Sie mit beiden Dokumenten vertraut sein.

Die Microsoft WinSNMP-Implementierung ist mit WinSNMP 2,0 kompatibel. Es werden alle Funktionen und Vorgänge unterstützt, die sowohl in der WinSNMP 1.1-Spezifikation als auch in der WinSNMP 2,0-Nachtrag definiert sind.
