# <a name="snmp-structures"></a>SNMP-Strukturen

\[SNMP ist für die Verwendung in den im Abschnitt Anforderungen angegebenen Betriebssystemen verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden. Verwenden Sie stattdessen [Windows Remoteverwaltung,](/windows/desktop/WinRM/portal)bei der es sich um die Microsoft-Implementierung von WS-Man handelt.\]

Die folgende Tabelle enthält Strukturen, die mit SNMP verwendet werden.



| SNMP-Struktur                                         | BESCHREIBUNG                                                                                             |
|--------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [**AsnAny**](/windows/desktop/api/Snmp/ns-snmp-asnany)                           | Beschreibt den Typ und den Wert einer angegebenen SNMP-Variablen.                                              |
| [**AsnCounter64**](/previous-versions/windows/desktop/legacy/aa377953(v=vs.85))               | Enthält einen 64-Bit-Zähler, der von zwei Werten angegeben wird, die seine niedrigen und hohen Bits beschreiben. |
| [**AsnObjectIdentifier**](/windows/desktop/api/Snmp/ns-snmp-asnobjectidentifier) | Beschreibt einen Objektbezeichner (OID).                                                                   |
| [**AsnOctetString**](/windows/desktop/api/Snmp/ns-snmp-asnoctetstring)           | Stellt eine Zeichenfolge von Oktetten dar, in der Regel in Bytewerten.                                                  |
| [**SnmpVarBind**](/windows/desktop/api/Snmp/ns-snmp-snmpvarbind)                 | Beschreibt eine SNMP-Variablenbindung.                                                                     |
| [**SnmpVarBindList**](/windows/desktop/api/Snmp/ns-snmp-snmpvarbindlist)         | Enthält eine Liste von SNMP-Variablenbindungen.                                                              |
