# <a name="snmp-structures"></a>SNMP-Strukturen

\[SNMP ist für die Verwendung in den Betriebssystemen verfügbar, die im Abschnitt "Anforderungen" angegeben sind. Es kann in nachfolgenden Versionen geändert oder entfernt werden. Verwenden Sie stattdessen [Windows-Remoteverwaltung](/windows/desktop/WinRM/portal), bei dem es sich um die Microsoft-Implementierung von WS-man handelt.\]

In der folgenden Tabelle sind die-Strukturen aufgelistet, die mit SNMP verwendet werden.



| SNMP-Struktur                                         | BESCHREIBUNG                                                                                             |
|--------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [**Asnany**](/windows/desktop/api/Snmp/ns-snmp-asnany)                           | Beschreibt den Typ und den Wert einer angegebenen SNMP-Variablen.                                              |
| [**AsnCounter64**](/previous-versions/windows/desktop/legacy/aa377953(v=vs.85))               | Enthält einen 64-Bit-Leistungs Bewert, der durch zwei Werte angegeben wird, die die nieder wertigen und höherwertigen Bits beschreiben. |
| [**Asnobjectidentifier**](/windows/desktop/api/Snmp/ns-snmp-asnobjectidentifier) | Beschreibt einen Objekt Bezeichner (OID).                                                                   |
| [**Asnoctetstring**](/windows/desktop/api/Snmp/ns-snmp-asnoctetstring)           | Stellt eine Zeichenfolge von Oktetten dar, normalerweise in Byte Werten.                                                  |
| [**Snmpvarbind**](/windows/desktop/api/Snmp/ns-snmp-snmpvarbind)                 | Beschreibt eine SNMP-Variablen Bindung.                                                                     |
| [**Snmpvarbindlist**](/windows/desktop/api/Snmp/ns-snmp-snmpvarbindlist)         | Enthält eine Liste von SNMP-Variablen Bindungen.                                                              |
