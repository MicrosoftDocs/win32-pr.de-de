# <a name="winsnmp-data-types"></a>WinSNMP-Datentypen

\[SNMP ist für die Verwendung in den Betriebssystemen verfügbar, die im Abschnitt "Anforderungen" angegeben sind. Es kann in nachfolgenden Versionen geändert oder entfernt werden. Verwenden Sie stattdessen [Windows-Remoteverwaltung](/windows/desktop/WinRM/portal), bei dem es sich um die Microsoft-Implementierung von WS-man handelt.\]

Die standardmäßigen WinSNMP-Datentypen werden in WinSNMP definiert. H-Datei.

Beachten Sie, dass WinSNMP einige Parameter mit dem Typ "Long Integer" mit Vorzeichen, **smiint**, angibt. Dies ermöglicht es WinSNMP, die Datenelemente zu erfüllen, insbesondere die PDU-Komponenten, die in den entsprechenden RFCs definiert sind.
