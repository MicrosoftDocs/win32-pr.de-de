# <a name="configuring-snmp-agent-information"></a>Konfigurieren von SNMP-Agent-Informationen

Mithilfe von SNMP-Agent-Informationen können Sie Kommentare zum Benutzer und den physischen Speicherort des Computers angeben und die Dienst Typen angeben, die gemeldet werden sollen. Die Dienst Typen, die gemeldet werden können, basieren auf der Konfiguration des Computers.
