# <a name="configuring-snmp-agent-information"></a>Konfigurieren von SNMP-Agent-Informationen

Mit SNMP-Agent-Informationen können Sie Kommentare zum Benutzer und zum physischen Speicherort des Computers angeben und die Zu meldende Dienstart angeben. Die Typen von Diensten, die gemeldet werden können, basieren auf der Konfiguration des Computers.
