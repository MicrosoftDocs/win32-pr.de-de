# <a name="winsnmp-sessions"></a>WinSNMP-Sitzungen

Eine *Sitzung* ist die grundlegende Einheit der Ressourcen-und Kommunikations Verwaltung zwischen einer aufrufenden WinSNMP-Anwendung und der Microsoft WinSNMP-Implementierung. Es wird empfohlen, dass eine Anwendung Sitzungen verwendet, um Ihre Vorgänge zu organisieren und den Bedarf an den Ressourcen der Implementierung zu verringern.

Weitere Informationen finden Sie unter [Öffnen und Schließen einer WinSNMP-Sitzung](opening-and-closing-a-winsnmp-session.md).
