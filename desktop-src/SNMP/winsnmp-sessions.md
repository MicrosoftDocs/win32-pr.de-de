# <a name="winsnmp-sessions"></a>WinSNMP-Sitzungen

Eine *Sitzung* ist die grundlegende Einheit der Ressourcen- und Kommunikationsverwaltung zwischen einer aufrufenden WinSNMP-Anwendung und der Microsoft WinSNMP-Implementierung. Es wird empfohlen, dass eine Anwendung Sitzungen verwendet, um ihre Vorgänge zu organisieren und den Bedarf an den Ressourcen der Implementierung zu reduzieren.

Weitere Informationen finden Sie unter [Öffnen und Schließen einer WinSNMP-Sitzung.](opening-and-closing-a-winsnmp-session.md)
