# <a name="rfcs-relevant-to-winsnmp"></a>Für WinSNMP relevante RFCs

TCP/IP-Standards werden in Anforderungen für Kommentare (RFCs) definiert, die von der Internet Engineering Task Force (IETF) veröffentlicht werden. Die RFCs, die für die WinSNMP-Funktionen relevant sind, sind in der folgenden Tabelle aufgeführt.



| RFC-Nummer | Titel                                                                                                  |
|------------|--------------------------------------------------------------------------------------------------------|
| 1155       | "Struktur und Identifizierung von Verwaltungsinformationen für TCP/IP-basiertes Internet"                    |
| 1157       | "A Simple Network Management Protocol (SNMP)"                                                          |
| 1213       | "Verwaltungs Informationsbasis für die Netzwerkverwaltung von TCP/IP-basierten Internet Informationsnetzen: MIB-II"                 |
| 1901       | "Einführung in Community-based SNMPv2"                                                               |
| 1902       | "Struktur der Verwaltungsinformationen für Version 2 des Simple Network Management-Protokolls (SNMPv2)" |
| 1903       | "Textkonventionen für Version 2 des Simple Network Management-Protokolls (SNMPv2)"                 |
| 1904       | "Konformitäts Anweisungen für Version 2 des Simple Network Management-Protokolls (SNMPv2)"              |
| 1905       | "Protokoll Vorgänge für Version 2 des Simple Network Management-Protokolls (SNMPv2)"                 |
| 1906       | "Transport Zuordnungen für Version 2 des Simple Network Management-Protokolls (SNMPv2)"                  |
| 1907       | "Verwaltungs Informationsbasis für Version 2 des Simple Network Management-Protokolls (SNMPv2)"         |
| 1908       | "Koexistenz zwischen Version 1 und Version 2 des Internet-Standard-Netzwerk Verwaltungs-Frameworks"    |
| 2089       | "V2ToV1 Mapping SNMPv2 auf SNMPv1 in einem bidirektionalen SNMP-Agent"                                     |
