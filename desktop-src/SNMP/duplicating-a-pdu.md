# <a name="duplicating-a-pdu"></a>Duplizieren eines PDU

Die [**snmpdupligenepdu**](/windows/desktop/api/Winsnmp/nf-winsnmp-snmpduplicatepdu) -Funktion dupliziert ein PDU, wobei der erforderliche Arbeitsspeicher zugeordnet wird. Um von **snmpdupliaseepdu** zugeordnete Ressourcen für ein neues PDU freizugeben, muss eine WinSNMP-Anwendung die Funktion [**snmpfreepdu**](/windows/desktop/api/Winsnmp/nf-winsnmp-snmpfreepdu) aufgerufen haben.
