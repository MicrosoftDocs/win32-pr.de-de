# <a name="about-snmp-versions"></a>Informationen zu SNMP-Versionen

Das ursprüngliche Internetstandard-Netzwerkverwaltungsframework, das unter RFCs 1155, 1157 und 1213 beschrieben wird, wird als SNMP-Framework der Version 1 (SNMPv1) bezeichnet. Relevante Teile des vorgeschlagenen Frameworks für Version 2C des Simple Network Management Protocol (SNMPv2C) werden in RFCs 1901 bis 1908 beschrieben.

Die WinSNMP-API unterstützt die in den relevanten Internet-RFCs beschriebene SNMP-Protokollfunktionalität. WinSNMP stellt keine Einschränkungen für die Verwendung von SNMPv1- oder SNMPv2C durch WinSNMP-Anwendungen.

Eine Verwaltungsentität kann eine andere Version von SNMP unterstützen als die, die von der WinSNMP-Anwendung unterstützt wird. Die Microsoft WinSNMP-Implementierung führt die entsprechenden Übersetzungen von SNMPv1 in SNMPv2C in Übereinstimmung mit der relevanten RFC durch.
