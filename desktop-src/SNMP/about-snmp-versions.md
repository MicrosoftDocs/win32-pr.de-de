# <a name="about-snmp-versions"></a>Informationen zu SNMP-Versionen

Das ursprüngliche Internet Standard Network Management-Framework, das in RFCs 1155, 1157 und 1213 beschrieben wird, wird als SNMP-Framework Version 1 (SNMPv1) bezeichnet. Relevante Teile des vorgeschlagenen Frameworks für Version 2C des Simple Network Management-Protokolls (SNMPv2C) werden in RFCs 1901 bis 1908 beschrieben.

Die WinSNMP-API unterstützt die SNMP-Protokollfunktionen, die in den relevanten Internet-RFCs beschrieben werden. WinSNMP stellt keine Einschränkungen bei der Verwendung von SNMPv1 oder SNMPv2C von WinSNMP-Anwendungen her.

Eine Verwaltungs Entität kann eine andere Version von SNMP unterstützen, als die, die die WinSNMP-Anwendung unterstützt. Die Microsoft WinSNMP-Implementierung führt die entsprechenden Übersetzungen von SNMPv1 bis SNMPv2C in Übereinstimmung mit der relevanten RFC aus.
