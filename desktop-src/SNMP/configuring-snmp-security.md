# <a name="configuring-snmp-security"></a>Konfigurieren der SNMP-Sicherheit

Mit den Sicherheitsfeatures von SNMP können Sie die Communitys und Hosts angeben, von denen ein Computeranforderungen annimmt, sowie die Art der Vorgänge, die von den Computern, die zu einer Community gehören, akzeptiert werden sollen. Mit SNMP können Sie auch angeben, ob ein Authentifizierungs Trap gesendet werden soll, wenn eine nicht autorisierte Community oder ein Hostinformationen anfordert.

Weitere Informationen finden Sie im Windows Resource Kit und in der Online Dokumentation für den SNMP-Dienst.

**Windows Server 2003:** Standardmäßig sind die Communitynamen nicht konfiguriert, und der SNMP-Dienst ist so konfiguriert, dass SNMP-Anforderungen nur vom Hostnamen "localhost" akzeptiert werden.
