# <a name="configuring-snmp-security"></a>Konfigurieren der SNMP-Sicherheit

Mit den Sicherheitsfeatures von SNMP können Sie die Communitys und Hosts angeben, von denen ein Computer Anforderungen akzeptiert, sowie die Art der Vorgänge, die von den Computern akzeptiert werden sollen, die zu einer Community gehören. Mit SNMP können Sie auch angeben, ob eine Authentifizierungsfalle gesendet werden soll, wenn eine nicht autorisierte Community oder ein Host Informationen anfordert.

Weitere Informationen finden Sie im Windows Resource Kit und in der Onlinedokumentation zum SNMP-Dienst.

**Windows Server 2003:** Standardmäßig sind Communitynamen nicht konfiguriert, und der SNMP-Dienst ist so konfiguriert, dass er nur SNMP-Anforderungen vom Hostnamen "localhost" akzeptiert.
