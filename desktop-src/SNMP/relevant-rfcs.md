# <a name="relevant-rfcs"></a>Relevante RFCs

TCP/IP-Standards werden unter Requests for Comments (RFCs) definiert, die von der Internet Engineering Task Force (IETF) veröffentlicht werden. In der folgenden Tabelle sind die RFCs aufgeführt, die für SNMP relevant sind.



| RFC-Nummer | Titel                                                                                                           |
|------------|-----------------------------------------------------------------------------------------------------------------|
| 1155       | "Struktur und Identifikation von Verwaltungsinformationen für TCP/IP-basierte Internets". Er definiert SMI. Mib.        |
| 1157       | "A Simple Network Management Protocol (SNMP)". Er definiert SNMP selbst.                                          |
| 1213       | "Verwaltungsinformationsbasis für die Netzwerkverwaltung von TCP/IP-basierten Internets: MIB-II." Sie definiert MIB \_ II.MIB. |
| 1901       | "Einführung in Community SNMPv2"                                                                        |
| 1902       | "Structure of Management Information for Version 2 of the Simple Network Management Protocol (SNMPv2)" (Struktur der Verwaltungsinformationen für Version 2 des Simple Network Management Protocol (SNMPv2))          |
| 1903       | "Textkonventionen für Version 2 des Simple Network Management Protocol (SNMPv2)"                          |
| 1904       | "Konformitätserklärungen für Version 2 des Simple Network Management Protocol (SNMPv2)"                       |
| 1905       | "Protokollvorgänge für Version 2 des Simple Network Management Protocol (SNMPv2)"                          |
| 1906       | "Transport mappings for Version 2 of the Simple Network Management Protocol (SNMPv2)" (Transportzuordnungen für Version 2 des Simple Network Management Protocol (SNMPv2))                           |
| 1907       | "Verwaltungsinformationsbasis für Version 2 des Simple Network Management Protocol (SNMPv2)"                  |
| 1908       | "Koexistenz zwischen Version 1 und Version 2 des Netzwerkverwaltungsframework nach Internetstandard"             |
| 2089       | "V2ToV1 Mapping SNMPv2 to SNMPv1 within a bi-lingual SNMP agent" (V2ToV1: Zuordnen von SNMPv2 zu SNMPv1 innerhalb eines bisprachlichen SNMP-Agents)                                              |
