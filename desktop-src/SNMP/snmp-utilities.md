# <a name="snmp-utilities"></a>SNMP-Hilfsprogramme

In der folgenden Tabelle sind die SNMP-Hilfsprogramme aufgelistet, die im Microsoft Windows Resource Kit verfügbar sind.



| Dateiname     | BESCHREIBUNG                                                                             |
|--------------|-----------------------------------------------------------------------------------------|
| EVNTCMD.EXE  | Eine Befehlszeilen Anwendung zum Konfigurieren des SNMP-Ereignis Konvertierers.                    |
| EVNTWIN.EXE  | Eine Anwendung, die eine Benutzeroberfläche zum Konfigurieren des SNMP-Ereignis Konvertierers bereitstellt. |
| MIBCC.EXE    | Der SNMP-MIB-Compiler                                                                   |
| SNMPUTIL.EXE | Ein Beispiel für eine SNMP-Manager-Konsolenanwendung                                               |



 

Der SNMP-Anbieter im Windows-Verwaltungsinstrumentation (WMI) SDK ermöglicht Client Anwendungen den Zugriff auf statische und dynamische SNMP-Informationen über WMI. Weitere Informationen finden Sie unter [SNMP-Anbieter](/windows/desktop/WmiSdk/snmp-provider) in der WMI SDK-Dokumentation.
