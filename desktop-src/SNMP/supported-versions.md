# <a name="supported-versions"></a>Unterstützte Versionen

\[SNMP ist für die Verwendung in den Betriebssystemen verfügbar, die im Abschnitt "Anforderungen" angegeben sind. Es kann in nachfolgenden Versionen geändert oder entfernt werden. Verwenden Sie stattdessen [Windows-Remoteverwaltung](/windows/desktop/WinRM/portal), bei dem es sich um die Microsoft-Implementierung von WS-man handelt.\]

Windows Vista und Windows Server 2008 unterstützen SNMPv1 und SNMPv2C. Dieses Thema enthält grundlegende Informationen zu jeder dieser Versionen von SNMP.

Das Simple Network Management-Protokoll, Version 1 (SNMPv1), ist die früheste Version des Protokolls. SNMPv1 wird von RFC 1157, RFC 1155 und RFC 1212 definiert. Für SNMPv1 basiert die Sicherheit auf Community-Zeichen folgen.

Das Simple Network Management-Protokoll, Version 2C (SNMPv2C), wird von RFC 1901, RFC 1905 und RFC 1906 definiert. SNMPv2C aktualisiert die Protokoll Vorgänge und Datentypen, die in SNMPv2p definiert wurden. SNMPv2C, bekannt als "Community String-based SNMPv2", verwendet die Community-basierte Sicherheit von SNMPv1.

SNMPv2x wird auf Client-und Server-Betriebssystemplattformen unter Windows 2000 und höher unterstützt, darunter Windows XP und Windows Server 2003.
