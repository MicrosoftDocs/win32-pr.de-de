# <a name="supported-versions"></a>Unterstützte Versionen

\[SNMP ist für die Verwendung in den Betriebssystemen verfügbar, die im Abschnitt Anforderungen angegeben sind. Es kann in nachfolgenden Versionen geändert oder entfernt werden. Verwenden Sie stattdessen [Windows Remoteverwaltung,](/windows/desktop/WinRM/portal)bei der es sich um die Microsoft-Implementierung von WS-Man handelt.\]

Windows Vista und Windows Server 2008 unterstützen SNMPv1 und SNMPv2c. Dieses Thema enthält grundlegende Informationen zu jeder dieser Versionen von SNMP.

Simple Network Management Protocol Version 1 (SNMPv1) ist die früheste Version des Protokolls. SNMPv1 wird durch RFC 1157, RFC 1155 und RFC 1212 definiert. Für SNMPv1 basiert die Sicherheit auf Communityzeichenfolgen.

Simple Network Management Protocol Version 2c (SNMPv2c) wird durch RFC 1901, RFC 1905 und RFC 1906 definiert. SNMPv2c aktualisiert die Protokollvorgänge und Datentypen, die in SNMPv2p definiert wurden. SNMPv2c, bekannt als "community string-based SNMPv2", verwendet die communitybasierte Sicherheit von SNMPv1.

SNMPv2x wird auf Windows 2000- und höher-Client- und Serverbetriebssystemplattformen unterstützt, einschließlich Windows XP und Windows Server 2003.
