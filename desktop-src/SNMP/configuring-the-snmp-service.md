# <a name="configuring-the-snmp-service"></a>Konfigurieren des SNMP-Dienstanbieter

Zum Konfigurieren des SNMP-Diensts müssen Sie die Communitynamen kennen, die vom Netzwerk verwendet werden, das Trap-Ziel für jede Community sowie den Computernamen oder die IP-Adresse für jeden SNMP-Verwaltungs Host.

In der folgenden Tabelle sind Themen aufgeführt, die Informationen zum Konfigurieren des SNMP-Dienstanbieter enthalten.

| Thema                                                                        | Inhalte                                                                                                                              |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [Communitynamen](community-names.md)                                       | Beschreibt die Verwendung von Communitynamen in SNMP.                                                                                         |
| [Hostnamen und IP-Adressen](host-names-and-ip-addresses.md)               | Beschreibt die Verwendung von Hostnamen und IP-Adressen in SNMP.                                                                             |
| [Konfigurieren der SNMP-Sicherheit](configuring-snmp-security.md)                   | Beschreibt die SNMP-Sicherheitsfunktionen.                                                                                                     |
| [Konfigurieren von SNMP-Gruppenrichtlinie](configuring-snmp-group-policy.md)           | Beschreibt die Unterschlüssel, die mit der Microsoft Management Console verknüpft sind, um Registrierungs basierte Richtlinien Einstellungen für SNMP festzulegen. |
| [Konfigurieren von SNMP-Agent-Informationen](configuring-snmp-agent-information.md) | Beschreibt die Verwendung von Agentinformationen in SNMP.                                                                                       |
