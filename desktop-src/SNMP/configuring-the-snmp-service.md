# <a name="configuring-the-snmp-service"></a>Konfigurieren des SNMP-Diensts

Zum Konfigurieren des SNMP-Diensts müssen Sie die vom Netzwerk verwendeten Communitynamen, das Trapziel für jede Community und den Computernamen oder die IP-Adresse für jeden SNMP-Verwaltungshost kennen.

In der folgenden Tabelle sind Themen aufgeführt, die Informationen zum Konfigurieren des SNMP-Diensts enthalten.

| Thema                                                                        | Inhalte                                                                                                                              |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [Community Namen](community-names.md)                                       | Beschreibt die Verwendung von Communitynamen in SNMP.                                                                                         |
| [Hostnamen und IP-Adressen](host-names-and-ip-addresses.md)               | Beschreibt die Verwendung von Hostnamen und IP-Adressen in SNMP.                                                                             |
| [Konfigurieren der SNMP-Sicherheit](configuring-snmp-security.md)                   | Beschreibt SNMP-Sicherheitsfeatures.                                                                                                     |
| [Konfigurieren von SNMP-Gruppenrichtlinie](configuring-snmp-group-policy.md)           | Beschreibt die Unterschlüssel, die mit dem Microsoft Management Console verknüpft sind, um registrierungsbasierte Richtlinieneinstellungen für SNMP festzulegen. |
| [Konfigurieren von SNMP-Agent-Informationen](configuring-snmp-agent-information.md) | Beschreibt die Verwendung von Agent-Informationen in SNMP.                                                                                       |
