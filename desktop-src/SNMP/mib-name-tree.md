# <a name="mib-name-tree"></a>MIB-Namensstruktur

Der Namensraum für MIB-Objektbezeichner ist hierarchisch. Sie ist so strukturiert, dass jedem verwaltbaren Objekt ein global eindeutiger Name zugewiesen werden kann.

Die Autorität für Teile des Namensraums wird einzelnen Organisationen zugewiesen. Dadurch können Organisationen Namen zuweisen, ohne bei jeder Zuweisung eine Internetautorität zu konsultieren. Der Microsoft zugewiesene Namensraum ist beispielsweise 1.3.6.1.4.1.311, der in MSFT definiert ist. Mib. Microsoft verfügt über die Autorität, Namen objekten an einer beliebigen Stelle unterhalb dieses Namensraums zuzuweisen.

Der Objektbezeichner in der Hierarchie wird als Sequenz von Unteridentitäten geschrieben, die am Stamm und am Objekt enden. Unteridentitäten werden durch einen Punkt getrennt.
