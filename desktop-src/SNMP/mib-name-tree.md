# <a name="mib-name-tree"></a>MIB-namens Struktur

Der Namespace für MIB-Objekt Bezeichner ist hierarchisch. Es ist so strukturiert, dass jedem verwaltbaren Objekt ein global eindeutiger Name zugewiesen werden kann.

Die Autorität für Teile des Namensraums wird einzelnen Organisationen zugewiesen. Dadurch können Organisationen Namen zuweisen, ohne eine Internet Autorität für jede Zuweisung zu konsultieren. Der Namespace, der Microsoft zugewiesen ist, lautet z. b. 1.3.6.1.4.1.311, der in MSFT definiert ist. MIB. Microsoft hat die Berechtigung, Objekten an einer beliebigen Stelle unterhalb dieses Namensraums Namen zuzuweisen.

Der Objekt Bezeichner in der Hierarchie wird als Sequenz von subidentifiers geschrieben, beginnend am Stamm und am Ende des Objekts. Unter Elemente werden durch einen bestimmten Zeitraum getrennt.
