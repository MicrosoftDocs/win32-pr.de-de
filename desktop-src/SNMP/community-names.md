# <a name="community-names"></a>Communitynamen

Ein Communityname identifiziert eine Sammlung von SNMP-Managern und-Agents. Die Verwendung eines Communitynamens bietet eine primitive Sicherheits-und Kontext Prüfung für Agents und Manager, die Anforderungen empfangen und Trap-Vorgänge initiieren. Ein Agent akzeptiert keine Anforderung von einem Vorgesetzten außerhalb der Community.

Beachten Sie, dass Standard Sicherheitsempfehlungen für die Verwendung von SNMP außer vertrauenswürdigen Netzwerken empfohlen werden, da das Protokoll im Entwurfs Modus nur minimale Sicherheit bietet. Weitere Informationen finden Sie unter [Konfigurieren der SNMP-Sicherheit](configuring-snmp-security.md).
