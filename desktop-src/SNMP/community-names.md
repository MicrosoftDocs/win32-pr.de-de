# <a name="community-names"></a>Community Namen

Ein Communityname bezeichnet eine Sammlung von SNMP-Managern und -Agents. Die Verwendung eines Communitynamens bietet primitive Sicherheits- und Kontextüberprüfung für Agents und Manager, die Anforderungen empfangen und Trapvorgänge initiieren. Ein Agent akzeptiert keine Anforderung von einem Vorgesetzten außerhalb der Community.

Beachten Sie, dass die Standardsicherheitsempfehlungen die Verwendung von SNMP mit Ausnahme von vertrauenswürdigen Netzwerken empfehlen, da das Protokoll standardmäßig minimale Sicherheit bietet. Weitere Informationen finden Sie unter [Konfigurieren der SNMP-Sicherheit.](configuring-snmp-security.md)
