# <a name="parental-controls-minimum-compliance-api"></a>Mindestrecompliance-API für Eltern Kontrollen

Die API für die minimale Konformität implementiert einfache Schnittstellen Methoden für Anwendungen, um grundlegende Zustandsinformationen zu einem bestimmten, von einem Benutzer überwachten Benutzerkonto, den webeinschränkungs Einstellungen und den Einstellungen für die Spiele Einschränkung zu erhalten.

-   [**IWindowsParentalControls**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwindowsparentalcontrols)
-   [**Iwpcgamessettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcgamessettings)
-   [**Iwpcsettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcsettings)
-   [**Iwpcwebsettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcwebsettings)
