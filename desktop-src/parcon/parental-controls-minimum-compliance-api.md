# <a name="parental-controls-minimum-compliance-api"></a>Mindestkonformitäts-API für Jugendschutz

Die Mindestkonformitäts-API implementiert einfache Schnittstellenmethoden für Anwendungen, um grundlegende Zustandsinformationen zu einem bestimmten, von der Eltern kontrollierten Benutzerkonto, seinen Webeinschränkungseinstellungen und Spieleeinschränkungseinstellungen abzurufen.

-   [**IWindowsParentalControls**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwindowsparentalcontrols)
-   [**IWPCGamesSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcgamessettings)
-   [**IWPCSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcsettings)
-   [**IWPCWebSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcwebsettings)
