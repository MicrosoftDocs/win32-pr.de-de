# <a name="parental-controls"></a>Jugendschutz

## <a name="purpose"></a>Zweck

Im Rahmen der Family Safety von Microsoft bietet Windows Funktionen für die Jugendschutzfunktionen, um die Gefährdung ausgewählter Computerbenutzer durch Onlinebehinderung und ungeeignete Inhalte zu überwachen und einzuschränken.

Die Technologie der Jugendschutzfunktionen in Windows soll eltern- oder pflegeberechtigten Eltern oder Wächtern dabei helfen, den Zugriff auf geeignete Materialien nach Alter oder Reifegrad für personen unter ihrer Wächterschaft sicherzustellen. Sie bietet zusätzlich zu den integrierten Funktionen eine erweiterbare Infrastruktur.

## <a name="developer-audience"></a>Entwicklergruppe

Unabhängige Softwarehersteller (Independent Software Vendors, ISVs) von Produkten, die Benutzer potenziell Sicherheitsrisiken aussetzen oder derzeit daran arbeiten, Benutzer vor solchen Risiken zu schützen, werden empfohlen, die Jugendschutzfunktionen zu integrieren.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Diese Funktion wurde in Windows Vista eingeführt. Die Funktionalität der Jugendschutzfunktionen ist nur für ausgewählte Consumerbetriebssysteme verfügbar. sie ist nicht für die Verwendung mit Versionen von Windows Server konzipiert. Diese Funktionalität wird in Windows 10 nicht unterstützt.
