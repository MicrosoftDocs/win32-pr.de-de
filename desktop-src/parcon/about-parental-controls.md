# <a name="about-parental-controls"></a>Informationen zu Eltern Steuerelementen

Die Jugend Steuerungsfunktion umfasst Folgendes:

-   Zentrale Benutzeroberflächen Funktionalität, integriert in die Systemsteuerung.
-   Ausgewählte Einschränkungs Implementierungen.
-   Eine Infrastruktur im Betriebssystem, die Einstellungen für allgemeine Richtlinien und implementierte Einschränkungen bereitstellt und die zielerweiterbarkeits Features verfügbar macht.
