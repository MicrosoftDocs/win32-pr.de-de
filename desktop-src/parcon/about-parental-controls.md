# <a name="about-parental-controls"></a>Informationen zu Jugendschutzfunktionen

Das Feature "Jugendschutz" umfasst Folgendes:

-   Zentrale, in Systemsteuerung integrierte Benutzeroberfläche.
-   Ausgewählte Einschränkungsimplementierungen.
-   Eine Infrastruktur im Betriebssystem, die Einstellungen für allgemeine Richtlinien und implementierte Einschränkungen bereitstellt und gezielte Erweiterbarkeitsfunktionen verfügbar macht.
