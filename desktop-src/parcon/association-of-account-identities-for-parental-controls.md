# <a name="association-of-account-identities-for-parental-controls"></a>Zuordnung von Konto Identitäten für Eltern Steuerelemente

Wie in der Einführung erläutert, hängt die effektive Verwendung der Konfigurations-, Protokollierungs-und Einschränkungs Funktionen der Windows Vista-Infrastruktur für Jugendschutz von Windows-Konten ab, die als Standardbenutzer eingerichtet werden. Es wird empfohlen, dass Anbieter, die Online Zugriff (wie z. b. ISP Suites) bereitstellen, eine Zuordnung von Online Benutzernamen erstellen und verwalten, die für jedes kontrollierte Windows-Konto von der übergeordneten oder dem Wächter Auf diese Weise wird die Umgehung von cloudbasierten Jugend Steuerungsmechanismen lediglich durch die Verwendung alternativer Online Identitäten reduziert.
