# <a name="parental-controls-top-level-user-interface"></a>Eltern Steuerelemente Top-Level Benutzeroberfläche

Die starke Verknüpfung zwischen Familien Sicherheit und Windows-Benutzerkonten hat eine kombinierte Kategorie mit **Benutzerkonten und Familien Sicherheit** in der Systemsteuerung für kundenorientierte SKUs von Windows Vista gesteuert.

> [!Note]  
> Die Kategorie wird als **Benutzerkonten** angezeigt, wenn ein Computer mit Domänen Verknüpfungs Funktion verknüpft ist.

 

Wenn ein Standardbenutzer Konto noch nicht für den kontrollierten Benutzer eingerichtet wurde, bietet ein Link in der Systemsteuerung einen einfachen Workflow zum Hinzufügen eines neuen, über das Konto kontrollierten Kontos.

Nachdem ein Konto erstellt wurde, werden in der Systemsteuerung "Jugendschutz" die Benutzer seitigen Funktionen der obersten Ebene für den kontrollierten Benutzer verfügbar gemacht. Aspekte

-   Die Kontroll Schaltflächen für die allgemeine Kontrolle über ein oder aus.
-   Options Felder für die unabhängige Aktivitäts Berichterstellung oder-Steuerung.
-   Ein Abschnitt mit den Einstellungen mit den von Microsoft implementierten Webanwendungs Links wird bedingt angezeigt, und die verfügbaren, von Microsoft implementierten Einschränkungs Typen für Microsoft
-   Ein Bereich mit weiteren Einstellungen, der angezeigt wird, wenn Benutzeroberflächen Erweiterungen von Microsoft oder Drittanbietern registriert werden.
-   Ein Zusammenfassungs Statusbereich, der den Benutzernamen und die Kachel anzeigt, den Link Aktivitätsberichte anzeigen und den Status der Einstellungen der Jugend Steuerungseinstellungen. Wenn ON festgelegt ist, enthält die Anzeige die Einstellungsebene für den Webinhalts Filter (wenn der Microsoft-Filter aktiv ist) oder Filter Name, wenn Sie von einer Drittanbieter Erweiterung außer Kraft gesetzt wird, und den Status der Offline Einstellungen für den Benutzer.

Das Optionsfeld für den gesamten Jugendschutz aktivieren ist anfänglich auf OFF festgelegt. Wenn die Aktivierung durch einen Administrator erfolgt, ist die Aktivitäts Berichterstattung ebenfalls standardmäßig aktiviert, kann jedoch unabhängig deaktiviert werden. Wie bei fast allen Einstellungen in Jugendschutz-Steuerelementen wird die Konfiguration möglicherweise Programm gesteuert durch die verfügbaren APIs durchgeführt.
