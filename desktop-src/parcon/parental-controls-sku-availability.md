# <a name="parental-controls-sku-availability"></a>Verfügbarkeit der Jugend Steuerungs SKU

Die Jugend Schnittstellen, Features und APIs, die in diesem Abschnitt beschrieben werden, sind zurzeit nur für kundenorientierte SKUs verfügbar, wie z. b.:

-   Windows Vista Starter
-   Windows Vista Home Basic
-   Windows Vista Home Premium
-   Windows Vista Ultimate

Elterliche Kontrollen werden nur auf diesen SKUs installiert. Lösungen, die eine Anforderung für die Bereitstellung auf Geschäfts-SKUs haben, müssen Folgendes erfüllen:

-   Erkennen Sie die Funktionen der Jugend Steuerungsfunktionen auf Geschäfts-SKUs
-   Erkennen und Bereitstellen alternativer Konfigurations-, Protokollierungs-und Einschränkungs Möglichkeiten.

Es wird empfohlen, dass Lösungen die Verfügbarkeit der Jugend Steuerungs Infrastruktur erkennen, indem Sie den Erfolg von com cokreateinstance für die Konformitäts-API überprüfen.
