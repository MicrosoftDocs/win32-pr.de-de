Die Implementierung der Aktivitätsprotokollierung erfordert die Verwendung der Minimum Compliance-API für die Jugendschutzfunktionen, um zu bestimmen, wann eine Protokollierung erforderlich ist, und der Windows Event Logging Publishing-API für die Generierung von Protokollereignissen.
