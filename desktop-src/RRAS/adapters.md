# <a name="adapters"></a>Adapter

Ein Adapter stellt die Hardware dar, die eine Verbindung mit einem bestimmten Netzwerksegment herstellt. Eine gebundene LAN-Karte ist ein Beispiel für einen Adapter. Analog dazu verfügt ein Computer mit zwei Modems, die jeweils eine Verbindung mit einem Remote Netzwerk herstellen können, über zwei Adapter, eine zur Darstellung der einzelnen Modem.
