# <a name="adapters"></a>Adapter

Ein Adapter stellt die Hardware dar, die eine Verbindung mit einem bestimmten Netzwerksegment herstellt. Eine gebundene LAN-Karte ist ein Beispiel für einen Adapter. Auf ähnliche Weise verfügt ein Computer mit zwei Modems, die jeweils eine Verbindung mit einem Remotenetzwerk herstellen können, über zwei Adapter, einen, der jedes Modem darstellt.
