<span id="_rras_autonomous_system_gly"></span><span id="_RRAS_AUTONOMOUS_SYSTEM_GLY"></span>**Autonomes System (AS)**

Ein autonomes System ist eine Gruppe von Routern, die eine einzige Administrative Richtlinie gemeinsam verwenden. Diese Router verwenden alle das gleiche Routingprotokoll für die Kommunikation (als [*inneres Gatewayprotokoll bezeichnet).*](i-gly.md) Siehe auch [*äußeres Gatewayprotokoll*](e-gly.md).

<span id="_rras_autostatic_route_gly"></span><span id="_RRAS_AUTOSTATIC_ROUTE_GLY"></span>**Autostatic Route**
</dt> <dd>

Eine automatisch statische Route ist eine Route, die dynamisch aus einem Routingprotokoll ermittelt wird. Nachdem eine Route auf diese Weise erhalten wurde, verhält sie sich wie eine [*statische Route.*](s-gly.md)
