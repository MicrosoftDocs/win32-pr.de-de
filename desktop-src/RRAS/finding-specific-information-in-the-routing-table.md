# <a name="finding-specific-information-in-the-routing-table"></a>Suchen bestimmter Informationen in der Routingtabelle

Clients müssen in der Lage sein, bestimmte Informationen in der Routingtabelle zu finden, anstatt alle Daten aufzählen zu müssen. Die RTMv2-API bietet die Möglichkeit, basierend auf bestimmten Kriterien nach bestimmten Zielen, Routen und nächsten Hops zu suchen.
