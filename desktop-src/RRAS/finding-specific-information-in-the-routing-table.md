# <a name="finding-specific-information-in-the-routing-table"></a>Suchen nach bestimmten Informationen in der Routing Tabelle

Clients müssen in der Lage sein, bestimmte Informationen in der Routing Tabelle zu finden, anstatt alle Daten aufzuzählen. Die RTMv2-API bietet die Möglichkeit, anhand bestimmter Kriterien nach bestimmten Zielen, Routen und nächsten Hops zu suchen.
