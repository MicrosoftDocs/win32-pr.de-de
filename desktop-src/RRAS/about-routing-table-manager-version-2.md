# <a name="about-routing-table-manager-version-2"></a>Informationen zum Routingtabellen-Manager Version 2

In der folgenden Dokumentation wird die RtMv2-Technologie (Routing Table Manager Version 2) beschrieben. Die RTMv2-API ist ein Feature der Betriebssysteme Windows 2000 und höher, mit denen Sie Routingprotokolle schreiben können, die mit dem Routingtabellen-Manager interagieren.

Der Routingtabellen-Manager ist das zentrale Repository mit Routinginformationen für alle Routingprotokolle, die unter dem Routing- und Ras-Dienst (RRAS) arbeiten.

RTMv2 ist für nt Windows 4.0 nicht verfügbar. Darüber hinaus kann RTMv2 nicht für IPX-Routingprotokolle verwendet werden, die unter Windows NT 4.0 oder Windows 2000 ausgeführt werden. Wenn Sie IPX verwenden oder Routingprotokolle für Windows NT 4.0 schreiben, lesen Sie Informationen zur [Routingtabellen-Manager-Version 1.](about-routing-table-manager-version-1.md)
