# <a name="address-family"></a>Adressfamilie

Zu den Adressfamilien gehören Internet Paket Austausch (IPX), Internetprotokoll Version 4 (IPv4) und IPv6 (Internet Protocol Version 6). Adressfamilien können auch als Protokoll Familien bezeichnet werden. Diese Adressfamilien sind in [RFC 1700](routing-protocols-request-for-comments.md)definiert.
