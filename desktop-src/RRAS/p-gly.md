# <a name="p-rras"></a>P (RRAS)

<dl> <dt>

<span id="_rras_point_to_multipoint_interface_gly"></span><span id="_RRAS_POINT_TO_MULTIPOINT_INTERFACE_GLY"></span>**Point-to-Multipoint-Schnittstelle**
</dt> <dd>

Die Multicast-Gruppen-Manager-Technologie verwendet diesen Begriff für eine Schnittstelle, die die Kommunikation zwischen einer einzelnen Quelle und einem einzelnen Ziel (z. b. einem PPP-Link) bereitstellt.

</dd> <dt>

<span id="_rras_point_to_point_interface_gly"></span><span id="_RRAS_POINT_TO_POINT_INTERFACE_GLY"></span>**Punkt-zu-Punkt-Schnittstelle**
</dt> <dd>

Die Multicast-Gruppen-Manager-Technologie verwendet diesen Begriff, um eine Schnittstelle zu verwenden, die die Kommunikation zwischen einem einzelnen Host und mehreren Zielen ermöglicht. Punkt-zu-Multipoint-Schnittstellen können als eine Auflistung von Punkt-zu-Punkt-Verknüpfungen mit einer einzelnen Beendigung (z. b. einem ATM-Link) betrachtet werden.

</dd> </dl>
