<dl> <dt>

<span id="_rras_point_to_multipoint_interface_gly"></span><span id="_RRAS_POINT_TO_MULTIPOINT_INTERFACE_GLY"></span>**Point-to-Multipoint-Schnittstelle**
</dt> <dd>

Die Multicast Group Manager-Technologie verwendet diesen Begriff als Schnittstelle, die die Kommunikation zwischen einer einzelnen Quelle und einem einzelnen Ziel ermöglicht, z. B. einem HYPERLINK-Link.

</dd> <dt>

<span id="_rras_point_to_point_interface_gly"></span><span id="_RRAS_POINT_TO_POINT_INTERFACE_GLY"></span>**Point-to-Point-Schnittstelle**
</dt> <dd>

Die Multicast Group Manager-Technologie verwendet diesen Begriff als Schnittstelle, die die Kommunikation zwischen einem einzelnen Host und mehreren Zielen ermöglicht. Punkt-zu-Multipoint-Schnittstellen können als Sammlung von Punkt-zu-Punkt-Links mit einer einzelnen Beendigung, z. B. einem ATM-Link, bezeichnet werden.

</dd> </dl>
