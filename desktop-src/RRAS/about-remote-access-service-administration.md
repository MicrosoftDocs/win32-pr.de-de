# <a name="about-remote-access-service-administration"></a>Informationen zur Remotezugriffsdienstverwaltung

Windows Betriebssystemen 2000 und höher stellen eine Reihe von Funktionen zum Verwalten von Benutzerberechtigungen und Ports auf RAS-Servern zur Verfügung. Mit diesen Funktionen können Sie eine RAS-Serververwaltungsanwendung entwickeln, um die folgenden Aufgaben auszuführen:

-   Aufzählen der Benutzer, die über einen angegebenen Ras-Berechtigungssatz verfügen
-   Zuweisen oder Widerrufen von RAS-Berechtigungen für einen angegebenen Benutzer
-   Aufzählen der konfigurierten Ports auf einem RAS-Server
-   Informationen und Statistiken zu einem angegebenen Port auf einem RAS-Server
-   Zurücksetzen der Statistikindikatoren für einen angegebenen Port
-   Trennen eines angegebenen Ports

Sie können auch eine RAS-Serververwaltungs-DLL installieren, um Benutzerverbindungen zu überwachen und Einwählbenutzern IP-Adressen zu zuweisen. Die DLL exportiert eine Reihe von Funktionen, die der RAS-Server aufruft, wenn ein Benutzer versucht, eine Verbindung herzustellen oder die Verbindung zu trennen.
