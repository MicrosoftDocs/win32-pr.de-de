# <a name="routing-table-manager-version-1-protocol-family-identifiers"></a>Routingtabellen-Manager Version 1: Protokollfamilienbezeichner

**Windows Server 2003:** Diese API wurde von der Routing Table Manager Version [2-API](about-routing-table-manager-version-2.md) abgelöst und ist nicht mehr als Windows Server 2003 verfügbar. Neue Anwendungen sollten die Routing Table Manager Version 2-API verwenden.

Die folgenden Transportbezeichner sind in Rtm.h aufgeführt. Verwenden Sie diese Bezeichner mit der RTMv1-API.



| Konstante                   | Beschreibung                        |
|----------------------------|------------------------------------|
| \_ \_ IPX DER \_ RTM-PROTOKOLLFAMILIE | Identifiziert die IPX-Adressfamilie. |
| \_ \_ \_ IP-Adresse der RTM-PROTOKOLLFAMILIE  | Identifiziert die IP-Adressfamilie.  |
