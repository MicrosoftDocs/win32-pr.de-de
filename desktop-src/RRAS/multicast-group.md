# <a name="multicast-group"></a>Multicast Gruppe

Eine Multicast Gruppe ist eine Klassen-D-IP-Adresse (D. h. im Bereich von 224.0.0.0-239.255.255.255). Multicast Daten, die an eine Adresse in diesem Bereich gesendet werden, sind für ein einzelnes Ziel nicht bestimmt. Stattdessen können diese Daten von jedem Host empfangen werden, von dem eine Anforderung zum beitreten zu einer bestimmten Gruppe gesendet wird. Auf eine Multicast Gruppe wird entweder das Symbol G oder g verwiesen.
