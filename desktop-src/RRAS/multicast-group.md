# <a name="multicast-group"></a>Multicastgruppe

Eine Multicastgruppe ist eine IP-Adresse der Klasse D (d. h. im Bereich von 224.0.0.0 bis 239.255.255.255). Multicastdaten, die an eine Adresse in diesem Bereich gesendet werden, sind nicht für ein einzelnes Ziel bestimmt. Stattdessen können diese Daten von jedem Host empfangen werden, der eine Anforderung zum Beitreten zu einer bestimmten Gruppe stellt. Auf eine Multicastgruppe wird entweder durch das Symbol G oder g verwiesen.
