# <a name="client"></a>Client

Ein Client von RTMv2 ist entweder eine Instanz eines Routing Protokolls oder eine Verwaltungs Komponente, die den Routing Tabellen-Manager mithilfe der RTMv2-API registriert und interagiert. Sofern nicht anders angegeben, verweist jeder Verweis auf einen Routing Tabellen-Manager-Client auf einen Client von RTMv2.

Ein Client muss in demselben Prozess wie der routermanager ausgeführt werden.
