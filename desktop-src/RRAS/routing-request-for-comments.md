# <a name="routing-request-for-comments"></a>Routing Anforderung für Kommentare

Bei Anforderungen für Kommentare (RFCs) handelt es sich um eine fortlaufende Reihe von Dokumenten, in denen die Internet-und Internet bezogenen Technologien erörtert werden. Bestimmte RFCs werden nach Zahl identifiziert. Im Abschnitt RAS (Remote Access Service) des Platform Software Development Kit (SDK) wird auf die folgenden RFCs verwiesen. Weitere Informationen zu RFCs und zum Auffinden von Kopien einer bestimmten RFC finden Sie unter [https://www.rfc-editor.org/rfc.html](https://www.rfc-editor.org/rfc.html) .

-   [RFC 791](https://www.ietf.org/rfc/rfc791.txt) – Internetprotokoll, DARPA Internet Program Protocol Specification, September 1981
-   [RFC 2863](https://www.ietf.org/rfc/rfc2863.txt) – Schnittstellen Gruppen-MIB, Juni 2000
-   [RFC 1354](https://www.ietf.org/rfc/rfc1354.txt) – IP-Weiterleitungs Tabelle MIB, Juli, 1992
