# <a name="next-hop"></a>Nächster Hop

Ein nächster Hop ist der nächste Router auf dem Pfad zu einem Ziel. Pakete aus einer Quelle werden auf Hop-by-Hop-Basis an ein Ziel weitergeleitet.

Die Adresse des Routers, bei der es sich um den nächsten Hop in einer Route handelt, wird verwendet, um Links auf einer Point-to-Multipoint-Schnittstelle eindeutig zu identifizieren, wobei alle Verknüpfungen denselben Schnittstellen Index gemeinsam verwenden.
