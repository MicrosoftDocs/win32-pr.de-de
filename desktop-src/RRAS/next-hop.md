# <a name="next-hop"></a>Nächster Hop

Ein nächster Hop ist der nächste Router auf dem Pfad zu einem Ziel. Pakete aus einer Quelle werden auf Hop-by-Hop-Basis an ein Ziel weitergeleitet.

Die Adresse des Routers, der der nächste Hop in einer Route ist, wird verwendet, um Links auf einer Point-to-Multipoint-Schnittstelle eindeutig zu identifizieren, wobei alle Links denselben Schnittstellenindex verwenden.
