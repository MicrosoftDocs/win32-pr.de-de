# <a name="ipx-service-table-management"></a>Verwaltung von IPX-Dienst Tabellen

Ein IPX-Routing Protokoll, das für die Dienst Behandlung registriert wird, verwaltet eine Dienst Bindery-Tabelle gemäß den Spezifikationen von Novell des IPX Service Advertising-Protokolls (SAP). Das Routing Protokoll ermöglicht außerdem dem routermanager den Zugriff auf diese Tabelle über die folgenden Funktionen und Strukturen.

-   [Funktionen der Dienst Tabellen Verwaltung](service-table-management-functions.md)
-   [Strukturen der Dienst Tabellen Verwaltung](service-table-management-structures.md)

Weitere Informationen zur Registrierung eines Routing Protokolls für die Dienst Behandlung finden Sie in der [**RegisterProtocol**](/windows/desktop/api/Routprot/nc-routprot-pregister_protocol) -Funktion.
