# <a name="overview-of-mgm-callbacks"></a>Übersicht über MGM-Rückrufe

Es gibt zwei Arten von Rückrufen in der MGM-API:

-   [Routingprotokollrückrufe](routing-protocol-callbacks.md)
-   [IGMP: Aktivieren und Deaktivieren von Rückrufen](igmp-enable-and-disable-callbacks.md)

Diese Rückrufe ermöglichen in Kombination mit den MGM-APIs die fortlaufende Kommunikation zwischen Routingprotokollen, IGMP und dem Multicastgruppen-Manager.
