# <a name="overview-of-mgm-callbacks"></a>Übersicht über die MGM-Rückrufe

In der MGM-API gibt es zwei Arten von Rückrufe:

-   [Routing Protokoll Rückrufe](routing-protocol-callbacks.md)
-   [IGMP-Rückrufe aktivieren und deaktivieren](igmp-enable-and-disable-callbacks.md)

Diese Rückrufe in Kombination mit den MGM-APIs ermöglichen die fortlaufende Kommunikation zwischen Routing Protokollen, IGMP und dem Multicast-Gruppen-Manager.
