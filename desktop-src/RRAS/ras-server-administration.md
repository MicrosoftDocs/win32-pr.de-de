# <a name="ras-server-administration"></a>RAS-Serververwaltung

Windows NT Server 4.0 bietet eine Reihe von Funktionen zum Verwalten von Benutzerberechtigungen und Ports auf Windows NT/Windows 2000 RAS-Servern. Windows 95 unterstützt diese Funktionen nicht. Mit diesen Funktionen können Sie eine RAS-Serververwaltungsanwendung entwickeln, um die folgenden Aufgaben auszuführen:

-   Aufzählen der Benutzer, die über einen angegebenen Satz von RAS-Berechtigungen verfügen
-   Zuweisen oder Widerrufen von RAS-Berechtigungen für einen angegebenen Benutzer
-   Aufzählen der konfigurierten Ports auf einem RAS-Server
-   Abrufen von Informationen und Statistiken zu einem angegebenen Port auf einem RAS-Server
-   Zurücksetzen der Statistikindikatoren für einen angegebenen Port
-   Trennen eines angegebenen Ports

Sie können auch eine RAS-Server-Verwaltungs-DLL installieren, um Benutzerverbindungen zu überwachen und einwählenden Benutzern IP-Adressen zuzuweisen. Die DLL exportiert eine Reihe von Funktionen, die der RAS-Server aufruft, wenn ein Benutzer versucht, eine Verbindung herzustellen oder die Verbindung zu trennen.
