# <a name="enumerating-routing-table-entries"></a>Auflisten von Routing Tabellen Einträgen

Die Enumerationsfunktionen ermöglichen einem Client das Abrufen von Informationen zu einem bestimmten Typ von Routing Tabellen Objekten (Routen, Ziele und nächste Hops). Sowohl Routing Protokolle als auch Verwaltungsprogramme können diese Funktionen verwenden, um nach bestimmten Daten zu suchen.

Beispielcode, der die Verwendung dieser Funktionen veranschaulicht, finden Sie unter Auflisten [aller Ziele](enumerate-all-destinations.md) und auflisten [aller Routen](enumerate-all-routes.md).
