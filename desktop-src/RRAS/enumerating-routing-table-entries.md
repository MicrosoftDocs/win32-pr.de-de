# <a name="enumerating-routing-table-entries"></a>Aufzählen von Routingtabelleneinträgen

Mit den Enumerationsfunktionen kann ein Client Informationen zu einem bestimmten Typ von Routingtabellenobjekten (Routen, Ziele und nächste Hops) abrufen. Sowohl Routingprotokolle als auch Verwaltungsprogramme können diese Funktionen verwenden, um bestimmte Daten zu suchen.

Beispielcode, der die Verwendung dieser Funktionen veranschaulicht, finden Sie unter [Aufzählen](enumerate-all-destinations.md) aller Ziele und [Aufzählen aller Routen.](enumerate-all-routes.md)
