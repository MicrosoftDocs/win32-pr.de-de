# <a name="routing-table"></a>Routingtabelle

Der Routingtabellen-Manager verwaltet für jede Adressfamilie (z. B. IPX und IPv4) eine eigene Routingtabelle. Die IPX-Adressfamilie wird von der RTMv1-API unterstützt. IPv4 wird von der RTMv2-API unterstützt.

Die Routingtabelle besteht aus Zielen, Routen und nächsten Hops. Diese Einträge definieren eine Route zu einem Zielnetzwerk.
