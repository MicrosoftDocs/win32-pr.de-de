# <a name="routing-table"></a>Routing Tabelle

Der Routing Tabellen-Manager verwaltet eine eindeutige Routing Tabelle für jede Adressfamilie (z. b. IPX und IPv4). Die IPX-Adressfamilie wird von der RTMv1-API unterstützt. IPv4 wird von der RTMv2-API unterstützt.

Die Routing Tabelle besteht aus Zielen, Routen und nächsten Hops. Mit diesen Einträgen wird eine Route zu einem Zielnetzwerk definiert.
