# <a name="maintaining-client-specific-lists"></a>Verwalten von Client-Specific Listen

RTMv2 stellt Funktionen bereit, mit denen Clients eine private Liste von Routen erstellen können, die in der Routing Tabelle gespeichert sind. Die Verwendung dieser Liste kann effizienter sein als das Auflisten von Routen aus der Routing Tabelle.

Beispielcode, der die Verwendung dieser Funktionen veranschaulicht, finden Sie unter [Verwenden einer Client-Specific Routenliste](use-a-client-specific-route-list.md).
