# <a name="view"></a>Sicht

Eine Sicht ist eine Teilmenge der Routing Tabelle und enthält eine Gruppe verwandter Routen (z. b. Multicast Routen). Derzeit werden nur Unicast-und Multicast Sichten unterstützt. Sichten werden manchmal auch als Routing Informationsbasen (RIBs) bezeichnet.
