# <a name="remote-access-request-for-comments"></a>Remote Zugriffs Anforderung für Kommentare

Bei Anforderungen für Kommentare (RFCs) handelt es sich um eine fortlaufende Reihe von Dokumenten, in denen die Internet-und Internet bezogenen Technologien erörtert werden. Bestimmte RFCs werden nach Zahl identifiziert. Im Abschnitt RAS (Remote Access Service) des Platform Software Development Kit (SDK) wird auf die folgenden RFCs verwiesen. Weitere Informationen zu RFCs und zum Auffinden von Kopien einer bestimmten RFC finden Sie unter [https://www.rfc-editor.org/rfc.html](https://www.rfc-editor.org/rfc.html) .

-   [RFC 791](https://www.ietf.org/rfc/rfc791.txt) – Internetprotokoll, DARPA Internet Program Protocol Specification, September 1981
-   [RFC 1171](https://www.ietf.org/rfc/rfc1171.txt) – die Point-to-Point-Protokoll, Juli 1990
-   [RFC 1172](https://www.ietf.org/rfc/rfc1172.txt) – die PPP-Optionen für die Erstkonfiguration, Juli, 1990
-   [RFC 1570](https://www.ietf.org/rfc/rfc1570.txt) – PPP LCP Extensions, Januar, 1994
-   [RFC 1962](https://www.ietf.org/rfc/rfc1962.txt) – das PPP Compression Control Protocol (CCP), Juni 1996
-   [RFC 1974](https://www.ietf.org/rfc/rfc1974.txt) – PPP Stac LZS-Komprimierungs Protokoll, August 1996
-   [RFC 1990](https://www.ietf.org/rfc/rfc1990.txt) – das PPP-Multilink-Protokoll (MP), August 1996
-   [RFC 2118](https://www.ietf.org/rfc/rfc2118.txt) – Microsoft Point-to-Point Compression (MPPC)-Protokoll, März 1997
-   [RFC 2284](https://www.ietf.org/rfc/rfc2284.txt) – PPP Extensible Authentication Protocol, März 1998
-   [RFC 2433](https://www.ietf.org/rfc/rfc2433.txt) – Microsoft PPP CHAP Extensions, Oktober 1998
-   [RFC 2484](https://www.ietf.org/rfc/rfc2484.txt) – PPP LCP-Internationalisierungs Konfigurations Option, Januar 1999
-   [RFC 2548](https://www.ietf.org/rfc/rfc2848.txt) – herstellerspezifische RADIUS-Attribute von Microsoft
-   [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt) – Remote Authentication Dial in User Service (RADIUS), Juni 2000
-   [RFC 2866](https://www.ietf.org/rfc/rfc2866.txt) – RADIUS-Buchhaltung, Juni 2000
-   [RFC 2868](https://www.ietf.org/rfc/rfc2868.txt) – RADIUS-Attribute für die Tunnel Protokoll Unterstützung, Juni 2000
-   [RFC 2869](https://www.ietf.org/rfc/rfc2869.txt) – RADIUS-Erweiterungen, Juni 2000
-   [Draft RADIUS MSCHAP attr 01](https://www.freeradius.org/rfc/draft-ietf-radius-mschap-attr-01.txt) – RADIUS-Attribute für MS-CHAP-Unterstützung, November, 1997
