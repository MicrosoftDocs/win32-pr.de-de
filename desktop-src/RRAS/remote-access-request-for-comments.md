# <a name="remote-access-request-for-comments"></a>Remotezugriffsanforderung für Kommentare

Requests for Comments (RFCs) sind eine fortlaufende Reihe von Dokumenten, in denen internet- und internetbezogene Technologien erörtert werden. Bestimmte RFCs werden anhand der Zahl identifiziert. Auf die folgenden RFCs wird im Abschnitt RAS (Remote Access Service) des Platform Software Development Kit (SDK) verwiesen. Weitere Informationen zu RFCs und zum Suchen von Kopien einer bestimmten RFC finden Sie unter [https://www.rfc-editor.org/rfc.html](https://www.rfc-editor.org/rfc.html) .

-   [RFC 791](https://www.ietf.org/rfc/rfc791.txt) – Internet Protocol, DARPA Internet Program Protocol Specification, September 1981
-   [RFC 1171](https://www.ietf.org/rfc/rfc1171.txt) – The Point-to-Point-Protokoll, July 1990
-   [RFC 1172](https://www.ietf.org/rfc/rfc1172.txt) – The PP Initial Configuration Options, July, 1990
-   [RFC 1570](https://www.ietf.org/rfc/rfc1570.txt) – VPNS-LCP-Erweiterungen, Januar 1994
-   [RFC 1962](https://www.ietf.org/rfc/rfc1962.txt) – THE PROTOM (COMPRESSION Compression Control Protocol), Juni 1996
-   [RFC 1974](https://www.ietf.org/rfc/rfc1974.txt) – PP Stac LZS Compression Protocol, August 1996
-   [RFC 1990](https://www.ietf.org/rfc/rfc1990.txt) – The PP Multilink Protocol (MP), August 1996
-   [RFC 2118](https://www.ietf.org/rfc/rfc2118.txt) – Microsoft Point-to-Point Compression (MPPC)-Protokoll, März 1997
-   [RFC 2284](https://www.ietf.org/rfc/rfc2284.txt) – EXTENSIBle Authentication Protocol, März 1998
-   [RFC 2433](https://www.ietf.org/rfc/rfc2433.txt) – Microsoft CHAP-Erweiterungen, Oktober 1998
-   [RFC 2484](https://www.ietf.org/rfc/rfc2484.txt) – RFC LCP Internationalization Configuration Option, January 1999
-   [RFC 2548](https://www.ietf.org/rfc/rfc2848.txt) – Microsoft-anbieterspezifische RADIUS-Attribute
-   [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt) – Remote Authentication Dial In User Service (RADIUS), Juni 2000
-   [RFC 2866](https://www.ietf.org/rfc/rfc2866.txt) – Radius Accounting, Juni 2000
-   [RFC 2868](https://www.ietf.org/rfc/rfc2868.txt) – Radius Attributes for Tunnel Protocol Support, Juni 2000
-   [RFC 2869](https://www.ietf.org/rfc/rfc2869.txt) – RADIUS-Erweiterungen, Juni 2000
-   [Draft Radius MSCHAP Attr 01](https://www.freeradius.org/rfc/draft-ietf-radius-mschap-attr-01.txt) – RADIUS Attributes for MS-CHAP Support, November, 1997
