# <a name="remote-hardware-management"></a><span data-ttu-id="67dc9-104">Remote Hardware Verwaltung</span><span class="sxs-lookup"><span data-stu-id="67dc9-104">Remote Hardware Management</span></span>

<span data-ttu-id="67dc9-105">Windows-Remoteverwaltung Hardware Verwaltung soll die Gesamtkosten für die IT-Verwaltung verringern, indem Sie die Überwachung und Steuerung von Remote Hardwarekomponenten bereitstellt, insbesondere vor dem Start des Systems und nach einem Betriebssystem Fehler.</span><span class="sxs-lookup"><span data-stu-id="67dc9-105">Windows Remote Management hardware management is intended to reduce overall IT administration costs by providing monitoring and control of remote hardware components, especially before the system is started and after an operating system failure.</span></span>

class="sxs-lookup"><span data-stu-id="67dc9-106">Original Equipment Manufacturers (OEMs) have developed a common architecture to address the need for hardware management.</span></span> <span data-ttu-id="67dc9-107">Ein wichtiger Teil dieser Architektur ist der [*Baseboard-Verwaltungs Controller*](windows-remote-management-glossary.md) (BMC).</span><span class="sxs-lookup"><span data-stu-id="67dc9-107">An important piece of this architecture is the [*baseboard management controller*](windows-remote-management-glossary.md) (BMC).</span></span> <span data-ttu-id="67dc9-108">Ein BMC ist ein spezielles Gerät, das den Status des Server Computers überwacht.</span><span class="sxs-lookup"><span data-stu-id="67dc9-108">A BMC is a specialized device that monitors the state of the server computer.</span></span> <span data-ttu-id="67dc9-109">Der BMC bietet Remote Steuerung der Server Hardware, ruft Statusdaten ab und empfängt Benachrichtigungen zu kritischen Fehlern und anderen critical errors and other hardware status changes.</span></span> <span data-ttu-id="67dc9-110">Skripts oder Anwendungen, die einen Remote Server überwachen, können Daten vom Server entweder [*in-Band*](windows-remote-management-glossary.md), über das Remote Betriebssystem oder [*out-of-Band*](windows-remote-management-glossary.md)direkt vom BMC abrufen.</span><span class="sxs-lookup"><span data-stu-id="67dc9-110">A script or application that is monitoring a remote server can obtain data from the server either [*in-band*](windows-remote-management-glossary.md), <span data-ttu-id="67dc9-111">Ein BMC verfügt über Sensoren, die erkennen können, z. b. wenn der Server Computer eine Überhitzung aufweist oder die Spannung außerhalb des zulässigen Bereichs liegt.</span><span class="sxs-lookup"><span data-stu-id="67dc9-111">A BMC has sensors that can detect, for example, when the server computer is overheating or when voltage is out of the acceptable range.</span></span> <span data-ttu-id="67dc9-112">Zum Definieren der Architektur von BMC gibt es mehrere Standards.</span><span class="sxs-lookup"><span data-stu-id="67dc9-112">Several standards exist to define the architecture of BMC.</span></span> <span data-ttu-id="67dc9-113">Die [*Intelligent Platform Management Interface (IPMI)*](windows-remote-management-glossary.md) ist ein solcher Standard, der häufig verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="67dc9-113">The [*Intelligent Platform Management Interface (IPMI)*](windows-remote-management-glossary.md) is one such standard that is used frequently.</span></span> <span data-ttu-id="67dc9-114">Trotz des IPMI-Standards ist der Verwaltungs Zugriff auf die Server Hardware proprietäre und erfordert die Verwendung class="sxs-lookup"><span data-stu-id="67dc9-115">Also, remote access to a BMC is provided using a specialized wire protocol, Remote Management Control Protocol (RMCP), which has nonstandard security <span data-ttu-id="67dc9-116">Mit dem Microsoft [IPMI-Anbieter](/previous-versions/windows/desktop/ipmiprv/ipmi-provider) und dem IPMI-Treiber können Sie BMC-Daten von Remote Server Computern über einen standardmäßigen WMI-Anbieter mit WMI- [Klassen](/previous-versions/windows/desktop/ipmiprv/ipmi-provider)abrufen.</span><span class="sxs-lookup"><span data-stu-id="67dc9-116">The Microsoft [IPMI provider](/previous-versions/windows/desktop/ipmiprv/ipmi-provider) and IPMI driver, allow you to obtain BMC data from remote server computers through a standard WMI provider with WMI [classes](/previous-versions/windows/desktop/ipmiprv/ipmi-provider).</span></span> <span data-ttu-id="67dc9-117">Sie können zwar ein normales WMI-Skript schreiben, das Remote Daten über DCOM erhält, aber in vielen Fällen ist die bevorzugte Methode zum Abrufen von IPMI-Daten das **WinRM** -Befehlszeilen Dienstprogramm oder die [WinRM-Skript-API](winrm-scripting-api.md) oder die [WinRM-C++-API](winrm-c---api.md).</span><span class="sxs-lookup"><span data-stu-id="67dc9-117">While you can write a normal WMI script that obtains remote data through DCOM, in many cases the preferred method of obtaining IPMI data is through the **Winrm** command-line utility or the [WinRM Scripting API](winrm-scripting-api.md) or [WinRM C++ API](winrm-c---api.md).</span></span> <span data-ttu-id="67dc9-118">Das WinRM-Hilfsprogramm und die WinRM-Dienst-APIs basieren auf dem WS-Management-Protokoll und können IPMI-Daten von lokalen Computern oder Remote Computern abrufen, ohne DCOM zu verwenden.</span><span class="sxs-lookup"><span data-stu-id="67dc9-118">The Winrm utility and the WinRM service APIs rely on the WS-Management protocol and can obtain IPMI data either from local or remote computer without using DCOM.</span></span>

<span data-ttu-id="67dc9-119">Der BMC verfügt auch über eine Ereignis Datenbank mit dem Namen System Ereignisprotokoll (SEL), das Ereignisse auf dem überwachten Computer aufzeichnet.</span><span class="sxs-lookup"><span data-stu-id="67dc9-119">The BMC also has an event database called the System Event Log (SEL) which records events in the monitored computer.</span></span> <span data-ttu-id="67dc9-120">Sie können nicht abonnieren, dass diese Ereignisse wie bei WMI-Ereignis Klassen an ein Skript übermittelt werden.</span><span class="sxs-lookup"><span data-stu-id="67dc9-120">You cannot subscribe to have these events delivered to a script as you can with WMI event classes.</span></span> <span data-ttu-id="67dc9-121">Sie können Sie jedoch mit dem Befehlszeilen Tool Wecutil.exe abonnieren.</span><span class="sxs-lookup"><span data-stu-id="67dc9-121">However, you can use the Wecutil.exe command-line tool to subscribe to them.</span></span> <span data-ttu-id="67dc9-122">Weitere Informationen zur Verwendung dieses Tools erhalten Sie, wenn Sie an einer Eingabeaufforderung **wecutil/?** eingeben.</span><span class="sxs-lookup"><span data-stu-id="67dc9-122">For more information about how to use this tool, at a command prompt type **wecutil /?**.</span></span>
