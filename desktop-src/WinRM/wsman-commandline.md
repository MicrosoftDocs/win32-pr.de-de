# <a name="wsmancommandline-property"></a>WSMan.CommandLine-Eigenschaft

Ruft die Befehlszeile des Prozesses ab, der die Automatisierungskomponente lädt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
WSMan.CommandLine As BSTR
```



## <a name="property-value"></a>Eigenschaftswert

Zeichenfolge, die die Befehlszeile enthält.
