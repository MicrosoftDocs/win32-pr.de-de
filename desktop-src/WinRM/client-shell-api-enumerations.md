# <a name="client-shell-api-enumerations"></a>Client Shell-API-Enumerationen

Die folgende Tabelle enthält eine Übersicht über die Enumerationen in der Windows Client Shell-Clientschnittstelle (APPLICATION Programming Interface, API) der Windows-Clientverwaltung (WinRM).



| Enumerationen                                                 | Beschreibung                                                                                      |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**WSManAuthenticationFlags**](/windows/desktop/api/Wsman/ne-wsman-wsmanauthenticationflags) | Definiert das Authentifizierungsschema, das sowohl für die Server- als auch für die Proxyauthentifizierung verwendet werden soll.               |
| [**WSManCallbackFlags**](/windows/desktop/api/Wsman/ne-wsman-wsmancallbackflags)             | Definiert einen Satz von Flags, die von allen Rückruffunktionen verwendet werden.                                           |
| [**WSManDataType**](/windows/desktop/api/Wsman/ne-wsman-wsmandatatype)                       | Gibt den aktuellen Datentyp der Union in der [**WSMAN \_ DATA-Struktur**](/windows/desktop/api/Wsman/ns-wsman-wsman_data) an. |
| [**WSManProxyAccessType**](/windows/desktop/api/Wsman/ne-wsman-wsmanproxyaccesstype)         | Definiert den Proxyzugriffstyp.                                                                   |
| [**WSManSessionOption**](/windows/desktop/api/Wsman/ne-wsman-wsmansessionoption)             | Definiert einen Satz erweiterter Optionen für die Sitzung.                                               |
