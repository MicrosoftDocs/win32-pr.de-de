# <a name="client-shell-api-enumerations"></a>Clientshellapi-Enumerationen

In der folgenden Tabelle finden Sie eine Übersicht über die Enumerationen in der-API (Application Programming Interface) für die-Windows-Remoteverwaltung (WinRM)-Client-Shell.



| Enumerationen                                                 | BESCHREIBUNG                                                                                      |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**Wsmanauthenticationflags**](/windows/desktop/api/Wsman/ne-wsman-wsmanauthenticationflags) | Definiert das Authentifizierungsschema, das für die Server-und Proxy Authentifizierung verwendet werden soll.               |
| [**Wsmancallbackflags**](/windows/desktop/api/Wsman/ne-wsman-wsmancallbackflags)             | Definiert einen Satz von Flags, die von allen Rückruf Funktionen verwendet werden.                                           |
| [**Wsmandatatype**](/windows/desktop/api/Wsman/ne-wsman-wsmandatatype)                       | Gibt den aktuellen Datentyp der Union in der [**WSMAN- \_ Daten**](/windows/desktop/api/Wsman/ns-wsman-wsman_data) Struktur an. |
| [**Wsmanproxyaccesstype**](/windows/desktop/api/Wsman/ne-wsman-wsmanproxyaccesstype)         | Definiert den Proxy Zugriffstyp.                                                                   |
| [**WSManSessionOption**](/windows/desktop/api/Wsman/ne-wsman-wsmansessionoption)             | Definiert eine Reihe erweiterter Optionen für die Sitzung.                                               |
