# <a name="winrm-plug-in-api-structures"></a>WinRM-Plug-in-API-Strukturen

In der folgenden Tabelle finden Sie eine Übersicht über die Strukturen in der Plug-in-API (Windows-Remoteverwaltung (WinRM)-Plug-in.



| Struktur                                                        | BESCHREIBUNG                                                                              |
|------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| [**WSMAN- \_ Authz- \_ Kontingent**](/windows/desktop/api/Wsman/ns-wsman-wsman_authz_quota)                 | Definiert Kontingent Informationen pro Benutzer.                                           |
| [**WSMAN- \_ Zertifikat \_ Details**](/windows/desktop/api/Wsman/ns-wsman-wsman_certificate_details) | Stellt die Felder innerhalb des Client Zertifikats dar.                                     |
| [**WSMAN- \_ Befehl \_ arg \_ festgelegt**](/windows/desktop/api/Wsman/ns-wsman-wsman_command_arg_set)        | Stellt den Satz von Argumenten dar, die an die Befehlszeile übermittelt werden.                  |
| [**WSMAN- \_ Filter**](/windows/desktop/api/Wsman/ns-wsman-wsman_filter)                            | Definiert die für einen Vorgang verwendete Filterung.                                             |
| [**WSMAN- \_ Fragment**](/windows/desktop/api/Wsman/ns-wsman-wsman_fragment)                        | Definiert die Fragmentinformationen für einen Vorgang.                                       |
| [**Informationen zum WSMAN- \_ Vorgang \_**](/windows/desktop/api/Wsman/ns-wsman-wsman_operation_info)           | Stellt einen bestimmten Endpunkt der Ressource dar, für den das Plug-in die Anforderung ausführen muss. |
| [**WSMAN- \_ Plugin- \_ Anforderung**](/windows/desktop/api/Wsman/ns-wsman-wsman_plugin_request)           | Enthält Informationen über die Anforderung und wird an jeden Plug-in-Vorgang übermittelt.       |
| [**WSMAN- \_ Absender \_ Details**](/windows/desktop/api/Wsman/ns-wsman-wsman_sender_details)           | Gibt Client Details für jede eingehende Anforderung an.                                      |
