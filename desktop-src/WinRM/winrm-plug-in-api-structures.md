# <a name="winrm-plug-in-api-structures"></a>WinRM-Plug-In-API-Strukturen

Die folgende Tabelle enthält eine Übersicht über die Strukturen in der WinRM-Plug-In-Api (Windows Remote Management).



| Struktur                                                        | Beschreibung                                                                              |
|------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| [**WSMAN \_ \_ AUTHZ-KONTINGENT**](/windows/desktop/api/Wsman/ns-wsman-wsman_authz_quota)                 | Definiert Kontingentinformationen pro Benutzer.                                           |
| [**\_WSMAN-ZERTIFIKATDETAILS \_**](/windows/desktop/api/Wsman/ns-wsman-wsman_certificate_details) | Stellt die Felder im Clientzertifikat dar.                                     |
| [**WSMAN \_ COMMAND \_ ARG \_ SET**](/windows/desktop/api/Wsman/ns-wsman-wsman_command_arg_set)        | Stellt den Satz von Argumenten dar, die an die Befehlszeile übergeben werden.                  |
| [**\_WSMAN-FILTER**](/windows/desktop/api/Wsman/ns-wsman-wsman_filter)                            | Definiert die für einen Vorgang verwendete Filterung.                                             |
| [**\_WSMAN-FRAGMENT**](/windows/desktop/api/Wsman/ns-wsman-wsman_fragment)                        | Definiert die Fragmentinformationen für einen Vorgang.                                       |
| [**\_WSMAN-VORGANGSINFORMATIONEN \_**](/windows/desktop/api/Wsman/ns-wsman-wsman_operation_info)           | Stellt einen bestimmten Ressourcenendpunkt dar, für den das Plug-In die Anforderung ausführen muss. |
| [**\_WSMAN-PLUG-IN-ANFORDERUNG \_**](/windows/desktop/api/Wsman/ns-wsman-wsman_plugin_request)           | Enthält Informationen über die Anforderung und wird an jeden Plug-In-Vorgang übergeben.       |
| [**WSMAN \_ SENDER \_ DETAILS**](/windows/desktop/api/Wsman/ns-wsman-wsman_sender_details)           | Gibt Clientdetails für jede eingehende Anforderung an.                                      |
