# <a name="enumeratorerror-property"></a>Enumerator.Error-Eigenschaft

Ruft eine XML-Darstellung zusätzlicher Fehlerinformationen ab.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
Enumerator.Error As String
```



## <a name="property-value"></a>Eigenschaftswert

Die XML-Darstellung zusätzlicher Fehlerinformationen.
