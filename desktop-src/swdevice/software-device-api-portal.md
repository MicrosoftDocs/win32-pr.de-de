# <a name="software-device-api"></a>Softwaregeräte-API

## <a name="purpose"></a>Zweck

Sie können die Softwaregeräte-API verwenden, um ein PnP-Gerät aus einer App zu erstellen. Mit der API können Sie das Gerät als untergeordnetes Element eines vorhandenen übergeordneten Geräts aufzählen. Mit der API können Sie auch Geräteschnittstellen für die aufzählten Geräte registrieren und Eigenschaften für die Geräte und Schnittstellen festlegen.

## <a name="developer-audience"></a>Entwicklergruppe

Die Softwaregeräte-API ist für entwickler konzipiert, die Gerätefunktionen im PnP-Verzeichnis veröffentlichen oder einen Gerätetreiber laden möchten.
