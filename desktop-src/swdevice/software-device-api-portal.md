# <a name="software-device-api"></a>Software Geräte-API

## <a name="purpose"></a>Zweck

Sie können die Software Geräte-API verwenden, um ein PNP-Gerät über eine APP zu erstellen. Mit der API können Sie das Gerät als untergeordnetes Element eines beliebigen vorhandenen übergeordneten Geräts auflisten. Mit der API können Sie auch Geräteschnittstellen bei den aufgelisteten Geräten registrieren und Eigenschaften für die Geräte und Schnittstellen festlegen.

## <a name="developer-audience"></a>Entwicklergruppe

Die Software Geräte-API ist für die Verwendung durch Entwickler vorgesehen, die Gerätefunktionen im PNP-Verzeichnis veröffentlichen oder einen Gerätetreiber laden möchten.
