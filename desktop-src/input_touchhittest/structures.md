# <a name="touch-hit-testing-structures"></a>Touch-Trefferteststrukturen

Die Themen in diesem Abschnitt enthalten die Referenzspezifikationen für [Touch-Trefferteststrukturen.](touch-hit-testing-portal.md)

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
| --- | --- |
| [**TOUCH_HIT_TESTING_INPUT**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_input)<br/> | Enthält Informationen zum Berührungskontaktbereich, der vom Touchdigitalisierer gemeldet wird.<br/> |
| [**TOUCH_HIT_TESTING_PROXIMITY_EVALUATION**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_proximity_evaluation)<br/> | Enthält die Treffertestbewertung, die angibt, ob das Objekt das wahrscheinliche Ziel des Berührungskontaktbereichs ist, relativ zu anderen Objekten, die den Berührungskontaktbereich überschneiden.<br/> |
