# <a name="touch-hit-testing-structures"></a>Berührungs Treffer-Test Strukturen

Die Themen in diesem Abschnitt enthalten die Referenz Spezifikationen für die [Berührungs Treffer Test](touch-hit-testing-portal.md) -Strukturen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
| --- | --- |
| [**TOUCH_HIT_TESTING_INPUT**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_input)<br/> | Enthält Informationen zum Berührungs Kontaktbereich, der vom Fingerabdruck-Digitalisierer gemeldet wird.<br/> |
| [**TOUCH_HIT_TESTING_PROXIMITY_EVALUATION**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_proximity_evaluation)<br/> | Enthält das Treffer Testergebnis, das angibt, ob es sich bei dem Objekt um das wahrscheinliche Ziel des Berührungs Kontakt Bereichs handelt, relativ zu anderen Objekten, die den Berührungs Kontaktbereich überschneiden.<br/> |
