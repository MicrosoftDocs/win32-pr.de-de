# <a name="touch-hit-testing-scores"></a>Treffer Treffer Test-Ergebnisse

Die folgenden Konstanten identifizieren die möglichen Treffer Testergebnisse für ein Objekt relativ zu anderen Objekten, die den Berührungs Kontaktbereich überschneiden.

| Konstante/Wert | BESCHREIBUNG |
|---|---|
| **TOUCH_HIT_TESTING_PROXIMITY_CLOSEST** 0x0      | Das-Objekt ist das wahrscheinlichste Ziel.<br/>  |
| **TOUCH_HIT_TESTING_PROXIMITY_FARTHEST** 0xfff | Das-Objekt ist das am wenigsten wahrscheinliche Ziel.<br/> |
