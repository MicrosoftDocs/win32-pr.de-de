# <a name="touch-hit-testing-constants"></a>Berührungs Treffer Test-Konstanten

Die Themen in diesem Abschnitt enthalten die Referenz Spezifikationen für Konstanten, die von den [Touchscreen](touchhittest-reference.md) -APIs verwendet werden.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
| --- | --- |
| [**Verhalten der Berührungs Treffer Tests**](hit-testing-behaviors.md) | Die folgenden Konstanten werden von Anwendungen oder Benutzeroberflächen-Frameworks verwendet, um zu identifizieren, wie Nachrichten für touchtreffer Tests von Fenstern verarbeitet werden, die über die [**registertouchhittestingwindow**](/windows/win32/api/winuser/nf-winuser-registertouchhittestingwindow) -Funktion registriert werden. |
| [**Treffer Treffer Test-Ergebnisse**](hit-testing-scores.md)       | Die folgenden Konstanten identifizieren die möglichen Treffer Testergebnisse für ein Objekt relativ zu anderen Objekten, die den Berührungs Kontaktbereich überschneiden. |
