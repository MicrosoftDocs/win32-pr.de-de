# <a name="touch-hit-testing-constants"></a>Touch-Treffertest-Konstanten

Die Themen in diesem Abschnitt enthalten die Referenzspezifikationen für Konstanten, die von den [Touch-Treffertest-APIs](touchhittest-reference.md) verwendet werden.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
| --- | --- |
| [**Verhalten von Touchtreffertests**](hit-testing-behaviors.md) | Die folgenden Konstanten werden von Anwendungen oder Benutzeroberflächenframeworks verwendet, um zu identifizieren, wie Nachrichten für Touchtreffertests von Fenstern verarbeitet werden, die über die [**RegisterTouchHitTestingWindow-Funktion**](/windows/win32/api/winuser/nf-winuser-registertouchhittestingwindow) registriert werden. |
| [**Touch-Treffertestergebnisse**](hit-testing-scores.md)       | Die folgenden Konstanten identifizieren die möglichen Treffertestergebnisse für ein Objekt relativ zu anderen Objekten, die den Berührungskontaktbereich überschneiden. |
