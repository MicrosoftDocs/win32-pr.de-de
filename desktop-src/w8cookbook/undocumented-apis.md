# <a name="undocumented-apis"></a>Nicht dokumentierte APIs

Ihre Apps sollten keine nicht dokumentierten Windows-APIs aufrufen oder von bestimmten Windows-Dateiexporten oder Registrierungsschlüsseln abhängig sein. Dies kann zu Funktionsproblemen, Datenverlusten und potenziellen Sicherheitsprobleme führen. Wenn Sie für Ihre App Funktionen benötigen, die nicht verfügbar sind, nutzen Sie Ihre üblichen Feedbackkanäle und das Windows-Insider-Programm, um Feedback zu übermitteln.
