# <a name="windows-app-certification-kit"></a>Zertifizierungskit für Windows-Apps

## <a name="platforms"></a>Plattformen

**Clients:** Windows 7 \| Windows 8 \| Windows 8.1  


## <a name="description"></a>BESCHREIBUNG

Windows App Certification Kit 3.1 ist im Windows SDK für Windows 8.1. Das Kit kann vor dem Onboarding zur Vorabzertifizierung von Windows Store-Apps für Windows 8 und Windows 8.1 sowie für die Windows 7-, Windows 8- und Windows 8.1 Desktop-App-Zertifizierungsprogramme und Windows 8.1 Telefon-Apps verwendet werden.

Das Windows App Certification Kit 3.1 wurde aktualisiert, um entwicklern eine nahtlose Benutzererfahrung zu bieten, einschließlich der parallelen Ausführung von Tests, um Gesamtzeit, selektive Testauswahl und verbesserte Berichterstellung zu sparen, um einige der Verbesserungen zu nennen.

**Hinweis:** Wenn Sie zur Eingabe eines Upgrades aufgefordert werden, aktualisieren Sie Ihre Version des Windows App Certification Kit, um von den neuesten Features im Kit zu profitieren. Wenn Ihre App mit der neuesten Version des Windows App Certification Kit getestet wird, wird sichergestellt, dass sie besser mit Windows Store Richtlinien und Desktopzertifizierungsprogrammen konform ist.
