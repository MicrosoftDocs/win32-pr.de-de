# <a name="windows-app-certification-kit"></a>Zertifizierungskit für Windows-Apps

## <a name="platforms"></a>Plattformen

**Clients:** Windows 7 \| Windows 8 \| Windows 8.1  


## <a name="description"></a>BESCHREIBUNG

Das Windows-zertifizierungskit für apps 3,1 ist im Windows SDK für Windows 8.1 enthalten. Das Kit kann verwendet werden, um Windows Store-Apps für Windows 8 vorab zu zertifizieren, und Windows 8.1 vor dem Onboarding sowie für die Zertifizierungsprogramme Windows 7, Windows 8 und Windows 8.1 Desktop-App und Windows 8.1 Phone-apps.

Das Windows-zertifizierungsk3,1 IT für Windows-Apps wurde aktualisiert, um den Entwicklern eine nahtlose Benutzerfunktion bereitzustellen, einschließlich der parallelen Ausführung von Tests, um die gesamte Zeit, die selektive Testauswahl und die erweiterte Berichterstellung zu verbessern, um einige der Verbesserungen zu nennen.

**Hinweis:** Wenn Sie zur Eingabe eines Upgrades aufgefordert werden, aktualisieren Sie Ihre Version des zertifizierungskit für Windows-apps, um von den neuesten Features im Kit zu profitieren. Wenn Ihre APP mit der neuesten Version des zertifizierungskit für Windows-apps getestet wird, ist eine bessere Konformität mit Windows Store-Richtlinien und Desktop Zertifizierungsprogrammen gewährleistet.
