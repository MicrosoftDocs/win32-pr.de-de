# <a name="bluetooth-drivers"></a>Bluetooth-Treiber

IHVs und Unternehmenskunden sollten sich bewusst sein, dass Bluetooth-Treiber und Profil Pakete von Drittanbietern nicht als Teil des Upgrades von Windows 7, Windows 8 oder Windows 8.1 zu Windows 10 migriert werden. Weitere Informationen zu Bluetooth-Änderungen für Windows 10 finden Sie in [diesem Artikel](/windows-hardware/design/component-guidelines/bluetooth) .
