# <a name="windows-store-apps"></a>Windows Store-Apps

## <a name="resources"></a>Ressourcen

Die unten aufgeführten Ressourcen sollen Sie bei der Migration von Apps unterstützen, die in Fenster 8 erstellt wurden, um Windows 8.1.

-   [Aktualisieren Ihrer App von Windows 8 auf Windows 8.1](/previous-versions/windows/dn376326(v=win.10)) &mdash; Ein Whitepaper, das Anleitungen und Tipps zum Migrieren Ihrer App von Windows 8 zu Windows 8.1 von Anfang bis Ende enthält. Enthält bewährte Methoden, Verwerfungen und Anleitungen zum erneuten Senden Ihrer App an Windows Store.
-   [Neue APIs und Features für Entwickler](https://msdn.microsoft.com/library/windows/apps/dn751496.aspx) &mdash; Lernen Sie die Windows 8.1 kennen, indem Sie sich neue APIs und Features an schauen.
-   [Neutarget your Windows 8 Store app to Windows 8.1](/previous-versions/windows/apps/dn263114(v=win.10)) &mdash; Schritt-für-Schritt-Anleitung zum Migrieren Ihrer App von Windows 8 zu Windows 8.1.
-   [Aktualisierte Beispiele](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples) &mdash; Beispiele für Windows 8.1, die veranschaulichen, wie neue APIs und Features verwendet werden und wie veraltete Funktionen entschärft werden.
-   [DirectX-Programmierung](https://msdn.microsoft.com/library/windows/apps/bg182880.aspx) &mdash; Neue Features und Verbesserungen für Spiele und Grafik-Apps.
-   Informationen zu API-Änderungen in Windows 8.1 im Zusammenhang mit der Programmierung mit JavaScript und Internet Explorer finden Sie unter API-Änderungen für [Windows 8.1 (Windows Store-Apps](/previous-versions/windows/apps/dn263112(v=win.10))mit JavaScript und HTML).
-   Informationen zu API-Änderungen in Windows 8.1 im Zusammenhang mit der Programmierung mit C#/VB/C++ und XAML finden Sie unter [API-Änderungen für Windows 8.1 (Windows Store-Apps mit C#/VB/C++](/previous-versions/windows/apps/dn263110(v=win.10)) und XAML).
