# <a name="windows-store-apps"></a>Windows Store-Apps

## <a name="resources"></a>Ressourcen

Die unten aufgeführten Ressourcen dienen als Hilfe beim Migrieren von in Windows 8 erstellten apps in Windows 8.1.

-   [Aktualisieren Ihrer APP von Windows 8 auf Windows 8.1](/previous-versions/windows/dn376326(v=win.10)) &mdash; Ein Whitepaper, das Anleitungen und Tipps zum Migrieren Ihrer APP von Windows 8 zu Windows 8.1 enthält, von Anfang bis Ende. Enthält bewährte Methoden, veraltete Anwendungen und Anleitungen zum erneuten Senden Ihrer APP an den Windows Store.
-   [Neue APIs und Features für Entwickler](https://msdn.microsoft.com/library/windows/apps/dn751496.aspx) &mdash; Informieren Sie sich Windows 8.1 durch die neuen APIs und Features.
-   [Neuzuweisen Ihrer Windows 8 Store-App auf Windows 8.1](/previous-versions/windows/apps/dn263114(v=win.10)) &mdash; Schritt-für-Schritt-Anleitung zum Migrieren Ihrer APP von Windows 8 zu Windows 8.1.
-   [Aktualisierte Beispiele](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples) &mdash; Beispiele für Windows 8.1, die veranschaulichen, wie neue APIs und Features verwendet werden, und wie veraltete Anwendungen minimiert werden.
-   [DirectX-Programmierung](https://msdn.microsoft.com/library/windows/apps/bg182880.aspx) &mdash; Neue Features und Verbesserungen für Spiele und Grafik-apps.
-   Informationen zu API-Änderungen in Windows 8.1 im Zusammenhang mit der Programmierung mit JavaScript und Internet Explorer finden Sie unter [API-Änderungen für Windows 8.1 (Windows Store-Apps mit JavaScript und HTML)](/previous-versions/windows/apps/dn263112(v=win.10)).
-   Informationen zu API-Änderungen in Windows 8.1 im Zusammenhang mit der Programmierung mit c#/VB/C + + und XAML finden Sie unter [API-Änderungen für Windows 8.1 (Windows Store-Apps mit c#/VB/C + + und XAML)](/previous-versions/windows/apps/dn263110(v=win.10)) .
