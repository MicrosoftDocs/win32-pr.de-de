# <a name="new-features-and-enhancements"></a>Neue Features und Verbesserungen

In diesem Abschnitt des Dokuments werden Verbesserungen und neue Features beschrieben, die sowohl die Benutzer- als auch die Entwicklererfahrung verbessern können. Einige davon haben jedoch Auswirkungen, die Entwickler in Bezug auf ihre vorhandenen und neuen Produkte beachten müssen. Im Folgenden finden Sie eine Liste der neuen Features und Erweiterungen, die nach Funktionsbereich zusammengefasst sind:

## <a name="security"></a>Sicherheit

-   Antischadsoftware-Frühstart
-   Anforderungen an die Signierung sicherer Startfeatures für Kernelmodustreiber
-   Kontrollierter Start

## <a name="performance"></a>Leistung

-   Start-Apps
-   Robuste Dateisysteme
-   Dateiserver-API-Unterstützung
-   Automatische Wartung

## <a name="usability"></a>Benutzerfreundlichkeit

-   Eingabemethoden-Editoren von Drittanbietern

## <a name="storage-and-file-system"></a>Storage und Dateisystem

-   Nachverfolgen geänderter Bereiche einer Datei in Windows 8.1
-   Mit der neuen API können Apps TRIM- und Unmap-Hinweise an Speichermedien senden.
-   Multipfad-E/A unterstützt jetzt erweiterte Speicheranforderungsblöcke.
-   Robuste Dateisysteme
-   Dateiserver-API-Unterstützung
-   Neues Dateiversionsverlauf feature
-   Das Betriebssystem steuert jetzt die Stromversorgung für optische Laufwerke

## <a name="usb"></a>USB

-   Unterstützung für USB 3.0
