# <a name="windows-and-windows-server-compatibility-cookbook-windows-10-windows-8-windows-81-windows-server-2012-and-windows-server-2012-r2"></a>Cookbook zur Kompatibilität von Windows und Windows Server: Windows 10, Windows 8, Windows 8.1, Windows Server 2012 und Windows Server 2012 R2

## <a name="purpose"></a>Zweck

Im *Cookbook* bieten wir Informationen zu Änderungen an und neuen Features der Betriebssysteme Windows 10, Windows 8, Windows 8.1, Windows Server 2012 und Windows Server 2012 R2. Außerdem bieten wir Richtlinien für Sie, um die Kompatibilität Ihrer vorhandenen und geplanten Programme mit den neuen Betriebssystemen zu überprüfen. Wir gehen davon aus, dass Sie mit früheren Windows-Versionen vertraut sind.

Der Inhalt gilt für:

-   Windows 10
-   Windows 8.1
-   Windows Server 2012 R2
-   Windows 8
-   Windows Server 2012
-   Windows 7
-   Windows Server 2008 R2
