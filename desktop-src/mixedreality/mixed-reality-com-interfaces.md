# <a name="windows-mixed-reality-com-interfaces"></a>Windows Mixed Reality COM-Schnittstellen

Die in diesem Abschnitt enthaltenen Themen enthalten die Referenzspezifikationen für Windows Mixed Reality COM-Schnittstellen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [**ISpatialInteractionManagerInterop**](/windows/desktop/api/spatialinteractionmanagerinterop/nn-spatialinteractionmanagerinterop-ispatialinteractionmanagerinterop) | Ermöglicht die Interoperabilität mit einem [SpatialInteractionManager-Objekt](/uwp/api/Windows.UI.Input.Spatial.SpatialInteractionManager) (Universal Windows Platform, UWP) und ermöglicht den Zugriff auf **SpatialInteractionManager-Member** für den Zugriff auf Benutzereingaben von Händen, Motion Controllern und Systemstimmenbefehlen. |
| [**IHolographicSpaceInterop**](/windows/desktop/api/holographicspaceinterop/nn-holographicspaceinterop-iholographicspaceinterop) | Ermöglicht die Interoperabilität mit einem [HolographicSpace-Objekt](/uwp/api/windows.graphics.holographic.holographicspace) (Universal Windows Platform, UWP) und ermöglicht den Zugriff auf **HolographicSpace-Member** zur Darstellung einer holografischen Szene. |
