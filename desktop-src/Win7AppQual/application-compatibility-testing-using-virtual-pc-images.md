# <a name="application-compatibility-testing-using-virtual-pc-images"></a>Anwendungskompatibilitätstests mit Abbildern virtueller Computer

Um die Webanwendungs Kompatibilität zu testen, müssen Sie eine Testumgebung einrichten. Der Prozess zum Erstellen und Konfigurieren von Computern für diese Aufgabe ist zeitaufwändig und oft teuer.

Um Kompatibilitätstests zu vereinfachen, können Sie die kostenlosen, vorab erstellten VPC-Images (Application Compatibility Virtual PC) verwenden. VPC-Abbilder sind vorkonfigurierte Festplattenlaufwerke, die Sie auf vorhandene Computer laden können, auf denen die [Virtual PC-Software](https://support.microsoft.com/help/14223/windows-xp-end-of-support)ausgeführt wird. Mit diesen Images können Sie verschiedene Versionen von Microsoft Windows und Windows Internet Explorer mit vorhandenen Anwendungen in kurzer Zeit gründlich testen.

Die VPC-Images sind für alle derzeit unterstützten Versionen von Windows verfügbar und enthalten zahlreiche Kompatibilitätstest Ressourcen und-Anwendungen. Die Test Images finden Sie unter [Internet Explorer Anwendungskompatibilitäts-VPC-Image](https://www.microsoft.com/Downloads/details.aspx?FamilyID=21eabb90-958f-4b64-b5f1-73d0a413c8ef&amp;displaylang=en).
