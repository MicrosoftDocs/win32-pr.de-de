# <a name="application-compatibility-testing-using-virtual-pc-images"></a>Anwendungskompatibilitätstests mit Abbildern virtueller Computer

Um die Kompatibilität von Webanwendungen zu testen, müssen Sie eine Testumgebung einrichten. Der Prozess zum Erstellen und Konfigurieren von Computern für diese Aufgabe ist zeitaufwändig und häufig kostspielig.

Um Kompatibilitätstests zu vereinfachen, können Sie die kostenlosen, vordefinierten VPC-Images (Application Compatibility Virtual PC) verwenden. VPC-Images sind vorkonfigurierte Festplattenlaufwerk-Images, die Sie auf vorhandene Computer laden können, auf denen die [Virtual PC-Software ausgeführt wird.](https://support.microsoft.com/help/14223/windows-xp-end-of-support) Mit diesen Images können Sie verschiedene Versionen von Microsoft Windows und Windows Internet Explorer mit vorhandenen Anwendungen in kurzer Zeit gründlich testen.

Die VPC-Images sind für alle derzeit unterstützten Versionen von Windows verfügbar und enthalten viele Ressourcen und Anwendungen für Kompatibilitätstests. Informationen zum Suchen der Testimages finden Sie unter [Internet Explorer Application Compatibility VPC Image (VPC-Image zur Anwendungskompatibilität).](https://www.microsoft.com/Downloads/details.aspx?FamilyID=21eabb90-958f-4b64-b5f1-73d0a413c8ef&amp;displaylang=en)
