# <a name="windows-troubleshooting"></a>Windows Problembehandlung

## <a name="affected-platforms"></a>Betroffene Plattformen

**Clients** – Windows 7  
**Server** – Windows Server 2008 R2  









## <a name="description"></a>Beschreibung

Ein neues Windows 7 Action Center-Feature (ehemals Solutions Center) Windows Problembehandlung bietet eine systematischen Problembehandlung für den Benutzer. Das Aktionscenter ist eines der fünf Symbole, die im Systray angeheftet sind. Windows Mit der Problembehandlung können Sie In-Box-Problembehandlungspakete und Problembehandlungspakete durchsuchen oder suchen, die auf einem Microsoft-Server im Internet gespeichert sind – eine bessere BWC-Erfahrung (When Connected). Sie können ein Problembehandlungspaket auswählen und ausführen, um zu versuchen, das Problem zu beheben. Wenn Sie keine Lösung für das Problem identifizieren können, haben Sie die Möglichkeit, Hilfe-, Community- und Supportartikel oder andere Problembehandlungspakete nach relevanten verwandten Inhalten zu durchsuchen. Sollte dadurch keine Antwort auf das Problem gegeben werden, können Sie den PC auf einen Zeitpunkt wiederherstellen, bevor das Problem aufgetreten ist, oder Hilfe über Remoteunterstützung erhalten. Die Absicht besteht darin, Ihnen zu ermöglichen, eine Lösung für das Problem einfach und schnell zu finden.

## <a name="usage"></a>Verbrauch

Das Action Center ist von mehreren Standorten aus deutlich sichtbar und verfügbar. Sie können es über das Kontextmenü des Aktionscenters in systray, über die Systemsteuerung als Verknüpfungslink unter System und Wartung, über die Hauptseite des Aktionscenters und über Hilfeinhalt starten.

Problembehandlungspakete basieren auf PowerShell-Skripts. Der Prozess der Erstellung und Veröffentlichung eines Problembehandlungspakets wird öffentlich gemacht, damit OEMs, IHVs, ISVs und IT-Experten eigene Problembehandlungsinhalte entwickeln und versenden können.

Um eine konsistente Benutzererfahrung zu gewährleisten, achten Sie darauf, beim Entwerfen eigener Problembehandlungspakete die bewährten Methoden und Richtlinien zu befolgen, die im Windows-Toolkit zur Problembehandlung beschrieben sind.
