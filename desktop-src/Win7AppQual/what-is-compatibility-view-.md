# <a name="what-is-compatibility-view"></a><span data-ttu-id="6521f-103">Was ist Kompatibilitätsansicht?</span><span class="sxs-lookup"><span data-stu-id="6521f-103">What Is Compatibility View?</span></span>

<span data-ttu-id="6521f-104">*Kompatibilitätsansicht* ist ein Feature von Windows Internet Explorer 8, mit dem der Browser eine Webseite nahezu genauso rendern kann wie Windows Internet Explorer 7.</span><span class="sxs-lookup"><span data-stu-id="6521f-104">*Compatibility View* is a feature of Windows Internet Explorer 8 that enables the browser to render a webpage nearly identically to the way that Windows Internet Explorer 7 would render it.</span></span>

<span data-ttu-id="6521f-105">In Internet Explorer 8 ändert Kompatibilitätsansicht, wie der Browser Code interpretiert, der in CSS, HTML und der Dokumentobjektmodell (DOM) geschrieben ist, um zu versuchen, Internet Explorer 7 zu entsprechen.</span><span class="sxs-lookup"><span data-stu-id="6521f-105">In Internet Explorer 8, Compatibility View changes how the browser interprets code that is written in CSS, HTML, and the Document Object Model (DOM) to try to match Internet Explorer 7.</span></span> <span data-ttu-id="6521f-106">Eine Website, die ein Benutzer in Internet Explorer 8 Kompatibilitätsansicht anzeigt, ist fast identisch mit einer Website, die der Benutzer in Internet Explorer 7 anzeigt.</span><span class="sxs-lookup"><span data-stu-id="6521f-106">A site that a user views in Internet Explorer 8 Compatibility View is almost identical to a site that the user views in Internet Explorer 7.</span></span> <span data-ttu-id="6521f-107">Kompatibilitätsansicht ändert jedoch nicht, wie der Browser den gesamten Code interpretiert.</span><span class="sxs-lookup"><span data-stu-id="6521f-107">However, Compatibility View does not change how the browser interprets all code.</span></span> <span data-ttu-id="6521f-108">Beispielsweise können die Änderungen in Internet Explorer 8 für die Verarbeitung von ActiveX, dem Parser, AJAX, JavaScript, Netzwerk und Sicherheit durch den Browser weiterhin Kompatibilitätsprobleme verursachen.</span><span class="sxs-lookup"><span data-stu-id="6521f-108">For example, the changes in Internet Explorer 8 for how the browser handles ActiveX, the parser, AJAX, JavaScript, networking, and security might still cause compatibility issues.</span></span> <span data-ttu-id="6521f-109">Kompatibilitätsansicht ändert diese Verhaltensweisen nicht.</span><span class="sxs-lookup"><span data-stu-id="6521f-109">Compatibility View does not change these behaviors.</span></span>

<span data-ttu-id="6521f-110">In einer Unternehmensumgebung besteht in einigen Bereichen ein geringeres Risiko von Kompatibilitätsproblemen.</span><span class="sxs-lookup"><span data-stu-id="6521f-110">In an enterprise environment, some areas have lower risk for compatibility issues.</span></span> <span data-ttu-id="6521f-111">Websites der Intranetzone verwenden die Kompatibilitätsansicht beispielsweise standardmäßig.</span><span class="sxs-lookup"><span data-stu-id="6521f-111">For example, Intranet Zone websites use Compatibility View by default.</span></span> <span data-ttu-id="6521f-112">Clientwebanwendungen, die mithilfe des Webbrowsersteuerelements oder der WebOC (Internet Explorer Rendering-Engine) gerendert werden, weisen ebenfalls ein geringes Risiko für Kompatibilitätsprobleme auf, da Internet Explorer 8 standardmäßig einen Kompatibilitätsmodus für webOC verwendet.</span><span class="sxs-lookup"><span data-stu-id="6521f-112">Client web applications that render by using the Web Browser Control, or the WebOC (Internet Explorer rendering engine), also have a low risk for compatibility issues because Internet Explorer 8 defaults to a compatibility mode for the WebOC.</span></span> <span data-ttu-id="6521f-113">Die Standardkonfigurationseinstellungen für Kompatibilitätsansicht stellen jedoch möglicherweise keine vollständige Kompatibilität sicher.</span><span class="sxs-lookup"><span data-stu-id="6521f-113">However, the default configuration settings for Compatibility View might not ensure complete compatibility.</span></span> <span data-ttu-id="6521f-114">Um festzustellen, ob eine Website oder Webanwendung mit Internet Explorer 8 kompatibel ist, sollten Sie die Website oder Webanwendung testen.</span><span class="sxs-lookup"><span data-stu-id="6521f-114">To determine if a website or web application is compatible with Internet Explorer 8, you should test the website or web application.</span></span>
