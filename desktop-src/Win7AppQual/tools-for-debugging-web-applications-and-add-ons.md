# <a name="tools-for-debugging-web-applications-and-add-ons"></a>Tools zum Debuggen von Webanwendungen und Add-Ons

Anstatt [Kompatibilitätsprobleme in Webanwendungen und Add-Ons](remediating-web-applications-and-add-ons.md)zu beheben, können Sie die Webanwendung oder Website nativ im Windows Internet Explorer 8-Standardmodus verwenden, indem Sie Ihren Code auf die neuesten Webstandards schreiben.

Um Probleme in Ihrer Anwendung oder Website vollständig zu bewerten, sollten Sie die Grundursache des Problems untersuchen. Beispielsweise hat das Internet Explorer-Team die Grundursachen mithilfe der Entwicklertools ermittelt und mögliche Probleme logisch reduziert. In den folgenden Abschnitten werden mehrere Tools beschrieben, um den Debugprozess zu vereinfachen.



| `Section`                                                                                                                    | BESCHREIBUNG                                                                                                                                                                         |
|----------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Microsoft Expression Web SuperPreview](microsoft-expression-web-superpreview.md)                                         | Ein eigenständiges Visuelles Debugtool, mit dem Sie Ihre Websites schneller und einfacher von Microsoft Internet Explorer 6 zu Windows Internet Explorer 7 oder Internet Explorer 8 migrieren können. |
| [Internet Explorer-Kompatibilitätstest-Tool (Internet Explorer Compatibility Test Tool, IECTT)](inernet-explorer-compatibility-test-tool--iectt-.md)                  | Eine Komponente des [Microsoft Application Compatibility Toolkit,](/windows-hardware/get-started/adk-install) mit der Sie Probleme in Ihren Webanwendungen diagnostizieren können.       |
| [Internet Explorer 8-Entwicklertools](internet-explorer-8-developer-tools.md)                                             | Die Entwicklertools, die in Internet Explorer 8 für das Debuggen enthalten sind.                                                                                                         |
| [Internet Explorer 8-Kompatibilitäts-Assistent (Aakurno)](inernet-explorer-8-compatibility-wizard--aggiorno-.md)              | Ein Drittanbietertool, das Websites automatisch aktualisiert, sodass sie in Internet Explorer 8 ordnungsgemäß gerendert werden, ohne die Kompatibilität in älteren Browsern zu beeinträchtigen.                                |
| [Fiddler-Webdebuggertool](fiddler-web-debugger-tool.md)                                                                 | Ein Drittanbietertool zum Erfassen von Netzwerkdatenverkehr zwischen dem Internet und Testcomputern.                                                                                           |
| [Anwendungskompatibilitätstests mit Abbildern virtueller Computer](application-compatibility-testing-using-virtual-pc-images.md) | Virtuelle PC-Images, die Kompatibilitätstests vereinfachen.                                                                                                                              |
