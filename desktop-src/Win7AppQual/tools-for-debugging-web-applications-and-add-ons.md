# <a name="tools-for-debugging-web-applications-and-add-ons"></a>Tools zum Debuggen von Webanwendungen und Add-Ons

Anstatt [Kompatibilitätsprobleme bei Webanwendungen und Add-ons zu beheben](remediating-web-applications-and-add-ons.md), können Sie die Webanwendung oder Website im Windows Internet Explorer 8-Standardmodus nativ ausführen lassen, indem Sie Ihren Code auf die neuesten Webstandards schreiben.

Um Probleme in Ihrer Anwendung oder Website vollständig zu bewerten, sollten Sie die Ursache des Problems untersuchen. Das Internet Explorer-Team hat z. b. die Hauptursachen mithilfe des Entwicklertools ermittelt und mögliche Probleme logisch reduziert. In den folgenden Abschnitten werden verschiedene Tools beschrieben, um den Debugprozess zu vereinfachen.



| `Section`                                                                                                                    | BESCHREIBUNG                                                                                                                                                                         |
|----------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Microsoft Expression Web SuperPreview](microsoft-expression-web-superpreview.md)                                         | Ein eigenständiges visuelles Debugtool, das die Migration Ihrer Websites von Microsoft Internet Explorer 6 zu Windows Internet Explorer 7 oder Internet Explorer 8 schneller und einfacher macht. |
| [Internet Explorer-Kompatibilitätstest-Tool (Internet Explorer Compatibility Test Tool, IECTT)](inernet-explorer-compatibility-test-tool--iectt-.md)                  | Eine Komponente des [Microsoft Anwendungskompatibilitäts-Toolkits](/windows-hardware/get-started/adk-install) , die Ihnen bei der Diagnose von Problemen in Ihren Webanwendungen helfen kann.       |
| [Internet Explorer 8-Entwicklertools](internet-explorer-8-developer-tools.md)                                             | Die Entwicklertools, die in Internet Explorer 8 zum Debuggen enthalten sind.                                                                                                         |
| [Internet Explorer 8-Kompatibilitäts-Assistent (Aggiorno)](inernet-explorer-8-compatibility-wizard--aggiorno-.md)              | Ein Drittanbieter Tool, das Websites automatisch aktualisiert, sodass Sie in Internet Explorer 8 richtig gerenstet werden, ohne dass die Kompatibilität in älteren Browsern unterbrochen wird.                                |
| [Tool für den webdebugger von "Tool"](fiddler-web-debugger-tool.md)                                                                 | Ein Drittanbieter Tool zum Erfassen von Netzwerk Datenverkehr zwischen dem Internet und den Testcomputern.                                                                                           |
| [Anwendungskompatibilitätstests mit Abbildern virtueller Computer](application-compatibility-testing-using-virtual-pc-images.md) | Virtual PC-Images, die Kompatibilitätstests vereinfachen.                                                                                                                              |
