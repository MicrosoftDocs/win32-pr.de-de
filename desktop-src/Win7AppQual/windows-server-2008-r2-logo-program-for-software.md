# <a name="windows-server-2008-r2-logo-program-for-software"></a>Windows Server 2008 R2-Logo Programm für Software

## <a name="affected-platforms"></a>Betroffene Plattformen

 **Server:** Windows Server 2008 R2  

### <a name="description"></a>BESCHREIBUNG

Das Windows Server 2008 R2-Logo-Programm wurde eingestellt.

Windows Server 2008 R2 2012-Test Übermittlungen werden nicht mehr für Microsoft Partner Network (MPN) oder das Certified for Windows Server-Logo akzeptiert.

Windows Server 2012 R2 ist derzeit die bevorzugte Plattform für das Logo "Certified for Windows Server". Weitere Informationen finden Sie unter [Enterprise Cloud Application Logo Certification Program](https://azure.microsoft.com/overview/commercial-marketplace/) .
