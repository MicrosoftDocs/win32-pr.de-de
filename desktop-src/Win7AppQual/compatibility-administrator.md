# <a name="compatibility-administrator"></a>Kompatibilitätsadministrator

## <a name="affected-platforms"></a>Betroffene Plattformen

 **Clients:** Windows 2000, Windows XP, Windows Vista, Windows 7  
**Server:** Windows Server 2003, Windows Server 2008, Windows Server 2008 R2  


## <a name="description"></a>BESCHREIBUNG

Mit dem vom Application Compatibility Toolkit (ACT) bereitgestellten Kompatibilitätsadministratortool können Sie viele Ihrer potenziellen Probleme mit der Anwendungskompatibilität beheben, bevor Sie eine neue Version von Windows in Ihrer Organisation bereitstellen, indem Sie:

-   Bereitstellen von individuellen Kompatibilitätskorrekturen, Kompatibilitätsmodi und AppHelp-Meldungen, die Sie zum Beheben bestimmter Kompatibilitätsprobleme verwenden können
-   Sie können benutzerdefinierte Kompatibilitätskorrekturen, Kompatibilitätsmodi, AppHelp-Meldungen und Kompatibilitätsdatenbanken erstellen.
-   Bereitstellen eines Abfragetools, mit dem Sie nach installierten Fehlerbehebungen auf Ihren lokalen Computern suchen können
