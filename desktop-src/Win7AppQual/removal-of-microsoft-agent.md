# <a name="removal-of-microsoft-agent"></a>Entfernen des Microsoft-Agents

## <a name="platform"></a>Plattform

**Clients** -Windows 7  
**Server** -Windows Server 2008 R2  









## <a name="feature-impact"></a>Auswirkungen von Features

 **Schweregrad** -Mittel  
**Häufigkeit** -niedrig  


## <a name="description"></a>BESCHREIBUNG

Microsoft hat beschlossen, die Entwicklung von Microsoft-Agent-Technologien zu unterbrechen. Ab Windows 7 wird der Microsoft-Agent in zukünftigen Versionen des Microsoft Windows-Betriebssystems nicht mehr unterstützt. Wir empfehlen Microsoft-Agent-Anwendungsentwicklern und-reverteilern, ihre Aktivitäten im Hinblick auf diese Entscheidung zu evaluieren.

## <a name="manifestation"></a>Ausstrahlung

-   Microsoft bietet keine neuen Lizenzen oder Erneuerungen an
-   Microsoft-Agent-Komponenten werden von Microsoft nicht aktualisiert, einschließlich:
    -   Die "msagent.exe"-Laufzeit und-Bibliotheken
    -   Die Zeichen "Merlin", "Genie", "Peer" und "Robbo"
    -   Der Agent-Zeichen-Editor
    -   Alle zugehörigen Microsoft-Agent-Software,-Tools und-Dokumentationen
-   Die Entwicklung von Microsoft-Agent-Anwendungen oder-Zeichen wird von Windows 7 und höheren Versionen nicht unterstützt.
