# <a name="removal-of-microsoft-agent"></a>Entfernen von Microsoft Agent

## <a name="platform"></a>Plattform

**Clients** – Windows 7  
**Server** – Windows Server 2008 R2  









## <a name="feature-impact"></a>Auswirkung von Features

 **Schweregrad:** Mittel  
**Häufigkeit** – Niedrig  


## <a name="description"></a>BESCHREIBUNG

Microsoft hat sich entschieden, die Entwicklung von Microsoft Agent-Technologien einzustellen. Ab Windows 7 wird Microsoft Agent in zukünftigen Versionen des Microsoft Windows-Betriebssystems nicht mehr enthalten oder unterstützt. Wir empfehlen Entwicklern und Weiterverteilern von Microsoft Agent-Anwendungen, ihre Aktivitäten im Licht dieser Entscheidung zu bewerten.

## <a name="manifestation"></a>Manifestation

-   Microsoft bietet keine neuen Lizenzen oder Verlängerungen an.
-   Microsoft aktualisiert die Microsoft Agent-Komponenten nicht, einschließlich:
    -   Die "msagent.exe" Runtime und Bibliotheken
    -   Die Zeichen Merlin,Ige, Peedy und Sollo
    -   Der Zeichen-Editor des Agents
    -   Alle zugehörigen Microsoft Agent-Software, -Tools und -Dokumentation
-   Windows 7 und neuere Versionen unterstützen nicht die Entwicklung von Microsoft Agent-Anwendungen oder -Zeichen.

## <a name="mitigation"></a>Minderung

Entwickler können weiterhin die folgenden Ressourcen für frühere Versionen von Windows verwenden:

-   [Microsoft Agent Core Components (nur zur Verwendung unter Microsoft Windows 95, Windows 98 und Windows NT 4.0)](https://www.microsoft.com/downloads/details.aspx?familyid=E11BF712-7862-45BA-826D-44AE3A11836F)
-   [Merlin-Zeichendatei](https://www.microsoft.com/downloads/details.aspx?FamilyID=fee1dadd-2f23-41d0-8a81-2affd74c0aa5)
-   [Peedy-Zeichendatei](https://www.microsoft.com/downloads/details.aspx?FamilyID=bd3c4655-79e4-4791-ab9d-abc7bbd133ef)

-   [Zeichenfolgendatei](https://www.microsoft.com/downloads/details.aspx?FamilyID=fa36d1d5-d828-494a-ad0a-7b571db5bd2e)
