# <a name="removal-of-microsoft-agent"></a>Entfernen des Microsoft-Agents

## <a name="platform"></a>Plattform

**Clients** – Windows 7  
**Server** – Windows Server 2008 R2  









## <a name="feature-impact"></a>Auswirkungen auf Features

 **Schweregrad –** Mittel  
**Häufigkeit** : Niedrig  


## <a name="description"></a>Beschreibung

Microsoft hat sich entschieden, die Entwicklung von Microsoft Agent-Technologien einzustellen. Ab Windows 7 wird der Microsoft-Agent in zukünftigen Versionen des Microsoft Windows Betriebssystems nicht mehr enthalten oder unterstützt. Wir empfehlen Entwicklern und Weiterverteilern von Microsoft-Agent-Anwendungen, ihre Aktivitäten im Hinblick auf diese Entscheidung zu bewerten.

## <a name="manifestation"></a>Manifestation

-   Microsoft bietet keine neuen Lizenzen oder Verlängerungen an.
-   Microsoft aktualisiert die Komponenten des Microsoft-Agents nicht, einschließlich:
    -   Laufzeit und Bibliotheken "msagent.exe"
    -   Die Zeichen Merlin, Genie, Peedy und Sollo
    -   Der Agent-Zeichen-Editor
    -   Alle zugehörigen Microsoft Agent-Software, -Tools und -Dokumentation
-   Windows 7 und höher unterstützt die Entwicklung von Microsoft-Agent-Anwendungen oder -Zeichen nicht.

Entwickler können weiterhin die folgenden Ressourcen für frühere Versionen von Windows verwenden:

-   [Microsoft Agent Core Components (nur zur Verwendung in Microsoft Windows 95, Windows 98 und Windows NT 4.0)](https://www.microsoft.com/downloads/details.aspx?familyid=E11BF712-7862-45BA-826D-44AE3A11836F)
-   [Merlin-Zeichendatei](https://www.microsoft.com/downloads/details.aspx?FamilyID=fee1dadd-2f23-41d0-8a81-2affd74c0aa5)
