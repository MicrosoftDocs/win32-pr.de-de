# <a name="application-compatibility-toolkit-act"></a>Anwendungskompatibilitäts-Toolkit (Application Compatibility Toolkit, ACT)

> [!IMPORTANT]
> Die in diesem Artikel behandelten Versionen des Application Compatibility Toolkit werden nicht mehr unterstützt. Die letzte unterstützte Version ist im [Windows 10 Assessment and Deployment Kit](/archive/blogs/yongrhee/download-windows-10-assessment-and-deployment-kit-adk) enthalten.

## <a name="affected-platforms"></a>Betroffene Plattformen

**Clients:** Windows XP \| Windows Vista \| Windows 7  


**Server:** Windows Server 2003 \| Windows Server 2008 \| Windows Server 2008 R2  


## <a name="description"></a>BESCHREIBUNG

Das Microsoft Application Compatibility Toolkit (ACT) ist ein Tool für die Lebenszyklusverwaltung, das Sie bei der Identifizierung und Verwaltung Ihres gesamten Anwendungsportfolios unterstützt, die Kosten und den Zeitaufwand für die Behebung von Anwendungskompatibilitätsproblemen reduziert und Sie dabei unterstützt, Windows und Windows Updates schnell bereitzustellen.

**Verwendung**

Mit dem ACT haben Sie folgende Möglichkeiten:

-   Analysieren Ihres Anwendungs-, Websites- und Computerportfolios
-   Bewerten von Betriebssystembereitstellungen, der Auswirkungen von Betriebssystemupdates und ihrer Kompatibilität mit Websites
-   Zentrales Verwalten von Kompatibilitätsauswertungen und Konfigurationseinstellungen
-   Rationalisieren und Organisieren von Anwendungen, Websites und Computern
-   Priorisieren von Anwendungskompatibilitätsbemühungen mit gefilterter Berichterstellung
-   Hinzufügen und Verwalten von Problemen und Lösungen für Ihre Enterprise Computing-Umgebung
-   Bereitstellen automatisierter Gegenmaßnahmen für bekannte Kompatibilitätsprobleme
-   Senden und Empfangen von Kompatibilitätsinformationen vom Microsoft Compatibility Exchange
