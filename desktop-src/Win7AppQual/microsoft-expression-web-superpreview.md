# <a name="microsoft-expression-web-superpreview"></a>Microsoft Expression Web SuperPreview

Microsoft Expression Web Super Preview für Internet Explorer ist ein eigenständiges visuelles Debugtool, das Sie beim Debuggen Ihrer Websites für Browser übergreifende Kompatibilität unterstützt. Web SuperPreview stellt sicher, dass Ihre Websites in Windows Internet Explorer 8 ordnungsgemäß funktionieren und die Kompatibilität mit früheren Versionen von Windows Internet Explorer aufrechterhalten.

Web SuperPreview zeigt, wie Ihre Webseiten in Microsoft Internet Explorer 6 und entweder Windows Internet Explorer 7 oder Internet Explorer 8 dargestellt werden, abhängig von der Version, die Sie auf Ihrem Computer installieren. Sie können die Seiten nebeneinander oder als Überlagerung anzeigen. Sie können Lineale, Führungslinien und Zoom-und Schwenken-Tools verwenden, um Unterschiede im Layout exakt zu identifizieren. Sie können Ihre Seite sogar mit dem Rendering der Seite durch die Ziel Browser vergleichen.

Web SuperPreview identifiziert auch das Tag, die Größe, die Position, die angewendeten Stile und den Speicherort eines Elements in der Struktur des Dokumentobjektmodell (DOM), damit Sie den Fehler schnell beheben können.

> [!Note]  
> Web SuperPreview ist eine kostenlose Anwendung, die über keinen Ablauf und keinen technischen Support von Microsoft verfügt.

 

Mit dem websuperpreview-Tool können Sie folgende Aktionen ausführen:

-   Debuggen von Seiten auf dem Entwicklungs Computer. Web Super Preview funktioniert direkt mit installierten Microsoft Windows-Browsern und umfasst das Internet Explorer 6-Renderingmodul. Sie können diese Engine nebeneinander zusammen mit anderen Versionen von Internet Explorer oder Mozilla Firefox verwenden.
-   Schnelleres Debuggen mithilfe von Element-und Dom-Strukturinformationen. Web SuperPreview kombiniert auch Bild Rendering mit Dom-Daten, sodass Sie bestimmte DOM-Elemente innerhalb des Bilds hervorheben und Unterschiede zwischen den Browsern anzeigen können. Das Tool stellt Element Informationen wie Position, Dimensionen und ID bereit, um das Element zu finden, das das Kompatibilitätsproblem verursacht.
-   Unterstützung für Windows-und Macintosh-Browser, einschließlich Internet Explorer 6 und höher, Firefox 3,6 und Macintosh Safari.

Weitere Informationen zu diesem Tool und anderen Entwicklertools finden Sie unter [Expression Studio 4 Web Overview](https://www.microsoft.com/expression/products/StudioWebPro_Overview.aspx), [Expression Web Demo](https://msdn.microsoft.com/expression/ff723803.aspx)und [Expression Forums](https://social.msdn.microsoft.com/forums/announce/threads).
