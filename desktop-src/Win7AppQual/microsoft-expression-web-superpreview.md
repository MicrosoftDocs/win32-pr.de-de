# <a name="microsoft-expression-web-superpreview"></a>Microsoft Expression Web SuperPreview

Microsoft Expression Web SuperPreview for Internet Explorer ist ein eigenständiges visuelles Debugtool, mit dem Sie Ihre Websites debuggen können, um browserübergreifende Kompatibilität zu gewährleisten. Web SuperPreview stellt sicher, dass Ihre Websites in Windows Internet Explorer 8 ordnungsgemäß funktionieren, und behält die Kompatibilität mit früheren Versionen von Windows Internet Explorer bei.

Web SuperPreview zeigt, wie Ihre Webseiten in Microsoft Internet Explorer 6 und windows Internet Explorer 7 oder Internet Explorer 8 gerendert werden, je nachdem, welche Version Sie auf Ihrem Computer installieren. Sie können die Seiten nebeneinander oder als Überlagerung anzeigen. Sie können Lineals, Handbücher und Zoom- und Schwenktools verwenden, um Unterschiede im Layout genau zu identifizieren. Sie können Ihre Seite sogar mit dem Rendern der Seite durch die Zielbrowser vergleichen.

Web SuperPreview identifiziert auch das Tag, die Größe, die Position, die angewendeten Stile und die Position eines Elements in der DOM-Struktur (Dokumentobjektmodell), sodass Sie den Fehler schnell beheben können.

> [!Note]  
> Web SuperPreview ist eine kostenlose Anwendung ohne Ablauf und ohne technischen Support von Microsoft.

 

Mit dem Web SuperPreview-Tool haben Sie folgende Möglichkeiten:

-   Debuggen von Seiten auf Ihrem Entwicklungscomputer. Web SuperPreview funktioniert direkt mit installierten Microsoft Windows-Browsern und enthält die Internet Explorer 6-Rendering-Engine. Sie können diese Engine zusammen mit anderen Versionen von Internet Explorer oder Mozilla Firefox verwenden.
-   Debuggen Sie schneller, indem Sie Element- und DOM-Strukturinformationen verwenden. Web SuperPreview kombiniert auch Bildrendering mit DOM-Daten, sodass Sie bestimmte DOM-Elemente innerhalb des Bilds hervorheben und Unterschiede zwischen Browsern anzeigen können. Das Tool stellt Elementinformationen wie Position, Dimensionen und ID bereit, damit Sie das Element finden können, das das Kompatibilitätsproblem verursacht.
-   Unterstützung von Windows- und Macintosh-Browsern, einschließlich Internet Explorer 6 und höher, Firefox 3.6 und Macintosh Safari.
