# <a name="microsoft-expression-web-superpreview"></a>Microsoft Expression Web SuperPreview

Microsoft Expression Web SuperPreview for Internet Explorer ist ein eigenständiges Tool zum Debuggen visueller Elemente, mit dem Sie Ihre Websites für browserübergreifende Kompatibilität debuggen können. Web SuperPreview stellt sicher, dass Ihre Websites in Windows Internet Explorer 8 ordnungsgemäß funktionieren und die Kompatibilität mit früheren Versionen von Windows Internet Explorer.

Web SuperPreview zeigt, wie Ihre Webseiten in Microsoft Internet Explorer 6 und entweder Windows Internet Explorer 7 oder Internet Explorer 8 gerendert werden, je nachdem, welche Version Sie auf Ihrem Computer installieren. Sie können die Seiten nebeneinander oder als Überlagerung anzeigen. Sie können Lineals, Leitfäden und Zoom- und Schwenktools verwenden, um Unterschiede im Layout genau zu identifizieren. Sie können ihre Seite sogar mit dem Rendern der Seite durch die Zielbrowser vergleichen.

Web SuperPreview identifiziert auch tag, size, position, applied styles und location eines Elements in der Dokumentobjektmodell(DOM)-Struktur, sodass Sie den Fehler schnell beheben können.

> [!Note]  
> Web SuperPreview ist eine kostenlose Anwendung ohne Ablauf und ohne technischen Support von Microsoft.

 

Mit dem Web SuperPreview-Tool können Sie:

-   Debugseiten auf dem Entwicklungscomputer. Web SuperPreview funktioniert direkt mit installierten Microsoft Windows-Browsern und enthält die Internet Explorer 6-Rendering-Engine. Sie können diese Engine zusammen mit anderen Versionen von Internet Explorer Oder Mozilla Firefox verwenden.
-   Debuggen Sie schneller, indem Sie Element- und DOM-Strukturinformationen verwenden. Web SuperPreview kombiniert auch Bildrendering mit DOM-Daten, sodass Sie bestimmte DOM-Elemente innerhalb des Bilds hervorheben und Unterschiede zwischen Browsern anzeigen können. Das Tool stellt Elementinformationen wie Position, Dimensionen und ID zur Verfügung, damit Sie das Element finden können, das das Kompatibilitätsproblem verursacht.
-   Unterstützung Windows und Macintosh-Browsern, einschließlich Internet Explorer 6 und neueren Versionen, Firefox 3.6 und Macintosh Safari.

Weitere Informationen zu diesem Tool und anderen Entwicklertools finden Sie in den [Foren Expression Studio 4 Web overview (Expression Studio 4 Web overview),](https://www.microsoft.com/expression/products/StudioWebPro_Overview.aspx) [Expression Web demo (Ausdruckswebdemo)](https://msdn.microsoft.com/expression/ff723803.aspx)und [Expression (Ausdruck).](https://social.msdn.microsoft.com/forums/announce/threads)
