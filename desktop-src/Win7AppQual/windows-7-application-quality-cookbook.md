# <a name="windows-7-and-windows-server-2008-r2-application-quality-cookbook"></a>Cookbook zur Anwendungsqualität in Windows 7 und Windows Server 2008 R2

Das *Cookbook zur Anwendungsqualität in Windows 7 und Windows Server 2008 R2* macht Anwendungsentwickler mit der Überprüfung der Kompatibilität Ihrer Anwendungen mit dem neuen Betriebssystem vertraut und bietet einen Überblick über die wenigen bekannten Probleme mit der Anwendungs Kompatibilität in Windows 7 und Windows Server 2008 R2. Außerdem werden Unterschiede in der Leistung, Zuverlässigkeit und Nutzbarkeit hervorgeht, und es werden Links zu detaillierten Whitepapers und anderen Anleitungen für Entwickler bereitstellen.

## <a name="overview"></a>Übersicht

Windows 7 und Windows Server 2008 R2 stellen die neueste Betriebssystem Technologie und Software Entwicklungsplattform für die Verwendung durch Anwendungsentwickler und Unternehmen weltweit bereit. Im Rahmen der weiteren Verbesserung der Sicherheit, Zuverlässigkeit, Leistung und Benutzer Leistung von Windows wurden viele neue Features eingeführt, vorhandene Features wurden verbessert, und einige Features wurden entfernt.

Obwohl Windows 7 und Windows Server 2008 R2 mit den meisten der Anwendungen, die für Windows XP, Windows Server 2003, Windows Vista, Windows Server 2008, Windows Server 2008 R2 und die zugehörigen Service Packs geschrieben wurden, stark kompatibel sind, sind einige Kompatibilitäts Umbrüche aufgrund von Innovationen, Sicherheits verschärbungen und erhöhter Zuverlässigkeit unvermeidlich. Insgesamt ist die Kompatibilität von Windows 7 und Windows Server 2008 R2 mit vorhandenen Anwendungen hoch.

Dieses Dokument baut auf den Konzepten auf, die im [Cookbook für die Anwendungs Kompatibilität von Windows Vista und Windows Server 2008](/previous-versions/bb757005(v=msdn.10))enthalten sind. (Diese Ressource ist möglicherweise in einigen Sprachen und Ländern/Regionen nicht verfügbar.) Dieses Dokument bietet Ihnen die Möglichkeit, sich mit der Überprüfung der Kompatibilität Ihrer Anwendungen mit dem neuen Betriebssystem vertraut zu machen und bietet einen Überblick über die wenigen bekannten Anwendungs Inkompatibilitäts Probleme in Windows 7 und Windows Server 2008 R2. Aber darüber hinaus zeigt Sie auch Unterschiede in der Leistung, Zuverlässigkeit und Benutzerfreundlichkeit und bietet Links zu detaillierten Whitepapers und anderen Anleitungen für Entwickler.

Außerdem investiert Microsoft in verschiedene neue und erweiterte Features und Tools, mit denen Sie Anwendungen höherer Qualität erstellen und Probleme beheben können, wenn Anwendungen unter Windows 7 und Windows Server 2008 R2 nicht ordnungsgemäß funktionieren. Zu diesen Tools gehören Anweisungen zum Qualifizieren von Client-und Server Anwendungen für die Teilnahme am Windows-Logo-Programm.

Dieses *Cookbook* enthält mehr als drei Dutzend Themen, die in drei Hauptabschnitte unterteilt sind:

-   [Kompatibilität](compatibility.md) (sowohl allgemein als auch Server spezifisch)
-   [Neue Features und Verbesserungen](new-features-and-enhancements.md)
-   [Tools, bewährte Methoden und Anleitungen](tools--resources--and-guidance.md)

Wir laden Sie ein, sich mit diesen Themen vertraut zu machen, um zu erfahren, wie Sie Ihre Anwendung optimieren, um von Windows 7 profitieren zu können.

Wir werden Ihre Vorschläge für Verbesserungen an diesem Dokument ernst nehmen.
