# <a name="net-30-and-35-now-on-server-core"></a>.NET 3.0 und 3.5 jetzt unter Server Core

## <a name="platform"></a>Plattform

**Server** – Windows Server 2008 R2  



## <a name="feature-impact"></a>Auswirkung von Features

 **Schweregrad:** Niedrig  
**Häufigkeit** – Niedrig  





## <a name="description"></a>BESCHREIBUNG

Die Server Core-Installationsoption für Windows Server 2008 R2 enthält jetzt eine Teilmenge der .NET Framework 3.0 und 3.5, insbesondere WCF, WF, LINQ.

Die Server Core-Installationsoption für Windows Server 2008 wurde nicht von .NET unterstützt.

## <a name="manifestation-of-impact"></a>Auswirkungen

Dadurch können einige WCF- und WF-basierte Anwendungen sowie Anwendungen, die LINQ verwenden, unter Server Core in Windows Server 2008 R2 ausgeführt werden.

## <a name="testing"></a>Testen

Testen Sie Anwendungen, die WCF-, WF- und LINQ-Code in Server Core integrieren.
