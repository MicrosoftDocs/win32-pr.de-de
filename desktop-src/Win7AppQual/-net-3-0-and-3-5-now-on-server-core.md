# <a name="net-30-and-35-now-on-server-core"></a>.NET 3,0 und 3,5 jetzt auf Server Core

## <a name="platform"></a>Plattform

**Server** -Windows Server 2008 R2  



## <a name="feature-impact"></a>Auswirkungen von Features

 **Schweregrad** -niedrig  
**Häufigkeit** -niedrig  





## <a name="description"></a>BESCHREIBUNG

Die Server Core-Installationsoption für Windows Server 2008 R2 enthält jetzt eine Teilmenge der .NET Framework 3,0 und 3,5, insbesondere WCF, WF, LINQ.

In der Server Core-Installationsoption für Windows Server 2008 wurde keine .NET-Unterstützung unterstützt.

## <a name="manifestation-of-impact"></a>Erscheinung der Auswirkung

Dadurch können einige WCF-und WF-basierte Anwendungen sowie Anwendungen, die LINQ verwenden, auf Server Core in Windows Server 2008 R2 ausgeführt werden.

## <a name="testing"></a>Test

Testen Sie Anwendungen, die WCF-, WF-und LINQ-Code auf Server Core enthalten.
