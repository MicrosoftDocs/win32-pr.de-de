## <a name="integrated-and-simple-to-use"></a><span data-ttu-id="15d9b-107">Integriert und einfach zu verwenden</span><span class="sxs-lookup"><span data-stu-id="15d9b-107">Integrated and Simple to Use</span></span>

<span data-ttu-id="15d9b-108">Die Entwicklertools sind in jeder Installation von Internet Explorer 8 enthalten, sodass Sie Probleme auf allen Computern Debuggen können, auf denen Internet Explorer 8 ausgeführt wird.</span><span class="sxs-lookup"><span data-stu-id="15d9b-108">The Developer Tools are included with every installation of Internet Explorer 8, so you can debug issues on any computer that is running Internet Explorer 8.</span></span> <span data-ttu-id="15d9b-109">Außerdem werden die Tools nur dann geladen, wenn Sie Sie benötigen, um die Auswirkungen auf die Leistung des Browsers einzuschränken.</span><span class="sxs-lookup"><span data-stu-id="15d9b-109">In addition, the tools load only when you need them to limit how they impact the browser's performance.</span></span> <span data-ttu-id="15d9b-110">Wenn Sie die Entwicklertools verwenden, können Sie schnell nur den aktuellen Internet Explorer-Prozess Debuggen, anstatt den gesamten Internet Explorer zu debuggen.</span><span class="sxs-lookup"><span data-stu-id="15d9b-110">By using the Developer Tools, you can rapidly debug only the current Internet Explorer process, instead of debugging for all of Internet Explorer.</span></span>

## <a name="provide-a-visual-interface-to-the-platform"></a><span data-ttu-id="15d9b-111">Bereitstellen einer visuellen Oberfläche für die Plattform</span><span class="sxs-lookup"><span data-stu-id="15d9b-111">Provide a Visual Interface to the Platform</span></span>

<span data-ttu-id="15d9b-112">Anstatt Reverse Engineering, wie Ihre Website funktioniert, oder Ihre Website so zu ändern, dass Debuginformationen ausgegeben werden, können Sie mit dem Entwicklertools Internet Explorer anzeigen, um die Darstellung Ihrer Website anzuzeigen.</span><span class="sxs-lookup"><span data-stu-id="15d9b-112">Instead of reverse engineering how your website works or modifying your site to output debug information, the Developer Tools let you look into Internet Explorer to view its representation of your site.</span></span> <span data-ttu-id="15d9b-113">Diese Ansicht reduziert den Zeitaufwand für das Debuggen dynamischer Websites, bei dem die Quell Überprüfung nicht sinnvoll ist, oder das Untersuchen eines Internet Explorer-spezifischen Verhaltens, bei dem ein generisches Authoring Tool nicht hilfreich ist.</span><span class="sxs-lookup"><span data-stu-id="15d9b-113">This view reduces the time that you spend debugging dynamic sites, where source inspection is not useful, or investigating an Internet Explorer-specific behavior, where a generic authoring tool cannot help.</span></span>

## <a name="enable-fast-experimentation"></a><span data-ttu-id="15d9b-114">Schnelles experimentieren aktivieren</span><span class="sxs-lookup"><span data-stu-id="15d9b-114">Enable Fast Experimentation</span></span>
