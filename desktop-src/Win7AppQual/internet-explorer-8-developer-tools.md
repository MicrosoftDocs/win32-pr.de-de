# <a name="internet-explorer-8-developer-tools"></a>Internet Explorer 8-Entwicklertools

Windows Internet Explorer 8 umfasst das Entwicklertools Feature. Diese Funktion ermöglicht es Ihnen, Microsoft JScript schnell zu debuggen, ein Windows Internet Explorer-spezifisches Verhalten zu untersuchen oder schnell zu Prototypen oder eine neue Entwurfslösung zu testen.

In den folgenden Abschnitten werden die wichtigen Eigenschaften der Entwicklertools in Internet Explorer 8 beschrieben.

## <a name="integrated-and-simple-to-use"></a>Integriert und einfach zu verwenden

Die Entwicklertools sind in jeder Installation von Internet Explorer 8 enthalten, sodass Sie Probleme auf allen Computern Debuggen können, auf denen Internet Explorer 8 ausgeführt wird. Außerdem werden die Tools nur dann geladen, wenn Sie Sie benötigen, um die Auswirkungen auf die Leistung des Browsers einzuschränken. Wenn Sie die Entwicklertools verwenden, können Sie schnell nur den aktuellen Internet Explorer-Prozess Debuggen, anstatt den gesamten Internet Explorer zu debuggen.

## <a name="provide-a-visual-interface-to-the-platform"></a>Bereitstellen einer visuellen Oberfläche für die Plattform

Anstatt Reverse Engineering, wie Ihre Website funktioniert, oder Ihre Website so zu ändern, dass Debuginformationen ausgegeben werden, können Sie mit dem Entwicklertools Internet Explorer anzeigen, um die Darstellung Ihrer Website anzuzeigen. Diese Ansicht reduziert den Zeitaufwand für das Debuggen dynamischer Websites, bei dem die Quell Überprüfung nicht sinnvoll ist, oder das Untersuchen eines Internet Explorer-spezifischen Verhaltens, bei dem ein generisches Authoring Tool nicht hilfreich ist.

## <a name="enable-fast-experimentation"></a>Schnelles experimentieren aktivieren

Wenn Sie in früheren Versionen von Internet Explorer ein neues Design oder Test Korrekturen erstellen, bearbeiten Sie Ihre Quelle wahrscheinlich, speichern Sie, aktualisieren Sie die Seite im Browser, und wiederholen Sie dann den Vorgang. Der Entwicklertools dieses Szenario optimieren, indem es Ihnen ermöglicht, Ihre Website im Browser zu bearbeiten und die Änderungen sofort anzuzeigen.

## <a name="optimize-application-performance"></a>Optimieren der Anwendungsleistung

Um Leistungsprobleme zu identifizieren und zu beheben, werden Sie wahrscheinlich durchlaufen, indem Sie sich jeweils auf ein Szenario konzentrieren. Mithilfe des Skript-Profilers in der Entwicklertools können Sie Statistiken erfassen, einschließlich der Ausführungszeit und der Häufigkeit, mit der eine JavaScript-Funktion aufgerufen wird. Diese Informationen können Sie verwenden, wenn Sie die Anwendung testen und den Profil Bericht verwenden, um Leistungsengpässe schnell zu identifizieren und zu beheben.

Um auf die Entwicklertools zuzugreifen, drücken Sie F12, während Sie eine Webseite anzeigen, oder klicken Sie **im Menü Extras** auf **Entwicklertools** .
