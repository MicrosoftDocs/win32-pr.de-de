# <a name="internet-explorer-8-developer-tools"></a>Internet Explorer 8-Entwicklertools

Windows Internet Explorer 8 enthält das feature Entwicklertools. Mit diesem Feature können Sie Microsoft JScript schnell debuggen, ein Windows-Internet Explorer-spezifisches Verhalten untersuchen oder schnell einen Prototyp erstellen oder eine neue Entwurfslösung testen.

In den folgenden Abschnitten werden die wichtigen Merkmale der Entwicklertools in Internet Explorer 8 beschrieben.

## <a name="integrated-and-simple-to-use"></a>Integriert und einfach zu verwenden

Die Entwicklertools sind in jeder Installation von Internet Explorer 8 enthalten, sodass Sie Probleme auf jedem Computer debuggen können, auf dem Internet Explorer 8 ausgeführt wird. Darüber hinaus werden die Tools nur geladen, wenn Sie sie benötigen, um die Auswirkungen auf die Leistung des Browsers einzuschränken. Mithilfe der Entwicklertools können Sie schnell nur den aktuellen Internet Explorer Prozess debuggen, anstatt alle Internet Explorer zu debuggen.

## <a name="provide-a-visual-interface-to-the-platform"></a>Bereitstellen einer grafischen Benutzeroberfläche für die Plattform

Anstatt die Funktionsweise Ihrer Website umgekehrt zu gestalten oder Ihre Website so zu ändern, dass Debuginformationen ausgegeben werden, können Sie mit dem Entwicklertools Internet Explorer untersuchen, um ihre Darstellung Ihrer Website anzuzeigen. Diese Ansicht reduziert die Zeit für das Debuggen dynamischer Websites, bei denen die Quelluntersuchung nicht sinnvoll ist, oder das Untersuchen eines Internet Explorer-spezifischen Verhaltens, bei dem ein generisches Erstellungstool nicht helfen kann.

## <a name="enable-fast-experimentation"></a>Aktivieren von schnellen Experimenten

Wenn Sie in früheren Versionen von Internet Explorer einen neuen Entwurf erstellen oder Fixes testen, bearbeiten Sie wahrscheinlich Ihre Quelle, speichern sie, aktualisieren Ihre Seite im Browser und wiederholen sie. Die Entwicklertools dieses Szenario optimieren, indem Sie Es Ihnen ermöglichen, Ihre Website im Browser zu bearbeiten und Änderungen sofort anzuzeigen.

## <a name="optimize-application-performance"></a>Optimieren der Anwendungsleistung

Um Leistungsprobleme zu identifizieren und zu beheben, iterieren Sie wahrscheinlich, indem Sie sich jeweils auf ein Szenario konzentrieren. Mithilfe des Skriptprofilors im Entwicklertools können Sie Statistiken erfassen, einschließlich der Ausführungszeit und der Anzahl der Aufrufe einer JavaScript-Funktion. Sie können diese Informationen verwenden, während Sie Ihre Anwendung testen und den Profilbericht verwenden, um Leistungsengpässe schnell zu identifizieren und zu beheben.

Drücken Sie F12, während Sie eine Webseite anzeigen, oder klicken Sie im Menü **Extras** auf **Entwicklertools,** um auf die Entwicklertools zuzugreifen.
