# <a name="internet-explorer-8-compatibility-wizard-aggiorno"></a>Internet Explorer 8-Kompatibilitäts-Assistent (Aggiorno)

Der Kompatibilitäts-Assistent in Windows Internet Explorer 8 aktualisiert Websites automatisch so, dass Sie in Internet Explorer 8 richtig gerehüsselt werden, ohne die Kompatibilität in älteren Browsern zu beeinträchtigen Der Assistent automatisiert die Zielversion der Browserversion und verwaltet das Aussehen und die Funktionalität der Seiten. Weitere Informationen zu diesem Assistenten finden Sie unter [Internet Explorer 8-Kompatibilitäts-Assistent Schnellstart](https://www.mobilize.net/).
