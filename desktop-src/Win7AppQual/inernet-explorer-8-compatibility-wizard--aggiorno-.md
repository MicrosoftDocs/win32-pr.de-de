# <a name="internet-explorer-8-compatibility-wizard-aggiorno"></a>Internet Explorer 8-Kompatibilitäts-Assistent (Aakurno)

Der Kompatibilitäts-Assistent in Windows Internet Explorer 8 aktualisiert Websites automatisch so, dass sie in Internet Explorer 8 ordnungsgemäß gerendert werden, ohne ihre Kompatibilität in älteren Browsern zu beeinträchtigen. Der Assistent automatisiert die Zielaufgabe für browserversionen, wobei das Aussehen und die Funktionalität der Seiten beibehalten werden. Weitere Informationen zu diesem Assistenten finden Sie unter [Internet Explorer 8-Kompatibilitäts-Assistent Schnellstart](https://www.mobilize.net/).
