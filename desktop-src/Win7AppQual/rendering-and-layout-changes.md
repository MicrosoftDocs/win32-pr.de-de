# <a name="rendering-and-layout-changes"></a>Rendering-und Layoutänderungen

Windows Internet Explorer 8 interpretiert das HTML-Markup genauer als vorherige Versionen, und diese Änderung kann dazu führen, dass Webseiten falsch angezeigt werden. Wenn ein Standort beispielsweise eine Liste oder Tabelle in Windows Internet Explorer 7 ordnungsgemäß anzeigt, wird dieser in Internet Explorer 8 möglicherweise nicht angezeigt.

Sie können diese Probleme beheben, indem Sie die Webseite so reduzieren, dass Sie das falsch formatierte HTML-Format findet. Wenn Sie die meisten Anzeigeprobleme beheben, lösen die Änderungen in früheren Versionen von Windows Internet Explorer keine Probleme aus, da in den vorherigen Versionen einfach das überflüssige HTML ignoriert wurde. Weitere Informationen zu diesen Änderungen finden Sie im Blogbeitrag [Website Kompatibilität und IE8](/archive/blogs/ie/site-compatibility-and-ie8) .
