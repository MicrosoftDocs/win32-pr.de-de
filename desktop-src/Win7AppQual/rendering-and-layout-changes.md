# <a name="rendering-and-layout-changes"></a>Rendering- und Layoutänderungen

Windows Internet Explorer 8 interpretiert HTML-Markup strenger als frühere Versionen, und diese Änderung kann dazu führen, dass Webseiten falsch angezeigt werden. Wenn eine Website beispielsweise eine Liste oder Tabelle in Windows Internet Explorer 7 ordnungsgemäß anzeigt, wird sie möglicherweise in Internet Explorer 8 als fehlerhaft angezeigt.

Sie können diese Arten von Problemen beheben, indem Sie die Webseite reduzieren, um den falsch formatierten HTML-Code zu finden. Wenn Sie die meisten Anzeigeprobleme beheben, verursachen die Änderungen keine Probleme in früheren Versionen von Windows Internet Explorer, da die vorherigen Versionen einfach den überflüssigen HTML-Code ignorieren. Weitere Informationen zu diesen Änderungen finden Sie im Blogbeitrag [Site Compatibility and IE8 (Websitekompatibilität und IE8).](/archive/blogs/ie/site-compatibility-and-ie8)
