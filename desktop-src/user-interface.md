# <a name="desktop-environment"></a>Desktop Umgebung

Dieser Abschnitt enthält Anleitungen zum integrieren und Erweitern der Desktop-Benutzer seitigen Features von Windows. Sie erfahren, wie Sie sicherstellen, dass Ihre apps und Dateiformate im Startmenü, auf der Taskleiste, auf dem Desktop und im Datei-Explorer ordnungsgemäß angezeigt werden. Sie können auch sicherstellen, dass die Dateiformate und Datenspeicher indizierbar und durchsuchbar sind.

| [Windows-Shell](./shell/shell-entry.md)<br/>                                      | Die Windows-Desktop Benutzeroberfläche ermöglicht Benutzern den Zugriff auf eine Vielzahl von Objekten, die für die Ausführung von Anwendungen und die Verwaltung des Betriebssystems erforderlich sind. Die häufigsten und vertrauten Objekte sind die Ordner und Dateien, die sich auf Computer Laufwerken befinden. Es gibt auch eine Reihe von virtuellen Objekten, die es dem Benutzer ermöglichen, Aufgaben auszuführen, z. b. das Senden von Dateien an Remote Drucker oder den Zugriff auf den Papierkorb. Die Shell organisiert diese Objekte in einem hierarchischen Namespace und bietet Benutzern und Anwendungen eine konsistente und effiziente Möglichkeit, auf Objekte zuzugreifen und diese zu verwalten.<br/> |
| [Windows-Eigenschaften System](./properties/windows-properties-system.md)<br/>         | Das Windows-Eigenschaften System ist ein erweiterbares Lese-/Schreibsystem mit Daten Definitionen, das eine einheitliche Methode zum Ausdrücken von Metadaten über shellelemente bietet. Mit dem Windows-Eigenschaften System können Sie Metadaten für shellelemente speichern und abrufen. Ein shellelement ist ein beliebiges einzelnes Element, z. b. eine Datei, ein Ordner, eine e-Mail oder ein Kontakt. Eine Eigenschaft ist ein einzelner Metadatenelement, das einem shellelement zugeordnet ist.<br/>                                                                                                                                                                             |
| [Windows-Suche](./search/windows-search.md)<br/>                                 | Windows Search ist eine Plattform für die Desktop Suche, die über sofortige Suchfunktionen für die meisten gängigen Datei-und Datentypen verfügt, z. b. e-Mail, Kontakte, Kalendertermine, Dokumente, Fotos, Multimedia und andere Formate, die von Entwicklern von Drittanbietern erweitert werden können. Diese Funktionen ermöglichen es Benutzern, die zunehmende Menge von Daten zu finden, zu verwalten und zu organisieren, die in Heim-und Unternehmensumgebungen üblich sind.<br/>                                                                                                                                                                                    |

Kernel Objekte sind Sicherungs fähig, Benutzer Objekte und GDI-Objekte hingegen nicht. Um zusätzliche Sicherheit zu gewährleisten, werden Benutzeroberflächen Objekte daher mithilfe von Fenster Stationen und Desktops verwaltet, die selbst Sicherungs fähige Objekte sind.<br/>                                                                                                                                                                                                                                              |
