# <a name="desktop-environment"></a>Desktopumgebung

Dieser Abschnitt enthält Anleitungen zum Integrieren und Erweitern der desktopbenutzerorientierten Features Windows. Sie erfahren, wie Sie sicherstellen, dass Ihre Apps und Dateiformate in Start, in der Taskleiste, auf dem Desktop und im Datei-Explorer ordnungsgemäß angezeigt werden und sich verhalten. Sie können auch sicherstellen, dass Ihre Dateiformate und Datenspeicher indizierbar und durchsuchbar sind.
