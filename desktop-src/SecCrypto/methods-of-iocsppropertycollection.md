# <a name="methods-of-iocsppropertycollection"></a>Methoden von iocsppropertycollection

Die folgenden Methoden werden von der [**iocsppropertycollection**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) -Schnittstelle definiert. Die Eigenschaften Zugriffsmethoden werden hier nicht angezeigt. Informationen zu den Eigenschaften für **iocsppropertycollection** finden Sie unter [Eigenschaften von iocsppropertycollection](properties-of-iocsppropertycollection.md) .



| Methode                                                                                      | BESCHREIBUNG                                                                               |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**CreateProperty**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-createproperty)                     | Erstellt eine neue Eigenschaft und fügt Sie einem Eigenschaften Satz hinzu.                                    |
| [**DeleteProperty**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-deleteproperty)                     | Entfernt eine benannte Eigenschaft aus einem Eigenschaften Satz.                                             |
| [**GetAllProperties**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-getallproperties)                 | Ruft alle Eigenschaften in einem Eigenschaften Satz ab.                                                    |
| [**Initializefromproperties**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-initializefromproperties) | Erstellt einen Eigenschaften Satz aus den Eigenschaften, die in einer vorhandenen Serverkonfiguration enthalten sind. |
