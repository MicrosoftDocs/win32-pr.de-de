# <a name="session-keys"></a>Sitzungsschlüssel

[*Sitzungsschlüssel*](../secgloss/s-gly.md) sind Schlüssel, die generiert werden, um in einer einzelnen Kommunikationssitzung verwendet zu werden. Sitzungsschlüssel werden häufig geändert und verworfen, wenn sie nicht mehr benötigt werden. In der Regel wird ein [*symmetrischer Schlüssel*](../secgloss/s-gly.md) als Sitzungsschlüssel verwendet.

Sitzungsschlüssel sind flüchtig. Anwendungen können diese Schlüssel zur späteren Verwendung oder zur Übertragung an andere Benutzer speichern. Weitere Informationen finden Sie unter [Kryptografieschlüssel Storage und Exchange](cryptographic-key-storage-and-exchange.md).
