# <a name="signed-data-additions"></a>Ergänzungen signierter Daten

Die folgenden Änderungen wurden an den Funktionen für signierte Daten vorgenommen:

-   [*Innere Inhalte*](../secgloss/i-gly.md) nicht Daten werden in einer Oktett-Zeichenfolge gekapselt.
-   Die Bereitstellung erfolgt für verschiedene Nachrichten Versionen.
-   Signatur Geber können entweder durch Aussteller und Seriennummer, wie in PKCS \# 7 1,5 oder durch den Antragsteller Schlüssel Bezeichner, identifiziert werden.
-   Mehrere Signatur Geber sind zulässig.
-   Sowohl RSA-, DSA-als auch ECDSA-Signaturen sind zulässig.
-   Auf der Grundlage von ECC-Algorithmen (Elliptic Curve Cryptography) und AES ist Windows Vista oder höher erforderlich.
