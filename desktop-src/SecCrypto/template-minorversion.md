# <a name="templateminorversion-property"></a>Template. MinorVersion (Eigenschaft)

\[Die **MinorVersion** -Eigenschaft ist für die Verwendung in den Betriebssystemen verfügbar, die im Abschnitt "Anforderungen" angegeben sind. Verwenden Sie stattdessen die [**X509Extension-Klasse**](/dotnet/api/system.security.cryptography.x509certificates.x509extension?view=netcore-3.1) im [**System. Security. Cryptography. X509Certificates**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1) -Namespace, indem Sie den Konstruktor aufrufen, der eine OID als Parameter annimmt, und verwenden Sie dann die OID für die Zertifikat Vorlage, um die Zertifikat Erweiterungs Vorlage abzurufen.\]

Die **MinorVersion** -Eigenschaft ruft die neben Versionsnummer der Vorlage ab.

## <a name="syntax"></a>Syntax


```VB
Template.MinorVersion As Long
```



## <a name="property-value"></a>Eigenschaftswert

Die neben Versionsnummer der Vorlage.
