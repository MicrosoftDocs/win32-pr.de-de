# <a name="capicom-versions"></a>CAPICOM-Versionen

\[CAPICOM 2.1.0.2 ist für die Verwendung in den folgenden Betriebssystemen verfügbar: Windows Server 2008, Windows Vista und Windows XP. Verwenden Sie stattdessen die .NET Framework, um Sicherheitsfunktionen zu implementieren. Weitere Informationen finden Sie unter [Alternativen zur Verwendung von CAPICOM](alternatives-to-using-capicom.md).\]

CAPICOM 1,0 bietet die folgenden Funktionen:

-   Generierung und Überprüfung von PKCS \# 7-basierten digitalen Signaturen
-   Verschlüsselung und Entschlüsselung von Daten mithilfe von Zertifikaten
-   Verschlüsselung und Entschlüsselung von Daten mithilfe von Kenn Wörtern

CAPICOM 2,0 umfasst die Funktionalität von CAPICOM 1,0 und bietet die folgenden Funktionen:

-   Generierung und Überprüfung von digitalen Authenticode-Signaturen
-   Unterstützung für dateibasierte Anmelde Informationen
-   Stabile Suche nach dem Zertifikat Speicher
-   Generierung von Hashes beliebiger Daten
-   Unterstützung für den Advanced Encryption Standard (AES)-Algorithmus
-   Unterstützung für erweiterte Zertifikat Eigenschaften, z. b. Richtlinien und Vorlagen

Informationen zu den Methoden und Eigenschaften, die auf den verschiedenen Plattformen unterstützt werden, finden Sie in den Referenz Themen für die CAPICOM-Objekte. Die weitervertreibbare CAPICOM 2,0 ist über [Platform SDK Redistributable: CAPICOM](https://www.microsoft.com/download/details.aspx?id=25281)verfügbar.
