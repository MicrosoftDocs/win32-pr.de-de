# <a name="capicom-versions"></a>CAPICOM-Versionen

\[CAPICOM 2.1.0.2 ist für die Verwendung in den folgenden Betriebssystemen verfügbar: Windows Server 2008, Windows Vista und Windows XP. Verwenden Sie stattdessen die .NET Framework, um Sicherheitsfeatures zu implementieren. Weitere Informationen finden Sie unter [Alternativen zur Verwendung von CAPICOM](alternatives-to-using-capicom.md).\]

CAPICOM 1.0 bietet die folgenden Funktionen:

-   Generierung und Überprüfung von PKCS \# 7-basierten digitalen Signaturen
-   Ver- und Entschlüsselung von Daten mithilfe von Zertifikaten
-   Ver- und Entschlüsselung von Daten mithilfe von Kennwörtern

CAPICOM 2.0 umfasst die Funktionalität von CAPICOM 1.0 und fügt die folgenden Funktionen hinzu:

-   Generierung und Überprüfung digitaler Authenticode-Signaturen
-   Unterstützung für dateibasierte Anmeldeinformationen
-   Robuste Zertifikatspeichersuche
-   Generierung von Hashes beliebiger Daten
-   Unterstützung für den Advanced Encryption Standard-Algorithmus (AES)
-   Unterstützung für erweiterte Zertifikateigenschaften, z. B. Richtlinien und Vorlagen

Um zu bestimmen, welche Methoden und Eigenschaften auf den verschiedenen Plattformen unterstützt werden, lesen Sie die Referenzthemen für die CAPICOM-Objekte. Die verteilbare CAPICOM 2.0-Version ist über [platform SDK Redistributable verfügbar: CAPICOM](https://www.microsoft.com/download/details.aspx?id=25281).
