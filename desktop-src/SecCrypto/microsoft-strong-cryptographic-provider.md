# <a name="microsoft-strong-cryptographic-provider"></a>Microsoft starker Kryptografieanbieter

Der Microsoft starke Kryptografieanbieter wird als standardmäßiger RSA Full [*kryptografischen Service Provider*](../secgloss/c-gly.md) (CSP) verwendet. Sie unterstützt alle Algorithmen des erweiterten Kryptografieanbieters von Microsoft und die gleichen [*Schlüssellängen*](../secgloss/k-gly.md). Weitere Informationen zu Standardschlüssel Längen und-Algorithmen finden Sie unter Microsoft Base Cryptographic Provider. Weitere Informationen über längere Schlüssellängen und verfügbare Algorithmen finden Sie in der Tabelle mit [Microsoft Enhanced Cryptographic Provider](microsoft-enhanced-cryptographic-provider.md).



|                |                      |
|----------------|----------------------|
| Anbietertyp: | **Prov \_ RSA \_ Full**  |
| Anbieter Name: | **MS \_ Strong \_ Prov** |
