# <a name="microsoft-strong-cryptographic-provider"></a>Microsoft starker Kryptografieanbieter

Der starke Kryptografieanbieter von Microsoft wird als standardmäßiger RSA Full [*Cryptographic Service Provider*](../secgloss/c-gly.md) (CSP) verwendet. Sie unterstützt alle Algorithmen des Microsoft Enhanced Cryptographic Provider und alle [*schlüssellängen.*](../secgloss/k-gly.md) Informationen zu Standardschlüssellängen und -algorithmen finden Sie unter Microsoft Base Cryptographic Provider. Informationen zu längeren Schlüssellängen und verfügbaren Algorithmen finden Sie in der Tabelle mit [Microsoft Enhanced Cryptographic Provider](microsoft-enhanced-cryptographic-provider.md).



|                   | Wert             |
|-------------------|-------------------|
| **Anbietertyp** | PROV \_ RSA \_ FULL   |
| **Anbietername** | MS \_ STRONG \_ PROV  |
