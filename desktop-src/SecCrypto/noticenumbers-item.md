# <a name="noticenumbersitem-property"></a>Noticenenbers. Item (Eigenschaft)

\[Die **Item** -Eigenschaft ist für die Verwendung in den Betriebssystemen verfügbar, die im Abschnitt "Anforderungen" angegeben sind. Weitere Informationen finden Sie unter [**Qualifizierer**](qualifier.md).\]

Die **Item** -Eigenschaft ruft die **Länge** ab, die die indizierte Benachrichtigungs Nummer der Auflistung darstellt. Dies ist die Standard Eigenschaft.

## <a name="syntax"></a>Syntax


```VB
NoticeNumbers.Item( _
  ByVal Index _
) As Variant
```



## <a name="property-value"></a>Eigenschaftswert

Ein [**Erweiterungs**](extension.md) Objekt, das die indizierte Benachrichtigungs Nummer der Auflistung darstellt.
