# <a name="using-a-cert_info-data-structure"></a>Verwenden einer CERT \_ Info-Datenstruktur

[**Zertifikat \_ Kontext**](/windows/desktop/api/Wincrypt/ns-wincrypt-cert_context) -und [**Zertifikat \_ Informationen**](/windows/desktop/api/Wincrypt/ns-wincrypt-cert_info) sind die Hauptdaten Strukturen, in denen die Informationen in einem Zertifikat gehalten werden. Wenn ein Zertifikat verwendet wird, muss es möglicherweise decodiert werden. Weitere Informationen finden Sie unter [Decodieren einer Zertifikat \_ Informationsstruktur](decoding-a-cert-info-structure.md). Informationen können einer decodierten Zertifikat Informationsstruktur **hinzu \_** gefügt werden, und dann kann die Struktur erneut codiert werden. Weitere Informationen finden Sie unter [Codieren einer Zertifikat \_ Informationsstruktur](encoding-a-cert-info-structure.md).
