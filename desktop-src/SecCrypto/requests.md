# <a name="requests"></a>Requests

Zertifikatdienste unterstützen [*Zertifikatanforderungen basierend*](../secgloss/c-gly.md) auf dem PKCS \# 10-Format und dem Keygen-Format (Netscape). Zertifikatdienste unterstützen auch PKCS \# 7-Verlängerungsanforderungen und CMS-Anforderungen (Cryptographic Message Syntax).

Zertifikatdienste stellen auch eine [ICertRequest-Schnittstelle](/windows/desktop/api/Certcli/nn-certcli-icertrequest) bereit, die Methoden zum Übermitteln einer Zertifikatanforderung und (wenn die Anforderung genehmigt wird) zum Empfangen des resultierenden ausgestellten Zertifikats enthält.

Zusätzliche sicherheitsbezogene Schnittstellen sind in der [Zertifikatregistrierungssteuerung verfügbar.](certificate-enrollment-control.md)
