# <a name="cryptography-essentials"></a>Grundlagen der Kryptografie

Microsoft CryptoAPI, Application Programming Interface (API) und CAPICOM stellen Dienste bereit, die es Entwicklern ermöglichen, Sicherheit basierend auf der [*Kryptografie*](../secgloss/c-gly.md) für Anwendungen hinzuzufügen. CryptoAPI umfasst Funktionen für das Codieren und Decodieren von [*abstrakten Syntax Notation One*](../secgloss/a-gly.md) (ASN. 1), das Verschlüsseln und Entschlüsseln von Daten, die Authentifizierung mithilfe [*digitaler Zertifikate*](../secgloss/d-gly.md)und die digitale Zertifikat Verwaltung mithilfe von Zertifikat speichern. CryptoAPI und CAPICOM unterstützen sowohl PKI-als auch symmetrische Schlüssel Kryptografie.

Cryptography Essentials umfasst die folgenden Themen:

-   [Kryptografie, CryptoAPI und CAPICOM](cryptography--cryptoapi--and-capicom.md)
-   [Kryptografische Schlüssel](cryptographic-keys.md)
-   [Codierte und decodierte Daten](encoded-and-decoded-data.md)
-   [Verschlüsselung und Entschlüsselung von Daten](data-encryption-and-decryption.md)
-   [Hashes und digitale Signaturen](hashes-and-digital-signatures.md)
-   [Zeitstempel für Authenticode-Signaturen](time-stamping-authenticode-signatures.md)
-   [Digitale Zertifikate](digital-certificates.md)
-   [Digitale Nachrichten](digital-messages.md)
