# <a name="cryptography-essentials"></a>Kryptografie-Essentials

Microsoft CryptoAPI, Die Anwendungsprogrammierschnittstelle (APPLICATION Programming Interface, API) und CAPICOM bieten Dienste, mit denen Entwickler Anwendungen Sicherheit basierend auf [*Kryptografie*](../secgloss/c-gly.md) hinzufügen können. CryptoAPI umfasst Funktionen zum Codieren in und Decodieren von ASN.1 [*(Abstract Syntax Notation One),*](../secgloss/a-gly.md) Verschlüsseln und Entschlüsseln von Daten, Authentifizierung mit digitalen Zertifikaten und digitale [*Zertifikatverwaltung*](../secgloss/d-gly.md)mithilfe von Zertifikatspeichern. CryptoAPI und CAPICOM unterstützen sowohl PKI- als auch Kryptografie mit symmetrischem Schlüssel.

Cryptography Essentials enthält die folgenden Themen:

-   [Kryptografie, CryptoAPI und CAPICOM](cryptography--cryptoapi--and-capicom.md)
-   [Kryptografische Schlüssel](cryptographic-keys.md)
-   [Codierte und decodierte Daten](encoded-and-decoded-data.md)
-   [Datenverschlüsselung und -entschlüsselung](data-encryption-and-decryption.md)
-   [Hashes und digitale Signaturen](hashes-and-digital-signatures.md)
-   [Zeitstempeln von Authenticode-Signaturen](time-stamping-authenticode-signatures.md)
-   [Digitale Zertifikate](digital-certificates.md)
-   [Digitale Nachrichten](digital-messages.md)
