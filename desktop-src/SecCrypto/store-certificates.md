# <a name="storecertificates-property"></a>Store. Certificates-Eigenschaft

\[Die **Certificates-Eigenschaft** ist für die Verwendung in den Betriebssystemen verfügbar, die im Abschnitt Anforderungen angegeben sind. Verwenden Sie stattdessen die [**X509Store-Klasse**](/previous-versions/windows/embedded/hh424027(v=msdn.10)) im [**Namespace System.Security.Cryptography.X509Certificates.**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1)\]

Die **Certificates-Eigenschaft** ruft die Auflistung ab, die alle Zertifikate im Speicher enthält. Dies ist die Standardeigenschaft.

## <a name="syntax"></a>Syntax


```VB
Store.Certificates As Certificates
```



## <a name="property-value"></a>Eigenschaftswert

Die Auflistung von Zertifikaten im Speicher.
