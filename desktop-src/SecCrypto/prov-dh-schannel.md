# <a name="prov_dh_schannel"></a>Prov \_ dh \_ SChannel

Der Prov \_ dh \_ SChannel-Anbietertyp unterstützt sowohl Diffie-Hellman-als auch [*SChannel*](../secgloss/s-gly.md) -Protokolle.

## <a name="algorithms-supported"></a>Unterstützte Algorithmen

Beschreibungen der einzelnen Algorithmen finden Sie im Glossar.



| Zweck      | Unterstützte Algorithmen                                                                                                                                   |
|--------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| Schlüsselaustausch | DH (kurzlebig)                                                                                                                                         |
| Signatur    | [*DSS*](../secgloss/d-gly.md)                                                                                            |
| Verschlüsselung   | [*Des*](../secgloss/d-gly.md)[*Triple des*](../secgloss/t-gly.md)<br/> |
| Hashing      | [*MD5*](../secgloss/m-gly.md)-[*SHA*](../secgloss/s-gly.md)<br/>                      |
