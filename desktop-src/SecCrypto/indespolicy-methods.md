# <a name="indespolicy-methods"></a>Indespolicy-Methoden

Die [**indespolicy**](/windows/desktop/api/certpol/nn-certpol-indespolicy) -Schnittstelle unterstützt die folgenden Methoden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                        | BESCHREIBUNG                                                                                       |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**Generatechallenge-Methode**](/windows/desktop/api/certpol/nf-certpol-indespolicy-generatechallenge)<br/> | Führt die Richtlinien Entscheidung aus, ob ein Abfrage Kennwort an den SCEP-Client ausgegeben werden soll.<br/> |
| [**Initialize-Methode**](/windows/desktop/api/certpol/nf-certpol-indespolicy-initialize)<br/>               | Initialisiert das NDES-Richtlinien Modul.<br/>                                                    |
| [**Notify-Methode**](/windows/desktop/api/certpol/nf-certpol-indespolicy-notify)<br/>                       | Benachrichtigt das Plug-in über den Transaktionsstatus der SCEP-Zertifikat Anforderung.<br/>        |
| [**Uninitialize-Methode**](/windows/desktop/api/certpol/nf-certpol-indespolicy-uninitialize)<br/>           | Hebt die Initialisierung des NDES-Richtlinien Moduls auf.<br/>                                                  |
| [**Verifyrequest-Methode**](/windows/desktop/api/certpol/nf-certpol-indespolicy-verifyrequest)<br/>         | Überprüft die NDES-Zertifikat Anforderung für die Übermittlung an die Zertifizierungsstelle.<br/>                        |
