# <a name="indespolicy-methods"></a>INDESPolicy-Methoden

Die [**INDESPolicy-Schnittstelle**](/windows/desktop/api/certpol/nn-certpol-indespolicy) unterstützt die folgenden Methoden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                        | BESCHREIBUNG                                                                                       |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**GenerateChallenge-Methode**](/windows/desktop/api/certpol/nf-certpol-indespolicy-generatechallenge)<br/> | Führt die Richtlinienentscheidung aus, ob ein Abfragekennwort für den SCEP-Client ausgegeben werden soll.<br/> |
| [**Initialize-Methode**](/windows/desktop/api/certpol/nf-certpol-indespolicy-initialize)<br/>               | Initialisiert das NDES-Richtlinienmodul.<br/>                                                    |
| [**Notify-Methode**](/windows/desktop/api/certpol/nf-certpol-indespolicy-notify)<br/>                       | Benachrichtigt das Plug-In über den Transaktionsstatus der SCEP-Zertifikatanforderung.<br/>        |
| [**Uninitialize-Methode**](/windows/desktop/api/certpol/nf-certpol-indespolicy-uninitialize)<br/>           | Initialisiert das NDES-Richtlinienmodul.<br/>                                                  |
| [**VerifyRequest-Methode**](/windows/desktop/api/certpol/nf-certpol-indespolicy-verifyrequest)<br/>         | Überprüft die NDES-Zertifikatanforderung für die Übermittlung an die Zertifizierungsstelle.<br/>                        |
