# <a name="properties-of-imscepsetup"></a>Eigenschaften von imscepsetup

Die folgenden Eigenschaften werden von der [**imscepsetup**](/windows/desktop/api/Casetup/nn-casetup-imscepsetup) -Schnittstelle definiert.



| Eigenschaft                                                 | BESCHREIBUNG                                                                                                                                              |
|----------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msceperrorid**](/windows/desktop/api/Casetup/nf-casetup-imscepsetup-get_msceperrorid)         | Ruft die ID für zusätzliche Fehlerinformationen im Zusammenhang mit einer fehlgeschlagenen NDES-Spezifikation ab. Alle Methodenaufrufe für das übergeordnete Objekt setzen diese Eigenschaft zurück.          |
| [**Msceperrorstring**](/windows/desktop/api/Casetup/nf-casetup-imscepsetup-get_msceperrorstring) | Ruft die Zeichen folgen Daten für zusätzliche Fehlerinformationen im Zusammenhang mit einer fehlgeschlagenen NDES-Spezifikation ab. Alle Methodenaufrufe für das übergeordnete Objekt setzen diese Eigenschaft zurück. |
