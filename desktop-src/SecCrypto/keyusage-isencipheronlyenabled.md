# <a name="keyusageisencipheronlyenabled-property"></a>KeyUsage. isencipheronlyaktivierte Eigenschaft

\[Die Eigenschaft " **isencipheronlyaktivierte** " ist für die Verwendung in den Betriebssystemen verfügbar, die im Abschnitt "Anforderungen" angegeben sind. Verwenden Sie stattdessen die [**X509EnhancedKeyUsageExtension-Klasse**](/dotnet/api/system.security.cryptography.x509certificates.x509enhancedkeyusageextension?view=netcore-3.1) im [**System. Security. Cryptography. X509Certificates**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1) -Namespace.\]

Die Eigenschaft " **isencipheronlyaktivierte** " Ruft einen booleschen Wert ab, der angibt, ob das encipheronly-Bit festgelegt ist.

## <a name="syntax"></a>Syntax


```VB
KeyUsage.IsEncipherOnlyEnabled As Boolean
```



## <a name="property-value"></a>Eigenschaftswert

**True** gibt an, dass das encipheronly-Bit festgelegt wird.
