# <a name="methods-of-icenroll4"></a>Methoden von ICEnroll4

Die folgenden Methoden werden von der [**ICEnroll4-Schnittstelle**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll4) definiert oder sind Methoden, die die **ICEnroll4-Schnittstelle** von [**ICEnroll,**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll) [**ICEnroll2**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll2)und [**ICEnroll3**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll3)erbt. Die Methoden für den Eigenschaftenzugriff werden hier nicht angezeigt. Die Eigenschaften für **ICEnroll4** finden Sie unter [Eigenschaften von ICEnroll4.](properties-of-icenroll4.md)



| Methode                                                                         | Beschreibung                                                                                                                                                                                                                                                                                                                                                                            |
|--------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**acceptFilePKCS7**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-acceptfilepkcs7)                           | Akzeptiert und verarbeitet eine Datei, die eine PKCS \# 7-Nachricht mit einem Zertifikat enthält.<br/>                                                                                                                                                                                                                                                                                     |
| [**acceptFileResponse**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-acceptfileresponse)                     | Akzeptiert die Übermittlung der Anmeldeinformationen, die als Reaktion auf einen früheren Aufruf von [**createFileRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-createfilerequest)ausgegeben wurden, und platziert die Anmeldeinformationen im entsprechenden Speicher.<br/>                                                                                                                                                                              |
| [**acceptPKCS7**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-acceptpkcs7)                                   | Akzeptiert und verarbeitet eine PKCS \# 7-Nachricht, die ein Zertifikat enthält. PKCS \# 7 ist eine Eingabe als Parameter.<br/>                                                                                                                                                                                                                                                                    |
| [**acceptResponse**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-acceptresponse)                             | Akzeptiert die Übermittlung der Anmeldeinformationen, die als Reaktion auf einen früheren Aufruf von [**createRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-createrequest) ausgegeben wurden, und platziert die Anmeldeinformationen im entsprechenden Speicher.<br/>                                                                                                                                                                                          |
| [**addAttributeToRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-addattributetorequest)               | Fügt der [*Zertifikatanforderung*](../secgloss/c-gly.md)ein [*Attribut*](../secgloss/a-gly.md) hinzu.<br/>                                                                                                                                                                   |
| [**addBlobPropertyToCertificate**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-addblobpropertytocertificate) | Fügt einem Zertifikat eine [*BLOB-Eigenschaft*](../secgloss/b-gly.md) hinzu.<br/>                                                                                                                                                                                                                                                                            |
| [**addCertTypeToRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll2-addcerttypetorequest)                 | Fügt einer Anforderung (die zur Unterstützung der [*Unternehmenszertifizierungsstelle*](../secgloss/c-gly.md) verwendet wird) eine Zertifikatvorlage hinzu.<br/>                                                                                                                                                                     |
| [**addCertTypeToRequestEx**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-addcerttypetorequestex)             | Fügt einer Anforderung eine Zertifikatvorlage (oder "Zertifikattyp") hinzu.<br/>                                                                                                                                                                                                                                                                                                           |
| [**addExtensionToRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-addextensiontorequest)               | Fügt der Anforderung eine Erweiterung hinzu.<br/>                                                                                                                                                                                                                                                                                                                                           |
| [**addNameValuePairToRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-addnamevaluepairtorequest)       | Fügt der Anforderung ein Name-Wert-Zeichenfolgenpaar hinzu.<br/>                                                                                                                                                                                                                                                                                                                               |
| [**addNameValuePairToSignature**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll2-addnamevaluepairtosignature)   | Fügt der Anforderung das Name-Wert-Paar eines [*Attributs*](../secgloss/a-gly.md) hinzu. Es liegt an der Zertifizierungsstelle, die Bedeutung des Name-Wert-Paars zu interpretieren.<br/>                                                                                                                                                                        |
| [**binaryToString**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-binarytostring)                             | Konvertiert ein Blob für binäre [*Daten*](../secgloss/b-gly.md) in eine Zeichenfolge.<br/>                                                                                                                                                                                                                                                                          |
| [**createFilePFX**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-createfilepfx)                               | Speichert die akzeptierte Zertifikatkette und den [*privaten Schlüssel*](../secgloss/p-gly.md) in einer Datei im PFX-Format (Personal Information Exchange).<br/>                                                                                                                                                                                       |
| [**createFilePKCS10**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-createfilepkcs10)                         | Erstellt eine Base64-codierte PKCS \# 10-Zertifikatanforderung und speichert sie in einer Datei. [](../secgloss/c-gly.md)<br/>                                                                                                                                                                                                      |
| [**createFileRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-createfilerequest)                       | Erstellt eine PKCS \# 10-Zertifikatanforderung, eine \) PKCS-Anforderung (CMC) und speichert sie in einer Datei.<br/>                                                                                                               |
| [**createPFX**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-createpfx)                                       | Speichert die akzeptierte Zertifikatkette und den privaten Schlüssel in einer PFX-Formatzeichenfolge (Personal Information Exchange). Das PFX-Format wird auch als PKCS \# 12 bezeichnet.<br/>                                                                                                                                                                                                                     |
| [**createPKCS10**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-createpkcs10)                                 | Erstellt eine Base64-codierte PKCS \# 10-Zertifikatanforderung.<br/>                                                                                                                                                                                                                                                                                                                     |
| [**createRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-createrequest)                               | Erstellt eine \# ZERTIFIKATanforderung im PKCS 10-, \) PKCS-Format (CMC) und speichert sie in einer Zeichenfolge. [](../secgloss/c-gly.md)<br/>                                  |
| [**EnumAlgs**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-enumalgs)                                         | Ruft die IDs kryptografischer Algorithmen in einer angegebenen Algorithmusklasse ab, die vom aktuellen CSP unterstützt werden.<br/>                                                                                                                                                                                                                                                             |
| [**enumContainers**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-enumcontainers)                             | Ruft die Namen der Container für den [*kryptografischen Dienstanbieter (Cryptographic Service Provider,*](../secgloss/c-gly.md) CSP) ab, der durch die [**ProviderName-Eigenschaft**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_providername) angegeben wird.<br/>                                                                                                  |
| [**enumPendingRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-enumpendingrequest)                     | Listet ausstehende [*Zertifikatanforderungen*](../secgloss/c-gly.md) auf und ruft jeweils eine angegebene Eigenschaft ab.<br/>                                                                                                                                                                                               |
| [**enumProviders**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-enumproviders)                               | Ruft die Namen der verfügbaren CSPs ab, die von der [**ProviderType-Eigenschaft**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_providertype) angegeben werden.<br/>                                                                                                                                                                                                                                                         |
| [**freeRequestInfo**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-freerequestinfo)                           | Bereinigt die Speicher, wenn ein Fehler auftritt. Derzeit nicht implementiert.<br/>                                                                                                                                                                                                                                                                                                         |
| [**GetAlgName**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-getalgname)                                     | Ruft den Namen eines kryptografischen Algorithmus unter Angabe seiner ID ab. Die von dieser Methode abgerufenen Werte hängen vom aktuellen CSP ab.<br/>                                                                                                                                                                                                                                                |
| [**getCertFromFileResponse**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-getcertfromfileresponse)           | Ruft das Zertifikat aus einer Datei ab, die eine Antwort von einer [*Zertifizierungsstelle*](../secgloss/c-gly.md)enthält.<br/>                                                                                                                                                                                 |
| [**getCertFromPKCS7**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-getcertfrompkcs7)                         | Ruft das In einer PKCS 7-Nachricht enthaltene Zertifikat \# ab, das als Antwort auf eine PKCS \# 10-Zertifikatanforderung ausgestellt wurde.<br/>                                                                                                                                                                                                                                                 |
| [**getCertFromResponse**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-getcertfromresponse)                   | Ruft das Zertifikat aus der Antwort einer [*Zertifizierungsstelle*](../secgloss/c-gly.md) ab.<br/>                                                                                                                                                                                                        |
| [**GetKeyLen**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-getkeylen)                                       | Ruft die minimale und maximale Schlüssellänge für die [*Signatur*](../secgloss/s-gly.md) und [*den Austauschschlüssel*](../secgloss/e-gly.md)ab.<br/>                                                                                                                |
| [**GetKeyLenEx**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-getkeylenex)                                   | Ruft Größeninformationen für die [*Signatur*](../secgloss/s-gly.md) ab und [*tauscht Schlüssel aus.*](../secgloss/e-gly.md)<br/>                                                                                                                                   |
| [**GetSupportedKeySpec**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-getsupportedkeyspec)                   | Ruft Informationen zur Unterstützung von Signatur- oder Austauschschlüsseln durch den CSP ab.<br/>                                                                                                                                                                                                                                                                                           |
| [**InstallPKCS7**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-installpkcs7)                                 | Verarbeitet ein Zertifikat oder eine Zertifikatkette und platziert sie in den entsprechenden [*Zertifikatspeichern.*](../secgloss/c-gly.md) Diese Methode unterscheidet sich von der [**acceptPKCS7-Methode**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-acceptpkcs7) darin, dass [**InstallPKCS7**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-installpkcs7) kein Anforderungszertifikat empfängt.<br/> |
| [**InstallPKCS7Ex**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-installpkcs7ex)                             | Identisch mit [**InstallPKCS7,**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-installpkcs7) mit dem Unterschied, dass die Anzahl der Zertifikate zurückgegeben wird, die tatsächlich in lokalen Speichern installiert sind.<br/>                                                                                                                                                                                                                            |
| [**removePendingRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-removependingrequest)                 | Entfernt eine ausstehende Anforderung aus dem Anforderungsspeicher des Clients.<br/>                                                                                                                                                                                                                                                                                                                  |
| [**Zurücksetzen**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-reset)                                               | Gibt das Zertifikatregistrierungssteuerelement-Objekt in seinen [*Anfangszustand*](../secgloss/s-gly.md)zurück.<br/>                                                                                                                                                                                                                                         |
| [**resetAttributes**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-resetattributes)                           | Entfernt alle Attribute aus der Anforderung.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**resetBlobProperties**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-resetblobproperties)                   | Setzt die Eigenschaften eines [*BLOB*](../secgloss/b-gly.md)zurück.<br/>                                                                                                                                                                                                                                                                                  |
| [**resetExtensions**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-resetextensions)                           | Entfernt alle Erweiterungen aus der Anforderung.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**setPendingRequestInfo**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-setpendingrequestinfo)               | Legt Eigenschaften für eine ausstehende Anforderung fest.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**stringToBinary**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-stringtobinary)                             | Konvertiert eine codierte Zeichenfolge in ein Blob für binäre [*Daten.*](../secgloss/b-gly.md)<br/>                                                                                                                                                                                                                                                                 |
