# <a name="hardware-cryptography"></a>Hardware Kryptografie

Zertifikat Dienste unterstützen Hardware bezogene Kryptografie, z. b. das Ausstellen von Zertifikaten, die auf [*Smartcards*](../secgloss/s-gly.md)gespeichert sind. Die Unternehmens [*Zertifizierungsstelle (Certification Authority*](../secgloss/c-gly.md) , ca) unterstützt die Ausstellung von Smartcardzertifikaten für die Domänen Anmeldung beim Betriebssystem. Außerdem unterstützen sowohl die Unternehmens Zertifizierungsstelle als auch die eigenständige Zertifizierungsstelle die Ausstellung anderer Arten von Smartcardzertifikaten.

Weitere Informationen finden Sie in der Produktdokumentation. Entwicklerinformationen zu Smartcards finden Sie unter [Smartcard-Authentifizierung](../secauthn/smart-card-authentication.md). Informationen über das Steuerelement, mit dem ein Administrator Smartcardzertifikate im Namen von Benutzern anfordern kann, finden Sie unter [Smartcard](/previous-versions/windows/desktop/secsmart/smart-card-enrollment-control-usage-scenario)-Registrierungs Steuerung.
