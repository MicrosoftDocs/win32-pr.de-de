# <a name="hardware-cryptography"></a>Hardwarekryptografie

Zertifikatdienste unterstützen hardwarebezogene Kryptografie, z. B. das Ausstellen von Zertifikaten, die auf [*Smartcards gespeichert sind.*](../secgloss/s-gly.md) Die [*Unternehmenszertifizierungsstelle*](../secgloss/c-gly.md) unterstützt die Ausstellung von Smartcardzertifikaten für die Domänenanmeldung auf dem Betriebssystem. Darüber hinaus unterstützen sowohl die Unternehmens-ZS als auch die eigenständige Zertifizierungsstelle die Ausstellung anderer Arten von Smartcardzertifikaten.

Weitere Informationen finden Sie in der Produktdokumentation. Entwicklerinformationen zu Smartcards finden Sie unter [Smartcard-Authentifizierung.](../secauthn/smart-card-authentication.md) Informationen zu dem Steuerelement, mit dem ein Administrator Smartcardzertifikate im Namen von Benutzern anfordern kann, finden Sie unter [Smartcard-Registrierungssteuerung.](/previous-versions/windows/desktop/secsmart/smart-card-enrollment-control-usage-scenario)
