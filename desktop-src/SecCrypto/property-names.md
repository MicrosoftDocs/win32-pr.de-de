# <a name="property-names"></a>Eigenschaftsnamen

Ein Eigenschaftsname für [*Zertifikat Anforderungen*](../secgloss/c-gly.md) kann entweder der "Anzeige Name" einer der allgemeinen Eigenschaften sein, die von den Zertifikat Diensten verstanden werden, oder ein [*Objekt Bezeichner*](../secgloss/o-gly.md) (OID). Weitere Informationen zu Anforderungs Eigenschaftsnamen finden Sie unter [Name Properties](name-properties.md).
