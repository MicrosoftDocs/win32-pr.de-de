# <a name="signeddatasigners-property"></a>SignedData. Signers (Eigenschaft)

\[Die **Signatur ners** -Eigenschaft ist für die Verwendung in den Betriebssystemen verfügbar, die im Abschnitt "Anforderungen" angegeben sind. Verwenden Sie stattdessen die [**SignedCms-Klasse**](/dotnet/api/system.security.cryptography.pkcs.signedcms?view=dotnet-plat-ext-3.1&preserve-view=true) im [**System. Security. Cryptography. Pkcs**](/dotnet/api/system.security.cryptography.pkcs?view=dotnet-plat-ext-3.1&preserve-view=true) -Namespace.\]

Die Signatur **ners** -Eigenschaft ruft die Signatur Ersteller der Daten ab.

## <a name="syntax"></a>Syntax


```VB
SignedData.Signers As Signers
```



## <a name="property-value"></a>Eigenschaftswert

Die Signatur Geber Auflistung [**, die die**](signers.md) Signatur Ersteller der Daten darstellt.
