# <a name="prov_ms_exchange"></a>Prov \_ MS \_ Exchange

Der Prov \_ MS \_ Exchange-Anbietertyp ist für die kryptografischen Anforderungen der Microsoft Exchange-e-Mail-Anwendung und anderer mit Microsoft Mail kompatibler Anwendungen konzipiert.

## <a name="algorithms-supported"></a>Unterstützte Algorithmen

Beschreibungen der einzelnen Algorithmen finden Sie im Glossar.



| Zweck      | Unterstützte Algorithmen                                           |
|--------------|----------------------------------------------------------------|
| Schlüsselaustausch | [*RSA*](../secgloss/r-gly.md)    |
| Signatur    | RSA                                                            |
| Verschlüsselung   | [*CAST*](../secgloss/c-gly.md) |
| Hashing      | [*MD5*](../secgloss/m-gly.md)    |
