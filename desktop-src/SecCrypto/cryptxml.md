# <a name="cryptxml"></a>Cryptxml

Cryptxml bietet einen Low-Level-Satz von systemeigenen APIs, die die Erstellung und Überprüfung von XML Digital Signature basierend auf der W3C-Empfehlung XML-Signatur Syntax und-Verarbeitung (Second Edition) implementieren. Entwickler müssen mit diesem Standard vertraut sein, um die digitalen Signatur-APIs von cryptxml effektiv verwenden zu können.

Cryptxml enthält die folgenden Themen:

-   [Übersicht über die digitale XML-Signatur](xml-digital-signature-overview.md)
-   [XML-Funktionen der digitalen Signatur-API](xml-digital-signature-api-functionality.md)
-   [Kryptografiealgorithmen für die digitale XML-Signatur](xml-digital-signature-cryptographic-algorithms.md)
-   [Kryptografieerweiterungen für die digitale XML-Signatur](xml-digital-signature-cryptographic-extensions.md)
-   [Sicherheitsüberlegungen zur XML-digitalen Signatur](xml-digital-signature-security-considerations.md)
