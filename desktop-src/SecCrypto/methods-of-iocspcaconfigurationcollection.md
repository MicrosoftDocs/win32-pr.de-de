# <a name="methods-of-iocspcaconfigurationcollection"></a>Methoden von IOCSPCAConfigurationCollection

Die folgenden Methoden werden von der [**IOCSPCAConfigurationCollection-Schnittstelle**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) definiert. Die Eigenschaftenzugriffsmethoden werden hier nicht angezeigt. Informationen zu den Eigenschaften für **IOCSPCAConfigurationCollection finden** Sie unter [Eigenschaften von IOCSPCAConfigurationCollection.](properties-of-iocspcaconfigurationcollection.md)



| Methode                                                                                       | Beschreibung                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-createcaconfiguration) | Erstellt eine neue [*Zertifizierungsstellenkonfiguration*](../secgloss/c-gly.md) und fügt sie dem Konfigurationssatz hinzu. |
| [**DeleteCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-deletecaconfiguration) | Entfernt eine benannte Zertifizierungsstellenkonfiguration aus dem Konfigurationssatz.                                                                                                                                   |
