# <a name="capicom_certificates_save_as_type-enumeration"></a>CAPICOM- \_ Zertifikate \_ Save-As-Type- \_ \_ Enumeration

Der-Enumerationstyp " **CAPICOM- \_ Zertifikate \_ Save \_ As \_** " definiert das Format einer Datei mit Zertifikat Informationen. Dieser Enumerationstyp wurde von CAPICOM 2,0 eingeführt.

## <a name="members"></a>Member



| Member                                          | BESCHREIBUNG                                          | Wert |
|-------------------------------------------------|------------------------------------------------------|-------|
| **CAPICOM- \_ Zertifikate werden \_ \_ als \_ serialisiert gespeichert.** | Die gespeicherten Zertifikate werden serialisiert.<br/>    | 0     |
| **CAPICOM- \_ Zertifikate \_ speichern \_ als \_ PKCS7**      | Die Zertifikate werden als PKCS 7 gespeichert \# .<br/> | 1     |
| **CAPICOM \_ - \_ Zertifikate \_ als \_ PFX speichern**        | Die Zertifikate werden als PFX gespeichert.<br/>      | 2     |



## <a name="remarks"></a>Bemerkungen

Der- \_ Enumerationstyp "CAPICOM-Zertifikate" \_ \_ \_ wird von der " [**Zertifikats. Save**](certificates-save.md) "-Methode verwendet.
