<span data-ttu-id="1421c-104">Diffie-Hellman, Version 3, werden [*öffentliche Schlüssel-BLOBs*](../secgloss/p-gly.md) (Type PublicKeyBlob) zum Exportieren und Importieren von Informationen über einen öffentlichen DH-Schlüssel verwendet.</span><span class="sxs-lookup"><span data-stu-id="1421c-104">Diffie-Hellman version 3 [*Public Key BLOBs*](../secgloss/p-gly.md) (type PUBLICKEYBLOB) are used to export and import information about a DH public key.</span></span> <span data-ttu-id="1421c-105">Sie haben das folgende Format:</span><span class="sxs-lookup"><span data-stu-id="1421c-105">They have the following format:</span></span>


```C++

                 // As explained under "Data Structures"
DHPUBKEY_VER3 dhpubkeyver3;
BYTE p[dhpubkeyver3.bitlenP/8]; 
                 // Where P is the prime modulus

                 // Where G is the generator parameter

                 // Where J is (P-1)/Q

                 // Where Y is (G^X) mod P
```
