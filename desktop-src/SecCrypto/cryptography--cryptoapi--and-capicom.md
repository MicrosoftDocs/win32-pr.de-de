# <a name="cryptography-cryptoapi-and-capicom"></a>Kryptografie, CryptoAPI und CAPICOM

CryptoAPI und CAPICOM stellen Dienste bereit, mit denen Anwendungsentwickler die Verschlüsselung/Entschlüsselung von Daten, die Authentifizierung mithilfe digitaler [*Zertifikate*](../secgloss/c-gly.md)und die Codierung von [*abstrakten Syntax Notation One*](../secgloss/a-gly.md) (ASN. 1) in Ihre Windows-basierten Anwendungen hinzufügen können. Anwendungsentwickler können die Funktionen in CryptoAPI verwenden, ohne die Details der zugrunde liegenden Implementierung zu kennen, ähnlich wie Sie eine Grafikbibliothek verwenden können, ohne dass Sie etwas über die jeweilige Grafikhardware Konfiguration wissen müssen. CryptoAPI kann mit einer Reihe von [Kryptografiedienstanbietern](microsoft-cryptographic-service-providers.md) verwendet werden, die die eigentlichen Kryptografiefunktionen ausführen.
