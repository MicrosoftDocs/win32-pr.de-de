# <a name="capicom_key_algorithm-enumeration"></a>CAPICOM \_ KEY \_ ALGORITHM-Enumeration

Die **CAPICOM \_ KEY \_ ALGORITHM-Enumeration** definiert Schlüsselalgorithmen.

## <a name="members"></a>Member



| Member                             | Beschreibung                                | Wert |
|------------------------------------|--------------------------------------------|-------|
| **CAPICOM \_ KEY \_ ALGORITHM \_ OTHER** | Verwenden Sie einen alternativen Schlüsselalgorithmus.<br/> | 0     |
| **CAPICOM \_ KEY \_ ALGORITHM \_ RSA**   | Verwenden Sie den RSA-Schlüsselalgorithmus.<br/>      | 1     |
| **CAPICOM \_ KEY \_ ALGORITHM \_ DSS**   | Verwenden Sie den DSS-Schlüsselalgorithmus.<br/>      | 2     |



## <a name="remarks"></a>Hinweise

Diese Enumeration ist für die zukünftige Verwendung reserviert.
