# <a name="capicom_key_algorithm-enumeration"></a>CAPICOM \_ - \_ schlüsselalgorithmusenumeration

Die **CAPICOM \_ - \_ schlüsselalgorithmusenumeration** definiert Schlüssel Algorithmen.

## <a name="members"></a>Member



| Member                             | BESCHREIBUNG                                | Wert |
|------------------------------------|--------------------------------------------|-------|
| **anderer CAPICOM- \_ Schlüssel \_ Algorithmus \_** | Verwenden Sie einen alternativen Schlüssel Algorithmus.<br/> | 0     |
| **CAPICOM- \_ Schlüssel \_ Algorithmus \_ RSA**   | Verwenden Sie den RSA-Schlüssel Algorithmus.<br/>      | 1     |
| **CAPICOM- \_ Schlüssel \_ Algorithmus \_ DSS**   | Verwenden Sie den DSS Key-Algorithmus.<br/>      | 2     |



## <a name="remarks"></a>Bemerkungen

Diese Enumeration ist für die zukünftige Verwendung reserviert.
