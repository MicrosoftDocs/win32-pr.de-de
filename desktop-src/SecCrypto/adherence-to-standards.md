# <a name="adherence-to-standards"></a>Einhaltung von Standards

Zertifikatdienste akzeptieren PKCS \# 10-Standardanforderungen und stellen X.509 3.0-Zertifikate aus. Zertifikatdienste können mit Nicht-Microsoft-Clients und -Browsern und Nicht-Microsoft-Webservern verwendet werden.
