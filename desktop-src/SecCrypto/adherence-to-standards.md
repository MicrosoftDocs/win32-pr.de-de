# <a name="adherence-to-standards"></a>Einhaltung von Standards

Zertifikat Dienste akzeptieren Standard-PKCS \# 10-Anforderungen und-Probleme X. 509 3,0-Zertifikate. Zertifikat Dienste können mit nicht-Microsoft-Clients und-Browsern und nicht-Microsoft-Webservern verwendet werden.
