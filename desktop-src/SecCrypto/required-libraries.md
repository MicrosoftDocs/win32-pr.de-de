# <a name="required-libraries"></a>Erforderliche Bibliotheken

Das Verknüpfen von CryptoAPI-Funktionen erfordert die Verfügbarkeit von crypt32. lib. Programme, die einige der Funktionen verwenden, benötigen möglicherweise auch advapi32. lib. Bei Verwendung des Visual C++-Compilers sollten diese Dateien unter den Objekt-/Bibliotheksmodulen aufgelistet werden.
