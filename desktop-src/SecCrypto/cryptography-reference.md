# <a name="cryptography-reference"></a>Kryptografiereferenz

Kryptografiereferenzseiten enthalten ausführliche Beschreibungen der Kryptografiefunktionen, Schnittstellen, Objekte und anderen Programmierelemente von Microsoft. Diese Seiten enthalten Referenzbeschreibungen der API für die Arbeit mit digitalen Zertifikaten, Zertifikatdiensten und der Zertifikatregistrierungssteuerung.

Verweisseiten sind in die folgenden Gruppen unterteilt. 

| `Section`                                                      | BESCHREIBUNG                                                                                        |

| [Kryptografiedatentypen](cryptography-data-types.md)       | Datentypen, die von Kryptografiefunktionen, Schnittstellen und Objekten verwendet werden.                                |
| [Kryptografieenumerationen](cryptography-enumerations.md)   | Enumerationen, die von kryptografischen Funktionen und Methoden verwendet werden.                                          |
| [Kryptografiefunktionen](cryptography-functions.md)         | Funktionen, die mit Kryptografie- und Zertifikatdiensten verwendet werden.                                         |
| [Kryptografieschnittstellen](cryptography-interfaces.md)       | Schnittstellen, die mit Zertifikatserver-Engines, Zertifikatcodierung und Zertifikatregistrierung verwendet werden. |
| [Kryptografieobjekte](cryptography-objects.md)             | Objekte, die mit CAPICOM und der Zertifikatregistrierung verwendet werden.                                              |
| [Kryptografierückgabewerte](cryptography-return-values.md) | Gibt von den Kryptografiefunktionen verwendete Werte zurück.                                                  |

| [CAPICOM-Referenz](capicom-reference.md)                   | Liste der CAPICOM-Objekte und -Enumerationstypen.                                                      |
