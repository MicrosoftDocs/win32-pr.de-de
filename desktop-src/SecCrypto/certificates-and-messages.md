# <a name="certificates-and-messages"></a>Zertifikate und Nachrichten

Die grundlegenden Kryptografiefunktionen verwenden [*Zertifikate*](../secgloss/c-gly.md)und [*Low-Level-Nachrichten Funktionen*](../secgloss/l-gly.md) sowie [*vereinfachte Nachrichten Funktionen*](../secgloss/s-gly.md) . Um Nachrichten zu senden oder zu empfangen, die Zertifikate enthalten, verwenden Sie Nachrichten Funktionen auf niedriger Ebene.

Beispiele für die Verwendung von Zertifikaten mit den Funktionen auf niedriger Ebene finden Sie unter [Beispiel c-Programm: Codieren und Decodieren einer gegen signierten Nachricht](example-c-program-encoding-and-decoding-a-countersigned-message.md), [Beispiel c-Programm: Codieren und Decodieren einer Hash Nachricht](example-c-program-encoding-and-decoding-a-hashed-message.md).

Beispiele für die Verwendung von Zertifikaten mit den vereinfachten Nachrichten Funktionen finden Sie unter [Beispiel C-Programm: Verwenden von cryptencryptmessage und cryptdecryptmessage](example-c-program-using-cryptencryptmessage-and-cryptdecryptmessage.md).

Informationen zu Funktionen auf niedriger Ebene finden Sie unter [Nachrichten auf niedriger](low-level-messages.md)Ebene.

Informationen zu vereinfachten Nachrichten Funktionen finden Sie unter [vereinfachte Nachrichten](simplified-messages.md).
