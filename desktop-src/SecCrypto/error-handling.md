# <a name="error-handling-cryptography"></a>Fehlerbehandlung (Kryptografie)

In allen C-Beispielen, die die Dokumentation der einzelnen Funktionen begleitet, wird die Beispielfunktion [**MyHandleError**](myhandleerror.md) verwendet, um eine Fehlermeldung zu drucken und eine Prozedur zu beenden. In realen Situationen sind möglicherweise umfangreichere Fehlerbehandlungsverfahren erforderlich. Weitere Informationen finden Sie unter [Beispielprogrammeinschränkungen.](example-program-limitations.md)
