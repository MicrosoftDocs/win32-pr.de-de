# <a name="error-handling-cryptography"></a>Fehlerbehandlung (Kryptografie)

In allen C-Beispielen, die die einzelnen Funktionen der-Dokumentation begleiten, wird die Beispiel Funktion [**myhanderror**](myhandleerror.md) verwendet, um eine Fehlermeldung zu drucken und eine Prozedur zu beenden. In realen Situationen sind möglicherweise ausführlichere Fehler Behandlungs Prozeduren erforderlich. Weitere Informationen finden Sie unter [Beispielprogramm Einschränkungen](example-program-limitations.md).
