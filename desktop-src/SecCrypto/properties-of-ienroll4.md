# <a name="properties-of-ienroll4"></a>Eigenschaften von IEnroll4

Die folgenden Eigenschaften werden von der [**IEnroll4**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll4) -Schnittstelle definiert, oder die **IEnroll4** -Schnittstelle erbt von [**ienroll**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll) und [**IEnroll2**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll2).



| Eigenschaft                                                                              | BESCHREIBUNG                                                                                                                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Castoreflags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castoreflags)                                         | Legt ein Flag fest, das den Zertifikat Speicher steuert, wenn es geöffnet wird, oder ruft es ab.                                                                                                                                                                                                                                         |
| [**Castorenamewstr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castorenamewstr)                                   | Legt den Namen des Stores fest oder ruft ihn ab, in dem alle nicht-"root"-und "My"-Zertifikate aufbewahrt werden.                                                                                                                                                                                                                        |
| [**Castoretypeer WSTR**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castoretypewstr)                                   | Legt den Typ des Stores fest, der für den von der [**castorenamewstr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castorenamewstr) -Eigenschaft angegebenen Speicher verwendet werden soll, oder ruft ihn ab.                                                                                                                                                                                 |

| [**Limitexchangekeyumcipherment**](/windows/win32/api/xenroll/nf-xenroll-ienroll2-get_limitexchangekeytoencipherment)     | Legt einen booleschen Wert fest, der steuert, ob eine bei \_ keyexchange-Anforderung eine [*digitale Signatur*](../secgloss/d-gly.md) und einen nicht abstreitbaren Schlüssel Verwendungen enthält, oder ruft ihn ab.                                                                                       |

| [**Mystoretypeer WSTR**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_mystoretypewstr)                                   | Legt den von der [**mystorenamewstr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_mystorenamewstr) -Eigenschaft angegebenen Speichertyp fest oder ruft ihn ab.                                                                                                                                                                                                      |
