# <a name="utilitiesbinarytohex-method"></a>Utilities. binarydehex-Methode

\[Die **binarytohex** -Methode ist für die Verwendung in den Betriebssystemen verfügbar, die im Abschnitt "Anforderungen" angegeben sind.\]

Die **binaryyhex** -Methode konvertiert eine Binär Zeichenfolge in eine hexadezimale Zeichenfolge.

## <a name="syntax"></a>Syntax


```VB
Utilities.BinaryToHex( _
  ByVal BinaryString _
)
```



## <a name="parameters"></a>Parameter

<dl> <dt>

*Binarystring* \[ in\]
</dt> <dd>

Die Binär gepackte Zeichenfolge, die in eine hexadezimale Zeichenfolge konvertiert werden soll.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Eine hexadezimale Zeichenfolge, die der angegebenen Binär geladenen Zeichenfolge entspricht.
