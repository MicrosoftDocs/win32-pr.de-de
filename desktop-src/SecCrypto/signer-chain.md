# <a name="signerchain-property"></a>Signer. Chain (Eigenschaft)

\[Die **Chain** -Eigenschaft ist für die Verwendung in den Betriebssystemen verfügbar, die im Abschnitt "Anforderungen" angegeben sind. Verwenden Sie stattdessen die [**CmsSigner-Klasse**](/dotnet/api/system.security.cryptography.pkcs.cmssigner?view=dotnet-plat-ext-3.1&preserve-view=true) im [**System. Security. Cryptography. Pkcs**](/dotnet/api/system.security.cryptography.pkcs?view=dotnet-plat-ext-3.1&preserve-view=true) -Namespace.\]

Die **Chain** -Eigenschaft ruft die Kette des Signatur Gebers ab.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
Signer.Chain As Chain
```



## <a name="property-value"></a>Eigenschaftswert

Die Kette des Signatur Gebers.
