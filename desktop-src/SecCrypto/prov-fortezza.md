# <a name="prov_fortezza"></a>Prov- \_ Fortezza

Der Prov \_ Fortezza-Anbietertyp enthält einen Satz kryptografischer Protokolle und Algorithmen im Besitz des [*National Institute of Standards and Technology*](../secgloss/n-gly.md) (NIST).

## <a name="algorithms-supported"></a>Unterstützte Algorithmen

Beschreibungen der einzelnen Algorithmen finden Sie im Glossar.



| Zweck      | Unterstützte Algorithmen                                                                                      |
|--------------|-----------------------------------------------------------------------------------------------------------|
| Schlüsselaustausch | [*KEA*](../secgloss/k-gly.md)         |
| Signatur    | [*DSS*](../secgloss/d-gly.md) |
| Verschlüsselung   | [*Skipjack*](../secgloss/s-gly.md)                                |
| Hashing      | [*SHA*](../secgloss/s-gly.md)           |



 

## <a name="related-documentation"></a>Verwandte Dokumentationen

Weitere Informationen finden Sie direkt vom [*National Institute of Standards and Technology*](../secgloss/n-gly.md) (NIST).
