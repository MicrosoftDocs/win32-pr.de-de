# <a name="digital-messages"></a>Digitale Nachrichten

Eine Nachricht wird als alle Daten definiert, die codiert wurden. Nachrichten sind Daten, die für die Übertragung an eine andere Person oder Entität codiert sind, oder codierte Daten, die von einer anderen Person oder Entität empfangen werden. In der Krypto-Umgebung werden Nachrichten in der Regel signiert, verschlüsselt oder beides. Der Begriff Meldungen enthält häufig Zertifikate.

In CryptoAPI Version 2.0 sind Nachrichtenbehandlungsfunktionen in [*Nachrichtenfunktionen*](../secgloss/l-gly.md) auf niedriger Ebene und [*vereinfachte Nachrichtenfunktionen unterteilt.*](../secgloss/s-gly.md)
