# <a name="digital-messages"></a>Digitale Nachrichten

Eine Nachricht wird als alle codierten Daten definiert. Bei Nachrichten handelt es sich um Daten, die für die Übertragung an andere Personen oder Entitäten oder codierte Daten, die von einer anderen In der kryptografieumgebung sind Nachrichten in der Regel signiert, verschlüsselt oder beides. Der Begriff Nachrichten enthält häufig Zertifikate.

In CryptoAPI, Version 2,0, sind die Funktionen zur Nachrichten Behandlung in [*Nachrichten Funktionen auf niedriger Ebene*](../secgloss/l-gly.md) und in [*vereinfachte Nachrichten Funktionen*](../secgloss/s-gly.md)unterteilt.
