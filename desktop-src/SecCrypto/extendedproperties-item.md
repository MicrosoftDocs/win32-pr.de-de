Die **Item** -Eigenschaft ruft ein [**ExtendedProperty**](extendedproperty.md) -Objekt aus der Auflistung ab. Dies ist die Standard Eigenschaft.

## <a name="syntax"></a>Syntax


```VB
ExtendedProperties.Item( _
  ByVal Index _
) As Variant
```



## <a name="property-value"></a>Eigenschaftswert

Ein [**ExtendedProperty**](extendedproperty.md) -Objekt, das die indizierte erweiterte Eigenschaft der Auflistung darstellt.
