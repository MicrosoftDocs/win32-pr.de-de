# <a name="microsoft-dss-cryptographic-provider"></a>Microsoft DSS Cryptographic Provider

Der Microsoft DSS-Kryptografieanbieter unterstützt Hashing, Datensignatur und Signaturüberprüfung mithilfe der Algorithmen [*Secure Hash Algorithm*](../secgloss/s-gly.md) (SHA) und Digital Signature [*Standard*](../secgloss/d-gly.md) (DSS). Sie kann außerhalb der USA und Kanada exportiert werden.



|                   | Wert               |
|-------------------|---------------------|
| **Anbietertyp** | PROV \_ DSS           |
| **Anbietername** | MS \_ DEF \_ DSS \_ PROV  |
