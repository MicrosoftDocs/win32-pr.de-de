# <a name="methods-of-ienroll4"></a>Methoden von IEnroll4

Die folgenden Methoden werden von der [**IEnroll4-Schnittstelle**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll4) definiert oder sind Methoden, die die **IEnroll4-Schnittstelle** von [**IEnroll**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll) und [**IEnroll2 erbt.**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll2) Die Eigenschaftenzugriffsmethoden werden hier nicht angezeigt. Die Eigenschaften für **IEnroll4 finden Sie** unter Eigenschaften von [IEnroll4](properties-of-ienroll4.md).
