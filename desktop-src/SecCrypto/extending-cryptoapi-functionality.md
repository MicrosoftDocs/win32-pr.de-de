# <a name="extending-cryptoapi-functionality"></a>Erweitern der CryptoAPI-Funktionalität

Die Zukunft der [*Kryptografie*](../secgloss/c-gly.md) und der sicheren Kommunikation kann nicht einfach vorhergesagt werden. Es können neue Zertifikattypen entstehen, verschiedene Zertifikaterweiterungen werden möglicherweise häufig verwendet, und es werden möglicherweise neue Nachrichtentypen eingeführt. Aus diesem Grund ist die Erweiterbarkeit Teil des Entwurfs wichtiger [*CryptoAPI-Funktionen.*](../secgloss/c-gly.md)

In den folgenden Abschnitten finden Sie Übersichten zur Verwendung von OIDs zum Erweitern von CryptoAPI-Funktionen.



| Thema                                                                              | Inhalte                                                                                                                            |
|------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [Übersicht über die OID](oid-overview.md)                                                   | Grundlegende OID-Konzepte.                                                                                                           |
| [Erstellen der neuen Funktionalität](creating-the-new-functionality.md)               | Erstellen von OIDs und Funktionen, um die Verwendung vorhandener APIs zu erweitern.                                                                     |
| [Registrieren der neuen Funktionalität](registering-the-new-functionality.md)         | Einrichten neuer OID-bezogener Funktionen.                                                                                               |
| [Installieren der neuen Funktionalität](installing-the-new-functionality.md)           | Installieren von OID-Funktionen im Arbeitsspeicher, um die Leistung zu verbessern.                                                                          |
| [Erweitern der CertOpenStore-Funktionalität](extending-certopenstore-functionality.md) | Erweitern der [**CertOpenStore-Funktionalität**](/windows/desktop/api/Wincrypt/nf-wincrypt-certopenstore) mithilfe der installierbaren oder registrierten Zertifikatspeicheranbieterfunktion. |
