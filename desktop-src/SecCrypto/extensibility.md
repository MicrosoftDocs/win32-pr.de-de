# <a name="extensibility-cryptography"></a>Erweiterbarkeit (Kryptografie)

Die Zertifikat Dienste unterstützen [*X. 509*](../secgloss/x-gly.md) -Zertifikat Erweiterungen. Richtlinien Module können Erweiterungen hinzufügen oder ändern, und ein Beendigungs Modul kann diese Erweiterungen abrufen.

Weitere Informationen finden Sie unter [Erweiterungs Handler](extension-handlers.md) .
