<span data-ttu-id="ef8ec-105">Ein [*symmetrischer Schlüssel*](../secgloss/s-gly.md) ist ein einzelner Schlüssel, der mit den Vorgängen eines [*symmetrischen Verschlüsselungs*](../secgloss/s-gly.md) Schemas verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="ef8ec-105">A [*Symmetric key*](../secgloss/s-gly.md) is a single key that is used with the operations of a [*symmetric encryption*](../secgloss/s-gly.md) scheme.</span></span> <span data-ttu-id="ef8ec-106">Ein solcher Schlüssel kann z. b. in einem symmetrischen Verschlüsselungsalgorithmus für die Verschlüsselung und Entschlüsselung verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="ef8ec-106">For example, such a key may be used in a symmetric encryption algorithm for both encryption and decryption.</span></span>
