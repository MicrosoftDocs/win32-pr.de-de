# <a name="using-certificate-enrollment-control"></a>Verwenden der Zertifikat Registrierungs Steuerung

Bei der Zertifikat Registrierungs Steuerung handelt es sich um ein einzelnes Objekt mit vielen Eigenschaften und mehreren Methoden, mit denen bestimmt werden kann, wie die Zertifikat Registrierungs Steuerung die Benutzerinformationen verarbeitet, welche Art von Zertifikat angefordert werden soll, wie die Schlüssel generiert werden sollen, wo das Zertifikat gespeichert werden soll und welche Prozesse verwendet werden. Es gibt zwei primäre Verwendungsmöglichkeiten der Zertifikat Registrierungs Steuerung: zum Erstellen einer [*Zertifikat Anforderung*](../secgloss/c-gly.md) (PKCS \# 10) und zum Empfangen des zurückgegebenen Zertifikats und zum Speichern dieses Zertifikats in einem [*Zertifikat Speicher*](../secgloss/c-gly.md). Ausführliche Informationen und eine Syntax zum Erstellen einer Instanz des Zertifikatregistrierungs-Steuerungs Objekts, zum Festlegen der zugehörigen Eigenschaften und zum Verwenden der zugehörigen Methoden finden Sie in den folgenden Themen.
