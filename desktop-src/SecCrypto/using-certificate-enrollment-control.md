# <a name="using-certificate-enrollment-control"></a>Verwenden der Zertifikatregistrierungssteuerung

Die Zertifikatregistrierungssteuerung ist ein einzelnes Objekt mit vielen Eigenschaften und mehreren Methoden, mit denen bestimmt werden kann, wie die Zertifikatregistrierungssteuerung die Benutzerinformationen verarbeitet, welche Art von Zertifikat angefordert werden soll, wie die Schlüssel generiert werden sollen, wo das Zertifikat gespeichert werden soll, und verwandte Prozesse. Es gibt zwei primäre Verwendungsmöglichkeiten der Zertifikatregistrierungssteuerung: zum Erstellen einer [*Zertifikatanforderung*](../secgloss/c-gly.md) (PKCS \# 10) und zum Empfangen des zurückgegebenen Zertifikats und zum Speichern in einem [*Zertifikatspeicher.*](../secgloss/c-gly.md) Ausführliche Informationen und Syntax zum Erstellen einer Instanz des Zertifikatregistrierungssteuerelement-Objekts, zum Festlegen seiner Eigenschaften und zum Verwenden seiner Methoden finden Sie in den folgenden Themen.
