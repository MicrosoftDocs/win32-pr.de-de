# <a name="tools-to-sign-files-and-check-signatures"></a>Tools zum Signieren von Dateien und Überprüfen von Signaturen

CryptoAPI-Tools sind die folgenden Tools, die Sie zum digitalen Signieren von Dateien, zum Bestätigen von Signaturen für Dateien und zum Erstellen von Katalogdateien verwenden können.



| Tool                     | Bemerkungen                                                                                                                                                                                                                                                                            |
|--------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [MakeCat](makecat.md)   | Erstellt eine nicht signierte Katalog Datei. Diese Katalog Datei wird anschließend zum Vergleichen der Hashwerte in der Katalog Datei mit den [*Hashes einzelner*](../secgloss/h-gly.md) Dateien verwendet, um zu beweisen, dass die ursprünglichen Dateien nicht manipuliert werden können. |
| [SetReg](setreg.md)     | Legt Registrierungsschlüssel fest, mit denen der Zertifikat Überprüfungsprozess gesteuert wird.                                                                                                                                                                                                              |
| [SignTool](signtool.md) | Signiert Dateien Digital und bestätigt digitale Signaturen von Dateien.                                                                                                                                                                                                                    |
