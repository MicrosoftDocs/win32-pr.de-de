# <a name="prov_rsa_sig"></a>Prov \_ RSA \_ sig

Der Prov \_ RSA \_ sig-Anbietertyp ist eine Teilmenge von Prov \_ RSA \_ Full. Es werden nur die Funktionen und Algorithmen unterstützt, die für [*Hashes*](../secgloss/h-gly.md) und [*digitale Signaturen*](../secgloss/d-gly.md)erforderlich sind.

## <a name="algorithms-supported"></a>Unterstützte Algorithmen

Beschreibungen der einzelnen Algorithmen finden Sie im Glossar.



| Zweck      | Unterstützte Algorithmen                                                                                                              |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------|
| Schlüsselaustausch | Keine                                                                                                                              |
| Signatur    | [*RSA*](../secgloss/r-gly.md)                                                                       |
| Verschlüsselung   | Keine                                                                                                                              |
| Hashing      | [*MD5*](../secgloss/m-gly.md)-[*SHA*](../secgloss/s-gly.md)<br/> |



 

## <a name="related-documentation"></a>Verwandte Dokumentationen

Die Dokumentation für den [Prov \_ RSA \_ Full](prov-rsa-full.md) -Anbietertyp gilt auch für den Prov \_ RSA \_ sig-Anbietertyp.
