# <a name="prov_rsa_full"></a>Prov \_ RSA \_ Full

Der \_ vollständige Prov RSA \_ -Anbietertyp unterstützt sowohl [*digitale Signaturen*](../secgloss/d-gly.md) als auch Datenverschlüsselung. Es wird als universell eingestuft. Der RSA-Algorithmus für öffentliche Schlüssel wird für alle Vorgänge mit öffentlichem Schlüssel verwendet.

## <a name="algorithms-supported"></a>Unterstützte Algorithmen

Beschreibungen der einzelnen Algorithmen finden Sie im Glossar.



| Zweck      | Unterstützte Algorithmen                                                                                                              |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------|
| Schlüsselaustausch | [*RSA*](../secgloss/r-gly.md)                                                                       |
| Signatur    | RSA                                                                                                                               |
| Verschlüsselung   | [*RC2*](../secgloss/r-gly.md)[*RC4*](../secgloss/r-gly.md)<br/> |
| Hashing      | [*MD5*](../secgloss/m-gly.md)-[*SHA*](../secgloss/s-gly.md)<br/> |



 

## <a name="related-documentation"></a>Verwandte Dokumentationen

Dieser Anbietertyp wird von Microsoft und RSA-Datensicherheit definiert. Dies wird in den folgenden Dokumenten beschrieben:

-   RSA-Laboratorien, Kryptografiestandards für öffentliche Schlüssel, RSA-Datensicherheit, November 1993.
