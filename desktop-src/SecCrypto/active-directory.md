# <a name="active-directory"></a>Active Directory

Zertifikatdienste unterstützen die Verwendung von Active Directory. Im Fall der [*Unternehmenszertifizierungsstelle*](../secgloss/c-gly.md) erfordert Certificate Services die Verwendung von Active Directory. Die Unternehmenszertifizierungsstelle veröffentlicht Benutzerzertifikate und [*Zertifikatsperrlisten*](../secgloss/c-gly.md) (Certificate Revocation Lists, CRL) in Active Directory. Eine eigenständige Zertifizierungsstelle kann verwaltet werden, um die gleichen Aktionen auszuführen, sofern Active Directory für die eigenständige Zertifizierungsstelle verfügbar ist und ein benutzerdefiniertes Richtlinienmodul verwendet wird.
