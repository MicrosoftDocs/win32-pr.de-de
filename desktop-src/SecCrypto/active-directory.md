# <a name="active-directory"></a>Active Directory

Die Verwendung von Active Directory wird von den Zertifikat Diensten unterstützt. Im Fall der Unternehmens Zertifizierungsstelle ( [*Certification Authority*](../secgloss/c-gly.md) , ca) erfordert die Zertifikat Dienste die Verwendung der Active Directory. Die Unternehmens Zertifizierungsstelle veröffentlicht Benutzerzertifikate und [*Zertifikat Sperr Listen*](../secgloss/c-gly.md) (CRL) für die Active Directory. Eine eigenständige Zertifizierungsstelle kann verwaltet werden, um dieselben Aktionen auszuführen, sofern die Active Directory der eigenständigen Zertifizierungsstelle zur Verfügung steht und ein benutzerdefiniertes Richtlinien Modul verwendet wird.
