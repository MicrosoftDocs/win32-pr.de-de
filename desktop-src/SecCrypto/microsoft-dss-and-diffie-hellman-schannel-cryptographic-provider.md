# <a name="microsoft-dss-and-diffie-hellmanschannel-cryptographic-provider"></a>Microsoft DSS-und Diffie-Hellman/SChannel-Kryptografieanbieter

Der Kryptografieanbieter Microsoft DSS und [*Diffie-Hellman*](../secgloss/d-gly.md) / [*SChannel*](../secgloss/s-gly.md) unterstützt das hasht, das Signieren von Daten mit DSS, das Erzeugen von Diffie-Hellman-Schlüsseln (d-h), das Austauschen von d-h-Schlüsseln und das Exportieren eines d-h-Schlüssels. Dieser CSP unterstützt die Schlüssel Ableitung für die Protokolle SSL3 und das TLS1 verwendet. Sie kann in andere Länder/Regionen exportiert werden.
