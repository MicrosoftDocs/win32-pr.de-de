# <a name="hcryptprov"></a>HCRYPTPROV

Der **HCRYPTPROV-Datentyp** wird verwendet, um Handles für CSPs darzustellen. Diese Handles werden verwendet, um anzugeben, welches CSP-Modul bestimmte Vorgänge ausführt.


```C++
typedef ULONG_PTR HCRYPTPROV;
```
