# <a name="microsoft-cryptographic-service-providers"></a>Kryptografiedienstanbieter von Microsoft

Die folgenden [*Kryptografiedienstanbieter*](../secgloss/c-gly.md) (CSP) sind derzeit bei Microsoft verfügbar.



| Anbieter                                                                                                                                 | Beschreibung                                                                                                                                                                                                                                              |
|------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Kryptografieanbieter von Microsoft](microsoft-base-cryptographic-provider.md)                                                       | Ein umfassender Satz an grundlegenden Kryptografiefunktionen, die in andere Länder oder Regionen exportiert werden können.                                                                                                                                                     |
| [Microsoft starker Kryptografieanbieter](microsoft-strong-cryptographic-provider.md)                                                   | Eine Erweiterung des Kryptografieanbieters von Microsoft, die unter Windows XP und höher verfügbar ist.                                                                                                                                                           |
| [Microsoft Enhanced Cryptographic Provider](microsoft-enhanced-cryptographic-provider.md)                                               | Der Kryptografieanbieter von Microsoft Base mit über längere Schlüssel und zusätzliche Algorithmen.                                                                                                                                                                |
| [Kryptografieanbieter von Microsoft AES](microsoft-aes-cryptographic-provider.md)                                                         | Microsoft Enhanced Cryptographic Provider mit Unterstützung für AES-Verschlüsselungsalgorithmen.                                                                                                                                                                    |
| [Microsoft DSS-Kryptografieanbieter](microsoft-dss-cryptographic-provider.md)                                                         | Bietet Hash-, Daten Signatur-und Signatur Überprüfungs Funktionen mithilfe der Algorithmen Secure Hash-Algorithmus ([*SHA*](../secgloss/s-gly.md)) und Digital Signature Standard (DSS). |
| [Microsoft Base DSS und Diffie-Hellman Kryptografieanbieter](microsoft-base-dss-and-diffie-hellman-cryptographic-provider.md)         | Eine supermenge des DSS-Kryptografieanbieters, der auch Diffie-Hellman Schlüsselaustausch, Hashing, Daten Signatur und Signatur Überprüfung unter Verwendung der Algorithmen des Secure Hash Algorithmus (SHA) und des Digital Signature Standard (DSS) unterstützt.                    |
| [Microsoft Enhanced DSS und Diffie-Hellman Kryptografieanbieter](microsoft-enhanced-dss-and-diffie-hellman-cryptographic-provider.md) | Unterstützt Diffie-Hellman Schlüsselaustausch (40-Bit-des-Ableitung), SHA-Hashwert, DSS-Daten Signierung und DSS-Signatur Überprüfung.                                                                                                                           |
| [Microsoft DSS-und Diffie-Hellman/SChannel-Kryptografieanbieter](microsoft-dss-and-diffie-hellman-schannel-cryptographic-provider.md) | Unterstützt Hashs, Daten Signierung mit DSS, das Erzeugen von Diffie-Hellman-Schlüsseln (d-h), das Austauschen von d-h-Schlüsseln und das Exportieren eines d-h-Schlüssels. Dieser CSP unterstützt die Schlüssel Ableitung für die Protokolle SSL3 und das TLS1 verwendet.                                                           |
| [Kryptografieanbieter für Microsoft RSA/SChannel](microsoft-rsa-schannel-cryptographic-provider.md)                                       | Unterstützt Hashwert, Daten Signierung und Signatur Überprüfung. Der Algorithmusbezeichner calg \_ SSL3 \_ SHAMD5 wird für SSL 3,0-und TLS 1,0-Client Authentifizierung verwendet. Dieser CSP unterstützt die Schlüssel Ableitung für die Protokolle SSL2 verwendet, das PCT1 verwendet, SSL3 und das TLS1 verwendet.             |
| [Kryptografieanbieter für Microsoft RSA Signature](microsoft-rsa-signature-cryptographic-provider.md)                                     | Stellt Daten Signierung und Signatur Überprüfung bereit.                                                                                                                                                                                                        |
