# <a name="capicom_secret_type-enumeration"></a>CAPICOM \_ Secret \_ Type-Enumeration

Die **CAPICOM \_ Secret \_ Type** -Enumeration gibt die Art des Geheimnisses an, das zum Ableiten eines Schlüssels verwendet wird, der zum Verschlüsseln/Entschlüsseln von Daten verwendet werden soll.

## <a name="members"></a>Member



| Member                        | BESCHREIBUNG                                                     | Wert |
|-------------------------------|-----------------------------------------------------------------|-------|
| **geheimer CAPICOM- \_ \_ Kennwort** | Der Verschlüsselungsschlüssel muss von einem Kennwort abgeleitet werden.<br/> | 0     |
