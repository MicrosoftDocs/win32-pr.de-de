# <a name="capicom_secret_type-enumeration"></a>CAPICOM \_ SECRET \_ TYPE-Enumeration

Die **CAPICOM \_ SECRET \_ TYPE-Enumeration** gibt die Art des Geheimnisses an, das zum Ableiten eines Schlüssels verwendet wird, der für die Verschlüsselung/Entschlüsselung von Daten verwendet werden soll.

## <a name="members"></a>Member



| Member                        | BESCHREIBUNG                                                     | Wert |
|-------------------------------|-----------------------------------------------------------------|-------|
| **CAPICOM \_ SECRET \_ PASSWORD** | Der Verschlüsselungsschlüssel muss von einem Kennwort abgeleitet werden.<br/> | 0     |
