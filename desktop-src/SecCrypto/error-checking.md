# <a name="error-checking-cryptography"></a>Fehlerüberprüfung (Kryptografie)

Für C++ ist der Rückgabewert immer ein **HRESULT** und gibt an, ob der Methodenaufrufe erfolgreich war oder fehlgeschlagen ist. Wenn der-Befehl fehlgeschlagen ist, gibt der Rückgabewert den Grund für den Fehler an.
