# <a name="error-checking-cryptography"></a>Fehlerüberprüfung (Kryptografie)

Für C++ ist der Rückgabewert immer ein **HRESULT** und gibt an, ob der Methodenaufruf erfolgreich war oder fehlgeschlagen ist. Wenn der Aufruf fehlgeschlagen ist, gibt der Rückgabewert an, warum er fehlgeschlagen ist.
