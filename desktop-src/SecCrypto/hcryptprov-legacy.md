# <a name="hcryptprov_legacy"></a>HCRYPTPROV \_ LEGACY

Der **HCRYPTPROV \_ LEGACY-Datentyp** wird verwendet, um den [**HCRYPTPROV-Datentyp**](hcryptprov.md) zu ersetzen, bei dem der **HCRYPTPROV-Datentyp** nicht mehr verwendet wird. Funktionsparameter und Datenstrukturen vom Typ **HCRYPTPROV \_ LEGACY** sollten auf **NULL** festgelegt werden.


```C++
typedef ULONG_PTR HCRYPTPROV_LEGACY;
```
