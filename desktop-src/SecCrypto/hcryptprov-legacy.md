# <a name="hcryptprov_legacy"></a>hcryptprov- \_ Legacy

Der **hcryptprov- \_ Legacy** Datentyp wird zum Ersetzen des [**hcryptprov**](hcryptprov.md) -Datentyps verwendet, bei dem der **hcryptprov** -Datentyp nicht mehr verwendet wird. Funktionsparameter und Datenstrukturen vom Typ " **hcryptprov \_ Legacy** " sollten auf **null** festgelegt werden.


```C++
typedef ULONG_PTR HCRYPTPROV_LEGACY;
```
