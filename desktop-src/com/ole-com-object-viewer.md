# <a name="ole-com-object-viewer"></a>OLE-COM-Objekt-Viewer

Der OLE-COM-Objekt-Viewer Oleview.exe ist eine Anwendung, die mit Visual C++ bereitgestellt wird, die die auf Ihrem Computer installierten COM-Objekte und die unterstützten Schnittstellen anzeigt. Sie können diese Objektanzeige verwenden, um Typbibliotheken anzuzeigen.

So zeigen Sie die Typbibliothek eines COM-Objekts an:

1.  Klicken Sie im **Menü** Datei des Objekt-Viewers auf **TypeLib anzeigen.** Ein Dialogfeld **Öffnen** wird angezeigt.
2.  Geben Sie die Typbibliotheksdatei an, die Sie öffnen möchten, und klicken Sie auf **OK.**

Der Objekt-Viewer zeigt die Schnittstellen des Objekts an.

Weitere Informationen finden Sie unter [Verwenden des OLE/COM-Objekt-Viewers](/previous-versions/d0kh9f4c(v=vs.140)) in der Visual C++-Dokumentation.
