# <a name="ole-com-object-viewer"></a>OLE-COM-Objekt-Viewer

Der OLE-COM-Objekt-Viewer (Oleview.exe) ist eine mit Visual C++ bereitgestellte Anwendung, die die auf Ihrem Computer installierten COM-Objekte und die von Ihnen unterstützten Schnittstellen anzeigt. Sie können diesen Objekt-Viewer verwenden, um Typbibliotheken anzuzeigen.

So zeigen Sie die Typbibliothek eines COM-Objekts an:

1.  Klicken Sie im Menü **Datei** der Objekt **Anzeige auf TypeLib anzeigen**. Ein Dialogfeld **Öffnen** wird angezeigt.
2.  Geben Sie die zu öffnende Typbibliotheks Datei an, und klicken Sie auf **OK**.

Der Objekt-Viewer zeigt die Schnittstellen des Objekts an.

Weitere Informationen finden Sie unter [Verwenden des OLE/COM-Objekt-Viewers](/previous-versions/d0kh9f4c(v=vs.140)) in der Visual C++-Dokumentation.
