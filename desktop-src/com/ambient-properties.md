# <a name="ambient-properties"></a>Umgebungseigenschaften

ActiveX-Steuerelement Container müssen mindestens die folgenden Ambient-Eigenschaften unterstützen, wobei die Standard-DispIds verwendet werden.



| Ambient-Eigenschaft             | DISPID          | Kommentare/Bedingungen                                                      |
|------------------------------|-----------------|--------------------------------------------------------------------------|
| LocaleID<br/>          | -705<br/> |   <br/>                                                          |
| UserMode<br/>          | -709<br/> | Für Container mit unterschiedlichen Benutzer-und Lauf Umgebungen.<br/> |
| DisplayAsDefault <br/> | -713<br/> | Für Container, in denen eine Standard Schaltfläche relevant ist.<br/>      |
