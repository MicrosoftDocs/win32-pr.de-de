# <a name="what-support-for-an-interface-means"></a>Was bedeutet Unterstützung für eine Schnittstelle

Neben der [**IUnknown**](/windows/desktop/api/Unknwn/nn-unknwn-iunknown) -Schnittstelle werden von einem ActiveX-Steuerelement oder COM-Objekt für dieses Element die unterstützten optionalen Funktionen durch zusätzliche Schnittstellen ausgedrückt. Dies heißt, dass über **IUnknown** keine anderen Schnittstellen erforderlich sind. In der folgenden Tabelle werden die Schnittstellen aufgelistet, die von einem ActiveX-Steuerelement unterstützt werden können, und es wird die Bedeutung dieser Schnittstelle aufgeführt.

Dies ist die einzige Möglichkeit, dass das Steuerelement diese Bedingung kennt.
