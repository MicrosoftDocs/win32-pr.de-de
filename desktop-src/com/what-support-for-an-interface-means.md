# <a name="what-support-for-an-interface-means"></a>Was bedeutet Die Unterstützung für eine Schnittstelle?

Neben der [**IUnknown-Schnittstelle**](/windows/desktop/api/Unknwn/nn-unknwn-iunknown) drückt ein ActiveX Control- oder COM-Objekt jede optionale Funktionalität aus, die es über zusätzliche Schnittstellen unterstützt. Das heißt, dass über **IUnknown** keine weiteren Schnittstellen erforderlich sind. Zu diesem Zweck werden in der folgenden Tabelle die Schnittstellen aufgeführt, die ein ActiveX-Steuerelement unterstützen kann, und was es bedeutet, diese Schnittstelle zu unterstützen.
