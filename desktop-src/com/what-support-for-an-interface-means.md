# <a name="what-support-for-an-interface-means"></a>Welche Unterstützung für eine Schnittstelle bedeutet

Neben der [**IUnknown-Schnittstelle**](/windows/desktop/api/Unknwn/nn-unknwn-iunknown) drückt ein ActiveX-Steuerelement oder COM-Objekt für diese Frage die optionale Funktionalität aus, die es über zusätzliche Schnittstellen unterstützt. Das heißt, dass über **IUnknown** keine weiteren Schnittstellen erforderlich sind. Zu diesem Zweck werden in der folgenden Tabelle die Schnittstellen aufgeführt, die von einem ActiveX-Steuerelement möglicherweise unterstützt werden, und was es bedeutet, diese Schnittstelle zu unterstützen.
