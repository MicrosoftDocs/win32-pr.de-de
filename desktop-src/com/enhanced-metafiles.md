# <a name="enhanced-metafiles"></a>Erweiterte Metadateien

Es ist nicht überraschend, dass erweiterte Metadateien mehr Funktionalität als Standardmetadateien bieten. Die Verwendung erweiterter Metadateien vereinfacht im Allgemeinen das Rendern von Code. Ein erweiterter Metadatei-DC wird genauso wie ein Standardmetadatei-DC verwendet. OLE unterstützt erweiterte Metadateien und enthält Abwärtskompatibilität mit Standardmetadateien.

ActiveX Steuerelementcontainer sollten erweiterte Metadateien anstelle von Standardmetadateien verwenden.
