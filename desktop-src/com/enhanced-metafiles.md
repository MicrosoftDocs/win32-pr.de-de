# <a name="enhanced-metafiles"></a>Erweiterte Metadatendateien

Nicht überraschend bieten erweiterte Metadatendateien mehr Funktionalität als standardmetadatendateien. die Verwendung erweiterter Metadatendateien vereinfacht das Rendern von Code. Ein erweiterter metadateidc wird genauso wie ein standardmäßiger dateidc verwendet. OLE unterstützt erweiterte Metadatendateien und umfasst Abwärtskompatibilität mit standardmetadatendateien.

ActiveX-Steuerelement Container sollten erweiterte Metadatendateien anstelle von standardmetadatendateien verwenden.
