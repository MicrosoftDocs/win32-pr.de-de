# <a name="adding-a-component-to-a-visual-basic-project"></a>Hinzufügen einer Komponente zu einem Visual Basic Projekt

Im folgenden Verfahren wird beschrieben, wie ein COM-Objekt als Komponente zu einem Visual Basic-Projekt hinzugefügt wird. Die Anwendung kann dann die Klassen dieses Objekts verwenden.

Wenn Sie ein-Steuerelement als Komponente hinzufügen, werden die Eigenschaften und Methoden des Visual Basic Extender so verfügbar gemacht, als wären Sie Teil des Steuer Elements Wenn Sie dagegen ein Objekt als Verweis hinzufügen, können Sie nur die Typbibliothek verwenden, die vom Steuerelement bereitgestellt wird, oder die Typbibliothek "RAW".

**So fügen Sie ein Steuerelement in ein Visual Basic Projekt ein**

1.  Klicken Sie im Menü **Projekt** auf **Komponenten**.
2.  Aktivieren Sie das Kontrollkästchen neben der Komponente, die Sie hinzufügen möchten. Wenn die Komponente nicht aufgelistet ist, suchen Sie die dll-oder OCX-Datei mithilfe von **Durchsuchen**.
3.  Klicken Sie auf **OK**.

Die Komponente ist nun Teil des Projekts und wird in der Objekt Symbolleiste angezeigt. Die Anwendung kann Instanzen des Steuer Elements erstellen, indem Sie das Steuerelement von der Symbolleiste ziehen und in einem Formular oder Dialogfeld ablegen.
