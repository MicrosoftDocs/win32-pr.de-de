# <a name="adding-a-component-to-a-visual-basic-project"></a>Hinzufügen einer Komponente zu Visual Basic Project

Im folgenden Verfahren wird beschrieben, wie Sie einem Projekt ein COM-Objekt als Komponente Visual Basic hinzufügen. Ihre Anwendung kann dann die Klassen dieses Objekts verwenden.

Das Hinzufügen eines Steuerelements als Komponente macht die Visual Basic Extendereigenschaften und -methoden verfügbar, als ob sie Teil des Steuerelements sind. Wenn Sie dagegen ein Objekt als Verweis hinzufügen, können Sie nur die vom Steuerelement bereitgestellte Typbibliothek oder die "rohe" Typbibliothek verwenden.

**So fügen Sie ein Steuerelement in ein Visual Basic ein**

1.  Klicken Sie **Project** menü auf **Komponenten**.
2.  Klicken Sie auf diese Option, um das Kontrollkästchen neben der Komponente zu aktivieren, die Sie hinzufügen möchten. Wenn die Komponente nicht aufgeführt ist, suchen Sie die .dll oder OCX-Datei, indem **Sie durchsuchen.**
3.  Klicken Sie auf **OK**.

Die Komponente ist jetzt Teil des Projekts und wird auf der Objektsymbolleiste angezeigt. Ihre Anwendung kann Instanzen des Steuerelements erstellen, indem sie das Steuerelement aus der Symbolleiste zieht und es in ein Formular oder Dialogfeld ablegen.
