# <a name="nummethods"></a>NumMethods

Enthält die Anzahl der Methoden in der zugeordneten-Schnittstelle, einschließlich Methoden aus abgeleiteten Schnittstellen.

## <a name="registry-entry"></a>Registrierungseintrag

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Interface
   {IID}
      NumMethods = value
```

## <a name="remarks"></a>Bemerkungen

Dies ist ein **reg \_ SZ** -Wert.
