# <a name="nummethods"></a>NumMethods

Enthält die Anzahl der Methoden in der zugeordneten Schnittstelle, einschließlich Methoden von abgeleiteten Schnittstellen.

## <a name="registry-entry"></a>Registrierungseintrag

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Interface
   {IID}
      NumMethods = value
```

## <a name="remarks"></a>Hinweise

Dies ist ein **REG \_ SZ-Wert.**
