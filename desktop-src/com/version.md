# <a name="version-com"></a>Version (COM)

Gibt die Versionsnummer des Steuerelements an.

## <a name="registry-entry"></a>Registrierungseintrag

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID
   {CLSID}
      Version = value
```

## <a name="remarks"></a>Hinweise

Dies ist ein **REG \_ SZ-Wert.**

Die Versionsnummer sollte mit der Version der Typbibliothek übereinstimmen, die dem Steuerelement zugeordnet ist.
