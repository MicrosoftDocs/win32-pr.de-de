# <a name="version-com"></a>Version (com)

Gibt die Versionsnummer des Steuer Elements an.

## <a name="registry-entry"></a>Registrierungseintrag

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID
   {CLSID}
      Version = value
```

## <a name="remarks"></a>Bemerkungen

Dies ist ein **reg \_ SZ** -Wert.

Die Versionsnummer muss mit der Version der Typbibliothek, die dem Steuerelement zugeordnet ist, entsprechen.
