# <a name="java-type-library-wizard"></a>Assistent für die Java-Typbibliothek

Der Assistent für die Java-Typbibliothek ist eine Komponente von Visual J++ 5,0. Dieser Assistent generiert Klassendateien und Java-Typbibliotheks Informationen für ein COM-Objekt.

**So generieren Sie Java-Klassendateien und Typbibliotheks Informationen**

1.  Klicken Sie **im Menü Extras** auf **Assistent für die Java-Typbibliothek**. Eine Liste der installierten COM-Komponenten wird angezeigt.
2.  Aktivieren Sie diese Option, um die Kontrollkästchen neben den Elementen auszuwählen, die Sie konvertieren möchten.
3.  Klicken Sie auf **OK**. Der Typbibliotheks-Assistent generiert Klassendateien für die ausgewählten Komponenten. Er gibt die **Import** Anweisung aus, die erforderlich ist, um jede Komponente in eine Java-Anwendung oder ein Applet einzufügen, sowie eine Zusammenfassungs Textdatei, die die Java-Typbibliotheks Informationen für das Objekt enthält.
