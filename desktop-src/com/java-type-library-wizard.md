# <a name="java-type-library-wizard"></a>Java-Typbibliotheks-Assistent

Der Java-Typbibliotheks-Assistent ist eine Komponente von Visual J++ 5.0. Dieser Assistent generiert Klassendateien und Java-Typbibliotheksinformationen für ein COM-Objekt.

**So generieren Sie Java-Klassendateien und Typbibliotheksinformationen**

1.  Klicken Sie **im Menü Extras** auf **Java-Typbibliotheks-Assistent**. Eine Liste der installierten COM-Komponenten wird angezeigt.
2.  Klicken Sie auf diese Option, um die Kontrollkästchen neben den Elementen zu aktivieren, die Sie konvertieren möchten.
3.  Klicken Sie auf **OK**. Der Typbibliotheks-Assistent generiert Klassendateien für die ausgewählten Komponenten. Es gibt die **import-Anweisung** aus, die zum Einfügen der einzelnen Komponenten in eine Java-Anwendung oder ein Applet erforderlich ist, sowie eine Zusammenfassungstextdatei, die die Java-Typbibliotheksinformationen für das Objekt enthält.
