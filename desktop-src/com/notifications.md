# <a name="notifications-com"></a>Benachrichtigungen (COM)

Benachrichtigungen sind Rückrufe, die von einem Objekt generiert werden, wenn es eine Änderung des Namens, des Zustands, der Daten oder der Darstellung erkennt. Container und andere Clients erfordern Benachrichtigungen, um angemessen auf diese Änderungen zu reagieren. Ein Container registriert sich für den Empfang von Benachrichtigungen, indem eine Beratungsverbindung mit einem objekt von Interesse eingerichtet wird. Andere interessierten Clients können dies auch tun. Der Container erstellt auch eine Empfehlungssenke zum Empfangen der Benachrichtigungen. Wenn die vom Container hergestellte Verbindung verwendet wird, benachrichtigt ein Objekt, bei dem eine Änderung festgestellt wird, die Empfehlungssenke. Nach dem Empfang einer Benachrichtigung führt der Container jede Aktion aus, die für den Typ der aufgetretenen Änderung definiert wurde.

-   [Funktionsweise von Benachrichtigungen](how-notifications-work.md)
