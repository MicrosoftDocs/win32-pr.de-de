# <a name="notifications-com"></a>Benachrichtigungen (com)

Benachrichtigungen sind Rückrufe, die von einem Objekt generiert werden, wenn es eine Änderung des Namens, des Zustands, der Daten oder der Präsentation erkennt. Für Container und andere Clients sind Benachrichtigungen erforderlich, damit Sie entsprechend auf diese Änderungen reagieren können. Ein Container registriert sich für den Empfang von Benachrichtigungen durch Einrichten einer Beratungs Verbindung mit einem Objekt, das von Interesse ist. Andere interessierte Clients können dasselbe tun. Der Container erstellt außerdem eine Beratungs Senke, um die Benachrichtigungen zu empfangen. Bei Verwendung der vom Container eingerichteten Verbindung wird von einem Objekt, das eine Änderung erfährt, die Beratungs Senke benachrichtigt. Beim Empfang einer Benachrichtigung führt der Container die für den Typ der aufgetretenen Änderung definierten Aktionen aus.
