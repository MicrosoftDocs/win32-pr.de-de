# <a name="control-methods"></a>Steuerungsmethoden

Es gibt drei Standardmethoden, die Steuerelemente unterstützen können: aktualisieren, DoClick und AboutBox. Diese Standardmethoden verfügen über negative DISPID-Werte, die ihren Standardstatus angeben.
