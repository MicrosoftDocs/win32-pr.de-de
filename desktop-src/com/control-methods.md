# <a name="control-methods"></a>Steuerelementmethoden

Es gibt drei Standardmethoden, die Steuerelemente unterstützen können: Refresh, DoClick und AboutBox. Diese Standardmethoden weisen negative DISPID-Werte auf, die ihren Standardstatus angeben.
