# <a name="using-oleview"></a>Verwenden von OleView

Während der Registrierungs-Editor eine nicht verarbeitete Ansicht der Registrierung zeigt und hilfreich ist, um die Organisation der Registrierung zu erlernen, kann die Verwendung des Registrierungs-Editors zeitaufwändig sein.

OleView bietet eine Ansicht der in der Registrierung enthaltenen Informationen auf höherer Ebene. Anstelle einer langen Liste von CLSIDs bietet OleView Struktursteuerelemente mit Benutzerfreundlichen Namen.
