# <a name="using-oleview"></a>Verwenden von OLEVIEW

Der Registrierungs-Editor zeigt zwar eine nicht verarbeitete Ansicht der Registrierung an und ist hilfreich zum Erlernen der Organisation der Registrierung, aber die Verwendung des Registrierungs-Editors kann zeitaufwändig sein.

OLEVIEW bietet eine allgemeine Übersicht über die in der Registrierung enthaltenen Informationen. Anstatt eine lange Liste von CLSIDs anzusehen, bietet OLEVIEW Struktur Steuerelemente mit anzeigen Amen.
