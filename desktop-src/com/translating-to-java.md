# <a name="translating-to-java"></a>Übersetzen in Java

Das Microsoft Visual J++-Entwicklungssystem für Java bietet zwei Tools, mit denen Sie Typbibliotheksinformationen für ein COM-Objekt in der Java-Syntax generieren können: den Java-Typbibliotheks-Assistenten und das JActiveX-Befehlszeilentool. Mit einem dieser Tools können Sie Java-Klassendateien für ein COM-Objekt generieren.

In Visual J++ Version 6.0 ersetzt das JActiveX-Tool das JavaTLB-Befehlszeilentool. Weitere Informationen finden Sie in der Visual J++-Dokumentation.

Wenn Sie eine andere Java-Entwicklungsumgebung als Visual J++ verwenden, finden Sie in der Dokumentation dieses Produkts Informationen zu COM-Konvertierungstools.

Nachdem Sie die Klassendateien für eine Komponente generiert haben, können Sie die Komponentenklassen ihrer Java-Anwendung mithilfe der **import-Anweisung** hinzufügen.

Anschließend können Sie die Klassen des Objekts in Ihrer Java-Anwendung erstellen und verwenden. Weitere Informationen zur **Java-Import-Anweisung** finden Sie in einer Referenz zur Programmiersprache Java.
