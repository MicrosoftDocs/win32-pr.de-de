# <a name="translating-to-java"></a>Übersetzen in Java

Das Microsoft Visual J++-Entwicklungssystem für Java bietet zwei Tools, mit denen Sie Typbibliotheks Informationen für ein COM-Objekt in der Java-Syntax, den Assistenten für die Java-Typbibliothek und das Befehlszeilen Tool JActiveX generieren können. Mit einem dieser Tools können Sie Java-Klassendateien für ein COM-Objekt generieren.

In Visual J++, Version 6,0, ersetzt das Tool "JActiveX" das Befehlszeilen Tool JAVATLB. Weitere Informationen finden Sie in der Visual J++-Dokumentation.

Wenn Sie eine andere Java-Entwicklungsumgebung als Visual J++ verwenden, finden Sie in der Dokumentation des Produkts Informationen zu den Tools für die com-Konvertierung.

Nachdem Sie die Klassendateien für eine Komponente generiert haben, können Sie die Komponenten Klassen mithilfe der **Import** -Anweisung zu Ihrer Java-Anwendung hinzufügen.

Anschließend können Sie die-Klassen des-Objekts in Ihrer Java-Anwendung erstellen und verwenden. Weitere Informationen zur Java- **Import** -Anweisung finden Sie unter Referenz zur Programmiersprache Java.
