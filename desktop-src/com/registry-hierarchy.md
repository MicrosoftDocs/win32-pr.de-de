# <a name="registry-hierarchy"></a>Registrierungshierarchie

Die Registrierung ist als Hierarchie von Schlüsseln, Unterschlüsseln und benannten Werten oder einzelnen Standardwerten strukturiert. Unterschlüssel können mehrere Unterschlüssel und Werte aufweisen. Schlüssel werden durch Zeichenfolgen mit umgekehrten Schrägstrichen benannt. Jeder Schlüssel in der Registrierung kann einen oder mehrere Werte enthalten, die Zeichenfolgen, ganzzahlige Werte oder Binärdaten enthalten können.
