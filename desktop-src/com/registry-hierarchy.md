# <a name="registry-hierarchy"></a>Registrierungs Hierarchie

Die Registrierung ist als eine Hierarchie von Schlüsseln, unter Schlüsseln und benannten Werten oder einzelnen Standardwerten strukturiert. Unterschlüssel können mehrere Unterschlüssel und Werte aufweisen. Schlüssel werden durch umgekehrten Schrägstriche mit Trennzeichen getrennt benannt. Jeder Schlüssel in der Registrierung kann über einen oder mehrere Werte verfügen, die Zeichen folgen, ganzzahlige Werte oder Binärdaten enthalten können.
