# <a name="containers"></a>Container

Ein ActiveX-Steuerelement Container ist ein OLE-Container, der die folgenden zusätzlichen Funktionen unterstützt:

-   Eingebettete Objekte von in-Process-Servern
-   Direkte Aktivierung
-   OLEMISC \_ activateeinvisible
-   Ereignisbehandlung

ActiveX-Steuerelement Container müssen Unterstützung für alle diese Funktionen bieten.
