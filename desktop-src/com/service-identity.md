# <a name="service-identity"></a>Dienst Identität

Die Auswahl dieser Identität bewirkt, dass die Anwendung als Dienst ausgeführt wird. Sie können Dienst Konten festlegen, indem Sie in der **Systemsteuerung** die Option **Dienste** auswählen. Weitere Informationen finden Sie unter [Installieren von as-a-Service-Anwendungen](installing-as-a-service-application.md).
