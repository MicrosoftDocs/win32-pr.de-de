# <a name="type-library-viewers-and-conversion-tools"></a>Typbibliotheks-Viewer und Konvertierungstools

Typbibliotheken enthalten die Spezifikation für mindestens ein COM-Element, einschließlich Klassen, Schnittstellen, Enumerationen und mehr. Diese Dateien werden in einem Standardbinärformat gespeichert. Eine Typbibliothek kann eine eigenständige Datei mit der Erweiterung TLB sein oder als Ressource in einer ausführbaren Datei gespeichert werden, die eine OCX-, .dll- oder .exe Dateinamenerweiterung aufweisen kann. Die im Folgenden beschriebenen Typbibliotheks-Viewer und Konvertierungstools lesen dieses Format, um Informationen zu den COM-Elementen in der Bibliothek zu erhalten.

Bevor Sie ein Objekt in einer bestimmten Programmiersprache programmieren können, müssen Sie dessen Typbibliothek in dieser Sprache anzeigen können. Dadurch erhalten Sie die richtige Syntax für die Klassen, Schnittstellen, Methoden, Eigenschaften und Ereignisse des COM-Objekts.

Microsoft-Entwicklungsprodukte bieten die folgenden Tools, mit denen Sie Typbibliotheksinformationen generieren, extrahieren und anzeigen können.
