# <a name="type-library-viewers-and-conversion-tools"></a>Typbibliotheks-Viewer und Konvertierungs Tools

Typbibliotheken enthalten die Spezifikation für ein oder mehrere com-Elemente, einschließlich Klassen, Schnittstellen, Enumerationen usw. Diese Dateien werden in einem standardmäßigen Binärformat gespeichert. Eine Typbibliothek kann eine eigenständige Datei mit der Dateinamenerweiterung ". tlb" sein oder als Ressource in einer ausführbaren Datei gespeichert werden, die eine Dateinamenerweiterung ". ocx", ". dll" oder ". exe" aufweisen kann. Die nachfolgend beschriebenen Typbibliotheks-Viewer und Konvertierungs Tools lesen dieses Format, um Informationen über die com-Elemente in der Bibliothek zu erhalten.

Bevor Sie ein Objekt in einer bestimmten Programmiersprache programmieren können, müssen Sie in der Lage sein, die Typbibliothek in dieser Sprache anzuzeigen. Dadurch erhalten Sie die richtige Syntax für die Klassen, Schnittstellen, Methoden, Eigenschaften und Ereignisse des COM-Objekts.

Microsoft-Entwicklungsprodukte bieten die folgenden Tools, mit denen Sie Typbibliotheks Informationen generieren, extrahieren und anzeigen können.
