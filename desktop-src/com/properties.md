# <a name="exposing-properties-through-idispatch"></a>Verfügbar machen von Eigenschaften über IDispatch

Obwohl die meisten Steuerelemente über Eigenschaften verfügen, ist es nicht erforderlich, dass Steuerelemente Eigenschaften verfügbar machen, sodass das Steuerelement nicht **IDispatch** benötigt. Wenn das Steuerelement über Eigenschaften verfügt, gibt es keine Anforderungen, welche Eigenschaften ein Steuerelement verfügbar machen muss.
