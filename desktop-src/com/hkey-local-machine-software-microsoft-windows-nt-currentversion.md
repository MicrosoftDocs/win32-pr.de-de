# <a name="hkey_local_machinesoftwaremicrosoftwindows-ntcurrentversion"></a>HKEY \_ LOCAL MACHINE SOFTWARE Microsoft Windows NT \_ \\ \\ \\ \\ CurrentVersion

Die Unterschlüssel und Registrierungswerte, die **HKEY \_ LOCAL MACHINE SOFTWARE Microsoft Windows NT \_ \\ \\ \\ \\ CurrentVersion** zugeordnet sind, enthalten Informationen zur COM RPC-Debugfunktion.



| Unterschlüssel                                                 | Beschreibung                                         |
|--------------------------------------------------------|-----------------------------------------------------|
| [**DebugObjectRPCEnabled**](debugobjectrpcenabled.md) | Konfigurationsoptionen für COM RPC-Remotedebuggen. |
