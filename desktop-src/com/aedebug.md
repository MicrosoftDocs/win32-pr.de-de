# <a name="aedebug"></a>AEDebug

Das **AEDebug** definiert remotedebuggingoptionen für com.

## <a name="registry-key"></a>Registrierungsschlüssel

**HKEY \_ local \_ Machine \\ Software \\ Microsoft \\ Windows NT \\ CurrentVersion \\ debugobjectrpcenabled \\ AEDebug**



| Registrierungs Wert               | BESCHREIBUNG                                           |
|------------------------------|-------------------------------------------------------|
| [**Automatisch**](auto.md)         | Bestimmt, ob der Debugger automatisch gestartet wird. |
| [**Debugger**](debugger.md) | Speichert den Namen des Debuggers.                             |
