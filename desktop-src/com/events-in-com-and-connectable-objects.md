<span data-ttu-id="7e857-104">Wenn ein Programm etwas erkennt, das aufgetreten ist, kann es seine Clients benachrichtigen.</span><span class="sxs-lookup"><span data-stu-id="7e857-104">When a program detects something that has happened, it can notify its clients.</span></span> <span data-ttu-id="7e857-105">Wenn z. b. ein Börsen Ticker Programm eine Änderung am Preis eines Aktien Ereignisses entdeckt, kann es alle Clients über die Änderung benachrichtigen.</span><span class="sxs-lookup"><span data-stu-id="7e857-105">For example, if a stock ticker program detects a change in the price of a stock, it can notify all clients of the change.</span></span> <span data-ttu-id="7e857-106">Dieser Benachrichtigungs Prozess wird als auslösen *eines Ereignisses* bezeichnet.</span><span class="sxs-lookup"><span data-stu-id="7e857-106">This notification process is referred to as *firing an event*.</span></span>

<span data-ttu-id="7e857-107">Mit com können Server Objekte com-Ereignisse verwenden, um Ereignisse ohne Informationen darüber auszulösen, welche Objekte benachrichtigt werden.</span><span class="sxs-lookup"><span data-stu-id="7e857-107">With COM, server objects can use COM events to fire events without any information about what objects will be notified.</span></span> <span data-ttu-id="7e857-108">Objekte können auch *Verbindungs fähige Objekte* verwenden, um ausführliche Informationen über Clients zu erhalten, die Benachrichtigungen angefordert haben.</span><span class="sxs-lookup"><span data-stu-id="7e857-108">Objects can also use *connectable objects* to maintain detailed information about clients who have requested notifications.</span></span>

<span data-ttu-id="7e857-109">COM-Verbindungs fähige Objekte stellen neben Ihren eingehenden Schnittstellen ausgehende Schnittstellen für Ihre Clients bereit.</span><span class="sxs-lookup"><span data-stu-id="7e857-109">COM connectable objects provide outgoing interfaces to their clients in addition to their incoming interfaces.</span></span> <span data-ttu-id="7e857-110">Demzufolge können Objekte und Ihre Clients an der bidirektionalen Kommunikation beteiligt sein.</span><span class="sxs-lookup"><span data-stu-id="7e857-110">As a result, objects and their clients can engage in bidirectional communication.</span></span> <span data-ttu-id="7e857-111">Eingehende Schnittstellen werden für ein Objekt implementiert und empfangen Aufrufe von externen Clients eines Objekts, während ausgehende Schnittstellen auf der Senke des Clients implementiert werden und Aufrufe vom-Objekt empfangen werden.</span><span class="sxs-lookup"><span data-stu-id="7e857-111">Incoming interfaces are implemented on an object and receive calls from external clients of an object, while outgoing interfaces are implemented on the client's sink and receive calls from the object.</span></span> <span data-ttu-id="7e857-112">Das-Objekt definiert eine Schnittstelle, die Sie verwenden möchten, und der Client implementiert diese.</span><span class="sxs-lookup"><span data-stu-id="7e857-112">The object defines an interface it would like to use, and the client implements it.</span></span>

<span data-ttu-id="7e857-113">Ein-Objekt definiert seine eingehenden Schnittstellen und stellt Implementierungen dieser Schnittstellen bereit.</span><span class="sxs-lookup"><span data-stu-id="7e857-113">An object defines its incoming interfaces and provides implementations of these interfaces.</span></span> <span data-ttu-id="7e857-114">Eingehende Schnittstellen sind für Clients über die [**IUnknown:: QueryInterface**](/windows/desktop/api/Unknwn/nf-unknwn-iunknown-queryinterface(q)) -Methode des Objekts verfügbar.</span><span class="sxs-lookup"><span data-stu-id="7e857-114">Incoming interfaces are available to clients through the object's [**IUnknown::QueryInterface**](/windows/desktop/api/Unknwn/nf-unknwn-iunknown-queryinterface(q)) method.</span></span> <span data-ttu-id="7e857-115">Clients Ruft die Methoden einer eingehenden Schnittstelle für das-Objekt auf, und das-Objekt führt gewünschte Aktionen im Namen des Clients aus.</span><span class="sxs-lookup"><span data-stu-id="7e857-115">Clients call the methods of an incoming interface on the object, and the object performs desired actions on behalf of the client.</span></span>

<span data-ttu-id="7e857-116">Ausgehende Schnittstellen werden auch von einem Objekt definiert, aber der Client stellt die Implementierungen der ausgehenden Schnittstellen für ein Sink-Objekt bereit, das vom Client erstellt wird.</span><span class="sxs-lookup"><span data-stu-id="7e857-116">Outgoing interfaces are also defined by an object, but the client provides the implementations of the outgoing interfaces on a sink object that the client creates.</span></span> <span data-ttu-id="7e857-117">Das-Objekt ruft dann Methoden der ausgehenden Schnittstelle für das sink-Objekt auf, um den Client über Änderungen im Objekt zu benachrichtigen, Ereignisse im Client zu lösen, etwas vom Client anzufordern, oder tatsächlich für einen beliebigen Zweck, mit dem der Objekt Ersteller kommt.</span><span class="sxs-lookup"><span data-stu-id="7e857-117">The object then calls methods of the outgoing interface on the sink object to notify the client of changes in the object, to trigger events in the client, to request something from the client, or, in fact, for any purpose the object creator comes up with.</span></span>

<span data-ttu-id="7e857-118">Ein Beispiel für eine ausgehende Schnittstelle ist eine ibuttonsink-Schnittstelle, die von einem Push-Schaltflächen-Steuerelement definiert wird, um die Clients über die</span><span class="sxs-lookup"><span data-stu-id="7e857-118">An example of an outgoing interface is an IButtonSink interface defined by a push button control to notify its clients of its events.</span></span> <span data-ttu-id="7e857-119">Beispielsweise ruft das Schaltflächen Objekt ibuttonsink:: OnClick für das sink-Objekt des Clients auf, wenn der Benutzer auf die Schaltfläche auf dem Bildschirm klickt.</span><span class="sxs-lookup"><span data-stu-id="7e857-119">For example, the button object calls IButtonSink::OnClick on the client's sink object when the user clicks the button on the screen.</span></span> <span data-ttu-id="7e857-120">Das Button-Steuerelement definiert die ausgehende Schnittstelle.</span><span class="sxs-lookup"><span data-stu-id="7e857-120">The button control defines the outgoing interface.</span></span> <span data-ttu-id="7e857-121">Damit ein Client der Schaltfläche das Ereignis behandelt, muss der Client diese ausgehende Schnittstelle in einem Sink-Objekt implementieren und dann die Senke mit dem Schaltflächen-Steuerelement verbinden.</span><span class="sxs-lookup"><span data-stu-id="7e857-121">For a client of the button to handle the event, the client must implement that outgoing interface on a sink object and then connect that sink to the button control.</span></span> <span data-ttu-id="7e857-122">Wenn dann Ereignisse in der Schaltfläche auftreten, ruft die Schaltfläche die Senke auf. zu diesem Zeitpunkt kann der Client jede beliebige Aktion ausführen, die diesem Schaltflächen Klick zugewiesen werden soll.</span><span class="sxs-lookup"><span data-stu-id="7e857-122">Then, when events occur in the button, the button will call the sink, at which time the client can execute whatever action it wishes to assign to that button click.</span></span>

<span data-ttu-id="7e857-123">Verbindungs fähige Objekte bieten einen allgemeinen Mechanismus für die Kommunikation zwischen Objekten.</span><span class="sxs-lookup"><span data-stu-id="7e857-123">Connectable objects provide a general mechanism for object-to-client communication.</span></span> <span data-ttu-id="7e857-124">Diese Technologie kann von jedem Objekt verwendet werden, das Ereignisse oder Benachrichtigungen verfügbar machen möchte.</span><span class="sxs-lookup"><span data-stu-id="7e857-124">Any object that wishes to expose events or notifications of any kind can use this technology.</span></span> <span data-ttu-id="7e857-125">Zusätzlich zur allgemeinen Verbindungs fähigen Objekttechnologie bietet com viele besondere Zweck senken und Standort Schnittstellen, die von Objekten verwendet werden, um Clients über bestimmte Ereignisse zu benachrichtigen, die für den Client von Interesse sind.</span><span class="sxs-lookup"><span data-stu-id="7e857-125">In addition to the general connectable object technology, COM provides many special purpose sink and site interfaces used by objects to notify clients of specific events of interest to the client.</span></span> <span data-ttu-id="7e857-126">Beispielsweise kann [**IAdviseSink**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink) von Objekten verwendet werden, um Clients von Daten zu benachrichtigen und Änderungen im Objekt anzuzeigen.</span><span class="sxs-lookup"><span data-stu-id="7e857-126">For example, [**IAdviseSink**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink) may be used by objects to notify clients of data and view changes in the object.</span></span>
