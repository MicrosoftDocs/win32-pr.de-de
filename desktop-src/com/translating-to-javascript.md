# <a name="translating-to-javascript"></a>Übersetzen in JavaScript

JavaScript ist eine Skriptsprache, die von Netscape erstellt wurde. Er wurde ursprünglich als LiveScript bezeichnet. Trotz der Ähnlichkeiten in Name und Syntax steht JavaScript nicht im Zusammenhang mit der Programmiersprache Java. Java ist eine Programmiersprache mit vollem Funktionsumfang, die von Sun Microsystems, Inc. entwickelt wurde.

JavaScript wird von einer Vielzahl von Webbrowsern unterstützt, einschließlich Internet Explorer und Netscape Navigator.

Zusätzlich zu den von JavaScript bereitgestellten sprach Features kann der Skript Host auch ein Objektmodell verfügbar machen, auf das Ihr Skript zugreifen kann. Internet Explorer stellt z. b. ein Objektmodell zur Verfügung, das es Skripts ermöglicht, mit dem Browser Programm gesteuert zu interagieren. Weitere Informationen zum Arbeiten mit solchen Objekt Modellen finden Sie in der Dokumentation zu Ihrem Skript Host.

Weitere Informationen zu JavaScript finden Sie in der JavaScript-Referenz.

In den folgenden Themen werden Probleme beschrieben, die Sie beim Übersetzen eines Skripts in JavaScript aus VBScript oder JScript berücksichtigen sollten:

-   [Übersetzen von JScript in JavaScript](translating-to-javascript-from-jscript.md)
