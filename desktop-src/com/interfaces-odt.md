# <a name="interfaces-ole-and-data-transfer"></a>Schnittstellen (OLE und Datenübertragung)

Die folgenden Schnittstellen werden verwendet, um zusammengesetzte Dokumente zu implementieren und Datenübertragungen zwischen Anwendungen durchzuführen.



| Schnittstelle                                                          | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                               |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Iadvisesink**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink)                                 | Ermöglicht Containern und anderen Objekten den Empfang von Benachrichtigungen über Datenänderungen, Anzeigeänderungen und Verbunddokumentänderungen, die in objekten von Interesse auftreten.                                                                                                                                                                                              |
| [**IAdviseSink2**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink2)                               | Eine Erweiterung für [**IAdviseSink,**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink) die eine Methode zum Behandeln einer Änderung im Moniker eines verknüpften Objekts hinzufügt.                                                                                                                                                                                                                               |
| [**IAdviseSinkEx**](/windows/desktop/api/OCIdl/nn-ocidl-iadvisesinkex)                             | Eine Erweiterung für [**IAdviseSink,**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink) die die Möglichkeit hinzufügt, die Senke über Änderungen im Ansichtsstatus eines Objekts zu benachrichtigen.                                                                                                                                                                                                                        |
| [**IDataAdviseHolder**](/windows/desktop/api/ObjIdl/nn-objidl-idataadviseholder)                     | Erstellt und verwaltet Advise-Verbindungen zwischen einem Datenobjekt und einem oder mehreren Advise-Senken.                                                                                                                                                                                                                                                              |
| [**Idataobject**](/windows/desktop/api/ObjIdl/nn-objidl-idataobject)                                 | Ermöglicht die Datenübertragung und Benachrichtigung über Änderungen an Daten.                                                                                                                                                                                                                                                                                                |
| [**IDropSource**](/windows/desktop/api/OleIdl/nn-oleidl-idropsource)                                 | Eine der Schnittstellen, die Sie implementieren müssen, um Drag & Drop-Vorgänge in Ihrer Anwendung bereitzustellen.                                                                                                                                                                                                                                                         |
| [**IDropSourceNotify**](/windows/desktop/api/OleIdl/nn-oleidl-idropsourcenotify)                     | Empfängt Benachrichtigungen, wenn ein Benutzer den Mauszeiger in ein oder aus einem potenziellen Ablagezielfenster zieht.                                                                                                                                                                                                                                                         |
| [**Idroptarget**](/windows/desktop/api/OleIdl/nn-oleidl-idroptarget)                                 | Eine der Schnittstellen, die Sie implementieren müssen, um Drag & Drop-Vorgänge in Ihrer Anwendung bereitzustellen.                                                                                                                                                                                                                                                         |
| [**IEnterpriseDropTarget**](/windows/desktop/api/OleIdl/nn-oleidl-ienterprisedroptarget)             | Bei Der Implementierung durch die Ablagezielanwendung bietet diese Schnittstelle der OLE-Drag & Drop-Engine die Möglichkeit, zu bestimmen, ob die Dropzielanwendung die Unternehmensschutzrichtlinie auswerten möchte, und bietet der OLE-Drag & Drop-Engine eine Möglichkeit, die Unternehmens-ID der Anwendung zum Löschen der Quelle für die Ablagezielanwendung bereitzustellen. |
| [**IEnumFORMATETC**](/windows/desktop/api/ObjIdl/nn-objidl-ienumformatetc)                           | Listet die [**FORMATETC-Strukturen**](/windows/win32/api/objidl/ns-objidl-formatetc) auf, die die Formate und Medien definieren, die von einem bestimmten Datenobjekt unterstützt werden.                                                                                                                                                                                                                              |
| [**IEnumMoniker**](/windows/desktop/api/ObjIdl/nn-objidl-ienummoniker)                               | Listet die Moniker in einer Tabelle von Monikern auf.                                                                                                                                                                                                                                                                                                           |
| [**IEnumOleDocumentViews**](/windows/desktop/api/DocObj/nn-docobj-ienumoledocumentviews)             | Listet die ansichten auf, die von einem Dokumentobjekt unterstützt werden.                                                                                                                                                                                                                                                                                                      |
| [**IEnumOleUndoUnits**](/windows/desktop/api/OCIdl/nn-ocidl-ienumoleundounits)                     | Listet die Rückgängig-Komponenten auf dem Rückgängigmachen- oder Wiederholen-Stapel auf.                                                                                                                                                                                                                                                                                                      |
| [**IEnumOLEVERB**](/windows/desktop/api/OleIdl/nn-oleidl-ienumoleverb)                               | Listet die verschiedenen für ein Objekt verfügbaren Verben in aufsteigender Verbnummer auf.                                                                                                                                                                                                                                                                 |
| [**IEnumSTATDATA**](/windows/desktop/api/ObjIdl/nn-objidl-ienumstatdata)                             | Listet die Beratungsverbindungsinformationen für ein Datenobjekt auf.                                                                                                                                                                                                                                                                                         |
| [**Iobjectwithsite**](/windows/desktop/api/OCIdl/nn-ocidl-iobjectwithsite)                         | Bietet eine einfache Möglichkeit, die Kommunikation zwischen einem Objekt und seinem Standort im Container zu unterstützen.                                                                                                                                                                                                                                                           |
| [**IOleAdviseHolder**](/windows/desktop/api/OleIdl/nn-oleidl-ioleadviseholder)                       | Verwaltet Beratungverbindungen und zusammengesetzte Dokumentbenachrichtigungen auf einem Objektserver.                                                                                                                                                                                                                                                                     |
| [**IOleCache**](/windows/desktop/api/OleIdl/nn-oleidl-iolecache)                                     | Stellt die Steuerung der Präsentationsdaten bereit, die innerhalb eines Objekts zwischengespeichert werden.                                                                                                                                                                                                                                                                             |
| [**IOleCache2**](/windows/desktop/api/OleIdl/nn-oleidl-iolecache2)                                   | Ermöglicht Objektclients das selektive Aktualisieren jedes Caches, der mit [**IOleCache::Cache**](/windows/desktop/api/OleIdl/nf-oleidl-iolecache-cache)erstellt wurde.                                                                                                                                                                                                                                |
| [**IOleCacheControl**](/windows/desktop/api/OleIdl/nn-oleidl-iolecachecontrol)                       | Bietet eine ordnungsgemäße Wartung von Caches.                                                                                                                                                                                                                                                                                                                    |
| [**IOleClientSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleclientsite)                           | Stellt ein eingebettetes Objekt mit Informationen zum Speicherort und Umfang seiner Anzeigewebsite, seines Monikers, seiner Benutzeroberfläche und anderer Ressourcen bereit, die vom Container bereitgestellt werden.                                                                                                                                                                           |
| [**Iolecommandtarget**](/windows/desktop/api/DocObj/nn-docobj-iolecommandtarget)                     | Ermöglicht Es Objekten und ihren Containern, Befehle aneinander zu senden.                                                                                                                                                                                                                                                                                  |
| [**IOleContainer**](/windows/desktop/api/OleIdl/nn-oleidl-iolecontainer)                             | Listet Objekte in einem zusammengesetzten Dokument auf oder sperrt einen Container im Ausführungszustand.                                                                                                                                                                                                                                                                       |
| [**IOleDocument**](/windows/desktop/api/DocObj/nn-docobj-ioledocument)                               | Ermöglicht einem Dokumentobjekt die Kommunikation mit Containern, um Ansichten seiner Daten zu erstellen.                                                                                                                                                                                                                                                           |
| [**IOleDocumentSite**](/windows/desktop/api/DocObj/nn-docobj-ioledocumentsite)                       | Aktiviert ein Dokument, das als Dokumentobjekt implementiert wurde, um die normale Aktivierungssequenz für direkt aktive Objekte zu umgehen und seine Clientwebsite direkt anzuweisen, es als Dokumentobjekt zu aktivieren.                                                                                                                                      |
| [**IOleDocumentView**](/windows/desktop/api/DocObj/nn-docobj-ioledocumentview)                       | Ermöglicht einem Container die Kommunikation mit jeder Ansicht, die von einem Dokumentobjekt unterstützt wird.                                                                                                                                                                                                                                                                         |
| [**IOleInPlaceActiveObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceactiveobject)         | Stellt einen direkten Kommunikationskanal zwischen einem direkten Objekt und dem äußeren Framefenster der zugeordneten Anwendung und dem Dokumentfenster in der Anwendung bereit, das das eingebettete Objekt enthält.                                                                                                                                          |
| [**IOleInPlaceFrame**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceframe)                       | Steuert das Rahmenfenster der obersten Ebene des Containers.                                                                                                                                                                                                                                                                                                          |
| [**IOleInPlaceObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceobject)                     | Verwaltet die Aktivierung und Deaktivierung von direkten Objekten und bestimmt, wie viel des direkten Objekts angezeigt werden soll.                                                                                                                                                                                                                            |
| [**IOleInPlaceObjectWindowless**](/windows/desktop/api/OCIdl/nn-ocidl-ioleinplaceobjectwindowless) | Ermöglicht es einem fensterlosen Objekt, Fenstermeldungen zu verarbeiten und an Drag & Drop-Vorgängen teilzunehmen.                                                                                                                                                                                                                                                       |
| [**IOleInPlaceSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplacesite)                         | Verwaltet die Interaktion zwischen dem Container und dem eigentlichen Clientstandort des Objekts.                                                                                                                                                                                                                                                                          |
| [**IOleInPlaceSiteEx**](/windows/desktop/api/OCIdl/nn-ocidl-ioleinplacesiteex)                     | Stellt einen zusätzlichen Satz von Aktivierungs- und Deaktivierungsbenachrichtigungsmethoden bereit, die es einem Objekt ermöglichen, unnötiges Blinken auf dem Bildschirm zu vermeiden, wenn das Objekt aktiviert und deaktiviert wird.                                                                                                                                                            |
| [**IOleInPlaceSiteWindowless**](/windows/desktop/api/OCIdl/nn-ocidl-ioleinplacesitewindowless)     | Ermöglicht es einem fensterlosen Objekt, Fenstermeldungen zu verarbeiten, an Drag & Drop-Vorgängen teilzunehmen und Zeichnungsvorgänge auszuführen.                                                                                                                                                                                                                       |
| [**IOleInPlaceUIWindow**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceuiwindow)                 | Handelt Rahmenbereich für das Dokument- oder Rahmenfenster aus.                                                                                                                                                                                                                                                                                                  |
| [**IOleLink**](/windows/desktop/api/OleIdl/nn-oleidl-iolelink)                                       | Ermöglicht einem verknüpften Objekt, seinen Container mit Funktionen für das Verknüpfen bereitzustellen.                                                                                                                                                                                                                                                                    |
| [**IOleObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleobject)                                   | Ermöglicht es einem eingebetteten Objekt, grundlegende Funktionen für seinen Container bereitzustellen und mit ihm zu kommunizieren.                                                                                                                                                                                                                                                       |
| [**IOleParentUndoUnit**](/windows/desktop/api/OCIdl/nn-ocidl-ioleparentundounit)                   | Aktiviert Rückgängigeinheiten, um untergeordnete Rückgängig-Einheiten zu enthalten.                                                                                                                                                                                                                                                                                                           |
| [**IOleUILinkContainer**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuilinkcontainera)                 | Allgemeine OLE-Dialogfelder verwenden diese Schnittstelle, um die Eigenschaften der Links eines Containers zu verwalten.                                                                                                                                                                                                                                                               |
| [**IOleUILinkInfo**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuilinkinfoa)                           | Eine Erweiterung der [**IOleUILinkContainer-Schnittstelle,**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuilinkcontainera) mit der Container die Seite **Link** des Dialogfelds **Objekteigenschaften** unterstützen können.                                                                                                                                                                            |
| [**IOleUIObjInfo**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuiobjinfoa)                             | Stellt Informationen bereit, die von den Seiten **Allgemein** und **Ansicht** des Dialogfelds **Objekteigenschaften** verwendet werden, in denen Informationen zur Größe, position, zum Typ und zum Namen des Objekts angezeigt werden. Außerdem kann das Objekt über das Dialogfeld **Konvertieren** konvertiert werden.                                                                                          |
| [**IOleUndoManager**](/windows/desktop/api/OCIdl/nn-ocidl-ioleundomanager)                         | Ermöglicht Containern das Implementieren von Rückgängig- und Wiederholungsvorgängen auf mehreren Ebenen für Aktionen, die in enthaltenen Steuerelementen auftreten.                                                                                                                                                                                                                                    |
| [**IOleUndoUnit**](/windows/desktop/api/OCIdl/nn-ocidl-ioleundounit)                               | Der Rückgängig-Manager ruft die Methoden in dieser Schnittstelle auf, um Rückgängigaktionen auszuführen und Zeichenfolgen abzurufen, die auf der Benutzeroberfläche angezeigt werden können, um die Rückgängig-Aktion zu beschreiben.                                                                                                                                                                                  |
| [**IOleWindow**](/windows/desktop/api/OleIdl/nn-oleidl-iolewindow)                                   | Ermöglicht es einer Anwendung, das Handle für die verschiedenen Fenster abzurufen, die an der aktivierungsbasierten Aktivierung beteiligt sind, sowie den kontextbezogenen Hilfemodus zu aktivieren und zu beenden.                                                                                                                                                                                       |
| [**IViewObject**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject)                                 | Ermöglicht es einem Objekt, sich selbst direkt anzuzeigen, ohne ein Datenobjekt an den Aufrufer zu übergeben. Darüber hinaus kann diese Schnittstelle eine Verbindung mit einer Advise-Senke erstellen und verwalten, damit der Aufrufer über Änderungen im Ansichtsobjekt benachrichtigt werden kann.                                                                                                                 |
| [**IViewObject2**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject2)                               | Eine Erweiterung der [**IViewObject-Schnittstelle,**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject) die die Größe der Zeichnung für eine bestimmte Ansicht eines Objekts zurückgibt.                                                                                                                                                                                                                      |
| [**IViewObjectEx**](/windows/desktop/api/OCIdl/nn-ocidl-iviewobjectex)                             | Eine Erweiterung der [**IViewObject2-Schnittstelle,**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject2) die Unterstützung für flimmerfreies Zeichnen, Treffertests und die Größenansteuerung bietet.                                                                                                                                                                                                       |
