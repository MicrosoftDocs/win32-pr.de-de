# <a name="adding-an-activex-control-to-a-visual-c-project"></a>Hinzufügen eines ActiveX-Steuer Elements zu einem Visual C++ Projekt

Mit dem folgenden Verfahren werden einem Visual C++ Projekt die Dispatch-und Header Dateien für ein ActiveX-Steuerelement hinzugefügt.

**So fügen Sie einem Visual C++ Projekt ein ActiveX-Steuerelement hinzu**

1.  Klicken Sie im Menü **Projekt** auf **zu Projekt hinzufügen**. Ein Kontextmenü wird angezeigt.
2.  Klicken Sie auf **Komponenten und Steuerelemente**. Das Dialogfeld **Komponenten und Steuerelemente Gallery** wird angezeigt.
3.  Klicken Sie auf die Komponente, die Sie dem Projekt hinzufügen möchten. Visual C++ zeigt ein Dialogfeld an, in dem Sie eine Teilmenge der Komponenten Klassen auswählen können, die dem Projekt hinzugefügt werden sollen.
4.  Aktivieren Sie die Kontrollkästchen für die Klassen, die Sie hinzufügen möchten, und klicken Sie auf **OK**.

Visual C++ generiert Dispatchklassen-und Header Dateien für die Komponente und fügt Sie dem Projekt hinzu.
