# <a name="exposing-methods-through-idispatch"></a>Verfügbar machen von Methoden über IDispatch

Obwohl die meisten Steuerelemente mehrere Methoden verfügbar machen und unterstützen, sind Steuerelemente nicht erforderlich, um Methoden verfügbar zu machen oder zu unterstützen, sodass das Steuerelement keine **IDispatch** erfordert. Wenn das Steuerelement über Methoden verfügt, gibt es keine Anforderungen, welche Methoden ein Steuerelement verfügbar machen muss.
