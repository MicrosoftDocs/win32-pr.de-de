# <a name="ipropertybag-and-ipersistpropertybag"></a>IPropertyBag und IPersistPropertyBag

[**IPropertyBag**](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/aa768196(v=vs.85)) und [**IPersistPropertyBag**](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/aa768205(v=vs.85)) optimieren die Speicherung als Text-Mechanismen und werden daher für ActiveX-Steuerelement Container empfohlen, die einen Save as Text-Mechanismus implementieren. **IPropertyBag** wird von einem Container implementiert und ist in etwa analog zu [**IStream**](/windows/desktop/api/objidl/nn-objidl-istream). **IPersistPropertyBag** wird von Steuerelementen implementiert und ist in etwa analog zu [**IPersistStream**](/windows/desktop/api/ObjIdl/nn-objidl-ipersiststream).
