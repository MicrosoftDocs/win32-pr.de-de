# <a name="extended-properties-events-and-methods"></a>Erweiterte Eigenschaften, Ereignisse und Methoden

ActiveX-Steuerelement Container müssen nicht erweiterte Steuerelemente unterstützen. Wenn der Steuerelement Container jedoch erweiterte Eigenschaften unterstützt, muss er die folgende minimale Menge unterstützen:

-   Sichtbar
-   Parent
-   Standard
-   Abbrechen

Derzeit haben erweiterte Eigenschaften, Ereignisse und Methoden keine Standard-DispIds.
