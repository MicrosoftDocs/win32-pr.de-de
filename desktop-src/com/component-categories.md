# <a name="component-categories"></a>Komponentenkategorien

Mit den Komponenten Kategorien von OLE können die Fähigkeiten und Anforderungen einer Softwarekomponente durch Einträge in der Registrierung identifiziert werden. In einem Szenario, in dem ein Container möglicherweise nicht in der Lage ist, einen Funktionsbereich wie z. b. Datenbindung zu unterstützen, möchte der Container keine Steuerelemente hosten, für die eine Datenbindung erforderlich ist, um seine Job-Funktion auszuführen. Mit Komponenten Kategorien können Bereiche von Funktionen wie z. b. Datenbindung identifiziert werden, damit der Steuerelement Container solche Steuerelemente vermeiden kann, die es als Anforderung festlegen. Komponenten Kategorien werden separat als Teil von OLE angegeben und sind nicht spezifisch für die Architektur des ActiveX-Steuer Elements. die Spezifikation für Komponenten Kategorien umfasst APIs zur Bearbeitung der Komponenten Kategorie-Registrierungsschlüssel.
