# <a name="component-categories"></a>Komponentenkategorien

Mit den Komponentenkategorien von OLE können die Fähigkeiten und Anforderungen einer Softwarekomponente durch Einträge in der Registrierung identifiziert werden. In einem Szenario, in dem ein Container einen Funktionsbereich wie z. B. die Datenbindung möglicherweise nicht unterstützen möchte oder nicht kann, möchte der Container keine Steuerelemente hosten, die eine Datenbindung erfordern, um seine Auftragsfunktion ausführen zu können. Komponentenkategorien ermöglichen das Identifizieren von Funktionalitätsbereichen, z. B. die Datenbindung, damit der Steuerelementcontainer die Steuerelemente vermeiden kann, die ihn als anforderungssteuerbar einräumen. Komponentenkategorien werden separat als Teil von OLE angegeben und sind nicht spezifisch für die ActiveX-Steuerelementarchitektur. Die Spezifikation für Komponentenkategorien enthält APIs zur Bearbeitung der Registrierungsschlüssel der Komponentenkategorie.
