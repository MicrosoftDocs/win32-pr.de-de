# <a name="translating-to-jscript"></a>Übersetzen in JScript

JScript ist die Microsoft-Implementierung der ECMA 262-Sprachspezifikation. Sie wird sowohl von Internet Explorer als auch von Netscape Navigator unterstützt. JScript ist die erste Skriptsprache, die dem ECMA 262-Standard entspricht.

JScript wird von einer Vielzahl von Webbrowsern unterstützt, einschließlich Internet Explorer und Navigator.

Zusätzlich zu den sprach Features, die von JScript bereitgestellt werden, kann der Skript Host auch ein Objektmodell verfügbar machen, auf das Ihr Skript zugreifen kann. Internet Explorer stellt z. b. ein Objektmodell zur Verfügung, das es Skripts ermöglicht, mit dem Browser Programm gesteuert zu interagieren. Weitere Informationen zum Arbeiten mit solchen Objekt Modellen finden Sie in der Dokumentation für Ihren Skript Host.

Weitere Informationen zu JScript finden Sie in der JScript-Sprachreferenz. Weitere Informationen finden Sie in der [ECMA 262-Sprachspezifikation](https://www.ecma-international.org/publications/standards/Ecma-262.htm).

In den folgenden Themen werden Probleme beschrieben, die Sie beim Übersetzen eines Skripts in JScript aus JavaScript oder VBScript berücksichtigen sollten:

-   [Übersetzen von VBScript in JScript](translating-to-jscript-from-vbscript.md)
-   [Übersetzen in JScript aus JavaScript](translating-to-jscript-from-javascript.md)
