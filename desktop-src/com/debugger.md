# <a name="debugger"></a>Debugger

Speichert den Namen des bevorzugten Debuggers, der für das Remotedebuggen von COM über RPC verwendet werden soll.

## <a name="registry-entry"></a>Registrierungseintrag

```
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\DebugObjectRPCEnabled\AeDebug
   Debugger = value
```

## <a name="remarks"></a>Hinweise

Dies ist ein **REG \_ SZ-Wert.**
