# <a name="events-in-controls"></a>Ereignisse in Steuerelementen

Obwohl die meisten Steuerelemente mehrere Ereignisse verfügbar machen und aussetzen, sind Steuerelemente nicht erforderlich, um Ereignisse verfügbar zu machen oder aus denen das Steuerelement keine [**IConnectionPointContainer erfordert.**](/windows/desktop/api/OCIdl/nn-ocidl-iconnectionpointcontainer) Wenn das Steuerelement über Ereignisse verfügt, gibt es keine Anforderungen, für welche Ereignisse ein Steuerelement verfügbar machen muss.
