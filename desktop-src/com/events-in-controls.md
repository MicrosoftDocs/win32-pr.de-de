# <a name="events-in-controls"></a>Ereignisse in Steuerelementen

Obwohl die meisten Steuerelemente mehrere Ereignisse verfügbar machen und auslösen, ist es nicht erforderlich, dass Steuerelemente Ereignisse verfügbar machen oder auslösen, sodass das Steuerelement nicht [**IConnectionPointContainer**](/windows/desktop/api/OCIdl/nn-ocidl-iconnectionpointcontainer)benötigt. Wenn das Steuerelement über Ereignisse verfügt, gibt es keine Anforderungen, welche Ereignisse ein Steuerelement verfügbar machen muss.
