# <a name="simple-data-binding"></a>einfache Datenbindung

Die Architektur der ActiveX-Steuerelemente definiert einen Daten Bindungs Mechanismus, bei dem ein ActiveX-Steuerelement angeben kann, dass eine oder mehrere Eigenschaften gebunden werden können. In den meisten Fällen sollte ein Daten gebundenes Steuerelement nicht unbedingt eine Datenbindung erfordern, damit es in einen Container eingefügt werden kann, der die Datenbindung nicht unterstützt. In einer solchen Situation kann die Funktionalität des Steuer Elements natürlich reduziert werden.
