# <a name="simple-data-binding"></a>einfache Datenbindung

Die ActiveX-Steuerelementarchitektur definiert einen Datenbindungsmechanismus, bei dem ein ActiveX-Steuerelement angeben kann, dass mindestens eine seiner Eigenschaften bindbar ist. In den meisten Fällen sollte ein datengebundenes Steuerelement nicht unbedingt eine Datenbindung erfordern, sodass es in einen Container eingefügt werden kann, der keine Datenbindung unterstützt. Natürlich kann in einer solchen Situation die Funktionalität des Steuerelements reduziert werden.
