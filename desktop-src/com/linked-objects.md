# <a name="linked-objects"></a>Verknüpfte Objekte

Wenn ein Link zu einem Objekt in einem Verbund Dokument eingefügt wird, befinden sich die Quelldaten oder die Verknüpfungs *Quelle* weiterhin an jedem Ort, an dem Sie ursprünglich erstellt wurde, in der Regel in einem anderen Dokument. Das Verbund Dokument enthält nur einen Verweis oder einen *Link* zu den tatsächlichen Daten, die in der Verknüpfungs Quelle gespeichert sind, sowie Informationen darüber, wie diese Daten dem Benutzer präsentiert werden.

Wenn Sie einen Link aktivieren, wird die Serveranwendung der Link Quelle ausgeführt, die der Benutzer benötigt, um die Linkdaten zu bearbeiten oder anderweitig zu bearbeiten. Durch die Verknüpfung wird die Größe eines zusammengesetzten Dokuments gering gehalten. Dies ist auch nützlich, wenn die Datenquelle von einer anderen Person verwaltet wird und von vielen Benutzern gemeinsam genutzt werden muss. Wenn die Person, die die Link Quelle verwaltet, die Daten ändert, wird die Änderung automatisch in allen Dokumenten aktualisiert, die einen Link zu diesen Daten enthalten. Neben dem Erstellen einfacher Verknüpfungen können Benutzer Verknüpfungen Schachteln und verknüpfte und eingebettete Objekte kombinieren, um komplexe Dokumente zu erstellen.
