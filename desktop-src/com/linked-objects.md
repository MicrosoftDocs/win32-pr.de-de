# <a name="linked-objects"></a>Verknüpfte Objekte

Wenn ein Link zu einem Objekt in ein Verbunddokument eingefügt wird, befinden sich die Quelldaten oder die Linkquelle weiterhin an dem Ort, an dem sie ursprünglich erstellt wurden, in der Regel in einem anderen Dokument. Das Verbunddokument enthält nur einen Verweis oder Link zu den tatsächlichen Daten, die in der Linkquelle gespeichert sind, sowie Informationen zur Präsentation dieser Daten für den Benutzer.

Wenn Sie einen Link aktivieren, wird die Serveranwendung der Linkquelle ausgeführt, die der Benutzer benötigt, um die Linkdaten zu bearbeiten oder anderweitig zu bearbeiten. Beim Verknüpfen wird die Größe eines zusammengesetzten Dokuments klein. Dies ist auch nützlich, wenn die Datenquelle von einer anderen Person verwaltet wird und von vielen Benutzern gemeinsam genutzt werden muss. Wenn die Person, die die Linkquelle verwalten, die Daten ändert, wird die Änderung automatisch in allen Dokumenten aktualisiert, die einen Link zu diesen Daten enthalten. Zusätzlich zum Erstellen einfacher Links können Benutzer Links schachteln und verknüpfte und eingebettete Objekte kombinieren, um komplexe Dokumente zu erstellen.
