# <a name="channel-com"></a>Channel (com)

Der Kanal ist für die Übertragung aller Nachrichten zwischen Client und Objekt über die Prozess Grenze verantwortlich. Der Kanal ist so konzipiert, dass er transparent mit verschiedenen Channeltypen arbeitet, mit dem OSF-DCE-Standard-RPC kompatibel ist und Single-und Multithreadanwendungen unterstützt.
