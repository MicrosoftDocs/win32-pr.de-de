# <a name="ws_group-and-ws_tabstop-flags-in-controls"></a>WS \_ -Gruppen-und WS \_ tabstoppflags in Steuerelementen

Ein Steuerelement sollte die WS \_ -Gruppe und die WS \_ tabstoppflags intern nicht verwenden. einige Container verwenden diese Flags, um die Tastatur Verarbeitung zu verwalten.
