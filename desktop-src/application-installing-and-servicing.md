# <a name="application-installation-and-servicing"></a>Anwendungsinstallation und -wartung

Nutzen Sie die verfügbaren APIs und Dienste, die von Windows, um Ihre Desktop-Apps zu installieren, zu verwalten und zu bedienen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema | Beschreibung |
| --- | --- |
| [Spieleexplorer](/previous-versions/windows/desktop/legacy/hh437965(v=vs.85)) | Beschreibt die Verwendung des Spiele-Explorers zum Anzeigen von Spielen auf Windows. |
| [Isolierte Anwendungen und parallele Assemblys](/windows/desktop/SbsCs/isolated-applications-and-side-by-side-assemblies-portal) | Isolierte Anwendungen und nebenseitige Assemblys ist eine Microsoft Windows-Lösung, die Versionskonflikte in Windows-Clientanwendungen reduziert. |
| [Verpacken, Bereitstellung und Abfragen von Windows-Apps](/windows/desktop/appxpkg/appx-portal) | Programmgesteuertes Erstellen Windows App-Paketen (einschließlich UWP- und Desktop-Apps) und Installieren, Aktualisieren, Abfragen und Deinstallieren von Apps. |
| [Neustart-Manager](/windows/desktop/RstMgr/restart-manager-portal) | Die Restart Manager-API kann die Anzahl von Systemneustarts, die zum Abschließen einer Installation oder eines Updates erforderlich sind, beseitigen oder verringern. |
| [Aktualisieren von Orchestrator](./updateorchestrator/index.md) | Mit der Orchestrator-API für Updates werden Ihre Softwareupdates im Hintergrund mit Blick auf die Benutzerfreundlichkeit geplant. | 
| [Windows Installer](/windows/desktop/Msi/windows-installer-portal) | Microsoft Windows Installer ist ein Installations- und Konfigurationsdienst, der mit Windows. Der Installerdienst ermöglicht Kunden eine bessere Unternehmensbereitstellung und stellt ein Standardformat für die Komponentenverwaltung zur Verfügung. Das Installationsprogramm ermöglicht auch die Ankündigung von Anwendungen und Features gemäß dem Betriebssystem. |
| [Windows Setup und Migration](/previous-versions/windows/desktop/wnf/windows-setup-and-migration-portal) | In diesem Abschnitt werden die Benachrichtigungen beschrieben, die verwendet werden, um eine Anwendung nach einem Setup oder einer Migration zu erkennen und ggf. zu reparieren. Diese Benachrichtigungen können auch verwendet werden, um Vorgänge während des flüchtigen Setups oder der Migration anzubesetzen. |
