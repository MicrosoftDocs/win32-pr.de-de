# <a name="application-installation-and-servicing"></a>Anwendungsinstallation und -wartung

Nutzen Sie die verfügbaren APIs und Dienste, die von Windows zur Installation, Verwaltung und Wartung Ihrer Desktop-Apps bereitgestellt werden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema | BESCHREIBUNG |
| --- | --- |
| [Spieleexplorer](/previous-versions/windows/desktop/legacy/hh437965(v=vs.85)) | Beschreibt die Verwendung von Games Explorer zum Anzeigen von spielen unter Windows. |
| [Isolierte Anwendungen und parallele Assemblys](/windows/desktop/SbsCs/isolated-applications-and-side-by-side-assemblies-portal) | Isolierte Anwendungen und parallele Assemblys sind eine Microsoft Windows-Lösung, mit der die Versions Konflikte in Windows-Client Anwendungen verringert werden. |
| [Verpacken, Bereitstellung und Abfrage von Windows-apps](/windows/desktop/appxpkg/appx-portal) | Programm gesteuertes Erstellen von Windows-App-Paketen (einschließlich UWP-und Desktop-Apps) und installieren, aktualisieren, Abfragen und Deinstallieren von apps. |
| [Neustart-Manager](/windows/desktop/RstMgr/restart-manager-portal) | Mit der API für den Neustart-Manager können Sie die Anzahl der Systemneustarts, die zum Durchführen einer Installation oder eines Updates erforderlich sind, eliminieren oder verringern. |
| [Aktualisieren von Orchestrator](./updateorchestrator/index.md) | Die Update Orchestrator-API plant Ihre Hintergrund Software Updates mit Benutzer Beeinträchtigung. | 
| [Windows Installer](/windows/desktop/Msi/windows-installer-portal) | Microsoft Windows Installer ist ein Installations-und Konfigurations Dienst, der mit Windows bereitgestellt wird. Der Installer-Dienst ermöglicht es Kunden, eine bessere Unternehmens Bereitstellung bereitzustellen und ein Standardformat für die Komponenten Verwaltung bereitzustellen. Das Installationsprogramm ermöglicht außerdem die Ankündigung von Anwendungen und Features gemäß dem Betriebssystem. |
| [Windows Setup und Migration](/previous-versions/windows/desktop/wnf/windows-setup-and-migration-portal) | In diesem Abschnitt werden die Benachrichtigungen beschrieben, mit denen eine Anwendung erkannt und möglicherweise repariert wird, nachdem eine Installation oder Migration aufgetreten ist. Diese Benachrichtigungen können auch verwendet werden, um Vorgänge während der flüchtigen Einrichtung oder Migration anzuhalten. |
