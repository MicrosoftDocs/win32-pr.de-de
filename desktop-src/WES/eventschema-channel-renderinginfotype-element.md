# <a name="channel-renderinginfotype-element"></a>Channel-Element (renderinginfotype)

Die gerenderte Meldungs Zeichenfolge des Kanals, der im Ereignis angegeben ist.

``` syntax
<xs:element name="Channel"
    type="string"
 />
```

Das **Channel** -Element wird durch den komplexen [**renderinginfotype**](eventschema-renderingtype-complextype.md) -Typ definiert.
