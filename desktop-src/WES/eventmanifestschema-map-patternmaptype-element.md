# <a name="map-patternmaptype-element"></a>MAP-Element (patternmaptype)

Definiert die regulären Ausdrücke, mit denen eine übereinstimmende Zeichenfolge in der Meldungs Zeichenfolge gesucht und geändert wird.

``` syntax
<xs:element name="map"
    type="PatternMapValueType"
 />
```

Das **map** -Element wird durch den komplexen Typ " [**patternmaptype**](eventmanifestschema-patternmaptype-complextype.md) " definiert.
