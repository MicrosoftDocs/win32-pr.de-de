# <a name="intypes-typelisttype-element"></a>Intypes (typelisttype)-Element

Enthält eine Liste von Eingabetyp Definitionen.

``` syntax
<xs:element name="inTypes"
    type="InputTypeListType"
 />
```

Das **Intypes** -Element wird durch den komplexen [**typelisttype**](eventmanifestschema-typelisttype-complextype.md) -Typ definiert.
