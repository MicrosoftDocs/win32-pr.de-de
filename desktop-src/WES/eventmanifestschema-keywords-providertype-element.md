# <a name="keywords-providertype-element"></a>Keywords-Element (ProviderType)

Definiert eine Liste von Schlüsselwörtern zum Kategorisieren von Ereignissen.

``` syntax
<xs:element name="keywords"
    type="KeywordListType"
 />
```

Das **Keywords** -Element wird durch den komplexen [**ProviderType**](eventmanifestschema-providertype-complextype.md) -Typ definiert.
