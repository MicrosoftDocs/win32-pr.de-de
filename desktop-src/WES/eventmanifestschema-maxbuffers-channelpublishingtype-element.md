# <a name="maxbuffers-channelpublishingtype-element"></a>MaxBuffers (channelpublishingtype)-Element

Die maximale Anzahl von Puffern, die für die Sitzung zuzuordnen sind.

``` syntax
<xs:element name="maxBuffers"
    type="unsignedInt"
 />
```

Das **MaxBuffers** -Element wird durch den komplexen [**channelpublishingtype**](eventmanifestschema-channelpublishingtype-complextype.md) -Typ definiert.
