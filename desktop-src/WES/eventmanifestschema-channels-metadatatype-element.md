# <a name="channels-metadatatype-element"></a>channels (MetadataType)-Element

Definiert eine Liste von Kanälen, für die Anbieter Ereignisse protokollieren können.

``` syntax
<xs:element name="channels"
    type="ChannelListType"
 />
```

Das **channels-Element** wird durch den komplexen [**MetadataType-Typ**](eventmanifestschema-metadatatype-complextype.md) definiert.
