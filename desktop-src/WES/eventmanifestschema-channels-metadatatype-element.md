# <a name="channels-metadatatype-element"></a>Channels (metadataType)-Element

Definiert eine Liste von Kanälen, zu denen Anbieter Ereignisse protokollieren können.

``` syntax
<xs:element name="channels"
    type="ChannelListType"
 />
```

Das **Channels** -Element wird durch den komplexen [**metadataType**](eventmanifestschema-metadatatype-complextype.md) -Typ definiert.
