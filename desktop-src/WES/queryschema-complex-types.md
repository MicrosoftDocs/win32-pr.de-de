# <a name="query-schema-complex-types"></a>Komplexe Typen von Abfrage Schemas

Im folgenden finden Sie die komplexen Typen, die das Abfrage Schema definiert.



| Komplexer Typ                                                   | BESCHREIBUNG                                                                                                                                                         |
|----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Querylisttype**](queryschema-querylisttype-complextype.md) | Definiert eine Liste von Abfragen, die eine Reihe von Selektoren und Unterdrückern enthalten können, mit denen Ereignisse in das Resultset eingeschlossen oder daraus ausgeschlossen werden. <br/> |
| [**QueryType**](queryschema-querytype-complextype.md)         | Definiert einen Satz von Auswahl-und Unterdrückungs Abfragen, die verwendet werden, um Ereignisse in das Resultset einzuschließen oder Ereignisse aus dem Resultset auszuschließen.<br/>                               |
