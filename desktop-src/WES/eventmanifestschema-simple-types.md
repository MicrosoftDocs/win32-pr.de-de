# <a name="eventmanifest-schema-simple-types"></a>Einfache Typen von eventmanifest-Schemas

Im folgenden finden Sie die einfachen Typen, die das eventmanifest-Schema definiert.



| Einfacher Typ                                                       | Beschreibung                                                                                                                                |
|-------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [**"Count Type"**](eventmanifestschema-counttype-simpletype.md)     | Definiert einen Zähltyp, der verwendet wird, um die Anzahl von Elementen in einem Array anzugeben.<br/>                                                |
| [**Csymboltype**](eventmanifestschema-csymboltype-simpletype.md) | Definiert einen gültigen C/C++-Symbolnamen.<br/>                                                                                              |
| [**FilePath**](eventmanifestschema-filepath-simpletype.md)       | Definiert eine Zeichenfolge, die einen voll qualifizierten Pfad zu einer Datei enthält.<br/>                                                                |
| [**Guidtype**](eventmanifestschema-guidtype-simpletype.md)       | Definiert einen Globally Unique Identifier Typ im Registrierungs Format.<br/>                                                                  |
| [**HexInt8Type**](eventmanifestschema-hex8type-simpletype.md)    | Definiert einen hexadezimalen 1-Byte-Typ.<br/>                                                                                              |
| [**HexInt16Type**](eventmanifestschema-hex16type-simpletype.md)  | Definiert einen 2-Byte-hexadezimal-Typ.<br/>                                                                                              |
| [**HexInt32Type**](eventmanifestschema-hex32type-simpletype.md)  | Definiert einen 4-Byte-hexadezimal-Typ.<br/>                                                                                              |
| [**HexInt64Type**](eventmanifestschema-hex64type-simpletype.md)  | Definiert einen 8-Byte-hexadezimal-Typ.<br/>                                                                                             |
| [**UInt8Type**](eventmanifestschema-hexint8type-simpletype.md)   | Definiert einen Byte-Typ ohne Vorzeichen.<br/>                                                                                                  |
| [**UInt16Type**](eventmanifestschema-hexint16type-simpletype.md) | Definiert einen Ganzzahl ohne Vorzeichen Short-Typ.<br/>                                                                                                 |
| [**UInt32Type**](eventmanifestschema-hexint32type-simpletype.md) | Definiert einen ganzzahligen Typ ohne Vorzeichen.<br/>                                                                                               |
| [**UInt64Type**](eventmanifestschema-hexint64type-simpletype.md) | Definiert einen Long-Typ ohne Vorzeichen.<br/>                                                                                                  |
| [**Längen-Typ**](eventmanifestschema-lengthtype-simpletype.md)   | Definiert einen length-Typ, der verwendet wird, um die Anzahl von Bytes oder Zeichen in einem Datenelement variabler Länge anzugeben.<br/>                 |
| [**Qnamelist**](eventmanifestschema-qnamelist-simpletype.md)     | Definiert eine Liste qualifizierter Namen.<br/>                                                                                              |
| [**"Strauch"**](eventmanifestschema-strtableref-simpletype.md) | Definiert eine Zeichenfolge, die auf eine Meldungs Zeichenfolge verweist, die in einer Zeichen folgen Tabelle im Manifest oder in einer Nachrichtendatei (. MC) definiert ist.<br/> |
