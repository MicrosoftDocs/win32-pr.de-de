# <a name="event-schema-elements"></a>Ereignis Schema Elemente

Im folgenden finden Sie die Elemente, die vom Ereignis Schema definiert werden. Dieser Abschnitt enthält die Namen der Elemente, die in einem protokollierten Ereignis gefunden werden. Informationen zu den einzelnen Elementen finden Sie jedoch unter dem komplexen Typ, der das Element enthält.



| Element                                                                                                    | BESCHREIBUNG                                                                                                                                                                                            |
|------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Binaryeventdata (EventType)**](eventschema-binaryeventdata-eventtype-element.md)                       | Enthält die Ereignisdaten als binäres Blob.<br/>                                                                                                                                                   |
| [**Binary (eventdatatype)**](eventschema-binary-eventdatatype-element.md)                                 | Ein binäres Daten-BLOB für Ereignisse, die mithilfe der [Ereignisprotokollierung](/windows/desktop/EventLog/event-logging)geschrieben werden.<br/>                                                                                                   |
| [**Channel (renderinginfotype)**](eventschema-channel-renderinginfotype-element.md)                       | Die gerenderte Meldungs Zeichenfolge des Kanals, der im Ereignis angegeben ist.<br/>                                                                                                                          |
| [**Channel (systempropertiestype)**](eventschema-channel-systempropertiestype-element.md)                 | Der Kanal, auf dem das Ereignis protokolliert wurde.<br/>                                                                                                                                                  |
| [**Complexdata (eventdatatype)**](eventschema-complexdata-eventdatatype-element.md)                       | Eine-Struktur, die in der Vorlage für das-Ereignis definiert ist.<br/>                                                                                                                                  |
| [**Komponente (debugdatatype)**](eventschema-component-debugdatatype-element.md)                           | Der Name der Komponente, die die Ablauf Verfolgungs Meldung protokolliert hat.<br/>                                                                                                                                    |
| [**Computer (systempropertiestype)**](eventschema-computer-systempropertiestype-element.md)               | Der Name des Computers, auf dem das Ereignis aufgetreten ist.<br/>                                                                                                                                       |
| [**Korrelation (systempropertiestype)**](eventschema-correlation-systempropertiestype-element.md)         | Die Aktivitäts Bezeichner, mit denen Consumer verwandte Ereignisse gruppieren können.<br/>                                                                                                           |
| [**Dataitemname (processingerrordatatype)**](eventschema-dataitemname-processingerrordatatype-element.md) | Enthält den Namen des Ereignisdaten Elements, das einen Fehler verursacht hat, als die Ereignisdaten verarbeitet wurden.<br/>                                                                                            |
| [**Daten (complexdatatype)**](eventschema-data-complexdatatype-element.md)                                 | Die Liste der Datenelemente in der-Struktur. Die Liste der Elemente wird in derselben Reihenfolge wie in der Vorlage definiert.<br/>                                                                                |
| [**Daten (eventdatatype)**](eventschema-data-eventdatatype-element.md)                                     | Ein Datenelement der obersten Ebene, das in der Vorlage für das-Ereignis definiert ist.<br/>                                                                                                                        |
| [**DebugData (EventType)**](eventschema-debugdata-eventtype-element.md)                                   | Enthält die Daten, die für Windows-Software-Ablaufverfolgungs-präprozessorereignisse (WPP) protokolliert werden können.<br/>                                                                                                  |
| [**ErrorCode (processingerrordatatype)**](eventschema-errorcode-processingerrordatatype-element.md)       | Enthält den Fehlercode, der ausgelöst wurde, als bei der Verarbeitung von Ereignisdaten ein Fehler aufgetreten ist. <br/>                                                                                                     |
| [**Ereignis**](eventschema-event-element.md)                                                                 | Dies ist der Stamm Knoten der gerenderten Ereignisdaten.<br/>                                                                                                                                           |
| [**EVENTDATA (EventType)**](eventschema-eventdata-eventtype-element.md)                                   | Enthält die Ereignisdaten.<br/>                                                                                                                                                                    |
| [**EventID (systempropertiestype)**](eventschema-eventid-systempropertiestype-element.md)                 | Der Bezeichner, den der Anbieter verwendet hat, um das Ereignis zu identifizieren.<br/>                                                                                                                                |
| [**EventPayload (processingerrordatatype)**](eventschema-eventpayload-processingerrordatatype-element.md) | Enthält binäre Ereignisdaten für das Ereignis, das einen Fehler verursacht hat, als die Ereignisdaten verarbeitet wurden. <br/>                                                                                           |
| [**Eventrecordid (systempropertiestype)**](eventschema-eventrecordid-systempropertiestype-element.md)     | Die Datensatznummer, die dem Ereignis bei der Protokollierung zugewiesen wurde.<br/>                                                                                                                                 |
| [**Ausführung (systempropertiestype)**](eventschema-execution-systempropertiestype-element.md)             | Enthält Informationen über den Prozess und den Thread, der das Ereignis protokolliert hat.<br/>                                                                                                                    |
| [**FileLine (Debug DataType)**](eventschema-fileline-debugdatatype-element.md)                             | Der Name der Quelldatei und die Zeile in der Quelldatei, die die Ablauf Verfolgungs Meldung protokolliert hat.<br/>                                                                                              |
| [**FlagName (Debug DataType)**](eventschema-flagname-debugdatatype-element.md)                             | Der Flagwert, der beim Aktivieren an den Anbieter übermittelt wurde.<br/>                                                                                                                                  |
| [**Function (Debug DataType)**](eventschema-function-debugdatatype-element.md)                             | Der Name der Funktion, die die Ablauf Verfolgungs Meldung protokolliert hat.<br/>                                                                                                                                     |
| [**Schlüsselwort (Schlüsselwörter)**](eventschema-keyword-keywords-element.md)                                         | Enthält ein gerendertes Schlüsselwort.<br/>                                                                                                                                                                |
| [**Schlüsselwörter (renderinginfotype)**](eventschema-keywords-renderingtype-element.md)                         | Eine Liste von gerenderten Schlüsselwörtern.<br/>                                                                                                                                                                |
| [**Schlüsselwörter (systempropertiestype)**](eventschema-keywords-systempropertiestype-element.md)               | Eine Bitmaske der Schlüsselwörter, die im Ereignis definiert sind.<br/>                                                                                                                                             |
| [**LevelName (Debug DataType)**](eventschema-levelname-debugdatatype-element.md)                           | Der bei der Aktivierung an den Anbieter über gegebene levelwert.<br/>                                                                                                                                 |
| [**Level (renderinginfotype)**](eventschema-level-renderingtype-element.md)                               | Die gerenderte Meldungs Zeichenfolge der im Ereignis angegebenen Ebene.<br/>                                                                                                                            |
| [**Level (systempropertiestype)**](eventschema-level-systempropertiestype-element.md)                     | Enthält den Schweregrad des Ereignisses.<br/>                                                                                                                                                   |
| [**Message (debugdatatype)**](eventschema-message-debugdatatype-element.md)                               | Die Meldungszeichenfolge. Der XML-Code enthält dieses Element, wenn das Feld "formattedstring" vom WPP-Ereignis angegeben wurde.<br/>                                                                                     |
| [**Message (renderinginfotype)**](eventschema-message-renderingtype-element.md)                           | Enthält die Ereignismeldung, die für das-Ereignis gerendert wird.<br/>                                                                                                                                  |
| [**Opcode (renderinginfotype)**](eventschema-opcode-renderingtype-element.md)                             | Die gerenderte Meldungs Zeichenfolge des im Ereignis angegebenen OpCodes.<br/>                                                                                                                           |
| [**Opcode (systempropertiestype)**](eventschema-opcode-systempropertiestype-element.md)                   | Der im Ereignis definierte Opcode.<br/>                                                                                                                                                            |
| [**Processingerrordata (EventType)**](eventschema-processingerrordata-eventtype-element.md)               | Enthält Details zu dem Fehler, der beim Versuch aufgetreten ist, das Ereignis zu erzeugen.<br/>                                                                                                               |
| [**Anbieter (systempropertiestype)**](eventschema-provider-systempropertiestype-element.md)               | Identifiziert den Anbieter, der das Ereignis protokolliert hat.<br/>                                                                                                                                              |
| [**Anbieter (renderinginfotype)**](eventschema-publisher-renderinginfotype-element.md)                    | Die gerenderte Nachrichten Zeichenfolge für den Anbieter.<br/>                                                                                                                                               |
| [**Renderinginfo (EventType)**](eventschema-renderinginfo-eventtype-element.md)                           | Enthält die gerenderten Meldungs Zeichenfolgen für das Ereignis (einschließlich der Meldungs Zeichenfolge des Ereignisses und der Meldungs Zeichenfolgen für alle Eigenschaften des Ereignisses, z. b. Ebene, Aufgabe und OpCode).<br/>        |
| [**Sicherheit (systempropertiestype)**](eventschema-security-systempropertiestype-element.md)               | Identifiziert den Benutzer, der das Ereignis protokolliert hat.<br/>                                                                                                                                                  |
| [**Sequencenumschlag (Debug DataType)**](eventschema-sequencenumber-debugdatatype-element.md)                 | Die lokale oder globale Sequenznummer der Ablauf Verfolgungs Meldung.<br/>                                                                                                                                   |
| [**Subcomponent (debugdatatype)**](eventschema-subcomponent-debugdatatype-element.md)                     | Gibt das für das Debuggen von Ereignissen in debugkanälen verwendete subcomponentname WPP-Debug-Ablauf Verfolgungs Feld an.                                                                                                 |
| [**System (EventType)**](eventschema-system-eventtype-element.md)                                         | Enthält Informationen, die den Anbieter und die Art der Aktivierung, das Ereignis, den Kanal, in den das Ereignis geschrieben wurde, und Systeminformationen, wie z. b. den Prozess und die Thread-IDs, identifiziert.<br/> |
| [**Aufgabe (renderinginfotype)**](eventschema-task-renderingtype-element.md)                                 | Die gerenderte Meldungs Zeichenfolge der im Ereignis angegebenen Aufgabe.<br/>                                                                                                                             |
| [**Aufgabe (systempropertiestype)**](eventschema-task-systempropertiestype-element.md)                       | Die im Ereignis definierte Aufgabe.<br/>                                                                                                                                                              |
| [**TimeCreated (systempropertiestype)**](eventschema-timecreated-systempropertiestype-element.md)         | Der Zeitstempel, der angibt, wann das Ereignis protokolliert wurde.<br/>                                                                                                                                   |
| [**UserData (EventType)**](eventschema-userdata-eventtype-element.md)                                     | Enthält die Ereignisdaten.<br/>                                                                                                                                                                    |
| [**Version (systempropertiestype)**](schema-version-systempropertiestype-element.md)                      | Enthält die Versionsnummer der Ereignis Definition.<br/>                                                                                                                                      |
