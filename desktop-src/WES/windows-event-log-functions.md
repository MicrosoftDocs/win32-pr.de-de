# <a name="windows-event-log-functions"></a>Windows-Ereignisprotokoll Funktionen

Das Windows-Ereignisprotokoll definiert die folgenden Funktionen, die Sie verwenden können, um Ereignisse aus einem Kanal-oder Ereignisprotokoll zu erhalten und die Metadaten für einen Anbieter und die generierten Ereignisse zu erhalten.



| Funktion                                                                   | BESCHREIBUNG                                                                                                                                                                                                       |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**EVT- \_ Abonnement \_ Rückruf**](/windows/win32/api/winevt/nc-winevt-evt_subscribe_callback)                 | Implementieren Sie diesen Rückruf, wenn Sie die [**evtsubscribe**](/windows/desktop/api/WinEvt/nf-winevt-evtsubscribe) -Funktion zum Empfangen von Ereignissen, die Ihrer Abfrage entsprechen, aufruft.                                                                                    |
| [**Evtarchiveexportedlog**](/windows/desktop/api/WinEvt/nf-winevt-evtarchiveexportedlog)                     | Fügt den Ereignissen in der angegebenen Protokolldatei lokalisierte Zeichen folgen hinzu.                                                                                                                                                   |
| [**Ereignis Abbruch**](/windows/desktop/api/WinEvt/nf-winevt-evtcancel)                                             | Bricht alle ausstehenden Vorgänge für ein Handle ab.                                                                                                                                                                       |
| [**Evtclearlog**](/windows/desktop/api/WinEvt/nf-winevt-evtclearlog)                                         | Entfernt alle Ereignisse aus dem angegebenen Kanal und schreibt Sie in die Ziel Protokolldatei.                                                                                                                             |
| [**Evtclose**](/windows/desktop/api/WinEvt/nf-winevt-evtclose)                                               | Schließt ein geöffnetes handle.                                                                                                                                                                                            |
| [**Evtkreatebookmark**](/windows/desktop/api/WinEvt/nf-winevt-evtcreatebookmark)                             | Erstellt ein Lesezeichen, das ein Ereignis in einem Kanal identifiziert.                                                                                                                                                         |
| [**Evtkreaterendercontext**](/windows/desktop/api/WinEvt/nf-winevt-evtcreaterendercontext)                   | Erstellt einen Kontext, der die Informationen im Ereignis angibt, das Sie wiedergeben möchten.                                                                                                                            |
| [**Evtexportlog**](/windows/desktop/api/WinEvt/nf-winevt-evtexportlog)                                       | Kopiert Ereignisse aus dem angegebenen Kanal oder der angegebenen Protokolldatei und schreibt Sie in die Ziel Protokolldatei.                                                                                                                      |
| [**Evtformatmessage**](/windows/desktop/api/WinEvt/nf-winevt-evtformatmessage)                               | Formatiert eine Nachrichten Zeichenfolge.                                                                                                                                                                                         |
| [**Evtgetchannelconfigproperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgetchannelconfigproperty)         | Ruft die angegebene kanalkonfigurations Eigenschaft ab.                                                                                                                                                                |
| [**Evtgeteventinfo**](/windows/desktop/api/WinEvt/nf-winevt-evtgeteventinfo)                                 | Ruft Informationen ab, die die strukturierte XML-Abfrage, die das Ereignis ausgewählt hat, und den Kanal oder die Protokolldatei, aus der es stammt, identifiziert.                                                                                 |
| [**EvtGetEventMetadataProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgeteventmetadataproperty)         | Ruft die angegebene ereignismetadateneigenschaft ab.                                                                                                                                                                       |
| [**Evtgetextendedstatus**](/windows/desktop/api/WinEvt/nf-winevt-evtgetextendedstatus)                       | Ruft eine Text Meldung ab, die die erweiterten Fehlerinformationen für den aktuellen Fehler enthält.                                                                                                                           |
| [**Evtgetloginfo**](/windows/desktop/api/WinEvt/nf-winevt-evtgetloginfo)                                     | Ruft Informationen zu einem Kanal oder einer Protokolldatei ab.                                                                                                                                                                     |
| [**Evtgetobjectarrayproperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgetobjectarrayproperty)             | Ruft eine anbietemetadateneigenschaft aus dem angegebenen-Objekt im-Array ab.                                                                                                                                         |
| [**Evtgetobjectarraysize**](/windows/desktop/api/WinEvt/nf-winevt-evtgetobjectarraysize)                     | Ruft die Anzahl der Elemente im Array von-Objekten ab.                                                                                                                                                              |
| [**EvtGetPublisherMetadataProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgetpublishermetadataproperty) | Ruft die angegebene anbietenmetadateneigenschaft ab.                                                                                                                                                                    |
| [**Evtgetqueryinfo**](/windows/desktop/api/WinEvt/nf-winevt-evtgetqueryinfo)                                 | Ruft Informationen zu einer ausgeführten Abfrage ab, die die Liste der Kanäle oder Protokolldateien identifiziert, auf die die Abfrage zugegriffen hat, sowie eine Liste der Rückgabecodes, die angibt, ob die einzelnen Zugriffe erfolgreich waren oder fehlgeschlagen sind. |
| [**EvtNext**](/windows/desktop/api/WinEvt/nf-winevt-evtnext)                                                 | Ruft das nächste Ereignis aus den Ergebnissen der Abfrage oder des Abonnements ab.                                                                                                                                                       |
| [**Evtnextchannelpath**](/windows/desktop/api/WinEvt/nf-winevt-evtnextchannelpath)                           | Ruft einen Kanalnamen aus dem Enumerator ab.                                                                                                                                                                          |
| [**Evtnexteventmetadata**](/windows/desktop/api/WinEvt/nf-winevt-evtnexteventmetadata)                       | Ruft eine Ereignis Definition aus dem Enumerator ab.                                                                                                                                                                     |
| [**Evtnextpublisherid**](/windows/desktop/api/WinEvt/nf-winevt-evtnextpublisherid)                           | Ruft den Bezeichner eines Anbieters aus dem Enumerator ab.                                                                                                                                                            |
| [**Evtopenchannelconfig**](/windows/desktop/api/WinEvt/nf-winevt-evtopenchannelconfig)                       | Ruft ein Handle ab, das Sie verwenden, um die Konfigurations Eigenschaft eines Kanals zu lesen oder zu ändern.                                                                                                                                  |
| [**Evtopenchannelumum**](/windows/desktop/api/WinEvt/nf-winevt-evtopenchannelenum)                           | Ruft ein Handle ab, mit dem Sie die Liste der auf dem Computer registrierten Kanäle auflisten.                                                                                                                 |
| [**EvtOpenEventMetadataEnum**](/windows/desktop/api/WinEvt/nf-winevt-evtopeneventmetadataenum)               | Ruft ein Handle ab, das Sie zum Auflisten der Liste der vom Anbieter definierten Ereignisse verwenden.                                                                                                                             |
| [**Evtoperlog**](/windows/desktop/api/WinEvt/nf-winevt-evtopenlog)                                           | Ruft ein Handle für eine Kanal-oder Protokolldatei ab, die Sie verwenden können, um Informationen über den Kanal oder die Protokolldatei zu erhalten.                                                                                                    |
| [**Evtopumpublishererum**](/windows/desktop/api/WinEvt/nf-winevt-evtopenpublisherenum)                       | Ruft ein Handle ab, das Sie zum Auflisten der Liste der registrierten Anbieter auf dem Computer verwenden.                                                                                                                         |
| [**Evtopumpublishermetadata**](/windows/desktop/api/WinEvt/nf-winevt-evtopenpublishermetadata)               | Ruft ein Handle ab, das zum Lesen der Metadaten des angegebenen Anbieters verwendet wird.                                                                                                                                             |
| [**Evtopensession**](/windows/desktop/api/WinEvt/nf-winevt-evtopensession)                                   | Stellt eine Verbindung mit einem Remote Computer her, den Sie verwenden können, wenn Sie die anderen Windows-Ereignisprotokoll Funktionen aufrufen.                                                                                                |
| [**Evtquery**](/windows/desktop/api/WinEvt/nf-winevt-evtquery)                                               | Führt eine Abfrage zum Abrufen von Ereignissen aus einer Kanal-oder Protokolldatei aus, die den angegebenen Abfrage Kriterien entsprechen.                                                                                                               |
| [**EvtRender**](/windows/desktop/api/WinEvt/nf-winevt-evtrender)                                             | Rendert ein XML-Fragment auf der Grundlage des von Ihnen angegebenen renderingkontexts.                                                                                                                                          |
| [**Evtsavechannelconfig**](/windows/desktop/api/WinEvt/nf-winevt-evtsavechannelconfig)                       | Speichert die Änderungen, die an der Konfiguration eines Kanals vorgenommen wurden.                                                                                                                                                              |
| [**Evtseek**](/windows/desktop/api/WinEvt/nf-winevt-evtseek)                                                 | Sucht ein bestimmtes Ereignis in einem Abfrageresultset.                                                                                                                                                                  |
| [**Evtsetchannelconfigproperty**](/windows/desktop/api/WinEvt/nf-winevt-evtsetchannelconfigproperty)         | Legt die angegebene Konfigurations Eigenschaft eines Kanals fest.                                                                                                                                                           |
| [**Event abonnieren**](/windows/desktop/api/WinEvt/nf-winevt-evtsubscribe)                                       | Erstellt ein Abonnement, das aktuelle und zukünftige Ereignisse aus einer Kanal-oder Protokolldatei empfängt, die den angegebenen Abfrage Kriterien entsprechen.                                                                            |
| [**Evtupdatebookmark**](/windows/desktop/api/WinEvt/nf-winevt-evtupdatebookmark)                             | Aktualisiert das Lesezeichen mit Informationen, die das angegebene Ereignis identifizieren.                                                                                                                                        |
