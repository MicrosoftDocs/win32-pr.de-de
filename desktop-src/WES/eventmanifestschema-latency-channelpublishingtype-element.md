# <a name="latency-channelpublishingtype-element"></a>Latenz (channelpublishingtype)-Element

Die Wartezeit in Millisekunden, bevor die Puffer geleert werden.

``` syntax
<xs:element name="latency"
    type="unsignedInt"
 />
```

Das **Latency** -Element wird durch den komplexen Typ [**channelpublishingtype**](eventmanifestschema-channelpublishingtype-complextype.md) definiert.
