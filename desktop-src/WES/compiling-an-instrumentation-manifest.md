# <a name="compiling-an-instrumentation-manifest"></a>Kompilieren eines Instrumentierungs Manifests

Verwenden Sie nach dem [Schreiben des Manifests](writing-an-instrumentation-manifest.md)den Nachrichten Compiler, um das Manifest zu validieren und die Ressourcen-und Header Dateien zu generieren, die Sie in Ihren Anbieter einschließen. Ausführliche Informationen zur Verwendung des Compilers finden Sie unter [**Message Compiler**](message-compiler--mc-exe-.md). Schließen Sie die Header-und Ressourcen Dateien ein, die der Compiler in dem Projekt generiert, das den Rest des Quellcodes des Anbieters enthält. Informationen zum Schreiben von Ereignissen, die im Manifest definiert sind, finden Sie unter [Entwickeln eines Anbieters](developing-a-provider.md).
