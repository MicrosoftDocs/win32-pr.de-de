# <a name="compiling-an-instrumentation-manifest"></a>Kompilieren eines Instrumentierungsmanifests

Verwenden [Sie nach dem Schreiben des Manifests](writing-an-instrumentation-manifest.md)den Nachrichtencompiler, um das Manifest zu überprüfen und die Ressourcen- und Headerdateien zu generieren, die Sie in Ihren Anbieter ein schließen. Weitere Informationen zur Verwendung des Compilers finden Sie unter [**Message Compiler**](message-compiler--mc-exe-.md). Schließen Sie die Header- und Ressourcendateien, die der Compiler generiert, in das Projekt ein, das den rest des Quellcodes Ihres Anbieters enthält. Informationen zum Schreiben der Ereignisse, die in Ihrem Manifest definiert sind, finden Sie unter [Entwickeln eines Anbieters.](developing-a-provider.md)
