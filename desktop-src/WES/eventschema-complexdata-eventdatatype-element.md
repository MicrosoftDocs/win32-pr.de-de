# <a name="complexdata-eventdatatype-element"></a>Complexdata (eventdatatype)-Element

Eine-Struktur, die in der Vorlage für das-Ereignis definiert ist.

``` syntax
<xs:element name="ComplexData"
    type="ComplexDataType"
 />
```

Das **complexdata** -Element wird durch den komplexen [**eventdatatype**](eventschema-eventdatatype-complextype.md) -Typ definiert.
