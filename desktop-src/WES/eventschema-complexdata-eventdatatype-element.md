# <a name="complexdata-eventdatatype-element"></a>ComplexData-Element (EventDataType)

Eine -Struktur, die in der Vorlage für das Ereignis definiert ist.

``` syntax
<xs:element name="ComplexData"
    type="ComplexDataType"
 />
```

Das **ComplexData-Element** wird durch den komplexen [**EventDataType-Typ**](eventschema-eventdatatype-complextype.md) definiert.
