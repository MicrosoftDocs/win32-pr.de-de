# <a name="opcodes-tasktype-element"></a>Opcodes (TaskType)-Element

Definiert eine Liste von aufgabenspezifischen OpCodes.

``` syntax
<xs:element name="opcodes"
    type="OpcodeListType"
 />
```

Das **Opcodes** -Element wird durch den komplexen [**TaskType**](eventmanifestschema-tasktype-complextype.md) -Typ definiert.
