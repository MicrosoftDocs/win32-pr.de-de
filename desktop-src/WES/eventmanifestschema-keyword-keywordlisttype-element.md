# <a name="keyword-keywordlisttype-element"></a>Schlüsselwort (keywordlisttype)-Element

Definiert ein Schlüsselwort, das eine Kategorie von Ereignissen bezeichnet. Ein Schlüsselwort ist ein Tag, das Sie an ein Ereignis anfügen, um Ereignisse auf der Grundlage ihrer Verwendung zu gruppieren.

``` syntax
<xs:element name="keyword"
    type="KeywordType"
 />
```

Das **Schlüsselwort** Element wird durch den komplexen Typ [**keywordlisttype**](eventmanifestschema-keywordlisttype-complextype.md) definiert.
