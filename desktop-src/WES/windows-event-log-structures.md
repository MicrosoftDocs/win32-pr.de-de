# <a name="windows-event-log-structures"></a>Windows-Ereignisprotokoll Strukturen

Das Windows-Ereignisprotokoll definiert die folgenden Strukturen.



| Struktur                                | BESCHREIBUNG                                                    |
|------------------------------------------|----------------------------------------------------------------|
| [**EVT- \_ RPC- \_ Anmeldung**](/windows/desktop/api/WinEvt/ns-winevt-evt_rpc_login) | Enthält die Informationen, die zum Herstellen einer Verbindung mit einem Remote Computer verwendet werden. |
| [**EVT- \_ Variante**](/windows/desktop/api/WinEvt/ns-winevt-evt_variant)      | Enthält Ereignisdaten oder Eigenschaftswerte.                        |
