# <a name="windows-event-log-structures"></a>Windows Ereignisprotokollstrukturen

Windows Das Ereignisprotokoll definiert die folgenden Strukturen.



| Struktur                                | BESCHREIBUNG                                                    |
|------------------------------------------|----------------------------------------------------------------|
| [**EVT \_ RPC \_ LOGIN**](/windows/desktop/api/WinEvt/ns-winevt-evt_rpc_login) | Enthält die Informationen, die zum Herstellen einer Verbindung mit einem Remotecomputer verwendet werden. |
| [**EVT \_ VARIANT**](/windows/desktop/api/WinEvt/ns-winevt-evt_variant)      | Enthält Ereignisdaten oder Eigenschaftswerte.                        |
