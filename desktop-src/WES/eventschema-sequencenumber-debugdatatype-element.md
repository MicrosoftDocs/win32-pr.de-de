Die lokale oder globale Sequenznummer der Ablauf Verfolgungs Meldung.

``` syntax
<xs:element name="SequenceNumber"
    type="unsignedInt"
    minOccurs="0"
 />
```

Das **sequencenumschlag** -Element wird durch den komplexen [**debugdatatype**](eventschema-debugdatatype-complextype.md) -Typ definiert.
