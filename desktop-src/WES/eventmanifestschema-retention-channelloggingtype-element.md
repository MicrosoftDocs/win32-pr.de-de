# <a name="retention-channelloggingtype-element"></a>Retention (channelloggingtype)-Element

Bestimmt, ob die Protokolldatei eine sequenzielle oder zirkuläre Protokolldatei ist.

``` syntax
<xs:element name="retention"
    type="boolean"
 />
```

Das **Beibehaltungs** Element wird durch den komplexen Typ [**channelloggingtype**](eventmanifestschema-channelloggingtype-complextype.md) definiert.
