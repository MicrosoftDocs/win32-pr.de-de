# <a name="stringtable-resources-element"></a>STRINGTABLE (Resources)-Element

Definiert eine Liste lokalisierter Zeichen folgen, auf die Sie im Manifest verweisen können.

``` syntax
<xs:element name="stringTable"
    type="StringTableType"
 />
```

Das **STRINGTABLE** -Element wird durch das [**Resources**](eventmanifestschema-resources-localizationtype-element.md) -Element definiert.
