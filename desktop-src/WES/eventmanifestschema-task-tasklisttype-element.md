# <a name="task-tasklisttype-element"></a>task (TaskListType)-Element

Definiert eine Komponente oder Unterkomponente einer Anwendung.

``` syntax

    type="TaskType"
 />
```

Das **Taskelement** wird durch den komplexen [**TaskListType-Typ**](eventmanifestschema-tasklisttype-complextype.md) definiert.
