# <a name="task-tasklisttype-element"></a>Task (tasklisttype)-Element

Definiert eine Komponente oder eine Unterkomponente einer Anwendung.

``` syntax
<xs:element name="task"
    type="TaskType"
 />
```

Das **Task** -Element wird durch den komplexen [**tasklisttype**](eventmanifestschema-tasklisttype-complextype.md) -Typ definiert.
