# <a name="map-bitmaptype-element"></a>MAP-Element (bitmaptype)

Definiert die Zuordnung zwischen einem Bitwert und einem Zeichen folgen Wert.

``` syntax
<xs:element name="map"
    type="BitMapValueType"
 />
```

Das **map** -Element wird durch den komplexen [**bitmaptype**](eventmanifestschema-bitmaptype-complextype.md) -Typ definiert.
