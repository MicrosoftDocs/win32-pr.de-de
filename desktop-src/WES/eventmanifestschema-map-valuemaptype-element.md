# <a name="map-valuemaptype-element"></a>Map (valuemaptype)-Element

Definiert die Zuordnung zwischen einem ganzzahligen Wert und einem Zeichen folgen Wert.

``` syntax
<xs:element name="map"
    type="ValueMapValueType"
 />
```

Das **map** -Element wird durch den komplexen Typ [**valuemaptype**](eventmanifestschema-valuemaptype-complextype.md) definiert.
