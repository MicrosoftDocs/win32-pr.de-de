# <a name="map-valuemaptype-element"></a>map(ValueMapType)-Element

Definiert die Zuordnung zwischen einem ganzzahligen Wert und einem Zeichenfolgenwert.

``` syntax
<xs:element name="map"
    type="ValueMapValueType"
 />
```

Das **Map-Element** wird durch den komplexen [**ValueMapType-Typ**](eventmanifestschema-valuemaptype-complextype.md) definiert.
