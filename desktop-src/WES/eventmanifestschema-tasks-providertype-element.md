# <a name="tasks-providertype-element"></a>Tasks (ProviderType)-Element

Definiert eine Liste von Aufgaben, die ein Anbieter zum Gruppieren von Ereignissen verwenden kann.

``` syntax
<xs:element name="tasks"
    type="TaskListType"
 />
```

Das **Tasks** -Element wird durch den komplexen [**ProviderType**](eventmanifestschema-providertype-complextype.md) -Typ definiert.
