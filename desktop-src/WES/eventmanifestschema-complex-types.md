# <a name="eventmanifest-schema-complex-types"></a>Komplexe EventManifest-Schematypen

Im Folgenden sind die komplexen Typen angegeben, die vom EventManifest-Schema definiert werden.



| Komplexer Typ                                                                                       | BESCHREIBUNG                                                                                                                                                                                                                             |
|----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BitMapType**](eventmanifestschema-bitmaptype-complextype.md)                                   | Definiert eine Liste von Name-Wert-Zuordnungen zwischen Bitwerten und Zeichenfolgenwerten.<br/>                                                                                                                                                  |
| [**BitMapValueType**](eventmanifestschema-bitmapvaluetype-complextype.md)                         | Definiert die Zuordnung zwischen einem Bitwert und einem Zeichenfolgenwert.<br/>                                                                                                                                                                  |
| [**ChannelListType**](eventmanifestschema-channellisttype-complextype.md)                         | Definiert eine Liste von Kanälen, in denen Anbieter Ereignisse protokollieren können.<br/>                                                                                                                                                                |
| [**ChannelLoggingType**](eventmanifestschema-channelloggingtype-complextype.md)                   | Definiert die Eigenschaften der Protokolldatei, die den Kanal sichern, z. B. die Kapazität und ob sie sequenziell oder kreisförmig ist.<br/>                                                                                                |
| [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md)             | Definiert die Protokollierungseigenschaften für die Sitzung, die der Kanal verwendet.<br/>                                                                                                                                                        |
| [**Channeltype**](eventmanifestschema-channeltype-complextype.md)                                 | Definiert einen Kanal, an den Anbieter Ereignisse protokollieren können.<br/>                                                                                                                                                                         |
| [**DataDefinitionType**](eventmanifestschema-datadefinitiontype-complextype.md)                   | Definiert ein Datenelement, das Sie in das Ereignis ein-/ausdingen möchten.<br/>                                                                                                                                                                 |
| [**DefinitionType**](eventmanifestschema-definitiontype-complextype.md)                           | Definiert eine Liste von Ereignissen, die Ihr Anbieter protokollieren kann.<br/>                                                                                                                                                                         |
| [**EventDefinitionType**](eventmanifestschema-eventdefinitiontype-complextype.md)                 | Definiert ein Ereignis, das ihr Anbieter schreiben kann.<br/>                                                                                                                                                                               |
| [**EventsType**](eventmanifestschema-eventstype-complextype.md)                                   | Enthält eine Liste der Anbieter, die im Manifest definiert sind.<br/>                                                                                                                                                               |
| [**FilterType**](eventmanifestschema-filtertype-complextype.md)                                   | Definiert einen Datenfilter, mit dem eine Sitzung Ereignisse basierend auf Ereignisdaten filtert.<br/>                                                                                                                                              |
| [**FilterListType**](eventmanifestschema-filterlisttype-complextype.md)                           | Definiert eine Liste von Filtern, die ein ETW-Controller an Ihren Anbieter übergeben kann, um die von ihm zu schreibenden Ereignisse weiter zu begrenzen.<br/>                                                                                                       |
| [**ImportChannelType**](eventmanifestschema-importchanneltype-complextype.md)                     | Identifiziert einen Kanal, der von einem anderen Anbieter oder in einem Manifest definiert wurde, das einen Metadatenabschnitt enthält.<br/>                                                                                                            |
| [**InputType**](eventmanifestschema-inputtype-complextype.md)                                     | Definiert einen Eingabedatentyp.<br/>                                                                                                                                                                                                  |
| [**InputTypeListType**](eventmanifestschema-inputtypelisttype-complextype.md)                     | Definiert eine Liste von Eingabetypen.<br/>                                                                                                                                                                                               |
| [**InstrumentationManifestType**](eventmanifestschema-instrumentationmanifesttype-complextype.md) | Definiert den Basistyp für das [**instrumentationManifest-Element.**](eventmanifestschema-instrumentationmanifest-element.md)<br/>                                                                                                |
| [**Instrumentationtype**](eventmanifestschema-instrumentationtype-complextype.md)                 | Definiert den Inhalt des Instrumentierungsabschnitts des Manifests.<br/>                                                                                                                                                         |
| [**KeywordListType**](eventmanifestschema-keywordlisttype-complextype.md)                         | Definiert eine Liste von Schlüsselwörtern, die Ereignisse kategorisieren.<br/>                                                                                                                                                                           |
| [**KeywordType**](eventmanifestschema-keywordtype-complextype.md)                                 | Definiert ein Schlüsselwort, das eine Kategorie von Ereignissen identifiziert.<br/>                                                                                                                                                                      |
| [**LevelListType**](eventmanifestschema-levellisttype-complextype.md)                             | Definiert eine Liste von Schweregraden, die die Ausführlichkeit eines Ereignisses angeben.<br/>                                                                                                                                                    |
| [**LevelType**](eventmanifestschema-leveltype-complextype.md)                                     | Definiert einen Schweregrad, der die Ausführlichkeit der protokollierten Ereignisse bestimmt.<br/>                                                                                                                                           |
| [**LocalizationType**](eventmanifestschema-localizationtype-complextype.md)                       | Definiert eine Gruppe lokalisierter Ressourcen, auf die Sie in Ihrem Manifest verweisen.<br/>                                                                                                                                                  |
| [**MapType**](eventmanifestschema-maptype-complextype.md)                                         | Definiert eine Liste von Name-Wert-Paaren.<br/>                                                                                                                                                                                          |
| [**MetadataType**](eventmanifestschema-metadatatype-complextype.md)                               | Definiert die Metadatentypen, die Sie im Metadatenabschnitt des Manifests definieren können.<br/>                                                                                                                                      |
| [**NamedQueryType**](eventmanifestschema-namedquerytype-complextype.md)                           | Definiert eine Liste benannter Abfragen, die die Ereignismeldungszeichenfolge nach einem Wert abfragen und eine angegebene Aktion ausführen, wenn sie gefunden wird.<br/>                                                                                                     |
| [**OpcodeListType**](eventmanifestschema-opcodelisttype-complextype.md)                           | Definiert eine Liste von Opcodes, die verwendet werden, um die Vorgänge einer Komponente der Anwendung zu identifizieren.<br/>                                                                                                                        |
| [**OpcodeType**](eventmanifestschema-opcodetype-complextype.md)                                   | Definiert einen Vorgang innerhalb einer Komponente der Anwendung.<br/>                                                                                                                                                                  |
| [**Outputtype**](eventmanifestschema-outputtype-complextype.md)                                   | Definiert einen Ausgabedatentyp, der bestimmt, wie die Daten gerendert werden.<br/>                                                                                                                                                        |
| [**PatternMapListType**](eventmanifestschema-patternmaplisttype-complextype.md)                   | Definiert eine Liste regulärer Ausdruckspaare, die zum Ändern der Meldungszeichenfolge verwendet werden.<br/>                                                                                                                                        |
| [**PatternMapType**](eventmanifestschema-patternmaptype-complextype.md)                           | Definiert eine Zuordnung zwischen zwei regulären Ausdrücken. Ein Ausdruck wird verwendet, um eine übereinstimmende Zeichenfolge in der Meldungszeichenfolge zu finden, und der andere wird verwendet, um die Zeichenfolge zu ändern, bevor der Dienst sie wieder in die Meldungszeichenfolge eingibt.<br/> |
| [**PatternMapValueType**](eventmanifestschema-patternmapvaluetype-complextype.md)                 | Definiert die regulären Ausdrücke, die verwendet werden, um eine übereinstimmende Zeichenfolge in der Meldungszeichenfolge zu suchen und zu ändern.<br/>                                                                                                                           |
| [**ProviderType**](eventmanifestschema-providertype-complextype.md)                               | Definiert einen Anbieter und die Metadaten, die er zum Definieren seiner Ereignisse verwendet.<br/>                                                                                                                                                       |
| [**StringTableType**](eventmanifestschema-stringtabletype-complextype.md)                         | Definiert eine Liste lokalisierter Zeichenfolgen, auf die Sie in Ihrem Manifest verweisen können.<br/>                                                                                                                                                 |
| [**StructDefinitionType**](eventmanifestschema-structdefinitiontype-complextype.md)               | Definiert eine -Struktur, die ein oder mehrere Datenelemente enthält, die In das Ereignis eingeschlossen werden soll.<br/>                                                                                                                            |
| [**TaskEventDefinitionType**](eventmanifestschema-taskeventdefinitiontype-complextype.md)         | Definiert ein aufgabenspezifisches Ereignis, das ihr Anbieter protokollieren kann.<br/>                                                                                                                                                                    |
| [**TaskListType**](eventmanifestschema-tasklisttype-complextype.md)                               | Definiert eine Liste von Aufgaben, die zum Identifizieren der Komponenten einer Anwendung verwendet werden.<br/>                                                                                                                                          |
| [**TaskType**](eventmanifestschema-tasktype-complextype.md)                                       | Definiert eine Komponente oder Unterkomponente einer Anwendung.<br/>                                                                                                                                                                       |
| [**TemplateItemType**](eventmanifestschema-templateitemtype-complextype.md)                       | Eine Vorlage, die die Daten definiert, die mit einem Ereignis enthalten sein müssen.<br/>                                                                                                                                                                   |
| [**TemplateListType**](eventmanifestschema-templatelisttype-complextype.md)                       | Definiert eine Liste von Vorlagen, die die Daten angeben, die in die Ereignisse aufgenommen werden.<br/>                                                                                                                                                |
| [**TypeListType**](eventmanifestschema-typelisttype-complextype.md)                               | Definiert die Typen, die im Manifest verwendet werden.<br/>                                                                                                                                                                             |
| [**ValueMapType**](eventmanifestschema-valuemaptype-complextype.md)                               | Definiert eine Liste von Name-Wert-Zuordnungen zwischen ganzzahligen Werten und Zeichenfolgenwerten.<br/>                                                                                                                                              |
| [**ValueMapValueType**](eventmanifestschema-valuemapvaluetype-complextype.md)                     | Definiert die Zuordnung zwischen einem ganzzahligen Wert und einem Zeichenfolgenwert.<br/>                                                                                                                                                             |
| [**Xmltype**](eventmanifestschema-xmltype-complextype.md)                                         | Definiert ein XML-Fragment.<br/>                                                                                                                                                                                                     |
| [**XmlTypeListType**](eventmanifestschema-xmltypelisttype-complextype.md)                         | Definiert listenausgabetypen, die der Dienst verwendet, um zu bestimmen, wie ein Eingabedatentyp gerendert wird.<br/>                                                                                                                             |
