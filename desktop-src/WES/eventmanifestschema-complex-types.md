# <a name="eventmanifest-schema-complex-types"></a>Komplexe Typen von eventmanifest-Schemas

Im folgenden finden Sie die komplexen Typen, die das eventmanifest-Schema definiert.



| Komplexer Typ                                                                                       | BESCHREIBUNG                                                                                                                                                                                                                             |
|----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Bitmaptype**](eventmanifestschema-bitmaptype-complextype.md)                                   | Definiert eine Liste von Name-Wert-Zuordnungen zwischen Bitwerten und Zeichen folgen Werten.<br/>                                                                                                                                                  |
| [**Bitmapvaluetype**](eventmanifestschema-bitmapvaluetype-complextype.md)                         | Definiert die Zuordnung zwischen einem Bitwert und einem Zeichen folgen Wert.<br/>                                                                                                                                                                  |
| [**Channelellisttype**](eventmanifestschema-channellisttype-complextype.md)                         | Definiert eine Liste von Kanälen, zu denen Anbieter Ereignisse protokollieren können.<br/>                                                                                                                                                                |
| [**Channelloggingtype**](eventmanifestschema-channelloggingtype-complextype.md)                   | Definiert die Eigenschaften der Protokolldatei, die den Kanal sichert, z. b. die Kapazität und ob Sie sequenziell oder zirkulär ist.<br/>                                                                                                |
| [**Channelpublishingtype**](eventmanifestschema-channelpublishingtype-complextype.md)             | Definiert die Protokollierungs Eigenschaften für die Sitzung, die vom Kanal verwendet wird.<br/>                                                                                                                                                        |
| [**ChannelType**](eventmanifestschema-channeltype-complextype.md)                                 | Definiert einen Kanal, zu dem Anbieter Ereignisse protokollieren können.<br/>                                                                                                                                                                         |
| [**DataDefinitionType**](eventmanifestschema-datadefinitiontype-complextype.md)                   | Definiert ein Datenelement, das Sie mit dem-Ereignis einschließen möchten.<br/>                                                                                                                                                                 |
| [**DefinitionType**](eventmanifestschema-definitiontype-complextype.md)                           | Definiert eine Liste von Ereignissen, die von Ihrem Anbieter protokolliert werden können.<br/>                                                                                                                                                                         |
| [**EventDefinitionType**](eventmanifestschema-eventdefinitiontype-complextype.md)                 | Definiert ein Ereignis, das von Ihrem Anbieter geschrieben werden kann.<br/>                                                                                                                                                                               |
| [**Eventstype**](eventmanifestschema-eventstype-complextype.md)                                   | Enthält eine Liste von Anbietern, die im Manifest definiert sind.<br/>                                                                                                                                                               |
| [**Filter Type**](eventmanifestschema-filtertype-complextype.md)                                   | Definiert einen Daten Filter, den eine Sitzung verwendet, um Ereignisse auf der Grundlage von Ereignisdaten zu filtern.<br/>                                                                                                                                              |
| [**Filterlisttype**](eventmanifestschema-filterlisttype-complextype.md)                           | Definiert eine Liste von Filtern, die von einem etw-Controller an Ihren Anbieter übergeben werden können, um die zu schreibenden Ereignisse weiter einzuschränken.<br/>                                                                                                       |
| [**Importchanneltype**](eventmanifestschema-importchanneltype-complextype.md)                     | Identifiziert einen Kanal, der von einem anderen Anbieter oder einem Manifest definiert wurde, das einen Metadatenabschnitt enthält.<br/>                                                                                                            |
| [**InputType**](eventmanifestschema-inputtype-complextype.md)                                     | Definiert einen Eingabe Datentyp.<br/>                                                                                                                                                                                                  |
| [**Inputtypelisttype**](eventmanifestschema-inputtypelisttype-complextype.md)                     | Definiert eine Liste von Eingabetypen.<br/>                                                                                                                                                                                               |
| [**InstrumentationManifestType**](eventmanifestschema-instrumentationmanifesttype-complextype.md) | Definiert den Basistyp für das [**instrumentationmanifest**](eventmanifestschema-instrumentationmanifest-element.md) -Element.<br/>                                                                                                |
| [**InstrumentationType**](eventmanifestschema-instrumentationtype-complextype.md)                 | Definiert den Inhalt des Instrumentations Abschnitts des Manifests.<br/>                                                                                                                                                         |
| [**Keywordlisttype**](eventmanifestschema-keywordlisttype-complextype.md)                         | Definiert eine Liste von Schlüsselwörtern zum Kategorisieren von Ereignissen.<br/>                                                                                                                                                                           |
| [**Keywordtype**](eventmanifestschema-keywordtype-complextype.md)                                 | Definiert ein Schlüsselwort, das eine Kategorie von Ereignissen bezeichnet.<br/>                                                                                                                                                                      |
| [**Levellisttype**](eventmanifestschema-levellisttype-complextype.md)                             | Definiert eine Liste mit Schweregraden, die den ausführlichkeits Grad eines Ereignisses angeben.<br/>                                                                                                                                                    |
| [**LevelType**](eventmanifestschema-leveltype-complextype.md)                                     | Definiert einen Schweregrad Wert, der die Ausführlichkeit der protokollierten Ereignisse bestimmt.<br/>                                                                                                                                           |
| [**Localizationtype**](eventmanifestschema-localizationtype-complextype.md)                       | Definiert eine Gruppe lokalisierter Ressourcen, auf die im Manifest verwiesen wird.<br/>                                                                                                                                                  |
| [**MapType**](eventmanifestschema-maptype-complextype.md)                                         | Definiert eine Liste von Name-Wert-Paaren.<br/>                                                                                                                                                                                          |
| [**MetadataType**](eventmanifestschema-metadatatype-complextype.md)                               | Definiert die Metadatentypen, die Sie im Metadatenabschnitt des Manifests definieren können.<br/>                                                                                                                                      |
| [**Namedquerytype**](eventmanifestschema-namedquerytype-complextype.md)                           | Definiert eine Liste benannter Abfragen, die die Ereignis Meldungs Zeichenfolge nach einem Wert Abfragen und eine angegebene Aktion ausführen, sofern gefunden.<br/>                                                                                                     |
| [**Opcodelisttype**](eventmanifestschema-opcodelisttype-complextype.md)                           | Definiert eine Liste von Opcodes, die verwendet werden, um die Vorgänge einer Komponente der Anwendung zu identifizieren.<br/>                                                                                                                        |
| [**OpCodeType**](eventmanifestschema-opcodetype-complextype.md)                                   | Definiert einen Vorgang innerhalb einer Komponente der Anwendung.<br/>                                                                                                                                                                  |
| [**OutputType**](eventmanifestschema-outputtype-complextype.md)                                   | Definiert einen Ausgabedatentyp, der bestimmt, wie die Daten gerendert werden.<br/>                                                                                                                                                        |
| [**Patternmaplisttype**](eventmanifestschema-patternmaplisttype-complextype.md)                   | Definiert eine Liste von Paaren für reguläre Ausdrücke, die zum Ändern der Meldungs Zeichenfolge verwendet werden.<br/>                                                                                                                                        |
| [**Patternmaptype**](eventmanifestschema-patternmaptype-complextype.md)                           | Definiert eine Zuordnung zwischen zwei regulären Ausdrücken. Ein Ausdruck wird verwendet, um nach einer übereinstimmenden Zeichenfolge in der Meldungs Zeichenfolge zu suchen, und die andere wird verwendet, um die Zeichenfolge zu ändern<br/> |
| [**Patternmapvaluetype**](eventmanifestschema-patternmapvaluetype-complextype.md)                 | Definiert die regulären Ausdrücke, mit denen eine übereinstimmende Zeichenfolge in der Meldungs Zeichenfolge gesucht und geändert wird.<br/>                                                                                                                           |
| [**ProviderType**](eventmanifestschema-providertype-complextype.md)                               | Definiert einen Anbieter und die Metadaten, die er verwendet, um seine Ereignisse zu definieren.<br/>                                                                                                                                                       |
| [**Stringtabletype**](eventmanifestschema-stringtabletype-complextype.md)                         | Definiert eine Liste lokalisierter Zeichen folgen, auf die Sie im Manifest verweisen können.<br/>                                                                                                                                                 |
| [**StructDefinitionType**](eventmanifestschema-structdefinitiontype-complextype.md)               | Definiert eine-Struktur, die ein oder mehrere Datenelemente enthält, die Sie mit dem-Ereignis einschließen möchten.<br/>                                                                                                                            |
| [**TaskEventDefinitionType**](eventmanifestschema-taskeventdefinitiontype-complextype.md)         | Definiert ein Aufgaben spezifisches Ereignis, das von Ihrem Anbieter protokolliert werden kann.<br/>                                                                                                                                                                    |
| [**Tasklisttype**](eventmanifestschema-tasklisttype-complextype.md)                               | Definiert eine Liste von Aufgaben, die zum Identifizieren der Komponenten einer Anwendung verwendet werden.<br/>                                                                                                                                          |
| [**TaskType**](eventmanifestschema-tasktype-complextype.md)                                       | Definiert eine Komponente oder eine Unterkomponente einer Anwendung.<br/>                                                                                                                                                                       |
| [**Templateitemtype**](eventmanifestschema-templateitemtype-complextype.md)                       | Eine Vorlage, die die Daten definiert, die in einem Ereignis enthalten sein sollen.<br/>                                                                                                                                                                   |
| [**Templatelisttype**](eventmanifestschema-templatelisttype-complextype.md)                       | Definiert eine Liste von Vorlagen, die die Daten angeben, die in die Ereignisse eingeschlossen werden sollen.<br/>                                                                                                                                                |
| [**Typelisttype**](eventmanifestschema-typelisttype-complextype.md)                               | Definiert die Typen, die im Manifest verwendet werden.<br/>                                                                                                                                                                             |
| [**Valuemaptype**](eventmanifestschema-valuemaptype-complextype.md)                               | Definiert eine Liste von Name-Wert-Zuordnungen zwischen ganzzahligen Werten und Zeichen folgen Werten.<br/>                                                                                                                                              |
| [**Valuemapvaluetype**](eventmanifestschema-valuemapvaluetype-complextype.md)                     | Definiert die Zuordnung zwischen einem ganzzahligen Wert und einem Zeichen folgen Wert.<br/>                                                                                                                                                             |
| [**XmlType**](eventmanifestschema-xmltype-complextype.md)                                         | Definiert ein XML-Fragment.<br/>                                                                                                                                                                                                     |
| [**Xmltypelisttype**](eventmanifestschema-xmltypelisttype-complextype.md)                         | Definiert einen Listen Ausgabetyp, den der Dienst verwendet, um zu bestimmen, wie ein Eingabe Datentyp dargestellt werden soll.<br/>                                                                                                                             |
