# <a name="templates-providertype-element"></a>Templates (ProviderType)-Element

Definiert eine Liste von Vorlagen, die die Daten angeben, die in die Ereignisse eingeschlossen werden sollen.

``` syntax
<xs:element name="templates"
    type="TemplateListType"
 />
```

Das **Templates** -Element wird durch den komplexen [**ProviderType**](eventmanifestschema-providertype-complextype.md) -Typ definiert.
