# <a name="stringtable-localizationtype-element"></a>stringTable -Element (LocalizationType)

Definiert eine Liste lokalisierter Zeichenfolgen, auf die Sie im Manifest verweisen können.

``` syntax
<xs:element name="stringTable"
    type="StringTableType"
 />
```

Das **stringTable-Element** wird durch den komplexen [**LocalizationType-Typ**](eventmanifestschema-localizationtype-complextype.md) definiert.
