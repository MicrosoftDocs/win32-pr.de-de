# <a name="event-schema-complex-types"></a>Komplexe Typen von Ereignis Schemas

Im folgenden finden Sie die komplexen Typen, die vom Ereignis Schema definiert werden.



| Komplexer Typ                                                                       | BESCHREIBUNG                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Complexdatatype**](eventschema-complexdatatype-complextype.md)                 | Definiert eine-Struktur, die ein oder mehrere Datenelemente enthält.<br/>                                                                                                                                      |
| [**DataType**](eventschema-datafieldtype-complextype.md)                          | Definiert ein Datenelement.<br/>                                                                                                                                                                           |
| [**Debug DataType**](eventschema-debugdatatype-complextype.md)                     | Definiert die Daten, die für Windows-Software-Ablaufverfolgungs-präprozessorereignisse (WPP) protokolliert werden können.<br/>                                                                                                      |
| [**Eventdatatype**](eventschema-eventdatatype-complextype.md)                     | Definiert die Ereignisdaten Elemente und-Strukturen, die die Ereignisdaten enthalten.<br/>                                                                                                                      |
| [**EventType**](eventschema-eventtype-complextype.md)                             | Definiert den Stamm Knoten des Ereignis Schemas.<br/>                                                                                                                                                     |
| [**Processingerrordatatype**](eventschema-processingerrordatatype-complextype.md) | Definiert einen Fehler, der beim Rendern der Ereignisdaten für das Ereignis aufgetreten ist.<br/>                                                                                                                   |
| [**Renderinginfotype**](eventschema-renderingtype-complextype.md)                 | Definiert die gerenderten Nachrichten für das Ereignis.<br/>                                                                                                                                                   |
| [**Systempropertiestype**](eventschema-systempropertiestype-complextype.md)       | Definiert die Informationen, die den Anbieter identifizieren und wie er aktiviert wurde, das Ereignis, den Kanal, in den das Ereignis geschrieben wurde, und Systeminformationen, wie z. b. die Prozess-und Thread-IDs.<br/> |
| [**Userdatatype**](eventschema-userdatatype-complextype.md)                       | Definiert ein XML-Fragment, das das Layout der Ereignisdaten angibt.<br/>                                                                                                                           |
