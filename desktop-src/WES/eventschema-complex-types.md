# <a name="event-schema-complex-types"></a>Komplexe Ereignisschematypen

Im Folgenden sind die komplexen Typen angegeben, die vom Ereignisschema definiert werden.



| Komplexer Typ                                                                       | Beschreibung                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ComplexDataType**](eventschema-complexdatatype-complextype.md)                 | Definiert eine -Struktur, die ein oder mehrere Datenelemente enthält.<br/>                                                                                                                                      |
| [**Datatype**](eventschema-datafieldtype-complextype.md)                          | Definiert ein Datenelement.<br/>                                                                                                                                                                           |
| [**DebugDataType**](eventschema-debugdatatype-complextype.md)                     | Definiert die Daten, die für Windows WPP-Ereignisse (Software Trace Preprocessor) protokolliert werden können.<br/>                                                                                                      |
| [**EventDataType**](eventschema-eventdatatype-complextype.md)                     | Definiert die Ereignisdatenelemente und -strukturen, die die Ereignisdaten enthalten.<br/>                                                                                                                      |
| [**Eventtype**](eventschema-eventtype-complextype.md)                             | Definiert den Stammknoten des Ereignisschemas.<br/>                                                                                                                                                     |
| [**ProcessingErrorDataType**](eventschema-processingerrordatatype-complextype.md) | Definiert einen Fehler, der beim Rendern der Ereignisdaten für das Ereignis aufgetreten ist.<br/>                                                                                                                   |
| [**RenderingInfoType**](eventschema-renderingtype-complextype.md)                 | Definiert die gerenderten Nachrichten für das Ereignis.<br/>                                                                                                                                                   |
| [**SystemPropertiesType**](eventschema-systempropertiestype-complextype.md)       | Definiert die Informationen, die den Anbieter identifizieren und wie er aktiviert wurde, das Ereignis, den Kanal, in den das Ereignis geschrieben wurde, und Systeminformationen wie prozess- und thread-IDs.<br/> |
| [**UserDataType**](eventschema-userdatatype-complextype.md)                       | Definiert ein XML-Fragment, das das Layout der Ereignisdaten angibt.<br/>                                                                                                                           |
