# <a name="using-event-logging"></a>Verwenden der Ereignisprotokollierung

Anwendungen können mithilfe der Ereignisprotokollierungs-API Ereignisse melden und anzeigen. Weitere Informationen zum Melden von Ereignissen finden Sie unter [Reporting Events](reporting-an-event.md).

Informationen zum Anzeigen von Ereignissen, die in Protokolldateien geschrieben wurden, finden Sie unter Abfragen von [Ereignisquellennachrichten](querying-for-event-source-messages.md) und [Empfangen von Ereignisbenachrichtigungen.](receiving-event-notification.md) Sie können auch die -Ereignisanzeige zum Anzeigen von Ereignissen verwenden.
