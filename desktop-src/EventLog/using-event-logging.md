# <a name="using-event-logging"></a>Verwenden der Ereignisprotokollierung

Anwendungen können die Ereignis Protokollierungs-API verwenden, um Ereignisse zu melden und anzuzeigen. Ausführliche Informationen zu Berichterstattungs Ereignissen finden Sie unter Bericht Erstellungs [Ereignisse](reporting-an-event.md).

Informationen zum Anzeigen von Ereignissen, die in Protokolldateien geschrieben wurden, finden Sie unter [Abfragen von Ereignis Quell Nachrichten](querying-for-event-source-messages.md) und [empfangen der Ereignis Benachrichtigung](receiving-event-notification.md). Sie können die Ereignisanzeige auch zum Anzeigen von Ereignissen verwenden.
