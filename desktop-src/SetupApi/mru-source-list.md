# <a name="mru-source-list"></a>MRU-Quell Liste

Die Setup-API stellt Funktionen bereit, die die zuletzt verwendeten (MRU) Quellverzeichnisse speichern. Diese Informationen werden im System des Benutzers gespeichert, und bei nachfolgenden Installationen kann darauf zugegriffen werden.

In den folgenden Themen wird die Liste der MRU-Quellen beschrieben und erläutert, wie Sie mit den Setup Funktionen geöffnet, geändert, überprüft und geschlossen werden:

-   [Informationen zur MRU-Quell Liste](about-the-mru-source-list.md)
-   [Verwenden der MRU-Quell Liste](using-the-mru-source-list.md)
-   [Referenz zur MRU-Quell Liste](mru-source-list-reference.md)
