# <a name="file-log-functions"></a>Datei Protokollfunktionen

Die folgenden Funktionen werden mit Datei Protokollen verwendet.



| Funktion                                                   | BESCHREIBUNG                                 |
|------------------------------------------------------------|---------------------------------------------|
| [**Setupinitializefilelog**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitializefileloga)   | Initialisiert eine Protokolldatei für die Verwendung.             |
| [**SetupLog file**](/windows/desktop/api/Setupapi/nf-setupapi-setuplogfilea)                       | Fügt der Protokolldatei einen Eintrag hinzu.              |
| [**Setupqueryfilelog**](/windows/desktop/api/Setupapi/nf-setupapi-setupqueryfileloga)             | Ruft Informationen aus einer Protokolldatei ab.      |
| [**Setupremuvefilelogentry**](/windows/desktop/api/Setupapi/nf-setupapi-setupremovefilelogentrya) | Entfernt einen Eintrag aus einer Protokolldatei.           |
| [**Setupterminatefilelog**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog)     | Gibt die einer Protokolldatei zugeordneten Ressourcen frei. |
