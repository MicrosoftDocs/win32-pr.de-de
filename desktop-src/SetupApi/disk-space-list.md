# <a name="disk-space-list"></a>Disk-Space Liste

Die Setup-API bietet Funktionen für die Speicherplatz Liste. Eine Speicherplatz Liste ist eine Liste von Datei Vorgängen, die Sie Abfragen können, um den für diese Vorgänge erforderlichen Speicherplatz zu ermitteln.

In den folgenden Themen werden die Speicherplatz Listen detailliert beschrieben. Außerdem wird erläutert, wie Sie den erforderlichen Speicherplatz berechnen und wie Sie in einer Setup Anwendung verwendet werden:

-   [Informationen zur Disk-Space Liste](about-the-disk-space-list.md)
-   [Verwenden der Disk-Space Liste](using-the-disk-space-list.md)
-   [Referenz zur Speicherplatz Liste](disk-space-list-reference.md)
