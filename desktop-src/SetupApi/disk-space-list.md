# <a name="disk-space-list"></a>Disk-Space Liste

Die Setup-API stellt Funktionen für die Speicherplatzliste bereit. Eine Datenträgerspeicherplatzliste ist eine Liste von Dateivorgängen, die Sie abfragen können, um den erforderlichen Speicherplatz für diese Vorgänge zu bestimmen.

In den folgenden Themen werden Die Speicherplatzlisten im Detail beschrieben, wie sie den erforderlichen Speicherplatz berechnen und wie sie in einer Setupanwendung verwendet werden:

-   [Informationen zur Disk-Space Liste](about-the-disk-space-list.md)
-   [Verwenden der Disk-Space Liste](using-the-disk-space-list.md)
-   [Referenz zur Datenträgerspeicherplatzliste](disk-space-list-reference.md)
