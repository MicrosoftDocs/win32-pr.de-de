# <a name="using-cabinet-files"></a>Verwenden von CAB-Dateien

Die Setup Funktionen behandeln Schränke intern. Zum expliziten auflisten und Extrahieren von Dateien aus einer CAB-Datei können Sie die [**setupiteratecabinet**](/windows/desktop/api/Setupapi/nf-setupapi-setupiteratecabineta) -Funktion aufrufen.

Im folgenden Thema wird die interne CAB-Verarbeitung der Setup Funktionen und die Verwendung von [**setupiteratecabinet**](/windows/desktop/api/Setupapi/nf-setupapi-setupiteratecabineta)beschrieben. Außerdem werden die von **setupiteratecabinet** gesendeten Benachrichtigungen und das erforderliche Format einer CAB-Rückruf Routine zum Verarbeiten dieser Benachrichtigungen erläutert.

-   [Extrahieren von Dateien aus kabinatendateien](extracting-files-from-cabinets.md)
-   [Erstellen einer CAB-Rückruf Routine](creating-a-cabinet-callback-routine.md)
