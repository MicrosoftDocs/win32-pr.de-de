# <a name="default-queue-callback-routine-functions"></a>Standardfunktionen für Warteschlangen Rückruf Routinen

Die folgenden Funktionen werden mit der standardmäßigen Warteschlangen Rückruf Routine verwendet.



| Funktion                                                                   | BESCHREIBUNG                                                                                                                                                                            |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Setupdefaultqueuecallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupdefaultqueuecallbacka)             | Die standardmäßige Warteschlangen Rückruf-Routine.                                                                                                                                                    |
| [**SetupInitDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallback)     | Initialisiert Kontextinformationen, die von der Standard Warteschlangen-Rückruf Routine benötigt werden.                                                                                                          |
| [**Setupinitdefaultqueuecallbackex**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallbackex) | Initialisiert Kontextinformationen, die von der Standard Warteschlangen Rückruf Routine benötigt werden, und gibt ein alternatives Fenster zum Anzeigen von Statusmeldungen an.                                           |
| [**Setuptermdefaultqueuecallback**](/windows/desktop/api/Setupapi/nf-setupapi-setuptermdefaultqueuecallback)     | Gibt die von [**SetupInitDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallback) und [**setupinitdefaultqueuecallbackex**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallbackex)zugeordneten Ressourcen frei. |
