# <a name="using-inf-files"></a>Verwenden von INF-Dateien

Nachdem Sie eine INF-Datei für den Installationsvorgang erstellt haben, können die Setup Funktionen auf diese Informationen zugreifen, um Dateien zu kopieren, zu löschen oder umzubenennen und um INIs und Registrierungseinträge zu aktualisieren.

In den folgenden Themen wird beschrieben, wie Sie die Setup Funktionen zum Abrufen von Informationen aus INF-Dateien und zum Durchführen von Installations Vorgängen durch Verweisen auf eine INF-Datei verwenden:

-   [Erstellen einer INF-Datei](authoring-an-inf-file.md)
-   [Öffnen und Schließen einer INF-Datei](opening-and-closing-an-inf-file.md)
-   [Abrufen von Informationen aus einer INF-Datei](retrieving-information-from-an-inf-file.md)
-   [Installieren aus einer INF-Datei](installing-from-an-inf-file.md)
-   [Angeben einer Sicherheits Beschreibung aus einer INF-Datei](specifying-a-security-descriptor-from-an-inf-file.md)
