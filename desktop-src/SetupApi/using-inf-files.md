# <a name="using-inf-files"></a>Verwenden von INF-Dateien

Nachdem Sie eine INF-Datei für den Installationsvorgang erstellt haben, können die Setupfunktionen auf diese Informationen zugreifen, die beim Kopieren, Löschen oder Umbenennen von Dateien und Aktualisieren von INIs und Registrierungseinträgen verwendet werden.

In den folgenden Themen wird beschrieben, wie Sie die Setupfunktionen verwenden, um Informationen aus INF-Dateien abzurufen, und wie Sie Installationsvorgänge ausführen, indem Sie auf eine INF-Datei verweisen:

-   [Erstellen einer INF-Datei](authoring-an-inf-file.md)
-   [Öffnen und Schließen einer INF-Datei](opening-and-closing-an-inf-file.md)
-   [Abrufen von Informationen aus einer INF-Datei](retrieving-information-from-an-inf-file.md)
-   [Installieren aus einer INF-Datei](installing-from-an-inf-file.md)
-   [Angeben eines Sicherheitsdeskriptors aus einer INF-Datei](specifying-a-security-descriptor-from-an-inf-file.md)
