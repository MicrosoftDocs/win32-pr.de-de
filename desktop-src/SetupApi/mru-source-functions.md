# <a name="mru-source-functions"></a>MRU-Quell Funktionen

Die folgenden Funktionen werden mit MRU-Quelldatei Listen verwendet.



| Funktion                                                                 | BESCHREIBUNG                                                                                       |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**Setupaddin SourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupaddtosourcelista)                     | Fügt einer Quell Liste einen Eintrag hinzu.                                                                   |
| [**Setupcanceltemporarysourcelist**](/windows/desktop/api/Setupapi/nf-setupapi-setupcanceltemporarysourcelist) | Bricht alle temporären Listen-oder No-Browse-Verhaltensweisen ab und stellt das Standard Listen Verhalten wieder her.        |
| [**Setupfreesourcelist**](/windows/desktop/api/Setupapi/nf-setupapi-setupfreesourcelista)                       | Gibt die einer Quell Liste zugeordneten Ressourcen frei.                                                       |
| [**Setupquerysourcelist**](/windows/desktop/api/Setupapi/nf-setupapi-setupquerysourcelista)                     | Fragt die aktuelle Liste der Installations Quellen ab.                                                 |
| [**Setupremuvefromsourcelist**](/windows/desktop/api/Setupapi/nf-setupapi-setupremovefromsourcelista)           | Entfernt einen Eintrag aus einer Installations Quell Liste.                                                |
| [**Setupsetsourcelist**](/windows/desktop/api/Setupapi/nf-setupapi-setupsetsourcelista)                         | Legt die Liste der Installations Quellen auf die System-MRU-Liste, die MRU-Liste des Benutzers oder eine temporäre Liste fest. |
