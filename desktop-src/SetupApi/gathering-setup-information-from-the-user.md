# <a name="gathering-setup-information-from-the-user"></a>Sammeln von Setupinformationen vom Benutzer

Es wird dringend empfohlen, dass Ihre Setupanwendung eine Benutzeroberfläche enthält, die alle erforderlichen Informationen vom Benutzer erhält. Diese Funktionalität wird nicht von den Setupfunktionen bereitgestellt und muss von Ihrer Setupanwendung implementiert werden, in der Regel in Form eines Installations-Assistenten.
