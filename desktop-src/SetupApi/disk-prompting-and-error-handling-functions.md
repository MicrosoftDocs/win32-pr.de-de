# <a name="disk-prompting-and-error-handling-functions"></a>Datenträger Aufforderung und Fehler Behandlungs Funktionen

Die folgenden Funktionen stellen Datenträger Aufforderungen und eine Benutzeroberfläche zur Fehlerbehandlung bereit.



| Funktion                                         | BESCHREIBUNG                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------|
| [**Setupcopyerror**](/windows/desktop/api/Setupapi/nf-setupapi-setupcopyerrora)         | Generiert ein Dialogfeld, in dem der Benutzer über einen Kopierfehler informiert wird.                          |
| [**Setupdeleteerror**](/windows/desktop/api/Setupapi/nf-setupapi-setupdeleteerrora)     | Generiert ein Dialogfeld, in dem der Benutzer über einen Lösch Fehler informiert wird.                      |
| [**Setuppromptfordisk**](/windows/desktop/api/Setupapi/nf-setupapi-setuppromptfordiska) | Generiert ein Dialogfeld, das den Benutzer zur Eingabe eines Medien Datenträgers oder Quelldatei Speicher Orts auffordert. |
| [**Setuprenameerror**](/windows/desktop/api/Setupapi/nf-setupapi-setuprenameerrora)     | Generiert ein Dialogfeld, in dem der Benutzer über einen Umbenennungs Fehler informiert wird.                        |
