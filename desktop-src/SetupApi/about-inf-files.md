# <a name="about-inf-files"></a>Informationen zu INF-Dateien

Eine INF-Datei ist eine Textdatei, die in verschiedene Typen formatierter Abschnitte unterteilt ist. Jeder Abschnitt ist für einen bestimmten Zweck konzipiert. beispielsweise zum Kopieren von Dateien oder zum Hinzufügen von Einträgen zur Registrierung. Anzahl und Typ der Abschnitte in einer INF-Datei hängen von der jeweiligen Installationsprozedur ab. Jeder Abschnitt besteht aus Zeilen, die zum Zweck dieses Abschnitts beitragen.

INF-Dateien speichern Informationen in einem Format, das in den Abschnitten *"Allgemeine Richtlinien für INF-Datei"* und *"INF-Datei" und "Direktiven"* des Microsoft Windows 2000 Driver Development Kit beschrieben wird. Sie sollten auf diese Ressourcen im Microsoft Windows Driver Development Kit (DDK) verweisen, wenn Sie INF-Dateien erstellen möchten. Die Setupfunktionen verwenden dieses Format, um Informationen aus der INF-Datei abzurufen.

Die folgenden INF-Abschnitte können mit den Setupfunktionen verwendet werden, um eine Installationsanwendung zu erstellen. Wenn Sie eine INF-Datei erstellen, müssen Sie nicht jeden Abschnittstyp einschließen. Die verwendeten Abschnitte hängen vom Installationsvorgang ab.



| `Section`                          | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                      |
|----------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \[**Version**\]                  | Enthält Informationen zur INF-Datei. Dies ist der einzige erforderliche Abschnitt. Die Setupfunktionen verwenden diesen Abschnitt, um zu erkennen, für welche Version von Windows die INF-Datei vorgesehen war.                                                                                                                                                                                                                                    |
| \[**Installieren**\]                  | Bietet eine Übersicht über den Installationsvorgang und enthält eine Liste der INF-Abschnitte, die während der Installation von den Setupfunktionen verarbeitet werden. Weitere Informationen finden Sie im [Inf-Installationsabschnitt – Beispiel.](inf-install-section-example.md)                                                                                                                                                                                                           |
|     \[**Hinzufügen einer Registrierung**\]         | Enthält Informationen, die zum Hinzufügen von Unterschlüsseln oder Wertnamen zur Registrierung verwendet werden.                                                                                                                                                                                                                                                                                                                                                         |
|     \[**Kopieren von Dateien**\]           | Enthält Quelldateinamen und kann zusätzliches Kopierverhalten angeben.                                                                                                                                                                                                                                                                                                                                                           |
|     \[**Löschen der Registrierung**\]      | Enthält Informationen, die zum Löschen eines Unterschlüssels oder Wertnamens aus der Registrierung verwendet werden.                                                                                                                                                                                                                                                                                                                                                    |
|     \[**Löschen von Dateien**\]         | Enthält die Dateinamen der zu löschenden Dateien und kann ein zusätzliches Löschverhalten angeben.                                                                                                                                                                                                                                                                                                                                         |
|     \[**INI-Datei in Registrierung**\] | Enthält Informationen zum Verschieben von Zeilen oder Abschnitten aus einer INI-Datei in die Registrierung.                                                                                                                                                                                                                                                                                                                                            |
|     \[**ProfileItems**\]         | Enthält Informationen zum Hinzufügen oder Entfernen von Elementen aus dem Startmenü. Nur mit Windows 2000 verfügbar.                                                                                                                                                                                                                                                                                                                          |
|     \[**Umbenennen von Dateien**\]         | Enthält die Quell- und Zieldateinamen für Umbenennungsvorgänge.                                                                                                                                                                                                                                                                                                                                                                |
|     \[**Aktualisieren von INI-Feldern**\]    | Enthält Informationen zum Ersetzen, Hinzufügen oder Löschen von Feldern in einem INI-Eintrag.                                                                                                                                                                                                                                                                                                                                                     |
|     \[**Aktualisieren der INI-Datei**\]      | Enthält Informationen zum Ersetzen, Hinzufügen oder Löschen eines INI-Eintrags.                                                                                                                                                                                                                                                                                                                                                               |
| \[**Dienstleistungen**\]                 | Listet die Dienste auf, die hinzugefügt oder installiert werden sollen.                                                                                                                                                                                                                                                                                                                                                                                            |
|     \[**Dienstinstallation**\]      | Enthält Informationen, die zum Installieren des Diensts verwendet werden, der im Abschnitt **Dienste** im AddService-Schlüssel aufgeführt ist.                                                                                                                                                                                                                                                                                                                       |
| \[**DestinationDirs**\]          | Karten Abschnitte **"Dateien kopieren"** in ein Zielverzeichnis. Definiert das Zielverzeichnis für Dateien, die in den Abschnitten **Dateien kopieren,** **Dateien umbenennen** oder **Dateien löschen** aufgeführt sind.                                                                                                                                                                                                                                                   |
| \[**SourceDisksFiles**\]         | Karten die Quelldateien den Ordnungswerten zu, die im Abschnitt **SourceDisksNames** zugewiesen sind. Siehe [INF SourceDisksNames and SourceDisksFiles Sections Example](inf-sourcedisksnames-and-sourcedisksfiles-sections-example.md).                                                                                                                                                                                                      |
| \[**SourceDisksNames**\]         | Weist jedem Quelldatenträger einen Ordinalwert zu und kann zusätzliche Informationen zu den Quelldatenträgern speichern. Siehe [INF SourceDisksNames and SourceDisksFiles Sections Example](inf-sourcedisksnames-and-sourcedisksfiles-sections-example.md).                                                                                                                                                                                     |
| \[**EventLog-Installation**\]         | Enthält Informationen, die zum Hinzufügen einer Ereignismeldung zur Registrierung verwendet werden.                                                                                                                                                                                                                                                                                                                                                               |
| \[**Zeichenfolgen**\]                  | Werte in INF-Dateien können als ersetzbare Zeichenfolgen ausgedrückt werden, indem das Format %*strkey*% verwendet wird. Jeder *Strkey* in einer INF-Datei muss einen eindeutigen Namen angeben, der aus druckbaren Zeichen besteht. Um ein %-Zeichen in die Zeichenfolge einzufügen, verwenden Sie %%. Der Strkey muss in einem **Zeichenfolgenabschnitt** der INF-Datei definiert werden. Sie können Zeichenfolgenschlüssel als Platzhalter in einer INF-Datei verwenden, um Informationen zu erhalten, die sich häufig ändern oder lokalisiert werden müssen. |



 

Eine INF-Datei kann auch private Abschnitte enthalten. Das Format eines privaten Abschnitts hängt von der Setupanwendung ab. Private Abschnitte in einer INF-Datei können verwendet werden, um spezielle Informationen zu speichern, die von einer bestimmten Setupanwendung verwendet werden. Die Setup-API enthält Funktionen wie [**SetupGetLineText**](/windows/desktop/api/Setupapi/nf-setupapi-setupgetlinetexta)und [**SetupGetIntField,**](/windows/desktop/api/Setupapi/nf-setupapi-setupgetintfield)die Informationen aus den privaten Abschnitten von INF-Dateien in den Ebenenzeilen und Feldern innerhalb von Zeilen abrufen können.
