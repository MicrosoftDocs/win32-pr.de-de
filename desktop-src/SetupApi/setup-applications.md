# <a name="setup-applications"></a>Einrichten von Anwendungen

In der Regel erstellen Sie eine *INF-Datei,* bevor Sie eine Setupanwendung erstellen. Eine INF-Datei ist eine Textdatei, die Informationen enthält, die von Ihrer Setupanwendung während einer Installation verwendet werden. Beispiele für informationen, die in einer INF-Datei gespeichert werden können, sind INI- und Registrierungsänderungen, Dateinamen und Speicherorte der Quelldateien auf Quellmedien.

Nachdem Sie eine INF-Datei erstellt haben, schreiben Sie den Quellcode für Ihre Setupanwendung. Sie können die Setupfunktionen in Ihrer Setupanwendung verwenden, um auf die Informationen in der INF-Datei zu zugreifen und Installationsvorgänge auszuführen.

In den folgenden Abschnitten werden die Verwendung von INF-Dateien und die Verarbeitungsfunktionen für das Setup beschrieben:

-   [Informationen zu INF-Dateien](about-inf-files.md)
-   [Verwenden von INF-Dateien](using-inf-files.md)
-   [INF-Dateieinrichtungsfunktionen](inf-file-setup-functions.md)

Weitere Informationen zur Verwendung der Setupfunktionen in einer Anwendung finden Sie unter [Erstellen von Setupanwendungen.](creating-setup-applications.md)
