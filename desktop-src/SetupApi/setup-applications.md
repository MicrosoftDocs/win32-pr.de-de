# <a name="setup-applications"></a>Setup Anwendungen

In der Regel erstellen Sie eine *INF-Datei* , bevor Sie eine Setup Anwendung erstellen. Eine INF-Datei ist eine Textdatei, die Informationen enthält, die von der Setup Anwendung während einer Installation verwendet werden. Beispiele für die Informationen, die in einer INF-Datei gespeichert werden können, sind u. a. ini-und Registrierungs Änderungen, Dateinamen und Speicherorte der Quelldateien auf den Quell Medien.

Nachdem Sie eine INF-Datei erstellt haben, schreiben Sie den Quellcode für die Setup Anwendung. Sie können die Setup Funktionen in der Setup Anwendung verwenden, um auf die Informationen in der INF-Datei zuzugreifen und Installations Vorgänge auszuführen.

In den folgenden Abschnitten wird die Verwendung von INF-Dateien und der Setup Verarbeitungsfunktionen beschrieben:

-   [Informationen zu INF-Dateien](about-inf-files.md)
-   [Verwenden von INF-Dateien](using-inf-files.md)
-   [Setup Funktionen für die INF-Datei](inf-file-setup-functions.md)

Weitere Informationen zum Verwenden der Setup Funktionen in einer Anwendung finden Sie unter [Erstellen von Setup Anwendungen](creating-setup-applications.md).
