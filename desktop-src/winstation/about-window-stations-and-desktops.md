# <a name="about-window-stations-and-desktops"></a>Informationen zu Fensterstationen und Desktops

Eine *Fensterstation ist* ein sicherungsfähiges Objekt, das einem Prozess zugeordnet ist und eine Zwischenablage, eine Atomtabelle und mindestens ein Desktopobjekt enthält.

Ein *Desktop ist* ein sicherungsfähiges Objekt, das in einer Fensterstation enthalten ist. Ein Desktop verfügt über eine logische Anzeigeoberfläche und enthält Benutzeroberflächenobjekte wie Fenster, Menüs und Hooks.
