# <a name="about-window-stations-and-desktops"></a>Informationen zu Fenster Stationen und Desktops

Eine *Fenster Station* ist ein Sicherungs fähiges Objekt, das einem Prozess zugeordnet ist und eine Zwischenablage, eine Atom-Tabelle und ein oder mehrere Desktop Objekte enthält.

Ein *Desktop* ist ein Sicherungs fähiges Objekt, das in einer Fenster Station enthalten ist. Ein Desktop verfügt über eine logische Anzeige Oberfläche und enthält Benutzeroberflächen Objekte wie Fenster, Menüs und Hooks.
