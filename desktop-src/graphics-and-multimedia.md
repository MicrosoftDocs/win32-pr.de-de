# <a name="graphics-and-gaming"></a>Grafik und Spiele

Windows bietet APIs und Komponenten, die Grafiken, Spiele und Bildverarbeitung unterstützen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [DirectX-Grafiken und -Spiele](./directx.md)<br/> | DirectX-Grafiken stellen eine Reihe von APIs zur Verfügung, mit denen Sie Spiele und andere hochleistungsbasierte Multimediaanwendungen erstellen können. <br/> |
| [DirectComposition](./directcomp/directcomposition-portal.md)<br/> | DirectComposition ermöglicht eine leistungsstarke Bitmapkomposition mit Transformationen, Effekten und Animationen. Sie können die DirectComposition-API verwenden, um visuell ansprechende Benutzeroberflächen zu erstellen, die umfangreiche und fließende animierte Übergänge von einem Visual zu einem anderen bieten.<br/> |
| [Spielmodus](/previous-versions/windows/desktop/gamemode/game-mode-portal)<br/> | Mithilfe der Game Mode-APIs für die Universelle Windows-Plattform (UWP) sorgen Sie für ein optimiertes Spielerlebnis, indem Sie den Spielmodus in Windows 10 nutzen.<br/> |
| [Gaming-Geräteinformationen](/previous-versions/windows/desktop/gamingdvcinfo/gaming-device-information-portal)<br/> | Die Gaming-Geräteinformations-APIs ermöglichen UWP-Spielentwicklern, den Typ der Konsole zu bestimmen, auf der das Spiel ausgeführt wird, um zur Laufzeit Entscheidungen zur besten Verwendung der Hardware treffen zu können.<br/> |
| [Windows Imaging-Komponente (WIC)](./wic/-wic-lh.md)<br/> | Die Windows Imaging Component (WIC) ist eine erweiterbare Plattform, die low-level API für digitale Bilder bietet. WIC unterstützt die Standardformate für Webbilder, Bilder mit hohem dynamischen Bereich und Unformatieren von Kameradaten.<br/> |
| [Win2D (externe Website)](https://github.com/Microsoft/Win2D)<br/> | Win2D ist eine einfach zu verwendende Windows-Runtime-API für 2D-Grafikrendering im unmittelbaren Modus mit GPU-Beschleunigung. Es steht C- und C++-Entwicklern zur Verfügung, die Windows-Apps für Windows 8.1, Windows Phone \# 8.1 und Windows 10. Sie nutzt die Leistung von Direct2D und lässt sich nahtlos in XAML und CoreWindow integrieren.<br/> |
| [ANGLE für Windows Store (externe Website)](https://github.com/microsoft/angle/wiki)<br/> | ANGLE für Windows Store ist ein Open-Source-Projekt, mit dem Entwickler OpenGL ES-Inhalte auf Windows ausführen können, indem Sie OpenGL ES-API-Aufrufe in DirectX 11-API-Aufrufe übersetzen. ANGLE für Windows Store unterstützt Windows 8.1, Windows Phone 8.1 und Windows 10.<br/> |

> [!NOTE]
> Ab Windows 10, Version 1809 (10.0, Build 17763), wird TruePlay aus Windows entfernt. Die TruePlay-Dokumentation wird nicht veröffentlicht.
