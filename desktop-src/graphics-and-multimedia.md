# <a name="graphics-and-gaming"></a>Grafik und Spiele

Windows stellt APIs und Komponenten bereit, die Grafiken, Spiele und Bild Erstellung unterstützen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [DirectX-Grafiken und -Spiele](./directx.md)<br/> | DirectX-Grafiken stellen eine Reihe von APIs bereit, mit denen Sie Spiele und andere hochleistungsfähige Multimedia-Anwendungen erstellen können. <br/> |
| [DirectComposition](./directcomp/directcomposition-portal.md)<br/> | Directcomposition ermöglicht eine leistungsstarke bitmapkomposition mit Transformationen, Effekten und Animationen. Sie können die directcomposition-API verwenden, um visuell ansprechende Benutzeroberflächen zu erstellen, die umfassende und fließende animierte Übergänge von einem visuellen Element in ein anderes Feature haben.<br/> |
| [Spielmodus](/previous-versions/windows/desktop/gamemode/game-mode-portal)<br/> | Mithilfe der Game Mode-APIs für die Universelle Windows-Plattform (UWP) sorgen Sie für ein optimiertes Spielerlebnis, indem Sie den Spielmodus in Windows 10 nutzen.<br/> |
| [Gaming-Geräteinformationen](/previous-versions/windows/desktop/gamingdvcinfo/gaming-device-information-portal)<br/> | Die Gaming-Geräte Informations-APIs ermöglichen es UWP-Spiel Entwicklern, den Typ der Konsole zu bestimmen, auf der das Spiel ausgeführt wird, um Lauf Zeit Entscheidungen zur optimalen Verwendung der Hardware zu treffen.<br/> |
| [Windows Imaging-Komponente (WIC)](./wic/-wic-lh.md)<br/> | Bei der Windows Imaging Component (WIC) handelt es sich um eine erweiterbare Plattform, die eine Low-Level-API für digitale Images bereitstellt. WIC unterstützt die Standardweb Bildformate, hohe dynamische Bereichs Bilder und unformatierte Kameradaten.<br/> |
| [Win2D (externe Website)](https://github.com/Microsoft/Win2D)<br/> | Win2D ist eine einfach zu verwendende Windows-Runtime-API für 2D-Grafikrendering im unmittelbaren Modus mit GPU-Beschleunigung. Es ist für C \# -und C++-Entwickler verfügbar, die Windows-Apps für Windows 8.1 schreiben, Windows Phone 8,1 und Windows 10. Es nutzt die Leistungsfähigkeit von Direct2D und lässt sich nahtlos in XAML und corewindow integrieren.<br/> |
| [Winkel für Windows Store (externe Site)](https://github.com/microsoft/angle/wiki)<br/> | Der Winkel für Windows Store ist ein Open-Source-Projekt, mit dem Entwickler OpenGL es-Inhalte unter Windows ausführen können, indem Sie OpenGL es-API-Aufrufe in DirectX 11-API-Aufrufe übersetzen. Der Winkel für Windows Store unterstützt Windows 8.1, Windows Phone 8,1 und Windows 10.<br/> |

> [!NOTE]
> Ab Windows 10, Version 1809 (10.0, Build 17763), wird TruePlay aus Windows entfernt. Die TruePlay-Dokumentation wird nicht veröffentlicht.
