# <a name="storyboard-interfaces"></a>Storyboardschnittstellen

Dieser Abschnitt enthält die Referenz Spezifikationen für die Windows Animation Manager-Schnittstellen, die Storyboards unterstützen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                                 | BESCHREIBUNG                                                                                                           |
|-------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**Iuianimationstoryboard**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboard)<br/>                                   | Definiert ein Storyboard, das eine Gruppe von Übergängen enthält, die relativ zueinander synchronisiert werden.<br/> |
| [**IUIAnimationStoryboard2**](/windows/win32/api/uianimation/nn-uianimation-iuianimationstoryboard2)<br/>                                 | Definiert ein Storyboard, das eine Gruppe von Übergängen enthält, die relativ zueinander synchronisiert werden.<br/> |
| [**Iuianimationstoryboarde venthandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboardeventhandler)<br/>           | Definiert Methoden für die Verarbeitung von Status-und Update Ereignissen für ein Storyboard.<br/>                                    |
| [**IUIAnimationStoryboardEventHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboardeventhandler2)<br/>         | Definiert Methoden zum Verarbeiten von Storyboard-Ereignissen. <br/>                                                           |
| [**IUIAnimationLoopIterationChangeHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationloopiterationchangehandler2)<br/> | Definiert eine Methode zum Verarbeiten von Storyboard-Schleifen Iterations Ereignissen.<br/>                                            |
| [**Iuianimationprioritycomparison**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationprioritycomparison)<br/>                   | Definiert eine Methode für den Prioritäts Vergleich, die der Animations-Manager verwendet, um Planungs Konflikte aufzulösen.<br/>  |
| [**IUIAnimationPriorityComparison2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationprioritycomparison2)<br/>                 | Definiert eine Methode, die Planungs Konflikte mithilfe des Prioritäts Vergleichs auflöst.<br/>                           |
