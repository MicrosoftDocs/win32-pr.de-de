# <a name="storyboard-interfaces"></a>Storyboardschnittstellen

Dieser Abschnitt enthält die Referenzspezifikationen für die Windows Animation Manager-Schnittstellen, die Storyboards unterstützen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                                 | BESCHREIBUNG                                                                                                           |
|-------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**IUIAnimationStoryboard**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboard)<br/>                                   | Definiert ein Storyboard, das eine Gruppe von Übergängen enthält, die relativ zueinander synchronisiert werden.<br/> |
| [**IUIAnimationStoryboard2**](/windows/win32/api/uianimation/nn-uianimation-iuianimationstoryboard2)<br/>                                 | Definiert ein Storyboard, das eine Gruppe von Übergängen enthält, die relativ zueinander synchronisiert werden.<br/> |
| [**IUIAnimationStoryboardEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboardeventhandler)<br/>           | Definiert Methoden zum Behandeln von Status- und Aktualisierungsereignissen für ein Storyboard.<br/>                                    |
| [**IUIAnimationStoryboardEventHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboardeventhandler2)<br/>         | Definiert Methoden für die Behandlung von Storyboardereignissen. <br/>                                                           |
| [**IUIAnimationLoopIterationChangeHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationloopiterationchangehandler2)<br/> | Definiert eine Methode für die Behandlung von Storyboardschleifeniterationsereignissen.<br/>                                            |
| [**IUIAnimationPriorityComparison**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationprioritycomparison)<br/>                   | Definiert eine Methode für den Prioritätsvergleich, mit der der Animations-Manager Zeitplanungskonflikte löst.<br/>  |
| [**IUIAnimationPriorityComparison2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationprioritycomparison2)<br/>                 | Definiert eine Methode, die Planungskonflikte durch Prioritätsvergleiche löst.<br/>                           |
