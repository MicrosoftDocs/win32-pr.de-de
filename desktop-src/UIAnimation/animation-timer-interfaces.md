# <a name="animation-timer-interfaces"></a>Zeit Geber Schnittstellen für Animationen

Dieser Abschnitt enthält die Referenz Spezifikationen für die Windows Animation Manager-Schnittstellen, die Animations Timer unterstützen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                         | BESCHREIBUNG                                                                                   |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**Iuianimationtimer**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimer)<br/>                                     | Definiert einen Animations Zeit Geber, der Dienste zum Verwalten der Animation der Animation bereitstellt.<br/> |
| [**Iuianimationtimerclienteventhandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimerclienteventhandler)<br/> | Definiert eine Methode für die Behandlung von Ereignissen, die sich auf Änderungen im Zeit Geber Client Status beziehen.<br/>    |
| [**Iuianimationtimereventhandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimereventhandler)<br/>             | Definiert Methoden für die Behandlung von Zeit Steuerungs Ereignissen.<br/>                                        |
| [**Iuianimationtimerupdatehandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimerupdatehandler)<br/>           | Definiert Methoden für die Behandlung von Zeit Steuerungs Ereignissen.<br/>                                 |
