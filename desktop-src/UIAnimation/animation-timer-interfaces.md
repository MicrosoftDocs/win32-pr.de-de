# <a name="animation-timer-interfaces"></a>Animations-Timerschnittstellen

Dieser Abschnitt enthält die Referenzspezifikationen für die Windows Animation Manager-Schnittstellen, die Animationszeitr unterstützen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                         | Beschreibung                                                                                   |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**IUIAnimationTimer**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimer)<br/>                                     | Definiert einen Animationszeitgeber, der Dienste zum Verwalten des Animationszeitpunkts bietet.<br/> |
| [**IUIAnimationTimerClientEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimerclienteventhandler)<br/> | Definiert eine Methode zum Behandeln von Ereignissen im Zusammenhang mit Änderungen im Timerclientstatus.<br/>    |
| [**IUIAnimationTimerEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimereventhandler)<br/>             | Definiert Methoden zum Behandeln von Zeitsteuerungsereignissen.<br/>                                        |
| [**IUIAnimationTimerUpdateHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimerupdatehandler)<br/>           | Definiert Methoden zum Behandeln von Zeitsteuerungsaktualisierungsereignissen.<br/>                                 |
