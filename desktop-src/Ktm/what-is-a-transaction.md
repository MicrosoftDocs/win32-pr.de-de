# <a name="what-is-a-transaction"></a><span data-ttu-id="732b8-103">Was ist eine Transaktion?</span><span class="sxs-lookup"><span data-stu-id="732b8-103">What is a Transaction?</span></span>

<span data-ttu-id="732b8-104">Bei einer Transaktion handelt es sich um eine Gruppe von Vorgängen, die über die folgenden Eigenschaften verfügen: atomarisch, konsistent, isoliert und dauerhaft (ACID).</span><span class="sxs-lookup"><span data-stu-id="732b8-104">A transaction is a group of operations that have the following properties: atomic, consistent, isolated, and durable (ACID).</span></span> <span data-ttu-id="732b8-105">Durch die Unterstützung von Transaktionen können neue Arten von Anwendungen entwickelt werden, die den Entwicklungsprozess vereinfachen und die Anwendung stabiler werden.</span><span class="sxs-lookup"><span data-stu-id="732b8-105">The support of transactions enables new types of applications to be developed, while simplifying the development process and making the application more robust.</span></span> <span data-ttu-id="732b8-106">Der Rest dieses Themas enthält Szenarios, in denen die Notwendigkeit dieser Eigenschaften veranschaulicht wird, und dann eine Tabelle, in der jede Eigenschaft definiert wird.</span><span class="sxs-lookup"><span data-stu-id="732b8-106">The remainder of this topic provides scenarios that demonstrate the need for these properties, then a table that defines each property.</span></span>

<span data-ttu-id="732b8-107">In einer *atomaren* Gruppe von Vorgängen muss entweder jeder Vorgang in der Gruppe erfolgreich ausgeführt werden, oder die Auswirkungen all dieser Vorgänge müssen rückgängig gemacht werden (auch als *Rollback* bezeichnet).</span><span class="sxs-lookup"><span data-stu-id="732b8-107">In an *atomic* group of operations, either every operation in the group must succeed, or the effects of all of them must be undone (also known as *rolling back*).</span></span> <span data-ttu-id="732b8-108">Eine Bank Übertragung muss beispielsweise ein atomarischer Satz von zwei Vorgängen sein: ein Abgleich von einem Konto und eine Gutschrift für ein anderes Konto.</span><span class="sxs-lookup"><span data-stu-id="732b8-108">For example, a bank transfer must be an atomic set of two operations: a debit from one account and a credit to another account.</span></span> <span data-ttu-id="732b8-109">Der Debit und das Guthaben müssen als atomarische Gruppe implementiert werden.</span><span class="sxs-lookup"><span data-stu-id="732b8-109">The debit and credit must be implemented as an atomic group.</span></span> <span data-ttu-id="732b8-110">Wenn diese beiden Vorgänge nicht beides erfolgreich sind, wird die Übertragung entweder von der Bank oder dem Kontoinhaber nicht durchgeführt.</span><span class="sxs-lookup"><span data-stu-id="732b8-110">If those two operations do not both succeed, then the transfer is either unfairly in favor of the bank or the account holder.</span></span>

<span data-ttu-id="732b8-111">Die *Konsistenz* Anforderung bedeutet, dass die Daten nach der Transaktion konsistent sind (vorausgesetzt, dass wir mit einem konsistenten System vor der Transaktion gestartet haben).</span><span class="sxs-lookup"><span data-stu-id="732b8-111">The requirement of *consistency* means that the data is consistent after the transaction (assuming that we started with a consistent system before the transaction).</span></span> <span data-ttu-id="732b8-112">Für das Beispiel "Bank Übertragung" kann die Konsistenz so definiert werden, dass der kombinierte Konto Saldo der beiden Konten eine Konstante ist.</span><span class="sxs-lookup"><span data-stu-id="732b8-112">For the bank transfer example, consistency may be defined as having the combined account balance of the two accounts be a constant.</span></span> <span data-ttu-id="732b8-113">Um die Konsistenz im Beispiel der Bank Übertragung zu implementieren, müssen die Vorgänge "Debit" und "Credit" lediglich für die gleiche Menge an Geld erfolgen.</span><span class="sxs-lookup"><span data-stu-id="732b8-113">To implement consistency in the bank transfer example, the debit and credit operations simply need to be for the same amount of money.</span></span>

<span data-ttu-id="732b8-114">Ein weiteres Beispiel für eine Transaktion ist ein Update für eine Website.</span><span class="sxs-lookup"><span data-stu-id="732b8-114">Another example of a transaction is an update to a website.</span></span> <span data-ttu-id="732b8-115">Für eine Electronic Commerce-Website ist es erforderlich, dass eine neue Navigationsseite der Produktkategorie genau zur gleichen Zeit wie die Produktdetailseiten angezeigt wird, die die neuen Produkte beschreiben.</span><span class="sxs-lookup"><span data-stu-id="732b8-115">An electronic commerce site requires that a new product category navigation page to appear at exactly the same time as the product detail pages that describe the new products.</span></span> <span data-ttu-id="732b8-116">In diesem Fall müssen Sie unter Kontrolle einer Transaktion mehrere Verzeichniseinträge aktualisieren und hinzufügen.</span><span class="sxs-lookup"><span data-stu-id="732b8-116">In this case, there is a need to update and add multiple directory entries under the control of a transaction.</span></span> <span data-ttu-id="732b8-117">Es ist nicht nur erforderlich, dass die Updates atomarisch sind, aber es ist auch erforderlich, dass ein Kunde, der derzeit Einkäufe durchläuft, die laufenden Updates nicht sehen kann.</span><span class="sxs-lookup"><span data-stu-id="732b8-117">Not only is it necessary to have the updates be atomic, but it is also necessary that a customer who is currently shopping must not see the updates in progress.</span></span> <span data-ttu-id="732b8-118">Dies ist ein Beispiel für die *Isolations* Eigenschaft der Transaktionen.</span><span class="sxs-lookup"><span data-stu-id="732b8-118">This is an example of the *isolation* property of transactions.</span></span>

<span data-ttu-id="732b8-119">Die Eigenschaft der *Dauerhaftigkeit* erfordert, dass nach Abschluss eines Updates die Auswirkungen weiterhin bestehen, auch wenn das System nicht mehr reagiert.</span><span class="sxs-lookup"><span data-stu-id="732b8-119">The property of *durability* requires that after an update is finished, its effects persist even if the system stops responding.</span></span> <span data-ttu-id="732b8-120">Im vorherigen Beispiel kann die Dauerhaftigkeit einfach durch sicherstellen einer ausreichenden Datenwiederherstellung gewährleistet werden, sodass alle neuen Dateisystem Einträge, die das Hinzufügen eines neuen Produkts zum Standort darstellen, angezeigt werden, nachdem ein System nicht mehr reagiert.</span><span class="sxs-lookup"><span data-stu-id="732b8-120">In the previous example, durability can be provided simply by ensuring adequate data recovery so that all new file system entries that represent the addition of a new product to the site appear after a system stops responding.</span></span> <span data-ttu-id="732b8-121">Hierfür ist ein System mit Daten Sicherungs-, Wiederherstellungs-und hoch Verfügbarkeits Mechanismen erforderlich.</span><span class="sxs-lookup"><span data-stu-id="732b8-121">This requires a system with data backup, recovery, and high availability mechanisms.</span></span>

<span data-ttu-id="732b8-122">Die Gewährleistung der Atomizität einer Transaktion sowie der anderen Eigenschaften ist bei einer beliebigen Anzahl von Fehlern vorhanden, einschließlich Fehlern, die während der Wiederherstellungs Phase eines vorherigen Fehlers auftreten.</span><span class="sxs-lookup"><span data-stu-id="732b8-122">The guarantee of the atomicity of a transaction, as well as the other properties, is present in the face of any number of failures, including failures that occur during the recovery phase of a prior failure.</span></span> <span data-ttu-id="732b8-123">Letztendlich erreicht das System entweder einen von zwei Zuständen: alle Vorgänge wurden angewendet, oder es wurden keine Vorgänge angewendet.</span><span class="sxs-lookup"><span data-stu-id="732b8-123">Eventually the system will reach either one of two states: all operations have been applied or none of the operations have been applied.</span></span>
