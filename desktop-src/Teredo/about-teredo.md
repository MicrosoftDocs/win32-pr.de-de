# <a name="about-teredo"></a>Informationen zu Teredo

Aufgrund der begrenzten Anzahl verfügbarer IPv4-Adressen müssen Internet Dienstanbieter (Internet Service Provider, ISP) mit der Umstellung auf IPv6 beginnen. Windows Vista bietet vollständige Unterstützung für IPv6, sodass Entwickler sicherstellen können, dass Ihre Anwendungen Morgen bereit sind, indem Sie die IPv6-Unterstützung noch heute aktivieren. Mit Teredo können Windows Vista-Desktops auch dann IPv6-Verbindungen herstellen, wenn ISP-Netzwerke nur IPv4 sind.

Technologien wie Peer-to-Peer erfordern eine End-to-End-Konnektivität für den gesamten Benutzer. Viele Heim Häuser haben NAT-Geräte (Network Address Translation, Netzwerk Adressübersetzung), mit denen diese Technologien eingeschränkt werden, und Anwendungsentwickler müssen sich auf effiziente Möglichkeiten zum Umgehen dieser Probleme konzentrieren. Die Teredo-Unterstützung mit Windows Vista verringert jedoch die Belastung der Anwendungsentwickler durch die Bereitstellung einer nahtlosen NAT-Durchquerung.
