# <a name="teredo"></a>Teredo

## <a name="purpose"></a>Zweck

Teredo ist eine IPv6-Übergangstechnologie, die Adresszuweisung und automatisches Host-zu-Host-Tunnelingverfahren für IPv6-Unicastdatenverkehr bereitstellt, wenn sich IPv6/IPv4-Hosts hinter einem oder mehreren IPv4-Netzwerk Adress Übersetzer (NATs) befinden. Zum Durchlaufen von IPv4-NATs werden IPv6-Pakete als IPv4-UDP (User Datagram Protocol)-Nachrichten gesendet.

## <a name="developer-audience"></a>Entwicklergruppe

Teredo ist für die Verwendung durch C/C++-Entwickler mit IPv6-Netzwerk Programmierung konzipiert.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Die Teredo-Schnittstelle wird hauptsächlich von Windows Vista und Windows Server 2008 unterstützt. Die eingeschränkte Funktionalität der Teredo-Schnittstelle, die von Windows XP mit Service Pack 2 (SP2) und Windows Server 2003 unterstützt wird, wird beim [empfangen von angeforderter Datenverkehr über Teredo](receiving-solicited-traffic-over-teredo.md)ausführlich erläutert.
