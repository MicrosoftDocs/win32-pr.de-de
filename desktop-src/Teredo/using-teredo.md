# <a name="using-teredo"></a>Verwenden von Teredo

Beim Entwickeln einer Anwendung, die [Teredo](about-teredo.md)verwendet, besteht das Ziel darin, IPv6-Konnektivität zu erreichen und sich nicht auf die Verwendung von Teredo zu konzentrieren. Windows-Teamarbeit und Remote Unterstützung sind z. b. zwei Anwendungen, die die von IPv6 bereitgestellte End-to-End-Konnektivität nutzen. Diese Anwendungen können Verbindungen mithilfe von systemeigenen IPv6-, IPv6-zu-IPv4-oder Teredo-Adressen abrufen. Für diese Anwendungen und alle anderen Anwendungen, die möglicherweise für IPv6 entwickelt werden, wird Teredo wie jedes andere Protokoll behandelt, das Konnektivität ermöglicht.

-   [Empfangen von angeforderten Datenverkehr über Teredo](receiving-solicited-traffic-over-teredo.md)
-   [Empfangen von angefordertem Datenverkehr über Teredo](receiving-unsolicited-traffic-over-teredo.md)
