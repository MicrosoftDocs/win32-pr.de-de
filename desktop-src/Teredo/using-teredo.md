# <a name="using-teredo"></a>Verwenden von Teredo

Bei der Entwicklung einer Anwendung, die [Teredo verwendet,](about-teredo.md)besteht das Ziel im Erreichen der IPv6-Konnektivität und nicht im Fokus der Verwendung von Teredo. Beispielsweise sind Windows Besprechungsraum und Remoteunterstützung zwei Anwendungen, die die end-to-End-Konnektivität von IPv6 nutzen. Diese Anwendungen können Konnektivität mit nativen IPv6-, 6to4- oder Teredo-Adressen abrufen. Für diese und alle anderen Anwendungen, die für IPv6 entwickelt werden können, wird Teredo wie jedes andere Protokoll behandelt, das Konnektivität ermöglicht.

-   [Empfangen von angeforderten Datenverkehr über Teredo](receiving-solicited-traffic-over-teredo.md)
-   [Empfangen von nicht angeforderten Datenverkehr über Teredo](receiving-unsolicited-traffic-over-teredo.md)
-   [Verwenden von Teredo mit dem IP-Hilfsfeld](using-teredo-with-ip-helper.md)
