# <a name="the-teredo-security-model"></a>Das Teredo-Sicherheitsmodell

Die [Teredo](about-teredo.md) -Technologie bietet, wenn Sie mit Windows-Security Center und der [Windows-Firewall](/previous-versions/windows/desktop/ics/windows-firewall-start-page)gekoppelt ist, eine verbesserte Sicherheit für den Endbenutzer. Bei der Windows-Firewall wird der über die Teredo-Schnittstelle gesendete und empfangene IPv6-Verkehr genauso wie der systemeigene IPv6-Datenverkehr verarbeitet Die Teredo-Schnittstelle wird im System als weiterer Miniport erkannt.

Weitere Informationen zur Verwendung finden Sie unter [Implementieren des Teredo-Sicherheitsmodells](implementing-the-teredo-security-model.md).
