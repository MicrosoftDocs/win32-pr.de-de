# <a name="the-teredo-security-model"></a>Das Teredo-Sicherheitsmodell

Die [Teredo-Technologie](about-teredo.md) bietet in Verbindung mit Windows-Sicherheit Center und [Windows Firewall](/previous-versions/windows/desktop/ics/windows-firewall-start-page)eine höhere Sicherheit für den Endbenutzer. Für Windows Firewall wird der über die Teredo-Schnittstelle gesendete und empfangene IPv6-Datenverkehr genauso verarbeitet wie nativer IPv6-Datenverkehr. Die Teredo-Schnittstelle wird als anderer Miniport im System erkannt.

Weitere Informationen zur Verwendung finden Sie unter [Implementieren des Teredo-Sicherheitsmodells.](implementing-the-teredo-security-model.md)
