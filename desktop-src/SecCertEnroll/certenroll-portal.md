# <a name="certificate-enrollment-api"></a>Certificate Enrollment API (Zertifikatregistrierungs-API, in englischer Sprache)

## <a name="purpose"></a>Zweck

Die Zertifikatregistrierungs-API kann verwendet werden, um eine Clientanwendung zum Anfordern eines Zertifikats und Installieren einer Zertifikatantwort zu erstellen. Diese API wird ab CertEnroll.dll Vista Windows implementiert. er ersetzt Xenroll.dll.

## <a name="developer-audience"></a>Entwicklergruppe

Die Zertifikatregistrierungs-API wird von Entwicklern von Anwendungen verwendet, mit denen Benutzer Zertifikate über Medien wie das Internet oder ein Intranet erstellen, anfordern und abrufen können, die nicht grundsätzlich sicher sind. Entwickler sollten mit den Programmiersprachen C und C++, dem Component Object Model (COM) und der Windows-basierten Programmierumgebung vertraut sein. Obwohl dies nicht erforderlich ist, wird ein Verständnis der Kryptografie- und Public Key-Infrastruktur empfohlen.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Die Zertifikatregistrierungs-API wird ab Windows Server 2008 und Windows Vista unterstützt. Informationen zu Laufzeitanforderungen für ein bestimmtes Programmierelement finden Sie im Abschnitt Anforderungen der Referenzseite für dieses Element.
