# <a name="certification-authority-interfaces"></a>Zertifizierungsstellen Schnittstellen

Die folgenden Schnittstellen können zum Abrufen von Eigenschafts Werten von einer Zertifizierungsstelle verwendet werden.



| Schnittstelle                                                      | BESCHREIBUNG                                                                                 |
|----------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| [**Icertificationautoritäten**](/windows/desktop/api/Certenroll/nn-certenroll-icertificationauthorities) | Verwaltet eine Auflistung von [**icertificationauthority**](/windows/desktop/api/certenroll/nn-certenroll-icertificationauthority) -Objekten. |
| [**Icertificationauthority**](/windows/desktop/api/certenroll/nn-certenroll-icertificationauthority)     | Stellt eine einzelne Zertifizierungsstelle dar.                                                |
