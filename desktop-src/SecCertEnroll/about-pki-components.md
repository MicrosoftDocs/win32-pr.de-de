# <a name="pki-elements"></a>PKI-Elemente

Eine typische Public Key-Infrastruktur (PKI) von Microsoft umfasst die folgenden Elemente:

| Element                                                                     | BESCHREIBUNG                                                                                                                                                                                              |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Zertifizierungsstellen](about-certification-authorities.md)<br/> | Bereitstellen von Diensten, die die Identität von Einzelpersonen, Computern und anderen Entitäten in einem Netzwerk authentifizieren. Dies umfasst sowohl Stamm Zertifizierungsstellen als auch untergeordnete Autoritäten.<br/> |
| [Zertifikatsverzeichnis](about-certificate-directory.md)<br/>         | Speichert Zertifikat Anforderungen und ausgestellte und widerrufene Zertifikate und Zertifikat Anforderungen.<br/>                                                                                                      |
| [Schlüssel Wiederherstellungs Server](about-key-recovery-server.md)<br/>             | Speichert verschlüsselte private Schlüssel in der Zertifikat Datenbank zur Wiederherstellung nach dem Verlust.<br/>                                                                                                             |
