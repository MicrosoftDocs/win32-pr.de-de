# <a name="trust-models"></a>Trust-Modelle

Wenn sich die Anzahl der Organisationen, die die Public Key-Infrastruktur (PKI) implementieren, erhöht und die Anzahl von Endbenutzern und Computern in diesen Organisationen zunimmt, erhöhen sich auch die logistischen Probleme im Zusammenhang mit der Zertifizierung von Identitäten und dem Einrichten von Zwei primäre Vertrauens Modelle, Zertifikat Hierarchien und eine Kreuz Zertifizierung wurden entwickelt, um diese Probleme zu beheben. In beiden Modellen ist das Konzept einer Zertifikat Kette enthalten. Weitere Informationen finden Sie unter den folgenden Themen:

-   [Zertifikat Hierarchie](about-certificate-hierarchy.md)
-   [Kreuz Zertifizierung](about-cross-certification.md)
-   [Zertifikatkette](about-certificate-chain.md)
