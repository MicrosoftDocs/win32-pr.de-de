# <a name="certenroll-enumerations"></a>CertEnroll-Enumerationen

Die folgenden Enumerationen werden von der Zertifikatregistrierungs-API unterstützt.

| Enumeration                                                                              | Beschreibung                                                                                                                                                                                                                                                                                                              |
|------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Algorithmflags**](/windows/desktop/api/CertEnroll/ne-certenroll-algorithmflags)                                            | Enthält Flags, die verwendet werden können, um die Suche nach einem Kryptografiealgorithmus zu verfeinern.                                                                                                                                                                                                                                      |
| [**Algorithmoperationflags**](/windows/desktop/api/CertEnroll/ne-certenroll-algorithmoperationflags)                          | Gibt die Vorgänge an, die von einem Algorithmus durchgeführt werden können. Hiermit werden die operativen Funktionen eines CSP-Algorithmus oder CSP-Statusinformationen basierend auf diesen Funktionen abgerufen.                                                                                                                                  |
| [**AlgorithmType**](/windows/desktop/api/CertEnroll/ne-certenroll-algorithmtype)                                              | Gibt den beabsichtigten Zweck eines Kryptografiealgorithmus an.                                                                                                                                                                                                                                                             |
| [**Alternativenametype**](/windows/desktop/api/CertEnroll/ne-certenroll-alternativenametype)                                  | Gibt die alternativen namens Typen an, die beim Erstellen der Erweiterung **alternativename** angegeben werden können.                                                                                                                                                                                                               |
| [**CertEnroll- \_ objectID**](/windows/desktop/api/CertEnroll/ne-certenroll-certenroll_objectid)                                 | Enthält die von der Zertifikatregistrierungs-API unterstützten vordefinierten [*Objekt*](/windows/desktop/SecGloss/o-gly) -IDs (OIDs).                                                                                                                                          |
| [**CertEnroll \_ PropertyId**](/windows/desktop/api/CertEnroll/ne-certenroll-certenroll_propertyid)                             | Enthält vordefinierte OIDs für externe Eigenschaften, die von der Zertifikatregistrierungs-API unterstützt werden.                                                                                                                                                                                                                            |
| [**Committemplateflags**](/windows/desktop/api/Certenroll/ne-certenroll-committemplateflags)                                       | Gibt Optionen zum Speichern und Löschen von Vorlagen an.                                                                                                                                                                                                                                                                     |
| [**EncodingType**](/windows/desktop/api/CertEnroll/ne-certenroll-encodingtype)                                                | Gibt den Codierungstyp an, der zu Anzeige Zwecken auf ein Bytearray angewendet wird.                                                                                                                                                                                                                                             |
| [**Registrimentcaproperty**](/windows/desktop/api/Certenroll/ne-certenroll-enrollmentcaproperty)                                     | Gibt die Eigenschaftswerte der Zertifizierungsstelle an.                                                                                                                                                                                                                                                                       |
| [**Registrierungs Display Status**](/windows/desktop/api/CertEnroll/ne-certenroll-enrollmentdisplaystatus)                          | Gibt an, ob die Informationen zum Registrierungsstatus auf einer Benutzeroberfläche angezeigt werden sollen.                                                                                                                                                                                                                                          |
| [**Registrierungs Registrierungsstatus**](/windows/desktop/api/CertEnroll/ne-certenroll-enrollmentenrollstatus)                            | Gibt den Registrierungsstatus einer Zertifikat Anforderung an.                                                                                                                                                                                                                                                                |
| [**Registrierungspolicyflags**](/windows/desktop/api/Certenroll/ne-certenroll-enrollmentpolicyflags)                                   | Gibt Gruppen Richtlinienflags an.                                                                                                                                                                                                                                                                                            |
| [**Registrierungspolicyserverpropertyflags**](/windows/desktop/api/Certenroll/ne-certenroll-enrollmentpolicyserverpropertyflags)       | Gibt den Standardrichtlinien Server an.                                                                                                                                                                                                                                                                                     |
| [**Registrierungsstatus**](/windows/desktop/api/CertEnroll/ne-certenroll-enrollmentselectionstatus)                      | Gibt an, ob der Registrierungsstatus eines Objekts während des Registrierungsvorgangs überwacht wird.                                                                                                                                                                                                                    |
| [**Anmelmenttemplateproperty**](/windows/desktop/api/Certenroll/ne-certenroll-enrollmenttemplateproperty)                         | Enthält Eigenschaftswerte für eine angegebene Vorlage.                                                                                                                                                                                                                                                                           |
| [**Innerrequestlevel**](/windows/desktop/api/CertEnroll/ne-certenroll-innerrequestlevel)                                      | Gibt die Kapselungs Ebene einer Zertifikat Anforderung in einer PKCS \# 7-oder einer CMC-Anforderung an.                                                                                                                                                                                                                               |
| [**Installresponserestrictionflags**](/windows/desktop/api/CertEnroll/ne-certenroll-installresponserestrictionflags)          | Enthält Flags, mit denen die Einschränkungen für die lokale Installation einer Zertifikat Kette identifiziert werden.                                                                                                                                                                                                                   |
| [**Keyidentifierhashalgorithm**](/windows/desktop/api/CertEnroll/ne-certenroll-keyidentifierhashalgorithm)                    | Gibt den Algorithmus an, mit dem der öffentliche Schlüssel in einer Zertifikat Anforderung als Hash verwendet wird.                                                                                                                                                                                                                                            |
| [**Objectidgroupid**](/windows/desktop/api/CertEnroll/ne-certenroll-objectidgroupid)                                          | Gibt die Kategorie oder Gruppe an, zu der eine OID gehört.                                                                                                                                                                                                                                                                 |
| [**Objectidpublickeyflags**](/windows/desktop/api/CertEnroll/ne-certenroll-objectidpublickeyflags)                            | Gibt an, ob ein Algorithmus für öffentliche Schlüssel zum Signieren oder zur Verschlüsselung verwendet wird.                                                                                                                                                                                                                                          |
| [**Pfxexportoptions**](/windows/desktop/api/CertEnroll/ne-certenroll-pfxexportoptions)                                        | Gibt an, wie viel eine Zertifikat Kette beim Erstellen einer PFX-Nachricht (Personal Information Exchange) enthalten ist.                                                                                                                                                                                                       |
| [**Pkcs10AllowedSignatureTypes**](/windows/desktop/api/CertEnroll/ne-certenroll-pkcs10allowedsignaturetypes)                  | Gibt den Typ der Signatur an, die beim Signieren einer Zertifikat Anforderung zulässig ist.                                                                                                                                                                                                                                            |
| [**Policyqualifiertype**](/windows/desktop/api/CertEnroll/ne-certenroll-policyqualifiertype)                                  | Gibt den Typ des auf eine Zertifikat Richtlinie angewendeten Qualifizierers an.                                                                                                                                                                                                                                                         |
| [**PolicyServerUrlFlags**](/windows/desktop/api/Certenroll/ne-certenroll-policyserverurlflags)                                     | Enthält Zertifikatregistrierungs-Richtlinien Server-Flags (CEP).                                                                                                                                                                                                                                                               |
| [**PolicyServerUrlPropertyID**](/windows/desktop/api/Certenroll/ne-certenroll-policyserverurlpropertyid)                           | Enthält Werte, die den Typ des Eigenschafts Werts angeben, der von der [**getStringProperty**](/windows/desktop/api/Certenroll/nf-certenroll-ix509policyserverurl-getstringproperty) -Methode zurückgegeben oder von der [**setStringProperty**](/windows/desktop/api/Certenroll/nf-certenroll-ix509policyserverurl-setstringproperty) -Methode in der [**IX509PolicyServerUrl**](/windows/desktop/api/Certenroll/nn-certenroll-ix509policyserverurl) -Schnittstelle festgelegt wird. |
| [**Requestclientinfoclientid**](/windows/desktop/api/CertEnroll/ne-certenroll-requestclientinfoclientid)                      | Gibt den Anwendungstyp an, von dem eine Zertifikat Anforderung erstellt wurde.                                                                                                                                                                                                                                                    |
| [**Webenrollmentflags**](/windows/desktop/api/Certenroll/ne-certenroll-webenrollmentflags)                                         | Gibt das Webregistrierungs Verhalten an.                                                                                                                                                                                                                                                                                       |
| [**Websecuritylevel**](/windows/desktop/api/CertEnroll/ne-certenroll-websecuritylevel)                                        | Gibt an, ob eine webfähige Methode oder Eigenschaft für die Skripterstellung sicher ist.                                                                                                                                                                                                                                                |
| [**X500NameFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-x500nameflags)                                              | Gibt die Anzeige-und Codierungs Eigenschaften eines Distinguished Name oder eines relativen Distinguished Name an.                                                                                                                                                                                                               |
| [**X509CertificateEnrollmentContext**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificateenrollmentcontext)        | Gibt die Art der Endentität an, für die das Zertifikat vorgesehen ist.                                                                                                                                                                                                                                            |
| [**X509CertificateTemplateEnrollmentFlag**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificatetemplateenrollmentflag)   | Enthält Werte, die während der Anmeldung Server-und Client Aktionen angeben.                                                                                                                                                                                                                                                |
| [**X509CertificateTemplateGeneralFlag**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificatetemplategeneralflag)         | Enthält Verwendungs-und Änderungs Informationen zu Vorlagen und zugeordneten Zertifikaten.                                                                                                                                                                                                                                   |
| [**X509CertificateTemplatePrivateKeyFlag**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificatetemplateprivatekeyflag)   | Enthält Werte, die Client Aktionen in Bezug auf einen privaten Schlüssel angeben.                                                                                                                                                                                                                                                     |
| [**X509CertificateTemplateSubjectNameFlag**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificatetemplatesubjectnameflag) | Enthält Werte, die Server-und Client Aktionen für Betreffnamen angeben.                                                                                                                                                                                                                                         |
| [**X509EnrollmentPolicyExportFlags**](/windows/desktop/api/Certenroll/ne-certenroll-x509enrollmentpolicyexportflags)               | Gibt an, welche Elemente vom Richtlinien Server exportiert werden sollen.                                                                                                                                                                                                                                                                   |
| [**X509EnrollmentPolicyLoadOption**](/windows/desktop/api/Certenroll/ne-certenroll-x509enrollmentpolicyloadoption)                 | Gibt an, wie Richtlinien vom Richtlinien Server abgerufen werden.                                                                                                                                                                                                                                                                 |
| [**X509KeySpec**](/windows/desktop/api/CertEnroll/ne-certenroll-x509keyspec)                                                  | Gibt die beabsichtigte Verwendung eines Schlüssels für einen Legacy-CSP an.                                                                                                                                                                                                                                                                    |
| [**X509KeyUsageFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-x509keyusageflags)                                      | Gibt den Zweck eines Schlüssels an, der in einem Zertifikat enthalten ist.                                                                                                                                                                                                                                                               |
| [**X509PrivateKeyExportFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-x509privatekeyexportflags)                           | Gibt die Export Richtlinie für einen privaten Schlüssel an.                                                                                                                                                                                                                                                                           |
| [**X509PrivateKeyProtection**](/windows/desktop/api/CertEnroll/ne-certenroll-x509privatekeyprotection)                             | Gibt die Ebene des Schutzes privater Schlüssel an, die von einem KSP oder CSP unterstützt wird.                                                                                                                                                                                                                                                 |
| [**X509PrivateKeyUsageFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-x509privatekeyusageflags)                             | Gibt die zulässigen Verwendungen eines privaten Schlüssels an.                                                                                                                                                                                                                                                                           |
| [**X509PrivateKeyVerify**](/windows/desktop/api/CertEnroll/ne-certenroll-x509privatekeyverify)                                     | Gibt an, ob während der Überprüfung des privaten Schlüssels eine Benutzeroberfläche angezeigt wird und ob die Überprüfung fortgesetzt werden kann, wenn der CSP oder KSP eine Smartcard darstellt                                                                                                                                                           |
| [**X509ProviderType**](/windows/desktop/api/CertEnroll/ne-certenroll-x509providertype)                                        | Gibt den Typ eines Kryptografiedienstanbieters oder Schlüssel Dienstanbieters an.                                                                                                                                                                                                                                          |
| [**X509RequestInheritOptions**](/windows/desktop/api/CertEnroll/ne-certenroll-x509requestinheritoptions)                      | Gibt an, wie Schlüssel, Erweiterungs Werte und externe Eigenschaften geerbt werden, wenn eine neue Anforderung aus einem vorhandenen Zertifikat erstellt wird.                                                                                                                                                                                  |
| [**X509RequestType**](/windows/desktop/api/CertEnroll/ne-certenroll-x509requesttype)                                          | Gibt den zertifikatanforderungstyp an.                                                                                                                                                                                                                                                                                  |
| [**X509SCEPDisposition**](/windows/desktop/api/certpol/ne-certpol-x509scepdisposition)                                       | Beschreibt die resultierende Disposition einer Anforderung zum Verarbeiten einer Antwortnachricht.                                                                                                                                                                                                                                          |
| [**X509SCEPFailInfo**](/windows/desktop/api/certpol/ne-certpol-x509scepfailinfo)                                             | Beschreibt die Art eines Fehlers bei der Registrierung eines SCEP-Zertifikats.                                                                                                                                                                                                                                                          |
