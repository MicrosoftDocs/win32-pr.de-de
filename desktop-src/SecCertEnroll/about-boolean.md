# <a name="boolean-certificate-enrollment-api"></a>BOOLEAN (Zertifikatregistrierungs-API)

Ein boolescher Wert kann **TRUE** oder **FALSE sein.** **FALSE** wird als TLV-Triplet codiert, bei dem das **Feld Wert** null (0) ist. **TRUE** wird als Wert ungleich 0 (null) codiert. Das **Tag** für den **BOOLEAN-Datentyp** ist 0x01.
