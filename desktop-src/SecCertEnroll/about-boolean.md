# <a name="boolean-certificate-enrollment-api"></a>Boolescher Wert (Zertifikatregistrierungs-API)

Ein boolescher Wert kann " **true** " oder " **false**" sein. **False** wird als TLV-dreier-Objekt codiert, bei dem das **Wertfeld** 0 (null) ist. **True** wird als Wert ungleich 0 (null) codiert. Das **Tag** für den **booleschen** Datentyp ist 0x01.
