# <a name="certificate-request-encoding"></a>Zertifikatanforderungscodierung

In einer Microsoft-PKI werden Zertifikatanforderungen von Clientcomputern als binäre Zeichenfolge, die eine Sequenz codierter Datenstrukturen enthält, an Zertifizierungsstellen (Certification Authorities, CAs) gesendet. Die Zertifikatregistrierungs-API verwendet die abstrakte Syntax Notation One (ASN.1), um diese Strukturen zu beschreiben und zu codieren. Für die Zwecke dieser Dokumentation kann ASN.1 konzeptionell in die Syntaxregeln (ISO 8824) unterteilt werden, die die Datenstrukturen und die Codierungsregeln (ISO 8825) beschreiben, die angeben, wie die Daten für die Netzwerkübertragung codiert werden sollen. Weitere Informationen finden Sie in den folgenden Themen:

-   [Einführung in die ASN.1-Syntax und -Codierung](about-introduction-to-asn-1-syntax-and-encoding.md)
-   [ASN.1-Typsystem](about-asn-1-type-system.md)
-   [Distinguished Encoding Rules](distinguished-encoding-rules.md)
