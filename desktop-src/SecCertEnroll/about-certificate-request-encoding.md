# <a name="certificate-request-encoding"></a>Zertifikat Anforderungs Codierung

In einer Microsoft-PKI werden Zertifikat Anforderungen von Client Computern an Zertifizierungsstellen (CAS) als binäre Zeichenfolge gesendet, die eine Sequenz von codierten Datenstrukturen enthält. Die Zertifikatregistrierungs-API verwendet die abstrakte Syntax Notation One (ASN. 1), um diese Strukturen zu beschreiben und zu codieren. Im Rahmen dieser Dokumentation kann ASN. 1 konzeptionell in die Syntax Regeln (ISO 8824) unterteilt werden, die die Datenstrukturen und Codierungsregeln (ISO 8825) beschreiben, die angeben, wie die Daten für die Netzwerkübertragung codiert werden sollen. Weitere Informationen finden Sie unter den folgenden Themen:

-   [Einführung in die ASN. 1-Syntax und-Codierung](about-introduction-to-asn-1-syntax-and-encoding.md)
-   [ASN. 1-Typsystem](about-asn-1-type-system.md)
-   [Distinguished Encoding Rules](distinguished-encoding-rules.md)
