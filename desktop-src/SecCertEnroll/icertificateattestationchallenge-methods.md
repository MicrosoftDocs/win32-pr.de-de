# <a name="icertificateattestationchallenge-methods"></a>ICertificateAttestationC workstationge-Methoden

Die [**ICertificateAttestationCllerge-Schnittstelle unterstützt**](/windows/desktop/api/Certenroll/nn-certenroll-icertificateattestationchallenge) die folgenden Methoden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                           | Beschreibung                                                                                                                                                                                                                                                    |
|-------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DecryptChallenge-Methode**](/windows/desktop/api/Certenroll/nf-certenroll-icertificateattestationchallenge-decryptchallenge)<br/> | Entschlüsselt die Herausforderung aus der CMC-Antwort [(Certificate Management over CMS)](/windows/desktop/SecGloss/c-gly) und erstellt eine neu verschlüsselte Antwort, die an die Zertifizierungsstelle gesendet wird.<br/> |
| [**Initialize-Methode**](/windows/desktop/api/Certenroll/nf-certenroll-icertificateattestationchallenge-initialize)<br/>             | Initialisieren Sie mit der [vollständigen CMC-Antwort (Certificate Management over CMS),](/windows/desktop/SecGloss/c-gly) die von der Zertifizierungsstelle zurückgegeben wurde.<br/>                                        |
