# <a name="icertificateattestationchallenge-methods"></a>Icertifitoreattestationchallenge-Methoden

Die [**icertifitoreattestationchallenge**](/windows/desktop/api/Certenroll/nn-certenroll-icertificateattestationchallenge) -Schnittstelle unterstützt die folgenden Methoden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                           | BESCHREIBUNG                                                                                                                                                                                                                                                    |
|-------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Decryptchallenge-Methode**](/windows/desktop/api/Certenroll/nf-certenroll-icertificateattestationchallenge-decryptchallenge)<br/> | Entschlüsselt die Herausforderung von der [Certificate Management over CMS](/windows/desktop/SecGloss/c-gly) -Antwort (CMC) und erstellt eine erneut verschlüsselte Antwort, die an die Zertifizierungsstelle gesendet wird.<br/> |
| [**Initialize-Methode**](/windows/desktop/api/Certenroll/nf-certenroll-icertificateattestationchallenge-initialize)<br/>             | Initialisieren Sie mithilfe der von der Zertifizierungsstelle zurückgegebenen Antwort auf die vollständige [Zertifikat Verwaltung über CMS](/windows/desktop/SecGloss/c-gly) (CMC).<br/>                                        |
