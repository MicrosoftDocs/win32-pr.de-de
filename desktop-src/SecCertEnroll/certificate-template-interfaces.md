# <a name="certificate-template-interfaces"></a>Zertifikat Vorlagen Schnittstellen

Die folgenden Schnittstellen können verwendet werden, um Zertifikat Anforderungs Vorlagen Programm gesteuert zuzugreifen, zu ändern und zu speichern.



| Schnittstelle                                                                    | BESCHREIBUNG                                                                                   |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**IX509CertificateTemplate**](/windows/desktop/api/Certenroll/nn-certenroll-ix509certificatetemplate)                 | Stellt eine Vorlage für Zertifikat Anforderungen dar.                                                    |
| [**IX509CertificateTemplates**](/windows/desktop/api/Certenroll/nn-certenroll-ix509certificatetemplates)               | Verwaltet eine Auflistung von [**IX509CertificateTemplate**](/windows/desktop/api/Certenroll/nn-certenroll-ix509certificatetemplate) -Objekten. |
| [**IX509CertificateTemplateWritable**](/windows/desktop/api/Certenroll/nn-certenroll-ix509certificatetemplatewritable) | Fügt eine Vorlage in einen Vorlagen Speicher ein oder löscht sie.                                       |
