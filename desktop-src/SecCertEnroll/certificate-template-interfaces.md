# <a name="certificate-template-interfaces"></a>Schnittstellen für Zertifikatvorlagen

Die folgenden Schnittstellen können verwendet werden, um programmgesteuert auf Zertifikatanforderungsvorlagen zu zugreifen, sie zu ändern und zu speichern.



| Schnittstelle                                                                    | BESCHREIBUNG                                                                                   |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**IX509CertificateTemplate**](/windows/desktop/api/Certenroll/nn-certenroll-ix509certificatetemplate)                 | Stellt eine Zertifikatanforderungsvorlage dar.                                                    |
| [**IX509CertificateTemplates**](/windows/desktop/api/Certenroll/nn-certenroll-ix509certificatetemplates)               | Verwaltet eine Auflistung von [**IX509CertificateTemplate-Objekten.**](/windows/desktop/api/Certenroll/nn-certenroll-ix509certificatetemplate) |
| [**IX509CertificateTemplateWritable**](/windows/desktop/api/Certenroll/nn-certenroll-ix509certificatetemplatewritable) | Fügt einer Vorlage eine Vorlage hinzu oder löscht sie aus einem Vorlagenspeicher.                                       |
