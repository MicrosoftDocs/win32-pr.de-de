# <a name="disablerollback-property"></a>DISABLEROLLBACK-Eigenschaft

Durch Festlegen **der DISABLEROLLBACK-Eigenschaft** wird die [Rollbackoption](rollback-installation.md) für die aktuelle Konfiguration deaktiviert. Legen Sie diese Eigenschaft auf 1 fest, um zu verhindern, dass das Installationsprogramm während der Installation ein Rollbackskript generiert und Kopien gelöschter Dateien gespeichert.

## <a name="remarks"></a>Hinweise

Standardmäßig ist ein Rollback aktiviert.
