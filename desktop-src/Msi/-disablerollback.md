# <a name="disablerollback-property"></a>DISABLEROLLBACK (Eigenschaft)

Durch Festlegen der **DISABLEROLLBACK** -Eigenschaft wird die [Rollback](rollback-installation.md) -Option für die aktuelle Konfiguration deaktiviert. Legen Sie diese Eigenschaft auf 1 fest, um zu verhindern, dass das Installationsprogramm während der Installation ein Rollback-Skript erstellt und Kopien gelöschter Dateien speichert.

## <a name="remarks"></a>Bemerkungen

Standardmäßig ist das Rollback aktiviert.
