# <a name="msisfcbypass-table"></a>Msisfcbypass-Tabelle

Die msisfcbypass-Tabelle enthält eine Liste von Dateien, mit denen der Windows-Datei Schutz umgangen werden muss, wenn die Dateien unter Microsoft Windows Me installiert werden.

Die msisfcbypass-Tabelle weist die folgende Spalte auf.



| Spalte | Typ                         | Schlüssel | Nullwerte zulässig |
|--------|------------------------------|-----|----------|
| Datei\_ | [Bezeichner](identifier.md) | J   | N        |



 

## <a name="columns"></a>Spalten

<dl> <dt>

<span id="File_"></span><span id="file_"></span><span id="FILE_"></span>Datei\_
</dt> <dd>

Der Fremdschlüssel für die [Dateitabelle](file-table.md) , der die Datei angibt, die den Windows-Datei Schutz umgehen soll, wenn die Datei unter Windows Me installiert ist.

</dd> </dl>
