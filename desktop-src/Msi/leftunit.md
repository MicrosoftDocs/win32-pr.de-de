# <a name="leftunit-property"></a>LeftUnit-Eigenschaft

Wenn die **LeftUnit-Eigenschaft** festgelegt ist, wird die Einheit links von der Zahl anstatt auf der typischen rechten Seite platziert. Dies ist für Sprachen geeignet, in denen dies das akzeptierte Format ist. Die Benutzeroberfläche verwendet diese Eigenschaft, um die Größe der Dateien und den verfügbaren Speicherplatz auf verschiedenen Volumes anzuzeigen.
