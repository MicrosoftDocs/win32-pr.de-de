# <a name="leftunit-property"></a>Leftunit (Eigenschaft)

Wenn die **leftunit** -Eigenschaft festgelegt ist, wird die Einheit auf der linken Seite der Zahl anstelle der typischen rechten Seite platziert. Dies gilt für Sprachen, in denen dies das akzeptierte Format ist. Die Benutzeroberfläche verwendet diese Eigenschaft, um die Größe der Dateien und den verfügbaren Speicherplatz auf unterschiedlichen Volumes anzuzeigen.
