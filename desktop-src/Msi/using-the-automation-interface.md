# <a name="using-the-automation-interface"></a>Verwenden der Automatisierungsschnittstelle

Für den Zugriff auf die Automatisierung muss die DLL selbst registriert sein. Die COM-ProgID, die zum Erstellen des [**Installations Objekts**](installer-object.md) verwendet wurde, ist **WindowsInstaller. Installer**.

Informationen zur Automatisierung und zum Zugriff auf C++-Bibliotheken finden Sie unter Informationen zur [Automation-Schnittstelle](about-the-automation-interface.md).

Referenzmaterial für die Windows Installer-Objekte finden Sie unter [Automation Interface Reference](automation-interface-reference.md).
