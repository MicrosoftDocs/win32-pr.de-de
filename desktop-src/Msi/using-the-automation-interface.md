# <a name="using-the-automation-interface"></a>Verwenden der Automatisierungsschnittstelle

Für den Zugriff auf die Automatisierung muss die DLL selbst registriert sein. Die COM ProgId, die zum Erstellen des [**Installerobjekts**](installer-object.md) verwendet wird, ist **WindowsInstaller.Installer**.

Informationen zur Automatisierung und zum Zugriff auf C++-Bibliotheken finden Sie unter [Informationen zur Automation-Schnittstelle.](about-the-automation-interface.md)

Referenzmaterial zu den Windows Installer-Objekten finden Sie unter [Referenz zur Automatisierungsschnittstelle.](automation-interface-reference.md)
