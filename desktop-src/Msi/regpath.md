# <a name="regpath"></a>RegPath

Der RegPath-Datentyp ist eine Textzeichenfolge, die einen Registrierungspfad enthält. Registrierungspfade können Eigenschaften enthalten, wie beim [formatierten](formatted.md) Datentyp. Ein RegPath darf nicht mit einem zurücken Schrägstrich () beginnen oder \\ enden. Der Dateischlüssel und $component können in den Pfad eingebettet oder anderen \[ \# Zeichen \] \[ \] vorangehende werden.
