# <a name="sessionlanguage-property"></a>Session.Language (Eigenschaft)

Die **Language-Eigenschaft** des [**Session-Objekts**](session-object.md) ist eine schreibgeschützte Eigenschaft, die die numerische Sprach-ID darstellt, die von der aktuellen Installationssitzung verwendet wird.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Session.Language
```



## <a name="property-value"></a>Eigenschaftswert
