# <a name="dependencymodule-property"></a>Abhängigkeits. Module (Eigenschaft)

Die schreibgeschützte **Modul** Eigenschaft des [**Abhängigkeits Objekts**](dependency-object.md) gibt die ModuleID des Moduls zurück, das von der aktuellen Zeichenfolge in Form eines **BSTR** benötigt wird. Die ModuleID-Eigenschaft ist das gleiche Formular, das in der [ModuleSignature-Tabelle](modulesignature-table.md)verwendet wird.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Dependency.Module
```
