# <a name="msideploymentcompliant-property"></a>MSIDEPLOYMENTCOMPLIANT (Eigenschaft)

Die **MSIDEPLOYMENTCOMPLIANT-Eigenschaft** kann festgelegt werden, um dem Installationsprogramm anzugeben, dass das Paket erstellt und getestet wurde, um die Einhaltung der Benutzerkontensteuerung (User [*Account Control,*](u-gly.md) UAC) in Windows Vista zu erfüllen. Wenn diese Eigenschaft nicht festgelegt ist, bestimmt das Installationsprogramm, ob das Paket mit der UAC auf Windows Vista konform ist.

Weitere Informationen zu UAC und Windows Installer finden Sie unter [Using Windows Installer with UAC](using-windows-installer-with-uac.md) and Guidelines for Packages (Verwenden des Windows Installers mit UAC und Richtlinien für [Pakete).](guidelines-for-packages.md)

**Windows Installer 3.1, Windows Installer 3.0 und Windows Installer 2.0:** Diese Eigenschaft wird ignoriert. Diese Eigenschaft wird nur von Windows Installer 4.0 in Windows Vista verwendet.
