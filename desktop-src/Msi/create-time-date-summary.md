# <a name="create-timedate-summary-property"></a>Erstellen der Eigenschaft "Zeit-/Datumszusammenfassung"

Die **Eigenschaft Create Time/Date Summary (Zeit-/Datumszusammenfassung** erstellen) gibt die Uhrzeit und das Datum an, zu denen ein Autor das Installationspaket, die Transformation oder das Patchpaket erstellt hat.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Version<br/> | Windows Installer 5.0 auf Windows Server 2012, Windows 8, Windows Server 2008 R2 oder Windows 7. Windows Installer 4.0 oder Windows Installer 4.5 auf Windows Server 2008 oder Windows Vista. Windows Installationsprogramm auf Windows Server 2003 oder Windows XP<br/> |
