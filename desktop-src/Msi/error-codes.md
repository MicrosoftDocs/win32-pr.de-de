# <a name="msiexecexe-and-instmsiexe-error-messages-for-developers"></a>MsiExec.exe und InstMsi.exe Fehlermeldungen (für Entwickler)

> [!NOTE]  
> Dieser Artikel richtet sich an Softwareentwickler, die Windows Installer verwenden, um Installationspakete für ihre Anwendungen zu erstellen. Wenn sie während oder nach der Installation oder Deinstallation einer Anwendung Probleme mit Ihrem Computer haben, sollten Sie sich an den Kundensupport für die Software wenden, die Sie installieren oder entfernen möchten. Wenn Sie den Support für ein Microsoft-Produkt benötigen, besuchen Sie unsere Website für technischen Support unter [support.microsoft.com](https://support.microsoft.com).

Diese Fehlercodes werden von den Windows Installer-Funktionen **MsiExec.exe** und **InstMsi.exe** zurückgegeben. Beachten Sie, dass jeder Fehler in "Winerror.h" (z. B. ERROR \_ INVALID \_ DATA, hier enthalten) ebenfalls zurückgegeben werden kann. Weitere Fehlercodes, die vom Windows Installer zurückgegeben werden, finden Sie unter [Windows Installer-Fehlermeldungen.](windows-installer-error-messages.md)
 
> Die Fehlercodes ERROR \_ SUCCESS, ERROR \_ SUCCESS REBOOT \_ \_ INITIATED und ERROR SUCCESS REBOOT \_ REQUIRED sind ein Hinweis auf den \_ \_ Erfolg. Wenn ERROR \_ SUCCESS REBOOT REQUIRED zurückgegeben \_ \_ wird, wurde die Installation erfolgreich abgeschlossen, aber ein Neustart ist erforderlich, um den Installationsvorgang abzuschließen.
