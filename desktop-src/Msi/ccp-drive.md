# <a name="ccp_drive-property"></a>EIGENSCHAFT "DRIVE \_ DRIVE"

Die **EIGENSCHAFTMEDIEN \_ DRIVE** wird auf den Stammpfad des Wechseldatenträgers festgelegt, der von [RMCCPSearch durchsucht werden soll.](rmccpsearch-action.md) Die RMCCPSearch-Aktion verwendet Dateisignaturen, um zu überprüfen, ob qualifizierte Produkte auf einem System installiert sind, bevor eine Upgradeinstallation vorgenommen wird.

Diese Eigenschaft wird in der Regel über die Benutzeroberfläche oder eine benutzerdefinierte Aktion festgelegt. Diese Eigenschaft sollte festgelegt werden, bevor die [RMCCPSearch-Aktion](rmccpsearch-action.md) ausgeführt wird.
