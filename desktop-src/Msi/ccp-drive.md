# <a name="ccp_drive-property"></a>CCP- \_ Laufwerks Eigenschaft

Die Eigenschaft für das **CCP- \_ Laufwerk** wird auf den Stammpfad des Wechsel Datenträgers festgelegt, das von [RMCCPSEARCH](rmccpsearch-action.md)durchsucht werden soll. Die RMCCPSEARCH-Aktion verwendet Datei Signaturen, um zu überprüfen, ob qualifizierte Produkte auf einem System installiert sind, bevor eine Upgrade-Installation durchgeführt wird.

Diese Eigenschaft wird in der Regel über die Benutzeroberfläche oder eine benutzerdefinierte Aktion festgelegt. Diese Eigenschaft muss festgelegt werden, bevor die [RMCCPSEARCH](rmccpsearch-action.md) -Aktion ausgeführt wird.
