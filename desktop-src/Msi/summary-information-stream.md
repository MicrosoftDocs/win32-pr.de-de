# <a name="summary-information-stream"></a>Zusammenfassungs Informationsdaten Strom

Das Material in diesem Abschnitt richtet sich an Entwickler, die eigene Setup Programme schreiben, die mehr über Windows Installer Datenbanktabellen erfahren möchten. Allgemeine Informationen finden Sie unter Informationen [zu Windows Installer](about-windows-installer.md).

Der Datenstrom für Zusammenfassungs Informationen wird vom Installationsprogramm für zwei Zwecke verwendet. Zuerst enthält Sie Informationen über das Paket, das über Microsoft Windows-Explorer angezeigt werden kann. Diese Informationen sind über die **IStream** -Schnittstelle zugänglich. Zweitens enthält Sie Eigenschaften, die vom Installationsprogramm zum Installieren des Pakets verwendet werden. In den folgenden Themen finden Sie Informationen zur Verwendung des Zusammenfassungs Informationsdaten Stroms mit dem Installationsprogramm:

-   [Informationen zum Datenstrom für Zusammenfassungs Informationen](about-the-summary-information-stream.md)
-   [Verwenden des Zusammenfassungs Informationsdaten Stroms](using-the-summary-information-stream.md)
-   [Datenstrom Referenz für Zusammenfassungs Informationen](summary-information-stream-reference.md)
