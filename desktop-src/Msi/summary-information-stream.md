# <a name="summary-information-stream"></a>Stream mit Zusammenfassungsinformationen

Das Material in diesem Abschnitt richtet sich an Entwickler, die eigene Setupprogramme schreiben, die mehr über Windows Installer-Datenbanktabellen erfahren möchten. Allgemeine Informationen finden Sie unter [Informationen Windows Installer.](about-windows-installer.md)

Der Zusammenfassungsinformationsstream wird vom Installationsprogramm für zwei Zwecke verwendet. Zunächst enthält es Informationen zum Paket, die über Microsoft Windows Explorer angezeigt werden können. Auf diese Informationen kann über die **IStream-Schnittstelle** zugegriffen werden. Zweitens enthält sie Eigenschaften, die vom Installationsprogramm zum Installieren des Pakets verwendet werden. Die folgenden Themen enthalten Informationen zur Verwendung des Zusammenfassungsinformationsdatenstroms mit dem Installationsprogramm:

-   [Informationen zum Zusammenfassungsinformationsdatenstrom](about-the-summary-information-stream.md)
-   [Verwenden des Zusammenfassungsinformationsdatenstroms](using-the-summary-information-stream.md)
-   [Streamreferenz zu Zusammenfassungsinformationen](summary-information-stream-reference.md)
