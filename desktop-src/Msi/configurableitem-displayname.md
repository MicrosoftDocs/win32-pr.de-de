# <a name="configurableitemdisplayname-property"></a>Configurableitem. Display Name (Eigenschaft)

Die **DisplayName** -Eigenschaft des [**configurableitem**](configurableitem-object.md) -Objekts gibt den Wert aus der DisplayName-Spalte der [ModuleConfiguration-Tabelle](moduleconfiguration-table.md)zurück.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = ConfigurableItem.DisplayName
```



## <a name="property-value"></a>Eigenschaftswert

## <a name="c"></a>C++

Weitere Informationen finden [**Sie unter Get \_ Display Name**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_displayname) function.
