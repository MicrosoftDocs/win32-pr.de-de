# <a name="configurableitemdisplayname-property"></a>ConfigurableItem.DisplayName-Eigenschaft

Die **DisplayName-Eigenschaft** des [**ConfigurableItem-Objekts**](configurableitem-object.md) gibt den Wert aus der DisplayName-Spalte der [ModuleConfiguration-Tabelle](moduleconfiguration-table.md)zurück.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = ConfigurableItem.DisplayName
```



## <a name="property-value"></a>Eigenschaftswert
