# <a name="leftscroll-dialog-style-bit"></a>Leftscroll-Dialog Feld Stil

Wenn dieses Stilbit festgelegt ist, befindet sich die Schiebe Leiste auf der linken Seite des Dialog Felds.

## <a name="value"></a>Wert



| Decimal | Hexadezimal | Konstante                            |
|---------|-------------|-------------------------------------|
| 512     | 0x00000200  | **msidbdialogattributesleftscroll** |
