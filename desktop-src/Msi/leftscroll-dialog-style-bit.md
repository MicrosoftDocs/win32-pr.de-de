# <a name="leftscroll-dialog-style-bit"></a>LeftScroll Dialog Style Bit

Wenn dieses Formatbit festgelegt ist, befindet sich die Scrollleiste auf der linken Seite des Dialogfelds.

## <a name="value"></a>Wert



| Decimal | Hexadezimal | Konstante                            |
|---------|-------------|-------------------------------------|
| 512     | 0x00000200  | **msidbDialogAttributesLeftScroll** |
