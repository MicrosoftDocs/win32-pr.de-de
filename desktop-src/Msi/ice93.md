# <a name="ice93"></a>ICE93

ICE93 gibt eine Warnung aus, wenn eine benutzerdefinierte Aktion denselben Namen wie eine Standardaktion verwendet. Eine Liste aller Standard Aktions Namen finden Sie unter [Standard Aktionen-Referenz](standard-actions-reference.md).

## <a name="result"></a>Ergebnis

ICE93 gibt die folgende Warnung aus.



| ICE93-Warnung                                                      | BESCHREIBUNG                                                                                                                    |
|--------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| Die benutzerdefinierte Aktion ' \[ 1 \] ' verwendet den gleichen Namen wie eine Standardaktion. | In der Spalte Aktion der Tabelle CustomAction gibt es einen Bezeichner, bei dem es sich um den Namen einer Windows Installer Standardaktion handelt. |
