# <a name="ice93"></a>ICE93

ICE93 gibt eine Warnung aus, wenn eine benutzerdefinierte Aktion den gleichen Namen wie eine Standardaktion verwendet. Eine Liste aller Standardaktionsnamen finden Sie unter [Referenz zu Standardaktionen.](standard-actions-reference.md)

## <a name="result"></a>Ergebnis

ICE93 gibt die folgende Warnung aus.



| ICE93-Warnung                                                      | Beschreibung                                                                                                                    |
|--------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| Die benutzerdefinierte Aktion \[ "1" \] verwendet den gleichen Namen wie eine Standardaktion. | In der Spalte Aktion der CustomAction-Tabelle befindet sich ein Bezeichner, der der Name einer Windows Installer-Standardaktion ist. |
