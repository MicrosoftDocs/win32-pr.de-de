# <a name="version9x-property"></a>Version9X-Eigenschaft

Die **Version9X-Eigenschaft** gibt die Versionsnummer für 9x-Versionen Windows Betriebssystemen an.

Der Wert dieser Eigenschaft ist eine ganze Zahl: MajorVersion \* 100 + MinorVersion. Die Eigenschaft bleibt nicht definiert, wenn das Betriebssystem keine 9x-Version ist. Weitere Informationen finden Sie unter [Betriebssystem-Eigenschaftswerte.](operating-system-property-values.md)

## <a name="remarks"></a>Hinweise

Bedingungsausdrücke können mithilfe des Eigenschaftennamens auf die Version testen oder die Version mithilfe eines Vergleichsoperator überprüfen.

Bei allen Eigenschaftsnamen wird die Kleinschreibung beachtet. Beachten Sie, dass das X im Namen **Version9X** groß geschrieben ist.
