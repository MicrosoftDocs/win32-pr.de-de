# <a name="version9x-property"></a>Version9X-Eigenschaft

Die **Version9X** -Eigenschaft gibt die Versionsnummer für 9X-Versionen von Windows-Betriebssystemen an.

Der Wert dieser Eigenschaft ist eine ganze Zahl: MajorVersion \* 100 + MinorVersion. Die-Eigenschaft bleibt undefiniert, wenn das Betriebssystem keine 9X-Version ist. Weitere Informationen finden Sie unter [Betriebs System-Eigenschaftswerte](operating-system-property-values.md).

## <a name="remarks"></a>Bemerkungen

Bedingungs Ausdrücke können mit dem Eigenschaftsnamen auf die Version testen oder die Version mithilfe eines Vergleichs Operators überprüfen.

Bei allen Eigenschaftsnamen wird Groß-/Kleinschreibung beachtet Beachten Sie, dass das X im Name **Version9X** Großbuchstaben ist.
