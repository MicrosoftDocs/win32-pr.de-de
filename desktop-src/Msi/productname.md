# <a name="productname-property"></a>ProductName-Eigenschaft

Die **ProductName-Eigenschaft** enthält den Namen der zu installierenden Anwendung. Dies wird nur zu Anzeigezwecken verwendet. Wird als Produkteigenschaft angekündigt.

Diese Eigenschaft ist REQUIRED.

Diese Eigenschaft kann durch eine Transformation geändert werden.

## <a name="remarks"></a>Hinweise

Die **ProductName-Eigenschaft** darf nicht größer als 63 Zeichen sein. Die Länge des Registrierungsschlüssels für DisplayName ist nicht begrenzt.
