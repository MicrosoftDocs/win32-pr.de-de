# <a name="productname-property"></a>ProductName-Eigenschaft

Die **ProductName** -Eigenschaft enthält den Namen der zu installierenden Anwendung. Dies wird nur zu Anzeige Zwecken verwendet. Wird als Produkt Eigenschaft angekündigt.

Diese Eigenschaft ist erforderlich.

Diese Eigenschaft kann durch eine Transformation geändert werden.

## <a name="remarks"></a>Bemerkungen

Die **ProductName** -Eigenschaft darf nicht länger als 63 Zeichen sein. Es gibt keine Beschränkung für die Länge des Registrierungsschlüssels für Display Name.
