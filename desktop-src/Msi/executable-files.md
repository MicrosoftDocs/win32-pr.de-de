# <a name="executable-files"></a>Ausführbare Dateien

Eine benutzerdefinierte Aktion kann eine ausführbare Datei starten, die auf dem Computer des Benutzers gespeichert oder im Installationspaket enthalten ist.

Die folgenden Typen von benutzerdefinierten Aktionen starten eine ausführbare Datei.



| Benutzerdefinierter Aktionstyp                                 | Beschreibung                                         |
|----------------------------------------------------|-----------------------------------------------------|
| [Benutzerdefinierter Aktionstyp 2](custom-action-type-2.md)   | EXE-Datei, die in einem Binärtabellenstream gespeichert ist.           |
| [Benutzerdefinierter Aktionstyp 18](custom-action-type-18.md) | EXE-Datei mit einem Produkt installiert.                  |
| [Benutzerdefinierter Aktionstyp 50](custom-action-type-50.md) | EXE-Datei mit einem Pfad, der durch einen Eigenschaftswert angegeben wird. |
| [Benutzerdefinierter Aktionstyp 34](custom-action-type-34.md) | EXE-Datei mit einem Pfad, der auf ein Verzeichnis verweist.       |



 

Im Abschnitt [Zusammenfassungsliste aller benutzerdefinierten Aktionstypen](summary-list-of-all-custom-action-types.md) finden Sie eine Zusammenfassung aller Typen von benutzerdefinierten Aktionen und deren Codierung in der [Tabelle CustomAction.](customaction-table.md)
