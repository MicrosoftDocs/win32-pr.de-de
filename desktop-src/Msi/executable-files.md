# <a name="executable-files"></a>Ausführbare Dateien

Eine benutzerdefinierte Aktion kann eine ausführbare Datei starten, die auf dem Computer des Benutzers gespeichert ist oder im Installationspaket enthalten ist.

Die folgenden Typen von benutzerdefinierten Aktionen starten eine ausführbare Datei.



| Benutzerdefinierter Aktionstyp                                 | BESCHREIBUNG                                         |
|----------------------------------------------------|-----------------------------------------------------|
| [Benutzerdefinierter Aktionstyp 2](custom-action-type-2.md)   | Die exe-Datei, die in einem binären Tabellenstream gespeichert ist.           |
| [Benutzerdefinierter Aktionstyp 18](custom-action-type-18.md) | EXE-Datei, die mit einem Produkt installiert wird.                  |
| [Benutzerdefinierter Aktionstyp 50](custom-action-type-50.md) | EXE-Datei mit einem Pfad, der von einem Eigenschafts Wert angegeben wird. |
| [Benutzerdefinierter Aktionstyp 34](custom-action-type-34.md) | EXE-Datei mit einem Pfad, der auf ein Verzeichnis verweist.       |



 

Eine Zusammenfassung aller Typen von benutzerdefinierten Aktionen und deren Codierung in die [CustomAction-Tabelle](customaction-table.md)finden Sie im Abschnitt [zusammenfassende Liste aller benutzerdefinierten Aktions Typen](summary-list-of-all-custom-action-types.md) .
