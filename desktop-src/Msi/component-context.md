# <a name="componentcontext-property"></a>Component. Context-Eigenschaft

Der Kontext, der für die Anwendung auf die betreffende Komponente bestimmt wurde. Diese Eigenschaft ruft die [**msienumschlag**](/windows/desktop/api/Msi/nf-msi-msienumcomponentsexa) -Funktion auf.

**[Windows Installer 4,5 oder früher](not-supported-in-windows-installer-4-5.md):** Nicht unterstützt. Diese Eigenschaft ist ab Windows Installer 5,0 verfügbar.
