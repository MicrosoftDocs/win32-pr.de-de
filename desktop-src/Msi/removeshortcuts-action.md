# <a name="removeshortcuts-action"></a>Removeshortcuts-Aktion

Mit der removeshortcuts-Aktion wird das Entfernen einer angekündigten Verknüpfung, deren Funktion für die Deinstallation ausgewählt ist, oder eine nicht angekündigte Verknüpfung, deren Komponente für die Deinstallation ausgewählt ist, verwaltet. Weitere Informationen finden Sie in der Verknüpfungs [Tabelle](shortcut-table.md).

## <a name="sequence-restrictions"></a>Sequenz Einschränkungen

Die removeshortcuts-Aktion muss vor der Aktion " [kreateshortcuts](createshortcuts-action.md) " erfolgen.

## <a name="actiondata-messages"></a>Aktions Daten Meldungen



| Feld | Beschreibung der Aktions Daten      |
|-------|---------------------------------|
| \[1\] | Bezeichner der entfernten Verknüpfung. |
