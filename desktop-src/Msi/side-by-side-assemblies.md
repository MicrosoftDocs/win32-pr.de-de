# <a name="side-by-side-assemblies"></a>Nebensassemblys

Die gemeinsame Nutzung von side-by-Side-Assemblys ist eine Infrastruktur, die für folgende Schritte verwendet wird:

-   Sicheres Freigeben von Assemblys für mehrere Anwendungen
-   Offset einiger negativer Auswirkungen der Freigabe, z. B. DLL-Konflikte.

Anstatt über eine einzelne Version einer Assembly zu verfügen, die Abwärtskompatibilität mit allen Anwendungen annimmt, ermöglicht die parallele Assemblyfreigabe die gleichzeitige Ausführung mehrerer Versionen einer COM- oder Win32-Assembly auf einem System. Weitere Informationen finden Sie unter [Isolierte Anwendungen und Nebenassemblys.](../sbscs/isolated-applications-and-side-by-side-assemblies-portal.md)

Nebenassemblys können als [freigegebene Assemblys](shared-assemblies.md) oder als [private Assemblys](private-assemblies.md)installiert werden.

Auf Systemen vor Windows XP sind keine nebenseitigen Assemblys verfügbar.
