# <a name="side-by-side-assemblies"></a>Parallele Assemblys

Die parallele assemblyfreigabe ist eine-Infrastruktur, die für die folgenden Aufgaben verwendet wird:

-   Sicheres Freigeben von Assemblys für mehrere Anwendungen
-   Versetzt einige negative Auswirkungen der Freigabe, z. b. dll-Konflikte, ab.

Anstatt eine einzige Version einer Assembly zu verwenden, die die Abwärtskompatibilität mit allen Anwendungen annimmt, ermöglicht die parallele assemblyfreigabe das gleichzeitige Ausführen mehrerer Versionen einer com-oder Win32-Assembly auf einem System. Weitere Informationen finden Sie unter [isolierte Anwendungen und](../sbscs/isolated-applications-and-side-by-side-assemblies-portal.md)parallele Assemblys.

Parallele Assemblys können als frei [gegebene](shared-assemblies.md) Assemblys oder als [private](private-assemblies.md)Assemblys installiert werden.

Parallele Assemblys sind auf Systemen vor Windows XP nicht verfügbar.
