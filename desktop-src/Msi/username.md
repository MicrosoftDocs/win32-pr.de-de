# <a name="username-property"></a>USERNAME-Eigenschaft

Die **USERNAME-Eigenschaft** ist ein Benutzer, der die Installation vorsteuert.

## <a name="default-value"></a>Standardwert

Der Standardwert kann in die Tabelle Property (Eigenschaft) festgelegt werden. Wenn die **USERNAME-Eigenschaft** nicht festgelegt ist, wird sie automatisch mithilfe von Werten aus der Registrierung festgelegt.

## <a name="remarks"></a>Hinweise

Legen Sie die [**NOUSERNAME-Eigenschaft**](nousername.md) fest, um die automatische Einstellung von USERNAME zu **unterdrücken.**
