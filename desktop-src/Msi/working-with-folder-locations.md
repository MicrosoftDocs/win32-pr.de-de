# <a name="working-with-folder-locations"></a>Arbeiten mit Ordnerspeicherorten

Verwenden Sie die folgenden Funktionen, um die Quelle oder das Ziel eines angegebenen Ordners in der [Verzeichnistabelle](directory-table.md)zu suchen:

-   Rufen Sie den Pfad des angegebenen Quellordners ab, indem Sie die [**MsiGetSourcePath-Funktion**](/windows/desktop/api/Msiquery/nf-msiquery-msigetsourcepatha) aufrufen.
-   Rufen Sie den Pfad des angegebenen Zielordners ab, indem Sie die [**MsiGetTargetPath-Funktion**](/windows/desktop/api/Msiquery/nf-msiquery-msigettargetpatha) aufrufen.
-   Ändern Sie den Zielspeicherort für einen Ordner mit der [**MsiSetTargetPath-Funktion.**](/windows/desktop/api/Msiquery/nf-msiquery-msisettargetpatha)
