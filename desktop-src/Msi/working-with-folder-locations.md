# <a name="working-with-folder-locations"></a>Arbeiten mit Ordner Standorten

Verwenden Sie die folgenden Funktionen, um die Quelle oder das Ziel eines angegebenen Ordners in der [Verzeichnis Tabelle](directory-table.md)zu finden:

-   Rufen Sie den Pfad des angegebenen Quell Ordners ab, indem Sie die [**msigetsourcepath**](/windows/desktop/api/Msiquery/nf-msiquery-msigetsourcepatha) -Funktion aufrufen.
-   Rufen Sie den Pfad des angegebenen Ziel Ordners ab, indem Sie die [**msigettargetpath**](/windows/desktop/api/Msiquery/nf-msiquery-msigettargetpatha) -Funktion aufrufen.
-   Ändern Sie den Zielort für einen Ordner mit der [**msisettargetpath**](/windows/desktop/api/Msiquery/nf-msiquery-msisettargetpatha) -Funktion.
