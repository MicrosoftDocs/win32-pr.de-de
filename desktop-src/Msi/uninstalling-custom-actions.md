# <a name="uninstalling-custom-actions"></a>Deinstallieren von benutzerdefinierten Aktionen

Entwickler von Installationspaketen, die benutzerdefinierte Aktionen verwenden, müssen in jedem Fall bestimmen, wie die benutzerdefinierte Aktion ordnungsgemäß funktioniert, wenn das Paket deinstalliert wird. Der Paketautor muss entscheiden, ob die benutzerdefinierte Aktion bei der Deinstallation ausgeführt werden soll, nicht bei der Deinstallation, oder ob eine alternative benutzerdefinierte Aktion bei der Deinstallation ausgeführt werden soll.
