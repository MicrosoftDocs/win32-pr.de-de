# <a name="uninstalling-custom-actions"></a>Deinstallieren von benutzerdefinierten Aktionen

Entwickler von Installationspaketen, die benutzerdefinierte Aktionen nutzen, müssen in jedem Fall bestimmen, wie die benutzerdefinierte Aktion beim Deinstallieren des Pakets ordnungsgemäß ausgeführt werden soll. Der Paket Ersteller muss entscheiden, ob die benutzerdefinierte Aktion bei der Deinstallation ausgeführt werden soll, nicht bei der Deinstallation ausgeführt werden soll oder eine Alternative benutzerdefinierte Aktion bei der Deinstallation ausgeführt werden soll.
