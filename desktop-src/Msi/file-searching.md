# <a name="file-searching"></a>Dateisuche

Bei der Installation ist es häufig erforderlich, nach einer bestimmten Datei zu suchen. Das Installationsprogramm verfügt über vier standardmäßige Suchaktionen, um diesen Zweck zu erfüllen. Die [AppSearch-Aktion](appsearch-action.md) verwendet die [AppSearch-Tabelle](appsearch-table.md) , um zu bestimmen, ob eine Datei im System eines Benutzers vorhanden ist. Während einer Upgradeinstallation können die [ccpsearch-Aktion](ccpsearch-action.md) und die [RMCCPSEARCH-Aktion](rmccpsearch-action.md) die entsprechenden Tabellen verwenden, um das System des Benutzers auf eine frühere Version eines Produkts zu überprüfen.
