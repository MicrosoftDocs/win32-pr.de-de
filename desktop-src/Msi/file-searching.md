# <a name="file-searching"></a>Dateisuche

Während der Installation ist es häufig erforderlich, nach einer bestimmten Datei zu suchen. Das Installationsprogramm verfügt über vier Standardsuchaktionen, um diesen Zweck zu erfüllen. Die [AppSearch-Aktion](appsearch-action.md) verwendet die [Tabelle AppSearch,](appsearch-table.md) um zu bestimmen, ob eine Datei auf dem System eines Benutzers vorhanden ist. Während einer Upgradeinstallation können die [AKTION INSTRUMENTSEARCH](ccpsearch-action.md) und die [RMCCPSearch-Aktion](rmccpsearch-action.md) ihre entsprechenden Tabellen verwenden, um das System des Benutzers auf eine frühere Version eines Produkts zu überprüfen.
