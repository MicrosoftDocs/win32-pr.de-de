# <a name="licenseagreement-dialog"></a>Dialogfeld "LicenseAgreement"

In diesem modalen Dialogfeld wird der Lizenzvertrag für den Benutzer angezeigt. In der Regel zeigt das Dialogfeld den Text der Vereinbarung mithilfe eines [ScrollableText-Steuerelements](scrollabletext-control.md) und eines Paars von Optionsschaltflächen an, mit denen der Benutzer die Vereinbarung akzeptieren oder ablehnen kann. Aus rechtlichen Gründen empfiehlt es sich, dem Benutzer keine der schaltflächen zu öffnen, die standardmäßig bereits ausgewählt sind. Dies zwingt den Benutzer, eine aktive Auswahl zu treffen. Autoren verwenden häufig ein [RadioButtonGroup-Steuerelement](radiobuttongroup-control.md) für diesen Zweck. Beachten Sie jedoch, dass der Fokus auf einem Dialogfeld erst dann zu einem RadioButtonGroup-Steuerelement bewegt wird, wenn eine der Schaltflächen in der Gruppe ausgewählt wurde.
