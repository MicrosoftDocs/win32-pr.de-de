# <a name="licenseagreement-dialog"></a>Dialog Feld "licensagreement"

In diesem modalen Dialogfeld wird dem Benutzer die Lizenzvereinbarung angezeigt. In der Regel zeigt das Dialogfeld den Text der Vereinbarung mit einem [scrollabletext-Steuer](scrollabletext-control.md) Element und einem Paar Options Schaltflächen an, mit denen der Benutzer die Vereinbarung annehmen oder ablehnen können. Aus rechtlichen Gründen empfiehlt es sich, dass dem Benutzer keine Schaltflächen angezeigt werden, da er bereits standardmäßig ausgewählt ist. Dadurch wird der Benutzer gezwungen, eine aktive Auswahl zu treffen. Autoren verwenden zu diesem Zweck häufig ein [RadioButton Group-Steuer](radiobuttongroup-control.md) Element. Beachten Sie jedoch, dass sich der Fokus auf einem Dialogfeld nicht auf ein RadioButton Group-Steuerelement verschiebt, bis eine der Schaltflächen in der Gruppe ausgewählt wurde.
