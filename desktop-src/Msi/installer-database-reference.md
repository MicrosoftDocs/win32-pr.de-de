# <a name="installer-database-reference"></a>Referenz zur Installer-Datenbank

Um Ihrer Anwendung Installerfunktionen hinzuzufügen, können Sie die Datenbanktabellen Windows Installer und die Datenbankfunktionen des Installationsprogramms verwenden. In den Themen in diesem Abschnitt werden die einzelnen Datenbankfunktionen und Datenbanktabellen beschrieben.

Eine Liste aller Installationsdatenbanktabellen finden Sie unter [Datenbanktabellen](database-tables.md).

Eine Liste der Funktionen, die zum Arbeiten mit Informationen in Installationsdatenbanken verwendet werden, finden Sie unter [Datenbankfunktionen.](database-functions.md)
