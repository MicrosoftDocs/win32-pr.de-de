# <a name="installer-database-reference"></a>Referenz zur Installer-Datenbank

Zum Hinzufügen der Installer-Funktionalität zu Ihrer Anwendung können Sie die Windows Installer-Datenbanktabellen und die Installer-Datenbankfunktionen verwenden. In den Themen in diesem Abschnitt werden die einzelnen Datenbankfunktionen und Datenbanktabellen beschrieben.

Eine Liste aller Installationsdaten Bank Tabellen finden Sie unter [Datenbanktabellen](database-tables.md).

Eine Liste der Funktionen, die für die Arbeit mit Informationen in Installations Datenbanken verwendet werden, finden Sie unter [Datenbankfunktionen](database-functions.md).
