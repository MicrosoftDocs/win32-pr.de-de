# <a name="dependencyversion-property"></a>Dependency.Version-Eigenschaft

Die schreibgeschützte **Version-Eigenschaft** gibt die Version des Moduls zurück, die für die aktuelle Zusammenführung erforderlich ist.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Dependency.Version
```
