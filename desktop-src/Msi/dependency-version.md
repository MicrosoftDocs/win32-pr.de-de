# <a name="dependencyversion-property"></a>Abhängigkeit. Version (Eigenschaft)

Die Eigenschaft schreibgeschützte **Version** gibt die Version des Moduls zurück, das für den aktuellen Merge erforderlich ist.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Dependency.Version
```



## <a name="property-value"></a>Eigenschaftswert

## <a name="c"></a>C++

Siehe [**get \_ Version**](/windows/win32/api/mergemod/nf-mergemod-imsmdependency-get_version) function.
