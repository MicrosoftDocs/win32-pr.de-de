## <a name="orca-documentation"></a>Orca-Dokumentation

[Orca](orca-exe.md) ist ein GUI-.msi-Datei-Editor, der mit dem Windows Installer SDK bereitgestellt wird. Sie bietet Vollzugriff auf die Windows Installer-Datenbanktabellen.

Ein Satz von Hilfedateien wird mit Orca bereitgestellt. Zum Abrufen der orca.exe- und Orca-Dokumentation müssen Sie das Windows Installer SDK herunterladen und installieren, das als Orca.msi bereitgestellt wird. Doppelklicken Sie nach der Installation [des Microsoft Windows Software Development Kit (SDK)](https://developer.microsoft.com/windows/downloads/windows-10-sdk/)auf die Orca.msi, um Orca zu installieren.

> [!Note]  
> Es ist nicht möglich, Orca separat herunterzuladen.

 

Obwohl Orca Zugriff auf alle Features des Windows Installers bietet, ist er nicht dazu gedacht, eine voll funktionsfähige Paketerstellungsumgebung zu ersetzen. In vielen Fällen ist es einfacher, eine Installation des Windows Installers für eine Anwendung mithilfe eines der kommerziellen Tools zur Paketerstellung zu erstellen, die von unabhängigen Softwareanbietern zur Verfügung gestellt werden.

## <a name="windows-installer-software-vendors"></a>Windows Softwarehersteller für Installationsprogramme

Sie können sich über Tools an einen unabhängigen Softwarehersteller wenden, um ein Windows Installer-Paket für eine Anwendung zu erstellen. Diese Tools können eine Paketerstellungsumgebung bereitstellen, die möglicherweise einfacher zu verwenden ist als die im Windows Installer SDK bereitgestellten Tools.
