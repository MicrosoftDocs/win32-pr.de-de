# <a name="last-saved-timedate-summary-property"></a>Zuletzt gespeicherte Zeit-/datzusammenfassungs-Eigenschaft

Die **zuletzt gespeicherte Zeit-/uhrzeitzusammenfassungs-** Eigenschaft gibt den Zeitpunkt an, zu dem das Installationspaket, die Transformation oder das Patchpaket zuletzt geändert wurde.

Anfänglich sollte ein Autor den Wert der **zuletzt gespeicherten Zeit-/datezusammenfassungs-** Eigenschaft auf NULL festlegen, um anzugeben, dass noch keine Änderungen am Paket vorgenommen wurden. Ein Autor sollte dann die **zuletzt gespeicherte Zeit-/uhrzeitzusammenfassungs-** Eigenschaft auf die aktuelle Systemzeit/das aktuelle Datum aktualisieren, wenn eine geänderte Installations Datenbank, eine geänderte Transformation oder ein Patchpaket gespeichert wird.
