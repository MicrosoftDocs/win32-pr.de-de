# <a name="userexit-dialog"></a>Userexit-Dialog Feld

Dieses modale Dialogfeld wird am Ende einer vom Benutzer beendeten Installation angezeigt. Dieses Dialogfeld enthält in der Regel nur Text und eine Schaltfläche **OK** . Der Name dieses Dialog Felds muss in der [Tabelle AdminUISequence](adminuisequence-table.md) und in der [Tabelle InstallUISequence](installuisequence-table.md) mit-2 als Sequenznummer angezeigt werden.
