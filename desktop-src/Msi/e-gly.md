<dl> <dt>

<span id="_msi_elevated_gly"></span><span id="_MSI_ELEVATED_GLY"></span>**erhöht**
</dt> <dd>

Mit System Privilegien ausgeführte Aktionen werden als "Erweitert" bezeichnet.

</dd> <dt>

<span id="_msi_execution_phase_gly"></span><span id="_MSI_EXECUTION_PHASE_GLY"></span>**Ausführungsphase**
</dt> <dd>

Wenn das Installationsprogramm ein Skript für Installationsprogramm Aktionen ausführt. In Microsoft Windows 2000 wird dieser Vorgang vom Installerdienst durchgeführt. In einer verwalteten Anwendung wird das Skript mit System Berechtigungen ausgeführt. Weitere Informationen finden Sie unter [Installationsmechanismus](installation-mechanism.md).

</dd> <dt>

<span id="_msi_execution_script_gly"></span><span id="_MSI_EXECUTION_SCRIPT_GLY"></span>**Ausführungs Skript**
</dt> <dd>

Installationsprogramm Aktionen für eine-Installation. Das Ausführungs Skript wird während der [*Erfassungs Phase*](a-gly.md) der Installation generiert und während der [*Ausführungsphase*](/windows)ausgeführt. Weitere Informationen finden Sie unter [Installationsmechanismus](installation-mechanism.md).

</dd> <dt>

<span id="_msi_external_source_files_gly"></span><span id="_MSI_EXTERNAL_SOURCE_FILES_GLY"></span>**externe Quelldateien**
</dt> <dd>

Quelldateien der zu installierenden Anwendung, die außerhalb der [*MSI-Datei*](m-gly.md)gespeichert werden. Mehrere externe Quelldateien können komprimiert und zusammen in einer CAB- [*Datei*](c-gly.md) gespeichert werden, die im [*Paket*](p-gly.md)enthalten ist.

</dd> <dt>

<span id="_msi_external_user_interface_gly"></span><span id="_MSI_EXTERNAL_USER_INTERFACE_GLY"></span>**externe Benutzeroberfläche**
</dt> <dd>

Die Benutzeroberfläche, die vom Autor eines Installationspakets bereitgestellt wird. Es verwendet nicht die internen Funktionen der Benutzeroberfläche des Installers. Weitere Informationen finden Sie unter Informationen zur [Benutzeroberfläche](about-the-user-interface.md).

</dd> </dl>
