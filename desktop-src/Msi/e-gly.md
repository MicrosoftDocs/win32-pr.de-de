# <a name="e-windows-installer"></a>E (Windows Installer)

<dl> <dt>

<span id="_msi_elevated_gly"></span><span id="_MSI_ELEVATED_GLY"></span>**Erhöhten**
</dt> <dd>

Aktionen, die mit Systemberechtigungen ausgeführt werden, werden als erhöhte Rechte bezeichnet.

</dd> <dt>

<span id="_msi_execution_phase_gly"></span><span id="_MSI_EXECUTION_PHASE_GLY"></span>**Ausführungsphase**
</dt> <dd>

Wenn das Installationsprogramm ein Skript der Installationsprogrammaktionen ausführt. In Microsoft Windows 2000 wird dieser Vorgang vom Installerdienst ausgeführt. In einer verwalteten Anwendung wird das Skript mit Systemberechtigungen ausgeführt. Weitere Informationen finden Sie unter [Installationsmechanismus.](installation-mechanism.md)

</dd> <dt>

<span id="_msi_execution_script_gly"></span><span id="_MSI_EXECUTION_SCRIPT_GLY"></span>**Ausführungsskript**
</dt> <dd>

Installationsprogrammaktionen für eine Installation. Das Ausführungsskript wird während der [*Beschaffungsphase*](a-gly.md) der Installation generiert und während der [*Ausführungsphase*](/windows)ausgeführt. Weitere Informationen finden Sie unter [Installationsmechanismus.](installation-mechanism.md)

</dd> <dt>

<span id="_msi_external_source_files_gly"></span><span id="_MSI_EXTERNAL_SOURCE_FILES_GLY"></span>**Externe Quelldateien**
</dt> <dd>

Quelldateien der installierten Anwendung, die außerhalb der [*.msi Datei*](m-gly.md)gespeichert sind. Mehrere externe Quelldateien können komprimiert und zusammen in einer im [*Paket*](p-gly.md)enthaltenen [*Cab-Datei*](c-gly.md) gespeichert werden.

</dd> <dt>

<span id="_msi_external_user_interface_gly"></span><span id="_MSI_EXTERNAL_USER_INTERFACE_GLY"></span>**Externe Benutzeroberfläche**
</dt> <dd>

Vom Autor eines Installationspakets bereitgestellte Benutzeroberfläche. Es werden nicht die internen Benutzeroberflächenfunktionen des Installationsprogramms verwendet. Weitere Informationen finden Sie unter [Informationen zum Benutzeroberfläche](about-the-user-interface.md).

</dd> </dl>
