# <a name="a-localization-example"></a>Ein Lokalisierungsbeispiel

In diesem Beispiel wird veranschaulicht, wie das Windows Installer-Paket lokalisiert wird, das in [Ein Installationsbeispiel](an-installation-example.md)beschrieben wird. Das ursprüngliche Installationspaket wird von einer englischen in eine französische Version geändert.

Das Lokalisierungsbeispiel weist die folgenden Spezifikationen auf:

-   Die Benutzeroberfläche für die Installation der Anwendung MNP2000 sollte von Englisch in Französisch geändert werden.
-   Die französische Version von MNP2000 enthält Readme.txt und Help.txt Dateien, die in französischer Sprache geschrieben wurden.
-   Die französische Version enthält eine Liste von Reise-Agents in Frankreich, die Tickets für Red ParkUnges bereitstellen können. Diese Liste (als neue Datei Fre.txt bereitgestellt) ist kein Teil der englischen Version.

Das allgemeine Verfahren zum Lokalisieren einer Installation wird im Abschnitt [Lokalisieren eines Windows Installer-Pakets](localizing-a-windows-installer-package.md)beschrieben.

Kopieren Sie die englische Version von MNP2000.msi und alle unter [Planen der Installation](planning-the-installation.md) beschriebenen Quelldateien in einen neuen Ordner. Ändern Sie den Namen des MNP2000.msi im Ordner in MNPFren.msi. In den folgenden Abschnitten ändern Sie diese Kopie, um die Anwendung in Französisch zu lokalisieren.
