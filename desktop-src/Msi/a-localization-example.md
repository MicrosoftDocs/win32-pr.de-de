# <a name="a-localization-example"></a>Ein Lokalisierungsbeispiel

In diesem Beispiel wird veranschaulicht, wie sie das Windows Installer-Paket lokalisieren, das in [Einem Installationsbeispiel beschrieben wird.](an-installation-example.md) Das ursprüngliche Installationspaket wird von einer englischen in eine französische Version geändert.

Das Lokalisierungsbeispiel hat die folgenden Spezifikationen:

-   Die Installationsbenutzeroberfläche für die Anwendung MNP2000 sollte von Englisch in Französisch geändert werden.
-   Die französische Version von MNP2000 enthält Readme.txt und Help.txt, die in französisch geschrieben wurden.
-   Die französische Version enthält eine Liste von Reise-Agents in Frankreich, die Tickets für Red Park Gegen bereitstellen können. Diese Liste (als neue Datei bereitgestellt, Fre.txt) ist kein Teil der englischen Version.

Das allgemeine Verfahren zum Lokalisieren einer Installation wird im Abschnitt Lokalisieren eines Windows [Installer-Pakets beschrieben.](localizing-a-windows-installer-package.md)

Kopieren Sie die englische Version von MNP2000.msi und alle unter Planning the Installation (Planen der [Installation)](planning-the-installation.md) beschriebenen Quelldateien in einen neuen Ordner. Ändern Sie den Namen der MNP2000.msi im Ordner in MNPFren.msi. In den folgenden Abschnitten ändern Sie diese Kopie, um die Anwendung in Französisch zu lokalisieren.
