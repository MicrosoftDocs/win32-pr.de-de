# <a name="a-localization-example"></a>Beispiel für eine Lokalisierung

In diesem Beispiel wird veranschaulicht, wie Sie das Windows Installer Paket lokalisieren, das in [einem Installations Beispiel](an-installation-example.md)beschrieben wird. Das ursprüngliche Installationspaket wird von der englischen Sprachversion in Französisch geändert.

Das Lokalisierungs Beispiel verfügt über die folgenden Spezifikationen:

-   Die Benutzeroberfläche für die Installation der Anwendung MNP2000 sollte von Englisch in Französisch geändert werden.
-   Die französische Version von MNP2000 enthält Readme.txt-und Help.txt Dateien, die in französischer Sprache geschrieben wurden.
-   Die französische Version enthält eine Liste der Reise-Agents in Frankreich, die Tickets in der Red Park-Arena bereitstellen können. Diese Liste (bereitgestellt als neue Datei, Fre.txt) ist kein Teil der englischen Version.

Das allgemeine Verfahren zum Lokalisieren einer Installation wird im Abschnitt [lokalisieren eines Windows Installer Pakets](localizing-a-windows-installer-package.md)beschrieben.

Kopieren Sie die englische Version von MNP2000.msi und alle in [Planen der Installation](planning-the-installation.md) beschriebenen Quelldateien in einen neuen Ordner. Ändern Sie den Namen der MNP2000.msi im Ordner in MNPFren.msi. In den folgenden Abschnitten ändern Sie diese Kopie, um die Anwendung in Französisch zu lokalisieren.
