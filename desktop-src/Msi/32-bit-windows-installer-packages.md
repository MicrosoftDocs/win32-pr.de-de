# <a name="32-bit-windows-installer-packages"></a>32-Bit-Windows Installer Pakete

Ein 32-Bit-Paket besteht nur aus 32-Bit- [Windows Installer Komponenten](windows-installer-components.md) und muss im Feld Plattform der Eigenschaft [**Vorlagen Zusammenfassung**](template-summary.md) den Wert "Intel" eingegeben haben.

Weitere Informationen finden Sie unter [Windows Installer auf 64-Bit-Betriebssystemen](windows-installer-on-64-bit-operating-systems.md) und [64-Bit-Windows Installer Paketen](64-bit-windows-installer-packages.md).
