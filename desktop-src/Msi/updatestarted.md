# <a name="updatestarted-property"></a>UpdateStarted-Eigenschaft

Das Installationsprogramm legt die **UpdateStarted-Eigenschaft** fest, wenn Änderungen am System für diese Installation begonnen haben, einschließlich des Fortsetzens einer angehaltenen Installation.

Benutzeroberflächenbedingungsanweisungen verwenden diesen Wert, um ein Dialogfeld auszuwählen. Wenn diese Eigenschaft festgelegt ist, wird der Benutzer nach einem Fehler und Abbruch gefragt, ob die Wiederherstellung oder später fortgesetzt werden soll.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Version<br/> | Windows Installationsprogramm 5.0 auf Windows Server 2012, Windows 8, Windows Server 2008 R2 oder Windows 7. Windows Installer 4.0 oder Windows Installer 4.5 auf Windows Server 2008 oder Windows Vista. Windows Installationsprogramm auf Windows Server 2003 oder Windows XP. Informationen zu den mindestens erforderlichen Windows Service Packs, die für eine Windows Installer-Version erforderlich sind, finden Sie unter Run-Time [Anforderungen](windows-installer-portal.md) für Windows Installer.<br/> |
