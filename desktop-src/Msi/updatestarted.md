# <a name="updatestarted-property"></a>Updatestarted (Eigenschaft)

Der Installer legt die **updatestarted** -Eigenschaft fest, wenn Änderungen am System für diese Installation begonnen haben, einschließlich des fort Setzens einer angehaltenen Installation.

Bedingungs Anweisungen für die Benutzeroberfläche verwenden diesen Wert, um ein Dialogfeld auszuwählen. Wenn diese Eigenschaft festgelegt ist, wird der Benutzer nach einem Fehler und einem Abbruch aufgefordert, ob er wieder hergestellt oder später fortgesetzt werden soll.

|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Version<br/> | Windows Installer 5,0 unter Windows Server 2012, Windows 8, Windows Server 2008 R2 oder Windows 7. Windows Installer 4,0 oder Windows Installer 4,5 unter Windows Server 2008 oder Windows Vista. Windows Installer unter Windows Server 2003 oder Windows XP. Informationen zu den minimalen Windows-Service Pack, die für eine Windows Installer Version erforderlich sind, finden Sie in den [Windows Installer Run-Time Anforderungen](windows-installer-portal.md) .<br/> |
