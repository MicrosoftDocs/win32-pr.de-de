# <a name="rightaligned-dialog-style-bit"></a>Rightausgerichtete-Dialog Feld Stil

Wenn dieses Stilbit festgelegt ist, wird der Text auf der rechten Seite des Dialog Felds ausgerichtet.

## <a name="value"></a>Wert



| Decimal | Hexadezimal | Konstante                              |
|---------|-------------|---------------------------------------|
| 256     | 0x00000100  | **msidbdialogattributesrightausgerichteten** |
