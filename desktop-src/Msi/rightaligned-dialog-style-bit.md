# <a name="rightaligned-dialog-style-bit"></a>RightAligned Dialog Style Bit

Wenn dieses Formatbit festgelegt ist, wird der Text rechts im Dialogfeld ausgerichtet.

## <a name="value"></a>Wert



| Decimal | Hexadezimal | Konstante                              |
|---------|-------------|---------------------------------------|
| 256     | 0x00000100  | **msidbDialogAttributesRightAligned** |
