# <a name="disablerollback-action"></a>DISABLEROLLBACK-Aktion

Mit der DISABLEROLLBACK-Aktion wird das [Rollback](rollback-installation.md) für den Rest der Installation deaktiviert. Das Rollback ist nur für die Aktionen deaktiviert, die nach der DISABLEROLLBACK-Aktion in den Sequenz Tabellen sequenziert werden. Ein Rollback ist für die gesamte Installation deaktiviert, wenn die DISABLEROLLBACK-Aktion vor der [InstallInitialize-Aktion](installinitialize-action.md)sequenziert wird.

## <a name="sequence-restrictions"></a>Sequenz Einschränkungen

Es gibt keine Sequenz Einschränkungen.

## <a name="actiondata-message"></a>Aktions Daten Nachricht

Es sind keine Aktions Daten Meldungen vorhanden.
