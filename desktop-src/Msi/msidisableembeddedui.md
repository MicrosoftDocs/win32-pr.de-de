# <a name="msidisableembeddedui"></a>Msidisableembeddedui

Wenn diese pro-Computer- [System Richtlinie](system-policy.md) auf 1 festgelegt ist, ist die Funktion zur Verwendung der eingebetteten Benutzeroberfläche auf dem Computer deaktiviert. Der Standardwert ist 0 (null), wodurch die Funktion der eingebetteten Benutzeroberflächen Handler aktiviert wird.

Zum Deaktivieren der eingebetteten Benutzeroberfläche eines einzelnen Pakets können Sie die [**msidisableeeui**](msidisableeeui.md) -Eigenschaft verwenden.

**[Windows Installer 4,0 und früher](not-supported-in-windows-installer-4-0.md):** Nicht unterstützt. Diese Funktion ist ab Windows Installer 4,5 verfügbar.

## <a name="registry-key"></a>Registrierungsschlüssel

**HKEY \_ Software Richtlinien für lokale \_ Computer** \\  \\  \\ **Microsoft** \\ **Windows** \\ **Installer**

## <a name="data-type"></a>Datentyp

**REG \_ DWORD**
