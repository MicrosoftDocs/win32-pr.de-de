# <a name="nethoodfolder-property"></a>Eigenschaft "netthoodfolder"

Die Eigenschaft " **netthoodfolder** " ist der vollständige Pfad zum Ordner "fthood".

|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
