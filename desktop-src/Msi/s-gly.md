<span id="_msi_standard_user_gly"></span><span id="_MSI_STANDARD_USER_GLY"></span>**Standardbenutzer**

<span id="_msi_structured_query_language_gly"></span><span id="_MSI_STRUCTURED_QUERY_LANGUAGE_GLY"></span>**Structured Query Language (SQL)**
</dt> <dd>

Sprache, die zum Verarbeiten von Daten in einer relationalen Datenbank verwendet wird. Sie können SQL verwenden, um Abfragen an die Installer-Datenbank zu stellen. Weitere Informationen finden Sie unter [Arbeiten mit Abfragen.](working-with-queries.md)

</dd> <dt>

<span id="_msi_synchronous_execution_gly"></span><span id="_MSI_SYNCHRONOUS_EXECUTION_GLY"></span>**Synchrone Ausführung**
</dt> <dd>

Wenn das Installationsprogramm auf den Abschluss des separaten Threads einer benutzerdefinierten Aktion wartet, bevor der Thread der aktuellen Installation fortgesetzt wird. Weitere Informationen finden Sie unter [Synchrone und asynchrone benutzerdefinierte Aktionen.](synchronous-and-asynchronous-custom-actions.md)

</dd> <dt>

<span id="_msi_system_policy_using_windows_installer__gly"></span><span id="_MSI_SYSTEM_POLICY_USING_WINDOWS_INSTALLER__GLY"></span>**Systemrichtlinie**
</dt> <dd>

Wird von Systemadministratoren verwendet, um Benutzer- und Computerkonfigurationen von einem zentralen Ort in einem Netzwerk aus zu steuern. Systemrichtlinien geben Registrierungseinstellungen an eine große Anzahl von Computern weiter, ohne dass der Administrator über ausführliche Kenntnisse der Registrierung verfügen muss. Die Systemrichtlinie des Installationsprogramms kann mithilfe des Gruppenrichtlinie Editors (GPE) konfiguriert werden, der in Windows 2000 enthalten ist. Weitere Informationen finden Sie unter [Systemrichtlinie](system-policy.md).

</dd> </dl>
