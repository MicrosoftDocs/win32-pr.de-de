# <a name="clientcontext-property"></a>Client.Context-Eigenschaft

Der Kontext des Produkts. Diese Eigenschaft ruft die [**MsiEnumClientsEx-Funktion**](/windows/desktop/api/Msi/nf-msi-msienumclientsexa) auf.

**[Windows Installer 4.5 oder früher:](not-supported-in-windows-installer-4-5.md)** Wird nicht unterstützt. Diese Eigenschaft ist ab Windows Installer 5.0 verfügbar.
