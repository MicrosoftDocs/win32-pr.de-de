# <a name="ice01"></a>ICE01

ICE01 überprüft, ob der ICE-Mechanismus funktioniert. Dieses ICE verwendet die [**Time-Eigenschaft,**](time.md) um die Zeit zu erhalten, und gibt entweder die Systemzeit oder Keine zurück.

## <a name="result"></a>Ergebnis

ICE01 veröffentlicht eine Meldung, in der die Uhrzeit des Installationsprogramms namens ICE01 angezeigt wird. Dieses ICE sollte niemals einen Fehler zurückgeben.
