# <a name="ice01"></a>ICE01

ICE01 überprüft, ob der ICE-Mechanismus funktioniert. Dieses Eis verwendet die [**time**](time.md) -Eigenschaft, um die Zeit zu erhalten, und gibt entweder die Systemzeit oder keine zurück.

## <a name="result"></a>Ergebnis

ICE01 stellt eine Nachricht bereit, die den Zeitpunkt der Installation des Installers ICE01. Dieses Eis sollte nie einen Fehler zurückgeben.
