# <a name="disablepatch"></a>DisablePatch

Wenn diese computerspezifische [Systemrichtlinie](system-policy.md) auf "1" festgelegt ist, werden keine Patches vom Installationsprogramm angewendet. Diese Richtlinie kann verwendet werden, um Sicherheit in Umgebungen zu bieten, in denen das Patchen eingeschränkt werden muss.

## <a name="registry-key"></a>Registrierungsschlüssel

**HKEY \_ LOCAL \_** \\ **MACHINE-Softwarerichtlinien** \\  \\ **Microsoft** \\ **Windows** \\ **Installer**

## <a name="data-type"></a>Datentyp

**REG \_ DWORD**
