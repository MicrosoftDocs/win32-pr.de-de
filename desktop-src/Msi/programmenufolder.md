# <a name="programmenufolder-property"></a>ProgramMenuFolder-Eigenschaft

Das Installationsprogramm legt die **ProgramMenuFolder-Eigenschaft** auf den vollständigen Pfad des Ordners Programmmenü für den aktuellen Benutzer fest. Wenn ein Profil "Alle Benutzer" vorhanden ist und die [**ALLUSERS-Eigenschaft**](allusers.md) festgelegt ist, wird diese Eigenschaft auf den Ordner im Profil "Alle Benutzer" festgelegt.
