<dl> <dt>

<span id="_msi_transaction_processing_gly"></span><span id="_MSI_TRANSACTION_PROCESSING_GLY"></span>**Transaktionsverarbeitung**
</dt> <dd>

Mindestens ein Vorgang, der zusammen als einzelnes unteilbares Ganzes verarbeitet wird, wird als Transaktion bezeichnet. Alle konstituierenden Vorgänge müssen erfolgreich ausgeführt werden, damit die Transaktion erfolgreich ist. Andernfalls wird für alle Vorgänge ein Rollback auf den ursprünglichen Zustand ausgeführt.

</dd> <dt>

<span id="_msi_transform_gly"></span><span id="_MSI_TRANSFORM_GLY"></span>**Verwandeln**
</dt> <dd>

Vorlage der Unterschiede zwischen zwei [*Installer-Datenbanken,*](i-gly.md) die angewendet werden können, um ähnliche Änderungen in anderen Datenbanken zu erzeugen. Beispielsweise kann eine Lokalisierungstransformation verwendet werden, um die Sprache einer Anwendung zu ändern. Weitere Informationen finden Sie unter [Zusammenführungen und Transformationen.](merges-and-transforms.md)

</dd> <dt>

<span id="_msi_transform_error_condition_flags_gly"></span><span id="_MSI_TRANSFORM_ERROR_CONDITION_FLAGS_GLY"></span>**Transformieren von Fehlerbedingungsflags**
</dt> <dd>

Satz von Eigenschaften, die zum Kennzeichnen der Fehlerbedingungen einer [*Transformation*](/windows)verwendet werden. Weitere Informationen finden Sie unter [**MsiCreateTransformSummaryInfo**](/windows/desktop/api/Msiquery/nf-msiquery-msicreatetransformsummaryinfoa).

</dd> <dt>

<span id="_msi_transform_validation_flags_gly"></span><span id="_MSI_TRANSFORM_VALIDATION_FLAGS_GLY"></span>**Transformieren von Validierungsflags**
</dt> <dd>

Satz von Eigenschaften, mit denen überprüft wird, ob die [*Transformation*](/windows) auf die Datenbank angewendet werden kann. Eine Liste dieser Eigenschaften finden Sie unter [**MsiCreateTransformSummaryInfo**](/windows/desktop/api/Msiquery/nf-msiquery-msicreatetransformsummaryinfoa).

</dd> </dl>
