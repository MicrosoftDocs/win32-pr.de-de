<dl> <dt>

<span id="_msi_transaction_processing_gly"></span><span id="_MSI_TRANSACTION_PROCESSING_GLY"></span>**Transaktionsverarbeitung**
</dt> <dd>

Mindestens ein Vorgang, der zusammen als einzelnes unteilbares Ganzes verarbeitet wird, das als Transaktion bezeichnet wird. Alle konstituierenden Vorgänge müssen erfolgreich sein, damit die Transaktion erfolgreich ist. Andernfalls wird für alle Vorgänge ein Rollback in den ursprünglichen Zustand ausgeführt.

</dd> <dt>

<span id="_msi_transform_gly"></span><span id="_MSI_TRANSFORM_GLY"></span>**Verwandeln**
</dt> <dd>

Vorlage der Unterschiede zwischen zwei [*Installerdatenbanken,*](i-gly.md) die angewendet werden können, um ähnliche Änderungen in anderen Datenbanken zu erzeugen. Beispielsweise kann eine Lokalisierungstransformation verwendet werden, um die Sprache einer Anwendung zu ändern. Weitere Informationen finden Sie unter [Merges und Transformationen.](merges-and-transforms.md)

</dd> <dt>

<span id="_msi_transform_error_condition_flags_gly"></span><span id="_MSI_TRANSFORM_ERROR_CONDITION_FLAGS_GLY"></span>**Fehlerbedingungsflags transformieren**
</dt> <dd>

Satz von Eigenschaften, die verwendet werden, um die Fehlerbedingungen einer Transformation [*zu kennzeichnen.*](/windows) Weitere Informationen finden Sie unter [**MsiCreateTransformSummaryInfo**](/windows/desktop/api/Msiquery/nf-msiquery-msicreatetransformsummaryinfoa).

</dd> <dt>

<span id="_msi_transform_validation_flags_gly"></span><span id="_MSI_TRANSFORM_VALIDATION_FLAGS_GLY"></span>**Transformationsvalidierungsflags**
</dt> <dd>

Satz von Eigenschaften, mit denen überprüft wird, ob [*die Transformation*](/windows) auf die Datenbank angewendet werden kann. Eine Liste dieser Eigenschaften finden Sie unter [**MsiCreateTransformSummaryInfo**](/windows/desktop/api/Msiquery/nf-msiquery-msicreatetransformsummaryinfoa).

</dd> </dl>
