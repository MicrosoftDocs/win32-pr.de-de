# <a name="t-windows-installer"></a>T (Windows Installer)

<dl> <dt>

<span id="_msi_transaction_processing_gly"></span><span id="_MSI_TRANSACTION_PROCESSING_GLY"></span>**Transaktionsverarbeitung**
</dt> <dd>

Mindestens ein Vorgang, der als einzelner unteilbarer gesamter Vorgang (Transaktion) verarbeitet wird. Alle konstituierenden Vorgänge müssen erfolgreich sein, damit die Transaktion erfolgreich ausgeführt werden kann. andernfalls werden alle Vorgänge in den ursprünglichen Zustand zurückgesetzt.

</dd> <dt>

<span id="_msi_transform_gly"></span><span id="_MSI_TRANSFORM_GLY"></span>**Sin**
</dt> <dd>

Vorlage der Unterschiede zwischen zwei [*Installer-Datenbanken*](i-gly.md) , die angewendet werden können, um ähnliche Änderungen in anderen Datenbanken zu erstellen. Beispielsweise kann eine Lokalisierungs Transformation verwendet werden, um die Sprache einer Anwendung zu ändern. Weitere Informationen finden Sie unter Zusammenführungen [und Transformationen](merges-and-transforms.md).

</dd> <dt>

<span id="_msi_transform_error_condition_flags_gly"></span><span id="_MSI_TRANSFORM_ERROR_CONDITION_FLAGS_GLY"></span>**Transformations fehlerbedingungs-Flags**
</dt> <dd>

Ein Satz von Eigenschaften, die zum Markieren der Fehlerbedingungen einer [*Transformation*](/windows)verwendet werden. Weitere Informationen finden Sie unter [**msikreatetransformsummaryinfo**](/windows/desktop/api/Msiquery/nf-msiquery-msicreatetransformsummaryinfoa).

</dd> <dt>

<span id="_msi_transform_validation_flags_gly"></span><span id="_MSI_TRANSFORM_VALIDATION_FLAGS_GLY"></span>**Transformations-Validierungs Flags**
</dt> <dd>

Ein Satz von Eigenschaften, mit denen überprüft wird, ob die [*Transformation*](/windows) auf die Datenbank angewendet werden kann. Eine Liste dieser Eigenschaften finden Sie unter [**msikreatetransformsummaryinfo**](/windows/desktop/api/Msiquery/nf-msiquery-msicreatetransformsummaryinfoa).

</dd> </dl>
