# <a name="publishproduct-action"></a>Publishproduct-Aktion

Die publishproduct-Aktion verwaltet die Ankündigung der Produktinformationen mit dem System. Mit dieser Aktion wird das Produkt veröffentlicht, wenn sich das Produkt im Ankündigungs Modus befindet oder wenn eine Funktion installiert oder neu installiert wird.

## <a name="sequence-restrictions"></a>Sequenz Einschränkungen

Die publishproduct-Aktion muss nach der [publishfeatures](publishfeatures-action.md) -Aktion erfolgen.

## <a name="actiondata-messages"></a>Aktions Daten Meldungen



| Feld | Beschreibung der Aktions Daten        |
|-------|-----------------------------------|
| \[1\] | Bezeichner des angekündigten Produkts. |
