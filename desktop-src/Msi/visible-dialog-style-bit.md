# <a name="visible-dialog-style-bit"></a>Visible Dialog Style Bit

Wenn dieses Bit festgelegt ist, wird das Dialogfeld ursprünglich als sichtbar erstellt, andernfalls wird es ausgeblendet.

## <a name="value"></a>Wert



| Decimal | Hexadezimal | Konstante                         |
|---------|-------------|----------------------------------|
| 1       | 0x00000001  | **msidbDialogAttributesVisible** |
