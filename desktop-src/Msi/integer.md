# <a name="integer"></a>Integer

Der Integer-Datentyp ist ein ganzzahliger Wert mit 2 Bytes. Sofern nicht anders eingeschränkt, liegt der Bereich der zulässigen Werte zwischen – 32.767 und + 32.767.
