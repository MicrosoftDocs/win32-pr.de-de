# <a name="character-count-summary-property"></a>Character Count Summary -Eigenschaft

Die **Eigenschaft Zeichenanzahlzusammenfassung** wird nur in Transformationen verwendet. Dieser Teil des Zusammenfassungsinformationsdatenstroms ist in zwei 16-Bit-Wörter unterteilt. Das obere Wort enthält die [*Transformationsvalidierungsflags*](t-gly.md). Das untere Wort enthält die [*Transformationsfehlerbedingungsflags*](t-gly.md).

Diese Eigenschaft sollte in einem Installations- oder Patchpaket NULL sein.
