# <a name="character-count-summary-property"></a>Zusammenfassungs Eigenschaft für Zeichen Anzahl

Die **Zusammenfassungs** Eigenschaft für die Zeichen Anzahl wird nur in Transformationen verwendet. Dieser Teil des Zusammenfassungs Informationsdaten Stroms ist in 2 16-Bit-Wörter unterteilt. Das obere Wort enthält die [*Transformations-Validierungsflags*](t-gly.md). Das untere Wort enthält die [*Transformations fehlerbedingungs-Flags*](t-gly.md).

Diese Eigenschaft sollte in einem Installationspaket oder Patchpaket NULL sein.
