# <a name="msiwin32assemblysupport-property"></a>MsiWin32AssemblySupport-Eigenschaft

Die **MsiWin32AssemblySupport-Eigenschaft** gibt an, ob der Computer Win32-Assemblys unterstützt. Auf Systemen, die Win32-Assemblys unterstützen, legt das Installationsprogramm den Wert von **MsiWin32AssemblySupport** auf die Dateiversion von sxs.dll fest. Das Installationsprogramm legt diese Eigenschaft nicht fest, wenn das Betriebssystem keine Win32-Assemblys unterstützt. Weitere Informationen finden Sie unter [Assemblys](assemblies.md).

Windows Installer 4.0 oder Windows Installer 4.5 auf Windows Server 2008 oder Windows Vista. Windows Installationsprogramm auf Windows Server 2003 oder Windows XP. Informationen zu den mindestens erforderlichen Windows Service Packs, die für eine Windows Installer-Version erforderlich sind, finden Sie unter Run-Time [Anforderungen](windows-installer-portal.md) für Windows Installer.<br/> |
