# <a name="propertyname-control-attribute"></a>PropertyName-Steuerelement Attribut

Dies ist der Name der Eigenschaft, die diesem Steuerelement zugeordnet ist. Wenn das [indirekte](indirect-control-attribute.md) Bit nicht festgelegt ist, zeigt das Steuerelement den Wert der Eigenschaft mit diesem Namen an oder ändert ihn. Dieses Attribut wird in der-Eigenschaften Spalte der [Steuerelement Tabelle](control-table.md)angegeben. Dieses Attribut gibt NULL für inertsteuerelemente zurück.

## <a name="valid-controls"></a>Gültige Steuerelemente

Alle-Steuerelemente.

## <a name="associated-bit-flags"></a>Zugehörige Bitflags

Dieses Attribut verwendet keine Bitflags.

## <a name="remarks"></a>Bemerkungen

Siehe [Steuerelement Attribute](control-attributes.md) und das Steuerelement, das Sie unter Steuer [Elementen](controls.md)erstellen müssen.
