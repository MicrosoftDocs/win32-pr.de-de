# <a name="recordcleardata-method"></a>Record. ClearData-Methode

Die **ClearData** -Methode des [**Datensatz**](record-object.md) -Objekts löscht die Daten in allen Feldern und legt Sie auf NULL fest. Alle Objekte, die in den Feldern gespeichert sind, werden freigegeben.

## <a name="syntax"></a>Syntax


```JScript
Record.ClearData()
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.
