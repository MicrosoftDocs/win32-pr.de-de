# <a name="listbox-control"></a>ListBox-Steuerelement

Das ListBox-Steuerelement ist ein reguläres Listenfeld, das es dem Benutzer ermöglicht, eine einzelne Auswahl aus einer Liste von vordefinierten Werten zu treffen. Die möglichen Werte werden aus der [ListBox-Tabelle](listbox-table.md)gelesen. Sie können eine Zeichenfolge oder eine ganzzahlige Eigenschaft zuordnen, indem Sie den Eigenschaftsnamen in der Eigenschafts Spalte der [Steuerelement Tabelle](control-table.md)eingeben.

## <a name="control-attributes"></a>Steuerelement Attribute

Mit diesem Steuerelement können Sie die folgenden Attribute verwenden. ## <a name="remarks"></a>Bemerkungen

Dieses Steuerelement kann über die ListBox-Klasse mithilfe der Funktion "up- [**windowex**](/windows/win32/api/winuser/nf-winuser-createwindowexa) " erstellt werden. Sie verfügt über die untergeordneten Stile " **WS \_ Tabstopps**", " **WS \_ Group**" und " **WS \_** ". Wenn das sortierte Steuerelement für das Steuerelement auf ON fest steht, wird das Steuerelement mit den Rahmen Stilen " **lbs \_ Notify**", " **WS \_ VScroll**" und " **WS \_** " erstellt. andernfalls wird das Steuerelement mit dem **lbs- \_ Standard** Stil erstellt
