# <a name="mergemodulecabinet"></a>MergeModule.CABinet

Jede Datei, die vom Mergemodul bereitgestellt wird, muss in einer CAB-Datei gespeichert werden, die als Stream in den strukturierten Speicher des Mergemoduls eingebettet ist. In einem standardmäßigen Mergemodul ist der Name dieses CAB immer: MergeModule.CABinet. Weitere Informationen finden Sie unter [Erstellen von MergeModule.CABinet](generating-mergemodule-cabinet-cabinet-files.md)-CAB-Dateien.
