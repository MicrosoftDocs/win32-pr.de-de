# <a name="table-names"></a>Tabellennamen

Tabellennamen sind auf 31 Zeichen beschränkt. Wenn ein Tabellenname einen Punkt enthält (eine Vorgehensweise, die zulässig ist, aber nicht empfohlen wird), muss der Name in große Akzentzeichen \` (ASCII-Wert 96) eingeschlossen werden, z \` . b. Dateiname. ext \` . Diese Escapesyntax verhindert Konflikte zwischen dem Tabellennamen und den reservierten Wörtern, wie z. b. SQL-Schlüsselwörtern. Weitere Informationen finden Sie unter [SQL-Syntax](sql-syntax.md).
