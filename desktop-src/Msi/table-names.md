# <a name="table-names"></a>Tabellennamen

Tabellennamen sind auf 31 Zeichen beschränkt. Wenn ein Tabellenname einen Zeitraum enthält (eine Vorgehensweise, die zulässig, aber nicht empfohlen wird), muss der Name in Die-Akzent-Zeichen (ASCII-Wert 96) eingeschlossen werden, z. B. \` \` dateiname.ext \` . Diese Escapesyntax verhindert Konflikte zwischen dem Tabellennamen und reservierten Wörtern, z. B. SQL Schlüsselwörtern. Weitere Informationen finden Sie unter [SQL Syntax](sql-syntax.md).
