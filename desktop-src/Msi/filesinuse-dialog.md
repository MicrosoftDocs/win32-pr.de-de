# <a name="filesinuse-dialog"></a>FilesInUse (Dialog Feld)

Ein FilesInUse-Dialogfeld warnt den Benutzer von Prozessen, die derzeit Dateien ausführen, die bei der Installation überschrieben oder gelöscht werden müssen. Dadurch haben Sie die Möglichkeit, diese Prozesse zu beenden und den Computer nicht neu starten zu müssen, um die Dateien zu ersetzen oder zu löschen. Siehe [System Neustarts](system-reboots.md).

Dieses Dialogfeld wird entsprechend der [InstallValidate-Aktion](installvalidate-action.md)erstellt.
