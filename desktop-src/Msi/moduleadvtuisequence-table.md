# <a name="moduleadvtuisequence-table"></a>Moduleadvtuisequence-Tabelle

Erstellen Sie diese Tabelle nicht. Der Windows Installer verwendet nicht die [advtuisequence-Tabelle](advtuisequence-table.md). Die advtuisequence-Tabelle darf nicht vorhanden sein oder sollte in der Installations Datenbank leer gelassen werden. Erstellen Sie auch nicht die Tabelle moduleadvtuisequence.
