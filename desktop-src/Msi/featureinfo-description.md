# <a name="featureinfodescription-property"></a>FeatureInfo. Description (Eigenschaft)

Die **Description** -Eigenschaft ist eine schreibgeschützte Eigenschaft, die die Beschreibung des Features in der Beschreibungs Spalte der [Funktions Tabelle](feature-table.md)zurückgibt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = FeatureInfo.Description
```
