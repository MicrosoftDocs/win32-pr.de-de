# <a name="featureinfodescription-property"></a>FeatureInfo.Description-Eigenschaft

Die **Description-Eigenschaft** ist eine schreibgeschützte Eigenschaft, die die Beschreibung des Features in der Description -Spalte der [Featuretabelle](feature-table.md)zurückgibt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = FeatureInfo.Description
```



## <a name="property-value"></a>Eigenschaftswert
