# <a name="released-versions-tools-and-redistributables"></a>Veröffentlichte Versionen, Tools und verteilbare Dateien

Die folgenden Themen enthalten weitere Informationen zu freigegebenen Versionen, Tools und verteilbaren Komponenten von Windows Installer.

-   In [Windows SDK Komponenten für Windows Installer Entwickler](platform-sdk-components-for-windows-installer-developers.md) wird erläutert, wo Sie die weitervertreibbaren Installer-Komponenten, die Dokumentation, das Daten Bank Überprüfungs Tool, den Datenbanktabellen-Editor, das Datenbankschema, Entwicklungs Tools, VBScript-Tools, Beispiel Produkt und Codebeispiele erhalten.
-   [Windows Installer Redistributables](windows-installer-redistributables.md) enthält Informationen dazu, wo die verteilbaren Installer-Komponenten abgerufen werden.
-   In den [veröffentlichten Versionen von Windows Installer](released-versions-of-windows-installer.md) werden die veröffentlichten Versionen des Installers aufgelistet, die mit unterschiedlichen Betriebssystemen ausgeliefert wurden.
-   [Windows Installer Entwicklungs Tools](windows-installer-development-tools.md) enthält Referenzseiten für die Verwendung der Installer-Entwicklungs Tools.
-   In [Instmsi.exe](instmsi-exe.md) wird beschrieben, welche Instmsi auf welchem Betriebssystem kompatibel ist. Instmsi ist das verteilbare Paket zum Installieren des Installers.
