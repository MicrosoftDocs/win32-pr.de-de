# <a name="changing-the-product-code"></a><span data-ttu-id="35ef8-104">Ändern des Produktcodes</span><span class="sxs-lookup"><span data-stu-id="35ef8-104">Changing the Product Code</span></span>

<span data-ttu-id="35ef8-105">Der Produktcode ist eine GUID, bei der es sich um die Prinzipal Identifizierung einer Anwendung oder eines Produkts handelt.</span><span class="sxs-lookup"><span data-stu-id="35ef8-105">The product code is a GUID that is the principal identification of an application or product.</span></span> <span data-ttu-id="35ef8-106">Siehe [Produkt Codes](product-codes.md).</span><span class="sxs-lookup"><span data-stu-id="35ef8-106">See [Product Codes](product-codes.md).</span></span>

<span data-ttu-id="35ef8-107">Ein Update, das die folgenden Richtlinien erfüllt, erfordert im Allgemeinen keine Änderung des Produktcodes und kann als [kleines Update](small-updates.md)behandelt werden, oder wenn die Version geändert werden soll, als [geringfügige](minor-upgrades.md)Aktualisierung:</span><span class="sxs-lookup"><span data-stu-id="35ef8-107">An update that meets the following guidelines generally does not require a change of the product code and can be handled as a [small update](small-updates.md), or if the version is to change, as a [minor upgrade](minor-upgrades.md):</span></span>

-   <span data-ttu-id="35ef8-108">Das Update kann die Funktionskomponenten Struktur vergrößern oder verkleinern, aber die vorhandene Hierarchie von Features und Komponenten, die in den Tabellen [Feature](feature-table.md) und [FeatureComponents](featurecomponents-table.md) beschrieben werden, darf nicht neu organisiert werden.</span><span class="sxs-lookup"><span data-stu-id="35ef8-108">The update can enlarge or reduce the feature-component tree but it must not reorganize the existing hierarchy of features and components described by the [Feature](feature-table.md) and [FeatureComponents](featurecomponents-table.md) tables.</span></span> <span data-ttu-id="35ef8-109">Der vorhandenen Funktionskomponenten Struktur kann ein neues Feature hinzugefügt werden.</span><span class="sxs-lookup"><span data-stu-id="35ef8-109">It can add a new feature to the existing feature-component tree.</span></span> <span data-ttu-id="35ef8-110">Wenn eine übergeordnete Funktion entfernt wird, müssen auch alle untergeordneten Funktionen der entfernten Funktion entfernt werden.</span><span class="sxs-lookup"><span data-stu-id="35ef8-110">If it removes a parent feature, it must also remove all the child features of the removed feature.</span></span>
-   <span data-ttu-id="35ef8-111">Mit dem Update kann eine neue Komponente zu einem neuen oder vorhandenen Feature hinzugefügt werden.</span><span class="sxs-lookup"><span data-stu-id="35ef8-111">The update can add a new component to a new or an existing feature.</span></span>
-   <span data-ttu-id="35ef8-112">Das Update darf den Komponenten Code einer beliebigen Komponente nicht ändern.</span><span class="sxs-lookup"><span data-stu-id="35ef8-112">The update must not change the component code of any component.</span></span> <span data-ttu-id="35ef8-113">Folglich muss bei einem kleinen Update oder einem geringfügigen Upgrade nie der Name der Schlüsseldatei einer Komponente geändert werden, da dies eine Änderung des Komponenten Codes erfordern würde.</span><span class="sxs-lookup"><span data-stu-id="35ef8-113">Consequently, a small update or minor upgrade must never change the name of a component's key file because this would require changing the component code.</span></span>
-   <span data-ttu-id="35ef8-114">Der Name der MSI-Datei des Installationspakets darf nicht durch das Update geändert werden.</span><span class="sxs-lookup"><span data-stu-id="35ef8-114">The update must not change the name of the .msi file of the installation package.</span></span> <span data-ttu-id="35ef8-115">Stattdessen sollte der Paketcode geändert werden, da er das Paket ändert.</span><span class="sxs-lookup"><span data-stu-id="35ef8-115">Instead, because it modifies the package, it should change the package code.</span></span> <span data-ttu-id="35ef8-116">Beachten Sie, dass das Update die Tabellen, benutzerdefinierten Aktionen und Dialogfelder in der MSI-Datei ändern kann, ohne den Dateinamen zu ändern.</span><span class="sxs-lookup"><span data-stu-id="35ef8-116">Note that this means that the update can change the tables, custom actions, and dialogs in the .msi file without changing the file's name.</span></span>
-   <span data-ttu-id="35ef8-117">Mit dem Update können Dateien, Registrierungsschlüssel oder Verknüpfungen von Komponenten, die nicht von zwei oder mehr Features gemeinsam genutzt werden, hinzugefügt, entfernt oder geändert werden.</span><span class="sxs-lookup"><span data-stu-id="35ef8-117">The update can add, remove, or modify the files, registry keys, or shortcuts of components that are not shared by two or more features.</span></span> <span data-ttu-id="35ef8-118">Wenn das Update eine Datei mit Versions Angabe ändert, muss die Version der Datei in der [Dateitabelle](file-table.md)inkrementiert werden.</span><span class="sxs-lookup"><span data-stu-id="35ef8-118">If the update modifies a versioned file, that file's version must be incremented in the [File table](file-table.md).</span></span> <span data-ttu-id="35ef8-119">Wenn beim Update Ressourcen entfernt werden, sollten auch die Tabellen [RemoveFile](removefile-table.md) und [removeregistry](removeregistry-table.md) aktualisiert werden, um nicht verwendete Dateien, Registrierungsschlüssel oder Verknüpfungen zu entfernen, die bereits installiert wurden.</span><span class="sxs-lookup"><span data-stu-id="35ef8-119">If the update removes resources, it should also update the [RemoveFile](removefile-table.md) and [RemoveRegistry](removeregistry-table.md) tables to remove any unused files, registry keys, or shortcuts that have already been installed.</span></span>
-   <span data-ttu-id="35ef8-120">Das Update einer Komponente, die von zwei oder mehr Features gemeinsam genutzt wird, muss abwärts kompatibel mit allen Anwendungen und Features sein, die die Komponente verwenden.</span><span class="sxs-lookup"><span data-stu-id="35ef8-120">The update of a component that is shared by two or more features must be backward compatible with all applications and features that use the component.</span></span> <span data-ttu-id="35ef8-121">Das Update kann die Ressource einer freigegebenen Komponente (z. b. Dateien, Registrierungseinträge und Verknüpfungen) ändern, solange die Änderungen abwärts kompatibel sind.</span><span class="sxs-lookup"><span data-stu-id="35ef8-121">The update can modify the resource of a shared component, such as files, registry entries, and shortcuts, as long as the changes are backward compatible.</span></span> <span data-ttu-id="35ef8-122">Es wird nicht empfohlen, Dateien, Registrierungseinträge oder Verknüpfungen aus einer freigegebenen Komponente hinzuzufügen oder zu entfernen.</span><span class="sxs-lookup"><span data-stu-id="35ef8-122">It is not recommended that the update add or remove files, registry entries, or shortcuts from a shared component.</span></span>
-   <span data-ttu-id="35ef8-123">Ein kleines Update wird als Windows Installer [Patchpaket](patch-packages.md)ausgeliefert.</span><span class="sxs-lookup"><span data-stu-id="35ef8-123">A small update is shipped as a Windows Installer [patch package](patch-packages.md).</span></span> <span data-ttu-id="35ef8-124">(Eine vollständige Produkt-CD-ROM wird in der Regel nicht mit einem kleinen Update bereitgestellt.)</span><span class="sxs-lookup"><span data-stu-id="35ef8-124">(A full product CD-ROM is usually not provided with a small update.)</span></span>

<span data-ttu-id="35ef8-125">Der Produktcode muss geändert werden, wenn für das Update Folgendes zutrifft:</span><span class="sxs-lookup"><span data-stu-id="35ef8-125">The product code must be changed if any of the following are true for the update:</span></span>

-   <span data-ttu-id="35ef8-126">Vorhandene Installationen von ursprünglichen und aktualisierten Produkten auf demselben System müssen möglich sein.</span><span class="sxs-lookup"><span data-stu-id="35ef8-126">Coexisting installations of both original and updated products on the same system must be possible.</span></span>
-   <span data-ttu-id="35ef8-127">Der Name der MSI-Datei wurde geändert.</span><span class="sxs-lookup"><span data-stu-id="35ef8-127">The name of the .msi file has been changed.</span></span>
-   <span data-ttu-id="35ef8-128">Der Komponenten Code einer vorhandenen Komponente hat sich geändert.</span><span class="sxs-lookup"><span data-stu-id="35ef8-128">The component code of an existing component has changed.</span></span>
-   <span data-ttu-id="35ef8-129">Eine Komponente wird aus einer vorhandenen Funktion entfernt.</span><span class="sxs-lookup"><span data-stu-id="35ef8-129">A component is removed from an existing feature.</span></span>
-   <span data-ttu-id="35ef8-130">Eine vorhandene Funktion wurde in einem untergeordneten Element eines vorhandenen Features erstellt.</span><span class="sxs-lookup"><span data-stu-id="35ef8-130">An existing feature has been made into a child of an existing feature.</span></span>
-   <span data-ttu-id="35ef8-131">Eine vorhandene untergeordnete Funktion wurde aus der übergeordneten Funktion entfernt.</span><span class="sxs-lookup"><span data-stu-id="35ef8-131">An existing child feature has been removed from its parent feature.</span></span>

<span data-ttu-id="35ef8-132">Beachten Sie, dass das Ändern des Produktcodes nicht erforderlich ist, wenn Sie ein neues untergeordnetes Feature hinzufügen, das ausschließlich aus neuen Komponenten besteht.</span><span class="sxs-lookup"><span data-stu-id="35ef8-132">Note that adding a new child feature, consisting entirely of new components, to an existing feature does not require changing the product code.</span></span>

<span data-ttu-id="35ef8-133">Neue untergeordnete Funktionen können erstellt werden, indem "msidbfeatureattributesfollowparent" und "msidbfeatureattributesuidisallowmissing" in das Feld "Attribute" der [Funktions Tabelle](feature-table.md)eingeschlossen werden.</span><span class="sxs-lookup"><span data-stu-id="35ef8-133">New child features can be authored by including msidbFeatureAttributesFollowParent and msidbFeatureAttributesUIDisallowAbsent in the Attributes field of the [Feature table](feature-table.md).</span></span> <span data-ttu-id="35ef8-134">Wenn beim geringfügigen Upgrade nur neue untergeordnete Funktionen hinzugefügt werden, ist REINSTALL = ALL ausreichend, um die Installation der neuen untergeordneten Funktionen zu erzwingen.</span><span class="sxs-lookup"><span data-stu-id="35ef8-134">If the minor upgrade only adds new child features, then REINSTALL=ALL is sufficient to force the installation of the new child features.</span></span> <span data-ttu-id="35ef8-135">Weitere Informationen finden Sie unter [Steuern von Funktionsauswahl Zuständen](controlling-feature-selection-states.md).</span><span class="sxs-lookup"><span data-stu-id="35ef8-135">For more information, see [Controlling Feature Selection States](controlling-feature-selection-states.md).</span></span>

<span data-ttu-id="35ef8-136">Ein neues untergeordnetes Feature ist möglicherweise für den Benutzer ausgeblendet.</span><span class="sxs-lookup"><span data-stu-id="35ef8-136">A new child feature may be hidden from the user.</span></span> <span data-ttu-id="35ef8-137">Um den Installationsstatus einer neuen untergeordneten Funktion mit ihrem übergeordneten Feature zu synchronisieren, legen Sie die Bits "msidbfeatureattributesfollowparent" und "msidbfeatureattributesuidisallowmissing" für die untergeordnete Funktion fest.</span><span class="sxs-lookup"><span data-stu-id="35ef8-137">To synchronize the installation state of a new child feature with its parent feature, set the msidbFeatureAttributesFollowParent and msidbFeatureAttributesUIDisallowAbsent bits for the child feature.</span></span>
