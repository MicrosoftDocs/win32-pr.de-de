# <a name="remoteadmints-property"></a>Remoteadmints (Eigenschaft)

Der Installer legt die **remoteadmints** -Eigenschaft fest, wenn das System mithilfe von Windows-Terminal Diensten f端r die Remote Verwaltung konfiguriert ist. Beachten Sie, dass das Installationsprogramm in diesem Fall die Eigenschaft [**Terminalserver**](terminalserver.md) nicht festgelegt hat.

## <a name="default-value"></a>Standardwert

Keine. Die-Eigenschaft ist nicht definiert, wenn Sie nicht auf einem Remote Verwaltungs Server mit Terminal Diensten ausgef端hrt wird.
