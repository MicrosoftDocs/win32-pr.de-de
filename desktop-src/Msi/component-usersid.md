# <a name="componentusersid-property"></a>Component. UserSID (Eigenschaft)

Die Benutzer-SID für die aufgelistete Komponente. Diese Eigenschaft ruft die [**msienumschlag**](/windows/desktop/api/Msi/nf-msi-msienumcomponentsexa) -Funktion auf.

**[Windows Installer 4,5 oder früher](not-supported-in-windows-installer-4-5.md):** Nicht unterstützt. Diese Eigenschaft ist ab Windows Installer 5,0 verfügbar.
