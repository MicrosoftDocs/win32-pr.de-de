# <a name="ice38"></a><span data-ttu-id="0f5ab-103">ICE38</span><span class="sxs-lookup"><span data-stu-id="0f5ab-103">ICE38</span></span>

<span data-ttu-id="0f5ab-104">ICE38 überprüft, ob jede Komponente, die unter dem Profil des aktuellen Benutzers installiert wird, auch einen Registrierungsschlüssel unter dem **aktuellen HKEY- \_ \_ Benutzer** Stamm in der KEYPATH-Spalte der [Komponenten Tabelle](component-table.md)angibt.</span><span class="sxs-lookup"><span data-stu-id="0f5ab-104">ICE38 validates that every component being installed under the current user's profile also specifies a registry key under the **HKEY\_CURRENT\_USER** root in the KeyPath column of the [Component table](component-table.md).</span></span>

## <a name="result"></a><span data-ttu-id="0f5ab-105">Ergebnis</span><span class="sxs-lookup"><span data-stu-id="0f5ab-105">Result</span></span>

<span data-ttu-id="0f5ab-106">ICE38 gibt einen Fehler aus, wenn eine Komponente, die im Profil des Benutzers installiert ist, keinen HKCU-Registrierungsschlüssel angibt.</span><span class="sxs-lookup"><span data-stu-id="0f5ab-106">ICE38 posts an error if a component installed under the user's profile does not specify a HKCU registry key.</span></span>
