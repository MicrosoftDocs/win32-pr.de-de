# <a name="configurableitemcontext-property"></a>Configurableitem. Context-Eigenschaft

Die **context** -Eigenschaft gibt den Wert aus der Kontext Spalte der [ModuleConfiguration-Tabelle](moduleconfiguration-table.md)zurück.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = ConfigurableItem.Context
```



## <a name="property-value"></a>Eigenschaftswert

## <a name="c"></a>C++

Siehe [**\_ Kontext**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_context) Funktion anzeigen
