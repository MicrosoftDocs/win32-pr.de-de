<dl> <dt>

<span id="_msi_reduced_ui_gly"></span><span id="_MSI_REDUCED_UI_GLY"></span>**Reduzierte Benutzeroberfläche**
</dt> <dd>

Ebene der internen [*Benutzeroberflächenfunktionen des Installationsprogramms.*](i-gly.md) Zeigt nur die erstellungsmoduslosen Dialogfelder an. Weitere Informationen finden Sie unter [Benutzeroberfläche Levels](user-interface-levels.md).

</dd> <dt>

<span id="_msi_reinstall_gly"></span><span id="_MSI_REINSTALL_GLY"></span>**Installieren**
</dt> <dd>

Teilweise oder vollständige Neuinstallation einer installierten Anwendung. Wird häufig durchgeführt, um eine Anwendung zu reparieren, wenn Dateien oder Registrierungseinträge beschädigt wurden oder fehlen. Weitere Informationen finden Sie unter [Neuinstallation eines Features oder einer Anwendung.](reinstalling-a-feature-or-application.md)

</dd> <dt>

<span id="_msi_resiliency_gly"></span><span id="_MSI_RESILIENCY_GLY"></span>**Ausfallsicherheit**
</dt> <dd>

Anwendungsfunktion zur ordnungsgemäßen Wiederherstellung nach Einem Fehler ohne Generierung von Fehlermeldungen. Weitere Informationen finden Sie unter [Resilienz.](resiliency.md)

</dd> <dt>

<span id="_msi_rollback_gly"></span><span id="_MSI_ROLLBACK_GLY"></span>**Rollback**
</dt> <dd>

Automatische Wiederherstellung des ursprünglichen Zustands des Computers, falls bei der Installation ein Fehler auftauen sollte. Weitere Informationen finden Sie unter [Rollbackinstallation (Windows Installer).](rollback-installation.md)

</dd> </dl>
