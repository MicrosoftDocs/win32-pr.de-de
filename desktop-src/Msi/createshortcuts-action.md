# <a name="createshortcuts-action"></a>Aktion "kreateshortcuts"

Mit der Aktion "Aktion erstellen" wird die Erstellung von Verknüpfungen verwaltet.

## <a name="sequence-restrictions"></a>Sequenz Einschränkungen

Die Aktion "Aktion durchsetzen" muss nach der Aktion " [InstallFiles](installfiles-action.md) " und der Aktion " [removeshortcuts](removeshortcuts-action.md) " erfolgen.

## <a name="actiondata-messages"></a>Aktions Daten Meldungen



| Feld | Beschreibung der Aktions Daten      |
|-------|---------------------------------|
| \[1\] | Bezeichner der erstellten Verknüpfung. |



 

## <a name="remarks"></a>Bemerkungen

Mit der Aktion "| ateshortcuts" werden Verknüpfungen zu den Schlüsseldateien von Komponenten von Features erstellt, die für die Installation oder Ankündigung ausgewählt werden.
