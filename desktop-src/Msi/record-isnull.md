# <a name="recordisnull-property"></a>Record. IsNull (Eigenschaft)

Die **IsNull** -Eigenschaft des [**Datensatz**](record-object.md) -Objekts ist eine schreibgeschützte Eigenschaft, die true zurückgibt, wenn das Feld NULL ist, und false, wenn das Feld Daten enthält.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Record.IsNull
```



## <a name="property-value"></a>Eigenschaftswert

Erforderliche Feldnummer des Werts im Datensatz, 1-basiert.
