# <a name="using-windows-installer-with-wmi"></a>Verwenden von Windows Installer mit WMI

Der Windows Installer-Anbieter ermöglicht Windows-Verwaltungsinstrumentation (WMI) den Zugriff auf Informationen, die von Windows Installer Anwendungen gesammelt wurden.

Der [Windows Installer-Anbieter](/previous-versions/windows/desktop/msiprov/windows-installer-provider) ist eine optionale Windows-Komponente. Durch die optionale Installation des Windows Installer Anbieters wird die Abwärtskompatibilität sichergestellt, aber es wird nicht auf die zukünftige Verfügbarkeit des Anbieters hingewiesen. Weitere Informationen zur Verfügbarkeit des Windows Installer Anbieters finden Sie unter [Betriebs System Verfügbarkeit von WMI-Komponenten](../wmisdk/operating-system-availability-of-wmi-components.md) in der [Windows-Verwaltungsinstrumentation-Dokumentation (WMI)](../wmisdk/wmi-start-page.md) .

Die aktuelle Beschreibung dieses WMI-Anbieters finden Sie unter [Windows Installer Provider](/previous-versions/windows/desktop/msiprov/windows-installer-provider) in der Haupt [Windows-Verwaltungsinstrumentation Dokumentation (WMI)](../wmisdk/wmi-start-page.md) . Eine Liste der WMI-Klassen, die von Windows Installer unterstützt werden, finden Sie unter [installierte Anwendungs Klassen](/previous-versions//aa390887(v=vs.85)).
