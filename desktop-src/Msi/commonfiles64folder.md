# <a name="commonfiles64folder-property"></a>CommonFiles64Folder(Eigenschaft)

Das Installationsprogramm legt die **CommonFiles64Folder-Eigenschaft** auf den vollständigen Pfad des vordefinierten 64-Bit-Ordners Common Files fest. Die vorhandene [**CommonFilesFolder-Eigenschaft**](commonfilesfolder.md) wird auf den entsprechenden 32-Bit-Ordner festgelegt.

## <a name="remarks"></a>Hinweise

Das Installationsprogramm legt diese Eigenschaft auf 64-Bit-Windows. Diese Eigenschaft wird nicht auf 32-Bit-Windows. Wenn Sie 64-Bit-Windows, kann der Wert "C: \\ Programme \\ Common Files" sein.
