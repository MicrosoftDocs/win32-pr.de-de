# <a name="commonfiles64folder-property"></a>CommonFiles64Folder-Eigenschaft

Der Installer legt die **CommonFiles64Folder** -Eigenschaft auf den vollständigen Pfad des vordefinierten 64-Bit-Ordners für allgemeine Dateien fest. Die vorhandene [**CommonFilesFolder**](commonfilesfolder.md) -Eigenschaft wird auf den entsprechenden 32-Bit-Ordner festgelegt.

## <a name="remarks"></a>Bemerkungen

Das Installationsprogramm legt diese Eigenschaft auf 64-Bit-Fenstern fest. Diese Eigenschaft wird auf 32-Bit-Fenstern nicht verwendet. Bei Verwendung von 64-Bit-Fenstern ist der Wert möglicherweise "C: \\ Dateien für \\ Allgemeine Dateien".
