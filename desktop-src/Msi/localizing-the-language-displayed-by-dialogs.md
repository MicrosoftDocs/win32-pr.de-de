# <a name="localizing-the-language-displayed-by-dialogs"></a>Lokalisieren der von Dialogen angezeigten Sprache

Der Windows Installer-Dienst verwendet die Sprache des aktuellen Benutzers in allen Dialogfeldern, bis ein Windows Installer-Paket geöffnet wird. Dies wird vom Windows Installer mit **getUserDefaultUILanguage** bestimmt. Wenn ein Windows Installer-Paket geöffnet wird, zeigt das Installationsprogramm Dialoge mit der Sprache des Pakets an, wie in der [**Vorlagenzusammenfassungseigenschaft**](template-summary.md) angegeben.

Weitere Informationen zum Lokalisieren der Sprache eines Windows Installer-Pakets finden Sie auch unter [Ein Lokalisierungsbeispiel.](a-localization-example.md)
