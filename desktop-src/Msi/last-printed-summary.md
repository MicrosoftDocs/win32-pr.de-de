# <a name="last-printed-summary-property"></a>Letzte gedruckte Zusammenfassungs Eigenschaft

In einem Installationspaket kann die **zuletzt gedruckte Zusammenfassungs** Eigenschaft auf das Datum und die Uhrzeit während einer [administrativen Installation](administrative-installation.md) festgelegt werden, um aufzuzeichnen, wann das administrative Image erstellt wurde. Bei nicht administrativen Installationen ist diese Eigenschaft mit der Eigenschaft [**Create time/date Summary**](create-time-date-summary.md) identisch.

In einer Transformation sollte diese Eigenschaft NULL sein.

In einem Patchpaket sollte diese Eigenschaft NULL sein.
