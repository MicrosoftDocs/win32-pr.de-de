# <a name="using-installer-functions"></a>Verwenden von Installerfunktionen

Um Ihrer Anwendung Windows Installer-Funktionalität hinzuzufügen, verwenden Sie die Installer-Dienstfunktionen. Da es sich beim Installationsprogramm um ein Setuphilfsprogramm und ein Komponentenverwaltungssystem handelt, muss eine Anwendung, die das Installationsprogramm verwendet, die Funktionen des Installationsprogrammdiensts im Rahmen des normalen Betriebs integrieren.

In den folgenden Themen und Verfahren wird erläutert, wie Sie Installationsprogrammfunktionen in Ihrer Anwendung implementieren:

-   [Initialisieren einer Anwendung](initializing-an-application.md)
-   [Abrufen von Anwendungsinformationen](getting-application-information.md)
-   [Anfordern eines Features](requesting-a-feature.md)
-   [Installieren einer Anwendung](installing-an-application.md)
-   [Erneutes Installieren eines Features oder einer Anwendung](reinstalling-a-feature-or-application.md)
-   [Entfernen einer angekündigten Anwendung](removing-an-advertised-application.md)
-   [Arbeiten mit Features und Komponenten](working-with-features-and-components.md)
-   [Installieren einer fehlenden Komponente](installing-a-missing-component.md)
-   [Inventurprodukte und Patches](inventory-products-and-patches-.md)
-   [Verwalten von Installationsquellen](managing-installation-sources.md)
