# <a name="recordfieldcount-property"></a>Record.FieldCount-Eigenschaft

Die **FieldCount-Eigenschaft** des [**Record-Objekts**](record-object.md) ist eine schreibgeschützte Eigenschaft, die die Anzahl der Felder im Datensatz zurückgibt. Der Lesezugriff auf Felder, die über diese Anzahl hinausgehen, gibt NULL-Werte zurück. Fehler beim Schreibzugriff.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Record.FieldCount
```
