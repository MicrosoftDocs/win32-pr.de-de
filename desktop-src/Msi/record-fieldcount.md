# <a name="recordfieldcount-property"></a>Record. FieldCount (Eigenschaft)

Die **FieldCount** -Eigenschaft des [**Datensatz**](record-object.md) -Objekts ist eine schreibgeschützte Eigenschaft, die die Anzahl der Felder im Datensatz zurückgibt. Lesezugriff auf Felder über diese Anzahl hinaus gibt NULL-Werte zurück. Schreibzugriff schlägt fehl.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Record.FieldCount
```
