# <a name="authoring-merge-module-summary-information-streams"></a>Erstellen von Zusammenfassungsinformationen zum Mergemodul Streams

Beim Erstellen eines Mergemoduls ist der Zusammenfassungsinformationsstream erforderlich, um diese erforderlichen Zusammenfassungsinformationseigenschaften ein include.

Jedes Mergemodul muss über einen Zusammenfassungsinformationsstream des Mergemoduls mit den folgenden Zusammenfassungsinformationseigenschaften verfügen:

-   [**Vorlagenzusammenfassung**](template-summary.md)
-   [**Zusammenfassung der Revisionsnummer**](revision-number-summary.md)
-   [**Zusammenfassung der Seitenanzahl**](page-count-summary.md)
-   [**Zusammenfassung der Wortanzahl**](word-count-summary.md)

Eine vollständige Liste der Zusammenfassungsinformationseigenschaften in Mergemodulen finden Sie unter [Merge Module Summary Information Stream Reference](merge-module-summary-information-stream-reference.md).
