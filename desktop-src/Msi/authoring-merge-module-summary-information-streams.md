# <a name="authoring-merge-module-summary-information-streams"></a>Erstellen von Zusammenfassungs Datenströmen für Zusammenfassungs Module

Beim Erstellen eines Mergemoduls ist der Datenstrom für Zusammenfassungs Informationen erforderlich, um die erforderlichen Eigenschaften für Zusammenfassungs Informationen einzubeziehen.

Jedes Mergemodul muss über einen Zusammenfassungs Informationsdaten Strom für Zusammenfassungs Module mit den folgenden Eigenschaften für Zusammenfassungs Informationen verfügen:

-   [**Vorlagen Zusammenfassung**](template-summary.md)
-   [**Zusammenfassung der Revisionsnummer**](revision-number-summary.md)
-   [**Zusammenfassung der Seitenanzahl**](page-count-summary.md)
-   [**Zusammenfassung der Wort Zählung**](word-count-summary.md)

Eine umfassende Liste der Eigenschaften für Zusammenfassungs Informationen in Mergemodulen finden Sie unter [Zusammenfassungs Datenstrom Referenz für Zusammenfassungs Modul](merge-module-summary-information-stream-reference.md).
