# <a name="icem01"></a>ICEM01

ICEM01 überprüft, ob der ICE-Mechanismus funktioniert. Dieser ICE verwendet die [**Time-Eigenschaft,**](time.md) um die Zeit abzurufen, und gibt entweder die Systemzeit oder Keine zurück.

Mergemodul-ICEs werden in einer Mergemodul-CUB-Datei namens Mergemod.cub und nicht in der CUB-Datei gespeichert, die die für die Paketvalidierung verwendeten ICEs enthält.

## <a name="result"></a>Ergebnis

ICEM01 sendet eine Nachricht, die den Zeitpunkt angibt, zu dem das Installationsprogramm ICEM01 aufgerufen hat. Dieser ICE sollte nie einen Fehler zurückgeben.
