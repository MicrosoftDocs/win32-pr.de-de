# <a name="icem01"></a>ICEM01

ICEM01 überprüft, ob der ICE-Mechanismus funktioniert. Dieses Eis verwendet die [**time**](time.md) -Eigenschaft, um die Zeit zu erhalten, und gibt entweder die Systemzeit oder keine zurück.

## <a name="result"></a>Ergebnis

ICEM01 stellt eine Nachricht bereit, die den Zeitpunkt der Installation des Installers ICEM01. Dieses Eis sollte nie einen Fehler zurückgeben.
