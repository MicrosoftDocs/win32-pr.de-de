# <a name="validation-automation"></a>Validierungs Automatisierung

Sie können den Evalcom2.dll verwenden, um Validierungs Vorgänge für Installationspakete und Mergemodule zu implementieren.

Die folgenden Abschnitte dieser Dokumentation enthalten Informationen zum Verwenden der Windows Installer Validierung innerhalb der Authoring Tools:

-   [Verwenden von Evalcom2](using-evalcom2.md)
-   [EvalCom2-Schnittstellen](evalcom2-interfaces.md)

Es wird empfohlen, Evalcom2.dll mithilfe des Windows Installer zu installieren. Das Orca.msi Paket, das mit dem Microsoft Windows Software Development Kit (SDK) für Windows XP mit Service Pack 2 (SP2) oder höher bereitgestellt wird, wird Evalcom2.dll installiert. Die Komponenten-ID der Komponente, die die COM-Schnittstelle enthält, ist {53315473-B8D6-470a-9951-D9AE5C11FC91}.

Weitere Informationen zum Abrufen des Windows SDK finden Sie unter [Windows SDK Komponenten für Windows Installer Entwickler](platform-sdk-components-for-windows-installer-developers.md).
