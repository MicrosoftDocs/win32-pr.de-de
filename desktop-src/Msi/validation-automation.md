# <a name="validation-automation"></a>Validierungsautomatisierung

Sie können die Evalcom2.dll verwenden, um Validierungsvorgänge für Installationspakete und Mergemodule zu implementieren.

Die folgenden Abschnitte dieser Dokumentation enthalten Informationen zur Verwendung der Windows Installer-Überprüfung in den Erstellungstools:

-   [Verwenden von Evalcom2](using-evalcom2.md)
-   [EvalCom2-Schnittstellen](evalcom2-interfaces.md)

Es wird empfohlen, Evalcom2.dll installation mit dem Windows installieren. Das Orca.msi-Paket, das mit dem Microsoft Windows Software Development Kit (SDK) für Windows XP mit Service Pack 2 (SP2) oder höher bereitgestellt wird, installiert Evalcom2.dll. Die Komponenten-ID für die Komponente, die die COM-Schnittstelle enthält, ist {53315473-B8D6-470A-9951-D9AE5C11FC91}.

Informationen zum Abrufen des Windows SDK finden Sie unter [Windows SDK Components for Windows Installer Developers](platform-sdk-components-for-windows-installer-developers.md).
