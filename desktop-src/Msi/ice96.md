# <a name="ice96"></a>ICE96

ICE96 überprüft, ob die [publishfeatures-Aktion](publishfeatures-action.md) und die [publishproduct-Aktion](publishproduct-action.md) in der [AdvtExecuteSequence-Tabelle](advtexecutesequence-table.md)eingegeben werden. Ein Paket kann ohne diese Aktionen nicht angekündigt werden.

## <a name="result"></a>Ergebnis

ICE96 gibt die folgenden Warnungen aus.



| ICE96-Warnung                                                            | BESCHREIBUNG                                                                                                  |
|--------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| Die publishfeatures-Aktion ist in der AdvtExecuteSequence-Tabelle erforderlich. | Ein Paket kann nicht ohne die publishfeatures-Aktion angekündigt werden, die in der AdvtExecuteSequence-Tabelle enthalten ist. |
| Die publishproduct-Aktion ist in der AdvtExecuteSequence-Tabelle erforderlich.  | Ein Paket kann nicht ohne die publishproduct-Aktion angekündigt werden, die in der AdvtExecuteSequence-Tabelle enthalten ist.  |
