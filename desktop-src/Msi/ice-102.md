# <a name="ice-102"></a>ICE 102

Überprüft die Tabellen [MsiServiceConfig](msiserviceconfig-table.md) und [MsiServiceConfigFailureActions.](msiserviceconfigfailureactions-table.md)

Überprüft, ob der Wert in der Spalte Ereignis in den Tabellen [MsiServiceConfig](msiserviceconfig-table.md) und [MsiServiceConfigFailureActions](msiserviceconfigfailureactions-table.md) **msidbServiceConfigEventInstall,** **msidbServiceConfigEventUninstall** oder **msidbServiceConfigEventReinstall** ist. Überprüft, ob der Wert in der Spalte Komponente \_ in den Tabellen MsiServiceConfig und MsiServiceConfigFailureActions ein gültiger Schlüssel in der [Komponententabelle](component-table.md)ist.

Überprüft, ob die Werte in der Spalte ConfigType der [MsiServiceConfig-Tabelle](msiserviceconfig-table.md) einer der zulässigen Werte sind.

Überprüft, ob die Werte in den Spalten Aktionen und Verzögerte Aktionen der [MsiServiceConfigFailureActions-Tabelle](msiserviceconfigfailureactions-table.md) durch getrennt \[ ~ \] sind. Überprüft, ob die Anzahl der werte, die in der Spalte Aktionen aufgeführt sind, und die Anzahl der Werte, die in jeder Zeile der MsiServiceConfigFailureActions -Tabelle in der Spalte Verzögerte Aktionen aufgeführt sind, identisch sind.

Wenn der Wert im Feld ConfigType in der [MsiServiceConfig-Tabelle](msiserviceconfig-table.md) **SERVICE \_ CONFIG \_ DELAYED \_ AUTO \_ START** lautet, muss der Wert im Feld Argument 0 oder 1 sein.

Wenn der Wert im Feld ConfigType in der [MsiServiceConfig-Tabelle](msiserviceconfig-table.md) **SERVICE \_ CONFIG \_ REQUIRED PRIVILEGES \_ \_ INFO** lautet, muss der Wert im Feld Argument eine gültige Zeichenfolge von **Privilege Constants** enthalten.

Wenn der Wert im Feld ConfigType in der [MsiServiceConfig-Tabelle](msiserviceconfig-table.md) **SERVICE \_ CONFIG \_ SERVICE SID \_ \_ INFO** lautet, muss der Wert im Feld Argument **SERVICE SID TYPE \_ \_ \_ NONE**, **SERVICE SID TYPE \_ \_ \_ RESTRICTED** oder **SERVICE SID TYPE \_ \_ \_ UNRESTRICTED** sein.

Wenn der Wert im Feld ConfigType in der [MsiServiceConfig-Tabelle](msiserviceconfig-table.md) **SERVICE \_ CONFIG \_ PRESHUTDOWN \_ INFO** lautet, muss der Wert im Feld Argument positiv oder leer sein. Wenn der Wert im Feld ConfigType in der MsiServiceConfig-Tabelle **SERVICE \_ CONFIG \_ FAILURE ACTIONS \_ \_ FLAG** lautet, muss der Wert im Feld Argument 0 oder 1 enthalten.

**[Windows Installer 4.5 oder früher:](not-supported-in-windows-installer-4-5.md)** Wird nicht unterstützt. Dieser ICE ist ab Windows Installer 5.0 verfügbar.
