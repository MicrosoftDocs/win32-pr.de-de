# <a name="overview-of-windows-installer"></a>Übersicht über Windows Installer

Durch die Windows Installer werden die Gesamtbetriebskosten (TCO) für Ihre Kunden reduziert, indem Sie Ihnen ermöglichen, ihre Produkte und Anwendungen effizient zu installieren und zu konfigurieren. Das Installationsprogramm kann Ihrem Produkt auch neue Funktionen zur Verfügung stellen, ohne diese zu installieren, Produkte bei Bedarf zu installieren und benutzerdefinierte Anpassungen hinzuzufügen.
