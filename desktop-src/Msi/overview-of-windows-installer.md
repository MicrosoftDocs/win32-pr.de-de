# <a name="overview-of-windows-installer"></a>Übersicht über Windows Installer

Der Windows Installer reduziert die Gesamtbetriebskosten (TCO) für Ihre Kunden, indem er es ihnen ermöglicht, Ihre Produkte und Anwendungen effizient zu installieren und zu konfigurieren. Das Installationsprogramm kann Ihrem Produkt auch neue Funktionen bereitstellen, um Features ohne Installation ankündigungen, Produkte bei Bedarf zu installieren und Benutzeranpassungen hinzuzufügen.
