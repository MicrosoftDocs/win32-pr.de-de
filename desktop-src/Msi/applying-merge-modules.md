# <a name="applying-merge-modules"></a>Anwenden von Mergemodulen

Mergemodule müssen mithilfe eines mergetools in ein Installationspaket zusammengeführt werden. Die beste Alternative besteht darin, ein frei verteiltes Zusammenführungs Tool zu erwerben oder eines der Zusammenführungs Tools zu erwerben, die von unabhängigen Softwareanbietern zur Verfügung gestellt werden können. Beispielsweise können Sie die von Mergemod.dll bereitgestellte Funktionalität nutzen.

Mergemodule mit mehreren Sprachen können in verschiedenen Sprachen geöffnet und dann in ein Ziel zusammengeführt werden, um eine andere Sprachversion des Pakets zu entwickeln. Weitere Informationen finden Sie unter

-   [Verwenden von Automation zum Zusammenführen eines Mergemoduls zu einer Datenbank](using-automation-to-merge-a-merge-module-into-a-database.md)
-   [Verwenden der API zum Zusammenführen eines Mergemoduls zu einer Datenbank](using-the-api-to-merge-a-merge-module-into-a-database.md)
-   [Öffnen eines Multiple-Language Merge-Moduls in einer bestimmten Sprache](opening-a-multiple-language-merge-module-in-a-specific-language.md)
-   [Mehrmals Zusammenführen eines Moduls mit mehreren Sprachen in demselben Paket](merging-a-multiple-language-module-into-the-same-package-multiple-times.md)
-   [Verbinden eines Mergemoduls mit mehreren Features](connecting-a-merge-module-to-multiple-features.md)
