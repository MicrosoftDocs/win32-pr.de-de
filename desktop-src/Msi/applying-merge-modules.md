# <a name="applying-merge-modules"></a>Anwenden von Mergemodulen

Mergemodule müssen mithilfe eines Mergetools in einem Installationspaket zusammengeführt werden. Die beste Alternative ist es, ein frei verteiltes Mergetool zu erhalten oder eines der Zusammenführungstools zu erwerben, um von unabhängigen Softwareherstellern zur Verfügung zu stehen. Sie können z. B. die funktionalität verwenden, die von Mergemod.dll.

Mergemodule mit mehreren Sprachen können in verschiedenen Sprachen geöffnet und dann zu einem Ziel zusammengeführt werden, um eine andere Sprachversion des Pakets zu erzeugen. Weitere Informationen finden Sie unter

-   [Verwenden der API zum Zusammenführen eines Mergemoduls in einer Datenbank](using-the-api-to-merge-a-merge-module-into-a-database.md)
-   [Öffnen eines Multiple-Language Mergemoduls in einer bestimmten Sprache](opening-a-multiple-language-merge-module-in-a-specific-language.md)
-   [Mehrfaches Zusammenführen eines Moduls mit mehreren Sprachen in demselben Paket](merging-a-multiple-language-module-into-the-same-package-multiple-times.md)
-   [Verbinden eines Mergemoduls mit mehreren Funktionen](connecting-a-merge-module-to-multiple-features.md)
