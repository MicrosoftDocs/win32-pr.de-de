# <a name="updating-directory-structure-for-an-upgrade"></a>Aktualisieren der Verzeichnisstruktur für ein Upgrade

Verwenden Sie den Datenbank-Editor, der mit dem SDK bereitgestellt wird, oder einen anderen Editor, um die [Verzeichnis Tabelle](directory-table.md) in MNP2001.msi zu öffnen. Die Verzeichnisstruktur des Beispiel Upgrades ist mit der Verzeichnisstruktur des ursprünglichen Produkts identisch, und es sind keine Änderungen an der ursprünglichen Verzeichnis Tabelle erforderlich, die unter [angeben der Verzeichnisstruktur](specifying-directory-structure.md)beschrieben wird.
