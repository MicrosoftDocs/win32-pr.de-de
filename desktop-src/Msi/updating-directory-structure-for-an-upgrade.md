# <a name="updating-directory-structure-for-an-upgrade"></a>Aktualisieren der Verzeichnisstruktur für ein Upgrade

Verwenden Sie den Datenbank-Editor Orca, der mit dem SDK oder einem anderen Editor bereitgestellt wird, um die [Verzeichnistabelle](directory-table.md) in MNP2001.msi zu öffnen. Die Verzeichnisstruktur des Beispielupgrades entspricht der Verzeichnisstruktur des ursprünglichen Produkts, und es sind keine Änderungen an der ursprünglichen Verzeichnistabelle erforderlich, die unter [Angeben der Verzeichnisstruktur](specifying-directory-structure.md)beschrieben wird.
