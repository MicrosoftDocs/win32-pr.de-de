# <a name="minimize-dialog-style-bit"></a>Minimieren von Dialogformatbits

Wenn dieses Bit festgelegt ist, kann das Dialogfeld minimiert werden. Dieses Bit wird für modale Dialogfelder ignoriert, die nicht minimiert werden können.

## <a name="value"></a>Wert



| Decimal | Hexadezimal | Konstante                          |
|---------|-------------|-----------------------------------|
| 4       | 0x00000004  | **msidbDialogAttributesMinimize** |
