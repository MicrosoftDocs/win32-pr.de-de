# <a name="minimize-dialog-style-bit"></a>Minimieren des Dialog Stils

Wenn dieses Bit festgelegt ist, kann das Dialogfeld minimiert werden. Dieses Bit wird bei modalen Dialogfeldern ignoriert, die nicht minimiert werden können.

## <a name="value"></a>Wert



| Decimal | Hexadezimal | Konstante                          |
|---------|-------------|-----------------------------------|
| 4       | 0x00000004  | **msidbdialogattributesminimi** |
