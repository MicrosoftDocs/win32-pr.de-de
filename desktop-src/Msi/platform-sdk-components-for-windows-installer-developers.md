# <a name="windows-sdk-components-for-windows-installer-developers"></a>Windows SDK-Komponenten für Windows Installer-Entwickler

Die Komponenten des Windows Installer Software Development Kit sind im Microsoft Windows Software Development Kit (SDK) enthalten. Das Windows SDK umfasst verteilbare Komponenten, Dokumentation, Tool zum Überprüfen der Installerdatenbank, Datenbanktabellen-Editor, Datenbankschema, Entwicklungstools, tools Visual Basic Scripting Edition (VBScript), Beispielprodukt und Codebeispiele.

Die Downloadseite für das Windows SDK finden Sie unter [Microsoft Windows Software Development Kit (SDK).](https://developer.microsoft.com/windows/downloads/windows-10-sdk)
