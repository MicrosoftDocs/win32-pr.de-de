# <a name="windows-sdk-components-for-windows-installer-developers"></a>Windows SDK Komponenten für Windows Installer Entwickler

Die Komponenten des Windows Installer Software Development Kit sind im Microsoft Windows Software Development Kit (SDK) enthalten. Die Windows SDK umfasst verteilbare Komponenten, Dokumentationen, das Validierungs Tool für die Daten Bank Überprüfung, den Datenbanktabellen-Editor, das Datenbankschema, Entwicklungs Tools, Visual Basic Scripting Edition Tools (VBScript), Beispiel Produkt und Codebeispiele.

Die Downloadseite für die Windows SDK finden Sie unter [Microsoft Windows Software Development Kit (SDK)](https://developer.microsoft.com/windows/downloads/windows-10-sdk).
