# <a name="object-model-for-mergemoddll-version-20"></a>Objektmodell für Mergemod.dll Version 2,0

Das Objektmodell für Mergemod.dll Version 2,0 ist wie folgt organisiert.

![Objektmodell für mergemod.dll Version 2,0](images/mergobj2.png)

Das [Merge-Objekt](merge-object.md) ist das primäre Objekt des Modells. Das [**GetFiles-Objekt**](getfiles-object.md) ist ein sekundäres Objekt. Abhängigkeiten sind Auflistungen von [**Abhängigkeits Objekten**](dependency-object.md). Fehler sind Auflistungen von [**Fehler Objekten**](error-object.md). Configurableitems sind Sammlungen von [**configurableitem-Objekten**](configurableitem-object.md).
