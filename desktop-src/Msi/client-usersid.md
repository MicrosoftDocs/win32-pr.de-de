# <a name="clientusersid-property"></a>Client. UserSID (Eigenschaft)

Die Benutzer-SID für die Komponente. Diese Eigenschaft ruft die [**msienenclientsex**](/windows/desktop/api/Msi/nf-msi-msienumclientsexa) -Funktion auf.

**[Windows Installer 4,5 oder früher](not-supported-in-windows-installer-4-5.md):** Nicht unterstützt. Diese Eigenschaft ist ab Windows Installer 5,0 verfügbar.
