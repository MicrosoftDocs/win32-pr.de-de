# <a name="ice-101"></a>Ice 101

Überprüft, ob kein Wert in der featurespalte der [Funktions](feature-table.md) Tabelle eine maximale Länge von 38 Zeichen überschreitet.

**[Windows Installer 4,5 oder früher](not-supported-in-windows-installer-4-5.md):** Nicht unterstützt. Dieser Ice ist ab Windows Installer 5,0 verfügbar.
