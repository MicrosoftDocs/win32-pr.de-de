# <a name="identifier"></a>Bezeichner

Der Bezeichnerdatentyp ist eine Textzeichenfolge. Bezeichner können die ASCII-Zeichen A-Z (a-z), Ziffern, Unterstriche ( \_ ) oder Punkte (.) enthalten. Jeder Bezeichner muss jedoch entweder mit einem Buchstaben oder einem Unterstrich beginnen.
