# <a name="identifier"></a>Bezeichner

Der BezeichnerDatentyp ist eine Text Zeichenfolge. Bezeichner können die ASCII-Zeichen a-z (a-z), Ziffern, Unterstriche ( \_ ) oder Punkte (.) enthalten. Allerdings muss jeder Bezeichner entweder mit einem Buchstaben oder einem Unterstrich beginnen.
