# <a name="ice-103"></a>Ice 103

Ice 103 überprüft das [msiprint](msiprint-controlevent.md) -und das [msilaunchapp](msilaunchapp-controlevent.md) -Steuerungs Ereignis.

Ice 103 überprüft, ob das [msiprint](msiprint-controlevent.md) -Steuerelement Ereignis nur in einem Dialogfeld mit einem [scrollabletext](scrollabletext-control.md) -Steuerelement verwendet wird.

**[Windows Installer 4,5 oder früher](not-supported-in-windows-installer-4-5.md):** Nicht unterstützt. Dieser Ice ist ab Windows Installer 5,0 verfügbar.
