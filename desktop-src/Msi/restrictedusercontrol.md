# <a name="restrictedusercontrol-property"></a>RestrictedUserControl(Eigenschaft)

Die **RestrictedUserControl-Eigenschaft** sollte auf 1 festgelegt werden, wenn eingeschränkt wird, welche öffentlichen Eigenschaften [an](public-properties.md) die Serverseite gesendet werden können. Autoren können diese Eigenschaft in einer bedingten Anweisung verwenden, um die Ausführung von Dialogen oder Aktionen davon abhängig zu machen, ob Einschränkungen für öffentliche Eigenschaften bestehen.
