# <a name="restrictedusercontrol-property"></a>Restrictedusercontrol (Eigenschaft)

Die **restrictedusercontrol** -Eigenschaft sollte auf 1 festgelegt werden, wenn Sie einschränken, welche [öffentlichen Eigenschaften](public-properties.md) an die Serverseite gesendet werden können. Autoren können diese Eigenschaft in der Bedingungs Anweisung verwenden, damit die Ausführung von Dialogfeldern oder Aktionen davon abhängt, ob es Einschränkungen für öffentliche Eigenschaften gibt.

Windows Installer 4,0 oder Windows Installer 4,5 unter Windows Server 2008 oder Windows Vista. Windows Installer unter Windows Server 2003 oder Windows XP.
