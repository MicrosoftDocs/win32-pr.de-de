# <a name="using-isolated-components"></a>Verwenden isolierter Komponenten

Mithilfe [isolierter Komponenten](isolated-components.md)können Autoren von Installationspaketen angeben, dass das Installationsprogramm die freigegebenen Dateien einer Anwendung direkt in den Ordner der Anwendung anstatt an einen freigegebenen Speicherort kopiert. Dieser private Satz von Dateien wird dann ausschließlich von der Anwendung verwendet.

Weitere Informationen finden Sie unter [Installieren einer COM-Komponente an einem privaten Speicherort,](installing-a-com-component-to-a-private-location.md) [Installieren einer Nicht-COM-Komponente an einem privaten Speicherort,](installing-a-non-com-component-to-a-private-location.md) [Erstellen einer COM-Komponente in einem vorhandenen Paket als privat](make-a-com-component-in-an-existing-package-private.md) oder Erstellen einer [Nicht-COM-Komponente in einem vorhandenen privaten Paket.](make-a-non-com-component-in-an-existing-package-private.md)
