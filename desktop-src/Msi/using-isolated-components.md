# <a name="using-isolated-components"></a>Verwenden isolierter Komponenten

Mithilfe [isolierter Komponenten](isolated-components.md)können Autoren von Installationspaketen angeben, dass das Installationsprogramm die freigegebenen Dateien einer Anwendung direkt in den Anwendungsordner und nicht in einen freigegebenen Speicherort kopiert. Dieser private Satz von Dateien wird dann exklusiv von der Anwendung verwendet.

Weitere Informationen finden Sie unter [Installieren einer COM-Komponente an einem privaten Speicherort](installing-a-com-component-to-a-private-location.md), [Installieren einer nicht-COM-Komponente an einem privaten Speicherort](installing-a-non-com-component-to-a-private-location.md), [Erstellen einer COM-Komponente in einem vorhandenen Paket](make-a-com-component-in-an-existing-package-private.md) oder [Erstellen einer nicht-COM-Komponente in einem vorhandenen Paket](make-a-non-com-component-in-an-existing-package-private.md).
