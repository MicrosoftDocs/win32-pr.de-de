# <a name="ice89"></a>ICE89

ICE89 überprüft, ob der Wert in der übergeordneten ProgID-Spalte in der ProgID- \_ [Tabelle](progid-table.md) ein gültiger Fremdschlüssel in der ProgID-Spalte in der ProgID-Tabelle ist. Jedes übergeordnete ProgID-Element muss einen Datensatz in der ProgID-Tabelle aufweisen.

## <a name="result"></a>Ergebnis

ICE89 gibt die folgenden Fehler aus.



| ICE89-Fehler                                                           | BESCHREIBUNG                                                                |
|-----------------------------------------------------------------------|----------------------------------------------------------------------------|
| Das übergeordnete ProgID-Element \_ ' \[ 1 \] ' in der ProgID-Tabelle ist keine gültige ProgID. | Es wurde eine übergeordnete ProgID angegeben, die nicht in der ProgID-Tabelle aufgeführt ist. |
