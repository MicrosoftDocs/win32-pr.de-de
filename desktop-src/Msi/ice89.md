# <a name="ice89"></a>ICE89

ICE89 überprüft, ob der Wert in der Spalte Progid Parent in der \_ [ProgId-Tabelle](progid-table.md) ein gültiger Fremdschlüssel in der ProgId-Spalte in der ProgId-Tabelle ist. Jedes übergeordnete ProgId-Element sollte über einen Datensatz in der ProgId-Tabelle verfügen.

## <a name="result"></a>Ergebnis

ICE89 gibt die folgenden Fehler aus.



| ICE89-Fehler                                                           | BESCHREIBUNG                                                                |
|-----------------------------------------------------------------------|----------------------------------------------------------------------------|
| Das übergeordnete ProgId-Element \_ \[ '1' \] in der ProgId-Tabelle ist keine gültige ProgId. | Es ist ein übergeordnetes ProgId-Element angegeben, das nicht in der ProgId-Tabelle aufgeführt ist. |
