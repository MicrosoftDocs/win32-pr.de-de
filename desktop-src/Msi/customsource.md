# <a name="customsource"></a>CustomSource

Der CustomSource-Datentyp ist entweder ein gültiger [Bezeichner](identifier.md) oder ein externer Schlüssel, abhängig von der Type-Spalte der [CustomAction-Tabelle](customaction-table.md). Dabei kann es sich um einen externen Schlüssel in die [binäre Tabelle](binary-table.md), [Dateitabelle](file-table.md), [Verzeichnis Tabelle](directory-table.md)und [Eigenschaften Tabelle](property-table.md)handeln. Beachten Sie, dass externe Schlüssel in die Eigenschaften Tabelle nicht überprüft werden, da Eigenschaften zur Laufzeit hinzugefügt werden können.
