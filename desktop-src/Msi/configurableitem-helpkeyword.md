# <a name="configurableitemhelpkeyword-property"></a>Configurableitem. HelpKeyword-Eigenschaft

Die **HelpKeyword** -Eigenschaft gibt den Wert aus der HelpKeyword-Spalte der [ModuleConfiguration-Tabelle](moduleconfiguration-table.md)zurück.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = ConfigurableItem.HelpKeyword
```



## <a name="property-value"></a>Eigenschaftswert

## <a name="c"></a>C++

Siehe [**get \_ HelpKeyword**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_helpkeyword) -Funktion.
