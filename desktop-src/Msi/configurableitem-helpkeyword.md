# <a name="configurableitemhelpkeyword-property"></a>ConfigurableItem.HelpKeyword (Eigenschaft)

Die **HelpKeyword-Eigenschaft** gibt den Wert aus der HelpKeyword-Spalte der [ModuleConfiguration-Tabelle zurück.](moduleconfiguration-table.md)

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = ConfigurableItem.HelpKeyword
```



## <a name="property-value"></a>Eigenschaftswert

## <a name="c"></a>C++
