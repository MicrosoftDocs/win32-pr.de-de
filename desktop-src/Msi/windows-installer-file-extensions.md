# <a name="windows-installer-file-name-extensions"></a>Dateinamen Erweiterungen Windows Installer

Die folgenden Dateinamen Erweiterungen werden mit dem Windows Installer verwendet.



| Durchwahl | BESCHREIBUNG                                |
|-----------|--------------------------------------------|
| .msi      | Windows Installer Datenbank.                |
| MSM      | Windows Installer Mergemodul.            |
| MSP      | Windows Installer Patch.                   |
| MST      | Windows Installer Transformation.               |
| IDT      | Windows Installer Datenbanktabelle exportiert. |
| . Cub      | Validierungs Modul.                         |
| PCP      | Datei für Windows Installer Patcherstellung      |
