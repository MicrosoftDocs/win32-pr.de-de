# <a name="maintenance-installation"></a>Wartungsinstallation

Nachdem eine Anwendung erfolgreich installiert wurde, kann der Benutzer zur Installation zurückkehren, um Features hinzuzufügen oder zu entfernen. Dies wird als Wartungsinstallation bezeichnet.

[Transformationen,](transforms.md) die auf eine Installation angewendet wurden, müssen bei nachfolgenden Wartungsinstallationen verfügbar sein.
