# <a name="maintenance-installation"></a>Wartungs Installation

Nach der erfolgreichen Installation einer Anwendung möchte der Benutzer möglicherweise zur Installation zurückkehren, um Features hinzuzufügen oder zu entfernen. Dies wird als Wartungs Installation bezeichnet.

[Transformationen](transforms.md) , die auf eine Installation angewendet wurden, müssen bei nachfolgenden Wartungs Installationen verfügbar sein.
