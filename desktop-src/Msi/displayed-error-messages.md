# <a name="displayed-error-messages"></a>Angezeigte Fehlermeldungen

Eine [installerfunktion](installer-function-reference.md) zeigt möglicherweise ein Dialogfeld Fehlermeldung an, das einen der folgenden Fehler zurückgibt. Ein Fehler Feld wird nur angezeigt, wenn sich die Benutzeroberflächen Ebene auf der Ebene "Full", "Reduced" oder "Basic" befindet. Weitere Informationen finden Sie unter [Benutzeroberflächen Ebenen](user-interface-levels.md).

Die Installer-Funktionen zeigen nur Fehlermeldungen an, die in der folgenden Tabelle enthalten sind. Bei Fehlern, die nicht in dieser Liste enthalten sind, ist der Aufrufer der Funktion für die Behandlung der Anzeige von Fehlermeldungen verantwortlich.



| Fehlercode               | Fehler                        |
|--------------------------|------------------------------|
| Fehler beim \_ Installieren von \_ Suspend  | Die Installation wurde angehalten.   |
| Fehler beim \_ Installieren von \_ Userexit. | Der Benutzer hat die Installation beendet. |
| Fehler bei der \_ Installation. \_  | Fehler bei der Installation.              |
