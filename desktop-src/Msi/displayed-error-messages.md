# <a name="displayed-error-messages"></a>Angezeigte Fehlermeldungen

Eine [Installerfunktion](installer-function-reference.md) zeigt möglicherweise ein Fehlermeldungsdialogfeld an, das einen der folgenden Fehler zurückgibt. Ein Fehlerfeld wird nur angezeigt, wenn sich die Benutzeroberflächenebene auf der vollständigen, reduzierten oder grundlegenden Ebene befindet. Weitere Informationen finden Sie unter [Benutzeroberfläche Ebenen.](user-interface-levels.md)

Die Installationsprogrammfunktionen zeigen nur Fehlermeldungen an, die in der folgenden Tabelle aufgeführt sind. Bei Fehlern, die nicht in dieser Liste enthalten sind, ist der Aufrufer der Funktion für die Behandlung der Anzeige von Fehlermeldungen verantwortlich.



| Fehlercode               | Fehler                        |
|--------------------------|------------------------------|
| FEHLER \_ BEIM INSTALLIEREN VON \_ "SUSPEND"  | Die Installation wurde angehalten.   |
| FEHLER \_ BEI DER INSTALLATION VON \_ USEREXIT | Der Benutzer hat die Installation beendet. |
| FEHLER BEI \_ DER INSTALLATION \_  | Fehler bei der Installation.              |
