# <a name="ice33"></a>ICE33

ICE33 verarbeitet Einträge in der [Registrierungs Tabelle](registry-table.md) und gibt möglicherweise eine Warnung für jeden Tabelleneintrag aus, der Klassen, Dateinamen Erweiterungen, ProgIDs, Shellverben, Remote Server-APPIDs, MIME-Typen oder typelisb registriert. Die Warnungen sind veraltet und können ignoriert werden.
