# <a name="componentcomponentcode-property"></a>Component.ComponentCode-Eigenschaft

Der Komponentencode der betreffenden Komponente. Diese Eigenschaft ruft die [**MsiEnumComponentsEx-Funktion auf.**](/windows/desktop/api/Msi/nf-msi-msienumcomponentsexa)

**[Windows Installer 4.5 oder früher:](not-supported-in-windows-installer-4-5.md)** Wird nicht unterstützt. Diese Eigenschaft ist ab Windows Installer 5.0 verfügbar.
