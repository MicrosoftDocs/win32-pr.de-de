<span data-ttu-id="c9e45-104">Ein Administrator kann die folgenden Methoden verwenden, um einem Benutzer ohne Administratorrechte die Installation einer Anwendung mit erhöhten System Berechtigungen zu ermöglichen.</span><span class="sxs-lookup"><span data-stu-id="c9e45-104">An administrator can use the following methods to enable a non-administrator user to install an application with elevated system privileges.</span></span>

    <span data-ttu-id="c9e45-106">**Windows Vista:** Erforderlich.</span><span class="sxs-lookup"><span data-stu-id="c9e45-106">**Windows Vista:** Required.</span></span>

-   <span data-ttu-id="c9e45-115">Ein nicht privilegierter Benutzer kann eine angekündigte Anwendung installieren, die erhöhte Rechte erfordert, wenn ein lokaler System-Agent die Anwendung ankündigt.</span><span class="sxs-lookup"><span data-stu-id="c9e45-115">A non-privileged user can install an advertised application that requires elevated privileges if a local system agent advertises the application.</span></span> <span data-ttu-id="c9e45-116">Die Anwendung kann für eine Installation pro Benutzer oder pro Computer angekündigt werden.</span><span class="sxs-lookup"><span data-stu-id="c9e45-116">The application can be advertised for a per-user or per-machine installation.</span></span> <span data-ttu-id="c9e45-117">Eine mit dieser Methode installierte Anwendung wird als verwaltet betrachtet.</span><span class="sxs-lookup"><span data-stu-id="c9e45-117">An application installed using this method is considered managed.</span></span> <span data-ttu-id="c9e45-118">Weitere Informationen finden Sie unter ankündigen [einer Per-User Anwendung, die mit erhöhten Rechten installiert werden soll](advertising-a-per-user-application-to-be-installed-with-elevated-privileges.md).</span><span class="sxs-lookup"><span data-stu-id="c9e45-118">For more information, see [Advertising a Per-User Application To Be Installed with Elevated Privileges](advertising-a-per-user-application-to-be-installed-with-elevated-privileges.md).</span></span>
-   <span data-ttu-id="c9e45-119">Ein Administrator kann die [alwaysinstallerhöhten](alwaysinstallelevated.md) -Richtlinie sowohl für Einzelbenutzer-als auch pro-Computer-Installationen festlegen.</span><span class="sxs-lookup"><span data-stu-id="c9e45-119">An administrator can set the [AlwaysInstallElevated](alwaysinstallelevated.md) policy for both per-user and per-machine installations.</span></span> <span data-ttu-id="c9e45-120">Diese Methode kann einen Computer mit einem Sicherheitsrisiko öffnen, denn wenn diese Richtlinie festgelegt ist, kann ein Benutzer, der kein Administrator ist, Installationen mit erhöhten Rechten ausführen und auf sichere Speicherorte auf dem Computer zugreifen, wie z. b. den Systemordner oder den **HKLM** -Registrierungsschlüssel.</span><span class="sxs-lookup"><span data-stu-id="c9e45-120">This method can open a computer to a security risk, because when this policy is set, a non-administrator user can run installations with elevated privileges and access secure locations on the computer, such as the SystemFolder or the **HKLM** registry key.</span></span>

    <span data-ttu-id="c9e45-121">Wenn die Anwendung pro Computer installiert wird, während die [alwaysinstallerhöhten](alwaysinstallelevated.md) -Richtlinie festgelegt ist, wird das Produkt als verwaltet behandelt.</span><span class="sxs-lookup"><span data-stu-id="c9e45-121">If the application is installed per-machine while the [AlwaysInstallElevated](alwaysinstallelevated.md) policy is set, the product is treated as managed.</span></span> <span data-ttu-id="c9e45-122">In diesem Fall kann die Anwendung weiterhin eine Reparatur mit erhöhten Rechten durchführen, wenn die Richtlinie entfernt wird.</span><span class="sxs-lookup"><span data-stu-id="c9e45-122">In this case, the application can still perform a repair with elevated privileges if the policy is removed.</span></span> <span data-ttu-id="c9e45-123">Auch wenn die Anwendung pro Benutzer installiert wird, während die alwaysinstallerhöhten-Richtlinie festgelegt ist, kann die Anwendung keine Reparatur durchführen, wenn die Richtlinie entfernt wird.</span><span class="sxs-lookup"><span data-stu-id="c9e45-123">Also, if the application is installed per-user while the AlwaysInstallElevated policy is set, the application is unable to perform a repair if the policy is removed.</span></span>
