# <a name="formatted-text-custom-actions"></a>Formatierte Text benutzerdefinierte Aktionen

Diese Art von benutzerdefinierter Aktion legt eine Eigenschaft oder ein Installationsverzeichnis aus einer formatierten Text Zeichenfolge fest. Die Parameter in dieser Zeichenfolge können Eigenschaften, Umgebungsvariablen, Dateipfade oder Verzeichnispfade von Komponenten sein. Dies ist hilfreich, wenn Informationen an nachfolgende benutzerdefinierte Aktionen übergeben werden.

Die folgenden grundlegenden benutzerdefinierten Aktions Typen legen eine Eigenschaft oder ein Verzeichnis fest. Weitere Informationen finden Sie unter der jeweilige benutzerdefinierte Aktionstyp.



| Benutzerdefinierter Aktionstyp                                 | BESCHREIBUNG                                    |
|----------------------------------------------------|------------------------------------------------|
| [Benutzerdefinierter Aktionstyp 51](custom-action-type-51.md) | Legt eine Eigenschaft mit einer formatierten Text Zeichenfolge fest.  |
| [Benutzerdefinierter Aktionstyp 35](custom-action-type-35.md) | Legt ein Verzeichnis mit einer formatierten Text Zeichenfolge fest. |



 

Eine Zusammenfassung aller Typen von benutzerdefinierten Aktionen und deren Codierung in die [CustomAction-Tabelle](customaction-table.md)finden Sie im Abschnitt [zusammenfassende Liste aller benutzerdefinierten Aktions Typen](summary-list-of-all-custom-action-types.md) .
