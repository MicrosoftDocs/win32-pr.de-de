# <a name="ice24"></a>ICE24

ICE24 überprüft die folgenden Eigenschaften in der [Tabelle Property](property-table.md):

-   Die [**ProductCode-Eigenschaft**](productcode.md) ist ein gültiger [GUID-Datentyp.](guid.md)
-   Die [**ProductVersion-Eigenschaft**](productversion.md) ist eine gültige Produktversion.
-   Die [**ProductLanguage-Eigenschaft**](productlanguage.md) ist ein gültiger [Language-Datentyp.](language.md)

## <a name="result"></a>Ergebnis

ICE24 sendet eine Fehlermeldung, wenn eine dieser Eigenschaften nicht in Form eines gültigen Datentyps vorliegt.
