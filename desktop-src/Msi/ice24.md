# <a name="ice24"></a>ICE24

ICE24 überprüft die folgenden Eigenschaften in der Eigenschaften [Tabelle](property-table.md):

-   Die [**ProductCode**](productcode.md) -Eigenschaft ist ein gültiger [GUID](guid.md) -Datentyp.
-   Die [**ProductVersion**](productversion.md) -Eigenschaft ist eine gültige Produktversion.
-   Die [**productlanguage**](productlanguage.md) -Eigenschaft ist ein gültiger [sprach](language.md) Datentyp.

## <a name="result"></a>Ergebnis

ICE24 gibt eine Fehlermeldung aus, wenn eine dieser Eigenschaften nicht in Form eines gültigen Datentyps vorliegt.
