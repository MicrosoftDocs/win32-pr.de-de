# <a name="templatefolder-property"></a>TemplateFolder(Eigenschaft)

Das Installationsprogramm legt die **TemplateFolder-Eigenschaft** auf den vollständigen Pfad zum Vorlagenordner für den aktuellen Benutzer fest.

Nur Windows 2000 verweist diese Eigenschaft auf ein Dateisystemverzeichnis, das die Vorlagen enthält, die für alle Benutzer verfügbar sind, wenn die [**ALLUSERS-Eigenschaft**](allusers.md) festgelegt ist.
