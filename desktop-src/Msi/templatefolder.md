# <a name="templatefolder-property"></a>TemplateFolder-Eigenschaft

Das Installationsprogramm legt die **TemplateFolder-Eigenschaft** auf den vollständigen Pfad zum Ordner Template für den aktuellen Benutzer fest.

Wenn die [**ALLUSERS-Eigenschaft**](allusers.md) festgelegt ist, verweist diese Eigenschaft auf ein Dateisystemverzeichnis, das die Vorlagen enthält, die für alle Benutzer verfügbar sind.
