# <a name="desktopfolder-property"></a>DesktopFolder-Eigenschaft

Der Installer legt die **DesktopFolder** -Eigenschaft auf den vollständigen Pfad des Desktop Ordners des aktuellen Benutzers fest. Wenn das Profil "alle Benutzer" vorhanden ist und die Eigenschaft [**ALLUSERS**](allusers.md) festgelegt ist, wird diese Eigenschaft auf den Ordner im Profil "alle Benutzer" festgelegt.
