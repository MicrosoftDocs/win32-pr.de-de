# <a name="desktopfolder-property"></a>DesktopFolder-Eigenschaft

Das Installationsprogramm legt die **DesktopFolder-Eigenschaft** auf den vollst채ndigen Pfad des Desktopordners des aktuellen Benutzers fest. Wenn ein Profil "Alle Benutzer" vorhanden ist und die [**ALLUSERS-Eigenschaft**](allusers.md) festgelegt ist, wird diese Eigenschaft auf den Ordner im Profil "Alle Benutzer" festgelegt.
