# <a name="msix64-property"></a>Msix64-Eigenschaft

Die **Msix64** -Eigenschaft wird nur definiert, wenn Sie auf einem x64-Prozessor ausgeführt wird. Der Wert wird vom Installer auf die numerische Prozessor Ebene festgelegt. Die Werte sind identisch mit dem **wprocessorlevel** -Feld der [**System \_ Info**](/windows/win32/api/sysinfoapi/ns-sysinfoapi-system_info) -Struktur.

## <a name="remarks"></a>Bemerkungen

**Windows Installer 2,0 und Windows Installer 3,0:** Nicht unterstützt. Die **Msix64** -Eigenschaft ist ab Windows Installer 3,1 verfügbar.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Version<br/> | Windows Installer 5,0 unter Windows Server 2012, Windows 8, Windows Server 2008 R2 oder Windows 7. Windows Installer 4,0 oder Windows Installer 4,5 unter Windows Server 2008 oder Windows Vista. Windows Installer 3,1 oder höher unter Windows Server 2003, Windows XP und Windows 2000<br/> |
