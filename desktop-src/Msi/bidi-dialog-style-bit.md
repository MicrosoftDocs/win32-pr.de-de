# <a name="bidi-dialog-style-bit"></a>BiDi Dialog Style Bit

Dies ist eine Kombination aus der Rechts-nach-Links-Leserichtung [RTLRO,](rtlro-dialog-style-bit.md)der [RightAligned](rightaligned-dialog-style-bit.md)und den [LeftScroll-Dialogformatbits](leftscroll-dialog-style-bit.md).

## <a name="value"></a>Wert



| Decimal | Hexadezimal | Konstante                                                                                                                                       |
|---------|-------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| 896     | 0x00000380  | **msidbDialogAttributesBiDi**  =  **msidbDialogAttributesRTLRO** \| **msidbDialogAttributesRightAligned** \| **msidbDialogAttributesLeftScroll** |
