# <a name="setting-public-property-values-on-the-command-line"></a>Festlegen von Werten für öffentliche Eigenschaften in der Befehlszeile

Wenn Sie eine öffentliche Eigenschaft auf einen literalzeichenwert festlegen möchten, schließen Sie die Zeichenfolge in Anführungszeichen ein.

Property = "String"

Die Anführungszeichen sind nur erforderlich, wenn die Zeichenfolge Leerzeichen enthält. Um eine öffentliche Eigenschaft in einer Befehlszeile zu löschen (auf NULL festzulegen), legen Sie den Wert der Eigenschaft auf eine leere Zeichenfolge fest. In diesem Fall sind Anführungszeichen erforderlich.

Property = "".
