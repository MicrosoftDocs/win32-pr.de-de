# <a name="initialization-of-property-values"></a>Initialisierung von Eigenschafts Werten

Es wird empfohlen, dass alle Eigenschaften, die von der Installation verwendet werden sollen, in die Eigenschaften [Tabelle](property-table.md) mit einem Anfangswert eingegeben werden. Der Installer legt die Eigenschaften beim Start der Installation auf diese Werte fest. Eigenschaften, für die ein leeres Wert zulässig ist, und in das Installationsprogramm integrierte Eigenschaften müssen nicht initialisiert werden.
