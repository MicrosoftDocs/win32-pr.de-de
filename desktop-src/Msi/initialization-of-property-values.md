# <a name="initialization-of-property-values"></a>Initialisierung von Eigenschaftswerten

Es wird empfohlen, dass alle Eigenschaften, die von der Installation verwendet werden sollen, mit einem Anfangswert in die [Tabelle Eigenschaft](property-table.md) eingegeben werden. Das Installationsprogramm legt die Eigenschaften beim Start der Installation auf diese Werte fest. Eigenschaften, bei denen ein leerer Wert ein zulässiger Wert ist, und eigenschaften, die in das Installationsprogramm integriert sind, müssen nicht initialisiert werden.
