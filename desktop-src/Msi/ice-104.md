# <a name="ice-104"></a>ICE 104

ICE 104 überprüft, ob nur eine der beiden Tabellen vorhanden ist: [MsiLockPermissionsEx](msilockpermissionsex-table.md) und [LockPermissions](lockpermissions-table.md) im Paket.

ICE 104 überprüft die Syntax, die in den Feldern LockObject, Table und SDDLText in der [Tabelle MsiLockPermissionsEx](msilockpermissionsex-table.md) verwendet wird. ICE 104 überprüft nicht, ob der Wert im Feld SDDLText in der Tabelle MsiLockPermissionsEx ein gültiger Sicherheitsdeskriptor ist.

**[Windows Installer 4.5 oder früher:](not-supported-in-windows-installer-4-5.md)** Wird nicht unterstützt. Dieser ICE ist ab Windows Installer 5.0 verfügbar.
