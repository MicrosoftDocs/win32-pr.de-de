# <a name="shortcut"></a>Abkürzung

Der Datentyp für die Verknüpfung wird in der Regel in der Ziel Spalte der Verknüpfungs [Tabelle](shortcut-table.md)verwendet. Wenn Sie eckige Klammern ( \[ \] ) enthält, wird das Verknüpfungs Ziel als formatierte Zeichenfolge ausgewertet. Andernfalls wird die Verknüpfung als [Bezeichner](identifier.md) ausgewertet, und es muss sich um einen gültigen Fremdschlüssel in der [Featuretabelle](feature-table.md)handeln.
