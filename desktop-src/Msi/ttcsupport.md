# <a name="ttcsupport-property"></a>TTCSupport-Eigenschaft

Das Installationsprogramm legt die **TTCSupport-Eigenschaft** fest, um anzugeben, ob das Betriebssystem TTC-Dateien (True Type Font Collections) unterstützt.

## <a name="remarks"></a>Hinweise

Diese Eigenschaft wird festgelegt, wenn das Installationsprogramm Windows 2000 oder eines der folgenden Betriebssysteme erkennt:

-   Japan – 932
-   China – 936
-   Taiwan – 950
-   Korea – 949
-   Hongkong SAR – 950
