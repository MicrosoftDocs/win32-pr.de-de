# <a name="about-custom-actions"></a>Informationen zu benutzerdefinierten Aktionen

Obwohl Standardaktionen ausreichen, um eine Installation in den meisten Fällen auszuführen, ermöglichen benutzerdefinierte Aktionen dem Autor eines Installationspakets, die Funktionen von Standardaktionen zu erweitern, indem ausführbare Dateien, Dynamic Link-Bibliotheken und Skripts enthalten sind. Benutzerdefinierte Aktionen können nicht in den Sequenztabellen verwendet werden, die für die Ankündigung, die [AdvtUISequence-Tabelle](advtuisequence-table.md) und die [AdvtExecuteSequence-Tabelle verwendet werden.](advtexecutesequence-table.md)
