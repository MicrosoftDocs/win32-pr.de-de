# <a name="defaultdir"></a>DefaultDir

Der DefaultDir-Datentyp ist eine Textzeichenfolge, die entweder einen gültigen [Dateinamen oder](filename.md) einen gültigen [Bezeichner enthält.](identifier.md) Dies wird nur in der [Directory-Tabelle verwendet.](directory-table.md) Es muss ein Bezeichner sein, wenn das Verzeichnis ein Stammverzeichnis ist. Wenn das Verzeichnis kein Root-Verzeichnis ist, muss dieser Wert ein Dateiname- oder ein Dateiname-Dateiname-Paar sein. Beachten Sie, dass "." als Dateiname zulässig ist und in der Directory-Tabelle eine besondere Bedeutung hat.
