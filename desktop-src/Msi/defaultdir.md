# <a name="defaultdir"></a>DefaultDir

Der DefaultDir-Datentyp ist eine Text Zeichenfolge, die entweder einen gültigen [Dateinamen](filename.md) oder einen gültigen [Bezeichner](identifier.md)enthält. Diese wird nur in der [Verzeichnis Tabelle](directory-table.md)verwendet. Es muss ein Bezeichner sein, wenn das Verzeichnis ein Stammverzeichnis ist. Wenn das Verzeichnis ein nicht-Stammverzeichnis ist, muss dieser Wert ein Dateiname oder ein Dateiname: Dateiname sein. Beachten Sie, dass "." als Dateiname zulässig ist und in der Verzeichnis Tabelle eine besondere Bedeutung hat.
