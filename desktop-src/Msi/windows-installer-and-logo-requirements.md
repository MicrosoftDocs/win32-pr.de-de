# <a name="windows-installer-and-logo-requirements"></a>Windows Installer-und Logo-Anforderungen

Mit dem Certified for Microsoft Windows-Logo werden Produkte identifiziert, die über unabhängige Tests überprüft wurden, um die Anwendungs Spezifikation für Windows einzuhalten. Diese Spezifikation wurde von Microsoft in Zusammenarbeit mit Kunden und anderen Entwicklern entwickelt, um eine Roadmap zum Erstellen zuverlässiger und verwaltbarer Anwendungen bereitzustellen. Software Hersteller, die der Spezifikation entsprechen, qualifizieren sich für das Certified for Microsoft Windows-Logo und lizenzieren dann das Logo für die Verwendung in Bezug auf Produktverpackung, Werbung, Begleitmaterialien und andere Marketingmaterialien.

Weitere Informationen zur Zertifizierung von Windows Vista, Windows XP und Windows Server 2008 für Ihre Anwendungen finden Sie in der [Microsoft Partner Network]( https://partner.microsoft.com).

Weitere Informationen zur Zertifizierung von Windows Server 2003 und Windows 2000 für Ihre Anwendungen finden Sie unter [https://msdn.microsoft.com/certification](https://www.microsoft.com/windowsserver2003/partners/isvs/cfw.mspx) .

Wenn Sie ein Installationspaket für Ihre Anwendung erstellen, können Sie die Funktionen des Microsoft Windows Installer nutzen, um einige der Zertifizierungsanforderungen zu erfüllen. In der folgenden Tabelle finden Sie Links zu Informationen in Windows Installer Dokumentation, die für einige der Zertifizierungsanforderungen relevant sind. Beachten Sie, dass die nicht alle Voraussetzungen für Zertifizierungen, die in der Anwendungs Spezifikation erläutert werden, von Windows Installer abhängig sind.
