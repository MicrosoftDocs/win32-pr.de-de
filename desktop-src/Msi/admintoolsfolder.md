# <a name="admintoolsfolder-property"></a>Admintoolsfolder (Eigenschaft)

Die **admintoolsfolder** -Eigenschaft enthält den vollständigen Pfad zum Dateisystem Verzeichnis, in dem die Verwaltungs Tools gespeichert werden. Microsoft Management Console (MMC) speichert angepasste Konsolen in diesem Verzeichnis und wechselt mit dem Benutzer. Wenn die Eigenschaft " [**ALLUSERS**](allusers.md) " festgelegt ist, verweist diese Eigenschaft auf das Dateisystem Verzeichnis, das die Verwaltungs Tools für alle Benutzer des Computers enthält.
