# <a name="admintoolsfolder-property"></a>AdminToolsFolder-Eigenschaft

Die **AdminToolsFolder-Eigenschaft** enthält den vollständigen Pfad zum Dateisystemverzeichnis, in dem Verwaltungstools gespeichert sind. Microsoft Management Console (MMC) speichert benutzerdefinierte Konsolen in diesem Verzeichnis und roamingt mit dem Benutzer. Wenn die [**ALLUSERS-Eigenschaft**](allusers.md) festgelegt ist, verweist diese Eigenschaft auf das Dateisystemverzeichnis, das die Verwaltungstools für alle Benutzer des Computers enthält.
