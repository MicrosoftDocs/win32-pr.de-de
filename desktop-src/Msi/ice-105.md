# <a name="ice-105"></a>Ice 105

Überprüft das Paket mit einer Reihe von Anforderungen für die Installation im Benutzer bezogenen Kontext. Ice 105 kann auch Pakete mit zwei Zweck validieren. Mit einem Paket mit zwei Zweck kann der Benutzer auswählen, ob die Anwendung im Kontext Benutzer-oder Computer bezogen installiert werden soll. Informationen zum Entwickeln eines Pakets mit zwei Zweck, finden Sie unter [Erstellen einzelner Pakete](single-package-authoring.md).

Ice 105 führt die folgende Überprüfung des Pakets aus.

-   Hier wird überprüft, ob die Tabelle " [CustomAction](customaction-table.md) " keine benutzerdefinierten Aktionen enthält, die für die Verwendung mit erhöhten Rechten markiert wurden. Weitere Informationen zu erweiterten benutzerdefinierten Aktionen finden Sie unter [Sicherheit für benutzerdefinierte](custom-action-security.md)Aktionen.
-   Hier wird überprüft, ob die [Verzeichnis](directory-table.md) Tabelle eine der folgenden Eigenschaften des System Ordners enthält.
    -   [**Ordner "admintools"**](admintoolsfolder.md)
    -   [**Commonappdatafolder**](commonappdatafolder.md)
    -   [**Fonsordner**](fontsfolder.md)
    -   [**System16Folder**](system16folder.md)
    -   [**System64Folder**](system64folder.md)
    -   [**Systemordner**](systemfolder.md)
    -   [**TempFolder**](tempfolder.md)
    -   [**Verzeichnis Windows befinden**](windowsfolder.md)
    -   [**Windows Volume**](windowsvolume.md)
-   Überprüft, ob das Paket eine Common Language Runtime-Assembly im globalen Assemblycache (GAC) installiert. Weitere Informationen zum Installieren von Assemblys im globalen Assemblycache finden Sie unter [Hinzufügen](adding-assemblies-to-a-package.md) von Assemblys zu Paketen und installieren [von Common Language Runtime](installation-of-common-language-runtime-assemblies.md)-Assemblys
-   Überprüft die [ODBCDatasource](odbcdatasource-table.md) -Tabelle, um zu überprüfen, ob das Paket keine Datenquellen installiert.
-   Überprüft die Tabelle " [ServiceInstall](serviceinstall-table.md) ", um zu überprüfen, ob das Paket keine Dienste installiert.
-   Überprüft, ob in der [Registrierungs](registry-table.md) Tabelle keine Einträge unter den Schlüssel für den **lokalen HKEY- \_ \_ Computer** geschrieben werden.

**[Windows Installer 4,5 oder früher](not-supported-in-windows-installer-4-5.md):** Nicht unterstützt. Dieser Ice ist ab Windows Installer 5,0 verfügbar.
