# <a name="unpublishfeatures-action"></a>Unpublishfeatures-Aktion

Wenn ein gesamtes Produkt deinstalliert wird, entfernt die unpublishfeatures-Aktion die Zustellungs Informationen für das Auswahl-und FeatureComponent aus der Systemregistrierung. Mit dieser Aktion wird die [FeatureComponents-Tabelle](featurecomponents-table.md)abgefragt.

## <a name="sequence-restrictions"></a>Sequenz Einschränkungen

Es gibt keine Sequenz Einschränkungen.

## <a name="actiondata-messages"></a>Aktions Daten Meldungen



| Feld | Beschreibung der Aktions Daten             |
|-------|----------------------------------------|
| \[1\] | Bezeichner eines nicht angekündigten Features. |
