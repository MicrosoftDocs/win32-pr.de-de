# <a name="clientcomponentcode-property"></a>Client.ComponentCode-Eigenschaft

Der Komponentencode der betreffenden Komponente. Diese Eigenschaft ruft die [**MsiEnumClientsEx-Funktion**](/windows/desktop/api/Msi/nf-msi-msienumclientsexa) auf.

**[Windows Installer 4.5 oder früher:](not-supported-in-windows-installer-4-5.md)** Wird nicht unterstützt. Diese Eigenschaft ist ab Windows Installer 5.0 verfügbar.
