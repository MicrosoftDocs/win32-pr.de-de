# <a name="computername-property"></a>Computername (Eigenschaft)

Die Eigenschaft " **Computername** " ist der Computername des aktuellen Systems. Das Installationsprogramm legt diese Eigenschaft durch einen Systemaufrufe von [**GetComputerName**](/windows/desktop/api/winbase/nf-winbase-getcomputernamea)fest.
