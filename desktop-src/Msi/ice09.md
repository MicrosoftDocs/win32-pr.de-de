# <a name="ice09"></a>ICE09

ICE09 überprüft, ob das permanente Bit für jede für die Installation markierte Komponente im Systemordner festgelegt ist. ICE09 gibt eine Warnung aus, da Sie vermeiden sollten, nicht permanente Systemkomponenten im Systemordner zu installieren. Wenn Sie diese Warnung vermeiden möchten, legen Sie das permanente Bit in der Spalte Attribute der [Komponenten Tabelle](component-table.md) für jede Komponente fest, die für die Installation im Systemordner markiert ist. Wenn die Datenbank keine Komponenten Tabelle enthält, erfolgt keine Validierung.
