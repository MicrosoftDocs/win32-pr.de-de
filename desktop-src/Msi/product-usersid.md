# <a name="productusersid-property"></a>Product. UserSID (Eigenschaft)

Die **UserSID** -Eigenschaft gibt die Benutzer Sicherheits-ID (SID) zurück, unter der diese Produkt Instanz verfügbar ist.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Product.UserSid
```



## <a name="property-value"></a>Eigenschaftswert
