# <a name="properties-table-patchwizdll"></a>Properties-Tabelle (Patchwiz.dll)

Die Properties-Tabelle enthält globale Einstellungen für das Patchpaket. Die Properties-Tabelle ist in der Datenbank für die Patcherstellung (PCP-Datei) erforderlich und wird von der [uikreatepatchpackageex](uicreatepatchpackageex--patchwiz-dll-.md) -Funktion verwendet.

Die Properties-Tabelle enthält die folgenden Spalten.



| Spalte | Typ | Schlüssel | Nullwerte zulässig |
|--------|------|-----|----------|
| Name   | text | J   | N        |
| Wert  | text |     | J        |



 

## <a name="columns"></a>Spalten

<dl> <dt>

<span id="Name"></span><span id="name"></span><span id="NAME"></span>Benennen
</dt> <dd>

Der Name einer patcherstellungs-Eigenschaft.

</dd> <dt>

<span id="Value"></span><span id="value"></span><span id="VALUE"></span>Wert
</dt> <dd>

Der Wert der patcherstellungs-Eigenschaft.

</dd> </dl>
