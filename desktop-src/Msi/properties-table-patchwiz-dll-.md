# <a name="properties-table-patchwizdll"></a>Eigenschaftentabelle (Patchwiz.dll)

Die Eigenschaftentabelle enthält globale Einstellungen für das Patchpaket. Die Properties-Tabelle ist in der Patcherstellungsdatenbank (PCP-Datei) erforderlich und wird von der [UiCreatePatchPackageEx-Funktion](uicreatepatchpackageex--patchwiz-dll-.md) verwendet.

Die Eigenschaftentabelle enthält die folgenden Spalten.



| Spalte | Typ | Key | Nullwerte zulässig |
|--------|------|-----|----------|
| Name   | text | J   | N        |
| Wert  | text |     | J        |



 

## <a name="columns"></a>Spalten

<dl> <dt>

<span id="Name"></span><span id="name"></span><span id="NAME"></span>Namen
</dt> <dd>

Der Name einer Patcherstellungseigenschaft.

</dd> <dt>

<span id="Value"></span><span id="value"></span><span id="VALUE"></span>Wert
</dt> <dd>

Der Wert der Patcherstellungseigenschaft.

</dd> </dl>
