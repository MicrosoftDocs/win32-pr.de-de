# <a name="exit-dialog"></a>Dialogfeld "Beenden"

Das Dialogfeld Beenden wird am Ende einer erfolgreichen Installation angezeigt. Dieses Dialogfeld enthält in der Regel nur Text und eine **Schaltfläche OK.** Der Name dieses Dialogfelds muss in der [Tabelle AdminUISequence](adminuisequence-table.md) und in der [InstallUISequence-Tabelle](installuisequence-table.md) mit -1 als Sequenznummer angezeigt werden. Dies ist ein modales Dialogfeld.
