# <a name="exit-dialog"></a>Dialog Feld Beenden

Das Dialogfeld beenden wird am Ende einer erfolgreichen Installation angezeigt. Dieses Dialogfeld enthält in der Regel nur Text und eine Schaltfläche **OK** . Der Name dieses Dialog Felds muss in der [Tabelle AdminUISequence](adminuisequence-table.md) und in der [Tabelle InstallUISequence](installuisequence-table.md) mit-1 als Sequenznummer angezeigt werden. Dies ist ein modales Dialogfeld.
