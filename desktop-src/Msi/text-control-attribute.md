# <a name="text-control-attribute"></a>Text-Steuerelement Attribut

Dieses Attribut ist die Text Zeichenfolge, die im Steuerelement angezeigt wird. Wenn das Feld 0 des Datensatzes nicht NULL ist, wird der Datensatz bei der Einstellung mithilfe von formattext formatiert. Wenn das Feld 0 NULL ist, definiert das erste Feld des Datensatzes den Text. Beim erzielen des Werts wird immer im ersten Feld zurückgegeben. Bei manchen Steuerelementen ist dieser Text möglicherweise nicht sichtbar. In Windows wird die Zugriffstaste für ein Steuerelement definiert, indem vor dem gewünschten Zeichen in dieser Zeichenfolge ein "&" platziert wird.

## <a name="valid-controls"></a>Gültige Steuerelemente

Alle Steuerelemente außer dem [Zeilen Steuer](line-control.md)Element.

## <a name="associated-bit-flags"></a>Zugehörige Bitflags

Keine.

## <a name="remarks"></a>Bemerkungen

Siehe [Steuerelement Attribute](control-attributes.md) und das Steuerelement, das Sie unter Steuer [Elementen](controls.md)erstellen müssen.
