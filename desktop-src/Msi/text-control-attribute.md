# <a name="text-control-attribute"></a>Textsteuerelementattribut

Dieses Attribut ist die Textzeichenfolge, die im -Steuerelement angezeigt wird. Wenn bei der Einstellung das Feld 0 des Datensatzes nicht NULL ist, wird der Datensatz mit FormatText formatiert. Wenn das Feld 0 NULL ist, definiert das erste Feld des Datensatzes den Text. Beim Abrufen des Werts wird immer im ersten Feld zurückgegeben. Bei einigen Steuerelementen ist dieser Text möglicherweise nicht sichtbar. In Windows wird die Zugriffstaste für ein Steuerelement definiert, indem ein "&" vor dem gewünschten Zeichen in dieser Zeichenfolge platziert wird.

## <a name="valid-controls"></a>Gültige Steuerelemente

Alle Steuerelemente mit Ausnahme des [Liniensteuerelements](line-control.md).

## <a name="associated-bit-flags"></a>Zugeordnete Bitflags

Keine.
