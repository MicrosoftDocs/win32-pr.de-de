# <a name="configurableitemdescription-property"></a>Configurableitem. Description (Eigenschaft)

Die **Description** -Eigenschaft gibt den Wert aus der Beschreibungs Spalte der [ModuleConfiguration-Tabelle](moduleconfiguration-table.md)zurück.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = ConfigurableItem.Description
```
