# <a name="configurableitemdescription-property"></a>ConfigurableItem.Description (Eigenschaft)

Die **Description-Eigenschaft** gibt den Wert aus der Description -Spalte der [ModuleConfiguration-Tabelle zurück.](moduleconfiguration-table.md)

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = ConfigurableItem.Description
```
