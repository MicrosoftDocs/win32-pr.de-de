# <a name="executeaction-property"></a>ExecuteAction (Eigenschaft)

Die **ExecuteAction** -Eigenschaft wird festgelegt, um zu bestimmen, welche Aktion der obersten Ebene von der [ExecuteAction-Aktion](executeaction-action.md) initiiert wird.

## <a name="remarks"></a>Bemerkungen

Die Aktionen der obersten Ebene, wie z. [](install-action.md)b. die [Aktion "installieren", "](advertise-action.md)ankündigen" und " [Administrator](admin-action.md) ", legen die **ExecuteAction** -Eigenschaft auf den Namen der Aktion der obersten Ebene fest.
