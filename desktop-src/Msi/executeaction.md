# <a name="executeaction-property"></a>EXECUTEACTION-Eigenschaft

Die **EXECUTEACTION-Eigenschaft** wird festgelegt, um zu bestimmen, welche Aktion der obersten Ebene von [der ExecuteAction-Aktion](executeaction-action.md) initiiert wird.

## <a name="remarks"></a>Hinweise

Die Aktionen der obersten Ebene, z. B. die [Aktion INSTALL,](install-action.md) [die AKTION ADVERTISE](advertise-action.md)und die [ADMIN-Aktion,](admin-action.md) legen die **EXECUTEACTION-Eigenschaft** auf den Namen der Aktion der obersten Ebene fest.
