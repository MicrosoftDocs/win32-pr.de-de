# <a name="validating-an-installation-upgrade"></a>Überprüfen eines Installationsupgrades

Jedes Mal, wenn Sie Änderungen am Paket vornehmen, sollten Autoren von Upgradepaketen immer eine Validierung für ihre Pakete ausführen, bevor sie versuchen, das Paket zum ersten Mal zu installieren und die Überprüfung erneut auszuführen. Führen Sie die Validierung für ein Upgradepaket auf die gleiche Weise wie für ein Installationspaket aus. Weitere Informationen finden Sie unter [Überprüfen einer Installationsdatenbank.](validating-an-installation-database.md)

Dadurch wird das Beispielupgradepaket abgeschlossen. Um das Upgrade zu installieren, installieren Sie zuerst MNP2000.msi und dann MNP2001.msi. Wenn Sie MNP2001 deinstallieren, werden sowohl das ursprüngliche Produkt als auch das Upgradeprodukt von Ihrem Computer entfernt.
