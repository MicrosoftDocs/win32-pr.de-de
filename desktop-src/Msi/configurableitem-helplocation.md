# <a name="configurableitemhelplocation-property"></a>Configurableitem. helplocation (Eigenschaft)

Die **helplocation** -Eigenschaft gibt den Wert aus der helplocation-Spalte der [ModuleConfiguration-Tabelle](moduleconfiguration-table.md)zurück.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = ConfigurableItem.HelpLocation
```



## <a name="property-value"></a>Eigenschaftswert

## <a name="c"></a>C++

Weitere Informationen finden [**Sie unter Get \_ helplocation**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_helplocation) function.
