<dl> <dt>

<span id="_msi_managed_application_gly"></span><span id="_MSI_MANAGED_APPLICATION_GLY"></span>**verwaltete Anwendung**
</dt> <dd>

System Privilegien, die auch als erweiterte Berechtigungen bezeichnet werden, werden zum Installieren der Anwendung verwendet. Eine Anwendung wird auf einem System verwaltet, wenn Sie eine Computer spezifische Installation unter Windows 2000 ist, oder wenn die Anwendung mithilfe von Gruppenrichtlinie unter Windows 2000 zugewiesen oder veröffentlicht wird.

</dd> <dt>

<span id="_msi_merge_using_windows_installer_gly"></span><span id="_MSI_MERGE_USING_WINDOWS_INSTALLER_GLY"></span>**Merge**
</dt> <dd>

Kombiniert zwei Installations Datenbanken, um eine einzelne Datenbank zu bilden. Weitere Informationen finden Sie unter Zusammenführen von [Datenbanken](merging-databases.md).

</dd> <dt>

<span id="_msi_merge_module_gly"></span><span id="_MSI_MERGE_MODULE_GLY"></span>**Mergemodul**
</dt> <dd>

Die Datenbank wurde mit einem Installationspaket zusammengeführt, um neue Komponenten an eine Anwendung zu übermitteln. Mergemodule können nicht allein installiert werden. Weitere Informationen finden Sie unter zusammen [führen von Modulen](merge-modules.md).

</dd> <dt>

<span id="_msi_microsoft_windows_installer_gly"></span><span id="_MSI_MICROSOFT_WINDOWS_INSTALLER_GLY"></span>**Microsoft Windows Installer**
</dt> <dd>

Client seitiger Installer-Dienst zum Verwalten der Installation von Anwendungen auf 32-Bit-Plattformen. Die Anwendung muss in einem [*Paket*](p-gly.md)gekapselt werden. Weitere Informationen finden Sie unter Informationen [zu Windows Installer](about-windows-installer.md).

</dd> <dt>

<span id="_msi_.msi_file_gly"></span><span id="_MSI_.MSI_FILE_GLY"></span>**MSI-Datei**
</dt> <dd>

COM-strukturierte Speicherdatei, die die Anweisungen und Daten enthält, die zum Installieren einer Anwendung erforderlich sind. Jedes [*Paket*](p-gly.md) enthält mindestens eine MSI-Datei. Die MSI-Datei enthält die [*Installerdatenbank*](i-gly.md), einen [*Zusammenfassungs Datenstrom*](s-gly.md)und möglicherweise eine oder mehrere [*Transformationen*](t-gly.md) und [*interne Quelldateien*](i-gly.md). Weitere Informationen finden Sie unter [Windows Installer Dateierweiterungen](windows-installer-file-extensions.md).

</dd> <dt>

<span id="_msi_.msm_file_gly"></span><span id="_MSI_.MSM_FILE_GLY"></span>**MSM-Datei**
</dt> <dd>

[Mergemodul](merge-modules.md). Weitere Informationen finden Sie unter [Windows Installer Dateierweiterungen](windows-installer-file-extensions.md).

</dd> <dt>

<span id="_msi_.msp_file"></span><span id="_MSI_.MSP_FILE"></span>**MSP-Datei**
</dt> <dd>

[Patchpaket](patch-packages.md). Weitere Informationen finden Sie unter [Windows Installer Dateierweiterungen](windows-installer-file-extensions.md).

</dd> <dt>

<span id="_msi_.mst_file_gly"></span><span id="_MSI_.MST_FILE_GLY"></span>**MST-Datei**
</dt> <dd>

Transformieren Sie das Paket. Weitere Informationen finden Sie unter Zusammenführungen [und Transformationen](merges-and-transforms.md). Weitere Informationen finden Sie unter [Windows Installer Dateierweiterungen](windows-installer-file-extensions.md).

</dd> </dl>
