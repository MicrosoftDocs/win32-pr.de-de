<dl> <dt>

<span id="_msi_managed_application_gly"></span><span id="_MSI_MANAGED_APPLICATION_GLY"></span>**Verwaltete Anwendung**
</dt> <dd>

Systemberechtigungen, auch als erhöhte Rechte bezeichnet, werden verwendet, um die Anwendung zu installieren. Eine Anwendung wird auf einem System verwaltet, wenn es sich um eine Computerinstallation auf Windows 2000 handelt oder wenn die Anwendung mithilfe von Gruppenrichtlinie auf Windows 2000 zugewiesen oder veröffentlicht wird.

</dd> <dt>

<span id="_msi_merge_using_windows_installer_gly"></span><span id="_MSI_MERGE_USING_WINDOWS_INSTALLER_GLY"></span>**Zusammenführen**
</dt> <dd>

Kombiniert zwei Installationsdatenbanken zu einer einzelnen Datenbank. Weitere Informationen finden Sie unter [Zusammenführen von Datenbanken.](merging-databases.md)

</dd> <dt>

<span id="_msi_merge_module_gly"></span><span id="_MSI_MERGE_MODULE_GLY"></span>**Mergemodul**
</dt> <dd>

Die Datenbank wurde mit einem Installationspaket zusammengeführt, um neue Komponenten an eine Anwendung zu liefern. Mergemodule können nicht allein installiert werden. Weitere Informationen finden Sie unter [MergeModule](merge-modules.md).

</dd> <dt>

<span id="_msi_microsoft_windows_installer_gly"></span><span id="_MSI_MICROSOFT_WINDOWS_INSTALLER_GLY"></span>**Microsoft Windows Installer**
</dt> <dd>

Clientseitiger Installerdienst zum Verwalten der Installation von Anwendungen auf 32-Bit-Plattformen. Die Anwendung muss in einem Paket [*gekapselt sein.*](p-gly.md) Weitere Informationen finden Sie unter [Informationen zum Windows Installer.](about-windows-installer.md)

</dd> <dt>

<span id="_msi_.msi_file_gly"></span><span id="_MSI_.MSI_FILE_GLY"></span>**.msi-Datei**
</dt> <dd>

COM-strukturierte Speicherdatei mit den Anweisungen und Daten, die zum Installieren einer Anwendung erforderlich sind. Jedes [*Paket*](p-gly.md) enthält mindestens eine .msi Datei. Die .msi enthält [*die*](i-gly.md)Installer-Datenbank, einen [*Zusammenfassungsinformationsstream*](t-gly.md) und möglicherweise eine oder mehrere Transformationen und interne [*Quelldateien.*](i-gly.md) [](s-gly.md) Weitere Informationen finden Sie unter [Windows Installer-Dateierweiterungen.](windows-installer-file-extensions.md)

</dd> <dt>

<span id="_msi_.msm_file_gly"></span><span id="_MSI_.MSM_FILE_GLY"></span>**MSM-Datei**

</dd> <dt>

<span id="_msi_.msp_file"></span><span id="_MSI_.MSP_FILE"></span>**MSP-Datei**

</dd> <dt>

<span id="_msi_.mst_file_gly"></span><span id="_MSI_.MST_FILE_GLY"></span>**MST-Datei**

</dd> </dl>
