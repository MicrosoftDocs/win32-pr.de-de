# <a name="merge-module-ice-reference"></a>Eisverweis für Mergemodul

In der folgenden Liste finden Sie Links zu den einzelnen mergemoduleis.



| ICEM                 | Kurze Beschreibung des mergemodulice                                                                                                                                                                                                                    |
|----------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [ICEM01](icem01.md) | Überprüft, ob der ICE-Mechanismus funktioniert.                                                                                                                                                                                                             |
| [ICEM02](icem02.md) | Überprüft, ob alle Modul Ausschlüsse und Abhängigkeiten mit dem aktuellen Modul in Beziehung stehen.                                                                                                                                                                      |
| [ICEM03](icem03.md) | Überprüft, ob alle Aktionen im Modul entweder Basis Aktionen sind oder von einer Basis Aktion abgeleitet werden.                                                                                                                                                           |
| [ICEM04](icem04.md) | Überprüft, ob die erforderlichen leeren Tabellen des Mergemoduls tatsächlich leer sind.                                                                                                                                                                                 |
| [ICEM05](icem05.md) | Prüft auf ungültige Zuordnungen zu Komponenten.                                                                                                                                                                                                         |
| [ICEM06](icem06.md) | Überprüft, ob vom Modul ungültige Verweise auf Features angezeigt werden.                                                                                                                                                                                                 |
| [ICEM07](icem07.md) | Überprüft, ob die Reihenfolge der Dateien in den Sequenz Tabellen und in [MergeModule.CABinet](mergemodule-cabinet.md) -Datei entspricht.                                                                                                                               |
| [ICEM08](icem08.md) | Stellt sicher, dass ein Modul nichts ausschließt, von dem es abhängt.                                                                                                                                                                                          |
| [ICEM09](icem09.md) | Überprüft, ob das Mergemodul vordefinierte Verzeichnisse sicher verarbeitet.                                                                                                                                                                                    |

| [ICEM11](icem11.md) | Überprüft, ob ein konfigurierbares Mergemodul die [ModuleConfiguration-Tabelle](moduleconfiguration-table.md) und die [ModuleSubstitution](modulesubstitution-table.md) -Tabelle in der [moduleignoretable-Tabelle](moduleignoretable-table.md) des Moduls auflistet. |
| [ICEM12](icem12.md) | Überprüft, ob in einer modulesequence-Tabelle Standard Aktionen Sequenznummern aufweisen und benutzerdefinierte Aktionen baseaction-und After-Werte aufweisen.                                                                                                                     |
| [ICEM13](icem13.md) | Überprüft, ob Herausgeber Richtlinien und konfigurationsassemblys nicht im Mergemodul enthalten sind.                                                                                                                                                        |
| [ICEM14](icem14.md) | Überprüft die Value-Spalte der [ModuleSubstitution-Tabelle](modulesubstitution-table.md).                                                                                                                                                              |
