# <a name="ice85"></a>ICE85

ICE85 überprüft, ob die Spalte SourceName der [Tabelle "muvefile](movefile-table.md) " ein gültiger langer Dateiname ist. Dieses Feld kann Platzhalter Zeichen ( \* und?) enthalten.

## <a name="result"></a>Ergebnis

ICE85 gibt die folgenden Fehler aus.



| ICE85-Fehler                                                      | BESCHREIBUNG                                                              |
|------------------------------------------------------------------|--------------------------------------------------------------------------|
| SourceName " \[ 2 \] " in der Tabelle "muvefile" weist ein ungültiges Format auf. | Das Feld "SourceName" in der Tabelle "muvefile" ist kein gültiger langer Dateiname. |
