# <a name="text-control"></a>Text Control

Im Text Steuerelement wird statischer Text angezeigt, in dem ein vordefinierter Stil verwendet werden kann.

Die empfohlene Methode zum Anzeigen von Text mit angegebenen Zeilenumbrüchen besteht darin, mehrere einzeilige Text Steuerelemente unter einander zu verwenden. Die Zeichen folgen \\ n, \\ r \\ n oder \\ n \\ r im Textfeld für das-Steuerelement werden nicht als Zeilenumbruch angezeigt. Diese Zeichen folgen werden buchstäblich vom-Steuerelement angezeigt.

## <a name="remarks"></a>Bemerkungen

Dieses Steuerelement kann mithilfe der Funktion "up- [**windowex**](/windows/win32/api/winuser/nf-winuser-createwindowexa) " aus der statischen Klasse erstellt werden. Sie verfügt über die Stile **SS \_ left**, **WS \_ Child** und **WS \_ Group** .

Platzieren Sie keine transparenten Text Steuerelemente oberhalb von farbigen Bitmaps. Der Text ist möglicherweise nicht sichtbar, wenn der Benutzer das Anzeige Farbschema ändert. Beispielsweise kann Text unsichtbar werden, wenn der Benutzer den Parameter für hohe Kontraste aus Barrierefreiheits Gründen festlegt.
