# <a name="mediasourcedir-property"></a>Mediasourcedir (Eigenschaft)

Der Installer legt die **mediasourcedir** -Eigenschaft auf 1 fest, wenn die Installation eine Quelle verwendet, die sich auf einem Medium, z. b. einer CD-ROM, befindet. Diese Eigenschaft wird nicht festgelegt, wenn die Installation eine Quelle verwendet, die sich an einem Netzwerk Speicherort befindet. Beispielsweise verwendet das [SelectionTree-Steuer](selectiontree-control.md) Element **mediasourcedir** , um zu bestimmen, ob die Installation von der Quelle ausgeführt oder von einem Netzwerk Speicherort aus ausgeführt wird.
