# <a name="programfiles64folder-property"></a>ProgramFiles64Folder-Eigenschaft

Der Installer legt die **ProgramFiles64Folder** -Eigenschaft auf den vollständigen Pfad des vordefinierten 64-Bit-Programmdatei Ordners fest. Die vorhandene [**ProgramFilesFolder**](programfilesfolder.md) -Eigenschaft wird auf den entsprechenden 32-Bit-Ordner festgelegt.

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft wird vom Installationsprogramm festgelegt. Beispielsweise kann auf 64-Bit-Fenstern der Wert "C: \\ Program Files" lauten. Diese Eigenschaft wird auf 32-Bit-Fenstern nicht verwendet.
