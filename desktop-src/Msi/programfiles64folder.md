# <a name="programfiles64folder-property"></a>ProgramFiles64Folder (Eigenschaft)

Das Installationsprogramm legt die **ProgramFiles64Folder-Eigenschaft** auf den vollständigen Pfad des vordefinierten Ordners 64-Bit-Programmdateien fest. Die vorhandene [**ProgramFilesFolder-Eigenschaft**](programfilesfolder.md) wird auf den entsprechenden 32-Bit-Ordner festgelegt.

## <a name="remarks"></a>Hinweise

Das Installationsprogramm legt diese Eigenschaft fest. Auf 64-Bit-Windows kann der Wert beispielsweise C: \\ Programme sein. Diese Eigenschaft wird nicht auf 32-Bit-Windows.
