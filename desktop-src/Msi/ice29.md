# <a name="ice29"></a>ICE29

ICE29 überprüft, ob abgeschnittene Datenstromnamen eindeutig bleiben. Jede Tabelle mit einer Binary- oder Object-Spalte wird überprüft. Weitere Informationen finden Sie unter Datentyp der [Binärspalte.](binary.md)

Die Verarbeitung von Streams durch die Strukturierte Speicherimplementierung von Win32 OLE schränkt Streamnamen ein. Weitere Informationen finden Sie unter [OLE-Einschränkungen für Streams](ole-limitations-on-streams.md). Das Installationsprogramm kann Streamnamen mit einer Länge von bis zu 62 Zeichen komprimieren. Namen, die länger als diese sind, werden abgeschnitten.
