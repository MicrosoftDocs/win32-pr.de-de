# <a name="ice29"></a>ICE29

ICE29 überprüft, ob abgeschnittene Streamnamen eindeutig sind. Jede Tabelle, die eine Binär-oder Objekt Spalte aufweist, wird überprüft. Weitere Informationen finden Sie unter Datentyp der [Binär](binary.md) Spalte.

Durch die Verarbeitung von Streams durch die Win32 OLE-strukturierte Speicher Implementierung werden Streamnamen eingeschränkt. Siehe [OLE-Einschränkungen für Datenströme](ole-limitations-on-streams.md). Der Installer kann Streamnamen mit bis zu 62 Zeichen komprimieren. Namen, die länger sind als diese, werden abgeschnitten.
