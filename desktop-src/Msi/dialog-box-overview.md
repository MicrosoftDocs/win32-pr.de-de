# <a name="dialog-box-overview"></a><span data-ttu-id="ebc2f-103">Dialog Feld Übersicht</span><span class="sxs-lookup"><span data-stu-id="ebc2f-103">Dialog Box Overview</span></span>

<span data-ttu-id="ebc2f-104">Das Erstellen eines Dialog Felds in Windows Installer ähnelt dem Prozess der programmgesteuerten Erstellung eines Dialog Felds mithilfe einer Microsoft Windows-API-Dialogfeld Vorlage.</span><span class="sxs-lookup"><span data-stu-id="ebc2f-104">The process of creating a dialog box in Windows Installer is similar to the process of creating a dialog box programmatically using a Microsoft Windows API dialog box template.</span></span> <span data-ttu-id="ebc2f-105">Während eine Windows-Dialogfeld Vorlage in einer null-terminierten –-Zeichenfolge gespeichert wird, speichert Windows Installer die Dialogfeld Parameter in der Dialogfeld Tabelle.</span><span class="sxs-lookup"><span data-stu-id="ebc2f-105">While a Windows dialog box template is stored in a null terminated–character string, Windows Installer stores the dialog box parameters in the Dialog table.</span></span> <span data-ttu-id="ebc2f-106">Die Dialog Feld Tabelle enthält eine Spalte Attribute, die analog zu Fenster Stilen in der Microsoft Windows-API für die Benutzeroberfläche ist.</span><span class="sxs-lookup"><span data-stu-id="ebc2f-106">The Dialog table contains an attributes column that is analogous to Window styles in the Microsoft Windows user interface API.</span></span> <span data-ttu-id="ebc2f-107">Die Anzahl der Dialogfeld Stil-Bits in Windows Installer ist jedoch ein reduzierter und spezieller Satz.</span><span class="sxs-lookup"><span data-stu-id="ebc2f-107">However, the number of dialog style bits in Windows Installer is a reduced and specialized set.</span></span>

<span data-ttu-id="ebc2f-108">Um das Dialogfeld mithilfe der API der Windows-Benutzeroberfläche physisch zu erstellen, wird [**Dialogbox**](/windows/win32/api/winuser/nf-winuser-dialogboxa) aufgerufen und übergibt einen Zeiger auf die Vorlage.</span><span class="sxs-lookup"><span data-stu-id="ebc2f-108">To physically create the dialog box using the Windows user interface API, [**DialogBox**](/windows/win32/api/winuser/nf-winuser-dialogboxa) is called and passes a pointer to the template.</span></span> <span data-ttu-id="ebc2f-109">In Windows Installer wird das Dialogfeld während der Ausführung einer der drei UI-Sequenz Tabellen erstellt.</span><span class="sxs-lookup"><span data-stu-id="ebc2f-109">In Windows Installer, the dialog box is created during execution of one of the three UI sequence tables.</span></span>

<span data-ttu-id="ebc2f-110">Windows Installer enthält keine Standardbenutzer Oberfläche, die Paket Autoren für Ihre Pakete verwenden können.</span><span class="sxs-lookup"><span data-stu-id="ebc2f-110">Windows Installer does not contain a default UI that package authors can utilize for their packages.</span></span> <span data-ttu-id="ebc2f-111">Sie enthält einen begrenzten Satz von Standard Dialogfeldern, die Fehler-und Informationsmeldungen anzeigen. <span data-ttu-id="ebc2f-114">Einige Standard Aktionen und interne Windows Installer Fehlerbedingungen geben einen bestimmten Satz von Benutzeroberflächen Daten zurück und benötigen daher ein Dialogfeld mit einer bestimmten Gruppe von Steuerelementen, um diese Daten zu unterstützen.</span><span class="sxs-lookup"><span data-stu-id="ebc2f-114">Some standard actions and internal Windows Installer error conditions return a specific set of UI data and therefore require a dialog box with a specific set of controls to accommodate that data.</span></span> <span data-ttu-id="ebc2f-115">Windows Installer überprüft zwei separate Speicherorte für die Bezeichner für diese Dialogfelder.</span><span class="sxs-lookup"><span data-stu-id="ebc2f-115">Windows Installer checks two separate locations for the identifiers for these dialog boxes.</span></span>

-   <span data-ttu-id="ebc2f-116">Windows Installer enthält einen Satz von eingebetteten Dialogfeld Namen.</span><span class="sxs-lookup"><span data-stu-id="ebc2f-116">Windows Installer contains a set of embedded dialog box names.</span></span> <span data-ttu-id="ebc2f-117">Die benutzerdefinierte Aktion fragt die Dialog Tabelle nach diesen reservierten Namen ab.</span><span class="sxs-lookup"><span data-stu-id="ebc2f-117">The custom action queries the Dialog table for these reserved names.</span></span>
-   <span data-ttu-id="ebc2f-118">In jeder der drei UI-Sequenz Tabellen werden Dialog Namen mit der Sequenznummer-1,-2 oder-3 als Reaktion auf beenden, vom Benutzer angeforderte Beendigung und schwerwiegende Fehlermeldungen aus Windows Installer angezeigt.</span><span class="sxs-lookup"><span data-stu-id="ebc2f-118">In each of the three UI sequence tables, dialog names with a sequence number of -1,-2, or -3 are displayed in response to exit, user requested exit, and fatal error messages from Windows Installer.</span></span>
