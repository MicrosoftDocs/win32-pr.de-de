# <a name="sessiondatabase-property"></a>Session.Database (Eigenschaft)

Die **Database-Eigenschaft** des [**Session-Objekts**](session-object.md) ist eine schreibgeschützte Eigenschaft, die die Datenbank für die aktuelle Installationssitzung als Datenbankobjekt [**zurückgibt.**](database-object.md)

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Session.Database
```



## <a name="property-value"></a>Eigenschaftswert
