# <a name="sessiondatabase-property"></a>Session. Database (Eigenschaft)

Die **Database** -Eigenschaft des [**Session**](session-object.md) -Objekts ist eine schreibgeschützte Eigenschaft, die die Datenbank für die aktuelle Installationssitzung als [**Daten Bank**](database-object.md) Objekt zurückgibt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Session.Database
```



## <a name="property-value"></a>Eigenschaftswert
