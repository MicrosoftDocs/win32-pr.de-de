# <a name="resume-property"></a>RESUME-Eigenschaft

Die **RESUME-Eigenschaft** wird festgelegt, wenn eine angehaltene Installation fortgesetzt wird. Diese Eigenschaft kann bestimmen, welcher Text auf der Benutzeroberfläche angezeigt werden soll. Wenn sich der Benutzer beispielsweise nicht im Wiederaufnahmemodus befindet, könnte er gefragt werden: "Sind Sie bereit für die Installation?" Während sie sich im Wiederaufnahmemodus befinden, kann der Benutzer gefragt werden, ob Sie bereit sind, ihre Installation abzuschließen.

## <a name="remarks"></a>Hinweise

Aus Gründen der Abwärtskompatibilität unterstützt das Installationsprogramm auch den Namen **Resume** für diese Eigenschaft.
