# <a name="resume-property"></a>Resume-Eigenschaft

Die **Resume** -Eigenschaft wird festgelegt, wenn eine angehaltene Installation fortgesetzt wird. Mit dieser Eigenschaft kann festgelegt werden, welcher Text auf der Benutzeroberfläche angezeigt werden soll. Wenn sich beispielsweise der Benutzer nicht im fort Setz Modus befindet, könnte der Benutzer gefragt werden: "sind Sie bereit für die Installation?" im Fortsetzungs Modus könnte der Benutzer gefragt werden: "sind Sie bereit, die Installation abzuschließen?".

## <a name="remarks"></a>Bemerkungen

Aus Gründen der Abwärtskompatibilität unterstützt das Installationsprogramm auch den Namen **Resume** für diese Eigenschaft.
