# <a name="user-interface-wizard-behavior"></a>Verhalten des Benutzeroberflächen-Assistenten

Um eine einfache Benutzeroberfläche zu erstellen, können Entwickler von Installationspaketen einen Benutzeroberflächen Entwurf verwenden, der dem Schnittstellen-Assistenten Verhalten entspricht.

Der Begriff "Verhalten der Benutzeroberfläche" bedeutet, dass jedes Dialogfeld in einer Sequenz eine Schaltfläche " **Next>>** " enthält, auf die der Benutzer klickt, um nach dem Eingeben von Daten oder dem Konfigurieren von Informationen im aktuellen Dialogfeld zum nächsten Dialogfeld zu wechseln. Wenn sich der Benutzer entscheidet, zurückzugehen und alle Informationen zu ändern, die in einem vorherigen Dialogfeld eingegeben wurden, enthält jedes Dialogfeld eine **<<** Schaltfläche "zurück", auf die der Benutzer klickt, um zurückzukehren. Am Ende der Sequenz des Assistenten klickt der Benutzer auf die Schaltfläche **Fertig** stellen, um den Installationsvorgang zu starten.

Die im Benutzeroberflächen Beispiel beschriebene Benutzeroberfläche entspricht dem Verhalten des Benutzeroberflächen-Assistenten. Sehen Sie sich [ein Installations Beispiel an](an-installation-example.md).
