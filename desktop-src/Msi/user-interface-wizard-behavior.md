# <a name="user-interface-wizard-behavior"></a>Benutzeroberfläche des Assistenten

Um eine einfache Benutzeroberfläche zu erstellen, können Entwickler von Installationspaketen einen Benutzeroberflächenentwurf verwenden, der dem Verhalten des Schnittstellen-Assistenten entspricht.

Der Begriff Benutzeroberflächen-Assistentverhalten bedeutet, dass jedes Dialogfeld in einer Sequenz eine Schaltfläche Weiter **>>** enthält, auf die der Benutzer klickt, um zum nächsten Dialogfeld zu wechseln, nachdem er Daten eingeben oder Informationen im aktuellen Dialogfeld konfiguriert hat. Wenn sich der Benutzer entschließt, zurück zu wechseln und informationen zu  ändern, die in einem vorherigen Dialogfeld eingegeben wurden, enthält jedes Dialogfeld eine Schaltfläche<<Zurück, auf die der Benutzer klickt, um zurück zu wechseln. Am Ende der Assistentensequenz klickt der  Benutzer auf die Schaltfläche Fertig stellen, um den Installationsvorgang zu starten.

Die im Beispiel für die Benutzeroberfläche beschriebene Benutzeroberfläche entspricht dem Verhalten des Benutzeroberflächen-Assistenten. Weitere Informationen [finden Sie unter Installationsbeispiel.](an-installation-example.md)
