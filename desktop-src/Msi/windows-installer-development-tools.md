# <a name="windows-installer-development-tools"></a>Windows Installationsentwicklungstools

Die folgenden Tools sind nur in den [Windows SDK-Komponenten für Windows Installer-Entwickler](platform-sdk-components-for-windows-installer-developers.md)verfügbar.



| Hilfsprogramm                          | Beschreibung                                                                                                                                                                                                                                      |
|----------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Instmsi.exe](instmsi-exe.md)   | Redistributable package for installing the Windows Installer on Windows operating systems earlier than Windows Me.                                                                                                                               |
| [Msicert.exe](msicert-exe.md)   | Füllt die Tabelle MsiDigitalSignature und die Tabelle MsiDigitalCertificate mit den Informationen zur digitalen Signatur auf, die zu externen Cab-Dateien in der Tabelle Media gehören.                                                                           |
| [Msidb.exe](msidb-exe.md)       | Importiert und exportiert Datenbanktabellen und -streams, führt Datenbanken zusammen und wendet Transformationen an.                                                                                                                                                       |
| [Msifiler.exe](msifiler-exe.md) | Füllt die Dateitabelle mit Dateiversionen, Sprachen und Größen basierend auf einem Quellverzeichnis auf. Außerdem kann die MsiFileHash-Tabelle mit Dateihashes aktualisiert werden.                                                                                      |
| [Msiinfo.exe](msiinfo-exe.md)   | Bearbeitet oder zeigt [den Zusammenfassungsinformationsdatenstrom](summary-information-stream.md)an.                                                                                                                                                                  |
| [Msimerg.exe](msimerg-exe.md)   | Führt eine Datenbank in einer anderen zusammen.                                                                                                                                                                                                                |
| [Msimsp.exe](msimsp-exe.md)     | Patcherstellungstool. Die empfohlene Methode zum Generieren eines Patchpakets besteht darin, ein Patcherstellungstool wie Msimsp.exe mit PATCHWIZ.DLL zu verwenden.                                                                                                 |
| [Msistuff.exe](msistuff-exe.md) | Zeigt die Ressourcen im Setup.exe ausführbaren Bootstrapdatei an oder konfiguriert sie.                                                                                                                                                                      |
| [Msitool.mak](msitool-mak.md)   | Makefile, das zum Erstellen von Tools und benutzerdefinierten Aktionen verwendet werden kann.                                                                                                                                                                                      |
| [Msitran.exe](msitran-exe.md)   | Generiert eine Transformation oder wendet eine Transformationsdatei auf eine Datenbank an.                                                                                                                                                                                 |
| [Msival2.exe](msival2-exe.md)   | Führt eine oder mehrere [interne Konsistenzauswertungen](internal-consistency-evaluators-ices.md)aus: ICEs.                                                                                                                                       |
| [Msizap.exe](msizap-exe.md)     | Entfernt Windows Installer-Informationen für ein Produkt oder alle produkte, die auf einem Computer installiert sind.                                                                                                                                                      |
| [Orca.exe](orca-exe.md)         | Datenbank-Editor. Erstellt und bearbeitet .msi Dateien und Mergemodule.                                                                                                                                                                                 |
| [PATCHWIZ.DLL](patchwiz-dll.md) | Generiert ein Windows [Installer-Patchpaket](patch-packages.md) aus einer Patcherstellungseigenschaftendatei (PCP-Datei). Die empfohlene Methode zum Generieren eines Patchpakets besteht darin, ein Patcherstellungstool wie Msimsp.exe mit PATCHWIZ.DLL zu verwenden. |
| [Wilogutl.exe](wilogutl-exe.md) | Unterstützt die Analyse von Protokolldateien aus einer installation Windows Installer und zeigt vorgeschlagene Lösungen für Fehler an.                                                                                                                              |
