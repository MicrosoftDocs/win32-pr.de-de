# <a name="windows-installer-development-tools"></a>Entwicklungs Tools für Windows Installer

Die folgenden Tools sind nur in den [Windows SDK Komponenten für Windows Installer Entwickler](platform-sdk-components-for-windows-installer-developers.md)verfügbar.



| Hilfsprogramm                          | BESCHREIBUNG                                                                                                                                                                                                                                      |
|----------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Instmsi.exe](instmsi-exe.md)   | Verteilbares Paket für die Installation des Windows Installer auf Windows-Betriebssystemen vor Windows Me.                                                                                                                               |
| [Msicert.exe](msicert-exe.md)   | Füllt die msidigitalsignature-Tabelle und die msidigitalcertificate-Tabelle mit den digitalen Signatur Informationen aus, die zu externen CAB-Dateien in der Medien Tabelle gehören.                                                                           |
| [Msidb.exe](msidb-exe.md)       | Importieren und Exportieren von Datenbanktabellen und Streams, Zusammenführungen von Datenbanken und Anwenden von Transformationen.                                                                                                                                                       |
| [Msifiler.exe](msifiler-exe.md) | Füllt die Dateitabelle mit Dateiversionen, Sprachen und Größen auf Grundlage eines Quell Verzeichnisses auf. Außerdem kann die msiflehash-Tabelle mit Dateihashes aktualisiert werden.                                                                                      |
| [Msiinfo.exe](msiinfo-exe.md)   | Bearbeitet oder zeigt einen [Zusammenfassungs Informationsdaten Strom](summary-information-stream.md)an.                                                                                                                                                                  |
| [Msimerg.exe](msimerg-exe.md)   | Führt eine Datenbank in einer anderen zusammen.                                                                                                                                                                                                                |
| [Msimsp.exe](msimsp-exe.md)     | Tool zum Erstellen von Patches. Die empfohlene Methode zum Erstellen eines Patchpakets ist die Verwendung eines patcherstellungs Tools, wie z. b. Msimsp.exe mit PATCHWIZ.DLL.                                                                                                 |
| [Msistuff.exe](msistuff-exe.md) | Hiermit werden die Ressourcen in der ausführbaren Datei Setup.exe Bootstrap angezeigt oder konfiguriert.                                                                                                                                                                      |
| [Msitool. MAK](msitool-mak.md)   | Makefile, das zum Erstellen von Tools und benutzerdefinierten Aktionen verwendet werden kann.                                                                                                                                                                                      |
| [Msitran.exe](msitran-exe.md)   | Generiert eine Transformation oder wendet eine Transformations Datei auf eine Datenbank an.                                                                                                                                                                                 |
| [Msival2.exe](msival2-exe.md)   | Führt eine oder eine Suite [interner Konsistenz Auswertung-ICES](internal-consistency-evaluators-ices.md)aus.                                                                                                                                       |
| [Msizap.exe](msizap-exe.md)     | Entfernt Windows Installer Informationen für ein Produkt oder alle auf einem Computer installierten Produkte.                                                                                                                                                      |
| [Orca.exe](orca-exe.md)         | Datenbank-Editor. Erstellt und bearbeitet MSI-Dateien und Mergemodule.                                                                                                                                                                                 |
| [PATCHWIZ.DLL](patchwiz-dll.md) | Generiert ein Windows Installer [Patchpaket](patch-packages.md) aus einer patcherstellungs-Eigenschaften Datei (PCP-Datei). Die empfohlene Methode zum Erstellen eines Patchpakets ist die Verwendung eines patcherstellungs Tools, wie z. b. Msimsp.exe mit PATCHWIZ.DLL. |
| [Wilogutl.exe](wilogutl-exe.md) | Unterstützt die Analyse von Protokolldateien aus einer Windows Installer-Installation und zeigt vorgeschlagene Lösungen für Fehler an.                                                                                                                              |
