# <a name="diskprompt-property"></a>DiskPrompt-Eigenschaft

Der **DiskPrompt-Eigenschaftswert** enthält eine Zeichenfolge, die in einem Dialogfeld angezeigt wird, in dem zur Eingabe eines Datenträgers aufgefordert wird. Diese Zeichenfolge sollte den Namen des Produkts und einen Platzhalter für den Text enthalten, der auf der CD-ROM-Oberfläche gedruckt wird.

## <a name="remarks"></a>Hinweise

Wird als Produkteigenschaft angekündigt.
