# <a name="position-control-attribute"></a>Positionssteuerelementattribut

Dieses Attribut bestimmt die Position des Steuerelements im Dialogfeld. Die vier Felder des Attributs sind: links, oben, Breite und Höhe. Geben Sie die Breite, Höhe und Koordinaten der linken Ecke des Steuerelements in die Spalten Width, Height, X und Y der [Control-Tabelle](control-table.md)ein. Verwenden Sie [Installationseinheiten](installer-units.md) für Längen- und Entfernungseinheiten.

## <a name="valid-controls"></a>Gültige Steuerelemente

Alle aktiven Steuerelemente.

## <a name="associated-bit-flags"></a>Zugeordnete Bitflags

Dieses Attribut verwendet keine Bitflags.

## <a name="remarks"></a>Hinweise

Weitere Informationen finden Sie unter [Steuerelementattribute](control-attributes.md) und das Steuerelement, das Sie unter [Steuerelemente](controls.md)erstellen müssen.

Eine Installationseinheit entspricht einem Zwölftel der Höhe des 10-Punkt-MS Sans Serif-Schriftgrads.
