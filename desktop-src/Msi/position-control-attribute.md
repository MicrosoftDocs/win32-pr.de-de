# <a name="position-control-attribute"></a>Positions Steuerungs Attribut

Dieses Attribut bestimmt die Position des-Steuer Elements im Dialogfeld. Die vier Felder des Attributs lauten: Left, Top, Width und Height. Geben Sie die Breite, Höhe und Koordinaten der Steuerelemente der linken Ecke des Steuer Elements in die Spalten Width, Height, X und Y der [Steuerelement Tabelle](control-table.md)ein. Verwenden Sie [Installer-Einheiten](installer-units.md) für Längen-und Entfernungs Einheiten.

## <a name="valid-controls"></a>Gültige Steuerelemente

Alle aktiven Steuerelemente.

## <a name="associated-bit-flags"></a>Zugehörige Bitflags

Dieses Attribut verwendet keine Bitflags.

## <a name="remarks"></a>Bemerkungen

Siehe [Steuerelement Attribute](control-attributes.md) und das Steuerelement, das Sie unter Steuer [Elementen](controls.md)erstellen müssen.

Eine installereinheit ist gleich 1-zwölfte Höhe des 10-Punkt-MS Sans Serif-Schrift Grads.
