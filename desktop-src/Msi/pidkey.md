# <a name="pidkey-property"></a>PIDKEY (Eigenschaft)

Die **PIDKEY** -Eigenschaft enthält den Teil der vom Benutzer eingegebenen Produkt-ID. Mit der [ValidateProductID-Aktion](validateproductid-action.md) wird die eingegebene Produkt-ID überprüft, indem die **PIDKEY** -Eigenschaft mit der [**pidtemplate**](pidtemplate.md) -Eigenschaft verglichen wird.

Die **PIDKEY** -Eigenschaft kann in der Befehlszeile oder über die Benutzeroberfläche festgelegt werden. Diese Eigenschaft enthält nur den Teil der Produkt-ID, die vom Benutzer eingegeben wird.
