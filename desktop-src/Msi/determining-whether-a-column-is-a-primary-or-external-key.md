# <a name="determining-whether-a-column-is-a-primary-or-external-key"></a>Bestimmen, ob eine Spalte ein Primärschlüssel oder ein externer Schlüssel ist

Sie können bestimmen, ob eine bestimmte Spalte ein Primärschlüssel oder ein externer Schlüssel der Tabelle ist, indem Sie die Verweisseite der Tabelle konsultieren. Eine Liste mit Links zu den Referenzseiten der Tabellen mit den Installations Datenbanken finden Sie unter [Datenbanktabellen](database-tables.md). Jede der Verweis Seiten identifiziert Spalten, bei denen es sich um einen Primärschlüssel oder einen externen Schlüssel handelt.

Bei Daten Bank Spalten, bei denen es sich um externe Schlüssel handelt, wird der Name der Primärschlüssel Spalte mit einem Unterstrich hinzugefügt. Ein externer Schlüssel für die Datei Spalte der Dateitabelle lautet z. b. immer mit dem Namen "file" \_ .

Informationen zum Anzeigen von Entitäts Beziehungs Diagrammen, die Tabellen Beziehungen veranschaulichen, finden Sie unter [Kern Tabellen Gruppe](core-tables-group.md), [Datei Tabellen Gruppe](file-tables-group.md), [Registrierungs Tabellen Gruppe](registry-tables-group.md), [System Tabellen Gruppe](system-tables-group.md)oder [Benutzeroberflächen Schema](user-interface-schema.md).
