# <a name="determining-whether-a-column-is-a-primary-or-external-key"></a>Bestimmen, ob eine Spalte ein Primärschlüssel oder ein externer Schlüssel ist

Sie können ermitteln, ob eine bestimmte Spalte ein Primärschlüssel oder ein externer Schlüssel der Tabelle ist, indem Sie die Referenzseite der Tabelle aufrufen. Eine Liste mit Links zu den Referenzseiten aller Installationsdatenbanktabellen finden Sie unter [Datenbanktabellen.](database-tables.md) Jede der Verweisseiten identifiziert Spalten, die ein Primärschlüssel oder ein externer Schlüssel sind.

Datenbankspalten, bei denen es sich um externe Schlüssel handelt, verwenden den Namen der Primärschlüsselspalte mit einem zusätzlichen Unterstrich. Beispielsweise hat ein externer Schlüssel für die Spalte Datei der Dateitabelle immer den Namen Datei \_ .

Informationen zum Anzeigen von Entitätsbeziehungsdiagrammen, die Tabellenbeziehungen veranschaulichen, finden Sie unter [Kerntabellengruppe,](core-tables-group.md) [Dateitabellengruppe,](file-tables-group.md) [Registrierungstabellengruppe,](registry-tables-group.md) [Systemtabellengruppe](system-tables-group.md)oder [Benutzeroberfläche Schema.](user-interface-schema.md)
