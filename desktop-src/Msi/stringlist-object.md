# <a name="stringlist-object"></a>Stringlist-Objekt

Das **stringlist** -Objekt ist eine Auflistung von Zeichen folgen. Der Client muss überprüfen, ob das **stringlist** -Objekt vorhanden und nicht leer ist, bevor auf seine Eigenschaften verwiesen wird.

## <a name="members"></a>Member

Das **stringlist** -Objekt verfügt über diese Typen von Membern:

-   [Eigenschaften](#properties)

### <a name="properties"></a>Eigenschaften

Das **stringlist** -Objekt verfügt über diese Eigenschaften.



| Eigenschaft                                     | BESCHREIBUNG                                                          |
|:---------------------------------------------|:---------------------------------------------------------------------|
| [**Countdown**](stringlist-count.md)<br/> | Gibt die Anzahl der Elemente im **stringlist** -Objekt zurück.<br/> |
| [**Element**](stringlist-item.md)<br/>   | Gibt eine Zeichenfolge in der **stringlist** -Objekt Auflistung zurück.<br/> |
