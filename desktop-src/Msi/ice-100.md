# <a name="ice-100"></a>Ice 100

ICE100 überprüft die Erstellung der [msiembeddedui](msiembeddedui-table.md) -Tabelle und der [msiembeddedchainer](msiembeddedchainer-table.md) -Tabelle.

## <a name="result"></a>Ergebnis

ICE100 gibt die folgenden Fehler aus.



| ICE100-Fehler                                                  | BESCHREIBUNG                                                                                                                                                                                                                                             |
|---------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Zu viele DLLs, die als Benutzeroberfläche gekennzeichnet sind: \[ 1\]                             | Die Spalte Attribute der Tabelle [msiembeddedui](msiembeddedui-table.md) enthält zu viele DLLs, die als Benutzeroberflächen-DLL gekennzeichnet sind. Nur eine DLL muss als Benutzeroberflächen-DLL gekennzeichnet werden.                                                               |
| Die UI-DLL darf keinen NULL/NULL-Nachrichtenfilter aufweisen: \[ 1\]            | Eine DLL, die in der Spalte Attribute der Tabelle [msiembeddedui](msiembeddedui-table.md) als Benutzeroberflächen-DLL gekennzeichnet ist, weist im Feld MessageFilter den Wert NULL auf. Wenn eine Zeile auf eine Benutzeroberflächen-DLL verweist, sollte der Wert im Feld MessageFilter nicht NULL sein. |
| Ressourcen-DLLs müssen Nachrichtenfilter NULL/NULL aufweisen: \[ 1\]      | Eine DLL, die in der Tabelle " [msiembeddedui](msiembeddedui-table.md) " nicht als Benutzeroberflächen-DLL gekennzeichnet ist, muss im Feld "MessageFilter" den Wert NULL aufweisen. Dieses Feld sollte NULL sein, wenn eine Zeile auf eine Ressourcen Datei verweist und der Wert von Attributen NULL ist.            |
| Mehrere DLLs, die mit diesem Dateinamen gekennzeichnet sind: \[ 2 \] (Index: \[ 1 \] ) | Mehrere DLLs in der Tabelle " [msiembeddedui](msiembeddedui-table.md) " haben denselben Dateinamen. Die FileName-Spalte darf keine doppelten Namen enthalten.                                                                                                    |



 

ICE100 gibt die folgende Warnung aus.



| ICE100-Warnung                                                                                          | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                         |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| In der Tabelle "msiembeddecodchainer" sind mehrere Chainer vorhanden. Stellen Sie sicher, dass nur eine für die ausgeführt werden kann. | In der Tabelle " [msiembeddecodchainer](msiembeddedchainer-table.md) " sind mehrere Einträge vorhanden. Es kann nur ein eingebetteter Chainer gestartet werden. Wenn mehrere Einträge für die Durchführung von bedingt sind, ist nicht definiert, welcher Eintrag ausgeführt wird. Wenn der Benutzer sicher ist, dass nur die Bedingung eines Eintrags in "true" aufgelöst werden kann, kann diese Warnung ignoriert werden. |
