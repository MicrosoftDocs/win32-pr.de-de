# <a name="ice-100"></a>ICE 100

ICE100 überprüft die Erstellung der [MsiEmbeddedUI-Tabelle](msiembeddedui-table.md) und der [MsiEmbeddedChainer-Tabelle.](msiembeddedchainer-table.md)

## <a name="result"></a>Ergebnis

ICE100 sendet die folgenden Fehler.



| ICE100-Fehler                                                  | Beschreibung                                                                                                                                                                                                                                             |
|---------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Zu viele DLLs, die als Benutzeroberfläche markiert sind: \[ 1\]                             | Die Spalte Attributes der [MsiEmbeddedUI-Tabelle](msiembeddedui-table.md) enthält zu viele DLLs, die als BENUTZEROBERFLÄCHEN-DLL markiert sind. Nur eine DLL sollte als BENUTZEROBERFLÄCHEN-DLL gekennzeichnet werden.                                                               |
| DIE UI-DLL darf keinen NULL/Null-Nachrichtenfilter aufweisen: \[ 1\]            | Eine DLL, die in der Spalte Attribute der [MsiEmbeddedUI-Tabelle](msiembeddedui-table.md) als BENUTZEROBERFLÄCHEN-DLL markiert ist, weist null im Feld MessageFilter auf. Wenn eine Zeile auf eine Benutzeroberflächen-DLL verweist, darf der Wert im Feld MessageFilter nicht NULL sein. |
| Ressourcen-DLLs müssen NULL/Null-Nachrichtenfilter aufweisen: \[ 1\]      | Eine DLL, die nicht als Benutzeroberflächen-DLL in der [MsiEmbeddedUI-Tabelle](msiembeddedui-table.md) markiert ist, muss null im Feld MessageFilter enthalten. Dieses Feld sollte NULL sein, wenn eine Zeile auf eine Ressourcendatei verweist und der Wert von Attribute NULL ist.            |
| Mehrere DLLs, die mit diesem Dateinamen gekennzeichnet sind: \[ 2 \] (Index: \[ 1 \] ) | Mehrere DLLs in der [MsiEmbeddedUI-Tabelle](msiembeddedui-table.md) haben den gleichen Dateinamen. Die FileName -Spalte darf keine doppelten Namen enthalten.                                                                                                    |



 

ICE100 gibt die folgende Warnung aus.



| ICE100-Warnung                                                                                          | Beschreibung                                                                                                                                                                                                                                                                                                                         |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| In der MsiEmbeddedChainer-Tabelle sind mehrere Chainer vorhanden. Stellen Sie sicher, dass nur eine ausführungsbedingt ist. | Die [MsiEmbeddedChainer-Tabelle](msiembeddedchainer-table.md) enthält mehrere Einträge. Es kann nur ein eingebetteter Chainer gestartet werden. Wenn mehrere Einträge zur Ausführung konditioniert sind, ist nicht definiert, welcher Eintrag ausgeführt wird. Wenn der Benutzer sicher ist, dass nur die Bedingung eines Eintrags in TRUE aufgelöst werden kann, kann diese Warnung ignoriert werden. |
