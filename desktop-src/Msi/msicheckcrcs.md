# <a name="msicheckcrcs-property"></a>Msicheckcrcs (Eigenschaft)

Die zyklische Redundanz Überprüfung (CRC) ist mit Windows Installer Version 2,0 und höher verfügbar. Das Installationsprogramm führt ein CRC für Dateien nur aus, wenn die **msicheckcrcs** -Eigenschaft festgelegt ist. Weitere Informationen finden Sie unter [CRC-Überprüfung während einer Installation](crc-checking-during-an-installation.md).

|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

Windows Installer 4,0 oder Windows Installer 4,5 unter Windows Server 2008 oder Windows Vista. Windows Installer unter Windows Server 2003 oder Windows XP. Informationen zu den minimalen Windows-Service Pack, die für eine Windows Installer Version erforderlich sind, finden Sie in den [Windows Installer Run-Time Anforderungen](windows-installer-portal.md) .<br/> |
