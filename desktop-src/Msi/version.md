# <a name="version-windows-installer"></a>Version (Windows Installer)

Der Datentyp Version ist eine Text Zeichenfolge, die eine gültige Versions Zeichenfolge enthält. Eine Versions Zeichenfolge hat das Format.

``` syntax
xxxxx.xxxxx.xxxxx.xxxxx
```

wobei *x* eine Ziffer ist.

Die maximal zulässige Versions Zeichenfolge ist 65535.65535.65535.65535.

Im folgenden finden Sie Beispiele für gültige Versions Zeichenfolgen:

-   1
-   1.0
-   1.00
-   10,00
-   1.00.1
-   1.0.1
-   1.00.10
-   1.00.100
-   1.0.1000.0
