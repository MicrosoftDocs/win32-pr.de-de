# <a name="doubleinteger"></a>Doubleiteger

Der doubleinteger-Datentyp ist ein 4-Byte-ganzzahliger Wert. Sofern nicht anders eingeschränkt, liegt der Bereich der zulässigen Werte zwischen – 2.147.483.647 und + 2.147.483.647.
