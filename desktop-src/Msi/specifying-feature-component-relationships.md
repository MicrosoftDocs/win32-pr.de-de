# <a name="specifying-feature-component-relationships"></a>Angeben Feature-Component Beziehungen

Jedes [Windows Installer-Feature](windows-installer-features.md) verwendet eine oder mehrere [Windows Installer-Komponenten,](windows-installer-components.md)und Features können Komponenten gemeinsam nutzen. [Die Tabelle FeatureComponents](featurecomponents-table.md) definiert die Beziehung zwischen Features und Komponenten. Weitere Informationen finden Sie unter Core Tables Group and Components and Features in the Windows Installer overview (Haupttabellengruppe und [Komponenten und Features](components-and-features.md) in der Übersicht über Windows Installer). [](core-tables-group.md) In diesem Abschnitt fügen Sie informationen zur Tabelle FeatureComponents des Editor Beispiels hinzu.

Verwenden Sie den Datenbank-Editor, um MNP2000.msi zu öffnen und die folgenden Daten in die leere Tabelle FeatureComponents einzugeben.

[Tabelle "FeatureComponents"](featurecomponents-table.md)



| Komponente\_ | Komponente\_ |
|-----------|-------------|
| Baseball  | Baseball    |
| Konzert   | Konzert     |
| Tanz     | Tanz       |
| Fußball  | Fußball    |
| Hilfe      | Hilfe        |
| January   | January     |
| NewYears  | NewYears    |
| Editor   | Editor     |
| Infodatei    | Editor     |
