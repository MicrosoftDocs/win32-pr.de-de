# <a name="indirectpropertyname-control-attribute"></a>IndirectPropertyName-Steuerelementattribut

Dieses Attribut ist der Name einer indirekten Eigenschaft, die dem Steuerelement zugeordnet ist. Wenn das [indirekte](indirect-control-attribute.md) Bit festgelegt ist, zeigt das Steuerelement den Wert der Eigenschaft mit diesem Namen an oder ändert diesen, und dieser Name ist der Wert der Eigenschaft, die in der Property -Spalte der [Control-Tabelle aufgeführt ist.](control-table.md)

Bei inert-Steuerelementen gibt dieses Attribut NULL zurück.

## <a name="valid-controls"></a>Gültige Steuerelemente

Alle aktiven Steuerelemente.

## <a name="associated-bit-flags"></a>Zugeordnete Bitflags

Dieses Attribut verwendet keine Bitflags.

## <a name="remarks"></a>Hinweise

Weitere [Informationen finden Sie unter Steuerelementattribute](control-attributes.md) und das Steuerelement, das Sie unter Steuerelemente erstellen [müssen.](controls.md)
