# <a name="indirectpropertyname-control-attribute"></a>Deretpropertyname-Steuerelement Attribut

Dieses Attribut ist der Name einer indirekten Eigenschaft, die dem Steuerelement zugeordnet ist. Wenn das [indirekte](indirect-control-attribute.md) Bit festgelegt ist, zeigt das Steuerelement den Wert der Eigenschaft mit diesem Namen an oder ändert ihn. dieser Name ist der Wert der Eigenschaft, die in der Eigenschafts Spalte der [Steuerelement Tabelle](control-table.md)aufgeführt ist.

Für inertsteuerelemente gibt dieses Attribut NULL zurück.

## <a name="valid-controls"></a>Gültige Steuerelemente

Alle aktiven Steuerelemente.

## <a name="associated-bit-flags"></a>Zugehörige Bitflags

Dieses Attribut verwendet keine Bitflags.

## <a name="remarks"></a>Bemerkungen

Siehe [Steuerelement Attribute](control-attributes.md) und das Steuerelement, das Sie unter Steuer [Elementen](controls.md)erstellen müssen.
