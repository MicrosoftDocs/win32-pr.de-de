# <a name="condition"></a>Bedingung

Der Condition-Datentyp ist eine Text Zeichenfolge, die eine gültige bedingte Anweisung enthält, die als true oder false ausgewertet werden kann. Weitere Informationen zur Syntax von Bedingungs Anweisungen finden Sie unter [Syntax für bedingte](conditional-statement-syntax.md)Anweisungen.
