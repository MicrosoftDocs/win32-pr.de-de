# <a name="condition"></a>Bedingung

Der Condition-Datentyp ist eine Textzeichenfolge, die eine gültige bedingte Anweisung enthält, die als TRUE oder FALSE ausgewertet werden kann. Informationen zur Syntax von bedingten Anweisungen finden Sie unter [Conditional Statement Syntax](conditional-statement-syntax.md).
