<span id="setup.delta_patch_gly"></span><span id="SETUP.DELTA_PATCH_GLY"></span>**Delta Patch**
</dt> <dd>

Ein Delta Patch ist ein Delta komprimierter Windows Installer Patch, der mit einem Tool erstellt wird, z. b. Patchwiz.dll, das Delta Komprimierung unterstützt. Patches, die Delta Komprimierung verwenden, können die Größe eines Updates verringern, indem Sie nur die Unterschiede (Delta) zwischen vorhandenen Dateien auf einem Bereitstellungs Zielcomputer und den gewünschten neuen Dateien bereitstellen. Die gewünschten neuen Dateien werden dann aus den vorhandenen Dateien und den heruntergeladenen Delta Dateien synthetisiert. Weitere Informationen zur Delta Komprimierungs Technologie finden Sie in der [Anwendungsprogrammierschnittstelle für die Delta Komprimierung](https://msdn.microsoft.com/library/ms811406.aspx).

</dd> </dl>
