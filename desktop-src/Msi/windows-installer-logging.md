# <a name="windows-installer-logging"></a>Windows Installer Protokollierung

Windows Installer zeichnet Fehler und Ereignisse in einem eigenen Fehlerprotokoll und im Ereignisprotokoll auf. Die Diagnoseinformationen, die das Installationsprogramm in diese Protokolle schreibt, können Benutzern und Administratoren helfen, die Ursache einer fehlgeschlagenen Installation zu verstehen.

-   [Normale Protokollierung](normal-logging.md)
-   [Ereignisprotokollierung](event-logging.md)
-   [Verhindern, dass vertrauliche Informationen in die Protokolldatei geschrieben werden](preventing-confidential-information-from-being-written-into-the-log-file.md)
-   [Wilogutl.exe](wilogutl-exe.md) unterstützt die Analyse von Protokolldateien von einer Windows Installer-Installation und zeigt vorgeschlagene Lösungen für Fehler an, die in einer Protokolldatei gefunden werden.

Weitere Informationen zur Interpretation Windows Installer Protokolldateien finden Sie im Whitepaper auf der TechNet-Website: [Windows Installer: Vorteile und Implementierung für System Administratoren](https://www.microsoft.com/technet/prodtechnol/windows2000serv/maintain/featusability/winmsi.mspx).
