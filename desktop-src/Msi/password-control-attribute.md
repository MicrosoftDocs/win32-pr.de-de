# <a name="password-control-attribute"></a>Password-Steuerelement Attribut

Mit diesem Attribut wird ein Bearbeitungs Steuerelement zum Eingeben von Kenn Wörtern erstellt. Das-Steuerelement zeigt jedes Zeichen als Sternchen () an, \* Wenn Sie in das-Steuerelement eingegeben werden.

Durch Festlegen des Attributs "Password Control" wird verhindert, dass das Installationsprogramm die dem Bearbeitungs Steuerelement zugeordnete Eigenschaft in die Protokolldatei schreibt. Weitere Informationen finden Sie unter [verhindern, dass vertrauliche Informationen in die Protokolldatei geschrieben werden](preventing-confidential-information-from-being-written-into-the-log-file.md).

## <a name="valid-controls"></a>Gültige Steuerelemente

[Bearbeitungssteuerelement](edit-control.md)

## <a name="associated-bit-flags"></a>Zugehörige Bitflags

Dieses Attribut verwendet keine Bitflags.
