# <a name="password-control-attribute"></a>Kennwortsteuerungsattribut

Dieses Attribut erstellt ein Bearbeitungssteuerelement für die Eingabe von Kennwörtern. Das -Steuerelement zeigt jedes Zeichen als Sternchen \* () an, wenn sie in das Steuerelement eingegeben werden.

Das Festlegen des Kennwortsteuerungsattributs verhindert, dass das Installationsprogramm die dem Steuerelement Bearbeiten zugeordnete Eigenschaft in die Protokolldatei schreibt. Weitere Informationen finden Sie unter [Verhindern, dass vertrauliche Informationen in die Protokolldatei geschrieben werden.](preventing-confidential-information-from-being-written-into-the-log-file.md)

## <a name="valid-controls"></a>Gültige Steuerelemente

[Bearbeitungssteuerelement](edit-control.md)

## <a name="associated-bit-flags"></a>Zugeordnete Bitflags

Dieses Attribut verwendet keine Bitflags.
