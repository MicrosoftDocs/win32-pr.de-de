# <a name="sign-setupexe-and-mysetupmsi"></a>Signieren von Setup.exe und MySetup.msi

Bevor Sie Setup.exe und MySetup.msi auf dem Webserver platzieren, sollten Sie die Dateien mit Ihrem digitalen Zertifikat und dem privaten Schlüssel Mycert.cer und Mycert.pvk signieren, indem Sie das [SignTool-Hilfsprogramm](../seccrypto/signtool.md) verwenden. Weitere Informationen zur Verwendung des Hilfsprogramms SignTool finden Sie unter [CryptoAPI Tools Reference](../seccrypto/cryptoapi-tools-reference.md) im Microsoft Windows Software Development Kit (SDK).
