# <a name="sign-setupexe-and-mysetupmsi"></a>Signieren Setup.exe und MySetup.msi

Bevor Sie Setup.exe und MySetup.msi auf dem Webserver platzieren, sollten Sie die Dateien mit Ihrem digitalen Zertifikat und dem privaten Schlüssel, mycert. CER und mycert. pvk, mithilfe des [SignTool](../seccrypto/signtool.md) -Hilfsprogramms signieren. Weitere Informationen zur Verwendung des SignTool-Hilfsprogramms finden Sie unter [CryptoAPI Tools Reference](../seccrypto/cryptoapi-tools-reference.md) im Microsoft Windows Software Development Kit (SDK).
