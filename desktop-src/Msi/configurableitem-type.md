# <a name="configurableitemtype-property"></a>ConfigurableItem.Type-Eigenschaft

Die **Type-Eigenschaft** gibt den Wert aus der Type-Spalte der ModuleConfiguration-Tabelle zurück.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = ConfigurableItem.Type
```



## <a name="property-value"></a>Eigenschaftswert

## <a name="c"></a>C++
