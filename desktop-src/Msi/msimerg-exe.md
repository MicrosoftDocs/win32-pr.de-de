# <a name="msimergexe"></a>Msimerg.exe

Msimerg.exe ist ein Befehlszeilen-Hilfsprogramm, das [**msidatabasemerge**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabasemergea) verwendet, um eine Verweis Datenbank in einer Basisdatenbank zusammen [zuführen](merges-and-transforms.md) .

Dieses Tool ist nur in den [Windows SDK Komponenten für Windows Installer Entwickler](platform-sdk-components-for-windows-installer-developers.md)verfügbar.

## <a name="syntax"></a>Syntax

**Msimerg** *{Base Database} {Verweis Datenbank}*

Wenn ein Mergekonflikt zwischen den beiden Datenbanken auftritt, werden die Konflikt Informationen in der \_ Tabelle mergeerrors abgelegt.

## <a name="command-line-options"></a>Befehlszeilenoptionen

Keine.
