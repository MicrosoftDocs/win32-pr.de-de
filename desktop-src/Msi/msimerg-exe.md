# <a name="msimergexe"></a>Msimerg.exe

Msimerg.exe ist ein Befehlszeilenprogramm, das [**MsiDatabaseMerge**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabasemergea) verwendet, um eine Verweisdatenbank in einer Basisdatenbank [zusammenzuführen.](merges-and-transforms.md)

Dieses Tool ist nur in den [Windows SDK-Komponenten für Windows Installer-Entwickler](platform-sdk-components-for-windows-installer-developers.md)verfügbar.

## <a name="syntax"></a>Syntax

**Msimerg** *{Basisdatenbank}{Referenzdatenbank}*

Wenn ein Mergekonflikt zwischen den beiden Datenbanken vorliegt, werden die Konfliktinformationen in die \_ MergeErrors-Tabelle aufgenommen.

## <a name="command-line-options"></a>Befehlszeilenoptionen

Keine.
