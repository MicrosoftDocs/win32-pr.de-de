# <a name="installer-units"></a>Installer-Einheiten

Eine Windows Installer Benutzeroberflächen Einheit ist ungefähr gleich 1-zwölf (1/12) Höhe der 10-Punkt-MS Sans Serif-Schriftgröße. Abhängig von der Schriftart Größe, die in der Benutzeroberfläche verwendet wird, kann die relative Größe von Dialogfeldern, Steuerelementen und Schriftarten geändert werden. Um die richtige Anzeige der Benutzeroberfläche zu gewährleisten, sollten Setup Entwickler ihre Anwendung immer mit den tatsächlichen Schriftgrößen testen.
