<span data-ttu-id="659bc-110">In verschiedenen Versionen desselben Produkts müssen Sie den [**UpgradeCode**](upgradecode.md)möglicherweise nie ändern.</span><span class="sxs-lookup"><span data-stu-id="659bc-110">Across different versions of the same product, you may never need to change the [**UpgradeCode**](upgradecode.md).</span></span> <span data-ttu-id="659bc-111">Jedes eigenständige Produkt sollte über einen eigenen **UpgradeCode** verfügen.</span><span class="sxs-lookup"><span data-stu-id="659bc-111">Each stand-alone product should have its own **UpgradeCode**.</span></span> <span data-ttu-id="659bc-112">Eine Produktsuite sollte auch über einen eigenen **UpgradeCode** verfügen.</span><span class="sxs-lookup"><span data-stu-id="659bc-112">A product suite should also have its own **UpgradeCode**.</span></span> <span data-ttu-id="659bc-113">Dies ermöglicht es der Suite, frühere Versionen der Suite oder eigenständigen Produkte zu aktualisieren, indem mehrere Zeilen in der [upgradetabelle](upgrade-table.md)verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="659bc-113">Doing so will allow the suite to upgrade previous versions of the suite or stand-alone products by using multiple rows in the [Upgrade table](upgrade-table.md).</span></span>

<span data-ttu-id="659bc-114">Die folgenden beiden Szenarien veranschaulichen die Verwendung von [**UpgradeCode**](upgradecode.md).</span><span class="sxs-lookup"><span data-stu-id="659bc-114">The following two scenarios illustrate the use of the [**UpgradeCode**](upgradecode.md).</span></span>

-   <span data-ttu-id="659bc-115">Produkt a und Produkt B waren mit den gleichen [**productlanguage**](productlanguage.md), [**ProductVersion**](productversion.md)und [**UpgradeCode**](upgradecode.md)ausgeliefert.</span><span class="sxs-lookup"><span data-stu-id="659bc-115">Product A and Product B were shipped with the same [**ProductLanguage**](productlanguage.md), [**ProductVersion**](productversion.md), and [**UpgradeCode**](upgradecode.md).</span></span> <span data-ttu-id="659bc-116">Produkt a und Produkt B verfügen über unterschiedliche [**productcodes**](productcode.md).</span><span class="sxs-lookup"><span data-stu-id="659bc-116">Product A and Product B have different [**ProductCodes**](productcode.md).</span></span> <span data-ttu-id="659bc-117">Da den Produkten der gleiche **UpgradeCode** zugewiesen wurde, kann die [upgradetabelle](upgrade-table.md) nicht erstellt werden, um die ältere Version von Produkt a von der älteren Version von Produkt B zu unterscheiden. Because these were different products, they should have each been assigned a different **UpgradeCode**.</span></span>
