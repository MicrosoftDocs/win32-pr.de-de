# <a name="logaction-property"></a>Logaction (Eigenschaft)

Die **logaction** -Eigenschaft ist eine Liste von Aktions Namen, die nur durch Semikolons und ohne Leerzeichen voneinander getrennt sind. Bei den Aktions Namen in dieser Eigenschaft wird die Groß-/Kleinschreibung nicht beachtet, und die Liste kann mit einem Semikolon enden.

## <a name="remarks"></a>Bemerkungen

Der Installer protokolliert Aktions Daten Meldungen für die Aktionen in der Liste. Die-Eigenschaft hat keine Auswirkung auf andere Protokollierungs Typen.
