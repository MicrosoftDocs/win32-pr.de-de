# <a name="logaction-property"></a>LOGACTION-Eigenschaft

Die **LOGACTION-Eigenschaft** ist eine Liste von Aktionsnamen, die nur durch Semikolons und ohne Leerzeichen getrennt sind. Bei den Aktionsnamen in dieser Eigenschaft wird die Groß-/Kleinschreibung nicht beachtet, und die Liste kann mit einem Semikolon enden.

## <a name="remarks"></a>Hinweise

Das Installationsprogramm protokolliert Aktionsdatenmeldungen für die Aktionen, die in der Liste enthalten sind. Die -Eigenschaft hat keine Auswirkungen auf andere Protokollierungstypen.
