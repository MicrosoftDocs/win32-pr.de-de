# <a name="arpsystemcomponent-property"></a>ARPSYSTEMCOMPONENT (Eigenschaft)

Wenn die Eigenschaft **ARPSYSTEMCOMPONENT** mithilfe der Befehlszeile oder einer Transformation auf 1 festgelegt wird, wird verhindert, dass **die Anwendung** in der **Systemsteuerung** in der Liste Software angezeigt wird.
