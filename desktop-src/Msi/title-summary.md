# <a name="title-summary-property"></a>Titel Zusammenfassungs Eigenschaft

Mit der Eigenschaft **Titel Zusammenfassung** wird der Typ des Installationspakets kurz beschrieben. Ausdrücke wie "Installation Database" oder "Transform" oder "Patch" können für diese Eigenschaft verwendet werden.
