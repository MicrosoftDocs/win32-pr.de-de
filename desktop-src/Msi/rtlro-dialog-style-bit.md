# <a name="rtlro-dialog-style-bit"></a>RTLRO Dialog Style Bit

Wenn dieses Formatbit festgelegt ist, wird der Text im Dialogfeld in der Leserichtung von rechts nach links angezeigt.

## <a name="value"></a>Wert



| Decimal | Hexadezimal | Konstante                       |
|---------|-------------|--------------------------------|
| 128     | 0x00000080  | **msidbDialogAttributesRTLRO** |
