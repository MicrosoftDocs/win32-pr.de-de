# <a name="rtlro-dialog-style-bit"></a>Rtlro-Dialog Feld Stil

Wenn dieses Stilbit festgelegt ist, wird der Text im Dialogfeld in der Reihenfolge von rechts nach links angezeigt.

## <a name="value"></a>Wert



| Decimal | Hexadezimal | Konstante                       |
|---------|-------------|--------------------------------|
| 128     | 0x00000080  | **msidbdialogattributesrtlro** |
