# <a name="properties-windows-installer"></a>Eigenschaften (Windows Installer)

Eigenschaften sind globale Variablen, die Windows Installer während einer Installation verwendet. In den folgenden Abschnitten werden die eigenschaften beschrieben, die vom Installationsprogramm verwendet werden:

-   [Informationen zu Eigenschaften](about-properties.md)
-   [Verwenden von Eigenschaften](using-properties.md)
-   [Eigenschaftenverweis](property-reference.md)

Der Begriff "Eigenschaft" bezieht sich auch auf ein Attribut eines Automatisierungsobjekts. Weitere Informationen [finden Sie unter Automation Interface](automation-interface.md).
