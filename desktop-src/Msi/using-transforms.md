# <a name="using-transforms"></a>Verwenden von Transformationen

Durch Transformationen können Sie die Flexibilität der Anwendungs Installation erhöhen, indem Sie eine Möglichkeit zum Anwenden von Änderungen auf eine Datenbank bereitstellen, ohne die ursprüngliche Datenbank zu ändern. Weitere Informationen finden Sie unter [Daten Bank Transformationen](database-transforms.md).

In den folgenden Abschnitten wird die Verwendung von Transformationen erläutert:

[Anwenden von Transformationen](applying-transforms.md)

 

[Erstellen einer Transformation](generating-a-transform.md)

 

[Verwenden von Transformationen zum Hinzufügen von Ressourcen](using-transforms-to-add-resources.md)
