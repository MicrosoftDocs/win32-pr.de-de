# <a name="errortype-property"></a>Error.Type-Eigenschaft

Die schreibgeschützte **Type-Eigenschaft** gibt einen der Werte von msmErrorType zurück, der den Typ des Fehlers angibt, der durch dieses Objekt dargestellt wird.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Error.Type
```
