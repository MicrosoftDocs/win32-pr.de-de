# <a name="errortype-property"></a>Error. Type-Eigenschaft

Die schreibgeschützte **Type** -Eigenschaft gibt einen der Werte von msmerrortype zurück, der den Fehlertyp angibt, der durch dieses-Objekt dargestellt wird.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Error.Type
```



## <a name="property-value"></a>Eigenschaftswert
