# <a name="interaction-context-structures"></a>Interaktionskontextstrukturen

Die Themen in diesem Abschnitt enthalten die Referenzspezifikationen für [Interaktionskontextstrukturen.](interaction-context-portal.md)

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|---|---|
| [**CROSS \_ SLIDE \_ PARAMETER**](/windows/win32/api/interactioncontext/ns-interactioncontext-cross_slide_parameter)<br/>                           | Definiert den Schwellenwert für schiebeübergreifende Schieberegler und dessen Entfernungsschwellenwert.<br/>                                                                                                    |
| [**\_INTERAKTIONSARGUMENTE \_ \_ SCHIEBEREGLER ÜBERGREIFEND**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_cross_slide)<br/>  | Definiert den Zustand der schiebeübergreifenden Interaktion.<br/>                                                                                                                |
| [**\_BEARBEITUNG VON INTERAKTIONSARGUMENTEN \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_manipulation)<br/> | Definiert den Zustand einer Bearbeitung.<br/>                                                                                                                             |
| [**\_ \_ INTERAKTIONSARGUMENTE TIPPEN**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_tap)<br/>                   | Definiert den Zustand der Tippinteraktion.<br/>                                                                                                                        |
| [**\_KONFIGURATION DES INTERAKTIONSKONTEXTS \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_context_configuration)<br/>   | Definiert die Konfiguration eines [Interaktionskontextobjekts,](interaction-context-portal.md) das das Verhalten einer Interaktion aktiviert, deaktiviert oder ändert.<br/> |
| [**\_ \_ INTERAKTIONSKONTEXTAUSGABE**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_context_output)<br/>                 | Definiert die Ausgabe des [Interaktionskontextobjekts.](interaction-context-portal.md)<br/>                                                                          |
| [**MANIPULATION \_ TRANSFORM**](/windows/win32/api/interactioncontext/ns-interactioncontext-manipulation_transform)<br/>                          | Definiert die Transformationsdaten für eine Bearbeitung.<br/>                                                                                                              |
| [**\_MANIPULATIONSGESCHWINDIGKEIT**](/windows/win32/api/interactioncontext/ns-interactioncontext-manipulation_velocity)<br/>                            | Definiert die Geschwindigkeitsdaten einer Manipulation.<br/>                                                                                                                     |
