# <a name="interaction-context"></a>Interaktions Kontext

## <a name="purpose"></a>Zweck

Die Themen in diesem Abschnitt bieten eine Übersicht über die Unterstützung für Interaktions Kontext in Windows 8. Mit dem Interaktions Kontext können Windows 8-Anwendungen und Benutzeroberflächen-Frameworks mehrere gleichzeitige Interaktionen unterstützen, indem Gestenerkennung und Manipulations Verarbeitung bereitgestellt werden.

In der hier bereitgestellten Themensammlung wird beschrieben, wie Ihre APP Interaktions Kontext verwenden kann, um die Benutzerfreundlichkeit für Ihren Kunden zu optimieren.

## <a name="developer-audience"></a>Entwicklergruppe

Die Interaktions Kontext-APIs sind für Entwickler konzipiert, die Benutzeroberflächen-Frameworks erstellen, die eine konsistente, Berührungs optimierte Benutzeroberfläche über Desktop Anwendungen hinweg bereitstellen.
