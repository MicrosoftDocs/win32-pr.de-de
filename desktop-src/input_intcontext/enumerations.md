# <a name="interaction-context-enumerations"></a>Interaktionskontextenumeration

Die Themen in diesem Abschnitt enthalten die Referenzspezifikationen für [Enumerationen des](interaction-context-portal.md) Interaktionskontexts.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | Beschreibung |
|---|---|
| [**ÜBERGREIFENDE \_ \_ GLEITFLAGS**](/windows/win32/api/interactioncontext/ne-interactioncontext-cross_slide_flags)<br/> | Gibt den Zustand der schiebeübergreifenden Interaktion an.<br/> |
| [**SCHWELLENWERT \_ FÜR \_ ÜBERGREIFENDE SCHIEBEREGLER**](/windows/win32/api/interactioncontext/ne-interactioncontext-cross_slide_threshold)<br/> | Gibt die Schwellenwerte für das schiebeübergreifende Verhalten an.<br/> |
| [**TRÄGHEITSPARAMETER \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-inertia_parameter)<br/> | Gibt die Trägheitswerte für eine Manipulation an (Übersetzung, Drehung, Skalierung).<br/> |
| [**\_ \_ KONFIGURATIONSFLAGS FÜR INTERAKTIONEN**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_configuration_flags)<br/> | Gibt die Interaktionen an, die beim Konfigurieren eines [Interaktionskontextobjekts aktiviert werden](interaction-context-portal.md) müssen.<br/> |
| [**\_ \_ INTERAKTIONSKONTEXTEIGENSCHAFT**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_context_property)<br/> | Gibt Eigenschaften des [Interaktionskontextobjekts](interaction-context-portal.md) an. <br/> |
| [**\_INTERAKTIONSFLAGS**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_flags)<br/> | Gibt den Zustand einer Interaktion an.<br/> |
| [**\_INTERAKTIONS-ID**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_id)<br/> | Gibt die Interaktionszustände an, die zum Konfigurieren eines [Interaktionskontextobjekts verwendet](interaction-context-portal.md) werden. Interaktionen können statisch (einzelner Kontakt ohne Bearbeitung, z. B. Tippen, Doppeltippen, Tippen mit der rechten Taste, Halten und Halten) oder dynamisch (ein oder mehrere Kontakte mit Manipulation, z. B. Übersetzung, Drehung oder Skalierung) sein.<br/> |
| [**\_INTERAKTIONSZUSTAND**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_state)<br/> | Gibt den Zustand des [Interaktionskontextobjekts](interaction-context-portal.md) an.<br/> |
| [**\_MANIPULATIONSSCHIENENZUSTAND \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-manipulation_rails_state)<br/> | Gibt die Schienenzustände für eine Interaktion an.<br/> |
| [**\_ \_ MAUSRADPARAMETER**](/windows/win32/api/interactioncontext/ne-interactioncontext-mouse_wheel_parameter)<br/> | Gibt die Bearbeitungen an, die der Mausraddrehung zugeordnet werden können.<br/> |
