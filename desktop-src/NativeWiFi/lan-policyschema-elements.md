# <a name="lan_policy-schema-elements"></a>LAN- \_ Richtlinien Schema Elemente

Ein kabelgebundenes (LAN-) Richtlinien Profil enthält die folgenden Schema Elemente. Alle benannten Elemente befinden sich im-Namespace `https://www.microsoft.com/networking/LAN/policy/v1` .

In der folgenden Liste werden die definierten Elemente in der Reihenfolge angezeigt, in der die Elemente in einem Profil angezeigt werden. Die Reihenfolge der Elemente wird erzwungen. Nicht alle Elemente befinden sich in jedem Profil, da einige Elemente optional sind.

In dieser Liste werden nicht alle möglichen Elemente angezeigt, die in einem Profil angezeigt werden können, da Elemente in **xs: beliebige** Einfügepunkte hinzugefügt werden können.

-   [**Lanpolicy**](lan-policyschema-lanpolicy-element.md)
    -   [**Name (lanpolicy)**](lan-policyschema-name-lanpolicy-element.md)
    -   [**Beschreibung (lanpolicy)**](lan-policyschema-description-lanpolicy-element.md)
    -   [**globalflags (lanpolicy)**](lan-policyschema-globalflags-lanpolicy-element.md)
        -   [**enableautoconfig (globalflags)**](lan-policyschema-enableautoconfig-globalflags-element.md)
    -   [**ProfileList (lanpolicy)**](lan-policyschema-profilelist-lanpolicy-element.md)
