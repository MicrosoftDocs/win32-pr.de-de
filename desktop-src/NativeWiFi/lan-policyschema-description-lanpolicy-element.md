# <a name="description-lanpolicy-element"></a>description (LANPolicy)-Element

Das DESCRIPTION-Element (LANPolicy) enthält die Beschreibung einer LAN-Richtlinie für kabelgebundene Verbindungen.

``` syntax
<xs:element name="description"
    type="nameType"
 />
```

Das **description-Element** wird durch das [**LANPolicy-Element**](lan-policyschema-lanpolicy-element.md) definiert.
