# <a name="description-lanpolicy-element"></a>Description-Element (lanpolicy)

Das Element Description (lanpolicy) enthält die Beschreibung einer Richtlinie für ein kabelgebundenes LAN.

``` syntax
<xs:element name="description"
    type="nameType"
 />
```

Das **Description** -Element wird durch das [**lanpolicy**](lan-policyschema-lanpolicy-element.md) -Element definiert.
