# <a name="lan_profile-schema-elements"></a>\_LAN-Profilschemaelemente

Ein Kabelnetzwerkprofil enthält die folgenden Schemaelemente. Alle benannten Elemente befinden sich im Namespace `https://www.microsoft.com/networking/LAN/profile/v1` .

In der folgenden Liste werden die definierten Elemente in der Reihenfolge angezeigt, in der die Elemente in einem Profil angezeigt werden. Die Reihenfolge der Elemente wird erzwungen. In dieser Liste werden nicht alle möglichen Elemente angezeigt, die in einem Profil angezeigt werden können, da Elemente in **xs:any-Einfügepunkten hinzugefügt werden** können.

-   [**LANProfile**](lan-profileschema-lanprofile-element.md)
    -   [**MSM (LANProfile)**](lan-profileschema-msm-lanprofile-element.md)
        -   [**Sicherheit (MSM)**](lan-profileschema-security-msm-element.md)
            -   [**OneXEnforced (Sicherheit)**](lan-profileschema-onexenforced-security-element.md)
            -   [**OneXEnabled (Sicherheit)**](lan-profileschema-onexenabled-security-element.md)
