# <a name="lan_profile-schema-elements"></a>LAN- \_ Profil Schema Elemente

Ein Kabelnetzwerk Profil enthält die folgenden Schema Elemente. Alle benannten Elemente befinden sich im-Namespace `https://www.microsoft.com/networking/LAN/profile/v1` .

In der folgenden Liste werden die definierten Elemente in der Reihenfolge angezeigt, in der die Elemente in einem Profil angezeigt werden. Die Reihenfolge der Elemente wird erzwungen. In dieser Liste werden nicht alle möglichen Elemente angezeigt, die in einem Profil angezeigt werden können, da Elemente in **xs: beliebige** Einfügepunkte hinzugefügt werden können.

-   [**Lanprofile**](lan-profileschema-lanprofile-element.md)
    -   [**MSM (lanprofile)**](lan-profileschema-msm-lanprofile-element.md)
        -   [**Sicherheit (MSM)**](lan-profileschema-security-msm-element.md)
            -   [**Onexerzwungene (Sicherheit)**](lan-profileschema-onexenforced-security-element.md)
            -   [**Onexaktiviert (Sicherheit)**](lan-profileschema-onexenabled-security-element.md)
