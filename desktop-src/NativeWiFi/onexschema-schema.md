# <a name="onex-schema"></a>Onex-Schema

Das Onex-Schema definiert 802.1 x-Konfigurationselemente. Alle Onex-Schema Elemente gelten für Kabel-und drahtlos Profile. Eine Liste der definierten Elemente finden Sie unter [Onex-Schema Elemente](onexschema-elements.md).

Das Stamm Element eines 802.1 x-Profils ist das [**Onex**](onexschema-onex-element.md) -Element. Jedes Profil weist genau ein Stamm Element auf. Das **Onex** -Element befindet sich im- `https://www.microsoft.com/networking/OneX/v1` Namespace.

Beispiel Profile für Drahtlos Netzwerke, die 802.1 x-Konfigurationselemente enthalten, finden Sie in den folgenden Profil Beispielen:

-   [Bootstrap-Profil Beispiel](bootstrap-profile-sample.md)
-   [Beispiel für ein PPS-Profil](fips-profile-sample.md)
-   [Beispiel für Einzel Sign-On Profil](single-sign-on-profile-sample.md)
-   [Beispiel für WPA-Enterprise mit PEAP-MSCHAPv2 profile](wpa-enterprise-with-peap-mschapv2-profile-sample.md)
-   [Beispiel für WPA-Enterprise mit TLS-Profil](wpa-enterprise-with-tls-profile-sample.md)
-   [Beispiel für WPA2-Enterprise mit PEAP-MSCHAPv2 profile](wpa2-enterprise-with-peap-mschapv2-profile-sample.md)
-   [Beispiel für WPA2-Enterprise mit TLS-Profil](wpa2-enterprise-with-tls-profile-sample.md)

Beispiel Profile für Kabel Netzwerke, die 802.1 x-Konfigurationselemente enthalten, finden Sie in den folgenden Profil Beispielen:
