Sie können ein drahtlos Profil zum Profil Speicher Programm gesteuert hinzufügen, indem Sie [**WLanSetProfile**](/windows/desktop/api/wlanapi/nf-wlanapi-wlansetprofile)aufrufen. Sie können dem Profil Speicher auch mit dem Befehlszeilenprogramm **netsh** ein Profil hinzufügen. Weitere Informationen finden Sie unter [Netsh Commands for Wireless Local Area Network (WLAN)](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc755301(v=ws.10)).
