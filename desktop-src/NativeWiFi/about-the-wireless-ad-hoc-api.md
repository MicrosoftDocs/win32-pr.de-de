# <a name="about-the-wireless-ad-hoc-api"></a>Informationen zur Drahtlosen Ad-hoc-API

Die Drahtlose Ad-hoc-API macht Schnittstellen zum Aufzählen und Herstellen einer Verbindung mit Ad-hoc-Netzwerken in 802.11 verfügbar. Diese Schnittstellen bieten einen vereinfachten objektorientierten Ansatz für die Ad-hoc-Netzwerkverwaltung. Standardenumeratoren werden für Netzwerke und Schnittstellen bereitgestellt. Diese Enumeratoren können gefiltert werden, um nur Ad-hoc-Netzwerke aufzuzählen, die von Ihrer Anwendung erstellt wurden.

> [!Note]  
> Der Ad-hoc-Modus ist in zukünftigen Versionen von Windows möglicherweise nicht mehr verfügbar. Verwenden Sie ab Windows 8.1 und Windows Server 2012 R2 stattdessen [Wi-Fi Direct.](about-the-wi-fi-direct-api.md)

 

Entwickler, die die Ad-hoc-API verwenden, sollten beachten, dass Netzwerke und Profile, die mit der Ad-hoc-API erstellt wurden, mithilfe der Native Wifi-API bearbeitet werden können, ohne die entsprechenden Ad-hoc-API-Benachrichtigungslistener zu benachrichtigen.

Die Drahtlose Ad-hoc-API wird in der [Drahtlosen Ad-hoc-Referenz](wireless-ad-hoc-reference.md)ausführlich beschrieben.

> [!Note]  
> Die Drahtlose Ad-hoc-API wird nur für Windows Vista und höhere Versionen des Betriebssystems unterstützt.
