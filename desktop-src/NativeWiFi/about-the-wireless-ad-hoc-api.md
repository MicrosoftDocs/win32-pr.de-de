# <a name="about-the-wireless-ad-hoc-api"></a>Informationen zur drahtlosen Ad-hoc-API

Die drahtlose Ad-hoc-API macht Schnittstellen zum Auflisten und verbinden mit 802,11 Ad-hoc-Netzwerken verfügbar. Diese Schnittstellen bieten einen vereinfachten objektorientierten Ansatz für die Ad-hoc-Netzwerkverwaltung. Standard-Enumeratoren werden für Netzwerke und Schnittstellen bereitgestellt. Diese Enumeratoren können so gefiltert werden, dass nur die von Ihrer Anwendung erstellten Ad-hoc-Netzwerke aufgelistet werden.

> [!Note]  
> Der Ad-hoc-Modus ist möglicherweise in zukünftigen Versionen von Windows nicht verfügbar. Beginnen Sie mit Windows 8.1 und Windows Server 2012 R2, und verwenden Sie stattdessen [Wi-Fi Direct](about-the-wi-fi-direct-api.md) .

 

Entwickler, die die Ad-hoc-API verwenden, sollten sich bewusst sein, dass Netzwerke und Profile, die mithilfe der Ad-hoc-API erstellt wurden, mithilfe der nativen WiFi-API bearbeitet werden können, ohne die relevanten Benachrichtigungs Listener für

Die drahtlose Ad-hoc-API wird ausführlich in der [drahtlosen Ad-hoc-Referenz](wireless-ad-hoc-reference.md)beschrieben.

> [!Note]  
> Die drahtlose Ad-hoc-API wird nur unter Windows Vista und höheren Versionen des Betriebssystems unterstützt.
