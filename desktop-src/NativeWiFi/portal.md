# <a name="native-wifi"></a>Natives WiFi

## <a name="purpose"></a>Zweck

Die systemeigene drahtlose WLAN-Konfigurationskomponente konfiguriert, stellt eine Verbindung mit Drahtlos Netzwerken her und trennt diese. Native WiFi kann Profile in den Netzwerken speichern, mit denen es in Form von XML-Dokumenten interagiert.

## <a name="developer-audience"></a>Entwicklergruppe

Die native WiFi-API wurde für C/C++-Entwickler entwickelt. Programmierer sollten mit den Konzepten und der Terminologie von Drahtlos Netzwerken vertraut sein.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Die native WiFi-Komponente erfordert Clients, auf denen Windows Vista, Windows XP mit Service Pack 3 (SP3) oder Wireless LAN API für Windows XP mit Service Pack 2 (SP2) ausgeführt wird.
