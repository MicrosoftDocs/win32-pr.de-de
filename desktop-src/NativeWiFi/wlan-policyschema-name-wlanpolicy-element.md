# <a name="name-wlanpolicy-element"></a>Name (wlanpolicy)-Element

Das Elementname (wlanpolicy) enthält den Namen einer Drahtlos-LAN-Richtlinie.

``` syntax
<xs:element name="name"
    type="nameType"
 />
```

Das **Name** -Element wird durch das [**wlanpolicy**](wlan-policyschema-wlanpolicy-element.md) -Element definiert.

## <a name="remarks"></a>Bemerkungen

Bei Namen wird die Groß-/Kleinschreibung nicht beachtet.
