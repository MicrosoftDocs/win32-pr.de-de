# <a name="getting-started-with-directx-graphics"></a>Erste Schritte mit DirectX-Grafiken

Microsoft DirectX-Grafiken bieten eine Reihe von APIs, mit denen Sie Spiele und andere Hochleistungs-Multimediaanwendungen erstellen können. DirectX-Grafiken unterstützen leistungsstarke 2D- und 3D-Grafiken.

Verwenden Sie für 3D-Grafiken die Microsoft Direct3D 11-API. Auch wenn Sie über Microsoft Direct3D-Hardware auf 9- oder Microsoft Direct3D 10-Ebene verfügen, können Sie die Direct3D 11-API verwenden und ein [Gerät auf Featureebene](/windows/desktop/direct3d11/overviews-direct3d-11-devices-downlevel-intro) 9 \_ x oder Featureebene 10 x als Ziel \_ verwenden. Informationen zum Entwickeln von 3D-Grafiken mit DirectX finden Sie unter [Erstellen von 3D-Grafiken mit DirectX.](/previous-versions/windows/apps/hh465137(v=win.10)
)

Verwenden Sie für 2D-Grafiken und -Text Direct2D und [DirectWrite](./directwrite/direct-write-portal.md) anstelle von Windows Graphics Device Interface (GDI).

Verwenden Sie [DirectComposition,](./directcomp/directcomposition-portal.md)um Bitmaps zu erstellen, die direct3D 11 oder Direct2D aufgefüllt haben.

Informationen zum Erstellen einer Windows Store-App, die DirectX verwendet, finden Sie unter [Erstellen Ihrer ersten Windows Store-App mit DirectX.](/previous-versions/windows/apps/br229580(v=win.10)
) Sie können die [**Klasse Windows.UI::Xaml::Controls::SwapChainPanel**](/uwp/api/Windows.UI.Xaml.Controls.SwapChainPanel?view=winrt-19041) verwenden, um leistungsstarke DirectX-Apps mit einer XAML-UI-Überlagerung zu erstellen. Weitere Informationen zum Kombinieren von XAML und DirectX in einer Windows-App finden Sie unter [DirectX- und XAML-Interop.](/previous-versions/windows/apps/hh825871(v=win.10))

Informationen zum Erstellen eines Anzeigetreibers für Windows 8 finden Sie unter [Roadmap für das Windows Display Driver Model (WDDM).](/windows-hardware/drivers/display/roadmap-for-developing-drivers-for-the-windows-vista-display-driver-mo)

Wenn Sie die Dokumentation für frühere DirectX-Versionen benötigen, finden Sie weitere Informationen unter [Klassische DirectX-Grafiken.](/windows/desktop/classic-directx-graphics)
