# <a name="getting-started-with-directx-graphics"></a>Erste Schritte mit DirectX-Grafiken

Microsoft DirectX-Grafiken stellen eine Reihe von APIs zur Verfügung, mit denen Sie Spiele und andere hochleistungsbasierte Multimediaanwendungen erstellen können. DirectX-Grafiken unterstützen leistungsstarke 2D- und 3D-Grafiken.

Verwenden Sie für 3D-Grafiken die Microsoft Direct3D 11-API. Auch wenn Sie über Microsoft Direct3D 9-Level- oder Microsoft Direct3D 10-Level-Hardware verfügen, können Sie die Direct3D 11-API verwenden und ein Gerät der Featureebene [9](/windows/desktop/direct3d11/overviews-direct3d-11-devices-downlevel-intro) x oder Featureebene 10 x als Ziel \_ \_ verwenden. Informationen zum Entwickeln von 3D-Grafiken mit DirectX finden Sie unter [Erstellen von 3D-Grafiken mit DirectX.](/previous-versions/windows/apps/hh465137(v=win.10)
)

Verwenden Sie für 2D-Grafiken und -Text Direct2D [und DirectWrite](./directwrite/direct-write-portal.md) anstelle Windows Graphics Device Interface (GDI).

Verwenden Sie [DirectComposition,](./directcomp/directcomposition-portal.md)um Bitmaps zu erstellen, die mit Direct3D 11 oder Direct2D aufgefüllt wurden.

Informationen zum Erstellen einer Windows Store-App, die DirectX verwendet, finden Sie unter Erstellen Ihrer ersten [Windows Store-App mit DirectX.](/previous-versions/windows/apps/br229580(v=win.10)
) Sie können die [**-Windows. UI::Xaml::Controls::SwapChainPanel-Klasse**](/uwp/api/Windows.UI.Xaml.Controls.SwapChainPanel?view=winrt-19041) zum Erstellen leistungsstarker DirectX-Apps mit einer XAML-UI-Überlagerung. Weitere Informationen zum Kombinieren von XAML und DirectX in einer Windows-App finden Sie unter [DirectX und XAML-Interop](/previous-versions/windows/apps/hh825871(v=win.10)).

Informationen zum Erstellen eines Anzeigetreibers für Windows 8 finden Sie unter [Roadmap für das Windows Display Driver Model (WDDM).](/windows-hardware/drivers/display/roadmap-for-developing-drivers-for-the-windows-vista-display-driver-mo)

Wenn Sie die Dokumentation für frühere DirectX-Versionen benötigen, finden Sie weitere Informationen unter [Klassische DirectX-Grafiken.](/windows/desktop/classic-directx-graphics)
