# <a name="idxcoreadapterlistgetadaptercount-method"></a>Idxcoreadapterlist:: getadaptercount-Methode

Ruft die Anzahl von Adaptern in einem DXCore-Adapter Listen Objekt ab. Programmieranleitungen und Codebeispiele finden [Sie unter Verwenden von DXCore zum Aufzählen von Adaptern](../dxcore-enum-adapters.md).

## <a name="syntax"></a>Syntax

```cpp
virtual uint32_t STDMETHODCALLTYPE GetAdapterCount() = 0;
```

## <a name="returns"></a>Gibt zurück

Typ: **uint32_t**

Gibt die Anzahl der Adapter Elemente in der Liste zurück.
