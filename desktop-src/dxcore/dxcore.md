# <a name="dxcore"></a>DXCore

DXCore ist eine adapterenumerationsapi für Grafiken und Compute-Geräte, sodass sich einige ihrer Funktionen mit denen von [DXGI](../direct3ddxgi/dx-graphics-dxgi.md)überlappen. DXCore wird von Direct3D 12 verwendet.

Dieser Dokumentations Satz enthält Informationen über die Programmierung mit DXCore sowie die DXCore-Referenz.

## <a name="requirements"></a>Anforderungen

| &nbsp; | &nbsp; |
| ---- |:---- |
| **Unterstützte Laufzeitumgebungen** | Windows/C++ |
| **Empfohlene Programmiersprachen** | C++ |
| **Unterstützte Mindestversion (Client)** | Windows 10, Version 2004 (10,0; Build 19041) |
| **Header** | DXCore. h und dxcore_interface. h |
| **Bibliothek** | DXCore. lib |
| **DLL** | dxcore.dll |
