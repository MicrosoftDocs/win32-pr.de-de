# <a name="dxcore"></a>DXCore

DXCore ist eine Adapterenumerations-API für Grafik- und Computegeräte, sodass sich einige seiner Funktionen mit denen von [DXGI überschneiden.](../direct3ddxgi/dx-graphics-dxgi.md) DXCore wird von Direct3D 12 verwendet.

Dieser Dokumentationssatz enthält Informationen zur Programmierung mit DXCore sowie die DXCore-Referenz.

## <a name="requirements"></a>Anforderungen

| &nbsp; | &nbsp; |
| ---- |:---- |
| **Unterstützte Laufzeitumgebungen** | Windows/C++ |
| **Empfohlene Programmiersprachen** | C++ |
| **Unterstützte Mindestversion (Client)** | Windows 10, Version 2004 (10.0; Build 19041) |
| **Header** | dxcore.h und dxcore_interface.h |
| **Bibliothek** | dxcore.lib |
| **Dll** | dxcore.dll |
