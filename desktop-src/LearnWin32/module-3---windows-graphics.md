# <a name="module-3-windows-graphics"></a>Modul 3. Windows Graphics

[In Modul 1](your-first-windows-program.md) dieser Reihe wurde gezeigt, wie Sie ein leeres Fenster erstellen. [Modul 2](module-2--using-com-in-your-windows-program.md) führte einen kleinen Umweg durch das Component Object Model (COM), das die Grundlage für viele der modernen Windows-APIs ist. Nun ist es an der Zeit, dem leeren Fenster, das wir in Modul 1 erstellt haben, Grafiken hinzuzufügen.

Dieses Modul beginnt mit einer umfassenden Übersicht über die Windows Grafikarchitektur. Wir sehen uns dann Direct2D an, eine leistungsstarke Grafik-API, die in Version Windows 7 eingeführt wurde.
