# <a name="get-started-with-win32-and-c"></a>Erste Schritte mit Win32 und C++

Das Ziel dieser Reihe von ersten Schritten besteht darin, Ihnen zu zeigen, wie Sie ein Desktop Programm in C++ mithilfe von Win32-und com-APIs schreiben.

Im ersten Modul erfahren Sie Schritt für Schritt, wie Sie ein Fenster erstellen und anzeigen. Spätere Module stellen die Component Object Model (com), Grafiken und Text sowie Benutzereingaben vor.

Bei dieser Serie wird davon ausgegangen, dass Sie über ein gutes Wissen in der C++-Programmierung verfügen. Es wird nicht mit der Windows-Programmierung begonnen. Wenn Sie noch nicht mit C++ vertraut sind, finden Sie im [Visual C++ Developer Center](https://msdn.microsoft.com/vstudio//default.aspx)Lernmaterial. (Diese Ressource ist möglicherweise nicht in einigen Sprachen und Ländern verfügbar.)
