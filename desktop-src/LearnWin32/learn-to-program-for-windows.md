# <a name="get-started-with-win32-and-c"></a>Erste Schritte mit Win32 und C++

Das Ziel dieser Erste Schritte ist es, Ihnen beizubringen, wie Sie ein Desktopprogramm in C++ mit win32- und COM-APIs schreiben.

Im ersten Modul erfahren Sie Schritt für Schritt, wie Sie ein Fenster erstellen und anzeigen. In späteren Modulen werden die Component Object Model (COM), Grafiken und Text sowie Benutzereingaben einführen.

Für diese Reihe wird davon ausgegangen, dass Sie über gute Kenntnisse der C++-Programmierung verfügen. Es wird keine Erfahrung mit Windows Programmierung vorausgesetzt. Wenn Sie noch nicht mit C++ arbeiten, finden Sie Lernmaterial im [Visual C++ Developer Center.](https://msdn.microsoft.com/vstudio//default.aspx) (Diese Ressource ist in einigen Sprachen und Ländern möglicherweise nicht verfügbar.)
