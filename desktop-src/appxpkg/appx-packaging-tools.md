# <a name="packaging-and-deployment-tools-and-powershell-cmdlets"></a>Paket- und Bereitstellungstools und PowerShell-Cmdlets

Verwenden Sie diese Tools und Cmdlets zum Erstellen, Bereitstellen und Inventarieren von App-Paketen für Windows Apps.

## <a name="app-packaging-tools-and-guidance"></a>Tools und Anleitungen für die App-Paketierung

Erfahren Sie, wie Sie eine App packen, ein Testzertifikat erstellen und Ihre App über die Eingabeaufforderung und PowerShell signieren. Weitere Informationen:

-   [Erstellen eines App-Pakets mit dem Tool „MakeAppx.exe“](/windows/msix/package/create-app-package-with-makeappx-tool)
-   [Erstellen eines Paketsignaturzertifikats](/windows/msix/package/create-certificate-package-signing)
-   [Signieren eines App-Pakets mit SignTool](/windows/msix/package/sign-app-package-using-signtool)

Alternativ können Sie ein App Visual Studio paket erstellen und signieren. Weitere Informationen finden Sie unter [Packen einer UWP-App mit Visual Studio](/windows/msix/package/packaging-uwp-apps).

## <a name="powershell-cmdlets"></a>PowerShell-Cmdlets

Verwenden Windows PowerShell Zum Installieren, Inventar und Deinstallieren von branchenspezifischen Apps (LOB) Windows apps, die nicht Store signiert sind. Weitere Informationen finden Sie unter: [Appx-Modul-Cmdlets](/powershell/module/appx/index?view=win10-ps).
