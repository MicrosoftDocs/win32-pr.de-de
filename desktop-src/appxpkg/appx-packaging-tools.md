# <a name="packaging-and-deployment-tools-and-powershell-cmdlets"></a>Verpackungs-und Bereitstellungs Tools und PowerShell-Cmdlets

Verwenden Sie diese Tools und Cmdlets zum Erstellen, bereitstellen und Inventarisieren von App-Paketen für Windows-apps.

## <a name="app-packaging-tools-and-guidance"></a>Tools und Anleitungen zur APP-Paket Erstellung

Erfahren Sie, wie Sie eine APP packen, ein Test Zertifikat erstellen und Ihre APP mit der Eingabeaufforderung und PowerShell signieren. Weitere Informationen:

-   [Erstellen eines App-Pakets mit dem Tool „MakeAppx.exe“](/windows/msix/package/create-app-package-with-makeappx-tool)
-   [Erstellen eines Paketsignaturzertifikats](/windows/msix/package/create-certificate-package-signing)
-   [Signieren eines App-Pakets mit SignTool](/windows/msix/package/sign-app-package-using-signtool)

Alternativ dazu können Sie auch Visual Studio verwenden, um ein App-Paket zu erstellen und zu signieren. Weitere Informationen finden Sie unter [Packen einer UWP-App mit Visual Studio](/windows/msix/package/packaging-uwp-apps).

## <a name="powershell-cmdlets"></a>PowerShell-Cmdlets

Verwenden Sie Windows PowerShell, um branchenspezifische Windows-apps, die nicht im Speicher signiert sind, zu installieren, zu inventarisieren und zu deinstallieren. Weitere Informationen finden Sie unter: [AppX-Modul-Cmdlets](/powershell/module/appx/index?view=win10-ps).
