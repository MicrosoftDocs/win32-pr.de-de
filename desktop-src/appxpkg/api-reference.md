# <a name="packaging-deployment-and-query-apis"></a>Paket Erstellung, Bereitstellung und Abfrage von APIs

Verwenden Sie diese APIs in Ihren Desktop-Apps, um App-Pakete für Windows-apps zu erstellen, zu nutzen, bereitzustellen oder abzufragen:

-   [Verpacken von APIs](interfaces.md)
-   [Paketbereitstellungs-API](package-deployment-api.md)
-   [Paket Abfrage-API](functions.md)

Verwenden Sie diese APIs in Ihren Desktop-Apps, um App-Pakete bereitzustellen oder abzufragen:

-   [**Windows. applicationmodel. Package**](/uwp/api/Windows.ApplicationModel.Package)
-   [**Windows. applicationmodel. PackageID**](/uwp/api/Windows.ApplicationModel.PackageId)
-   [**Windows.Management.Deployment.PackageManager**](/uwp/api/Windows.Management.Deployment.PackageManager)
-   [**Windows.Management.Deployment.PackageUserInformation**](/uwp/api/Windows.Management.Deployment.PackageUserInformation)

Verwenden Sie diese APIs in ihren UWP-apps, um App-Pakete abzufragen:

-   [**Windows. applicationmodel. Package**](/uwp/api/Windows.ApplicationModel.Package)
-   [**Windows. applicationmodel. PackageID**](/uwp/api/Windows.ApplicationModel.PackageId)
