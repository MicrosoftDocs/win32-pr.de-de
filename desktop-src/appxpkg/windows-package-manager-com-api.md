# <a name="windows-package-manager-com-api"></a>Windows-Paket-Manager com-API

Gilt für Windows 10 und ist in der Familie mobiler Geräte implementiert. Wenn Sie weitere Informationen zum Windows-Paket-Manager haben, wenden Sie sich an Ihren Microsoft-Kundenbetreuer.

> [!Note]
>
> Diese API ist nicht für alle Windows-apps verfügbar. Wenn Ihr Entwicklerkonto nicht speziell von Microsoft bereitgestellt wird, schlagen Aufrufe dieser APIs zur Laufzeit fehl.
