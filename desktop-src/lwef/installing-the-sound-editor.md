# <a name="installing-the-sound-editor"></a>Installieren des Sound-Editors

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Die empfohlene Systemkonfiguration für die Verwendung des Sound-Editors ist ein PC mit Pentium 166, mindestens 48 MB RAM und einer Windows-kompatiblen Soundkarte. Wenn Sie mit dem Tool gesprochene Eingaben aufzeichnen möchten, benötigen Sie auch ein kompatibles Mikrofon.

Um das Microsoft linguistische Sound Bearbeitungs Tool zu installieren, öffnen Sie die selbst extrahierende Installationsdatei. Dadurch werden automatisch die entsprechenden Dateien auf dem System installiert. Wenn Sie den Sound-Editor von der Microsoft Agent-Website herunterladen, können Sie den Editor nach dem herunterladen oder speichern auf dem Datenträger installieren, damit er anschließend geöffnet und installiert wird. Das Installations Tool schlägt vor, sich im Unterverzeichnis "Tools" des Microsoft-Agents zu installieren. Es wird empfohlen, diesen Speicherort zu verwenden.

Die sprach Erkennungs-Engine von Microsoft Command and Control (Version. 4,0) muss auch installiert sein, bevor Sie den Sound-Editor verwenden können. Dies wird normalerweise mit dem Sound-Editor installiert, aber wenn es später deinstalliert wurde, können Sie es über das Microsoft Platform SDK oder die Microsoft Agent-Website unter neu installieren <https://www.microsoft.com/downloads/details.aspx?FamilyID=914fef6b-d764-4eca-88dc-d49d8912625c> . Der Sound-Editor kann nur linguistische Informationen auf Grundlage der Sprache generieren, die von der Sprach-Engine unterstützt wird. Zum Generieren von Informationen für andere Sprachen muss eine kompatible sprach Erkennungs-Engine für diese Sprache installiert sein. Wenden Sie sich an Ihren sprach Hersteller, um zu ermitteln, ob er das Microsoft linguistische Sound Bearbeitungs Tool unterstützt.
