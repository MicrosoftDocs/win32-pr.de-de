# <a name="iagentpropertysheet"></a>Iagentpropertysheet

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

**Iagentpropertysheet** definiert eine Schnittstelle, die es Anwendungen ermöglicht, Eigenschaften für das Eigenschaften Blatt von Microsoft-Agent (Fenster) festzulegen und abzufragen.

**Methoden in Vtable-Reihenfolge**



| Iagentpropertysheet-Methoden                             | BESCHREIBUNG                                                                                  |
|---------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**GetVisible**](iagentpropertysheet--getvisible.md)   | Gibt zurück, ob das Eigenschaften Blatt "Microsoft-Agent" sichtbar ist.                               |
| [**SetVisible**](iagentpropertysheet--setvisible.md)   | Legt die [**sichtbare**](visible-property.md) Eigenschaft des Microsoft-Agent-Eigenschaften Blatts fest. |
| [**GetPosition**](iagentpropertysheet--getposition.md) | Gibt die Position des Eigenschaften Blatts von Microsoft-Agent zurück.                                  |
| [**GetSize**](iagentpropertysheet--getsize.md)         | Gibt die Größe des Eigenschaften Blatts des Microsoft-Agents zurück.                                      |
| [**GetPage**](iagentpropertysheet--getpage.md)         | Gibt die aktuelle Seite für das Eigenschaften Blatt von Microsoft-Agent zurück.                             |
| [**Setpage**](iagentpropertysheet--setpage.md)         | Legt die aktuelle Seite für das Eigenschaften Blatt von Microsoft-Agent fest.                                |
