# <a name="iagentpropertysheet"></a>IAgentPropertySheet

\[Microsoft Agent ist ab Version Windows 7 veraltet und in nachfolgenden Versionen von Windows.\]

**IAgentPropertySheet definiert** eine Schnittstelle, mit der Anwendungen Eigenschaften für das Microsoft Agent-Eigenschaftenblatt (Fenster) festlegen und abfragen können.

**Methoden in Vtable-Reihenfolge**



| IAgentPropertySheet-Methoden                             | BESCHREIBUNG                                                                                  |
|---------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**Getvisible**](iagentpropertysheet--getvisible.md)   | Gibt zurück, ob das Microsoft Agent-Eigenschaftenblatt sichtbar ist.                               |
| [**Setvisible**](iagentpropertysheet--setvisible.md)   | Legt die [**Visible-Eigenschaft**](visible-property.md) des Microsoft Agent-Eigenschaftenblatts fest. |
| [**Getposition**](iagentpropertysheet--getposition.md) | Gibt die Position des Microsoft Agent-Eigenschaftenblatts zurück.                                  |
| [**GetSize**](iagentpropertysheet--getsize.md)         | Gibt die Größe des Microsoft Agent-Eigenschaftenblatts zurück.                                      |
| [**GetPage**](iagentpropertysheet--getpage.md)         | Gibt die aktuelle Seite für das Microsoft Agent-Eigenschaftenblatt zurück.                             |
| [**SetPage**](iagentpropertysheet--setpage.md)         | Legt die aktuelle Seite für das Microsoft Agent-Eigenschaftenblatt fest.                                |
