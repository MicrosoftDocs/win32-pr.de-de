# <a name="iagentballoonsetvisible"></a>Iagentballoon:: setVisible

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

``` syntax
HRESULT SetVisible(
   long bVisible  // word balloon Visible setting 
);
```

Legt die [**sichtbare**](visible-property.md) Eigenschaft für die Word-Sprechblase fest.

-   Gibt S \_ OK zurück, um anzugeben, dass der Vorgang erfolgreich war.

<dl> <dt>

<span id="bVisible"></span><span id="bvisible"></span><span id="BVISIBLE"></span>*bvisible*
</dt> <dd>

Sichtbare Eigenschafts Einstellung. Der Wert **true** zeigt die Word-Sprechblase an. der Wert **false** blendet ihn aus.

</dd> </dl>
