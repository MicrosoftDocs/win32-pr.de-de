# <a name="accessing-microsoft-agent-services-directly"></a>Direktes Zugreifen auf Microsoft-Agent-Dienste

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Wenn Sie C, C++ oder Java verwenden, können Sie direkt über die OLE-Schnittstellen (ActiveX) auf den Microsoft-Agent-Server zugreifen. Weitere Informationen zu diesen Schnittstellen finden Sie unter [Programmieren der Microsoft-Agent-Serverschnittstelle.](programming-the-microsoft-agent-server-interface.md)
