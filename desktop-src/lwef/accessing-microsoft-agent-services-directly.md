# <a name="accessing-microsoft-agent-services-directly"></a>Direktes Zugreifen auf Microsoft-Agent-Dienste

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Wenn Sie C, C++ oder Java verwenden, können Sie auf den Microsoft-Agent-Server direkt über seine ActiveX-Schnittstellen (OLE) zugreifen. Weitere Informationen zu diesen Schnittstellen finden Sie unter [Programmieren der Microsoft-Agent-Server Schnittstelle](programming-the-microsoft-agent-server-interface.md).
