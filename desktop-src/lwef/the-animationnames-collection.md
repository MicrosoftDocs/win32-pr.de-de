# <a name="the-animationnames-collection"></a>Die AnimationNames-Auflistung

\[Microsoft Agent ist ab Version Windows 7 veraltet und in nachfolgenden Versionen von Windows.\]

Die [**AnimationNames-Auflistung**](https://www.bing.com/search?q=**AnimationNames**) ist eine spezielle Sammlung, die die Liste der Animationsnamen enthält, die für ein Zeichen kompiliert wurden. Sie können die Auflistung verwenden, um die Namen der Animationen für ein Zeichen aufzählen. Beispielsweise können Sie in Visual Basic oder VBScript (2.0 oder höher) mit for Each... auf diese **Namen zugreifen. Nächste Anweisungen:**


```
   For Each Animation in Genie.AnimationNames
      Genie.Play Animation
   Next
```



Elemente in der Auflistung haben keine Eigenschaften, sodass nicht direkt auf einzelne Elemente zugegriffen werden kann.

Für. ACF-Zeichen gibt die Auflistung alle Animationen zurück, die für das Zeichen definiert wurden, nicht nur die, die mit der [**Get-Methode abgerufen**](get-method.md) wurden.
