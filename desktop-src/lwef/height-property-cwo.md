# <a name="height-property-commandswindow-object"></a>Height-Eigenschaft (commandswindow-Objekt)

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt einen ganzzahligen Wert zurück, der die aktuelle Höhe des Sprachbefehle Fensters in Pixel angibt.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Agent * * *. Commandswindow. Height**

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

Der Server zeigt das Fenster "Sprachbefehle" basierend auf der Position und Größe an, die vom Benutzer festgelegt wurden.
