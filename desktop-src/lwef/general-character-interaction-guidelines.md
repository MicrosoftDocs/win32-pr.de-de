# <a name="general-character-interaction-guidelines"></a>Allgemeine Richtlinien für die Zeichen Interaktion

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Grundsätze von gutem Design der Benutzeroberfläche gelten auch für das Entwerfen einer Schnittstelle mit dem Microsoft-Agent. Sie sollten sich mit ihren Benutzern und ihren Zielen vertraut machen und einen guten Entwurfsprozess für die Benutzeroberfläche befolgen. Die Richtlinien für die Windows-Benutzeroberfläche für den Software Entwurf bieten einen guten Überblick über allgemeine Entwurfs Prinzipien und-Methoden.

-   [Nicht exklusiv](be-non-exclusive.md)
-   [Entsprechendes Feedback bereitstellen](provide-appropriate-feedback.md)
-   [Natürliche Variation verwenden](use-natural-variation.md)
