# <a name="general-character-interaction-guidelines"></a>Allgemeine Richtlinien für die Zeicheninteraktion

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Die Prinzipien eines guten Entwurfs der Benutzeroberfläche gelten auch für das Entwerfen einer Schnittstelle mithilfe des Microsoft-Agents. Sie sollten Ihre Benutzer und ihre Ziele verstehen und einen guten Entwurfsprozess für die Benutzeroberfläche befolgen. Die Windows Interface Guidelines for Software Design bietet einen hervorragenden Überblick über allgemeine Entwurfsprinzipien und Methodiken.

-   [Nicht exklusiv sein](be-non-exclusive.md)
-   [Bereitstellen von entsprechendem Feedback](provide-appropriate-feedback.md)
-   [Verwenden natürlicher Variationen](use-natural-variation.md)
