# <a name="charsperline-property"></a>Charsperline (Eigenschaft)

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt die Zeichen pro Zeile zurück, die für die Wort Sprechblase für das angegebene Zeichen unterstützt werden.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Agent*[**. Zeichen ("**](/windows/desktop/lwef/the-characters-object)* Merkmal-ID ***").** Sprechblase. charsperline

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft gibt die durchschnittliche Anzahl von Zeichen (Buchstaben) zurück, die in der Wort Sprechblase als Long Integer-Wert angezeigt werden. Sie können den Wert mithilfe der [**Style**](style-property.md) -Eigenschaft festlegen.
