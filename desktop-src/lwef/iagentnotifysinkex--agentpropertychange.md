# <a name="iagentnotifysinkexagentpropertychange"></a>Iagentnotifysinkex:: agentpropertychange

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

``` syntax
HRESULT AgentPropertyChange(
);
```

Benachrichtigt eine Client Anwendung, wenn der Benutzer eine Microsoft-Agent-Eigenschaften Einstellung ändert.

-   Kein Rückgabewert.

Wenn der Benutzer eine Microsoft-Agent-Eigenschaften Einstellung auf der Eigenschaften Seite des Microsoft-Agents ändert, sendet der Server dieses Ereignis an alle Clients, es sei denn, der Server ist zurzeit angehalten.
