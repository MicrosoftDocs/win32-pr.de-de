# <a name="height-property-propertysheet-object"></a>Height-Eigenschaft (PropertySheet-Objekt)

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt einen ganzzahligen Wert zurück, der die aktuelle Höhe des Microsoft-Agent-Eigenschaften Blatt Fensters in Pixel angibt.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Agent * * *. PropertySheet. Height**

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

Der Server zeigt das Fenster basierend auf dem vom Benutzer festgelegten Speicherort an.
