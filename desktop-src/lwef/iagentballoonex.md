# <a name="iagentballoonex"></a>Iagentballoonex

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

[**Iagentballoonex**](/windows/desktop/lwef/iagentballoon) wird von der **iagentballoon** -Schnittstelle abgeleitet. Sie enthält alle **iagentballoon** -Methoden und ermöglicht den Zugriff auf zusätzliche Funktionen.

**Methoden in Vtable-Reihenfolge**



| Iagentballoonex-Methoden                                  | BESCHREIBUNG                                                        |
|----------------------------------------------------------|--------------------------------------------------------------------|
| [**GetStyle**](https://www.bing.com/search?q=**GetStyle**)                     | Gibt den Ausgabe Stil der Wort Sprechblase zurück.                           |
| [**SetStyle**](https://www.bing.com/search?q=**SetStyle**)                     | Legt den Ausgabe Stil der Wort Sprechblase fest.                              |
| [**Setnumlines**](https://www.bing.com/search?q=**SetNumLines**)               | Legt die Anzahl der Zeilen in der Wort Sprechblase fest.                  |
| [**Setnumcharsperline**](https://www.bing.com/search?q=**SetNumCharsPerLine**) | Legt die Anzahl der Zeichen pro Zeilen Ausgabe in der Wort Sprechblase fest. |
