# <a name="width-property-propertysheet-object"></a>Width-Eigenschaft (PropertySheet-Objekt)

\[Microsoft Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt einen ganzzahligen Wert zurück, der die aktuelle Breite des Microsoft Agent-Eigenschaftenblattfensters in Pixel angibt.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Syntax** 
</dt> <dd>

*agent**. PropertySheet.Width**

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

Der Server zeigt das Fenster basierend auf dem vom Benutzer festgelegten Speicherort an.
