# <a name="word-balloon-support"></a>Word-Sprechblasen Unterstützung

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Die gesprochene Ausgabe kann auch als Textausgabe in Form einer Zeichen Sprechblase für den Zeichen Begriff angezeigt werden. Dies kann verwendet werden, um die gesprochene Ausgabe eines Zeichens oder als Alternative zur Audioausgabe zu ergänzen, wenn Sie die [**Sprech**](speak-method.md) Methode verwenden.

![Ja, Master Word-Sprechblase](images/f3ballon.gif)

Sie können auch eine Word-Sprechblase verwenden, um zu übermitteln, was ein Zeichen [**mit der-**](think-method.md) Methode "Thinking" ist. Dadurch wird der Text angezeigt, den Sie in einer immer noch "Denk" Sprechblase angeben. Die **Think** -Methode unterscheidet sich auch von der [**Sprech**](speak-method.md) Methode, da Sie keine Audioausgabe erzeugt.

Word-Luftballons unterstützen nur die überprüfte Kommunikation vom Zeichen und nicht durch die Benutzereingabe. Daher unterstützt die Word-Sprechblase keine Eingabe Steuerelemente. Sie können jedoch problemlos Benutzereingaben für ein Zeichen bereitstellen, indem Sie Schnittstellen aus ihrer Programmiersprache oder die anderen vom Microsoft-Agent bereitgestellten Eingabe Dienste verwenden, z. b. das Popupmenü.

Wenn Sie ein Zeichen definieren, können Sie angeben, ob die Unterstützung der Wort Sprechblase enthalten sein soll. Wenn Sie jedoch ein Zeichen verwenden, das die Unterstützung der Wort Sprechblase umfasst, können Sie die Unterstützung nicht deaktivieren.
