# <a name="requirements-for-supporting-the-microsoft-linguistic-information-sound-editing-tool"></a>Anforderungen für die Unterstützung des Microsoft-Tools für linguistische Informationen Sound Bearbeitung

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Das Microsoft-Tool für die Audiobearbeitung verwendet eine Spracherkennungs-Engine zum Entwickeln von Wort Umbrüchen und phonetischen Informationen für standardmäßige Windows Wave Sound-Dateien (WAV). Die Version 2,0 unterstützt jetzt die Verwendung mit anderen Sprachmodulen. Anbieter, die den Sound-Editor unterstützen möchten, müssen sicherstellen, dass Ihre Engines die SAPI 4,0-Spezifikation für kontextfreie Grammatik-Engines und die folgenden Anforderungen vollständig unterstützen.

-   [Isrresgraphex und IAttribute](isrresgraphex-and-iattributes.md)
