# <a name="the-microsoft-agent-user-interface"></a>Die Benutzeroberfläche des Microsoft-Agents

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Mit dem Microsoft-Agent können Websites und konventionelle Anwendungen eine erweiterte Form der Benutzerinteraktion einschließen. Es werden mehrere Benutzeroberflächen Komponenten bereitstellt, mit denen Benutzer auf das Zeichen zugreifen und damit interagieren können, den Status des Zeichens kennen und globale Einstellungen ändern können, die alle Zeichen betreffen.

In den folgenden Abschnitten werden diese grundlegenden Elemente der Benutzeroberfläche des Microsoft-Agents beschrieben:

-   [Das Zeichen Fenster](the-character-window.md)
-   [Symbol "Taskleiste" für Zeichen](the-character-taskbar-icon.md)
-   [Das Fenster "Sprachbefehle"](the-voice-commands-window.md)
-   [Die Word-Sprechblase](the-word-balloon.md)
-   [Das Fenster "lausch Tipp"](the-listening-tip-window.md)
-   [Das Fenster "erweiterte Zeichen Optionen"](the-advanced-character-options-window.md)
-   [Das standardmäßige Zeichen Eigenschaften Fenster](the-default-character-properties-window.md)
