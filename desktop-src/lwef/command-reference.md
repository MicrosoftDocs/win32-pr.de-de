# <a name="command-reference-linguistic-information-sound-editing-tool"></a>Befehlsreferenz (Audiobearbeitungstool für linguistische Informationen)

\[Microsoft Agent ist ab Version Windows 7 veraltet und in nachfolgenden Versionen von Windows.\]

### <a name="the-file-menu"></a>Menü "Datei"

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Neu**
</dt> <dd>

Setzt den Sound-Editor zurück, um eine neue erweiterte Sounddatei zu erstellen. Wenn eine vorhandene Sounddatei geladen wird und über nicht gespeicherte Bearbeitungen verfügt, zeigt der Sound-Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden.

</dd> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Öffnen**
</dt> <dd>

Zeigt das Dialogfeld Öffnen an, in dem Sie eine vorhandene Sounddatei öffnen können. Wenn eine vorhandene Sounddatei geladen wird und über nicht gespeicherte Bearbeitungen verfügt, zeigt der Sound-Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden.

</dd> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Speichern**
</dt> <dd>

Speichert eine Sounddatei. Wenn die Sounddatei nicht vorhanden ist (nicht benannt wurde), zeigt der Sound-Editor das Dialogfeld Speichern unter für die Eingabe des Dateinamens an.

</dd> <dt>

<span id="Save_As"></span><span id="save_as"></span><span id="SAVE_AS"></span>**Speichern unter**
</dt> <dd>

Zeigt das Dialogfeld Speichern unter an, in dem Sie einen neuen Namen für die Sounddatei eingeben können.

</dd> <dt>

<span id="Save_Selection_As"></span><span id="save_selection_as"></span><span id="SAVE_SELECTION_AS"></span>**Auswahl speichern unter**
</dt> <dd>

Zeigt das Dialogfeld Auswahl speichern unter an, in dem Sie einen Namen für den ausgewählten Teil der Sounddatei eingeben können.

</dd> <dt>

<span id="Most_Recently_Open_Files"></span><span id="most_recently_open_files"></span><span id="MOST_RECENTLY_OPEN_FILES"></span>**Zuletzt geöffnete Dateien**
</dt> <dd>

Verfolgt die zuletzt geöffneten Zeichendefinitionsdateien nach. Wenn Sie eine Datei auswählen, wird diese Datei automatisch zur Bearbeitung geöffnet. Wenn ein vorhandenes Zeichen geladen wird und nicht gespeicherte Änderungen an einer Datei enthält, zeigt der Sound-Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden.

</dd> <dt>

<span id="Exit"></span><span id="exit"></span><span id="EXIT"></span>**Ausfahrt**
</dt> <dd>

Beendet den Sound-Editor. Wenn eine vorhandene Datei geladen wird und nicht gespeicherte Bearbeitungen enthält, zeigt der Sound-Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden.

</dd> </dl>

### <a name="the-edit-menu"></a>Menü "Bearbeiten"

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Rückgängig**
</dt> <dd>

Entfernt eine Im Sound-Editor vorgenommene Änderung.

</dd> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Wiederholen**
</dt> <dd>

Kehrt eine Rückgängig-Aktion im Sound-Editor um.

</dd> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Schneiden**
</dt> <dd>

Entfernt den ausgewählten Text und platziert ihn in der Zwischenablage.

</dd> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Kopieren**
</dt> <dd>

Kopiert den ausgewählten Text in die Zwischenablage.

</dd> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Einfügen**
</dt> <dd>

Kopiert Text in der Zwischenablage in die Einfügemarke oder -auswahl im Textfeld Textdarstellung.

</dd> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Löschen**
</dt> <dd>

Entfernt den ausgewählten Text.

</dd> <dt>

<span id="Select_All"></span><span id="select_all"></span><span id="SELECT_ALL"></span>**Wählen Sie Alle aus.**
</dt> <dd>

Wählt den Text im Textfeld Textdarstellung aus.

</dd> <dt>

<span id="Generate_Linguistic_Info"></span><span id="generate_linguistic_info"></span><span id="GENERATE_LINGUISTIC_INFO"></span>**Generieren von linguistischen Informationen**
</dt> <dd>

Beginnt mit der Generierung von Wörtern und Phoneminformationen für eine Sounddatei.

</dd> <dt>

<span id="Insert_Phoneme"></span><span id="insert_phoneme"></span><span id="INSERT_PHONEME"></span>**Einfügen von Phoneme**
</dt> <dd>

Zeigt das Dialogfeld Phoneme einfügen an, in dem Sie eine ausgewählte Phonembezeichnung einfügen können.

</dd> <dt>

<span id="Replace_Phoneme"></span><span id="replace_phoneme"></span><span id="REPLACE_PHONEME"></span>**Phoneme ersetzen**
</dt> <dd>

Zeigt das Dialogfeld Phoneme ersetzen an, in dem Sie die ausgewählte Phonembezeichnung ersetzen können.

</dd> <dt>

<span id="Delete_Phoneme"></span><span id="delete_phoneme"></span><span id="DELETE_PHONEME"></span>**Phoneme löschen**
</dt> <dd>

Löscht die ausgewählte Phonembezeichnung.

</dd> <dt>

<span id="Insert_Word"></span><span id="insert_word"></span><span id="INSERT_WORD"></span>**Wort einfügen**
</dt> <dd>

Zeigt das Dialogfeld Wort einfügen an, mit dem Sie eine Wortbezeichnung in die Audiodarstellung einfügen können.

</dd> <dt>

<span id="Replace_Word"></span><span id="replace_word"></span><span id="REPLACE_WORD"></span>**Word ersetzen**
</dt> <dd>

Zeigt das Dialogfeld Word ersetzen an, in dem Sie die ausgewählte Wortbezeichnung in der Audiodarstellung ersetzen können.

</dd> <dt>

<span id="Delete_Word"></span><span id="delete_word"></span><span id="DELETE_WORD"></span>**Löschen von Wörtern**
</dt> <dd>

Löscht die ausgewählte Wortbezeichnung in der Audiodarstellung.

</dd> <dt>

<span id="Phoneme_Label_Display"></span><span id="phoneme_label_display"></span><span id="PHONEME_LABEL_DISPLAY"></span>**Anzeige von Phoneme-Bezeichnungen**
</dt> <dd>

Ändert die Anzeige der Phonembezeichnung zwischen beschreibenden Namen und IPA-Bytewerten.

</dd> <dt>

<span id="Speech_Engine"></span><span id="speech_engine"></span><span id="SPEECH_ENGINE"></span>**Speech-Engine**
</dt> <dd>

Ermöglicht es Ihnen, die Sprach-Engine zu ändern, die Sie zum Generieren der Wörterbruch- und Phoneminformationen verwenden.

</dd> </dl>

### <a name="the-audio-menu"></a>Das Menü "Audio"

<dl> <dt>

<span id="Play"></span><span id="play"></span><span id="PLAY"></span>**Spielen**
</dt> <dd>

Gibt die Sounddatei oder den ausgewählten Teil der Sounddatei wieder.

</dd> <dt>

<span id="Record"></span><span id="record"></span><span id="RECORD"></span>**Aufzeichnung**
</dt> <dd>

Zeichnet eine neue Sounddatei auf.

</dd> <dt>

<span id="Pause"></span><span id="pause"></span><span id="PAUSE"></span>**Anhalten**
</dt> <dd>

Hält die Wiedergabe der Sounddatei oder des ausgewählten Teils der Sounddatei an. Verwenden Sie Wiedergabe, um die Wiedergabe wieder aufzunehmen.

</dd> <dt>

<span id="Stop"></span><span id="stop"></span><span id="STOP"></span>**Stoppen**
</dt> <dd>

Beendet die Aufzeichnung oder Wiedergabe der Sounddatei oder des ausgewählten Teils der Sounddatei.

</dd> </dl>
