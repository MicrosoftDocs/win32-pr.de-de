# <a name="command-reference-linguistic-information-sound-editing-tool"></a>Befehlsreferenz (Sound Bearbeitungs Tool für linguistische Informationen)

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

### <a name="the-file-menu"></a>Menü "Datei"

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Neu**
</dt> <dd>

Setzt den Sound-Editor zum Erstellen einer neuen erweiterten Audiodatei zurück. Wenn eine vorhandene Audiodatei geladen wird und ungespeicherte Änderungen aufweist, zeigt der Sound-Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden sollen.

</dd> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Eren**
</dt> <dd>

Zeigt das Dialogfeld Öffnen an, in dem Sie eine vorhandene Audiodatei öffnen können. Wenn eine vorhandene Audiodatei geladen wird und ungespeicherte Änderungen aufweist, zeigt der Sound-Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden sollen.

</dd> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Sicher**
</dt> <dd>

Speichert eine Audiodatei. Wenn die Audiodatei nicht vorhanden ist (wurde nicht benannt), zeigt der Sound-Editor das Dialogfeld Speichern unter für die Eingabe des Datei namens an.

</dd> <dt>

<span id="Save_As"></span><span id="save_as"></span><span id="SAVE_AS"></span>**Speichern unter**
</dt> <dd>

Zeigt das Dialogfeld Speichern unter an, in dem Sie einen neuen Namen für die Audiodatei eingeben können.

</dd> <dt>

<span id="Save_Selection_As"></span><span id="save_selection_as"></span><span id="SAVE_SELECTION_AS"></span>**Auswahl speichern unter**
</dt> <dd>

Zeigt das Dialogfeld Auswahl speichern unter an, in dem Sie einen Namen für den ausgewählten Teil der Audiodatei eingeben können.

</dd> <dt>

<span id="Most_Recently_Open_Files"></span><span id="most_recently_open_files"></span><span id="MOST_RECENTLY_OPEN_FILES"></span>**Zuletzt geöffnete Dateien**
</dt> <dd>

Verfolgt die zuletzt geöffneten Zeichen Definitions Dateien, die Sie geöffnet haben. Wenn Sie eine Datei auswählen, wird diese Datei automatisch zur Bearbeitung geöffnet. Wenn ein vorhandenes Zeichen geladen wird und ungespeicherte Änderungen an einer Datei vorgenommen wurden, zeigt der Sound-Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden sollen.

</dd> <dt>

<span id="Exit"></span><span id="exit"></span><span id="EXIT"></span>**Abstiegs**
</dt> <dd>

Beendet den Sound-Editor. Wenn eine vorhandene Datei geladen wird und ungespeicherte Änderungen aufweist, zeigt der Sound-Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden sollen.

</dd> </dl>

### <a name="the-edit-menu"></a>Menü "Bearbeiten"

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Machen**
</dt> <dd>

Entfernt eine im Sound-Editor vorgenommene Änderung.

</dd> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Holen**
</dt> <dd>

Kehrt eine Rückgängig-Aktion im Sound-Editor um.

</dd> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Schnitts**
</dt> <dd>

Entfernt den markierten Text und legt ihn in der Zwischenablage ab.

</dd> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Skopie**
</dt> <dd>

Kopiert den ausgewählten Text in die Zwischenablage.

</dd> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Kle**
</dt> <dd>

Kopiert Text in der Zwischenablage in die Einfügemarke oder Auswahl im Textfeld Textdarstellung.

</dd> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Lösch**
</dt> <dd>

Entfernt den markierten Text.

</dd> <dt>

<span id="Select_All"></span><span id="select_all"></span><span id="SELECT_ALL"></span>**Alle auswählen**
</dt> <dd>

Wählt den Text im Textfeld Textdarstellung aus.

</dd> <dt>

<span id="Generate_Linguistic_Info"></span><span id="generate_linguistic_info"></span><span id="GENERATE_LINGUISTIC_INFO"></span>**Linguistische Informationen generieren**
</dt> <dd>

Beginnt mit dem Erstellen von Wort Umbruch-und Phonem-Informationen für eine Audiodatei.

</dd> <dt>

<span id="Insert_Phoneme"></span><span id="insert_phoneme"></span><span id="INSERT_PHONEME"></span>**Einfügen von Phoneme**
</dt> <dd>

Zeigt das Dialogfeld Phonem einfügen an, in dem Sie eine ausgewählte Phonem-Bezeichnung einfügen können.

</dd> <dt>

<span id="Replace_Phoneme"></span><span id="replace_phoneme"></span><span id="REPLACE_PHONEME"></span>**Phoneme ersetzen**
</dt> <dd>

Zeigt das Dialogfeld Phonem ersetzen an, in dem Sie die ausgewählte Phonem-Bezeichnung ersetzen können.

</dd> <dt>

<span id="Delete_Phoneme"></span><span id="delete_phoneme"></span><span id="DELETE_PHONEME"></span>**Löschen von Phoneme**
</dt> <dd>

Löscht die ausgewählte Phonem-Bezeichnung.

</dd> <dt>

<span id="Insert_Word"></span><span id="insert_word"></span><span id="INSERT_WORD"></span>**Wort einfügen**
</dt> <dd>

Zeigt das Dialogfeld Wort einfügen an, in dem Sie eine Wort Bezeichnung in die Audiodarstellung einfügen können.

</dd> <dt>

<span id="Replace_Word"></span><span id="replace_word"></span><span id="REPLACE_WORD"></span>**Wort ersetzen**
</dt> <dd>

Zeigt das Dialogfeld Wort ersetzen an, in dem Sie die ausgewählte Word-Bezeichnung in der Audiodarstellung ersetzen können.

</dd> <dt>

<span id="Delete_Word"></span><span id="delete_word"></span><span id="DELETE_WORD"></span>**Wort löschen**
</dt> <dd>

Löscht die ausgewählte Wort Bezeichnung in der Audiodarstellung.

</dd> <dt>

<span id="Phoneme_Label_Display"></span><span id="phoneme_label_display"></span><span id="PHONEME_LABEL_DISPLAY"></span>**Anzeige der Phoneme-Bezeichnung**
</dt> <dd>

Ändert die Anzeige der Phonem-Bezeichnung zwischen beschreibenden Namen und IPA-Byte Werten.

</dd> <dt>

<span id="Speech_Engine"></span><span id="speech_engine"></span><span id="SPEECH_ENGINE"></span>**Sprach-Engine**
</dt> <dd>

Ermöglicht es Ihnen, die Sprach-Engine zu ändern, mit der Sie die Wörter Trennungs-und Phonem-Informationen generieren.

</dd> </dl>

### <a name="the-audio-menu"></a>Das Audiomenü

<dl> <dt>

<span id="Play"></span><span id="play"></span><span id="PLAY"></span>**Theater**
</dt> <dd>

Gibt die Audiodatei oder den ausgewählten Teil der Audiodatei wieder.

</dd> <dt>

<span id="Record"></span><span id="record"></span><span id="RECORD"></span>**Aufnahme**
</dt> <dd>

Zeichnet eine neue Audiodatei auf.

</dd> <dt>

<span id="Pause"></span><span id="pause"></span><span id="PAUSE"></span>**Brechung**
</dt> <dd>

Hält die Wiedergabe der Audiodatei oder den ausgewählten Teil der Audiodatei an. Verwenden von Play zum Fortsetzen der Wiedergabe.

</dd> <dt>

<span id="Stop"></span><span id="stop"></span><span id="STOP"></span>**Anzuhalten**
</dt> <dd>

Beendet die Aufzeichnung oder Wiedergabe der Audiodatei oder des ausgewählten Teils der Audiodatei.

</dd> </dl>
