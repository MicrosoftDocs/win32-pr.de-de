# <a name="use-appropriate-timing-and-emphasis"></a>Verwenden der richtigen zeitlichen Steuerung und Hervorhebung

\[Microsoft Agent ist ab Version Windows 7 veraltet und möglicherweise in nachfolgenden Versionen von Windows.\]

Wie bei jedem Feedback hängt die Effektivität der Sprachausgabe von der zeitlichen Steuerung und hervorhebung ab. Viele Informationen können in Der Geschwindigkeit, Lautstärke und Tonhöhe kommuniziert werden, die verwendet werden, wenn etwas gesprochen wird. Wenn Sie eine Text-to-Speech-Engine als Stimme Ihrer Figur verwenden, können Sie in den meisten Engines geschwindigkeit, Pausen, Tonhöhe und Hervorhebung von Wörtern oder Ausdrücken festlegen. Sie können diese Attribute verwenden, um Interesse und Verständnis des Zeichens anzugeben sowie die Aufmerksamkeit des Benutzers oder die Absicht Ihres Zeichens anzugeben. Weitere Informationen zum Festlegen von Sprachattributen finden Sie unter [Microsoft Agent Speech Output Tags](microsoft-agent-speech-output-tags.md). Wenn Sie Sounddateien als Ausgabe Ihres Zeichens verwenden, berücksichtigen Sie diese Faktoren auch in Ihren aufgezeichneten Audiodaten.
