# <a name="use-appropriate-timing-and-emphasis"></a>Angemessene zeitliche Planung und Betonung verwenden

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Wie bei allen Feedback hängt auch die Effektivität der Sprachausgabe von der zeitlichen Steuerung und dem Fokus ab. Eine gute Menge an Informationen kann in der Geschwindigkeit, dem Volume und der Tonhöhe kommuniziert werden, die verwendet werden, wenn etwas gesprochen wird. Wenn Sie eine Text-zu-Sprache-Engine als Stimme Ihres Zeichens verwenden, können Sie mit den meisten Modulen Geschwindigkeit, Pausen, Tonhöhe und Betonung von Wörtern oder Ausdrücken festlegen. Sie können diese Attribute verwenden, um die Interessen und das Verständnis des Zeichens anzugeben und die Aufmerksamkeit des Benutzers zu lenken oder die Absicht Ihres Zeichens anzugeben. Weitere Informationen zum Festlegen von sprach Attributen finden Sie unter [Microsoft Agent Speech Output Tags](microsoft-agent-speech-output-tags.md). Wenn Sie Audiodateien als Ausgabe Ihres Zeichens verwenden, sollten Sie diese Faktoren auch in der aufgezeichneten Audiodatei berücksichtigen.
