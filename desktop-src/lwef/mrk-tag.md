# <a name="mrk-tag"></a>Mrk-Tag

\[Microsoft Agent ist ab Version Windows 7 veraltet und in nachfolgenden Versionen von Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Definiert ein Lesezeichen im gesprochenen Text.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

**\\ Mrk=**_number_*_\\_*



| Teil     | BESCHREIBUNG                                        |
|----------|----------------------------------------------------|
| *Zahl* | Ein ganzzahliger Long-Wert, der das Lesezeichen identifiziert. |



 

</dd> </dl>

### <a name="remarks"></a>Hinweise

Wenn der Server ein Lesezeichen verarbeitet, generiert er ein Lesezeichenereignis. Sie müssen eine Zahl angeben, die größer als 0 (null) und nicht gleich 2147483647 oder 2147483646.
