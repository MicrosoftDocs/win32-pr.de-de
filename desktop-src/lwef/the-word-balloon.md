# <a name="the-word-balloon"></a>Die Word-Sprechblase

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Zusätzlich zu der gesprochenen Audioausgabe unterstützt die Microsoft-Agent-Schnittstelle Text Beschriftungen in Form von Textausgabe in Wort Blasen im Zeichenstil. Wörter werden in der Sprechblase angezeigt, wenn Sie gesprochen werden. Die Sprechblase wird ausgeblendet, wenn die gesprochene Ausgabe abgeschlossen ist. Das Fenster Erweiterte Zeichen Optionen stellt Optionen zum Deaktivieren der Anzeige des Sprechblasen sowie zum Steuern von Attributen in Bezug auf seine Darstellung bereit.
