# <a name="the-word-balloon"></a>Die Wortsprechblase

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Zusätzlich zur Ausgabe von gesprochenen Audiodaten unterstützt die Microsoft-Agent-Schnittstelle Textbeschriftungen in Form von Textausgaben in Wortsprechblasen im Stil von Wörtern. Wörter werden im Sprechblasen angezeigt, während sie gesprochen werden. Der Sprechblasen wird ausgeblendet, wenn die gesprochene Ausgabe abgeschlossen ist. Das Fenster Erweiterte Zeichenoptionen bietet Optionen zum Deaktivieren der Anzeige des Sprechblasens sowie Steuerungsattribute im Zusammenhang mit seiner Darstellung.
