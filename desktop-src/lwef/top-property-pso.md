# <a name="top-property-propertysheet-object"></a>Top-Eigenschaft (PropertySheet-Objekt)

\[Microsoft Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt einen ganzzahligen Wert zurück, der den aktuellen oberen Rand des Eigenschaftenblattfensters von Microsoft Agent in Pixel angibt.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*agent**. PropertySheet.Top**

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

Der Server zeigt das Fenster basierend auf dem vom Benutzer festgelegten Speicherort an.
