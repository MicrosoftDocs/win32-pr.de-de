# <a name="using-the-microsoft-agent-character-editor"></a>Verwenden des Microsoft-Agent-Zeichen-Editors

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Mit dem Zeichen-Editor von Microsoft-Agent können Sie Zeichen Animationen für die Verwendung mit dem Microsoft-Agent kompilieren. Sie können Animationen definieren, indem Sie Windows-Bitmapbilder importieren, deren Dauer festlegen und optional Verzweigungen, Soundeffekte und Sprech Überlagerungen einschließen. Weitere Informationen zum Entwerfen der Animationen eines Zeichens finden Sie unter [Entwerfen von Zeichen für den Microsoft-Agent](designing-characters-for-microsoft-agent.md).

Mit dem Agent-Zeichen-Editor können Sie den Namen und die Beschreibung des Zeichens sowie Ausgabeoptionen definieren, einschließlich Text-to-Speech (TTS), synthetischer Sprachausgabe, Popup-Menü Unterstützung und der Wort Sprechblasen-Design der Zeichen.

In diesem Dokument wird der Microsoft-Agent-Zeichen-Editor für Microsoft Agent beschrieben, den Sie unter herunterladen können <https://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=c21be0ec-1712-427d-a43d-94c122ef8eb6> .
