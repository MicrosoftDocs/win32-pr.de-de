# <a name="iagentnotifysinkex"></a>Iagentnotifysinkex

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

**Iagentnotifysinkex** wird von der [**iagentnotifysink**](https://www.bing.com/search?q=**IAgentNotifySink**) -Schnittstelle abgeleitet. Sie enthält alle **iagentnotifysink** -Methoden und ermöglicht den Zugriff auf zusätzliche Funktionen.

**Methoden in Vtable-Reihenfolge**



| Iagentnotifysinkex-Methoden                                                   | BESCHREIBUNG                                                        |
|------------------------------------------------------------------------------|--------------------------------------------------------------------|
| [**Helpcomplete**](iagentnotifysinkex--helpcomplete.md)                     | Tritt ein, wenn der Benutzer ein Menü oder das Zeichen im Hilfe Modus auswählt. |
| [**Listeningstate**](iagentnotifysinkex--listeningstate.md)                 | Tritt auf, wenn sich der gelesende Zustand des Zeichens ändert.               |
| [**Defaultcharacters-Änderung**](iagentnotifysinkex--defaultcharacterchange.md) | Tritt ein, wenn der Benutzer das Standard Zeichen ändert.                |
| [**Agentpropertychange**](iagentnotifysinkex--agentpropertychange.md)       | Tritt ein, wenn der Benutzer eine agenteigenschafteneinstellung ändert.            |
| [**Activeclientchange**](iagentnotifysinkex--activeclientchange.md)         | Tritt auf, wenn sich der aktive Client eines Zeichens ändert.              |
