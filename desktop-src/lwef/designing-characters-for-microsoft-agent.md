Sie enthält konzeptionelle und technische Informationen zum Entwurf von Zeichen, Bildern und Animationen. Die Größe, die Verwendung von Farben und die Typen von Bildern, die Sie erstellen müssen; vorgeschlagene Animationen; Sprechanimationen; - und -Agent-Zustände.
