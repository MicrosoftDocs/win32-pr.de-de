# <a name="designing-characters-for-microsoft-agent"></a>Entwerfen von Zeichen für den Microsoft-Agent

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Dieses Dokument enthält Informationen, die Sie beim Entwerfen und Entwickeln eines Zeichens für die Verwendung mit Microsoft Agent unterstützen. Sie enthält konzeptionelle und technische Informationen zum Zeichen-, Bild- und Animationsentwurf. Größe, Verwendung von Farben und Bildtypen, die Sie erstellen müssen; vorgeschlagene Animationen; Sprechanimationen; - und -Agent-Status. Effektive Animationsprinzipien werden auch für Ihre Verwendung beim Erstellen visuell animierter animierter Figuren beschrieben.

-   [Animationen](animations.md)
-   [Agent-Status](agent-states.md)
