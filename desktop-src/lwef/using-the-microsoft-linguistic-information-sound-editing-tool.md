# <a name="using-the-linguistic-information-sound-editing-tool"></a>Verwenden des Audiobearbeitungs Tools für linguistische Informationen

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Das Tool für die Audiobearbeitung von linguistischen Informationen ermöglicht es Ihnen, Phonem-und Wort Umbruch Informationen zu generieren, um Windows Sound-Dateien (WAV-Dateien) zur Unterstützung der Animation mit hoher Qualität zu unterstützen. In den folgenden Abschnitten wird beschrieben, wie das Tool verwendet wird.

-   [Installieren des Sound-Editors](installing-the-sound-editor.md)
-   [Starten des Sound-Editors](starting-the-sound-editor.md)
-   [Erstellen einer neuen Audiodatei](creating-a-new-sound-file.md)
-   [Laden einer vorhandenen Audiodatei](loading-an-existing-sound-file.md)
-   [Erstellen von linguistischen Informationen](generating-linguistic-information.md)
-   [Speichern einer Audiodatei](saving-a-sound-file.md)
-   [Befehlsreferenz](command-reference.md)
-   [Symbolleistenschaltflächen](toolbar-buttons.md)

Sie können linguistisch erweiterte Audiodateien verwenden, die mit dem Sound-Editor generiert wurden, um die über-und Ausgabe von Microsoft-agentzeichen zu unterstützen. Übergeben Sie zu diesem Zweck einfach die Datei als Parameter an die [**Sprech**](speak-method.md) Methode. Weitere Informationen finden Sie unter [Programmieren des Microsoft-Agent-Steuer](programming-the-microsoft-agent-control.md) Elements oder [Programmieren der Microsoft-Agent-Server Schnittstelle](programming-the-microsoft-agent-server-interface.md).
