# <a name="using-the-linguistic-information-sound-editing-tool"></a>Verwenden des Audiobearbeitungstools für linguistische Informationen

\[Microsoft Agent ist ab Version Windows 7 veraltet und in nachfolgenden Versionen von Windows.\]

Mit dem Bearbeitungstool für linguistische Informationen können Sie Phonem- und Wörterpauseninformationen generieren, um Windows-Sounddateien (WAV)-Dateien zu erweitern, um qualitativ hochwertige Zeichenanimationen mit Synchronisierungszeichen in hoher Qualität zu unterstützen. In den folgenden Abschnitten wird die Verwendung des Tools beschrieben.

-   [Installieren des Sound-Editors](installing-the-sound-editor.md)
-   [Starten des Sound-Editors](starting-the-sound-editor.md)
-   [Erstellen einer neuen Sounddatei](creating-a-new-sound-file.md)
-   [Laden einer vorhandenen Sounddatei](loading-an-existing-sound-file.md)
-   [Generieren linguistischer Informationen](generating-linguistic-information.md)
-   [Speichern einer Sounddatei](saving-a-sound-file.md)
-   [Befehlsreferenz](command-reference.md)
-   [Symbolleistenschaltflächen](toolbar-buttons.md)

Sie können linguistisch erweiterte Sounddateien verwenden, die mit dem Sound-Editor generiert wurden, um die synchrone Microsoft Agent-Zeichenausgabe zu unterstützen. Übergeben Sie dazu einfach die Datei als Parameter an die [**Speak-Methode.**](speak-method.md) Weitere Informationen finden Sie unter [Programmieren des Microsoft Agent-Steuerelements oder](programming-the-microsoft-agent-control.md) [Programmieren der Microsoft Agent-Serverschnittstelle.](programming-the-microsoft-agent-server-interface.md)
