# <a name="originalheight-property"></a>OriginalHeight-Eigenschaft

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt einen ganzzahligen Wert zurück, der die ursprüngliche Höhe des Zeichens angibt.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*agent.***Characters("**_CharacterID_*_"). OriginalHeight_*

</dd> </dl>

## <a name="remarks"></a>Hinweise

Diese Eigenschaft gibt die Framehöhe des Zeichens zurück, wie mit dem Microsoft-Agent-Zeichen-Editor erstellt.
