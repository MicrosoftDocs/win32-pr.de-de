# <a name="originalheight-property"></a>Originalheight-Eigenschaft

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt einen ganzzahligen Wert zurück, der die ursprüngliche Höhe des Zeichens angibt.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Büros. ***Zeichen ("*** Merkmal-ID * * *"). OriginalHeight**

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft gibt die Rahmenhöhe des Zeichens zurück, die mit dem Microsoft-Agent-Zeichen-Editor erstellt wurde.
