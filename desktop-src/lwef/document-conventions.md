# <a name="document-conventions"></a>Dokument Konventionen

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

In dieser Dokumentation werden die folgenden typografischen Konventionen verwendet:



| Konvention                       | BESCHREIBUNG                                                                    |
|----------------------------------|--------------------------------------------------------------------------------|
| **Sub, Visible, Caption**        | Fett formatierte Wörter mit dem ersten Buchstaben in Großbuchstaben.               |
| *Agent, Zeichenfolge, jetzt*             | Kursiv geschriebene Wörter geben Platzhalter für Informationen an, die Sie angeben.                 |
| EINGABETASTE, F1                        | Wörter in allen Großbuchstaben weisen auf Dateinamen, Schlüsselnamen und schlüsselsequenzen hin. |
| `Agent1.Commands.Enabled = True` | Text in dieser Schriftart zeigt Codebeispiele an.                                       |
| `' This is a comment`            | Ein Apostroph ( `'` ) gibt einen Code Kommentar an.                                  |
| `Agent1.Commands.Add "Test1", _` | Ein Leerzeichen und ein Unterstrich ( \_ ) werden eine Codezeile fortgesetzt.                       |
| `[words or expression]`          | Elemente in eckigen Klammern sind optional.                                     |
| `This | That`                    | Ein senkrechter Strich gibt eine Auswahl zwischen zwei oder mehr Elementen an.                   |
| *Büros*                          | Das Wort "Agent" in kursiv Schrift steht für den Namen des verwendeten agentsteuerelements.  |



 

Die Beschreibungen der Programmierschnittstellen in diesem Dokument entsprechen den Konventionen für Microsoft VBScript. Sie sollten jedoch in der Regel auch für andere Sprachen gelten.
