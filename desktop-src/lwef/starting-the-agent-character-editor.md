# <a name="starting-the-agent-character-editor"></a>Starten des Agent-Zeichen-Editors

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Wählen Sie zum Ausführen des Agent-Zeichen-Editors im **Startmenü** der Windows Taskleiste die Option **Agent-Zeichen-Editor** aus, oder doppelklicken Sie auf das Symbol **Microsoft-Agent-Zeichen-Editor** auf Ihrem Desktop. Das Fenster des Editors wird geöffnet und zeigt seine Menüs, eine Symbolleiste mit häufig verwendeten Befehlen, eine Struktur mit den Komponenten, aus denen die Definition eines Zeichens besteht, und eine Reihe von Seiten im Registerkartenbereich an, die sich basierend auf Ihrer Auswahl in der Komponentenstruktur ändern.

![Screenshot: Seite "Eigenschaften" des Zeichens "Microsoft-Agent-Zeichen-Editor"](images/f1chared.gif)

Um über die Tastatur auf Felder im Fenster zuzugreifen, können Sie die TAB-TASTE und UMSCHALT+TAB verwenden, um zwischen Steuerelementen zu navigieren, oder die Zugriffstaste (ALT+*unterstrichener Buchstabe*) verwenden, um zu einem bestimmten Steuerelement zu wechseln. Nachdem der Editor gestartet wurde, können Sie mit dem Erstellen einer neuen Zeichendefinition beginnen oder eine vorhandene Zeichendefinition laden.

Die Statusleiste zeigt Informationen zu Befehlen oder Symbolleistenschaltflächen an, wenn Sie den Zeiger darauf bewegen. Außerdem werden zusammenfassende Informationen zu Ihren Zeichenanimationsdaten und Statusinformationen angezeigt, wenn Sie ein Zeichen erstellen.
