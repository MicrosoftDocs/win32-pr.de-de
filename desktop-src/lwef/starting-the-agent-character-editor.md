# <a name="starting-the-agent-character-editor"></a>Starten des Agent-Zeichen-Editors

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Um den Agent-Zeichen-Editor auszuführen, wählen Sie im **Startmenü** der Windows-Taskleiste die Option **Agent-Zeichen-Editor** aus, oder Doppelklicken Sie auf dem Desktop auf das Symbol Microsoft Agent-Zeichen- **Editor** . Das Fenster des Editors wird geöffnet und zeigt die Menüs, eine Symbolleiste mit häufig verwendeten Befehlen, eine Struktur mit den Komponenten, die die Definition eines Zeichens bilden, und einen Satz von Seiten im Registerkarten Format an, die sich basierend auf Ihrer Auswahl in der Komponenten Struktur ändern.

![Screenshot, der die Seite "Eigenschaften" des Zeichen-Editors "Zeichen-Microsoft-Agent" anzeigt.](images/f1chared.gif)

Um über die Tastatur auf Felder im Fenster zuzugreifen, können Sie die Tab-Taste und die UMSCHALT + TAB-Taste verwenden, um zwischen Steuerelementen zu navigieren, oder Sie können die Zugriffstaste (alt +*unterstrichener Buchstabe*) verwenden, um zu einem bestimmten Steuerelement zu wechseln Nachdem der Editor gestartet wurde, können Sie beginnen, eine neue Zeichen Definition zu erstellen oder eine vorhandene Zeichen Definition zu laden.

In der Statusleiste werden Informationen zu Befehlen oder Symbolleisten-Schaltflächen angezeigt, wenn Sie den Mauszeiger darüber bewegen. Außerdem werden zusammenfassende Informationen zu ihren Zeichen Animationsdaten und Statusinformationen angezeigt, wenn Sie ein Zeichen erstellen.
