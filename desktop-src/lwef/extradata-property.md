# <a name="extradata-property"></a>ExtraData-Eigenschaft

\[Microsoft Agent ist ab Version Windows 7 veraltet und in nachfolgenden Versionen von Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt eine Zeichenfolge zurück, die zusätzliche Daten angibt, die als Teil des Zeichens gespeichert sind.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*-Agent.* **Characters("**_CharacterID_*_"). ExtraData_*

</dd> </dl>

## <a name="remarks"></a>Hinweise

Der Standardwert für die **ExtraData-Eigenschaft** für ein Zeichen wird definiert, wenn das Zeichen mit dem Microsoft Agent-Zeichen-Editor kompiliert wird. Sie kann zur Laufzeit nicht geändert oder angegeben werden.

> [!Note]  
> Die **Eigenschaftseinstellung ExtraData** ist optional und wird möglicherweise nicht für alle Zeichen angegeben.
