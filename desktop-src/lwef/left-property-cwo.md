# <a name="left-property-commandswindow-object"></a>Left-Eigenschaft (CommandsWindow-Objekt)

\[Microsoft Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt einen ganzzahligen Wert zurück, der den linken Rand des Sprachbefehlsfensters in Pixel angibt.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*agent**. CommandsWindow.Left**

</dd> </dl>

## <a name="remarks"></a>Hinweise

Der Server zeigt das Sprachbefehlsfenster basierend auf der vom Benutzer festgelegten Position und Größe an.
