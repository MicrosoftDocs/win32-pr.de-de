# <a name="toolbar-buttons-linguistic-information-sound-editing-tool"></a>Schaltflächen der Symbolleiste (Linguistic Information Sound Editing Tool)

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

:::image type="icon" source="images/f9charnew.gif":::

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Neu**
</dt> <dd>

Setzt den Sound-Editor zum Erstellen einer neuen Audiodatei zurück. Wenn eine vorhandene Audiodatei geladen wird und ungespeicherte Änderungen aufweist, zeigt der Sound-Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden sollen.

</dd> </dl>

:::image type="icon" source="images/f10charopen.gif":::

<dl> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Eren**
</dt> <dd>

Zeigt das Dialogfeld Datei öffnen an, in dem Sie eine vorhandene Audiodatei öffnen können. Wenn eine vorhandene Audiodatei geladen wird und ungespeicherte Änderungen aufweist, zeigt der Sound-Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden sollen.

</dd> </dl>

:::image type="icon" source="images/f11charsave.gif":::

<dl> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Sicher**
</dt> <dd>

Speichert die Audiodatei. Wenn die Datei nicht vorhanden ist (wurde nicht benannt), zeigt der Editor das Dialogfeld Speichern unter für die Eingabe des Datei namens an.

</dd> </dl>

:::image type="icon" source="images/f14charcut.gif":::

<dl> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Schnitts**
</dt> <dd>

Entfernt den ausgewählten Text aus dem Editor und platziert ihn in der Windows-Zwischenablage.

</dd> </dl>

:::image type="icon" source="images/f15charcopy.gif":::

<dl> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Skopie**
</dt> <dd>

Kopiert den markierten Text im Editor in die Windows-Zwischenablage.

</dd> </dl>

:::image type="icon" source="images/f16charpaste.gif":::

<dl> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Kle**
</dt> <dd>

Kopiert Text aus der aktuellen Windows-Zwischenablage an die ausgewählte Position im Textfeld Textdarstellung.

</dd> </dl>

:::image type="icon" source="images/f17chardel.gif":::

<dl> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Lösch**
</dt> <dd>

Entfernt den markierten Text aus dem Sound-Editor.

</dd> </dl>

:::image type="icon" source="images/f18charundo.gif":::

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Machen**
</dt> <dd>

Entfernt eine im Sound-Editor vorgenommene Änderung.

</dd> </dl>

:::image type="icon" source="images/f19charredo.gif":::

<dl> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Holen**
</dt> <dd>

Kehrt eine Rückgängig-Aktion im Sound-Editor um.

</dd> </dl>

:::image type="icon" source="images/flistlinguist.gif":::

<dl> <dt>

<span id="Generate_Linguistic_Info"></span><span id="generate_linguistic_info"></span><span id="GENERATE_LINGUISTIC_INFO"></span>**Linguistische Informationen generieren**
</dt> <dd>

Generiert Phonem-und Word-Bezeichnungen für die Audiodatei.

</dd> </dl>

:::image type="icon" source="images/flistpause.gif":::

<dl> <dt>

<span id="Pause"></span><span id="pause"></span><span id="PAUSE"></span>**Brechung**
</dt> <dd>

Hält das Abspielen der Audiodatei an.

</dd> </dl>

:::image type="icon" source="images/flistplay.gif":::

<dl> <dt>

<span id="Play_"></span><span id="play_"></span><span id="PLAY_"></span>**Theater** 
</dt> <dd>

Gibt die Audiodatei oder den ausgewählten Teil der Audiodatei wieder.

</dd> </dl>

:::image type="icon" source="images/fliststop.gif":::

<dl> <dt>

<span id="Stop_"></span><span id="stop_"></span><span id="STOP_"></span>**Anzuhalten** 
</dt> <dd>

Beendet die Aufzeichnung oder Wiedergabe der Audiodatei oder des ausgewählten Teils der Audiodatei.

</dd> </dl>

:::image type="icon" source="images/flistrecord.gif":::

<dl> <dt>

<span id="Record"></span><span id="record"></span><span id="RECORD"></span>**Aufnahme**
</dt> <dd>

Startet die Aufzeichnung einer Audiodatei.

</dd> </dl>
