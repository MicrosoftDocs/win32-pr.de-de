# <a name="speech-engine-support-requirements"></a>Sprach-Engine-Support Anforderungen

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Der Microsoft-Agent verwendet die Spracheingabe (Spracherkennung, SR) und Sprachausgabe (Text-zu-Sprache oder TTS) mithilfe der Microsoft Speech Application Programming Interface (SAPI). Durch die Unterstützung dieses Standards können die Sprachdienste von Microsoft Agent von anderen Sprachmodulen unterstützt werden. Dieses Dokument beschreibt die erforderlichen SAPI-Schnittstellen, die vom Microsoft-Agent verwendet werden. Weitere Informationen zu SAPI finden Sie auf der Website der [Microsoft-sprach](https://msdn.microsoft.com/library/ee705648.aspx) Gruppe.

-   [Anforderungen für Text-zu-Sprache-Engines](requirements-for-text-to-speech-engines.md)
-   [Anforderungen für Spracherkennungs-Engines](requirements-for-speech-recognition-engines.md)
-   [Anforderungen für die Unterstützung des Microsoft-Tools für linguistische Informationen Sound Bearbeitung](requirements-for-supporting-the-microsoft-linguistic-information-sound-editing-tool.md)
