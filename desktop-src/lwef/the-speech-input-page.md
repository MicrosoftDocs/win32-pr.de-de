# <a name="the-speech-input-page"></a>Die Spracheingabe Seite

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Ein Benutzer kann die Optionen für die Spracheingabe auf dieser Eigenschaften Seite anpassen. Der Benutzer kann die Spracheingabe deaktivieren, den abzurufenden Eingabe Schlüssel festlegen, auswählen, ob das Überwachungsfenster angezeigt werden soll, und einen MIDI-Ton abspielen, um anzugeben, wann die Spracheingabe verfügbar ist.
