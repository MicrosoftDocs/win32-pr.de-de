# <a name="left-property-propertysheet-object"></a>Left-Eigenschaft (PropertySheet-Objekt)

\[Microsoft Agent ist ab Version Windows 7 veraltet und in nachfolgenden Versionen von Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt einen ganzzahligen Wert zurück, der den aktuellen linken Rand des Eigenschaftenblattfensters von Microsoft Agent in Pixel angibt.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*agent**. PropertySheet.Left**

</dd> </dl>

## <a name="remarks"></a>Hinweise

Der Server zeigt das Fenster basierend auf dem vom Benutzer festgelegten Speicherort an.
