# <a name="name-property-agent-control"></a>Name-Eigenschaft (agentsteuerelement)

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt den Namen zurück, der im Code zum Identifizieren des Steuer Elements verwendet wird. Diese Eigenschaft ist zur Laufzeit schreibgeschützt.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Agent * * *. Benennen**

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

In einigen Programmierumgebungen wie Visual Basic wird durch das Hinzufügen des Steuer Elements automatisch ein Standardname für das Steuerelement generiert, das zur Entwurfszeit geändert werden kann. Für HTML-Skripts können Sie den Namen im- <OBJECT> Tag definieren. Wenn Sie den Namen definieren, befolgen Sie die Konventionen der Programmiersprache, um Objektnamen zu definieren.
