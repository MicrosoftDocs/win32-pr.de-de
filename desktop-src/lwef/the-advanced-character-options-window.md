# <a name="advanced-character-options-window-microsoft-agent-user-interface"></a>Fenster "Erweiterte Zeichenoptionen" (Microsoft-Agent-Benutzeroberfläche)

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Der Microsoft-Agent verwaltet bestimmte globale Einstellungen, die es einem Benutzer ermöglichen, die Interaktion mit allen Zeichen zu steuern. Im Fenster Erweiterte Zeichenoptionen werden diese Optionen und ihre aktuellen Einstellungen angezeigt, und sie können von jeder Hostinganwendung über die Programmierschnittstelle des Microsoft-Agents angezeigt werden. Der Microsoft-Agent speichert die zuletzt angezeigte Seite und zeigt diese Seite an, wenn das Eigenschaftenblatt angezeigt wird.

-   [Seite "Spracheingabe"](the-speech-input-page.md)
-   [Seite "Copyright"](the-copyright-page.md)
