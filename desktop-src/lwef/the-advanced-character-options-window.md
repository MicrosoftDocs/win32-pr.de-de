# <a name="advanced-character-options-window-microsoft-agent-user-interface"></a>Fenster "erweiterte Zeichen Optionen" (Benutzeroberfläche des Microsoft-Agents)

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Der Microsoft-Agent behält bestimmte globale Einstellungen bei, mit denen ein Benutzer die Interaktion mit allen Zeichen steuern kann. Das Fenster Erweiterte Zeichen Optionen zeigt diese Optionen und ihre aktuellen Einstellungen an und kann von jeder Host Anwendung mithilfe der Microsoft-Agent-Programmierschnittstelle angezeigt werden. Der Microsoft-Agent speichert die zuletzt angezeigte Seite und zeigt diese Seite an, wenn das Eigenschaften Blatt angezeigt wird.

-   [Die Seite "Ausgabe"](the-output-page.md)
-   [Die Spracheingabe Seite](the-speech-input-page.md)
-   [Die Seite "Copyright"](the-copyright-page.md)
