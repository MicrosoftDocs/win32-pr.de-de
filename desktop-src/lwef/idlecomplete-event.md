# <a name="idlecomplete-event"></a>Idlecomplete-Ereignis

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Tritt auf, wenn der Server den **Leerlauf** Zustand eines Zeichens beendet.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

**Sub** - *Agent * * * \_ idlecomplete* *  **(ByVal** -Merkmal- *ID * * *)**



| Teil          | BESCHREIBUNG                                         |
|---------------|-----------------------------------------------------|
| *Merkmal-ID* | Gibt die ID des idult-Zeichens als Zeichenfolge zurück. |



 

</dd> </dl>

### <a name="remarks"></a>Bemerkungen

Der Server sendet dieses Ereignis an alle Clients des Zeichens.
