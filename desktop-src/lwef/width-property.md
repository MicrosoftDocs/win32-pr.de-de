# <a name="width-property-commandswindow-object"></a>Width-Eigenschaft (CommandsWindow-Objekt)

\[Microsoft Agent ist ab Version Windows 7 veraltet und in nachfolgenden Versionen von Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt einen ganzzahligen Wert zurück, der die Breite des Sprachbefehlsfensters in Pixel angibt.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Syntax** 
</dt> <dd>

*agent**. CommandsWindow.Width**

</dd> </dl>

### <a name="remarks"></a>Hinweise

Der Server zeigt das Fenster für Sprachbefehle basierend auf der vom Benutzer festgelegten Position und Größe an.
