# <a name="width-property-commandswindow-object"></a>Width-Eigenschaft (commandswindow-Objekt)

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt einen ganzzahligen Wert zurück, der die Breite des sprach Befehls Fensters in Pixel angibt.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Syntax** 
</dt> <dd>

*Agent * * *. Commandswindow. Width**

</dd> </dl>

### <a name="remarks"></a>Bemerkungen

Der Server zeigt das Fenster "Sprachbefehle" basierend auf der Position und Größe an, die vom Benutzer festgelegt wurden.
