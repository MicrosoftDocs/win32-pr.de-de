# <a name="soundeffectson-property"></a>SoundEffectsOn-Eigenschaft

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt zurück oder legt fest, ob Soundeffekte für Ihr Zeichen aktiviert sind.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Agent .* **Characters("**_CharacterID_*_"). SoundEffectsOn_* \[  =  *boolean*\]



| Teil      | Beschreibung                                                                                                                                                        |
|-----------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *boolean* | Ein boolescher Ausdruck, der angibt, ob Soundeffekte aktiviert sind. **True** Soundeffekte sind aktiviert.<br/> **False** Soundeffekte sind deaktiviert.<br/> |



 

</dd> </dl>

## <a name="remarks"></a>Hinweise

Diese Eigenschaft bestimmt, ob Soundeffekte, die als Teil der Animationen eines Zeichens enthalten sind, wiedergegeben werden, wenn eine Animation wiedergegeben wird. Die Einstellung dieser Eigenschaft gilt für alle Clients des Zeichens.
