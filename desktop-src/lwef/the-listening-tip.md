# <a name="the-listening-tip"></a>Der Lauschentipp

\[Microsoft Agent ist ab Version Windows 7 veraltet und möglicherweise in nachfolgenden Versionen von Windows.\]

Der Lauschentipp ist ein weiterer Spracheingabedienst, der von Microsoft Agent bereitgestellt wird. Wenn die Spracheingabe installiert ist, enthält der -Agent ein spezielles QuickInfo-Fenster, das angezeigt wird, wenn der Benutzer den Hotkey Lauschen drückt oder die Listen-Methode aufruft. Der Tipp zum Lauschen wird nur angezeigt, wenn die Sprachdienste verfügbar sind. Wenn kein Client einen Sprachbefehl erstellt hat oder erfolgreich eine Sprach-Engine lädt, wird der Tipp zum Lauschen nicht angezeigt. Darüber hinaus müssen sowohl die Spracheingabe als auch die Option Tipps anzeigen in den erweiterten Zeichenoptionen aktiviert sein, damit der Tipp angezeigt wird.

Beim Tipp zum Lauschen wird nach derEntität automatisch ein Zeitsende ausgeführt. Wenn das Time out für den Text "Heard" abgeschlossen ist, während der Benutzer den Hotkey immer noch hält, wird der Tipp auf den lauschenden Text zurückverwendet, es sei denn, der Server erhält eine weitere übereinstimmende Äußerung. In diesem Fall zeigt der Tipp den neuen Text "Heard" an und beginnt mit dem Time out für diesen Trinkgeldtext. Wenn der Benutzer den Hotkey frei gibt und der Server den Text "Heard" zeigt, wird das Time out fortgesetzt, und das Fenster Lauschende Tipp wird ausgeblendet, wenn das Time out-Intervall abgeht.

Wenn der Server noch nicht versucht hat, eine Spracherkennungs-Engine zu laden, wird der Tipp zum Lauschen nicht angezeigt. Wenn der Benutzer die Anzeige des Lauschenden Tipps oder die Spracheingabe in erweiterten Zeichenoptionen deaktiviert hat, wird der Tipp zum Lauschen ebenfalls nicht angezeigt.

Der Lauschende Tipp wird nicht angezeigt, wenn sich der Zeiger über dem Taskleistensymbol des Zeichens befindet. Stattdessen wird das Standardfenster für Benachrichtigungstipps angezeigt, in dem der Name des Zeichens angezeigt wird.

Clientanwendungen können nicht direkt in den Lauschen-Tipp schreiben, aber Sie können alternativen Text angeben, der vom Server bei der Erkennung eines übereinstimmenden Sprachbefehls angezeigt wird. Legen Sie hierzu die [**Confidence-Eigenschaft**](confidence-property.md) und die neue [**ConfidenceText-Eigenschaft**](confidencetext-property.md)für den Befehl fest. Wenn gesprochene Eingaben mit dem Befehl übereinstimmen, aber die beste Übereinstimmung die Vertrauenseinstellung nicht überschreitet, verwendet der Server den Text, der in der **ConfidenceText-Eigenschaft** im Trinkgeldfenster festgelegt ist. Wenn der Client diesen Wert nicht anfing, zeigt der Server den Übereinstimmenden Text (Grammatik) an.

Der Text "Lauschende Tipps" wird in der Sprache basierend auf der Einstellung für die Zeichensprachen-ID des Eingabeaktiven Clients angezeigt, unabhängig davon, ob eine sprachkompatible Spracherkennungs-Engine verfügbar ist.
