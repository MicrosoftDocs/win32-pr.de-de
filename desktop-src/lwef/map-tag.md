# <a name="map-tag"></a>Map-Tag

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Ordnet gesprochenen Text Text zu, der in der Word-Sprechblase angezeigt wird.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

\\**Map**= "SpokenText" = "balloontext"\\



| Teil          | BESCHREIBUNG                                           |
|---------------|-------------------------------------------------------|
| *SpokenText*  | Eine Zeichenfolge, die den Text für die gesprochene Ausgabe angibt.       |
| *balloontext* | Eine Zeichenfolge, die den Text für die Ausgabe der Wort Blase angibt. |



 

</dd> </dl>

### <a name="remarks"></a>Bemerkungen

Dieses Tag ermöglicht es Ihnen, anderen gesprochenen Text als den in der Word-Sprechblase angezeigten Text zu verwenden.
