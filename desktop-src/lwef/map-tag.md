# <a name="map-tag"></a>Kartentag

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Karten gesprochenen Text in Text ein, der im Wortsprechblasen angezeigt wird.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

\\**Map**="spokentext"="balloontext"\\



| Teil          | BESCHREIBUNG                                           |
|---------------|-------------------------------------------------------|
| *Spokentext*  | Eine Zeichenfolge, die den Text für die gesprochene Ausgabe angibt.       |
| *balloontext* | Eine Zeichenfolge, die den Text für die Ausgabe der Wortsprechblase angibt. |



 

</dd> </dl>

### <a name="remarks"></a>Hinweise

Mit diesem Tag können Sie einen anderen gesprochenen Text als den im Wortsprechblasen angezeigten Text verwenden.
