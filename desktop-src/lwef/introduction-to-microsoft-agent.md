# <a name="introduction-to-microsoft-agent"></a>Einführung in den Microsoft-Agent

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Bei Microsoft Agent handelt es sich um eine Reihe von programmierbaren Software Diensten, die die Darstellung von interaktiven animierten Zeichen unterstützen. Entwickler können Zeichen als interaktive Assistenten verwenden, um Ihre Webseiten oder Anwendungen zusätzlich zur herkömmlichen Verwendung von Fenstern, Menüs und Steuerelementen einzuführen, zu unterhalten oder anderweitig zu verbessern.

Mit dem Microsoft-Agent können Softwareentwickler und Webautoren eine neue Form von Benutzerinteraktion integrieren, die als "konvertigende Schnittstellen" bezeichnet wird, die natürliche Aspekte der menschlichen sozialen Kommunikation nutzt. Neben der Maus-und Tastatureingabe umfasst der Microsoft-Agent optionale Unterstützung für die Spracherkennung, damit Anwendungen auf Sprachbefehle reagieren können. Zeichen können mithilfe von synthetischer Sprache, aufgezeichneter Audiodaten oder Text in einer Zeichen Sprechblase mit Zeichenfolge Antworten.

Der Ansatz der von den Microsoft-Agent-Diensten unterstützten Herangehensweise an die konverational Interface ersetzt den herkömmlichen grafischen Benutzeroberflächen Entwurf nicht. Stattdessen kann die Zeichen Interaktion problemlos mit den konventionellen Schnittstellen Komponenten, z. b. Fenstern, Menüs und Steuerelementen, kombiniert werden, um die Benutzeroberfläche der Anwendung zu erweitern und zu verbessern.

Die Programmierschnittstellen von Microsoft Agent erleichtern das Animieren eines Zeichens, um auf Benutzereingaben zu reagieren. Animierte Zeichen werden in einem eigenen Fenster angezeigt und bieten so maximale Flexibilität, wo Sie auf dem Bildschirm angezeigt werden können. Der Microsoft-Agent enthält ein ActiveX-Steuerelement, das die Verfügbarkeit der Dienste für Programmiersprachen ermöglicht, die ActiveX unterstützen, einschließlich Webskript Sprachen wie Visual Basic Scripting Edition (VBScript). Dies bedeutet, dass die Zeichen Interaktion auch aus HTML-Seiten mit dem-Tag programmiert werden kann <OBJECT> .
