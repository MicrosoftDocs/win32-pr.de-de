# <a name="introduction-to-microsoft-agent"></a>Einführung in Microsoft Agent

\[Microsoft Agent ist ab Version Windows 7 veraltet und in nachfolgenden Versionen von Windows.\]

Microsoft Agent ist eine Reihe von programmierbaren Softwarediensten, die die Präsentation interaktiver animierter Figuren unterstützen. Entwickler können Zeichen als interaktive Assistenten verwenden, um ihre Webseiten oder Anwendungen zusätzlich zur herkömmlichen Verwendung von Fenstern, Menüs und Steuerelementen einzuführen, zu steuern, zu beplanen oder anderweitig zu verbessern.

Mit Microsoft Agent können Softwareentwickler und Webautoren eine neue Form der Benutzerinteraktion integrieren, die als Konversationsschnittstellen bezeichnet wird und natürliche Aspekte der menschlichen sozialen Kommunikation nutzt. Zusätzlich zur Maus- und Tastatureingabe bietet Microsoft Agent optionale Unterstützung für die Spracherkennung, damit Anwendungen auf Sprachbefehle reagieren können. Zeichen können mit synthetisierter Sprache, aufgezeichneten Audiodaten oder Text in einem Sprechblasenwort antworten.

Der von den Microsoft Agent-Diensten ermöglichte Ansatz der Konversationsschnittstelle ersetzt nicht den herkömmlichen GUI-Entwurf (Grafische Benutzeroberfläche). Stattdessen kann die Zeicheninteraktion problemlos mit den herkömmlichen Schnittstellenkomponenten wie Fenstern, Menüs und Steuerelementen kombiniert werden, um die Oberfläche Ihrer Anwendung zu erweitern und zu verbessern.

Die Programmierschnittstellen von Microsoft Agent machen es einfach, ein Zeichen zu animieren, um auf Benutzereingaben zu reagieren. Animierte Figuren werden in ihrem eigenen Fenster angezeigt und bieten maximale Flexibilität, wo sie auf dem Bildschirm angezeigt werden können. Microsoft Agent enthält ein ActiveX-Steuerelement, das seine Dienste für Programmiersprachen zugänglich macht, die ActiveX unterstützen, einschließlich Webskriptsprachen wie Visual Basic Scripting Edition (VBScript). Dies bedeutet, dass zeicheninteraktion auch von HTML-Seiten mithilfe des &lt; OBJECT-Tags programmiert werden &gt; kann.
