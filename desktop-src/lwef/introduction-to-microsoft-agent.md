# <a name="introduction-to-microsoft-agent"></a>Einführung in den Microsoft-Agent

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Microsoft Agent ist eine Reihe von programmierbaren Softwarediensten, die die Darstellung interaktiver animierter Zeichen unterstützen. Entwickler können Zeichen als interaktive Assistenten verwenden, um ihre Webseiten oder Anwendungen zusätzlich zur herkömmlichen Verwendung von Fenstern, Menüs und Steuerelementen einzuführen, zu leiten, zu leiten oder anderweitig zu verbessern.

Microsoft Agent ermöglicht Softwareentwicklern und Webautoren, eine neue Form der Benutzerinteraktion zu integrieren, die als Konversationsschnittstellen bezeichnet wird und natürliche Aspekte der menschlichen sozialen Kommunikation nutzt. Zusätzlich zur Maus- und Tastatureingabe bietet Der Microsoft-Agent optionale Unterstützung für die Spracherkennung, damit Anwendungen auf Sprachbefehle reagieren können. Zeichen können mithilfe von synthetisierter Sprache, aufgezeichneten Audiodaten oder Text in einem Sprechblasentext reagieren.

Der von den Microsoft-Agent-Diensten angebotene Ansatz der Konversationsschnittstelle ersetzt nicht den herkömmlichen GUI-Entwurf (Graphical User Interface). Stattdessen kann die Zeicheninteraktion problemlos mit den herkömmlichen Schnittstellenkomponenten wie Fenstern, Menüs und Steuerelementen kombiniert werden, um die Schnittstelle Ihrer Anwendung zu erweitern und zu verbessern.

Die Programmierschnittstellen des Microsoft-Agents erleichtern das Animieren eines Zeichens, um auf Benutzereingaben zu reagieren. Animierte Zeichen werden in ihrem eigenen Fenster angezeigt und bieten maximale Flexibilität, wo sie auf dem Bildschirm angezeigt werden können. Der Microsoft-Agent enthält ein ActiveX Steuerelement, das die Dienste für Programmiersprachen zugänglich macht, die ActiveX unterstützen, einschließlich Webskriptsprachen wie Visual Basic Scripting Edition (VBScript). Dies bedeutet, dass die Zeicheninteraktion auch von HTML-Seiten mit dem -Tag programmiert werden <OBJECT> kann.
