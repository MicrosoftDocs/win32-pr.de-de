# <a name="version-property"></a>Version-Eigenschaft

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt die Version des-Zeichens zurück.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Büros. ***Zeichen ("*** Merkmal-ID * * *"). Version**

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

Die **Version** -Eigenschaft gibt eine Zeichenfolge zurück, die der Version der standardmäßigen Animations Satz Definition entspricht, für die das Zeichen kompiliert wurde. Die Versionsnummer des Zeichens wird automatisch festgelegt, wenn Sie Sie mit dem Microsoft-Agent-Zeichen-Editor erstellen.
