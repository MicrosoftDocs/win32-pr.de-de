# <a name="the-voice-commands-window"></a>Das Fenster "Sprachbefehle"

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Wenn eine kompatible Sprach-Engine installiert ist, stellt der Microsoft-Agent ein spezielles Fenster mit dem Namen " **Sprachbefehle** " bereit, in dem die Befehle angezeigt werden, die für die Spracherkennung sprach fähig waren. Das **Fenster "Sprachbefehle** " dient als visuelle Eingabeaufforderung für das, was als Eingabe gesprochen werden kann (Befehle können nicht mit der Maus ausgewählt werden).

![Sprachbefehle (Dialogfeld)](images/f2voice.gif)

Das Fenster wird angezeigt, wenn ein Benutzer den Befehl **Sprachbefehle öffnen** auswählt, indem er entweder den Befehl anspricht oder mit der rechten Maustaste auf das Zeichen klickt und im Popup Menü des Zeichens den Befehl auswählt. Wenn der Benutzer jedoch die Spracheingabe deaktiviert, kann nicht auf das Fenster für die **Sprachbefehle** zugegriffen werden.

Das **Fenster "Sprachbefehle** " zeigt sprach aktivierte Befehle als Baum an. Wenn die aktuelle Hostinganwendung Sprachbefehle liefert, werden Sie erweitert und am oberen Rand des Fensters angezeigt. Einträge werden auch für andere Anwendungen angezeigt, die das-Zeichen verwenden. Das Fenster enthält auch die globalen Sprachbefehle, die vom Microsoft-Agent bereitgestellt werden. Wenn die aktuelle Hostinganwendung über keine Sprachbefehle verfügt, werden die globalen Sprachbefehle erweitert und am oberen Rand des Fensters angezeigt.

Der Benutzer kann das **Fenster "Sprachbefehle**" anpassen und verschieben. Der Microsoft-Agent speichert den letzten Speicherort des Fensters und zeigt ihn an diesem Speicherort erneut an, wenn der Benutzer das Fenster schließt und erneut öffnet. Wenn die Einträge im Fenster die aktuelle Anzeige Größe des Fensters überschreiten, werden Scrollleisten angezeigt.
