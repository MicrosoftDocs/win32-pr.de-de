# <a name="the-voice-commands-window"></a><span data-ttu-id="99ac6-103">Das Fenster "Sprachbefehle"</span><span class="sxs-lookup"><span data-stu-id="99ac6-103">The Voice Commands Window</span></span>

<span data-ttu-id="99ac6-104">\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]</span><span class="sxs-lookup"><span data-stu-id="99ac6-104">\[Microsoft Agent is deprecated as of Windows 7, and may be unavailable in subsequent versions of Windows.\]</span></span>

<span data-ttu-id="99ac6-105">Wenn eine kompatible Sprach-Engine installiert ist, stellt der Microsoft-Agent ein spezielles Fenster mit dem Namen " **Sprachbefehle** " bereit, in dem die Befehle angezeigt werden, die für die Spracherkennung sprach fähig waren.</span><span class="sxs-lookup"><span data-stu-id="99ac6-105">If a compatible speech engine is installed, Microsoft Agent supplies a special window called the **Voice Commands Window** that displays the commands that have been voice-enabled for speech recognition.</span></span> <span data-ttu-id="99ac6-106">Das **Fenster "Sprachbefehle** " dient als visuelle Eingabeaufforderung für das, was als Eingabe gesprochen werden kann (Befehle können nicht mit der Maus ausgewählt werden).</span><span class="sxs-lookup"><span data-stu-id="99ac6-106">The **Voice Commands Window** serves as a visual prompt for what can be spoken as input (commands cannot be selected with the mouse).</span></span>

![Sprachbefehle (Dialogfeld)](images/f2voice.gif)

<span data-ttu-id="99ac6-108">Das Fenster wird angezeigt, wenn ein Benutzer den Befehl **Sprachbefehle öffnen** auswählt, indem er entweder den Befehl anspricht oder mit der rechten Maustaste auf das Zeichen klickt und im Popup Menü des Zeichens den Befehl auswählt.</span><span class="sxs-lookup"><span data-stu-id="99ac6-108">The window appears when a user selects the **Open Voice Commands Window** command, either by speaking the command or right-clicking the character and choosing the command from the character's pop-up menu.</span></span> <span data-ttu-id="99ac6-109">Wenn der Benutzer jedoch die Spracheingabe deaktiviert, kann nicht auf das Fenster für die **Sprachbefehle** zugegriffen werden.</span><span class="sxs-lookup"><span data-stu-id="99ac6-109">However, if the user disables speech input, the **Voice Commands Window** is not accessible.</span></span>

<span data-ttu-id="99ac6-110">Das **Fenster "Sprachbefehle** " zeigt sprach aktivierte Befehle als Baum an.</span><span class="sxs-lookup"><span data-stu-id="99ac6-110">The **Voice Commands Window** displays voice-enabled commands as a tree.</span></span> <span data-ttu-id="99ac6-111">Wenn die aktuelle Hostinganwendung Sprachbefehle liefert, werden Sie erweitert und am oberen Rand des Fensters angezeigt.</span><span class="sxs-lookup"><span data-stu-id="99ac6-111">If the current hosting application supplies voice commands, they appear expanded and at the top of the window.</span></span> <span data-ttu-id="99ac6-112">Einträge werden auch für andere Anwendungen angezeigt, die das-Zeichen verwenden.</span><span class="sxs-lookup"><span data-stu-id="99ac6-112">Entries also appear for other applications using the character.</span></span> <span data-ttu-id="99ac6-113">Das Fenster enthält auch die globalen Sprachbefehle, die vom Microsoft-Agent bereitgestellt werden.</span><span class="sxs-lookup"><span data-stu-id="99ac6-113">The window also includes the global voice commands supplied by Microsoft Agent.</span></span> <span data-ttu-id="99ac6-114">Wenn die aktuelle Hostinganwendung über keine Sprachbefehle verfügt, werden die globalen Sprachbefehle erweitert und am oberen Rand des Fensters angezeigt.</span><span class="sxs-lookup"><span data-stu-id="99ac6-114">If the current hosting application has no voice commands, the global voice commands appear expanded and at the top of the window.</span></span>

<span data-ttu-id="99ac6-115">Der Benutzer kann das **Fenster "Sprachbefehle**" anpassen und verschieben.</span><span class="sxs-lookup"><span data-stu-id="99ac6-115">The user can size and move the **Voice Commands Window**.</span></span> <span data-ttu-id="99ac6-116">Der Microsoft-Agent speichert den letzten Speicherort des Fensters und zeigt ihn an diesem Speicherort erneut an, wenn der Benutzer das Fenster schließt und erneut öffnet.</span><span class="sxs-lookup"><span data-stu-id="99ac6-116">Microsoft Agent remembers the last location of the window and redisplays it at that location if the user closes and re-opens the window.</span></span> <span data-ttu-id="99ac6-117">Wenn die Einträge im Fenster die aktuelle Anzeige Größe des Fensters überschreiten, werden Scrollleisten angezeigt.</span><span class="sxs-lookup"><span data-stu-id="99ac6-117">If the entries in the window exceed the current display size of the window, scroll bars appear.</span></span>
