# <a name="characters"></a>Zeichen

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Die menschliche Kommunikation ist grundsätzlich soziale Netzwerke. Mit dem Microsoft-Agent können Sie diesen Aspekt der Interaktion mit animierten Zeichen nutzen. Benutzer erwarten, dass ein Zeichen dem gleichen sozialen, aber nicht notwendigerweise physischen, bei der Interaktion mit anderen Personen verwendeten Zeichen entspricht, selbst wenn Sie wissen, dass das Zeichen synthetisch ist. In dem Umfang, in dem Sie Zeichen erstellen, die Ihren Erwartungen entsprechen, können Benutzer die Zeichen leichter und besser ansehen. Daher kann die Art und Weise, wie Sie ein Zeichen entwerfen, eine drastische Auswirkung auf seinen Erfolg haben.

Wenn Sie ein Zeichen entwerfen, sollten Sie zunächst das Profil Ihrer Zielgruppe und die darin begriffenden Aktionen sowie die Aufgaben, die Sie ausführen, überprüfen. Beachten Sie auch, wie gut der Entwurf und Stil Ihres Zeichens zusätzlich zu der von ihm unterstützten Anwendung übereinstimmt. Beispielsweise kann ein hundezeichen in Abhängigkeit von der Gesamtdarstellung für eine Abruf-oder Sicherheitsanwendung gut funktionieren. Häufig wird der Erfolg in den Details erläutert. Research hat gezeigt, dass das Ändern der Flächen haftigkeit eines Tieres und der Form des ragens sehr unterschiedliche Reaktionen auf das Zeichen generieren kann.

Beachten Sie auch den grundlegenden Typ der Zeichenfolge: "dominant" oder "submissive", "emotional" oder "reserviert", "hoch" oder vielleicht möchten Sie Ihre Persönlichkeit auf der Grundlage der Benutzerinteraktion anpassen. Beispielsweise können Sie ein Steuerelement bereitstellen, das es einem Benutzer ermöglicht, festzulegen, ob das Zeichen mehr Informationen zur Verfügung stellt oder auf die Frage wartet. Die erste wäre eher ausgehender als letztere.

Der Name, den Sie für Ihr Zeichen angeben, kann einen bestimmten Typ von Persönlichkeit ableiten. Beispielsweise können "Max" und "Linus" sehr unterschiedliche Personen vermitteln. Mit dem Zeichen-Editor von Microsoft-Agent können Sie den Namen Ihres Zeichens festlegen und eine kurze Beschreibung einschließen. Diese Attribute können zur Laufzeit abgefragt werden.

Außerdem sollten Sie entscheiden, ob Sie eine synthetische Stimme (mit einer Text-zu-Sprache-Engine) oder eine aufgezeichnete Stimme (. WAV-Datei). Diese Entscheidung hängt möglicherweise von der Art des verwendeten Zeichens, von den Sprachen ab, die Sie unterstützen möchten, und von dem, was das Zeichen sagen soll. Beispielsweise kann ein Zeichen mit einer synthetisierten Stimme fast alles sagen. Das Programmieren Ihrer Zeichen ist einfach und schnell: Sie geben einfach den Text an, der von dem Zeichen gesprochen wird. Allerdings erfordert die Verwendung einer computergenerierten Sprach-Engine für die Erstinstallation einen zusätzlichen Aufwand und ist sprachspezifisch. Darüber hinaus werden die meisten mit der Groß-und groß-und mehrwertigen Stimmen Sie entsprechen nicht der Klarheit und prosody der meisten menschlichen Spracheingaben. Es ist möglicherweise schwierig, eine Stimme zu simulieren, die mit Ihrem Zeichen übereinstimmt. Dies gilt insbesondere, wenn Sie ein Zeichen verwenden, das bereits über eine festgelegte Identität oder eine mit sehr unterschiedliche Stimme verfügt. In einem solchen Fall möchten Sie möglicherweise aufgezeichnete Sprachdateien für Ihre Ausgabe verwenden. Der Microsoft-Agent unterstützt auch die Lip-Synchronisierung für die aufgezeichnete Sprachausgabe. Obwohl Audiodateien eine natürliche Stimme darstellen und leichter in anderen Sprachen implementiert werden können, müssen Sie auf lokale Computer kopiert oder heruntergeladen werden. Aufgezeichnete Sprachdateien beschränken auch das Zeichen auf das darin enthaltene Vokabular. Egal, ob Sie synthetische oder aufgezeichnete Sprachausgabe auswählen, denken Sie daran, dass eine Stimme zusätzliche soziale Informationen über das Geschlecht, das Alter und die Persönlichkeit des Referenten enthält.

Sie können auch die Wort Sprechblase für die Ausgabe und die Standardeinstellungen für Schriftart und Farbe der Sprechblase verwenden. Beachten Sie jedoch, dass der Benutzer die Schriftart-und Farb Attribute ändern kann. Außerdem können Sie nicht davon ausgehen, dass der Zustand der Word-Sprechblase konstant bleibt, da der Benutzer die Word-Sprechblase deaktivieren kann.
