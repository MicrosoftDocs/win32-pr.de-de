# <a name="characters"></a>Zeichen

\[Microsoft Agent ist ab Version Windows 7 veraltet und in nachfolgenden Versionen von Windows.\]

Die kommunikation zwischen Menschen ist im Wesentlichen ein soziales Netzwerk. Microsoft Agent ermöglicht es Ihnen, diesen Aspekt der Interaktion mit animierten Figuren zu nutzen. Benutzer erwarten, dass ein Zeichen denselben sozialen, aber nicht notwendigerweise physischen Regeln entspricht, die sie bei der Interaktion mit anderen Personen verwenden, auch wenn sie verstehen, dass das Zeichen synthetisch ist. In dem Umfang, in dem Sie Zeichen erstellen, die ihren Erwartungen entsprechen, finden Benutzer Ihre Zeichen verständlicher und likabler. Daher kann die Art und Weise, wie Sie ein Zeichen entwerfen, einen drastischen Einfluss auf den Erfolg haben.

Berücksichtigen Sie beim Entwerfen eines Zeichens zunächst das Profil Ihrer Zielgruppe, was für sie von Bewertern ist und welche Aufgaben sie ausführen. Berücksichtigen Sie auf ähnliche Weise, wie gut das Design und der Stil Ihres Zeichens neben der unterstützten Anwendung dem Zweck des Zeichens entspricht. Beispielsweise kann ein Hundezeichen je nach Aussehen gut für einen Abruf oder eine Sicherheitsanwendung funktionieren. Häufig liegt der Erfolg in den Details. Untersuchungen haben gezeigt, dass das Ändern der Augen- und Augenform eines Tierzeichens zu sehr unterschiedlichen Reaktionen auf das Zeichen führen kann.

Berücksichtigen Sie auch den grundlegenden Persönlichkeitstyp Ihrer Figur: dominant oder unterlegt, emotional oder reserviert, ausgefeilt oder bodenständig; oder vielleicht möchten Sie seine Persönlichkeit basierend auf der Benutzerinteraktion anpassen. Beispielsweise können Sie ein Steuerelement bereitstellen, mit dem ein Benutzer anpassen kann, ob das Zeichen weitere Informationen enthält oder auf die Frage wartet. Ersteres wäre ausgehender als letzteres.

Der Name, den Sie für Ihr Zeichen angeben, kann einen bestimmten Persönlichkeitstyp abgeleitet werden. Beispielsweise können "Max" und "Linus" sehr unterschiedliche Persönlichkeiten vermitteln. Mit dem Zeichen-Editor von Microsoft Agent können Sie den Namen Ihres Zeichens festlegen und eine kurze Beschreibung angeben. Diese Attribute können zur Laufzeit abgefragt werden.

Entscheiden Sie außerdem, ob Sie eine synthetische Stimme (mit einer Text-zu-Sprache-Engine) oder eine aufgezeichnete Stimme () verwenden möchten. WAV-Datei). Diese Entscheidung hängt möglicherweise vom Typ des zeichens ab, den Sie verwenden, den Sprachen, die Sie unterstützen möchten, und dem, was das Zeichen sagen kann. Eine synthetisierte Stimme ermöglicht es Ihrem Zeichen beispielsweise, fast alles zu sagen. Das Programmieren des Zeichens ist einfach und schnell: Sie geben einfach den Text an, den das Zeichen spricht. Die Verwendung einer vom Computer generierten Sprach-Engine erfordert jedoch zusätzlichen Mehraufwand für die Erstinstallation und ist sprachspezifisch. Darüber hinaus klingen die meisten synthetisierten Stimmen computergeneriert. Sie passen nicht zur Klarheit und Prosodie der meisten menschlichen Sprache. Es kann schwierig sein, eine Stimme zu simulieren, die Ihrem Zeichen entspricht, insbesondere, wenn Sie ein Zeichen verwenden, das bereits über eine identität verfügt, oder eine Stimme mit einer sehr unterschiedlichen Stimme. In einem solchen Fall können Sie aufgezeichnete Sprachdateien für Ihre Ausgabe verwenden. Microsoft Agent unterstützt auch die Sync-Synchronisierung für aufgezeichnete Sprachausgabe. Obwohl Audiodateien eine natürliche Stimme bieten und in anderen Sprachen einfacher zu implementieren sind, müssen sie kopiert oder auf lokale Computer heruntergeladen werden. Aufgezeichnete Sprachdateien beschränken ihr Zeichen auch auf das darin enthaltene Vokabular. Unabhängig davon, ob Sie synthetische oder aufgezeichnete Sprachausgabe auswählen, denken Sie daran, dass eine Stimme zusätzliche soziale Informationen zu Geschlecht, Alter und Persönlichkeit des Sprechers mit sich bringt.

Sie können auch das Wort Balloon für die Ausgabe und die Standardeinstellungen für die Schriftart und Farbe des Sprechblasens verwenden. Beachten Sie jedoch, dass der Benutzer die Schriftart- und Farbattribute ändern kann. Darüber hinaus können Sie nicht davon ausgehen, dass der Zustand des Wortsprechblasens konstant bleibt, da der Benutzer den Wortsprechblasen ausschalten kann.
