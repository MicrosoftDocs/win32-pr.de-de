# <a name="command-method"></a>Command-Methode

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt ein [**Befehls**](/windows/desktop/lwef/the-command-object) Objekt in einer [**Commands**](/windows/desktop/lwef/the-commands-collection-object) -Auflistung zurück.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Agent ***. Zeichen ("*** Merkmal ID * * *"). Commands. Command** "*Name*"

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

Mit dieser Methode können Sie auf die Eigenschaften eines [**Befehls**](/windows/desktop/lwef/the-command-object) Objekts zugreifen.

> [!Note]  
> Diese Methode ist möglicherweise für einige Programmiersprachen erforderlich. Dies ist für VBScript oder Visual Basic nicht erforderlich. Weitere Informationen zum Angeben von **Befehls** Methoden finden Sie unter Command Object Properties.
