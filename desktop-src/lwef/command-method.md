# <a name="command-method"></a>Command-Methode

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt ein [**Command-Objekt**](/windows/desktop/lwef/the-command-object) in einer [**Commands-Auflistung zurück.**](/windows/desktop/lwef/the-commands-collection-object)

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*agent***. Zeichen ("**_CharacterID_*_"). Commands.Command_* "*Name*"

</dd> </dl>

## <a name="remarks"></a>Hinweise

Sie können diese Methode verwenden, um auf die Eigenschaften eines [**Command-Objekts**](/windows/desktop/lwef/the-command-object) zuzugreifen.

> [!Note]  
> Diese Methode ist möglicherweise für einige Programmiersprachen erforderlich. Sie ist für VBScript oder Visual Basic nicht erforderlich. Weitere Informationen zum Angeben von **Befehlsmethoden** finden Sie unter Befehlsobjekteigenschaften.
