# <a name="the-office-animation-set"></a>Der Office Animationssatz

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Die folgende Tabelle enthält die Animationen, die für die Microsoft Office 2.000 Zeichen definiert sind. Wenn Sie ihr Zeichen in Microsoft Office verwenden möchten, sollten Sie alle Animationen in dieser Tabelle unterstützen. Darüber hinaus können Sie alle anderen Animationen hinzufügen, die Sie live verwenden, aber denken Sie daran, dass Microsoft Office sie nicht aufrufen. Animationen mit Sternchen \* () sollten zu 100 % schleifend sein. Andere Animationen sollten kurz sein.

|-------------------------|----------------|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| **Warnung**               | Keine           | Wenn das Zeichen den Benutzer warnen möchte                                                           | Zeichen sucht nach Benutzer.                                                                                                              |
| **CheckingSomething\*** | Keine           | Rechtschreibprüfung, Grammatikprüfung                                                                            | Zeichen sucht etwas in einem Referenzbuch                                                                                          |

| **EmptyTrash**          | Keine           | Papierkorb wird in Outlook geleert                                                                          | Zeichenlichter: Papierkorb in Brand                                                                                                        |

| **GestureDown**         | GesturingDown  | Zeichen zeigt auf etwas auf dem Bildschirm                                                         | Das Zeichen zeigt auf den Benutzer und zeigt dann auf den Bildschirm.                                                                           |
| **GestureLeft**         | GesturingLeft  | Zeichen zeigt etwas auf dem Bildschirm an, z. B. ein Hilfethema oder eine Benutzeroberfläche.                  | Das Zeichen zeigt auf den Benutzer und zeigt dann auf den Bildschirm.                                                                           |
