# <a name="the-office-animation-set"></a>Der Office-Animations Satz

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

In der folgenden Tabelle sind die für die Microsoft Office 2000-Zeichen definierten Animationen aufgeführt. Wenn Sie Ihr Zeichen in Microsoft Office verwenden möchten, sollten Sie alle Animationen in dieser Tabelle unterstützen. Außerdem können Sie alle anderen Animationen hinzufügen, die Sie leben, aber denken Sie daran, dass Microsoft Office Sie nicht aufruft. Animationen mit Sternchen () müssen \* 100% Schleifen aufweisen. Andere Animationen sollten kurz sein.
