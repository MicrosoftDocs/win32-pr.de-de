# <a name="animation-services"></a>Animations Dienste

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Die Animations Dienste von Microsoft Agent verwalten die Animation und Bewegung des Bilds eines Zeichens in einem eigenen Fenster auf dem Bildschirm. Eine Animation wird als Sequenz von zeitgesteuerten und optional verzweigten Frames definiert, bestehend aus mindestens einem Bild.

-   [Laden eines Zeichens](loading-a-character.md)
-   [Laden des Standard Zeichens](loading-the-default-character.md)
-   [Animieren eines Zeichens](animating-a-character.md)
