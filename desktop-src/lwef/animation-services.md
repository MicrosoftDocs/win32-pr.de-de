# <a name="animation-services"></a>Animationsdienste

\[Microsoft Agent ist ab Version Windows 7 veraltet und in nachfolgenden Versionen von Windows.\]

Die Animationsdienste von Microsoft Agent verwalten die Animation und Bewegung des Bilds eines Zeichens in einem eigenen Fenster auf dem Bildschirm. Eine Animation wird als Sequenz von zeit- und optional verzweigten Frames definiert, die aus einem oder mehreren Bildern bestehen.

-   [Laden eines Zeichens](loading-a-character.md)
-   [Laden des Standardzeichens](loading-the-default-character.md)
-   [Animieren eines Zeichens](animating-a-character.md)
