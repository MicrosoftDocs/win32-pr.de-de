# <a name="iagentaudiooutputpropertiesex"></a>IAgentAudioOutputPropertiesEx

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

IAgentAudioOutputPropertiesEx wird von der IAgentAudioOutputPropertiesEx-Schnittstelle abgeleitet. Sie enthält alle IAgentAudioOutputPropertiesEx-Methoden und bietet Zugriff auf zusätzliche Funktionen.

**Methoden in Vtable-Reihenfolge**



| IAgentAudioOutputPropertiesEx-Methoden                         | Beschreibung                                     |
|---------------------------------------------------------------|-------------------------------------------------|
| [**GetStatus**](iagentaudiooutputpropertiesex--getstatus.md) | Gibt den Status des Audioausgabekanals zurück. |
