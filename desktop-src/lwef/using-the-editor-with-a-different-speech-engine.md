# <a name="using-the-editor-with-a-different-speech-engine"></a>Verwenden des Editors mit einer anderen Sprach-Engine

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Während der Sound-Editor die Microsoft-Spracherkennungs-Engine (4,0) installiert, kann Sie mit einer anderen Sprach-Engine verwendet werden, wenn diese Engine die erforderlichen Schnittstellen unterstützt, die im Dokument "Sprachmodul Anforderungen" dokumentiert sind. Bevor Sie versuchen, den Editor mit einem anderen Modul zu verwenden, vergewissern Sie sich, dass der Anbieter diese Anforderungen erfüllt.

Wenn Sie den Sound-Editor mit einer anderen Sprach-Engine verwenden möchten, wählen Sie im Menü **Bearbeiten** den Befehl Sprach-Engine aus. Dadurch wird ein Dialogfeld mit der aktuellen Engine angezeigt. Um ein anderes Modul auszuwählen, zeigen Sie die Liste der Engines an, und klicken Sie dann auf **OK**. Wenn keine anderen Module aufgelistet sind, sind keine weiteren kompatiblen Module enthalten.
