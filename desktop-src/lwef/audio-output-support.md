# <a name="audio-output-support"></a>Audioausgabeunterstützung

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Mit dem Microsoft-Agent können Sie Audiodateien für die gesprochene Ausgabe eines Zeichens verwenden. Sie können Audiodateien aufzeichnen und die [**Sprech**](speak-method.md) Methode verwenden, um diese Daten wiederzugeben. Microsoft-Agent-Animations Dienste unterstützen die automatische Synchronisierung des Zeichen Mundes mit den audiomerkmalen der Audiodatei automatisch. Der Microsoft-Agent unterstützt auch ein spezielles Format für Audiodateien, das zusätzliche Phonem-und Wörter Trennungs Informationen für eine verbesserte Unterstützung für die Unterstützung von Unterstützung bietet. Sie können dieses spezielle Format mit dem Microsoft-Tool für die Audiobearbeitung von Sprachinformationen generieren.
