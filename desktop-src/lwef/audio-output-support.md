# <a name="audio-output-support"></a>Audioausgabeunterstützung

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Mit dem Microsoft-Agent können Sie Audiodateien für die gesprochene Ausgabe eines Zeichens verwenden. Sie können Audiodateien aufzeichnen und die [**Speak-Methode**](speak-method.md) verwenden, um diese Daten wiederzuspielen. Microsoft-Agent-Animationsdienste unterstützen automatisch die Synchronisierung des Zeichengehörs mithilfe der Audiomerkmale der Audiodatei. Der Microsoft-Agent unterstützt auch ein spezielles Format für Audiodateien, das zusätzliche Phoneme- und Wörterpauseninformationen enthält, um die Unterstützung für die Synchronisierung von Lisen zu verbessern. Sie können dieses spezielle Format mit dem Microsoft Linguistic Information Sound Editing Tool generieren.
