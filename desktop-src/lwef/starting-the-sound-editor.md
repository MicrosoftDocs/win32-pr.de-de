# <a name="starting-the-sound-editor"></a>Starten des Sound-Editors

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Um das Microsoft Linguistic Information Sound Editing Tool auszuführen, wählen Sie es im **Startmenü** aus, oder doppelklicken Sie auf das Symbol des Sound-Editors. Das Fenster des Sound-Editors wird geöffnet und zeigt seine Menüs, eine Symbolleiste für häufig verwendete Befehle, ein Textfeld zum Eingeben der Wörter, die der Editor zum Verarbeiten der Sounddatei verwendet, und einen Anzeigebereich zum Anzeigen und Bearbeiten der Audio- und linguistischen Daten an.

![Screenshot: Startseite des Microsoft Linguistic Information Sound Editing Tools](images/f1listliq.gif)

Nachdem der Sound-Editor gestartet wurde, können Sie mit der Aufzeichnung einer neuen Sounddatei beginnen oder eine vorhandene Sounddatei laden.
