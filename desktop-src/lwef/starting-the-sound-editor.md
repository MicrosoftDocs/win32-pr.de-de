# <a name="starting-the-sound-editor"></a>Starten des Sound-Editors

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Um das Microsoft-Tool für die Audiobearbeitung von Sprachinformationen auszuführen, wählen Sie es im **Startmenü** aus, oder Doppelklicken Sie auf das Symbol des Sound-Editors. Das Fenster des Sound-Editors wird geöffnet und zeigt die Menüs, eine Symbolleiste für häufig verwendete Befehle, ein Textfeld zum Eingeben der Wörter, die der Editor zum Verarbeiten der Audiodatei verwendet, und einen Anzeigebereich zum Anzeigen und Bearbeiten der Audiodaten und linguistischen Daten an.

![Screenshot, der die Startseite des Microsoft-Tools für die Audiobearbeitung von linguistischen Informationen anzeigt.](images/f1listliq.gif)

Nachdem der Sound-Editor gestartet wurde, können Sie beginnen, eine neue Audiodatei aufzuzeichnen oder eine vorhandene Audiodatei zu laden.
