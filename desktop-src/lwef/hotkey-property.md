# <a name="hotkey-property"></a>Hotkey-Eigenschaft

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt eine Zeichenfolge zurück, die die aktuelle Einstellung des Benutzers für den Abhör Schlüssel angibt.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Syntax** 
</dt> <dd>

*Agent * * *. Speechinput. Hotkey**

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

Die **Hotkey** -Eigenschaft gibt die aktuelle Einstellung des abhörenden Schlüssels auf der Seite Spracheingabe des agenteigenschaftenblatts an (erweiterte Zeichen Optionen). Die Eigenschaft ist schreibgeschützt. nur der Benutzer kann die Einstellung ändern.
