# <a name="the-propertysheet-object"></a>Das PropertySheet-Objekt

\[Microsoft Agent ist ab Version Windows 7 veraltet und in nachfolgenden Versionen von Windows.\]

Das [**PropertySheet-Objekt**](https://www.bing.com/search?q=**PropertySheet**) stellt mehrere Eigenschaften zur Verfügung, die Sie verwenden können, wenn Sie das Zeichen relativ zum Microsoft Agent-Eigenschaftenblatt (auch als Fenster Erweiterte Zeichenoptionen bekannt) bearbeiten möchten.

-   [**Höhe**](height-property-pso.md)
-   [**Links**](left-property-pso.md)
-   [**Seite**](page-property.md)
-   [**Nach oben**](top-property-pso.md)
-   [**Sichtbar**](visible-property.md)
-   [**Breite**](width-property-pso.md)

Wenn Sie die [**Eigenschaften Height**](height-property-pso.md), [**Left,**](left-property-pso.md) [**Top**](top-property-pso.md)und [**Width**](width-property-pso.md) abfragen, bevor das Eigenschaftenblatt jemals angezeigt wurde, geben ihre Werte als 0 (null) zurück. Nach der Anzeige geben diese Eigenschaften die letzte Position und Größe des Fensters zurück (relativ zur aktuellen Bildschirmauflösung).
