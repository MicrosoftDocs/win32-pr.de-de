# <a name="the-propertysheet-object"></a>Das PropertySheet-Objekt

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Das [**PropertySheet**](https://www.bing.com/search?q=**PropertySheet**) -Objekt bietet verschiedene Eigenschaften, die Sie verwenden können, wenn Sie das Zeichen relativ zum Microsoft-Agent-Eigenschaften Blatt (auch als Fenster für erweiterte Zeichen Optionen bezeichnet) bearbeiten möchten.

-   [**Höhe**](height-property-pso.md)
-   [**Linken**](left-property-pso.md)
-   [**Seite**](page-property.md)
-   [**Nach oben**](top-property-pso.md)
-   [**Sichtbar**](visible-property.md)
-   [**Breite**](width-property-pso.md)

Wenn Sie die Eigenschaften [**height**](height-property-pso.md), [**left**](left-property-pso.md), [**Top**](top-property-pso.md)und [**Width**](width-property-pso.md) Abfragen, bevor das Eigenschaften Blatt angezeigt wird, geben die Werte als NULL (0) zurück. Wenn diese Eigenschaften angezeigt werden, geben Sie die letzte Position und Größe des Fensters zurück (relativ zur aktuellen Bildschirmauflösung).
