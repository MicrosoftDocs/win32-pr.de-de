# <a name="the-characters-object"></a>Das Characters-Objekt

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Ihre Clientanwendung kann ein oder mehrere Zeichen unterstützen. Darüber hinaus können Sie ein Zeichen für mehrere Anwendungen freigeben. Der Microsoft-Agent definiert das [**Characters-Objekt**](/windows/desktop/lwef/the-characters-object) als Eine Auflistung von Zeichen. Um auf ein Zeichen zuzugreifen, laden Sie die Daten des Zeichens in die [**Characters-Auflistung,**](/windows/desktop/lwef/the-characters-object) und geben Sie an, dass das Element in der Auflistung die für dieses Zeichen unterstützten Methoden und Eigenschaften verwendet.

-   [Characters-Objektmethoden](characters-object-methods.md)
-   [Characters-Objektmethoden](character-object-methods.md)
-   [Eigenschaften des Characters-Objekts](character-object-properties.md)
