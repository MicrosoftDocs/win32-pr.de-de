# <a name="the-characters-object"></a>Das Character-Objekt

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Die Client Anwendung kann ein oder mehrere Zeichen unterstützen. Darüber hinaus können Sie ein Zeichen für mehrere Anwendungen freigeben. Der Microsoft-Agent definiert das [**Zeichen**](/windows/desktop/lwef/the-characters-object) Objekt als eine Auflistung von Zeichen. Wenn Sie auf ein Zeichen zugreifen möchten, laden Sie die Daten des Zeichens in die [**Zeichen**](/windows/desktop/lwef/the-characters-object) Auflistung, und geben Sie an, dass das Element in der Auflistung die für dieses Zeichen unterstützten Methoden und Eigenschaften verwendet.

-   [Zeichen Objektmethoden](characters-object-methods.md)
-   [Zeichen Objektmethoden](character-object-methods.md)
-   [Zeichen Objekteigenschaften](character-object-properties.md)
