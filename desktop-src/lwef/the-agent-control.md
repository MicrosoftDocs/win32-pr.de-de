# <a name="the-agent-control"></a>Das agentsteuerelement

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Der Verweis auf das agentsteuerelement ermöglicht den Zugriff auf Ereignisse und die meisten anderen vom Microsoft-Agent unterstützten Objekte. Das agentsteuerelement macht auch direkt seinen eigenen Satz von Eigenschaften verfügbar.

-   [Agentsteuerungsmethoden](agent-control-methods.md)
-   [Agentsteuerungseigenschaften](agent-control-properties.md)
-   [Agentsteuerungsereignisse](agent-control-events.md)
