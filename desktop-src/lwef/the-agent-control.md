# <a name="the-agent-control"></a>Das Agent-Steuerelement

\[Microsoft Agent ist ab Version Windows 7 veraltet und möglicherweise in nachfolgenden Versionen von Windows.\]

Das Verweisen auf das -Agent-Steuerelement ermöglicht den Zugriff auf Ereignisse und die meisten anderen Objekte, die von Microsoft Agent unterstützt werden. Das -Agent-Steuerelement macht auch direkt einen eigenen Satz von Eigenschaften verfügbar.

-   [Agent-Steuerelementeigenschaften](agent-control-properties.md)
