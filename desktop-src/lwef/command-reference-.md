# <a name="command-reference-microsoft-agent-character-editor"></a>Befehlsreferenz (Zeichen-Editor für Microsoft-Agent)

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

## <a name="the-file-menu"></a>Menü "Datei"

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Neu**
</dt> <dd>

Setzt den Agent-Zeichen-Editor zum Erstellen einer neuen Zeichen Definition zurück. Wenn ein vorhandenes Zeichen geladen wird und ungespeicherte Änderungen aufweist, zeigt der Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden sollen.

</dd> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Eren**
</dt> <dd>

Zeigt das Dialogfeld Datei öffnen an, in dem Sie eine vorhandene Zeichen Definitionsdatei zum Bearbeiten öffnen können. Wenn ein vorhandenes Zeichen geladen wird und ungespeicherte Änderungen an einer Datei vorgenommen wurden, zeigt der Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden sollen.

</dd> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Sicher**
</dt> <dd>

Speichert die Zeichen Definition. Wenn die Zeichen Definition nicht vorhanden ist (wurde nicht benannt), zeigt der Editor das Dialogfeld Speichern unter für die Eingabe des Datei namens an.

</dd> <dt>

<span id="Save_As"></span><span id="save_as"></span><span id="SAVE_AS"></span>**Speichern unter**
</dt> <dd>

Zeigt das Dialogfeld Speichern unter an, in dem Sie einen neuen Namen für die Zeichen Definitionsdatei eingeben können.

</dd> <dt>

<span id="Print"></span><span id="print"></span><span id="PRINT"></span>**Gedruck**
</dt> <dd>

Zeigt das Dialogfeld Drucken an, in dem Sie eine Druckoption auswählen und die Zeichen Definitionsdatei drucken können.

</dd> <dt>

<span id="Build_Character"></span><span id="build_character"></span><span id="BUILD_CHARACTER"></span>**Buildzeichen**
</dt> <dd>

Zeigt das Dialogfeld "buildzeichen" an, das Optionen zum Definieren der Daten-und Animationsdateien eines Zeichens für die Verwendung mit dem Microsoft-Agent enthält.

</dd> <dt>

<span id="Page_Setup"></span><span id="page_setup"></span><span id="PAGE_SETUP"></span>**Seite einrichten**
</dt> <dd>

Zeigt das Dialogfeld Seite einrichten an, in dem Sie die Druckoptionen für die Zeichen Definitionsdatei festlegen können.

</dd> <dt>

<span id="Most_Recently_Open_Files"></span><span id="most_recently_open_files"></span><span id="MOST_RECENTLY_OPEN_FILES"></span>**Zuletzt geöffnete Dateien**
</dt> <dd>

Verfolgt die zuletzt geöffneten Zeichen Definitions Dateien, die Sie geöffnet haben. Wenn Sie eine Datei auswählen, wird diese Datei automatisch zur Bearbeitung geöffnet. Wenn ein vorhandenes Zeichen geladen wird und ungespeicherte Änderungen an einer Datei vorgenommen wurden, zeigt der Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden sollen.

</dd> <dt>

<span id="Exit"></span><span id="exit"></span><span id="EXIT"></span>**Abstiegs**
</dt> <dd>

Beendet den Agent-Zeichen-Editor. Wenn ein vorhandenes Zeichen geladen wird und ungespeicherte Änderungen an einer Datei vorgenommen wurden, zeigt der Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden sollen.

</dd> </dl>

## <a name="the-edit-menu"></a>Menü "Bearbeiten"

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Machen**
</dt> <dd>

Entfernt eine im Editor vorgenommene Änderung.

</dd> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Holen**
</dt> <dd>

Kehrt eine Rückgängig-Aktion im Editor um.

</dd> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Schnitts**
</dt> <dd>

Entfernt das ausgewählte Element aus dem Editor und platziert es in der Windows-Zwischenablage.

</dd> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Skopie**
</dt> <dd>

Kopiert das ausgewählte Element im Editor in die Windows-Zwischenablage.

</dd> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Kle**
</dt> <dd>

Kopiert Daten aus der aktuellen Windows-Zwischenablage in den ausgewählten Speicherort.

</dd> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Lösch**
</dt> <dd>

Entfernt das ausgewählte Element aus dem Editor.

</dd> <dt>

<span id="New_Animation"></span><span id="new_animation"></span><span id="NEW_ANIMATION"></span>**Neue Animation**
</dt> <dd>

Erstellt ein neues Animations Objekt im Editor.

</dd> <dt>

<span id="New_Frame"></span><span id="new_frame"></span><span id="NEW_FRAME"></span>**Neuer Frame**
</dt> <dd>

Erstellt einen neuen Frame für eine Animation.

</dd> <dt>

<span id="New_Frames_from_Files"></span><span id="new_frames_from_files"></span><span id="NEW_FRAMES_FROM_FILES"></span>**Neue Frames aus Dateien**
</dt> <dd>

Zeigt das Dialogfeld Bilddateien auswählen an und erstellt Frames mithilfe der ausgewählten Dateien.

</dd> <dt>

<span id="Open_Frame_Window"></span><span id="open_frame_window"></span><span id="OPEN_FRAME_WINDOW"></span>**Rahmen Fenster öffnen**
</dt> <dd>

Zeigt den aktuellen Frame und seine Images in einem separaten Fenster an, ohne dass die in den Frame geladenen Bilder skaliert werden.

</dd> <dt>

<span id="Preview___Stop_Preview"></span><span id="preview___stop_preview"></span><span id="PREVIEW___STOP_PREVIEW"></span>**Vorschau auf Vorschau \| Abbrechen**
</dt> <dd>

Gibt eine Animation wieder (oder beendet Sie), beginnend mit dem ausgewählten Frame.

</dd> <dt>

<span id="Preview___Stop_Preview_Exit_Branching"></span><span id="preview___stop_preview_exit_branching"></span><span id="PREVIEW___STOP_PREVIEW_EXIT_BRANCHING"></span>**Vorschau der Beendigungs \| Verzweigung beenden**
</dt> <dd>

Gibt die Beendigungs Verzweigung einer Animation an (oder beendet Sie), beginnend mit dem ausgewählten Frame.

</dd> </dl>
