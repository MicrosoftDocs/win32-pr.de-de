# <a name="command-reference-microsoft-agent-character-editor"></a>Befehlsreferenz (Microsoft Agent-Zeichen-Editor)

\[Microsoft Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

## <a name="the-file-menu"></a>Menü "Datei"

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Neu**
</dt> <dd>

Setzt den Zeichen-Editor des -Agents zurück, um eine neue Zeichendefinition zu erstellen. Wenn ein vorhandenes Zeichen geladen wird und nicht gespeicherte Bearbeitungen enthält, zeigt der Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden.

</dd> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Öffnen**
</dt> <dd>

Zeigt das Dialogfeld Datei öffnen an, in dem Sie eine vorhandene Zeichendefinitionsdatei zur Bearbeitung öffnen können. Wenn ein vorhandenes Zeichen geladen wird und nicht gespeicherte Änderungen an einer Datei enthält, zeigt der Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden.

</dd> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Speichern**
</dt> <dd>

Speichert die Zeichendefinition. Wenn die Zeichendefinition nicht vorhanden ist (nicht benannt wurde), zeigt der Editor das Dialogfeld Speichern unter für die Eingabe des Dateinamens an.

</dd> <dt>

<span id="Save_As"></span><span id="save_as"></span><span id="SAVE_AS"></span>**Speichern unter**
</dt> <dd>

Zeigt das Dialogfeld Speichern unter an, in dem Sie einen neuen Namen für die Zeichendefinitionsdatei eingeben können.

</dd> <dt>

<span id="Print"></span><span id="print"></span><span id="PRINT"></span>**Drucken**
</dt> <dd>

Zeigt das Dialogfeld Drucken an, in dem Sie eine Druckoption auswählen und die Zeichendefinitionsdatei drucken können.

</dd> <dt>

<span id="Build_Character"></span><span id="build_character"></span><span id="BUILD_CHARACTER"></span>**Buildzeichen**
</dt> <dd>

Zeigt das Dialogfeld Zeichen erstellen an, das Optionen zum Definieren der Erstellung der Daten- und Animationsdateien eines Zeichens für die Verwendung mit Microsoft Agent enthält.

</dd> <dt>

<span id="Page_Setup"></span><span id="page_setup"></span><span id="PAGE_SETUP"></span>**Seiteneinrichtung**
</dt> <dd>

Zeigt das Dialogfeld Seiteneinrichtung an, in dem Sie die Druckoptionen für die Zeichendefinitionsdatei festlegen können.

</dd> <dt>

<span id="Most_Recently_Open_Files"></span><span id="most_recently_open_files"></span><span id="MOST_RECENTLY_OPEN_FILES"></span>**Zuletzt geöffnete Dateien**
</dt> <dd>

Verfolgt die zuletzt geöffneten Zeichendefinitionsdateien nach. Wenn Sie eine Datei auswählen, wird diese Datei automatisch zur Bearbeitung geöffnet. Wenn ein vorhandenes Zeichen geladen wird und nicht gespeicherte Änderungen an einer Datei enthält, zeigt der Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden.

</dd> <dt>

<span id="Exit"></span><span id="exit"></span><span id="EXIT"></span>**Ausfahrt**
</dt> <dd>

Beendet den Zeichen-Editor des Agents. Wenn ein vorhandenes Zeichen geladen wird und nicht gespeicherte Änderungen an einer Datei enthält, zeigt der Editor eine Meldung an, um zu bestimmen, ob nicht gespeicherte Änderungen gespeichert oder verworfen werden.

</dd> </dl>

## <a name="the-edit-menu"></a>Menü "Bearbeiten"

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Rückgängig**
</dt> <dd>

Entfernt eine im Editor vorgenommene Änderung.

</dd> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Wiederholen**
</dt> <dd>

Kehrt eine Rückgängig-Aktion im Editor um.

</dd> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Schneiden**
</dt> <dd>

Entfernt das ausgewählte Element aus dem Editor und platziert es in der Windows-Zwischenablage.

</dd> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Kopieren**
</dt> <dd>

Kopiert das ausgewählte Element im Editor in die Windows-Zwischenablage.

</dd> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Einfügen**
</dt> <dd>

Kopiert Daten aus der aktuellen Windows-Zwischenablage an den ausgewählten Speicherort.

</dd> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Löschen**
</dt> <dd>

Entfernt das ausgewählte Element aus dem Editor.

</dd> <dt>

<span id="New_Animation"></span><span id="new_animation"></span><span id="NEW_ANIMATION"></span>**Neue Animation**
</dt> <dd>

Erstellt ein neues Animationsobjekt im Editor.

</dd> <dt>

<span id="New_Frame"></span><span id="new_frame"></span><span id="NEW_FRAME"></span>**Neuer Frame**
</dt> <dd>

Erstellt einen neuen Frame für eine Animation.

</dd> <dt>

<span id="New_Frames_from_Files"></span><span id="new_frames_from_files"></span><span id="NEW_FRAMES_FROM_FILES"></span>**Neue Frames aus Dateien**
</dt> <dd>

Zeigt das Dialogfeld Bilddateien auswählen an und erstellt Frames mithilfe der ausgewählten Dateien.

</dd> <dt>

<span id="Open_Frame_Window"></span><span id="open_frame_window"></span><span id="OPEN_FRAME_WINDOW"></span>**Fenster "Frame öffnen"**
</dt> <dd>

Zeigt den aktuellen Frame und seine Bilder in einem separaten Fenster an, ohne die in den Frame geladenen Bilder zu skalieren.

</dd> <dt>

<span id="Preview___Stop_Preview"></span><span id="preview___stop_preview"></span><span id="PREVIEW___STOP_PREVIEW"></span>**Preview \| Stop Preview (Vorschauversion beenden)**
</dt> <dd>

Gibt eine Animation ab dem ausgewählten Frame wieder (oder beendet sie).

</dd> <dt>

<span id="Preview___Stop_Preview_Exit_Branching"></span><span id="preview___stop_preview_exit_branching"></span><span id="PREVIEW___STOP_PREVIEW_EXIT_BRANCHING"></span>**Vorschauversion \| beenden Vorschau beenden Branching beenden**
</dt> <dd>

Gibt die Exit-Verzweigung einer Animation ab dem ausgewählten Frame wieder (oder beendet sie).

</dd> </dl>
