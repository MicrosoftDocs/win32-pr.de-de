# <a name="lst-tag"></a>Lst-Tag

\[Microsoft Agent ist ab Version Windows 7 veraltet und in nachfolgenden Versionen von Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Wiederholt die letzte gesprochene Anweisung für das Zeichen.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

\\**Lst**\\

</dd> </dl>

### <a name="remarks"></a>Hinweise

Mit diesem Tag kann ein Zeichen seine letzte gesprochene Anweisung wiederholen. Dieses Tag muss allein in der [**Speak-Methode angezeigt**](speak-method.md) werden. Es kann kein anderer Text oder keine anderen Parameter eingeschlossen werden. Wenn der gesprochene Text wiederholt wird, werden alle anderen Tags, die im ursprünglichen Text enthalten sind, mit Ausnahme von Lesezeichen wiederholt. jegliche. WAV und . Im Text enthaltene LWV-Dateien werden ebenfalls wiederholt.
