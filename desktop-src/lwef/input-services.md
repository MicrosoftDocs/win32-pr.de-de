# <a name="input-services"></a>Eingabe Dienste

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Eine Client Anwendung stellt die primäre Benutzeroberfläche für die Interaktion mit einem Zeichen bereit. Sie können ein Zeichen programmieren, um auf eine beliebige Form von Eingaben zu reagieren, von Schaltflächen Klicks zum typisierten Text. Darüber hinaus stellt der Microsoft-Agent Ereignisse bereit, damit Sie programmieren können, was geschieht, wenn der Benutzer klickt, doppelklickt oder das Zeichen zieht. Der Server übergibt die Koordinaten des Zeigers und einen beliebigen modifiziererschlüsselzustand für diese Ereignisse.
