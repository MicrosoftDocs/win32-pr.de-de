# <a name="the-listening-tip-window"></a>Fenster "Lauschende Trinkgeld"

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Wenn die Spracheingabe aktiviert ist, wird ein spezielles QuickInfo-Fenster angezeigt, wenn der Benutzer die Push-to-Talk-Taste drückt, um mit der Spracheingabe zu beginnen. Der Lauschendtipp zeigt Kontextinformationen im Zusammenhang mit dem aktuellen Eingabestatus des Microsoft-Agents an. Wenn die kompatible Spracherkennung nicht installiert oder deaktiviert wurde, wird der Lauschende Tipp nicht angezeigt.
