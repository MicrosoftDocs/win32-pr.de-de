# <a name="the-listening-tip-window"></a>Das Fenster "lausch Tipp"

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Wenn Sprache aktiviert ist, wird ein spezielles QuickInfo-Fenster angezeigt, wenn der Benutzer den Push-to-Talk-Schlüssel drückt, um mit der Spracheingabe zu beginnen. Der lausch Tipp zeigt Kontextinformationen im Zusammenhang mit dem aktuellen Eingabe Status des Microsoft-Agents an. Wenn die kompatible Spracherkennung nicht installiert oder deaktiviert wurde, wird der lausch Tipp nicht angezeigt.
