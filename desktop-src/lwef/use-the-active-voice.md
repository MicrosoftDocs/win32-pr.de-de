# <a name="use-the-active-voice"></a>Verwenden der aktiven Stimme

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Wenn Sie die Sprachausgabe verwenden, um Direktiveninformationen bereitzustellen oder eine Benutzer Antwort zu verwenden, verwenden Sie die aktive Stimme, und geben Sie die erwartete Aktion des Benutzers eindeutig an. Das folgende Beispiel veranschaulicht die Unterschiede:



|                                      |                               |
|--------------------------------------|-------------------------------|
| Wiederholen Sie die Nummer.           | Keine Benutzeraktion                |
| Die Zahl wird wiederholt.         | Passive Stimme, keine Benutzeraktion |
| Lauschen Sie, während die Zahl wiederholt wird. | Passive Stimme                 |
| Lauschen Sie auf die Wiederholung.            | Beste Wahl                   |



 

Erstellen Sie außerdem die Ausgabe, um die Schlüsselinformationen am Ende des Ausdrucks zu erweitern, wie in den folgenden Beispielen gezeigt:



|             |                                      |                                            |
|-------------|--------------------------------------|--------------------------------------------|
| anstelle von: | "Ist drei die nächste Ziffer?"           | Verwenden Sie: "ist die nächste Ziffer 3?"            |
| anstelle von: | "Klicken Sie auf" OK ", um zu beginnen.                 | Verwenden Sie Folgendes: "beginnen Sie, klicken Sie auf" OK ".                 |
| anstelle von: | Sagen Sie "Fertig", um Ihre Bestellung abzuschließen. | Verwenden Sie: ", um Ihre Bestellung abzuschließen, beispielsweise" Fertig ". |
