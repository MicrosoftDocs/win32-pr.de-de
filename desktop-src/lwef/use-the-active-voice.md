# <a name="use-the-active-voice"></a>Verwenden von Active Voice

\[Microsoft Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Wenn Sie die Sprachausgabe verwenden, um Direktiveinformationen anzugeben oder eine Benutzerantwort zu erhüren, verwenden Sie die aktive Stimme, und geben Sie die erwartete Aktion des Benutzers eindeutig an. Im folgenden Beispiel werden die Unterschiede veranschaulicht:



| Anweisung                            |  Auswertung                   |
|--------------------------------------|-------------------------------|
| Lassen Sie mich Ihre Zahl wiederholen.           | Keine Benutzeraktion                |
| Die Zahl wird wiederholt.         | Passive Stimme, keine Benutzeraktion |
| Lauschen Sie, während die Zahl wiederholt wird. | Passive Stimme                 |
| Lauschen Sie auf die Wiederholung.            | Beste Wahl                   |



 

Erstellen Sie außerdem Ihre Ausgabe, um die wichtigsten Informationen am Ende des Ausdrucks zu verfingen, wie in den folgenden Beispielen gezeigt:



| Anstatt...                                                 | Option                                           |
|--------------------------------------------------|--------------------------------------------|
| "Ist drei die nächste Ziffer?"           | "Ist die nächste Ziffer drei?"            |
| "Klicken Sie auf OK, um zu beginnen."                 | "Klicken Sie zunächst auf OK."                 |
| "Say 'Done' to complete your order." | "Wenn Sie Ihre Bestellung abschließen, sagen Sie "Fertig"." |
