# <a name="programming-the-microsoft-agent-server-interface"></a>Programmieren der Microsoft-Agent-Server Schnittstelle

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Der Microsoft-Agent bietet Dienste, mit denen Sie animierte Zeichen aus einer Anwendung programmieren können. Diese Dienste werden als OLE-Automatisierungsserver implementiert. Mithilfe der OLE-Automatisierung kann eine Anwendung das-Objekt einer anderen Anwendung Programm gesteuert steuern. In diesem Dokument wird davon ausgegangen, dass die Component Object Model (com) und OLE verstanden wurden. Eine Einführung in diese Dienste finden Sie in der [Übersicht über die Programmierschnittstelle](microsoft-agent-programming-interface-overview.md).
