# <a name="programming-the-microsoft-agent-server-interface"></a>Programmieren der Microsoft-Agent-Serverschnittstelle

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Microsoft Agent stellt Dienste bereit, mit denen Sie animierte Zeichen aus einer Anwendung programmieren können. Diese Dienste werden als OLE-Automatisierungsserver implementiert. Ole-Automatisierung ermöglicht es einer Anwendung, das Objekt einer anderen Anwendung programmgesteuert zu steuern. In diesem Dokument wird davon ausgegangen, dass Sie die Component Object Model (COM) und OLE verstehen. Eine Einführung in diese Dienste finden Sie in der [Übersicht über die Programmierschnittstelle.](microsoft-agent-programming-interface-overview.md)
