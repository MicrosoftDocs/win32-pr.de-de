# <a name="windows-defender-structures"></a>Windows Defender Strukturen

Strukturen, die von Apps beim Aufrufen von zum Anfordern von Überprüfungen, Signaturaktualisierungen oder Informationen von Windows Defender verwendet werden.



| Struktur                                                      | BESCHREIBUNG                                                                             |
|----------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [**\_MPCALLBACK-DATEN**](mpcallback-data.md)                    | Daten, die an die Rückruffunktion übergeben werden.<br/>                                        |
| [**MPCLEAN \_ DATA**](mpclean-data.md)                          | Benachrichtigungsdaten, die an eine bereinigte Rückruffunktion übergeben werden.<br/>                         |
| [**MPCLEAN \_ PRECHECK \_ DATA**](mpclean-precheck-data.md)       | Benachrichtigungsdaten, die an die Bereinigung der Rückruffunktion "Precheck" übergeben werden.<br/>                |
| [**MPCOMPONENT-STATUS \_**](mpcomponent-status.md)              | Komponentenstatusinformationen.<br/>                                                |
| [**MPCOMPONENT-VERSION \_**](mpcomponent-version.md)            | Version und Updatezeit für eine einzelne Komponente.<br/>                         |
| [**\_MPCONFIGURATION-DATEN**](mpconfiguration-data.md)          | Enthält Daten zu Konfigurationsänderungen, einschließlich der alten und neuen Werte.<br/> |
| [**MPENDOFLIFE-DATEN \_**](mpendoflife-data.md)                  | Benachrichtigungsdaten zum Ende der Lebensdauer.<br/>                                             |
| [**\_MPEXPIRATION-DATEN**](mpexpiration-data.md)                | Benachrichtigung zum Produktablaufstatus.<br/>                                      |
| [**\_MPFASTPATH-DATEN**](mpfastpath-data.md)                    | FastPath-Updatebenachrichtigung.<br/>                                                |
| [**\_MPHEALTH-DATEN**](mphealth-data.md)                        | Integritätsbenachrichtigungsdaten.<br/>                                                    |
| [**MPMALWARETOAST-DATEN \_**](mpmalwaretoast-data.md)            | Daten zu Popupbenachrichtigungen durch Schadsoftware.<br/>                                             |
| [**PRIVATE \_ \_ MPNIS-DATEN**](mpnis-private-data.md)             | Private NIS-Benachrichtigungen.<br/>                                                   |
| [**M BEIBEHALTENE \_ DATEN**](mpreserved-data.md)                    | Reservierte Benachrichtigungsdaten.<br/>                                                  |
| [**\_MPRESOURCE-INFORMATIONEN**](mpresource-info.md)                    | Struktur der Ressourceninformationen.<br/>                                              |
| [**\_MPRESOURCE-STATISTIKEN**](mpresource-stats.md)                  | Ressourcenbezogene Statistiken.<br/>                                                 |
| [**\_MPSAMPLE-DATEN**](mpsample-data.md)                        | Benachrichtigungsdaten, die an die Rückruffunktion für die Beispielübermittlung übergeben werden.<br/>         |
| [**\_MPSCAN-DATEN**](mpscan-data.md)                            | Überprüfen Sie die an den Rückruf übergebenen Daten.<br/>                                            |
| [**\_MPSCAN-RESSOURCEN**](mpscan-resources.md)                  | Ressourceninformationen, die während eines Überprüfungsvorgangs übergeben werden.<br/>                         |
| [**\_MPSCAN-ERGEBNIS**](mpscan-result.md)                        | Die Ergebnisse einer Überprüfung.<br/>                                                       |
| [**\_MPSIGUPDATE-DATEN**](mpsigupdate-data.md)                  | Benachrichtigungsdaten, die an die Rückruffunktion der Signaturaktualisierung übergeben werden.<br/>          |
| [**\_MPSTATUS-DATEN**](mpstatus-data.md)                        | Enthält Daten zum aktuellen Status einer Komponente des Produkts.<br/>        |
| [**MPSTATUS \_ DATAEX \_ NICHT VERWENDET**](mpstatus-dataex-unused.md)     | Dummystruktur für Nicht-SRP.<br/>                                                 |
| [**MPSTATUS \_ INFO**](mpstatus-info.md)                        | Statusinformationen für den Schadsoftwareschutz-Manager.<br/>                       |
| [**\_MPTHREAT-DATEN**](mpthreat-data.md)                        | Benachrichtigungsdaten, die aufgrund von Bedrohungserkennung oder -änderung übergeben werden.<br/>            |
| [**\_MPTHREAT-INFORMATIONEN**](mpthreat-info.md)                        | Enthält Informationen zu einer Bedrohung.<br/>                                         |
| [**MPTHREAT \_ \_ INFOEX-VERHALTEN**](mpthreat-infoex-behavior.md) | Enthält verhaltensänderungsspezifische Informationen.<br/>                         |
| [**MPTHREAT \_ INFOEX \_ NIS**](mpthreat-infoex-nis.md)           | Enthält NIS-spezifische Informationen.<br/>                                           |
| [**MPTHREAT \_ INFOEX \_ UNUSED**](mpthreat-infoex-unused.md)     | Dummystruktur für Nichtverhaltensänderungstypbedrohungen.<br/>                  |
| [**\_LOKALISIERTE \_ MPTHREAT-INFORMATIONEN**](mpthreat-localized-info.md)   | Lokalisierte Informationen für eine Bedrohung.<br/>                                          |
| [**\_MPTHREAT-STATISTIKEN**](mpthreat-stats.md)                      | Bedrohungsbezogene Statistiken.<br/>                                                   |
| [**\_MPTHREAT-STATISTIKDATEN \_**](mpthreat-stats-data.md)           | Zusätzliche Daten zu Bedrohungsstatistiken.<br/>                                           |
| [**MPVERSION \_ INFO**](mpversion-info.md)                      | Versionsinformationen für die Komponenten des Schadsoftwareschutz-Managers.<br/>         |
