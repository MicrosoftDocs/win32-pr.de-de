# <a name="iagentaudiooutputproperties"></a>IAgentAudioOutputProperties

\[Microsoft Agent ist ab Version Windows 7 veraltet und in nachfolgenden Versionen von Windows.\]

IAgentAudioOutputProperties bietet Zugriff auf Audioausgabeeigenschaften, die vom Microsoft Agent-Server verwaltet werden. Diese Funktionen sind auch über IAgentAudioOutputProperties verfügbar. Die Eigenschaften sind schreibgeschützt, aber der Benutzer kann sie auf dem Microsoft Agent-Eigenschaftenblatt ändern.

**Methoden in Vtable-Reihenfolge**



| IAgentAudioOutputProperties-Methoden                                               | Beschreibung                                     |
|-----------------------------------------------------------------------------------|-------------------------------------------------|
| [**Getenabled**](iagentaudiooutputproperties--getenabled.md)                     | Gibt zurück, ob die Audioausgabe aktiviert ist.        |
| [**GetUsingSoundEffects**](iagentaudiooutputproperties--getusingsoundeffects.md) | Gibt zurück, ob die Soundeffektausgabe aktiviert ist. |
