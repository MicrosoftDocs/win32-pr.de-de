# <a name="iagentaudiooutputproperties"></a>Iagentaudiooutputproperties

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Iagentaudiooutputproperties ermöglicht den Zugriff auf audioausgabeeigenschaften, die vom Microsoft-Agent-Server verwaltet werden. Diese Funktionen sind auch über iagentaudiooutputproperties verfügbar. Die Eigenschaften sind schreibgeschützt, aber der Benutzer kann Sie im Eigenschaften Blatt Microsoft-Agent ändern.

**Methoden in Vtable-Reihenfolge**



| Iagentaudiooutputproperties-Methoden                                               | BESCHREIBUNG                                     |
|-----------------------------------------------------------------------------------|-------------------------------------------------|
| [**GetEnabled**](iagentaudiooutputproperties--getenabled.md)                     | Gibt zurück, ob die Audioausgabe aktiviert ist.        |
| [**Getusingsoundebug**](iagentaudiooutputproperties--getusingsoundeffects.md) | Gibt zurück, ob die Ausgabe der Soundeffekte aktiviert ist. |
