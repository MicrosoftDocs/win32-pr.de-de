# <a name="spd-tag"></a>SPD-Tag

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Legt die durchschnittliche durchschnittliche Sprechgeschwindigkeit der Sprachausgabe fest.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

**\\SPD =***Zahl***\\**



| Teil     | BESCHREIBUNG                                          |
|----------|------------------------------------------------------|
| *Zahl* | Mittlere durchschnittliche Sprechgeschwindigkeit in Wörtern pro Minute. |



 

</dd> </dl>

### <a name="remarks"></a>Bemerkungen

Dieses Tag wird nur für die von TTS generierte Ausgabe unterstützt. Der Wertebereich für den Parameter kann abhängig von der installierten TTS-Engine variieren.
