# <a name="animation-sound-effects"></a>Sound Effekte der Animation

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Mit dem Microsoft-Agent können Sie auch Soundeffekte als Teil der Animation eines Zeichens einschließen. Mithilfe des Zeichen-Editors von Microsoft-Agent können Sie den Dateinamen des standardmäßigen Windows-Sounds () angeben. WAV-Dateien, die für einen bestimmten Frame auslöst sind. Beachten Sie, dass der Microsoft-Agent keine Soundeffekte und gesprochenen Ausgaben kombiniert, sodass die gesprochene Ausgabe erst beginnt, wenn ein Soundeffekt abgeschlossen ist. Vermeiden Sie daher einen langen oder Schleifen Soundeffekt als Teil der Animation eines Zeichens.
