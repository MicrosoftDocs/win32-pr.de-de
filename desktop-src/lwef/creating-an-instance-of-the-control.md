# <a name="creating-an-instance-of-the-control"></a>Erstellen einer Instanz des Steuerelements

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht mehr verfügbar.\]

Für den Zugriff auf das Steuerelement des Microsoft-Agents müssen Sie zunächst eine Instanz des Steuerelements erstellen. Die einfachste Möglichkeit hierfür besteht darin, eine Instanz des Steuerelements auf einem Formular zu platzieren. Möglicherweise müssen Sie das Steuerelement der Toolbox hinzufügen, bevor Sie es dem Formular hinzufügen. Je nach Programmiersprache können Sie das Steuerelement möglicherweise auch zur Laufzeit erstellen.
