# <a name="creating-an-instance-of-the-control"></a>Erstellen einer Instanz des Steuer Elements

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

Der Zugriff auf das Steuerelement von Microsoft-Agent erfordert, dass Sie zunächst eine Instanz des Steuer Elements erstellen. Die einfachste Möglichkeit hierfür ist, eine Instanz des Steuer Elements in einem Formular zu platzieren. Möglicherweise müssen Sie der Toolbox das-Steuerelement hinzufügen, bevor Sie es dem Formular hinzufügen. Abhängig von ihrer Programmiersprache können Sie das Steuerelement möglicherweise auch zur Laufzeit erstellen.
