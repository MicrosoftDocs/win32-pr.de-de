# <a name="numberoflines-property"></a>Nummerioflines-Eigenschaft

\[Der Microsoft-Agent ist ab Windows 7 veraltet und in nachfolgenden Versionen von Windows möglicherweise nicht verfügbar.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Beschreibung**
</dt> <dd>

Gibt die Anzahl der Zeilen zurück, die für die Wort Sprechblase für das angegebene Zeichen unterstützt werden.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Agent ***. Zeichen ("*** Merkmal-ID * * *"). Sprechblase. anzahlungszeilen**

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

Die Eigenschaft " **numoflines** " gibt die Anzahl von Textzeilen als Long Integer-Wert zurück.
