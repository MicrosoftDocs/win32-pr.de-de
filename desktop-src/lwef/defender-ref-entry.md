# <a name="windows-defender-reference"></a>Windows Defender-Referenz

Windows Defender trägt zum Schutz von Computern vor Popups, leistungsschwachen Und Sicherheitsbedrohungen bei, die durch Spyware und andere unerwünschte Software verursacht werden. Apps können Informationen wie Computerzustand, spezifische Schadsoftwareinformationen und den Verlauf der Schadsoftware für den Client abfragen und bedarfsorientierte Überprüfungen aufrufen.
