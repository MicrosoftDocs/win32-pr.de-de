# <a name="windows-defender-reference"></a>Windows Defender Verweis

Windows Defender hilft, Computer vor Popups, langsamer Leistung und Sicherheitsbedrohungen zu schützen, die durch Spyware und andere unerwünschte Software verursacht werden. Apps können Informationen wie Computerzustand, bestimmte Schadsoftwareinformationen und den Malwareverlauf für den Client abfragen und bedarfsbasierte Scans aufrufen.
