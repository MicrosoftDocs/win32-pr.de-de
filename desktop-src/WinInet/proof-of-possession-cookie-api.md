# <a name="proof-of-possession-cookie-api"></a>Cookie-API zum Nachweis des Besitzes

Ermöglicht ihnen das Erstellen von Besitznachweiscookies, die einmaliges Anmelden für Webanwendungen unterstützen, die über das OAuth-Internetauthentifizierungsprotokoll authentifiziert werden. Sie können die von dieser API zurückgegebenen Informationen verwenden, um das einmalige Anmelden über Microsoft-Konten zu implementieren.
