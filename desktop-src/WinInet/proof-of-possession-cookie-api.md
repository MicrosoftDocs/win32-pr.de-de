# <a name="proof-of-possession-cookie-api"></a>Nachweis der Besitz-Cookie-API

Ermöglicht das Erstellen von Eigentumsnachweis-Cookies, die Single Sign-on für Webanwendungen unterstützen, die durch das OAuth-Internet Authentifizierungsprotokoll authentifiziert werden. Sie können die von dieser API zurückgegebenen Informationen verwenden, um Single Sign-on Formular Microsoft-Konten zu implementieren.
