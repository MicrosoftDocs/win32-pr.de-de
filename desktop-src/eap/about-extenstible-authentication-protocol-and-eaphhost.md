# <a name="about-eap-and-eaphost"></a>Informationen zu EAP und EAPHost

## <a name="about-eaphost"></a>Informationen zu EAPHost

EAPHost implementiert den EAP-Zustands Automat (Extensible Authentication Protocol) und das EAP-Protokoll Framework gemäß [RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063). Weitere Informationen finden Sie unter Informationen [zu EAPHost](about-eaphost.md).

## <a name="about-extensible-authentication-protocol"></a>Informationen über das Extensible Authentication-Protokoll

EAP ist eine Microsoft Windows-Netzwerkkomponente, die eine EAP-Infrastruktur (Extensible Authentication Protocol) bereitstellt. Weitere Informationen finden Sie unter Informationen [zu EAP](about-extensible-authentication-protocol.md).
