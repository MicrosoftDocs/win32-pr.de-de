# <a name="learn-about-the-eap-and-eaphost-relationship"></a>Weitere Informationen zur EAP-und EAPHost-Beziehung

In diesem Thema wird die Beziehung zwischen dem Extensible Authentication Protocol (EAP) und dem Extensible Authentication Protocol-Host beschrieben. Informationen zur Entwicklung neuer EAP-Methoden finden Sie unter [Extensible Authentication Protocol Host](../eaphost/portal.md).

## <a name="eaphost-framework"></a>EAPHost-Framework

EAPHost wird aus zwei Gründen als Framework für EAP betrachtet.

-   EAPHost ist eine Komponente der Windows-Infrastruktur, die die EAP-Plug-ins hostet.
-   EAPHost implementiert das EAP-Protokoll gemäß [RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063). Eine EAP-Authentifizierung besteht aus dem EAP-Protokoll gemäß [RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063)und den für die EAP-Methode spezifischen Aspekten des Protokolls.
