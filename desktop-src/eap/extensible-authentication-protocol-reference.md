# <a name="extensible-authentication-protocol-reference"></a>Extensible Authentication-Protokoll Referenz

In diesem Abschnitt werden die Verweis Elemente beschrieben, die zum Implementieren von EAP (Extensible Authentication Protocol) verwendet werden. Diese Verweis Elemente sind Funktionen, die zum Implementieren von Authentifizierungs Protokollen, Authentifizierungs Anbietern und Kontoführungs Anbietern verwendet werden. Dieser Abschnitt enthält auch die Strukturen und Enumerationstypen, die von diesen Funktionen verwendet werden.

In dieser Dokumentation werden die folgenden Themen beschrieben.

-   [EAP-Enumerationen](eap-enumerations.md)
-   [EAP-Funktionen](eap-functions.md)
-   [EAP-Schnittstellen](eap-interfaces.md)
-   [EAP-Strukturen](eap-structures.md)
