# <a name="extensible-authentication-protocol-reference"></a>Extensible Authentication-Protokollreferenz

In diesem Abschnitt werden die Verweiselemente beschrieben, die zum Implementieren des Extensible Authentication Protocol (EAP) verwendet werden. Zu diesen Referenzelementen gehören Funktionen, die zum Implementieren von Authentifizierungsprotokollen, Authentifizierungsanbietern und Buchhaltungsanbietern verwendet werden. Dieser Abschnitt enthält auch die Strukturen und Enumerationstypen, die von diesen Funktionen verwendet werden.

In dieser Dokumentation werden die folgenden Themen beschrieben.

-   [EAP-Funktionen](eap-functions.md)
-   [EAP-Schnittstellen](eap-interfaces.md)
-   [EAP-Strukturen](eap-structures.md)
