# <a name="text-input-panel-enumerations"></a>Text Eingabe Panel-Enumerationen

Dieser Abschnitt enthält die Enumerationen, die von der [**itextinputpanel-Schnittstelle**](/windows/desktop/api/peninputpanel/nn-peninputpanel-itextinputpanel)verwendet werden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Enumeration                                                  | Beschreibung                                                                                                                               |
|--------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [**CorrectionMode-Enumeration**](/windows/win32/api/peninputpanel/ne-peninputpanel-correctionmode)         | Definiert Werte, die die Korrektur Modi des Tablet PC-Eingabe Bereichs darstellen.<br/>                                               |
| [**CorrectionPosition-Enumeration**](/windows/win32/api/peninputpanel/ne-peninputpanel-correctionposition) | Definiert Werte, die die Korrektur Positionen nach dem Einfügen des Eingabe Bereichs darstellen.<br/>                                      |
| [**EventMask**](/windows/win32/api/peninputpanel/ne-peninputpanel-eventmask)                               | Gibt die Ereignisse an, für die Sie die Aufmerksamkeit festlegen können.                                                                                      |
| [**InPlaceDirection-Enumeration**](/windows/win32/api/peninputpanel/ne-peninputpanel-inplacedirection)     | Definiert Werte, die die Positionen darstellen, an denen der Eingabebereich relativ zur aktuellen Text Einfügemarke platziert werden kann.<br/> |
| [**InPlaceState-Enumeration**](/windows/win32/api/peninputpanel/ne-peninputpanel-inplacestate)             | Definiert Werte, die darstellen, wie der Eingabebereich auf dem Bildschirm angezeigt wird.<br/>                                                       |
| [**Interaktionmode-Enumeration**](/windows/win32/api/peninputpanel/ne-peninputpanel-interactionmode)       | Definiert Werte, die die Positionierung des Eingabe Bereichs vorschreiben.<br/>                                                                |
| [**PanelInputArea-Enumeration**](/windows/win32/api/peninputpanel/ne-peninputpanel-panelinputarea)         | Definiert Werte, die die Eingabe Bereiche des Eingabe Bereichs darstellen.<br/>                                                              |
