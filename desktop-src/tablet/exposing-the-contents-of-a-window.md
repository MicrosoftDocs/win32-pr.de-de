# <a name="exposing-the-contents-of-a-window"></a>Verfügbar machen des Inhalts eines Fensters

Eine Anwendung muss den Inhalt von benutzerdefinierten Fenster Klassen verfügbar machen, die Microsoft Active Accessibility verwenden. Das gesamte Fenster muss als benutzerdefiniertes Steuerelement behandelt werden, bei dem es sich um einen Container für andere standardmäßige, angepasste oder benutzerdefinierte Steuerelemente handelt.

Weitere Informationen zum verfügbar machen der Inhalte benutzerdefinierter Fenster mithilfe Active Accessibility finden Sie im Abschnitt zur [Barrierefreiheit](/windows/desktop/accessibility) .
