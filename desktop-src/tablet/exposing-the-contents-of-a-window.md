# <a name="exposing-the-contents-of-a-window"></a>Verfügbar machen des Inhalts eines Fensters

Eine Anwendung muss den Inhalt benutzerdefinierter Fensterklassen verfügbar machen, die Microsoft Active Accessibility. Das gesamte Fenster sollte als benutzerdefiniertes Steuerelement behandelt werden, bei dem es sich um einen Container für andere standard-, benutzerdefinierte oder benutzerdefinierte Steuerelemente handelt.

Weitere Informationen zum Verfügbar machen des Inhalts von benutzerdefinierten Fenstern mit Active Accessibility Sie im Abschnitt [Barrierefreiheit.](/windows/desktop/accessibility)
