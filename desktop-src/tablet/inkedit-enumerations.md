# <a name="inkedit-enumerations"></a>InkEdit-Enumerationen

Dieser Abschnitt enthält Enumerationen, die zum InkEdit-Steuerelement gehören.



| Enumeration                                            | Beschreibung                                                                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AppearanceConstants**](/windows/desktop/api/inked/ne-inked-appearanceconstants)     | Gibt an, ob das Steuerelement flach oder 3D angezeigt wird.<br/>                                                                                            |
| [**BorderStyleConstants**](/windows/desktop/api/inked/ne-inked-borderstyleconstants)   | Gibt an, ob das Steuerelement über einen Rahmen verfügt.<br/>                                                                                                   |
| [**InkApplicationGesture**](/windows/desktop/api/msinkaut/ne-msinkaut-inkapplicationgesture) | Gibt das Interesse an einer Reihe anwendungsspezifischer Gesten an.<br/>                                                                             |
| [**InkDisplayMode**](/windows/desktop/api/inked/ne-inked-inkdisplaymode)               | Gibt an, ob eine Auswahl als Ink oder Text angezeigt wird.<br/>                                                                                         |
| [**InkEditStatus**](/windows/desktop/api/inked/ne-inked-inkeditstatus)                 | Gibt an, ob sich das [InkEdit-Steuerelement](inkedit-control-reference.md) im Leerlauf befindet, Ink erfasst oder Ink erkennt.<br/>                           |
| [**InkInsertMode**](/windows/desktop/api/inked/ne-inked-inkinsertmode)                 | Gibt an, wie Ink in das [InkEdit-Steuerelement](inkedit-control-reference.md) eingefügt wird.<br/>                                                      |
| [**InkMode**](/windows/desktop/api/inked/ne-inked-inkmode)                             | Gibt die Sammlungsmoduseinstellungen für gezeichnete Ink-Sammlungen an– unabhängig davon, ob die Ink-Sammlung deaktiviert ist, Ink gesammelt wird oder Ink und Gesten gesammelt werden.<br/> |
| [**InkMouseButton**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousebutton)               | Gibt an, welche Maustaste gedrückt wurde.<br/>                                                                                                     |
| [**InkMousePointer**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousepointer)             | Gibt den Typ des angezeigten Mauszeigers an.<br/>                                                                                             |
| [**Mousebutton**](/windows/desktop/api/inked/ne-inked-mousebutton)                     | Gibt an, welche Maustaste gedrückt wurde.<br/>                                                                                                     |
| [**ScrollBarsConstants**](/windows/desktop/api/inked/ne-inked-scrollbarsconstants)     | Gibt an, wie die Bildlaufleisten eines [InkEdit-Steuerelements](inkedit-control-reference.md) auf dem Bildschirm angezeigt werden.<br/>                                    |
| [**SelAlignmentConstants**](/windows/desktop/api/inked/ne-inked-selalignmentconstants) | Gibt die Ausrichtung des Absatzes relativ zu den Rändern des [InkEdit-Steuerelements](inkedit-control-reference.md) an.<br/>                     |
