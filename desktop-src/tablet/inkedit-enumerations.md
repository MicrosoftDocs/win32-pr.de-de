# <a name="inkedit-enumerations"></a>InkEdit-Enumerationen

Dieser Abschnitt enthält Enumerationen, die zum InkEdit-Steuerelement gehören.



| Enumeration                                            | Beschreibung                                                                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Aussehen von anceconstants**](/windows/desktop/api/inked/ne-inked-appearanceconstants)     | Gibt an, ob das Steuerelement flach oder 3D angezeigt wird.<br/>                                                                                            |
| [**Borderstyleconstants**](/windows/desktop/api/inked/ne-inked-borderstyleconstants)   | Gibt an, ob das Steuerelement über einen Rahmen verfügt.<br/>                                                                                                   |
| [**Inkapplicationgeste**](/windows/desktop/api/msinkaut/ne-msinkaut-inkapplicationgesture) | Gibt das Interesse an einer Reihe von anwendungsspezifischen Gesten an.<br/>                                                                             |
| [**InkDisplayMode**](/windows/desktop/api/inked/ne-inked-inkdisplaymode)               | Gibt an, ob eine Auswahl als frei Hand oder Text angezeigt wird.<br/>                                                                                         |
| [**InkEditStatus**](/windows/desktop/api/inked/ne-inked-inkeditstatus)                 | Gibt an, ob sich das [InkEdit](inkedit-control-reference.md) -Steuerelement im Leerlauf befindet, Freihand sammelt oder frei Hand Eingaben erkennt<br/>                           |
| [**InkInsertMode**](/windows/desktop/api/inked/ne-inked-inkinsertmode)                 | Gibt an, wie frei Hand Eingaben in das [InkEdit](inkedit-control-reference.md) -Steuerelement eingefügt werden.<br/>                                                      |
| [**InkMode**](/windows/desktop/api/inked/ne-inked-inkmode)                             | Gibt die Auflistungs Modus-Einstellungen für gezeichneten frei Hand Eingaben an: gibt an, ob die frei Hand Auflistung deaktiviert ist<br/> |
| [**Inkmoueinschaltfläche**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousebutton)               | Gibt an, welche Maustaste gedrückt wurde.<br/>                                                                                                     |
| [**Inkmousekunden-Zeiger**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousepointer)             | Gibt den Typ des Mauszeigers an, der angezeigt wird.<br/>                                                                                             |
| [**MoU. Button**](/windows/desktop/api/inked/ne-inked-mousebutton)                     | Gibt an, welche Maustaste gedrückt wurde.<br/>                                                                                                     |
| [**Scrollbarskonstanten**](/windows/desktop/api/inked/ne-inked-scrollbarsconstants)     | Gibt an, wie die Bild Lauf leisten eines [InkEdit](inkedit-control-reference.md) -Steuer Elements auf dem Bildschirm angezeigt werden.<br/>                                    |
| [**Selalignmentconstants**](/windows/desktop/api/inked/ne-inked-selalignmentconstants) | Gibt die Ausrichtung des Absatzes relativ zu den Rändern des [InkEdit](inkedit-control-reference.md) -Steuer Elements an.<br/>                     |
