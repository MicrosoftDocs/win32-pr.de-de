# <a name="inkedit-methods"></a>InkEdit-Methoden

Dieser Abschnitt enthält Methoden, die zum InkEdit-Steuerelement gehören.



| Methode                                               | BESCHREIBUNG                                                                                                           |
|------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**GetGestureStatus**](/windows/desktop/api/inked/nf-inked-iinkedit-getgesturestatus) | Ruft das Interesse des [InkEdit](inkedit-control-reference.md) -Steuer Elements in einem bekannten Satz von Gesten ab.<br/> |
| [**Recognize**](/windows/desktop/api/inked/nf-inked-iinkedit-recognize)               | Gibt an, dass eine Erkennung erfolgen soll.<br/>                                                                   |
| [**Aktualisieren**](/windows/desktop/api/inked/nf-inked-iinkedit-refresh)                   | Bewirkt, dass das Steuerelement neu gezeichnet wird.<br/>                                                                              |
| [**SetGestureStatus**](/windows/desktop/api/inked/nf-inked-iinkedit-setgesturestatus) | Ändert das Interesse des [InkEdit](inkedit-control-reference.md) -Steuer Elements in einem bekannten Satz von Gesten.<br/>  |
