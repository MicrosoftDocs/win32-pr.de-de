# <a name="core-tablet-pc-constants"></a>Core-Tablet PC-Konstanten

Dieser Abschnitt enthält die Kern Konstanten für Tablet PC.

## <a name="in-this-section"></a>In diesem Abschnitt



| Konstante                                                 | BESCHREIBUNG                                                                                                                                                                                                                                               |
|----------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Faktoid](factoid-constants.md)                         | Definiert konstante Zeichen folgen Werte, die verwendet werden, um die Erkennungsgenauigkeit durch Bereitstellen von Kontextinformationen für die Erkennung zu erhöhen.<br/>                                                                                                           |
| [Packetpropertyguids](packetpropertyguids-constants.md) | Definiert Werte, die die Paket Eigenschaften angeben. Die Tablet PC-API (Application Programming Interface) verwendet Global Unique Bezeichner (GUIDs), um Paket Eigenschaften zu identifizieren, die in Automation Konstante Zeichen folgen sind.<br/>                     |
| [Erkennungs Eigenschaft](recognitionproperty-constants.md) | Definiert Werte, die die Eigenschaften einer Erkennungs Alternative angeben. Die Tablet PC-API (Application Programming Interface) verwendet Global Unique Bezeichner (GUIDs), um Paket Eigenschaften zu identifizieren, die in Automation Konstante Zeichen folgen sind.<br/> |
