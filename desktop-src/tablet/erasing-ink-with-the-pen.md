# <a name="erasing-ink-with-the-pen"></a>Löschen von frei Hand Eingaben mit dem Stift

Zusätzlich zum Aktivieren von Handschrift und Zeichnung können Sie die Anwendung so erstellen, dass die Verwendung des Tablettstifts zum Löschen von frei Hand Eingaben unterstützt wird. Verwenden Sie hierfür das [**InkOverlay**](inkoverlay-class.md) -Objekt.

In den folgenden Abschnitten wird das Löschen von frei Hand Eingaben mit dem Tablettstift beschrieben.

-   [Löschen mithilfe des InkOverlay-Objekts](erasing-by-using-the-inkoverlay-object.md)
-   [Löschen mithilfe des Stifts](erasing-by-using-the-pen.md)
