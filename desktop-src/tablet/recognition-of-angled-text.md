# <a name="recognition-of-angled-text"></a>Erkennung von Abgewinkelter Text

Die Microsoft-erkennende verarbeiten das Schreiben mit einem Winkel von +/-20 Grad, aber die Genauigkeit wird beeinträchtigt, wenn der Schreibvorgang von der idealen horizontalen abweicht. Für frei Hand Eingaben, die auf mehr als 20 Grad von der horizontalen Achse geschrieben wurden, empfiehlt es sich, dass der Entwickler die frei Hand Eingaben vor der Übergabe an die Erkennung wieder in den horizontalen rotiert.
