# <a name="recognition-of-angled-text"></a>Erkennung von abgewinkelten Texten

Die Microsoft-Erkennungen verarbeiten das Schreiben in einem Winkel von +/- 20 Grad, aber die Genauigkeit nimmt ab, wenn das Schreiben weiter von der idealen horizontalen Abweicht. Für Freihandschrift, die bei mehr als 20 Grad von der horizontalen Achse geschrieben wird, wird empfohlen, dass der Entwickler die Freihand wieder in horizontal dreht, bevor sie an die Erkennung übergeben wird.
