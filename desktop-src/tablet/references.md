# <a name="references-tablet-pc"></a>Verweise (Tablet PC)

Informationen zum Aktivieren der Spracherkennung in Ihrer Anwendung finden Sie im Microsoft Speech SDK, das Sie im [Microsoft Speech SDK](https://www.microsoft.com/speech/download/sdk51/)finden. Diese Website enthält auch eine Übersicht über die Microsoft Speech-API und Richtlinien für den Anwendungs Entwurf.

Informationen zur Barrierefreiheit von Microsoft finden Sie unter [Microsoft-Barrierefreiheit](https://www.microsoft.com/enable/).

Weitere Informationen zu den Microsoft Active Accessibility-Richtlinien für Windows-basierte Anwendungen finden Sie im Abschnitt zur [Barrierefreiheit](../accessibility/accessibility.md) .
