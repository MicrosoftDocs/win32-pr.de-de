# <a name="recognizer-dll-sample-template"></a>Beispielvorlage für die Dll-Recognizer-Datei

Das Beispiel für die Recognizer-DLL enthält eine Vorlage zum Erstellen einer Recognizer-DLL. Die Vorlage enthält Funktionen zum Registrieren und Aufheben der Registrierung des Servers sowie Gerüste für primäre Recognizer-Funktionen.

Diese Vorlage zeigt, wie eine CLSID in der Registrierung registriert wird. Sie können dies auch tun, wenn Sie die Recognizer mit dem Setupprogramm installieren. Stellen Sie sicher, dass Sie Ihre eigene eindeutige CLSID für Ihre Sprachen erstellen und die im Beispiel gezeigte CLSID nicht wiederverwenden.

Standardmäßig werden die Beispiele im Verzeichnis Samples unter dem Verzeichnis installiert, in dem Sie das SDK installiert haben.

Ein Beispiel für die Verwendung von Erkennungen aus der Tablet PC-Plattform-API finden Sie im [Beispiel für erweiterte Erkennung.](advanced-recognition-sample.md)
