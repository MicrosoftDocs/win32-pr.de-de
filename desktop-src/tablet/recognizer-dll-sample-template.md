# <a name="recognizer-dll-sample-template"></a>Beispiel Vorlage für Erkennungs Modul-dll

Das Erkennungs Modul-dll-Beispiel enthält eine Vorlage zum Erstellen einer Erkennungs-DLL. Die Vorlage enthält Funktionen zum Registrieren und Aufheben der Registrierung des Servers sowie zu den Skeletten für primäre Erkennungsfunktionen.

Diese Vorlage zeigt, wie Sie eine CLSID in der Registrierung registrieren. Dies können Sie auch tun, wenn Sie Ihre Erkennung mithilfe des Setup Programms installieren. Stellen Sie sicher, dass Sie eine eigene eindeutige CLSID für Ihre Sprachen erstellen und die im Beispiel gezeigte CLSID nicht wieder verwenden.

Standardmäßig werden die Beispiele im Verzeichnis "Samples" unter dem Verzeichnis installiert, in dem Sie das SDK installiert haben.

Ein Beispiel für die Verwendung von erkenzern aus der Tablet PC Platform-API finden Sie im Beispiel für die [erweiterte Erkennung](advanced-recognition-sample.md).
