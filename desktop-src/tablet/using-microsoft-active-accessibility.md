# <a name="using-microsoft-active-accessibility"></a>Verwenden von Microsoft Active Accessibility

Die Spracherkennungs-Engine auf dem Tablet PC verwendet Active Accessibility, um Informationen direkt aus dem Objekt abzurufen und bei Änderungen der Objekte benachrichtigt zu werden. Diese Schnittstelle wird von den standardmäßigen Microsoft Windows-Steuerelementen unterstützt und kann auch von benutzerdefinierten Steuerelementen und benutzerdefinierten Fenster Klassen verwendet werden.
