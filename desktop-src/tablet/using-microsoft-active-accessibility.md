# <a name="using-microsoft-active-accessibility"></a>Verwenden Microsoft Active Accessibility

Die Spracherkennungs-Engine auf dem Tablet PC verwendet Active Accessibility, um Informationen direkt aus dem Objekt abzurufen und benachrichtigt zu werden, wenn sich Objekte ändern. Diese Schnittstelle wird von den standardmäßigen Microsoft Windows-Steuerelementen unterstützt und kann auch von benutzerdefinierten Steuerelementen und benutzerdefinierten Fensterklassen verwendet werden.
