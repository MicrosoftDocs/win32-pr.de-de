# <a name="ianalysisregionmakeinfinite-method"></a>IAnalysisRegion::MakeInfinite-Methode

```C++
HRESULT MakeInfinite();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.
