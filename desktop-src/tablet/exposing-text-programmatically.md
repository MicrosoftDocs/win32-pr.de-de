# <a name="exposing-text-programmatically"></a>Programm gesteuertes verfügbar machen von Text

Die programmgesteuerte Offenlegung von Text bietet einen großen Vorteil für hilfshilfen, indem er eine viel zuverlässigere Bereitstellung bietet und in einigen Fällen eine umfassendere Interpretation der Informationen bietet.

Es gibt mehrere grundlegende Methoden zum verfügbar machen von Objekten, um Textprogramm gesteuert verfügbar zu machen. Allerdings ist keine Methode mit allen Barrierefreiheits Hilfen wirksam, daher ist es auch erforderlich, Text auf dem Bildschirm zu zeichnen, damit er erkannt werden kann. Weitere Informationen zum verfügbar machen von Text für große Mengen finden [Sie unter Verwenden von Active Accessibility zur](using-active-accessibility-to-expose-text.md)Offenlegung von Text.
