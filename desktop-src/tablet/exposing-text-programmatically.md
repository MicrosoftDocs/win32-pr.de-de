# <a name="exposing-text-programmatically"></a>Programmgesteuertes Verfügbarmachen von Text

Das programmgesteuerte Verfügbarmachen von Text bietet einen großen Vorteil für Barrierefreiheitshilfen, da eine wesentlich zuverlässigere und in einigen Fällen umfangreichere Interpretation der Informationen ermöglicht wird.

Es gibt mehrere grundlegende Methoden zum verfügbarmachen von Objekten, um Text programmgesteuert verfügbar zu machen. Allerdings ist keine Methode bei allen Barrierefreiheitshilfen effektiv, daher ist es auch erforderlich, Text auf dem Bildschirm zu zeichnen, damit er erkannt werden kann. Informationen zum Verfügbarmachen großer Mengen von Text finden Sie unter [Verwenden von Active Accessibility zum Verfügbarmachen von Text.](using-active-accessibility-to-expose-text.md)
