# <a name="adding-ink-controls-to-a-project"></a>Hinzufügen von Ink-Steuerelementen zu einem Projekt

So fügen Sie einem Microsoft Visual Basic .net-oder Microsoft Visual C-Projekt ein [InkEdit](inkedit-control-reference.md) -oder [InkPicture](inkpicture-control-reference.md) -Steuerelement hinzu \# :

1.  Klicken Sie mit der rechten Maustaste in die Toolbox, und klicken Sie dann auf **Toolbox anpassen**.
2.  Klicken Sie im Dialogfeld **Toolbox anpassen** auf die Registerkarte **.NET Framework Komponenten** .
3.  Wählen Sie im Listenfeld die Steuerelemente InkEdit und InkPicture aus, und klicken Sie dann auf **OK**. Die Steuerelemente werden der Toolbox auf der Registerkarte **Komponenten** hinzugefügt.
4.  Wählen Sie das gewünschte Steuerelement aus der Toolbox aus, und ziehen Sie das Steuerelement dann an die Stelle, an der es angezeigt werden soll.

> [!Note]  
> Die von Tablet PC verwaltete Bibliothek kann websteuer Elemente in teilweise vertrauenswürdigen Umgebungen unterstützen, nämlich die Ausführung verwalteter Assemblys von Webseiten. Frei Hand fähige ActiveX-Steuerelemente auf Webseiten werden jedoch nicht unterstützt.
