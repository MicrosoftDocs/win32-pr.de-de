# <a name="recognizer-stroke-reordering-support"></a>Unterstützung für das Unterzeichnen von Stroke-Befehlen

Die Microsoft-Erkennens der lateinischen Skripts unterstützen die Neuanordnung von Hubs.

Die Microsoft-Handschrifterkennung von lateinischen Skripts ist nicht abhängig von den Strichen in einer Zeile, die in einer bestimmten Reihenfolge geschrieben wird. Stattdessen ordnen die erkennungsstriche die Striche intern basierend auf räumlichen Beziehungen zwischen den Strichen an. Beispielsweise unterstützen die Erkennungs Tools des lateinischen Skripts das gängige Szenario, in dem jemand eine Zeile zurückgibt, um eine linke Klammer ("(") zu schreiben oder Anführungszeichen hinzuzufügen.
