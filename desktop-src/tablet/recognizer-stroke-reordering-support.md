# <a name="recognizer-stroke-reordering-support"></a>Unterstützung für die Neuanordnung von Erkennungsstrichen

Die Microsoft-Erkennungen lateinischer Skripts unterstützen die Neuanordnung von Strichen.

Die Microsoft-Handschrifterkennungen des lateinischen Skripts hängen nicht von den Strichen in einer Zeile ab, die in einer bestimmten Reihenfolge geschrieben wird. Stattdessen ordnen die Erkennungen die Striche intern basierend auf räumlichen Beziehungen zwischen den Strichen an. Die Erkennungen des lateinischen Skripts unterstützen z. B. das allgemeine Szenario, in dem jemand auf eine Zeile zurückgeht, um eine linke Klammer ("" zu schreiben oder Anführungszeichen hinzuzufügen).
