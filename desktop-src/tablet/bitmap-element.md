# <a name="bitmap-element"></a>Bitmap-Element

Enthält die Base64-codierten Daten für eine Bitmap, die in der Journaldatei als Hintergrundbild enthalten ist.

## <a name="definition"></a>Definition

``` syntax
<xs:element name="Bitmap" type="xs:base64Binary" minOccurs="0" />
```

## <a name="parent-elements"></a>Übergeordnete Elemente

[**Image**](image-element.md)

## <a name="child-elements"></a>Untergeordnete Elemente

Keine

## <a name="attributes"></a>Attribute

Keine

## <a name="element-information"></a>Elementinformationen



|                  | Wert                                      |
|------------------|--------------------------------------------|
| **Elementtyp** | xs:base64Binary                            |
| **Namespace**    | urn:schemas-microsoft-com:tabletpc:richink |
| **Schemaname**  | Journalreader                             |
