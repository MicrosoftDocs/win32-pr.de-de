# <a name="inkedit-control-events"></a>InkEdit-Steuerelementereignisse

In der folgenden Tabelle wird beschrieben, für welche Threads die [InkEdit-Steuerelementereignisse](inkedit-control-reference.md) ausgeführt werden können.



| Ereignis                                                                          | Threads                                                          |
|--------------------------------------------------------------------------------|------------------------------------------------------------------|
| [**Change**](inkedit-change.md)                                               | Wird auf dem Benutzeroberflächenthread der Anwendung ausgeführt.<br/> |
| [**Klicken**](inkedit-click.md)                                                 | Wird im UI-Thread der Anwendung ausgeführt.<br/>                  |
| [**Dblclick**](inkedit-dblclick.md)                                           | Wird im UI-Thread der Anwendung ausgeführt.<br/>                  |
| [**Geste**](inkedit-gesture.md)                                             | Wird im UI-Thread der Anwendung ausgeführt.<br/>                  |
| [**Keydown**](inkedit-keydown.md)                                             | Wird im UI-Thread der Anwendung ausgeführt.<br/>                  |
| [**Keypress**](inkedit-keypress.md)                                           | Wird im UI-Thread der Anwendung ausgeführt.<br/>                  |
| [**Keyup**](inkedit-keyup.md)                                                 | Wird im UI-Thread der Anwendung ausgeführt.<br/>                  |
| [**Mousedown**](inkedit-mousedown.md)                                         | Wird im UI-Thread der Anwendung ausgeführt.<br/>                  |
| [**Mousemove**](inkedit-mousemove.md)                                         | Wird im UI-Thread der Anwendung ausgeführt.<br/>                  |
| [**Mouseup**](inkedit-mouseup.md)                                             | Wird im UI-Thread der Anwendung ausgeführt.<br/>                  |
| [**Erkennung**](/previous-versions/ms567627(v=vs.100)) (nur verwaltete Bibliothek). | Wird im UI-Thread der Anwendung ausgeführt.<br/>                  |
| [**Recognitionresult**](inkedit-recognitionresult.md)                         | Wird im UI-Thread der Anwendung ausgeführt.<br/>                  |
| [**SelChange**](inkedit-selchange.md)                                         | Wird im UI-Thread der Anwendung ausgeführt.<br/>                  |
| [**Takt**](inkedit-stroke.md)                                               | Wird im UI-Thread der Anwendung ausgeführt.<br/>                  |
