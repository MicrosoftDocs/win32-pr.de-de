# <a name="ink-analysis-enumerations"></a>Enumerationen der Ink-Analyse

Dieser Abschnitt enthält Informationen zu den Enumerationen, die bei der Ink-Analyse verwendet werden.

## <a name="enumerations"></a>Enumerationen



| Enumeration                                              | Beschreibung                                                                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [**Analysismodes**](analysismodes.md)                   | Gibt an, wie [**der IInkAnalyzer**](iinkanalyzer.md) eine Ink-Analyse durchführt.<br/>                                                  |
| [**Analysiswarningcode**](analysiswarningcode.md)       | Gibt den Satz verfügbarer Warnungen an, die während der Freik-Analyse auftreten können.<br/>                                                    |
| [**ConfirmationType**](confirmationtype.md)             | Gibt den Typ der Bestätigung an, die für ein [**IContextNode-Objekt auftreten**](icontextnode.md) kann.<br/>                           |
| [**Contextlinkdirection**](contextlinkdirection.md)     | Gibt die Typen von [**IContextLink-Objekten**](icontextlink.md) an.<br/>                                                           |
| [**RecognitionConfidence**](recognitionconfidence.md)   | Gibt den Grad der Zuverlässigkeit an, den [**der IInkAnalyzer**](iinkanalyzer.md) in der Genauigkeit des Erkennungsergebnis hat.<br/> |
| [**RecognizerCapabilities**](recognizercapabilities.md) | Gibt die Attribute eines [**IInkAnalysisRecognizer an.**](iinkanalysisrecognizer.md)<br/>                                       |
| [**Stroketype**](stroketype.md)                         | Gibt an, wie [**der IInkAnalyzer**](iinkanalyzer.md) einen Strich während der Analyse klassifizieren soll.<br/>                               |
