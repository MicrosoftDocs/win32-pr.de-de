<span data-ttu-id="c614e-105">Die [**RealTimeStylus**](realtimestylus-class.md) -Klasse ermöglicht es Ihnen, mit dem Datenstrom vom Tablettstift zu interagieren.</span><span class="sxs-lookup"><span data-stu-id="c614e-105">The [**RealTimeStylus**](realtimestylus-class.md) class enables you to interact with the data stream from the tablet pen.</span></span> <span data-ttu-id="c614e-106">Wenn Sie mit dem Datenstrom interagieren möchten, fügen Sie der Anwendung ein **RealTimeStylus** -Objekt hinzu, und fügen Sie dem **RealTimeStylus** -Objekt Plug-Ins hinzu.</span><span class="sxs-lookup"><span data-stu-id="c614e-106">To interact with the data stream, add a **RealTimeStylus** object to your application, and add plug-ins to the **RealTimeStylus** object.</span></span>

<span data-ttu-id="c614e-107">Die Plug-Ins können die Daten ändern, die mit in-Air-Paketen, tablettdown-, Paket-und tablettstiftbenachrichtigungs Methoden verknüpft sind.</span><span class="sxs-lookup"><span data-stu-id="c614e-107">The plug-ins can modify the data associated with in-air packets, stylus down, packets, and stylus up notification methods.</span></span> <span data-ttu-id="c614e-108">Mit den Plug-Ins können die in-Air-Pakete und Benachrichtigungs Methoden für Pakete abgebrochen werden.</span><span class="sxs-lookup"><span data-stu-id="c614e-108">The plug-ins can cancel the in-air packets and packets notification methods.</span></span> <span data-ttu-id="c614e-109">Die Plug-Ins können dem Stream auch Anwendungsdaten in Form von [CustomStylusData](/previous-versions/ms575208(v=vs.100)) -Objekten hinzufügen.</span><span class="sxs-lookup"><span data-stu-id="c614e-109">The plug-ins can also add application data to the stream in the form of [CustomStylusData](/previous-versions/ms575208(v=vs.100)) objects.</span></span> <span data-ttu-id="c614e-110">Die folgende Liste enthält Ideen für gängige Kategorien von Plug-ins, die Sie möglicherweise verwenden oder erstellen möchten.</span><span class="sxs-lookup"><span data-stu-id="c614e-110">The following list offers ideas for common categories of plug-ins that you may want to use or create.</span></span>

-   <span data-ttu-id="c614e-111">Filter-Plug-in: ein Objekt, das Daten im Tablet Pen-Datenstrom selektiv entfernt oder abbricht.</span><span class="sxs-lookup"><span data-stu-id="c614e-111">Filter plug-in: An object that selectively removes or cancels data in the tablet pen data stream.</span></span>
-   <span data-ttu-id="c614e-112">Modifiziererplug-in: ein Objekt, das Daten im Tablet Pen-Datenstrom selektiv ändert.</span><span class="sxs-lookup"><span data-stu-id="c614e-112">Modifier plug-in: An object that selectively modifies data in the tablet pen data stream.</span></span>
-   <span data-ttu-id="c614e-113">Dynamic-Renderer-Plug-in: ein Objekt, das die Tablet Pen-Daten in Echtzeit anzeigt, wie Sie vom [**RealTimeStylus**](realtimestylus-class.md) -Objekt verarbeitet werden.</span><span class="sxs-lookup"><span data-stu-id="c614e-113">Dynamic-renderer plug-in: An object that displays the tablet pen data in real-time as it is being handled by the [**RealTimeStylus**](realtimestylus-class.md) object.</span></span> <span data-ttu-id="c614e-114">Für Ereignisse wie z. b. eine Formular Aktualisierung könnte das dynamische Renderer-Plug-in oder ein frei Hand Eingabe-Plug-in das frei Handzeichen neu zeichnen.</span><span class="sxs-lookup"><span data-stu-id="c614e-114">Later, for events such as a form refresh, the dynamic renderer plug-in or an ink collection plug-in might redraw the ink.</span></span>
-   <span data-ttu-id="c614e-115">Erkennungs-Plug-in: ein Objekt, das die Bewegung des Tablettstifts für Gesten, Handschrift oder andere Symbole scannt.</span><span class="sxs-lookup"><span data-stu-id="c614e-115">Recognizer plug-in: An object that scans the movement of the tablet pen for gestures, handwriting, or other glyphs.</span></span>
-   <span data-ttu-id="c614e-116">Ink-Collector-Plug-in: ein Objekt, das vom Tablet Pen Data Stream erstellt und gespeichert wird.</span><span class="sxs-lookup"><span data-stu-id="c614e-116">Ink-collector plug-in: An object that from the tablet pen data stream creates and stores ink.</span></span>
-   <span data-ttu-id="c614e-117">Wrapper-Plug-in: ein Plug-in, das als Schnittstelle zwischen dem [**RealTimeStylus**](realtimestylus-class.md) -Objekt und einem anderen Plug-in oder Objekt fungiert, um das Verhalten des Wrapper Objekts zu ändern.</span><span class="sxs-lookup"><span data-stu-id="c614e-117">Wrapper plug-in: A plug-in that acts as an interface between the [**RealTimeStylus**](realtimestylus-class.md) object and another plug-in or object as a way of modifying the behavior of the wrapped object.</span></span>

<span data-ttu-id="c614e-118">Sowohl Dynamic-Renderer-als auch Freihand-Auflistungs-Plug-Ins können erstellt werden, um Sie in verschiedenen Kontexten zu rendern – z. b. in einer Datei, einem Stream oder einem Anzeigegerät.</span><span class="sxs-lookup"><span data-stu-id="c614e-118">Both dynamic-renderer and ink-collection plug-ins can be created to render to various contexts—such as to a file, a stream, or to a display device.</span></span> <span data-ttu-id="c614e-119">Frei Hand Eingaben können auch in verschiedenen Formaten gespeichert werden, z. b. in einem [Ink](/previous-versions/aa515768(v=msdn.10)) -Objekt, einer GIF-Datei (angereichert Graphics Interchange Format), einer ISF-Datei (Ink Serialized Format) oder anderen Formaten.</span><span class="sxs-lookup"><span data-stu-id="c614e-119">Ink can also be stored in various formats, such as an [Ink](/previous-versions/aa515768(v=msdn.10)) object, a fortified Graphics Interchange Format (GIF) file, an Ink Serialized Format (ISF) file, or other formats.</span></span>

<span data-ttu-id="c614e-120">Mit den StylusInput-APIs werden zwei Plug-ins bereitgestellt: die [**DynamicRenderer**](/previous-versions/windows/desktop/legacy/ms701168(v=vs.85)) -Klasse und die [**GestureRecognizer**](gesturerecognizer-class.md) -Klasse.</span><span class="sxs-lookup"><span data-stu-id="c614e-120">Two plug-ins are provided with the StylusInput APIs: the [**DynamicRenderer**](/previous-versions/windows/desktop/legacy/ms701168(v=vs.85)) class and the [**GestureRecognizer**](gesturerecognizer-class.md) class.</span></span> <span data-ttu-id="c614e-121">Die **DynamicRenderer** -Klasse stellt das grundlegende Rendering der frei Hand Daten in Echtzeit bereit und ist so optimiert, dass Sie eine minimale Auswirkung auf die Leistung haben.</span><span class="sxs-lookup"><span data-stu-id="c614e-121">The **DynamicRenderer** class provides basic rendering of the ink data in real time and is streamlined to have a minimal performance impact.</span></span> <span data-ttu-id="c614e-122">Die **GestureRecognizer** -Klasse stellt Gestenerkennung für die [**RealTimeStylus**](realtimestylus-class.md) -Klasse bereit.</span><span class="sxs-lookup"><span data-stu-id="c614e-122">The **GestureRecognizer** class provides gesture recognition for the [**RealTimeStylus**](realtimestylus-class.md) class.</span></span>
