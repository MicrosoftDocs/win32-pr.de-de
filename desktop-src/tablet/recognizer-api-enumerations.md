# <a name="recognizer-enumerations"></a>Erkennungsenumerationen

In diesem Abschnitt werden die Recognizer-Enumerationen beschrieben.

## <a name="in-this-section"></a>In diesem Abschnitt



| Enumeration                                   | Beschreibung                                                                                |
|-----------------------------------------------|--------------------------------------------------------------------------------------------|
| [**\_ALT-UMBRÜCHE**](/windows/win32/api/rectypes/ne-rectypes-alt_breaks)             | Gibt an, wie Alternativen aus einer Zeichenfolge mit dem besten Ergebnis erstellt werden.<br/>                   |
| [**\_KONFIDENZSTUFE**](/windows/win32/api/rectypes/ne-rectypes-confidence_level) | Gibt den Vertrauensgrad an, den die Erkennung im Erkennungsergebnis hat.<br/> |
| [**\_ZEILENMETRIKEN**](/windows/win32/api/rectypes/ne-rectypes-line_metrics)         | Stellt die in einem Segment gefundenen Zeilen dar.<br/>                                        |
| [**\_EIGENSCHAFTENEINHEITEN**](/windows/desktop/api/tpcshrd/ne-tpcshrd-property_units)     | Gibt die Maßeinheit einer Eigenschaft an.<br/>                                |
