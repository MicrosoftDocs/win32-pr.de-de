# <a name="inkwordtype-complex-type"></a>Komplexer inkwordtype-Typ

Definiert den Typ, der die frei Hand Daten, die Alternativen und das Vertrauen für frei Hand Inhalte enthält, bei denen es sich um ein [InkWord-Element](inkword-element.md) handelt (im Gegensatz zu einem [Zeichnungs Element](drawing-element.md)).

``` syntax
<xs:complexType name="InkWordType" />
```
