# <a name="whats-new-in-tablet-pc-development"></a>Neues bei der Entwicklung von Tablet-PCs

In diesem Abschnitt wird beschrieben, was neu bei der Entwicklung von Tablet PC Technology ist.

## <a name="tablet-pc-development-kit-now-in-windows-sdk"></a>Tablet PC Development Kit jetzt im Windows SDK

Ab dem Jahr 2006 und in Zukunft ist das Windows SDK der einzige Verteilungsmechanismus für die neueste Tablet PC-Technologie. Es gibt kein eigenständiges Tablet PC SDK mehr. Die Plattformbinärdateien, Header, Bibliotheken, Dokumentation und Beispielcode sind im Windows SDK verfügbar. Diese Distribution umfasst die Binärdateien der Tablet PC-Plattform 1.7 und die verteilbaren Mergemodule für diese Binärdateien sowie dokumentation für die Binärdateien der Version 1.7 und die neuen Binärdateien der Windows Vista Tablet PC Technology Platform (Version 6).

Weitere Informationen [zum Einrichten Ihres Computers für](the-development-environment.md) die Tablet PC-Entwicklung finden Sie unter The Development Environment (Entwicklungsumgebung).

## <a name="windows-server-2008-r2-now-supports-handwriting-recognition-and-custom-dictionaries"></a>Windows Server 2008 R2 unterstützt jetzt handschriftliche Erkennung und benutzerdefinierte Wörterbücher.

Windows Server 2008 R2 kann jetzt Die Ink-Erkennung durchführen, und es wurden benutzerdefinierte Wörterbuchfeatures hinzugefügt, um neue Erkennungsszenarien zu unterstützen.

Weitere Informationen finden Sie unter Handschrifterkennung in Windows Server [](creating-custom-dictionaries-for-handwriting-recognition-in-windows-7-and-windows-server-2008-r2.md) [2008 R2](handwriting-recognition-in-windows-server-2008-r2.md) und Erstellen benutzerdefinierter Wörterbücher für die Handschrifterkennung.

Für die Tablet PC Technology-Plattform wurden neue Features für die Ink-Analyse erstellt. Konzeptionelle [Details finden Sie unter Übersicht](ink-analysis-overview.md) über die Ink-Analyse.

## <a name="stylus-input-api-com-version"></a>COM-Version der Stifteingabe-API

Die Eingabe-API für Stifte ist jetzt in einer COM-Version verfügbar. Die COM-Version der API-Referenzthemen finden Sie in der [RealTimeStylus-Referenz.](realtimestylus-reference.md) Konzeptionelle [Informationen zur Eingabe-API](accessing-and-manipulating-stylus-input.md) für Stifte finden Sie unter Zugreifen auf und Bearbeiten von Stifteingaben.

## <a name="textinputpanel-class"></a>TextInputPanel-Klasse

Die [TextInputPanel-Klasse](/previous-versions/ms573640(v=vs.100)) ist als anfingbares Objekt verfügbar, sodass Sie vorhandenen Steuerelementen Tablet PC-Eingabebereichsfunktionen hinzufügen können. Die Benutzeroberfläche wird größtenteils von der aktuellen Eingabesprache beauftragt. Sie haben die Möglichkeit, entweder Handschrift oder Tastatur als Standardeingabemethode für das TextInputPanel-Objekt zu wählen. Der Endbenutzer kann mithilfe von Schaltflächen auf der Benutzeroberfläche zwischen Eingabemethoden wechseln.
