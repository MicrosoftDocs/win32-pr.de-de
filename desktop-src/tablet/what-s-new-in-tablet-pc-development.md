# <a name="whats-new-in-tablet-pc-development"></a>Neues bei der Tablet PC-Entwicklung

In diesem Abschnitt werden die Neuerungen bei der Entwicklung von Tablet PC-Technologien beschrieben.

## <a name="tablet-pc-development-kit-now-in-windows-sdk"></a>Tablet PC Development Kit jetzt in Windows SDK

Ab dem Spring of 2006 ist der Windows SDK der einzige Verteilungsmechanismus für die neueste Tablet PC-Technologie. Es ist kein eigenständiges Tablet PC-SDK mehr vorhanden. Die Platt Form Binärdateien, Header, Bibliotheken, Dokumentationen und Beispielcode sind im Windows SDK verfügbar. Diese Verteilung umfasst die Binärdateien der 1,7-Tablet PC-Plattform und die verteilbaren Mergemodule für diese Binärdateien sowie die Dokumentation für die Binärdateien der Version 1,7 und die neuen Windows Vista Tablet PC Technology Platform (Version 6)-Binärdateien.

Weitere Informationen zum Einrichten des Computers für die Tablet PC-Entwicklung finden Sie in [der Entwicklungsumgebung](the-development-environment.md) .

## <a name="windows-server-2008-r2-now-supports-handwriting-recognition-and-custom-dictionaries"></a>Windows Server 2008 R2 unterstützt jetzt die Handschrifterkennung und benutzerdefinierte Wörterbücher

Windows Server 2008 R2 kann nun frei Handerkennung durchführen, und es wurden benutzerdefinierte Wörterbuch Features hinzugefügt, die neue Erkennungs Szenarien unterstützen.

Weitere Informationen finden Sie [unter Handschrifterkennung in Windows Server 2008 R2](handwriting-recognition-in-windows-server-2008-r2.md) und [Erstellen von benutzerdefinierten Wörterbüchern für die Handschrifterkennung](creating-custom-dictionaries-for-handwriting-recognition-in-windows-7-and-windows-server-2008-r2.md) .

Die mathematische Eingabe-API ist jetzt in einer com-Version verfügbar. Weitere Informationen finden Sie unter [Programmieren des mathematischen Eingabe Steuer](programming-the-math-input-control.md) Elements und der [Referenz zu mathematischen Eingabe Steuer](math-input-control-reference.md) Elementen.

Für die Tablet PC-Technologieplattform wurden neue Funktionen zur frei Hand Analyse erstellt. Konzeptionelle Details finden Sie unter [Übersicht über](ink-analysis-overview.md) die frei Hand Analyse.

Die Stift Eingabe-API ist jetzt in einer com-Version verfügbar. Weitere Informationen zur com-Version der API-Referenz Themen finden Sie unter [RealTimeStylus-Referenz](realtimestylus-reference.md) . Weitere Informationen finden Sie unter Zugreifen auf und Bearbeiten von [Stift Eingaben](accessing-and-manipulating-stylus-input.md) für konzeptionelle Eingabe-API-Informationen.

## <a name="textinputpanel-class"></a>TextInputPanel-Klasse

Die [TextInputPanel](/previous-versions/ms573640(v=vs.100)) -Klasse steht als anfügbares Objekt zur Verfügung, sodass Sie vorhandenen Steuerelementen Tablet PC-Eingabe Bereichs Funktionen hinzufügen können. Die Benutzeroberfläche wird größtenteils von der aktuellen Eingabe Sprache vorgeschrieben. Sie haben die Möglichkeit, entweder Handschrift oder Tastatur als Standardeingabe Methode für das TextInputPanel-Objekt auszuwählen. Der Endbenutzer kann mithilfe von Schaltflächen auf der Benutzeroberfläche zwischen Eingabemethoden wechseln.
