# <a name="text-input-panel-reference"></a>Referenz zum Texteingabebereich

Bietet Zugriff auf die programmierbaren Funktionen des Tablet PC-Eingabebereichs.

## <a name="in-this-section"></a>In diesem Abschnitt

-   [Texteingabebereich– Meldungen](tip-messages.md)
-   [Texteingabebereichsenumerungen](text-input-panel-enumerations.md)
-   [Schnittstellen des Texteingabebereichs](text-input-panel-interfaces.md)

## <a name="com-implementation"></a>COM-Implementierung

Das Thema enthält Verweise auf die COM-Schnittstellen und -Enumerationen des Texteingabebereichs.

Diese Schnittstellen können durch Aufrufen der [**CoCreateInstance-Methode**](/windows/win32/api/combaseapi/nf-combaseapi-cocreateinstance) in C++ instanziiert werden.

## <a name="remarks"></a>Hinweise

Die Schnittstellen und Enumerationen des Texteingabebereichs werden verwendet, um die Darstellung und das Verhalten des Eingabebereichs zu steuern.

Diese Elemente werden in Peninputpanel.h deklariert.
