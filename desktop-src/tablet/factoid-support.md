# <a name="factoid-support"></a>Factoid-Unterstützung

*Factoids* verbessern die Erkennungsgenauigkeit, indem sie der Erkennung eine Definition und die Reihenfolge der erwarteten Wörter, Zahlen und Satzzeichen bereitstellen.

Weitere Informationen zu Factoiden finden Sie unter [Verwenden von Kontext zur Verbesserung der Genauigkeit.](using-context-to-improve-accuracy.md)
