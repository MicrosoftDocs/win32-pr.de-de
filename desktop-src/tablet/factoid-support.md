# <a name="factoid-support"></a>Unterstützung von Faktoids

*Faktoide* verbessern die Erkennungsgenauigkeit, indem der Erkennungs Modul eine Definition und die Reihenfolge der erwarteten Wörter, Ziffern und Interpunktions Zeichen bereitgestellt werden.

Weitere Informationen zu Faktoiden finden Sie unter [Verwenden von Kontext zur Verbesserung der Genauigkeit](using-context-to-improve-accuracy.md).
