# <a name="canreclassify-element"></a>CanReClassify-Element

True **gibt** an, dass die Ink-Datei erneut analysiert werden kann. Dieses Element ist **false,** wenn der Benutzer den Typ der Ink-Datei explizit mithilfe der Features Convert Handwriting to Text (Handschrift in Text konvertieren) oder Change Shape To (Form ändern in) von Journal festgelegt hat.

## <a name="definition"></a>Definition

``` syntax
<xs:element name="CanReclassify" type="xs:boolean" />
```

## <a name="child-elements"></a>Untergeordnete Elemente

Keine
