# <a name="inkpicture-methods"></a>InkPicture-Methoden

Dieser Abschnitt enthält Methoden für das InkPicture-Steuerelement.



| Methode                                                                                   | Beschreibung                                                                                                                                                                      |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetEventInterest-Methode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-geteventinterest)                           | Ruft einen Wert ab, der angibt, ob das [InkPicture-Steuerelement](inkpicture-control-reference.md) für ein bestimmtes Ereignis von Interesse ist.<br/>                               |
| [**GetGestureStatus**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-getgesturestatus)                                  | Ruft einen Wert ab, der angibt, ob das [InkPicture-Steuerelement](inkpicture-control-reference.md) an einer bestimmten Anwendungsgeste interessiert ist.<br/>                 |
| [**GetWindowInputRectangle-Methode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-getwindowinputrectangle)             | Gibt das Fensterrechteck in Pixel zurück, innerhalb dessen Ink gezeichnet wird.<br/>                                                                                                   |
| [**HitTestSelection**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-hittestselection)                                  | Ruft einen Member der [**SelectionHitResult-Enumeration**](/windows/desktop/api/msinkaut/ne-msinkaut-selectionhitresult) ab, der angibt, welcher Teil einer Auswahl ggf. während eines Treffertests erreicht wurde.<br/> |
| [**SetAllTabletsMode-Methode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-setalltabletsmode)                         | Aktiviert das [InkPicture-Steuerelement](inkpicture-control-reference.md) zum Erfassen von Inkk von jedem Tablet, das an den Tablet PC angefügt ist.<br/>                                          |
| [**SetEventInterest-Methode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-seteventinterest)                           | Ändert einen Wert, der angibt, ob ein [InkPicture-Steuerelement](inkpicture-control-reference.md) an einem angegebenen Ereignis interessiert ist.<br/>                                  |
| [**SetGestureStatus-Methode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-setgesturestatus)                           | Ändert das Interesse des [InkPicture-Objekts](inkpicture-control-reference.md) in einer angegebenen Anwendungsgeste.<br/>                                                |
| [**SetSingleTabletIntegratedMode-Methode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-setsingletabletintegratedmode) | Ändert das [InkPicture-Steuerelement,](inkpicture-control-reference.md) um Nur-Ink-Dateien von einem Tablet zu erfassen, das an den Tablet-PC angefügt ist. Ink von anderen Tablets wird ignoriert.<br/> |
| [**SetWindowInputRectangle-Methode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-setwindowinputrectangle)             | Gibt das Fensterrechteck an, das in Fensterkoordinaten festgelegt werden soll, in dem Ink gezeichnet wird.<br/>                                                                              |
