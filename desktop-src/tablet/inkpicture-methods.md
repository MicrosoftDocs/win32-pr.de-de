# <a name="inkpicture-methods"></a>InkPicture-Methoden

Dieser Abschnitt enthält Methoden für das InkPicture-Steuerelement.



| Methode                                                                                   | BESCHREIBUNG                                                                                                                                                                      |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Geteventinterest-Methode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-geteventinterest)                           | Ruft einen Wert ab, der angibt, ob das [InkPicture](inkpicture-control-reference.md) -Steuerelement an einem bestimmten Ereignis interessiert ist.<br/>                               |
| [**GetGestureStatus**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-getgesturestatus)                                  | Ruft einen Wert ab, der angibt, ob das [InkPicture](inkpicture-control-reference.md) -Steuerelement an einer bestimmten Anwendungs Geste interessiert ist.<br/>                 |
| [**Getwindowinputrechteck-Methode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-getwindowinputrectangle)             | Gibt das Fenster Rechteck in Pixel zurück, in dem frei Hand Eingaben gezeichnet werden.<br/>                                                                                                   |
| [**HitTestSelection**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-hittestselection)                                  | Ruft einen Member der [**SelectionHitResult**](/windows/desktop/api/msinkaut/ne-msinkaut-selectionhitresult) -Enumeration ab, der angibt, welcher Teil einer Auswahl, falls vorhanden, während eines Treffer Tests getroffen wurde.<br/> |
| [**Setalltablezmode-Methode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-setalltabletsmode)                         | Ermöglicht dem [InkPicture](inkpicture-control-reference.md) -Steuerelement, frei Hand Eingaben von jedem Tablet zu erfassen, das an den Tablet PC angeschlossen ist<br/>                                          |
| [**""-Methode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-seteventinterest)                           | Ändert einen Wert, der angibt, ob ein [InkPicture](inkpicture-control-reference.md) -Steuerelement an einem angegebenen Ereignis interessiert ist.<br/>                                  |
| [**SetGestureStatus-Methode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-setgesturestatus)                           | Ändert das Interesse des [InkPicture](inkpicture-control-reference.md) -Objekts in einer angegebenen Anwendungs Geste.<br/>                                                |
| [**SetSingleTabletIntegratedMode-Methode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-setsingletabletintegratedmode) | Ändert das [InkPicture](inkpicture-control-reference.md) -Steuerelement, um frei Hand Eingaben von nur einem Tablet zu erfassen, das an den Tablet PC angeschlossen ist. Frei Hand Eingaben von anderen Tablets werden ignoriert.<br/> |
| [**Setwindowinputrechteck-Methode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-setwindowinputrectangle)             | Gibt das Fenster Rechteck an, das in Fenster Koordinaten festgelegt werden soll, in dem frei Hand Eingaben gezeichnet werden.<br/>                                                                              |
