# <a name="flicks-structures"></a>Flicks-Strukturen

Dieser Abschnitt enthält die Strukturen, die in der [**WM \_ TABLET \_ FLICK Message verwendet werden.**](wm-tablet-flick-message.md)



| Struktur                           | Beschreibung                                                                                 |
|-------------------------------------|---------------------------------------------------------------------------------------------|
| [**\_FLICKPUNKT**](/windows/desktop/api/tabflicks/ns-tabflicks-flick_point) | Enthält die X- und Y-Datenmitglieder, die den Punkt beschreiben, an dem der Flick aufgetreten ist.<br/> |
| [**\_FLICK-DATEN**](/windows/desktop/api/tabflicks/ns-tabflicks-flick_data)   | Enthält ausführliche Informationen über die Flickbewegung und ihre Umgebung.<br/>       |
