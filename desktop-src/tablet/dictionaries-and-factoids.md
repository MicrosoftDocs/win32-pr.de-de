# <a name="dictionaries-and-factoids"></a>Wörterbücher und Factoids

Ein Wörterbuch ist ein internes Lexikon, das eine Erkennung verwendet, um Worttreffer und Alternative zu bestimmen. Ein Factoid erhöht die Erkennungsgenauigkeit, indem ein bestimmter Kontext für ein bestimmtes Feld ausgewählt wird.

Weitere Informationen zu Wörterbüchern finden Sie unter [Verwenden von Anwendungswörterbüchern.](using-application-dictionaries.md)

Weitere Informationen zu Factoiden finden Sie unter [Verwenden von Kontext zur Verbesserung der Genauigkeit.](using-context-to-improve-accuracy.md)
