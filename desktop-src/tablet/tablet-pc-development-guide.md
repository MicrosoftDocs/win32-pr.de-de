# <a name="tablet-pc"></a>Tablet PC

Der Tablet PC ist ein voll funktionsfähiger PCs, der für Stift fähige, Handschrift fähige und sprach aktivierte Anwendungen konzipiert ist. Die Kombination von Software und Hardware auf einem Tablet PC ermöglicht diese Methoden der Benutzerinteraktion und ermöglicht eine umfangreiche, interaktive und produktive Computer Darstellung für Benutzer.

Die Tablet PC-Technologie in Windows ermöglicht die Eingabe und Ausgabe von Handschrift-und sprach Daten auf einem Tablet PC sowie den Austausch dieser Daten mit anderen Computern.

Die in Windows 7 eingeführte Windows-Toucheingabe ermöglicht multitouchgesten und eine verbesserte Touchfunktion für Benutzer.
