# <a name="listening-to-system-events"></a>Lauschen auf System Ereignisse

Anwendungen können Systemereignisse überwachen, indem Sie das [**InkCollector**](inkcollector-class.md) -Objekt verwenden und das System [**Gesten**](inkcollector-systemgesture.md) -Ereignis darauf lauschen. Sie können festlegen, auf welche Ereignisse eine Anwendung lauscht. Wenn eine tablettstiftaktion auftritt, wird das entsprechende **systemgesten** -Ereignis an die Anwendung im **InkCollector** -Objekt gesendet. Eine Anwendung kann die Maus Meldung abbrechen, die einem bestimmten **systemgesten** -Ereignis entspricht, wenn das Ereignis empfangen wird. Ausführliche Informationen zum Abbrechen von Maus Meldungen finden Sie unter **systemgesten** -Ereignis.
