# <a name="listening-to-system-events"></a>Lauschen auf Systemereignisse

Anwendungen können auf Systemereignisse lauschen, indem sie das [**InkCollector-Objekt**](inkcollector-class.md) verwenden und auf das [**SystemGesture-Ereignis**](inkcollector-systemgesture.md) darauf lauschen. Sie können festlegen, auf welche Ereignisse eine Anwendung lauscht. Wenn eine Tablettstiftaktion auftritt, wird das entsprechende **SystemGesture-Ereignis** an die Anwendung für ihr **InkCollector-Objekt** gesendet. Eine Anwendung kann die Mausnachricht abbrechen, die einem bestimmten **SystemGesture-Ereignis** entspricht, wenn sie das Ereignis empfängt. Ausführliche Informationen zum Abbrechen von Mausnachrichten finden Sie unter **SystemGesture-Ereignis.**
