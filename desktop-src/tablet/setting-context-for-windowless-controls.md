# <a name="setting-context-for-windowless-controls"></a>Festlegen des Kontexts für fensterlose Steuerelemente

Fensterlose Steuerelemente können die Funktion "*" immer dann aufzurufen, [**Wenn der Fokus**](/windows/win32/api/inputscope/nf-inputscope-setinputscope) zwischen Elementen des fensterlosen Steuer Elements verschoben wird. Dies wird jedoch aus Leistungsgründen nicht empfohlen.
