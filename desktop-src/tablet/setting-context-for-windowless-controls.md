# <a name="setting-context-for-windowless-controls"></a>Festlegen des Kontexts für fensterlose Steuerelemente

Fensterlose Steuerelemente können die [**SetInputScope-Funktion**](/windows/win32/api/inputscope/nf-inputscope-setinputscope) aufrufen, wenn der Fokus zwischen Elementen des fensterlosen Steuerelements verschoben wird. Dies wird jedoch aus Leistungsgründen nicht empfohlen.
