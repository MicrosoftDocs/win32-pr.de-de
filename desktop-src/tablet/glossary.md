# <a name="tablet-pc-api-glossary"></a>Tablet PC-API-Glossar

<dl> <dt>

<span id="tablet.tablet_glossary_alternate"></span><span id="TABLET.TABLET_GLOSSARY_ALTERNATE"></span>**nder**
</dt> <dd>

Eine potenzielle Wort Übereinstimmung für eine frei Hand Einheit. Alternativen werden von einer Erkennung generiert und basieren auf akzeptablen Übereinstimmungen der frei Hand Eingaben mit einem Wörterbuch, einem Grammatik Satz oder einem Eingabebereich.

</dd> <dt>

<span id="tablet.tablet_glossary_application_dictionary"></span><span id="TABLET.TABLET_GLOSSARY_APPLICATION_DICTIONARY"></span>**Anwendungs Wörterbuch**
</dt> <dd>

Eine Liste von Wörtern, die für eine Anwendung spezifisch sind und weder im System Wörterbuch noch im Benutzerwörterbuch gefunden werden.

</dd> <dt>

<span id="tablet.tablet_glossary_application_gesture"></span><span id="TABLET.TABLET_GLOSSARY_APPLICATION_GESTURE"></span>**Anwendungs Geste**
</dt> <dd>

Eine Geste, die von einer bestimmten Anwendung unterstützt wird.

</dd> <dt>

<span id="tablet.tablet_glossary_bezier_curve"></span><span id="TABLET.TABLET_GLOSSARY_BEZIER_CURVE"></span>**Bezier-Kurve**
</dt> <dd>

Eine Kurve, die mathematisch berechnet wird, um separate Punkte in eine glatte, frei Form Kurve zu verbinden.

</dd> <dt>

<span id="tablet.tablet_glossary_confidence_level"></span><span id="TABLET.TABLET_GLOSSARY_CONFIDENCE_LEVEL"></span>**Vertrauensgrad**
</dt> <dd>

Ein Wert, der den Grad der Vertrauenswürdigkeit für jede mögliche Übereinstimmung angibt, die die Erkennung für eine bestimmte frei Hand Einheit zurückgibt.

</dd> <dt>

<span id="tablet.tablet_glossary_correction_area"></span><span id="TABLET.TABLET_GLOSSARY_CORRECTION_AREA"></span>**Korrekturbereich**
</dt> <dd>

Der Bereich, der im Schreibbereich angezeigt wird, wenn Text angetippt oder ausgewählt wird. Der Benutzer kann dann ein falsch erkanntes Zeichen neu schreiben oder ein Wort durch eine vorgeschlagene Alternative ersetzen.

</dd> <dt>

<span id="tablet.tablet_glossary_cusp"></span><span id="TABLET.TABLET_GLOSSARY_CUSP"></span>**Cusp**
</dt> <dd>

Der Punkt, an dem ein Strich abrupt die Richtung ändert. Die Endpunkte eines Strichs werden auch als "cusps" betrachtet. Der Großbuchstabe "L" hat beispielsweise drei cusps: einen in der Mitte und einen an jedem Ende.

</dd> <dt>

<span id="tablet.tablet_glossary_dictionary"></span><span id="TABLET.TABLET_GLOSSARY_DICTIONARY"></span>**Buch**
</dt> <dd>

Ein Satz von Wörtern, den eine Erkennung verwendet, um Übereinstimmungen und Alternativen zu bestimmen.

</dd> <dt>

<span id="tablet.tablet_glossary_digitizer"></span><span id="TABLET.TABLET_GLOSSARY_DIGITIZER"></span>**Digitizer**
</dt> <dd>

Ein hochauflösende Hardware Gerät, das Eingaben von einem Stift oder Finger erkennt.

</dd> <dt>

<span id="tablet.tablet_glossary_factoid"></span><span id="TABLET.TABLET_GLOSSARY_FACTOID"></span>**Faktoid**
</dt> <dd>

Siehe Eingabebereich.

</dd> <dt>

<span id="tablet.tablet_glossary_flick"></span><span id="TABLET.TABLET_GLOSSARY_FLICK"></span>**umzu**
</dt> <dd>

Ein schneller, geradliniger Strich eines Fingers oder Stifts auf einem Bildschirm. Ein Flick wird als Geste erkannt und als Navigations-oder Bearbeitungs Befehl interpretiert.

</dd> <dt>

<span id="tablet.tablet_glossary_flick_action"></span><span id="TABLET.TABLET_GLOSSARY_FLICK_ACTION"></span>**Flick Aktion**
</dt> <dd>

Die Aktion oder Verknüpfung, die als Reaktion auf einen Flick ausgeführt wird.

</dd> <dt>

<span id="tablet.tablet_glossary_gesture"></span><span id="TABLET.TABLET_GLOSSARY_GESTURE"></span>**Gestaltungs**
</dt> <dd>

Eine Bewegung eines Stifts oder Fingers auf einem Bildschirm, der vom Computer als Befehl interpretiert wird.

</dd> <dt>

<span id="tablet.tablet_glossary_grammar_set"></span><span id="TABLET.TABLET_GLOSSARY_GRAMMAR_SET"></span>**Grammatik Satz**
</dt> <dd>

Der Erkennungs Kontext für frei Hand Eingaben, der nicht in einem Wörterbuch vorhanden ist, z. b. Zahlen, Währungen oder Datumsangaben.

</dd> <dt>

<span id="tablet.tablet_glossary_handwriting_recognition"></span><span id="TABLET.TABLET_GLOSSARY_HANDWRITING_RECOGNITION"></span>**Handschrifterkennung**
</dt> <dd>

Die Fähigkeit eines Programms, handgeschriebenen Text zu interpretieren und in den computerlesbaren Text umzuwandeln. Handschrift Erkennungs Programme akzeptieren Eingaben von einem Stift oder einem anderen Handschrift Eingabegerät, nicht von einer Tastatur.

</dd> <dt>

<span id="tablet.tablet_glossary_hot_point"></span><span id="TABLET.TABLET_GLOSSARY_HOT_POINT"></span>**Hotspot**
</dt> <dd>

Der Unterscheidungs Punkt in der Geometrie einer Geste. Beispielsweise ist die Spitze eines Caretzeichen (^) der Spitze.

</dd> <dt>

<span id="tablet.tablet_glossary_human_interface_device__hid_"></span><span id="TABLET.TABLET_GLOSSARY_HUMAN_INTERFACE_DEVICE__HID_"></span>**Personal Interface-Gerät (HID)**
</dt> <dd>

Eine Spezifikation, die zum standardmäßigen USB-Standard (Universal Serial Bus) für eine Klasse von Peripheriegeräten gehört. Diese Geräte, wie z. b. eine Maus, Tastatur und Joysticks, ermöglichen es Benutzern, Daten einzugeben oder direkt mit dem Computer zu interagieren.

</dd> <dt>

<span id="tablet.tablet_glossary_ink"></span><span id="TABLET.TABLET_GLOSSARY_INK"></span>**Haken**
</dt> <dd>

Die Schreib-oder Zeichnungs Striche, die mit einem Tablettstift oder einer Maus erstellt werden.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_analysis"></span><span id="TABLET.TABLET_GLOSSARY_INK_ANALYSIS"></span>**Ink-Analyse**
</dt> <dd>

Der Prozess der programmgesteuerten Untersuchung von frei Hand Eingaben.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_classification"></span><span id="TABLET.TABLET_GLOSSARY_INK_CLASSIFICATION"></span>**Ink-Klassifizierung**
</dt> <dd>

Die Komponente der Ink-Analyse, die das Aufteilen von frei Hand Eingaben in sinnvolle Einheiten, wie z. b. Absätze, Linien, Wörter und Zeichnungen, einschließt

</dd> <dt>

<span id="tablet.tablet_glossary_ink_collection"></span><span id="TABLET.TABLET_GLOSSARY_INK_COLLECTION"></span>**Ink-Auflistung**
</dt> <dd>

Der Prozess des Sammelns von frei Hand Daten von einem Digitalisierer.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_recognition"></span><span id="TABLET.TABLET_GLOSSARY_INK_RECOGNITION"></span>**frei Handerkennung**
</dt> <dd>

Der Prozess, bei dem frei Hand Eingaben in andere Datentypen, z. b. Text, umgewandelt werden.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_serialized_format__isf_"></span><span id="TABLET.TABLET_GLOSSARY_INK_SERIALIZED_FORMAT__ISF_"></span>**Ink-serialisiertes Format (ISF)**
</dt> <dd>

Ein grundlegendes Format zum Speichern von frei Hand Daten.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_space"></span><span id="TABLET.TABLET_GLOSSARY_INK_SPACE"></span>**Leerraum**
</dt> <dd>

Der virtuelle Koordinaten Bereich, dem die Koordinaten des Tablet-Kontexts zugeordnet sind.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_stroke"></span><span id="TABLET.TABLET_GLOSSARY_INK_STROKE"></span>**frei Hand Strich**
</dt> <dd>

Eine Textzeile, die auf einem Tablet geschrieben oder gezeichnet wurde, beginnend an der Position, an der der Stift zuerst den Bildschirm berührt hat, und das Beenden des Stifts.

</dd> <dt>

<span id="tablet.tablet_glossary_input_panel"></span><span id="TABLET.TABLET_GLOSSARY_INPUT_PANEL"></span>**Eingabebereich**
</dt> <dd>

Ein Zubehör zum Eingeben von Text mithilfe eines Tablettstifts anstelle einer Standardtastatur. Der Eingabebereich umfasst einen Schreibbereich und einen Zeichenbereich, um die Handschrift in typisierten Text umzuwandeln, und eine Bildschirmtastatur zum eingeben einzelner Zeichen.

</dd> <dt>

<span id="tablet.tablet_glossary_input_scope"></span><span id="TABLET.TABLET_GLOSSARY_INPUT_SCOPE"></span>**Eingabebereich**
</dt> <dd>

Eine Reihe von Wörtern, Ziffern, Interpunktions Zeichen und syntaktischen Sortierungen, die innerhalb eines bestimmten Sprach Modells zulässig sind. Eingabe Bereiche können von Anwendungen verwendet werden, um das Sprachmodell, das von der Erkennung verwendet wird, auf einen bestimmten Kontext einzuschränken. Früher als FA bezeichnet

</dd> <dt>

<span id="tablet.tablet_glossary_intersection"></span><span id="TABLET.TABLET_GLOSSARY_INTERSECTION"></span>**Kreuz**
</dt> <dd>

Ein Punkt, an dem sich ein Strich mit sich selbst oder einem anderen Strich überschneidet.

</dd> <dt>

<span id="tablet.tablet_glossary_language_model"></span><span id="TABLET.TABLET_GLOSSARY_LANGUAGE_MODEL"></span>**Sprachmodell**
</dt> <dd>

Eine Reihe von Regeln für eine bestimmte Sprache, die als primärer Kontext zum Erkennen von Wörtern verwendet werden.

</dd> <dt>

<span id="tablet.tablet_glossary_lattice"></span><span id="TABLET.TABLET_GLOSSARY_LATTICE"></span>**tten**
</dt> <dd>

Eine Gruppe von-Strukturen, die verwendet werden, um Erkennungsergebnisse an Tablet PC-Bibliotheken zu übergeben. Die Flexibilität der Gitterstrukturen ermöglicht die logische Verwaltung der großen Anzahl von Erkennungs Ergebnissen, die zu komplexen Beziehungen zusammengefasst werden können.

</dd> <dt>

<span id="tablet.tablet_glossary_layout_analysis"></span><span id="TABLET.TABLET_GLOSSARY_LAYOUT_ANALYSIS"></span>**Layoutanalyse**
</dt> <dd>

Die Komponente der frei Hand Analyse, bei der die Position von frei Hand Eingaben auf der Erstellungs Oberfläche und die Beziehung zwischen den Strichen räumlich und semantisch bestimmt werden. So kann z. b. eine Layoutanalyse bestimmen, dass eine bestimmte frei Hand Einheit eine Anmerkung ist.

</dd> <dt>

<span id="tablet.tablet_glossary_manipulation"></span><span id="TABLET.TABLET_GLOSSARY_MANIPULATION"></span>**Bearbeitung**
</dt> <dd>

Eine physische Aktion (z. b. Drehung), die auf ein Objekt angewendet wird.

</dd> <dt>

<span id="tablet.tablet_glossary_math_input_control"></span><span id="TABLET.TABLET_GLOSSARY_MATH_INPUT_CONTROL"></span>**Math-Eingabe Steuerelement**
</dt> <dd>

Eine Komponente einer Anwendung, die verwendet wird, um mathematische Symbole in Handschrift zu erkennen.

</dd> <dt>

<span id="tablet.tablet_glossary_microsoft_gesture_recognizer"></span><span id="TABLET.TABLET_GLOSSARY_MICROSOFT_GESTURE_RECOGNIZER"></span>**Gestenerkennung von Microsoft**
</dt> <dd>

Eine Engine, die frei Hand Eingaben als System-oder Anwendungs Befehle erkennt und verarbeitet.

</dd> <dt>

<span id="tablet.tablet_glossary_microsoft_windows_journal_note_reader"></span><span id="TABLET.TABLET_GLOSSARY_MICROSOFT_WINDOWS_JOURNAL_NOTE_READER"></span>**Microsoft Windows Journal-Notiz Leser**
</dt> <dd>

Eine Komponente, die einen Stream mit einer JNT-Datei annimmt und einen Stream mit dem Dateiinhalt im XML-Format zurückgibt, der dem Journal-leseleseschema entspricht.

</dd> <dt>

<span id="tablet.tablet_glossary_multitouch"></span><span id="TABLET.TABLET_GLOSSARY_MULTITOUCH"></span>**Multitouch**
</dt> <dd>

Eine physische Echt Zeit Aktion, die auf ein Objekt auf einem Bildschirm angewendet wird, auf dem mehrere Kontaktpunkte gleichzeitig verwendet werden.

</dd> <dt>

<span id="tablet.tablet_glossary_object_recognizer"></span><span id="TABLET.TABLET_GLOSSARY_OBJECT_RECOGNIZER"></span>**Objekterkennung**
</dt> <dd>

Eine Engine, die allgemeine Formen nach ihrem Zweck erkennt. Zu diesen Objekten gehören Musik Notizen, geometrische Formen, mathematische Gleichungen und Flussdiagramm Elemente.

</dd> <dt>

<span id="tablet.tablet_glossary_packet"></span><span id="TABLET.TABLET_GLOSSARY_PACKET"></span>**Lohn**
</dt> <dd>

Der Satz von Daten, die vom Tablet-Gerät an jedem Stichproben Punkt in einem Strich gesendet werden.

</dd> <dt>

<span id="tablet.tablet_glossary_recognition_guide"></span><span id="TABLET.TABLET_GLOSSARY_RECOGNITION_GUIDE"></span>**Erkennungs Handbuch**
</dt> <dd>

Eine-Struktur, die die frei Hand Begrenzungen an die Erkennung mithilfe von frei Hand Raumkoordinaten angibt. Im Allgemeinen gibt ein Erkennungs Handbuch an, welche Markierungen auf dem Bildschirm gezeichnet werden, damit der Benutzer Bereiche für Eingaben isolieren kann.

</dd> <dt>

<span id="tablet.tablet_glossary_recognition_segment"></span><span id="TABLET.TABLET_GLOSSARY_RECOGNITION_SEGMENT"></span>**Erkennungs Segment**
</dt> <dd>

Eine frei Hand Einheit, die von einer Erkennung interpretiert wird. Für westliche Sprachen ist dies normalerweise ein Wort. für ostasiatische Sprachen ist es normalerweise ein Zeichen.

</dd> <dt>

<span id="tablet.tablet_glossary_recognizer"></span><span id="TABLET.TABLET_GLOSSARY_RECOGNIZER"></span>**Erkennung**
</dt> <dd>

Ein Softwaremodul, das Komponenten einer Sprache oder Gruppe verwandter Sprachen erkennt und verarbeitet, oder eine Klasse verwandter Objekte, wie z. b. Musik Notizen, System Gesten und geometrische Formen.

</dd> <dt>

<span id="tablet.tablet_glossary_recognizer_context"></span><span id="TABLET.TABLET_GLOSSARY_RECOGNIZER_CONTEXT"></span>**Erkennungs Kontext**
</dt> <dd>

Anwendungsspezifische Informationen, die die Erkennung verwendet, um die frei Handerkennung durchzuführen.

</dd> <dt>

<span id="tablet.tablet_glossary_sketch_ink_object"></span><span id="TABLET.TABLET_GLOSSARY_SKETCH_INK_OBJECT"></span>**skeink-Objekt**
</dt> <dd>

Ein OLE-Objekt, das das frei Handzeichen darstellt, das anstelle eines Worts als Zeichnung interpretiert wird.

</dd> <dt>

<span id="tablet.tablet_glossary_system_dictionary"></span><span id="TABLET.TABLET_GLOSSARY_SYSTEM_DICTIONARY"></span>**System Wörterbuch**
</dt> <dd>

Eine Wort Liste, die alle häufig geschriebenen Wörter in einer Sprache enthält.

</dd> <dt>

<span id="tablet.tablet_glossary_system_gesture"></span><span id="TABLET.TABLET_GLOSSARY_SYSTEM_GESTURE"></span>**System Geste**
</dt> <dd>

Eine Stift Entsprechung einer herkömmlichen Maus Nachricht, z. b. durch Klicken mit der linken Maustaste oder Doppelklick.

</dd> <dt>

<span id="tablet.tablet_glossary_tablet_pc_managed_library"></span><span id="TABLET.TABLET_GLOSSARY_TABLET_PC_MANAGED_LIBRARY"></span>**Tablet PC Managed Library**
</dt> <dd>

Eine Reihe von verwalteten Objekten, die das Microsoft .NET Framework erweitern, um die Eingabe und Ausgabe von Handschrift auf Tablet PC und Datenaustausch mit anderen Computern zu unterstützen.

</dd> <dt>

<span id="tablet.tablet_glossary_text_ink_object"></span><span id="TABLET.TABLET_GLOSSARY_TEXT_INK_OBJECT"></span>**textink-Objekt**
</dt> <dd>

Ein OLE-Objekt, das frei Hand Eingaben darstellt, die als Word interpretiert werden.

</dd> <dt>

<span id="tablet.tablet_glossary_user_dictionary"></span><span id="TABLET.TABLET_GLOSSARY_USER_DICTIONARY"></span>**Benutzerwörterbuch**
</dt> <dd>

Eine Wort Liste, die Wörter enthält, die von Benutzern zum Wörterbuch hinzugefügt werden.

</dd> <dt>

<span id="tablet.tablet_glossary_word_list"></span><span id="TABLET.TABLET_GLOSSARY_WORD_LIST"></span>**Wortliste**
</dt> <dd>

Eine Datei, die eine Auflistung von Wörtern enthält, aus denen sich ein Anwendungs Wörterbuch besteht.

</dd> </dl>
