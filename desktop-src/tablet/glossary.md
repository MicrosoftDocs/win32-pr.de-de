# <a name="tablet-pc-api-glossary"></a>Glossar zur Tablet PC-API

<dl> <dt>

<span id="tablet.tablet_glossary_alternate"></span><span id="TABLET.TABLET_GLOSSARY_ALTERNATE"></span>**Alternative**
</dt> <dd>

Eine mögliche Wort übereinstimmung für eine Einheit von Ink. Alternativen werden von einer -Erkannten generiert und basieren auf akzeptablen Übereinstimmungen der Freieingabe für ein Wörterbuch, einen Grammatiksatz oder einen Eingabebereich.

</dd> <dt>

<span id="tablet.tablet_glossary_application_dictionary"></span><span id="TABLET.TABLET_GLOSSARY_APPLICATION_DICTIONARY"></span>**Anwendungswörterbuch**
</dt> <dd>

Eine Anwendungsspezifische Liste von Wörtern, die weder im System- noch im Benutzerwörterbuch gefunden werden.

</dd> <dt>

<span id="tablet.tablet_glossary_application_gesture"></span><span id="TABLET.TABLET_GLOSSARY_APPLICATION_GESTURE"></span>**Anwendungsgeste**
</dt> <dd>

Eine Geste, die von einer bestimmten Anwendung unterstützt wird.

</dd> <dt>

<span id="tablet.tablet_glossary_bezier_curve"></span><span id="TABLET.TABLET_GLOSSARY_BEZIER_CURVE"></span>**Bézierkurve**
</dt> <dd>

Eine Kurve, die mathematisch berechnet wird, um separate Punkte mit einer reibungslosen Freiformkurve zu verbinden.

</dd> <dt>

<span id="tablet.tablet_glossary_confidence_level"></span><span id="TABLET.TABLET_GLOSSARY_CONFIDENCE_LEVEL"></span>**Vertrauensgrad**
</dt> <dd>

Ein -Wert, der den Grad der Konfidenz für jede mögliche Übereinstimmung angibt, die von der Erkannten für eine bestimmte Einheit von Ink zurückgegeben wird.

</dd> <dt>

<span id="tablet.tablet_glossary_correction_area"></span><span id="TABLET.TABLET_GLOSSARY_CORRECTION_AREA"></span>**Korrekturbereich**
</dt> <dd>

Der Bereich, der im Schreibpad angezeigt wird, wenn Text angeknippt oder ausgewählt wird. Der Benutzer kann dann ein falsch erkanntes Zeichen neu schreiben oder ein Wort durch eine vorgeschlagene Alternative ersetzen.

</dd> <dt>

<span id="tablet.tablet_glossary_cusp"></span><span id="TABLET.TABLET_GLOSSARY_CUSP"></span>**Schwelle**
</dt> <dd>

Der Punkt, an dem ein Strich die Richtung plötzlich ändert. Die Endpunkte eines Strichs werden ebenfalls als Cusps betrachtet. Der Großbuchstaben "L" hat beispielsweise drei Hinweise: einen in der Mitte und einen an jedem Ende.

</dd> <dt>

<span id="tablet.tablet_glossary_dictionary"></span><span id="TABLET.TABLET_GLOSSARY_DICTIONARY"></span>**Wörterbuch**
</dt> <dd>

Eine Gruppe von Wörtern, die von der -Ermittlung verwendet werden, um Übereinstimmungen und Alternativen zu bestimmen.

</dd> <dt>

<span id="tablet.tablet_glossary_digitizer"></span><span id="TABLET.TABLET_GLOSSARY_DIGITIZER"></span>**Digitizer**
</dt> <dd>

Ein hochauflösendes Hardwaregerät, das Eingaben von einem Stift oder Finger erkennt.

</dd> <dt>

<span id="tablet.tablet_glossary_factoid"></span><span id="TABLET.TABLET_GLOSSARY_FACTOID"></span>**Factoid**
</dt> <dd>

Siehe Eingabebereich.

</dd> <dt>

<span id="tablet.tablet_glossary_flick"></span><span id="TABLET.TABLET_GLOSSARY_FLICK"></span>**Flick**
</dt> <dd>

Ein schneller, gerader Strich eines Fingers oder Stifts auf einem Bildschirm. Ein Flick wird als Geste erkannt und als Navigations- oder Bearbeitungsbefehl interpretiert.

</dd> <dt>

<span id="tablet.tablet_glossary_flick_action"></span><span id="TABLET.TABLET_GLOSSARY_FLICK_ACTION"></span>**Aktion "Blättern"**
</dt> <dd>

Die Aktion oder Verknüpfung, die als Reaktion auf einen Flick ausgeführt wird.

</dd> <dt>

<span id="tablet.tablet_glossary_gesture"></span><span id="TABLET.TABLET_GLOSSARY_GESTURE"></span>**Geste**
</dt> <dd>

Eine Bewegung eines Stifts oder fingers auf einem Bildschirm, die der Computer als Befehl interpretiert.

</dd> <dt>

<span id="tablet.tablet_glossary_grammar_set"></span><span id="TABLET.TABLET_GLOSSARY_GRAMMAR_SET"></span>**Grammatiksatz**
</dt> <dd>

Der Erkennungskontext für Ink, der sich nicht in einem Wörterbuch befindet, z. B. Zahlen, Währungen oder Datumsangaben.

</dd> <dt>

<span id="tablet.tablet_glossary_handwriting_recognition"></span><span id="TABLET.TABLET_GLOSSARY_HANDWRITING_RECOGNITION"></span>**Handschrifterkennung**
</dt> <dd>

Die Fähigkeit eines Programms, handschriftlichen Text zu interpretieren und in computerlesbaren Text zu konvertieren. Handschrifterkennungsprogramme akzeptieren Eingaben von einem Stift oder einem anderen Handschrifteingabegerät, nicht von einer Tastatur.

</dd> <dt>

<span id="tablet.tablet_glossary_hot_point"></span><span id="TABLET.TABLET_GLOSSARY_HOT_POINT"></span>**Hot Point**
</dt> <dd>

Der Unterscheidungspunkt in der Geometrie einer Geste. Der heiße Punkt einer Caretbewegung (^) ist z. B. der Apex.

</dd> <dt>

<span id="tablet.tablet_glossary_human_interface_device__hid_"></span><span id="TABLET.TABLET_GLOSSARY_HUMAN_INTERFACE_DEVICE__HID_"></span>**Human Interface Device (HID)**
</dt> <dd>

Eine Spezifikation, die Teil des USB-Standards (Universal Serial Bus) für eine Klasse von Peripheriegeräten ist. Diese Geräte, z. B. ein Maus, Tastaturen und Maske, ermöglichen es Benutzern, Daten ein- oder direkt mit dem Computer zu interagieren.

</dd> <dt>

<span id="tablet.tablet_glossary_ink"></span><span id="TABLET.TABLET_GLOSSARY_INK"></span>**Tinte**
</dt> <dd>

Das Schreiben oder Zeichnen von Strichen, die mit einem Tablettstift oder einer Maus gemacht werden.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_analysis"></span><span id="TABLET.TABLET_GLOSSARY_INK_ANALYSIS"></span>**Ink-Analyse**
</dt> <dd>

Der Prozess der programmgesteuerten Untersuchung von Freieingaben.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_classification"></span><span id="TABLET.TABLET_GLOSSARY_INK_CLASSIFICATION"></span>**Ink-Klassifizierung**
</dt> <dd>

Die Komponente der Ink-Analyse, die die Unterteilung von Ink in sinnvolle Einheiten wie Absätze, Linien, Wörter und Zeichnungen umfasst.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_collection"></span><span id="TABLET.TABLET_GLOSSARY_INK_COLLECTION"></span>**Ink-Sammlung**
</dt> <dd>

Der Prozess der Erfassung von Ink-Daten von einem Digitizer.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_recognition"></span><span id="TABLET.TABLET_GLOSSARY_INK_RECOGNITION"></span>**Ink-Erkennung**
</dt> <dd>

Der Prozess der Konvertierung von Ink in andere Datentypen, z. B. Text.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_serialized_format__isf_"></span><span id="TABLET.TABLET_GLOSSARY_INK_SERIALIZED_FORMAT__ISF_"></span>**Serialisiertes Ink-Format (ISF)**
</dt> <dd>

Ein grundlegendes Format zum Speichern von Ink-Daten.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_space"></span><span id="TABLET.TABLET_GLOSSARY_INK_SPACE"></span>**Freiraum**
</dt> <dd>

Der virtuelle Koordinatenraum, dem die Koordinaten des Tablet-Kontexts zugeordnet werden.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_stroke"></span><span id="TABLET.TABLET_GLOSSARY_INK_STROKE"></span>**Ink-Strich**
</dt> <dd>

Eine auf einem Tablet geschriebene oder gezeichnete Ink-Linie, die an der Stelle beginnt, an der der Stift zuerst den Bildschirm berührt hat, und an der Stelle endet, an der der Stift gezeichnet wurde.

</dd> <dt>

<span id="tablet.tablet_glossary_input_panel"></span><span id="TABLET.TABLET_GLOSSARY_INPUT_PANEL"></span>**Eingabebereich**
</dt> <dd>

Ein Accessory zum Eingeben von Text mithilfe eines Tablettstifts anstelle einer Standardtastatur. Der Eingabebereich enthält ein Schreibpad und ein Zeichenpad zum Konvertieren der Handschrift in typierten Text sowie eine Bildschirmtastatur zum Eingeben einzelner Zeichen.

</dd> <dt>

<span id="tablet.tablet_glossary_input_scope"></span><span id="TABLET.TABLET_GLOSSARY_INPUT_SCOPE"></span>**Eingabebereich**
</dt> <dd>

Eine Gruppe von Wörtern, Zahlen, Satzzeichen und syntaktischen Sortierungen, die innerhalb eines bestimmten Sprachmodells zulässig sind. Eingabebereich können von Anwendungen verwendet werden, um das von der -Erkennen verwendete Sprachmodell auf einen bestimmten Kontext zu beschränken. Früher als fa bekannt

</dd> <dt>

<span id="tablet.tablet_glossary_intersection"></span><span id="TABLET.TABLET_GLOSSARY_INTERSECTION"></span>**Schnittpunkt**
</dt> <dd>

Ein Punkt, an dem sich ein Strich mit sich selbst oder einem anderen Strich überschneidet.

</dd> <dt>

<span id="tablet.tablet_glossary_language_model"></span><span id="TABLET.TABLET_GLOSSARY_LANGUAGE_MODEL"></span>**Sprachmodell**
</dt> <dd>

Ein Satz von Regeln für eine bestimmte Sprache, die als primärer Kontext für die Erkennung von Wörtern verwendet werden.

</dd> <dt>

<span id="tablet.tablet_glossary_lattice"></span><span id="TABLET.TABLET_GLOSSARY_LATTICE"></span>**Gitter**
</dt> <dd>

Eine von strukturen, die verwendet werden, um Erkennungsergebnisse an Tablet PC-Bibliotheken zu übergeben. Die Flexibilität der Latticestrukturen ermöglicht die logische Verwaltung der großen Anzahl von Erkennungsergebnissen, die in komplexen Beziehungen kombiniert werden können.

</dd> <dt>

<span id="tablet.tablet_glossary_layout_analysis"></span><span id="TABLET.TABLET_GLOSSARY_LAYOUT_ANALYSIS"></span>**Layoutanalyse**
</dt> <dd>

Die Komponente der Ink-Analyse, bei der die Position von Ink auf der Inking-Oberfläche bestimmt wird und wie die Striche räumlich und semantisch miteinander in Beziehung stehen. Die Layoutanalyse kann z. B. feststellen, dass eine bestimmte Einheit von Ink eine Annotatio ist.

</dd> <dt>

<span id="tablet.tablet_glossary_manipulation"></span><span id="TABLET.TABLET_GLOSSARY_MANIPULATION"></span>**Manipulation**
</dt> <dd>

Eine physische Aktion, z. B. Drehung, die auf ein Objekt angewendet wird.

</dd> <dt>

<span id="tablet.tablet_glossary_math_input_control"></span><span id="TABLET.TABLET_GLOSSARY_MATH_INPUT_CONTROL"></span>**Mathematisches Eingabesteuerfeld**
</dt> <dd>

Eine Komponente einer Anwendung, die verwendet wird, um mathematische Symbole in der Handschrift zu erkennen.

</dd> <dt>

<span id="tablet.tablet_glossary_microsoft_gesture_recognizer"></span><span id="TABLET.TABLET_GLOSSARY_MICROSOFT_GESTURE_RECOGNIZER"></span>**Microsoft-Gestenerkennung**
</dt> <dd>

Eine Engine, die Ink-Eingaben entweder als System- oder Anwendungsbefehle erkennt und verarbeitet.

</dd> <dt>

<span id="tablet.tablet_glossary_microsoft_windows_journal_note_reader"></span><span id="TABLET.TABLET_GLOSSARY_MICROSOFT_WINDOWS_JOURNAL_NOTE_READER"></span>**Microsoft Windows Journal Note Reader**
</dt> <dd>

Eine Komponente, die einen Stream mit einer JNT-Datei verwendet und einen Stream zurückgibt, der den Inhalt der Datei im XML-Format enthält, das dem Schema des Journalnotizlesers entspricht.

</dd> <dt>

<span id="tablet.tablet_glossary_multitouch"></span><span id="TABLET.TABLET_GLOSSARY_MULTITOUCH"></span>**Multitouch**
</dt> <dd>

Eine physische Echtzeitaktion, die auf ein Objekt auf einem Bildschirm angewendet wird, auf dem mehrere Kontaktpunkte gleichzeitig verwendet werden.

</dd> <dt>

<span id="tablet.tablet_glossary_object_recognizer"></span><span id="TABLET.TABLET_GLOSSARY_OBJECT_RECOGNIZER"></span>**Objekt-Recognizer**
</dt> <dd>

Eine Engine, die allgemeine Formen entsprechend ihrem Zweck erkennt. Zu diesen Objekten gehören Musiknotizen, geometrische Formen, mathematische Gleichungen und Flussdiagrammelemente.

</dd> <dt>

<span id="tablet.tablet_glossary_packet"></span><span id="TABLET.TABLET_GLOSSARY_PACKET"></span>**Paket**
</dt> <dd>

Der Satz von Daten, die vom Tablettgerät an jedem Beispielpunkt eines Strichs gesendet werden.

</dd> <dt>

<span id="tablet.tablet_glossary_recognition_guide"></span><span id="TABLET.TABLET_GLOSSARY_RECOGNITION_GUIDE"></span>**Leitfaden zur Erkennung**
</dt> <dd>

Eine -Struktur, die die Freiraumgrenzen für die Erkennen angibt, indem Freiraumkoordinaten verwendet werden. Im Allgemeinen gibt ein Erkennungsleitfaden an, welche Markierungen auf dem Bildschirm gezeichnet werden, damit der Benutzer Bereiche für die Eingabe isolieren kann.

</dd> <dt>

<span id="tablet.tablet_glossary_recognition_segment"></span><span id="TABLET.TABLET_GLOSSARY_RECOGNITION_SEGMENT"></span>**Erkennungssegment**
</dt> <dd>

Eine Einheit von Ink, die von einer Recognizer interpretiert wird. Für westeskliche Sprachen ist dies in der Regel ein Wort. für ostasiatische Sprachen ist es in der Regel ein Zeichen.

</dd> <dt>

<span id="tablet.tablet_glossary_recognizer"></span><span id="TABLET.TABLET_GLOSSARY_RECOGNIZER"></span>**Erkennungsmodul**
</dt> <dd>

Ein Softwaremodul, das Komponenten einer Sprache oder Gruppe verwandter Sprachen oder eine Klasse verwandter Objekte wie Notizen, Systemgesten und geometrische Formen erkennt und verarbeitet.

</dd> <dt>

<span id="tablet.tablet_glossary_recognizer_context"></span><span id="TABLET.TABLET_GLOSSARY_RECOGNIZER_CONTEXT"></span>**Erkennungskontext**
</dt> <dd>

Anwendungsspezifische Informationen, die die Erkennung verwendet, um Diekerkennung durchzuführen.

</dd> <dt>

<span id="tablet.tablet_glossary_sketch_ink_object"></span><span id="TABLET.TABLET_GLOSSARY_SKETCH_INK_OBJECT"></span>**Sketch-Ink-Objekt**
</dt> <dd>

Ein OLE-Objekt, das die Ink darstellt, die als Zeichnung anstelle eines Worts interpretiert wird.

</dd> <dt>

<span id="tablet.tablet_glossary_system_dictionary"></span><span id="TABLET.TABLET_GLOSSARY_SYSTEM_DICTIONARY"></span>**Systemwörterbuch**
</dt> <dd>

Eine Wortliste, die alle häufig geschriebenen Wörter in einer Sprache enthält.

</dd> <dt>

<span id="tablet.tablet_glossary_system_gesture"></span><span id="TABLET.TABLET_GLOSSARY_SYSTEM_GESTURE"></span>**Systemgeste**
</dt> <dd>

Ein Stift, der einer herkömmlichen Mausnachricht entspricht, z. B. Linksklick oder Doppelklick.

</dd> <dt>

<span id="tablet.tablet_glossary_tablet_pc_managed_library"></span><span id="TABLET.TABLET_GLOSSARY_TABLET_PC_MANAGED_LIBRARY"></span>**Verwaltete Tablet PC-Bibliothek**
</dt> <dd>

Eine Reihe von verwalteten Objekten, die die Microsoft .NET Framework erweitern, um Unterstützung für die Eingabe und Ausgabe von Handschrift auf Tablet-PCs und den Datenaustausch mit anderen Computern bereitzustellen.

</dd> <dt>

<span id="tablet.tablet_glossary_text_ink_object"></span><span id="TABLET.TABLET_GLOSSARY_TEXT_INK_OBJECT"></span>**Text-Ink-Objekt**
</dt> <dd>

Ein OLE-Objekt, das Ink darstellt, das als Wort interpretiert wird.

</dd> <dt>

<span id="tablet.tablet_glossary_user_dictionary"></span><span id="TABLET.TABLET_GLOSSARY_USER_DICTIONARY"></span>**Benutzerwörterbuch**
</dt> <dd>

Eine Wortliste, die Wörter enthält, die dem Wörterbuch von Benutzern hinzugefügt wurden.

</dd> <dt>

<span id="tablet.tablet_glossary_word_list"></span><span id="TABLET.TABLET_GLOSSARY_WORD_LIST"></span>**Wortliste**
</dt> <dd>

Eine Datei, die eine Auflistung von Wörtern enthält, aus denen ein Anwendungswörterbuch besteht.

</dd> </dl>
