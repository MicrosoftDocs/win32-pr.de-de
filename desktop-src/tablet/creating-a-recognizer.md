# <a name="creating-a-recognizer"></a>Erstellen einer Recognizer-Endung

Wenn Sie sich dafür entscheiden, kann Ihre Anwendung eigene Implementierungen für die Wiedererkennung bereitstellen. In diesem Abschnitt werden die Anforderungen zum Erstellen einer benutzerdefinierten Anwendungs-Recognizer-Funktion für Ihre Anwendung beschrieben, die Sie mit der Tablet PC-Plattform-API verwenden können.

> [!Note]  
> Benutzerdefinierte Anwendungs-Recognizer werden nicht vom Tablet PC-Eingabebereich oder microsoft Windows Journal verwendet. Für dieses Zubehör werden nur Diesserken verwendet, mit denen sie getestet wurden.
