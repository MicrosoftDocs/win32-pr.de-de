> [!Note]  
> Benutzerdefinierte Anwendungs Erkennungs Tools werden nicht vom Tablet PC-Eingabebereich oder Microsoft Windows-Journal verwendet. Diese Zubehör verwenden nur Erkennungs Tools, mit denen Sie getestet wurden.
