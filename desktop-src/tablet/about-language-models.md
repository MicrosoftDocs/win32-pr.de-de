# <a name="about-language-models"></a>Informationen zu Sprachmodellen

Handschrifterkennungen verwenden einen definierten Satz von Regeln für eine bestimmte Sprache als primären Kontext für die Erkennung von Wörtern. Diese Regeln werden als Sprachmodelle bezeichnet. Das Sprachmodell wird von der -Erkennen implementiert und kann nicht direkt von einem Entwickler mithilfe des Microsoft Windows XP Tablet PC Edition Software Development Kit (SDK) geändert werden. Es gibt jedoch Möglichkeiten, den Kontext für bestimmte Felder in einer Anwendung zu verfeinern. Diese Verfahren umfassen Folgendes:

-   Eingabebereich
-   Factoids
-   Wörterbücher
-   Begriffslisten
