# <a name="about-language-models"></a>Informationen zu Sprachmodellen

Handschrift Erkennungs Tools verwenden einen definierten Satz von Regeln für eine bestimmte Sprache als primären Kontext zum Erkennen von Wörtern. Diese Regeln werden als Sprachmodelle bezeichnet. Das Sprachmodell wird von der Erkennung implementiert und kann nicht direkt von einem Entwickler mithilfe des Microsoft Windows XP Tablet PC Edition Software Development Kit (SDK) geändert werden. Es gibt jedoch Möglichkeiten, den Kontext für bestimmte Felder in einer Anwendung zu verfeinern. Diese Verfahren umfassen Folgendes:

-   Eingabe Bereiche
-   Faktoide
-   Wörterbücher
-   Begriffslisten
