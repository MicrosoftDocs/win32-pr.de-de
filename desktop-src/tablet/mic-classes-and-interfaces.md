# <a name="math-input-classes-and-interfaces"></a>Mathematische Eingabe Klassen und-Schnittstellen

Dieser Abschnitt enthält die Klassen und Schnittstellen, die für die mathematische Eingabesteuerung verwendet werden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Schnittstelle                                                     | BESCHREIBUNG                                                       |
|---------------------------------------------------------------|-------------------------------------------------------------------|
| [**\_Imathinputcontrolevents**](/windows/win32/api/micaut/nn-micaut-_imathinputcontrolevents) | Macht die Ereignishandler für die mathematische Eingabesteuerung verfügbar.                    |
| [**Imathinputcontrol**](/windows/desktop/api/micaut/nn-micaut-imathinputcontrol)                | Macht Methoden verfügbar, die frei Hand Eingaben in interpretierte mathematische Ausgabe umwandeln. |
