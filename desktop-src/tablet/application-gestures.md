# <a name="application-gestures"></a>Anwendungs Gesten

Anwendungs Gesten sind Gesten, die Sie zur Unterstützung Ihrer Anwendung auswählen. Sie sind am häufigsten in Anwendungen, die speziell für die Arbeit mit einem Tablettstift entwickelt wurden. Eine Legacy Anwendung kann Gesten verwenden, aber vorhandene Auswahl Modelle innerhalb der Anwendung erschweren möglicherweise die Unterstützung für Gesten. Nur die Gesten, die im Tablet PC-Eingabe Panel unterstützt werden, sind für ältere Anwendungen verfügbar.

In den folgenden Abschnitten werden Anwendungs Gesten und deren Verwendung beschrieben.

-   [Anwendungs Gesten und Semantik Verhalten](application-gestures-and-semantic-behavior.md)
-   [Nur die Gestenerkennung von Microsoft verwenden](using-the-microsoft-gesture-recognizer-only.md)
-   [Verwenden einer benutzerdefinierten Gestenerkennung](using-a-custom-gesture-recognizer.md)
-   [Mögliche Konflikte mit Anwendungs Gesten](potential-conflicts-with-application-gestures.md)
-   [Dokumentieren der Gesten, die von Ihren Anwendungen verwendet werden](documenting-the-gestures-used-by-your-applications.md)
-   [Nicht implementierte Symbole](unimplemented-glyphs.md)
