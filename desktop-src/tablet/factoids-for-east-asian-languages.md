# <a name="factoids-for-east-asian-languages"></a>Factoids für ostasiatische Sprachen

Ostasiatische Sprachen sind als Japanisch, Chinesisch (vereinfacht), Chinesisch (traditionell) und Koreanisch definiert. Die Formate innerhalb der Factoids in der folgenden Tabelle sind spezifisch für die Spracherkennung.

Beispielsweise ist das **Telefon-Factoid** in jeder Sprache unterschiedlich. Darüber hinaus ist jedes Factoid spezifisch für eine bestimmte Erkennende. Beispielsweise kann nur  das japanische Telefon-Factoid mit der japanischen Recognizer verwendet werden. Zusätzlich zu den Factoids in der folgenden Tabelle verwenden alle Sprachen die in Factoids Common Across Languages aufgelisteten [Factoids.](factoids-common-across-languages.md)

> [!Note]  
> Die Factoids für ostasiatische Sprachen werden implementiert, indem eine Liste zulässiger Unicode-Zeichen angegeben wird. Die Factoids für westestische Sprachen werden mit regulären Ausdrücken implementiert, die die erwartete Eingabe beschreiben. Dies liegt daran, dass westasiatische Sprachen aus Buchstaben bestehen, die zu Wörtern kombiniert werden, während ostasiatische Sprachen zeichenbasierte Sprachen sind.

 

Erkennen ostasiatischer Zeichen unterstützen die Kombination von bis zu zehn Factoiden. Diese Factoidkombinationen verwenden einen logischen OR-Operator. Daher kann die Eingabe mit jedem der Factoids im Ausdruck übereinstimmen.
