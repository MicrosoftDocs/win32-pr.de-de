# <a name="winsock-ipxspx-structures"></a>Winsock-IPX-/SPX-Strukturen

Die folgende Liste enthält präzise Beschreibungen der einzelnen Winsock-IPX-/SPX-Strukturen. Klicken Sie auf den Strukturnamen, um weitere Informationen zu einer Struktur zu erhalten.



| IPX/SPX-Struktur                                            | Beschreibung                                                          |
|--------------------------------------------------------------|----------------------------------------------------------------------|
| [**\_IPX-ADRESSDATEN \_**](/windows/desktop/api/Wsnwlink/ns-wsnwlink-ipx_address_data)             | Stellt Informationen zu einem bestimmten Adapter zur Verfügung, an den IPX gebunden ist. |
| [**IPX \_ NETNUM \_ DATA**](/windows/desktop/api/Wsnwlink/ns-wsnwlink-ipx_netnum_data)               | Stellt Informationen zu einer angegebenen IPX-Netzwerknummer zur Verfügung.           |
| [**IPX \_ SPXCONNSTATUS-DATEN \_**](/windows/desktop/api/Wsnwlink/ns-wsnwlink-ipx_spxconnstatus_data) | Stellt Informationen zu einem verbundenen SPX-Socket zur Verfügung.                   |
