# <a name="using-sendto-while-connected"></a>Verwenden von SendTo während der Verbindung

[**Wspsendto**](/previous-versions/windows/desktop/legacy/ms742291(v=vs.85)) übermittelt die Daten immer an die angegebene Adresse, auch wenn in [**wspconnect**](/previous-versions/windows/hardware/network/ff566275(v=vs.85))ein benannter Peer für den sendenden Socket eingerichtet wurde.
