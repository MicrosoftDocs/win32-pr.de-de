# <a name="sio_multipoint_loopback-command-code-for-wsaioctl"></a>SIO- \_ Multipoint- \_ Loopback-Befehls Code für WSAIoctl

Wenn d- \_ Blatt Sockets in einer Datenebene verwendet werden, die keine Daten besitzt, ist es wünschenswert, dass Datenverkehr, der an denselben Socket gesendet wird, an denselben Socket zurückgesendet wird. Der \_ Befehl Code für die Befehlszeilen-Multipoint- \_ Loopback für [**WSAIoctl**](/windows/desktop/api/Winsock2/nf-winsock2-wsaioctl) dient zum Aktivieren oder Deaktivieren der Loopbacks von Multipoint-Datenverkehr.
