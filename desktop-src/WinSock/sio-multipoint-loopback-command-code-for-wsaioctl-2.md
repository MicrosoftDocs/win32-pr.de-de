# <a name="sio_multipoint_loopback-command-code-for-wsaioctl"></a>SIO \_ MULTIPOINT \_ LOOPBACK-Befehlscode für WSAIoctl

Wenn d \_ Blattsockets in einer nicht entrooteten Datenebene verwendet werden, ist es wünschenswert, dass ausgehender Datenverkehr an denselben Socket zurückgesendet wird. Der SIO \_ MULTIPOINT \_ LOOPBACK-Befehlscode für [**WSAIoctl**](/windows/desktop/api/Winsock2/nf-winsock2-wsaioctl) wird verwendet, um loopback von Multipoint-Datenverkehr zu aktivieren oder zu deaktivieren.
