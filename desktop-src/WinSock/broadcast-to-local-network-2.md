# <a name="broadcast-to-local-network"></a>Übertragung an das lokale Netzwerk

Es kann eine Übertragung an das lokal angeschlossene Netzwerk erfolgen, indem **sa \_ netnum** auf binäre Nullen (0) und **sa- \_ nodenum** auf Binärdateien (1) festgelegt wird. Diese Übertragung kann an das primäre Netzwerk für das Gerät oder an alle lokal angeschlossenen Netzwerke mit der Option des Dienstanbieters gesendet werden. Übertragungen an das lokale Netzwerk werden von Routern nicht weitergegeben.
