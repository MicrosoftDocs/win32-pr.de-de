# <a name="broadcast-to-local-network"></a>Übertragung an lokales Netzwerk

Eine Übertragung kann an das lokal angefügte Netzwerk erfolgen, indem **sa \_ netnum** auf binäre Nullen (0) und **sa \_ nodenum** auf binäre Nullen (1) festgelegt wird. Diese Übertragung kann an das primäre Netzwerk für das Gerät oder an alle lokal angeschlossenen Netzwerke mit der Option des Dienstanbieters gesendet werden. Broadcasts an das lokale Netzwerk werden nicht von Routern weitergegeben.
