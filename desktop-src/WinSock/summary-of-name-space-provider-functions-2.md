# <a name="summary-of-namespace-provider-functions"></a>Zusammenfassung der Namespace-Anbieter Funktionen

Die Namespace-Dienstanbieter Funktionen können in fünf Kategorien unterteilt werden:

-   Konfiguration von Namespace Anbietern (und Installation)
-   Anbieter Initialisierung
-   Dienstinstallation
-   Client Abfragen
-   Hilfsfunktionen (und Makros)

In den folgenden Abschnitten werden die Funktionen in den einzelnen Kategorien identifiziert und die beabsichtigte Verwendung kurz beschrieben. Wichtige Datenstrukturen werden ebenfalls beschrieben.

-   [Konfiguration und Installation von Namespace Anbietern](name-space-provider-configuration-and-installation-2.md)
-   [Initialisierung und Bereinigung von Namespace Anbietern](name-space-provider-initialization-and-cleanup-2.md)
-   [Dienst Installation in Windows Sockets 2 SPI](service-installation-in-the-windows-sockets-2-spi-2.md)
-   [Dienst Abfrage](service-query-2.md)
-   [Hilfsfunktionen in der SPI](helper-functions-in-the-spi-2.md)
