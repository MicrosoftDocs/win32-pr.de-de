# <a name="multicast"></a>Multicast

Generische Winsock-Multipoint-Funktionen unterstützen IP-Multicast. Allerdings müssen die TCP/IP-Transportanbieter, die Multicast unterstützen, auch den BSD-Stil (Berkeley Software Distribution) – Multicast Unterstützung bereitstellen, indem die entsprechenden Multicast Optionen unterstützt werden. Dies vereinfacht das Portieren vorhandener Multicast Anwendungen auf Windows Sockets 2.
