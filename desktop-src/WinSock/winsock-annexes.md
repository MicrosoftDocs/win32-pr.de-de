# <a name="winsock-annexes"></a>Winsock-Anhänge

Winsock-Anhänge stellen Implementierungs Informationen für gängige Winsock-Protokoll Sammlungen bereit und erörtern, wie jedes Protokoll mit Winsock verwendet werden kann.

Jedes Protokoll in diesem Anhang weist Konventionen, Verhalten oder besondere Features auf, die sich nicht für die generische Implementierung eignen. In diesem Abschnitt werden die Details erläutert, die Entwickler beim implementieren oder verwenden der beschriebenen Protokolle berücksichtigen müssen.

Die Winsock-Anhänge enthalten die folgenden Abschnitte:

-   [Winsock ATM-Anhang](winsock-atm-annex.md)
-   [WinSock-IPX/SPX-Anhang](winsock-ipx-spx-annex.md)
-   [Winsock-TCP/IP-Anhang](winsock-tcp-ip-annex.md)
