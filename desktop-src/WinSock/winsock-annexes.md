# <a name="winsock-annexes"></a>Winsock-1600

Winsock-Schutz stellt Implementierungsinformationen für allgemeine Winsock-Protokollsammlungen bereit und erläutert, wie die einzelnen Protokolle mit Winsock verwendet werden.

Jedes Protokoll in diesem Anhang verfügt über Konventionen, Verhaltensweisen oder spezielle Features, die sich nicht für die generische Implementierung eignen. In diesem Abschnitt werden Details beschrieben, die Entwickler beim Implementieren oder Verwenden der beschriebenen Protokolle berücksichtigen müssen.

Die Winsock-Ausschnitte umfassen die folgenden Abschnitte:

-   [Winsock ATM-Anhang](winsock-atm-annex.md)
-   [Winsock IPX/SPX Anhang](winsock-ipx-spx-annex.md)
-   [Winsock TCP/IP Anhang](winsock-tcp-ip-annex.md)
