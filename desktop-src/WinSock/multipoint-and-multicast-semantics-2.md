# <a name="multipoint-and-multicast-semantics"></a>Multipoint- und Multicastsemantik

In Bezug auf die Unterstützung von Multipoint- und Multicastsemantik in Windows Sockets 2 (Winsock) wurde eine Reihe von vorhandenen und vorgeschlagenen Schemas (einschließlich IP-Multicast, Punkt-zu-Multipoint-Atm-Verbindung, ST-II, T.120, H.320-MCU) untersucht. Um eine konsistente Diskussion der verschiedenen Schemas zu ermöglichen, ist es wichtig, zuerst eine Taxonomie zu erstellen, die die wesentlichen Attribute der einzelnen Schemas kennzeichnet. In diesem Dokument stellt der Begriff Multipoint sowohl *Multipoint* als auch Multicast dar.
