# <a name="multipoint-and-multicast-semantics"></a>Multipoint-und Multicast-Semantik

In der Betrachtung der Multipoint-und Multicast Semantik in Windows Sockets 2 (Winsock) wurden mehrere vorhandene und vorgeschlagene Schemas (einschließlich IP-Multicast, ATM Point-to-Multipoint-Verbindung, St-II, T. 120, H. 320-MCU) untersucht. Um eine kohärente Erörterung der verschiedenen Schemas zu ermöglichen, ist es wichtig, zuerst eine Taxonomie zu erstellen, die die wesentlichen Attribute der einzelnen Schemas kennzeichnet. In diesem Dokument stellt der Begriff *Multipoint* sowohl Multipoint als auch Multicast dar.
