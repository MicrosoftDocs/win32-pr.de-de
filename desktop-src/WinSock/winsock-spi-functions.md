## <a name="32-bit-spi-functions-for-64-bit-platforms"></a>32-Bit-SPI-Funktionen für 64-Bit-Plattformen

> [!Note]  
> Diese Aufrufe sind streng 32-Bit-Versionen von systemeigenen WSC-Funktionsaufrufen für die Verwendung auf 64-Bit-Plattformen durch 32-Bit-Prozesse. Die Definitionen und die Semantik dieser spezifischen 32-Bit-Aufrufe sind identisch mit ihren systemeigenen Entsprechungen.

 



| 32-Bit-Winsock-SPI-Funktion                                                   | BESCHREIBUNG                                                                                                                                                 |
|-------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WSCDeinstallProvider32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscdeinstallprovider32)                      | Entfernt den angegebenen 32-Bit-Transport Dienstanbieter aus der Systemkonfigurations-Datenbank auf einer 64-Bit-Plattform.                                        |
| [**WSCEnableNSProvider32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenablensprovider32)                        | Ändert den Zustand eines angegebenen 32-Bit-Namespace Anbieters auf einer 64-Bit-Plattform.                                                                                |
| [**WSCEnumNameSpaceProviders32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenumnamespaceproviders32)            | Gibt eine Gruppe von verfügbaren 32-Bit-Namespace Anbietern auf einer 64-Bit-Plattform zurück.                                                                                 |
| [**WSCEnumNameSpaceProvidersEx32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenumnamespaceprovidersex32)        | Gibt Informationen zu verfügbaren 32-Bit-Namespace Anbietern auf einer 64-Bit-Plattform zurück.                                                                           |
| [**WSCEnumProtocols32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenumprotocols32)                              | Ruft Informationen zu verfügbaren 32-Bit-Transportprotokollen auf einer 64-Bit-Plattform ab.                                                                      |
| [**WSCGetProviderInfo32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscgetproviderinfo32)                          | Ruft die Daten ab, die einer Informations Klasse für einen 32-Bit-Schicht-Dienstanbieter auf einer 64-Bit-Plattform zugeordnet sind.                                         |
| [**WSCGetProviderPath32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscgetproviderpath32)                          | Ruft den DLL-Pfad für den angegebenen 32-Bit-Anbieter auf einer 64-Bit-Plattform ab.                                                                              |
| [**WSCInstallNameSpace32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscinstallnamespace32)                        | Installiert einen 32-Bit-Namespace Anbieter auf einer 64-Bit-Plattform.                                                                                                  |
| [**WSCInstallNameSpaceEx32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscinstallnamespaceex32)                    | Installiert einen Namespace Anbieter.                                                                                                                              |
| [**WSCInstallProvider64 \_ 32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscinstallprovider64_32)                   | Installiert einen Transport Dienstanbieter in den 32-Bit-und 64-Bit-Systemkonfigurations-Datenbanken auf einer 64-Bit-Plattform.                                       |
| [**WSCInstallProviderAndChains64 \_ 32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscinstallproviderandchains64_32) | Installiert einen Transport Dienstanbieter und seine spezifischen Protokoll Ketten in den 32-Bit-und 64-Bit-Systemkonfigurations-Datenbanken auf einer 64-Bit-Plattform. |
| [**WSCSetProviderInfo32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscsetproviderinfo32)                          | Legt den Datenwert für die angegebene Informations Klasse für einen geschichteten Dienstanbieter auf einer 64-Bit-Plattform fest.                                                    |
| [**WSCUnInstallNameSpace32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscuninstallnamespace32)                    | Deinstalliert einen 32-Bit-Namespace Anbieter auf einer 64-Bit-Plattform.                                                                                                |
| [**WSCUpdateProvider32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscupdateprovider32)                            | Ändert einen 32-Bit-Transport Dienstanbieter in der Systemkonfigurations-Datenbank auf einer 64-Bit-Plattform.                                                     |
| [**WSCWriteNameSpaceOrder32**](/windows/desktop/api/Sporder/nf-sporder-wscwritenamespaceorder32)                  | Ändert die Reihenfolge der verfügbaren Namespace Anbieter in einem 32-Bit-Katalog auf einer 64-Bit-Plattform.                                                               |
