## <a name="32-bit-spi-functions-for-64-bit-platforms"></a>32-Bit-SPI-Funktionen für 64-Bit-Plattformen

> [!Note]  
> Diese Aufrufe sind ausschließlich 32-Bit-Versionen nativer WSC-Funktionsaufrufe zur Verwendung auf 64-Bit-Plattformen durch 32-Bit-Prozesse. Die Definitionen und die Semantik dieser spezifischen 32-Bit-Aufrufe sind identisch mit ihren nativen Gegenstücken.

 



| 32-Bit-Winsock-SPI-Funktion                                                   | Beschreibung                                                                                                                                                 |
|-------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WSCDeinstallProvider32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscdeinstallprovider32)                      | Entfernt den angegebenen 32-Bit-Transportdienstanbieter aus der Systemkonfigurationsdatenbank auf einer 64-Bit-Plattform.                                        |
| [**WSCEnableNSProvider32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenablensprovider32)                        | Ändert den Status eines angegebenen 32-Bit-Namespaceanbieters auf einer 64-Bit-Plattform.                                                                                |
| [**WSCEnumNameSpaceProviders32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenumnamespaceproviders32)            | Gibt einen Satz verfügbarer 32-Bit-Namespaceanbieter auf einer 64-Bit-Plattform zurück.                                                                                 |
| [**WSCEnumNameSpaceProvidersEx32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenumnamespaceprovidersex32)        | Gibt Informationen zu verfügbaren 32-Bit-Namespaceanbietern auf einer 64-Bit-Plattform zurück.                                                                           |
| [**WSCEnumProtocols32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenumprotocols32)                              | Ruft Informationen zu verfügbaren 32-Bit-Transportprotokollen auf einer 64-Bit-Plattform ab.                                                                      |
| [**WSCGetProviderInfo32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscgetproviderinfo32)                          | Ruft die Daten ab, die einer Informationsklasse für einen 32-Bit-Mehrschicht-Dienstanbieter auf einer 64-Bit-Plattform zugeordnet sind.                                         |
| [**WSCGetProviderPath32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscgetproviderpath32)                          | Ruft den DLL-Pfad für den angegebenen 32-Bit-Anbieter auf einer 64-Bit-Plattform ab.                                                                              |
| [**WSCInstallNameSpace32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscinstallnamespace32)                        | Installiert einen 32-Bit-Namespaceanbieter auf einer 64-Bit-Plattform.                                                                                                  |
| [**WSCInstallNameSpaceEx32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscinstallnamespaceex32)                    | Installiert einen Namespaceanbieter.                                                                                                                              |
| [**WSCInstallProvider64 \_ 32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscinstallprovider64_32)                   | Installiert einen Transportdienstanbieter in den 32-Bit- und 64-Bit-Systemkonfigurationsdatenbanken auf einer 64-Bit-Plattform.                                       |
| [**WSCInstallProviderAndChains64 \_ 32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscinstallproviderandchains64_32) | Installiert einen Transportdienstanbieter und seine spezifischen Protokollketten in den 32-Bit- und 64-Bit-Systemkonfigurationsdatenbanken auf einer 64-Bit-Plattform. |
| [**WSCSetProviderInfo32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscsetproviderinfo32)                          | Legt den Datenwert für die angegebene Informationsklasse für einen mehrschichtigen Dienstanbieter auf einer 64-Bit-Plattform fest.                                                    |
| [**WSCUnInstallNameSpace32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscuninstallnamespace32)                    | Deinstalliert einen 32-Bit-Namespaceanbieter auf einer 64-Bit-Plattform.                                                                                                |
| [**WSCUpdateProvider32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscupdateprovider32)                            | Ändert einen 32-Bit-Transportdienstanbieter in der Systemkonfigurationsdatenbank auf einer 64-Bit-Plattform.                                                     |
| [**WSCWriteNameSpaceOrder32**](/windows/desktop/api/Sporder/nf-sporder-wscwritenamespaceorder32)                  | Ändert die Reihenfolge der verfügbaren Namespaceanbieter in einem 32-Bit-Katalog auf einer 64-Bit-Plattform.                                                               |
