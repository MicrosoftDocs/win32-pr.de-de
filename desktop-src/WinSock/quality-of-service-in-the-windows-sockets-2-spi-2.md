# <a name="quality-of-service-in-the-windows-sockets-2-spi"></a>Quality of Service in der Windows Sockets 2 SPI

Quality of Service (QoS) wird in Windows über verschiedene QoS-Komponenten implementiert, die unter Windows Server 2003, Windows XP und Windows 2000 unterstützt werden. Eine ausführliche Erläuterung von QoS und seinen Komponenten finden Sie in einem Abschnitt für Quality of Service, der sich im Microsoft Windows Software Development Kit (SDK) befindet. Jede QoS-Komponente und die Rolle, die sie in der QoS-Implementierung spielt, werden in diesem QoS-Abschnitt mit zusätzlichen Anleitungen für die Implementierung von QoS-fähigen Anwendungen und Diensten erläutert. Ausführlichere Informationen [finden Sie im Abschnitt Quality of Service (QoS)](/previous-versions/windows/desktop/qos/qos-start-page) des Windows SDK.

In diesem Abschnitt werden die Quality of Service, die Winsock-Entwicklern zur Verfügung stehen. In der folgenden Liste werden die Themen in diesem Abschnitt beschrieben:

-   [Verwendungsmodell](usage-model-2.md)
-   [QoS-Updates](qos-updates-2.md)
-   [Standardwerte in der SPI](default-values-in-the-spi-2.md)
-   [QoS-Vorlagen in der SPI](qos-templates-in-the-spi-2.md)
