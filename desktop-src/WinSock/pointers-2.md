# <a name="pointers-windows-sockets-2"></a>Zeiger (Windows Sockets 2)

Alle Zeiger, die von Anwendungen mit Windows Sockets verwendet werden, sollten viel zu tun haben, obwohl dies nur für 16-Bit-Anwendungen relevant ist und in einem 32-Bit bedeutungslos ist. Um dies zu vereinfachen, werden Datentyp Definitionen, z. b. **lphostent** , bereitgestellt.
