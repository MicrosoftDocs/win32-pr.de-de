# <a name="usage-model"></a>Verwendungs Modell

Verwendungs-und Implementierungs Leit Faden für QoS-aktivierte Anwendungen und Dienste werden in einem separaten Abschnitt des Microsoft Windows Software Development Kit (SDK) bereitgestellt, das für QoS reserviert ist. Ausführlichere Informationen zum QoS-Verwendungs Modell finden Sie im Abschnitt [Quality of Service (QoS)](/previous-versions/windows/desktop/qos/qos-start-page) des Windows SDK.
