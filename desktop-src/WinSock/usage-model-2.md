# <a name="usage-model"></a>Nutzungsmodell

Nutzungs- und Implementierungsleitfäden für QoS-fähige Anwendungen und Dienste werden in einem separaten Abschnitt im Microsoft Windows Software Development Kit (SDK) bereitgestellt, das für QoS vorgesehen ist. Ausführlichere Informationen zum QoS-Nutzungsmodell finden Sie im Abschnitt [Quality of Service (QoS)](/previous-versions/windows/desktop/qos/qos-start-page) des Windows SDK.
