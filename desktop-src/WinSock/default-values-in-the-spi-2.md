# <a name="default-values-in-the-spi"></a>Standardwerte in der SPI

Die Quality of Service (QoS)-Implementierungsdetails werden in einem separaten, dedizierten QoS-Abschnitt des Microsoft Windows Software Development Kit (SDK) bereitgestellt. Informationen zu den QoS-Verwendungs Richtlinien in Bezug auf die Standardwerte finden Sie im Abschnitt [Quality of Service](/previous-versions/windows/desktop/qos/qos-start-page) des Windows SDK.
