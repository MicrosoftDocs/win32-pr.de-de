# <a name="sol_socket-socket-options"></a>SOL SOCKET Socket Options (SOL \_ SOCKET-Socketoptionen)

In den folgenden Tabellen werden **SOL \_ SOCKET-Socketoptionen** beschrieben. Weitere Informationen zum Abrufen und Festlegen von Socketoptionen finden Sie auf den Referenzseiten der Funktionen [**getsockopt**](/windows/desktop/api/winsock/nf-winsock-getsockopt) und [**setsockopt.**](/windows/desktop/api/winsock/nf-winsock-setsockopt)

Verwenden Sie die Funktionen [**WSAEnumProtocols, WSCEnumProtocols oder WSCEnumProtocols32,**](/windows/desktop/api/Winsock2/nf-winsock2-wsaenumprotocolsa)um Protokolle aufzulisten und unterstützte Eigenschaften für jedes [**installierte**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenumprotocols32) Protokoll zu ermitteln. [](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenumprotocols)

Einige Socketoptionen erfordern mehr Erklärung, als diese Tabellen vermitteln können. diese Optionen enthalten Links zu zusätzlichen Seiten.

> [!Note]  
> Alle **SOL \_ SOCKET-Socketoptionen** gelten gleichermaßen für IPv4 und IPv6 (mit Ausnahme von SO \_ BROADCAST, da Broadcast in IPv6 nicht implementiert ist).

## <a name="remarks"></a>Hinweise

Die **SOL \_ SOCKET-Socketoptionen** werden in mehreren Winsock-Headerdateien definiert:

-   *Winsock2.h*
-   *Mswsock.h*
-   *Ws2def.h*

Auf dem Microsoft Windows Software Development Kit (SDK), das für Windows Vista und höher veröffentlicht wurde, hat sich die Organisation der Headerdateien geändert, und **die SOL \_ SOCKET-Ebene** wird in der *Headerdatei Ws2def.h* definiert, die automatisch in der *Winsock2.h-Headerdatei* enthalten ist. Einige der **SOL \_ SOCKET-Socketoptionen** werden in den *Headerdateien Winsock2.h* und *Mswsock.h* definiert. Die **verbleibenden SOL \_ SOCKET-Socketoptionen** werden in der *Ws2def.h-Headerdatei* definiert, die automatisch in der *Winsock2.h-Headerdatei* enthalten ist. *Ws2def.h* sollte nie direkt verwendet werden.

Im Platform Software Development Kit (SDK), das für Windows Server 2003 und Windows XP veröffentlicht wurde, wird die **SOL \_ SOCKET-Ebene** in der *Winsock2.h-Headerdatei* definiert. Die **SOL \_ SOCKET-Socketoptionen** werden in den *Headerdateien Winsock2.h* und *Mswsock.h* definiert.
