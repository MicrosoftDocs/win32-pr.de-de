# <a name="sol_socket-socket-options"></a>Optionen des Sol- \_ socketsockets

In den folgenden Tabellen werden die Optionen für den **Sol- \_ socketsocket** Weitere Informationen zum Ermitteln und Festlegen von Socketoptionen finden Sie auf den Referenzseiten zu den Funktionen [**getsockopt**](/windows/desktop/api/winsock/nf-winsock-getsockopt) und [**setsockopt**](/windows/desktop/api/winsock/nf-winsock-setsockopt) .

Verwenden Sie die [**wsaenumprotokolls**](/windows/desktop/api/Winsock2/nf-winsock2-wsaenumprotocolsa)-, [**wscenumprotokolle**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenumprotocols)-oder [**WSCEnumProtocols32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenumprotocols32) -Funktion, um Protokolle aufzulisten und die unterstützten Eigenschaften für jedes installierte Protokoll zu ermitteln.

Einige Socketoptionen erfordern eine ausführlichere Erläuterung, als diese Tabellen vermitteln können. Diese Optionen enthalten Links zu weiteren Seiten.

> [!Note]  
> Alle Optionen für den **Sol- \_ socketsocket** gelten gleichermaßen für IPv4 und IPv6 (mit Ausnahme von \_ Broadcast, da Broadcast nicht in IPv6 implementiert ist).

## <a name="remarks"></a>Bemerkungen

Die Optionen für den **Sol- \_ socketsocket** werden in mehreren Winsock-Header Dateien definiert:

-   *Winsock2. h*
-   *Mtauscht. h*
-   *Ws2def. h*

Im Microsoft Windows Software Development Kit (SDK), das für Windows Vista und höher veröffentlicht wurde, hat sich die Organisation der Header Dateien geändert, und die Ebene " **Sol \_ Socket** " wird in der Header Datei " *Ws2def. h* " definiert, die automatisch in der Header Datei " *Winsock2. h* " enthalten ist. Einige der Optionen für den **Sol- \_ socketsocket** sind in den Header Dateien *Winsock2. h* und *mtauscht. h* definiert. Die übrigen Optionen für den **Sol- \_ socketsocket** werden in der Header Datei " *Ws2def. h* " definiert, die automatisch in der Header Datei " *Winsock2. h* " enthalten ist. *Ws2def. h* sollte nie direkt verwendet werden.

Auf dem Platform Software Development Kit (SDK), das für Windows Server 2003 und Windows XP veröffentlicht wurde, wird die Ebene des **Sol- \_ Sockets** in der Header Datei " *Winsock2. h* " definiert. Die Optionen für den **Sol- \_ socketsocket** werden in den Header Dateien *Winsock2. h* und *mtauscht. h* definiert.
