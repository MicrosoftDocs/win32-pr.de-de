# <a name="internet-protocol-version-6-ipv6"></a>Internetprotokoll Version 6 (IPv6)

Es ist wichtig sicherzustellen, dass neue Winsock-Anwendungen und vorhandene Anwendungen vollständig mit IPv6 kompatibel sind. Die Verfügbarkeit des IPv4-Adressraums für neue IPv4-Adress Zuordnungen war für Asien und den Pazifik in 2011 erschöpft. Es wird erwartet, dass andere Teile der Welt in wenigen Jahren erschöpft sind.

Ein wachsender Prozentsatz der neuen Websites und Dienste ist über IPv6-Adressen verfügbar. Viele Internet Benutzer in den neuen Märkten sind von IPv6 für den Internet Zugang abhängig.

Microsoft hat eine lange Verpflichtung zum unterstützen von IPv6. Vollständige IPv6-Unterstützung wird unter Windows XP mit Service Pack 1 (SP1) und höher bereitgestellt.

Dieses Dokument enthält Informationen über die Winsock-Unterstützung für IPv6:

-   [Verwenden von IPv6](using-ipv6-2.md)
-   [Verwenden von Internet Explorer für den Zugriff auf IPv6-Websites](using-internet-explorer-to-access-ipv6-web-sites-2.md)
-   [Empfohlene Konfigurationen für IPv6](recommended-configurations-2.md)
-   [Weitere IPv6-Themen](additional-topics-2.md)

Weitere Informationen zum Hinzufügen von IPv6-Funktionen zu Ihren Windows Sockets-Anwendungen finden Sie unter [IPv6-Handbuch für Windows Sockets-Anwendungen](ipv6-guide-for-windows-sockets-applications-2.md).

Eine Einführung in das IPv6-Protokoll sowie Übersichten zur Bereitstellung und zu IPv6-Übergangs Technologien finden Sie auf TechNet unter [Microsoft Internet Protocol Version 6 (IPv6)](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/dd379473(v=ws.10)).
