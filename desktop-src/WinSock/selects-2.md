# <a name="selects"></a>Wählt

Die " [**Select**](/windows/desktop/api/Winsock2/nf-winsock2-select) "-Funktion in der ursprünglichen "Berkeley Software Distribution (BSD)"-Socketschnittstelle war das standardmäßige (und einzige) Mittel zum Abrufen von Netzwerk Ereignis hinweisen. Für jeden Socket können Informationen zu Lese-, Schreib-oder Fehlerstatus abgerufen oder gewartet werden. Weitere Informationen finden [**Sie unter wspselect**](/previous-versions/windows/desktop/legacy/ms742289(v=vs.85)) . Beachten Sie, dass das Netzwerk Ereignis FD- \_ QoS von diesem Ansatz nicht abgerufen werden kann.
