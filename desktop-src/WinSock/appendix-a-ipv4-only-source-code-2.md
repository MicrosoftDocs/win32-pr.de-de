# <a name="appendix-a-ipv4-only-source-code"></a>Anhang A: nur-IPv4-Quellcode

Dieser Anhang enthält den Quellcode für die Platform Software Development Kit (SDK)-Beispiele simplec. c und simples. c, die in diesem IPv6-Handbuch als Beispiele verwendet werden:

-   [Nur-IPv4-Client Code](ipv4-only-client-code-2.md)
-   [Nur-IPv4-Server Code](ipv4-only-server-code-2.md)

> [!WARNING]
>
> Dieser Code ist in dieser Anleitung enthalten, um den in Anhang B bereitgestellten, in der IP-Version unabhängigen Code zu vergleichen. Dieser Beispielcode dient zu Vergleichszwecken und zum Bereitstellen eines klar und sofort verfügbaren Verweises beim Lesen dieses IPv6-Handbuchs. Eine Versions unabhängige Version dieses Quellcodes für die IP-Version finden Sie in [Anhang B: Quell Code für die unabhängige IP-Version](appendix-b-ip-version-agnostic-source-code-2.md).

 

> [!Note]  
> Wenn Sie den Quellcode in Anhang A (nur IPv4) und Anhang B (IP-Versions agnostisch) vergleichen, erhalten Sie einen Eindruck von den Änderungen, die erforderlich sind, um die Windows Sockets-Anwendung so zu ändern, dass Unterstützung für IPv6 hinzugefügt wird.
