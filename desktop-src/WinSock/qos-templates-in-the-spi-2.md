# <a name="qos-templates-in-the-spi"></a>QoS-Vorlagen in der SPI

Die Implementierungsdetails für Quality of Service (QoS) werden im Microsoft Windows Software Development Kit (SDK) in einem separaten, dedizierten QoS-Abschnitt bereitgestellt. Informationen zu QoS-Nutzungsrichtlinien, einschließlich verfügbarer QoS-Vorlagen und der zugehörigen [**FLOWSPEC-Werte,**](/windows/win32/api/qos/ns-qos-flowspec) finden Sie im Abschnitt [Quality of Service](/previous-versions/windows/desktop/qos/qos-start-page) des Windows SDK.
