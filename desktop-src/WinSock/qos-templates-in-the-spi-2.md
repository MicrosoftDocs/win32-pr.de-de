# <a name="qos-templates-in-the-spi"></a>QoS-Vorlagen in der SPI

Die Quality of Service (QoS)-Implementierungsdetails werden in einem separaten, dedizierten QoS-Abschnitt des Microsoft Windows Software Development Kit (SDK) bereitgestellt. Informationen zu QoS-Verwendungs Richtlinien, einschließlich der verfügbaren QoS-Vorlagen und der zugehörigen [**Flowspec**](/windows/win32/api/qos/ns-qos-flowspec) -Werte, finden Sie im Abschnitt [Quality of Service](/previous-versions/windows/desktop/qos/qos-start-page) des Windows SDK.
