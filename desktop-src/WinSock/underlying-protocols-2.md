# <a name="underlying-protocols-for-ipv6-winsock-applications"></a>Zugrunde liegende Protokolle für IPv6-Winsock-Anwendungen

Einige Protokolle, von denen Ihre Anwendung für bestimmte Dienste abhängt, z. b. remote Prozedur Aufruf (RPC), verfügen möglicherweise über von der IP-Version abhängige Funktionen und Strukturen, die eine Änderung Ihrer Anwendung im Hinblick auf Ihre Nutzung erfordern.

Ein Teil des Prozesses zum Ändern Ihres Codes zur Unterstützung von IPv6 sollte die Entscheidung umfassen, ob die Verwendung der zugrunde liegenden Protokolle (aus der Perspektive des Stapels, diese Protokolle gelten als Protokolle der höheren Schicht) erforderlich sind, damit IPv6 ordnungsgemäß verwendet werden kann.
