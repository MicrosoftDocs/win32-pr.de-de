# <a name="underlying-protocols-for-ipv6-winsock-applications"></a>Zugrunde liegende Protokolle für IPv6-Winsock-Anwendungen

Einige Protokolle, von denen Ihre Anwendung für bestimmte Dienste abhängig ist, z. B. Remoteprozeduraufrufe (REMOTE Procedure Call, RPC), verfügen möglicherweise über von der IP-Version abhängige Funktionen und Strukturen, von denen Sie Ihre Anwendung hinsichtlich ihrer Verwendung ändern müssen.

Ein Teil des Prozesses zum Ändern ihres Codes zur Unterstützung von IPv6 sollte die Bestimmung umfassen, ob die Verwendung der zugrunde liegenden Protokolle (aus Sicht des Stapels, diese Protokolle gelten als Protokolle auf höherer Ebene) geändert werden muss, um IPv6 ordnungsgemäß zu verwenden.
