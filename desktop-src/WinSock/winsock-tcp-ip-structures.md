# <a name="winsock-tcpip-structures"></a>Winsock-TCP/IP-Strukturen

Die folgende Liste enthält präzise Beschreibungen der einzelnen Winsock-TCP/IP-Strukturen. Wenn Sie weitere Informationen zu einer Struktur haben, klicken Sie auf den Struktur Namen.



| TCP/IP-Struktur                                 | BESCHREIBUNG                                                                                                                              |
|--------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**Schnittstellen \_ Informationen**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info)      | Wird zusammen mit dem IOCTL-Befehl verwendet, um Informationen zu einer Schnittstellen-IP-Adresse abzurufen.                                                         |
| [**Schnittstellen \_ Info \_ Ex**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info_ex) | Dient zum Abrufen von Informationen zu einer Schnittstellen-IP-Adresse, einschließlich IPv6-Adressen, mit dem **\_ \_ \_ ioctl-Befehl der "sio Get Interface List** " |
| [**sockaddr- \_ gen**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-sockaddr_gen)            | Stellt generische socketadressinformationen bereit. Wird mit der [**Schnittstellen \_ Info**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info) -Struktur verwendet.                        |
