# <a name="winsock-tcpip-structures"></a>Winsock-TCP/IP-Strukturen

Die folgende Liste enthält präzise Beschreibungen der einzelnen Winsock-TCP/IP-Strukturen. Klicken Sie auf den Strukturnamen, um weitere Informationen zu einer beliebigen Struktur zu finden.



| TCP/IP-Struktur                                 | Beschreibung                                                                                                                              |
|--------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_SCHNITTSTELLENINFORMATIONEN**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info)      | Wird mit dem Befehl ioctl verwendet, um Informationen zu einer Schnittstellen-IP-Adresse abzurufen.                                                         |
| [**INTERFACE \_ INFO \_ EX**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info_ex) | Wird mit dem **\_ SIO GET \_ INTERFACE \_ LIST-IOCTL-Befehl** verwendet, um Informationen zu einer Schnittstellen-IP-Adresse abzurufen, einschließlich IPv6-Adressen. |
| [**sockaddr \_ gen**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-sockaddr_gen)            | Stellt generische Socketadresseninformationen bereit. Wird mit der [**INTERFACE \_ INFO-Struktur**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info) verwendet.                        |
