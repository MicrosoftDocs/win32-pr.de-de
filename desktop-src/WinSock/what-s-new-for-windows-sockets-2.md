## <a name="updated-for-windows-8-and-windows-server-2012"></a>Aktualisiert für Windows 8 und Windows Server 2012

Mit Microsoft Windows 8 und Windows Server 2012 werden neue Windows Sockets-Programmier Elemente eingeführt.

Eine Reihe von Erweiterungen für hoch Geschwindigkeits Netzwerke sind für eine höhere Netzwerkleistung mit geringerer Latenz und Jitter verfügbar. Diese Erweiterungen, die primär für Server Anwendungen vorgesehen sind, verwenden vorab registrierte Datenpuffer und Vervollständigungs Warteschlangen, um die Leistung zu steigern
