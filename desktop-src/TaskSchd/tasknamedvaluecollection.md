# <a name="tasknamedvaluecollection-object"></a>Tasknamedvaluecollection-Objekt

Skript Objekt, das eine Auflistung von [**tasknamedvaluepair**](tasknamedvaluepair.md) -Objektname-Wert-Paaren enthält.

## <a name="members"></a>Member

Das **tasknamedvaluecollection** -Objekt verfügt über diese Typen von Membern:

-   [Methoden](#methods)
-   [Eigenschaften](#properties)

### <a name="methods"></a>Methoden

Das **tasknamedvaluecollection** -Objekt verfügt über diese Methoden.



| Methode                                            | BESCHREIBUNG                                                        |
|:--------------------------------------------------|:-------------------------------------------------------------------|
| [**Klartext**](tasknamedvaluecollection-clear.md)   | Löscht die gesamte Auflistung von Name-Wert-Paaren.<br/>       |
| [**Stelle**](tasknamedvaluecollection-create.md) | Erstellt ein Name-Wert-Paar in der Auflistung.<br/>            |
| [**Aufgeh**](tasknamedvaluecollection-remove.md) | Entfernt ein ausgewähltes Name-Wert-Paar aus der Auflistung.<br/> |



 

### <a name="properties"></a>Eigenschaften

Das **tasknamedvaluecollection** -Objekt verfügt über diese Eigenschaften.



| Eigenschaft                                                   | BESCHREIBUNG                                                        |
|:-----------------------------------------------------------|:-------------------------------------------------------------------|
| [**Countdown**](tasknamedvaluecollection-count.md)<br/> | Ruft die Anzahl der Name-Wert-Paare in der-Auflistung ab.<br/>  |
| [**Element**](tasknamedvaluecollection-item.md)<br/>   | Ruft das angegebene Name-Wert-Paar aus der Auflistung ab.<br/> |
