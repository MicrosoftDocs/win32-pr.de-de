# <a name="registrationinfoxmltext-property"></a>RegistrationInfo.XmlText-Eigenschaft

Ruft für die Skripterstellung eine XML-formatierte Version der Registrierungsinformationen für den Task ab oder legt diese fest.

## <a name="syntax"></a>Syntax


```VB
RegistrationInfo.XmlText As String
```



## <a name="property-value"></a>Eigenschaftswert

Eine XML-formatierte Version der Taskregistrierungsinformationen.
