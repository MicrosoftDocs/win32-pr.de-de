# <a name="registeredtaskenabled-property"></a>Registeredtask. aktivierte Eigenschaft

Ruft bei der Skripterstellung einen booleschen Wert ab, der angibt, ob die registrierte Aufgabe aktiviert ist, oder legt diesen fest.

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.Enabled As Boolean
```



## <a name="property-value"></a>Eigenschaftswert

Ein boolescher Wert, der angibt, ob die registrierte Aufgabe aktiviert ist.
