# <a name="registeredtaskcollectionitem-property"></a>RegisteredTaskCollection.Item-Eigenschaft

Ruft für die Skripterstellung den angegebenen registrierten Task aus der Auflistung ab.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
RegisteredTaskCollection.Item( _
  ByVal Index _
) As RegisteredTask
```



## <a name="property-value"></a>Eigenschaftswert

Ein [**RegisteredTask-Objekt,**](registeredtask.md) das die registrierte Aufgabe enthält.
