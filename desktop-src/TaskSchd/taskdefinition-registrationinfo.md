# <a name="taskdefinitionregistrationinfo-property"></a>Taskdefinition. RegistrationInfo-Eigenschaft

Ruft bei der Skripterstellung die Registrierungsinformationen ab, die zum Beschreiben einer Aufgabe verwendet werden, z. b. die Beschreibung der Aufgabe, den Autor der Aufgabe und das Datum, an dem die Aufgabe registriert ist, oder legt diese fest.

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.RegistrationInfo As RegistrationInfo
```



## <a name="property-value"></a>Eigenschaftswert

Die Registrierungsinformationen, die verwendet werden, um eine Aufgabe zu beschreiben, z. b. die Beschreibung der Aufgabe, der Autor der Aufgabe und das Datum, an dem die Aufgabe registriert ist.
