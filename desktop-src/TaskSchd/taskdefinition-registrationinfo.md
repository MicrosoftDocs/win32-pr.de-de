# <a name="taskdefinitionregistrationinfo-property"></a>TaskDefinition.RegistrationInfo (Eigenschaft)

F端r die Skripterstellung ruft die Registrierungsinformationen ab, die zum Beschreiben einer Aufgabe verwendet werden, z. B. die Beschreibung der Aufgabe, den Autor der Aufgabe und das Datum, an dem die Aufgabe registriert wird, oder legt diese fest.

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.RegistrationInfo As RegistrationInfo
```



## <a name="property-value"></a>Eigenschaftswert

Die Registrierungsinformationen, die verwendet werden, um eine Aufgabe zu beschreiben, z. B. die Beschreibung der Aufgabe, den Autor der Aufgabe und das Datum, an dem die Aufgabe registriert wird.
