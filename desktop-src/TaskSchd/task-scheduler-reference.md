# <a name="task-scheduler-reference"></a>Taskplaner Referenz

In diesem Abschnitt werden die APIs, Skript Objekte und XML-Schemas beschrieben, die von der Taskplaner bereitgestellt werden.

API-Themen enthalten eine Beschreibung der API, die Deklarations Syntax der API, Hinweise, die besondere Bedingungen für die API beschreiben, und einen Anforderungs Block, der beschreibt, welche Plattform, Header Dateien und Bibliotheken erforderlich sind.

Zu den Themen zu Skript Objekten gehören eine Beschreibung des Objekts, die Deklarations Syntax des Objekts, Hinweise, die besondere Bedingungen für das Objekt beschreiben, und ein Anforderungs Block, der beschreibt, welche Plattform-und Typbibliotheken erforderlich sind.

Zu den XML-Schema Themen zählen eine Beschreibung der Informationen über das Element, das übergeordnete Element, das untergeordnete Element und das Attribut sowie Hinweise, die besondere Bedingungen beschreiben
