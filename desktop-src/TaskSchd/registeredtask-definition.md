# <a name="registeredtaskdefinition-property"></a>Registeredtask. Definition (Eigenschaft)

Ruft bei der Skripterstellung die Definition der Aufgabe ab.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.Definition As TaskDefinition
```



## <a name="property-value"></a>Eigenschaftswert

Die Definition der Aufgabe.
