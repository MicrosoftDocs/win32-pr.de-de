# <a name="networksettingsid-property"></a>NetworkSettings.ID-Eigenschaft

Ruft bei der Skripterstellung einen GUID-Wert ab, der ein Netzwerk Profil identifiziert, oder legt ihn fest.

## <a name="syntax"></a>Syntax


```VB
NetworkSettings.Id As String
```



## <a name="property-value"></a>Eigenschaftswert

Ein GUID-Wert, der ein Netzwerk Profil identifiziert.
