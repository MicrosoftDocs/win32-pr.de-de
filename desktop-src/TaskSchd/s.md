<span id="_msb_scheduled_tasks_folder_gly"></span><span id="_MSB_SCHEDULED_TASKS_FOLDER_GLY"></span>**Ordner "Geplante Aufgaben"**
</dt> <dd>

Ein ordner, der vom Taskplaner erstellt wird, der alle Aufgaben enthält, die für die Ausführung auf dem Computer geplant sind.

Um diesen Ordner in den Betriebssystemen Windows Server 2003, Windows XP oder Windows 2000, zu öffnen, öffnen Sie Systemsteuerung , und doppelklicken Sie **dann** auf **Geplante Aufgaben**.
