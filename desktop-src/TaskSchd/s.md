Ein Ordner, der vom Taskplaner-Dienst erstellt wird und alle Tasks enthält, die für die Ausführung auf dem Computer geplant sind.

Öffnen Sie die **Systemsteuerung**, und doppelklicken Sie dann auf **geplante Aufgaben**, um diesen Ordner in den Betriebssystemen Windows Server 2003, Windows XP oder Windows 2000 anzuzeigen.
