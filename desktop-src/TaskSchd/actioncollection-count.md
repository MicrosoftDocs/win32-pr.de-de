# <a name="actioncollectioncount-property"></a>Aktions Collection. Count-Eigenschaft

Bei der Skripterstellung wird die Anzahl der Aktionen in der Auflistung abgerufen.

## <a name="syntax"></a>Syntax


```VB
ActionCollection.Count As long
```



## <a name="property-value"></a>Eigenschaftswert

Die Anzahl von Aktionen in der Auflistung. Die Sammlung kann bis zu 32 Aktionen enthalten.
