# <a name="registeredtaskpath-property"></a>Registeredtask. Path-Eigenschaft

Bei der Skripterstellung wird der Pfad abgerufen, in dem die registrierte Aufgabe gespeichert wird.

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.Path As String
```



## <a name="property-value"></a>Eigenschaftswert

Der Pfad, in dem die registrierte Aufgabe gespeichert wird.
