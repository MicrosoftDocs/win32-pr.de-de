# <a name="actionid-property"></a>Action.Id-Eigenschaft

Für die Skripterstellung ruft den Bezeichner der Aktion ab oder legt diesen fest.

## <a name="syntax"></a>Syntax


```VB
Action.Id As String
```



## <a name="property-value"></a>Eigenschaftswert

Der benutzerdefinierte Bezeichner für die Aktion. Dieser Bezeichner wird von der Taskplaner protokollierungszwecken verwendet.

## <a name="remarks"></a>Hinweise

Informationen zur Zusammenarbeit von Aktionen und Aufgaben finden Sie unter [Aufgabenaktionen](task-actions.md).
