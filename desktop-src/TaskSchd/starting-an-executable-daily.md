# <a name="starting-an-executable-daily"></a>Täglich eine ausführbare Datei starten

Wenn Sie eine Aufgabe schreiben, die eine ausführbare Datei täglich startet, definieren Sie einen täglichen und eine ausführbare Aktion.

## <a name="daily-triggers"></a>Tägliche Trigger

Tägliche Trigger verwenden Ihre Start Grenze, um den Trigger zu aktivieren und die Uhrzeit anzugeben, zu der die Aufgabe ausgeführt wird. Nachdem der Trigger aktiviert wurde, wird das triggerintervall verwendet, um anzugeben, ob die Aufgabe täglich, täglich, an jedem dritten Tag oder so weiter ausgeführt wird.
