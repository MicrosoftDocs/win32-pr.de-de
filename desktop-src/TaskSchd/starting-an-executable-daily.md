# <a name="starting-an-executable-daily"></a>Tägliches Starten einer ausführbaren Datei

Das Schreiben einer Aufgabe, die täglich eine ausführbare Datei startet, erfolgt durch Definieren eines täglichen Triggers und einer ausführbaren Aktion.

## <a name="daily-triggers"></a>Tägliche Trigger

Tägliche Trigger verwenden ihre Startgrenze, um den Trigger zu aktivieren und die Tageszeit anzugeben, zu der die Aufgabe ausgeführt wird. Nachdem der Trigger aktiviert wurde, wird das Triggerintervall verwendet, um anzugeben, ob die Aufgabe täglich, jeden anderen Tag, jeden dritten Tag oder so weiter ausgeführt wird.
