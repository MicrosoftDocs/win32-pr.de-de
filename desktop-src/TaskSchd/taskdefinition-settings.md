# <a name="taskdefinitionsettings-property"></a>Taskdefinition. Settings-Eigenschaft

Ruft bei der Skripterstellung die Einstellungen ab, die definieren, wie der Taskplaner Dienst die Aufgabe ausführt, oder legt diese fest.

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.Settings As TaskSettings
```



## <a name="property-value"></a>Eigenschaftswert

Die Einstellungen, die definieren, wie der Taskplaner Dienst den Task ausführt.
