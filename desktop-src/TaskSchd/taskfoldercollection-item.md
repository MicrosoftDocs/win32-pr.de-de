# <a name="taskfoldercollectionitem-property"></a>Taskfoldercollection. Item-Eigenschaft

Bei der Skripterstellung wird der angegebene Ordner aus der Auflistung abgerufen.

## <a name="syntax"></a>Syntax


```VB
TaskFolderCollection.Item( _
  ByVal Index _
) As TaskFolder
```



## <a name="property-value"></a>Eigenschaftswert

Ein [**taskfolder**](taskfolder.md) -Objekt, das den angeforderten Ordner darstellt.
