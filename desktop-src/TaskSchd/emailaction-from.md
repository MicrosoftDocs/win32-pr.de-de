# <a name="emailactionfrom-property"></a>EmailAction.From-Eigenschaft

\[Dieses Objekt wird nicht mehr unterstützt. Verwenden Sie IExecAction mit dem [**PowerShell-Cmdlet Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) als Problemumgehung.\]

Für die Skripterstellung ruft die E-Mail-Adresse ab, von der aus Sie die E-Mail senden möchten, oder legt sie fest.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
EmailAction.From As String
```



## <a name="property-value"></a>Eigenschaftswert

Die E-Mail-Adresse, von der Sie die E-Mail senden möchten.
