# <a name="task-scheduler-10-enumerated-types"></a>Taskplaner 1,0-Enumerationstypen

\[\[Diese API kann in nachfolgenden Versionen des Betriebssystems oder des Produkts geändert werden oder nicht verfügbar sein. Verwenden Sie stattdessen die [Taskplaner 2,0-Schnittstellen](task-scheduler-2-0-interfaces.md) oder die [Taskplaner 2,0-Enumerationstypen](task-scheduler-2-0-enumerated-types.md) . \]\]

Beschreibt die Enumerationstypen, die die Konstanten definieren, die von den Taskplaner 1,0-APIs verwendet werden.


Die folgenden Enumerationstypen werden von Taskplaner 1,0 eingeführt.



| Name                                             | BESCHREIBUNG                                                                        |
|--------------------------------------------------|------------------------------------------------------------------------------------|
| [**Task \_ \_ auslösertyp**](/windows/desktop/api/Mstask/ne-mstask-task_trigger_type) | Definiert die Typen von [*Triggern*](t.md) , die einer Aufgabe zugeordnet sind. |
| [**Taskpage**](/windows/desktop/api/Mstask/ne-mstask-taskpage)                     | Definiert den Typ der abzurufenden Aufgabenseite.                                     |
