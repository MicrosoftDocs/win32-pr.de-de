# <a name="task-scheduler-10-enumerated-types"></a>Taskplaner 1.0-Aufzähltypen

\[\[Diese API kann in nachfolgenden Versionen des Betriebssystems oder Produkts geändert oder nicht verfügbar sein. Verwenden Sie stattdessen [Taskplaner 2.0-Schnittstellen](task-scheduler-2-0-interfaces.md) [oder Taskplaner 2.0-Aufzähltypen.](task-scheduler-2-0-enumerated-types.md) \]\]

Beschreibt die aufzählten Typen, die die Konstanten definieren, die von den Taskplaner 1.0-APIs verwendet werden.


Die folgenden aufzählten Typen werden von Taskplaner 1.0 eingeführt.



| Name                                             | Beschreibung                                                                        |
|--------------------------------------------------|------------------------------------------------------------------------------------|
| [**\_ \_ TASKTRIGGERTYP**](/windows/desktop/api/Mstask/ne-mstask-task_trigger_type) | Definiert die Einem Task [*zugeordneten*](t.md) Triggertypen. |
| [**TASKPAGE**](/windows/desktop/api/Mstask/ne-mstask-taskpage)                     | Definiert den Typ der abzurufenden Aufgabenseite.                                     |
