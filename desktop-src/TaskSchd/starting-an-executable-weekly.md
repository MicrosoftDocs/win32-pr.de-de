# <a name="starting-an-executable-weekly"></a>Wöchentliches Starten einer ausführbaren Datei

Das Schreiben einer Aufgabe, die eine ausführbare Datei wöchentlich startet, erfolgt durch Definieren eines wöchentlichen Auslösers und einer ausführbaren Aktion.

## <a name="weekly-triggers"></a>Wöchentliche Trigger

Wöchentliche Trigger verwenden Ihre Start Grenze, um den Trigger zu aktivieren und die Uhrzeit anzugeben, zu der die Aufgabe ausgeführt wird. Wenn der-Triggervorgang aktiviert ist, gibt der-Triggerwert auch an, wann die Aufgabe während der Woche ausgeführt wird und ob die Aufgabe wöchentlich, jede Woche, jede dritte Woche usw. ausgeführt wird.
