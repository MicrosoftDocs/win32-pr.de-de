# <a name="taskfolderpath-property"></a>TaskFolder.Path (Eigenschaft)

Ruft für die Skripterstellung den Pfad zum Ort ab, in dem der Ordner gespeichert ist.

## <a name="syntax"></a>Syntax


```VB
TaskFolder.Path As String
```



## <a name="property-value"></a>Eigenschaftswert

Der Pfad, in dem der Ordner gespeichert ist. Der Stammordner des Task wird mit einem schrägen Schrägstrich () \\ angegeben. Ein Beispiel für einen Taskordnerpfad unter dem Stammaufgabenordner ist \\ MyTaskFolder.
