# <a name="taskfolderpath-property"></a>TaskFolder.Path-Eigenschaft

Ruft für die Skripterstellung den Pfad zum Speicherort des Ordners ab.

## <a name="syntax"></a>Syntax


```VB
TaskFolder.Path As String
```



## <a name="property-value"></a>Eigenschaftswert

Der Pfad zum Speicherort des Ordners. Der Stammtaskordner wird mit einem umgekehrten Schrägstrich \\ () angegeben. Ein Beispiel für einen Taskordnerpfad im Stammtaskordner ist \\ MyTaskFolder.
