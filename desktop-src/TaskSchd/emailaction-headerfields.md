# <a name="emailactionheaderfields-property"></a>Emailaction. headerfields (Eigenschaft)

\[Dieses Objekt wird nicht mehr unterstützt. Verwenden Sie "IExecAction" mit dem PowerShell-Cmdlet " [**Send-Mail Message**](/powershell/module/microsoft.powershell.utility/send-mailmessage) " als Problem Umgehung.\]

Ruft bei der Skripterstellung die Header Informationen in der e-Mail ab, die Sie senden möchten, oder legt diese fest.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
EmailAction.HeaderFields As TaskNamedValueCollection
```



## <a name="property-value"></a>Eigenschaftswert

Die Header Informationen in der e-Mail, die Sie senden möchten.
