# <a name="emailactionheaderfields-property"></a>EmailAction.HeaderFields(Eigenschaft)

\[Dieses Objekt wird nicht mehr unterstützt. Verwenden Sie IExecAction mit dem [**PowerShell-Cmdlet Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) als Problemumgehung.\]

Für die Skripterstellung ruft die Headerinformationen in der E-Mail ab, die Sie senden möchten, oder legt sie fest.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
EmailAction.HeaderFields As TaskNamedValueCollection
```



## <a name="property-value"></a>Eigenschaftswert

Die Headerinformationen in der E-Mail, die Sie senden möchten.
