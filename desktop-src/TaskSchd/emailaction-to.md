# <a name="emailactionto-property"></a>EmailAction.To-Eigenschaft

\[Dieses Objekt wird nicht mehr unterstützt. Verwenden Sie IExecAction mit dem [**PowerShell-Cmdlet Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) als Problemumgehung.\]

Für die Skripterstellung ruft die E-Mail-Adresse oder die Adressen ab, an die Sie die E-Mail senden möchten, oder legt sie fest.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
EmailAction.To As String
```



## <a name="property-value"></a>Eigenschaftswert

Die E-Mail-Adresse oder die E-Mail-Adressen, an die bzw. die Sie die E-Mail senden möchten.
