# <a name="triggercollectioncount-property"></a>TriggerCollection. Count-Eigenschaft

Bei der Skripterstellung wird die Anzahl der Trigger in der Auflistung abgerufen.

## <a name="syntax"></a>Syntax


```VB
TriggerCollection.Count As long
```



## <a name="property-value"></a>Eigenschaftswert

Die Anzahl der Trigger in der Auflistung.
