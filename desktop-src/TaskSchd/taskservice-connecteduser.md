# <a name="taskserviceconnecteduser-property"></a>TaskService. connecteduser (Eigenschaft)

Ruft bei der Skripterstellung den Namen des Benutzers ab, der mit dem Taskplaner-Dienst verbunden ist.

## <a name="syntax"></a>Syntax


```VB
TaskService.ConnectedUser As String
```



## <a name="property-value"></a>Eigenschaftswert

Der Name des Benutzers, der mit dem Taskplaner-Dienst verbunden ist.
