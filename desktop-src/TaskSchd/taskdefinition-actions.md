# <a name="taskdefinitionactions-property"></a>Taskdefinition. Actions (Eigenschaft)

Ruft bei der Skripterstellung eine Auflistung von Aktionen ab, die vom Task ausgeführt werden, oder legt diese fest.

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.Actions As ActionCollection
```



## <a name="property-value"></a>Eigenschaftswert

Eine Auflistung von Aktionen, die vom Task ausgeführt werden.
