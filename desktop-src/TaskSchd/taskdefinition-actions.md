# <a name="taskdefinitionactions-property"></a>TaskDefinition.Actions (Eigenschaft)

Für die Skripterstellung ruft eine Auflistung von Aktionen ab, die von der Aufgabe ausgeführt werden, oder legt diese fest.

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.Actions As ActionCollection
```



## <a name="property-value"></a>Eigenschaftswert

Eine Auflistung von Aktionen, die von der Aufgabe ausgeführt werden.
