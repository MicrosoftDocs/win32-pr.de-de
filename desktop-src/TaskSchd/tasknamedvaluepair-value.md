# <a name="tasknamedvaluepairvalue-property"></a>Tasknamedvaluepair. Value-Eigenschaft

Ruft bei der Skripterstellung den Wert ab, der einem Namen in einem Name-Wert-Paar zugeordnet ist, oder legt diesen fest.

## <a name="syntax"></a>Syntax


```VB
TaskNamedValuePair.Value As String
```



## <a name="property-value"></a>Eigenschaftswert

Der Wert, der einem Namen in einem Name-Wert-Paar zugeordnet ist.
