> Für einen Computer mit Windows Server 2003, Windows XP oder Windows 2000 zum Erstellen, überwachen oder Steuern von Aufgaben auf einem Computer mit Windows Vista müssen bestimmte Vorgänge auf dem Computer mit Windows Vista ausgeführt werden.
