# <a name="tasknamedvaluecollectioncount-property"></a>Tasknamedvaluecollection. Count-Eigenschaft

Bei der Skripterstellung wird die Anzahl der Name-Wert-Paare in der Auflistung abgerufen.

## <a name="syntax"></a>Syntax


```VB
TaskNamedValueCollection.Count As long
```



## <a name="property-value"></a>Eigenschaftswert

Die Anzahl der Name-Wert-Paare in der Auflistung.
