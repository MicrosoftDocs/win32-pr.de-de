# <a name="tasksettingsxmltext-property"></a>TaskSettings.XmlText-Eigenschaft

Ruft bei der Skripterstellung eine XML-formatierte Definition der Task Einstellungen ab oder legt diese fest.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
TaskSettings.XmlText As String
```



## <a name="property-value"></a>Eigenschaftswert

Eine XML-formatierte Definition der Task Einstellungen.
