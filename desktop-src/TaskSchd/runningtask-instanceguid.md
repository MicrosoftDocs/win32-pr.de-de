# <a name="runningtaskinstanceguid-property"></a>Runningtask. instanceGUID-Eigenschaft

Ruft bei der Skripterstellung den GUID-Bezeichner für diese Instanz der Aufgabe ab.

## <a name="syntax"></a>Syntax


```VB
RunningTask.InstanceGuid As string
```



## <a name="property-value"></a>Eigenschaftswert

Der GUID-Bezeichner für diese Instanz der Aufgabe. Ein Bezeichner wird von der Taskplaner-Dienst bei jedem Ausführen der Aufgabe generiert.
