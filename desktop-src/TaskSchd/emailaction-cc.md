# <a name="emailactioncc-property"></a>EmailAction.CC-Eigenschaft

\[Dieses Objekt wird nicht mehr unterstützt. Verwenden Sie "IExecAction" mit dem PowerShell-Cmdlet " [**Send-Mail Message**](/powershell/module/microsoft.powershell.utility/send-mailmessage) " als Problem Umgehung.\]

Dient zum Abrufen oder Festlegen der e-Mail-Adresse oder der Adressen, die Sie in der e-Mail-Nachricht für CC festlegen möchten.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```VB
EmailAction.Cc As String
```



## <a name="property-value"></a>Eigenschaftswert

Die e-Mail-Adresse oder Adressen, die Sie in der e-Mail-Nachricht CC angeben möchten.
