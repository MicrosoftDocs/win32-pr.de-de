# <a name="taskdefinitiondata-property"></a>Taskdefinition. Data (Eigenschaft)

Ruft bei der Skripterstellung die Daten ab, die der Aufgabe zugeordnet sind, oder legt diese fest. Diese Daten werden vom Taskplaner-Dienst ignoriert, aber von Drittanbietern verwendet, die das Aufgaben Format erweitern möchten.

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.Data As String
```



## <a name="property-value"></a>Eigenschaftswert

Die Daten, die der Aufgabe zugeordnet sind.
