# <a name="runningtaskpath-property"></a>Runningtask. Path-Eigenschaft

Bei der Skripterstellung wird der Pfad zum Speicherort der Aufgabe abgerufen.

## <a name="syntax"></a>Syntax


```VB
RunningTask.Path As string
```



## <a name="property-value"></a>Eigenschaftswert

Der Pfad, in dem die Aufgabe gespeichert wird.
