# <a name="taskfoldercollection-object"></a>Taskfoldercollection-Objekt

Skript Objekt, das Informationen und Steuerelemente für eine Sammlung von Ordnern bereitstellt, die Tasks enthalten.

## <a name="members"></a>Member

Das **taskfoldercollection** -Objekt verfügt über diese Typen von Membern:

-   [Eigenschaften](#properties)

### <a name="properties"></a>Eigenschaften

Das **taskfoldercollection** -Objekt verfügt über diese Eigenschaften.



| Eigenschaft                                               | BESCHREIBUNG                                                |
|:-------------------------------------------------------|:-----------------------------------------------------------|
| [**Countdown**](taskfoldercollection-count.md)<br/> | Ruft die Anzahl der Ordner in der Auflistung ab.<br/>   |
| [**Element**](taskfoldercollection-item.md)<br/>   | Ruft den angegebenen Ordner aus der Auflistung ab. <br/> |
