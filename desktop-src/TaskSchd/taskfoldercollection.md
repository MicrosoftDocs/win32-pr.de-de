# <a name="taskfoldercollection-object"></a>TaskFolderCollection-Objekt

Skripterstellungsobjekt, das Informationen und Steuerung für eine Auflistung von Ordnern bereitstellt, die Aufgaben enthalten.

## <a name="members"></a>Member

Das **TaskFolderCollection-Objekt** verfügt über folgende Typen von Membern:

-   [Eigenschaften](#properties)

### <a name="properties"></a>Eigenschaften

Das **TaskFolderCollection-Objekt** verfügt über diese Eigenschaften.



| Eigenschaft                                               | Beschreibung                                                |
|:-------------------------------------------------------|:-----------------------------------------------------------|
| [**Anzahl**](taskfoldercollection-count.md)<br/> | Ruft die Anzahl der Ordner in der Auflistung ab.<br/>   |
| [**Element**](taskfoldercollection-item.md)<br/>   | Ruft den angegebenen Ordner aus der Auflistung ab. <br/> |
