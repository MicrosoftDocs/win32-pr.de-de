# <a name="ivmvirtualpcminimummemorypervm-property"></a>Ivmvirtualpc:: minimummemorypervm (Eigenschaft)

\[Windows Virtual PC ist nicht mehr für die Verwendung ab Windows 8 verfügbar. Verwenden Sie stattdessen den [Hyper-V-WMI-Anbieter (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Ruft die minimal zulässige Menge an physischem Arbeitsspeicher pro virtuellem Computer (in Megabyte) ab.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```C++
HRESULT get_MinimumMemoryPerVM(
  [out, retval] long *megabytesOfMemory
);
```



## <a name="property-value"></a>Eigenschaftswert

Die minimale zulässige Menge (in Megabyte) des physischen Speichers pro virtuellem Computer.
