# <a name="windows-virtual-pc"></a>Windows Virtual PC

\[Windows Der virtuelle PC ist ab Windows 8 nicht mehr für die Verwendung verfügbar. Verwenden Sie stattdessen den [Hyper-V-WMI-Anbieter (V2).](../hyperv_v2/windows-virtualization-portal.md)\]

## <a name="purpose"></a>Zweck

Diese Dokumentation enthält Informationen zur Windows VIRTUAL PC-COM-Schnittstelle.

## <a name="where-applicable"></a>Anwendungsbereich

Windows Virtual PC ist die neueste Virtualisierungstechnologie von Microsoft. Sie ermöglicht es Ihnen, viele Produktivitätsanwendungen in einer virtuellen Windows-Umgebung mit einem einzigen Klick direkt aus Windows 7 auszuführen. Sie können separate virtuelle Computer auf Ihrem Windows 7-Desktop erstellen. Jeder virtuelle Computer emuliert ein vollständiges Hardwaresystem – vom Prozessor bis zur Netzwerkkarte – in einer eigenständigen, isolierten Softwareumgebung, die den gleichzeitigen Betrieb von ansonsten inkompatiblen Systemen ermöglicht.

## <a name="developer-audience"></a>Entwicklergruppe

Die Windows VIRTUAL PC-COM-Schnittstellen sind für Entwickler vorgesehen, die Clientanwendungen erstellen, die die Bereitstellung und den Betrieb virtueller Computer automatisieren.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Windows Virtual PC erfordert eine der folgenden Windows 7 Editionen: Home Basic, Home Premium, Professional, Ultimate oder Enterprise Edition.

Um die zugehörigen Header- und Bibliotheksdateien abzurufen, installieren Sie das Windows SDK für Windows 7 aus dem [Microsoft Download Center.](https://www.microsoft.com/download/details.aspx?id=8279)
