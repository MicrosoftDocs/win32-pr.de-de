# <a name="ivmvirtualmachinehassse2-property"></a>Ivmvirtualmachine:: HasSSE2-Eigenschaft

\[Windows Virtual PC ist nicht mehr für die Verwendung ab Windows 8 verfügbar. Verwenden Sie stattdessen den [Hyper-V-WMI-Anbieter (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Bestimmt, ob der Prozessor den SSE2-Anweisungs Satz unterstützt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```C++
HRESULT get_HasSSE2(
  [out, retval] VARIANT_BOOL *sse2Enabled
);
```



## <a name="property-value"></a>Eigenschaftswert

**True** , wenn der Prozessor den SSE2-Anweisungs Satz unterstützt, andernfalls **false** .
