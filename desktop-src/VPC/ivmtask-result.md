# <a name="ivmtaskresult-property"></a>Ivmtask:: result (Eigenschaft)

\[Windows Virtual PC ist nicht mehr für die Verwendung ab Windows 8 verfügbar. Verwenden Sie stattdessen den [Hyper-V-WMI-Anbieter (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Ruft das Ergebnis der Aufgabe ab.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```C++
HRESULT get_Result(
  [out, retval] VMTaskResult *result
);
```



## <a name="property-value"></a>Eigenschaftswert

Das Ergebnis der Aufgabe. Eine Liste der Werte finden Sie unter [**vmtaskresult**](vmtaskresult.md).
