# <a name="ivmvirtualmachineserialports-property"></a>Ivmvirtualmachine:: serialports (Eigenschaft)

\[Windows Virtual PC ist nicht mehr für die Verwendung ab Windows 8 verfügbar. Verwenden Sie stattdessen den [Hyper-V-WMI-Anbieter (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Ruft eine Aufzähl Bare Auflistung von seriellen Anschlüssen ab.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```C++
HRESULT get_SerialPorts(
  [out, retval] IVMSerialPortCollection **serialPortCollection
);
```



## <a name="property-value"></a>Eigenschaftswert

Ein [**ivmserialportcollection**](ivmserialportcollection.md) -Objekt.
