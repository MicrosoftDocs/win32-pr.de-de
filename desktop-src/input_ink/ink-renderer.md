<span data-ttu-id="b7d22-104">Die Ink-Renderer-APIs ermöglichen das Rendern von frei Hand Strichen auf den festgelegten Direct2D-Gerätekontext einer universellen Windows-APP anstelle des standardmäßigen [**InkCanvas**](/uwp/api/Windows.UI.Xaml.Controls.InkCanvas) -Steuer Elements.</span><span class="sxs-lookup"><span data-stu-id="b7d22-104">The Ink renderer APIs enable the rendering of ink strokes onto the designated Direct2D device context of a Universal Windows app, instead of the default [**InkCanvas**](/uwp/api/Windows.UI.Xaml.Controls.InkCanvas) control.</span></span>
