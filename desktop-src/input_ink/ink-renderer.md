# <a name="ink-renderer"></a>Ink-Renderer

Die Ink-Renderer-APIs ermöglichen das Rendern von frei Hand Strichen auf den festgelegten Direct2D-Gerätekontext einer universellen Windows-APP anstelle des standardmäßigen [**InkCanvas**](/uwp/api/Windows.UI.Xaml.Controls.InkCanvas) -Steuer Elements.

Der Ink-Renderer ist für die Verwendung durch universelle Windows-apps konzipiert, die anpassen möchten, wie Handschlag Striche als "trocken" und von identischer Qualität für das [**InkCanvas**](/uwp/api/Windows.UI.Xaml.Controls.InkCanvas) -Steuerelement gerendert werden.
