# <a name="ink-renderer"></a>Ink-Renderer

Die Ink-Renderer-APIs ermöglichen das Rendern von Ink-Strichen im angegebenen Direct2D-Gerätekontext einer Universellen Windows-App anstelle des Standardsteuerelements [**InkCanvas.**](/uwp/api/Windows.UI.Xaml.Controls.InkCanvas)

Der Ink-Renderer ist für die Verwendung durch Entwickler von Universal Windows-Apps konzipiert, die daran interessiert sind, anzupassen, wie Striche mit InkK-Strichen "dry" gerendert werden und die gleiche Qualität wie das [**InkCanvas-Steuerelement**](/uwp/api/Windows.UI.Xaml.Controls.InkCanvas) aufweisen.
