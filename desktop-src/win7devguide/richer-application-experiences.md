# <a name="richer-application-experiences"></a>Umfangreichere Anwendungserfahrungen

Windows 7 ermöglicht Es Entwicklern, differenzierte und intuitive Anwendungen zu erstellen, die die Auffindbarkeit, Benutzerfreundlichkeit und Freundlichkeit erheblich verbessern. Neue Methoden der Desktopintegration stellen die Anwendungsfunktionalität direkt auf die Fingerspitzen des Benutzers, und Windows Explorer und *Bibliotheken* bieten einfachen Zugriff auf hochwertige Informationen. Das Windows Menüband-Steuerelement und Windows Animation Framework erleichtern das Erstellen interaktiver und ansprechender Benutzeroberflächen. Neue Touch-APIs ermöglichen natürliche Interaktionen durch Multitouching und Fingerschwenken, und Manipulations- und Trägheits-APIs ermöglichen beeindruckende visuelle Effekte.

Schnelle Fortschritte in der Hardware- und Softwaretechnologie führen auch zu einer benutzerorientierten Benutzerfreundlichkeit. Windows 7 bringt diese Fortschritte mit neuen und flexiblen APIs, die die Technologie vollständig nutzen, unter Entwicklerkontrolle, während es noch einfacher wird, überzeugende Anwendungen zu entwickeln.
