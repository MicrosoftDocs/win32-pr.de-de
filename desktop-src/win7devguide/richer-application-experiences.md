# <a name="richer-application-experiences"></a>Umfangreichere Anwendungsumgebungen

Mit Windows 7 können Entwickler besondere und intuitive Anwendungen erstellen, die die Auffindbarkeit, die Benutzerfreundlichkeit und den reinen Genuss erheblich verbessern. Mit den neuen Methoden der Desktop Integration können Sie die Anwendungs Funktionalität direkt auf dem Benutzer ablegen, und Windows-Explorer und *Bibliotheken* bieten einfachen Zugriff auf Informationen mit hohem Wert. Das Windows-Menüband-Steuerelement und Windows Animation Framework vereinfachen das Erstellen interaktiver und ansprechender Benutzeroberflächen. Neue touchapis ermöglichen natürliche Interaktionen durch Multitouch-und Finger schwenken, und Manipulations-und Trägheits-APIs ermöglichen beeindruckende visuelle Effekte.

Schnelle Fortschritte bei der Hardware-und Softwaretechnologie sorgen auch für eine höhere Benutzerfreundlichkeit. Windows 7 bietet diese Fortschritte unter Entwickler Steuerung mit neuen und flexiblen APIs, die die Technologie optimal nutzen und gleichzeitig die Entwicklung von überzeugenden Anwendungen erleichtern.
