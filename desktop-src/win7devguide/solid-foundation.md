# <a name="solid-foundation"></a>Solide Grundlage

Windows 7 bietet eine hochgradig produktive Entwicklerplattform und Tools, die grundlegende Grundlagen des Betriebssystems bereitstellen. Windows 7 baut auf der Windows Vista-Plattform auf und bietet Entwicklern die Möglichkeit, Anwendungen zu erstellen, die mit beiden Plattformen kompatibel sind, während Anwendungs Kompatibilitätsprobleme eingeschränkt werden. Windows 7 verbessert die Leistung und Energie Verwaltung erheblich, sodass Sie Anwendungen erstellen können, die das Mobile Verhalten optimieren. Folglich sind Ihre Anwendungen visuell ansprechender, einfacher zu erstellen und unterstützen eine breitere Palette internationaler Standards. Es wurden auch Verbesserungen an globalen Unterstützung, Barrierefreiheit und Anwendungs Bereitstellung vorgenommen.
