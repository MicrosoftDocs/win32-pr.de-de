# <a name="solid-foundation"></a>Solid Foundation

Windows 7 bietet eine äußerst produktive Entwicklerplattform und Tools, die grundlegende Betriebssystemgrundwerte bereitstellen. Windows 7 baut auf der Windows Vista-Plattform auf und bietet Entwicklern die Möglichkeit, Anwendungen zu erstellen, die mit beiden Plattformen kompatibel sind, während Anwendungskompatibilitätsprobleme eingeschränkt werden. Windows 7 verbessert die Leistung und Energieverwaltung erheblich, sodass Sie Anwendungen erstellen können, die die mobile Benutzeroberfläche optimieren. Daher sind Ihre Anwendungen visuell ansprechender, einfacher zu erstellen und unterstützen eine größere Bandbreite an internationalen Standards. Fortschritte wurden auch bei der globalen Unterstützung, der Barrierefreiheit und der Anwendungsbereitstellung erzielt.
