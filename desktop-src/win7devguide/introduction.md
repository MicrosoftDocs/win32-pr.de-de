# <a name="introduction-windows-7-developer-guide"></a>Einführung (entwicklerhandbuch für Windows 7)

Die Entwicklung von Software war noch nie so interessant. Die Welt war noch nie verbundener, interaktiver und mobiler. Benutzeroberflächen und Inhalte hatten noch nie eine höhere Genauigkeit. Fortschritte in den Bereichen Konnektivität, Mobilität, natürliche Schnittstellen, Grafiken und Medien ermöglichen Entwicklern neue Szenarien und Möglichkeiten. Gleichzeitig waren die Erwartungen hinsichtlich Leistung, Flexibilität und Interoperabilität noch nie höher. Eine erstklassige Entwicklerplattform muss Entwicklern und ihren Lösungen eine flexible und vollständige Grundlage bieten, die es ihnen ermöglicht, Anwendungen zu erstellen, die sie vom Wettbewerb abheben und gleichzeitig ihre Investitionen maximieren.

Das Betriebssystem Windows 7 ist die wichtige Plattform für Entwickler. Microsoft lauscht unseren Kunden, blickt auf die Szenarien von morgen und baut auf den Funktionen von Windows Vista auf und bietet Entwicklern eine Vielzahl von Auswahlmöglichkeiten und Funktionen auf der Windows Entwicklerplattform, während Entwickler in die Lage versetzt werden, kreativ lösungen bereitzustellen, die mobil, verbunden und hochgradig intuitiv sind. Am wichtigsten ist, dass die Grundlagen der Entwicklerplattform wie Sicherheit, Leistung und Kompatibilität in Windows 7 oberste Priorität haben.

Dieser Leitfaden fasst die wichtigsten Entwicklerfortschritte in jedem der folgenden drei Bereiche zusammen:

[Solid Foundation](solid-foundation.md)

Hervorragende Benutzerfreundlichkeit beginnt mit einer soliden Grundlage. Windows 7 bietet eine solide Entwicklungsplattform und innovative Tools, die Ihnen mehr Optionen, eine höhere Anwendungskompatibilität, eine bessere Leistung und eine anspruchsvolle Dokumentunterstützung bieten. Dies führt zu einer einfachen, zuverlässigeren Umgebung für Ihre Anwendungen und einer vertrauten, intuitiven Umgebung für Ihre Benutzer.

Wir haben die Benutzeroberfläche konsistent, vorhersagbar und einfach zu verwenden gehalten und gleichzeitig Multitouchunterstützung, Scrollen und andere intuitive Interaktionsfeatures hinzugefügt, die die neuesten PC- und Mobilen Gerätefunktionen nutzen. Mit Windows 7 können Entwickler die richtigen Benutzeroberflächen für ihre Anwendungen erstellen. (Weitere Informationen finden Sie [unter Windows Richtlinien für die Interaktion mit der Benutzeroberfläche.)](/windows/apps/desktop/)

[Umfangreichere Anwendungserfahrungen](richer-application-experiences.md)

mit Windows 7 können Entwickler heute Anwendungen erstellen, die auf den PCs von morgen ausgeführt werden. Dienstfähige Software und Geräte wie Mobiltelefone, tragbare Medienplayer und Digitalkameras erfordern konstante Konnektivität und erweiterte Anwendungen. Windows 7 bietet eine Plattform für beide, sodass Entwickler die Funktionen und Features der Hardware der nächsten Generation problemlos nutzen und gleichzeitig sicherstellen können, dass Benutzer immer aktiviert und immer verbunden sind. (Siehe [Windows Hardware Developer Central](https://www.microsoft.com/whdc/default.mspx).)

[Das Beste aus Windows und dem Web](the-best-of-windows-and-the-web.md)

Mit Windows 7 ist es einfach, verbunden zu sein und verbunden zu bleiben. Windows Netzwerk bietet Entwicklern Optionen für eine bessere Zwischenspeicherung und Freigabe von Daten, um die Netzwerkleistung zu verbessern. Die Netzwerkdiagnose in Windows 7 gibt Entwicklern relevante Informationen zur Überwachung von Netzwerkproblemen. Windows Internet Explorer 8 ist die wichtige Plattform zum Erstellen schneller, umfangreicher Weberfahrungen. Unabhängig davon, ob das Ziel darin besteht, Geräte zu verbinden, eine Verbindung mit einem Netzwerk herzustellen oder eine Verbindung mit dem *Internet* herzustellen, bietet Windows 7 eine erweiterte Plattform. (Siehe [Webentwicklung](https://msdn.microsoft.com/library/aa155073.aspx).)
