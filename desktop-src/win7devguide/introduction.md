# <a name="introduction-windows-7-developer-guide"></a>Einführung (Windows 7-Entwicklerhandbuch)

Die Entwicklung von Software war noch nie spannend. Die Welt war nie mehr verbunden, interaktiv und mobil. Die Benutzeroberfläche und der Inhalt hatten nie höhere Treue. Der Fortschritt bei Konnektivität, Mobilität, natürlichen Schnittstellen, Grafiken und Medien ermöglicht Entwicklern neue Szenarien und Möglichkeiten. Gleichzeitig waren die Erwartungen für Leistung, Flexibilität und Interoperabilität nie höher. Eine erstklassige Entwicklerplattform muss eine flexible und umfassende Grundlage für Entwickler und ihre Lösungen bieten. Sie ermöglicht Ihnen, Anwendungen zu erstellen, die Sie vom Wettbewerb trennen und gleichzeitig ihre Investitionen maximieren.

Das Betriebssystem Windows 7 ist die wichtigste Plattform für Entwickler. Wenn wir uns an unsere Kunden halten, die nach den Szenarien von morgen und auf den Funktionen von Windows Vista aufbauen, bietet Microsoft Entwicklern eine Vielzahl von Optionen und Funktionen innerhalb der Windows-Entwicklerplattform, während Entwickler gleichzeitig kreative Lösungen bereitstellen können, die mobile Geräte unterstützen, verbunden sind und eine hochgradig intuitive Benutzer Funktionalität bieten. Vor allem sind die Grundlagen der Entwicklerplattform, wie Sicherheit, Leistung und Kompatibilität, die wichtigsten Prioritäten in Windows 7.

In diesem Leitfaden werden die wichtigsten Entwicklungsfortschritte in den folgenden drei Bereichen zusammengefasst:

[Solide Grundlage](solid-foundation.md)

Sehr gute Benutzererfahrungen beginnen mit einer soliden Grundlage. Windows 7 bietet eine solide Entwicklungsplattform und innovative Tools, mit denen Sie mehr Optionen, eine höhere Anwendungs Kompatibilität, eine bessere Leistung und ausgereifte Dokumenten Unterstützung erhalten. Dies ergibt eine einfache, zuverlässigere Umgebung für Ihre Anwendungen und eine vertraute, intuitive Umgebung für Ihre Benutzer.

Wir haben die Benutzeroberfläche konsistent, vorhersagbar und einfach zu verwenden und gleichzeitig Multitouch-Unterstützung, Bildlauf und andere intuitive Interaktions Features hinzugefügt, die die neuesten Funktionen von PCs und mobilen Geräten nutzen. Mit Windows 7 haben Entwickler die Möglichkeit, die richtigen Benutzeroberflächen für Ihre Anwendungen zu entwickeln. (Weitere Informationen finden Sie unter [Windows User Interaktion Guidelines](/windows/apps/desktop/).)

[Umfangreichere Anwendungsumgebungen](richer-application-experiences.md)

Mit Windows 7 können Entwickler heute Anwendungen erstellen, die auf den PCs von morgen ausgeführt werden. Dienst fähige Software und Geräte, z. b. Mobiltelefone, tragbare Medien Player und digitale Kameras, erfordern Konstante Konnektivität und erweiterte Anwendungen. Windows 7 bietet eine Plattform für beides, wodurch Entwickler auf einfache Weise die Funktionen und Features der Next Generation-Hardware nutzen können, während gleichzeitig sichergestellt wird, dass die Benutzer immer eingeschaltet und immer verbunden sind. (Siehe [Windows Hardware Developer Central](https://www.microsoft.com/whdc/default.mspx).)

[Das beste von Windows und dem Web](the-best-of-windows-and-the-web.md)

Mit Windows 7 können Sie problemlos verbunden werden und bleiben verbunden. Windows-Netzwerke bieten Entwicklern Optionen zur besseren Zwischenspeicherung und Freigabe von Daten, um die Netzwerkleistung zu verbessern. Die Netzwerk Diagnose in Windows 7 bietet Entwicklern relevante Informationen zum Überwachen von Netzwerkproblemen. Windows Internet Explorer 8 ist die wichtigste Plattform zum entwickeln schneller, umfassender webollen. Ob das Ziel darin besteht, Geräte zu verbinden, eine Verbindung mit einem Netzwerk herzustellen oder eine Verbindung mit dem *Internet* herzustellen, bietet Windows 7 eine verbesserte Plattform. (Siehe [Webentwicklung](https://msdn.microsoft.com/library/aa155073.aspx).)
