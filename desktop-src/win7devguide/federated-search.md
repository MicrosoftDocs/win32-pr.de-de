# <a name="federated-search"></a>Verbundsuche

Windows 7 unterstützt die Suche nach Dokumenten, die über den eigenen PC des Benutzers hinausgehen. Entwickler und IT-Experten können in Windows 7 ihre Suchmaschinen, Dokumentrepos repositorys, Webanwendungen und proprietären Datenspeicher durchsuchen, ohne Clientcode schreiben und bereitstellen zu müssen. Dadurch können Endbenutzer ihr Unternehmensintranet oder das Web so einfach durchsuchen, wie sie ihre lokalen Dateien suchen können– und das alles über dieselbe vertraute Windows Benutzeroberfläche.

Windows 7-kompatible  OpenSearch-Unterstützung kann von Entwicklern und IT-Experten zu jeder vorhandenen durchsuchbaren Webanwendung hinzugefügt werden, indem *eine RSS-* oder *ATOM-Ausgabe* gemäß der Definition im öffentlichen OpenSearch [wird.](http://www.opensearch.org/) *OpenSearch* beschreibungsdateien können dann von Entwicklern und erweiterten Benutzern erstellt werden, um die Verbindungsparameter zu definieren, die es dem Desktopclient ermöglichen, einen Suchconnector für den Dienst zu erstellen.

SharePoint Der Suchserver kann diese kompatiblen Dienste auch *OpenSearch* abfragen. Dies ermöglicht auch den Zugriff auf Ihren Suchdienst *über SharePoint-Weboberfläche* des Suchcenters.

Windows 7 unterstützt die Suche nach Dokumenten, die über den eigenen PC des Benutzers hinausgehen, sowie eine Vorschau der Suchergebnisse.
