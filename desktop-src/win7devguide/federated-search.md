# <a name="federated-search"></a>Verbund Suche

Windows 7 unterstützt das Suchen nach Dokumenten, die über den benutzereigenen PC hinausgehen. Entwickler und IT-Experten können Ihre Suchmaschinen, dokumenttrepositorys, Webanwendungen und proprietären Datenspeicher für die Suche von Windows 7 aktivieren, ohne Client Code schreiben und bereitstellen zu müssen. Dies ermöglicht es Endbenutzern, Ihr Unternehmens Intranet oder das Web so einfach zu durchsuchen, wie Sie nach lokalen Dateien suchen können, und das alles innerhalb derselben vertrauten Windows-Schnittstelle.

Windows 7-kompatible *OpenSearch* -Unterstützung kann von Entwicklern und IT-Experten zu jeder vorhandenen durchsuchbaren Webanwendung hinzugefügt werden, indem Sie eine *RSS* -oder *Atom* -Ausgabe hinzufügen, wie Sie im öffentlichen [OpenSearch](http://www.opensearch.org/) *OpenSearch* -Beschreibungsdateien können dann von Entwicklern und fortgeschrittenen Benutzern erstellt werden, um die Verbindungsparameter zu definieren, die es dem Desktop Client ermöglichen, einen Suchconnector für den Dienst zu erstellen.

SharePoint Search Server kann auch diese kompatiblen *OpenSearch* -Dienste Abfragen. Dies ermöglicht auch den Zugriff auf Ihren Suchdienst über die Webschnittstelle des *SharePoint* -Suchcenters.

![SharePoint-Suche in Windows 7](images/windows7-12.jpg)

Windows 7 unterstützt das Suchen nach Dokumenten, die über den benutzereigenen PC hinausgehen, sowie eine Vorschau der Suchergebnisse.
