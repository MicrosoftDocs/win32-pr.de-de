# <a name="federated-search"></a>Verbundsuche

Windows 7 unterstützt die Suche nach Dokumenten, die über den eigenen PC des Benutzers hinausgehen. Entwickler und IT-Experten können ihre Suchmaschinen, Dokumentrepositorys, Webanwendungen und proprietären Datenspeicher aus Windows 7 durchsuchen, ohne Clientcode schreiben und bereitstellen zu müssen. Auf diese Weise können Endbenutzer ihr Unternehmensintranet oder das Web so einfach wie ihre lokalen Dateien durchsuchen– alles innerhalb der gleichen vertrauten Windows Schnittstelle.

Windows 7 kompatible *OpenSearch-Unterstützung* kann von Entwicklern und IT-Experten jeder vorhandenen durchsuchbaren Webanwendung hinzugefügt werden, indem *rss-* oder *ATOM-Ausgabe* hinzugefügt wird, wie im öffentlichen [OpenSearch](http://www.opensearch.org/) Standard definiert. *OpenSearch* Beschreibungsdateien können dann von Entwicklern und erweiterten Benutzern erstellt werden, um die Verbindungsparameter zu definieren, mit denen der Desktopclient einen Suchconnector für den Dienst erstellen kann.

SharePoint Der Suchserver kann diese kompatiblen *OpenSearch* Dienste auch abfragen. Dies ermöglicht auch den Zugriff auf Ihren Suchdienst über die *Weboberfläche SharePoint* Suchcenters.

Windows 7 unterstützt die Suche nach Dokumenten über den eigenen PC des Benutzers hinaus sowie eine Vorschau der Suchergebnisse.
