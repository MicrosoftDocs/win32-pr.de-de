# <a name="dialog-boxes-dialog-boxes"></a>Dialogfelder (Dialogfelder)

Ein Dialogfeld ist ein temporäres Fenster, das von einer Anwendung erstellt wird, um Benutzereingaben abzurufen. Eine Anwendung verwendet in der Regel Dialogfelder, um den Benutzer zur Eingabe zusätzlicher Informationen für Menü Elemente aufzufordern. Ein Dialogfeld enthält in der Regel mindestens ein Steuerelement (untergeordnete Fenster), mit dem der Benutzer Text eingibt, Optionen auswählt oder die Aktion weiterleitet.

Windows stellt auch vordefinierte Dialogfelder bereit, die allgemeine Menü Elemente wie " **Öffnen** " und " **Drucken**" unterstützen. Anwendungen, die diese Menü Elemente verwenden, sollten die allgemeinen Dialogfelder verwenden, um diese Benutzereingabe unabhängig vom Anwendungstyp einzugeben.
