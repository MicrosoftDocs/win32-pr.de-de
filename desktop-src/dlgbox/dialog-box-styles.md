# <a name="dialog-box-styles"></a>Dialogfeldstile

In der folgenden Tabelle sind die Dialogfeldstile aufgeführt, die Sie beim Erstellen eines Dialogfelds angeben können. Sie können diese Stile in Aufrufen der Funktionen [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) und [**CreateWindowEx,**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) im **Stilmember** der [**DLGTEMPLATE-**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate) und [**DLGTEMPLATEEX-Strukturen**](dlgtemplateex.md) und in der Anweisung einer Dialogfelddefinition in einer Ressourcendatei verwenden.



| Konstante                                                                                                                                                               | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="DS_3DLOOK"></span><span id="ds_3dlook"></span><dl> <dt>**DS \_ 3DLOOK**</dt> </dl>                      | Veraltet. Das System wendet die dreidimensionale Darstellung automatisch auf Dialogfelder an, die von Anwendungen erstellt wurden.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="DS_ABSALIGN"></span><span id="ds_absalign"></span><dl> <dt>**DS \_ ABSALIGN**</dt> </dl>                | Gibt an, dass die Koordinaten des Dialogfelds Bildschirmkoordinaten sind. Wenn dieser Stil nicht angegeben ist, sind die Koordinaten Clientkoordinaten.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="DS_CENTER"></span><span id="ds_center"></span><dl> <dt>**DS \_ CENTER**</dt> </dl>                      | Zentriere das Dialogfeld im Arbeitsbereich des Monitors, der das Besitzerfenster enthält. Wenn kein Besitzerfenster angegeben wird, wird das Dialogfeld im Arbeitsbereich eines vom System bestimmten Monitors zentriert. Der Arbeitsbereich ist der Bereich, der nicht von der Taskleiste oder appbars verdeckt wird.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="DS_CENTERMOUSE"></span><span id="ds_centermouse"></span><dl> <dt>**DS \_ CENTERMOUSE**</dt> </dl>       | Zentrt das Dialogfeld auf dem Mauszeiger.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DS_CONTEXTHELP"></span><span id="ds_contexthelp"></span><dl> <dt>**DS \_ CONTEXTHELP**</dt> </dl>       | Schließt ein Fragezeichen in die Titelleiste des Dialogfelds ein. Wenn der Benutzer auf das Fragezeichen klickt, wird der Cursor zu einem Fragezeichen geändert. Wenn der Benutzer dann im Dialogfeld auf ein Steuerelement klickt, empfängt das Steuerelement eine [**\_ WM-HILFEmeldung.**](../shell/wm-help.md) Das Steuerelement sollte die Meldung an die Dialogfeldprozedur übergeben, die die Funktion mithilfe des **HELP \_ WM \_ HELP-Befehls** aufrufen sollte. Die Hilfeanwendung zeigt ein Popupfenster an, das in der Regel Hilfe für das Steuerelement enthält.<br/> Beachten Sie, dass **DS \_ CONTEXTHELP** nur ein Platzhalter ist. Wenn das Dialogfeld erstellt wird, sucht das System nach **DS \_ CONTEXTHELP** und fügt **WS \_ EX \_ CONTEXTHELP** dem erweiterten Stil des Dialogfelds hinzu, wenn es vorhanden ist. **WS \_ EX \_ CONTEXTHELP** kann nicht mit den **Stilen WS \_ MAXIMIZEBOX** oder **WS \_ MINIMIZEBOX** verwendet werden.<br/>                                                                              |
| <span id="DS_CONTROL"></span><span id="ds_control"></span><dl> <dt>**\_DS-STEUERELEMENT**</dt> </dl>                   | Erstellt ein Dialogfeld, das als untergeordnetes Fenster eines anderen Dialogfelds funktioniert, ähnlich wie eine Seite in einem Eigenschaftenblatt. Dieser Stil ermöglicht es dem Benutzer, zwischen den Steuerfenstern eines untergeordneten Dialogfelds zu tabstoppen, seine Zugriffstasten zu verwenden usw.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DS_FIXEDSYS"></span><span id="ds_fixedsys"></span><dl> <dt>**DS \_ FIXEDSYS**</dt> </dl>                | Bewirkt, dass das Dialogfeld die **\_ SYSTEM FIXED \_ FONT** anstelle der systemstandardschriftart verwendet. **\_** Dies ist eine Monospaceschriftart, die mit der Schriftart System in 16-Bit-Versionen von Windows vor 3.0 kompatibel ist.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="DS_LOCALEDIT"></span><span id="ds_localedit"></span><dl> <dt>**DS \_ LOCALEDIT**</dt> </dl>             | Gilt nur für 16-Bit-Anwendungen. Dieser Stil weist Bearbeitungssteuerelemente im Dialogfeld an, Arbeitsspeicher aus dem Datensegment der Anwendung zuzuordnen. Andernfalls weisen Bearbeitungssteuerelemente Speicher aus einem globalen Speicherobjekt zu.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="DS_MODALFRAME"></span><span id="ds_modalframe"></span><dl> <dt>**DS \_ MODALFRAME**</dt> </dl>          | Erstellt ein Dialogfeld mit einem modalen Dialogfeldrahmen, der mit einer Titelleiste und einem Fenstermenü kombiniert werden kann, indem die **WS \_ CAPTION-** und **WS \_ SYSMENU-Stile** angegeben werden.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="DS_NOFAILCREATE"></span><span id="ds_nofailcreate"></span><dl> <dt>**DS \_ NOFAILCREATE**</dt> </dl>    | Erstellt das Dialogfeld auch dann, wenn Fehler auftreten, z. B. wenn ein untergeordnetes Fenster nicht erstellt werden kann oder wenn das System kein spezielles Datensegment für ein Bearbeitungssteuerelement erstellen kann.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="DS_NOIDLEMSG"></span><span id="ds_noidlemsg"></span><dl> <dt>**DS \_ NOIDLEMSG**</dt> </dl>             | Unterdrückt [**\_ WM-ENTERIDLE-Nachrichten,**](wm-enteridle.md) die das System andernfalls an den Besitzer des Dialogfelds senden würde, während das Dialogfeld angezeigt wird.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="DS_SETFONT"></span><span id="ds_setfont"></span><dl> <dt>**DS \_ SETFONT**</dt> </dl>                   | Gibt an, dass der Header der Dialogfeldvorlage (standard oder erweitert) zusätzliche Daten enthält, die die Schriftart angeben, die für Text im Clientbereich und Steuerelemente des Dialogfelds verwendet werden soll. Nach Möglichkeit wählt das System eine Schriftart gemäß den angegebenen Schriftartdaten aus. Das System übergibt ein Handle an die Schriftart an das Dialogfeld und an jedes Steuerelement, indem es die [**WM \_ SETFONT-Nachricht**](/windows/desktop/winmsg/wm-setfont) sendet. Beschreibungen des Formats dieser Schriftartdaten finden Sie unter [**DLGTEMPLATE**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate) und [**DLGTEMPLATEEX.**](dlgtemplateex.md)<br/> Wenn weder **DS \_ SETFONT** noch **DS \_ SHELLFONT** angegeben ist, enthält die Dialogfeldvorlage keine Schriftartdaten.<br/>                                                                                                                                                                                                                                 |
| <span id="DS_SETFOREGROUND"></span><span id="ds_setforeground"></span><dl> <dt>**DS \_ SETFOREGROUND**</dt> </dl> | Bewirkt, dass das System die [**SetForegroundWindow-Funktion**](/windows/desktop/api/winuser/nf-winuser-setforegroundwindow) verwendet, um das Dialogfeld in den Vordergrund zu bringen. Dieser Stil ist nützlich für modale Dialogfelder, die sofortige Aufmerksamkeit des Benutzers erfordern, unabhängig davon, ob das Besitzerfenster das Vordergrundfenster ist.<br/> Das System schränkt ein, welche Prozesse das Vordergrundfenster festlegen können. Weitere Informationen finden Sie unter [Vordergrund und Hintergrund Windows](/windows/desktop/winmsg/window-features).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="DS_SHELLFONT"></span><span id="ds_shellfont"></span><dl> <dt>**DS \_ SHELLFONT**</dt> </dl>             | Gibt an, dass das Dialogfeld die Systemschriftart verwenden soll. Der **Schriftartmember** der Vorlage für erweiterte Dialogfelder muss auf MS Shell Dlg festgelegt werden. Andernfalls hat dieser Stil keine Auswirkungen. Es wird auch empfohlen, die [DIALOGEX-Ressource](../menurc/dialogex-resource.md)anstelle der [DIALOG-Ressource](../menurc/dialog-resource.md)zu verwenden. Weitere Informationen finden Sie unter [Dialogfeldschriftarten](about-dialog-boxes.md).<br/> Das System wählt eine Schriftart mithilfe der Schriftartdaten aus, die in den **punktisierten**, **Gewichtungs-** und **italischen Membern** angegeben sind. Das System übergibt ein Handle an die Schriftart an das Dialogfeld und an jedes Steuerelement, indem es die [**WM \_ SETFONT-Nachricht**](/windows/desktop/winmsg/wm-setfont) sendet. Beschreibungen des Formats dieser Schriftartdaten finden Sie unter [**DLGTEMPLATEEX**](dlgtemplateex.md). <br/> Wenn weder **DS \_ SHELLFONT** noch **DS \_ SETFONT** angegeben ist, enthält die Vorlage für erweiterte Dialogfelder keine Schriftartdaten.<br/> |
| <span id="DS_SYSMODAL"></span><span id="ds_sysmodal"></span><dl> <dt>**DS \_ SYSMODAL**</dt> </dl>                | Dieser Stil ist veraltet und ist aus Kompatibilitätsgründen mit 16-Bit-Versionen von Windows enthalten. Wenn Sie diesen Stil angeben, erstellt das System das Dialogfeld mit dem **WS \_ EX \_ TOPMOST-Stil.** Dieser Stil verhindert nicht, dass der Benutzer auf andere Fenster auf dem Desktop zugreift.<br/> Kombinieren Sie diesen Stil nicht mit dem **DS \_ CONTROL-Stil.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
