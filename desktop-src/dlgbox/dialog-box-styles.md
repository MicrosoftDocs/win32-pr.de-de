# <a name="dialog-box-styles"></a>Dialog Feld Stile

In der folgenden Tabelle sind die Dialogfeld Stile aufgelistet, die Sie beim Erstellen eines Dialog Felds angeben können. Sie können diese Stile in Aufrufen der Funktionen " [**deatewindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) " und " [**deatewindowex**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) ", im **Stil** Element der [**DLGTEMPLATE**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate) -und [**dlgtemplateex**](dlgtemplateex.md) -Strukturen und in der-Anweisung einer Dialogfeld Definition in einer Ressourcen Datei verwenden.



| Konstante                                                                                                                                                               | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="DS_3DLOOK"></span><span id="ds_3dlook"></span><dl> <dt>**DS \_ 3dlook**</dt> </dl>                      | Veraltet. Das System wendet automatisch das dreidimensionale Aussehen auf Dialogfelder an, die von-Anwendungen erstellt wurden.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="DS_ABSALIGN"></span><span id="ds_absalign"></span><dl> <dt>**DS \_ absalign**</dt> </dl>                | Gibt an, dass die Koordinaten des Dialog Felds Bildschirm Koordinaten sind. Wenn dieser Stil nicht angegeben wird, sind die Koordinaten Client Koordinaten.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="DS_CENTER"></span><span id="ds_center"></span><dl> <dt>**DS \_ Center**</dt> </dl>                      | Zentriert das Dialogfeld im Arbeitsbereich des Monitors, der das Besitzer Fenster enthält. Wenn kein Besitzer Fenster angegeben ist, wird das Dialogfeld im Arbeitsbereich eines Monitors zentriert, der vom System festgelegt wird. Der Arbeitsbereich ist der Bereich, der von der Taskleiste oder von appbars nicht verdeckt wird.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="DS_CENTERMOUSE"></span><span id="ds_centermouse"></span><dl> <dt>**DS- \_ centermouse**</dt> </dl>       | Zentriert das Dialogfeld mit dem Mauszeiger.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DS_CONTEXTHELP"></span><span id="ds_contexthelp"></span><dl> <dt>**DS- \_ CONTEXTHELP**</dt> </dl>       | Schließt ein Fragezeichen in die Titelleiste des Dialog Felds ein. Wenn der Benutzer auf das Fragezeichen klickt, wird der Cursor zu einem Fragezeichen geändert. Wenn der Benutzer auf ein Steuerelement im Dialogfeld klickt, empfängt das Steuerelement eine [**WM- \_ Hilfe**](../shell/wm-help.md) Nachricht. Das-Steuerelement sollte die Nachricht an die Dialogfeld Prozedur übergeben, die die-Funktion mit dem Help- **\_ WM- \_ Hilfe** Befehl aufruft. Die Hilfe Anwendung zeigt ein Popup Fenster an, das in der Regel Hilfe für das Steuerelement enthält.<br/> Beachten Sie, dass **DS \_ CONTEXTHELP** nur ein Platzhalter ist. Wenn das Dialogfeld erstellt wird, überprüft das System nach **DS \_ CONTEXTHELP** und fügt ggf. den erweiterten Stil des Dialog Felds **WS \_ Ex \_ CONTEXTHELP** hinzu. **WS \_ Die " \_ CONTEXTHELP** " kann nicht mit den Stilen " **WS \_ MaximizeBox** " oder " **WS \_ MinimizeBox** " verwendet werden.<br/>                                                                              |
| <span id="DS_CONTROL"></span><span id="ds_control"></span><dl> <dt>**DS- \_ Steuerelement**</dt> </dl>                   | Erstellt ein Dialogfeld, das als untergeordnetes Fenster eines anderen Dialog Felds funktioniert, ähnlich wie eine Seite in einem Eigenschaften Blatt. Dieser Stil ermöglicht dem Benutzer die Tab-Taste zwischen den Steuer Fenstern eines untergeordneten Dialog Felds, der Verwendung seiner Zugriffstasten usw.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DS_FIXEDSYS"></span><span id="ds_fixedsys"></span><dl> <dt>**DS- \_ Fixedsys**</dt> </dl>                | Bewirkt, dass im Dialogfeld anstelle der standardmäßigen **System \_ Schriftart** die **\_ fixierte \_ Schriftart des Systems** verwendet wird. Dies ist eine fest breiten-Schriftart, die mit der System Schriftart in 16-Bit-Versionen von Windows vor 3,0 kompatibel ist.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="DS_LOCALEDIT"></span><span id="ds_localedit"></span><dl> <dt>**DS \_ localedit**</dt> </dl>             | Gilt nur für 16-Bit-Anwendungen. Dieser Stil weist Bearbeitungs Steuerelemente im Dialogfeld an, Arbeitsspeicher aus dem Daten Segment der Anwendung zuzuordnen. Andernfalls weisen Bearbeitungs Steuerelemente Speicher aus einem globalen Speicher Objekt zu.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="DS_MODALFRAME"></span><span id="ds_modalframe"></span><dl> <dt>**DS- \_ modalframe**</dt> </dl>          | Erstellt ein Dialogfeld mit einem modalen Dialogfeld Rahmen, das mit einer Titelleiste und einem Fenstermenü kombiniert werden kann, indem die **WS- \_ Beschriftung** und die **WS \_ sysmenu** -Stile angegeben werden.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="DS_NOFAILCREATE"></span><span id="ds_nofailcreate"></span><dl> <dt>**DS \_ nofailcreate**</dt> </dl>    | Erstellt das Dialogfeld, auch wenn Fehler auftreten, z. b., wenn ein untergeordnetes Fenster nicht erstellt werden kann oder wenn das System kein spezielles Daten Segment für ein Bearbeitungs Steuerelement erstellen kann.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="DS_NOIDLEMSG"></span><span id="ds_noidlemsg"></span><dl> <dt>**DS \_ noidlemsg**</dt> </dl>             | Unterdrückt [**die \_ WM**](wm-enteridle.md) -Nachrichten, die das System andernfalls an den Besitzer des Dialog Felds sendet, während das Dialogfeld angezeigt wird.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="DS_SETFONT"></span><span id="ds_setfont"></span><dl> <dt>**DS- \_ setFont**</dt> </dl>                   | Gibt an, dass der Header der Dialogfeld Vorlage (entweder Standard oder erweitert) zusätzliche Daten enthält, die die Schriftart angeben, die für den Text im Client Bereich und die Steuerelemente des Dialog Felds verwendet werden soll. Wenn möglich, wählt das System eine Schriftart gemäß den angegebenen Schriftart Daten aus. Das System übergibt ein Handle an die Schriftart an das Dialogfeld und an jedes Steuerelement, indem es die [**WM- \_ setFont**](/windows/desktop/winmsg/wm-setfont) -Nachricht sendet. Beschreibungen des Formats dieser Schriftart Daten finden Sie unter [**DLGTEMPLATE**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate) und [**dlgtemplateex**](dlgtemplateex.md).<br/> Wenn weder **DS \_ setFont** noch **DS \_ shellfont** angegeben ist, enthält die Dialogfeld Vorlage keine Schriftart Daten.<br/>                                                                                                                                                                                                                                 |
| <span id="DS_SETFOREGROUND"></span><span id="ds_setforeground"></span><dl> <dt>**DS \_ setvorder Grund**</dt> </dl> | Bewirkt, dass das System die [**SetForegroundWindow**](/windows/desktop/api/winuser/nf-winuser-setforegroundwindow) -Funktion verwendet, um das Dialogfeld in den Vordergrund zu bringen. Dieser Stil eignet sich für modale Dialogfelder, die sofortige Aufmerksamkeit durch den Benutzer erfordern, unabhängig davon, ob das Besitzer Fenster das Vordergrund Fenster ist.<br/> Das System schränkt ein, welche Prozesse das Vordergrund Fenster festlegen können. Weitere Informationen finden Sie unter [Vordergrund-und Hintergrund Fenster](/windows/desktop/winmsg/window-features).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="DS_SHELLFONT"></span><span id="ds_shellfont"></span><dl> <dt>**DS \_ shellfont**</dt> </dl>             | Gibt an, dass im Dialogfeld die System Schriftart verwendet werden soll. Der **Schriftart** -Member der erweiterten Dialogfeld Vorlage muss auf MS Shell Dlg festgelegt werden. Andernfalls hat dieser Stil keine Auswirkung. Es wird auch empfohlen, die [DIALOGEX-Ressource](../menurc/dialogex-resource.md)anstelle der [Dialog Feld Ressource](../menurc/dialog-resource.md)zu verwenden. Weitere Informationen finden Sie unter [Dialog Feld Schriftarten](about-dialog-boxes.md).<br/> Das System wählt eine Schriftart mithilfe der Schriftart Daten aus, die in den Elementen **pointsize**, **Weight** und **kursiv** angegeben sind. Das System übergibt ein Handle an die Schriftart an das Dialogfeld und an jedes Steuerelement, indem es die [**WM- \_ setFont**](/windows/desktop/winmsg/wm-setfont) -Nachricht sendet. Beschreibungen des Formats dieser Schriftart Daten finden Sie unter [**dlgtemplateex**](dlgtemplateex.md). <br/> Wenn weder **DS \_ shellfont** noch **DS \_ setFont** angegeben ist, enthält die erweiterte Dialogfeld Vorlage keine Schriftart Daten.<br/> |
| <span id="DS_SYSMODAL"></span><span id="ds_sysmodal"></span><dl> <dt>**DS \_ sysmodal**</dt> </dl>                | Dieser Stil ist veraltet und wird aus Gründen der Kompatibilität mit 16-Bit-Versionen von Windows berücksichtigt. Wenn Sie diesen Stil angeben, erstellt das System das Dialogfeld mit dem **\_ \_ obersten WS** -Stil. Dieser Stil verhindert nicht, dass der Benutzer auf dem Desktop auf andere Fenster zugreift.<br/> Kombinieren Sie diesen Stil nicht mit dem **DS- \_ Steuer** Element Stil.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
