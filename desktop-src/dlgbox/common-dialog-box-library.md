# <a name="common-dialog-box-library"></a>Allgemeine Dialog Feld Bibliothek

Die Bibliothek für das allgemeine Dialogfeld enthält eine Reihe von Dialog Feldern zum Ausführen allgemeiner Anwendungsaufgaben, z. b. das Öffnen von Dateien, das Auswählen von Farbwerten und das Drucken von Dokumenten. Die allgemeinen Dialogfelder ermöglichen es Ihnen, einen konsistenten Ansatz für die Benutzeroberfläche Ihrer Anwendung zu implementieren. Dadurch wird der Aufwand reduziert, den Benutzer beim Erlernen des Verhaltens der Benutzeroberfläche für Ihre Anwendung aufwenden.

In diesem Abschnitt werden die allgemeinen Dialogfelder beschrieben, in denen die Dialogfelder **Öffnen** und **Speichern** unter gespeichert sind. die Dialogfelder zum **Suchen** und **ersetzen** von Bearbeitung; die **Dialogfelder Druck,** **Druck Einrichtung**, **Druckeigenschaften Blatt** und Druck **Seiten Einrichtung** und die Dialogfelder **Farbe** und **Schriftart** .

> [!Note]  
> Ab Windows Vista wurden die Dialogfelder " **Öffnen** " und " **Speichern** unter" im allgemeinen [Element](/previous-versions/windows/desktop/legacy/bb776913(v=vs.85))ersetzt. Es wird empfohlen, dass Sie die allgemeine Element Dialogfeld-API anstelle dieser Dialogfelder aus der allgemeinen Dialogfeld Bibliothek verwenden.
