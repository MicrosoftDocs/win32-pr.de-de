# <a name="common-dialog-box-types"></a>Allgemeine Dialog Feld Typen

Die Bibliothek für das allgemeine Dialogfeld stellt eine Erstellungs Funktion und eine Struktur für die einzelnen Typen von allgemeinen Dialogfeldern bereit. Um ein gemeinsames Dialogfeld in seiner einfachsten Form zu verwenden, müssen Sie seine Erstellungs Funktion und einen Zeiger auf eine Struktur angeben, die Anfangswerte und Options-Flags enthält. Nach dem Initialisieren des Dialog Felds verwendet die Dialogfeld Prozedur die-Struktur, um Informationen zur Benutzereingabe zurückzugeben. Sie können auch ein häufig verwendeter Dialogfeld Anpassen, das den Anforderungen Ihrer Anwendung entspricht.

Die folgende Tabelle enthält eine kurze Beschreibung der verschiedenen Typen von allgemeinen Dialogfeldern und zeigt die Funktion und die Struktur an, die für die einzelnen Typen verwendet werden.



| Dialogfeld                | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|---------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Farbe**<br/>      | Zeigt verfügbare Farben an und ermöglicht es dem Benutzer, benutzerdefinierte Farben zu erstellen. Der Benutzer kann eine einfache oder benutzerdefinierte Farbe auswählen. Verwenden Sie die [**choosetcolor**](/previous-versions/windows/desktop/legacy/ms646912(v=vs.85)) -Funktion und die [**Chooin Color**](/windows/win32/api/commdlg/ns-commdlg-choosecolora-r1) -Struktur. Weitere Informationen finden Sie unter [Dialog Feld "Farbe](color-dialog-box.md)".<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **Suchen**<br/>       | Zeigt ein Dialogfeld an, in dem der Benutzer die zu suchende Zeichenfolge eingeben kann. Der Benutzer kann auch Suchoptionen angeben, z. b. die Suchrichtung und ob bei der Suche die Groß-/Kleinschreibung beachtet wird. Verwenden Sie die [**FindText**](/windows/desktop/api/Commdlg/nf-commdlg-findtexta) -Funktion und die [**FindReplace**](/windows/win32/api/commdlg/ns-commdlg-findreplacea) -Struktur. Weitere Informationen finden Sie untersuchen [und Ersetzen von Dialog Feldern](find-and-replace-dialog-boxes.md).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **Schriftart**<br/>       | Zeigt ein Dialogfeld an, in dem der Benutzer eine Schriftfamilie und zugehörige Schriftart, Punktgrößen und andere Schriftart Attribute wie Schriftart Farbe, Unterstreichung oder durchgestrichen auswählen kann. Verwenden Sie die [**Auswahl**](/windows/win32/api/commdlg/ns-commdlg-choosefonta) Funktion und die [**Auswahl Schriftart**](/windows/win32/api/commdlg/ns-commdlg-choosefonta) . Weitere Informationen finden Sie unter [Dialog Feld "Schriftart](font-dialog-box.md)".<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Öffnen**<br/>       | Zeigt ein Dialogfeld an, in dem der Benutzer den Namen eines zu öffnenden Datei-oder Shellname-Space-Objekts eingeben oder auswählen kann. Das Dialogfeld enthält Listen mit Laufwerken, Verzeichnissen und Shell-Namespace Erweiterungen, mit denen der Benutzer den shellnamensbereich durchsuchen kann. Sie enthält auch eine Liste der Dateinamen Erweiterungen, mit deren Hilfe der Benutzer die angezeigten Dateinamen filtern kann. Verwenden Sie die [**GetOpenFileName**](/windows/desktop/api/Commdlg/nf-commdlg-getopenfilenamea) -Funktion und die [**OpenFileName**](/windows/win32/api/commdlg/ns-commdlg-openfilenamea) -Struktur. Weitere Informationen finden Sie unter [Dialog Felder "Öffnen" und "Speichern](open-and-save-as-dialog-boxes.md)unter".<br/>                                                                                                                                                                                                                                                                                     |

Verwenden Sie die [**pagesetupdlg**](/previous-versions/windows/desktop/legacy/ms646937(v=vs.85)) -Funktion und die [**pagesetupdlg**](/windows/win32/api/commdlg/ns-commdlg-pagesetupdlga) -Struktur. Die Seite **Allgemein** des Eigenschaften Blatts ähnelt dem Dialogfeld **Drucken** . Auf dem Eigenschaften Blatt können zusätzliche anwendungsspezifische und Treiber spezifische Eigenschaften Seiten nach der Seite **Allgemein** vorhanden sein. Weitere Informationen finden Sie unter [Print Property Sheet](print-property-sheet.md).<br/> |

Der Benutzer kann Suchoptionen angeben, z. b. ob bei der Suche die Groß-/Kleinschreibung beachtet wird, und Ersetzungs Optionen, wie z. b Verwenden Sie die [**ReplaceText**](/windows/desktop/api/Commdlg/nf-commdlg-replacetexta) -Funktion und die [**FindReplace**](/windows/win32/api/commdlg/ns-commdlg-findreplacea) -Struktur. Weitere Informationen finden Sie untersuchen [und Ersetzen von Dialog Feldern](find-and-replace-dialog-boxes.md).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                        |

Das Dialogfeld enthält Listen mit Laufwerken, Verzeichnissen und Shell-Namespace Erweiterungen, mit denen der Benutzer den shellnamensbereich durchsuchen kann. Sie enthält auch eine Liste der Dateinamen Erweiterungen, mit deren Hilfe der Benutzer die angezeigten Dateinamen filtern kann. Verwenden Sie die [**GetSaveFileName**](/windows/desktop/api/Commdlg/nf-commdlg-getsavefilenamea) -Funktion und die [**OpenFileName**](/windows/win32/api/commdlg/ns-commdlg-openfilenamea) -Struktur. Weitere Informationen finden Sie unter [Dialog Felder "Öffnen" und "Speichern](open-and-save-as-dialog-boxes.md)unter".<br/>                                                                                                                                                                                                                                                                             |



 

Obwohl ein **Druck Installations** Dialogfeld verfügbar ist, wurde es durch das Dialogfeld **Seite einrichten** ersetzt. Anwendungen sollten das Dialogfeld **Seite einrichten** anstelle des Dialog Felds **Druck Setup** verwenden.

Alle allgemeinen Dialogfelder sind modal, ausgenommen die Dialogfelder **Suchen** und **ersetzen** . Modale Dialogfelder müssen vom Benutzer geschlossen werden, bevor die Funktion, die zum Erstellen des Dialog Felds verwendet wird, zurückgeben kann. Die **Dialogfelder Suchen und** **ersetzen** sind nicht modelliert. die Funktion gibt zurück, bevor das Dialogfeld geschlossen wird. Wenn Sie die Dialogfelder **Suchen** und **ersetzen** verwenden, müssen Sie auch die [**IsDialogMessage**](/windows/desktop/api/Winuser/nf-winuser-isdialogmessagea) -Funktion in der Hauptnachrichten Schleife Ihrer Anwendung verwenden, um sicherzustellen, dass in diesen Dialogfeldern Tastatureingaben ordnungsgemäß verarbeitet werden, z. b. die Tab-Taste und die ESC-Taste.
