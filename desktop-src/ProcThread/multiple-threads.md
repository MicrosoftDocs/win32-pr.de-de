# <a name="multiple-threads"></a>Mehrere Threads

Ein *Thread* ist die Entität innerhalb eines Prozesses, die für die Ausführung geplant werden kann. Alle Threads eines Prozesses verwenden den virtuellen Adressraum und die Systemressourcen gemeinsam. Jeder Prozess wird mit einem einzelnen Thread gestartet, kann jedoch zusätzliche Threads aus allen Threads erstellen.
