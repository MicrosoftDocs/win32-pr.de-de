# <a name="scheduling"></a>Scheduling

Der Systemplaner steuert das Multitasking, indem ermittelt wird, welche der konkurrierenden Threads den nächsten Prozessorzeit Slice erhalten. Der Planer bestimmt mithilfe von Planungs Prioritäten, welcher Thread als nächstes ausgeführt wird.
