# <a name="scheduling"></a>Scheduling

Der Systemplaner steuert multitasking, indem er bestimmt, welcher der konkurrierenden Threads den nächsten Prozessorzeitslice empfängt. Der Scheduler bestimmt anhand von Planungsprioritäten, welcher Thread als Nächstes ausgeführt wird.
