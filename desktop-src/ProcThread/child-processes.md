# <a name="child-processes"></a>Untergeordnete Prozesse

Jeder *Prozess* stellt die Ressourcen zur Verfügung, die zum Ausführen eines Programms erforderlich sind. Ein *untergeordneter Prozess* ist ein Prozess, der von einem anderen Prozess erstellt wird, der als *übergeordneter Prozess bezeichnet wird.*
