# <a name="child-processes"></a>Untergeordnete Prozesse

Jeder *Prozess* stellt die Ressourcen bereit, die zum Ausführen eines Programms benötigt werden. Ein untergeordneter *Prozess* ist ein Prozess, der von einem anderen Prozess erstellt wird, der als über *geordneter Prozess* bezeichnet wird.
