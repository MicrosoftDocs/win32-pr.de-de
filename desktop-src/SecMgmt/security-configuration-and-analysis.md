# <a name="security-configuration-and-analysis"></a>Sicherheitskonfiguration und -analyse

Die Sicherheitskonfiguration und-Analyse ist ein eigenständiges Snap-in-Tool, das Benutzer verwenden können, um eine oder mehrere gespeicherte Konfigurationen in eine private Sicherheitsdatenbank zu importieren. Beim Importieren von Konfigurationen wird eine Computer spezifische Sicherheitsdatenbank erstellt, in der eine zusammengesetzte Konfiguration gespeichert wird. Sie können diese zusammengesetzte Konfiguration auf den Computer anwenden und die aktuelle Systemkonfiguration anhand der in der-Datenbank gespeicherten gespeicherten zusammengesetzten Konfiguration analysieren.

Die Sicherheits [Vorlagen](security-templates.md) und die Sicherheitskonfiguration und-Analyse bilden die Sicherheitskonfigurations-Snap-Ins.
