# <a name="the-policy-object-type"></a>Der Richtlinien Objekttyp

Das [**Richtlinien**](policy-object.md) Objekt ist ein Container Objekt. Die folgenden Objekttypen befinden sich möglicherweise in diesem Container:

-   [**Konto**](account-object.md) Objekte
-   [**Treuhänder-Domänen**](trusteddomain-object.md) Objekt
-   Geheimes Objekt

Es ist nur eine einzige Instanz des [**Policy**](policy-object.md) -Objekts vorhanden. Sie kann weder erstellt noch zerstört werden.
