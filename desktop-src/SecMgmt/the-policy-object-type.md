# <a name="the-policy-object-type"></a>Der Richtlinienobjekttyp

Das [**Policy-Objekt**](policy-object.md) ist ein Containerobjekt. Die folgenden Objekttypen können sich in diesem Container befinden:

-   [**Kontoobjekte**](account-object.md)
-   [**TrustedDomain-Objekt**](trusteddomain-object.md)
-   Geheimnisobjekt

Es gibt nur eine einzelne Instanz des [**Policy-Objekts.**](policy-object.md) Sie kann weder erstellt noch zerstört werden.
