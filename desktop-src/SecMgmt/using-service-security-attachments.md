# <a name="using-service-security-attachments"></a>Verwenden von Dienst Sicherheitsanlagen

In diesem Abschnitt wird beschrieben, wie die Anlagen Komponenten erstellt werden, die zur Unterstützung einer Dienst spezifischen Konfiguration in den Snap-Ins für die Sicherheitskonfiguration erforderlich sind. Sie müssen zwei Komponenten implementieren und registrieren: ein Erweiterungs-Snap-in, um eine Benutzeroberfläche und eine Anlage-Engine zum Verarbeiten von Konfigurations-und Analyseanforderungen anzuzeigen.

Weitere Informationen finden Sie in den folgenden Themen:

-   [Erstellen einer Anlagen-Snap-in-Erweiterung](creating-an-attachment-snap-in-extension.md)
-   [Registrieren einer Anlage-Snap-in-Erweiterung](registering-an-attachment-snap-in-extension.md)
-   [Erstellen einer Anlage-Engine](creating-an-attachment-engine.md)
-   [Registrieren einer Anlage-Engine](registering-an-attachment-engine.md)
