# <a name="using-service-security-attachments"></a>Verwenden von Dienstsicherheitsanlagen

In diesem Abschnitt wird beschrieben, wie Sie die Anlagenkomponenten erstellen, die zur Unterstützung einer dienstspezifischen Konfiguration in den Sicherheitskonfigurations-Snap-Ins erforderlich sind. Sie müssen zwei Komponenten implementieren und registrieren: ein Erweiterungs-Snap-In zum Anzeigen einer Benutzeroberfläche und eine Anlagen-Engine zum Verarbeiten von Konfigurations- und Analyseanforderungen.

Weitere Informationen finden Sie in den folgenden Themen:

-   [Erstellen einer Erweiterung für das Anfüge-Snap-In](creating-an-attachment-snap-in-extension.md)
-   [Registrieren einer Anlagen-Snap-In-Erweiterung](registering-an-attachment-snap-in-extension.md)
-   [Erstellen einer Anlagen-Engine](creating-an-attachment-engine.md)
-   [Registrieren einer Anlagen-Engine](registering-an-attachment-engine.md)
