# <a name="security-management"></a>Sicherheitsverwaltung

## <a name="purpose"></a>Zweck

Die Sicherheits Verwaltungs Technologien können verwendet werden, um die Richtlinie für die [*Lokale Sicherheits Autorität*](/windows/desktop/SecGloss/l-gly) (LSA) und die Kenn [*Wortfilter*](/windows/desktop/SecGloss/p-gly) Richtlinie zu verwalten, die Fähigkeit von Programmen aus externen Quellen abzufragen und Dienst Anlagen, die die Funktionalität des Sicherheits Konfigurationstools erweitern.

Zu den Microsoft-Sicherheits Verwaltungs Technologien zählen die LSA-Richtlinien-API, die Kenn Wort Filter-API, die sicherere API und die Dienst Sicherheitsanlagen-API. Diese Technologien ermöglichen Softwareentwicklern das Erstellen von Anwendungen, die Systeme und Anwendungen verwalten.

## <a name="developer-audience"></a>Entwicklergruppe

Die LSA-Richtlinien-API, die Kenn Wort Filter-API, die sicherere API und die dienstsicherheitsanlagen-API sind für die Verwendung durch Entwickler von Anwendungen gedacht, die Administratoren die Verwaltung und Sicherung Ihrer Systeme ermöglichen. Entwickler sollten mit den Programmiersprachen C und C++ und mit Windows-basierter Programmierung vertraut sein.
