# <a name="security-templates"></a>Sicherheitsvorlagen

Die Sicherheitsvorlagen sind ein eigenständiges Snap-In-Tool, mit dem Benutzer computerunabhängige Sicherheitskonfigurationen definieren können. Diese Konfigurationen werden als textbasierte INF-Dateien gespeichert.

Die Sicherheitsvorlagen und die [Sicherheitskonfiguration und -analyse](security-configuration-and-analysis.md) bilden die Sicherheitskonfigurations-Snap-Ins.
