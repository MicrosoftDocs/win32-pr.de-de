# <a name="security-templates"></a>Sicherheits Vorlagen

Bei den Sicherheits Vorlagen handelt es sich um ein eigenständiges Snap-in-Tool, das Benutzer verwenden können, um Computer unabhängige Sicherheits Konfigurationen zu definieren. Diese Konfigurationen werden als textbasierte INF-Dateien gespeichert.

Die Sicherheits Vorlagen und die [Sicherheitskonfiguration und-Analyse](security-configuration-and-analysis.md) bilden die Sicherheitskonfigurations-Snap-Ins.
