# <a name="safer"></a>Sicherheit

Funktionen in der sichereren API stellen eine beliebige Anwendung bereit, die Programme aus externen Quellen startet, um die Sicherheitsrichtlinie für die Genehmigung abzufragen, bevor eine ausführbare Datei gestartet wird. Diese Funktionen können vor dem Laden und Ausführen einer ausführbaren Datei oder eines aktiven Inhalts aufgerufen werden. Anwendungs Typen, bei denen die sicherere API nützlich ist, sind Anwendungen, die Anlagen (z. b. e-Mail-Clients und Instant Messenger, die Dateien übertragen können) und Skript Interpreter verarbeiten. Die Unterstützung für Funktionen in der sichereren API beginnt mit Windows XP. Weitere Informationen finden Sie unter [sicherere Funktionen](management-functions.md).
