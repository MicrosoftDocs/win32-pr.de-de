# <a name="modifying-configuration-information-in-the-database"></a>Ändern von Konfigurationsinformationen in der Datenbank

Die Erweiterung für das Anfüge-Snap-In muss es Benutzern ermöglichen, Konfigurationsinformationen über ihren Analyseknoten zu ändern. Dadurch können Benutzer Änderungen vornehmen und die Konfiguration dann im Laufe der Zeit erneut anwenden. Die Logik hierfür sollte mit dem Ändern von Informationen in den Konfigurationsdateien identisch sein. Die vorgenommenen Änderungen werden auf die gespeicherte Konfiguration in der Sicherheitsdatenbank wirksam.
