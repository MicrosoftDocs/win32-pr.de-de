# <a name="policy-object"></a><span data-ttu-id="d235b-103">Policy-Objekt</span><span class="sxs-lookup"><span data-stu-id="d235b-103">Policy Object</span></span>

<span data-ttu-id="d235b-104">Das **Policy** -Objekt wird verwendet, um den Zugriff auf die [*Lokale Sicherheits Autorität*](/windows/desktop/SecGloss/l-gly) (LSA) zu steuern, und enthält Informationen, die für das gesamte System gelten oder Standardwerte für das System festlegen.</span><span class="sxs-lookup"><span data-stu-id="d235b-104">The **Policy** object is used to control access to the [*Local Security Authority*](/windows/desktop/SecGloss/l-gly) (LSA) database and contains information that applies to the entire system or establishes defaults for the system.</span></span> <span data-ttu-id="d235b-105">Jedes System verfügt nur über ein **Richtlinien** Objekt.</span><span class="sxs-lookup"><span data-stu-id="d235b-105">Each system has only one **Policy** object.</span></span> <span data-ttu-id="d235b-106">Dieses **Richtlinien** Objekt wird vom LSA erstellt, wenn das System gestartet wird, und Anwendungen können es nicht erstellen oder zerstören.</span><span class="sxs-lookup"><span data-stu-id="d235b-106">This **Policy** object is created by the LSA when the system starts up, and applications cannot create or destroy it.</span></span>

<span data-ttu-id="d235b-107">Zu den in einem **Richtlinien** Objekt gespeicherten Informationen gehören:</span><span class="sxs-lookup"><span data-stu-id="d235b-107">The information stored in a **Policy** object includes:</span></span>

-   <span data-ttu-id="d235b-108">Standardmäßiges System Arbeitsspeicher-Kontingent.</span><span class="sxs-lookup"><span data-stu-id="d235b-108">System default memory quota.</span></span> <span data-ttu-id="d235b-109">Sofern nicht anders angegeben, wird jedem Benutzer, der sich am System anmeldet, dieses Speicher Kontingent zugewiesen.</span><span class="sxs-lookup"><span data-stu-id="d235b-109">Unless otherwise specified, each user logging on to the system will be assigned this memory quota.</span></span> <span data-ttu-id="d235b-110">Besondere Arbeitsspeicher Kontingente können Einzelpersonen oder Gruppenmitgliedern oder lokalen Gruppen über ein [**Konto**](account-object.md) Objekt zugewiesen werden.</span><span class="sxs-lookup"><span data-stu-id="d235b-110">Special memory quotas can be assigned to individuals or members of groups or local groups through an [**Account**](account-object.md) object.</span></span>
-   <span data-ttu-id="d235b-111">System weite Sicherheits Überprüfungsanforderungen.</span><span class="sxs-lookup"><span data-stu-id="d235b-111">System-wide security auditing requirements.</span></span>
-   <span data-ttu-id="d235b-112">Der Name und die SID der Konto Domäne dieses Systems.</span><span class="sxs-lookup"><span data-stu-id="d235b-112">The name and SID of the account domain of this system.</span></span>
-   <span data-ttu-id="d235b-113">Informationen zur primären Domäne dieses Systems.</span><span class="sxs-lookup"><span data-stu-id="d235b-113">Information about the primary domain of this system.</span></span> <span data-ttu-id="d235b-114">Diese Informationen umfassen den Namen und die SID der primären Domäne, den Namen des Kontos innerhalb der primären Domäne, das für Authentifizierungsanforderungen verwendet werden soll, die Namen-und sid-Übersetzungen und das Abrufen der Namen von Domänen Controllern innerhalb der Domäne.</span><span class="sxs-lookup"><span data-stu-id="d235b-114">This information includes the name and SID of the primary domain, the name of the account within the primary domain that is to be used for authentication requests, name and SID translations, and obtaining the names of domain controllers within the domain.</span></span> <span data-ttu-id="d235b-115">Diese Namen sind möglicherweise veraltet und sollten nur als Hinweis verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="d235b-115">These names may be out of date and should be taken only as a hint.</span></span> <span data-ttu-id="d235b-116">Die Reihenfolge dieser Liste wird als signifikant angesehen und wird beibehalten.</span><span class="sxs-lookup"><span data-stu-id="d235b-116">The order of this list is assumed to be significant and will be maintained.</span></span> <span data-ttu-id="d235b-117">Dies ermöglicht beispielsweise, dass der Vorname in der Liste den letzten bekannten primären Domänen Controller darstellt.</span><span class="sxs-lookup"><span data-stu-id="d235b-117">This allows, for example, the first name in the list to represent the last-known primary domain controller.</span></span>
-   <span data-ttu-id="d235b-118">Informationen dazu, ob die LSA die Master Kopie der Richtlinien Informationen oder eines Replikats enthält.</span><span class="sxs-lookup"><span data-stu-id="d235b-118">Information about whether the LSA holds the master copy of the policy information or a replica.</span></span> <span data-ttu-id="d235b-119">Es wird nur ein Teil der Richtlinien Informationen repliziert. der Rest wird pro System festgelegt.</span><span class="sxs-lookup"><span data-stu-id="d235b-119">Only part of the policy information is replicated; the remainder is established on a per-system basis.</span></span>

<span data-ttu-id="d235b-120">Die Felder " **accountdomain** " und " **primarydomain** " des Objekts " **Policy** " werden je nach Art der System-und Vertrauens Stellungen für verschiedene Zwecke verwendet:</span><span class="sxs-lookup"><span data-stu-id="d235b-120">The **AccountDomain** and **PrimaryDomain** fields of the **Policy** object are used for different purposes depending upon the type of system and trust relationships:</span></span>

-   <span data-ttu-id="d235b-121">Auf einem System, das nicht über eine primäre Domäne verfügt, enthält das Feld **accountdomain** den Namen und die SID der lokalen Konto Domäne des Systems, die mit dem Computernamen identisch ist.</span><span class="sxs-lookup"><span data-stu-id="d235b-121">On a system that does not have a primary domain, the **AccountDomain** field contains the name and SID of the system's local account domain, which is the same as the computer name.</span></span> <span data-ttu-id="d235b-122">Das Feld **primarydomain** enthält den Namen der Arbeitsgruppe, in der dieser Computer Mitglied ist.</span><span class="sxs-lookup"><span data-stu-id="d235b-122">The **PrimaryDomain** field contains the name of the workgroup this machine is a member of.</span></span> <span data-ttu-id="d235b-123">Die Objekte der [**Treuhänder Domäne**](trusteddomain-object.md) werden mit einer Ausnahme ignoriert – es darf kein " **Treuhänder** "-Objekt mit dem gleichen Namen wie die Arbeitsgruppe vorhanden sein, da es so angezeigt wird, als ob es sich um die primäre Domäne des Computers handelt.</span><span class="sxs-lookup"><span data-stu-id="d235b-123">[**TrustedDomain**](trusteddomain-object.md) objects are ignored with one exception—there cannot be a **TrustedDomain** object with the same name as the workgroup because it will appear as though it is the machine's primary domain.</span></span>
-   <span data-ttu-id="d235b-124">Auf einem System mit einer primären Domäne identifiziert das Feld **accountdomain** den Namen und die SID der lokalen Konto Domäne wie zuvor.</span><span class="sxs-lookup"><span data-stu-id="d235b-124">On a system that has a primary domain, the **AccountDomain** field identifies the name and SID of the local account domain, as before.</span></span> <span data-ttu-id="d235b-125">Das Feld **primarydomain** enthält jedoch den Namen und die SID der primären Domäne für das System.</span><span class="sxs-lookup"><span data-stu-id="d235b-125">However, the **PrimaryDomain** field contains the name and SID of the primary domain for the system.</span></span> <span data-ttu-id="d235b-126">Außerdem sollte ein " [**treuddomain**](trusteddomain-object.md) "-Objekt mit dem Namen und der SID im Feld " **primarydomain** " angegeben werden.</span><span class="sxs-lookup"><span data-stu-id="d235b-126">Additionally, there should be a [**TrustedDomain**](trusteddomain-object.md) object with the name and SID identified in the **PrimaryDomain** field.</span></span> <span data-ttu-id="d235b-127">Dieses Objekt " **Treuhänder Domäne** " enthält die Konto-und Server Informationen, die erforderlich sind, um einen sicheren Kanal zu einem Domänen Controller in der primären Domäne einzurichten.</span><span class="sxs-lookup"><span data-stu-id="d235b-127">This **TrustedDomain** object contains the account and server information necessary to establish a secure channel to a domain controller in the primary domain.</span></span> <span data-ttu-id="d235b-128">Alle anderen Objekte der **Treuhänder Domäne** werden ignoriert.</span><span class="sxs-lookup"><span data-stu-id="d235b-128">Any other **TrustedDomain** objects are ignored.</span></span>
-   <span data-ttu-id="d235b-129">Auf Domänen Controllern identifiziert das Feld **accountdomain** die lokale Konto Domäne für das System. der Kontoname ist jedoch vom Benutzer zugewiesen, anstelle eines bekannten namens.</span><span class="sxs-lookup"><span data-stu-id="d235b-129">On domain controllers, the **AccountDomain** field identifies the local account domain for the system; however, the account name is user assigned rather than being a well-known name.</span></span> <span data-ttu-id="d235b-130">Da die primäre Domäne mit der Konto Domäne identisch ist, muss das Feld **primarydomain** denselben Wert wie das Feld **accountdomain** enthalten.</span><span class="sxs-lookup"><span data-stu-id="d235b-130">Since the primary domain is the same as the account domain, the **PrimaryDomain** field must contain the same value as the **AccountDomain** field.</span></span> <span data-ttu-id="d235b-131">Außerdem wird erwartet, dass alle [**trustddomain**](trusteddomain-object.md) -Objekte gültig sind und Vertrauens Stellungen mit anderen Domänen darstellen.</span><span class="sxs-lookup"><span data-stu-id="d235b-131">Furthermore, all [**TrustedDomain**](trusteddomain-object.md) objects are expected to be valid and represent trust relationships with other domains.</span></span> <span data-ttu-id="d235b-132">Wenn das System keine anderen Domänen als vertrauenswürdig einstuft, sollten keine vertrauenswürdigen **Domänen** Objekte vorhanden sein.</span><span class="sxs-lookup"><span data-stu-id="d235b-132">If the system does not trust any other domains, there should not be any **TrustedDomain** objects.</span></span>
