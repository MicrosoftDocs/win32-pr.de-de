# <a name="account-object"></a>Konto Objekt

Das **Konto** Objekt wird verwendet, um einzelnen Benutzern oder Mitgliedern von lokalen Gruppen oder Gruppen [*Berechtigungen*](/windows/desktop/SecGloss/p-gly), den System Zugriff und spezielle Kontingente zuzuweisen.

Zu den in einem **Konto** Objekt gespeicherten Informationen gehören:

-   Die sid für ein Benutzerkonto, ein Gruppenkonto oder ein lokales Gruppenkonto.
-   Eine Liste von Berechtigungen, die dem Benutzer oder den Mitgliedern einer Gruppe zugewiesen werden sollen. Wenn einem Benutzer von mehreren **Konto** Objekten Berechtigungen zugewiesen werden, enthält die Liste alle zugewiesenen Berechtigungen.
