# <a name="local-security-policy"></a>Lokale Sicherheitsrichtlinie

Die lokale Sicherheitsrichtlinie eines Systems ist ein Satz von Informationen zur Sicherheit eines lokalen Computers.

Die Informationen zu den lokalen Sicherheitsrichtlinien umfassen Folgendes:

-   Die vertrauenswürdigen Domänen zum Authentifizieren von Anmelde versuchen.
-   Welche Benutzerkonten auf das System zugreifen können und wie. Beispielsweise interaktiv, über ein Netzwerk oder als Dienst.
-   Die den Konten zugewiesenen Rechte und [*Berechtigungen*](/windows/desktop/SecGloss/p-gly) .
-   Die Sicherheits Überwachungsrichtlinie.

Die lokale [*Sicherheits Autorität (Local Security Authority*](/windows/desktop/SecGloss/l-gly) , LSA) speichert die Informationen der lokalen Richtlinie in einem Satz von [LSA-Richtlinien Objekten](lsa-policy-objects.md).
