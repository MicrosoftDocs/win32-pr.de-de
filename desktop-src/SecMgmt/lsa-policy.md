# <a name="lsa-policy"></a>LSA-Richtlinie

Bei der [*lokalen Sicherheits Autorität (Local Security Authority*](/windows/desktop/SecGloss/l-gly) , LSA) handelt es sich um ein geschütztes Subsystem von Windows, das Informationen zu allen Aspekten der lokalen Sicherheit auf einem System verwaltet, die als [lokale Sicherheitsrichtlinie](local-security-policy.md) des Systems bezeichnet werden. Zusätzlich zu den Informationen zu den Informationen zu den Informationen bietet die LSA Dienste für die Übersetzung zwischen Namen und [*Sicherheits*](/windows/desktop/SecGloss/s-gly) -IDs (SIDs).
