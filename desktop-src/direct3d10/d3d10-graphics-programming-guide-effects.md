<span data-ttu-id="6b6aa-104">Ein DirectX-Effekt ist eine Auflistung von Pipeline Zuständen, die durch in [HLSL](../direct3dhlsl/dx-graphics-hlsl-reference.md) geschriebene Ausdrücke und eine bestimmte, für das Effect-Framework spezifische Syntax festgelegt ist.</span><span class="sxs-lookup"><span data-stu-id="6b6aa-104">A DirectX effect is a collection of pipeline state, set by expressions written in [HLSL](../direct3dhlsl/dx-graphics-hlsl-reference.md) and some syntax that is specific to the effect framework.</span></span> <span data-ttu-id="6b6aa-105">Verwenden Sie nach dem Kompilieren eines Effekts die Effekt-Framework-APIs zum Rendering.</span><span class="sxs-lookup"><span data-stu-id="6b6aa-105">After compiling an effect, use the effect framework APIs to render.</span></span> <span data-ttu-id="6b6aa-106">Die Auswirkung der Funktionalität kann von etwas einfacher sein wie ein Vertexshader, der Geometrie transformiert, und ein Pixelshader, der eine voll Tonfarbe ausgibt, an eine renderingtechnik, die mehrere Durchgänge erfordert, jede Phase der Grafik Pipeline verwendet und den shaderzustand und den Pipeline Zustand, der den programmierbaren Shadern nicht zugeordnet ist, ändert.</span><span class="sxs-lookup"><span data-stu-id="6b6aa-106">Effect functionality can range from something as simple as a vertex shader that transforms geometry and a pixel shader that outputs a solid color, to a rendering technique that requires multiple passes, uses every stage of the graphics pipeline, and manipulates shader state as well as the pipeline state not associated with the programmable shaders.</span></span>
