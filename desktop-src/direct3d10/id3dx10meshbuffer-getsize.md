# <a name="id3dx10meshbuffergetsize-method"></a>ID3DX10MeshBuffer:: GetSize-Methode

Gibt die Größe des Gitter Puffers in Bytes an.

## <a name="syntax"></a>Syntax


```C++
SIZE_T GetSize();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **Größe \_ T**](../winprog/windows-data-types.md)**

Die Größe des Gitter Puffers in Bytes.
