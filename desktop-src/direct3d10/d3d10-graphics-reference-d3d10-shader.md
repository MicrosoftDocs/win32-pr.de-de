# <a name="shader-reference-direct3d-10-graphics"></a>Shaderverweis (Direct3D 10-Grafiken)

Die Direct3D-API definiert mehrere API-Elemente, die Sie beim Erstellen und Verwalten von programmierbaren Shadern unterstützen.

-   [Shader-Schnittstellen](d3d10-graphics-reference-d3d10-shader-interfaces.md)
-   [Shaderfunktionen](d3d10-graphics-reference-d3d10-shader-functions.md)
-   [Shader-Strukturen](d3d10-graphics-reference-d3d10-shader-structures.md)
-   [Shader-Enumerationen](d3d10-graphics-reference-d3d10-shader-enums.md)
-   [Shaderkonstanten](d3d10-graphics-reference-d3d10-shader-constants.md)

Shader sind ausführbare Programme, die die [Pipeline Stufen](d3d10-graphics-programming-guide-pipeline-stages.md)Vertex-Shader, Geometry-Shader und Pixel-Shader steuern. Shaders werden ausschließlich mit [HLSL](../direct3dhlsl/dx-graphics-hlsl-reference.md)programmiert.
