# <a name="resource-enumerations-direct3d-10-graphics"></a>Ressourcen Enumerationen (Direct3D 10-Grafiken)

Die folgenden Enumerationen werden verwendet, um Informationen darüber anzugeben, wie Ressourcen während des Renderings erstellt und aufgerufen werden.



| Enumeration                                                     | Beschreibung                                                                                                               |
|-----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**D3d10 \_ Bind- \_ Flag**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_bind_flag)                    | Gibt an, wie eine Ressource von der Pipeline verwendet wird, und bestimmt, an welche Pipeline Stufen eine Ressource gebunden werden kann. |
| [**D3d10 \_ CPU \_ - \_ Zugriffsflag**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_cpu_access_flag)       | Gibt die Typen des für eine Ressource zulässigen CPU-Zugriffs an.                                                                 |
| [**D3d10 \_ DSV- \_ Dimension**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_dsv_dimension)            | Gibt an, wie auf eine Ressource zugegriffen wird, die in einer Ansicht der tiefen Schablone verwendet wird.                                                  |
| [**D3d10- \_ Karte**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_map)                                 | Identifiziert eine Ressource, die für das Lesen und Schreiben durch die CPU zugeordnet werden soll.                                                    |
| [**D3d10 \_ map- \_ Flag**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_map_flag)                      | Gibt an, wie die CPU auf Karten Methoden reagieren soll, wenn die GPU noch nicht mit der Ressource fertig ist.               |
| [**D3d10- \_ Ressourcen \_ Dimension**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_resource_dimension)  | Identifiziert den Typ der Ressource, die verwendet wird.                                                                           |
| [**D3d10 \_ \_ ressourcenmisc- \_ Flag**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_resource_misc_flag) | Identifiziert weniger häufige Optionen zum Erstellen von Ressourcen.                                                                         |
| [**D3d10 \_ RTV- \_ Dimension**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_rtv_dimension)            | Gibt an, wie auf eine in einer renderzielansicht verwendete Ressource zugegriffen werden kann.                                                          |
| [**D3d10- \_ Verwendung**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_usage)                             | Gibt an, wie eine Ressource von der CPU und/oder der GPU gelesen und geschrieben werden soll.                                   |
