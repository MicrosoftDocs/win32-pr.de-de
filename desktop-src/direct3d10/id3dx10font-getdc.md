# <a name="id3dx10fontgetdc-method"></a>ID3DX10Font:: GetDC-Methode

Gibt ein Handle für einen Anzeigegeräte Kontext (DC) zurück, für den die Schriftart festgelegt ist.

## <a name="syntax"></a>Syntax


```C++
HDC GetDC();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **hdc**](../winprog/windows-data-types.md)**

Handle für einen Anzeige-DC.
