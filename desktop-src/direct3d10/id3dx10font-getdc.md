# <a name="id3dx10fontgetdc-method"></a>ID3DX10Font::GetDC-Methode

Gibt ein Handle an einen Anzeigegerätekontext (DC) zurück, auf dem die Schriftart festgelegt ist.

## <a name="syntax"></a>Syntax


```C++
HDC GetDC();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **HDC**](../winprog/windows-data-types.md)**

Handle für einen Anzeigedomänencontroller.
