# <a name="effect-reference-direct3d-10-graphics"></a>Effekt Referenz (Direct3D 10-Grafiken)

Die Direct3D-API definiert mehrere API-Elemente, die Sie beim Erstellen und Verwalten des Effekt Systems unterstützen.

-   [Schnittstellen](d3d10-graphics-reference-effect-interfaces.md)
-   [Funktionen](d3d10-graphics-reference-effect-functions.md)
-   [Strukturen](d3d10-graphics-reference-effect-structures.md)
-   [Konstanten](d3d10-graphics-reference-effect-constants.md)

Das Effektsystem kapselt Zustands Zuweisungen in einem Effekt. Jeder Effekt ist eine Auflistung von Pipeline Zuständen, die unter Verwendung von Zustands Blöcken, Ressourcen und Shadern in Zustands Zuweisungen aufgeteilt werden können. Abhängig von der Größe eines Effekts können Sie auswählen, ob Sie einen in einer ASCII-Text Zeichenfolge oder einer Effekt Datei (. fx) implementieren möchten. Informationen zum Organisieren von Informationen finden Sie im [Effekt Format](d3d10-effect-format.md).

Wenn Sie einen Effekt entwickelt haben, verwenden Sie die Effekt-API, um den Status im Gerät beim Rendern festzulegen. Das Effect-System unterstützt auch eine Reihe von reflektionsapis, um Effekt Daten zu erhalten und festzulegen. Weitere Informationen finden Sie unter [Effect System Interfaces (Direct3D 10)](d3d10-graphics-programming-guide-effects-interfaces.md).
