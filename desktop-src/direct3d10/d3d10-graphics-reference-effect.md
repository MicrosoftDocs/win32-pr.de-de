# <a name="effect-reference-direct3d-10-graphics"></a>Effektverweis (Direct3D 10-Grafiken)

Die Direct3D-API definiert mehrere API-Elemente, mit denen Sie das Effektsystem erstellen und verwalten können.

-   [Schnittstellen](d3d10-graphics-reference-effect-interfaces.md)
-   [Funktionen](d3d10-graphics-reference-effect-functions.md)
-   [Strukturen](d3d10-graphics-reference-effect-structures.md)
-   [Konstanten](d3d10-graphics-reference-effect-constants.md)

Das Effektsystem kapselt Zustandszuweisungen in einem Effekt. Jeder Effekt ist eine Sammlung von Pipelinestatus, die mithilfe von Zustandsblöcken, Ressourcen und Shadern in Zustandszuweisungen unterteilt werden kann. Abhängig von der Größe eines Effekts können Sie einen in einer ASCII-Textzeichenfolge oder einer Effektdatei (FX) implementieren. Informationen zum Organisieren von Informationen in einem Effekt finden Sie im [Effektformat](d3d10-effect-format.md).

Nachdem Sie einen Effekt entworfen haben, verwenden Sie die Effekt-API, um den Zustand auf dem Gerät während des Renderings festzulegen. Außerdem unterstützt das Effektsystem eine Reihe von Reflektions-APIs zum Abrufen und Festlegen von Effektdaten. Weitere Informationen finden Sie unter [Effektsystemschnittstellen (Direct3D 10).](d3d10-graphics-programming-guide-effects-interfaces.md)
