# <a name="d3dx-enumerations-direct3d-10-graphics"></a>D3DX-Enumerationen (Direct3D 10-Grafiken)

Dieser Abschnitt enthält Informationen zu den folgenden aufzähligen Typen und Flags, die mit D3DX verwendet werden:



| Enumeration                                                       | Beschreibung                                                                                                                                                                            |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_D3DX10-KANALFLAG \_**](d3dx10-channel-flag.md)              | Diese Flags werden von Funktionen verwendet, die auf einem oder mehreren Kanälen in einer Textur arbeiten.                                                                                                  |
| [**\_D3DX-CPU-OPTIMIERUNG \_**](d3dx-cpu-optimization.md)          | Gibt den Anweisungssatz an, für den D3DX derzeit optimiert ist.                                                                                                                         |
| [**D3DX10 \_ ERR**](d3dx10-err.md)                                 | Fehler werden durch negative Werte dargestellt und können nicht kombiniert werden.                                                                                                                      |
| [**\_D3DX10-FILTERFLAG \_**](d3dx10-filter-flag.md)                | Flags für die Texturfilterung.                                                                                                                                                               |
| [**\_D3DX10-BILDDATEIFORMAT \_ \_**](d3dx10-image-file-format.md)   | Beschreibt die unterstützten Bilddateiformate.                                                                                                                                            |
| [**D3DX10 \_ MESH**](d3dx10-mesh.md)                               | Flags, die zum Angeben von Erstellungsoptionen für ein Gitternetz verwendet werden.                                                                                                                                     |
| [**D3DX10 \_ MESH \_ DISCARD \_ FLAGS**](d3dx10-mesh-discard-flags.md) | Gibt an, welche Gitternetzdaten vom Gerät verworfen werden sollen. Wird mit [**ID3DX10Mesh::D iscard**](id3dx10mesh-discard.md)verwendet.                                                         |
| [**D3DX10 \_ MESHOPT**](d3dx10-meshopt.md)                         | Gibt den Typ der durchzuführenden Meshoptimierung an.                                                                                                                               |
| [**D3DX10 \_ \_ NORMALMAP-FLAG**](d3dx10-normalmap-flag.md)          | Diese Flags werden verwendet, um zu steuern, wie [**D3DX10ComputeNormalMap**](d3dx10computenormalmap.md) normale Zuordnungen generiert. Eine beliebige Anzahl dieser Flags kann OR in beliebiger Kombination sein. |
| [**D3DX10 \_ SAVE \_ TEXTURE \_ FLAG**](d3dx10-save-texture-flag.md)   | Optionen zum Speichern von Texturen.                                                                                                                                                                  |
