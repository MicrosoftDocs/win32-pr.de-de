# <a name="d3dx-enumerations-direct3d-10-graphics"></a>D3DX-Enumerationen (Direct3D 10-Grafiken)

Dieser Abschnitt enthält Informationen zu den folgenden enumerierten Typen und Flags, die mit D3DX verwendet werden:



| Enumeration                                                       | Beschreibung                                                                                                                                                                            |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**D3dx10 \_ - \_ kanalflag**](d3dx10-channel-flag.md)              | Diese Flags werden von Funktionen verwendet, die auf einem oder mehreren Kanälen in einer Textur arbeiten.                                                                                                  |
| [**D3DX- \_ CPU- \_ Optimierung**](d3dx-cpu-optimization.md)          | Gibt an, dass der Anweisungs Satz D3DX derzeit für optimiert ist.                                                                                                                         |
| [**D3dx10 \_ Err**](d3dx10-err.md)                                 | Fehler werden durch negative Werte dargestellt und können nicht kombiniert werden.                                                                                                                      |
| [**D3dx10 \_ - \_ filterflag**](d3dx10-filter-flag.md)                | Textur Filterflags.                                                                                                                                                               |
| [**D3dx10- \_ Bild \_ Datei \_ Format**](d3dx10-image-file-format.md)   | Beschreibt die unterstützten Bild Dateiformate.                                                                                                                                            |
| [**D3dx10 \_ Mesh**](d3dx10-mesh.md)                               | Flags, die zum Angeben der Erstellungs Optionen für ein Mesh verwendet werden.                                                                                                                                     |
| [**D3dx10 \_ Mesh- \_ Verwerfungs \_ Flags**](d3dx10-mesh-discard-flags.md) | Gibt an, welche Teile von Mesh-Daten vom Gerät verworfen werden sollen. Wird mit [**ID3DX10Mesh::D iscard**](id3dx10mesh-discard.md)verwendet.                                                         |
| [**D3dx10 \_ meshopt**](d3dx10-meshopt.md)                         | Gibt den Typ der auszuführenden Mesh-Optimierung an.                                                                                                                               |
| [**D3dx10 \_ normalmap- \_ Flag**](d3dx10-normalmap-flag.md)          | Diese Flags werden verwendet, um zu steuern, wie [**D3DX10ComputeNormalMap**](d3dx10computenormalmap.md) normale Zuordnungen generiert. Eine beliebige Anzahl dieser Flags kann in beliebiger Kombination oder zusammengefasst werden. |
| [**D3dx10 \_ - \_ Textur Kennzeichen speichern \_**](d3dx10-save-texture-flag.md)   | Texturspeicher Optionen.                                                                                                                                                                  |
