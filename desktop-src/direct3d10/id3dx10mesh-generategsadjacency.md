# <a name="id3dx10meshgenerategsadjacency-method"></a>ID3DX10Mesh::GenerateGSAdjazenzmethode

Fügt dem Indexpuffer des Gitters Adjazenzdaten hinzu. Wenn das Gitternetz an einen Geometrie-Shader gesendet werden soll, der Adjazenzdaten annimmt, muss der Indexpuffer des Gitters Adjazenzdaten enthalten.

## <a name="syntax"></a>Syntax


```C++
HRESULT GenerateGSAdjacency();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **HRESULT**](https://msdn.microsoft.com/library/Bb401631(v=MSDN.10).aspx)**

Der Rückgabewert ist einer der In [Direct3D 10-Rückgabecodes aufgeführten](d3d10-graphics-reference-returnvalues.md)Werte.
