# <a name="single-instance-store-and-sis-backup"></a>Single-Instance Speicherung und SIS-Sicherung

*Single Instance Store*(SIS) ist eine Architektur, die so konzipiert ist, dass doppelte Dateien mit minimalem Datenträger-, Cache-und Sicherungsmedien Aufwand gewartet werden. Eine *Einzelinstanzspeicher-Sicherung* oder SIS-Sicherung ist eine API, die Sicherungs Anwendungen aufrufen, um auf die SIS-Architektur zuzugreifen.
