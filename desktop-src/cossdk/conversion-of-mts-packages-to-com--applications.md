# <a name="conversion-of-mts-packages-to-com-applications"></a>Konvertierung von MTS-Paketen in com+-Anwendungen

Com+ kann zusammen mit der Einführung neuer Features als Zusammenführung von Microsoft Transaction Server (MTS) und com angezeigt werden.

Zum größten Teil werden Ihre MTS-Pakete während des Windows Setup-Verfahrens in com+-Anwendungen konvertiert. Wenn Sie nichts über das typische Setup hinaus tun, können Sie alle neuen com+-Funktionen nutzen. Eine Liste und eine kurze Erläuterung der signifikantesten com+-Verbesserungen finden Sie unter [com+-Programmier Übersicht](com--programming-overview.md).

Weitere Informationen zur automatischen Konvertierung von MTS-Paketen in com+-Anwendungen während des Windows-Setup Vorgangs sowie zur Behandlung von Fehlern, die während dieses Vorgangs auftreten können, finden Sie unter [Automatische Konvertierung von MTS](automatic-conversion-from-mts.md).

Weitere Informationen zum manuellen Konvertieren von MTS-Paketen in com+-Anwendungen finden Sie unter [manuelle Konvertierung von MTS](manual-conversion-from-mts.md).

Beachten Sie außerdem, dass bei der Installation von com+ mehrere neue Funktionen eingeführt werden, die von ihren MTS-Paketen möglicherweise als com+-Anwendungen verwendet werden können. Diese werden unter [com+-Konvertierungs Ergebnisse und-Probleme](com--conversion-results-and-issues.md)erläutert.
