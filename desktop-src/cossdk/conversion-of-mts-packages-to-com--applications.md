# <a name="conversion-of-mts-packages-to-com-applications"></a>Konvertierung von MTS-Paketen in COM+-Anwendungen

COM+ kann zusammen mit der Einführung neuer Features als Zusammenführung von Microsoft Transaction Server (MTS) und COM angezeigt werden.

Zum größten Teil werden Ihre MTS-Pakete während des Setups in COM+-Windows konvertiert. Ohne etwas, das über das typische Setup hinaus geht, können Sie alle neuen COM+-Features nutzen. Eine Liste und eine kurze Erläuterung der wichtigsten COM+-Verbesserungen finden Sie unter [ÜBERSICHT ÜBER COM+-Programmierung.](com--programming-overview.md)

Weitere Informationen zur automatischen Konvertierung von MTS-Paketen in COM+-Anwendungen während des Windows-Setupprozesses sowie zur Behandlung von Fehlern, die während dieses Prozesses auftreten können, finden Sie unter Automatische Konvertierung von [MTS](automatic-conversion-from-mts.md).

Informationen zum manuellen Konvertieren von MTS-Paketen in COM+-Anwendungen finden Sie unter [Manuelle Konvertierung von MTS.](manual-conversion-from-mts.md)

Sie sollten auch beachten, dass die Installation von COM+ mehrere neue Funktionen mit sich bringt, die Ihre MTS-Pakete möglicherweise als COM+-Anwendungen verwenden können. Diese werden unter [COM+-Konvertierungsergebnisse und Probleme erläutert.](com--conversion-results-and-issues.md)
