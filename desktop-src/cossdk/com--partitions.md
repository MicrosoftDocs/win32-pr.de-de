# <a name="com-partitions"></a>Com+-Partitionen

Sie können den com+-Partitions Dienst verwenden, um unterschiedliche Versionen einer COM+-Anwendung auf einem Computer zu installieren und diese gleichzeitig auszuführen.

**Windows XP:** Die Möglichkeit, com+-Partitionen zu erstellen, zu konfigurieren oder zu delegieren, ist nicht verfügbar. Die globale Partition ist die einzige verfügbare com+-Partition.

* * Windows 2000: * * der com+-Partitions Dienst ist in Windows 2000 nicht verfügbar.

Die folgenden Themen enthalten Informationen über den com+-Partitions Dienst.



| Thema                                                               | BESCHREIBUNG                                                                 |
|---------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [Konzepte der COM+-Partitionen](com--partitions-concepts.md)<br/> | Bietet eine Übersicht über die Verwendung von COM+-Partitionen.<br/>                   |
| [Aufgaben für COM+-Partitionen](com--partitions-tasks.md)<br/>       | Enthält Anweisungen zum Erstellen und Verwalten von COM+-Partitionen.<br/> |



 

> [!Note]  
> Der com+-Partitions Dienst ist standardmäßig nicht aktiviert. Um den com+-Partitions Dienst zu verwenden, müssen Sie ihn über das Verwaltungs Tool Komponenten Dienste oder durch Ändern der partitionsenabled-Eigenschaft in der [**LocalComputer**](localcomputer.md) -Sammlung auf true aktivieren.
