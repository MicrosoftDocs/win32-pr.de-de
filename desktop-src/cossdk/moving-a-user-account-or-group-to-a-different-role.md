# <a name="moving-a-user-account-or-group-to-a-different-role"></a>Verschieben eines Benutzerkontos oder einer Gruppe in eine andere Rolle

Sie können Benutzerkonten oder Gruppen aus einer Rolle entfernen, wenn den Benutzern oder Mitgliedern der Gruppen kein Zugriff mehr auf die Anwendungsressourcen gewährt werden soll, denen die Rolle zugewiesen ist.

**So entfernen Sie ein Benutzerkonto oder eine Gruppe aus einer Rolle**

1.  Suchen Sie in der Konsolenstruktur des Component Services-Verwaltungstools die COM+-Anwendung, die die Rolle enthält, an der Sie interessiert sind. Erweitern Sie die Ansicht in der Konsolenstruktur, bis die Benutzer für die Rolle sichtbar sind.

2.  Klicken Sie mit der rechten Maustaste auf das Benutzerkonto oder die Gruppe, das bzw. die Sie entfernen möchten, und klicken Sie dann auf **Löschen.**

3.  Wenn Sie vom Dialogfeld **Elementlöschung bestätigen** aufgefordert werden, klicken Sie auf **Ja,** um das Benutzerkonto oder die Gruppe zu löschen.

Das Benutzerkonto oder die Gruppe, das bzw. die Sie aus der Rolle entfernt haben, wird nicht mehr im Ordner **Benutzer** angezeigt. Die neue Rollenmitgliedschaft wird wirksam, wenn die Anwendung das nächste Mal gestartet wird. Wenn Sie Änderungen an der **Systemanwendung** vorgenommen haben, müssen Sie den Computer neu starten, damit die Änderungen wirksam werden.
