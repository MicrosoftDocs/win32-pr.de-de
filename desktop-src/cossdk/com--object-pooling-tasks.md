# <a name="com-object-pooling-tasks"></a>Com+-objektpoolingaufgaben

Sie können eine Komponente konfigurieren, die in einem Pool zusammengefasst werden soll, indem Sie entweder das Verwaltungs Programmkomponenten Dienste oder die com+-Verwaltungsfunktionen verwenden. Vor der Konfiguration sollten Sie sicher sein, dass die Komponente ordnungsgemäß für die Unterstützung von Pooling geschrieben wurde.

Die folgenden Themen in diesem Abschnitt enthalten Schritt-für-Schritt-Anleitungen zum Verwenden des COM+-Objekt Pooling.



| Thema                                                                                       | BESCHREIBUNG                                                                                                                                 |
|---------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [Konfigurieren einer Komponente, die in einem Pool zusammengefasst werden soll](configuring-a-component-to-be-pooled.md)<br/> | Beschreibt, wie eine in einem Pool zusammengefasste Komponente erstellt wird.<br/>                                                                                      |
| [Objekt Statistik überwachen](monitoring-object-statistics.md)<br/>                 | Beschreibt das Überwachen von Objekten, sodass Sie die Eigenschaften des Objekt Pools optimieren können, um die gewünschte Leistung zu erzielen. <br/> |
