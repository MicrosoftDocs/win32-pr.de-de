# <a name="com-applications-running-as-service-applications"></a>Com+-Anwendungen, die als Dienst Anwendungen ausgeführt werden

Com+-Entwickler können das Verwaltungs Programmkomponenten Dienste verwenden, um eine COM+-Serveranwendung als Dienst Anwendung zu konfigurieren und zu implementieren. Dies bedeutet, dass der Server automatisch gestartet oder neu gestartet werden kann, wenn die Anwendung immer ausgeführt werden muss. Ihre COM+-Anwendung kann als lokales Systemkonto ausgeführt werden, wenn Sie privilegierte Vorgänge ausführen muss. und die abhängigen Dienste Ihrer Anwendung können nun automatisch gestartet werden.

Die folgenden Themen enthalten Hintergrundinformationen und Aufgabeninformationen zum Konfigurieren Ihrer Anwendungen für die Anwendung als Dienst Anwendungen.

| [Aufgaben der com+-Dienst Anwendung](com--service-application-tasks.md)       | Enthält Anweisungen zum Konfigurieren einer COM+-Serveranwendung als Dienst.                       |
