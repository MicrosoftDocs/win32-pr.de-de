# <a name="setting-the-synchronization-attribute"></a>Festlegen des Synchronisierungsattributs

Legen Sie das Synchronisierungsattribut entweder mit dem Component Services-Verwaltungstool oder der COM+-Verwaltungsbibliothek fest. (Weitere Informationen [finden Sie unter Automatisieren der COM+-Verwaltung.)](automating-com--administration.md)

**So legen Sie das Attribut mithilfe des Component Services-Verwaltungstools fest**

1.  Öffnen Sie nach der Installation der Komponente das Dialogfeld **Eigenschaften** der Komponente.

2.  Klicken Sie **auf die Registerkarte Parallelität.**

3.  Klicken **Sie unter Synchronisierungsunterstützung** auf die Optionsschaltfläche neben dem Wert, den Sie für die Komponente auswählen möchten.

4.  Klicken Sie auf **OK**.
