# <a name="setting-the-synchronization-attribute"></a>Festlegen des Synchronisierungs Attributs

Legen Sie das Synchronisierungs Attribut entweder mithilfe des Verwaltungs Tools Komponenten Dienste oder der com+-Verwaltungs Bibliothek fest. (Weitere Informationen finden Sie unter [Automatisieren der com+-Verwaltung](automating-com--administration.md).)

**So legen Sie das-Attribut mithilfe des Verwaltungs Tools "Komponenten Dienste" fest**

1.  Nachdem Sie die Komponente installiert haben, öffnen Sie das Dialogfeld **Eigenschaften** der Komponente.

2.  Klicken Sie **auf die Register** Karte Parallelität.

3.  Klicken Sie unter **Synchronisierungs Unterstützung** auf das Optionsfeld neben dem Wert, den Sie für die Komponente auswählen möchten.

4.  Klicken Sie auf **OK**.
