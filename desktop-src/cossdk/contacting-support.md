# <a name="contacting-support"></a>Kontaktieren des Supports

Wenn ein Problem auftritt, das Sie nicht lösen können, wenden Sie sich mit den folgenden Informationen an den technischen Support von Microsoft:

-   Die Topografie der Anwendung, in der der Fehler aufgetreten ist, z. b. eine Beschreibung der Anwendungen, Komponenten und Schnittstellen.
-   Anwendungs Ereignisprotokoll auf allen Computern.
-   Beschreibung der Art der Reproduktion des Fehlers, wenn möglich.
