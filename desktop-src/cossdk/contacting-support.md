# <a name="contacting-support"></a>Kontaktieren des Supports

Wenn ein Problem auftritt, das Sie nicht lösen können, können Sie sich mit den folgenden Informationen an den technischen Support von Microsoft wenden:

-   Topografie der Anwendung, in der der Fehler aufgetreten ist, z. B. eine Beschreibung von Anwendungen, Komponenten und Schnittstellen.
-   Anwendungsereignisprotokoll auf allen Computern.
-   Beschreibung, wie der Fehler nach Möglichkeit reproduziert wird.
