# <a name="com-compensating-resource-manager"></a>Kompensierende com+-Ressourcen-Manager

Das com+-kompensierende Ressourcen-Manager (CRM) bietet eine schnelle und einfache Möglichkeit, Anwendungs Ressourcen in Microsoft Distributed Transaction Coordinator (DTC)-Transaktionen zu integrieren. Die Verwendung von CRM ist eine Alternative zum Entwickeln eines vollständigen Ressourcen-Managers von Microsoft Transaction Services (MTS).

Die folgenden Themen enthalten Hintergrundinformationen und Aufgabeninformationen zur Verwendung von CRM.
