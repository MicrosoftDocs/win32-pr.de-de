# <a name="com-administration-classes"></a>Com+-Verwaltungs Klassen

Die com+-Verwaltungs Klassen sind die folgenden:



| Klasse                                                          | BESCHREIBUNG                                                                                                                                            |
|----------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Comadmincatalog**](comadmincatalog.md)                     | Greift auf die Daten zu, die im com+-Katalog gespeichert sind.                                                                                                  |
| [**Comadmincatalogcollection**](comadmincatalogcollection.md) | Stellt eine beliebige Auflistung im com+-Katalog dar. Verwenden Sie diese Option, um Elemente in einer Auflistung aufzulisten, hinzuzufügen, zu entfernen und abzurufen und um auf verwandte Auflistungen zuzugreifen. |
| [**COMAdminCatalogObject**](comadmincatalogobject.md)         | Stellt Elemente in Auflistungen im com+-Katalog dar. Verwenden Sie es zum Abrufen und Ändern von Eigenschaften, die von einem Element in einer Sammlung verfügbar gemacht werden.                      |
