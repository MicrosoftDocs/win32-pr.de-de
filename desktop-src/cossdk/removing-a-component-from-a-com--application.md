# <a name="removing-a-component-from-a-com-application"></a>Entfernen einer Komponente aus einer COM+-Anwendung

Mit dem Verwaltungs Programmkomponenten Dienste können Sie Komponenten aus einer COM+-Anwendung entfernen.

> [!Note]  
> Achten Sie darauf, wenn Sie einzelne Komponenten aus einer COM+-Anwendung löschen und neu installieren. Dieser Prozess erfordert ausführliche Informationen über die Klassen der Komponente, einschließlich ihrer Attribute, Sicherheits Konfigurationen für Schnittstellen und Methoden sowie anderer Einstellungen.

 

**So entfernen Sie eine Komponente aus einer COM+-Anwendung**

1.  Wählen Sie in der Konsolen Struktur des Verwaltungs Programms Komponenten Dienste die Anwendung aus, die die Komponente enthält, die Sie entfernen möchten.

2.  Öffnen Sie den Ordner **Komponenten** der Anwendung.

3.  Wählen Sie die Komponente aus, die Sie entfernen möchten.

4.  Klicken Sie im Menü **Aktion** auf **Löschen**. Sie können auch die Komponente auswählen, mit der rechten Maustaste klicken und dann auf **Löschen** klicken.

5.  Klicken Sie auf **Ja** , wenn Sie zur Bestätigung Ihrer Aktion aufgefordert werden.
