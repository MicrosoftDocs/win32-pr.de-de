# <a name="com-partitions-tasks"></a>Aufgaben für COM+-Partitionen

Die folgenden Themen in diesem Abschnitt enthalten Schritt-für-Schritt-Anleitungen für die Verwendung von COM+-Partitionen.

**Windows XP:** Die Möglichkeit, com+-Partitionen zu erstellen, zu konfigurieren oder zu delegieren, ist nicht verfügbar. Die globale Partition ist die einzige verfügbare com+-Partition.

* * Windows 2000: * * der com+-Partitions Dienst ist in Windows 2000 nicht verfügbar.



| Thema                                                                                                         | BESCHREIBUNG                                                                                    |
|---------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------|
| [Erstellen und Konfigurieren von COM+-Partitionen](creating-and-configuring-com--partitions.md)<br/>           | Hier wird beschrieben, wie eine COM+-Partition erstellt und konfiguriert wird.<br/>                             |
| [Verwalten von lokalen Partitionen](managing-local-partitions.md)<br/>                                         | Beschreibt, wie COM+-Partitionen verwaltet werden, die nicht in Active Directory vorhanden sind.<br/>       |
| [Verwalten von Partitionen in Active Directory](managing-partitions-within-active-directory.md)<br/>     | Beschreibt, wie COM+-Partitionen verwaltet werden, die in Active Directory angegeben werden.<br/> |
| [Festlegen von Administratorrechten für eine Partition](setting-administrative-rights-for-a-partition.md)<br/> | Beschreibt, wie Sicherheits Privilegien für eine COM+-Partition festgelegt werden.<br/>                      |
| [Gruppieren von Anwendungen in Partitionen](grouping-applications-into-partitions.md)<br/>                 | Beschreibt, wie COM+-Anwendungen für eine COM+-Partition konfiguriert werden. <br/>        |

> [!Note]  
> Der com+-Partitions Dienst ist standardmäßig nicht aktiviert. Um den com+-Partitions Dienst zu verwenden, müssen Sie ihn über das Verwaltungs Tool Komponenten Dienste oder durch Ändern der partitionsenabled-Eigenschaft in der [**LocalComputer**](localcomputer.md) -Sammlung auf true aktivieren.
