# <a name="com-object-constructor-strings"></a>Com+-objektkonstruktorzeichenfolgen

Com+-objektkonstruktorzeichenfolgen unterstützen generische Programmierung. Sie ermöglichen das Schreiben einer einzelnen Komponente mit einer gewissen Generalität, damit Sie für eine bestimmte Aufgabe mithilfe administrativer festgelegter Konstruktorzeichenfolgen angepasst werden kann.
