# <a name="designing-com-applications"></a>Entwerfen von COM+-Anwendungen

Unabhängig davon, ob Sie eine Anwendung für den Onlinehandel, das Nachverfolgen von Informationen, den Zugriff auf Daten in Legacysystemen, Messaging, Ressourcenplanung, Veröffentlichung und Wissensverwaltung oder eine kritische Branchenanwendung (Branchenanwendung) entwerfen, können Sie Ihren Entwurf der mittleren Ebene verbessern und optimieren, indem Sie COM+-Dienste integrieren. Die gängigste Methode hierfür ist die Gruppierung von Komponenten in COM+-Anwendungen.

Während eine vollständige Methodiküberprüfung für die Verwendung von COM+ in allen Situationen den Rahmen dieses Dokuments sprengen würde, enthalten die folgenden Themen Informationen, die Sie beim Erstellen einer COM+-Anwendung verwenden können.
