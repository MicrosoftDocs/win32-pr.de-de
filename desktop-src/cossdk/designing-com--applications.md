# <a name="designing-com-applications"></a>Entwerfen von COM+-Anwendungen

Unabhängig davon, ob Sie eine Anwendung zum Durchführen von Onlinehandel, Nachverfolgen von Informationen, Zugreifen auf Daten in älteren Systemen, Messaging, Ressourcenplanung, Veröffentlichung und Wissensverwaltung oder eine kritische BRANCHENanwendung (Line of Business) entwerfen, können Sie Ihren Entwurf auf mittlerer Ebene durch die Integration von COM+-Diensten verbessern und optimieren. Die gängigste Methode besteht in der Gruppierung von Komponenten in COM+-Anwendungen.

Während eine vollständige Methodiküberprüfung für die Verwendung von COM+ in allen Situationen über den Rahmen dieses Dokuments hinaus geht, stellen die folgenden Themen Informationen vor, die Sie beim Erstellen einer COM+-Anwendung verwenden können.
