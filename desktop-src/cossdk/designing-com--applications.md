# <a name="designing-com-applications"></a>Entwerfen von com+-Anwendungen

Unabhängig davon, ob Sie eine Anwendung für die Durchführung von Online-Commerce, das Nachverfolgen von Informationen, den Zugriff auf Daten in Legacy Systemen, Messaging, Ressourcenplanung, Veröffentlichung und Wissensverwaltung oder eine wichtige LOB-Anwendung (Line of Business) entwickeln, können Sie den Entwurf der mittleren Ebene durch die Integration von COM+-Diensten verbessern und optimieren. Die gängigste Methode hierfür ist das Gruppieren von Komponenten in com+-Anwendungen.

Während eine vollständige Methoden Überprüfung für die Verwendung von com+ in allen Situationen über den Rahmen dieses Dokuments hinausgeht, enthalten die folgenden Themen Informationen, die Sie beim Erstellen einer COM+-Anwendung verwenden können.
