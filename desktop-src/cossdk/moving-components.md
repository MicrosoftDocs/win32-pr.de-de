# <a name="moving-components"></a>Verschieben von Komponenten

Sie können eine Komponente von einer COM+-Anwendung in eine andere verschieben. Tatsächlich wird durch das Verschieben einer Komponente die Komponente aus der ursprünglichen COM+-Anwendung entfernt und in der com+-Zielanwendung installiert.

**So verschieben Sie eine Komponente von einer COM+-Anwendung in eine andere**

1.  Klicken Sie im Detailbereich des Verwaltungs Programms Komponenten Dienste mit der rechten Maustaste auf die Komponente, die Sie verschieben möchten, und klicken Sie dann auf **verschieben**.

2.  Wählen Sie im Dialogfeld **Komponente (n) verschieben** die com+-Zielanwendung aus, zu der die Komponente verschoben werden soll.

3.  Klicken Sie auf **OK**.
