# <a name="com-instrumentation-interfaces"></a>Com+-Instrumentierungs Schnittstellen

Die com+-instrumentierungsmetriken werden mithilfe von [com+-Ereignissen](com--events.md), einem locker gekoppelten Ereignis (LCE), das Ereignis Informationen von verschiedenen Verlegern in einem Ereignisspeicher im com+-Katalog speichert, auf einem Client veröffentlicht. Die Metriken werden basierend auf dem Objekttyp, für den Sie Informationen bereitstellen, in Kategorien gruppiert. Für jede Kategorie gibt es eine entsprechende Schnittstelle, die der Client abonnieren kann, und der abonnierende Client empfängt alle Metriken in dieser Kategorie. Der Client kann auch bestimmte Methoden der Schnittstelle abonnieren, um bestimmte Metriken durch Filtern von Metriken zu erhalten.

Die in der folgenden Tabelle beschriebenen com+-Instrumentations Schnittstellen ermöglichen Clients das Empfangen von Benachrichtigungen für eine Reihe von com+-Instrumentations Metriken.
