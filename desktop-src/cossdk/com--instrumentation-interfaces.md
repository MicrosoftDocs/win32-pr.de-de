# <a name="com-instrumentation-interfaces"></a>COM+-Instrumentierungsschnittstellen

Die COM+-Instrumentierungsmetriken werden auf einem Client mithilfe von [COM+-Ereignissen](com--events.md)veröffentlicht. Dabei handelt es sich um ein lose gekoppeltes Ereignissystem (LCE), das Ereignisinformationen von verschiedenen Herausgebern in einem Ereignisspeicher im COM+-Katalog speichert. Die Metriken werden basierend auf dem Typ des Objekts, für das sie Informationen bereitstellen, in Kategorien gruppiert. Für jede Kategorie gibt es eine freundliche Schnittstelle, die der Client abonnieren kann, und der abonnierende Client empfängt alle Metriken in dieser Kategorie. Der Client kann auch bestimmte Methoden auf der Schnittstelle abonnieren, um bestimmte Metriken durch Filtern von Metriken zu empfangen.

Die in der folgenden Tabelle beschriebenen COM+-Instrumentierungsschnittstellen ermöglichen Clients den Empfang von Benachrichtigungen für eine Vielzahl von COM+-Instrumentierungsmetriken.
