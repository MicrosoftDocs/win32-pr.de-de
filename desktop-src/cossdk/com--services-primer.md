# <a name="com-services-primer"></a>COM+ Services Primer

COM+ bietet ein einfaches Programmiermodell für die Verwendung seiner automatischen Dienste. Sie können diese Dienste einfach als deklarative Attribute für Komponenten und deren Methoden festlegen. Wenn Sie diese Attribute festlegen, übermittelt COM+ die angeforderten Dienste nach Bedarf automatisch an das Objekt.

In dieser Einführung werden COM+-Dienste in Aktion gezeigt, indem Beispielkomponentencode in drei definierten Schritten durchlaufen wird. Die Komplexität der Informationen wird im Verlauf der Schritte erhöht. Der Code konzentriert sich auf das vereinfachte Programmiermodell, wobei die Transaktionsverarbeitung verwendet wird, um grundlegende COM+-Prinzipien zu veranschaulichen.

Jeder der folgenden Schritte veranschaulicht einen grundlegenden Aspekt der COM+-Dienste:

-   [Schritt 1: Erstellen einer Transaktionskomponente](step-1--creating-a-transactional-component.md)
-   [Schritt 2: Erweitern einer Transaktion über mehrere Komponenten hinweg](step-2--extending-a-transaction-across-multiple-components.md)
-   [Schritt 3: Wiederverwenden von Komponenten](step-3--reusing-components.md)
