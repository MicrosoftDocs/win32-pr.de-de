# <a name="com-services-primer"></a>Einführung in COM+-Dienste

Com+ stellt ein einfaches Programmiermodell für die Verwendung der automatischen Dienste bereit. Sie können diese Dienste einfach als deklarative Attribute für Komponenten und deren Methoden festlegen. Wenn Sie diese Attribute festlegen, übergibt com+ die angeforderten Dienste automatisch nach Bedarf an das Objekt.

Diese Einführung zeigt com+-Dienste in Aktion, indem Sie den Beispiel Komponenten Code in drei definierten Schritten durchlaufen. Die Komplexität der Informationen wird im Verlauf der Schritte erstellt. Der Code konzentriert sich auf das vereinfachte Programmiermodell, wobei die Transaktionsverarbeitung verwendet wird, um grundlegende com+-Prinzipien zu veranschaulichen.

Jeder der folgenden Schritte veranschaulicht einen grundlegenden Aspekt der com+-Dienste:

-   [Schritt 1: Erstellen einer transaktionalen Komponente](step-1--creating-a-transactional-component.md)
-   [Schritt 2: Erweitern einer Transaktion über mehrere Komponenten hinweg](step-2--extending-a-transaction-across-multiple-components.md)
-   [Schritt 3: wieder verwenden von Komponenten](step-3--reusing-components.md)
