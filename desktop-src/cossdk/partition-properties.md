# <a name="partition-properties"></a>Partitionseigenschaften

Partitionen verfügen über fünf verfügbare Eigenschaften. Diese Eigenschaften und ihre Beschreibungen sind wie folgt:

<dl> <dt>

<span id="Partition_Name"></span><span id="partition_name"></span><span id="PARTITION_NAME"></span>Partitionsname
</dt> <dd>

Eine der Partition zugeordnete Textzeichenfolge.

</dd> <dt>

<span id="Partition_ID"></span><span id="partition_id"></span><span id="PARTITION_ID"></span>Partitions-ID
</dt> <dd>

Eine GUID, die das System verwendet, um einen Benutzer mit einer Partition zu korrelieren.

</dd> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>Beschreibung
</dt> <dd>

Eine Textbeschreibung der Partition.

</dd> <dt>

<span id="Disable_Deletion"></span><span id="disable_deletion"></span><span id="DISABLE_DELETION"></span>Löschen deaktivieren
</dt> <dd>

Ein Mechanismus, mit dem verhindert wird, dass ein Benutzer eine Partition löscht.

</dd> <dt>

<span id="Disable_Changes"></span><span id="disable_changes"></span><span id="DISABLE_CHANGES"></span>Deaktivieren von Änderungen
</dt> <dd>

Ein Mechanismus, mit dem verhindert wird, dass ein Benutzer Änderungen an einer Partition vornimmt.

</dd> </dl>

Diese Eigenschaften können angezeigt werden, indem Sie im Verwaltungstool Komponentendienste mit der rechten Maustaste auf einen Partitionsnamen klicken und **Eigenschaften** auswählen.

Weitere Informationen zur Partitions-ID-Eigenschaft finden Sie unter [Registrieren und Aktivieren von Komponenten in Partitionen.](registering-and-activating-components-in-partitions.md)
