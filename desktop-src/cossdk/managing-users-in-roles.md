# <a name="managing-users-in-roles"></a>Verwalten von Benutzern in Rollen

Die meisten Verwaltungsaufgaben für die Konfiguration der com+-Anwendungssicherheit beinhalten das Auffüllen von vordefinierten Rollen mit Benutzern, sowohl bei der Installation von Anwendungen als auch fortlaufend. Einführende Informationen zur Funktionsweise dieser Sicherheitsfeatures finden Sie unter [Konfigurieren von Role-Based Sicherheit](configuring-role-based-security.md).

In diesem Abschnitt werden Schritt-für-Schritt-Anweisungen für die folgenden Aufgaben bereitgestellt:

-   [Aktivieren der Role-Based Autorisierungs Überprüfung](enabling-role-based-authorization-checking.md)
-   [Zuweisen eines Benutzerkontos oder einer Gruppe zu einer Rolle](assigning-a-user-account-or-group-to-a-role.md)
-   [Entfernen eines Benutzerkontos oder einer Gruppe aus einer Rolle](removing-a-user-account-or-group-from-a-role.md)
-   [Verschieben eines Benutzerkontos oder einer Gruppe in eine andere Rolle](moving-a-user-account-or-group-to-a-different-role.md)
