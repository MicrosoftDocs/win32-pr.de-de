# <a name="copying-components"></a>Kopieren von Komponenten

Sie können eine Komponente von einer COM+-Anwendung in eine andere kopieren. Nachdem Sie eine Komponente in eine andere COM+-Anwendung kopiert haben, können Sie diese Komponente anders konfigurieren als die ursprüngliche Komponente.

**So kopieren Sie eine Komponente von einer COM+-Anwendung in eine andere**

1.  Klicken Sie im Detailbereich des Verwaltungs Programms Komponenten Dienste mit der rechten Maustaste auf die Komponente, die Sie kopieren möchten, und klicken Sie dann auf **Kopieren**.

2.  Wählen Sie im Dialogfeld **Komponente kopieren** im Bereich **Ziel auswählen** die COM+-Anwendung aus, auf die die Komponente kopiert werden soll.

3.  Geben Sie die neue ProgID für die kopierte Komponente ein.

    > [!Note]  
    > Die ProgID für die ursprüngliche Komponente wird als Verweis angezeigt und kann nicht geändert werden.

     

4.  Geben Sie die neue CLSID für die kopierte Komponente ein. Com+ zeigt eine automatisch generierte CLSID an. In den meisten Fällen ist dies ausreichend, um die kopierte Komponente eindeutig zu identifizieren. Geben Sie eine neue CLSID nur dann ein, wenn Sie die von com+ bereitgestellte ändern möchten.

5.  Klicken Sie auf **OK**.
