# <a name="copying-components"></a>Kopieren von Komponenten

Sie können eine Komponente aus einer COM+-Anwendung in eine andere kopieren. Nachdem Sie eine Komponente in eine andere COM+-Anwendung kopiert haben, können Sie diese Komponente anders konfigurieren als die ursprüngliche Komponente.

**So kopieren Sie eine Komponente aus einer COM+-Anwendung in eine andere**

1.  Klicken Sie im Detailbereich des Verwaltungstool Komponentendienste mit der rechten Maustaste auf die Komponente, die Sie kopieren möchten, und klicken Sie dann auf **Kopieren.**

2.  Wählen Sie **im Dialogfeld** Komponente  kopieren im Bereich Ziel auswählen die COM+-Anwendung aus, in die die Komponente kopiert werden soll.

3.  Geben Sie die neue ProgID für die kopierte Komponente ein.

    > [!Note]  
    > Die ProgID für die ursprüngliche Komponente wird als Referenz angezeigt und kann nicht geändert werden.

     

4.  Geben Sie die neue CLSID für die kopierte Komponente ein. COM+ zeigt eine automatisch generierte CLSID an. In den meisten Fällen reicht dies aus, um die kopierte Komponente eindeutig zu identifizieren. Geben Sie eine neue CLSID nur ein, wenn Sie die von COM+ bereitgestellte ÄNDERN möchten.

5.  Klicken Sie auf **OK**.
