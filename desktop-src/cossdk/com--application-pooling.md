# <a name="com-application-pooling"></a>Com+-Anwendungs Pooling

Das com+-Anwendungs Pooling ermöglicht das Skalieren von Single Thread-Prozessen und unterstützt Sie außerdem bei der Wiederherstellung nach Fehlern in einzelnen Prozessen, indem andere laufende Prozesse zur Handhabung von Aktivierungen bereitgestellt werden.

Die folgenden Themen enthalten Hintergrundinformationen und Aufgabeninformationen zur Verwendung des com+-Anwendungs Pooling.
