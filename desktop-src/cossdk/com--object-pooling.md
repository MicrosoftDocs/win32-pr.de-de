# <a name="com-object-pooling"></a>COM+-Objektpooling

Mit dem COM+-Objektpoolingdienst können Sie eine Komponente so konfigurieren, dass Instanzen von sich selbst in einem Pool aktiv bleiben und von jedem Client verwendet werden können, der die Komponente anfordert.

Die in der folgenden Tabelle beschriebenen Themen enthalten Hintergrund- und Aufgabeninformationen zur Verwendung von Objektpooling in COM+.



| Thema                                                                       | BESCHREIBUNG                                                                                                |
|-----------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [COM+-Objektpoolingkonzepte](com--object-pooling-concepts.md)<br/> | Stellt Informationen zur Verwendung des COM+-Objektpoolingdiensts bereit, um die Systemleistung zu verbessern.<br/> |
| [COM+-Objektpoolingtasks](com--object-pooling-tasks.md)<br/>       | Enthält ausführliche Anweisungen zum Implementieren und Überwachen des COM+-Objektpoolings.<br/>             |
