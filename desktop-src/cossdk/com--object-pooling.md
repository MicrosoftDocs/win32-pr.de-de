# <a name="com-object-pooling"></a>Com+-Objekt Pooling

Der com+-objektpoolingdienst ermöglicht es Ihnen, eine Komponente so zu konfigurieren, dass Instanzen von selbst in einem Pool aktiv gehalten werden und von jedem Client verwendet werden können, der die Komponente anfordert.

Die in der folgenden Tabelle beschriebenen Themen enthalten Hintergrundinformationen und Aufgabeninformationen zur Verwendung von Objekt Pooling in com+.
