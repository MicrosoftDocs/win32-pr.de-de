# <a name="com-transactions-concepts"></a>COM+-Transaktionskonzepte

Obwohl COM+ viele der mühsamen Programmierdetails im Zusammenhang mit der Transaktionsverarbeitung verarbeitet, ist es hilfreich, ein konzeptionelles Verständnis der Transaktionstheorie zu haben, wenn Sie Transaktionen in COM+ programmieren müssen.

In den in der folgenden Tabelle beschriebenen Themen werden die Konzepte behandelt, die für die Transaktionsverarbeitung gelten.



| Thema                                                                                                                     | BESCHREIBUNG                                                                                                                                                                                 |
|---------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [ACID-Eigenschaften](acid-properties.md)<br/>                                                                         | Beschreibt die atomaren, konsistenten, isolierten und dauerhaften Eigenschaften von Transaktionen.<br/>                                                                                              |
| [Konfigurieren von Transaktionen](configuring-transactions.md)<br/>                                                       | Beschreibt die Transaktionsattributwerte, die Sie Ihren Komponenten zuweisen können, um den Grad des zu erzwingenden Transaktionsschutzes zu bestimmen.<br/>                              |
| [Konfigurieren von Transaktionsisolationsstufen](configuring-transaction-isolation-levels.md)<br/>                       | Beschreibt die Isolationsstufen, die Sie Ihren Transaktionen zuweisen können, sodass Sie die Parallelität je nach Ihren Leistungs- und Skalierbarkeitsanforderungen erhöhen oder verringern können.<br/> |
| [Verwalten automatischer Transaktionen in COM+](managing-automatic-transactions-in-com-.md)<br/>                         | Übersicht über den automatisierten Transaktionsprozess, der von COM+ unterstützt wird. <br/>                                                                                                                |
| [Verwenden nicht transaktionaler Komponenten in einer Transaktion](using-non-transactional-components-in-a-transaction.md)<br/> | Beschreibt, wie nicht transaktionale Komponenten in einer Transaktion verwendet werden und wann Sie sie verwenden würden.<br/>                                                                                  |
| [Bring Your Own Transaction (BYOT)](bring-your-own-transaction--byot-.md)<br/>                                     | Beschreibt, wie eine Komponente eine externe Transaktion erben kann.<br/>                                                                                                              |
