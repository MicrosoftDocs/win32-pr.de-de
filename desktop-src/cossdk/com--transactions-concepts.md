# <a name="com-transactions-concepts"></a>Konzepte der com+-Transaktionen

Obwohl com+ viele der mühsamen Programmier Details verarbeitet, die mit der Transaktionsverarbeitung in Zusammenhang stehen, ist es hilfreich, wenn Sie Transaktionen in com+ programmieren müssen, um ein konzeptionelles Verständnis der Transaktions Theorie zu erhalten.

In den in der folgenden Tabelle beschriebenen Themen werden die Konzepte behandelt, die für die Transaktionsverarbeitung gelten.



| Thema                                                                                                                     | BESCHREIBUNG                                                                                                                                                                                 |
|---------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [ACID-Eigenschaften](acid-properties.md)<br/>                                                                         | Beschreibt die atomarischen, konsistenten, isolierten und dauerhaften Eigenschaften von Transaktionen.<br/>                                                                                              |
| [Konfigurieren von Transaktionen](configuring-transactions.md)<br/>                                                       | Beschreibt die Transaktions Attributwerte, die Sie den-Komponenten zuweisen können, um zu bestimmen, welcher Grad an Transaktions Schutz erzwungen werden soll.<br/>                              |
| [Konfigurieren von Transaktions Isolations Stufen](configuring-transaction-isolation-levels.md)<br/>                       | Beschreibt die Isolations Stufen, die Sie Ihren Transaktionen zuweisen können, sodass Sie die Parallelität abhängig von den Leistungs-und Skalierbarkeits Anforderungen erhöhen oder verringern können.<br/> |
| [Verwalten von automatischen Transaktionen in com+](managing-automatic-transactions-in-com-.md)<br/>                         | Übersicht über den automatisierten Transaktionsprozess, der von com+ unterstützt wird. <br/>                                                                                                                |
| [Verwenden von nicht transaktionalen Komponenten in einer Transaktion](using-non-transactional-components-in-a-transaction.md)<br/> | Beschreibt, wie nicht transaktionale Komponenten in einer Transaktion verwendet werden und wann Sie verwendet werden.<br/>                                                                                  |
| [Bring Your Own Transaction (BYOT)](bring-your-own-transaction--byot-.md)<br/>                                     | Beschreibt, wie eine Komponente eine externe Transaktion erben kann.<br/>                                                                                                              |
