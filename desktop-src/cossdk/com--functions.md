# <a name="com-functions"></a>Com+-Funktionen

Im folgenden finden Sie die com+-Funktionen.



| Funktion                                                                 | BESCHREIBUNG                                                                                                                             |
|--------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**Cokreateactivity**](/windows/desktop/api/ComSvcs/nf-comsvcs-cocreateactivity)                             | Erstellt eine Aktivität für die synchrone oder asynchrone Batchverarbeitung, die COM+-Dienste verwenden kann, ohne dass eine COM+-Komponente erstellt werden muss. |
| [**Coenterservicedomain**](/windows/desktop/api/ComSvcs/nf-comsvcs-coenterservicedomain)                     | Wird verwendet, um Code einzugeben, der dann com+-Dienste verwenden kann.                                                                                     |
| [**Cogetdefaultcontext**](/windows/desktop/api/combaseapi/nf-combaseapi-cogetdefaultcontext)                       | Ruft einen Verweis auf den Standardkontext des angegebenen Apartment ab.                                                                |
| [**Coleaveservicedomain**](/windows/desktop/api/ComSvcs/nf-comsvcs-coleaveservicedomain)                     | Wird verwendet, um Code zu verlassen, der com+-Dienste verwendet.                                                                                             |
| **Compluscompletecbbsetup**               | Schließt eine Katalog Migration auf dem Zielcomputer ab.                                                                              |
| **Getcompluspackageingestallstatus** | Gibt an, ob die Common Language Runtime (CLR) von 64 Bit installiert ist.                                                                |
| [**Getdispenser Server-Manager**](/windows/desktop/api/MtxDM/nf-mtxdm-getdispensermanager)                       | Ruft die [**idispenser Manager Manager**](/windows/desktop/api/ComSvcs/nn-comsvcs-idispensermanager) -Schnittstelle des Dispenser-Managers ab.                                             |
| [**Getmanagedextensions**](/windows/desktop/api/ComSvcs/nf-comsvcs-getmanagedextensions)                     | Bestimmt, ob die installierte Version von com+ spezielle Features unterstützt, die zum Verwalten von Serviced Components (verwalteten Objekten) bereitgestellt werden.    |
| [**GetObjectContext**](/windows/desktop/api/ComSvcs/nf-comsvcs-getobjectcontext)                             | Ruft einen Verweis auf den Kontext ab, der dem aktuellen com+-Objekt zugeordnet ist.                                                   |
| [**Mtscreateactivity**](/windows/desktop/api/ComSvcs/nf-comsvcs-mtscreateactivity)                           | Erstellt eine Aktivität in einem Single Thread-Apartment, um synchrone oder asynchrone Batch Vorgänge auszuführen.                                        |
| [**Recycleersatz**](/windows/desktop/api/ComSvcs/nf-comsvcs-recyclesurrogate)                             | Wieder verwendet den aufrufenden Prozess.                                                                                                           |
| [**Saferef**](/windows/desktop/api/ComSvcs/nf-comsvcs-saferef)                                               | Veralteten Verwenden Sie nicht.                                                                                                                   |
