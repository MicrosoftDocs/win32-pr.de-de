# <a name="com-administration-interfaces"></a>Com+-Verwaltungs Schnittstellen

Im folgenden sind die com+-Verwaltungs Schnittstellen aufgeführt.



| Schnittstelle                                        | BESCHREIBUNG                                                                                                                                                                                      |
|--------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Icatalogcollection**](/windows/desktop/api/ComAdmin/nn-comadmin-icatalogcollection) | Ermöglicht das auflisten, hinzufügen, entfernen und Abrufen von Elementen in einer beliebigen Auflistung im com+-Katalog und den Zugriff auf verwandte Auflistungen.                                                               |
| [**Icatalogobject**](/windows/desktop/api/ComAdmin/nn-comadmin-icatalogobject)         | Ermöglicht das Get und Platzieren von Eigenschaften, die von Objekten im com+-Katalog verfügbar gemacht werden.                                                                                                                    |
| [**Icomadmincatalog**](/windows/desktop/api/ComAdmin/nn-comadmin-icomadmincatalog)     | Initiiert eine Sitzung für die programmgesteuerte com+-Verwaltung, den Zugriff auf Auflistungen im Katalog, die Installation von com+-Anwendungen und-Komponenten, das Starten und die Beendigung von Diensten und das Herstellen einer Verbindung mit |
| [**ICOMAdminCatalog2**](/windows/desktop/api/ComAdmin/nn-comadmin-icomadmincatalog2)   | Steuert die Interaktionen von Anwendungen, Komponenten und Partitionen.                                                                                                                           |
