# <a name="general-design-tips-for-using-com"></a>Allgemeiner Entwurf Tipps für die Verwendung von COM+

Der Schlüssel zu einem guten Entwurf ist die Planung. Wenn Sie mit dem Entwerfen einer Anwendung mit drei Ebenen nicht vertraut sind, lesen Sie den Abschnitt [Entwerfen der COM+-Anwendung mit UML.](designing-the-com--application-using-uml.md)

Wie Sie die COM-Komponenten erstellen, die die mittlere Ebene Ihrer COM+-Anwendung bilden, kann sich erheblich auf leistung, skalierbarkeit und einfache Bereitstellung auswirken. Die folgenden Themen enthalten Entwurfsüberlegungen und Implementierungstipps, die Sie beim Entwerfen von COM-Komponenten kennen sollten.



| Thema                                                                   | BESCHREIBUNG                                                                                                                |
|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [Entwerfen für Skalierbarkeit](designing-for-scalability.md)<br/>   | Schlägt Möglichkeiten vor, die Skalierbarkeit Ihrer COM+-Anwendung zu erhöhen.<br/>                                             |
| [Entwerfen für Verfügbarkeit](designing-for-availability.md)<br/> | Erläutert Dienste, die Sie verwenden können, um die Verfügbarkeit Ihrer Funktionen der mittleren Ebene in COM+-Anwendungen zu verbessern.<br/> |
| [Entwerfen für Die Sicherheit](designing-for-security.md)<br/>         | Erläutert Möglichkeiten zum effizientesten Einrichten der Sicherheit für eine COM+-Anwendung.<br/>                                      |
| [Entwerfen für die Bereitstellung](designing-for-deployment.md)<br/>     | Erläutert Möglichkeiten zum Minimieren von Bereitstellungsproblem beim Entwerfen einer verteilten Anwendung.<br/>                       |



 

Stellen Sie außerdem sicher, dass Sie unter [Verbessern der Leistung mit Objektpooling](improving-performance-with-object-pooling.md) nach Möglichkeiten zur effektivsten Verwendung von Objektpooling suchen.
