# <a name="partitions-and-active-directory"></a><span data-ttu-id="df880-103">Partitionen und Active Directory</span><span class="sxs-lookup"><span data-stu-id="df880-103">Partitions and Active Directory</span></span>

<span data-ttu-id="df880-104">Zusätzlich zu Partitionen, die eine oder mehrere Anwendungen enthalten, umfasst com+ auch *Partitions Sätze*, die eine oder mehrere Partitionen enthalten.</span><span class="sxs-lookup"><span data-stu-id="df880-104">In addition to partitions, which contain one or more applications, COM+ also includes *partition sets*, which contain one or more partitions.</span></span> <span data-ttu-id="df880-105">Die in Active Directory erstellten Partitions Sätze ermöglichen Benutzern in einer Domäne den Zugriff auf COM+-Anwendungen in der gesamten Domäne.</span><span class="sxs-lookup"><span data-stu-id="df880-105">Created in Active Directory, partition sets allow users in a domain to access COM+ applications throughout the domain.</span></span> <span data-ttu-id="df880-106">Während der Erstellung wird jeder bestimmte Benutzer oder jede Organisationseinheit (OE) einem Partitions Satz zugewiesen oder zugeordnet.</span><span class="sxs-lookup"><span data-stu-id="df880-106">During creation, each specific user or organizational unit (OU) is assigned, or mapped, to a partition set.</span></span>

<span data-ttu-id="df880-119">Der Administrator erstellt eine neue Partition und gibt den gleichen Partitionsnamen und die gleiche Partitions-ID (GUID) wie die der Partition an, die in Active Directory auf dem Domänen Controller erstellt wurde.</span><span class="sxs-lookup"><span data-stu-id="df880-119">The administrator creates a new partition, specifying the same partition name and partition ID (GUID) as that of the partition that was created within Active Directory on the domain controller.</span></span> <span data-ttu-id="df880-120">Dies erfolgt über den **com+** -Ordner "Partitionen" im Verwaltungs Programm "Komponenten Dienste".</span><span class="sxs-lookup"><span data-stu-id="df880-120">This is done using the **COM+ Partitions** folder within the Component Services administrative tool.</span></span> <span data-ttu-id="df880-121">Um diese Aufgabe zu vereinfachen, ermöglicht das Komponenten Dienst Tool dem Administrator, Active Directory zu durchsuchen, um die gewünschte Partition und ihre GUID auszuwählen.</span><span class="sxs-lookup"><span data-stu-id="df880-121">To simplify this task, the Component Services tool allows the administrator to browse Active Directory to select the desired partition and its GUID.</span></span>

<span data-ttu-id="df880-124">Weitere Informationen zum Verwalten von Partitionen für den Zugriff durch Domänen Benutzer finden Sie in der Hilfe im Zusammenhang mit dem Verwaltungs Tool "Active Directory-Benutzer und-Computer".</span><span class="sxs-lookup"><span data-stu-id="df880-124">For more information about administering partitions for access by domain users, see the help associated with the Active Directory Users and Computers administrative tool.</span></span>
