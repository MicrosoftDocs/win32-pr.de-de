# <a name="local-partitions"></a>Lokale Partitionen

Partitionen können lokal auf einem Anwendungsserver verwendet werden. In diesem Szenario kann ein Administrator mithilfe des Verwaltungs Programms Komponenten Dienste eine Partition erstellen. Lokale Partitionen, die nicht in Active Directory implementiert sind, sind nur für lokale Benutzer zugänglich, nicht für Benutzer auf anderen Computern in der Domäne. Nachdem eine Partition erstellt und eine COM+-Anwendung darin installiert wurde, kann der Administrator diese Partition als Standard Partition für einen oder mehrere Benutzer auf diesem Server zuweisen. Um einem Benutzer eine Standard Partition zuzuweisen, kann der Administrator das Verwaltungs Programmkomponenten Dienste verwenden.
