# <a name="pointer-device-input-stack"></a>Zeigergeräteeingabestapel

## <a name="purpose"></a>Zweck

Die Themen in diesem Abschnitt bieten eine Übersicht über den Eingabestapel des Windows-Zeigergeräts. Die [Eingabestapel-APIs](/windows/win32/api/_input_pointerdevice/) für Zeigergeräte stellen Informationen zu den Eingabegeräten bereit, die mit dem System verbunden sind.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|---|---|
| [Zeigergeräteeingabestapelreferenz](unified-input-stack-reference.md)<br/> | Die Themen in diesem Abschnitt enthalten die Referenzspezifikationen für den Zeigergeräteeingabestapel in Windows 8.<br/> |

## <a name="developer-audience"></a>Entwicklergruppe

Die Pointer Device Input StackAPIs sind für Entwickler konzipiert, die Informationen über das Zeigergerät benötigen, das für die Interaktion mit ihrer App verwendet wird.
