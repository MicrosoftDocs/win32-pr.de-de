# <a name="pointer-device-input-stack"></a>Eingabe Stapel für Zeiger Geräte

## <a name="purpose"></a>Zweck

Die Themen in diesem Abschnitt bieten eine Übersicht über den Geräte Eingabe Stapel für Windows-Zeiger. Die [Zeiger Geräte-Eingabe Stapel](/windows/win32/api/_input_pointerdevice/) -APIs stellen Informationen zu den Eingabegeräten bereit, die mit dem System verbunden sind.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|---|---|
| [Verweis auf Zeiger Geräte-Eingabe Stapel Referenz](unified-input-stack-reference.md)<br/> | Die Themen in diesem Abschnitt enthalten die Referenz Spezifikationen für den Zeiger Geräte-Eingabe Stapel in Windows 8.<br/> |

## <a name="developer-audience"></a>Entwicklergruppe

Die Zeiger Geräte Eingabe stackapis sind für Entwickler konzipiert, die Informationen über das Zeiger Gerät benötigen, das für die Interaktion mit Ihrer APP verwendet wird.
