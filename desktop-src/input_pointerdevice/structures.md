# <a name="pointer-device-input-stack-structures"></a>Eingabe Stapel Strukturen für Zeiger Geräte

Die Themen in diesem Abschnitt enthalten die Referenz Spezifikationen für [Zeiger Geräte-Eingabe Stapel](pointer-device-stack-portal.md) Strukturen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|---|---|
| [**POINTER_DEVICE_CURSOR_INFO**](/windows/win32/api/winuser/ns-winuser-pointer_device_cursor_info)<br/> | Enthält Cursor-ID-Zuordnungen für Zeiger Geräte.<br/> |
| [**POINTER_DEVICE_INFO**](/windows/win32/api/winuser/ns-winuser-pointer_device_info)<br/> | Enthält Informationen zu einem Zeiger Gerät. Ein Array dieser Strukturen wird von der [**getpointerdevices**](/windows/win32/api/winuser/nf-winuser-getpointerdevices) -Funktion zurückgegeben. Eine einzelne Struktur wird von einem Aufrufen der [**getpointerdevice**](/windows/win32/api/winuser/nf-winuser-getpointerdevice) -Funktion zurückgegeben. <br/> |
| [**POINTER_DEVICE_PROPERTY**](/windows/win32/api/winuser/ns-winuser-pointer_device_property)<br/> | Enthält Geräteeigenschaften (Eingabegeräte (HID) globale Elemente, die den verborgenen Verwendungen entsprechen).<br/> |
