# <a name="pointer-device-input-stack-structures"></a>Zeigergeräteeingabestapelstrukturen

Die Themen in diesem Abschnitt enthalten die Referenzspezifikationen für [Zeigergeräte-Eingabestapelstrukturen.](pointer-device-stack-portal.md)

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|---|---|
| [**POINTER_DEVICE_CURSOR_INFO**](/windows/win32/api/winuser/ns-winuser-pointer_device_cursor_info)<br/> | Enthält Cursor-ID-Zuordnungen für Zeigergeräte.<br/> |
| [**POINTER_DEVICE_INFO**](/windows/win32/api/winuser/ns-winuser-pointer_device_info)<br/> | Enthält Informationen zu einem Zeigergerät. Ein Array dieser Strukturen wird von der [**GetPointerDevices-Funktion**](/windows/win32/api/winuser/nf-winuser-getpointerdevices) zurückgegeben. Eine einzelne Struktur wird von einem Aufruf der [**GetPointerDevice-Funktion**](/windows/win32/api/winuser/nf-winuser-getpointerdevice) zurückgegeben. <br/> |
| [**POINTER_DEVICE_PROPERTY**](/windows/win32/api/winuser/ns-winuser-pointer_device_property)<br/> | Enthält globale Geräteeigenschaften (Eingabegeräte (HID), die HID-Verwendungen entsprechen.<br/> |
