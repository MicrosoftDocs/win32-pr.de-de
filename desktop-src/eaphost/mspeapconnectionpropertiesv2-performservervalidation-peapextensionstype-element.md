# <a name="performservervalidation-peapextensionstype-element-v2-schema"></a>PerformServerValidation (PeapExtensionsType)-Element (V2-Schema)

Das **Element PerformServerValidation (PeapExtensionsType)** gibt an, ob die Servervalidierung ausgeführt wird.

``` syntax
<xs:element name="PerformServerValidation"
    type="xs:boolean"
 />
```

Das **PerformServerValidation-Element** wird durch das [**PeapExtensionsType-Element**](mspeapconnectionpropertiesv1schema-peapextensionstype-complextype.md) definiert.

## <a name="remarks"></a>Hinweise

Das **PerformServerValidation-Element** ist optional.

## <a name="requirements"></a>Anforderungen



| Role | Minimale Version des Betriebssystems |
|------|--------------------|
| Client<br/> | nur Windows 7 \[ Desktop-Apps\]<br/>              |
| Server<br/> | Windows Nur Server 2008 \[ R2-Desktop-Apps\]<br/> |
