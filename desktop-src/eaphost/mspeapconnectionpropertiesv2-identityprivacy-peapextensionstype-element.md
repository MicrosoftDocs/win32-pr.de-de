# <a name="the-identityprivacy-peapextensionstype-element"></a>Das identityprivacy (Peer-extensionstype)-Element

Das **identityprivacy (papextensionstype)** -Element gibt an, ob die tatsächliche Identität eines Benutzers oder eine anonyme Identität gesendet wird.

``` syntax
<xs:element name="IdentityPrivacy"
    type="IdentityPrivacyParameters"
 />
```

Das **identityprivacy** -Element wird durch das " [**Peer-extensionstype**](mspeapconnectionpropertiesv1schema-peapextensionstype-complextype.md) "-Element definiert.

## <a name="remarks"></a>Bemerkungen

Das **identityprivacy** -Element ist optional.
