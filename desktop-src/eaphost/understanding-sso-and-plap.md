# <a name="sso-and-plap"></a>SSO und PLAP

Die Single Sign-On (SSO)-API stellt eine Reihe von Methoden dar, mit denen auf sichere Weise EAP-Methoden spezifische Anmelde Informationen für ein Netzwerk Benutzer-oder Computer Konto abgerufen werden, ohne dass mehrere Benutzeroberflächen Instanzen erhoben werden müssen.

> [!Note]  
> Der Begriff "Pre-Logon Access Provider" (PLAP) ist veraltet. Wenn "Pre-Logon Access Provider" oder "PLAP" im Code oder in der Dokumentation gefunden wird, ersetzen Sie es durch "einmaliges Anmelden" bzw. "SSO".
