# <a name="eaphost-supplicant-callbacks"></a>EAPHost Supplicant-Rückrufe

Die EAPHost Supplicant-API-Rückruffunktionen lauten wie folgt.



| Rückruf                                           | Beschreibung                                                                                                                                                                                                                                     |
|----------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**NotificationHandler**](/previous-versions/windows/desktop/api) | Ein Rückrufprototyp, der die Annahme benachrichtigt, dass die SoH-Anweisung (Statement of Health) und die erneute Authentifizierung einer NAP-Systemverbindung [(Network Access Protection)](/windows/desktop/NAP/network-access-protection-start-page) geändert werden müssen. |
