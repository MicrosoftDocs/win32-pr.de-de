# <a name="eaphost-supplicant-callbacks"></a>EAPHost-Supplicant-Rückrufe

Die Rückruf Funktionen der EAPHost-Supplicant-API lauten wie folgt.



| Rückruf                                           | BESCHREIBUNG                                                                                                                                                                                                                                     |
|----------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Notificationhandler**](/previous-versions/windows/desktop/api) | Ein Rückruf Prototyp, der den Supplicant benachrichtigt, dass eine Änderung des SoH (Statement of Health) und die erneute Authentifizierung einer Systemverbindung für den [Netzwerk Zugriffsschutz (Network Access Protection](/windows/desktop/NAP/network-access-protection-start-page) , NAP) erforderlich sind. |
