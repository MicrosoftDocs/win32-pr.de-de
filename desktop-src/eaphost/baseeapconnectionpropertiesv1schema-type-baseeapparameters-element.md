# <a name="type-baseeapparameters-element"></a>Type (baseeapparameters)-Element

Das **Type (baseeapparameters)** -Element definiert das Platzhalter Element für den ausgewählten Methodentyp und die Methoden spezifische Konfiguration.

``` syntax
<xs:element name="Type"
    type="integer"
 />
```

Das **Type** -Element wird durch den komplexen [**baseeapparameters**](baseeapconnectionpropertiesv1schema-baseeapparameters-complextype.md) -Typ definiert.
