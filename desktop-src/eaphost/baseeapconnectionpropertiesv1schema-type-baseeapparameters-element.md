# <a name="type-baseeapparameters-element"></a>Type(BaseEapParameters)-Element

Das **Type-Element (BaseEapParameters)** definiert das Platzhalterelement für den ausgewählten Methodentyp und die methodenspezifische Konfiguration.

``` syntax
<xs:element name="Type"
    type="integer"
 />
```

Das **Type-Element** wird durch den komplexen [**BaseEapParameters-Typ**](baseeapconnectionpropertiesv1schema-baseeapparameters-complextype.md) definiert.
