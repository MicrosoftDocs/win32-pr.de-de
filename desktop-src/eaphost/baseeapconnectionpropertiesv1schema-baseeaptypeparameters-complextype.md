# <a name="baseeaptypeparameters-complex-type---connection-properties"></a>Baseeaptypeparameters komplexer Typ-Verbindungs Eigenschaften

Der komplexe Typ **baseeaptypeparameters** ist ein Basiselement des Methoden spezifischen Elements, das im abgeleiteten Element des [BaseEapConnectionPropertiesV1-Schemas](baseeapconnectionpropertiesv1schema-schema.md)angezeigt wird.

``` syntax
<xs:complexType name="BaseEapTypeParameters"
    abstract="true"
 />
```

## <a name="remarks"></a>Bemerkungen

**Baseeaptypeparameters** ist ein abstraktes Element. Ein abgeleitetes Element von **baseeaptypeparameters** muss in Instanzdokumenten verwendet werden.
