# <a name="baseeaptypeparameters-complex-type---connection-properties"></a>Komplexer BaseEapTypeParameters-Typ – Verbindungseigenschaften

Der komplexe **BaseEapTypeParameters-Typ** ist ein Basiselement des methodenspezifischen Elements, das im abgeleiteten Element des [BaseEapConnectionPropertiesV1-Schemas](baseeapconnectionpropertiesv1schema-schema.md)angezeigt wird.

``` syntax
<xs:complexType name="BaseEapTypeParameters"
    abstract="true"
 />
```

## <a name="remarks"></a>Hinweise

**BaseEapTypeParameters** ist ein abstraktes Element. Ein abgeleitetes Element von **BaseEapTypeParameters** muss in Instanzdokumenten verwendet werden.
