# <a name="servervalidation-eaptype-element-tls"></a>ServerValidation -Element (EapType) (TLS)

Das **Element ServerValidation (EapType)** enthält Informationen zum Durchführen der Servervalidierung.

``` syntax
<xs:element name="ServerValidation"
    type="ServerValidationParameters"
 />
```

Das **ServerValidation-Element** wird durch das [**EapType-Element**](eaptlsconnectionpropertiesv1schema-eaptype-element.md) definiert.

## <a name="remarks"></a>Hinweise

Das **ServerValidation-Element** ist optional.
