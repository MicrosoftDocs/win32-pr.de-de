# <a name="about-eaphost"></a>Informationen zu EAPHost

EAPHost implementiert den EAP-Zustands Automat (Extensible Authentication Protocol) und das EAP-Protokoll Framework gemäß [RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063).

EAPHost unterstützt EAP-basierten [Netzwerk Zugriffsschutz (Network Access Protection](/windows/desktop/NAP/network-access-protection-start-page) , NAP).
