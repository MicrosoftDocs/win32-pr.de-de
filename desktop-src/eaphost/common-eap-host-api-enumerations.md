# <a name="common-eaphost-api-enumerations"></a>Allgemeine EAPHost-API-Enumerationen

Die folgenden Enumerationen gelten für alle EAPHost-API-Technologien.



| Enumeration                                                                  | Beschreibung                                                                                   |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**\_EAP-ATTRIBUTTYP \_**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_attribute_type)                           | Definiert den Satz möglicher EAP-Attributtypen, die für eine authentifizierende Entität verfügbar sind.        |
| [**EINGABEFELDTYP \_ "EAP-KONFIGURATION" \_ \_ \_**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_config_input_field_type)       | Definiert einen Satz möglicher Eingabefeldtypen, die beim Abfragen von Anmeldeinformationen für den Benutzer verfügbar sind. |
| [**\_ \_ EAP-METHODENEIGENSCHAFTSTYP \_**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_type)              | Definiert den Satz möglicher EAP-Methodeneigenschaften.                                            |
| [**\_ \_ EAP-METHODENEIGENSCHAFTSWERTTYP \_ \_**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_value_type) | Definiert den Satz möglicher Datentypen für die Methodeneigenschaften.                             |
| [**EapCode**](/windows/win32/api/eapmethodtypes/ne-eapmethodtypes-eapcode)                                                   | Definiert den Satz von EAP-Pakettypen.                                                          |
