# <a name="common-eaphost-api-enumerations"></a>Allgemeine EAPHost-API-Enumerationen

Die folgenden Enumerationen sind für alle EAPHost-API-Technologien üblich.



| Enumeration                                                                  | Beschreibung                                                                                   |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**EAP \_ - \_ Attributtyp**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_attribute_type)                           | Definiert den Satz möglicher EAP-Attributtypen, die für eine authentifizier Ende Entität verfügbar sind.        |
| [**\_ \_ Eingabe \_ \_ Feldtyp der EAP-Konfiguration**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_config_input_field_type)       | Definiert einen Satz möglicher Eingabefeld Typen, die bei der Abfrage des Benutzers nach Anmelde Informationen verfügbar sind. |
| [**EAP \_ - \_ Methoden \_ Eigenschaftentyp**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_type)              | Definiert den Satz möglicher EAP-Methoden Eigenschaften.                                            |
| [**Eigenschafts \_ \_ \_ \_ Werttyp der EAP-Methode**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_value_type) | Definiert den Satz möglicher Datentypen für die Methoden Eigenschaften.                             |
| [**Eapcode**](/windows/win32/api/eapmethodtypes/ne-eapmethodtypes-eapcode)                                                   | Definiert den Satz von EAP-Pakettypen.                                                          |
