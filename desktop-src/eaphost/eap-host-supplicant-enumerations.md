# <a name="eaphost-supplicant-enumerations"></a>EAPHost-Supplicant-Enumerationen

Die Enumerationen der EAPHost-Supplicant-API lauten wie folgt.



| Enumeration                                                                  | Beschreibung                                                                                                        |
|------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**interaktiver EAP- \_ \_ UI- \_ \_ Datentyp**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_interactive_ui_data_type)     | Gibt den Satz von Typen interaktiver Kontext Daten der Benutzeroberfläche an, die für bestimmte Supplicant-API-Aufrufe bereitgestellt werden.    |
| [**EAP \_ - \_ Methoden \_ Eigenschaftentyp**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_type)              | Windows 7 oder höher: gibt den Eigenschaftentyp der Methode an.                                                            |
| [**Eigenschafts \_ \_ \_ \_ Werttyp der EAP-Methode**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_value_type) | Windows 7 oder höher: gibt den Datentyp des Eigenschafts Werts einer Methode an.                                            |
| [**EAPHost-Authentifizierungs \_ \_ Status**](/windows/desktop/api/eaphostpeertypes/ne-eaphostpeertypes-eaphost_auth_status)                         | Definiert den Satz möglicher EAP-Authentifizierungs Sitzungs Statuswerte.                                              |
| [**Eaphostpeer-authparametriams**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeerauthparams)                       | Definiert den Satz möglicher Authentifizierungs Parameterwerte.                                                       |
| [**Eaphostpeer-methodresulverrat**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeermethodresultreason)       | Definiert den Satz möglicher Gründe, die die von einer EAP-Methode zurückgegebenen Ergebnisse an einen Supplicant beschreiben.           |
| [**Eaphostpeer-Response Action**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeerresponseaction)               | Definiert den Satz von Aktionen, die ein EAP-Authentifikator oder eine Peer Methode für einen Supplicant während der Authentifizierung angeben kann. |
| [**Isolations \_ Status**](/windows/desktop/api/eaphostpeertypes/ne-eaphostpeertypes-isolation_state)                                  | Definiert den Satz möglicher Isolations Statuswerte eines Computers.                                                  |
