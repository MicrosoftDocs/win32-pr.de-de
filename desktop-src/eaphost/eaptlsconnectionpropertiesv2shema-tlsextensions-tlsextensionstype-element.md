# <a name="performservervalidation-element"></a>Performservervalidation-Element

Das **performservervalidation (eaptype)** -Element gibt an, ob die Server Validierung ausgeführt wird.

``` syntax
<xs:element name="PerformServerValidation"
    type="xs:boolean"
 />
```

Das Element " **performservervalidation** " wird durch das [**eaptype**](eaptlsconnectionpropertiesv1schema-eaptype-element.md) -Element definiert.

## <a name="remarks"></a>Bemerkungen

Das **performservervalidation** -Element ist optional.

## <a name="requirements"></a>Anforderungen



| Role | Minimale Version des Betriebssystems |
|------|--------------------|
| Client<br/> | Nur Windows 7 \[ -Desktop-Apps\]<br/>              |
| Server<br/> | Nur Windows Server 2008 R2 \[ -Desktop-Apps\]<br/> |
