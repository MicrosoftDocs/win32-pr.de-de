# <a name="performservervalidation-element"></a>PerformServerValidation-Element

Das **Element PerformServerValidation (EapType)** gibt an, ob die Serverüberprüfung ausgeführt wird.

``` syntax
<xs:element name="PerformServerValidation"
    type="xs:boolean"
 />
```

Das **PerformServerValidation-Element** wird durch das [**EapType-Element**](eaptlsconnectionpropertiesv1schema-eaptype-element.md) definiert.

## <a name="remarks"></a>Hinweise

Das **PerformServerValidation-Element** ist optional.

## <a name="requirements"></a>Anforderungen



| Role | Minimale Version des Betriebssystems |
|------|--------------------|
| Client<br/> | nur Windows 7 \[ Desktop-Apps\]<br/>              |
| Server<br/> | Windows Nur Server 2008 \[ R2-Desktop-Apps\]<br/> |
