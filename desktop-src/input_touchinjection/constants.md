# <a name="touch-injection-constants"></a>Touch Injection-Konstanten

Dieser Abschnitt enthält die Referenzspezifikationen für [Touch Injection-Konstanten.](touch-injection-portal.md)

| Konstante/Wert | BESCHREIBUNG |
|---|---|
| **MAX_TOUCH_COUNT** 256                            | Gibt die maximale Anzahl gleichzeitiger Kontakte an.<br/> |
| **TOUCH_FEEDBACK_DEFAULT** 0x1    | Gibt standardmäßige Touchvisualisierungen an.<br/>                |
| **TOUCH_FEEDBACK_INDIRECT** 0x2 | Gibt indirekte Touchvisualisierungen an.<br/>               |
| **TOUCH_FEEDBACK_NONE** 0x3             | Gibt keine Touchvisualisierungen an.<br/>                     |
