# <a name="touch-injection"></a>Touch Injection

## <a name="purpose"></a>Zweck

Die Themen in diesem Abschnitt bieten eine Übersicht über Touch Injection für Windows Anwendungen. Die [Touch Injection-APIs](/windows/win32/api/_input_touchinjection/) ermöglichen Windows Entwicklern, Toucheingaben programmgesteuert zu simulieren.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|---|---|
| [Referenz zur Toucheinjektion](touch-injection-reference.md)<br/> | Die Themen in diesem Abschnitt enthalten die Referenzspezifikationen für Touch Injection.<br/> |

## <a name="developer-audience"></a>Entwicklergruppe

Die Touch Injection-APIs sind für Entwickler konzipiert, die Toucheingabeautomatisierungs- und Testframeworks mit C++ erstellen.
