# <a name="ms-ds-entry-time-to-die-attribute"></a>ms-DS-Entry-Time-to-das-Attribut

Enthält die absolute Ablaufzeit eines dynamischen Objekts im Verzeichnis.



| Eingabe | Wert |
|-------------------|---------------------------------------------------------------|
| CN                | ms-DS-Entry-Time-to-be                                       |
| LDAP-Display-Name | MSDS-Entry-Time-to-be                                        |
| Size              | \-                                                            |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                              |
| Aktualisierungshäufigkeit  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.1622                                       |
| System-ID-GUID    | e1e9bad7-c6dd-4101-A843-794cec85b038                          |
| Syntax            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
