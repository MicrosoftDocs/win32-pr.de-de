# <a name="ms-ds-entry-time-to-die-attribute"></a>ms-DS-Entry-Time-To-Die-Attribut

Enthält die absolute Ablaufzeit eines dynamischen Objekts im Verzeichnis.



| Eingabe | Wert |
|-------------------|---------------------------------------------------------------|
| CN                | ms-DS-Entry-Time-To-Die                                       |
| Ldap-Anzeigename | msDS-Entry-Time-To-Die                                        |
| Size              | \-                                                            |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.                              |
| Updatehäufigkeit  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.1622                                       |
| System-ID-GUID    | e1e9bad7-c6dd-4101-a843-794cec85b038                          |
| Syntax            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
