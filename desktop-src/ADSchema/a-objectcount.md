# <a name="object-count-attribute"></a>Object-Count-Attribut

Das überwachte Datei Kontingent für ein bestimmtes Volume.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Object-Count                         |
| LDAP-Display-Name | objectCount                          |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.506               |
| System-ID-GUID    | 34aaa216-b699-11D0-Afee-0000b C1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
