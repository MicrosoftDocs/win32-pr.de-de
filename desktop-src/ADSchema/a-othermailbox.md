# <a name="other-mailbox-attribute"></a>Other-Mailbox-Attribut

Enthält andere zusätzliche e-Mail-Adressen in einem Formular wie z. b. CCMAIL: brucekeever.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------|
| CN                | Other-Mailbox                                            |
| LDAP-Display-Name | othermailbox                                             |
| Size              | \-                                                       |
| Berechtigung aktualisieren  | Domänen Administrator                                     |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn der Benutzer eine zusätzliche e-Mail-Adresse ausgibt. |
| Attribute-Id      | 1.2.840.113556.1.4.651                                   |
| System-ID-GUID    | 0296c123-40DA-11d1-a9c0-0000b80367c1                     |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)              |
