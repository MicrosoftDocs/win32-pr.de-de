# <a name="other-mailbox-attribute"></a>Other-Mailbox-Attribut

Enthält weitere zusätzliche E-Mail-Adressen in einem Formular, z. B. CCMAIL:IttKeever.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------|
| CN                | Other-Mailbox                                            |
| Ldap-Anzeigename | otherMailbox                                             |
| Size              | \-                                                       |
| Aktualisieren von Berechtigungen  | Domänenadministrator                                     |
| Updatehäufigkeit  | Jedes Mal, wenn dem Benutzer eine zusätzliche E-Mail-Adresse ausgegeben wird. |
| Attribute-Id      | 1.2.840.113556.1.4.651                                   |
| System-Id-Guid    | 0296c123-40da-11d1-a9c0-0000f80367c1                     |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)              |
