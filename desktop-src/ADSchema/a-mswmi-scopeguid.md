# <a name="ms-wmi-scopeguid-attribute"></a>MS-WMI-scopeguid-Attribut

Der GUID für den Bereich, in dem sich die zugeordnete Codierung befindet.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------|
| CN                | MS-WMI-scopeguid                                                |
| LDAP-Display-Name | mswap-scopeguid                                                 |
| Size              | 16 Bytes                                                        |
| Berechtigung aktualisieren  | Domänen Administrator                                            |
| Aktualisierungshäufigkeit  | Wenn eine Klasse, die das Attribut enthält, hinzugefügt oder geändert wird. |
| Attribute-Id      | 1.2.840.113556.1.4.1686                                         |
| System-ID-GUID    | 87b78d51-405f -4b7f -80ed-2bd28786f 48d                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                     |
