# <a name="acs-minimum-latency-attribute"></a>ACS-Attribut mit minimaler Latenzzeit

Das Attribut " **ACS-minimaler Latenz** " ist nur für die interne Verwendung vorgesehen. Basierend auf RFC2210.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-minimale Latenzzeit                  |
| LDAP-Display-Name | acsminimumlatency                    |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1316              |
| System-ID-GUID    | 9517fef b-3b90-11d2-90cc-00c04f d91ab1 |
| Syntax            | [**Intervall**](s-interval.md)       |
