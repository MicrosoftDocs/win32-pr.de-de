# <a name="address-book-roots-attribute"></a>Address-Book-Roots-Attribut

Wird von Exchange verwendet. Exchange konfiguriert Strukturen von Adressbuchcontainern, die im MAPI-Adressbuch angezeigt werden. Dieses Attribut für das Exchange Config-Objekt listet die Stammstrukturen der Adressbuchcontainerstrukturen auf.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Address-Book-Roots                      |
| Ldap-Anzeigename | addressBookRoots                        |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | \-                                      |
| Updatehäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1244                 |
| System-ID-GUID    | f70b6e48-06f4-11d2-aa53-00c04fd7d83a    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
