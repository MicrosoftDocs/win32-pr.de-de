# <a name="pki-overlap-period-attribute"></a>PKI-Überschneidungszeit Attribut

Der Zeitraum, nach dem das Zertifikat erneuert werden soll, bevor es abgelaufen ist.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | PKI-Überlappung-Zeitraum                                    |
| LDAP-Display-Name | pkioverlapperiod                                      |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1332                               |
| System-ID-GUID    | 1219a3ec-3b9e-11d2-90cc-00c04f d91ab1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
