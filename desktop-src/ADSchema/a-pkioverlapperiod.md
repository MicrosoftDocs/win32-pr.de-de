# <a name="pki-overlap-period-attribute"></a>PKI-Overlap-Period-Attribut

Der Zeitraum, bis zu dem das Zertifikat erneuert werden soll, bevor es abgelaufen ist.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | PKI-Überlappungszeitraum                                    |
| Ldap-Anzeigename | pKIOverlapPeriod                                      |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | \-                                                    |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1332                               |
| System-Id-Guid    | 1219a3ec-3b9e-11d2-90cc-00c04fd91ab1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
