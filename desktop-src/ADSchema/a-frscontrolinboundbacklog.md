| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | FRS-Steuerung-eingehender Rückstand                 |
| LDAP-Display-Name | frscontrolinboundrückstand                    |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.872                      |
| System-ID-GUID    | 2a13257b-9373-11d1-AEbc-0000e80367c1        |
