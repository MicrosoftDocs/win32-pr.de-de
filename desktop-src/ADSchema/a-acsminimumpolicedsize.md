# <a name="acs-minimum-policed-size-attribute"></a>ACS-minimal-Richtlinien Attribut

Das **ACS-minimal-Richtwert** -Attribut ist nur für die interne Verwendung vorgesehen. Basierend auf RFC2210.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-minimal-Richtlinien Größe             |
| LDAP-Display-Name | acsminimumpolicedsize                |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1315              |
| System-ID-GUID    | 8d0e7195-3b90-11d2-90cc-00c04f d91ab1 |
| Syntax            | [**Intervall**](s-interval.md)       |
