# <a name="frs-time-last-command-attribute"></a>FRS-Time-Last-Command-Attribut

Der Zeitpunkt, zu dem der letzte Befehl ausgeführt wurde.



| Eingabe | Wert |
|-------------------|---------------------------------------------------------------|
| CN                | FRS-Time-Last-Command                                         |
| LDAP-Display-Name | frstimelastcommand                                            |
| Size              | \-                                                            |
| Berechtigung aktualisieren  | \-                                                            |
| Aktualisierungshäufigkeit  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.880                                        |
| System-ID-GUID    | 2a132583-9373-11d1-AEbc-0000 C1                          |
| Syntax            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
