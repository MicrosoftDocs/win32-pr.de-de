# <a name="active-directory-schema-ad-schema"></a>Active Directory-Schema (AD-Schema)

Das Microsoft Active Directory-Schema enthält formale Definitionen jeder Objektklasse, die in einer Active Directory-Gesamtstruktur erstellt werden kann. Das Schema enthält auch formale Definitionen jedes Attributs, das in einem Active Directory-Objekt vorhanden sein kann. Dieser Abschnitt enthält die Referenz für jedes Schemaobjekt und eine kurze Erläuterung der Attribute, Klassen und anderen Objekte, aus denen das Active Directory-Schema erstellt wird.

> [!Note]  
> Die folgende Dokumentation enthält die Programmierreferenz für das Active Directory-Schema. Wenn Sie ein Endbenutzer sind, der versucht, einen Druckerfehler zu debuggen, versuchen Sie, auf der [Microsoft-Communitywebsite zu suchen.](https://answers.microsoft.com) Wenn Sie ein Entwickler sind, der eine allgemeine Übersicht über das Active Directory-Schema sucht, lesen Sie die [Übersichtsthemen zum Active Directory-Schema.](/windows/desktop/AD/active-directory-schema) Wenn Sie nach Programmierrichtlinien zum Aktualisieren oder Ändern des Schemas suchen, finden Sie weitere Informationen zum Erweitern und Anpassen des Schemas unter Erweitern des [Schemas](/windows/desktop/AD/extending-the-schema)sowie viele der Themen in den Programmierhandbüchern [Active Directory Domain Services](/windows/desktop/AD/active-directory-domain-services) und [Active Directory Lightweight Directory Services.](/previous-versions/windows/desktop/adam/active-directory-lightweight-directory-services)

 

In jedem der Referenzthemen gibt es einen Abschnitt für jedes Betriebssystem, für das das Thema gilt. Die folgenden Betriebssysteme werden derzeit unterstützt. 

| Plattform                                                      | Name im Thema                               |
|---------------------------------------------------------------|---------------------------------------------|
| Microsoft Windows Server 2003<br/>                      | Windows Server 2003<br/>              |
| Microsoft Active Directory-Anwendungsmodus (ADAM)<br/> | Adam<br/>                             |
| Microsoft Windows Server 2003 R2<br/>                   | Windows Server 2003 R2<br/>           |
| Microsoft Windows Server 2008<br/>                      | Microsoft Windows Server 2008<br/>    |
| Microsoft Windows Server 2008 R2<br/>                   | Microsoft Windows Server 2008 R2<br/> |
| Microsoft Windows Server 2012<br/>                      | Microsoft Windows Server 2012<br/>    |



 

Wenn ein Betriebssystem nicht im Thema aufgeführt ist, wird das Thema unter diesem Betriebssystem nicht unterstützt. Wenn ein Thema beispielsweise nur Windows Server 2003 und ADAM auflistet, gilt das Thema nicht für Windows Server 2003 R2.
