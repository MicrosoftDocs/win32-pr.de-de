# <a name="acs-max-duration-per-flow-attribute"></a>ACS-Max-Duration-pro-Flow-Attribut

Die maximale Dauer (in Sekunden) eines einzelnen Flows.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-Max-Dauer pro Flow            |
| LDAP-Display-Name | acsmaxdurationperflow                |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.761               |
| System-ID-GUID    | 7F 56127e-5301-11d1-a9c5-0000 C1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
