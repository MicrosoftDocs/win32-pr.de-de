# <a name="phone-pager-other-attribute"></a>Phone-Pager-anderes Attribut

Eine Liste alternativer Pager-Nummern.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Telefon-Pager-Sonstiges                                                                |
| LDAP-Display-Name | otherPager                                                                       |
| Size              | \-                                                                               |
| Berechtigung aktualisieren  | Domänen Administrator oder Konto Besitzer.                                           |
| Aktualisierungshäufigkeit  | Wenn der Benutzerdaten Satz erstellt und die Telefonnummer geändert werden muss. |
| Attribute-Id      | 1.2.840.113556.1.2.118                                                           |
| System-ID-GUID    | f0f8ffa4-1191-11d0-a060-00aa006c33ed                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
