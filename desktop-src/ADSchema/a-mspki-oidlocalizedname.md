# <a name="ms-pki-oid-localizedname-attribute"></a>MS-PKI-OID-LocalizedName-Attribut

Enthält eine Liste mit anzeigen Amen, die zum Beschreiben einer OID nach Gebiets Schema verwendet werden.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-PKI-OID-LocalizedName                    |
| LDAP-Display-Name | mspki-oidlocalizedname                      |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänen Administrator                        |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1712                     |
| System-ID-GUID    | 7d59a816-bb05-4a72-971F-5c1331f67559        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
