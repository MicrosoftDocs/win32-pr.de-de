# <a name="domain-wide-policy-attribute"></a>Domain-Wide-Policy-Attribut

Dies gilt für die erweiterbare Benutzer Richtlinie, die auf den Clients repliziert werden soll.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Domänen weite Richtlinie                                    |
| LDAP-Display-Name | domainwidepolicy                                      |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.421                                |
| System-ID-GUID    | 80a67e29-9F 22-11D0-AFDD-00c04 C9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
