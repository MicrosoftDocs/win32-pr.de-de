# <a name="domain-wide-policy-attribute"></a>Domain-Wide-Policy-Attribut

Dies ist für die benutzererweiterbare Richtlinie, die auf die Clients repliziert werden soll.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Domänenweite Richtlinie                                    |
| Ldap-Anzeigename | domainWidePolicy                                      |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | \-                                                    |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.421                                |
| System-Id-Guid    | 80a67e29-9f22-11d0-afdd-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
