|-------------------|-----------------------------------------|
| CN                | MS-DFSR-mitgliedsverweis                 |
| LDAP-Display-Name | msdfsr-mitgliedverweisverweis                  |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.100             |
| System-ID-GUID    | 261337aa-f 1C3-44b2-bbea-c88d49e6b0c7    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
