# <a name="ms-com-defaultpartitionlink-attribute"></a>MS-com-defaultpartitionlink-Attribut

Ein Link, der verwendet wird, um die com+-Standard Partition in einer com+-partitionmenge zu identifizieren.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | MS-com-defaultpartitionlink             |
| LDAP-Display-Name | MScom-defaultpartitionlink              |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1427                 |
| System-ID-GUID    | 998b10fi7-aa1a-4364-b867-753d197fe670    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
