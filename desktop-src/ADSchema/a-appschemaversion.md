# <a name="app-schema-version-attribute"></a>App-Schema-Version-Attribut

Dieses Attribut speichert die Schema Version des Klassen Speichers. Es wird verwendet, um ein korrektes Verhalten für Schema Änderungen bereitzustellen.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | App-Schema-Version                   |
| LDAP-Display-Name | appschemaversion                     |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.848               |
| System-ID-GUID    | 96a7dd65-9118-11d1-AEbc-0000f 80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
