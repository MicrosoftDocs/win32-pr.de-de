# <a name="app-schema-version-attribute"></a>Attribut "App-Schema-Version"

Dieses Attribut speichert die Schemaversion des Klassenspeichers. Sie wird verwendet, um das richtige Verhalten für Schemaänderungen bereitzustellen.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | App-Schema-Version                   |
| Ldap-Anzeigename | appSchemaVersion                     |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.848               |
| System-ID-GUID    | 96a7dd65-9118-11d1-aebc-0000f80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
