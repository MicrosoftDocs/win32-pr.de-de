# <a name="ms-wmi-class-attribute"></a>MS-WMI-class-Attribut

Der Name eines WMI-Klassen Objekts in einer zugeordneten Codierung (z. b \_ . Win32 Computersystem). Bei der Codierung kann es sich entweder um eine Instanz oder ein Klassenobjekt handeln.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------|
| CN                | MS-WMI-Klasse                                                    |
| LDAP-Display-Name | mtaumi-Klasse                                                     |
| Size              | \-                                                              |
| Berechtigung aktualisieren  | Domänen Administrator                                            |
| Aktualisierungshäufigkeit  | Wenn eine Klasse, die das Attribut enthält, hinzugefügt oder geändert wird. |
| Attribute-Id      | 1.2.840.113556.1.4.1676                                         |
| System-ID-GUID    | 90c1925f -4a24-4b07-B202-be32eb3c8b74                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                     |
