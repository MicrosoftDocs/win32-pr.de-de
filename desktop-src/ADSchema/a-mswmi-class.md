# <a name="ms-wmi-class-attribute"></a>ms-WMI-Class-Attribut

Der Name eines WMI-Klassenobjekts in einer zugeordneten Codierung (z. B. Win32 \_ ComputerSystem). Die Codierung kann entweder eine Instanz oder ein Klassenobjekt sein.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------|
| CN                | ms-WMI-Class                                                    |
| Ldap-Anzeigename | msWMI-Class                                                     |
| Size              | \-                                                              |
| Aktualisieren von Berechtigungen  | Domänenadministrator                                            |
| Updatehäufigkeit  | Jedes Mal, wenn eine Klasse, die das Attribut enthält, hinzugefügt oder geändert wird. |
| Attribute-Id      | 1.2.840.113556.1.4.1676                                         |
| System-Id-Guid    | 90c1925f-4a24-4b07-b202-be32eb3c8b74                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                     |
