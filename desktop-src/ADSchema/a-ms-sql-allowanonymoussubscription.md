# <a name="ms-sql-allowanonymoussubscription-attribute"></a>MS-SQL-zubezeichnungs Abonnement-Attribut

True, wenn die Veröffentlichung anonymen Abonnenten das Abonnieren ermöglicht.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MS-SQL-zuverteilungsabonnement    |
| LDAP-Display-Name | MS-SQL-zuverteilungsabonnement    |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | Wenn die Replikation eingerichtet ist.           |
| Attribute-Id      | 1.2.840.113556.1.4.1394              |
| System-ID-GUID    | db77be4a-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
