# <a name="catalogs-attribute"></a>Kataloge-Attribut

Die Liste der Kataloge, die Speicher auf einem bestimmten Computer indizieren.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Kataloge                                    |
| LDAP-Display-Name | catalogs                                    |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.675                      |
| System-ID-GUID    | 7bf dcb81-4807-11d1-a9c3-0000b80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
