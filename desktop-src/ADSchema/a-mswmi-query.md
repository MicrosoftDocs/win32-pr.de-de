# <a name="ms-wmi-query-attribute"></a>MS-WMI-Query-Attribut

Eine einzelne WQL-Abfrage.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-Query                                |
| LDAP-Display-Name | mswap-Abfrage                                 |
| Size              | Weniger als 250 Zeichen.                   |
| Berechtigung aktualisieren  | Gruppenrichtlinie-Administrator                  |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1642                     |
| System-ID-GUID    | 65fff93e-35e3-45a3-85ae-876c6718297f        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
