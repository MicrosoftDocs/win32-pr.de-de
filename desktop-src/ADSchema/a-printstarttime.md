# <a name="print-start-time-attribute"></a>Print-Start-Time-Attribut

Die Zeit, zu der eine Druckwarteschlange mit der Wartung von Aufträgen beginnt.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Druckstartzeit                     |
| Ldap-Anzeigename | printStartTime                       |
| Size              | 4 Bytes                              |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.233               |
| System-ID-GUID    | 281416c9-1968-11d0-a28f-00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
