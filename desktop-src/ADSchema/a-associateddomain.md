# <a name="associateddomain-attribute"></a>associatedDomain-Attribut

Der attributtyp associatedDomain gibt eine DNS-Domäne an, die einem -Objekt zugeordnet ist.

|-------------------|--------------------------------------|
| CN                | associatedDomain                     |
| Ldap-Anzeigename | associatedDomain                     |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 0.9.2342.19200300.100.1.37           |
| System-ID-GUID    | 3320fc38-c379-4c17-a510-1bdf6133c5da |
