# <a name="associateddomain-attribute"></a>associateddomain-Attribut

Der associateddomain-Attributtyp gibt eine DNS-Domäne an, die einem Objekt zugeordnet ist.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | associateddomain                     |
| LDAP-Display-Name | associateddomain                     |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 0.9.2342.19200300.100.1.37           |
| System-ID-GUID    | 3320fc38-c379-4c17-A510-1bdf6133c5da |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
