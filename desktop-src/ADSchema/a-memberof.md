# <a name="is-member-of-dl-attribute"></a>Is-Member-of-DL-Attribut

Der Distinguished Name der Gruppen, zu denen dieses-Objekt gehört.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Is-Member-of-DL                         |
| LDAP-Display-Name | memberOf                                |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Domänen Administrator                    |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.102                  |
| System-ID-GUID    | bf967991-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
