# <a name="dhcp-servers-attribute"></a>DHCP-Server-Attribut

Enthält eine Liste von Servern, die im Unternehmen autorisiert sind.



| Eingabe | Wert |
|-------------------|--------------------------------------------------------|
| CN                | DHCP-Server                                           |
| LDAP-Display-Name | Gruppe dhcpservers gespeicherte                                            |
| Size              | \-                                                     |
| Berechtigung aktualisieren  | Domänen Administrator.                                  |
| Aktualisierungshäufigkeit  | Wenn ein neuer Server der Gesamtstruktur hinzugefügt oder daraus entfernt wird. |
| Attribute-Id      | 1.2.840.113556.1.4.704                                 |
| System-ID-GUID    | 963d2745-48be-11d1-a9c3-0000 C1                   |
| Syntax            | [**String(IA5)**](s-string-ia5.md)                    |
