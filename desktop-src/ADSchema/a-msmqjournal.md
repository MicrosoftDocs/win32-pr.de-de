# <a name="msmq-journal-attribute"></a>MSMQ-Journal-Attribut

Gibt an, ob von der Warteschlange abgerufene Nachrichten im Journal aufbewahrt werden sollen.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MSMQ-Journal                         |
| LDAP-Display-Name | msmqjournal                          |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.918               |
| System-ID-GUID    | 9a0dc321-C100-11d1-bbc5-0080c76670c0 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
