# <a name="phone-fax-other-attribute"></a>Telefon-Fax-anderes Attribut

Eine Liste alternativer Faxnummern.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Telefon-Fax-Sonstiges                                                                  |
| LDAP-Display-Name | otherfaksimiletelefonienumber                                                    |
| Size              | \-                                                                               |
| Berechtigung aktualisieren  | Domänen Administrator oder Konto Besitzer.                                           |
| Aktualisierungshäufigkeit  | Wenn der Benutzerdaten Satz erstellt und die Telefonnummer geändert werden muss. |
| Attribute-Id      | 1.2.840.113556.1.4.646                                                           |
| System-ID-GUID    | 0296c11d-40DA-11d1-a9c0-0000b80367c1                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
