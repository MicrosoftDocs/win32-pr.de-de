# <a name="ms-pki-ra-signature-attribute"></a>ms-PKI-RA-Signature-Attribut

Gibt die Anzahl der Registrierungsstelle-Signaturen an, die in einer Registrierungsanforderung erforderlich sind.



| Eingabe | Wert |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-RA-Signature                                                                               |
| Ldap-Anzeigename | msPKI-RA-Signature                                                                                |
| Size              | 4 Bytes                                                                                           |
| Aktualisieren von Berechtigungen  | Domänenadministrator                                                                              |
| Updatehäufigkeit  | Wenn das Zertifikatvorlagenobjekt (ms-PKI-Certificate-Template) bearbeitet, erstellt oder geklont wird. |
| Attribute-Id      | 1.2.840.113556.1.4.1429                                                                           |
| System-Id-Guid    | fe17e04b-937d-4f7e-8e0e-9292c8d5683e                                                              |
| Syntax            | [**Enumeration**](s-enumeration.md)                                                              |
