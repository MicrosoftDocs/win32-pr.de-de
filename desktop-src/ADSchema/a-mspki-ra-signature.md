# <a name="ms-pki-ra-signature-attribute"></a>MS-PKI-RA-Signature-Attribut

Gibt die Anzahl der Registrierungs Registrierungsstellen-Signaturen an, die bei einer Registrierungs Anforderung erforderlich sind.



| Eingabe | Wert |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-RA-Signature                                                                               |
| LDAP-Display-Name | mspki-RA-Signature                                                                                |
| Size              | 4 Bytes                                                                                           |
| Berechtigung aktualisieren  | Domänen Administrator                                                                              |
| Aktualisierungshäufigkeit  | Wenn das Objekt Vorlage (MS-PKI-Certificate-template) bearbeitet, erstellt oder geklont wird. |
| Attribute-Id      | 1.2.840.113556.1.4.1429                                                                           |
| System-ID-GUID    | fe17e04b-937d-4f7e-8e0e-9292c8d5683e                                                              |
| Syntax            | [**Enumeration**](s-enumeration.md)                                                              |
