| CN                | Ipsec-Data                                            |
| LDAP-Display-Name | ipsecdata                                             |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.623                                |
| System-ID-GUID    | b40ff81f-427a-11d1-a9c2-0000f80367c1                  |
