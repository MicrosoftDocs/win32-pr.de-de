# <a name="options-location-attribute"></a>Options-Location-Attribut

Für DHCP enthält der Optionsspeicherort den DN für alternative Standorte, die die Optionsinformationen enthalten.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Options-Location                     |
| Ldap-Anzeigename | optionsLocation                      |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.713               |
| System-Id-Guid    | 963d274e-48be-11d1-a9c3-0000f80367c1 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
