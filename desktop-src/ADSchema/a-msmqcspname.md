# <a name="msmq-csp-name-attribute"></a>MSMQ-CSP-Name-Attribut

Der Typ des Kryptografieanbieters, der von MSMQ verwendet wird.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MSMQ-CSP-Name                               |
| LDAP-Display-Name | msmqcspname                                 |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.940                      |
| System-ID-GUID    | 9a0dc334-C100-11d1-bbc5-0080c76670c0        |
| Syntax            | [**String(Teletex)**](s-string-teletex.md) |
