# <a name="ms-tpm-srk-pub-thumbprint-attribute"></a>ms-TPM-Srk-Pub-Thumbprint-Attribut

Dieses Attribut enthält den Fingerabdruck von SrkPub, der einem bestimmten TPM entspricht. Dies hilft bei der Indizierung der TPM-Geräte im Verzeichnis.

| Aktualisieren von Berechtigungen  | \-                                                    |

| Attribute-Id      | 1.2.840.113556.1.4.2107                               |
