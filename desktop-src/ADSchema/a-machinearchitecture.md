# <a name="machine-architecture-attribute"></a>Machine-Architecture-Attribut

Enthält eine Liste der Hardware Prozessoren, die von einer bestimmten Anwendung unterstützt werden.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Machine-Architecture                 |
| LDAP-Display-Name | machinearchitecture                  |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.68                |
| System-ID-GUID    | bf9679af-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
