# <a name="server-reference-attribute"></a>Server-Reference-Attribut

Gefunden in einem Standort Computer Objekt. Enthält den Distinguished Name des Domänen Controllers im Domänen namens Kontext.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Server-Reference                        |
| LDAP-Display-Name | ServerReference                         |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.        |
| Aktualisierungshäufigkeit  | Immer dann, wenn eine Website erstellt wird.             |
| Attribute-Id      | 1.2.840.113556.1.4.515                  |
| System-ID-GUID    | 26d9736d-6070-11d1-a9c6-0000b80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
