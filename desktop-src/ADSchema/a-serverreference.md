# <a name="server-reference-attribute"></a>Server-Reference-Attribut

Gefunden in einem Standortcomputerobjekt. Enthält den Distinguished Name des Domänencontrollers im Domänennamenskontext.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Server-Reference                        |
| Ldap-Anzeigename | serverReference                         |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.        |
| Updatehäufigkeit  | Immer dann, wenn ein Standort erstellt wird.             |
| Attribute-Id      | 1.2.840.113556.1.4.515                  |
| System-ID-GUID    | 26d9736d-6070-11d1-a9c6-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
