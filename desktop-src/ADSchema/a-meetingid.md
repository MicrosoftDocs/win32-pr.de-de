# <a name="meetingid-attribute"></a>MeetingID-Attribut

Die ID für die Besprechung.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------------|
| CN                | MeetingID                                                                        |
| LDAP-Display-Name | MeetingID                                                                        |
| Size              | \-                                                                               |
| Berechtigung aktualisieren  | Jeder kann dieses Objekt basierend auf der Sicherheit des Objekts, das erstellt wird, aktualisieren. |
| Aktualisierungshäufigkeit  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.565                                                           |
| System-ID-GUID    | 11b6cc7c-48C4-11d1-a9c3-0000t80367c1                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
