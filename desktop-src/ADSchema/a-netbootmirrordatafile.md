# <a name="netboot-mirror-data-file-attribute"></a>NetBoot-Mirror-Data-file-Attribut

Das **netboot-Mirror-Data-File-** Attribut ist für die interne Verwendung reserviert.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | NetBoot-Spiegelung-Datendatei                    |
| LDAP-Display-Name | netbootmirrordatafile                       |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.            |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1241                     |
| System-ID-GUID    | 2df90d85-009F -11d2-aa4c-00c04f d83a        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
