# <a name="change-schema-master-extended-right"></a>Change-Schema-Master erweitert rechts

Das erweiterte Recht ist erforderlich, um den Besitzer der FSMO-Rolle Schema Master zu ändern.



| Eingabe | Wert |
|--------------|--------------------------------------|
| CN           | Change-Schema-Master                 |
| Anzeigename | Schema Master ändern                 |
| Rights-GUID  | e12b56b6-0a95-11d1-adbb-00c04fd8d5cd |
