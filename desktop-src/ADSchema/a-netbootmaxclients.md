# <a name="netboot-max-clients-attribute"></a>NetBoot-Max-Clients-Attribut

Das **netboot-Max-Clients-** Attribut ist für die interne Verwendung reserviert.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | NetBoot-Max-Clients                  |
| LDAP-Display-Name | netbootmaxclients                    |
| Size              | \-                                   |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.851               |
| System-ID-GUID    | 07383078-91df-11d1-AEbc-0000e80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
