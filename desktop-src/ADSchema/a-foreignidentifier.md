| LDAP-Display-Name | fremd-nidentifier                                     |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.356                                |
| System-ID-GUID    | 3e97891e-8c01-11D0-AFDA-00c04f 930c9                  |
