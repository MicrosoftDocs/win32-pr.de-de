# <a name="uniqueidentifier-attribute"></a>uniqueIdentifier-Attribut

Der uniqueIdentifier-Attributtyp gibt einen "eindeutigen Bezeichner" für ein Objekt an, das im Verzeichnis dargestellt wird.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Uniqueidentifier                            |
| Ldap-Anzeigename | Uniqueidentifier                            |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | \-                                          |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.44                  |
| System-Id-Guid    | ba0184c7-38c5-4bed-a526-75421470580c        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
