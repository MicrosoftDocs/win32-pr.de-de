# <a name="frs-control-data-creation-attribute"></a>FRS-Control-Attribute für die Daten Erstellung

Warn-/fehlerebenenpaar für die Datei Daten Erstellung (Megabytes pro Sekunde).



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | FRS-Control-Daten Erstellung                   |
| LDAP-Display-Name | frscontroldatacreations                      |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.871                      |
| System-ID-GUID    | 2a13257a-9373-11d1-AEbc-0000e80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
