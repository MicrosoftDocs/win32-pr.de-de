# <a name="ms-pki-oid-attribute-attribute"></a>MS-PKI-OID-Attribute-Attribut

Das-Attribut für die Unternehmens-OID.



| Eingabe | Wert |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-OID-Attribute                                                                       |
| LDAP-Display-Name | mspki-OID-Attribute                                                                        |
| Size              | 4 Bytes                                                                                    |
| Berechtigung aktualisieren  | Unternehmensadministrator                                                                   |
| Aktualisierungshäufigkeit  | Wenn eine neue Vorlage erstellt wird oder die Attribute vorhandener Vorlagen bearbeitet werden. |
| Attribute-Id      | 1.2.840.113556.1.4.1671                                                                    |
| System-ID-GUID    | 8c9e1288-5028-4f 4B-A704-76d026l246ef                                                       |
| Syntax            | [**Enumeration**](s-enumeration.md)                                                       |
