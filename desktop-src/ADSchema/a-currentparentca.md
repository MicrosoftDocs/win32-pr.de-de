# <a name="current-parent-ca-attribute"></a>Current-Parent-ca-Attribut

Verweis auf die Zertifizierungsstellen, die die aktuellen Zertifikate für eine Zertifizierungsstelle ausgestellt haben.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Current-Parent-ca                       |
| LDAP-Display-Name | currentparameentca                         |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.696                  |
| System-ID-GUID    | 963d273b-48be-11d1-a9c3-0000b80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
