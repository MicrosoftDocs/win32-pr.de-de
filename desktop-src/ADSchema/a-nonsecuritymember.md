# <a name="non-security-member-attribute"></a>Nicht-Security-Member-Attribut

Nicht Sicherheits Mitglieder einer Gruppe. Wird für Exchange-Verteilerlisten verwendet.



| Eingabe | Wert |
|-------------------|--------------------------------------------------|
| CN                | Nicht sicherheitsmitglied                              |
| LDAP-Display-Name | nonsecuritymember                                |
| Size              | \-                                               |
| Berechtigung aktualisieren  | Domänen Administrator                             |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn ein Benutzer der DL hinzugefügt oder aus dieser entfernt wird. |
| Attribute-Id      | 1.2.840.113556.1.4.530                           |
| System-ID-GUID    | 52458018-ca6a-11D0-affinf-0000f 80367c1             |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)          |
