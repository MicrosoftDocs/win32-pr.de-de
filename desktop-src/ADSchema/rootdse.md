# <a name="rootdse-ad-schema"></a>RootDSE (AD-Schema)

In LDAP 3.0 ist rootDSE als Stamm der Verzeichnisdatenstruktur auf einem Verzeichnisserver definiert. RootDSE ist nicht Teil eines Namespace. Der Zweck von rootDSE besteht in der Bereitstellung von Daten über den Verzeichnisserver. Weitere Informationen zu rootDSE finden Sie unter [Serverlose Bindung und RootDSE](/windows/desktop/AD/serverless-binding-and-rootdse) in der Active Directory SDK-Dokumentation.

rootDSE enthält die folgenden Attribute. Alle Attribute sind einwertige Attribute, sofern nicht anders angegeben.



| attribute                                    | Syntax                                                 | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                            |
|----------------------------------------------|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **configurationNamingContext**<br/>    | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Distinguished Name für den Konfigurationscontainer.<br/>                                                                                                                                                                                                                                                                                                                                            |
| **Currenttime**<br/>                   | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den aktuellen Zeitsatz auf diesem Verzeichnisserver im koordinierte Weltzeit Format.<br/>                                                                                                                                                                                                                                                                                                                |
| **defaultNamingContext**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Distinguished Name für die Domäne, der dieser Verzeichnisserver mitglied ist.<br/>                                                                                                                                                                                                                                                                                                                  |
| **Dnshostname**<br/>                   | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält die DNS-Adresse für diesen Verzeichnisserver.<br/>                                                                                                                                                                                                                                                                                                                                                         |
| **domainControllerFunctionality**<br/> | [**String(Teletex)**](s-string-teletex.md)<br/> | Gibt die Funktionsebene des Domänencontrollers an. Dies kann einer der folgenden Werte sein.<br/> "0" – Windows 2000-Modus<br/> "2" – Windows Server 2003-Modus<br/> "3" – Windows Server 2008-Modus<br/>                                                                                                                                                                                    |
| **domainFunctionality**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Gibt die Funktionsebene der Domäne an. Dies kann einer der folgenden Werte sein.<br/> "0" – Windows 2000-Domänenmodus<br/> "1" – Windows Server 2003-Zwischendomänenmodus<br/> "2" – Windows Server 2003-Domänenmodus<br/> "3" – Windows Server 2008-Domänenmodus<br/> "4" – Windows Server 2008 R2-Domänenmodus<br/>                                                             |
| **dsServiceName**<br/>                 | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Distinguished Name des NTDS-Einstellungsobjekts für diesen Verzeichnisserver.<br/>                                                                                                                                                                                                                                                                                                                      |
| **forestFunctionality**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Gibt die Funktionsebene der Gesamtstruktur an. Dies kann einer der folgenden Werte sein.<br/> "0" – Windows Gesamtstrukturmodus 2000<br/> "1" – Windows Server 2003– Zwischenstrukturmodus<br/> "2": Windows Server 2003-Gesamtstrukturmodus<br/> "3" – Windows Server 2008-Gesamtstrukturmodus<br/> "4" – Windows Server 2008 R2-Gesamtstrukturmodus<br/>                                                             |
| **highestCommittedUSN**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält die höchste Updatesequenznummer (USN) auf diesem Verzeichnisserver. Wird von der Verzeichnisreplikation verwendet.<br/>                                                                                                                                                                                                                                                                                                  |
| **isGlobalCatalogReady**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Gibt an, ob der globale Katalog voll funktionsfähig ist. Enthält entweder "TRUE" oder "FALSE".<br/>                                                                                                                                                                                                                                                                                                                    |
| **isSynchronized**<br/>                | [**String(Teletex)**](s-string-teletex.md)<br/> | Gibt an, ob der Verzeichnisserver vollständig synchronisiert ist. Enthält entweder "TRUE" oder "FALSE".<br/>                                                                                                                                                                                                                                                                                                                 |
| **ldapServiceName**<br/>               | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Dienstprinzipalnamen (Service Principal Name, SPN) für den LDAP-Server. Wird für die gegenseitige Authentifizierung verwendet.<br/>                                                                                                                                                                                                                                                                                                              |
| **namingContexts**<br/>                | [**String(Teletex)**](s-string-teletex.md)<br/> | Ein mehrwertiges Attribut, das die Distinguished Names für alle Aufnamenskontexte enthält, die auf diesem Verzeichnisserver gespeichert sind. Standardmäßig enthält ein Windows 2000-Domänencontroller mindestens drei Namenskontexte: Schema, Konfiguration und einen für die Domäne, in der der Server Mitglied ist.<br/>                                                                                                             |
| **rootDomainNamingContext**<br/>       | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Distinguished Name für die erste Domäne in der Gesamtstruktur, die die Domäne enthält, in der dieser Verzeichnisserver Mitglied ist.<br/>                                                                                                                                                                                                                                                                     |
| **schemaNamingContext**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Distinguished Name für den Schemacontainer.<br/>                                                                                                                                                                                                                                                                                                                                                   |
| **serverName**<br/>                    | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Distinguished Name für das Serverobjekt für diesen Verzeichnisserver im Konfigurationscontainer.<br/>                                                                                                                                                                                                                                                                                             |
| **subschemaSubentry**<br/>             | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Distinguished Name für das [**subSchema-Objekt.**](c-subschema.md) Das **subSchema-Objekt** enthält Eigenschaften, die die unterstützten Attribute (in der [**attributeTypes-Eigenschaft)**](a-attributetypes.md) und Klassen (in der [**objectClasses-Eigenschaft) verfügbar**](a-objectclasses.md) machen.<br/> Die **subschemaSubentry-Eigenschaft** und das Unterschema sind in LDAP 3.0 definiert (siehe RFC 2251).<br/> |
| **supportedCapabilities**<br/>         | [**String(Teletex)**](s-string-teletex.md)<br/> | Ein Mehrwertattribut, das die von diesem Verzeichnisserver unterstützten Funktionen enthält.<br/>                                                                                                                                                                                                                                                                                                              |
| **supportedControl**<br/>              | [**String(Teletex)**](s-string-teletex.md)<br/> | Ein mehrwertige Attribut, das die OIDs für Erweiterungssteuerelemente enthält, die von diesem Verzeichnisserver unterstützt werden. Eine Liste der möglichen Kontroll-OIDs finden Sie in der folgenden Tabelle.<br/>                                                                                                                                                                                                                                  |
| **supportedLDAPPolicies**<br/>         | [**String(Teletex)**](s-string-teletex.md)<br/> | Ein mehrwertiges Attribut, das die Namen der unterstützten LDAP-Verwaltungsrichtlinien enthält.<br/>                                                                                                                                                                                                                                                                                                              |
| **supportedLDAPVersion**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Ein mehrwertiges Attribut, das die LDAP-Versionen enthält (angegeben durch die Hauptversionsnummer), die von diesem Verzeichnisserver unterstützt werden.<br/>                                                                                                                                                                                                                                                                         |
| **supportedSASLMechanisms**<br/>       | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält die Sicherheitsmechanismen, die für die SASL-Aushandlung unterstützt werden (siehe LDAP-RFCs). Standardmäßig wird GSSAPI unterstützt.<br/>                                                                                                                                                                                                                                                                                           |
