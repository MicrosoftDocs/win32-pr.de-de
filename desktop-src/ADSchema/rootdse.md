# <a name="rootdse-ad-schema"></a>RootDSE (AD-Schema)

In LDAP 3,0 ist RootDSE als Stammverzeichnis der Verzeichnis Datenstruktur auf einem Verzeichnisserver definiert. RootDSE ist nicht Teil eines Namespace. Der Zweck von RootDSE besteht darin, Daten zum Verzeichnisserver bereitzustellen. Weitere Informationen zu RootDSE finden Sie unter [Server lose Bindung und RootDSE](/windows/desktop/AD/serverless-binding-and-rootdse) in der Active Directory SDK-Dokumentation.

RootDSE enthält die folgenden Attribute. Alle Attribute sind einwertig, sofern nicht anders angegeben.



| Attribut                                    | Syntax                                                 | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                            |
|----------------------------------------------|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **configurationNamingContext**<br/>    | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Distinguished Name für den Konfigurations Container.<br/>                                                                                                                                                                                                                                                                                                                                            |
| **CurrentTime**<br/>                   | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält die aktuelle Zeit, die auf diesem Verzeichnisserver im koordinierte Weltzeit festgelegt ist.<br/>                                                                                                                                                                                                                                                                                                                |
| **defaultNamingContext**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Distinguished Name für die Domäne, deren Mitglied dieser Verzeichnisserver ist.<br/>                                                                                                                                                                                                                                                                                                                  |
| **DNSHostName**<br/>                   | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält die DNS-Adresse für diesen Verzeichnisserver.<br/>                                                                                                                                                                                                                                                                                                                                                         |
| **domaincontrollerfunktionsfunktion**<br/> | [**String(Teletex)**](s-string-teletex.md)<br/> | Gibt die Funktionsebene des Domänen Controllers an. Dies kann einer der folgenden Werte sein:<br/> "0"-Windows 2000-Modus<br/> "2"-Windows Server 2003-Modus<br/> "3"-Windows Server 2008-Modus<br/>                                                                                                                                                                                    |
| **domainfunktionalität**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Gibt die Funktionsebene der Domäne an. Dies kann einer der folgenden Werte sein:<br/> "0"-Windows 2000-Domänen Modus<br/> "1"-Windows Server 2003-zwischen Domänen Modus<br/> "2"-Windows Server 2003-Domänen Modus<br/> "3"-Windows Server 2008-Domänen Modus<br/> "4"-Windows Server 2008 R2-Domänen Modus<br/>                                                             |
| **dsservicename**<br/>                 | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Distinguished Name des NTDS-Einstellungs Objekts für diesen Verzeichnisserver.<br/>                                                                                                                                                                                                                                                                                                                      |
| **Forest-Funktionalität**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Gibt die Funktionsebene der Gesamtstruktur an. Dies kann einer der folgenden Werte sein:<br/> "0"-Windows 2000-Gesamtstruktur Modus<br/> "1"-Windows Server 2003-zwischen Gesamtstruktur Modus<br/> "2"-Windows Server 2003-Gesamtstruktur Modus<br/> "3"-Windows Server 2008-Gesamtstruktur Modus<br/> "4"-Windows Server 2008 R2-Gesamtstruktur Modus<br/>                                                             |
| **highestCommittedUSN**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält die höchste Aktualisierungs Sequenznummer (Update Sequence Number, US) auf diesem Verzeichnisserver. Wird von der Verzeichnisreplikation verwendet.<br/>                                                                                                                                                                                                                                                                                                  |
| **isGlobalCatalogReady**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Gibt an, ob der globale Katalog voll funktionsfähig ist. Enthält entweder "true" oder "false".<br/>                                                                                                                                                                                                                                                                                                                    |
| **IsSynchronized**<br/>                | [**String(Teletex)**](s-string-teletex.md)<br/> | Gibt an, ob der Verzeichnisserver vollständig synchronisiert ist. Enthält entweder "true" oder "false".<br/>                                                                                                                                                                                                                                                                                                                 |
| **ldapservicename**<br/>               | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Dienst Prinzipal Namen (SPN) für den LDAP-Server. Wird für die gegenseitige Authentifizierung verwendet.<br/>                                                                                                                                                                                                                                                                                                              |
| **namingContexts**<br/>                | [**String(Teletex)**](s-string-teletex.md)<br/> | Ein Attribut mit mehreren Werten, das die Distinguished Names für alle auf diesem Verzeichnisserver gespeicherten Namenskontexte enthält. Standardmäßig enthält ein Windows 2000-Domänen Controller mindestens drei Namenskontexte: Schema, Konfiguration und einen für die Domäne, der der Server angehört.<br/>                                                                                                             |
| **rootDomainNamingContext**<br/>       | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Distinguished Name für die erste Domäne in der Gesamtstruktur, die die Domäne enthält, von der dieser Verzeichnisserver Mitglied ist.<br/>                                                                                                                                                                                                                                                                     |
| **schemaNamingContext**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Distinguished Name für den Schema Container.<br/>                                                                                                                                                                                                                                                                                                                                                   |
| **serverName**<br/>                    | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Distinguished Name für das Server Objekt für diesen Verzeichnisserver im Konfigurations Container.<br/>                                                                                                                                                                                                                                                                                             |
| **subschemasubentry**<br/>             | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält den Distinguished Name für das [**unter Schema**](c-subschema.md) Objekt. Das **subschema** -Objekt enthält Eigenschaften, die die unterstützten Attribute (in der [**attributetypes**](a-attributetypes.md) -Eigenschaft) und-Klassen (in der [**ObjectClasses**](a-objectclasses.md) -Eigenschaft) verfügbar machen.<br/> Die **subschema asubentry** -Eigenschaft und das unter Schema sind in LDAP 3,0 definiert (siehe RFC 2251).<br/> |
| **supportedfunktionen**<br/>         | [**String(Teletex)**](s-string-teletex.md)<br/> | Ein Attribut mit mehreren Werten, das die von diesem Verzeichnisserver unterstützten Funktionen enthält.<br/>                                                                                                                                                                                                                                                                                                              |
| **supportedcontrol**<br/>              | [**String(Teletex)**](s-string-teletex.md)<br/> | Ein Attribut mit mehreren Werten, das die OIDs für Erweiterungs Steuerelemente enthält, die von diesem Verzeichnisserver unterstützt werden. Eine Liste der möglichen OID-OIDs finden Sie in der folgenden Tabelle.<br/>                                                                                                                                                                                                                                  |
| **supportedldappolicies**<br/>         | [**String(Teletex)**](s-string-teletex.md)<br/> | Ein Attribut mit mehreren Werten, das die Namen der unterstützten LDAP-Verwaltungsrichtlinien enthält.<br/>                                                                                                                                                                                                                                                                                                              |
| **supportedldapversion**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Ein Attribut mit mehreren Werten, das die von diesem Verzeichnisserver unterstützten LDAP-Versionen (angegeben durch die Hauptversionsnummer) enthält.<br/>                                                                                                                                                                                                                                                                         |
| **supportedsaslmechanisms**<br/>       | [**String(Teletex)**](s-string-teletex.md)<br/> | Enthält die Sicherheitsmechanismen, die für die SASL-Aushandlung unterstützt werden (siehe LDAP-RFCs). Standardmäßig wird GSSAPI unterstützt.<br/>                                                                                                                                                                                                                                                                                           |
