# <a name="site-server-attribute"></a>Site-Server-Attribut

Lizenzierung des Hauptservers für einen bestimmten Standort.



| Eingabe | Wert |
|-------------------|----------------------------------------------|
| CN                | Site-Server                                  |
| Ldap-Anzeigename | siteServer                                   |
| Size              | \-                                           |
| Aktualisieren von Berechtigungen  | Domänenadministrator                         |
| Updatehäufigkeit  | Immer dann, wenn sich der Lizenzierungsstandort ändern muss. |
| Attribute-Id      | 1.2.840.113556.1.4.494                       |
| System-ID-GUID    | 1be8f17c-a9ff-11d0-afe2-00c04fd930c9         |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)      |
