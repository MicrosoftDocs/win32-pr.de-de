# <a name="site-server-attribute"></a>Site-Server-Attribut

Lizenzierungs Hauptserver für einen bestimmten Standort.



| Eingabe | Wert |
|-------------------|----------------------------------------------|
| CN                | Site-Server                                  |
| LDAP-Display-Name | Standort Server                                   |
| Size              | \-                                           |
| Berechtigung aktualisieren  | Domänen Administrator                         |
| Aktualisierungshäufigkeit  | Immer, wenn die Lizenzierungs Website geändert werden muss. |
| Attribute-Id      | 1.2.840.113556.1.4.494                       |
| System-ID-GUID    | 1be8f17c-a9ff-11D0-afe2-00c04fd930c9         |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)      |
