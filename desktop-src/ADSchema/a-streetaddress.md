| CN                | Adresse                                     |
| LDAP-Display-Name | streetAddress                               |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänen Administrator                        |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.256                      |
| System-ID-GUID    | f0f8ff84-1191-11d0-a060-00aa006c33ed        |
