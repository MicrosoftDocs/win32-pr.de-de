# <a name="priority-attribute"></a>Priority-Attribut

Die aktuelle Priorität (eines Prozesses, eines Druckauftrags usw.).



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Priorität                             |
| LDAP-Display-Name | priority                             |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.231               |
| System-ID-GUID    | 281416c7-1968-11D0-a28f -00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
