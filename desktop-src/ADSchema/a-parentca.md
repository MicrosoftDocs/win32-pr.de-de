# <a name="parent-ca-attribute"></a>Übergeordnetes ZS-Attribut

Der Distinguished Name eines Zertifizierungsstellen Objekts (ca) für eine übergeordnete Zertifizierungsstelle.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Übergeordnete Zertifizierungsstelle                               |
| LDAP-Display-Name | parametrica                                |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.557                  |
| System-ID-GUID    | 5245801b-ca6a-11D0-AFFF -0000f 80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
