# <a name="rid-used-pool-attribute"></a>Rid-used-Pool-Attribut

Die RID-Pools, die von einem DC verwendet wurden.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Rid-used-Pool                        |
| LDAP-Display-Name | ridusedpool                          |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn ein RID-Pool geleert wird.      |
| Attribute-Id      | 1.2.840.113556.1.4.373               |
| System-ID-GUID    | 6617188b-8B f -11D0-AFDA-00c04f 930c9 |
| Syntax            | [**Intervall**](s-interval.md)       |
