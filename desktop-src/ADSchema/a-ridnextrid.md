# <a name="rid-next-rid-attribute"></a>Rid-Next-Rid-Attribut

Der nächste freie relative Bezeichner im aktuellen Pool.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Rid-Next-Rid                         |
| LDAP-Display-Name | ridnextrd                           |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.374               |
| System-ID-GUID    | 6617188c-8F c04afda-00c04f 930c9 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
