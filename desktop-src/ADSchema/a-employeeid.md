# <a name="employee-id-attribute"></a>Employee-ID-Attribut

Die ID eines Mitarbeiters.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Mitarbeiter-ID                                 |
| LDAP-Display-Name | employeeID                                  |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänen Administrator                        |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.35                       |
| System-ID-GUID    | bf967962-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
