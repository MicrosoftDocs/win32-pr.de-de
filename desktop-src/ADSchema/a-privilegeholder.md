# <a name="privilege-holder-attribute"></a>Privilege-Holder-Attribut

Liste der Distinguished Names von Prinzipale, denen diese Berechtigung gewährt wurde.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Privilege-Holder                        |
| LDAP-Display-Name | privilegigehalter                         |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.637                  |
| System-ID-GUID    | 19405b9b-3cfa-11d1-a9c0-0000f 80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
