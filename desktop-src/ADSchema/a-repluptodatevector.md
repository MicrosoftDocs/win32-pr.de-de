# <a name="repl-uptodate-vector-attribute"></a>Repl-UpToDate-Vector-Attribut

Verfolgt interne Replikationsstatusinformationen für einen gesamten NC nach. Informationen können hier in öffentlicher Form über die API DsReplicaGetInfo() extrahiert werden. Auf allen NC-Stammobjekten vorhanden.



| Eingabe | Wert |
|-------------------|--------------------------------------------------------------------------------|
| CN                | Repl-UpToDate-Vector                                                           |
| Ldap-Anzeigename | replUpToDateVector                                                             |
| Size              | Die Länge ist proportional zur Anzahl der Replikate (früher und vorhanden) des NC. |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.                                               |
| Updatehäufigkeit  | Änderungen als Reaktion auf abgeschlossene eingehende Replikationszyklen.                   |
| Attribute-Id      | 1.2.840.113556.1.4.4                                                           |
| System-ID-GUID    | bf967a16-0de6-11d0-a285-00aa003049e2                                           |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                          |
