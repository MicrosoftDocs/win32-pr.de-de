# <a name="repl-uptodate-vector-attribute"></a>REPL-upydate-Vector-Attribut

Verfolgt interne Replikations Zustandsinformationen für einen gesamten NC. Informationen können hier über die API dsreplicagetinfo () im öffentlichen Format extrahiert werden. In allen NC-Stamm Objekten vorhanden.



| Eingabe | Wert |
|-------------------|--------------------------------------------------------------------------------|
| CN                | REPL-Update-Vector                                                           |
| LDAP-Display-Name | befindet                                                             |
| Size              | Die Länge ist proportional zur Anzahl der Replikate ("Past" und "Present") des NC. |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                                               |
| Aktualisierungshäufigkeit  | Änderungen in Reaktion auf abgeschlossene eingehende Replikations Zyklen.                   |
| Attribute-Id      | 1.2.840.113556.1.4.4                                                           |
| System-ID-GUID    | bf967a16-0de6-11d0-a285-00aa003049e2                                           |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                          |
