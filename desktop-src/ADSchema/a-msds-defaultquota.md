# <a name="ms-ds-default-quota-attribute"></a>ms-DS-Default-Quota-Attribut

Das Standardkontingent, das für einen Sicherheitsprinzipal gilt, der ein Objekt im NC erstellt, wenn keine Kontingentspezifikation vorhanden ist, die den Sicherheitsprinzipal abdeckt.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ms-DS-Default-Quota                  |
| Ldap-Anzeigename | msDS-DefaultQuota                    |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1846              |
| System-Id-Guid    | 6818f726-674b-441b-8a3a-f40596374cea |
| Syntax            | [**Enumeration**](s-enumeration.md) |
