# <a name="ms-ds-default-quota-attribute"></a>ms-DS-default-Quota-Attribut

Das Standard Kontingent, das auf einen Sicherheits Prinzipal angewendet wird, der ein Objekt im NC erstellt, wenn keine Kontingent Spezifikation vorhanden ist, die den Sicherheits Prinzipal abdeckt.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ms-DS-default-Quota                  |
| LDAP-Display-Name | MSDS-defaultquota                    |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1846              |
| System-ID-GUID    | 6818f 726-674b-441b-8a3a-f 40596374cea |
| Syntax            | [**Enumeration**](s-enumeration.md) |
