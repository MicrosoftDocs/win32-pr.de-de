# <a name="frs-staging-path-attribute"></a>FRS-Staging-Pfad-Attribut

Der Pfad zum Stagingbereich der Datei Replikation.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | FRS-Staging-Pfad                            |
| LDAP-Display-Name | frsstagingpath                              |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.488                      |
| System-ID-GUID    | 1be8f175-a9ff-11D0-afe2-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
