# <a name="locality-name-attribute"></a>Locality-Name-Attribut

Stellt den Namen einer Lokalität dar, z. b. eine Stadt oder ein Ort.



| Eingabe | Wert |
|-------------------|------------------------------------------------------------------------------|
| CN                | Locality-Name                                                                |
| LDAP-Display-Name | l                                                                            |
| Size              | \-                                                                           |
| Berechtigung aktualisieren  | Domänen Administrator oder Konto Besitzer.                                       |
| Aktualisierungshäufigkeit  | Wenn der Benutzerdaten Satz erstellt wird, und wenn der Ort geändert werden muss. |
| Attribute-Id      | 2.5.4.7                                                                      |
| System-ID-GUID    | bf9679a2-0de6-11d0-a285-00aa003049e2                                         |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                  |
