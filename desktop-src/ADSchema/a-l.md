# <a name="locality-name-attribute"></a>Locality-Name-Attribut

Stellt den Namen eines Orts dar, z. B. eine Stadt oder einen Ort.



| Eingabe | Wert |
|-------------------|------------------------------------------------------------------------------|
| CN                | Locality-Name                                                                |
| Ldap-Anzeigename | l                                                                            |
| Size              | \-                                                                           |
| Aktualisieren von Berechtigungen  | Domänenadministrator oder Kontobesitzer.                                       |
| Updatehäufigkeit  | Wenn der Datensatz des Benutzers erstellt wird und wann immer die Lokalität geändert werden muss. |
| Attribute-Id      | 2.5.4.7                                                                      |
| System-ID-GUID    | bf9679a2-0de6-11d0-a285-00aa003049e2                                         |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                  |
