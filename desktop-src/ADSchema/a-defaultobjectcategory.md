# <a name="default-object-category-attribute"></a>Default-Object-Category-Attribut

Die für das Objekt zu verwendende Objekt Kategorie, wenn kein Wert angegeben ist.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Default-Object-Category                 |
| LDAP-Display-Name | defaultObjectCategory                   |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Schema Administrator                    |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.783                  |
| System-ID-GUID    | 26d97367-6070-11d1-a9c6-0000b80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
