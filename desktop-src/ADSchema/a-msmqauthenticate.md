# <a name="msmq-authenticate-attribute"></a>MSMQ-Authenticate-Attribut

Gibt an, ob die Warteschlange nur authentifizierte Nachrichten akzeptiert.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MSMQ-Authenticate                    |
| LDAP-Display-Name | msmqauthenticate                     |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.923               |
| System-ID-GUID    | 9a0dc326-C100-11d1-bbc5-0080c76670c0 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
