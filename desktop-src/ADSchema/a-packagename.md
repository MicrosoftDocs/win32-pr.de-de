# <a name="package-name-attribute"></a>Package-Name-Attribut

Dieses Attribut enthält den Bereitstellungs Namen für eine Anwendung.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Package-Name                                |
| LDAP-Display-Name | packageName                                 |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.326                      |
| System-ID-GUID    | 7d6c0e98-7E20-11D0-afd6-00c04f d930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
