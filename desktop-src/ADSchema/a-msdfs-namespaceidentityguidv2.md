# <a name="ms-dfs-namespace-identity-guid-v2-attribute"></a>MS-DFS-Namespace-Identity-GUID-v2-Attribut

Soll nur festgelegt werden, wenn der Namespace erstellt wird. Über umbenennen oder verschieben, solange der Namespace nicht durch einen anderen Namespace mit demselben Namen ersetzt wird.

| CN                | MS-DFS-Namespace-Identity-GUID-v2                     |
| LDAP-Display-Name | msdfs-NamespaceIdentityGUIDv2                         |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.2033                               |
| System-ID-GUID    | 200432ce-ec5f-4931-A525-d7f4afe34e68                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
