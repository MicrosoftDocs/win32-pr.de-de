# <a name="acs-non-reserved-max-sdu-size-attribute"></a>ACS-Non-reserved-Max-SDU-size-Attribut

Das Attribut " **ACS-Non-reserved-Max-SDU-size** " ist nur für die interne Verwendung vorgesehen. Basierend auf RFC2814.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-Non-reserved-Max-SDU-size        |
| LDAP-Display-Name | acsnonreservedmaxsdusize             |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1320              |
| System-ID-GUID    | aec2cfe3-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**Intervall**](s-interval.md)       |
