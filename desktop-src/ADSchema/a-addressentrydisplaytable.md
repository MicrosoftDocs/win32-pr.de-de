| CN                | Address-Entry-Display-Table                           |
| Ldap-Anzeigename | addressEntryDisplayTable                              |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | \-                                                    |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.324                                |
| System-Id-Guid    | 5fd42461-1262-11d0-a060-00aa006c33ed                  |
