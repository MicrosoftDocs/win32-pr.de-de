# <a name="address-entry-display-table-attribute"></a>Address-Entry-Display-Table-Attribut

Die Anzeige Tabelle für einen Adresseintrag.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Address-Entry-Display-Table                           |
| LDAP-Display-Name | addressentrydisplaytable                              |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.324                                |
| System-ID-GUID    | 5F d42461-1262-11D0-a060-00aa006c33ed                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
