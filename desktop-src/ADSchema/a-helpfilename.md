# <a name="help-file-name-attribute"></a>Help-File Name-Attribut

Dieses Attribut wurde für Exchange 4,0 verwendet. Es enthält den Namen, der für die Datei verwendet werden sollte, wenn der Anbieter Hilfe Daten auf einen Client Computer heruntergeladen hat. Er wird nicht für andere Versionen von Exchange verwendet.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Hilfe-Dateiname                              |
| LDAP-Display-Name | helpfilename                                |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Diese wird vom System verwendet.                 |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.327                      |
| System-ID-GUID    | 5F d424a9-1262-11D0-a060-00aa006c33ed        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
