# <a name="enabled-attribute"></a>Enabled-Attribut

Dieses Attribut wird verwendet, um zu bestimmen, ob eine bestimmte crossRef aktiviert ist.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Aktiviert                              |
| Ldap-Anzeigename | Aktiviert                              |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.     |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.557               |
| System-Id-Guid    | a8df73f2-c5ea-11d1-lautb-0080c76670c0 |
| Syntax            | [**Boolean**](s-boolean.md)         |
