# <a name="enabled-attribute"></a>Aktiviertes Attribut

Dieses Attribut wird verwendet, um anzugeben, ob ein bestimmtes CrossRef aktiviert ist.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Aktiviert                              |
| LDAP-Display-Name | Aktiviert                              |
| Size              | \-                                   |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.557               |
| System-ID-GUID    | a8df73f2-c5ea-11d1-bbcb-0080c76670c0 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
