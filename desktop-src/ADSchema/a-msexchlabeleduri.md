# <a name="ms-exch-labeleduri-attribute"></a>ms-Exch-LabeledURI-Attribut

Ein Exchange-URI, auf den eine Bezeichnung folgt. Die Bezeichnung wird verwendet, um die Ressource zu beschreiben, auf die der URI verweist und die als lesbarer Name gedacht ist.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-Exch-LabeledURI                          |
| LDAP-Display-Name | msExchLabeledURI                            |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.593                      |
| System-ID-GUID    | 16775820-47F 3-11d1-a9c3-0000 C1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
