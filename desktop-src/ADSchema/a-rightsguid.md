# <a name="rights-guid-attribute"></a>Rights-Guid-Attribut

Die GUID, die verwendet wird, um ein erweitertes Recht innerhalb eines Zugriffssteuerungseintrags darzustellen.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Rights-Guid                                 |
| Ldap-Anzeigename | rightsGuid                                  |
| Size              | 16 Bytes                                    |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.            |
| Updatehäufigkeit  | Wenn ein neues erweitertes Recht erstellt wird.       |
| Attribute-Id      | 1.2.840.113556.1.4.340                      |
| System-ID-GUID    | 8297931c-86d3-11d0-afda-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
