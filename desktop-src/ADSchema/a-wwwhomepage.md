# <a name="www-home-page-attribute"></a>WWW-Home-Page-Attribut

Eine Webseite, die die primäre Landing Page einer Website ist.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------------------|
| CN                | WWW-Startseite                                                               |
| Ldap-Anzeigename | wWWHomePage                                                                 |
| Size              | \-                                                                          |
| Aktualisieren von Berechtigungen  | Domänenadministrator oder Kontobesitzer.                                      |
| Updatehäufigkeit  | Wenn der Datensatz des Benutzers erstellt wird und wenn die Webseite geändert werden muss. |
| Attribute-Id      | 1.2.840.113556.1.2.464                                                      |
| System-ID-GUID    | bf967a7a-0de6-11d0-a285-00aa003049e2                                        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                 |
