# <a name="www-home-page-attribute"></a>WWW-Home-Page-Attribut

Eine Webseite, bei der es sich um die primäre Landing Page einer Website handelt.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------------------|
| CN                | WWW-Homepage                                                               |
| LDAP-Display-Name | wWWHomePage                                                                 |
| Size              | \-                                                                          |
| Berechtigung aktualisieren  | Domänen Administrator oder Konto Besitzer.                                      |
| Aktualisierungshäufigkeit  | Wenn der Benutzerdaten Satz erstellt wird, und wenn die Webseite geändert werden muss. |
| Attribute-Id      | 1.2.840.113556.1.2.464                                                      |
| System-ID-GUID    | bf967a7a-0de6-11d0-a285-00aa003049e2                                        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                 |
