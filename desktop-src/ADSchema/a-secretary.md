# <a name="secretary-attribute"></a>Secretary-Attribut

Enthält den Distinguished Name des Sekretärs für ein Konto.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | secretary                               |
| LDAP-Display-Name | secretary                               |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 0.9.2342.19200300.100.1.21              |
| System-ID-GUID    | 01072d9a-98ad-4a53-9744-e83e287278fb    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
