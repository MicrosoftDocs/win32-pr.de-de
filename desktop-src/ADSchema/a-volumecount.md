# <a name="volume-count-attribute"></a>Volume-Count-Attribut

Das nach verfolgte Volumen Kontingent für einen bestimmten Computer.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Volume-Count                         |
| LDAP-Display-Name | volumecount                          |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.507               |
| System-ID-GUID    | 34aaa217-b699-11D0-Afee-0000b C1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
