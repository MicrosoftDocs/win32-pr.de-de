# <a name="volume-count-attribute"></a>Volume-Count-Attribut

Das nachverfolgte Volumekontingent für einen bestimmten Computer.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Volume-Count                         |
| Ldap-Anzeigename | volumeCount                          |
| Size              | 4 Bytes                              |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.507               |
| System-Id-Guid    | 34aaa217-b699-11d0-afee-0000f80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
