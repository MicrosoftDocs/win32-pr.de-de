# <a name="acs-dsbm-deadtime-attribute"></a>ACS-DSBM-Deadtime-Attribut

Dieses Attribut enthält das Wahlzeit Intervall (dsbmdeadinterval) für eine Domäne. Wenn das designierte Subnetz-Bandbreiten-Manager \_ \_ während dieses Intervalls keine I am DSBM-Ankündigung sendet, wählt die anderen subnetzbandbreiten-Manager (sbms) in der Domäne einen neuen DSBM aus.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-DSBM-Deadtime                    |
| LDAP-Display-Name | acsdsbmdeadtime                      |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.778               |
| System-ID-GUID    | 1cb355a0-56d0-11d1-a9c6-0000b80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
