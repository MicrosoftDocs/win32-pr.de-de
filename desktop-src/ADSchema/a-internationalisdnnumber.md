# <a name="international-isdn-number-attribute"></a>International-ISDN-Number-Attribut

Gibt eine internationale ISDN-Nummer an, die einem Objekt zugeordnet ist.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | International-ISDN-Number                   |
| LDAP-Display-Name | internationalisdnnumber                     |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 2.5.4.25                                    |
| System-ID-GUID    | bf96798d-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Numeric)**](s-string-numeric.md) |
