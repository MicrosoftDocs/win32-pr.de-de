# <a name="bridgehead-transport-list-attribute"></a>Bridgehead-Transport-List-Attribut

Transporte, für die dieser Server ein Bridgehead ist.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Bridgehead-Transport-List               |
| Ldap-Anzeigename | bridgeheadTransportList                 |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.        |
| Updatehäufigkeit  | Immer dann, wenn ein Standort eingerichtet wird.               |
| Attribute-Id      | 1.2.840.113556.1.4.819                  |
| System-ID-GUID    | d50c2cda-8951-11d1-aebc-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
