# <a name="account-name-history-attribute"></a>Account-Name-History-Attribut

Die Zeitspanne, für die das Konto aktiv war.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Konto Name-Verlauf                        |
| LDAP-Display-Name | accountnamehistory                          |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänen Administrator                        |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1307                     |
| System-ID-GUID    | 031952ec-3b72-11d2-90cc-00c04f d91ab1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
