# <a name="ms-wmi-intflags1-attribute"></a>MS-WMI-intFlags1-Attribut

Das **MS-WMI-intFlags1-** Attribut ist für die interne Verwendung reserviert.

|-------------------|--------------------------------------|
| CN                | MS-WMI-intFlags1                     |
| LDAP-Display-Name | mswap-intFlags1                      |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1678              |
| System-ID-GUID    | 18e006b9-6445-48e3-9dcf-b5ecfbc4df8e |
