# <a name="print-keep-printed-jobs-attribute"></a>Print-Keep-Print-Jobs-Attribut

| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Print-Keep-Print-Jobs              |
| LDAP-Display-Name | printKeepPrintedJobs                 |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.275               |
| System-ID-GUID    | ba305f6d-47e3-11d0-a1a6-00c04fd930c9 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
