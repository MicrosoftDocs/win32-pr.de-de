# <a name="ms-ds-creator-sid-attribute"></a>MS-DS-Creator-SID-Attribut

Die Sicherheits-ID des Erstellers des Objekts, das dieses Attribut enthält.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MS-DS-Creator-sid                    |
| LDAP-Display-Name | mS-DS-kreatorsid                     |
| Size              | \-                                   |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn ein neues-Objekt erstellt wird.    |
| Attribute-Id      | 1.2.840.113556.1.4.1410              |
| System-ID-GUID    | c5e60132-1480-11d3-91c1-0000f87a57d4 |
| Syntax            | [**Zeichenfolge (SID)**](s-string-sid.md)  |
