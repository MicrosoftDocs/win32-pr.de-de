# <a name="anr-attribute"></a>ANR-Attribut

Mehrdeutiges namens Auflösungs Attribut, das bei der Auswahl zwischen Objekten verwendet werden soll.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------------------|
| CN                | ANR                                                               |
| LDAP-Display-Name | aNR                                                               |
| Size              | \-                                                                |
| Berechtigung aktualisieren  | Schema Administrator                                              |
| Aktualisierungshäufigkeit  | Wenn ein Attribut hinzugefügt oder aus der ANR-Liste entfernt werden muss. |
| Attribute-Id      | 1.2.840.113556.1.4.1208                                           |
| System-ID-GUID    | 45b01500-c419-11d1-bbc9-0080c76670c0                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                       |
