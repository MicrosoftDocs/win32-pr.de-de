# <a name="anr-attribute"></a>ANR-Attribut

Mehrdeutiges Namensauflösungsattribut, das bei der Auswahl zwischen Objekten verwendet werden soll.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------------------|
| CN                | Anr                                                               |
| Ldap-Anzeigename | Anr                                                               |
| Size              | \-                                                                |
| Aktualisieren von Berechtigungen  | Schemaadministrator                                              |
| Updatehäufigkeit  | Wenn ein Attribut der ANR-Liste hinzugefügt oder daraus entfernt werden muss. |
| Attribute-Id      | 1.2.840.113556.1.4.1208                                           |
| System-ID-GUID    | 45b01500-c419-11d1-sender9-0080c76670c0                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                       |
