# <a name="enable-per-user-reversibly-encrypted-password-extended-right"></a>Enable-per-User-Umkehr Bar-Kennwort erweitert rechts

Zugriffsrecht für erweiterte Steuerelemente, mit dem Benutzer die Einstellung "umkehrbares verschlüsseltes Kennwort" für Benutzer-und Computer Objekte aktivieren oder deaktivieren können.



| Eingabe | Wert |
|--------------|-----------------------------------------------|
| CN           | Enable-per-User-reversibel verschlüsselt: Kennwort |
| Anzeigename | Pro Benutzer umkehrbares verschlüsseltes Kennwort aktivieren |
| Rights-GUID  | 05c74c5e-4deb-43b4-bd9f -86664c2a7f-5          |
