# <a name="enable-per-user-reversibly-encrypted-password-extended-right"></a>Erweitertes Recht "Enable-Per-User-Reversibly-Encrypted-Password"

Erweitertes Zugriffsrecht für die Steuerung, mit dem Benutzer die Einstellung "Umkehrbares verschlüsseltes Kennwort" für Benutzer- und Computerobjekte aktivieren oder deaktivieren können.



| Eingabe | Wert |
|--------------|-----------------------------------------------|
| CN           | Enable-Per-User-Reversibly-Encrypted-Password |
| Anzeigename | Aktivieren eines reversibly verschlüsselten Kennworts pro Benutzer |
| Rights-GUID  | 05c74c5e-4deb-43b4-bd9f-86664c2a7fd5          |
