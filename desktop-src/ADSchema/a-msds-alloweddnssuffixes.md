# <a name="ms-ds-allowed-dns-suffixes-attribute"></a>ms-DS-Allowed-DNS-Suffixes-Attribut

Die Liste der zulässigen Suffixe für das dNSHostName-Attribut in Computer Objekten.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-DS-zulässige DNS-Suffixe                  |
| LDAP-Display-Name | MSDS-Zuweisung                     |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänen Administrator                        |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1710                     |
| System-ID-GUID    | 8469441b-9ac4-4e45-8205-bd219dbf 672D        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
