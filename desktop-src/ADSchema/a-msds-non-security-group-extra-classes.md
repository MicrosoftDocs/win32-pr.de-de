# <a name="ms-ds-non-security-group-extra-classes-attribute"></a>ms-DS-Non-Security-Group-extra-Classes-Attribut

Die allgemeinen Namen der nicht standardmäßigen Klassen, die über das Snap-in "Active Directory-Benutzer und-Computer" einer nicht-Sicherheitsgruppe hinzugefügt werden können.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Non-Security-Group-extra-Classes      |
| LDAP-Display-Name | MSDS-Non-Security-Group-extra-Classes       |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänenadministrator                        |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1689                     |
| System-ID-GUID    | 2debug FC-1F 52-486-bdf4-16lcc3084e54        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
