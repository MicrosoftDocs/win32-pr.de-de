# <a name="ms-ds-revealed-users-attribute"></a>ms-DS-enthüllte-users-Attribut

Wird mit RODCs verwendet, um die Benutzer Objekte zu identifizieren, deren Geheimnisse diesem RODC offengelegt wurden.



| Eingabe | Wert |
|-------------------|-------------------------------------------------|
| CN                | ms-DS-offen: Benutzer                            |
| LDAP-Display-Name | MSDS-revealedusers                              |
| Size              | \-                                              |
| Berechtigung aktualisieren  | \-                                              |
| Aktualisierungshäufigkeit  | \-                                              |
| Attribute-Id      | 1.2.840.113556.1.4.1924                         |
| System-ID-GUID    | 185c7821-3749-443a-bd6a-288899071adb            |
| Syntax            | [**Object(DN-Binary)**](s-object-dn-binary.md) |
