# <a name="ms-pki-certificate-application-policy-attribute"></a>MS-PKI-Certificate-Application-Policy-Attribut

Die Anwendungsrichtlinien-OID in einem Zertifikat.



| Eingabe | Wert |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-Certificate-Application-Policy                                                      |
| LDAP-Display-Name | mspki-Zertifikat-Anwendungs Richtlinie                                                       |
| Size              | \-                                                                                         |
| Berechtigung aktualisieren  | Unternehmensadministrator                                                                   |
| Aktualisierungshäufigkeit  | Wenn eine neue Vorlage erstellt wird oder die Attribute vorhandener Vorlagen bearbeitet werden. |
| Attribute-Id      | 1.2.840.113556.1.4.1674                                                                    |
| System-ID-GUID    | dbd90548-aa37-4202-9966-8c537ba5ce32                                                       |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                |
