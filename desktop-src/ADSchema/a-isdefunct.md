# <a name="is-defunct-attribute"></a>Is-Defunct-Attribut

**True** gibt an, dass die Klasse oder das Attribut nicht mehr verwendet werden kann. Alte Versionen dieses Objekts können vorhanden sein, es können jedoch keine neuen Versionen erstellt werden.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Is-Defunct                           |
| LDAP-Display-Name | isDefunct                            |
| Size              | \-                                   |
| Berechtigung aktualisieren  | Schema Administrator                 |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.661               |
| System-ID-GUID    | 28630ebe-41d5-11d1-a9c1-0000b80367c1 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
