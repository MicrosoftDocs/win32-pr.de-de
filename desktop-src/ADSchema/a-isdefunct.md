# <a name="is-defunct-attribute"></a>Is-Defunct-Attribut

True **gibt an,** dass die Klasse oder das Attribut nicht mehr verwendet werden kann. Alte Versionen dieses Objekts sind möglicherweise vorhanden, aber es können keine neuen erstellt werden.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Is-Defunct                           |
| Ldap-Anzeigename | isDefunct                            |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | Schemaadministrator                 |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.661               |
| System-Id-Guid    | 28630ebe-41d5-11d1-a9c1-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
