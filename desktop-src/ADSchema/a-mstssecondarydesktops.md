|-------------------|-----------------------------------------|
| CN                | ms-TS-Secondary-Desktops                |
| Ldap-Anzeigename | msTSSecondaryDesktops                   |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | \-                                      |
| Updatehäufigkeit  | \-                                      |

| System-Id-Guid    | f63aa29a-bb31-48e1-bfab-0a6c5a1d39c2    |
