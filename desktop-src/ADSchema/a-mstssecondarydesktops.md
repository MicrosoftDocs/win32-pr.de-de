# <a name="ms-ts-secondary-desktops-attribute"></a>MS-TS-Sekundär-Desktops-Attribut

Dieses Attribut stellt das Array von vorwärts Links zu den sekundären Desktops eines Benutzers dar.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | MS-TS-sekundäre Desktops                |
| LDAP-Display-Name | mstssecondarydesktops                   |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2075                 |
| System-ID-GUID    | f63aa29a-bb31-48e1-bfab-0a6c5a1d39c2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
