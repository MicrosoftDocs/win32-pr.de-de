Liste der Mitglieder Anwendungs Gruppen oder Benutzer, die mit AZ-Role verknüpft sind.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Members-für-AZ-Role               |
| LDAP-Display-Name | MSDS-mitgliedsforazrole                   |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Azrollen-Administrator                           |
| Aktualisierungshäufigkeit  | Während der Initialisierung oder Richtlinien Änderung. |
| Attribute-Id      | 1.2.840.113556.1.4.1806                 |
| System-ID-GUID    | cbf7e6cd-85a4-4314-8939-8bfe80597835    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
