# <a name="com-typelib-id-attribute"></a>COM-TypeLib-ID-Attribut

Dieses Attribut speichert die Liste der Typbibliotheks-IDs, die in diesem Anwendungspaket enthalten sind.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------------|
| CN                | COM-TypeLib-ID                                                                   |
| LDAP-Display-Name | comtypelibid                                                                     |
| Size              | \-                                                                               |
| Berechtigung aktualisieren  | Jeder kann dieses Objekt basierend auf der Sicherheit des Objekts, das erstellt wird, aktualisieren. |
| Aktualisierungshäufigkeit  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.254                                                           |
| System-ID-GUID    | 281416un-1968-11D0-a28f -00aa003049e2                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
