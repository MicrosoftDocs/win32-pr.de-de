# <a name="ms-sql-sortorder-attribute"></a>MS-SQL-sortor-Attribut

Die Sortierreihenfolge für die aktuelle Instanz von SQL Server.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-sortor                            |
| LDAP-Display-Name | MS-SQL-sortor                            |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänen Administrator                        |
| Aktualisierungshäufigkeit  | Beim System Setup.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1371                     |
| System-ID-GUID    | 6ddc42c0-CCEE-11d2-9993-0000f 87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
