# <a name="meetingendtime-attribute"></a>meetingendtime-Attribut

Das Datum und die Uhrzeit, zu der die Besprechung enden soll.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------------|
| CN                | meetingendtime                                                                   |
| LDAP-Display-Name | meetingendtime                                                                   |
| Size              | \-                                                                               |
| Berechtigung aktualisieren  | Jeder kann dieses Objekt basierend auf der Sicherheit des Objekts, das erstellt wird, aktualisieren. |
| Aktualisierungshäufigkeit  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.588                                                           |
| System-ID-GUID    | 11b6cc91-48C4-11d1-a9c3-0000t80367c1                                             |
| Syntax            | [**String(Generalized-Time)**](s-string-generalized-time.md)                    |
