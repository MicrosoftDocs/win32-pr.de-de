# <a name="builtin-modified-count-attribute"></a>Builtin-Modified-Count-Attribut

Das **Attribut Builtin-Modified-Count** wird verwendet, um die Replikation in Windows NT 4.0-Domänen zu unterstützen.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Builtin-Modified-Count               |
| Ldap-Anzeigename | builtinModifiedCount                 |
| Size              | 8 Bytes                              |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.     |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.14                |
| System-Id-Guid    | bf967930-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervall**](s-interval.md)       |
