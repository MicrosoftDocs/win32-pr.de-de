# <a name="builtin-modified-count-attribute"></a>Builtin-modified-count-Attribut

Das Attribut " **builtin-modified-count** " wird zur Unterstützung der Replikation in Windows NT 4,0-Domänen verwendet.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Builtin-modified-count               |
| LDAP-Display-Name | builtinmodifiedcount                 |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.14                |
| System-ID-GUID    | bf967930-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervall**](s-interval.md)       |
