# <a name="msmq-site-2-attribute"></a>MSMQ-Site-2-Attribut

Der DN der zweiten Site eines verbundenen Paars.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | MSMQ-Site-2                             |
| LDAP-Display-Name | mSMQSite2                               |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn eine MSMQ-Site-Line erstellt wird. |
| Attribute-Id      | 1.2.840.113556.1.4.944                  |
| System-ID-GUID    | 9a0dc338-C100-11d1-bbc5-0080c76670c0    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
