# <a name="ms-ds-cached-membership-attribute"></a>ms-DS-zwischengespeicherte Mitgliedschafts Attribut

Das Attribut " **ms-DS-Cached-Membership** " wird vom Sicherheits Konten-Manager für die Gruppen Erweiterung während der tokenbewertung verwendet.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | ms-DS-zwischengespeicherte Mitgliedschaft                               |
| LDAP-Display-Name | zwischengespeicherte MSDS-Mitgliedschaft                                |
| Size              | Binäres Blob von bis zu 3 Kilobyte.                     |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                      |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1441                               |
| System-ID-GUID    | 69cab008-cdd4-4bc9-bab8-0ff37efe1b20                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
