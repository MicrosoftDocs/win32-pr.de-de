# <a name="ms-ds-cached-membership-attribute"></a>ms-DS-Cached-Membership-Attribut

Das **attribut ms-DS-Cached-Membership** wird vom Sicherheitskonten-Manager für die Gruppenerweiterung während der Tokenauswertung verwendet.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | ms-DS-Cached-Membership                               |
| Ldap-Anzeigename | msDS-Cached-Membership                                |
| Size              | Binäres BLOB von bis zu 3 KB.                     |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.                      |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1441                               |
| System-Id-Guid    | 69cab008-cdd4-4bc9-blu8-0ff37efe1b20                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
