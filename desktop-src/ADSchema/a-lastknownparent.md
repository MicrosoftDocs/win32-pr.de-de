# <a name="last-known-parent-attribute"></a>Letztes-bekannt/übergeordnetes Attribut

Der Distinguished Name (DN) des letzten bekannten übergeordneten Elements eines verwaisten Objekts.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Letztes-bekannt/übergeordnetes Element                       |
| LDAP-Display-Name | lastKnownParent                         |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.        |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn ein Objekt verwaist ist.         |
| Attribute-Id      | 1.2.840.113556.1.4.781                  |
| System-ID-GUID    | 52ab8670-5709-11d1-a9c6-0000b80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
