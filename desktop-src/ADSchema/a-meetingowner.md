# <a name="meetingowner-attribute"></a>meetingowner-Attribut

Die Person, die für die Besprechung zuständig ist.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------------|
| CN                | meetingowner                                                                     |
| LDAP-Display-Name | meetingowner                                                                     |
| Size              | \-                                                                               |
| Berechtigung aktualisieren  | Jeder kann dieses Objekt basierend auf der Sicherheit des Objekts, das erstellt wird, aktualisieren. |
| Aktualisierungshäufigkeit  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.579                                                           |
| System-ID-GUID    | 11b6cc88-48C4-11d1-a9c3-0000t80367c1                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
