# <a name="frs-computer-reference-attribute"></a>FRS-Computer-Verweis Attribut

Verweis auf das Computer Objekt eines Replikat Satz Mitglieds.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | FRS-Computer-Referenz                  |
| LDAP-Display-Name | frsComputerReference                    |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.869                  |
| System-ID-GUID    | 2a132578-9373-11d1-AEbc-0000 C1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
