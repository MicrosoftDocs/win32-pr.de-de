# <a name="flat-name-attribute"></a>Flat-Name-Attribut

Bei Windows NT-Domänen ist der flache Name der NetBIOS-Name. Für Verknüpfungen mit nicht-Windows NT-Domänen ist der flache Name der identifizierende Name dieser Domäne, oder der Name ist **null**.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Flat-Name                                   |
| LDAP-Display-Name | flatname                                    |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.511                      |
| System-ID-GUID    | b7b13117-b82e-11d0-afee-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
