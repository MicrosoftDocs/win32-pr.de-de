# <a name="system-may-contain-attribute"></a>System-May-Contain-Attribut

Die Liste der optionalen Attribute für eine Klasse. Die Liste der Attribute kann nur vom System geändert werden.



| Eingabe | Wert |
|-------------------|------------------------------------------------------------------------------|
| CN                | System-May-Contain                                                           |
| Ldap-Anzeigename | systemMayContain                                                             |
| Size              | \-                                                                           |
| Aktualisieren von Berechtigungen  | Schemaadministrator                                                         |
| Updatehäufigkeit  | Wenn die Klasse erstellt wird oder der Klasse ein neues optionales Attribut hinzugefügt wird. |
| Attribute-Id      | 1.2.840.113556.1.4.196                                                       |
| System-ID-GUID    | bf967a44-0de6-11d0-a285-00aa003049e2                                         |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md)              |
