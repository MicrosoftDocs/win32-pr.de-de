# <a name="ms-ds-executescriptpassword-attribute"></a>ms-DS-ExecuteScriptPassword-Attribut

Wird während der Umbenennung der Domäne verwendet. Dieser Wert kann nicht mitHILFE von LDAP in ldap geschrieben oder daraus gelesen werden.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | ms-DS-ExecuteScriptPassword                           |
| Ldap-Anzeigename | msDS-ExecuteScriptPassword                            |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.                      |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1783                               |
| System-ID-GUID    | 9d054a5a-d187-46c1-9d85-42dfc44a56dd                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
