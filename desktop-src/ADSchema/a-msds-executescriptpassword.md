# <a name="ms-ds-executescriptpassword-attribute"></a>ms-DS-executescriptpassword-Attribut

Wird während der Domänen Umbenennung verwendet. Dieser Wert kann nicht mit LDAP geschrieben oder aus diesem gelesen werden.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | ms-DS-executescriptpassword                           |
| LDAP-Display-Name | MSDS-executescriptpassword                            |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                      |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1783                               |
| System-ID-GUID    | 9d054a5a-D187-46c1-9d85-42dfc44a56dd                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
