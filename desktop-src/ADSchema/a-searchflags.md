## <a name="remarks"></a>Bemerkungen

Dieses Attribut kann NULL sein oder eine Kombination aus einem oder mehreren der folgenden Werte aufweisen.



| Wert            | Beschreibung                                                                                                                                                                                                                                                                                                                                                               |
|------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 (0x00000001)   | Erstellen Sie einen Index für das-Attribut.                                                                                                                                                                                                                                                                                                                                        |
| 2 (0x00000002)   | Erstellen Sie in jedem Container einen Index für das-Attribut.                                                                                                                                                                                                                                                                                                                      |
| 4 (0x00000004)   | Fügen Sie dieses Attribut dem ANR-Satz (mehrdeutige Namensauflösung) hinzu. Diese dient zur Unterstützung bei der Suche nach einem Objekt, wenn nur partielle Informationen angegeben werden. Wenn der LDAP-Filter beispielsweise (ANR = Jeff) lautet, findet die Suche jedes Objekt, bei dem Vorname, Nachname, e-Mail-Adresse oder anderes ANR-Attribut gleich Jeff ist. Für diesen Index muss das Bit 0 festgelegt werden. |
| 8 (0x00000008)   | Behalten Sie dieses Attribut im Tombstone-Objekt für gelöschte Objekte bei.                                                                                                                                                                                                                                                                                                      |
| 16 (0x00000010)  | Kopieren Sie den Wert für dieses Attribut, wenn das Objekt kopiert wird.                                                                                                                                                                                                                                                                                                              |
| 32 (0x00000020)  | Erstellen Sie einen Tupelindex für das Attribut. Dies verbessert die Suche, bei der der Platzhalter am Anfang der Such Zeichenfolge angezeigt wird. Beispiel: (SN = \* mith).                                                                                                                                                                                                                |
| 64 (0x00000040)   | Wird ab Adam unterstützt. Erstellt einen Index, der die Leistung von VLV bei beliebigen Attributen erheblich unterstützt.                                                                                                                                                                                                                                                                  |
| 128 (0x00000080) | Markieren Sie das Attribut als vertraulich. Ignoriert für Basis Schema Attribute (systemFlags = 0x10).                                                                                                                                                                                                                                                                                    |
| 64 (0x00000040)  | Unterstützt ab Windows Server 2008. Erstellen Sie einen Index zum Verbessern der Leistung der VLV-Suche für dieses Attribut.                                                                                                                                                                                                                                                        |
