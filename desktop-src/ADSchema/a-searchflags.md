## <a name="remarks"></a>Hinweise

Dieses Attribut kann 0 (null) oder eine Kombination aus mindestens einem der folgenden Werte sein.



| Wert            | Beschreibung                                                                                                                                                                                                                                                                                                                                                               |
|------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 (0x00000001)   | Erstellen Sie einen Index für das Attribut.                                                                                                                                                                                                                                                                                                                                        |
| 2 (0x00000002)   | Erstellen Sie einen Index für das Attribut in jedem Container.                                                                                                                                                                                                                                                                                                                      |
| 4 (0x00000004)   | Fügen Sie dieses Attribut dem AnR-Satz (Mehrdeutige Namensauflösung) hinzu. Dies wird verwendet, um beim Suchen eines Objekts zu helfen, wenn nur partielle Informationen angegeben werden. Wenn der LDAP-Filter beispielsweise (ANR=JEFF) ist, sucht die Suche nach jedem Objekt, bei dem der Vorname, Nachname, die E-Mail-Adresse oder ein anderes ANR-Attribut gleich JEFF ist. Bit 0 muss festgelegt werden, damit dieser Index Auswirkungen hat. |
| 8 (0x00000008)   | Behalten Sie dieses Attribut im Tombstoneobjekt für gelöschte Objekte bei.                                                                                                                                                                                                                                                                                                      |
| 16 (0x00000010)  | Kopieren Sie den Wert für dieses Attribut, wenn das -Objekt kopiert wird.                                                                                                                                                                                                                                                                                                              |
| 32 (0x00000020)  | Erstellen Sie einen Tupelindex für das Attribut. Dadurch werden Suchvorgänge verbessert, bei denen der Platzhalter am Anfang der Suchzeichenfolge angezeigt wird. Beispiel: (sn= \* mith).                                                                                                                                                                                                                |
| 64(0x00000040)   | Wird ab ADAM unterstützt. Erstellt einen Index, um die VLV-Leistung bei beliebigen Attributen zu verbessern.                                                                                                                                                                                                                                                                  |
| 128 (0x00000080) | Markieren Sie das Attribut als vertraulich. Wird für Basisschemaattribute ignoriert (systemFlags=0x10).                                                                                                                                                                                                                                                                                    |
| 64 (0x00000040)  | Wird ab Windows Server 2008 unterstützt. Erstellen Sie einen Index, um die VLV-Suchleistung für dieses Attribut zu verbessern.                                                                                                                                                                                                                                                        |
