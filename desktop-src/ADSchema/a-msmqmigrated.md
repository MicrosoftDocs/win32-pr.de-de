# <a name="msmq-migrated-attribute"></a>MSMQ-Migrated-Attribut

Das Attribut " **MSMQ-migriert** " enthält MSMQ-Informationen im gemischten Modus.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MSMQ-Migrated                        |
| LDAP-Display-Name | msmqmigriert                         |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.952               |
| System-ID-GUID    | 9a0dc33f-C100-11d1-bbc5-0080c76670c0 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
