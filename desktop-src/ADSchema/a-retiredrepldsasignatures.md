# <a name="retired-repl-dsa-signatures-attribute"></a>Zurückgezogene-repl-DSA-Signaturen-Attribut

Verfolgt die letzten DS-Replikations Identitäten eines bestimmten Domänen Controllers.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------------------------------------|
| CN                | Veraltet-repl-DSA-Signaturen                                                         |
| LDAP-Display-Name | retiredrepldsasignaturen                                                            |
| Size              | Die Länge ist proportional zur Häufigkeit, mit der der DC von der Sicherung wieder hergestellt wurde. |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                                                    |
| Aktualisierungshäufigkeit  | \-                                                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.673                                                              |
| System-ID-GUID    | 7bf dcb7f -4807-11d1-a9c3-0000b80367c1                                                |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                               |
