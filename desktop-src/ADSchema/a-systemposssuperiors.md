# <a name="system-poss-superiors-attribute"></a>System-Poss-Vorgesetzten-Attribut

Die Liste der Klassen, die diese Klasse enthalten können. Diese Liste kann nur vom System geändert werden.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------------------|
| CN                | System-Poss-Vorgesetzten                                                       |
| LDAP-Display-Name | systempossvorgesetzten                                                         |
| Size              | \-                                                                          |
| Berechtigung aktualisieren  | Schema Administrator                                                        |
| Aktualisierungshäufigkeit  | Wenn die Klasse erstellt oder eine neue mögliche übergeordnete Klasse der Klasse hinzugefügt wird. |
| Attribute-Id      | 1.2.840.113556.1.4.195                                                      |
| System-ID-GUID    | bf967a47-0de6-11d0-a285-00aa003049e2                                        |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md)             |
