# <a name="system-poss-superiors-attribute"></a>System-Poss-Superiors-Attribut

Die Liste der Klassen, die diese Klasse enthalten können. Diese Liste kann nur vom System geändert werden.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------------------|
| CN                | System-Poss-Superiors                                                       |
| Ldap-Anzeigename | systemPossSuperiors                                                         |
| Size              | \-                                                                          |
| Aktualisieren von Berechtigungen  | Schemaadministrator                                                        |
| Updatehäufigkeit  | Wenn die -Klasse erstellt wird oder der -Klasse ein neuer möglicher übergeordneter Hinzugefügt wird. |
| Attribute-Id      | 1.2.840.113556.1.4.195                                                      |
| System-ID-GUID    | bf967a47-0de6-11d0-a285-00aa003049e2                                        |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md)             |
