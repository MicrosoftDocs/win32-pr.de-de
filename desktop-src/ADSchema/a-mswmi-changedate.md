# <a name="ms-wmi-changedate-attribute"></a>MS-WMI-changeDate-Attribut

Das letzte Datum, an dem dieses Objekt geändert wurde.



| Eingabe | Wert |
|-------------------|---------------------------------------------------|
| CN                | MS-WMI-changeDate                                 |
| LDAP-Display-Name | mslimi-changeDate                                  |
| Size              | Darstellung des WMI-datetime-Datentyps-26 Zeichen. |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                  |
| Aktualisierungshäufigkeit  | \-                                                |
| Attribute-Id      | 1.2.840.113556.1.4.1624                           |
| System-ID-GUID    | f9cdf7a0-ec44-4937-a79b-cd91522b3aa8              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)       |
