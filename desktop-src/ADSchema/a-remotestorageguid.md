# <a name="remote-storage-guid-attribute"></a>Remote-Storage-GUID-Attribut

Dieses Attribut enthält die GUID für ein Remotespeicherobjekt.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Remote-Storage-GUID                         |
| Ldap-Anzeigename | remoteStorageGUID                           |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | \-                                          |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.809                      |
| System-ID-GUID    | 2a39c5b0-8960-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
