# <a name="remote-storage-guid-attribute"></a>Remote-Storage-GUID-Attribut

Dieses Attribut enthält die GUID für ein Remote Speicher Objekt.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Remote-Storage-GUID                         |
| LDAP-Display-Name | remotestorageguid                           |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.809                      |
| System-ID-GUID    | 2a39c5b0-8960-11d1-AEbc-0000b80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
