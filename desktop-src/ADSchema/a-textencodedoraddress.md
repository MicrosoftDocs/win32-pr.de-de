# <a name="text-encoded-or-address-attribute"></a>Text-codiertes-or-address-Attribut

Dieses Attribut wird verwendet, um X. 400-Adressen in einem Textformat zu unterstützen.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Text-codiert-or-address                     |
| LDAP-Display-Name | textEncodedORAddress                        |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.2                   |
| System-ID-GUID    | a8df7489-c5ea-11d1-bbcb-0080c76670c0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
