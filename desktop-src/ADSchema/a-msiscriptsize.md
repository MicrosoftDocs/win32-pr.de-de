|-------------------|--------------------------------------|
| CN                | Msi-Script-Size                      |
| Ldap-Anzeigename | msiScriptSize                        |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.846               |
| System-ID-GUID    | 96a7dd63-9118-11d1-aebc-0000f80367c1 |
