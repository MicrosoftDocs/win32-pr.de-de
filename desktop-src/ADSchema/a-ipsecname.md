# <a name="ipsec-name-attribute"></a>Ipsec-Name-Attribut

Das **IPSec-Name-** Attribut ist nur für die interne Verwendung vorgesehen.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Ipsec-Name                                  |
| LDAP-Display-Name | ipsecname                                   |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.620                      |
| System-ID-GUID    | b40ff81c-427a-11d1-a9c2-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
