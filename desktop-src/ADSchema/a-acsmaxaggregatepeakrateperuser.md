# <a name="acs-max-aggregate-peak-rate-per-user-attribute"></a>ACS-Max-Aggregate-Spitze-Rate-pro-Benutzer-Attribut

Die maximale Spitzen Rate, die jeder Benutzer für alle Flows haben kann.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-Max-Aggregate-Spitzen Rate pro Benutzer |
| LDAP-Display-Name | acsmaxaggregatepeer akrateperuser       |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.897               |
| System-ID-GUID    | f072230c-aef5-11d1-bdcf-0000f80367c1 |
| Syntax            | [**Intervall**](s-interval.md)       |
