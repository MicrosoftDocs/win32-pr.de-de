# <a name="ms-ts-endpoint-data-attribute"></a>MS-TS-Endpoint-Data-Attribut

Dieses Attribut stellt den VM-Namen für einen Computer in einer TSV-Bereitstellung dar.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-TS-Endpoint-Data                         |
| LDAP-Display-Name | mstsendpointdata                            |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2070                     |
| System-ID-GUID    | 40e1c407-4344-40l3-ab43-3625a34a63a2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
