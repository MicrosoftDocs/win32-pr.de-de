# <a name="ms-ds-cached-membership-time-stamp-attribute"></a>ms-DS-zwischen gespeicherter Mitgliedschafts Zeitstempel-Attribut

Das Attribut **ms-DS-Cached-Membership-Zeitstempel** wird vom Sicherheits Konten-Manager für die Gruppen Erweiterung während der tokenauswertung verwendet.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ms-DS-zwischengespeicherte Mitgliedschafts Zeitstempel   |
| LDAP-Display-Name | MSDS-Cache-Membership-Zeitstempel    |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1442              |
| System-ID-GUID    | 3566bf1f-Beee-4dcb-8abe-ef89fcfec6c1 |
| Syntax            | [**Intervall**](s-interval.md)       |
