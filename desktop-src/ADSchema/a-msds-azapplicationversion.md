# <a name="ms-ds-az-application-version-attribute"></a>ms-DS-AZ-Application-Version-Attribut

Eine Versionsnummer, um anzugeben, dass die AzApplication aktualisiert wurde.



| Eingabe | Wert |
|-------------------|------------------------------------------------------|
| CN                | ms-DS-AZ-Application-Version                         |
| LDAP-Display-Name | MSDS-azapplicationversion                            |
| Size              | \-                                                   |
| Berechtigung aktualisieren  | Azrollen-Administrator                                        |
| Aktualisierungshäufigkeit  | Wenn die Versionsnummer für die Anwendung geändert wird. |
| Attribute-Id      | 1.2.840.113556.1.4.1817                              |
| System-ID-GUID    | 7184a120-3ac4-47ae-848f-fe0ab20784d4                 |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)          |
