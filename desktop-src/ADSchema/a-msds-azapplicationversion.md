# <a name="ms-ds-az-application-version-attribute"></a>ms-DS-Az-Application-Version-Attribut

Eine Versionsnummer, die angibt, dass AzApplication aktualisiert wird.



| Eingabe | Wert |
|-------------------|------------------------------------------------------|
| CN                | ms-DS-Az-Application-Version                         |
| Ldap-Anzeigename | msDS-AzApplicationVersion                            |
| Size              | \-                                                   |
| Aktualisieren von Berechtigungen  | AzRoles admin                                        |
| Updatehäufigkeit  | Wenn sich die Versionsnummer für die Anwendung ändert. |
| Attribute-Id      | 1.2.840.113556.1.4.1817                              |
| System-ID-GUID    | 7184a120-3ac4-47ae-848f-fe0ab20784d4                 |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)          |
