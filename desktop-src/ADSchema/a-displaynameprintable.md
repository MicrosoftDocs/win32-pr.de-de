# <a name="display-name-printable-attribute"></a>Anzeige Name-druckbares Attribut

Der druckbare Anzeige Name für ein Objekt. Der Druck Bare Anzeige Name ist normalerweise die Kombination aus dem Vornamen, dem mittleren Anfangs-und Nachnamen des Benutzers.



| Eingabe | Wert |
|-------------------|----------------------------------------|
| CN                | Anzeige Name-druckdruck                 |
| LDAP-Display-Name | Display Name Print                   |
| Size              | \-                                     |
| Berechtigung aktualisieren  | Domänen Administrator oder Konto Besitzer. |
| Aktualisierungshäufigkeit  | \-                                     |
| Attribute-Id      | 1.2.840.113556.1.2.353                 |
| System-ID-GUID    | bf967954-0de6-11d0-a285-00aa003049e2   |
| Syntax            | [**String(IA5)**](s-string-ia5.md)    |
