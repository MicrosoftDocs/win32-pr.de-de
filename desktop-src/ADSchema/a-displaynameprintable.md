# <a name="display-name-printable-attribute"></a>Display-Name-Printable-Attribut

Der druckbare Anzeigename für ein Objekt. Der druckbare Anzeigename ist in der Regel die Kombination aus Vorname, Anfangsname und Nachname des Benutzers.



| Eingabe | Wert |
|-------------------|----------------------------------------|
| CN                | Display-Name-Printable                 |
| Ldap-Anzeigename | displayNamePrintable                   |
| Size              | \-                                     |
| Aktualisieren von Berechtigungen  | Domänenadministrator oder Kontobesitzer. |
| Updatehäufigkeit  | \-                                     |
| Attribute-Id      | 1.2.840.113556.1.2.353                 |
| System-Id-Guid    | bf967954-0de6-11d0-a285-00aa003049e2   |
| Syntax            | [**String(IA5)**](s-string-ia5.md)    |
