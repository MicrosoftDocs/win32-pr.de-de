# <a name="msmq-sites-attribute"></a>MSMQ-Sites-Attribut

Die Liste der Websites, zu denen der Computer gehört (ein Array von objectguids der Standorte, in der Regel nicht mehr als eine GUID).



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Sites                                            |
| LDAP-Display-Name | msmqsites                                             |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.927                                |
| System-ID-GUID    | 9a0dc32a-C100-11d1-bbc5-0080c76670c0                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
