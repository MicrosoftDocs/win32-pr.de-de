# <a name="msmq-sites-attribute"></a>MSMQ-Sites-Attribut

Die Liste der Websites, zu denen der Computer gehört (ein Array der objectGUIDs der Standorte, in der Regel nicht mehr als eine GUID).



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Sites                                            |
| Ldap-Anzeigename | mSMQSites                                             |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | \-                                                    |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.927                                |
| System-ID-GUID    | 9a0dc32a-c100-11d1-sender5-0080c76670c0                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
