# <a name="ms-sql-name-attribute"></a>MS-SQL-Name-Attribut

Der Name einer SQL Server-Instanz. Der Standardwert ist MSSQLSERVER.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Name                                 |
| LDAP-Display-Name | MS-SQL-Name                                 |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.            |
| Aktualisierungshäufigkeit  | Beim System Setup.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1363                     |
| System-ID-GUID    | 3532dfd8-CCEE-11d2-9993-0000f 87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
