# <a name="original-display-table-attribute"></a>Original-Display-Table-Attribut

Die MAPI-Anzeigetabelle (original) für einen Adresseintrag.

|-------------------|-------------------------------------------------------|
| CN                | Original-Display-Table                                |
| Ldap-Anzeigename | originalDisplayTable                                  |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | \-                                                    |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.445                                |
| System-Id-Guid    | 5fd424ce-1262-11d0-a060-00aa006c33ed                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
