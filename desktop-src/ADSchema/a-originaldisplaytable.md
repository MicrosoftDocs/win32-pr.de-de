# <a name="original-display-table-attribute"></a>Original-Display-Table-Attribut

Die MAPI-Anzeige Tabelle (Original) für einen Adresseintrag.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Original-Display-Table                                |
| LDAP-Display-Name | originaldisplaytable                                  |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.445                                |
| System-ID-GUID    | 5F d424ce-1262-11D0-a060-00aa006c33ed                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
