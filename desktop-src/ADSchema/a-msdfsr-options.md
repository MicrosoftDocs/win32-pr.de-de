# <a name="ms-dfsr-options-attribute"></a>MS-DFSR-options-Attribut

Enthält die Optionen für die verteiltes Dateisystem (DFS)-Replikations Objekte.

|-------------------|--------------------------------------|
| CN                | MS-DFSR-Optionen                      |
| LDAP-Display-Name | msdfsr-Optionen                       |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.17           |
| System-ID-GUID    | d6d67084-C720-417d-8647-b696237a114c |
| Syntax            | [**Enumeration**](s-enumeration.md) |
