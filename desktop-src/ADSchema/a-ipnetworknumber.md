# <a name="ipnetworknumber-attribute"></a>ipnetworknumber-Attribut

Enthält eine IP-Netzwerk Nummer in punktierter Dezimal Schreibweise, wobei die führenden Nullen weggelassen werden.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ipnetworknumber                      |
| LDAP-Display-Name | ipnetworknumber                      |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.20                     |
| System-ID-GUID    | 4e3854f4-3087-42a4-A813-bb0c528958d3 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
