# <a name="trust-parent-attribute"></a>Trust-Parent-Attribut

Das übergeordnete Element in der KERBEROS-Vertrauenshierarchie.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Trust-Parent                            |
| Ldap-Anzeigename | trustParent                             |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.        |
| Updatehäufigkeit  | Wenn eine untergeordnete Domäne erstellt wird.         |
| Attribute-Id      | 1.2.840.113556.1.4.471                  |
| System-Id-Guid    | b000ea7a-a086-11d0-afdd-00c04fd930c9    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
