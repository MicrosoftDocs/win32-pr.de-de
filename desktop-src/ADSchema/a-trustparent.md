# <a name="trust-parent-attribute"></a>Trust-Parent-Attribut

Das übergeordnete Element in der Kerberos-Vertrauens Hierarchie.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Trust-Parent                            |
| LDAP-Display-Name | Trust Parent                             |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.        |
| Aktualisierungshäufigkeit  | Wenn eine untergeordnete Domäne erstellt wird.         |
| Attribute-Id      | 1.2.840.113556.1.4.471                  |
| System-ID-GUID    | b000ea7a-a086-11d0-afdd-00c04fd930c9    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
