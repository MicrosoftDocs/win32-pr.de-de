# <a name="ms-ts-work-directory-attribute"></a>ms-TS-Work-Directory-Attribut

| CN                | ms-TS-Work-Directory                        |
| Ldap-Anzeigename | msTSWorkDirectory                           |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | \-                                          |
