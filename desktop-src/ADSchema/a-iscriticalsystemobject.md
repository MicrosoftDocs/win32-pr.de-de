# <a name="is-critical-system-object-attribute"></a>Is-critical-System-Object-Attribut

Wenn **true**, muss das Objekt, das dieses Attribut hostet, während der Installation eines neuen Replikats repliziert werden.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Is-critical-System-Object            |
| LDAP-Display-Name | isCriticalSystemObject               |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.868               |
| System-ID-GUID    | 00bbfi30d-91fe-11d1-AEbc-0000fi80367c1 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
