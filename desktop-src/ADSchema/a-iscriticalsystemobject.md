# <a name="is-critical-system-object-attribute"></a>Is-Critical-System-Object-Attribut

**True** gibt an, dass das Objekt, das dieses Attribut hostet, während der Installation eines neuen Replikats repliziert werden muss.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Is-Critical-System-Object            |
| Ldap-Anzeigename | isCriticalSystemObject               |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.868               |
| System-ID-GUID    | 00fbf30d-91fe-11d1-aebc-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
