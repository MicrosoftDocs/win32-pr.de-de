# <a name="ms-sql-keywords-attribute"></a>MS-SQL-Keywords-Attribut

Nicht verwendet. Der Standardwert ist auf Key 1, Key 2, Key 3, Key 4 und Key 5 festgelegt.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Schlüsselwörter                             |
| LDAP-Display-Name | MS-SQL-Schlüsselwörter                             |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | Beim System Setup.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1401                     |
| System-ID-GUID    | 01e9a98a-ccef-11d2-9993-0000f 87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
