Eine Zeichenfolge, die ein Bereichs Objekt eindeutig identifiziert.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-Scope-Name                         |
| LDAP-Display-Name | MSDS-AzScopeName                            |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Azrollen-Administrator                               |
| Aktualisierungshäufigkeit  | Während der Initialisierung oder Richtlinien Änderung.     |
| Attribute-Id      | 1.2.840.113556.1.4.1799                     |
| System-ID-GUID    | 515a6b06-2617-4173-8099-d5605df043c6        |
