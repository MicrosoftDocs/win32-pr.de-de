# <a name="ms-ds-nc-repl-inbound-neighbors-attribute"></a>ms-DS-NC-Repl-Inbound-Neighbors-Attribut

Replikationspartner für diese Partition. Dieser Server ruft Replikationsdaten von diesen anderen Servern ab, die als Quellen fungieren.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-DS-NC-Repl-Inbound-Neighbors             |
| Ldap-Anzeigename | msDS-NCReplInboundNeighbors                 |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.            |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1705                     |
| System-ID-GUID    | 9edba85a-3e9e-431b-9b1a-a5b6e9eda796        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
