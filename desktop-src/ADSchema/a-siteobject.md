# <a name="site-object-attribute"></a>Site-Object-Attribut

Der Distinguished Name für den Standort, zu dem dieses Subnetz gehört.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Site-Object                             |
| Ldap-Anzeigename | siteObject                              |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | Domänenadministrator                    |
| Updatehäufigkeit  | Jedes Mal, wenn eine Website erstellt wird.             |
| Attribute-Id      | 1.2.840.113556.1.4.512                  |
| System-Id-Guid    | 3e10944c-c354-11d0-aff8-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
