# <a name="site-object-attribute"></a>Site-Object-Attribut

Der Distinguished Name für den Standort, zu dem dieses Subnetz gehört.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Site-Object                             |
| LDAP-Display-Name | siteObject                              |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Domänen Administrator                    |
| Aktualisierungshäufigkeit  | Immer dann, wenn eine Website erstellt wird.             |
| Attribute-Id      | 1.2.840.113556.1.4.512                  |
| System-ID-GUID    | 3e10944c-C354-11D0-aff8-0000service 80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
