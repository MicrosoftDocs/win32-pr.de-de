# <a name="enabled-connection-attribute"></a>Enabled-Connection-Attribut

Gibt an, ob eine Verbindung zur Verwendung verfügbar ist.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Enabled-Connection                   |
| LDAP-Display-Name | enabledconnection                    |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.36                |
| System-ID-GUID    | bf967963-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
