# <a name="ms-sql-lastbackupdate-attribute"></a>MS-SQL-lastbackupdate-Attribut

Das letzte Datum, an dem die Datenbank gesichert wurde.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-lastbackupdate                       |
| LDAP-Display-Name | MS-SQL-lastbackupdate                       |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.            |
| Aktualisierungshäufigkeit  | Wenn die Datenbank gesichert wird.             |
| Attribute-Id      | 1.2.840.113556.1.4.1398                     |
| System-ID-GUID    | f2b6abca-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
