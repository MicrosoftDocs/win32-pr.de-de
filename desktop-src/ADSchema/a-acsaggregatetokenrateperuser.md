# <a name="acs-aggregate-token-rate-per-user-attribute"></a>ACS-Aggregate-Token-Rate-pro-Benutzer-Attribut

Die maximale TokenRate, die jeder Benutzer für alle Flows aufweisen kann.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-Aggregate-Token-Rate pro Benutzer    |
| LDAP-Display-Name | acsaggregatepkenrateperuser         |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.760               |
| System-ID-GUID    | 7b56127d-5301-11d1-a9c5-0000b80367c1 |
| Syntax            | [**Intervall**](s-interval.md)       |
