# <a name="ms-ds-has-domain-ncs-attribute"></a>ms-DS-has-Domain-NCS-Attribut

Verzeichnisdienst-Replikations Informationen, die Details zu den Domänen Namen Kontexten auf einem bestimmten Server enthalten.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------|
| CN                | ms-DS-has-Domain-NCS                                |
| LDAP-Display-Name | MSDS-hasdomainncs                                   |
| Size              | \-                                                  |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                     |
| Aktualisierungshäufigkeit  | Sie wird von Dcpromo festgelegt und danach nie geändert. |
| Attribute-Id      | 1.2.840.113556.1.4.1820                             |
| System-ID-GUID    | 6f 17e347-a842-4498-b8b3-15e007da4fed                |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)             |
