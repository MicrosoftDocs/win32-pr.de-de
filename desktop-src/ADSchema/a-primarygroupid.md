# <a name="primary-group-id-attribute"></a>Primary-Group-ID-Attribut

Enthält den relativen Bezeichner (RID) für die primäre Gruppe des Benutzers. Standardmäßig ist dies die RID für die Gruppe der Domänen Benutzer.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Primary-Group-ID                     |
| LDAP-Display-Name | primaryGroupId                       |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.98                |
| System-ID-GUID    | bf967a00-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
