# <a name="msmq-in-routing-servers-attribute"></a>MSMQ-in-Routing-Servers-Attribut

DN Links zu MSMQ-Routing Servern, über die der gesamte eingehende Datenverkehr an diesen Computer weitergeleitet werden soll.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | MSMQ-in-Routing-Server                 |
| LDAP-Display-Name | msmqinroutingservers                    |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.929                  |
| System-ID-GUID    | 9a0dc32c-C100-11d1-bbc5-0080c76670c0    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
