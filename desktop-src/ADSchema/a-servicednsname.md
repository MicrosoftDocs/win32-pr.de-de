# <a name="service-dns-name-attribute"></a>Attribut "Service-DNS-Name"

Der DNS-Name, der gesucht werden soll, um einen Server zu finden, auf dem dieser Dienst ausgeführt wird.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Dienst-DNS-Name                            |
| Ldap-Anzeigename | serviceDNSName                              |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | \-                                          |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.657                      |
| System-ID-GUID    | 28630eb8-41d5-11d1-a9c1-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
