# <a name="ms-wmi-genus-attribute"></a>MS-WMI-Genus-Attribut

Identifiziert den Objekttyp einer Codierung.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------|
| CN                | MS-WMI--Gattung                                                    |
| LDAP-Display-Name | mtauscht-Gattung                                                     |
| Size              | 4 Bytes. 1 gibt eine Klasse an. 2 gibt eine-Instanz an.          |
| Berechtigung aktualisieren  | Domänen Administrator                                            |
| Aktualisierungshäufigkeit  | Wenn eine Klasse, die das Attribut enthält, hinzugefügt oder geändert wird. |
| Attribute-Id      | 1.2.840.113556.1.4.1677                                         |
| System-ID-GUID    | 50c8673a-8F 56-4614-9308-9e1340b                            |
| Syntax            | [**Enumeration**](s-enumeration.md)                            |
