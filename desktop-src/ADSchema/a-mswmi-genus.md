# <a name="ms-wmi-genus-attribute"></a>ms-WMI-Attribut

Identifiziert den Objekttyp einer Codierung.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------|
| CN                | ms-WMI-Soll                                                    |
| Ldap-Anzeigename | msWMI-Soll                                                     |
| Size              | 4 Bytes. 1 gibt eine Klasse an. 2 gibt eine -Instanz an.          |
| Aktualisieren von Berechtigungen  | Domänenadministrator                                            |
| Updatehäufigkeit  | Jedes Mal, wenn eine Klasse, die das Attribut enthält, hinzugefügt oder geändert wird. |
| Attribute-Id      | 1.2.840.113556.1.4.1677                                         |
| System-Id-Guid    | 50c8673a-8f56-4614-9308-9e1340fb9af3                            |
| Syntax            | [**Enumeration**](s-enumeration.md)                            |
