# <a name="usn-source-attribute"></a>USN-Source-Attribut

Wert des Attributs, das von der Attribut-Änderung des [**-**](a-usnchanged.md) Objekts aus dem Remote Verzeichnis abgeleitet wurde, von dem die Änderung auf den lokalen Server repliziert wurde.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------------------------------------------|
| CN                | USN-Source                                                                                          |
| LDAP-Display-Name | Quelle                                                                                           |
| Size              | 8 Bytes                                                                                             |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                                                                    |
| Aktualisierungshäufigkeit  | Wenn das Objekt in einem Remote Verzeichnis geändert wird, das die Änderung in das lokale Verzeichnis repliziert hat. |
| Attribute-Id      | 1.2.840.113556.1.4.896                                                                              |
| System-ID-GUID    | 167758ad-47b3-11d1-a9c3-0000f 80367c1                                                                |
| Syntax            | [**Intervall**](s-interval.md)                                                                      |
