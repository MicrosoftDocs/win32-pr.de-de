# <a name="friendly-names-attribute"></a>Friendly-Names-Attribut

Liste der standardmäßigen Anzeige Namen Definitionen, die von einem Katalog unterstützt werden.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Friendly-Names                              |
| LDAP-Display-Name | friendlynames                               |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.682                      |
| System-ID-GUID    | 7bf dcb88-4807-11d1-a9c3-0000b80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
