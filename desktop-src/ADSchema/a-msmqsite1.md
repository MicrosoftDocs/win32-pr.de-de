# <a name="msmq-site-1-attribute"></a>MSMQ-Site-1-Attribut

Der DN der ersten Site eines verbundenen Paars.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | MSMQ-Site-1                             |
| LDAP-Display-Name | mSMQSite1                               |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn eine MSMQ-Site-Line erstellt wird. |
| Attribute-Id      | 1.2.840.113556.1.4.943                  |
| System-ID-GUID    | 9a0dc337-C100-11d1-bbc5-0080c76670c0    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
