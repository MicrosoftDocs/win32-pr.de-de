| CN                | Group-Type                           |
| Ldap-Anzeigename | groupType                            |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | Domänenadministrator                 |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.750               |
| System-ID-GUID    | 9a9a021e-4a5b-11d1-a9c3-0000f80367c1 |

## <a name="remarks"></a>Hinweise

Dieses Attribut kann 0 (null) oder eine Kombination aus mindestens einem der folgenden Werte sein.



| Wert                   | Beschreibung                                                                                  |
|-------------------------|----------------------------------------------------------------------------------------------|
| 1 (0x00000001)          | Gibt eine Gruppe an, die vom System erstellt wird.                                             |
| 2 (0x00000002)          | Gibt eine Gruppe mit globalem Gültigkeitsbereich an.                                                         |
| 4 (0x00000004)          | Gibt eine Gruppe mit lokalem Domänenbereich an.                                                   |
| 8 (0x00000008)          | Gibt eine Gruppe mit universellem Gültigkeitsbereich an.                                                      |
| 16 (0x00000010)         | Gibt eine APP \_ BASIC-Gruppe für Windows Serverautorisierungs-Manager an.                      |
| 32 (0x00000020)         | Gibt eine APP \_ QUERY-Gruppe für Windows Serverautorisierungs-Manager an.                      |
| 2147483648 (0x80000000) | Gibt eine Sicherheitsgruppe an. Wenn dieses Flag nicht festgelegt ist, ist die Gruppe eine Verteilergruppe. |



 

Weitere Informationen zum Gruppentyp und -bereich finden Sie in den Themen [Gruppentypen](/previous-versions/windows/it-pro/windows-server-2003/cc781446(v=ws.10)) und [Gruppenbereich](/previous-versions/windows/it-pro/windows-server-2003/cc755692(v=ws.10)) auf Microsoft TechNet.
