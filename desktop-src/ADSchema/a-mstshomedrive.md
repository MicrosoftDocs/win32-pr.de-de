# <a name="ms-ts-home-drive-attribute"></a>ms-TS-Home-Drive-Attribut

Terminal services Home Drive gibt ein Startlaufwerk für den Benutzer an.

|-------------------|---------------------------------------------|
| CN                | ms-TS-Home-Drive                            |
| Ldap-Anzeigename | msTSHomeDrive                               |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | \-                                          |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1978                     |
| System-ID-GUID    | 5f0a24d9-dffa-4cd9-acbf-a0680c03731e        |
