# <a name="ms-ds-filter-containers-attribute"></a>ms-DS-Filter-Containers-Attribut

Eine Zeichenfolge mit mehreren Werten, die die Namen der Klassen enthält, mit denen bestimmt wird, welche Containertypen beim Filtern von dem Active Directory-Benutzer und-Computer-Snap-in angezeigt werden sollen.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Filter-Container                     |
| LDAP-Display-Name | MSDS-filtercontainers                       |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänen Administrator                        |
| Aktualisierungshäufigkeit  | Wenn die Liste der Containertypen geändert wird.   |
| Attribute-Id      | 1.2.840.113556.1.4.1703                     |
| System-ID-GUID    | fb00dcdf-ac37-483a-9c12-ac53a6603033        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
