# <a name="governs-id-attribute"></a>Regelt-ID-Attribut

Die eindeutige Objekt-ID der Klasse, die von diesem Class-Schema Objekt definiert wird.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------|
| CN                | Regelt-ID                                                      |
| LDAP-Display-Name | governsID                                                       |
| Size              | \-                                                              |
| Berechtigung aktualisieren  | \-                                                              |
| Aktualisierungshäufigkeit  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.22                                           |
| System-ID-GUID    | bf96797d-0de6-11d0-a285-00aa003049e2                            |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
