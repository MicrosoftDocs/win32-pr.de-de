# <a name="content-indexing-allowed-attribute"></a>Content-Indizierung-zulässiges Attribut

Gibt an, ob das Volumeobjekt Inhalts indiziert werden kann.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Inhalts Indizierung (zulässig)             |
| LDAP-Display-Name | contentindexingallowed               |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.24                |
| System-ID-GUID    | bf967943-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
