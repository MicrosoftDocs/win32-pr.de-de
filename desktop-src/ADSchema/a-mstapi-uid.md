# <a name="ms-tapi-unique-identifier-attribute"></a>ms-TAPI-Unique-Identifier-Attribut

Gibt den Namen einer TAPI-Multicastkonferenz an. Es ist das Benennungsattribut der Rt-Conference Klasse.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------------|
| CN                | ms-TAPI-Unique-Identifier                                             |
| Ldap-Anzeigename | msTAPI-uid                                                            |
| Size              | Kann eine beliebige Zeichenfolge sein, in der Regel unter 100 Zeichen. |
| Aktualisieren von Berechtigungen  | Es sind keine besonderen Berechtigungen erforderlich.                                       |
| Updateh√§ufigkeit  | Legen Sie zum Zeitpunkt der Erstellung des -Objekts Rt-Conference fest.            |
| Attribute-Id      | 1.2.840.113556.1.4.1698                                               |
| System-Id-Guid    | 70a4e7ea-b3b9-4643-8918-e6dd2471bfd4                                  |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                           |
