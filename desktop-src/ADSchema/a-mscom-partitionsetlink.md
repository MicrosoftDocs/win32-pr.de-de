# <a name="ms-com-partitionsetlink-attribute"></a>MS-com-partitionsetlink-Attribut

Ein Link, der verwendet wird, um eine COM+-Partition einem com+-partitionsetobjekt zuzuordnen.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | MS-com-partitionsetlink                 |
| LDAP-Display-Name | MScom-partitionsetlink                  |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.        |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1424                 |
| System-ID-GUID    | 67f 121dc-7d02-4c7d-82F 5-9ad4c950 ac34    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
