# <a name="schema-version-attribute"></a>Schema-Version-Attribut

Die Versionsnummer für das Schema.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Schema-Version                       |
| LDAP-Display-Name | schemaVersion                        |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn das Schema freigegeben wird.    |
| Attribute-Id      | 1.2.840.113556.1.2.471               |
| System-ID-GUID    | bf967a2c-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
