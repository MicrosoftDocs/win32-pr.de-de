# <a name="min-ticket-age-attribute"></a>Min-Ticket-Age-Attribut

Dieses Attribut bestimmt den minimalen Zeitraum (in Stunden), für den das Ticket Erteilungs Ticket (TGT) eines Benutzers für die Kerberos-Authentifizierung verwendet werden kann, bevor eine Anforderung zum Erneuern des Tickets erteilt werden kann.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Min. Ticket-Alter                       |
| LDAP-Display-Name | minticketage                         |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.80                |
| System-ID-GUID    | bf9679c4-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervall**](s-interval.md)       |
