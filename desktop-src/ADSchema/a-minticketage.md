# <a name="min-ticket-age-attribute"></a>Min-Ticket-Age-Attribut

Dieses Attribut bestimmt den Mindestzeitraum in Stunden, den ein Ticket-Granting Ticket (TGT) eines Benutzers für die Kerberos-Authentifizierung verwenden kann, bevor eine Anforderung zur Verlängerung des Tickets gestellt werden kann.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Min-Ticket-Age                       |
| Ldap-Anzeigename | minTicketAge                         |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.80                |
| System-Id-Guid    | bf9679c4-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervall**](s-interval.md)       |
