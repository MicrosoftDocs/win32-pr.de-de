# <a name="documentlocation-attribute"></a>documentlocation-Attribut

Der documentlocation-Attributtyp gibt den Speicherort des ursprünglichen Dokuments an.

|-------------------|---------------------------------------------|
| CN                | documentlocation                            |
| LDAP-Display-Name | documentlocation                            |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.15                  |
| System-ID-GUID    | b958b14e-ac6d-4ec4-8892-be70b69f7281        |
