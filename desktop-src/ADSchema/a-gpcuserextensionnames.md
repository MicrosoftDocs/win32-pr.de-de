# <a name="gpc-user-extension-names-attribute"></a>GPC-User-Extension-names-Attribut

Wird vom Gruppenrichtlinie-Objekt für Benutzerrichtlinien verwendet.



| Eingabe | Wert |
|-------------------|---------------------------------------------------------------------------------|
| CN                | GPC-Benutzer-Erweiterungs Namen                                                        |
| LDAP-Display-Name | gpcuserextensionnames                                                           |
| Size              | Hängt von der Anzahl der Client seitigen Erweiterungen ab, die in diesem Gruppenrichtlinien Objekt über eine Richtlinie verfügen. |
| Berechtigung aktualisieren  | Domänen-oder Richtlinien Administrator.                                                 |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn ein GPO über das gpe aktualisiert wird.                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1349                                                         |
| System-ID-GUID    | 42a75fc6-783f -11d2-9916-0000f 87a57d4                                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                     |
