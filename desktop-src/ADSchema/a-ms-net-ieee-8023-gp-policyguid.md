# <a name="ms-net-ieee-8023-gp-policyguid-attribute"></a>ms-net-IEEE-8023-GP-policyguid-Attribut

Enthält eine GUID, die ein bestimmtes 802,3 Gruppenrichtlinie Objekt in der Domäne identifiziert.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-net-IEEE-8023-GP-policyguid              |
| LDAP-Display-Name | ms-net-IEEE-8023-GP-policyguid              |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1954                     |
| System-ID-GUID    | 94a7b05a-b8b2-4l59-9c25-39e69baa1684        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
