# <a name="com-progid-attribute"></a>COM-ProgID-Attribut

Dieses Attribut speichert die Liste der COM-Objekt Programm-IDs, die in diesem Anwendungspaket implementiert werden.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------------|
| CN                | COM-ProgID                                                                       |
| LDAP-Display-Name | comprogid                                                                        |
| Size              | \-                                                                               |
| Berechtigung aktualisieren  | Jeder kann dieses Objekt basierend auf der Sicherheit des Objekts, das erstellt wird, aktualisieren. |
| Aktualisierungshäufigkeit  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.21                                                            |
| System-ID-GUID    | bf96793d-0de6-11d0-a285-00aa003049e2                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
