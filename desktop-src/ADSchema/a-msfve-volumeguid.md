# <a name="ms-fve-volumeguid-attribute"></a>MS-f-VolumeGuid-Attribut

Enthält die GUID, die einem von BitLocker unterstützten Datenträger Volume zugeordnet ist.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | MS-f-VolumeGuid                                     |
| LDAP-Display-Name | MSF-VolumeGuid                                      |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1998                               |
| System-ID-GUID    | 85e5a5cf-dcee-4075-9cfd-ac9db6a2f245                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
