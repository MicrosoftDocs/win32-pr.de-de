# <a name="ms-fve-volumeguid-attribute"></a>ms-FVE-VolumeGuid-Attribut

Enthält die GUID, die einem von BitLocker unterstützten Datenträgervolumen zugeordnet ist.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | ms-FVE-VolumeGuid                                     |
| Ldap-Anzeigename | msFVE-VolumeGuid                                      |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | \-                                                    |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1998                               |
| System-Id-Guid    | 85e5a5cf-dcee-4075-9cfd-ac9db6a2f245                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
