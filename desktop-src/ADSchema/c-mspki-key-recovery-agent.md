# <a name="ms-pki-key-recovery-agent-class"></a>ms-PKI-Key-Recovery-Agent-Klasse

Während der Einrichtung des Zertifikatservers wird für jeden installierten Cert-Server (mit einem eindeutigen allgemeinen Namen) eine SCHLÜSSELWIEDERHERSTELLUNGS-Objektinstanz (Key Recovery Agent) erstellt. Wenn zwei Zertifizierungsstellen während der Einrichtung der Zertifizierungsstelle den gleichen allgemeinen Namen erhalten haben, wird eine einzelne INSTANZ des OBJEKTS gemeinsam verwendet.



| Eingabe | Wert |
|-------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Key-Recovery-Agent                                                                                                                                                    |
| Ldap-Anzeigename | msPKI-Key-Recovery-Agent                                                                                                                                                     |
| Aktualisieren von Berechtigungen  | Ein Administrator, der eine Zertifizierungsstelle installiert, muss in der Lage sein, eine EIGENSTÄNDIGE-Instanz im ZIP-Container zu erstellen. Installierte Zertifikatserver müssen in der Lage sein, das userCertificate-Attribut zu aktualisieren. |
| Updatehäufigkeit  | Einige Zertifikate werden höchstens alle paar Monate hinzugefügt.                                                                                                                   |
| Schema-ID-GUID    | 26ccf238-a08e-4b86-9a82-a8c9ac7ee5cb                                                                                                                                         |
