# <a name="ms-dns-signing-key-descriptors-attribute"></a>MS-DNS-Signing-Key-Descriptors-Attribut

Ein Attribut, das den Satz von DNSSEC-Signatur Schlüssel Deskriptoren (SKDS) enthält, die vom DNS-Server verwendet werden, um Schlüssel zu generieren und die DNS-Zone zu signieren.

| LDAP-Display-Name | MSDNs-signingkeydescriptors                           |

| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.2143                               |
| System-ID-GUID    | 3443d8cd-e5b6-4b098-659a0214a079                  |
