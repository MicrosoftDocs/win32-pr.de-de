# <a name="file-ext-priority-attribute"></a>File-ext-priority-Attribut

Dieses Attribut enthält eine Liste von Dateinamen Erweiterungen in einem Anwendungspaket und die zugehörigen Prioritäten.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Datei-ext-Priorität                           |
| LDAP-Display-Name | fileextpriority                             |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.816                      |
| System-ID-GUID    | d9e18315-8939-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
