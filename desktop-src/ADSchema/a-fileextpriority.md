# <a name="file-ext-priority-attribute"></a>File-Ext-Priority-Attribut

Dieses Attribut enthält eine Liste der Dateierweiterungen in einem Anwendungspaket und die zugehörigen Prioritäten.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | File-Ext-Priority                           |
| Ldap-Anzeigename | fileExtPriority                             |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | \-                                          |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.816                      |
| System-Id-Guid    | d9e18315-8939-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
