# <a name="nisnetgrouptriple-attribute"></a>das Attribut "-netgrouptriple"

Stellt einen Eintrag aus einer Netzgruppen Herkunft-Zuordnung dar.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | nisnetgrouptriple                    |
| LDAP-Display-Name | nisnetgrouptriple                    |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.14                     |
| System-ID-GUID    | a8032e74-30ef-4ff5-affc-0fc217783fec |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
