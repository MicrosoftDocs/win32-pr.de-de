| CN                | RPC-NS-Interface-ID                         |
| LDAP-Display-Name | rpcnsinterfakeid                            |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |

| Attribute-Id      | 1.2.840.113556.1.4.115                      |
