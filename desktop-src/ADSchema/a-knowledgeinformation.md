# <a name="knowledge-information-attribute"></a>Knowledge-Information-Attribut

Gibt eine lesbare Beschreibung der Wissensquelle an, die von einem bestimmten Verzeichnis System-Agent verwaltet wird.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Knowledge-Information                       |
| LDAP-Display-Name | Wissens dgeinformation                        |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 2.5.4.2                                     |
| System-ID-GUID    | 1677581f -47b3-11d1-a9c3-0000b80367c1        |
| Syntax            | [**String(Teletex)**](s-string-teletex.md) |
