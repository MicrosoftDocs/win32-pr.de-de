# <a name="ms-ds-isrodc-attribute"></a>ms-DS-isrodc-Attribut

Zeigt an, ob ein Domänen Controller ein RODC ist.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ms-DS-isrodc                         |
| LDAP-Display-Name | MSDS-isrodc                          |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1960              |
| System-ID-GUID    | a8e8aa23-3e67-4af1-9d7a-2f1a1d633ac9 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
