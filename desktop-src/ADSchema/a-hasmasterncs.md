# <a name="has-master-ncs-attribute"></a>Has-Master-NCS-Attribut

Der Distinguished Name für die Benennungs Kontexte für den DC. Forward-Link für das Mastered-By-Attribut.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Has-Master-NCS                          |
| LDAP-Display-Name | hasmasterncs                            |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.14                   |
| System-ID-GUID    | bf967982-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
