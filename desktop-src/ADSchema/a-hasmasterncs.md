# <a name="has-master-ncs-attribute"></a>Has-Master-NCs-Attribut

Der Distinguished Name für die Benennungskontexte für den Domänencontroller. Forward-Link für das Mastered-By Attribut.

|-------------------|-----------------------------------------|
| CN                | Has-Master-NCs                          |
| Ldap-Anzeigename | hasMasterNCs                            |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | \-                                      |
| Updatehäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.14                   |
| System-Id-Guid    | bf967982-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
