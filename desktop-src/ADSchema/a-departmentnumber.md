# <a name="departmentnumber-attribute"></a>departmentnumber-Attribut

| LDAP-Display-Name | departmentnumber                            |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |

| System-ID-GUID    | be9ef6ee-cbc7-4f22-b27b-96967e7ee585        |
