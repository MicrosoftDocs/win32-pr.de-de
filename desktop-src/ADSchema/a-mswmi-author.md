# <a name="ms-wmi-author-attribute"></a>MS-WMI-Author-Attribut

Der Autor einer Instanz einer-Klasse.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-Autor                               |
| LDAP-Display-Name | mswap-Autor                                |
| Size              | Weniger als 25 Zeichen.                    |
| Berechtigung aktualisieren  | Gruppenrichtlinie-Administrator                  |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1623                     |
| System-ID-GUID    | 6366c0c1-6972-4e66-b3a5-1d52ad0c0547        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
