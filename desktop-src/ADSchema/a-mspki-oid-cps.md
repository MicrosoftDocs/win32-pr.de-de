# <a name="ms-pki-oid-cps-attribute"></a>MS-PKI-OID-CPS-Attribut

Die CPS (Certificate Policy-Anweisung) für die Richtlinien-OID des Unternehmens.



| Eingabe | Wert |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-OID-CPS                                                                             |
| LDAP-Display-Name | mspki-OID-CPS                                                                              |
| Size              | \-                                                                                         |
| Berechtigung aktualisieren  | Unternehmensadministrator                                                                   |
| Aktualisierungshäufigkeit  | Wenn eine neue Vorlage erstellt wird oder die Attribute vorhandener Vorlagen bearbeitet werden. |
| Attribute-Id      | 1.2.840.113556.1.4.1672                                                                    |
| System-ID-GUID    | 5f49940e-a79f-4a51-bb6f-3d446a54dc6b                                                       |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                |
