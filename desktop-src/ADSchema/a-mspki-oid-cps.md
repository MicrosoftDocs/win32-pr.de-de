# <a name="ms-pki-oid-cps-attribute"></a>ms-PKI-OID-CPS-Attribut

Die CPS (Certificate Policy Statement, Zertifikatrichtlinien-Anweisung) für die Richtlinien-OID des Unternehmensausstellers.



| Eingabe | Wert |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | ms-PKI-OID-CPS                                                                             |
| Ldap-Anzeigename | msPKI-OID-CPS                                                                              |
| Size              | \-                                                                                         |
| Aktualisieren von Berechtigungen  | Unternehmensadministrator                                                                   |
| Updatehäufigkeit  | Jedes Mal, wenn eine neue Vorlage erstellt wird oder die Attribute einer vorhandenen Vorlage bearbeitet werden. |
| Attribute-Id      | 1.2.840.113556.1.4.1672                                                                    |
| System-Id-Guid    | 5f49940e-a79f-4a51-bb6f-3d446a54dc6b                                                       |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                |
