# <a name="default-class-store-attribute"></a>Default-Class-Store-Attribut

Der Standardklassen Speicher für einen angegebenen Benutzer.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Standard-Class-Store                     |
| LDAP-Display-Name | defaultclassstore                       |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.213                  |
| System-ID-GUID    | bf967948-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
