Das **msnpcalledstationid** -Attribut wird intern verwendet. Ändern Sie diesen Wert nicht direkt.

| CN                | msnpcalledstationid                  |
| LDAP-Display-Name | msnpcalledstationid                  |
| Size              | \-                                   |
