# <a name="netboot-server-attribute"></a>NetBoot-Server-Attribut

Der Distinguished Name eines netboot-Servers.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | NetBoot-Server                          |
| LDAP-Display-Name | netbootserver                           |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.        |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.860                  |
| System-ID-GUID    | 07383081-91df-11d1-AEbc-0000e80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
