# <a name="ms-dfsr-rdcenabled-attribute"></a>MS-DFSR-rdcenabled-Attribut

Enthält den Wert, der angibt, ob RDC aktiviert oder deaktiviert ist.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-rdcenabled                   |
| LDAP-Display-Name | msdfsr-rdcenabled                    |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.19           |
| System-ID-GUID    | e3b44e05-f4a7-4078-a730-f48670a743f8 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
