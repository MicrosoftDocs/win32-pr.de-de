# <a name="acs-identity-name-attribute"></a>ACS-Identity-Name-Attribut

Dieses Attribut enthält den DN eines Benutzers oder einer Organisationseinheit und ist die Identität eines Benutzers oder einer Gruppe von Benutzern, auf die diese QoS-Richtlinie angewendet wird.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ACS-Identitäts Name                           |
| LDAP-Display-Name | acsidentityname                             |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänen Administrator                        |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.784                      |
| System-ID-GUID    | dab029b6-ddf7-11d1-90a5-00c04fd91ab1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
