# <a name="ms-tapi-ip-address-attribute"></a>MS-TAPI-IP-address-Attribut

IP-Adressen eines TAPI-Client Computers, bei dem ein Benutzer angemeldet ist. Dieses Attribut kann als generisches Attribut für Computer-IP-Adressen verwendet werden.



| Eingabe | Wert |
|-------------------|------------------------------------------------------------|
| CN                | MS-TAPI-IP-Adresse                                         |
| LDAP-Display-Name | MSTAPI-IPAddress                                           |
| Size              | Jede IP-Adresse wird als Zeichenfolge in einer. B. C. D-Notation gespeichert. |
| Berechtigung aktualisieren  | Es sind keine besonderen Berechtigungen erforderlich.                            |
| Aktualisierungshäufigkeit  | Normalerweise sehr niedrig.                                        |
| Attribute-Id      | 1.2.840.113556.1.4.1701                                    |
| System-ID-GUID    | efd7d7f7-178e-4767-87fa-f8a16b840544                       |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                |
