# <a name="ms-tapi-ip-address-attribute"></a>ms-TAPI-Ip-Address-Attribut

IP-Adressen eines TAPI-Clientcomputers, bei dem ein Benutzer angemeldet ist. Dieses Attribut kann als generisches Attribut zum Speichern von Computer-IP-Adressen verwendet werden.



| Eingabe | Wert |
|-------------------|------------------------------------------------------------|
| CN                | ms-TAPI-Ip-Address                                         |
| Ldap-Anzeigename | msTAPI-IpAddress                                           |
| Size              | Jede IP-Adresse wird als Zeichenfolge in A.B.C.D-Notation gespeichert. |
| Aktualisieren von Berechtigungen  | Es sind keine besonderen Berechtigungen erforderlich.                            |
| Updateh√§ufigkeit  | In der Regel sehr niedrig.                                        |
| Attribute-Id      | 1.2.840.113556.1.4.1701                                    |
| System-Id-Guid    | efd7d7f7-178e-4767-87fa-f8a16b840544                       |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                |
