# <a name="implemented-categories-attribute"></a>Implemented-Categories-Attribut

Dieses Attribut enthält eine Liste der Komponenten Kategorie-IDs, die von diesem Objekt implementiert werden.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Implemented-Categories                                |
| LDAP-Display-Name | implementedcategories                                 |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.320                                |
| System-ID-GUID    | 7d6c0e92-7E20-11D0-afd6-00c04f d930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
