# <a name="bad-pwd-count-attribute"></a>Ungültiges-pwd-count-Attribut

Gibt an, wie oft der Benutzer versucht hat, sich mit einem falschen Kennwort beim Konto anzumelden. Der Wert 0 gibt an, dass der Wert unbekannt ist.



| Eingabe | Wert |
|-------------------|-------------------------------------------|
| CN                | Falsch-pwd-Anzahl                             |
| LDAP-Display-Name | BadPwdCount                               |
| Size              | 4 Bytes                                   |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.          |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn der Benutzer ein ungültiges Kennwort eingibt. |
| Attribute-Id      | 1.2.840.113556.1.4.12                     |
| System-ID-GUID    | bf96792e-0de6-11d0-a285-00aa003049e2      |
| Syntax            | [**Enumeration**](s-enumeration.md)      |

## <a name="remarks"></a>Bemerkungen

Dieses Attribut wird nicht repliziert und auf jedem Domänen Controller in der Domäne separat verwaltet.

Dieses Attribut wird auf einem bestimmten Domänen Controller zurückgesetzt, wenn sich der Benutzer erfolgreich auf diesem Domänen Controller anmeldet.
