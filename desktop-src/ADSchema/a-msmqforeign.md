# <a name="msmq-foreign-attribute"></a>MSMQ-Foreign-Attribut

Gibt an, ob es sich bei diesem Computer um einen MSMQ-Computer handelt.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MSMQ-Foreign                         |
| LDAP-Display-Name | msmqforeign                          |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.934               |
| System-ID-GUID    | 9a0dc32f-C100-11d1-bbc5-0080c76670c0 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
