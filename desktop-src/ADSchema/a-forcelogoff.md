# <a name="force-logoff-attribute"></a>Force-Logoff-Attribut

Wird zum Berechnen der Anfangszeit in samigetaccountrestrictions verwendet. Die Abmelde Zeit minus "Abmelde Zeit" ist gleich der Start Zeit.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Force-Logoff                         |
| LDAP-Display-Name | ForceLogoff                          |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.39                |
| System-ID-GUID    | bf967977-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervall**](s-interval.md)       |
