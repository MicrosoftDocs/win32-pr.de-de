# <a name="ms-pki-template-minor-revision-attribute"></a>ms-PKI-Template-Minor-Revision-Attribut

Verfolgt Attribute in der Klasse nach, die sich ändern. Dies löst jedoch keine automatische Registrierung aus.



| Eingabe | Wert |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Template-Minor-Revision                                                                    |
| Ldap-Anzeigename | msPKI-Template-Minor-Revision                                                                     |
| Size              | 4 Bytes                                                                                           |
| Aktualisieren von Berechtigungen  | Domänenadministrator                                                                              |
| Updatehäufigkeit  | Wenn das Zertifikatvorlagenobjekt (ms-PKI-Certificate-Template) bearbeitet, erstellt oder geklont wird. |
| Attribute-Id      | 1.2.840.113556.1.4.1435                                                                           |
| System-ID-GUID    | 13f5236c-1884-46b1-b5d0-484e38990d58                                                              |
| Syntax            | [**Enumeration**](s-enumeration.md)                                                              |
