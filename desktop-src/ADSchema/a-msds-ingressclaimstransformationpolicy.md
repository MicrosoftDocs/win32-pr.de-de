# <a name="ms-ds-ingress-claims-transformation-policy-attribute"></a>ms-DS-Ingress-Claims-Transformation-Policy-Attribut

Dies ist ein Link zu einem Anspruchs Transformations-Richtlinien Objekt für die Eingangs Ansprüche (Ansprüche, die in diese Gesamtstruktur eintreten) von der vertrauenswürdigen Domäne. Dies gilt nur für eine ausgehende oder bidirektionale Gesamtstruktur übergreifende Vertrauensstellung. Wenn dieser Link fehlt, werden alle eingehenden Ansprüche gelöscht.
