# <a name="ms-ds-ingress-claims-transformation-policy-attribute"></a>ms-DS-Ingress-Claims-Transformation-Policy-Attribut

Dies ist ein Link zu einem Anspruchstransformationsrichtlinienobjekt für die Eingangsansprüche (Ansprüche, die in diese Gesamtstruktur eingegeben werden) aus der vertrauenswürdigen Domäne. Dies gilt nur für eine ausgehende oder bidirektionale gesamtstrukturübergreifende Vertrauensstellung. Wenn dieser Link nicht vorhanden ist, werden alle Eingangsansprüche gelöscht.
