# <a name="ms-ds-all-users-trust-quota-attribute"></a>MS-DS-all-users-Trust-Quota-Attribut

Wird verwendet, um ein kombiniertes Benutzer Kontingent für die Gesamtanzahl der Trusted-Domain Objekte zu erzwingen, die mit dem neuen Zugriffsrecht "Create-Inbound-Forest-Trust" erstellt wurden.



| Eingabe | Wert |
|-------------------|-------------------------------------------|
| CN                | MS-DS-all-users-Trust-Quota               |
| LDAP-Display-Name | MSDS-alluserstrustquota                   |
| Size              | \-                                        |
| Berechtigung aktualisieren  | Domänen Administrator                      |
| Aktualisierungshäufigkeit  | Bei der Gesamtstruktur Erstellung und selten danach. |
| Attribute-Id      | 1.2.840.113556.1.4.1789                   |
| System-ID-GUID    | d3aa4a5c-4e03-4810-97aa-2b339e7a434b      |
| Syntax            | [**Enumeration**](s-enumeration.md)      |
