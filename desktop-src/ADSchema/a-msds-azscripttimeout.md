# <a name="ms-ds-az-script-timeout-attribute"></a>ms-DS-AZ-Script-Timeout-Attribut

Die maximale Zeit in Millisekunden, die auf das Abschließen der Überwachung einer bestimmten Richtlinie durch ein Skript gewartet werden soll.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | ms-DS-AZ-Script-Timeout                 |
| LDAP-Display-Name | MSDS-azscripttimeout                    |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Azrollen-Administrator                           |
| Aktualisierungshäufigkeit  | Während der Initialisierung oder Richtlinien Änderung. |
| Attribute-Id      | 1.2.840.113556.1.4.1797                 |
| System-ID-GUID    | 87d0b41-2c8b-41b6-b972-11b-bbd50d6b0    |
| Syntax            | [**Enumeration**](s-enumeration.md)    |
