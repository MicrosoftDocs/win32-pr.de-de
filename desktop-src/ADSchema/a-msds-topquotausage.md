# <a name="ms-ds-top-quota-usage-attribute"></a>ms-DS-Top-Quota-Usage-Attribut

Die Liste der obersten Kontingent Benutzer, die sich zurzeit in der Verzeichnis Datenbank befinden, geordnet nach absteigender Kontingent Auslastung.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Top-Quota-Usage                       |
| LDAP-Display-Name | MSDS-topquotausage                          |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1850                     |
| System-ID-GUID    | 7b7cce4f-f1f5-4bb6-b7eb-23504af19e75        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
