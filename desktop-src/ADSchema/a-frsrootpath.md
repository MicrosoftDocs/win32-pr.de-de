# <a name="frs-root-path-attribute"></a>FRS-root-path-Attribut

Der Pfad zum Stamm der replizierten Dateisystem Struktur.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | FRS-Root-Pfad                               |
| LDAP-Display-Name | frsrootpath                                 |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.487                      |
| System-ID-GUID    | 1be8f174-a9ff-11D0-afe2-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
