# <a name="domain-replica-attribute"></a>Domain-Replica-Attribut

Unicode-Zeichen folgen Attribut gibt die Liste der Windows NT 4,0-Replikations Domänen Controller an.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Domain-Replica                              |
| LDAP-Display-Name | domainreplica                               |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.158                      |
| System-ID-GUID    | bf96795e-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
