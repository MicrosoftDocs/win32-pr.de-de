# <a name="domain-replica-attribute"></a>Domain-Replica-Attribut

Unicode-Zeichenfolgenattribut gibt die Liste der Windows NT 4.0-Replikationsdomänencontroller an.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Domain-Replica                              |
| Ldap-Anzeigename | domainReplica                               |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | \-                                          |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.158                      |
| System-Id-Guid    | bf96795e-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
