# <a name="acs-non-reserved-tx-size-attribute"></a>Attribut "ACS-Non-reserved-size"

Die tokenbucket-Größe, die eine Anwendung verwenden kann, bevor eine Reservierung erfolgt.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-nicht reserviert-TX-Größe             |
| LDAP-Display-Name | acsnonreservedtxsize                 |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.898               |
| System-ID-GUID    | f072230d-aef5-11d1-bdcf-0000f80367c1 |
| Syntax            | [**Intervall**](s-interval.md)       |
