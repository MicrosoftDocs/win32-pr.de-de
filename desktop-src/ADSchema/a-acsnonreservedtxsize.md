# <a name="acs-non-reserved-tx-size-attribute"></a>ACS-Non-Reserved-Tx-Size-Attribut

Die Token-Bucketgröße, die eine Anwendung verwenden kann, bevor eine Reservierung eingerichtet wird.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-Non-Reserved-Tx-Size             |
| Ldap-Anzeigename | aCSNonReservedTxSize                 |
| Size              | 8 Bytes                              |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.898               |
| System-ID-GUID    | f072230d-aef5-11d1-bdcf-0000f80367c1 |
| Syntax            | [**Intervall**](s-interval.md)       |
