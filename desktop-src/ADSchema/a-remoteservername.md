# <a name="remote-server-name-attribute"></a>Remote Server Name-Attribut

Wird immer dann verwendet, wenn ein oder mehrere Computernamen gespeichert werden müssen.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Remote Server Name                          |
| LDAP-Display-Name | Remote Servername                            |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.105                      |
| System-ID-GUID    | bf967a12-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
