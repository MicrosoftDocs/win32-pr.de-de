# <a name="next-level-store-attribute"></a>Next-Level-Store-Attribut

Dieses Attribut gibt den nächsten klassenspeicher an, der durchsucht werden soll.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Nächste ebener Store                        |
| Ldap-Anzeigename | nextLevelStore                          |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | \-                                      |
| Updatehäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.214                  |
| System-Id-Guid    | bf9679da-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
