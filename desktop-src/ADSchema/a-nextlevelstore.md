# <a name="next-level-store-attribute"></a>Next-Level-Store-Attribut

Dieses Attribut gibt den nächsten zu durchsuchenden Klassen Speicher an.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Next-Level-Store                        |
| LDAP-Display-Name | nextlevelstore                          |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.214                  |
| System-ID-GUID    | bf9679da-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
