# <a name="ms-wmi-intflags2-attribute"></a>MS-WMI-intFlags2-Attribut

Das **MS-WMI-intFlags2-** Attribut ist für die interne Verwendung reserviert.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MS-WMI-intFlags2                     |
| LDAP-Display-Name | mswap-intFlags2                      |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1679              |
| System-ID-GUID    | 075a42c9-c55a-45b1-ac93-eb086b31f |
| Syntax            | [**Enumeration**](s-enumeration.md) |
