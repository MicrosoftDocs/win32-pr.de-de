# <a name="documentversion-attribute"></a>DocumentVersion-Attribut

Der DocumentVersion-Attributtyp gibt die Versionsnummer eines Dokuments an.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | DocumentVersion                             |
| LDAP-Display-Name | DocumentVersion                             |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.13                  |
| System-ID-GUID    | 94b3a8a9-D613-4cec-9aad-5 b43        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
