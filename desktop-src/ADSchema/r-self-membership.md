# <a name="self-membership-validated-writes"></a>Self-Membership überprüfte Schreibvorgänge

Die Schreibberechtigung wurde überprüft, um das Aktualisieren der Mitgliedschaft einer Gruppe in Bezug auf das Hinzufügen/Entfernen eines eigenen Kontos zu ermöglichen.



| Eingabe | Wert |
|--------------|--------------------------------------|
| CN           | Self-Membership                      |
| Anzeigename | Add/Remove self as member (Selbst als Mitglied hinzufügen/entfernen)            |
| Rechte-GUID  | bf9679c0-0de6-11d0-a285-00aa003049e2 |
