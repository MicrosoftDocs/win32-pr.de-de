# <a name="non-security-member-bl-attribute"></a>Nicht-Security-Member-BL-Attribut

Liste der nicht Sicherheits Mitglieder für eine Exchange-Verteilerliste.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Nicht sicherheitsmitglied-BL                  |
| LDAP-Display-Name | nonsecuritymitgliedbl                     |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.        |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.531                  |
| System-ID-GUID    | 52458019-ca6a-11D0-affinf-0000f 80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
