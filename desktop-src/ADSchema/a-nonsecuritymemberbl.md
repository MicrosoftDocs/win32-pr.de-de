# <a name="non-security-member-bl-attribute"></a>Attribut "Non-Security-Member-BL"

Liste der Nichtsicherheitsmitglieder für eine Exchange Verteilerliste.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Non-Security-Member-BL                  |
| Ldap-Anzeigename | nonSecurityMemberBL                     |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.        |
| Updatehäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.531                  |
| System-ID-GUID    | 52458019-ca6a-11d0-afff-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
