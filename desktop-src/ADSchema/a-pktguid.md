# <a name="pkt-guid-attribute"></a>PKT-Guid-Attribut

Eindeutige ID einer angegebenen verteiltes Dateisystem Partitions-Wissens Tabelle.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | PKT-Guid                                              |
| LDAP-Display-Name | pktguid                                               |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.205                                |
| System-ID-GUID    | 8447f 0-1027-11D0-a05f -00aa006c33ed                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
