# <a name="userclass-attribute"></a>userclass-Attribut

Der userclass-Attributtyp gibt eine Kategorie von Computerbenutzer an.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | userClass                                   |
| LDAP-Display-Name | userClass                                   |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.8                   |
| System-ID-GUID    | 11732a8a-e14d-4cc5-b92a-d93f 51c6d8e4        |
