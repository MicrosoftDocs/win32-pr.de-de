# <a name="ds-execute-intentions-script-extended-right"></a>DS-Execute-Absichten-Skript erweitert rechts

Steuern des Zugriffsrechts, das dem Partitions Container erteilt werden soll, sodass der Rendom.exe-oder Vorbereitungs Vorgang in einer Domänen Umbenennung verwendet werden kann. Dieses Zugriffsrecht für das Steuerelement wird auch als nur Überwachungs Recht angezeigt, wenn der Vorgang Rendom.exe oder Schritt ausführen ausgeführt wird.



| Eingabe | Wert |
|--------------|--------------------------------------|
| CN           | DS-Execute-Absichten-Skript         |
| Anzeigename | Skript für Gesamtstruktur Update ausführen         |
| Rights-GUID  | 2b16c4a5-b98e-432c-952a-cb388ba33f 2E |
