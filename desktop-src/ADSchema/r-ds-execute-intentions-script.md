# <a name="ds-execute-intentions-script-extended-right"></a>DS-Execute-Intentions-Script erweitert rechts

Steuern Sie das Zugriffsrecht, das dem Partitionscontainer gewährt werden sollte, wodurch der Rendom.exe- oder Vorbereitungsvorgang in einer Domänenumbenennung verwendet werden kann. Dieses Steuerelementzugriffsrecht wird auch als Nur-Überwachung-Recht angezeigt, wenn der Rendom.exe oder Schritt ausführen ausgeführt wird.



| Eingabe | Wert |
|--------------|--------------------------------------|
| CN           | DS-Execute-Intentions-Script         |
| Anzeigename | Skript zum Aktualisieren der Gesamtstruktur ausführen         |
| Rights-GUID  | 2f16c4a5-b98e-432c-952a-cb388ba33f2e |
