# <a name="phone-ip-primary-attribute"></a>"Phone-IP-Primary"-Attribut

Die TCP/IP-Adresse für das Telefon. Wird von Telefoniediensten verwendet.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Telefon-IP-primär                                                                 |
| LDAP-Display-Name | ipPhone                                                                          |
| Size              | \-                                                                               |
| Berechtigung aktualisieren  | Domänen Administrator oder Konto Besitzer.                                           |
| Aktualisierungshäufigkeit  | Wenn der Benutzerdaten Satz erstellt und die Telefonnummer geändert werden muss. |
| Attribute-Id      | 1.2.840.113556.1.4.721                                                           |
| System-ID-GUID    | 4d146 E4A-48d4-11d1-a9c3-0000e80367c1                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
