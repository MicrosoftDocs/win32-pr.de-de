# <a name="ms-wmi-id-attribute"></a>MS-WMI-ID-Attribut

Eine eindeutige ID für eine Objektinstanz. Im globalen Katalog indiziert.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-ID                                   |
| LDAP-Display-Name | mswap-ID                                    |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.            |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1627                     |
| System-ID-GUID    | 9339a803-94b8-47f7-9123-a853b9ff7e45        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
