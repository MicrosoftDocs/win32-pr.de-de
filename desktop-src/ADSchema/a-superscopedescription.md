# <a name="super-scope-description-attribute"></a>Superscope-Description-Attribut

Dieses Attribut stellt eine Beschreibung für eine Bereichs Gruppierung bereit.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Superscope-Beschreibung                     |
| LDAP-Display-Name | superscopedescription                       |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.711                      |
| System-ID-GUID    | 963d274c-48be-11d1-a9c3-0000b80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
