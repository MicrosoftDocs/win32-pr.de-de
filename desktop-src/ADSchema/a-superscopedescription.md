# <a name="super-scope-description-attribute"></a>Super-Scope-Description-Attribut

Dieses Attribut enthält eine Beschreibung für eine Bereichsgruppierung.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Super-Scope-Description                     |
| Ldap-Anzeigename | superScopeDescription                       |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | \-                                          |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.711                      |
| System-Id-Guid    | 963d274c-48be-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
