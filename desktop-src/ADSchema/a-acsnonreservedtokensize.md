# <a name="acs-non-reserved-token-size-attribute"></a>ACS-Non-reserved-Token-size-Attribut

Das Attribut " **ACS-Non-reserved-Token-size** " ist nur für die interne Verwendung vorgesehen. Basierend auf RFC2814.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-nicht reserviert-Tokengröße          |
| LDAP-Display-Name | acsnonreservedtekensize              |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1319              |
| System-ID-GUID    | a916d7c9-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**Intervall**](s-interval.md)       |
