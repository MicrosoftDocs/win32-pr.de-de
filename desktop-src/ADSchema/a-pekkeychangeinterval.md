# <a name="pek-key-change-interval-attribute"></a>PEK-Key-Change-Interval-Attribut

Änderungs Intervall für das Kennwort-Verschlüsselungsschlüssel.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | PEK-Key-Change-Interval              |
| LDAP-Display-Name | "Peer keychangeinterval"                 |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.866               |
| System-ID-GUID    | 07383084-91df-11d1-AEbc-0000e80367c1 |
| Syntax            | [**Intervall**](s-interval.md)       |
