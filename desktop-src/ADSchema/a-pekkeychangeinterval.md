# <a name="pek-key-change-interval-attribute"></a>Pek-Key-Change-Interval-Attribut

Änderungsintervall für Kennwortverschlüsselungsschlüssel.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Pek-Key-Change-Interval              |
| Ldap-Anzeigename | pekKeyChangeInterval                 |
| Size              | 8 Bytes                              |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.     |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.866               |
| System-Id-Guid    | 07383084-91df-11d1-aebc-0000f80367c1 |
| Syntax            | [**Intervall**](s-interval.md)       |
