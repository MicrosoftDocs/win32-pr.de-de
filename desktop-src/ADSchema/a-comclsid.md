# <a name="com-clsid-attribute"></a>COM-CLSID-Attribut

Dieses Attribut enthält die GUID, die dieser Objektklasse zugeordnet ist.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | COM-CLSID                                   |
| LDAP-Display-Name | comclsid                                    |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.249                      |
| System-ID-GUID    | 281416d9-1968-11D0-a28f -00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
