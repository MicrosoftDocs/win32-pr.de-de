# <a name="ms-sql-unicodesortorder-attribute"></a>MS-SQL-unicodesortor-Attribut

Die Unicode-Sortierreihenfolge für die aktuelle Instanz von SQL Server.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MS-SQL-unicodesortor              |
| LDAP-Display-Name | MS-SQL-unicodesortor              |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | Domänen Administrator                 |
| Aktualisierungshäufigkeit  | Beim Systemstart.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1372              |
| System-ID-GUID    | 72dc918a-CCEE-11d2-9993-0000f 87a57d4 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
