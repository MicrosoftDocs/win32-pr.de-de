# <a name="user-shared-folder-other-attribute"></a>Benutzer-Shared-Folder-anderes Attribut

Gibt einen UNC-Pfad zum Ordner "weitere freigegebene Dokumente" des Benutzers an. Der Pfad muss ein Netzwerk-UNC-Pfad des Formular **\\\\** _Server_ *_\\_* _Freigabe_- *_\\_* _Verzeichnisses_ sein. Dieser Wert kann eine NULL-Zeichenfolge sein.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------------------------|
| CN                | Benutzer-Shared-Folder-Sonstiges                                                          |
| LDAP-Display-Name | usersharedfolderother                                                             |
| Size              | \-                                                                                |
| Berechtigung aktualisieren  | Domänen Administrator oder Konto Besitzer.                                            |
| Aktualisierungshäufigkeit  | Wenn der Benutzerdaten Satz erstellt und der freigegebene Ordner geändert werden muss. |
| Attribute-Id      | 1.2.840.113556.1.4.752                                                            |
| System-ID-GUID    | 9a9a0220-4a5b-11d1-a9c3-0000t80367c1                                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                       |
