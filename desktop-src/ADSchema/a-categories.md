# <a name="categories-attribute"></a>Categories-Attribut

Liste der Kategorie-IDs (GUIDs) für Kategorien, die auf diese Anwendung angewendet werden.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Kategorien                                  |
| LDAP-Display-Name | categories                                  |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.672                      |
| System-ID-GUID    | 7bf dcb7e-4807-11d1-a9c3-0000b80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
