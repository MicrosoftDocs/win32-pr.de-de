# <a name="object-class-category-attribute"></a>Object-Class-Category-Attribut

Dieses Attribut enthält den Klassentyp, z. b. abstrakt, Zusatz oder strukturiert.



| Eingabe | Wert |
|-------------------|---------------------------------------------------------------------------------|
| CN                | Object-Class-Category                                                           |
| LDAP-Display-Name | objectClassCategory                                                             |
| Size              | 4 Bytes. Strukturell 1, abstract 2, Zusatz 3. Die Klasse 88, 0 sollte nicht verwendet werden. |
| Berechtigung aktualisieren  | Dieser Wert wird vom Designer des-Objekts festgelegt.                                |
| Aktualisierungshäufigkeit  | Dieser Wert sollte sich nie ändern.                                                 |
| Attribute-Id      | 1.2.840.113556.1.2.370                                                          |
| System-ID-GUID    | bf9679e6-0de6-11d0-a285-00aa003049e2                                            |
| Syntax            | [**Enumeration**](s-enumeration.md)                                            |
