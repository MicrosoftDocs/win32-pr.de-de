# <a name="ms-wmi-intmax-attribute"></a>MS-WMI-IntMax-Attribut

Der maximale Wert für ein WMI-32-Bit-Ganzzahl-Parameter Objekt.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MS-WMI-IntMax                        |
| LDAP-Display-Name | mswap-IntMax                         |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | Gruppenrichtlinie-Administrator           |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1629              |
| System-ID-GUID    | fb920c2c-f294-4426-8ac1-d24b42aa2bce |
| Syntax            | [**Enumeration**](s-enumeration.md) |
