# <a name="ms-dfsr-version-attribute"></a>MS-DFSR-Version-Attribut

Enthält die Versionsnummer des DFS-Replikations Dienstanbieter (verteiltes Dateisystem).



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-DFSR-Version                             |
| LDAP-Display-Name | msdfsr-Version                              |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.1                   |
| System-ID-GUID    | 1a861408-38c3-49ea-Ba75-85481a77c655        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
