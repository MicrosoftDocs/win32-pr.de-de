# <a name="shell-property-pages-attribute"></a>Shell-Eigenschaften Seiten-Attribut

Die Bestellnummer und die GUID von Eigenschaften Seiten zum Verwalten von Active Directory-Objekten. Auf diese Eigenschaften Seiten kann von der Windows-Shell zugegriffen werden. Weitere Informationen finden Sie im Dokument Erweitern der Benutzeroberfläche für Verzeichnisobjekte.



| Eingabe | Wert |
|-------------------|------------------------------------------------|
| CN                | Shell-Eigenschaften Seiten                           |
| LDAP-Display-Name | shellpropertypages                             |
| Size              | \-                                             |
| Berechtigung aktualisieren  | Domänen Administrator oder Anwendungsentwickler. |
| Aktualisierungshäufigkeit  | Immer dann, wenn eine Eigenschaften Seite hinzugefügt oder entfernt wird.  |
| Attribute-Id      | 1.2.840.113556.1.4.563                         |
| System-ID-GUID    | 52458039-ca6a-11D0-affinf-0000f 80367c1           |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)    |
