# <a name="shell-property-pages-attribute"></a>Shell-Property-Pages-Attribut

Die Bestellnummer und GUID der Eigenschaftenseiten zum Verwalten von Active Directory-Objekten. Auf diese Eigenschaftenseiten kann über die Windows Shell zugegriffen werden. Weitere Informationen finden Sie im Dokument Erweitern der Benutzeroberfläche für Verzeichnisobjekte.



| Eingabe | Wert |
|-------------------|------------------------------------------------|
| CN                | Shell-Property-Pages                           |
| Ldap-Anzeigename | shellPropertyPages                             |
| Size              | \-                                             |
| Aktualisieren von Berechtigungen  | Domänenadministrator oder Anwendungsentwickler. |
| Updatehäufigkeit  | Immer dann, wenn eine Eigenschaftenseite hinzugefügt oder entfernt wird.  |
| Attribute-Id      | 1.2.840.113556.1.4.563                         |
| System-ID-GUID    | 52458039-ca6a-11d0-afff-0000f80367c1           |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)    |
