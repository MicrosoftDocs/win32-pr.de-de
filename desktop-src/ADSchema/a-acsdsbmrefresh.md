# <a name="acs-dsbm-refresh-attribute"></a>ACS-DSBM-Refresh-Attribut

Dieses Attribut enthält den Zeit Geber Wert des Intervalls, der bestimmt, wann der designierte subnetzbandbreiten-Manager (DSBM) eine Aktualisierungs Nachricht ( \_ \_ DSBM) an alle subnetzbandbreiten-Manager in einer Domäne sendet.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-DSBM-aktualisieren                     |
| LDAP-Display-Name | acsdsbmrefresh                       |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.777               |
| System-ID-GUID    | 1cb3559b-56d0-11d1-a9c6-0000b80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
