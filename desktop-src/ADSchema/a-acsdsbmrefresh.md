# <a name="acs-dsbm-refresh-attribute"></a>ACS-DSBM-Refresh-Attribut

Dieses Attribut enthält den Wert des Intervallzeiters, der bestimmt, wann der Designated Subnet Bandwidth Manager (DSBM) eine Aktualisierungsmeldung (I \_ AM DSBM) an alle Subnetzbandbreiten-Manager in einer Domäne \_ sendet.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-DSBM-Refresh                     |
| Ldap-Anzeigename | aCSDSBMRefresh                       |
| Size              | 4 Bytes                              |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.777               |
| System-Id-Guid    | 1cb3559f-56d0-11d1-a9c6-0000f80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
