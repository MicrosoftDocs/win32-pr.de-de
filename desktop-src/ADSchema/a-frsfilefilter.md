# <a name="frs-file-filter-attribute"></a>FRS-File-Filter-Attribut

Eine Liste der von der Dateireplikation ausgeschlossenen Dateierweiterungen.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | FRS-File-Filter                             |
| Ldap-Anzeigename | fRSFileFilter                               |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | Domänenadministrator                        |
| Updatehäufigkeit  | Wenn die Replikation eingerichtet ist.                  |
| Attribute-Id      | 1.2.840.113556.1.4.483                      |
| System-Id-Guid    | 1be8f170-a9ff-11d0-afe2-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
