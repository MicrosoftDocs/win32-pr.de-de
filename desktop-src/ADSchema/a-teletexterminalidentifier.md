# <a name="teletex-terminal-identifier-attribute"></a>Teletex-terminalbezeichnerattribut

Gibt den Teletex-Terminal Bezeichner und optional Parameter für ein einem-Objekt zugeordnetes Teletex-Terminal an.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Teletex-Terminal-Identifier                           |
| LDAP-Display-Name | teletexterminalidentifier                             |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 2.5.4.22                                              |
| System-ID-GUID    | bf967a4a-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
