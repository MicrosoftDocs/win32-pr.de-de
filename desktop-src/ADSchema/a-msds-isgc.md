# <a name="ms-ds-isgc-attribute"></a>ms-DS-isgc-Attribut

Identifiziert den Status des globalen Katalogs auf dem DC.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ms-DS-isgc                           |
| LDAP-Display-Name | MSDS-isgc                            |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1959              |
| System-ID-GUID    | 1df5cf33-0fe5-499e-90e1-e94b42718a46 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
