| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Is-Privilege-Inhaber                     |
| LDAP-Display-Name | isprivilegeholder                       |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.638                  |
| System-ID-GUID    | 19405b9c-3cfa-11d1-a9c0-0000f 80367c1    |
