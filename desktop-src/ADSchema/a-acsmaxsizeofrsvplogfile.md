# <a name="acs-max-size-of-rsvp-log-file-attribute"></a>ACS-maximum-size-of-RSVP-log-file-Attribut

Die maximale Größe einer RSVP-Protokolldatei in Byte.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-max-size-of-RSVP-Log-File        |
| LDAP-Display-Name | acsmaxsizeofrsvplogfile              |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.775               |
| System-ID-GUID    | 1cb3559d-56d0-11d1-a9c6-0000b80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
