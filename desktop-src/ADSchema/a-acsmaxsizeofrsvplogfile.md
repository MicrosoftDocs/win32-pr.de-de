# <a name="acs-max-size-of-rsvp-log-file-attribute"></a>ACS-Max-Size-Of-RSVP-Log-File-Attribut

Die maximale Größe einer RSVP-Protokolldatei in Bytes.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-Max-Size-of-RSVP-Log-File        |
| Ldap-Anzeigename | aCSMaxSizeOfRSVPLogFile              |
| Size              | 4 Bytes                              |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.775               |
| System-Id-Guid    | 1cb3559d-56d0-11d1-a9c6-0000f80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
