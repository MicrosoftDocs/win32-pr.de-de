# <a name="ms-exch-owner-bl-attribute"></a>ms-Exch-Owner-BL-Attribut

Der Rückwärtslink zum Owner-Attribut. Enthält eine Liste der Besitzer für ein Objekt.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | ms-Exch-Owner-BL                        |
| Ldap-Anzeigename | ownerBL                                 |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.        |
| Updatehäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.104                  |
| System-ID-GUID    | bf9679f4-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
