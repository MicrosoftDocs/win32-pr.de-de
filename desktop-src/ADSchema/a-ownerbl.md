# <a name="ms-exch-owner-bl-attribute"></a>ms-Exch-Owner-BL-Attribut

Der rückwärts Link zum Besitzer Attribut. Enthält eine Liste der Besitzer für ein Objekt.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | ms-Exch-Owner-BL                        |
| LDAP-Display-Name | Besitzer                                 |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.        |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.104                  |
| System-ID-GUID    | bf9679f4-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
