# <a name="modified-count-attribute"></a>Modified-Count-Attribut

Die Seriennummer für die Protokoll Änderungs Protokollierung.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Modified-Count                       |
| LDAP-Display-Name | modifiedcount                        |
| Size              | \-                                   |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.168               |
| System-ID-GUID    | bf9679c5-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervall**](s-interval.md)       |
