# <a name="initials-attribute"></a>Initials-Attribut

Enthält die Initialen für Teile des vollständigen Namens des Benutzers. Dies kann als mittlere Initialisierung in der -Windows Adressbuch.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Initialen                                    |
| Ldap-Anzeigename | Initials                                    |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | Domänenadministrator oder Kontobesitzer.      |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 2.5.4.43                                    |
| System-Id-Guid    | f0f8ff90-1191-11d0-a060-00aa006c33ed        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
