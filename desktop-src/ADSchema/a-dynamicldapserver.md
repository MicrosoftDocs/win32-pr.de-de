# <a name="dynamic-ldap-server-attribute"></a>Dynamic-LDAP-Server-Attribut

DNS-Name des Servers, der dynamische Eigenschaften für dieses Konto übermittelt.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Dynamic-LDAP-Server                     |
| Ldap-Anzeigename | dynamicLDAPServer                       |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | \-                                      |
| Updatehäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.537                  |
| System-ID-GUID    | 52458021-ca6a-11d0-afff-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
