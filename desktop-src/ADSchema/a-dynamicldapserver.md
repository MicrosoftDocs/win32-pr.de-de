# <a name="dynamic-ldap-server-attribute"></a>Dynamic-LDAP-Server-Attribut

DNS-Name des Servers, der dynamische Eigenschaften für dieses Konto übergibt.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Dynamic-LDAP-Server                     |
| LDAP-Display-Name | dynamicldapserver                       |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.537                  |
| System-ID-GUID    | 52458021-ca6a-11D0-affinf-0000f 80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
