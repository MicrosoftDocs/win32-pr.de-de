Enthält das Suffix des Warteschlangen namens (sofern vorhanden).

|-------------------|---------------------------------------------|
| CN                | MSMQ-Queue-Name-ext                         |
| LDAP-Display-Name | msmqqueuenameext                            |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1243                     |
| System-ID-GUID    | 2df90d87-009F -11d2-aa4c-00c04f d83a        |
