# <a name="ms-ds-dnsrootalias-attribute"></a>ms-DS-dnsrootalias-Attribut

Wird verwendet, um den Domänenalias zu speichern.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-DS-dnsrootalias                          |
| LDAP-Display-Name | MSDS-dnsrootalias                           |
| Size              | Maximale Größe 255 Bytes.                     |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.            |
| Aktualisierungshäufigkeit  | Nur während der Domänen Umstrukturierung.             |
| Attribute-Id      | 1.2.840.113556.1.4.1719                     |
| System-ID-GUID    | 2143acca-eead-4D29-b591-85fa49ce9173        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
