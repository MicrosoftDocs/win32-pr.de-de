# <a name="ms-ds-machine-account-quota-attribute"></a>MS-DS-Machine-Account-Quota-Attribut

Die Anzahl der Computer Konten, die ein Benutzer in einer Domäne erstellen darf.



| Eingabe | Wert |
|-------------------|--------------------------------------------------|
| CN                | MS-DS-Computer-Account-Quota                      |
| LDAP-Display-Name | ms-DS-machineaccountquota                        |
| Size              | 4 Bytes                                          |
| Berechtigung aktualisieren  | Domänen Administrator                             |
| Aktualisierungshäufigkeit  | Wann immer das Kontingent für eine Domäne geändert werden muss. |
| Attribute-Id      | 1.2.840.113556.1.4.1411                          |
| System-ID-GUID    | d064fb68-1480-11d3-91c1-0000f87a57d4             |
| Syntax            | [**Enumeration**](s-enumeration.md)             |
