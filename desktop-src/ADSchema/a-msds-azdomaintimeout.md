# <a name="ms-ds-az-domain-timeout-attribute"></a>ms-DS-AZ-Domain-Timeout-Attribut

Zeit (in Millisekunden), nach der erkannt wird, dass eine Domäne nicht erreichbar ist und bevor der DC erneut versucht wird.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | ms-DS-AZ-Domain-Timeout                 |
| LDAP-Display-Name | MSDS-azdomaintimeout                    |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Azrollen-Administrator                           |
| Aktualisierungshäufigkeit  | Während der Initialisierung oder Richtlinien Änderung. |
| Attribute-Id      | 1.2.840.113556.1.4.1795                 |
| System-ID-GUID    | 6448l56a-ca70-4e2e-b0af-d20e4ce653d0    |
| Syntax            | [**Enumeration**](s-enumeration.md)    |
