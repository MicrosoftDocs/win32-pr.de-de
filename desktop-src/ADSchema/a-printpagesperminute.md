# <a name="print-pages-per-minute-attribute"></a>Print-pages-pro-Minute-Attribut

Vom Treiber bereitgestellte Druck Rate in Seiten pro Minute.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Druckseiten pro Minute               |
| LDAP-Display-Name | printPagesPerMinute                  |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.631               |
| System-ID-GUID    | 19405b97-3cfa-11d1-a9c0-0000f 80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
