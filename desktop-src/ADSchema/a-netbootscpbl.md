# <a name="netboot-scp-bl-attribute"></a>NetBoot-SCP-BL-Attribut

Eine Liste der Dienst Verbindungspunkte, die auf diesen NetBoot-Server verweisen.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | NetBoot-SCP-BL                          |
| LDAP-Display-Name | netbootSCPBL                            |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.        |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.864                  |
| System-ID-GUID    | 07383082-91df-11d1-AEbc-0000e80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
