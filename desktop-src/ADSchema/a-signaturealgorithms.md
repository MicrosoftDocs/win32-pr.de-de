# <a name="signature-algorithms-attribute"></a>Signature-Algorithms-Attribut

Dieses Attribut gibt den Algorithmustyp an, der während des Authentifizierungsprozesses zum Decodieren einer digitalen Signatur verwendet werden muss.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Signature-Algorithms                        |
| Ldap-Anzeigename | signatureAlgorithms                         |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | \-                                          |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.824                      |
| System-ID-GUID    | 2a39c5b2-8960-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
