# <a name="signature-algorithms-attribute"></a>Signature-Algorithms-Attribut

Dieses Attribut gibt den Algorithmustyp an, der verwendet werden muss, um eine digitale Signatur während des Authentifizierungs Vorgangs zu decodieren.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Signature-Algorithms                        |
| LDAP-Display-Name | signaturealgorithms                         |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.824                      |
| System-ID-GUID    | 2a39c5b2-8960-11d1-AEbc-0000b80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
