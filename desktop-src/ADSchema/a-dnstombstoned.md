# <a name="dns-tombstoned-attribute"></a>DNS-Tombstoned-Attribut

TRUE, wenn dieses Objekt mit Tombstones vererbt wurde. Dieses Attribut ist vorhanden, um die Suche nach tombstoned-Datensätzen zu vereinfachen und zu beschleunigen. Tombstoneobjekte sind Objekte, die gelöscht, aber noch nicht aus dem Verzeichnis entfernt wurden.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | DNS-Tombstoned                       |
| Ldap-Anzeigename | dNSTombstoned                        |
| Size              | 4 Bytes                              |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.     |
| Updatehäufigkeit  | Jedes Mal, wenn ein Objekt gelöscht wird.       |
| Attribute-Id      | 1.2.840.113556.1.4.1414              |
| System-ID-GUID    | d5eb2eb7-be4e-463b-a214-634a44d7392e |
| Syntax            | [**Boolesch**](s-boolean.md)         |
