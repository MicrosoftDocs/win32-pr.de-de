# <a name="dns-tombstoned-attribute"></a>DNS-Tombstoned-Attribut

True, wenn für dieses Objekt ein tombstoning durchgearbeitet wurde. Dieses Attribut ist vorhanden, um die Suche nach tombstoning-Datensätzen zu vereinfachen und zu beschleunigen. Objekte mit tombstoning sind Objekte, die gelöscht, aber noch nicht aus dem Verzeichnis entfernt wurden.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | DNS-Tombstoned                       |
| LDAP-Display-Name | dnstombstoning                        |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn ein Objekt gelöscht wird.       |
| Attribute-Id      | 1.2.840.113556.1.4.1414              |
| System-ID-GUID    | d5eb2eb7-be4e-463b-a214-634a44d7392e |
| Syntax            | [**Booleschen**](s-boolean.md)         |
