# <a name="omt-guid-attribute"></a>OMT-Guid-Attribut

Der eindeutige Bezeichner für einen Link-Track-Object-Move-Tabelleneintrag.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | OMT-Guid                                              |
| LDAP-Display-Name | omtguid                                               |
| Size              | 16 Bytes                                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                      |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn sich der Link für eine Datei ändert.                 |
| Attribute-Id      | 1.2.840.113556.1.4.505                                |
| System-ID-GUID    | ddac0cf3-af8f-11d0-afeb-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
