# <a name="ms-sql-size-attribute"></a>MS-SQL-size-Attribut

Die Größe der Datenbank in Kilobyte.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MS-SQL-size                          |
| LDAP-Display-Name | MS-SQL-size                          |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | Wenn die Größe der Datenbank geändert wird.      |
| Attribute-Id      | 1.2.840.113556.1.4.1396              |
| System-ID-GUID    | e9098084-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**Intervall**](s-interval.md)       |
