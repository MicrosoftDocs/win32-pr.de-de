# <a name="ms-dfsr-readonly-attribute"></a>ms-DFSR-ReadOnly-Attribut

Gibt an, ob der Inhalt schreibgeschützt oder lese-/schreibgeschützt ist.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ms-DFSR-ReadOnly                     |
| Ldap-Anzeigename | msDFSR-ReadOnly                      |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.28           |
| System-ID-GUID    | 5ac48021-e447-46e7-9d23-92c0c6a90dfb |
| Syntax            | [**Boolesch**](s-boolean.md)         |
