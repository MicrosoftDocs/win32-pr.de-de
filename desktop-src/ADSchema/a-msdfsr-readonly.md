# <a name="ms-dfsr-readonly-attribute"></a>MS-DFSR-Schreib geschütztes Attribut

Gibt an, ob der Inhalt schreibgeschützt ist oder Lese-/Schreibzugriff hat.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-schreibgeschützt                     |
| LDAP-Display-Name | msdfsr-schreibgeschützt                      |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.28           |
| System-ID-GUID    | 5ac48021-e447-46e7-9d23-92c0c6a90dfb |
| Syntax            | [**Booleschen**](s-boolean.md)         |
