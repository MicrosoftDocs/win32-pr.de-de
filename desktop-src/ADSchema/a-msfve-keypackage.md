# <a name="ms-fve-keypackage-attribute"></a>ms-FVE-KeyPackage-Attribut

Enthält den BitLocker-Verschlüsselungsschlüssel eines Volumes, der durch das entsprechende Wiederherstellungskennwort geschützt ist.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | ms-FVE-KeyPackage                                     |
| Ldap-Anzeigename | msFVE-KeyPackage                                      |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | \-                                                    |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1999                               |
| System-ID-GUID    | 1fd55ea8-88a7-47dc-8129-0daa97186a54                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
