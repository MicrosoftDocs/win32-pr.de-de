# <a name="ms-fve-keypackage-attribute"></a>MS-f-KeyPackage-Attribut

Enthält den BitLocker-Verschlüsselungsschlüssel eines Volumes, der durch das entsprechende Wiederherstellungs Kennwort geschützt ist.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | MS-f-KeyPackage                                     |
| LDAP-Display-Name | MSF-KeyPackage                                      |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1999                               |
| System-ID-GUID    | 1gd55ea8-88a7-47dc-8129-0daa97186a54                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
