| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Members-für-AZ-Role-BL            |

| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | Dieser Wert wird vom System festgelegt.        |
