# <a name="presentation-address-attribute"></a>Presentation-Address-Attribut

Gibt eine Präsentations Adresse an, die einem Objekt zugeordnet ist, das eine OSI-Anwendungs Entität darstellt.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------------|
| CN                | Presentation-Address                                                  |
| LDAP-Display-Name | presentationaddress                                                   |
| Size              | \-                                                                    |
| Berechtigung aktualisieren  | \-                                                                    |
| Aktualisierungshäufigkeit  | \-                                                                    |
| Attribute-Id      | 2.5.4.29                                                              |
| System-ID-GUID    | a8df744b-c5ea-11d1-bbcb-0080c76670c0                                  |
| Syntax            | [**Object(Presentation-Address)**](s-object-presentation-address.md) |
