# <a name="product-code-attribute"></a>Product-Code-Attribut

Dieses Attribut enthält einen eindeutigen Bezeichner für eine Anwendung für eine bestimmte Produktversion, die als Zeichen folgen-GUID dargestellt wird, z {12345678-1234-1234-1234-123456789012} . b. "". Die in dieser GUID verwendeten Buchstaben müssen in Großbuchstaben angegeben werden. Diese ID muss für verschiedene Versionen und Sprachen variieren.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Product-Code                                          |
| LDAP-Display-Name | productCode                                           |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.818                                |
| System-ID-GUID    | d9e18317-8939-11d1-aebc-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
