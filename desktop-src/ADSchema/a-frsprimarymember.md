# <a name="frs-primary-member-attribute"></a>FRS-Primary-Member-Attribut

Verweis auf den primären Member einer Replikat Gruppe.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | FRS-primär-Member                      |
| LDAP-Display-Name | frsprimarymember                        |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.878                  |
| System-ID-GUID    | 2a132581-9373-11d1-AEbc-0000 C1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
