# <a name="initial-auth-incoming-attribute"></a>Ursprüngliches Authentifizierungs Attribut

Enthält Informationen über eine anfängliche eingehende Authentifizierungsanforderung durch einen Client an diesen Server. Diese Anforderung wird dann von diesem Server an den Authentifizierungsserver für die Domäne gesendet.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Anfängliche Authentifizierung-eingehende                       |
| LDAP-Display-Name | initialauthincoming                         |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.539                      |
| System-ID-GUID    | 52458023-ca6a-11D0-affinf-0000f 80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
