# <a name="ms-sql-characterset-attribute"></a>MS-SQL-Attributset-Attribut

Der Zeichensatz für die aktuelle Instanz von SQL Server.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MS-SQL-Merkmal Satz                  |
| LDAP-Display-Name | MS-SQL-Merkmal Satz                  |
| Size              | \-                                   |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | Beim Systemstart.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1370              |
| System-ID-GUID    | 696177a6-CCEE-11d2-9993-0000f 87a57d4 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
