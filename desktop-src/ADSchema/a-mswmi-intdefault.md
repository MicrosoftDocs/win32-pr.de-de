# <a name="ms-wmi-intdefault-attribute"></a>MS-WMI-intdefault-Attribut

Der Standardwert für ganzzahlige Parameter Objekte von WMI 32 Bit.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MS-WMI-intdefault                    |
| LDAP-Display-Name | mswap-intdefault                     |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | Gruppenrichtlinie-Administrator           |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1628              |
| System-ID-GUID    | 1b0c07f-76dd-4060-A1E1-70084619dc90 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
