# <a name="rpc-ns-group-attribute"></a>RPC-NS-Group-Attribut

Ein Verweis auf einen RPC-Server Eintrag oder eine andere RPC-Gruppe.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | RPC-NS-Gruppe                                |
| LDAP-Display-Name | rpcnsgroup                                  |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.114                      |
| System-ID-GUID    | bf967a24-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
