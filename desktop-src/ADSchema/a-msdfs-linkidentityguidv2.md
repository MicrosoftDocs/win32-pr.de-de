# <a name="ms-dfs-link-identity-guid-v2-attribute"></a>MS-DFS-Link-Identity-GUID-v2-Attribut

Wird nur festgelegt, wenn der Link erstellt wird. Über umbenennen oder verschieben, solange der Link nicht durch einen anderen Link ersetzt wird, der denselben Namen aufweist.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | MS-DFS-Link-Identity-GUID-v2                          |
| LDAP-Display-Name | msdfs-LinkIdentityGUIDv2                              |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.2041                               |
| System-ID-GUID    | edb027f3-5726-4dee-8d4e-dbf07e1ad1f1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
