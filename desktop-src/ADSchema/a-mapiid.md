# <a name="mapi-id-attribute"></a>MAPI-ID-Attribut

Eine ganze Zahl, durch die MAPI-Clients dieses Attribut identifizieren.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MAPI-ID                              |
| LDAP-Display-Name | mAPIID                               |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.49                |
| System-ID-GUID    | bf9679b7-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
