# <a name="dhcp-identification-attribute"></a>DHCP-Identifizierungs Attribut

Das **DHCP-Identification-** Attribut wird derzeit nicht verwendet.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | DHCP-Identifikation                         |
| LDAP-Display-Name | dhcpidentifizierung                          |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.701                      |
| System-ID-GUID    | 963d2742-48be-11d1-a9c3-0000 C1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
