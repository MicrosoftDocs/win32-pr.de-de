# <a name="must-contain-attribute"></a>Must-Contain-Attribut

Die Liste der obligatorischen Attribute für eine Klasse. Diese Attribute müssen angegeben werden, wenn eine Instanz der-Klasse erstellt wird.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------|
| CN                | Must-Contain                                                    |
| LDAP-Display-Name | mustContain                                                     |
| Size              | \-                                                              |
| Berechtigung aktualisieren  | Schema Administrator                                            |
| Aktualisierungshäufigkeit  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.24                                           |
| System-ID-GUID    | bf9679d3-0de6-11d0-a285-00aa003049e2                            |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
