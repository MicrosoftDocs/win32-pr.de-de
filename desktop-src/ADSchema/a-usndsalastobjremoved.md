# <a name="usn-dsa-last-obj-removed-attribute"></a>Das Attribut "US-DSA-Last-obj" wurde entfernt.

Enthält die Aktualisierungs Sequenznummer (Update Sequence Number, US) für das letzte Systemobjekt, das von einem Server entfernt wurde.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | USA-DSA-letzte-obj-entfernt             |
| LDAP-Display-Name | "", "".                 |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn sich ein Verzeichnis Objekt ändert. |
| Attribute-Id      | 1.2.840.113556.1.2.267               |
| System-ID-GUID    | bf967a71-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervall**](s-interval.md)       |
