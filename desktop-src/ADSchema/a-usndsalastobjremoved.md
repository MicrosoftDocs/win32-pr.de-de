# <a name="usn-dsa-last-obj-removed-attribute"></a>USN-DSA-Last-Obj-Removed-Attribut

Enthält die Updatesequenznummer (USN) für das letzte Systemobjekt, das von einem Server entfernt wurde.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | USN-DSA-last-obj-removed             |
| Ldap-Anzeigename | uSNDSALastObjRemoved                 |
| Size              | 8 Bytes                              |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.     |
| Updatehäufigkeit  | Jedes Mal, wenn sich ein Verzeichnisobjekt ändert. |
| Attribute-Id      | 1.2.840.113556.1.2.267               |
| System-ID-GUID    | bf967a71-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervall**](s-interval.md)       |
