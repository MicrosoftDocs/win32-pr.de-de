# <a name="max-renew-age-attribute"></a>Max-Renew-Age-Attribut

Dieses Attribut bestimmt den Zeitraum (in Tagen), in dem das Ticket Erteilungs Ticket (TGT) eines Benutzers für die Kerberos-Authentifizierung erneuert werden kann. Die Standardeinstellung ist 7 Tage im Standard-Domänen Gruppenrichtlinie Objekt (GPO).



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Max-Renew-Alter                        |
| LDAP-Display-Name | maxverlänage                          |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.75                |
| System-ID-GUID    | bf9679bc-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervall**](s-interval.md)       |
