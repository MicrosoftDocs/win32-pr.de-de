# <a name="print-media-ready-attribute"></a>Print-Media-Ready-Attribut

Eine Liste der verfügbaren Medien für einen Drucker.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Druck Medien-bereit                           |
| LDAP-Display-Name | printMediaReady                             |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.289                      |
| System-ID-GUID    | 3bcbfcf5-4d3d-11D0-A1A6-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
