# <a name="context-menu-attribute"></a>Context-Menu-Attribut

Die Bestellnummer und die GUID des Kontextmenüs, die für ein Objekt verwendet werden sollen.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Context-Menu                                |
| LDAP-Display-Name | ContextMenu                                 |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Schema Administrator                        |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.499                      |
| System-ID-GUID    | 4d8601ee-AC85-11D0-afe3-00c04f 930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
