# <a name="context-menu-attribute"></a>Context-Menu-Attribut

Die Bestellnummer und GUID des Kontextmenüs, das für ein Objekt verwendet werden soll.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Context-Menu                                |
| Ldap-Anzeigename | Contextmenu                                 |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | Schemaadministrator                        |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.499                      |
| System-ID-GUID    | 4d8601ee-ac85-11d0-afe3-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
