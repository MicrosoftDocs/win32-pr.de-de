# <a name="print-share-name-attribute"></a>Print-Share-Name-Attribut

Der Freigabe Name des Druckers.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Druckfreigabe Name                            |
| LDAP-Display-Name | printShareName                              |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.270                      |
| System-ID-GUID    | ba305f68-47e3-11d0-a1a6-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
