# <a name="dit-content-rules-attribute"></a>DIT-Content-Rules-Attribut

Hiermit wird der zulässige Inhalt von Einträgen einer bestimmten strukturellen Objektklasse mithilfe der Identifizierung eines optionalen Satzes von hilfsobjektklassen und obligatorischen, optionalen und ausgelösten Attributen angegeben. Kollektive Attribute sind in dit-Content-Rules enthalten, wie in RFC 2251 definiert.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | DIT-Content-Rules                           |
| LDAP-Display-Name | ditcontentrules                             |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.            |
| Aktualisierungshäufigkeit  | Immer dann, wenn eine neue Klasse erstellt wird.            |
| Attribute-Id      | 2.5.21.2                                    |
| System-ID-GUID    | 9a7ad946-ca53-11d1-bbd0-0080c76670c0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
