# <a name="garbage-coll-period-attribute"></a>Garbage-Coll-period-Attribut

Dieses Attribut befindet sich im Verzeichnisdienst CN = Verzeichnisdienst, CN = Windows NT, CN = Services, CN = Configuration,... Objekt. Sie stellt die Zeit in Stunden zwischen den Ausführungen der DS Garbage Collection dar.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Garbage Koll-Zeitraum                  |
| LDAP-Display-Name | garbageCollPeriod                    |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | Domänen Administrator                 |
| Aktualisierungshäufigkeit  | Sehr selten.                           |
| Attribute-Id      | 1.2.840.113556.1.2.301               |
| System-ID-GUID    | 5F d424a1-1262-11D0-a060-00aa006c33ed |
| Syntax            | [**Enumeration**](s-enumeration.md) |
