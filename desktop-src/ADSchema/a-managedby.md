# <a name="managed-by-attribute"></a>Managed-By-Attribut

Der Distinguished Name des Benutzers, der zugewiesen ist, um dieses Objekt zu verwalten.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------------------------------------|
| CN                | Managed-By                                                                          |
| LDAP-Display-Name | managedBy                                                                           |
| Size              | \-                                                                                  |
| Berechtigung aktualisieren  | Domänen Administrator oder Konto Besitzer.                                              |
| Aktualisierungshäufigkeit  | Wenn der Benutzerdaten Satz erstellt wird, und wenn die verwalteten Objekte geändert werden müssen. |
| Attribute-Id      | 1.2.840.113556.1.4.653                                                              |
| System-ID-GUID    | 0296c120-40DA-11d1-a9c0-0000b80367c1                                                |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)                                             |
