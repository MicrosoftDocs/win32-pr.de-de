# <a name="upgrade-product-code-attribute"></a>Upgrade-Product-Code-Attribut

Dieses Attribut enthält den Produktcode anderer Pakete, z. B. Anwendungen, die von diesem Paket aktualisiert werden können, oder die dieses Paket aktualisieren können.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Upgrade-Product-Code                                  |
| Ldap-Anzeigename | upgradeProductCode                                    |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | \-                                                    |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.813                                |
| System-Id-Guid    | d9e18312-8939-11d1-aebc-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
