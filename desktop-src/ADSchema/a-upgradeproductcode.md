# <a name="upgrade-product-code-attribute"></a>Upgrade-Product-Code-Attribut

Dieses Attribut enthält den Produktcode anderer Pakete (z. b. Anwendungen), die durch dieses Paket aktualisiert werden können oder für die ein Upgrade dieses Pakets ausgeführt werden kann.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Upgrade-Product-Code                                  |
| LDAP-Display-Name | upgradeproductcode                                    |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.813                                |
| System-ID-GUID    | d9e18312-8939-11d1-aebc-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
