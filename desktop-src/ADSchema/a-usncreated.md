# <a name="usn-created-attribute"></a>USN-Created-Attribut

Die bei der Objekt Erstellung zugewiesene Aktualisierungs Sequenznummer (Update Sequence Number, en). Siehe [**auch.**](a-usnchanged.md)



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | USN-Created                          |
| LDAP-Display-Name | uSNCreated                           |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | Wenn das Objekt erstellt wird.          |
| Attribute-Id      | 1.2.840.113556.1.2.19                |
| System-ID-GUID    | bf967a70-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervall**](s-interval.md)       |
