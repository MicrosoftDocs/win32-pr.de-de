# <a name="physical-location-object-attribute"></a>Physical-Location-Object-Attribut

Wird verwendet, um ein Gerät (z. b. einen Drucker, einen Computer usw.) einem physischen Speicherort zuzuordnen.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Physical-Location-Object                |
| LDAP-Display-Name | physicalLocationObject                  |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.514                  |
| System-ID-GUID    | b7b13119-b82e-11d0-afee-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
