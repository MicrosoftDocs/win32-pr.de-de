# <a name="applies-to-attribute"></a>Applies-To-Attribut

Enthält die Liste der Objektklassen, für die das erweiterte Recht gilt. In der Liste wird eine Objektklasse durch die schemaIDGUID-Eigenschaft für das zugehörige Schema Schema dargestellt.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Applies-To                                  |
| LDAP-Display-Name | appliesTo                                   |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.341                      |
| System-ID-GUID    | 8297931d-86d3-11D0-AFDA-00c04f 930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
