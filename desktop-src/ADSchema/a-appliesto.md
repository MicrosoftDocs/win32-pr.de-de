# <a name="applies-to-attribute"></a>Applies-To-Attribut

Enthält die Liste der Objektklassen, für die das erweiterte Recht gilt. In der Liste wird eine Objektklasse durch die schemaIDGUID-Eigenschaft für das schemaClass-Objekt dargestellt.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Applies-To                                  |
| Ldap-Anzeigename | appliesTo                                   |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | \-                                          |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.341                      |
| System-Id-Guid    | 8297931d-86d3-11d0-afda-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
