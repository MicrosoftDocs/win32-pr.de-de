# <a name="server-reference-bl-attribute"></a>Server-Reference-BL-Attribut

Gefunden im Domänen namens Kontext. Der Distinguished Name eines Computers unter dem Ordner Sites.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Server-Reference-BL                     |
| LDAP-Display-Name | ServerReferenceBL                       |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.        |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.516                  |
| System-ID-GUID    | 26d9736e-6070-11d1-a9c6-0000l80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
