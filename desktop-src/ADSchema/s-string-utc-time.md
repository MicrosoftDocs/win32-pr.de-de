# <a name="stringutc-time-syntax"></a>Zeichenfolgensyntax (UTC-Zeit)

Ein durch ASN.1-Standards definiertes Zeitzeichenfolgenformat. Weitere Informationen finden Sie unter ISO 8601 und X680. Verwenden Sie diese Syntax zum Speichern von Zeitwerten in UTC-Time Format.



| Eingabe | Wert |
|--------------|----------------------------------------------------------------------------|
| Name         | String(UTC-Time)                                                           |
| Syntax-ID    | 2.5.5.11                                                                   |
| OM-ID        | 23                                                                         |
| MAPI-Typ    | Systime                                                                    |
| ADS-Typ     | ADSTYPE \_ UTC \_ TIME                                                         |
| Variant-Typ | VT \_ DATE                                                                   |
| SDS-Typ     | [System.DateTime](/dotnet/api/system.datetime) |
