# <a name="domain-certificate-authorities-attribute"></a>Domain-Certificate-Autoritäten-Attribut

Liste der Zertifizierungsstellen für eine bestimmte Domäne.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Domänen Zertifizierungsstellen          |
| LDAP-Display-Name | Domänen CAS                               |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.668                  |
| System-ID-GUID    | 7bf dcb7a-4807-11d1-a9c3-0000b80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
