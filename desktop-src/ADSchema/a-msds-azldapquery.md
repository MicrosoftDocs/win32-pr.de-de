# <a name="ms-ds-az-ldap-query-attribute"></a>ms-DS-AZ-LDAP-Query-Attribut

Eine Zeichenfolge, die die LDAP-Abfrage definiert (maximale Länge 4096), mit der die Mitgliedschaft eines Benutzer Objekts zur Gruppe bestimmt wird.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-LDAP-Query                         |
| LDAP-Display-Name | MSDS-azldapquery                            |
| Size              | 4096 Zeichen                             |
| Berechtigung aktualisieren  | Azrollen-Administrator                               |
| Aktualisierungshäufigkeit  | Während der Initialisierung oder Richtlinien Änderung.     |
| Attribute-Id      | 1.2.840.113556.1.4.1792                     |
| System-ID-GUID    | 5e53368b-fc94-45c8-9d7d-DAF 31ee7112d        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
