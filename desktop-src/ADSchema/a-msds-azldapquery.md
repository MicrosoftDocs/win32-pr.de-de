# <a name="ms-ds-az-ldap-query-attribute"></a>ms-DS-Az-LDAP-Query-Attribut

Eine Zeichenfolge, die die LDAP-Abfrage (max. Länge 4096) definiert, die die Mitgliedschaft eines Benutzerobjekts zur Gruppe bestimmt.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Az-LDAP-Query                         |
| Ldap-Anzeigename | msDS-AzLDAPQuery                            |
| Size              | 4096 Zeichen                             |
| Aktualisieren von Berechtigungen  | AzRoles admin                               |
| Updatehäufigkeit  | Während der Initialisierung oder Richtlinienänderung.     |
| Attribute-Id      | 1.2.840.113556.1.4.1792                     |
| System-ID-GUID    | 5e53368b-fc94-45c8-9d7d-daf31ee7112d        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
