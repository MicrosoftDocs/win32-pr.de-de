# <a name="locale-id-attribute"></a>Locale-ID-Attribut

Dieses Attribut enthält eine Liste von Gebiets Schema-IDs, die von dieser Anwendung unterstützt werden. Eine Gebiets Schema-ID ist ein geografischer Standort, z. b. ein Land/eine Region, eine Stadt, ein Landkreis usw.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------|
| CN                | Gebiets Schema-ID                                                                  |
| LDAP-Display-Name | localeID                                                                   |
| Size              | 4 Bytes                                                                    |
| Berechtigung aktualisieren  | Domänen Administrator oder Konto Besitzer.                                     |
| Aktualisierungshäufigkeit  | Wenn der Benutzerdaten Satz erstellt wird, und wenn der Ort geändert werden muss. |
| Attribute-Id      | 1.2.840.113556.1.4.58                                                      |
| System-ID-GUID    | bf9679a1-0de6-11d0-a285-00aa003049e2                                       |
| Syntax            | [**Enumeration**](s-enumeration.md)                                       |
