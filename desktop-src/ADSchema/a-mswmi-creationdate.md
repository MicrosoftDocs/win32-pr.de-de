# <a name="ms-wmi-creationdate-attribute"></a>MS-WMI-kreationdate-Attribut

Der Datums-und Zeitstempel der Erstellung einer Objektinstanz.



| Eingabe | Wert |
|-------------------|---------------------------------------------------|
| CN                | MS-WMI--erationdate                               |
| LDAP-Display-Name | mslimi-"deationdate"                                |
| Size              | Darstellung des WMI-datetime-Datentyps-26 Zeichen. |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                  |
| Aktualisierungshäufigkeit  | \-                                                |
| Attribute-Id      | 1.2.840.113556.1.4.1626                           |
| System-ID-GUID    | 748b0a2e-3351-4b3f-B171-2 ea779              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)       |
