# <a name="ms-sql-version-attribute"></a>MS-SQL-Version-Attribut

Die Version der aktuellen Instanz von SQL Server.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Version                              |
| LDAP-Display-Name | MS-SQL-Version                              |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.            |
| Aktualisierungshäufigkeit  | Beim System Setup.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1388                     |
| System-ID-GUID    | c07cc1d0-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
