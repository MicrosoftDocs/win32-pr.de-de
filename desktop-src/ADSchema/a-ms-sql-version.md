# <a name="ms-sql-version-attribute"></a>MS-SQL-Version-Attribut

Die Version für die aktuelle Instanz von SQL Server.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Version                              |
| Ldap-Anzeigename | mS-SQL-Version                              |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.            |
| Updatehäufigkeit  | Beim Systemsetup.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1388                     |
| System-ID-GUID    | c07cc1d0-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
