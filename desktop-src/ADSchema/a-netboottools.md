# <a name="netboot-tools-attribute"></a>netboot-tools-Attribut

Das **netboot-tools-** Attribut ist für die interne Verwendung reserviert.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | netboot-tools                               |
| LDAP-Display-Name | netboottools                                |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.            |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.858                      |
| System-ID-GUID    | 0738307f -91df-11d1-AEbc-0000        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
