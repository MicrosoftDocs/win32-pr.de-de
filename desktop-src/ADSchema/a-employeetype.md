# <a name="employee-type-attribute"></a>Employee-Type-Attribut

Die Auftrags Kategorie für einen Mitarbeiter.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Employee-Type                               |
| LDAP-Display-Name | employeeType                                |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänen Administrator                        |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.613                      |
| System-ID-GUID    | a8df73f0-c5ea-11d1-bbcb-0080c76670c0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
