# <a name="print-rate-attribute"></a>Print-Rate-Attribut

Vom Treiber bereitgestellte Druck Rate.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Print-Rate                           |
| LDAP-Display-Name | printRate                            |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.285               |
| System-ID-GUID    | ba305f77-47e3-11d0-a1a6-00c04fd930c9 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
