# <a name="phone-isdn-primary-attribute"></a>Telefon-ISDN-Primary-Attribut

Der primäre ISDN.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Telefon-ISDN-Primary                                                               |
| Ldap-Anzeigename | primaryInternationalISDNNumber                                                   |
| Size              | \-                                                                               |
| Aktualisieren von Berechtigungen  | Domänenadministrator oder Kontobesitzer.                                           |
| Updatehäufigkeit  | Wenn der Datensatz des Benutzers erstellt wird und wann immer sich die Telefonnummer ändern muss. |
| Attribute-Id      | 1.2.840.113556.1.4.649                                                           |
| System-Id-Guid    | 0296c11f-40da-11d1-a9c0-0000f80367c1                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
