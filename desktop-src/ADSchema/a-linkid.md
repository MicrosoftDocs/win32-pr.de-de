# <a name="link-id-attribute"></a>Link-ID-Attribut

Eine ganze Zahl, die angibt, dass das Attribut ein verknüpftes Attribut ist. Eine gerade ganze Zahl ist ein Vorwärtslink, und eine ungerade ganze Zahl ist ein Rückwärtslink.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Link-ID                              |
| Ldap-Anzeigename | Linkid                               |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | Schemaadministrator                 |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.50                |
| System-ID-GUID    | bf96799b-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
