# <a name="msmq-routing-service-attribute"></a>MSMQ-Routing-Dienst-Attribut

Gibt an, ob dieser Server Routing Dienste bereitstellt.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MSMQ-Routing-Service                 |
| LDAP-Display-Name | msmqroutingservice                   |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1237              |
| System-ID-GUID    | 2df90d81-009F -11d2-aa4c-00c04f d83a |
| Syntax            | [**Booleschen**](s-boolean.md)         |
