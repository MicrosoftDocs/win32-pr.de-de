# <a name="moniker-display-name-attribute"></a>Moniker-Display-Name-Attribut

Der Anzeige Name für einen Moniker.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Moniker-Anzeige Name                        |
| LDAP-Display-Name | monikerdisplayname                          |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.83                       |
| System-ID-GUID    | bf9679c8-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
