# <a name="print-color-attribute"></a>Print-Color-Attribut

**True** , wenn ein Drucker in Farbe drucken kann.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Print-Color                          |
| LDAP-Display-Name | printColor                           |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.243               |
| System-ID-GUID    | 281416d3-1968-11D0-a28f -00aa003049e2 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
