Eine Zeichenfolge, die von einzelnen Anwendungen verwendet wird, um die Informationen zu speichern, die Sie möglicherweise benötigen.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-Application-Data                   |
| LDAP-Display-Name | MSDS-azapplicationdata                      |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1819                     |
| System-ID-GUID    | 503fc3e8-1cc6-461A-99a3-9eee04f 402a7        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
