# <a name="ms-ds-has-master-ncs-attribute"></a>ms-DS-has-Master-NCS-Attribut

Enthält die von einem Domänen Controller gehaltenen Namenskontexte. Mit diesem Attribut wird "-Master-NCS" als veraltet markiert.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | ms-DS-has-Master-NCS                    |
| LDAP-Display-Name | MSDS-hasmasterncs                       |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.         |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn ein neuer NC erstellt wird.          |
| Attribute-Id      | 1.2.840.113556.1.4.1836                 |
| System-ID-GUID    | ae2de0e2-59d7-4d47-8d47-ed4dfe4357ad    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |



## <a name="implementations"></a>Implementierungen

-   [**Adam**](#adam)
-   [**Windows Server 2003 R2**](#windows-server-2003-r2)
-   [**Windows Server 2008**](#windows-server-2008)
