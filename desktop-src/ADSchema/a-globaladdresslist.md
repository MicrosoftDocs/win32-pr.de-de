# <a name="global-address-list-attribute"></a>Global-Address-List-Attribut

Dieses Attribut wird in einem Microsoft Exchange-Container verwendet, um den Distinguished Name einer neu erstellten globalen Adressliste (GAL) zu speichern. Dieses Attribut muss über einen Eintrag verfügen, bevor Sie Messaging Application Programming Interface (MAPI)-Clients für die Verwendung eines GAL aktivieren können.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Global-Address-List                     |
| LDAP-Display-Name | GlobalAddressList                       |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1245                 |
| System-ID-GUID    | f754c748-06f4-11d2-aa53-00c04fd7d83a    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
