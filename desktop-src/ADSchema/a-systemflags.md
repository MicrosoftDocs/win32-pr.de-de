# <a name="system-flags-attribute"></a>System-Flags-Attribut

Ein ganzzahliger Wert, der Flags enthält, die zusätzliche Eigenschaften der-Klasse definieren. Siehe Hinweise.



| Eingabe | Wert |
|-------------------|------------------------------------------------|
| CN                | System-Flags                                   |
| LDAP-Display-Name | systemFlags                                    |
| Size              | \-                                             |
| Berechtigung aktualisieren  | Dieser Wert wird vom Schema Administrator festgelegt. |
| Aktualisierungshäufigkeit  | Wenn sich der Zustand des-Objekts ändert.      |
| Attribute-Id      | 1.2.840.113556.1.4.375                         |
| System-ID-GUID    | e0fa1e62-9b45-11d0-afdd-00c04fd930c9           |
| Syntax            | [**Enumeration**](s-enumeration.md)           |

## <a name="remarks"></a>Bemerkungen

Dieses Attribut kann NULL sein oder eine Kombination aus einem oder mehreren der folgenden Werte aufweisen.



| Wert                   | Beschreibung                                                                                                                                                                                                                                                                                     |
|-------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 (0x00000001)          | Wenn das Attribut auf ein Attribut angewendet wird, wird es nicht repliziert. Wenn Sie auf ein [**quer**](c-crossref.md) Verweis Objekt angewendet wird, befindet sich der namens Kontext in NTDS.<br/>                                                                                                                          |
| 2 (0x00000002)          | Wenn das Attribut auf ein Attribut angewendet wird, wird es in den globalen Katalog repliziert. Der namens Kontext ist eine Domäne, wenn er auf ein [**quer**](c-crossref.md) Verweis Objekt angewendet wird.<br/>                                                                                                       |
| 4 (0x00000004)          | Wenn das Attribut auf ein Attribut angewendet wird, wird es erstellt.                                                                                                                                                                                                                                     |
| 16 (0x00000010)         | Gibt an, dass das Objekt ein Objekt der Kategorie 1 ist, wenn festgelegt. Ein Objekt der Kategorie 1 ist eine Klasse oder ein Attribut, das in dem im System enthaltenen Basis Schema enthalten ist.                                                                                                                                |
| 33554432 (0x02000000)   | Das Objekt wird nicht in den Container für gelöschte Objekte verschoben, wenn es gelöscht wird. Sie wird sofort gelöscht.                                                                                                                                                                                    |
| 67108864 (0x04000000)   | Das Objekt kann nicht verschoben werden.                                                                                                                                                                                                                                                                     |
| 134217728 (0x08000000)  | Das Objekt kann nicht umbenannt werden.                                                                                                                                                                                                                                                                   |
| 268435456 (0x10000000)  | Wenn dieses Flag festgelegt ist, kann das Objekt bei Objekten in der Konfigurations Partition mit Einschränkungen verschoben werden. Andernfalls kann das Objekt nicht verschoben werden. Standardmäßig ist dieses Flag nicht für neue Objekte festgelegt, die unter der Konfigurations Partition erstellt werden. Dieses Flag kann nur während der Objekt Erstellung festgelegt werden. |
| 536870912 (0x20000000)  | Wenn dieses Flag festgelegt ist, kann das Objekt bei Objekten in der Konfigurations Partition verschoben werden. Andernfalls kann das Objekt nicht verschoben werden. Standardmäßig ist dieses Flag nicht für neue Objekte festgelegt, die unter der Konfigurations Partition erstellt werden. Dieses Flag kann nur während der Objekt Erstellung festgelegt werden.                   |
| 1073741824 (0x40000000) | Wenn dieses Flag festgelegt ist, kann das Objekt für Objekte in der Konfigurations Partition umbenannt werden. Andernfalls kann das Objekt nicht umbenannt werden. Standardmäßig ist dieses Flag nicht für neue Objekte festgelegt, die unter der Konfigurations Partition erstellt werden. Dieses Flag kann nur während der Objekt Erstellung festgelegt werden.               |
| 2147483648 (0x80000000) | Das Objekt kann nicht gelöscht werden.                                                                                                                                                                                                                                                                   |
