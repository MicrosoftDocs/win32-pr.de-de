# <a name="system-flags-attribute"></a>System-Flags-Attribut

Ein ganzzahliger Wert, der Flags enthält, die zusätzliche Eigenschaften der Klasse definieren. Siehe Hinweise.



| Eingabe | Wert |
|-------------------|------------------------------------------------|
| CN                | System-Flags                                   |
| Ldap-Anzeigename | systemFlags                                    |
| Size              | \-                                             |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom Schemaadministrator festgelegt. |
| Updatehäufigkeit  | Immer dann, wenn sich der Zustand des Objekts ändert.      |
| Attribute-Id      | 1.2.840.113556.1.4.375                         |
| System-ID-GUID    | e0fa1e62-9b45-11d0-afdd-00c04fd930c9           |
| Syntax            | [**Enumeration**](s-enumeration.md)           |

## <a name="remarks"></a>Hinweise

Dieses Attribut kann 0 (null) oder eine Kombination aus mindestens einem der folgenden Werte sein.



| Wert                   | Beschreibung                                                                                                                                                                                                                                                                                     |
|-------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 (0x00000001)          | Wenn es auf ein Attribut angewendet wird, wird das Attribut nicht repliziert. Wenn der [**Benennungskontext auf ein Ref-übergreifendes**](c-crossref.md) Objekt angewendet wird, befindet er sich in NTDS.<br/>                                                                                                                          |
| 2 (0x00000002)          | Wenn das Attribut auf ein Attribut angewendet wird, wird es in den globalen Katalog repliziert. Bei Anwendung auf ein [**Cross-Ref-Objekt**](c-crossref.md) ist der Namenskontext eine Domäne.<br/>                                                                                                       |
| 4 (0x00000004)          | Wenn das Attribut auf ein Attribut angewendet wird, wird es erstellt.                                                                                                                                                                                                                                     |
| 16 (0x00000010)         | Gibt bei Festlegung an, dass das Objekt ein Objekt der Kategorie 1 ist. Ein Objekt der Kategorie 1 ist eine Klasse oder ein Attribut, die bzw. das im Basisschema des Systems enthalten ist.                                                                                                                                |
| 33554432 (0x02000000)   | Das Objekt wird nicht in den Container Gelöschte Objekte verschoben, wenn es gelöscht wird. Er wird sofort gelöscht.                                                                                                                                                                                    |
| 67108864 (0x04000000)   | Das Objekt kann nicht verschoben werden.                                                                                                                                                                                                                                                                     |
| 134217728 (0x08000000)  | Das Objekt kann nicht umbenannt werden.                                                                                                                                                                                                                                                                   |
| 268435456 (0x10000000)  | Wenn dieses Flag für Objekte in der Konfigurationspartition festgelegt ist, kann das Objekt mit Einschränkungen verschoben werden. Andernfalls kann das Objekt nicht verschoben werden. Standardmäßig ist dieses Flag nicht für neue Objekte festgelegt, die unter der Konfigurationspartition erstellt wurden. Dieses Flag kann nur während der Objekterstellung festgelegt werden. |
| 536870912 (0x20000000)  | Wenn dieses Flag für Objekte in der Konfigurationspartition festgelegt ist, kann das Objekt verschoben werden. Andernfalls kann das Objekt nicht verschoben werden. Standardmäßig ist dieses Flag nicht für neue Objekte festgelegt, die unter der Konfigurationspartition erstellt wurden. Dieses Flag kann nur während der Objekterstellung festgelegt werden.                   |
| 1073741824 (0x40000000) | Wenn dieses Flag für Objekte in der Konfigurationspartition festgelegt ist, kann das Objekt umbenannt werden. Andernfalls kann das Objekt nicht umbenannt werden. Standardmäßig ist dieses Flag nicht für neue Objekte festgelegt, die unter der Konfigurationspartition erstellt wurden. Dieses Flag kann nur während der Objekterstellung festgelegt werden.               |
| 2147483648 (0x80000000) | Das Objekt kann nicht gelöscht werden.                                                                                                                                                                                                                                                                   |
