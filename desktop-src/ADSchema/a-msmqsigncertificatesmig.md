# <a name="msmq-sign-certificates-mig-attribute"></a>MSMQ-Sign-Zertifikats-MiG-Attribut

Im gemischten Modus von MSMQ enthält das-Attribut den vorherigen Wert von msmqsignzertifikats. MSMQ unterstützt die Migration von MSMQ 1,0 DS zu Windows 2000 DS, und der gemischte Modus gibt einen Status an, in dem einige der DS-Schweregrade nicht auf Windows 2000 aktualisiert wurden.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Sign-Zertifikats-MiG                            |
| LDAP-Display-Name | msmqsigncertifikatesmig                               |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.967                                |
| System-ID-GUID    | 3881b8ea-da3b-11d1-90a5-00c04f d91ab1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
