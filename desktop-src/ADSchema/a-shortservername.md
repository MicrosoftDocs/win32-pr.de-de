# <a name="short-server-name-attribute"></a>Short-Server-Name-Attribut

Windows-kompatibler Servername (Pre-Windows 2000) für Druckserver.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Short-Server-Name                           |
| LDAP-Display-Name | shortServerName                             |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänen Administrator                        |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1209                     |
| System-ID-GUID    | 45b01501-c419-11d1-bbc9-0080c76670c0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
