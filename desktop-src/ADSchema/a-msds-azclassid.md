# <a name="ms-ds-az-class-id-attribute"></a>ms-DS-AZ-Class-ID-Attribut

Eine Klassen-ID, die von der azrollen-Benutzeroberfläche für das AzApplication-Objekt benötigt wird.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-Class-ID                           |
| LDAP-Display-Name | MSDS-azclassid                              |
| Size              | 36 Zeichen                               |
| Berechtigung aktualisieren  | Azrollen-Administrator                               |
| Aktualisierungshäufigkeit  | Während der Initialisierung oder Richtlinien Änderung.     |
| Attribute-Id      | 1.2.840.113556.1.4.1816                     |
| System-ID-GUID    | 013a7277-5c2d-49ef-a7de-b765b36a3f 6F        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
