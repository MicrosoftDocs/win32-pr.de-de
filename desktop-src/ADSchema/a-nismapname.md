# <a name="nismapname-attribute"></a>nismapname-Attribut

Enthält den Namen der Zuordnung, zu der das Objekt gehört.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | nmapname                           |
| LDAP-Display-Name | nmapname                           |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.26                     |
| System-ID-GUID    | 969d3c79-0e9a-4d95-b0ac-bdde7ff8f3a1 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
