# <a name="frs-computer-reference-bl-attribute"></a>FRS-Computer-Reference-BL-Attribut

Verweis auf die Replikat Gruppen, zu denen dieser Computer gehört.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | FRS-Computer-Reference-BL               |
| LDAP-Display-Name | frscomputerreferencebl                  |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.        |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.870                  |
| System-ID-GUID    | 2a132579-9373-11d1-AEbc-0000 C1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
