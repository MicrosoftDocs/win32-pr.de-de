# <a name="treat-as-leaf-attribute"></a>Treat-As-Leaf-Attribut

Anzeigespezifizierer, bei der dieses Flag auf **TRUE** festgelegt ist, erzwingen, dass die verknüpfte Klasse auch dann als Blattklasse angezeigt wird, wenn sie über untergeordnete Elemente verfügt.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Treat-As-Leaf                        |
| Ldap-Anzeigename | treatAsLeaf                          |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.806               |
| System-ID-GUID    | 8fd044e3-771f-11d1-aeae-0000f80367c1 |
| Syntax            | [**Boolesch**](s-boolean.md)         |
