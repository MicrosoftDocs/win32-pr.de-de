# <a name="membernisnetgroup-attribute"></a>attributnetgroup-Attribut

Enthält die Liste der Netzgruppen, die Mitglieder dieser netgroup sind.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | mitgliednisnetzgruppe                    |
| LDAP-Display-Name | mitgliednisnetzgruppe                    |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.13                     |
| System-ID-GUID    | 0F 6a17dc-53e5-4be8-9442-8f. |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
