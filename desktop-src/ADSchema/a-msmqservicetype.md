# <a name="msmq-service-type-attribute"></a>MSMQ-Service-Type-Attribut

Der Diensttyp, der von MSMQ auf diesem Computer installiert wird.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MSMQ-Diensttyp                    |
| LDAP-Display-Name | msmqservicetype                      |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.930               |
| System-ID-GUID    | 9a0dc32d-C100-11d1-bbc5-0080c76670c0 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
