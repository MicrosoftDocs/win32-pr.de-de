# <a name="ms-ds-replication-notify-subsequent-dsa-delay-attribute"></a>ms-DS-Replication-Notify-Subsequent-DSA-Delay-Attribut

Dieses Attribut steuert die Verzögerung zwischen der Benachrichtigung jedes nachfolgenden Replikatpartners für einen NC.



| Eingabe | Wert |
|-------------------|-----------------------------------------------|
| CN                | ms-DS-Replication-Notify-Subsequent-DSA-Delay |
| Ldap-Anzeigename | msDS-Replication-Notify-Subsequent-DSA-Delay  |
| Size              | \-                                            |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.              |
| Updatehäufigkeit  | \-                                            |
| Attribute-Id      | 1.2.840.113556.1.4.1664                       |
| System-Id-Guid    | d63db385-dd92-4b52-b1d8-0d3ecc0e86b6          |
| Syntax            | [**Enumeration**](s-enumeration.md)          |
