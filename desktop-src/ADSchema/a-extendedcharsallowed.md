# <a name="extended-chars-allowed-attribute"></a>Extended-chars-Allowed-Attribut

Gibt an, ob im Wert dieses Attributs erweiterte Zeichen zulässig sind. Gilt nur für IA5-, numerische, druckbare und Teletex-Zeichen folgen Attribute.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Extended-chars-Allowed               |
| LDAP-Display-Name | extendedcharsallowed                 |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.380               |
| System-ID-GUID    | bf967966-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
