# <a name="picture-attribute"></a>Bild Attribut

Ein Bild des Benutzers. Ein Speicher fähiges Format wie JPEG oder GIF wird empfohlen.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Picture                                               |
| LDAP-Display-Name | thumbnailPhoto                                        |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 2.16.840.1.113730.3.1.35                              |
| System-ID-GUID    | 8d3bca50-1d7e-11D0-a081-00aa006c33ed                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
