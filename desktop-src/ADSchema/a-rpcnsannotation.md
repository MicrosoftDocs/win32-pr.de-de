# <a name="rpc-ns-annotation-attribute"></a>RPC-NS-Annotation-Attribut

Eine Zeichenfolge, die ein bestimmtes RPC-Profil Element beschreibt.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | RPC-NS-Anmerkung                           |
| LDAP-Display-Name | rpcnsannotation                             |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.366                      |
| System-ID-GUID    | 88611bde-8cf4-11D0-AFDA-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
