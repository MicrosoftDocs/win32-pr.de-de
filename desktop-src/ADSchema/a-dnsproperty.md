# <a name="dns-property-attribute"></a>DNS-Property-Attribut

Wird zum Speichern von Binär Einstellungen (Eigenschaften) in DNS-Zonen Objekten verwendet.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | DNS-Property                                          |
| LDAP-Display-Name | dnsproperty                                           |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                      |
| Aktualisierungshäufigkeit  | Wenn sich die DNS-Zonendaten Sätze ändern.                     |
| Attribute-Id      | 1.2.840.113556.1.4.1306                               |
| System-ID-GUID    | 675a15fe-3b70-11d2-90cc-00c04f d91ab1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
