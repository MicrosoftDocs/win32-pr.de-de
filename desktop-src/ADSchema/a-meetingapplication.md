# <a name="meetingapplication-attribute"></a>meetingapplication-Attribut

Dieses Attribut enthält den Namen der Anwendung, die für eine Online Besprechung verwendet wird, z. b. Microsoft NetMeeting, Windows Media Services oder Microsoft Exchange-Konferenzen.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | meetingapplication                          |
| LDAP-Display-Name | meetingapplication                          |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.573                      |
| System-ID-GUID    | 11b6cc83-48C4-11d1-a9c3-0000t80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
