# <a name="meetingapplication-attribute"></a>meetingApplication-Attribut

Dieses Attribut enthält den Namen der Anwendung, die für eine Onlinebesprechung wie Microsoft NetMeeting, Windows Media-Dienste oder Microsoft Exchange Conferenceing verwendet wird.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | meetingApplication                          |
| Ldap-Anzeigename | meetingApplication                          |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | \-                                          |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.573                      |
| System-ID-GUID    | 11b6cc83-48c4-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
