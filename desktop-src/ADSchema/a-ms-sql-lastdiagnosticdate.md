# <a name="ms-sql-lastdiagnosticdate-attribute"></a>MS-SQL-lastdiagnosticdate-Attribut

Das letzte Datum, an dem DBCC CHECKDB ausgeführt wurde.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-lastdiagnosticdate                   |
| LDAP-Display-Name | MS-SQL-lastdiagnosticdate                   |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.            |
| Aktualisierungshäufigkeit  | Wenn DBCC CHECKDB ausgeführt wird.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1399                     |
| System-ID-GUID    | f6d6dd88-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
