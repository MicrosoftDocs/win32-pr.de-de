# <a name="gpc-machine-extension-names-attribute"></a>GPC-Machine-Extension-names-Attribut

Wird vom Gruppenrichtlinie-Objekt für Computer Richtlinien verwendet.



| Eingabe | Wert |
|-------------------|---------------------------------------------------------------------------------|
| CN                | GPC-Computer-Erweiterungs Namen                                                     |
| LDAP-Display-Name | gpcmachineextensionnames                                                        |
| Size              | Hängt von der Anzahl der Client seitigen Erweiterungen ab, die in diesem Gruppenrichtlinien Objekt über eine Richtlinie verfügen. |
| Berechtigung aktualisieren  | Domänen-oder Richtlinien Administrator.                                                 |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn ein GPO über das gpe aktualisiert wird.                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1348                                                         |
| System-ID-GUID    | 32ff8ecc-783f-11d2-9916-0000f87a57d4                                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                     |
