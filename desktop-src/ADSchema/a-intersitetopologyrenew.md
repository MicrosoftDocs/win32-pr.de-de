# <a name="inter-site-topology-renew-attribute"></a>Attribut "Inter-Site-Topology-Renew"

Diese Klasse gibt an, wie oft der Generator für die standortübergreifende Topologie die Keep-Alive-Nachricht aktualisiert, die an Domänencontroller gesendet wird, die am gleichen Standort enthalten sind.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Standortübergreifende Topologieverlängerung            |
| Ldap-Anzeigename | interSiteTopologyRenew               |
| Size              | 4 Bytes                              |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1247              |
| System-Id-Guid    | b7c69e5f-2cc7-11d2-854e-00a0c983f608 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
