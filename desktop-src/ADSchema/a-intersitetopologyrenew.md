# <a name="inter-site-topology-renew-attribute"></a>Inter-Site-Topology-Renew-Attribut

Diese Klasse gibt an, wie oft der standortübergreifende Topologiegenerator die Keep-Alive-Nachricht aktualisiert, die an Domänen Controller am selben Standort gesendet wird.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Standortübergreifende Topologie-Erneuerung            |
| LDAP-Display-Name | intersitetopologyrenew               |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1247              |
| System-ID-GUID    | b7c69e5f-2cc7-11d2-854e-00a0c983f608 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
