# <a name="msi-script-path-attribute"></a>MSI-Script-path-Attribut

Der Dateipfad für die Microsoft Installer-Ankündigungs Skriptdatei für diese Anwendung.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MSI-Skript Pfad                             |
| LDAP-Display-Name | msiscriptpath                               |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.15                       |
| System-ID-GUID    | bf967937-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
