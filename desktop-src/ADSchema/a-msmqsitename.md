# <a name="msmq-site-name-attribute"></a>MSMQ-Site-Name-Attribut

| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MSMQ-Site-Name                              |
| LDAP-Display-Name | msmqsitename                                |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.965                      |
| System-ID-GUID    | ffadb4b2-de39-11d1-90a5-00c04fd91ab1        |
| Syntax            | [**String(Teletex)**](s-string-teletex.md) |
