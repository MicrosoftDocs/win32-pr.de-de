# <a name="sync-with-object-attribute"></a>Sync-with-Object-Attribut

Der Distinguished Name des Objekts, das für die SAM-Builtin Group/Local Policy-Synchronisierung synchronisiert wird.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Sync-with-Object                        |
| LDAP-Display-Name | syncwithobject                          |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.664                  |
| System-ID-GUID    | 037651e2-441d-11d1-a9c3-0000b80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
