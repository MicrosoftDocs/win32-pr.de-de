# <a name="ms-sql-publisher-attribute"></a>MS-SQL-Publisher-Attribut

Der Name der Verleger Datenbank für die Replikation.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Publisher                            |
| LDAP-Display-Name | MS-SQL-Publisher                            |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänen Administrator                        |
| Aktualisierungshäufigkeit  | Wenn die Replikation eingerichtet ist.                  |
| Attribute-Id      | 1.2.840.113556.1.4.1402                     |
| System-ID-GUID    | c1676858-d34b-11d2-999a-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
