# <a name="ms-wmi-normalizedclass-attribute"></a>MS-WMI-normalizedclass-Attribut

Der Name einer WMI-kernrichtlinienklasse.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-normalizedclass                      |
| LDAP-Display-Name | mswap-normalizedclass                       |
| Size              | Weniger als 20 Zeichen.                |
| Berechtigung aktualisieren  | Gruppenrichtlinie-Administrator                  |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1640                     |
| System-ID-GUID    | eaba628f-eb8e-4fe9-83fc-693be695559b        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
