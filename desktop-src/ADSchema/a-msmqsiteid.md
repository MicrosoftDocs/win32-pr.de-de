# <a name="msmq-site-id-attribute"></a>MSMQ-Site-ID-Attribut

Das **MSMQ-Site-ID-Attribut** enthält MSMQ-Informationen im gemischten Modus.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Site-ID                                          |
| Ldap-Anzeigename | mSMQSiteID                                            |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | \-                                                    |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.953                                |
| System-Id-Guid    | 9a0dc340-c100-11d1-bb5-0080c76670c0                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
