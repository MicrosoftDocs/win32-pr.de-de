# <a name="msmq-site-id-attribute"></a>MSMQ-Site-ID-Attribut

Das Attribut " **MSMQ-Site-ID** " enthält MSMQ-Informationen im gemischten Modus.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Site-ID                                          |
| LDAP-Display-Name | msmqsiteid                                            |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.953                                |
| System-ID-GUID    | 9a0dc340-C100-11d1-bbc5-0080c76670c0                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
