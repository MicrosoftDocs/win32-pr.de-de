# <a name="netboot-allow-new-clients-attribute"></a>NetBoot-Allow-New-Clients-Attribut

Das **netboot-Allow-New-Clients-** Attribut ist für die interne Verwendung reserviert.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | NetBoot-Allow-New-Clients            |
| LDAP-Display-Name | netbootallownewclients               |
| Size              | \-                                   |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.849               |
| System-ID-GUID    | 07383076-91df-11d1-AEbc-0000e80367c1 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
