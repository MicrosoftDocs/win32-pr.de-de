# <a name="ms-ds-never-reveal-group-attribute"></a>ms-DS-Never-Reveal-Group-Attribut

Wird mit RODCs verwendet, um zu definieren, welchen Benutzern, Computern und Gruppen die Kenn Wörter auf einem RODC zwischengespeichert werden dürfen.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Never-Reveal-Group                |
| LDAP-Display-Name | msDS-NeverRevealGroup                   |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1926                 |
| System-ID-GUID    | 15585999-fd49-4d66-b25d-eeb96aba8174    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
