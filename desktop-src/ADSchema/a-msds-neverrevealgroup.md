# <a name="ms-ds-never-reveal-group-attribute"></a>ms-DS-Never-Reveal-Group-Attribut

Wird mit RODCs verwendet, um zu definieren, welche Benutzer, Computer und Gruppen ihre Kennwörter nicht auf einem RODC zwischenspeichern dürfen.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Never-Reveal-Group                |
| Ldap-Anzeigename | msDS-NeverRevealGroup                   |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | \-                                      |
| Updatehäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1926                 |
| System-Id-Guid    | 15585999-fd49-4d66-b25d-eeb96aba8174    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
