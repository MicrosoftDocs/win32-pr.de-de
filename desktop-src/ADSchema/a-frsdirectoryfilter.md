# <a name="frs-directory-filter-attribute"></a>FRS-Directory-Filter-Attribut

Die Liste der Verzeichnisse, die von der Datei Replikation ausgeschlossen sind, z. b. Temp oder obj.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | FRS-Verzeichnis-Filter                        |
| LDAP-Display-Name | frsdirecteryfilter                          |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.484                      |
| System-ID-GUID    | 1be8f171-a9ff-11D0-afe2-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
