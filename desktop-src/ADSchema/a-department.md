# <a name="department-attribute"></a>Department-Attribut

Enthält den Namen für die Abteilung, in der der Benutzer arbeitet.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Department                                  |
| LDAP-Display-Name | department                                  |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänen Administrator oder Konto Besitzer.      |
| Aktualisierungshäufigkeit  | Immer, wenn die Abteilung geändert werden muss.    |
| Attribute-Id      | 1.2.840.113556.1.2.141                      |
| System-ID-GUID    | bf96794f-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
