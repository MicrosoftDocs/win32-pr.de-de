# <a name="print-separator-file-attribute"></a>Print-Separator-file-Attribut

Der Dateipfad der Druckertrennseite.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Druck Trennzeichen-Datei                        |
| LDAP-Display-Name | printseparatorfile                          |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.230                      |
| System-ID-GUID    | 281416c6-1968-11D0-a28f -00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
