# <a name="ms-sql-alias-attribute"></a>MS-SQL-Alias-Attribut

Ein Alias, der zum Herstellen der Verbindung mit der Datenbank verwendet wird. Default ist auf Alias festgelegt.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Alias                                |
| LDAP-Display-Name | MS-SQL-Alias                                |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1395                     |
| System-ID-GUID    | e0c6baae-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
