# <a name="extension-name-attribute"></a>Extension-Name-Attribut

Der Name einer Eigenschaften Seite, die zum Erweitern der Benutzeroberfläche eines Verzeichnis Objekts verwendet wird.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Extension-Name                              |
| LDAP-Display-Name | extensionName                               |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Der Ersteller des Erweiterungs Objekts.        |
| Aktualisierungshäufigkeit  | Immer dann, wenn eine neue Benutzeroberflächen Erweiterung erstellt wird.     |
| Attribute-Id      | 1.2.840.113556.1.2.227                      |
| System-ID-GUID    | bf967972-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
