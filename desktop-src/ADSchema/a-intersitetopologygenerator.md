# <a name="inter-site-topology-generator-attribute"></a>Standort übergreifender Topologiegenerator-Attribut

Dieses Attribut wird verwendet, um ein Failover für den Computer zu unterstützen, der die Wissens Konsistenzprüfung der standortübergreifenden Topologiegenerierung an einem bestimmten Standort ausführt.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Standort übergreifender Topologiegenerator           |
| LDAP-Display-Name | InterSiteTopologyGenerator              |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1246                 |
| System-ID-GUID    | b7c69e5e-2cc7-11d2-854e-00a0c983f608    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |



## <a name="implementations"></a>Implementierungen

-   [**Windows 2000 Server**](#windows-2000-server)
-   [**Windows Server 2003**](#windows-server-2003)
-   [**Adam**](#adam)

-   [**Windows Server 2008**](#windows-server-2008)
