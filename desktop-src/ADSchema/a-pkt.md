# <a name="pkt-attribute"></a>Pkt-Attribut

Wissens Tabelle der DFS-Partition. Beschreibt die Struktur einer verteiltes Dateisystem Hierarchie.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | PKT                                                   |
| LDAP-Display-Name | pKT                                                   |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.206                                |
| System-ID-GUID    | 8447f 1-1027-11D0-a05f -00aa006c33ed                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
