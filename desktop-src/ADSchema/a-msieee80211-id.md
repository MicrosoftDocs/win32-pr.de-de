# <a name="ms-ieee-80211-id-attribute"></a>ms-ieee-80211-ID-Attribut

Ein Bezeichner, der für ein Drahtlosrichtlinienobjekt in AD verwendet wird.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------------|
| CN                | ms-ieee-80211-ID                                                                 |
| Ldap-Anzeigename | msieee80211-ID                                                                   |
| Size              | \-                                                                               |
| Aktualisieren von Berechtigungen  | Domänenadministrator                                                             |
| Updatehäufigkeit  | Jedes Mal, wenn ein Domänenadministrator die Drahtlosnetzwerkrichtlinie für eine Domäne oder Organisationseinheit ändert. |
| Attribute-Id      | 1.2.840.113556.1.4.1823                                                          |
| System-ID-GUID    | 7f73ef75-14c9-4c23-81de-dd07a06f9e8b                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
