# <a name="ms-ieee-80211-id-attribute"></a>MS-IEEE-80211-ID-Attribut

Ein Bezeichner, der für ein drahtlos Richtlinien Objekt in AD verwendet wird.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------------|
| CN                | MS-IEEE-80211-ID                                                                 |
| LDAP-Display-Name | msieee80211-ID                                                                   |
| Size              | \-                                                                               |
| Berechtigung aktualisieren  | Domänen Administrator                                                             |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn ein Domänen Administrator die Richtlinie für Drahtlos Netzwerke für eine Domäne oder Organisationseinheit ändert. |
| Attribute-Id      | 1.2.840.113556.1.4.1823                                                          |
| System-ID-GUID    | 7F 73ef75-14c9-4c23-81de-dd07a06s9e8b                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
