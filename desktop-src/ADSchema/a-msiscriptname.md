# <a name="msi-script-name-attribute"></a>Msi-Script-Name-Attribut

Der Name der Microsoft Installer-Ankündigungsskriptdatei für diese Anwendung.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Msi-Script-Name                             |
| Ldap-Anzeigename | msiScriptName                               |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | \-                                          |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.845                      |
| System-Id-Guid    | 96a7dd62-9118-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
