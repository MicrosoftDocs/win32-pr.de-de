# <a name="msi-script-name-attribute"></a>MSI-Script-Name-Attribut

Der Name der Microsoft Installer-Ankündigungs Skriptdatei für diese Anwendung.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MSI-Skript Name                             |
| LDAP-Display-Name | msiscriptname                               |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.845                      |
| System-ID-GUID    | 96a7dd62-9118-11d1-AEbc-0000f 80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
