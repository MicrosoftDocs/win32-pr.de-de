# <a name="ms-ds-az-task-is-role-definition-attribute"></a>ms-DS-Az-Task-Is-Role-Definition-Attribut

Ein boolesches Feld, das angibt, ob AzTask eine klassische Aufgabe oder eine Rollendefinition ist.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Az-Task-Is-Role-Definition        |
| Ldap-Anzeigename | msDS-AzTaskIsRoleDefinition             |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | AzRoles admin                           |
| Updatehäufigkeit  | Während der Initialisierung oder Richtlinienänderung. |
| Attribute-Id      | 1.2.840.113556.1.4.1818                 |
| System-Id-Guid    | 7b078544-6c82-4fe9-872f-ff48ad2b2e26    |
| Syntax            | [**Boolesch**](s-boolean.md)            |
