# <a name="last-logoff-attribute"></a>Last-Logoff-Attribut

Dieses Attribut wird nicht verwendet.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Last-Logoff                          |
| LDAP-Display-Name | lastlogoff                           |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn sich der Benutzer abmeldet.         |
| Attribute-Id      | 1.2.840.113556.1.4.51                |
| System-ID-GUID    | bf967996-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervall**](s-interval.md)       |
