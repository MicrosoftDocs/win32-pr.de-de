# <a name="msmq-owner-id-attribute"></a>MSMQ-Owner-ID-Attribut

Das Attribut " **MSMQ-Owner-ID** " enthält MSMQ-Informationen im gemischten Modus.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Besitzer-ID                                         |
| LDAP-Display-Name | msmqownerid                                           |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.925                                |
| System-ID-GUID    | 9a0dc328-C100-11d1-bbc5-0080c76670c0                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
