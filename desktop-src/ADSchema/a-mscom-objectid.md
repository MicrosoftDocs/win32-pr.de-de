# <a name="ms-com-objectid-attribute"></a>MS-com-ObjectID-Attribut

GUID, die zum Identifizieren einer COM+-Partition oder eines com+-partitionsets verwendet wird.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | MS-com-ObjectID                                       |
| LDAP-Display-Name | MScom-ObjectID                                        |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1428                               |
| System-ID-GUID    | 430f 678b-889b f -41 b5a65572f                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
