| CN                | Localized-Description                       |
| LDAP-Display-Name | localizeddescription                        |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.817                      |
| System-ID-GUID    | d9e18316-8939-11d1-aebc-0000f80367c1        |
