# <a name="ms-ds-security-group-extra-classes-attribute"></a>ms-DS-Security-Group-extra-Classes-Attribut

Die allgemeinen Namen der nicht standardmäßigen Klassen, die über das Snap-in "Active Directory-Benutzer und-Computer" zu einer Sicherheitsgruppe hinzugefügt werden können.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Security-Group-extra-Classes          |
| LDAP-Display-Name | MSDS-Security-Group-extra-Classes           |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänenadministrator                        |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1688                     |
| System-ID-GUID    | 4f 146 AE8-a4fe-4801-a731-f51848a4f4e4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
