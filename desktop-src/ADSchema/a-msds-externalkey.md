# <a name="ms-ds-external-key-attribute"></a>ms-DS-externes-Key-Attribut

Eine Zeichenfolge, die ein Objekt in einem externen Speicher identifiziert, z. b. ein Datensatz in einer Datenbank.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-DS-extern-Schlüssel                          |
| LDAP-Display-Name | MSDS-ExternalKey                            |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1833                     |
| System-ID-GUID    | b92fd528-38ac-40d4-818d-0433380837c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
