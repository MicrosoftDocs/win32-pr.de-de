# <a name="ms-ds-is-user-cachable-at-rodc-attribute"></a>ms-DS-is-User-cachable-at-RODC-Attribut

Gibt bei einem Read-Only Domänen Controller (RODC) an, ob die geheimen Schlüssel des angegebenen Benutzers zwischengespeichert werden können.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ms-DS-ist-User-cachable-at-RODC       |
| LDAP-Display-Name | MSDS-isusercachableatrodc            |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2025              |
| System-ID-GUID    | fe01245a-341f-4556-951f-48c033a89050 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
