# <a name="nt-group-members-attribute"></a>NT-Group-Members-Attribut

Dieses Attribut wird nicht verwendet.

Das [**Member**](a-member.md) -Attribut wird verwendet, um die Mitglieder einer Gruppe zu enthalten.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | NT-Gruppenmitglieder                                      |
| LDAP-Display-Name | ntgroupmembers                                        |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.89                                 |
| System-ID-GUID    | bf9679df-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
