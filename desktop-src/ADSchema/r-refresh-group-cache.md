# <a name="refresh-group-cache-extended-right"></a>Rechte "Refresh-Group-Cache" erweitert

Dies gilt für keine GC-Anmeldung. Keine GC-Anmeldung basiert auf Zwischenspeichern von Gruppenmitgliedschaften, und dieses Steuerelementzugriffsrecht wird verwendet, um Administratoren und Operatoren Berechtigungen zu erteilen, um eine sofortige Aktualisierung des Caches zu verursachen und eine verfügbare GC zu kontaktieren.



| Eingabe | Wert |
|--------------|--------------------------------------|
| CN           | Refresh-Group-Cache                  |
| Anzeigename | Aktualisieren des Gruppencaches für Anmeldungen       |
| Rechte-GUID  | 9432c620-033c-4db7-8b58-14ef6d0bf477 |
