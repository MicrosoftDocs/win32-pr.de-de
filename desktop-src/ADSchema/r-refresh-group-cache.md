# <a name="refresh-group-cache-extended-right"></a>Refresh-Group-Cache erweitert rechts

Dies gilt für keine GC-Anmeldung. Keine GC-Anmeldung stützt sich auf das Zwischenspeichern von Gruppenmitgliedschaften, und dieses Zugriffsrecht wird verwendet, um Administratoren und Operatoren Berechtigungen zu erteilen, um eine sofortige Aktualisierung des Caches zu ermöglichen und eine Verbindung mit einem verfügbaren GC aufzunehmen.



| Eingabe | Wert |
|--------------|--------------------------------------|
| CN           | Refresh-Group-Cache                  |
| Anzeigename | Gruppen Cache für Anmeldungen aktualisieren       |
| Rights-GUID  | 9432c620-033c-4db7-8b58-14ef6d0bf 477 |
