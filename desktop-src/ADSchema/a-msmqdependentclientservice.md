# <a name="msmq-dependent-client-service-attribute"></a>MSMQ-Dependent-Client-Service-Attribut

Gibt an, ob dieser Server Dienste für abhängige Clients bereitstellt.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MSMQ-abhängige-Client-Service        |
| LDAP-Display-Name | msmqdependentclientservice           |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1239              |
| System-ID-GUID    | 2df90d83-009F -11d2-aa4c-00c04f d83a |
| Syntax            | [**Booleschen**](s-boolean.md)         |
