# <a name="reps-to-attribute"></a>Reps-To-Attribut

Listet die Server auf, über die das Verzeichnis über Änderungen und Server benachrichtigt wird, an die das Verzeichnisänderungen an der Anforderung für den definierten Namens Kontext sendet.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Reps-To                                               |
| LDAP-Display-Name | repsto                                                |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.83                                 |
| System-ID-GUID    | bf967a1e-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
