# <a name="ms-ds-schema-extensions-attribute"></a>ms-DS-Schema-Extensions-Attribut

Ein binäres Blob, das zum Speichern von Informationen über Erweiterungen von Schema Objekten verwendet wird.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | ms-DS-Schema-Extensions                               |
| LDAP-Display-Name | MSDS-Schema-Extensions                                |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                      |
| Aktualisierungshäufigkeit  | Wenn das Schema erweitert wird.                          |
| Attribute-Id      | 1.2.840.113556.1.4.1440                               |
| System-ID-GUID    | b39a61be-ed07-4cab-9a4a-4963ed0141e1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
