# <a name="operating-system-attribute"></a>Operating-System-Attribut

Der Name des Betriebssystems, z. b. Windows Vista Enterprise.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Operating-System                            |
| LDAP-Display-Name | operatingSystem                             |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.363                      |
| System-ID-GUID    | 3e978925-8c01-11D0-AFDA-00c04f 930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
