# <a name="acs-cache-timeout-attribute"></a>ACS-Cache-Timeout-Attribut

Die Zeitspanne vor dem Ablauf von ACS-Objekten, die vom ACS-Dienst zwischengespeichert werden.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-Cache-Timeout                    |
| LDAP-Display-Name | acscachetimeout                      |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.779               |
| System-ID-GUID    | 1cb355a1-56d0-11d1-a9c6-0000b80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
