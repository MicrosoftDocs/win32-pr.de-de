# <a name="rid-manager-reference-attribute"></a>RID-Manager-Verweis Attribut

Der Distinguished Name für den RID-Manager eines Objekts.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | RID-Manager-Referenz                   |
| LDAP-Display-Name | ridmanagerreference                     |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.        |
| Aktualisierungshäufigkeit  | Wenn der RID-Manager geändert wird.       |
| Attribute-Id      | 1.2.840.113556.1.4.368                  |
| System-ID-GUID    | 66171886-8F -11D0-AFDA-00c04f-930c9    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
