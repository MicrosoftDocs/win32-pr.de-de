Das **msradiherframedroute** -Attribut wird intern verwendet. Ändern Sie diesen Wert nicht direkt.

|-------------------|--------------------------------------|
| CN                | msradius framedroute                  |
| LDAP-Display-Name | msradius framedroute                  |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
