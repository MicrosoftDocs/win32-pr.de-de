# <a name="curr-machine-id-attribute"></a>Currr-Machine-ID-Attribut

Die ID des Computers, auf dem sich ein Link-Track-Vol-Entry-Objekt befindet.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Cursor-Computer-ID                                       |
| LDAP-Display-Name | currmachineid                                         |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                      |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.337                                |
| System-ID-GUID    | 1F 0075fe-7E40-11D0-afd6-00c04f 930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
