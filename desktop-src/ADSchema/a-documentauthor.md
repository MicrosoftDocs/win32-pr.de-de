# <a name="documentauthor-attribute"></a>DocumentAuthor-Attribut

Der DocumentAuthor-Attributtyp gibt den Distinguished Name des Autors eines Dokuments an.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | DocumentAuthor                          |
| LDAP-Display-Name | DocumentAuthor                          |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 0.9.2342.19200300.100.1.14              |
| System-ID-GUID    | f18a8e19-AF5F-4478-B096-6f 35c27eb83f    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
