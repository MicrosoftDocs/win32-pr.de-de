# <a name="shadowlastchange-attribute"></a>shadowLastChange-Attribut

| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | shadowLastChange                     |
| Ldap-Anzeigename | shadowLastChange                     |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.5                      |
| System-ID-GUID    | f8f2689c-29e8-4843-8177-e8b98e15eeac |
