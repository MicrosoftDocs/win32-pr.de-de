# <a name="pwd-last-set-attribute"></a>PWD-letztes Set-Attribut

Das Datum und die Uhrzeit der letzten Änderung des Kennworts für dieses Konto. Dieser Wert wird als große ganze Zahl gespeichert, die die Anzahl von 100 Nanosekunden-Intervallen seit dem 1. Januar 1601 (UTC) darstellt. Wenn dieser Wert auf 0 festgelegt ist und das Attribut " [**User-Account-Control**](a-useraccountcontrol.md) " nicht das Kennwort " **UF \_ 't \_ expire \_ passwd** " enthält, muss der Benutzer das Kennwort bei der nächsten Anmeldung festlegen.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | PWD-letztes Set                         |
| LDAP-Display-Name | pwdLastSet                           |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn das Kennwort geändert wird.   |
| Attribute-Id      | 1.2.840.113556.1.4.96                |
| System-ID-GUID    | bf967a0a-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervall**](s-interval.md)       |
