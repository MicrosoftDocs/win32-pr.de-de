# <a name="control-access-rights-attribute"></a>Control-Access-Rights-Attribut

Wird von der DS-Sicherheit verwendet, um zu bestimmen, welche Benutzer bestimmte Vorgänge für das Hostobjekt ausführen können.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Control-Access-Rights                                 |
| Ldap-Anzeigename | controlAccessRights                                   |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | \-                                                    |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.200                                |
| System-ID-GUID    | 6da8a4fc-0e52-11d0-a286-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
