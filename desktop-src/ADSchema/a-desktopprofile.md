# <a name="desktop-profile-attribute"></a>Desktop-Profile-Attribut

Der Speicherort des Desktop Profils für einen Benutzer oder eine Gruppe von Benutzern. Nicht verwendet.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Desktop-Profile                             |
| LDAP-Display-Name | desktopprofile                              |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.346                      |
| System-ID-GUID    | eea65906-8ac6-11d0-afda-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
