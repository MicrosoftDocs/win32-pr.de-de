# <a name="extended-class-info-attribute"></a>Extended-Class-Info-Attribut

Eine mehrwertige Eigenschaft, die Zeichen folgen enthält, die zusätzliche Informationen für jede Klasse darstellen. Jeder Wert enthält die governsID, den ldapDisplayName und die schemaIDGUID der Klasse.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Extended-Class-Info                         |
| LDAP-Display-Name | extendedclassinfo                           |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.908                      |
| System-ID-GUID    | 9a7ad948-ca53-11d1-bbd0-0080c76670c0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
