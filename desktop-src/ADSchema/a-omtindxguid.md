# <a name="omt-indx-guid-attribute"></a>OMT-INDX-GUID-Attribut

Der Index Bezeichner für einen Link-Track-Object-Move-Tabelleneintrag.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | OMT-INDX-GUID                                         |
| LDAP-Display-Name | omtindxguid                                           |
| Size              | 16 Bytes                                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                      |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn sich der Link für eine Datei ändert.                 |
| Attribute-Id      | 1.2.840.113556.1.4.333                                |
| System-ID-GUID    | 1F 0075fa-7E40-11D0-afd6-00c04f 930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
