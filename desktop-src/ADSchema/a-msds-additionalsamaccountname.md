# <a name="ms-ds-additional-sam-account-name-attribute"></a>ms-DS-additional-Sam-Account-Name-Attribut

Dieses Attribut wird verwendet, um die SAM-Kontonamen zu speichern, die den DNS-Hostnamen in "ms-DS-additional-DNS-Host-Name" entsprechen.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-DS-additional-Sam-Account-Name           |
| LDAP-Display-Name | MSDS-additionalsamaccountname               |
| Size              | Weniger als 20 Bytes.                         |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.            |
| Aktualisierungshäufigkeit  | Wenn der DC umbenannt wird.                     |
| Attribute-Id      | 1.2.840.113556.1.4.1718                     |
| System-ID-GUID    | 975571df-a4d5-429A-9F 59-cdc6581d91e6        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
