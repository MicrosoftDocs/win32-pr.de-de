# <a name="change-pdc-extended-right"></a>Ändern-PDC-Recht erweitert

Das erweiterte Recht ist erforderlich, um den FSMO-Emulator (Primary Domain Controller) der FSMO-Rolle zu ändern.



| Eingabe | Wert |
|--------------|--------------------------------------|
| CN           | Ändern-PDC                           |
| Anzeigename | Ändern des PDC                           |
| Rights-GUID  | bae50096-4752-11d1-9052-00c04fc2d4cf |
