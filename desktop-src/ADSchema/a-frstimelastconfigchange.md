# <a name="frs-time-last-config-change-attribute"></a>FRS-Time-Last-config-Change-Attribut

Der Zeitpunkt, zu dem die letzte Konfigurationsänderung akzeptiert wurde.



| Eingabe | Wert |
|-------------------|---------------------------------------------------------------|
| CN                | FRS-Zeit-letzte-config-Änderung                                   |
| LDAP-Display-Name | frstimelastconfigchange                                       |
| Size              | \-                                                            |
| Berechtigung aktualisieren  | \-                                                            |
| Aktualisierungshäufigkeit  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.881                                        |
| System-ID-GUID    | 2a132584-9373-11d1-AEbc-0000 C1                          |
| Syntax            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
