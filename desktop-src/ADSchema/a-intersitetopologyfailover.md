# <a name="inter-site-topology-failover-attribute"></a>Standortübergreifende Topologie: Failover-Attribut

Gibt an, wie viel Zeit seit dem letzten Keep-Alive für den standortübergreifenden Topologiegenerator in Erwägung gezogen werden muss.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Standortübergreifende Topologie-Failover         |
| LDAP-Display-Name | intersitetopologyfailover            |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1248              |
| System-ID-GUID    | b7c69e60-2cc7-11d2-854e-00a0c983f608 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
