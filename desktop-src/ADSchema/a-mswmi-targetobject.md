# <a name="ms-wmi-targetobject-attribute"></a>MS-WMI-TargetObject-Attribut

Enthält mindestens eine binäre Sequenz, die kompilierte WMI-Objekte darstellt.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | MS-WMI-TargetObject                                   |
| LDAP-Display-Name | mswap-TargetObject                                    |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                      |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1647                               |
| System-ID-GUID    | c44f67a5-7de5-4a1f-92d9-662b57364b77                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
