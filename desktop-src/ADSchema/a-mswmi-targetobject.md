# <a name="ms-wmi-targetobject-attribute"></a>ms-WMI-TargetObject-Attribut

Dieser enthält eine oder mehrere binäre Sequenzen, die kompilierte WMI-Objekte darstellen.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | ms-WMI-TargetObject                                   |
| Ldap-Anzeigename | msWMI-TargetObject                                    |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.                      |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1647                               |
| System-Id-Guid    | c44f67a5-7de5-4a1f-92d9-662b57364b77                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
