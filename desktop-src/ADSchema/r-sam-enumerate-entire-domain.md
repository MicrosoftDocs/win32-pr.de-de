# <a name="sam-enumerate-entire-domain-extended-right"></a>Erweitertes Recht "SAM-Enumerate-Entire-Domain"

Dieses Zugriffssteuerungsrecht kann verwendet werden, um einzuschränken, wer eine kompatible API wie NetQueryDisplayInformation und NetUser/GroupEnum verwenden darf, und die gesamte Domäne aufzuzählen.



| Eingabe | Wert |
|--------------|--------------------------------------|
| CN           | SAM-Enumerate-Entire-Domain          |
| Anzeigename | Aufzählen der gesamten SAM-Domäne          |
| Rechte-GUID  | 91d67418-0135-4acc-8d79-c08e857cfbec |
