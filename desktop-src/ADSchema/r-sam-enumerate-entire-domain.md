# <a name="sam-enumerate-entire-domain-extended-right"></a>Sam-Enumerate-gesamte Domäne erweitert rechts

Mit diesem Zugriffsrecht steuern können Sie einschränken, wer eine kompatible-API wie NetQueryDisplayInformation und nettuser/groupum verwenden darf, und die gesamte Domäne auflisten.



| Eingabe | Wert |
|--------------|--------------------------------------|
| CN           | Sam-Enumerate-gesamte Domäne          |
| Anzeigename | Gesamte Sam-Domäne aufzählen          |
| Rights-GUID  | 91d67418-0135-4acc-8d79-c08e857cfbec |
