# <a name="last-backup-restoration-time-attribute"></a>Letztes Backup-Wiederherstellungszeit Attribut

Zeitpunkt der letzten Systemwiederherstellung.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Letzte Sicherung-Wiederherstellungszeit         |
| LDAP-Display-Name | lastbackuprestorationtime            |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.519               |
| System-ID-GUID    | 1fbb0be8-BA63-11D0-Afef-0000f80367c1 |
| Syntax            | [**Intervall**](s-interval.md)       |
