# <a name="employee-number-attribute"></a>Employee-Number-Attribut

Die Zahl, die einem anderen Mitarbeiter als der ID zugewiesen ist.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Employee-Number                             |
| LDAP-Display-Name | employeeNumber                              |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Domänen Administrator                        |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.610                      |
| System-ID-GUID    | a8df73ef-c5ea-11d1-bbcb-0080c76670c0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
