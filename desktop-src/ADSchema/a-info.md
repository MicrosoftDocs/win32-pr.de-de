# <a name="comment-attribute-ad-schema"></a>Comment-Attribut (AD-Schema)

Die Kommentare des Benutzers. Diese Zeichenfolge kann eine NULL-Zeichenfolge sein.



| Eingabe | Wert |
|-------------------|--------------------------------------------------------------------------|
| CN                | Kommentar                                                                  |
| LDAP-Display-Name | info                                                                     |
| Size              | \-                                                                       |
| Berechtigung aktualisieren  | Domänen Administrator oder Konto Besitzer.                                   |
| Aktualisierungshäufigkeit  | Immer dann, wenn der Benutzer oder Administrator Kommentare zum Konto hinzufügen muss. |
| Attribute-Id      | 1.2.840.113556.1.2.81                                                    |
| System-ID-GUID    | bf96793e-0de6-11d0-a285-00aa003049e2                                     |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                              |
