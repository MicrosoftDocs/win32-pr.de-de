# <a name="user-shared-folder-attribute"></a>Benutzer-Shared-Folder-Attribut

Gibt einen UNC-Pfad zum freigegebenen Dokument Ordner des Benutzers an. Der Pfad muss ein Netzwerk-UNC-Pfad des Formular **\\\\** _Server_ *_\\_* _Freigabe_- *_\\_* _Verzeichnisses_ sein. Dieser Wert kann eine NULL-Zeichenfolge sein.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------------------------|
| CN                | Benutzer-Shared-Folder                                                                |
| LDAP-Display-Name | usersharedfolder                                                                  |
| Size              | \-                                                                                |
| Berechtigung aktualisieren  | Domänen Administrator oder Konto Besitzer.                                            |
| Aktualisierungshäufigkeit  | Wenn der Benutzerdaten Satz erstellt und der freigegebene Ordner geändert werden muss. |
| Attribute-Id      | 1.2.840.113556.1.4.751                                                            |
| System-ID-GUID    | 9a9a021b-4a5b-11d1-a9c3-0000b80367c1                                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                       |
