# <a name="msmq-routing-services-attribute"></a>MSMQ-Routing-Dienste-Attribut

Gibt an, ob MSMQ, das auf diesem Computer installiert ist, MSMQ-Routing Dienste bereitstellt.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MSMQ-Routing Dienste                |
| LDAP-Display-Name | msmqroutingservices                  |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1227              |
| System-ID-GUID    | 2df90d77-009F -11d2-aa4c-00c04f d83a |
| Syntax            | [**Booleschen**](s-boolean.md)         |
