# <a name="transport-dll-name-attribute"></a>Transport-DLL-Name-Attribut

Der Name der dll, von der ein Transport verwaltet wird.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Transport-DLL-Name                          |
| LDAP-Display-Name | transportdllname                            |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.            |
| Aktualisierungshäufigkeit  | Beim Verbinden von Standorten.                      |
| Attribute-Id      | 1.2.840.113556.1.4.789                      |
| System-ID-GUID    | 26d97372-6070-11d1-a9c6-0000b80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
