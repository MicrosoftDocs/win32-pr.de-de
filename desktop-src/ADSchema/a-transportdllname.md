# <a name="transport-dll-name-attribute"></a>Transport-DLL-Name-Attribut

Der Name der DLL, die einen Transport verwaltet.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Transport-DLL-Name                          |
| Ldap-Anzeigename | transportDLLName                            |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.            |
| Updatehäufigkeit  | Beim Verbinden von Standorten.                      |
| Attribute-Id      | 1.2.840.113556.1.4.789                      |
| System-ID-GUID    | 26d97372-6070-11d1-a9c6-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
