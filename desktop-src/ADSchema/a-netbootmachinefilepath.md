# <a name="netboot-machine-file-path-attribute"></a>Netboot-Machine-File-Path-Attribut

Dieses Attribut gibt den Server an, der den Client beantwortet. Ab dem Betriebssystem Windows Server 2003 kann dies die Startrom.com angeben, die der Client erhält.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Netboot-Machine-File-Path                   |
| Ldap-Anzeigename | netbootMachineFilePath                      |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.            |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.361                      |
| System-Id-Guid    | 3e978923-8c01-11d0-afda-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
