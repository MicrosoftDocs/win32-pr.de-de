# <a name="netboot-machine-file-path-attribute"></a>NetBoot-Machine-File-path-Attribut

Dieses Attribut gibt den Server an, der den Client beantwortet. Beginnend mit dem Betriebssystem Windows Server 2003 können Sie den Startrom.com-Wert angeben, der vom Client abgerufen wird.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | NetBoot-Machine-Dateipfad                   |
| LDAP-Display-Name | netbootMachineFilePath                      |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.            |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.361                      |
| System-ID-GUID    | 3e978923-8c01-11D0-AFDA-00c04f 930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
