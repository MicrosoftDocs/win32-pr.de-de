# <a name="netboot-guid-attribute"></a>NetBoot-GUID-Attribut

Start ohne Datenträger: die on-Board-GUID eines Computers. Entspricht der Mac-Adresse für die Netzwerkkarte des Computers.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | NetBoot-GUID                                          |
| LDAP-Display-Name | netbootguid                                           |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                      |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.359                                |
| System-ID-GUID    | 3e978921-8c01-11D0-AFDA-00c04f 930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
