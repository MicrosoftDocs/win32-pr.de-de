# <a name="rdn-att-id-attribute"></a>RDN-ATT-ID-Attribut

Der RDN für das Attribut, das verwendet wird, um eine Klasse zu benennen.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------------------|
| CN                | RDN-ATT-ID                                                      |
| LDAP-Display-Name | rDNAttID                                                        |
| Size              | \-                                                              |
| Berechtigung aktualisieren  | Schema Administrator                                            |
| Aktualisierungshäufigkeit  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.26                                           |
| System-ID-GUID    | bf967a0f-0de6-11d0-a285-00aa003049e2                            |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
