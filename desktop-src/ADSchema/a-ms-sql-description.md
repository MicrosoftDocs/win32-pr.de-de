# <a name="ms-sql-description-attribute"></a>MS-SQL-Description-Attribut

Die Beschreibung der SQL Server Datenbank.



| Eingabe | Wert |
|-------------------|--------------------------------------------------|
| CN                | MS-SQL-Description                               |
| LDAP-Display-Name | MS-SQL-Description                               |
| Size              | \-                                               |
| Berechtigung aktualisieren  | Domänen Administrator                             |
| Aktualisierungshäufigkeit  | Wenn der Benutzer entscheidet, die Beschreibung zu ändern. |
| Attribute-Id      | 1.2.840.113556.1.4.1390                          |
| System-ID-GUID    | 8386603c-ccef-11d2-9993-0000f 87a57d4             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)      |
