# <a name="ms-ieee-80211-data-attribute"></a>MS-IEEE-80211-Data-Attribut

Speichert eine Liste der bevorzugten Netzwerkkonfigurationen in Gruppenrichtlinie für drahtlose Netzwerke.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------------|
| CN                | MS-IEEE-80211-Data                                                               |
| LDAP-Display-Name | msieee80211-Daten                                                                 |
| Size              | \-                                                                               |
| Berechtigung aktualisieren  | Domänen Administrator                                                             |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn ein Domänen Administrator die Richtlinie für Drahtlos Netzwerke für eine Domäne oder Organisationseinheit ändert. |
| Attribute-Id      | 1.2.840.113556.1.4.1821                                                          |
| System-ID-GUID    | 0e0d0938-2658-4580-a9f 6-7a0ac7b566cb                                             |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                            |
