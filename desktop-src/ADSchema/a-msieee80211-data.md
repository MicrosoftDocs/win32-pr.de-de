# <a name="ms-ieee-80211-data-attribute"></a>ms-ieee-80211-Data-Attribut

Speichert eine Liste der bevorzugten Netzwerkkonfigurationen in Gruppenrichtlinie für Drahtlosnetzwerke.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------------|
| CN                | ms-ieee-80211-Data                                                               |
| Ldap-Anzeigename | msieee80211-Data                                                                 |
| Size              | \-                                                                               |
| Aktualisieren von Berechtigungen  | Domänenadministrator                                                             |
| Updatehäufigkeit  | Jedes Mal, wenn ein Domänenadministrator die Drahtlosnetzwerkrichtlinie für eine Domäne oder Organisationseinheit ändert. |
| Attribute-Id      | 1.2.840.113556.1.4.1821                                                          |
| System-ID-GUID    | 0e0d0938-2658-4580-a9f6-7a0ac7b566cb                                             |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                            |
