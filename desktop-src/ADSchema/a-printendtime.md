# <a name="print-end-time-attribute"></a>Print-End-Time-Attribut

Die Zeit, zu der eine Druckwarteschlange die Wartung von Aufträgen beendet.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Druckendezeit                       |
| Ldap-Anzeigename | printEndTime                         |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.234               |
| System-ID-GUID    | 281416ca-1968-11d0-a28f-00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
