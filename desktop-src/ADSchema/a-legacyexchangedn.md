# <a name="legacy-exchange-dn-attribute"></a>Legacy-Exchange-DN-Attribut

Der Distinguished Name, der zuvor von Exchange verwendet wurde.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Legacy-Exchange-DN                          |
| LDAP-Display-Name | legacyExchangeDN                            |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.655                      |
| System-ID-GUID    | 28630ebc-41d5-11d1-a9c1-0000b80367c1        |
| Syntax            | [**String(Teletex)**](s-string-teletex.md) |
