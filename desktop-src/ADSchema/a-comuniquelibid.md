# <a name="com-unique-libid-attribute"></a>COM-Unique-LIBID-Attribut

Einwertige Zeichen folgen-GUID LIBID für eine Typbibliothek.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | COM-Unique-LIBID                            |
| LDAP-Display-Name | comuniquelibid                              |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.250                      |
| System-ID-GUID    | 281416da-1968-11D0-a28f -00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
