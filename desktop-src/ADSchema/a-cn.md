# <a name="common-name-attribute"></a>Common-Name-Attribut

Der Name, der ein Objekt darstellt. Wird zum Ausführen von Suchvorgängen verwendet.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Common-Name                                 |
| Ldap-Anzeigename | cn                                          |
| Size              | Die durchschnittliche Größe beträgt 64 Bytes.                   |
| Aktualisieren von Berechtigungen  | Domänenadministrator                        |
| Updatehäufigkeit  | Nur wenn ein neues -Objekt erstellt wird.          |
| Attribute-Id      | 2.5.4.3                                     |
| System-Id-Guid    | bf96793f-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
