# <a name="common-name-attribute"></a>Common-Name-Attribut

Der Name, der ein Objekt darstellt. Wird verwendet, um Suchvorgänge auszuführen.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Common-Name                                 |
| LDAP-Display-Name | cn                                          |
| Size              | Die durchschnittliche Größe beträgt 64 Bytes.                   |
| Berechtigung aktualisieren  | Domänen Administrator                        |
| Aktualisierungshäufigkeit  | Nur, wenn ein neues-Objekt erstellt wird.          |
| Attribute-Id      | 2.5.4.3                                     |
| System-ID-GUID    | bf96793f-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
