# <a name="frs-version-guid-attribute"></a>FRS-Version-GUID-Attribut

Falls vorhanden, wird durch Ändern dieses Werts angegeben, dass für diese Replikat Gruppe eine Konfigurationsänderung vorgenommen wurde.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | FRS-Version-GUID                                      |
| LDAP-Display-Name | frsversionguid                                        |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.43                                 |
| System-ID-GUID    | 26d9736c-6070-11d1-a9c6-0000 C1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
