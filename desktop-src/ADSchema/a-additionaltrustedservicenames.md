# <a name="additional-trusted-service-names-attribute"></a>Zusätzliches-Trusted-Service-names-Attribut

Eine Liste der Dienste in der Domäne, die als vertrauenswürdig eingestuft werden können. Wird nicht von AD verwendet.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Zusätzliche-vertrauenswürdige Dienst-Namen            |
| LDAP-Display-Name | additionaltreuhändservicenames               |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.889                      |
| System-ID-GUID    | 032160be-9824-11d1-aec0-0000e80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
