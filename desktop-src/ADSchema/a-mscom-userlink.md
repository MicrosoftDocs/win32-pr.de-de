# <a name="ms-com-userlink-attribute"></a>MS-com-userlink-Attribut

Ein Link, der verwendet wird, um ein com+-partitionset einem Benutzerobjekt zuzuordnen.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | MS-com-userlink                         |
| LDAP-Display-Name | MScom-userlink                          |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.        |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1425                 |
| System-ID-GUID    | 9e6f 3a4d-242c-4B B068-36b57f fc852    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
