# <a name="nt-pwd-history-attribute"></a>NT-pwd-History-Attribut

Der Kenn Wort Verlauf des Benutzers im unidirektionalen Windows NT-Format (owf). Windows 2000 verwendet Windows NT owf.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | NT-pwd-Verlauf                                        |
| LDAP-Display-Name | ntpwdhistory                                          |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                      |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn der Benutzer Kenn Wörter ändert.                 |
| Attribute-Id      | 1.2.840.113556.1.4.94                                 |
| System-ID-GUID    | bf9679e2-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
