# <a name="gidnumber-attribute"></a>gidNumber-Attribut

Enthält einen ganzzahligen Wert, der eine Gruppe in einer administrativen Domäne eindeutig identifiziert.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | gidNumber                            |
| LDAP-Display-Name | gidNumber                            |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.1                      |
| System-ID-GUID    | c5b95f0c-ec9e-41c4-849c-b46597ed6696 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
