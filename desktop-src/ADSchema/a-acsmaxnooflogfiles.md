| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-Max-No-of-Log-Files              |
| LDAP-Display-Name | acsmaxnooflogfiles                   |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.774               |
| System-ID-GUID    | 1cb3559c-56d0-11d1-a9c6-0000b80367c1 |
