# <a name="msmq-ds-service-attribute"></a>MSMQ-DS-Service-Attribut

| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MSMQ-DS-Service                      |
| LDAP-Display-Name | msmqdsservice                        |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1238              |
| System-ID-GUID    | 2df90d82-009F -11d2-aa4c-00c04f d83a |
| Syntax            | [**Booleschen**](s-boolean.md)         |
