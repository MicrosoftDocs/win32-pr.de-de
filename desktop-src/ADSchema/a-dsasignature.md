# <a name="dsa-signature-attribute"></a>DSA-Signature-Attribut

Der DSA-Signature eines Objekts ist die Aufruf-ID des letzten Verzeichnisses, um das Objekt zu ändern.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | DSA-Signature                                         |
| LDAP-Display-Name | DSASignature                                          |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.                      |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn ein Objekt geändert wird.                       |
| Attribute-Id      | 1.2.840.113556.1.2.74                                 |
| System-ID-GUID    | 167757bc-47b3-11d1-a9c3-0000-C1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
