# <a name="admin-property-pages-attribute"></a>Admin-Property-Pages-Attribut

Die Bestellnummer und die GUID der Eigenschaften Seiten für ein Objekt, das auf Active Directory Verwaltungs Bildschirmen angezeigt werden soll. Weitere Informationen finden Sie im Dokument Erweitern der Benutzeroberfläche für Verzeichnisobjekte.



| Eingabe | Wert |
|-------------------|------------------------------------------------|
| CN                | Admin-Eigenschaften Seiten                           |
| LDAP-Display-Name | adminPropertyPages                             |
| Size              | \-                                             |
| Berechtigung aktualisieren  | Domänen Administrator oder Anwendungsentwickler. |
| Aktualisierungshäufigkeit  | Immer dann, wenn eine Eigenschaften Seite hinzugefügt oder entfernt wird.  |
| Attribute-Id      | 1.2.840.113556.1.4.562                         |
| System-ID-GUID    | 52458038-ca6a-11D0-affinf-0000f 80367c1           |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)    |
