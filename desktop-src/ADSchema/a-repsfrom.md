# <a name="reps-from-attribute"></a>Reps-From-Attribut

Listet die Server auf, von denen das Verzeichnis Änderungen für den definierten Namenskontext akzeptiert.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Reps-From                                             |
| Ldap-Anzeigename | repsFrom                                              |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | \-                                                    |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.91                                 |
| System-ID-GUID    | bf967a1d-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
