# <a name="reps-from-attribute"></a>Reps-From-Attribut

Listet die Server auf, von denen das Verzeichnisänderungen für den definierten Namens Kontext annimmt.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Reps-From                                             |
| LDAP-Display-Name | repsfrom                                              |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.91                                 |
| System-ID-GUID    | bf967a1d-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
