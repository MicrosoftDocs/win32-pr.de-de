# <a name="unexpire-password-extended-right"></a>Erweiterte Rechte Unexpire-Password

Zugriffsrecht für erweiterte Steuerelemente, mit dem ein Benutzer ein abgelaufenes Kennwort für ein Benutzerobjekt wiederherstellen kann.



| Eingabe | Wert |
|--------------|--------------------------------------|
| CN           | Abgelaufenes Kennwort wiederherstellen                    |
| Anzeigename | Kennwort nicht ablaufen                    |
| Rights-GUID  | ccc2dc7d-a6ad-4a7a-8846-c04e3cc53501 |
