# <a name="unexpire-password-extended-right"></a>Unexpire-Password erweitertes Recht

Erweitertes Steuerelementzugriffsrecht, das es einem Benutzer ermöglicht, ein abgelaufenes Kennwort für ein Benutzerobjekt wiederherzustellen.



| Eingabe | Wert |
|--------------|--------------------------------------|
| CN           | Abgelaufenes Kennwort wiederherstellen                    |
| Anzeigename | Unexpire Password                    |
| Rights-GUID  | pflicht2dc7d-a6ad-4a7a-8846-c04e3cc53501 |
