# <a name="msi-file-list-attribute"></a>Msi-file-List-Attribut

Dieses Attribut enthält eine Liste von Microsoft Installer-Dateien, z. b. die MSI-Basisdatei (MSI) und MST-Transformations Dateien (. MST).



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MSI-Datei Liste                               |
| LDAP-Display-Name | msifilelist                                 |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.671                      |
| System-ID-GUID    | 7bf dcb7d-4807-11d1-a9c3-0000b80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
