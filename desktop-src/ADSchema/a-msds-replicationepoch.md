# <a name="ms-ds-replicationepoch-attribute"></a>ms-DS-ReplicationEpoch-Attribut

Diese wird verwendet, um die Epoche aufzunehmen, in der alle DCS replizieren. Eine Epoche ist die Zeitspanne, in der eine Domäne einen bestimmten Namen aufweist. Eine neue Epoche beginnt, wenn eine Domänen Namensänderung auftritt.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ms-DS-ReplicationEpoch               |
| LDAP-Display-Name | MSDS-ReplicationEpoch                |
| Size              | \-                                   |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | Nur während der Domänen Umstrukturierung.      |
| Attribute-Id      | 1.2.840.113556.1.4.1720              |
| System-ID-GUID    | 08e3aa79-eb1c-45b5-af7b-8l94246c8e41 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
