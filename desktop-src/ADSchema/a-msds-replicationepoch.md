# <a name="ms-ds-replicationepoch-attribute"></a>ms-DS-ReplicationEpoch-Attribut

Dies wird verwendet, um die Epoche zu speichern, in der alle Dcs repliziert werden. Eine Epoche ist der Zeitraum, in dem eine Domäne über einen bestimmten Namen verfügt. Eine neue Epoche beginnt, wenn eine Änderung des Domänennamens auftritt.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ms-DS-ReplicationEpoch               |
| Ldap-Anzeigename | msDS-ReplicationEpoch                |
| Size              | \-                                   |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.     |
| Updatehäufigkeit  | Nur während der Domänenumstrukturierung.      |
| Attribute-Id      | 1.2.840.113556.1.4.1720              |
| System-ID-GUID    | 08e3aa79-eb1c-45b5-af7b-8f94246c8e41 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
