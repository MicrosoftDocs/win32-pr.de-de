# <a name="telex-number-attribute"></a>Telex-Number-Attribut

Eine Liste alternativer Telex-Nummern.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Telex-Number                                                                     |
| LDAP-Display-Name | telexnumber                                                                      |
| Size              | \-                                                                               |
| Berechtigung aktualisieren  | Jeder kann dieses Objekt basierend auf der Sicherheit des Objekts, das erstellt wird, aktualisieren. |
| Aktualisierungshäufigkeit  | \-                                                                               |
| Attribute-Id      | 2.5.4.21                                                                         |
| System-ID-GUID    | bf967a4b-0de6-11d0-a285-00aa003049e2                                             |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                            |
