# <a name="print-form-name-attribute"></a>Print-Form-Name-Attribut

Der Name des zurzeit geladenen Formulars.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Print-Formular Name                             |
| LDAP-Display-Name | printformname                               |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.235                      |
| System-ID-GUID    | 281416cb-1968-11D0-a28f -00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
