# <a name="site-list-attribute"></a>Site-List-Attribut

Liste der mit diesem Link Objekt verbundenen Websites.



| Eingabe | Wert |
|-------------------|----------------------------------------------|
| CN                | Site-List                                    |
| LDAP-Display-Name | sitelist                                     |
| Size              | \-                                           |
| Berechtigung aktualisieren  | Domänen Administrator                         |
| Aktualisierungshäufigkeit  | Jedes Mal, wenn eine Site hinzugefügt oder daraus entfernt wird. |
| Attribute-Id      | 1.2.840.113556.1.4.821                       |
| System-ID-GUID    | d50c2cdc-8951-11d1-aebc-0000f80367c1         |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)      |
