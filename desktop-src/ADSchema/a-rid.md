# <a name="rid-attribute"></a>Rid-Attribut

Der relative Bezeichner eines Objekts.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Gesagt                                         |
| LDAP-Display-Name | rid                                         |
| Size              | 4 Bytes                                     |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.            |
| Aktualisierungshäufigkeit  | Wenn ein Objekt erstellt wird, das eine Rid benötigt. |
| Attribute-Id      | 1.2.840.113556.1.4.153                      |
| System-ID-GUID    | bf967a22-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**Enumeration**](s-enumeration.md)        |
