# <a name="trust-posix-offset-attribute"></a>Trust-POSIX-Offset-Attribut

Der Offset der portablen Betriebs System Schnittstelle (POSIX) für die vertrauenswürdige Domäne.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Trust-POSIX-Offset                   |
| LDAP-Display-Name | trustposixoffset                     |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | Wenn eine neue Vertrauensstellung erstellt wird.         |
| Attribute-Id      | 1.2.840.113556.1.4.134               |
| System-ID-GUID    | bf967a5e-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
