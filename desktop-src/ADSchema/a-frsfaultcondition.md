# <a name="frs-fault-condition-attribute"></a>FRS-Fault-Condition-Attribut

| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | FRS-Fehlerbedingung                         |
| LDAP-Display-Name | frsfehlercondition                           |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.491                      |
| System-ID-GUID    | 1be8f178-a9ff-11D0-afe2-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
