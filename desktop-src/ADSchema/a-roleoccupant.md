|-------------------|-----------------------------------------|
| CN                | Role-Occupant                           |
| Ldap-Anzeigename | roleOccupant                            |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | \-                                      |
| Updatehäufigkeit  | \-                                      |
| Attribute-Id      | 2.5.4.33                                |
| System-Id-Guid    | a8df7465-c5ea-11d1-bbb-0080c76670c0    |
