# <a name="role-occupant-attribute"></a>Role-Occupant-Attribut

Der Distinguished Name eines Objekts, das eine Organisations Rolle erfüllt.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Role-Occupant                           |
| LDAP-Display-Name | roleOccupant                            |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 2.5.4.33                                |
| System-ID-GUID    | a8df7465-c5ea-11d1-bbcb-0080c76670c0    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
