# <a name="ms-pki-oid-user-notice-attribute"></a>MS-PKI-OID-User-Notice-Attribut

Die Benutzer Benachrichtigung für die Richtlinien-OID der Unternehmens Aussteller.



| Eingabe | Wert |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-OID-Benutzer Hinweis                                                                     |
| LDAP-Display-Name | mspki-OID-Benutzer Hinweis                                                                      |
| Size              | \-                                                                                         |
| Berechtigung aktualisieren  | Unternehmensadministrator                                                                   |
| Aktualisierungshäufigkeit  | Wenn eine neue Vorlage erstellt wird oder die Attribute vorhandener Vorlagen bearbeitet werden. |
| Attribute-Id      | 1.2.840.113556.1.4.1673                                                                    |
| System-ID-GUID    | 04c4da7a-e114-4E69-88de-e293f 2d3b395                                                       |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                |
