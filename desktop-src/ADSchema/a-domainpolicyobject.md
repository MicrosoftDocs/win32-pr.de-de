# <a name="domain-policy-object-attribute"></a>Domain-Policy-Object-Attribut

Verweis auf das Richtlinienobjekt, das die Richtlinie der lokalen Sicherheitsautorität für die Hostdomäne definiert.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Domain-Policy-Object                    |
| Ldap-Anzeigename | domainPolicyObject                      |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | \-                                      |
| Updatehäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.32                   |
| System-ID-GUID    | bf96795d-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
