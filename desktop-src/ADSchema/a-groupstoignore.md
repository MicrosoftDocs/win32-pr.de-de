# <a name="groups-to-ignore-attribute"></a>Gruppen-zu-ignorieren-Attribut

Das **Gruppen-zu-ignorieren-** Attribut wird derzeit nicht verwendet.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Zu ignorierende Gruppen                            |
| LDAP-Display-Name | groupstoignore                              |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.344                      |
| System-ID-GUID    | eea65904-8ac6-11d0-afda-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
