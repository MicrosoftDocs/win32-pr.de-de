# <a name="ms-ds-consistency-guid-attribute"></a>MS-DS-Consistency-Guid-Attribut

Dieses Attribut wird verwendet, um die Konsistenz zwischen dem Verzeichnis und einem anderen Objekt, einer Datenbank oder anwendung zu überprüfen, indem GUIDs verglichen werden.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | MS-DS-Consistency-Guid                                |
| Ldap-Anzeigename | mS-DS-ConsistencyGuid                                 |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | \-                                                    |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1360                               |
| System-Id-Guid    | 23773dc2-b63a-11d2-90e1-00c04fd91ab1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
