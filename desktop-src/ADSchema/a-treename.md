# <a name="tree-name-attribute"></a>Tree-Name-Attribut

DNS-Name der Domäne im Stammverzeichnis einer Struktur.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Tree-Name                                   |
| LDAP-Display-Name | treename                                    |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.660                      |
| System-ID-GUID    | 28630ebd-41d5-11d1-a9c1-0000b80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
