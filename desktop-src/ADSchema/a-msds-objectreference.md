# <a name="ms-ds-object-reference-attribute"></a>ms-DS-Object-Reference-Attribut

Ein Link zum-Objekt, das die im-Objekt gespeicherten Daten verwendet, das dieses Attribut enthält.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Object-Reference                  |
| LDAP-Display-Name | MSDS-objectreferenzierung                    |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1840                 |
| System-ID-GUID    | 638ec2e8-22e7-409c-85d2-11 b21bee72de    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
