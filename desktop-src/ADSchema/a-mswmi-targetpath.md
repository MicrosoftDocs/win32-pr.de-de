# <a name="ms-wmi-targetpath-attribute"></a>MS-WMI-TargetPath-Attribut

Liste der Schlüssel-Wert-Paare zum eindeutigen Identifizieren eines WMI-Objekts.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-TargetPath                           |
| LDAP-Display-Name | mswap-TargetPath                            |
| Size              | Weniger als 100 Zeichen.                   |
| Berechtigung aktualisieren  | Gruppenrichtlinie-Administrator                  |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1648                     |
| System-ID-GUID    | 5006a79a-6bfe-4561-9f52-13cf4dd3e560        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
