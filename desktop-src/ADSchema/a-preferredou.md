# <a name="preferred-ou-attribute"></a>Bevorzugtes OU-Attribut

Die Organisationseinheit, die standardmäßig auf dem Desktop des Benutzers angezeigt werden soll.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Bevorzugte Organisationseinheit                            |
| LDAP-Display-Name | preferredou                             |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.97                   |
| System-ID-GUID    | bf9679ff-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
