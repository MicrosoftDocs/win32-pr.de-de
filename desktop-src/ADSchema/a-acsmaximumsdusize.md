# <a name="acs-maximum-sdu-size-attribute"></a>ACS-Maximum-SDU-Size-Attribut

Das **Attribut ACS-Maximum-SDU-Size** ist nur für die interne Verwendung vorgesehen. Basierend auf RFC2210.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | ACS-Maximum-SDU-Size                 |
| Ldap-Anzeigename | aCSMaximumSDUSize                    |
| Size              | 8 Bytes                              |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1314              |
| System-ID-GUID    | 87a2d8f9-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**Intervall**](s-interval.md)       |
