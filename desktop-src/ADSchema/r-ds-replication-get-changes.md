# <a name="ds-replication-get-changes-extended-right"></a>DS-Replication-Get-Changes Extended right

|--------------|--------------------------------------|

| Anzeigename | Replizieren von Verzeichnisänderungen        |
| Rights-GUID  | 1131f -9c07-11d1-f c04fc2dcd2 |
