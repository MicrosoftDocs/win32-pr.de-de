# <a name="bridgehead-server-list-bl-attribute"></a>Bridgehead-Server-List-BL-Attribut

Die Liste der Server, die als Bridgeheads für die Replikation dienen.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Bridgehead-Server-List-BL               |
| LDAP-Display-Name | bridgeheadServerListBL                  |
| Size              | \-                                      |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.        |
| Aktualisierungshäufigkeit  | Immer dann, wenn eine Site eingerichtet ist.               |
| Attribute-Id      | 1.2.840.113556.1.4.820                  |
| System-ID-GUID    | d50c2cdb-8951-11d1-aebc-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
