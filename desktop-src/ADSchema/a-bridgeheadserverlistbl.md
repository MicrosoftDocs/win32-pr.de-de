# <a name="bridgehead-server-list-bl-attribute"></a>Bridgehead-Server-List-BL-Attribut

Die Liste der Server, die Bridgeheads für die Replikation sind.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | Bridgehead-Server-List-BL               |
| Ldap-Anzeigename | bridgeheadServerListBL                  |
| Size              | \-                                      |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.        |
| Updatehäufigkeit  | Jedes Mal, wenn ein Standort eingerichtet wird.               |
| Attribute-Id      | 1.2.840.113556.1.4.820                  |
| System-Id-Guid    | d50c2cdb-8951-11d1-aebc-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
