# <a name="last-set-time-attribute"></a>Last-Set-Time-Attribut

Der Zeitpunkt, zu dem das Geheimnis zuletzt geändert wurde. Dieser Wert wird als große ganze Zahl gespeichert, die die Anzahl von 100-Nanosekunden-Intervallen seit dem 1. Januar 1601 (UTC) darstellt.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Zeitpunkt der letzten Festlegung                        |
| Ldap-Anzeigename | lastSetTime                          |
| Size              | 8 Bytes                              |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.53                |
| System-ID-GUID    | bf967998-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervall**](s-interval.md)       |
