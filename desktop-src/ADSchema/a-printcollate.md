# <a name="print-collate-attribute"></a>Print-Collate-Attribut

**True** , wenn ein Drucker Sortier Container hat.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Print-Collate                        |
| LDAP-Display-Name | printCollate                         |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.242               |
| System-ID-GUID    | 281416d2-1968-11D0-a28f -00aa003049e2 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
