# <a name="msmq-multicast-address-attribute"></a>MSMQ-Multicast-Address-Attribut

Dies ist Teil eines MSMQ-Objekts und wird durch Aufrufen der API auf MQCreateQueue oder MQSetProperties festgelegt. Sie steuert, ob MSMQ Nachrichten von einer Multicastadresse akzeptiert.
