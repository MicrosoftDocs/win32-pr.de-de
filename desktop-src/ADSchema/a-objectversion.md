# <a name="object-version-attribute"></a>Object-Version-Attribut

Dies kann verwendet werden, um eine Versionsnummer für das -Objekt zu speichern.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------|
| CN                | Object-Version                                                 |
| Ldap-Anzeigename | Objectversion                                                  |
| Size              | 4 Bytes                                                        |
| Aktualisieren von Berechtigungen  | Der Designer des -Objekts würde diesen Wert festlegen.               |
| Updatehäufigkeit  | Dieser Wert sollte jedes Mal erhöht werden, wenn sich das Objekt ändert. |
| Attribute-Id      | 1.2.840.113556.1.2.76                                          |
| System-ID-GUID    | 16775848-47f3-11d1-a9c3-0000f80367c1                           |
| Syntax            | [**Enumeration**](s-enumeration.md)                           |
