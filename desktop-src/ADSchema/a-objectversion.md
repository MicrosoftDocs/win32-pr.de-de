# <a name="object-version-attribute"></a>Object-Version-Attribut

Diese kann verwendet werden, um eine Versionsnummer für das-Objekt zu speichern.



| Eingabe | Wert |
|-------------------|----------------------------------------------------------------|
| CN                | Object-Version                                                 |
| LDAP-Display-Name | ob objectVersion für                                                  |
| Size              | 4 Bytes                                                        |
| Berechtigung aktualisieren  | Dieser Wert wird vom Designer des-Objekts festgelegt.               |
| Aktualisierungshäufigkeit  | Dieser Wert sollte jedes Mal erhöht werden, wenn sich das Objekt ändert. |
| Attribute-Id      | 1.2.840.113556.1.2.76                                          |
| System-ID-GUID    | 16775848-47F 3-11d1-a9c3-0000 C1                           |
| Syntax            | [**Enumeration**](s-enumeration.md)                           |
