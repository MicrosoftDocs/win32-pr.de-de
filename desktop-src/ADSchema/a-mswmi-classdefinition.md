# <a name="ms-wmi-classdefinition-attribute"></a>ms-WMI-ClassDefinition-Attribut

Enthält eine Klassendefinition, die in einem WMI-Namespace deklariert werden soll.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-ClassDefinition                      |
| Ldap-Anzeigename | msWMI-ClassDefinition                       |
| Size              | \-                                          |
| Aktualisieren von Berechtigungen  | Dieser Wert wird vom System festgelegt.            |
| Updatehäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1625                     |
| System-Id-Guid    | 2b9c0ebc-c272-45cb-99d2-4d0e691632e0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
