# <a name="ms-wmi-classdefinition-attribute"></a>MS-WMI-classdefinition-Attribut

Enthält eine Klassendefinition, die in einem WMI-Namespace festgelegt werden muss.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-classdefinition                      |
| LDAP-Display-Name | mswap-classdefinition                       |
| Size              | \-                                          |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.            |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1625                     |
| System-ID-GUID    | 2b9c0ebc-C272-45cb-99d2-4d0e691632e0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
