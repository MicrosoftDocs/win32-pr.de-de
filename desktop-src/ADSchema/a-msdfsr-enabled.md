# <a name="ms-dfsr-enabled-attribute"></a>MS-DFSR-aktiviertes Attribut

Gibt an, ob das Objekt aktiviert ist.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-aktiviert                      |
| LDAP-Display-Name | msdfsr-aktiviert                       |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.9            |
| System-ID-GUID    | 03726ae7-8e7d-4446-8aae-a91657c00993 |
| Syntax            | [**Booleschen**](s-boolean.md)         |
