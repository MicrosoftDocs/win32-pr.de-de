# <a name="invocation-id-attribute"></a>Invocation-Id-Attribut

Dient zum eindeutigen Identifizieren der einzelnen Microsoft Exchange Server-Verzeichnisse in der Organisation.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Invocation-Id                                         |
| LDAP-Display-Name | invocationId                                          |
| Size              | \-                                                    |
| Berechtigung aktualisieren  | \-                                                    |
| Aktualisierungshäufigkeit  | Wenn der Exchange-Server installiert ist.                |
| Attribute-Id      | 1.2.840.113556.1.2.115                                |
| System-ID-GUID    | bf96798e-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
