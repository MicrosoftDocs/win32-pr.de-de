# <a name="ms-com-partitionlink-attribute"></a>MS-com-partitionlink-Attribut

Ein Link, der verwendet wird, um ein com+-partitionset mit den zugehörigen COM+-Partitionen zuzuordnen.



| Eingabe | Wert |
|-------------------|-----------------------------------------|
| CN                | MS-com-partitionlink                    |
| LDAP-Display-Name | MScom-partitionlink                     |
| Size              | \-                                      |
| Berechtigung aktualisieren  | \-                                      |
| Aktualisierungshäufigkeit  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1423                 |
| System-ID-GUID    | 09abac62-043f-4702-ac2b-6ca15eee5754    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
