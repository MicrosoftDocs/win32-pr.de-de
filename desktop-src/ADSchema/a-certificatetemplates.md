# <a name="certificate-templates-attribute"></a>Certificate-Templates-Attribut

Enthält Informationen für ein Zertifikat, das von einem Zertifikat Server ausgestellt wurde.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Certificate-Templates                       |
| LDAP-Display-Name | certificatetemplates                        |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.823                      |
| System-ID-GUID    | 2a39c5b1-8960-11d1-AEbc-0000b80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
