# <a name="ms-sql-allowqueuedupdatingsubscription-attribute"></a>MS-SQL-allowqueuedupdatingabonnement-Attribut

True, um Transaktionen in der Warteschlange zuzulassen, wenn Abonnements aktualisiert werden.



| Eingabe | Wert |
|-------------------|----------------------------------------|
| CN                | MS-SQL-allowqueuedupdatingabonnement |
| LDAP-Display-Name | MS-SQL-allowqueuedupdatingabonnement |
| Size              | 4 Bytes                                |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.       |
| Aktualisierungshäufigkeit  | Wenn die Replikation eingerichtet ist.             |
| Attribute-Id      | 1.2.840.113556.1.4.1405                |
| System-ID-GUID    | c458ca80-d34b-11d2-999a-0000f87a57d4   |
| Syntax            | [**Booleschen**](s-boolean.md)           |
