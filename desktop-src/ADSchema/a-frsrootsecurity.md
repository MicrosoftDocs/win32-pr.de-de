# <a name="frs-root-security-attribute"></a>FRS-root-Security-Attribut

Die Sicherheits Beschreibung des Replikat Gruppen Stamms für die Datei Replikation.



| Eingabe | Wert |
|-------------------|-----------------------------------------------------|
| CN                | FRS-root-Sicherheit                                   |
| LDAP-Display-Name | frsrootsecurity                                     |
| Size              | \-                                                  |
| Berechtigung aktualisieren  | \-                                                  |
| Aktualisierungshäufigkeit  | \-                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.535                              |
| System-ID-GUID    | 5245801f-ca6a-11D0-AFFF -0000f 80367c1                |
| Syntax            | [**String(NT-Sec-Desc)**](s-string-nt-sec-desc.md) |
