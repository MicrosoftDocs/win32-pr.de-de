# <a name="ms-ds-behavior-version-attribute"></a>ms-DS-Behavior-Version-Attribut

Dieses Attribut wird verwendet, um die Version der Domänen-oder Gesamtstruktur Verhalten zu verfolgen. Es handelt sich um eine monoton steigende Zahl, die verwendet wird, um bestimmte Active Directory Features zu aktivieren.



| Eingabe | Wert |
|-------------------|------------------------------------------------|
| CN                | ms-DS-Behavior-Version                         |
| LDAP-Display-Name | msDS-Behavior-Version                          |
| Size              | \-                                             |
| Berechtigung aktualisieren  | Dieser Wert wird vom Domänen Administrator festgelegt. |
| Aktualisierungshäufigkeit  | \-                                             |
| Attribute-Id      | 1.2.840.113556.1.4.1459                        |
| System-ID-GUID    | d31a8757-2447-4545-8081-3bb610cacbf2           |
| Syntax            | [**Enumeration**](s-enumeration.md)           |
