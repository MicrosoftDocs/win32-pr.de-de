# <a name="mssfu-30-domains-attribute"></a>msSFU-30-Domains-Attribut

Enthält die Liste der UNIX-NIS-Domänen, die alle zur gleichen AD NIS-Domäne migriert wurden.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | msSFU-30-Domänen                     |
| LDAP-Display-Name | msSFU30Domains                       |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.340          |
| System-ID-GUID    | 93095ed3-6f 30-4bdd-B734-65d569 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
