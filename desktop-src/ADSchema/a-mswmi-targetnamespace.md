# <a name="ms-wmi-targetnamespace-attribute"></a>MS-WMI-targetNamespace-Attribut

Der Namespace, in dem das Objekt erstellt werden soll.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-targetNamespace                      |
| LDAP-Display-Name | mswap-targetNamespace                       |
| Size              | Weniger als 20 Zeichen.                |
| Berechtigung aktualisieren  | Gruppenrichtlinie-Administrator                  |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1646                     |
| System-ID-GUID    | 1c4ab61b-3420-44e5-849d-8b5dbf 60feb7        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
