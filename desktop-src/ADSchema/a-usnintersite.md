# <a name="usn-intersite-attribute"></a>USN-Intersite-Attribut

Die Updatesequenznummer (USN) für die standortübergreifende Replikation.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | USN-Intersite                        |
| Ldap-Anzeigename | USNIntersite                         |
| Size              | 4 Bytes                              |
| Aktualisieren von Berechtigungen  | \-                                   |
| Updatehäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.469               |
| System-Id-Guid    | a8df7498-c5ea-11d1-bbb-0080c76670c0 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
