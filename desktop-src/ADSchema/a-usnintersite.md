# <a name="usn-intersite-attribute"></a>USN-Intersite-Attribut

Die Aktualisierungs Sequenznummer für die standortübergreifende Replikation.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | USN-Intersite                        |
| LDAP-Display-Name | Standortübergreifende Website                         |
| Size              | 4 Bytes                              |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.469               |
| System-ID-GUID    | a8df7498-c5ea-11d1-bbcb-0080c76670c0 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
