# <a name="ms-sql-status-attribute"></a>MS-SQL-Status-Attribut

Der Status für den SQL-Server. Der Status ist " **true** ", wenn der Server ausgeführt wird.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | MS-SQL-Status                        |
| LDAP-Display-Name | MS-SQL-Status                        |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | Beim Systemstart.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1380              |
| System-ID-GUID    | 9a7d4770-CCEE-11d2-9993-0000f 87a57d4 |
| Syntax            | [**Intervall**](s-interval.md)       |
