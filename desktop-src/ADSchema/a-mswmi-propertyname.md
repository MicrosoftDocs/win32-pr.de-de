# <a name="ms-wmi-propertyname-attribute"></a>MS-WMI-PropertyName-Attribut

Der Ziel Richtlinien Objektname für ein Parameter Objekt.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-PropertyName                         |
| LDAP-Display-Name | mgomi-PropertyName                          |
| Size              | Weniger als 20 Zeichen.                |
| Berechtigung aktualisieren  | Gruppenrichtlinie-Administrator                  |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1641                     |
| System-ID-GUID    | ab920883-e7f8-4d72-b4a0-c0449897509d        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
