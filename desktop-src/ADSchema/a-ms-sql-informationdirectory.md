# <a name="ms-sql-informationdirectory-attribute"></a>MS-SQL-informationdirectory-Attribut

Das Informations Verzeichnis für die aktuelle Instanz von SQL Server.



| Eingabe | Wert |
|-------------------|------------------------------------------|
| CN                | MS-SQL-informationdirectory              |
| LDAP-Display-Name | MS-SQL-informationdirectory              |
| Size              | \-                                       |
| Berechtigung aktualisieren  | Domänen Administrator                     |
| Aktualisierungshäufigkeit  | Beim Systemstart oder beim Aktualisieren von AD. |
| Attribute-Id      | 1.2.840.113556.1.4.1392                  |
| System-ID-GUID    | d0aedb2e-ccee-11d2-9993-0000f87a57d4     |
| Syntax            | [**Booleschen**](s-boolean.md)             |
