# <a name="print-notify-attribute"></a>Print-Notify-Attribut

Eine vom Benutzer angegebene Zeichenfolge, die den Benachrichtigungs Kontakt angibt.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | Print-Notify                                |
| LDAP-Display-Name | printNotify                                 |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.272                      |
| System-ID-GUID    | ba305f6a-47e3-11d0-a1a6-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
