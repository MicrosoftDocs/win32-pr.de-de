# <a name="help-data16-attribute"></a>Help-Data16-Attribut

Dieses Attribut wurde für das Win16-Hilfedateiformat für Exchange 4.0 verwendet. Es wird nicht für andere Versionen von Exchange.



| Eingabe | Wert |
|-------------------|-------------------------------------------------------|
| CN                | Help-Data16                                           |
| Ldap-Anzeigename | helpData16                                            |
| Size              | \-                                                    |
| Aktualisieren von Berechtigungen  | Dies wird vom System verwendet.                           |
| Updatehäufigkeit  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.402                                |
| System-Id-Guid    | 5fd424a7-1262-11d0-a060-00aa006c33ed                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
