# <a name="com-classid-attribute"></a>COM-ClassID-Attribut

Mit diesem Attribut wird die Liste der ClassIDs gespeichert, die in diesem Anwendungspaket implementiert werden.



| Eingabe | Wert |
|-------------------|---------------------------------------------|
| CN                | COM-ClassID                                 |
| LDAP-Display-Name | comclassid                                  |
| Size              | \-                                          |
| Berechtigung aktualisieren  | \-                                          |
| Aktualisierungshäufigkeit  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.19                       |
| System-ID-GUID    | bf96793b-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
