# <a name="rid-allocation-pool-attribute"></a>Rid-Allocation-Pool-Attribut

Ein Pool, der für die Verwendung durch den RID-Manager vorab abgerufen wurde, als der RID-Previous-Allocation-Pool aufgebraucht wurde.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | RID-Zuordnung-Pool                  |
| LDAP-Display-Name | ridzugecationpool                    |
| Size              | 8 Bytes                              |
| Berechtigung aktualisieren  | Dieser Wert wird vom System festgelegt.     |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.371               |
| System-ID-GUID    | 66171889-8F -11D0-AFDA-00c04f-930c9 |
| Syntax            | [**Intervall**](s-interval.md)       |
