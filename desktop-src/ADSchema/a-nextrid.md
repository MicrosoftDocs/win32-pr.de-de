# <a name="next-rid-attribute"></a>Next-Rid-Attribut

Das nächste Rid-Feld, das von der Zuweisung im gemischten Modus verwendet wird.



| Eingabe | Wert |
|-------------------|--------------------------------------|
| CN                | Nächste Rid                             |
| LDAP-Display-Name | nextrd                              |
| Size              | \-                                   |
| Berechtigung aktualisieren  | \-                                   |
| Aktualisierungshäufigkeit  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.88                |
| System-ID-GUID    | bf9679db-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
