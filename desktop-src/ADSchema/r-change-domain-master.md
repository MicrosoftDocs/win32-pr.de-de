# <a name="change-domain-master-extended-right"></a>Erweitertes Recht "Change-Domain-Master"

|--------------|--------------------------------------|
| CN           | Change-Domain-Master                 |
| Anzeigename | Ändern des Domänenmasters                 |
| Rights-GUID  | 014bf69c-7b3b-11d1-85f6-08002be74fab |
