# <a name="user-profiles"></a>Benutzerprofile

Konfigurationsinformationen, die für jeden Benutzer beibehalten werden können, werden in [Benutzerprofilen](about-user-profiles.md)gespeichert. Zu diesen Informationen gehören die Desktop Anordnung, persönliche Programm Gruppen, Programmelemente in den persönlichen Programm Gruppen, Bildschirmfarben, Bildschirmschoner, Netzwerkverbindungen, Drucker Verbindungen, Mauseinstellungen, Fenstergröße und-Position usw. Wenn sich ein Benutzer anmeldet, lädt das System das Profil des Benutzers und konfiguriert die Umgebung gemäß den Informationen im Profil.

In diesem Abschnitt werden folgende Themen behandelt.

-   [Informationen zu Benutzerprofilen](about-user-profiles.md)
-   [Referenz zu Benutzerprofilen](user-profiles-reference.md)
