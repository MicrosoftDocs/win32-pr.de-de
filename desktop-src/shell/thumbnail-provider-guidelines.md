# <a name="thumbnail-handler-guidelines"></a><span data-ttu-id="a4079-103">Richtlinien für Miniaturansichten</span><span class="sxs-lookup"><span data-stu-id="a4079-103">Thumbnail Handler Guidelines</span></span>

<span data-ttu-id="a4079-104">Wenn Sie eine Miniaturansicht bereitstellen, sollten die folgenden Richtlinien befolgt werden.</span><span class="sxs-lookup"><span data-stu-id="a4079-104">When you provide a thumbnail, the following guidelines should be followed.</span></span>

-   <span data-ttu-id="a4079-105">Stellen Sie Miniaturansichten bereit, die bei einer Auflösung von 256 x 256 Pixel in 32-Bit-Farbe gut angezeigt werden.</span><span class="sxs-lookup"><span data-stu-id="a4079-105">Provide thumbnails that display well at a resolution of 256x256 pixels in 32-bit color.</span></span> <span data-ttu-id="a4079-106">Eine Miniaturansicht dieser Größe wird vom Lesebereich von Windows Vista verwendet, wenn kein registrierter Vorschau Handler vorhanden ist.</span><span class="sxs-lookup"><span data-stu-id="a4079-106">A thumbnail of this size is be used by the Windows Vista reading pane in the absence of a registered preview handler.</span></span> <span data-ttu-id="a4079-107">Ein Vorschau Handler ist jedoch die bevorzugte Option und sollte nach Möglichkeit bereitgestellt werden.</span><span class="sxs-lookup"><span data-stu-id="a4079-107">However, a preview handler is the preferred option and should be provided whenever possible.</span></span>
-   <span data-ttu-id="a4079-108">Wenn Sie mehrere Abbilder mit unterschiedlichen Größen erstellen, erstellen Sie nicht die kleineren Bilder aus der größeren, indem Sie die Seite, den Frame oder das Bild zuschneiden.</span><span class="sxs-lookup"><span data-stu-id="a4079-108">When you create multiple images of different sizes, do not create the smaller images from the larger by cropping the page, frame, or image.</span></span> <span data-ttu-id="a4079-109">Skalieren Sie das gesamte Bild herunter.</span><span class="sxs-lookup"><span data-stu-id="a4079-109">Scale down the entire image.</span></span>
-   <span data-ttu-id="a4079-110">Nicht mehrere Seiten, Frames oder Bilder gleichzeitig anzeigen; Verwenden Sie einfach eine.</span><span class="sxs-lookup"><span data-stu-id="a4079-110">Do not show multiple pages, frames, or images at once; just use one.</span></span> <span data-ttu-id="a4079-111">Wenn ein Dokument aus mehreren Seiten besteht, z. b. einem Textdokument oder einem Arbeitsblatt, das aus mehreren Arbeitsblättern besteht, ist die Seite Abdeckung oft die beste Wahl, aber unabhängig davon, welche Option Sie verwenden, verwenden Sie nur eine.</span><span class="sxs-lookup"><span data-stu-id="a4079-111">If a document consists of several pages, such as a text document or a spreadsheet that consists of several worksheets, the cover page is often the best choice, but regardless of which you use, only use one.</span></span> <span data-ttu-id="a4079-112">Aggregieren Sie keine unterschiedlichen Seiten, was ein übersichtliches aussehen ermöglicht.</span><span class="sxs-lookup"><span data-stu-id="a4079-112">Do not aggregate different pages, which gives a cluttered look.</span></span>
-   <span data-ttu-id="a4079-113">Windows Vista ist für das horizontale herunter-oder Herunterskalieren von Sampling-Images verantwortlich.</span><span class="sxs-lookup"><span data-stu-id="a4079-113">Windows Vista is responsible for scaling down or down sampling images.</span></span> <span data-ttu-id="a4079-114">Wenn Ihr Handler nach einem größeren Abbild gefragt wird, als Sie verfügbar sind, geben Sie die nächstgelegene Größe an.</span><span class="sxs-lookup"><span data-stu-id="a4079-114">If your handler is asked for a larger image than you have available, provide the closest size you have.</span></span> <span data-ttu-id="a4079-115">Versuchen Sie nicht, die Größe Ihres eigenen Bilds dynamisch zu ändern.</span><span class="sxs-lookup"><span data-stu-id="a4079-115">Do not attempt to dynamically resize your own image.</span></span>
-   <span data-ttu-id="a4079-116">Gibt immer ein Miniaturbild von Ihrem Handler zurück, anstatt eine spezielle Logik zum Zurückgeben von herkömmlichen Symbolen auszuführen.</span><span class="sxs-lookup"><span data-stu-id="a4079-116">Always return a thumbnail image from your handler instead of performing special logic to return traditional icons.</span></span> <span data-ttu-id="a4079-117">Unter einer bestimmten Größe zeigt Windows Vista automatisch ein herkömmliches Symbol anstelle der Miniaturansicht an.</span><span class="sxs-lookup"><span data-stu-id="a4079-117">Below a certain size, Windows Vista automatically displays a traditional icon in place of the thumbnail.</span></span> <span data-ttu-id="a4079-118">Weitere Informationen finden Sie im Abschnitt "Miniaturansicht: *Cache und Größen* Anpassung" von Miniatur Ansichts [Handlern](thumbnail-providers.md) .</span><span class="sxs-lookup"><span data-stu-id="a4079-118">See the *Thumbnail Cache and Sizing* section of [Thumbnail Handlers](thumbnail-providers.md) for more details.</span></span>
-   <span data-ttu-id="a4079-119">Gibt immer eine Miniaturansicht mit dem Seitenverhältnis der Seite, des Frames oder Bilds zurück.</span><span class="sxs-lookup"><span data-stu-id="a4079-119">Always return a thumbnail with the aspect ratio of the page, frame, or image.</span></span> <span data-ttu-id="a4079-120">Verwenden Sie Alpha nicht zum Vervollständigen eines Quadrats.</span><span class="sxs-lookup"><span data-stu-id="a4079-120">Do not use alpha to complete a square.</span></span> <span data-ttu-id="a4079-121">Windows Vista ist für die ordnungsgemäße Positionierung eines nicht quadratischen Bilds verantwortlich.</span><span class="sxs-lookup"><span data-stu-id="a4079-121">Windows Vista is responsible for correctly positioning a non-square image.</span></span>
-   <span data-ttu-id="a4079-122">Fügen Sie Ihren Miniaturansichten keine Zusatzelemente hinzu.</span><span class="sxs-lookup"><span data-stu-id="a4079-122">Do not add adornments to your thumbnails.</span></span> <span data-ttu-id="a4079-123">Windows Vista wendet ggf. automatisch Drop Shadows und andere Zusatzelemente an.</span><span class="sxs-lookup"><span data-stu-id="a4079-123">Windows Vista automatically applies drop shadows and other adornments where appropriate.</span></span> <span data-ttu-id="a4079-124">Es wendet auch spezielle Zusatzelemente für bestimmte Dateitypen an, z. b. Bilder oder Videos.</span><span class="sxs-lookup"><span data-stu-id="a4079-124">It also applies special adornments for specific file types such as pictures or videos.</span></span>
-   <span data-ttu-id="a4079-125">Überlagern Sie keine Dateityp-oder Anwendungsinformationen in der Miniaturansicht.</span><span class="sxs-lookup"><span data-stu-id="a4079-125">Do not overlay file type or application information on your thumbnail.</span></span> <span data-ttu-id="a4079-126">Windows Vista zeigt in der unteren rechten Ecke des Bilds eine typüberlagerung an.</span><span class="sxs-lookup"><span data-stu-id="a4079-126">Windows Vista displays a type overlay for you in the lower right corner of the image.</span></span> <span data-ttu-id="a4079-127">Diese Überlagerung basiert auf einem wahrgenommenen Typ, kann jedoch für einzelne Dateitypen festgelegt werden.</span><span class="sxs-lookup"><span data-stu-id="a4079-127">This overlay is based on perceived type but can be set for individual file types.</span></span>
-   <span data-ttu-id="a4079-128">Um eine bessere Leistung zu erzielen, wenn die Miniaturansicht auf Dateiinhalt basiert – eine Seite eines Dokuments, z. b. –, speichern Sie das Vorschaubild, wenn die Datei gespeichert wird (und daher wahrscheinlich geändert wird), anstatt Sie in Echtzeit zu berechnen.</span><span class="sxs-lookup"><span data-stu-id="a4079-128">For better performance, when your thumbnail is based on file content—a page of a document, for example—store the preview image when the file is saved (and therefore probably changed) instead of calculating it in real time.</span></span> <span data-ttu-id="a4079-129">Dies sollte erfolgen, wenn die Berechnung Speicher intensiv ist (mehr als ein oder zwei Sekunden).</span><span class="sxs-lookup"><span data-stu-id="a4079-129">This should be done if the calculation is memory-intensive (more than one or two seconds).</span></span> <span data-ttu-id="a4079-130">Wenn dies nicht der Fall ist, werden Sichten, für die eine große Anzahl von Dateien angezeigt wird, deren Miniaturansichten von unterschiedlichen Handlern verarbeitet werden, etwas Zeit in Anspruch nehmen – eine schlechte Benutzer Leistung.</span><span class="sxs-lookup"><span data-stu-id="a4079-130">If this is not done, views showing a large number of files whose thumbnails are handled by different handlers is going to take some time to display—a bad user experience.</span></span> <span data-ttu-id="a4079-131">In Windows Vista werden Miniaturansichten zwischengespeichert, und es wird auf den Zeitpunkt der letzten Änderung verwiesen, um zu bestimmen, ob eine Miniaturansicht</span><span class="sxs-lookup"><span data-stu-id="a4079-131">Windows Vista caches thumbnails and refers to the last modified time to determine whether a thumbnail should be updated.</span></span>
-   <span data-ttu-id="a4079-132">Beachten Sie, dass der Explorer eine Miniaturansicht möglicherweise nicht anzeigt, auch wenn ein Anbieter verfügbar ist.</span><span class="sxs-lookup"><span data-stu-id="a4079-132">Be aware that Explorer may choose not to display a thumbnail even if a provider is available.</span></span> <span data-ttu-id="a4079-133">Beispielsweise wird eine Datei, die auf einem Band archiviert wurde, nicht zurückgerufen, um die Miniaturansicht zu erhalten.</span><span class="sxs-lookup"><span data-stu-id="a4079-133">For example, a file that has been archived to tape will not be recalled to obtain its thumbnail.</span></span>
