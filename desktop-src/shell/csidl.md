# <a name="csidl"></a>CSIDL

> [!Note]  
> Ab Windows Vista wurden diese Werte durch [**KNOWNFOLDERID**](knownfolderid.md) -Werte ersetzt. In diesem Thema finden Sie eine Liste der neuen Konstanten und ihrer entsprechenden CSIDL-Werte. Der benutzerfreundlicher Name werden auch die entsprechenden **KNOWNFOLDERID** -Werte für jeden CSIDL-Wert angegeben.
>
> Das CSIDL-System wird unter Windows Vista aus Kompatibilitätsgründen unterstützt. Die neue Entwicklung sollte jedoch anstelle von CSIDL-Werten [**KNOWNFOLDERID**](knownfolderid.md) -Werte verwenden.

 

CSIDL-Werte (Konstante sonderelement-ID-Liste) stellen eine eindeutige systemunabhängige Methode dar, um spezielle Ordner zu identifizieren, die häufig von Anwendungen verwendet werden, aber nicht denselben Namen oder Speicherort auf einem beliebigen System haben. Der Systemordner kann z. b. "c: \\ Windows" auf einem System und "c: \\ Winnt" auf einem anderen System sein. Diese Konstanten werden in "shlobj. h" definiert.

## <a name="remarks"></a>Bemerkungen

Diese Werte ersetzen die Verwendung von Umgebungsvariablen zu diesem Zweck. Sie werden wiederum in Windows Vista und höher durch die [**KNOWNFOLDERID**](knownfolderid.md) -Werte ersetzt.

Eine CSIDL wird in Verbindung mit einer der vier Shellfunktionen " [**shgetfolderlocation**](/windows/desktop/api/shlobj_core/nf-shlobj_core-shgetfolderlocation)", " [**SHGetFolderPath**](/windows/desktop/api/shlobj_core/nf-shlobj_core-shgetfolderpatha)", " [**SHGetSpecialFolderLocation**](/windows/desktop/api/shlobj_core/nf-shlobj_core-shgetspecialfolderlocation)" und " [**SHGetSpecialFolderPath**](/windows/desktop/api/shlobj_core/nf-shlobj_core-shgetspecialfolderpatha)" verwendet, um den Pfad eines speziellen Ordners oder einen Zeiger auf eine Element Bezeichner-Liste (PIDL) abzurufen.

Kombinieren Sie * * * * CSIDL- \_ Flag \_ Create * * * * mit allen anderen CSIDLs, außer * * * * CSIDL \_ Flag \_ Not \_ Verify * * * *, um die Erstellung des zugeordneten Ordners zu erzwingen. Die verbleibenden CSIDLs entsprechen entweder Dateisystem Ordnern oder virtuellen Ordnern. Wenn die CSIDL einen Dateisystem Ordner identifiziert, wird ein häufig verwendeter Pfad als Beispiel angegeben. Es können auch andere Pfade verwendet werden. Einige CSIDLs können einer entsprechenden Umgebungsvariablen " *% VariableName%* " zugeordnet werden. CSIDLs sind jedoch zuverlässiger und sollten nach Möglichkeit verwendet werden.
