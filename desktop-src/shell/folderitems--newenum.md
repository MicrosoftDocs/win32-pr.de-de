# <a name="folderitems_newenum-method"></a>FolderItems. \_ NewEnum-Methode

Erstellt ein neues [**FolderItems-Objekt,**](folderitems.md) das eine Kopie dieses FolderItemsss-Objekts ist, und gibt es zurück.

## <a name="syntax"></a>Syntax


```JScript
FolderItems._NewEnum()
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.
