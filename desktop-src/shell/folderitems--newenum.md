# <a name="folderitems_newenum-method"></a>Folderitems. \_ Methode ' netwenum '

Erstellt ein neues [**folderitems**](folderitems.md) -Objekt, das eine Kopie dieses folderitemsss-Objekts ist, und gibt es zurück.

## <a name="syntax"></a>Syntax


```JScript
FolderItems._NewEnum()
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.
