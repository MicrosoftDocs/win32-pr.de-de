# <a name="using-hardware-autoplay"></a>Verwenden der Hardware AutoPlay

AutoPlay ist ein neues Feature in Windows XP, mit dem Inhalte wie Bilder, Musik oder Videodateien auf Wechselmedien und Wechselmedien erkannt werden. Bei der automatischen Wiedergabe werden Anwendungen automatisch gestartet, um diesen Inhalt zu wiedergeben oder anzuzeigen. Dadurch wird die Verwendung von spezialisierten Peripheriegeräten, wie z. b. MPEG-Layer 3 (MP3)-Playern und digitalen Foto Lesern, vereinfacht. Außerdem erleichtert es Benutzern, die mit der für den Zugriff auf verschiedene Inhaltstypen benötigten Software nicht vertraut sind.

Eine erweiterte Auswahl von Symbolen, einschließlich Marken Symbolen für Haupt Hersteller, ist enthalten, um eine verbesserte visuelle Differenzierung von Laufwerken und anderen angeschlossenen Geräten in **Arbeitsplatz** zu bieten. Symbole für Geräte mit Wechselmedien, mit Ausnahme von Diskettenlaufwerken, werden sowohl in bereit-als auch in bereit verfügbaren Versionen bereitgestellt, um den aktuellen Zustand des Geräts anzugeben. Dadurch wird bestätigt, dass ein Gerät, das in das System integriert ist, zur Verwendung bereit ist, und es trägt dazu bei, Geräte, die nicht bereit sind, von Geräten ohne eingefügte Medien zu vermeiden.

> [!Note]  
> Die automatische Wiedergabe sollte nicht mit Autorun verwechselt werden. Autorun ermöglicht es einer Compact Disk, eine Funktion, z. b. ein Installationsprogramm oder Spiel Wiedergabe, automatisch zu starten, wenn Sie auf dem CD-ROM-Laufwerk platziert wird. Dies wird durch die Verwendung einer Autorun. inf-Datei im Stammverzeichnis der Compact Disk erreicht. Weitere Informationen finden Sie unter [Erstellen einer Autorun-aktivierten CD-ROM-Anwendung](autoplay.md).
