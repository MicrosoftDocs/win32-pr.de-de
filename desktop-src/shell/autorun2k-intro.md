# <a name="using-hardware-autoplay"></a>Verwenden der automatischen Hardwarewiedergabe

AutoPlay ist ein neues Feature, das in Windows XP eingeführt wurde und Inhalte wie Bilder, Musik oder Videodateien auf Wechselmedien und Wechselmedien erkennt. Die automatische Wiedergabe startet dann automatisch Anwendungen, um diesen Inhalt wiederzugeben oder anzuzeigen. Dies vereinfacht die Verwendung spezialisierter Peripheriegeräte wie MPEG-Layer 3-Player (MP3) und digitaler Fotoleser. Außerdem erleichtert sie Benutzern, die mit der Software nicht vertraut sind, die für den Zugriff auf verschiedene Inhaltstypen erforderlich ist.

Eine erweiterte Auswahl von Symbolen, einschließlich Markensymbolen für große Hersteller, ist enthalten, um eine verbesserte visuelle Unterscheidung von Laufwerken und anderen angeschlossenen Geräten in **Arbeitsplatz** zu ermöglichen. Symbole für Geräte mit Wechselmedien, mit Ausnahme von Diskettenlaufwerken, werden sowohl in bereiten als auch in nicht bereiten Versionen bereitgestellt, um den aktuellen Zustand des Geräts anzugeben. Dadurch wird bestätigt, dass ein Gerät mit hot-Plug-In in das System einsatzbereit ist, und es wird verhindert, dass Nachrichten vom Gerät nicht bereit sind, ohne dass Medien eingefügt werden.

> [!Note]  
> AutoPlay darf nicht mit AutoRun verwechselt werden. AutoRun ermöglicht einem Compact Disc das automatische Starten einer Funktion, z. B. eines Installationsprogramms oder eines Spielspiels, wenn sie auf dem CD-ROM-Laufwerk platziert wird. Dies wird durch die Verwendung einer Datei Autorun.inf im Stammverzeichnis der Compact Disc erreicht. Weitere Informationen finden Sie unter [Creating an AutoRun-enabled CD-ROM Application](autoplay.md).
