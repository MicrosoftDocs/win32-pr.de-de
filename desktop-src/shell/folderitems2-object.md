# <a name="folderitems2-object"></a>FolderItems2-Objekt

Erweitert das [**FolderItems-Objekt.**](folderitems.md) Es unterstützt eine zusätzliche Methode.

## <a name="members"></a>Member

Das **FolderItems2-Objekt** verfügt über diese Typen von Membern:

-   [Methoden](#methods)

### <a name="methods"></a>Methoden

Das **FolderItems2-Objekt** verfügt über diese Methoden.



| Methode                                            | Beschreibung                                                                                                                                                                                                                                         |
|:--------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**InvokeVerbEx**](folderitems2-invokeverbex.md) | Führt ein Verb für eine Auflistung von [**FolderItem-Objekten**](folderitem.md) aus. Diese Methode ist eine Erweiterung der [**InvokeVerb-Methode,**](folderitem-invokeverb.md) die eine zusätzliche Steuerung des Vorgangs über einen Satz von Flags ermöglicht.<br/> |
