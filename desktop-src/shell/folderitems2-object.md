# <a name="folderitems2-object"></a>FolderItems2-Objekt

Erweitert das [**folderitems**](folderitems.md) -Objekt. Es wird eine zusätzliche Methode unterstützt.

## <a name="members"></a>Member

Das **FolderItems2** -Objekt verfügt über diese Typen von Membern:

-   [Methoden](#methods)

### <a name="methods"></a>Methoden

Das **FolderItems2** -Objekt verfügt über diese Methoden.



| Methode                                            | BESCHREIBUNG                                                                                                                                                                                                                                         |
|:--------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Invokeverbex**](folderitems2-invokeverbex.md) | Führt ein Verb für eine Auflistung von [**folderItem**](folderitem.md) -Objekten aus. Bei dieser Methode handelt es sich um eine Erweiterung der [**invokeverb**](folderitem-invokeverb.md) -Methode, die eine zusätzliche Steuerung des Vorgangs über einen Satz von Flags ermöglicht.<br/> |
