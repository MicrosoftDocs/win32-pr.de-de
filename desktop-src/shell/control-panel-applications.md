# <a name="implementing-control-panel-items"></a>Implementieren von System Steuerungselementen

System Steuerungselemente sind DLLs oder ausführbare Dateien (exe-Dateien), mit denen Benutzer die Umgebung von Windows konfigurieren können. Der Zugriff erfolgt in der Regel durch Klicken auf ein Symbol in der Systemsteuerung.

In diesem Abschnitt werden die System Steuerungselemente beschrieben und erläutert, wie Sie erstellt und registriert werden, damit Sie in der Systemsteuerung ordnungsgemäß angezeigt werden. Für Windows Vista sind Informationen enthalten, in denen Sie erfahren, wie Sie Aufgaben Verknüpfungen hinzufügen, die unter dem System Steuerungselement und in den Suchergebnissen der Systemsteuerung angezeigt werden.
