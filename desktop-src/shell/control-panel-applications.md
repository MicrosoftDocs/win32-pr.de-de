# <a name="implementing-control-panel-items"></a>Implementieren Systemsteuerung Elementen

Systemsteuerung Elemente sind DLLs oder ausführbare Dateien (.exe), mit denen Benutzer die Umgebung des Windows. Auf sie wird in der Regel durch Klicken auf ein Symbol im Systemsteuerung.

In diesem Abschnitt werden Systemsteuerung elemente beschrieben und erläutert, wie sie erstellt und registriert werden, damit sie in der Tabelle entsprechend Systemsteuerung. Für Windows Vista sind Informationen enthalten, die Ihnen zeigen, wie Aufgabenlinks hinzugefügt werden, die unter dem Systemsteuerung-Element und in Systemsteuerung angezeigt werden.
