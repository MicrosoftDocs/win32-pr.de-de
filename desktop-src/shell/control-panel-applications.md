# <a name="implementing-control-panel-items"></a>Implementieren von Systemsteuerung Elementen

Systemsteuerung Elemente sind DLLs oder ausführbare Dateien (.exe), mit denen Benutzer die Umgebung von Windows konfigurieren können. Auf sie wird in der Regel durch Klicken auf ein Symbol im Systemsteuerung zugegriffen.

In diesem Abschnitt werden Systemsteuerung Elemente beschrieben und erläutert, wie sie erstellt und registriert werden, damit sie im Systemsteuerung entsprechend angezeigt werden. Für Windows Vista sind Informationen enthalten, die Sie darüber informieren, wie Sie Aufgabenlinks hinzufügen, die unter dem element Systemsteuerung und in Systemsteuerung Suchergebnissen angezeigt werden.
