# <a name="syntax-argument-the-windows-shell"></a>SYNTAX-Argument (die Windows Shell)

Mit `syntax` dem -Argument können Sie die Syntax Ihrer Abfrage identifizieren. Zulässige Werte sind entweder Natural Query Syntax (NQS) oder Advanced Query Syntax (AQS). AQS ist die Standardeinstellung und wird immer angenommen, dass sie analysiert und unterstützt wird.

## <a name="example"></a>Beispiel

``` syntax
search:query=from:paolo&syntax=AQS
```

### <a name="argument-information"></a>Argumentinformationen



|                              | Wert                                   |
|------------------------------|-----------------------------------------|
| **Mindestbetriebssystem** | Windows Vista mit Service Pack 1 (SP1) |
