# <a name="sync-and-share-verbs"></a>Synchronisieren und Freigeben von Verben

Dieses Thema enthält folgende Abschnitte:

-   [Anforderungen](#requirements)
-   [Herunterladen des Beispiels](#downloading-the-sample)
-   [Ausführen des Beispiels](#running-the-sample)
-   [Entfernen des Beispiels](#removing-the-sample)

## <a name="running-the-sample"></a>Ausführen des Beispiels

So führen Sie das Beispiel aus (Sync):

1.  Navigieren Sie zu dem Verzeichnis, das die Datei enthält. `sync.reg`
2.  Geben Sie `sync.reg ` in der Befehlszeile ein, oder Doppelklicken Sie auf das Symbol, um `sync.reg` es zu registrieren.
3.  Öffnen Sie den Windows-Explorer, und wählen Sie eine Datei aus.
4.  Klicken Sie in der Befehlsleiste auf die Option **Synchronisieren** , und wählen Sie eine unter Option wie **Paint** aus.

So führen Sie das Beispiel (Freigabe) aus:

1.  Navigieren Sie zu dem Verzeichnis, das die `share.reg` Datei enthält.
2.  Geben Sie `share.reg` in der Befehlszeile ein, oder Doppelklicken Sie auf das Symbol, um `share.reg` es zu registrieren.
3.  Öffnen Sie Windows Explorer, und wählen Sie eine Datei aus. Klicken Sie in der Befehlsleiste auf die Option **Freigeben** .
4.  Klicken Sie in der Befehlsleiste auf die Option **Freigeben mit** , und wählen Sie eine unter Option wie **Paint** aus.

## <a name="removing-the-sample"></a>Entfernen des Beispiels

So entfernen Sie das Beispiel (Sync):

1.  Navigieren Sie zu dem Verzeichnis, das die Datei "uninstallsync. reg" enthält.
2.  Geben `uninstallsync.reg` Sie in der Befehlszeile ein, oder Doppelklicken Sie auf das Symbol für `uninstallsync.reg` .

So entfernen Sie das Beispiel (Freigabe):

1.  Navigieren Sie zu dem Verzeichnis, das die Datei "uninstallshare. reg" enthält.
2.  Geben `uninstallshare.reg` Sie in der Befehlszeile ein, oder Doppelklicken Sie auf das Symbol für. `uninstallshare.reg.`
