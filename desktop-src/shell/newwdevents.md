# <a name="newwdevents-object"></a>NewWDEvents-Objekt

Erweitert das [**WebWizardHost-Objekt,**](webwizardhost.md) indem serverseitige Seiten, die in einem Assistenten gehostet werden, ermöglicht werden, um zu überprüfen, ob der Benutzer über eine Microsoft-Konto.

## <a name="members"></a>Member

Das **NewWDEvents-Objekt** verfügt über die folgenden Membertypen:

-   [Methoden](#methods)

### <a name="methods"></a>Methoden

Das **NewWDEvents-Objekt** verfügt über diese Methoden.



| Methode                                                            | BESCHREIBUNG                                                                                                                         |
|:------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------|
| [**PassportAuthenticate**](inewwdevents-passportauthenticate.md) | Ermöglicht es serverseitigen Seiten, die in einem Assistenten gehostet werden, zu überprüfen, ob der Benutzer über eine Microsoft-Konto.<br/> |
