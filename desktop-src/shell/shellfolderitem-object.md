# <a name="shellfolderitem-object"></a>Shellfolderitem-Objekt

Erweitert das [**folderItem**](folderitem.md) -Objekt. Zusätzlich zu den Eigenschaften und Methoden, die von **folderItem** unterstützt werden, verfügt **shellfolderitem** über zwei zusätzliche Methoden.

## <a name="members"></a>Member

Das **shellfolderitem** -Objekt verfügt über diese Typen von Membern:

-   [Methoden](#methods)

### <a name="methods"></a>Methoden

Das **shellfolderitem** -Objekt verfügt über diese Methoden.



| Methode                                                       | BESCHREIBUNG                                                                                                                                                                                         |
|:-------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ExtendedProperty**](shellfolderitem-extendedproperty.md) | Ruft den Wert einer Eigenschaft aus dem Eigenschaften Satz eines Elements ab. Die-Eigenschaft kann entweder anhand des Namens oder anhand des Format Bezeichners (fmtid) und des Eigenschafts Bezeichners (PID) des Eigenschaften Satzes angegeben werden.<br/> |
| [**Invokeverbex**](invokeverbex.md)                         | Führt ein Verb für ein shellelement aus.<br/>                                                                                                                                                         |
