# <a name="shellfolderitem-object"></a>ShellFolderItem-Objekt

Erweitert das [**FolderItem-Objekt.**](folderitem.md) Zusätzlich zu den Eigenschaften und Methoden, die von **FolderItem** unterstützt werden, verfügt **ShellFolderItem** über zwei zusätzliche Methoden.

## <a name="members"></a>Member

Das **ShellFolderItem-Objekt** verfügt über diese Typen von Membern:

-   [Methoden](#methods)

### <a name="methods"></a>Methoden

Das **ShellFolderItem-Objekt** verfügt über diese Methoden.



| Methode                                                       | BESCHREIBUNG                                                                                                                                                                                         |
|:-------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Extendedproperty**](shellfolderitem-extendedproperty.md) | Ruft den Wert einer Eigenschaft aus dem Eigenschaftensatz eines Elements ab. Die Eigenschaft kann entweder anhand des Namens oder des Formatbezeichners (FMTID) und des Eigenschaftenbezeichners (PID) des Eigenschaftensatzes angegeben werden.<br/> |
| [**InvokeVerbEx**](invokeverbex.md)                         | Führt ein Verb für ein Shellelement aus.<br/>                                                                                                                                                         |
