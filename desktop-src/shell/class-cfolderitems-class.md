# <a name="cfolderitems-class"></a>Cfolderitems-Klasse

**Cfolderitems** ist eine Sammlung von [**folderitems**](folderitems.md). Es implementiert die folgenden Schnittstellen: [**ipersistfolder**](/windows/desktop/api/shobjidl_core/nn-shobjidl_core-ipersistfolder), [**IObjectWithSite**](/windows/win32/api/ocidl/nn-ocidl-iobjectwithsite), [**FolderItems3**](folderitems3-object.md), [**iobjectafety**](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/aa768224(v=vs.85)), [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch).

## <a name="remarks"></a>Bemerkungen

Die **cfolderitems** -Klasse listet nur Elemente aus einem bestimmten Ordner auf.
