# <a name="about-synchronization-manager"></a>Informationen zur Synchronisierungs Verwaltung

Der Synchronisierungs-Manager stellt eine zentralisierte Standardtechnologie zum Synchronisieren von Dateien für die Offline Verwendung auf einem lokalen Computer bereit. Der Benutzer kann einen Computer für die Offline Verwendung vorbereiten, indem er die benötigten Dateien aus dem Netzwerkspeicher aktualisiert. Wenn der Computer wieder online geschaltet wird, können alle Offline Änderungen wieder in das Netzwerk kopiert werden. Syncmgr bietet eine gemeinsame Benutzeroberfläche, die von allen Anwendungen zum Synchronisieren Ihrer Dateien freigegeben werden kann.

Dateien werden unabhängig vom Protokoll übertragen. Ein e-Mail-Programm kann z. b. seine Nachrichten mit SMTP, NMTP oder POP3 übertragen, während ein Browser http verwenden kann und eine Datenbank RPC verwenden kann. Das Protokoll ist für syncmgr transparent.

Syncmgr ist auch Speicher unabhängig und kann Dateien, e-Mail-Nachrichten, HTML-Seiten oder Datenbankinformationen transparent übertragen.
