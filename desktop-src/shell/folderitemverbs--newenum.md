# <a name="folderitemverbs_newenum-method"></a>Folderitemverbs. \_ Methode ' netwenum '

Erstellt ein neues [**folderitemverbs**](folderitemverbs.md) -Objekt, das eine Kopie dieses folderitemverbs-Objekts ist, und gibt es zurück.

## <a name="syntax"></a>Syntax


```JScript
FolderItemVerbs._NewEnum()
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.
