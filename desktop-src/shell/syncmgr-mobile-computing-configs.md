# <a name="mobile-computing-configurations"></a>Mobile Computing-Konfigurationen

Der Synchronisierungs-Manager eignet sich für Computer, die wie folgt konfiguriert sind:

-   Mobiler Computer, der in einer Dockingstation in einem Netzwerk mit hoher Bandbreite verwendet wird, aber gelegentlich über eine Einwahlverbindung verwendet wird.
-   Mobiler Computer, der hauptsächlich über eine Einwahlverbindung verwendet wird.
-   Desktopcomputer, der ausschließlich über eine DFÜ-Verbindung verwendet wird, z. B. ein Computer, der von einem Telekommunikationsanbieter verwendet wird.
-   Desktopcomputer, der ausschließlich über ein Netzwerk mit hoher Bandbreite verwendet wird.

Obwohl der letzte Fall kein typisches mobiles Szenario ist, können Latenzprobleme beim Netzwerkzugriff das lokale Zwischenspeichern von Ressourcen erleichtern. In all diesen Konfigurationen kann die Anwendung SyncMgr verwenden, um Dateien und andere Ressourcen lokal zwischenzuspeichern und zwischen online und offline zu synchronisieren.
