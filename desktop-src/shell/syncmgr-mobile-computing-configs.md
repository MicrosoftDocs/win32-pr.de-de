# <a name="mobile-computing-configurations"></a>Mobile Computing-Konfigurationen

Der Synchronisierungs-Manager eignet sich für Computer, die wie folgt konfiguriert werden:

-   Mobiler Computer, der auf einer Docking Station in einem Netzwerk mit hoher Bandbreite verwendet wird, aber gelegentlich über eine DFÜ-Verbindung verwendet wird.
-   Mobiler Computer, der größtenteils über eine DFÜ-Verbindung verwendet wird.
-   Der Desktop Computer wird streng über eine DFÜ-Verbindung verwendet, z. b. einen von einem telependler verwendeten Computer.
-   Der Desktop Computer wird streng über ein Netzwerk mit hoher Bandbreite verwendet.

Obwohl es sich bei dem letzten Fall nicht um ein typisches mobiles Szenario handelt, kann es bei Latenzproblemen mit dem Netzwerk Zugriff sinnvoll sein, Ressourcen lokal zwischenzuspeichern. In all diesen Konfigurationen kann die Anwendung syncmgr verwenden, damit Dateien und andere Ressourcen lokal zwischengespeichert und zwischen Online-und Offline Verwendung synchronisiert werden.
