# <a name="shellfolderviewscript-property"></a>Shellfolderview. Script (Eigenschaft)

\[Diese Eigenschaft wird in Windows XP oder höher nicht unterstützt.\]

Enthält das Skript Objekt für die Sicht.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
objScript = ShellFolderView.Script
```



## <a name="property-value"></a>Eigenschaftswert

Eine Variable vom Typ [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) , die das Skript Objekt empfängt.
