# <a name="ownersid-element-library-schema"></a>ownerSID-Element (Bibliotheksschema)

Das &lt; &gt; ownerSID-Element gibt die Sicherheits-ID (SID) des Erstellers der Bibliothek an. Dieses Element ist optional und verfügt über keine Attribute oder untergeordneten Elemente.

## <a name="syntax"></a>Syntax

``` syntax

<xs:element name="ownerSID" minOccurs="0"/>
```
