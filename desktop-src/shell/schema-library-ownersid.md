# <a name="ownersid-element-library-schema"></a>Besitzer-SID-Element (Bibliotheks Schema)

Das <ownerSID> -Element gibt die Sicherheits-ID (SID) des Erstellers der Bibliothek an. Dieses Element ist optional und besitzt keine Attribute oder untergeordneten Elemente.

## <a name="syntax"></a>Syntax

``` syntax
<!-- ownerSID -->
<xs:element name="ownerSID" minOccurs="0"/>
```
