# <a name="didiskquotauserid-property"></a>DIDiskQuotaUser.ID-Eigenschaft

Ruft eine ID ab, die den Benutzer eindeutig identifiziert.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
iID = DIDiskQuotaUser.ID
```



## <a name="property-value"></a>Eigenschaftswert

Ein **ganzzahliger** Wert, der das [**didiskquotauser**](didiskquotauser-object.md) -Objekt des Benutzers innerhalb eines bestimmten [**diskquotacontrol**](diskquotacontrol-object.md) -Prozesses eindeutig identifiziert.

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist für die Verwendung durch Sprachen vorgesehen, z. b. Microsoft Visual Basic, die keine Zeiger unterstützen.
