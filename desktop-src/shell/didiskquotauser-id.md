# <a name="didiskquotauserid-property"></a>DIDiskQuotaUser.ID-Eigenschaft

Ruft eine ID ab, die den Benutzer eindeutig identifiziert.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
iID = DIDiskQuotaUser.ID
```



## <a name="property-value"></a>Eigenschaftswert

Ein **Ganzzahlwert,** der das [**DIDiskQuotaUser-Objekt**](didiskquotauser-object.md) des Benutzers innerhalb eines bestimmten [**DiskQuotaControl-Prozesses eindeutig identifiziert.**](diskquotacontrol-object.md)

## <a name="remarks"></a>Hinweise

Diese Eigenschaft ist für die Verwendung durch Sprachen wie Microsoft Visual Basic, die keine Zeiger unterstützen.
