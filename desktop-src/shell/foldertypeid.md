## <a name="remarks"></a>Bemerkungen

**Foldertypeid** -Werte sind GUIDs und werden in shlguid. h deklariert.

In der Praxis können alle Werte von **foldertypeid** unabhängig vom Inhalt dieses Ordners auf einen beliebigen Ordner angewendet werden. Die **foldertypeid** ist nur eine Ansichts Vorlage, die für den erwarteten Ordner Inhalt entsprechendes Layout und Spalten bereitstellt. Es wird keine Überprüfung des Ordner Inhalts mit dem **foldertypeid** -Wert durchgeführt, um die Eignung sicherzustellen.

Diese und andere Ordner Typen sind in der Registrierung unter dem Unterschlüssel **foldertypes** definiert, der sich wie folgt befindet:

```
HKEY_LOCAL_MACHINE
   SOFTWARE
      Microsoft
         Windows
            CurrentVersion
               Explorer
                  FolderTypes
```
