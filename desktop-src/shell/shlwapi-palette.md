# <a name="shell-color-palette-handling-functions"></a>Verarbeitungsfunktionen für shellfarb Palette

In diesem Abschnitt werden die Funktionen zur Handhabung der Farbpalette von Windows Shell beschrieben. Die in dieser Dokumentation erläuterten Programmier Elemente werden von Shlwapi.dll exportiert und in "shlwapi. h" und "shlwapi. lib" definiert.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                           | BESCHREIBUNG                                                                           |
|-----------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**Color-Luma**](/windows/desktop/api/Shlwapi/nf-shlwapi-coloradjustluma)<br/>           | Ändert die Leuchtkraft eines RGB-Werts. Farbton und Sättigung sind nicht betroffen.<br/> |
| [**Colorhlstorgb**](/windows/desktop/api/Shlwapi/nf-shlwapi-colorhlstorgb)<br/>               | Konvertiert Farben von "Hue-Luminance-Sättigung (HLS)" in das RGB-Format.<br/>         |
| [**Colorrgbper HLS**](/windows/desktop/api/Shlwapi/nf-shlwapi-colorrgbtohls)<br/>               | Konvertiert Farben von RGB in das HLS-Format (Hue-Luminance-Sättigung).<br/>         |
| [**Shkreateshellpalette**](/windows/desktop/api/Shlwapi/nf-shlwapi-shcreateshellpalette)<br/> | Erstellt eine halbftone-Palette für den angegebenen Gerätekontext.<br/>               |
