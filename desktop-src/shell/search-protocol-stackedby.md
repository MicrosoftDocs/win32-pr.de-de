# <a name="stackedby-argument-the-windows-shell"></a>Stackedby-Argument (die Windows-Shell)

Das- `stackedby` Argument gibt die Eigenschaften Spalte an, nach der die Ergebnisse gestapelt werden. Sie können eine beliebige gültige Eigenschaft im Eigenschaften System stapeln.

## <a name="example"></a>Beispiel


```
search:query=vacation&stackedby=System.Author
```



### <a name="argument-information"></a>Argument Informationen



|                          |                                         |
|--------------------------|-----------------------------------------|
| Mindestens Betriebs System | Windows Vista mit Service Pack 1 (SP1) |
