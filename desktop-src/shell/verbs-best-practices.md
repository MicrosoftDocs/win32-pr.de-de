### <a name="best-practices-for-verb-implementations"></a><span data-ttu-id="2bf24-113">Bewährte Methoden für Verbimplementierungen</span><span class="sxs-lookup"><span data-stu-id="2bf24-113">Best Practices for Verb Implementations</span></span>

<span data-ttu-id="2bf24-114">Die folgende Liste enthält bewährte Methoden für Verbimplementierungen:</span><span class="sxs-lookup"><span data-stu-id="2bf24-114">The following list represents best practices for verb implementations:</span></span>

-   <span data-ttu-id="2bf24-115">Wählen Sie immer die einfachste Verbmethode aus, die Ihren Anforderungen entspricht.</span><span class="sxs-lookup"><span data-stu-id="2bf24-115">Always chose the simplest verb method that meets your needs.</span></span>
-   <span data-ttu-id="2bf24-116">Verwenden Sie nach Möglichkeit eine statische Verbmethode.</span><span class="sxs-lookup"><span data-stu-id="2bf24-116">Use a static verb method, if possible.</span></span>

-   <span data-ttu-id="2bf24-120">Stellen Sie Verben den Namen des unabhängigen Softwareherstellers (Independent Software Vendor, ISV) wie folgt `ISVName.verb` voran: .</span><span class="sxs-lookup"><span data-stu-id="2bf24-120">Prefix verbs with the independent software vendor (ISV) name as follows, `ISVName.verb`.</span></span> <span data-ttu-id="2bf24-121">Die Verwendung von nicht qualifizierten Namen kann zu Kollisionen mit mehreren ISVs führen, die denselben Verbnamen ausgewählt haben.</span><span class="sxs-lookup"><span data-stu-id="2bf24-121">Using unqualified names can result in collisions with multiple ISVs that chose the same verb name.</span></span>
-   <span data-ttu-id="2bf24-122">Verwenden Sie immer eine anwendungsspezifische ProgID.</span><span class="sxs-lookup"><span data-stu-id="2bf24-122">Always use an application-specific ProgID.</span></span> <span data-ttu-id="2bf24-123">Da einige Elementtypen diese Zuordnung nicht verwenden, ist es notwendig, herstellerspezifische Namen zu verwenden.</span><span class="sxs-lookup"><span data-stu-id="2bf24-123">Because some item types do not use this mapping, there is a need for vendor-unique names.</span></span>

-   <span data-ttu-id="2bf24-125">Akzeptieren Sie nicht den Rückgabewert **S \_ OK für** kanonische Verben, die an die [**IContextMenu::InvokeCommand-Methode übergeben**](/windows/desktop/api/shobjidl_core/nf-shobjidl_core-icontextmenu-invokecommand) werden.</span><span class="sxs-lookup"><span data-stu-id="2bf24-125">Do not accept the return value **S\_OK** for canonical verbs passed to the [**IContextMenu::InvokeCommand**](/windows/desktop/api/shobjidl_core/nf-shobjidl_core-icontextmenu-invokecommand) method.</span></span> <span data-ttu-id="2bf24-126">Dies führt dazu, dass ein Fehler bei der Implementierung des echten Verbs aufgerufen wird, und gibt einen Fehlercode für kanonische Verben zurück.</span><span class="sxs-lookup"><span data-stu-id="2bf24-126">Doing so causes a failure for the real verb implementation to be invoked, and returns a failure code for canonical verbs.</span></span> <span data-ttu-id="2bf24-127">Wenn Sie keine kanonischen Verben unterstützen, geben Sie einen Fehler zurück, wenn ein HIWORD(lpVerb)-Wert ungleich 0 (null) gefunden wird.</span><span class="sxs-lookup"><span data-stu-id="2bf24-127">If you do not support canonical verbs, then return failure when a nonzero HIWORD(lpVerb) value is encountered.</span></span>
-   <span data-ttu-id="2bf24-128">Vermeiden Sie die Verwendung **rundll32.exe** als Host für Ihr Verb.</span><span class="sxs-lookup"><span data-stu-id="2bf24-128">Avoid the use of **rundll32.exe** as the host for your verb.</span></span>
-   <span data-ttu-id="2bf24-129">Stellen Sie beim Implementieren von [**IContextMenu::QueryContextMenu**](/windows/desktop/api/shobjidl_core/nf-shobjidl_core-icontextmenu-querycontextmenu) sicher, dass Sie die Anzahl der Verben zurückgeben, die dem Menü über den **HRESULT-Wert** mithilfe des ResultFromShort-Makros hinzugefügt wurden.</span><span class="sxs-lookup"><span data-stu-id="2bf24-129">When implementing [**IContextMenu::QueryContextMenu**](/windows/desktop/api/shobjidl_core/nf-shobjidl_core-icontextmenu-querycontextmenu) be sure to return the number of verbs that have been added to the menu through the **HRESULT** value using the ResultFromShort macro.</span></span>
-   <span data-ttu-id="2bf24-130">Wenn Sie sich für einen der folgenden Registrierungsschlüsseleinträge registrieren, sollten Sie vorsichtig sein, und achten Sie darauf, den Handler für den spezifischsten Typ zu registrieren, um die möglicherweise unbeabsichtigten Folgen zu reduzieren:</span><span class="sxs-lookup"><span data-stu-id="2bf24-130">If you register on one of the following registry key entries, then exercise caution and be sure to register your handler on the most specific type to reduce the possibly unintended consequences:</span></span>
    -   <span data-ttu-id="2bf24-131">**HKEY \_ CLASSES \_ ROOT\\\***</span><span class="sxs-lookup"><span data-stu-id="2bf24-131">**HKEY\_CLASSES\_ROOT\\\***</span></span>
    -   <span data-ttu-id="2bf24-132">**HKEY \_ CLASSES \_ ROOT \\ AllFileSystemObjects**</span><span class="sxs-lookup"><span data-stu-id="2bf24-132">**HKEY\_CLASSES\_ROOT\\AllFileSystemObjects**</span></span>
    -   <span data-ttu-id="2bf24-133">**HKEY \_ CLASSES \_ ROOT \\ Folder**</span><span class="sxs-lookup"><span data-stu-id="2bf24-133">**HKEY\_CLASSES\_ROOT\\Folder**</span></span>
    -   <span data-ttu-id="2bf24-134">**HKEY \_ CLASSES \_ ROOT \\ Directory**</span><span class="sxs-lookup"><span data-stu-id="2bf24-134">**HKEY\_CLASSES\_ROOT\\Directory**</span></span>
-   <span data-ttu-id="2bf24-135">Legen Sie **die MayChangeDefaultMenu-Taste** nur fest, wenn Sie davon ausgehen, dass Sie möglicherweise das Standardverb im Kontextmenü ändern müssen.</span><span class="sxs-lookup"><span data-stu-id="2bf24-135">Set the **MayChangeDefaultMenu** key only if you anticipate that you might need to change the default verb in the shortcut menu.</span></span> <span data-ttu-id="2bf24-136">Wenn ihr Handler das Standardverb nicht ändert, sollten Sie diesen Schlüssel nicht festlegen, da dies dazu führt, dass das System die DLL unnötigerweise geladen.</span><span class="sxs-lookup"><span data-stu-id="2bf24-136">If your handler does not change the default verb, then you should not set this key because doing so causes the system to load your DLL unnecessarily.</span></span>
-   <span data-ttu-id="2bf24-137">Minimieren Sie die Arbeit, die Sie in [**IShellExtInit::Initialize ausführen.**](/windows/desktop/api/shobjidl_core/nf-shobjidl_core-ishellextinit-initialize)</span><span class="sxs-lookup"><span data-stu-id="2bf24-137">Minimize the work you perform in [**IShellExtInit::Initialize**](/windows/desktop/api/shobjidl_core/nf-shobjidl_core-ishellextinit-initialize).</span></span> <span data-ttu-id="2bf24-138">Erfassen Sie für Kontextmenühandler das an **IShellExtInit::Initialize** übergebene Datenobjekt, und verarbeiten Sie es dann in [**IContextMenu::QueryContextMenu**](/windows/desktop/api/shobjidl_core/nf-shobjidl_core-icontextmenu-querycontextmenu)oder [**IContextMenu::InvokeCommand**](/windows/desktop/api/shobjidl_core/nf-shobjidl_core-icontextmenu-invokecommand).</span><span class="sxs-lookup"><span data-stu-id="2bf24-138">For shortcut menu handlers, capture the data object passed to **IShellExtInit::Initialize**, and then process it in [**IContextMenu::QueryContextMenu**](/windows/desktop/api/shobjidl_core/nf-shobjidl_core-icontextmenu-querycontextmenu), or [**IContextMenu::InvokeCommand**](/windows/desktop/api/shobjidl_core/nf-shobjidl_core-icontextmenu-invokecommand).</span></span>

## <a name="best-practices-for-multiple-selection-verbs"></a><span data-ttu-id="2bf24-139">Bewährte Methoden für Mehrfachauswahlverben</span><span class="sxs-lookup"><span data-stu-id="2bf24-139">Best Practices for Multiple Selection Verbs</span></span>

<span data-ttu-id="2bf24-140">Da die Anzahl der Elemente in einem Mehrfachauswahl-Verbszenario groß sein kann, ist es wichtig, dass Sie die Auswirkungen Ihrer Verbimplementierung auf die Leistung berücksichtigen.</span><span class="sxs-lookup"><span data-stu-id="2bf24-140">Because the number of items in a multi-selection verb scenario can be large, it is important that you consider the performance implications of your verb implementations.</span></span> <span data-ttu-id="2bf24-141">Wenn ein Benutzer beispielsweise in einem Bereich, der eine große Anzahl von Elementen enthält, nach " " sucht und dann auf Alles auswählen klickt und mit der rechten Maustaste klickt, wird dem Verb eine Auswahl angezeigt, die Tausende von Elementen enthalten \* kann. </span><span class="sxs-lookup"><span data-stu-id="2bf24-141">For example, when a user searches for "\*" over a scope that includes large numbers of items, and then clicks **Select All** and right-clicks, the verb is presented with a selection that can have thousands of items in it.</span></span> <span data-ttu-id="2bf24-142">Daher sollten Verben nur das erste Element in der Auswahl und die Gesamtanzahl der Elemente berücksichtigen.</span><span class="sxs-lookup"><span data-stu-id="2bf24-142">As a result, verbs should only consider the first item in the selection, and the overall item count.</span></span> <span data-ttu-id="2bf24-143">Das erste Element wird entweder als das Element oben in der Ansicht oder als das Element definiert, auf das der Benutzer zuerst mit der rechten Maustaste geklickt hat.</span><span class="sxs-lookup"><span data-stu-id="2bf24-143">The first item is defined as either the item at the top of the view, or the item that the user first right-clicked.</span></span>

<span data-ttu-id="2bf24-144">In Windows 7 und höher ist die Anzahl von Elementen, die an ein Verb übergeben werden, auf 16 beschränkt, wenn ein Kontextmenü abgefragt wird.</span><span class="sxs-lookup"><span data-stu-id="2bf24-144">In Windows 7 and later, the number of items passed to a verb is limited to 16 when a shortcut menu is queried.</span></span> <span data-ttu-id="2bf24-145">Das Verb wird dann neu erstellt und mit der vollständigen Auswahl neu initialisiert, wenn dieses Verb aufgerufen wird.</span><span class="sxs-lookup"><span data-stu-id="2bf24-145">The verb is then re-created and re-initialized with the full selection when that verb is invoked.</span></span>

<span data-ttu-id="2bf24-146">In einigen Fällen ist es sinnvoll, eine kleine Anzahl fester Elemente zu berücksichtigen.</span><span class="sxs-lookup"><span data-stu-id="2bf24-146">It is appropriate in some cases to consider a small number of fixed items.</span></span> <span data-ttu-id="2bf24-147">Beispielsweise ist es für ein Diff-Verb geeignet, nur die ersten beiden Elemente zu berücksichtigen.</span><span class="sxs-lookup"><span data-stu-id="2bf24-147">For example, it is appropriate for a "diff" verb to consider only the first two items.</span></span> <span data-ttu-id="2bf24-148">Im Allgemeinen müssen Sie nicht jedes Element in der Auswahl testen, um zu prüfen, ob es sich um einen bestimmten Typ handelt, oder jedes Element in der Auswahl nach seinen Eigenschaften abfragen.</span><span class="sxs-lookup"><span data-stu-id="2bf24-148">Generally, you need not test every item in the selection to see if it is a certain type, or query every item in the selection for its properties.</span></span> <span data-ttu-id="2bf24-149">Sehen Sie sich stattdessen das erste Element an, und entscheiden Sie, ob es sinnvoll ist, Ihr Verb hinzuzufügen.</span><span class="sxs-lookup"><span data-stu-id="2bf24-149">Look rather at the first item and decide if it is appropriate to add your verb.</span></span>

### <a name="heterogenous-selections"></a><span data-ttu-id="2bf24-150">Heterogene Auswahl</span><span class="sxs-lookup"><span data-stu-id="2bf24-150">Heterogenous Selections</span></span>

<span data-ttu-id="2bf24-151">Optimistische Verben werden im Mehrfachauswahlfall automatisch hinzugefügt, vorausgesetzt, dass nicht spektivische Elemente vom Verb behandelt werden können.</span><span class="sxs-lookup"><span data-stu-id="2bf24-151">Optimistic verbs are automatically added in the multi-selection case, assuming that uninspected items can be handled by the verb.</span></span> <span data-ttu-id="2bf24-152">Im Gegensatz dazu werden pessimistische Verben nicht hinzugefügt, wenn die Auswahl unbeabsichtigte Elemente enthält, und sie werden nur in Fällen hinzugefügt, in denen die Anzahl der Elemente klein ist.</span><span class="sxs-lookup"><span data-stu-id="2bf24-152">In contrast, pessimistic verbs are not added when the selection contains uninspected items, and are only added in cases where the number of items is small.</span></span>

<span data-ttu-id="2bf24-153">Verben im Playerstil sollten optimistisch sein und die Elemente, die nicht behandelt werden, im Hintergrund überspringen.</span><span class="sxs-lookup"><span data-stu-id="2bf24-153">Player style verbs should be optimistic, and silently skip the items that are not handled.</span></span> <span data-ttu-id="2bf24-154">Wenn ein Fehler bei der Bearbeitung von Elementen zu Datenverlusten oder Verwirrung führen kann, sollte das Verb Benutzer vor Elementen warnen, die nicht verarbeitet werden können.</span><span class="sxs-lookup"><span data-stu-id="2bf24-154">If a failure to act on items can cause data loss or confusion, then the verb should warn users about items that cannot be processed.</span></span> <span data-ttu-id="2bf24-155">Beispielsweise sollte ein "backup"-Verb angeben, dass einige Elemente nicht gesichert werden konnten.</span><span class="sxs-lookup"><span data-stu-id="2bf24-155">For example a "backup" verb should indicate that some items could not be backed up.</span></span>
