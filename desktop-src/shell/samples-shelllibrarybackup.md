# <a name="shell-library-backup-sample"></a>Shellbibliothekssicherung (Beispiel)

Veranschaulicht, wie Bibliotheken als Container aufgelistet werden. Bibliotheken sind das neue Speicher Paradigma für Benutzer Dateien und werden in Windows 7 eingeführt.

Dieses Thema enthält folgende Abschnitte:

-   [Beschreibung](#description)
-   [Anforderungen](#requirements)
-   [Herunterladen des Beispiels](#downloading-the-sample)
-   [Beispiel zum Aufbau](#building-the-sample)
-   [Ausführen des Beispiels](#running-the-sample)

## <a name="description"></a>BESCHREIBUNG

Bei diesem Beispiel handelt es sich um eine fiktive Sicherungs Anwendung, die zeigt, wie Bibliotheken über das Dialogfeld "allgemeine Datei" ausgewählt werden Außerdem wird veranschaulicht, wie Bibliotheken mithilfe des Shell-Namespace Walker gesichert werden.
