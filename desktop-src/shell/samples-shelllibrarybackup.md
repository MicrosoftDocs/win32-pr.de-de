# <a name="shell-library-backup-sample"></a>Shellbibliothekssicherung (Beispiel)

Veranschaulicht, wie Bibliotheken als Container aufzählt werden. Bibliotheken sind das neue Speicherparadigma für Benutzerdateien und werden in Windows 7 eingeführt.

Dieses Thema enthält folgende Abschnitte:

-   [Beschreibung](#description)
-   [Requirements](#requirements)
-   [Herunterladen des Beispiels](#downloading-the-sample)
-   [Erstellen des Beispiels](#building-the-sample)
-   [Ausführen des Beispiels](#running-the-sample)

## <a name="description"></a>Beschreibung

Dieses Beispiel ist eine fiktive Sicherungsanwendung, die zeigt, wie Bibliotheken über das allgemeine Dateidialogfeld ausgewählt werden. Außerdem wird veranschaulicht, wie Bibliotheken mithilfe des Shell-Namespacespazierers gesichert werden.
