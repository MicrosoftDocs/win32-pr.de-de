# <a name="citemidfactorygetpropertyfromidlist-methods"></a>CItemIDFactory::GetPropertyFromIDList-Methoden

Gibt eine Eigenschaft aus [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore) in der IDList zurück.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                        | BESCHREIBUNG                                                                                                                                   |
|:------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetPropertyFromIDList**](/windows/win32/api/shidfact/nf-shidfact-citemidfactory-getpropertyfromidlist(pcuidlist_relative_pcwstr_propvariant))     | Ruft eine Eigenschaft aus [**dem IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore) in der IDList als Variante ab, wobei der Schlüssel verwendet wird.<br/>            |
| [**GetPropertyFromIDList**](/windows/win32/api/shidfact/nf-shidfact-citemidfactory-getpropertyfromidlist(pcuidlist_relative_pcwstr_propvariant))    | Ruft eine Eigenschaft aus [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore) in der IDList als Variante ab, wobei die benannte Eigenschaft verwendet wird.<br/> |
| [**GetPropertyFromIDList**]/windows/win32/api/shidfact/nf-shidfact-citemidfactory-getpropertyfromidlist(pcuidlist_relative_pcwstr_propvariant))  | Ruft mithilfe des Schlüssels eine Eigenschaft aus [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore) in der IDList ab.<br/>                         |
| [**GetPropertyFromIDList**](/windows/win32/api/shidfact/nf-shidfact-citemidfactory-getpropertyfromidlist(pcuidlist_relative_pcwstr_propvariant)) | Ruft eine Eigenschaft aus [**dem IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore) innerhalb der IDList unter Verwendung der benannten Eigenschaft ab.<br/>              |
