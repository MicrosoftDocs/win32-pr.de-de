# <a name="citemidfactorygetpropertyfromidlist-methods"></a>Citemittelfactory:: getpropertyfromittel List-Methoden

Gibt eine Eigenschaft aus dem [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore) innerhalb der idlist zurück.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                        | BESCHREIBUNG                                                                                                                                   |
|:------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------|
| [**Getpropertyfromittlerlist**](/windows/win32/api/shidfact/nf-shidfact-citemidfactory-getpropertyfromidlist(pcuidlist_relative_pcwstr_propvariant))     | Ruft eine Eigenschaft aus dem [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore) innerhalb der idlist als Variante mithilfe des Schlüssels ab.<br/>            |
| [**Getpropertyfromittlerlist**](/windows/win32/api/shidfact/nf-shidfact-citemidfactory-getpropertyfromidlist(pcuidlist_relative_pcwstr_propvariant))    | Ruft eine Eigenschaft aus dem [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore) innerhalb der idlist als Variante ab, wobei die benannte Eigenschaft verwendet wird.<br/> |
| [**Getpropertyfromittlerlist**]/Windows/Win32/API/shidfact/NF-shidfact-citemidfactory-getpropertyfromidlist (pcuidlist_relative_pcwstr_propvariant))  | Ruft eine Eigenschaft aus dem [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore) innerhalb der idlist ab, wobei der Schlüssel verwendet wird.<br/>                         |
| [**Getpropertyfromittlerlist**](/windows/win32/api/shidfact/nf-shidfact-citemidfactory-getpropertyfromidlist(pcuidlist_relative_pcwstr_propvariant)) | Ruft eine Eigenschaft aus dem [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore) innerhalb der idlist ab, wobei die benannte Eigenschaft verwendet wird.<br/>              |
