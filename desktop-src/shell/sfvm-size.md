# <a name="sfvm_size-message"></a>Sfvm- \_ Größen Meldung

Benachrichtigt das Rückruf Objekt, dass die Größe der Ordneransicht geändert wurde. Wird von [**ishellfolderviewcb:: messagesfvcb**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb)verwendet.


```C++
SFVM_SIZE
```



## <a name="parameters"></a>Parameter

Diese Nachricht weist keine Parameter auf.
