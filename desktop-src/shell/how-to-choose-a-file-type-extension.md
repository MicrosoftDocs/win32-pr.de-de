# <a name="how-to-choose-a-file-type-extension"></a>Auswählen einer Dateityperweiterung

Dateitypen werden durch eine Dateinamenerweiterung identifiziert. Bevor Sie einen neuen Dateityp definieren, sollten Sie zunächst eine Dateinamenerweiterung auswählen.

## <a name="instructions"></a>Anweisungen

### <a name="step-1"></a>Schritt 1:

Wählen Sie eine lange Dateinamenerweiterung aus, die Sie zum Identifizieren des Dateityps verwenden möchten. Weitere Informationen finden Sie unter [Dateityphandler.](fa-file-extensions.md)

### <a name="step-2"></a>Schritt 2:

Überprüfen Sie die MIME-Datenbank der [Internet Assigned Numbers Authority (IANA),](https://www.iana.org/assignments/media-types) um zu ermitteln, ob die Dateierweiterung bereits einer anderen Anwendung zugeordnet ist.

### <a name="step-3"></a>Schritt 3:

Definieren Sie einen MIME-Typ, wenn Sie einen neuen öffentlichen Dateityp definieren, und registrieren Sie ihn bei der IANA.
