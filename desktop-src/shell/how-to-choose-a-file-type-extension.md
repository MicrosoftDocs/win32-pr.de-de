# <a name="how-to-choose-a-file-type-extension"></a>Vorgehensweise beim Auswählen einer Dateityp Erweiterung

Dateitypen werden durch eine Dateinamenerweiterung identifiziert. Bevor Sie einen neuen Dateityp definieren, sollten Sie zuerst eine Dateinamenerweiterung auswählen.

## <a name="instructions"></a>Instructions

### <a name="step-1"></a>Schritt 1:

Wählen Sie eine lange Dateinamenerweiterung aus, die Sie verwenden möchten, um den Dateityp zu identifizieren. Weitere Informationen finden Sie unter [Dateityp Handler](fa-file-extensions.md).

### <a name="step-2"></a>Schritt 2:

Überprüfen Sie die MIME-Datenbank der [Internet Assigned Numbers Authority (IANA)](https://www.iana.org/assignments/media-types) , um zu ermitteln, ob die Dateinamenerweiterung bereits einer anderen Anwendung zugeordnet ist.

### <a name="step-3"></a>Schritt 3:

Definieren Sie einen MIME-Typ, wenn Sie einen neuen öffentlichen Dateityp definieren, und registrieren Sie ihn bei der IANA.
