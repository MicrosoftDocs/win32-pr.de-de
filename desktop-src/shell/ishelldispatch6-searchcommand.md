# <a name="ishelldispatch6searchcommand-method"></a>IShellDispatch6.SearchCommand-Methode

Zeigt den Bereich Apps-Suche an, der normalerweise angezeigt wird, wenn Sie beginnen, einen Suchbegriff aus dem Startbildschirm.

## <a name="syntax"></a>Syntax


```JScript
IShellDispatch6.SearchCommand()
```


```VB

IShellDispatch6.SearchCommand()
```





## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

### <a name="jscript"></a>JScript

Diese Methode gibt keinen Wert zurück.

### <a name="vb"></a>VB

Diese Methode gibt keinen Wert zurück.
