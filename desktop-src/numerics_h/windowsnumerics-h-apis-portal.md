# <a name="windowsnumericsh-apis"></a>windowsnumerics. h-APIs

Die Header Datei windowsnumerics. h definiert C++ Vector-und Matrix Typen im [**Windows. Foundation. Numerics**](/uwp/api/Windows.Foundation.Numerics) -Namespace. Die Strukturen werden von **Windows. Foundation. Numerics** durch einen Bereich mathematischer Operatoren und Funktionen erweitert.

Dieser Namespace ist nur in C++ verfügbar. Die .NET-Entsprechung ist [System. Numerics](/dotnet/api/system.numerics?view=netframework-4.8).

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [**float2-Struktur**](float2-structure.md) | Ein Vektor mit zwei Komponenten. |
| [**float3-Struktur**](float3-structure.md) | Ein Vektor mit drei Komponenten. |
| [**float3x2-Struktur**](float3x2-structure.md) | Eine 3 x 2-Matrix, die für 2D-Transformationen verwendet wird. |
| [**FLOAT4-Struktur**](float4-structure.md) | Ein Vektor mit vier Komponenten. |
| [**float4x4-Struktur**](float4x4-structure.md) | Eine 4 x 4-Matrix, die für 3D-Transformationen verwendet wird. |
| [**Plane-Struktur**](plane-structure.md) | Diese Struktur stellt eine Ebene mit einem 3D-Vektor normal und einem Entfernungs Wert dar. |
| [**Quaternion-Struktur**](quaternion-structure.md) | Ein vierdimensionaler Vektor, der zum Darstellen einer Drehung verwendet wird. |
| [**Windows-Numerics und directxmath-Interop-APIs**](windows-numerics-and-directxmath-interop-apis.md) | Diese Funktionen konvertieren [**Windows. Foundation. Numerics**](/uwp/api/Windows.Foundation.Numerics) -Typen in und aus den directxmath-SIMD-Typen [xmvector](../dxmath/xmvector-data-type.md) und [xmmatrix](/windows/win32/api/directxmath/ns-directxmath-xmmatrix). |
