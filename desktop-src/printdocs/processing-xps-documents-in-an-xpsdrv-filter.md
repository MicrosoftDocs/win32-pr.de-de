# <a name="processing-xps-documents-in-an-xpsdrv-filter"></a>Verarbeiten von XPS-Dokumenten in einem XPSDrv-Filter

Weitere Informationen zu XPS-Filtern und XPSDrv-Druckertreibern finden Sie unter [XPS-Filter](/windows-hardware/drivers/print/xps-filters) im Windows-Treiberkit.
