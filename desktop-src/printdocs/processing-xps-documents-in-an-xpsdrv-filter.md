# <a name="processing-xps-documents-in-an-xpsdrv-filter"></a>Verarbeiten von XPS-Dokumenten in einem XPSDrv-Filter

Die XPS OM kann verwendet werden, um über einen XPSDrv-Druckertreiberfilter auf die Komponenten eines XPS-Dokuments zuzugreifen.

Weitere Informationen zu XPS-Filtern und XPSDrv-Druckertreibern finden Sie unter [XPS-Filter](/windows-hardware/drivers/print/xps-filters) im Windows Driver Kit.
