# <a name="print-ticket-api-enumerations"></a>API-Enumerationen der Druck Ticket

Die folgenden Enumerationen werden von der Print Ticket-API verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt



| Enumeration                                                   | Beschreibung                                                                                                                                                             |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Edefaultdevmodetype**](/windows/win32/api/prntvpt/ne-prntvpt-edefaultdevmodetype)<br/> | Ermöglicht es Benutzern, den [**DEVMODE**](/windows/win32/api/wingdi/ns-wingdi-devmodea) anzugeben, der als Quelle für Standardwerte verwendet werden soll, wenn ein Druck Ticket nicht alle möglichen Einstellungen angibt.<br/> |
| [**Eprintticketscope**](/windows/desktop/api/prntvpt/ne-prntvpt-eprintticketscope)<br/>     | Gibt den Bereich eines Druck Tickets an.<br/>                                                                                                                       |
