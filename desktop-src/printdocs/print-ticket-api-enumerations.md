# <a name="print-ticket-api-enumerations"></a>Druckticket-API-Enumerationen

Die folgenden Enumerationen werden von der Druckticket-API verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt



| Enumeration                                                   | Beschreibung                                                                                                                                                             |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**EDefaultDevmodeType**](/windows/win32/api/prntvpt/ne-prntvpt-edefaultdevmodetype)<br/> | Ermöglicht Es Benutzern, anzugeben, [**welcher DEVMODE**](/windows/win32/api/wingdi/ns-wingdi-devmodea) als Quelle für Standardwerte verwendet werden soll, wenn ein Druckticket nicht alle möglichen Einstellungen an enthält.<br/> |
| [**EPrintTicketScope**](/windows/desktop/api/prntvpt/ne-prntvpt-eprintticketscope)<br/>     | Gibt den Bereich eines Drucktickets an.<br/>                                                                                                                       |
