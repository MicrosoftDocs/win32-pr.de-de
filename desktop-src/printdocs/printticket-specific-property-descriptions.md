# <a name="printticket-specific-property-descriptions"></a>PrintTicket-spezifische Eigenschaftenbeschreibungen

Dieses Thema ist nicht aktuell. Die aktuellsten Informationen finden Sie unter Print Schema Specification (Spezifikation des [Druckschemas).](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip)

In diesem Abschnitt werden die öffentlichen Schlüsselwörter beschrieben, die in einem PrintTicket angezeigt werden können.

Es müssen nicht alle Schlüsselwörter unterstützt werden. Implementierer von PrintTicket MAY unterstützen Funktionen nur, wenn sie auf ihre eigene spezifische Implementierung anwendbar sind.

## <a name="element-type-implementations-for-printticket-specific-property-descriptions"></a>Elementtypimplementierungen für PrintTicket-spezifische Eigenschaftenbeschreibungen

Es ist wichtig, auf die übergeordneten und untergeordneten Elemente zu achten, die im Strukturinhalt für jedes Schlüsselwort zulässig sind. Weitere Informationen finden Sie im Abschnitt [Zusammenfassung der Elementtypen.](./summary-of-element-types.md)
