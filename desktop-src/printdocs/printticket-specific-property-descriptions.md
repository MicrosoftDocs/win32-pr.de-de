# <a name="printticket-specific-property-descriptions"></a>Beschreibungen der PrintTicket-spezifischen Eigenschaften

Dieses Thema ist nicht aktuell. Die aktuellsten Informationen finden Sie in der [PrintSchema-Spezifikation](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

In diesem Abschnitt werden die öffentlichen Schlüsselwörter erläutert, die möglicherweise in einem PrintTicket angezeigt werden.

Es ist nicht erforderlich, dass alle Schlüsselwörter unterstützt werden. Implementierer von PrintTicket können Funktionen nur unterstützen, wenn er auf eine eigene spezifische Implementierung anwendbar ist.

## <a name="element-type-implementations-for-printticket-specific-property-descriptions"></a>Elementtyp Implementierungen für PrintTicket-spezifische Eigenschafts Beschreibungen

Es ist wichtig, dass Sie auf die übergeordneten und untergeordneten Elemente achten, die im strukturellen Inhalt der einzelnen Schlüsselwörter zulässig sind. Weitere Informationen finden Sie im Abschnitt [Zusammenfassung der Element Typen](./summary-of-element-types.md) .
