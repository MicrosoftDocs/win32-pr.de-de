# <a name="schema-imposed-limitations"></a>Einschränkungen für Schema-Imposed

Dieses Thema ist nicht aktuell. Die aktuellsten Informationen finden Sie in der [PrintSchema-Spezifikation](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Das printfunktionalitäten-Schema bietet printexpressions-Autoren eine beträchtliche Menge an Flexibilität und Ausdruckskraft, die in Geräte Beschreibungen verwendet werden kann. Konfigurations Abhängigkeiten stellen jedoch eine Einschränkung für diese Flexibilität und Ausdruckskraft dar. ParameterDef-Instanzen, die Liste der Funktionselemente, die Liste der Options Instanzen in den einzelnen Features und die ScoredProperty-Instanzen in jeder Option dürfen keine Konfigurations Abhängigkeiten enthalten. Der printfunktionalitäten-Dokument Anbieter muss ungültige Kombinationen von Konfigurationen erkennen und diese auflösen.
