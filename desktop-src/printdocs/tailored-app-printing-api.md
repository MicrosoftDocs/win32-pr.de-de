# <a name="print-document-package-api"></a>Druckdokumentpaket-API

Stellt eine Schnittstelle bereit, mit der eine Anwendung auf das Druckdokumentpaket zugreifen und es verwalten kann.

Die Windows Store App-Druck-API ist mit Windows 8 und neueren Versionen von Windows Desktop verfügbar.
