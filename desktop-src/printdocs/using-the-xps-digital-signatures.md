# <a name="xps-om-digital-signatures"></a>Digitale XPS-OM-Signaturen

In diesem Thema werden die Schnittstellen für digitale Signaturen der XPS-Dokument-API aufgelistet.

Das XPS-OM unterstützt digitale Signaturen. auf digitale Signaturen in einem XPS-Dokument kann jedoch auch direkt zugegriffen werden, ohne dass ein XPS-OM erstellt werden muss. Weitere Informationen zum Zugreifen auf XPS Digital Signaturen ohne XPS-OM finden Sie unter [XPS Digital Signature API](xps-digital-signatures.md).
