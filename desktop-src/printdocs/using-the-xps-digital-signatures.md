# <a name="xps-om-digital-signatures"></a>Digitale XPS OM-Signaturen

In diesem Thema werden die digitalen Signaturschnittstellen der XPS-Dokument-API aufgelistet.

XPS OM unterstützt digitale Signaturen. auf digitale Signaturen in einem XPS-Dokument kann jedoch auch direkt zugegriffen werden, ohne dass eine XPS OM erstellt werden muss. Weitere Informationen zum Zugriff auf digitale XPS-Signaturen ohne XPS OM finden Sie unter [XPS Digital Signature API](xps-digital-signatures.md).
