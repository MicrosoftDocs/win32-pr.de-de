# <a name="overview-of-the-printcapabilities"></a>Übersicht über Print-Funktionen

Dieses Thema ist nicht aktuell. Die aktuellsten Informationen finden Sie in der [PrintSchema-Spezifikation](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Die printfunktionen beschreiben die Konfigurationen oder Zustände, die auf einem Gerät verfügbar sind. Ein Gerät, das von printfunktionen beschrieben wird, kann häufig in einer Reihe von verschiedenen Konfigurationen platziert werden. Bei einem Druck Gerät enthalten typische Druck Attribute den Duplexdruck, die Auflösung und die Mediengröße. Wenn das Gerät diese Attribute unterstützt, sind Sie Teil der Konfiguration für das Gerät. Der Benutzer wählt eine bestimmte Konfiguration aus, indem er jedem der verfügbaren Attribute einen bestimmten Wert zuweist. Beispiel: "Duplex: Onesided", "Resolution: 600 dpi" und "mediasize: Legal".

Die Print-Funktionen basieren auf dem Print Schema-Framework, das die Elemente definiert, die in printfunktionen verwendet werden können. Die Schlüsselwörter des Druck Schemas definieren häufig verwendete Element Hierarchien oder Schlüsselwörter, um die Portabilität der Informationen und der Absicht zwischen einzelnen Clients zu fördern. Die Schlüsselwörter des Druck Schemas lassen jedoch auch private Erweiterungen zu, damit printfunktionen an bestimmte Anforderungen angepasst werden können.
