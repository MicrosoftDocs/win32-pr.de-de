# <a name="printticket-syntax-and-schema"></a>PrintTicket-Syntax und -Schema

Dieses Thema ist nicht aktuell. Die aktuellsten Informationen finden Sie unter Print Schema Specification (Spezifikation des [Druckschemas).](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip)

PrintTickets werden in der Syntax des XML-basierten Druckschemaframeworks ausgedrückt. Daher sollte der Leser die [Zusammenfassung der Elementtypen](summary-of-element-types.md) auf die erforderliche Syntax überprüfen.

Das PrintTicket-Schema ist hinsichtlich der verwendeten Elementtypen und in der allgemeinen Struktur, die von übergeordneten und untergeordneten Elementen ausgedrückt wird, eng parallel zum Print Schema Framework. Beachten Sie, dass ParameterDef-Elemente in einem PrintTicket nicht zulässig sind und ParameterInit-Elemente nur in einem PrintTicket zulässig sind.
