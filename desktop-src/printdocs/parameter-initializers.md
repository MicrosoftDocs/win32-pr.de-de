# <a name="parameter-initializers"></a>Parameterinitialisierer

Dieses Thema ist nicht aktuell. Die aktuellsten Informationen finden Sie unter Print Schema Specification (Spezifikation des [Druckschemas).](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip)

Eine Zusammenfassung des ParameterInit-Elementtyps finden Sie im Abschnitt [ParameterInit.](parameterinit.md)

Ein ParameterInit-Elementtyp weist einer ParameterDef einen Wert zu. Die ParameterDef wird daher im PrintTicket initialisiert. Ein Beispiel für einen ParameterInit-Elementtyp finden Sie im [PrintTicket-Beispiel](printticket-example.md)für das Druckschema.

Weitere Informationen zur Interaktion zwischen ParameterDef-Elementen und ParameterInit-Elementen finden Sie im Abschnitt [ParameterDef- und ParameterInit-Elemente.](./parameterdef-and-parameterinit-elements.md)
