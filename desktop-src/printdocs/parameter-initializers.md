# <a name="parameter-initializers"></a>Parameterinitialisierer

Dieses Thema ist nicht aktuell. Die aktuellsten Informationen finden Sie in der [PrintSchema-Spezifikation](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Eine Zusammenfassung des parameterinit-Elementtyps finden Sie im Abschnitt [parameterinit](parameterinit.md) .

Ein parameterinit-Elementtyp weist einem ParameterDef einen Wert zu. Der ParameterDef wird daher in PrintTicket initialisiert. Ein Beispiel für einen parameterinit-Elementtyp finden Sie im Beispiel Print Schema Print [Ticket](printticket-example.md).

Weitere Informationen zur Interaktion zwischen ParameterDef-Elementen und parameterinit-Elementen finden Sie im Abschnitt [ParameterDef und parameterinit Elements](./parameterdef-and-parameterinit-elements.md) .
