# <a name="xps-om-color-management"></a>XPS OM-Farbverwaltung

In diesem Thema werden die Farbverwaltungsschnittstellen der XPS-Dokument-API in einer XPS OM aufgelistet.

-   [**IXpsOMColorProfileResource-Schnittstelle**](/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomcolorprofileresource)
-   [**IXpsOMColorProfileResourceCollection-Schnittstelle**](/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomcolorprofileresourcecollection)

Bei Verwendung einer [**XPS \_ COLOR-Struktur**](xps-color.md) ist eine Farbprofilressource erforderlich, wenn der Wert des **colorType-Members** **XPS \_ COLOR TYPE \_ \_ CONTEXT** ist.
