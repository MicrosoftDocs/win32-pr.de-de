# <a name="print-schema-language-notes"></a>Drucken von Schemasprachhinweisen

Dieses Thema ist nicht aktuell. Die aktuellen Informationen finden Sie unter [Print Schema Specification (Spezifikation des Druckschemas).](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip)

In dieser Spezifikation werden die Wörter, mit denen die Bedeutung der einzelnen Anforderungen definiert wird, in Großbuchstaben geschrieben. Diese Wörter werden in Übereinstimmung mit ihren Definitionen in RFC 2119 verwendet, und ihre jeweiligen Bedeutungen werden unten reproduziert:



| Begriff                                                                | BESCHREIBUNG                                                                                                                                                                                                                                                                                         |
|---------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="MUST"></span><span id="must"></span>muss<br/>       | Dieses Wort oder das Adjektiv REQUIRED bedeutet, dass das Element eine absolute Anforderung der Spezifikation ist.<br/>                                                                                                                                                                               |
| <span id="SHOULD"></span><span id="should"></span>Sollten<br/> | Dieses Wort oder das Adjektiv RECOMMENDED bedeutet, dass es unter bestimmten Umständen gültige Gründe geben kann, dieses Element zu ignorieren, aber die vollständigen Auswirkungen sollten verstanden und der Fall sorgfältig abgewogen werden, bevor Sie einen anderen Kurs auswählen.<br/>                                     |
| <span id="MAY"></span><span id="may"></span>Kann<br/>          | Dieses Wort oder das Adjektiv OPTIONAL bedeutet, dass dieses Element wirklich optional ist. Beispielsweise kann eine Implementierung das Element enthalten, da es für einen bestimmten Marketplace oder ein bestimmtes Szenario erforderlich ist oder weil es das Produkt verbessert. Eine andere Implementierung kann dasselbe Element weglassen.<br/> |
