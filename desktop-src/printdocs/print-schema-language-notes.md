# <a name="print-schema-language-notes"></a>Sprach Hinweise zum Druck Schema

Dieses Thema ist nicht aktuell. Die aktuellsten Informationen finden Sie in der [PrintSchema-Spezifikation](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

In dieser Spezifikation werden die Wörter, die verwendet werden, um die Bedeutung der einzelnen Anforderungen zu definieren, in Großbuchstaben geschrieben. Diese Wörter werden in Übereinstimmung mit ihren Definitionen in RFC 2119 verwendet, und die entsprechende Bedeutung wird im folgenden dargestellt:



| Begriff                                                                | BESCHREIBUNG                                                                                                                                                                                                                                                                                         |
|---------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="MUST"></span><span id="must"></span>Müssen<br/>       | Dieses Wort oder das erforderliche Adjektiv bedeutet, dass es sich bei dem Element um eine absolute Anforderung der Spezifikation handelt.<br/>                                                                                                                                                                               |
| <span id="SHOULD"></span><span id="should"></span>Sollte<br/> | Dieses Wort oder das empfohlene Adjektiv bedeutet, dass es in bestimmten Situationen möglicherweise gültige Gründe gibt, dieses Element zu ignorieren, aber die vollständigen Implikationen sollten verstanden werden, und die Groß-/Kleinschreibung sollte vor der Auswahl eines anderen Kurses sorgfältig abgewogen werden.<br/>                                     |
| <span id="MAY"></span><span id="may"></span>Durchaus<br/>          | Dieses Wort oder das optionale Adjektiv bedeutet, dass dieses Element wirklich optional ist. Beispielsweise kann eine Implementierung das Element einschließen, da es für einen bestimmten Marketplace oder ein bestimmtes Szenario erforderlich ist oder das Produkt erweitert wird. Eine andere Implementierung kann das gleiche Element weglassen.<br/> |
