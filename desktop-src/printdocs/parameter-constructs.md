# <a name="parameter-constructs"></a>Parameterkonstrukte

Dieses Thema ist nicht aktuell. Die aktuellen Informationen finden Sie unter [Print Schema Specification (Spezifikation des Druckschemas).](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip)

Parameter, die im Druckschema verwendet werden, beschreiben präzise einen zulässigen Wertebereich anstelle einer diskreten Enumeration von Werten. Das Druckschemaframework definiert drei Parameterspezifische Elemente: ParameterDef-Elemente beschreiben den Parameter und seine Merkmale, ParameterInit-Elemente definieren eine bestimmte Instanz oder einen Wert für einen Parameter, und ParameterRef-Elemente verweisen auf ein bestimmtes ParameterInit-Element, das an anderer Stelle definiert ist.
