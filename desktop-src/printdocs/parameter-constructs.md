# <a name="parameter-constructs"></a>Parameterkonstrukte

Dieses Thema ist nicht aktuell. Die aktuellsten Informationen finden Sie in der [PrintSchema-Spezifikation](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Parameter, die im Druck Schema verwendet werden, beschreiben einen akzeptablen Wertebereich und keine diskrete Enumeration von Werten. Das Print Schema Framework definiert drei Elemente, die für Parameter spezifisch sind: ParameterDef-Elemente beschreiben den Parameter und seine Merkmale, parameterinit-Elemente definieren eine bestimmte Instanz oder einen Wert für einen Parameter, und ParameterRef-Elemente verweisen auf ein bestimmtes parameterinit-Element, das an anderer Stelle definiert ist.
