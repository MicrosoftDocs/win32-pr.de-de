# <a name="user-configurable-elements"></a>Vom benutzerkonfigurierbare Elemente

Dieses Thema ist nicht aktuell. Die aktuellsten Informationen finden Sie in der [PrintSchema-Spezifikation](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

In diesem Abschnitt werden die öffentlichen Schlüsselwörter erläutert, die möglicherweise in einem Print-Funktionen-Dokument oder in PrintTicket angezeigt werden.

Es ist nicht erforderlich, dass alle Schlüsselwörter und Elemente von Schlüsselwörtern unterstützt werden. Implementierer von PrintTicket und Print-Funktionen sollten die-Funktionalität nur dann unterstützen, wenn er auf das Zielgerät und/oder den Treiber anwendbar ist.

## <a name="element-type-implementations-for-user-configurable-elements"></a>Elementtyp Implementierungen für vom benutzerkonfigurierbare Elemente

Es ist wichtig, dass Sie auf die übergeordneten und untergeordneten Elemente achten, die im strukturellen Inhalt der einzelnen Schlüsselwörter zulässig sind. Weitere Informationen finden Sie im Abschnitt [Zusammenfassung der Element Typen](./summary-of-element-types.md) .
