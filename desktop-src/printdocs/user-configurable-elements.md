# <a name="user-configurable-elements"></a>Vom Benutzer konfigurierbare Elemente

Dieses Thema ist nicht aktuell. Die aktuellen Informationen finden Sie unter [Print Schema Specification (Spezifikation des Druckschemas).](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip)

In diesem Abschnitt werden die öffentlichen Schlüsselwörter beschrieben, die entweder in einem PrintCapabilities-Dokument oder in PrintTicket angezeigt werden können.

Alle Schlüsselwörter und Elemente von Schlüsselwörtern müssen nicht unterstützt werden. Implementierer von PrintTicket und PrintCapabilities sollten Funktionen nur unterstützen, wenn sie auf das Zielgerät und/oder den Treiber anwendbar sind.

## <a name="element-type-implementations-for-user-configurable-elements"></a>Elementtypimplementierungen für vom Benutzer konfigurierbare Elemente

Es ist wichtig, auf die übergeordneten und untergeordneten Elemente zu achten, die im strukturellen Inhalt für jedes Schlüsselwort zulässig sind. Weitere Informationen finden Sie im Abschnitt [Zusammenfassung der Elementtypen.](./summary-of-element-types.md)
