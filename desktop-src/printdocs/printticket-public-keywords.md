# <a name="printticket-public-keywords"></a>Öffentliche PrintTicket-Schlüsselwörter

Dieses Thema ist nicht aktuell. Die aktuellen Informationen finden Sie unter [Print Schema Specification (Spezifikation des Druckschemas).](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip)

Im folgenden Abschnitt werden sowohl PrintTicket-spezifische Eigenschaftenbeschreibungen als auch Parameterinitialisierer angegeben, die auf ein PrintTicket anwendbar sein können.

## <a name="printticket-specific-property-descriptions-usage"></a>PrintTicket Specific Property Descriptions Usage

PrintTicket-spezifische Eigenschaftendeskriptoren stellen öffentliche Schlüsselwörter des Druckschemas dar, die möglicherweise in einem PrintTicket angezeigt werden. Sie enthalten bestimmte Informationen, die eine Anwendung oder ein Benutzer möglicherweise in einem PrintTicket angeben möchte.

## <a name="parameter-initializers-usage"></a>Verwendung von Parameterinitialisierern

Parameterinitialisierer haben die Form von ParameterInit-Elementtypen in einem PrintTicket. ParameterInit-Elementtypen enthalten den Wert des Parameters und werden in einem PrintTicket initialisiert. Weitere Informationen finden Sie im Abschnitt [Parameterkonstrukte.](parameter-constructs.md)
