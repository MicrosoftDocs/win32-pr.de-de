# <a name="printticket-public-keywords"></a>Öffentliche Schlüsselwörter von PrintTicket

Dieses Thema ist nicht aktuell. Die aktuellsten Informationen finden Sie in der [PrintSchema-Spezifikation](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Im folgenden Abschnitt werden sowohl PrintTicket-spezifische Eigenschaften Beschreibungen als auch parameterinitialisierer angegeben, die möglicherweise auf ein Print Ticket anwendbar sind.

## <a name="printticket-specific-property-descriptions-usage"></a>Verwendung von PrintTicket-spezifischen Eigenschafts Beschreibungen

PrintTicket-spezifische Eigenschaften Deskriptoren stellen öffentliche Schlüsselwörter des Druck Schemas dar, die in einem PrintTicket vorkommen können. Sie enthalten bestimmte Informationen, die von einer Anwendung oder einem Benutzer möglicherweise in einem PrintTicket angegeben werden.

## <a name="parameter-initializers-usage"></a>Verwendung von parameterinitialisierern

Parameterinitialisierer verwenden die Form von parameterinit-Elementtypen in einem PrintTicket. Parameterinit-Elementtypen enthalten den Wert des-Parameters und werden in einem PrintTicket initialisiert. Weitere Informationen finden Sie im Abschnitt [Parameterkonstrukte](parameter-constructs.md) .
