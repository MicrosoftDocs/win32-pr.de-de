# <a name="xps-document-api"></a>XPS-Dokument-API

Die XPS-Dokument-API ist eine native Windows-API, die es Programmen ermöglicht, XPS-Dokumente zu erstellen, zu lesen, zu bearbeiten, zu speichern und zu drucken. Die XPS-Dokument-API wird in Windows 7 und höheren Versionen von Windows unterstützt und kann im benutzermodusprogramm und in XPSDrv-Druckertreibern verwendet werden.

Die XPS-Dokument-API unterstützt sowohl XPS als auch openxps und ist für C++-Programmierer konzipiert, die Native Windows-Anwendungen schreiben. Weitere Informationen zur Open XPS-Spezifikation finden Sie unter [Open XML Paper Specification (openxps)](https://www.ecma-international.org/publications/standards/Ecma-388.htm).
