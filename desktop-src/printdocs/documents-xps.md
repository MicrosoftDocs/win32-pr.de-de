# <a name="xps-document-api"></a>XPS-Dokument-API

Die XPS-Dokument-API ist eine native Windows-API, mit der Programme XPS-Dokumente erstellen, lesen, bearbeiten, speichern und drucken können. Die XPS-Dokument-API wird in Windows 7 und höher von Windows unterstützt und kann in Benutzermodusprogrammen und XPSDrv-Druckertreibern verwendet werden.

Die XPS-Dokument-API bietet sowohl XPS als auch OpenXPS und ist für C++-Programmierer konzipiert, die native Windows Anwendungen schreiben. Weitere Informationen zur Open XPS-Spezifikation finden Sie unter [Open XML Paper Specification (OpenXPS)](https://www.ecma-international.org/publications/standards/Ecma-388.htm).
