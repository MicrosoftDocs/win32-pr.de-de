# <a name="quick-starter-guide"></a>Schnellstartanleitung

Dieses Thema ist nicht aktuell. Die aktuellsten Informationen finden Sie unter Print Schema Specification (Spezifikation des [Druckschemas).](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip)

Diese Seite soll Ihnen einen schnellen Einstieg in die Implementierung von PrintTicket und PrintCapabilities für Ihre Anwendung oder Ihren Gerätetreiber ermöglichen. Wir empfehlen Ihnen zwar, die Spezifikation des Druckschemas vollständig zu lesen, aber auf dieser Seite erhalten Sie einen Einstieg in wichtige Bereiche, auf die Sie achten sollten.

1) Lesen Sie [die Druck-Schema-Related-Technologien,](print-schema-related-technologies.md) um ihnen einen Überblick über printTicket- und Druckfunktionen-Technologien zu geben.

2) Stellen Sie sicher, dass Sie die [Zusammenfassung der Elementtypen](summary-of-element-types.md) verstanden haben, die zum Ausdrücken von Technologien im Zusammenhang mit dem Druckschema verwendet werden.

3) PrintCapabilities-Dokumente und PrintTickets werden auf drei [Bereichspräfixebenen](scoping-prefix.md) definiert: Auftrag, Dokument und Seite.

4) Lesen der verschiedenen [XML-Attribute,](xml-attributes.md) die in verschiedenen Print Schema-Elementtypen verwendet werden

5) Lesen Sie die Checkliste für die Erstellung von [PrintCapabilities-Dokumenten](printcapabilities-document-construction-checklist.md) sowie das bereitgestellte [PrintCapabilities-Dokumentbeispiel.](printcapabilities-document-example.md)

6) Sehen Sie sich die Prüflisten für die [PrintTicket-Konstruktion](printticket-construction-checklists.md) und auch das [bereitgestellte PrintTicket-Beispiel](printticket-example.md) an.

7) PrintTicket-Anbieter sollten auch die Prüfliste für die [PrintTicket-Überprüfung](printticket-validation-checklist.md) überprüfen, um die Konformität mit der PrintSchema-Spezifikation sicherzustellen.

8) Lesen Sie sowohl [benutzerkonfigurierbare Elemente](user-configurable-elements.md) als [auch parameterdefinitionen](parameter-definitions.md) öffentliche Druckschemaschlüsselwörter, die in einem PrintCapabilities-Dokument angezeigt werden können.
