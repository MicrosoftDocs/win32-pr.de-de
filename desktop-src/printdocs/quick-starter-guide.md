# <a name="quick-starter-guide"></a>Schnellstartanleitung

Dieses Thema ist nicht aktuell. Die aktuellen Informationen finden Sie unter [Print Schema Specification (Spezifikation des Druckschemas).](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip)

Diese Seite soll Ihnen einen schnellen Einstieg in die Implementierung von PrintTicket und PrintCapabilities für Ihre Anwendung oder Ihren Gerätetreiber bieten. Wir empfehlen Ihnen zwar, die Spezifikation für das Druckschema vollständig zu lesen, aber diese Seite hilft Ihnen bei einem Einstieg in wichtige Bereiche, auf die Sie achten sollten.

1) Lesen Sie [print Schema-Related Technologies ,](print-schema-related-technologies.md) um einen Überblick über die Technologien PrintTicket und Print Capabilities zu erhalten.

2) Stellen Sie sicher, dass Sie die Zusammenfassung der [Elementtypen](summary-of-element-types.md) verstanden haben, die verwendet werden, um Druckschema-bezogene Technologien auszudrücken.

3) PrintCapabilities-Dokumente und PrintTickets werden auf drei [Bereichspräfixebenen](scoping-prefix.md) definiert: Auftrag, Dokument und Seite.

4) Lesen der verschiedenen [XML-Attribute,](xml-attributes.md) die in verschiedenen Druckschema-Elementtypen verwendet werden

5) Lesen Sie die [Prüfliste für die Dokumentkonstruktion von PrintCapabilities](printcapabilities-document-construction-checklist.md) und das [bereitgestellte PrintCapabilities-Dokumentbeispiel.](printcapabilities-document-example.md)

6) Überprüfen Sie die [Prüflisten für die PrintTicket-Konstruktion und](printticket-construction-checklists.md) das bereitgestellte [PrintTicket-Beispiel.](printticket-example.md)

7) PrintTicket-Anbieter sollten auch die [Prüfliste für die PrintTicket-Überprüfung überprüfen,](printticket-validation-checklist.md) um die Konformität mit der PrintSchema-Spezifikation sicherzustellen.

8) Überprüfen Sie sowohl [vom Benutzer konfigurierbare Elemente](user-configurable-elements.md) als auch von [Parameterdefinitionen](parameter-definitions.md) öffentliche Druckschemaschlüsselwörter, die in einem PrintCapabilities-Dokument enthalten sein können.
