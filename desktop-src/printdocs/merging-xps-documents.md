Beschreibt die Verwendung der XPS-Dokument-API zum Zusammenführen mehrerer XPS-Dokumente in einem einzelnen XPS-Dokument.

Weitere Informationen zum Zusammenführen von XPS-Dokumenten finden Sie im Beispielprogramm unter [XpsRollup](/previous-versions/windows/desktop/ff819182(v=vs.85)).
