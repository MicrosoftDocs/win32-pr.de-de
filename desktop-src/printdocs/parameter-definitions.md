# <a name="parameter-definitions"></a>Parameterdefinitionen

Dieses Thema ist nicht aktuell. Die aktuellsten Informationen finden Sie in der [PrintSchema-Spezifikation](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

In diesem Abschnitt werden die Definitionen der öffentlichen Parameter beschrieben, die möglicherweise in einem printfunktionalitäten-Dokument angezeigt werden.

Parameter werden verwendet, um einen akzeptablen Wertebereich anstelle einer diskreten Enumeration von Werten zu beschreiben.

## <a name="parameterdef"></a>ParameterDef

Eine Zusammenfassung des ParameterDef-Elementtyps finden Sie im Abschnitt [ParameterDef](parameterdef.md) .

Weitere Informationen zur Interaktion zwischen ParameterDef-Elementen und parameterinit-Elementen finden Sie im Abschnitt [ParameterDef und parameterinit Elements](./parameterdef-and-parameterinit-elements.md) .

ParameterDef-Elemente, die in den Schlüsselwörtern des Druck Schemas definiert sind, müssen in einem Printworks-Dokument vollständig definiert sein.

## <a name="parameterref"></a>ParameterRef

Eine Zusammenfassung des ParameterRef-Elementtyps finden Sie im Abschnitt [ParameterRef](parameterref.md) .

Ein vom Benutzer konfigurierbares Element Schlüsselwort kann einen Verweis auf einen Parameter enthalten. ParameterRef-Elemente werden als untergeordnetes Element eines ScoredProperty-Elements angegeben, um weitere Informationen zu erhalten. Weitere Informationen finden Sie im Abschnitt [Parameter Verweis Elemente](parameter-reference-elements.md) .
