# <a name="parameter-definitions"></a>Parameterdefinitionen

Dieses Thema ist nicht aktuell. Die aktuellen Informationen finden Sie unter [Print Schema Specification (Spezifikation des Druckschemas).](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip)

In diesem Abschnitt werden die öffentlichen Parameterdefinitionen beschrieben, die in einem PrintCapabilities-Dokument enthalten sein können.

Parameter werden verwendet, um einen akzeptablen Wertebereich zu beschreiben, anstatt eine diskrete Enumeration von Werten zu beschreiben.

## <a name="parameterdef"></a>ParameterDef

Eine Zusammenfassung des ParameterDef-Elementtyps finden Sie im [Abschnitt ParameterDef.](parameterdef.md)

Weitere Informationen zur Interaktion zwischen ParameterDef-Elementen und ParameterInit-Elementen finden Sie im Abschnitt [ParameterDef- und ParameterInit-Elemente.](./parameterdef-and-parameterinit-elements.md)

ParameterDef-Elemente, die in den Print Schema Keywords definiert sind, müssen vollständig in einem PrintCapabilities-Dokument definiert werden.

## <a name="parameterref"></a>ParameterRef

Eine Zusammenfassung des ParameterRef-Elementtyps finden Sie im [Abschnitt ParameterRef.](parameterref.md)

Ein vom Benutzer konfigurierbares Elementschlüsselwort kann einen Verweis auf einen Parameter enthalten. ParameterRef-Elemente werden als untergeordnetes Element eines ScoredProperty-Elements angegeben. Weitere Informationen finden Sie im Abschnitt [Parameterverweiselemente.](parameter-reference-elements.md)
