# <a name="whats-new-for-printing"></a>Neues beim Drucken

Windows 8 unterstützt das [Drucken für Windows Store-Apps mit JavaScript und HTML](/previous-versions/windows/apps/hh465225(v=win.10)) und das [Drucken für Windows Store-Apps mithilfe von c#/VB/C + + und XAML](/previous-versions/windows/apps/hh465196(v=win.10)).

Windows 8 umfasst auch eine neue COM-basierte API, die vollständige Unterstützung für das Drucken und zugreifen von [openxps](app-support-for-openxps-printing.md) auf Teile der neuen Druckertreiber bietet, die von Windows 8 unterstützt werden. Weitere Informationen zur neuen API finden Sie unter [Print Document Package API](tailored-app-printing-api.md).

Das Eingangsbox-Programm für den Windows-Drucktreiber stellt sicher, dass Windows 8 Unterstützung für einen hohen Prozentsatz der beliebten Drucker umfasst
