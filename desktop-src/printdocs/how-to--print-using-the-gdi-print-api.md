# <a name="how-to-print-using-the-gdi-print-api"></a>How To: Print Using the GDI Print API

In diesem Thema wird das Drucken aus einem nativen Windows mithilfe der [GDI-Druck-API beschrieben.](gdi-printing.md)

## <a name="overview"></a>Übersicht

Das Drucken ist in der Regel ein integraler Bestandteil eines nativen Windows-Programms, daher ist es kein Feature, das Sie nach dem Schreiben des Programms problemlos hinzufügen können. Beim Entwerfen eines Programms für Windows 7 sollten Sie die Verwendung der [XPS-Druck-API](xps-printing.md) in Betracht ziehen, um die Druckfunktionalität zur Verfügung zu stellen, da sie für die Zukunft die größte Kompatibilität bietet. Programme, die unter Windows Vista und früheren Versionen von Windows ausgeführt werden müssen, verwenden wahrscheinlich die [GDI-Druck-API,](gdi-printing.md) um Druckfunktionen zur Verfügung zu stellen. Die GDI-Druck-API wird auch in Version Windows 7 unterstützt, ist jedoch eingeschränkter als die XPS-Druck-API.
