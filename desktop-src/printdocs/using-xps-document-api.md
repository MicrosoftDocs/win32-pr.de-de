# <a name="using-xps-document-api"></a>Verwenden der XPS-Dokument-API

In diesem Abschnitt wird beschrieben, wie Sie die [XPS-Dokument-API](documents-xps.md) zum Ausführen von Programmieraufgaben verwenden.

Beispiele für die Verwendung der [XPS-Dokument-API](documents-xps.md) in einem Programm finden Sie im Abschnitt [Programmiertasks der XPS-Dokument-API.](#xps-document-api-programming-tasks)

Informationen zur Verwendung des XPS-Objektmodells und zur Implementierung durch die [XPS-Dokument-API](documents-xps.md)finden Sie unter Informationen zur [XPS-Dokument-API.](about-xps-document-api.md)

## <a name="getting-started-with-the-xps-document-api"></a>Erste Schritte mit der XPS-Dokument-API

Bevor Sie mit der Verwendung der XPS-Dokument-API beginnen, stellen Sie sicher, dass Sie mit den folgenden Programmierthemen vertraut sind:<dl>

[COM-Programmierung](/windows/desktop/com/component-object-model--com--portal)  
[Fehlerbehandlung in COM](/windows/desktop/com/error-handling-in-com)  
</dl>

Wenn Sie die XPS-Dokument-API verwenden, sollten Sie auch die folgenden Technologien verwenden:<dl>

[DirectWrite](/windows/desktop/DirectWrite/direct-write-portal)  
[XPS-Druck-API](./printing-with-the-xpsprint-api.md)  
[API für digitale XPS-Signaturen](xps-digital-signatures.md)  
</dl>
