# <a name="whats-new-for-printing-in-windows-7"></a>Neues beim Drucken in Windows 7

Hier wird beschrieben, was für Dokumente und das Drucken in Windows 7 neu ist.

## <a name="documents-and-printing"></a>Dokumente und Drucken 

Dokumente und Drucken wurden für Windows 7 verbessert, um die folgenden neuen Features und Verbesserungen zu bieten.



|                                                                                                                |                                                                                                                                                                                                                                                                                                                          |
|----------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [XPS-Dokument-API](/previous-versions/windows/desktop/dd316976(v=vs.85))<br/>                                                        | Mit der [XPS-Dokument-API](/previous-versions/windows/desktop/dd316976(v=vs.85)) können Sie XPS-Dokumente aus Ihrem Win32-Programm erstellen und als Dokument Datei speichern oder direkt an einen Drucker senden.<br/> Sie können auch die [XPS-Dokument-API](/previous-versions/windows/desktop/dd316976(v=vs.85)) verwenden, um vorhandene XPS-Dokumente in einem Win32-Programm zu ändern.<br/> |
| [XPS-API für digitale Signaturen](/previous-versions/windows/desktop/ff819108(v=vs.85))<br/>                                      | Mit der [XPS Digital Signature-API](/previous-versions/windows/desktop/ff819108(v=vs.85)) können Sie XPS-Dokumenten digitale Signaturen aus Ihrem Win32-Programm hinzufügen.<br/> Sie können auch die XPS Digital Signature-API verwenden, um zu überprüfen, ob ein XPS-Dokument authentisch und unverändert ist.<br/>                                          |
| [XPS-Druck-API](xps-printing.md)<br/>                                                                   | Mit der [XPS-Druck-API](xpsprint-api.md) können Sie XPS-Dokumente aus Ihrem Win32-Programm drucken.<br/>                                                                                                                                                                                                                   |
| Verbesserungen an [Microsoft XPS Document Converter (mxdc)](microsoft-xps-document-converter--mxdc-.md)<br/> | Der [mxdc](microsoft-xps-document-converter--mxdc-.md) wurde verbessert, sodass Sie vorhandene GDI-Funktionen verwenden können, um XPS-Inhalte zu erstellen und in den XPS-Druckpfad von ihren Win32-Programmen mit höherer grafischer Genauigkeit zu drucken, als dies in Windows Vista möglich war.<br/>                                   |
