# <a name="print-spooler-api-enumerations"></a>Enumerationen der Druck Spooler-API

Dieser Abschnitt enthält eine alphabetische Auflistung der Enumerationen, die von der druckspoolerapi verwendet werden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Enumeration                                                             | Beschreibung                                                                                                        |
|-------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**Eprintxpsjoboperation**](eprintxpsjoboperation.md)<br/>       | Gibt an, ob sich ein XPS-Druckauftrag in der spoolingphase oder der Renderingphase befindet.<br/>                           |
| [**Eprintxpsjobprogress**](eprintxpsjobprogress.md)<br/>         | Gibt an, wie der Spooler gerade ausgeführt wird, während er einen XPS-Druckauftrag verarbeitet.<br/>                         |
| [**Druck \_ Ausführungs \_ Kontext**](print-execution-context.md)<br/> | Stellt den Ausführungs Kontext dar, wenn [**getprintexecutiondata**](getprintexecutiondata.md) aufgerufen wird.<br/> |
| [**\_druckeroptionsflags \_**](printer-option-flags.md)<br/>       | Gibt das Caching eines Handles für einen mit [**OpenPrinter2**](openprinter2.md)geöffneten Drucker an.<br/>       |
