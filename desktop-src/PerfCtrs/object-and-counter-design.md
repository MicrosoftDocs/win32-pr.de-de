<span data-ttu-id="34514-112">Nachdem Sie entschieden haben, ob das Objekt eine einzelne Instanz oder mehrere Instanzen unterstützt, müssen Sie den Typ der Indikatoren festlegen, die das Objekt bereitstellen soll.</span><span class="sxs-lookup"><span data-stu-id="34514-112">After deciding if your object supports a single instance or multiple instances, you need to decide on the type of counters that you want the object to provide.</span></span> <span data-ttu-id="34514-113">Beispielsweise können Sie Leistungswerte angeben, die als Rohwerte, als Raten oder als Prozentsätze angezeigt werden.</span><span class="sxs-lookup"><span data-stu-id="34514-113">For example, you can provide counter values that are displayed as raw values, as rates, or as percentages.</span></span>

<span data-ttu-id="34514-116">Die Methode, die Sie verwenden, um die Daten zu erfassen, kann so einfach sein wie das erhöhen eines Zählers, wenn eine bestimmte Routine in der Anwendung aufgerufen wird, oder es kann zeitaufwändige Berechnungen einschließen.</span><span class="sxs-lookup"><span data-stu-id="34514-116">The method you use to collect the data can be as simple as incrementing a counter each time a particular routine in the application is called, or it can involve time-consuming calculations.</span></span> <span data-ttu-id="34514-117">Zähler und Timer sollten Inkrement und nie gelöscht werden.</span><span class="sxs-lookup"><span data-stu-id="34514-117">Counters and timers should increment and never be cleared.</span></span> <span data-ttu-id="34514-118">Leistungsindikatoren können eingebunden werden, solange Sie nicht zweimal zwischen den Stichproben durch den Consumer gebunden werden.</span><span class="sxs-lookup"><span data-stu-id="34514-118">Counters can wrap, as long as they do not wrap twice between being sampled by the consumer.</span></span> <span data-ttu-id="34514-119">Die Anwendung kann Daten während ihrer normalen Ausführung erfassen und speichern, solange Sie sich nicht auf Ihre Leistung auswirkt.</span><span class="sxs-lookup"><span data-stu-id="34514-119">Your application can collect and store data during its normal execution, as long as it does not affect its performance.</span></span>
