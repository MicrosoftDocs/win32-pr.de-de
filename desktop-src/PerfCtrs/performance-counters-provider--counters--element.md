# <a name="provider-counters-element"></a>Provider (Counters)-Element

Identifiziert den Anbieter und die Leistungsindikatoren, die er bereitstellt.

``` syntax
<xs:element name="provider"
    type="man:provider"
 />
```

Das **Provider** -Element wird durch den [**komplexen Typ**](performance-counters-counters-complex-type.md) "Indikatoren" definiert.
