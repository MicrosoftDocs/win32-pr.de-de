# <a name="limited-user-access-support"></a>Eingeschränkte Benutzer Zugriffs Unterstützung

Nur der Administrator des Computers oder der Benutzer in der Benutzergruppe Leistungs Protokolle kann die Leistungsdaten der Leistungs Protokolle protokollieren. Benutzer in der Gruppe "Administratoren" können nur die Daten der Daten protokollieren, wenn das Tool, das zum Protokollieren von Counter-Daten verwendet wird, über ein Eingabe Aufforderungs Fenster gestartet wird, das mit **als Administrator ausführen** geöffnet ist. Alle Benutzer in interaktiven Anmelde Sitzungen können gegen Daten anzeigen. Benutzer in nicht interaktiven Anmelde Sitzungen müssen sich jedoch in der Gruppe der Benutzer der Leistungsüberwachung befinden, um Leistungsdaten anzeigen zu können.

**Windows XP:** Der Administrator oder die Benutzer in der Gruppe "Administratoren" können gegen Daten ohne Einschränkung protokollieren und anzeigen.
