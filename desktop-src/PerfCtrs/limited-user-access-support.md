# <a name="limited-user-access-support"></a>Eingeschränkte Benutzerzugriffsunterstützung

Nur der Administrator des Computers oder Benutzer in der Benutzergruppe "Leistungsprotokolle" kann Indikatordaten protokollieren. Benutzer in der Gruppe Administrator können Indikatordaten nur protokollieren, wenn das Tool, mit dem sie Indikatordaten protokollieren, über ein Eingabeaufforderungsfenster gestartet wird, das mit Als Administrator **ausführen... geöffnet wird.** Alle Benutzer in interaktiven Anmeldesitzungen können Indikatordaten anzeigen. Benutzer in nicht interaktiven Anmeldesitzungen müssen sich jedoch in der Gruppe Leistungsüberwachungsbenutzer befingen, um Leistungsindikatordaten anzeigen zu können.

**Windows XP:** Der Administrator oder Benutzer in der Gruppe Administrator können Indikatordaten ohne Einschränkung protokollieren und anzeigen.
