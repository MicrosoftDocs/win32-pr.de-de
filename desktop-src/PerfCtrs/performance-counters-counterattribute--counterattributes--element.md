# <a name="counterattribute-counterattributes-element"></a>counterAttribute(counterAttributes)-Element

Identifiziert ein Attribut eines Indikators, der beschreibt, wie der Zähler angezeigt werden soll.

``` syntax
<xs:element name="counterAttribute"
    type="man:counterAttribute"
 />
```

Das **counterAttribute-Element** wird durch den [**komplexen counterAttributes-Typ**](performance-counters-counterattributes-complex-type.md) definiert.
