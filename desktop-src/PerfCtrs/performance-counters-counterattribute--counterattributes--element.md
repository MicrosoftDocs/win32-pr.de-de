# <a name="counterattribute-counterattributes-element"></a>Counter Attribute-Element (Counter Attribute)

Identifiziert ein Attribut eines Zählers, der beschreibt, wie der-Wert angezeigt werden soll.

``` syntax
<xs:element name="counterAttribute"
    type="man:counterAttribute"
 />
```

Das **counterAttribute** -Element wird durch den komplexen [**counterAttribute**](performance-counters-counterattributes-complex-type.md) -Typ definiert.
