# <a name="struct-structs-element"></a>struct-Element (Strukturen)

Identifiziert eine -Struktur, die die Werte der Leistungsindikatoren im übergeordneten Indikatorsatz enthält.

``` syntax
<xs:element name="struct"
    type="man:struct"
 />
```

Das **Strukturelement wird** durch den [**komplexen Strukturtyp**](performance-counters-structs-complex-type.md) definiert.
