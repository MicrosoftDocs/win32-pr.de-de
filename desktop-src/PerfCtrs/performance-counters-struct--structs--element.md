# <a name="struct-structs-element"></a>struct (structs)-Element

Identifiziert eine-Struktur, die die Werte der Zähler im übergeordneten Indikatorensatz enthält.

``` syntax
<xs:element name="struct"
    type="man:struct"
 />
```

Das **struct** -Element wird durch den komplexen [**Strukturen**](performance-counters-structs-complex-type.md) -Typ definiert.
