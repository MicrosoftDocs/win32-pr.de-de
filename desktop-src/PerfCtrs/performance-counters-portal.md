# <a name="performance-counters"></a>Leistungsindikatoren

Windows Leistungsindikatoren bieten eine abstraktionsschicht auf hoher Ebene, die eine konsistente Schnittstelle zum Sammeln verschiedener Arten von Systemdaten bietet, z. B. CPU-, Arbeitsspeicher- und Datenträgerauslastung. Systemadministratoren verwenden häufig Leistungsindikatoren, um Systeme auf Leistungs- oder Verhaltensprobleme zu überwachen. Softwareentwickler verwenden häufig Leistungsindikatoren, um die Ressourcennutzung ihrer Programme zu untersuchen.
