# <a name="performance-counters"></a>Leistungsindikatoren

Windows-Leistungsindikatoren stellen eine Abstraktions Ebene auf hoher Ebene bereit, die eine konsistente Schnittstelle zum Erfassen verschiedener Arten von Systemdaten bietet, z. b. CPU, Arbeitsspeicher und Datenträger Nutzung. System Administratoren verwenden häufig Leistungsindikatoren, um Systeme auf Leistungs-oder Verhaltensprobleme zu überwachen. Software Entwickler verwenden häufig Leistungsindikatoren, um die Ressourcenverwendung ihrer Programme zu untersuchen.
