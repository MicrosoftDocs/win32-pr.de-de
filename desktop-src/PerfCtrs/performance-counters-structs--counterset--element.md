# <a name="structs-counterset-element"></a>Strukturen (CounterSet)-Element

Eine Liste von-Strukturen, die die Werte der Zähler im übergeordneten Indikatorensatz enthalten.

``` syntax
<xs:element name="structs"
    type="man:structs"
 />
```

Das **Strukturen** -Element wird durch den komplexen [**CounterSet**](performance-counters-counterset-complex-type.md) -Typ definiert.

## <a name="remarks"></a>Bemerkungen

Wird nur von kernelmodusanbietern verwendet.
