# <a name="getting-counter-information"></a>Abrufen von Indikatorinformationen

Rufen Sie die [**PdhGetCounterInfo-Funktion**](/windows/desktop/api/Pdh/nf-pdh-pdhgetcounterinfoa) auf, um Informationen zu einem Indikator abzurufen, z. B. den Indikatortyp, den Hilfetext und den Pfad. Die Funktion gibt die Indikatorinformationen als [**PDH \_ COUNTER \_ INFO-Struktur**](/windows/desktop/api/Pdh/ns-pdh-pdh_counter_info_a) zurück.
