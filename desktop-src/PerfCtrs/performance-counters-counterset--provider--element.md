# <a name="counterset-provider-element"></a>Counter Set-Element (Anbieter)

Enthält eine Liste mit Leistungsindikatoren, die logisch verknüpft sind.

``` syntax
<xs:element name="counterSet"
    type="man:counterSet"
 />
```

Das **CounterSet** -Element wird durch den komplexen [**Anbietertyp**](performance-counters-provider-complex-type.md) definiert.
