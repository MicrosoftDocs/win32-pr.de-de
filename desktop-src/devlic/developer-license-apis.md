# <a name="developer-license-apis"></a>Entwicklerlizenz-APIs

Verwenden Sie diese APIs, um eine Entwicklerlizenz zu überprüfen, zu erwerben und zu entfernen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                 | Beschreibung                                             |
|-----------------------------------------------------------------------|---------------------------------------------------------|
| [**CheckDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-checkdeveloperlicense)<br/>     | Überprüft, ob eine Entwicklerlizenz vorhanden ist.<br/> |
| [**AcquireDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-acquiredeveloperlicense)<br/> | Erwirbt eine Entwicklerlizenz.<br/>                |
| [**RemoveDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-removedeveloperlicense)<br/>   | Entfernt eine Entwicklerlizenz.<br/>                 |
