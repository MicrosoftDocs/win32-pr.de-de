# <a name="developer-license-apis"></a>Entwicklerlizenz-APIs

Verwenden Sie diese APIs, um eine Entwicklerlizenz zu überprüfen, zu erwerben und zu entfernen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                 | BESCHREIBUNG                                             |
|-----------------------------------------------------------------------|---------------------------------------------------------|
| [**Checkdeveloperlicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-checkdeveloperlicense)<br/>     | Prüft, ob eine Entwicklerlizenz vorhanden ist.<br/> |
| [**Acquiredeveloperlicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-acquiredeveloperlicense)<br/> | Erhält eine Entwicklerlizenz.<br/>                |
| [**Removedeveloperlicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-removedeveloperlicense)<br/>   | Entfernt eine Entwicklerlizenz.<br/>                 |
