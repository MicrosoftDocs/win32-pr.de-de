# <a name="power-management-control-codes"></a>Steuerungs Codes der Energie Verwaltung

Die folgenden Steuerungs Codes werden mit [Akkus](battery-information.md)verwendet.



| Steuerungscode                                                                  | BESCHREIBUNG                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------|
| [**IOCTL- \_ Akku \_ Abfrage \_ Informationen**](ioctl-battery-query-information.md) | Ruft eine Vielzahl von Informationen für den Akku ab. |
| [**IOCTL \_ Akku \_ Query- \_ Status**](ioctl-battery-query-status.md)           | Ruft den aktuellen Status für den Akku ab.      |
| [**IOCTL \_ Akku \_ - \_ abfragetag**](ioctl-battery-query-tag.md)                 | Ruft das aktuelle Tag des Akkus ab.               |
| [**IOCTL- \_ Akku \_ Satz \_ Informationen**](ioctl-battery-set-information.md)     | Legt verschiedene Akku Informationen fest.                  |



 

Die folgenden Steuercodes sind Teil der [Back-Light-Steuerelement Schnittstelle](backlight-control-interface.md).



| Steuerungscode                                                                                 | BESCHREIBUNG                                                                   |
|----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**Bildschirmhelligkeit der IOCTL- \_ Video \_ Abfrage \_ \_**](ioctl-video-query-display-brightness.md)     | Ruft die aktuellen AC-und DC-Timeout-Ebenen und den aktuellen Energiezustand ab. |
| [**\_ \_ \_ unterstützte \_ Helligkeit der IOCTL-Video Abfrage**](ioctl-video-query-supported-brightness.md) | Ruft die unterstützten Timeout-Ebenen ab.                                     |
| [**Anzeige Helligkeit für IOCTL- \_ Video \_ Satz \_ \_**](ioctl-video-set-display-brightness.md)         | Legt die aktuellen AC-und DC-Timeout-Ebenen fest.                                   |
