# <a name="power-schemes"></a>Energie Schemas

Ein Energie Schema ist eine Sammlung von Einstellungen, mit denen der Energieverbrauch des Computers gesteuert wird. Mithilfe von Energie Schemas können Sie den Stromverbrauch einzelner Geräte oder des gesamten Systems reduzieren. Sie können optionale Features konfigurieren, z. b. die Unterstützung für den Ruhezustand, den Zeitraum, in dem sich das System im Leerlauf befinden muss, bevor die Anzeige ausgeschaltet wird

Es gibt mehrere Möglichkeiten, Energie Schemas zu erstellen: die Anwendung Energieoptionen in der Systemsteuerung, das Befehlszeilen Tool PowerCfg.exe und die API für das Energie Schema. Weitere Informationen zu Energieoptionen oder PowerCfg.exe finden Sie in der Hilfe zum Betriebssystem.

-   [Energie Schema Verwaltung](managing-power-schemes.md)
-   [Administrator Überschreibungen](administrator-overrides.md)
