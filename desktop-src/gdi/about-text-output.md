# <a name="about-text-output"></a>Informationen zur Text Ausgabe

Die Text Ausgabe ist der häufigste Typ der Grafikausgabe, der im Client Bereich gefunden wird. Sie wird von Anwendungen auf unterschiedliche Weise verwendet. Textverarbeitungs-und Desktop Veröffentlichungs Anwendungen erstellen Dokumente mit formatiertem Text; Arbeitsblatt Anwendungen verwenden Text, Zahlen und Symbole zum Angeben von Formeln, Beschriftungs Spalten und Listen Werten. Datenbankanwendungen erstellen Datensätze und zeigen Abfragen mit Text an, und in CAD-Anwendungen wird Text zum bezeichnen von Objekten und Anzeigen von Dimensionen verwendet.

Es sind Funktionen zum Formatieren und Zeichnen von Text im Client Bereich einer Anwendung und auf einer Seite mit Drucker Papier verfügbar. Diese Funktionen können in zwei Kategorien unterteilt werden: solche, die den Text formatieren (oder für die Ausgabe vorbereiten), und diejenigen, die den Text tatsächlich zeichnen. Die Formatierungsfunktionen richten Text aus, legen den Abstand zwischen Zeichen fest, legen die Text-und Text Hintergrundfarben fest und rechtfertigen Text. Die Zeichnungsfunktionen zeichnen einzelne Zeichen (oder Symbole) oder gesamte Text Zeichenfolgen.

Wenn Sie in Microsoft Windows arbeiten, werden harte Zeilenumbrüche mit dem Wagen Rücklauf/Zeilenvorschub-Paar ( \\ r \\ n) angegeben.
