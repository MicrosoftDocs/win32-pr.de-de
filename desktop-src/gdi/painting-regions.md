# <a name="painting-regions"></a>Zeichnungs Bereiche

Eine Anwendung füllt das Innere eines Bereichs mithilfe des aktuell ausgewählten Pinsels durch die Funktion " [**paintrgn**](/windows/desktop/api/Wingdi/nf-wingdi-paintrgn) " in einem Gerätekontext aus. Diese Funktion verwendet die aktuellen Polygon Füll Modi (Alternative und Auffüllung).
