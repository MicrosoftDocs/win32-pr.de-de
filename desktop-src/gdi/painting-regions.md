# <a name="painting-regions"></a>Zeichnen von Regionen

Eine Anwendung füllt das Innere eines Bereichs mithilfe des Pinsels aus, der derzeit von der [**PaintRgn-Funktion**](/windows/desktop/api/Wingdi/nf-wingdi-paintrgn) in einen Gerätekontext ausgewählt wurde. Diese Funktion verwendet die aktuellen Polygonfüllmodi (alternative und Windung).
