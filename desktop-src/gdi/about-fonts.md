# <a name="about-fonts"></a>Informationen zu Schriftarten

Für eine Person, die in der Mechanik der Handschrift Komposition trainiert oder mit der Standard Typografie vertraut ist, können einige der typografischen Begriffe, die in dieser Übersicht verwendet werden, ungewöhnlich sein. Die meisten Unterschiede zwischen der Standard Typografie und dem Text in Microsoft Windows spiegeln die Technologie Änderungen wider. Die ursprünglichen typografischen Begriffe basieren auf der Hot-Metal-Komposition, während die hier verwendeten Begriffe, die als Elementnamen für die Schriftart-und Textausgabe Strukturen angezeigt werden, auf eine neue Technologie basierend auf der Ausgabe des Laserdruckers und der Komposition auf einem PC mit Desktop Publishing Software hinweisen.
