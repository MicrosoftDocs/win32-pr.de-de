# <a name="filled-shape-functions"></a>Gefüllte Shape-Funktionen

Die folgenden Funktionen werden mit gefüllten Formen verwendet.



| Funktion                           | BESCHREIBUNG                                                     |
|------------------------------------|-----------------------------------------------------------------|
| [**Chord**](/windows/desktop/api/Wingdi/nf-wingdi-chord)             | Zeichnet einen Bereich, der durch eine Ellipse und ein Liniensegment begrenzt ist.         |
| [**Ellipse**](/windows/desktop/api/Wingdi/nf-wingdi-ellipse)         | Zeichnet eine Ellipse.                                               |
| [**FillRect**](/windows/desktop/api/Winuser/nf-winuser-fillrect)       | Füllt ein Rechteck mit einem Pinsel.                                |
| [**FrameRect**](/windows/desktop/api/Winuser/nf-winuser-framerect)     | Zeichnet mit einem Pinsel einen Rahmen um ein Rechteck.                |
| [**Invertrect**](/windows/desktop/api/Winuser/nf-winuser-invertrect)   | Kehrt die Farbwerte der Pixel in einem Rechteck um.          |
| [**Kreis**](/windows/desktop/api/Wingdi/nf-wingdi-pie)                 | Zeichnet einen kreisförmigen Keil, der durch eine Ellipse und zwei Radials begrenzt ist. |
| [**Polygon**](/windows/desktop/api/Wingdi/nf-wingdi-polygon)         | Zeichnet ein Polygon.                                                |
| [**Polypolygon**](/windows/desktop/api/Wingdi/nf-wingdi-polypolygon) | zeichnet eine Reihe von geschlossenen Polygonen.                              |
| [**Rollen**](/windows/desktop/api/Wingdi/nf-wingdi-rectangle)     | Zeichnet ein Rechteck.                                              |
| [**RoundRect**](/windows/desktop/api/Wingdi/nf-wingdi-roundrect)     | Zeichnet ein Rechteck mit abgerundeten Ecken.                         |
