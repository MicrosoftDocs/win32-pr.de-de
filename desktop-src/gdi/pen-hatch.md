# <a name="pen-hatch"></a>Stift Schraffur

Das Attribut "Schraffur" gibt den Schraffurtyp eines geometrischen Stifts mit dem Schraffurmuster an. Es stehen sechs Muster zur Verfügung. In der folgenden Abbildung werden die mit verschiedenen Schraffurmustern gezeichneten Zeilen dargestellt
