# <a name="pen-hatch"></a>Stiftschraffur

Das Schraffurattribut gibt den Schraffurtyp eines geometrischen Stifts mit dem Schraffurmusterattribut an. Es sind sechs Muster verfügbar. Die folgende Abbildung zeigt Linien, die mit verschiedenen Schraffurmustern gezeichnet wurden.
