# <a name="metafiles-windows-gdi"></a>Metafiles (Windows GDI)

Eine *Metadatei* ist eine Auflistung von-Strukturen, die ein Bild in einem geräteunabhängigen Format speichern. Die Geräteunabhängigkeit ist das einzige Feature, bei dem Metadateien von Bitmaps getrennt festgelegt werden. Anders als bei einer Bitmap gewährleistet eine Metadatei die Unabhängigkeit von Geräten. Es gibt jedoch einen Nachteil für Metadateien, dass Sie in der Regel langsamer als Bitmaps gezeichnet werden. Wenn eine Anwendung schnelles zeichnen und Geräteunabhängigkeit erfordert, sollte daher anstelle von Metadateien Bitmaps verwendet werden.

-   [Informationen zu Metadateien](about-metafiles.md)
-   [Verwenden von Metafiles](using-metafiles.md)
-   [Metadateiverweis](metafile-reference.md)

Weitere Informationen zu Bitmaps finden Sie unter [Bitmaps](bitmaps.md).
