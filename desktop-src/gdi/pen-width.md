# <a name="pen-width"></a>Stift Breite

Das width-Attribut gibt die Breite des kosmetischen Stifts in Geräte Einheiten an. Bei Verwendung mit einem geometrischen Stift wird jedoch die Breite des Stifts in logischen Einheiten angegeben. Weitere Informationen zu Geräte Einheiten finden Sie unter [Koordinaten Bereiche und Transformationen](coordinate-spaces-and-transformations.md).

Derzeit beschränkt das System die Breite von kosmetischen Stiften auf ein einzelnes Pixel. Diese Einschränkung kann jedoch in zukünftigen Versionen entfernt werden.
