# <a name="pen-width"></a>Stiftbreite

Das Width-Attribut gibt eine breite des Stifts in Geräteeinheiten an. Bei Verwendung mit einem geometrischen Stift wird jedoch die Breite des Stifts in logischen Einheiten angegeben. Weitere Informationen zu Geräteeinheiten finden Sie unter [Koordinatenräume und Transformationen](coordinate-spaces-and-transformations.md).

Derzeit beschränkt das System die Breite der Stifte auf ein einzelnes Pixel. In zukünftigen Versionen wird diese Einschränkung jedoch möglicherweise entfernt.
