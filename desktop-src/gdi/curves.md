# <a name="curves"></a><span data-ttu-id="a2356-103">Kurven</span><span class="sxs-lookup"><span data-stu-id="a2356-103">Curves</span></span>

<span data-ttu-id="a2356-104">Eine reguläre Kurve ist ein Satz hervorgehobener Pixel in einer Raster Anzeige (oder Punkte auf einer gedruckten Seite), die den Umkreis (oder einen Teil des Umkreis Bereichs) eines gleich enden Abschnitts definieren.</span><span class="sxs-lookup"><span data-stu-id="a2356-104">A regular curve is a set of highlighted pixels on a raster display (or dots on a printed page) that define the perimeter (or part of the perimeter) of a conic section.</span></span> <span data-ttu-id="a2356-105">Eine unregelmäßige Kurve ist ein Satz von Pixeln, die eine Kurve definieren, die nicht in den Umkreis eines gleich enden Abschnitts passt.</span><span class="sxs-lookup"><span data-stu-id="a2356-105">An irregular curve is a set of pixels that define a curve that does not fit the perimeter of a conic section.</span></span> <span data-ttu-id="a2356-106">Der Endpunkt wird von einer Kurve ausgeschlossen, so wie er von einer Zeile ausgeschlossen wird.</span><span class="sxs-lookup"><span data-stu-id="a2356-106">The ending point is excluded from a curve just as it is excluded from a line.</span></span>

<span data-ttu-id="a2356-107">Wenn eine Anwendung eine der Kurven Zeichnungsfunktionen aufruft, unterbricht GDI die Kurve in eine Reihe von extrem kleinen, diskreten Liniensegmenten.</span><span class="sxs-lookup"><span data-stu-id="a2356-107">When an application calls one of the curve-drawing functions, GDI breaks the curve into a number of extremely small, discrete line segments.</span></span> <span data-ttu-id="a2356-108">Nach dem Bestimmen der Endpunkte (Startpunkt und Endpunkt) für jedes dieser Zeilen Segmente bestimmt GDI, welche Pixel (oder Punkte) die einzelnen Zeilen durch Anwenden des zugehörigen DDA definieren.</span><span class="sxs-lookup"><span data-stu-id="a2356-108">After determining the endpoints (starting point and ending point) for each of these line segments, GDI determines which pixels (or dots) define each line by applying its DDA.</span></span>

<span data-ttu-id="a2356-109">Eine Anwendung kann eine Ellipse oder einen Teil einer Ellipse durch Aufrufen der [**Arc**](/windows/desktop/api/Wingdi/nf-wingdi-arc) -Funktion zeichnen.</span><span class="sxs-lookup"><span data-stu-id="a2356-109">An application can draw an ellipse or part of an ellipse by calling the [**Arc**](/windows/desktop/api/Wingdi/nf-wingdi-arc) function.</span></span> <span data-ttu-id="a2356-110">Diese Funktion zeichnet die Kurve innerhalb des Umkreis Bereichs eines unsichtbaren Rechtecks, das als umschließendes Rechteck bezeichnet wird.</span><span class="sxs-lookup"><span data-stu-id="a2356-110">This function draws the curve within the perimeter of an invisible rectangle called a bounding rectangle.</span></span> <span data-ttu-id="a2356-111">Die Größe der Ellipse wird durch zwei unsichtbare radiale angegeben, die von der Mitte des Rechtecks auf die Seiten des Rechtecks ausgedehnt werden.</span><span class="sxs-lookup"><span data-stu-id="a2356-111">The size of the ellipse is specified by two invisible radials extending from the center of the rectangle to the sides of the rectangle.</span></span> <span data-ttu-id="a2356-112">Die folgende Abbildung zeigt einen Bogen (Teil einer Ellipse), der mithilfe der **Bogen** Funktion gezeichnet wurde.</span><span class="sxs-lookup"><span data-stu-id="a2356-112">The following illustration shows an arc (part of an ellipse) drawn by using the **Arc** function.</span></span>

<span data-ttu-id="a2356-114">Beim Aufrufen der [**Arc**](/windows/desktop/api/Wingdi/nf-wingdi-arc) -Funktion gibt eine Anwendung die Koordinaten des umgebenden Rechtecks und der radiale an.</span><span class="sxs-lookup"><span data-stu-id="a2356-114">When calling the [**Arc**](/windows/desktop/api/Wingdi/nf-wingdi-arc) function, an application specifies the coordinates of the bounding rectangle and radials.</span></span> <span data-ttu-id="a2356-115">In der vorangehenden Abbildung werden das Rechteck und die radiale mit gestrichelten Linien dargestellt, während der eigentliche Bogen mithilfe einer durchgezogenen Linie gezeichnet wurde.</span><span class="sxs-lookup"><span data-stu-id="a2356-115">The preceding illustration shows the rectangle and radials with dashed lines while the actual arc was drawn using a solid line.</span></span>
