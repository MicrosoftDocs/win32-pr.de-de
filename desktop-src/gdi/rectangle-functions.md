# <a name="rectangle-functions"></a>Rechteckfunktionen

Die folgenden Funktionen werden mit Rechtecke verwendet.



| Funktion                               | BESCHREIBUNG                                                                                                                                   |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [**CopyRect**](/windows/desktop/api/Winuser/nf-winuser-copyrect)           | Kopiert die Koordinaten eines Rechtecks in ein anderes.                                                                                           |
| [**EqualRect**](/windows/desktop/api/Winuser/nf-winuser-equalrect)         | Bestimmt, ob die beiden angegebenen Rechtecke gleich sind, indem die Koordinaten ihrer oberen linken und unteren rechten Ecken verglichen werden.           |
| [**InflateRect**](/windows/desktop/api/Winuser/nf-winuser-inflaterect)     | Erhöht oder verringert die Breite und Höhe des angegebenen Rechtecks.                                                                       |
| [**IntersectRect**](/windows/desktop/api/Winuser/nf-winuser-intersectrect) | Berechnet die Schnittmenge zweier Quellrechtecke und platziert die Koordinaten des Schnittpunktrechtecks im Zielrechteck. |
| [**IsRectEmpty**](/windows/desktop/api/Winuser/nf-winuser-isrectempty)     | Bestimmt, ob das angegebene Rechteck leer ist.                                                                                          |
| [**OffsetRect**](/windows/desktop/api/Winuser/nf-winuser-offsetrect)       | Verschiebt das angegebene Rechteck um die angegebenen Offsets.                                                                                       |
| [**PtInRect**](/windows/desktop/api/Winuser/nf-winuser-ptinrect)           | Bestimmt, ob sich der angegebene Punkt innerhalb des angegebenen Rechtecks befindet.                                                                   |
| [**SetRect**](/windows/desktop/api/Winuser/nf-winuser-setrect)             | Legt die Koordinaten des angegebenen Rechtecks fest.                                                                                              |
| [**SetRectEmpty**](/windows/desktop/api/Winuser/nf-winuser-setrectempty)   | Erstellt ein leeres Rechteck, in dem alle Koordinaten auf 0 (null) festgelegt sind.                                                                          |
| [**SubtractRect**](/windows/desktop/api/Winuser/nf-winuser-subtractrect)   | Bestimmt die Koordinaten eines Rechtecks, das durch Subtrahieren eines Rechtecks von einem anderen rechteckig ist.                                                   |
| [**UnionRect**](/windows/desktop/api/Winuser/nf-winuser-unionrect)         | Erstellt die Vereinigung zweier Rechtecke.                                                                                                          |
