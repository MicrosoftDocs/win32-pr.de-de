# <a name="rectangle-functions"></a>Rechteck Funktionen

Die folgenden Funktionen werden mit Rechtecke verwendet.



| Funktion                               | BESCHREIBUNG                                                                                                                                   |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [**Copyrect**](/windows/desktop/api/Winuser/nf-winuser-copyrect)           | Kopiert die Koordinaten eines Rechtecks in ein anderes.                                                                                           |
| [**Equalrect**](/windows/desktop/api/Winuser/nf-winuser-equalrect)         | Bestimmt, ob die beiden angegebenen Rechtecke gleich sind, indem die Koordinaten ihrer oberen linken und unteren rechten Ecke verglichen werden.           |
| [**Inflaterect**](/windows/desktop/api/Winuser/nf-winuser-inflaterect)     | Erhöht oder verringert die Breite und Höhe des angegebenen Rechtecks.                                                                       |
| [**IntersectRect**](/windows/desktop/api/Winuser/nf-winuser-intersectrect) | Berechnet die Schnittmenge zweier Quell Rechtecke und platziert die Koordinaten des Schnittstellen Rechtecks in das Ziel Rechteck. |
| [**Isrectempty**](/windows/desktop/api/Winuser/nf-winuser-isrectempty)     | Bestimmt, ob das angegebene Rechteck leer ist.                                                                                          |
| [**Offsetrect**](/windows/desktop/api/Winuser/nf-winuser-offsetrect)       | Verschiebt das angegebene Rechteck um die angegebenen Offsets.                                                                                       |
| [**PtInRect**](/windows/desktop/api/Winuser/nf-winuser-ptinrect)           | Bestimmt, ob der angegebene Punkt innerhalb des angegebenen Rechtecks liegt.                                                                   |
| [**SetRect**](/windows/desktop/api/Winuser/nf-winuser-setrect)             | Legt die Koordinaten des angegebenen Rechtecks fest.                                                                                              |
| [**Setrectempty**](/windows/desktop/api/Winuser/nf-winuser-setrectempty)   | Erstellt ein leeres Rechteck, in dem alle Koordinaten auf 0 (null) festgelegt sind.                                                                          |
| [**Subtractrect**](/windows/desktop/api/Winuser/nf-winuser-subtractrect)   | Bestimmt die Koordinaten eines Rechtecks, das durch Subtraktion eines Rechtecks von einem anderen formatiert wird.                                                   |
| [**Unionrect**](/windows/desktop/api/Winuser/nf-winuser-unionrect)         | Erstellt die Union von zwei Rechtecke.                                                                                                          |
