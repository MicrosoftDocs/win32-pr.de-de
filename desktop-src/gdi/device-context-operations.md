# <a name="device-context-operations"></a>Gerätekontext Vorgänge

Eine Anwendung kann die folgenden Vorgänge für einen Gerätekontext ausführen:

-   Listet vorhandene Grafik Objekte auf.
-   Wählen Sie neue Grafik Objekte aus.
-   Löschen Sie vorhandene Grafik Objekte.
-   Speichern Sie die aktuellen Grafik Objekte, ihre Attribute und die Grafikmodi.
-   Stellen Sie zuvor gespeicherte Grafik Objekte, ihre Attribute und die Grafikmodi wieder her.

Außerdem kann eine Anwendung einen Gerätekontext für Folgendes verwenden:

-   Bestimmen Sie, wie die Grafikausgabe übersetzt wird.
-   Beenden Sie lange Zeichnungsvorgänge (begonnen von einem Thread in einer Multithreadanwendung).
-   Setzt einen Drucker auf einen bestimmten Status zurück.
