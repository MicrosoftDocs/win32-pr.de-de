# <a name="device-context-operations"></a>Gerätekontextvorgänge

Eine Anwendung kann die folgenden Vorgänge in einem Gerätekontext ausführen:

-   Aufzählen vorhandener Grafikobjekte.
-   Wählen Sie neue Grafikobjekte aus.
-   Löschen vorhandener Grafikobjekte.
-   Speichern Sie die aktuellen Grafikobjekte, ihre Attribute und die Grafikmodi.
-   Stellen Sie zuvor gespeicherte Grafikobjekte, deren Attribute und die Grafikmodi wieder her.

Darüber hinaus kann eine Anwendung einen Gerätekontext verwenden, um:

-   Bestimmen Sie, wie die Grafikausgabe übersetzt wird.
-   Brechen Sie lange Zeichnungsvorgänge ab (gestartet von einem Thread in einer Multithreadanwendung).
-   Setzen Sie einen Drucker auf einen bestimmten Zustand zurück.
