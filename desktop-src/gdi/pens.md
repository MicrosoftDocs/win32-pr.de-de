# <a name="pens"></a>Stifte

Ein Stift ist ein Grafik Tool, das von einer Anwendung zum Zeichnen von Linien und Kurven verwendet werden kann. Zeichen Anwendungen verwenden Stifte zum Zeichnen von Zeilen Freihand, geraden Linien und Kurven. Anwendungen mit Computer gest√ºtztem Design (CAD) verwenden Stifte zum Zeichnen von sichtbaren Linien, ausgeblendeten Linien, Abschnitts Linien, Mittelpunkten usw. Word-Verarbeitung und Desktop Publishing Anwendungen verwenden Stifte zum Zeichnen von Rahmen und Regeln. Arbeitsblatt Anwendungen verwenden Stifte zum Bestimmen von Trends in Diagrammen und zum Gliedern von Balkendiagrammen und Kreis Diagrammen.
