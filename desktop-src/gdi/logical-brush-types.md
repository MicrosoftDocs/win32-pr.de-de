# <a name="logical-brush-types"></a>Logische Pinseltypen

Es gibt vier Arten von logischen Pinseln: [Voll,](solid-brush.md) [Lager,](stock-brush.md) [Schraffur](hatch-brush.md)und [Muster](pattern-brush.md). Diese Pinsel werden in der folgenden Abbildung dargestellt.

![Abbildung mit vier Kreisen, die jeweils mit einem anderen Pinseltyp gefüllt sind: Voll, Stock, Schraffur und Muster](images/csbru-02.png)

Die Stock- und Schraffurtypen verfügen jeweils über mehrere vordefinierte Pinsel.

Die [**CreateBrushIndirect-Funktion**](/windows/desktop/api/Wingdi/nf-wingdi-createbrushindirect) erstellt einen logischen Pinsel mit einem angegebenen Stil, einer angegebenen Farbe und einem angegebenen Muster.
