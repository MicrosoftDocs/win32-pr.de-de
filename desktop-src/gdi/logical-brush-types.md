# <a name="logical-brush-types"></a>Logische Pinseltypen

Es gibt vier Arten von logischen Pinseln: [Solid](solid-brush.md), [Stock](stock-brush.md), [Schraffur](hatch-brush.md)und [Pattern](pattern-brush.md). Diese Pinsel sind in der folgenden Abbildung dargestellt.

![Abbildung, die vier Kreise anzeigt, die jeweils von einem anderen Pintyp ausgefüllt sind: Solid, Stock, Schraffur und Pattern](images/csbru-02.png)

Die Stock-und Schraffurtypen verfügen jeweils über mehrere vordefinierte Pinsel.

Die Funktion " [**interatebrushindirekte**](/windows/desktop/api/Wingdi/nf-wingdi-createbrushindirect) " erstellt einen logischen Pinsel mit einem angegebenen Stil, einer angegebenen Farbe und einem angegebenen Muster.
