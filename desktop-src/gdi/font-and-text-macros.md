# <a name="font-and-text-macros"></a>Schriftart und Text Makros

Die folgenden Makros werden mit Schriftarten und Text verwendet.



| Makro                            | Beschreibung                                            |
|----------------------------------|--------------------------------------------------------|
| [**Deletefont**](/windows/desktop/api/Windowsx/nf-windowsx-deletefont) | Löscht eine angegebene Schriftart.                              |
| [**Selectfont**](/windows/desktop/api/Windowsx/nf-windowsx-selectfont) | Wählt eine Schriftart im angegebenen Gerätekontext (DC) aus. |
