# <a name="regions-windows-gdi"></a>Regionen (Windows-GDI)

Ein *Bereich* ist ein Rechteck, ein Polygon oder eine Ellipse (oder eine Kombination aus mindestens zwei Formen), die ausgefüllt, gezeichnet, invertiert, gerahmt und zum Ausführen von Treffer Tests (Testen der Cursorposition) verwendet werden können.
