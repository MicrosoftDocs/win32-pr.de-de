# <a name="regions-windows-gdi"></a>Regionen (Windows GDI)

Ein  Bereich ist ein Rechteck, ein Polygon oder eine Ellipse (oder eine Kombination aus zwei oder mehr dieser Formen), die gefüllt, gestrichen, umgekehrt, umrahmend und verwendet werden können, um Treffertests (Tests für die Cursorposition) durchzuführen.

-   [Verwenden von Bereichen](using-regions.md)
