# <a name="moving-regions"></a>Verschieben von Regionen

Eine Anwendung verschiebt einen Bereich durch Aufrufen der [**offsetrgn**](/windows/desktop/api/Wingdi/nf-wingdi-offsetrgn) -Funktion. Die angegebenen Offsets entlang der x-und der y-Achse bestimmen die Anzahl der logischen Einheiten, die nach links oder rechts bzw. nach oben oder unten verschoben werden.
