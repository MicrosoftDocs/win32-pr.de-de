# <a name="raster-operation-codes"></a>Raster-Vorgangscodes

Rastervorgangcodes definieren, wie die Grafikgeräteschnittstelle (GDI) die Bits des ausgewählten Stifts mit den Bits in der Zielbitmap kombiniert.

In dieser Übersicht werden die binären und ternären Rastervorgänge aufgeführt und beschrieben, die von GDI verwendet werden. Ein [binärer Rastervorgang](binary-raster-operations.md) umfasst zwei Operanden: einen Stift und eine Zielbitmap. Ein [ternärer Rastervorgang](ternary-raster-operations.md) umfasst drei Operanden: eine Quellbitmap, einen Pinsel und eine Zielbitmap. Sowohl binäre als auch ternäre Rastervorgänge verwenden boolesche Operatoren.
