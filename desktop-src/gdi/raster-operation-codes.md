# <a name="raster-operation-codes"></a>Raster Vorgangs Codes

Mithilfe von Raster-Vorgangs Codes wird definiert, wie die Grafikgeräte Schnittstelle die Bits aus dem ausgewählten Stift mit den Bits in der Ziel Bitmap kombiniert.

In dieser Übersicht werden die binären und ternären Raster Vorgänge aufgeführt und beschrieben, die von GDI verwendet werden. Ein [binärer Raster Vorgang](binary-raster-operations.md) umfasst zwei Operanden: einen Stift und eine Ziel Bitmap. Ein [ternärer Raster Vorgang](ternary-raster-operations.md) umfasst drei Operanden: eine Quell Bitmap, einen Pinsel und eine Ziel Bitmap. Sowohl binäre als auch ternäre Raster Vorgänge verwenden boolesche Operatoren.
