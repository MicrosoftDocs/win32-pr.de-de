# <a name="paths-windows-gdi"></a>Pfade (Windows-GDI)

Bei einem *Pfad* handelt es sich um eine oder mehrere Abbildungen (oder Formen), die ausgefüllt, beschrieben oder ausgefüllt und dargestellt werden. Pfade werden zum Zeichnen und Zeichnen von Anwendungen verwendet. Anwendungen, die vom Computer unterstützt werden, verwenden Pfade, um eindeutige Clippingbereiche zu erstellen, um Kontur von unregelmäßigen Formen zu zeichnen und um das Innere von unregelmäßigen Formen zu füllen. Eine unregelmäßige Form ist eine Form, die aus Bézier-Kurven und geraden Linien besteht. (Reguläre Formen sind Ellipsen, Kreise, Rechtecke und Polygone.)
