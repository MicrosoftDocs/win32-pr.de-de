# <a name="pen-attributes"></a>Stift Attribute

Es gibt sieben Stift Attribute, die den Typ des Stifts und seine Merkmale definieren: Breite, Stil, Farbe, Muster, Schraffur, Endstil und joinstil. Sowohl kosmetische als auch geometrische Stifte verfügen über die Attribute "width", "Style" und "Color". Nur geometrische Stifte verfügen über die Attribute Muster, Schraffur, Endstil und joinstil. Das Muster und optionale schraffurattribute sind normalerweise mit einem Pinsel verknüpft, können aber auch mit geometrischen stiften verwendet werden.

Weitere Informationen finden Sie unter den folgenden Themen:

-   [Stift Stil](pen-style.md)
-   [Stift Farbe](pen-color.md)
-   [Stift Muster](pen-pattern.md)
