# <a name="about-coordinate-spaces-and-transformations"></a>Informationen zu Koordinaten Räumen und Transformationen

Koordinaten Bereiche und Transformationen werden von den folgenden Anwendungs Typen verwendet:

-   Anwendungen zur Desktop Veröffentlichung (um Teile einer Seite zu vergrößern oder angrenzende Seiten in einem Fenster anzuzeigen).
-   Computer gestützte Design-(CAD) Anwendungen (zum Drehen von Objekten, Skalieren von Zeichnungen oder zum Erstellen von Perspektiven Ansichten).
-   Tabellenkalkulationsanwendungen (zum Verschieben und Anpassen von Diagrammen).
