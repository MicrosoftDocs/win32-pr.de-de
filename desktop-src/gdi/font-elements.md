# <a name="font-elements"></a>Schriftart Elemente

Eine *Schriftart* ist eine Auflistung von Zeichen und Symbolen, die einen gemeinsamen Entwurf aufweisen. Die drei Hauptelemente dieses Entwurfs werden als Schriftart, Stil und Größe bezeichnet.

## <a name="typeface"></a>Schriftart

Der Begriff Schriftart bezieht sich auf bestimmte Merkmale von Zeichen und Symbolen in der Schriftart, z. b. die Breite der dichten und dünnen Striche, die die Zeichen und das vorhanden sein oder Nichtvorhandensein von serifs bilden. Ein Serif ist die kurze Kreuz Linie an den Enden eines nicht verbundenen Strichs. Eine Schriftart oder Schriftart ohne Serifen an wird in der Regel als "Sans-Serif"-Schriftart bezeichnet.

## <a name="style"></a>Stil

Der Begriffs Stil bezieht sich auf die Gewichtung und die Rechts einer Schriftart. Schriftart Gewichtungen können von schlank zu schwarz reichen. Im folgenden finden Sie eine Liste möglicher Gewichtungen für Schriftarten (von der leichtesten bis zum stärksten):

<dl> Thin  
ExtraLight  
Leicht  
Normal  
Medium (Mittel)  
Halbfett  
Fett  
ExtraBold  
Schwer  
</dl>

Drei Begriffe kategorisieren den Rechts einer Schriftart: Römer, schrägen und kursiv.

Die Zeichen in einer römischen Schriftart sind aufsteigenden Zeichen. Die Zeichen in einer schrägen Schriftart sind künstlich schlank. Der Rechts wird erreicht, indem eine scherungs Transformation für die Zeichen aus einer römischen Schriftart durchgeführt wird. Die Zeichen in einer kursiv formatierten Schriftart sind tatsächlich schlank und werden angezeigt, sobald Sie entworfen wurden. Weitere Informationen zum Scheren finden Sie unter [Koordinaten Bereiche und Transformationen](coordinate-spaces-and-transformations.md).

## <a name="size"></a>Size

Der *Schrift* Grad ist ein unpräziser Wert. Der Wert kann im Allgemeinen bestimmt werden, indem der Abstand zwischen dem unteren Rand eines Kleinbuchstaben g und dem oberen Rand eines angrenzenden Großbuchstaben M gemessen wird, wie in der folgenden Abbildung dargestellt.

![Abbildung mit einem Kleinbuchstaben g und einem Großbuchstaben m](images/csftx-01.png)

Die Größe einer Schriftart wird in Einheiten angegeben, die als Punkte bezeichnet werden. Ein Punkt ist. 013837 Zoll. Im Anschluss an das von Pierre Simon Fournier entwickelte Punkt System ist es üblich, einen Punkt als 1/72 Zoll zu nähern.
