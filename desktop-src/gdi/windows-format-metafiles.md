# <a name="windows-format-metafiles"></a>Metadatendateien Windows-Format

Metadatendateien im Microsoft Windows-Format sind in ihren Funktionen eingeschränkt und sollten nur selten verwendet werden. Die Windows-Formatfunktionen werden unterstützt, um die Abwärtskompatibilität mit Anwendungen zu gewährleisten, die als Windows-basierte 16-Bit-Anwendungen geschrieben wurden. Stattdessen sollten Sie die Funktionen des erweiterten Formats verwenden.

Eine *Metadatei im Windows-Format* wird von Windows-basierten 16-Bit-Anwendungen verwendet. Das Format besteht aus einem Header und einem Array von Metadatei-Datensätzen.

Im folgenden sind die Einschränkungen dieses Formats aufgeführt:

-   Eine Metadatei im Windows-Format ist abhängig von der Anwendung und dem Gerät. Änderungen der zuordnungsmodi der Anwendung oder der Geräte Auflösung beeinflussen die Darstellung von Metadateien, die in diesem Format erstellt werden.
-   Eine Metadatei im Windows-Format enthält keinen umfassenden Header, der die ursprünglichen Bild Dimensionen, die Auflösung des Geräts, auf dem das Bild erstellt wurde, eine optionale Textbeschreibung oder eine optionale Palette beschreibt.
-   Eine Metadatei im Windows-Format unterstützt die neuen Kurven-, Pfad-und Transformations Funktionen nicht. Weitere Informationen finden Sie in der Liste der unterstützten Funktionen in der folgenden Tabelle.
-   Einige Metadatei-Datensätze im Windows-Format können nicht skaliert werden.
-   Der Metadatei-Gerätekontext, der einer Metadatei im Windows-Format zugeordnet ist, kann nicht abgefragt werden (d. h., eine Anwendung kann keine Geräte Auflösungs Daten, Schriftart Metriken usw. Abrufen).

Um eine Metadatei im Windows-Format in eine Metadatei mit erweitertem Format zu konvertieren, rufen Sie die [**GetMetaFileBitsEx**](/windows/desktop/api/Wingdi/nf-wingdi-getmetafilebitsex) -Funktion auf, um die Daten aus der Metadatei im Windows-Format abzurufen, und rufen Sie dann die [**setwinmetafilebits**](/windows/desktop/api/Wingdi/nf-wingdi-setwinmetafilebits) -Funktion auf, um diese Daten in eine Datei mit erweitertem Format zu konvertieren. Um einen erweiterten Format Daten Satz in einen Windows-Format-Datensatz zu konvertieren, müssen Sie die [**getwinmetafilebits**](/windows/desktop/api/Wingdi/nf-wingdi-getwinmetafilebits) -Funktion aufrufen.
