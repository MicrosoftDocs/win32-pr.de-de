# <a name="windows-format-metafiles"></a>Windows-Format Metadateien

Metadateien im Microsoft Windows-Format sind in ihren Funktionen eingeschränkt und sollten nur selten verwendet werden. Die Windows-Formatfunktionen werden unterstützt, um die Abwärtskompatibilität mit Anwendungen aufrechtzuerhalten, die als 16-Bit-Windows-basierte Anwendungen ausgeführt wurden. Stattdessen sollten Sie die Funktionen im erweiterten Format verwenden.

Eine Metadatei im *Windows-Format* wird von 16-Bit-Windows-basierten Anwendungen verwendet. Das Format besteht aus einem Header und einem Array von Metadateidatensätzen.

Folgende Einschränkungen gelten für dieses Format:

-   Eine Metadatei im Windows-Format ist anwendungs- und geräteabhängig. Änderungen in den Zuordnungsmodi der Anwendung oder die Geräteauflösung wirken sich auf die Darstellung von Metadateien aus, die in diesem Format erstellt werden.
-   Eine Metadatei im Windows-Format enthält keinen umfassenden Header, der die ursprünglichen Bilddimensionen, die Auflösung des Geräts, auf dem das Bild erstellt wurde, eine optionale Textbeschreibung oder eine optionale Palette beschreibt.
-   Eine Metadatei im Windows-Format unterstützt die neuen Kurven-, Pfad- und Transformationsfunktionen nicht. Sehen Sie sich die Liste der unterstützten Funktionen in der folgenden Tabelle an.
-   Einige Metadateidatensätze im Windows-Format können nicht skaliert werden.
-   Der metafile-Gerätekontext, der einer Metadatei im Windows-Format zugeordnet ist, kann nicht abgefragt werden (d. b. eine Anwendung kann keine Geräteauflösungsdaten, Schriftartmetriken usw.) abrufen.

Im Folgenden sind die einzigen Funktionen, die von metafiles im Windows-Format unterstützt werden.

Um eine Metadatei im Windows-Format in eine Metadatei im erweiterten Format zu konvertieren, rufen Sie die [**GetMetaFileBitsEx-Funktion**](/windows/desktop/api/Wingdi/nf-wingdi-getmetafilebitsex) auf, um die Daten aus der Metadatei im Windows format abzurufen, und rufen Sie dann die [**SetWinMetaFileBits-Funktion**](/windows/desktop/api/Wingdi/nf-wingdi-setwinmetafilebits) auf, um diese Daten in eine Metadatei im erweiterten Format zu konvertieren. Um einen Datensatz im erweiterten Format in einen Windows-Formatdatensatz zu konvertieren, rufen Sie die [**GetWinMetaFileBits-Funktion**](/windows/desktop/api/Wingdi/nf-wingdi-getwinmetafilebits) auf.
