# <a name="pen-end-cap"></a>Stift Ende Obergrenze

Das End-Cap-Attribut gibt die Form eines geometrischen Stifts an: Round, Square oder Flat. Die folgende Abbildung zeigt parallele Linien, die mit jedem Typ von EndCap gezeichnet werden.

![Abbildung mit drei horizontalen Linien, von denen jede eine andere Art von EndCap hat](images/cspen-04.png)

Die Runden und quadratischen Endpunkte werden nach den Anfangs-und Endpunkten einer mit einem geometrischen Stift gezeichneten Linie erweitert. die flache Obergrenze ist nicht.
