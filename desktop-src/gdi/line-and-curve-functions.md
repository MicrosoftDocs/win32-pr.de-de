# <a name="line-and-curve-functions"></a>Linien-und Kurven Funktionen

Die folgenden Funktionen werden bei Linien und Kurven verwendet.



| Funktion                                   | BESCHREIBUNG                                                                                                   |
|--------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**Anglearc**](/windows/desktop/api/Wingdi/nf-wingdi-anglearc)               | Zeichnet ein Liniensegment und einen Bogen.                                                                              |
| [**Arc**](/windows/desktop/api/Wingdi/nf-wingdi-arc)                         | Zeichnet einen elliptischen Bogen.                                                                                      |
| [**ArcTo**](/windows/desktop/api/Wingdi/nf-wingdi-arcto)                     | Zeichnet einen elliptischen Bogen.                                                                                      |
| [**Getarcdirection**](/windows/desktop/api/Wingdi/nf-wingdi-getarcdirection) | Ruft die aktuelle Bogen Richtung für den angegebenen Gerätekontext ab.                                         |
| [**LineDDA**](/windows/desktop/api/Wingdi/nf-wingdi-linedda)                 | Bestimmt, welche Pixel für eine Linie hervorgehoben werden sollen, die durch die angegebenen Start-und Endpunkte definiert ist. |
| [**Lineddaproc**](/windows/desktop/api/Wingdi/nc-wingdi-lineddaproc)         | Eine von der Anwendung definierte Rückruffunktion, die mit der LineDDA-Funktion verwendet wird.                                      |
| [**LineTo**](/windows/desktop/api/Wingdi/nf-wingdi-lineto)                   | Zeichnet eine Linie von der aktuellen Position bis zum angegebenen Punkt, aber nicht einschließlich.                         |
| [**"Muvezu Ex"**](/windows/desktop/api/Wingdi/nf-wingdi-movetoex)               | Aktualisiert die aktuelle Position auf den angegebenen Punkt und gibt optional die vorherige Position zurück.             |
| [**Polybezier**](/windows/desktop/api/Wingdi/nf-wingdi-polybezier)           | Zeichnet eine oder mehrere Bézier-Kurven.                                                                              |
| [**PolyBezierTo**](/windows/desktop/api/Wingdi/nf-wingdi-polybezierto)       | Zeichnet eine oder mehrere Bézier-Kurven.                                                                              |
| [**Polydraw**](/windows/desktop/api/Wingdi/nf-wingdi-polydraw)               | Zeichnet eine Reihe von Liniensegmenten und Bézier-Kurven.                                                               |
| [**Polylinie**](/windows/desktop/api/Wingdi/nf-wingdi-polyline)               | Zeichnet eine Reihe von Liniensegmenten, indem die Punkte im angegebenen Array verbunden werden.                              |
| [**PolyLineTo**](/windows/desktop/api/Wingdi/nf-wingdi-polylineto)           | Zeichnet eine oder mehrere gerade Linien.                                                                             |
| [**Polypolyline**](/windows/desktop/api/Wingdi/nf-wingdi-polypolyline)       | Zeichnet mehrere Reihe verbundener Liniensegmente.                                                             |
| [**"Tartarcdirection"**](/windows/desktop/api/Wingdi/nf-wingdi-setarcdirection) | Legt die Zeichnungs Richtung fest, die für Bogen-und Rechteck Funktionen verwendet werden soll.                                        |
