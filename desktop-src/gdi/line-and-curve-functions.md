# <a name="line-and-curve-functions"></a>Linien- und Kurvenfunktionen

Die folgenden Funktionen werden mit Linien und Kurven verwendet.



| Funktion                                   | BESCHREIBUNG                                                                                                   |
|--------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**AngleArc**](/windows/desktop/api/Wingdi/nf-wingdi-anglearc)               | Zeichnet ein Liniensegment und einen Bogen.                                                                              |
| [**Arc**](/windows/desktop/api/Wingdi/nf-wingdi-arc)                         | Zeichnet einen elliptischen Bogen.                                                                                      |
| [**ArcTo**](/windows/desktop/api/Wingdi/nf-wingdi-arcto)                     | Zeichnet einen elliptischen Bogen.                                                                                      |
| [**GetArcDirection**](/windows/desktop/api/Wingdi/nf-wingdi-getarcdirection) | Ruft die aktuelle Bogenrichtung für den angegebenen Gerätekontext ab.                                         |
| [**LineDDA**](/windows/desktop/api/Wingdi/nf-wingdi-linedda)                 | Bestimmt, welche Pixel für eine Zeile hervorgehoben werden sollen, die durch die angegebenen Start- und Endpunkte definiert wird. |
| [**LineDDAProc**](/windows/desktop/api/Wingdi/nc-wingdi-lineddaproc)         | Eine anwendungsdefinierte Rückruffunktion, die mit der LineDDA-Funktion verwendet wird.                                      |
| [**Lineto**](/windows/desktop/api/Wingdi/nf-wingdi-lineto)                   | Zeichnet eine Linie von der aktuellen Position bis zum angegebenen Punkt, aber nicht einschließlich.                         |
| [**MoveToEx**](/windows/desktop/api/Wingdi/nf-wingdi-movetoex)               | Aktualisiert die aktuelle Position auf den angegebenen Punkt und gibt optional die vorherige Position zurück.             |
| [**PolyBezier**](/windows/desktop/api/Wingdi/nf-wingdi-polybezier)           | Zeichnet eine oder mehrere Bézierkurven.                                                                              |
| [**PolyBezierTo**](/windows/desktop/api/Wingdi/nf-wingdi-polybezierto)       | Zeichnet eine oder mehrere Bézierkurven.                                                                              |
| [**PolyDraw**](/windows/desktop/api/Wingdi/nf-wingdi-polydraw)               | Zeichnet eine Reihe von Liniensegmenten und Bézierkurven.                                                               |
| [**Polylinie**](/windows/desktop/api/Wingdi/nf-wingdi-polyline)               | Zeichnet eine Reihe von Liniensegmenten, indem die Punkte im angegebenen Array verbunden werden.                              |
| [**PolylineTo**](/windows/desktop/api/Wingdi/nf-wingdi-polylineto)           | Zeichnet eine oder mehrere gerade Linien.                                                                             |
| [**PolyPolyline**](/windows/desktop/api/Wingdi/nf-wingdi-polypolyline)       | Zeichnet mehrere Reihen verbundener Liniensegmente.                                                             |
| [**SetArcDirection**](/windows/desktop/api/Wingdi/nf-wingdi-setarcdirection) | Legt die Zeichnungsrichtung fest, die für Arc- und Rechteckfunktionen verwendet werden soll.                                        |
