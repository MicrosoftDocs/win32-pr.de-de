# <a name="lines"></a><span data-ttu-id="b97cd-103">Linien</span><span class="sxs-lookup"><span data-stu-id="b97cd-103">Lines</span></span>

<span data-ttu-id="b97cd-104">Eine Linie ist ein Satz hervorgehobener Pixel in einem Raster Display (oder einer Gruppe von Punkten auf einer gedruckten Seite), die durch zwei Punkte identifiziert werden: ein Anfangspunkt und ein Endpunkt.</span><span class="sxs-lookup"><span data-stu-id="b97cd-104">A line is a set of highlighted pixels on a raster display (or a set of dots on a printed page) identified by two points: a starting point and an ending point.</span></span> <span data-ttu-id="b97cd-105">Das Pixel am Ausgangspunkt ist immer in der Zeile enthalten, und das Pixel am Endpunkt wird immer ausgeschlossen.</span><span class="sxs-lookup"><span data-stu-id="b97cd-105">The pixel located at the starting point is always included in the line, and the pixel located at the ending point is always excluded.</span></span> <span data-ttu-id="b97cd-106">(Diese Art von Zeile wird manchmal als inklusiv bezeichnet.)</span><span class="sxs-lookup"><span data-stu-id="b97cd-106">(This kind of line is sometimes called inclusive-exclusive.)</span></span>

<span data-ttu-id="b97cd-107">Wenn eine Anwendung eine der Zeilen Zeichnungsfunktionen aufruft (Graphics Device Interface, GDI), oder in einigen Fällen ein Gerätetreiber, bestimmt, welche Pixel
