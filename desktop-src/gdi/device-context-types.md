# <a name="device-context-types"></a>Gerätekontexttypen

Es gibt vier Arten von DCs: Anzeige, Drucker, Arbeitsspeicher (oder kompatibel) und Informationen. Jeder Typ erfüllt einen bestimmten Zweck, wie in der folgenden Tabelle beschrieben.



| Gerätekontext                                 | BESCHREIBUNG                                          |
|------------------------------------------------|------------------------------------------------------|
| [Anzeige](display-device-contexts.md)         | Unterstützt Zeichnungsvorgänge auf einer Videoanzeige.      |
| [Drucker](printer-device-contexts.md)         | Unterstützt Zeichnungsvorgänge für einen Drucker oder Plotter. |
| [Memory](memory-device-contexts.md)           | Unterstützt Zeichnungsvorgänge für eine Bitmap.             |
| [Information](information-device-contexts.md) | Unterstützt das Abrufen von Gerätedaten.               |
