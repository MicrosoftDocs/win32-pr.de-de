# <a name="device-context-types"></a>Gerätekontext Typen

Es gibt vier Typen von DCS: Anzeige, Drucker, Arbeitsspeicher (oder kompatibel) und Informationen. Jeder Typ dient einem bestimmten Zweck, wie in der folgenden Tabelle beschrieben.



| Gerätekontext                                 | BESCHREIBUNG                                          |
|------------------------------------------------|------------------------------------------------------|
| [Anzeige](display-device-contexts.md)         | Unterstützt Zeichnungsvorgänge in einer Videoanzeige.      |
| [Drucker](printer-device-contexts.md)         | Unterstützt Zeichnungsvorgänge für einen Drucker oder einen Plotter. |
| [Arbeitsspeicher](memory-device-contexts.md)           | Unterstützt Zeichnungsvorgänge in einer Bitmap.             |
| [Information](information-device-contexts.md) | Unterstützt das Abrufen von Gerätedaten.               |
