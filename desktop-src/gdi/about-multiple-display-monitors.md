# <a name="about-multiple-display-monitors"></a><span data-ttu-id="3fb4a-103">Informationen zu mehreren Anzeige Monitoren</span><span class="sxs-lookup"><span data-stu-id="3fb4a-103">About Multiple Display Monitors</span></span>

<span data-ttu-id="3fb4a-104">Wenn mehrere Monitore Teil des Desktops sind, können Objekte nahtlos zwischen Monitoren übertragen werden.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-104">When multiple monitors are part of the desktop, objects can travel seamlessly between monitors.</span></span> <span data-ttu-id="3fb4a-105">Das heißt, Sie können Fenster oder Verknüpfungen von einem Monitor auf einen anderen ziehen, und Sie können die Größe von Fenstern anpassen, um mehr als einen Monitor abzudecken.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-105">That is, you can drag windows or shortcuts from one monitor to another, and you can size windows to cover more than one monitor.</span></span> <span data-ttu-id="3fb4a-106">Wenn außerdem ein Monitor oberhalb eines anderen Monitors installiert wird, wird ein Cursor, der den unteren Rand des oberen Monitors verlässt, oben im unteren Monitor angezeigt.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-106">Also, if one monitor is installed above another, a cursor that leaves the bottom of the upper monitor appears at the top of the lower monitor.</span></span>

<span data-ttu-id="3fb4a-107">In der Regel ordnet ein Benutzer die Monitore im System an, um die Anordnung der physischen Anzeigeeinheiten widerzuspiegeln. z. b. nebeneinander oder eins auf der obersten Seite der anderen.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-107">Typically, a user arranges the monitors in the system to reflect the arrangement of the physical display units; for example, side-by-side or one-on-top-of-the-other.</span></span> <span data-ttu-id="3fb4a-108">Der Benutzer führt dies über die Registerkarte Monitore durch, die im Dialogfeld **Anzeigeeigenschaften** über die Systemsteuerung die Registerkarte **Einstellungen** ersetzt.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-108">The user does this with the Monitors tab, which replaces the **Settings** tab in the **Display Properties** dialog box through Control Panel.</span></span> <span data-ttu-id="3fb4a-109">Die Monitore müssen einen zusammenhängenden Bereich bilden, d. h. jeder Monitor berührt einen anderen Monitor auf mindestens einem Teil eines Edge.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-109">The monitors must form one contiguous region, that is, each monitor touches another monitor on at least part of one edge.</span></span>

<span data-ttu-id="3fb4a-110">Wenn ein Fenster verschoben oder seine Größe geändert wird, ist ein Teil der Beschriftung immer sichtbar, sodass der Benutzer das Fenster mit der Maus verschieben und die Größe ändern kann.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-110">When a window is moved or resized, some part of the caption is always visible so the user can move and resize the window using the mouse.</span></span> <span data-ttu-id="3fb4a-111">Die Cursor Bewegung ist auf den Bereich der Monitore beschränkt, sodass Sie immer sichtbar ist.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-111">Cursor movement is restricted to the area of the monitors, so it is always visible.</span></span> <span data-ttu-id="3fb4a-112">Shellsymbole befinden sich auf demselben Monitor wie die Taskleiste, und die Taskleiste kann sich auf einem beliebigen Monitor befinden. Weitere Informationen finden Sie unter über [Legungen zum Monitor für ältere Programme](multiple-monitor-considerations-for-older-programs.md).</span><span class="sxs-lookup"><span data-stu-id="3fb4a-112">Shell icons are positioned on the same monitor as the taskbar, and the taskbar can be on any monitor, see [Multiple Monitor Considerations for Older Programs](multiple-monitor-considerations-for-older-programs.md).</span></span>

<span data-ttu-id="3fb4a-113">Ein System mit mehreren Monitoren wirkt sich auf bestimmte Tastenkombinationen aus.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-113">A multiple monitor system affects certain key combinations.</span></span> <span data-ttu-id="3fb4a-114">Die Tastenkombination ALT + PRINTSCRN erstellt eine Momentaufnahme des Vordergrund Fensters, wie immer.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-114">The ALT+PRINTSCRN key combination takes a snapshot of the foreground window, as always.</span></span> <span data-ttu-id="3fb4a-115">Der PRINTSCRN-Schlüssel erstellt jedoch eine Momentaufnahme des Monitors mit der Maus.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-115">However, the PRINTSCRN key takes a snapshot of the monitor that has the mouse.</span></span> <span data-ttu-id="3fb4a-116">Die Tastenkombination STRG + PRINTSCRN erstellt eine Momentaufnahme des gesamten virtuellen Bildschirms. Weitere Informationen finden Sie auf [dem virtuellen Bildschirm](the-virtual-screen.md).</span><span class="sxs-lookup"><span data-stu-id="3fb4a-116">The CTRL+PRINTSCRN key combination takes a snapshot of the entire virtual screen, see [The Virtual Screen](the-virtual-screen.md).</span></span>

<span data-ttu-id="3fb4a-117">Die Unterstützung für mehrere Monitore wirkt sich nicht auf die Leistung von Anwendungen aus, wenn Sie in einer einzigen Anzeige Umgebung ausgeführt wird.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-117">The support for multiple monitors does not affect the performance of applications when running in a single display environment.</span></span> <span data-ttu-id="3fb4a-118">Dies bedeutet, dass bei der Ausführung auf einem einzelnen Anzeigesystem kein zusätzlicher mehr Aufwand im Hochleistungs-Grafik Vorgangs Code vorhanden ist.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-118">That is, when running on a single display system, no additional overhead will be present in the high-performance graphics operations code.</span></span> <span data-ttu-id="3fb4a-119">In einem System mit mehreren Monitoren ist die Leistung jedoch geringfügig beeinträchtigt, wenn eine Anwendung nur auf einem der Grafikgeräte ausgeführt wird.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-119">However, in a multiple monitor system, performance is slightly affected if an application runs only on one of the graphics devices.</span></span> <span data-ttu-id="3fb4a-120">Außerdem kann die Leistung erheblich beeinträchtigt werden, wenn eine Anwendung mehrere anzeigen umfasst, insbesondere für grafikintensive Vorgänge.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-120">Also, performance may be greatly affected if an application spans multiple displays, especially for graphics-intensive operations.</span></span>

<span data-ttu-id="3fb4a-121">Der *voll Bildschirm* ist eine vom Betriebssystem bereitgestellte Funktion, mit der ein Benutzer eine Anwendung in einen bestimmten Zustand umschalten kann, in dem die Anwendung direkt auf die VGA-Grafikhardware zugreifen kann.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-121">*Full-screen* is a feature provided by the operating system that allows a user to toggle an application into a special state where the application can access VGA graphics hardware directly.</span></span> <span data-ttu-id="3fb4a-122">Dies ist ein wichtiges Feature für Spiele und andere Grafik orientierte Anwendungen, die eine hohe Leistung erfordern.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-122">This is a key feature for games and other graphics-focused applications that require high performance.</span></span> <span data-ttu-id="3fb4a-123">Außerdem wird es häufig von Entwicklern für die Textbearbeitung verwendet, da es sehr schnelles Scrollen von Text ermöglicht.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-123">Also, it is often used by developers for text editing since it enables very fast text scrolling.</span></span>

<span data-ttu-id="3fb4a-124">In einer Umgebung mit mehreren Monitoren kann nur ein Grafikgerät mit VGA kompatibel sein.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-124">In a multiple monitor environment, only one graphics device can be VGA compatible.</span></span> <span data-ttu-id="3fb4a-125">Dies ist eine Einschränkung der Computer Hardware, die erfordert, dass nur ein Gerät auf Hardware Adressen antwortet.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-125">This is a limitation of computer hardware which requires that only one device respond to any hardware address.</span></span> <span data-ttu-id="3fb4a-126">Da der VGA-Hardware Kompatibilitäts Standard bestimmte Hardware Adressen erfordert, kann nur ein VGA-Grafikgerät auf einem Computer vorhanden sein, und nur dieses Gerät kann physisch auf VGA-Adressen reagieren.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-126">Because the VGA hardware compatibility standard requires specific hardware addresses, only one VGA graphics device can be present in a machine and only this device can physically respond to VGA addresses.</span></span> <span data-ttu-id="3fb4a-127">Anwendungen, die einen Vollbildmodus erfordern, werden daher nur auf einem bestimmten Gerät ausgeführt, das die VGA-Hardware Kompatibilität unterstützt.</span><span class="sxs-lookup"><span data-stu-id="3fb4a-127">Thus, applications that require going full screen will only run on the particular device that supports VGA hardware compatibility.</span></span>
