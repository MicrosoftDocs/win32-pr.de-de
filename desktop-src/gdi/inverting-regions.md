# <a name="inverting-regions"></a>Umkehren von Regionen

Eine Anwendung stellt die Farben in einem Bereich um, indem die [**InvertRgn-Funktion**](/windows/desktop/api/Wingdi/nf-wingdi-invertrgn) aufgerufen wird. Bei monofarbigen Anzeigen macht **InvertRgn** weiße Pixel schwarz und schwarze Pixel weiß. Auf Farbbildschirmen hängt diese Umkehrung von der Art der Technologie ab, die zum Generieren der Farben für den Bildschirm verwendet wird.
