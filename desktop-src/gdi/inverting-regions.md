# <a name="inverting-regions"></a>Umkehren von Regionen

Eine Anwendung kehrt durch Aufrufen der Funktion [**invertrgn**](/windows/desktop/api/Wingdi/nf-wingdi-invertrgn) die Farben um, die innerhalb eines Bereichs angezeigt werden. Bei monochrom-anzeigen werden durch **invertrgn** weiße Pixel schwarz und schwarz Pixel weiß. Auf Farb Schirmen hängt diese Inversion vom Typ der Technologie ab, die zum Generieren der Farben für den Bildschirm verwendet wird.
