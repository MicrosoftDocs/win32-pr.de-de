# <a name="comparing-regions"></a>Vergleichen von Regionen

Eine Anwendung vergleicht zwei Bereiche, um zu bestimmen, ob Sie identisch sind, indem Sie die [**equalrgn**](/windows/desktop/api/Wingdi/nf-wingdi-equalrgn) -Funktion aufrufen. Equalrgn betrachtet zwei Bereiche als identisch, wenn Sie gleich Größe und Form haben.
