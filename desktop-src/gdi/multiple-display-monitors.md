# <a name="multiple-display-monitors"></a>Mehrere Anzeige Monitore

Mehrere Anzeige Monitore sind eine Reihe verwandter Features, mit denen Anwendungen gleichzeitig mehrere Anzeigegeräte verwenden können. Es gibt zwei Möglichkeiten, wie mehrere Monitore verwendet werden können: als ein großer Desktop oder als Zahl unabhängige Anzeige. Bei Verwendung als ein großer Desktop erstellen die Monitore mehr Bildschirmspeicher für Anwendungen. Dies ist hilfreich, wenn Sie Ihren Bildschirm-Arbeitsbereich maximieren müssen, und ist besonders bei der Desktop Veröffentlichung, der Webentwicklung oder der Videobearbeitung nützlich. Bei der Verwendung als unabhängige Anzeige können die Monitore zum Spielen von spielen, zum Debuggen von Vollbildanwendungen, zum untersuchen oder zum Erstellen von Präsentationen verwendet werden.
