# <a name="multiple-display-monitors"></a>Mehrere Anzeigemonitore

Bei mehreren Anzeigemonitoren handelt es sich um eine Reihe verwandter Features, mit denen Anwendungen mehrere Anzeigegeräte gleichzeitig verwenden können. Es gibt zwei Möglichkeiten, mehrere Monitore zu verwenden: als einen großen Desktop oder als eine Reihe unabhängiger Anzeigen. Bei Verwendung als ein großer Desktop erstellen die Monitore mehr Bildschirmfläche für Anwendungen. Dies ist nützlich, wenn Sie Ihren Arbeitsbereich auf dem Bildschirm maximieren müssen, und ist besonders nützlich bei der Desktopveröffentlichung, Webentwicklung oder Videobearbeitung. Wenn sie als unabhängige Anzeigen verwendet werden, können die Monitore zum Spielen von Spielen, Debuggen von Vollbildanwendungen, Zum Teaching oder zum Erstellen von Präsentationen verwendet werden.
