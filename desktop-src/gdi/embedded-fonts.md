# <a name="embedded-fonts"></a><span data-ttu-id="3be19-103">Eingebettete Schriftarten</span><span class="sxs-lookup"><span data-stu-id="3be19-103">Embedded Fonts</span></span>

<span data-ttu-id="3be19-104">Das Einbetten einer Schriftart ist das Verfahren, bei dem ein Dokument und die darin enthaltenen Schriftarten in eine Datei für die Übertragung an einen anderen Computer gebündelt werden.</span><span class="sxs-lookup"><span data-stu-id="3be19-104">Embedding a font is the technique of bundling a document and the fonts it contains into a file for transmission to another computer.</span></span> <span data-ttu-id="3be19-105">Durch das Einbetten einer Schriftart wird sichergestellt, dass auf dem Computer, der die Datei empfängt, eine in einer übertragenen Datei angegebene Schriftart vorhanden ist.</span><span class="sxs-lookup"><span data-stu-id="3be19-105">Embedding a font guarantees that a font specified in a transmitted file will be present on the computer receiving the file.</span></span> <span data-ttu-id="3be19-106">Es können jedoch nicht alle Schriftarten von Computer zu Computer verschoben werden, da die meisten Schriftarten jeweils nur auf einem Computer lizenziert sind.</span><span class="sxs-lookup"><span data-stu-id="3be19-106">Not all fonts can be moved from computer to computer, however, since most fonts are licensed to only one computer at a time.</span></span> <span data-ttu-id="3be19-107">Nur TrueType-und OpenType-Schriftarten können eingebettet werden.</span><span class="sxs-lookup"><span data-stu-id="3be19-107">Only TrueType and OpenType fonts can be embedded.</span></span>

<span data-ttu-id="3be19-111">Es kann einen Verstoß gegen die proprietären Rechte oder den Benutzer Lizenzvertrag eines Schriftart Herstellers darstellen, um Schriftarten einzubetten, wenn Einbettungen nicht zulässig sind oder die folgenden Richtlinien zum Einbetten von Schriftarten nicht beachtet werden müssen.</span><span class="sxs-lookup"><span data-stu-id="3be19-111">It may be a violation of a font vendor's proprietary rights or user license agreement to embed any fonts where embedding is not permitted or to fail to observe the following guidelines on embedding fonts.</span></span> <span data-ttu-id="3be19-112">Die Lizenz einer Schriftart erteilt möglicherweise nur Lese-/Schreibberechtigung für eine Schriftart, die auf dem Zielcomputer installiert und verwendet werden soll.</span><span class="sxs-lookup"><span data-stu-id="3be19-112">A font's license may give only read/write permission for a font to be installed and used on the destination computer.</span></span> <span data-ttu-id="3be19-113">Oder die Lizenz erteilt eine Leseberechtigung.</span><span class="sxs-lookup"><span data-stu-id="3be19-113">Or the license may give read-only permission.</span></span> <span data-ttu-id="3be19-114">Mit der Berechtigung "schreibgeschützt" kann ein Dokument vom Zielcomputer angezeigt und gedruckt (aber nicht geändert) werden. <span data-ttu-id="3be19-116">Eine Anwendung kann den Lizenzstatus ermitteln, indem Sie die [**getoutlinetextmetrics**](/windows/desktop/api/Wingdi/nf-wingdi-getoutlinetextmetricsa) -Funktion aufrufen und den **otmfstype** -Member der [**outlinetextmetric**](/windows/desktop/api/Wingdi/ns-wingdi-outlinetextmetrica) -Struktur untersucht.</span><span class="sxs-lookup"><span data-stu-id="3be19-116">An application can determine the license status by calling the [**GetOutlineTextMetrics**](/windows/desktop/api/Wingdi/nf-wingdi-getoutlinetextmetricsa) function and examining the **otmfsType** member of the [**OUTLINETEXTMETRIC**](/windows/desktop/api/Wingdi/ns-wingdi-outlinetextmetrica) structure.</span></span> <span data-ttu-id="3be19-117">Wenn Bit 1 von **otmarstype** festgelegt ist, ist die Einbettung für die Schriftart nicht zulässig.</span><span class="sxs-lookup"><span data-stu-id="3be19-117">If bit 1 of **otmfsType** is set, embedding is not permitted for the font.</span></span> <span data-ttu-id="3be19-118">Wenn Bit 1 eindeutig ist, kann die Schriftart eingebettet werden.</span><span class="sxs-lookup"><span data-stu-id="3be19-118">If bit 1 is clear, the font can be embedded.</span></span> <span data-ttu-id="3be19-119">Wenn Bit 2 festgelegt ist, ist die Einbettung schreibgeschützt.</span><span class="sxs-lookup"><span data-stu-id="3be19-119">If bit 2 is set, the embedding is read-only.</span></span>

<span data-ttu-id="3be19-120">Zum Einbetten einer TrueType-Schriftart kann eine Anwendung die [**getfontdata**](/windows/desktop/api/Wingdi/nf-wingdi-getfontdata) -Funktion verwenden, um die Schriftart Datei zu lesen.</span><span class="sxs-lookup"><span data-stu-id="3be19-120">To embed a TrueType font, an application can use the [**GetFontData**](/windows/desktop/api/Wingdi/nf-wingdi-getfontdata) function to read the font file.</span></span> <span data-ttu-id="3be19-121">Wenn die Parameter *dwtable* und *dwOffset* von [**getfontdata**](/windows/win32/api/wingdi/nf-wingdi-getfontdata) auf 0l und der *cbData* -Parameter auf 1L festgelegt werden, wird sichergestellt, dass die Anwendung die gesamte Schriftart Datei von Anfang an liest.</span><span class="sxs-lookup"><span data-stu-id="3be19-121">Setting the *dwTable* and *dwOffset* parameters of [**GetFontData**](/windows/win32/api/wingdi/nf-wingdi-getfontdata) to 0L and the *cbData* parameter to 1L ensures that the application reads the entire font file from the beginning.</span></span>
