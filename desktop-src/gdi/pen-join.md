# <a name="pen-join"></a>Stift Verknüpfung

Das Join-Attribut gibt an, wie die Enden von zwei geometrischen Linien verknüpft werden: ein-oder abgerundet. Die folgende Abbildung zeigt Paare verbundener Linien, die mit den einzelnen Jointypen gezeichnet werden.

![Abbildung, die drei Zeilenpaare anzeigt, die in der rechten Ecke verknüpft sind. jedes Paar veranschaulicht einen anderen Jointyp.](images/cspen-05.png)
