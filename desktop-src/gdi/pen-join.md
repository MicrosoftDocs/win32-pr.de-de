# <a name="pen-join"></a>Stift-Join

Das Join-Attribut gibt an, wie die Enden von zwei geometrischen Linien verknüpft werden: abgeschrägt, mitered oder round. Die folgende Abbildung zeigt Paare verbundener Linien, die mit den einzelnen Jointypen gezeichnet werden.
