# <a name="formatting-text-windows-gdi"></a>Formatieren von Text (Windows GDI)

Die Formatierungsfunktionen können in drei Kategorien unterteilt werden: diejenigen, die die [Text Formatierungs Attribute](text-formatting-attributes.md) für einen Gerätekontext abrufen oder festlegen, die die [Zeichen breiten](character-widths.md)abrufen, und diejenigen, die Zeichen folgen [breiten und-Höhen](string-widths-and-heights.md)abrufen.
