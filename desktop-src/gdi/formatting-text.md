# <a name="formatting-text-windows-gdi"></a>Formatieren von Text (Windows GDI)

Die [Formatierungsfunktionen](character-widths.md)können in drei Kategorien unterteilt werden: diejenigen, die die Textformatierungsattribute für einen Gerätekontext abrufen oder festlegen, diejenigen, die Zeichenbreiten abrufen, und diejenigen, die Zeichenfolgenbreiten und -höhen [](text-formatting-attributes.md) [abrufen.](string-widths-and-heights.md)
