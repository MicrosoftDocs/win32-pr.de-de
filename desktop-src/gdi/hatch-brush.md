# <a name="hatch-brush"></a>Pinsel Schraffur

Es gibt sechs vordefinierte logische Schraffurpinsel, die von GDI verwaltet werden. Die folgenden Rechtecke wurden mit den sechs vordefinierten Schraffurpinseln gezeichnet.

![Abbildung, die sechs Felder anzeigt, eine von jedem der sechs Schraffurpinsel.](images/hatchbrush.png)

Eine Anwendung kann einen Schraffurpinsel erstellen, indem Sie die [**CreateHatchBrush**](/windows/desktop/api/Wingdi/nf-wingdi-createhatchbrush) -Funktion aufruft und eine der sechs Schraffurstile angibt.
