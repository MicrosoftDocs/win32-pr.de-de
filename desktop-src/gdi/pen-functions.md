# <a name="pen-functions-windows-gdi"></a>Stiftfunktionen (Windows GDI)

Die folgenden Funktionen werden mit Stiften verwendet.



| Funktion                                       | Beschreibung                                                                                            |
|------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**CreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-createpen)                 | Erstellt einen logischen Stift mit dem angegebenen Stil, der angegebenen Breite und Farbe.                                  |
| [**CreatePenIndirect**](/windows/desktop/api/Wingdi/nf-wingdi-createpenindirect) | Erstellt einen logischen logischen Stift, der über den Stil, die Breite und die Farbe verfügt, die in einer -Struktur angegeben sind.          |
| [**ExtCreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-extcreatepen)           | Erstellt einen logischen logischen oder geometrischen Stift, der über die angegebenen Stil-, Breiten- und Pinselattribute verfügt. |
| [**SetDCPenColor**](/windows/desktop/api/Wingdi/nf-wingdi-setdcpencolor)         | Legt die aktuelle Gerätekontextstiftfarbe fest.                                                             |
