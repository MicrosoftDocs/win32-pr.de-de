# <a name="brushes-windows-gdi"></a>Pinsel (Windows GDI)

Ein *Pinsel* ist ein Grafiktool, mit dem Anwendungen das Innere von Polygonen, Ellipsen und Pfaden zeichnen. Zeichnungsanwendungen verwenden Pinsel, um Formen zu zeichnen. Textverarbeitungsanwendungen verwenden Pinsel, um Regeln zu zeichnen.
