# <a name="brushes-windows-gdi"></a>Pinsel (Windows GDI)

Ein *Pinsel* ist ein Grafik Tool, das von Anwendungen verwendet wird, um das Innere von Polygonen, Ellipsen und Pfaden zu zeichnen. Zeichen Anwendungen verwenden Pinsel zum Zeichnen von Formen. Textverarbeitungsanwendungen verwenden Pinsel zum Zeichnen von Regeln.
