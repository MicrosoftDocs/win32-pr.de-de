# <a name="polyline-records"></a>Polyline-Datensätze

[Polyliniendatensätze](/windows/desktop/api/Wingdi/nf-wingdi-polyline) sind eine Reihe von Punkten. die Linien, die zwischen den Punkten gezeichnet werden, beschreiben den Umriss des Zeichens. Ein polyliniendatensatz beginnt mit dem letzten Punkt im vorherigen Datensatz (oder für den ersten Datensatz in der Kontur, dem Ausgangspunkt). Jeder Punkt im Datensatz befindet sich in der Symbol Gliederung und kann einfach mithilfe von geraden Linien verbunden werden.
