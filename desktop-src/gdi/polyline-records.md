# <a name="polyline-records"></a>Polylinedatensätze

[Polyliniendatensätze](/windows/desktop/api/Wingdi/nf-wingdi-polyline) sind eine Reihe von Punkten. Linien, die zwischen den Punkten gezeichnet werden, beschreiben die Kontur des Zeichens. Ein Polyliniendatensatz beginnt mit dem letzten Punkt im vorherigen Datensatz (oder für den ersten Datensatz in der Kontur der Ausgangspunkt). Jeder Punkt im Datensatz befindet sich auf der Glyphengliederung und kann einfach über gerade Linien verbunden werden.
