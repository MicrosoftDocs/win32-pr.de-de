# <a name="device-space-to-physical-device-transformation"></a>Device-Space Physical-Device Transformation

Die Transformation für das Gerät zwischen Geräten und physischen Geräten ist in vielerlei Hinsicht eindeutig. Beispielsweise ist es auf die Übersetzung beschränkt und wird vom System gesteuert. Der einzige Zweck dieser Transformation besteht darin sicherzustellen, dass der Ursprung des geräteraums dem richtigen Punkt auf dem physischen Gerät zugeordnet ist. Es gibt keine Funktionen, mit denen diese Transformation festgelegt werden kann, und es sind keine Funktionen zum Abrufen verwandter Daten vorhanden.
