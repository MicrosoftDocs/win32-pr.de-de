# <a name="window-coordinate-system"></a><span data-ttu-id="85861-103">Fenster Koordinaten System</span><span class="sxs-lookup"><span data-stu-id="85861-103">Window Coordinate System</span></span>

<span data-ttu-id="85861-104">Das Koordinatensystem für ein Fenster basiert auf dem Koordinatensystem des Anzeige Geräts.</span><span class="sxs-lookup"><span data-stu-id="85861-104">The coordinate system for a window is based on the coordinate system of the display device.</span></span> <span data-ttu-id="85861-105">Die grundlegende Maßeinheit ist die Geräteeinheit (in der Regel das Pixel).</span><span class="sxs-lookup"><span data-stu-id="85861-105">The basic unit of measure is the device unit (typically, the pixel).</span></span> <span data-ttu-id="85861-106">Die Punkte auf dem Bildschirm werden durch x-und y-Koordinatenpaare beschrieben.</span><span class="sxs-lookup"><span data-stu-id="85861-106">Points on the screen are described by x- and y-coordinate pairs.</span></span> <span data-ttu-id="85861-107">Die x-Koordinaten werden auf der rechten Seite vergrößert. y-Koordinaten werden von oben nach unten vergrößert.</span><span class="sxs-lookup"><span data-stu-id="85861-107">The x-coordinates increase to the right; y-coordinates increase from top to bottom.</span></span> <span data-ttu-id="85861-108">Der Ursprung (0, 0) für das System hängt vom Typ der verwendeten Koordinaten ab.</span><span class="sxs-lookup"><span data-stu-id="85861-108">The origin (0,0) for the system depends on the type of coordinates being used.</span></span>

<span data-ttu-id="85861-109">Das System und die Anwendungen geben die Position eines Fensters auf dem Bildschirm in *Bildschirm Koordinaten* an.</span><span class="sxs-lookup"><span data-stu-id="85861-109">The system and applications specify the position of a window on the screen in *screen coordinates*.</span></span> <span data-ttu-id="85861-110">Bei Bildschirm Koordinaten wird der Ursprung in der oberen linken Ecke des Bildschirms angezeigt.</span><span class="sxs-lookup"><span data-stu-id="85861-110">For screen coordinates, the origin is the upper-left corner of the screen.</span></span> <span data-ttu-id="85861-111">Die vollständige Position eines Fensters wird häufig durch eine [**Rect**](/previous-versions//dd162897(v=vs.85)) -Struktur beschrieben, die die Bildschirm Koordinaten zweier Punkte enthält, die die oberen linken und unteren rechten Ecke des Fensters definieren.</span><span class="sxs-lookup"><span data-stu-id="85861-111">The full position of a window is often described by a [**RECT**](/previous-versions//dd162897(v=vs.85)) structure containing the screen coordinates of two points that define the upper-left and lower-right corners of the window.</span></span>

<span data-ttu-id="85861-112">Das System und die Anwendungen geben die Position der Punkte in einem Fenster mithilfe von *Client Koordinaten* an.</span><span class="sxs-lookup"><span data-stu-id="85861-112">The system and applications specify the position of points in a window by using *client coordinates*.</span></span> <span data-ttu-id="85861-113">Der Ursprung ist in diesem Fall die linke obere Ecke des Fensters oder des Client Bereichs.</span><span class="sxs-lookup"><span data-stu-id="85861-113">The origin in this case is the upper-left corner of the window or client area.</span></span> <span data-ttu-id="85861-114">Mithilfe von Client Koordinaten wird sichergestellt, dass eine Anwendung während des Zeichnens im Fenster konsistente Koordinaten Werte verwenden kann, unabhängig von der Position des Fensters auf dem Bildschirm.</span><span class="sxs-lookup"><span data-stu-id="85861-114">Client coordinates ensure that an application can use consistent coordinate values while drawing in the window, regardless of the position of the window on the screen.</span></span>

<span data-ttu-id="85861-115">Die Abmessungen des Client Bereichs werden auch durch eine [**Rect**](/previous-versions//dd162897(v=vs.85)) -Struktur beschrieben, die Client Koordinaten für den Bereich enthält.</span><span class="sxs-lookup"><span data-stu-id="85861-115">The dimensions of the client area are also described by a [**RECT**](/previous-versions//dd162897(v=vs.85)) structure that contains client coordinates for the area.</span></span> <span data-ttu-id="85861-116">In allen Fällen ist die obere linke Koordinate des Rechtecks im Fenster-oder Client Bereich enthalten, während die Koordinate der unteren rechten Seite ausgeschlossen wird.</span><span class="sxs-lookup"><span data-stu-id="85861-116">In all cases, the upper-left coordinate of the rectangle is included in the window or client area, while the lower-right coordinate is excluded.</span></span> <span data-ttu-id="85861-117">Grafik Vorgänge in einem Fenster-oder Client Bereich werden vom rechten und unteren Rand des einschließenden Rechtecks ausgeschlossen.</span><span class="sxs-lookup"><span data-stu-id="85861-117">Graphics operations in a window or client area are excluded from the right and lower edges of the enclosing rectangle.</span></span>

<span data-ttu-id="85861-118">Gelegentlich ist es erforderlich, dass Anwendungen in einem Fenster Koordinaten in einem anderen Fenster zuordnen.</span><span class="sxs-lookup"><span data-stu-id="85861-118">Occasionally, applications may be required to map coordinates in one window to those of another window.</span></span> <span data-ttu-id="85861-119">Eine Anwendung kann Koordinaten mithilfe der [**mapwindowpoints**](/windows/desktop/api/Winuser/nf-winuser-mapwindowpoints) -Funktion zuordnen.</span><span class="sxs-lookup"><span data-stu-id="85861-119">An application can map coordinates by using the [**MapWindowPoints**](/windows/desktop/api/Winuser/nf-winuser-mapwindowpoints) function.</span></span> <span data-ttu-id="85861-120">Handelt es sich bei einem der Fenster um das Desktop Fenster, konvertiert die Funktion Bildschirm Koordinaten effektiv in Client Koordinaten und umgekehrt. das Desktop Fenster wird immer in Bildschirm Koordinaten angegeben.</span><span class="sxs-lookup"><span data-stu-id="85861-120">If one of the windows is the desktop window, the function effectively converts screen coordinates to client coordinates and vice versa; the desktop window is always specified in screen coordinates.</span></span>
