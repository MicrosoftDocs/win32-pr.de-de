# <a name="sizesize-constructors"></a>Size.Size-Konstruktoren

In diesem Thema werden die Konstruktoren der [**Size-Klasse**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) aufgeführt. Eine vollständige Klassenauflistung finden Sie unter **Size Class**.

### <a name="overload-list"></a>Überladeliste



| Konstruktor                                                     | BESCHREIBUNG                                                                                                                                          |
|:----------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Size(Size&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size(inconstsize_))           | Erstellt ein [**Size-Objekt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) und initialisiert seine Member, indem die Member eines anderen **Size-Objekts kopiert** werden.<br/> |
| [**Size(INT,INT)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size(inint_inint)) | Erstellt ein [**Size-Objekt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) und initialisiert dessen **Width-** und **Height-Datenmitglieder.**<br/>                     |
| [**Size()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size)                     | Erstellt ein neues [**Size-Objekt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) und initialisiert die Member auf 0 (null). Dies ist der Standardkonstruktor<br/>      |
