# <a name="sizesize-constructors"></a>Size. Size-Konstruktoren

In diesem Thema werden die Konstruktoren der [**size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) -Klasse aufgelistet. Eine komplette Klassen Auflistung finden Sie unter **size-Klasse**.

### <a name="overload-list"></a>Überladeliste



| Konstruktor                                                     | BESCHREIBUNG                                                                                                                                          |
|:----------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Größe (Größe&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size(inconstsize_))           | Erstellt ein [**size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) -Objekt und initialisiert seine Member, indem die Member eines anderen **size** -Objekts kopiert werden.<br/> |
| [**Größe (int, int)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size(inint_inint)) | Erstellt ein [**size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) -Objekt und initialisiert seine **Width** -und **height** -Datenmember.<br/>                     |
| [**Größe ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size)                     | Erstellt ein neues [**size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) -Objekt und initialisiert die Elemente mit 0 (null). Dies ist der Standardkonstruktor<br/>      |
