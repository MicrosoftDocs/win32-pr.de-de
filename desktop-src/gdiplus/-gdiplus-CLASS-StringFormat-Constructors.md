# <a name="stringformatstringformat-constructors"></a>StringFormat.StringFormat-Konstruktoren

In diesem Thema werden die Konstruktoren der [**StringFormat-Klasse**](/windows/win32/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) aufgeführt. Eine vollständige Klassenauflistung finden Sie unter **StringFormat-Klasse**.

### <a name="overload-list"></a>Überladeliste



| Konstruktor                                                                                        | BESCHREIBUNG                                                                                                                        |
|:---------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**StringFormat(INT,LANGID)**](/windows/win32/api/gdiplusstringformat/nf-gdiplusstringformat-stringformat-stringformat(inint_inlangid)) | Erstellt ein [**StringFormat-Objekt**](/windows/win32/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) basierend auf Zeichenfolgenformatflags und einer Sprache.<br/> |
| [**StringFormat(StringFormat \* )**](/windows/win32/api/gdiplusstringformat/nf-gdiplusstringformat-stringformat-stringformat(conststringformat_))           | Erstellt ein [**StringFormat-Objekt**](/windows/win32/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) aus einem anderen **StringFormat-Objekt.**<br/>        |
