# <a name="colorcolor-constructors"></a>Color. Color-Konstruktoren

In diesem Thema werden die Konstruktoren der [**Color**](/windows/win32/api/gdipluscolor/nl-gdipluscolor-color) -Klasse aufgelistet. Eine komplette Klassen Auflistung finden Sie unter **Color-Klasse**.

### <a name="overload-list"></a>Überladeliste



| Konstruktor                                                               | BESCHREIBUNG                                                                                                                                                                                                         |
|:--------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Farbe (ARGB)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inargb))                   | Erstellt ein [**Color:: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inargb)) -Objekt mit einem **ARGB** -Wert.<br/>                                                                                                    |
| [**Farbe (Byte, Byte, Byte)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte))        | Erstellt mithilfe der angegebenen Werte für die roten, grünen und blauen Komponenten ein [**Color:: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte)) -Objekt. Dieser Konstruktor legt die Alpha Komponente auf 255 (nicht transparent) fest.<br/> |
| [**Farbe (Byte, Byte, Byte, Byte)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte_inbyte)) | Erstellt mit den angegebenen Werten für die Alpha-, rot-, grün-und Blau-Komponenten ein [**Color:: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte_inbyte)) -Objekt.<br/>                                                   |
| [**Farbe ()**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color)                            | Erstellt ein [**Color:: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color) -Objekt und initialisiert es mit undurchsichtigem schwarz. Dies ist der Standardkonstruktor<br/>                                                                |
