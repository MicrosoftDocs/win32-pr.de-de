# <a name="colorcolor-constructors"></a>Color.Color-Konstruktoren

In diesem Thema werden die Konstruktoren der [**Color-Klasse**](/windows/win32/api/gdipluscolor/nl-gdipluscolor-color) aufgelistet. Eine vollständige Klassenauflistung finden Sie unter **Color Class**.

### <a name="overload-list"></a>Überladeliste



| Konstruktor                                                               | BESCHREIBUNG                                                                                                                                                                                                         |
|:--------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Color(ARGB)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inargb))                   | Erstellt ein [**Color::Color-Objekt**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inargb)) mithilfe eines **ARGB-Werts.**<br/>                                                                                                    |
| [**Color(BYTE, BYTE, BYTE)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte))        | Erstellt ein [**Color::Color-Objekt**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte)) unter Verwendung der angegebenen Werte für die Rot-, Grün- und Blaukomponenten. Dieser Konstruktor legt die Alphakomponente auf 255 (nicht transparent) fest.<br/> |
| [**Color(BYTE, BYTE, BYTE, BYTE)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte_inbyte)) | Erstellt ein [**Color::Color-Objekt**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte_inbyte)) unter Verwendung der angegebenen Werte für die Alpha-, Rot-, Grün- und Blaukomponenten.<br/>                                                   |
| [**Color()**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color)                            | Erstellt ein [**Color::Color-Objekt**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color) und initialisiert es in deckendes Schwarz. Dies ist der Standardkonstruktor<br/>                                                                |
