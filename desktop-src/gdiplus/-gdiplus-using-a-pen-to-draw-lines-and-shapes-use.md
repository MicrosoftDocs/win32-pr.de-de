# <a name="using-a-pen-to-draw-lines-and-shapes"></a>Verwenden eines Stiftes zum Zeichnen von Linien und Formen

Die [**Graphics-Klasse**](/windows/win32/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) stellt eine Vielzahl von Zeichnungsmethoden bereit, einschließlich der in der folgenden Liste gezeigten:

-   [DrawLine-Methoden](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawline(inconstpen_inint_inint_inint_inint))
-   [DrawRectangle-Methoden](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawrectangle(inconstpen_inconstrectf_))
-   [DrawEllipse-Methoden](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawellipse(inconstpen_inconstrectf_))
-   [DrawArc-Methoden](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawarc(inconstpen_inconstrectf__inreal_inreal))
-   [**Graphics::D rawPath**](/windows/win32/api/Gdiplusgraphics/nf-gdiplusgraphics-graphics-drawpath)
-   [DrawCurve-Methoden](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawcurve(inconstpen_inconstpoint_inint))
-   [DrawBezier-Methoden](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawbezier(inconstpen_inconstpointf__inconstpointf__inconstpointf__inconstpointf_))

Eines der Argumente, die Sie an eine solche [](/windows/win32/api/gdipluspen/nl-gdipluspen-pen) Zeichnungsmethode übergeben, ist die Adresse eines Stiftobjekts.

-   [Festlegen der Stiftbreite und -ausrichtung](-gdiplus-setting-pen-width-and-alignment-use.md)
-   [Zeichnen einer Linie mit Linienobergrenzen](-gdiplus-drawing-a-line-with-line-caps-use.md)
-   [Verknüpfen von Zeilen](-gdiplus-joining-lines-use.md)
-   [Zeichnen einer benutzerdefinierten gestrichelten Linie](-gdiplus-drawing-a-custom-dashed-line-use.md)
-   [Zeichnen einer mit einer Textur gefüllten Linie](-gdiplus-drawing-a-line-filled-with-a-texture-use.md)
