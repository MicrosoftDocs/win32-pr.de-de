# <a name="using-a-pen-to-draw-lines-and-shapes"></a>Verwenden eines Stiftes zum Zeichnen von Linien und Formen

Die [**Grafik**](/windows/win32/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) Klasse bietet eine Vielzahl von Zeichnungs Methoden, einschließlich der in der folgenden Liste aufgeführten Zeichnungs Methoden:

-   [DrawLine-Methoden](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawline(inconstpen_inint_inint_inint_inint))
-   [Drawrechteck-Methoden](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawrectangle(inconstpen_inconstrectf_))
-   [DrawEllipse-Methoden](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawellipse(inconstpen_inconstrectf_))
-   [DrawArc-Methoden](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawarc(inconstpen_inconstrectf__inreal_inreal))
-   [**Grafiken::D rawpath**](/windows/win32/api/Gdiplusgraphics/nf-gdiplusgraphics-graphics-drawpath)
-   [DrawCurve-Methoden](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawcurve(inconstpen_inconstpoint_inint))
-   [DrawBezier-Methoden](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawbezier(inconstpen_inconstpointf__inconstpointf__inconstpointf__inconstpointf_))

Eines der Argumente, das Sie an eine solche Zeichnungs Methode übergeben, ist die Adresse eines [**Stift**](/windows/win32/api/gdipluspen/nl-gdipluspen-pen) Objekts.

-   [Zeichnen einer Linie mit Linien Deckeln](-gdiplus-drawing-a-line-with-line-caps-use.md)
-   [Joinlinien](-gdiplus-joining-lines-use.md)
-   [Zeichnen einer benutzerdefinierten gestrichelten Linie](-gdiplus-drawing-a-custom-dashed-line-use.md)
