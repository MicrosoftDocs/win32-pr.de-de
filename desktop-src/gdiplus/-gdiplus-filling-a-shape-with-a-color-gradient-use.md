# <a name="filling-a-shape-with-a-color-gradient"></a>Auffüllen einer Form mit einem Farbverlauf

Mithilfe eines Farbverlaufs Pinsels können Sie eine Form mit einer allmählich veränderlichen Farbe ausfüllen. Windows GDI+ bietet lineare Farbverlaufs Pinsel und Pfad Farbverlaufs Pinsel. Die folgende Abbildung zeigt ein Rechteck mit einem linearen Farbverlaufs Pinsel und einer Ellipse, die mit einem Pfad Farbverlaufs Pinsel gefüllt ist.

![Darstellung eines Rechtecks, das von oben links nach unten rechts aufgefüllt ist, und einer Ellipse, die von Center zu Edge füllt](images/gradient2.png)

Weitere Informationen zu Farbverlaufs Pinseln finden Sie unter [Ausfüllen von Formen mit einem Farbverlaufs Pinsel](-gdiplus-filling-shapes-with-a-gradient-brush-use.md).
