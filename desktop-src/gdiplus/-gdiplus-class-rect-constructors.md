# <a name="rectrect-constructors"></a>Rect. Rect-Konstruktoren

In diesem Thema werden die Konstruktoren der [**Rect**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) -Klasse aufgelistet. Eine komplette Klassen Auflistung finden Sie unter **Rect-Klasse**.

### <a name="overload-list"></a>Überladeliste



| Konstruktor                                                                 | BESCHREIBUNG                                                                                                                                                                                                                                                                                        |
|:----------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Rect (Punkt&, Größe&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rect-rect(inconstpoint__inconstsize_))       | Erstellt ein [**Rect**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) -Objekt, indem ein [**Punkt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) Objekt verwendet wird, um die **X** -und **Y** -Datenmember zu initialisieren, und ein [**size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) -Objekt, um die Datenmember **Width** und **height** zu initialisieren.<br/> |
| [**Rect (int, int, int, int)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rect-rect(inint_inint_inint_inint)) | Erstellt ein [**Rect**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) -Objekt mit vier ganzen Zahlen, um die Datenmember **X**, **Y**, **Width** und **height** zu initialisieren.<br/>                                                                                                                               |
| [**Rect ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rect-rect)                                 | Erstellt ein [**Rect**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) -Objekt, dessen x-Koordinate, y-Koordinate, Breite und Höhe Null sind. Dies ist der Standardkonstruktor <br/>                                                                                                                          |
