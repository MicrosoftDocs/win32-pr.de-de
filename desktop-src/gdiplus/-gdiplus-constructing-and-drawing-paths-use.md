# <a name="constructing-and-drawing-paths"></a>Erstellen und Zeichnen von Pfaden

Ein Pfad ist eine Sequenz von Grafikprimitiven (Linien, Rechtecke, Kurven, Text usw.), die als einzelne Einheit bearbeitet und gezeichnet werden können. Ein Pfad kann in *Abbildungen* unterteilt werden, die entweder offen oder geschlossen sind. Eine Abbildung kann mehrere Primitive enthalten.

Sie können einen Pfad zeichnen, indem Sie die [**Graphics::D rawPath-Methode**](/windows/desktop/api/Gdiplusgraphics/nf-gdiplusgraphics-graphics-drawpath) der [**Graphics-Klasse**](/windows/desktop/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) aufrufen, und Sie können einen Pfad ausfüllen, indem Sie die [**Graphics::FillPath-Methode**](/windows/desktop/api/Gdiplusgraphics/nf-gdiplusgraphics-graphics-fillpath) der **Graphics-Klasse** aufrufen.

In den folgenden Themen werden Pfade ausführlicher behandelt:

-   [Erstellen von Abbildungen Figuren aus Linien, Kurven und Formen](-gdiplus-creating-figures-from-lines-curves-and-shapes-use.md)
-   [Auffüllen von offenen Zahlen](-gdiplus-filling-open-figures-use.md)
