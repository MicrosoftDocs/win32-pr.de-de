# <a name="texturebrushtexturebrush-constructors"></a>TextureBrush.TextureBrush-Konstruktoren

In diesem Thema werden die Konstruktoren der [**TextureBrush-Klasse**](/windows/win32/api/gdiplusbrush/nl-gdiplusbrush-texturebrush) aufgelistet. Eine vollständige Klassenauflistung finden Sie unter **TextureBrush-Klasse.**

### <a name="overload-list"></a>Überladeliste



| Konstruktor                                                                                                                                                        | BESCHREIBUNG                                                                                                                                                                                                                 |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**TextureBrush(Image \* , WrapMode)**](/windows/win32/api/gdiplusbrush/nf-gdiplusbrush-texturebrush-texturebrush(inimage_inwrapmode))                                                                 | Erstellt ein [**TextureBrush-Objekt**](/windows/win32/api/gdiplusbrush/nl-gdiplusbrush-texturebrush) basierend auf einem Bild und einem Umbruchmodus. Die Größe des Pinsels ist standardmäßig auf die Größe des Bilds eingestellt, sodass das gesamte Bild vom Pinsel verwendet wird.<br/> |
| [**TextureBrush(Image \* , WrapMode, Rect&)**](/windows/win32/api/gdiplusbrush/nf-gdiplusbrush-texturebrush-texturebrush(inimage_inwrapmode_constinrect_))                               | Erstellt ein [**TextureBrush-Objekt**](/windows/win32/api/gdiplusbrush/nl-gdiplusbrush-texturebrush) basierend auf einem Bild, einem Umbruchmodus und einem definierenden Rechteck.<br/>                                                                             |
| [**TextureBrush(Image \* , wrapMode, RectF&)**](/windows/win32/api/gdiplusbrush/nf-gdiplusbrush-texturebrush-texturebrush(inimage_inwrapmode_inconstrectf_))                             | Erstellt ein [**TextureBrush-Objekt**](/windows/win32/api/gdiplusbrush/nl-gdiplusbrush-texturebrush) basierend auf einem Bild, einem Umbruchmodus und einem definierenden Rechteck.<br/>                                                                             |
| [**TextureBrush(Image, \* Rect&,ImageAttributes \* )**](/windows/win32/api/gdiplusbrush/nf-gdiplusbrush-texturebrush-texturebrush(inimage_inconstrect__inconstimageattributes))        | Erstellt ein [**TextureBrush-Objekt**](/windows/win32/api/gdiplusbrush/nl-gdiplusbrush-texturebrush) basierend auf einem Bild, einem definierenden Rechteck und einer Reihe von Bildeigenschaften.<br/>                                                               |
| [**TextureBrush(Image, \* RectF&,ImageAttributes \* )**](/windows/win32/api/gdiplusbrush/nf-gdiplusbrush-texturebrush-texturebrush(inimage_inconstrectf__inconstimageattributes))      | Erstellt ein [**TextureBrush-Objekt**](/windows/win32/api/gdiplusbrush/nl-gdiplusbrush-texturebrush) basierend auf einem Bild, einem definierenden Rechteck und einer Reihe von Bildeigenschaften.<br/>                                                               |
| [**TextureBrush(Image, \* WrapMode, INT, INT, INT, INT)**](/windows/win32/api/gdiplusbrush/nf-gdiplusbrush-texturebrush-texturebrush(inimage_inwrapmode_inint_inint_inint_inint))     | Erstellt ein [**TextureBrush-Objekt**](/windows/win32/api/gdiplusbrush/nl-gdiplusbrush-texturebrush) basierend auf einem Bild, einem Umbruchmodus und einem definierenden Satz von Koordinaten.<br/>                                                                    |
| [**TextureBrush(Image, \* WrapMode, REAL, REAL, REAL, REAL)**](/windows/win32/api/gdiplusbrush/nf-gdiplusbrush-texturebrush-texturebrush(inimage_inwrapmode_inreal_inreal_inreal_inreal)) | Erstellt ein [**TextureBrush-Objekt**](/windows/win32/api/gdiplusbrush/nl-gdiplusbrush-texturebrush) basierend auf einem Bild, einem Umbruchmodus und einem definierenden Satz von Koordinaten.<br/>                                                                    |
