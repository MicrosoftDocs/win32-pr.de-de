# <a name="pointpoint-constructors"></a>Point. Point-Konstruktoren

In diesem Thema werden die Konstruktoren der [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) -Klasse aufgelistet. Eine komplette Klassen Auflistung finden Sie unter **Point-Klasse**.

### <a name="overload-list"></a>Überladeliste



| Konstruktor                                                | BESCHREIBUNG                                                                                                                                                                      |
|:-----------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Punkt (Größe&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inconstsize_))   | Erstellt ein [**Punkt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) Objekt mithilfe eines [**size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) -Objekts, um die **X** -und **Y** -Datenmember zu initialisieren.<br/> |
| [**Punkt (Punkt&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inconstpoint_)) | Erstellt ein neues [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) -Objekt und kopiert die Datenmember von einem anderen **Point** -Objekt.<br/>                                           |
| [**Punkt (int, int)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inint_inint))  | Erstellt ein [**Punkt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) Objekt mit zwei Ganzzahlen, um die **X** -und **Y** -Datenmember zu initialisieren.<br/>                                       |
| [**Punkt ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point)             | Erstellt ein [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) -Objekt und initialisiert die **X** -und **Y** -Datenmember mit 0 (null). Dies ist der Standardkonstruktor<br/>               |
