# <a name="matrixtransformpoints-methods"></a>Matrix.TransformPoints-Methoden

In diesem Thema werden die TransformPoints-Methoden der [**Matrix-Klasse**](/windows/win32/api/gdiplusmatrix/nl-gdiplusmatrix-matrix) aufgelistet. Eine vollständige Liste der Methoden für die **Matrix-Klasse** finden Sie unter [Matrixmethoden.](-gdiplus-class-matrix-methods.md)

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                               | Beschreibung                                                                                                                                                                                                                                                                                                 |
|:-----------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**TransformPoints(Point \* , INT)**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-transformpoints(inoutpoint_inint))   | Die [**Matrix::TransformPoints-Methode**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-transformpoints(inoutpoint_inint)) multipliziert jeden Punkt in einem Array mit dieser Matrix. Jeder Punkt wird als Zeilenmatrix behandelt. Die Multiplikation wird mit der Zeilenmatrix links und dieser Matrix auf der rechten Seite ausgeführt.<br/>  |
| [**TransformPoints(PointF, \* INT)**](/previous-versions//ms535321(v=vs.85)) | Die [**Matrix::TransformPoints-Methode**](/previous-versions//ms535321(v=vs.85)) multipliziert jeden Punkt in einem Array mit dieser Matrix. Jeder Punkt wird als Zeilenmatrix behandelt. Die Multiplikation wird mit der Zeilenmatrix links und dieser Matrix auf der rechten Seite ausgeführt.<br/> |
