# <a name="sizefsizef-constructors"></a>SizeF.SizeF-Konstruktoren

In diesem Thema werden die Konstruktoren der [**SizeF-Klasse**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) aufgeführt. Eine vollständige Klassenauflistung finden Sie unter **SizeF-Klasse**.

### <a name="overload-list"></a>Überladeliste



| Konstruktor                                                          | BESCHREIBUNG                                                                                                                                             |
|:---------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SizeF(SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-sizef-sizef(inconstsizef_))            | Erstellt ein [**SizeF-Objekt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) und initialisiert seine Member, indem die Member eines anderen **SizeF-Objekts kopiert** werden.<br/> |
| [**SizeF(REAL,REAL)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-sizef-sizef(inreal_inreal)) | Erstellt ein [**SizeF-Objekt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) und initialisiert dessen **Width-** und **Height-Datenmitglieder.**<br/>                      |
| [**SizeF()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-sizef-sizef)                       | Erstellt ein [**SizeF-Objekt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) und initialisiert die Member auf 0 (null). Dies ist der Standardkonstruktor<br/>           |
