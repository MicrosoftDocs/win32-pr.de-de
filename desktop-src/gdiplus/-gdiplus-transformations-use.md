# <a name="transformations"></a>Transformationen

Affine Transformationen umfassen das Drehen, skalieren, reflektieren, Scheren und übersetzen. In Windows GDI+ stellt die [**Matrix**](/windows/desktop/api/gdiplusmatrix/nl-gdiplusmatrix-matrix) Klasse die Grundlage für die Durchführung von affinen Transformationen für Vektorzeichnungen, Bilder und Text dar.

In den folgenden Themen werden Transformationen ausführlicher behandelt:

-   [Verwenden der globalen Transformation](-gdiplus-using-the-world-transformation-use.md)
-   [Bedeutung der Transformationsreihenfolge](-gdiplus-why-transformation-order-is-significant-use.md)
