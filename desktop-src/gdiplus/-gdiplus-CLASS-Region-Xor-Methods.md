# <a name="regionxor-methods"></a>Region.Xor-Methoden

In diesem Thema werden die Xor-Methoden der [**Region-Klasse**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) aufgelistet. Eine vollständige Liste der Methoden für die **Region-Klasse** finden Sie unter [Regionsmethoden.](-gdiplus-class-region-methods.md)

### <a name="overload-list"></a>Überladeliste



| Methode                                                         | Beschreibung                                                                                                                                                                              |
|:---------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Xor(Rect&)**](/previous-versions//ms534787(v=vs.85))     | Die [**Region::Xor-Methode**](/previous-versions//ms534787(v=vs.85)) aktualisiert diesen Bereich auf die nicht schneidenden Teile von sich selbst und das Innere des angegebenen Rechtecks.<br/>  |
| [**Xor(RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstrectf_))   | Die [**Region::Xor-Methode**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstrectf_)) aktualisiert diesen Bereich auf die nicht schneidenden Teile von sich selbst und das Innere des angegebenen Rechtecks.<br/> |
| [**Xor(Region) \***](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstregion))     | Die [**Region::Xor-Methode**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstregion)) aktualisiert diese Region auf die nicht überschneidenden Teile von sich selbst und eine andere Region.<br/>                         |
| [**Xor(GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstgraphicspath)) | Die [**Region::Xor-Methode**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstgraphicspath)) aktualisiert diesen Bereich auf die nicht schneidenden Teile von sich selbst und das Innere des angegebenen Pfads.<br/>            |
