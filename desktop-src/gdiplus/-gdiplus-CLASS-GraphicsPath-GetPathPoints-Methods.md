# <a name="graphicspathgetpathpoints-methods"></a>GraphicsPath.GetPathPoints-Methoden

In diesem Thema werden die GetPathPoints-Methoden der [**GraphicsPath-Klasse**](/windows/win32/api/gdipluspath/nl-gdipluspath-graphicspath) aufgeführt. Eine vollständige Liste der Methoden für die **GraphicsPath-Klasse** finden Sie unter [**GraphicsPath**](/windows/win32/api/gdipluspath/nl-gdipluspath-graphicspath).

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                    | Beschreibung                                                                                                                                                                                                                                                                           |
|:----------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetPathPoints(Point \* ,INT)**](/windows/win32/api/gdipluspath/nf-gdipluspath-graphicspath-getpathpoints(outpoint_inint))   | Die [**GraphicsPath::GetPathPoints-Methode**](/windows/win32/api/gdipluspath/nf-gdipluspath-graphicspath-getpathpoints(outpoint_inint)) ruft das Array von Punkten dieses Pfads ab. Das -Array enthält die Endpunkte und Steuerungspunkte der Linien und Bézier-Splines, die zum Zeichnen des Pfads verwendet werden.<br/>  |
| [**GetPathPoints(PointF, \* INT)**](/previous-versions//ms535582(v=vs.85)) | Die [**GraphicsPath::GetPathPoints-Methode**](/previous-versions//ms535582(v=vs.85)) ruft das Array von Punkten dieses Pfads ab. Das -Array enthält die Endpunkte und Steuerungspunkte der Linien und Bézier-Splines, die zum Zeichnen des Pfads verwendet werden.<br/> |
