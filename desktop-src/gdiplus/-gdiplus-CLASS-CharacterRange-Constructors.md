# <a name="characterrangecharacterrange-constructors"></a>CharacterRange.CharacterRange-Konstruktoren

In diesem Thema werden die Konstruktoren der [**CharacterRange-Klasse**](/windows/win32/api/gdiplustypes/nl-gdiplustypes-characterrange) aufgeführt. Eine vollständige Klassenauflistung finden Sie unter **CharacterRange-Klasse**.

### <a name="overload-list"></a>Überladeliste



| Konstruktor                                                                                   | BESCHREIBUNG                                                                                                                                                                                |
|:----------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CharacterRange()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-characterrange-characterrange-r2)                     | Erstellt ein [**CharacterRange-Objekt,**](/windows/win32/api/gdiplustypes/nl-gdiplustypes-characterrange) bei dem die Datenmitglieder auf 0 (null) festgelegt sind.<br/>                                     |
| [**CharacterRange(INT,INT)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-characterrange-characterrange(int_int)) | Erstellt ein [**CharacterRange-Objekt**](/windows/win32/api/gdiplustypes/nl-gdiplustypes-characterrange) und initialisiert die Datenmitglieder mit den angegebenen Werten.<br/> |
