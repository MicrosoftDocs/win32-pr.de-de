# <a name="rectfrectf-constructors"></a>RectF. RectF-Konstruktoren

In diesem Thema werden die Konstruktoren der [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) -Klasse aufgelistet. Eine komplette Klassen Auflistung finden Sie unter **RectF Class**.

### <a name="overload-list"></a>Überladeliste



| Konstruktor                                                                        | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                     |
|:-----------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**RectF (PointF&, SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inconstpointf__inconstsizef_))         | Erstellt ein [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) -Objekt mithilfe eines [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) -Objekts, um die **X** -und **Y** -Datenmember zu initialisieren, und verwendet ein [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) -Objekt, um die Datenmember für die **Breite** und **Höhe** dieses Rechtecks zu initialisieren.<br/> |
| [**RectF (Real, Real, Real, Real)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inreal_inreal_inreal_inreal)) | Erstellt ein [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) -Objekt mit vier ganzen Zahlen, um die Datenmember **X**, **Y**, **Width** und **height** zu initialisieren.<br/>                                                                                                                                                          |
| [**RectF ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inconstpointf__inconstsizef_))                                     | Erstellt ein [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) -Objekt und initialisiert die Datenmember **X**, **Y**, **Width** und **height** auf 0 (null). Dies ist der Standardkonstruktor<br/>                                                                                                                                      |
