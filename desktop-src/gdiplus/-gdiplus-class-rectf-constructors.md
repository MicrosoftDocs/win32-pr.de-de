# <a name="rectfrectf-constructors"></a>RectF.RectF-Konstruktoren

In diesem Thema werden die Konstruktoren der [**RectF-Klasse**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) aufgeführt. Eine vollständige Klassenauflistung finden Sie unter **RectF-Klasse**.

### <a name="overload-list"></a>Überladeliste



| Konstruktor                                                                        | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                     |
|:-----------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**RectF(PointF&,SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inconstpointf__inconstsizef_))         | Erstellt ein [**RectF-Objekt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) mithilfe eines [**PointF-Objekts,**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) um die **X-** und **Y-Datenmitglieder** zu initialisieren, und verwendet ein [**SizeF-Objekt,**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) um die **Width-** und **Height-Datenmitglieder** dieses Rechtecks zu initialisieren.<br/> |
| [**RectF(REAL,REAL,REAL,REAL)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inreal_inreal_inreal_inreal)) | Erstellt ein [**RectF-Objekt,**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) indem vier ganze Zahlen verwendet werden, um die **Datenmitglieder X,** **Y,** **Width** und **Height** zu initialisieren.<br/>                                                                                                                                                          |
| [**RectF()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inconstpointf__inconstsizef_))                                     | Erstellt ein [**RectF-Objekt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) und initialisiert die **X-,** **Y-,** **Width-** und **Height-Datenmitglieder** auf 0 (null). Dies ist der Standardkonstruktor<br/>                                                                                                                                      |
