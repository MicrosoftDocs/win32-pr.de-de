# <a name="rectfcontains-methods"></a>RectF.Contains-Methoden

In diesem Thema werden die Contains-Methoden der [**RectF-Klasse**](/windows/win32/api/gdiplustypes/nl-gdiplustypes-rectf) aufgeführt. Eine vollständige Liste der Methoden für die **RectF-Klasse** finden Sie unter [RectF-Methoden.](-gdiplus-class-rectf-methods.md)

### <a name="overload-list"></a>Überladeliste



| Methode                                                            | Beschreibung                                                                                                                                           |
|:------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Contains(RectF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstrectf_))   | Die [**RectF::Contains-Methode**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstrectf_)) bestimmt, ob sich ein anderes Rechteck innerhalb dieses Rechtecks befindet.<br/>   |
| [**Contains(PointF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstpointf_))    | Die [**RectF::Contains-Methode**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstpointf_)) bestimmt, ob sich ein Punkt in diesem Rechteck befindet.<br/>               |
| [**Contains(REAL,REAL)**](/previous-versions//ms534956(v=vs.85)) | Die [**RectF::Contains-Methode**](/previous-versions//ms534956(v=vs.85)) bestimmt, ob sich der Punkt (*x*, *y*) innerhalb dieses Rechtecks befindet.<br/> |
