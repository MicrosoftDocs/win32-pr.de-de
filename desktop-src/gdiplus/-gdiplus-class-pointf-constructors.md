# <a name="pointfpointf-constructors"></a>PointF.PointF-Konstruktoren

In diesem Thema werden die Konstruktoren der [**PointF-Klasse**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) aufgeführt. Eine vollständige Klassenauflistung finden Sie unter **PointF-Klasse**.

### <a name="overload-list"></a>Überladeliste



| Konstruktor                                                    | BESCHREIBUNG                                                                                                                                                                       |
|:---------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PointF(SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inconstsizef_))   | Erstellt ein [**PointF-Objekt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) mithilfe eines [**SizeF-Objekts,**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) um die **X-** und **Y-Datenmitglieder** anzugeben.<br/> |
| [**PointF(PointF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inconstpointf_)) | Erstellt ein neues [**PointF-Objekt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) und kopiert die Daten aus einem **anderen PointF-Objekt.**<br/>                                                 |
| [**PointF(REAL,REAL)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inreal_inreal)) | Erstellt ein [**PointF-Objekt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) mit zwei echten Zahlen, um die **X-** und **Y-Datenmitglieder** anzugeben.<br/>                                     |
| [**PointF()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf)              | Erstellt ein [**PointF-Objekt**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) und initialisiert die **X-** und **Y-Datenmitglieder** auf 0 (null). Dies ist der Standardkonstruktor<br/>              |
