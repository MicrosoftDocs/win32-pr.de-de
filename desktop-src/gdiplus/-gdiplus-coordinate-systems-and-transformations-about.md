# <a name="coordinate-systems-and-transformations"></a>Koordinatensysteme und Transformationen

Windows GDI+ bietet eine globale Transformation und eine Seiten Transformation, sodass Sie die von Ihnen gezeichneten Elemente transformieren (drehen, skalieren, übersetzen usw.) können. Mit den beiden Transformationen können Sie auch in einer Vielzahl von Koordinatensystemen arbeiten.

-   [Globale und lokale Transformationen](-gdiplus-global-and-local-transformations-about.md)
