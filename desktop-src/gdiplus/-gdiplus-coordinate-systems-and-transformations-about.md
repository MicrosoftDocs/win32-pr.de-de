# <a name="coordinate-systems-and-transformations"></a>Koordinatensysteme und Transformationen

Windows GDI+ bietet eine Welttransformation und eine Seitentransformation, sodass Sie die elemente, die Sie zeichnen, transformieren (drehen, skalieren, übersetzen und so weiter) transformieren können. Mit den beiden Transformationen können Sie auch in einer Vielzahl von Koordinatensystemen arbeiten.

-   [Matrixdarstellung von Transformationen](-gdiplus-matrix-representation-of-transformations-about.md)
-   [Globale und lokale Transformationen](-gdiplus-global-and-local-transformations-about.md)
