# <a name="introduction-to-gdi"></a>Einführung in GDI+

Windows GDI+ ist eine Grafikgeräte Schnittstelle, mit der Programmierer geräteunabhängige Anwendungen schreiben können. Die Dienste von GDI+ werden durch eine Reihe von C++-Klassen verfügbar gemacht.
