# <a name="image-frame-dimension-constants"></a>Bild Frame-Dimensions Konstanten

Die Bild Dateiformate Graphics Interchange Format (GIF) und Tagged Image File Format (TIFF) ermöglichen es Ihnen, mehrere Frames in einer einzelnen Bilddatei zu speichern. Mehrere Frames in einer GIF-Datei werden für die Animation verwendet, sodass die Frames als Zeitdimension bezeichnet werden. Mehrere Frames in einer TIFF-Datei werden normalerweise als separate Seiten verwendet, sodass sich die Frames in der Seiten Dimension befinden.

Die folgenden Konstanten, die in "gdiplusimaging. h" definiert sind, stellen GUIDs dar, die die Seiten-und Zeitdimensionen identifizieren.

-   Framedimensionpage
-   Framedimensiontime
