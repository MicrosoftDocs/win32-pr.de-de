# <a name="image-frame-dimension-constants"></a>Bildrahmendimensionskonstanten

Mit den Bilddateiformaten Graphics Interchange Format (GIF) und Tagged Image File Format (TIFF) können Sie mehrere Frames in einer einzelnen Bilddatei speichern. Mehrere Frames in einer GIF-Datei werden für Animationen verwendet, sodass die Frames in der Zeitdimension angegeben werden. Mehrere Frames in einer TIFF-Datei werden in der Regel als separate Seiten verwendet, sodass die Frames in der Seitendimension angegeben werden.

Die folgenden Konstanten, die in Gdiplusimaging.h definiert sind, stellen GUIDs dar, die die Seiten- und Zeitdimensionen identifizieren.

-   FrameDimensionPage
-   FrameDimensionTime
