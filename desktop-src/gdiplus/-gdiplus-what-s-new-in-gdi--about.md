# <a name="whats-new-in-gdi"></a>Neues in GDI+

Windows GDI+ unterscheidet sich auf verschiedene Weise von Windows Graphics Device Interface (GDI). Zuerst erweitert GDI+ die Features von GDI durch die Bereitstellung neuer Funktionen, wie z. b. Farbverlaufs Pinsel und Alpha Blending. Zweitens wurde das Programmiermodell überarbeitet, um die Grafik Programmierung zu vereinfachen und flexibler zu gestalten.

-   [Neue Funktionen](-gdiplus-new-features-about.md)
-   [Änderungen am Programmiermodell](-gdiplus-changes-in-the-programming-model-about.md)
