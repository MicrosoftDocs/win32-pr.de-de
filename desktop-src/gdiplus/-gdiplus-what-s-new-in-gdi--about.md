# <a name="whats-new-in-gdi"></a>Neues in GDI+?

Windows GDI+ von Windows Graphics Device Interface (GDI) auf verschiedene Weise. Erstens GDI+ die Features von GDI erweitert, indem neue Funktionen wie Farbverlaufspinsel und Alphablending zur Verfügung stehen. Zweitens wurde das Programmiermodell überarbeitet, um die Grafikprogrammierung einfacher und flexibler zu gestalten.
