# <a name="matrixmatrix-constructors"></a>Matrix.Matrix-Konstruktoren

In diesem Thema werden die Konstruktoren der [**Matrix-Klasse**](/windows/desktop/api/gdiplusmatrix/nl-gdiplusmatrix-matrix) aufgelistet. Eine vollständige Klassenauflistung finden Sie unter **Matrixklasse.**

### <a name="overload-list"></a>Überladeliste



| Konstruktor                                                                                          | BESCHREIBUNG                                                                                                                                                                          |
|:-----------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Matrix(Rect&,Point \* )**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(inconstrect__inconstpoint))                | Erstellt ein [**Matrix::Matrix-Objekt**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(inconstrect__inconstpoint)) basierend auf einem Rechteck und einem Punkt.<br/>                                         |
| [**Matrix(RectF&,PointF \* )**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(inconstrectf__inconstpointf))            | Erstellt ein [**Matrix::Matrix-Objekt**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(inconstrectf__inconstpointf)) basierend auf einem Rechteck und einem Punkt.<br/>                                       |
| [**Matrix (REAL, REAL, REAL, REAL, REAL, REAL, REAL)**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(inreal_inreal_inreal_inreal_inreal_inreal)) | Erstellt und initialisiert ein [**Matrix::Matrix-Objekt**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(inreal_inreal_inreal_inreal_inreal_inreal)) basierend auf sechs Zahlen, die eine affine Transformation definieren.<br/> |
| [**Matrix()**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(constmatrix_))                                                    | Erstellt und initialisiert ein [**Matrix::Matrix-Objekt,**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(constmatrix_)) das die Identitätsmatrix darstellt.<br/>                                             |
