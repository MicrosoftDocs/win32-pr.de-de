# <a name="updating-windows-update-agent-header-files"></a>Aktualisieren von Windows Update-Agent-Header Dateien

Die APIs für den Windows Update-Agent waren zuvor ein separater Download. Sie sind jetzt als Teil der Windows SDK enthalten. Um das Windows SDK herunterzuladen, wechseln Sie zu [dieser Seite](https://developer.microsoft.com/windows/downloads/windows-10-sdk).

Weitere Informationen zum Aktualisieren von WUA finden Sie unter [Aktualisieren von Windows Update-Agent](updating-the-windows-update-agent.md).
