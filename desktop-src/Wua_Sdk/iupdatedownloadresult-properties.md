# <a name="iupdatedownloadresult-properties"></a>Iupdatedownloadresult-Eigenschaften

Die [**iupdatedownloadresult**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadresult) -Schnittstelle definiert die folgenden Eigenschaften.



| Eigenschaft                                               | BESCHREIBUNG                                                                                                                          |
|--------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| [**HRESULT**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadresult-get_hresult)       | Ruft den Ausnahme- **HRESULT** -Wert ab, sofern vorhanden, der während eines Vorgangs bei einem Update ausgelöst wird.                                       |
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadresult-get_resultcode) | Ruft einen [**operationresultcode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) -Enumerationswert ab, der das Ergebnis eines Vorgangs bei einem Update angibt. |
