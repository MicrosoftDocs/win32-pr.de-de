# <a name="iupdatedownloadresult-properties"></a>IUpdateDownloadResult-Eigenschaften

Die [**IUpdateDownloadResult-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadresult) definiert die folgenden Eigenschaften.



| Eigenschaft                                               | Beschreibung                                                                                                                          |
|--------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| [**Hresult**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadresult-get_hresult)       | Ruft ggf. den **HRESULT-Ausnahmewert** ab, der während eines Vorgangs bei einem Update ausgelöst wird.                                       |
| [**Resultcode**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadresult-get_resultcode) | Ruft einen [**OperationResultCode-Enumerationswert**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) ab, der das Ergebnis eines Vorgangs bei einem Update angibt. |
