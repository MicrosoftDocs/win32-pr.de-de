# <a name="iwindowsdriverupdate-properties"></a>Iwindowsdriverupdate-Eigenschaften

Die [**iwindowsdriverupdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) -Schnittstelle definiert die folgenden Eigenschaften.



| Eigenschaft                                                                                                    | BESCHREIBUNG                                                                                           |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [**Deviceproblemnumber**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_deviceproblemnumber)-[**Adresse**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_address) | Ruft die Problem Nummer des Geräts ab, das mit dem Windows-Treiberupdate übereinstimmt.                     |
| [**DeviceStatus**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_devicestatus)                                                   | Ruft den Status des Geräts ab, das mit dem Windows-Treiberupdate übereinstimmt.                             |
| [**Driverclass**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverclass)                                                     | Ruft die Klasse des Windows-Treiber Updates ab.                                                          |
| [**Driverhardwareid**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverhardwareid)                                           | Ruft die Hardware-ID oder die kompatible ID ab, mit der das Windows-Treiberupdate installiert werden muss.    |
| [**Driverhersteller**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermanufacturer)                                       | Ruft den sprach invarianten Namen des Herstellers des Windows-Treiber Updates ab.                    |
| [**Drivermodel**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermodel)                                                     | Ruft den sprach invarianten Modellnamen des Geräts ab, für das das Windows-Treiberupdate vorgesehen ist. |
| [**Driverprovider**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverprovider)                                               | Ruft den sprach invarianten Namen des Anbieters des Windows-Treiber Updates ab.                        |
| [**Driverdatum**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driververdate)                                                 | Ruft das Treiber Versions Datum des Windows-Treiber Updates ab.                                            |
