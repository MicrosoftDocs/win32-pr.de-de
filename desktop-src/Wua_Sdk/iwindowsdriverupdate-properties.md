# <a name="iwindowsdriverupdate-properties"></a>IWindowsDriverUpdate-Eigenschaften

Die [**IWindowsDriverUpdate-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) definiert die folgenden Eigenschaften.



| Eigenschaft                                                                                                    | BESCHREIBUNG                                                                                           |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [**DeviceProblemNumber-Adresse**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_deviceproblemnumber)[](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_address) | Ruft die Problemnummer des Geräts ab, das mit dem Windows Treiberupdate übereinstimmt.                     |
| [**DeviceStatus**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_devicestatus)                                                   | Ruft den Status des Geräts ab, das mit dem Windows Treiberupdate übereinstimmt.                             |
| [**DriverClass**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverclass)                                                     | Ruft die Klasse des Windows Treiberupdates ab.                                                          |
| [**DriverHardwareID**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverhardwareid)                                           | Ruft die Hardware-ID oder kompatible ID ab, mit der das Windows Treiberupdate übereinstimmen muss, damit es installiert werden kann.    |
| [**DriverManufacturer**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermanufacturer)                                       | Ruft den sprachinvarianten Namen des Herstellers des Windows Treiberupdates ab.                    |
| [**DriverModel**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermodel)                                                     | Ruft den sprachinvarianten Modellnamen des Geräts ab, für das das Windows Treiberupdate vorgesehen ist. |
| [**DriverProvider**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverprovider)                                               | Ruft den sprachinvarianten Namen des Anbieters des Windows Treiberupdates ab.                        |
| [**DriverVerDate**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driververdate)                                                 | Ruft das Datum der Treiberversion des Windows Treiberupdates ab.                                            |
