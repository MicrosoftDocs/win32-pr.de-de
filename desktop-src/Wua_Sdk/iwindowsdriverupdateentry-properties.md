# <a name="iwindowsdriverupdateentry-properties"></a>IWindowsDriverUpdateEntry-Eigenschaften

Die [**IWindowsDriverUpdateEntry-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdateentry) definiert die folgenden Eigenschaften.



| Eigenschaft                                                                     | Beschreibung                                                                                                          |
|------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**DeviceProblemNumber**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_deviceproblemnumber) | Ruft die Problemnummer des Geräts ab, das mit dem Update des Windows Treibers übereinstimmt.                                    |
| [**DeviceStatus**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_devicestatus)               | Ruft den Status des Geräts ab, das mit dem Windows Treiberupdate übereinstimmt.                                            |
| [**DriverClass**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverclass)                 | Ruft die Klasse des Windows Treiberupdates ab.                                                                         |
| [**DriverHardwareID**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverhardwareid)       | Ruft die Hardware oder den kompatiblen Bezeichner ab, mit dem das Windows Treiberupdate übereinstimmen muss, damit es installiert werden kann. |
| [**DriverManufacturer**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_drivermanufacturer)   | Ruft den sprachinvarianten Namen des Herstellers des Windows Treiberupdates ab.                                   |
| [**DriverModel**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_drivermodel)                 | Ruft den sprachinvarianten Modellnamen des Geräts ab, für das das Windows Treiberupdate vorgesehen ist.                |
| [**DriverProvider**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverprovider)           | Ruft den sprachinvarianten Namen des Anbieters des Windows Treiberupdates ab.                                       |
| [**DriverVerDate**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driververdate)             | Ruft das Treiberversionsdatum des Windows Treiberupdates ab.                                                           |
