# <a name="iautomaticupdatessettings2-properties"></a>IAutomaticUpdatesSettings2-Eigenschaften

Die [**IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2) -Schnittstelle definiert die folgende Eigenschaft.



| Eigenschaft                                                                                  | BESCHREIBUNG                                                                                                                                                         |
|-------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IncludeRecommendedUpdates**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings2-get_includerecommendedupdates) | Ruft einen booleschen Wert ab, der angibt, ob bei der Suche nach Updates und der Installation von Updates optionale oder empfohlene Updates eingeschlossen werden sollen, oder legt diesen fest. |



 

Informationen zu den Membern, die von dieser Schnittstelle geerbt werden, finden Sie in der folgenden Schnittstelle.

-   [**Iautomaticupdatessettings**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
