# <a name="iinstallationprogresschangedcallbackargs-properties"></a>Iinstallationprogresschangedcallbackargs-Eigenschaften

Die [**iinstallationprogresschangedcallbackargs**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogresschangedcallbackargs) -Schnittstelle definiert die folgende Eigenschaft.



| Eigenschaft                                                              | BESCHREIBUNG                                                                                                                        |
|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [**Fortschritt**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogresschangedcallbackargs-get_progress) | Ruft eine Schnittstelle ab, die den Fortschritt der asynchronen Installation oder der Deinstallation zum Zeitpunkt der Rückruf Ausführung enthält. |
