# <a name="iinstallationprogresschangedcallback-methods"></a>IInstallationProgressChangedCallback-Methoden

Die [**IInstallationProgressChangedCallback-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogresschangedcallback) definiert die folgende Methode.



| Methode                                                        | BESCHREIBUNG                                                                                                                                                                                                                                                                                           |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Invoke**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogresschangedcallback-invoke) | Behandelt die Benachrichtigung über die Statusänderung einer asynchronen Installation oder Deinstallation, die durch einen Aufruf der [**IUpdateInstaller.BeginInstall-**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-begininstall) oder [**IUpdateInstaller.BeginUninstall-Methode**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-beginuninstall) initiiert wurde. |
