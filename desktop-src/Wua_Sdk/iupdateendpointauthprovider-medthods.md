# <a name="iupdateendpointauthprovider-methods"></a>Iupdateendpointauthprovider-Methoden

Die [**iupdateendpointauthprovider**](iupdateendpointauthprovider.md) -Schnittstelle definiert die folgenden Methoden.



| Methode                                                                                             | BESCHREIBUNG                                                                         |
|----------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**Getendpointtoken**](iupdateendpointauthprovider-getendpointtoken.md)                           | Fordern Sie ein Token für den Endpunkt des Diensts mit den angegebenen Anmelde Informationen an.    |
| [**Getpreferredendpointpokentype**](iupdateendpointauthprovider-getpreferredendpointtokentype.md) | Gibt den bevorzugten Authentifizierungstyp für den Endpunkt des Diensts zurück. |
