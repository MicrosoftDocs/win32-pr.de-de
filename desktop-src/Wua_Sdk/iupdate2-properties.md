# <a name="iupdate2-properties"></a>IUpdate2-Eigenschaften

Die [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2) -Schnittstelle definiert die folgenden Eigenschaften.



| Eigenschaft                                          | BESCHREIBUNG                                                                                                                                             |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Cveids**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_cveids)                 | Ruft eine Auflistung allgemeiner CVE-IDs (Sicherheitsanfälligkeiten und Exposures) ab, die mit dem Update verknüpft sind.                                                |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_ispresent)           | Ruft einen booleschen Wert ab, der angibt, ob ein Update auf einem Computer vorhanden ist.                                                                         |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_rebootrequired) | Ruft einen booleschen Wert ab, der angibt, ob auf einem Computer ein Systemneustart erforderlich ist, um die Installation oder die Installation eines Updates abzuschließen. |



 

Informationen zu den Membern, die von dieser Schnittstelle geerbt werden, finden Sie in der folgenden Schnittstelle:

-   [**Iupdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
