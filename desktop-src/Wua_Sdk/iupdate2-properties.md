# <a name="iupdate2-properties"></a>IUpdate2-Eigenschaften

Die [**IUpdate2-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2) definiert die folgenden Eigenschaften.



| Eigenschaft                                          | BESCHREIBUNG                                                                                                                                             |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CveIDs**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_cveids)                 | Ruft eine Auflistung gängiger CVE-IDs (Vulnerabilities and Exposures) ab, die dem Update zugeordnet sind.                                                |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_ispresent)           | Ruft einen booleschen Wert ab, der angibt, ob ein Update auf einem Computer vorhanden ist.                                                                         |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_rebootrequired) | Ruft einen booleschen Wert ab, der angibt, ob ein Systemneustart auf einem Computer erforderlich ist, um die Installation oder Deinstallation eines Updates abzuschließen. |



 

Informationen zu den von dieser Schnittstelle geerbten Membern finden Sie auf der folgenden Schnittstelle:

-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
