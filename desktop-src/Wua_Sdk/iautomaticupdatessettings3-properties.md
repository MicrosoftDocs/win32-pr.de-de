# <a name="iautomaticupdatessettings3-properties"></a>IAutomaticUpdatesSettings3-Eigenschaften

Die [**IAutomaticUpdatesSettings3-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings3) definiert die folgenden Eigenschaften.



| Eigenschaft                                                                                  | Beschreibung                                                                                                                                     |
|-------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FeaturedUpdatesEnabled**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings3-get_featuredupdatesenabled)       | Ruft einen booleschen Wert ab, der angibt, ob Benachrichtigungen für ausgewählte Updates angezeigt werden sollen, und legt diesen fest.                                             |
| [**NonAdministratorsElevated**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings3-get_nonadministratorselevated) | Ruft einen booleschen Wert ab, der angibt, ob Nichtadministratoren einige updatebezogene Aktionen ohne Genehmigung durch den Administrator ausführen können, und legt diesen fest. |



 

Informationen zu den Membern, die von dieser Schnittstelle geerbt werden, finden Sie in den folgenden Schnittstellen.

-   [**IAutomaticUpdatesSettings**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
-   [**IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2)
