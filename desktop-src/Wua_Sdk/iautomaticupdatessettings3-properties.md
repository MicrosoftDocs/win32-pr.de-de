# <a name="iautomaticupdatessettings3-properties"></a>IAutomaticUpdatesSettings3-Eigenschaften

Die [**IAutomaticUpdatesSettings3**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings3) -Schnittstelle definiert die folgenden Eigenschaften.



| Eigenschaft                                                                                  | BESCHREIBUNG                                                                                                                                     |
|-------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Featuredupdatesaktivierte**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings3-get_featuredupdatesenabled)       | Ruft einen booleschen Wert ab, der angibt, ob Benachrichtigungen für die vorgestellten Updates angezeigt werden sollen.                                             |
| [**NonAdministratorsElevated**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings3-get_nonadministratorselevated) | Ruft einen booleschen Wert ab, der angibt, ob nicht-Administratoren einige Aktualisierungs bezogene Aktionen ohne Genehmigung durch den Administrator ausführen können, oder legt diesen fest. |



 

Informationen zu den Membern, die von dieser Schnittstelle geerbt werden, finden Sie unter den folgenden Schnittstellen.

-   [**Iautomaticupdatessettings**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
-   [**IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2)
