# <a name="iinstallationjob-methods"></a>Iinstallationjob-Methoden

Die [**iinstallationjob**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationjob) -Schnittstelle definiert die folgenden Methoden.



| Methode                                                | BESCHREIBUNG                                                                                                                                           |
|-------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Bereinigung**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationjob-cleanup)           | Wartet, bis ein asynchroner Vorgang abgeschlossen ist, und gibt dann alle Rückrufe frei.                                                              |
| [**GetProgress**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationjob-getprogress)   | Gibt eine [**iinstallationprogress**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogress) -Schnittstelle zurück, die den aktuellen Status einer Installation oder der Uninstallation beschreibt. |
| [**Requestabort**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationjob-requestabort) | Stellt eine Anforderung zum Abbrechen der Installation oder der Installation von bereit.                                                                                         |
