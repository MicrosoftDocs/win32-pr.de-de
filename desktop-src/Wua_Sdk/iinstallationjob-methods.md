# <a name="iinstallationjob-methods"></a>IInstallationJob-Methoden

Die [**IInstallationJob-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationjob) definiert die folgenden Methoden.



| Methode                                                | BESCHREIBUNG                                                                                                                                           |
|-------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Bereinigung**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationjob-cleanup)           | Wartet auf den Abschluss eines asynchronen Vorgangs und gibt dann alle Rückrufe frei.                                                              |
| [**GetProgress**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationjob-getprogress)   | Gibt eine [**IInstallationProgress-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogress) zurück, die den aktuellen Fortschritt einer Installation oder Deinstallation beschreibt. |
| [**RequestAbort**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationjob-requestabort) | Stellt eine Anforderung zum Abbrechen der Installation oder Deinstallation.                                                                                         |
