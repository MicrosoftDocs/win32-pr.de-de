# <a name="iautomaticupdates-properties"></a>Iautomaticupdates-Eigenschaften

Die [**iautomaticupdates**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates) -Schnittstelle definiert die folgenden Eigenschaften.



| Eigenschaft                                                   | BESCHREIBUNG                                                                                                   |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**Servicefähig**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_serviceenabled) | Ruft einen booleschen Wert ab, der angibt, ob alle automatische Updates erforderlichen Komponenten verfügbar sind. |
| [**Einstellungen**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_settings)             | Ruft die Konfigurationseinstellungen für automatische Updates ab.                                                        |
