# <a name="iautomaticupdates-properties"></a>IAutomaticUpdates-Eigenschaften

Die [**IAutomaticUpdates-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates) definiert die folgenden Eigenschaften.



| Eigenschaft                                                   | Beschreibung                                                                                                   |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**ServiceEnabled**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_serviceenabled) | Ruft einen booleschen Wert ab, der angibt, ob alle Komponenten verfügbar sind, Automatische Updates erforderlich sind. |
| [**Einstellungen**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_settings)             | Ruft die Konfigurationseinstellungen für Automatische Updates.                                                        |
