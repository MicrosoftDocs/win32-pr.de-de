# <a name="isearchjob-properties"></a>ISearchJob-Eigenschaften

Die [**ISearchJob-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-isearchjob) definiert die folgenden Eigenschaften.



| Eigenschaft                                      | Beschreibung                                                                                                                                                 |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AsyncState**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-get_asyncstate)   | Ruft das aufruferspezifische Zustandsobjekt ab, das an die [**IUpdateSearch.BeginSearch-Methode**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch) übergeben wird.                         |
| [**IsCompleted**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-get_iscompleted) | Ruft einen booleschen Wert ab, der angibt, ob der Aufruf der [**IUpdateSearch.BeginSearch-Methode**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch) vollständig verarbeitet wird. |
