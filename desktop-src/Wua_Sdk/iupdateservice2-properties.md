# <a name="iupdateservice2-properties"></a>IUpdateService2-Eigenschaften

Die **IUpdateService2** -Schnittstelle definiert die folgende Eigenschaft.



| Eigenschaft                                                         | BESCHREIBUNG                                                                                                                                                                         |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Isdefaultauservice**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice2-get_isdefaultauservice) | Ruft einen booleschen Wert ab, der angibt, ob der Dienst bei automatische Updates registriert ist und ob der Dienst zurzeit von Automatische Updates als Standard Dienst verwendet wird. |



 

Informationen zu den Membern, die von dieser Schnittstelle geerbt werden, finden Sie in der folgenden Schnittstelle.

-   [**Iupdateservice**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservice)
