# <a name="istringcollection-methods"></a>Istringcollection-Methoden

Die [**istringcollection**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection) -Schnittstelle definiert die folgenden Methoden.



| Methode                                         | BESCHREIBUNG                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**Eren**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-add)           | Fügt der Auflistung ein Element hinzu.                                |
| [**Klartext**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-clear)       | Entfernt alle Elemente aus der Auflistung.                  |
| [**Kopieren**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-copy)         | Erstellt eine Tiefe Lese-/Schreibkopie der Auflistung.              |
| [**Setze**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-insert)     | Fügt ein Element an der angegebenen Position in die Auflistung ein. |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-removeat) | Entfernt das Element am angegebenen Index aus der Auflistung.   |
