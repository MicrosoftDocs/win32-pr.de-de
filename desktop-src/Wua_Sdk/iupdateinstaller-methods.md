# <a name="iupdateinstaller-methods"></a>IUpdateInstaller-Methoden

Die [**IUpdateInstaller-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller) definiert die folgenden Methoden.



| Methode                                                    | BESCHREIBUNG                                                                          |
|-----------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**BeginInstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-begininstall)     | Startet eine asynchrone Installation der Updates.                                  |
| [**BeginUninstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-beginuninstall) | Startet eine asynchrone Deinstallation der Updates.                                |
| [**EndInstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-endinstall)         | Schließt eine asynchrone Installation der Updates ab.                               |
| [**EndUninstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-enduninstall)     | Schließt eine asynchrone Deinstallation der Updates ab.                             |
| [**Installieren**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-install)               | Startet eine synchrone Installation der Updates.                                    |
| [**RunWizard**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-runwizard)           | Startet einen Assistenten, der den lokalen Benutzer durch die Schritte zum Installieren der Updates führt. |
| [**Deinstallieren**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-uninstall)           | Startet eine synchrone Deinstallation der Updates.                                  |
