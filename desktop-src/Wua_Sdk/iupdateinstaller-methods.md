# <a name="iupdateinstaller-methods"></a>Iupdateingestaller-Methoden

Die [**iupdaterstaller**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller) -Schnittstelle definiert die folgenden Methoden.



| Methode                                                    | BESCHREIBUNG                                                                          |
|-----------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**Begininstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-begininstall)     | Startet eine asynchrone Installation der Updates.                                  |
| [**Beginuninstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-beginuninstall) | Startet eine asynchrone Installation der Updates.                                |
| [**Umdinstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-endinstall)         | Schließt eine asynchrone Installation der Updates ab.                               |
| [**Enduninstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-enduninstall)     | Schließt eine asynchrone Installation der Updates ab.                             |
| [**Installieren**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-install)               | Startet eine synchrone Installation der Updates.                                    |
| [**Runwizard**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-runwizard)           | Hiermit wird ein Assistent gestartet, der den lokalen Benutzer durch die Schritte zum Installieren der Updates führt. |
| [**Deinstallieren**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-uninstall)           | Startet eine synchrone Neuinstallation der Updates.                                  |
