# <a name="idownloadjob-methods"></a>IDownloadJob-Methoden

Die [**IDownloadJob-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadjob) definiert die folgenden Methoden.



| Methode                                            | Beschreibung                                                                                                            |
|---------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**Bereinigung**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadjob-cleanup)           | Wartet auf den Abschluss eines asynchronen Vorgangs und gibt alle Rückrufe frei.                                        |
| [**GetProgress**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadjob-getprogress)   | Gibt eine [**IDownloadProgress-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadprogress) zurück, die den aktuellen Fortschritt eines Downloads beschreibt. |
| [**RequestAbort**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadjob-requestabort) | Stellt eine Anforderung zum Beenden eines asynchronen Downloads.                                                                       |
