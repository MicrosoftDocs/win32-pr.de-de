# <a name="iupdate4-properties"></a>IUpdate4-Eigenschaften

Die [**IUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate4) -Schnittstelle definiert die folgende Eigenschaft.



| Eigenschaft                            | BESCHREIBUNG                                                            |
|-------------------------------------|------------------------------------------------------------------------|
| [**Peruser**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate4-get_peruser) | Ruft einen booleschen Wert ab, der angibt, ob es sich um ein Update pro Benutzer handelt. |



 

Informationen zu den Membern, die von dieser Schnittstelle geerbt werden, finden Sie unter den folgenden Schnittstellen:

-   [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3)
-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**Iupdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
