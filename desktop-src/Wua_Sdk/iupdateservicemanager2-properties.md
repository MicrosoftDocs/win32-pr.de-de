# <a name="iupdateservicemanager2-properties"></a>IUpdateServiceManager2-Eigenschaften

Die [**IUpdateServiceManager2-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager2) definiert die folgende Eigenschaft.



| Eigenschaft                                                                  | Beschreibung                                                     |
|---------------------------------------------------------------------------|-----------------------------------------------------------------|
| [**ClientApplicationID**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicemanager2-get_clientapplicationid) | Ruft den Bezeichner der aktuellen Clientanwendung ab und legt den Bezeichner fest. |



 

Informationen zu den von dieser Schnittstelle geerbten Membern finden Sie auf der folgenden Schnittstelle.

-   [**IUpdateServiceManager**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager)
