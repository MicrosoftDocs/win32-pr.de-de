# <a name="iupdateservice-properties"></a>Iupdateservice-Eigenschaften

Die [**iupdateservice**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservice) -Schnittstelle definiert die folgenden Eigenschaften.



| Eigenschaft                                                              | BESCHREIBUNG                                                                                     |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**Canregisterwithau**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice-get_canregisterwithau)         | Ruft einen booleschen Wert ab, der angibt, ob der Dienst bei automatische Updates registriert werden kann.    |
| [**Contentvalidationcert**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice-get_contentvalidationcert) | Ruft einen SHA-1-Hash des Zertifikats ab, das verwendet wird, um den Inhalt des Dienstanbieter zu signieren.         |
| [**ExpirationDate**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice-get_expirationdate)               | Ruft das Datum ab, an dem die Autorisierungs CAB-Datei abläuft.                                  |
| [**IsManaged**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice-get_ismanaged)                         | Ruft einen booleschen Wert ab, der angibt, ob ein Dienst ein verwalteter Dienst ist.                     |
| [**Isregisteredwithau**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice-get_isregisteredwithau)       | Ruft einen booleschen Wert ab, der angibt, ob ein Dienst bei automatische Updates registriert ist.     |
| [**Isscanpackageservice**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice-get_isscanpackageservice)   | Ruft einen booleschen Wert ab, der angibt, ob ein Dienst auf einem Überprüfungspaket basiert.               |
| [**IssueDate**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice-get_issuedate)                         | Ruft das Datum ab, an dem die Autorisierungs CAB-Datei ausgegeben wurde.                               |
| [**Name**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice-get_name)                                   | Ruft den Namen des Diensts ab.                                                                   |
| [**Offerswindowsupdates**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice-get_offerswindowsupdates)   | Ruft einen booleschen Wert ab, der angibt, ob der aktuelle Dienst Updates von Windows-Updates anbietet. |
| [**Redirecturls**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice-get_redirecturls)                   | Ruft die URL für die Redirector-CAB-Datei ab.                                                   |
| [**ServiceID**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice-get_serviceid)                         | Ruft den Bezeichner für einen Dienst ab.                                                              |
| [**ServiceUrl**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice-get_serviceurl)                       | Ruft die URL für den Dienst ab.                                                                   |
| [**Setupprefix**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice-get_setupprefix)                     | Ruft das Präfix für die Setup Dateien ab.                                                            |
