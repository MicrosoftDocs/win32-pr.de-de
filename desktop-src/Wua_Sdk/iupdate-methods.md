# <a name="iupdate-methods"></a>IUpdate-Methoden

Die [**IUpdate-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate) definiert die folgenden Methoden.



| Methode                                         | Beschreibung                                                                                      |
|------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**AcceptEula**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate-accepteula)       | Akzeptiert die Microsoft-Software-Lizenzbedingungen, die dem Update Windows sind.<br/> |
| [**CopyFromCache**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate-copyfromcache) | Kopiert den Inhalt eines Updates in einen angegebenen Pfad.<br/>                                 |
