# <a name="iupdate-methods"></a>Iupdate-Methoden

Die [**iupdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate) -Schnittstelle definiert die folgenden Methoden.



| Methode                                         | BESCHREIBUNG                                                                                      |
|------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**AcceptEula**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate-accepteula)       | Akzeptiert die Microsoft-Software-Lizenzbedingungen, die Windows Update zugeordnet sind.<br/> |
| [**Copyfromcache**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate-copyfromcache) | Kopiert den Inhalt eines Updates in einen angegebenen Pfad.<br/>                                 |
