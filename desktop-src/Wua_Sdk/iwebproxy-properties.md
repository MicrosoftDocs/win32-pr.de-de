# <a name="iwebproxy-properties"></a>IWebProxy-Eigenschaften

Die [**IWebProxy-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy) definiert die folgenden Eigenschaften.



| Eigenschaft                                                   | Beschreibung                                                                                                               |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**Adresse**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_address)                       | Ruft die Adresse und die Dezimalportnummer des Proxyservers ab und legt diese fest.                                                |
| [**Autodetect**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_autodetect)                 | Ruft einen booleschen Wert ab, der angibt, ob [**IWebProxy**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy) Proxyeinstellungen automatisch erkennt, und legt diesen fest. |
| [**Bypasslist**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_bypasslist)                 | Ruft eine Auflistung von Adressen ab, die den Proxyserver nicht verwenden, und legt diese fest.                                                 |
| [**BypassProxyOnLocal**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_bypassproxyonlocal) | Ruft einen booleschen Wert ab, der angibt, ob lokale Adressen den Proxyserver umgehen, und legt diesen fest.                             |
| [**Readonly**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_readonly)                     | Ruft einen booleschen Wert ab, der angibt, ob das [**WebProxy-Objekt**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy) schreibgeschützt ist.                        |
| [**Nutzername**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_username)                     | Ruft den Benutzernamen ab, der zur Authentifizierung an den Proxyserver übermittelt werden soll, und legt ihn fest.                                             |
