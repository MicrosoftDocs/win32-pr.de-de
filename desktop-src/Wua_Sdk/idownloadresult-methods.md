# <a name="idownloadresult-methods"></a>IDownloadResult-Methoden

Die [**IDownloadResult-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadresult) definiert die folgende Methode.



| Methode                                                     | Beschreibung                                                                                                                              |
|------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetUpdateResult**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadresult-getupdateresult) | Gibt eine [**IUpdateDownloadResult-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadresult) zurück, die Downloadinformationen für das angegebene Update enthält. |
