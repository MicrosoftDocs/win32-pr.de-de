# <a name="iautomaticupdates-methods"></a>IAutomaticUpdates-Methoden

Die [**IAutomaticUpdates-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates) definiert die folgenden Methoden.



| Methode                                                             | Beschreibung                                                         |
|--------------------------------------------------------------------|---------------------------------------------------------------------|
| [**DetectNow**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-detectnow)                   | Startet die automatische Aktualisierung, wenn sie noch nicht gestartet wurde.            |
| [**EnableService**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-enableservice)           | Aktiviert alle Komponenten, die Automatische Updates erforderlich sind.         |
| [**Anhalten**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-pause)                           | Hält die automatische Aktualisierung an.                                          |
| [**Fortsetzen**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-resume)                         | Startet die automatische Aktualisierung neu, wenn sie angehalten ist.                        |
| [**ShowSettingsDialog**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-showsettingsdialog) | Zeigt ein Dialogfeld an, das Einstellungen für Automatische Updates. |
