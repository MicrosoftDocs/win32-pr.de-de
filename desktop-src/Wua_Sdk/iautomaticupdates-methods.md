# <a name="iautomaticupdates-methods"></a>Iautomaticupdates-Methoden

Die [**iautomaticupdates**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates) -Schnittstelle definiert die folgenden Methoden.



| Methode                                                             | BESCHREIBUNG                                                         |
|--------------------------------------------------------------------|---------------------------------------------------------------------|
| [**Detectnow**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-detectnow)                   | Startet die automatische Aktualisierung, wenn Sie nicht bereits gestartet wurde.            |
| [**Enableservice**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-enableservice)           | Aktiviert alle Komponenten, die automatische Updates erfordern.         |
| [**Anhalten**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-pause)                           | Hält die automatische Aktualisierung an.                                          |
| [**Fortsetzen**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-resume)                         | Startet die automatische Aktualisierung neu, wenn Sie angehalten wurde.                        |
| [**Showsettingsdialog**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-showsettingsdialog) | Zeigt ein Dialogfeld an, das Einstellungen für automatische Updates enthält. |
