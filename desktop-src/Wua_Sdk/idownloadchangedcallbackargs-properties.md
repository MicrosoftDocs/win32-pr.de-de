# <a name="idownloadchangedcallbackargs-properties"></a>IDownloadChangedCallbackArgs-Eigenschaften

Die [**IDownloadProgressChangedCallbackArgs-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadprogresschangedcallbackargs) definiert die folgende Eigenschaft.



| Eigenschaft                                                          | Beschreibung                                                                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [**Fortschritt**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadprogresschangedcallbackargs-get_progress) | Ruft eine Schnittstelle ab, die den Fortschritt des asynchronen Downloads zum Zeitpunkt des Rückrufs enthält. |
