# <a name="idownloadchangedcallbackargs-properties"></a>Idownloadchangedcallbackargs-Eigenschaften

Die [**idownloadprogresschangedcallbackargs**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadprogresschangedcallbackargs) -Schnittstelle definiert die folgende Eigenschaft.



| Eigenschaft                                                          | BESCHREIBUNG                                                                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [**Fortschritt**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadprogresschangedcallbackargs-get_progress) | Ruft eine Schnittstelle ab, die den Fortschritt des asynchronen Downloads zum Zeitpunkt der Rückruf Ausführung enthält. |
