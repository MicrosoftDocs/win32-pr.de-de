# <a name="iupdateservicemanager2-methods"></a>IUpdateServiceManager2-Methoden

Die [**IUpdateServiceManager2-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager2) definiert die folgenden Methoden.



| Methode                                                                                      | Beschreibung                                                                                                                                                                                |
|---------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddService2**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicemanager2-addservice2)                           | Registriert einen Dienst bei WUA, ohne dass eine Autorisierungs-Cab-Datei (.cab) erforderlich ist, und gibt einen Zeiger auf eine [**IUpdateServiceRegistration-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicecollection) zurück. |
| [**QueryServiceRegistration**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicemanager2-queryserviceregistration) | Gibt einen Zeiger auf eine [**IUpdateServiceRegistration-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateserviceregistration) zurück.                                                                                        |



 

Informationen zu den Membern, die von dieser Schnittstelle geerbt werden, finden Sie in der folgenden Schnittstelle.

-   [**IUpdateServiceManager**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager)
