# <a name="iupdateservicemanager2-methods"></a>IUpdateServiceManager2-Methoden

Die [**IUpdateServiceManager2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager2) -Schnittstelle definiert die folgenden Methoden.



| Methode                                                                                      | BESCHREIBUNG                                                                                                                                                                                |
|---------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddService2**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicemanager2-addservice2)                           | Registriert einen Dienst bei WUA, ohne dass eine Autorisierungs CAB-Datei (CAB-Datei) erforderlich ist, und gibt einen Zeiger auf eine [**iupdateserviceregistration**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicecollection) -Schnittstelle zurück. |
| [**Queryserviceregistration**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicemanager2-queryserviceregistration) | Gibt einen Zeiger auf eine [**iupdateserviceregistration**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateserviceregistration) -Schnittstelle zurück.                                                                                        |



 

Informationen zu den Membern, die von dieser Schnittstelle geerbt werden, finden Sie in der folgenden Schnittstelle.

-   [**Iupdateservicemanager**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager)
