# <a name="iwindowsdriverupdate2-properties"></a>IWindowsDriverUpdate2-Eigenschaften

Die [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) -Schnittstelle definiert die folgenden Eigenschaften.



| Eigenschaft                                                       | BESCHREIBUNG                                                                                                             |
|----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**Cveids**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_cveids)                 | Enthält eine Sammlung der allgemeinen Bezeichner für Schwachstellen und exposures (CVE), die einem Update zugeordnet sind. |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_ispresent)           | Ruft einen booleschen Wert ab, der angibt, ob ein Update auf dem Computer installiert ist.                                     |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_rebootrequired) | Ruft einen booleschen Wert ab, der angibt, ob der Computer nach der Installation oder Deinstallation eines Updates neu gestartet werden muss.    |



 

Informationen zu den Membern, die von dieser Schnittstelle geerbt werden, finden Sie in der folgenden Schnittstelle.

-   [**Iwindowsdriverupdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
