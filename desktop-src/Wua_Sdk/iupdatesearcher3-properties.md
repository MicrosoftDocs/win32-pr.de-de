# <a name="iupdatesearcher3-methods"></a>IUpdateSearcher3-Methoden

Die [**IUpdateSearcher3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher3) -Schnittstelle definiert die folgende Methode.



| Methode                                              | BESCHREIBUNG                                                                                                                                    |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SearchScope**](/previous-versions/windows/desktop/ee694850(v=vs.85)) | Ruft einen [**SearchScope**](/windows/win32/api/wuapi/ne-wuapi-searchscope) -Enumerationswert ab, der den Gültigkeitsbereich der von der Suche zurückgegebenen Updates angibt, oder legt ihn fest. |



 

Informationen zu den Membern, die von dieser Schnittstelle geerbt werden, finden Sie unter den folgenden Schnittstellen.

-   [**Iupdatesearcher**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher)
-   [**IUpdateSearcher2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher2)
