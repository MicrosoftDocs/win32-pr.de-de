# <a name="iwebproxy-methods"></a>IWebProxy-Methoden

Die [**IWebProxy**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy) -Schnittstelle definiert die folgenden Methoden.



| Methode                                                                         | BESCHREIBUNG                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**Promptfor-Anmelde Informationen**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentials)                 | Fordert den Benutzer zur Eingabe des Kennworts auf, das für die Proxy Authentifizierung verwendet wird.  |
| [**Promptforkredentialsfromhwnd**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentialsfromhwnd) | Fordert den Benutzer zur Eingabe des Kennworts auf, das für die Proxy Authentifizierung verwendet wird.  |
| [**SetPassword**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-setpassword)                                   | Legt das Kennwort fest, das für die Authentifizierung an den Proxy Server gesendet werden soll. |
