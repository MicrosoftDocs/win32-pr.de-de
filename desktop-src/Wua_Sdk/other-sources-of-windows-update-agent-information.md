# <a name="other-sources-of-windows-update-agent-information"></a>Weitere Quellen für Windows Update-Agent-Informationen

Das [Windows Update-Agent-API](portal-client.md)-SDK enthält die umfassendsten und aktuellen Beschreibungen der API. Dies ist die Hauptquelle für Entwicklerreferenz Material für WUA.

## <a name="software-development-kit-sdk-for-windows-update-agent-wua"></a>Software Development Kit (SDK) für Windows Update-Agent (WUA)

In den folgenden Ressourcen wird beschrieben, wie Windows Server Update Services und WUA bereitgestellt und verwaltet werden:

-   [Windows Server Update Services](/documentation/)

    Sie können das Windows Server Update Services SDK herunterladen und andere Dokumente überprüfen, in denen beschrieben wird, wie Sie Windows Update Dienste installieren, bereitstellen und warten.

-   [MSDN-Download und Code Center](/powerapps/developer/common-data-service/org-service/subscribe-sdk-assembly-updates-using-nuget)

    Sie können Updates für die WUA SDK-Beispiele,-Header und-Bibliotheken herunterladen. Downloads im MSDN-Download und im Code Center sind nach Themen organisiert. Mit dem [Microsoft Download Center](https://go.microsoft.com/fwlink/p/?linkid=83468) -Suchtool können Sie auch Downloads auf MSDN, dem Microsoft Developer Network, finden.

## <a name="windows-update-agent-wua-newsgroups"></a>Windows Update-Agent (WUA)-Newsgroups

Sie können Informationen in einer Newsgroup lesen und bereitstellen, die WUA erläutert, oder archivierte Newsgroup-Beiträge durchsuchen.

MSDN bietet Zugriff auf die folgenden öffentlichen Newsgroups:

-   [Öffentliche Newsgroups von Microsoft](https://go.microsoft.com/fwlink/p/?linkid=84395)

    Diese Newsgroup richtet sich an IT-Experten, die Fragen zur Bereitstellung und Anpassung haben.

-   [MSDN-Newsgroups](https://go.microsoft.com/fwlink/p/?linkid=83943)

    Diese Newsgroup ist an Entwickler gerichtet, die Fragen zur Anwendungs-und Infrastrukturentwicklung haben.
