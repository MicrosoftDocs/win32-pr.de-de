# <a name="iwindowsdriverupdate4-properties"></a>IWindowsDriverUpdate4-Eigenschaften

Die [**IWindowsDriverUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate4) -Schnittstelle definiert die folgenden Eigenschaften.



| Eigenschaft                                                                               | BESCHREIBUNG                                                                 |
|----------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**Peruser**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate4-get_peruser)                                       | Ruft einen booleschen Wert ab, der angibt, ob ein Update ein Update pro Benutzer ist. |
| [**Windowsdriverupdateentries**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate4-get_windowsdriverupdateentries) | Ruft die Treiberupdate Einträge ab, die für das Update gelten.          |



 

Informationen zu den Membern, die von dieser Schnittstelle geerbt werden, finden Sie in der folgenden Schnittstelle.

-   [**Iwindowsdriverupdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
-   [**IWindowsDriverUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate3)
