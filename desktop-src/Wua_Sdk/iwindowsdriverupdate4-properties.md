# <a name="iwindowsdriverupdate4-properties"></a>IWindowsDriverUpdate4-Eigenschaften

Die [**IWindowsDriverUpdate4-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate4) definiert die folgenden Eigenschaften.



| Eigenschaft                                                                               | Beschreibung                                                                 |
|----------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**PerUser**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate4-get_peruser)                                       | Ruft einen booleschen Wert ab, der angibt, ob es sich bei einem Update um ein Benutzerupdate handelt. |
| [**WindowsDriverUpdateEntries**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate4-get_windowsdriverupdateentries) | Ruft die Treiberupdateeinträge ab, die für das Update gelten.          |



 

Informationen zu den Membern, die von dieser Schnittstelle geerbt werden, finden Sie in der folgenden Schnittstelle.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
-   [**IWindowsDriverUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate3)
