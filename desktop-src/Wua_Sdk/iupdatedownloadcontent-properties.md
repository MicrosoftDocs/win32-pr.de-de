# <a name="iupdatedownloadcontent-properties"></a>IUpdateDownloadContent-Eigenschaften

Die [**IUpdateDownloadContent-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatecollection) definiert die folgende Eigenschaft.



| Eigenschaft                                                  | Beschreibung                                                                    |
|-----------------------------------------------------------|--------------------------------------------------------------------------------|
| [**DownloadUrl**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontent-get_downloadurl) | Ruft den Speicherort des Downloadinhalts auf dem Server ab, der das Update hostet. |
