# <a name="iupdatedownloadcontent-properties"></a>Iupdatedownloadcontent-Eigenschaften

Die [**iupdatedownloadcontent**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatecollection) -Schnittstelle definiert die folgende Eigenschaft.



| Eigenschaft                                                  | BESCHREIBUNG                                                                    |
|-----------------------------------------------------------|--------------------------------------------------------------------------------|
| [**Download-URL**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontent-get_downloadurl) | Ruft den Speicherort des heruntergeladenen Inhalts auf dem Server ab, der das Update hostet. |
