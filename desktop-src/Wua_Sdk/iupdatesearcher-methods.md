# <a name="iupdatesearcher-methods"></a>Iupdatesearcher-Methoden

Die [**iupdatesearcher**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher) -Schnittstelle definiert die folgenden Methoden.



| Methode                                                              | BESCHREIBUNG                                                                                                               |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**Beginsearch**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch)                   | Startet die Ausführung einer asynchronen Suche nach Updates. Bei der Suche werden die Suchoptionen verwendet, die derzeit konfiguriert sind. |
| [**Endsuche**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-endsearch)                       | Schließt eine asynchrone Suche nach Updates ab.                                                                             |
| [**Escapezeichenfolge**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-escapestring)                 | Konvertiert eine Zeichenfolge in eine Zeichenfolge, die als Literalwert in einer Such Kriterienzeichenfolge verwendet werden kann.                          |
| [**Gettotalhistorycount**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-gettotalhistorycount) | Gibt die Anzahl der Update Ereignisse auf dem Computer zurück.                                                                      |
| [**Queryhistory**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-queryhistory)                | Fragt synchron den Computer nach dem Verlauf von Update Ereignissen ab.                                                      |
| [**Search**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-search)                             | Führt eine synchrone Suche nach Updates aus. Bei der Suche werden die Suchoptionen verwendet, die derzeit konfiguriert sind.              |
