# <a name="iupdateservicemanager-properties"></a>IUpdateServiceManager-Eigenschaften

Die [**IUpdateServiceManager-Schnittstelle**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager) definiert die folgende Eigenschaft.



| Eigenschaft                                           | Beschreibung                                                                                                        |
|----------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**Dienste**](/windows/win32/api/wuapi/nf-wuapi-iupdateservicemanager-get_services) | Ruft eine [**IUpdateServiceCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicecollection) der Dienste ab, die bei WUA registriert sind. |
