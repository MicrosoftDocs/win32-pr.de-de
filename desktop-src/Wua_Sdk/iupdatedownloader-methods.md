# <a name="iupdatedownloader-methods"></a>Iupdatedownloader-Methoden

Die [**iupdatedownloader**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloader) -Schnittstelle definiert die folgenden Methoden.



| Methode                                                   | BESCHREIBUNG                                                                                |
|----------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**BeginDownload**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-begindownload) | Startet einen asynchronen Download der Inhalts Dateien, die mit den Updates verknüpft sind. |
| [**Herunterladen**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-download)           | Startet einen synchronen Download der Inhalts Dateien, die mit den Updates verknüpft sind.   |
| [**EndDownload**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-enddownload)     | Schließt einen asynchronen Download ab.                                                        |
