# <a name="iautomaticupdatesresults-properties"></a>Iautomaticupdatesresults-Eigenschaften

Die [**iautomaticupdatesresults**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatesresults) -Schnittstelle definiert die folgenden Eigenschaften.



| Eigenschaft                                                                                    | BESCHREIBUNG                                                                                                                                             |
|---------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Lastinstallationsuccess Date**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatesresults-get_lastinstallationsuccessdate) | Ruft das Datum der letzten Uhrzeit und der koordinierten Weltzeit (UTC) ab, zu dem automatische Updates Updates erfolgreich installiert haben, auch wenn einige Fehler aufgetreten sind. |
| [**Lastsearchsuccess Date**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatesresults-get_lastsearchsuccessdate)             | Ruft das Datum der letzten Uhrzeit und der koordinierten Weltzeit (UTC) ab, zu dem automatische Updates erfolgreich nach Updates durchsucht wurden.                                   |
