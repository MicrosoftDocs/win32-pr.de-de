# <a name="isysteminformation-properties"></a>Isysteminformation-Eigenschaften

Die [**isysteminformation**](/windows/desktop/api/Wuapi/nn-wuapi-isysteminformation) -Schnittstelle definiert die folgenden Eigenschaften.



| Eigenschaft                                                                    | BESCHREIBUNG                                                                                                                                     |
|-----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Oemhardwaresupportlink**](/windows/desktop/api/Wuapi/nf-wuapi-isysteminformation-get_oemhardwaresupportlink) | Ruft einen Link zu technischen Support Informationen für OEM-Hardware ab.                                                                             |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-isysteminformation-get_rebootrequired)                 | Ruft einen booleschen Wert ab, der angibt, ob ein Systemneustart erforderlich ist, um die Installation oder die Installation von mindestens einem Update abzuschließen. |
