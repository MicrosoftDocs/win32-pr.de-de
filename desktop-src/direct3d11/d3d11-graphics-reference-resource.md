# <a name="resource-reference-direct3d-11-graphics"></a>Ressourcenreferenz (Direct3D 11-Grafiken)

Die Direct3D-API definiert mehrere API-Elemente, die Sie beim Erstellen und Verwalten von Ressourcen unterstützen.


## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                              | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Ressourcenschnittstellen](d3d11-graphics-reference-resource-interfaces.md)<br/> | Es gibt eine Reihe von Schnittstellen für die beiden grundlegenden Ressourcentypen: Puffer und Texturen. Es gibt auch Schnittstellen für Ansichten von Ressourcen. <br/> Eine Anwendung verwendet eine Sicht, um eine Ressource an eine Pipelinephase zu binden. Die Ansicht definiert, wie während des Renderings auf die Ressource zugegriffen werden kann. In diesem Abschnitt werden die Ansichtsschnittstellen beschrieben. <br/> |
| [Ressourcenfunktionen](d3d11-graphics-reference-resource-functions.md)<br/>   | Dieser Abschnitt enthält Informationen zu den Ressourcenfunktionen.<br/>                                                                                                                                                                                                                                                                                |
| [Ressourcenstrukturen](d3d11-graphics-reference-resource-structures.md)<br/> | Strukturen werden zum Erstellen und Verwenden von Ressourcen verwendet.<br/>                                                                                                                                                                                                                                                                                               |
| [Ressourcenenumerationen](d3d11-graphics-reference-resource-enums.md)<br/>    | Enumerationen werden verwendet, um Informationen darüber anzugeben, wie Ressourcen während des Renderings erstellt und darauf zugegriffen wird.<br/>                                                                                                                                                                                                                                    |
