# <a name="resource-reference-direct3d-11-graphics"></a>Ressourcen Verweis (Direct3D 11-Grafik)

Die Direct3D-API definiert mehrere API-Elemente, die Sie beim Erstellen und Verwalten von Ressourcen unterstützen.


## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                              | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Ressourcen Schnittstellen](d3d11-graphics-reference-resource-interfaces.md)<br/> | Es gibt eine Reihe von Schnittstellen für die beiden grundlegenden Arten von Ressourcen: Puffer und Texturen. Es gibt auch Schnittstellen für Ansichten von Ressourcen. <br/> Eine Anwendung verwendet eine Ansicht, um eine Ressource an eine Pipeline Stufe zu binden. Die Sicht definiert, wie während des Renderings auf die Ressource zugegriffen werden kann. In diesem Abschnitt werden die Ansichts Schnittstellen beschrieben. <br/> |
| [Ressourcen Funktionen](d3d11-graphics-reference-resource-functions.md)<br/>   | Dieser Abschnitt enthält Informationen zu den Ressourcen Funktionen.<br/>                                                                                                                                                                                                                                                                                |
| [Ressourcen Strukturen](d3d11-graphics-reference-resource-structures.md)<br/> | Strukturen werden verwendet, um Ressourcen zu erstellen und zu verwenden.<br/>                                                                                                                                                                                                                                                                                               |
| [Ressourcenenumerationen](d3d11-graphics-reference-resource-enums.md)<br/>    | Enumerationen werden verwendet, um Informationen darüber anzugeben, wie Ressourcen während des Renderings erstellt und aufgerufen werden.<br/>                                                                                                                                                                                                                                    |
