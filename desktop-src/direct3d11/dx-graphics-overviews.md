# <a name="programming-guide-for-direct3d-11"></a>Programmierhandbuch für Direct3D 11

Der Programmierleitfaden enthält Informationen zur Verwendung der programmierbaren Microsoft Direct3D 11-Pipeline zum Erstellen von 3D-Echtzeitgrafiken für Spiele sowie wissenschaftliche anwendungen und Desktopanwendungen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | Beschreibung |
|-|-|
| [Geräte](overviews-direct3d-11-devices.md) | In diesem Abschnitt werden Direct3D 11-Geräte- und Gerätekontextobjekte beschrieben. |
| [Ressourcen](overviews-direct3d-11-resources.md) | In diesem Abschnitt werden Aspekte von Direct3D 11-Ressourcen beschrieben. |
| [Grafikpipeline](overviews-direct3d-11-graphics-pipeline.md) | In diesem Abschnitt wird die programmierbare Direct3D 11-Pipeline beschrieben. |
| [Übersicht über Compute-Shader](direct3d-11-advanced-stages-compute-shader.md) | Ein Compute-Shader ist eine programmierbare Shaderstufe, die Direct3D 11 über die Grafikprogrammierung hinaus erweitert. Die Compute-Shadertechnologie wird auch als DirectCompute-Technologie bezeichnet. |
| [Rendering](overviews-direct3d-11-render.md) | Dieser Abschnitt enthält Informationen zu mehreren Direct3D 11-Renderingtechnologien. |
| [Effekte](d3d11-graphics-programming-guide-effects.md) | Ein DirectX-Effekt ist eine Auflistung des Pipelinezustands, die durch in [HLSL](/windows/desktop/direct3dhlsl/dx-graphics-hlsl-reference) geschriebene Ausdrücke und eine syntaxspezifische Syntax festgelegt wird, die für das Effektframework spezifisch ist. |
| [Migrieren zu Direct3D 11](d3d11-programming-guide-migrating.md) | Dieser Abschnitt enthält Informationen zur Migration von einer früheren Version von Direct3D zu Direct3D 11. |
| [Direct3D-Videoschnittstellen](direct3d-video-interfaces.md) | In diesem Thema werden die Direct3D-Videoschnittstellen aufgelistet, die in Direct3D 9Ex verfügbar sind und auf Windows 7 und höher von Windows Clientbetriebssystemen und auf Windows Server 2008 R2 und höher von Windows Serverbetriebssystemen unterstützt werden. |
