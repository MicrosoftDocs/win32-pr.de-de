# <a name="common-version-interfaces"></a>Allgemeine Versions Schnittstellen

Dieser Abschnitt enthält Informationen zu den allgemeinen Versions Schnittstellen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [**ID3D10Blob**](/windows/win32/api/d3dcommon/nn-d3dcommon-id3d10blob) | Diese Schnittstelle wird verwendet, um Daten beliebiger Länge zurückzugeben. |
| [**ID3DBlob**](/previous-versions/windows/desktop/legacy/ff728743(v=vs.85)) | Diese Schnittstelle wird verwendet, um Daten beliebiger Länge zurückzugeben. |
| [**ID3DDestructionNotifier**](/windows/win32/api/d3dcommon/nn-d3dcommon-id3ddestructionotifier) | Wird verwendet, um für Rückrufe zu registrieren, wenn ein direktes 3D-Nano-com-Objekt zerstört wird. |
| [**ID3DInclude**](/windows/desktop/api/D3Dcommon/nn-d3dcommon-id3dinclude) | [**ID3DInclude**](/windows/desktop/api/D3Dcommon/nn-d3dcommon-id3dinclude) ist eine include-Schnittstelle, die der Benutzer implementiert, um es einer Anwendung zu ermöglichen, Benutzer über schreibbare Methoden zum Öffnen und Schließen von Shader-Includedateien aufzurufen \# . |
| [**ID3DUserDefinedAnnotation**](/windows/desktop/api/D3D11_1/nn-d3d11_1-id3duserdefinedannotation) | Die [**ID3DUserDefinedAnnotation**](/windows/desktop/api/D3D11_1/nn-d3d11_1-id3duserdefinedannotation) -Schnittstelle ermöglicht es einer Anwendung, konzeptionelle Abschnitte und Marker innerhalb des Code Flusses der Anwendung zu beschreiben. Ein entsprechend aktiviertes Tool, wie z. b. Microsoft Visual Studio Ultimate 2012, kann diese Abschnitte und Markierungen visuell entlang der Microsoft Direct3D-Zeilenlinie des Tools anzeigen, während das Tool die Anwendung debuggt. Mit diesen visuellen Notizen können Benutzer eines solchen Tools zu den Teilen der zeitanzahl navigieren, die von Interesse sind, oder um zu verstehen, welche Direct3D-Aufrufe von bestimmten Abschnitten des Anwendungs Codes erstellt werden. |
