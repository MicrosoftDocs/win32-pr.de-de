# <a name="common-version-reference"></a>Allgemeine Versionsreferenz

Die Direct3D-API definiert mehrere API-Elemente, die für Direct3D 12, Direct3D 11, Direct3D 10 und Direct3D 10.1 gelten. Sie können diese API-Elemente in Ihrem Code für jede dieser Direct3D-Versionen verwenden. Diese API-Elemente werden als versionsneutral bezeichnet.
