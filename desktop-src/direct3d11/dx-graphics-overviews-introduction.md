# <a name="whats-new-in-direct3d-11"></a>Neuerungen in Direct3D 11

In diesem Abschnitt werden die in Direct3D 11, Direct3D 11,1 und Direct3D 11,2 hinzugefügten Features beschrieben.

Microsoft Direct3D 11 ist eine Erweiterung der Rendering-API von Microsoft Direct3D 10/Microsoft Direct3D 10,1. Ausführlichere Informationen zur Verwendung von Direct3D 11 finden Sie im [Programmier Handbuch für Direct3D 10](/windows/desktop/direct3d10/d3d10-graphics-programming-guide).
