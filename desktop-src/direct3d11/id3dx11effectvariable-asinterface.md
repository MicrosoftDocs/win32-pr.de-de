# <a name="id3dx11effectvariableasinterface-method"></a>ID3DX11EffectVariable::AsInterface-Methode

Hier erhalten Sie eine Schnittstellenvariable.

## <a name="syntax"></a>Syntax


```C++
ID3DX11EffectInterfaceVariable* AsInterface();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **ID3DX11EffectInterfaceVariable**](id3dx11effectinterfacevariable.md)\***

Ein Zeiger auf eine Schnittstellenvariable. Siehe [**ID3DX11EffectInterfaceVariable**](id3dx11effectinterfacevariable.md).

## <a name="remarks"></a>Hinweise

> [!Note]  
> Das DirectX SDK stellt keine kompilierten Binärdateien für Effekte bereit. Sie müssen die Quelle Effects 11 verwenden, um Ihre Effekttypanwendung zu erstellen. Weitere Informationen zur Verwendung der Quelle Effects 11 finden Sie unter [Unterschiede zwischen Effekten 10 und Effekten 11.](d3d11-graphics-programming-guide-effects-differences.md)
