# <a name="id3dx11effectunorderedaccessviewvariable-interface"></a>ID3DX11EffectUnorderedAccessViewVariable-Schnittstelle

Greift auf eine ungeordnete Zugriffs Ansicht zu.

## <a name="members"></a>Member

Die **ID3DX11EffectUnorderedAccessViewVariable** -Schnittstelle erbt von [**ID3DX11EffectVariable**](id3dx11effectvariable.md). **ID3DX11EffectUnorderedAccessViewVariable** verfügt auch über die folgenden Typen von Membern:

-   [Methoden](#methods)

### <a name="methods"></a>Methoden

Die **ID3DX11EffectUnorderedAccessViewVariable** -Schnittstelle verfügt über diese Methoden.



| Methode                                                                                                      | BESCHREIBUNG                                        |
|:------------------------------------------------------------------------------------------------------------|:---------------------------------------------------|
| [**Getunorderedaccessview**](id3dx11effectunorderedaccessviewvariable-getunorderedaccessview.md)           | Erhalten Sie eine ungeordnete Zugriffs Ansicht.<br/>           |
| [**Getunorderedaccessviewarray**](id3dx11effectunorderedaccessviewvariable-getunorderedaccessviewarray.md) | Abrufen eines Arrays von ungeordneten Access-Ansichten.<br/> |
| [**"*"**](id3dx11effectunorderedaccessviewvariable-setunorderedaccessview.md)           | Legen Sie eine ungeordnete Zugriffs Ansicht fest.<br/>           |
| [**"*"**](id3dx11effectunorderedaccessviewvariable-setunorderedaccessviewarray.md) | Legen Sie ein Array von ungeordneten Zugriffs Ansichten fest.<br/> |



 

## <a name="remarks"></a>Bemerkungen

> [!Note]  
> Das DirectX SDK stellt keine kompilierten Binärdateien für Effekte bereit. Sie müssen die Effekte 11-Quelle verwenden, um die Effekte-Anwendung zu erstellen. Weitere Informationen zum Verwenden der Effekte 11-Quelle finden Sie [unter Unterschiede zwischen den Effekten 10 und Effekte 11](d3d11-graphics-programming-guide-effects-differences.md).
