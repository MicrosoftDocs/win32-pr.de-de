# <a name="windows-api-index"></a>Windows-API-Index

Im folgenden finden Sie eine Liste der Referenz Inhalte für die Windows-Anwendungsprogrammierschnittstelle (Application Programming Interface, API) für Desktop-und Server Anwendungen.

Mithilfe der Windows-API können Sie Anwendungen entwickeln, die in allen Versionen von Windows erfolgreich ausgeführt werden, und dabei die Features und Funktionen nutzen, die für jede Version eindeutig sind. (Beachten Sie, dass diese zuvor als Win32-API bezeichnet wurde. Der Name der Windows-API spiegelt die Stamm Elemente in 16-Bit-Fenstern genauer wider und unterstützt 64-Bit-Windows.)
