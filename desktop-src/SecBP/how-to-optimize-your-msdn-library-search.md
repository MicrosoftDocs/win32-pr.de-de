# <a name="how-to-optimize-your-msdn-library-search"></a>Optimieren der MSDN Library-Suche

Um MSDN zu durchsuchen, geben Sie ein Schlüsselwort oder eine Zeichenfolge in das Feld **MSDN mit Bing suchen** ein. Nachdem Sie die [Msdn-Website (Microsoft-Entwickler Network)](https://msdn.microsoft.com/default.aspx) durchsucht haben, können Sie Ihre Suchergebnisse eingrenzen, indem Sie auf die optionen klicken, die in der Spalte **Suche optimieren** unter **Nach Quelle** oder Nach **Thema** verfügbar sind. Diese Optionen sind nach dem ersten Suchversuch verfügbar.

**So geben Sie nur Themen im Abschnitt Windows Entwicklung der MSDN Library zurück**

1.  Klicken Sie in der Liste **Nach Thema** auf **Win32 & COM**. Eine zusätzliche Liste mit dem Namen **Current Verfeinerungen** wird angezeigt. Die Zeile **Aktuelle Verfeinerungen** enthält **Win32 & COM.**
2.  Klicken Sie in der Liste **Nach Quelle** auf Dokumentation **& Artikel**. Die Liste **Der aktuellen Optimierungen** enthält jetzt **Dokumentation & Artikel**.

Wenn Sie suchen, stehen möglicherweise zusätzliche Filter zur Verfügung, z. B. **Windows-Sicherheit**, **Windows Grafiken und Multimedia** oder Windows Messaging und **Zusammenarbeit.**

**So suchen Sie nur Themen im Sicherheits-SDK**

-   Klicken Sie in der Liste **Nach Thema** auf **Windows-Sicherheit**. Die Liste **Aktuelle Verfeinerungen** enthält jetzt **Windows-Sicherheit**.

Wenn Sie eine Suchzeichenfolge eingeben, werden Vorschläge automatisch unterhalb des Suchfelds angezeigt. Um nach SDK-Inhalten zu suchen, suchen Sie in den Vorschlägen nach **API** oder **Fenstern.**

Wenn Sie in den Suchergebnissen auf ein Thema klicken, können Sie schnell ermitteln, ob das Thema SDK-Inhalt bereitstellt, indem Sie den Navigationspfad anzeigen, der oben im Thema angezeigt wird. Beispielsweise generiert eine Suche nach **Kryptografie** Vorschläge, die die **Kryptografie-API** enthalten. Wenn Sie auf die **Vorschlagskryptografie-API** klicken, wird ein Suchergebnissebereich generiert, der **Cryptography Functions (Windows)** enthält. Wenn Sie auf **Cryptography Functions (Windows)** klicken, gelangen Sie zu einem Thema, das den folgenden Navigationspfad oben im Thema enthält:

**MSDN > MSDN Library > Windows Development > Security > Cryptography > Cryptography Reference > Cryptography Functions >**

Der gesamte Inhalt des Sicherheits-SDK enthält diesen Teil dieses Navigationspfads:

**MSDN > MSDN Library > Windows Development > Security >**

**So entfernen Sie Suchergebnissefilter aus Ihrer MSDN-Suche**

-   Klicken Sie in der Liste **Aktuelle Verfeinerungen** auf das **(x),** das dem Suchfilter folgt, den Sie entfernen möchten. Um z. **B. Die Dokumentation & Artikel** aus **aktuellen Verfeinerungen** zu entfernen, klicken Sie auf das **(x),** das der **Dokumentation & Artikel** folgt. Sie können auch auf **Alle entfernen** klicken, um alle Filter aus Ihrer MSDN-Suche zu entfernen.
