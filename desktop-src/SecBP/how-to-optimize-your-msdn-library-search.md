# <a name="how-to-optimize-your-msdn-library-search"></a>Optimieren der MSDN Library-Suche

Um MSDN zu durchsuchen, geben Sie ein Schlüsselwort oder eine Zeichenfolge in das Feld **MSDN mit dem Namen durchsuchen** ein. Nachdem Sie die MSDN-Website ( [Microsoft Developer Network](https://msdn.microsoft.com/default.aspx) ) durchsucht haben, können Sie Ihre Suchergebnisse eingrenzen, indem Sie auf die Optionen klicken, die in der Spalte zum **verfeinern der Suche** unter nach **Quelle** oder **Thema** verfügbar sind. Diese Optionen werden nach dem ersten suchversuch verfügbar.

**So geben Sie nur Themen im Abschnitt zur Windows-Entwicklung der MSDN Library zurück**

1.  Klicken Sie in der Liste **nach Thema** auf **Win32 & com**. Eine zusätzliche Liste mit dem Namen **aktuelle** Optimierungen wird angezeigt. Die **aktuelle** Verarbeitungen haben **Win32-& com** darin.
2.  Klicken Sie in der Liste **nach Quelle** auf **Dokumentation & Artikel**. Die aktuelle Liste der Verb **esse rungen** enthält jetzt **Dokumentations & Artikel**.

Bei der Suche werden möglicherweise zusätzliche Filter verfügbar sein, z. b. **Windows-Sicherheit**, **Windows-Grafiken und-Multimedia** oder **Windows-Messaging und-Zusammenarbeit**.

**So durchsuchen Sie nur Themen im Security SDK**

-   Klicken Sie in der Liste **nach Thema** auf **Windows-Sicherheit**. Die **aktuelle Liste der Verfeinerungen** enthält jetzt die **Windows-Sicherheit**.

Wenn Sie eine Such Zeichenfolge eingeben, werden Vorschläge automatisch unter dem Suchfeld angezeigt. Suchen Sie in den Vorschlägen nach **API** oder **Windows** , um nach SDK-Inhalten zu suchen.

Wenn Sie in den Suchergebnissen auf ein Thema klicken, können Sie schnell feststellen, ob das Thema den SDK-Inhalt bereitstellt, indem Sie den Navigationspfad anzeigen, der am Anfang des Themas angezeigt wird. Eine Suche nach **Kryptografie** generiert beispielsweise Vorschläge, die die **kryptografieapi** enthalten. Wenn Sie auf die Vorschlags- **kryptografieapi** klicken, wird ein Suchergebnis Bereich generiert, der **Kryptografiefunktionen (Windows)** enthält. Wenn Sie auf **Kryptografiefunktionen (Windows)** klicken, gelangen Sie zu einem Thema, das den folgenden Navigationspfad am Anfang des Themas enthält:

**MSDN > MSDN Library > Windows-Entwicklungs > Sicherheit > Kryptografie > kryptografiereferenz > Kryptografiefunktionen >**

Alle Security SDK-Inhalte enthalten diesen Teil dieses Navigations Pfads:

**MSDN > MSDN Library > Windows-Entwicklungs > Sicherheit >**

**So entfernen Sie Suchergebnis Filter aus der MSDN-Suche**

-   Klicken Sie in der Liste **aktuelle Verbesserungen** auf das **(x)** , das auf den Suchfilter folgt, den Sie entfernen möchten. Wenn Sie z. b. die **Dokumentation & Artikel** aus den **aktuellen Verfeinerungen** entfernen möchten, klicken Sie auf das **(x)** , das der **Dokumentation & Artikeln** folgt. Sie können auch auf **Alle entfernen** klicken, um alle Filter aus der MSDN-Suche zu entfernen.
