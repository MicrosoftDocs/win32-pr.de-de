# <a name="direct3d"></a>Direct3D

Direct3D ist eine API auf niedriger Ebene zum Zeichnen primitiver Elemente mit der Renderingpipeline oder zum Durchführen paralleler Vorgänge mit dem Compute-Shader. Weitere Informationen finden Sie im folgenden Inhalt.

Informationen zum Abrufen und Installieren von Direct3D finden Sie unter [Direct3D 12 Programming Environment Setup](./direct3d12/directx-12-programming-environment-set-up.md).

| [Direct3D 12-Grafiken](./direct3d12/direct3d-12-graphics.md) | Direct3D 12 bietet eine API und Plattform, mit der Ihre Anwendung die Grafik-und Berechnungsfunktionen von PCs nutzen kann, die mit mindestens einem Direct3D 12-kompatiblen GPUs ausgestattet sind. |
| [Direct3D 11-Grafik](./direct3d11/atoc-dx-graphics-direct3d-11.md) | Sie können Microsoft Direct3D 11-Grafiken verwenden, um 3D-Grafiken für Spiele und wissenschaftliche und Desktop Anwendungen zu erstellen. |
| [DXGI](./direct3ddxgi/dx-graphics-dxgi.md) | DXGI behandelt das Auflisten von Grafikadaptern, das Auflisten der Anzeigemodi, das Auswählen von Puffer Formaten, das Freigeben von Ressourcen zwischen Prozessen und das darstellen von gerenderten Frames für ein Fenster oder einen Monitor zur Anzeige. |
