# <a name="restart-manager-callback-functions"></a>Neustart-Manager-Rückruffunktionen

Die Restart Manager-API verwendet die Rückruffunktionen in der folgenden Tabelle.



| Rückruffunktion                                               | Beschreibung                                                                                                                                                            |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**RM \_ WRITE \_ STATUS \_ CALLBACK**](/windows/win32/api/restartmanager/nc-restartmanager-rm_write_status_callback) | Die Anwendung, die die Neustart-Manager-Sitzung gestartet hat, kann einen Zeiger auf diese Funktion an die Restart Manager-Funktionen übergeben, um einen Prozentsatz der Vollständigkeit zu erhalten. |
