# <a name="idcompositionvisualsetoffsety-methods"></a>IDCompositionVisual::SetOffsetY-Methoden

Ändert oder animiert den Wert der OffsetY-Eigenschaft dieses Visuals und ändert die vertikale Position des Visuals relativ zum übergeordneten Element.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                | Beschreibung                                           |
|:------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetOffsetY(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual-setoffsety(float))                                     | Ändert den Wert der OffsetY-Eigenschaft.<br/> |
| [**SetOffsetY(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual-setoffsety(idcompositionanimation)) | Animiert den Wert der OffsetY-Eigenschaft.<br/> |
