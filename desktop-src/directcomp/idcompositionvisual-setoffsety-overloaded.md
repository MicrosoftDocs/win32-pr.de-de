# <a name="idcompositionvisualsetoffsety-methods"></a>Idcompositionvisual:: Seto ffsety-Methoden

Ändert oder animiert den Wert der OffsetY-Eigenschaft dieses visuellen Elements und ändert die vertikale Position des visuellen Elements relativ zu seinem übergeordneten Element.
