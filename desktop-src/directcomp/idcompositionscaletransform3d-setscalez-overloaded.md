# <a name="idcompositionscaletransform3dsetscalez-methods"></a>IDCompositionScaleTransform3D::SetScaleZ-Methoden

Ändert oder animiert den Wert der ScaleZ-Eigenschaft einer Skalierungstransformation. Die ScaleZ-Eigenschaft gibt den Skalierungsfaktor entlang der Z-Achse an.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                        | Beschreibung                                           |
|:--------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetScaleZ(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscalez(float))                                     | Ändert den Wert der ScaleZ-Eigenschaft.<br/>  |
| [**SetScaleZ(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscalez(idcompositionanimation)) | Animiert den Wert der ScaleZ-Eigenschaft.<br/> |
