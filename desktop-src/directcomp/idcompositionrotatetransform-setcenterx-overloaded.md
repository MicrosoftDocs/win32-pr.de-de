# <a name="idcompositionrotatetransformsetcenterx-methods"></a>IDCompositionRotateTransform::SetCenterX-Methoden

Ändert oder animiert den Wert der CenterX-Eigenschaft einer 2D-Drehungstransformation. Die CenterX-Eigenschaft gibt die x-Koordinate des Punkts an, um den die Drehung ausgeführt wird.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                         | Beschreibung                                            |
|:---------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setcenterx(float))                                     | Ändert den Wert der CenterX-Eigenschaft.<br/>  |
| [**SetCenterX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setcenterx(idcompositionanimation)) | Animiert den Wert der CenterX-Eigenschaft.<br/> |
