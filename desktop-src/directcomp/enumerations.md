# <a name="directcomposition-enumerations"></a>DirectComposition-Enumerationen

In diesem Abschnitt werden die von der Microsoft DirectComposition-API bereitgestellten Enumerationen beschrieben.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                                  | Beschreibung                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DCOMPOSITION \_ BACKFACE \_ VISIBILITY**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_backface_visibility)<br/>              | Gibt die Aufschriftsichtbarkeit an, die auf ein Visual angewendet werden soll. <br/>                                                                                                                            |
| [**DCOMPOSITION \_ BITMAP \_ INTERPOLATION \_ MODE**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_bitmap_interpolation_mode)<br/> | Gibt den Interpolationsmodus an, der verwendet werden soll, wenn eine Bitmap mit einer Transformation zusammengesetzt wird, bei der die Pixel in der Bitmap nicht genau 1:1 mit Pixeln auf dem Bildschirm angezeigt werden. <br/>         |
| [**\_DCOMPOSITION-RAHMENMODUS \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_border_mode)<br/>                              | Gibt den Rahmenmodus an, der beim Erstellen einer Bitmap oder Beim Anwenden eines Clips mit einer beliebigen Transformation verwendet werden soll, damit die Ränder der Bitmap oder des Clips nicht an ganzzahligen Koordinaten ausgerichtet sind. <br/> |
| [**ZUSAMMENGESETZTER \_ DCOMPOSITION-MODUS \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_composite_mode)<br/>                        | Der Modus, der zum Mischen des Bitmapinhalts eines Visuals mit dem Renderziel verwendet werden soll.<br/>                                                                                                          |
| [**OPACITY-MODUS \_ DER DCOMPOSITION \_**](/windows/desktop/api/dcomptypes/ne-dcomptypes-dcomposition_opacity_mode)<br/>                            | Gibt an, wie der effektive Deckkraftwert eines Visuals auf den Inhalt und die unteren Elemente dieses Visuals angewendet wird.<br/>                                                                                  |
| [**TRANSFORMATIONSMODUS "DCOMPOSITION" \_ \_**](/previous-versions/windows/desktop/legacy/dn904487(v=vs.85))<br/>                        | Gibt den Transformationsstil an, der auf ein Visual angewendet werden soll. <br/>                                                                                                                                |
