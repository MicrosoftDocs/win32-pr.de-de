# <a name="directcomposition-enumerations"></a>Directcomposition-Enumerationen

In diesem Abschnitt werden die von der Microsoft directcomposition-API bereitgestellten Enumerationen beschrieben.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                                  | BESCHREIBUNG                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Sichtbarkeit der dcomposition- \_ Backface \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_backface_visibility)<br/>              | Gibt die Sichtbarkeit der Hintergrund Anzeige an, die auf ein visuelles Element angewendet werden soll. <br/>                                                                                                                            |
| [**dcomposition- \_ Bitmap- \_ Interpolations \_ Modus**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_bitmap_interpolation_mode)<br/> | Gibt den Interpolations Modus an, der verwendet werden soll, wenn eine Bitmap mit einer Transformation zusammengesetzt wird, bei der die Pixel in der Bitmap nicht genau eins zu eins mit Pixel auf dem Bildschirm anordnen. <br/>         |
| [**dcomposition-Rahmen \_ \_ Modus**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_border_mode)<br/>                              | Gibt den Rahmen Modus an, der beim Erstellen einer Bitmap oder beim Anwenden eines Clips mit einer beliebigen Transformation verwendet werden soll, sodass die Ränder der Bitmap oder des Clips nicht an ganzzahligen Koordinaten ausgerichtet sind. <br/> |
| [**zusammengesetzter dcomposition- \_ \_ Modus**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_composite_mode)<br/>                        | Der Modus, der verwendet wird, um den Bitmapinhalt eines visuellen Elements mit dem Renderziel zu kombinieren.<br/>                                                                                                          |
| [**dcomposition- \_ Deckkraft \_ Modus**](/windows/desktop/api/dcomptypes/ne-dcomptypes-dcomposition_opacity_mode)<br/>                            | Gibt an, wie der effektive Deckkraft Wert eines visuellen Elements auf den Inhalt und die untergeordneten Elemente der visuellen Elemente angewendet wird.<br/>                                                                                  |
| [**dcomposition- \_ Transformations \_ Modus**](/previous-versions/windows/desktop/legacy/dn904487(v=vs.85))<br/>                        | Gibt den Transformations Stil an, der auf ein visuelles Element angewendet werden soll. <br/>                                                                                                                                |
