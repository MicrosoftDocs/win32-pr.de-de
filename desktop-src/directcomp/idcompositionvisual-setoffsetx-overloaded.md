# <a name="idcompositionvisualsetoffsetx-methods"></a>Idcompositionvisual:: sedeffsetx-Methoden

Ändert oder animiert den Wert der OffsetX-Eigenschaft dieses visuellen Elements und ändert die horizontale Position des visuellen Elements relativ zu seinem übergeordneten Element.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                | BESCHREIBUNG                                           |
|:------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Seto ffsetx (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual-setoffsetx(float))                                     | Ändert den Wert der OffsetX-Eigenschaft.<br/> |
| [**Seto ffsetx (idcompositionanimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual-setoffsetx(idcompositionanimation)) | Animiert den Wert der OffsetX-Eigenschaft.<br/> |
