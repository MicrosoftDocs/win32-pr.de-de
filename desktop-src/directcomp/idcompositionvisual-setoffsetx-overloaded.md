# <a name="idcompositionvisualsetoffsetx-methods"></a>IDCompositionVisual::SetOffsetX-Methoden

Ändert oder animiert den Wert der OffsetX-Eigenschaft dieses Visuals und ändert die horizontale Position des Visuals relativ zum übergeordneten Element.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                | Beschreibung                                           |
|:------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetOffsetX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual-setoffsetx(float))                                     | Ändert den Wert der OffsetX-Eigenschaft.<br/> |
| [**SetOffsetX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual-setoffsetx(idcompositionanimation)) | Animiert den Wert der OffsetX-Eigenschaft<br/> |
