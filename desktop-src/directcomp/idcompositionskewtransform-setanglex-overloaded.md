# <a name="idcompositionskewtransformsetanglex-methods"></a>Idcompositionschräwtransform::-Methode

Ändert oder animiert den Wert der AngleX-Eigenschaft einer 2D-Schiefe-Transformation. Die AngleX-Eigenschaft gibt den Drehwinkel in Grad an. Der Standardwert ist 0 (null).

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                     | BESCHREIBUNG                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**"S" (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setanglex(float))                                     | Ändert den Wert der AngleX-Eigenschaft.<br/>  |
| [**"S" ("idcompositionanimation" \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setanglex(idcompositionanimation)) | Animiert den Wert der AngleX-Eigenschaft.<br/> |
