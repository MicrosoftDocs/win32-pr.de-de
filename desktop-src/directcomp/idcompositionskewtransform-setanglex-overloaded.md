# <a name="idcompositionskewtransformsetanglex-methods"></a>IDCompositionSkewTransform::SetAngleX-Methoden

Ändert oder animiert den Wert der AngleX-Eigenschaft einer 2D-Schiefetransformation. Die AngleX-Eigenschaft gibt den Drehwinkel in Grad an. Der Standardwert ist 0 (null).

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                     | Beschreibung                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetAngleX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setanglex(float))                                     | Ändert den Wert der AngleX-Eigenschaft.<br/>  |
| [**SetAngleX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setanglex(idcompositionanimation)) | Animiert den Wert der AngleX-Eigenschaft.<br/> |
