# <a name="idcompositionscaletransformsetscalex-methods"></a>Idcompositionscaletransform:: setscalex-Methoden

Ändert oder animiert den Wert der ScaleX-Eigenschaft einer 2D-Skalierungs Transformation. Die ScaleX-Eigenschaft gibt den Skalierungsfaktor entlang der x-Achse an.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                      | BESCHREIBUNG                                           |
|:------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Setscalex (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setscalex(float))                                     | Ändert den Wert der ScaleX-Eigenschaft.<br/>  |
| [**Setscalex (idcompositionanimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setscalex(idcompositionanimation)) | Animiert den Wert der ScaleX-Eigenschaft.<br/> |
