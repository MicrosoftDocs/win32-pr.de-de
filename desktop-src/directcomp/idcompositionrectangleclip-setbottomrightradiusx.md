# <a name="idcompositionrectangleclipsetbottomrightradiusx-methods"></a>IDCompositionRectangleClip::SetBottomRightRadiusX-Methoden

Ändert oder animiert den Wert der BottomRightRadiusX-Eigenschaft dieses Clips. Die BottomRightRadiusX-Eigenschaft gibt den x-Radius der Ellipse an, die die untere rechte Ecke des Clips rundet.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                                             | Beschreibung                                                                    |
|:-----------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------|
| [**SetBottomRightRadiusX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottomrightradiusx(idcompositionanimation))                                     | Ändert den Wert der BottomRightRadiusX-Eigenschaft dieses Clips.<br/>  |
| [**SetBottomRightRadiusX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottomrightradiusx(idcompositionanimation)) | Animiert den Wert der BottomRightRadiusX-Eigenschaft dieses Clips.<br/> |
