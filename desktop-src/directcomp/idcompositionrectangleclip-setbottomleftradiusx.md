# <a name="idcompositionrectangleclipsetbottomleftradiusx-methods"></a>IDCompositionRectangleClip::SetBottomLeftRadiusX-Methoden

Ändert oder animiert den Wert der BottomLeftRadiusX-Eigenschaft dieses Clips. Die BottomLeftRadiusX-Eigenschaft gibt den x-Radius der Ellipse an, die die untere linke Ecke des Clips rundet.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                                           | Beschreibung                                                                   |
|:---------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------|
| [**SetBottomLeftRadiusX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottomleftradiusx(idcompositionanimation))                                     | Ändert den Wert der BottomLeftRadiusX-Eigenschaft dieses Clips.<br/>  |
| [**SetBottomLeftRadiusX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottomleftradiusx(idcompositionanimation)) | Animiert den Wert der BottomLeftRadiusX-Eigenschaft dieses Clips.<br/> |
