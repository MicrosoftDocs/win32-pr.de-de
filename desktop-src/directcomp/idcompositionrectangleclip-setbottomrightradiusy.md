# <a name="idcompositionrectangleclipsetbottomrightradiusy-methods"></a>IDCompositionRectangleClip::SetBottomRightRadiusY-Methoden

Ändert oder animiert den Wert der BottomRightRadiusY-Eigenschaft dieses Clips. Die BottomRightRadiusY-Eigenschaft gibt den y-Radius der Ellipse an, die die untere rechte Ecke des Clips rundet.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                                             | Beschreibung                                                                    |
|:-----------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------|
| [**SetBottomRightRadiusY(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottomrightradiusy(idcompositionanimation))                                     | Ändert den Wert der BottomRightRadiusY-Eigenschaft dieses Clips.<br/>  |
| [**SetBottomRightRadiusY(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottomrightradiusy(idcompositionanimation)) | Animiert den Wert der BottomRightRadiusY-Eigenschaft dieses Clips.<br/> |
