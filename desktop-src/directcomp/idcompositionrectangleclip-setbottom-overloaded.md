# <a name="idcompositionrectangleclipsetbottom-methods"></a>IDCompositionRectangleClip::SetBottom-Methoden

Animiert oder ändert den Wert der Bottom-Eigenschaft eines Cliprechtecks. Die Bottom-Eigenschaft gibt die y-Koordinate der unteren rechten Ecke des Cliprechtecks an.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                     | Beschreibung                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetBottom(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottom(float))                                     | Ändert den Wert der Bottom-Eigenschaft.<br/>  |
| [**SetBottom(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottom(idcompositionanimation)) | Animiert den Wert der Bottom-Eigenschaft.<br/> |
