# <a name="idcompositionrectangleclipsetbottom-methods"></a>Idcompositionrechgleclip:: SetBottom-Methoden

Animiert oder ändert den Wert der untersten-Eigenschaft eines Clip Rechtecks. Die Bottom-Eigenschaft gibt die y-Koordinate der unteren rechten Ecke des Clip Rechtecks an.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                     | BESCHREIBUNG                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetBottom (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottom(float))                                     | Ändert den Wert der untersten-Eigenschaft.<br/>  |
| [**SetBottom (idcompositionanimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottom(idcompositionanimation)) | Animiert den Wert der untersten-Eigenschaft.<br/> |
