# <a name="idcompositionrectangleclipsettopleftradiusx-methods"></a>Idcompositionrechgleclip:: settopleftradius-Methoden

Ändert oder animiert den Wert der topleftradi-x-Eigenschaft dieses Clips. Die topleftradiusx-Eigenschaft gibt den x-Radius der Ellipse an, der die linke obere Ecke des Clips rundet.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                                     | BESCHREIBUNG                                                                |
|:---------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------|
| [**Settopleftradius x (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settopleftradiusx(idcompositionanimation))                                     | Ändert den Wert der topLeftRadius x-Eigenschaft dieses Clips.<br/>  |
| [**Settopleftradius x (idcompositionanimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settopleftradiusx(idcompositionanimation)) | Animiert den Wert der topLeftRadius x-Eigenschaft dieses Clips.<br/> |
