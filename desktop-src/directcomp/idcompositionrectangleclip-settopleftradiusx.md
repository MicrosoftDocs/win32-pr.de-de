# <a name="idcompositionrectangleclipsettopleftradiusx-methods"></a>IDCompositionRectangleClip::SetTopLeftRadiusX-Methoden

Ändert oder animiert den Wert der TopLeftRadiusX-Eigenschaft dieses Clips. Die TopLeftRadiusX-Eigenschaft gibt den x-Radius der Ellipse an, die die linke obere Ecke des Clips rundet.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                                     | Beschreibung                                                                |
|:---------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------|
| [**SetTopLeftRadiusX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settopleftradiusx(idcompositionanimation))                                     | Ändert den Wert der TopLeftRadiusX-Eigenschaft dieses Clips.<br/>  |
| [**SetTopLeftRadiusX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settopleftradiusx(idcompositionanimation)) | Animiert den Wert der TopLeftRadiusX-Eigenschaft dieses Clips.<br/> |
