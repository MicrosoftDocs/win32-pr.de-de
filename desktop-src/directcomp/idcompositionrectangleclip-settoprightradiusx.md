# <a name="idcompositionrectangleclipsettoprightradiusx-methods"></a>Idcompositionrechgleclip:: settoprightradius-Methoden

Ändert oder animiert den Wert der toprightradi-x-Eigenschaft dieses Clips. Die toprightradiusx-Eigenschaft gibt den x-Radius der Ellipse an, der die obere rechte Ecke des Clips rundet.
