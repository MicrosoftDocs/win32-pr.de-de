# <a name="idcompositionscaletransformsetcentery-methods"></a>Idcompositionscaletransform:: setcentery-Methoden

Ändert oder animiert den Wert der CenterY-Eigenschaft einer 2D-Skalierungs Transformation. Die CenterY-Eigenschaft gibt die y-Koordinate des Punkts an, an dem die Skalierung ausgeführt wird.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                        | BESCHREIBUNG                                            |
|:--------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Setcentery (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setcentery(float))                                     | Ändert den Wert der CenterY-Eigenschaft.<br/>  |
| [**Setcentery (idcompositionanimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setcentery(idcompositionanimation)) | Animiert den Wert der CenterY-Eigenschaft.<br/> |
