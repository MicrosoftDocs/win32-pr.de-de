# <a name="idcompositionskewtransformsetangley-methods"></a>IDCompositionSkewTransform::SetAngleY-Methoden

Ändert oder animiert den Wert der AngleY-Eigenschaft einer 2D-Schiefetransformation. Die AngleY-Eigenschaft gibt den Drehwinkel in Grad an. Der Standardwert ist 0 (null).

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                     | Beschreibung                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetAngleY(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setangley(float))                                     | Ändert den Wert der AngleY-Eigenschaft.<br/>  |
| [**SetAngleY(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setangley(idcompositionanimation)) | Animiert den Wert der AngleY-Eigenschaft.<br/> |
