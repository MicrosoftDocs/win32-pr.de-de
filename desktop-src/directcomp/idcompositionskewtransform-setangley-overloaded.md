# <a name="idcompositionskewtransformsetangley-methods"></a>Idcompositionschräwtransform::-Methode

Ändert oder animiert den Wert der AngleY-Eigenschaft einer 2D-Schiefe-Transformation. Die AngleY-Eigenschaft gibt den Drehwinkel in Grad an. Der Standardwert ist 0 (null).

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                     | BESCHREIBUNG                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**"S" (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setangley(float))                                     | Ändert den Wert der AngleY-Eigenschaft.<br/>  |
| [**"S" ("idcompositionanimation" \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setangley(idcompositionanimation)) | Animiert den Wert der AngleY-Eigenschaft.<br/> |
