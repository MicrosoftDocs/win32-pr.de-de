# <a name="directcomposition-interfaces"></a>DirectComposition-Schnittstellen

In diesem Abschnitt werden die Schnittstellen beschrieben, die von der Microsoft DirectComposition-API bereitgestellt werden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                               | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                               |
|-----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IDCompositionAffineTransform2DEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionaffinetransform2deffect)<br/>     | Der arithmetische zusammengesetzte Effekt wird verwendet, um zwei Bilder mithilfe einer gewichteten Summe von Pixeln aus den Eingabebildern zu kombinieren. <br/>                                                                                                                                                                                                                                                                             |
| [**IDCompositionAnimation**](/windows/desktop/api/DcompAnimation/nn-dcompanimation-idcompositionanimation)<br/>                                 | Stellt eine Funktion zum Animieren einer oder mehrerer Eigenschaften eines oder mehrerer DirectComposition-Objekte dar.<br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionArithmeticCompositeEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionarithmeticcompositeeffect)<br/> | Der arithmetische zusammengesetzte Effekt wird verwendet, um zwei Bilder mithilfe einer gewichteten Summe von Pixeln aus den Eingabebildern zu kombinieren.<br/>                                                                                                                                                                                                                                                                              |
| [**IDCompositionBlendEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionblendeffect)<br/>                             | Blend Effect wird verwendet, um zwei Bilder zu kombinieren. <br/>                                                                                                                                                                                                                                                                                                                                                 |
| [**IDCompositionBrightnessEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionbrightnesseffect)<br/>                   | Der Helligkeitseffekt steuert die Helligkeit des Bilds.<br/>                                                                                                                                                                                                                                                                                                                                    |

| [**IDCompositionCompositeEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioncompositeeffect)<br/>                     | Der zusammengesetzte Effekt wird verwendet, um 2 oder mehr Bilder zu kombinieren. Dieser Effekt verfügt über 13 verschiedene zusammengesetzte Modi. Der zusammengesetzte Effekt akzeptiert 2 oder mehr Eingaben. Wenn Sie zwei Bilder angeben, ist das Ziel die erste Eingabe (Index 0) und die Quelle die zweite Eingabe (Index 1). Wenn Sie mehr als zwei Eingaben angeben, werden die Bilder zusammengesetzt, beginnend mit der ersten Eingabe und der zweiten usw. <br/> |
| [**IDCompositionClip**](/windows/win32/api/dcomp/nn-dcomp-idcompositionclip)<br/>                                           | Stellt ein Clipobjekt dar, das verwendet wird, um das Rendering einer visuellen Teilstruktur auf einen rechteckigen Bereich zu beschränken. <br/>                                                                                                                                                                                                                                                                                    |
| [**IDCompositionDesktopDevice**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondesktopdevice)<br/>                         | Eine Anwendung muss die IDCompositionDesktopDevice-Schnittstelle verwenden, um DirectComposition in einer Win32-Desktopanwendung zu verwenden. Diese Schnittstelle ermöglicht der Anwendung, eine visuelle Struktur mit einem Fenster zu verbinden und mehrstufige untergeordnete Fenster für die Komposition zu hosten.<br/>                                                                                                                                |
| [**IDCompositionDevice**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice)<br/>                                       | Dient als Factory für alle anderen DirectComposition-Objekte und stellt Methoden zum Steuern der Transaktionskomposition bereit.<br/>                                                                                                                                                                                                                                                                         |
| [**IDCompositionDevice2**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice2)<br/>                                     | Dient als Factory für alle anderen DirectComposition-Objekte und stellt Methoden zum Steuern der Transaktionskomposition bereit.<br/>                                                                                                                                                                                                                                                                         |
| [**IDCompositionDevice3**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice3)<br/>                                     | Dient als Factory für alle anderen DirectComposition-Objekte und stellt Methoden zum Steuern der Transaktionskomposition bereit. <br/>                                                                                                                                                                                                                                                                        |
| [**IDCompositionDeviceDebug**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevicedebug)<br/>                             | Bietet Zugriff auf Renderingfunktionen, die beim Debuggen und Optimieren der Leistung von Anwendungen helfen. Diese Schnittstelle kann über die DirectComposition-Geräteschnittstelle abgefragt werden.<br/>                                                                                                                                                                                                                  |
| [**IDCompositionEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioneffect)<br/>                                       | Stellt einen Bitmapeffekt dar, der die Rasterung der Unterstruktur eines Visuals ändert. <br/>                                                                                                                                                                                                                                                                                                             |
| [**IDCompositionEffectGroup**](/windows/win32/api/dcomp/nn-dcomp-idcompositioneffectgroup)<br/>                             | Stellt eine Gruppe von Bitmapeffekten dar, die zusammen angewendet werden, um die Rasterung der Unterstruktur eines Visuals zu ändern. <br/>                                                                                                                                                                                                                                                                             |
| [**IDCompositionFilterEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionfiltereffect)<br/>                           | Stellt einen Filtereffekt dar.<br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionFloodEffect**](/previous-versions/windows/desktop/legacy/dn919732(v=vs.85))<br/>                             | Der Überflutungseffekt wird verwendet, um eine Bitmap basierend auf der angegebenen Farbe und dem angegebenen Alphawert zu generieren. Sie können diesen Effekt verwenden, wenn Sie eine bestimmte Farbe als Eingabe für einen Effekt wie eine Hintergrundfarbe verwenden möchten. <br/>                                                                                                                                                                                        |
| [**IDCompositionVertriebssianBlurEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositiongaussianblureffect)<br/>               |                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IDCompositionHueRotationEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionhuerotationeffect)<br/>                 | Der Farbtonrotationseffekt ändert den Farbton eines Bilds, indem eine Farbmatrix basierend auf dem Drehwinkel angewendet wird. <br/>                                                                                                                                                                                                                                                                                      |
| [**IDCompositionLinearTransferEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionlineartransfereffect)<br/>           | Der lineare Übertragungseffekt wird verwendet, um die Farbdichten eines Bilds mithilfe einer linearen Funktion zuzuordnen, die aus einer Liste von Werten erstellt wurde, die Sie für jeden Kanal angeben. <br/>                                                                                                                                                                                                                               |
| [**IDCompositionMatrixTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionmatrixtransform)<br/>                     | Stellt eine beliebige affine 2D-Transformation dar, die durch eine 3:2-Matrix definiert wird.<br/>                                                                                                                                                                                                                                                                                                                   |
| [**IDCompositionMatrixTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionmatrixtransform3d)<br/>                 | Stellt eine beliebige 3D-Transformation dar, die durch eine 4-mal-4-Matrix definiert wird.<br/>                                                                                                                                                                                                                                                                                                                          |
| [**IDCompositionRectangleClip**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrectangleclip)<br/>                         | Stellt ein Clipobjekt dar, das das Rendering einer visuellen Teilstruktur auf den angegebenen rechteckigen Bereich beschränkt. Optional kann für das Clipobjekt abgerundete Ecken angegeben sein.<br/>                                                                                                                                                                                                                 |
| [**IDCompositionRotateTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrotatetransform)<br/>                     | Stellt eine 2D-Transformation dar, die sich auf die Drehung eines Visuals um die Z-Achse auswirkt. Das Koordinatensystem wird um den angegebenen Mittelpunkt gedreht. <br/>                                                                                                                                                                                                                                   |
| [**IDCompositionSaturationEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsaturationeffect)<br/>                   | Dieser Effekt wird verwendet, um die Sättigung eines Bilds zu ändern. Der Sättigungseffekt ist eine Spezialisierung des Farbmatrixeffekts. <br/>                                                                                                                                                                                                                                                                |
| [**IDCompositionRotateTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrotatetransform3d)<br/>                 | Stellt eine 3D-Transformation dar, die sich auf die Drehung eines Visuals entlang einer beliebigen Achse im 3D-Raum auswirkt. Das Koordinatensystem wird um den angegebenen Mittelpunkt gedreht. <br/>                                                                                                                                                                                                                 |
| [**IDCompositionScaleTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionscaletransform)<br/>                       | Stellt eine 2D-Transformation dar, die sich auf die Skalierung eines Visuals entlang der x- und y-Achse auswirkt. Das Koordinatensystem wird vom angegebenen Mittelpunkt aus skaliert. <br/>                                                                                                                                                                                                                               |
| [**IDCompositionScaleTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionscaletransform3d)<br/>                   | Stellt einen 3D-Transformationseffekt dar, der sich auf die Skalierung eines Visuals entlang der x-Achse, y-Achse und Z-Achse auswirkt. Das Koordinatensystem wird vom angegebenen Mittelpunkt aus skaliert. <br/>                                                                                                                                                                                                               |
| [**IDCompositionSkewTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionskewtransform)<br/>                         | Stellt eine 2D-Transformation dar, die sich auf die Schiefe eines Visuals entlang der x-Achse und der y-Achse auswirkt. Das Koordinatensystem wird um den angegebenen Mittelpunkt verzerrt. <br/>                                                                                                                                                                                                                              |
| [**IDCompositionShadowEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionshadoweffect)<br/>                           | Der Schatteneffekt wird verwendet, um einen Schatten aus dem Alphakanal eines Bilds zu generieren. Der Schatten ist für höhere Alphawerte nicht transparent und für niedrigere Alphawerte transparenter. Sie können die Menge an Weichzeichnern und die Farbe des Schattens festlegen. <br/>                                                                                                                                                 |
| [**IDCompositionSurface**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsurface)<br/>                                     | Stellt eine physische Bitmap dar, die einem Visuellen für die Komposition in einer visuellen Struktur zugeordnet werden kann. Diese Schnittstelle kann auch verwendet werden, um den Bitmapinhalt zu aktualisieren. <br/>                                                                                                                                                                                                                            |
| [**IDCompositionTableTransferEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontabletransfereffect)<br/>             | Der Tabellenübertragungseffekt wird verwendet, um die Farbdichten eines Bilds mithilfe einer Übertragungsfunktion zuzuordnen, die aus der Interpolation einer liste von Ihnen angegebener Werte erstellt wurde.<br/>                                                                                                                                                                                                                                  |
| [**IDCompositionSurfaceFactory**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsurfacefactory)<br/>                       | Erstellt Oberflächen- und virtuelle Oberflächenobjekte, die einem von der Anwendung bereitgestellten Renderinggerät zugeordnet sind.<br/>                                                                                                                                                                                                                                                                                          |
| [**IDCompositionTarget**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontarget)<br/>                                       | Stellt eine Bindung zwischen einer visuellen DirectComposition-Struktur und einem Ziel dar, auf dem die visuelle Struktur zusammengesetzt werden soll. <br/>                                                                                                                                                                                                                                                            |
| [**IDCompositionTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontransform)<br/>                                 | Stellt eine 2D-Transformation dar, die verwendet werden kann, um den Koordinatenbereich einer visuellen Teilstruktur zu ändern. <br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontransform3d)<br/>                             | Stellt einen 3D-Transformationseffekt dar, der verwendet werden kann, um die Rasterung einer visuellen Teilstruktur zu ändern. <br/>                                                                                                                                                                                                                                                                                       |
| [**IDCompositionTranslateTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontranslatetransform)<br/>               | Stellt eine 2D-Transformation dar, die sich nur auf den Offset eines Visuals entlang der x-Achse und der y-Achse auswirkt.<br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionTurbulenceEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionturbulenceeffect)<br/>                   | Der Effekt wird verwendet, um eine Bitmap basierend auf der Perlin-Rauschfunktion zu generieren. Der Effekt hat kein Eingabebild. <br/>                                                                                                                                                                                                                                                              |
| [**IDCompositionTranslateTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontranslatetransform3d)<br/>           | Stellt eine 3D-Transformation dar, die sich auf den Offset eines Visuals entlang der x-Achse, y-Achse und Z-Achse auswirkt. <br/>                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVirtualSurface**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvirtualsurface)<br/>                       | Stellt eine wenig zugeordnete Bitmap dar, die einem Visuellen für die Komposition in einer visuellen Struktur zugeordnet werden kann.<br/>                                                                                                                                                                                                                                                                                  |
| [**IDCompositionVisual**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual)<br/>                                       | Stellt ein DirectComposition-Visual dar. <br/>                                                                                                                                                                                                                                                                                                                                                        |

| [**IDCompositionVisual3**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual3)<br/>                                     | Stellt ein DirectComposition-Visual in einer visuellen Struktur dar.<br/>                                                                                                                                                                                                                                                                                                                                      |
