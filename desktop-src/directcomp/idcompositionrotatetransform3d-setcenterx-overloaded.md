# <a name="idcompositionrotatetransform3dsetcenterx-methods"></a>IDCompositionRotateTransform3D::SetCenterX-Methoden

Ändert oder animiert den Wert der CenterX-Eigenschaft einer 3D-Drehungstransformation. Die CenterX-Eigenschaft gibt die x-Koordinate des Punkts an, um den die Drehung ausgeführt wird. Der Standardwert ist 0 (null).

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                           | BESCHREIBUNG                                            |
|:-----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterx(float))                                     | Ändert den Wert der CenterX-Eigenschaft.<br/>  |
| [**SetCenterX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterx(idcompositionanimation)) | Animiert den Wert der CenterX-Eigenschaft.<br/> |
