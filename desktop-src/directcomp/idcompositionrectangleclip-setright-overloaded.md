# <a name="idcompositionrectangleclipsetright-methods"></a>Idcompositionrechgleclip::-Methode

Animiert oder ändert den Wert der Right-Eigenschaft eines Clip Rechtecks. Die Rechte Eigenschaft gibt die x-Koordinate der unteren rechten Ecke des Clip Rechtecks an.
