# <a name="idcompositionrectangleclipsetright-methods"></a>IDCompositionRectangleClip::SetRight-Methoden

Animiert oder ändert den Wert der Right-Eigenschaft eines Cliprechtecks. Die Right-Eigenschaft gibt die x-Koordinate der unteren rechten Ecke des Cliprechtecks an.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                   | Beschreibung                                          |
|:---------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetRight(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setright(float))                                     | Ändert den Wert der Right-Eigenschaft.<br/>  |
| [**SetRight(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setright(idcompositionanimation)) | Animiert den Wert der Right-Eigenschaft.<br/> |
