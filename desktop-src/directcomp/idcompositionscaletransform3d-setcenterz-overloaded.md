# <a name="idcompositionscaletransform3dsetcenterz-methods"></a>IDCompositionScaleTransform3D:: setcenterz-Methoden

Ändert oder animiert den Wert der CenterZ-Eigenschaft einer 3D-Skalierungs Transformation. Die CenterZ-Eigenschaft gibt die z-Koordinate des Punkts an, an dem die Skalierung ausgeführt wird.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                          | BESCHREIBUNG                                            |
|:----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Setcenterz (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterz(float))                                     | Ändert den Wert der CenterZ-Eigenschaft.<br/>  |
| [**Setcenterz (idcompositionanimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterz(idcompositionanimation)) | Animiert den Wert der CenterZ-Eigenschaft.<br/> |
