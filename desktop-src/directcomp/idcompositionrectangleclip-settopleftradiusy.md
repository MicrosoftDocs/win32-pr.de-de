# <a name="idcompositionrectangleclipsettopleftradiusy-methods"></a>Idcompositionrechgleclip:: settopleftradi-y-Methoden

Ändert oder animiert den Wert der topleftradi-y-Eigenschaft dieses Clips. Die topleftradiusy-Eigenschaft gibt den y-Radius der Ellipse an, der die linke obere Ecke des Clips rundet.
