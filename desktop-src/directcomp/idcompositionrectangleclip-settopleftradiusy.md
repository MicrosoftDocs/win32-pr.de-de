# <a name="idcompositionrectangleclipsettopleftradiusy-methods"></a>IDCompositionRectangleClip::SetTopLeftRadiusY-Methoden

Ändert oder animiert den Wert der TopLeftRadiusY-Eigenschaft dieses Clips. Die TopLeftRadiusY-Eigenschaft gibt den y-Radius der Ellipse an, die die linke obere Ecke des Clips rundet.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                                     | Beschreibung                                                                |
|:---------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------|
| [**SetTopLeftRadiusY(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settopleftradiusy(idcompositionanimation))                                     | Ändert den Wert der TopLeftRadiusY-Eigenschaft dieses Clips.<br/>  |
| [**SetTopLeftRadiusY(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settopleftradiusy(idcompositionanimation)) | Animiert den Wert der TopLeftRadiusY-Eigenschaft dieses Clips.<br/> |
