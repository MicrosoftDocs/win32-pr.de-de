# <a name="idcompositiontranslatetransformsetoffsety-methods"></a>Idcompositiontranslatetransform::-Methode

Ändert oder animiert den Wert der Off/Ty-Eigenschaft einer 2D-Übersetzungs Transformation. Die offsty-Eigenschaft gibt die Übersetzung entlang der y-Achse an.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                            | BESCHREIBUNG                                            |
|:------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**"Sekunden" (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform-setoffsety(float))                                     | Ändert den Wert der offsty-Eigenschaft.<br/>  |
| [**"*" (Idcompositionanimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform-setoffsety(idcompositionanimation)) | Animiert den Wert der offsty-Eigenschaft.<br/> |
