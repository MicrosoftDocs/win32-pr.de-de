# <a name="idcompositionrotatetransform3dsetaxisz-methods"></a>IDCompositionRotateTransform3D:: setaxisz-Methoden

Ändert oder animiert den Wert der axisz-Eigenschaft einer 3D-Drehungs Transformation. Die axisz-Eigenschaft gibt die z-Koordinate für den Achsen Vektor der Drehung an. Der Standardwert ist 1,0.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                       | BESCHREIBUNG                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**Setaxisz (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisz(float))                                     | Ändert den Wert der axisz-Eigenschaft.<br/>  |
| [**Setaxisz (idcompositionanimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisz(idcompositionanimation)) | Animiert den Wert der axisz-Eigenschaft.<br/> |
