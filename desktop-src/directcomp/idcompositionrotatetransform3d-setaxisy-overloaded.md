# <a name="idcompositionrotatetransform3dsetaxisy-methods"></a>IDCompositionRotateTransform3D:: */taxisy-Methoden

Ändert oder animiert den Wert der AxisY-Eigenschaft einer Drehungs Transformation. Die AxisY-Eigenschaft gibt die y-Koordinate für den Achsen Vektor der Drehung an. Der Standardwert ist 0 (null).

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                       | BESCHREIBUNG                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**"-Taxisy" (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(float))                                     | Ändert den Wert der AxisY-Eigenschaft.<br/>  |
| [**"Eintaxisy" (idcompositionanimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(idcompositionanimation)) | Animiert den Wert der AxisY-Eigenschaft.<br/> |
