# <a name="idcompositiontranslatetransform3dsetoffsety-methods"></a>IDCompositionTranslateTransform3D:: enumerty-Methoden

Ändert oder animiert den Wert der offationseigenschaft eines 3D-Transformations Effekts. Die offsty-Eigenschaft gibt die Übersetzung entlang der y-Achse an.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                              | BESCHREIBUNG                                            |
|:--------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**"Sekunden" (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsety(float))                                     | Ändert den Wert der offsty-Eigenschaft.<br/>  |
| [**"*" (Idcompositionanimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsety(idcompositionanimation)) | Animiert den Wert der offsty-Eigenschaft.<br/> |
