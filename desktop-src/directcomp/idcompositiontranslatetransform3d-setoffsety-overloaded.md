# <a name="idcompositiontranslatetransform3dsetoffsety-methods"></a>IDCompositionTranslateTransform3D::SetOffsetY-Methoden

Ändert oder animiert den Wert der OffsetY-Eigenschaft eines 3D-Übersetzungstransformationseffekts. Die OffsetY-Eigenschaft gibt die Übersetzung entlang der y-Achse an.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                              | Beschreibung                                            |
|:--------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetOffsetY(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsety(float))                                     | Ändert den Wert der OffsetY-Eigenschaft.<br/>  |
| [**SetOffsetY(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsety(idcompositionanimation)) | Animiert den Wert der OffsetY-Eigenschaft.<br/> |
