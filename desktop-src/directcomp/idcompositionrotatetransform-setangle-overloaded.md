# <a name="idcompositionrotatetransformsetangle-methods"></a>Idcompositionrotatetransform::-Methode

Ändert oder animiert den Wert der Angle-Eigenschaft einer Drehungs Transformation. Die Angle-Eigenschaft gibt den Drehungs Winkel in Grad an. Der Standardwert ist 0 (null).

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                     | BESCHREIBUNG                                          |
|:-----------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**(Float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setangle(float))                                     | Ändert den Wert der Angle-Eigenschaft.<br/>  |
| [**"Eintangle" (idcompositionanimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setangle(idcompositionanimation)) | Animiert den Wert der Angle-Eigenschaft.<br/> |
