# <a name="idcompositionrotatetransform3dsetcenterz-methods"></a>IDCompositionRotateTransform3D:: setcenterz-Methoden

Ändert oder animiert den Wert der CenterZ-Eigenschaft einer 3D-Drehungs Transformation. Die CenterZ-Eigenschaft gibt die z-Koordinate des Punkts an, an dem die Drehung ausgeführt wird. Der Standardwert ist 0 (null).

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                           | BESCHREIBUNG                                            |
|:-----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Setcenterz (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterz(float))                                     | Ändert den Wert der CenterZ-Eigenschaft.<br/>  |
| [**Setcenterz (idcompositionanimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterz(idcompositionanimation)) | Animiert den Wert der CenterZ-Eigenschaft.<br/> |
