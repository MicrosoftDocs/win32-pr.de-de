# <a name="idcompositionscaletransformsetcenterx-methods"></a>IDCompositionScaleTransform::SetCenterX-Methoden

Ändert oder animiert den Wert der CenterX-Eigenschaft einer 2D-Skalierungstransformation. Die CenterX-Eigenschaft gibt die x-Koordinate des Punkts an, über den die Skalierung durchgeführt wird.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                        | Beschreibung                                            |
|:--------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setcenterx(float))                                     | Ändert den Wert der CenterX-Eigenschaft.<br/>  |
| [**SetCenterX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setcenterx(idcompositionanimation)) | Animiert den Wert der CenterX-Eigenschaft.<br/> |
