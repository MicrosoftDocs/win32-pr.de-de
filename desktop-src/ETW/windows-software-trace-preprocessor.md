# <a name="writing-wpp-events"></a>Schreiben von WPP-Ereignissen

Der Windows Software Trace Processor (WPP) bietet einen effizienten Mechanismus zum Protokollieren von Ereignissen, die während der Ausführung einer Anwendung oder eines Treibers auftreten. WPP verwendet die ETW-API.

Ausführliche Informationen zur Verwendung von WPP zum Protokollieren von Ereignissen finden Sie unter Häufig gestellte Fragen zur [WPP-Software Ablauf Verfolgung](/windows-hardware/drivers/devtest/wpp-software-tracing) und [Software Ablauf Verfolgung](/windows-hardware/drivers/devtest/software-tracing-faq) im Microsoft Windows Driver Development Kit (DDK).
