# <a name="event-tracing-tools"></a>Ereignisablaufverfolgungstools

Die folgenden Tools für die Ereignisablaufverfolgung sind für Ihre Verwendung verfügbar.



| Tool     | Beschreibung                                                                                                                                                                                                                                                                                                                                |
|----------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Logman   | Verwaltet und plant Sammlungen von Leistungsindikatoren und Ereignisablaufverfolgungsprotokollen auf lokalen und Remotesystemen. Mit diesem Tool können Sie die Anbieter auflisten, [die das Layout ihrer Ereignisdaten](publishing-your-event-schema.md) im \\ wmi-Stammnamespace veröffentlicht haben. Ausführliche Informationen zur Verwendung dieses Tools finden Sie im **Hilfe- und Supportcenter**. |
| Tracefmt | Formatiert Informationen in einer Ablaufverfolgungsprotokolldatei in lesbarer Form. Weitere Informationen finden Sie unter Tools für die Softwareablaufverfolgung im Microsoft Windows Driver Development Kit (DDK). Wird nur von der WPP-Softwareablaufverfolgung verwendet.                                                                                                                                      |
| Ablaufverfolgungsprotokoll | Steuert eine Ereignisablaufverfolgungssitzung. Weitere Informationen finden Sie unter Tools für die Softwareablaufverfolgung im DDK.                                                                                                                                                                                                                                                   |
| Tracepdb | Erstellt eine Formatdatei für Ablaufverfolgungsmeldungen, die das Tracefmt-Tool verwendet, um protokollierte Nachrichten in ein lesbares Formular zu konvertieren. Weitere Informationen finden Sie unter Tools für die Softwareablaufverfolgung im DDK. Wird nur von der WPP-Softwareablaufverfolgung verwendet.                                                                                                                            |
| Tracerpt | Verarbeitet Ereignisablaufverfolgungsprotokolle oder Echtzeitereignisse von instrumentierten Ereignisablaufverfolgungsanbietern und ermöglicht ihnen das Generieren von Berichten zur Ablaufverfolgungsanalyse und CSV-Dateien (durch Trennzeichen getrennte Dateien) für die generierten Ereignisse. Weitere Informationen finden Sie im **Hilfe- und Supportcenter**.                                                                                       |
