# <a name="writing-related-events-in-an-end-to-end-scenario"></a>Schreiben verwandter Ereignisse in einem End-to-End-Szenario

ETW bietet eine Möglichkeit, verwandte Ereignisse aus mehr als einer Komponente zu gruppen. Wenn beispielsweise mehrere Komponenten (entweder auf demselben Computer oder auf verschiedenen Computern) an der Verarbeitung derselben Daten beteiligt sind und jede Komponente Ereignisse für ihren Teil der Aktivität protokolliert, können Sie alle zugehörigen Ereignisse gruppen. Dadurch kann ein Consumer alle Ereignisse vom Anfang des Prozesses bis zum Ende des Prozesses verarbeiten.

Informationen zum Schreiben verwandter Ereignisse in [einem manifestbasierten](about-event-tracing.md) Anbieter finden Sie unter [Schreiben verwandter Ereignisse in einem manifestbasierten Anbieter.](writing-related-events-in-a-manifest-base-provider.md)

Informationen zum Schreiben verwandter Ereignisse in einem [klassischen Anbieter](about-event-tracing.md) finden Sie unter [Schreiben verwandter Ereignisse in einem klassischen Anbieter.](tracing-event-instances.md)
