# <a name="device-drivers"></a>Gerätetreiber

Diese Dokumentation ist für Benutzermodusanwendungen vorgesehen, die etw für die Ereignis Ablauf Verfolgung verwenden möchten. Diese Dokumentation ist nicht für Gerätetreiber vorgesehen, die im Kernel Modus ausgeführt werden. Informationen zum Erstellen von Problemen mit der zeitlichen Steuerung finden Sie unter [Ereignis Ablauf Verfolgung für Windows](/windows-hardware/drivers/devtest/event-tracing-for-windows--etw-), [WPP-Software](/windows-hardware/drivers/devtest/wpp-software-tracing)Ablauf Verfolgung und [WMI-Ereignis Ablauf Verfolgung](/windows-hardware/drivers/kernel/wmi-event-tracing) im Windows-Treiberkit (WDK), um Informationen zum Instrumentieren Ihres Treibers zu erhalten.
