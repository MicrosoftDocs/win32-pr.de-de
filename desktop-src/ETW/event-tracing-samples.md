# <a name="event-tracing-samples"></a>Beispiele zur Ereignis Ablauf Verfolgung

Das Platform Software Development Kit (SDK) enthält Beispiele für die komplette Ereignis Ablauf Verfolgung. Diese Beispiele befinden sich im \\ Verzeichnis Winbase \\ EventTrace. Der Stamm dieses Pfades ist das Basis Installationsverzeichnis des PSDK. Sie können das PSDK von herunterladen [https://www.microsoft.com/downloads/details.aspx?FamilyId=A55B6B43-E24F-4EA3-A93E-40C0EC4F68E5](https://msdn.microsoft.com/windowsserver/bb980924.aspx) .

In der folgenden Tabelle werden die Beispiele beschrieben, die in den Beispielen \\ Winbase \\ EventTrace-Verzeichnis enthalten sind.



| Beispiel   | BESCHREIBUNG                                                                                                                            |
|----------|----------------------------------------------------------------------------------------------------------------------------------------|
| TraceDmp | Ein Ereignis Ablauf Verfolgungs Consumer. Die Ereignisdaten werden mithilfe der von WMI erhaltenen Formatinformationen decodiert, und die Daten werden in einer CSV-Datei ausgegeben. |
| Tracedp  | Verwendet die Ereignis Ablauf Verfolgungs Anbieter-API zur Bereitstellung von Ereignis Ablauf Verfolgungs Daten für die Protokollierung oder einen Consumer.                                             |
| TraceLog | Verwendet die Ereignis Ablauf Verfolgungs Controller-API, um Protokollierungs Sitzungen zu verwalten.                                                                        |



 

Ausführliche Beschreibungen finden Sie in der Readme.txt-Datei, die in diesen Beispielen enthalten ist.
