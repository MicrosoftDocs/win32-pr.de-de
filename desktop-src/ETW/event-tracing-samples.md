# <a name="event-tracing-samples"></a>Beispiele für die Ereignisablaufverfolgung

Das Platform Software Development Kit (SDK) enthält vollständige Beispiele für die Ereignisablaufverfolgung. Diese Beispiele befinden sich im \\ WinBase \\ EventTrace-Beispielverzeichnis. Der Stamm dieses Pfads ist das Basisinstallationsverzeichnis des PSDK. Sie können das PSDK unter [https://www.microsoft.com/downloads/details.aspx?FamilyId=A55B6B43-E24F-4EA3-A93E-40C0EC4F68E5](https://msdn.microsoft.com/windowsserver/bb980924.aspx) herunterladen.

In der folgenden Tabelle werden die Beispiele beschrieben, die im Verzeichnis Samples \\ WinBase \\ EventTrace enthalten sind.



| Beispiel   | BESCHREIBUNG                                                                                                                            |
|----------|----------------------------------------------------------------------------------------------------------------------------------------|
| TraceDmp | Ein Ereignis-Ablaufverfolgungs-Consumer. Sie decodiert die Ereignisdaten mithilfe der von WMI erhaltenen Formatinformationen und gibt die Daten in einer .csv aus. |
| TraceDp  | Verwendet die Ereignisverfolgungsanbieter-API, um Ereignisverfolgungsdaten für die Protokollierung oder einen Consumer zur Verfügung zu stellen.                                             |
| TraceLog | Verwendet die Ereignisverfolgungscontroller-API zum Verwalten von Protokollierungssitzungen.                                                                        |



 

Ausführliche Beschreibungen finden Sie in der Readme.txt in diesen Beispielen enthaltenen Datei.
