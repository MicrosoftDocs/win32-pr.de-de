# <a name="thread_v1-class"></a>Thread \_ v1-Klasse

Diese Klasse ist die übergeordnete Klasse für Thread Ereignisse.

Die folgende Syntax wird durch den MOF-Code vereinfacht.

## <a name="syntax"></a>Syntax

``` syntax
[Guid("{3d6fa8d1-fe05-11d0-9dda-00c04fd7ba7c}"), EventVersion(1)]
class Thread_V1 : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Member

Die **Thread \_ v1** -Klasse definiert keine Member.
