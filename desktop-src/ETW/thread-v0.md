Diese Klasse ist die übergeordnete Klasse für Threadereignisse.

``` syntax
[Guid("{3d6fa8d1-fe05-11d0-9dda-00c04fd7ba7c}"), EventVersion(0)]
class Thread_V0 : MSNT_SystemTrace
```
