# <a name="retrieving-event-data"></a>Abrufen von Ereignisdaten

Um ereignisspezifische Daten zu nutzen, muss der Consumer das Format der Ereignisdaten kennen. Dies ist kein Problem, wenn Sie Ereignisse von Ihrem eigenen Anbieter nutzen, da Sie das Format der Daten kennen. Um jedoch ein Ereignis von einem beliebigen Anbieter zu nutzen, muss der Anbieter das Layout der Ereignisdaten veröffentlichen. Einen Überblick darüber, wie Ereignis Metadaten funktionieren und wie Sie Ihre Ereignis Metadaten veröffentlichen, damit andere Benutzer Sie verwenden können, finden Sie unter [Übersicht über Ereignis Metadaten](event-metadata-overview.md).

Informationen zur Nutzung von Ereignissen unter Windows Vista, die mithilfe von Manifest-, MOF-oder TMF-Dateien sowie tracelogging-Ereignissen veröffentlicht wurden, finden Sie unter [Abrufen von Ereignisdaten mit TDH](retrieving-event-data-using-tdh.md).

Informationen zur Nutzung von Ereignissen vor Windows Vista, die mithilfe von MOF veröffentlicht wurden, finden Sie unter [Abrufen von Ereignisdaten mithilfe von MOF](retrieving-event-data-using-mof.md).
