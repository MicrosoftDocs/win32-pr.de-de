# <a name="retrieving-event-data"></a>Abrufen von Ereignisdaten

Um ereignisspezifische Daten zu nutzen, muss der Consumer das Format der Ereignisdaten kennen. Dies ist kein Problem, wenn Sie Ereignisse von Ihrem eigenen Anbieter nutzen, da Sie das Format der Daten kennen. Um jedoch ein Ereignis von einem beliebigen Anbieter zu nutzen, muss der Anbieter das Layout der Ereignisdaten veröffentlichen. Eine Übersicht über die Funktionsweise von Ereignismetadaten sowie die Veröffentlichung Ihrer Ereignismetadaten, damit andere sie verwenden können, finden Sie unter [Übersicht über Ereignismetadaten.](event-metadata-overview.md)

Informationen zum Nutzen von Ereignissen auf Windows Vista, die mithilfe eines Manifests, mof oder TMF-Dateien veröffentlicht wurden, sowie TraceLogging-Ereignisse finden Sie unter [Abrufen von Ereignisdaten mit TDH.](retrieving-event-data-using-tdh.md)

Informationen zum Nutzen von Ereignissen vor Windows Vista, die mit MOF veröffentlicht wurden, finden Sie unter [Abrufen von Ereignisdaten mit MOF.](retrieving-event-data-using-mof.md)
