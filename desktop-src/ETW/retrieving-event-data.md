# <a name="retrieving-event-data"></a>Abrufen von Ereignisdaten

Um ereignisspezifische Daten zu nutzen, muss der Consumer das Format der Ereignisdaten kennen. Dies ist kein Problem, wenn Sie Ereignisse von Ihrem eigenen Anbieter nutzen, da Sie das Format der Daten kennen. Um jedoch ein Ereignis von einem beliebigen Anbieter nutzen zu können, muss der Anbieter das Layout der Ereignisdaten veröffentlichen. Eine Übersicht über die Funktionsweise von Ereignismetadaten sowie darüber, wie Sie Ihre Ereignismetadaten veröffentlichen, damit andere sie verwenden können, finden Sie unter [Übersicht über Ereignismetadaten.](event-metadata-overview.md)

Informationen zum Nutzen von Ereignissen unter Windows Vista, die mithilfe von Manifest-, MOF- oder TMF-Dateien veröffentlicht wurden, sowie TraceLogging-Ereignisse finden Sie unter [Abrufen von Ereignisdaten mit TDH.](retrieving-event-data-using-tdh.md)

Informationen zur Nutzung von Ereignissen vor Windows Vista, die mit MOF veröffentlicht wurden, finden Sie unter [Abrufen von Ereignisdaten mit MOF.](retrieving-event-data-using-mof.md)
