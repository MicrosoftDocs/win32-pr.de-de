# <a name="retrieving-event-data-using-tdh"></a>Abrufen von Ereignisdaten mithilfe von TDH

Um ereignisspezifische Daten zu nutzen, muss der Consumer das Format der Ereignisdaten kennen. Wenn der Anbieter eine Manifest-, MOF-oder TMF-Datei verwendet hat, um das Format der Ereignisdaten zu veröffentlichen, können Sie die Ereignisdaten mithilfe der Funktionen des Trace Data Helper (TDH) analysieren. Die TDH-Funktionen sind ab Windows Vista verfügbar.

In den folgenden Themen wird gezeigt, wie Sie mit TDH Ereignisdaten und Metadaten abrufen.

-   [Verwenden von tdhformatproperty zum Verarbeiten von Ereignisdaten](using-tdhformatproperty-to-consume-event-data.md)
-   [Verwenden von tdhgetproperty zum Verarbeiten von Ereignisdaten](using-tdhgetproperty-to-consume-event-data.md)
-   [Abrufen von Ereignis Metadaten](retrieving-event-metadata.md)
-   [Auflisten von Anbietern](enumerating-providers.md)
