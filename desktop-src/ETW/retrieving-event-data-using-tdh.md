# <a name="retrieving-event-data-using-tdh"></a>Abrufen von Ereignisdaten mit TDH

Um ereignisspezifische Daten zu nutzen, muss der Consumer das Format der Ereignisdaten kennen. Wenn der Anbieter eine Manifest-, MOF- oder TMF-Datei verwendet hat, um das Format der Ereignisdaten zu veröffentlichen, können Sie die Funktionen des Ablaufverfolgungsdaten-Hilfsprogramm (Trace Data Helper, TDH) verwenden, um die Ereignisdaten zu analysieren. Die TDH-Funktionen sind ab Windows Vista verfügbar.

In den folgenden Themen wird gezeigt, wie Sie TDH zum Abrufen von Ereignisdaten und Metadaten verwenden.

-   [Verwenden von TdhFormatProperty zum Verarbeiten von Ereignisdaten](using-tdhformatproperty-to-consume-event-data.md)
-   [Verwenden von TdhGetProperty zum Nutzen von Ereignisdaten](using-tdhgetproperty-to-consume-event-data.md)
-   [Abrufen von Ereignismetadaten](retrieving-event-metadata.md)
-   [Auflisten von Anbietern](enumerating-providers.md)
