# <a name="shape-drawing-functions"></a>Formen Zeichnungsfunktionen

D3DX stellt die folgenden Funktionen zum Zeichnen von Formen bereit.

-   [**D3DXCreateBox**](d3dxcreatebox.md)
-   [**D3DXCreateCylinder**](d3dxcreatecylinder.md)
-   [**D3DXCreatePolygon**](d3dxcreatepolygon.md)
-   [**D3DXCreateSphere**](d3dxcreatesphere.md)
-   [**D3DXCreateTeapot**](d3dxcreateteapot.md)
-   [**D3DXCreateText**](d3dxcreatetext.md)
-   [**D3DXCreateTorus**](d3dxcreatetorus.md)

Alle Formen werden als Gitter Objekte zurückgegeben.

Diese Funktionen können entweder in einem von Links oder rechts gerichteten Koordinatensystem verwendet werden.
