# <a name="colorrgba"></a>Colorrgba

Definiert ein Color-Objekt mit einer Alpha Komponente. Diese wird für die Vordergrundfarbe in der Material Template-Definition verwendet.

``` syntax
template ColorRGBA
{
    < 35FF44E0-6C7C-11cf-8F52-0040333594A3 >
    float red;
    float green;
    float blue;
    float alpha;
} 
```
