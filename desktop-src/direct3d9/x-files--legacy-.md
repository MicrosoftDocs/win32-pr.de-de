# <a name="x-files-legacy-direct3d-9"></a>X Files (Legacy) (Direct3D 9)

Das X-Dateiformat bezieht sich auf Dateien mit der Dateierweiterung .x. X-Dateien wurden mit DirectX 2.0 eingeführt. Eine Binärversion dieses Formats wurde anschließend mit DirectX 3.0 veröffentlicht, was auch in dieser Dokumentation beschrieben wird. Mit DirectX 6.0 wurden Schnittstellen und Methoden eingeführt, die das Lesen aus und Schreiben in X-Dateien ermöglichen.

X-Dateien bieten ein vorlagengesteuertes Format, das die Speicherung von Gitternetzen, Texturen, Animationen und benutzerdefinierbaren Objekten ermöglicht. Mit der Unterstützung für Animationssätze können Sie vordefinierte Pfade für die Wiedergabe in Echtzeit speichern. Instancing und Hierarchien werden ebenfalls unterstützt. Die -Instancing ermöglicht mehrere Verweise auf ein Objekt, z. B. ein Gittermodell, während seine Daten nur einmal pro Datei gespeichert werden. Hierarchien werden verwendet, um Beziehungen zwischen Datensätzen auszudrücken.

Das X-Dateiformat bietet Datenprimitiven auf niedriger Ebene, für die Anwendungen primitivere Primitive auf höherer Ebene über Vorlagen definieren.

Dreidimensionale Modelle, die in maya-Anwendungen von 3ds max oder Alias Wavefront erstellt wurden, können mit den DirectX-Erweiterungen für Alias Maya in \| X-Dateien konvertiert werden.
