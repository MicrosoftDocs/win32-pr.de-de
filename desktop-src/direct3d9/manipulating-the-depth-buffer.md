# <a name="manipulating-the-depth-buffer-direct3d-9"></a>Bearbeiten des tiefen Puffers (Direct3D 9)

Tiefen Puffer werden dem Gerät zugeordnet. Anwendungen müssen die tiefen Puffer verschieben, wenn Sie Renderziele festlegen. Die [**IDirect3DDevice9:: getdepthstencilsurface**](/windows/desktop/api) -Methode und die [**IDirect3DDevice9:: setdepthstencilsurface**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-setdepthstencilsurface) -Methode werden verwendet, um Tiefe Puffer zu bearbeiten.
