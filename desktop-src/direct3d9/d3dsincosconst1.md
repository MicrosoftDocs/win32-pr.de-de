# <a name="d3dsincosconst1"></a>D3DSINCOSCONST1

Erstellt die erste Konstante (in Register Quelle2) für die [SinCos-PS-Pixel-](../direct3dhlsl/sincos---ps.md) Shader-Anweisung.

``` syntax
#define D3DSINCOSCONST1  -1.5500992e-006f, -2.1701389e-005f,  0.0026041667f, 0.00026041668f
```

## <a name="return-value"></a>Rückgabewert

Gibt die erste Konstante zurück, die von [SinCos-PS](../direct3dhlsl/sincos---ps.md)benötigt wird.
