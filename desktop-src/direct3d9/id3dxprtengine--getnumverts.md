# <a name="id3dxprtenginegetnumverts-method"></a>ID3DXPRTEngine::GetNumVerts-Methode

Ruft die Anzahl der Scheitelpunkte im Netz ab, einschließlich aller neuen Scheitelpunkte, die als Ergebnis der adaptiven räumlichen Stichprobenentnahme hinzugefügt wurden.

## <a name="syntax"></a>Syntax


```C++
UINT GetNumVerts();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **UINT**](../winprog/windows-data-types.md)**

Anzahl der Scheitelpunkte im Netz, einschließlich aller neuen Scheitelpunkte, die als Ergebnis der adaptiven Stichprobenentnahme hinzugefügt wurden.
