# <a name="id3dxprtenginegetnumverts-method"></a>ID3DXPRTEngine:: getnumverts-Methode

Ruft die Anzahl der Scheitel Punkte im Mesh ab, einschließlich aller neuen Scheitel Punkte, die als Ergebnis der adaptiven räumlichen Stichprobenentnahme hinzugefügt wurden.

## <a name="syntax"></a>Syntax


```C++
UINT GetNumVerts();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **uint**](../winprog/windows-data-types.md)**

Anzahl der Scheitel Punkte im Mesh, einschließlich neuer Scheitel Punkte, die als Ergebnis der adaptiven Stichprobenentnahme hinzugefügt wurden.
