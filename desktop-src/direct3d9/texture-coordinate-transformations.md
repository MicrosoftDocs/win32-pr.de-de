## <a name="setting-and-retrieving-texture-coordinate-transformations"></a><span data-ttu-id="402ad-114">Festlegen und Abrufen von Texturkoordinaten Transformationen</span><span class="sxs-lookup"><span data-stu-id="402ad-114">Setting and Retrieving Texture Coordinate Transformations</span></span>

<span data-ttu-id="402ad-117">Mit dem folgenden Code wird eine Matrix festgelegt, die auf die Texturkoordinaten für Textur Phase 0 angewendet werden soll.</span><span class="sxs-lookup"><span data-stu-id="402ad-117">The following code sets a matrix to apply to the texture coordinates for texture stage 0.</span></span>




## <a name="enabling-texture-coordinate-transformations"></a><span data-ttu-id="402ad-118">Aktivieren von Texturkoordinaten Transformationen</span><span class="sxs-lookup"><span data-stu-id="402ad-118">Enabling Texture Coordinate Transformations</span></span>

<span data-ttu-id="402ad-119">Der D3DTSS \_ texturetransformflags Textur Phasen Zustand steuert die Anwendung von Texturkoordinaten Transformationen.</span><span class="sxs-lookup"><span data-stu-id="402ad-119">The D3DTSS\_TEXTURETRANSFORMFLAGS texture stage state controls the application of texture coordinate transformations.</span></span> <span data-ttu-id="402ad-120">Werte für diesen Textur Stufen Zustand werden durch den [**D3DTEXTURETRANSFORMFLAGS**](./d3dtexturetransformflags.md) -enumerierten Typ definiert.</span><span class="sxs-lookup"><span data-stu-id="402ad-120">Values for this texture stage state are defined by the [**D3DTEXTURETRANSFORMFLAGS**](./d3dtexturetransformflags.md) enumerated type.</span></span>

<span data-ttu-id="402ad-121">Texturkoordinaten Transformationen werden deaktiviert, wenn D3DTSS \_ texturetransformflags auf D3DTTFF \_ deaktiviert (Standardwert) festgelegt ist.</span><span class="sxs-lookup"><span data-stu-id="402ad-121">Texture coordinate transformations are disabled when D3DTSS\_TEXTURETRANSFORMFLAGS is set to D3DTTFF\_DISABLE (the default value).</span></span> <span data-ttu-id="402ad-122">Wenn Texturkoordinaten Transformationen für Phase 0 aktiviert wurden, werden Sie durch den folgenden Code deaktiviert.</span><span class="sxs-lookup"><span data-stu-id="402ad-122">Assuming that texture coordinate transformations were enabled for stage 0, the following code disables them.</span></span>


```

                                 D3DTTFF_DISABLE );
```



<span data-ttu-id="402ad-123">Die anderen in [**D3DTEXTURETRANSFORMFLAGS**](./d3dtexturetransformflags.md) definierten Werte werden verwendet, um Texturkoordinaten Transformationen zu aktivieren und um zu steuern, wie viele resultierende Texturkoordinaten Elemente an den Rasterizer übermittelt werden.</span><span class="sxs-lookup"><span data-stu-id="402ad-123">The other values defined in [**D3DTEXTURETRANSFORMFLAGS**](./d3dtexturetransformflags.md) are used to enable texture coordinate transformations, and to control how many resulting texture coordinate elements are passed to the rasterizer.</span></span> <span data-ttu-id="402ad-124">Nehmen Sie z. b. den folgenden Code.</span><span class="sxs-lookup"><span data-stu-id="402ad-124">For example, take the following code.</span></span>


```
// For this example, assume the d3dDevice variable contains a 
//   valid pointer to an IDirect3DDevice9 interface.

d3dDevice->SetTextureStageState( 0, D3DTSS_TEXTURETRANSFORMFLAGS, 
                                 D3DTTFF_COUNT2 );
```



<span data-ttu-id="402ad-125">Der D3DTTFF \_ COUNT2-Wert weist das System an, den Transformationsmatrix Satz für Textur Phase 0 anzuwenden und die ersten beiden Elemente der geänderten Texturkoordinaten an den Rasterizer zu übergeben.</span><span class="sxs-lookup"><span data-stu-id="402ad-125">The D3DTTFF\_COUNT2 value instructs the system to apply the transformation matrix set for texture stage 0, and then pass the first two elements of the modified texture coordinates to the rasterizer.</span></span>

<span data-ttu-id="402ad-126">Das \_ Flag für die D3DTTFF projizierte Textur Transformation gibt Koordinaten für eine projizierte Textur an.</span><span class="sxs-lookup"><span data-stu-id="402ad-126">The D3DTTFF\_PROJECTED texture transformation flag indicates coordinates for a projected texture.</span></span> <span data-ttu-id="402ad-127">Wenn dieses Flag angegeben wird, dividiert der Raster die Elemente, die durch das letzte Element übertragen wurden.</span><span class="sxs-lookup"><span data-stu-id="402ad-127">When this flag is specified, the rasterizer divides the elements passed-in by the last element.</span></span> <span data-ttu-id="402ad-128">Nehmen Sie z. b. den folgenden Code.</span><span class="sxs-lookup"><span data-stu-id="402ad-128">Take the following code, for example.</span></span>


```

//   valid pointer to an IDirect3DDevice9 interface.

                                 D3DTTFF_COUNT3 | D3DTTFF_PROJECTED );
```



<span data-ttu-id="402ad-129">In diesem Beispiel wird dem System mitgeteilt, dass drei Texturkoordinaten Elemente an den Rasterizer übergeben werden.</span><span class="sxs-lookup"><span data-stu-id="402ad-129">This example informs the system to pass three texture coordinate elements to the rasterizer.</span></span> <span data-ttu-id="402ad-130">Der Raster dividiert die ersten beiden Elemente durch den dritten und erzeugt die 2D-Texturkoordinaten, die zur Adressierung der Textur benötigt werden.</span><span class="sxs-lookup"><span data-stu-id="402ad-130">The rasterizer divides the first two elements by the third, producing the 2D texture coordinates needed to address the texture.</span></span>
