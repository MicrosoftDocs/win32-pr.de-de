# <a name="shadowing-direct3d-9"></a>Shadowing (Direct3D 9)

Schablonenpuffer sind eine Tiefenpuffertechnik, die beim Rendern der Geometrie aktualisiert und wieder als Maske zum Zeichnen weiterer Geometrie verwendet werden kann. Häufige Effekte sind Spiegel, Schatten (eine erweiterte Technik), Beeinträchtigungen usw. Das [ShadowVolume-Beispiel](https://msdn.microsoft.com/library/Ee418792(v=VS.85).aspx) verwendet Schablonenpuffer, um Echtzeitschatten zu implementieren. Zusätzliche Informationen sind in [der zweiseitigen Schablone (Direct3D 9)](two-sided-stencil.md)enthalten.
