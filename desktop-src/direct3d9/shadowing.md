# <a name="shadowing-direct3d-9"></a>Shadowing(Direct3D 9)

Schablonen Puffer sind ein tiefpufferverfahren, das aktualisiert werden kann, wenn Geometry gerendert wird, und wieder als Maske zum Zeichnen von mehr Geometrie verwendet. Zu den häufigen Effekten zählen Spiegelungen, Schatten (eine erweiterte Technik), eine Aufwirkung usw. Das [shadowvolume-Beispiel](https://msdn.microsoft.com/library/Ee418792(v=VS.85).aspx) verwendet Schablonen Puffer, um Echt Zeit Schatten zu implementieren. Weitere Informationen sind in der [zweiseitigen Schablone (Direct3D 9)](two-sided-stencil.md)enthalten.
