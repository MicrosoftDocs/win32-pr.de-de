# <a name="d3dusage"></a>D3DUSAGE

Verwendungsoptionen, die bestimmen, wie Ressourcen verwendet werden sollen.

In der folgenden Tabelle sind die verfügbaren Verwendungsoptionen zusammengefasst.



| Konstante                                                                                                                                                                                                                                          | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="D3DUSAGE_AUTOGENMIPMAP"></span><span id="d3dusage_autogenmipmap"></span><dl> <dt>**D3DUSAGE \_ AUTOGENMIPMAP**</dt> </dl>                                                          | Die Ressource generiert automatisch Mipmaps. Weitere Informationen finden Sie unter [Automatische Generierung von Mipmaps (Direct3D 9).](automatic-generation-of-mipmaps.md) Die automatische Generierung von Mipmaps wird für Volumentexturen und Tiefenschablonenoberflächen/Texturen nicht unterstützt. Diese Verwendung ist für eine Ressource im Systemspeicher (**D3DPOOL \_ SYSTEMMEM**) ungültig.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="D3DUSAGE_DEPTHSTENCIL"></span><span id="d3dusage_depthstencil"></span><dl> <dt>**D3DUSAGE \_ DEPTHSTENCIL**</dt> </dl>                                                             | Die Ressource ist ein Tiefenschablonenpuffer. **D3DUSAGE \_ DEPTHSTENCIL** kann nur mit **D3DPOOL \_ DEFAULT** verwendet werden.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="D3DUSAGE_DMAP"></span><span id="d3dusage_dmap"></span><dl> <dt>**D3DUSAGE \_ DMAP**</dt> </dl>                                                                                     | Die Ressource ist eine Verschiebungszuordnung.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="D3DUSAGE_DONOTCLIP"></span><span id="d3dusage_donotclip"></span><dl> <dt>**D3DUSAGE \_ DONOTCLIP**</dt> </dl>                                                                      | Legen Sie diese Einstellung fest, um anzugeben, dass für den Inhalt des Scheitelpunktpuffers kein Clipping erforderlich ist. Beim Rendern mit Puffern, für die dieses Flag festgelegt ist, muss der **D3DRS \_ CLIPPING-Renderzustand** auf FALSE festgelegt werden.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <span id="D3DUSAGE_DYNAMIC"></span><span id="d3dusage_dynamic"></span><dl> <dt>**D3DUSAGE \_ DYNAMIC**</dt> </dl>                                                                            | Legen Sie diese Einstellung fest, um anzugeben, dass der Scheitelpunktpuffer eine dynamische Speicherverwendung erfordert. Dies ist für Treiber nützlich, da sie entscheiden können, wo der Puffer abgelegt werden soll. Im Allgemeinen werden statische Scheitelpunktpuffer im Videospeicher und dynamische Scheitelpunktpuffer im AGP-Speicher platziert. Beachten Sie, dass es keine separate statische Verwendung gibt. Wenn Sie **D3DUSAGE \_ DYNAMIC** nicht angeben, wird der Scheitelpunktpuffer als statisch festgelegt. **D3DUSAGE \_ DYNAMIC** wird streng durch die **Sperrflags D3DLOCK \_ DISCARD** und **D3DLOCK \_ NOOVERWRITE** erzwungen. **D3DLOCK \_ DISCARD** und **D3DLOCK \_ NOOVERWRITE** sind daher nur für Scheitelpunktpuffer gültig, die mit **D3DUSAGE \_ DYNAMIC** erstellt wurden. Sie sind keine gültigen Flags für statische Scheitelpunktpuffer. Weitere Informationen finden Sie unter [Verwalten von Ressourcen (Direct3D 9).](managing-resources.md) <br/> Weitere Informationen zur Verwendung dynamischer Scheitelpunktpuffer finden Sie unter [Leistungsoptimierungen (Direct3D 9).](performance-optimizations.md)<br/> **D3DUSAGE \_ DYNAMIC** und **D3DPOOL \_ MANAGED** sind inkompatibel und sollten nicht zusammen verwendet werden. Siehe [**D3DPOOL**](./d3dpool.md).<br/> Texturen können **D3DUSAGE \_ DYNAMIC** angeben. Verwaltete Texturen können jedoch nicht **D3DUSAGE \_ DYNAMIC** verwenden. Weitere Informationen zu dynamischen Texturen finden Sie unter [Verwenden von dynamischen Texturen.](performance-optimizations.md) <br/> |
| <span id="D3DUSAGE_NONSECURE"></span><span id="d3dusage_nonsecure"></span><dl> <dt>**D3DUSAGE \_ NICHT UNSICHER**</dt> </dl>                                                                      | Lassen Sie zu, dass eine freigegebene Oberfläche, die von einer sicheren Anwendung erstellt wurde, von einer nicht sicheren Anwendung geöffnet wird, die über das freigegebene Handle verfügt. <br/> **Unterschiede zwischen Direct3D 9 und Direct3D 9Ex:** Dieses Flag ist nur in Direct3D 9Ex verfügbar.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="D3DUSAGE_NPATCHES"></span><span id="d3dusage_npatches"></span><dl> <dt>**\_D3DUSAGE-NPATCHES**</dt> </dl>                                                                         | Legen Sie diese Einstellung fest, um anzugeben, dass der Scheitelpunktpuffer zum Zeichnen von N-Patches verwendet werden soll.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <span id="D3DUSAGE_POINTS"></span><span id="d3dusage_points"></span><dl> <dt>**D3DUSAGE-PUNKTE \_**</dt> </dl>                                                                               | Legen Sie diese Einstellung fest, um anzugeben, dass der Scheitelpunkt oder Indexpuffer für Zeichnungspunkt-Sprites verwendet wird. Der Puffer wird in den Systemspeicher geladen, wenn die Verarbeitung von Softwarevertex zum Emulieren von Punkt-Sprites erforderlich ist.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="D3DUSAGE_RENDERTARGET"></span><span id="d3dusage_rendertarget"></span><dl> <dt>**D3DUSAGE \_ RENDERTARGET**</dt> </dl>                                                             | Die Ressource ist ein Renderziel. **D3DUSAGE \_ RENDERTARGET** kann nur mit **D3DPOOL \_ DEFAULT** verwendet werden.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="D3DUSAGE_RTPATCHES"></span><span id="d3dusage_rtpatches"></span><dl> <dt>**D3DUSAGE \_ RTPATCHES**</dt> </dl>                                                                      | Legen Sie diese Einstellung fest, um anzugeben, dass der Scheitelpunktpuffer zum Zeichnen von primitiven Typen hoher Ordnung verwendet werden soll.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="D3DUSAGE_SOFTWAREPROCESSING"></span><span id="d3dusage_softwareprocessing"></span><dl> <dt>**D3DUSAGE \_ SOFTWAREPROCESSING**</dt> </dl>                                           | Wenn dieses Flag verwendet wird, erfolgt die Scheitelpunktverarbeitung in der Software. Wenn dieses Flag nicht verwendet wird, erfolgt die Scheitelpunktverarbeitung auf Hardware.<br/> Das **D3DUSAGE \_ SOFTWAREPROCESSING-Flag** kann festgelegt werden, wenn die Verarbeitung im gemischten Modus oder die Verarbeitung von Softwarevertex (**D3DCREATE \_ MIXED \_ VERTEXPROCESSING**  /  **D3DCREATE \_ SOFTWARE \_ VERTEXPROCESSING**) für dieses Gerät aktiviert ist. **D3DUSAGE \_ SOFTWAREPROCESSING** muss für Puffer festgelegt werden, die mit der Softwarevertexverarbeitung im gemischten Modus verwendet werden sollen. Sie sollte jedoch nicht für die bestmögliche Leistung festgelegt werden, wenn die Hardwareindexverarbeitung im gemischten Modus verwendet wird (**D3DCREATE \_ HARDWARE \_ VERTEXPROCESSING**). Das Festlegen von **D3DUSAGE \_ SOFTWAREPROCESSING** ist jedoch die einzige Option, wenn ein einzelner Puffer sowohl bei der Hardware- als auch bei der Softwarevertexverarbeitung verwendet wird. **D3DUSAGE \_ SOFTWAREPROCESSING** ist für gemischte und Softwaregeräte zulässig.<br/> **D3DUSAGE \_ SOFTWAREPROCESSING** wird mit [**CheckDeviceFormat**](/windows/win32/api/d3d9/nf-d3d9-idirect3d9-checkdeviceformat) verwendet, um herauszufinden, ob ein bestimmtes Texturformat während der Verarbeitung von Softwarevertex als Scheitelpunkttextur verwendet werden kann. Wenn dies dere ist, muss die Textur in **D3DPOOL \_ SCRATCH** erstellt werden.<br/>                                                                                                                                                                 |
| <span id="D3DUSAGE_TEXTAPI"></span><span id="d3dusage_textapi"></span><dl> <dt>**D3DUSAGE \_ TEXTAPI**</dt> </dl>                                                                            | Dieses Verwendungsflag muss für Scheitelpunktpuffer und Quelloberflächen angegeben werden, die in Aufrufen von [**ComposeRects**](/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9ex-composerects)verwendet werden. Mit diesem Verwendungsflag erstellte T-Extures können nicht für die Texturfilterung verwendet werden. Vertexpuffer, die mit diesem Verwendungsflag erstellt wurden, können nicht als Eingabestreamquellen verwendet werden. <br/> **Unterschiede zwischen Direct3D 9 und Direct3D 9Ex:** Dieses Flag ist nur in Direct3D 9Ex verfügbar.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="D3DUSAGE_WRITEONLY"></span><span id="d3dusage_writeonly"></span><dl> <dt>**D3DUSAGE \_ WRITEONLY**</dt> </dl>                                                                      | Informiert das System, dass die Anwendung nur in den Scheitelpunktpuffer schreibt. Die Verwendung dieses Flags ermöglicht es dem Treiber, den besten Speicherort für effiziente Schreibvorgänge und Rendering auszuwählen. Beim Lesen aus einem Scheitelpunktpuffer, der mit dieser Funktion erstellt wird, tritt ein Fehler auf. Bei Puffern, die mit **D3DPOOL \_ DEFAULT** erstellt wurden und **D3DUSAGE \_ WRITEONLY** nicht angeben, kann es zu schwerwiegenden Leistungseinbußen kommen. **D3DUSAGE \_ WRITEONLY** wirkt sich nur auf die Leistung von **D3DPOOL \_ DEFAULT-Puffern** aus.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="D3DUSAGE_RESTRICTED_CONTENT"></span><span id="d3dusage_restricted_content"></span><dl> <dt>**D3DUSAGE \_ RESTRICTED \_ CONTENT**</dt> </dl>                                          | Das Festlegen dieses Flags gibt an, dass die Ressource möglicherweise geschützten Inhalt enthält.<br/> **Unterschiede zwischen Direct3D 9 und Direct3D 9Ex:** Dieses Flag ist nur in Direct3D 9Ex verfügbar.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <span id="D3DUSAGE_RESTRICT_SHARED_RESOURCE"></span><span id="d3dusage_restrict_shared_resource"></span><dl> <dt>**D3DUSAGE \_ RESTRICT \_ SHARED \_ RESOURCE**</dt> </dl>                       | Das Festlegen dieses Flags gibt an, dass der Zugriff auf die freigegebene Ressource eingeschränkt werden soll. <br/> **Unterschiede zwischen Direct3D 9 und Direct3D 9Ex:** Dieses Flag ist nur in Direct3D 9Ex verfügbar.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="D3DUSAGE_RESTRICT_SHARED_RESOURCE_DRIVER"></span><span id="d3dusage_restrict_shared_resource_driver"></span><dl> <dt>**D3DUSAGE \_ RESTRICT \_ SHARED \_ RESOURCE \_ DRIVER**</dt> </dl> | Durch Festlegen dieses Flags wird angegeben, dass der Treiber den Zugriff auf die freigegebene Ressource einschränken soll. Der Aufrufer muss einen authentifizierten Kanal mit dem Treiber erstellen. Der Treiber sollte dann den Zugriff auf Prozesse zulassen, die versuchen, diese freigegebene Ressource zu öffnen.<br/> **Unterschiede zwischen Direct3D 9 und Direct3D 9Ex:** Dieses Flag ist nur in Direct3D 9Ex verfügbar.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |

|----------------------------------|----------------------|---------------------|----------------|---------------------|-----------------------|----------------|---------------------|
| **D3DUSAGE \_ AUTOGENMIPMAP**      |                      |                     | x              | x                   |                       |                | x                   |
| **D3DUSAGE \_ DEPTHSTENCIL**       |                      |                     | x              | x                   |                       | x              | x                   |

| **D3DUSAGE \_ DONOTCLIP**          | x                    | x                   |                |                     |                       |                |                     |
| **D3DUSAGE \_ DYNAMIC**            | x                    | x                   | x              | x                   | x                     |                | x                   |
| **D3DUSAGE \_ NICHT UNSICHER**          | x                    | x                   | x              | x                   | x                     | x              | x                   |
| **\_D3DUSAGE-NPATCHES**           | x                    | x                   |                |                     |                       |                |                     |

-   [**CreateTexture**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-createtexture)
-   [**CreateVertexBuffer**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-createvertexbuffer)
-   [**CreateVolumeTexture**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-createvolumetexture)

Die Texturfunktionen [D3DXCreatexxx](dx9-graphics-reference-d3dx-functions-texture.md) verwenden auch einige dieser konstanten Werte für die Ressourcenerstellung.

Weitere Informationen zu Pooltypen und deren Einschränkungen bei bestimmten Verwendungen finden Sie unter [**D3DPOOL**](./d3dpool.md).
