# <a name="other-d3dx-constants"></a>Andere D3DX-Konstanten

Andere D3DX-Konstanten umfassen Folgendes:

## <a name="16-bit-floating-point-numbers"></a>16-Bit-Gleit Komma Zahlen



|                         |                                         |
|-------------------------|-----------------------------------------|
| \#definieren                | BESCHREIBUNG                             |
| D3DX \_ 16f- \_ dig          | Anzahl der Dezimalstellen der Genauigkeit   |
| D3DX \_ 16f \_ Epsilon      | Der kleinste Wert: 1,0 + Epsilon! = 1,0 |
| D3DX \_ 16f \_ - \_ dig    | Anzahl der Bits der Mantisse              |
| D3DX \_ 16f \_ Max.          | Maximalwert                           |
| D3DX \_ 16f \_ Max. \_ 10 \_ Exp | Der höchstmögliche dezimale Exponent                |
| D3DX \_ 16f \_ Max. \_ Exp     | Der höchstmögliche binäre Exponent                 |
| D3DX \_ 16f \_ Min.          | Der positive Mindestwert                  |
| D3DX \_ 16f \_ Min. \_ 10 \_ Exp | Der dezimale Mindestwert                |
| D3DX \_ 16f \_ Min. \_ Exp     | Der binäre Mindestwert                 |
| D3DX \_ 16f \_ Radix        | Die Exponentenbasis                          |
| D3DX \_ 16f- \_ Runden       | Runden der Addition: nah                 |
| D3DX \_ 1bypi             | 1/pi                                    |
| D3DX \_ Pi                | pi                                      |
| D3DX \_ Standardwert \_ float    | Maximaler float-Wert                     |



 

Diese \# Definitionen werden in d3dx9. h und d3dx9math. h deklariert.

## <a name="prt-constants"></a>PRT-Konstanten



|                  |                                            |
|------------------|--------------------------------------------|
| \#definieren         | BESCHREIBUNG                                |
| D3DXSH \_ minorder | Niedrigste zulässige Reihenfolge der Simulation.  |
| D3DXSH \_ maxorder | Höchste zulässige Reihenfolge der Simulation. |



 

Diese \# Definitionen werden in d3dx9math. h deklariert. Weitere Informationen zu PRT finden Sie unter [Voraus berechnete Strahlungs Übertragung (Direct3D 9)](precomputed-radiance-transfer.md).

## <a name="texture-constants"></a>Textur Konstanten



|                        |                                                                    |
|------------------------|--------------------------------------------------------------------|
| \#definieren               | BESCHREIBUNG                                                        |
| \_Aus \_ Datei D3DFMT     | Nehmen Sie das Format genau aus einer Datei.                               |
| D3DX \_ Standard          | Ein Standardwert.                                                   |
| D3DX \_ Standard \_ NONPOW2 | Runden Sie keine Zahlen, wie z. b. Breite oder Höhe, auf eine Potenz von zwei aus. |
| \_Aus \_ Datei D3DX       | Nehmen Sie die Textur Dimensionen genau aus einer Datei.                   |



 

Diese \# Definitionen werden in d3dx9. h deklariert.

## <a name="other-d3dx-constants"></a>Andere D3DX-Konstanten

Die UNUSED16-und D3DX- \_ Versions Konstanten, die in D3dx9mesh. h und D3dx9core. h definiert sind, werden intern verwendet. Verwenden Sie diese Konstanten nicht.
