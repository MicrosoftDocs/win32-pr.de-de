# <a name="texture-wrapping-state-direct3d-9"></a>Textur Wrapping Zustand (Direct3D 9)

Die D3DRS \_ WRAP0 through D3DRS \_ WRAP7-Rendering-Zustände aktivieren und deaktivieren u-und v-Wrapping für verschiedene Texturen in der Multitextur Cascade des Geräts. Sie können diese vererbungszustände auf eine Kombination der \_ Flags D3DWRAPCOORD 0, D3DWRAPCOORD \_ 1, D3DWRAPCOORD \_ 2 und D3DWRAPCOORD 3 festlegen, \_ um das umspringen in die ersten, zweiten, dritten und vierten Richtungen der Textur zu ermöglichen. Verwenden Sie den Wert 0, um das umwickeln vollständig zu deaktivieren. Das Textur umwickeln ist in allen Richtungen für alle Textur Stufen standardmäßig deaktiviert.

Eine konzeptionelle Übersicht finden Sie unter [Textur Wrapping (Direct3D 9)](texture-wrapping.md).
