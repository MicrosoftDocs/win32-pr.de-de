# <a name="d3dpresent_back_buffers"></a>D3DPRESENT- \_ Back- \_ Puffer

Gibt die Anzahl der Back Puffer an, die vom Gerät verwendet werden können.



|                                    |       |                                                           |
|------------------------------------|-------|-----------------------------------------------------------|
| \#definieren                           | Wert | BESCHREIBUNG                                               |
| D3DPRESENT \_ \_ backpuffer (max.) \_     | 3     | Maximale Anzahl der in Direct3D 9 unterstützten Back Puffer.   |
| D3DPRESENT \_ \_ backpuffern \_ Max. \_ | 30    | Maximale Anzahl der in Direct3D 9Ex unterstützten Back Puffer. |
