# <a name="d3dpresent_back_buffers"></a>D3DPRESENT \_ BACK \_ BUFFERS

Gibt die Anzahl der Backpuffer an, die vom Gerät verwendet werden können.



| \#Definieren                           | Wert | BESCHREIBUNG                                               |
|------------------------------------|-------|-----------------------------------------------------------|
| D3DPRESENT \_ BACK \_ BUFFERS \_ MAX     | 3     | Maximale Anzahl von Backpuffern, die in Direct3D 9 unterstützt werden.   |
| D3DPRESENT \_ BACK \_ BUFFERS \_ \_ MAX. EX | 30    | Maximale Anzahl von Backpuffern, die in Direct3D 9Ex unterstützt werden. |

|                          |            |
|--------------------------|------------|
| Header                   | d3d9.h     |
| Mindestbetriebssystem | Windows 98 |
