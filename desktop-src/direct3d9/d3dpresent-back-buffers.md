# <a name="d3dpresent_back_buffers"></a>\_D3DPRESENT-ZURÜCK-PUFFER \_

Gibt die Anzahl der Zurückpuffer an, die vom Gerät verwendet werden können.



| \#Definieren                           | Wert | BESCHREIBUNG                                               |
|------------------------------------|-------|-----------------------------------------------------------|
| D3DPRESENT \_ BACK \_ BUFFERS \_ MAX     | 3     | Maximale Anzahl der in Direct3D 9 unterstützten Backpuffer.   |
| D3DPRESENT \_ BACK \_ BUFFERS \_ MAX \_ EX | 30    | Maximale Anzahl der in Direct3D 9Ex unterstützten Hintergrundpuffer. |



 

## <a name="constant-information"></a>Konstante Informationen



|  Anforderung                        | Wert           |
|--------------------------|------------|
| Header                   | d3d9.h     |
| Mindestbetriebssystem | Windows 98 |
