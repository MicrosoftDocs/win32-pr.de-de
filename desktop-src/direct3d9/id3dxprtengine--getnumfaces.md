# <a name="id3dxprtenginegetnumfaces-method"></a>ID3DXPRTEngine:: getnumfaces-Methode

Ruft die Anzahl der Gesichter im Mesh ab, einschließlich aller neuen Flächen, die als Ergebnis der adaptiven räumlichen Stichprobenentnahme hinzugefügt wurden.

## <a name="syntax"></a>Syntax


```C++
UINT GetNumFaces();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **uint**](../winprog/windows-data-types.md)**

Anzahl der Gesichter im Mesh, einschließlich aller neuen Flächen, die als Ergebnis der adaptiven Stichprobenentnahme hinzugefügt wurden.
