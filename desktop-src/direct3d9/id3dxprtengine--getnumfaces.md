# <a name="id3dxprtenginegetnumfaces-method"></a>ID3DXPRTEngine::GetNumFaces-Methode

Ruft die Anzahl der Gesichter im Netz ab, einschließlich aller neuen Gesichter, die als Ergebnis der adaptiven räumlichen Stichprobenentnahme hinzugefügt wurden.

## <a name="syntax"></a>Syntax


```C++
UINT GetNumFaces();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **UINT**](../winprog/windows-data-types.md)**

Anzahl der Gesichter im Netz, einschließlich aller neuen Gesichter, die als Ergebnis der adaptiven Stichprobenentnahme hinzugefügt wurden.
