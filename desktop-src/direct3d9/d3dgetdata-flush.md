# <a name="d3dgetdata_flush"></a>D3DGETDATA \_ leeren

Dieses Makro erstellt den an [**GetData**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3dquery9-getdata) übergebenen Wert, um Abfrage Daten zu leeren.

``` syntax
#define D3DGETDATA_FLUSH (1 << 0)
```

## <a name="remarks"></a>Bemerkungen

Weitere Informationen zum Verwenden von Abfragen finden Sie unter [Abfragen (Direct3D 9)](queries.md).
