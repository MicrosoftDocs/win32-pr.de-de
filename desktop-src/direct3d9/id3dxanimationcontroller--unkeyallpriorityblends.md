# <a name="id3dxanimationcontrollerunkeyallpriorityblends-method"></a>ID3DXAnimationController::UnkeyAllPriorityBlends-Methode

Entfernt alle geplanten Prioritätsmischungsereignisse aus dem Animationscontroller.

## <a name="syntax"></a>Syntax


```C++
HRESULT UnkeyAllPriorityBlends();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **HRESULT**](https://msdn.microsoft.com/library/Bb401631(v=MSDN.10).aspx)**

Wenn die Methode erfolgreich ist, ist der Rückgabewert S \_ OK. Wenn bei der Methode ein Fehler auftritt, wird der folgende Wert zurückgegeben: D3DERR \_ INVALIDCALL.
