# <a name="id3dxbaseeffectsetboolarray-method"></a>ID3DXBaseEffect::SetBoolArray-Methode

Legt ein Array von booleschen Werten fest.

## <a name="syntax"></a>Syntax


```C++
HRESULT SetBoolArray(
  [in]       D3DXHANDLE hParameter,
  [in] const BOOL       *pB,
  [in]       UINT       Count
);
```
