# <a name="id3dxfiledataisreference-method"></a>ID3DXFileData::IsReference-Methode

Gibt an, ob dieses Dateidatenobjekt ein Verweisobjekt ist, das auf ein anderes untergeordnetes Datenobjekt verweist.

## <a name="syntax"></a>Syntax


```C++
BOOL IsReference();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **BOOL**](../winprog/windows-data-types.md)**

Gibt **TRUE** zurück, wenn das Dateidatenobjekt ein Verweisobjekt ist. gibt **andernfalls FALSE** zurück.
