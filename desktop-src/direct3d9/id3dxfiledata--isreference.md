# <a name="id3dxfiledataisreference-method"></a>ID3DXFileData:: IsReference-Methode

Gibt an, ob dieses Datei Datenobjekt ein Verweis Objekt ist, das auf ein anderes untergeordnetes Datenobjekt zeigt.

## <a name="syntax"></a>Syntax


```C++
BOOL IsReference();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **bool**](../winprog/windows-data-types.md)**

Gibt " **true** " zurück, wenn das Datei Datenobjekt ein Verweis Objekt ist. gibt andernfalls **false** zurück.
