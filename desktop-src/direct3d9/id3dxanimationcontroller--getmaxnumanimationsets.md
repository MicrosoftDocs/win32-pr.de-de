# <a name="id3dxanimationcontrollergetmaxnumanimationsets-method"></a>ID3DXAnimationController::GetMaxNumAnimationSets-Methode

Ruft die maximale Anzahl von Animationssätzen ab, die der Animationscontroller unterstützen kann.

## <a name="syntax"></a>Syntax


```C++
UINT GetMaxNumAnimationSets();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **UINT**](../winprog/windows-data-types.md)**

Die maximale Anzahl von Animationssätzen, die der Controller verwalten kann.
