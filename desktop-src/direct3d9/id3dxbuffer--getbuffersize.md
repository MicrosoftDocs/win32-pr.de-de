# <a name="id3dxbuffergetbuffersize-method"></a>ID3DXBuffer::GetBufferSize-Methode

Ruft die Gesamtgröße der Daten im Puffer ab.

## <a name="syntax"></a>Syntax


```C++
DWORD GetBufferSize();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **DWORD**](../winprog/windows-data-types.md)**

Gibt die Gesamtgröße der Daten im Puffer in Bytes zurück.
