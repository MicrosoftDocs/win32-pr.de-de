# <a name="d3dcolor"></a>D3DCOLOR

Definiert den grundlegenden Direct3D-Farbtyp.


```C++
typedef DWORD D3DCOLOR;
```



## <a name="remarks"></a>Bemerkungen

Ein 4-Byte-Wert, der in der Regel die Alpha-, rot-, grün-und Blau-Komponenten einer Farbe darstellt. Sie kann auch die Beleuchtung und die Helligkeit darstellen.

Sie können den **D3DCOLOR** -Typ mithilfe eines der folgenden Makros festlegen.

-   [**D3DCOLOR \_ ARGB**](d3dcolor-argb.md)
-   [**D3DCOLOR \_ ayuv**](d3dcolor-ayuv.md)
-   [**D3DCOLOR \_ ColorValue**](d3dcolor-colorvalue.md)
-   [**D3DCOLOR \_ RGBA**](d3dcolor-rgba.md)
-   [**D3DCOLOR \_ xrgb**](d3dcolor-xrgb.md)
-   [**D3DCOLOR \_ XYUV**](d3dcolor-xyuv.md)
