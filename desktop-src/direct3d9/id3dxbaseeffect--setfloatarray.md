# <a name="id3dxbaseeffectsetfloatarray-method"></a>ID3DXBaseEffect::SetFloatArray-Methode

Legt ein Array von Gleitkommawerten fest.

## <a name="syntax"></a>Syntax


```C++
HRESULT SetFloatArray(
  [in]       D3DXHANDLE hParameter,
  [in] const FLOAT      *pf,
  [in]       UINT       Count
);
```
