# <a name="id3dxcompressedanimationsetgetplaybacktype-method"></a>ID3DXCompressedAnimationSet:: getplaybacktype-Methode

## <a name="syntax"></a>Syntax


```C++
D3DXPLAYBACK_TYPE GetPlaybackType();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Type: **[ **D3DXPLAYBACK- \_ Typ**](./d3dxplayback-type.md)**

Der Typ der Wiedergabe Schleife für Animations Sätze. Siehe [**D3DXPLAYBACK \_ Type**](./d3dxplayback-type.md).
