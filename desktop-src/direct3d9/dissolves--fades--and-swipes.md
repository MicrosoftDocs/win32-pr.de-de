# <a name="dissolves-fades-and-swipes-direct3d-9"></a><span data-ttu-id="9565b-103">Wird aufgelöst, ausgeblendet und durch Schwenken (Direct3D 9)</span><span class="sxs-lookup"><span data-stu-id="9565b-103">Dissolves, Fades, and Swipes (Direct3D 9)</span></span>

<span data-ttu-id="9565b-104">Anwendungen nutzen zunehmend besondere Effekte, die häufig in Filmen und Videos verwendet werden, wie z. b. das Auflösen, wischen und durchlaufen von Anwendungen.</span><span class="sxs-lookup"><span data-stu-id="9565b-104">Increasingly, applications employ special effects that are commonly used in movies and video, such as dissolves, swipes, and fades.</span></span>

<span data-ttu-id="9565b-105">In einer Auflösung wird ein Bild nach und nach durch einen anderen in einer glatten Sequenz von Frames ersetzt.</span><span class="sxs-lookup"><span data-stu-id="9565b-105">In a dissolve, one image is gradually replaced by another in a smooth sequence of frames.</span></span> <span data-ttu-id="9565b-106">Obwohl Direct3D Methoden zur Verwendung mehrerer Textur Mischungs Funktionen bereitstellt, um denselben Effekt zu erzielen, können Anwendungen, die den Schablonen Puffer für die Auflösung verwenden, Textur Mischungs Funktionen für andere Effekte verwenden, während Sie eine Auflösung durchführen.</span><span class="sxs-lookup"><span data-stu-id="9565b-106">Although Direct3D provides methods of using multiple texture blending to achieve the same effect, applications that use the stencil buffer for dissolves can use texture-blending capabilities for other effects while they do a dissolve.</span></span>

<span data-ttu-id="9565b-107">Wenn Ihre Anwendung eine Auflösung durchführt, muss Sie zwei unterschiedliche Bilder darstellen.</span><span class="sxs-lookup"><span data-stu-id="9565b-107">When your application performs a dissolve, it must render two different images.</span></span> <span data-ttu-id="9565b-108">Er verwendet den Schablonen Puffer, um zu steuern, welche Pixel aus jedem Bild auf die Renderingzieloberfläche gezeichnet werden.</span><span class="sxs-lookup"><span data-stu-id="9565b-108">It uses the stencil buffer to control which pixels from each image are drawn to the rendering target surface.</span></span> <span data-ttu-id="9565b-109">Sie können eine Reihe von Schablone-Masken definieren und diese bei aufeinander folgenden Frames in den Schablonen Puffer kopieren.</span><span class="sxs-lookup"><span data-stu-id="9565b-109">You can define a series of stencil masks and copy them into the stencil buffer on successive frames.</span></span> <span data-ttu-id="9565b-110">Alternativ können Sie eine Basis Schablonen Maske für den ersten Frame definieren und inkrementell ändern.</span><span class="sxs-lookup"><span data-stu-id="9565b-110">Alternately, you can define a base stencil mask for the first frame and alter it incrementally.</span></span>

<span data-ttu-id="9565b-111">Am Anfang der Auflösung legt ihre Anwendung die Schablone-und Schablonen Maske fest, sodass die meisten Pixel aus dem Startimage den Schablonen Test bestehen.</span><span class="sxs-lookup"><span data-stu-id="9565b-111">At the beginning of the dissolve, your application sets the stencil function and stencil mask so that most of the pixels from the starting image pass the stencil test.</span></span> <span data-ttu-id="9565b-112">Die meisten Pixel des endbilds sollten den Schablonen Test nicht bestanden haben.</span><span class="sxs-lookup"><span data-stu-id="9565b-112">Most of the pixels from the ending image should fail the stencil test.</span></span> <span data-ttu-id="9565b-113">Bei aufeinander folgenden Frames wird die Schablonen Maske so aktualisiert, dass weniger und weniger Pixel im Start Abbild den Test bestehen.</span><span class="sxs-lookup"><span data-stu-id="9565b-113">On successive frames, the stencil mask is updated so that fewer and fewer of the pixels in the starting image pass the test.</span></span> <span data-ttu-id="9565b-114">Während der Fortschritt des Frames schlägt weniger und weniger der Pixel im Endbild bei dem Test fehl.</span><span class="sxs-lookup"><span data-stu-id="9565b-114">As the frames progress, fewer and fewer of the pixels in the ending image fail the test.</span></span> <span data-ttu-id="9565b-115">Auf diese Weise kann die Anwendung mithilfe beliebiger Auflösungs Muster aufgelöst werden.</span><span class="sxs-lookup"><span data-stu-id="9565b-115">In this manner, your application can perform a dissolve using any arbitrary dissolve pattern.</span></span>

<span data-ttu-id="9565b-116">Das Ausblenden oder Auschecken ist ein Sonderfall der Auflösung.</span><span class="sxs-lookup"><span data-stu-id="9565b-116">Fading in or fading out is a special case of dissolving.</span></span> <span data-ttu-id="9565b-117">Beim Ausblenden wird der Schablone-Puffer verwendet, um von einem schwarzen oder weißen Bild zu einem Rendering einer 3D-Szene zu lösen.</span><span class="sxs-lookup"><span data-stu-id="9565b-117">When fading in, the stencil buffer is used to dissolve from a black or white image to a rendering of a 3D scene.</span></span> <span data-ttu-id="9565b-118">Das Ausblenden ist das Gegenteil, Ihre Anwendung beginnt mit einem Rendering einer 3D-Szene und wird in schwarz oder weiß aufgelöst.</span><span class="sxs-lookup"><span data-stu-id="9565b-118">Fading out is the opposite, your application starts with a rendering of a 3D scene and dissolves to black or white.</span></span> <span data-ttu-id="9565b-119">Das ausblenden kann mithilfe eines beliebigen Musters erfolgen, das Sie verwenden möchten.</span><span class="sxs-lookup"><span data-stu-id="9565b-119">The fade can be done using any arbitrary pattern you want to employ.</span></span>

<span data-ttu-id="9565b-120">Direct3D-Anwendungen verwenden eine ähnliche Methode für das Schwenken.</span><span class="sxs-lookup"><span data-stu-id="9565b-120">Direct3D applications use a similar technique for swipes.</span></span> <span data-ttu-id="9565b-121">Wenn eine Anwendung z. b. einen Schwenk von links nach rechts durchführt, wird das Ende Bild nach dem Start Bild von links nach rechts angezeigt.</span><span class="sxs-lookup"><span data-stu-id="9565b-121">For example, when an application performs a left-to-right swipe, the ending image appears to slide gradually on top of the starting image from left to right.</span></span> <span data-ttu-id="9565b-122">Wie bei einer Auflösung müssen Sie eine Reihe von Schablone-Masken definieren, die bei aufeinander folgenden Frames in den Schablonen Puffer geladen werden, oder die Anfangs Schablone-Maske nacheinander ändern.</span><span class="sxs-lookup"><span data-stu-id="9565b-122">As in a dissolve, you must define a series of stencil masks that are loaded into the stencil buffer on successive frames, or successively modify the starting stencil mask.</span></span> <span data-ttu-id="9565b-123">Die Schablonen Masken werden verwendet, um das Schreiben von Pixeln aus dem Startbild zu deaktivieren und das Schreiben von Pixeln aus dem Endbild zu aktivieren.</span><span class="sxs-lookup"><span data-stu-id="9565b-123">The stencil masks are used to disable the writing of pixels from the starting image and to enable the writing of pixels from the ending image.</span></span>

<span data-ttu-id="9565b-124">Ein Schwenk ist etwas komplexer als eine Auflösung, in der die Anwendung Pixel aus dem Endbild in umgekehrter Reihenfolge des Schwenken lesen muss.</span><span class="sxs-lookup"><span data-stu-id="9565b-124">A swipe is somewhat more complex than a dissolve in that your application must read pixels from the ending image in the reverse order of the swipe.</span></span> <span data-ttu-id="9565b-125">Das heißt, wenn der schwenkvorgang von links nach rechts verschoben wird, muss die Anwendung Pixel aus dem Endbild von rechts nach links lesen.</span><span class="sxs-lookup"><span data-stu-id="9565b-125">That is, if the swipe is moving from left to right, your application must read pixels from the ending image from right to left.</span></span>
