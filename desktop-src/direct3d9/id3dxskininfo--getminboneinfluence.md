# <a name="id3dxskininfogetminboneinfluence-method"></a>ID3DXSkinInfo:: getminboneingefluence-Methode

Ruft den minimalen Knochen Einfluss ab. Werte, die kleiner als dieser Wert sind, werden ignoriert.

## <a name="syntax"></a>Syntax


```C++
FLOAT GetMinBoneInfluence();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **float**](../winprog/windows-data-types.md)**

Gibt den minimalen Wert für den Knochen Einfluss zurück.
