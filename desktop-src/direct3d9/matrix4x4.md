# <a name="matrix4x4"></a>Matrix4x4

Definiert eine 4 x 4-Matrix. Diese wird als Frame-Transformationsmatrix verwendet.

``` syntax
template Matrix4x4
{
    < F6F23F45-7686-11cf-8F52-0040333594A3 >
    array float matrix[16];
} 
```

Hierbei gilt:

-   Array-float \[ -Matrix 16 \] -Array von 16 Gleit Komma Zahlen.
