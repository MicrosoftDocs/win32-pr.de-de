# <a name="animationset"></a>AnimationSet

Enthält mindestens ein [**Animationsobjekt.**](animation.md) Jede Animation innerhalb eines Animationssatzes hat zu einem bestimmten Zeitpunkt die gleiche Zeit. Wenn Sie die Zeit des Animationssatzes erhöhen, verlängert sich die Zeit für alle darin enthaltenen Animationen.

``` syntax
template AnimationSet
{
    < 3D82AB50-62DA-11cf-AB39-0020AF71E433 >
    [ Animation < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 > ]
} 
```

Hierbei gilt:

-   \[ Animation < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 > \] – Optionale Animationsvorlage.
