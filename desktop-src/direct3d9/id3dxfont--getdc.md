# <a name="id3dxfontgetdc-method"></a>ID3DXFont::GetDC-Methode

Gibt ein Handle für einen Anzeigegerätekontext (DC) zurück, für den die Schriftart festgelegt ist.

## <a name="syntax"></a>Syntax


```C++
HDC GetDC();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **HDC**](../winprog/windows-data-types.md)**

Handle für einen Anzeigedomänencontroller.
