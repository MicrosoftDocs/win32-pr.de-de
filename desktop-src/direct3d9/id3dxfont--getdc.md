# <a name="id3dxfontgetdc-method"></a>ID3DXFont:: GetDC-Methode

Gibt ein Handle für einen Anzeigegeräte Kontext (DC) zurück, der über den Schriftart Satz verfügt.

## <a name="syntax"></a>Syntax


```C++
HDC GetDC();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **hdc**](../winprog/windows-data-types.md)**

Handle für einen Anzeige-DC.
