# <a name="boolean2d"></a>Boolean2d

Definiert einen Satz von zwei booleschen Werten, die in der [**MeshFaceWraps-Vorlage**](meshfacewraps.md) verwendet werden, um die Texturtopologie eines einzelnen Gesichts zu definieren.

``` syntax
template Boolean2d
{
    < 4885AE63-78E8-11cf-8F52-0040333594A3 >
    Boolean u;
    Boolean v;
} 
```

Hierbei gilt:

-   u : Boolescher Wert. Siehe [**Boolescher Wert.**](boolean.md)
-   v : Boolescher Wert. Siehe [**Boolescher Wert.**](boolean.md)
