# <a name="d3dcursorcaps"></a>D3DCURSORCAPS

Treibercursorfunktionsflags.



| \#Definieren              | Wert       | Beschreibung                                                                                     |
|-----------------------|-------------|-------------------------------------------------------------------------------------------------|
| D3DCURSORCAPS-FARBE \_  | 0x00000001L | Der Treiber unterstützt den Hardwarefarbcursor mindestens im Modus mit hoher Auflösung (Höhe >= 400). |
| D3DCURSORCAPS \_ LOWRES | 0x00000002L | Der Treiber unterstützt Hardwarefarbcursor in Modi mit niedriger Auflösung (Höhe < 400).            |



 

## <a name="constant-information"></a>Konstanteninformationen



| Anforderung                         | Wert           |
|--------------------------|------------|
| Header                   | d3d9caps.h |
| Mindestbetriebssystem | Windows 98 |
