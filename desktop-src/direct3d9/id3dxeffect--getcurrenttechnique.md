# <a name="id3dxeffectgetcurrenttechnique-method"></a>ID3DXEffect:: getcurrenttechnique-Methode

Ruft die aktuelle Technik ab.

## <a name="syntax"></a>Syntax


```C++
D3DXHANDLE GetCurrentTechnique();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[D3DXHANDLE](dx9-graphics-reference-effects-constants.md)**

Ein eindeutiger Bezeichner für die aktuelle Technik. Weitere Informationen finden Sie unter [Handles (Direct3D 9)](handles.md).
