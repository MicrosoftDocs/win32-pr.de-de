# <a name="d3dissue_begin"></a>D3DISSUE \_ Begin

Dieses Makro erstellt einen Wert, der von [**Issue**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3dquery9-issue) verwendet wird, um einen Abfrage Anfang auszugeben.

``` syntax
#define D3DISSUE_BEGIN (1 << 1)
```

## <a name="remarks"></a>Bemerkungen

D3DISSUE \_ Begin ist für den folgenden Abfragetyp gültig.

-   D3DQUERYTYPE- \_ oksion
