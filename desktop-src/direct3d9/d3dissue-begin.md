# <a name="d3dissue_begin"></a>D3DISSUE \_ BEGIN

Dieses Makro erstellt einen Wert, der von [**Issue verwendet wird,**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3dquery9-issue) um einen Abfragebeginn aus zu geben.

``` syntax
#define D3DISSUE_BEGIN (1 << 1)
```

## <a name="remarks"></a>Hinweise

D3DISSUE \_ BEGIN ist für den folgenden Abfragetyp gültig.

-   D3DQUERYTYPE \_ OCCLUSION
