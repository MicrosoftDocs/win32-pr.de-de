# <a name="meshfacewraps"></a>Meshfacewrapper

Wird verwendet, um die Textur Topologie der einzelnen Gesichter in einem Umbruch zu definieren. Der Wert des nfacewrapvalues-Members sollte gleich der Anzahl der Gesichter in einem Mesh sein.

``` syntax
template MeshFaceWraps
{
    < ED1EC5C0-C0A8-11D0-941C-0080C80CFA7B >
    DWORD nFaceWrapValues;
    array Boolean2d faceWrapValues[nFaceWrapValues];
} 
```

Diese Vorlage wird nicht mehr verwendet.
