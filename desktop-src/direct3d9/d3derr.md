## <a name="remarks"></a>Bemerkungen

Fehler werden durch negative Werte dargestellt und können nicht kombiniert werden.

Andere Rückgabecodes sind in [S \_ vorhanden](device-state-return-codes.md).

Der Einrichtungs Code \_ FACD3D wird verwendet, um Fehlercodes zu generieren, wie in den folgenden Makros.


```
// Direct3D Errors
#define D3DERR_UNSUPPORTEDFACTORVALUE  MAKE_D3DHRESULT(2079)
#define D3DERR_WASSTILLDRAWING         MAKE_D3DHRESULT(540)
    
#define _FACD3D                  0x876   // Direct3D facility code
#define MAKE_D3DHRESULT( code )  MAKE_HRESULT( 1, _FACD3D, code )
#define MAKE_D3DSTATUS( code )   MAKE_HRESULT( 0, _FACD3D, code )
```
