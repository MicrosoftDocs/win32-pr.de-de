# <a name="direct3d-textures-direct3d-9"></a>Direct3D-Texturen (Direct3D 9)

Texturen sind ein leistungsstarkes Tool, um mit dem Computer realistische 3D-Bilder zu erzeugen. Direct3D unterstützt einen umfangreichen Texturfunktionssatz und ermöglicht Entwicklern dadurch den einfachen Zugriff auf erweiterte Texturtechniken.

In diesem Abschnitt erfahren Sie, wie Sie Texturen verwenden.

-   [Grundlegende Texturkonzepte (Direct3D 9)](basic-texturing-concepts.md)
-   [Texturkoordinaten (Direct3D 9)](texture-coordinates.md)
-   [Texturfilterung (Direct3D 9)](texture-filtering.md)
-   [Texturressourcen (Direct3D 9)](texture-resources.md)
-   [Texturumbruch (Direct3D 9)](texture-wrapping.md)
-   [Texturmischung (Direct3D 9)](texture-blending.md)

In diesen Themen werden weitere Details zu zusätzlichen Texturfunktionen behandelt.

-   [Automatische Generierung von Mipmaps (Direct3D 9)](automatic-generation-of-mipmaps.md)
-   [Automatische Texturverwaltung (Direct3D 9)](automatic-texture-management.md)
-   [Komprimierte Texturressourcen (Direct3D 9)](compressed-texture-resources.md)
-   [Hardwareüberlegungen zur Texturierung (Direct3D 9)](hardware-considerations-for-texturing.md)
-   [Volumetexturressourcen (Direct3D 9)](volume-texture-resources.md)

Um die Leistung zu verbessern, sollten Sie dynamische Texturen verwenden. Eine dynamische Textur kann für jeden Frame gesperrt, in diesen geschrieben und entsperrt werden. Weitere Informationen finden Sie [unter Verwenden dynamischer Texturen.](performance-optimizations.md)
