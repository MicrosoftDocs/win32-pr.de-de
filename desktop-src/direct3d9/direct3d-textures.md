# <a name="direct3d-textures-direct3d-9"></a>Direct3D-Texturen (Direct3D 9)

Texturen sind ein leistungsstarkes Tool, um mit dem Computer realistische 3D-Bilder zu erzeugen. Direct3D unterstützt einen umfangreichen Texturfunktionssatz und ermöglicht Entwicklern dadurch den einfachen Zugriff auf erweiterte Texturtechniken.

In diesem Abschnitt erhalten Sie einen Einstieg in die Verwendung von Texturen.

-   [Grundlegende texturierungskonzepte (Direct3D 9)](basic-texturing-concepts.md)
-   [Texturkoordinaten (Direct3D 9)](texture-coordinates.md)
-   [Textur Filterung (Direct3D 9)](texture-filtering.md)
-   [Textur Ressourcen (Direct3D 9)](texture-resources.md)
-   [Textur Wrapping (Direct3D 9)](texture-wrapping.md)
-   [Textur Mischung (Direct3D 9)](texture-blending.md)

Diese Themen werden ausführlicher über zusätzliche texturierungsfunktionen behandelt.

-   [Automatische Generierung von Mipmaps (Direct3D 9)](automatic-generation-of-mipmaps.md)
-   [Automatische Textur Verwaltung (Direct3D 9)](automatic-texture-management.md)
-   [Komprimierte Textur Ressourcen (Direct3D 9)](compressed-texture-resources.md)
-   [Überlegungen zur Hardware für die Texturierung (Direct3D 9)](hardware-considerations-for-texturing.md)
-   [Volumetextur-Ressourcen (Direct3D 9)](volume-texture-resources.md)

Um die Leistung zu verbessern, sollten Sie dynamische Texturen verwenden. Eine dynamische Textur kann gesperrt, geschrieben und in jedem Frame entsperrt werden. Siehe [Verwenden dynamischer Texturen](performance-optimizations.md).
