# <a name="id3dxanimationcontrollergetpriorityblend-method"></a>ID3DXAnimationController:: getpriorityblend-Methode

Ruft die aktuelle Prioritäts Mischungs Gewichtung ab, die vom Animations Controller verwendet wird.

## <a name="syntax"></a>Syntax


```C++
FLOAT GetPriorityBlend();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **float**](../winprog/windows-data-types.md)**

Gibt die aktuelle Prioritäts Mischungs Gewichtung zurück.

## <a name="remarks"></a>Bemerkungen

Mit der Gewichtungs Gewichtungs Gewichtung werden High-und Low-Priority-Spuren kombiniert.
