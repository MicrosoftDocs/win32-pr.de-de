# <a name="id3dxlinegetgllines-method"></a>ID3DXLine::GetGLLines-Methode

Ruft den Strichzeichnungsmodus im OpenGL-Stil ab.

## <a name="syntax"></a>Syntax


```C++
BOOL GetGLLines();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **BOOL**](../winprog/windows-data-types.md)**

Gibt **TRUE** zurück, wenn OpenGL-Stillinien aktiviert sind, und **FALSE,** wenn Linien im Direct3D-Stil aktiviert sind.
