# <a name="id3dxlinegetgllines-method"></a>ID3DXLine:: getgllines-Methode

Ruft den Modus im OpenGL-Stil ab.

## <a name="syntax"></a>Syntax


```C++
BOOL GetGLLines();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **bool**](../winprog/windows-data-types.md)**

Gibt **true** zurück, wenn die Zeilen im OpenGL-Stil aktiviert sind, und **false** , wenn Direct3D-Stile aktiviert sind.
