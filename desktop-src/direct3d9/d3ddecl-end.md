# <a name="d3ddecl_end-macro"></a>D3DDECL- \_ End-Makro

Initialisieren Sie das letzte Vertex-Element in einem Vertex-Deklarations Array.

## <a name="syntax"></a>Syntax


```C++
void D3DDECL_END(void);
```



## <a name="parameters"></a>Parameter

Dieses Makro weist keine Parameter auf.

## <a name="return-value"></a>Rückgabewert

Dieses Makro gibt keinen Wert zurück.

## <a name="remarks"></a>Bemerkungen

Wird verwendet, um das letzte Vertex-Element in einem Vertex-Deklarations Array zu initialisieren.
