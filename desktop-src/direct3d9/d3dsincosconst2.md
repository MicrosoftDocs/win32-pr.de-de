# <a name="d3dsincosconst2"></a>D3DSINCOSCONST2

Erstellt die zweite Konstante (in Register src3) für die [SinCos-PS-Pixel-](../direct3dhlsl/sincos---ps.md) Shader-Anweisung.

``` syntax
#define D3DSINCOSCONST2 -0.020833334f, -0.12500000f, 1.0f, 0.50000000f
```

## <a name="return-value"></a>Rückgabewert

Gibt die zweite Konstante zurück, die von [SinCos-PS](../direct3dhlsl/sincos---ps.md)benötigt wird.
