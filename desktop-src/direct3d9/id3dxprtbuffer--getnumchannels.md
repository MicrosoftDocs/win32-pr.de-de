# <a name="id3dxprtbuffergetnumchannels-method"></a>ID3DXPRTBuffer::GetNumChannels-Methode

Ruft die Anzahl der Farbkanäle ab, die im Arbeitsspeicher zum Speichern von Stichproben verwendet werden.

## <a name="syntax"></a>Syntax


```C++
UINT GetNumChannels();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **UINT**](../winprog/windows-data-types.md)**

Gibt die Anzahl der Farbkanäle zurück, die im Arbeitsspeicher zum Speichern von Stichproben verwendet werden. Der Wert ist im Allgemeinen entweder 1 zur Darstellung von Leuchtdichtewerten oder 3 für RGB-Werte.
