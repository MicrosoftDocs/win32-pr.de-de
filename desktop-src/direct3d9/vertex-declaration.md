# <a name="vertex-declaration-direct3d-9"></a>Scheitelpunktdeklaration (Direct3D 9)

Eine Scheitelpunktdeklaration definiert das Scheitelpunktpufferlayout und programmiert die Mosaik-Engine. Ab Direct3D 9 werden Scheitelpunktstreams als Array von [**D3DVERTEXELEMENT9-Strukturen**](d3dvertexelement9.md) ausgedrückt (anstatt FVF-Codes (Flexible Vertex Format) zu verwenden). Jedes Arrayelement beschreibt jeden Scheitelpunkt-Datentyp. Die Konvertierung von FVF> Codes in das neue Format wird in den folgenden Themen behandelt:

-   [Zuordnen von FVF-Codes zu einer Direct3D 9-Deklaration (Direct3D 9)](mapping-fvf-codes-to-a-directx-9-declaration.md)
-   [FVF-Codes für feste Funktionen (Direct3D 9)](fixed-function-fvf-codes.md)
-   [Zuordnung zwischen einer Direct3D-Deklaration und FVF-Codes (Direct3D 9)](mapping-between-a-directx-9-declaration-and-fvf-codes.md)
-   [Zuordnung zwischen einer Direct3D 9-Deklaration und einer Direct3D 8-Deklaration (Direct3D 9)](mapping-between-a-directx-9-declaration-and-directx-8.md)
-   [Programmieren eines oder Streams (Direct3D 9)](programming-one-or-more-streams.md)
