# <a name="vertex-declaration-direct3d-9"></a>Vertex-Deklaration (Direct3D 9)

Eine Scheitelpunkt Deklaration definiert das vertexpufferlayout und die Programme der Mosaik-Engine. Ab Direct3D 9 werden Scheitelpunkt Datenströme als Array von [**D3DVERTEXELEMENT9**](d3dvertexelement9.md) -Strukturen ausgedrückt (anstatt flexible Scheitelpunkt Formate (FVF)) zu verwenden. Jedes Array Element beschreibt jeden Datentyp pro Scheitelpunkt. Das Umrechnen von f-> Codes in das neue Format wird in den folgenden Themen behandelt:

-   [Zuordnung von FVF-Codes zu einer Direct3D 9-Deklaration (Direct3D 9)](mapping-fvf-codes-to-a-directx-9-declaration.md)
-   [Code der Fixed-Funktion (Direct3D 9)](fixed-function-fvf-codes.md)
-   [Zuordnung zwischen einer Direct3D-Deklaration und einem f-Code (Direct3D 9)](mapping-between-a-directx-9-declaration-and-fvf-codes.md)
-   [Zuordnung zwischen einer Direct3D 9-Deklaration und einer Direct3D 8-Deklaration (Direct3D 9)](mapping-between-a-directx-9-declaration-and-directx-8.md)
-   [Programmieren von mindestens einem Stream (Direct3D 9)](programming-one-or-more-streams.md)
