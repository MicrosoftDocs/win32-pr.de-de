# <a name="animation-direct3d-9-graphics"></a>Animation (Direct3D 9-Grafiken)

Enthält Animationen, die auf einen vorherigen Frame verweisen. Sie sollte einen Verweis auf einen Frame und mindestens einen Satz [**von AnimationKey-Vorlagen**](animationkey.md) enthalten. Sie kann auch ein [**AnimationOptions-Datenobjekt**](animationoptions.md) enthalten.

``` syntax
template Animation
{
    < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 >
    [...]
} 
```

Hierbei gilt:

-   \[ ... \] – Hier kann eine beliebige X-Dateivorlage verwendet werden. Dadurch wird die Architektur erweiterbar.
