# <a name="animation-direct3d-9-graphics"></a>Animation (Direct3D 9-Grafiken)

Enthält Animationen, die auf einen vorherigen Frame verweisen. Sie sollte einen Verweis auf einen Frame und mindestens einen Satz von [**animationkey**](animationkey.md) -Vorlagen enthalten. Sie kann auch ein [**animationoptions**](animationoptions.md) -Datenobjekt enthalten.

``` syntax
template Animation
{
    < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 >
    [...]
} 
```

Hierbei gilt:

-   \[ ... \] -Eine beliebige x-Datei Vorlage kann hier verwendet werden. Dadurch wird die Architektur erweiterbar.
