# <a name="id3dxprtenginefreebouncedata-method"></a>ID3DXPRTEngine::FreeButnceData-Methode

Gibt Arbeitsspeicher frei, der für temporäre hüpfende Lichtsimulationsdaten verwendet wird.

## <a name="syntax"></a>Syntax


```C++
HRESULT FreeBounceData();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **HRESULT**](https://msdn.microsoft.com/library/Bb401631(v=MSDN.10).aspx)**

Der Rückgabewert ist S \_ OK.
