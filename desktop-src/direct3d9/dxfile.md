# <a name="dxfile-constants"></a>DXFILE-Konstanten

Die folgenden Flags werden verwendet, um anzugeben, welche Kanäle in einer Textur verwendet werden sollen. Veraltet.

## <a name="dxfileformat"></a>DXFILEFORMAT



| \#Definieren                 | Wert | Beschreibung      |
|--------------------------|-------|------------------|
| DXFILEFORMAT \_ BINARY     | 0     | Binärdatei.     |
| DXFILEFORMAT \_ TEXT       | 1     | Textdatei.       |
| DXFILEFORMAT \_ COMPRESSED | 2     | Komprimierte Datei. |



 

Diese \# Definitionen werden in Dxfile.h deklariert.

## <a name="dxfileloadoptions"></a>DXFILELOADOPTIONS



| \#Definieren                 | Wert | Beschreibung                  |
|--------------------------|-------|------------------------------|
| DXFILELOAD \_ FROMFILE     | 0x00L | Laden sie eine Datei aus einer Datei.     |
| DXFILELOAD \_ FROMRESOURCE | 0x01L | Laden sie eine Datei aus einer Ressource. |
| DXFILELOAD \_ FROMMEMORY   | 0x02L | Laden sie eine Datei aus dem Arbeitsspeicher.     |
| DXFILELOAD \_ FROMSTREAM   | 0x04L | Laden sie eine Datei aus einem Stream.   |
| DXFILELOAD \_ FROMURL      | 0x08L | Laden sie eine Datei aus einer URL.      |



 

Diese \# Definitionen werden in Dxfile.h deklariert.
