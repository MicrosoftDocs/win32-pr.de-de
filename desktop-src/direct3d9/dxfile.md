# <a name="dxfile-constants"></a>Dxfile-Konstanten

Die folgenden Flags werden verwendet, um anzugeben, auf welchen Kanälen in einer Textur gearbeitet werden soll. Veraltet.

## <a name="dxfileformat"></a>Dxfileformat



|                          |       |                  |
|--------------------------|-------|------------------|
| \#definieren                 | Wert | BESCHREIBUNG      |
| dxfileformat- \_ Binärdatei     | 0     | Binärdatei.     |
| dxfileformat- \_ Text       | 1     | Textdatei.       |
| dxfileformat \_ komprimiert | 2     | Komprimierte Datei. |



 

Diese \# Definitionen werden in "dxfile. h" deklariert.

## <a name="dxfileloadoptions"></a>Dxfileloadoption



|                          |       |                              |
|--------------------------|-------|------------------------------|
| \#definieren                 | Wert | BESCHREIBUNG                  |
| dxfileload \_ FromFile     | 0x00l | Lädt eine Datei aus einer Datei.     |
| dxfileload \_ FromResource | 0x01l | Lädt eine Datei aus einer Ressource. |
| dxfileload \_ frommemory   | 0x02l | Lädt eine Datei aus dem Arbeitsspeicher.     |
| dxfileload \_ FromStream   | 0x04l | Lädt eine Datei aus einem Stream.   |
| dxfileload \_ FromUrl      | 0x08l | Lädt eine Datei aus einer URL.      |



 

Diese \# Definitionen werden in "dxfile. h" deklariert.
