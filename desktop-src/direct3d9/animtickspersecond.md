# <a name="animtickspersecond"></a>Animtickspersecond

Beschreibt einen Animations Skalierungsfaktor.

``` syntax
template AnimTicksPerSecond
{
    < 9E415A43-7BA6-4a73-8743-B73D47E88476 >
    DWORD AnimTicksPerSecond;
} 
```

Hierbei gilt:

-   Animtickspersecond: ein Skalierungsfaktor zum Konvertieren von Animations Zeitstempeln in globale Zeit. Der Wert dividiert die Animations Zeit um 30, 60, 24 oder 25.
