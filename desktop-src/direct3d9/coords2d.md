# <a name="coords2d"></a>Coords2d

Definiert einen zweidimensionalen Vektor, der verwendet wird, um die Texturkoordinaten eines Netzes (u, v) zu definieren.

``` syntax
template Coords2d
{
    < F6F23F44-7686-11cf-8F52-0040333594A3 >
    float u;
    float v;
} 
```

-   u-u-Koordinaten Wert.
-   v-v-Koordinaten Wert.
