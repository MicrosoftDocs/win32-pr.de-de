# <a name="d3dadapter_default"></a>D3DADAPTER \_ Standard

Gibt den primären Anzeige Adapter an.



|                     |       |                                              |
|---------------------|-------|----------------------------------------------|
| \#definieren            | Wert | BESCHREIBUNG                                  |
| D3DADAPTER \_ Standard | 0     | Wird verwendet, um den primären Anzeige Adapter anzugeben. |
