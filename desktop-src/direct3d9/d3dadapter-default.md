# <a name="d3dadapter_default"></a>D3DADAPTER \_ DEFAULT

Gibt den primären Anzeigeadapter an.



| \#Definieren            | Wert | BESCHREIBUNG                                  |
|---------------------|-------|----------------------------------------------|
| D3DADAPTER \_ DEFAULT | 0     | Wird verwendet, um den primären Anzeigeadapter anzugeben. |
