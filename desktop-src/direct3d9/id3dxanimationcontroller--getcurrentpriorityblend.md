# <a name="id3dxanimationcontrollergetcurrentpriorityblend-method"></a>ID3DXAnimationController:: getcurrentpriorityblend-Methode

Gibt ein Ereignis Handle an ein Priority Blend-Ereignis zurück, das gerade ausgeführt wird.

## <a name="syntax"></a>Syntax


```C++
D3DXEVENTHANDLE GetCurrentPriorityBlend();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **D3DXEVENTHANDLE**](id3dxanimationcontroller.md)**

Ereignis Handle für das aktuell aktuell aktuell laufende Ereignis für die Prioritäts Mischung. **Null** wird zurückgegeben, wenn zurzeit kein Priority-Blend-Ereignis ausgeführt wird.
