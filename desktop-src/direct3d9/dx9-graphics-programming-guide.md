# <a name="programming-guide-for-direct3d-9"></a>Programmierhandbuch für Direct3D 9

Dieser Leitfaden enthält eine Beschreibung der grafikpipeline, die von Microsoft Direct3D implementiert wird. Es ist ein Leitfaden für Entwickler, die Direct3D-Grafikfunktionen in ihre Anwendungen implementieren. Der Leitfaden enthält Architekturbeschreibungen, Funktionsblockdiagramme und Beschreibungen der Bausteine in der Pipeline sowie Codeausschnitte und Beispielanwendungen. Die Informationen sind in die folgenden Abschnitte unterteilt:

-   [Erste Schritte:](getting-started.md) Dieser Abschnitt enthält sowohl eine Übersicht über die Pipeline als auch Tutorials, mit denen Sie eine einfache Grafikanwendung in wenigen Minuten ausführen können.
-   [Effekte:](effects.md) In diesem Abschnitt werden Effekt- und Effektdateien zum Erstellen von Anwendungen behandelt, die auf einer Vielzahl von Hardwareplattformen ausgeführt werden können.
-   [Erweiterte Themen:](advanced-topics.md) Dieser Abschnitt enthält Beispiele für verschiedene Arten von Sondereffekten, die Sie implementieren können. Themen wie Umgebungs- und Bumpzuordnung, Antialiasing, Vertexblending, Tweening, HDR-Beleuchtung (High Dynamic Range) und Precomputed Radiance Transfer (PRT) zeigen, wie Sie führende Sondereffekte auf Ihre Anwendung anwenden.
-   [Programmieren Tipps:](programming-tips.md) Dieser Abschnitt enthält Informationen zur effizienten Entwicklung von Direct3D-Grafikanwendungen.

Weitere Informationen zu bestimmten API-Methoden finden Sie auf den [Referenzseiten.](dx9-graphics-reference.md)
