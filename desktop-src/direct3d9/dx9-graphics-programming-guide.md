# <a name="programming-guide-for-direct3d-9"></a>Programmier Handbuch für Direct3D 9

Dieses Handbuch enthält eine Beschreibung der von Microsoft Direct3D implementierten Grafik Pipeline. Es ist ein Leitfaden für Entwickler, die Direct3D-Grafikfunktionen in Ihre Anwendungen implementieren. Das Handbuch enthält Architektur Beschreibungen, funktionale Blockdiagramme und Beschreibungen der Bausteine in der Pipeline sowie Code Ausschnitte und Beispielanwendungen. Die Informationen sind in folgende Abschnitte unterteilt:

-   Einführung [: dieser](getting-started.md) Abschnitt enthält eine Übersicht über die Pipeline und Lernprogramme, mit deren Hilfe Sie in wenigen Minuten eine einfache Grafikanwendung ausführen können.
-   [Effekte](effects.md) : in diesem Abschnitt werden Effekte und Wirkungs Dateien zum Entwickeln von Anwendungen behandelt, die auf einer Vielzahl von Hardwareplattformen ausgeführt werden können.
-   [Erweiterte Themen](advanced-topics.md) : Dieser Abschnitt enthält Beispiele für verschiedene Arten von speziellen Effekten, die Sie implementieren können. Themen wie z. b. Umgebungs-und Bump-Zuordnung, Antialiasing, Vertex-Blending, twiening, eine hohe dynamische Bereichs Beleuchtung (HDR) und preberechnungsübertragungs-Übertragung (PRT) veranschaulichen, wie Sie die wichtigsten besonderen Auswirkungen auf Ihre Anwendung anwenden können.
-   [Programmiertipps](programming-tips.md) : Dieser Abschnitt enthält Informationen, die Ihnen helfen, Direct3D-Grafikanwendungen effizient zu entwickeln.

Weitere Informationen zu bestimmten API-Methoden finden Sie auf den [Referenz](dx9-graphics-reference.md) Seiten.
