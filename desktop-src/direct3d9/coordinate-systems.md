# <a name="coordinate-systems-direct3d-9"></a><span data-ttu-id="d6751-103">Koordinatensysteme (Direct3D 9)</span><span class="sxs-lookup"><span data-stu-id="d6751-103">Coordinate Systems (Direct3D 9)</span></span>

<span data-ttu-id="d6751-104">In der Regel verwenden 3D-Grafikanwendungen zwei Arten von kartesischen Koordinatensystemen: Links-und rechtsbündig.</span><span class="sxs-lookup"><span data-stu-id="d6751-104">Typically 3D graphics applications use two types of Cartesian coordinate systems: left-handed and right-handed.</span></span> <span data-ttu-id="d6751-105">In beiden Koordinatensystemen zeigt die positive x-Achse nach rechts, und die positive y-Achse zeigt nach oben.</span><span class="sxs-lookup"><span data-stu-id="d6751-105">In both coordinate systems, the positive x-axis points to the right, and the positive y-axis points up.</span></span> <span data-ttu-id="d6751-106">Sie können merken, welche Richtung die positive z-Achse zeigt, indem Sie die Finger entweder in der linken oder rechten Ecke in der positiven x-Richtung zeigen und Sie in die positive y-Richtung hineinbewegen.</span><span class="sxs-lookup"><span data-stu-id="d6751-106">You can remember which direction the positive z-axis points by pointing the fingers of either your left or right hand in the positive x-direction and curling them into the positive y-direction.</span></span> <span data-ttu-id="d6751-107">Die Richtung, auf die sich der Ziehpunkt von Ihnen befindet, ist die Richtung, auf die die positive z-Achse für dieses Koordinatensystem zeigt.</span><span class="sxs-lookup"><span data-stu-id="d6751-107">The direction your thumb points, either toward or away from you, is the direction that the positive z-axis points for that coordinate system.</span></span> <span data-ttu-id="d6751-108">In der folgenden Abbildung werden diese beiden Koordinatensysteme veranschaulicht.</span><span class="sxs-lookup"><span data-stu-id="d6751-108">The following illustration shows these two coordinate systems.</span></span>

![Abbildung der links gerichteten und rechts gerichteten kartesischen Koordinatensysteme](images/leftrght.png)

<span data-ttu-id="d6751-110">Direct3D verwendet ein Links händiges Koordinatensystem.</span><span class="sxs-lookup"><span data-stu-id="d6751-110">Direct3D uses a left-handed coordinate system.</span></span> <span data-ttu-id="d6751-111">Wenn Sie eine Anwendung portieren, die auf einem rechts gerichteten Koordinatensystem basiert, müssen Sie zwei Änderungen an den Daten vornehmen, die an Direct3D übergeben werden.</span><span class="sxs-lookup"><span data-stu-id="d6751-111">If you are porting an application that is based on a right-handed coordinate system, you must make two changes to the data passed to Direct3D.</span></span>

-   <span data-ttu-id="d6751-112">Kippen Sie die Reihenfolge der Dreiecks Scheitel Punkte, sodass Sie vom System im Uhrzeigersinn nach vorne durchlaufen werden.</span><span class="sxs-lookup"><span data-stu-id="d6751-112">Flip the order of triangle vertices so that the system traverses them clockwise from the front.</span></span> <span data-ttu-id="d6751-113">Anders ausgedrückt: Wenn es sich bei den Scheitel Punkten um v0, v1, v2 handelt, übergeben Sie Sie an Direct3D als v0, v2, v1.</span><span class="sxs-lookup"><span data-stu-id="d6751-113">In other words, if the vertices are v0, v1, v2, pass them to Direct3D as v0, v2, v1.</span></span>
-   <span data-ttu-id="d6751-114">Verwenden Sie die Ansichts Matrix, um den Welt Raum um-1 in der z-Richtung zu skalieren.</span><span class="sxs-lookup"><span data-stu-id="d6751-114">Use the view matrix to scale world space by -1 in the z-direction.</span></span> <span data-ttu-id="d6751-115">Um dies zu erreichen, kippen Sie das Vorzeichen der \_ Member 31, \_ 32, \_ 33 und \_ 34 der [**D3DMATRIX**](d3dmatrix.md) -Struktur, die Sie für die Sicht Matrix verwenden.</span><span class="sxs-lookup"><span data-stu-id="d6751-115">To do this, flip the sign of the \_31, \_32, \_33, and \_34 member of the [**D3DMATRIX**](d3dmatrix.md) structure that you use for your view matrix.</span></span>

<span data-ttu-id="d6751-116">Verwenden Sie die [**D3DXMatrixPerspectiveRH**](d3dxmatrixperspectiverh.md) -Funktion und die [**D3DXMatrixOrthoRH**](d3dxmatrixorthorh.md) -Funktion, um die Projektions Transformation zu definieren.</span><span class="sxs-lookup"><span data-stu-id="d6751-116">To obtain what amounts to a right-handed world, use the [**D3DXMatrixPerspectiveRH**](d3dxmatrixperspectiverh.md) and [**D3DXMatrixOrthoRH**](d3dxmatrixorthorh.md) functions to define the projection transform.</span></span> <span data-ttu-id="d6751-117">Achten Sie jedoch darauf, dass Sie die entsprechende [**D3DXMatrixLookAtRH**](d3dxmatrixlookatrh.md) -Funktion verwenden, die Backface-cullinger-Reihenfolge umkehren und die Cubemaps entsprechend aufstellen.</span><span class="sxs-lookup"><span data-stu-id="d6751-117">However, be careful to use the corresponding [**D3DXMatrixLookAtRH**](d3dxmatrixlookatrh.md) function, reverse the backface-culling order, and lay out the cube maps accordingly.</span></span>

<span data-ttu-id="d6751-118">Obwohl Links ausgegebene und rechts gerichtete Koordinaten die gängigsten Systeme sind, gibt es eine Vielzahl anderer Koordinatensysteme, die in 3D-Software verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="d6751-118">Although left-handed and right-handed coordinates are the most common systems, there is a variety of other coordinate systems used in 3D software.</span></span> <span data-ttu-id="d6751-119">Beispielsweise ist es nicht ungewöhnlich, dass 3D-Modellierungs Anwendungen ein Koordinatensystem verwenden, in dem die y-Achse auf den Viewer hin-und Weg zeigt, und die z-Achse zeigt nach oben.</span><span class="sxs-lookup"><span data-stu-id="d6751-119">For example, it is not unusual for 3D modeling applications to use a coordinate system in which the y-axis points toward or away from the viewer, and the z-axis points up.</span></span>

<span data-ttu-id="d6751-120">Formal kann die Ausrichtung eines Satzes von Basis Vektoren (d. h. ein Koordinatensystem) durch das Berechnen der Determinanten der Matrix gefunden werden, die durch einen bestimmten Satz von Basis Vektoren definiert wird.</span><span class="sxs-lookup"><span data-stu-id="d6751-120">Formally, the orientation of a set of basis vectors (i.e. a coordinate system) can be found by the computing the determinant of the matrix defined by the particular set of basis vectors.</span></span> <span data-ttu-id="d6751-121">Wenn die Determinante positiv ist, wird die Basis als "positiv" (oder Rechtshändig) bezeichnet.</span><span class="sxs-lookup"><span data-stu-id="d6751-121">If the determinant is positive, the basis is said to be "positively" oriented (or right-handed).</span></span> <span data-ttu-id="d6751-122">Wenn die Determinante negativ ist, wird die Basis als "negativ" (oder linksbündig) bezeichnet.</span><span class="sxs-lookup"><span data-stu-id="d6751-122">If the determinant is negative, the basis is said to be "negatively" oriented (or left-handed).</span></span> <span data-ttu-id="d6751-123">Eine Erläuterung dazu, was eine Determinante ist, finden Sie unter beliebige lineare Algebra-Ressourcen.</span><span class="sxs-lookup"><span data-stu-id="d6751-123">For an explanation of what a determinant is, see any linear algebra resource.</span></span>

<span data-ttu-id="d6751-124">Informell können Sie mithilfe der "right/left-Rule"-Regel ermitteln, ob ein bestimmter Satz von Basis Vektoren entweder ein Rechtes oder Links händiges Koordinatensystem bildet.</span><span class="sxs-lookup"><span data-stu-id="d6751-124">Informally, you can use the "right/left hand rule" to determine if a given set of basis vectors form either a right or left handed coordinate system.</span></span>

<span data-ttu-id="d6751-125">Die wesentlichen Vorgänge, die für in einem 3D-Koordinatensystem definierte Objekte ausgeführt werden, sind Übersetzung, Drehung und Skalierung.</span><span class="sxs-lookup"><span data-stu-id="d6751-125">The essential operations performed on objects defined in a 3D coordinate system are translation, rotation, and scaling.</span></span> <span data-ttu-id="d6751-126">Sie können diese grundlegenden Transformationen kombinieren, um eine Transformationsmatrix zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="d6751-126">You can combine these basic transformations to create a transform matrix.</span></span> <span data-ttu-id="d6751-127">Weitere Informationen finden Sie unter [Transformationen (Direct3D 9)](transforms.md).</span><span class="sxs-lookup"><span data-stu-id="d6751-127">For details, see [Transforms (Direct3D 9)](transforms.md).</span></span>

<span data-ttu-id="d6751-128">Wenn Sie diese Vorgänge kombinieren, sind die Ergebnisse nicht kommutativ. die Reihenfolge, in der Sie Matrizen multiplizieren, ist wichtig.</span><span class="sxs-lookup"><span data-stu-id="d6751-128">When you combine these operations, the results are not commutative; the order in which you multiply matrices is important.</span></span>
