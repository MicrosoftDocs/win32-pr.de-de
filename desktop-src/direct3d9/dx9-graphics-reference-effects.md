# <a name="effect-reference-direct3d-9-graphics"></a>Effect Reference (Direct3D 9 Graphics)

Effekte sind eine Abstraktion auf höherer Ebene, die Scheitelpunkt- und Pixel-Shader nutzen, um Renderingeffekte zu kapseln. Effekte können mit der Sprache auf hoher Ebene oder mit der Shader-Assemblysprache geschrieben werden.

Hier sind die API-Themen.

-   [Effektschnittstellen](dx9-graphics-reference-effects-interfaces.md)
-   [Effect-Funktionen](dx9-graphics-reference-effects-functions.md)
-   [Effektstrukturen](dx9-graphics-reference-effects-structures.md)
-   [Effektkonst constants](dx9-graphics-reference-effects-constants.md)
-   [Effect-Format (Direct3D 9)](dx9-graphics-reference-effects-file-format.md)
-   [DirectX-Standardanmerkungen und Semantikreferenz](dx9-graphics-reference-effects-dxsas.md)

Als Alternative zur Verwendung der API zum Erstellen von Effekten können Sie auch ein Befehlszeilen-Kompilierungstool verwenden. Weitere Informationen [finden Sie unter Effect-Compiler Tool](../direct3dtools/fxc.md).
