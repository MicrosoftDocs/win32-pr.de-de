# <a name="effect-reference-direct3d-9-graphics"></a>Effekt Referenz (Direct3D 9-Grafiken)

Effekte sind eine Abstraktion auf höherer Ebene, die Scheitelpunkt-und Pixel-Shader zum Kapseln von renderingeffekten nutzen. Effekte können mit der Sprache auf hoher Ebene oder mit der Shader-Assemblysprache geschrieben werden.

Hier finden Sie die API-Themen.

-   [Effekt Schnittstellen](dx9-graphics-reference-effects-interfaces.md)
-   [Effekt Funktionen](dx9-graphics-reference-effects-functions.md)
-   [Effekt Strukturen](dx9-graphics-reference-effects-structures.md)
-   [Effekt Konstanten](dx9-graphics-reference-effects-constants.md)
-   [Effekt Format (Direct3D 9)](dx9-graphics-reference-effects-file-format.md)
-   [DirectX-Standard Anmerkungen und Semantik Referenz](dx9-graphics-reference-effects-dxsas.md)

Als Alternative zur Verwendung der-API zum Erstellen von Effekten können Sie auch ein Befehlszeilen-Kompilierungs Tool verwenden. Siehe [Effect-Compiler-Tool](../direct3dtools/fxc.md).
