# <a name="id3dxprtbuffergetnumcoeffs-method"></a>ID3DXPRTBuffer::GetNumCoeffs-Methode

Ruft die Anzahl der Skalar pro Farbkanal ab, die im Arbeitsspeicher zum Speichern von Beispielen verwendet werden.

## <a name="syntax"></a>Syntax


```C++
UINT GetNumCoeffs();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **UINT**](../winprog/windows-data-types.md)**

Gibt die Anzahl von Skalaren pro Farbkanal zurück, die im Arbeitsspeicher zum Speichern von Stichproben verwendet werden.
