# <a name="text-encoding"></a>Textcodierung

Das DirectX.x-Dateiformat ist nicht anwendungsspezifisch. Es werden Vorlagen verwendet, die nicht davon abhängen, wie die Datei verwendet wird. Dadurch kann das X-Dateiformat von jeder Clientanwendung verwendet werden.

In den folgenden Abschnitten werden der Inhalt und die Syntax des Dateiformats beschrieben, das die Erweiterung .x im DirectX SDK verwendet.
