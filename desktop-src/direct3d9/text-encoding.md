# <a name="text-encoding"></a>Textcodierung

Das Dateiformat DirectX. x ist nicht spezifisch für eine Anwendung. Es werden Vorlagen verwendet, die nicht von der Verwendung der Datei abhängig sind. Dadurch kann das Dateiformat. x von jeder Client Anwendung verwendet werden.

In den folgenden Abschnitten werden der Inhalt und die Syntax des Datei Formats behandelt, in dem die Erweiterung ". x" im DirectX SDK verwendet wird.

-   [Vorlagen](dx9-graphics-reference-x-file-textencoding-templates.md)
