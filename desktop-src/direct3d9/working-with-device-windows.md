# <a name="working-with-device-windows-direct3d-9"></a>Arbeiten mit Device Windows (Direct3D 9)

In diesem Abschnitt wird ein Problem aufgeführt, das beim Arbeiten mit Gerätefenstern in Direct3D-Anwendungen auftreten kann.

-   Direct3D richtet nur die Fokusfenster anstelle des Gerätefensters mit der Direct3D-Nachrichtenverarbeitungsfunktion ein und verarbeitet nur die Nachrichten im Fokusfenster. Daher sollte das Fokusfenster das übergeordnete Fenster eines beliebigen Gerätefensters sein.
