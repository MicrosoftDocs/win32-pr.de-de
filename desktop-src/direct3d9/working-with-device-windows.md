# <a name="working-with-device-windows-direct3d-9"></a>Arbeiten mit Geräte Fenstern (Direct3D 9)

In diesem Abschnitt wird ein Problem aufgeführt, das bei der Arbeit mit Geräte Fenstern in Direct3D-Anwendungen auftreten kann.

-   Direct3D verknüpft nur die Fokus Fenster anstelle des Geräte Fensters mit der Direct3D-Nachrichten Verarbeitungs Funktion und verarbeitet nur die Fokus Fenster Meldungen. Daher sollte das Fokus Fenster das übergeordnete Element jedes Geräte Fensters sein.
