# <a name="id3dxlinegetpattern-method"></a>ID3DXLine:: GetPattern-Methode

Ruft das Zeilen stippingmuster ab.

## <a name="syntax"></a>Syntax


```C++
DWORD GetPattern();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **DWORD**](../winprog/windows-data-types.md)**

Gibt das Zeilen stippingmuster zurück: 1 ist undurchsichtig, 0 ist transparent.
