# <a name="id3dxbasemeshgetnumbytespervertex-method"></a>ID3DXBaseMesh::GetNumBytesPerVertex-Methode

Ruft die Anzahl der Bytes pro Scheitelpunkt ab.

## <a name="syntax"></a>Syntax


```C++
DWORD GetNumBytesPerVertex();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Typ: **[ **DWORD**](../winprog/windows-data-types.md)**

Gibt die Anzahl der Bytes pro Scheitelpunkt zurück.
