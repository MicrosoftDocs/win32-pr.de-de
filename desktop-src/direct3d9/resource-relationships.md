# <a name="resource-relationships-direct3d-9"></a>Ressourcenbeziehungen (Direct3D 9)

Das folgende Diagramm zeigt die Vorgänge, die syntaktisch für bestimmte Ressourcen und deren Inhalt möglich sind. Weitere Informationen zur Verwendung von Ressourcen finden Sie unter [Bearbeiten von Ressourcen (Direct3D 9).](manipulating-resources.md)

![Diagramm der Ressourcenbeziehungen](images/resourcerelationship.png)

Pfeile, die von links nach rechts ausgeführt werden, geben an, dass die Zieltypen von den angegebenen Methoden erstellt werden, während Pfeile, die von rechts nach links ausgeführt werden, angeben, dass diese Ressourcentypen als Argumente an die angegebenen Methoden übergeben werden können.
