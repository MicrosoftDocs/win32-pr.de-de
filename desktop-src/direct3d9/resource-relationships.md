# <a name="resource-relationships-direct3d-9"></a>Ressourcen Beziehungen (Direct3D 9)

Das folgende Diagramm zeigt die Vorgänge, die für bestimmte Ressourcen und ihren Inhalt syntaktisch möglich sind. Weitere Informationen zum Verwenden von Ressourcen finden Sie unter Bearbeiten von [Ressourcen (Direct3D 9)](manipulating-resources.md).

![Diagramm der Ressourcen Beziehungen](images/resourcerelationship.png)

Pfeile, die von links nach rechts ausgeführt werden, zeigen an, dass die Zieltypen von den angegebenen Methoden erstellt werden, während Pfeile von rechts nach links anzeigen, dass diese Ressourcentypen als Argumente an die angegebenen Methoden übergeben werden können.
