# <a name="frametransformmatrix"></a>Frametransformmatrix

Definiert eine lokale Transformation für einen Frame (und alle seine untergeordneten Objekte).

``` syntax
template FrameTransformMatrix
{
    < F6F23F41-7686-11cf-8F52-0040333594A3 >
    Matrix4x4 frameMatrix;
} 
```

Hierbei gilt:

-   framematrix: eine Matrix4x4-Vorlage. Siehe [**Matrix4x4**](matrix4x4.md).
