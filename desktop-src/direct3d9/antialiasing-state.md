# <a name="antialiasing-state-direct3d-9"></a>Antialiasing-Zustand (Direct3D 9)

[Antialiasing (Direct3D 9)](antialiasing.md) ist eine Methode, mit der Linien und Kanten auf dem Bildschirm glatter angezeigt werden. Standardmäßig führt Direct3D keine Antialiasing durch. Um das Vollbild-Antialiasing zu aktivieren, legen \_ Sie den D3DRS multisampleantialias-Rendering-Zustand auf **true** fest. Legen Sie D3DRS \_ multisampleantialias auf **false** fest, um es zu deaktivieren.
