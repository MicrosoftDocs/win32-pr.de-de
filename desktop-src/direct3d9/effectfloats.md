# <a name="effectfloats"></a>EffectFloats

Definiert einen Legacyeffekt FLOAT. Diese Vorlage wurde durch die [**EffectParamFloats-Vorlage**](effectparamfloats.md) ersetzt.

``` syntax
template EffectFloats
{
    < F1CFE2B3-0DE3-4e28-AFA1-155A750A282D >
    DWORD nFloats;
    array float Floats[nFloats];
} 
```

Hierbei gilt:

-   nFloats: Anzahl der Gleitkommazahlen.
-   Floats \[ nFloats: \] Array von float-Werten.
