# <a name="effectfloats"></a>Effectfloat

Definiert einen Legacy Effekt-float-Wert. Diese Vorlage wurde durch die [**effectparamfloat**](effectparamfloats.md) -Vorlage ersetzt.

``` syntax
template EffectFloats
{
    < F1CFE2B3-0DE3-4e28-AFA1-155A750A282D >
    DWORD nFloats;
    array float Floats[nFloats];
} 
```

Hierbei gilt:

-   nfloat-Anzahl von Gleit Komma Zahlen.
-   Schwebt \[ \] nfloat-Array von float-Werten.
