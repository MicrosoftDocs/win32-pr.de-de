# <a name="wmdm-programming-guide"></a>WMDM-Programmier Handbuch

In diesem Abschnitt werden Aktionen beschrieben, die eine Anwendung oder ein Dienstanbieter, die auf Windows Media Device Manager erstellt wurde, ausführen sollte

-   [Zu den Aufgaben, die von Anwendungen und Dienstanbietern gemeinsam](tasks-common-to-applications-and-service-providers.md) verwendet werden, werden Aufgaben beschrieben, die sowohl von Anwendungen als auch von Dienstanbietern ausgeführt werden, z. b. Protokollierung, Verschlüsselung und Entschlüsselung
-   Beim [Erstellen einer Windows Media Device Manager-Anwendung](creating-a-windows-media-device-manager-application.md) werden alle Aktionen beschrieben, die eine Anwendung in der Regel ausführen muss.
-   Beim [Erstellen eines Dienstanbieters](creating-a-service-provider.md) werden alle Aktionen beschrieben, die ein Dienstanbieter normalerweise ausführen muss.
