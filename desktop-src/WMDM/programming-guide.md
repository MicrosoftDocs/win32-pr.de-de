# <a name="wmdm-programming-guide"></a>WMDM-Programmierhandbuch

In diesem Abschnitt werden Aktionen beschrieben, die von einer Anwendung oder einem Dienstanbieter, die auf Windows Media Geräte-Manager, durchgeführt werden sollten.

-   [Allgemeine Aufgaben für Anwendungen und Dienstanbieter](tasks-common-to-applications-and-service-providers.md) beschreibt Aufgaben, die sowohl anwendungen als auch Dienstanbieter ausführen, z. B. Protokollierung, Verschlüsselung und Entschlüsselung und Authentifizierung.
-   [Beim Erstellen Windows Media Geräte-Manager-Anwendung](creating-a-windows-media-device-manager-application.md) werden alle Aktionen beschrieben, die eine Anwendung in der Regel ausführen muss.
-   [Beim Erstellen eines Dienstanbieters](creating-a-service-provider.md) werden alle Aktionen beschrieben, die ein Dienstanbieter normalerweise ausführen muss.
