# <a name="privacy-statement-windows-media-device-manager-11-sdk"></a>Datenschutzbestimmungen (Windows Media Device Manager 11 SDK)

Entwickler, die Windows Media Device Manager-Anwendungen erstellen, sollten Ihren Benutzern eindeutig mitteilen, wie Ihr Datenschutz beeinträchtigt werden könnte. Zur Unterstützung beim Erstellen einer Datenschutzerklärung für Endbenutzer hat Microsoft auf der [Windows Media-Webseite](https://www.microsoft.com/windows/windowsmedia/privacy/10ssdk.aspx)Datenschutzrichtlinien veröffentlicht.
