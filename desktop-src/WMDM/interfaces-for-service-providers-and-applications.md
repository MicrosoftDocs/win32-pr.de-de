# <a name="interfaces-for-service-providers-and-applications"></a>Schnittstellen für Dienstanbieter und Anwendungen

Die folgenden Schnittstellen können von Dienstanbietern und Anwendungen verwendet werden, die das Windows Media Device Manager SDK verwenden.



| Schnittstelle                                                | BESCHREIBUNG                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**Iwmdmlogger**](/windows/desktop/api/wmdmlog/nn-wmdmlog-iwmdmlogger)                       | Protokolliert Einträge in einer allgemeinen Protokolldatei.                                                        |
| [**Icomponentauthenticate**](/windows/desktop/api/mswmdm/nn-mswmdm-icomponentauthenticate) | Bietet sichere, verschlüsselte Kommunikation zwischen Modulen von Windows Media Device Manager. |
