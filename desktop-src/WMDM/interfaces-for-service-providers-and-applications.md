# <a name="interfaces-for-service-providers-and-applications"></a>Schnittstellen für Dienstanbieter und Anwendungen

Die folgenden Schnittstellen können sowohl von Dienstanbietern als auch von Anwendungen verwendet werden, die das Windows Media Geräte-Manager SDK verwenden.



| Schnittstelle                                                | BESCHREIBUNG                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**IWMDMLogger**](/windows/desktop/api/wmdmlog/nn-wmdmlog-iwmdmlogger)                       | Protokolliert Einträge in einer gemeinsamen Protokolldatei.                                                        |
| [**IComponentAuthenticate**](/windows/desktop/api/mswmdm/nn-mswmdm-icomponentauthenticate) | Bietet eine sichere, verschlüsselte Kommunikation zwischen Modulen Windows Media Geräte-Manager. |
