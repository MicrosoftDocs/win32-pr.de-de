# <a name="compiling-the-sample-service-provider-using-nmake"></a>Kompilieren des Beispieldienstanbieters mithilfe von NMAKE

Um den Beispieldienstanbieter mit NMAKE zu kompilieren, öffnen Sie ein Visual Studio-Befehlsfenster, navigieren Sie zum <SDK-Installationspfad > \\ WMFSDK95 \\ WMDM \\ mdsp, und geben Sie an der Eingabeaufforderung `nmake` ein.
