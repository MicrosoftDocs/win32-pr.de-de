# <a name="compiling-the-sample-service-provider-using-nmake"></a>Kompilieren des Beispiel Dienstanbieters mithilfe von NMAKE

Um den Beispiel Dienstanbieter mithilfe von NMAKE zu kompilieren, öffnen Sie ein Visual Studio-Befehlsfenster, navigieren Sie zu <*SDK-Installationspfad* > \\ WMFSDK95 \\ WMDM \\ mdsp, und geben Sie `nmake` an der Eingabeaufforderung ein.
