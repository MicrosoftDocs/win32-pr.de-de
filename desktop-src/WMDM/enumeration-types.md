# <a name="wmdm-enumerations"></a>WMDM-Enumerationen

In diesem Thema wird ein Feature des Windows Media Geräte-Manager SDK dokumentiert. Es wird empfohlen, Dass Sie Ihre Anwendung migrieren, um die api Windows Portable Devices zu verwenden. Weitere Informationen finden Sie im Windows PORTABLE DEVICES SDK.

Windows Media Geräte-Manager definiert die folgenden Enumerationen.



| Enumerationstyp                                                                  | BESCHREIBUNG                                                                                     |
|-----------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**WMDM \_ ENUM \_ PROP \_ VALID \_ VALUES \_ FORM**](wmdm-enum-prop-valid-values-form.md) | Beschreibt mögliche Formen gültiger Werte für eine Eigenschaft.                                        |
| [**\_WMDM-FIND-BEREICH \_**](wmdm-find-scope.md)                                      | Definiert den Bereich des Speicherobjekts.                                                        |
| [**\_WMDM-FORMATCODE**](wmdm-formatcode.md)                                       | Definiert eine Liste von Formatcodes, die Inhaltstypen beschreiben, die auf und von einem Gerät übertragen werden. |
| [**\_WMDM-SITZUNGSTYP \_**](wmdm-session-type.md)                                  | Definiert den Sitzungstyp.                                                                       |
| [**\_ \_ WMDM-SPEICHER-ENUM-MODUS \_**](wmdm-storage-enum-mode.md)                       | Definiert, wie der Inhalt im Speicher aufzählt werden soll.                                     |
| [**\_WMDM-TAGDATENTYP \_**](wmdm-tag-datatype.md)                                  | Definiert einen Datentyp.                                                                            |
| [**WMDMMessage**](wmdmmessage.md)                                                | Definiert Nachrichtentypen und -zustände.                                                               |
