# <a name="wmdm-reference"></a>WMDM-Referenz

Das Windows Media Geräte-Manager SDK besteht aus einer Sammlung von Schnittstellen, Strukturen, Enumerationen und Konstanten. Der Verweisabschnitt unterteilt die Schnittstellen in Gruppen gemäß dem Typ des Objekts, das sie verwendet.



| `Section`                                                                                                    | BESCHREIBUNG                                                                                                                                                                                         |
|------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Schnittstellen für Anwendungen](interfaces-for-applications.md)                                             | Beschreibt Schnittstellen, die nur von Desktopanwendungen, Windows Media Player Plug-Ins oder COM-Objekten verwendet oder implementiert werden, die zugriff auf ein portables Gerät auf hoher Ebene benötigen.                     |
| [Schnittstellen für Dienstanbieter](interfaces-for-service-providers.md)                                   | Beschreibt Schnittstellen, die nur von Dienstanbietern verwendet oder implementiert werden, die die eigentliche Low-Level-Kommunikation mit einem portablen Gerät verarbeiten.                                                |
| [Windows Medienschnittstellen DRM-Implemented Medienschnittstellen](windows-media-drm-implemented-interfaces.md)                   | Beschreibt Schnittstellen, die nicht von Drittanbietern implementiert werden sollen, sondern intern vom Windows Media DRM und Windows Media Geräte-Manager. |
| [Schnittstellen für Dienstanbieter und Anwendungen](interfaces-for-service-providers-and-applications.md) | Beschreibt Schnittstellen, die entweder von Anwendungen oder von Dienstanbietern verwendet werden können.                                                                                                               |
| [Schnittstellen für sichere Inhaltsanbieter](interfaces-for-secure-content-providers.md)                     | Beschreibt Schnittstellen, die von einem Gerät oder einer Anwendung implementiert werden müssen, das bzw. die DRM-geschützte Inhalte verwendet.                                                                                       |
| [Strukturen](structures.md)                                                                               | Beschreibt Strukturen, die für Windows Media Geräte-Manager verwendet werden.                                                                                                                                 |
| [Enumerationstypen](enumeration-types.md)                                                                 | Beschreibt Enumerationen, die für Windows Media Geräte-Manager verwendet werden.                                                                                                                               |
| [Metadatenkonst constants](metadata-constants.md)                                                               | Beschreibt die Konstanten, die vom Windows Media Geräte-Manager SDK zum Festlegen oder Abrufen verschiedener Metadateneigenschaften für ein Gerät oder Objekt auf dem Gerät definiert werden.                                 |
| [Fehlercodes](error-codes.md)                                                                             | Beschreibt die Fehlercodes, die von den Media Windows SDK-Methoden Geräte-Manager werden können.                                                                                                         |
