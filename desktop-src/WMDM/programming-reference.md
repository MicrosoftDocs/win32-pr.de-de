# <a name="wmdm-reference"></a>WMDM-Referenz

Das Windows Media Device Manager SDK besteht aus einer Auflistung von Schnittstellen, Strukturen, Enumerationen und Konstanten. Im Referenz Abschnitt werden die Schnittstellen gemäß dem Objekttyp, der Sie verwendet, in Gruppen aufgeteilt.



| `Section`                                                                                                    | BESCHREIBUNG                                                                                                                                                                                         |
|------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Schnittstellen für Anwendungen](interfaces-for-applications.md)                                             | Beschreibt Schnittstellen, die nur von Desktop Anwendungen, Windows Media Player-Plug-ins oder COM-Objekten verwendet werden, die Zugriff auf hoher Ebene auf ein tragbares Gerät benötigen.                     |
| [Schnittstellen für Dienstanbieter](interfaces-for-service-providers.md)                                   | Beschreibt Schnittstellen, die nur von Dienstanbietern verwendet oder implementiert werden, die die tatsächliche Kommunikation auf niedriger Ebene mit einem tragbaren Gerät verarbeiten.                                                |
| [Windows Media DRM-Implemented-Schnittstellen](windows-media-drm-implemented-interfaces.md)                   | Beschreibt Schnittstellen, die nicht für die Implementierung durch Dienstanbieter von Drittanbietern vorgesehen sind, aber intern von Windows Media DRM und Windows Media Device Manager implementiert und aufgerufen werden. |
| [Schnittstellen für Dienstanbieter und Anwendungen](interfaces-for-service-providers-and-applications.md) | Beschreibt Schnittstellen, die von Anwendungen oder Dienstanbietern verwendet werden können.                                                                                                               |
| [Schnittstellen für sichere Inhaltsanbieter](interfaces-for-secure-content-providers.md)                     | Beschreibt Schnittstellen, die von einem Gerät oder einer Anwendung implementiert werden müssen, das von DRM geschützte Inhalte verwendet.                                                                                       |
| [Strukturen](structures.md)                                                                               | Beschreibt Strukturen, die für Windows Media Device Manager-Methoden verwendet werden.                                                                                                                                 |
| [Enumerationstypen](enumeration-types.md)                                                                 | Beschreibt Enumerationen, die für Windows Media Device Manager-Methoden verwendet werden.                                                                                                                               |
| [Metadatenkonstanten](metadata-constants.md)                                                               | Beschreibt die Konstanten, die vom Windows Media Device Manager SDK zum Festlegen oder Abrufen verschiedener Metadateneigenschaften für ein Gerät oder Objekt auf dem Gerät definiert werden.                                 |
| [Fehlercodes](error-codes.md)                                                                             | Beschreibt die Fehlercodes, die von den Windows Media Device Manager SDK-Methoden zurückgegeben werden können.                                                                                                         |
