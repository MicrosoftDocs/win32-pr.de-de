# <a name="wmdm-reference"></a>WMDM-Referenz

Das Windows Media Geräte-Manager SDK besteht aus einer Auflistung von Schnittstellen, Strukturen, Enumerationen und Konstanten. Im Verweisabschnitt werden die Schnittstellen entsprechend dem Objekttyp, der sie verwendet, in Gruppen unterteilt.



| `Section`                                                                                                    | BESCHREIBUNG                                                                                                                                                                                         |
|------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Schnittstellen für Anwendungen](interfaces-for-applications.md)                                             | Beschreibt Schnittstellen, die nur von Desktopanwendungen, Windows Media Player Plug-Ins oder COM-Objekten verwendet oder implementiert werden, die zugriff auf ein portables Gerät auf hoher Ebene benötigen.                     |
| [Schnittstellen für Dienstanbieter](interfaces-for-service-providers.md)                                   | Beschreibt Schnittstellen, die nur von Dienstanbietern verwendet oder implementiert werden, die die tatsächliche Kommunikation auf niedriger Ebene mit einem portablen Gerät verarbeiten.                                                |
| [Windows Media DRM-Implemented Interfaces](windows-media-drm-implemented-interfaces.md)                   | Beschreibt Schnittstellen, die nicht von Drittanbietern implementiert werden sollen, sondern intern vom Windows Media DRM- und Windows Media-Geräte-Manager implementiert und aufgerufen werden. |
| [Schnittstellen für Dienstanbieter und Anwendungen](interfaces-for-service-providers-and-applications.md) | Beschreibt Schnittstellen, die von Anwendungen oder Dienstanbietern verwendet werden können.                                                                                                               |
| [Schnittstellen für sichere Inhaltsanbieter](interfaces-for-secure-content-providers.md)                     | Beschreibt Schnittstellen, die von einem Gerät oder einer Anwendung implementiert werden müssen, die DRM-geschützte Inhalte verwenden.                                                                                       |
| [Strukturen](structures.md)                                                                               | Beschreibt Strukturen, die für Windows Media Geräte-Manager-Methoden verwendet werden.                                                                                                                                 |
| [Enumerationstypen](enumeration-types.md)                                                                 | Beschreibt Enumerationen, die für Windows Media Geräte-Manager-Methoden verwendet werden.                                                                                                                               |
| [Metadatenkonstanten](metadata-constants.md)                                                               | Beschreibt die Konstanten, die vom Windows Media Geräte-Manager SDK zum Festlegen oder Abrufen verschiedener Metadateneigenschaften für ein Gerät oder Objekt auf dem Gerät definiert werden.                                 |
| [Fehlercodes](error-codes.md)                                                                             | Beschreibt die Fehlercodes, die von Windows Media Geräte-Manager SDK-Methoden zurückgegeben werden können.                                                                                                         |
