# <a name="methods-imanipulationprocessor"></a>Methoden (IManipulationProcessor)

Dieser Abschnitt enthält die Methoden für die [**IManipulationProcessor-Schnittstelle.**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor)

Die folgenden Methoden werden von der [**IManipulationProcessor-Schnittstelle**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) verfügbar gemacht.



| Methode                                                                      | BESCHREIBUNG                                                                              |
|-----------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| [**CompleteManipulation**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-completemanipulation) | Diese Methode wird aufgerufen, wenn der Entwickler die Bearbeitung beendet.                |
| [**GetAngularVelocity**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getangularvelocity)     | Berechnet die Drehgeschwindigkeit, mit der sich das Zielobjekt bewegt.                  |
| [**GetExpansionVelocity**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getexpansionvelocity) | Berechnet die Rate, mit der das Zielobjekt erweitert wird.                              |
| [**GetVelocityX**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getvelocityx)                 | Berechnet die horizontale Geschwindigkeit für das Zielobjekt und gibt sie zurück.                    |
| [**GetVelocityY**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getvelocityy)                 | Berechnet die vertikale Geschwindigkeit und gibt sie zurück.                                            |
| [**ProcessDown**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processdown)                   | Gibt Daten an den Bearbeitungsprozessor weiter, der einem Ziel zugeordnet ist.                       |
| [**ProcessMove**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processmove)                   | Gibt Daten an den Bearbeitungsprozessor weiter, der einem Ziel zugeordnet ist.                       |
| [**ProcessUp**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processup)                       | Gibt Daten an den Bearbeitungsprozessor weiter, der einem Ziel zugeordnet ist.                       |
| [**ProcessDownWithTime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processdownwithtime)   | Gibt Daten einschließlich eines Zeitstempels an den einem Ziel zugeordneten Bearbeitungsprozessor weiter. |
| [**ProcessMoveWithTime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processmovewithtime)   | Gibt Daten einschließlich eines Zeitstempels an den einem Ziel zugeordneten Bearbeitungsprozessor weiter. |
| [**ProcessUpWithTime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processupwithtime)       | Gibt Daten einschließlich eines Zeitstempels an den einem Ziel zugeordneten Bearbeitungsprozessor weiter. |
