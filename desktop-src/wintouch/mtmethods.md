# <a name="methods-imanipulationprocessor"></a>Methoden (IManipulationProcessor)

Dieser Abschnitt enthält die Methoden für die [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) -Schnittstelle.

Die folgenden Methoden werden von der [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) -Schnittstelle verfügbar gemacht.



| Methode                                                                      | BESCHREIBUNG                                                                              |
|-----------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| [**CompleteManipulation**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-completemanipulation) | Diese Methode wird aufgerufen, wenn der Entwickler entscheidet, die Bearbeitung zu beenden.                |
| [**Getangularvelocity**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getangularvelocity)     | Berechnet die Rotationsgeschwindigkeit, in der das Zielobjekt bewegt wird.                  |
| [**Getexpansionvelocity**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getexpansionvelocity) | Berechnet die Rate, um die das Zielobjekt erweitert wird.                              |
| [**Getvelocityx**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getvelocityx)                 | Berechnet die horizontale Geschwindigkeit für das Zielobjekt und gibt Sie zurück.                    |
| [**Getvelocityy**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getvelocityy)                 | Berechnet die vertikale Geschwindigkeit und gibt Sie zurück.                                            |
| [**Processdown**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processdown)                   | Feeds von Daten an den Manipulations Prozessor, der einem Ziel zugeordnet ist.                       |
| [**Processmove**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processmove)                   | Feeds von Daten an den Manipulations Prozessor, der einem Ziel zugeordnet ist.                       |
| [**Processup**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processup)                       | Feeds von Daten an den Manipulations Prozessor, der einem Ziel zugeordnet ist.                       |
| [**Processdownwithtime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processdownwithtime)   | Feeds von Daten, einschließlich eines Zeitstempels, auf den einem Ziel zugeordneten Bearbeitungs Prozessor. |
| [**Processmuvewithtime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processmovewithtime)   | Feeds von Daten, einschließlich eines Zeitstempels, auf den einem Ziel zugeordneten Bearbeitungs Prozessor. |
| [**Processupwithtime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processupwithtime)       | Feeds von Daten, einschließlich eines Zeitstempels, auf den einem Ziel zugeordneten Bearbeitungs Prozessor. |
