# <a name="properties-imanipulationprocessor"></a>Eigenschaften (IManipulationProcessor)

Dieser Abschnitt enthält die Eigenschaften für die [**IManipulationProcessor-Schnittstelle.**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor)

Die folgenden Eigenschaften werden von der IManipulationProcessor-Schnittstelle verfügbar gemacht.



| Eigenschaft                                                                            | Beschreibung                                                                                                |
|-------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**MinimumScaleRotateRadius**](imanipulationprocessor-minimumscalerotateradius.md) | Gibt an, wie groß die Abstandskontakte in einer Skalierungs- oder Drehbewegung sein müssen, um die Bearbeitung auszulösen. |
| [**PivotPointX**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_pivotpointx)                           | Diese Eigenschaft ist der horizontale Mittelpunkt des Objekts.                                                      |
| [**PivotPointY**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_pivotpointy)                           | Diese Eigenschaft ist die vertikale Mitte des Objekts.                                                        |
| [**PivotRadius**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_pivotradius)                           | Diese Eigenschaft wird verwendet, um zu bestimmen, wie viel Drehung bei der Bearbeitung mit einem Finger verwendet wird.                |
| [**SupportedManipulations**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_supportedmanipulations)     | Diese Eigenschaft wird verwendet, um anzugeben, welche Bearbeitungen von einem -Objekt unterstützt werden.                          |
