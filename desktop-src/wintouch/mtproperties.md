# <a name="properties-imanipulationprocessor"></a>Eigenschaften (IManipulationProcessor)

Dieser Abschnitt enthält die Eigenschaften für die [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) -Schnittstelle.

Die folgenden Eigenschaften werden von der IManipulationProcessor-Schnittstelle verfügbar gemacht.



| Eigenschaft                                                                            | BESCHREIBUNG                                                                                                |
|-------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**MinimumScaleRotateRadius**](imanipulationprocessor-minimumscalerotateradius.md) | Gibt an, wie groß die Entfernungs Kontakte in einer Skala oder Drehung sein müssen, um eine Manipulation zu initiieren. |
| [**Pivotpointx**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_pivotpointx)                           | Diese Eigenschaft ist die horizontale Mitte des Objekts.                                                      |
| [**Pivotpointy**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_pivotpointy)                           | Diese Eigenschaft ist die vertikale Mitte des Objekts.                                                        |
| [**Pivotradius**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_pivotradius)                           | Diese Eigenschaft wird verwendet, um zu bestimmen, wie viel Drehung bei einfacher Finger Bearbeitung verwendet wird.                |
| [**Supportedmanipulationen**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_supportedmanipulations)     | Diese Eigenschaft wird verwendet, um anzugeben, welche Manipulationen von einem Objekt unterstützt werden.                          |
