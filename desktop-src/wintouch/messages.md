# <a name="messages-touch-input"></a>Nachrichten (Toucheingabe)

Dieser Abschnitt enthält Meldungen für Multitoucheingaben.

Die folgenden Meldungen werden für multitouch-Eingaben verwendet.



| `Message`                           | BESCHREIBUNG                                                                                                         |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**WM \_ TOUCH**](wm-touchdown.md) | Gibt an, dass ein oder mehrere Berührungspunkte, z. B. ein Finger oder Stift, eine touchabhängige Digitizeroberfläche berührt haben. |
