# <a name="functions-boundary-feedback"></a>Funktionen (Begrenzungs Feedback)

In diesem Abschnitt werden die Funktionen zum Grenzen von Feedback beschrieben.



| Funktion                                               | BESCHREIBUNG                                                                                        |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [**Beginpanningfeedback**](/windows/win32/api/uxtheme/nf-uxtheme-beginpanningfeedback)     | Diese Funktion wird vom Benutzer vor der Verwendung des Begrenzungs Feedbacks aufgerufen.                               |
| [**Endpanningfeedback**](/windows/win32/api/uxtheme/nf-uxtheme-endpanningfeedback)     | Diese Funktion wird nach Abschluss des Begrenzungs Feedbacks aufgerufen und löst optional eine Animation aus. |
| [**Updatepanningfeedback**](/windows/win32/api/uxtheme/nf-uxtheme-updatepanningfeedback) | Diese Funktion wird während des Begrenzungs Feedbacks aufgerufen und löst optional eine Animation aus.             |
