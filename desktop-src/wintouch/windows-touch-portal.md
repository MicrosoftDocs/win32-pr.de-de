# <a name="touch-input"></a>Fingereingabe

## <a name="purpose"></a>Zweck

Windows unterstützt Berührungs Eingaben durch das Interaktionsmodell, das von der End-to-End-Plattform und dem Benutzeroberflächen Framework von Windows-Fingerabdruck verfügbar gemacht wird.

## <a name="developer-audience"></a>Entwicklergruppe

Die Features von Windows-Finger Eingaben sind für die Verwendung durch erfahrene C/C++-Entwickler konzipiert, die mit Component Object Model (com), Programmier Konzepten für die Benutzeroberfläche und allgemeinen Konzepten der Berührungs Interaktion vertraut sind.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Windows 7
