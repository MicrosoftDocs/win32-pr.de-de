# <a name="methods-_imanipulationevents"></a>Methoden (_IManipulationEvents)

Dieser Abschnitt enthält die Methoden für die [**\_ imanipulationevents**](/windows/win32/api/manipulations/nn-manipulations-_imanipulationevents) -Schnittstelle.



| Methode                                                                      | BESCHREIBUNG                                                   |
|-----------------------------------------------------------------------------|---------------------------------------------------------------|
| [**ManipulationStarted**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationstarted)     | Behandelt das-Ereignis für den Beginn der Bearbeitung oder Trägheit.    |
| [**ManipulationDelta**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationdelta)         | Behandelt Ereignisse, die auftreten, wenn sich ein manipuliertes Objekt ändert. |
| [**ManipulationCompleted**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationcompleted) | Behandelt das-Ereignis, wenn die Bearbeitung oder Trägheit beendet ist.      |
