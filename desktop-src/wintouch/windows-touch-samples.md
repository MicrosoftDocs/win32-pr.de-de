# <a name="windows-touch-samples"></a>Beispiele für Windows Touch

Beschreibt den Pfad zu den Windows-Berührungs Beispielen und bietet kurze Übersichten zu den Beispielen.

Wenn Sie die [Windows 7-Berührungs Beispiele]()installieren, können Sie auswählen, wo die Beispiele installiert werden. Wenn Sie z. b. auf Laufwerk c installieren, wird für das SDK standardmäßig "c: \\ Programme \\ Microsoft SDKs \\ Windows \\ v 7.0 \\ Samples" verwendet. Im Verzeichnis "Samples" befindet sich der Ordner "berühren". Dieser Ordner enthält alle Windows-Berührungs Beispiele.
