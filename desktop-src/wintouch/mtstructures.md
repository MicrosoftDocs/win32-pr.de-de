# <a name="structures-windows-touch-gestures"></a>Strukturen (Windows Touchgesten)

Dieser Abschnitt enthält Strukturen für Windows Touchgesten.

Die folgenden Strukturen werden in diesem Abschnitt angegeben.



| Struktur                                          | Beschreibung                                                                              |
|----------------------------------------------------|------------------------------------------------------------------------------------------|
| [**GESTURECONFIG**](/windows/win32/api/winuser/ns-winuser-gestureconfig)             | Wird verwendet, um die Konfiguration festzulegen, für die Gestenmeldungen aktiviert oder deaktiviert werden.      |
| [**GESTUREINFO**](/windows/win32/api/winuser/ns-winuser-gestureinfo)                 | Enthält Informationen zu einer Geste.                                                    |
| [**GESTURENOTIFYSTRUCT**](/windows/win32/api/winuser/ns-winuser-gesturenotifystruct) | Enthält Informationen zu einer [**\_ WM-GESTENNOTIFY-Benachrichtigung.**](wm-gesturenotify.md) |
