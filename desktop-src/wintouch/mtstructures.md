# <a name="structures-windows-touch-gestures"></a>Strukturen (Windows-Touchgesten)

Dieser Abschnitt enthält Strukturen für Windows-Touchgesten.

In diesem Abschnitt werden die folgenden Strukturen angegeben.



| Struktur                                          | BESCHREIBUNG                                                                              |
|----------------------------------------------------|------------------------------------------------------------------------------------------|
| [**Gestureconfig**](/windows/win32/api/winuser/ns-winuser-gestureconfig)             | Wird verwendet, um die Konfiguration festzulegen, welche Gesten Nachrichten aktiviert oder deaktiviert werden.      |
| [**GESTUREINFO**](/windows/win32/api/winuser/ns-winuser-gestureinfo)                 | Enthält Informationen über eine Geste.                                                    |
| [**Gesturenotifystruct**](/windows/win32/api/winuser/ns-winuser-gesturenotifystruct) | Enthält Informationen zu einer [**WM- \_ gesturenotify**](wm-gesturenotify.md) -Benachrichtigung. |
