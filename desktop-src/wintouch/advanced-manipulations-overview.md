# <a name="advanced-manipulations-overview"></a>Übersicht über erweiterte Manipulationen

In diesem Abschnitt werden erweiterte Bearbeitungen für Anwendungen erläutert.

Aus Gründen der Benutzerfreundlichkeit sollten Sie Ihrer Anwendung komplexe Bearbeitungen hinzufügen, damit Objekte mit einem hohen Maß an Granularität bearbeitet werden können. In den folgenden Abschnitten werden erweiterte Bearbeitungen beschrieben.

### <a name="advanced-expansion"></a>Erweiterte Erweiterung

Die folgende Abbildung zeigt zwei Interpretationen der Erweiterung.

![Abbildung der einfachen Erweiterung um den Mittelpunkt eines Objekts und der erweiterten Erweiterung um den Mittelpunkt der Manipulation](images/expansion.png)

In Beispiel A, dem einfachen Erweiterungsbeispiel, wird das Objekt um seinen Mittelpunkt erweitert. In Beispiel B wird das -Objekt um den Mittelpunkt der Bearbeitung erweitert.

### <a name="advanced-rotation"></a>Erweiterte Drehung

Die folgende Abbildung zeigt zwei Interpretationen der Drehung.

![Abbildung, die zwei Arten der Drehung mit einem Finger zeigt: um den Mittelpunkt oder um den Rand, wobei der Rand sowohl Drehung als auch Übersetzung umfasst](images/rotation.png)

In Beispiel A, dem einfachen Drehungsbeispiel, wird das Objekt um seinen Mittelpunkt gedreht. In Beispiel B wird das -Objekt um den Mittelpunkt der Bearbeitung gedreht. Weitere Informationen zur komplexen Rotation finden Sie im Abschnitt [Erweiterte Rotation.](advanced-rotation.md)

## <a name="advanced-translation"></a>Erweiterte Übersetzung

Die folgende Abbildung zeigt zwei Interpretationen der Übersetzung.

![Abbildung der einfachen Übersetzung, bei der ein Objekt ohne Drehung verschoben wird, und der erweiterten Übersetzung, die das Verschieben und Drehen umfasst](images/translation.png)

In Beispiel A, dem einfachen Übersetzungsbeispiel, wird das Objekt ohne Drehung verschoben. In Beispiel B wird das Objekt während der Übersetzung gedreht, je nachdem, wo sich der Objektkontaktpunkt befindet. Wenn Sie die Drehung mit einem Finger aktivieren, wie unter [Einfingerrotation](single-finger-rotation.md)beschrieben, können Sie die komplexe Übersetzung aktivieren.
