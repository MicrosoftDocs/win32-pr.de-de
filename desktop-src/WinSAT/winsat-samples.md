# <a name="winsat-samples"></a>WinSAT-Beispiele

\[WinSAT-Beispiele können nach Windows 8.1 geändert oder für Releases nicht verfügbar sein.\]

Das Windows-System Bewertungs Tool (WinSAT) bietet die folgenden Beispiele in der [Windows SDK](https://msdn.microsoft.com/windows/bb980924.aspx).



| Beispiel             | BESCHREIBUNG                                                                    |
|--------------------|--------------------------------------------------------------------------------|
| Initiateassessment | Zeigt, wie formale und benutzerdefinierte Bewertungen initiiert werden.                           |
| Querywinsat        | Zeigt, wie die WinSAT-Datenspeicher abgefragt werden, um die Ergebnisse einer Bewertung zu erhalten. |
