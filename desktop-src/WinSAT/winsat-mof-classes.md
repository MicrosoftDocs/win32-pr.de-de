# <a name="winsat-mof-classes"></a>WinSAT MOF-Klassen

\[WinSAT MOF-Klassen können nach Windows 8.1 geändert oder für Releases nicht verfügbar sein.\]

Das Windows System Assessment Tool (WinSAT) definiert die folgende Windows Management Instrumentation (WMI) Managed Object Format (MOF)-Klasse, die im \\ \\ Cimv2-Stammnamespace vorhanden ist:

-   [**Win32 \_ WinSAT**](win32-winsat.md)

Sie können diese Klasse verwenden, um zusammenfassende Informationen zur letzten formalen Bewertung abzurufen.
