# <a name="winsat-mof-classes"></a>WinSAT-MOF-Klassen

\[WinSAT-MOF-Klassen können nach Windows 8.1 geändert werden oder für Releases nicht verfügbar sein.\]

Mit dem Windows-System Bewertungs Tool (WinSAT) wird die folgende Windows-Verwaltungsinstrumentation (WMI) Managed Object Format (MOF)-Klasse definiert, die im \\ root \\ CIMV2-Namespace vorhanden ist:

-   [**Win32- \_ WinSAT**](win32-winsat.md)

Sie können diese Klasse verwenden, um zusammenfassende Informationen über die aktuellste formale Bewertung abzurufen.
