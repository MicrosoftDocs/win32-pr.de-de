# <a name="windows-system-assessment-tool-provider"></a>Anbieter des Windows-System Bewertungs Tools

\[WinSAT kann nach Windows 8.1 geändert werden oder für Releases nicht verfügbar sein.\]

## <a name="purpose"></a>Zweck

Das Windows-System Bewertungs Tool (WinSAT) stellt eine Reihe von Klassen zur Verfügung, mit denen die Leistungsmerkmale und-Funktionen eines Computers bewertet werden. Entwickler können diese API verwenden, um Software zu entwickeln, die auf die Leistungs-und Funktions Informationen eines Computers zugreifen kann, um die optimalen Anwendungseinstellungen basierend auf den Leistungs Funktionen dieses Computers zu ermitteln.

Benutzer können die Ergebnisse einer formalen Bewertung verwenden, um zu bestimmen, welche Szenarien und Anwendungen auf einem Computer gut funktionieren. Wenn beispielsweise ein Softwarepaket eine Bewertung der Verpackung enthält, kann ein Benutzer anhand der Bewertung feststellen, ob die Software auf dem Computer ordnungsgemäß ausgeführt wird.

## <a name="developer-audience"></a>Entwicklergruppe

WinSAT ist für C-, C++-und Visual Basic Entwickler sowie für Entwickler konzipiert, die Skripts schreiben.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

WinSAT ist auf Client Computern ab dem Betriebssystem Windows Vista verfügbar.
