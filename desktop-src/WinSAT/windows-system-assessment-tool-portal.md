# <a name="windows-system-assessment-tool-provider"></a>Windows Anbieter des Systembewertungstools

\[WinSAT kann nach der Veröffentlichung geändert oder nicht mehr Windows 8.1.\]

## <a name="purpose"></a>Zweck

Das Windows System Assessment Tool (WinSAT) macht eine Reihe von Klassen verfügbar, die die Leistungsmerkmale und Funktionen eines Computers bewerten. Entwickler können diese API verwenden, um Software zu entwickeln, die auf die Leistungs- und Funktionsinformationen eines Computers zugreifen kann, um die optimalen Anwendungseinstellungen basierend auf den Leistungsfunktionen dieses Computers zu bestimmen.

Benutzer können die Ergebnisse einer formalen Bewertung verwenden, um zu bestimmen, welche Szenarien und Anwendungen auf einem Computer eine gute Leistung bieten. Wenn ein Softwarepaket z. B. eine Bewertung für die Paketierung enthält, kann ein Benutzer die Bewertung verwenden, um zu bestimmen, ob die Software gut auf dem Computer ausgeführt wird.

## <a name="developer-audience"></a>Entwicklergruppe

WinSAT ist für C-, C++- und Visual Basic und Entwickler konzipiert, die Skripts schreiben.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

WinSAT ist ab dem Betriebssystem Windows Vista auf Clientcomputern verfügbar.
