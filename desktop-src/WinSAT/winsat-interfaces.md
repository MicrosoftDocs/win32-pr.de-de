# <a name="winsat-interfaces"></a>WinSAT-Schnittstellen

\[WinSAT-Schnittstellen können nach Windows 8.1 geändert werden oder für Releases nicht verfügbar sein.\]

Das Windows-System Bewertungs Tool (WinSAT) definiert die folgenden Schnittstellen, die Sie verwenden können, um Bewertungen auf dem Computer zu initiieren und um formale Bewertungen im WinSAT-Datenspeicher abzufragen:

-   [**Iinitiatewinsatassessment**](/windows/desktop/api/Winsatcominterfacei/nn-winsatcominterfacei-iinitiatewinsatassessment)
-   [**Iprovidewinsatgutamentinfo**](/windows/desktop/api/Winsatcominterfacei/nn-winsatcominterfacei-iprovidewinsatassessmentinfo)
-   [**Iprovide winsatresultsinfo**](/windows/desktop/api/Winsatcominterfacei/nn-winsatcominterfacei-iprovidewinsatresultsinfo)
-   [**Iprovide winsatvisuals**](/windows/desktop/api/Winsatcominterfacei/nn-winsatcominterfacei-iprovidewinsatvisuals)
-   [**Iqueryallwinsatassessments**](/windows/desktop/api/Winsatcominterfacei/nn-winsatcominterfacei-iqueryallwinsatassessments)
-   [**Iqueryrecentwinsatassessment**](/windows/desktop/api/Winsatcominterfacei/nn-winsatcominterfacei-iqueryrecentwinsatassessment)
-   [**Iwinsatinitiateevents**](/windows/desktop/api/Winsatcominterfacei/nn-winsatcominterfacei-iwinsatinitiateevents)

WinSAT definiert auch eine Windows-Verwaltungsinstrumentation (WMI) Managed Object Format (MOF)- [Klasse](winsat-mof-classes.md) , die eine ähnliche Funktionalität bereitstellt.
