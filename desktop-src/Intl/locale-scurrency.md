# <a name="locale_scurrency"></a>Gebiets Schema- \_ scurrency

Zeichenfolge, die als Lokales Währungssymbol verwendet wird. Die maximal zulässige Anzahl von Zeichen für diese Zeichenfolge ist 13, einschließlich eines abschließenden NULL-Zeichens. In der USA lautet das Symbol beispielsweise "$".
