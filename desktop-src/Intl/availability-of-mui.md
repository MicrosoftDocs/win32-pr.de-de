# <a name="availability-of-mui"></a>Verfügbarkeit von AVAILABILITY

Die folgenden Versionen von Windows ENTHALTEN APIs und Konstrukte, um native Unterstützung für die Entwicklung mehrsprachiger Anwendungen mithilfe von TECHNOLOGY zu bieten:

-   Windows 7 und höher.
-   Windows Vista

Eine downlevel-Bibliothek, [**LoadMUILibrary,**](/windows/desktop/api/Muiload/nf-muiload-loadmuilibrarya)wird bereitgestellt, um das Schreiben mehrsprachiger Anwendungen für Versionen von Windows vor Windows Vista zu ermöglichen und gleichzeitig ein Verhalten zu bieten, das mit dem für Windows Vista und spätere Versionen von Windows geschriebenen VERHALTEN kompatibel ist.
