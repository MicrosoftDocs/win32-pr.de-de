# <a name="availability-of-mui"></a>Verfügbarkeit von MUI

Die folgenden Versionen von Windows enthalten APIs und Konstrukte zum Bereitstellen von nativer Unterstützung für die Entwicklung mehrsprachiger Anwendungen mithilfe der MUI-Technologie

-   Windows 7 und höher.
-   Windows Vista

Die Bibliothek " [**loadmuilibrary**](/windows/desktop/api/Muiload/nf-muiload-loadmuilibrarya)" der Vorgängerversion ermöglicht das Schreiben mehrsprachiger Anwendungen für Windows-Versionen vor Windows Vista und bietet ein Verhalten, das mit der Verwendung von MUI-Anwendungen kompatibel ist, die für Windows Vista und spätere Versionen von Windows geschrieben werden.
