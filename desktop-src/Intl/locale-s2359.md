# <a name="locale_s2359-and-locale_spm"></a>LOCALE \_ S2359 und LOCALE \_ SPM

Zeichenfolge für den PM-Designator (zweite 12 Stunden des Tages). Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, ist für verschiedene Releases von Windows.

**Windows XP:** Unerkennlich, einschließlich eines beendenden NULL-Zeichens [**für SetLocaleInfo**](/windows/desktop/api/Winnls/nf-winnls-setlocaleinfoa). 15, einschließlich eines beendenden NULL-Zeichens [**für GetLocaleInfo**](/windows/desktop/api/Winnls/nf-winnls-getlocaleinfoa).

**Windows Me/98/95, Windows NT 4.0, Windows 2000:** Neun, einschließlich eines beendenden NULL-Zeichens.

**Windows Server 2003 und höher:** 15, einschließlich eines beendenden NULL-Zeichens.

Windows 10 den Wert **LOCALE \_ SPM** als lesbareres Synonym für **LOCALE \_ S2359 hinzugefügt.**
