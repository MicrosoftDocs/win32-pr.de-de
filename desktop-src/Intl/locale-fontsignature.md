# <a name="locale_fontsignature"></a>LOCALE \_ FONTSIGNATURE

**Windows Me/98/95, Windows NT 4.0:** Ein bestimmtes Bitmuster, das die Beziehung zwischen der Zeichenabdeckung bestimmt, die zur Unterstützung des Locale und des Schriftartinhalts erforderlich ist. Beachten Sie, dass LOCALE \_ FONTSIGNATURE-Daten ein anderes Format als alle anderen Informationen zum Locale haben. Alle anderen Locale-Informationen können in einer Zeichenfolgenform oder als Zahl ausgedrückt werden. LOCALE \_ FONTSIGNATURE-Daten werden in einer [LOCALESIGNATURE-Struktur](/windows/win32/api/wingdi/ns-wingdi-localesignature) abgerufen.
