# <a name="locale_fontsignature"></a>Gebiets Schema \_ fontSignature

**Windows Me/98/95, Windows NT 4,0:** Ein bestimmtes Bitmuster, das die Beziehung zwischen der Zeichen Abdeckung bestimmt, die zur Unterstützung des Gebiets Schemas und des Schriftart Inhalts erforderlich ist. Beachten Sie, dass Gebiets Schema- \_ fontSignature-Daten eine andere Form als alle anderen Gebiets Schema Informationen haben. Alle anderen Gebiets Schema Informationen können in Form einer Zeichenfolge oder als Zahl ausgedrückt werden. LOCALE \_ fontSignature-Daten werden in einer [LOCALESIGNATURE](/windows/win32/api/wingdi/ns-wingdi-localesignature) -Struktur abgerufen.
