# <a name="time-and-date"></a>Uhrzeit und Datum

NLS unterstützt das Abrufen von Zeichen folgen für beliebige Zeit oder Datumsangaben in einem Format, das für das aktuelle Gebiets Schema geeignet ist. Unter Windows Vista und höher unterstützt nls auch das Abrufen von Dauer Zeichenfolgen für beliebige Gebiets Schemas, einschließlich benutzerdefinierter Gebiets Schemas. Anweisungen zur Entwicklung finden Sie unter [Abrufen von Zeit-und Datumsangaben](retrieving-time-and-date-information.md).
