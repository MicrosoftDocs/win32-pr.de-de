# <a name="locale_icurrdigits"></a>Gebiets Schema- \_ icurrdigits

Anzahl der Bruch Ziffern für das lokale Währungs Format. Die maximal zulässige Anzahl von Zeichen für diese Zeichenfolge ist 2, einschließlich der Werte 0-9 und eines abschließenden NULL-Zeichens. 200,00 hat z. b. den Wert 2, da es zwei Dezimalstellen gibt, während 200 den Wert 0 hat, da keine Bruch Ziffern angezeigt werden.
