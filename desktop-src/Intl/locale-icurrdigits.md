# <a name="locale_icurrdigits"></a>LOCALE \_ ICURRDIGITS

Anzahl der Dezimalstellen für das lokale Währungsformat. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt zwei, einschließlich der Werte 0-9 und eines abschließenden NULL-Zeichens. Beispielsweise hat 200,00 den Wert 2, da zwei Dezimalstellen vorhanden sind, während 200 den Wert 0 hat, da keine Dezimalstellen angezeigt werden.
