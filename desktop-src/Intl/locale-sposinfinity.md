# <a name="locale_sposinfinity"></a>Gebiets Schema \_ sposinfinity

**Windows Vista und höher:** Zeichen folgen Wert für "positiv unendlich", z. b. "unendlich" für das englische Gebiets Schema (USA). Es gibt keine Beschränkung für die Anzahl der Zeichen, die für die Zeichenfolge zulässig sind.
