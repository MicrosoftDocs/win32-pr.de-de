# <a name="locale_imeasure"></a>LOCALE \_ IMEASURE

System der Messung. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt zwei, einschließlich eines beendenden NULL-Zeichens. Dieser Wert ist 0 , wenn das Metriksystem (Sysmetricme International d'Units oder S.I.) verwendet wird, und 1, wenn das USA verwendet wird.
