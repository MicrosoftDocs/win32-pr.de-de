System der Messung. Die maximal zulässige Anzahl von Zeichen für diese Zeichenfolge ist 2, einschließlich eines abschließenden NULL-Zeichens. Dieser Wert ist 0 (null), wenn das metriksystem (systyme International, oder S.I.) verwendet wird, und 1, wenn das USA System verwendet wird.
