# <a name="language-names"></a>Sprachnamen

Ein Sprachname verwendet im Wesentlichen die gleiche Formatierung wie ein [Gebietsschemaname.](locale-names.md) Beispielsweise lautet der Sprachname für das Gebietsschema "English (Soll)" "en-BZ". Jeder Sprachname entspricht einem [Sprachbezeichner.](language-identifiers.md)

Die Hauptunterschiede zwischen der Formatierung eines Sprachnamens und der Formatierung eines Gebietsschemanamens sind:

-   Ein Sprachname enthält nie eine Sortierreihenfolge. Daher ist der Sprachname, der dem Gebietsschemanamen "hu-HU \_ technl" entspricht (ungarisch, Französisch, mit technischer Sortierung), nur "hu-HU".
-   Ein Sprachname kann in Bezug auf das Gebietsschema "neutral" sein. "en" (Englisch) ist z. B. ein gültiger neutraler Sprachname, der sich von "en-BZ" (Englisch, Domänennamen) unterscheidet.
