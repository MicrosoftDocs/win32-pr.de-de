# <a name="language-names"></a>Sprachnamen

Ein Sprachen Name verwendet im Wesentlichen die gleiche Formatierung wie ein Gebiets Schema [Name](locale-names.md). Der Sprachen Name für das Gebiets Schema Englisch (Belize) lautet z. b. "en-BZ". Jeder sprach Name entspricht einem [sprach Bezeichner](language-identifiers.md).

Die Hauptunterschiede zwischen der Formatierung für einen Sprachen Namen und der Formatierung für einen Gebiets Schema Namen lauten wie folgt:

-   Ein Sprachen Name enthält nie eine Sortierreihenfolge. Daher ist der Name der Sprache, die dem Gebiets Schema Namen "HU-HU \_ technl" (ungarischer Sprache, Ungarn, mit technischer Sortierung) entspricht, einfach "HU-HU".
-   Ein Sprachen Name kann in Bezug auf das Gebiets Schema "neutral" sein. "En" (Englisch) ist beispielsweise ein gültiger neutraler Sprachen Name, der sich von "en-BZ" (Englisch, Belize) unterscheidet.
