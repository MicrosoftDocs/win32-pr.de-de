# <a name="locale_icurrency"></a>Gebiets Schema- \_ icurrency

Die Position des Währungs Symbols im positiven Währungs Modus.



| Wert | Bedeutung                                            |
|-------|----------------------------------------------------|
| 0     | Prefix, keine Trennung, z. b. $1,1           |
| 1     | Suffix, keine Trennung, z. b. $1,1           |
| 2     | Präfix, 1-Zeichen-Trennung, z. b. $1,1 |
| 3     | Suffix, 1-Zeichen-Trennung, z. b. $1,1 |
