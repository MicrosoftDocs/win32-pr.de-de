# <a name="locale_sdayname-constants"></a>LOCALE \_ \* SDAYNAME-Konstanten

In diesem Thema werden die LOCALE \_ SDAYNAME-Konstanten \* definiert, die von NLS verwendet werden.



| Wert             | Bedeutung                                                                                                                                 |
|-------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ SDAYNAME1 | Nativer langer Name für Montag. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt 80, einschließlich eines beendenden NULL-Zeichens.    |
| LOCALE \_ SDAYNAME2 | Nativer langer Name für Dienstag. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt 80, einschließlich eines beendenden NULL-Zeichens.   |
| LOCALE \_ SDAYNAME3 | Nativer langer Name für Mittwoch. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt 80, einschließlich eines beendenden NULL-Zeichens. |
| LOCALE \_ SDAYNAME4 | Nativer langer Name für Donnerstag. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt 80, einschließlich eines beendenden NULL-Zeichens.  |
| LOCALE \_ SDAYNAME5 | Nativer long-Name für Friday. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt 80, einschließlich eines beendenden NULL-Zeichens.    |
| LOCALE \_ SDAYNAME6 | Nativer langer Name für Samstag. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt 80, einschließlich eines beendenden NULL-Zeichens.  |
| LOCALE \_ SDAYNAME7 | Nativer langer Name für Sonntag. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt 80, einschließlich eines beendenden NULL-Zeichens.    |
