# <a name="locale_sdayname-constants"></a>LOCALE \_ sdayname- \* Konstanten

In diesem Thema werden die \_ von NLS verwendeten locale-sdayname- \* Konstanten definiert.



| Wert             | Bedeutung                                                                                                                                 |
|-------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ SDAYNAME1 | Der Native lange Name für Montag. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig ist, ist 80, einschließlich eines abschließenden NULL-Zeichens.    |
| LOCALE \_ SDAYNAME2 | Der Native lange Name für Dienstag. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig ist, ist 80, einschließlich eines abschließenden NULL-Zeichens.   |
| LOCALE \_ SDAYNAME3 | Der Native lange Name für Mittwoch. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig ist, ist 80, einschließlich eines abschließenden NULL-Zeichens. |
| LOCALE \_ SDAYNAME4 | Der Native lange Name für Donnerstag. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig ist, ist 80, einschließlich eines abschließenden NULL-Zeichens.  |
| LOCALE \_ SDAYNAME5 | Der Native lange Name für Freitag. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig ist, ist 80, einschließlich eines abschließenden NULL-Zeichens.    |
| LOCALE \_ SDAYNAME6 | Der Native lange Name für Samstag. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig ist, ist 80, einschließlich eines abschließenden NULL-Zeichens.  |
| LOCALE \_ SDAYNAME7 | Der Native lange Name für Sonntag. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig ist, ist 80, einschließlich eines abschließenden NULL-Zeichens.    |
