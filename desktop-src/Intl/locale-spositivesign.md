# <a name="locale_spositivesign"></a>Gebiets Schema \_ spositivesign

Lokalisierter Zeichen folgen Wert für das positive Zeichen für das Gebiets Schema. Beispielsweise wird **locale \_ spositivesign** als "+" für das Gebiets Schema "Englisch (USA)" betrachtet. Beachten Sie, dass das vorhanden sein von Daten für diese Zeichenfolge nicht bedeutet, dass beim Formatieren einer Zahl ein positives Vorzeichen verwendet wird. Wenn diese Daten leer/leer sind, wird davon ausgegangen, dass ein "+" verwendet wird. Die maximal zulässige Anzahl von Zeichen für diese Zeichenfolge ist fünf, einschließlich eines abschließenden NULL-Zeichens.
