# <a name="locale_use_cp_acp"></a>\_GEBIETSSCHEMA: VERWENDEN VON CP \_ \_ ACP

**Windows Me/98, Windows 2000:** Systemstandard Windows ANSI-Codepage (ACP) anstelle der gebietsschema-Codepage, die für die Zeichenfolgenübersetzung verwendet wird. Eine Liste mit ANSI und anderen Codepages finden Sie unter [Codepagebezeichner.](code-page-identifiers.md)
