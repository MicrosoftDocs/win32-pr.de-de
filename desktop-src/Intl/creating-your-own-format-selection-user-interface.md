# <a name="international-text-display"></a>Internationale Text Anzeige

Es gibt vier mögliche Optionen für die Anzeige von internationalem Text, der auch die Ausgabe komplexer Skripts umfasst:

-   Aufrufen von Windows-Text-APIs
-   Instanziieren von Windows-Standard Bearbeitungs Steuerelementen
-   Instanziieren von Rich-Edit-Steuerelementen
-   Aufrufen von unischreiber

Die Erläuterung und die Vorteile der einzelnen Optionen finden Sie unter [Optionen für die Anzeige von Text](https://msdn.microsoft.com/globalization/mt662335) . Sie sollten sich dann entscheiden, welche Option für Ihre Anwendung am besten geeignet ist, und zwar basierend auf der Komplexität der Anwendung und den zugehörigen Entwurfs Features.
