# <a name="hot-keys"></a>Abkürzungstasten

Hot Keys ermöglichen dem Benutzer, den Eingabemodus einer IME schnell zu ändern oder zu einer anderen IME zu wechseln. Obwohl Anwendungen dem Betriebssystem keine Hot Keys hinzufügen können, können sie mithilfe der [**ImmSimulateHotKey-Funktion**](/windows/desktop/api/Imm/nf-imm-immsimulatehotkey) die gleiche Aktion wie ein Hot-Schlüssel initiieren.

Die HexToUnicode-IME ermöglicht auch die Konvertierung zwischen Hexadezimal- und Unicode-Zeichen. Weitere Informationen finden Sie unter [HexToUnicode IME](hextounicode-ime.md).
