# <a name="hot-keys"></a>Abkürzungstasten

Mit Abkürzungs Schlüsseln kann der Benutzer schnell den Eingabemodus eines IME ändern oder zu einem anderen IME wechseln. Obwohl Anwendungen dem Betriebssystem keine Hotkeys hinzufügen können, können Sie mit der [**immsimulatehotkey**](/windows/desktop/api/Imm/nf-imm-immsimulatehotkey) -Funktion dieselbe Aktion wie ein Hot Key initiieren.

Das hexadeunicode-IME ermöglicht außerdem die Konvertierung zwischen hexadezimalen und Unicode-Zeichen. Weitere Informationen finden Sie unter [hexyunicode-IME](hextounicode-ime.md).
