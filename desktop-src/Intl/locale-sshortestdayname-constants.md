| LOCALE \_ SSHORTESTDAYNAME4 | **Windows Vista und höher:** Kurzer nativer Name des vierten Tags der Woche. Dieser Name ist oft kürzer als die [locale \_ SABBREVDAYNAME4](locale-sabbrev-constants.md)und ist für Kalender Titel nützlich.  |
| LOCALE \_ SSHORTESTDAYNAME5 | **Windows Vista und höher:** Kurzer nativer Name des fünften wochentags. Dieser Name ist oft kürzer als die [locale \_ SABBREVDAYNAME5](locale-sabbrev-constants.md)und ist für Kalender Titel nützlich.   |
