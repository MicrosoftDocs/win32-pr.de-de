# <a name="locale_sshortestdayname-constants"></a>LOCALE \_ SSHORTESTDAYNAME-Konstanten \*

In diesem Thema werden die \_ locale SSHORTESTDAYNAME-Konstanten definiert, \* die von NLS verwendet werden.



| Wert                     | Bedeutung                                                                                                                                                                                                       |
|---------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ SSHORTESTDAYNAME1 | **Windows Vista und höher:** Kurzer nativer Name des ersten Wochentags. Dieser Name ist häufig kürzer als [LOCALE \_ CABBREVDAYNAME1](locale-sabbrev-constants.md)und eignet sich für Kalendertitel.   |
| LOCALE \_ SSHORTESTDAYNAME2 | **Windows Vista und höher:** Kurzer nativer Name des zweiten Wochentags. Dieser Name ist häufig kürzer als [LOCALE \_ CABBREVDAYNAME2](locale-sabbrev-constants.md)und eignet sich für Kalendertitel.  |
| LOCALE \_ SSHORTESTDAYNAME3 | **Windows Vista und höher:** Kurzer nativer Name des dritten Wochentags. Dieser Name ist häufig kürzer als [ \_ LOCALEBREVDAYNAME3](locale-sabbrev-constants.md)und eignet sich für Kalendertitel.   |
| LOCALE \_ SSHORTESTDAYNAME4 | **Windows Vista und höher:** Kurzer nativer Name des vierten Tags der Woche. Dieser Name ist häufig kürzer als [LOCALE \_ CABBREVDAYNAME4](locale-sabbrev-constants.md)und eignet sich für Kalendertitel.  |
| LOCALE \_ SSHORTESTDAYNAME5 | **Windows Vista und höher:** Kurzer nativer Name des fünften Tags der Woche. Dieser Name ist häufig kürzer als [LOCALE \_ CABBREVDAYNAME5](locale-sabbrev-constants.md)und eignet sich für Kalendertitel.   |
| LOCALE \_ SSHORTESTDAYNAME6 | **Windows Vista und höher:** Kurzer nativer Name des sechsten Wochentags. Dieser Name ist häufig kürzer als [ \_ LOCALEBREVDAYNAME6](locale-sabbrev-constants.md)und eignet sich für Kalendertitel.   |
| LOCALE \_ SSHORTESTDAYNAME7 | **Windows Vista und höher:** Kurzer nativer Name des siebenten Wochentags. Dieser Name ist häufig kürzer als [LOCALE \_ CABBREVDAYNAME7](locale-sabbrev-constants.md)und eignet sich für Kalendertitel. |
