# <a name="locale_senglish-constants"></a>LOCALE \_ \* SENGLISH-Konstanten

In diesem Thema werden die LOCALE \_ SENGLISH-Konstanten \* definiert, die von NLS verwendet werden.



| Wert                        | Bedeutung                                                                                                                                                                                                                                                  |
|------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ SENGLISHCOUNTRYNAME  | **Windows 7 und höher:** Der englische Name des Landes bzw. der Region, z. B. Deutschland für Deutschland. Dieser Name ist immer auf Zeichen beschränkt, die der ASCII-Teilmenge mit 127 Zeichen zugeordnet werden können. <br/>                                      |
| LOCALE \_ SENGLISHDISPLAYNAME  | **Windows 7 und höher:** Anzeigename des Locale auf Englisch. Normalerweise besteht der Anzeigename aus der Sprache und dem Land/der Region, z. B. Deutsch (Deutschland) für Deutsch (Deutschland). <br/>                                           |
| LOCALE \_ SENGLISHLANGUAGENAME | **Windows 7 und höher:** Der englische Name der Sprache in Englisch, z. B. Deutsch für Deutsch, aus dem internationalen ISO-Standard 639. Dieser Name ist immer auf Zeichen beschränkt, die der ASCII-Teilmenge mit 127 Zeichen zugeordnet werden können. <br/> |
