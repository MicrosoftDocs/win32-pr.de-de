# <a name="locale_senglish-constants"></a>LOCALE- \_ senglish- \* Konstanten

In diesem Thema werden die \_ von NLS verwendeten locale-senglish- \* Konstanten definiert.



| Wert                        | Bedeutung                                                                                                                                                                                                                                                  |
|------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ senglishcountryname  | **Windows 7 und höher:** Der englische Name des Landes bzw. der Region, z. b. Deutschland für Deutschland. Dieser Name ist immer auf Zeichen beschränkt, die der ASCII-Zeichenfolge 127 zugeordnet werden können. <br/>                                      |
| LOCALE \_ senglishdisplayname  | **Windows 7 und höher:** Anzeige Name des Gebiets Schemas in englischer Sprache. Normalerweise besteht der Anzeige Name aus der Sprache und dem Land bzw. der Region, z. b. Deutsch (Deutschland) für Deutsch (Deutschland). <br/>                                           |
| LOCALE \_ senglishlanguagename | **Windows 7 und höher:** Englischer Name der Sprache in englischer Sprache, z. b. Deutsch für Deutsch, vom internationalen ISO-Standard 639. Dieser Name ist immer auf Zeichen beschränkt, die der ASCII-Zeichenfolge 127 zugeordnet werden können. <br/> |
