# <a name="international-fonts-and-text-display"></a>Internationale Schriftarten und Text Anzeige

Internationale Schriftarten und Textanzeige bietet Platt Form Unterstützung für Internationale Schriftarten, internationalen Text und eine fein Typografie auf der Windows-Plattform.

Internationale Schriftarten und Texte baut auf den grundlegenden Typografie-, Text-und erweiterten Lesetechnologien von Windows auf, um Schriftarten und Text für Welt weite Benutzer der Windows-Plattform bereitzustellen. Durch die Technologie werden die End-to-End-Anzeige Szenarios für Schriftart und Text erweitert.

Die folgenden Technologien sind enthalten:

-   [Internationale Schriftart Verwaltung](about-international-fonts-and-text.md). Bietet eine Einführung in die grundlegenden Funktionen internationaler Schriftarten.
-   [Verwenden von MS Shell Dlg und MS Shell Dlg 2](using-ms-shell-dlg-and-ms-shell-dlg-2.md). Übersicht über die Schriftarten von MS Shell Dlg und MS Shell Dlg 2 sowie die Schriftart Zuordnung von MS Shell Dlg und MS Shell Dlg 2 auf verschiedenen Windows-Versionen.
-   [Internationale Schriftart Enumeration und-Auswahl](using-international-fonts-and-text.md). Erläutert mögliche Möglichkeiten zum Auflisten internationaler Schriftarten, damit Benutzer die gewünschte Schriftart auswählen können.
-   [Internationale Text Anzeige](creating-your-own-format-selection-user-interface.md). Erläutert mögliche Möglichkeiten zum Anzeigen von internationalem Text.
-   [Unischreiber](uniscribe.md). Ermöglicht ein hohes Maß an Kontrolle für die fein Typografie und die Verarbeitung komplexer Skripts.
