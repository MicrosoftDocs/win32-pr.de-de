# <a name="international-fonts-and-text-display"></a>Internationale Schriftarten und Textanzeige

International Fonts and Text Display bietet Plattformunterstützung für internationale Schriftarten, internationalen Text und feinen Typografie auf Windows Plattform.

Internationale Schriftarten und Text bauen auf den grundlegenden Typografie-, Text- und erweiterten Lesetechnologien von Windows auf, um Schriftarten und Text für weltweiten Benutzer der Windows zu liefern. Die Technologie verbessert End-to-End-Schriftart- und Textanzeigeszenarien.

Die folgenden Technologien sind enthalten:

-   [Internationale Schriftverwaltung](about-international-fonts-and-text.md). Stellt die grundlegende Funktionalität von internationalen Schriftarten vor.
-   [Verwenden von MS Shell Dlg und MS Shell Dlg 2](using-ms-shell-dlg-and-ms-shell-dlg-2.md). Übersicht über MS Shell Dlg- und MS Shell Dlg 2-Schriftarten und Schriftartzuordnung von MS Shell Dlg und MS Shell Dlg 2 in verschiedenen Windows Versionen.
-   [Internationale Schriftartenenumeration und Auswahl](using-international-fonts-and-text.md). Erläutert mögliche Möglichkeiten zum Aufzählen von internationalen Schriftarten, damit Benutzer die schriftart auswählen können, die sie wünschen.
-   [Internationale Textanzeige](creating-your-own-format-selection-user-interface.md). Erläutert mögliche Möglichkeiten zum Anzeigen von internationalem Text.
-   [Uniscribe](uniscribe.md). Ermöglicht ein hohes Maß an Kontrolle für die feinen Typografie und die Verarbeitung komplexer Skripts.
