# <a name="locale_allow_neutral_names"></a>Gebiets \_ Schema \_ neutrale \_ Namen zulassen

**Windows 7 und höher:** Hiermit wird die Rückgabe von neutralen Namen oder LCIDs bei der Umstellung zwischen Gebiets Schema Namen und Gebiets Schema Bezeichnerzeichen zugelassen.
