# <a name="locale_ssort_constants"></a>LOCALE- \_ ssort- \_ Konstanten

In diesem Thema werden die \_ von NLS verwendeten locale-ssort- \* Konstanten definiert.



| Wert               | Bedeutung                                                                                                                                                                                                           |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Gebiets Schema \_ ssortlocale | **Windows 7 und höher:** Der Name des Gebiets Schemas, das für Sortier-oder Schreibverhalten verwendet werden soll.                                                                                                                                |
| LOCALE \_ ssortname   | **Windows Me/98, Windows 2000:** Der vollständig lokalisierte Name der Sortierung für den angegebenen Gebiets Schema Bezeichner, abhängig von der Sprache der Shell. Diese Konstante wird verwendet, um die Schreibweise und das Sortier Verhalten zu bestimmen. |
