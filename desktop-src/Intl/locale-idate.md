# <a name="locale_idate"></a>Gebiets Schema- \_ IDATE

Kurzformatierungsspezifizierungsspezifizierer für Datum. Der Spezifizierer muss einen der folgenden Werte aufweisen. Es sind keine vom Benutzer angegebenen Werte zulässig. Es wird empfohlen, dass Ihre Anwendung [locale \_ sshortdate](locale-sshortdate.md) anstelle von locale \_ IDATE verwendet.



| Wert | Bedeutung        |
|-------|----------------|
| 0     | Monat-Tag-Jahr |
| 1     | Tag-Monat-Jahr |
| 2     | Jahr-Monat-Tag |
