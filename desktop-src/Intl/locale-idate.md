# <a name="locale_idate"></a>LOCALE \_ IDATE

Der Formatbezeichner für kurze Datumsreihenfolge. Der Spezifizierer muss einer der folgenden Werte sein. Es sind keine benutzerdefinierten Werte zulässig. Es wird für Ihre Anwendung [bevorzugt, LOCALE \_ SSHORTDATE](locale-sshortdate.md) anstelle von LOCALE IDATE zu \_ verwenden.



| Wert | Bedeutung        |
|-------|----------------|
| 0     | Month-Day-Year |
| 1     | Tag-Monat-Jahr |
| 2     | Year-Month-Day |
