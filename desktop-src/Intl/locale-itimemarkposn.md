# <a name="locale_itimemarkposn"></a>LOCALE \_ ITIMEMARKPOSN

Spezifizierer, der angibt, ob die Zeitmarkerzeichenfolge (AM oder PM) der Zeitzeichenfolge vorangestellt ist oder dieser folgt. Der Registrierungswert ist aus Gründen der Kompatibilität mit früheren versionen von Windows in Asien iTimePrefix. Der Bezeichner muss einen der folgenden Werte aufweisen. Es sind keine benutzerdefinierten Werte zulässig. Es wird für Ihre Anwendung bevorzugt, die [LOCALE \_ STIMEFORMAT-Konstante](locale-stime-constants.md) anstelle von LOCALE \_ ITIMEMARKPOSN zu verwenden.



| Wert | Bedeutung        |
|-------|----------------|
| 0     | Verwenden Sie als Suffix. |
| 1     | Verwenden Sie als Präfix. |
