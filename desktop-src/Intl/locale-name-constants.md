# <a name="locale_name-constants"></a>LOCALE \_ \* NAME-Konstanten

In diesem Thema werden die LOCALE \_ \* NAME-Konstanten definiert, die von NLS verwendet werden.



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>Wert</th>
<th>Bedeutung</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td>LOCALE_NAME_INVARIANT</td>
<td>Der Name eines invarianten Locales, das stabile Locale- und Kalenderdaten bietet.</td>
</tr>
<tr class="even">
<td>LOCALE_NAME_MAX_LENGTH</td>
<td>Maximale Länge eines Locale-Namens. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt 85, einschließlich eines beendenden NULL-Zeichens.
<blockquote>
[!Note]<br />
Ihre Anwendung muss die Konstante für die maximale Länge des Locale-Namens verwenden, anstatt den Wert &quot; 85 hart zu &quot; codieren.
</blockquote>
<br/></td>
</tr>
<tr class="odd">
<td>LOCALE_NAME_SYSTEM_DEFAULT</td>
<td>Name des aktuellen Betriebssystem-Locales.</td>
</tr>
<tr class="even">
<td>LOCALE_NAME_USER_DEFAULT</td>
<td>Name des aktuellen Benutzer-Locales, der mit dem Einstellungssatz im Bereich "Regionale Optionen" und "Sprachoptionen" des Systemsteuerung. Dieses Locale kann sich vom -Locale für die aktuelle Benutzeroberflächensprache unterscheiden.</td>
</tr>
</tbody>
</table>
