# <a name="locale_name-constants"></a>LOCALE \_ \* NAME-Konstanten

In diesem Thema werden die LOCALE \_ \* NAME-Konstanten definiert, die von NLS verwendet werden.




| Wert | Bedeutung | 
|-------|---------|
| LOCALE_NAME_INVARIANT | Der Name eines invarianten Locales, das stabile Locale- und Kalenderdaten bietet. | 
| LOCALE_NAME_MAX_LENGTH | Maximale Länge eines Locale-Namens. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt 85, einschließlich eines beendenden NULL-Zeichens.<blockquote>[!Note]<br />Ihre Anwendung muss die Konstante für die maximale Länge des Locale-Namens verwenden, anstatt den Wert "85" hart zu codieren.</blockquote><br /> | 
| LOCALE_NAME_SYSTEM_DEFAULT | Name des aktuellen Betriebssystem-Locales. | 
| LOCALE_NAME_USER_DEFAULT | Name des aktuellen Benutzer-Locales, der mit dem Einstellungssatz im Bereich "Regionale Optionen" und "Sprachoptionen" des Systemsteuerung. Dieses Locale kann sich vom -Locale für die aktuelle Benutzeroberflächensprache unterscheiden. |
