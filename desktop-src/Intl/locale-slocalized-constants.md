# <a name="locale_slocalized-constants"></a>\_Slocalized- \* Konstanten

In diesem Thema werden die \_ \* von NLS verwendeten slocalized-Konstanten definiert.



| Wert                          | Bedeutung                                                                                                                                                                                                                                                                                                                                     |
|--------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Gebiets Schema \_ slocalizedcountryname  | **Windows 7 und höher:** Vollständig lokalisierter Name des Landes bzw. der Region, z. b. Deutschland für Deutschland. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig ist, ist 80, einschließlich eines abschließenden NULL-Zeichens. Da dieser Name auf der Lokalisierung des Produkts basiert, wird er für jede lokalisierte Version geändert. <br/>           |
| LOCALE \_ slocalizeddisplayname  | **Windows 7 und höher:** Vollständig lokalisierter Name des Gebiets Schemas für die Sprache der Benutzeroberfläche, z. b. Deutsch (Deutschland) für Deutsch (Deutschland). es gibt keine Beschränkung für die Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind. Da dieser Name auf der Lokalisierung des Produkts basiert, wird er für jede lokalisierte Version geändert. <br/> |
| Gebiets Schema \_ slocalizedlanguagename | **Windows Vista:** Vollständiger lokalisierter primär Name der Sprache der Benutzeroberfläche, die in einem lokalisierten anzeigen Amen enthalten ist, z. b. Deutsch, das Deutsch darstellt. Da dieser Name auf der Lokalisierung des Produkts basiert, wird er für jede lokalisierte Version geändert. <br/>                                                               |
