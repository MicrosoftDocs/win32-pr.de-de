# <a name="locale_ildate"></a>Gebiets Schema- \_ ildate

Long Date Format-Order specifier. Es wird empfohlen, dass Ihre Anwendung die [locale \_ slongdate](locale-slongdate.md) -Konstante anstelle von locale \_ ildate verwendet.



| Wert | Bedeutung        |
|-------|----------------|
| 0     | Monat-Tag-Jahr |
| 1     | Tag-Monat-Jahr |
| 2     | Jahr-Monat-Tag |
