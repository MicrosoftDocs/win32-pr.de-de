# <a name="about-unicode-and-character-sets"></a>Informationen zu Unicode und Zeichensätzen

Die zeichenbasierten Daten der Welt wurden sowohl mit [Unicode](unicode.md) als auch mit herkömmlichen Zeichensätzen entwickelt. Aus diesem Grund stellt Windows Zeichensatzfunktionen zur Verfügung, mit denen Anwendungen die zeichenbasierten Daten aus ihrem ursprünglichen Zeichensatz in Unicode oder einen anderen herkömmlichen Zeichensatz konvertieren können. Mithilfe dieser Zeichensatzfunktionen können Anwendungen auch zeichenbasierte Daten erstellen, die in ein beliebiges Betriebssystem übertragen und verwendet werden können, einschließlich derErsteller, die Unicode nicht unterstützen.

Weitere Informationen zu Unicode, die über den Rahmen dieser Übersicht hinausgehen, finden Sie unter [Der Unicode-Standard](https://www.unicode.org/standard/standard.html).
