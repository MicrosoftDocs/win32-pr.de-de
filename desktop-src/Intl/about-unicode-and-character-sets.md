# <a name="about-unicode-and-character-sets"></a>Informationen zu Unicode und Zeichensätzen

Die Zeichen basierten Daten der Welt wurden sowohl mit [Unicode](unicode.md) als auch mit herkömmlichen Zeichensätzen entwickelt. Aus diesem Grund stellt Windows Zeichensatz Funktionen bereit, mit deren Hilfe Anwendungen die Zeichen basierten Daten aus Ihrem ursprünglichen Zeichensatz in Unicode oder einen anderen herkömmlichen Zeichensatz konvertieren können. Mit diesen Zeichensatz Funktionen können Anwendungen auch zeichenbasierte Daten erstellen, die auf ein beliebiges Betriebssystem übertragen und dort verwendet werden können, einschließlich derjenigen, die Unicode nicht unterstützen.

Weitere Informationen zu Unicode außerhalb des Umfangs dieser Übersicht finden Sie [im Unicode-Standard](https://www.unicode.org/standard/standard.html).
