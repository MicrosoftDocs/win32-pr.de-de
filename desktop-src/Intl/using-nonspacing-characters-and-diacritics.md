# <a name="using-nonspacing-characters-and-diacritics"></a>Verwenden von Zeichen ohne zwischen Raum und Diakritik

Wenn Sie Zeilenumbruch oder anderweitig voneinander trennen, muss ihre Unicode-Anwendung keine Abstands Zeichen mit ihren Basiszeichen enthalten. Viele Skripts enthalten Zeichen, die mit anderen Zeichen in der Ausgabe visuell kombiniert werden, z. b. Diakritische Markierungen ("Gleit Ende Akzente").
