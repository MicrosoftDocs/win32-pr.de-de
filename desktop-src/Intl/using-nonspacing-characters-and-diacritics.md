# <a name="using-nonspacing-characters-and-diacritics"></a>Verwenden von Nicht-Paczeichen und diakritischen Zeichen

Wenn Zeilen bricht oder Text anderweitig getrennt wird, muss Ihre Unicode-Anwendung zeichen, die keine Leerzeichen enthalten, mit ihren Basiszeichen behalten. Viele Skripts enthalten Zeichen, die visuell mit anderen Zeichen in der Ausgabe kombiniert werden, z. B. diakritische Markierungen ("unverankerte Akzente").
