# <a name="locale_sname"></a>Name des Gebiets Schemas \_

**Windows Vista und höher:** Gebiets Schema Name, ein mehrstufiges Tag zur eindeutigen Identifizierung des Gebiets Schemas. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig ist, ist 85, einschließlich eines abschließenden NULL-Zeichens. Das-Tag basiert auf den sprachtagging-Konventionen von RFC 4646. Das zu verwendende Muster wird in Gebiets Schema [Namen](locale-names.md)beschrieben.
