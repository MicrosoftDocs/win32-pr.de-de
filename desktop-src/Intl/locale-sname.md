# <a name="locale_sname"></a>LOCALE \_ SNAME

**Windows Vista und höher:** Gebietsschemaname, ein mehrteiliges Tag zur eindeutigen Identifizierung des Gebietsschemas. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt 85, einschließlich eines abschließenden NULL-Zeichens. Das Tag basiert auf den Sprachtaggingkonventionen von RFC 4646. Das zu verwendende Muster wird unter [Gebietsschemanamen](locale-names.md)beschrieben.
