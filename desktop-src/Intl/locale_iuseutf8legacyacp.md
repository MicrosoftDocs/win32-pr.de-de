# <a name="locale_iuseutf8legacyacp"></a>LOCALE \_ IUSEUTF8LEGACYACP

**Ab der nächsten Version von Windows:** Es wird empfohlen, dass Anwendungen stattdessen Unicode verwenden. Auf Computern, die der Benutzer für die Verwendung einer UTF-8-Codepage konfiguriert hat, ist LOCALE_IUSEUTF8LEGACYACP die ältere Windows-Codepage, die auf einem Nicht-UTF-8-Legacycomputer Windows wurde.
