# <a name="locale_ipapersize"></a>LOCALE \_ IPAPERSIZE

**Windows 2000:** Standarddokumentgröße, die dem -Locale zugeordnet ist. Die Größe hat in der Regel einen der folgenden Werte, obwohl sie auf jede der definierten Papiergrößen festgelegt werden kann, die vom Spooler verstanden werden.



| Wert | Bedeutung   |
|-------|-----------|
| 1     | US-Buchstabe |
| 5     | US-rechtliches Recht  |
| 8     | A3        |
| 9     | A4        |
