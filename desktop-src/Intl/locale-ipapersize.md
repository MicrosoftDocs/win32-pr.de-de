# <a name="locale_ipapersize"></a>LOCALE \_ ipapersize

**Windows 2000:** Standardpapiergröße, die dem Gebiets Schema zugeordnet ist. Die Größe weist in der Regel einen der folgenden Werte auf, obwohl Sie auf alle definierten Papiergrößen festgelegt werden kann, die vom Spooler verstanden werden.



| Wert | Bedeutung   |
|-------|-----------|
| 1     | US-Buchstabe |
| 5     | US legal  |
| 8     | A3        |
| 9     | A4        |
