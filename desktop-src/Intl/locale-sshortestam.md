# <a name="locale_sshortestam"></a>LOCALE \_ sshortestam

Zeichenfolge für den kürzesten möglichen am-Indikator für ein Gebiets Schema. Für Englisch, das möglicherweise so kurz ist wie "a"; andere Sprachen können jedoch mehr Code Punkte enthalten.
