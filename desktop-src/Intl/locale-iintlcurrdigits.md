# <a name="locale_iintlcurrdigits"></a>LOCALE \_ IINTLCURRDIGITS

\[LOCALE \_ IINTLCURRDIGITS wird nicht unterstützt und kann in Zukunft geändert oder nicht mehr verfügbar sein. Verwenden Sie stattdessen [LOCALE \_ ICURRDIGITS](locale-icurrdigits.md).\]

Anzahl der Dezimalstellen für das internationale Währungsformat. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt drei, einschließlich eines abschließenden NULL-Zeichens. Beispielsweise ist die Anzahl der Dezimalstellen für das internationale Währungsformat von US-Dollar 2, wie in 345,25 USD.
