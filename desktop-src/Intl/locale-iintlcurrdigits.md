# <a name="locale_iintlcurrdigits"></a>Gebiets Schema \_ iintlcurrdigits

\[Gebiets Schema \_ -iintlcurrdigits wird nicht unterstützt und kann in Zukunft geändert oder nicht verfügbar sein. Verwenden Sie stattdessen [locale \_ icurrdigits](locale-icurrdigits.md).\]

Anzahl von Bruch Ziffern für das internationale Währungs Format. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig ist, ist drei, einschließlich eines abschließenden NULL-Zeichens. Beispielsweise ist die Anzahl der Bruch Ziffern für das internationale Währungs Format von US-Dollar 2, wie in $345,25.
