# <a name="locale_itlzero"></a>Gebiets Schema- \_ itlzero

Spezifizierer für führende Nullen in Zeit Feldern. Es wird empfohlen, dass Ihre Anwendung die [locale \_ stimeformat](locale-stime-constants.md) -Konstante anstelle von locale \_ itlzero verwendet.



| Wert | Bedeutung                    |
|-------|----------------------------|
| 0     | Keine führenden Nullen für Stunden |
| 1     | Führende Nullen für Stunden    |
