# <a name="locale_itlzero"></a>LOCALE \_ ITLZERO

Spezifizierer für führende Nullen in Zeitfeldern. Es wird bevorzugt, dass Ihre Anwendung die [LOCALE \_ STIMEFORMAT-Konstante](locale-stime-constants.md) anstelle von LOCALE \_ ITLZERO verwendet.



| Wert | Bedeutung                    |
|-------|----------------------------|
| 0     | Keine führenden Nullen für Stunden |
| 1     | Führende Nullen für Stunden    |
