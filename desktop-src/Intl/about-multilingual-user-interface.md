# <a name="about-multilingual-user-interface"></a>Informationen mehrsprachige Benutzeroberfläche

mehrsprachige Benutzeroberfläche (USERNAME) ist eine Technologie, die Benutzern eine lokalisierte Benutzeroberfläche für globalisierte Anwendungen und die Ressourcenverwaltung der Benutzeroberflächensprache im Windows Betriebssystem bereitstellt. Unterstützung für das Hinzufügen von FUNKTIONEN zu globalisierten Anwendungen, die auf Windows Vista und höher ausgeführt werden sollen, sowie vielen Betriebssystemen vor Windows Vista. Die LOCALIZATION-Lokalisierungs- und Ressourcenverwaltungsmodelle verbessern Entwicklung, Tests und Unterstützung für weltweit einsatzbereite Software.

Zu den Features der TECHNOLOGY-Technologie gehören:

-   Vereinfachte Sprachverwaltung für die Softwarelokalisierung. MIT DER OPTION KÖNNEN die Spracheinstellungen für das Betriebssystem gemäß den Benutzereinstellungen in eine unterstützte Sprache geändert werden.
-   Eine innovative Ressourcentechnologie, die auf der Aufteilung der Binärdatei des Anwendungscodes aus Sprachressourcendateien basiert. Sie können Ressourcen für eine zusätzliche Sprache hinzufügen, ohne ihre Anwendung erneut kompilieren oder verknüpfen zu müssen. Zusätzliche lokalisierte Ressourcen werden zu optionalen Add-Ins.
-   Vollständige Anwendungsprogrammierschnittstelle (APPLICATION PROGRAMMING Interface, API). Windows Vista und höher machen dieAPI-API FÜR SIE verfügbar, die Sie beim Hinzufügen von MSI-Funktionen zu globalisierten Anwendungen verwenden können. Sprachverwaltungsfunktionen ermöglichen Es Ihren Anwendungen, eine Vielzahl von Sprachen der Benutzeroberfläche zu unterstützen. Die API enthält auch Funktionen zum Laden von Ressourcen, mit denen das Ressourcenladeprogramm Ressourcen auf Windows Vista und höher sowie vor Windows Vista-Betriebssystemen laden kann.
-   Eine Reihe von Sprachpaket-Verwaltungstools, die Flexibilität bei der internationalen Imageverwaltung für die Bereitstellung bieten. Diese Tools sind nicht für Entwickler ausgelegt und werden daher nicht im SDK behandelt. Weitere Informationen finden Sie im Windows Automated Installation Kit, das auf Technet verfügbar ist.

Der sichtbarste Vorteil von USERNAME besteht darin, dass mehrere Benutzer dieselbe Arbeitsstation gemeinsam nutzen und die Benutzeroberfläche in verschiedenen Sprachen anzeigen können. Unternehmen und OEMs profitieren von der Möglichkeit, mehrsprachige Images mit einer einzigen Installation herauszuführen, zu unterstützen und zu warten. Aber vielleicht ist der Hauptvorteil von CSV die Effizienz, die bei der Entwicklung, Erstellung und Wartung Ihrer Anwendung erzielt wird. Sie können eine Kernfunktionalitätsbinärdatei liefern, die unabhängig von der Benutzeroberflächensprache für alle Plattformen gilt, wodurch der Entwicklungs- und Testaufwand erheblich reduziert wird. Wenn Sie ein Update oder ein Service Pack ausstellen müssen, gilt es ohne zusätzlichen Engineeringaufwand für alle unterstützten Sprachen. Spätere Unterstützung für zusätzliche Sprachen wird zu einem Lokalisierungsprojekt anstelle eines vollständigen Softwareentwicklungsprojekts.
