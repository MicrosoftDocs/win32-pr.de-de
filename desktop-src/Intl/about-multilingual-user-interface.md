# <a name="about-multilingual-user-interface"></a>Informationen über mehrsprachige Benutzeroberfläche

Mi (mehrsprachige Benutzeroberfläche) ist eine Technologie, die Benutzern eine lokalisierte Benutzeroberfläche für globalisierte Anwendungen und die Sprachressourcen Verwaltung von Benutzeroberflächen im Windows-Betriebssystem bereitstellt. Es wird Unterstützung für das Hinzufügen von MUI-Funktionen zu globalisierten Anwendungen bereitgestellt, die unter Windows Vista und höher sowie in vielen Betriebssystemen vor Windows Vista ausgeführt werden. Die MUI-Lokalisierungs-und-Ressourcen Verwaltungsmodelle verbessern die Entwicklung, das Testen und die Unterstützung für weltweit einsetzbaren Software.

Zu den Features der MUI-Technologie zählen:

-   Vereinfachte sprach Verwaltung für die Softwarelokalisierung. MUI ermöglicht, dass die Spracheinstellungen für das Betriebssystem gemäß den Benutzereinstellungen in eine unterstützte Sprache geändert werden.
-   Eine innovative Ressourcen Technologie, die auf dem Aufteilen der Anwendungscode Binärdatei von Sprachressourcen Dateien basiert. Sie können Ressourcen für eine zusätzliche Sprache hinzufügen, ohne dass die Anwendung neu kompiliert oder neu verknüpft werden muss. Weitere lokalisierte Ressourcen werden zu optionalen Add-Ins.
-   Vervollständigen Sie die API (Application Programming Interface). Windows Vista und höher machen die MUI-API verfügbar, die Sie beim Hinzufügen von MUI-Funktionen zu globalisierten Anwendungen verwenden können. Sprach Verwaltungsfunktionen ermöglichen es Ihren Anwendungen, eine Vielzahl von Benutzeroberflächen Sprachen zu unterstützen. Die API umfasst auch Funktionen zum Laden von Ressourcen, die dem Ressourcen Lader das Laden von Ressourcen unter Windows Vista und höher sowie vor Windows Vista-Betriebssystemen ermöglichen.
-   Eine Reihe von Sprachpaket-Verwaltungs Tools, die Flexibilität bei der internationalen Bereitstellungs Abbild Verwaltung bieten. Diese Tools sind nicht für Entwickler bestimmt und werden daher nicht im SDK behandelt. Weitere Informationen finden Sie im Windows Automated Installation Kit, das auf TechNet verfügbar ist.

Der offensichtlichste Vorteil von MUI besteht darin, dass mehrere Benutzer dieselbe Arbeitsstation freigeben und die Benutzeroberfläche in verschiedenen Sprachen anzeigen können. Unternehmen und OEMs profitieren von der Funktion, mit der Sie mehrsprachige Images durch eine einzelne Installation einführen, unterstützen und warten können. Der größte Vorteil von MUI liegt jedoch in der Effizienz, die bei der Entwicklung, Erstellung und Wartung Ihrer Anwendung erzielt wurde. Sie können eine binäre Binär Funktions Binärdatei liefern, die unabhängig von der Benutzeroberflächen Sprache auf alle Plattformen anwendbar ist, was die Entwicklungs-und Test Bemühungen erheblich reduziert. Wenn Sie ein Update oder ein Service Pack ausgeben müssen, gilt es für alle unterstützten Sprachen ohne zusätzlichen Engineering-Aufwand. Die spätere Unterstützung für weitere Sprachen wird zu einem Lokalisierungsprojekt anstatt zu einem vollständigen Softwareentwicklungsprojekt.
