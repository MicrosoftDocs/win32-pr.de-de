# <a name="locale_neutraldata"></a>LOCALE \_ NEUTRALDATA

**Windows 7 und höher:** Neutrale Gebietsschemadaten, d. h. Daten, die nur durch sprachedefiniert sind. Für Länder-/Regionsdaten wird die Standardeinstellung verwendet.
