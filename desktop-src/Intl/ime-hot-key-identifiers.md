# <a name="ime-hot-key-identifiers"></a>IME-Hot-Key-Identifier

Diese Bezeichner werden mit der Funktion " [**immsimulatehotkey**](/windows/desktop/api/Imm/nf-imm-immsimulatehotkey) " verwendet.



| Bezeichner                       | Bedeutung                                                                                                                                                                                                 |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| IME-Schalter " \_ chotkey \_ IME \_ nonime" \_ | (Vereinfachtes Chinesisch) Wechseln Sie zwischen dem IME-und dem nicht-IME-Vorgang, wenn die Sprache Chinesisch (vereinfacht) ist.                                                                                                  |
| Taste für die IME- \_ chotkey- \_ Form \_ Umschalten       | (Vereinfachtes Chinesisch) Schalten Sie den Shape-Konvertierungsmodus von IME um.                                                                                                                                           |
| IME- \_ chotkey- \_ Symbol \_ Umschalten      | (Vereinfachtes Chinesisch) Schalten Sie den Modus für die Symbol Konvertierung von IME um. Der Symbol Modus gibt an, dass der Benutzer chinesische Interpunktions Zeichen und Symbole eingeben kann, indem er dem Satzzeichen und Symbolen auf der Tastatur zuordnet. |
| IME \_ ithotkey \_ reconvertstring    | Neukonvertierung von **Windows Me/98, Windows 2000, Windows XP:** (traditionelles Chinesisch).                                                                                                                |
| IME- \_ jhotkey \_ Schließen \_ geöffnet         | Japanischen Alternativ öffnen und schließen Sie den IME.                                                                                                                                                          |
| IME \_ khotkey \_ Englisch             | ERS Wechseln Sie zu Englisch.                                                                                                                                                                             |
| Taste für IME- \_ khotkey- \_ Form \_ Umschalten       | ERS Schalten Sie den Shape-Konvertierungsmodus von IME um.                                                                                                                                                       |
| IME \_ khotkey \_ hanjaconvert        | ERS Wechseln Sie zur Hanja-Konvertierung.                                                                                                                                                                    |
| IME \_ thotkey \_ IME \_ nonime- \_ Umschalten | (Traditionelles Chinesisch) Wechseln Sie zwischen dem IME-und dem nicht-IME-Vorgang, wenn die Sprache Chinesisch (traditionell) ist.                                                                                                |
| IME- \_ thotkey- \_ Shape \_ Umschalten       | (Traditionelles Chinesisch) Schalten Sie den Shape-Konvertierungsmodus von IME um.                                                                                                                                          |
| IME- \_ thotkey- \_ Symbol \_ Umschalten      | (Traditionelles Chinesisch) Schalten Sie den Modus für die Symbol Konvertierung von IME um.                                                                                                                                         |
