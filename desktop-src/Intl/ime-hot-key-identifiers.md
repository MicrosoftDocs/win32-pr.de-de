# <a name="ime-hot-key-identifiers"></a>IME Hot Key-Bezeichner

Diese Bezeichner werden mit der [**ImmSimulateHotKey-Funktion**](/windows/desktop/api/Imm/nf-imm-immsimulatehotkey) verwendet.



| Bezeichner                       | Bedeutung                                                                                                                                                                                                 |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| IME \_ CHOTKEY \_ IME \_ NONIME \_ TOGGLE | (Vereinfachtes Chinesisch) Umschalten zwischen IME- und Nicht-IME-Vorgang, wenn die Sprache Chinesisch vereinfacht ist.                                                                                                  |
| IME \_ CHOTKEY \_ SHAPE \_ TOGGLE       | (Vereinfachtes Chinesisch) Umschalten des Formkonvertierungsmodus von IME.                                                                                                                                           |
| IME \_ \_ CHOTKEY-SYMBOL \_ UMSCHALTTASTE      | (Vereinfachtes Chinesisch) Umschalten des Symbolkonvertierungsmodus von IME. Der Symbolmodus gibt an, dass der Benutzer chinesische Interpunktion und Symbole eingeben kann, indem er die Interpunktion und Symbole auf der Tastatur zuordnen kann. |
| IME \_ ITHOTKEY \_ RECONVERTSTRING    | **Windows Me/98, Windows 2000, Windows XP:** (Traditionelles Chinesisch) Trigger reconversion.                                                                                                                |
| IME \_ JHOTKEY \_ CLOSE \_ OPEN         | (Japanisch) Öffnen und schließen Sie alternativ den IME.                                                                                                                                                          |
| \_IME-ZEICHENSCHLÜSSEL \_ ENGLISCH             | (Koreanisch) Wechseln Sie zu Englisch.                                                                                                                                                                             |
| UMSCHALTTASTE \_ FÜR IME-ZEICHENSCHLÜSSEL-FORM \_ \_       | (Koreanisch) Umschalten des Formkonvertierungsmodus von IME.                                                                                                                                                       |
| IME \_ OHNE SCHLÜSSEL \_ HANJACONVERT        | (Koreanisch) Wechseln Sie zur Konvertierung von Hanja.                                                                                                                                                                    |
| IME \_ THOTKEY \_ IME \_ NONIME \_ TOGGLE | (Chinesisch (traditionell) Umschalten zwischen IME- und Nicht-IME-Vorgang, wenn die Sprache Chinesisch (traditionell) ist.                                                                                                |
| IME \_ THOTKEY \_ SHAPE \_ TOGGLE       | (Chinesisch (traditionell) Umschalten des Formkonvertierungsmodus von IME.                                                                                                                                          |
| \_IME-THOTKEY-SYMBOL \_ \_ UMSCHALTTASTE      | (Chinesisch (traditionell) Umschalten des Symbolkonvertierungsmodus von IME.                                                                                                                                         |
