# <a name="locale_ireadinglayout"></a>Gebiets Schema \_ ileslayout

**Windows 7 und höher:** Das Lese Layout für Text. Mögliche Werte sind in der folgenden Tabelle definiert.



| Wert | Bedeutung                                                                                                                                                             |
|-------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0     | Lesen von links nach rechts, wie für das Gebiets Schema Englisch (USA).                                                                                                 |
| 1     | Lesen von rechts nach links, wie für arabische Gebiets Schemata.                                                                                                                     |
| 2     | Lesen Sie entweder vertikal von oben nach unten mit Spalten, die von rechts nach links verlaufen, oder lesen Sie horizontale Zeilen von links nach rechts, wie für das japanische Gebiets Schema (Japan). |
| 3     | Lesen Sie vertikal von oben nach unten mit den Spalten von links nach rechts, wie für das Mongolisch (Mongolisch) Gebiets Schema.                                                  |
