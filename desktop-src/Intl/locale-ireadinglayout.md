# <a name="locale_ireadinglayout"></a>LOCALE \_ IREADINGLAYOUT

**Windows 7 und höher:** Das Leselayout für Text. Mögliche Werte sind in der folgenden Tabelle definiert.



| Wert | Bedeutung                                                                                                                                                             |
|-------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0     | Lesen Sie von links nach rechts, wie für das Gebietsschema Englisch (USA).                                                                                                 |
| 1     | Lesen von rechts nach links, wie bei arabischen Gebietsschemas.                                                                                                                     |
| 2     | Lesen Sie entweder vertikal von oben nach unten mit Spalten von rechts nach links, oder lesen Sie horizontale Zeilen von links nach rechts ein, wie beim japanischen Gebietsschema (Japan). |
| 3     | Lesen Sie vertikal von oben nach unten mit Spalten, die von links nach rechts verlaufen, wie für das Gebietsschema "Wird".                                                  |
