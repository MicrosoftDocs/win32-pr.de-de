# <a name="globalization-services"></a>Globalisierungsdienste

Globalization Services bietet grundlegende Globalisierungsunterstützung für Sprachen und Regionen, um die Erstellung von globalisierten Anwendungen zu ermöglichen, die für eine vielzahl von internationalen Kunden gelten. Diese Technologie stellt die Globalisierungsinfrastruktur zur Verfügung, um eine Windows Benutzererfahrung zu unterstützen, die konsistent und linguistisch und kulturell angemessen ist. Globalisierungsdienste umfassen die folgenden Technologien:

-   [Erweiterte linguistische Dienste](extended-linguistic-services.md). Bietet eine Reihe textbasierter Dienste für Windows-Entwickler (einschließlich Spracherkennung, Transliteration und anderer linguistischer Kernfunktionen), die Plattform und Schnittstellen für Anwendungen, um Dienste einfach zu nutzen, sowie Schnittstellen, mit denen Entwickler eigene Dienste hinzufügen können.
-   [International Components for Unicode (ICU)](international-components-for-unicode--icu-.md). Bietet einen ausgereiften, weit verbreiteten Satz von Open-Source-Globalisierungs-APIs. ICU nutzt das umfangreiche Common Locale Data Repository (CLDR) von Unicode als Datenbibliothek und bietet Globalisierungsunterstützung für Softwareanwendungen. ICU ist weit verbreitet portabel und bietet Anwendungen auf allen Plattformen die gleichen Ergebnisse.
-   [Unterstützung der Landessprache](national-language-support.md). Stellt internationale Daten und APIs zur Unterstützung von Datums-, Uhrzeit- und Kalenderformatierung, Tastaturlayouts, Sortierverhalten, Aspekten der Textermöglichung und anderen grundlegenden Informationen zum Locale zur Verfügung.
-   [Rechtschreibprüfungs-API](spell-checker-api.md). Stellt eine API bereit, mit der Entwickler die Rechtschreibprüfung nutzen können, um Text zu überprüfen, Vorschläge zu erhalten und Einstellungen und Benutzerwörterbücher zu verwalten. Entwickler können auch Rechtschreibprüfungstools erstellen, die als Anbieter in das Rechtschreibprüfungsframework integriert sind.
