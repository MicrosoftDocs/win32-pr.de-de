# <a name="globalization-services"></a>Globalisierungs Dienste

Globalisierungs Dienste bieten Unterstützung der Kern Globalisierung für Sprachen und Regionen, um die Erstellung globalisierter Anwendungen zu ermöglichen, die für eine Vielzahl internationaler Kunden gelten. Diese Technologie stellt die Globalisierungs Infrastruktur bereit, um eine konsistente und kulturelle Windows-Benutzerfunktion zu unterstützen. Globalisierungs Dienste beinhalten die folgenden Technologien:

-   [Erweiterte linguistische Dienste](extended-linguistic-services.md). Bietet eine Reihe von textbasierten Diensten für Windows-Entwickler (einschließlich Spracherkennung, Transliterierung und anderer linguistischer Grundfunktionen) sowie die Plattform und Schnittstellen, mit denen Anwendungen problemlos Dienste nutzen können, sowie Schnittstellen, die es Entwicklern ermöglichen, eigene Dienste hinzuzufügen.
-   [Internationale Komponenten für Unicode (ICU)](international-components-for-unicode--icu-.md). Bietet einen ausgereiften, häufig verwendeten Satz von Open Source-Globalisierungs-APIs. ICU nutzt das riesige Common Locale Data Repository (CLDR) von Unicode als Datenbibliothek und bietet Globalisierungs Unterstützung für Softwareanwendungen. ICU ist umfassend portierbar und bietet Anwendungen die gleichen Ergebnisse auf allen Plattformen.
-   [Unterstützung für nationale Sprache](national-language-support.md). Bietet internationale Daten und APIs zur Unterstützung von Datums-, Uhrzeit-und Kalender Formatierung, Tastaturlayouts, Sortier Verhalten, Aspekten der Text Aktivierung und anderen grundlegenden Gebiets Schema Informationen.
-   [Rechtschreibprüfungs-API](spell-checker-api.md). Bietet Entwicklern eine API, mit der Sie Rechtschreibprüfung nutzen können, um Text zu überprüfen, Vorschläge zu erhalten und Einstellungen und Benutzer Wörterbücher zu verwalten. Entwickler können auch Rechtschreibprüfungen erstellen, die als Anbieter in das Framework für die Rechtschreibprüfung integriert sind.
