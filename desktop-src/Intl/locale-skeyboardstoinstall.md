# <a name="locale_skeyboardstoinstall"></a>LOCALE \_ skeyboardstoinstall

**Windows Vista und höher:** Eine durch Semikolons getrennte Liste von Tastaturen, die möglicherweise für das Gebiets Schema installiert und intern von Windows verwendet werden soll. Es gibt keine Beschränkung für die Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind. Um den Namen des aktiven Eingabe Gebiets Schema Bezeichners (früher als Tastaturlayout bezeichnet) abzurufen, kann die Anwendung die [**getkeyboardlayoutname**](/windows/win32/api/winuser/nf-winuser-getkeyboardlayoutnamea) -Funktion aufrufen.
