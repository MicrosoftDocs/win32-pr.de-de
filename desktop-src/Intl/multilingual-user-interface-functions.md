# <a name="multilingual-user-interface-functions"></a>Mehrsprachige Benutzeroberflächen Funktionen

Die folgenden Funktionen werden mit der mehrsprachigen Benutzeroberfläche (MUI) verwendet.



| Funktion                                                                 | BESCHREIBUNG                                                                                                             |
|--------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**Enumeranguages**](/windows/desktop/api/Winnls/nf-winnls-enumuilanguagesa)                               | Listet die Sprachen der Benutzeroberfläche auf, die auf dem Betriebssystem verfügbar sind.                                     |
| [**"Umumuilanguagesproc"**](/windows/win32/api/winnls/nc-winnls-uilanguage_enumproca)                       | Eine Anwendungs definierte Funktion, die mit der [**enumuilanguages**](/windows/desktop/api/Winnls/nf-winnls-enumuilanguagesa) -Funktion verwendet wird.                      |
| [**Freihand**](/windows/desktop/api/Muiload/nf-muiload-freemuilibrary)                                 | Dekremente den Verweis Zähler eines von [ **loadmuilibrary** geladenen Ressourcen Moduls.](/windows/desktop/api/Muiload/nf-muiload-loadmuilibrarya)                  |
| [**Getfilemuiinfo**](/windows/desktop/api/Winnls/nf-winnls-getfilemuiinfo)                                 | Ruft ressourcenbezogene Informationen zu einer Datei ab.                                                                    |
| [**Getfilemuipath**](/windows/desktop/api/Winnls/nf-winnls-getfilemuipath)                                 | Ruft den Pfad zu einer sprachspezifischen Ressourcen Datei ab, die der angegebenen LN-Datei zugeordnet ist.                           |
| [**Getprocesliniferreduilanguages**](/windows/desktop/api/Winnls/nf-winnls-getprocesspreferreduilanguages) | Ruft die bevorzugten Benutzeroberflächen Sprachen des Prozesses ab.                                                                           |
| [**GetSystemDefaultUILanguage**](/windows/desktop/api/Winnls/nf-winnls-getsystemdefaultuilanguage)         | Ruft den sprach Bezeichner für die standardmäßige Benutzeroberflächen Sprache des Systems ab, die unter Windows Vista als "Installationssprache" bezeichnet wird. |
| [**GetSystemPreferredUILanguages**](/windows/desktop/api/Winnls/nf-winnls-getsystempreferreduilanguages)   | Ruft die bevorzugten Benutzeroberflächen Sprachen des Systems ab.                                                                            |
| [**Getthreadpreferreduilanguages**](/windows/desktop/api/Winnls/nf-winnls-getthreadpreferreduilanguages)   | Ruft die für den aktuellen Thread bevorzugten Benutzeroberflächen Sprachen des Threads ab.                                                     |
| [**Getthreaduilanguage**](/windows/desktop/api/Winnls/nf-winnls-getthreaduilanguage)                       | Gibt den sprach Bezeichner der ersten Benutzeroberflächen Sprache für den aktuellen Thread zurück.                            |
| [**Getuilanguagefallbacklist**](/windows/desktop/api/Muiload/nf-muiload-getuilanguagefallbacklist)           | Ruft eine Fall Back Liste von Benutzeroberflächen Sprachen ab, die als Sprachnamen dargestellt werden.                                         |
| [**Getuilanguageinfo**](/windows/desktop/api/Winnls/nf-winnls-getuilanguageinfo)                           | Ruft eine Vielzahl von Informationen über eine Benutzeroberflächen Sprache ab.                                                     |
| [**GetUserDefaultUILanguage**](/windows/desktop/api/Winnls/nf-winnls-getuserdefaultuilanguage)             | Gibt den sprach Bezeichner für die Benutzeroberflächen Sprache des aktuellen Benutzers zurück.                                          |
| [**Getuserpreferreduilanguages**](/windows/desktop/api/Winnls/nf-winnls-getuserpreferreduilanguages)       | Ruft die bevorzugten Benutzeroberflächen Sprachen des Benutzers ab.                                                                              |
| [**Loadmuilibrary**](/windows/desktop/api/Muiload/nf-muiload-loadmuilibrarya)                                 | Gibt ein Handle für die sprachspezifischen Ressourcen zurück, die einer bestimmten Sprach neutralen Datei (LN) zugeordnet sind.            |
| [**Setprocesliniferreduilanguages**](/windows/desktop/api/Winnls/nf-winnls-setprocesspreferreduilanguages) | Legt die Prozess bevorzugten Benutzeroberflächen Sprachen für den Anwendungsprozess fest.                                                    |
| [**Setthreadpreferreduilanguages**](/windows/desktop/api/Winnls/nf-winnls-setthreadpreferreduilanguages)   | Legt die bevorzugten Benutzeroberflächen Sprachen des Threads fest.                                                                                 |
| [**Setthreaduilanguage**](/windows/desktop/api/Winnls/nf-winnls-setthreaduilanguage)                       | Legt die bevorzugte Benutzeroberflächen Sprache für den aktuellen Thread fest.                                                      |
