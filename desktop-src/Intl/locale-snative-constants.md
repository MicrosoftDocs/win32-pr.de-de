# <a name="locale_snative-constants"></a>\_LOCALE-SNATIVE-Konstanten \*

In diesem Thema werden die \_ LOCALE-SNATIVE-Konstanten definiert, die \* von NLS zur Darstellung von Namen nativer Sprachen verwendet werden.



| Wert                       | Bedeutung                                                                                                                                                                                                                                                            |
|-----------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ SNATIVECOUNTRYNAME  | **Windows 7 und höher:** Nativer Name des Landes bzw. der Region, z. B. "Soll für Spanien". Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, ist 80, einschließlich eines abschließenden NULL-Zeichens.                                                                 |
| LOCALE \_ SNATIVECTRYNAME     | Für Windows 7 und höher veraltet. Nativer Name des Landes/der Region. Siehe LOCALE \_ SNATIVECOUNTRYNAME.                                                                                                                                                             |
| LOCALE \_ SNATIVECURRNAME     | **Windows Me/98, Windows 2000:** Der native Name der Währung, die dem Gebietsschema in der nativen Sprache des Gebietsschemas zugeordnet ist. Es gibt keine Beschränkung für die Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind.                                                          |
| LOCALE \_ SNATIVEDIGITS       | Native Entsprechungen von ASCII 0 bis 9. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, ist elf, einschließlich eines abschließenden NULL-Zeichens. Arabisch verwendet beispielsweise "٠١٢٣٤٥ ٦٧٨٩". Siehe auch [LOCALE \_ IDIGITSUBS CYBERSECURITY](locale-idigitsubstitution.md). |
| LOCALE \_ SNATIVEDISPLAYNAME  | **Windows 7 und höher:** Anzeigename des Gebietsschemas in seiner nativen Sprache, z. B. Deutsch (Deutschland) für das Gebietsschema Deutsch (Deutschland). <br/>                                                                                                        |
| LOCALE \_ SNATIVELANGNAME     | Für Windows 7 und höher veraltet. Nativer Name der Sprache. Siehe LOCALE \_ SNATIVELANGUAGENAME.                                                                                                                                                                  |
| LOCALE \_ SNATIVELANGUAGENAME | **Windows 7 und höher:** Der native Name der Sprache, z. B. Հայերեն für "Wird(1)". Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, ist 80, einschließlich eines abschließenden NULL-Zeichens.                                                         |
