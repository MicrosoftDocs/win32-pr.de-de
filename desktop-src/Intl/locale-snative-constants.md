# <a name="locale_snative-constants"></a>Gebiets Schema- \_ snative \* Konstanten

In diesem Thema werden die \_ von NLS verwendeten Gebiets Schema-snativen \* Konstanten zum Darstellen von systemeigenen Sprachen Namen definiert.



| Wert                       | Bedeutung                                                                                                                                                                                                                                                            |
|-----------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Gebiets \_ Schema snativecountryname  | **Windows 7 und höher:** Der Native Name des Landes bzw. der Region, z. b. "España für Spanien". Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig ist, ist 80, einschließlich eines abschließenden NULL-Zeichens.                                                                 |
| Gebiets \_ Schema snativectryname     | Veraltet für Windows 7 und höher. Der Native Name des Landes bzw. der Region. Siehe locale \_ snativecountryname.                                                                                                                                                             |
| LOCALE \_ snativecurrname     | **Windows Me/98, Windows 2000:** Der Native Name der dem Gebiets Schema zugeordneten Währung in der nativen Sprache des Gebiets Schemas. Es gibt keine Beschränkung für die Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind.                                                          |
| Gebiets \_ Schema-snativedigits       | Systemeigene Entsprechungen von ASCII 0 bis 9. Die maximal zulässige Anzahl von Zeichen für diese Zeichenfolge ist elf, einschließlich eines abschließenden NULL-Zeichens. Arabisch verwendet beispielsweise "012345 6789". Siehe auch [locale \_ idigitsubstitution](locale-idigitsubstitution.md). |
| LOCALE \_ snativedisplayname  | **Windows 7 und höher:** Anzeige Name des Gebiets Schemas in der nativen Sprache, z. b. Deutsch (Deutschland) für das Gebiets Schema Deutsch (Deutschland). <br/>                                                                                                        |
| Gebiets \_ Schema snativelangname     | Veraltet für Windows 7 und höher. Der Native Name der Sprache. Siehe locale \_ snativelanguagename.                                                                                                                                                                  |
| LOCALE \_ snativelanguagename | **Windows 7 und höher:** Der Native Name der Sprache, z. b. Հայերեն für Armenisch (Armenien). Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig ist, ist 80, einschließlich eines abschließenden NULL-Zeichens.                                                         |
