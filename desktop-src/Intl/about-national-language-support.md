# <a name="about-national-language-support"></a>Informationen zur Unterstützung der Landessprache

Die Funktionen der National Language Support (NLS) ermöglichen Anwendungen Folgendes:

-   Festlegen des Gebiets Schemas für den Benutzer
-   Identifizieren der Sprache, in der der Benutzer arbeitet
-   Abrufen von Zeichen folgen, die Uhrzeiten, Datumsangaben und andere Informationen für die angegebene Sprache und das angegebene Gebiets Schema darstellen

NLS bietet auch Unterstützung für Tastaturlayouts und sprachspezifische Schriftarten. Weitere Informationen finden Sie unter [Eingabemethoden-Manager](input-method-manager.md) und [Internationale Schriftarten und Text Anzeige](international-fonts-and-text-display.md).
