# <a name="locale_ipos-constants"></a>Gebiets Schema- \_ IPOs- \* Konstanten

In diesem Thema werden die \_ von NLS verwendeten Gebiets Schema-IPOs- \* Konstanten definiert.



| Wert                   | Bedeutung                                                                                                                                                                   |
|-------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Gebiets Schema \_ ipossepbyspace  | Trennung des monetären Symbols in einem positiven monetären Wert. Dieser Wert ist 1, wenn das Währungssymbol durch ein Leerzeichen von einem positiven Betrag getrennt ist. andernfalls ist der Wert 0.        |
| LOCALE \_ ipossignposn    | Formatierungs Index für positive Werte. Der Index verwendet die gleichen Werte wie das Gebiets Schema [ \_ inegsignposn](locale-ineg-constants.md), mit dem Unterschied, dass der Index NULL nicht verwendet wird.    |
| Gebiets Schema \_ ipossymprees | Die Position des Währungs Symbols in einem positiven monetären Wert. Dieser Wert ist 1, wenn das Währungssymbol dem positiven Betrag vorangestellt ist, oder 0, wenn das Symbol dem Betrag folgt. |
