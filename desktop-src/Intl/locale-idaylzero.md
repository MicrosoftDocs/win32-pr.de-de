# <a name="locale_idaylzero"></a>Gebiets Schema \_ idaylzero

Spezifizierer für führende Nullen in Tages Feldern nur für ein kurzes Datum. Die maximal zulässige Anzahl von Zeichen für diese Zeichenfolge ist 2, einschließlich eines abschließenden NULL-Zeichens. Es wird empfohlen, dass Ihre Anwendung die [locale \_ sshortdate](locale-sshortdate.md) -Konstante anstelle von locale \_ idaylzero verwendet.



| Wert | Bedeutung                   |
|-------|---------------------------|
| 0     | Keine führenden Nullen für Tage |
| 1     | Führende Nullen für Tage    |
