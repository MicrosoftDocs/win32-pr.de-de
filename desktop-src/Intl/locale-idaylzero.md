# <a name="locale_idaylzero"></a>LOCALE \_ IDAYLZERO

Spezifizierer für führende Nullen in Tagesfeldern nur für ein kurzes Datum. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, ist zwei, einschließlich eines abschließenden NULL-Zeichens. Es wird für Ihre Anwendung bevorzugt, die [ \_ LOCALE-SSHORTDATE-Konstante](locale-sshortdate.md) anstelle von LOCALE \_ IDAYLZERO zu verwenden.



| Wert | Bedeutung                   |
|-------|---------------------------|
| 0     | Keine führenden Nullen für Tage |
| 1     | Führende Nullen für Tage    |
