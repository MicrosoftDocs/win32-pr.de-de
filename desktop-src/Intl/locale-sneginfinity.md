# <a name="locale_sneginfinity"></a>LOCALE \_ SNEGINFINITY

**Windows Vista und höher:** Zeichenfolgenwert für "negative unendlich", z. B. "-Infinity" für das englische Gebietsschema (USA). Es gibt keine Beschränkung für die Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind.
