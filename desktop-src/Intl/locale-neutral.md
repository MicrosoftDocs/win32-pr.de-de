# <a name="locale_neutral"></a>Gebiets Schema \_ neutral

Das neutrale Gebiets Schema. Diese Konstante wird im Allgemeinen nicht verwendet, wenn NLS-APIs aufgerufen werden. Verwenden Sie stattdessen entweder " [locale \_ System \_ default](locale-system-default.md) " oder " [locale \_ User \_ default](locale-user-default.md)". Der Wert von locale \_ neutral ist 0x0000.
