# <a name="locale_neutral"></a>\_GEBIETSSCHEMANEUTRAL

Das neutrale Gebietsschema. Diese Konstante wird im Allgemeinen nicht verwendet, wenn NLS-APIs aufgerufen werden. Verwenden Sie stattdessen [ENTWEDER LOCALE \_ SYSTEM \_ DEFAULT](locale-system-default.md) oder [LOCALE USER \_ \_ DEFAULT](locale-user-default.md). Der Wert von LOCALE \_ NEUTRAL ist 0x0000.
