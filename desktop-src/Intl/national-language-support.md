# <a name="national-language-support"></a>Unterstützung für nationale Sprache

Mithilfe der NLS-Funktionen (National Language Support) unterstützen Anwendungen die verschiedenen Sprach-und Gebiets Schema spezifischen Anforderungen von Benutzern auf der ganzen Welt. In dieser Übersicht werden die NLS-Funktionen beschrieben und erläutert, wie Sie in Ihren Anwendungen verwendet werden.
