# <a name="locale_ipositivepercent"></a>Gebiets Schema \_ ipositiveprozent

**Windows 7 und höher:** Positives Prozentsatz Formatierungs Muster für das Gebiets Schema. Es kann nur ein Muster angegeben werden.



| Wert | Format                                    |
|-------|-------------------------------------------|
| 0     | Anzahl, Leerraum, Prozent Zum Beispiel \# % |
| 1     | Anzahl, Prozent; Zum Beispiel \#%         |
| 2     | Prozent, Zahl; Beispiel:%\#         |
| 3     | Prozent, Leerraum, Zahl; Beispiel:% \# |
