# <a name="locale_smonthname-constants"></a>LOCALE \_ \* SMONTHNAME-Konstanten

In diesem Thema werden die LOCALE \_ SMONTHNAME-Konstanten \* definiert, die von NLS verwendet werden.




| Wert | Bedeutung | 
|-------|---------|
| LOCALE_SMONTHNAME1 | Nativer long-Name für Januar. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt 80, einschließlich eines beendenden NULL-Zeichens.<blockquote>[!Note]<br />Das Aufrufen <a href="/windows/desktop/api/Winnls/nf-winnls-getlocaleinfoa"><strong>der GetLocaleInfo-</strong></a> oder <a href="/windows/desktop/api/Winnls/nf-winnls-getlocaleinfoex"><strong>GetLocaleInfoEx-Funktion</strong></a> mit einer LOCALE_SMONTHNAME*-Konstante gibt die eigenständige form des Monatsnamens zurück. Um die genitive Form des Monatsnamens abzurufen, ruft die Anwendung <a href="/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformata">GetDateFormat</a> oder <a href="/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformatex">GetDateFormatEx</a> mit einem Datumsbild von ddMMMM auf und entfernt die beiden Ziffern vom Anfang der abgerufenen Zeichenfolge.</blockquote><br /> |
