## <a name="mui-benefits-for-developers"></a><span data-ttu-id="67f20-107">MUI-Vorteile für Entwickler</span><span class="sxs-lookup"><span data-stu-id="67f20-107">MUI benefits for developers</span></span>

<span data-ttu-id="67f20-108">Es gibt viele Möglichkeiten, eine MUI-Lösung in-Anwendungen zu implementieren. jede Möglichkeit ist jedoch eine Variation einer von drei grundlegenden Methoden:</span><span class="sxs-lookup"><span data-stu-id="67f20-108">There are many possible ways to implement a MUI solution in applications, but each possibility is a variation of one of three basic methods:</span></span>

1.  <span data-ttu-id="67f20-109">Kompilieren einer Binärdatei (mit integrierten Ressourcen) für jede Sprache.</span><span class="sxs-lookup"><span data-stu-id="67f20-109">Compiling one binary (with built-in resources) for each language.</span></span> <span data-ttu-id="67f20-110">Dies ist der *de-facto* -Standard für Legacy Anwendungen, da dies das primäre Modell ist, das von Standard Entwicklungs Tools wie Microsoft Visual Studio unterstützt wird.</span><span class="sxs-lookup"><span data-stu-id="67f20-110">This is the *de facto* standard for legacy applications, as this is the primary model supported by standard development tools such as Microsoft Visual Studio.</span></span> <span data-ttu-id="67f20-111">Dieses Modell erfordert mehrere Binärdateien für mehrere Sprachen und weist Einschränkungen hinsichtlich der Bereitstellung einzelner Images und mehrsprachige Szenarien auf.</span><span class="sxs-lookup"><span data-stu-id="67f20-111">This model does require multiple binaries for multiple languages and has limitations with regards to single image deployment and multi-lingual scenarios.</span></span> <span data-ttu-id="67f20-112">Beachten Sie, dass die mit diesem Modell entwickelten Anwendungen weiterhin unter Windows Vista funktionieren. diese Tools werden bereitgestellt, die Entwicklern dabei helfen, von diesem Modell zu dem moderneren Modell zu wechseln, das in der dritten Methode beschrieben wird.</span><span class="sxs-lookup"><span data-stu-id="67f20-112">It should be noted that applications developed with this model will continue to work on Windows Vista, and that tools are provided that help developers move from this model to the more modern model outlined in the third method.</span></span>

3.  <span data-ttu-id="67f20-117">Die Verwendung einer sprach neutralen Kern Binärdatei mit einem Satz von Ressourcen-DLLs pro Sprache.</span><span class="sxs-lookup"><span data-stu-id="67f20-117">Having one language-neutral core binary with one set of resource DLLs per language.</span></span> <span data-ttu-id="67f20-118">Dies ist die Art und Weise, wie das Betriebssystem in Windows Vista implementiert wird. <span data-ttu-id="67f20-119">Vor der Windows Vista-Version machte das Fehlen integrierter Unterstützung für dieses letztere Modell die Übernahme schwierig.</span><span class="sxs-lookup"><span data-stu-id="67f20-119">Prior to the Windows Vista release, the lack of built-in support for this latter model made it hard to adopt.</span></span> <span data-ttu-id="67f20-120">Dies hat sich jedoch geändert, und die Vorteile dieses Modells sind zahlreiche Vorteile, die das Modell für Ihre Anwendungen hervorragend gestalten:</span><span class="sxs-lookup"><span data-stu-id="67f20-120">However, this has changed, and the benefits of this model are numerous and make it a great model for your applications:</span></span>

-   <span data-ttu-id="67f20-121">Anwendungen können mehrsprachig gemacht werden und können sich auf die gleiche Weise wie Windows Vista Verhalten, was eine konsistente Anzeige Sprache für Benutzer bietet.</span><span class="sxs-lookup"><span data-stu-id="67f20-121">Applications can be made multi-lingual and can behave in the same way as Windows Vista, providing a consistent display language experience for users.</span></span>
-   <span data-ttu-id="67f20-122">Flexibilität bei der Freigabe zusätzlicher Sprachen für eine Anwendung.</span><span class="sxs-lookup"><span data-stu-id="67f20-122">Flexibility is increased in releasing additional languages for an application.</span></span> <span data-ttu-id="67f20-123">Weitere Sprachen können unabhängig vom Kerncode freigegeben werden. Dies bedeutet, dass die Unterstützung für neue Sprachen im Laufe der Zeit nach Bedarf hinzugefügt werden kann.</span><span class="sxs-lookup"><span data-stu-id="67f20-123">Additional languages can be released independently of the core code, which means that support for new languages can be added over time as needed.</span></span>
-   <span data-ttu-id="67f20-124">Die Kosten werden bei der Erstellung und Wartung von sprach Releases reduziert.</span><span class="sxs-lookup"><span data-stu-id="67f20-124">Cost is reduced in creating and servicing more language releases.</span></span>
-   <span data-ttu-id="67f20-125">OEMs und Unternehmen können problemlos Anwendungen in Ihr globalisiertes PC-Image integrieren – bereit für die Lieferung in verschiedenen Ländern.</span><span class="sxs-lookup"><span data-stu-id="67f20-125">OEMs and enterprises can easily integrate applications into their globalized PC image—ready for shipment to different countries.</span></span>
-   <span data-ttu-id="67f20-126">Tools und Richtlinien, die Sie beim Migrieren Ihrer Anwendung zum Windows Vista MUI-Modell unterstützen, sind verfügbar.</span><span class="sxs-lookup"><span data-stu-id="67f20-126">Tools and guidelines that help you migrate your application to the Windows Vista MUI model are available.</span></span> <span data-ttu-id="67f20-127">Insbesondere bietet MSDN eine [bedeutende Dokumentation](multilingual-user-interface.md) zu MUI.</span><span class="sxs-lookup"><span data-stu-id="67f20-127">In particular, MSDN provides a [significant set of documentation](multilingual-user-interface.md) on MUI.</span></span>

## <a name="mui-benefits-for-enterprises"></a><span data-ttu-id="67f20-128">MUI-Vorteile für Unternehmen</span><span class="sxs-lookup"><span data-stu-id="67f20-128">MUI benefits for enterprises</span></span>

<span data-ttu-id="67f20-129">MUI bietet zwei wesentliche Vorteile für Unternehmen:</span><span class="sxs-lookup"><span data-stu-id="67f20-129">MUI offers two major benefits for enterprises:</span></span>

-   <span data-ttu-id="67f20-130">Installation mit einem einzelnen Image: mit MUI können Unternehmen das gleiche weltweite Image (oder eines beliebigen Teils davon) mit einer einzelnen Installation einführen, unterstützen und verwalten.</span><span class="sxs-lookup"><span data-stu-id="67f20-130">Single image installation: MUI allows enterprises to roll out, support and maintain the same worldwide (or any part thereof) image with a single installation.</span></span> <span data-ttu-id="67f20-131">Windows Vista erweitert das Rollout des Betriebssystems mit einem einzelnen Image, damit Geschäftsanwendungen auch als Teil des gleichen Images bereitgestellt werden können.</span><span class="sxs-lookup"><span data-stu-id="67f20-131">Windows Vista extends the single-image rollout of the operating system, so that business applications can also be deployed as part of the same image.</span></span>
-   <span data-ttu-id="67f20-132">Unterstützung für mehrsprachige Desktops: Es können mehrere lokalisierte UI-Sprachpakete auf einem Desktop installiert werden. ## <a name="mui-benefits-for-oems"></a><span data-ttu-id="67f20-134">Vorteile von MUI für OEMs</span><span class="sxs-lookup"><span data-stu-id="67f20-134">MUI benefits for OEMs</span></span>

<span data-ttu-id="67f20-135">Der größte Vorteil für OEMs ist die Installation von einem einzelnen Image, die MUI ermöglicht, da es möglich ist, Abbilder zu erstellen, die alle notwendigen Sprachen enthalten, um eine geografische Zone effektiv als Ziel festzustellen, und die Sprachauswahl für die Erstinstallation des Computers des Benutzers zu verzögern.</span><span class="sxs-lookup"><span data-stu-id="67f20-135">The major benefit for OEMs is the single image installation that MUI enables, as it makes it possible to create images that contain all necessary languages to target a geographic zone effectively, and to delay the language choice to the user's initial installation of the computer.</span></span> <span data-ttu-id="67f20-136">Dies ermöglicht insbesondere eine effektivere Verwaltung des Inventars für OEMs.</span><span class="sxs-lookup"><span data-stu-id="67f20-136">In particular, this enables a more effective management of inventory for OEMs.</span></span>

<span data-ttu-id="67f20-137">Durch die Bereitstellung von MUI-Unterstützung für Anwendungen ermöglicht Windows Vista auch OEMs das Bereitstellen von Anwendungen zum Hinzufügen von Werten auf Ihren Images, während Sie von der Installation mit einem einzelnen Image profitieren, solange diese Anwendungen MUI-fähig sind.</span><span class="sxs-lookup"><span data-stu-id="67f20-137">By providing MUI support for applications, Windows Vista also enables OEMs to provide value-add applications on their images while benefiting from the single image installation, as long as these applications are MUI enabled.</span></span>
