# <a name="multilingual-user-interface"></a>Multilingual User Interface

Die mehrsprachige Benutzeroberfläche (MUI) ermöglicht die Lokalisierung von Benutzeroberflächen für globalisierte Anwendungen. MUI unterstützt auch die Erstellung von Ressourcen für eine beliebige Anzahl von Sprachen in der Benutzeroberfläche.

In dieser Dokumentation wird MUI beschrieben, und Sie erfahren, wie Sie Ihrer Anwendung eine MUI-Funktionalität hinzufügen. Es wird jedoch nicht versucht, alle Details der globalisierter Anwendungsentwicklung unter Windows zu beschreiben. Ausführliche Informationen zu diesem Typ finden Sie im Microsoft [Go Global Developer Center](https://msdn.microsoft.com/goglobal).

Die MUI-Dokumentation enthält die folgenden Abschnitte:

-   [Informationen über mehrsprachige Benutzeroberfläche](about-multilingual-user-interface.md)
-   [Verwenden der mehrsprachigen Benutzeroberfläche](using-multilingual-user-interface.md)
-   [Mehrsprachige Benutzeroberflächen Referenz](multilingual-user-interface-reference.md)
