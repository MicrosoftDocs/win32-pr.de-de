# <a name="input-method-manager"></a>Eingabemethoden-Manager

Der Eingabemethoden-Manager (IMM) ist eine Technologie, die von einer Anwendung verwendet wird, um mit einem Input Method Editor (IME) zu kommunizieren, der als Dienst ausgeführt wird. Mit dem IME können Computerbenutzer komplexe Zeichen und Symbole (z. b. japanische Kanji-Zeichen) mithilfe einer Standardtastatur eingeben.

In diesem Abschnitt wird die imm-API beschrieben und erläutert, wie Sie die Funktionen zum Erstellen und Verwalten von IME-Fenstern verwenden. Er enthält folgende Abschnitte:

-   [Informationen über den Eingabemethoden-Manager](about-input-method-manager.md)
-   [Verwenden des Eingabemethoden-Managers](using-input-method-manager.md)
-   [Eingabemethoden-Manager-Referenz](input-method-manager-reference.md)
