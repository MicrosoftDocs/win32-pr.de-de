# <a name="input-method-manager"></a>Eingabemethoden-Manager

Input Method Manager (IMM) ist eine Technologie, die von einer Anwendung für die Kommunikation mit einem Eingabemethoden-Editor (INPUT Method Editor, IME) verwendet wird, der als Dienst ausgeführt wird. Mit der IME können Computerbenutzer komplexe Zeichen und Symbole wie japanische Kanjizeichen mithilfe einer Standardtastatur eingeben.

In diesem Abschnitt wird die IMM-API beschrieben und erläutert, wie Sie die Funktionen zum Erstellen und Verwalten von IME-Fenstern verwenden. Er enthält folgende Abschnitte:

-   [Informationen zum Eingabemethoden-Manager](about-input-method-manager.md)
-   [Verwenden des Eingabemethoden-Managers](using-input-method-manager.md)
-   [Referenz zum Eingabemethoden-Manager](input-method-manager-reference.md)
