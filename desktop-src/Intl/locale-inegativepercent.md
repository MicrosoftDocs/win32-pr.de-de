# <a name="locale_inegativepercent"></a>Gebiets \_ Schema inegativeprozent

**Windows 7 und höher:** Negatives Prozentsatz Formatierungs Muster für das Gebiets Schema. Es kann nur ein Muster angegeben werden. Wenn mehr als ein Format für das Gebiets Schema verwendet wird, wählen Sie die bevorzugte Option aus. Wenn beispielsweise ein negativer Prozentsatz "-9%" für "negative neun Prozent" angezeigt wird, ist die geeignete Auswahl für diese Konstante "0".



| Wert | Format                                                    |
|-------|-----------------------------------------------------------|
| 0     | Negatives Vorzeichen, Zahl, Leerraum, Prozentsatz; Beispiel:-\# % |
| 1     | Negatives Vorzeichen, Zahl, Prozent; Beispiel:-\#%         |
| 2     | Negatives Vorzeichen, Prozent, Zahl; Beispiel:-%\#         |
| 3     | Prozent, negatives Vorzeichen, Zahl; Beispiel:%-\#         |
| 4     | Prozent, Zahl, negatives Vorzeichen; Beispiel:%\#-         |
| 5     | Zahl, negatives Vorzeichen, Prozent Zum Beispiel \#-%         |
| 6     | Zahl, Prozent, negatives Vorzeichen; Zum Beispiel \#%-         |
| 7     | Negatives Vorzeichen, Prozent, Leerraum, Zahl; Beispiel:-% \# |
| 8     | Anzahl, Leerraum, Prozentsatz, Minuszeichen; Zum Beispiel \# %- |
| 9     | Prozent, Leerraum, Zahl, negatives Vorzeichen; Beispiel:% \#- |
| 10    | Prozent, Leerraum, negatives Vorzeichen, Zahl; Beispiel:%-\# |
| 11    | Zahl, negatives Zeichen, Leerzeichen, Prozent Beispiel: \# -% |
