# <a name="locale_igeoid"></a>LOCALE \_ IGEOID

Eine 32-Bit-Zahl mit Vorzeichen, die einen [geografischen Standort](table-of-geographical-locations.md)eindeutig identifiziert. Die Anwendung verwendet diese Konstante, um Kunden gebietsschemaspezifische Dienste bereitzustellen. Beispielsweise kann er als Schlüssel für den Zugriff auf einen Datenbankdatensatz verwendet werden, der spezifische Informationen zu einem Land/einer Region enthält.
