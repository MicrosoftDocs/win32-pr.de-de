# <a name="candidate-lists"></a>Kandidatenlisten

Eine Kandidatenliste wird durch eine [**candidatelist**](/windows/win32/api/imm/ns-imm-candidatelist) -Struktur dargestellt, die ein Array von Zeichen folgen enthält, das die Zeichen oder Zeichen folgen angibt, aus denen der Benutzer auswählen kann. Ihre IME-fähige Anwendung kann Kandidatenlisten Informationen mithilfe der [**immgetcandidatelistcount**](/windows/desktop/api/Imm/nf-imm-immgetcandidatelistcounta) -Funktion und der [**immgetcandidatelist**](/windows/desktop/api/Imm/nf-imm-immgetcandidatelista) -Funktion abrufen.
