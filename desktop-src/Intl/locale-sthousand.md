# <a name="locale_sthousand"></a>\_Gebiets Schema stausendstel

Zeichen, die verwendet werden, um Gruppen von Ziffern links vom Dezimaltrennzeichen zu trennen. Die maximal zulässige Anzahl von Zeichen für diese Zeichenfolge ist vier, einschließlich eines abschließenden NULL-Zeichens. In der Regel stellen diese Gruppen Tausende dar. Abhängig von dem für die Gebiets Schema- [ \_ sgruppierung](locale-sgrouping.md)angegebenen Wert können Sie jedoch etwas anderes darstellen.
