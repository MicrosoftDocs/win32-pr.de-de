# <a name="locale_sthousand"></a>LOCALE \_ STHOUSAND

Zeichen, die verwendet werden, um Gruppen von Ziffern links vom Dezimaltrennzeichen zu trennen. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt vier, einschließlich eines beendenden NULL-Zeichens. In der Regel stellen diese Gruppen Tausende dar. Je nach dem für [LOCALE \_ SGROUPING](locale-sgrouping.md)angegebenen Wert können sie jedoch etwas anderes darstellen.
