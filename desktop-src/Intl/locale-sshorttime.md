# <a name="locale_sshorttime"></a>LOCALE \_ sshorttime

**Windows 7 und höher:** Kurzzeit-Formatierungs Zeichenfolge für das Gebiets Schema Muster werden in der Regel abgeleitet, indem der Wert "SS" (Sekunden) aus dem langen Zeitformat Muster entfernt wird. Wenn das lange Zeitformat z. b. "h:mm: ss tt" ist, ist das kurze Zeitformat höchstwahrscheinlich "h:mm tt". Mit dieser Konstante können mehrere Formate in einer durch Semikolons getrennten Liste angegeben werden. Allerdings sollte das bevorzugte kurze Zeitformat der erste aufgeführte Wert sein.
