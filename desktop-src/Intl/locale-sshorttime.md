# <a name="locale_sshorttime"></a>LOCALE \_ SSHORTTIME

**Windows 7 und höher:** Kurze Formatierungszeichenfolge für das Gebietsschema. Muster werden in der Regel abgeleitet, indem der Wert "ss" (Sekunden) aus dem Formatmuster für lange Zeit entfernt wird. Wenn das lange Zeitformat beispielsweise "h:mm:ss tt" lautet, ist das kurze Zeitformat höchstwahrscheinlich "h:mm tt". Diese Konstante kann mehrere Formate in einer durch Semikolons getrennten Liste angeben. Das bevorzugte Format für kurze Zeit sollte jedoch der erste aufgeführte Wert sein.
