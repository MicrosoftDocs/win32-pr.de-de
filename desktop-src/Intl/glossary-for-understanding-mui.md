# <a name="glossary-for-understanding-mui"></a>Glossar zum Verständnis von MUI

-   [Globalisierung](#globalization)
-   [Lokalisierbarkeit](#localizability)
-   [Lokalisierung](#localization)
-   [Internationalisierung](#internationalization)

## <a name="globalization"></a>Globalisierung

Der Begriff *Globalisierung* gibt an, dass die Plattform von überall aus arbeiten muss. In der Praxis wird diese Vision innerhalb von Windows implementiert, indem eine einzige weltweite Binärdatei für das Windows-Betriebssystem mit der Windows Vista-Version bereitgestellt wird, indem die Bereitstellung einzelner Images in einem mehrsprachigen Kontext unterstützt wird und NLS-APIs zum Abrufen Gebiets Schema spezifischer Daten bereitgestellt werden.

## <a name="localizability"></a>Lokalisierbarkeit

Der Begriff " *Lokalisier barkeit* " bedeutet, dass das Produkt einfacher lokalisiert werden kann, was wiederum zu einer Verlangsamung der Lokalisierungskosten führt. Dies kann durch die effiziente Verwendung von Ressourcen-APIs erreicht werden, wobei der Umfang und die Anzahl der dem System hinzugefügten Ressourcen gesteuert werden und die Tools verbessert werden, die für die Arbeit mit diesen Ressourcen verwendet werden.

## <a name="localization"></a>Lokalisierung

*Lokalisierung* ist der Prozess, bei dem ein Programm für einen bestimmten lokalen Markt angepasst wird. dazu gehören das Übersetzen der Benutzeroberfläche, das Ändern der Größe von Dialogfeldern, das Anpassen von Features (falls erforderlich) und das Testen von Ergebnissen, um sicherzustellen, dass das Programm weiterhin funktioniert. Im am häufigsten verwendeten Kontext konzentriert sich die Lokalisierung auf die Übersetzung der Benutzeroberfläche.

## <a name="internationalization"></a>Internationalisierung

Die *Internationalisierung* wird als Synonym für die Globalisierung an manchen Stellen oder bei der Globalisierung und Lokalisierung an anderen Orten verwendet, um alle Aktivitäten zu beschreiben, die ein Produkt im weltweiten Kontext verbessern.
