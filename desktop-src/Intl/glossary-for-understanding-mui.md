## <a name="globalization"></a><span data-ttu-id="cb390-108">Globalisierung</span><span class="sxs-lookup"><span data-stu-id="cb390-108">Globalization</span></span>

<span data-ttu-id="cb390-109">Der Begriff *Globalisierung* gibt an, dass die Plattform von überall aus arbeiten muss.</span><span class="sxs-lookup"><span data-stu-id="cb390-109">The term *globalization* indicates the need for the platform to work everywhere out of the box.</span></span> <span data-ttu-id="cb390-110">In der Praxis wird diese Vision innerhalb von Windows implementiert, indem eine einzige weltweite Binärdatei für das Windows-Betriebssystem mit der Windows Vista-Version bereitgestellt wird, indem die Bereitstellung einzelner Images in einem mehrsprachigen Kontext unterstützt wird und NLS-APIs zum Abrufen Gebiets Schema spezifischer Daten bereitgestellt werden.</span><span class="sxs-lookup"><span data-stu-id="cb390-110">In practice, this vision is realized within Windows by having a single worldwide binary for the Windows operating system starting with the Windows Vista release, by supporting single image deployment in a multilingual context and by providing NLS APIs to retrieve locale-specific data.</span></span>

## <a name="localizability"></a><span data-ttu-id="cb390-111">Lokalisierbarkeit</span><span class="sxs-lookup"><span data-stu-id="cb390-111">Localizability</span></span>

<span data-ttu-id="cb390-112">Der Begriff " *Lokalisier barkeit* " bedeutet, dass das Produkt einfacher lokalisiert werden kann, was wiederum zu einer Verlangsamung der Lokalisierungskosten führt.</span><span class="sxs-lookup"><span data-stu-id="cb390-112">The term *localizability* means making the product easier to localize, which in turn drives down localization costs.</span></span> <span data-ttu-id="cb390-113">Dies kann durch die effiziente Verwendung von Ressourcen-APIs erreicht werden, wobei der Umfang und die Anzahl der dem System hinzugefügten Ressourcen gesteuert werden und die Tools verbessert werden, die für die Arbeit mit diesen Ressourcen verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="cb390-113">This can be accomplished by efficient use of resource APIs, controlling the extent and number of resources added to the system, and improving the tools used to work with these resources.</span></span>

<span data-ttu-id="cb390-115">*Lokalisierung* ist der Prozess, bei dem ein Programm für einen bestimmten lokalen Markt angepasst wird. dazu gehören das Übersetzen der Benutzeroberfläche, das Ändern der Größe von Dialogfeldern, das Anpassen von Features (falls erforderlich) und das Testen von Ergebnissen, um sicherzustellen, dass das Programm weiterhin funktioniert.</span><span class="sxs-lookup"><span data-stu-id="cb390-115">*Localization* is the process of adapting a program for a specific local market, which includes translating the user interface, resizing dialog boxes, customizing features (if necessary), and testing results to ensure that the program still works.</span></span> <span data-ttu-id="cb390-116">Im am häufigsten verwendeten Kontext konzentriert sich die Lokalisierung auf die Übersetzung der Benutzeroberfläche.</span><span class="sxs-lookup"><span data-stu-id="cb390-116">In the most commonly used context, localization focuses on translating the user interface.</span></span>
