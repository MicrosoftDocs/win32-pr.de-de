# <a name="locale_iuseutf8legacyoemcp"></a>LOCALE \_ IUSEUTF8LEGACYOEMCP

**Ab der nächsten Version von Windows:** Es wird empfohlen, dass Anwendungen stattdessen Unicode verwenden. Auf Computern, die der Benutzer für die Verwendung einer UTF-8-Codepage konfiguriert hat, ist LOCALE_IUSEUTF8LEGACYOEMCP die Legacy-OEM-Codepage, die auf einem Nicht-UTF-8-Legacycomputer Windows verwendet worden wäre.
