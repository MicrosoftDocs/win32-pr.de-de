# <a name="locale_snan"></a>\_GEBIETSSCHEMA-SNAN

**Windows Vista und höher:** Zeichenfolgenwert für "Not a number", z.B. "Nan" für das gebietsschema "English" (USA). Es gibt keine Beschränkung für die Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind.
