# <a name="locale_snan"></a>Gebiets Schema- \_ Snan

**Windows Vista und höher:** Der Zeichen folgen Wert für "not a Number", z. b. "NaN" für das englische Gebiets Schema (USA). Es gibt keine Beschränkung für die Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind.
