# <a name="locale_sdecimal"></a>LOCALE \_ SDECIMAL

Zeichen, die für das Dezimaltrennzeichen verwendet werden, z.B. "." in "3.14" oder "," in "3,14". Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt vier, einschließlich eines beendenden NULL-Zeichens.
