# <a name="locale_sdecimal"></a>sdecimal (locale) \_

Zeichen, die für das Dezimaltrennzeichen verwendet werden, z. b. "." in "3,14" oder "," in "3, 14". Die maximal zulässige Anzahl von Zeichen für diese Zeichenfolge ist vier, einschließlich eines abschließenden NULL-Zeichens.
