# <a name="locale_sintlsymbol"></a>LOCALE \_ SINTLSYMBOL

Drei Zeichen des internationalen Währungssymbols, das in ISO 4217 angegeben ist, gefolgt von dem Zeichen, das diese Zeichenfolge vom Betrag trennt. Die maximale Anzahl von Zeichen, die für diese Zeichenfolge zulässig sind, beträgt 9, einschließlich eines beendenden NULL-Zeichens.
