# <a name="spell-checking-api"></a>Rechtschreibprüfung-API

Die Rechtschreibprüfungs-API ermöglicht Entwicklern das Verwenden von Rechtschreibprüfung, um Text zu überprüfen, Vorschläge zu erhalten und Einstellungen und Benutzer Wörterbücher zu verwalten. Entwickler können auch Rechtschreibprüfungen erstellen, die als Anbieter in das Rechtschreib Prüfungs Framework integriert sind.

-   [Informationen zur Rechtschreibprüfungs-API](about-the-spell-checker-api.md)
-   [API-Referenz zur Rechtschreibprüfung](spell-checker-api-reference.md)
