# <a name="codecapi_avencvideosupportedcontrols-property"></a>Codecapi \_ avencvideosupportedcontrols-Eigenschaft

Legt die unterstützten Steuerelemente für einen Video Encoder fest.

## <a name="data-type"></a>Datentyp

**UINT64 (VT \_ UI8)**

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideosupportedcontrols**

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft wird nur mit [H. 264 UVC 1,5-Kamera Encodern](camera-encoder-h264-uvc-1-5.md)verwendet.
