# <a name="codecapi_avencvideosupportedcontrols-property"></a>CODECAPI \_ AVEncVideoSupportedControls-Eigenschaft

Legt die unterstützten Steuerelemente für einen Videoencoder fest.

## <a name="data-type"></a>Datentyp

**UINT64 (VT \_ UI8)**

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncVideoSupportedControls**

## <a name="remarks"></a>Hinweise

Diese Eigenschaft wird nur mit [H.264 UVC 1.5-Kameraencodern](camera-encoder-h264-uvc-1-5.md)verwendet.
