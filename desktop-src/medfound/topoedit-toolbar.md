# <a name="topoedit-toolbar"></a>Topoedit-Symbolleiste

Standardmäßig wird die Symbolleiste am oberen Rand des Anwendungsfensters und unterhalb der Menüleiste angezeigt. Die Symbolleiste ermöglicht den schnellen Maus Zugriff auf die Menü Elemente der Steuer [Elemente](controls-menu.md) . Die Symbolleisten Schaltflächen werden erst aktiviert, nachdem eine Topologie erfolgreich aufgelöst wurde.

Der folgende Screenshot zeigt die topoedit-Symbolleiste.

![Screenshot der Symbolleiste "topoedit"](images/410257c5-ae5c-454c-a9a8-ba05fc5235b5.jpg)
