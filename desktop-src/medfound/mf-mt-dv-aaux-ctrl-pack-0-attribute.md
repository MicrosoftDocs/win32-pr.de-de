# <a name="mf_mt_dv_aaux_ctrl_pack_0-attribute"></a>MF \_ MT \_ DV \_ Aaux \_ STRG \_ Pack \_ 0-Attribut

Das Quell Code Verwaltungspaket für die audiohilfsquelle (Aaux) für den ersten Audioblock in einem Medien Typ Digital Video (DV).

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut entspricht dem **dwdvaauxctl** -Member der DirectShow [**dvinfo**](/windows/win32/api/strmif/ns-strmif-dvinfo) -Struktur.

Die GUID-Konstante für dieses Attribut wird aus "mfuuid. lib" exportiert.
