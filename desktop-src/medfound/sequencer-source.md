# <a name="sequencer-source"></a>Sequencerquelle

Mit der Sequencerquelle kann eine Anwendung eine Sammlung von Medienquellen sequenziell mit nahtlosen Übergängen zwischen den Quellen wieder geben. Sie können damit Wiedergabelisten erstellen oder Streams aus mehreren Quellen gleichzeitig wieder geben.

|--------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [Informationen zur Sequencerquelle](about-the-sequencer-source.md)       | Bietet eine Übersicht über die Sequencerquelle und deren Umgang mit Topologien.                                      |
| [Verwenden der Sequencerquelle](using-the-sequencer-source.md)       | Beschreibt, wie die Sequencerquelle in einer Anwendung verwendet wird.                                                     |
| [Sequencer-Quellereignisse](sequencer-source-events.md)             | Beschreibt die Ereignisse, die von der Sequencerquelle generiert werden.                                                 |
| [Erstellen einer Wiedergabeliste](how-to-create-a-playlist.md)           | Beschreibt, wie die Sequencerquelle verwendet wird, um eine Wiedergabeliste zu erstellen.                                                  |
| [Sequencer Source Example Code](sequencer-source-example-code.md) | Enthält den vollständigen Beispielcode für das vorherige Thema.                                                       |
| [Sequenzpräsentationszeiten](sequence-presentation-times.md)     | Beschreibt, wie die Mediensitzung die Präsentationsuhr steuert, wenn die Präsentation die Sequencerquelle verwendet. |
