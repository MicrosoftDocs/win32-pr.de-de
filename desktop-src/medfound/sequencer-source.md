# <a name="sequencer-source"></a>Sequencer-Quelle

Mithilfe der Sequencer-Quelle kann eine Anwendung eine Auflistung von Medienquellen sequenziell wiedergeben, wobei nahtlose Übergänge zwischen den Quellen möglich sind. Sie können Sie zum Erstellen von Wiedergabelisten oder zum gleichzeitigen Abspielen von Streams aus mehreren Quellen verwenden.

In diesem Abschnitt werden die folgenden Themen behandelt:

| [Informationen über die Sequencer-Quelle](about-the-sequencer-source.md)       | Bietet eine Übersicht über die Sequencer-Quelle und die Behandlung von Topologien.                                      |
| [Verwenden der Sequencer-Quelle](using-the-sequencer-source.md)       | Beschreibt, wie die Sequencer-Quelle in einer Anwendung verwendet wird.                                                     |
| [Sequencer-Quell Ereignisse](sequencer-source-events.md)             | Beschreibt die Ereignisse, die von der Sequencer-Quelle generiert werden.                                                 |
| [Erstellen einer Wiedergabeliste](how-to-create-a-playlist.md)           | Beschreibt, wie die Sequencer-Quelle verwendet wird, um eine Wiedergabeliste zu erstellen.                                                  |
| [Beispiel Code für die Sequencer-Quelle](sequencer-source-example-code.md) | Enthält den kompletten Beispielcode für das vorherige Thema.                                                       |
| [Sequenz Präsentations Zeiten](sequence-presentation-times.md)     | Beschreibt, wie die Präsentations Uhr von der Medien Sitzung gesteuert wird, wenn die Präsentation die Sequencer-Quelle verwendet. |
