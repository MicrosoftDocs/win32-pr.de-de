# <a name="imfmediasourceextensiongetreadystate-method"></a>Imfmediasourceextension:: getreadystate-Methode

Ruft den Status "bereit" der Medienquelle ab.

## <a name="syntax"></a>Syntax


```C++
MF_MSE_READY GetReadyState();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Der bereite Zustand der Medienquelle.
