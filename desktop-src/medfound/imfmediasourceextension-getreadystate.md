# <a name="imfmediasourceextensiongetreadystate-method"></a>ANDROMediaSourceExtension::GetReadyState-Methode

Ruft den Bereitschaftszustand der Medienquelle ab.

## <a name="syntax"></a>Syntax


```C++
MF_MSE_READY GetReadyState();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Der Zustand "Bereit" der Medienquelle.
