# <a name="mf_media_engine_audio_category-attribute"></a>MF \_ MEDIA ENGINE AUDIO \_ \_ \_ CATEGORY-Attribut

Gibt die Kategorie des Audiostreams an.

## <a name="data-type"></a>Datentyp

**[**\_KATEGORIE \_ "AUDIOSTREAM"**](/windows/win32/api/audiosessiontypes/ne-audiosessiontypes-audio_stream_category)**

## <a name="remarks"></a>Hinweise

Der Wert dieses Attributs ist ein Member der [**AUDIO \_ STREAM \_ CATEGORY-Enumeration.**](/windows/win32/api/audiosessiontypes/ne-audiosessiontypes-audio_stream_category)

Dieses Attribut wird zusammen mit der [**METHODE VONMEDIAENGINEClassFactory::CreateInstance**](/windows/desktop/api/mfmediaengine/nf-mfmediaengine-imfmediaengineclassfactory-createinstance) verwendet, um die Medien-Engine zu initialisieren. Das Attribut ist optional.
