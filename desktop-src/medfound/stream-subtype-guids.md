# <a name="stream-subtype-guids"></a>GuiDs des Streamuntertyps

Die folgenden Untertyp-GUIDs werden in der Headerdatei mfapi.h definiert. Um den Untertyp anzugeben, legen Sie das [**MF \_ MT \_ SUBTYPE-Attribut**](mf-mt-subtype-attribute.md) für den Medientyp fest.

Wenn diese Untertypen verwendet werden, legen Sie das [MF \_ MT MAJOR \_ \_ TYPE-Attribut](mf-mt-major-type-attribute.md) auf **MFMediaType \_ Stream fest.**



| GUID                               | Beschreibung              |
|------------------------------------|--------------------------|
| **MFStreamFormat \_ MPEG2Program**   | MPEG-2-Programmstream    |
| **MFStreamFormat \_ MPEG2Transport** | MPEG-2-Transportstream. |
