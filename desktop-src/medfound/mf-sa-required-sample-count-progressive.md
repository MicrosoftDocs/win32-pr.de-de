# <a name="mf_sa_required_sample_count_progressive-attribute"></a>MF \_ SA REQUIRED SAMPLE COUNT \_ \_ \_ \_ PROGRESSIVE-Attribut

Gibt die Anzahl der Stichproben an, die eine Microsoft Media Foundation Transform (MFT) für progressiven Inhalt zugeordnet werden muss.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Hinweise

Dieser Wert wird verwendet, wenn der nächste Knoten downstream über [**einen NSDVideoSampleAllocator verfügt.**](/windows/desktop/api/mfidl/nn-mfidl-imfvideosampleallocator)
