# <a name="mf_sa_required_sample_count_progressive-attribute"></a>Erforderliche Anzahl von ggf. \_ \_ erforderlichen \_ \_ \_ Attributen der MF-SA

Gibt die Anzahl der Stichproben an, die eine Microsoft Media Foundation Transformation (MFT) benötigt, um progressiven Inhalten zugewiesen zu werden.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Bemerkungen

Dieser Wert wird verwendet, wenn der nächste Knoten Downstream über einen [**IMF videosamplezuordcator**](/windows/desktop/api/mfidl/nn-mfidl-imfvideosampleallocator)verfügt.
