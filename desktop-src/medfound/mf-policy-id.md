# <a name="mf_policy_id-attribute"></a>\_Richtlinien- \_ ID-Attribut für MF

Ein Bezeichner, der für [IMF outputpolicy](/windows/win32/api/mfidl/nn-mfidl-imfoutputpolicy)festgelegt werden kann.

## <a name="data-type"></a>Datentyp

**UInt32** (als **bool** behandelt)

## <a name="remarks"></a>Bemerkungen

Der Wert kann vom [MEPolicySet](./mepolicyset.md) -Medienereignis empfangen werden. Sie wird als **VT_UI4** Type-Nutzlast im **MEPolicySet** -Ereignis gespeichert.
