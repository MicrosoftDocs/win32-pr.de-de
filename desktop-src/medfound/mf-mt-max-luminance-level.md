# <a name="mf_mt_max_luminance_level-attribute"></a>Maximal zulässige Größe für das MF \_ MT- \_ \_ \_ Attribut

Gibt die maximale Leuchtkraft Ebene des Inhalts in Nits an. Dieses Attribut weist die gleiche Semantik wie **maxcll** auf, wie im CEA-861,3-Standard definiert.

## <a name="data-type"></a>Datentyp

**UINT32**
