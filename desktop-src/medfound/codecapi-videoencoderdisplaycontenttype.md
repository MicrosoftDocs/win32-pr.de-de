# <a name="codecapi_videoencoderdisplaycontenttype-property"></a>Codecapi \_ videoencoderdisplaycontenttype (Eigenschaft)

Legt fest, ob es sich bei dem Inhalt um Vollbildvideos handelt, im Gegensatz zu Bildschirm Inhalten, die möglicherweise ein kleineres Videofenster enthalten oder überhaupt kein Video enthalten.

## <a name="data-type"></a>Datentyp

**VT \_ UI4**

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ videoencoderdisplaycontenttype**

## <a name="property-value"></a>Eigenschaftswert

Wenn der Wert ungleich 0 (null) ist, handelt es sich bei dem Inhalt um Vollbildvideos. Wenn der Wert 0 (null) ist, handelt es sich bei dem Inhalt nicht um Vollbildvideos.

## <a name="remarks"></a>Bemerkungen

Unter Windows 10 wird der Wert dieser Eigenschaft nicht verwendet.
