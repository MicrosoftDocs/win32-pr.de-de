# <a name="codecapi_avdecddstereodownmixmode-property"></a>Codecapi \_ avdecddstereodownmixmode (Eigenschaft)

Gibt den Stereo downmischmodus für einen Dolby Digital-Audiodecoder an.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avdecddstereodownmixmode**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eavdecddstereodownmixmode**](/windows/win32/api/codecapi/ne-codecapi-eavdecddstereodownmixmode) -Enumeration.

## <a name="remarks"></a>Bemerkungen

Dieses Attribut gilt, wenn es sich bei der Eingabe für den Decoder um Multichannel PCM-Audiodaten handelt und die Ausgabe Stereo-Audiodaten ist.
