# <a name="mf_user_data_payload-attribute"></a>MF \_ USER \_ DATA \_ PAYLOAD-Attribut

Legt fest, ob eine Benutzerdatennutzlast in das Ausgabebeispiel enthalten sein soll.

## <a name="data-type"></a>Datentyp

**Blob**

## <a name="remarks"></a>Hinweise

Benutzerdaten können Daten wie Untertiteldaten oder Balkendaten enthalten.

Benutzerdaten werden ohne Decodierung übergeben.
