# <a name="mf_user_data_payload-attribute"></a>\_Benutzer \_ Daten- \_ Nutz Last Attribut für MF

Legt fest, ob eine Benutzerdaten Nutzlast mit dem Ausgabe Beispiel eingeschlossen wird.

## <a name="data-type"></a>Datentyp

**BLOB**

## <a name="remarks"></a>Bemerkungen

Benutzerdaten können Daten wie das Schließen von Beschriftungs Daten oder Balken Daten enthalten.

Benutzerdaten werden ohne Decodierung übermittelt.
