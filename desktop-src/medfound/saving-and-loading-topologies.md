# <a name="saving-and-loading-topologies"></a>Speichern und Laden von Topologien

Mit topoedit können Sie eine Topologie als XML-Datei speichern und Sie später laden. Wenn eine vollständige Topologie geladen wird, muss Sie erneut aufgelöst werden. Weitere Informationen zu diesem Verfahren finden Sie unter [Auflösen einer Topologie mit topoedit](resolving-a-topology-with-topoedit.md).

## <a name="to-save-a-topology"></a>So speichern Sie eine Topologie

1.  Klicken Sie im Menü **Datei** auf **Speichern**.

    Das Dialogfeld **Datei speichern** wird geöffnet.

2.  Navigieren Sie zu dem Ordner, in dem die topologiedatei gespeichert werden muss.

3.  Geben Sie im Feld **Dateiname:** den Namen der Datei ein.

4.  Klicken Sie auf **Speichern**.

Die topologiedatei wird als XML-Datei gespeichert.

## <a name="to-load-a-saved-topology"></a>So laden Sie eine gespeicherte Topologie

1.  Klicken Sie im Menü **Datei** auf **Öffnen**.

    Das Dialogfeld **Struktur laden** wird geöffnet.

2.  Navigieren Sie zu dem Ordner, in dem sich die XML-Topologie befindet.

3.  Geben Sie den Dateinamen im Feld **Dateiname:** an, oder wählen Sie die Datei aus der Liste aus.

4.  Klicken Sie auf **Öffnen**.

Die topologiedatei wird im **Topologiebereich** geladen.
