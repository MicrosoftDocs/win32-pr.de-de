# <a name="media-foundation-pipeline"></a>Media Foundation Pipeline

Die *Pipelineebene* in Microsoft Media Foundation ist die Ebene der Architektur, die Mediendaten direkt generiert oder verarbeitet.

Die meisten Anwendungen rufen Methoden nicht direkt in den Pipelinekomponenten auf, obwohl dies möglich ist. Lesen Sie diese Themen, wenn Sie eine benutzerdefinierte Pipelinekomponente schreiben.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                     | Beschreibung                                                                                                                           |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [Medienquellen](media-sources.md)<br/>                             | Medienquellen generieren Mediendaten, in der Regel aus einer Datei, einem Erfassungsgerät oder einem Netzwerkstream. <br/>                              |
| [Media Foundation Transformationen](media-foundation-transforms.md)<br/> | Media Foundation (MFTs) verarbeiten Mediendaten. Beispielsweise werden Codecs in Media Foundation als MFTs implementiert. <br/>   |
| [Mediensenken](media-sinks.md)<br/>                                 | Mediensenken nutzen Mediendaten. Beispielsweise kann eine Mediensenke die Daten in eine Datei schreiben oder die Daten über ein Netzwerk senden.<br/> |
