# <a name="media-foundation-pipeline"></a>Media Foundation Pipeline

Die *Pipeline* Schicht in Microsoft Media Foundation ist die Ebene der Architektur, die Mediendaten direkt generiert oder verarbeitet.

Bei den meisten Anwendungen werden Methoden nicht direkt auf den Pipeline Komponenten aufgerufen, obwohl dies möglich ist. Lesen Sie diese Themen, wenn Sie eine benutzerdefinierte Pipeline Komponente schreiben.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                     | BESCHREIBUNG                                                                                                                           |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [Medienquellen](media-sources.md)<br/>                             | Medienquellen generieren Mediendaten, in der Regel über eine Datei, ein Erfassungsgerät oder einen Netzwerkstream. <br/>                              |
| [Media Foundation Transformationen](media-foundation-transforms.md)<br/> | Media Foundation Transformationen (MFTs) verarbeiten von Mediendaten. Beispielsweise werden Codecs in Media Foundation als MFTs implementiert. <br/>   |
| [Medien senken](media-sinks.md)<br/>                                 | Medien senken verbrauchen Mediendaten. Beispielsweise könnte eine Medien Senke die Daten in eine Datei schreiben oder die Daten über ein Netzwerk senden.<br/> |
