# <a name="mf_sensorstream_required_capabilities-attribute"></a>MF \_ SENSORSTREAM \_ REQUIRED \_ CAPABILITIES-Attribut

Dieses Attribut enthält eine durch Semikolons getrennte Liste von Funktionszeichenfolgen, die die für einen bestimmten Stream erforderlichen Funktionen angibt. Eine Liste der Funktionszeichenfolgen, die in dieses Attribut aufgenommen werden können, finden Sie unter [DeviceCapability](/uwp/schemas/appxpackage/appxmanifestschema/element-devicecapability).

## <a name="data-type"></a>Datentyp

**Wchar\***
