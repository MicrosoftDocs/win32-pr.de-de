# <a name="mf_mt_source_content_hint-attribute"></a>MF \_ MT SOURCE CONTENT \_ \_ \_ HINT-Attribut

Beschreibt das beabsichtigte Seitenverhältnis für einen Videomedientyp.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Hinweise

Der Wert dieses Attributs ist ein Member der [**MFVideoSrcContentHintFlags-Enumeration.**](/windows/desktop/api/mfapi/ne-mfapi-mfvideosrccontenthintflags)

Die GUID-Konstante für dieses Attribut wird aus mfuuid.lib exportiert.
