# <a name="mfpkey_closedentrypoint-property"></a>Mfpkey \_ closedentrypoint-Eigenschaft

Gibt das Codierungs Muster für den Encoder an, der am Anfang einer Gruppe von Bildern verwendet werden soll.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

**g \_ wszwmvcclosedentrypoint**

## <a name="data-type"></a>Datentyp

**VT \_ bool**

## <a name="default-value"></a>Standardwert

**Variant \_ false**

## <a name="remarks"></a>Bemerkungen

Wenn Sie diese Eigenschaft auf **Variant \_ true** festlegen, ist das Codierungs Muster bbibbp. Wenn Sie diese Eigenschaft auf **Variant \_ false** festlegen, ist das Codierungs Muster ibbpbbp.
