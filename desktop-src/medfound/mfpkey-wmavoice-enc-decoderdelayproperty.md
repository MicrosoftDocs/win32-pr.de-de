# <a name="mfpkey_wmavoice_enc_decoderdelay-property"></a>Mfpkey \_ wmavoice \_ ENC- \_ decoderdelay-Eigenschaft

Gibt die encoderwartezeit in Paketeinheiten an.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

Nur mit [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore)verfügbar.

## <a name="data-type"></a>Datentyp

VT \_ I4

## <a name="remarks"></a>Bemerkungen

Legen Sie den Wert dieser Eigenschaft auf 3, 4 oder 5 fest. Da der Wert von 3 auf 5 zunimmt, erhöhen sich auch die Qualität und die Verzögerung.
