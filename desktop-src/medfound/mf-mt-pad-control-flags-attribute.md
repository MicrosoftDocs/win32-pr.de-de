# <a name="mf_mt_pad_control_flags-attribute"></a>MF- \_ MT- \_ Pad- \_ steuerungflagattribut \_

Gibt das Seitenverhältnis des Ausgabe Rechtecks für einen Video Medientyp an.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Bemerkungen

Der Wert dieses Attributs ist ein Member der [**mfvideopadflags**](/windows/desktop/api/mfapi/ne-mfapi-mfvideopadflags) -Enumeration.

Die GUID-Konstante für dieses Attribut wird aus "mfuuid. lib" exportiert.
