## <a name="remarks"></a>Bemerkungen

Wenn ein Connector als *eingebettet* oder *integriert* bezeichnet wird, bedeutet dies, dass der Connector intern ist. Diese Connectors verfügen über "Embedded" im Namen der Enumerationskonstante.

Anwendungen sollten das interne Flag " **OPM \_ COPP \_ Compatible \_ Connector \_ Type \_** " ignorieren und stattdessen die Connector-Typen überprüfen, deren konstanter Name "Embedded" ist.

Mit Ausnahme von **OPM \_ COPP \_ Compatible \_ Connector \_ Type \_ internal** sind die hier aufgeführten Werte keine Bitflags und können nicht kombiniert werden.

Einige dieser Werte entsprechen Werten aus der **COPP \_ Connector Type** -Enumeration, die im Certified Output Protection Protocol (COPP) verwendet wird.
