# <a name="mf_mt_transfer_function-attribute"></a>MF \_ MT- \_ Übertragungs \_ Funktions Attribut

Gibt die Konvertierungs Funktion für einen Video Medientyp zwischen RGB und r' B ' an.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Bemerkungen

Der Wert dieses Attributs ist ein Member der [**mfvideotransferfunction**](/windows/desktop/api/mfobjects/ne-mfobjects-mfvideotransferfunction) -Enumeration.

Die GUID-Konstante für dieses Attribut wird aus "mfuuid. lib" exportiert.
