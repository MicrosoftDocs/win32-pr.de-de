# <a name="mfpkey_conv_inputframerate-property"></a>MFPKEY \_ CONV \_ INPUTFRAMERATE-Eigenschaft

Gibt die Eingabebildrate als Verhältnis an.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

Nur mit [**IPropertyStore verfügbar.**](/windows/win32/api/propsys/nn-propsys-ipropertystore)

## <a name="data-type"></a>Datentyp

VT \_ UI8

## <a name="applies-to"></a>Gilt für

-   [Frameratenkonverter](framerateconverter.md)

## <a name="remarks"></a>Hinweise

Store den Zähler in den höheren 4 Bytes und den Nenner in den unteren 4 Bytes.
