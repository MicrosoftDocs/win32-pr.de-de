# <a name="mf_mt_audio_num_channels-attribute"></a>MF \_ MT \_ - \_ Attribut für audionum- \_ Kanäle

Anzahl von Audiokanälen in einem audiomedientyp.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut entspricht dem **nchannels** -Member der [WaveFormatEx](mf-mt-audio-prefer-waveformatex-attribute.md) -Struktur.

Die GUID-Konstante für dieses Attribut wird aus "mfuuid. lib" exportiert.
