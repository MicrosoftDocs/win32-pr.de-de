# <a name="mf_mt_audio_num_channels-attribute"></a>MF \_ MT AUDIO NUM \_ \_ \_ CHANNELS-Attribut

Anzahl der Audiokanäle in einem Audiomedientyp.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Hinweise

Dieses Attribut entspricht dem **nChannels-Member** der [WAVEFORMATEX-Struktur.](mf-mt-audio-prefer-waveformatex-attribute.md)

Die GUID-Konstante für dieses Attribut wird aus mfuuid.lib exportiert.
