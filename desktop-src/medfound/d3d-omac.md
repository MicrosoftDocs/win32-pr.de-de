# <a name="d3d_omac-structure"></a>D3D \_ OMAC-Struktur

Enthält einen Nachrichtenauthentifizierungscode (Mac).

## <a name="syntax"></a>Syntax


```C++
typedef struct _D3D_OMAC {
  BYTE Omac[D3D_OMAC_SIZE];
} D3D_OMAC;
```



## <a name="members"></a>Member

<dl> <dt>

**OMAC**
</dt> <dd>

Ein Bytearray, das den kryptografischen Mac-Wert der Nachricht enthält.
