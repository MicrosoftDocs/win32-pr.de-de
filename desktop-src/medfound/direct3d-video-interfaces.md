# <a name="direct3d-9-video-interfaces"></a>Direct3D 9-Video Schnittstellen

Beschreibt die Microsoft Direct3D 9-Video Schnittstellen.



| Schnittstelle                                                                | BESCHREIBUNG                                                                                                |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IDirect3D9ExOverlayExtension**](/windows/desktop/api/d3d9/nn-d3d9-idirect3d9exoverlayextension)     | Fragt die Überlagerungs Hardwarefunktionen eines Direct3D-Geräts ab.                                            |
| [**IDirect3DAuthenticatedChannel9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dauthenticatedchannel9) | Stellt einen Kommunikationskanal mit dem Grafiktreiber oder der Direct3D-Laufzeit bereit.                        |
| [**IDirect3DCryptoSession9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dcryptosession9)               | Stellt eine Kryptografiesitzung dar.                                                                        |
| [**IDirect3DDevice9Video**](/windows/desktop/api/d3d9/nn-d3d9-idirect3ddevice9video)                   | Ermöglicht es einer Anwendung, Inhalts Schutz-und Verschlüsselungsdienste zu verwenden, die von einem Grafiktreiber implementiert werden. |
| [**IDirect3DDXVADevice9**](idirect3ddxvadevice9.md)                     | Stellt einen Hardwarebeschleuniger für die DirectX-Video Beschleunigung (DXVA) 1,0 dar.                               |
| [**IDirect3DVideoDevice9**](idirect3dvideodevice9.md)                   | Ermöglicht die hardwarebeschleunigte Decodierung von einem Direct3D-Gerät unter Verwendung der DXVA-Version 1,0.                      |
