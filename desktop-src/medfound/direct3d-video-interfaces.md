# <a name="direct3d-9-video-interfaces"></a>Direct3D 9-Videoschnittstellen

Beschreibt die Microsoft Direct3D 9-Videoschnittstellen.



| Schnittstelle                                                                | BESCHREIBUNG                                                                                                |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IDirect3D9ExOverlayExtension**](/windows/desktop/api/d3d9/nn-d3d9-idirect3d9exoverlayextension)     | Fragt die Überlagerungshardwarefunktionen eines Direct3D-Geräts ab.                                            |
| [**IDirect3DAuthenticatedChannel9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dauthenticatedchannel9) | Stellt einen Kommunikationskanal für den Grafiktreiber oder die Direct3D-Runtime bereit.                        |
| [**IDirect3DCryptoSession9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dcryptosession9)               | Stellt eine kryptografische Sitzung dar.                                                                        |
| [**IDirect3DDevice9Video**](/windows/desktop/api/d3d9/nn-d3d9-idirect3ddevice9video)                   | Ermöglicht einer Anwendung die Verwendung von Inhaltsschutz- und Verschlüsselungsdiensten, die von einem Grafiktreiber implementiert werden. |
| [**IDirect3DDXVADevice9**](idirect3ddxvadevice9.md)                     | Stellt eine Hardwarebeschleunigung für DirectX Video Acceleration (DXVA) 1.0 dar.                               |
| [**IDirect3DVideoDevice9**](idirect3dvideodevice9.md)                   | Ermöglicht die hardwarebeschleunigte Decodierung von einem Direct3D-Gerät mit dxva Version 1.0.                      |
