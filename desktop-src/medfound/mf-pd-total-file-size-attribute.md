# <a name="mf_pd_total_file_size-attribute"></a>\_ \_ \_ Datei \_ Größen Attribut für MF-PD Gesamt

Gibt die Gesamtgröße der Quelldatei in Bytes an. Dieses Attribut gilt für Präsentations Deskriptoren. Eine Medienquelle kann dieses Attribut optional festlegen.

## <a name="data-type"></a>Datentyp

**UINT64**

## <a name="remarks"></a>Bemerkungen

Die GUID-Konstante für dieses Attribut wird aus "mfuuid. lib" exportiert.
