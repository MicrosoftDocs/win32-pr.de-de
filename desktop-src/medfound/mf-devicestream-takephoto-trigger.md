# <a name="mf_devicestream_takephoto_trigger-attribute"></a>MF-debug-Attribut (Debug- \_ \_ \_ Attribut)

Gibt an, ob der Take Photo-auslöst in der Geräte Quelle gekapselt ist.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Bemerkungen

Legen Sie dieses Attribut auf einen Wert ungleich 0 (null) fest, um den Take Photo--Auslösewert in der Geräte Quelle
