# <a name="mf_devicestream_takephoto_trigger-attribute"></a>MF \_ DEVICESTREAM \_ TAKEPHOTO \_ TRIGGER-Attribut

Gibt an, ob der Fotoaufnahmetrigger in der Gerätequelle gekapselt ist.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Hinweise

Legen Sie dieses Attribut auf einen Wert von 0 (null) fest, um den Take Photo-Trigger in der Gerätequelle zu kapseln.
