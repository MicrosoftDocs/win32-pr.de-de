# <a name="codecapi_avencvideoroienabled-property"></a>Codecapi- \_ Eigenschaft "avencvideoroiaktivi"

Gibt an, ob das für das Eingabe Beispiel festgelegte Attribut " [MF Sample Extension \_ roirechteck](mfsampleextension-roirectangle.md) " berücksichtigt wird oder nicht.

## <a name="data-type"></a>Datentyp

**Ulong** (VT \_ UI4)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideoroiaktivierte**

## <a name="remarks"></a>Bemerkungen

Der Standardwert ist 0.

Wenn ein Encoder-MFT einen Wert ungleich 0 (null) akzeptiert, wird erwartet, dass der Encoder das in der Eingabe Stichprobe festgelegte Attribut " [mfsampleextension \_ roirechteck](mfsampleextension-roirectangle.md) " berücksichtigt.
