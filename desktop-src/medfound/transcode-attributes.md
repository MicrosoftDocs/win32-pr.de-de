# <a name="transcode-attributes"></a>Transcode-Attribute

Die folgenden Attribute werden mit der transcodieren-API verwendet.



| Attribut                                                                          | BESCHREIBUNG                                                                                  |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [\_transcode- \_ Anpassungs \_ Profil für MF](mf-transcode-adjust-profile.md)                  | Profilflags, die die streameinstellungen für die transcodieren-Topologie definieren.                    |
| [MF- \_ transcode- \_ ContainerType](mf-transcode-containertype.md)                     | Gibt den Containertyp einer codierten Datei an.                                             |
| [MF- \_ transcode- \_ \_ Anfüge- \_ Encoder](mf-transcode-donot-insert-encoder.md)     | Gibt an, ob ein Encoder in der transcodieren-Topologie enthalten sein muss.                     |
| [MF- \_ transcode- \_ Codierungs Profil](mf-transcode-encodingprofile.md)                 | Gibt das Geräte Konformitäts Profil für die Codierung von ASF-Dateien (Advanced Streaming Format) an. |
| [MF- \_ transcode \_ qualityvsspeed](mf-transcode-qualityvsspeed.md)                   | Gibt den Kompromiss zwischen Codierungsqualität und Codierungs Geschwindigkeit an.                          |
| [MF- \_ transcode \_ Skip- \_ \_ metadatenübertragung](mf-transcode-skip-metadata-transfer.md) | Gibt an, ob Metadaten in die transcodierte Datei geschrieben werden.                                |
| [**MF- \_ transcode- \_ topologymode**](/windows/desktop/api/mfidl/ne-mfidl-mf_transcode_topologymode_flags)             | Gibt an, ob das topologielader hardwarebasierte Transformationen lädt.                   |
