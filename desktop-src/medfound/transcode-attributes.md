# <a name="transcode-attributes"></a>Transcodieren von Attributen

Die folgenden Attribute werden mit der Transcodierungs-API verwendet.



| attribute                                                                          | Beschreibung                                                                                  |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [ANPASSEN DES \_ MF-TRANSCODIERUNGSPROFILS \_ \_](mf-transcode-adjust-profile.md)                  | Profilflags, die die Streameinstellungen für die Transcodierungstopologie definieren.                    |
| [MF \_ TRANSCODE \_ CONTAINERTYPE](mf-transcode-containertype.md)                     | Gibt den Containertyp einer codierten Datei an.                                             |
| [MF \_ TRANSCODE \_ DONOT \_ INSERT \_ ENCODER](mf-transcode-donot-insert-encoder.md)     | Gibt an, ob ein Encoder in die Transcodierungstopologie eingeschlossen werden muss.                     |
| [MF \_ TRANSCODE \_ ENCODINGPROFILE](mf-transcode-encodingprofile.md)                 | Gibt das Gerätekonformitätsprofil für die Codierung von ASF-Dateien (Advanced Streaming Format) an. |
| [\_ \_ MF-TRANSCODIERUNGSQUALITÄTVSSPEED](mf-transcode-qualityvsspeed.md)                   | Gibt den Kompromiss zwischen Codierungsqualität und Codierungsgeschwindigkeit an.                          |
| [\_MF-TRANSCODIERUNG \_ \_ ÜBERSPRINGT \_ METADATENÜBERTRAGUNG](mf-transcode-skip-metadata-transfer.md) | Gibt an, ob Metadaten in die transcodierte Datei geschrieben werden.                                |
| [**MF \_ TRANSCODE \_ TOPOLOGYMODE**](/windows/desktop/api/mfidl/ne-mfidl-mf_transcode_topologymode_flags)             | Gibt an, ob das Topologielader hardwarebasierte Transformationen geladen.                   |
