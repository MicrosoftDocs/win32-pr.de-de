# <a name="mf_mt_video_3d-attribute"></a>MF \_ MT \_ VIDEO \_ 3D-Attribut

Gibt an, ob ein Videostream 3D-Inhalt enthält.

## <a name="data-type"></a>Datentyp

**BOOL als** **UINT32 gespeichert**

## <a name="remarks"></a>Hinweise

Dieses Attribut gilt für Videomedientypen. Wenn dieses Attribut **TRUE ist,** enthält der Videostream 3D-Inhalt. Der Standardwert ist **FALSE**.
