# <a name="mf_mt_video_3d-attribute"></a>Das \_ MF \_ MT \_ -Video 3D-Attribut

Gibt an, ob ein Videostream 3D-Inhalt enthält.

## <a name="data-type"></a>Datentyp

**Bool** gespeichert als **UInt32**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut gilt für Video Medientypen. Wenn dieses Attribut **true** ist, enthält der Videostream 3D-Inhalt. Der Standardwert ist **FALSE**.
