# <a name="codecapi_avencvideocontenttype-property"></a>Codecapi- \_ Eigenschaft "avencvideocontenttype"

Gibt den Typ des Video Inhalts eines Video Encoders an.

## <a name="data-type"></a>Datentyp

**ULONG**

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideocontenttype**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eavencvideocontenttype**](/windows/win32/api/codecapi/ne-codecapi-eavencvideocontenttype) -Enumeration.

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist ein Hinweis für den Encoder, seine Codierungs Einstellungen zu optimieren.
