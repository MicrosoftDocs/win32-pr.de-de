# <a name="codecapi_avencnumworkerthreads-property"></a>CODECAPI \_ AVEncNumWorkerThreads-Eigenschaft

Legt die Anzahl von Arbeitsthreads fest, die von einem Videoencoder verwendet werden.

## <a name="data-type"></a>Datentyp

**ULONG** (VT \_ UI4)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncNumWorkerThreads**

## <a name="remarks"></a>Hinweise

Wenn der Wert 0 (null) ist, wählt der Encoder die Anzahl der Threads aus.
