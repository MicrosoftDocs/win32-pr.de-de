# <a name="codecapi_avencnumworkerthreads-property"></a>Codecapi- \_ Eigenschaft ' avencnumworkerthreads '

Legt die Anzahl von Arbeitsthreads fest, die von einem Video Encoder verwendet werden.

## <a name="data-type"></a>Datentyp

**Ulong** (VT \_ UI4)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi- \_ avencnumworkerthreads**

## <a name="remarks"></a>Bemerkungen

Wenn der Wert 0 (null) ist, wählt der Encoder die Anzahl der Threads aus.
