# <a name="mfpkey_stat_ravg-property"></a>\_MFPKEY-STAT-EIGENSCHAFT \_ "NAGG"

Gibt die durchschnittliche Bitrate eines codierten Streams in Bits pro Sekunde an. Schreibgeschützt.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

**g \_ wszWMVCAvgBitrate**

## <a name="data-type"></a>Datentyp

**VT \_ I4**

## <a name="remarks"></a>Hinweise

Um die durchschnittliche Bitrate eines codierten Streams zu bestimmen, lesen Sie diese Eigenschaft am Ende der Codierung.
