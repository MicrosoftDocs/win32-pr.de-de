# <a name="mfpkey_stat_ravg-property"></a>Mfpkey \_ stat \_ Ravg (Eigenschaft)

Gibt die durchschnittliche Bitrate eines codierten Streams in Bits pro Sekunde an. Schreibgeschützt.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

**g \_ wszwmvcavgbitrate**

## <a name="data-type"></a>Datentyp

**VT \_ I4**

## <a name="remarks"></a>Bemerkungen

Um die durchschnittliche Bitrate eines codierten Streams zu ermitteln, lesen Sie diese Eigenschaft am Ende der Codierung.
