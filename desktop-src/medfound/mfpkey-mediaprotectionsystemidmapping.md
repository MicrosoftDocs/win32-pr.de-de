# <a name="mfpkey_mediaprotectionsystemidmapping-property"></a>Mfpkey \_ mediaschutzsystememappeneigenschaft

Gibt ein [**ipropertyset**](/uwp/api/Windows.Foundation.Collections.IPropertySet?view=winrt-19041) an, das die Zuordnung von einer Eigenschaften System-ID zu einer Eigenschafts System-Aktivierungs-ID definiert.



Datentyp

PROPVARIANT-Typ (VT)

PROPVARIANT-Member

**IUnknown**

VT \_ unbekannt

**punkVal**



## <a name="remarks"></a>Bemerkungen

Dieser Eigenschafts Schlüssel wird verwendet, um ein [**ipropertyset**](/uwp/api/Windows.Foundation.Collections.IPropertySet?view=winrt-19041) zu übergeben, das Zuordnungen zwischen den GUIDs des Schutzsystems und Lauf Zeit Klassen Zeichenfolgen zu einer Medienquelle enthält.
