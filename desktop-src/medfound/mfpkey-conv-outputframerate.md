# <a name="mfpkey_conv_outputframerate-property"></a>MFPKEY \_ CONV \_ OUTPUTFRAMERATE-Eigenschaft

Gibt die Ausgabebildrate als Verhältnis an.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

Nur mit [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore)verfügbar.

## <a name="data-type"></a>Datentyp

VT \_ UI8

## <a name="applies-to"></a>Gilt für

-   [Bildfrequenzkonverter](framerateconverter.md)

## <a name="remarks"></a>Hinweise

Store den Zähler in den höheren 4 Bytes und den Nenner in den unteren 4 Bytes.
