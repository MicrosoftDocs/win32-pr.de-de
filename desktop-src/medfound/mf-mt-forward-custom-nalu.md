# <a name="mf_mt_forward_custom_nalu-attribute"></a>\_ \_ \_ Benutzerdefiniertes \_ Nalu-Attribut für MF MT

Gibt an, dass die Einheiten Typen der Netzwerk Abstraktionsschicht (NAL) vom Decoder an Ausgabe Beispiele weitergeleitet werden sollen.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Bemerkungen

Wenn der Decoder eine Nalu-Datei analysiert, wird er nicht weitergeleitet.
