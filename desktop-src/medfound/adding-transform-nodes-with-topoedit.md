## <a name="to-add-a-registered-mft-to-the-topology"></a><span data-ttu-id="0f202-109">So fügen Sie der Topologie eine registrierte MFT hinzu</span><span class="sxs-lookup"><span data-stu-id="0f202-109">To add a registered MFT to the topology</span></span>

1.  <span data-ttu-id="0f202-110">Klicken Sie im Menü **Topologie** auf **Transformation hinzufügen**.</span><span class="sxs-lookup"><span data-stu-id="0f202-110">On the **Topology** menu, click **Add Transform**.</span></span>

    <span data-ttu-id="0f202-111">Das Dialogfeld **Transformation auswählen** wird geöffnet.</span><span class="sxs-lookup"><span data-stu-id="0f202-111">The **Select Transform** dialog box opens.</span></span> <span data-ttu-id="0f202-112">Es wird eine kategorisierte Liste registrierter MFTs angezeigt, die durch das Auflisten der registrierten Einträge in der Registrierung durch Aufrufen der [**mftenum**](/windows/desktop/api/mfapi/nf-mfapi-mftenum) -Funktion generiert wird.</span><span class="sxs-lookup"><span data-stu-id="0f202-112">It displays a categorized list of registered MFTs that is generated by enumerating the registered entries in the registry by calling the [**MFTEnum**](/windows/desktop/api/mfapi/nf-mfapi-mftenum) function.</span></span>

2.  <span data-ttu-id="0f202-113">Erweitern Sie die Kategorie, und wählen Sie die MFT aus, die Sie der Topologie hinzufügen möchten.</span><span class="sxs-lookup"><span data-stu-id="0f202-113">Expand the category and select the MFT that you want to add to the topology.</span></span>

<span data-ttu-id="0f202-115">Topoedit erstellt den angegebenen Transformations Knoten.</span><span class="sxs-lookup"><span data-stu-id="0f202-115">TopoEdit creates the specified transform node.</span></span> <span data-ttu-id="0f202-116">Der Bereich " **Topologie** " zeigt den Transformations Knoten als grünes Feld an, in dem der Name der MFT angezeigt wird.</span><span class="sxs-lookup"><span data-stu-id="0f202-116">The **Topology Pane** shows the transform node as a green box that displays the name of the MFT.</span></span>
