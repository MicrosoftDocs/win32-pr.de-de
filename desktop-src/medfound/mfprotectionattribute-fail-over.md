# <a name="mfprotectionattribute_fail_over-attribute"></a>MFPROTECTIONATTRIBUTE-FAIL \_ \_ OVER-Attribut

Gibt an, ob für den Schutz ein Fehler auf diesen erfolgt, wenn die beste Leistung fehlschlägt. Dieses Attribut kann mit [**DEM ATTRIBUTEOutputSchema-Objekt**](/windows/desktop/api/mfidl/nn-mfidl-imfoutputschema) verwendet werden.

## <a name="data-type"></a>Datentyp

**BOOL** als **UINT32** gespeichert

## <a name="remarks"></a>Hinweise

Dieses Attribut kann mit [**DEM ATTRIBUTEOutputSchema-Objekt**](/windows/desktop/api/mfidl/nn-mfidl-imfoutputschema) verwendet werden.
