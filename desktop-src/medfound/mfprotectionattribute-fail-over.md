# <a name="mfprotectionattribute_fail_over-attribute"></a>Mfschutzattribute-Attribut für Failover \_ \_

Gibt an, ob für den Schutz ein Failover zu diesem ausgeführt wird, wenn die bestmögliche Vorgehens Dieses Attribut kann mit [**IMF OutputSchema**](/windows/desktop/api/mfidl/nn-mfidl-imfoutputschema) -Objekten verwendet werden.

## <a name="data-type"></a>Datentyp

**Bool** gespeichert als **UInt32**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut kann mit [**IMF OutputSchema**](/windows/desktop/api/mfidl/nn-mfidl-imfoutputschema) -Objekten verwendet werden.
