# <a name="mf_mt_spatial_audio_max_dynamic_objects-attribute"></a>MF \_ MT SPATIAL AUDIO MAX DYNAMIC \_ \_ \_ \_ \_ OBJECTS-Attribut

Gibt die maximale Anzahl von dynamischen Audioobjekten an, die vom Audioendpunkt auf ähnliche Art gerendert werden können.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Hinweise

Ein [](/windows/desktop/api/mfspatialaudio/nn-mfspatialaudio-imfspatialaudiosample) 50-000-000-30-000-Wert enthält möglicherweise weniger Stichproben als die zahl, die von diesem Attribut angegeben wird. Wenn ein Beispiel mehr Audioobjekte enthält, als von diesem Attribut angegeben werden, ist das Verhalten nicht definiert.
