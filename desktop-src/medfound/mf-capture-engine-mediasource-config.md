# <a name="mf_capture_engine_mediasource_config-attribute"></a>\_ \_ \_ MediaSource- \_ Konfigurations Attribut der MF-Erfassungs-Engine

Enthält Konfigurations Eigenschaften für die Erfassungs Quelle.

## <a name="data-type"></a>Datentyp

**IPropertyStore \* *_ gespeichert als _* IUnknown\***

## <a name="remarks"></a>Bemerkungen

Derzeit wird dieses Attribut ignoriert. In Zukunft kann Sie verwendet werden, um Konfigurations Eigenschaften an die Medienquelle zu übergeben.
