# <a name="imfsourcebufferlistgetlength-method"></a>IMF sourcebufferlist:: GetLength-Methode

Ruft die Anzahl der [**IMF sourceBuffer**](/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfsourcebuffer) -Objekte in der Liste ab.

## <a name="syntax"></a>Syntax


```C++
DWORD GetLength();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Die Anzahl der Quell Puffer in der Liste.
