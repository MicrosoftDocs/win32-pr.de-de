# <a name="mfpkey_pixelaspectratio-property"></a>MFPKEY \_ PIXELASPECTRATIO-Eigenschaft

Gibt das Pixelseitenverhältnis des Ausgabemedientyps an.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

Nur mit [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore)verfügbar.

## <a name="data-type"></a>Datentyp

VT \_ UI8

## <a name="applies-to"></a>Gilt für

-   [Video Resizer DSP](videoresizer.md)

## <a name="remarks"></a>Hinweise

Speichert den Zähler in den höheren 4 Bytes und den Nenner in den unteren 4 Bytes.
