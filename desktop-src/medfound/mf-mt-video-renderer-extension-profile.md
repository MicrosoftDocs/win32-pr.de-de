# <a name="mf_mt_video_renderer_extension_profile-attribute"></a>MF \_ MT \_ VIDEO \_ RENDERER EXTENSION \_ \_ PROFILE-Attribut

Enthält eine Zeichenfolge, die einem Eintrag in der **VideoRendererExtensionProfiles-Liste** eines UWP-App-Manifests entspricht, um auszuwählen, welche Auswirkung geladen werden soll.

## <a name="data-type"></a>Datentyp

**LPCWSTR**
