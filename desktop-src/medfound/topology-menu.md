# <a name="topology-menu"></a>Topologiemenü

Im Menü **Topologie** werden die folgenden Befehle angezeigt:




| Befehl | BESCHREIBUNG | 
|---------|-------------|
| <strong>Quelle hinzufügen...</strong> | Fügt einen Quellknoten für die angegebene Mediendatei hinzu. Weitere Informationen finden <a href="adding-source-nodes-with-topoedit.md">Sie unter Hinzufügen von Quellknoten mit TopoEdit</a>. | 
| <strong>Hinzufügen von SAR</strong> | Fügt einen Ausgabeknoten für den StreamingAudiorenderer (SAR) hinzu. Weitere Informationen finden <a href="adding-output-nodes-with-topoedit.md">Sie unter Hinzufügen von Ausgabeknoten mit TopoEdit.</a> | 
| <strong>Hinzufügen von EVR</strong> | Fügt einen Ausgabeknoten für den erweiterten Videorenderer (EVR) hinzu. Weitere Informationen finden <a href="adding-output-nodes-with-topoedit.md">Sie unter Hinzufügen von Ausgabeknoten mit TopoEdit.</a> | 
| <strong>Transformieren hinzufügen...</strong> | Fügt den angegebenen Transformationsknoten hinzu. Weitere Informationen finden Sie unter <a href="adding-transform-nodes-with-topoedit.md">Hinzufügen von Transformationsknoten mit TopoEdit</a>. | 
| <strong>Tee hinzufügen</strong> | Fügt einen Teeknoten zum Senden der Ausgabe an mehrere Ausgabeknoten hinzu. | 
| <strong>Benutzerdefiniertes MFT hinzufügen...</strong> | Fügt eine benutzerdefinierte Media Foundation Transform (MFT) hinzu. Weitere Informationen finden Sie unter <a href="adding-transform-nodes-with-topoedit.md">Hinzufügen von Transformationsknoten mit TopoEdit</a>. | 
| <strong>Benutzerdefinierte Senke hinzufügen...</strong> | Fügt einen Ausgabeknoten für eine benutzerdefinierte Streamsenke hinzu. Weitere Informationen finden <a href="adding-output-nodes-with-topoedit.md">Sie unter Hinzufügen von Ausgabeknoten mit TopoEdit.</a> | 
| <strong>Videoaufnahmequelle hinzufügen...</strong> | Fügt einen Quellknoten für ein Videoaufnahmegerät hinzu.<br /><blockquote>[!Note]<br />Für dieses Feature ist Windows 7 erforderlich.</blockquote><br /> | 
| <strong>Audioaufnahmequelle hinzufügen...</strong> | Fügt einen Quellknoten für ein Audioaufnahmegerät hinzu.<br /><blockquote>[!Note]<br />Für dieses Feature ist Windows 7 erforderlich.</blockquote><br /> | 
| <strong>Auflösen der Topologie</strong> | Erstellt eine vollständige Topologie für die Wiedergabe. Weitere Informationen finden Sie unter <a href="resolving-a-topology-with-topoedit.md">Auflösen einer Topologie mit TopoEdit.</a> | 
| <strong>Löschen</strong> | Entfernt ein Element, Topologieknoten oder Knotenverbindungen aus dem <strong>Topologiebereich</strong>. |
