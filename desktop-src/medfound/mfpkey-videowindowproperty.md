# <a name="mfpkey_videowindow-property"></a>Mfpkey- \_ videowindow-Eigenschaft

Gibt den Umfang des Inhalts in Millisekunden an, der in den Modell Puffer passen kann.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

g \_ wszwmvcvideowindow

## <a name="data-type"></a>Datentyp

VT \_ I4

## <a name="default-value"></a>Standardwert

3000

## <a name="remarks"></a>Bemerkungen

Das Puffer Fenster ist einer der "Leaky Bucket"-Modellparameter, die in der Codec-Raten Steuerung verwendet werden.
