# <a name="codecapi_avencvideominqp-property"></a>CODECAPI \_ AVEncVideoMinQP (Eigenschaft)

Gibt den minimalen Quantisierungsparameter (QP) für die Videocodierung an.

## <a name="data-type"></a>Datentyp

**ULONG** (VT \_ UI4)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncVideoMinQP**

## <a name="property-value"></a>Eigenschaftswert

Der Wert ist der Mindestwert für QP, der in der Variablen-QP-Codierung verwendet wird.
