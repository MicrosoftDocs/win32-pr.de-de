# <a name="overview-of-table-of-contents-parser"></a><span data-ttu-id="e825e-103">Übersicht über den parserinhalts-Parser</span><span class="sxs-lookup"><span data-stu-id="e825e-103">Overview of Table of Contents Parser</span></span>

<span data-ttu-id="e825e-104">Der Inhaltsverzeichnis-Parser (Inhaltsverzeichnis) ist eine Windows-Komponente, mit der Sie erweiterte Indexinformationen in Mediendateien speichern können.</span><span class="sxs-lookup"><span data-stu-id="e825e-104">Table of Contents Parser (TOC Parser) is a Windows component that enables you to store advanced index information in media files.</span></span> <span data-ttu-id="e825e-105">Beispiele für erweiterte Indexinformationen sind Inhalts Tabellen, semantische Indizes und Storyboards.</span><span class="sxs-lookup"><span data-stu-id="e825e-105">Examples of advanced index information are tables of contents, semantic indices, and story boards.</span></span>

<span data-ttu-id="e825e-106">Mit dem Inhaltsverzeichnis können Sie Software schreiben, die es Benutzern ermöglicht, interessante Szenen in einer Videodatei zu finden.</span><span class="sxs-lookup"><span data-stu-id="e825e-106">With TOC Parser, you can write software that enables users to find scenes of interest in a video file.</span></span> <span data-ttu-id="e825e-107">Beispielsweise können Sie dem Benutzer ein Inhaltsverzeichnis bereitstellen, damit er die primären Abschnitte des Videos identifizieren und dann einen bestimmten Abschnitt anzeigen kann.</span><span class="sxs-lookup"><span data-stu-id="e825e-107">For example, you can provide the user with a table of contents so that he or she can identify the primary sections of the video and then view a particular section.</span></span> <span data-ttu-id="e825e-108">Sie können dem Benutzer auch einen Index bereitstellen, sodass er eine Liste von Szenen finden kann, die mit einem bestimmten Wort oder Ausdruck verknüpft sind.</span><span class="sxs-lookup"><span data-stu-id="e825e-108">You can also provide the user with an index so that he or she can find a list of scenes that are related to a particular word or phrase.</span></span>

<span data-ttu-id="e825e-109">Der Content Generator (Inhalts Generator) ist eine Windows-Komponente, die zusammen mit dem TOC-Parser funktioniert.</span><span class="sxs-lookup"><span data-stu-id="e825e-109">Table of Contents Generator (TOC Generator) is a Windows component that works along with TOC Parser.</span></span> <span data-ttu-id="e825e-110">Der TOC-Generator kann Szenen und Cluster von Szenen in einer Videodatei erkennen und dann ein Inhaltsverzeichnis generieren, das Sie als Ausgangspunkt in der Software verwenden können.</span><span class="sxs-lookup"><span data-stu-id="e825e-110">TOC Generator can detect scenes and clusters of scenes in a video file and then generate a table of contents that you can use as a starting point in your software.</span></span>

<span data-ttu-id="e825e-111">In der folgenden Liste finden Sie einige Szenarien, in denen Sie die Dienste verwenden können, die von TOC-Parser oder TOC-Generator bereitgestellt werden.</span><span class="sxs-lookup"><span data-stu-id="e825e-111">The following list gives some scenarios in which you could use the services provided by TOC Parser or TOC Generator.</span></span>

-   <span data-ttu-id="e825e-112">Erstellen Sie von Grund auf ein Inhaltsverzeichnis, und Betten Sie es in eine Videodatei ein.</span><span class="sxs-lookup"><span data-stu-id="e825e-112">Create a table of contents from scratch and embed it in a video file.</span></span>
-   <span data-ttu-id="e825e-113">Dient zum Lesen eines Inhaltsverzeichnisses, das bereits in eine Videodatei eingebettet ist.</span><span class="sxs-lookup"><span data-stu-id="e825e-113">Read a table of contents that is already embedded in a video file.</span></span>
-   <span data-ttu-id="e825e-114">Bearbeiten Sie ein Inhaltsverzeichnis, das bereits in eine Videodatei eingebettet ist.</span><span class="sxs-lookup"><span data-stu-id="e825e-114">Edit a table of contents that is already embedded in a video file.</span></span>
-   <span data-ttu-id="e825e-115">Verwenden Sie den TOC-Generator, um auf der Grundlage von Szenen und Clustern, die in einer Videodatei erkannt werden, ein Inhaltsverzeichnis zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="e825e-115">Use TOC Generator to create a table of contents based on scenes and clusters that it detects in a video file.</span></span> <span data-ttu-id="e825e-116">Bearbeiten Sie das generierte Inhaltsverzeichnis, und Betten Sie es in die Videodatei ein.</span><span class="sxs-lookup"><span data-stu-id="e825e-116">Edit the generated table of contents and embed it in the video file.</span></span>
