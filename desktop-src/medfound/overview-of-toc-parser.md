# <a name="overview-of-table-of-contents-parser"></a>Übersicht über den Inhaltsverzeichnisparser

Der Inhaltsverzeichnisparser (TOC Parser) ist eine Windows Komponente, mit der Sie erweiterte Indexinformationen in Mediendateien speichern können. Beispiele für erweiterte Indexinformationen sind Inhaltstabellen, semantische Indizes und Storyboards.

Mit dem TOC-Parser können Sie Software schreiben, mit der Benutzer interessante Szenen in einer Videodatei finden können. Beispielsweise können Sie dem Benutzer ein Inhaltsverzeichnis bereitstellen, damit er die primären Abschnitte des Videos identifizieren und dann einen bestimmten Abschnitt anzeigen kann. Sie können dem Benutzer auch einen Index bereitstellen, damit er eine Liste von Szenen finden kann, die sich auf ein bestimmtes Wort oder einen bestimmten Ausdruck beziehen.

Table of Contents Generator (TOC Generator) ist eine Windows-Komponente, die zusammen mit dem TOC-Parser funktioniert. Der TOC-Generator kann Szenen und Szenencluster in einer Videodatei erkennen und dann ein Inhaltsverzeichnis generieren, das Sie als Ausgangspunkt in Ihrer Software verwenden können.

Die folgende Liste enthält einige Szenarien, in denen Sie die von TOC Parser oder TOC Generator bereitgestellten Dienste verwenden können.

-   Erstellen Sie ein Inhaltsverzeichnis von Grund auf neu, und betten Sie es in eine Videodatei ein.
-   Liest ein Inhaltsverzeichnis, das bereits in eine Videodatei eingebettet ist.
-   Bearbeiten Sie ein Inhaltsverzeichnis, das bereits in eine Videodatei eingebettet ist.
-   Verwenden Sie den TOC-Generator, um ein Inhaltsverzeichnis basierend auf Szenen und Clustern zu erstellen, die in einer Videodatei erkannt werden. Bearbeiten Sie das generierte Inhaltsverzeichnis, und betten Sie es in die Videodatei ein.
