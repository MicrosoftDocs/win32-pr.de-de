# <a name="overview-of-table-of-contents-parser"></a>Übersicht über den parserinhalts-Parser

Der Inhaltsverzeichnis-Parser (Inhaltsverzeichnis) ist eine Windows-Komponente, mit der Sie erweiterte Indexinformationen in Mediendateien speichern können. Beispiele für erweiterte Indexinformationen sind Inhalts Tabellen, semantische Indizes und Storyboards.

Mit dem Inhaltsverzeichnis können Sie Software schreiben, die es Benutzern ermöglicht, interessante Szenen in einer Videodatei zu finden. Beispielsweise können Sie dem Benutzer ein Inhaltsverzeichnis bereitstellen, damit er die primären Abschnitte des Videos identifizieren und dann einen bestimmten Abschnitt anzeigen kann. Sie können dem Benutzer auch einen Index bereitstellen, sodass er eine Liste von Szenen finden kann, die mit einem bestimmten Wort oder Ausdruck verknüpft sind.

Der Content Generator (Inhalts Generator) ist eine Windows-Komponente, die zusammen mit dem TOC-Parser funktioniert. Der TOC-Generator kann Szenen und Cluster von Szenen in einer Videodatei erkennen und dann ein Inhaltsverzeichnis generieren, das Sie als Ausgangspunkt in der Software verwenden können.

In der folgenden Liste finden Sie einige Szenarien, in denen Sie die Dienste verwenden können, die von TOC-Parser oder TOC-Generator bereitgestellt werden.

-   Erstellen Sie von Grund auf ein Inhaltsverzeichnis, und Betten Sie es in eine Videodatei ein.
-   Dient zum Lesen eines Inhaltsverzeichnisses, das bereits in eine Videodatei eingebettet ist.
-   Bearbeiten Sie ein Inhaltsverzeichnis, das bereits in eine Videodatei eingebettet ist.
-   Verwenden Sie den TOC-Generator, um auf der Grundlage von Szenen und Clustern, die in einer Videodatei erkannt werden, ein Inhaltsverzeichnis zu erstellen. Bearbeiten Sie das generierte Inhaltsverzeichnis, und Betten Sie es in die Videodatei ein.
