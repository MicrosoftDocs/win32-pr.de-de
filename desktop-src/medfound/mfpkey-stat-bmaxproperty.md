# <a name="mfpkey_stat_bmax-property"></a>Mfpkey \_ stat ( \_ bmax-Eigenschaft)

Gibt das maximale Puffer Fenster (in Millisekunden) eines codierten Streams an. Schreibgeschützt.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

**g \_ wszwmvcmaxbitrate**

## <a name="data-type"></a>Datentyp

**VT \_ I4**

## <a name="remarks"></a>Bemerkungen

Um die maximalen Puffer Fenster eines codierten Streams zu ermitteln, lesen Sie diese Eigenschaft am Ende der Codierung.
