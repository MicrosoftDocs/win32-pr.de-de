# <a name="mfpkey_stat_bmax-property"></a>MFPKEY \_ STAT \_ BMAX-Eigenschaft

Gibt das maximale Pufferfenster eines codierten Streams in Millisekunden an. Schreibgeschützt.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

**g \_ wszWMVCMaxBitrate**

## <a name="data-type"></a>Datentyp

**VT \_ I4**

## <a name="remarks"></a>Hinweise

Um die maximalen Pufferfenster eines codierten Streams zu bestimmen, lesen Sie diese Eigenschaft am Ende der Codierung.
