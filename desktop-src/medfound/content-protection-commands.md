# <a name="content-protection-commands"></a>Content Protection-Befehle

Listet die Befehle für die [**IDirect3DAuthenticatedChannel9:: Configure**](/windows/desktop/api/d3d9/nf-d3d9-idirect3dauthenticatedchannel9-configure) -Methode auf.



| Get-Help                                                                                                           | BESCHREIBUNG                                                                                                           |
|-------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**D3DAUTHENTICATEDCONFIGURE \_ initialisieren**](d3dauthenticatedconfigure-initialize.md)                             | Initialisiert den authentifizierten Kanal.                                                                                |
| [**D3DAUTHENTICATEDCONFIGURE \_ CryptoSession**](d3dauthenticatedconfigure-cryptosession.md)                       | Verknüpft eine Kryptografiesitzung mit einem DXVA-2-Decodergerät (DirectX Video Acceleration 2) und einem Direct3D-Gerät. |
| [**D3DAUTHENTICATEDCONFIGURE \_ verschlüsselt**](d3dauthenticatedconfigure-encryptionwhenaccessible.md) | Legt den Verschlüsselungs Grad fest, der vor dem Zugriff auf geschützte Inhalte für die CPU oder den Bus ausgeführt wird.         |
| [**D3DAUTHENTICATEDCONFIGURE \_ Schutz**](d3dauthenticatedconfigure-protection.md)                             | Aktiviert oder deaktiviert den Schutz für das Gerät.                                                                        |
| [**D3DAUTHENTICATEDCONFIGURE \_ SharedResource**](d3dauthenticatedconfigure-sharedresource.md)                     | Ermöglicht einem Prozess, eine freigegebene Ressource zu öffnen, oder deaktiviert das Öffnen von freigegebenen Ressourcen durch einen Prozess.                     |
