# <a name="mfsampleextension_multiplexed_manager-attribute"></a>MFSampleExtension \_ MULTIPLEXED \_ MANAGER-Attribut

Stellt eine Instanz von [**VORMuxStreamSampleManager**](/windows/desktop/api/mfobjects/nn-mfobjects-imfmuxstreamsamplemanager) zur Verfügung, die für den Zugriff auf die Sammlung von Beispielen aus den Unterstreams einer multiplexierten Medienquelle verwendet wird.

## <a name="data-type"></a>Datentyp

**[**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown)**

## <a name="remarks"></a>Hinweise

Übergeben Sie diesen Wert [**an DIE ATTRIBUTEAttributes::GetUnknown,**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-getunknown) um eine Instanz von [**DURCHMuxStreamSampleManager abzurufen.**](/windows/desktop/api/mfobjects/nn-mfobjects-imfmuxstreamsamplemanager)
