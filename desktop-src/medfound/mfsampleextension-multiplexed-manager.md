# <a name="mfsampleextension_multiplexed_manager-attribute"></a>MF SampleExtension- \_ multiplexed- \_ Manager-Attribut

Stellt eine Instanz von [**imfmuxstreamsamplemanager**](/windows/desktop/api/mfobjects/nn-mfobjects-imfmuxstreamsamplemanager) bereit, die für den Zugriff auf die Auflistung von Beispielen aus den unter strömen einer multiplext-Medienquelle verwendet wird.

## <a name="data-type"></a>Datentyp

**[**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown)**

## <a name="remarks"></a>Bemerkungen

Übergeben Sie diesen Wert an [**imfattributes:: getunknown**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-getunknown) , um eine Instanz von [**imfmuxstreamsamplemanager**](/windows/desktop/api/mfobjects/nn-mfobjects-imfmuxstreamsamplemanager)zu erhalten.
