# <a name="codecapi_avdecsoftwaredynamicformatchange-property"></a>Codecapi \_ avdecsoftwaredynamicformatchange (Eigenschaft)

Legt fest, ob die dynamische Änderung von Software für die interne Größe verwendet werden soll.

## <a name="data-type"></a>Datentyp

**BOOL**

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avdecsoftwaredynamicformatchange**

## <a name="property-value"></a>Eigenschaftswert

Wenn diese Einstellung auf " **true**" festgelegt ist, wird die Änderung des dynamischen Formats für die interne Größe verwendet.
