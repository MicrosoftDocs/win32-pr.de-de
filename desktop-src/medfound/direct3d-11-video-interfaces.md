# <a name="direct3d-11-video-interfaces"></a>Direct3D 11-Video Schnittstellen

Dieser Abschnitt enthält Referenzinformationen für die Video Schnittstellen von Microsoft Direct3D 11.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                 | BESCHREIBUNG                                                                                              |
|---------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [**ID3D11AuthenticatedChannel**](/windows/desktop/api/d3d11/nn-d3d11-id3d11authenticatedchannel)<br/>           | Stellt einen Kommunikationskanal mit dem Grafiktreiber oder der Microsoft Direct3D-Laufzeit bereit. <br/> |
| [**ID3D11CryptoSession**](/windows/desktop/api/d3d11/nn-d3d11-id3d11cryptosession)<br/>                         | Stellt eine Kryptografiesitzung dar. <br/>                                                          |
| [**ID3D11VideoContext**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videocontext)<br/>                           | Bietet die Videofunktionen eines Direct3D 11-Geräts. <br/>                                    |
| [**ID3D11VideoContext1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videocontext1)<br/>                         | Bietet die Videofunktionen eines Direct3D 11-Geräts.<br/>                                     |
| [**ID3D11VideoContext2**](/windows/win32/api/d3d11_4/nn-d3d11_4-id3d11videocontext2)<br/>                         | Bietet die Videofunktionen eines Direct3D 11-Geräts. <br/>                                    |
| [**ID3D11VideoDecoder**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodecoder)<br/>                           | Stellt einen Hardware beschleunigten Video Decoder für Direct3D 11 dar.<br/>                              |
| [**ID3D11VideoDecoderOutputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodecoderoutputview)<br/>       | Gibt die Ausgabe Oberflächen an, auf die während der Video Decodierung zugegriffen werden kann.<br/>                    |
| [**ID3D11VideoDevice**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodevice)<br/>                             | Bietet die videodecodierungs-und Video Verarbeitungsfunktionen eines Direct3D 11-Geräts. <br/>       |
| [**ID3D11VideoDevice1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videodevice1)<br/>                           | Bietet die videodecodierungs-und Video Verarbeitungsfunktionen eines Direct3D 11-Geräts. <br/>       |
| [**ID3D11VideoProcessor**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessor)<br/>                       | Stellt einen Videoprozessor für Direct3D 11 dar.<br/>                                                 |
| [**ID3D11VideoProcessorEnumerator**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessorenumerator)<br/>   | Listet die Videoprozessor Funktionen eines Direct3D 11-Geräts auf. <br/>                         |
| [**ID3D11VideoProcessorEnumerator1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videoprocessorenumerator1)<br/> | Listet die Videoprozessor Funktionen eines Direct3D 11-Geräts auf.<br/>                          |
| [**ID3D11VideoProcessorInputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessorinputview)<br/>     | Identifiziert die Eingabe Oberflächen, auf die während der Videoverarbeitung zugegriffen werden kann.<br/>                   |
| [**ID3D11VideoProcessorOutputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessoroutputview)<br/>   | Gibt die Ausgabe Oberflächen an, auf die während der Videoverarbeitung zugegriffen werden kann.<br/>                  |
