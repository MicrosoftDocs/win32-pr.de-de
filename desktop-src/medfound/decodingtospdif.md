# <a name="decoding-to-spdif-microsoft-media-foundation"></a>Decodierung in S/PDIF (Microsoft Media Foundation)

Der Klassen Bezeichner für das Decoder-Objekt, das Windows Media Audio 9,1 Professional-Inhalt für die Übermittlung über das digitale Interconnect-Format (S/PDIF) von Sony transformiert, ist CLSID \_ cwmaudiospdtxdmo. Sie verwenden dieses Objekt wie jeden anderen Decoder. Sie unterstützt die gleichen Eigenschaften wie der Windows Media Audio 9-Decoder. Die von diesem Decoder aufgelisteten Ausgabetypen sind alle S/PDIF-Formate.

Damit die S/PDIF-Ausgabe vollständig unterstützt wird, muss auf dem Client Computer eine spezielle Hardware installiert sein.
