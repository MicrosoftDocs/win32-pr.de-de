# <a name="decoding-to-spdif-microsoft-media-foundation"></a>Decodieren in S/PDIF (Microsoft Media Foundation)

Der Klassenbezeichner für das Decoderobjekt, das Windows Media Audio 9.1 Professional-Inhalt für die Übermittlung über das Format für digitale Verbindungen (S/PDIF) transformiert, ist CLSID \_ CWMAudioSpdTxDMO. Sie verwenden dieses -Objekt wie jeden anderen Decoder. Es unterstützt die gleichen Eigenschaften wie der Windows Media Audio 9-Decoder. Die von diesem Decoder aufzählten Ausgabetypen sind alle S/PDIF-Formate.

Um die S/PDIF-Ausgabe vollständig zu unterstützen, muss auf dem Clientcomputer eine spezielle Hardware installiert sein.
