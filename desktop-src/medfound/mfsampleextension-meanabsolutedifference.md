# <a name="mfsampleextension_meanabsolutedifference-attribute"></a>MFSampleExtension \_ MeanAbsoluteDifference-Attribut

Dieses Attribut gibt den mittleren absoluten Unterschied (Mean Absolute Difference, MAD) für alle Makroblöcke in der Y-Ebene zurück.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Hinweise

Die Anwendung kann den Encoder anfordern, um dieses Attribut für Ausgabebeispiele mithilfe von [CODECAPI \_ AVEncVideoMeanAbsoluteDifference](codecapi-avencvideomeanabsolutedifference.md)festzulegen.
