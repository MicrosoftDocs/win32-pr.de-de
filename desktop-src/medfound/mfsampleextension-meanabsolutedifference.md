# <a name="mfsampleextension_meanabsolutedifference-attribute"></a>Mnesampleextension- \_ Attribut "meanabsolutedifference"

Dieses Attribut gibt den Mean absolute Difference (Mad) über alle Makroblöcke in der Y-Ebene zurück.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Bemerkungen

Die Anwendung kann den Encoder zum Festlegen dieses Attributs für Ausgabe Beispiele mithilfe von [codecapi \_ avencvideomeanabsolutedifference](codecapi-avencvideomeanabsolutedifference.md)anfordern.
