# <a name="mf_device_thermal_state_changed-attribute"></a>\_ \_ \_ \_ Geändertes Attribut des MF-Geräts

Stellt ein Ereignis dar, das eine Änderung des thermischen Zustands im Gerät signalisiert.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Bemerkungen

Der Wert dieses Attributs gibt den thermischen Wert des Geräts in Prozent an.
