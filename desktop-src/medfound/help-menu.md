# <a name="help-menu"></a>Hilfemenü

Das **Menü** Hilfe enthält die folgenden Befehle:



| Befehl      | BESCHREIBUNG                                                                      |
|--------------|----------------------------------------------------------------------------------|
| **Hilfe...**  | Öffnet die TopoEdit-Hilfe auf der MSDN-Website.                                         |
| **Über...** | Öffnet das **Dialogfeld Info,** in dem Versionsinformationen zu TopoEdit angezeigt werden. |
