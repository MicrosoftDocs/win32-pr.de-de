# <a name="why-does-the-audio-encoder-report-no-supported-input-types"></a>Warum meldet der Audioencoder keine unterstützten Eingabetypen?

Der Audioencoder kann die unterstützten Eingabetypen erst ermitteln, wenn Sie einen Ausgabetyp festgelegt haben. Zählen Sie die Ausgabetypen auf, und wählen Sie den Wert aus, der Ihren Anforderungen entspricht. Nachdem Sie den Ausgabetyp festgelegt haben, gibt das Objekt unterstützte Eingabetypen zurück.
