# <a name="why-does-the-audio-encoder-report-no-supported-input-types"></a>Warum meldet der Audioencoder keine unterstützten Eingabetypen?

Der Audioencoder kann die unterstützten Eingabetypen erst bestimmen, wenn Sie einen Ausgabetyp festgelegt haben. Aufzählen Sie die Ausgabetypen, und wählen Sie den Typ aus, der Ihren Anforderungen entspricht. Nachdem Sie den Ausgabetyp festgelegt haben, gibt das Objekt unterstützte Eingabetypen zurück.
