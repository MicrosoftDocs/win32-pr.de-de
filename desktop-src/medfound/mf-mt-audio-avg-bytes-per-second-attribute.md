# <a name="mf_mt_audio_avg_bytes_per_second-attribute"></a>Attribut "MF \_ MT \_ -audiodurchschn. \_ \_ Byte \_ pro \_ Sekunde"

Die durchschnittliche Anzahl von Bytes pro Sekunde in einem audiomedientyp.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut entspricht dem **navgbytespersec** -Member der [**WaveFormatEx**](/previous-versions/dd757713(v=vs.85)) -Struktur.

Die GUID-Konstante für dieses Attribut wird aus "mfuuid. lib" exportiert.
