# <a name="mfpkey_stat_bavg-property"></a>Mfpkey \_ stat- \_ bavg (Eigenschaft)

Gibt das durchschnittliche Puffer Fenster (in Millisekunden) eines codierten Streams an. Schreibgeschützt.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

**g \_ wszwmvcbavg**

## <a name="data-type"></a>Datentyp

**VT \_ I4**

## <a name="remarks"></a>Bemerkungen

Um das durchschnittliche Puffer Fenster eines codierten Streams zu ermitteln, lesen Sie diese Eigenschaft am Ende der Codierung.
