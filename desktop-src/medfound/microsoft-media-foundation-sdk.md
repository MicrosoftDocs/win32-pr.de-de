# <a name="microsoft-media-foundation"></a>Microsoft Media Foundation

Microsoft Media Foundation ermöglicht die Entwicklung von Anwendungen und Komponenten für die Verwendung digitaler Medien unter Windows Vista und höher.

Media Foundation ist die Multimediaplattform der nächsten Generation für Windows, mit der Entwickler, Consumer und Inhaltsanbieter die neue Welle von Premium-Inhalten mit verbesserter Stabilität, einzigartiger Qualität und nahtloser Interoperabilität nutzen können.
