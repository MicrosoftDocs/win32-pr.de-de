# <a name="microsoft-media-foundation"></a>Microsoft Media Foundation

Microsoft Media Foundation ermöglicht die Entwicklung von Anwendungen und Komponenten für die Verwendung digitaler Medien auf Windows Vista und höher.

Media Foundation ist die Multimediaplattform der nächsten Generation für Windows, die Es Entwicklern, Kunden und Inhaltsanbietern ermöglicht, die neue Welle von Premium-Inhalten mit erhöhter Stabilität, erstklassiger Qualität und nahtloser Interoperabilität zu nutzen.
