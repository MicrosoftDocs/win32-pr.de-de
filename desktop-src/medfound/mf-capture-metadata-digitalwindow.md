# <a name="mf_capture_metadata_digitalwindow-attribute"></a>DIGITALWINDOW-Attribut für MF \_ \_ CAPTURE METADATA \_

Ein -Wert, der die Begrenzungseinstellungen des digitalen Fensters für die Videoaufnahme darstellt.

## <a name="data-type"></a>Datentyp

**Blob**

## <a name="remarks"></a>Hinweise

Der **BLOB-Wert** dieser Eigenschaft ist eine [DigitalWindowSetting-Struktur.](/windows/win32/api/mfapi/ns-mfapi-digitalwindowsetting)
