# <a name="mfpkey_stat_rmax-property"></a>Mfpkey \_ stat \_ Rmax (Eigenschaft)

Gibt die maximale Bitrate eines codierten Streams in Bits pro Sekunde an. Schreibgeschützt.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

**g \_ wszwmvcmaxbitrate**

## <a name="data-type"></a>Datentyp

**VT \_ I4**

## <a name="remarks"></a>Bemerkungen

Um die maximale Bitrate eines codierten Streams zu ermitteln, lesen Sie diese Eigenschaft am Ende der Codierung.
