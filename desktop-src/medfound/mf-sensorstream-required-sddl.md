# <a name="mf_sensorstream_required_sddl-attribute"></a>MF \_ SENSORSTREAM \_ REQUIRED \_ SDDL-Attribut

Dieses Attribut wird verwendet, um eine Sicherheitsdeskriptordefinitionssprache (Security Descriptor Definition Language, SDDL) im Stream anzugeben, um differenzierte Zugriffsrechte für einen bestimmten Sensor anzugeben.

## <a name="data-type"></a>Datentyp

**Wchar\***
