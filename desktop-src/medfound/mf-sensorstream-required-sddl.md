# <a name="mf_sensorstream_required_sddl-attribute"></a>Für das MF- \_ sensorstream \_ benötigtes \_ SDDL-Attribut

Dieses Attribut wird verwendet, um eine Sicherheits Beschreibung (Security Descriptor Definition Language, SDDL) für den Stream anzugeben, um differenzierte Zugriffsrechte für einen bestimmten Sensor anzugeben.
