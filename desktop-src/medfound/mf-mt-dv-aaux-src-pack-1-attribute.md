# <a name="mf_mt_dv_aaux_src_pack_1-attribute"></a>MF \_ MT \_ DV \_ Aaux \_ src \_ Pack \_ 1-Attribut

Audiohilfsquelle (Aaux) Source Pack für den zweiten Audioblock in einem Typ digitaler Video (DV).

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut entspricht dem **dwDVAAuxSrc1** -Member der DirectShow [**dvinfo**](/windows/win32/api/strmif/ns-strmif-dvinfo) -Struktur.

Die GUID-Konstante für dieses Attribut wird aus "mfuuid. lib" exportiert.
