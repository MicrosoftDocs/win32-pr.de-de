# <a name="mf_mt_dv_aaux_src_pack_1-attribute"></a>MF \_ MT \_ DV \_ AAUX \_ SRC PACK \_ \_ 1-Attribut

AAUX-Quellpaket (Audio Auxiliary) für den zweiten Audioblock in einem Medientyp für digitale Videos (DV).

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Hinweise

Dieses Attribut entspricht dem **dwDVAAuxSrc1-Member** der [**DirectShow DVINFO-Struktur.**](/windows/win32/api/strmif/ns-strmif-dvinfo)

Die GUID-Konstante für dieses Attribut wird aus mfuuid.lib exportiert.
