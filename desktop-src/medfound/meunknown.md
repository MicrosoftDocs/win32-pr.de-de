# <a name="meunknown-event"></a>Meunknown-Ereignis

Unbekannter Ereignistyp. Sie können diesen Wert verwenden, um Variablen des Typs **mediaeventtype** zu initialisieren, aber eine Komponente sollte nie das meunknown-Ereignis erhöhen.

## <a name="event-values"></a>Ereigniswerte

Mögliche Werte, die von [**imfmediaevent:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) abgerufen werden, sind folgende.



| VARTYPE              | BESCHREIBUNG                           |
|----------------------|---------------------------------------|
| VT \_ leer<br/> | Keine Ereignisdaten.<br/> <br/> |
