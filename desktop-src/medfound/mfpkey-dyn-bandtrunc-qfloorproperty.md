# <a name="mfpkey_dyn_bandtrunc_qfloor-property"></a>Mfpkey \_ dyn \_ bandtrunc- \_ qfloor (Eigenschaft)

Gibt die Qualität an, mit der die maximale codierte Bandbreite zulässig ist. Wenn die Qualität besser als oder gleich dem Wert dieser Eigenschaft ist, ist die maximale codierte Bandbreite zulässig.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

Nur mit [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore)verfügbar.

## <a name="data-type"></a>Datentyp

**VT \_ R4**
