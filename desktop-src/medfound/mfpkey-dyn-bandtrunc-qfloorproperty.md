# <a name="mfpkey_dyn_bandtrunc_qfloor-property"></a>MFPKEY \_ DYN \_ BANDTRUNC \_ QFLOOR-Eigenschaft

Gibt die Qualität an, mit der die maximale codierte Bandbreite zulässig ist. Bei einer Qualität, die größer oder gleich dem Wert dieser Eigenschaft ist, ist die maximale codierte Bandbreite zulässig.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

Nur mithilfe von [**IPropertyStore verfügbar.**](/windows/win32/api/propsys/nn-propsys-ipropertystore)

## <a name="data-type"></a>Datentyp

**VT \_ R4**
