# <a name="codecapi_avench264cabacenable-property"></a>Codecapi \_ AVEncH264CABACEnable Eigenschaft

Aktiviert oder deaktiviert CABAC (Kontext Adaptive binäre binäre Codierung) für die H. 264-Entropie Codierung.

## <a name="data-type"></a>Datentyp

**Variant \_ bool** (VT \_ bool)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ AVEncH264CABACEnable**

## <a name="property-value"></a>Eigenschaftswert

Wenn der Wert **Variant \_ true** ist, ist CABAC aktiviert.
