# <a name="file-menu"></a>Menü „Datei“

Das Menü **Datei** enthält die folgenden Befehle:



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>Get-Help</th>
<th>BESCHREIBUNG</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Neu...</strong></td>
<td>Löscht die vorhandene Topologie und entfernt topologieknoten aus dem <strong>Topologiebereich</strong>.<br/>
<ul>
<li>Wenn eine Topologie momentan geöffnet ist, wird der Benutzer aufgefordert, die Topologie zu speichern. Informationen zum Speichern der Topologie finden Sie unter <a href="saving-and-loading-topologies.md">Speichern und Laden von Topologien</a>.<br/></li>
<li>Erstellt eine leere Topologie. Der Benutzer kann der neuen Topologie topologieknoten hinzufügen. Weitere Informationen finden Sie unter <a href="building-topologies-by-using-topoedit.md">Building Topologien using topoedit</a>.<br/></li>
</ul></td>
</tr>
<tr class="even">
<td><strong>Datei Rendering...</strong></td>
<td>Erstellt eine Wiedergabe Topologie für eine Mediendatei. Mit topoedit werden die Knoten in der Topologie automatisch erstellt und verbunden. Weitere Informationen finden Sie unter <a href="building-a-playback-topology-with-topoedit.md">aufbauen einer Wiedergabe Topologie mit topoedit</a>.</td>
</tr>
<tr class="odd">
<td><strong>Transcode Rendering...</strong></td>
<td>Erstellt eine Transcodierungs Topologie. Weitere Informationen finden Sie unter <a href="building-a-transcode-topology-with-topoedit.md">aufbauen einer transcode-Topologie mit topoedit</a>.<br/>
<blockquote>
[!Note]<br />
Für dieses Feature ist Windows 7 erforderlich.
</blockquote>
<br/></td>
</tr>
<tr class="even">
<td><strong>Öffnen Sie...</strong></td>
<td>Lädt eine vorhandene Topologie, die als XML-Datei gespeichert ist. Weitere Informationen finden Sie unter <a href="saving-and-loading-topologies.md">Speichern und Laden von Topologien</a>.</td>
</tr>
<tr class="odd">
<td><strong>Speichern...</strong></td>
<td>Speichert die aktuell geöffnete Topologie als XML-Datei. Weitere Informationen finden Sie unter <a href="saving-and-loading-topologies.md">Speichern und Laden von Topologien</a>.</td>
</tr>
<tr class="even">
<td><strong>Beenden</strong></td>
<td>Schließt die topoedit-Anwendung.</td>
</tr>
</tbody>
</table>
