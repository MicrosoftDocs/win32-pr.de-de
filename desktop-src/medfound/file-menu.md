# <a name="file-menu"></a>Menü „Datei“

Das **Menü** Datei enthält die folgenden Befehle:




| Befehl | BESCHREIBUNG | 
|---------|-------------|
| <strong>Neu...</strong> | Clears the existing topology and removes topology nodes from <strong>the Topology Pane</strong>.<br /><ul><li>Wenn eine Topologie derzeit geöffnet ist, fordert den Benutzer auf, die Topologie zu speichern. Informationen zum Speichern der Topologie finden Sie unter <a href="saving-and-loading-topologies.md">Speichern und Laden von Topologien.</a><br /></li><li>Erstellt eine leere Topologie. Der Benutzer kann der neuen Topologie Topologieknoten hinzufügen. Weitere Informationen finden Sie unter <a href="building-topologies-by-using-topoedit.md">Building Topologies by Using TopoEdit (Erstellen von Topologien mithilfe von TopoEdit).</a><br /></li></ul> | 
| <strong>Datei rendern...</strong> | Erstellt eine Wiedergabetopologie für eine Mediendatei. TopoEdit erstellt und verbindet die Knoten in der Topologie automatisch. Weitere Informationen finden Sie unter <a href="building-a-playback-topology-with-topoedit.md">Erstellen einer Wiedergabetopologie mit TopoEdit</a>. | 
| <strong>Render Transcode...</strong> | Erstellt eine Transcodierungstopologie. Weitere Informationen finden Sie unter <a href="building-a-transcode-topology-with-topoedit.md">Erstellen einer Transcodierungstopologie mit TopoEdit</a>.<br /><blockquote>[!Note]<br />Dieses Feature erfordert Windows 7.</blockquote><br /> | 
| <strong>Öffnen...</strong> | Lädt eine vorhandene Topologie, die als XML-Datei gespeichert ist. Weitere Informationen <a href="saving-and-loading-topologies.md">finden Sie unter Speichern und Laden von Topologien.</a> | 
| <strong>Speichern...</strong> | Speichert die derzeit geöffnete Topologie als XML-Datei. Weitere Informationen <a href="saving-and-loading-topologies.md">finden Sie unter Speichern und Laden von Topologien.</a> | 
| <strong>Beenden</strong> | Schließt die TopoEdit-Anwendung. |
