# <a name="protectedplayback-sample"></a>Protectedplayback-Beispiel

Zeigt, wie geschützte Medieninhalte in Microsoft Media Foundation wiedergegeben werden.

## <a name="usage"></a>Verbrauch

Das protectedplayback-Beispiel erstellt eine Windows-Anwendung.

Um eine lokale Mediendatei wiederzugeben, wählen Sie im Menü **Datei** die Option **Datei öffnen** aus. Um eine Datei per URL anzugeben, wählen Sie im Menü **Datei** die Option **URL öffnen** aus. Nachdem Sie eine Datei ausgewählt haben, beginnt die Wiedergabe automatisch. Um die Wiedergabe anzuhalten, drücken Sie die Leertaste. Um die Wiedergabe fortzusetzen, drücken Sie erneut die Leertaste.

## <a name="apis-demonstrated"></a>Gezeigte APIs

In diesem Beispiel werden die folgenden Media Foundation Schnittstellen veranschaulicht:

-   [**IMF contentenabler**](/windows/desktop/api/mfidl/nn-mfidl-imfcontentenabler)
-   [**IMF contentprotection Manager**](/windows/desktop/api/mfidl/nn-mfidl-imfcontentprotectionmanager)
