# <a name="protectedplayback-sample"></a>ProtectedPlayback-Beispiel

Zeigt, wie geschützte Medieninhalte in Microsoft Media Foundation wiedergegeben werden.

## <a name="usage"></a>Verwendung

Im ProtectedPlayback-Beispiel wird eine Windows Anwendung erstellt.

Wählen Sie im Menü **Datei** die Option **Datei öffnen** aus, um eine lokale Mediendatei wiederzuspielen. Wählen Sie im Menü Datei die Option **URL öffnen** aus, um eine **Datei** nach URL anzugeben. Nachdem Sie eine Datei ausgewählt haben, beginnt die Wiedergabe automatisch. Um die Wiedergabe anzuhalten, drücken Sie die LEERTASTE. Drücken Sie erneut die LEERTASTE, um die Wiedergabe fortzusetzen.

## <a name="apis-demonstrated"></a>Gezeigte APIs

In diesem Beispiel werden die folgenden Media Foundation Schnittstellen veranschaulicht:

-   [**CONTENTContentEnabler**](/windows/desktop/api/mfidl/nn-mfidl-imfcontentenabler)
-   [**CONTENTContentProtectionManager**](/windows/desktop/api/mfidl/nn-mfidl-imfcontentprotectionmanager)
