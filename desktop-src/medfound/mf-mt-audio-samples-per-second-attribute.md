# <a name="mf_mt_audio_samples_per_second-attribute"></a>MF \_ MT AUDIO SAMPLES PER \_ \_ \_ \_ SECOND-Attribut

Anzahl von Audiobeispielen pro Sekunde in einem Audiomedientyp.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut entspricht dem **nSamplesPerSec-Member** der [**WAVEFORMATEX-Struktur.**](/previous-versions/dd757713(v=vs.85))

Die GUID-Konstante für dieses Attribut wird aus mfuuid.lib exportiert.
