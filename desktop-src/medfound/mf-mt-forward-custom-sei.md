# <a name="mf_mt_forward_custom_sei-attribute"></a>MF \_ MT FORWARD CUSTOM \_ \_ \_ SEI-Attribut

Gibt an, dass DIE-Einheitentypen (Supplemental Enhancement Information) an Ausgabebeispiele vom Decoder weitergeleitet werden sollen.

## <a name="data-type"></a>Datentyp

**UINT32**

## <a name="remarks"></a>Hinweise

Wenn der Decoder eine SEI analysiert, wird sie nicht weitergeleitet.
