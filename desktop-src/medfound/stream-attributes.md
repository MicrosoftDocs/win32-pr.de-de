# <a name="stream-attributes"></a>Streamattribute

Die folgenden Attribute gelten für Mediendaten Ströme. Um die Attribute aus einem Medien Beispiel zu erhalten, verwenden Sie die [**imfmediasourceex:: getstreamattributormethode**](/windows/desktop/api/mfidl/nf-mfidl-imfmediasourceex-getstreamattributes) .



| Attribut                                                                                   | BESCHREIBUNG                                   |
|---------------------------------------------------------------------------------------------|-----------------------------------------------|
| [MF-Erweiterung " \_ cameraextrinsics"](mfstreamextension-cameraextrinsics.md)               | Die Kamera ist extra insics für den Datenstrom.         |
| [MF-Erweiterung \_ pinholecameraintrinsics](mfstreamextension-pinholecameraintrinsics.md) | Die systeminternen Funktionen der pinloch Kamera für den Datenstrom. |



 

Nicht jeder Mediendaten Strom enthält jedes hier aufgeführte Attribut. In einigen Fällen gilt ein Attribut nur für bestimmte Arten von Daten.
