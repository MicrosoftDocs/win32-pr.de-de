# <a name="topoedit-main-window"></a>Topoedit-Hauptfenster

Nach dem Starten von topoedit wird das im folgenden Screenshot angezeigte Fenster angezeigt.

![Screenshot mit dem Hauptfenster "topoedit"](images/809bce6e-43b7-4399-a41b-deff894ba151.jpg)

Die Hauptfenster Elemente von topoedit lauten:

-   Bereich für die **Topologie**. Wird auf der linken Seite des Fensters angezeigt. Dieser Bereich wird verwendet, um topologieknoten hinzuzufügen, eine Topologie anzuzeigen und die topologieknoten zu verbinden. Weitere Informationen zur Verwendung dieses Bereichs finden Sie unter [Building Topologien using topoedit](building-topologies-by-using-topoedit.md).
-   Bereich **Attribute**. Wird auf der rechten Seite des Fensters angezeigt. In diesem Bereich werden die Attribute des aktuell ausgewählten topologieknotens angezeigt. Informationen zu den Attributen, die in diesem Bereich angezeigt werden, finden Sie unter [Anzeigen von Topologieinformationen](viewing-topology-information.md).
-   Kreiert. Siehe [topoedit-Menüs](topoedit-menus.md).
-   Symbolleiste Siehe [topoedit Toolbar](topoedit-toolbar.md).
