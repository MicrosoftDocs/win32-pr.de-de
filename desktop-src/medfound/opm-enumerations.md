# <a name="opm-enumerations"></a>OPM-Enumerationen

Der [Output Protection Manager](output-protection-manager.md) (OPM) verwendet die folgenden Konstanten und Enumerationstypen.



| Enumeration                                                                    | Beschreibung                                                                                                                                                                             |
|--------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**OPM-bustyps**](opm-bus-type-flags.md)                               | Gibt den Typ des e/a-Busses an, der vom Grafikadapter verwendet wird.                                                                                                                             |
| [**OPM-Connector-Typflags**](opm-connector-type-flags.md)                   | Gibt den Typ des physischen Verbindungs Typs für eine Videoausgabe an.                                                                                                                            |
| [**OPM-Schutztyp Flags**](opm-protection-type-flags.md)                 | Gibt die Ausgabe Schutzmechanismen für den Output Protection Manager (OPM) an.                                                                                                             |
| [**OPM-Statusflags**](opm-status-flags.md)                                   | Gibt den Status einer OPM-Sitzung an.                                                                                                                                                 |
| [**OPM \_ -ACP- \_ Schutz \_ Ebene**](/windows/desktop/api/opmapi/ne-opmapi-opm_acp_protection_level)                | Gibt die Schutz Ebene für den Schutz des analogen Kopierens (ACP) an.                                                                                                                        |
| [**OPM \_ dpcp- \_ Schutz \_ Ebene**](/windows/desktop/api/opmapi/ne-opmapi-opm_dpcp_protection_level)              | Gibt die Schutz Ebene für Display Port Content Protection (dpcp) an.                                                                                                               |
| [**OPM- \_ HDCP- \_ Schutz \_ Ebene**](/windows/desktop/api/opmapi/ne-opmapi-opm_hdcp_protection_level)              | Gibt die Schutz Ebene für High-Bandwidth Digital Content Protection (HDCP) an.                                                                                                    |
| [**Seitenverhältnis des OPM- \_ Bilds \_ \_ \_ EN300294**](/windows/desktop/api/opmapi/ne-opmapi-opm_image_aspect_ratio_en300294) | Gibt das Seitenverhältnis für ETSI EN 300 294 an.                                                                                                                                         |
| [**OPM- \_ Video \_ Ausgabe \_ Semantik**](/windows/desktop/api/opmapi/ne-opmapi-opm_video_output_semantics)            | Gibt an, ob die [**IOPMVideoOutput**](/windows/desktop/api/opmapi/nn-opmapi-iopmvideooutput) -Schnittstelle eine OPM-Semantik (Output Protection Manager) oder COPP-Semantik (Certified Output Protection Protocol) verwendet. |
