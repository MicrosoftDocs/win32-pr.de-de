# <a name="mfpkey_fullframerate-property"></a>Mfpkey \_ fullframerate (Eigenschaft)

Gibt an, ob der endoder Frames löschen darf.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

**g \_ wszwmvcfullframerate**

## <a name="data-type"></a>Datentyp

**VT \_ I4**

## <a name="remarks"></a>Bemerkungen

Wenn Sie diese Eigenschaft auf 1 festlegen, werden die Frames vom Encoder nicht gelöscht. Wenn Sie diese Eigenschaft auf 0 festlegen, kann der Encoder Frames löschen.
