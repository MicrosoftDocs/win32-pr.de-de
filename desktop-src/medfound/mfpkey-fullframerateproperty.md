# <a name="mfpkey_fullframerate-property"></a>MFPKEY \_ FULLFRAMERATE-Eigenschaft

Gibt an, ob der Literator Frames löschen darf.

## <a name="constant-for-ipropertybag"></a>Konstante für IPropertyBag

**g \_ wszWMVCFullFrameRate**

## <a name="data-type"></a>Datentyp

**VT \_ I4**

## <a name="remarks"></a>Hinweise

Wenn Sie diese Eigenschaft auf 1 festlegen, werden vom Encoder keine Frames gelöscht. Wenn Sie diese Eigenschaft auf 0 festlegen, kann der Encoder Frames löschen.
