# <a name="remoteapp-classes"></a>RemoteApp-Klassen

Der RemoteApp-WMI-Anbieter stellt die folgenden Klassen bereit.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Win32 \_ rdallowlistfile Association**](win32-rdallowlistfileassociation.md)
</dt> <dd>

Beschreibt eine veröffentlichte Dateityp Zuordnung zu einer RemoteApp.

</dd> <dt>

[**Win32 \_ rdfileassociation**](win32-rdfileassociation.md)
</dt> <dd>

Stellt eine Dateityp Zuordnung für eine RemoteApp dar.

</dd> <dt>

[**Win32 \_ remoteappchangeevent**](win32-remoteappchangeevent.md)
</dt> <dd>

Stellt eine Änderung an Windows Server 2008 R2 RemoteApp-Einstellungen auf dem Remotedesktop-Sitzungshost-Server (RD-Sitzungshost) dar.

</dd> <dt>

[**Win32- \_ applicationfileextensions**](win32-tsapplicationfileextensions.md)
</dt> <dd>

die Dateinamen Erweiterungen, die von einer Anwendung auf einem RD-Sitzungshost Server behandelt werden.

</dd> <dt>

[**Win32-terminalbereitungslizenzierung \_**](win32-tsdeploymentlicensing.md)
</dt> <dd>

Lizenzierungs Einstellungen für die Bereitstellung von Remotedesktop Virtualisierung (RDV).

</dd> <dt>

[**Win32-Datei- \_ deploymentsettings**](win32-tsdeploymentsettings.md)
</dt> <dd>

Definiert die Standardeinstellungen, die der RemoteApp-Manager verwendet, wenn Remotedesktopprotokoll (RDP)-Dateien erstellt werden.

</dd> <dt>

[**Win32 \_ tsexpandumgebungs variables**](win32-tsexpandenvironmentvariables.md)
</dt> <dd>

Erweitert System definierte Umgebungsvariablen.

</dd> <dt>

[**Win32-Zeit \_ Ticon**](win32-tsgeticon.md)
</dt> <dd>

Gibt den Inhalt des Symbols zurück, das durch den Dateipfad und Index angegeben wird.

</dd> <dt>

[**Win32- \_ tspublishedapplikation**](win32-tspublishedapplication.md)
</dt> <dd>

Definiert die Anwendungen, die für die Remote Verwendung über Windows Server 2008 R2 RemoteApp verfügbar gemacht werden.

</dd> <dt>

[**Win32 \_ tspublishedapplicationlist**](win32-tspublishedapplicationlist.md)
</dt> <dd>

Stellt die Liste der Programme dar, die in der Liste der RemoteApp-Programme auf einem RD-Sitzungshost Server enthalten sind.

</dd> <dt>

[**Win32 \_ -Remotedesktop**](win32-tsremotedesktop.md)
</dt> <dd>

Beschreibt eine Remote Desktop Verbindung, die über Remotedesktop Webzugriff (RD Webzugriff) verfügbar ist.

</dd> <dt>

[**Win32- \_ App-startmenuapplication**](win32-tsstartmenuapplication.md)
</dt> <dd>

Beschreibt die Anwendungen, die sich im Startmenü eines RD-Sitzungshost Servers befinden.

</dd> <dt>

[**Win32- \_ tssysteminfo**](win32-tssysteminfo.md)
</dt> <dd>

Stellt den zentralisierten Veröffentlichungs Server-Informationsdienst dar.

</dd> </dl>
