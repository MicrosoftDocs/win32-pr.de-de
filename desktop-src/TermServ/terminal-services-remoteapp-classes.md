# <a name="remoteapp-classes"></a>RemoteApp-Klassen

Der RemoteApp-WMI-Anbieter stellt die folgenden Klassen bereit.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Win32 \_ RDAllowListFileAssociation**](win32-rdallowlistfileassociation.md)
</dt> <dd>

Beschreibt eine veröffentlichte Dateitypzuordnung mit einer RemoteApp.

</dd> <dt>

[**Win32 \_ RDFileAssociation**](win32-rdfileassociation.md)
</dt> <dd>

Stellt eine Dateitypzuordnung für eine RemoteApp dar.

</dd> <dt>

[**Win32 \_ RemoteAppChangeEvent**](win32-remoteappchangeevent.md)
</dt> <dd>

Stellt eine Änderung an Windows Server 2008 R2-RemoteApp-Einstellungen auf dem Remotedesktop-Sitzungshost -Server (RD-Sitzungshost) dar.

</dd> <dt>

[**Win32 \_ TSApplicationFileExtensions**](win32-tsapplicationfileextensions.md)
</dt> <dd>

die Dateinamenerweiterungen, die von einer Anwendung auf einem RD-Sitzungshost-Server behandelt werden.

</dd> <dt>

[**Win32 \_ TSDeploymentLicensing**](win32-tsdeploymentlicensing.md)
</dt> <dd>

Lizenzierungs-Einstellungen für die Bereitstellung von Remotedesktop Virtualization (RDV).

</dd> <dt>

[**Win32 \_ TSDeploymentSettings**](win32-tsdeploymentsettings.md)
</dt> <dd>

Definiert die Standardeinstellungen, die der RemoteApp-Manager beim Erstellen von RDP-Dateien (Remotedesktopprotokoll) verwendet.

</dd> <dt>

[**Win32 \_ TSExpandEnvironmentVariables**](win32-tsexpandenvironmentvariables.md)
</dt> <dd>

Erweitert systemdefinierte Umgebungsvariablen.

</dd> <dt>

[**Win32 \_ TSGetIcon**](win32-tsgeticon.md)
</dt> <dd>

Gibt den Inhalt des Durch Dateipfad und Index angegebenen Symbols zurück.

</dd> <dt>

[**Win32 \_ TSPublishedApplication**](win32-tspublishedapplication.md)
</dt> <dd>

Definiert die Anwendungen, die über Windows Server 2008 R2 RemoteApp für die Remoteverwendung verfügbar gemacht werden.

</dd> <dt>

[**Win32 \_ TSPublishedApplicationList**](win32-tspublishedapplicationlist.md)
</dt> <dd>

Stellt die Liste der Programme dar, die sich auf einem RD-Sitzungshost Server in der Liste RemoteApp-Programme befinden.

</dd> <dt>

[**Win32 \_ TSRemoteDesktop**](win32-tsremotedesktop.md)
</dt> <dd>

Beschreibt eine Remotedesktopverbindung, die über Remotedesktop Webzugriff (RD-Webzugriff) verfügbar ist.

</dd> <dt>

[**Win32 \_ TSStartMenuApplication**](win32-tsstartmenuapplication.md)
</dt> <dd>

Beschreibt die Anwendungen, die sich auf dem Startmenü eines RD-Sitzungshost-Servers befinden.

</dd> <dt>

[**Win32 \_ TSSystemInfo**](win32-tssysteminfo.md)
</dt> <dd>

Stellt den zentralen Veröffentlichungsserver-Informationsdienst dar.

</dd> </dl>
