# <a name="whats-new-in-windows-7"></a>Neues in Windows 7

Windows 7 führt die folgenden neuen Programmierelemente für Remotedesktopdienste.




| Anforderung | Wert |
|--------|-------|
| Remotedesktop ActiveX-Steuerelementschnittstellen<br /> | Die folgenden Schnittstellen (und die zugehörigen Eigenschaften) wurden der Remotedesktop ActiveX <a href="remote-desktop-web-connection-reference.md">Control Interface hinzugefügt.</a><br /><ul><li><a href="imsrdpclientadvancedsettings7.md"><strong>IMsRdpClientAdvancedSettings7-Schnittstelle</strong></a></li></ul> | 
| Remotedesktopdienste Webzugriff-API<br /> | Remotedesktopdienste Webzugriff Control macht Schnittstellen verfügbar, die die Veröffentlichung von benutzerspezifischen Ressourcen in RemoteApp- und Desktopverbindung.<br /><ul><li><a href="/windows/desktop/TermServ/remote-desktop-services-web-access-control-interfaces">Remotedesktopdienste Webzugriff-Steuerelement-API</a></li></ul> | 
| RemoteApp- und Desktopverbindung Runtime-API<br /> | RemoteApp- und Desktopverbindung Runtime-API unterstützt die Entwicklung benutzerdefinierter Clients in RemoteApp- und Desktopverbindung.<br /><ul><li><a href="/windows/desktop/TermServ/remoteapp-and-desktop-connection-runtime-interfaces">RemoteApp- und Desktopverbindung Runtime-API</a></li></ul> |
