# <a name="remoteapp-and-desktop-connection-management-service-interfaces"></a>RemoteApp-und Desktopverbindungsverwaltung von Dienst Schnittstellen

Die RemoteApp-und Desktopverbindungsverwaltung-Dienst-API stellt die folgenden Schnittstellen bereit.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Itspubplugin**](/windows/desktop/api/tspubplugincom/nn-tspubplugincom-itspubplugin)
</dt> <dd>

Macht Eigenschaften und Methoden verfügbar, die Informationen zu Ressourcen bereitstellen, die Benutzern von RemoteApp-und Desktop Verbindungen zur Verfügung stehen.

</dd> <dt>

[**ItsPubPlugin2**](/windows/desktop/api/tspubplugin2com/nn-tspubplugin2com-itspubplugin2)
</dt> <dd>

Gibt Methoden an, die Informationen zu Ressourcen bereitstellen, die Benutzern von RemoteApp-und Desktop Verbindungen zur Verfügung stehen.

</dd> </dl>
