# <a name="remote-desktop-services-audioendpoint-functions"></a>Remotedesktopdienste AudioEndpoint-Funktionen

Die folgenden Funktionen werden mit der AudioEndpoint-API verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[*GetTSAudioEndpointEnumeratorForSession*](gettsaudioendpointenumeratorforsession.md)
</dt> <dd>

Gibt einen Verweis auf einen Audioenumerator für den Endpunkt für die angegebene Sitzungs-ID zurück.

</dd> </dl>

## <a name="remarks"></a>Hinweise

Die Remotedesktopdienste AudioEndpoint-API ist für die Verwendung in Remotedesktop Szenarien vorgesehen. sie ist nicht für Clientanwendungen vorgesehen.
