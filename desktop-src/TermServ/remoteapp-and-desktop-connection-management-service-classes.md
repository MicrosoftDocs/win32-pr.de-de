# <a name="remoteapp-and-desktop-connection-management-service-classes"></a>RemoteApp-und Desktopverbindungsverwaltung Dienst Klassen

Der WMI-Anbieter für den RemoteApp-und Desktopverbindungsverwaltung-Dienst stellt die folgenden Klassen bereit.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Win32 \_ centralpublishingchangeevent**](win32-centralpublishingchangeevent.md)
</dt> <dd>

Ein Ereignis, das eine Änderung an den zentralen RDV-Einstellungen darstellt.

</dd> <dt>

[**Win32 \_ rdcentralpublisheddeploymentsettings**](win32-rdcentralpublisheddeploymentsettings.md)
</dt> <dd>

Enthält die Bereitstellungs Einstellungen, die zum Generieren von RDP-Dateien für aus einer Farm veröffentlichte Ressourcen verwendet werden.

</dd> <dt>

[**Win32 \_ rdcentralpublishedfarm**](win32-rdcentralpublishedfarm.md)
</dt> <dd>

Die Liste der Farmen, von denen Desktops oder Anwendungen veröffentlicht wurden.

</dd> <dt>

[**Win32 \_ rdcentralpublishedfileassociation**](win32-rdcentralpublishedfileassociation.md)
</dt> <dd>

Informationen für eine Dateierweiterung, die einer Anwendung zugeordnet ist.

</dd> <dt>

[**Win32 \_ rdcentralpublishedremoteapplication**](win32-rdcentralpublishedremoteapplication.md)
</dt> <dd>

Beschreibt eine auf einem anderen Computer veröffentlichte Anwendung für die Remote Verwendung durch Terminal Dienste.

</dd> <dt>

[**Win32 \_ rdcentralpublishedremotedesktop**](win32-rdcentralpublishedremotedesktop.md)
</dt> <dd>

Auf einem anderen Computer veröffentlichter Desktop für die Remote Verwendung durch Terminal Dienste

</dd> <dt>

[**Win32 \_ rdpersonaldesktopzuweisung**](win32-rdpersonaldesktopassignment.md)
</dt> <dd>

Die Liste der persönlichen Desktop Zuweisungen.

</dd> <dt>

[**Win32- \_ tscpubplugin**](win32-tscpubplugin.md)
</dt> <dd>

Stellt ein Plug-in dar, das für die Verwendung durch den RemoteApp-und Desktopverbindungsverwaltung-Dienst konfiguriert ist.

</dd> <dt>

[**Win32-Datei-Plug-in \_**](win32-tslegacyplugin.md)
</dt> <dd>

Stellt einen Remotedesktop Server dar, der von den integrierten RemoteApp-und Desktopverbindungsverwaltung Dienst-Plug-Ins für RemoteApp-Programme abgefragt wird.

</dd> <dt>

[**Win32- \_ Arbeitsbereich**](win32-workspace.md)
</dt> <dd>

Gibt Remotedesktopdienste Konfigurationsinformationen für den Arbeitsbereich an.

</dd> </dl>
