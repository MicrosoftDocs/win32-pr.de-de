# <a name="remoteapp-and-desktop-connection-management-service-classes"></a>RemoteApp- und Desktopverbindungsverwaltung Service-Klassen

Der RemoteApp- und Desktopverbindungsverwaltung Service WMI-Anbieter stellt die folgenden Klassen zur Verfügung.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Win32 \_ CentralPublishingChangeEvent**](win32-centralpublishingchangeevent.md)
</dt> <dd>

Ein Ereignis, das eine Änderung der zentralen RDV-Einstellungen darstellt.

</dd> <dt>

[**Win32 \_ RDCentralPublishedDeploymentSettings**](win32-rdcentralpublisheddeploymentsettings.md)
</dt> <dd>

Enthält die Bereitstellungseinstellungen, die zum Generieren von RDP-Dateien für Ressourcen verwendet werden, die von einer Farm veröffentlicht wurden.

</dd> <dt>

[**Win32 \_ RDCentralPublishedFarm**](win32-rdcentralpublishedfarm.md)
</dt> <dd>

Die Liste der landwirtschaftlichen Betriebe, aus denen Desktops oder Anwendungen veröffentlicht wurden.

</dd> <dt>

[**Win32 \_ RDCentralPublishedFileAssociation**](win32-rdcentralpublishedfileassociation.md)
</dt> <dd>

Informationen zu einer einer Anwendung zugeordneten Dateierweiterung

</dd> <dt>

[**Win32 \_ RDCentralPublishedRemoteApplication**](win32-rdcentralpublishedremoteapplication.md)
</dt> <dd>

Beschreibt eine Anwendung, die auf einem anderen Computer für die Remotenutzung über Terminaldienste veröffentlicht wurde.

</dd> <dt>

[**Win32 \_ RDCentralPublishedRemoteDesktop**](win32-rdcentralpublishedremotedesktop.md)
</dt> <dd>

Desktop, der auf einem anderen Computer für die Remotenutzung über Terminaldienste veröffentlicht wird

</dd> <dt>

[**Win32 \_ RDPersonalDesktopAssignment**](win32-rdpersonaldesktopassignment.md)
</dt> <dd>

Die Liste der persönlichen Desktopzuweisungen.

</dd> <dt>

[**Win32 \_ TSCPubPlugin**](win32-tscpubplugin.md)
</dt> <dd>

Stellt ein Plug-In dar, das für die Verwendung über den RemoteApp- und Desktopverbindungsverwaltung ist.

</dd> <dt>

[**Win32 \_ TSLegacyPlugin**](win32-tslegacyplugin.md)
</dt> <dd>

Stellt einen Remotedesktop-Server dar, der von den integrierten RemoteApp- und Desktopverbindungsverwaltung Service-Plug-Ins für RemoteApp-Programme abgefragt wird.

</dd> <dt>

[**Win32-Arbeitsbereich \_**](win32-workspace.md)
</dt> <dd>

Gibt Remotedesktopdienste Informationen zur Arbeitsbereichskonfiguration an.

</dd> </dl>
