# <a name="personal-desktop-task-agent-api-enumerations"></a>API-Enumerationen des Personal Desktop-Task-Agents

Die API für den persönlichen Desktoptask-Agent unterstützt die folgenden Enumerationen.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**\_RDV-AUFGABENSTATUS \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-rdv_task_status)
</dt> <dd>

Wird mit der [**IRDVTaskPluginNotifySink::OnTaskStateChange-Methode**](irdvtaskpluginnotifysink-ontaskstatechange.md) verwendet, um den Status einer Aufgabe anzugeben.

</dd> </dl>
