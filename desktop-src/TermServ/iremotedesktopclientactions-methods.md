# <a name="iremotedesktopclientactions-methods"></a>IRemoteDesktopClientActions-Methoden

Die [**IRemoteDesktopClientActions-Schnittstelle**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclientactions) macht die folgenden Methoden verfügbar.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**ExecuteRemoteAction-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-executeremoteaction)
</dt> <dd>

Bewirkt, dass eine Aktion in der Remotesitzung ausgeführt wird.

</dd> <dt>

[**GetSnapshot-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-getsnapshot)
</dt> <dd>

Bewirkt, dass eine Momentaufnahme des Sitzungsdesktops des Remotedesktopprotokoll-App-Containers (RDP) erstellt wird.

</dd> <dt>

[**ResumeScreenUpdates-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-resumescreenupdates)
</dt> <dd>

Setzt die Bildschirmaktualisierungen wieder ein, die an den Client gesendet werden.

</dd> <dt>

[**SuspendScreenUpdates-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-suspendscreenupdates)
</dt> <dd>

Setzt die Bildschirmaktualisierungen aus, die an den Client gesendet werden.

</dd> </dl>
