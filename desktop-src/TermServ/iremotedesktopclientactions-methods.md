# <a name="iremotedesktopclientactions-methods"></a>Iremotedesktopclientactions-Methoden

Die [**iremotedesktopclientactions**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclientactions) -Schnittstelle stellt die folgenden Methoden zur Verfügung.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Executeremoteaction-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-executeremoteaction)
</dt> <dd>

Bewirkt, dass eine Aktion in der Remote Sitzung ausgeführt wird.

</dd> <dt>

[**Geznapshot-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-getsnapshot)
</dt> <dd>

Bewirkt, dass eine Momentaufnahme des Sitzungs basierten Desktops des Remotedesktopprotokoll (RDP)-app-Containers ausgeführt wird.

</dd> <dt>

[**Resumeskredeupdates-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-resumescreenupdates)
</dt> <dd>

Setzt Bildschirm Aktualisierungen fort, die an den Client gesendet werden.

</dd> <dt>

[**Suspendscreenupdates-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-suspendscreenupdates)
</dt> <dd>

Hält Bildschirm Aktualisierungen an, die an den Client gesendet werden.

</dd> </dl>
