# <a name="iremotedesktopclient-methods"></a>IRemoteDesktopClient-Methoden

Die [**IRemoteDesktopClient-Schnittstelle**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclient) macht die folgenden Methoden verfügbar.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**attachEvent-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-attachevent)
</dt> <dd>

Angefügt einen Ereignishandler an ein Ereignis.

</dd> <dt>

[**Verbinden-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-connect)
</dt> <dd>

Initiiert eine Verbindung mithilfe der Eigenschaften, die derzeit auf dem Clientsteuersatz Remotedesktopprotokoll -App (RDP) festgelegt sind.

</dd> <dt>

[**DeleteSavedCredentials-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-deletesavedcredentials)
</dt> <dd>

Löscht gespeicherte Anmeldeinformationen für den angegebenen Remotecomputer.

</dd> <dt>

[**detachEvent-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-detachevent)
</dt> <dd>

Trennt einen Ereignishandler von einem Ereignis.

</dd> <dt>

[**Disconnect-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-disconnect)
</dt> <dd>

Trennt die aktive Verbindung.

</dd> <dt>

[**Reconnect-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-reconnect)
</dt> <dd>

Initiiert eine automatische wiederhergestellte Verbindung des Remotedesktopprotokoll(RDP)-App-Containerclient-Steuerelements, um die Sitzung an die neue Breite und Höhe anzupassen.

</dd> <dt>

[**UpdateSessionDisplaySettings-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-updatesessiondisplaysettings)
</dt> <dd>

Aktualisiert die Breiten- und Höheneinstellungen für das Remotedesktopprotokoll(RDP)-App-Containerclientsteuer steuerelement.

</dd> </dl>
