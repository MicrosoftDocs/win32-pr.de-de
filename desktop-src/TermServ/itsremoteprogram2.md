# <a name="itsremoteprogram2-interface"></a>ITSRemoteProgram2-Schnittstelle

Definiert Eigenschaften für die Verwendung mit einer RemoteApp.

## <a name="members"></a>Member

Die **ITSRemoteProgram2-Schnittstelle** erbt von [**ITSRemoteProgram.**](itsremoteprogram.md) **ITSRemoteProgram2** verfügt auch über diese Typen von Membern:

-   [Eigenschaften](#properties)

### <a name="properties"></a>Eigenschaften

Die **ITSRemoteProgram2-Schnittstelle** verfügt über diese Eigenschaften.



| Eigenschaft                                                                                  | Zugriffstyp           | BESCHREIBUNG                                                 |
|:------------------------------------------------------------------------------------------|:----------------------|:------------------------------------------------------------|
| [**RemoteApplicationArgs**](itsremoteprogram2-remoteapplicationargs.md)<br/>       | Lesegeschützt<br/> | Die Befehlszeilenargumente für die RemoteApp.<br/>    |
| [**RemoteApplicationName**](itsremoteprogram2-remoteapplicationname.md)<br/>       | Lesegeschützt<br/> | Der Name der RemoteApp.<br/>                       |
| [**RemoteApplicationProgram**](itsremoteprogram2-remoteapplicationprogram.md)<br/> | Lesegeschützt<br/> | Der Pfad und der Dateiname des RemoteApp-Programms.<br/> |
