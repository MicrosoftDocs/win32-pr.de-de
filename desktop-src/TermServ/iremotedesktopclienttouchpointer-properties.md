# <a name="iremotedesktopclienttouchpointer-properties"></a>IRemoteDesktopClientTouchPointer-Eigenschaften

Die [**IRemoteDesktopClientTouchPointer-Schnittstelle**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclienttouchpointer) macht die folgenden Eigenschaften verfügbar.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Enabled-Eigenschaft**](iremotedesktopclienttouchpointer-enabled.md)
</dt> <dd>

Gibt an, ob das Touchzeigerfeature im RdP-App-Containerclientsteuerelement aktiviert ist.

</dd> <dt>

[**EventsEnabled-Eigenschaft**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_eventsenabled)
</dt> <dd>

Gibt an, ob Touchzeigerereignisbenachrichtigungen für die Clientsteuerung des RDP-App-Containers aktiviert sind.

</dd> <dt>

[**PointerSpeed-Eigenschaft**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_pointerspeed)
</dt> <dd>

Wie schnell der Touchzeigercursor relativ zur Geschwindigkeit der Geste auf dem Client auf dem virtuellen Desktop bewegt wird.

</dd> </dl>
