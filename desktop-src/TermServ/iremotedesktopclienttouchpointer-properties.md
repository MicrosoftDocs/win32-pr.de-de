# <a name="iremotedesktopclienttouchpointer-properties"></a>Iremotedesktopclienttouchpointer-Eigenschaften

Die [**iremotedesktopclienttouchpointer**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclienttouchpointer) -Schnittstelle stellt die folgenden Eigenschaften zur Verfügung.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Aktivierte Eigenschaft**](iremotedesktopclienttouchpointer-enabled.md)
</dt> <dd>

Gibt an, ob die Funktion für den Fingerabdruck für das Client Steuerelement der RDP-App aktiviert ist.

</dd> <dt>

[**EventsEnabled (Eigenschaft)**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_eventsenabled)
</dt> <dd>

Gibt an, ob Fingerabdruck Ereignis Benachrichtigungen für die RDP-App-Container-Client Steuerung aktiviert sind.

</dd> <dt>

[**Pointerspeed-Eigenschaft**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_pointerspeed)
</dt> <dd>

Gibt an, wie schnell sich der Fingerabdruck Cursor auf dem virtuellen Desktop bewegt, relativ zur Geschwindigkeit der Bewegung auf dem Client.

</dd> </dl>
