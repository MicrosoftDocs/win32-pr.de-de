# <a name="imsrdpclienttransportsettings4-interface"></a>IMsRdpClientTransportSettings4-Schnittstelle

Definiert zusätzliche Eigenschaften für den Remotedesktop Gateway-Server (RD-Gateway).

## <a name="members"></a>Member

Die **IMsRdpClientTransportSettings4** -Schnittstelle erbt von [**IMsRdpClientTransportSettings3**](imsrdpclienttransportsettings3.md). **IMsRdpClientTransportSettings4** verfügt auch über die folgenden Typen von Membern:

-   [Eigenschaften](#properties)

### <a name="properties"></a>Eigenschaften

Die **IMsRdpClientTransportSettings4** -Schnittstelle verfügt über diese Eigenschaften.



| Eigenschaft                                                                                       | Zugriffstyp           | BESCHREIBUNG                                          |
|:-----------------------------------------------------------------------------------------------|:----------------------|:-----------------------------------------------------|
| [**Gatewaybrokeringtype**](imsrdpclienttransportsettings4-gatewaybrokeringtype.md)<br/> | Lesegeschützt<br/> | Ruft den gatewaybrokeringtyp ab und legt ihn fest.<br/> |
