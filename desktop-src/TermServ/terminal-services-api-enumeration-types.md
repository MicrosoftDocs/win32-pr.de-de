# <a name="remote-desktop-services-api-enumeration-types"></a>API-Enumerationstypen Remotedesktopdienste

Die folgenden Enumerationstypen werden mit der Remotedesktopdienste-API verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**WTS- \_ Konfigurations \_ Klasse**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class)
</dt> <dd>

Enthält Werte, die den Typ der Benutzer Konfigurationsinformationen angeben, die in einem Aufrufen der [**wzqueryuserconfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) -und [**wtsetuserconfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga) -Funktionen festgelegt oder abgerufen werden sollen.

</dd> <dt>

[**WTS- \_ Konfigurations \_ Quelle**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_source)
</dt> <dd>

Gibt die Quelle der Konfigurationsinformationen an, die von der [**wzqueryuserconfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) -Funktion zurückgegeben werden.

</dd> <dt>

[**WTS \_ connectstate- \_ Klasse**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_connectstate_class)
</dt> <dd>

Gibt den Verbindungsstatus einer Remotedesktopdienste Sitzung an.

</dd> <dt>

[**WTS- \_ Informations \_ Klasse**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_info_class)
</dt> <dd>

Enthält Werte, die den Typ der Sitzungsinformationen angeben, die bei einem Aufrufen der [**wzquerysessioninformation**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsquerysessioninformationa) -Funktion abgerufen werden sollen.

</dd> <dt>

[**WTS \_ - \_ Typklasse**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_type_class)
</dt> <dd>

Gibt den Strukturtyp an, den eine Remotedesktopdienste Funktion in einem Puffer zurückgegeben hat.

</dd> <dt>

[**virtuelle WTS- \_ \_ Klasse**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_virtual_class)
</dt> <dd>

Enthält Werte, die den Typ der abzurufenden Informationen des virtuellen Kanals angeben.

</dd> <dt>

[**wtssbx- \_ Adress \_ Familie**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_address_family)
</dt> <dd>

Enthält Werte, die die Adressfamilie einer Netzwerkadresse angeben, die für die Umleitung verwendet wird.

</dd> <dt>

[**wtssbx- \_ Computer \_ Ableitung**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_drain)
</dt> <dd>

Enthält Werte, die den Ausgleichs Status eines Remotedesktop-Sitzungshost (RD-Sitzungshost)-Servers angeben.

</dd> <dt>

[**wtssbx- \_ Computer \_ Sitzungs \_ Modus**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_session_mode)
</dt> <dd>

Enthält Werte, die den Sitzungs Modus eines RD-Sitzungshost Servers angeben.

</dd> <dt>

[**wtssbx- \_ Computer \_ Status**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_state)
</dt> <dd>

Enthält Werte, die den aktuellen Status eines Servers angeben.

</dd> <dt>

[**wtssbx \_ - \_ Benachrichtigungstyp**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_notification_type)
</dt> <dd>

Enthält Werte, die den Typ der Statusänderung angeben, die auf einem RD-Sitzungshost Server oder einer Benutzersitzung aufgetreten ist.

</dd> <dt>

[**wtssbx- \_ Sitzungs \_ Status**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_session_state)
</dt> <dd>

Enthält Werte, die den Verbindungsstatus einer Benutzersitzung angeben.

</dd> </dl>
