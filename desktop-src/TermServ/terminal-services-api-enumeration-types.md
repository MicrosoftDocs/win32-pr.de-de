# <a name="remote-desktop-services-api-enumeration-types"></a>Remotedesktopdienste API-Enumerationstypen

Die folgenden Enumerationstypen werden mit der Remotedesktopdienste-API verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**\_WTS \_CONFIG-KLASSE**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class)
</dt> <dd>

Enthält Werte, die den Typ der Benutzerkonfigurationsinformationen angeben, die in einem Aufruf der [**Funktionen WTSQueryUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) und [**WTSSetUserConfig festgelegt**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga) oder abgerufen werden.

</dd> <dt>

[**\_WTS KONFIGURATIONSQUELLE \_**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_source)
</dt> <dd>

Gibt die Quelle der Konfigurationsinformationen an, die von der [**WTSQueryUserConfig-Funktion zurückgegeben**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) werden.

</dd> <dt>

[**\_WTS \_CONNECTSTATE-KLASSE**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_connectstate_class)
</dt> <dd>

Gibt den Verbindungsstatus einer Remotedesktopdienste an.

</dd> <dt>

[**\_WTS \_INFO-KLASSE**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_info_class)
</dt> <dd>

Enthält Werte, die den Typ der Sitzungsinformationen angeben, die in einem Aufruf der [**WTSQuerySessionInformation-Funktion abgerufen werden.**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsquerysessioninformationa)

</dd> <dt>

[**\_WTS \_TYPE-KLASSE**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_type_class)
</dt> <dd>

Gibt den Strukturtyp an, den eine Remotedesktopdienste in einem Puffer zurückgegeben hat.

</dd> <dt>

[**\_WTS VIRTUELLE \_ KLASSE**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_virtual_class)
</dt> <dd>

Enthält Werte, die den Typ der abzurufenden Informationen des virtuellen Kanals angeben.

</dd> <dt>

[**WTSSBX-ADRESSFAMILIE \_ \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_address_family)
</dt> <dd>

Enthält Werte, die die Adressfamilie einer Netzwerkadresse angeben, die für die Umleitung verwendet wird.

</dd> <dt>

[**WTSSBX \_ MACHINE \_ DRAIN**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_drain)
</dt> <dd>

Enthält Werte, die den Leerungszustand eines Remotedesktop-Sitzungshost (RD-Sitzungshost) angeben.

</dd> <dt>

[**\_WTSSBX-COMPUTERSITZUNGSMODUS \_ \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_session_mode)
</dt> <dd>

Enthält Werte, die den Sitzungsmodus eines RD-Sitzungshost angeben.

</dd> <dt>

[**\_WTSSBX-COMPUTERSTATUS \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_state)
</dt> <dd>

Enthält Werte, die den aktuellen Status eines Servers angeben.

</dd> <dt>

[**WTSSBX-BENACHRICHTIGUNGSTYP \_ \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_notification_type)
</dt> <dd>

Enthält Werte, die den Typ der Statusänderung angeben, die auf einem RD-Sitzungshost Server oder einer Benutzersitzung aufgetreten ist.

</dd> <dt>

[**WTSSBX-SITZUNGSZUSTAND \_ \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_session_state)
</dt> <dd>

Enthält Werte, die den Verbindungsstatus einer Benutzersitzung angeben.

</dd> </dl>
