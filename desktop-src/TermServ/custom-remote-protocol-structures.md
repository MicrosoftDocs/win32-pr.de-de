<dl> <dt>

</dd> <dt>

</dd> <dt>

[**wrds \_ - \_ Listenereinstellungen**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_listener_settings)

</dd> <dt>

[**Wrds \_ - \_ Listenereinstellungen \_ 1**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_listener_settings_1)
