# <a name="itspubplugin2-methods"></a>ItsPubPlugin2-Methoden

Die [**ItsPubPlugin2-Schnittstelle**](/windows/desktop/api/tspubplugin2com/nn-tspubplugin2com-itspubplugin2) macht die folgenden Methoden verfügbar.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**DeletePersonalDesktopAssignment-Methode**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-deletepersonaldesktopassignment)
</dt> <dd>

Wird aufgerufen, um eine Zuordnung zwischen dem angegebenen Benutzer und einem virtuellen Computer in einer persönlichen Sammlung virtueller Desktops zu löschen.

</dd> <dt>

[**GetResource2-Methode**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2)
</dt> <dd>

Diese Methode ist reserviert und sollte immer **E \_ NOTIMPL zurückgeben.**

</dd> <dt>

[**GetResource2List-Methode**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2list)
</dt> <dd>

Ruft eine Liste der Ressourcen ab, die dem angegebenen Benutzer zugewiesen sind.

</dd> <dt>

[**ResolvePersonalDesktop-Methode**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-resolvepersonaldesktop)
</dt> <dd>

Wird aufgerufen, um eine Zuordnung zwischen dem angegebenen Benutzer und einem virtuellen Computer in einer persönlichen Sammlung virtueller Desktops aufzulösen.

</dd> </dl>
