# <a name="itspubplugin2-methods"></a>ItsPubPlugin2-Methoden

Die [**ItsPubPlugin2**](/windows/desktop/api/tspubplugin2com/nn-tspubplugin2com-itspubplugin2) -Schnittstelle stellt die folgenden Methoden zur Verfügung.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Deletepersonaldesktopzuweisung-Methode**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-deletepersonaldesktopassignment)
</dt> <dd>

Wird aufgerufen, um eine Zuordnung zwischen dem angegebenen Benutzer und einer virtuellen Maschine in einer persönlichen Sammlung von virtuellen Desktops zu löschen.

</dd> <dt>

[**GetResource2-Methode**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2)
</dt> <dd>

Diese Methode ist reserviert und sollte immer **E \_ notimpl** zurückgeben.

</dd> <dt>

[**GetResource2List-Methode**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2list)
</dt> <dd>

Ruft eine Liste von Ressourcen ab, die dem angegebenen Benutzer zugewiesen sind.

</dd> <dt>

[**Resolvepersonaldesktop-Methode**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-resolvepersonaldesktop)
</dt> <dd>

Wird aufgerufen, um eine Zuordnung zwischen dem angegebenen Benutzer und einer virtuellen Maschine in einer persönlichen Sammlung von virtuellen Desktops aufzulösen.

</dd> </dl>
