# <a name="itssbprovisioning-methods"></a>Itssbprovisioning-Methoden

Die [**itssbprovisioning**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbprovisioning) -Schnittstellen stellen die folgenden Methoden zur Verfügung.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**CancelJob-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioning-canceljob)
</dt> <dd>

Bricht einen Bereitstellungs Auftrag ab.

</dd> <dt>

[**Methode "kreatevirtualmachines"**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioning-createvirtualmachines)
</dt> <dd>

Erstellt eine virtuelle Maschine asynchron.

</dd> <dt>

[**Deletevirtualmachines-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioning-deletevirtualmachines)
</dt> <dd>

Löscht eine virtuelle Maschine asynchron.

</dd> <dt>

[**Patchvirtualmachines-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioning-patchvirtualmachines)
</dt> <dd>

Patcht eine virtuelle Maschine asynchron.

</dd> </dl>
