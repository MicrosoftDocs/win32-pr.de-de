# <a name="itssbprovisioningpluginnotifysink-methods"></a>Itssbprovisioningpluginnotifysink-Methoden

Die [**itssbprovisioningpluginnotifysink**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbprovisioningpluginnotifysink) -Schnittstelle stellt die folgenden Methoden zur Verfügung.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Lockvirtualmachine-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-lockvirtualmachine)
</dt> <dd>

Benachrichtigt Remotedesktopverbindung Broker (RD-Verbindungsbroker), dass der virtuelle Computer gesperrt ist.

</dd> <dt>

[**Onjobabgeb Rochen-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcancelled)
</dt> <dd>

Benachrichtigt RD-Verbindungsbroker, dass der Auftrag abgebrochen wird.

</dd> <dt>

[**Onjobabgeschlossene-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcompleted)
</dt> <dd>

Benachrichtigt RD-Verbindungsbroker, dass der Auftrag beendet wurde.

</dd> <dt>

[**Onjobcreated-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcreated)
</dt> <dd>

Benachrichtigt RD-Verbindungsbroker, dass ein Bereitstellungs Auftrag erstellt wird.

</dd> <dt>

[**Onvirtualmachinehoststatuschangi-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onvirtualmachinehoststatuschanged)
</dt> <dd>

Benachrichtigt RD-Verbindungsbroker, dass der Status des Hosts einer virtuellen Maschine geändert wird.

</dd> <dt>

[**Onvirtualmachinestatuschge-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onvirtualmachinestatuschanged)
</dt> <dd>

Benachrichtigt RD-Verbindungsbroker, dass der Status eines virtuellen Computers geändert wird.

</dd> </dl>
