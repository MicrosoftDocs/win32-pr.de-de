# <a name="terminal-services-is-now-remote-desktop-services"></a>Terminaldienste sind jetzt Remotedesktopdienste

Terminaldienste wurden in Remotedesktopdienste.

In Windows Server 2008 R2 wurden alle Remotedesktopdienste Rollendienste umbenannt. In der folgenden Tabelle werden sowohl der erste als auch der neue Name der einzelnen Rollendienste Remotedesktopdienste aufgeführt.



| Vorheriger Name                                                   | Name in Windows Server 2008 R2                                         |
|-----------------------------------------------------------------|------------------------------------------------------------------------|
| Terminaldienste <br/>                                   | Remotedesktopdienste<br/>                                     |
| Terminalserver<br/>                                      | Remotedesktop-Sitzungshost (RD-Sitzungshost)<br/>               |
| Terminaldienstelizenzierung (TS-Lizenzierung)<br/>          | Remotedesktoplizenzierung (RD-Lizenzierung)<br/>                     |
| Terminaldienstegateway (TS-Gateway)<br/>               | Remotedesktopgateway (RD-Gateway)<br/>                         |
| Terminaldienste-Sitzungsbroker (TS-Sitzungsbroker)<br/> | Remotedesktopverbindung Broker (RD-Verbindungsbroker)<br/>     |
| Terminaldienste-Webzugriff (TS-Webzugriff)<br/>         | Remotedesktop Webzugriff (RD-Webzugriff)<br/>                  |
| Virtualisierung von Terminaldiensten<br/>                     | Remotedesktop-Virtualisierungshost (RD-Virtualisierungshost)<br/> |



 

Beachten Sie außerdem, dass ein Server, auf dem der RD-Sitzungshost-Rollendienst installiert ist, jetzt als RD-Sitzungshost Server und nicht als Terminalserver bezeichnet wird.
