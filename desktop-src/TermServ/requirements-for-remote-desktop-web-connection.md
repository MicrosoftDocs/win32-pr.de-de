# <a name="requirements-for-remote-desktop-web-connection"></a>Anforderungen für Remotedesktop-Webverbindung

Remotedesktop-Webverbindung erfordert zwei Systeme, die wie folgt konfiguriert sind:

-   Ein Hostserver, auf dem Windows und IIS 4.0 oder höher ausgeführt wird.
-   Ein Client, der das Remotedesktop ActiveX-Steuerelement herunterladen und ausführen kann. Jeder Client, auf dem Microsoft Internet Explorer 4.01 (vorzugsweise Service Pack 2) oder höher ausgeführt wird, kann das Remotedesktop ActiveX-Steuerelement ausführen.

Der Benutzer, der die Software installiert, muss über ausreichende Berechtigungen zum Installieren der Software verfügen. Beispielsweise muss der Benutzer auf Windows XP Mitglied der Gruppen Power Users oder Administrators sein.
