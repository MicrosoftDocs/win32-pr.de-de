# <a name="requirements-for-remote-desktop-web-connection"></a>Anforderungen für Remotedesktop-Webverbindung

Remotedesktop-Webverbindung müssen zwei Systeme wie folgt konfiguriert werden:

-   Einen Host Server, auf dem Windows und IIS 4,0 oder höher ausgeführt werden.
-   Ein Client, der das Remotedesktop ActiveX-Steuerelement herunterladen und ausführen kann. Alle Clients, auf denen Microsoft Internet Explorer 4,01 (vorzugsweise Service Pack 2) oder höher ausgeführt wird, können das Remotedesktop ActiveX-Steuerelement ausführen.

Der Benutzer, der die Software installiert, muss über ausreichende Berechtigungen zum Installieren von Software verfügen. Beispielsweise muss der Benutzer unter Windows XP Mitglied der Gruppe "Hauptbenutzer" oder "Administratoren" sein.
