# <a name="itssbtaskpluginnotifysink-methods"></a>Itssbtaskpluginnotifysink-Methoden

Die [**itssbtaskpluginnotifysink**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbtaskpluginnotifysink) -Schnittstelle stellt die folgenden Methoden zur Verfügung.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Ondeletetasktime-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-ondeletetasktime)
</dt> <dd>

Benachrichtigt Remotedesktopverbindung Broker (RD-Verbindungsbroker), dass eine Aufgabe aus der Warteschlange entfernt wurde.

</dd> <dt>

[**Onreporttasks-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onreporttasks)
</dt> <dd>

Benachrichtigt RD-Verbindungsbroker über einen neuen Aufgaben Bericht.

</dd> <dt>

[**Onsettasktime-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onsettasktime)
</dt> <dd>

Benachrichtigt RD-Verbindungsbroker, dass eine Aufgabe geplant wurde.

</dd> <dt>

[**Onupdatetaskstatus-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onupdatetaskstatus)
</dt> <dd>

Benachrichtigt RD-Verbindungsbroker, dass sich der Status einer Aufgabe geändert hat.

</dd> </dl>
