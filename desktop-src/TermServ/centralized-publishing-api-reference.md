# <a name="remoteapp-and-desktop-connection-management-service-api-reference"></a>Referenz zur RemoteApp- und Desktopverbindungsverwaltung-Dienst-API

Die RemoteApp- und Desktopverbindungsverwaltung-Dienst-API unterstützt eine Schnittstelle und eine Struktur, die Informationen zu Ressourcen bereitstellt, die Benutzern von RemoteApp- und Desktopverbindungen zur Verfügung stehen. Ihre Implementierung dieser Schnittstelle kann eine benutzerdefinierte Filterung von Ressourcen durchführen und Unterstützung für Dateitypen bereitstellen, die derzeit nicht unterstützt werden.
