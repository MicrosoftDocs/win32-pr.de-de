# <a name="remoteapp-and-desktop-connection-management-service-api-reference"></a>RemoteApp-und Desktopverbindungsverwaltung Dienst-API-Referenz

Die RemoteApp-und Desktopverbindungsverwaltung Service-API unterstützt eine Schnittstelle und eine Struktur, die Informationen zu Ressourcen bereitstellen, die Benutzern von RemoteApp-und Desktop Verbindungen zur Verfügung stehen. Die Implementierung dieser Schnittstelle kann eine benutzerdefinierte Filterung von Ressourcen durchführen und Unterstützung für Dateitypen bereitstellen, die derzeit nicht unterstützt werden. Weitere Informationen finden Sie in den nachfolgenden Themen.
