# <a name="remote-desktop-services-user-configuration"></a>Remotedesktopdienste Benutzerkonfiguration

Mit [**den Funktionen WTSQueryUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) und [**WTSSetUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga) können Sie Konfigurationsinformationen für ein angegebenes Benutzerkonto festlegen und abrufen. Der [**WTS \_ CONFIG \_ CLASS-Enumerationstyp**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class) definiert die Benutzerkonfigurationsparameter.
