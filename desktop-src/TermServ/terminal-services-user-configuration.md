# <a name="remote-desktop-services-user-configuration"></a>Remotedesktopdienste Benutzerkonfiguration

Mit den Funktionen [**wzqueryuserconfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) und [**wtsetuserconfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga) können Sie Konfigurationsinformationen für ein angegebenes Benutzerkonto festlegen und abrufen. Der Enumerationstyp der [**WTS- \_ Konfigurations \_ Klasse**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class) definiert die Benutzer Konfigurationsparameter.
