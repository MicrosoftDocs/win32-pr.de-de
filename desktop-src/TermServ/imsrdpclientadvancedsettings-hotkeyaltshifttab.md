# <a name="imsrdpclientadvancedsettingshotkeyaltshifttab-property"></a>Imsrdpclientadvancedsettings:: hotkeyaltshif ttab-Eigenschaft

Gibt den Code des virtuellen Schlüssels an, der alt addiert werden soll, um die Hotkey-Ersetzung für Alt + Umschalt + Tab zu bestimmen.

Diese Eigenschaft ist nur gültig, wenn die [**keyboardhuokmode**](imsrdpclientsecuredsettings-keyboardhookmode.md) -Eigenschaft nicht aktiviert ist.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```C++
HRESULT put_HotKeyAltShiftTab(
  [in]  LONG hotKeyAltShiftTab
);

HRESULT get_HotKeyAltShiftTab(
  [out] LONG *photKeyAltShiftTab
);
```



## <a name="property-value"></a>Eigenschaftswert

Der neue Code des virtuellen Schlüssels. **VK \_ Der nächste** Schritt ist der Standardwert, wobei Alt + Bild-ab als resultierende Sequenz angezeigt wird.

## <a name="error-codes"></a>Fehlercodes

Gibt bei Erfolg **S \_ OK** zurück.
