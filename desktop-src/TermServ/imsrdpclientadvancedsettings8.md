# <a name="imsrdpclientadvancedsettings8-interface"></a>IMsRdpClientAdvancedSettings8-Schnittstelle

Macht Methoden und Eigenschaften verfügbar, die erweiterte Einstellungen des Remotedesktop ActiveX-Steuer Elements verwalten.

Um eine Instanz dieser Schnittstelle abzurufen, verwenden Sie die [**IMsRdpClient8:: AdvancedSettings9**](imsrdpclient8-advancedsettings9.md) -Eigenschaft.

## <a name="members"></a>Member

Die **IMsRdpClientAdvancedSettings8** -Schnittstelle erbt von [**IMsRdpClientAdvancedSettings7**](imsrdpclientadvancedsettings7.md). **IMsRdpClientAdvancedSettings8** verfügt auch über die folgenden Typen von Membern:

-   [Eigenschaften](#properties)

| Eigenschaft                                                                                  | Zugriffstyp           | BESCHREIBUNG                                                           |
