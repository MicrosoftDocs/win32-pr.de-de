# <a name="imsrdpclientadvancedsettings8-interface"></a>IMsRdpClientAdvancedSettings8-Schnittstelle

Macht Methoden und Eigenschaften verfügbar, die erweiterte Einstellungen des Remotedesktop ActiveX-Steuerelements verwalten.

Um eine Instanz dieser Schnittstelle abzurufen, verwenden Sie die [**IMsRdpClient8::AdvancedSettings9-Eigenschaft.**](imsrdpclient8-advancedsettings9.md)

## <a name="members"></a>Member

Die **IMsRdpClientAdvancedSettings8-Schnittstelle** erbt von [**IMsRdpClientAdvancedSettings7**](imsrdpclientadvancedsettings7.md). **IMsRdpClientAdvancedSettings8** verfügt auch über diese Typen von Membern:

-   [Eigenschaften](#properties)

### <a name="properties"></a>Eigenschaften

Die **IMsRdpClientAdvancedSettings8-Schnittstelle** verfügt über diese Eigenschaften.



| Eigenschaft                                                                                  | Zugriffstyp           | Beschreibung                                                           |
|:------------------------------------------------------------------------------------------|:----------------------|:----------------------------------------------------------------------|
| [**BandwidthDetection**](imsrdpclientadvancedsettings8-bandwidthdetection.md)<br/> | Lesen/Schreiben<br/> | Gibt an, ob Bandbreitenänderungen automatisch erkannt werden.<br/> |
