# <a name="graphics-virtual-channels-interfaces"></a>Grafikschnittstellen für virtuelle Kanäle

Die API für virtuelle Grafikkanäle unterstützt die folgenden Schnittstellen.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**IWRdsGraphicsChannel**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannel)
</dt> <dd>

Diese Schnittstelle wird vom RemoteFX Grafikdiensten verwendet, um Daten an einen virtuellen Grafikkanal zu senden und zu empfangen.

</dd> <dt>

[**IWRdsGraphicsChannelEvents**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannelevents)
</dt> <dd>

Diese Schnittstelle empfängt Benachrichtigungen, die sich auf einen virtuellen Grafikkanal beziehen.

</dd> <dt>

[**IWRdsGraphicsChannelManager**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannelmanager)
</dt> <dd>

Diese Schnittstelle wird von der RemoteFX-Api für Grafikdienste verwendet, um die virtuellen Grafikkanäle zu erstellen, die für das Remoting von Grafikdaten erforderlich sind.

</dd> </dl>
