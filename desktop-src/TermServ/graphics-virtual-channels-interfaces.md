# <a name="graphics-virtual-channels-interfaces"></a>Virtual Channels-Schnittstellen für Grafiken

Die Grafik Virtual Channels-API unterstützt die folgenden Schnittstellen.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Iwrdsgraphicschannel**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannel)
</dt> <dd>

Diese Schnittstelle wird von den remotefx-Grafik Diensten verwendet, um Daten an einen virtuellen Grafik Kanal zu senden und zu empfangen.

</dd> <dt>

[**Iwrdsgraphicschannelevents**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannelevents)
</dt> <dd>

Diese Schnittstelle empfängt Benachrichtigungen, die sich auf einen virtuellen Grafik Kanal beziehen.

</dd> <dt>

[**Iwrdsgraphicschannelmanager**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannelmanager)
</dt> <dd>

Diese Schnittstelle wird von der remotefx-Grafik Dienste-API zum Erstellen der virtuellen Grafik Kanäle verwendet, die für das Remoting von Grafikdaten erforderlich sind.

</dd> </dl>
