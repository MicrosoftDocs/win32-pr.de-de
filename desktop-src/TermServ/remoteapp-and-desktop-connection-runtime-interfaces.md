# <a name="remoteapp-and-desktop-connection-runtime-interfaces"></a>Lauf Zeit Schnittstellen RemoteApp-und Desktopverbindung

Die RemoteApp-und Desktopverbindung Runtime-API unterstützt Schnittstellen, die die Entwicklung benutzerdefinierter Clients in RemoteApp-und Desktopverbindung ermöglichen.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Iworkspace**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspace)
</dt> <dd>

Macht Methoden verfügbar, die Informationen zu einer Verbindung in RemoteApp-und Desktopverbindung bereitstellen.

</dd> <dt>

[**IWorkspace2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspace2)
</dt> <dd>

Macht zusätzliche Methoden verfügbar, die Informationen zu einer Verbindung in RemoteApp-und Desktopverbindung bereitstellen.

</dd> <dt>

[**IWorkspace3**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspace3)
</dt> <dd>

Macht Methoden verfügbar, die Informationen zu einer Verbindung in RemoteApp-und Desktopverbindung bereitstellen, und bietet die Möglichkeit, ein Anspruchs Token abzurufen oder festzulegen.

</dd> <dt>

[**Iworkspaceclikotext**](/windows/desktop/api/Workspaceruntimeclientext/nn-workspaceruntimeclientext-iworkspaceclientext)
</dt> <dd>

Macht Methoden verfügbar, mit denen die Common Language Runtime einen benutzerdefinierten Client in RemoteApp-und Desktopverbindung trennen kann.

</dd> <dt>

[**Iworkspaceregistration**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspaceregistration)
</dt> <dd>

Macht Methoden verfügbar, mit denen in RemoteApp-und Desktopverbindung Verweise auf benutzerdefinierte Clients hinzugefügt und entfernt werden.

</dd> <dt>

[**IWorkspaceRegistration2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspaceregistration2)
</dt> <dd>

Macht Methoden verfügbar, mit denen in RemoteApp-und Desktopverbindung Verweise auf benutzerdefinierte Clients hinzugefügt und entfernt werden.

</dd> <dt>

[**Iworkspacereportmessage**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacereportmessage)
</dt> <dd>

Macht Methoden verfügbar, die die Behandlung von Fehlermeldungen für Remote Arbeitsbereiche unterstützen.

</dd> <dt>

[**Iworkspacerestyperegistry**](/windows/desktop/api/Workspaceax/nn-workspaceax-iworkspacerestyperegistry)
</dt> <dd>

Macht Methoden verfügbar, mit denen ein Plug-in Dateinamen Erweiterungen von Drittanbietern in RemoteApp-und Desktopverbindung Laufzeit verwalten kann.

</dd> <dt>

[**Iworkspacescriptable**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable)
</dt> <dd>

Macht Methoden verfügbar, die RemoteApp-und Desktopverbindung Anmelde Informationen und Verbindungen verwalten.

</dd> <dt>

[**IWorkspaceScriptable2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable2)
</dt> <dd>

Macht Methoden verfügbar, die RemoteApp-und Desktopverbindung Anmelde Informationen und Verbindungen verwalten.

</dd> <dt>

[**IWorkspaceScriptable3**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable3)
</dt> <dd>

Macht Methoden verfügbar, die RemoteApp-und Desktopverbindung Anmelde Informationen und Verbindungen verwalten.

</dd> </dl>
