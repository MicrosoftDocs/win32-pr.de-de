# <a name="imstscaxeventsondevicesbuttonpressed-method"></a>Imstscaxevents:: ondevicesbuttonpressed-Methode

Wird aufgerufen, wenn die Schaltfläche Geräte in der Verbindungs Leiste gedrückt wurde.

## <a name="syntax"></a>Syntax


```C++
void OnDevicesButtonPressed();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.

## <a name="remarks"></a>Bemerkungen

Die Schaltfläche Geräte wird in der Verbindungs Leiste angezeigt, wenn die USB-Umleitung sowohl auf dem Client als auch auf dem Server aktiviert ist.
