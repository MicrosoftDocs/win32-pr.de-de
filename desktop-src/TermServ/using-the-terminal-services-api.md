# <a name="using-the-remote-desktop-services-api"></a>Verwenden der Remotedesktopdienste-API

In diesem Abschnitt wird beschrieben, wie Sie mit der Remotedesktopdienste-API Anwendungen die folgenden Aufgaben in einer Remotedesktopdienste Umgebung ausführen können:

-   Ausführen von [Remotedesktopdienste Verwaltungs](terminal-services-administration.md) Aufgaben, z. b. das Auflisten und Verwalten von Remotedesktop-Sitzungshost (RD-Sitzungshost)-Servern in einer Domäne oder das Auflisten und Verwalten von Sitzungen und Prozessen auf einem RD-Sitzungshost Server.
-   Verbessern von Client/Server-Anwendungen in einer Remotedesktopdienste Umgebung.
-   [Verwenden von Remotedesktopdienste virtuellen Kanälen](using-terminal-services-virtual-channels.md) für die Kommunikation zwischen Client-und Server Modulen einer Anwendung.
-   Festlegen und Abrufen von [Remotedesktopdienste Benutzer Konfigurations](terminal-services-user-configuration.md) Informationen.

Wenn Ihre Anwendung in einer Umgebung ausgeführt wird, die keine Remotedesktopdienste Umgebung ist, Sie jedoch die Remotedesktopdienste-API verwenden möchten, um zusätzliche Funktionalität in einer Remotedesktopdienste Umgebung bereitzustellen, finden Sie weitere Informationen unter [Laufzeit-Verknüpfung mit Wtsapi32.dll](run-time-linking-to-wtsapi32-dll.md).
