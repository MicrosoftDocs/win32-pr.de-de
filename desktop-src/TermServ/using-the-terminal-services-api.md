# <a name="using-the-remote-desktop-services-api"></a>Verwenden der Remotedesktopdienste-API

In diesem Abschnitt wird beschrieben, wie Sie die Remotedesktopdienste-API verwenden können, um Anwendungen das Ausführen der folgenden Aufgaben in einer Remotedesktopdienste-Umgebung zu ermöglichen:

-   Ausführen [Remotedesktopdienste Verwaltungsaufgaben,](terminal-services-administration.md) z. B. Aufzählen und Verwalten von Remotedesktop-Sitzungshost -Servern (RD-Sitzungshost) in einer Domäne oder Aufzählen und Verwalten von Sitzungen und Prozessen auf einem RD-Sitzungshost-Server.
-   Erweitern von Client-/Serveranwendungen in einer Remotedesktopdienste-Umgebung.
-   [Verwenden von Remotedesktopdienste virtuellen Kanälen](using-terminal-services-virtual-channels.md) für die Kommunikation zwischen Client- und Servermodulen einer Anwendung.
-   Festlegen und Abrufen [Remotedesktopdienste Benutzerkonfigurationsinformationen.](terminal-services-user-configuration.md)

Wenn Ihre Anwendung in einer Umgebung ausgeführt wird, die keine Remotedesktopdienste Umgebung ist, Sie aber die Remotedesktopdienste-API verwenden möchten, um zusätzliche Funktionen in einer Remotedesktopdienste Umgebung bereitzustellen, finden Sie weitere Informationen unter [Laufzeitverknüpfung mit Wtsapi32.dll](run-time-linking-to-wtsapi32-dll.md).
