# <a name="remote-desktop-services-management-applications"></a>Remotedesktopdienste-Verwaltungsanwendungen

Remotedesktopdienste unterstützt die folgenden Verwaltungsanwendungen.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[ADSI-Erweiterung für Remotedesktopdienste Benutzerkonfiguration](adsi-extension-for-terminal-services-user-configuration.md)
</dt> <dd>

Eine ADSI-Erweiterung (Active Directory Service Interface) ist ein spezielles COM-Objekt, das die Erweiterung eines COM-Objekts ermöglicht.

</dd> <dt>

[Remotedesktopdienste Berechtigungen](terminal-services-permissions.md)
</dt> <dd>

Sie können die berechtigungen verwenden, die für Remotedesktopdienste, um zu steuern, wie Benutzer und Gruppen auf den Server zugreifen.

</dd> </dl>

Der [Remotedesktopdienste WMI-Anbieter](terminal-services-wmi-provider.md) stellt WMI-Zugriff auf Informationen zur Verfügung, die Administratoren mithilfe des MMC-Snap-Ins (Remotedesktopdienste Configuration/Connections Microsoft Management Console) konfigurieren können.

Die [ADSI-Erweiterung für Remotedesktopdienste-Benutzerkonfiguration](adsi-extension-for-terminal-services-user-configuration.md) erweitert ein ADSI-Objekt, mit dem Domänenadministratoren Skripts schreiben können, die die Wartung Remotedesktopdienste benutzerspezifischen Eigenschaften automatisieren.

Eine Liste der Berechtigungen, die Sie festlegen können, und der Aufgaben, die die Berechtigungen zulassen, finden Sie [Remotedesktopdienste Berechtigungen.](terminal-services-permissions.md)

Weitere Informationen finden Sie in Remotedesktopdienste [WMI-Anbieterreferenz](terminal-services-wmi-provider-reference.md) und [Verwenden der ADSI-Erweiterung für Remotedesktopdienste Benutzerkonfiguration.](using-the-adsi-extension-for-terminal-services-user-configuration.md)
