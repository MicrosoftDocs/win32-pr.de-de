# <a name="remotefx-media-redirection-api-functions"></a>RemoteFX Funktionen der Medienumleitungs-API

Die RemoteFX-API für die Medienumleitung unterstützt die folgenden Funktionen.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**WTSSetRenderHint**](/windows/desktop/api/WtsHintApi/nf-wtshintapi-wtssetrenderhint)
</dt> <dd>

Wird von einer Anwendung verwendet, die Inhalte zeigt, die für die Anzeige in einer Remotesitzung optimiert werden können, um den Bereich eines Fensters zu identifizieren, bei dem es sich um den tatsächlichen Inhalt handelt.

</dd> </dl>
