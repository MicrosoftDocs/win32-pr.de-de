# <a name="imsrdpclient10remoteprogram3-property"></a>IMsRdpClient10:: RemoteProgram3-Eigenschaft

Ein Objekt, das die [**ITSRemoteProgram3**](itsremoteprogram3.md) -Schnittstelle unterstützt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```C++
HRESULT get_RemoteProgram3(
  [out, retval] ITSRemoteProgram3 **ppRemoteProgram
);
```



## <a name="property-value"></a>Eigenschaftswert

Gibt einen [**ITSRemoteProgram3**](itsremoteprogram3.md) -Schnittstellen Zeiger zurück.
