# <a name="imsrdpclient10remoteprogram3-property"></a>IMsRdpClient10::RemoteProgram3 (Eigenschaft)

Ein Objekt, das die [**ITSRemoteProgram3-Schnittstelle**](itsremoteprogram3.md) unterstützt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```C++
HRESULT get_RemoteProgram3(
  [out, retval] ITSRemoteProgram3 **ppRemoteProgram
);
```



## <a name="property-value"></a>Eigenschaftswert

Gibt einen [**ITSRemoteProgram3-Schnittstellenzeiger**](itsremoteprogram3.md) zurück.
