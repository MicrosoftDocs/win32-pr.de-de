# <a name="remote-desktop-gateway-enumeration-types"></a>Remotedesktop Gateway-Enumerationstypen

Die Remotedesktop Gateway-API (RD-Gateway) unterstützt die folgenden Enumerationstypen.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Aaaccountingdatatype**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-aaaccountingdatatype)
</dt> <dd>

Gibt den Ereignistyp an, über den die [**itsgaccountingengine::D oaccounting**](/windows/desktop/api/TSGPolicyEngine/nf-tsgpolicyengine-itsgaccountingengine-doaccounting) -Methode benachrichtigt wird.

</dd> <dt>

[**Aaauthschemas**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-aaauthschemes)
</dt> <dd>

Gibt den Typ der Authentifizierung an, die zum Herstellen einer Verbindung mit RD-Gateway verwendet wird

</dd> <dt>

[**Policyattributetype**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-policyattributetype)
</dt> <dd>

Gibt die einer Verbindung zugeordneten Umleitungseinstellungen an.

</dd> </dl>
