# <a name="iremotedesktopclientsettings-methods"></a>Iremotedesktopclientsettings-Methoden

Die [**iremotedesktopclientsettings**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclientsettings) -Schnittstelle stellt die folgenden Methoden zur Verfügung.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Applysettings-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-applysettings)
</dt> <dd>

Speichert den angegebenen Inhalt in der RDP-Datei.

</dd> <dt>

[**Getrdpproperty-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-getrdpproperty)
</dt> <dd>

Ruft einen einzelnen benannten RDP-Eigenschafts Wert ab.

</dd> <dt>

[**Retrievesettings-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-retrievesettings)
</dt> <dd>

Ruft die gesamte RDP-Datei als Zeichenfolge ab.

</dd> <dt>

[**"Methode"-Methode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-setrdpproperty)
</dt> <dd>

Legt den Wert einer einzelnen benannten RDP-Eigenschaft fest.

</dd> </dl>
