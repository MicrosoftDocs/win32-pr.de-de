# <a name="imstscax-interface"></a>IMsTscAx-Schnittstelle

Ermöglicht es Ihnen, das Clientsteuerobjekt zu verbinden und zu trennen, virtuelle Kanalobjekte zu erstellen und Daten über einen virtuellen Kanal zu senden. Die -Schnittstelle enthält auch Methoden zum Abrufen und Festlegen von Eigenschaften, die mit dem Clientsteuersatz verknüpft sind. Diese Methoden sind auch über die [**IMsRdpClient-Schnittstelle verfügbar.**](imsrdpclient-interface.md)

## <a name="members"></a>Member

Die **IMsTscAx-Schnittstelle** erbt von der [**IDispatch-Schnittstelle.**](/windows/win32/api/oaidl/nn-oaidl-idispatch) **IMsTscAx** verfügt auch über diese Typen von Membern:

-   [Methoden](#methods)
-   [Eigenschaften](#properties)

### <a name="methods"></a>Methoden

Die **IMsTscAx-Schnittstelle** verfügt über diese Methoden.



| Methode                                                          | Beschreibung                                                                                                                                                                                           |
|:----------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Verbinden**](imstscax-connect.md)                             | Initiiert eine Verbindung mithilfe der Eigenschaften, die derzeit für das -Steuerelement festgelegt sind.<br/>                                                                                                                  |
| [**CreateVirtualChannels**](imstscax-createvirtualchannels.md) | Erstellt ein clientseitiges virtuelles Kanalobjekt für jeden angegebenen Namen des virtuellen Kanals.<br/>                                                                                                      |
| [**Trennen**](imstscax-disconnect.md)                       | Trennt die aktive Verbindung.<br/>                                                                                                                                                         |
| [**SendOnVirtualChannel**](imstscax-sendonvirtualchannel.md)   | Sendet Daten an den RD-Sitzungshost-Server über einen virtuellen Kanal, der zuvor mithilfe der [**IMsTscAx::CreateVirtualChannels-Methode erstellt**](imstscax-createvirtualchannels.md) wurde.<br/> |
