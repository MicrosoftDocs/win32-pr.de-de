# <a name="imsrdpclientnonscriptable7disabledpicursorscalingforprocess-method"></a>IMsRdpClientNonScriptable7::D isabledpicurrsorscalingforprocess-Methode

Deaktiviert die lokale Skalierung des Mauszeigers, der vom Server empfangen wurde. Dadurch wird sichergestellt, dass die Cursor Form ohne Änderung korrekt gerendert wird.

## <a name="syntax"></a>Syntax

```C++
HRESULT DisableDpiCursorScalingForProcess();
```

## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt **\_ OK** zurück, wenn erfolgreich.
