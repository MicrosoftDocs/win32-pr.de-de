# <a name="remote-desktop-protocol-provider-unions"></a>Remotedesktopprotokoll Anbieter-Unions

Die benutzerdefinierte Remoteprotokoll-API unterstützt die folgenden Unions.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**VERBINDUNGSEINSTELLUNG FÜR WRDS \_ \_**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_connection_setting)
</dt> <dd>

Enthält verschiedene Einstellungsebenen für eine Remotedesktopverbindung.

</dd> <dt>

[**\_WRDS-LISTENEREINSTELLUNG \_**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_listener_setting)
</dt> <dd>

Enthält verschiedene Ebenen von Listenereinstellungen für eine Remotedesktopverbindung.

</dd> <dt>

[**\_WRDS-EINSTELLUNG**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_setting)
</dt> <dd>

Enthält verschiedene Ebenen richtlinienbezogener Einstellungen für eine Remotedesktopverbindung.

</dd> <dt>

[**\_WTS CACHE \_ STATS \_ UN**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wts_cache_stats_un)
</dt> <dd>

Enthält Cachestatistiken.

</dd> </dl>
