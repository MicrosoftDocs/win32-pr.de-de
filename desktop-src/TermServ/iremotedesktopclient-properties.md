# <a name="iremotedesktopclient-properties"></a>Iremotedesktopclient-Eigenschaften

Die [**iremotedesktopclient**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclient) -Schnittstelle macht die folgenden Eigenschaften verfügbar.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Actions-Eigenschaft**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-get_actions)
</dt> <dd>

Ruft das Aktions Objekt für den Remotedesktopprotokoll (RDP)-app-Container Client ab.

</dd> <dt>

[**Settings-Eigenschaft**](iremotedesktopclient-settings.md)
</dt> <dd>

Ruft das Einstellungs Objekt für den Remotedesktopprotokoll (RDP)-app-Container Client ab.

</dd> <dt>

[**Touchpointer (Eigenschaft)**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-get_touchpointer)
</dt> <dd>

Enthält das [**remotedesktopclienttouchpointer**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclienttouchpointer) -Objekt für den Remotedesktopprotokoll (RDP)-app-Container Client.

</dd> </dl>
