# <a name="remoteapp-and-desktop-connection-management-service-structures"></a>RemoteApp- und Desktopverbindungsverwaltung Dienststrukturen

Die RemoteApp- und Desktopverbindungsverwaltung-Dienst-API stellt die folgenden Strukturen bereit.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**pluginResource**](/windows/win32/api/tspubplugincom/ns-tspubplugincom-pluginresource)
</dt> <dd>

Enthält Informationen zu einer Ressource, die Benutzern in RemoteApp- und Desktopverbindung zugewiesen werden kann.

</dd> <dt>

[**pluginResource2**](/windows/win32/api/tspubplugin2com/ns-tspubplugin2com-pluginresource2)
</dt> <dd>

Enthält zusätzliche Informationen zu einer Ressource, die Benutzern in RemoteApp- und Desktopverbindung zugewiesen werden kann.

</dd> <dt>

[**pluginResource2FileAssociation**](/windows/win32/api/tspubplugin2com/ns-tspubplugin2com-pluginresource2fileassociation)
</dt> <dd>

Enthält Informationen zu einer Dateizuordnung in RemoteApp- und Desktopverbindung.

</dd> </dl>
