# <a name="remote-desktop-management-services-provider"></a>Remotedesktop Management Services-Anbieter

Der RDMs-Anbieter (Remotedesktop Management Services) verwaltet virtuelle Desktop Infrastruktur (VDI)-Umgebungen.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Win32- \_ rdmscollectionproperties**](win32-rdmscollectionproperties.md)
</dt> <dd>

Verwaltet die Eigenschaften einer Sammlung virtueller Desktops.

</dd> <dt>

[**Win32 \_ rdmsdeploymentsettings**](win32-rdmsdeploymentsettings.md)
</dt> <dd>

Verwaltet die Bereitstellungs Einstellungen für eine Sammlung virtueller Desktops.

</dd> <dt>

[**Win32- \_ rdmsdesktopzuweisung**](win32-rdmsdesktopassignment.md)
</dt> <dd>

Beschreibt die Benutzer Desktop Zuweisung für die RD-Sammlung.

</dd> <dt>

[**Win32- \_ rdmsenvironment**](win32-rdmsenvironment.md)
</dt> <dd>

Verwaltet eine RDMs-Umgebung (Remotedesktop Management Services).

</dd> <dt>

[**Win32 \_ rdmsfiletypeassociation**](win32-rdmsfiletypeassociation.md)
</dt> <dd>

Verwaltet eine Dateityp Zuordnung für eine veröffentlichte Anwendung.

</dd> <dt>

[**Win32 \_ rdmsjoinednode**](win32-rdmsjoinednode.md)
</dt> <dd>

Stellt einen Server Knoten dar, der von Remotedesktop Verwaltungsdienste (RDMs) verwaltet wird.

</dd> <dt>

[**Win32 \_ rdmsmanagementdata**](win32-rdmsmanagementdata.md)
</dt> <dd>

Synchronisiert Veröffentlichungsdaten für Remotedesktop Management Services (RDMs).

</dd> <dt>

[**Win32- \_ rdmspublishedapplikation**](win32-rdmspublishedapplication.md)
</dt> <dd>

Verwaltet eine veröffentlichte Anwendung.

</dd> <dt>

[**Win32 \_ rdmsvirtualdesktop**](win32-rdmsvirtualdesktop.md)
</dt> <dd>

Stellt einen virtuellen Desktop dar.

</dd> <dt>

[**Win32 \_ rdmsvirtualdesktopcollection**](win32-rdmsvirtualdesktopcollection.md)
</dt> <dd>

Erstellt und verwaltet eine Sammlung virtueller Desktops.

</dd> <dt>

[**Win32- \_ rdshcollection**](win32-rdshcollection.md)
</dt> <dd>

Verwaltet eine Auflistung von Remotedesktop Sitzungs Hosts.

</dd> <dt>

[**Win32- \_ rdshserver**](win32-rdshserver.md)
</dt> <dd>

Verwaltet einen Remotedesktop-Sitzungshost Server (RDSH).

</dd> <dt>

[**Win32 \_ virtualdesktopsession**](win32-virtualdesktopsession.md)
</dt> <dd>

Verwaltet eine virtuelle Desktop Sitzung.

</dd> </dl>
