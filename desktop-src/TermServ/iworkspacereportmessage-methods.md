# <a name="iworkspacereportmessage-methods"></a>IWorkspaceReportMessage-Methoden

Die [**IWorkspaceReportMessage-Schnittstelle**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacereportmessage) unterstützt die folgenden Methoden.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**RegisterErrorLogMessage-Methode**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-registererrorlogmessage)
</dt> <dd>

Registriert die angegebene Fehlermeldung für die Verwendung in der Benutzeroberfläche.

</dd> <dt>

[**IsErrorMessageRegistered-Methode**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-iserrormessageregistered)
</dt> <dd>

Bestimmt, ob eine angegebene Fehlermeldung in einem angegebenen Arbeitsbereich registriert ist.

</dd> <dt>

[**RegisterErrorEvent-Methode**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-registererrorevent)
</dt> <dd>

Registriert die angegebene Fehlermeldung für die Verwendung in der Benutzeroberfläche.

</dd> </dl>
