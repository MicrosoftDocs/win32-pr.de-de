# <a name="imsrdpclientadvancedsettings8-properties"></a>IMsRdpClientAdvancedSettings8-Eigenschaften

Die [**IMsRdpClientAdvancedSettings8**](imsrdpclientadvancedsettings8.md) -Schnittstelle macht die folgenden Eigenschaften verfügbar.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Bandwidtherkennungs (Eigenschaft)**](imsrdpclientadvancedsettings8-bandwidthdetection.md)
</dt> <dd>

Gibt an, ob Bandbreiten Änderungen automatisch erkannt werden.

</dd> <dt>

[**Clientprotocolspec (Eigenschaft)**](imsrdpclientadvancedsettings8-clientprotocolspec.md)
</dt> <dd>

Gibt das Remote Desktop Protokoll an, das zwischen dem Client und dem Server verwendet wird.

</dd> </dl>
