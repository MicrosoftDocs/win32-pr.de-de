# <a name="imsrdpclientadvancedsettings8-properties"></a>IMsRdpClientAdvancedSettings8-Eigenschaften

Die [**IMsRdpClientAdvancedSettings8-Schnittstelle**](imsrdpclientadvancedsettings8.md) macht die folgenden Eigenschaften verfügbar.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**BandwidthDetection-Eigenschaft**](imsrdpclientadvancedsettings8-bandwidthdetection.md)
</dt> <dd>

Gibt an, ob Bandbreitenänderungen automatisch erkannt werden.

</dd> <dt>

[**ClientProtocolSpec-Eigenschaft**](imsrdpclientadvancedsettings8-clientprotocolspec.md)
</dt> <dd>

Gibt das Remotedesktopprotokoll an, das zwischen dem Client und dem Server verwendet wird.

</dd> </dl>
