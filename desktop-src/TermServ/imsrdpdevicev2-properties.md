# <a name="imsrdpdevicev2-properties"></a>IMsRdpDeviceV2-Eigenschaften

Die [**IMsRdpDeviceV2-Schnittstelle**](imsrdpdevicev2.md) macht die folgenden Eigenschaften verfügbar.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**CmClassGuid-Eigenschaft**](imsrdpdevicev2-cmclassguid.md)
</dt> <dd>

Enthält die Configuration Manager-Setupklassen-GUID für das Gerät.

</dd> <dt>

[**CmDeviceInstance-Eigenschaft**](imsrdpdevicev2-cmdeviceinstance.md)
</dt> <dd>

Enthält die Configuration Manager-Geräteinstanz des Geräts.

</dd> <dt>

[**DeviceText-Eigenschaft**](imsrdpdevicev2-devicetext.md)
</dt> <dd>

Enthält den Gerätetext.

</dd> <dt>

[**DriveLetterBitmap-Eigenschaft**](imsrdpdevicev2-driveletterbitmap.md)
</dt> <dd>

Enthält ein Bitfeld, das eine Zuordnung von Laufwerkbuchstaben darstellt, die im Gerät enthalten sind.

</dd> <dt>

[**IsCompositeDevice-Eigenschaft**](imsrdpdevicev2-iscompositedevice.md)
</dt> <dd>

Gibt an, ob das Gerät ein zusammengesetztes Gerät ist.

</dd> <dt>

[**IsOptionalDevice-Eigenschaft**](imsrdpdevicev2-isoptionaldevice.md)
</dt> <dd>

Gibt an, ob das Gerät für die USB-Umleitung optional ist.

</dd> <dt>

[**IsUSBDevice-Eigenschaft**](imsrdpdevicev2-isusbdevice.md)
</dt> <dd>

Gibt an, ob das Gerät für die USB-Umleitung vorgesehen ist.

</dd> </dl>
