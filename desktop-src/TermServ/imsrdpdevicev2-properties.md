# <a name="imsrdpdevicev2-properties"></a>IMsRdpDeviceV2-Eigenschaften

Die [**IMsRdpDeviceV2**](imsrdpdevicev2.md) -Schnittstelle macht die folgenden Eigenschaften verfügbar.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Cmclassguid (Eigenschaft)**](imsrdpdevicev2-cmclassguid.md)
</dt> <dd>

Enthält die Configuration Manager-Setup Klassen-GUID für das Gerät.

</dd> <dt>

[**Cmdebug-Eigenschaft**](imsrdpdevicev2-cmdeviceinstance.md)
</dt> <dd>

Enthält die Configuration Manager-Geräte Instanz des Geräts.

</dd> <dt>

[**Devicetext-Eigenschaft**](imsrdpdevicev2-devicetext.md)
</dt> <dd>

Enthält den Geräte Text.

</dd> <dt>

[**Driveletterbitmap (Eigenschaft)**](imsrdpdevicev2-driveletterbitmap.md)
</dt> <dd>

Enthält ein Bitfeld, das eine Zuordnung der im Gerät enthaltenen Laufwerk Buchstaben darstellt.

</dd> <dt>

[**Iscompositedevice (Eigenschaft)**](imsrdpdevicev2-iscompositedevice.md)
</dt> <dd>

Gibt an, ob es sich um ein zusammengesetztes Gerät handelt.

</dd> <dt>

[**Isoptionaldevice (Eigenschaft)**](imsrdpdevicev2-isoptionaldevice.md)
</dt> <dd>

Gibt an, ob das Gerät für die USB-Umleitung optional ist.

</dd> <dt>

[**Isusbdevice-Eigenschaft**](imsrdpdevicev2-isusbdevice.md)
</dt> <dd>

Gibt an, ob das Gerät für die USB-Umleitung vorgesehen ist.

</dd> </dl>
