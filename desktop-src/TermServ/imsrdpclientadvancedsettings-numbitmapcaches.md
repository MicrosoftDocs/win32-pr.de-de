# <a name="imsrdpclientadvancedsettingsnumbitmapcaches-property"></a>Imsrdpclientadvancedsettings:: numbitmapcaches (Eigenschaft)

Diese Eigenschaft wird nicht unterstützt.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="syntax"></a>Syntax


```C++
HRESULT put_NumBitmapCaches(
  [in]  LONG numBitmapCaches
);

HRESULT get_NumBitmapCaches(
  [out] LONG *pnumBitmapCaches
);
```



## <a name="property-value"></a>Eigenschaftswert

Die neue Anzahl von Caches. Der Standardwert ist 3. Dies führt zu einer optimalen Leistung.

## <a name="error-codes"></a>Fehlercodes

Gibt **" \_ false**" zurück.

## <a name="remarks"></a>Bemerkungen
