# <a name="remote-desktop-connection-broker-client-api"></a>Remotedesktopverbindung Broker-Client-API

Der Remotedesktopverbindung Broker ist eine Komponente der VDI-Bereitstellung (Virtual Desktop Infrastructure), die für eine Remote Verbindung ein geeignetes Ziel aus einem Pool von Computern findet, in dem die Verbindung umgeleitet werden kann, und die IP-Adresse des Ziels zurückgibt. Die Verbindungs Broker-Client-API ermöglicht es Protokoll Anbietern von Drittanbietern, den Verbindungs Broker zu nutzen, um die Verarbeitung von Verbindungen mithilfe Ihres Protokolls für virtuelle Maschinen oder Remotedesktop Server in einer Farm zu beschleunigen.
