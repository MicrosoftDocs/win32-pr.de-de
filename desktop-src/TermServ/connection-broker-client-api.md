# <a name="remote-desktop-connection-broker-client-api"></a>Remotedesktopverbindung Broker-Client-API

Der Remotedesktopverbindung Broker ist eine Komponente der VDI-Bereitstellung (Virtual Desktop Infrastructure), die für eine Remoteverbindung ein geeignetes Ziel aus einem Pool von Computern findet, in dem die Verbindung umgeleitet werden kann, und die IP-Adresse des Ziels zurückgibt. Die Client-API des Verbindungsbrokers ermöglicht es Protokollanbietern von Drittanbietern, den Verbindungsbroker zu nutzen, um die Verarbeitung von Verbindungen mithilfe ihres Protokolls für virtuelle Computer oder Remotedesktop Server in einer Farm zu beschleunigen.
