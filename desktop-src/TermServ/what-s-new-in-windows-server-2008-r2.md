# <a name="whats-new-in-windows-server-2008-r2"></a>Neues in Windows Server 2008 R2

Windows Server 2008 R2 führt die folgenden neuen Programmierelemente für Remotedesktopdienste.




| | | Terminaldienste sind jetzt Remotedesktopdienste<br /> | Terminaldienste wurden in Remotedesktopdienste. Ein Server, auf dem der Remotedesktop-Sitzungshost-Rollendienst (RD-Sitzungshost) installiert ist, wird jetzt als RD-Sitzungshost Server und nicht als Terminalserver bezeichnet.<br /><ul><li><a href="terminal-services-is-now-remote-desktop-services.md">Terminaldienste sind jetzt Remotedesktopdienste</a></li></ul> | | Remotedesktop-Virtualisierungs-API<br /> | Die Remotedesktop-Virtualisierungs-API stellt Enumerationen, Schnittstellen und Strukturen bereit, mit denen Sie benutzerdefinierte Plug-Ins erstellen können, die die Standardumleitungslogik von Remotedesktopverbindung Broker (RD-Verbindungsbroker) überschreiben. Der RD-Verbindungsbroker (früher als TS-Sitzungsbroker bezeichnet) wurde erweitert, um Verbindungen mit virtuellen Computern zu unterstützen.<br /><ul><li><a href="using-the-remote-desktop-virtualization-api.md">Verwenden der Remotedesktop-Virtualisierungs-API</a></li><li><a href="terminal-services-virtualization-api-reference.md">Remotedesktop Virtualisierungs-API-Referenz</a></li></ul> | | Remotedesktopprotokoll Provider<br /> | Die Remotedesktopprotokoll-Anbieter-API kann verwendet werden, um ein Protokoll zu erstellen, das die Interaktion zwischen dem Remotedesktopdienste-Dienst und Desktopclients anpasst.<br /><ul><li><a href="creating-a-custom-remote-protocol.md">Erstellen eines Remotedesktopprotokoll Anbieters</a></li><li><a href="custom-remote-protocol-reference.md">Remotedesktopprotokoll Anbieterreferenz</a></li></ul> | | Remotedesktopdienste AudioEndpoint-API<br /> | Die Remotedesktopdienste AudioEndpoint-API unterstützt Enumerationen, Schnittstellen und Strukturen für die Registrierung von Audioendpunkten und den Datentransport.<br /><ul><li><a href="terminal-services-audioendpoint-api-reference.md">Remotedesktopdienste AudioEndpoint-API-Referenz</a></li></ul> | | RemoteApp- und Desktopverbindungsverwaltung-Dienst-API<br /> | Die RemoteApp- und Desktopverbindungsverwaltung-Dienst-API unterstützt Schnittstellen und Strukturen, die Sie verwenden können, um benutzerdefinierte Filterung von Ressourcen durchzuführen und Unterstützung für Dateitypen zu bieten, die in Windows Server 2008 R2 nicht nativ unterstützt werden.<br /><ul><li><a href="centralized-publishing-api-reference.md">RemoteApp- und Desktopverbindungsverwaltung Service-API-Referenz</a></li></ul> |
