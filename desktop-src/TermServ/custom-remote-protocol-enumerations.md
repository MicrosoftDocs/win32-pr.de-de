# <a name="remote-desktop-protocol-provider-enumerations"></a>Remotedesktopprotokoll Anbieterenumeration

Die folgenden Enumerationstypen werden von der Remotedesktopprotokoll Provider-API unterstützt.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**\_ \_ WRDS-VERBINDUNGSEINSTELLUNGSEBENE \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_connection_setting_level)
</dt> <dd>

Gibt den Strukturtyp an, der im **WRdsConnectionSetting-Member** der [**WRDS \_ CONNECTION \_ SETTINGS-Struktur enthalten**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_connection_settings) ist.

</dd> <dt>

[**EINSTELLUNGSEBENE DES \_ \_ WRDS-LISTENERS \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_listener_setting_level)
</dt> <dd>

Wird verwendet, um den Strukturtyp anzugeben, der im **WRdsListenerSetting-Member** der [**WRDS \_ LISTENER \_ SETTINGS-Struktur enthalten**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_listener_settings) ist.

</dd> <dt>

[**\_WRDS-EINSTELLUNGSEBENE \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_level)
</dt> <dd>

Gibt den Typ der -Struktur an, der im **WRdsSetting-Member** der [**WRDS \_ SETTINGS-Struktur enthalten**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings) ist.

</dd> <dt>

[**\_ \_ WRDS-EINSTELLUNGSSTATUS**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_status)
</dt> <dd>

Gibt den Status einer Richtlinieneinstellung für verschiedene Member der [**WRDS \_ SETTINGS \_ 1-Struktur**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings_1) an.

</dd> <dt>

[**\_ \_ WRDS-EINSTELLUNGSTYP**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_type)
</dt> <dd>

Gibt die Kategorie der Einstellungen an, die in einer [**WRDS \_ SETTINGS-Struktur gespeichert**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings) werden.

</dd> <dt>

[**\_WTS \_CERT-TYP**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_cert_type)
</dt> <dd>

Enthält Werte, die den Typ des Zertifikats angeben, das zum Abrufen einer Lizenz verwendet wird.

</dd> <dt>

[**\_WTS \_UMLEITUNGSANTWORT FÜR \_ \_ ANMELDEFEHLER**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_logon_error_redirector_response)
</dt> <dd>

Enthält Werte, die die bevorzugte Antwort des Protokolls auf einen Anmeldefehler angeben.

</dd> <dt>

[**\_WTS \_RCM-ENTLEERUNGSSTATUS \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_rcm_drain_state)
</dt> <dd>

Enthält Informationen zum Entleerungsstatus des Remotedesktop-Sitzungshost (RD-Sitzungshost Server).

</dd> <dt>

[**\_WTS \_RCM-DIENSTSTATUS \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_rcm_service_state)
</dt> <dd>

Enthält Informationen zum Status des Remotedesktopdienste Diensts.

</dd> </dl>
