# <a name="remote-desktop-protocol-provider-enumerations"></a>Remotedesktopprotokoll Anbieter Enumerationen

Die folgenden Enumerationstypen werden von der Remotedesktopprotokoll Anbieter-API unterstützt.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**wrds- \_ Verbindungs \_ Einstellungs \_ Ebene**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_connection_setting_level)
</dt> <dd>

Gibt den Typ der Struktur an, die im **wrdsconnectionsetting** -Member der [**wrds- \_ Verbindungs \_ Einstellungs**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_connection_settings) Struktur enthalten ist.

</dd> <dt>

[**wrds- \_ Listener- \_ Einstellungs \_ Ebene**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_listener_setting_level)
</dt> <dd>

Wird verwendet, um den Strukturtyp anzugeben, der im **wrdslistenersetting** -Member der [**wrds- \_ \_ listenereinstellungs**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_listener_settings) -Struktur enthalten ist.

</dd> <dt>

[**wrds- \_ Einstellungs \_ Ebene**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_level)
</dt> <dd>

Gibt den Typ der Struktur an, die im **wrdssetting** -Member der [**wrds- \_ Einstellungs**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings) Struktur enthalten ist.

</dd> <dt>

[**wrds- \_ Einstellungs \_ Status**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_status)
</dt> <dd>

Gibt den Status einer Richtlinien Einstellung für verschiedene Mitglieder der wrds-Struktur " [**\_ Settings \_ 1**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings_1) " an.

</dd> <dt>

[**wrds \_ - \_ Einstellungstyp**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_type)
</dt> <dd>

Gibt die Kategorie der Einstellungen an, die in einer [**wrds- \_ Einstellungs**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings) Struktur gespeichert werden.

</dd> <dt>

[**WTS- \_ CERT- \_ Typ**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_cert_type)
</dt> <dd>

Enthält Werte, die den Typ des zum Abrufen einer Lizenz verwendeten Zertifikats angeben.

</dd> <dt>

[**Anmeldungs \_ \_ Fehler \_ Redirector- \_ Antwort von WTS**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_logon_error_redirector_response)
</dt> <dd>

Enthält Werte, die die bevorzugte Antwort des Protokolls auf einen Anmeldefehler angeben.

</dd> <dt>

[**\_RCM- \_ Ausgleichs \_ Status der WTS**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_rcm_drain_state)
</dt> <dd>

Enthält Informationen zum Ausgleichs Status des Remotedesktop-Sitzungshost (RD-Sitzungshost)-Servers.

</dd> <dt>

[**Status der WTS \_ RCM- \_ Dienste \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_rcm_service_state)
</dt> <dd>

Enthält Informationen zum Status des Remotedesktopdienste Dienstanbieter.

</dd> </dl>
