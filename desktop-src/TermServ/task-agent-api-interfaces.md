# <a name="personal-desktop-task-agent-api-interfaces"></a>API-Schnittstellen des Personal Desktop-Task-Agents

Die Agent-API für den persönlichen Desktop-Task unterstützt die folgenden Schnittstellen.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**IRDVTaskPluginNotifySink**](irdvtaskpluginnotifysink.md)
</dt> <dd>

Die [**IRDVTaskPluginNotifySink-Schnittstelle**](irdvtaskpluginnotifysink.md) wird vom Task-Agent für die Kommunikation mit dem Trigger-Agent verwendet.

</dd> <dt>

[**IRDVTaskPlugin**](irdvtaskplugin.md)
</dt> <dd>

Die [**IRDVTaskPlugin-Schnittstelle**](irdvtaskplugin.md) wird von  einem Task-Agent für die Aktualisierung virtueller Computer implementiert, damit der Task-Agent Systemupdates für einen virtuellen Computer verwalten kann.

</dd> </dl>
