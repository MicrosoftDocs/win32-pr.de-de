# <a name="scriptable-virtual-channels"></a>Skriptfähige virtuelle Kanäle

Informationen zur Verwendung von skriptfähigen virtuellen Kanälen finden Sie unter [Implementing Scriptable Virtual Channels Using Remotedesktop-Webverbindung](implementing-scriptable-virtual-channels-using-remote-desktop-web-connection.md).

Informationen zu virtuellen Kanälen finden Sie unter [Remotedesktopdienste Virtuelle Kanäle.](terminal-services-virtual-channels.md)

**Windows XP:** Windows XP enthält Remotedesktop-Webverbindung, bei dem es sich um eine aktualisierte Version der Terminaldienste Erweiterter Client (TSAC) handelt. Remotedesktop-Webverbindung unterstützt [Remotedesktopprotokoll](remote-desktop-protocol.md) (RDP) 5.1 und ist abwärtskompatibel mit RDP 5.0.
