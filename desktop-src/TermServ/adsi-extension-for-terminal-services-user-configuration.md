# <a name="adsi-extension-for-remote-desktop-services-user-configuration"></a>ADSI-Erweiterung für Remotedesktopdienste Benutzerkonfiguration

Eine ADSI-Erweiterung (Active Directory Service Interface) ist ein spezielles COM-Objekt, das die Erweiterung eines COM-Objekts ermöglicht. ADSI-Erweiterungen können einem Verzeichnisobjekt Funktionen hinzufügen und Verwaltungsaufgaben vereinfachen.

Die ADSI-Erweiterung für Remotedesktopdienste-Benutzerkonfiguration erweitert ein ADSI-Objekt, mit dem Domänenadministratoren Skripts schreiben können, die die Wartung Remotedesktopdienste benutzerspezifischen Eigenschaften automatisieren.

Die ADSI-Erweiterung für Remotedesktopdienste-Benutzerkonfiguration ist in Windows Server enthalten.

Weitere Informationen finden Sie in den folgenden Themen:

-   [Verwenden der ADSI-Erweiterung für Remotedesktopdienste Benutzerkonfiguration](using-the-adsi-extension-for-terminal-services-user-configuration.md)
-   [Referenz für die ADSI-Erweiterung für Remotedesktopdienste Benutzerkonfiguration](reference-for-the-adsi-extension-for-terminal-services-user-configuration.md)

Allgemeine Informationen zu ADSI finden Sie unter [ADSI Extensions](/windows/desktop/ADSI/adsi-extensions), [The ADSI Attribute Cache](/windows/desktop/ADSI/the-adsi-attribute-cache)und Active Directory Service [Interfaces](/windows/desktop/ADSI/active-directory-service-interfaces-adsi).
