# <a name="adsi-extension-for-remote-desktop-services-user-configuration"></a>ADSI-Erweiterung für Remotedesktopdienste Benutzerkonfiguration

Eine Erweiterung der Active Directory Service Interface (ADSI) ist ein spezielles com-Objekt, das die Erweiterung eines COM-Objekts ermöglicht. ADSI-Erweiterungen können einem Verzeichnis Objektfunktionen hinzufügen und Verwaltungsaufgaben vereinfachen.

Die ADSI-Erweiterung für Remotedesktopdienste Benutzerkonfiguration erweitert ein ADSI-Objekt, das Domänen Administratoren das Schreiben von Skripts ermöglicht, mit denen die Wartung von Remotedesktopdienste spezifischen Benutzereigenschaften automatisiert werden kann.

Die ADSI-Erweiterung für Remotedesktopdienste Benutzerkonfiguration ist in Windows Server enthalten.

Weitere Informationen finden Sie unter den folgenden Themen:

-   [Verwenden der ADSI-Erweiterung für Remotedesktopdienste Benutzerkonfiguration](using-the-adsi-extension-for-terminal-services-user-configuration.md)
-   [Referenz für die ADSI-Erweiterung für Remotedesktopdienste Benutzerkonfiguration](reference-for-the-adsi-extension-for-terminal-services-user-configuration.md)

Allgemeine Informationen zu ADSI finden Sie unter [ADSI-Erweiterungen](/windows/desktop/ADSI/adsi-extensions), [ADSI-Attribut Cache](/windows/desktop/ADSI/the-adsi-attribute-cache)und [Active Directory-Dienst Schnittstellen](/windows/desktop/ADSI/active-directory-service-interfaces-adsi).
