# <a name="remote-desktop-license-server-classes"></a>Remotedesktop Lizenzserverklassen

Der Remotedesktop Lizenzserver-WMI-Anbieter stellt die folgenden Klassen zur Folge.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Win32 \_ TSIssuedLicense**](win32-tsissuedlicense.md)
</dt> <dd>

Beschreibt Instanzen von Remotedesktopdienste Clientzugriffslizenzen pro Gerät (RDS per Device CALs), die von einem Remotedesktop ausgestellt werden.

</dd> <dt>

[**Win32 \_ TSLicenseKeyPack**](win32-tslicensekeypack.md)
</dt> <dd>

Stellt Methoden und Eigenschaften zum Anzeigen und Installieren Remotedesktopdienste Lizenzschlüsselpaketen zur Auswahl.

</dd> <dt>

[**Win32 \_ TSLicenseReport**](win32-tslicensereport.md)
</dt> <dd>

Stellt Instanzen von REMOTEDESKTOPDIENSTE Pro Benutzer-Clientzugriffslizenz (RDS Per User CAL) zur Verwendung von Auf dem Remotedesktop-Lizenzserver generierten Sowie Methoden für Generierungs-, Abruf- und Löschvorgänge von Lizenzberichten zur Anwendung.

</dd> <dt>

[**Win32 \_ TSLicenseReportEntry**](win32-tslicensereportentry.md)
</dt> <dd>

Enthält Details zur ausgestellten Remotedesktopdienste Clientzugriffslizenz pro Benutzer (RDS per User CAL).

</dd> <dt>

[**Win32 \_ TSLicenseReportFailedPerUserEntry**](win32-tslicensereportfailedperuserentry.md)
</dt> <dd>

Enthält Details zur fehlerhaften Remotedesktopdienste client access license (RDS Per User CAL) (Clientzugriffslizenz pro Benutzer (RDS pro Benutzer)).

</dd> <dt>

[**Win32 \_ TSLicenseReportFailedPerUserSummaryEntry**](win32-tslicensereportfailedperusersummaryentry.md)
</dt> <dd>

Enthält Details zu den Domänen, für die bei der Ausstellung pro Benutzer ein Fehler auft war.

</dd> <dt>

[**Win32 \_ TSLicenseReportPerDeviceEntry**](win32-tslicensereportperdeviceentry.md)
</dt> <dd>

Enthält Details zur fehlgeschlagenen Remotedesktopdienste Clientzugriffslizenz pro Gerät (RDS pro Geräte-CAL).

</dd> <dt>

[**Win32 \_ TSLicenseReportSummaryEntry**](win32-tslicensereportsummaryentry.md)
</dt> <dd>

Enthält eine Zusammenfassung der installierten und ausgestellten Remotedesktopdienste Clientzugriffslizenzen pro Benutzer (RDS-CALs pro Benutzer).

</dd> <dt>

[**Win32 \_ TSLicenseServer**](win32-tslicenseserver.md)
</dt> <dd>

Stellt Methoden und Eigenschaften zum Anzeigen und Konfigurieren Remotedesktop Lizenzierung (RD-Lizenzierung) auf einem Remotedesktop Lizenzservers zur Seite.

</dd> </dl>
