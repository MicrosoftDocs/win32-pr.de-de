# <a name="remote-desktop-license-server-classes"></a>Remotedesktop-Lizenz Server Klassen

Der Remotedesktop-Lizenz Server-WMI-Anbieter stellt die folgenden Klassen bereit.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Win32-" \_ tsissuedlicense"**](win32-tsissuedlicense.md)
</dt> <dd>

Beschreibt Instanzen von Remotedesktopdienste pro Geräte-Client Zugriffs Lizenzen (RDS pro Gerät CALs), die von einem Remotedesktop Lizenzserver ausgestellt werden.

</dd> <dt>

[**Win32-Schlüssel-Lizenz-Schlüssel \_ ACK**](win32-tslicensekeypack.md)
</dt> <dd>

Stellt Methoden und Eigenschaften zum Anzeigen und installieren Remotedesktopdienste Lizenzschlüssel Packs bereit.

</dd> <dt>

[**Win32-Datei- \_ /licensereport**](win32-tslicensereport.md)
</dt> <dd>

Stellt Instanzen von Remotedesktopdienste Client Zugriffslizenz-Verwendungs Berichte pro Benutzer (RDS pro Benutzer) bereit, die auf dem Remotedesktop-Lizenzserver generiert werden, sowie Methoden zum generieren, abrufen und Löschen von Lizenz Berichten.

</dd> <dt>

[**Win32-Wert für "- \_ Lizenzserver"**](win32-tslicensereportentry.md)
</dt> <dd>

Stellt Details der pro Benutzer-Client Zugriffslizenz ausgestellten Remotedesktopdienste (RDS pro Benutzer-CAL) bereit.

</dd> <dt>

[**Win32-Datei- \_ faisereportfailedperuserentry**](win32-tslicensereportfailedperuserentry.md)
</dt> <dd>

Enthält Details zur Client Zugriffslizenz für die fehlerhafte Remotedesktopdienste pro Benutzer (RDS pro Benutzer-CAL).

</dd> <dt>

[**Win32-Schlüssel- \_ /licensereportfailedperusersummaryentry**](win32-tslicensereportfailedperusersummaryentry.md)
</dt> <dd>

Gibt Details zu den Domänen an, für die pro Benutzer eine Ausstellung fehlgeschlagen ist.

</dd> <dt>

[**Win32-" \_ zlicensereportperdeviceentry"**](win32-tslicensereportperdeviceentry.md)
</dt> <dd>

Enthält Details zu den fehlerhaften Remotedesktopdienste pro Geräte-Client Zugriffslizenz (RDS pro Gerät CAL).

</dd> <dt>

[**Win32-Datei Eintrag ("-Name)" \_**](win32-tslicensereportsummaryentry.md)
</dt> <dd>

Bietet eine Zusammenfassung der installierten und ausgestellten Remotedesktopdienste pro Benutzer-Client Zugriffs Lizenzen (RDS pro Benutzer-CALs).

</dd> <dt>

[**Win32- \_ Lizenznehmer**](win32-tslicenseserver.md)
</dt> <dd>

Stellt Methoden und Eigenschaften zum Anzeigen und konfigurieren Remotedesktop Lizenzierung (RD-Lizenzierung) auf einem Remotedesktop Lizenzserver bereit.

</dd> </dl>
