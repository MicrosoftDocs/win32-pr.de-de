# <a name="itssbservicenotification-methods"></a>ITsSbServiceNotification-Methoden

Die [**ITsSbServiceNotification-Schnittstellen**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbservicenotification) machen die folgenden Methoden verfügbar.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**NotifyServiceFailure-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicefailure)
</dt> <dd>

Benachrichtigt registrierte Plug-Ins, dass der Remotedesktopverbindung Broker-Dienst (RD-Verbindungsbroker) beendet wurde.

</dd> <dt>

[**NotifyServiceSuccess-Methode**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicesuccess)
</dt> <dd>

Benachrichtigt registrierte Plug-Ins, dass der RD-Verbindungsbrokerdienst gestartet wurde.

</dd> </dl>
