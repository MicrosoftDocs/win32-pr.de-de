# <a name="imsrdpclientsecuredsettings2-interface"></a>IMsRdpClientSecuredSettings2-Schnittstelle

Definiert zusätzliche Eigenschaften des Remotedesktop ActiveX,die auf bestimmte URL Internet Explorer-Sicherheitszonen beschränkt sind.

## <a name="members"></a>Member

Die **IMsRdpClientSecuredSettings2-Schnittstelle** erbt von [**IMsRdpClientSecuredSettings**](imsrdpclientsecuredsettings-interface.md). **IMsRdpClientSecuredSettings2** verfügt auch über diese Membertypen:

-   [Eigenschaften](#properties)

### <a name="properties"></a>Eigenschaften

Die **IMsRdpClientSecuredSettings2-Schnittstelle** verfügt über diese Eigenschaften.



| Eigenschaft                                                   | Zugriffstyp           | Beschreibung                                                                                                          |
|:-----------------------------------------------------------|:----------------------|:---------------------------------------------------------------------------------------------------------------------|
| [**Pcb**](imsrdpclientsecuredsettings2-pcb.md)<br/> | Lesen/Schreiben<br/> | Gibt die Einstellung preconnection BLOB (BOARD) an, die vor dem Herstellen einer Verbindung für die Übertragung an den Server verwendet werden soll.<br/> |
