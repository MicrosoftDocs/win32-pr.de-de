Die Remotedesktopverbindung Broker-Client-API definiert die folgenden Schnittstellen.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**IConnectionBrokerClient**](iconnectionbrokerclient.md)
</dt> <dd>

</dd> <dt>

</dt> <dd>

</dd> </dl>
