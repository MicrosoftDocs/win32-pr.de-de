# <a name="imsrdppreferredredirectioninfo-interface"></a>IMsRdpPreferredRedirectionInfo-Schnittstelle

Stellt eine Eigenschaft zum Steuern mithilfe eines Umleitungsservers zur Anwendung.

## <a name="members"></a>Member

Die **IMsRdpPreferredRedirectionInfo-Schnittstelle** erbt von der [**IUnknown-Schnittstelle.**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) **IMsRdpPreferredRedirectionInfo** verfügt auch über diese Membertypen:

-   [Eigenschaften](#properties)

### <a name="properties"></a>Eigenschaften

Die **IMsRdpPreferredRedirectionInfo-Schnittstelle** verfügt über diese Eigenschaften.



| Eigenschaft                                                                                               | Zugriffstyp           | Beschreibung                                                          |
|:-------------------------------------------------------------------------------------------------------|:----------------------|:---------------------------------------------------------------------|
| [**UseRedirectionServerName**](imsrdppreferredredirectioninfo-useredirectionservername.md)<br/> | Lesen/Schreiben<br/> | Ruft ab und legt fest, ob der Umleitungsservername verwendet werden soll.<br/> |
