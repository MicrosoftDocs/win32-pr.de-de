# <a name="imsrdppreferredredirectioninfo-interface"></a>Imsrdppreferredredirectioninfo-Schnittstelle

Stellt eine Eigenschaft bereit, die mithilfe eines Umleitungs Servers gesteuert werden soll.

## <a name="members"></a>Member

Die **imsrdppreferredredirectioninfo** -Schnittstelle erbt von der [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) -Schnittstelle. **Imsrdppreferredredirectioninfo** verfügt auch über die folgenden Typen von Membern:

-   [Eigenschaften](#properties)

### <a name="properties"></a>Eigenschaften

Die **imsrdppreferredredirectioninfo** -Schnittstelle verfügt über diese Eigenschaften.



| Eigenschaft                                                                                               | Zugriffstyp           | BESCHREIBUNG                                                          |
|:-------------------------------------------------------------------------------------------------------|:----------------------|:---------------------------------------------------------------------|
| [**Useredirectionservername**](imsrdppreferredredirectioninfo-useredirectionservername.md)<br/> | Lesen/Schreiben<br/> | Ruft ab und legt fest, ob der Umleitungs Servername verwendet werden soll.<br/> |
