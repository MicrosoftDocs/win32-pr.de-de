# <a name="remote-desktop-virtual-machine-classes"></a>Klassen für den Remotedesktop virtuellen Computer

Der WMI-Anbieter für die Remotedesktop virtuellen Maschine stellt die folgenden Klassen bereit.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Win32-Datei- \_ virtualdesktopserversettings**](win32-tsvirtualdesktopserversettings.md)
</dt> <dd>

Enthält Konfigurationsinformationen für einen Remotedesktop-Virtualisierungshost Server (RD-Virtualisierungshost).

</dd> <dt>

[**Win32- \_ VM**](win32-tsvm.md)
</dt> <dd>

Stellt einen Remotedesktop virtuellen Computer dar.

</dd> <dt>

[**Win32- \_ TSVmMetadataAssociation**](win32-tsvmmetadataassociation.md)
</dt> <dd>

Stellt eine Zuordnung zwischen einer Remotedesktop virtuellen Maschine und ihren Metadateneigenschaften dar.

</dd> <dt>

[**Win32- \_ TSVmMetadataItem**](win32-tsvmmetadataitem.md)
</dt> <dd>

Stellt ein Metadatenelement für einen Remotedesktop virtuellen Computer dar.

</dd> <dt>

[**Win32- \_ rdvhmanagement**](win32-rdvhmanagement.md)
</dt> <dd>

Beschreibt einen Remotedesktop den Verwaltungsdienst für virtuelle Hosts (RDVH).

</dd> </dl>
