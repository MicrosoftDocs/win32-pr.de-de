# <a name="remote-desktop-virtual-machine-classes"></a>Remotedesktop Virtual Machine-Klassen

Der WMI-Anbieter Remotedesktop virtuellen Computers stellt die folgenden Klassen bereit.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Win32 \_ TSVirtualDesktopServerSettings**](win32-tsvirtualdesktopserversettings.md)
</dt> <dd>

Enthält Konfigurationsinformationen für einen Remotedesktop Virtualization Host-Server (RD Virtualization Host).

</dd> <dt>

[**Win32 \_ TSVm**](win32-tsvm.md)
</dt> <dd>

Stellt einen Remotedesktop virtuellen Computer dar.

</dd> <dt>

[**Win32 \_ TSVmMetadataAssociation**](win32-tsvmmetadataassociation.md)
</dt> <dd>

Stellt eine Zuordnung zwischen einem Remotedesktop virtuellen Computer und dessen Metadateneigenschaften dar.

</dd> <dt>

[**Win32 \_ TSVmMetadataItem**](win32-tsvmmetadataitem.md)
</dt> <dd>

Stellt ein Metadatenelement für einen Remotedesktop virtuellen Computer dar.

</dd> <dt>

[**Win32 \_ RdvhManagement**](win32-rdvhmanagement.md)
</dt> <dd>

Beschreibt einen Remotedesktop Virtual Host-Verwaltungsdienst (RDVH).

</dd> </dl>
