# <a name="method-call-sequence"></a>Methoden aufrufssequenz

Der Remotedesktopdienste-Dienst und Ihr Protokoll Anbieter werden einander in klar definierten Sequenzen aufgerufen, wie in den folgenden Themen beschrieben.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[Start Sequenz](start-sequence.md)
</dt> <dd>

Schritte zum Starten des benutzerdefinierten Protokolls.

</dd> <dt>

[Verbindungs Sequenz](connection-sequence.md)
</dt> <dd>

Eine Abbildung, in der die Methodenaufrufe zwischen dem Remotedesktopdienste-Dienst und Ihrem Protokoll während der Client Verbindungs Sequenz angezeigt werden.

</dd> <dt>

[Sequenz neu verbinden](reconnect-sequence.md)
</dt> <dd>

Eine Abbildung, in der die Methodenaufrufe angezeigt werden, die zwischen dem Remotedesktopdienste-Dienst und Ihrem Protokoll während einer neuverbindungs Sequenz vorgenommen werden.

</dd> </dl>
