# <a name="remote-desktop-services-audioendpoint-structures"></a>Remotedesktopdienste AudioEndpoint-Strukturen

Die folgenden Strukturen werden mit der AudioEndpoint-API verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**AKTUELLE \_ \_ AE-POSITION**](/windows/desktop/api/Audioengineendpoint/ns-audioengineendpoint-ae_current_position)
</dt> <dd>

Meldet die aktuelle Frameposition vom Gerät an die Clients.

</dd> <dt>

[**\_APO-VERBINDUNGSEIGENSCHAFT \_**](/windows/desktop/api/Audioapotypes/ns-audioapotypes-apo_connection_property)
</dt> <dd>

Enthält die dynamisch ändernden Verbindungseigenschaften.

</dd> </dl>

## <a name="remarks"></a>Hinweise

Die Remotedesktopdienste AudioEndpoint-API ist für die Verwendung in Remotedesktop Verwendet. Dies gilt nicht für Clientanwendungen.
