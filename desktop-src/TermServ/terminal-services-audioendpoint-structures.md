# <a name="remote-desktop-services-audioendpoint-structures"></a>Remotedesktopdienste audioendpoint-Strukturen

Die folgenden Strukturen werden mit der audioendpoint-API verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**aktuelle AE- \_ \_ Position**](/windows/desktop/api/Audioengineendpoint/ns-audioengineendpoint-ae_current_position)
</dt> <dd>

Meldet die aktuelle Frame Position vom Gerät an die Clients.

</dd> <dt>

[**APO- \_ Verbindungs \_ Eigenschaft**](/windows/desktop/api/Audioapotypes/ns-audioapotypes-apo_connection_property)
</dt> <dd>

Enthält die dynamisch veränderlichen Verbindungs Eigenschaften.

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

Die Remotedesktopdienste audioendpoint-API ist für die Verwendung in Remotedesktop Szenarien vorgesehen. Es handelt sich nicht um Client Anwendungen.
