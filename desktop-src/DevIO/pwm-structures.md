# <a name="pwm-structures"></a>PWM-Strukturen

\[Einige Informationen beziehen sich auf Vorabversionen, die vor der kommerziellen Freigabe grundlegend geändert werden können. Microsoft übernimmt keine Garantie, weder ausdrücklich noch stillschweigend, für die hier bereitgestellten Informationen.\]

Die folgenden Strukturen werden mit Pulse Width Pulse verwendet:

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                                                        | Beschreibung                                                                                                              |
|------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**PWM \_ CONTROLLER \_ GET \_ ACTUAL \_ PERIOD \_ OUTPUT**](pwm-controller-get-actual-period-output.md)<br/>                   | Enthält den effektiven Ausgabesignalzeitraum für einen PWM-Controller (Pulse Width Pw).<br/>                    |
| [**\_PWM-CONTROLLERINFORMATIONEN \_**](/windows/desktop/api/Pwm/ns-pwm-pwm_controller_info)<br/>                                                              | Stellt die statischen Informationen dar, die einen PWM-Controller (Pulse Width Pulse) charakterisieren. <br/>              |
| [**PWM \_ CONTROLLER \_ SET \_ DESIRED \_ PERIOD \_ INPUT**](/windows/desktop/api/Pwm/ns-pwm-pwm_controller_set_desired_period_input)<br/>                   | Enthält einen Eingabewert für einen vorgeschlagenen Signalzeitraum für den PWM-Controller (Pulse Width Pulse). <br/>       |
| [**PWM \_ CONTROLLER \_ SET \_ DESIRED \_ PERIOD \_ OUTPUT**](/windows/desktop/api/Pwm/ns-pwm-pwm_controller_set_desired_period_output)<br/>                 | Enthält den effektiven Ausgabesignalzeitraum des PWM-Controllers (Pulse Width Pw).<br/>                   |
| [**PWM \_ PIN \_ GET \_ ACTIVE \_ DUTY \_ CYCLE \_ PERCENTAGE \_ OUTPUT**](pwm-pin-get-active-duty-cycle-percentage-output.md)<br/> | Enthält den aktuellen Prozentsatz des Arbeitszyklus für eine Stecknadel oder einen Kanal in einem PWM-Controller (Pulse Width PwM).<br/> |
| [**PWM \_ PIN \_ SET \_ ACTIVE \_ DUTY \_ CYCLE \_ PERCENTAGE \_ INPUT**](pwm-pin-set-active-duty-cycle-percentage-input.md)<br/>   | Enthält einen gewünschten Prozentsatz für einen Arbeitszyklus für eine Stecknadel oder einen Kanal in einem PWM-Controller (Pulse Width PwM).<br/>   |
| [**PWM \_ PIN \_ GET \_ POLARITY \_ OUTPUT**](pwm-pin-get-polarity-output.md)<br/>                                            | Enthält einen zurück zu gebenden Polaritätswert.<br/>                                                                          |
| [**PWM \_ PIN \_ SET \_ POLARITY \_ INPUT**](/windows/desktop/api/Pwm/ns-pwm-pwm_pin_set_polarity_input)<br/>                                              | Enthält einen gewünschten Wert für die Polarität eines Pins oder Kanals.<br/>                                                    |
| [**PWM \_ PIN \_ IS \_ STARTED \_ OUTPUT**](pwm-pin-is-started-output.md)<br/>                                                | Enthält den aktuellen Signalgenerierungsstatus eines Pins.<br/>                                                        |
