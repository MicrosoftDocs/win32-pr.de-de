# <a name="about-device-management"></a>Informationen zur Geräteverwaltung

Das System verwendet Nachrichten, um Anwendungen über Geräteänderungen und Strom Änderungen zu benachrichtigen. Geräte Nachrichten Benachrichtigen Anwendungen über Geräte Änderungs Ereignisse; Energie Meldungen Benachrichtigen Anwendungen von Energie Verwaltungs Ereignissen. Anwendungen und Treiber können auch benutzerdefinierte Nachrichten definieren und verwenden, um Benachrichtigungen über andere Ereignis Typen zu ermöglichen. Diese Übersicht enthält Informationen zu den folgenden Themen:

-   [Gerätemeldungen](device-messages.md)
-   [Geräte Ereignis Typen](device-event-types.md)
-   [Geräte Benachrichtigungen](device-notifications.md)
-   [Geräte Ereignisse](device-events.md)
-   [Geräte Eingabe-und-Ausgabe Steuerung (IOCTL)](device-input-and-output-control-ioctl-.md)

Weitere Informationen zu Strom Meldungen finden Sie unter [Energie Verwaltung](/windows/desktop/Power/power-management-portal).
