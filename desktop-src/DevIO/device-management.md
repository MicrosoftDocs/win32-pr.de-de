# <a name="device-management"></a>Geräteverwaltung

Mit der Geräteverwaltung können Sie alle Anwendungen und System Features von Änderungen, die sich auf den Betrieb oder den Zugriff auf Ressourcen auswirken können, einheitlich benachrichtigen. Anwendungen und das System verwenden und verarbeiten Geräte Ereignisse, um neue Ressourcen zu nutzen, wenn Sie verfügbar werden, und um Datenverluste zu vermeiden, wenn vorhandene Ressourcen nicht mehr verfügbar sind.

-   [Informationen zur Geräteverwaltung](about-device-management.md)
-   [Verwenden der Geräteverwaltung](using-device-management.md)
-   [Referenz zur Geräteverwaltung](device-management-reference.md)
