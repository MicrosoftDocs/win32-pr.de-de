# <a name="windows-system-information"></a>Windows Systeminformationen

In den folgenden Übersichten werden die Verfügbaren Arten von Systeminformationen beschrieben.



| Übersicht                                                | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                         |
|---------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Handles und Objekte](handles-and-objects.md)          | Ein *Objekt* ist eine Datenstruktur, die eine Systemressource darstellt, z. B. eine Datei, ein Thread oder ein Grafikbild. Eine Anwendung kann nicht direkt auf Objektdaten oder die Systemressource zugreifen, die ein Objekt darstellt. Stattdessen muss eine Anwendung ein Objekthand handle *abrufen,* mit dem sie die Systemressource untersuchen oder ändern kann. |
| [Registrierung](registry.md)                                | Eine systemdefinierte Datenbank, in der Anwendungen und das System Konfigurationsinformationen speichern und abrufen.                                                                                                                                                                                                                        |
| [Systeminformationen](system-information.md)            | Ruft die Systemkonfiguration, Einstellungen, Version und Metriken ab oder legt diese fest.                                                                                                                                                                                                                                                             |
| [Time](time.md)                                        | Ruft die Systemzeit ab oder legt sie fest.                                                                                                                                                                                                                                                                                                  |
| [Zeitanbieter](time-provider.md)                      | Ruft genaue Zeitstempel von der Hardware oder dem Netzwerk ab und stellt Zeitstempel für andere Clients im Netzwerk zur Verfügung.                                                                                                                                                                                                              |
| [WaaS-Bewertungsplattform](update-assessor-service.md) | Die Windows-As-a-Service-Updatebewertungsplattform (WaaS) enthält Informationen zu den Updates Windows Geräts.                                                                                                                                                                                                                      |
