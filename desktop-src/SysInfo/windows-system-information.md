# <a name="windows-system-information"></a>Windows-System Informationen

In den folgenden Übersichten werden die Typen der verfügbaren Systeminformationen beschrieben.



| Übersicht                                                | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                         |
|---------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Handles und Objekte](handles-and-objects.md)          | Bei einem *Objekt* handelt es sich um eine Datenstruktur, die eine System Ressource darstellt, z. b. eine Datei, einen Thread oder ein Grafik Bild. Eine Anwendung kann nicht direkt auf Objektdaten oder die System Ressource zugreifen, die ein Objekt darstellt. Stattdessen benötigt eine Anwendung ein Objekt *handle*, das zum Überprüfen oder Ändern der System Ressource verwendet werden kann. |
| [Registrierung](registry.md)                                | Eine System definierte Datenbank, in der Anwendungen und das System Konfigurationsinformationen speichern und abrufen.                                                                                                                                                                                                                        |
| [Systeminformationen](system-information.md)            | Ruft Systemkonfiguration, Einstellungen, Version und Metriken ab oder legt Sie fest.                                                                                                                                                                                                                                                             |
| [Time](time.md)                                        | Ruft die Systemzeit ab oder legt Sie fest.                                                                                                                                                                                                                                                                                                  |
| [Zeit Anbieter](time-provider.md)                      | Ruft genaue Zeitstempel von der Hardware oder dem Netzwerk ab und stellt Zeitstempel für andere Clients im Netzwerk bereit.                                                                                                                                                                                                              |
| [WAAS-Bewertungsplattform](update-assessor-service.md) | Die Plattform "Windows as a Service (WAAS) Update Assessment" enthält Informationen zu den Windows-Updates eines Geräts.                                                                                                                                                                                                                      |
