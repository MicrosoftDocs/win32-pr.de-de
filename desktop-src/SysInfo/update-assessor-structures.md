# <a name="update-assessor-structures"></a>Aktualisieren von Assessor-Strukturen

Die folgenden Funktionen werden vom Update-Accessor-Anbieter verwendet.



| Funktion                                         | BESCHREIBUNG                                                           |
|--------------------------------------------------|-----------------------------------------------------------------------|
| [**Osupdateassessment**](/windows/win32/api/waasapitypes/ns-waasapitypes-osupdateassessment) | Definiert, wie aktuell das Betriebssystem auf einem Zielgerät ist.                |
| [**Updateassessment**](/windows/win32/api/waasapitypes/ns-waasapitypes-osupdateassessment)   | Enthält Informationen, die bewerten, wie ein installiertes Betriebssystem auf dem neuesten Stand ist. |
