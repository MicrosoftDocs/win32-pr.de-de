# <a name="handles-and-objects"></a>Handles und Objekte

Bei einem *Objekt* handelt es sich um eine Datenstruktur, die eine System Ressource darstellt, z. b. eine Datei, einen Thread oder ein Grafik Bild. Eine Anwendung kann nicht direkt auf Objektdaten oder die System Ressource zugreifen, die ein Objekt darstellt. Stattdessen benötigt eine Anwendung ein Objekt *handle*, das zum Überprüfen oder Ändern der System Ressource verwendet werden kann. Jedes Handle verfügt über einen Eintrag in einer intern verwalteten Tabelle. Diese Einträge enthalten die Adressen der Ressourcen und die Mittel zum Identifizieren des Ressourcentyps.
