# <a name="registry"></a>Registrierung

Die *Registrierung* ist eine System definierte Datenbank, in der Anwendungen und Systemkomponenten Konfigurationsdaten speichern und abrufen. Die Daten, die in der Registrierung gespeichert sind, variieren je nach Version von Microsoft Windows. Anwendungen verwenden die Registrierungs-API, um Registrierungsdaten abzurufen, zu ändern oder zu löschen.

Sie sollten keine Registrierungsdaten bearbeiten, die nicht zu Ihrer Anwendung gehören, es sei denn, es ist unbedingt erforderlich. Wenn in der Registrierung ein Fehler auftritt, funktioniert das System möglicherweise nicht ordnungsgemäß. Wenn dies der Fall ist, können Sie die Registrierung in dem Zustand wiederherstellen, in dem Sie sich beim letzten Starten des Computers befand. Weitere Informationen finden Sie in der Hilfe zu Ihrem Betriebssystem.

Weitere Informationen zur Registrierungsdatenbank und zum programmgesteuerten Zugriff auf die Registrierungsdaten finden Sie in den folgenden Themen:

-   [Informationen zur Registrierung](about-the-registry.md)
-   [Verwenden der Registrierung](using-the-registry.md)
-   [Registrierungs Verweis](registry-reference.md)
