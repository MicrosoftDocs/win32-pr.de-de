# <a name="object-categories"></a>Objektkategorien

Das System stellt drei Kategorien von Objekten bereit: Benutzer, Grafikgeräte Schnittstelle (GDI) und Kernel. Das System verwendet Benutzer Objekte zur Unterstützung der Fensterverwaltung, GDI-Objekte zur Unterstützung von Grafiken und Kernel Objekten zur Unterstützung der Speicherverwaltung, Prozess Ausführung und prozessübergreifenden Kommunikation (prozessübergreifende Communications, IPC). Weitere Informationen zum Erstellen und Verwenden eines bestimmten Objekts finden Sie in der zugehörigen Übersicht.
