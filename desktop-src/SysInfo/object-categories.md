# <a name="object-categories"></a>Objektkategorien

Das System stellt drei Kategorien von Objekten bereit: Benutzer, Grafikgeräteschnittstelle (GDI) und Kernel. Das System verwendet Benutzerobjekte zur Unterstützung der Fensterverwaltung, GDI-Objekte zur Unterstützung von Grafiken und Kernelobjekte zur Unterstützung der Speicherverwaltung, Prozessausführung und prozessübergreifenden Kommunikation (Interprocess Communications, IPC). Informationen zum Erstellen und Verwenden eines bestimmten Objekts finden Sie in der zugehörigen Übersicht.
