# <a name="update-assessor-enumerations"></a>Aktualisieren von Assessor-Enumerationen

Die folgenden Enumerationen werden vom Update-Accessor-Anbieter verwendet.



| Funktion                                                 | BESCHREIBUNG                                                                    |
|----------------------------------------------------------|--------------------------------------------------------------------------------|
| [**Updategutamentstatus**](updateassessmentstatus.md) | Hier wird beschrieben, wie das Betriebssystem auf einem Gerät auf dem neuesten Stand ist.                                |
| [**Updateimpactlevel**](updateimpactlevel.md)           | Gibt eine hohe, mittlere oder geringe Auswirkung auf ein Gerät an, auf dem ein veralteter Betriebssystem ausgeführt wird. |
