# <a name="update-assessor-enumerations"></a>Aktualisieren von Assessor-Enumerationen

Die folgenden Enumerationen werden vom Updateaccessoranbieter verwendet.



| Funktion                                                 | BESCHREIBUNG                                                                    |
|----------------------------------------------------------|--------------------------------------------------------------------------------|
| [**UpdateAssessmentStatus**](updateassessmentstatus.md) | Beschreibt, wie aktuell das Betriebssystem auf einem Gerät ist.                                |
| [**UpdateImpactLevel**](updateimpactlevel.md)           | Gibt eine hohe, mittlere oder geringe Auswirkung eines Geräts an, auf dem ein veraltetes Betriebssystem ausgeführt wird. |
