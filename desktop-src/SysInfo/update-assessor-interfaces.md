Die folgenden Schnittstellen werden vom Updatezugriffsanbieter verwendet.



| Funktion                               | BESCHREIBUNG                                                                                                                 |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [**IWaaSAssessor**](/windows/desktop/api/waasapi/nn-waasapi-iwaasassessor) | Ruft die Bewertung des Betriebssystemupdates ab, indem der neueste Build von Microsoft mit dem Build verglichen wird, der auf dem aktuellen Gerät ausgeführt wird. |
