# <a name="system-information-functions"></a>Systeminformationen Funktionen

Die folgenden Funktionen werden verwendet, um Systeminformationen abzurufen oder festzulegen.



| Funktion                                                                     | Beschreibung                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CeipIsOptedIn**](/windows/desktop/api/Windowsceip/nf-windowsceip-ceipisoptedin)                                       | Überprüft, ob sich der Benutzer im Rahmen des Programm zur Verbesserung der Benutzerfreundlichkeit (CEIP) für die DATENsammlung VOM DATENTYP "SQM" entschieden hat.                                                                                                       |
| [**DnsHostnameToComputerName**](/windows/desktop/api/Winbase/nf-winbase-dnshostnametocomputernamea)               | Konvertiert einen DNS-Namen in einen NetBIOS-Namen.                                                                                                                                                                                            |
| [**EnumSystemFirmwareTables**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-enumsystemfirmwaretables)                 | Listet alle Systemfirmwaretabellen des angegebenen Typs auf.                                                                                                                                                                      |
| [**ExpandEnvironmentStrings**](/windows/win32/api/processenv/nf-processenv-expandenvironmentstringsa)                 | Ersetzt Umgebungsvariablenzeichenfolgen durch ihre definierten Werte.                                                                                                                                                                  |
| [**GetComputerName**](/windows/desktop/api/Winbase/nf-winbase-getcomputernamea)                                   | Ruft den NetBIOS-Namen des lokalen Computers ab.                                                                                                                                                                                 |
| [**GetComputerNameEx**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getcomputernameexa)                               | Ruft den NetBIOS- oder DNS-Namen des lokalen Computers ab.                                                                                                                                                                          |
| [**GetComputerObjectName**](/windows/desktop/api/Secext/nf-secext-getcomputerobjectnamea)                       | Ruft den Namen des lokalen Computers in einem angegebenen Format ab.                                                                                                                                                                        |
| [**GetCurrentHwProfile**](/windows/desktop/api/Winbase/nf-winbase-getcurrenthwprofilea)                           | Ruft das aktuelle Hardwareprofil für den lokalen Computer ab.                                                                                                                                                                    |
| [**GetFirmwareEnvironmentVariable**](/windows/desktop/api/Winbase/nf-winbase-getfirmwareenvironmentvariablea)     | Ruft den Wert der angegebenen Firmwareumgebungsvariablen aus NVRAM ab.                                                                                                                                                    |
| [**GetFirmwareEnvironmentVariableEx**](/windows/desktop/api/Winbase/nf-winbase-getfirmwareenvironmentvariableexa) | Ruft den Wert der angegebenen Firmwareumgebungsvariablen und deren Attribute ab.                                                                                                                                            |
| [**GetFirmwareType**](/windows/desktop/api/Winbase/nf-winbase-getfirmwaretype)                                   | Ruft den Firmwaretyp des lokalen Computers ab.                                                                                                                                                                                |
| [**GetIntegratedDisplaySize**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getintegrateddisplaysize)                 | Ruft die beste Schätzung der diagonalen Größe des integrierten Bildschirms in Zoll ab.                                                                                                                                               |
| [**GetNativeSystemInfo**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getnativesysteminfo)                           | Ruft Informationen zum aktuellen System für eine Anwendung ab, die unter WOW64 ausgeführt wird.                                                                                                                                            |
| [**GetProductInfo**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getproductinfo)                                     | Ruft den Produkttyp für das Betriebssystem auf dem lokalen Computer ab und ordnet den Typ den Produkttypen zu, die vom angegebenen Betriebssystem unterstützt werden.                                                                    |
| [**GetSystemDirectory**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getsystemdirectorya)                             | Ruft den Pfad des Systemverzeichnisses ab.                                                                                                                                                                                       |
| [**GetSystemFirmwareTable**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getsystemfirmwaretable)                     | Ruft die angegebene Firmwaretabelle vom Firmwaretabellenanbieter ab.                                                                                                                                                          |
| [**GetSystemInfo**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getsysteminfo)                                       | Ruft Informationen zum aktuellen System ab.                                                                                                                                                                                   |
| [**GetSystemRegistryQuota**](/windows/desktop/api/Winbase/nf-winbase-getsystemregistryquota)                     | Ruft die aktuelle Größe der Registrierung und die maximale Größe ab, die die Registrierung auf dem System erreichen darf.                                                                                                             |
| [**GetSystemWindowsDirectory**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getsystemwindowsdirectorya)               | Ruft den Pfad des freigegebenen Windows-Verzeichnisses auf einem Mehrbenutzersystem ab.                                                                                                                                                        |
| [**GetSystemWow64Directory**](/windows/win32/api/wow64apiset/nf-wow64apiset-getsystemwow64directorya)                   | Ruft den Pfad des von WOW64 verwendeten Systemverzeichnisses ab.                                                                                                                                                                         |
| [**GetSystemWow64Directory2**](/windows/desktop/api/wow64apiset/nf-wow64apiset-getsystemwow64directory2a)                 | Ruft den Pfad des von WOW64 verwendeten Systemverzeichnisses unter Verwendung des angegebenen Imagedateicomputertyps ab.                                                                                                                            |
| [**GetUserName**](/windows/desktop/api/Winbase/nf-winbase-getusernamea)                                           | Ruft den Benutzernamen des aktuellen Threads ab.                                                                                                                                                                                    |
| [**GetUserNameEx**](/windows/desktop/api/Secext/nf-secext-getusernameexa)                                       | Ruft den Namen des Benutzers oder eines anderen Sicherheitsprinzipals ab, der dem aufrufenden Thread zugeordnet ist. Sie können das Format des zurückgegebenen Namens angeben.                                                                                   |
| [**Getversion**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getversion)                                             | Ruft die Versionsnummer des aktuellen Betriebssystems ab.                                                                                                                                                                     |
| [**Getversionex**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getversionexa)                                         | Ruft Informationen zum aktuellen Betriebssystem ab.                                                                                                                                                                         |
| [**GetWindowsDirectory**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getwindowsdirectorya)                           | Ruft den Pfad des Windows Verzeichnisses ab.                                                                                                                                                                                      |
| [**IsNativeVhdBoot**](/windows/desktop/api/Winbase/nf-winbase-isnativevhdboot)                                   | Gibt an, ob das Betriebssystem von einem VHD-Container gestartet wurde.                                                                                                                                                                              |
| [**IsWow64GuestMachineSupported**](/windows/desktop/api/wow64apiset/nf-wow64apiset-iswow64guestmachinesupported)         |                                                                                                                                                                                                                                   |
| [**IsProcessorFeaturePresent**](/windows/win32/api/processthreadsapi/nf-processthreadsapi-isprocessorfeaturepresent)               | Bestimmt, ob ein Prozessorfeature vom aktuellen Computer unterstützt wird.                                                                                                                                                      |
| [**IsWow64Message**](/windows/desktop/api/winuser/nf-winuser-iswow64message)                                    | Bestimmt, ob die letzte aus der Warteschlange des aktuellen Prozesses gelesene Nachricht von einem WOW64-Prozess stammt.                                                                                                                   |
| [**IsWow64Process**](/windows/desktop/api/wow64apiset/nf-wow64apiset-iswow64process)                                    | Bestimmt, ob ein Prozess unter WOW64 ausgeführt wird.                                                                                                                                                                              |
| [**QueryPerformanceCounter**](/windows/win32/api/profileapi/nf-profileapi-queryperformancecounter)                   | Ruft den aktuellen Wert des Leistungsindikators mit hoher Auflösung ab.                                                                                                                                                           |
| [**QueryPerformanceFrequency**](/windows/win32/api/profileapi/nf-profileapi-queryperformancefrequency)               | Ruft die Häufigkeit des Leistungsindikators mit hoher Auflösung ab.                                                                                                                                                               |
| [**RtlGetSuiteMask**](rtlgetsuitemask.md)                                   | Ruft eine Bitmaske ab, die die im System verfügbaren Produktsammlungen identifiziert. Wenn diese Funktion in einer Anwendung aufgerufen wird, die im Kontext eines Serversilos ausgeführt wird, wird stattdessen die Suitemaske für das Serversilos abgerufen. |
| [**SetComputerName**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-setcomputernamea)                                   | Legt einen neuen NetBIOS-Namen für den lokalen Computer fest.                                                                                                                                                                                   |
| [**SetComputerNameEx**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-setcomputernameexa)                               | Legt einen neuen NetBIOS- oder DNS-Namen für den lokalen Computer fest.                                                                                                                                                                            |
| [**SetFirmwareEnvironmentVariable**](/windows/desktop/api/Winbase/nf-winbase-setfirmwareenvironmentvariablea)     | Legt den Wert der angegebenen Firmwareumgebungsvariablen fest.                                                                                                                                                                    |
| [**SetFirmwareEnvironmentVariableEx**](/windows/desktop/api/Winbase/nf-winbase-setfirmwareenvironmentvariableexa) | Legt den Wert der angegebenen Firmwareumgebungsvariablen und deren Attribute fest.                                                                                                                                                 |
| [**TranslateName**](/windows/desktop/api/Secext/nf-secext-translatenamea)                                       | Konvertiert einen Verzeichnisdienstobjektnamen von einem Format in ein anderes.                                                                                                                                                              |
| [**VerifyVersionInfo**](/windows/desktop/api/Winbase/nf-winbase-verifyversioninfoa)                               | Vergleicht eine Reihe von Versionsanforderungen mit den Werten für das aktuelle Betriebssystem.                                                                                                                                            |
| [**VerSetConditionMask**](/windows/desktop/api/Winnt/nf-winnt-versetconditionmask)                           | Erstellt die Bedingungsmaske für die [**VerifyVersionInfo-Funktion.**](/windows/desktop/api/Winbase/nf-winbase-verifyversioninfoa)                                                                                                                                        |
| [**Wow64DisableWow64FsRedirection**](/windows/desktop/api/wow64apiset/nf-wow64apiset-wow64disablewow64fsredirection)    | Deaktiviert die Dateisystemumleitung für den aufrufenden Thread.                                                                                                                                                                          |
| [**Wow64EnableWow64FsRedirection**](/windows/desktop/api/winbase/nf-winbase-wow64enablewow64fsredirection)      | Aktiviert oder deaktiviert die Dateisystemumleitung für den aufrufenden Thread.                                                                                                                                                               |
| [**Wow64RevertWow64FsRedirection**](/windows/desktop/api/wow64apiset/nf-wow64apiset-wow64revertwow64fsredirection)      | Stellt die Dateisystemumleitung für den aufrufenden Thread wieder her.                                                                                                                                                                          |
