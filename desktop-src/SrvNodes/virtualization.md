# <a name="virtualization"></a>Virtualisierung

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[Microsoft Virtual Server](/previous-versions/windows/desktop/msvs/microsoft-virtual-server-portal)
</dt> <dd>

Mit Microsoft Virtual Server 2005 können Sie auf Ihrem Microsoft Windows-Desktop separate virtuelle Computer erstellen, auf denen Sie praktisch jedes beliebige Intel-Architektur-Betriebssystem installieren können. Jeder virtuelle Computer emuliert ein komplett Hardwaresystem – vom Prozessor bis zur Netzwerkkarte – in einer eigenständigen, isolierten Softwareumgebung, die den gleichzeitigen Betrieb von anderweitig inkompatiblen Systemen ermöglicht.

</dd> <dt>

[Windows Virtual PC](/windows/desktop/VPC/virtual-pc-portal)
</dt> <dd>

Windows Virtual PC ist die neueste Microsoft-Virtualisierungstechnologie. mit diesem Dienst können Sie viele Produktivitätsanwendungen in einer virtuellen Windows-Umgebung mit einem einzigen Mausklick direkt von Windows 7 ausführen. Sie können separate virtuelle Computer zusätzlich zu Ihrem Windows 7-Desktop erstellen. Jeder virtuelle Computer emuliert ein komplett Hardwaresystem – vom Prozessor bis zur Netzwerkkarte – in einer eigenständigen, isolierten Softwareumgebung, die den gleichzeitigen Betrieb von anderweitig inkompatiblen Systemen ermöglicht.

</dd> </dl>
