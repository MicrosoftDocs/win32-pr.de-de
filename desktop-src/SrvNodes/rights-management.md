# <a name="microsoft-rights-management-sdks"></a>Microsoft Rights Management SDKs

Drei Generationen von Rights Management SDK sind jetzt verfügbar: [Rights Management SDK 4.2](/previous-versions/windows/desktop/msipcthin2/active-directory-rights-management-services-multi-platform-thin-client-sdk-portal), [Microsoft Rights Management SDK 2.1](/previous-versions/windows/desktop/msipc/microsoft-information-protection-and-control-client-portal) und [Active Directory Rights Management Services SDK](/previous-versions/windows/desktop/adrms_sdk/active-directory-rights-management-services-sdk-portal)sowie eine Skript-API, Active Directory Rights Management Services [Scripting API,](/previous-versions/windows/desktop/adrms_script/adrms-script-portal)die die benutzerdefinierte Verwaltung eines RMS-Servers ermöglicht.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[Codebeispiele und Tools](rms-code-samples-and-tools.md)
</dt> <dd>

Eine Sammlung von von Microsoft bereitgestellten RMS-Codebeispielen und Entwicklersupporttools für alle unterstützten Betriebssysteme. Android, iOS/OS X, Windows Phone und Windows Desktop.

</dd> <dd>

RMS SDK 4.2 ist ein vereinfachter Toolsatz der nächsten Generation, der eine einfache Entwicklungserfahrung zum Aktivieren Ihrer Android-, iOS- und Mac OS X-Geräte-Apps mit Informationsschutz über Microsoft Rights Management Services.

</dd> <dd>

RMS SDK 2.1 ist ein leistungsstarkes SDK-Angebot für Windows Desktopanwendungsentwickler und serverbasierte Lösungsanbieter, um ihre Produkte mit Rights Management zu ermöglichen.

</dd> <dt>

[Active Directory Rights Management Services-SDK](/previous-versions/windows/desktop/adrms_sdk/active-directory-rights-management-services-sdk-portal)
</dt> <dd>

> [!Note]  
> AD RMS SDK, das funktionen nutzt, die vom Client in Msdrm.dll verfügbar gemacht werden, ist für die Verwendung in Windows Server 2012, Windows 8, Windows Server 2008 R2, Windows 7, Windows Server 2008 und Windows Vista verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden. Verwenden Sie stattdessen [Microsoft Rights Management Services SDK 2.1,](/previous-versions/windows/desktop/msipc/microsoft-information-protection-and-control-client-portal)das funktionen nutzt, die vom Client in der Msipc.dll.

 

</dd> <dt>

[Active Directory Rights Management Services Scripting API (Active Directory Rights Management Services-Skripterstellungs-API)](/previous-versions/windows/desktop/adrms_script/adrms-script-portal)
</dt> <dd>

Die Windows Active Directory Rights Management Services (AD RMS)-Skripterstellungs-API kann verwendet werden, um Skripts zum Verwalten einer AD RMS zu erstellen.

</dd> </dl>
