# <a name="directx-graphics-and-gaming"></a>DirectX-Grafiken und-Spiele

Dieser Inhalt konzentriert sich auf die Verwendung von DirectX in einer Win32-Anwendung. Informationen zur Verwendung von DirectX in einer UWP-Anwendung finden Sie im [Windows 10 Game Development Guide (UWP)](/windows/uwp/gaming/e2e).

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [Einstieg in DirectX-Grafiken](./getting-started-with-directx-graphics.md) | Microsoft DirectX Graphics bietet eine Reihe von APIs, die Sie zum Erstellen von spielen und anderen hochleistungsfähigen Multimedia-Apps verwenden können. Die DirectX-Grafik bietet Unterstützung für hochleistungsfähige 2D-und 3D-Grafiken. |
| [Programmieren von DirectX mit com](prog-dx-with-com.md) | Beim Microsoft-Component Object Model (com) handelt es sich um ein objektorientiertes Programmiermodell, das von verschiedenen Technologien verwendet wird, einschließlich des Massen Vorgangs der DirectX-API-Oberfläche. |
| [Direct2D](./direct2d/direct2d-portal.md) | Direct2D ist eine hardwarebeschleunigte 2D-Grafik-API mit unmittelbarem Modus, die das Rendern mit hoher Leistung und in hoher Qualität für 2D-Geometrie, Bitmaps und Text bereitstellt. |
| [Direct3D](./direct3d.md) | Direct3D ermöglicht es Ihnen, 3D-Grafiken für Spiele und wissenschaftliche apps zu erstellen. |
| [DXCore](./dxcore/dxcore.md) | DXCore ist eine adapterenumerationsapi für Grafiken und Compute-Geräte, sodass sich einige ihrer Funktionen mit denen von DXGI überlappen. |
| [DirectWrite](./directwrite/direct-write-portal.md) | DirectWrite unterstützt das hochwertige Text Rendering, auflösungsunabhängige Gliederungs Schriftarten sowie vollständigen Unicode-Text und-Layouts. |
| [DirectXMath](./dxmath/directxmath-portal.md) | Directxmath bietet eine optimale und tragbare Oberfläche für arithmetische und lineare Algebra-Vorgänge für Gleit Komma Vektoren mit einfacher Genauigkeit (2D, 3D und 4D) oder Matrizen (3 × 3 und 4 × 4). |
| [windowsnumerics. h-APIs](./numerics_h/windowsnumerics-h-apis-portal.md) | Die `windowsnumerics.h` Header Datei definiert C++ Vector-und Matrix Typen im Windows. Foundation. Numerics-Namespace. |
| [Klassische DirectX-Grafiken](./classic-directx-graphics.md) | Microsoft DirectX-Grafik Technologien, die derzeit minimal verwendet werden. Es wird nicht empfohlen, diese klassischen DirectX-Grafik Technologien für neue apps zu verwenden. |
