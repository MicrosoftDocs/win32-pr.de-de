# <a name="system-restore-wmi-classes"></a>WMI-Klassen für System Wiederherstellung

Die folgenden WMI-Klassen werden bei der System Wiederherstellung verwendet.



| WMI-Klasse                                          | BESCHREIBUNG                                                                                                                                |
|----------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [**SystemRestore**](systemrestore.md)             | Bietet Methoden zum Deaktivieren und Aktivieren der Überwachung, zum Auflisten verfügbarer Wiederherstellungspunkte und zum Initiieren einer Wiederherstellung auf dem lokalen System.    |
| [**Systemrestoreconfig**](systemrestoreconfig.md) | Stellt Eigenschaften zum Steuern der Häufigkeit der geplanten Wiederherstellungspunkt Erstellung und des auf jedem Laufwerk genutzten Speicherplatzes bereit. |
