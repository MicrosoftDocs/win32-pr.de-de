# <a name="system-restore-api"></a>API zur System Wiederherstellung

Die Systemwiederherstellungs-API ermöglicht es einer Anwendung, Wiederherstellungspunkte zu verwalten. Es gibt auch eine Skript fähige Schnittstelle, die über [Windows-Verwaltungsinstrumentation](/windows/desktop/WmiSdk/wmi-start-page) (WMI) verfügbar ist.

Weitere Informationen zur System Wiederherstellungs-API finden Sie in den folgenden Themen:

-   [Aufrufen von srsettrestorepoint](calling-srsetrestorepoint.md)
-   [Wiederherstellungspunkt-Beschreibungs Text](restore-point-description-text.md)
-   [Aufgerufenen Aufrufe von srsettrestorepoint](nested-calls-to-srsetrestorepoint.md)

Weitere Informationen zu den WMI-Klassen für die Systemwiederherstellung finden Sie unter [Konfigurieren der Systemwiederherstellung](configuring-system-restore.md).
