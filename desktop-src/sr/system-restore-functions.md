# <a name="system-restore-functions"></a>System Wiederherstellungs Funktionen

Die folgenden Funktionen werden bei der System Wiederherstellung verwendet.



| Funktion                                             | BESCHREIBUNG                                                                                                   |
|------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**Srremoverestorepoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srremoverestorepoint) | Löscht den angegebenen Wiederherstellungspunkt.                                                                          |
| [**Srsertrestorepoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srsetrestorepointa)       | Gibt den Anfang und das Beenden eines Satzes von Änderungen an, sodass die System Wiederherstellung einen Wiederherstellungspunkt erstellen kann. |
