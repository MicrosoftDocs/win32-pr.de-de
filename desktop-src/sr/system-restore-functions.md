# <a name="system-restore-functions"></a>Systemwiederherstellungsfunktionen

Die folgenden Funktionen werden mit der Systemwiederherstellung verwendet.



| Funktion                                             | BESCHREIBUNG                                                                                                   |
|------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**SRRemoveRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srremoverestorepoint) | Löscht den angegebenen Wiederherstellungspunkt.                                                                          |
| [**SRSetRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srsetrestorepointa)       | Gibt den Anfang und das Ende eines Satzes von Änderungen an, sodass die Systemwiederherstellung einen Wiederherstellungspunkt erstellen kann. |
