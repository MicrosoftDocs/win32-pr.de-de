# <a name="avenccommonmeanbitrate-property"></a>Avenccommonmeanbitrate (Eigenschaft)

Gibt die durchschnittliche codierte Bitrate in Bits pro Sekunde an. Diese Eigenschaft gilt nur für die Konstante Bitrate (CBR) und die Variablen Bitrate (VBR)-Codierungs Modi.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avenccommonmeanbitrate**

## <a name="property-value"></a>Eigenschaftswert

Encoder können diese Eigenschaft als enumerationsset oder als linearen Bereich implementieren.

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft wird auch mit [H. 264 UVC 1,5-Kamera Codierern](/windows/desktop/medfound/camera-encoder-h264-uvc-1-5)verwendet.
