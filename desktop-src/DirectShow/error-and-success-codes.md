# <a name="directshow-error-and-success-codes"></a>Fehler-und Erfolgs Codes von DirectShow

In diesem Thema werden die **HRESULT** -Rückgabecodes aufgelistet, die für DirectShow definiert sind. Beachten Sie, dass DirectShow-APIs andere **HRESULT** -Werte zurückgeben können, die hier nicht angezeigt werden Weitere **HRESULT** -Werte finden Sie unter [com-Fehler Codes](/windows/desktop/com/com-error-codes).

Die folgende Tabelle enthält DirectShow-spezifische Fehler-und Erfolgs Codes in numerischer Reihenfolge. Rückgabecodes, die mit VFW S beginnen, \_ \_ deuten auf Erfolg hin. Ein Vorgang ist möglicherweise teilweise erfolgreich und weist trotzdem einen erfolgreichen Rückgabewert auf. Beispielsweise geben **VFW \_ S \_ einige \_ Daten \_ ignoriert** an, dass eine Datei gerendert wurde, indem einige ihrer Eigenschafts Einstellungen ignoriert wurden. Rückgabecodes, die mit VFW E beginnen, \_ \_ weisen auf einen Fehler hin.

Diese Werte werden in der Header Datei vfwmsgs. h definiert.
