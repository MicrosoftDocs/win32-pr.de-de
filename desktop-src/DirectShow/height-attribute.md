# <a name="height-attribute"></a>height-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Releases von Windows entfernt werden.\]

 

Das `height` -Attribut gibt die Höhe des Ausgabevideos in Pixel an.

## <a name="possible-values"></a>Mögliche Werte

Gleitkommawert. Der Wert muss die führende 0 (null) vor der Dezimalstelle enthalten. Beispiel: 0.3, nicht .3. Verwenden Sie nicht mehr als sieben Dezimalstellen.

## <a name="applies-to"></a>Gilt für

[**group**](group-element.md)

## <a name="remarks"></a>Hinweise

Legen Sie dieses  Attribut nur fest, wenn das Typattribut `video` ist.
