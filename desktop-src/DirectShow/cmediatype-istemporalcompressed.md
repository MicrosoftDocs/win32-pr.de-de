# <a name="cmediatypeistemporalcompressed-method"></a>Cmediatype. istemporalcompressed-Methode

Die- `IsTemporalCompressed` Methode bestimmt, ob der Stream Temporale Komprimierung verwendet.

## <a name="syntax"></a>Syntax


```C++
BOOL IsTemporalCompressed() const;
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt den Wert des **btemporalcompression** -Members zurück.
