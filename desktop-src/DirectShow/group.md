# <a name="group-directshow"></a>Gruppe (DirectShow)

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das Group-Objekt stellt eine Komposition der obersten Ebene dar. Jede Gruppe definiert ein Video oder einen Audiostream im endgültigen Projekt. Um dieses Objekt zu erstellen, rufen Sie die [**iamtimeline:: erkreateemptynode**](iamtimeline-createemptynode.md) -Methode auf.

Das Group-Objekt macht die folgenden Schnittstellen verfügbar:

-   [**Iamtimelinecomp**](iamtimelinecomp.md)
-   [**Iamtimelinegroup**](iamtimelinegroup.md)
-   [**Iamtimelineobj**](iamtimelineobj.md)
