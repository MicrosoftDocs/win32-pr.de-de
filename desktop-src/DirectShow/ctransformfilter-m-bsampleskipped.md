# <a name="ctransformfilterm_bsampleskipped-member"></a>Ctransformfilter:: m \_ bsampleskipped-Member

Flag, das angibt, ob das letzte Beispiel gelöscht wurde. Wenn die [**Receive**](ctransformfilter-receive.md) -Methode ein Beispiel löscht, wird der Wert auf **true** festgelegt.

## <a name="syntax"></a>Syntax


```C++
BOOL m_bSampleSkipped;
```
