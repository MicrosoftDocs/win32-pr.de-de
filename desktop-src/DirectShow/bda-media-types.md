# <a name="bda-media-types"></a>BDA-Medientypen

In den folgenden Tabellen werden die Medientypen beschrieben, die für digitale BDA-Fernsehfilter verwendet werden.

## <a name="antenna"></a>Antenne

Dieser Medientyp wird für die Pinverbindung zwischen dem Netzwerkanbieterfilter und dem Tunerfilter verwendet.



| Bezeichnung | Wert |
|-------------|----------------------------------|
| Haupttyp  | KSDATAFORMAT-TYP \_ \_ \_ BDA-ANTENNE |
| Subtype     | MEDIASUBTYPE \_ None               |
| Formattyp | FORMAT \_ None                     |



 

## <a name="intermediate-frequency-if-signal"></a>Zwischenfrequenzsignal (IF)

Dieser Medientyp stellt das IF-Signal dar, das vom Tuner an den Demodulator gesendet wird.



| Bezeichnung | Wert |
|-------------|-------------------------------------|
| Haupttyp  | KSDATAFORMAT-TYP \_ \_ BDA \_ IF \_ SIGNAL |
| Subtype     | MEDIASUBTYPE \_ None                  |
| Formattyp | FORMAT \_ None                        |
