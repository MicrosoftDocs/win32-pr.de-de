# <a name="cimagedisplaygetdisplaydepth-method"></a>Cimagedisplay. getdisplaytiefe-Methode

Die- `GetDisplayDepth` Methode ruft die Bittiefe des aktuellen Anzeigemodus ab.

## <a name="syntax"></a>Syntax


```C++
WORD GetDisplayDepth();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt die Anzahl der Bits pro Pixel zurück.
