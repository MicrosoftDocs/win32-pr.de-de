# <a name="xtl-elements"></a>XTL-Elemente

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Im XTL-Format werden die folgenden Elemente definiert.



| Element                                  | BESCHREIBUNG                                                                    |
|------------------------------------------|--------------------------------------------------------------------------------|
| [**an**](at-element.md)                 | Gibt eine zeitbasierte Einstellung für ein **param** -Element an.                        |
| [**Techno**](clip-element.md)             | Gibt eine Quelldatei an.                                                       |
| [**Haufen**](composite-element.md)   | Definiert eine Komposition.                                                         |
| [**entsprechende**](effect-element.md)         | Definiert einen Effekt.                                                             |
| [**Kreis**](group-element.md)           | Definiert eine Gruppe (Audiodatei oder Video).                                              |
| [**AK**](linear-element.md)         | Gibt eine zeitbasierte Einstellung für ein **param** -Element an.                        |
| [**param**](param-element.md)           | Gibt den Wert einer Eigenschaft eines Übergangs, eines Effekts oder eines anderen unter Objekts an. |
| [**Zeitplan**](timeline-element.md)     | Definiert die Zeitachse.                                                          |
| [**track**](track-element.md)           | Definiert eine Spur.                                                               |
| [**Übergang**](transition-element.md) | Definiert einen Übergang.                                                          |
