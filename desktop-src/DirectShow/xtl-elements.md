# <a name="xtl-elements"></a>XTL-Elemente

> [!Note]  
> \[Veraltet. Diese API wird möglicherweise aus zukünftigen Releases von Windows.\]

 

Das XTL-Format definiert die folgenden Elemente.



| Element                                  | BESCHREIBUNG                                                                    |
|------------------------------------------|--------------------------------------------------------------------------------|
| [**Auf**](at-element.md)                 | Gibt eine zeitbasierte Einstellung für ein **param-Element** an.                        |
| [**clip**](clip-element.md)             | Gibt eine Quelldatei an.                                                       |
| [**Zusammengesetzten**](composite-element.md)   | Definiert eine Komposition.                                                         |
| [**Wirkung**](effect-element.md)         | Definiert einen Effekt.                                                             |
| [**Gruppe**](group-element.md)           | Definiert eine Gruppe (Audio oder Video).                                              |
| [**Lineare**](linear-element.md)         | Gibt eine zeitbasierte Einstellung für ein **param-Element** an.                        |
| [**param**](param-element.md)           | Gibt den Wert einer Eigenschaft für einen Übergang, einen Effekt oder ein anderes Unterobjekt an. |
| [**Zeitachse**](timeline-element.md)     | Definiert die Zeitachse.                                                          |
| [**track**](track-element.md)           | Definiert eine Spur.                                                               |
| [**Übergang**](transition-element.md) | Definiert einen Übergang.                                                          |
