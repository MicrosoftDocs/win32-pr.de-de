> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Wird nicht unterstützt.

## <a name="syntax"></a>Syntax


```C++
HRESULT IsRecompressFormatDirty(
   BOOL *pVal
);
```



## <a name="parameters"></a>Parameter

<dl> <dt>

*pVal* 
</dt> <dd>

Reserviert.

</dd> </dl>
