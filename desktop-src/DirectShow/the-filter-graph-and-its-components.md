# <a name="the-filter-graph-and-its-components"></a>Die Filter-Graph und deren Komponenten

In diesem Artikel werden die Hauptkomponenten von DirectShow beschrieben. Es dient als Einführung für Anwendungsentwickler und entwickler, die benutzerdefinierte DirectShow-Filter schreiben. Anwendungsentwickler können in der Regel viele der Low-Level-Details von DirectShow ignorieren. Es ist jedoch dennoch eine gute Idee, diesen Abschnitt zu lesen, um ein allgemeines Verständnis der DirectShow-Architektur zu haben.
