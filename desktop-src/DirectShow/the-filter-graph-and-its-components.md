# <a name="the-filter-graph-and-its-components"></a>Das Filter Diagramm und dessen Komponenten

In diesem Artikel werden die Hauptkomponenten von DirectShow beschrieben. Es ist eine Einführung für Anwendungsentwickler und Entwickler, die benutzerdefinierte DirectShow-Filter schreiben. Anwendungsentwickler können normalerweise viele der Details von DirectShow auf niedriger Ebene ignorieren. Es ist jedoch immer noch ratsam, diesen Abschnitt zu lesen, um ein allgemeines Verständnis der DirectShow-Architektur zu erhalten.
