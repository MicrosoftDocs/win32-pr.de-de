# <a name="media-locator"></a>Medienlocator

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das medienlocatorobjekt gibt die Dateien an, die Quell Objekten zugeordnet sind. Verwenden Sie dieses Objekt zum Überprüfen der Quellen in der Zeitachse. Um dieses Objekt zu erstellen, rufen Sie **cokreateinstance** auf. Der Klassen Bezeichner ist CLSID \_ medialocator.

Das Media Locator-Objekt stellt die folgende Schnittstelle bereit:

-   [**Imedialocator**](imedialocator.md)
