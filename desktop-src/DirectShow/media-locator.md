# <a name="media-locator"></a>Medienlocator

> [!Note]  
> \[Veraltet. Diese API wird möglicherweise aus zukünftigen Releases von Windows.\]

 

Das Medienlocatorobjekt sucht die Dateien, die Quellobjekten zugeordnet sind. Verwenden Sie dieses Objekt, um Quellen auf der Zeitachse zu überprüfen. Um dieses Objekt zu erstellen, rufen Sie **CoCreateInstance auf.** Der Klassenbezeichner ist CLSID \_ MediaLocator.

Das Medienlocatorobjekt macht die folgende Schnittstelle verfügbar:

-   [**IMediaLocator**](imedialocator.md)
