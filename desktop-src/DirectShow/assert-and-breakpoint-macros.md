# <a name="assert-and-breakpoint-macros"></a>Assert-und breakpointmakros

Die [DirectShow-Basisklassen](directshow-base-classes.md) stellen mehrere Makros bereit, die Assertionen ausführen oder Breakpoints verursachen.



| Makro                                        | Beschreibung                                                                                                                        |
|----------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [**Machung**](assert.md)                     | Wertet einen Ausdruck aus und zeigt eine Diagnose Meldung an, wenn der Ausdruck **false** ist.                                         |
| [**Dbgassertaligned**](dbgassertaligned.md) | Testet, ob ein Zeiger an einer angegebenen Grenze ausgerichtet ist.                                                                        |
| [**Dbgbreak**](dbgbreak.md)                 | Zeigt ein Meldungs Feld mit der angegebenen Zeichenfolge, dem Namen der Quelldatei und der Zeilennummer an.                                |
| [**\_Assert ausführen**](execute-assert.md)    | Wertet einen Ausdruck in Debug-und Retail-Builds aus. In Debugbuilds zeigt eine Diagnose Meldung an, wenn der Ausdruck **false** ist. |
| [**Kassert**](kassert.md)                   | Wertet einen Ausdruck aus und verursacht eine breakpointausnahme, wenn der Ausdruck **false** ist.                                         |
| [**Kdbgbreak**](kdbgbreak.md)               | Verursacht eine breakpointausnahme und protokolliert die angegebene Zeichenfolge.                                                                      |
