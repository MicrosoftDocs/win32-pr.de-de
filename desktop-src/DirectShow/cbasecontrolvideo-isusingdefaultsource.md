# <a name="cbasecontrolvideoisusingdefaultsource-method"></a>Cbasecontrolvideo. isusingdefaultsource-Methode

Die- `IsUsingDefaultSource` Methode bestimmt, ob der Renderer das standardmäßige Quell Code Fenster verwendet.

## <a name="syntax"></a>Syntax


```C++
virtual HRESULT IsUsingDefaultSource() = 0;
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt s \_ OK zurück, wenn der Renderer die Standard Quelle verwendet; andernfalls wird s \_ false zurückgegeben.
