# <a name="track-element"></a>Track-Element

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das- `track` Element definiert ein Track-Objekt.

## <a name="attributes"></a>Attribute

[**Lock**](lock-attribute.md), [**stumm**](mute-attribute.md), [**UserData**](userdata-attribute.md), [**UserID**](userid-attribute.md), [**username**](username-attribute.md)

|          |                                                                                                          |
|----------|----------------------------------------------------------------------------------------------------------|
| Parent   | zusammen [**gesetzte**](composite-element.md), [ **Gruppe**](group-element.md)                                   |
| Children | [**Clip**](clip-element.md), [**Effekt**](effect-element.md), [**Übergang**](transition-element.md) |
