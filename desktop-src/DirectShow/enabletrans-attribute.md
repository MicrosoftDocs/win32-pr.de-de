# <a name="enabletrans-attribute"></a>enabletrans-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das- `enabletrans` Attribut gibt an, ob Übergänge aktiviert sind. Wenn der Wert **true** ist, rendert die Rendering-Engine die Übergänge vollständig. Andernfalls werden Sie als Schnitte gerendert. Der Standardwert ist " **true**".

## <a name="possible-values"></a>Mögliche Werte

Die folgenden Werte sind als true definiert: y, y, t, t, 1. Die folgenden Werte sind als false definiert: n, n, f, f, 0 (null).
