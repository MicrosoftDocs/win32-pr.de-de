# <a name="cbasefilterisstopped-method"></a>CBaseFilter.IsStopped-Methode

Die `IsStopped` -Methode bestimmt, ob der Filter derzeit beendet wird.

## <a name="syntax"></a>Syntax


```C++
BOOL IsStopped();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt **TRUE zurück,** wenn der Filter beendet wurde, andernfalls **FALSE.**
