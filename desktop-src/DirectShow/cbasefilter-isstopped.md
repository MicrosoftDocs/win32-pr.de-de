# <a name="cbasefilterisstopped-method"></a>Cbasefilter. isbeendet-Methode

Die- `IsStopped` Methode bestimmt, ob der Filter aktuell beendet ist.

## <a name="syntax"></a>Syntax


```C++
BOOL IsStopped();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt **true** zurück, wenn der Filter beendet wurde, andernfalls **false** .
