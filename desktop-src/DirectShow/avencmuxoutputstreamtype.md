# <a name="avencmuxoutputstreamtype-property"></a>AVEncMuxOutputStreamType-Eigenschaft

Gibt den Typ des Ausgabestreams an, der von einem Multiplexer erzeugt wird.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncMuxOutputStreamType**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eAVEncMuxOutput-Enumeration.**](/windows/win32/api/codecapi/ne-codecapi-eavencmuxoutput)
