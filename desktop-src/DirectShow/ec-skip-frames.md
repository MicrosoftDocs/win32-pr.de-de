# <a name="ec_skip_frames"></a>EC- \_ Skip- \_ Frames

Wird vom DVD-Navigator gesendet, wenn die Rahmen übersprungen werden müssen.

## <a name="parameters"></a>Parameter

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

Anzahl der zu über springenden Frames.

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

Reserviert.

</dd> </dl>

## <a name="default-action"></a>Standardaktion

Dieses Ereignis wird nicht an die Anwendung gesendet. Anwendungen können die Standardaktion dieses Ereignisses nicht überschreiben.
