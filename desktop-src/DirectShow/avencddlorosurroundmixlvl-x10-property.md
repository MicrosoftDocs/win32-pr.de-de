# <a name="avencddlorosurroundmixlvl_x10-property"></a>AVEncDDLoRoSurroundMixLvl \_ x10-Eigenschaft

Gibt die Ebenenverschiebung an, die auf die Surround-Kanäle für Lo/Ro-Downmixing angewendet wird. Diese Eigenschaft gilt für Dolby Digital-Audioencoder.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncDDLoRoSurroundMixLvl \_ x10**

## <a name="property-value"></a>Eigenschaftswert

Der Wert wird als decibels (dB) x 10 angegeben.
