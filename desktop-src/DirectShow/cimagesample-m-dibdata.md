# <a name="cimagesamplem_dibdata-member"></a>CImageSample::m \_ DibData-Member

Die `m_DibData` Membervariable ist eine [**DIBDATA-Struktur,**](dibdata.md) die Informationen zu einer geräteunabhängigen GDI-Bitmap (DIB) enthält.

## <a name="syntax"></a>Syntax


```C++
DIBDATA m_DibData;
```
