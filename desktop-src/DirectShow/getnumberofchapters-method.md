# <a name="getnumberofchapters-method"></a>Getnumofchapters-Methode

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `GetNumberOfChapters` Methode ruft die Anzahl der Kapitel im angegebenen Titel ab.

``` syntax
[iChapters = ] MSWebDVD.GetNumberOfChapters(iTitle)
```

## <a name="parameters"></a>Parameter

<dl> <dt>

<span id="iTitle"></span><span id="ititle"></span><span id="ITITLE"></span>*ititle*
</dt> <dd>

Gibt den Titel als Ganzzahl an.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt einen ganzzahligen Wert zwischen 1 und 999 zurück, der die Anzahl von Kapiteln angibt.
