# <a name="working-with-sources"></a>Arbeiten mit Quellen

\[Diese API wird nicht unterstützt und kann in Zukunft geändert oder nicht mehr verfügbar sein.\]

Ein Quellobjekt stellt einen einzelnen Medienstream aus einer Quelldatei dar. Quellen stellen das Rohmaterial für ein Projekt bereit. Alles andere auf der Zeitachse dient entweder zur Bereitstellung von Struktur (Spuren, Kompositionen und Gruppen) oder zum Ändern von Quellen (Effekte und Übergänge).

In diesem Artikel werden verschiedene Details zu Quellen in Microsoft® DirectShow® Editing Services (DES) beschrieben. Informationen zu den grundlegenden Schritten, die zum Hinzufügen einer Quelle zur Zeitachse erforderlich sind, finden Sie unter [Erstellen einer Zeitachse.](constructing-a-timeline.md)

Dieser Artikel enthält folgende Abschnitte.

-   [Festlegen von Eigenschaften für Quellen](setting-properties-on-sources.md)
-   [Verwenden der Medienerkennung](using-the-media-detector.md)
-   [Verwenden des Medienlocators](using-the-media-locator.md)
-   [Andere Quellobjekte](other-source-objects.md)
