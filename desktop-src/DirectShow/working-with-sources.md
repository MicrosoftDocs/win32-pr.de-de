# <a name="working-with-sources"></a>Arbeiten mit Quellen

\[Diese API wird nicht unterstützt und kann in Zukunft geändert oder nicht verfügbar sein.\]

Ein-Quell Objekt stellt einen einzelnen Mediendaten Strom aus einer Quelldatei dar. Quellen stellen das Rohmaterial für ein Projekt bereit. Alles andere in der Zeitachse ist entweder zur Bereitstellung von Struktur (Spuren, Komposition und Gruppen) oder zum Ändern von Quellen (Effekten und Übergänge) vorhanden.

In diesem Artikel werden verschiedene Details im Zusammenhang mit Quellen in Microsoft® DirectShow® Bearbeitungs Diensten beschrieben. Informationen zu den grundlegenden Schritten zum Hinzufügen einer Quelle zur Zeitachse finden Sie unter [Erstellen einer Zeitachse](constructing-a-timeline.md).

Dieser Artikel enthält folgende Abschnitte.

-   [Festlegen von Eigenschaften für Quellen](setting-properties-on-sources.md)
-   [Verwenden des Medien Detektors](using-the-media-detector.md)
-   [Verwenden des medienlocators](using-the-media-locator.md)
-   [Andere Quell Objekte](other-source-objects.md)
