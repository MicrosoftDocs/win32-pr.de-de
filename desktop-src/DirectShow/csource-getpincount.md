# <a name="csourcegetpincount-method"></a>CSource. getpincount-Methode

Die- `GetPinCount` Methode ruft die Anzahl der Pins für den Filter ab. Diese Methode implementiert die rein virtuelle [**cbasefilter:: getpincount**](cbasefilter-getpincount.md) -Methode.

## <a name="syntax"></a>Syntax


```C++
int GetPinCount();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt die Anzahl der Pins für diesen Filter zurück.
