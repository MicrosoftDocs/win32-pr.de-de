# <a name="cutscene-sample"></a>Beispiel für Cutscene

## <a name="description"></a>BESCHREIBUNG

Das Beispiel "Cutscene" spielt eine Videodatei im Vollbildmodus, bis der Benutzer ESC, die EINGABETASTE oder die Leertaste eingibt.

Führen Sie dieses Beispiel in der Befehlszeile mit dem Namen einer Videodatei als Befehlszeilenargument aus.

## <a name="downloading-the-sample"></a>Herunterladen des Beispiels

Zum Herunterladen der DirectShow SDK-Beispiele installieren Sie die neueste Version der [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Dieses Beispiel wird unter folgendem Pfad installiert: *\[ SDK Root \]* \\ Samples \\ Multimedia \\ DirectShow \\ Players \\ Cutscene.
