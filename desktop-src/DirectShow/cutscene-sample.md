# <a name="cutscene-sample"></a>CutScene-Beispiel

## <a name="description"></a>BESCHREIBUNG

Das CutScene-Beispiel gibt eine Videodatei im Vollbildmodus wieder, bis der Benutzer ESC, DIE EINGABETASTE oder die Leerzeichenleiste ein gibt.

Führen Sie dieses Beispiel über die Befehlszeile mit dem Namen einer Videodatei als Befehlszeilenargument aus.

## <a name="downloading-the-sample"></a>Herunterladen des Beispiels

Installieren Sie zum Herunterladen der DirectShow SDK-Beispiele die neueste Version des [Windows SDK.](https://msdn.microsoft.com/windowsvista/bb980924.aspx)

Dieses Beispiel wird unter folgendem Pfad installiert: *\[ SDK Root \]* Samples Multimedia \\ \\ \\ DirectShow Players \\ \\ CutScene.
