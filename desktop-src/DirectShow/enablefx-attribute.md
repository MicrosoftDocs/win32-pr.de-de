# <a name="enablefx-attribute"></a>enablefx-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das- `enablefx` Attribut gibt an, ob Effekte aktiviert sind. Wenn der Wert **true** ist, rendert die Renderingengine Effekte. Andernfalls trifft dies nicht zu. Der Standardwert ist " **true**".

## <a name="possible-values"></a>Mögliche Werte

Die folgenden Werte sind als true definiert: y, y, t, t, 1. Die folgenden Werte sind als false definiert: n, n, f, f, 0 (null).

## <a name="applies-to"></a>Gilt für

[**Zeitplan**](timeline-element.md)
