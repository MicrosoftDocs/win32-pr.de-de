# <a name="directshow-editing-services"></a>DirectShow-Bearbeitungsdienste

\[Diese API wird nicht unterstützt und kann in Zukunft geändert oder nicht mehr verfügbar sein.\]

DirectShow Editing Services (DES) ist eine Anwendungsprogrammierschnittstelle (Application Programming Interface, API), die die Aufgaben bei der Videobearbeitung erheblich vereinfacht. DES basiert auf der DirectShow-Kernarchitektur. Sie abstrahiert einen Großen Teil der Komplexität von DirectShow und stellt eine Reihe von Schnittstellen bereit, die speziell für die Bearbeitung von Videobearbeitungsprojekten entwickelt wurden. Als Anwendungsentwickler profitieren Sie von den Vorteilen von DirectShow innerhalb eines Frameworks, das viel besser für die Erstellung von Anwendungen zur Videobearbeitung geeignet ist.
