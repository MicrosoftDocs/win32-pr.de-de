# <a name="directshow-editing-services"></a>DirectShow-Bearbeitungs Dienste

\[Diese API wird nicht unterstützt und kann in Zukunft geändert oder nicht verfügbar sein.\]

DirectShow-Bearbeitungs Dienste (Application Programming Interface, des) ist eine API (Application Programming Interface), die die Aufgaben bei der Videobearbeitung erheblich vereinfacht. Des baut auf der Kernarchitektur von DirectShow auf. Es abstrahiert einen Großteil der Komplexität von DirectShow und stellt eine Reihe von Schnittstellen bereit, die speziell für die Bearbeitung von Video Bearbeitungs Projekten entwickelt wurden. Als Anwendungsentwickler erhalten Sie die Vorteile von DirectShow innerhalb eines Frameworks, das für die Erstellung von Anwendungen zur Videobearbeitung wesentlich besser geeignet ist.
