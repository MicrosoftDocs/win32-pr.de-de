# <a name="track"></a>Track

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das Track-Objekt enthält Quell Objekte. Um dieses Objekt zu erstellen, rufen Sie die [**iamtimeline:: erkreateemptynode**](iamtimeline-createemptynode.md) -Methode auf.

Das Track-Objekt macht die folgenden Schnittstellen verfügbar:

-   [**Iamtimelineeffectable**](iamtimelineeffectable.md)
-   [**Iamtimelineobj**](iamtimelineobj.md)
-   [**Iamtimelinesplicustom**](iamtimelinesplittable.md)
-   [**Iamtimelinetrack**](iamtimelinetrack.md)
-   [**Iamtimelinetransable**](iamtimelinetransable.md)
-   [**Iamtimelinevirtualtrack**](iamtimelinevirtualtrack.md)
