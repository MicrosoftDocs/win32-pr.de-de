# <a name="defaultaudiolanguageext-property"></a>Defaultaudiolanguageext (Eigenschaft)

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die `DefaultAudioLanguageExt` -Eigenschaft ruft die standardmäßige DVD-Audiosprach Erweiterung ab.

``` syntax
[ iLangExt = ] MSWebDVD.DefaultAudioLanguageExt
```

## <a name="return-value"></a>Rückgabewert

Gibt einen ganzzahligen Wert zurück, der die standardmäßige audiosprachweiterung angibt Informationen zu den möglichen Werten finden Sie unter "Hinweise".

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist schreibgeschützt und weist keinen Standardwert auf. In der folgenden Tabelle sind die möglichen Werte für die Erweiterungen der DVD-Audiosprache aufgeführt.



| Wert | BESCHREIBUNG       |
|-------|-------------------|
| 0     | NotSpecified      |
| 1     | Untertitel          |
| 2     | Visualisierungen beeinträchtigt  |
| 3     | DirectorComments1 |
| 4     | DirectorComments2 |
