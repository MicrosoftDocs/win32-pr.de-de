# <a name="clsid-attribute"></a>clsid-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Releases von Windows entfernt werden.\]

 

Das `clsid` -Attribut gibt einen Klassenbezeichner (CLSID) für ein Objekt an.

## <a name="possible-values"></a>Mögliche Werte

Muss eine Zeichenfolge im Format {00000000-0000-0000-0000-000000000000} sein.

## <a name="applies-to"></a>Gilt für

[**bei**](at-element.md), [**Effekt**](effect-element.md), [**Übergang**](transition-element.md)
