# <a name="xml-parser"></a>XML-Parser

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das XML-Parserobjekt konvertiert zwischen XML-Projektdateien und-Zeitachsen. Um dieses Objekt zu erstellen, rufen Sie **cokreateinstance** auf.

Das XML-Parserobjekt stellt die folgende Schnittstelle bereit:

-   [**IXml2Dex**](ixml2dex.md)

**Klassen Bezeichner:** CLSID- \_ Xml2Dex
