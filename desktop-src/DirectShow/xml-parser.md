# <a name="xml-parser"></a>XML-Parser

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Releases von Windows entfernt werden.\]

 

Das XML Parser-Objekt konvertiert zwischen XML-Projektdateien und Zeitachsen. Um dieses Objekt zu erstellen, rufen Sie **CoCreateInstance** auf.

Das XML Parser-Objekt macht die folgende Schnittstelle verfügbar:

-   [**IXml2Dex**](ixml2dex.md)

**Klassenbezeichner:** CLSID \_ Xml2Dex
