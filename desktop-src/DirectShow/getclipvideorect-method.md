# <a name="getclipvideorect-method"></a>Getclipvideorect-Methode

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `GetClipVideoRect` Methode ruft das Clippingrechteck ab, das derzeit für die Videoanzeige definiert ist.

``` syntax
[ oClipRect = ] MSWebDVD.GetClipVideoRect()
```

## <a name="return-value"></a>Rückgabewert

Gibt ein [dvdrect](dvdrect-object.md) -Objekt zurück, das das Rechteck definiert.
