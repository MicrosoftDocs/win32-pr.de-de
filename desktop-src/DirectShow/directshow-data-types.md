# <a name="directshow-data-types"></a>DirectShow-Datentypen

DirectShow stellt die folgenden Typdefinitionen bereit.



| Datentyp                                 | Beschreibung                                                                                       |
|-------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**\_DVD-REGISTER**](dvd-register.md)     | Enthält einen Wert für ein ALLGEMEINES DVD-Parameterregister (GPRM) oder ein Systemparameterregister (SPRM). |
| [**GPRMARRAY**](gprmarray.md)            | Enthält ein Array von GPRM-Werten.                                                                 |
| [**OAEVENT**](oaevent.md)                | Definiert ein Automation-kompatibles Ereignishandle.                                                    |
| [**OAFilterState**](oafilterstate.md)    | Definiert den Zustand des Filterdiagramms.                                                            |
| [**OAHWND**](oahwnd.md)                  | Definiert ein Automation-kompatibles Fensterhandle.                                                   |
| [**\_REFERENZZEIT**](reference-time.md) | Definiert den Datentyp für die Verweiszeit.                                                         |
| [**REFTIME**](reftime.md)                | Definiert den Datentyp, mit dem die Zeit in Sekunden ausgedrückt wird.                                            |
| [**SPRMARRAY**](sprmarray.md)            | Enthält ein Array von SPRM-Werten.                                                                 |
