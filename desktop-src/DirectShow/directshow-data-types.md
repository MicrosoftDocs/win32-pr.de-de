# <a name="directshow-data-types"></a>DirectShow-Datentypen

DirectShow stellt die folgenden Typdefinitionen bereit.



| Datentyp                                 | BESCHREIBUNG                                                                                       |
|-------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**DVD- \_ Register**](dvd-register.md)     | Enthält einen Wert für eine DVD allgemeine Parameter Register (GPRM) oder ein Systemparameter Register (SPRM). |
| [**Gprmarray**](gprmarray.md)            | Enthält ein Array von GPRM-Werten.                                                                 |
| [**Oaevent**](oaevent.md)                | Definiert ein Automatisierungs kompatibles Ereignis handle.                                                    |
| [**Oafilterstate**](oafilterstate.md)    | Definiert den Status des Filter Diagramms.                                                            |
| [**Oahwnd**](oahwnd.md)                  | Definiert ein Automatisierungs kompatibles Fenster handle.                                                   |
| [**Verweis \_ Zeit**](reference-time.md) | Definiert den Datentyp für die Verweis Zeit.                                                         |
| [**Ref-Zeit**](reftime.md)                | Definiert den Datentyp, der verwendet wird, um die Zeit in Sekunden auszudrücken.                                            |
| [**Sprmarray**](sprmarray.md)            | Enthält ein Array von SPRM-Werten.                                                                 |
