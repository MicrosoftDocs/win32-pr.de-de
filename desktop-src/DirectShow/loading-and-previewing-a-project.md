# <a name="loading-and-previewing-a-project"></a>Laden und Anzeigen einer Vorschau Project

\[Diese API wird nicht unterstützt und kann in Zukunft geändert oder nicht mehr verfügbar sein.\]

In diesem Thema wird beschrieben, wie Sie eine Projektdatei mit [DirectShow Editing Services](directshow-editing-services.md) (DES) laden und eine Vorschau anzeigen. Es zeigt eine Beispielanwendung, die eine XML-Projektdatei in eine leere Zeitachse lädt und die Zeitachse für die Vorschau rendert. Sie können die Beispielanwendung so ändern, dass sie mit jeder XML-Beispielprojektdatei verwendet wird, die mit diesem SDK bereitgestellt wird.

Dieser Artikel enthält die folgenden Themen.

-   [Laden einer Project Datei](loading-a-project-file.md)
-   [Vorschau des Project](previewing-the-project.md)
-   [Vorschau einer Project: Beispielcode](previewing-a-project--example-code.md)
