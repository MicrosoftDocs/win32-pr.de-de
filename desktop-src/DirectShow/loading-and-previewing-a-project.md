# <a name="loading-and-previewing-a-project"></a>Laden und Anzeigen der Vorschau eines Projekts

\[Diese API wird nicht unterstützt und kann in Zukunft geändert oder nicht verfügbar sein.\]

In diesem Thema wird beschrieben, wie eine Projektdatei mit [DirectShow-Bearbeitungs Diensten](directshow-editing-services.md) geladen und in der Vorschau angezeigt wird. Es zeigt eine Beispielanwendung, die eine XML-Projektdatei in eine leere Zeitachse lädt und die Zeitachse für die Vorschau rendert. Sie können die Beispielanwendung so ändern, dass Sie mit jeder XML-Beispiel Projektdatei arbeitet, die in diesem SDK enthalten ist.

Dieser Artikel enthält die folgenden Themen:

-   [Laden einer Projektdatei](loading-a-project-file.md)
-   [Anzeigen der Vorschau des Projekts](previewing-the-project.md)
-   [Anzeigen einer Vorschau eines Projekts: Beispiel Code](previewing-a-project--example-code.md)
