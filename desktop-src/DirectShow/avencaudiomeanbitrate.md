# <a name="avencaudiomeanbitrate-property"></a>AVEncAudioMeanBitRate (Eigenschaft)

Gibt die durchschnittliche Bitrate des codierten Audiostreams in Bits pro Sekunde an. Diese Eigenschaft gilt nur für Codierungsmodi mit konstanter Bitrate (CBR) und variabler Bitrate (VBR).

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncCommonMeanBitRate**

## <a name="property-value"></a>Eigenschaftswert

Encoder können diese Eigenschaft als enumerierte Menge oder als linearen Bereich implementieren.
