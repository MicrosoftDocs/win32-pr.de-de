# <a name="tv-audio-filter"></a>TV-Audiofilter

Der Filter TV-Audio ermöglicht die Steuerung der Tv-Audiodecodierung, der Stereo- oder monoauralen Auswahl und der Auswahl des sekundären Audioprogramms (SECONDARY Audio Program, SAP).



| Bezeichnung | Wert |
|------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| Filterschnittstellen                        | [**IAMTVAudio,**](/windows/desktop/api/Strmif/nn-strmif-iamtvaudio) **IPersistPropertyBag,** **ISpecifyPropertyPages,** [**IKsPropertySet**](ikspropertyset.md) |
| Eingabe-Stecknadelmedientypen                    | MEDIATYPE \_ AnalogAudio                                                                                                         |
| Eingabe-Pin-Schnittstellen                     | **IKsPropertySet**                                                                                                             |
| Medientypen des Ausgabepins                   | MEDIATYPE \_ AnalogAudio                                                                                                         |
| Ausgabe-Pin-Schnittstellen                    | **IKsPropertySet**                                                                                                             |
| Filtern von CLSID                             | CLSID \_ TVAudioFilter                                                                                                           |
| CLSID der Eigenschaftenseite                      | AM \_ KSCATEGORY \_ TVAUDIO                                                                                                        |
| [Verdienst](merit.md)                       | NOT USE (NICHT \_ \_ \_ VERWENDEN)                                                                                                            |
| [Filterkategorie](filter-categories.md) | CLSID \_ AudioInputDeviceCategory                                                                                                |
