# <a name="tv-audio-filter"></a>TV-Audiofilter

Der TV-Audiofilter bietet Kontrolle über die Audiodecodierung von Fernsehgeräten, Stereo-oder monoaural-Auswahl und SAP-Auswahl (sekundär Audioprogramm).
