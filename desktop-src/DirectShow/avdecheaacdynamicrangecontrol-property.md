# <a name="avdecheaacdynamicrangecontrol-property"></a>AVDecHEAACDynamicRangeControl-Eigenschaft

Aktiviert oder deaktiviert die Steuerung des dynamischen Bereichs in einem AAC-Decoder.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVDecHEAACDynamicRangeControl**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eAVDecHEAACDynamicRangeControl-Enumeration.**](/windows/desktop/api/codecapi/ne-codecapi-eavdecheaacdynamicrangecontrol)
