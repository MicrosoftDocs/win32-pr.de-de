# <a name="avencvideooutputchromaresolution-property"></a>AVEncVideoOutputColoraResolution-Eigenschaft

Gibt die Auflösung des codierten Videos an.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncVideoOutputColoraResolution**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eAVEncVideoColoraResolution-Enumeration.**](/windows/win32/api/codecapi/ne-codecapi-eavencvideochromaresolution)
