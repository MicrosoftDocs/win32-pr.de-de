# <a name="avencvideooutputchromaresolution-property"></a>Avencvideooutputchromaresolution (Eigenschaft)

Gibt die Chroma-Auflösung des codierten Videos an.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideooutputchromaresolution**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eavencvideochromaresolution**](/windows/win32/api/codecapi/ne-codecapi-eavencvideochromaresolution) -Enumeration.
