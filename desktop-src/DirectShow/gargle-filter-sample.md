# <a name="gargle-filter-sample"></a>Gargle-Filter Beispiel

## <a name="description"></a>BESCHREIBUNG

Der Gargle-Filter ist ein Transformations Filter für Audioeffekte.

Der Gargle-Filter moduliert das Wellenform, indem es die Wellenform mit einem anderen Wellen Formular multipliziert, das im Filter mathematisch generiert wird. Die modulating-Wellenform ist standardmäßig eine dreieckige Welle. Das Eigenschaften Blatt bietet auch eine Alternative zu einer quadratischen Welle. Sie können die Häufigkeit der modulating-Welle über das Eigenschaften Blatt des Filters festlegen. Bei niedrigen modulhäufigkeits Frequenzen klingt der Effekt wie ein Tremolo. Bei höheren modulhäufigkeits Frequenzen klingt Sie wie eine Verzerrung und fügt zusätzliche Frequenzen oberhalb und unterhalb des ursprünglichen unmodulierten Sounds hinzu.

Zum Herunterladen der DirectShow SDK-Beispiele installieren Sie die neueste Version der [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Dieses Beispiel wird unter folgendem Pfad installiert: *\[ SDK Root \]* \\ Samples \\ Multimedia \\ DirectShow \\ Filters \\ Gargle.
