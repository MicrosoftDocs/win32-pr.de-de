# <a name="cdeferredcommandgetmethod-method"></a>Cdeferredcommand. GetMethod-Methode

Die `GetMethod` -Methode ruft den Dispatchbezeichner (DISPID) der Methode ab, die ausgeführt werden soll.

## <a name="syntax"></a>Syntax


```C++
long GetMethod();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt die DispID der Methode zurück, die ausgeführt werden soll.
