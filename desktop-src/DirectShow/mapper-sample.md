# <a name="mapper-sample"></a>Mapper-Beispiel

## <a name="description"></a>BESCHREIBUNG

Dieses Beispiel veranschaulicht die Verwendung der [**IFilterMapper2:: enummatchingfilters**](/windows/desktop/api/Strmif/nf-strmif-ifiltermapper2-enummatchingfilters) -Methode, um Filter in der Registrierung zu suchen.

Die Benutzeroberfläche der Beispielanwendung enthält Steuerelemente, die mit den Parametern der [**enummatchingfilters**](/windows/desktop/api/Strmif/nf-strmif-ifiltermapper2-enummatchingfilters) -Methode, z. b. Minimalwert, PIN-Kategorien und Medientypen, identisch sind. Wählen Sie die gewünschten Suchoptionen aus, und klicken Sie dann auf die Schaltfläche **Suchen** , um die Filter aufzuzählen, die diese Optionen erfüllen.

## <a name="downloading-the-sample"></a>Herunterladen des Beispiels

Zum Herunterladen der DirectShow SDK-Beispiele installieren Sie die neueste Version der [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Dieses Beispiel wird unter folgendem Pfad installiert: \[ SDK Root \] \\ Samples \\ Multimedia \\ DirectShow \\ misc \\ Mapper.
