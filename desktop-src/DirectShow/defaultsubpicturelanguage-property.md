# <a name="defaultsubpicturelanguage-property"></a>Defaultsubpicturelanguage (Eigenschaft)

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `DefaultSubpictureLanguage` Eigenschaft ruft die Standardsprache des unter Bilds ab.

``` syntax
[ iLang = ] MSWebDVD.DefaultSubpictureLanguage
```

## <a name="return-value"></a>Rückgabewert

Gibt einen LCID-Wert zurück, der die primäre Sprach-ID für die standardmäßige Audiosprache enthält.

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist schreibgeschützt und weist keinen Standardwert auf.
