# <a name="time-format-guids"></a>Zeit Format-GUIDs

Die folgenden global eindeutigen Bezeichner (GUIDs) definieren verschiedene Zeitformate.



| GUID                                                                                                                                                                                       | Beschreibung                                       |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------|
| <span id="TIME_FORMAT_NONE"></span><span id="time_format_none"></span><dl> <dt>**Uhrzeit \_ Format \_ None**</dt> </dl>                    | Kein Format.<br/>                             |
| <span id="TIME_FORMAT_FRAME"></span><span id="time_format_frame"></span><dl> <dt>**Zeit \_ Format \_ Rahmen**</dt> </dl>                 | Video Frames.<br/>                          |
| <span id="TIME_FORMAT_SAMPLE"></span><span id="time_format_sample"></span><dl> <dt>**Beispiel für Zeit \_ Format \_**</dt> </dl>              | Beispiele im Stream.<br/>                 |
| <span id="TIME_FORMAT_FIELD"></span><span id="time_format_field"></span><dl> <dt>**Feld für Zeit \_ Format \_**</dt> </dl>                 | Verschachtelte Videofelder.<br/>               |
| <span id="TIME_FORMAT_BYTE"></span><span id="time_format_byte"></span><dl> <dt>**Zeit \_ Format \_ Byte**</dt> </dl>                    | Byte Offset innerhalb des Streams.<br/>         |
| <span id="TIME_FORMAT_MEDIA_TIME"></span><span id="time_format_media_time"></span><dl> <dt>**Uhrzeit \_ Format \_ Medien \_ Zeit**</dt> </dl> | Verweis Zeit (100-Nanosecond-Einheiten).<br/> |
