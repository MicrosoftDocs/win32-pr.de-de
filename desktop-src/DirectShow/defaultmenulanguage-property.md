# <a name="defaultmenulanguage-property"></a>Defaultmenulanguage (Eigenschaft)

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `DefaultMenuLanguage` Eigenschaft ruft die Standardmenü Sprache ab.

``` syntax
[ iLang = ] MSWebDVD.DefaultMenuLanguage
```

## <a name="return-value"></a>Rückgabewert

Gibt einen LCID-Wert zurück, der die primäre Sprach-ID für die standardmäßige Audiosprache enthält.

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft verfügt über Lese-/Schreibzugriff ohne Standardwert.
