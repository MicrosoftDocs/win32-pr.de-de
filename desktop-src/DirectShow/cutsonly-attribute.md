# <a name="cutsonly-attribute"></a>cutsonly-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Releases von Windows entfernt werden.\]

 

Das `cutsonly` -Attribut gibt an, ob ein Übergang als Schnitt gerendert wird. Wenn der Wert **TRUE** ist, ist der Übergang ein sofortiger Schnitt. Andernfalls wird der Übergang normal gerendert. Der Standardwert ist **FALSE**.

## <a name="possible-values"></a>Mögliche Werte

Die folgenden Werte sind als TRUE definiert: y, Y, t, T, 1. Die folgenden Werte sind als FALSE definiert: n, N, f, F, 0 (null).

## <a name="applies-to"></a>Gilt für

[**Übergang**](transition-element.md)
