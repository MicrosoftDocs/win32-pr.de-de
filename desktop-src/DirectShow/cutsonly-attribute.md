# <a name="cutsonly-attribute"></a>cutsonly-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das- `cutsonly` Attribut gibt an, ob ein Übergang als Ausschneiden gerendert wird. Wenn der Wert **true** ist, ist der Übergang ein sofortiger Ausschneiden. Andernfalls wird der Übergang normal gerendert. Der Standardwert ist **FALSE**.

## <a name="possible-values"></a>Mögliche Werte

Die folgenden Werte sind als true definiert: y, y, t, t, 1. Die folgenden Werte sind als false definiert: n, n, f, f, 0 (null).

## <a name="applies-to"></a>Gilt für

[**Übergang**](transition-element.md)
