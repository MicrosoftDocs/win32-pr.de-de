# <a name="cmediatyperesetformatbuffer-method"></a>CMediaType.ResetFormatBuffer-Methode

Die `ResetFormatBuffer` -Methode löscht den Formatblock. Die -Methode legt die **cbFormat-** und **pbFormat-Member** der **AM MEDIA \_ \_ TYPE-Struktur** auf **NULL fest.**

## <a name="syntax"></a>Syntax


```C++
void ResetFormatBuffer();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.
