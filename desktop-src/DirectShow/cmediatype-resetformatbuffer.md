# <a name="cmediatyperesetformatbuffer-method"></a>Cmediatype. resetformatbuffer-Methode

Die- `ResetFormatBuffer` Methode löscht den Format Block. Die-Methode legt die Member **cbformat** und **pbformat** der **am \_ - \_ Medientyp** -Struktur auf **null** fest.

## <a name="syntax"></a>Syntax


```C++
void ResetFormatBuffer();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.
