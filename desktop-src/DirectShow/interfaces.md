# <a name="directshow-interfaces"></a>DirectShow-Schnittstellen

Dieses Thema enthält Referenzinformationen zu den Microsoft® DirectShow®-Schnittstellen. Um eine breite Unterstützung für verschiedene Arten von Hardware, Dateiformaten und Streamvorgängen zu bieten, implementiert eine große Anzahl von Schnittstellen. Einige werden von Anwendungen verwendet. andere werden intern von Filtern und anderen DirectShow-Komponenten verwendet.

Das folgende Thema enthält eine alphabetische Liste aller DirectShow-Codeschnittstellen.

-   [Alphabetische Liste der DirectShow-Schnittstellen](alphabetical-list-of-directshow-interfaces.md)

Jede bestimmte Anwendung verwendet nur eine kleine Teilmenge dieser Schnittstellen. In den folgenden Themen werden Schnittstellen nach Funktionskategorie aufgeführt. Nicht jede Schnittstelle wird auf diesen Seiten aufgeführt, und einige Schnittstellen werden auf mehr als einer Seite angezeigt.

-   [Schnittstellen zum Erstellen von Filterdiagrammen](interfaces-for-building-filter-graphs.md)
-   [Schnittstellen zum Steuern eines Filterfilters Graph](interfaces-for-controlling-a-filter-graph.md)
-   [Audioaufnahme- und Renderingschnittstellen](audio-capture-and-rendering-interfaces.md)
-   [DVD-Schnittstellen](dvd-interfaces.md)
-   [Videoaufnahmeschnittstellen](video-capture-interfaces.md)
-   [Dateicodierungs- und Decodierungsschnittstellen](file-encoding-and-decoding-interfaces.md)
-   [Schnittstellen für Videorendering und -überlagerung](interfaces-for-video-rendering-and-overlay.md)
-   [Externe Gerätesteuerungsschnittstellen](external-device-control-interfaces.md)
