# <a name="directshow-interfaces"></a>DirectShow-Schnittstellen

Dieses Thema enthält Referenzinformationen zu den Kern Schnittstellen von Microsoft® DirectShow®. Um eine umfassende Unterstützung für verschiedene Arten von Hardware, Dateiformaten und Streamvorgängen bereitzustellen, implementiert eine große Anzahl von Schnittstellen. Einige werden von Anwendungen verwendet. andere werden intern von Filtern und anderen DirectShow-Komponenten verwendet.

Das folgende Thema enthält eine alphabetische Liste aller Code-DirectShow-Schnittstellen.

-   [Alphabetische Liste der DirectShow-Schnittstellen](alphabetical-list-of-directshow-interfaces.md)

Eine bestimmte Anwendung verwendet nur eine kleine Teilmenge dieser Schnittstellen. In den folgenden Themen sind die Schnittstellen nach funktionaler Kategorie aufgelistet. Auf diesen Seiten ist nicht jede Schnittstelle aufgeführt, und eine Schnittstelle wird auf mehr als einer Seite angezeigt.

-   [Schnittstellen zum Entwickeln von Filter Diagrammen](interfaces-for-building-filter-graphs.md)
-   [Schnittstellen zum Steuern eines Filter Diagramms](interfaces-for-controlling-a-filter-graph.md)
-   [Schnittstellen für Audioerfassung und-Rendering](audio-capture-and-rendering-interfaces.md)
-   [DVD-Schnittstellen](dvd-interfaces.md)
-   [Video Erfassungs Schnittstellen](video-capture-interfaces.md)
-   [Datei Codierung und Decodierungs Schnittstellen](file-encoding-and-decoding-interfaces.md)
-   [Schnittstellen für Video Rendering und Overlay](interfaces-for-video-rendering-and-overlay.md)
-   [Externe Geräte Steuerungs Schnittstellen](external-device-control-interfaces.md)
