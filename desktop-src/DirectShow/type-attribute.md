# <a name="type-attribute"></a>type-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das- `type` Attribut gibt den Medientyp einer Gruppe an.

## <a name="possible-values"></a>Mögliche Werte

Der Wert muss "Audiodatei" oder "Video" lauten.
