# <a name="dvd-graph-builder"></a>DVD-Diagramm-Generator

Der DVD Graph Builder ist ein Hilfsobjekt zum Entwickeln von DVD-Wiedergabe Diagrammen. Anwendungen können diese Komponente verwenden, um Filter Diagramme für die DVD-Wiedergabe und-Navigation zu erstellen. Erstellen Sie dieses Objekt durch Aufrufen von **CoCreateInstance**.



|                  |                                              |
|------------------|----------------------------------------------|
| Klassen Bezeichner | CLSID \_ dvdgraphbuilder                       |
| Schnittstellen       | [**Idvdgraphbuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) |
