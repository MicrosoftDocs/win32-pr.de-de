# <a name="name-attribute"></a>name-Attribut

> [!Note]  
> \[Veraltet. Diese API wird möglicherweise aus zukünftigen Releases von Windows.\]

 

Das `name` -Attribut gibt den Namen eines Objekts an.

## <a name="possible-values"></a>Mögliche Werte

Kann ein beliebiger Wert sein.

## <a name="applies-to"></a>Gilt für

[**group,**](group-element.md) [ **param**](param-element.md)

## <a name="remarks"></a>Hinweise

Für das **Gruppenelement** gibt dieses Attribut einen anwendungsdefinierten Namen für die Gruppe an.

Für das **param-Element** gibt dieses Attribut den Namen der Eigenschaft an.
