# <a name="name-attribute"></a>name-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das- `name` Attribut gibt den Namen eines-Objekts an.

## <a name="possible-values"></a>Mögliche Werte

Kann ein beliebiger Wert sein.

## <a name="applies-to"></a>Gilt für

[**Gruppieren**](group-element.md), [ **param**](param-element.md)

## <a name="remarks"></a>Bemerkungen

Für das **Group** -Element gibt dieses Attribut einen von der Anwendung definierten Namen für die Gruppe an.

Für das **param** -Element gibt dieses Attribut den Namen der Eigenschaft an.
