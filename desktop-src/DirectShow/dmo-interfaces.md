# <a name="dmo-interfaces"></a>DMO Schnittstellen

Dieser Abschnitt enthält Verweiseinträge für alle COM-Schnittstellen von Microsoft DirectX Media Object (DMO) und deren Methoden.



| Schnittstelle                                                            | BESCHREIBUNG                                                              |
|----------------------------------------------------------------------|--------------------------------------------------------------------------|
| [**IDMOQualityControl**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-idmoqualitycontrol)                     | Unterstützt die Qualitätssteuerung für DMO.                                       |
| [**IDMOVideoOutputOptimizations**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-idmovideooutputoptimizations) | Unterstützt Videooptimierungen auf einem DMO.                                   |
| [**IEnumDMO**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-ienumdmo)                                         | Stellt Methoden zum Aufzählen von DMOs zur                                   |
| [**IMediaBuffer**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-imediabuffer)                                 | Stellt Methoden zum Bearbeiten eines Datenpuffers zur Auswahl.                         |
| [**IMediaObject**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-imediaobject)                                 | Stellt Methoden zum Bearbeiten eines -DMO.                                 |
| [**IMediaObjectInPlace**](/previous-versions/windows/desktop/api/mediaobj/nn-mediaobj-imediaobjectinplace)                   | Stellt Methoden zum Verarbeiten von Daten an Ort und Stelle zur                           |
| [**IMediaParamInfo**](/previous-versions/windows/desktop/api/Medparam/nn-medparam-imediaparaminfo)                           | Ruft Informationen zu den Medienparametern ab, die ein Objekt unterstützt. |
| [**IMediaParams**](/previous-versions/windows/desktop/api/Medparam/nn-medparam-imediaparams)                                 | Legt Medienparameter für ein Objekt fest und ruft sie ab.                        |
