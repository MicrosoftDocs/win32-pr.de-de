# <a name="dmo-interfaces"></a>DMO-Schnittstellen

Dieser Abschnitt enthält Referenz Einträge für alle COM-Schnittstellen (DMO) von Microsoft DirectX Media Object (DMO) und ihre Methoden.



| Schnittstelle                                                            | BESCHREIBUNG                                                              |
|----------------------------------------------------------------------|--------------------------------------------------------------------------|
| [**Idmuqualitycontrol**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-idmoqualitycontrol)                     | Unterstützt die Qualitätskontrolle für einen DMO.                                       |
| [**Idmuvideooutputoptimierungen**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-idmovideooutputoptimizations) | Unterstützt Video Optimierungen in einem DMO.                                   |
| [**Ienumdmo**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-ienumdmo)                                         | Stellt Methoden zum Auflisten von dmos bereit.                                   |
| [**Imediabuffer**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-imediabuffer)                                 | Stellt Methoden zum Bearbeiten eines Daten Puffers bereit.                         |
| [**Imediaobject**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-imediaobject)                                 | Stellt Methoden zum Bearbeiten eines DMO bereit.                                 |
| [**Imediaobjectinplace**](/previous-versions/windows/desktop/api/mediaobj/nn-mediaobj-imediaobjectinplace)                   | Stellt Methoden bereit, mit denen Daten direkt verarbeitet werden.                           |
| [**Imediaparaminfo**](/previous-versions/windows/desktop/api/Medparam/nn-medparam-imediaparaminfo)                           | Ruft Informationen zu den von einem Objekt unterstützten Medien Parametern ab. |
| [**Imediaparams**](/previous-versions/windows/desktop/api/Medparam/nn-medparam-imediaparams)                                 | Legt Medien Parameter für ein-Objekt fest und ruft Sie ab.                        |
