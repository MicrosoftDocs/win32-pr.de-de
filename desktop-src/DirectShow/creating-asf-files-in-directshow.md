# <a name="creating-asf-files-in-directshow-directshow"></a>Erstellen von ASF-Dateien in DirectShow (DirectShow)

In diesem Abschnitt wird beschrieben, wie Sie den DirectShow- [WM-ASF-Writer](wm-asf-writer-filter.md) -Filter verwenden, um Dateien im Advanced Systems Format (ASF) zu erstellen. Bei ASF handelt es sich um ein Containerformat, das beliebige komprimierte oder unkomprimierte Daten enthalten kann. Windows Media-Format Dateien sind ASF-Dateien, die bestimmte Codecs verwenden, wie im Windows Media-Format Software Development Kit (SDK) angegeben. Diese Dateien verwenden die Erweiterungen ". wma" für Windows Media Audio Dateien und ". wmv" für Windows Media Video Dateien.
