# <a name="avencstatvideooutputframerate-property"></a>AVEncStatVideoOutputFrameRate-Eigenschaft

Gibt die durchschnittliche Bildfrequenz des Videoinhalts in Frames pro Sekunde zurück.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncStatVideoOutputFrameRate**

## <a name="remarks"></a>Hinweise

Diese Eigenschaft ist verfügbar, nachdem die Codierung abgeschlossen wurde.
