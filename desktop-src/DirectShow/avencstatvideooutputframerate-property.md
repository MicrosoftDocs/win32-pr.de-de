# <a name="avencstatvideooutputframerate-property"></a>Avencstatus videooutputframerate (Eigenschaft)

Gibt die durchschnittliche Framerate des Video Inhalts in Frames pro Sekunde zurück.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencstatus videooutputframerate**

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist nach Abschluss der Codierung verfügbar.
