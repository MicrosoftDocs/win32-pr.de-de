# <a name="avencvideodefaultupperfielddominant-property"></a>Avencvideodefaultupperfielddominant (Eigenschaft)

Gibt an, welches Feld zuerst angezeigt wird.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**Variant \_ bool** (**VT \_ bool**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideodefaultupperfielddominant**

## <a name="remarks"></a>Bemerkungen

Wenn der Wert **Variant \_ true** ist, wird das obere Feld zuerst angezeigt. Andernfalls wird das untere Feld zuerst angezeigt.
