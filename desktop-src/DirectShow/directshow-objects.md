# <a name="directshow-objects"></a>DirectShow-Objekte

Dieses Thema enthält Referenzeinträge für die wichtigsten COM-Objekte, die in DirectShow enthalten sind, mit Anderen als Filtern. Eine Liste der in DirectShow enthaltenen Filter finden Sie unter [DirectShow-Filter.](directshow-filters.md)



| Object                                                   | BESCHREIBUNG                                                      |
|----------------------------------------------------------|------------------------------------------------------------------|
| [Capture Graph Builder](capture-graph-builder.md)       | Erstellt Videoerfassungsdiagramme.                                     |
| [DVD Graph Builder](dvd-graph-builder.md)               | Erstellt DVD-Wiedergabediagramme.                                      |
| [Filter Graph Manager](filter-graph-manager.md)         | Erstellt und steuert Filterdiagramme.                               |
| [Filterzuordnung](filter-mapper.md)                       | Durchsucht die Registrierung nach registrierten Filtern.                    |
| [Medieneigenschafts-Bag](media-property-bag.md)             | Legt INFO- und DISP-Block in AVI-Dateien fest oder ruft sie ab.             |
| [Speicherzuweisung](memory-allocator.md)                 | Ordnet Puffer für Medienbeispiele zu.                             |
| [Systemreferenzuhr](system-reference-clock.md)     | Implementiert eine Verweisuhr.                                    |
| [Systemgeräte-Enumerator](system-device-enumerator.md) | Aufzählen von Filtern und Hardwaregeräten, die auf dem System installiert sind. |
