# <a name="creftimemillisecs-method"></a>Methode "up-time. Millisekunden"

Die- `Millisecs` Methode konvertiert die Verweis Zeit in Millisekunden.

## <a name="syntax"></a>Syntax


```C++
LONG Millisecs();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt die Verweis Zeit in Millisekunden zurück.
