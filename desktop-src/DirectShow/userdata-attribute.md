# <a name="userdata-attribute"></a>userdata-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Releases von Windows entfernt werden.\]

 

Das `userdata` -Attribut gibt anwendungsdefinierte persistente Daten für ein Objekt an.

## <a name="possible-values"></a>Mögliche Werte

Der Wert muss eine hexadezimale Zahl mit einer geraden Anzahl von Ziffern sein. Die Werte A–F müssen großgeschrieben werden. Verwenden Sie kein Präfix wie 0x. Beispiel: 123ABC.
