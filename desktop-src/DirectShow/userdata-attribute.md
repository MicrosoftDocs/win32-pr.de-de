# <a name="userdata-attribute"></a>UserData-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das- `userdata` Attribut gibt Anwendungs definierte persistente Daten für ein-Objekt an.

## <a name="possible-values"></a>Mögliche Werte

Der Wert muss eine hexadezimale-Zahl sein, die eine gerade Anzahl von Ziffern enthält. Die Werte A – F müssen groß geschrieben werden. Verwenden Sie kein Präfix wie z. b. "0x". Beispiel: 123aBc.

## <a name="applies-to"></a>Gilt für

[**Clip**](clip-element.md), zusammen [**gesetzte**](composite-element.md), [**Effekte**](effect-element.md), [**Gruppe**](group-element.md), [**Zeitachse**](timeline-element.md), [**Übergang**](transition-element.md)
