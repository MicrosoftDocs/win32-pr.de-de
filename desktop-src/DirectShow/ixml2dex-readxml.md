> [!Note]  
> \[Veraltet. Diese API wird möglicherweise aus zukünftigen Versionen von Windows entfernt.\]

 

Nicht implementiert.

## <a name="syntax"></a>Syntax


```C++
HRESULT ReadXML(
   IUnknown *pTimeline,
   IUnknown *pXML
);
```
