# <a name="bitdepth-attribute"></a>Bittiefen Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das- `bitdepth` Attribut gibt die Bittiefe des Ausgabevideos in Bits pro Pixel an.

## <a name="possible-values"></a>Mögliche Werte

Muss 16, 24 oder 32 sein. Der Standardwert ist 16.

## <a name="applies-to"></a>Gilt für

[**Kreis**](group-element.md)

## <a name="remarks"></a>Bemerkungen

Legen Sie dieses Attribut nur dann fest, wenn das Type-Attribut ist `video` .
