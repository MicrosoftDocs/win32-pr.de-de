# <a name="interfaces-for-controlling-a-filter-graph"></a>Schnittstellen zum Steuern eines Filter Diagramms

Diese Schnittstellen stellen Methoden zum Steuern eines Filter Diagramms bereit.



| Schnittstelle                                  | BESCHREIBUNG                                                                                               |
|--------------------------------------------|-----------------------------------------------------------------------------------------------------------|
| [**Iamclock-Anpassung**](/windows/desktop/api/Strmif/nn-strmif-iamclockadjust)   | Passen Sie die Diagramm Uhr an.                                                                                   |
| [**Iamgraphstreams**](/windows/desktop/api/Strmif/nn-strmif-iamgraphstreams) | Synchronisieren von Livestreams in einem Filter Diagramm.                                                               |
| [**Ifilterchain**](/windows/desktop/api/Strmif/nn-strmif-ifilterchain)       | Steuerungs Ketten von Filtern.                                                                                |
| [**IMediaControl**](/windows/desktop/api/Control/nn-control-imediacontrol)     | Ausführen, anhalten und Anhalten des Filter Diagramms. (Bietet auch Automatisierungs kompatible Methoden zum Entwickeln von Diagrammen.) |
| [**Imediaeventex**](/windows/desktop/api/Control/nn-control-imediaeventex)     | Reagieren auf Ereignisse im Diagramm.                                                                           |
| [**Imediaseeking**](/windows/desktop/api/Strmif/nn-strmif-imediaseeking)     | Suchen Sie in einer Datei.                                                                                       |
| [**Iqueuecommand**](/windows/desktop/api/Control/nn-control-iqueuecommand)     | Warteschlangen Befehle, die zu einem späteren Zeitpunkt ausgeführt werden.                                                                    |
| [**Ivideoframestep**](/windows/desktop/api/Strmif/nn-strmif-ivideoframestep) | Frame: Durchlaufen eines Videodaten Stroms.                                                                        |
