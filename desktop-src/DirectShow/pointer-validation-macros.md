# <a name="pointer-validation-macros"></a>Zeiger Validierungs Makros

Microsoft DirectShow stellt mehrere Makros zum Validieren von Zeigern bereit.



| Makro                                                | Beschreibung                                                                   |
|------------------------------------------------------|-------------------------------------------------------------------------------|
| [**Checkpointer**](checkpointer.md)                 | Überprüft, ob ein Zeiger **null** ist.                                         |
| [**Validatereadptr**](validatereadptr.md)           | Überprüft, ob der aufrufenden Prozess über Lesezugriff auf einen Speicherblock verfügt.          |
| [**Validatereadschreiteptr**](validatereadwriteptr.md) | Überprüft, ob der aufrufende Prozess über Lese-/Schreibzugriff auf einen Speicherblock verfügt.    |
| [**Validatestringptr**](validatestringptr.md)       | Überprüft, ob der aufrufenden Prozess Lesezugriff auf eine Zeichenfolge hat.                |
| [**Validatestringptra**](validatestringptra.md)     | Überprüft, ob der aufrufende Prozess über Lesezugriff auf eine ANSI-Zeichenfolge verfügt.          |
| [**Validatestringptrw**](validatestringptrw.md)     | Überprüft, ob der aufrufenden Prozess Lesezugriff auf eine breit Zeichen-Zeichenfolge hat. |
| [**Validateschreiteptr**](validatewriteptr.md)         | Überprüft, ob der aufrufende Prozess über Schreibzugriff auf einen Speicherblock verfügt.         |
