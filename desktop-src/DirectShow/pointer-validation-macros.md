# <a name="pointer-validation-macros"></a>Zeigervalidierungsmakros

Microsoft DirectShow stellt mehrere Makros zum Überprüfen von Zeigern bereit.



| Makro                                                | Beschreibung                                                                   |
|------------------------------------------------------|-------------------------------------------------------------------------------|
| [**CheckPointer**](checkpointer.md)                 | Überprüft, ob ein Zeiger **NULL** ist.                                         |
| [**ValidateReadPtr**](validatereadptr.md)           | Überprüft, ob der aufrufende Prozess Lesezugriff auf einen Speicherblock hat.          |
| [**ValidateReadWritePtr**](validatereadwriteptr.md) | Überprüft, ob der aufrufende Prozess Über Lese-/Schreibzugriff auf einen Speicherblock verfügt.    |
| [**ValidateStringPtr**](validatestringptr.md)       | Überprüft, ob der aufrufende Prozess Lesezugriff auf eine Zeichenfolge hat.                |
| [**ValidateStringPtrA**](validatestringptra.md)     | Überprüft, ob der aufrufende Prozess Lesezugriff auf eine ANSI-Zeichenfolge hat.          |
| [**ValidateStringPtrW**](validatestringptrw.md)     | Überprüft, ob der aufrufende Prozess Lesezugriff auf eine Breitzeichenzeichenfolge hat. |
| [**ValidateWritePtr**](validatewriteptr.md)         | Überprüft, ob der aufrufende Prozess Schreibzugriff auf einen Speicherblock hat.         |
