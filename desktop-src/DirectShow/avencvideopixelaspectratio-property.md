# <a name="avencvideopixelaspectratio-property"></a>Avencvideopixelaspectratio (Eigenschaft)

Gibt das Pixel Seitenverhältnis an.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideopixelaspectratio**

## <a name="property-value"></a>Eigenschaftswert

Die oberen 16 Bits des Werts enthalten die Breite, und die unteren 16 Bits enthalten die Höhe.
