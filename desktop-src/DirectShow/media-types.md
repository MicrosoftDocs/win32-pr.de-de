# <a name="media-types-directshow"></a>Medientypen (DirectShow)

Microsoft® DirectShow® verwendet die [**AM \_ MEDIA \_ TYPE-Struktur,**](/windows/win32/api/strmif/ns-strmif-am_media_type) um Medienbeispiele zu beschreiben. Diese Struktur enthält **GUID-Member** für Haupttyp, Untertyp und Formattyp. In diesem Abschnitt werden die Haupttyp- und Untertyp-GUIDs zusammengefasst, die für DirectShow definiert sind. Filter von Drittanbietern können eigene Medientyp-GUIDs definieren.

-   [**Haupttypen**](major-types.md)
-   [**Audiountertypen**](audio-subtypes.md)
-   [**BDA-Medientypen**](bda-media-types.md)
-   [**DVD-Medientypen**](dvd-media-types.md)
-   [**Line 21 Media Types**](line-21-media-types.md)
-   [**MPEG-1-Medientypen**](mpeg-1-media-types.md)
-   [MPEG-2-Medientypen](mpeg-2-media-types.md)
-   [**Streamuntertypen**](stream-subtypes.md)
-   [VBI-Medientypen](vbi-media-types.md)
-   [Videountertypen](video-subtypes.md)
