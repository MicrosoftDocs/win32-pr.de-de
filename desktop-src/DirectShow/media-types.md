# <a name="media-types-directshow"></a>Medientypen (DirectShow)

Microsoft® DirectShow® verwendet die [**am \_ \_ Medientyp**](/windows/win32/api/strmif/ns-strmif-am_media_type) -Struktur, um Medien Beispiele zu beschreiben. Diese Struktur enthält **GUID** -Member für den Haupttyp, den Untertyp und den Formattyp. In diesem Abschnitt werden die wichtigsten Typen-und Untertyp-GUIDs zusammengefasst, die für DirectShow definiert sind. Filter von Drittanbietern können eigene Medientyp-GUIDs definieren.

-   [**Haupttypen**](major-types.md)
-   [**Audiountertypen**](audio-subtypes.md)
-   [**BDA-Medientypen**](bda-media-types.md)
-   [**DVD-Medientypen**](dvd-media-types.md)
-   [**Zeilen 21-Medientypen**](line-21-media-types.md)
-   [**MPEG-1-Medientypen**](mpeg-1-media-types.md)
-   [MPEG-2-Medientypen](mpeg-2-media-types.md)
-   [**Streamsubtypen**](stream-subtypes.md)
-   [VBI-Medientypen](vbi-media-types.md)
-   [Video Untertypen](video-subtypes.md)
