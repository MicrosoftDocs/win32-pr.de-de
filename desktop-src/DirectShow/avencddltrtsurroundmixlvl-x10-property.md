# <a name="avencddltrtsurroundmixlvl_x10-property"></a>Avencddltrtsurroundmixlvl \_ x10 (Eigenschaft)

Gibt die auf den umschließenden Kanälen für die LT/RT-downmischung angewendete Schicht Schicht an. Diese Eigenschaft gilt für Dolby Digital-Audioencoder.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencddltrtsurroundmixlvl \_ x10**

## <a name="property-value"></a>Eigenschaftswert

Der Wert wird als Dezibel (DB) x 10 angegeben.
