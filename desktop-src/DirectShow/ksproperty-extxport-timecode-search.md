# <a name="ksproperty_extxport_timecode_search"></a>ksproperty \_ extxport- \_ \_ timecodesuche

Diese Eigenschaft sendet einen Befehl an das Gerät, um nach einem angegebenen Zeit Code zu suchen. Der UVC-Gerätetreiber unterstützt diese Eigenschaft.



|                   |                                        |
|-------------------|----------------------------------------|
| Eigenschaftensatz-GUID | propseetid- \_ Ext- \_ Transport              |
| Eigenschafts-ID       | ksproperty \_ extxport- \_ \_ timecodesuche |
| Datentyp         | **Ksproperty \_ Extxport \_ S** -Struktur  |



 

## <a name="remarks"></a>Bemerkungen

Füllen Sie den **Timecode** -Member der **ksproperty \_ extxport \_ S** -Struktur mit dem gewünschten Frame, dem zweiten, der Minute und der Stunde aus. Die Struktur von **ksproperty \_ extxport \_ S** wird im Windows-DDK beschrieben.
