# <a name="iamtimelinetrackmoveeverythingby-method"></a>IAMTimelineTrack::MoveEverythingBy-Methode

```C++
HRESULT MoveEverythingBy(
   REFERENCE_TIME Start,
   REFERENCE_TIME MoveBy
);
```

<dl> <dt>

</dd> <dt>
