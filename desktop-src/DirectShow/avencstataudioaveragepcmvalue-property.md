# <a name="avencstataudioaveragepcmvalue-property"></a>AVEncStatAudioAveragePCMValue-Eigenschaft

Gibt die durchschnittliche Volumeebene des Audioinhalts zurück.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ AVEncStatAudioAveragePCMValue**

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist nach Abschluss der Codierung verfügbar.
