# <a name="avencstataudioaveragepcmvalue-property"></a>AVEncStatAudioAveragePCMValue-Eigenschaft

Gibt die durchschnittliche Lautstärke des Audioinhalts zurück.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncStatAudioAveragePCMValue**

## <a name="remarks"></a>Hinweise

Diese Eigenschaft ist nach Abschluss der Codierung verfügbar.
