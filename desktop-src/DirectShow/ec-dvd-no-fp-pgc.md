# <a name="ec_dvd_no_fp_pgc"></a>EC- \_ DVD \_ keine FP- \_ \_ PGC

Signalisiert, dass die DVD-Diskette keine FP \_ PGC (First Play Program Chain) hat und dass der DVD-Navigator keine PGC automatisch lädt und die Wiedergabe startet.

## <a name="remarks"></a>Bemerkungen

Dieses Ereignis wird von der ersten Wiedergabe Domäne ausgelöst.
