# <a name="directshow-structures"></a>DirectShow-Strukturen

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

DirectShow Editing Services definiert die folgenden Strukturen.



| Struktur                             | Beschreibung                                          |
|---------------------------------------|------------------------------------------------------|
| [**DEXTER \_ PARAM**](dexter-param.md) | Gibt eine Eigenschaft für einen Übergang oder Effekt an.      |
| [**\_DEXTER-WERT**](dexter-value.md) | Gibt den Wert einer Eigenschaft an.                   |
| [**SCompFmt0**](scompfmt0.md)        | Enthält Formatinformationen für die intelligente Neukomprimierung. |
