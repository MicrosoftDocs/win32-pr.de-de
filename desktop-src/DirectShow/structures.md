# <a name="directshow-structures"></a>DirectShow-Strukturen

> [!Note]  
> \[Veraltet. Diese API wird möglicherweise aus zukünftigen Releases von Windows.\]

 

DirectShow Editing Services definiert die folgenden Strukturen.



| Struktur                             | Beschreibung                                          |
|---------------------------------------|------------------------------------------------------|
| [**DEXTER \_ PARAM**](dexter-param.md) | Gibt eine Eigenschaft für einen Übergang oder Effekt an.      |
| [**DEXTER-WERT \_**](dexter-value.md) | Gibt den Wert einer Eigenschaft an.                   |
| [**SCompFmt0**](scompfmt0.md)        | Enthält Formatinformationen für die intelligente Rekomprimierung. |
