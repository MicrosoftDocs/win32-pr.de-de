# <a name="avencvideooutputframerateconversion-property"></a>Avencvideooutputframerateconversion (Eigenschaft)

Gibt an, ob der Encoder die Framerate konvertiert, wenn die Ausgabe Frame Rate nicht mit der Eingangs Frame Rate identisch ist.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideooutputframerateconversion**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eavencvideooutputframerateconversion**](/windows/win32/api/codecapi/ne-codecapi-eavencvideooutputframerateconversion) -Enumeration.
