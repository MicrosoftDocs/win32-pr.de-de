# <a name="dvd-region-change-support-in-windows"></a>Unterstützung der Änderung von DVD-Regionen in Windows

In diesem Abschnitt wird erläutert, wie der Auswahlprozess der DVD-Region in Windows 2000 und höher funktioniert. Im ersten Abschnitt werden die drei Quellen der DVD-Regions Informationen beschrieben. Im zweiten Abschnitt wird beschrieben, wie PC-Systemhersteller die anfängliche DVD-Region eines Systems festlegen können. Im dritten Abschnitt wird erläutert, wie der DVD-Navigator die Regions Übereinstimmung zwischen Laufwerk, Decoder und CD bestimmt. Im letzten Abschnitt wird gezeigt, wie Anwendungs-und Decoder-Entwickler die korrekte Unterstützung der DVD-Regions Auswahl in Ihrem Code implementieren können.
