# <a name="dvd-region-change-support-in-windows"></a>Unterstützung von DVD-Regionsänderungen in Windows

In diesem Abschnitt wird erläutert, wie der DVD-Regionsauswahlprozess in Windows 2000 und höher funktioniert. Im ersten Abschnitt werden die drei Quellen von DVD-Regionsinformationen beschrieben. Im zweiten Abschnitt wird beschrieben, wie PC-Systemhersteller den ursprünglichen DVD-Bereich eines Systems festlegen können. Im dritten Abschnitt wird erläutert, wie der DVD-Navigator die Regionseinstimmung zwischen Laufwerk, Decoder und Datenträger bestimmt. Im letzten Abschnitt wird gezeigt, wie Anwendungs- und Decoderentwickler die richtige Unterstützung für die DVD-Regionsauswahl in ihrem Code implementieren können.
