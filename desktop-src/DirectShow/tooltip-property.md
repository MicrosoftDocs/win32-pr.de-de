# <a name="tooltip-property"></a>QuickInfo-Eigenschaft

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die -Eigenschaft legt den Text für die QuickInfo fest, der angezeigt wird, wenn sich der Mauszeiger über dem `ToolTip` **MSWebDVD-Videorechteck** befindet.

``` syntax
[ sToolTip = ] MSWebDVD.ToolTip
```

## <a name="return-value"></a>Rückgabewert

Gibt den QuickInfo-Text als Zeichenfolge zurück.

## <a name="remarks"></a>Hinweise

Diese Eigenschaft ist Lese-/Schreibzugriff ohne Standardwert. Wenn die Zeichenfolge leer ist, wird keine QuickInfo angezeigt.
