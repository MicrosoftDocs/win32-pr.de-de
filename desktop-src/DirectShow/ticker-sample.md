# <a name="ticker-sample"></a>Tickerbeispiel

## <a name="description"></a>Beschreibung

In diesem Beispiel wird der Video Mixing Renderer verwendet, um Video und Text zu mischen. Sie verwendet die [**IVMRMixerBitmap9-Schnittstelle,**](/previous-versions/windows/desktop/api/Vmr9/nn-vmr9-ivmrmixerbitmap9) um Text im unteren Bereich des Videofensters zu mischen.

Die Anwendung erstellt eine Bitmap mit einer Standardtextzeichenfolge und führt einen Bildlauf über den unteren Bildschirmrand durch. Um den Text zu ändern, wählen Sie im **Menü Ticker** die Option Textzeichenfolge **festlegen...** aus. Wählen Sie zum Festlegen der Schriftart im **Menü Ticker** die Option Schriftart **festlegen...** aus. Wenn Sie im **Menü** **Ticker** die Option Statisches Bild verwenden auswählen, wechselt die Anwendung von einer dynamischen Textzeichenfolge zu einem statischen Bitmapbild.

## <a name="downloading-the-sample"></a>Herunterladen des Beispiels

Um die DirectShow SDK-Beispiele herunterzuladen, installieren Sie die neueste Version des [Windows SDK.](https://msdn.microsoft.com/windowsvista/bb980924.aspx)
