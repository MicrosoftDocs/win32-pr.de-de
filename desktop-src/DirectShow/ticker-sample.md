# <a name="ticker-sample"></a>Ticker-Beispiel

## <a name="description"></a>BESCHREIBUNG

In diesem Beispiel wird der Video Mischungs-Renderer verwendet, um Videos und Text zu mischen. Die [**IVMRMixerBitmap9**](/previous-versions/windows/desktop/api/Vmr9/nn-vmr9-ivmrmixerbitmap9) -Schnittstelle wird verwendet, um Text auf den unteren Teil des Videofensters zu umblenden.

Die Anwendung erstellt eine Bitmap mit einer Standardtext Zeichenfolge und führt einen Bildlauf zum unteren Rand des Bildschirms durch. Um den Text zu ändern, wählen Sie im Menü **Ticker** die Option **Text Zeichenfolge festlegen...** aus. Um die Schriftart festzulegen, wählen Sie im Menü " **Ticker** " die Option **Schriftart festlegen...** aus. Wenn Sie im Menü " **Ticker** " die Option " **statisches Bild verwenden** " auswählen, wechselt die Anwendung von einer dynamischen Text Zeichenfolge zu einem statischen Bitmapbild.

## <a name="downloading-the-sample"></a>Herunterladen des Beispiels

Zum Herunterladen der DirectShow SDK-Beispiele installieren Sie die neueste Version der [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).
