# <a name="currentangle-property"></a>CurrentAngle-Eigenschaft

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die `CurrentAngle` -Eigenschaft legt den aktuellen Winkel in einem Winkelblock fest oder ruft sie ab.

``` syntax
[ iCurAngle = ] MSWebDVD.CurrentAngle
```

## <a name="return-value"></a>Rückgabewert

Gibt einen ganzzahligen Wert zurück, der den Winkel darstellt. Muss ein Wert zwischen 1 und 9 sein.

## <a name="remarks"></a>Hinweise

Diese Eigenschaft ist Lese-/Schreibzugriff ohne Standardwert.
