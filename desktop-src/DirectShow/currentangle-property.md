# <a name="currentangle-property"></a>Currentangle (Eigenschaft)

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `CurrentAngle` Eigenschaft legt den aktuellen Winkel in einem Winkel Block fest oder ruft ihn ab.

``` syntax
[ iCurAngle = ] MSWebDVD.CurrentAngle
```

## <a name="return-value"></a>Rückgabewert

Gibt einen Integer-Wert zurück, der den Winkel darstellt. Muss ein Wert zwischen 1 und 9 sein.

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist Lese-/Schreibzugriff und hat keinen Standardwert.
