# <a name="smart-render-engine"></a>Smart-Rendering-Engine

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das Smart-Rendering-Engine-Objekt rendert komprimierte Ausgaben aus einer Zeitachse Um dieses Objekt zu erstellen, rufen Sie **cokreateinstance** auf. Verwenden Sie für die nicht komprimierte Ausgabe die einfache Renderingengine. Der Klassen Bezeichner ist CLSID \_ smartrenderengine.

Die Smart-Rendering-Engine macht die folgenden Schnittstellen verfügbar:

-   [**Iameinterrorlog**](iamseterrorlog.md)
-   **IObjectWithSite**
-   [**"Unenderengine"**](irenderengine.md)
-   [**IRenderEngine2**](irenderengine2.md)
-   [**Ismartrenderengine**](ismartrenderengine.md)
