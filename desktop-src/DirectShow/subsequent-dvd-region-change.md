# <a name="subsequent-dvd-region-change"></a>Nachfolgende Änderung des DVD-Region

Ab Windows 2000 ruft der DVD-Navigator eine Eigenschaftenseite auf dem DVD-ROM-Laufwerkgerät im Geräte-Manager. Diese Eigenschaftenseite wird auch von der DVDPlay.exe angezeigt, wenn eine Region geändert werden muss. Die Benutzeroberfläche dieser Eigenschaftenseite ähnelt der von DVDRgn.exe und ist im Betriebssystem regionalisiert.

Bei RPC1-Laufwerken verwalten die Windows Betriebssystemkomponenten Die Region wird verwaltet. RPC2-Laufwerke verwalten die Region auf ihrer eigenen Hardware. Wenn die Anzahl der zulässigen Änderungen auf einem RPC2-Laufwerk erschöpft ist, kann das Laufwerk den Aufruf zum Ändern der Region nicht mehr vornehmen, und die Komponente für die Änderung der Region im Betriebssystem gibt dies für den Benutzer an.
