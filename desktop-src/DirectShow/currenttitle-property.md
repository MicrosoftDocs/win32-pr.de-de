# <a name="currenttitle-property"></a>Currenttitle (Eigenschaft)

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `CurrentTitle` Eigenschaft ruft die Nummer des aktuell wiedergegebenen Titels ab.

``` syntax
[ iCurTitle = ] MSWebDVD.CurrentTitle
```

## <a name="return-value"></a>Rückgabewert

Gibt einen ganzzahligen Wert zwischen 1 und 99 zurück, der den aktuellen Titel der CD darstellt.

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist schreibgeschützt und weist keinen Standardwert auf.
