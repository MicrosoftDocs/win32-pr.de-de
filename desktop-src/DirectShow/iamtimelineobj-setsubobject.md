> \[Veraltet. Diese API wird möglicherweise aus zukünftigen Versionen von Windows entfernt.\]

 

Wird nicht unterstützt.

## <a name="syntax"></a>Syntax


```C++
HRESULT SetSubObject(
   IUnknown *newVal
);
```



## <a name="parameters"></a>Parameter

<dl> <dt>

*newVal* 
</dt> <dd>

Reserviert.

</dd> </dl>
