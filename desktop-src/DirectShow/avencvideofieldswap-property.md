# <a name="avencvideofieldswap-property"></a>Avencvideofieldswap (Eigenschaft)

Kehrt die Reihenfolge der Zeilen Sprung Felder im Quellvideo um.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**Variant \_ bool** (**VT \_ bool**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideofieldswap**

## <a name="remarks"></a>Bemerkungen

Wenn der Wert **Variant \_ true** ist, tauscht der Encoder die Feld Reihenfolge aus.
