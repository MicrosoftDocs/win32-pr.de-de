# <a name="per-pixel-alpha-blending"></a>Per-Pixel Alpha Blending

Es wurden vier Medien Untertypen für die Alpha Mischung pro Pixel definiert. Diese Untertypen werden nur unterstützt, wenn sich der VMR im Mischungs Modus befindet. Die VMR lehnt die Verbindung ab, wenn Sie sich nicht im Mischungs Modus befindet, wenn ein upstreamfilter versucht, eine Verbindung mit einem dieser Untertypen herzustellen. Diese Untertypen sind in UUIDs. h definiert:

-   Mediasubtype \_ ARGB1555
-   Mediasubtype \_ ARGB4444
-   Mediasubtype \_ ARGB32
-   mediasubtype \_ ayuv

Weitere Informationen finden Sie unter [unkomprimierte RGB-Video Untertypen](uncompressed-rgb-video-subtypes.md) und [**YUV-Video Untertypen**](yuv-video-subtypes.md).
