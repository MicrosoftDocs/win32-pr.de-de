# <a name="closed-captions-and-teletext"></a>Untertitel und Teletext

Analoge Fernsehsignale können zusätzliche Informationen im vertikalen Leerungsintervall (Vertical Blanking Interval, VBI) enthalten, einschließlich Untertiteln und World Standard Teletext (WST). DirectShow stellt eine Reihe von Filtern zum Decodieren und Anzeigen von CC- und WST-Text bereit.

-   [Anzeigen von Untertiteln](viewing-closed-captions.md)
-   [Viewing World Standard Teletext](viewing-world-standard-teletext.md)
