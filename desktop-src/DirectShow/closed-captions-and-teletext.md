# <a name="closed-captions-and-teletext"></a>Untertitel und Teletext

Analog zu Fernsehsignalen können zusätzliche Informationen im vertikalen Auslassungs Intervall (VBI), einschließlich der Untertitel und des World Standard-Teletexts (WST), enthalten. DirectShow stellt eine Reihe von Filtern für das Decodieren und Anzeigen von CC-und WST-Text bereit.

-   [Anzeigen von Untertiteln](viewing-closed-captions.md)
-   [Anzeigen von World Standard-Teletext](viewing-world-standard-teletext.md)
