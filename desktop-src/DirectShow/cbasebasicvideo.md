# <a name="cbasebasicvideo-class"></a>Cbasebasicvideo-Klasse

![cbasebasicvideo-Klassenhierarchie](images/wctrl02.png)

Die- `CBaseBasicVideo` Klasse implementiert die **IDispatch** -Methoden der [**ibasicvideo**](/windows/desktop/api/Control/nn-control-ibasicvideo) -Schnittstelle. Die verbleibenden **ibasicvideo** -Methoden werden als rein virtuelle Methoden belassen und müssen von einer abgeleiteten Klasse implementiert werden.

Bei den **IDispatch** -Methoden in dieser Klasse handelt es sich um Standard Implementierungen, die hier nicht ausführlich beschrieben werden.
