# <a name="composition"></a>Aufbau

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Releases von Windows entfernt werden.\]

 

Das Kompositionsobjekt ist ein Container für Spuren. Sie kann auch andere Kompositionen (für die Schachtelung von Spuren), Effekte und Übergänge enthalten. Um dieses Objekt zu erstellen, rufen Sie die [**IAMTimeline::CreateEmptyNode-Methode**](iamtimeline-createemptynode.md) auf.

Das Kompositionsobjekt macht die folgenden Schnittstellen verfügbar:

-   [**IAMTimelineComp**](iamtimelinecomp.md)
-   [**IAMTimelineEffectable**](iamtimelineeffectable.md)
-   [**IAMTimelineObj**](iamtimelineobj.md)
-   [**IAMTimelineTransable**](iamtimelinetransable.md)
-   [**IAMTimelineVirtualTrack**](iamtimelinevirtualtrack.md)
