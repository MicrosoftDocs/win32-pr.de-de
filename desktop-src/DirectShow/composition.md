# <a name="composition"></a>Aufbau

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das Kompositions Objekt ist ein Container für Spuren. Sie kann auch andere Kompositionen (für die Schachtelung von Spuren), Effekte und Übergänge enthalten. Um dieses Objekt zu erstellen, rufen Sie die [**iamtimeline:: erkreateemptynode**](iamtimeline-createemptynode.md) -Methode auf.

Das Kompositions Objekt macht die folgenden Schnittstellen verfügbar:

-   [**Iamtimelinecomp**](iamtimelinecomp.md)
-   [**Iamtimelineeffectable**](iamtimelineeffectable.md)
-   [**Iamtimelineobj**](iamtimelineobj.md)
-   [**Iamtimelinetransable**](iamtimelinetransable.md)
-   [**Iamtimelinevirtualtrack**](iamtimelinevirtualtrack.md)
