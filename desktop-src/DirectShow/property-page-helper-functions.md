# <a name="property-page-helper-functions"></a>Hilfsfunktionen für Eigenschaftenseiten

Die Videoctl.h-Headerdatei in den Microsoft® DirectShow®-Basisklassen bietet Funktionen zur Unterstützung bei Implementierungen von Eigenschaftenseiten.



| Funktion                                                 | Beschreibung                                                                            |
|----------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**GetDialogSize**](getdialogsize.md)                   | Ruft die Größe eines Ressourcendialogfelds in Bildschirmpixeln ab.                          |
| [**StringFromResource**](stringfromresource.md)         | Lädt eine Zeichenfolge aus einer Ressourcendatei mit dem angegebenen Ressourcenbezeichner.                |
| [**WideStringFromResource**](widestringfromresource.md) | Lädt eine Breitzeichenzeichenfolge aus einer Ressourcendatei mit dem angegebenen Ressourcenbezeichner. |
