# <a name="property-page-helper-functions"></a>Hilfsfunktionen für Eigenschaften Seiten

Die Header Datei "videoctl. h" in der Microsoft® DirectShow-®-Basisklasse stellt Funktionen bereit, die bei Implementierungen von Eigenschaften Seiten helfen.



| Funktion                                                 | BESCHREIBUNG                                                                            |
|----------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**Getdialogsize**](getdialogsize.md)                   | Ruft die Größe eines Ressourcen Dialogfelds in Bildschirm Pixel ab.                          |
| [**Stringfromresource**](stringfromresource.md)         | Lädt eine Zeichenfolge aus einer Ressourcen Datei mit dem angegebenen Ressourcen Bezeichner.                |
| [**Widestringfromresource**](widestringfromresource.md) | Lädt eine breit Zeichen-Zeichenfolge aus einer Ressourcen Datei mit dem angegebenen Ressourcen Bezeichner. |
