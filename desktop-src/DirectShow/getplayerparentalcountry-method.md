# <a name="getplayerparentalcountry-method"></a>Getplayerparameentalcountry-Methode

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die `GetPlayerParentalCountry` -Methode ruft das aktuelle Land bzw. die aktuelle Region ab, das im **mswebdvd** -Objekt festgelegt ist.

``` syntax
[ iCountryCode = ] MSWebDVD.GetPlayerParentalCountry()
```

## <a name="return-value"></a>Rückgabewert

Gibt einen ganzzahligen Wert mit dem Länder-/Regionscode zurück.
