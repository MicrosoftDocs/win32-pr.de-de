# <a name="getdvdtextstringtype-method"></a>Getdvdtextstringtype-Methode

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `GetDVDTextStringType` Methode ruft einen Wert ab, der den Typ der Informationen angibt, die in der angegebenen DVD-Text Zeichenfolge enthalten sind.

``` syntax
[ iStringType = ] MSWebDVD.GetDVDTextStringType(iLangIndex, iStringIndex)
```

## <a name="parameters"></a>Parameter

<dl> <dt>

<span id="iLangIndex"></span><span id="ilangindex"></span><span id="ILANGINDEX"></span>*ilangindex*
</dt> <dd>

Gibt die Sprache der Text Zeichenfolge als Ganzzahl an.

</dd> <dt>

<span id="iStringIndex"></span><span id="istringindex"></span><span id="ISTRINGINDEX"></span>*istringindex*
</dt> <dd>

Gibt die Indexnummer der Text Zeichenfolge als Ganzzahl an.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt einen ganzzahligen Wert zurück, der den Typ der in der Zeichenfolge enthaltenen Informationen angibt.
