# <a name="avencvideoforcesourcescantype-property"></a>AVEncVideoForceSourceScanType (Eigenschaft)

Gibt an, ob die Eingabeframes progressiv oder interlaced sind.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncVideoForceSourceScanType**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eAVEncVideoSourceScanType-Enumeration.**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideosourcescantype)
