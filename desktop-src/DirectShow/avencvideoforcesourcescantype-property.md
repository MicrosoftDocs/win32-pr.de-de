# <a name="avencvideoforcesourcescantype-property"></a>Avencvideoforcesourcescantype (Eigenschaft)

Gibt an, ob die Eingabe Frames progressiv oder Zeilen Sprung sind.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideoforcesourcescantype**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eavencvideosourcescantype**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideosourcescantype) -Enumeration.
