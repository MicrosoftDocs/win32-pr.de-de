# <a name="avencddpreferredstereodownmixmode-property"></a>Avencddpreferredstereodownmixmode (Eigenschaft)

Gibt den bevorzugten Stereo downmischmodus für einen Dolby Digital-Audiodatenstrom an. Diese Eigenschaft gilt für Dolby Digital-Audioencoder.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencddpreferredstereodownmixmode**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eavencddpreferredstereodownmixmode**](/windows/desktop/api/codecapi/ne-codecapi-eavencddpreferredstereodownmixmode) -Enumeration.

## <a name="remarks"></a>Bemerkungen

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.
