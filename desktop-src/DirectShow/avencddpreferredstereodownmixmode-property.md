# <a name="avencddpreferredstereodownmixmode-property"></a>AVEncDDPreferredStereoDownMixMode-Eigenschaft

Gibt den bevorzugten Stereo-Downmixmodus für einen Dolby Digital-Audiostream an. Diese Eigenschaft gilt für Dolby Digital-Audioencoder.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncDDPreferredStereoDownMixMode**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eAVEncDDPreferredStereoDownMixMode-Enumeration.**](/windows/desktop/api/codecapi/ne-codecapi-eavencddpreferredstereodownmixmode)

## <a name="remarks"></a>Hinweise

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.
