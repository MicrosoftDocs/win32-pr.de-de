# <a name="avencvideosourceisbw-property"></a>Avencvideosourceisbw (Eigenschaft)

Gibt an, ob das Video Monochrom (schwarz und weiß) oder Farbe enthält.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**Variant \_ bool** (**VT \_ bool**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideosourceisbw**

## <a name="property-value"></a>Eigenschaftswert

Wenn der Wert **Variant \_ true** ist, ist das Video Monochrom. Andernfalls enthält das Video die Farbe.
