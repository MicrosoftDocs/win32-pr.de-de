# <a name="coutputqueueisqueued-method"></a>Coutputqueue. IsQueued-Methode

Die- `IsQueued` Methode bestimmt, ob das Objekt einen Arbeits Thread zum Übermitteln von Beispielen verwendet.

## <a name="syntax"></a>Syntax


```C++
BOOL IsQueued();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt " **true** " zurück, wenn das Objekt einen Arbeits Thread verwendet, andernfalls " **false** ".
