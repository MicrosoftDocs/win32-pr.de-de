# <a name="avencvideodisplaydimension-property"></a>Avencvideodisplaydimension (Eigenschaft)

Gibt die Größe des Videodaten Stroms an, wenn er decodiert wird.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideodisplaydimension**

## <a name="property-value"></a>Eigenschaftswert

Die oberen 16 Bits des Werts enthalten die Breite in Pixel. Die unteren 16 Bits enthalten die Höhe in Pixel.
