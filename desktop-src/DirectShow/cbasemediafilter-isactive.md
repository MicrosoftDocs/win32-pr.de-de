# <a name="cbasemediafilterisactive-method"></a>Cbasemediafilter. IsActive-Methode

Die- `IsActive` Methode bestimmt, ob das-Objekt aktiv ist (wird ausgeführt oder angehalten).

## <a name="syntax"></a>Syntax


```C++
BOOL IsActive();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt " **true** " zurück, wenn das Objekt angehalten oder ausgeführt wird, oder " **false** ", wenn es beendet wurde.
