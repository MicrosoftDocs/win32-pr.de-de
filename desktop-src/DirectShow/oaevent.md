# <a name="oaevent"></a>Oaevent

Der **oaevent** -Datentyp definiert ein Ereignis handle, das von OLE-Automatisierungs Clients verwendet werden kann, wie z. b. Microsoft Visual Basic 6,0.


```C++
typedef LONG_PTR OAEVENT;
```
