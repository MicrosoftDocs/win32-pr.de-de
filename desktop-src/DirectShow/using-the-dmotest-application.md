# <a name="using-the-dmotest-application"></a>Verwenden der DMOTest-Anwendung

> [!Note]  
> Die DMOTest-Anwendung ist im SDK nicht mehr verfügbar.

 

Das DMOTest-Hilfsprogramm kann dabei helfen, zu überprüfen, ob ein DMO die DMO Spezifikationen erfüllt. Wenn Sie eine DMO entwickeln, sollten Sie die DMO Testanwendung als Teil der Tests einschließen.

DMOTest wird im Verzeichnis Bin \\ DXUtils unter dem SDK-Stammverzeichnis installiert. Um DMOTest verwenden zu können, müssen Sie zunächst einige Testdaten für die DMO generieren. Das SDK enthält zu diesem Zweck einen DirectShow-Filter. Details zum Filter und zur DMOTest-Anwendung finden Sie in der Hilfe zur DMOTest-Anwendung. Wählen Sie in der Anwendung im **Menü Hilfe** die Option **Inhalt** aus. Die Hilfe enthält schrittweise Anweisungen zum Generieren von Testdaten und Ausführen von Tests.
