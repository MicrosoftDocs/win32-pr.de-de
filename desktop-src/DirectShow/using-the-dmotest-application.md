# <a name="using-the-dmotest-application"></a>Verwenden der dmutest-Anwendung

> [!Note]  
> Die dmutest-Anwendung ist im SDK nicht mehr verfügbar.

 

Mithilfe des Hilfsprogramms dmotest können Sie überprüfen, ob ein DMO den DMO-Spezifikationen entspricht. Wenn Sie ein DMO entwickeln, sollten Sie die DMO-Testanwendung als Teil Ihrer Tests einschließen.

Dmutest wird im Verzeichnis "bin \\ dxutils" unter dem SDK-Stammverzeichnis installiert. Wenn Sie dmotest verwenden möchten, müssen Sie zuerst einige Testdaten für den DMO generieren. Das SDK enthält zu diesem Zweck einen DirectShow-Filter. Details zum Filter und zur dmutest-Anwendung finden Sie in der Hilfe zur dmutest-Anwendung. Wählen Sie in der Anwendung im Menü **Hilfe** die Option **Inhalt** aus. Die Hilfe enthält schrittweise Anleitungen zum Erstellen von Testdaten und zum Ausführen von Tests.
