# <a name="samplingrate-attribute"></a>Samplingrate-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das `samplingrate` -Attribut gibt die Samplingrate der Ausgabe Audiodaten in Hz an.

## <a name="possible-values"></a>Mögliche Werte

Der Wert muss 8000, 11025, 22050, 32000, 44100 oder 48000 lauten. Der Standardwert ist 44100.

## <a name="applies-to"></a>Gilt für

[**Kreis**](group-element.md)

## <a name="remarks"></a>Bemerkungen

Legen Sie dieses Attribut nur dann fest, wenn das **Type** -Attribut ist `audio` .
