# <a name="samplingrate-attribute"></a>Samplingrate-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Releases von Windows entfernt werden.\]

 

Das `samplingrate` -Attribut gibt die Samplingrate der Ausgabeaudiodaten in Hz an.

## <a name="possible-values"></a>Mögliche Werte

Der Wert muss 8000, 11025, 22050, 32000, 44100 oder 48000 sein. Der Standardwert ist 44100.

## <a name="applies-to"></a>Gilt für

[**Gruppe**](group-element.md)

## <a name="remarks"></a>Hinweise

Legen Sie dieses  Attribut nur fest, wenn das Typattribut `audio` ist.
