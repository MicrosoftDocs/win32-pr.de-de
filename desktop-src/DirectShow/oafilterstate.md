# <a name="oafilterstate"></a>OAFilterState

Der `OAFilterState` Datentyp definiert den Zustand des Filterdiagramms. Der Wert ist ein Member der [**FILTER \_ STATE-Enumeration.**](/windows/win32/api/strmif/ne-strmif-filter_state) Dieser Datentyp wird für OLE-Automatisierungsclients wie Microsoft Visual Basic 6.0 definiert.


```C++
typedef long OAFilterState;
```
