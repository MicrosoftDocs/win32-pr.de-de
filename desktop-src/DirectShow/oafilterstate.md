# <a name="oafilterstate"></a>Oafilterstate

Der- `OAFilterState` Datentyp definiert den Status des Filter Diagramms. Der Wert ist ein Member der [**Filter \_ Status**](/windows/win32/api/strmif/ne-strmif-filter_state) -Enumeration. Dieser Datentyp wird für OLE-Automatisierungs Clients definiert, wie z. b. Microsoft Visual Basic 6,0.


```C++
typedef long OAFilterState;
```
