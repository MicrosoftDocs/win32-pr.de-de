# <a name="amcap-sample"></a>Amcap-Beispiel

## <a name="description"></a>BESCHREIBUNG

Video Erfassungs Anwendung.

Diese Beispielanwendung veranschaulicht die folgenden Aufgaben im Zusammenhang mit der Audio-und Video Erfassung:

-   In einer Datei erfassen
-   Live Vorschau
-   Zuordnung der Erfassungs Datei
-   Anzeigen von Geräteeigenschaften Seiten
-   Geräte Enumeration
-   Stream-Steuerelement

Amcap unterstützt MPEG-2-Programm Datenstrom Eingaben, z. b. von analogen TV-Tuners, die MPEG-2-Inhalte streamen. Zum Decodieren der Streams ist ein DirectShow-kompatibler MPEG-2-Decoder erforderlich.

## <a name="downloading-the-sample"></a>Herunterladen des Beispiels

Zum Herunterladen der DirectShow SDK-Beispiele installieren Sie die neueste Version der [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Dieses Beispiel wird unter folgendem Pfad installiert: \[ *SDK Root* \] \\ Samples \\ Multimedia \\ DirectShow \\ Capture \\ amcap.
