# <a name="avencddlorocentermixlvl_x10-property"></a>Avencddlorocentermixlvl \_ x10 (Eigenschaft)

Gibt die Schicht Schicht an, die auf den Mittelpunkt für das LO/RO-Downmix angewendet wird. Diese Eigenschaft gilt für Dolby Digital-Audioencoder.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencddlorocentermixlvl \_ x10**

## <a name="property-value"></a>Eigenschaftswert

Der Wert wird als Dezibel (DB) x 10 angegeben.
