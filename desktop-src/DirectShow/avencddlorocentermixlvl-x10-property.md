# <a name="avencddlorocentermixlvl_x10-property"></a>AVEncDDLoRoCenterMixLvl \_ x10 (Eigenschaft)

Gibt die Pegelverschiebung an, die für das Lo/Ro-Downmixing auf den Mittelkanal angewendet wird. Diese Eigenschaft gilt für Dolby Digital-Audioencoder.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncDDLoRoCenterMixLvl \_ x10**

## <a name="property-value"></a>Eigenschaftswert

Der Wert wird als decbels (dB) x 10 angegeben.
