# <a name="directshow-base-classes"></a>DirectShow-Basisklassen

Die DirectShow-Basisklassen sind eine Reihe von C++-Klassen und Hilfsfunktionen, die für die Implementierung von DirectShow-Filtern entwickelt wurden. Einige der Hilfsklassen sind auch für Anwendungsentwickler nützlich.

In diesem Abschnitt werden die Basisklassen beschrieben. Sie enthält die folgenden Abschnitte:
