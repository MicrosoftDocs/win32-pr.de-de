# <a name="dmo-functions"></a>DMO-Funktionen

In diesem Abschnitt werden die Funktionen von Microsoft DirectX Media Object (DMO) beschrieben.



| Funktion                                             | BESCHREIBUNG                                                   |
|------------------------------------------------------|---------------------------------------------------------------|
| [**Dmuaufumum**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmoenum)                           | Listet DMOS auf, die in der Registrierung aufgeführt sind.                       |
| [**Dmugetname**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmogetname)                     | Ruft den Namen eines DMO aus der Registrierung ab.                |
| [**Dmuttypes**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmogettypes)                   | Ruft die für einen DMO registrierten Medientypen ab.               |
| [**Dmoregiester**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmoregister)                   | Registriert einen DMO.                                              |
| [**Dmounregister**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmounregister)               | Hebt die Registrierung eines DMO auf.                                            |
| [**"Mucopymediatype"**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mocopymediatype)           | Kopiert eine Medientyp Struktur in eine andere.                   |
| [**"" Von "".**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mocreatemediatype)       | Ordnet eine neue Medientyp Struktur zu.                         |

| [**Moduplicatemediatype**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-moduplicatemediatype) | Dupliziert eine Medientyp Struktur.                            |
| [**"Mufremediatype"**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mofreemediatype)           | Gibt die zugeordneten Member einer Medientyp Struktur frei.        |
| [**"Muitmediatype"**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-moinitmediatype)           | Initialisiert eine Medientyp Struktur.                           |
