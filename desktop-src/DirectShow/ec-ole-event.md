# <a name="ec_ole_event"></a>EC \_ OLE \_ EVENT

Ein Filter übergibt eine Textzeichenfolge an die Anwendung.

## <a name="parameters"></a>Parameter

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

(**BSTR**) Zeichenfolge vom Typ **BSTR**.

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

(**BSTR**) Zeichenfolge vom Typ **BSTR**.

</dd> </dl>

## <a name="default-action"></a>Standardaktion

Keine.

## <a name="remarks"></a>Hinweise

Standardmäßig enthält der erste Parameter Typinformationen (z. B. "Text"), und der zweite Parameter enthält die Textzeichenfolge.
