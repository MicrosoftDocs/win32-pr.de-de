# <a name="avencddatodconvertertype-property"></a>AVEncDDAtoDConverterType (Eigenschaft)

Gibt den Typ der Analog-Digital-Konvertierung (A/D) für einen Dolby Digital-Audiostream an. Diese Eigenschaft gilt für Dolby Digital-Audioencoder.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncDDAtoDConverterType**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eAVEncDDAtoDConverterType-Enumeration.**](/windows/win32/api/codecapi/ne-codecapi-eavencddatodconvertertype)
