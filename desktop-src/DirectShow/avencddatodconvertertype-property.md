# <a name="avencddatodconvertertype-property"></a>Avencddatodconvertertype (Eigenschaft)

Gibt den Typ der analog-zu-Digital-Konvertierung (a/D) für einen Dolby Digital-Audiodatenstrom an. Diese Eigenschaft gilt für Dolby Digital-Audioencoder.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencddatodconvertertype**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eavencddatodconvertertype**](/windows/win32/api/codecapi/ne-codecapi-eavencddatodconvertertype) -Enumeration.
