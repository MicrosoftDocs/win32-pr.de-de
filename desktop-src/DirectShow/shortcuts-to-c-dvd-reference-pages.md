# <a name="shortcuts-to-c-dvd-reference-pages"></a>Verknüpfungen zu C++-DVD-Referenzseiten

Die folgenden Schnittstellen werden normalerweise beim Schreiben einer DVD-Anwendung in C++ verwendet.



| Schnittstelle                                    | Zweck                                                                                          |
|----------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**IAMLine21Decoder**](/previous-versions/windows/desktop/api/il21dec/nn-il21dec-iamline21decoder) | Steuert die Anzeige von Untertiteln.                                                              |
| [**IDvdCmd**](/windows/desktop/api/Strmif/nn-strmif-idvdcmd)                   | Wird in erweiterten Befehlssynchronisierungstechniken verwendet, um ein Befehlsobjekt zu steuern.                 |
| [**IDvdControl2**](/windows/desktop/api/Strmif/nn-strmif-idvdcontrol2)         | Gibt Befehle für den [DVD Navigator-Filter](dvd-navigator-filter.md) aus (die "set"-Methoden).     |
| [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) | Erstellt das DVD-Filterdiagramm.                                                                    |
| [**IDvdInfo2**](/windows/desktop/api/Strmif/nn-strmif-idvdinfo2)               | Fragt den DVD-Navigator nach Zustandsinformationen und Informationen auf dem Datenträger ab (die "get"-Methoden). |
| [**IDvdState**](/windows/desktop/api/Strmif/nn-strmif-idvdstate)               | Speichert Informationen, z. B. Lesezeichen, auf einem Datenträger.                                                   |
| [**IVideoFrameStep**](/windows/desktop/api/Strmif/nn-strmif-ivideoframestep)   | Steuert die "Frameschrittung", wenn der Decoder dies unterstützt.                                            |
