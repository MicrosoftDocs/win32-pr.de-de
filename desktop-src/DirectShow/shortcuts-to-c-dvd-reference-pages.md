# <a name="shortcuts-to-c-dvd-reference-pages"></a>Verknüpfungen zu C++-DVD-Referenzseiten

Die folgenden Schnittstellen sind diejenigen, die normalerweise beim Schreiben einer DVD-Anwendung in C++ verwendet werden.



| Schnittstelle                                    | Zweck                                                                                          |
|----------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**IAMLine21Decoder**](/previous-versions/windows/desktop/api/il21dec/nn-il21dec-iamline21decoder) | Steuert die Untertitelanzeige.                                                              |
| [**Idvdcmd**](/windows/desktop/api/Strmif/nn-strmif-idvdcmd)                   | Wird in erweiterten Befehls Synchronisierungs Verfahren zum Steuern eines Befehls Objekts verwendet.                 |
| [**IDvdControl2**](/windows/desktop/api/Strmif/nn-strmif-idvdcontrol2)         | Gibt Befehle an den [DVD-navigatorfilter](dvd-navigator-filter.md) aus (die "Set"-Methoden).     |
| [**Idvdgraphbuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) | Erstellt das DVD-Filter Diagramm.                                                                    |
| [**IDvdInfo2**](/windows/desktop/api/Strmif/nn-strmif-idvdinfo2)               | Fragt den DVD-Navigator nach Zustandsinformationen und Informationen auf dem Datenträger (die "Get"-Methoden) ab. |
| [**Idvdstate**](/windows/desktop/api/Strmif/nn-strmif-idvdstate)               | Speichert Informationen, z. b. Lesezeichen, auf der Festplatte.                                                   |
| [**Ivideoframestep**](/windows/desktop/api/Strmif/nn-strmif-ivideoframestep)   | Steuert "Frame Step", wenn der Decoder dies unterstützt.                                            |
