# <a name="oahwnd"></a>Oahwnd

Der- `OAHWND` Datentyp definiert ein Fenster Handle, das von OLE-Automatisierungs Clients verwendet werden kann, wie z. b. Microsoft Visual Basic 6,0.


```C++
typedef LONG_PTR OAHWND;
```
