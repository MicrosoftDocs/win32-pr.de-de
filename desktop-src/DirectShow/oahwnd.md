# <a name="oahwnd"></a>OAHWND

Der `OAHWND` Datentyp definiert ein Fensterhandle, das von OLE-Automatisierungsclients wie Microsoft Visual Basic 6.0 verwendet werden kann.


```C++
typedef LONG_PTR OAHWND;
```
