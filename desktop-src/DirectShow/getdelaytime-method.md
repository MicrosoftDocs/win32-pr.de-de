# <a name="getdelaytime-method"></a>Getdelta-Time-Methode

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `GetDelayTime` Methode ruft die Verzögerungszeit für die QuickInfo ab, die dem **mswebdvd** -Objekt zugeordnet ist.

``` syntax
[ iDelay = ] MSWebDVD.GetDelayTime()
```

## <a name="return-value"></a>Rückgabewert

Gibt einen Integer-Wert zurück, der die QuickInfo-Verzögerungszeit in Millisekunden angibt.
