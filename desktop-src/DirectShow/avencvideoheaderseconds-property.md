# <a name="avencvideoheaderseconds-property"></a>Avencvideoheaderseconds (Eigenschaft)

Gibt die Anfangs zweite Zahl im GOP-Header (Group of Pictures) an.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideoheaderseconds**

## <a name="property-value"></a>Eigenschaftswert

Der gültige Bereich dieser Eigenschaft liegt zwischen 0 und 59 (einschließlich).
