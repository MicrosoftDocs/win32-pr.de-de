# <a name="avencvideooutputscantype-property"></a>Avencvideooutputscantype (Eigenschaft)

Gibt an, wie der Encoder das Ausgabevideo interziert.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideooutputscantype**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eavencvideooutputscantype**](/windows/win32/api/codecapi/ne-codecapi-eavencvideooutputscantype) -Enumeration.
