# <a name="cdynamicoutputpinstopusingoutputpin-method"></a>CDynamicOutputPin.StopUsingOutputPin-Methode

Die `StopUsingOutputPin` -Methode gibt den Zugriff auf den Pin nach einem Streamingvorgang frei.

## <a name="syntax"></a>Syntax


```C++
virtual void StopUsingOutputPin();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.
