# <a name="ec_paused"></a>EC \_ angehalten

Eine Pause-Anforderung wurde abgeschlossen.

## <a name="parameters"></a>Parameter

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

(**HRESULT**) Status des Anhaltevorgangs.

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

Keinen.

</dd> </dl>

## <a name="default-action"></a>Standardaktion

Keine.

## <a name="remarks"></a>Bemerkungen

Der Filter Graph-Manager sendet dieses Ereignis, wenn ein asynchroner Pause-Befehl abgeschlossen ist.
