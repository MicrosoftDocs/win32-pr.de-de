# <a name="ctransinplacefilteroutputpin-method"></a>Ctransinplacefilter. outputpin-Methode

Die `OutputPin` -Methode ruft einen Zeiger auf den Ausgabepin des Filters ab.

## <a name="syntax"></a>Syntax


```C++
CTransInPlaceOutputPin* OutputPin();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt die [**ctransformfilter:: m \_ poutput**](ctransformfilter-m-poutput.md) -Member-Variable zurück.
