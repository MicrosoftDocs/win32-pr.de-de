# <a name="lock-attribute"></a>Lock-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das- `lock` Attribut gibt an, ob ein Objekt bearbeitet werden soll. Wenn der Wert **true** ist, sollte die Anwendung das Objekt als gesperrt behandeln und Änderungen an diesem Objekt oder seinen untergeordneten Elementen nicht zulassen. Der Standardwert ist **FALSE**.

## <a name="possible-values"></a>Mögliche Werte

Mögliche Werte die folgenden Werte sind als true definiert: y, y, t, t, 1. Die folgenden Werte sind als false definiert: n, n, f, f, 0 (null).
