# <a name="ctransinplacefilterinputpin-method"></a>Ctransinplacefilter. InputPin-Methode

Die `InputPin` -Methode ruft einen Zeiger auf die Eingabe-PIN des Filters ab.

## <a name="syntax"></a>Syntax


```C++
CTransInPlaceInputPin* InputPin();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt die [**ctransformfilter:: m \_ pinput**](ctransformfilter-m-pinput.md) -Member-Variable zurück.
