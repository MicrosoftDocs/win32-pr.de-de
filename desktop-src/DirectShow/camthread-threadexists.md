# <a name="camthreadthreadexists-method"></a>CAMThread.ThreadExists-Methode

Die `ThreadExists` -Methode fragt ab, ob der Thread vorhanden ist.

## <a name="syntax"></a>Syntax


```C++
BOOL ThreadExists();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt **TRUE zurück,** wenn der Thread vorhanden ist, oder **FALSE,** wenn der Thread nicht vorhanden ist.
