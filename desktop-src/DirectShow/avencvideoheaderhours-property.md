# <a name="avencvideoheaderhours-property"></a>Avencvideoheaderhours (Eigenschaft)

Gibt die anfangsstunden Nummer im GOP-Header (Group of Pictures) an.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideoheaderhours**

## <a name="property-value"></a>Eigenschaftswert

Der gültige Bereich dieser Eigenschaft ist 0 bis 23 (einschließlich).
