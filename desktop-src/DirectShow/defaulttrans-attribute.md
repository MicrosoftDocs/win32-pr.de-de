# <a name="defaulttrans-attribute"></a>defaulttrans-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das- `defaulttrans` Attribut gibt den Klassen Bezeichner (CLSID) eines Standard Übergangs für die Zeitachse an.

## <a name="possible-values"></a>Mögliche Werte

Muss eine Zeichenfolge mit dem-Format sein {00000000-0000-0000-0000-000000000000} .

## <a name="applies-to"></a>Gilt für

[**Zeitplan**](timeline-element.md)
