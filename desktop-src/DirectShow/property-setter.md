# <a name="property-setter"></a>Eigenschaften Setter

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das Eigenschaften Setter-Objekt legt **IDispatch** -Eigenschaften für ein Timeline-Objekt fest. Um dieses Objekt zu erstellen, rufen Sie **cokreateinstance** auf. Der Klassen Bezeichner ist CLSID \_ propertysetter.

Das Eigenschaften Setter-Objekt macht die folgenden Schnittstellen verfügbar:

-   [**Iameinterrorlog**](iamseterrorlog.md)
-   [**Ipropertysetter**](ipropertysetter.md)
