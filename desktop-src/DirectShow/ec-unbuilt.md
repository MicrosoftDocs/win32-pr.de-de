# <a name="ec_unbuilt"></a>EC \_ UNBUILT

Das EC \_ UNBUILT-Ereignis wird vom Videosteuerelement gesendet, wenn ein Diagramm abreißt wurde. Dieses Ereignis wird nicht an Anwendungen weitergeleitet.
