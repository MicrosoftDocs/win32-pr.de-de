# <a name="csourceposition-class"></a>Csourceposition-Klasse

`CSourcePosition` ist eine abstrakte Klasse zum Implementieren der [**imediaposition**](/windows/desktop/api/Control/nn-control-imediaposition) -Schnittstelle für einen Quell Filter.

Diese Klasse ist veraltet. Wenn der Quell Filter suchbar ist, sollte er anstelle von **imediaposition** die [**imediaseeking**](/windows/desktop/api/Strmif/nn-strmif-imediaseeking) -Schnittstelle verfügbar machen. Hierfür können Sie die [**csourceseeking**](csourceseeking.md) -Basisklasse verwenden.
