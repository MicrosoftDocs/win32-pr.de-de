<span data-ttu-id="397af-187">**Domänen**</span><span class="sxs-lookup"><span data-stu-id="397af-187">**Domains**</span></span>

<span data-ttu-id="397af-205">Benutzer Vorgangs Steuerelemente (User Operation Controls, UOPs) sind Marker auf einer CD, die von DVD-Autoren an beliebiger Stelle eingefügt werden können, um die
