# <a name="cbaserendererisstreaming-method"></a>Cbaserderderer. isstreaming-Methode

Die- `IsStreaming` Methode fragt ab, ob der Filter Streamingdaten ist.

## <a name="syntax"></a>Syntax


```C++
BOOL IsStreaming();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt das [**\_ bstreaming-Flag "cbaserenderer:: m**](cbaserenderer-m-bstreaming.md) " zurück.
