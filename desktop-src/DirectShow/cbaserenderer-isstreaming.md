# <a name="cbaserendererisstreaming-method"></a>CBaseRenderer.IsStreaming-Methode

Die `IsStreaming` -Methode fragt ab, ob der Filter Daten streamt.

## <a name="syntax"></a>Syntax


```C++
BOOL IsStreaming();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt das [**\_ BStreamingflag CBaseRenderer::m**](cbaserenderer-m-bstreaming.md) zurück.
