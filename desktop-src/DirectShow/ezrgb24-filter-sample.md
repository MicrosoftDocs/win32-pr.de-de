# <a name="ezrgb24-filter-sample"></a>EZRGB24 Filter-Beispiel

## <a name="description"></a>BESCHREIBUNG

Der EZRGB24 Filter ist ein Transformations Filter für die Bildverarbeitung. Dieser Filter führt eine Reihe von einzelnen Video Effekten aus, einschließlich rot, grün, blau, Darken, Xor, weich, grau und Prägung-Effekten.

In diesem Beispiel wird gezeigt, wie mehrere DirectShow-Basisklassen verwendet werden, einschließlich [**ctransformfilter**](ctransformfilter.md), [**cpersiststream**](cpersiststream.md)und [**cbasepropertypage**](cbasepropertypage.md).

## <a name="downloading-the-sample"></a>Herunterladen des Beispiels

Zum Herunterladen der DirectShow SDK-Beispiele installieren Sie die neueste Version der [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Dieses Beispiel wird unter folgendem Pfad installiert: *\[ SDK \] root* \\ Samples \\ Multimedia \\ DirectShow \\ Filters \\ EZRGB24.
