# <a name="ezrgb24-filter-sample"></a>EZRGB24-Filterbeispiel

## <a name="description"></a>Beschreibung

Der EZRGB24-Filter ist ein Bildverarbeitungstransformationsfilter. Dieser Filter führt eine Reihe einzelner Videoeffekte aus, einschließlich Rot-, Grün-, Blau-, Dunkel-, XOR-, Weichzeichner-, Grau- und Betteffekten.

In diesem Beispiel wird die Verwendung mehrerer DirectShow-Basisklassen veranschaulicht, einschließlich [**CTransformFilter,**](ctransformfilter.md) [**CPersistStream**](cpersiststream.md)und [**CBasePropertyPage.**](cbasepropertypage.md)

## <a name="downloading-the-sample"></a>Herunterladen des Beispiels

Um die DirectShow SDK-Beispiele herunterzuladen, installieren Sie die neueste Version des [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Dieses Beispiel wird unter folgendem Pfad installiert: *\[ \] SDK-Stammbeispiele* \\ Multimedia \\ \\ DirectShow Filters \\ \\ EZRGB24.
