# <a name="cbaserenderergetrenderevent-method"></a>Cbaserderderer. GetRenderEvent-Methode

Die- `GetRenderEvent` Methode ruft das Ereignis ab, das das Rendering plant.

## <a name="syntax"></a>Syntax


```C++
CAMEvent* GetRenderEvent();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt einen Zeiger auf das [**cbaserenderer:: m \_ renderevent**](cbaserenderer-m-renderevent.md) -Ereignis zurück.
