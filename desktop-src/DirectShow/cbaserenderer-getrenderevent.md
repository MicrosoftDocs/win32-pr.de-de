# <a name="cbaserenderergetrenderevent-method"></a>CBaseRenderer.GetRenderEvent-Methode

Die `GetRenderEvent` -Methode ruft das Ereignis ab, das das Rendering geplant.

## <a name="syntax"></a>Syntax


```C++
CAMEvent* GetRenderEvent();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt einen Zeiger auf das [**\_ RenderEvent-Ereignis CBaseRenderer::m**](cbaserenderer-m-renderevent.md) zurück.
