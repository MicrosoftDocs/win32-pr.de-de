## <a name="remarks"></a><span data-ttu-id="56dd7-132">Bemerkungen</span><span class="sxs-lookup"><span data-stu-id="56dd7-132">Remarks</span></span>

<span data-ttu-id="56dd7-133">Im Kontext dieses Eigenschaftssatzes misst rate die Rate, mit der Zeitstempel relativ zur Referenzuhr vorn sind.</span><span class="sxs-lookup"><span data-stu-id="56dd7-133">In the context of this property set, rate measures the rate at which time stamps advance relative to the reference clock.</span></span> <span data-ttu-id="56dd7-134">Bewerten Sie die Umkehrung der Wiedergabegeschwindigkeit.</span><span class="sxs-lookup"><span data-stu-id="56dd7-134">Rate the inverse of playback speed.</span></span> <span data-ttu-id="56dd7-135">Wenn die Wiedergabegeschwindigkeit beispielsweise 2x beträgt, müssen die Zeitstempel mit 1/2 der normalen Rate erhöht werden.</span><span class="sxs-lookup"><span data-stu-id="56dd7-135">For example, if the playback speed is 2x, the time stamps must increase at 1/2 the normal rate.</span></span> <span data-ttu-id="56dd7-136">Dies führt zu einer schnelleren Wiedergabegeschwindigkeit, da Beispiele früher als normal gerendert werden.</span><span class="sxs-lookup"><span data-stu-id="56dd7-136">This translates to a faster playback speed, because samples are rendered earlier than normal.</span></span>
