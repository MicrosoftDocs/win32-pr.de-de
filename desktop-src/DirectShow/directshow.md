# <a name="directshow"></a>DirectShow

Die Microsoft DirectShow-API (Application Programming Interface) ist eine Medien Streaming-Architektur für Microsoft Windows. Mithilfe von DirectShow können Ihre Anwendungen eine qualitativ hochwertige Video-und Audiowiedergabe oder-Erfassung ausführen.

Die DirectShow-Header,-Bibliotheken,-SDK-Tools und-Beispiele sind im [Windows SDK](https://msdn.microsoft.com/windows/aa904949.aspx)verfügbar.

> [!Note]  
> Frühere Versionen des DirectShow SDK waren im DirectX SDK enthalten. Die letzte Version des DirectX SDK zum Einbeziehen von DirectShow war DirectX 9,0 SDK Update-(Februar 2005) Extras. Nach dieser Version wurde DirectShow in den Windows SDK verschoben. Um die neueste Version der DirectShow-Header,-Bibliotheken und-Beispiele zu erhalten, laden Sie die Windows SDK herunter.
