# <a name="coutputqueuem_heventpop-member"></a>Coutputqueue:: m \_ heventpop-Member

Optionales Ereignis, das signalisiert wird, wenn das Objekt ein Beispiel aus der Warteschlange entfernt. Der Wert ist anfänglich **null**. Aufrufen der [**coutputqueue:: setpopevent**](coutputqueue-setpopevent.md) -Methode, um ein Ereignis handle anzugeben.

## <a name="syntax"></a>Syntax


```C++
HANDLE m_hEventPop;
```
