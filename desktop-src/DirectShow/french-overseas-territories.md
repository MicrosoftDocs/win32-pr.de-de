# <a name="french-overseas-territories"></a>Französische Überseegebiete

Eine Erläuterung der folgenden Tabellen finden Sie unter [Tabellen und Zuweisungen](tables-and-assignments.md).

Die Häufigkeits Indizes in der französischen Broadcast Tabelle Übersee sind für die Verwendung als Kanalnummern geeignet. die Indizes für die Kabel Frequenz können nicht als Kanalnummern verwendet werden.

### <a name="french-overseas-territories-broadcast-assignments"></a>Französische Übertragungs Zuweisungen im Überseegebiet


```C++
         4L,            // Lowest channel
         9L,            // Highest channel

       175250000L,      // 4
       183250000L,      // 5
       191250000L,      // 6
       199250000L,      // 7
       207250000L,      // 8
       215250000L,      // 9
```



### <a name="french-overseas-territories-cable-assignments"></a>Französische Kabel Zuweisungen im Überseegebiet


```C++
         4L,            // Lowest channel
         9L,            // Highest channel

       175250000L,      // 4
       183250000L,      // 5
       191250000L,      // 6
       199250000L,      // 7
       207250000L,      // 8
       215250000L,      // 9
```
