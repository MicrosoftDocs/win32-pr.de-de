# <a name="currentdiscside-property"></a>CurrentDiscSide-Eigenschaft

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die `CurrentDiscSide` -Eigenschaft ruft die aktuelle Seite des Datenträgers ab.

``` syntax
[ iCurSide = ] MSWebDVD.CurrentDiscSide
```

## <a name="return-value"></a>Rückgabewert

Gibt einen ganzzahligen Wert zurück, der die aktuelle Seite darstellt. Seite A = 1 und Seite B = 2.

## <a name="remarks"></a>Hinweise

Diese Eigenschaft ist schreibgeschützt und hat keinen Standardwert.
