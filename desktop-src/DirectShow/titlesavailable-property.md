# <a name="titlesavailable-property"></a>Titlesavailable (Eigenschaft)

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `TitlesAvailable` Eigenschaft ruft die Anzahl der auf der Festplatte verfügbaren Titel ab.

``` syntax
[ iTitles = ] MSWebDVD.TitlesAvailable
```

## <a name="return-value"></a>Rückgabewert

Gibt die Anzahl der verfügbaren Titel als ganzzahligen Wert zwischen 1 und 99 zurück.

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist schreibgeschützt und weist keinen Standardwert auf.
