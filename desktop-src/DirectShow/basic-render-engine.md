# <a name="basic-render-engine"></a>Einfache Renderingengine

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das grundlegende Rendering-Engine-Objekt rendert eine nicht komprimierte Ausgabe aus einer Zeita Um dieses Objekt zu erstellen, rufen Sie **cokreateinstance** auf. Verwenden Sie für die komprimierte Ausgabe die Smart-Rendering-Engine. Der Klassen Bezeichner ist CLSID- \_ Renderengine.

Die einfache Renderingengine macht die folgenden Schnittstellen verfügbar:

-   [**Iameinterrorlog**](iamseterrorlog.md)
-   IObjectWithSite
-   [**"Unenderengine"**](irenderengine.md)
-   [**IRenderEngine2**](irenderengine2.md)
