# <a name="external-device-control-interfaces"></a>Externe Geräte Steuerungs Schnittstellen

Diese Schnittstellen unterstützen die Anwendungs Kontrolle über externe Geräte, z. b. DV-Camcorder.



| Schnittstelle                                            | BESCHREIBUNG                                                                             |
|------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [**IAMCameraControl**](/windows/desktop/api/Strmif/nn-strmif-iamcameracontrol)         | Steuern einer Kamera.                                                                       |
| [**IAMExtDevice**](/windows/desktop/api/Strmif/nn-strmif-iamextdevice)                 | Steuern Sie ein externes Gerät, z. b. einen VCR oder einen Camcorder.                                  |
| [**IAMExtTransport**](/windows/desktop/api/Strmif/nn-strmif-iamexttransport)           | Steuern Sie VCR-Transportfunktionen wie "Play", "Pause", "Record", "Fast Forward" und "rewind". |
| [**Iamtimecodedisplay**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodedisplay)     | Festlegen von Eigenschaften auf einem Gerät, das SMPTE-Timecode anzeigt.                                |
| [**Iamtimecodegenerator**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodegenerator) | Festlegen von Eigenschaften auf einem Gerät, das SMPTE-Timecode generiert.                               |
| [**IAMTimecodeReader**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodereader)       | Lesen von SMPTE-Zeitcode von einem externen Gerät.                                            |
