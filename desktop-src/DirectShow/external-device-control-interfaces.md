# <a name="external-device-control-interfaces"></a>Externe Gerätesteuerungsschnittstellen

Diese Schnittstellen unterstützen die Anwendungssteuerung über externe Geräte, z. B. DV-Dvds.



| Schnittstelle                                            | Beschreibung                                                                             |
|------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [**IAMCameraControl**](/windows/desktop/api/Strmif/nn-strmif-iamcameracontrol)         | Steuern einer Kamera.                                                                       |
| [**IAMExtDevice**](/windows/desktop/api/Strmif/nn-strmif-iamextdevice)                 | Steuern sie ein externes Gerät, z. B. einen VCR oder Einser.                                  |
| [**IAMExtTransport**](/windows/desktop/api/Strmif/nn-strmif-iamexttransport)           | Steuern Sie VCR-Transportfunktionen wie Wiedergabe, Pause, Aufzeichnung, Schnelles Vorwärts und Zurückspulen. |
| [**IAMTimecodeDisplay**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodedisplay)     | Legen Sie Eigenschaften auf einem Gerät fest, das den SMPTE-Zeitcode anzeigt.                                |
| [**IAMTimecodeGenerator**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodegenerator) | Legen Sie Eigenschaften auf einem Gerät fest, das einen SMPTE-Zeitcode generiert.                               |
| [**IAMTimecodeReader**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodereader)       | Lesen des SMPTE-Zeitcodes von einem externen Gerät                                            |
