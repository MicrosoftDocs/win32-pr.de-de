# <a name="filter-mapper"></a>Filterzuordnung

Die Filterzuordnung durchsucht die Registrierung nach registrierten Filtern. Der Filter Graph Manager verwendet diese Komponente, um Filterdiagramme zu erstellen. Anwendungen können damit Filter suchen, die den angegebenen Suchkriterien entsprechen. Erstellen Sie dieses Objekt, indem **Sie CoCreateInstance** aufrufen.
