# <a name="avencvideooutputcolorlighting-property"></a>Avencvideooutputcolorlighting (Eigenschaft)

Gibt die vorgesehenen Beleuchtungsbedingungen zum Anzeigen des codierten Videos an.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideooutputcolorbeleuchtung**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eavencvideocolorlighting**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolorlighting) -Enumeration.
