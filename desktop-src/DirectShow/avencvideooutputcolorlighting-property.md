# <a name="avencvideooutputcolorlighting-property"></a>AVEncVideoOutputColorLighting-Eigenschaft

Gibt die beabsichtigten Beleuchtungsbedingungen zum Anzeigen des codierten Videos an.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncVideoOutputColorLighting**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eAVEncVideoColorLighting-Enumeration.**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolorlighting)
