# <a name="capture-method"></a>Capture-Methode

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `Capture` Methode erfasst ein Image aus dem Videoframe, wenn sich das mswebdvd-Objekt im fensterlosen Modus befindet.

``` syntax
MSWebDVD.Capture()
```

## <a name="remarks"></a>Bemerkungen

Diese Methode erfasst den aktuellen Frame aus dem DVD-Video Bild und fügt ihn in ein Fenster ein, aus dem der Benutzer das Bild speichern oder bearbeiten kann. Das mswebdvd-Objekt muss sich im fensterlosen Modus befinden, damit diese Methode erfolgreich ausgeführt werden konnte.
