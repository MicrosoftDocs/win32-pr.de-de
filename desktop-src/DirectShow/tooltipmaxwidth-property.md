# <a name="tooltipmaxwidth-property"></a>ToolTipMaxWidth-Eigenschaft

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die `ToolTipMaxWidth` -Eigenschaft legt die maximale Breite für die QuickInfo fest, die dem **MSWebDVD-Objekt** zugeordnet ist, oder ruft sie ab.

``` syntax
[ iMaxWidth = ] MSWebDVD.ToolTipMaxWidth
```

## <a name="return-value"></a>Rückgabewert

Gibt die maximale Breite für die QuickInfo der Anwendung als ganze Zahl zurück.

## <a name="remarks"></a>Hinweise

Diese Eigenschaft wird mit einem Standardwert von 200 Pixeln gelesen/geschrieben.
