# <a name="tooltipmaxwidth-property"></a>Tooltipmaxwidth (Eigenschaft)

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `ToolTipMaxWidth` Eigenschaft legt die maximale Breite für die QuickInfo fest, die dem **mswebdvd** -Objekt zugeordnet ist, oder ruft diese ab.

``` syntax
[ iMaxWidth = ] MSWebDVD.ToolTipMaxWidth
```

## <a name="return-value"></a>Rückgabewert

Gibt die maximale Breite für die QuickInfo der Anwendung als Ganzzahl zurück.

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist Lese-/Schreibzugriff mit einem Standardwert von 200 Pixel.
