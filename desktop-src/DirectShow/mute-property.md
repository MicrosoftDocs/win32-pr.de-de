# <a name="mute-property"></a>Stumm Eigenschaft

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die `Mute` -Eigenschaft schaltet die audiostreamausgabe ein bzw. aus.

``` syntax
[ bMute = ] MSWebDVD.Mute
```

## <a name="return-value"></a>Rückgabewert

Gibt einen booleschen Wert zurück, der angibt, ob der Audiodatenstrom stumm geschaltet werden soll.

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist Lese-/Schreibzugriff und hat den Standardwert false.
