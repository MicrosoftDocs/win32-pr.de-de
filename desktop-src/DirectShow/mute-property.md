# <a name="mute-property"></a>Mute-Eigenschaft

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die -Eigenschaft aktiviert oder deaktiviert `Mute` die Ausgabe des Audiostreams.

``` syntax
[ bMute = ] MSWebDVD.Mute
```

## <a name="return-value"></a>Rückgabewert

Gibt einen booleschen Wert zurück, der angibt, ob der Audiostream stummgeschaltet werden soll.

## <a name="remarks"></a>Hinweise

Diese Eigenschaft ist mit dem Standardwert false gelesen/geschrieben.
