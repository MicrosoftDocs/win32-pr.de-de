# <a name="avencaudiomapdestchanneln-property"></a>Avencaudiomapdestchanneleln (Eigenschaft)

Gibt an, welcher Audiokanal Channel *N* im codierten Audiostream zugeordnet ist.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Siehe Hinweise.**

## <a name="remarks"></a>Bemerkungen

In diesem Referenz Thema wird ein Satz verwandter Eigenschaften beschrieben. Jede Eigenschaft ordnet einem angegebenen Ausgabekanal einen Eingabe Kanal zu. Die folgenden Eigenschaften sind definiert.



| Eigenschaften-GUID                            | BESCHREIBUNG             |
|------------------------------------------|-------------------------|
| **Codecapi \_ AVEncAudioMapDestChannel0**  | Zielchannel 0.  |
| **Codecapi \_ AVEncAudioMapDestChannel1**  | Ziel Kanal 1.  |
| **Codecapi \_ AVEncAudioMapDestChannel2**  | Zielchannel 2.  |
| **Codecapi \_ AVEncAudioMapDestChannel3**  | Ziel Kanal 3.  |
| **Codecapi \_ AVEncAudioMapDestChannel4**  | Zielchannel 4.  |
| **Codecapi \_ AVEncAudioMapDestChannel5**  | Zielchannel 5.  |
| **Codecapi \_ AVEncAudioMapDestChannel6**  | Ziel Kanal 6.  |
| **Codecapi \_ AVEncAudioMapDestChannel7**  | Ziel Kanal 7.  |
| **Codecapi \_ AVEncAudioMapDestChannel8**  | Zielchannel 8.  |
| **Codecapi \_ AVEncAudioMapDestChannel9**  | Zielchannel 9.  |
| **Codecapi \_ AVEncAudioMapDestChannel10** | Zielchannel 10. |
| **Codecapi \_ AVEncAudioMapDestChannel11** | Ziel Kanal 11. |
