# <a name="getdvdtextnumberoflanguages-method"></a>GetDVDTextNumberOfLanguages-Methode

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die `GetDVDTextNumberOfLanguages` -Methode ruft die Anzahl der auf dem Datenträger verfügbaren Textsprachen ab.

``` syntax
[ iLanguages = ] MSWebDVD.GetDVDTextNumberOfLanguages()
```

## <a name="return-value"></a>Rückgabewert

Gibt einen ganzzahligen Wert zurück, der die Anzahl der verfügbaren Textsprachen angibt.
