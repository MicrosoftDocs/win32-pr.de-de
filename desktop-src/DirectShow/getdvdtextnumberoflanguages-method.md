# <a name="getdvdtextnumberoflanguages-method"></a>Getdvdtextnumoflanguages-Methode

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `GetDVDTextNumberOfLanguages` Methode ruft die Anzahl der auf der Festplatte verfügbaren Text Sprachen ab.

``` syntax
[ iLanguages = ] MSWebDVD.GetDVDTextNumberOfLanguages()
```

## <a name="return-value"></a>Rückgabewert

Gibt einen ganzzahligen Wert zurück, der die Anzahl der verfügbaren Text Sprachen angibt.
