# <a name="cgenericlistaddtail-method-wxlisth---plist-parameter"></a>Cgenericlist. AddTail-Methode (wxlist. h)-plist-Parameter

Die- `AddTail` Methode fügt eine Liste an das Ende der Liste an.

## <a name="syntax"></a>Syntax


```C++
BOOL AddTail(
   CGenericList<OBJECT> pList
);
```



## <a name="parameters"></a>Parameter

<dl> <dt>

*pList* 
</dt> <dd>

Ein Zeiger auf die anzufügende Liste.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt **true** zurück, wenn erfolgreich, andernfalls **false** .
