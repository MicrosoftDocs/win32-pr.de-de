# <a name="ccactive-property"></a>CCActive-Eigenschaft

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die `CCActive` -Eigenschaft legt den aktuellen Status der Untertitelung fest oder ruft sie ab.

``` syntax
[ bCCActive = ] MSWebDVD.CCActive
```

## <a name="return-value"></a>Rückgabewert

Gibt einen booleschen Wert zurück, der angibt, ob Untertitel aktiviert sind.

## <a name="remarks"></a>Hinweise

Diese Eigenschaft ist lese-/schreibgeschützt und hat den Standardwert false.
