# <a name="ccactive-property"></a>Ccactive-Eigenschaft

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `CCActive` Eigenschaft legt den aktuellen Status von Untertiteln fest oder ruft ihn ab.

``` syntax
[ bCCActive = ] MSWebDVD.CCActive
```

## <a name="return-value"></a>Rückgabewert

Gibt einen booleschen Wert zurück, der angibt, ob Untertitel aktiviert ist.

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist Lese-/Schreibzugriff und hat den Standardwert false.
