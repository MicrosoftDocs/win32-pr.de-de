# <a name="width-property-directshow"></a>Width-Eigenschaft (DirectShow)

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `Width` Eigenschaft legt die Breite des Rechtecks fest oder ruft diese ab.

``` syntax
[ iWidth = ] DVDRect.Width
```

## <a name="return-value"></a>Rückgabewert

Gibt die Breite des Rechtecks in Pixel als Ganzzahl zurück.

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist Lese-/Schreibzugriff und hat keinen Standardwert.
