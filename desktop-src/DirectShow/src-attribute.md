# <a name="src-attribute"></a>src-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das- `src` Attribut gibt den Pfadnamen einer Mediendatei an. Bei der Datei kann es sich um jeden von DirectShow unterstützten Typ oder um ein Image handeln. Die unterstützten Formate für weiterhin Images sind Bitmap, geräteunabhängige Bitmap, GIF und JPEG.

## <a name="possible-values"></a>Mögliche Werte

Muss eine Zeichenfolge sein, die einen Pfadnamen angibt.
