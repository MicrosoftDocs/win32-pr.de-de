# <a name="src-attribute"></a>src-Attribut

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Releases von Windows entfernt werden.\]

 

Das `src` -Attribut gibt den Pfadnamen einer Mediendatei an. Die Datei kann ein beliebiger Typ sein, der von DirectShow unterstützt wird, oder ein Standbild. Die unterstützten Formate für Standbilder sind Bitmap, geräteunabhängige Bitmap, GIF und JPEG.

## <a name="possible-values"></a>Mögliche Werte

Muss eine Zeichenfolge sein, die einen Pfadnamen angibt.
