> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Nicht implementiert.

## <a name="syntax"></a>Syntax


```C++
HRESULT CopyXML(
   IUnknown *pTimeline,
   double   dStart,
   double   dEnd
);
```
