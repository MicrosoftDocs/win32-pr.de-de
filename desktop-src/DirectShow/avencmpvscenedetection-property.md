# <a name="avencmpvscenedetection-property"></a>Avencmpvsceneerkennungs (Eigenschaft)

Gibt an, wie sich der Encoder verhält, wenn er eine neue Szene erkennt. Diese Eigenschaft gilt für MPEG-Video Encoder.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencmpvsceneerkennungs**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eavencmpvsceneerkennungs**](/windows/win32/api/codecapi/ne-codecapi-eavencmpvscenedetection) -Enumeration.
