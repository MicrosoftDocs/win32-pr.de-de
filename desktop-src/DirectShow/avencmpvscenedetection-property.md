# <a name="avencmpvscenedetection-property"></a>AVEncMPVSceneDetection (Eigenschaft)

Gibt an, wie sich der Encoder verhält, wenn er eine neue Szene erkennt. Diese Eigenschaft gilt für MPEG-Videoencoder.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncMPVSceneDetection**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eAVEncMPVSceneDetection-Enumeration.**](/windows/win32/api/codecapi/ne-codecapi-eavencmpvscenedetection)
