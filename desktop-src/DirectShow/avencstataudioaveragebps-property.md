# <a name="avencstataudioaveragebps-property"></a>Avencstatuaudioaveragebps (Eigenschaft)

Gibt die durchschnittlichen Bits pro Sekunde der codierten Audiodaten zurück.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencstatus audioaveragebps**

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist nach Abschluss der Codierung verfügbar.

Diese Eigenschaft gilt nur für die Qualitäts basierte Variable Bitrate (VBR)-Codierung.
