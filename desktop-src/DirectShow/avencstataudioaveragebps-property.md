# <a name="avencstataudioaveragebps-property"></a>AVEncStatAudioAverageBPS-Eigenschaft

Gibt die durchschnittlichen Bits pro Sekunde des codierten Audios zurück.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncStatAudioAverageBPS**

## <a name="remarks"></a>Hinweise

Diese Eigenschaft ist verfügbar, nachdem die Codierung abgeschlossen wurde.

Diese Eigenschaft gilt nur für die qualitätsbasierte VBR-Codierung (Variable Bit Rate).
