# <a name="selectoractivatbutton"></a>Selectoractivatbutton

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Das `SelectOrActivatButton` Ereignis wird gesendet, wenn der Disc die Auswahl oder Aktivierung von Menü Schaltflächen aktiviert oder deaktiviert.

``` syntax
SelectOrActivatButton(bEnabled)
```

## <a name="parameters"></a>Parameter

<dl> <dt>

<span id="bEnabled"></span><span id="benabled"></span><span id="BENABLED"></span>*benabled*
</dt> <dd>

Gibt an, ob der Vorgang als boolescher Wert aktiviert oder deaktiviert ist.

</dd> </dl>
