# <a name="x-property"></a>x-Eigenschaft

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die `x` -Eigenschaft legt die x-Position der linken oberen Ecke des Rechtecks fest oder ruft diese ab.

``` syntax
[ iXPos = ] DVDRect.x
```

## <a name="return-value"></a>Rückgabewert

Gibt die X-Position der linken oberen Ecke des Rechtecks als Ganzzahl zurück.

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist Lese-/Schreibzugriff und hat keinen Standardwert.
