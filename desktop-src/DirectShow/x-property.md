# <a name="x-property"></a>x-Eigenschaft

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die -Eigenschaft legt die x-Position der oberen linken Ecke des `x` Rechtecks fest oder ruft sie ab.

``` syntax
[ iXPos = ] DVDRect.x
```

## <a name="return-value"></a>Rückgabewert

Gibt die X-Position der oberen linken Ecke des Rechtecks als ganze Zahl zurück.

## <a name="remarks"></a>Hinweise

Diese Eigenschaft ist Lese-/Schreibzugriff ohne Standardwert.
