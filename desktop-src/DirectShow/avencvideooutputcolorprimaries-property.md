# <a name="avencvideooutputcolorprimaries-property"></a>Avencvideooutputcolorprimaries (Eigenschaft)

Gibt die Farb Primärwerte für das codierte Video an.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**Codecapi \_ avencvideooutputcolorprimaries**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eavencvideocolorprimaries**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolorprimaries) -Enumeration.
