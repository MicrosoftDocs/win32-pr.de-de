# <a name="effect"></a>Wirkung

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das Effect-Objekt erzeugt ein Video oder einen Audioeffekt. Um dieses Objekt zu erstellen, rufen Sie die [**iamtimeline:: erkreateemptynode**](iamtimeline-createemptynode.md) -Methode auf.

Das Effect-Objekt macht die folgenden Schnittstellen verfügbar:

-   [**Iamtimelineeffect**](iamtimelineeffect.md)
-   [**Iamtimelineobj**](iamtimelineobj.md)
-   [**Iamtimelinesplicustom**](iamtimelinesplittable.md)
-   **IPropertyPage**
