# <a name="analog-television-audio"></a>Analog zum Fernsehgerät

Mit dem analogen Fernsehgerät stammt die Audiodatei vom TV-Tuner. Verschiedene Tuner-Karten müssen anders konfiguriert werden. Einige Karten verwenden einen externen Connector zum Patchen der Audiokarte in. Andere Benutzer verwenden einen internen Connector, um die Audiokarte auf die CD-Eingabe der Audiokarte zu patchen.

In den folgenden Themen wird beschrieben, wie Sie TV-Audiodaten in DirectShow anzeigen und erfassen:

-   [Vorschau der TV-Audiodatei](previewing-tv-audio.md)
-   [TV-Audiodatei](capturing-tv-audio.md)
