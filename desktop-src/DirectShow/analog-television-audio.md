# <a name="analog-television-audio"></a>Analoge Fernsehaudiodaten

Bei analogem Fernsehgerät stammt das Audio vom TV-Tuner. Verschiedene Tunerkarten müssen unterschiedlich konfiguriert werden. Einige Karten verwenden einen externen Connector, um die Audiodaten auf die Zeile in der Soundkarte zu patchen. Andere verwenden einen internen Connector, um die Audiodaten auf die CD-Eingabe der Soundkarte zu patchen.

In den folgenden Themen wird beschrieben, wie Tv-Audio in DirectShow in der Vorschau angezeigt und erfasst wird:

-   [Vorschau von TV-Audio](previewing-tv-audio.md)
-   [Erfassen von TV-Audio](capturing-tv-audio.md)
