# <a name="avencmpvintravlctable-property"></a>AVEncMPVIntraVLCTable(Eigenschaft)

Gibt an, welche VLC-Tabelle (Variable Length Coding) für die Entropiecodierung verwendet werden soll. Diese Eigenschaft gilt für MPEG-Videoencoder.

Dies ist eine Eigenschaft mit Lese- und Schreibzugriff.

## <a name="data-type"></a>Datentyp

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>Eigenschaften-GUID

**CODECAPI \_ AVEncMPVIntraVLCTable**

## <a name="property-value"></a>Eigenschaftswert

Der Wert dieser Eigenschaft ist ein Member der [**eAVEncMPVIntraVLCTable-Enumeration.**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpvintravlctable)
