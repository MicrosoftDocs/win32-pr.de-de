# <a name="overview-of-the-timeline-components"></a>Übersicht über die Zeitachsenkomponenten

\[Diese API wird nicht unterstützt und kann in Zukunft geändert oder nicht mehr verfügbar sein.\]

Die Zeitachse in [DirectShow Editing Services](directshow-editing-services.md) (DES) ist eine COM-Komponente. Die Objekte innerhalb der Zeitachse sind auch COM-Komponenten. Dieser Artikel bietet eine Übersicht über die Schnittstellen, die von den verschiedenen Objekten verfügbar gemacht werden.

Dieser Artikel enthält folgende Themen:

-   [Die Zeitachse](the-timeline.md)
-   [Zeitachsenobjekte](timeline-objects.md)
-   [Unterobjekte](subobjects.md)

Weitere Informationen zu Zeitachsen finden Sie unter [Das Zeitachsenmodell.](the-timeline-model.md) Eine Liste der Komponenten in DES finden Sie unter [DirectShow Editing Services Objects](directshow-editing-services-objects.md).
