# <a name="overview-of-the-timeline-components"></a>Übersicht über die Zeitachsen Komponenten

\[Diese API wird nicht unterstützt und kann in Zukunft geändert oder nicht verfügbar sein.\]

Die Zeitachse in [DirectShow-Bearbeitungs Diensten](directshow-editing-services.md) ist eine COM-Komponente. Die Objekte in der Zeitachse sind auch com-Komponenten. Dieser Artikel bietet eine Übersicht über die Schnittstellen, die von den verschiedenen Objekten verfügbar gemacht werden.

Dieser Artikel enthält folgende Themen:

-   [Die Zeitachse](the-timeline.md)
-   [Timeline-Objekte](timeline-objects.md)
-   [Unter Objekte](subobjects.md)

Weitere Informationen zu Zeitachsen finden Sie [im Zeitachsen Modell](the-timeline-model.md). Eine Liste der Komponenten in des finden Sie unter [DirectShow-Objekte zum Bearbeiten von Diensten](directshow-editing-services-objects.md).
