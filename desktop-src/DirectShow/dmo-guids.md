# <a name="dmo-guids"></a>DMO Guids

Um die in einer Kategorie registrierten DMOs aufzuzählen, übergeben Sie die entsprechende GUID an die [**DMOEnum-Funktion.**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmoenum)
