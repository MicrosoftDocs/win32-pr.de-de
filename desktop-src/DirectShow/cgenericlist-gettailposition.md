# <a name="cgenericlistgettailposition-method"></a>CGenericList.GetTailPosition-Methode

Die `GetTailPosition` -Methode ruft die Position des letzten Elements der Liste ab.

## <a name="syntax"></a>Syntax


```C++
POSITION GetTailPosition();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt einen POSITION-Wert oder **NULL zurück,** wenn die Liste leer ist.
