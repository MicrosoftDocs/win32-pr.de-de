# <a name="cbasepinm_bcanreconnectwhenactive-member"></a>Cbasepin:: m \_ bcanreconnect-Active Member

Flag, das angibt, ob die PIN dynamische erneute Verbindung unterstützt.

## <a name="syntax"></a>Syntax


```C++
bool m_bCanReconnectWhenActive;
```



## <a name="remarks"></a>Bemerkungen

Dieses Flag ist standardmäßig auf **false** eingestellt.
