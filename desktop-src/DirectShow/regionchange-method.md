# <a name="regionchange-method"></a>RegionChange-Methode

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die `RegionChange` -Methode zeigt ein Systemdialogfeld an, in dem der Benutzer die region ändern kann, die dem DVD-Laufwerk zugeordnet ist.

``` syntax
MSWebDVD.RegionChange()
```

## <a name="return-value"></a>Rückgabewert

Kein Rückgabewert.

## <a name="remarks"></a>Hinweise

Die Region für ein DVD-Laufwerk kann nur fünf mal geändert werden.
