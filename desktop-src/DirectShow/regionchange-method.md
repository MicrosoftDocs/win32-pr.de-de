# <a name="regionchange-method"></a>Regionchange-Methode

> [!Note]  
> Diese Komponente ist für die Verwendung in den Betriebssystemen Microsoft Windows 2000, Windows XP und Windows Server 2003 verfügbar. Es kann in nachfolgenden Versionen geändert oder entfernt werden.

 

Die- `RegionChange` Methode zeigt ein System Dialogfeld an, das es dem Benutzer ermöglicht, die dem DVD-Laufwerk zugeordnete Region zu ändern.

``` syntax
MSWebDVD.RegionChange()
```

## <a name="return-value"></a>Rückgabewert

Kein Rückgabewert.

## <a name="remarks"></a>Bemerkungen

Die Region für ein DVD-Laufwerk kann nur fünfmal geändert werden.
