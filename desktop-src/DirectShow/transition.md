# <a name="transition"></a>Übergang

> [!Note]  
> \[Veraltet. Diese API kann aus zukünftigen Versionen von Windows entfernt werden.\]

 

Das Übergangs Objekt erstellt einen Video Übergang Zwischentiteln oder Kompositionen. Um dieses Objekt zu erstellen, rufen Sie die [**iamtimeline:: erkreateemptynode**](iamtimeline-createemptynode.md) -Methode auf.

Das Übergangs Objekt macht die folgenden Schnittstellen verfügbar:

-   [**Iamtimelineobj**](iamtimelineobj.md)
-   [**Iamtimelinesplicustom**](iamtimelinesplittable.md)
-   [**Iamtimelinetrans**](iamtimelinetrans.md)
