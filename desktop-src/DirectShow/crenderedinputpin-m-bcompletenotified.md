# <a name="crenderedinputpinm_bcompletenotified-member"></a>Crenderedinputpin:: m \_ bcompletenoidentifizierte-Member

Die `m_bCompleteNotified` Member-Variable ist ein boolesches Flag, das angibt, ob die PIN ein [**EC \_ Complete**](ec-complete.md) -Ereignis an den Filter Graph-Manager gesendet hat.

## <a name="syntax"></a>Syntax


```C++
BOOL m_bCompleteNotified;
```
