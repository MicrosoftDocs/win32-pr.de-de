# <a name="cbasevideorendereronstartstreaming-method"></a>Cbasevideorenderer. onstartstreaming-Methode

Die- `OnStartStreaming` Methode setzt alle Male zurück, die das Streaming steuern.

## <a name="syntax"></a>Syntax


```C++
HRESULT OnStartStreaming();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt einen **HRESULT** -Wert zurück.

## <a name="remarks"></a>Bemerkungen

Diese Member-Funktion überschreibt [**cbaserenderer:: onstartstreaming**](cbaserenderer-onstartstreaming.md).
