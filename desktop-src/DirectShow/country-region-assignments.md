# <a name="countryregion-assignments"></a>Zuweisungen für Land/Region

Die standardmäßigen Länder Aufruf Codes der internationalen Telekommunikations Union (ITU) werden an die [**iamtuner::p UT \_ CountryCode**](/windows/desktop/api/Strmif/nf-strmif-iamtuner-put_countrycode) -Methode übermittelt, um Standard Zuordnungen für die Fernseh Häufigkeit auszuwählen.

Wenn Sie z. b. 20, den ITU-T-aufrufenden Code für Ägypten, übergeben, werden standardmäßige Fernsehfrequenzen für Ägypten ausgewählt.

Eine Liste der Länder Aufruf Codes finden Sie in der aktuellen ITU-T-Empfehlung der zugewiesenen Ländercodes.
