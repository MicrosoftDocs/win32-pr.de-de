# <a name="countryregion-assignments"></a>Zuweisungen von Ländern/Regionen

Standardcodes für Länderanrufe der International Telekommunikations-Union (INTERNATIONAL Telekommunikations-Union, ITU) werden an die [**IAMTuner::p ut \_ CountryCode-Methode**](/windows/desktop/api/Strmif/nf-strmif-iamtuner-put_countrycode) übergeben, um Standardzuordnungen für die Häufigkeit von Fernsehsendungen auszuwählen.

Wenn Sie z. B. 20 übergeben, den ITU-T-Anrufcode für "Hotels", würden Standardfrequenzen für Fernsehsendungen für "Usa" auswählen.

Eine Liste der Landesvorwahlen finden Sie in der aktuellen ITU-T-Empfehlung der zugewiesenen Ländercodes.
