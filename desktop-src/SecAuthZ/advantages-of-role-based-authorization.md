# <a name="advantages-of-role-based-authorization"></a>Vorteile der rollenbasierten Autorisierung

Die Autorisierungs-Manager-API bietet ein Autorisierungsmodell, das gegenüber der Autorisierung auf niedriger Ebene (ACL-basiert) mehrere Vorteile bietet:

-   Vereinfacht die Verwaltung der Zugriffssteuerung.
-   Macht die Autorisierung für Skripts und Anwendungen verfügbar, die mit dem Microsoft Visual Basic-Entwicklungssystem entwickelt wurden.
-   Stellt einen Mechanismus zum Anwenden von Laufzeit-Geschäftslogik beim Überprüfen des Zugriffs bereit.
