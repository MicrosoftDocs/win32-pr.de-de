# <a name="advantages-of-role-based-authorization"></a>Vorteile der rollenbasierten Autorisierung

Die Autorisierungs-Manager-API bietet ein Autorisierungs Modell mit mehreren Vorteilen gegenüber der (ACL-basierten) Autorisierung auf niedriger Ebene:

-   Vereinfacht die Verwaltung der Zugriffs Steuerung.
-   Stellt die Autorisierung für Skripts und Anwendungen zur Verfügung, die mithilfe des Microsoft Visual Basic Entwicklungssystems entwickelt wurden.
-   Bietet einen Mechanismus zum Anwenden der Lauf Zeit Geschäftslogik beim Überprüfen des Zugriffs.
