# <a name="primary-group-of-a-new-object"></a>Primäre Gruppe eines neuen Objekts

Die primäre Gruppe eines neuen Objekts ist [](/windows/desktop/SecGloss/s-gly) die primäre Gruppe aus der Sicherheitsbeschreibung, die vom Ersteller des Objekts angegeben wird. Wenn der Ersteller eines Objekts keine primäre Gruppe an gibt, ist die primäre Gruppe des Objekts die primäre Standardgruppe aus dem primären Token oder identitätswechseltoken des [*Erstellers.*](/windows/desktop/SecGloss/i-gly)
