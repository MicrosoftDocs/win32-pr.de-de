# <a name="allowing-anonymous-access"></a>Zulassen des anonymen Zugriffs

Die Standard Sicherheitsrichtlinie schränkt den anonymen lokalen Zugriff auf keine Rechte ein. Administratoren können dann nach Bedarf Rechte hinzufügen oder subtrahieren.

Eine lokale Zugriffs Gruppe ist für Anwendungen mit denselben Zugriffsrechten wie jeder vorhanden. Administratoren können dann die Anzahl der Benutzer in dieser Gruppe, die als **Pre-Windows 2000-kompatible Zugriffs Gruppe** bezeichnet wird, entsprechend erhöhen oder verringern.

Weitere Informationen finden Sie auf den Referenzseiten zu den Funktionen, die in der Tabelle des Themas " [lokale Gruppenfunktionen](/windows/desktop/NetMgmt/local-group-functions) " aufgeführt sind.
