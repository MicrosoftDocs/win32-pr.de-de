# <a name="authorization-objects"></a>Autorisierungs Objekte

Die folgenden Objekte werden mit Autorisierungs Anwendungen verwendet.

Das ActiveX-Installationsprogramm stellt das folgende Objekt bereit.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                               | BESCHREIBUNG                                                                                                                        |
|---------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [**Cieaxiinstallerservice**](cieaxiinstallerservice.md)<br/> | Implementiert die Schnittstellen [**iaxiservice**](ieaxiservice.md) und [**ieaxiservicecallback**](ieaxiservicecallback.md) .<br/> |



 


Der Autorisierungs-Manager stellt die folgenden Objekte bereit.



| Object                                               | BESCHREIBUNG                                                                          |
|------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**AzAuthorizationStore**](/windows/desktop/api/Azroles/nn-azroles-iazauthorizationstore) | Definiert den Container, der das Stammverzeichnis des Autorisierungs Richtlinien Speicher ist.<br/> |
| [**AzBizRuleContext**](/windows/desktop/api/Azroles/nn-azroles-iazbizrulecontext)         | Enthält Informationen zu einem Geschäftsregel Vorgang (BizRule).<br/>           |
