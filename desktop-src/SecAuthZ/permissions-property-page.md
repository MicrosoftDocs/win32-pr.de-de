# <a name="permissions-property-page"></a>Eigenschaften Seite "Berechtigungen"

Die Eigenschaften Seite **Berechtigungen** enthält Steuerelemente für die erweiterte Bearbeitung der [*ACEs*](/windows/desktop/SecGloss/a-gly) in der [*DACL*](/windows/desktop/SecGloss/d-gly)eines Objekts. Auf dieser Seite können Benutzer alle Informationen in der DACL anzeigen und bearbeiten, einschließlich der Informationen, die auf der [Eigenschaften Seite grundlegende Sicherheit](basic-security-property-page.md)nicht verfügbar sind. Zu den erweiterten Informationen zählen [objektspezifische ACEs](object-specific-aces.md) und Informationen zur [ACE-Vererbung](ace-inheritance.md).

Die Eigenschaften Seite **Berechtigungen** ist immer im [Eigenschaften Blatt erweiterte Sicherheit](advanced-security-property-sheet.md) enthalten, das angezeigt wird, wenn der Benutzer auf der Eigenschaften Seite grundlegende Sicherheit auf die Schaltfläche **erweitert** klickt.
