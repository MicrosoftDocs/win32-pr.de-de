# <a name="about-authorization"></a>Informationen zur Autorisierung

Die folgenden Abschnitte enthalten Informationen zur Autorisierung.



| `Section`                                                                                             | BESCHREIBUNG                                                                                                                                                                                                                                     |
|-----------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Zugriffssteuerung](access-control.md)<br/>                                                     | Sicherheitsfeatures, die steuern, wer auf Ressourcen im Betriebssystem zugreifen kann. Anwendungen rufen Zugriffssteuerungsfunktionen auf, um zu festlegen, wer auf bestimmte Ressourcen zugreifen oder den Zugriff auf von der Anwendung bereitgestellte Ressourcen steuern kann.<br/>     |
| [Access Control-Editor](access-control-editor.md)<br/>                                       | Erstellen und verwenden Sie Eigenschaftenblätter und Eigenschaftenseiten, mit denen der Benutzer die Komponenten des Sicherheitsdeskriptors eines Objekts anzeigen [*und ändern kann.*](/windows/desktop/SecGloss/s-gly)<br/> |
| [Client-/Server Access Control](client-server-access-control.md)<br/>                         | Serveranwendungen, die Dienste für Clients bereitstellen.<br/>                                                                                                                                                                                |
| [Access Control für Anwendungsressourcen](access-control-for-application-resources.md)<br/> | Rollenbasierte und ACL-basierte Zugriffssteuerung für Anwendungsressourcen.<br/>                                                                                                                                                                   |
| [bevollmächtigende Integritätskontrolle](mandatory-integrity-control.md)<br/>                           | Zugriffssteuerung auf Systemebene für sicherungsfähige Objekte.<br/>                                                                                                                                                                                   |
| [Benutzerkontensteuerung](user-account-control.md)<br/>                                         | Sicherheitsfunktion, mit der Benutzer allgemeine Aufgaben als Nichtadministratoren, als Standardbenutzer und als Administratoren ausführen können, ohne benutzerwechseln, sich abmelden oder Ausführen als **verwenden zu müssen.**<br/>                                           |
