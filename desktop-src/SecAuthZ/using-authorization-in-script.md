# <a name="using-authorization-in-script"></a>Verwenden der Autorisierung im Skript

Sie können die Autorisierungs-Manager-API verwenden, um den Zugriff auf Anwendungs Ressourcen zu steuern.

Die Beispiele in diesem Abschnitt werden mithilfe von Visual Basic Scripting Edition (VBScript) und Active Server Pages (ASP) geschrieben.

Weitere Informationen zu anderen Autorisierungs Aufgaben finden Sie [unter unterstützen von Aufgaben in Skripts](supporting-tasks-for-authorization-in-script.md).
