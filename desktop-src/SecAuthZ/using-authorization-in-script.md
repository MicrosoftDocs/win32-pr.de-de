# <a name="using-authorization-in-script"></a>Verwenden der Autorisierung in Skripts

Sie können die Autorisierungs-Manager-API verwenden, um den Zugriff auf Anwendungsressourcen zu steuern.

Die Beispiele in diesem Abschnitt werden mithilfe von Visual Basic Scripting Edition (VBScript) und Active Server Pages (ASP) geschrieben.

Informationen zu anderen Autorisierungsaufgaben finden Sie unter [Supporting Tasks in Script](supporting-tasks-for-authorization-in-script.md).



| Thema                                                                                                                      | BESCHREIBUNG                                                                                              |
|----------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [Definieren von Berechtigungen in Skripts](defining-permissions-in-script.md)                                                       | Legen Sie fest, welche Benutzer Zugriff auf welche Anwendungsressourcen haben, indem Sie einen Autorisierungsrichtlinienspeicher erstellen. |
| [Überprüfen des Clientzugriffs auf eine angeforderte Ressource im Skript](verifying-client-access-to-a-requested-resource-in-script.md) | Überprüfen Sie, ob der Client Zugriff auf einen oder mehrere Vorgänge hat.                                           |
| [Delegieren der Definition von Berechtigungen im Skript](delegating-the-defining-of-permissions-in-script.md)                   | Delegieren Sie die Verwaltung von Autorisierungsrichtlinienspeichern, die in Active Directory gespeichert sind.          |
