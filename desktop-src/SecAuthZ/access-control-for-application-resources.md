# <a name="access-control-for-application-resources"></a>Access Control für Anwendungs Ressourcen

Mit der Autorisierungs-Manager-API und dem MMC-Snap-in, die in Windows Server 2003 verfügbar sind, erhalten Anwendungen ein flexibles rollenbasiertes Zugriffs Steuerungs Framework. Dieses Framework umfasst eine Reihe von strippbaren com-Objekten für Web-und Line-of-Business-Anwendungen.

In den folgenden Abschnitten finden Sie Informationen zum Steuern des Zugriffs auf Anwendungs Ressourcen:

-   [Rollenbasierte Access Control](role-based-access-control.md)
-   [ACL-basiertes Access Control](acl-based-access-control.md)
