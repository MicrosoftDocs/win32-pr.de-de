# <a name="authorization"></a>Autorisierung

## <a name="purpose"></a>Zweck

Autorisierung ist das Recht, das einer Person gewährt wird, das System und die darin gespeicherten Daten zu verwenden. Die Autorisierung wird in der Regel von einem Systemadministrator eingerichtet und vom Computer anhand einer Form der Benutzeridentifikation überprüft, z. B. einer Codenummer oder einem Kennwort.

Zu den Microsoft-Autorisierungstechnologien gehören Authorization Manager und die Authz API.

## <a name="developer-audience"></a>Entwicklergruppe

Microsoft-Autorisierungstechnologien sind für die Verwendung durch Entwickler von Anwendungen vorgesehen, die auf dem Windows Server und Windows Betriebssystemen basieren, die den Zugriff auf Ressourcen steuern. Entwickler sollten mit der Windows-basierten Programmierung vertraut sein. Obwohl dies nicht erforderlich ist, wird empfohlen, die Autorisierung oder sicherheitsbezogene Themen zu verstehen.
