# <a name="authorization-data-types"></a>Autorisierungsdatentypen

Die folgenden Datentypen werden mit Autorisierungsanwendungen verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                           | Beschreibung                                                                                                                                                                                                                                                                                     |
|---------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ACCESS \_ MASK**](access-mask.md)<br/>                                  | Definiert Standard-, spezifische und generische Rechte. Diese Rechte werden in Zugriffssteuerungseinträgen (Access [Control Entries,](/windows/desktop/SecGloss/a-gly) ACEs) verwendet und sind das primäre Mittel zum Angeben des angeforderten oder gewährten Zugriffs auf ein Objekt.<br/> |
| [**\_SICHERHEITSBESCHREIBUNGSSTEUERUNG \_**](security-descriptor-control.md)<br/> | Eine Gruppe von Bitflags, die die Bedeutung eines [Sicherheitsdeskriptors](/windows/desktop/SecGloss/s-gly) oder seiner Komponenten qualifizieren.<br/>                                                                                                          |
| [**\_SICHERHEITSINFORMATIONEN**](security-information.md)<br/>                | Identifiziert die objektbezogenen Sicherheitsinformationen, die festgelegt oder abgefragt werden.<br/>                                                                                                                                                                                                             |
