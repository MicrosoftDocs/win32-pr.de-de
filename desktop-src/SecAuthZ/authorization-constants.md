# <a name="authorization-constants-authorization"></a>Autorisierungskonstanten (Autorisierung)

Autorisierungskonstanten werden gemäß der Verwendung wie folgt kategorisiert.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                         | Beschreibung                                                                                                                                                                                                                                                          |
|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Kontorechtekonstanten**](account-rights-constants.md)<br/>       | Kontorechte bestimmen den Anmeldetyp, den ein Benutzerkonto ausführen kann. Ein Administrator weist Benutzer- und Gruppenkonten Kontorechte zu. Die Kontorechte jedes Benutzers umfassen die dem Benutzer und den Gruppen, denen der Benutzer angehört, gewährten Rechte.<br/> |
| [**SID-Konstanten für App-Container**](app-container-sid-constants.md)<br/> | Geben Sie die Autorität des Anwendungspakets vor.<br/>                                                                                                                                                                                                                |
| [**Überwachen von Konstanten**](auditing-constants.md)<br/>                   | Stellen Kategorien und Unterkategorien von Überwachungsrichtlinienereignissen dar.<br/>                                                                                                                                                                                            |
| [**SID-Konstanten für Funktionen**](capability-sid-constants.md)<br/>       | Definieren Sie für Anwendungen bekannte Funktionen mithilfe der [**AllocateAndInitializeSid-Funktion.**](/windows/win32/api/securitybaseapi/nf-securitybaseapi-allocateandinitializesid)<br/>                                                                                                                       |
| [**Berechtigungskonstanten**](privilege-constants.md)<br/>                 | Berechtigungen bestimmen den Typ der Systemvorgänge, die ein Benutzerkonto ausführen kann. Ein Administrator weist Benutzer- und Gruppenkonten Berechtigungen zu. Zu den Berechtigungen jedes Benutzers gehören die Berechtigungen, die dem Benutzer und den Gruppen gewährt werden, denen der Benutzer angehört.<br/> |
