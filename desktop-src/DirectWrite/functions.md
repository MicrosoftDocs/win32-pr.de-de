# <a name="directwrite-functions"></a>DirectWrite-Funktionen

DirectWrite bietet die folgende Funktion. Alle anderen Vorgänge werden mithilfe von DirectWrite-Schnittstellenobjekten ausgeführt, die von dem [**IDWriteFactory-Objekt erstellt**](/windows/win32/api/dwrite/nn-dwrite-idwritefactory) werden, das mit der folgenden Funktion erstellt wird.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                         | Beschreibung                                                                                                             |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**DWriteCreateFactory**](/windows/win32/api/dwrite/nf-dwrite-dwritecreatefactory)<br/> | Erstellt ein DirectWrite-Factoryobjekt, das für die nachfolgende Erstellung einzelner DirectWrite-Objekte verwendet wird.<br/> |
| [**DWriteCoreCreateFactory (DWriteCore)**](/windows/windows-app-sdk/api/win32/dwrite_core/nf-dwrite_core-dwritecorecreatefactory)<br/> | Erstellt ein DirectWrite-Factoryobjekt, das für die nachfolgende Erstellung einzelner DWriteCore-Objekte verwendet wird.<br/> |
