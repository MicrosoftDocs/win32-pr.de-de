# <a name="directwrite-functions"></a>DirectWrite-Funktionen

DirectWrite stellt die folgende Funktion bereit. Alle anderen Vorgänge werden mithilfe von DirectWrite-Schnittstellen Objekten durchgeführt, die vom [**idschreitefactory**](/windows/win32/api/dwrite/nn-dwrite-idwritefactory) -Objekt erstellt werden, das mit der folgenden Funktion erstellt wird.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                         | BESCHREIBUNG                                                                                                             |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**DWriteCreateFactory**](/windows/win32/api/dwrite/nf-dwrite-dwritecreatefactory)<br/> | Erstellt ein DirectWrite-Factoryobjekt, das für die nachfolgende Erstellung einzelner DirectWrite-Objekte verwendet wird.<br/> |
