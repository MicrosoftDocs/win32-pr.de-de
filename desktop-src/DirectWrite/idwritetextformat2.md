# <a name="idwritetextformat2-interface"></a>IDWriteTextFormat2-Schnittstelle

Beschreibt die Schriftart- und Absatzeigenschaften, die zum Formatieren von Text verwendet werden, und beschreibt die Informationen zum -Locale.

## <a name="members"></a>Member

Die **IDWriteTextFormat2-Schnittstelle** erbt von [**IDWriteTextFormat1.**](idwritetextformat1.md) **IDWriteTextFormat2** verfügt auch über diese Membertypen:

-   [Methoden](#methods)

### <a name="methods"></a>Methoden

Die **IDWriteTextFormat2-Schnittstelle** verfügt über diese Methoden.



| Methode                                                      | Beschreibung                                                                      |
|:------------------------------------------------------------|:---------------------------------------------------------------------------------|
| [**GetLineSpacing**](/windows/win32/api/dwrite_3/nf-dwrite_3-idwritetextformat2-getlinespacing) | Ruft den Anpassungssatz für den Zeilenabstand für einen mehrzeilenigen Textumsatz ab. <br/> |
| [**SetLineSpacing**](idwritetextformat2-setlinespacing.md) | Legen Sie den Zeilenabstand fest.<br/>                                                     |
