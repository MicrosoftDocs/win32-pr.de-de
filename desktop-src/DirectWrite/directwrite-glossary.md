# <a name="directwrite-glossary"></a>DirectWrite-Glossar

<dl> <dt>

<span id="directwrite.directwrite_glossary_bidi__unicode_feature_"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_BIDI__UNICODE_FEATURE_"></span>**Bidi (Unicode-Feature)**
</dt> <dd>

Ausgesprochen "by-Dye". Text, der eine Mischung aus Wörtern aus unterschiedlichen Sprachen enthält, die in verschiedene Richtungen gelesen werden müssen. Erfordert häufig speziellen Code, der verarbeitet werden muss. "Arabisch und Hebräisch sind Bidi-Sprachen. der Text wird von rechts nach links, aber die Zahlen werden von links nach rechts angezeigt. "

</dd> <dt>

<span id="directwrite.directwrite_glossary_cleartype"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_CLEARTYPE"></span>**ClearType**
</dt> <dd>

Eine Schriftart Anzeige Technologie, die die Auflösung der Schriftart Anzeige erheblich verbessert, sodass die Buchstaben auf dem Computerbildschirm glatt, nicht verzweigt angezeigt werden. ClearType verbessert die Lesbarkeit von Text auf Farb-LCD-Monitoren mit einer digitalen Oberfläche, wie z. b. der in Laptops und hochwertigen Flatpanel-Desktop Anzeige.

</dd> <dt>

<span id="directwrite.directwrite_glossary_direct2d"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECT2D"></span>**Direct2D**
</dt> <dd>

Eine hardwarebeschleunigte, 2D-Grafik-API, die leistungsstarke und qualitativ hochwertige Rendering für 2-d-Geometrie, Bitmaps und Text bereitstellt. Die Direct2D-API ist für die Zusammenarbeit mit vorhandenem Code konzipiert, der GDI, GDI+ oder Direct3D verwendet.

</dd> <dt>

<span id="directwrite.directwrite_glossary_directwrite"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECTWRITE"></span>**DirectWrite**
</dt> <dd>

Eine DirectX-API, die ein verbessertes hochwertiges Text Rendering und Interoperabilität mit GDI und Direct2D bietet.

</dd> <dt>

<span id="directwrite.directwrite_glossary_directx"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECTX"></span>**DirectX**
</dt> <dd>

Eine Erweiterung des Microsoft Windows-Betriebssystems. Die DirectX-Technologie hilft spielen und anderen Programmen dabei, die erweiterten Multimedia-Funktionen Ihrer Hardware zu nutzen.

</dd> <dt>

<span id="directwrite.directwrite_glossary_gdi"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GDI"></span>**GDI**
</dt> <dd>

Ein ausführbares Programm, das grafische Funktionsaufrufe von einer Windows-basierten Anwendung verarbeitet und diese Aufrufe an den entsprechenden Gerätetreiber übergibt, der die Hardware spezifischen Funktionen ausführt, die eine Ausgabe generieren. Wenn Sie als Puffer zwischen Anwendungen und Ausgabegeräten agieren, zeigt GDI eine geräteunabhängige Ansicht der Welt für die Anwendung an, während Sie in einem geräteabhängigen Format mit dem Gerät interagiert.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH"></span>**Symbol**
</dt> <dd>

Die physische Darstellung eines Zeichens in einer angegebenen Schriftart. Zeichen können viele Glyphen aufweisen, wobei jede Schriftart in einem System möglicherweise ein anderes Symbol für dieses Zeichen definiert.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_composition"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_COMPOSITION"></span>**Symbol Komposition**
</dt> <dd>

Die Kombination von zwei oder mehr Symbolen zu einem einzelnen Symbol.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_decomposition"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_DECOMPOSITION"></span>**Glyphe-Zerlegung**
</dt> <dd>

Die Aufteilung eines einzelnen Symbols in mehrere Symbole.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_run"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_RUN"></span>**Glyphe ausführen**
</dt> <dd>

Eine Reihe von Symbolen in einer bestimmten Reihenfolge mit denselben Formatierungs Merkmalen, wie z. b. Schriftart, Größe, Gewichtung und Stil.

</dd> <dt>

<span id="directwrite.directwrite_glossary_hardware-accelerated_text"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_HARDWARE-ACCELERATED_TEXT"></span>**Hardware beschleunigter Text**
</dt> <dd>

Text, der von einer Technologie gerendert wird, die Hardwarebeschleunigung verwendet, um die Render

</dd> <dt>

<span id="directwrite.directwrite_glossary_interoperability"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_INTEROPERABILITY"></span>**interoper**
</dt> <dd>

Die Möglichkeit, mit zwei oder mehr APIs zu arbeiten und Informationen untereinander zu übertragen.

</dd> <dt>

<span id="directwrite.directwrite_glossary_kerning"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_KERNING"></span>**Unterschneidungen**
</dt> <dd>

Der Abstand zwischen bestimmten Buchstaben innerhalb eines bestimmten Worts.

</dd> <dt>

<span id="directwrite.directwrite_glossary_letterspacing"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_LETTERSPACING"></span>**Brief Abstände**
</dt> <dd>

Die Anpassung des Abstands zwischen zwei Zeichen, um die Darstellung des geraden Abstands zu erstellen, Text an einen angegebenen Bereich anzupassen und Zeilenumbrüche anzupassen.

</dd> <dt>

<span id="directwrite.directwrite_glossary_ligature"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_LIGATURE"></span>**Ligaturen**
</dt> <dd>

Zwei oder mehr Zeichen, die kombiniert werden, um ein einzelnes typografisches Zeichen darzustellen. Das moderne lateinische Skript verwendet einige. Andere Skripts verwenden viele Ligaturen, die von Schriftart und Stil abhängen. Einige Sprachen besitzen obligatorische Ligaturen, z. b. Arabisch.

</dd> <dt>

<span id="directwrite.directwrite_glossary_smart_pointers"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_SMART_POINTERS"></span>**intelligente Zeiger**
</dt> <dd>

Eine Klasse, die COM-Schnittstellen Zeiger umschließt, die das Schnittstellen Objekt automatisch freigeben.

</dd> <dt>

<span id="directwrite.directwrite_glossary_swashes"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_SWASHES"></span>**Schwung Schrift**
</dt> <dd>

Eine dekorative Ergänzung zu einem Zeichen, das ein mehr stilistisches Symbol enthält.

</dd> <dt>

<span id="directwrite.directwrite_glossary_unicode"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_UNICODE"></span>**Unicode-**
</dt> <dd>

Ein Standard für Zeichencodierung, der vom Unicode-Konsortium entwickelt wurde und fast alle Schriftsprachen der Welt darstellt. Das Unicode-Zeichenrepertoire umfasst mehrere Darstellungsformen, einschließlich UTF-8, UTF-16 und UTF-32. Die meisten Windows-Schnittstellen verwenden die Form UTF-16. (Gtmt hat 20 Definitionen).

</dd> <dt>

<span id="directwrite.directwrite_glossary_windows_presentation_foundation__wpf_"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_WINDOWS_PRESENTATION_FOUNDATION__WPF_"></span>**Windows Presentation Foundation (WPF)**
</dt> <dd>

Ein GUI-Framework, das von .net Micro Framework 3,0, Windows Vista und Microsoft Silverlight verwendet wird. Die .net Micro Framework-GUI-Klassen basieren auf WPF locker.

</dd> </dl>
