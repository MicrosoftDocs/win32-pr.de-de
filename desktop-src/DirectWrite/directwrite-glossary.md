# <a name="directwrite-glossary"></a>DirectWrite Glossar

<dl> <dt>

<span id="directwrite.directwrite_glossary_bidi__unicode_feature_"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_BIDI__UNICODE_FEATURE_"></span>**BiDi (Unicode-Feature)**
</dt> <dd>

Ausgesprochen "by-Aber". Text, der eine Mischung aus Wörtern aus verschiedenen Sprachen enthält, die in verschiedene Richtungen gelesen werden müssen. Erfordert häufig speziellen Code für die Verarbeitung. "Arabisch und Hebräisch sind BiDi-Sprachen; der Text geht von rechts nach links, aber die Zahlen gehen von links nach rechts."

</dd> <dt>

<span id="directwrite.directwrite_glossary_cleartype"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_CLEARTYPE"></span>**Cleartype**
</dt> <dd>

Eine Schriftanzeigetechnologie, die die Schriftanzeigeauflösung erheblich verbessert, sodass Buchstaben auf dem Computerbildschirm reibungslos und nicht verdrggt angezeigt werden. ClearType verbessert die Lesbarkeit von Text auf FARB-MONITOREN mit einer digitalen Schnittstelle erheblich, z. B. in Laptops und hochwertigen Flachpanel-Desktopanzeigen.

</dd> <dt>

<span id="directwrite.directwrite_glossary_direct2d"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECT2D"></span>**Direct2d**
</dt> <dd>

Eine hardwarebeschleunigte 2D-Grafik-API mit sofortigem Modus, die hochleistungs- und qualitativ hochwertiges Rendering für 2D-Geometrie, Bitmaps und Text bietet. Die Direct2D-API ist so konzipiert, dass sie gut mit vorhandenem Code zusammenarbeiten kann, der GDI, GDI+ oder Direct3D verwendet.

</dd> <dt>

<span id="directwrite.directwrite_glossary_directwrite"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECTWRITE"></span>**DirectWrite**
</dt> <dd>

Eine DirectX-API, die verbessertes, hochwertiges Textrendering und interoperabilität mit GDI und Direct2D bietet.

</dd> <dt>

<span id="directwrite.directwrite_glossary_directx"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECTX"></span>**Directx**
</dt> <dd>

Eine Erweiterung des Microsoft Windows Betriebssystems. DirectX-Technologie unterstützt Spiele und andere Programme bei der Verwendung der erweiterten Multimediafunktionen Ihrer Hardware.

</dd> <dt>

<span id="directwrite.directwrite_glossary_gdi"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GDI"></span>**Gdi**
</dt> <dd>

Ein ausführbares Programm, das grafische Funktionsaufrufe von einer Windows-basierten Anwendung verarbeitet und diese Aufrufe an den entsprechenden Gerätetreiber übergibt, der die hardwarespezifischen Funktionen ausführt, die die Ausgabe generieren. Durch die Verwendung als Puffer zwischen Anwendungen und Ausgabegeräten bietet GDI eine geräteunabhängige Ansicht der Welt für die Anwendung, während sie in einem geräteabhängigen Format mit dem Gerät interagiert.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH"></span>**Glyph**
</dt> <dd>

Die physische Darstellung eines Zeichens in einer angegebenen Schriftart. Zeichen können viele Glyphen enthalten, bei denen jede Schriftart auf einem System möglicherweise ein anderes Glyphen für dieses Zeichen definiert.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_composition"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_COMPOSITION"></span>**Glyphenkomposition**
</dt> <dd>

Die Kombination von zwei oder mehr Glyphen zu einem einzelnen Glyphen.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_decomposition"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_DECOMPOSITION"></span>**Glyphendekomposition**
</dt> <dd>

Die Aufteilung eines einzelnen Glyphen in mehrere Glyphen.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_run"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_RUN"></span>**Glyphen ausführen**
</dt> <dd>

Ein Satz von Glyphen in einer bestimmten Reihenfolge mit den gleichen Formatierungsmerkmalen wie Schriftart, Schriftgrad, Gewichtung und Stil.

</dd> <dt>

<span id="directwrite.directwrite_glossary_hardware-accelerated_text"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_HARDWARE-ACCELERATED_TEXT"></span>**hardwarebeschleunigten Text**
</dt> <dd>

Text, der von einer Technologie gerendert wird, die die Hardwarebeschleunigung verwendet, um die Renderingleistung zu verbessern.

</dd> <dt>

<span id="directwrite.directwrite_glossary_interoperability"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_INTEROPERABILITY"></span>**Interoperabilität**
</dt> <dd>

Die Möglichkeit, dass zwei oder mehr APIs miteinander arbeiten und Informationen zwischen diesen übertragen können.

</dd> <dt>

<span id="directwrite.directwrite_glossary_kerning"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_KERNING"></span>**Kerning**
</dt> <dd>

Der Abstand, der zwischen bestimmten Buchstabenpaaren innerhalb eines bestimmten Worts angewendet wird.

</dd> <dt>

<span id="directwrite.directwrite_glossary_letterspacing"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_LETTERSPACING"></span>**Letterspacing**
</dt> <dd>

Die Anpassung des Abstands zwischen zwei Zeichen, um die Darstellung eines gleichmäßigen Abstands zu erstellen, Text an ein bestimmtes Leerzeichen anzupassen und Zeilenumbrüche anzupassen.

</dd> <dt>

<span id="directwrite.directwrite_glossary_ligature"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_LIGATURE"></span>**Ligatur**
</dt> <dd>

Zwei oder mehr Zeichen, die kombiniert werden, um ein einzelnes typografisches Zeichen zu darstellen. Das moderne lateinische Skript verwendet einige. Andere Skripts verwenden vieleLigaturen, die von Schriftart und Stil abhängen. Einige Sprachen verfügen über obligatorischeLigaturen, z. B. Arabisch.

</dd> <dt>

<span id="directwrite.directwrite_glossary_smart_pointers"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_SMART_POINTERS"></span>**Intelligente Zeiger**
</dt> <dd>

Eine Klasse, die COM-Schnittstellenzeige umschließt, die das Schnittstellenobjekt automatisch frei gibt.

</dd> <dt>

<span id="directwrite.directwrite_glossary_swashes"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_SWASHES"></span>**Swashes**
</dt> <dd>

Eine zierliche Ergänzung zu einem Zeichen, das für ein stilistisches Glyphen sorgt.

</dd> <dt>

<span id="directwrite.directwrite_glossary_unicode"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_UNICODE"></span>**Unicode**
</dt> <dd>

Ein Standard für Zeichencodierung, der vom Unicode-Konsortium entwickelt wurde und fast alle Schriftsprachen der Welt darstellt. Das Unicode-Zeichenrepertoire umfasst mehrere Darstellungsformen, einschließlich UTF-8, UTF-16 und UTF-32. Die meisten Windows-Schnittstellen verwenden die Form UTF-16. (GTMT verfügt über 20 Definitionen).

</dd> <dt>

<span id="directwrite.directwrite_glossary_windows_presentation_foundation__wpf_"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_WINDOWS_PRESENTATION_FOUNDATION__WPF_"></span>**Windows Presentation Foundation (WPF)**
</dt> <dd>

Ein GUI-Framework, das von .NET Micro Framework 3.0, Windows Vista und Microsoft Silverlight verwendet wird. Die GUI-Klassen von .NET Micro Framework basieren lose auf WPF.

</dd> </dl>
