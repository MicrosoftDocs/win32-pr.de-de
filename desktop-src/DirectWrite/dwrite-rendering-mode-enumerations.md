# <a name="dwrite_rendering_mode-enumerations"></a>DWRITE \_ RENDERING \_ MODE-Enumerationen

Ab Windows 8 wurden durch die **DWRITE \_ RENDERING \_ MODE-Enumeration** neue Enumerationswerte hinzugefügt und andere veraltet.

Ab Windows 8 bestimmt die [**DWRITE \_ TEXT \_ ANTIALIAS \_ MODE-Enumeration,**](/windows/win32/api/Dwrite_1/ne-dwrite_1-dwrite_text_antialias_mode) ob Text mit ClearType gerendert wird. Daher sind alle ClearType-Renderingmodi in der **DWRITE \_ RENDERING \_ MODE-Enumeration** veraltet. Diese Enumerationswerte werden nun neuen Renderingmodi zugeordnet.

Die folgende Tabelle zeigt die alten Enumerationswerte und die neuen Werte, denen sie zugeordnet sind. Beschreibungen der neuen Werte finden Sie unter [**DWRITE \_ RENDERING \_ MODE**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode).



| Alter Modus                                                                                | Neuer Modus                                                                                |
|-----------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [**\_DWRITE-RENDERINGMODUS \_ \_ CLEARTYPE \_ GDI \_ CLASSIC**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode)       | [**DWRITE \_ RENDERING \_ MODE \_ GDI \_ CLASSIC**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode)                  |
| [**\_DWRITE-RENDERINGMODUS \_ \_ CLEARTYPE \_ GDI \_ NATURAL**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode)       | [**DWRITE \_ RENDERING \_ MODE \_ GDI \_ NATURAL**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode)                  |
| [**\_DWRITE-RENDERINGMODUS \_ \_ CLEARTYPE \_ NATURAL**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode)            | [**\_DWRITE-RENDERINGMODUS \_ \_ CLEARTYPE \_ NATURAL**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode)            |
| [**\_DWRITE-RENDERINGMODUS \_ \_ CLEARTYPE \_ NATURAL \_ SYMMETRIC**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode) | [**\_DWRITE-RENDERINGMODUS \_ \_ CLEARTYPE \_ NATURAL \_ SYMMETRIC**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode) |
