# <a name="directwrite-macros"></a>DirectWrite-Makros

DirectWrite definiert die folgenden Makros.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [**DWRITE_MAKE_FONT_AXIS_TAG**](/windows/win32/api/dwrite_3/nf-dwrite_3-dwrite_make_font_axis_tag) | Erstellt ein OpenType-Tag für eine Schriftart Achse. |
| [**dwrite \_ make \_ OpenType- \_ Tag**](/windows/win32/api/dwrite/nf-dwrite-dwrite_make_opentype_tag) | Erstellt ein OpenType-Tag als 32-Bit-Ganzzahl, sodass das erste Zeichen im-Tag das niedrigste Byte ist (am geringsten bei kleinen Endian-Architekturen maßgeblich), das zum Vergleichen mit Tags in der Schriftart Datei verwendet werden kann. Dieses Makro ist kompatibel mit dem [**\_ Feature- \_ \_ Tag der dwrite-Schriftart**](/windows/win32/api/dwrite/ne-dwrite-dwrite_font_feature_tag). |
