# <a name="windows-event-collector-enumerations"></a>Windows-Ereignis Sammler-Enumerationen

In der folgenden Tabelle sind die Enumerationen des Windows-Ereignis Sammler-SDK aufgeführt.



| Enumeration                                                                                               | Beschreibung                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ \_ Konfigurations \_ Modus des EC-Abonnements**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_configuration_mode)                       | Gibt verschiedene Konfigurations Modi an, die die Standardeinstellungen für ein Abonnement ändern.                                            |
| [**\_ \_ Inhalts \_ Format des EC-Abonnements**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_content_format)                               | Gibt an, wie Ereignisse auf dem Computer gerendert werden, der die Ereignisse sendet, bevor die Ereignisse an den Ereignis Sammler Computer gesendet werden. |
| [**Anmelde Informationen für das EC- \_ Abonnement \_ \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_credentials_type)                           | Gibt den Typ der Anmelde Informationen an, die bei der Kommunikation mit Ereignis Quellen verwendet werden sollen.                                                         |
| [**EC-Abonnement Übermittlungs \_ \_ \_ Modus**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_delivery_mode)                                 | Gibt an, wie Ereignisse über ein Ereignis Abonnement (mit einem Push-oder Pull-Modell) übermittelt werden.                                          |
| [**Eigenschaften-ID des EC- \_ Abonnements \_ \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_property_id)                                     | Definiert Werte zur Identifizierung von Ereignis Abonnement Eigenschaften, die für die Abonnement Konfiguration verwendet werden.                                           |
| [**Status \_ des \_ \_ \_ aktiven \_ Status des EC-Abonnements**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_active_status) | Gibt den Status eines Abonnements oder einer Ereignis Quelle in Bezug auf ein Abonnement an.                                               |
| [**\_ID des \_ Lauf \_ Zeit \_ Status \_ der EC-Abonnements**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_info_id)             | Gibt einen Wert an, der eine Eigenschaft des Lauf Zeit Status einer Ereignis Quelle oder eines Abonnements angibt.                                |
| [**EC \_ - \_ Abonnementtyp**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_type)                                                    | Gibt den Typ des zu verwendenden Abonnements an (ein von der Quelle initiiertes oder vom Collector initiiertes Abonnement).                                     |
| [**Typ der EC- \_ Variante \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_variant_type)                                                              | Definiert die Werte, die die Datentypen angeben, die in den Windows-Ereignis Sammlungs Funktionen verwendet werden.                                  |
