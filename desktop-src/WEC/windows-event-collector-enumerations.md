# <a name="windows-event-collector-enumerations"></a>Windows Event Collector-Enumerationen

In der folgenden Tabelle sind die Enumerationen des Windows Event Collector SDK aufgeführt.



| Enumeration                                                                                               | Beschreibung                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ \_ \_ EC-ABONNEMENTKONFIGURATIONSMODUS**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_configuration_mode)                       | Gibt verschiedene Konfigurationsmodi an, die die Standardeinstellungen für ein Abonnement ändern.                                            |
| [**INHALTSFORMAT \_ DES \_ \_ EC-ABONNEMENTS**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_content_format)                               | Gibt an, wie Ereignisse auf dem Computer gerendert werden, der die Ereignisse sendet, bevor die Ereignisse an den Ereignissammlercomputer gesendet werden. |
| [**TYP DER \_ \_ ANMELDEINFORMATIONEN FÜR DAS \_ EC-ABONNEMENT**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_credentials_type)                           | Gibt den Typ der Anmeldeinformationen an, die bei der Kommunikation mit Ereignisquellen verwendet werden.                                                         |
| [**\_ÜBERMITTLUNGSMODUS FÜR \_ \_ EC-ABONNEMENTS**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_delivery_mode)                                 | Gibt an, wie Ereignisse über ein Ereignisabonnement (mithilfe eines Push- oder Pullmodells) übermittelt werden.                                          |
| [**\_EIGENSCHAFTEN-ID DES \_ \_ EC-ABONNEMENTS**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_property_id)                                     | Definiert Werte zum Identifizieren von Ereignisabonnementeigenschaften, die für die Abonnementkonfiguration verwendet werden.                                           |
| [**STATUS \_ DER \_ \_ EC-ABONNEMENTLAUFZEIT \_ AKTIV \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_active_status) | Gibt den Status eines Abonnements oder einer Ereignisquelle in Bezug auf ein Abonnement an.                                               |
| [**EC SUBSCRIPTION RUNTIME STATUS INFO ID (ID DER \_ \_ \_ \_ LAUFZEITSTATUS DES \_ EC-ABONNEMENTS)**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_info_id)             | Gibt einen Wert an, der eine Eigenschaft des Laufzeitstatus einer Ereignisquelle oder eines Abonnements identifiziert.                                |
| [**\_ \_ EC-ABONNEMENTTYP**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_type)                                                    | Gibt den Typ des zu verwendenden Abonnements an (ein von der Quelle oder vom Collector initiiertes Abonnement).                                     |
| [**EC \_ VARIANT \_ TYPE**](/windows/desktop/api/Evcoll/ne-evcoll-ec_variant_type)                                                              | Definiert die Werte, mit denen die Datentypen angegeben werden, die in den Windows Event Collector-Funktionen verwendet werden.                                  |
