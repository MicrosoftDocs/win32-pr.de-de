# <a name="about-system-shutdown"></a>Zum Herunterfahren des Systems

Durch das Herunterfahren des Systems wird das System zu einer Bedingung, bei der es sicher ist, den Computer auszuschalten. Alle Dateisystem Puffer werden auf den Datenträger geleert. Daraufhin wird ein Meldungs Feld angezeigt, in dem der Benutzer darüber informiert wird, dass der Computer ausgeschaltet werden kann. Es gibt auch eine Neustart Option, mit der der Computer neu gestartet wird, anstatt dieses Meldungs Feld für das Herunterfahren des Systems anzuzeigen. Weitere Informationen finden [Sie](shutting-down.md)unter Herunterfahren. Ein Beispiel finden Sie unter Vorgehens [Weise beim Herunterfahren des Systems](how-to-shut-down-the-system.md).

Beim Abmelden werden alle Prozesse angehalten, die dem Sicherheitskontext des Prozesses zugeordnet sind, der die Funktion Exit aufgerufen hat, der aktuelle Benutzer wird vom System protokolliert und das Anmelde Dialogfeld angezeigt. Weitere Informationen finden Sie unter [Abmelden](logging-off.md). Ein Beispiel finden Sie unter [Abmelden des aktuellen Benutzers](how-to-log-off-the-current-user.md).

Das Sperren der Arbeitsstation schützt die Anzeige vor nicht autorisierter Verwendung, wenn Sie den Computer verlassen. Zum Entsperren der Arbeitsstation müssen Sie sich anmelden. Ein Beispiel finden Sie unter Vorgehens [Weise beim Sperren der Arbeits](how-to-lock-the-workstation.md)Station.

Mit der Ereignisprotokollierung für das [herunter](shutdown-event-tracker.md) fahren kann der Benutzer oder die Anwendung den Grund für das Herunterfahren oder Neustarten des Systems dokumentieren.
