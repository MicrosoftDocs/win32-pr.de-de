# <a name="control-point-programming-considerations"></a>Überlegungen zur Steuerungspunktprogrammierung

Entwickler, die UPnP-basierte Anwendungen (oder Steuerungspunkte) erstellen, müssen diese Anwendungen über einen COINIT \_ APARTMENTTHREADED-Client verwenden. Es gibt bekannte Probleme bei der Verwendung der Kontrollpunkt-API von einem COINIT \_ MULTITHREADED-Client, der unter Belastung ausgeführt wird.
