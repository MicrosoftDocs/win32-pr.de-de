# <a name="device-host-api-reference"></a>Referenz zur Geräte Host-API

Die folgenden Schnittstellen sind Teil der Geräte Host-API mit UPnP-Technologie. Der Geräte Host mit der UPnP-Technologie unterstützt Microsoft Visual Basic und C++.

Diese API bietet keine Unterstützung für Microsoft Visual Basic Scripting Edition (VBScript).



| Schnittstelle                                                  | BESCHREIBUNG                                                                                         |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**Iupnpde vicecontrol**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpdevicecontrol)           | Wird vom Geräte Host zum Verwalten von Geräten und zugehörigen Diensten verwendet.                                     |
| [**Iupnpdeviceprovider**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpdeviceprovider)         | Wird vom Geräte Host verwendet, um Geräte Anbieter zu starten und zu unterbinden.                                         |
| [**Iupnpeer-Sink**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpeventsink)                   | Wird von Geräten verwendet, um Ereignis Benachrichtigungen an den Geräte Host zu senden.                                     |
| [**Iupnpeer-Source**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpeventsource)               | Wird vom Geräte Host zum abonnieren oder kündigen von Ereignissen verwendet, die vom gehosteten Dienst bereitgestellt werden.       |
| [**Iupnpregistrinar**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpregistrar)                   | Wird von Geräten verwendet, um sich beim Geräte Host zu registrieren.                                                   |
| [**Iupnpremoteendpointinfo**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpremoteendpointinfo) | Wird von Geräten verwendet, um Informationen über einen Anforderer (d. h. einen Steuerungspunkt) und die Anforderung zu erhalten. |
| [**Iupnpreregistrinar**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpreregistrar)               | Wird von Geräten zum erneuten Registrieren beim Geräte Host verwendet.                                                |
