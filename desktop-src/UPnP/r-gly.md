# <a name="r-upnp-apis"></a>R (UPnP-APIs)

Enthält UPnP-bezogene Begriffe, die mit dem Buchstaben R beginnen.

[a](a-gly.md) B [C](c-gly.md) [D](d-gly.md) [E](e-gly.md) F G [H](h-gly.md) I J K L M [N](n-gly.md) O [P](p-gly.md) Q R [S](s-gly.md) T [U](u-gly.md) V W X Y Z

<dl> <dt>

<span id="upnp.r_1_gly"></span><span id="UPNP.R_1_GLY"></span>**Ressourcenverzeichnis**
</dt> <dd>

Beim Ressourcenverzeichnis handelt es sich um ein lokales Verzeichnis des Geräts, auf dem sich Dateien wie Symbol Dateien befinden.

</dd> <dt>

<span id="upnp.r_2_gly"></span><span id="UPNP.R_2_GLY"></span>**Stamm Gerät**
</dt> <dd>

Ein Stamm Gerät ist das oberste Gerät in einer Gerätestruktur. Wenn Sie z. b. auf ein VCR/TV-Kombinations Gerät zugreifen, das als Anzeige Station bezeichnet wird, gibt es drei Geräte: die Anzeige Station, VCR und das TV. Die Anzeige Station wird als Stamm Gerät betrachtet.

</dd> <dt>

<span id="upnp.r_3_gly"></span><span id="UPNP.R_3_GLY"></span>**Gerät wird ausgeführt**
</dt> <dd>

Ein Gerät, das gerade ausgeführt wird, ist ein Gerät, dessen Code bereits vor der Registrierung geladen wurde.

Dabei handelt es sich um Geräte, für die vor der Registrierung keine Dienst Beschreibung bekannt ist.

Beispiele für solche Geräte sind Bridges, z. b. eine UPnP-SCP-Bridge. Diese Bridge registriert die Ausführung von Geräten beim Geräte Host.

</dd> </dl>
