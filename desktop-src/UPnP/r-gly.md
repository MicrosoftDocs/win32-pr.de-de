# <a name="r-upnp-apis"></a>R (UPnP-APIs)

Enthält UPnP-bezogene Begriffe, die mit dem Buchstaben R beginnen.

[A](a-gly.md) B [C](c-gly.md) [D](d-gly.md) [E](e-gly.md) F G [H](h-gly.md) I J K L M [N](n-gly.md) O [P](p-gly.md) Q R [S](s-gly.md) T [U](u-gly.md) V W X Y Z

<dl> <dt>

<span id="upnp.r_1_gly"></span><span id="UPNP.R_1_GLY"></span>**Ressourcenverzeichnis**
</dt> <dd>

Das Ressourcenverzeichnis ist ein lokales Verzeichnis auf dem Gerät, in dem sich Dateien wie Symboldateien befinden.

</dd> <dt>

<span id="upnp.r_2_gly"></span><span id="UPNP.R_2_GLY"></span>**Stammgerät**
</dt> <dd>

Ein Stammgerät ist das oberste Gerät in einer Gerätestruktur. Wenn Sie beispielsweise auf ein VCR/TV-Kombinationsgerät zugreifen, das als Anzeigestation bezeichnet wird, gibt es drei Geräte: die Anzeigestation, den VCR und den TV. Die Anzeigestation wird als Stammgerät betrachtet.

</dd> <dt>

<span id="upnp.r_3_gly"></span><span id="UPNP.R_3_GLY"></span>**Ausgeführtes Gerät**
</dt> <dd>

Ein ausgeführtes Gerät ist ein Gerät, dessen Code bereits vor der Registrierung geladen wurde.

Hierbei handelt es sich um Geräte, für die vor der Registrierung keine Dienstbeschreibung bekannt ist.

Beispiele für solche Geräte sind Brücken, z. B. eine UPnP-SCP-Brücke. Diese Bridge registriert ausgeführte Geräte beim Gerätehost.

</dd> </dl>
