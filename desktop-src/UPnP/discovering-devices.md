# <a name="discovering-devices"></a>Entdecken von Geräten

Sie können auf drei Arten nach Geräten suchen: nach Typ, UDN und asynchroner Suche (bei der es sich um eine Suche nach Gerätetyp handelt).

![Fenster "Geräte entdecken"](images/ucp-disc.png)

**So entdecken Sie Geräte**

1.  Wählen Sie den Typ der Suche aus, den Sie verwenden möchten: Suchen nach **Typ,** **Suchen nach UDN** oder **Asynchrones Suchen.**
2.  Wählen Sie den Typ des Geräts oder den UDN aus der Liste aus (für Suchvorgänge nach Typ oder UDN). Der Inhalt der Liste basiert auf dem Inhalt der udn.txt und devtype.txt, die Sie zuvor bearbeitet haben.
3.  Klicken Sie auf **Ermittlung starten**. Die Suche wird gestartet. Die Ergebnisse werden in der Liste **Geräte gefunden** angezeigt. Wenn keine Geräte gefunden werden, wird eine Meldung angezeigt, die diese Bedingung angibt. Der Status des Suchfortschritts wird im Feld **Status** angezeigt.

Wenn Sie die asynchrone Suche verwenden, werden neue Geräte, die gefunden werden, in der Liste **Gefundene** Geräte und in der **Statusleiste** angezeigt. Wenn die asynchrone Suche abgeschlossen ist, wird auf **der Statusleiste** eine Meldung angezeigt, in der dies angezeigt wird. Da eine asynchrone Suche jedoch ausgeführt wird, bis  sie manuell beendet wird, werden neue Geräte in der Liste Gefundene Geräte und in der **Statusleiste** angezeigt, wenn die Geräte im Netzwerk angezeigt werden.

Sobald Sie Geräte gefunden haben, können Sie [sie steuern.](controlling-a-device.md)
