# <a name="about-the-device-host-api"></a>Informationen zur Gerätehost-API

Die Gerätehost-API mit UPnP-Technologie ist ein Framework zum Implementieren von UPnP-basierten Gerätefunktionen auf der Windows-Plattform. Entwickler, die Geräte mithilfe der Gerätehost-API mit UPnP-Technologie (als [*gehostete Geräte*](h-gly.md)bezeichnet) erstellen, müssen nur die Kernfunktionen des Geräts implementieren. Entwickler können sich auf den Gerätehost verlassen, um die UPnP-spezifischen Details der Ermittlung, Beschreibung, Steuerung, Ereigniserstellung und Präsentation zu verarbeiten. Der Gerätehost überprüft eingehende Daten von UPnP-basierten Clients und formatiert alle ausgehenden Daten von gehosteten Geräten gemäß der UPnP-Gerätearchitektur.
