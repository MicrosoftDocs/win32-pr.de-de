# <a name="about-the-device-host-api"></a>Informationen zur Geräte Host-API

Die Geräte Host-API mit UPnP-Technologie ist ein Framework für die Implementierung von UPnP-basierter Gerätefunktionalität auf der Windows-Plattform. Entwickler, die Geräte mithilfe der Geräte Host-API mit UPnP-Technologie (als [*gehostete Geräte*](h-gly.md)bezeichnet) erstellen, müssen nur die Kernfunktionen des Geräts implementieren. Entwickler können sich auf den Geräte Host verlassen, um die UPnP-spezifischen Details von Ermittlung, Beschreibung, Steuerung, Ereignis und Präsentation zu verarbeiten. Der Geräte Host überprüft eingehende Daten von UPnP-basierten Clients und formatiert alle ausgehenden Daten von gehosteten Geräten entsprechend der UPnP-Geräte Architektur.
