# <a name="using-the-dimmer-device-sample-code"></a>Verwenden des Beispielcodes für das Dimer-Gerät

Das Platform Software Development Kit (SDK) enthält Beispielcode für Einblendgeräte. Dieser Beispielcode veranschaulicht ein einfaches UPnP-basiertes Gerät, das mithilfe der Gerätehost-API mit UPnP-Technologie implementiert wird.

Das Verzeichnis DCO \_ DimerService enthält die Implementierung für das Gerät .dll. Das Verzeichnis RegisterDevice enthält die Implementierung zum Registrieren des Geräts .dll beim UPnP-Registrierungsdienst auf dem Computer. Das Verzeichnis StepByStep enthält VC6.0-Projekte, die der in der datei Readme.txt enthaltenen Schritt-für-Schritt-Anleitung entsprechen. Die "Schritt-für-Schritt-Anleitung" ist für Entwickler vorgesehen, die noch keine ATL-/COM-Erfahrung haben.
