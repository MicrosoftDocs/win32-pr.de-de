# <a name="using-the-dimmer-device-sample-code"></a>Verwenden des Beispielcodes für Dimmer Geräte

Das Platform Software Development Kit (SDK) enthält Beispielcode für schwächer-Geräte. Dieser Beispielcode veranschaulicht ein einfaches UPnP-basiertes Gerät, das mithilfe der Geräte Host-API mit UPnP-Technologie implementiert wird.

Das Verzeichnis DCO \_ dimmerservice enthält die Implementierung für die Datei "Device. dll". Das Verzeichnis "registerdevice" enthält die Implementierung für die Registrierung der Datei "Device. dll" beim UPnP-Registrierungsdienst auf dem Computer. Das stepbystep-Verzeichnis enthält VC 6.0-Projekte, die der "Schritt-für-Schritt-Anleitung" in der Readme.txt-Datei entsprechen. Die Schritt-für-Schritt-Anleitung ist für Entwickler vorgesehen, die nicht über vorherige ATL-/COM-Benutzererfahrung verfügen.
