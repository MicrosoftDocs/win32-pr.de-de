# <a name="a-upnp-apis"></a>A (UPnP-APIs)

Enthält UPnP-bezogene Begriffe, die mit dem Buchstaben A beginnen.

A B [C](c-gly.md) [D](d-gly.md) [E](e-gly.md) F G [H](h-gly.md) I J K L M [N](n-gly.md) O [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) T [U](u-gly.md) V W X Y Z

<dl> <dt>

<span id="upnp.a_1_gly"></span><span id="UPNP.A_1_GLY"></span>**Aktion**
</dt> <dd>

Eine Aktion stellt einen Befehl dar, den ein Client ausgibt, um eine bestimmte Funktion auszuführen. Sie benötigt null oder mehr Argumente.

</dd> <dt>

<span id="upnp.a_2_gly"></span><span id="UPNP.A_2_GLY"></span>**Verkünden**
</dt> <dd>

Die Ankündigung eines Geräts in einem Netzwerk bedeutet, dass Nachrichten über die Verfügbarkeit des Geräts an alle Knoten im Netzwerk übertragen werden.

</dd> <dt>

<span id="upnp.a_3_gly"></span><span id="UPNP.A_3_GLY"></span>**Argument**
</dt> <dd>

Ein Argument ist ein Parameter für eine Aktion.

</dd> </dl>
