<dl> <dt>

<span id="upnp.n_1_gly"></span><span id="UPNP.N_1_GLY"></span>**nicht laufendes Gerät**
</dt> <dd>

Ein Gerät, das nicht ausgeführt wird, ist ein Gerät, für das nur der Code geladen wird, wenn eine Anforderung für dieses Gerät empfangen wird. Dabei handelt es sich häufig um Geräte, für die Dienst Beschreibungen vor der Registrierung bekannt sind.

Ein typisches Beispiel für nicht-laufende Geräte ist ein vorläufiges Gerät, z. b. ein Media Player-Gerät.
