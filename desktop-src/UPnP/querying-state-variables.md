# <a name="querying-state-variables"></a>Abfragen von Zustandsvariablen

Die [**IUPnPService-Schnittstelle**](/windows/desktop/api/Upnp/nn-upnp-iupnpservice) stellt die [**QueryStateVariable-Methode**](/windows/desktop/api/Upnp/nf-upnp-iupnpservice-querystatevariable) bereit, die den Wert einer angegebenen Zustandsvariablen zurückgibt.

Das UPnP-Forum rät von der Verwendung von [**QueryStateVariable ab.**](/windows/desktop/api/Upnp/nf-upnp-iupnpservice-querystatevariable) Geräteautoren wurde empfohlen, entsprechende Aktionen zu schreiben, um diese Informationen zur Verfügung zu stellen. Geräteautoren sind nicht verpflichtet, **QueryStateVariable zu implementieren.**

Weitere Informationen finden Sie [im Abschnitt Aufrufen von Aktionen.](invoking-actions.md)
