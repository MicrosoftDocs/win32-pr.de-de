<dl> <dt>

<span id="upnp.c_1_gly"></span><span id="UPNP.C_1_GLY"></span>**Kontrollpunkt**
</dt> <dd>

Ein Steuerungspunkt ist ein Controller (z. B. eine Anwendung), der Geräte- und Dienstbeschreibungen abruft, Aktionen an Dienste sendet, Dienstzustandsvariablen abfragt und Ereignisse von Diensten empfängt. Anwendungen, die es einem Benutzer ermöglichen, ein Gerät zu verwenden oder ausführen, z. B. ein CD-Playergerät, werden als Kontrollpunkte betrachtet.

</dd> </dl>
