# <a name="what-is-the-control-point-object-model"></a>Was ist das Control Point Object Model?

Die folgende Abbildung zeigt das grundlegende Control Point-Objektmodell.

![Steuerelementpunktobjektmodell](images/upnp-object-model.png)

Bei der Suche nach Geräten mit der Gerätesuche-Schnittstelle wird eine Gerätesammlung erstellt. Eine Devices-Sammlung enthält null oder mehr Geräteobjekte. Anwendungen können die verschiedenen Gerätesammlungsmethoden verwenden, um auf einzelne Geräteobjekte zuzugreifen.

Geräteobjekte enthalten immer eine Services-Sammlung, die mindestens ein Dienstobjekt enthält. Diese Dienstobjekte werden von Anwendungen verwendet, um mit Geräten zu kommunizieren und diese zu steuern.
