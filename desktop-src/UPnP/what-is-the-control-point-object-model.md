# <a name="what-is-the-control-point-object-model"></a>Was ist das Steuerungspunkt-Objektmodell?

In der folgenden Abbildung wird das grundlegende Steuerungspunkt-Objektmodell veranschaulicht.

![Steuerungspunkt-Objektmodell](images/upnp-object-model.png)

Durch die Suche nach Geräten mithilfe der Device Finder-Schnittstelle wird eine Geräte Sammlung erstellt. Eine Geräte Sammlung enthält 0 (null) oder mehr Geräte Objekte. Anwendungen können mit den verschiedenen Geräte Sammlungs Methoden auf einzelne Geräte Objekte zugreifen.

Geräte Objekte enthalten immer eine Dienste-Sammlung, die ein oder mehrere Dienst Objekte enthält. Diese Dienst Objekte werden von Anwendungen verwendet, um mit Geräten zu kommunizieren und diese zu steuern.
