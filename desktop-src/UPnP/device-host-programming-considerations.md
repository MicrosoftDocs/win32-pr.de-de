# <a name="device-host-programming-considerations"></a>Überlegungen zur Programmierung von Gerätehosts

Eine leere Präsentations-URL in der Gerätebeschreibung ist nicht zulässig. Wenn Ihr Gerät nicht über eine Präsentations-URL verfügt, entfernen Sie die Präsentations-URL-Tags aus der Gerätebeschreibung.
