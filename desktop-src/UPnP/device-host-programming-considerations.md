# <a name="device-host-programming-considerations"></a>Überlegungen zur Programmierung von Geräte Hosts

Eine leere Präsentations-URL in der Geräte Beschreibung ist nicht zulässig. Wenn Ihr Gerät nicht über eine Präsentations-URL verfügt, entfernen Sie die Präsentations-URL-Tags aus der Geräte Beschreibung.
