# <a name="e-upnp-apis"></a>E (UPnP-APIs)

Enthält UPnP-bezogene Begriffe, die mit dem Buchstaben E beginnen.

[a](a-gly.md) B [C](c-gly.md) [D](d-gly.md) E F G [H](h-gly.md) I J K L M [N](n-gly.md) O [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) T [U](u-gly.md) V W X Y Z

<dl> <dt>

<span id="upnp.e_1_gly"></span><span id="UPNP.E_1_GLY"></span>**Veranstalter**
</dt> <dd>

Ein Ereignis ist eine Benachrichtigung über eine oder mehrere Änderungen in Zustandsvariablen, die von einem Dienst verfügbar gemacht werden.

</dd> <dt>

<span id="upnp.e_2_gly"></span><span id="UPNP.E_2_GLY"></span>**übergeordnete Variable**
</dt> <dd>

Eine evdown-Variable ist eine Zustands Variable für einen Dienst, die bei einer Änderung ein Ereignis auslöst. Alle Kontrollpunkte, die die Ereignis Quelle einer Variablen abonniert haben, erhalten dann eine Benachrichtigung über die Änderung. Eine nicht-Auswert-Zustands Variable ist in einer Ereignis Benachrichtigung nie enthalten.

</dd> </dl>
