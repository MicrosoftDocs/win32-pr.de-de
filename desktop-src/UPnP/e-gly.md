# <a name="e-upnp-apis"></a>E (UPnP-APIs)

Enthält UPnP-bezogene Begriffe, die mit dem Buchstaben E beginnen.

[A](a-gly.md) B [C](c-gly.md) [D](d-gly.md) E F G [H](h-gly.md) I J K L M [N](n-gly.md) O [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) T [U](u-gly.md) V W X Y Z

<dl> <dt>

<span id="upnp.e_1_gly"></span><span id="UPNP.E_1_GLY"></span>**Ereignis**
</dt> <dd>

Ein Ereignis ist eine Benachrichtigung über eine oder mehrere Änderungen an Zustandsvariablen, die von einem Dienst verfügbar gemacht werden.

</dd> <dt>

<span id="upnp.e_2_gly"></span><span id="UPNP.E_2_GLY"></span>**Evented Variable**
</dt> <dd>

Eine ereignisierte Variable ist eine Zustandsvariable für einen Dienst, der bei Änderung ein Ereignis auslöst. Alle Steuerungspunkte, die die Ereignisquelle einer Variablen abonniert haben, erhalten dann eine Benachrichtigung über die Änderung. Eine nicht ereignisbezogene Zustandsvariable ist nie in einer Ereignisbenachrichtigung enthalten.

</dd> </dl>
