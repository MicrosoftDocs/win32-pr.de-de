# <a name="imapiv1-support"></a>IMAPIv1-Unterstützung

Die Imagemastering-API Version 1 (IMAPIv1) ermöglicht es einer Anwendung, ein einfaches Audio- oder Datenbild auf CD-R- und CD-RW-Geräten zu verarbeiten. Die API unterstützt die Formate Joliet und ISO 9660 für Redbook-Audio- und Datendatenträger.

> [!Note]  
> DIE IMAPI wurde für Version 2.0 vollständig überarbeitet. Die meisten Funktionen der Version 1.0 sind in Version 2.0 verfügbar. Personen, die Imagemasteranwendungen schreiben oder die Entwicklung neuer Geräte und Formate durchführen, sollten [version 2.0](using-imapi.md) anstelle von Version 1.0 verwenden. Wenn Sie eine Anwendung der Version 1.0 auf Windows Vista ausführen, konvertieren die Schnittstellen der Version 1.0 diese Aufrufe in die entsprechenden Aufrufe der Version 2.0, um die gleiche Funktionalität zu erreichen.

 

Weitere Informationen zur Verwendung von Version 1.0 auf Windows XP finden Sie in den folgenden Themen:

-   [Informationen zur ImageMastering-API](about-the-image-mastering-api.md)
-   [Verwenden der Imagemastering-API](using-the-image-mastering-api.md)
-   [IMAPIv1-Referenz](imapiv1-reference.md)
