# <a name="imapiv1-support"></a>IMAPIv1-Unterstützung

Mit der Image-Mastering-API, Version 1 (IMAPIv1), kann eine Anwendung ein einfaches Audio-oder Daten Image auf CD-R-und CD-RW-Geräten bereitstellen und brennen. Die API unterstützt die Formate "Joliet" und "ISO 9660" für Redbook-Audiodaten und Datenscheiben.

> [!Note]  
> IMAPI wurde für Version 2,0 vollständig umgestaltet. Die meisten Funktionen der Version 1,0 sind in Version 2,0 verfügbar. Anwendungen, die Bild-oder Entwicklungsanwendungen schreiben oder eine neue Geräte-und Formatentwicklung durchführen, werden empfohlen, [Version 2,0](using-imapi.md) anstelle von Version 1,0 zu verwenden. Wenn Sie eine Anwendung der Version 1,0 unter Windows Vista ausführen, werden diese Aufrufe von den Schnittstellen der Version 1,0 in die entsprechenden Aufrufe der Version 2,0 konvertiert, um die gleiche Funktionalität zu erzielen.

 

Weitere Informationen zur Verwendung von Version 1,0 unter Windows XP finden Sie in den folgenden Themen:

-   [Informationen zur Image-Mastering-API](about-the-image-mastering-api.md)
-   [Verwenden der Image-Mastering-API](using-the-image-mastering-api.md)
-   [IMAPIv1-Referenz](imapiv1-reference.md)
