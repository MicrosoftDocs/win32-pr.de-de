# <a name="glossary-imapi"></a>Glossar (IMAPI)

<dl> <dt>

<span id="imapi.imapi2_glossary_ANSI"></span><span id="imapi.imapi2_glossary_ansi"></span><span id="IMAPI.IMAPI2_GLOSSARY_ANSI"></span>**Toleranz**
</dt> <dd>

Der American National Standards Institute.

</dd> <dt>

<span id="imapi.imapi2_glossary_BurnEngine"></span><span id="imapi.imapi2_glossary_burnengine"></span><span id="IMAPI.IMAPI2_GLOSSARY_BURNENGINE"></span>**Burn-Engine**
</dt> <dd>

Eine Komponente, die Daten aus einem bereitgestellten Datenstream liest (z. b. IStream oder IO). Stream) und schreibt diese Daten auf ein CD/DVD-Gerät. Die Komponente muss den verfügbaren Puffer auf dem Laufwerk in Erwägung gezogen, enthält jedoch keine Kenntnisse über das Format der Daten, bereitet das Laufwerk für das Schreiben vor oder informiert das Laufwerk darüber, dass der Schreibvorgang abgeschlossen wurde.

</dd> <dt>

<span id="imapi.imapi2_glossary_CDDVDDevice"></span><span id="imapi.imapi2_glossary_cddvddevice"></span><span id="IMAPI.IMAPI2_GLOSSARY_CDDVDDEVICE"></span>**CD/DVD-Gerät**
</dt> <dd>

Ein Gerät, das sich selbst als "Typ 5"-Gerät meldet. Dies schließt CD-ROM, DVD-ROM und die meisten CD-und DVD-Writer-Laufwerke ein. Darüber hinaus melden sich einige Laufwerke mit Magnetmedien (z. b. der Iomega-REV) als "Typ 5"-Geräte an und werden in Windows als CD-ROM-Geräte angezeigt.

</dd> <dt>

<span id="imapi.imapi2_glossary_CDG"></span><span id="imapi.imapi2_glossary_cdg"></span><span id="IMAPI.IMAPI2_GLOSSARY_CDG"></span>**CD-G** 
</dt> <dd>

Ein Akronym für CD-Grafiken, Technologie exklusiv für Audiocds, die das Hinzufügen von Grafiken mit niedriger Bitrate zu einem Audiotitel ermöglicht. Diese Technologie wird am häufigsten für CDs verwendet, die für Karaoke verwendet werden.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_R"></span><span id="imapi.imapi2_glossary_cd_r"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_R"></span>**CD-R**
</dt> <dd>

Eine CD-Once-CD.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_RandRW"></span><span id="imapi.imapi2_glossary_cd_randrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_RANDRW"></span>**CD-R/RW**
</dt> <dd>

CD-R oder CD-RW Compact Disk

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_ROM"></span><span id="imapi.imapi2_glossary_cd_rom"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_ROM"></span>**CD-ROM**
</dt> <dd>

Eine schreibgeschützte Compact-CD.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_RW"></span><span id="imapi.imapi2_glossary_cd_rw"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_RW"></span>**CD-RW**
</dt> <dd>

Eine reschreibbare Compact-CD.

</dd> <dt>

<span id="imapi.imapi2_glossaryCDText"></span><span id="imapi.imapi2_glossarycdtext"></span><span id="IMAPI.IMAPI2_GLOSSARYCDTEXT"></span>**CD-Text** 
</dt> <dd>

Eine Technologie exklusiv für Audiocds, die das Speichern von Informationen zu niedrigen Bitraten, wie z. b. dem Namen des Künstlers und des Namens, auf den physischen Medien ermöglicht. Anwendungen oder Geräte, die die resultierende CD wiedergeben, können dann diese Informationen, sofern vorhanden, abrufen, damit Sie dem Benutzer angezeigt werden.

</dd> <dt>

<span id="imapi.imapi2_glossary_Disc"></span><span id="imapi.imapi2_glossary_disc"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISC"></span>**Discs**
</dt> <dd>

Ein generischer Begriff für das Runde optische Medium, z. b. eine CD-ROM oder eine DVD.

</dd> <dt>

<span id="imapi.imapi2_glossary_DiscAtOnce"></span><span id="imapi.imapi2_glossary_discatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISCATONCE"></span>**CD-at-Once** 
</dt> <dd>

Eine Methode zum Schreiben auf ein CD-Medium, bei der der gesamte Inhalt des Burn-Abbilds auf die Festplatte in einem kontinuierlichen Stream geschrieben wird. Außerdem sind nicht barrierefreie Daten, wie z. b. "Lead-in" und "Leadout", im Burn-Image enthalten.

</dd> <dt>

<span id="imapi.imapi2_glossary_Disk"></span><span id="imapi.imapi2_glossary_disk"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISK"></span>**Diskette**
</dt> <dd>

Ein generischer Begriff für das Round-Magnet Medium, z. b. eine Diskette oder eine Festplatte.

</dd> <dt>

<span id="imapi.imapi2_glossary_Dispatch"></span><span id="imapi.imapi2_glossary_dispatch"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISPATCH"></span>**Übermittlung** 
</dt> <dd>

Eine COM-Schnittstellen Bezeichnung, bei der eine bestimmte Schnittstelle die **IDispatch** -Methode zum Aufrufen einer Funktion unterstützt. Weitere Informationen zu diesem erweiterten com-Thema finden Sie in der MSDN-Website.

</dd> <dt>

<span id="imapi.imapi2_glossary_Dual"></span><span id="imapi.imapi2_glossary_dual"></span><span id="IMAPI.IMAPI2_GLOSSARY_DUAL"></span>**Ales** 
</dt> <dd>

Eine COM-Schnittstellen Bezeichnung, bei der eine bestimmte Schnittstelle sowohl die Vtable-als auch die **IDispatch** -Methode zum Aufrufen einer Funktion unterstützt. Weitere Informationen zu diesen erweiterten com-Themen finden Sie in der MSDN-Website.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVD"></span><span id="imapi.imapi2_glossary_dvd"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVD"></span>**DVD**
</dt> <dd>

Eine digitale, vielseitige CD. ursprünglich bedeutete diese Abkürzung "Digital Video Disk".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusR"></span><span id="imapi.imapi2_glossary_dvdplusr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSR"></span>**DVD + R**
</dt> <dd>

Eine Write-Once-DVD. Das Akronym hat den Wert "DVD plus R".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRDL"></span><span id="imapi.imapi2_glossary_dvdplusrdl"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRDL"></span>**DVD + R-DL**
</dt> <dd>

Eine Dual-Layer-DVD + R-CD. Diese Art von Disc wird auch als "DVD + R Double Layer" bezeichnet.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRRW"></span><span id="imapi.imapi2_glossary_dvdplusrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRRW"></span>**DVD + R/RW**
</dt> <dd>

Eine DVD + R-CD und eine DVD + RW-CD.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRW"></span><span id="imapi.imapi2_glossary_dvdplusrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRW"></span>**DVD + RW**
</dt> <dd>

Eine neu beschreibbare DVD. Das Akronym ist als "DVD plus RW" ausgesprochen.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashR"></span><span id="imapi.imapi2_glossary_dvdplusdashr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHR"></span>**DVD ± R**
</dt> <dd>

Eine DVD + r-CD und eine DVD-r-CD.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashRRW"></span><span id="imapi.imapi2_glossary_dvdplusdashrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHRRW"></span>**DVD ± R/RW**
</dt> <dd>

Eine DVD + r-CD, eine DVD-r-CD, eine DVD + RW-CD und eine DVD-RW-CD.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashRW"></span><span id="imapi.imapi2_glossary_dvdplusdashrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHRW"></span>**DVD ± RW**
</dt> <dd>

Eine DVD + RW-CD und DVD + R-Medien

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashR"></span><span id="imapi.imapi2_glossary_dvddashr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHR"></span>**DVD-R**
</dt> <dd>

Eine Write-Once-DVD. Das Akronym hat den Wert "DVD Dash R".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRDL"></span><span id="imapi.imapi2_glossary_dvddashrdl"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRDL"></span>**DVD-R-DL**
</dt> <dd>

Ein Dual-Layer-DVD-R-Medium.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDRAM"></span><span id="imapi.imapi2_glossary_dvdram"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDRAM"></span>**DVD-RAM**
</dt> <dd>

Eine nach dem Zufallsprinzip beschreibbare DVD. Dieser Medientyp unterstützt die Hardware Mängel Verwaltung, mit der beschädigte Spuren identifiziert und vermieden werden. Laufwerke, die dieses Format unterstützen, sind in Japan häufiger als in den USA.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDROM"></span><span id="imapi.imapi2_glossary_dvdrom"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDROM"></span>**DVD-ROM**
</dt> <dd>

Eine schreibgeschützte DVD.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRRW"></span><span id="imapi.imapi2_glossary_dvddashrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRRW"></span>**DVD-R/RW**
</dt> <dd>

Eine DVD-R-CD und DVD-RW-CD.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRW"></span><span id="imapi.imapi2_glossary_dvddashrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRW"></span>**DVD-RW**
</dt> <dd>

Eine neu beschreibbare DVD. Das Akronym ist als "DVD Dash RW" ausgesprochen.

</dd> <dt>

<span id="imapi.imapi2_glossary_ECMA"></span><span id="imapi.imapi2_glossary_ecma"></span><span id="IMAPI.IMAPI2_GLOSSARY_ECMA"></span>**ECMA** 
</dt> <dd>

Die Zuordnung von Europäischen Computer Herstellern ist eine Branchen Verknüpfung, die in 1961 mit der Standardisierung von Informations-und Kommunikationssystemen begründet ist. ECMA bearbeitet Standards und technische Berichte. Alle ECMA-Veröffentlichungen sind kostenlos verfügbar.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPI"></span><span id="imapi.imapi2_glossary_imapi"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPI"></span>**IMAPI** 
</dt> <dd>

Die Image-Mastering-API, die zum Erstellen und Schreiben von verwalteten Images (im Gegensatz zu Live-FS) für CD/DVD-Geräte verwendet wird. Dieser Begriff gilt generisch für alle Versionen der Image-Mastering-API.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPIv1"></span><span id="imapi.imapi2_glossary_imapiv1"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPIV1"></span>**IMAPIv1** 
</dt> <dd>

Die Version von IMAPI, die zuerst in Windows XP verfügbar gemacht wurde. IMAPI 1,0 unterstützt CD-R-und CD-RW-Medien.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPIv2"></span><span id="imapi.imapi2_glossary_imapiv2"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPIV2"></span>**IMAPIv2** 
</dt> <dd>

Die zweite Version von IMAPI, die mit Windows Vista veröffentlicht wurde.

</dd> <dt>

<span id="imapi.imapi2_glossary_INCITS"></span><span id="imapi.imapi2_glossary_incits"></span><span id="IMAPI.IMAPI2_GLOSSARY_INCITS"></span>**INCITS**
</dt> <dd>

Das internationale Committee für Informationstechnologie Standards, eine von ANSI genehmigte und genehmigte Organisation.

</dd> <dt>

<span id="imapi.imapi2_glossary_ISO9660"></span><span id="imapi.imapi2_glossary_iso9660"></span><span id="IMAPI.IMAPI2_GLOSSARY_ISO9660"></span>**ISO9660**
</dt> <dd>

Ein Dateisystem, das für die erste Generierung von CD-Medien entwickelt wurde. In diesem Dateisystem sind wichtige Dateinamen-und Sortierungs Einschränkungen zu beachten.

</dd> <dt>

<span id="imapi.imapi2_glossary_ISOImage"></span><span id="imapi.imapi2_glossary_isoimage"></span><span id="IMAPI.IMAPI2_GLOSSARY_ISOIMAGE"></span>**ISO-Abbild**
</dt> <dd>

Ein allgemeiner Name für ein sektorbasiertes Image des gesamten Datenbereichs einer Daten-CD oder DVD-Diskette. Der Name wurde von der allgemeinen Verwendung von ISO9660 als Dateisystem für CD-Medien vergrößert.

</dd> <dt>

<span id="imapi.imapi2_glossary_Joliet"></span><span id="imapi.imapi2_glossary_joliet"></span><span id="IMAPI.IMAPI2_GLOSSARY_JOLIET"></span>**Joliet**
</dt> <dd>

Ein Satz standardisierter Erweiterungen für ISO9660, der Unicode-Dateinamen mit bis zu 64 Zeichen unterstützt. Weitere Informationen finden Sie in der ECMA-Spezifikations Nummer 119.

</dd> <dt>

<span id="imapi.imapi2_glossary_MMC"></span><span id="imapi.imapi2_glossary_mmc"></span><span id="IMAPI.IMAPI2_GLOSSARY_MMC"></span>**MMC**
</dt> <dd>

Der Multimedia-Befehlssatz, der von der T10-Arbeitsgruppe von Einschnitten definiert wird. Die MMC definiert Paket Befehle für CD-und DVD-Geräte. Die aktuellen funktionierenden Entwürfe der MMC finden Sie möglicherweise auf der T10-Website.

</dd> <dt>

<span id="imapi.imapi2_glossary_MtFuji"></span><span id="imapi.imapi2_glossary_mtfuji"></span><span id="IMAPI.IMAPI2_GLOSSARY_MTFUJI"></span>**Mt. Fuji**
</dt> <dd>

Eine Reihe von Befehlen zum Steuern von Multimedia-Geräten, ähnlich wie der MMC-Befehlssatz. Einige Elemente (z. b. Features oder Medientypen) werden im MT beschrieben. Fuji-Dokumente, während andere zuerst in den MMC-Dokumenten beschrieben werden. Aktuelle Arbeits Entwürfe des Befehlssatzes finden Sie unter [FTP://FTP.AVC-Pioneer.com](ftp://ftp.avc-pioneer.com/).

</dd> <dt>

<span id="imapi.imapi2_glossary_SCSI"></span><span id="imapi.imapi2_glossary_scsi"></span><span id="IMAPI.IMAPI2_GLOSSARY_SCSI"></span>**SCSI**
</dt> <dd>

Die System Schnittstelle des kleinen Computers.

</dd> <dt>

<span id="imapi.imapi2_glossary_SCSICommand"></span><span id="imapi.imapi2_glossary_scsicommand"></span><span id="IMAPI.IMAPI2_GLOSSARY_SCSICOMMAND"></span>**SCSI-Befehl**
</dt> <dd>

Eine beliebige Anzahl von Paket Befehlen, z. b. solche, die im MMC-Dokument definiert sind. Diese Befehle können mithilfe eines beliebigen geeigneten Transportmechanismus, z. b. SCSI, ATAPI, USB oder IEEE1394, gesendet werden.

</dd> <dt>

<span id="imapi.imapi2_glossary_SessonAtOnce"></span><span id="imapi.imapi2_glossary_sessonatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_SESSONATONCE"></span>**Sitzung-at-Once**
</dt> <dd>

Eine Methode zum Erstellen einer CD Plus CD. Eine Schreib Sitzung zeichnet Audiospuren auf der Festplatte auf, und eine zweite Sitzung schreibt einen einzelnen CD-XA-Track auf die Festplatte und schließt die Festplatte.

</dd> <dt>

<span id="imapi.imapi2_glossary_TrackAtOnce"></span><span id="imapi.imapi2_glossary_trackatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_TRACKATONCE"></span>**Track-at-Once**
</dt> <dd>

Eine Methode zum Schreiben auf ein CD-Medium, bei der die Daten für einen einzelnen Track gleichzeitig geschrieben werden. Wenn alle Spuren geschrieben sind, wird die Sitzung als abschließender Schritt geschlossen, der das Schreiben der führenden und der führenden Bereiche der Festplatte umfasst.

</dd> <dt>

<span id="imapi.imapi2_glossary_UDF"></span><span id="imapi.imapi2_glossary_udf"></span><span id="IMAPI.IMAPI2_GLOSSARY_UDF"></span>**Universal Disk Format (UDF)**
</dt> <dd>

Dieses Dateisystem wird von OSTA (der optischen Speichertechnologie Zuordnung) definiert, die Datenträger mit einer einzelnen Sitzung und mehreren Sitzungen unterstützt.

</dd> <dt>

<span id="imapi.imapi2_glossary_VTable"></span><span id="imapi.imapi2_glossary_vtable"></span><span id="IMAPI.IMAPI2_GLOSSARY_VTABLE"></span>**Vtable** 
</dt> <dd>

Eine COM-Schnittstellen Bezeichnung, bei der eine bestimmte Schnittstelle die Vtable-Methode zum Aufrufen einer Funktion unterstützt. Weitere Informationen zu diesem erweiterten com-Thema finden Sie in der MSDN-Website.

</dd> </dl>
