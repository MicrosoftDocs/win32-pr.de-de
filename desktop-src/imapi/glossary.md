# <a name="glossary-imapi"></a>Glossar (IMAPI)

<dl> <dt>

<span id="imapi.imapi2_glossary_ANSI"></span><span id="imapi.imapi2_glossary_ansi"></span><span id="IMAPI.IMAPI2_GLOSSARY_ANSI"></span>**Ansi**
</dt> <dd>

Die American National Standards Institute.

</dd> <dt>

<span id="imapi.imapi2_glossary_BurnEngine"></span><span id="imapi.imapi2_glossary_burnengine"></span><span id="IMAPI.IMAPI2_GLOSSARY_BURNENGINE"></span>**Burn-Engine**
</dt> <dd>

Eine Komponente, die Daten aus einem bereitgestellten Datenstrom liest (z. B. IStream oder E/A). Streamen) und schreibt diese Daten auf ein CD/DVD-Gerät. Die Komponente muss den verfügbaren Puffer auf dem Laufwerk berücksichtigen, enthält jedoch keine Kenntnisse über das Format der Daten, bereitet das Laufwerk für das Schreiben vor oder informiert das Laufwerk darüber, dass der Schreibvorgang abgeschlossen wurde.

</dd> <dt>

<span id="imapi.imapi2_glossary_CDDVDDevice"></span><span id="imapi.imapi2_glossary_cddvddevice"></span><span id="IMAPI.IMAPI2_GLOSSARY_CDDVDDEVICE"></span>**CD/DVD-Gerät**
</dt> <dd>

Ein Gerät, das sich selbst als "Typ 5"-Gerät meldet. Dazu gehören CD-ROM, DVD-ROM und die meisten CD- und DVD-Writer-Laufwerke. Darüber hinaus melden sich einige Laufwerke mit magnetischen Medien (z. B. IOMEGA REV) selbst als "Typ 5"-Geräte und werden als CD-ROM-Geräte in Windows.

</dd> <dt>

<span id="imapi.imapi2_glossary_CDG"></span><span id="imapi.imapi2_glossary_cdg"></span><span id="IMAPI.IMAPI2_GLOSSARY_CDG"></span>**CD-G** 
</dt> <dd>

Ein Akronym für CD Graphics, eine Technologie, die ausschließlich Audio-CDs verwendet, mit der Grafiken mit niedriger Bitrate zu einer Audiospur hinzugefügt werden können. Diese Technologie wird am häufigsten auf CDs verwendet, die für Gegensen verwendet werden.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_R"></span><span id="imapi.imapi2_glossary_cd_r"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_R"></span>**CD-R**
</dt> <dd>

Ein Write-Once-Datenträger.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_RandRW"></span><span id="imapi.imapi2_glossary_cd_randrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_RANDRW"></span>**CD-R/RW**
</dt> <dd>

Ein CD-R- oder CD-RW-Datenträger.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_ROM"></span><span id="imapi.imapi2_glossary_cd_rom"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_ROM"></span>**CD-ROM**
</dt> <dd>

Eine schreibgeschützte Compact Disk.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_RW"></span><span id="imapi.imapi2_glossary_cd_rw"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_RW"></span>**CD-RW**
</dt> <dd>

Ein wiederbeschreibbarer Datenträger.

</dd> <dt>

<span id="imapi.imapi2_glossaryCDText"></span><span id="imapi.imapi2_glossarycdtext"></span><span id="IMAPI.IMAPI2_GLOSSARYCDTEXT"></span>**CD-Text** 
</dt> <dd>

Eine Technologie, die für Audio-CDs exklusiv ist, die die Speicherung von Informationen mit niedriger Bitrate ermöglicht, z. B. Name des Interpreten und Musiknamens, auf den physischen Medien. Anwendungen oder Geräte, die die resultierende CD wieder geben, können diese Informationen dann abrufen, sofern vorhanden, um sie dem Benutzer anzuzeigen.

</dd> <dt>

<span id="imapi.imapi2_glossary_Disc"></span><span id="imapi.imapi2_glossary_disc"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISC"></span>**Disc**
</dt> <dd>

Ein generischer Begriff für runde optische Medien, z. B. CD-ROM oder DVD.

</dd> <dt>

<span id="imapi.imapi2_glossary_DiscAtOnce"></span><span id="imapi.imapi2_glossary_discatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISCATONCE"></span>**Datenträger auf einmal** 
</dt> <dd>

Eine Methode zum Schreiben auf CD-Medien, bei der der gesamte Inhalt des Brandbilds in einem fortlaufenden Stream auf den Datenträger geschrieben wird. Darüber hinaus sind nicht zugängliche Daten, z. B. lead-in und leadout, im Brandbild enthalten.

</dd> <dt>

<span id="imapi.imapi2_glossary_Disk"></span><span id="imapi.imapi2_glossary_disk"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISK"></span>**Datenträger**
</dt> <dd>

Ein generischer Begriff für runde magnetische Medien, z. B. eine Diskette oder eine Festplatte.

</dd> <dt>

<span id="imapi.imapi2_glossary_Dispatch"></span><span id="imapi.imapi2_glossary_dispatch"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISPATCH"></span>**Versand** 
</dt> <dd>

Eine COM-Schnittstellenbezeichnung, bei der eine bestimmte Schnittstelle die **IDispatch-Methode** zum Aufrufen einer Funktion unterstützt. Weitere Informationen zu diesem erweiterten COM-Thema finden Sie unter MSDN.

</dd> <dt>

<span id="imapi.imapi2_glossary_Dual"></span><span id="imapi.imapi2_glossary_dual"></span><span id="IMAPI.IMAPI2_GLOSSARY_DUAL"></span>**Dual** 
</dt> <dd>

Eine COM-Schnittstellenbezeichnung, bei der eine bestimmte Schnittstelle sowohl die VTable- als auch die **IDispatch-Methode** zum Aufrufen einer Funktion unterstützt. Weitere Informationen zu diesen erweiterten COM-Themen finden Sie unter MSDN.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVD"></span><span id="imapi.imapi2_glossary_dvd"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVD"></span>**Dvd**
</dt> <dd>

Digital Versatile Disc. Ursprünglich bedeutete diese Abkürzung "Digital Video Disc".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusR"></span><span id="imapi.imapi2_glossary_dvdplusr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSR"></span>**DVD+R**
</dt> <dd>

Eine Write-Once-DVD. Das Akronym wird als "DVD Plus R" ausgesprochen.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRDL"></span><span id="imapi.imapi2_glossary_dvdplusrdl"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRDL"></span>**DVD+R DL**
</dt> <dd>

Ein DVD+R-Datenträger mit dualer Ebene. Diese Art von Datenträger wird auch als "DVD+R Double Layer" bezeichnet.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRRW"></span><span id="imapi.imapi2_glossary_dvdplusrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRRW"></span>**DVD+R/RW**
</dt> <dd>

Ein DVD+R-Datenträger und ein DVD+RW-Datenträger.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRW"></span><span id="imapi.imapi2_glossary_dvdplusrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRW"></span>**DVD+RW**
</dt> <dd>

Eine wiederbeschreibbare DVD. Das Akronym wird als "DVD Plus RW" ausgesprochen.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashR"></span><span id="imapi.imapi2_glossary_dvdplusdashr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHR"></span>**DVD±R**
</dt> <dd>

Ein DVD+R-Datenträger und ein DVD-R-Datenträger.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashRRW"></span><span id="imapi.imapi2_glossary_dvdplusdashrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHRRW"></span>**DVD±R/RW**
</dt> <dd>

Dvd+R-Datenträger, DVD-R-Datenträger, DVD+RW-Datenträger und DVD-RW-Datenträger.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashRW"></span><span id="imapi.imapi2_glossary_dvdplusdashrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHRW"></span>**DVD±RW**
</dt> <dd>

DVD+RW-Datenträger und DVD+R-Medien

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashR"></span><span id="imapi.imapi2_glossary_dvddashr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHR"></span>**DVD-R**
</dt> <dd>

Eine Write-Once-DVD. Das Akronym wird als "DVD Dash R" ausgesprochen.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRDL"></span><span id="imapi.imapi2_glossary_dvddashrdl"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRDL"></span>**DVD-R DL**
</dt> <dd>

Ein DVD-R-Medium mit dualer Ebene.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDRAM"></span><span id="imapi.imapi2_glossary_dvdram"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDRAM"></span>**DVD-RAM**
</dt> <dd>

Eine zufällig beschreibbare DVD. Dieser Medientyp unterstützt die Verwaltung von Hardwarefehlern, mit der beschädigte Spuren identifiziert und vermieden werden. Laufwerke, die dieses Format unterstützen, sind in Japan häufiger als in den USA.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDROM"></span><span id="imapi.imapi2_glossary_dvdrom"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDROM"></span>**DVD-ROM**
</dt> <dd>

Eine schreibgeschützte DVD.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRRW"></span><span id="imapi.imapi2_glossary_dvddashrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRRW"></span>**DVD-R/RW**
</dt> <dd>

Ein DVD-R-Datenträger und ein DVD-RW-Datenträger.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRW"></span><span id="imapi.imapi2_glossary_dvddashrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRW"></span>**DVD-RW**
</dt> <dd>

Eine wiederbeschreibbare DVD. Das Akronym wird als "DVD Dash RW" ausgesprochen.

</dd> <dt>

<span id="imapi.imapi2_glossary_ECMA"></span><span id="imapi.imapi2_glossary_ecma"></span><span id="IMAPI.IMAPI2_GLOSSARY_ECMA"></span>**Ecma** 
</dt> <dd>

Die European Computer Manufacturers Association ist ein Branchenverbund aus dem Jahr 1961, der sich der Standardisierung von Informations- und Kommunikationssystemen widmen soll. ECMA bearbeitet Standards und technische Berichte. Alle ECMA-Veröffentlichungen sind kostenlos verfügbar.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPI"></span><span id="imapi.imapi2_glossary_imapi"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPI"></span>**Imapi** 
</dt> <dd>

Die Bildmaster-API, die zum Erstellen und Schreiben von Masterbildern (im Gegensatz zu einem Live-FS) auf CD/DVD-Geräten verwendet wird. Dieser Begriff gilt allgemein für alle Versionen der Bildmaster-API.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPIv1"></span><span id="imapi.imapi2_glossary_imapiv1"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPIV1"></span>**IMAPIv1** 
</dt> <dd>

Die version of IMAPI first exposed in Windows XP. IMAPI 1.0 unterstützt CD-R- und CD-RW-Medien.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPIv2"></span><span id="imapi.imapi2_glossary_imapiv2"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPIV2"></span>**IMAPIv2** 
</dt> <dd>

Die zweite Version von IMAPI, die mit Windows Vista veröffentlicht wurde.

</dd> <dt>

<span id="imapi.imapi2_glossary_INCITS"></span><span id="imapi.imapi2_glossary_incits"></span><span id="IMAPI.IMAPI2_GLOSSARY_INCITS"></span>**INCITS**
</dt> <dd>

Der InterNational-Ausschuss für Informationstechnologiestandards, eine Organisation, die von ANSI anerkannt und genehmigt wurde.

</dd> <dt>

<span id="imapi.imapi2_glossary_ISO9660"></span><span id="imapi.imapi2_glossary_iso9660"></span><span id="IMAPI.IMAPI2_GLOSSARY_ISO9660"></span>**ISO9660**
</dt> <dd>

Ein Dateisystem, das für die erste Generation von CD-Medien konzipiert ist. Für dieses Dateisystem gelten erhebliche Einschränkungen hinsichtlich Dateiname und Sortierung.

</dd> <dt>

<span id="imapi.imapi2_glossary_ISOImage"></span><span id="imapi.imapi2_glossary_isoimage"></span><span id="IMAPI.IMAPI2_GLOSSARY_ISOIMAGE"></span>**ISO-Image**
</dt> <dd>

Ein gängiger Name für ein Sektor-für-Sektor-Image des gesamten Datenbereichs einer Daten-CD oder eines DVD-Datenträgers. Der Name wurde aus der gängigen Verwendung von ISO9660 als Dateisystem für CD-Medien entwickelt.

</dd> <dt>

<span id="imapi.imapi2_glossary_Joliet"></span><span id="imapi.imapi2_glossary_joliet"></span><span id="IMAPI.IMAPI2_GLOSSARY_JOLIET"></span>**Joliet**
</dt> <dd>

Eine Reihe standardisierter Erweiterungen für ISO9660, die UNICODE-Dateinamen mit einer Länge von bis zu 64 Zeichen unterstützen. Weitere Informationen finden Sie unter ECMA-Spezifikationsnummer 119.

</dd> <dt>

<span id="imapi.imapi2_glossary_MMC"></span><span id="imapi.imapi2_glossary_mmc"></span><span id="IMAPI.IMAPI2_GLOSSARY_MMC"></span>**Mmc**
</dt> <dd>

Der Multimedia-Befehlssatz, der von der T10-Arbeitsgruppe von INCITS definiert wird. Die MMC definiert Paketbefehle für CD- und DVD-Geräte. Aktuelle Arbeitsentwürfe der MMC finden Sie auf der T10-Website.

</dd> <dt>

<span id="imapi.imapi2_glossary_MtFuji"></span><span id="imapi.imapi2_glossary_mtfuji"></span><span id="IMAPI.IMAPI2_GLOSSARY_MTFUJI"></span>**Mt. Gegenstück**
</dt> <dd>

Eine Reihe von Befehlen zum Steuern von Multimediageräten, ähnlich dem MMC-Befehlssatz. Einige Elemente (z. B. Features oder Medientypen) werden im Mt beschrieben. Während andere dokumente zuerst in den MMC-Dokumenten beschrieben werden. Aktuelle Arbeitsentwürfe des Befehlssatzes finden Sie unter [ftp://ftp.avc-pioneer.com](ftp://ftp.avc-pioneer.com/).

</dd> <dt>

<span id="imapi.imapi2_glossary_SCSI"></span><span id="imapi.imapi2_glossary_scsi"></span><span id="IMAPI.IMAPI2_GLOSSARY_SCSI"></span>**Scsi**
</dt> <dd>

Die kleine Computersystemschnittstelle.

</dd> <dt>

<span id="imapi.imapi2_glossary_SCSICommand"></span><span id="imapi.imapi2_glossary_scsicommand"></span><span id="IMAPI.IMAPI2_GLOSSARY_SCSICOMMAND"></span>**SCSI-Befehl**
</dt> <dd>

Eine Beliebige Anzahl von Paketbefehlen, z. B. die im MMC-Dokument definierten Befehle. Diese Befehle können mit einem geeigneten Transportmechanismus wie SCSI, ATAPI, USB oder IEEE1394 gesendet werden.

</dd> <dt>

<span id="imapi.imapi2_glossary_SessonAtOnce"></span><span id="imapi.imapi2_glossary_sessonatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_SESSONATONCE"></span>**Sitzungs-at-Once**
</dt> <dd>

Eine Methode zum Erstellen eines CD Plus-Datenträgers. Ein schreibende Sitzung zeichnet Audiospuren auf dem Datenträger auf, und eine zweite Sitzung schreibt eine einzelne CD-XA-Spur auf den Datenträger und schließt den Datenträger.

</dd> <dt>

<span id="imapi.imapi2_glossary_TrackAtOnce"></span><span id="imapi.imapi2_glossary_trackatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_TRACKATONCE"></span>**Track-At-Once**
</dt> <dd>

Eine Methode zum Schreiben auf CD-Medien, bei der die Daten für eine einzelne Spur gleichzeitig geschrieben werden. Wenn alle Spuren geschrieben werden, wird die Sitzung als letzter Schritt geschlossen, der das Schreiben der Lead-In- und Lead-Out-Bereiche des Datenträgers umfasst.

</dd> <dt>

<span id="imapi.imapi2_glossary_UDF"></span><span id="imapi.imapi2_glossary_udf"></span><span id="IMAPI.IMAPI2_GLOSSARY_UDF"></span>**Universelles Datenträgerformat (UDF)**
</dt> <dd>

Dieses Dateisystem wird durch die optische Storage Technology Association (Optical Storage Technology Association) definiert, die Einzelsitzungs- und Mehrfachsitzungs-Datenträger unterstützt.

</dd> <dt>

<span id="imapi.imapi2_glossary_VTable"></span><span id="imapi.imapi2_glossary_vtable"></span><span id="IMAPI.IMAPI2_GLOSSARY_VTABLE"></span>**Vtable** 
</dt> <dd>

Eine COM-Schnittstellenbezeichnung, bei der eine bestimmte Schnittstelle die vtable-Methode zum Aufrufen einer Funktion unterstützt. Weitere Informationen zu diesem erweiterten COM-Thema finden Sie unter MSDN.

</dd> </dl>
