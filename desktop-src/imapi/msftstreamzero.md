# <a name="msftstreamzero-object"></a>Msftstreamzero-Objekt

Verwenden Sie dieses Objekt, um einen schreibgeschützten Datenstrom zu erstellen, dessen Daten mit Nullen initialisiert werden. Sie müssen die **SetSize** -Methode aufrufen, um die angeforderte Größe des Streams festzulegen.

## <a name="members"></a>Member

Das **msftstreamzero** -Objekt definiert keine Member.
