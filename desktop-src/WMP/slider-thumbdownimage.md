# <a name="sliderthumbdownimage"></a>Slider. thumbdownimage

Das **thumbdownimage** -Attribut gibt das Bild an, das den Zustand des Zieh Punkts darstellt, oder ruft es ab.

``` syntax
        elementID.thumbDownImage
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese- **/schreibzeichenfolge** , die den Namen einer Bilddatei enthält.

## <a name="remarks"></a>Bemerkungen

" **Thumbdownimage** " ist optional und ist das Bild, das zum Darstellen des instand Zustands von " **thumbbimage**" verwendet wird. Wenn ein **thumbdownimage** nicht angegeben wird, verwendet der Schieberegler das **thumbbimage**.

Die unterstützten Formate sind BMP, JPG, PNG und GIF (keine animierten GIFs).
