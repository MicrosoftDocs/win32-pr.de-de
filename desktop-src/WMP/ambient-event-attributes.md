# <a name="ambient-event-attributes"></a>Ambient-Ereignis Attribute

Die folgenden Attribute sind für alle Skin-Ereignisse üblich. Der Zugriff erfolgt über das Schlüsselwort " **Event** " innerhalb eines Ereignis Handlers für das-Element.



| Attribut                              | BESCHREIBUNG                                                                                                  |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [altKey](event-altkey.md)             | Ruft einen Wert ab, der angibt, ob die Alt-Taste gedrückt wurde, als das Ereignis aufgetreten ist.                           |
| [gedrückt](event-button.md)             | Ruft einen Wert ab, der angibt, welche Maustasten beim Eintreten des Ereignisses angezeigt wurden.                          |
| [clientX](event-clientx.md)           | Ruft die x-Koordinate des Mauszeigers in Bezug auf den Client Bereich des Anwendungsfensters ab. |
| [clientY](event-clienty.md)           | Ruft die y-Koordinate des Mauszeigers in Bezug auf den Client Bereich des Anwendungsfensters ab. |
| [ctrlKey](event-ctrlkey.md)           | Ruft einen Wert ab, der angibt, ob die STRG-Taste gedrückt wurde, als das Ereignis aufgetreten ist.                          |
| [fromelements](event-fromelement.md)   | Ruft das Element ab, von dem das Ereignis stammt.                                                                   |
| [Keycode](event-keycode.md)           | Ruft den ASCII-Schlüsselcode ab, wenn eine Taste gedrückt wurde, als das Ereignis aufgetreten ist.                                   |
| [OffsetX](event-offsetx.md)           | Ruft die x-Koordinate des Mauszeigers in Bezug auf das Element ab, das das Ereignis auslöst.                |
| [offität](event-offsety.md)           | Ruft die y-Koordinate des Mauszeigers in Bezug auf das Element ab, das das Ereignis auslöst.                |
| [ScreenHeight](event-screenheight.md) | Ruft die Höhe der verfügbaren Bildschirmgröße in Pixel ab.                                                 |
| [Bildschirmbreite](event-screenwidth.md)   | Ruft die Breite der verfügbaren Bildschirmgröße in Pixel ab.                                                  |
| [screenX](event-screenx.md)           | Ruft die absolute x-Koordinate des Mauszeigers in Bezug auf den Bildschirm ab.                         |
| [Screeny](event-screeny.md)           | Ruft die absolute y-Koordinate des Mauszeigers in Bezug auf den Bildschirm ab.                         |
| [shiftKey](event-shiftkey.md)         | Ruft einen Wert ab, der angibt, ob die UMSCHALTTASTE gedrückt wurde, als das Ereignis aufgetreten ist.                         |
| [srcelements](event-srcelement.md)     | Ruft das Element ab, das das Ereignis ausgelöst hat.                                                                  |
| ["Element"](event-toelement.md)       | Ruft das Element ab, zu dem der Tastaturfokus verschoben wurde.                                                           |
| [x](event-x.md)                       | Ruft die x-Koordinate des Mauszeigers in Bezug auf das Anwendungsfenster ab.                      |
| [y](event-y.md)                       | Ruft die y-Koordinate des Mauszeigers in Bezug auf das Anwendungsfenster ab.                      |
