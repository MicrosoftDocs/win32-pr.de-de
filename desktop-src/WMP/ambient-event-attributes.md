# <a name="ambient-event-attributes"></a>Ambient-Ereignisattribute

Die folgenden Attribute sind allen Skinereignissen gemeinsam. Auf sie wird mit dem **Ereignisschlüsselwort** innerhalb eines Ereignishandlers für das Element zugegriffen.



| attribute                              | Beschreibung                                                                                                  |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [altKey](event-altkey.md)             | Ruft einen Wert ab, der angibt, ob die ALT-TASTE beim Ereignis heruntergefahren war.                           |
| [Schaltfläche](event-button.md)             | Ruft einen Wert ab, der angibt, welche Maustasten beim Ereignis heruntergefahren waren.                          |
| [clientX](event-clientx.md)           | Ruft die x-Koordinate des Mauszeigers in Bezug auf den Clientbereich des Anwendungsfensters ab. |
| [clientY](event-clienty.md)           | Ruft die y-Koordinate des Mauszeigers in Bezug auf den Clientbereich des Anwendungsfensters ab. |
| [STRGKEY](event-ctrlkey.md)           | Ruft einen Wert ab, der angibt, ob die STRG-TASTE beim Ereignis nicht gedrückt war.                          |
| [fromElement](event-fromelement.md)   | Ruft das Element ab, aus dem das Ereignis stammt.                                                                   |
| [Schlüsselcode](event-keycode.md)           | Ruft den ASCII-Schlüsselcode ab, wenn beim Ereignis eine Taste gedrückt wurde.                                   |
| [Offsetx](event-offsetx.md)           | Ruft die x-Koordinate des Mauszeigers in Bezug auf das Element ab, das das Ereignis ausspricht.                |
| [Offsety](event-offsety.md)           | Ruft die y-Koordinate des Mauszeigers in Bezug auf das Element ab, das das Ereignis ausspricht.                |
| [screenHeight](event-screenheight.md) | Ruft die Höhe der verfügbaren Bildschirmgröße in Pixel ab.                                                 |
| [screenWidth](event-screenwidth.md)   | Ruft die Breite der verfügbaren Bildschirmgröße in Pixel ab.                                                  |
| [screenX](event-screenx.md)           | Ruft die absolute x-Koordinate des Mauszeigers in Bezug auf den Bildschirm ab.                         |
| [Screeny](event-screeny.md)           | Ruft die absolute y-Koordinate des Mauszeigers in Bezug auf den Bildschirm ab.                         |
| [shiftKey](event-shiftkey.md)         | Ruft einen Wert ab, der angibt, ob die UMSCHALTTASTE beim Ereignis heruntergefahren war.                         |
| [srcElement](event-srcelement.md)     | Ruft das Element ab, das das Ereignis ausgelöst hat.                                                                  |
| [toElement](event-toelement.md)       | Ruft das Element ab, in das der Tastaturfokus verschoben wurde.                                                           |
| [x](event-x.md)                       | Ruft die x-Koordinate des Mauszeigers in Bezug auf das Anwendungsfenster ab.                      |
| [y](event-y.md)                       | Ruft die y-Koordinate des Mauszeigers in Bezug auf das Anwendungsfenster ab.                      |
