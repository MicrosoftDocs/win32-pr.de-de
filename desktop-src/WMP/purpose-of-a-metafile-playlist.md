# <a name="purpose-of-a-metafile-playlist"></a>Zweck einer Metafile-Wiedergabeliste

Der grundlegende Zweck einer Metadatei-Wiedergabeliste besteht darin, das Streaming von Medieninhalten von Browsern zu Windows Media Player umzuleiten, da die meisten Browser versuchen, den Inhalt herunterzuladen, anstatt ihn zu streamen. Eine Metadatei-Wiedergabeliste enthält Informationen, die Windows Media Player zum Empfangen von Streams verwendet, z. b. die Uniform Resource Locator (URL). Die Metadatei-Wiedergabeliste kann den Pfad zu Mediendateien über das Internet, ein Intranet oder nur gespeicherte Medien auf einem Netzwerk oder auf einem lokalen Laufwerk bereitstellen. Wenn die Verarbeitung an Windows Media Player umgeleitet wird, haben Sie eine bessere Kontrolle über die Streamingmedien.

Metadatendateien werden verwendet, um das Streaming zu steuern. Indem Sie verschiedene Mediendateien in einem einzelnen Inhaltsstream kombinieren, können Sie Ihre Präsentation steuern und anpassen.
