# <a name="purpose-of-a-metafile-playlist"></a>Zweck einer Metadateiwiedergabeliste

Der grundlegende Zweck einer Metadateiwiedergabeliste besteht im Umleiten von Streamingmedieninhalten von Browsern zu Windows Media Player da die meisten Browser versuchen, den Inhalt herunterzuladen, anstatt ihn zu streamen. Eine Metadateiwiedergabeliste stellt Informationen Windows Media Player, die zum Empfangen von Streams verwendet werden, z. B. die Uniform Resource Locator (URL). Die Metadateiwiedergabeliste kann den Pfad zu Mediendateien über das Internet, ein Intranet oder nur gespeicherte Medien in einem Netzwerk oder lokalen Laufwerk bereitstellen. Wenn die Verarbeitung an Windows Media Player umgeleitet wird, haben Sie mehr Kontrolle über die Streamingmedien.

Metafile-Wiedergabelisten werden verwendet, um die Streamingerfahrung zu steuern. Indem Sie verschiedene Mediendateien in einem einzelnen Inhaltsstream kombinieren, können Sie ihre Darstellung steuern und anpassen.
