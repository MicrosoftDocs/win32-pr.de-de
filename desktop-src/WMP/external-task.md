# <a name="externaltask"></a>Extern. Aufgabe

> [!Note]  
> In diesem Thema werden die Funktionen beschrieben, die für die Verwendung durch Online-Speicher Die Verwendung dieser Funktion außerhalb des Kontexts eines Online Stores wird nicht unterstützt.

 

Die **Task** -Eigenschaft ruft den Namen des aktuellen Aufgabenbereichs ab.

## <a name="syntax"></a>Syntax

``` syntax
window.external.task
```

## <a name="possible-values"></a>Mögliche Werte

Diese Eigenschaft ist eine schreibgeschützte **Zeichenfolge**. Mögliche Werte sind Durchsuchen, brennen und synchronisieren.
