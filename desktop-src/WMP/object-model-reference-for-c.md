# <a name="object-model-reference-for-c"></a>Objektmodell Referenz für C++

> [!Note]  
> Die Verwendung des Windows Media Player ActiveX-Steuer Elements mit C++ wird für Windows Media Player 9-Serie oder höher unterstützt.

 

Die Objektmodell Referenz für C++ bietet eine Dokumentation für Windows Media Player-und Windows Media Player Mobile-Schnittstellen und-Enumerationstypen.

Wenn Sie Remoting oder Skins mit dem Windows Media Player-Steuerelement verwenden möchten, müssen Sie sich auch mit den Schnittstellen **iwmpplayerservices** und **iwmpremotemediaservices** vertraut machen, die für die Verwendung mit C++ entworfen wurden.

Eine Demonstration der Einbettung des Windows Media Player-Steuer Elements in ein C++-Programm finden Sie in den C++ Samples, die in diesem SDK enthalten sind.

In den folgenden Abschnitten werden die C++-Schnittstellen und Enumerationen beschrieben, die vom Windows Media Player-Objektmodell bereitgestellt werden.



| `Section`                                    | BESCHREIBUNG                                                                                                     |
|--------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [Schnittstellen](interfaces.md)               | Listet die für C++-Programmierer verfügbaren Schnittstellen und Methoden auf, die das Windows Media Player ActiveX-Steuerelement verwenden. |
| [Enumerationstypen](enumeration-types.md) | Listet die in C++-Code verfügbaren Enumerationstypen auf.                                                              |
