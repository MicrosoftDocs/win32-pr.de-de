# <a name="object-model-reference-for-c"></a>Objektmodellreferenz für C++

> [!Note]  
> Die Verwendung des Windows Media Player ActiveX-Steuerelements mit C++ wird für Windows Media Player 9er Serie oder höher unterstützt.

 

Die Objektmodellreferenz für C++ bietet Dokumentation für Windows Media Player und Windows Media Player Mobile-Schnittstellen und -Enumerationstypen.

Wenn Sie Remoting oder Skins mit dem Windows Media Player-Steuerelement verwenden möchten, müssen Sie auch mehr über die **Schnittstellen IWMPPlayerServices** und **IWMPRemoteMediaServices** erfahren, die für die Verwendung mit C++ konzipiert sind.

Eine Demonstration des Einbettens des Windows Media Player-Steuerelements in ein C++-Programm finden Sie in den C++-Beispielen, die in diesem SDK enthalten sind.

In den folgenden Abschnitten werden die C++-Schnittstellen und -Enumerationen dokumentiert, die vom Windows Media Player-Objektmodell bereitgestellt werden.



| `Section`                                    | BESCHREIBUNG                                                                                                     |
|--------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [Schnittstellen](interfaces.md)               | Listet die Schnittstellen und Methoden auf, die C++-Programmierern zur Verfügung stehen, die das Windows Media Player ActiveX-Steuerelement verwenden. |
| [Enumerationstypen](enumeration-types.md) | Listet die im C++-Code verfügbaren Enumerationstypen auf.                                                              |
