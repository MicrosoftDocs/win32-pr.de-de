# <a name="listboxfocusitem"></a>LISTBOX.focusItem

Das **focusItem-Attribut** gibt die Zeile an, die den Fokus enthält, oder ruft sie ab.

``` syntax
        elementID.focusItem
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/Schreibnummer (**long**). 

## <a name="remarks"></a>Hinweise

Das Fokuselement und das ausgewählte Element können sich unterscheiden. Ein Element kann über ein Skript ausgewählt werden, aber ein anderes Element kann durch skript den Fokus erhalten. Das Element mit dem Fokus enthält ein gepunktete Rechteck und wird ausgewählt, wenn die EINGABETASTE gedrückt wird.
