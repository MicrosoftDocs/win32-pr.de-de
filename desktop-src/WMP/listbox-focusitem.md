# <a name="listboxfocusitem"></a>ListBox. focusitem

Das **focusitem** -Attribut gibt die Zeile mit dem Fokus an oder ruft Sie ab.

``` syntax
        elementID.focusItem
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/schreibzahl (**Long**). 

## <a name="remarks"></a>Bemerkungen

Das Fokus Element und das ausgewählte Element können unterschiedlich sein. Ein Element kann über ein Skript ausgewählt werden, aber einem anderen Element kann der Fokus über das Skript zugewiesen werden. Das Element mit dem Fokus verfügt über ein gepunktetes Rechteck, das ausgewählt wird, wenn die EINGABETASTE gedrückt wird.
