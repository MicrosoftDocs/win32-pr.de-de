# <a name="customsliderdownimage"></a>Customslider. downImage

Das **downImage** -Attribut gibt das Image des benutzerdefinierten Schiebereglers an oder ruft es ab.

``` syntax
        elementID.downImage
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese- **/schreibzeichenfolge** , die den Namen einer Bilddatei enthält.

## <a name="remarks"></a>Bemerkungen

Dieses Attribut ist optional. Wenn kein Wert angegeben ist, wird die im **Image** -Attribut angegebene Datei verwendet.

Das **Fenster Mage** stellt den Down-Zustand des **customslider** -Steuer Elements dar. Dabei kann es sich um ein einzelnes Bild oder eine Reihe von Bildern handeln, die die verschiedenen Zustände des Schiebereglers darstellen. Wenn mehrere Images verwendet werden, werden Sie auf die gleiche Weise wie die **Bilddatei** angeordnet.

Die unterstützten Bild Dateitypen sind BMP, JPG, PNG und GIF (keine animierten GIFs).
