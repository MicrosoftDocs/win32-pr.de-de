# <a name="other-skin-samples"></a>Weitere Skin-Beispiele

Um die Beispiele für die gesamten Arbeits eigenen Skins anzuzeigen, klicken Sie auf die Schaltfläche **Start** , zeigen Sie auf **Programme**, dann auf **Windows** Media und dann auf **Windows Media SDK**, und klicken Sie dann auf **Player SDK Samples**. Sie müssen zunächst das Windows Media Player SDK installieren, um die Beispiele zu verwenden.
