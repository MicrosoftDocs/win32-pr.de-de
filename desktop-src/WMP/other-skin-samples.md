# <a name="other-skin-samples"></a>Andere Skinbeispiele

Um Beispiele für vollständige funktionierende Skins anzuzeigen, klicken Sie auf die Schaltfläche **Start,** zeigen Sie dann auf **Programme**, **Windows Medien,** **Windows Media SDK**, und klicken Sie dann auf **Player SDK-Beispiele**. Sie müssen zuerst das Windows Media Player SDK installieren, um die Beispiele verwenden zu können.
