# <a name="other-skin-samples"></a>Andere Skinbeispiele

Klicken Sie auf die Schaltfläche Start, zeigen Sie auf Programme , dann auf **Windows Media**, Windows **Media SDK,** und klicken Sie dann auf Player SDK samples **(Player SDK-Beispiele),** um Beispiele für vollständige Arbeits-Skins zu sehen.   Sie müssen zuerst das Windows Media Player SDK installieren, um die Beispiele verwenden zu können.
