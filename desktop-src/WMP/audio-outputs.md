# <a name="audio-outputs"></a>Audioausgaben

Mit dem Systemsteuerung in Windows XP kann der Benutzer einem bestimmten Audioausgabegerät den Namen DirectSound-Standardgerät zuweisen. In Windows Vista ermöglicht der Systemsteuerung dem Benutzer, jedem der drei Namen (eConsole, eMultimedia und eCommunications) einem Audioausgabegerät zuzuweisen. Diese Namen können alle verschiedenen Geräten zugewiesen werden, oder demselben Gerät können mehrere Namen zugewiesen werden. Die folgende Tabelle zeigt, wie Windows Media Player activeX Windows Media Player Windows Media Player ActiveX-Steuerelement standardaudioausgabegeräte auswählen.



|       &nbsp;                               | Windows XP                                                                                                                                                                                | Windows Vista                                                                                                                                                              |
|--------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Windows Media Player                 | Standardmäßig verwendet der Player das Audiogerät, das als DirectSound-Standardgerät festgelegt ist. Der Player bietet ein Dialogfeld, in dem der Benutzer den Player auf ein anderes Audiogerät umschalten kann. | Standardmäßig verwendet der Player das audio-Gerät, das als eMultimedia festgelegt ist. Der Player bietet ein Dialogfeld, in dem der Benutzer den Player auf ein anderes Audiogerät umschalten kann. |
| Windows Media Player-ActiveX-Steuerelement | Standardmäßig verwendet das Player-Steuerelement das Audiogerät, das als DirectSound-Standardgerät festgelegt ist. Das Audioausgabegerät kann nicht programmgesteuert geändert werden.                                | Standardmäßig verwendet das Player-Steuerelement das Audiogerät, das als eMultimedia festgelegt ist. Das Audioausgabegerät kann nicht programmgesteuert geändert werden.                                |
