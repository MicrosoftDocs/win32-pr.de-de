# <a name="about-windows-media-metafiles"></a>Informationen zu Windows Media-Metadateien

Microsoft Windows Media-Metadateien sind Textdateien, die Informationen über einen Dateistream und seine Präsentation bereitstellen. Windows Media-Metadateien basieren auf der Syntax von Extensible Markup Language (XML) und können im ANSI-oder Unicode-Format (UTF-8) codiert werden. Sie bestehen aus verschiedenen Elementen mit ihren zugeordneten Tags und Attributen. Jedes Element in einer Windows Media-Metadatei definiert eine bestimmte Einstellung oder Aktion in Windows-Media Player. Einige Elemente müssen sich in der Datei relativ zu anderen Elementen an einer bestimmten Position befinden. Einige Elemente verfügen über erforderliche Tags und Attribute, die in der Windows Media-Metadatendatei definiert werden müssen.

Windows Media-Download Pakete bieten eine Möglichkeit zum Kombinieren von Windows Media Player Rahmen, Wiedergabelisten Informationen und Multimedia-Inhalten in einer einzelnen herunterladbaren Datei.
