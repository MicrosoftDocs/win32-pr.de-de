# <a name="about-windows-media-metafiles"></a>Informationen Windows Medienmetadateien

Microsoft Windows Media-Metadateien sind Textdateien, die Informationen zu einem Dateistream und seiner Darstellung bereitstellen. Windows Medienmetadateien basieren auf der EXTENSIBLE MARKUP LANGUAGE-Syntax (XML) und können im ANSI- oder UNICODE-Format (UTF-8) codiert werden. Sie besteht aus verschiedenen Elementen mit ihren zugeordneten Tags und Attributen. Jedes Element in einer Windows Media-Metadatei definiert eine bestimmte Einstellung oder Aktion in Windows Media Player. Einige Elemente müssen sich relativ zu anderen Elementen an einer bestimmten Position in der Datei befinden. Einige Elemente verfügen über erforderliche Tags und Attribute, die in der Medienmetadatei Windows werden müssen.

Windows Mediendownloadpakete bieten eine Möglichkeit, Windows Media Player Rahmen, Wiedergabelisteninformationen und Multimediainhalte in einer einzelnen herunterladbaren Datei zu kombinieren.
