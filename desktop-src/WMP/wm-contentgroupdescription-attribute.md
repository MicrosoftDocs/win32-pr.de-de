# <a name="wmcontentgroupdescription-attribute"></a>WM/ContentGroupDescription-Attribut

Das **WM/ContentGroupDescription-Attribut** ist die Beschreibung der Inhaltsgruppe, bei der es sich um eine Sammlung von Medienelementen handelt, z. B. einem CD-Geschachtelten Satz.

## <a name="applies-to"></a>Gilt für

-   [Audioelemente](audio-item-attributes.md)
-   [Häufig verwendete Windows Mediendateiattribute](commonly-used-windows-media-file-attributes.md)

## <a name="remarks"></a>Hinweise

Dieses Attribut wird sowohl in der Bibliothek als auch in der digitalen Mediendatei gespeichert.

**ContentGroupDescription ist** ein Alias für dieses Attribut.

Die Windows Media Format SDK-Konstante für dieses Attribut ist g \_ wszWMContentGroupDistribution.

Um zu bestimmen, ob Sie den Wert dieses Attributs ändern können, verwenden Sie die [Media.isReadOnlyItem-Methode.](media-isreadonlyitem.md)

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9er Serie oder höher<br/> |
