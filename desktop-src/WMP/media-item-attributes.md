# <a name="media-item-attributes"></a>Medienelementattribute

Wenn Sie sich die Bibliothek in Windows Media Player, werden in der Regel viele Informationen zu einem Medienelement angezeigt. Neben dem Namen einer Audiospur sehen Sie z. B. wahrscheinlich den Namen des Albums, auf dem sich der Titel befindet, die Namen des Interpreten und Composers, das Genre der Musik und so weiter.

Im Windows Media Player-Objektmodell werden diese Metadatenelemente als Attribute bezeichnet. Das Objektmodell enthält Eigenschaften und Methoden, die Sie zum Abfragen von Medienelementen und Wiedergabelisten für Attribute verwenden können. Anschließend können Sie Attributwerte in der Benutzeroberfläche anzeigen, oder Ihr Code kann Aktionen basierend auf dem Wert eines Attributs ausführen.

| [Lesen von Attributwerten](reading-attribute-values.md)                                   | Zeigt die Methoden zum Abrufen des Werts eines Attributs an.                                     |
| [Attribute mit mehreren Werten](attributes-with-multiple-values.md)                     | Zeigt die Methoden zum Abrufen des Werts eines mehrwertigen Attributs an.                         |
