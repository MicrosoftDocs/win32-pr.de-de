# <a name="media-item-attributes"></a>Medien Element Attribute

Wenn Sie sich die Bibliothek in Windows Media Player ansehen, sehen Sie in der Regel eine große Menge an Informationen zu einem Medien Element. Neben dem Namen eines Audiotitels sehen Sie z. b. wahrscheinlich den Namen des Albums, auf dem sich der Titel befindet, die Namen des Künstlers und Composer, das Genre der Musik usw.

Im Windows Media Player-Objektmodell werden diese Metadatenelemente als Attribute bezeichnet. Das Objektmodell enthält Eigenschaften und Methoden, die Sie verwenden können, um Medienelemente und Wiedergabelisten für Attribute abzufragen. Anschließend können Sie Attributwerte in der Benutzeroberfläche anzeigen, oder der Code kann basierend auf dem Wert eines Attributs Aktionen ausführen.
