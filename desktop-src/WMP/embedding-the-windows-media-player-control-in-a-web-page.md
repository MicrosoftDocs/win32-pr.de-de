# <a name="embedding-the-windows-media-player-control-in-a-web-page"></a>Einbetten des Windows Media Player-Steuer Elements in eine Webseite

Wenn Sie das Windows Media Player-Steuerelement in eine Webseite einbetten, können Sie einen grundlegenden Ansatz oder einen ausführlicheren Ansatz verwenden.

Bei der grundlegenden Vorgehensweise verwenden Sie das-Steuerelement, um Hintergrundmusik wiederzugeben oder der Seite eine Videoanzeige hinzuzufügen.

Beim ausführlicheren Ansatz machen Sie die Seite interaktiv, indem Sie Skriptcode hinzufügen, um das Steuerelement durch das Player-Objektmodell zu bearbeiten. Auf diese Weise können Sie eine Benutzeroberfläche Ihres eigenen Entwurfs bereitstellen oder dynamisch auf Skript Befehle oder Ereignisse wie Medien Übergänge reagieren.

Weitere Informationen finden Sie unter [Verwenden des Windows Media Player-Steuer Elements in einer Webseite](using-the-windows-media-player-control-in-a-web-page.md).
