# <a name="embedding-the-windows-media-player-control-in-a-web-page"></a>Einbetten des Windows Media Player-Steuerelements in eine Webseite

Wenn Sie das steuerelement Windows Media Player in eine Webseite einbetten, können Sie einen grundlegenden oder einen aufwendigeren Ansatz verfolgen.

Im grundlegenden Ansatz verwenden Sie das Steuerelement einfach, um Hintergrund musik wieder zu spielen oder ihrer Seite eine Videoanzeige hinzuzufügen.

In der aufwendigeren Vorgehensweise gestalten Sie Ihre Seite interaktiv, indem Sie Skriptcode hinzufügen, um das Steuerelement über das Player-Objektmodell zu bearbeiten. Auf diese Weise können Sie eine Benutzeroberfläche Ihres eigenen Entwurfs bereitstellen oder dynamisch auf Skriptbefehle oder Ereignisse wie Medienübergänge reagieren.

Weitere Informationen finden Sie unter [Verwenden des Windows Media Player-Steuerelements in einer Webseite.](using-the-windows-media-player-control-in-a-web-page.md)
