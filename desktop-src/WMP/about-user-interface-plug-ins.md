# <a name="about-user-interface-plug-ins"></a>Informationen zu Benutzeroberflächen-Plug-ins

Windows Media Player bietet eine Vielzahl von Steuerungs Bereichen, die es dem Benutzer ermöglichen, verschiedene Aspekte des Players zu ändern, wie z. b. Video-und Grafik-equalibeinstellungen. Diese Steuerelemente können die von Windows Media Player bereitgestellte Multimediafunktionalität verbessern. Sie bieten jedoch nicht unbedingt alle Funktionen, nach denen der Benutzer suchen kann.

Skins sind eine Möglichkeit, zusätzliche Funktionen bereitzustellen, aber Sie erfordern, dass der Entwickler die gesamte Benutzeroberfläche (User Interface, UI) neu erstellt. Als Alternative ermöglicht Windows Media Player die Erstellung benutzerdefinierter Benutzeroberflächen-Plug-ins, die im vollständigen Modus des Players angezeigt werden. Diese Funktion wird über eine Programmierschnittstelle bereitgestellt, die den standardmäßigen Richtlinien für Microsoft-Component Object Model (com) folgt. Sie können diese Schnittstelle in Microsoft Visual C++ implementieren, indem Sie den Windows Media Player Plug-in-Assistenten verwenden, um Ihnen den Einstieg zu erleichtern.
