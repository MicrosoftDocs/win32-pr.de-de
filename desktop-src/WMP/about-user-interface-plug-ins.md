# <a name="about-user-interface-plug-ins"></a>Informationen Benutzeroberfläche Plug-Ins

Windows Media Player bietet eine Vielzahl von Steuerelementen, die es dem Benutzer ermöglichen, verschiedene Aspekte des Players zu ändern, z. B. die Einstellungen für Video- und Grafikgleichheit. Diese Kontrollpanels können die multimediale Benutzeroberfläche verbessern, die von Windows Media Player bereitgestellt wird, aber sie bieten nicht unbedingt alle Funktionen, die der Benutzer möglicherweise sucht.

Skins sind eine Möglichkeit, zusätzliche Funktionen zur Verfügung zu stellen, aber der Entwickler muss die gesamte Benutzeroberfläche neu erstellen. Alternativ ermöglicht Windows Media Player die Erstellung benutzerdefinierter Benutzeroberflächen-Plug-Ins, die im vollständigen Modus des Players angezeigt werden. Diese Funktionalität wird über eine Programmierschnittstelle bereitgestellt, die den Standardrichtlinien von Microsoft Component Object Model (COM) entspricht. Sie können diese Schnittstelle in Microsoft Visual C++ implementieren, indem Sie den Windows Media Player Plug-In-Assistenten verwenden, um Ihnen den Einstieg zu unterstützen.
