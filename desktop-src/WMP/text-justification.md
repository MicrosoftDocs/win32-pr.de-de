# <a name="textjustification"></a>TEXT.justification

Das  Begründungsattribut gibt die Ausrichtung des Texts im Text-Steuerelement an oder ruft diese ab.

``` syntax
        elementID.justification
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine **Zeichenfolge** mit Lese-/Schreibzugriff.



| Wert  | Beschreibung                                                   |
|--------|---------------------------------------------------------------|
| Links   | Standard. Richtet den Text links vom Text-Steuerelement aus.     |
| Right  | Richtet den Text rechts neben dem Text-Steuerelement aus.             |
| Zentrum | Richtet den Text an der horizontalen Mitte des Text-Steuerelements aus. |



 

## <a name="remarks"></a>Hinweise

Ein Beispiel, das veranschaulicht, wie die Attribute des **TEXT-Elements** verwendet werden, finden Sie im [Value-Attribut.](text-value.md)

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player Version 7.0 oder höher<br/> |
