# <a name="textjustification"></a>Text. Begründung

Das Attribut " **Begründung** " gibt die Ausrichtung des Texts im Text Steuerelement an oder ruft diese ab.

``` syntax
        elementID.justification
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese- **/schreibzeichenfolge**.



| Wert  | BESCHREIBUNG                                                   |
|--------|---------------------------------------------------------------|
| Links   | Standard. Richtet den Text links neben dem Text Steuerelement aus.     |
| Rechts  | Richtet den Text rechts vom Text Steuerelement aus.             |
| Zentrum | Richtet den Text an der horizontalen Mitte des Text Steuer Elements aus. |



 

## <a name="remarks"></a>Bemerkungen

Im [value](text-value.md) -Attribut finden Sie ein Beispiel, das veranschaulicht, wie die Attribute des **Text** -Elements verwendet werden.
