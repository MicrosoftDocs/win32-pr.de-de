# <a name="playermediacollectionmediaadded-event"></a>Player.MediaCollectionMediaAdded-Ereignis

Das MediaCollectionMediaAdded-Ereignis tritt auf, wenn der lokalen Bibliothek ein Medienelement hinzugefügt wird.

## <a name="syntax"></a>Syntax


```JScript
Player.MediaCollectionMediaAdded(
  pdispMedia
)
```



## <a name="parameters"></a>Parameter

<dl> <dt>

*pdispMedia* 
</dt> <dd>

**Medienobjekt,** das hinzugefügt wurde.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Dieses Ereignis gibt keinen Wert zurück.

## <a name="remarks"></a>Hinweise

Dieses Ereignis tritt nur für die lokale Bibliothek auf.

**Windows Media Player 10 Mobile:** Dieses Ereignis wird nicht unterstützt.
