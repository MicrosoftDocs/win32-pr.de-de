# <a name="playermediacollectionmediaadded-event"></a>Player. mediacollectionmediaadded-Ereignis

Das mediacollectionmediaadded-Ereignis tritt auf, wenn der lokalen Bibliothek ein Medien Element hinzugefügt wird.

## <a name="syntax"></a>Syntax


```JScript
Player.MediaCollectionMediaAdded(
  pdispMedia
)
```



## <a name="parameters"></a>Parameter

<dl> <dt>

*pdispmedia* 
</dt> <dd>

**Medien** Objekt, das hinzugefügt wurde.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Dieses Ereignis gibt keinen Wert zurück.

## <a name="remarks"></a>Bemerkungen

Dieses Ereignis tritt nur für die lokale Bibliothek auf.

**Windows Media Player 10 Mobile:** Dieses Ereignis wird nicht unterstützt.
