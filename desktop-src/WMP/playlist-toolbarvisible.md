# <a name="playlisttoolbarvisible"></a>Wiedergabeliste. ToolbarVisible

Mit dem **ToolbarVisible** -Attribut wird ein Wert angegeben oder abgerufen, der angibt, ob die Wiedergabe Symbolleiste angezeigt wird.

``` syntax
        elementID.toolbarVisible
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist ein **boolescher** Wert mit Lese-/Schreibzugriff. Der Standardwert ist „FALSE“.

## <a name="remarks"></a>Bemerkungen

Die Wiedergabeliste der Wiedergabelisten stellt Benutzeroberflächen Elemente bereit, die die Arbeit mit Wiedergabelisten vereinfachen.
