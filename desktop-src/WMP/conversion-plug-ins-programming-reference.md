# <a name="conversion-plug-ins-programming-reference"></a>Programmierreferenz zu Konvertierungs-Plug-Ins

Das Microsoft Windows Media Player Software Development Kit (SDK) unterstützt eine Schnittstelle für Konvertierungs-Plug-Ins, die ein digitales Mediendateiformat in ein anderes konvertieren. Das SDK gibt auch Anforderungen für die Registrierung von Dateiformaten für die Konvertierung durch ein bestimmtes Plug-In an. In den folgenden Abschnitten werden diese ausführlich dokumentiert.



| `Section`                                                                | BESCHREIBUNG                                                                                                                                                                                           |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Registrieren von Konvertierungs-Plug-Ins](registering-conversion-plug-ins.md) | Beschreibt, wie Digitale Mediendateiformate für die Konvertierung registriert werden.                                                                                                                                  |
| [IWMPConvert-Schnittstelle](/previous-versions/windows/desktop/api/wmpservices/nn-wmpservices-iwmpconvert)                               | Stellt Methoden bereit, um Windows Media Player Konvertierungs-Plug-Ins zu aktivieren, um digitale Mediendateien, die mit technologien erstellt werden, die nicht von Microsoft bereitgestellt werden, in advanced Systems Format (ASF) zu konvertieren. |
