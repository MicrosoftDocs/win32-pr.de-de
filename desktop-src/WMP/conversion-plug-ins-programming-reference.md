# <a name="conversion-plug-ins-programming-reference"></a>Programmier Referenz für Konvertierungs-Plug-ins

Das Microsoft Windows Media Player Software Development Kit (SDK) unterstützt eine Schnittstelle für Konvertierungs-Plug-ins, die ein digitales Mediendatei Format in ein anderes konvertieren. Das SDK gibt auch Anforderungen für das Registrieren von Dateiformaten für die Konvertierung durch ein bestimmtes Plug-in an. In den folgenden Abschnitten werden diese ausführlich beschrieben.



| `Section`                                                                | BESCHREIBUNG                                                                                                                                                                                           |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Konvertieren von Konvertierungs-Plug-ins](registering-conversion-plug-ins.md) | Hier wird beschrieben, wie Sie Digital Media-Dateiformate für die Konvertierung registrieren.                                                                                                                                  |
| [Iwmpconvert-Schnittstelle](/previous-versions/windows/desktop/api/wmpservices/nn-wmpservices-iwmpconvert)                               | Bietet Methoden zum Aktivieren von Windows Media Player-Konvertierungs-Plug-ins zum Konvertieren digitaler Mediendateien, die mithilfe von nicht von Microsoft bereitgestellten Technologien erstellt wurden, in das Advanced Systems-Format (ASF). |
