# <a name="playlistdropdownlist"></a>Wiedergabeliste. Dropdown List

Mit dem **DropDownList** -Attribut wird ein Wert angegeben oder abgerufen, der angibt, welche Elemente im Dropdown-Listenfeld für eine bestimmte Instanz des **Wiedergabe** Listen Elements angezeigt werden.

``` syntax
        elementID.dropDownList
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese- **/schreibzeichenfolge** , die einen der folgenden Werte enthält.



| Wert       | BESCHREIBUNG                                                                                      |
|-------------|--------------------------------------------------------------------------------------------------|
| showalbums  | Zeigt Alben an.                                                                                    |
| ShowAll     | Standard. Zeigt alle verfügbaren Elemente an, einschließlich CD-Wiedergabelisten, Benutzer Wiedergabelisten und Options Voreinstellungen. |
| showcd      | Zeigt die CD-Wiedergabeliste an.                                                                           |
| showclips   | Zeigt alle Clips an.                                                                                 |
| showcurrent | Zeigt die aktuelle, nicht gespeicherte Wiedergabeliste an.                                                             |
| showlibrary | Zeigt nur Bibliotheks Wiedergabelisten an.                                                                    |
| showradio   | Zeigt Options Einstellungen an.                                                                             |
| showqueries | Zeigt Abfragen an.                                                                                   |
