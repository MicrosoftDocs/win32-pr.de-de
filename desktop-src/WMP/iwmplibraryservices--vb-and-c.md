# <a name="iwmplibraryservices-vb-and-c-interface"></a>Iwmplibraryservices (VB und c#)-Schnittstelle

Stellt Methoden zum Aufzählen von Bibliotheken bereit.

## <a name="members"></a>Member

Die **iwmplibraryservices (VB und c#)** -Schnittstelle verfügt über diese Typen von Membern:

-   [Methoden](#methods)

### <a name="methods"></a>Methoden

Die **iwmplibraryservices (VB und c#)** -Schnittstelle verfügt über diese Methoden.



| Methode                                                                                               | BESCHREIBUNG                                                                                                        |
|:-----------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------|
| [**getcountrytbytype**](wmplibiwmplibraryservices-iwmplibraryservices-getcountbytype--vb-and-c.md)     | Gibt die Anzahl der verfügbaren Bibliotheken eines angegebenen Typs zurück.<br/>                                           |
| [**getlibrarybytype**](wmplibiwmplibraryservices-iwmplibraryservices-getlibrarybytype--vb-and-c.md) | Gibt eine **iwmplibrary** -Schnittstelle zurück, die die Bibliothek mit dem angegebenen Typ und Index darstellt.<br/> |



 

Sie erhalten eine **iwmplibraryservices** -Schnittstelle durch Umwandeln des Objekts, das von der **AxWindowsMediaPlayer. gedecx** -Methode zurückgegeben wird.
