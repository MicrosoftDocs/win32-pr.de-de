# <a name="windows-media-player-object-model"></a>Windows Media Player-Objektmodell

Das Microsoft Windows Media Player ActiveX-Steuerelement-Objektmodell stellt Schnittstellen bereit, mit denen Entwickler Webseiten und anderen Programmen Windows Media Player und Windows Media Player Mobile-Funktionalität hinzufügen können. Die Dokumentation zum Windows Media Player ActiveX-Steuerelement Objektmodell wird in den folgenden Abschnitten dargestellt.

> [!Note]  
> Das Windows Media Player Mobile ActiveX-Steuerelement basiert auf dem Windows Media Player ActiveX-Steuerelement und verwendet daher dasselbe Objektmodell. Sofern nicht anders angegeben, gilt jede Dokumentation zum Einbetten des Windows Media Player ActiveX-Steuer Elements auf einer Webseite oder einer Anwendung auch für das Windows Media Player Mobile ActiveX-Steuerelement.
