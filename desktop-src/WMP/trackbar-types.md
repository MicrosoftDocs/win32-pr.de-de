# <a name="trackbar-types"></a>Trackbartypen

Sie müssen keine Schieberegler oder Trackleisten einschließen, aber sie können sehr nützlich sein, damit der Benutzer die Lautstärke und Wiedergabeposition anpassen kann. Es gibt zwei Arten von Trackbars:

-   Seek
-   Volume

Weitere Informationen zu Schiebereglern finden Sie im Abschnitt [Trackbars](trackbars.md) der Skin Reference.
