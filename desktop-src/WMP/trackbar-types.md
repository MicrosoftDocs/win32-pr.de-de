# <a name="trackbar-types"></a>TrackBar-Typen

Sie müssen keine Schieberegler oder trackbars einschließen, aber Sie können sehr nützlich sein, um dem Benutzer das Anpassen des Volumes und der Wiedergabe Position zu ermöglichen. Es gibt zwei Arten von trackbars:

-   Seek
-   Lautstärke

Weitere Informationen zu Schiebereglern finden Sie im Abschnitt " [trackbars](trackbars.md) " der Skin-Referenz.
