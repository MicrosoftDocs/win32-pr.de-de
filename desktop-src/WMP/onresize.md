# <a name="onresize"></a>OnResize

Der **OnResize** -Ereignishandler behandelt ein Ereignis, das auftritt, wenn die Größe eines Steuer Elements geändert wird.

``` syntax
onresize
```

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|-----------------------------------------------------|
| Version<br/> | Windows Media Player, Version 70 oder höher<br/> |
