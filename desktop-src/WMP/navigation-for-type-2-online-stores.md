# <a name="navigation-for-type-2-online-stores"></a>Navigation für Typ 2-Onlineshops

Webseiten, die von Windows Media Player 10 oder höher gehostet werden, z. B. in Aufgabenbereichen des Onlineshopdiensts, können die [External.NavigateTaskPaneURL-Methode](external-navigatetaskpaneurl.md) verwenden, um die Navigation innerhalb der Windows Media Player. Die Navigation kann auf folgende Weise erfolgen:

-   Zwischen Den Aufgabenbereichen des Diensts. Weitere Informationen [finden Sie unter Navigieren zwischen Dienst-Aufgabenbereichen.](navigating-between-service-task-panes.md)
-   Von Features wie Info Center View, HTMLView-Webseiten und der Albuminformationsseite bis zu Service-Aufgabenbereichen. Weitere [Informationen finden Sie unter Navigation über Windows Media Player Features.](navigating-from-windows-media-player-features.md)
-   Zwischen Onlineshops. Weitere [Informationen finden Sie unter Navigieren zu anderen Onlineshops.](navigating-to-other-online-stores.md)
