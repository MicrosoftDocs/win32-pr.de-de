# <a name="navigation-for-type-2-online-stores"></a>Navigation für den Typ 2-Online Speicher

Webseiten, die von Windows Media Player 10 oder höher gehostet werden, z. b. in den Aufgabenbereichen des Online Store-Diensts, können die Methode " [externe. navigatetaskpaneurl](external-navigatetaskpaneurl.md) " verwenden, um die Navigation innerhalb von Windows Media Player zu steuern. Die Navigation kann auf folgende Weise erfolgen:

-   Zwischen Dienstaufgaben Bereichen. Siehe [Navigieren zwischen Dienstaufgaben](navigating-between-service-task-panes.md) Bereichen
-   Über Features wie Info Center-Ansicht, HtmlView-Webseiten und die Seite "Albuminformationen" zu Dienstaufgaben Bereichen. Weitere [Informationen finden Sie unter Navigieren von Windows Media Player-Features](navigating-from-windows-media-player-features.md).
-   Zwischen Online speichern. Weitere Informationen finden [Sie unter Navigieren zu anderen Online Stores](navigating-to-other-online-stores.md).
