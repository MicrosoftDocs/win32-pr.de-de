# <a name="ambientattributesid"></a>AmbientAttributes.id

Das **id-Attribut** gibt den Bezeichner eines Steuerelements an oder ruft den Bezeichner ab. Dieses Attribut unterstützt alle Namen, die von Microsoft-Visual Basic for Applications. Kann nur zur Entwurfszeit festgelegt werden.

``` syntax
        elementID.id
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine **Zeichenfolge,** die zur Entwurfszeit angegeben und danach schreibgeschützt ist. Sie hat den Standardwert .

Unbenannte \_ *Elementtyp* \_ *num*

Wobei *elementtype* der Name des Steuerelementtyps und *num* eine Zahl ist, die der Position des Steuerelements in der Sequenz aller unbenannten Steuerelemente im Design entspricht. Diese Zahl wird für jedes neue Design auf 0 zurückgesetzt.
