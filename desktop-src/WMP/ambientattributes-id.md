# <a name="ambientattributesid"></a>AmbientAttributes.id

Das **ID** -Attribut gibt den Bezeichner eines Steuer Elements an oder ruft ihn ab. Dieses Attribut unterstützt jeden beliebigen Namen, der von Microsoft Visual Basic for Applications zugelassen wird. Kann nur zur Entwurfszeit festgelegt werden.

``` syntax
        elementID.id
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine **Zeichenfolge** , die zur Entwurfszeit angegeben wird, und anschließend schreibgeschützt. Der Standardwert ist

Unbenannter \_ *Element Type* \_ *NUM*

Dabei ist *Element Type* der Name des Steuerelement Typs, und *NUM* ist eine Zahl, die der Position des Steuer Elements in der Sequenz aller unbenannten Steuerelemente im Design entspricht. Diese Zahl wird für jedes neue Design auf 0 (null) zurückgesetzt.
