# <a name="_wmpocxevents-interface"></a>\_Wmpocxevents-Schnittstelle

Die **\_ wmpocxevents** -Schnittstelle bietet dieselben Ereignisse wie die **iwmpevents**-, **IWMPEvents2**-, **IWMPEvents3**-und **IWMPEvents4** -Schnittstellen, erbt aber von **IDispatch** anstelle von **IUnknown**. Auf diese Weise können Sie Windows Media Player-Ereignis Benachrichtigungen über eine **IDispatch** -Ereignis Senke empfangen.
