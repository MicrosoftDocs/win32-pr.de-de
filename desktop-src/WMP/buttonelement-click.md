# <a name="buttonelementclick"></a>ButtonElement. Click

Mit der **Click** -Methode wird der für das **ButtonElement** definierte **OnClick** -Ereignishandler aufgerufen.

``` syntax
        elementID.click()
```

## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.

## <a name="remarks"></a>Bemerkungen

Verwenden Sie diese Methode, um eine alternative Möglichkeit zur Ausführung des Codes bereitzustellen, der einer Schaltfläche zugeordnet ist.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9-Serie oder höher<br/> |
