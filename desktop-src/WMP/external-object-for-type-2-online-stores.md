# <a name="external-object-for-type-2-online-stores"></a>Externes Objekt für Onlinespeicher vom Typ 2

> [!Note]  
> In diesem Abschnitt werden Funktionen beschrieben, die für die Verwendung durch Onlineshops entwickelt wurden. Die Verwendung dieser Funktionalität außerhalb des Kontexts eines Onlineshops wird nicht unterstützt.

 

Das **External-Objekt** kann Funktionen für Webseiten bereitstellen, die von einem Onlineshop vom Typ 2 bereitgestellt und in einem Windows Media Player.

Das **External-Objekt** macht die folgenden Eigenschaften verfügbar.



| Eigenschaft                                                        | Beschreibung                                                                               |
|-----------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [appColorButtonHighlight](external-appcolorbuttonhighlight.md) | Ruft die aktuelle Hervorhebungsfarbe der Schaltfläche für die Windows Media Player ab. |
| [appColorButtonHoverFace](external-appcolorbuttonhoverface.md) | Ruft die aktuelle Schaltflächen-Hoverfarbe für die Windows Media Player ab.     |
| [appColorButtonShadow](external-appcolorbuttonshadow.md)       | Ruft die aktuelle Schaltflächenschattenfarbe für die benutzeroberfläche Windows Media Player ab.    |
| [appColorDark](external-appcolordark.md)                       | Ruft die aktuelle dunkel schattierte Farbe des Windows Media Player ab.       |
| [appColorLight](external-appcolorlight.md)                     | Ruft die aktuelle hell schattierte Farbe des Windows Media Player ab.      |
| [appColorMedium](external-appcolormedium.md)                   | Ruft die aktuelle mittlere schattierte Farbe der Windows Media Player ab.     |
| [DownloadManager](external-downloadmanager.md)                 | Ruft das **DownloadManager-Objekt** ab.                                                 |
| [SelectedTaskPane](external-selectedtaskpane.md)               | Gibt den aktuell ausgewählten Aufgabenbereich an oder ruft sie ab.                                  |
| [version](external-version.md)                                 | Ruft die aktuelle Version der Windows Media Player.                                    |



 

Das **External-Objekt** macht die folgenden Methoden verfügbar.



| Methode                                                  | Beschreibung                                                                          |
|---------------------------------------------------------|--------------------------------------------------------------------------------------|
| [NavigateTaskPaneURL](external-navigatetaskpaneurl.md) | Öffnet eine Webseite im angegebenen Aufgabenbereich und ändert den Fokus auf den angegebenen Bereich. |
| [playMedia (veraltet)](external-playmedia.md)        | Gibt die URL eines digitalen Medienelements an, das wiedergibt.                                   |



 

Das **External-Objekt** macht das folgende Ereignis verfügbar.



| Ereignis                                             | Beschreibung                                                               |
|---------------------------------------------------|---------------------------------------------------------------------------|
| [OnColorChange](external-oncolorchange-event.md) | Tritt ein, wenn sich die Farbe Windows Media Player Benutzeroberfläche ändert. |
