# <a name="customslideronpositionchange"></a>VERALTENLIDER.onPositionChange

Der **onPositionChange-Ereignishandler** behandelt ein Ereignis, das auftritt, wenn sich die Position des Schiebereglers ändert, wenn der Benutzer klickt oder zieht.

``` syntax
onPositionChange
```

## <a name="remarks"></a>Hinweise

Wenn sich die Position des benutzerdefinierten  Schiebereglers ändert, weil das Wertattribut im Skript geändert wird, wird dieses Ereignis nicht ausgelöst. Um diese Möglichkeit zu berücksichtigen, implementieren Sie stattdessen den **Wert \_ onchange-Ereignishandler.**
