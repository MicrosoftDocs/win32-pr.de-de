# <a name="art-file-formats-mobile"></a>Art-Dateiformate (mobil)

Bei Den Grafikdateien für Skins kann es sich entweder um Bitmapbilder (.bmp) oder UM GIF-Bilder (.gif) oder um Bilder vom .gif. JPEG-Bilder (.jpg) und PNG-Bilder (.png) werden ebenfalls unterstützt. Fügen Sie keine Leerzeichen oder Kommas in den Namen Ihrer Artdatei ein.

Wenn Sie Dateien mit einem portablen Gerät synchronisieren, stellen Sie sicher, dass die Synchronisierung die Dateierweiterung nicht ändert oder die Bilddateien in ein anderes Format konvertiert.

Viele Zeichnungsprogramme können zum Erstellen von Bitmaps und GIF-Dateien oder zum Konvertieren von Grafiken in anderen Formaten in Bitmaps oder GIF-Dateien verwendet werden. Sie können z. B. Microsoft Paint, das in Microsoft Windows ist, für einfache Zeichnungen verwenden.

Sie sollten die leeren Bereiche Ihrer Bilder um die Ränder kürzen, um die Dateigröße zu reduzieren. Sie können die Offsets im Abschnitt Bitmaps verwenden, um das Entfernen von Leerzeichen auf der linken und oberen Seite zu kompensieren. Sie können auch die unteren und rechten leeren Ränder der Bilder kürzen, da sie für Messungen nicht benötigt werden. Die einzigen Dateien, die Sie nicht kürzen sollten, sind die Bilder Background, SeekThumb und VolumeThumb.
