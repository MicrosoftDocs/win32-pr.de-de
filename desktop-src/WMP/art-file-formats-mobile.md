# <a name="art-file-formats-mobile"></a>Kunst Dateiformate (mobil)

Kunstdateien für Skins können entweder Bitmap-(BMP) oder GIF-Bilder (GIF) sein. JPEG-(JPG) und PNG-Bilder (PNG) werden ebenfalls unterstützt. Fügen Sie keine Leerzeichen oder Kommas in ihren Dateinamen ein.

Wenn Sie Dateien mit einem tragbaren Gerät synchronisieren, müssen Sie sicherstellen, dass die Dateinamenerweiterung durch die Synchronisierung nicht geändert oder die Bilddateien in ein anderes Format konvertiert werden.

Viele Zeichenprogramme können verwendet werden, um Bitmaps und GIF-Dateien zu erstellen oder um Grafiken in anderen Formaten in Bitmaps oder GIF-Dateien zu konvertieren. Beispielsweise können Sie Microsoft Paint, das in Microsoft Windows verfügbar ist, für einfache Zeichnungen verwenden.

Sie sollten die leeren Bereiche der Bilder um die Ränder kürzen, um die Dateigröße zu verringern. Sie können die Offsets im Abschnitt Bitmaps verwenden, um das Entfernen von leeren Leerzeichen auf der linken und oberen Seite auszugleichen. Sie können auch die unteren und rechten leeren Ränder der Bilder kürzen, da Sie nicht für Messungen benötigt werden. Die einzigen Dateien, die Sie nicht kürzen sollten, sind die Bilder background, seekthumb und volumethumschlag.
