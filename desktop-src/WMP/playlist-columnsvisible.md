# <a name="playlistcolumnsvisible"></a>Wiedergabeliste. columnsvisible

Mit dem **columnsvisible** -Attribut wird ein Wert angegeben oder abgerufen, der angibt, ob Spaltenüberschriften angezeigt werden.

``` syntax
        elementID.columnsVisible
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist ein **boolescher** Wert mit Lese-/Schreibzugriff.



| Wert | BESCHREIBUNG                        |
|-------|------------------------------------|
| true  | Standard. Spaltenüberschriften werden angezeigt. |
| false | Spaltenüberschriften werden nicht angezeigt.      |



 

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player, Version 7,0 oder höher<br/> |
