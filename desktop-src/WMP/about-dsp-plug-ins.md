# <a name="about-dsp-plug-ins"></a>Informationen zu DSP-Plug-ins

Microsoft Windows Media Player stellt eine Architektur für die Verbindung von Add-on-Software bereit, die DSP-Funktionen (Digital Signal Processing) für die Audiodaten oder Videostreams während der Wiedergabe ausführt. Diese Add-on-Programme werden als DSP-Plug-ins bezeichnet.

Die folgenden Abschnitte bieten eine konzeptionelle Übersicht über Windows Media Player DSP-Plug-ins.

|----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [Übersicht über den DSP-Plug-in-Benutzer](dsp-plug-in-user-overview.md)           | Beschreibt DSP-Plug-ins aus der Sicht des Endbenutzers.                                                |
| [Übersicht über den DSP-Plug-in-Entwickler](dsp-plug-in-developer-overview.md) | Enthält allgemeine Informationen, die Sie vor dem Erstellen eines DSP-Plug-Ins für Windows Media Player kennen müssen. |
| [Aufbau eines DSP-Plug-ins](building-a-dsp-plug-in.md)                 | Erläutert die Verwendung des Assistenten für den Windows Media Player-Plug-in zum Erstellen eines Beispiel-DSP-Plug-ins.            |
| [Implementieren des DSP-Codes](implementing-your-dsp-code.md)         | Erläutert Überlegungen zum Erstellen von benutzerdefinierten Audio-und Video-DSP-Plug-Ins für Windows Media Player.    |
