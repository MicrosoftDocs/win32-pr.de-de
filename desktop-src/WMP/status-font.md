# <a name="status-font"></a>Status Schriftart

Sie müssen die Schriftart definieren, die von der Statusanzeige verwendet wird, die Sie verwenden möchten. Die Schriftart wird durch Kommas getrennt durch Kommas definiert, die das Gesicht, die Größe und die Gewichtung der Schriftart darstellen.

**Schriftart Werte**

Jeder Schriftart Name kann verwendet werden, wenn er wahrscheinlich auf dem Computer des Benutzers installiert ist. Wenn keine Schriftart auf dem Computer gefunden wird, wird eine Alternative vom Betriebssystem ausgewählt. In der folgenden Tabelle sind die typegesichter aufgeführt, die normalerweise auf Windows Mobile 2003-basierten Geräten gefunden werden.



| Schriftart       | BESCHREIBUNG              |
|----------------|--------------------------|
| Tahoma         | Eine Sans-Serif-Schriftart.   |
| Lucida Console | Eine quadratische Serif-Schriftart. |



 

**Größen Werte**

Dies ist die Größe der Schriftart in Punkten. Jeder positive ganzzahlige Wert ist gültig, obwohl Zahlen zwischen 10 und 18 empfohlen werden. Größen, die kleiner als 10 sind, sind möglicherweise schwer lesbar, und die Größe oberhalb von 18 lässt nicht genügend Speicherplatz zu, um mehr als ein paar Buchstaben gleichzeitig anzuzeigen.

**Gewichtungswerte**

Die einzigen zulässigen Werte sind in der folgenden Tabelle aufgeführt.



| Wert | BESCHREIBUNG |
|-------|-------------|
| B     | Fett        |
| N     | Normal      |
