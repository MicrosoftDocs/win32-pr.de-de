# <a name="status-font"></a>Schriftart "Status"

Sie müssen die Schriftart definieren, die von der Statusanzeige verwendet werden soll, die Sie verwenden möchten. Die Schriftart wird durch drei Werte definiert, die durch Kommas getrennt sind und das Gesicht, die Größe und die Gewichtung der Schriftart darstellen.

**Schriftartwerte**

Ein beliebiger Schriftartname kann verwendet werden, wenn er wahrscheinlich auf dem Computer des Benutzers installiert ist. Wenn auf dem Computer keine Schriftart gefunden wird, wird vom Betriebssystem eine Alternative ausgewählt. Die folgende Tabelle zeigt die Schriftarten, die normalerweise auf Windows Mobile 2003-basierten Geräten zu finden sind.



| Schriftart       | Beschreibung              |
|----------------|--------------------------|
| Tahoma         | Eine sans-serif-Schriftart.   |
| Lucida Console | Eine quadratische Serifenschrift. |



 

**Größenwerte**

Dies ist die Größe der Schriftart in Punkten. Jeder positive ganzzahlige Wert ist gültig, obwohl Zahlen zwischen 10 und 18 empfohlen werden. Größen, die kleiner als 10 sind, können schwierig zu lesen sein, und Größen über 18 lassen möglicherweise nicht genügend Platz, um mehr als ein paar Buchstaben gleichzeitig anzuzeigen.

**Gewichtungswerte**

Die einzigen zulässigen Werte sind in der folgenden Tabelle aufgeführt.



| Wert | Beschreibung |
|-------|-------------|
| B     | Fett        |
| N     | Normal      |
