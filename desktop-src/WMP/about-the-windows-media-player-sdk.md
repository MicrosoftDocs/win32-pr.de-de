# <a name="about-the-windows-media-player-sdk"></a>Informationen zum Windows Media Player SDK

Das Microsoft Windows Media Player Software Development Kit (SDK) enthält Informationen und Tools, mit denen Sie Windows Media Player und Windows Media Player Mobile anpassen und das Windows Media Player ActiveX-Steuerelement und das Windows Media Player Mobile ActiveX-Steuerelement verwenden können. Diese Dokumentation enthält auch Informationen zur Verwendung von Windows Media-Metadatendateien.

Unterstützung für die Anpassung von Windows Media Player wird von folgenden Stellen bereitgestellt:

-   Windows Media Player Skins. Mit Skins können Sie die Benutzeroberfläche des Players anpassen und ihre Funktionalität mithilfe von XML erweitern.
-   Windows Media Player-Plug-ins. Windows Media Player bietet Unterstützung für Plug-ins, die Visualisierungs Effekte erstellen, die DSP-Aufgaben (Digital Signal Processing) ausführen, die benutzerdefinierte Benutzeroberflächen Elemente zum vollmoduspartor hinzufügen und benutzerdefinierte Datenströme in digitalen Mediendateien renderingdateien mit dem ASF-Dateiformat Rendering.
-   Windows Media Player Online Stores. Online Stores sind kommerzielle Unternehmen, die ihre Dienste in die Windows Media Player-Benutzeroberfläche integrieren, um digitale Medieninhalte zum vermieten oder zum Verkauf anzubieten.

Das Einbetten des Windows Media Player-Steuer Elements wird für eine Vielzahl von Technologien unterstützt, darunter:

-   HTML in Webbrowsern. Browser von Microsoft Internet Explorer, Firefox und Netscape Navigator Version 7,1 werden unterstützt.
-   Programme, die mit dem Microsoft Visual C++® Entwicklungssystem erstellt wurden.
-   Programme, die mit Microsoft Visual Basic® 6,0 erstellt wurden.
-   Programme, die mit dem .NET Framework erstellt wurden, einschließlich Programmen, die in den Programmiersprachen Visual Basic und c# geschrieben wurden.
-   Microsoft Office.

Das Windows Media Player Mobile-Steuerelement wird in den folgenden Technologien unterstützt:

-   HTML in Microsoft Pocket Internet Explorer.
-   Programme, die mit dem Microsoft Embedded Visual C++ 4,0-Entwicklungssystem erstellt wurden.
-   Programme, die auf Active Template Library (ATL) 3,0 oder höher basieren.
