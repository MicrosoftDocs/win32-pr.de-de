# <a name="windows-media-player-conversion-plug-ins"></a>Windows Media Player Konvertierungs-Plug-Ins

Microsoft Windows Media Player bietet eine Architektur für Plug-In-Programme, die Dateiformate für digitale Medien konvertieren.

Die Dokumentation zum Konvertierungs-Plug-In ist in die folgenden Abschnitte unterteilt.



| `Section`                                                                                    | BESCHREIBUNG                                                                                                                                           |
|--------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Informationen zu Konvertierungs-Plug-Ins](about-conversion-plug-ins.md)                                 | Bietet eine Übersicht über die Architektur, die für Konvertierungs-Plug-Ins verwendet wird. Lesen Sie diesen Abschnitt, um sich mit den allgemeinen Konzepten dieser Technologie vertraut zu machen. |
| [Programmierreferenz zu Konvertierungs-Plug-Ins](conversion-plug-ins-programming-reference.md) | Enthält eine ausführliche Referenz für die COM-Schnittstelle, die vom Windows Media Player SDK für Konvertierungs-Plug-Ins unterstützt wird.                                |
