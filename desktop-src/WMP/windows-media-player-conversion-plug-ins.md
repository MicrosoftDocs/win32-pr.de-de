# <a name="windows-media-player-conversion-plug-ins"></a>Windows Media Player-Konvertierungs-Plug-ins

Microsoft Windows Media Player stellt eine Architektur für Plug-in-Programme bereit, mit denen digitale Mediendatei Formate konvertiert werden.

Die Konvertierungs-Plug-in-Dokumentation ist in die folgenden Abschnitte unterteilt.



| `Section`                                                                                    | BESCHREIBUNG                                                                                                                                           |
|--------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Informationen zu Konvertierungs-Plug-ins](about-conversion-plug-ins.md)                                 | Bietet eine Übersicht über die Architektur, die für Konvertierungs-Plug-ins verwendet wird. Lesen Sie diesen Abschnitt, um sich mit den allgemeinen Konzepten dieser Technologie vertraut zu machen. |
| [Programmier Referenz für Konvertierungs-Plug-ins](conversion-plug-ins-programming-reference.md) | Bietet eine ausführliche Referenz für die COM-Schnittstelle, die vom Windows Media Player SDK für Konvertierungs-Plug-Ins unterstützt wird.                                |
