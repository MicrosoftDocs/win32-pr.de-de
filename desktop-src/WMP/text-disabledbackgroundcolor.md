# <a name="textdisabledbackgroundcolor"></a>Text. disabledbackgroundcolor

Das **disabledbackgroundcolor** -Attribut gibt die Hintergrundfarbe an oder ruft Sie ab, die für das Text Steuerelement verwendet wird, wenn es deaktiviert ist.

``` syntax
        elementID.disabledBackgroundColor
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese- **/schreibzeichenfolge** mit einem beliebigen Microsoft Internet Explorer-Farbwert.

## <a name="remarks"></a>Bemerkungen

Wenn **disabledbackgroundcolor** nicht angegeben wird, wird **BackgroundColor** verwendet.

Im [value](text-value.md) -Attribut finden Sie ein Beispiel, das veranschaulicht, wie die Attribute des **Text** -Elements verwendet werden.
