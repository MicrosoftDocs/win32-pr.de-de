# <a name="releasedateyearmonth-attribute"></a>Releasedatasearmonth-Attribut

Das **releasedatasearmonth** -Attribut ist der Jahres-und Monats Teil des Datums der ursprünglichen Version des Elements.

## <a name="applies-to"></a>Gilt für

-   [Audioelemente](audio-item-attributes.md)
-   [CD-Spuren](cd-track-attributes.md)
-   [Video Elemente](video-item-attributes.md)

## <a name="remarks"></a>Bemerkungen

Dieses Attribut wird nur in der Bibliotheks Datenbank (oder im Cache) gespeichert.

Um zu ermitteln, ob Sie den Wert dieses Attributs ändern können, verwenden Sie die [Media. isread onlyitem](media-isreadonlyitem.md) -Methode.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------|
| Version<br/> | Windows Media Player 11<br/> |
