# <a name="creating-the-skin-definition-file"></a>Erstellen der Skin-Definitionsdatei

Die Skin-Definitionsdatei ist eine einfache Textdatei. Sie können Skin-Definitions Dateien in einem beliebigen Text-Editor erstellen, der ASCII-nur-Text-Dateien speichert, z. b. Microsoft Notepad, Microsoft Visual InterDev® oder Microsoft Visual Studio. Da Sie XML, nicht HTML, schreiben, möchten Sie möglicherweise keinen Editor für HTML verwenden, da dadurch zusätzliche Codes hinzugefügt werden können, die Sie nicht benötigen, oder Sie erhalten möglicherweise Fehler, die Sie nicht haben. Wahrscheinlich möchten Sie Word-Prozessoren wie z. b. Microsoft Word vermeiden. Obwohl Word nur-Text-Dateien speichern kann, sollten Sie nicht vergessen, bei jedem Speichern als Text zu speichern.
