# <a name="equalizersettingsgainlevel5"></a>Equalizersettings. gainLevel5

Das **gainLevel5** -Attribut gibt die Gewinn Ebene von Band 5 an oder ruft Sie ab.

``` syntax
        elementID.gainLevel5
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/schreibnummer (**float**) mit einem Wert, der normalerweise zwischen 20 und + 20 liegt.  Der Standardwert ist 0 (null).

## <a name="remarks"></a>Bemerkungen

Mit diesem Attribut wird der Teil des Frequenzbereichs für die Audiofrequenz angepasst, der sich auf 500 Hz konzentriert.

Wenn dieses Attribut nicht angegeben wird, wird der vorherige Wert beibehalten.
