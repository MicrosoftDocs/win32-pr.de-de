# <a name="commonly-used-windows-media-file-attributes"></a>Häufig verwendete Windows Media-Dateiattribute

Windows Media-Dateien können beliebig viele Attribute enthalten, einschließlich benutzerdefinierter Attribute.

Wenn die Datei der Bibliothek hinzugefügt wurde, sind möglicherweise auch zusätzliche Attribute verfügbar, die dem Medientyp zugeordnet sind. Weitere Attribute finden Sie unter [Audioelemente](audio-item-attributes.md) oder [Video Elemente](video-item-attributes.md) .

Weitere Informationen zu den Windows Media-Dateiattributen finden Sie in den [Richtlinien zur Verwendung von Windows Media-Metadaten](/previous-versions/ms867702(v=msdn.10)).

In der folgenden Liste sind häufig verwendete Attribute enthalten.

-   [Author-Attribut](author-attribute.md)
-   [Title-Attribut](title-attribute.md)
-   [WM/albumartist-Attribut](wm-albumartist-attribute.md)
-   [WM-/contentverteilungs-Attribut](wm-contentdistributor-attribute.md)
-   [WM/Genre-Attribut](wm-genre-attribute.md)
-   [WM/MCDI-Attribut](wm-mcdi-attribute.md)
-   [WM/mediaclassprimaryid-Attribut](wm-mediaclassprimaryid-attribute.md)
-   [WM/MediaClassSecondaryID-Attribut](wm-mediaclasssecondaryid-attribute.md)
-   [WM/Provider-Attribut](wm-provider-attribute.md)
-   [WM/tracknumber-Attribut](wm-tracknumber-attribute.md)
