# <a name="playererror"></a>Player.error

Die **error-Eigenschaft** ruft das Error-Objekt ab.

## <a name="syntax"></a>Syntax

*Player* . **Fehler**

## <a name="possible-values"></a>Mögliche Werte

Diese Eigenschaft ist ein schreibgeschütztes Error-Objekt.
