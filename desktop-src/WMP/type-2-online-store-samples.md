# <a name="type-2-online-store-samples"></a>Type 2 Online Store-Beispiele

Das Windows Media Player SDK umfasst die folgenden Online Store-Beispiele:

-   **Beispiel-Online Stores.**
-   **Ein Plug-in für den Typ 2-Online Shop.** Dieses Beispiel wird von einem Visual Studio-Projekt-Assistenten generiert, der ein C++-Projekt erstellt, das Sie als Ausgangspunkt für Ihre eigene Komponente verwenden können. Mithilfe des Assistenten wird sichergestellt, dass die Komponente eindeutig ist. Weitere Informationen finden Sie unter [Type 2 Online Store-Plug-in](type-2-online-store-plug-in.md).
-   **Eine Online Store-Webseite.** Diese Beispiel Webseite zeigt eine Webseite, die von Windows Media Player 10 oder höher in einem Dienstaufgaben Bereich gehostet werden kann.
-   **Angeben des contentdistributor-Attributs.** In diesem Beispiel C++ Code wird gezeigt, wie Sie das Windows Media-Format-SDK zum Hinzufügen des **contentdistributor** -Attributs zum-Header einer Windows-Mediendatei verwenden. Dies ist das Attribut, das Windows Media Player 10 oder höher verwendet, um einzelne Online Store-Anbieter zu identifizieren. Weitere Informationen finden Sie unter [Hinzufügen des contentdistributor-Attributs](adding-the-contentdistributor-attribute.md).

**Anzeigen der Online-Beispiel Speicher des Typs 2**

Microsoft hat zwei Beispiel-Online Stores bereitgestellt, die Sie anzeigen können. Eine davon ist ein Beispiel für einen Geschäfts Speicher, der andere ein Beispiel für einen Musikspeicher. Die Online-Beispiel Speicher sind einfache Implementierungen, die einige der grundlegenden Features veranschaulichen, die für die Verwendung in ihren eigenen Online Stores verfügbar sind. Sie können den Quellcode für die Webseiten des Beispiel-Online Stores anzeigen.

Da die Beispiele über das Internet verfügbar sind, werden Sie mit zusätzlichen Features von Zeit zu Zeit aktualisiert.

Zum Anzeigen der Online-Beispiel Speicher für den Typ 2 Konfigurieren Sie Ihre Registrierung, wie in [Registrierungs Schlüsseln und Einträgen für einen Typ 2 Online Store](registry-keys-and-entries-for-a-type-2-online-store.md)beschrieben. In der folgenden Tabelle werden die Schlüssel für die Beispiel Speicher für den Typ 2 angezeigt.



| Speichertyp      | Schlüssel  | Player Versionen, die den Beispiel Speicher unterstützen |
|-----------------|------|-----------------------------------------------|
| Typ 2 Musik    | 8540 | 10 und 11                                     |
| Typ 2 Commerce | 8541 | 10                                            |
