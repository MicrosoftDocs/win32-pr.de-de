# <a name="type-2-online-store-samples"></a>Geben Sie 2 Online Store Beispiele ein.

Das Windows Media Player SDK enthält die folgenden Beispiele für Onlineshops:

-   **Beispiel-Onlineshops.**
-   **Ein Onlineshop-Plug-In vom Typ 2.** Dieses Beispiel wird von einem Visual Studio Projekt-Assistenten generiert, der ein C++-Projekt erstellt, das Sie als Ausgangspunkt für Ihre eigene Komponente verwenden können. Mithilfe des Assistenten wird sichergestellt, dass Ihre Komponente eindeutig ist. Weitere Informationen finden Sie unter [Typ 2 Online Store Plug-In](type-2-online-store-plug-in.md).
-   **Eine Webseite eines Onlineshops.** Diese Beispielwebseite zeigt eine Webseite, die Windows Media Player 10 oder höher in einem Dienstaufgabenbereich hosten kann.
-   **Angeben des ContentDistributor-Attributs.** In diesem C++-Beispielcode wird gezeigt, wie Sie das Windows Media Format SDK verwenden, um das **ContentDistributor-Attribut** dem Header einer Windows Mediendatei hinzuzufügen. Dies ist das Attribut, das Windows Media Player 10 oder höher verwendet, um einzelne Onlineshopanbieter zu identifizieren. Weitere Informationen finden Sie unter [Hinzufügen des ContentDistributor-Attributs.](adding-the-contentdistributor-attribute.md)

**Anzeigen der Beispiel-Onlineshops vom Typ 2**

Microsoft hat zwei Beispiel-Onlineshops vom Typ 2 bereitgestellt, die Sie anzeigen können. Das eine ist ein Beispiel-Commerce-Store, der andere ein Beispiel für einen Musikspeicher. Die Beispiel-Onlineshops sind einfache Implementierungen, die einige der grundlegenden Features zeigen, die für die Verwendung in Ihren eigenen Onlineshops verfügbar sind. Sie können den Quellcode für die Beispielwebseiten des Onlineshops anzeigen.

Da die Beispiele über das Internet verfügbar sind, werden sie von Zeit zu Zeit mit zusätzlichen Features aktualisiert.

Um die Beispiel-Onlineshops vom Typ 2 anzuzeigen, konfigurieren Sie Ihre Registrierung wie unter [Registrierungsschlüssel und -einträge für einen Typ 2 Online Store](registry-keys-and-entries-for-a-type-2-online-store.md)beschrieben. Die folgende Tabelle enthält die Schlüssel für die Typ 2-Beispielspeicher.



| Speichertyp      | Key  | Playerversionen, die den Beispielspeicher unterstützen |
|-----------------|------|-----------------------------------------------|
| Geben Sie 2 musik ein.    | 8540 | 10 und 11                                     |
| Typ 2: Commerce | 8541 | 10                                            |
