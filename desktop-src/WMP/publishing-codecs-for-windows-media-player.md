# <a name="publishing-codecs-for-windows-media-player"></a>Veröffentlichen von Codecs für Windows Media Player

Die WMPlugins-Website wird nicht mehr für die Veröffentlichung Ihres Codecs und die Erstellbarkeit durch Windows Media Player unterstützt. Sie können Ihren Codec stattdessen auf einer Website Ihrer Wahl veröffentlichen.
