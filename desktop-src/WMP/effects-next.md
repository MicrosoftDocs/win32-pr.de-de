# <a name="effectsnext"></a>Effekte. Next

Die **nächste** Methode zeigt die nächste Visualisierungs Voreinstellung an und wechselt bei Bedarf zur nächsten Visualisierung.

``` syntax
        elementID.next()
```

## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.

## <a name="remarks"></a>Bemerkungen

Wenn es sich bei der aktuellen Voreinstellung um die letzte Voreinstellung in der Reihe aller verfügbaren Visualisierungen handelt, wird die erste Voreinstellung der ersten Visualisierung als aktuell festgestellt.

| Anforderung | Wert |
|--------------------|------------------------------------------------------|
