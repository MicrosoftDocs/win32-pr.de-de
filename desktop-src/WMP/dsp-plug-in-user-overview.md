# <a name="dsp-plug-in-user-overview"></a>Übersicht über DSP-Plug-In-Benutzer

Für den Benutzer sind DSP-Plug-Ins Add-On-Features, die die Art und Weise ändern, in der der Benutzer die Audio- oder Videowiedergabe Windows Media Player abspielt. Das Ergebnis eines DSP-Plug-Ins könnte offensichtlich sein, z. B. die Farben im Videobild, das umgekehrt wird, sodass es wie ein bildliches Negatives aussieht, oder etwas sehr dezentes, z. B. ein lichter Halleffekt im Audio, um dem Sound ein Gefühl der Nähe zu geben.

Benutzer können DSP-Plug-Ins über die Registerkarte **Plug-Ins** des Dialogfelds Optionen in der Windows Media Player. Benutzer können installierte Plug-Ins dann mithilfe der Kontrollkästchen auf der Registerkarte **Plug-Ins** aktivieren und deaktivieren. DSP-Plug-Ins können eine Eigenschaftenseite implementieren, auf die Benutzer auch über die **Registerkarte Plug-Ins** zugreifen können. Eine Eigenschaftenseite bietet dem Benutzer die Möglichkeit, Einstellungen zu ändern, die sich auf die Art und Weise auswirken, wie ein DSP-Plug-In funktioniert. Beispielsweise kann eine Eigenschaftenseite ein Schieberegler-Steuerelement enthalten, mit dem der Benutzer die Abklingzeit eines Halleffekts ändern kann.
