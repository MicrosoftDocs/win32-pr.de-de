# <a name="dsp-plug-in-user-overview"></a>Übersicht über den DSP-Plug-in-Benutzer

Für den Benutzer sind DSP-Plug-Ins Add-on-Features, mit denen sich die Art und Weise ändert, in der der Benutzer das audiobild oder Video, das von Windows Media Player abgespielt wird, Das Ergebnis eines DSP-Plug-ins kann offensichtlich sein, z. b. die Farben in dem Video Bild, die umgekehrt werden, sodass es wie ein Foto negativ aussieht, oder etwas sehr fein, wie z. b. ein Licht im audiobild, um dem Sound ein Gefühl der räumlichen Darstellung zu verleihen.

Benutzer können DSP-Plug-Ins mithilfe der Registerkarte **Plug-ins** im Dialogfeld Optionen in der Windows-Media Player installieren und deinstallieren. Benutzer können installierte Plug-Ins mithilfe der Kontrollkästchen auf der Registerkarte **Plug-ins** aktivieren und deaktivieren. DSP-Plug-Ins können eine Eigenschaften Seite implementieren, auf die Benutzer auch über die Registerkarte **Plug-ins** zugreifen können. Eine Eigenschaften Seite bietet dem Benutzer die Möglichkeit, Einstellungen zu ändern, die sich auf die Funktionsweise eines DSP-Plug-ins auswirken. Eine Eigenschaften Seite könnte z. b. ein Schieberegler-Steuerelement enthalten, das es dem Benutzer ermöglicht, die verlaufzeit eines Hall-Effekts zu ändern.
