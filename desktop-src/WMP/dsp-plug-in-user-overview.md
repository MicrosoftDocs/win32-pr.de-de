# <a name="dsp-plug-in-user-overview"></a><span data-ttu-id="dfb92-109">Übersicht über den DSP-Plug-in-Benutzer</span><span class="sxs-lookup"><span data-stu-id="dfb92-109">DSP Plug-in User Overview</span></span>

<span data-ttu-id="dfb92-110">Für den Benutzer sind DSP-Plug-Ins Add-on-Features, mit denen sich die Art und Weise ändert, in der der Benutzer das audiobild oder Video, das von Windows Media Player abgespielt wird,</span><span class="sxs-lookup"><span data-stu-id="dfb92-110">To the user, DSP plug-ins are add-on features that somehow change the way the user experiences the audio or video that Windows Media Player is playing.</span></span> <span data-ttu-id="dfb92-111">Das Ergebnis eines DSP-Plug-ins kann offensichtlich sein, z. b. die Farben in dem Video Bild, die umgekehrt werden, sodass es wie ein Foto negativ aussieht, oder etwas sehr fein, wie z. b. ein Licht im audiobild, um dem Sound ein Gefühl der räumlichen Darstellung zu verleihen.</span><span class="sxs-lookup"><span data-stu-id="dfb92-111">The result of a DSP plug-in could be something obvious, like the colors in the video image being reversed so it looks like a photographic negative, or something very subtle, like a light reverb effect in the audio to add a feeling of spaciousness to the sound.</span></span>

<span data-ttu-id="dfb92-112">Benutzer können DSP-Plug-Ins mithilfe der Registerkarte **Plug-ins** im Dialogfeld Optionen in der Windows-Media Player installieren und deinstallieren.</span><span class="sxs-lookup"><span data-stu-id="dfb92-112">Users can install and uninstall DSP plug-ins by using the **Plug-ins** tab of the Options dialog box in Windows Media Player.</span></span> <span data-ttu-id="dfb92-113">Benutzer können installierte Plug-Ins mithilfe der Kontrollkästchen auf der Registerkarte **Plug-ins** aktivieren und deaktivieren. DSP-Plug-Ins können eine Eigenschaften Seite implementieren, auf die Benutzer auch über die Registerkarte **Plug-ins** zugreifen können. Eine Eigenschaften Seite bietet dem Benutzer die Möglichkeit, Einstellungen zu ändern, die sich auf die Funktionsweise eines DSP-Plug-ins auswirken.</span><span class="sxs-lookup"><span data-stu-id="dfb92-113">Users can then enable and disable installed plug-ins using the checkboxes provided on the **Plug-ins** tab. DSP plug-ins may implement a property page, which users can also access from the **Plug-ins** tab. A property page provides the user with a way to change settings that affect the way a DSP plug-in works.</span></span> <span data-ttu-id="dfb92-114">Eine Eigenschaften Seite könnte z. b. ein Schieberegler-Steuerelement enthalten, das es dem Benutzer ermöglicht, die verlaufzeit eines Hall-Effekts zu ändern.</span><span class="sxs-lookup"><span data-stu-id="dfb92-114">For example, a property page might include a slider control that allows the user to change the decay time of a reverb effect.</span></span>
