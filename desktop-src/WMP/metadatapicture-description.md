# <a name="metadatapicturedescription"></a>MetadataPicture.description

Die **description-Eigenschaft** ruft eine Beschreibung des Metadatenbilds ab.

## <a name="syntax"></a>Syntax

*Player*. *currentMedia*. **getItemInfoByType**( *name*, *language*, *index*). **Beschreibung**

## <a name="possible-values"></a>Mögliche Werte

Diese Eigenschaft ist eine schreibgeschützte **Zeichenfolge.**

## <a name="remarks"></a>Hinweise

Zum Abrufen des Werts dieser Eigenschaft ist Lesezugriff auf die Bibliothek erforderlich. Weitere Informationen finden Sie unter [Bibliothekszugriff.](library-access.md)

**Windows Media Player 10 Mobile:** Diese Eigenschaft wird nicht unterstützt.
