# <a name="slidermax"></a>Schieberegler. Max

Das **Maximum** -Attribut gibt den maximalen Wert des Bereichs an oder ruft ihn ab, der durch das Schieberegler-Steuerelement definiert wird.

``` syntax
        elementID.max
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/schreibnummer (**float**) mit dem Standardwert 100. 

## <a name="remarks"></a>Bemerkungen

Der für " **Max** " angegebene Wert muss größer als der Wert für " **Min**" sein.

Weitere Informationen finden Sie unter **customslider**. [positionImage](customslider-positionimage.md) -Attribut für ein Beispiel, das veranschaulicht, wie die Attribute des **Slider** -Elements verwendet werden.
