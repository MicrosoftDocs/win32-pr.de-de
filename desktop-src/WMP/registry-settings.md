# <a name="registry-settings"></a>Registrierungseinstellungen

Windows Media Player 9-Serie oder höher verwendet die Registrierung, um bestimmte Einstellungen zu speichern. Sie können änderungen an diesen Einstellungen vornehmen, um das Verhalten von Windows Media Player und des Windows Media Player-Steuerelements zu ändern.

In den folgenden Abschnitten werden die unterstützten Registrierungseinstellungen ausführlich beschrieben.



| `Section`                                                                                                        | BESCHREIBUNG                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Einstellungen der Dateinamenerweiterungsregistrierung](file-name-extension-registry-settings.md)                             | Beschreibt, wie eine benutzerdefinierte Dateinamenerweiterung registriert wird.                                                                                                               |
| [Benutzerdefinierte Einstellungen für die Schemaregistrierung](custom-scheme-registry-settings.md)                                         | Beschreibt, wie ein benutzerdefiniertes Schema registriert wird.                                                                                                                            |
| [Einstellungen für die Ordnerüberwachungsregistrierung](folder-monitoring-registry-settings.md)                                 | Beschreibt, wie Dateiordner registriert werden, die von Windows Media Player überwacht werden sollen.                                                                                       |
| [firewall port registry Einstellungen](firewall-port-registry-settings.md)                                         | Beschreibt einen Registrierungseintrag, der Windows Media Player festlegt, um anzugeben, ob Firewalls Ports für die Bibliotheksfreigabe geöffnet lassen sollen.                                   |
| [Registrierungseinstellung für den automatischen Erwerb](silent-acquisition-registry-setting.md)                                 | Beschreibt einen Registrierungseintrag, der Windows Media Player verwendet, um zu bestimmen, ob Nutzungsrechte automatisch heruntergeladen werden, wenn der Benutzer geschützte Inhalte wiedergibt oder synchronisiert. |
| [Registrieren der Anwendungsabhängigkeit](registering-application-dependency.md)                                   | Enthält Informationen zum Registrieren Ihrer Anwendung, um sicherzustellen, dass die richtigen Laufzeitdateien auf dem Computer des Benutzers vorhanden sind.                                         |
| [Registrierungseinträge zum Nachverfolgen des Installationsfortschritts](registry-entries-for-tracking-installation-progress.md) | Beschreibt Registrierungseinträge, mit denen Installationsprogramme den Fortschritt der Windows Media Player-Setups nachverfolgen können.                                                    |
