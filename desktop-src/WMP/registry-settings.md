# <a name="registry-settings"></a>Registrierungseinstellungen

Windows Media Player 9 oder höher verwendet die Registrierung, um bestimmte Einstellungen zu speichern. Sie können Änderungen an diesen Einstellungen vornehmen, um das Verhalten von Windows Media Player und dem Windows Media Player-Steuerelement zu ändern.

In den folgenden Abschnitten werden die unterstützten Registrierungs Einstellungen ausführlich erläutert.



| `Section`                                                                                                        | BESCHREIBUNG                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Registrierungs Einstellungen für die Dateinamenerweiterung](file-name-extension-registry-settings.md)                             | Beschreibt, wie eine benutzerdefinierte Dateinamenerweiterung registriert wird.                                                                                                               |
| [Registrierungs Einstellungen für benutzerdefinierte Schemas](custom-scheme-registry-settings.md)                                         | Beschreibt, wie ein benutzerdefiniertes Schema registriert wird.                                                                                                                            |
| [Registrierungs Einstellungen für die Ordner Überwachung](folder-monitoring-registry-settings.md)                                 | Hier wird beschrieben, wie Sie Datei Ordner registrieren, die von Windows Media Player überwacht werden sollen.                                                                                       |
| [Registrierungs Einstellungen für den Firewallport](firewall-port-registry-settings.md)                                         | Beschreibt einen Registrierungs Eintrag, der von Windows Media Player festgelegt wird, um anzugeben, ob Firewalls Ports für die Bibliotheks Freigabe geöffnet lassen sollen.                                   |
| [Registrierungs Einstellung für automatische Übernahme](silent-acquisition-registry-setting.md)                                 | Beschreibt einen Registrierungs Eintrag, den Windows Media Player verwendet, um zu bestimmen, ob die Nutzungsrechte automatisch heruntergeladen werden sollen, wenn der Benutzer geschützte Inhalte wieder gibt oder synchronisiert. |
| [Anwendungs Abhängigkeit wird registriert](registering-application-dependency.md)                                   | Enthält Informationen zum Registrieren Ihrer Anwendung, um sicherzustellen, dass auf dem Computer des Benutzers korrekte Laufzeitdateien vorhanden sind.                                         |
| [Registrierungseinträge zum Nachverfolgen des Installations Fortschritts](registry-entries-for-tracking-installation-progress.md) | Beschreibt Registrierungseinträge, die von Installationsprogrammen verwendet werden können, um den Fortschritt von Windows Media Player-Setup zu verfolgen.                                                    |
