# <a name="texthoverforegroundcolor"></a>Text. hoverforegroundcolor

Das **hoverforegroundcolor** -Attribut gibt die Textfarbe an, die für das Text Steuerelement verwendet wird, wenn der Mauszeiger darüber bewegt wird, oder ruft Sie ab.

``` syntax
        elementID.hoverForegroundColor
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese- **/schreibzeichenfolge** mit einem beliebigen Microsoft Internet Explorer-Farbwert.

## <a name="remarks"></a>Bemerkungen

Wenn **hoverforegroundcolor** nicht angegeben wird, wird **ForegroundColor** verwendet.

Im [value](text-value.md) -Attribut finden Sie ein Beispiel, das veranschaulicht, wie die Attribute des **Text** -Elements verwendet werden.
