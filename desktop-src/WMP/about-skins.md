# <a name="about-skins"></a>Informationen zu Skins

Mit Windows Media Player können Benutzer aus einer Vielzahl von Standard-Skins auswählen, von denen jedes eine zusätzliche visuelle Darstellung bietet, die das lauschen und Anzeigen von Genuss verbessert. Windows Media Player verfügt über mehrere zu wählbare Skins, aber es ist relativ einfach, benutzerdefinierte Skins zu erstellen und zu verteilen.

Skins sind einfache Auflistungen von einer oder mehreren Computergrafik Dateien, die nach einer Textdatei organisiert sind und Windows Media Player, wie diese Dateien zum Anzeigen eines Skin verwendet werden. Sie können eine Vielzahl von Programmen verwenden, um vorhandene Grafiken zu ändern oder einfache Kunst zu erstellen. Wenn Sie z. b. die in Microsoft PowerPoint bereitgestellten Zeichnungs Tools und Clip Grafiken verwenden, können Sie die Grafik erstellen, indem Sie einfach Drag und Drop verwenden. Ebenso müssen Sie nicht Programmierer sein, um die Textdatei zu erstellen, mit der eine einzelne Skin definiert wird.

Eine der Gründe, warum die Skins relativ einfach zu erstellen sind, besteht darin, dass Sie sich die einzelnen Teile der Kunst und den Quellcode für beliebige Skin ansehen können und sehen, wie die einzelnen Skin zusammengestellt werden. Mit diesem SDK werden mehrere einfache Beispiel-Skins bereitgestellt, sowie eine Referenz für Skin-Elemente und-Attribute.
