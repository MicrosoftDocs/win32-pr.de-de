# <a name="about-skins"></a>Informationen zu Skins

Mit Windows Media Player können Benutzer aus einer Vielzahl von Standardskins wählen, die jeweils eine zusätzliche visuelle Benutzeroberfläche bieten, die das Lauschen und Diebesen verbessert. Windows Media Player verfügt über mehrere Skins zur Auswahl, aber es ist relativ einfach, benutzerdefinierte Skins zu erstellen und zu verteilen.

Skins sind einfach Sammlungen von einer oder mehreren Computerartdateien, die durch eine Textdatei organisiert sind, die Windows Media Player angibt, wie diese Dateien zum Anzeigen einer Skin verwendet werden sollen. Sie können eine Vielzahl von Programmen verwenden, um vorhandene Artefakte zu ändern oder eigene einfache Artefakte zu erstellen. Wenn Sie z. B. die mit Microsoft PowerPoint bereitgestellten Zeichentools und ClipArts verwenden, können Sie Grafiken erstellen, indem Sie sie einfach ziehen und ablegen. Ebenso müssen Sie kein Programmierer sein, um die Textdatei zu erstellen, die zum Definieren einer einzelnen Skin verwendet wird.

Einer der Gründe, warum Skins relativ einfach zu erstellen sind, besteht darin, dass Sie sich die einzelnen Artefakte und den Quellcode für jede Skin ansehen und sehen können, wie die einzelnen Skins zusammengestellt werden. Dieses SDK enthält mehrere einfache Beispielskins sowie einen Verweis auf Skinelemente und Attribute.
