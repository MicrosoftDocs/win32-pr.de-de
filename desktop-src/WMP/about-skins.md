<span data-ttu-id="260fd-107">Mit Windows Media Player können Benutzer aus einer Vielzahl von Standard-Skins auswählen, von denen jedes eine zusätzliche visuelle Darstellung bietet, die das lauschen und Anzeigen von Genuss verbessert.</span><span class="sxs-lookup"><span data-stu-id="260fd-107">With Windows Media Player, users can choose from a variety of standard skins, each one providing an additional visual experience that enhances listening and viewing pleasure.</span></span> <span data-ttu-id="260fd-108">Windows Media Player verfügt über mehrere zu wählbare Skins, aber es ist relativ einfach, benutzerdefinierte Skins zu erstellen und zu verteilen.</span><span class="sxs-lookup"><span data-stu-id="260fd-108">Windows Media Player comes with several skins to choose from, but it is relatively easy to create and distribute custom skins.</span></span>

<span data-ttu-id="260fd-109">Skins sind einfache Auflistungen von einer oder mehreren Computergrafik Dateien, die nach einer Textdatei organisiert sind und Windows Media Player, wie diese Dateien zum Anzeigen eines Skin verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="260fd-109">Skins are simply collections of one or more computer art files, organized by a text file, which tells Windows Media Player how to use these files to display a skin.</span></span> <span data-ttu-id="260fd-110">Sie können eine Vielzahl von Programmen verwenden, um vorhandene Grafiken zu ändern oder einfache Kunst zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="260fd-110">You can use a variety of programs to modify existing art or create simple art of your own.</span></span> <span data-ttu-id="260fd-111">Wenn Sie z. b. die in Microsoft PowerPoint bereitgestellten Zeichnungs Tools und Clip Grafiken verwenden, können Sie die Grafik erstellen, indem Sie einfach Drag und Drop verwenden.</span><span class="sxs-lookup"><span data-stu-id="260fd-111">For example, using the drawing tools and clip art supplied with Microsoft PowerPoint, you can create artwork by just dragging and dropping.</span></span> <span data-ttu-id="260fd-112">Ebenso müssen Sie nicht Programmierer sein, um die Textdatei zu erstellen, mit der eine einzelne Skin definiert wird.</span><span class="sxs-lookup"><span data-stu-id="260fd-112">Similarly, you do not need to be a programmer to create the text file used to define an individual skin.</span></span>

<span data-ttu-id="260fd-113">Eine der Gründe, warum die Skins relativ einfach zu erstellen sind, besteht darin, dass Sie sich die einzelnen Teile der Kunst und den Quellcode für beliebige Skin ansehen können und sehen, wie die einzelnen Skin zusammengestellt werden.</span><span class="sxs-lookup"><span data-stu-id="260fd-113">One of the reasons that skins are relatively easy to create is that you can look at the individual pieces of art and the source code for any skin and see how each skin is put together.</span></span> <span data-ttu-id="260fd-114">Mit diesem SDK werden mehrere einfache Beispiel-Skins bereitgestellt, sowie eine Referenz für Skin-Elemente und-Attribute.</span><span class="sxs-lookup"><span data-stu-id="260fd-114">Several simple example skins are provided with this SDK, as well as a reference for skin elements and attributes.</span></span>
