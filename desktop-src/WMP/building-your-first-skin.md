# <a name="building-your-first-skin"></a>Aufbauen Ihrer ersten Skin

Wie in [about Skins](about-skins.md)ausführlicher erläutert, wird ein Skin in einer Skin-Definitionsdatei definiert und erfordert eine oder mehrere unterstützende kunstdateien, Microsoft JScript-Dateien und digitale Mediendateien. Dieses Tutorial für eine sehr einfache Skin erfordert nur die Skin-Definitionsdatei und die drei kunstdateien. Sie werden vier Schritte durchlaufen, die in den folgenden Themen beschrieben werden:

-   [Erstellen der primären Grafikdatei](creating-the-primary-art-file.md)
-   [Erstellen der Mapping-Datei](creating-the-mapping-file.md)
-   [Erstellen des Hover-Bilds](creating-the-hover-image.md)
-   [Erstellen der Skin-Definitionsdatei](creating-the-skin-definition-file.md)
