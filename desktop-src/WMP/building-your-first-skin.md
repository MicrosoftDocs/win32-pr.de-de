# <a name="building-your-first-skin"></a>Erstellen Ihrer ersten Skin

Wie unter [About Skins (Informationen zu Skins)](about-skins.md)ausführlicher erläutert wird, wird eine Skin in einer Skindefinitionsdatei definiert und erfordert mindestens eine unterstützende Grafikdatei, Microsoft JScript dateien und digitale Mediendateien. Dieses Tutorial für eine sehr einfache Skin erfordert nur die Skindefinitionsdatei und drei Grafikdateien. Sie durchlaufen vier Schritte, die in den folgenden Themen beschrieben werden:

-   [Erstellen der primären Art-Datei](creating-the-primary-art-file.md)
-   [Erstellen der Zuordnungsdatei](creating-the-mapping-file.md)
-   [Erstellen des Bilds mit dem Mauszeiger](creating-the-hover-image.md)
-   [Erstellen der Skindefinitionsdatei](creating-the-skin-definition-file.md)
