# <a name="wmalbumtitle-attribute"></a>WM/AlbumTitle-Attribut

Das **WM/AlbumTitle-Attribut** ist der Titel des Albums, auf dem der Inhalt ursprünglich veröffentlicht wurde.

## <a name="remarks"></a>Hinweise

Dieses Attribut wird sowohl in der Bibliothek (oder im Cache) als auch in der digitalen Mediendatei gespeichert.

**Album** ist ein Alias für dieses Attribut.

Die Windows Media Format SDK-Konstante für dieses Attribut ist g \_ wszWMMediumTitle.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9er Serie oder höher<br/> |
