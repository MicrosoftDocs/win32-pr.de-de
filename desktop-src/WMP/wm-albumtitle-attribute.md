# <a name="wmalbumtitle-attribute"></a>WM/Album Title-Attribut

Das **WM/albumtitle-** Attribut ist der Titel des Albums, auf dem der Inhalt ursprünglich veröffentlicht wurde.

## <a name="remarks"></a>Bemerkungen

Dieses Attribut ist sowohl in der Bibliothek (oder im Cache) als auch in der digitalen Mediendatei gespeichert.

" **Album** " ist ein Alias für dieses Attribut.

Die SDK-Konstante für das Windows Media-Format für dieses Attribut ist "g \_ wszwmalbumtitle".

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9-Serie oder höher<br/> |
