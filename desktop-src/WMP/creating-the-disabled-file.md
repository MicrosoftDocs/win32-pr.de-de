# <a name="creating-the-disabled-file"></a>Erstellen der deaktivierten Datei

Die Disabled-Datei ähnelt der Push-Datei, mit der Ausnahme, dass eine andere Farbe (rot) verwendet wird, um zu zeigen, wenn etwas deaktiviert ist.

Die folgende Abbildung zeigt die Disabled-Datei.

![Deaktivierte Datei](images/ceswmdis.png)

Beachten Sie, dass Dateien wie diese auf der rechten und unteren Seite abgeschnitten werden können. Die tatsächliche Größe der Bitmap ist nicht wichtig, sondern nur, dass die Elemente für Treffertests mit der Region-Bitmap in Einklang stehen, wie von den oberen linken Ecken (einschließlich Offsets) gemessen.
