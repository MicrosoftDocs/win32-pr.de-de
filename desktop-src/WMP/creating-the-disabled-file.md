# <a name="creating-the-disabled-file"></a>Erstellen der deaktivierten Datei

Die deaktivierte Datei ähnelt der gepushten Datei mit der Ausnahme, dass eine andere Farbe (rot) verwendet wird, um anzuzeigen, wenn etwas deaktiviert ist.

Die folgende Abbildung ist die deaktivierte Datei.

![deaktivierte Datei](images/ceswmdis.png)

Beachten Sie, dass Dateien wie diese auf der rechten und unteren Seite abgeschnitten werden könnten. Die tatsächliche Größe der Bitmap ist nicht wichtig, nur dass die Elemente mit der Bereichs Bitmap für Treffer Tests übereinstimmen, wie Sie von den oberen linken Ecken (einschließlich Offsets) gemessen werden.
