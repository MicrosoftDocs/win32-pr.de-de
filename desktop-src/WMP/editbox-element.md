# <a name="editbox-element"></a>EditBox-Element

Das **EditBox** -Element bietet Benutzern die Möglichkeit, Text in einem Skin einzugeben.

Das **EditBox** -Element unterstützt die folgenden Attribute.



| Attribut                                      | BESCHREIBUNG                                                                                                   |
|------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [Background Color](editbox-backgroundcolor.md) | Gibt die Hintergrundfarbe für das Bearbeitungsfeld-Steuerelement an oder ruft diese ab.                                         |
| [Aun](editbox-border.md)                   | Gibt einen Wert an, der angibt, ob das Bearbeitungsfeld-Steuerelement über einen Rahmen verfügt                          |
| [editstyle](editbox-editstyle.md)             | Gibt das Format des Bearbeitungsfeld-Steuer Elements an oder ruft es ab.                                                     |
| [fontface](editbox-fontface.md)               | Gibt die Schriftart an, die für Text im Bearbeitungsfeld-Steuerelement verwendet wird, oder ruft diese ab.                                        |
| [FontSize](editbox-fontsize.md)               | Gibt den Schrift Grad für das Bearbeitungsfeld-Steuerelement an oder ruft ihn ab.                                                |
| [FontStyle](editbox-fontstyle.md)             | Gibt den Schrift Schnitt für das Bearbeitungsfeld-Steuerelement an oder ruft ihn ab.                                               |
| [ForegroundColor](editbox-foregroundcolor.md) | Gibt die Textfarbe im Bearbeitungsfeld-Steuerelement an oder ruft diese ab.                                                |
| [Recht](editbox-justification.md)     | Gibt die Ausrichtung des Texts innerhalb des Bearbeitungsfeld-Steuer Elements an oder ruft diese ab.                                 |
| [Zeilen Anzahl](editbox-linecount.md)             | Ruft die Anzahl der Zeilen im Bearbeitungsfeld-Steuerelement ab.                                                        |
| [readOnly](editbox-readonly.md)               | Gibt einen Wert an, der angibt, ob der Text im Bearbeitungsfeld-Steuerelement schreibgeschützt ist oder bearbeitet werden kann, oder ruft diesen Wert ab. |
| [Textlimit](editbox-textlimit.md)             | Gibt die maximale Anzahl von Zeichen an, die der Benutzer im Bearbeitungsfeld-Steuerelement eingeben kann, oder ruft diese ab.       |
| [value](editbox-value.md)                     | Gibt den Text an, der im Bearbeitungsfeld-Steuerelement angezeigt wird, oder ruft diesen ab.                                    |
| [WordWrap](editbox-wordwrap.md)               | Gibt einen Wert an, der angibt, ob der Zeilenumbruch im Bearbeitungsfeld-Steuerelement aktiviert ist.               |



 

Das **EditBox** -Element unterstützt die folgenden Methoden.



| Methode                                             | BESCHREIBUNG                                                                                         |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [getLine](editbox-getline.md)                     | Ruft den Text für die Zeile mit dem angegebenen Index ab.                                           |
| [getlinefromchar](editbox-getlinefromchar.md)     | Ruft den Zeilen Index für den angegebenen Zeichen Index ab.                                         |
| [getlineingedex](editbox-getlineindex.md)           | Ruft den Zeichen Index für den angegebenen Zeilen Index ab.                                         |
| [getSelectionEnd](editbox-getselectionend.md)     | Ruft die Endposition des ausgewählten Texts im Bearbeitungsfeld-Steuerelement ab.                         |
| [getSelectionStart](editbox-getselectionstart.md) | Ruft die Anfangsposition des ausgewählten Texts im Bearbeitungsfeld-Steuerelement ab.                       |
| [ReplaceSelection](editbox-replaceselection.md)   | Ersetzt die aktuelle Auswahl durch den angegebenen Text.                                             |
| [setSelection](editbox-setselection.md)           | Wählt den Text im Bearbeitungsfeld-Steuerelement vom angegebenen Start Index bis zum angegebenen endIndex aus. |



 

Das **EditBox** -Element unterstützt die Ambient-Attribute und kann alle Ambient-Ereignishandler mit Ausnahme von " **OnClick**" implementieren. Weitere Informationen finden Sie unter [Ambient-Attribute](ambient-attributes.md) und [Ambient-Ereignishandler](ambient-event-handlers.md).

> [!Note]  
> Für dieses Element ist Windows Media Player für Windows XP oder höher erforderlich.
