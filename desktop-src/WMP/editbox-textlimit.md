# <a name="editboxtextlimit"></a>EDITBOX.textLimit

Das **textLimit-Attribut** gibt die maximale Anzahl von Zeichen an, die der Benutzer im Bearbeitungsfeld-Steuerelement eingeben kann, oder ruft sie ab.

``` syntax
        elementID.textLimit
```

## <a name="possible-values"></a>Mögliche Werte

Bei diesem Attribut handelt  es sich um eine Lese-/Schreibnummer **(long)** mit dem Standardwert 0, was auf keinen Grenzwert hinweist.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------------------|
| Version<br/> | Windows Media Player für Windows XP oder höher<br/> |
