# <a name="editboxtextlimit"></a>EditBox. Textlimit

Das **Textlimit** -Attribut gibt die maximale Anzahl von Zeichen an, die der Benutzer im Bearbeitungsfeld-Steuerelement eingeben kann, oder ruft diese ab.

``` syntax
        elementID.textLimit
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/schreibzahl (**Long**) mit einem Standardwert von 0 (null), was bedeutet, dass es keine Begrenzung gibt. 

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------------------|
| Version<br/> | Windows Media Player für Windows XP oder höher<br/> |
