# <a name="music-file-attributes"></a>Musik Dateiattribute

Die folgenden Attribute sind für eine Musikdatei verfügbar, die nicht in der Bibliothek enthalten ist. Diese Attribute entsprechen Eingaben, die im Dialogfeld **"Erweiterter Tag-Editor"** Windows Media Player.

-   [Description-Attribut](description-attribute.md)
-   [WM/OriginalAttributeTitle-Attribut](wm-originalalbumtitle-attribute.md)
-   [WM/OriginalArtist-Attribut](wm-originalartist-attribute.md)
-   [WM/OriginalLyricist-Attribut](wm-originallyricist-attribute.md)
-   [WM/PartOfSet-Attribut](wm-partofset-attribute.md)
