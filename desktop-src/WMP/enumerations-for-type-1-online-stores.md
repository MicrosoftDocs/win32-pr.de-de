# <a name="enumerations-for-type-1-online-stores"></a>Enumerationen für den Typ 1-Online Speicher

> [!Note]  
> In diesem Abschnitt werden die-Funktionen beschrieben, die für die Verwendung durch Online Stores Die Verwendung dieser Funktion außerhalb des Kontexts eines Online Stores wird nicht unterstützt.

 

Die folgenden Enumerationstypen sind für Type 1 Online Stores mit C++-Code verfügbar.



| Enumeration                                            | Beschreibung                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [Wmpaccounttype](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpaccounttype)                   | Enumeration, die Konto Typen für einen Online Shop definiert.                                              |
| [Wmpcallbacknotification](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpcallbacknotification) | Enumeration, die Zustände definiert, die von der **iwmpcontentpartnercallback:: notify** -Rückruffunktion verwendet werden. |
| [Wmppartnerbenachrichtigung](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmppartnernotification)   | Enumeration, die Betriebszustände eines Online Stores definiert.                                          |
| [Wmpstreamingtype](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpstreamingtype)               | Enumeration, die den Typ der Streamingmedien angibt.                                                  |
| [Wmptasktype](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptasktype)                         | Enumeration, die Aufgabenbereiche von Windows Media Player darstellt.                                             |
| [Wmptemplatesize](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptemplatesize)                 | Enumeration, die HTML-Vorlagen Größen darstellt.                                                         |
| [Wmptransaktionstyp](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptransactiontype)           | Enumeration, die einen Transaktionstyp darstellt.                                                          |
