# <a name="writing-code"></a>Schreiben von Code

Die tatsächliche Leistungsfähigkeit von Skins ist von der Fähigkeit, auf Ereignisse zu reagieren. Sie möchten, dass Ihre Skin etwas tun kann, wenn der Benutzer auf eine Schaltfläche klickt, und Sie möchten auch auf Änderungen reagieren, die auf Windows Media Player, wie z. b. den Fortschritt der Wiedergabe der digitalen Mediendatei oder des Datenstroms.
