# <a name="writing-code"></a>Schreiben von Code

Die eigentliche Leistung von Skins liegt in der Möglichkeit, auf Ereignisse zu reagieren. Sie möchten, dass Ihre Skin etwas tun kann, wenn der Benutzer auf eine Schaltfläche klickt, und Sie möchten auch auf Änderungen reagieren, die mit Windows Media Player geschehen, z. B. auf den Fortschritt der digitalen Mediendatei oder des streams, die abspielt.

In den folgenden Abschnitten wird das Schreiben von Code für Skins erläutert:

|----------------------------------------|---------------------------------------------|
| [Behandeln von Ereignissen](handling-events.md) | Erläutert, wie Ereignisse im Skincode behandelt werden. |
| [Verwenden JScript](using-jscript.md)     | Erläutert, wie Skriptcode in Skins geschrieben wird. |
