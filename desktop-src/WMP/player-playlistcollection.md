# <a name="playerplaylistcollection"></a>Player. playlistcollection

Die **playlistcollection** -Eigenschaft ruft das **playlistcollection** -Objekt ab.

## <a name="syntax"></a>Syntax

*Player* . **playlistcollection**

## <a name="possible-values"></a>Mögliche Werte

Diese Eigenschaft ist ein Schreib geschütztes **playlistcollection** -Objekt.

## <a name="remarks"></a>Bemerkungen

Zum Abrufen des Werts dieser Eigenschaft ist Lesezugriff auf die Bibliothek erforderlich. Weitere Informationen finden Sie unter [Bibliotheks Zugriff](library-access.md).
