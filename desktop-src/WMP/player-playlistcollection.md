# <a name="playerplaylistcollection"></a>Player.playlistCollection

Die **playlistCollection-Eigenschaft** ruft das **PlaylistCollection-Objekt** ab.

## <a name="syntax"></a>Syntax

*Player* . **playlistCollection**

## <a name="possible-values"></a>Mögliche Werte

Diese Eigenschaft ist ein schreibgeschütztes **PlaylistCollection-Objekt.**

## <a name="remarks"></a>Hinweise

Um den Wert dieser Eigenschaft abzurufen, ist Lesezugriff auf die Bibliothek erforderlich. Weitere Informationen finden Sie unter [Bibliothekszugriff.](library-access.md)
