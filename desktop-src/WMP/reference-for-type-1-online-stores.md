# <a name="reference-for-type-1-online-stores"></a>Referenz für Typ 1-Onlineshops

> [!Note]  
> In diesem Abschnitt werden die-Funktionen beschrieben, die für die Verwendung durch Online Stores Die Verwendung dieser Funktion außerhalb des Kontexts eines Online Stores wird nicht unterstützt.

 

Windows Media Player 11 stellt einen Katalog Compiler, ein Objekt und mehrere Schnittstellen bereit, die den Typ 1 Online-Speicher unterstützen. In den folgenden Abschnitten werden diese ausführlich beschrieben.



| `Section`                                                                                    | BESCHREIBUNG                                                                                                                                                                                                                        |
|--------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Katalog Compiler für Typ 1-Online Speicher](catalog-compiler-for-type-1-online-stores.md) | Enthält Referenzmaterial für den Compiler, der von einem Online Shop zum Kompilieren seines musikkatalogs verwendet wird.                                                                                                                               |
| [Iwmpcontentcontainer-Schnittstelle](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainer)                                 | Stellt Referenzseiten für Methoden bereit, die das Plug-in des Online Stores aufruft, um einen Inhalts Container zu überprüfen. Ein Inhalts Container stellt einen Satz von Medien Elementen dar, der heruntergeladen oder gekauft werden soll. Implementiert von Windows Media Player. |
| [Iwmpcontentcontainerlist-Schnittstelle](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainerlist)                         | Stellt Referenzseiten für Methoden bereit, die vom Plug-in des Online Stores aufgerufen werden, um eine Liste der Inhalts Container zu überprüfen. Implementiert von Windows Media Player.                                                                           |
| [Iwmpcontentpartner-Schnittstelle](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartner)                                     | Stellt Referenzseiten für Methoden bereit, die von Windows Media Player aufgerufen werden, um Informationen aus dem Online Shop zu erhalten und den Online Shop über die Aktionen des Benutzers zu benachrichtigen. Wird vom Plug-in des Online Stores implementiert.                  |
| [Iwmpcontentpartnercallback-Schnittstelle](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartnercallback)                     | Stellt Referenzseiten für Methoden bereit, die das Plug-in des Online Stores aufruft, um Windows Media Player über den Abschluss von asynchronen Vorgängen zu benachrichtigen. Implementiert von Windows Media Player.                                    |
| [Externes Objekt für den Typ 1-Online Speicher](external-object-for-type-1-online-stores.md)   | Stellt Referenzseiten für Eigenschaften, Methoden und Ereignisse bereit, die von Skripts in Webseiten verwendet werden, die vom Online Store bereitgestellt werden.                                                                                                              |
| [Enumerationen für den Typ 1-Online Speicher](enumerations-for-type-1-online-stores.md)         | Bietet Referenzseiten für die Enumerationen, die bei der Kommunikation zwischen Windows Media Player und dem Online Store-Plug-in verwendet werden.                                                                                               |
| [Strukturen für Typ 1-Online Speicher](structures-for-type-1-online-stores.md)             | Bietet Referenzseiten für die Strukturen, die bei der Kommunikation zwischen Windows Media Player und dem Online Store-Plug-in verwendet werden.                                                                                                 |
