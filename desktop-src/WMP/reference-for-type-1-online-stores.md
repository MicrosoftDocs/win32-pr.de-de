# <a name="reference-for-type-1-online-stores"></a>Referenz für Typ 1-Onlineshops

> [!Note]  
> In diesem Abschnitt werden funktionen beschrieben, die für die Verwendung durch Onlineshops entwickelt wurden. Die Verwendung dieser Funktionalität außerhalb des Kontexts eines Onlineshops wird nicht unterstützt.

 

Windows Media Player 11 bietet einen Katalogcompiler, ein -Objekt und mehrere Schnittstellen, die Onlineshops vom Typ 1 unterstützen. In den folgenden Abschnitten werden diese ausführlich dokumentiert.



| `Section`                                                                                    | BESCHREIBUNG                                                                                                                                                                                                                        |
|--------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Katalogcompiler für Onlineshops vom Typ 1](catalog-compiler-for-type-1-online-stores.md) | Stellt Referenzmaterial für den Compiler bereit, das ein Onlineshop verwendet, um seinen Musikkatalog zu kompilieren.                                                                                                                               |
| [IWMPContentContainer-Schnittstelle](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainer)                                 | Stellt Referenzseiten für Methoden bereit, die das Plug-In des Onlineshops aufruft, um einen Inhaltscontainer zu überprüfen. Ein Inhaltscontainer stellt einen Satz von Medienelementen dar, die heruntergeladen oder erworben werden sollen. Wird von Windows Media Player implementiert. |
| [IWMPContentContainerList-Schnittstelle](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainerlist)                         | Stellt Referenzseiten für Methoden bereit, die das Plug-In des Onlineshops aufruft, um eine Liste von Inhaltscontainern zu überprüfen. Wird von Windows Media Player implementiert.                                                                           |
| [IWMPContentPartner-Schnittstelle](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartner)                                     | Stellt Referenzseiten für Methoden bereit, die Windows Media Player aufrufen, um Informationen aus dem Onlineshop abzurufen und den Onlineshop über die Aktionen des Benutzers zu benachrichtigen. Wird vom Plug-In des Onlineshops implementiert.                  |
| [IWMPContentPartnerCallback-Schnittstelle](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartnercallback)                     | Stellt Referenzseiten für Methoden bereit, die das Plug-In des Onlineshops aufruft, um Windows Media Player über den Abschluss asynchroner Vorgänge zu benachrichtigen. Wird von Windows Media Player implementiert.                                    |
| [Externes Objekt für Onlineshops vom Typ 1](external-object-for-type-1-online-stores.md)   | Stellt Referenzseiten für Eigenschaften, Methoden und Ereignisse bereit, die von Skripts auf Webseiten verwendet werden, die vom Onlineshop bereitgestellt werden.                                                                                                              |
| [Enumerationen für Onlineshops vom Typ 1](enumerations-for-type-1-online-stores.md)         | Stellt Referenzseiten für die Enumerationen bereit, die bei der Kommunikation zwischen Windows Media Player und dem Plug-In des Onlineshops verwendet werden.                                                                                               |
| [Strukturen für Onlineshops vom Typ 1](structures-for-type-1-online-stores.md)             | Stellt Referenzseiten für die Strukturen bereit, die bei der Kommunikation zwischen Windows Media Player und dem Plug-In des Onlineshops verwendet werden.                                                                                                 |
