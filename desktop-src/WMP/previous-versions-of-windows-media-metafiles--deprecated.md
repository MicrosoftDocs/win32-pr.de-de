# <a name="previous-versions-of-windows-media-metafiles-deprecated"></a>Frühere Versionen von Windows Media-Metadatendateien (veraltet)

In diesem Thema wird eine Funktion beschrieben, die in der aktuellen Version von Windows Media Metafiles nicht mehr verwendet wird.

In diesem Abschnitt wird ein Teil der Version 2 von Windows Media-Metadatendateien dokumentiert. Version 2 ist nicht die aktuelle Version von Windows Media-Metadatendateien, es gibt jedoch einige Fälle, in denen Windows Media Services Clients Metadatendateien im Format Version 2 bereitstellt. Windows Media-Metadatendateien werden auch als erweiterte Stream Redirector-Dateien (ASX) bezeichnet.

In diesem Abschnitt werden die folgenden Elemente dokumentiert.



| Element                                                   | BESCHREIBUNG                                                    |
|--------------------------------------------------------|----------------------------------------------------------------|
| [Reference-Element](reference-element--deprecated.md) | Enthält eine Liste mit URLs von Mediendateien.                        |
| [Mswap-Parameter](mswmext-parameter--deprecated.md) | Gibt einem Client Programm das Format einer referenzierten Datei an. |
