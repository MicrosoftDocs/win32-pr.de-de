# <a name="previous-versions-of-windows-media-metafiles-deprecated"></a>Frühere Versionen von Windows Medienmetadateien (veraltet)

In diesem Thema wird ein Feature dokumentiert, das nicht mehr in der aktuellen Version von Windows Media Metafiles verwendet wird.

In diesem Abschnitt wird ein Teil der Version 2 von Windows Medienmetadateien dokumentiert. Version 2 ist nicht die aktuelle Version von Windows Media Metafiles, aber es gibt einige Fälle, in denen Windows Media-Dienste Clients Metadateien im Format Version 2 zur Verfügung stellt. Windows Medienmetadateien werden auch als ASX-Dateien (Advanced Stream Redirector) bezeichnet.

In diesem Abschnitt werden die folgenden Elemente dokumentiert.



| Element                                                   | Beschreibung                                                    |
|--------------------------------------------------------|----------------------------------------------------------------|
| [reference-Element](reference-element--deprecated.md) | Enthält eine Liste der URLs von Mediendateien.                        |
| [MSWMExt-Parameter](mswmext-parameter--deprecated.md) | Gibt für ein Clientprogramm das Format einer Datei an, auf die verwiesen wird. |
