# <a name="attribute-reference"></a>Attributverweis

Die Windows Media Player Attributreferenz dokumentiert Metadatenattribute, die für Windows Media Player SDK-Entwickler von Interesse sind. In der folgenden Tabelle sind diese Attribute in verschiedenen Kategorien aufgeführt.



| Category                                                                                       | BESCHREIBUNG                                                                                                              |
|------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [Alphabetische Referenz](alphabetical-attribute-reference.md)                                 | Stellt eine alphabetische Liste vordefinierter Attribute bereit.                                                                  |
| [Audioelemente](audio-item-attributes.md)                                                       | Listet Attribute auf, die für ein Audioelement in der Bibliothek verfügbar sind.                                                    |
| [CD-Wiedergabelisten](cd-playlist-attributes.md)                                                     | Listet Attribute auf, die für eine CD verfügbar sind.                                                                            |
| [CD-Spuren](cd-track-attributes.md)                                                           | Listet Attribute auf, die für eine CD-Spur verfügbar sind.                                                                      |
| [Häufig verwendete Windows Mediendateiattribute](commonly-used-windows-media-file-attributes.md) | Listet Attribute auf, die für eine digitale Mediendatei verfügbar sind.                                                            |
| [DVDs](dvd-attributes.md)                                                                     | Listet Attribute auf, die für eine DVD verfügbar sind.                                                                           |
| [Musik Dateien](music-file-attributes.md)                                                       | Listet Attribute auf, die für eine Musikdatei verfügbar sind, die sich nicht in der Bibliothek befindet.                                         |
| [Andere Elemente](other-item-attributes.md)                                                       | Listet Attribute auf, die für Elemente in der Bibliothek verfügbar sind, die keine Audio-, Video-, Wiedergabelisten-, Foto- oder Radioelemente sind. |
| [Fotoelemente](photo-item-attributes.md)                                                       | Listet Attribute auf, die für ein Fotoelement in der Bibliothek verfügbar sind.                                                     |
| [Wiedergabelisten](playlist-attributes-ref.md)                                                       | Listet Attribute auf, die für Wiedergabelisten verfügbar sind, die als Medienelemente aus der Bibliothek abgerufen werden.                             |
| [Funkelemente](radio-item-attributes.md)                                                       | Listet Attribute auf, die für ein Optionselement in der Bibliothek verfügbar sind.                                                     |
| [Videoelemente](video-item-attributes.md)                                                       | Listet Attribute auf, die für ein Videoelement in der Bibliothek verfügbar sind.                                                     |



 

Weitere Informationen zu den Metadatenattributen finden Sie in den [Richtlinien zur Verwendung von Windows Medienmetadaten.](/previous-versions/ms867702(v=msdn.10))
