# <a name="attribute-reference"></a>Attribut Verweis

Der Verweis auf das Windows Media Player-Attribut dokumentiert Metadatenattribute, die für Windows Media Player SDK-Entwickler von Interesse sind. In der folgenden Tabelle sind diese Attribute in verschiedenen Kategorien aufgeführt.



| Category                                                                                       | BESCHREIBUNG                                                                                                              |
|------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [Alphabetischer Verweis](alphabetical-attribute-reference.md)                                 | Stellt eine alphabetische Liste von vordefinierten Attributen bereit.                                                                  |
| [Audioelemente](audio-item-attributes.md)                                                       | Listet die Attribute auf, die für ein Audioelement in der Bibliothek verfügbar sind.                                                    |
| [CD-Wiedergabelisten](cd-playlist-attributes.md)                                                     | Listet die für eine CD verfügbaren Attribute auf.                                                                            |
| [CD-Spuren](cd-track-attributes.md)                                                           | Listet die Attribute auf, die für einen CD-Track verfügbar sind.                                                                      |
| [Häufig verwendete Windows Media-Dateiattribute](commonly-used-windows-media-file-attributes.md) | Listet die Attribute auf, die für eine digitale Mediendatei verfügbar sind.                                                            |
| [DVDs](dvd-attributes.md)                                                                     | Listet die Attribute auf, die für eine DVD verfügbar sind.                                                                           |
| [Musikdateien](music-file-attributes.md)                                                       | Listet die Attribute auf, die für eine Musikdatei verfügbar sind, die nicht in der Bibliothek vorhanden ist.                                         |
| [Andere Elemente](other-item-attributes.md)                                                       | Listet die Attribute auf, die für Elemente in der Bibliothek verfügbar sind, bei denen es sich nicht um Audioelemente, Video-, Wiedergabelisten-, Foto-oder Radio |
| [Foto Elemente](photo-item-attributes.md)                                                       | Listet die Attribute auf, die für ein Foto Element in der Bibliothek verfügbar sind.                                                     |
| [Wiedergabelisten](playlist-attributes-ref.md)                                                       | Listet die Attribute auf, die für Wiedergabelisten verfügbar sind, die als Medienelemente aus der Bibliothek abgerufen werden.                             |
| [Options Felder](radio-item-attributes.md)                                                       | Listet die Attribute auf, die für ein Radio Element in der Bibliothek verfügbar sind.                                                     |
| [Video Elemente](video-item-attributes.md)                                                       | Listet die Attribute auf, die für ein Video Element in der Bibliothek verfügbar sind.                                                     |



 

Weitere Informationen zu den Metadatenattributen finden Sie in den [Richtlinien zur Verwendung von Windows Media-Metadaten](/previous-versions/ms867702(v=msdn.10)).
