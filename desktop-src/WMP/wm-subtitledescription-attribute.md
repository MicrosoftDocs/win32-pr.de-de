# <a name="wmsubtitledescription-attribute"></a>WM/SubTitleDescription-Attribut

Das **WM/SubTitleDescription-Attribut** gibt zusätzliche Untertitelinformationen an.

## <a name="applies-to"></a>Gilt für

-   [Videoelemente](video-item-attributes.md)

## <a name="remarks"></a>Hinweise

Die Windows Media Format SDK-Konstante für dieses Attribut ist g \_ wszWMSubTitleDescription.

Um zu bestimmen, ob Sie den Wert dieses Attributs ändern können, verwenden Sie die [Media.isReadOnlyItem-Methode.](media-isreadonlyitem.md)

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------|
| Version<br/> | Windows Media Player 10 oder höher<br/> |
