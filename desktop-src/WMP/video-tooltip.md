# <a name="videotooltip"></a>Video. ToolTip

Das **ToolTip** -Attribut gibt den QuickInfo-Text für das Videofenster an oder ruft ihn ab.

``` syntax
        elementID.toolTip
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese- **/schreibzeichenfolge**, die nicht länger als 1024 Zeichen sein darf. Er besitzt keinen Standardwert.

## <a name="remarks"></a>Bemerkungen

Wenn dieses Attribut auf "" (leere Zeichenfolge) festgelegt ist, wird keine QuickInfo angezeigt.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player, Version 7,0 oder höher<br/> |
