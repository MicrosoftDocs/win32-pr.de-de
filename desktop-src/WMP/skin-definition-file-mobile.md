# <a name="skin-definition-file-mobile"></a>Skin-Definitionsdatei (Mobile)

In der Skin-Definitionsdatei werden alle Teile zusammengefasst. Sie definieren die zu verwendenden Images, geben die Benutzeroberflächen Elemente an, die der Benutzer sehen soll, und geben die Funktionen, Koordinaten, Dateinamen usw. ein.
