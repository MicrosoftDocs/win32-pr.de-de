# <a name="iwmpmetadatatextdescription-property"></a>IWMPMetadataText::D escription (Eigenschaft)

Die- `Description` Eigenschaft ruft eine Beschreibung des metadatentexts ab.

## <a name="syntax"></a>Syntax


```CSharp
public System.String Description {get; set;}
```


```VB

Public ReadOnly Property Description As System.String
```





## <a name="property-value"></a>Eigenschaftswert

Ein **System. String** -Wert, der die Beschreibung ist.

## <a name="remarks"></a>Bemerkungen

Vor der Verwendung dieser Eigenschaft müssen Sie über Lesezugriff auf die Bibliothek verfügen. Weitere Informationen finden Sie unter [Bibliotheks Zugriff](library-access.md).
