# <a name="iwmpnetworkmaxbitrate-property"></a>IWMPNetwork::maxBitRate (Eigenschaft)

Die **maxBitRate-Eigenschaft** ruft die maximal mögliche Videobitrate ab.

## <a name="syntax"></a>Syntax


```CSharp
public System.Int32 maxBitRate {get; set;}
```


```VB

Public ReadOnly Property maxBitRate As System.Int32
```





## <a name="property-value"></a>Eigenschaftswert

Ein **System.Int32,** das die maximale Bitrate ist.
