# <a name="playerapplicationswitchtoplayerapplication-method"></a>Playerapplication. switchtoplayerapplication-Methode

Die **switchtoplayerapplication** -Methode schaltet ein Remotes Windows Media Player-Steuerelement in den vollständigen Modus des Players.

## <a name="syntax"></a>Syntax


```JScript
PlayerApplication.switchToPlayerApplication()
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.

## <a name="remarks"></a>Bemerkungen

Diese Methode wird nur verwendet, wenn das Windows Media Player-Steuerelement Remoting ist.

**Windows Media Player 10 Mobile:** Diese Methode wird nicht unterstützt.
