# <a name="playerapplicationswitchtoplayerapplication-method"></a>PlayerApplication.switchToPlayerApplication-Methode

Die **switchToPlayerApplication-Methode** schaltet ein remotedes Windows Media Player-Steuerelement in den vollständigen Modus des Players um.

## <a name="syntax"></a>Syntax


```JScript
PlayerApplication.switchToPlayerApplication()
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.

## <a name="remarks"></a>Hinweise

Diese Methode wird nur beim Remoting des Windows Media Player verwendet.

**Windows Media Player 10 Mobile:** Diese Methode wird nicht unterstützt.
