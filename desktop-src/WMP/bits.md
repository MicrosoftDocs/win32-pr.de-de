# <a name="bits"></a>BITS

Die [Background Intelligent Transfer Service](/windows/desktop/Bits/background-intelligent-transfer-service-portal) ist ein Feature des Windows Betriebssystems. BITS überträgt Dateien (Downloads oder Uploads) zwischen einem Client und einem Server und stellt Statusinformationen im Zusammenhang mit den Übertragungen bereit. Wenn Sie Dateien mit C++-Code übertragen möchten, ist BITS die richtige Technologie. Wenn Sie Dateien mithilfe von JScript auf Ihrer Webseite im Onlineshop übertragen möchten, verwenden Sie stattdessen den Download-Manager.

Da der Windows Media Player Download-Manager BITS verwendet, können Sie Schritte ausführen, um Ihre BITS-Kopieraufträge so zu konfigurieren, dass Windows Media Player Aufträge für Sie verwaltet. Dazu müssen Sie die im Abschnitt Windows Media Player [BITS-Auftragskonvention](windows-media-player-bits-job-convention.md) beschriebene Konvention befolgen, wenn Sie Ihre Aufträge zur BITS-Übertragungswarteschlange hinzufügen. In diesem Abschnitt wird auch ein Mechanismus zum Abrufen von Informationen zu Ihren BITS-Aufträgen von der Webseite Ihrer Onlineshops beschrieben.
