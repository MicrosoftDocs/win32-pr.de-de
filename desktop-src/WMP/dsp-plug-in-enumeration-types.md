# <a name="dsp-plug-in-enumeration-types"></a>DSP-Plug-in-Enumerationstypen

Das Windows Media Player SDK implementiert den folgenden Enumerationstyp zum Erstellen von DSP-Plug-ins.



| Enumerationstyp                                        | BESCHREIBUNG                                                                                    |
|---------------------------------------------------------|------------------------------------------------------------------------------------------------|
| [Wmpplugin- \_ Caps](/previous-versions/windows/desktop/api/wmpservices/ne-wmpservices-wmpplugin_caps)                   | Wird mit **iwmpplugin:: getcaps** verwendet, um anzugeben, ob das Plug-in zwischen Formaten konvertieren kann. |
| [Wmpservices \_ streamstate](/previous-versions/windows/desktop/api/wmpservices/ne-wmpservices-wmpservices_streamstate) | Gibt an, ob der Stream derzeit beendet oder angehalten oder wiedergegeben wird.                     |
