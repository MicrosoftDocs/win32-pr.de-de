# <a name="windows-media-player-dsp-plug-ins"></a>Windows Media Player DSP-Plug-Ins

Microsoft Windows Media Player bietet eine Architektur, die es dem Benutzer ermöglicht, Plug-In-Programme zu installieren und zu aktivieren, die DSP-Funktionen (Digital Signal Processing) hinzufügen. Ein DSP-Plug-In ist ein Microsoft DirectX Media Object (DMO) oder eine Media Foundation Transform (MFT), die über COM-Schnittstellen eine Verbindung mit dem Player herstellt. Ein typisches DSP-Plug-In kann ein Audio equalizer oder ein Video Tönungssteuermaterial sein. Dieser Abschnitt des SDK enthält die Programmierinformationen, die Sie zum Erstellen Ihres eigenen DSP-Plug-Ins benötigen.

Die Dokumentation zum DSP-Plug-In ist in die folgenden drei Abschnitte unterteilt.
