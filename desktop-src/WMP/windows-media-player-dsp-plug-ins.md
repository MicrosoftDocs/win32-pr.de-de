# <a name="windows-media-player-dsp-plug-ins"></a>Windows Media Player DSP-Plug-ins

Microsoft Windows Media Player stellt eine Architektur bereit, die es dem Benutzer ermöglicht, Plug-in-Programme zu installieren und zu aktivieren, die DSP-Funktionen (Digital Signal Processing) hinzufügen. Ein DSP-Plug-in ist ein Microsoft DirectX Media Object (DMO) oder eine Media Foundation Transformation (MFT), die mithilfe von COM-Schnittstellen eine Verbindung mit dem Player herstellt. Ein typisches DSP-Plug-in kann ein Audioequalizer oder ein Video-Tönungs-Steuerelement sein. Dieser Abschnitt des SDK enthält die Programmierinformationen, die Sie benötigen, um Ihr eigenes DSP-Plug-in zu erstellen.

Die DSP-Plug-in-Dokumentation ist in die folgenden drei Abschnitte unterteilt.
