# <a name="wmpvideo"></a>Wmpvideo

Dies ist ein vordefiniertes **Video** Element mit den folgenden Standardwerten.

``` syntax
backgroundColor="black"
horizontalAlignment="stretch"
verticalAlignment="stretch"
```

## <a name="remarks"></a>Bemerkungen

Dadurch wird ein **Video** Element erstellt, mit dem das Videofenster gestreckt wird, wenn die Größe der Skin geändert wird. Das in diesem Fenster angezeigte digitale Video wird an den verfügbaren Platz angepasst oder entsprechend den Einstellungen im Menü Player **Ansicht** unter **Video size** zentral hoch-oder herunterskaliert. Das Menü **Ansicht** wird im vollständigen Modus des Players angezeigt, oder wenn das **View. TitleBar** -Attribut in einem Skin auf true festgelegt ist.

Alle Eigenschaften dieses **Video** Elements können überschrieben werden, indem Sie explizit angegeben werden.
