# <a name="wmpvideo"></a>WMPVIDEO

Dies ist ein vordefiniertes **VIDEO-Element** mit den folgenden Standardwerten.

``` syntax
backgroundColor="black"
horizontalAlignment="stretch"
verticalAlignment="stretch"
```

## <a name="remarks"></a>Hinweise

Dadurch wird ein **VIDEO-Element** erstellt, mit dem das Videofenster gestreckt wird, wenn die Größe der Skin geändert wird. Digitale Videos, die in diesem Fenster angezeigt werden, passen an den verfügbaren Platz an oder werden gemäß den Einstellungen im Menü **Playeransicht** unter **Videogröße** hoch- oder herunterskaliert. Das Menü **Ansicht** wird im vollständigen Modus des Players oder angezeigt, wenn das **VIEW.titleBar-Attribut** in einer Skin auf TRUE festgelegt ist.

Alle Eigenschaften dieses **VIDEO-Elements** können überschrieben werden, indem sie explizit angegeben werden.
