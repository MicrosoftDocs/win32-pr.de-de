# <a name="alternate-images"></a>Alternative Images

Alternative Bilder sind Bilder, die nicht sichtbar sind, wenn die Skin anfänglich angezeigt wird, sondern als Reaktion auf Mausereignisse sichtbar werden. Beispielsweise können Sie die Skin so einrichten, dass sich die Kunst in diesem Bereich ändert, wenn ein Mauszeiger über einen bestimmten Bereich bewegt wird. Dadurch können Benutzer visuell informiert werden, dass Sie auf einen Bereich klicken können, um etwas zu tun. Sie können auch ein alternatives Bild verwenden, wenn der Benutzer auf einen Bereich klickt, um ihm visuelles Feedback zu geben, dass der Klick von Windows Media Player empfangen wurde.
