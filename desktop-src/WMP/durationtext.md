# <a name="durationtext"></a>Durationtext

Dies ist ein vordefiniertes **Text** Element mit den folgenden Standardwerten.

``` syntax
value="wmpprop:player.currentMedia.DurationString"
tabstop="true"
justification="right"
```

## <a name="remarks"></a>Bemerkungen

Dadurch wird ein **Text** Element erstellt, das die Dauer der Wiedergabe der Medien anzeigt. Alle Eigenschaften dieses **Text** Elements können überschrieben werden, indem Sie explizit angegeben werden.
