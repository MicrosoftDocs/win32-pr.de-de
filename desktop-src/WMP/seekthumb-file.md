# <a name="seekthumb-file"></a>Seekthumb-Datei

Die seekthumb-Datei definiert das Bild, das die Wiedergabe Position innerhalb des Medien Elements für eine Such Trackleiste angibt. Sie können eine Datei verwenden und für die Verwendung durch seekthumb und volumethumschlag definieren. Im Gegensatz zu den anderen kunstdateien werden die Thumb-Dateien im trackbars-Abschnitt der Skin-Definitionsdatei definiert.

Das folgende Bild ist eine typische seekthumb-Datei.

![seekthumb-Datei](images/cesdksee.gif)

Diese beiden Schaltflächen Dateien sehen identisch aus, unterscheiden sich jedoch geringfügig. Das linke Bild ist die normale Ansicht, die dem Benutzer angezeigt wird, und das rechte Bild ist das gedrückte Bild, das dem Benutzer angezeigt wird, wenn Sie auf die Schaltfläche tippen.

> [!Note]  
> Seekthumb-Dateien, die für in Windows Media Player 10 Mobile oder höher verwendete Skins erstellt wurden, müssen die folgenden drei Bilder aufweisen (von links nach rechts): normal, per pushübertragung und deaktiviert.

 

Möglicherweise möchten Sie bestimmte Bereiche der Thumb-Bilder transparent machen. Auf diese Weise können Sie Thumb-Bilder in anderen Formen als rechteckig erstellen. Alle Bereiche eines Ziehpunkt Bilds, die Sie mit der Farbe ausfüllen, die durch den RGB-Wert 255, 0, 255 angegeben wird, werden in der Skin als transparent angezeigt.
