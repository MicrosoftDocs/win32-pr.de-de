# <a name="buttongroupclick"></a>ButtonGroup. Click

Die **Click** -Methode ruft den **OnClick** -Ereignishandler auf, der für das **ButtonElement** mit dem angegebenen Index definiert ist.

``` syntax
        elementID.click(button)
```

## <a name="parameters"></a>Parameter

<dl> <dt>

<span id="button"></span><span id="BUTTON"></span>*gedrückt*
</dt> <dd>

**Zahl** (**Long**), die den Index des **buttonelements** mit dem aufzurufenden **OnClick** -Ereignishandler enthält.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.

## <a name="remarks"></a>Bemerkungen

Verwenden Sie diese Methode, um eine alternative Methode zum Ausführen des Codes bereitzustellen, der einer Schaltfläche zugeordnet ist, die in der **ButtonGroup** enthalten ist.

|--------------------|---------------------------------------------------|
