# <a name="listboxfirstvisibleitem"></a>ListBox. firstvisibleitem

Das **firstvisibleitem** -Attribut gibt den Index der ersten sichtbaren Zeile im Listenfeld-Steuerelement an oder ruft ihn ab.

``` syntax
        elementID.firstVisibleItem
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/schreibzahl (**Long**). 

## <a name="remarks"></a>Bemerkungen

Dieses Attribut ist nützlich zum Bestimmen der ersten sichtbaren Zeile, wenn ein Bildlauf für das Listenfeld ausgeführt wird.
