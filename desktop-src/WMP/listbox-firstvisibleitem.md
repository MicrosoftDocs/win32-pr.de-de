# <a name="listboxfirstvisibleitem"></a>LISTBOX.firstVisibleItem

Das **firstVisibleItem-Attribut** gibt den Index der ersten sichtbaren Zeile im Listenfeld-Steuerelement an oder ruft den Index ab.

``` syntax
        elementID.firstVisibleItem
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/Schreibnummer (**long**). 

## <a name="remarks"></a>Hinweise

Dieses Attribut ist nützlich, um die erste sichtbare Zeile beim Scrollen des Listenfelds zu bestimmen.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------------------|
| Version<br/> | Windows Media Player für Windows XP oder höher<br/> |
