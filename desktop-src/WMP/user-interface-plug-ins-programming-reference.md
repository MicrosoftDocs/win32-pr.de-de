# <a name="user-interface-plug-ins-programming-reference"></a>Programmierreferenz zu Benutzeroberfläche Plug-Ins

Das Microsoft Windows Media Player Software Development Kit (SDK) unterstützt eine Schnittstelle für Benutzeroberflächen-Plug-Ins. Im folgenden Abschnitt wird dies ausführlich dokumentiert.



| `Section`                                                  | BESCHREIBUNG                                                                                                   |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [WMPNotifyPluginAddRemove](/previous-versions/windows/desktop/api/wmpplug/nf-wmpplug-wmpnotifypluginaddremove) | Eine unabhängige Funktion, mit der Windows Media Player benachrichtigt werden, dass ein Plug-In installiert oder deinstalliert wurde. |
| [IWMPPluginUI-Schnittstelle](/previous-versions/windows/desktop/api/wmpplug/nn-wmpplug-iwmppluginui)               | Eine Schnittstelle zu Benutzeroberflächen-Plug-Ins.                                                                                  |
| [Registrierungsflags](registration-flags.md)             | Eine Liste der Einstellungsflags, die sich auf das Verhalten von Benutzeroberflächen-Plug-Ins auswirken.                                             |
