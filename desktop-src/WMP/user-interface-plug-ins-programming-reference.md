# <a name="user-interface-plug-ins-programming-reference"></a>Programmier Referenz zu Benutzeroberflächen-Plug-ins

Das Microsoft Windows Media Player Software Development Kit (SDK) unterstützt eine Schnittstelle für UI-Plug-ins. Dies wird im folgenden Abschnitt ausführlich beschrieben.



| `Section`                                                  | BESCHREIBUNG                                                                                                   |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [Wmpnotifypluginadressmove](/previous-versions/windows/desktop/api/wmpplug/nf-wmpplug-wmpnotifypluginaddremove) | Eine unabhängige Funktion, mit der Windows Media Player benachrichtigt wird, dass ein Plug-in installiert oder deinstalliert wurde. |
| [Iwmppluginui-Schnittstelle](/previous-versions/windows/desktop/api/wmpplug/nn-wmpplug-iwmppluginui)               | Eine Schnittstelle zu UI-Plug-ins.                                                                                  |
| [Registrierungsflags](registration-flags.md)             | Eine Liste von einstellungsflags, die sich auf das Verhalten von UI-Plug-ins auswirken.                                             |
