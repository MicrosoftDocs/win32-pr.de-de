# <a name="object-model-migration-guide"></a>Handbuch für die Migration des Objektmodells

In Windows Media Player 7 wurde ein neues Objektmodell eingeführt, das einen umfangreichen neuen Funktions Satz bietet, mit dem Sie Ihre Webseiten erweitern können. Nachfolgende Versionen haben das Objektmodell der Version 7 mit zusätzlichen Eigenschaften, Methoden und Ereignissen erweitert. Das neue-Objektmodell unterscheidet sich jedoch erheblich vom Windows Media Player 6,4-Objektmodell. Sie müssen also wissen, wie Sie Ihren vorhandenen Code aktualisieren können, wenn Sie Windows Media Player 7 und höher in Ihren Projekten unterstützen möchten.

Ein Beispiel, das veranschaulicht, wie Sie die Windows Media Player-Version und den aktuellen Browser des Benutzers erkennen, finden Sie im Beispiel mit dem Namen "Detection", das mit diesem SDK installiert wird. Weitere Informationen zu Beispielen finden Sie unter [Beispiele](samples.md).

Dieses Handbuch bezieht sich auf das neue Objektmodell als das Objektmodell Windows Media Player 7 oder höher. Ausführliche Informationen zu den Funktionen, die in den einzelnen Versionen von Windows Media Player verfügbar sind, finden Sie unter [Objektmodell Referenz für die Skript](object-model-reference-for-scripting.md)Erstellung.
