# <a name="object-model-migration-guide"></a>Leitfaden zur Objektmodellmigration

Windows Media Player 7 wurde ein neues Objektmodell eingeführt, das einen neuen Funktionsumfang bietet, mit dem Sie Ihre Webseiten verbessern können. In nachfolgenden Versionen wurde das Objektmodell der Version 7 um zusätzliche Eigenschaften, Methoden und Ereignisse erweitert. Das neue Objektmodell unterscheidet sich jedoch erheblich vom Windows Media Player 6.4-Objektmodell, daher müssen Sie wissen, wie Sie Ihren vorhandenen Code aktualisieren, wenn Sie Windows Media Player 7 und höher in Ihren Projekten unterstützen möchten.

Ein Beispiel, das veranschaulicht, wie die Windows Media Player-Version und der aktuelle Browser des Benutzers erkannt werden, finden Sie im Beispiel mit dem Namen "detection", das mit diesem SDK installiert wird. Weitere Informationen zu Beispielen finden Sie unter [Beispiele.](samples.md)

Dieser Leitfaden bezieht sich auf das neue Objektmodell als Windows Media Player 7- oder höher-Objektmodell. Weitere Informationen dazu, welche Funktionen in den einzelnen Versionen von verfügbar sind, Windows Media Player unter [Object Model Reference for Scripting (Objektmodellreferenz für die Skripterstellung).](object-model-reference-for-scripting.md)
