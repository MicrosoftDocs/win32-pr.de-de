# <a name="windows-media-playlist-elements-reference"></a>Referenz zu Windows Media-Wiedergabelisten Elementen

Dieser Abschnitt enthält die Referenz Dokumentation für alle Elemente der Windows Media-Wiedergabeliste. Verweis Einträge enthalten Definitionen der Elemente, ihrer Attribute und Werte sowie spezielle Bedingungen im Zusammenhang mit den einzelnen Elementen.

Die folgenden Wiedergabelisten Elemente werden unterstützt.



| Element                                    | BESCHREIBUNG                                                                                                                                                                                          |
|--------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [argument](argument-element.md)           | Enthält einen Teil einer Bedingungs Zeichenfolge.                                                                                                                                                          |
| [body](body-element.md)                   | Enthält die-Elemente, die den Inhalt einer Wiedergabeliste definieren.                                                                                                                                        |
| [filter](filter-element.md)               | Enthält Elemente, die die Größe einer Wiedergabeliste, die Dauer einer Wiedergabeliste oder die Anzahl von Medien Elementen in einer Wiedergabeliste einschränken.                                                                                 |
| [Bruch](fragment-element.md)           | Gibt eine Bedingung der Abfrage an, die Elemente aus der Bibliothek auswählt.                                                                                                                            |
| [head](head-element.md)                   | Enthält Metadaten, die auf die gesamte Wiedergabeliste angewendet werden.                                                                                                                                               |
| [media](media-element.md)                 | Gibt eines der Medienelemente in einer Wiedergabeliste an.                                                                                                                                                      |
| [Analysen](meta-element.md)                   | Gibt Metadaten an, die auf die gesamte Wiedergabeliste angewendet werden.                                                                                                                                              |
| [Queryset](queryset-element.md)           | Enthält Elemente, die definieren, welche Medienelemente aus der Bibliothek ausgewählt werden.                                                                                                                   |
| [Seq](seq-element.md)                     | Enthält Elemente, die eine statische Wiedergabeliste oder Elemente definieren, die eine intelligente Wiedergabeliste definieren.                                                                                                            |
| [smartwiedergabe](smartplaylist-element.md) | Enthält den dynamisch definierten Teil einer Wiedergabeliste.                                                                                                                                              |
| [smil](smil-element.md)                   | Das **SMIL** -Element ist immer das Element der obersten Ebene in einer Windows Media-Wiedergabe Datei (WPL). Er gibt an, dass die Datei SMIL-Syntax und-Grammatik (synchronisiert Multimedia Integration Language) verwendet. |
| [sourceFilter](sourcefilter-element.md)   | Enthält Elemente, die Elemente aus einer Bibliothek auswählen.                                                                                                                                                  |
| [title](title-element--wpl.md)            | Gibt einen Titel für die Wiedergabeliste an.                                                                                                                                                                  |
