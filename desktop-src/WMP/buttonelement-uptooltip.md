# <a name="buttonelementuptooltip"></a>BUTTONELEMENT.upToolTip

Das **upToolTip-Attribut** gibt den QuickInfo-Text an oder ruft diesen ab, der angezeigt wird, wenn sich der Mauszeiger über **BUTTONELEMENT** befindet und **sich BUTTONELEMENT** im Auf-Zustand befindet.

``` syntax
        elementID.upToolTip
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine  Lese-/Schreibzeichenfolge mit dem Standardwert "" (leere Zeichenfolge) mit einer maximalen Länge von 1.024 Zeichen.

## <a name="remarks"></a>Hinweise

Wenn dieses Attribut auf "" (leere Zeichenfolge) festgelegt ist, wird keine QuickInfo angezeigt.

## <a name="examples"></a>Beispiele

Ein [Beispiel, das](buttonelement-mappingcolor.md) die Verwendung dieses Attributs veranschaulicht, finden Sie unter mappingColor-Attribut.
