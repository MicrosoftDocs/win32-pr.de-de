# <a name="buttonelementuptooltip"></a>ButtonElement. uptooltip

Das **uptooltip** -Attribut gibt den QuickInfo-Text an oder ruft ihn ab, der angezeigt wird, wenn sich der Mauszeiger über dem **Button-Element** befindet und das **ButtonElement** den Status "up" aufweist.

``` syntax
        elementID.upToolTip
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese- **/schreibzeichenfolge** mit dem Standardwert "" (leere Zeichenfolge) mit einer maximalen Länge von 1024 Zeichen.

## <a name="remarks"></a>Bemerkungen

Wenn dieses Attribut auf "" (leere Zeichenfolge) festgelegt ist, wird keine QuickInfo angezeigt.

## <a name="examples"></a>Beispiele

Ein Beispiel zur Veranschaulichung der Verwendung dieses Attributs finden Sie im [mappingColor](buttonelement-mappingcolor.md) -Attribut.
