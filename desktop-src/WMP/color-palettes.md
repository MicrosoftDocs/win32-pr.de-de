# <a name="color-palettes-windows-media-player-sdk"></a>Farbpaletten (Windows Media Player SDK)

Verschiedene Geräte verfügen über unterschiedliche Farbpaletten. Wahrscheinlich möchten Sie für jede Farbpalette verschiedene Skins erstellen. Die folgenden Farbpaletten werden empfohlen:

-   16 Bit
-   8 Bit
-   2-Bit-Graustufen
