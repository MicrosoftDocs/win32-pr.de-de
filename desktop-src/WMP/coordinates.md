# <a name="coordinates"></a>Koordinaten

Dadurch werden die x- und y-Koordinaten des Bilds relativ zur Hintergrunddatei definiert. Zwei positive ganze Zahlen sind erforderlich, getrennt durch ein Komma. Es gibt zwei Arten von Koordinaten, die Sie verwenden müssen, abhängig vom Typ des Bilds, das Sie definieren. Die folgende Tabelle zeigt die zulässigen Koordinatenwerte für Bitmaptypen.



| Bitmaptyp               | Koordinatenwerte                                                             |
|---------------------------|-------------------------------------------------------------------------------|
| Hintergrund oder Super       | 0,0                                                                           |
| Deaktiviert, Push oder Region | Die linke obere Ecke des Bilds relativ zum Hintergrundbild in Pixel. |
