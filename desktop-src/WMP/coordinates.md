# <a name="coordinates"></a>Koordinaten

Dadurch werden die x-und y-Koordinaten des Bilds relativ zur Hintergrund Datei definiert. Zwei positive ganze Zahlen sind erforderlich, getrennt durch ein Komma. Es gibt zwei Arten von Koordinaten, die Sie verwenden müssen, abhängig vom Typ des Bilds, das Sie definieren. In der folgenden Tabelle sind die zulässigen Koordinaten Werte für Bitmaptypen aufgeführt.



| Bitmaptyp               | Koordinaten Werte                                                             |
|---------------------------|-------------------------------------------------------------------------------|
| Hintergrund oder Super       | 0, 0                                                                           |
| Deaktiviert, Push oder Region | Die linke obere Ecke des Bilds relativ zum Hintergrundbild in Pixel. |
