# <a name="iwmplibrarymediacollection-property"></a>Iwmplibrary:: mediacollection (Eigenschaft)

Die **mediacollection** -Eigenschaft ruft eine **iwmpmediacollection** -Schnittstelle für die aktuelle Bibliothek ab.

## <a name="syntax"></a>Syntax


```CSharp
public IWMPMediaCollection mediaCollection {get; set;}
```


```VB

Public ReadOnly Property mediaCollection As IWMPMediaCollection
```





## <a name="property-value"></a>Eigenschaftswert

Eine WMPLib. iwmpmediacollection-Schnittstelle für die aktuelle Bibliothek.
