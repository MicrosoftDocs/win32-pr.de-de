# <a name="installing-visualizations"></a>Installieren von Visualisierungen

Sie müssen einen Installationsvorgang für den Benutzer Ihrer Visualisierung bereitstellen. Außerdem müssen Sie einen Deinstallations Vorgang für den Benutzer bereitstellen. Von der aktuellen Version von Windows Media Player werden keine Visualisierungen von der Benutzeroberfläche installiert.

## <a name="installing-to-the-visualization-folder"></a>Installieren im Ordner "Visualisierung"

Es wird empfohlen, dass Sie alle Visualisierungen im Unterordner Visualisierungen des Ordners installieren, in dem Windows Media Player installiert ist.

## <a name="registering-your-visualization"></a>Registrieren der Visualisierung

Visualisierungen sind COM-DLLs und befolgen alle normalen Installations-und Entfernungs Regeln. Sie können regsvr32.exe oder andere Installations Tools verwenden, um die Visualisierung zu registrieren.
