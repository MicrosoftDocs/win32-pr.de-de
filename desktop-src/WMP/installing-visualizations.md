# <a name="installing-visualizations"></a>Installieren von Visualisierungen

Sie müssen einen Installationsvorgang für den Benutzer Ihrer Visualisierung bereitstellen. Sie müssen auch einen Deinstallationsprozess für den Benutzer bereitstellen. Die aktuelle Version von Windows Media Player installiert keine Visualisierungen über die Benutzeroberfläche.

## <a name="installing-to-the-visualization-folder"></a>Installieren im Visualisierungsordner

Es wird empfohlen, alle Visualisierungen im Unterordner Visualisierungen des Ordners zu installieren, in dem Windows Media Player installiert ist.

## <a name="registering-your-visualization"></a>Registrieren Ihrer Visualisierung

Visualisierungen sind COM-DLLs und befolgen alle normalen Regeln für die Installation und Entfernung. Sie können ihre regsvr32.exe oder andere Installationstools registrieren.
