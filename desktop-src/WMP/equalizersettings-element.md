# <a name="equalizersettings-element"></a>EQUALIZERSETTINGS-Element

Das **EQUALIZERSETTINGS-Element** bietet eine Möglichkeit, den grafischen Equalizer und andere Audioeinstellungen von Windows Media Player mit den hier aufgeführten Attributen und Methoden zu bearbeiten.

Das **EQUALIZERSETTINGS-Element** unterstützt die folgenden Attribute.



| attribute                                                          | Beschreibung                                                                                             |
|--------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [Bänder](equalizersettings-bands.md)                               | Ruft die Anzahl der unterstützten Frequenzbereiche ab.                                                      |
| [Umgehen](equalizersettings-bypass.md)                             | Gibt einen Wert an, der angibt, ob der Equalizerfilter im Filterdiagramm umgangen wird, oder ruft einen Wert ab. |
| [Überblenden](equalizersettings-crossfade.md)                       | Gibt einen Wert an, der angibt, ob Das Überblenden aktiviert ist, oder ruft einen Wert ab.                                |
| [crossFadeWindow](equalizersettings-crossfadewindow.md)           | Gibt die Überlappungsmenge in Millisekunden an oder ruft diese ab.                                |
| [currentPreset](equalizersettings-currentpreset.md)               | Gibt den Index der aktuellen Voreinstellung an oder ruft den Index ab.                                                 |
| [currentPresetTitle](equalizersettings-currentpresettitle.md)     | Ruft den Titel der aktuellen Voreinstellung ab.                                                              |
| [currentSpeakerName](equalizersettings-currentspeakername.md)     | Ruft den Namen der aktuellen Sprechereinstellung ab.                                                      |
| [enableSplineTension](equalizersettings-enablesplinetension.md)   | Gibt einen Wert an, der angibt, ob spline-Beeinträchtigung aktiviert oder deaktiviert ist, oder ruft einen Wert ab.                |
| [enhancedAudio](equalizersettings-enhancedaudio.md)               | Gibt einen Wert an, der angibt, ob erweiterte Audiodaten aktiviert sind, oder ruft einen Wert ab.                          |
| [gainLevels](equalizersettings-gainlevels.md)                     | Gibt die Verstärkungsebene des Bands an, der dem bereitgestellten Index entspricht, oder ruft sie ab.                  |
| [gainLevel1](equalizersettings-gainlevel1.md)                     | Gibt die Verstärkungsebene von Band 1 an oder ruft sie ab.                                                        |
| [gainLevel2](equalizersettings-gainlevel2.md)                     | Gibt die Verstärkungsebene von Band 2 an oder ruft sie ab.                                                        |
| [gainLevel3](equalizersettings-gainlevel3.md)                     | Gibt die Verstärkungsebene von Band 3 an oder ruft sie ab.                                                        |
| [gainLevel4](equalizersettings-gainlevel4.md)                     | Gibt die Verstärkungsebene von Band 4 an oder ruft sie ab.                                                        |
| [gainLevel5](equalizersettings-gainlevel5.md)                     | Gibt die Verstärkungsebene von Band 5 an oder ruft sie ab.                                                        |
| [gainLevel6](equalizersettings-gainlevel6.md)                     | Gibt die Verstärkungsebene von Band 6 an oder ruft sie ab.                                                        |
| [gainLevel7](equalizersettings-gainlevel7.md)                     | Gibt die Verstärkungsebene von Band 7 an oder ruft sie ab.                                                        |
| [gainLevel8](equalizersettings-gainlevel8.md)                     | Gibt die Verstärkungsebene von Band 8 an oder ruft sie ab.                                                        |
| [gainLevel9](equalizersettings-gainlevel9.md)                     | Gibt die Verstärkungsebene von Band 9 an oder ruft sie ab.                                                        |
| [gainLevel10](equalizersettings-gainlevel10.md)                   | Gibt die Verstärkungsebene von Band 10 an oder ruft sie ab.                                                       |
| [Normalisierung](equalizersettings-normalization.md)               | Gibt einen Wert an, der angibt, ob die Normalisierung aktiviert ist, oder ruft einen Wert ab.                             |
| [normalisierungAverage](equalizersettings-normalizationaverage.md) | Ruft den durchschnittlichen Normalisierungswert ab.                                                              |
| [normalizationPeak](equalizersettings-normalizationpeak.md)       | Ruft den Höchstnormalisierungswert ab.                                                                 |
| [presetCount](equalizersettings-presetcount.md)                   | Ruft die Anzahl der verfügbaren Voreinstellungen ab.                                                              |
| [speakerSize](equalizersettings-speakersize.md)                   | Gibt den Index der aktuellen Sprechergröße an oder ruft den Index ab.                                           |
| [splineTension](equalizersettings-splinetension.md)               | Gibt die Splineauslösung für das Equalizer-Steuerelement an oder ruft sie ab.                                    |
| [truBassLevel](equalizersettings-trubasslevel.md)                 | Gibt die SRS TruBass-Ebene an oder ruft sie ab.                                                           |
| [wowLevel](equalizersettings-wowlevel.md)                         | Gibt die SRS WOW Effect-Ebene an oder ruft sie ab.                                                        |



 

Das **EQUALIZERSETTINGS-Element** unterstützt die folgenden Methoden.



| Methode                                                 | Beschreibung                                                          |
|--------------------------------------------------------|----------------------------------------------------------------------|
| [nextPreset](equalizersettings-nextpreset.md)         | Wendet die nächste Voreinstellung für den Equalizer an.                                   |
| [presetTitle](equalizersettings-presettitle.md)       | Ruft den Namen der Equalizervoreinstellung mit dem angegebenen Index ab. |
| [previousPreset](equalizersettings-previouspreset.md) | Wendet die vorherige Voreinstellung für den Equalizer an.                               |
| [reset](equalizersettings-reset.md)                   | Setzt die Verstärkungsebenen aller Bänder auf null Dezibel zurück.                |



 

Das **EQUALIZERSETTINGS-Element** kann die [ \_ Onchange-Ereignishandler](attribute-onchange.md) des Attributs implementieren.
