# <a name="buttongroupbuttoncount"></a>ButtonGroup. Button count

Das **ButtonCount** -Attribut ruft die Anzahl der Schaltflächen in der **ButtonGroup** ab.

``` syntax
        elementID.buttonCount
```

## <a name="possible-values"></a>Mögliche Werte

Bei diesem Attribut handelt es sich um eine schreibgeschützte **Zahl** (**Long**).

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9-Serie oder höher<br/> |
