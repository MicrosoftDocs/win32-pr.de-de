# <a name="initial-lines"></a>Anfängliche Zeilen

Jede Skin-Definitionsdatei für Windows Media Player Mobile muss mit einer Zeile beginnen, in der die Versionsnummer der Skin-Datei angegeben ist. Die folgende Zeile funktioniert für Windows Media Player 9-Serie für Windows Mobile 2003 oder höher:


```C++
[Pocket WMP Skin File v9.0.1]

```
