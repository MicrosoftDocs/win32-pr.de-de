# <a name="about-the-metadatapicture-object"></a>Informationen zum MetadataPicture-Objekt

Das **WM/Bild-** Metadatenattribut bietet ausführliche Informationen zu metadatenbildern, wie z. b. Albumkunst. Das **MetadataPicture** -Objekt bietet eine Möglichkeit zum Abrufen der einzelnen Daten, die mit dem Bild verknüpft sind. Sie können sich das **WM/Bild-** Metadatenattribut als Verbund Attribut vorstellen. Wenn Sie *Medien* verwenden. **getItemInfoByType** zum Abrufen von Informationen über ein metadatenimage rufen Sie ein **MetadataPicture** -Objekt ab, das über Eigenschaften verfügt, die die einzelnen Daten enthalten.
