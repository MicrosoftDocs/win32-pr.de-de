# <a name="about-the-metadatapicture-object"></a>Informationen zum MetadataPicture-Objekt

Das **WM/Picture-Metadatenattribut** enthält ausführliche Informationen zu Metadatenbildern, z. B. Albumart. Das **MetadataPicture-Objekt** bietet eine Möglichkeit, die einzelnen Daten im Zusammenhang mit dem Bild abzurufen. Sie können sich das **WM/Picture-Metadatenattribut** als Verbundattribut vorstellen. Wenn Sie *Medien* verwenden. **getItemInfoByType** zum Abrufen von Informationen zu einem Metadatenbild rufen Sie ein **MetadataPicture-Objekt** ab, das Eigenschaften enthält, die die einzelnen Daten enthalten.
