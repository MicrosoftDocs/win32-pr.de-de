# <a name="windows-media-player-sdk-enumeration-types"></a>Windows Media Player SDK-Enumerationstypen

Das Windows Media Player SDK unterstützt die folgenden Enumerationstypen.



| Enumerationstyp                                                             | BESCHREIBUNG                                                                                                            |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [Wmpburnformat](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpburnformat)                                           | Definiert die möglichen Arten von CDs zum Brennen.                                                                         |
| [Wmpburnstate](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpburnstate)                                             | Definiert die möglichen Betriebszustände von Windows Media Player beim Brennen einer CD.                                      |
| [Wmpdevicestatus](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpdevicestatus)                                       | Definiert die möglichen Werte für den aktuellen Status eines Geräts.                                                        |
| [Wmpfolderscanstate](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpfolderscanstate)                                 | Definiert die möglichen Betriebszustände von Windows-Media Player, da Datei Ordner für digitale Medieninhalte überwacht werden. |
| [Wmplibrarytype](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmplibrarytype)                                         | Definiert die möglichen Bibliothekstypen, mit denen Windows Media Player eine Verbindung herstellen kann.                                          |
| [Wmpopenstate](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpopenstate)                                             | Definiert die möglichen Betriebszustände von Windows Media Player beim Öffnen einer digitalen Mediendatei.                      |
| [Wmpplaylistchangeeventtype](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpplaylistchangeeventtype)                 | Definiert die Arten von Änderungen, die an einer Wiedergabeliste vorgenommen werden können.                                                           |
| [WMPPlayState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpplaystate)                                             | Definiert die möglichen Betriebszustände von Windows Media Player, wenn eine digitale Mediendatei wiedergegeben wird.                      |
| [Wmpripstate](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpripstate)                                               | Definiert die möglichen Betriebszustände von Windows Media Player beim Rips einer CD.                                       |
| [Wmpstringcollectionchangeeventtype](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpstringcollectionchangeeventtype) | Definiert die Typen von Änderungen, die in einer Zeichen folgen Auflistung auftreten können.                                                    |
| [Wmpsyncstate](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpsyncstate)                                             | Definiert die möglichen Betriebszustände von Windows Media Player während der Synchronisierung digitaler Medien mit einem Gerät.          |
