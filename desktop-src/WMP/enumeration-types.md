# <a name="windows-media-player-sdk-enumeration-types"></a>Windows Media Player SDK-Enumerationstypen

Das Windows Media Player SDK unterstützt die folgenden Enumerationstypen.



| Enumerationstyp                                                             | Beschreibung                                                                                                            |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [WMP Auswendformat](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpburnformat)                                           | Definiert die möglichen Typen von CDs für die Aushändigung.                                                                         |
| [WMPState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpburnstate)                                             | Definiert die möglichen Betriebszustände von Windows Media Player, wenn es eine CD verbraucht.                                      |
| [WMPDeviceStatus](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpdevicestatus)                                       | Definiert die möglichen Werte für den aktuellen Status eines Geräts.                                                        |
| [WMPFolderScanState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpfolderscanstate)                                 | Definiert die möglichen Betriebszustände von Windows Media Player beim Überwachen von Dateiordnern auf digitale Medieninhalte. |
| [WMPLibraryType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmplibrarytype)                                         | Definiert die möglichen Bibliothekstypen, mit denen Windows Media Player eine Verbindung herstellen können.                                          |
| [WMPOpenState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpopenstate)                                             | Definiert die möglichen Betriebszustände von Windows Media Player beim Öffnen einer digitalen Mediendatei.                      |
| [WMPPlaylistChangeEventType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpplaylistchangeeventtype)                 | Definiert die Arten von Änderungen, die an einer Wiedergabeliste vorgenommen werden können.                                                           |
| [WMPPlayState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpplaystate)                                             | Definiert die möglichen Betriebszustände von Windows Media Player, während eine digitale Mediendatei wiedergegeben wird.                      |
| [WMPRipState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpripstate)                                               | Definiert die möglichen Betriebszustände von Windows Media Player, wenn eine CD zerrissen wird.                                       |
| [WMPStringCollectionChangeEventType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpstringcollectionchangeeventtype) | Definiert die Arten von Änderungen, die in einer Zeichenfolgenauflistung auftreten können.                                                    |
| [WMPSyncState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpsyncstate)                                             | Definiert die möglichen Betriebszustände von Windows Media Player, wenn digitale Medien mit einem Gerät synchronisiert werden.          |
