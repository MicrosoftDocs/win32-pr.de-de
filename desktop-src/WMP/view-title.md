# <a name="viewtitle"></a>View. Title

Das **Title** -Attribut gibt den Titel der **Ansicht** an oder ruft ihn ab.

``` syntax
        elementID.title
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine **Zeichenfolge** , die zur Entwurfszeit angegeben wird, und anschließend schreibgeschützt. Der Wert kann bis zu 255 Zeichen lang sein und hat keinen Standardwert.

## <a name="remarks"></a>Bemerkungen

Der angegebene oder abgerufene Titel ist der Anzeige Name für das Design.

|--------------------|------------------------------------------------------|
