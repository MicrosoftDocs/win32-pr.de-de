# <a name="debugging-the-page"></a>Debuggen der Seite

Wenn Windows Media Player Ihre Online Store-Webseite hostet, wird die Möglichkeit deaktiviert, Dialogfelder über die Webseite zu öffnen. Zum Debuggen des Webseiten Codes sollten Sie stattdessen Code hinzufügen, der die Ausgabe temporär an ein HTML-Element auf der Webseite ausgibt, z. b. ein div-oder Text-Element.
