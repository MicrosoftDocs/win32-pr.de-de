# <a name="listitemcsv"></a>listitem.csv

Diese Datei enthält Einträge für jedes Element, das in einer Liste enthalten ist. Jeder Eintrag ist eine Zeile, die aus den Feldern mit Tabstopptrennzeichen besteht, die in der folgenden Tabelle beschrieben sind. Felder müssen in der aufgeführten Reihenfolge angezeigt werden.

Alle Felder sind erforderlich.

Die Spalte Format in der folgenden Tabelle beschreibt die Formatierung der einzelnen Unicode-Textfelder. Er bezieht sich nicht auf den Datentyp des Inhalts. Wenn beispielsweise Integer in der Spalte Format angegeben ist, bedeutet dies, dass das Feld eine Unicode-Zeichenfolge enthält, die einen ganzzahligen Wert anstelle einer tatsächlichen ganzen Zahl darstellt.



| Feld              | Format                                          | Beschreibung                                                                                                            |
|--------------------|-------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| Verknüpfte \_ ListID     | Nicht negative ganze Zahl. Beispiel: 465<br/>    | Die ID der Liste, zu der dieses Element gehört.                                                                               |
| Verknüpftes \_ ListItem   | Nicht negative ganze Zahl. Beispiel: 684793<br/> | Die ID des Elements. Kann die ID eines Titels, eines Albums, eines Interpreten, eines Genre, eines Untergenres, eines Radiofeeds oder einer anderen Liste sein. |
| ItemPositionInList | Nicht negative ganze Zahl. Beispiel: 5<br/>      | Die Position des Elements in seiner Liste. Das erste Element in einer Liste befindet sich an Position 0.                                   |
