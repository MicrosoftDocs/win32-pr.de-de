# <a name="listitemcsv"></a>listitem.csv

Diese Datei enthält Einträge für alle Elemente, die in einer Liste enthalten sind. Jeder Eintrag ist eine Zeile, die aus den durch Tabstopps getrennten Feldern besteht, die in der folgenden Tabelle beschrieben werden. Felder müssen in der aufgeführten Reihenfolge angezeigt werden.

Alle Felder sind erforderlich.

In der Spalte Format in der folgenden Tabelle wird beschrieben, wie jedes Unicode-Textfeld formatiert wird. Er verweist nicht auf den Datentyp des Inhalts. Wenn beispielsweise eine ganze Zahl in der Spalte Format angegeben ist, bedeutet dies, dass das Feld eine Unicode-Zeichenfolge enthält, die einen ganzzahligen Wert anstelle einer tatsächlichen Ganzzahl darstellt.



| Feld              | Format                                          | BESCHREIBUNG                                                                                                            |
|--------------------|-------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| Verknüpfte \_ ListId     | Nicht negative ganze Zahl. Beispiel: 465<br/>    | Die ID der Liste, zu der dieses Element gehört.                                                                               |
| Verknüpftes \_ ListItem   | Nicht negative ganze Zahl. Beispiel: 684793<br/> | Die ID des Elements. Kann die ID eines Titels, eines Albums, eines Künstlers, eines Genres, eines Subgenres, eines Radio Feeds oder einer anderen Liste sein. |
| Itempositioninlist | Nicht negative ganze Zahl. Beispiel: 5<br/>      | Die Position des Elements in der Liste. Das erste Element in einer Liste befindet sich an Position 0.                                   |
