# <a name="about-the-sample-video-dsp-plug-in"></a>Informationen zum Beispielvideo-DSP-Plug-In

Das Beispielvideo-DSP-Plug-In bietet eine einfache Verarbeitungsimplementierung, die die Farbsättigung des Videos um einen Faktor skaliert, der vom Benutzer auf der Eigenschaftenseite bereitgestellt wird. Das Plug-In akzeptiert Werte zwischen 0 und 1. Der Standardwert ist 1. Der Wert 1 hat keine Auswirkungen auf das Videobild. andere Skalierungsfaktoren desaturieren die Bildfarbe. Beispielsweise würde ein Wert von 0,5 zu einer Farbsättigung von 50 Prozent führen. Der Wert 0 (null) führt zu Graustufenvideos.

Das DSP-Beispielvideo-Plug-In funktioniert mit den folgenden Videoformaten:

-   NV12
-   YV12
-   YUY2
-   UYVIG
-   RGB32
-   RGB24
-   RGB555
-   RGB565
