# <a name="buttongroupradio"></a>ButtonGroup. Radio

Das **Radio** -Attribut gibt einen Wert an bzw. Ruft einen Wert ab, der angibt, ob die **ButtonGroup** aus Options Feldern besteht.

``` syntax
        elementID.radio
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist ein **boolescher** Wert mit Lese-/Schreibzugriff.



| Wert | BESCHREIBUNG                                      |
|-------|--------------------------------------------------|
| true  | Die **ButtonGroup** ist ein Optionsfeld.              |
| false | Standard. Die **ButtonGroup** ist kein Optionsfeld. |



 

## <a name="remarks"></a>Bemerkungen

Wenn **Radio** auf true festgelegt ist, werden alle Button **Element** -Elemente in der **ButtonGroup** als Kurznotiz angezeigt, aber immer nur eine Schaltfläche im Zustand "gedrückt".

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player, Version 7,0 oder höher<br/> |
