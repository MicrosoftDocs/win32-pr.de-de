# <a name="wmvideoheight-attribute"></a>WM/videoheight-Attribut

Das **WM/videoheight-** Attribut gibt die Höhe des Fotos oder des Videos in Pixel an.

## <a name="applies-to"></a>Gilt für

-   [Foto Elemente](photo-item-attributes.md)
-   [Video Elemente](video-item-attributes.md)

## <a name="remarks"></a>Bemerkungen

Dies ist ein Schreib geschütztes Attribut.

Die SDK-Konstante für das Windows Media-Format für dieses Attribut ist g \_ wszwmvideoheight.

Um zu ermitteln, ob Sie den Wert dieses Attributs ändern können, verwenden Sie die [Media. isread onlyitem](media-isreadonlyitem.md) -Methode.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------|
| Version<br/> | Windows Media Player 10 oder höher<br/> |
