# <a name="status-alignment"></a>Status Ausrichtung

Sie müssen die Ausrichtung der Statusanzeige definieren, die Sie verwenden möchten. In der folgenden Tabelle werden die gültigen Werte angezeigt.



| Wert  | BESCHREIBUNG                                                                       |
|--------|-----------------------------------------------------------------------------------|
| Links   | Der Text wird linksbündig ausgerichtet (mit dem linken Rand der Statusanzeige).   |
| Zentrum | Der Text wird in der Statusanzeige zentriert.                                       |
| Rechts  | Der Text wird rechtsbündig ausgerichtet (mit dem rechten Rand der Statusanzeige). |
