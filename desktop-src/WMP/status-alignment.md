# <a name="status-alignment"></a>Statusausrichtung

Sie müssen die Ausrichtung der Statusanzeige definieren, die Sie verwenden möchten. In der folgenden Tabelle sind die gültigen Werte aufgeführt.



| Wert  | BESCHREIBUNG                                                                       |
|--------|-----------------------------------------------------------------------------------|
| Links   | Der Text ist linksbündig ausgerichtet (am linken Rand der Statusanzeige angeordnet).   |
| Zentrum | Der Text wird in der Statusanzeige zentriert angezeigt.                                       |
| Right  | Der Text ist rechtsbündig ausgerichtet (am rechten Rand der Statusanzeige angeordnet). |
