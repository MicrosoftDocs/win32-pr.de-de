# <a name="trackbar-location"></a>Position der Trackleiste

Sie müssen den Speicherort der Trackleiste definieren, die Sie verwenden möchten. Sie müssen vier positive, durch Kommas getrennte ganze Zahlen angeben, die die linke, obere, breite und höhe der Trackleiste relativ zum Hintergrundbild in Pixel definieren.


```C++
180,46,50,30 

```
