# <a name="trackbar-location"></a>Position der TrackBar

Sie müssen den Speicherort der TrackBar definieren, die Sie verwenden möchten. Sie müssen vier positive ganze Zahlen, getrennt durch Kommas, bereitstellen, die die linke, obere, Breite und Höhe der Trackleiste in Pixel relativ zum Hintergrundbild definieren.


```C++
180,46,50,30 

```
