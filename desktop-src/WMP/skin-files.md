# <a name="skin-files"></a>Skin-Dateien

Wenn ein Skin angezeigt wird, sehen Sie nur die Oberfläche. Hinter der Oberfläche befinden sich mehrere Dateien, die die gesamte Skin bilden. Technisch gesehen handelt es sich bei einer Skin um eine Gruppe von Dateien, wobei jede Datei eine bestimmte Art von Informationen enthält. Hier sind die Dateitypen, die Sie verwenden können, um eine komplette Skin zu bilden.



| Dateityp                                        | BESCHREIBUNG                                                                                                                                                               |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Skin-Definitionsdatei](skin-definition-file.md) | Die Master Datei, die definiert, wie die anderen Dateien verwendet werden. Dies ist eine Textdatei mit der Dateinamenerweiterung. WMS.                                                    |
| [Kunstdateien](art-files.md)                       | Kunstdateien, die die grafischen Elemente Ihrer Skin enthalten. Dazu zählen Bitmap-, GIF-, JPEG-und PNG-Dateien.                                                                 |
| [JScript-Dateien](jscript-files.md)               | Wenn Sie kompliziertere Antworten auf Ereignisse erstellen möchten, können Sie mithilfe von Microsoft JScript Skriptdateien erstellen. Dabei handelt es sich um Textdateien mit der Dateinamenerweiterung. js. |
