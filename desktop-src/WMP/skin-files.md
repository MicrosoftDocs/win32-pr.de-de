# <a name="skin-files"></a>Skindateien

Wenn Sie eine Skin sehen, sehen Sie nur die Oberfläche. Hinter der Oberfläche befinden sich mehrere Dateien, aus denen sich die gesamte Skin macht. Technisch gesehen ist eine Skin eine Gruppe von Dateien, bei der jede Datei eine bestimmte Art von Informationen enthält. Hier sind die Dateitypen, die Sie verwenden können, um eine vollständige Skin zu erstellen.



| Dateityp                                        | BESCHREIBUNG                                                                                                                                                               |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Skindefinitionsdatei](skin-definition-file.md) | Die Masterdatei, die definiert, wie die anderen Dateien verwendet werden. Dies ist eine Textdatei mit der Erweiterung WMS.                                                    |
| [Art-Dateien](art-files.md)                       | Bilddateien, die die grafischen Elemente Ihrer Skin enthalten. Dazu gehören Bitmap-, GIF-, JPEG- und PNG-Dateien.                                                                 |
| [JScript Dateien](jscript-files.md)               | Wenn Sie kompliziertere Antworten auf Ereignisse erstellen möchten, können Sie Skriptdateien mithilfe von Microsoft JScript. Dabei handelt es sich um Textdateien, die eine .js haben. |
