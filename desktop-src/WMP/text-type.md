# <a name="text-type"></a>Texttyp

Sie müssen den Typ des Textanzeige Felds definieren, das Sie verwenden möchten. Die folgenden gültigen Typen sind in der folgenden Tabelle aufgeführt.



| type        | BESCHREIBUNG                                                                                                                                                                                                                                                                         |
|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Time        | Aktuelle Wiedergabe Position im Medien Element in Minuten und Sekunden.                                                                                                                                                                                                                 |
| Abspielen    | Der Name der aktuellen Wiedergabeliste.                                                                                                                                                                                                                                                       |
| Nachverfolgen\#     | Die Nummer des aktuellen Elements in der Wiedergabeliste.                                                                                                                                                                                                                                         |
| \#Spuren    | Anzahl der Elemente in der aktuellen Wiedergabeliste.                                                                                                                                                                                                                                            |
| Titel       | Der Titel des aktuellen Elements in der aktuellen Wiedergabeliste. Unter bestimmten Umständen wird dieser Typ möglicherweise nicht ordnungsgemäß angezeigt. Als Alternative können Sie Title als Teil eines Marquee verwenden.                                                                                                                 |
| Autor      | Der Autor des aktuellen Elements in der aktuellen Wiedergabeliste. Unter bestimmten Umständen wird dieser Typ möglicherweise nicht ordnungsgemäß angezeigt. Als Alternative können Sie den Autor als Teil eines Marquee verwenden.                                                                                                               |
| Copyright   | Copyright Daten des aktuellen Elements in der aktuellen Wiedergabeliste. Unter bestimmten Umständen wird dieser Typ möglicherweise nicht ordnungsgemäß angezeigt. Als Alternative können Sie das Copyright als Teil eines Marquee verwenden.                                                                                                    |
| Dateiname    | Der Dateiname des aktuellen Elements in der aktuellen Wiedergabeliste.                                                                                                                                                                                                                              |
| Dateinameext | Die Dateinamenerweiterung des aktuellen Elements in der aktuellen Wiedergabeliste. Die einzigen unterstützten Werte sind "WMA" und "WAV".                                                                                                                                                                 |
| Bitrate     | Die codierte Bitrate (in Kbit/s) des aktuellen Medien Elements.                                                                                                                                                                                                                           |
| Häufigkeit   | Die Stichproben Häufigkeit (in kHz) des aktuellen Medien Elements.                                                                                                                                                                                                                           |
| Stands\*    | Der Künstler, das Album, das Genre und die Bitrate des aktuellen Audioelements während der Wiedergabe oder die Bitrate und Untertitel (falls zutreffend) des aktuellen Video Elements. Wenn ein Medien Element nicht wiedergegeben wird, zeigt der statustexttyp den aktuellen Wiedergabe Zustand an (z. b. beendet oder angehalten). |



 

\*Verfügbar für Windows Media Player 10 Mobile oder höher.
