# <a name="eventaltkey"></a>Event. altKey

Das Attribut " **altKey** " Ruft einen Wert ab, der angibt, ob die Alt-Taste gedrückt wurde, als das Ereignis aufgetreten ist.

``` syntax
event.altKey
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist ein Schreib geschützter **boolescher** Wert.



| Wert | BESCHREIBUNG                                     |
|-------|-------------------------------------------------|
| true  | Gibt an, dass die Alt-Taste an der nach-unten-Position |
| false | Gibt an, dass sich die Alt-Taste an der Position oben befand.   |



 

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player, Version 7,0 oder höher<br/> |
