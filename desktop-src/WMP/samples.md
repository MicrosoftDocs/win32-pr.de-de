# <a name="wmp-sdk-samples"></a>WMP SDK-Beispiele

Das Windows Media Player SDK enthält eine Vielzahl von Beispielen, die viele der in diesem Dokument beschriebenen Programmiertechniken veranschaulichen.

-   Skriptbeispiele veranschaulichen das Einbetten des Player ActiveX-Steuerelements in eine Webseite. Ein vollständiges Webseitenbeispiel veranschaulicht, wie die aktuelle Version des Player-Steuerelements ActiveX wird.
-   Ein Skins-Beispiel veranschaulicht, wie sie die visuelle Darstellung und das Verhalten der Windows Media Player.
-   Ein Assistent für Microsoft Visual C++ kann ein einzigartiges Funktionierendes Beispiel für verschiedene Arten von Plug-Ins erstellen: ein Visualisierungs-Plug-In, ein Benutzeroberflächen-Plug-In, ein Rendering-Plug-In und ein Plug-In für die Verarbeitung digitaler Signale.
-   Beispiele für Onlineshops enthalten einen Assistenten für Microsoft Visual C++, der ein eindeutiges Funktionierendes Beispiel für ein COM-Objekt im Onlineshop erstellen kann.
-   Beispiele für Microsoft Visual C++ veranschaulichen das Einbetten des Windows Media Player ActiveX-Steuerelements, das Remoting des Player-ActiveX-Steuerelements und das Arbeiten mit der Bibliothek.
-   Ein C#-Beispiel veranschaulicht die Verwendung des Windows Media Player ActiveX -Steuerelements mit .NET Framework.
-   Ein Visual Basic .NET-Beispiel veranschaulicht die Verwendung des Windows Media Player ActiveX-Steuerelements mit dem .NET Framework.

Um die Beispiele verwenden zu können, müssen Sie das Windows SDK installieren, das das Windows Media Player SDK enthält. Um das Windows SDK herunterzuladen, wechseln Sie zur Seite [Windows SDK](https://msdn.microsoft.com/windows/bb980924.aspx) im Windows Developer Center.
