# <a name="user-interface-elements"></a>Benutzeroberflächenelemente

Obwohl Sie aus einer Vielzahl von Funktionen zum Steuern digitaler Medien, Wiedergabelisten und Volumes auswählen können, können Sie dem Benutzer Ihrer Skin eine Möglichkeit bereitstellen, um mit dieser Funktionalität zu interagieren. Wenn Sie z. b. möchten, dass der Benutzer mit der Wiedergabe eines Medien Elements beginnt, müssen Sie ein Benutzeroberflächen-Schaltflächen Element bereitstellen, das bewirkt, dass das Medien Element wiedergegeben wird, wenn der Benutzer auf das Bild der Schaltfläche tippt.

Es sind keine Elemente erforderlich. Wenn Sie jedoch nicht über einige Schaltflächen verfügen, um das Medien Element wiederzugeben und anzuhalten, wird nicht viel passiert. Es ist auch eine gute Idee, ein Volumen Steuerelement bereitzustellen.

Benutzeroberflächen Elemente entsprechen den Funktionstypen, die zum Steuern der Auswahl und Wiedergabe digitaler Medien verwendet werden.

Die folgenden Elemente werden bereitgestellt.



| Element                    | BESCHREIBUNG                                                                                                                                                                                                                                                             |
|----------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Schaltflächen](buttons.md)     | Viele Schaltflächen Typen werden bereitgestellt. Schaltflächen können verwendet werden, um Funktionen zu aktivieren oder zu deaktivieren oder zwischen zwei Funktionen zu wechseln.                                                                                                                                                        |
| [Trackbars](trackbars.md) | Sie können eine lineare Steuerung über trackbars bereitstellen. Sie möchten trackbars verwenden, um das Volume oder die Wiedergabe Position in einem Medien Element zu ändern.                                                                                                                         |
| [Text](text.md)           | Textfelder können verwendet werden, um dem Benutzer Text anzuzeigen. Eine gute Verwendung eines Textfelds besteht darin, den Namen des aktuell wiedergegebenen Titels anzuzeigen.                                                                                                                                      |
| [Marquee](marquee.md)     | Ein Marquee ist ein Bildlauf-Textfeld, das Sie verwenden können, um Text aus mehr als einer Text Kategorie anzuzeigen. Beispielsweise können Sie die Wiedergabeliste, die Anzahl der Spuren und die Wiedergabe von Zahlen anzeigen. Verwenden Sie einen Marquee, wenn Sie nicht über genügend Speicherplatz verfügen und eine große Anzahl an Anzeigemöglichkeiten haben. |
| [Status](status.md)       | Sie können eine Statusanzeige bereitstellen, die Sie verwenden können, um Textnachrichten automatisch anzuzeigen, die den Benutzer über den aktuellen Status von Windows-Media Player informieren. Beispielsweise wird der Benutzer mit der Statusanzeige informiert, wenn ein Medien Element gepuffert wird.                     |
| [Video](video.md)         | Wenn Sie Video abspielen möchten, können Sie in der Skin einen rechteckigen Videoanzeige Bereich bereitstellen.                                                                                                                                                                                |
| [Altersfreigabe](ratings.md)     | Sie können die Bewertung anzeigen, die allen Windows Media Audio basierten Inhalt zugeordnet ist, der zurzeit wiedergegeben wird.                                                                                                                                                             |
