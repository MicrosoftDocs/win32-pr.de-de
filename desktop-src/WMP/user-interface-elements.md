# <a name="user-interface-elements"></a>Benutzeroberflächenelemente

Obwohl Sie aus einer Vielzahl von Funktionen zum Steuern von digitalen Medien, Wiedergabelisten und Lautstärke wählen können, liegt es an Ihnen, dem Benutzer Ihrer Skin eine Möglichkeit zu bieten, mit dieser Funktionalität zu interagieren. Wenn sie beispielsweise möchten, dass der Benutzer mit der Wiedergabe eines Medienelements beginnt, müssen Sie ein Benutzeroberflächen-Schaltflächenelement bereitstellen, das die Wiedergabe des Medienelements verursacht, wenn der Benutzer auf das Bild ihrer Schaltfläche tippt.

Es sind keine Elemente erforderlich, aber wenn Sie nicht über einige Schaltflächen verfügen, um das Medienelement wieder zu spielen und zu beenden, passiert nicht viel. Es ist auch eine gute Idee, ein Volumesteuersystem zur Verfügung zu stellen.

Benutzeroberflächenelemente entsprechen Den Funktionstypen, die zum Steuern der Auswahl und Wiedergabe digitaler Medien verwendet werden.

Die folgenden Elemente werden bereitgestellt.



| Element                    | Beschreibung                                                                                                                                                                                                                                                             |
|----------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Schaltflächen](buttons.md)     | Es werden viele Schaltflächentypen bereitgestellt. Schaltflächen können verwendet werden, um Funktionen zu aktivieren oder zu deaktivieren oder zwischen zwei Funktionen zu wechseln.                                                                                                                                                        |
| [Trackbars](trackbars.md) | Sie können lineare Steuerung über Trackleisten bereitstellen. Sie sollten Trackleisten verwenden, um das Volume oder die Wiedergabeposition in einem Medienelement zu ändern.                                                                                                                         |
| [Text](text.md)           | Textfelder können verwendet werden, um dem Benutzer Text anzuzeigen. Eine gute Verwendung eines Textfelds besteht in der Anzeige des Namens des derzeit abspielten Titels.                                                                                                                                      |
| [Marquee](marquee.md)     | Ein Festzelt ist ein Bildlauftextfeld, mit dem Sie Text aus mehr als einer Textkategorie anzeigen können. Beispielsweise können Sie die Wiedergabeliste, die Anzahl der Spuren und die Wiedergabe der Titelnummer anzeigen. Verwenden Sie ein Festzelt, wenn Sie nicht viel Platz haben und viel anzuzeigen haben. |
| [Status](status.md)       | Sie können eine Statusanzeige bereitstellen, mit der Sie automatisch Textnachrichten anzeigen können, die den Benutzer über den aktuellen Zustand der Windows Media Player. Beispielsweise informiert die Statusanzeige den Benutzer, wenn ein Medienelement gepuffert wird.                     |
| [Video](video.md)         | Sie können einen rechteckigen Videoanzeigebereich in Ihrer Skin bereitstellen, wenn Sie Videos wieder geben möchten.                                                                                                                                                                                |
| [Altersfreigabe](ratings.md)     | Sie können die Bewertung anzeigen, die jedem Windows Medienaudio-basierten Inhalt zugeordnet ist, der gerade abspielt.                                                                                                                                                             |
