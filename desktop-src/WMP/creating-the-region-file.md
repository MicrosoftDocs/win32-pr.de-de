# <a name="creating-the-region-file"></a>Erstellen der Regions Datei

Sie sollten eine Regions Datei erstellen, in der die Bereiche angezeigt werden, auf die mit Abzweigungen reagiert wird. Sie können einfach die Basis Ebenen der einzelnen Elemente in eine neue Ebene kopieren und diese entsprechend den in der Skin-Definitionsdatei verwendeten Farbnummern Farben. Stellen Sie sicher, dass die Images, die Sie in dieser Ebene erstellen, einsolide sind und keine Effekte angewendet werden. Wahrscheinlich möchten Sie die Farbnummern der einzelnen Bilder notieren. Sie können die Farbnummern aus der Photoshop-Farbauswahl erhalten. Sie benötigen die RGB-Nummern, bei denen es sich um drei Dezimalzahlen handelt. Jede Zahl liegt zwischen 0 und 255. Ein voll roter roter Wert wäre z. b. 255, 0, 0.

> [!Note]  
> Regions Dateien werden in Windows Media Player 10 Mobile-Skins nicht verwendet, da Schaltflächen Typen in Windows Media Player 10 Mobile oder höher nicht unterstützt werden.
