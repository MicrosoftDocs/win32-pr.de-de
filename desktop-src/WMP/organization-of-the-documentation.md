# <a name="organization-of-the-documentation"></a>Organisation der Dokumentation

Das Windows Media Player SDK dokumentiert eine Vielzahl von Technologien im Zusammenhang mit Windows Media Player. Um die Suche nach den benötigten Informationen zu vereinfachen, ist die Dokumentation in Abschnitte unterteilt, die Sie dabei unterstützen, schnell Themen zu einem bestimmten Feature oder einer bestimmten Technologie zu finden.

Gegebenenfalls enthalten Hauptabschnitte drei Unterabschnitte: info, Programmier Handbuch und Referenz. Im Abschnitt about (Info) werden die jeweiligen Features oder Technologien beschrieben, damit Sie verstehen können, wie Sie Sie verwenden können. Im Abschnitt "Programmier Handbuch" werden Tutorials mit Schritt-für-Schritt-Anleitungen für die Technologie hinzugefügt. Im Referenz Abschnitt werden die Schnittstellen, Objekte, Methoden, Eigenschaften, Ereignisse, Elemente und Attribute erläutert, die für ihre Verwendung verfügbar sind.
