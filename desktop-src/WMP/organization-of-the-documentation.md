# <a name="organization-of-the-documentation"></a>Organisation der Dokumentation

Das Windows Media Player SDK dokumentiert eine Vielzahl von Technologien im Zusammenhang mit Windows Media Player. Um die Suche nach den benötigten Informationen zu vereinfachen, ist die Dokumentation in Abschnitte unterteilt, die Ihnen helfen, schnell Themen zu einem bestimmten Feature oder einer bestimmten Technologie zu finden.

Gegebenenfalls enthalten hauptabschnitte jeweils drei Unterabschnitte: Informationen, Programmierhandbuch und Referenz. Im Abschnitt Informationen wird das jeweilige Feature oder die Technologie beschrieben, damit Sie besser verstehen können, wie Sie es verwenden können. Im Abschnitt "Programmierhandbuch" werden Tutorials hinzugefügt, die Sie mit schrittweisen Anweisungen in die Technologie vertiefen. Im Abschnitt Referenz werden die Schnittstellen, Objekte, Methoden, Eigenschaften, Ereignisse, Elemente und Attribute beschrieben, die für Ihre Verwendung verfügbar sind.
