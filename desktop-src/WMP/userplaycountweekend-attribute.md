# <a name="userplaycountweekend-attribute"></a>Userplayzählweekend-Attribut

Das **userplayzählweekend** -Attribut gibt an, wie oft das Element an einem Samstag oder Sonntag wiedergegeben wurde.

## <a name="applies-to"></a>Gilt für

-   [Audioelemente](audio-item-attributes.md)
-   [Andere Elemente](other-item-attributes.md)
-   [Wiedergabelisten](playlist-attributes-ref.md)
-   [Video Elemente](video-item-attributes.md)

## <a name="remarks"></a>Bemerkungen

Dieses Attribut ist nur in der-Bibliothek gespeichert.

Um zu ermitteln, ob Sie den Wert dieses Attributs ändern können, verwenden Sie die [Media. isread onlyitem](media-isreadonlyitem.md) -Methode.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9-Serie oder höher<br/> |
