# <a name="windows-media-player-skins"></a>Windows Media Player Skins

Windows Media Player stellt eine Programmierplattform zum Erstellen von benutzerdefinierten Skins bereit. Skins sind Sätze von Skripts, Kunst-, Medien-und Textdateien, die kombiniert werden können, um eine neue Darstellung für Windows-Media Player zu erstellen. Mithilfe von Skins können Sie nicht nur die Art und Weise ändern, wie Windows Media Player aussieht, sondern auch, wie es funktioniert. Nicht nur an der Stelle, an der sich die Knöpfe befinden und wie Sie aussehen, sondern auch, was Sie tun, angesichts der Grenzen der zugrunde liegenden Windows Media Player-Kerntechnologie.

Die Skin-Technologie unterscheidet sich stark von anderen Programmier Typen. im wesentlichen mischen Sie die Programmierung und die Kunst in gleicher Teile. Sie müssen kein Experte Programmierer sein (es ist nicht viel mehr, als Sie bereits wissen, ob Sie Webseiten erstellt und eine einfache Skripterstellung durchgeführt haben), und Sie müssen kein Experte sein (solange Sie eine Grafik Bearbeitungs Anwendung verwenden können). Sie verwenden XML (ähnlich wie HTML), Microsoft JScript und die von Ihnen gewählten Kunstprogramme.
