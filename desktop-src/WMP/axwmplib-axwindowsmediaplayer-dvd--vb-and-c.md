# <a name="axwindowsmediaplayerdvd-property"></a>AxWindowsMediaPlayer.dvd-Eigenschaft

Die DVD-Eigenschaft ruft eine IWMPDVD-Schnittstelle ab, die Eigenschaften und Methoden für die Arbeit mit DVDs bereitstellt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```CSharp
public IWMPDVD dvd {get;}
```


```VB

Public ReadOnly Property dvd As IWMPDVD
```





## <a name="property-value"></a>Eigenschaftswert

Die WMPLib.IWMPDVD-Schnittstelle.
