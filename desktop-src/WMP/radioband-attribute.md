# <a name="radioband-attribute"></a>Radioband-Attribut

Das **Radioband-** Attribut ist das Frequency-Band der Radio Station.

## <a name="applies-to"></a>Gilt für

-   [Options Felder](radio-item-attributes.md)

## <a name="remarks"></a>Bemerkungen

Dieses Attribut ist nur in der-Bibliothek gespeichert.

Dieses Attribut weist einen der folgenden Werte auf: "FM", "am" oder "NET".

Um zu ermitteln, ob Sie den Wert dieses Attributs ändern können, verwenden Sie die [Media. isread onlyitem](media-isreadonlyitem.md) -Methode.
