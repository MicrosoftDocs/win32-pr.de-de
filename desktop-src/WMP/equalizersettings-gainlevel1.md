# <a name="equalizersettingsgainlevel1"></a>Equalizersettings. gainLevel1

Das **gainLevel1** -Attribut gibt die Gewinn Ebene von Band 1 an oder ruft Sie ab.

``` syntax
        elementID.gainLevel1
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/schreibnummer (**float**) mit einem Wert, der normalerweise zwischen 20 und + 20 liegt.  Der Standardwert ist 0 (null).

## <a name="remarks"></a>Bemerkungen

Mit diesem Attribut wird der Teil des Frequenzbereichs für die Audiofrequenz angepasst, der sich auf 31hz konzentriert.

Wenn dieses Attribut nicht angegeben wird, wird der vorherige Wert beibehalten.
