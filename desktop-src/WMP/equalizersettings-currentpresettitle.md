# <a name="equalizersettingscurrentpresettitle"></a>Equalizersettings. currentpresettitle

Das **currentpresettitle** -Attribut ruft den Titel der aktuellen Voreinstellung ab.

``` syntax
        elementID.currentPresetTitle
```

## <a name="possible-values"></a>Mögliche Werte

Bei diesem Attribut handelt es sich um eine schreibgeschützte **Zeichenfolge**.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player, Version 7,0 oder höher<br/> |
