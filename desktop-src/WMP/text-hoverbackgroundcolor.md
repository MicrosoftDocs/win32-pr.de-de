# <a name="texthoverbackgroundcolor"></a>Text. hoverbackgroundcolor

Das **hoverbackgroundcolor** -Attribut gibt die für das Text Steuerelement verwendete Hintergrundfarbe an oder ruft diese ab, wenn der Mauszeiger darüber bewegt wird.

``` syntax
        elementID.hoverBackgroundColor
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese- **/schreibzeichenfolge** mit einem beliebigen Microsoft Internet Explorer-Farbwert.

## <a name="remarks"></a>Bemerkungen

Wenn **hoverbackgroundcolor** nicht angegeben wird, wird **BackgroundColor** verwendet.

Im [value](text-value.md) -Attribut finden Sie ein Beispiel, das veranschaulicht, wie die Attribute des **Text** -Elements verwendet werden.

|--------------------|------------------------------------------------------|
