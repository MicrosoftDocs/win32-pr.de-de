Die Steuerung von Wiedergabelisten wird vom Client verwaltet. Wiedergabelisten werden von Windows Media Player analysiert, das auf den Elementen der Wiedergabeliste Aktionen ausführt. Der Client verfügt über die Kontrolle über das Streaming.
