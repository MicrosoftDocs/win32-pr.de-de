# <a name="control-of-playlists"></a>Steuerung von Wiedergabelisten

Die Steuerung von Wiedergabelisten wird vom Client verwaltet. Wiedergabelisten werden von der Windows Media Player analysiert, die Aktionen basierend auf den Elementen der Wiedergabeliste vor nimmt. Der Client hat die Kontrolle über die Streamingerfahrung.
