# <a name="videostretchtofit"></a>Video. stretchdefit

Das **stretchdefit** -Attribut gibt einen Wert an, der angibt, ob das Video auf die für das Video-Steuerelement definierte Breite und Höhe gestreckt wird.

``` syntax
        elementID.stretchToFit
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist ein **boolescher** Wert mit Lese-/Schreibzugriff.



| Wert | BESCHREIBUNG                                             |
|-------|---------------------------------------------------------|
| true  | Das Video wird so gestreckt, dass es dem Steuerelement entspricht.              |
| false | Standard. Das Video wird nicht an das Steuerelement angepasst. |



 

## <a name="remarks"></a>Bemerkungen

Wenn keine Breite oder Höhe angegeben wird, wird dieses Attribut ignoriert.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player, Version 7,0 oder höher<br/> |
