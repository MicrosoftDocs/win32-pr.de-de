# <a name="statustext"></a>Status Text

Dies ist ein vordefiniertes **Text** Element mit den folgenden Standardwerten.

``` syntax
value="wmpprop:player.status"
tabstop="true"
```

## <a name="remarks"></a>Bemerkungen

Dadurch wird ein **Text** Element erstellt, in dem der Wiedergabe Status angezeigt wird. Alle Eigenschaften dieses **Text** Elements können überschrieben werden, indem Sie explizit angegeben werden.
