# <a name="editboxbackgroundcolor"></a>EDITBOX.backgroundColor

Das **backgroundColor-Attribut** gibt die Hintergrundfarbe für das Bearbeitungsfeld-Steuerelement an oder ruft sie ab.

``` syntax
        elementID.backgroundColor
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine **Zeichenfolge** mit Lese-/Schreibzugriff, die einen beliebigen Microsoft Internet Explorer Farbwert oder den Wert "none" enthält. Er hat einen Standardwert, der der Hintergrundfarbe in Windows entspricht.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------------------|
| Version<br/> | Windows Media Player für Windows XP oder höher<br/> |
