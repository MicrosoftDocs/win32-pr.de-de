# <a name="playerapplication-element"></a>PLAYERAPPLICATION-Element

Das **PLAYERAPPLICATION-Element** kann von bestimmten Skins verwendet werden, um auf das **PlayerApplication-Objekt** zuzugreifen. Dieses Element bietet ähnliche Funktionen wie das globale **playerApplication-Attribut.** Dieses Element wird nur von Skins verwendet, die von C++-Anwendungen angezeigt werden, die eine Remoteinstanz von Windows Media Player einbetten.

Das **PLAYERAPPLICATION-Element** unterstützt die folgenden Attribute.



| attribute                                              | Beschreibung                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [hasDisplay](playerapplication-hasdisplay-scr.md)     | Ruft einen Wert ab, der angibt, ob das Video über das Remotesteuerelement Windows Media Player angezeigt werden kann. |
| [playerDocked](playerapplication-playerdocked-scr.md) | Ruft einen Wert ab, der angibt, ob sich Windows Media Player in einem andockten Zustand befindet.                          |
