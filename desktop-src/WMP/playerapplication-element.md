# <a name="playerapplication-element"></a>Playerapplication-Element

Das **playerapplication** -Element kann von bestimmten Skins verwendet werden, um auf das **playerapplication** -Objekt zuzugreifen. Dieses Element bietet eine ähnliche Funktionalität wie das globale Attribut **playerapplication** . Dieses Element wird nur von Skins verwendet, die von C++-Anwendungen angezeigt werden, die eine Remote Instanz von Windows Media Player-Steuerelement einbetten.

Das **playerapplication** -Element unterstützt die folgenden Attribute.



| Attribut                                              | BESCHREIBUNG                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [hasdisplay](playerapplication-hasdisplay-scr.md)     | Ruft einen Wert ab, der angibt, ob Videos über das Remote Fenster Media Player-Steuerelement angezeigt werden können. |
| [playerdocked](playerapplication-playerdocked-scr.md) | Ruft einen Wert ab, der angibt, ob sich Windows Media Player in einem angedockten Zustand befindet.                          |
