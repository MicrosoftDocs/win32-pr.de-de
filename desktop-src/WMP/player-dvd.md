# <a name="playerdvd"></a>Player.dvd

Die **dvd-Eigenschaft** ruft das DVD-Objekt ab.

## <a name="syntax"></a>Syntax

*Player* . **DVD**

## <a name="possible-values"></a>Mögliche Werte

Diese Eigenschaft ist ein schreibgeschütztes DVD-Objekt.
