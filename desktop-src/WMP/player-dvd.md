# <a name="playerdvd"></a>Player. DVD

Die **DVD** -Eigenschaft ruft das DVD-Objekt ab.

## <a name="syntax"></a>Syntax

*Player* . **DVD**

## <a name="possible-values"></a>Mögliche Werte

Diese Eigenschaft ist ein Schreib geschütztes DVD-Objekt.
