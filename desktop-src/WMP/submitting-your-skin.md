# <a name="submitting-your-skin"></a>Übermitteln der Skin

Die Verteilung ihrer Skin über die wmplugins-Website wird nicht mehr unterstützt. Sie können Ihre Skin stattdessen von einer Website Ihrer Wahl verteilen.
