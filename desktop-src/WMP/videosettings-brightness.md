# <a name="videosettingsbrightness"></a>Videosettings. Helligkeit

Das **Helligkeit** -Attribut gibt die Helligkeitseinstellung des Videos an oder ruft Sie ab.

``` syntax
        elementID.brightness
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/schreibzahl (**Long**) mit einem Wert im Bereich von-127 bis + 127 und einem Standardwert von 0. 

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|-----------------------------------------------------------------|
| Version<br/> | Windows Media Player Version für Windows XP oder höher<br/> |
