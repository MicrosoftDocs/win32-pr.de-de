# <a name="onclick"></a>OnClick

Der **OnClick** -Ereignishandler behandelt ein Ereignis, das auftritt, wenn der Benutzer auf das Element klickt.

``` syntax
onclick
```

## <a name="remarks"></a>Bemerkungen

Dieser Ereignishandler gilt nicht für das **Wiedergabe** Listenelement, das **Popup** Element, das **Video** Element, das **ListBox** -Element, wenn es sich um ein *Video* handelt. **Windows less** ist false, oder das **Effects** -Element bei *Effekten*. Das **Fenster ist "** true".
