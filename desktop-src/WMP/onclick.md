# <a name="onclick"></a>Onclick

Der **Onclick-Ereignishandler** behandelt ein Ereignis, das auftritt, wenn der Benutzer auf das Element klickt.

``` syntax
onclick
```

## <a name="remarks"></a>Hinweise

Dieser Ereignishandler gilt nicht für das **PLAYLIST-Element,** das **POPUP-Element,** das **VIDEO-Element,** das **LISTBOX-Element,** wenn *VIDEO*. **windowless ist** false oder das **EFFECTS-Element,** wenn *EFFECTS*. **windowed** ist true.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|-----------------------------------------------------|
| Version<br/> | Windows Media Player Version 70 oder höher<br/> |
