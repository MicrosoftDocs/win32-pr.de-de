# <a name="ambient-attributes"></a>Ambient-Attribute

Die Ambient-Attribute und-Methoden werden von allen geeigneten Skin-Elementen unterstützt, sofern nicht angegeben.

Die folgenden Attribute sind Ambient.



| Attribut                                                        | BESCHREIBUNG                                                                                                                               |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [accName](ambientattributes-accname.md)                         | Gibt einen Namen für ein beliebiges Element an oder ruft ihn ab.                                                                                            |
| [accdescription](ambientattributes-accdescription.md)           | Gibt eine Beschreibung für ein beliebiges Element an oder ruft diese ab.                                                                                     |
| [accKeyboardShortcut](ambientattributes-acckeyboardshortcut.md) | Gibt eine Tastenkombination für ein beliebiges Element an oder ruft diese ab.                                                                   |
| [AlphaBlend](ambientattributes-alphablend.md)                   | Gibt an, dass ein Wert für Alpha eine beliebige **Ansicht**, **unter Ansicht** oder ein UI-Widget kombiniert.                                                |
| [unten](ambientattributes-bottom.md)                           | Gibt die untere Koordinate des Steuer Elements an oder ruft diese ab.                                                                              |
| [clippingcolor](ambientattributes-clippingcolor.md)             | Gibt die Farbe an, die aus der **clippingimage** -Bitmap abgeschnitten werden soll, oder ruft diese ab.                                                           |
| [clippingimage](ambientattributes-clippingimage.md)             | Gibt den Bereich an, an den das Steuerelement zugeschnitten wird, oder ruft ihn ab.                                                                                 |
| [Element Type](ambientattributes-elementtype.md)                 | Ruft den Typ des Elements (z... **Schaltfläche**) ab.                                                                             |
| [enabled](ambientattributes-enabled.md)                         | Gibt einen Wert an, der angibt, ob das Steuerelement aktiviert oder deaktiviert ist, oder ruft diesen ab.                                                     |
| [height](ambientattributes-height.md)                           | Gibt die Höhe des Steuer Elements an oder ruft diese ab.                                                                                         |
| [horizontalAlignment](ambientattributes-horizontalalignment.md) | Gibt einen Wert an, der die horizontale Platzierung des Steuer Elements angibt, wenn die Größe der **Sicht** oder der übergeordneten **unter Ansicht** geändert wird, oder ruft diesen Wert ab. |
| [id](ambientattributes-id.md)                                   | Gibt den Bezeichner eines Steuer Elements an oder ruft ihn ab. Kann nur zur Entwurfszeit festgelegt werden.                                                       |
| [linken](ambientattributes-left.md)                               | Gibt die linke Koordinate des Steuer Elements an oder ruft diese ab.                                                                                |
| [ninegridmargin](ambientattributes-ninegridmargins.md)         | Gibt die Rand breiten für die nicht einheitliche Skalierung des Skin-Elements an.                                                                      |
| [passThrough](ambientattributes-passthrough.md)                 | Gibt einen Wert an, der angibt, ob das Steuerelement alle Mausereignisse an das Steuerelement unter ihm übergibt, oder ruft diesen Wert ab.                 |
| [resizeimages](ambientattributes-resizeimages.md)               | Gibt einen Wert an, der angibt, ob die Größe der im Steuerelement enthaltenen Bilder automatisch geändert wird, wenn sich die Größe des Steuer Elements ändert.     |
| [Richting](ambientattributes-right.md)                             | Gibt die rechte Koordinate des Steuer Elements an oder ruft diese ab.                                                                               |
| [tabStop](ambientattributes-tabstop.md)                         | Gibt einen Wert an, der angibt, ob das Steuerelement in der Reihenfolge der Tabstopps angezeigt wird, oder ruft diesen                                               |
| [top](ambientattributes-top.md)                                 | Gibt die obere Koordinate des Steuer Elements an oder ruft diese ab.                                                                                 |
| [verticalAlignment](ambientattributes-verticalalignment.md)     | Gibt einen Wert an, der die vertikale Platzierung des Steuer Elements angibt, wenn die Größe der **Sicht** oder der übergeordneten **unter Ansicht** geändert wird, oder ruft diesen Wert ab.   |
| [Barem](ambientattributes-visible.md)                         | Gibt die Sichtbarkeit des Steuer Elements an oder ruft diese ab.                                                                                     |
| [width](ambientattributes-width.md)                             | Gibt die Breite des Steuer Elements an oder ruft diese ab.                                                                                          |
| [ZIndex](ambientattributes-zindex.md)                           | Gibt die Reihenfolge an, in der das Steuerelement gerendert wird.                                                                        |



 

Die folgenden Methoden sind Ambient.



| Methode                                             | BESCHREIBUNG                                                                                                                                                          |
|----------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [alphablendto](ambientattributes-alphablendto.md) | Passt die **AlphaBlend** -Eigenschaft über einen Zeitraum an.                                                                                                           |
| ["muvesizeto"](ambientattributes-movesizeto.md)     | Verschiebt das-Steuerelement und gibt eine neue Größe für das-Steuerelement am neuen Speicherort an. Das-Steuerelement verschiebt und ändert seine Größe im Verlauf des angegebenen Zeitraums auf animierte Weise. |
| [MoveTo](ambientattributes-moveto.md)             | Verschiebt das-Steuerelement mit einer linearen Geschwindigkeit an eine neue Position.                                                                                                               |
| [mit der](ambientattributes-slideto.md)           | Verschiebt das-Steuerelement an eine neue Position. Das-Steuerelement bewegt sich im angegebenen Zeitraum nicht linear.                                                       |
