# <a name="ambient-attributes"></a>Ambient-Attribute

Die Ambient-Attribute und -Methoden werden von allen geeigneten Skinelementen unterstützt, sofern nicht angegeben.

Die folgenden Attribute sind ambient.



| attribute                                                        | Beschreibung                                                                                                                               |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [accName](ambientattributes-accname.md)                         | Gibt einen Namen für ein beliebiges Element an oder ruft einen Namen ab.                                                                                            |
| [accDescription](ambientattributes-accdescription.md)           | Gibt eine Beschreibung für ein beliebiges Element an oder ruft sie ab.                                                                                     |
| [accKeyboardShortcut](ambientattributes-acckeyboardshortcut.md) | Gibt eine Tastenkombinationsbeschreibung für ein beliebiges Element an oder ruft sie ab.                                                                   |
| [alphaBlend](ambientattributes-alphablend.md)                   | Gibt einen Wert für das Alphablending aller **VIEW-,** **SUBVIEW-** oder UI-Widgets an oder ruft sie ab.                                                |
| [Unteres](ambientattributes-bottom.md)                           | Gibt die untere Koordinate des Steuerelements an oder ruft sie ab.                                                                              |
| [clippingColor](ambientattributes-clippingcolor.md)             | Gibt die Farbe an, die aus der **ClippingImage-Bitmap** ausgeschnitten werden soll, oder ruft sie ab.                                                           |
| [clippingImage](ambientattributes-clippingimage.md)             | Gibt den Bereich an, an den das Steuerelement abgeschnitten werden soll, oder ruft diesen ab.                                                                                 |
| [Elementtype](ambientattributes-elementtype.md)                 | Ruft den Typ des Elements ab (z. B. **BUTTON**).                                                                             |
| [enabled](ambientattributes-enabled.md)                         | Gibt einen Wert an, der angibt, ob das Steuerelement aktiviert oder deaktiviert ist, oder ruft einen Wert ab.                                                     |
| [height](ambientattributes-height.md)                           | Gibt die Höhe des Steuerelements an oder ruft sie ab.                                                                                         |
| [Horizontalalignment](ambientattributes-horizontalalignment.md) | Gibt einen Wert an oder ruft einen Wert ab, der die horizontale Platzierung des Steuerelements angibt, wenn die GRÖßE der **VIEW-** oder übergeordneten **SUBVIEW** geändert wird. |
| [id](ambientattributes-id.md)                                   | Gibt den Bezeichner eines Steuerelements an oder ruft den Bezeichner ab. Kann nur zur Entwurfszeit festgelegt werden.                                                       |
| [Links](ambientattributes-left.md)                               | Gibt die linke Koordinate des Steuerelements an oder ruft sie ab.                                                                                |
| [nineGridMargins](ambientattributes-ninegridmargins.md)         | Gibt Randbreiten für die nicht einheitliche Skalierung des Skinelements an.                                                                      |
| [Passthrough](ambientattributes-passthrough.md)                 | Gibt einen Wert an, der angibt, ob das Steuerelement alle Mausereignisse an das steuerelement darunter übergibt, oder ruft ihn ab.                 |
| [resizeImages](ambientattributes-resizeimages.md)               | Gibt einen Wert an, der angibt, ob die Größe der im Steuerelement enthaltenen Bilder automatisch geändert wird, wenn sich die Größe des Steuerelements ändert, oder ruft diesen ab.     |
| [Richting](ambientattributes-right.md)                             | Gibt die rechte Koordinate des Steuerelements an oder ruft sie ab.                                                                               |
| [Tabstop](ambientattributes-tabstop.md)                         | Gibt einen Wert an, der angibt, ob sich das Steuerelement in der Tabulatorreihenfolge befindet, oder ruft einen Wert ab.                                               |
| [top](ambientattributes-top.md)                                 | Gibt die oberste Koordinate des Steuerelements an oder ruft sie ab.                                                                                 |
| [Verticalalignment](ambientattributes-verticalalignment.md)     | Gibt einen Wert an oder ruft einen Wert ab, der die vertikale Platzierung des Steuerelements angibt, wenn die GRÖßE der **VIEW-** oder übergeordneten **SUBVIEW** geändert wird.   |
| [Sichtbar](ambientattributes-visible.md)                         | Gibt die Sichtbarkeit des Steuerelements an oder ruft sie ab.                                                                                     |
| [width](ambientattributes-width.md)                             | Gibt die Breite des Steuerelements an oder ruft sie ab.                                                                                          |
| [Zindex](ambientattributes-zindex.md)                           | Gibt die Reihenfolge an, in der das Steuerelement gerendert wird, oder ruft sie ab.                                                                        |



 

Die folgenden Methoden sind ambient.



| Methode                                             | Beschreibung                                                                                                                                                          |
|----------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [alphaBlendTo](ambientattributes-alphablendto.md) | Passt die **alphaBlend-Eigenschaft** über einen bestimmten Zeitraum an.                                                                                                           |
| [moveSizeTo](ambientattributes-movesizeto.md)     | Verschiebt das Steuerelement und gibt eine neue Größe für das Steuerelement an der neuen Position an. Das Steuerelement wird im angegebenen Zeitraum animiert verschoben und seine Größe geändert. |
| [Moveto](ambientattributes-moveto.md)             | Verschiebt das Steuerelement mit linearer Geschwindigkeit an eine neue Position.                                                                                                               |
| [slideTo](ambientattributes-slideto.md)           | Verschiebt das Steuerelement an eine neue Position. Das Steuerelement bewegt sich im angegebenen Zeitraum nicht linear.                                                       |
