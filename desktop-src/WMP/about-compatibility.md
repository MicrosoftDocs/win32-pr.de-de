# <a name="about-compatibility"></a>Informationen zur Kompatibilität

Nicht jede Version von Windows Media Player Mobile funktioniert auf jedem Gerät. Außerdem sind einige Versionen auf dem Gerät vorinstalliert, während andere Versionen zum herunterladen und installieren auf der Microsoft-Website verfügbar sind. Spätere Versionen bieten möglicherweise Skin-Features, die von früheren Versionen nicht unterstützt werden.

In der folgenden Tabelle wird die Kompatibilität von Windows Media Player Mobile mit verschiedenen Geräten ausführlich erläutert.

| Windows Media Player 1,1          | **PCs der Größe von Palmen:** Geräte der Klasse "Casio Cassiopeia E-10 *x* "<br/> Compaq Aero 21 *xx* und Aero 15 *xx*<br/> HP Jornada 430<br/>                                      | Webrelease.                                                     | Nur Audiodaten.                                       |
