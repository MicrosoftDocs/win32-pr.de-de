# <a name="about-compatibility"></a>Informationen zur Kompatibilität

Nicht jede Version von Windows Media Player Mobile funktioniert auf jedem Gerät. Darüber hinaus sind einige Versionen auf dem Gerät vorinstalliert, während andere Versionen von der Microsoft-Website heruntergeladen und installiert werden können. Höhere Versionen bieten möglicherweise Skinfunktionen, die frühere Versionen nicht unterstützen.

In der folgenden Tabelle wird die Kompatibilität von Windows Media Player Mobile mit verschiedenen Geräten ausführlich aufgeführt.
