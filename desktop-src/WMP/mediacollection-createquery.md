# <a name="mediacollectioncreatequery-method"></a>Mediacollection. kreatequery-Methode

Die Methode " **kreatequery** " erstellt ein neues **Abfrage** Objekt.

## <a name="syntax"></a>Syntax


```JScript
retVal = MediaCollection.createQuery()
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt ein neues, leeres **Query** -Objekt zurück.

## <a name="remarks"></a>Bemerkungen

Das Erstellen eines neuen **Abfrage** Objekts ist der erste Schritt beim Erstellen einer Verbund Abfrage.
