# <a name="playlist-element"></a>Wiedergabelisten Element

Das **Wiedergabe** Listenelement bietet eine Möglichkeit, Medienelemente in einer Liste zu organisieren, um eine einfache Bearbeitung mithilfe der folgenden Attribute und Methoden zu ermöglichen. Vordefinierte **Wiedergabe** Listenelemente werden zur einfacheren Bereitstellung bereitgestellt. Angepasste Spalten können für eine Wiedergabeliste angegeben werden, indem Sie **Spalten** Elemente als untergeordnete Elemente des **Wiedergabe** Listen Elements einschließen.

Das **Wiedergabe** Listenelement unterstützt die folgenden Attribute.



| Attribut                                                                                 | BESCHREIBUNG                                                                                                                                |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [allowcolumnsortierung](playlist-allowcolumnsorting.md)                                     | Gibt einen Wert an, der angibt, ob die Sortierung für Spaltenheader zulässig ist, oder ruft diesen ab.                                                    |
| [Zuordnung](playlist-allowitemediting.md)                                         | Gibt einen Wert an, der angibt, ob Elemente in einer Wiedergabeliste direkte Bearbeitung unterstützen, oder ruft diesen Wert ab.                                            |
| [Background Color](playlist-backgroundcolor.md)                                           | Gibt die Hintergrundfarbe an oder ruft Sie ab.                                                                                               |
| [backgroundImage](playlist-backgroundimage.md)                                           | Gibt das Hintergrundbild an oder ruft es ab.                                                                                               |
| [checkboxesvisible](playlist-checkboxesvisible.md)                                       | Gibt einen Wert an, der angibt, ob Kontrollkästchen sichtbar sind, oder ruft ihn ab.                                                                  |
| [columnCount](playlist-columncount.md)                                                   | Ruft die Anzahl der angezeigten Spalten ab.                                                                                                     |
| [ColumnOrder](playlist-columnorder.md)                                                   | Gibt die Reihenfolge der Wiedergabelisten Spalten an oder ruft diese ab.                                                                                  |
| [Spalten](playlist-columns.md)                                                           | Definiert die Spalten, die im **Wiedergabe** Listenelement angezeigt werden.                                                                               |

| [Vorgänge](playlist-copying.md)                                                           | Ruft einen Wert ab, der angibt, ob das **Wiedergabe** Listenelement den Kopiervorgang durchläuft.                                                    |
| [disableditemcolor](playlist-disableditemcolor.md)                                       | Gibt die Farbe einer deaktivierten CD-Spur oder von Online Inhalten im Offline Modus an oder ruft diese ab.                                                 |

| [itemerrorcolor](playlist-itemerrorcolor.md)                                             | Gibt die Hervorhebungs Farbe an, die ein Wiedergabelisten Element mit einer Fehlerbedingung angibt, oder ruft diese ab.                                     |

| [itemplayingcolor](playlist-itemplayingcolor.md)                                         | Gibt die Hervorhebungs Farbe an oder ruft diese ab, die das aktuell wiedergegebene Element in der Wiedergabeliste angibt.                                      |

Das **Wiedergabe** Listenelement unterstützt die folgenden Methoden.



| Methode                                                              | BESCHREIBUNG                                                                                                               |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [Abbrechen](playlist-abortcopy.md)                                 | Bricht einen Kopiervorgang ab.                                                                                                 |
| [addselectedtoplaylist](playlist-addselectedtoplaylist.md)         | Fügt der Wiedergabeliste das ausgewählte Element hinzu.                                                                                   |
| [copy](playlist-copy.md)                                           | Startet einen Kopiervorgang von der CD.                                                                                      |
| [Delta gewählt](playlist-deleteselected.md)                       | Löscht das ausgewählte Element aus der Wiedergabeliste.                                                                              |
| [Delta-electedfromlibrary](playlist-deleteselectedfromlibrary.md) | Löscht das ausgewählte Element aus der Wiedergabeliste und aus der Bibliothek.                                                         |
| [getnextcheckeditem](playlist-getnextcheckeditem.md)               | Ruft den Index des nächsten aktivierten Elements in der Wiedergabeliste nach dem angegebenen Index ab.                               |
| [getNextCheckedItem2](playlist-getnextcheckeditem2.md)             | Ruft den Index des nächsten aktivierten Elements in der Wiedergabeliste nach dem angegebenen Index ab. Funktioniert mit in der Liste zusammengefügten Wiedergabelisten  |
| [getnextselecteditem](playlist-getnextselecteditem.md)             | Ruft den Index des nächsten ausgewählten Elements in der Wiedergabeliste nach dem angegebenen Index ab.                              |
| [getNextSelectedItem2](playlist-getnextselecteditem2.md)           | Ruft den Index des nächsten ausgewählten Elements in der Wiedergabeliste nach dem angegebenen Index ab. Funktioniert mit in der Liste zusammengefügten Wiedergabelisten |
| ["muveselecteddown"](playlist-moveselecteddown.md)                   | Verschiebt das ausgewählte Element in der Liste um eine Position nach unten.                                                                    |
| ["muveselectedup"](playlist-moveselectedup.md)                       | Verschiebt das ausgewählte Element in der Liste um eine Position nach oben.                                                                      |
| [setcheckedstate](playlist-setcheckedstate.md)                     | Gibt an, dass das indizierte Element in der Wiedergabeliste aktiviert ist.                                                               |
| [setCheckedState2](playlist-setcheckedstate2.md)                   | Legt den aktivierten Zustand des Elements mit dem angegebenen Index im **Wiedergabe** Listenelement fest. Funktioniert mit in der Liste zusammengefügten Wiedergabelisten     |
| [setcolumnresizemode](playlist-setcolumnresizemode.md)             | Gibt die Größe der indizierten Spalten an.                                                                            |
| [setcolumnwidth](playlist-setcolumnwidth.md)                       | Gibt die Spaltenbreite an und ändert den Größen Änderungs Modus der Spalte in "wmpcrmfixed".                                    |
| [setselectedstate](playlist-setselectedstate.md)                   | Gibt an, dass das indizierte Element in der Wiedergabeliste ausgewählt ist.                                                              |
| [setSelectedState2](playlist-setselectedstate2.md)                 | Legt den ausgewählten Zustand des Elements mit dem angegebenen Index im **Wiedergabe** Listenelement fest. Funktioniert mit in der Liste zusammengefügten Wiedergabelisten    |
| [SortColumn](playlist-sortcolumn.md)                               | Sortiert die Daten in der angegebenen Spalte.                                                                                   |



 

Das **Wiedergabe** Listenelement unterstützt die Ambient-Attribute und kann die Umgebungs Ereignishandler implementieren, sofern nicht angegeben. Weitere Informationen finden Sie unter [Ambient-Attribute](ambient-attributes.md) und [Ambient-Ereignishandler](ambient-event-handlers.md).

Vordefinierte Wiedergabelisten sind normale **Wiedergabe** Listenelemente mit verschiedenen allgemeinen Attribut Einstellungen, die standardmäßig angegeben werden. Die folgenden vordefinierten Wiedergabelisten sind verfügbar.



| Vordefinierte Wiedergabeliste                      | BESCHREIBUNG                                                       |
|------------------------------------------|-------------------------------------------------------------------|
| [Dropdownwiedergabe](dropdownplaylist.md) | Eine Dropdown **Liste** ohne sichtbare Elemente.                   |
| [Itemsplaylist](itemsplaylist.md)       | Eine Dropdown- **Liste** , für die keine Elemente oder Spaltenüberschriften sichtbar sind. |
