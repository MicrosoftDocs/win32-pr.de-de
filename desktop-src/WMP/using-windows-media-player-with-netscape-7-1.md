# <a name="using-windows-media-player-with-netscape-71"></a>Verwenden von Windows Media Player mit Netscape 7.1

Der Netscape 7,1-Webbrowser bietet native Unterstützung für Windows Media Player ActiveX-Steuerelemente mithilfe der Gecko-ActiveX-Komponente. Dies bedeutet, dass Sie das Windows Media Player-Steuerelement in Ihre Webseiten für Netscape 7,1 einbetten können, indem Sie die gleiche Syntax verwenden, die Sie für Microsoft Internet Explorer verwenden. Netscape 7,1 unterstützt auch die Skripterstellung für das Steuerelement und die Ereignis Behandlung auf ähnliche Weise.

Wenn Sie das Windows Media Player-Steuerelement in Netscape verwenden möchten, empfiehlt es sich, Netscape 7,1 zu verwenden.

Weitere Informationen zur Verwendung des Windows Media Player-Steuer Elements mit Netscape 7,1 finden Sie auf der [Netscape-Website](https://developer.mozilla.org/docs/Archive/Windows_Media_in_Netscape).

Weitere Informationen zur Gecko-ActiveX-Komponente finden Sie auf der [StreamingMedia.com-Website](https://www.streamingmedia.com/Articles/Editorial/Featured-Articles/The-Windows-Media-ActiveX-Control-8211%3b-Not-Just-for-Internet-Explorer-Anymore-64608.aspx).

Das vollständige herunterladen dieses SDK umfasst ein Beispiel, das die Webbrowser Erkennung veranschaulicht. Weitere Informationen zu den im SDK enthaltenen Beispielen finden Sie unter [Beispiele](samples.md).
