# <a name="using-windows-media-player-with-netscape-71"></a>Verwenden von Windows Media Player mit Netscape 7.1

Der Netscape 7.1-Webbrowser bietet native Unterstützung für Windows Media Player ActiveX-Steuerelemente mithilfe der Gecko ActiveX Komponente. Dies bedeutet, dass Sie das Windows Media Player-Steuerelement in Ihre Webseiten für Netscape 7.1 einbetten können, indem Sie dieselbe Syntax verwenden, die Sie für Microsoft Internet Explorer. Netscape 7.1 unterstützt auch die Skripterstellung für das Steuerelement und die Ereignisbehandlung auf ähnliche Weise.

Wenn Sie das steuerelement Windows Media Player in Netscape verwenden möchten, wird die Verwendung von Netscape 7.1 empfohlen.

Weitere Informationen zur Verwendung des Windows Media Player mit Netscape 7.1 finden Sie auf der [Netscape-Website](https://developer.mozilla.org/docs/Archive/Windows_Media_in_Netscape).

Weitere Informationen zur Gecko-ActiveX finden Sie auf der StreamingMedia.com [Website](https://www.streamingmedia.com/Articles/Editorial/Featured-Articles/The-Windows-Media-ActiveX-Control-8211%3b-Not-Just-for-Internet-Explorer-Anymore-64608.aspx).

Der vollständige Download dieses SDK enthält ein Beispiel, das die Erkennung von Webbrowsern veranschaulicht. Weitere Informationen zu beispielen, die im SDK enthalten sind, finden Sie unter [Beispiele](samples.md)für .
