# <a name="playlistarraycount"></a>PlaylistArray.count

Die **count-Eigenschaft** ruft die Anzahl der Wiedergabelisten im Wiedergabelistenarray ab.

## <a name="syntax"></a>Syntax

player.playlistCollection.getAll().count

## <a name="possible-values"></a>Mögliche Werte

Diese Eigenschaft ist eine schreibgeschützte **Zahl** (**long**).

## <a name="remarks"></a>Hinweise

Um den Wert dieser Eigenschaft abzurufen, ist Lesezugriff auf die Bibliothek erforderlich. Weitere Informationen finden Sie unter [Bibliothekszugriff.](library-access.md)
