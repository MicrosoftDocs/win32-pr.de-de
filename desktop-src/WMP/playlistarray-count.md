# <a name="playlistarraycount"></a>Playlistarray. Count

Die **count** -Eigenschaft ruft die Anzahl der Wiedergabelisten im Wiedergabelisten Array ab.

## <a name="syntax"></a>Syntax

Player. playlistcollection. GetAll (). Count

## <a name="possible-values"></a>Mögliche Werte

Diese Eigenschaft ist eine schreibgeschützte **Zahl** (**Long**).

## <a name="remarks"></a>Bemerkungen

Zum Abrufen des Werts dieser Eigenschaft ist Lesezugriff auf die Bibliothek erforderlich. Weitere Informationen finden Sie unter [Bibliotheks Zugriff](library-access.md).
