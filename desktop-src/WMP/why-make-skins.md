# <a name="why-make-skins"></a>Warum Make Skins?

Wenn ein Benutzer ein Programm in einer grafischen Computingumgebung wie Microsoft Windows verwendet, bilden die visuellen Komponenten, mit denen der Benutzer interagiert, die Benutzeroberfläche. Einer der Zwecke von Microsoft Windows besteht darin, eine Standard-Benutzeroberfläche bereitzustellen, sodass alle Programme auf ähnliche, vertraute Weise ausgeführt werden. Beispielsweise empfiehlt Microsoft in der Microsoft Windows-Benutzeroberfläche, dass jedes Programm eine Schaltfläche **Schließen** in der oberen rechten Ecke des Hauptfensters des Programms bereitstellt.

Windows Media Player bietet die Funktionen zum Erstellen einer eigenen Benutzeroberfläche. Wenn Sie die Schaltfläche **Schließen** in der Mitte des Bildschirms setzen möchten, können Sie dies tun. Vielleicht gefällt Ihnen die Art und Weise, wie die Schaltfläche **Schließen** aussieht (sie sieht wie ein X in einem Feld aus). Wenn Sie möchten, dass es wie ein Lötchen und Kreuzbonse aussieht, können Sie eine Benutzeroberfläche erstellen, auf der die Schaltfläche **Schließen** genau das ist! Windows Media Player bietet Ihnen alle Tools, die Sie benötigen, um eine benutzerdefinierte Benutzeroberfläche zum Wiedergeben von Musik und Video zu erstellen: Schaltflächen, Schieberegler, Videofenster, Visualisierungsfenster, Gleichheitsleisten usw.

Es gibt mehrere gute Gründe, warum Sie möglicherweise eine eigene Benutzeroberfläche für Windows Media Player erstellen möchten. Ein Grund dafür ist, dass Sie möglicherweise Funktionen hinzufügen möchten, die nicht bereits in Windows Media Player sind. Sie können z. B. einen Player erstellen, der Musik aus Wiedergabelisten abspielt, die auf der Tageszeit basieren, sodass Sie am Morgen upbeat rock und am Abend langsamen Jazz haben. Vielleicht möchten Sie auch eine Skin mit einer großen roten Schaltfläche erstellen, die die Musik schnell beendet. Windows Media Player enthält nicht die Schaltfläche "Play the same song over and again until my roommate goes ok" (Immer wieder den gleichen Titel wiedergeben, bis mein Zimmergefährte ausfällt), aber wenn Sie einen titel möchten, können Sie ihn erstellen.

Ein weiterer Grund für die Erstellung einer Skin ist, ein Markenbild für Windows Media Player zu erstellen. Wenn Sie Musik von Ihrer Website verteilen und ein bestimmtes Logo verwenden, sollten Sie eine Skin entwerfen, die Ihr Logo verwendet, um Personen an Ihre Website zu erinnern. Wenn Sie über eine Rockband verfügen, können Sie eine Skin mit Bildern Ihrer Band darauf erstellen.

Und ein weiterer Grund für die Erstellung von Skins besteht darin, etwas Einzigartiges zu machen, das Ihren Desktop optimieren kann. Wenn Ihre Freunde vorbeikommen und Sie fragen, was das coole Programm auf Ihrem Bildschirm ist, können Sie sagen, dass Sie es selbst erstellt haben. Sie können sogar ein Bild von Ihrem Hund machen, ihn auf Ihrem Computer scannen, einige Schaltflächen hinzufügen und eine Skin erstellen, in der das Klicken auf die Nasen Ihres Hundes die Musik beginnt und durch Klicken auf das Ende beendet wird. Sie können verschiedene Skins für verschiedene Arten von Musik erstellen oder für jeden Tag der Woche eine andere Skin haben.
