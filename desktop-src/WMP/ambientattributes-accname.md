# <a name="ambientattributesaccname"></a>AmbientAttributes.accName

Das **accName-Attribut** gibt einen Namen für ein beliebiges Element an oder ruft einen Namen ab.

``` syntax
        elementID.accName
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine  Lese-/Schreibzeichenfolge mit einem Standardwert, der dem **ID-Attribut** entspricht.

## <a name="remarks"></a>Hinweise

Dieses Attribut wird für Barrierefreiheitszwecke verwendet. Dadurch kann der Name eines beliebigen Elements von einem Readerprogramm laut vorgelesen werden.

Dieses Attribut gilt auch für Schaltflächenelemente innerhalb eines Schaltflächengruppen-Steuerelements.
