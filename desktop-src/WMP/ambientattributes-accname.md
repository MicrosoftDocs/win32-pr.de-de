# <a name="ambientattributesaccname"></a>Ambientattribute. accName

Das **accName** -Attribut gibt einen Namen für ein beliebiges Element an oder ruft ihn ab.

``` syntax
        elementID.accName
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/schreibzeichenfolge mit einem Standardwert, der dem **ID** -Attribut entspricht. 

## <a name="remarks"></a>Bemerkungen

Dieses Attribut wird für Barrierefreiheits Zwecke verwendet. Sie ermöglicht es, dass der Name eines beliebigen Elements von einem Reader-Programm gelesen wird.

Dieses Attribut gilt auch für Schaltflächen Elemente innerhalb eines Button-Steuer Elements.
