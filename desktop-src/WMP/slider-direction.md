# <a name="sliderdirection"></a>Schieberegler. Richtung

Das **Direction** -Attribut gibt die Richtung an, in der Schieberegler-Bilder angelegt werden, oder ruft Sie ab.

``` syntax
        elementID.direction
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese- **/schreibzeichenfolge**.



| Wert      | BESCHREIBUNG                                                                             |
|------------|-----------------------------------------------------------------------------------------|
| Horizontal | Standard. Die minimale Position befindet sich Links und die maximale Position rechts. |
| Rechtes   | Die minimale Position befindet sich am unteren Rand, und die maximale Position befindet sich am oberen Rand.          |



 

## <a name="remarks"></a>Bemerkungen

Die Füllung oder Bewegung von **ForegroundColor** oder **foregroundimage** erfolgt entsprechend dem angegebenen Wert entlang der vertikalen oder horizontalen Achse.

| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player, Version 7,0 oder höher<br/> |
