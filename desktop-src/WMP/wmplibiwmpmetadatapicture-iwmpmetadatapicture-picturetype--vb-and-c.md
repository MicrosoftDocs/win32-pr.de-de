# <a name="iwmpmetadatapicturepicturetype-property"></a>IWMPMetadataPicture::p icturetype-Eigenschaft

Die- `pictureType` Eigenschaft ruft den Bildtyp des Bilds ab, das durch das Metadata-Attribut dargestellt wird.

## <a name="syntax"></a>Syntax


```CSharp
public System.String pictureType {get; set;}
```


```VB

Public ReadOnly Property pictureType As System.String
```





## <a name="property-value"></a>Eigenschaftswert

Eine **System. String** , die den Bildtyp des Bilds ist.

## <a name="remarks"></a>Bemerkungen

Vor der Verwendung dieser Eigenschaft müssen Sie über Lesezugriff auf die Bibliothek verfügen. Weitere Informationen finden Sie unter [Bibliotheks Zugriff](library-access.md).
