# <a name="working-with-the-library"></a>Arbeiten mit der Bibliothek

Die Bibliothek ist eine Datenbank mit Informationen zu den Medieninhalten, die auf dem Computer des Benutzers gespeichert oder dort wiedergegeben wurden. Einige der Informationen werden in der Bibliotheksfunktion in Windows Media Player angezeigt. Auf einen größeren Satz von Informationen kann programmgesteuert zugegriffen werden.

Medienelemente können in Wiedergabelisten organisiert werden. Wiedergabelisten können von Benutzern oder über Code erstellt werden. Das Windows Media Player-Objektmodell stellt Objekte und Methoden bereit, mit deren Hilfe Sie sowohl einzelne Medienelemente als auch Wiedergabelisten bearbeiten können.

In den folgenden Abschnitten werden Techniken für die Arbeit mit der Bibliothek erläutert:

-   [Bibliothekszugriff](library-access.md)
-   [Verwalten von Medienelementen](managing-media-items.md)
-   [Medienelementattribute](media-item-attributes.md)
-   [Verwalten von Wiedergabelisten](managing-playlists.md)
