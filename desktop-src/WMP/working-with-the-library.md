# <a name="working-with-the-library"></a>Arbeiten mit der Bibliothek

Bei der Bibliothek handelt es sich um eine Datenbank mit Informationen zu den Medieninhalten, die auf dem Computer des Benutzers gespeichert sind oder dort abgespielt wurden. Einige der Informationen werden in der Bibliotheksfunktion in Windows Media Player angezeigt. Auf einen größeren Satz von Informationen kann Programm gesteuert zugegriffen werden.

Medienelemente können in Wiedergabelisten organisiert werden. Wiedergabelisten können von Benutzern oder durch Code erstellt werden. Das Windows Media Player-Objektmodell stellt Objekte und Methoden bereit, mit denen Sie sowohl einzelne Medienelemente als auch Wiedergabelisten bearbeiten können.

In den folgenden Abschnitten werden Techniken zum Arbeiten mit der-Bibliothek erläutert:

-   [Bibliotheks Zugriff](library-access.md)
-   [Verwalten von Medien Elementen](managing-media-items.md)
-   [Medien Element Attribute](media-item-attributes.md)
-   [Verwalten von Wiedergabelisten](managing-playlists.md)
