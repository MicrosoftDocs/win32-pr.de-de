# <a name="creating-the-volumethumb-file"></a>Erstellen der volumethumschlag Datei

Die volumethserib-Datei enthält das Thumb-Bild, das mit einer volumetrackleiste verwendet wird.

Die folgende Abbildung ist die volumethumschlag Datei.

![volumethumschlag Datei](images/ceswmvol.png)

Diese enthält die zwei Thumb-Bilder. Der linke auf der linken Seite ist der normale Zustand, und der Rechte auf der rechten Seite ist der pushzustand. Es ist kein deaktivierter Zustand vorhanden. Wenn eine TrackBar deaktiviert ist, wird das deaktivierte Hintergrundbild der TrackBar in der Super-Datei angezeigt.
