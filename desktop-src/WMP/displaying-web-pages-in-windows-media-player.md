# <a name="displaying-web-pages-in-windows-media-player"></a>Anzeigen von Webseiten in Windows Media Player

Das Windows Media Player HtmlView-Feature ermöglicht Internet Inhaltsanbietern (Internet Content Providers, ICPs) die Verwendung der **nun** wiedergegebenen Funktion des vollmodusplayers zum Anzeigen von webbasiertem Inhalt, während Inhalte digitaler Medien wiedergegeben werden. (HtmlView wird vom Windows Media Player-Steuerelement nicht unterstützt.) In der Vergangenheit mussten ICPs Ankündigungen in separaten Webbrowser Fenstern anzeigen (als "Popup Werbung" bezeichnet) oder benutzerdefinierte Digital Media-Player zu ihren Webseiten hinzufügen. Mit HtmlView können ICPs ein personalisiertes und integriertes, speziell für Windows Media Player 9-Serie und Windows Media Player 10 erstellte Verfahren bereitstellen und gleichzeitig sicherstellen, dass die Inhalte digitaler Medien weiterhin in früheren Versionen von Windows Media Player wiedergegeben werden.

Dieser Abschnitt enthält Informationen zur Verwendung von HtmlView, zum Erstellen von HtmlView-Inhalten und zum Beheben möglicher Probleme, die bei Verwendung der HtmlView-Funktion in den folgenden Themen auftreten können:

-   [Erstellen einer HtmlView-Präsentation](creating-an-htmlview-presentation.md)
-   [Hinzufügen eines eingebetteten Windows Media Player-Steuer Elements](adding-an-embedded-windows-media-player-control.md)
