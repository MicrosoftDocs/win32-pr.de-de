# <a name="displaying-web-pages-in-windows-media-player"></a>Anzeigen von Webseiten in Windows Media Player

Die Windows Media Player HTMLView-Funktion ermöglicht Internetinhaltsanbietern (ICPs), die **Funktion Jetzt wiedergeben** des Vollmodusplayers zu verwenden, um webbasierte Inhalte anzuzeigen, während inhalte digitaler Medien wiedergegeben werden. (HTMLView wird vom Windows Media Player-Steuerelement nicht unterstützt.) In der Vergangenheit mussten ICPs Ankündigungen in separaten Webbrowserfenstern (sogenannte "Popup-Werbung") anzeigen oder ihren Webseiten benutzerdefinierte player für digitale Medien hinzufügen. Mit HTMLView können ICPs eine personalisierte und integrierte Benutzeroberfläche bieten, die speziell für Windows Media Player 9-Serie und Windows Media Player 10 erstellt wurde, und gleichzeitig sicherstellen, dass digitale Medieninhalte in früheren Versionen von Windows Media Player weiterhin wiedergegeben werden.

Dieser Abschnitt enthält Informationen zur Verwendung von HTMLView, zum Erstellen von HTMLView-Inhalten und zum Beheben potenzieller Probleme, die bei verwendung des HTMLView-Features in den folgenden Themen auftreten können:
