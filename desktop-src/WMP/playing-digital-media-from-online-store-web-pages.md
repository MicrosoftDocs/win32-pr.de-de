# <a name="playing-digital-media-from-online-store-web-pages"></a>Wieder spielt digitale Medien von Online Store Web pages ab

Sie können digitale Medieninhalte von Webseiten Ihres Onlineshops wieder geben. Betten Sie einfach das Windows Media Player ein, wie unter [Verwenden des Windows Media Player-Steuerelements in einer Webseite beschrieben.](using-the-windows-media-player-control-in-a-web-page.md) Wenn Sie dies in einem Dienstaufgabebereich ausführen, stellt das eingebettete Steuerelement eine Verbindung mit dem Windows Media Player-Programm auf ähnliche Weise wie im Remotemodus (siehe [Remoting des Windows Media Player-Steuerelements) auf.](remoting-the-windows-media-player-control.md) Dies bedeutet, dass Benutzer die Transportsteuerelemente des Player im Vollmodus verwenden können, um die Wiedergabe digitaler Medien auf der Webseite zu steuern.

Es wird empfohlen, **uiMode** auf "none" oder "invisible" zu setzen, wenn Sie das Player-Steuerelement in eine Webseite mit dem Aufgabenbereich "Dienst" einbetten.
