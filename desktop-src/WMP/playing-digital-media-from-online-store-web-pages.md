# <a name="playing-digital-media-from-online-store-web-pages"></a>Wiedergabe digitaler Medien aus Online Store-Webseiten

Sie können digitale Medieninhalte auf Ihren Online Store-Webseiten abspielen. Betten Sie einfach das Windows Media Player-Steuerelement ein, wie unter [Verwenden des Windows Media Player-Steuer Elements in einer Webseite](using-the-windows-media-player-control-in-a-web-page.md)beschrieben. Wenn Sie dies in einem Dienstaufgaben Bereich durchführen, stellt das eingebettete Steuerelement eine Verbindung mit dem Windows-Media Player-Programm her, ähnlich wie im Remote Modus (siehe [Remoting the Windows Media Player Control](remoting-the-windows-media-player-control.md)). Dies bedeutet, dass Benutzer die Transport Steuerelemente des vollmodusplayers zum Steuern der Wiedergabe digitaler Medien auf der Webseite verwenden können.

Es wird empfohlen, dass Sie **uiMode** auf "None" oder "unsichtbar" festlegen, wenn Sie das Player-Steuerelement in eine Webseite des Dienstaufgaben Bereichs einbetten.
