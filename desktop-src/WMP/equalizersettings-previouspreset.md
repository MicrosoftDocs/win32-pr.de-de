# <a name="equalizersettingspreviouspreset"></a>Equalizersettings. previouationsset

Die Methode **previouredset** wendet die vorherige Voreinstellung für den equaliausgleich an.

``` syntax
        elementID.previousPreset()
```

## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.

## <a name="remarks"></a>Bemerkungen

Wenn die aktuelle Voreinstellung das erste verfügbare ist, wird die letzte Voreinstellung als aktuell festgestellt.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player, Version 7,0 oder höher<br/> |
