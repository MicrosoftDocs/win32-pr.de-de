# <a name="equalizersettingspreviouspreset"></a>EQUALIZERSETTINGS.previousPreset

Die **previousPreset-Methode** wendet die vorherige Voreinstellung für den Equalizer an.

``` syntax
        elementID.previousPreset()
```

## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.

## <a name="remarks"></a>Hinweise

Wenn es sich bei der aktuellen Voreinstellung um die erste verfügbare Voreinstellung handelt, wird die letzte Voreinstellung als aktuell betrachtet.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player Version 7.0 oder höher<br/> |
