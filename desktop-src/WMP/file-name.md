# <a name="file-name"></a>Dateiname

Dies ist der Name der Datei, in der das Image enthalten ist. Der Pfad sollte nicht angegeben werden, und die Datei sollte sich im Skin-Verzeichnis befinden. Die Dateinamenerweiterung sollte BMP oder GIF lauten.
