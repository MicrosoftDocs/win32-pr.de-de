# <a name="effectscurrentpreset"></a>Effects. currentvoreinstellung

Das **currentvoreinstellung** -Attribut gibt die aktuelle Voreinstellung der aktuellen Visualisierung an oder ruft diese ab.

``` syntax
        elementID.currentPreset
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/schreibzahl (**Long**), die den Index der Voreinstellung angibt.  Die Indizes beginnen mit 0 (null), was auch der Standardwert ist.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player, Version 7,0 oder höher<br/> |
