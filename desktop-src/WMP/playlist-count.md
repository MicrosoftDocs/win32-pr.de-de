# <a name="playlistcount"></a>Playlist.count

Die **count-Eigenschaft** ruft die Anzahl der Medienelemente in der Wiedergabeliste ab.

## <a name="syntax"></a>Syntax

*Player*. *currentPlaylist*. **count**

## <a name="possible-values"></a>Mögliche Werte

Diese Eigenschaft ist eine schreibgeschützte **Zahl** (**long**).

## <a name="example"></a>Beispiel

Beispielcode, der diese Eigenschaft verwendet, finden Sie in der [attributeCount-Eigenschaft.](playlist-attributecount.md)

## <a name="remarks"></a>Hinweise

Um den Wert dieser Eigenschaft abzurufen, ist Lesezugriff auf die Bibliothek erforderlich. Weitere Informationen finden Sie unter [Bibliothekszugriff.](library-access.md)
