# <a name="download-manager"></a>Download-Manager

Der Microsoft Windows Media Player Download-Manager ermöglicht den Typ 2-Onlinespeicher, um das Herunterladen von Inhalten aus einem Windows Media Player Online Store-Aufgabenbereich zu steuern. Ein Aufgabenbereich im Online Shop hostet eine Webseite, die angezeigt wird, wenn der Benutzer einen Online Shop verwendet.

Diese Einführung in den Download-Manager enthält die folgenden Abschnitte.



| `Section`                                                            | BESCHREIBUNG                                                     |
|--------------------------------------------------------------------|-----------------------------------------------------------------|
| [Download-Manager: Übersicht](download-manager-overview.md)         | Führt den Download-Manager und einige grundlegende Verwendungs Szenarien ein. |
| [Architektur des Download-Managers](download-manager-architecture.md) | Enthält eine Übersicht über das Objektmodell des Download-Managers.        |
