<span data-ttu-id="5f392-111">Mapping-Images werden für bestimmte Steuerelemente verwendet, um anzugeben, welche Bereiche auf Mausklicks reagieren, und um zu bestimmen, welche Steuerelemente welche Ereignisse empfangen.</span><span class="sxs-lookup"><span data-stu-id="5f392-111">Mapping images are used for specific controls to specify which regions will respond to mouse clicks and to determine which controls receive which events.</span></span> <span data-ttu-id="5f392-112">Das Zuordnungsbild muss dieselbe Höhe und Breite wie das primäre Image aufweisen und sollte so erstellt werden, dass die Bilder in beiden Dateien genau übereinstimmen.</span><span class="sxs-lookup"><span data-stu-id="5f392-112">The mapping image must be the same height and width as the primary image and should be created so that the images in both files line up exactly.</span></span> <span data-ttu-id="5f392-113">Aus diesem Grund wird empfohlen, ein Grafikbearbeitungsprogramm, das Ebenen unterstützt, zu empfehlen, da Sie sicher sein können, dass Ihre Bilder perfekt ausgerichtet sind.</span><span class="sxs-lookup"><span data-stu-id="5f392-113">For this reason, a graphics editing program that supports layers is recommended because you can be certain your images are perfectly aligned.</span></span>
