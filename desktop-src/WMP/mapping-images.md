# <a name="mapping-images"></a>Mapping von Bildern

Mapping-Images werden für bestimmte Steuerelemente verwendet, um anzugeben, welche Bereiche auf Mausklicks reagieren, und um zu bestimmen, welche Steuerelemente welche Ereignisse empfangen. Das Zuordnungsbild muss dieselbe Höhe und Breite wie das primäre Image aufweisen und sollte so erstellt werden, dass die Bilder in beiden Dateien genau übereinstimmen. Aus diesem Grund wird empfohlen, ein Grafikbearbeitungsprogramm, das Ebenen unterstützt, zu empfehlen, da Sie sicher sein können, dass Ihre Bilder perfekt ausgerichtet sind.

Für verschiedene Steuerelemente sind unterschiedliche Arten der Zuordnung erforderlich. Ein **ButtonElement** -Element benötigt eine Karte mit einer anderen Farbe für jede Schaltfläche. Die farbigen Bereiche in der Zuordnungs Datei müssen den Bereichen der Schaltflächen entsprechen, die Sie zuordnen möchten.
