# <a name="jscript-files"></a>JScript-Dateien

JScript-Dateien werden mit dem **scriptfile** -Attribut des **View** -Elements geladen. Dabei muss es sich um Textdateien handeln, und die Dateinamenerweiterung. js sollte verwendet werden. Wenn Sie über eine JScript-Datei mit demselben Namen wie die Skin-Definitionsdatei verfügen, wird die JScript-Datei zur gleichen Zeit geladen wie die Skin-Definitionsdatei. Wenn Sie z. b. eine Skin-Definitionsdatei mit dem Namen "Laure. WMS" haben und eine JScript-Datei namens "laure.js" haben, wird die laure.js Datei automatisch geladen.

Sie können JScript verwenden, um eine aufwändige Funktionalität hinter ihrer Skin zu erstellen. Durch das Erstellen von Funktionen in JScript können Sie nahezu alles, was Sie mit Skins machen möchten, erledigen. Sie können z. b. für jeden Tag der Woche eine andere Wiedergabeliste verwenden, aber immer dieselbe, die am Freitag vorhanden ist.

Weitere Informationen zur Verwendung von JScript mit Skins finden Sie unter [Verwenden von JScript](using-jscript.md) . Beachten Sie, dass Visual Basic Scripting Edition (VBScript), die beim Einbetten des Windows Media Player-Steuer Elements in einer Webseite verwendet werden kann, nicht für die Verwendung mit Skins unterstützt wird.
