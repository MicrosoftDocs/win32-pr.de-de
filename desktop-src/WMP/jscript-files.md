# <a name="jscript-files"></a>JScript Dateien

JScript Dateien werden mit dem **scriptFile-Attribut** des **VIEW-Elements** geladen. Dabei muss es sich um Textdateien mit der Dateinamenerweiterung .js. Wenn Sie über eine JScript verfügen, die den gleichen Namen wie die Skindefinitionsdatei hat, wird die JScript-Datei gleichzeitig mit der Skindefinitionsdatei geladen. Wenn Sie beispielsweise eine Skindefinitionsdatei mit dem Namen laure.wms und eine JScript-Datei namens laure.js haben, wird die laure.js-Datei automatisch geladen.

Sie können JScript verwenden, um ausführliche Funktionen hinter Ihrer Skin zu erstellen. Indem Sie Funktionen in JScript erstellen, können Sie fast alles erreichen, was mit Skins vorstellbar ist. Beispielsweise können Sie eine andere Wiedergabeliste für jeden Tag der Woche verwenden, aber immer die gleiche wiedergabeliste am Freitag verwenden.

Weitere [Informationen zur JScript](using-jscript.md) mit Skins finden Sie unter Using JScript (Verwenden von JScript mit Skins). Beachten Sie, dass Visual Basic Scripting Edition (VBScript), die beim Einbetten des Windows Media Player-Steuerelements in eine Webseite verwendet werden kann, nicht für die Verwendung mit Skins unterstützt wird.
