# <a name="ondblclick"></a>ondblclick

Der **ondblclick** -Ereignishandler behandelt ein Ereignis, das auftritt, wenn der Benutzer auf das Element doppelklickt.

``` syntax
ondblclick
```

## <a name="remarks"></a>Bemerkungen

Dieser Ereignishandler gilt nicht für das **Wiedergabe** Listenelement, das **Popup** Element, das **Video** Element bei *Video*. **Windows less** ist false, oder das **Effects** -Element bei *Effekten*. Das **Fenster ist "** true".

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|-----------------------------------------------------|
| Version<br/> | Windows Media Player, Version 70 oder höher<br/> |
