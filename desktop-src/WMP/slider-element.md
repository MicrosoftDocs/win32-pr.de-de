# <a name="slider-element"></a>Slider-Element

Das **Slider** -Element bietet eine Möglichkeit zum Erstellen und Bearbeiten eines einfachen horizontalen oder vertikalen Schieberegler-Steuer Elements. Es unterstützt die folgenden Attribute und Ereignishandler. Vordefinierte **Schieberegler** -Elemente werden zur einfacheren Bereitstellung bereitgestellt.

Das **Slider** -Element unterstützt die folgenden Attribute.



| Attribut                                                 | BESCHREIBUNG                                                                                                       |
|-----------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [Background Color](slider-backgroundcolor.md)             | Gibt die Hintergrundfarbe des Schieberegler-Steuer Elements an oder ruft diese ab.                                                |
| [backgroundendcolor](slider-backgroundendcolor.md)       | Gibt die Hintergrund Endfarbe des Schieberegler-Steuer Elements an oder ruft diese ab.                                         |
| [backgroundhoverimage](slider-backgroundhoverimage.md)   | Gibt das Hintergrundbild des Schiebereglers an oder ruft ihn ab, der angezeigt wird, wenn mit der Maus darauf gezeigt wird.      |
| [backgroundImage](slider-backgroundimage.md)             | Gibt das Standard Hintergrundbild des Schiebereglers an oder ruft es ab.                                                |
| [BorderSize](slider-bordersize.md)                       | Gibt die Rahmengröße in Pixel an oder ruft Sie ab.                                                                 |
| [Cursor](slider-cursor.md)                               | Gibt einen Wert an, der angibt, welcher Cursortyp angezeigt wird, wenn sich der Mauszeiger über dem Schieberegler-Steuerelement befindet |
| [direction](slider-direction.md)                         | Gibt die Richtung an, in der Schieberegler-Bilder angelegt werden, oder ruft Sie ab.                                             |
| [disabledColor](slider-disabledcolor.md)                 | Gibt die deaktivierte Farbe des Schieberegler-Steuer Elements an oder ruft diese ab.                                                  |
| [disabledImage](slider-disabledimage.md)                 | Gibt das Bild des Schiebereglers an oder ruft ihn ab, der angezeigt wird, wenn das Steuerelement deaktiviert ist.                         |
| [ForegroundColor](slider-foregroundcolor.md)             | Gibt die Vordergrundfarbe des Schieberegler-Steuer Elements an oder ruft diese ab.                                                |
| [foregroundendcolor](slider-foregroundendcolor.md)       | Gibt die Vordergrundfarbe des Schieberegler-Steuer Elements an oder ruft diese ab.                                         |
| [foregroundhoverimage](slider-foregroundhoverimage.md)   | Gibt das Vordergrundbild des Schiebereglers an oder ruft ihn ab, der angezeigt wird, wenn mit der Maus darauf gezeigt wird.      |
| [foregroundimage](slider-foregroundimage.md)             | Gibt das Standard-Vordergrundbild des Schiebereglers an oder ruft es ab.                                                |
| [foregroundprogress](slider-foregroundprogress.md)       | Gibt die aktuelle Position der Statusanzeige im Vordergrund als Prozentsatz des Schieberegler-Bereichs an oder ruft diese ab.    |
| [max](slider-max.md)                                     | Gibt den maximalen Wert des vom Schieberegler-Steuerelement definierten Bereichs an oder ruft ihn ab.                              |
| [min](slider-min.md)                                     | Gibt den minimalen Wert des Bereichs an, der vom Schieberegler-Steuerelement definiert wird, oder ruft diesen ab.                              |
| [Dia](slider-slide.md)                                 | Gibt einen Wert an, der angibt, ob das Vordergrundbild über das Hintergrundbild bewegt wird, oder ruft diesen ab          |
| [thumbdisabledimage](slider-thumbdisabledimage.md)       | Gibt das deaktivierte Ziehbild des Schieberegler-Steuer Elements an oder ruft es ab.                                            |
| [thumbdownimage](slider-thumbdownimage.md)               | Gibt das Bild an, das den Zustand des Zieh Punkts darstellt, oder ruft dieses ab.                                        |
| [thumbhoverimage](slider-thumbhoverimage.md)             | Gibt das Bild des Zieh Punkts an, das angezeigt wird, wenn mit der Maus darauf gezeigt wird, oder ruft es ab.                  |
| [thumbimage](slider-thumbimage.md)                       | Gibt das Bild an, das zum Darstellen der aktuellen Position des Schiebereglers verwendet wird, oder ruft es ab.               |
| [Ken](slider-tiled.md)                                 | Gibt einen Wert an, der angibt, ob die Schieberegler angezeigt werden, oder ruft ihn ab.                                |
| [QuickInfo](slider-tooltip.md)                             | Gibt den QuickInfo-Text für das Slider-Steuerelement an oder ruft ihn ab.                                                   |
| [transparendcolor](slider-transparencycolor.md)         | Gibt die transparente Farbe der Schieberegler-Bilder an oder ruft diese ab.                                                |
| [useforegroundprogress](slider-useforegroundprogress.md) | Gibt einen Wert an, der angibt, ob die Statusleiste des Vordergrunds verwendet wird, oder ruft diesen ab.                       |
| [value](slider-value.md)                                 | Gibt die aktuelle Position des Schiebereglers an und ruft diese ab.                                                       |



 

Das **Slider** -Element kann die folgenden Ereignishandler implementieren.



| Ereignishandler                                   | BESCHREIBUNG                                                                                                          |
|-------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [ondragbegin](slider-ondragbegin.md)           | Behandelt ein Ereignis, das auftritt, wenn der Benutzer auf die linke Maustaste klickt und mit der Maus bewegt wird. |
| [ondragend](slider-ondragend.md)               | Behandelt ein Ereignis, das auftritt, wenn die linke Maustaste nach einem Zieh Vorgang losgelassen wird.                      |
| [onpositionchange](slider-onpositionchange.md) | Behandelt ein Ereignis, das auftritt, wenn sich die Position des Schiebereglers ändert, wenn der Benutzer auf klicken oder ziehen klickt.   |



 

Das **Slider** -Element unterstützt die Ambient-Attribute und kann die Umgebungs Ereignishandler implementieren. Weitere Informationen finden Sie unter [Ambient-Attribute](ambient-attributes.md) und [Ambient-Ereignishandler](ambient-event-handlers.md).

Vordefinierte Schieberegler sind normale **Schieberegler** -Elemente mit verschiedenen allgemeinen Attribut Einstellungen, die standardmäßig angegeben werden. Die folgenden vordefinierten Schieberegler sind verfügbar.



| Vordefinierter Schieberegler                  | BESCHREIBUNG                                                    |
|------------------------------------|----------------------------------------------------------------|
| [Balanceslider](balanceslider.md) | Ein **Schieberegler** zum Festlegen des audioausgleichs.                        |
| [SeekSlider](seekslider.md)       | Ein **Schieberegler** , der verwendet wird, um eine beliebige Position innerhalb einer Mediendatei zu suchen. |
| [VolumeSlider](volumeslider.md)   | Ein **Schieberegler** , mit dem audiovolume festgelegt wird.                         |
