# <a name="metadata-area-plug-ins-deprecated"></a>Metadatenbereichs-Plug-Ins (veraltet)

Auf dieser Seite wird ein Feature dokumentiert, das in zukünftigen Versionen von Windows Media Player und dem Windows Media Player SDK möglicherweise nicht verfügbar ist.

Benutzeroberflächen-Plug-Ins für Metadatenbereiche werden im Metadatenbereich des Bereichs **Jetzt wiedergeben** rechts neben dem hauptvisualen Bereich und im Bereich Extras sowie unterhalb der Wiedergabelistenansicht angezeigt (sofern aktiviert). Es kann jeweils nur ein Metadatenbereichs-Plug-In aktiviert werden.

Metadatenbereichs-Plug-Ins sind nützlich, um präzise Informationen über das aktuell wiedergegebene Medienelement, einfache Steuerelemente oder Links zu zusätzlichen Informationen anzuzeigen.
