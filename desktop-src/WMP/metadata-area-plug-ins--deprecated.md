# <a name="metadata-area-plug-ins-deprecated"></a>Metadatenbereich-Plug-ins (veraltet)

Diese Seite dokumentiert eine Funktion, die in zukünftigen Versionen von Windows Media Player und dem Windows Media Player SDK möglicherweise nicht verfügbar ist.

Benutzeroberflächen-Plug-Ins für Metadatenbereiche werden im Metadatenbereich des Fensters " **jetzt** wiedergegeben" rechts neben der visuellen Hauptfläche und dem Bereich "Extras" und unterhalb der Wiedergabelisten Ansicht angezeigt (sofern aktiviert). Es kann jeweils nur ein metadatenbereichsplug-in aktiviert werden.

Metadatenbereich-Plug-ins sind hilfreich, um präzise Informationen über das aktuell wiedergegebene Medien Element, einfache Steuerelemente oder Hyperlinks zu zusätzlichen Informationen anzuzeigen.
