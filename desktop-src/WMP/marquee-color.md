# <a name="marquee-color"></a>Marquee Color

Der Farbwert des Typs muss angegeben werden. Der Wert muss mit drei positiven ganzen Zahlen angegeben werden, die durch Kommas getrennt sind. Die Werte stellen die Menge an Rot, Grün und Blau für eine Bitmapfarbe dar, und die Bereiche liegen zwischen 0 und 255.
