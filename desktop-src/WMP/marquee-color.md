# <a name="marquee-color"></a>Marquee-Farbe

Der Farbwert des Typs muss angegeben werden. Der Wert muss mithilfe von drei positiven ganzen Zahlen angegeben werden, die durch Kommas getrennt sind. Die Werte stellen die Menge von rot, grün und blau für eine bitmapfarbe dar und liegen zwischen 0 und 255.
