# <a name="erroritem-object"></a>ErrorItem-Objekt

Das **ErrorItem** -Objekt bietet eine Möglichkeit, auf Fehlerinformationen zuzugreifen.

Das **ErrorItem** -Objekt unterstützt die folgenden Eigenschaften.



| Eigenschaft                                           | BESCHREIBUNG                                                                                     |
|----------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [condition](erroritem-condition.md)               | Ruft einen Wert ab, der die Bedingung für den Fehler angibt.                                       |
| [CustomURL](erroritem-customurl.md)               | Ruft die URL einer Website ab, auf der bestimmte Informationen zum Download Fehler beim Codec angezeigt werden. |
| [ErrorCode](erroritem-errorcode.md)               | Ruft den aktuellen Fehlercode ab.                                                               |
| [errorcontext](erroritem-errorcontext.md)         | Ruft einen Wert ab, der den Kontext des Fehlers angibt.                                          |
| [errorDescription](erroritem-errordescription.md) | Ruft eine Beschreibung des Fehlers ab.                                                           |
| **Lösung**                                         | Für die zukünftige Verwendung reserviert.                                                                        |



 

Der Zugriff auf das **ErrorItem** -Objekt erfolgt über die folgenden Methoden.



| Object                    | Methode                   |
|---------------------------|--------------------------|
| [Fehler](error-object.md) | [item](error-item.md)   |
| [Medien](media-object.md) | [error](media-error.md) |



 

Dient zur Veranschaulichung des *Players*. *Fehler*. *Item*(*Index*) wird in den Abschnitten der Verweis Syntax verwendet.
