# <a name="sliderforegroundhoverimage"></a>Slider. foregroundhoverimage

Das **foregroundhoverimage** -Attribut gibt das Vordergrundbild des Schiebereglers an oder ruft es ab, das angezeigt wird, wenn mit der Maus darauf gezeigt wird.

``` syntax
        elementID.foregroundHoverImage
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese- **/schreibzeichenfolge** , die den Namen einer Bilddatei enthält.

## <a name="remarks"></a>Bemerkungen

Das **foregroundhoverimage** ist optional. Wenn keine Angabe erfolgt, wird **BackgroundImage** verwendet.

Die unterstützten Formate sind BMP, JPG, PNG und GIF (keine animierten GIFs).
