# <a name="about-the-dvd-object"></a>Informationen zum DVD-Objekt

Das **DVD-Objekt** fügt spezifische Funktionen für DVD-Medien hinzu. Im Allgemeinen werden DVD-Medien wie andere digitale Medien in der Windows Media Player. DVD-Laufwerke werden beispielsweise mithilfe des **CcollectionCollection-Objekts** aufzählt, und  DVD-Titel und Kapitel werden mit Wiedergabelistenobjekten und **Medienobjekten bearbeitet.** Einige Funktionen sind jedoch DVD-spezifisch, und das **DVD-Objekt** stellt dies zur Verfügung. Dvd verfügt beispielsweise über ein Konzept namens Domäne. Geben Sie Folgendes ein, um die aktuelle Domäne für DVD-Medien abzurufen:


```C++
mydomain = player.dvd.domain;

```
