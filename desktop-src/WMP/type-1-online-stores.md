# <a name="type-1-online-stores"></a>Geben Sie 1 Onlineshops ein.

> [!Note]  
> In diesem Abschnitt werden funktionen beschrieben, die für die Verwendung durch Onlineshops entwickelt wurden. Die Verwendung dieser Funktionalität außerhalb des Kontexts eines Onlineshops wird nicht unterstützt.

 

Windows Media Player 11 unterstützt Onlineshops vom Typ 1 und 2. Speicher vom Typ 1 sind stärker in die Benutzeroberfläche des Players integriert als Speicher vom Typ 2. Diese Dokumentation, die nur für Speicher vom Typ 1 gilt, ist in die folgenden Abschnitte unterteilt.



| `Section`                                                                                      | BESCHREIBUNG                                                                                                                                      |
|----------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [Informationen zu Onlineshops vom Typ 1](about-type-1-online-stores.md)                                 | Erläutert die wichtigsten Features von Onlineshops vom Typ 1, einschließlich herunterladbarer Kataloge, Bibliotheksintegration und Ermittlungsseiten.               |
| [Programmierhandbuch für Onlineshops vom Typ 1](programming-guide-for-type-1-online-stores.md) | Enthält Anleitungen zum Erstellen eines Onlineshop-Plug-Ins vom Typ 1.                                                                                     |
| [Referenz für Typ 1-Onlineshops](reference-for-type-1-online-stores.md)                 | Stellt detaillierte Verweise auf die Schnittstellen, Objekte, Enumerationen, Strukturen, Konstanten und Registrierungseinträge bereit, die von Onlineshops vom Typ 1 verwendet werden. |
