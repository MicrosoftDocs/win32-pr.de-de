# <a name="ambientattributesbottom"></a>AmbientAttributes.bottom

Das **bottom-Attribut** gibt die untere Koordinate des Steuerelements an oder ruft sie ab.

``` syntax
        elementID.bottom
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/Schreibnummer (**long**), die den Abstand zwischen dem Steuerelement und dem unteren Rand der übergeordneten **VIEW-** oder  **SUBVIEW-Ansicht** in Pixel darstellt. Der Standardwert ist 0 (null). Das Verhalten für negative Werte oder , wenn [AmbientAttributes.height](ambientattributes-height.md) nicht angegeben ist, ist nicht definiert.
