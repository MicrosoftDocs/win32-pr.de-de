# <a name="ambientattributesbottom"></a>Ambientattribute. Bottom

Das **Bottom** -Attribut gibt die untere Koordinate des Steuer Elements an oder ruft diese ab.

``` syntax
        elementID.bottom
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/schreibzahl (**Long**), die den Abstand zwischen dem Steuerelement und dem unteren Rand der übergeordneten **Ansicht** oder **unter Ansicht** darstellt.  Der Standardwert ist 0 (null). Das Verhalten für negative Werte oder, wenn " [ambientattribute. Height](ambientattributes-height.md) " nicht angegeben wird, ist nicht definiert.
