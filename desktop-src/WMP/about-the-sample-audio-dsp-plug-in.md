# <a name="about-the-sample-audio-dsp-plug-in"></a>Informationen zum Beispiel-audiodsp-Plug-in

Das Beispiel-audiodsp-Plug-in bietet eine einfache Verarbeitungs Implementierung, die die Amplitude der Audiodaten um einen vom Benutzer auf der Eigenschaften Seite bereitgestellten Faktor skaliert. Das Plug-in akzeptiert Werte zwischen 0 (null) und 1. Der Standardwert ist 1. Der Wert 1 hat keine Auswirkung auf den Sound. Weitere Skalierungsfaktoren sind Multiplikatoren für die Audiobeispiele. Der Wert 0,5 würde beispielsweise zu einer Abnahme von 6 Dezimalstellen führen. Der Wert 0 (null) führt zu einer Ruhe.

Das Beispiel-audiodsp-Plug-in funktioniert mit Stereo-oder Mono-PCM-Audiodaten.
