# <a name="totalduration-attribute"></a>Totalduration-Attribut

Das **Totalduration** -Attribut ist die Wiedergabedauer des Elements.

## <a name="applies-to"></a>Gilt für

-   [Audioelemente](audio-item-attributes.md)
-   [CD-Wiedergabelisten](cd-playlist-attributes.md)
-   [CD-Spuren](cd-track-attributes.md)

## <a name="remarks"></a>Bemerkungen

Dieses Attribut ist nur im Bibliotheks Cache gespeichert.

Um zu ermitteln, ob Sie den Wert dieses Attributs ändern können, verwenden Sie die [Media. isread onlyitem](media-isreadonlyitem.md) -Methode.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|-----------------------------------------------------------------------------------------------------------|
| Version<br/> | Windows Media Player 9-Serie Windows Media Player 10 oder höher unterstützt dieses Attribut nicht.<br/> |
