# <a name="bitrate-attribute"></a>Bitrate-Attribut

Das **Bitrate-Attribut** ist die Bitrate des Elements in Bits pro Sekunde.

## <a name="applies-to"></a>Gilt für

-   [Audioelemente](audio-item-attributes.md)
-   [Häufig verwendete Windows Mediendateien](commonly-used-windows-media-file-attributes.md)
-   [Andere Elemente](other-item-attributes.md)
-   [Funkelemente](radio-item-attributes.md)
-   [Videoelemente](video-item-attributes.md)

## <a name="remarks"></a>Hinweise

Dieses Attribut wird sowohl in der Bibliothek als auch in der digitalen Mediendatei gespeichert.

Die Windows Media Format SDK-Konstante für dieses Attribut ist g \_ wszWMBitrate.

Verwenden Sie die [Media.isReadOnlyItem-Methode,](media-isreadonlyitem.md) um zu bestimmen, ob Sie den Wert dieses Attributs ändern können.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9er Serie oder höher<br/> |
