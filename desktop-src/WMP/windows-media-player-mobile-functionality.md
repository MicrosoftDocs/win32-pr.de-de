# <a name="windows-media-player-mobile-functionality"></a>Windows Media Player Mobile-Funktionalität

Die folgenden Aktionen sind für den Benutzer Ihrer Skin verfügbar:

-   Tippen oder klicken auf Schaltflächen
-   Gleitende trackbars
-   Lesen von Scrolltext
-   Video ansehen
-   Anzeigen von albumgrafiken
-   Bewerten von Medien Elementen

In den folgenden Abschnitten werden die Typen von Elementen beschrieben, die Sie erstellen können, um diese Benutzeraktionen zu aktivieren:

-   [Schaltflächen Funktionen](button-functions.md)
-   [TrackBar-Typen](trackbar-types.md)
-   [Text Anzeigetypen](text-display-types.md)
-   [Die Video Anzeige](the-video-display.md)
-   [Ausrichtung](orientation.md)
