# <a name="windows-media-player-user-interface-plug-ins"></a>Windows Media Player Benutzeroberfläche Plug-Ins

Microsoft Windows Media Player bietet eine Architektur, mit der der Benutzer Plug-In-Programme installieren und aktivieren kann, die dem vollständigen Modus des Players Benutzeroberflächenfunktionen hinzufügen. Ein typisches Benutzeroberflächen-Plug-In kann es einem Benutzer ermöglichen, die CD zu erwerben, die die aktuell wiedergegebene Spur enthält, oder auf zusätzliche Informationen über den Interpreten zuzugreifen. Dieser Abschnitt des Windows Media Player Software Development Kit (SDK) enthält die Programmierinformationen, die Sie zum Erstellen Ihres eigenen Ui-Plug-Ins benötigen.

Die Dokumentation zum Ui-Plug-In ist in drei Abschnitte unterteilt:



| `Section`                                                                                            | BESCHREIBUNG                                                                                                                                   |
|----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [Informationen zu Benutzeroberfläche Plug-Ins](about-user-interface-plug-ins.md)                                 | Bietet eine Übersicht über die Architektur, die für Benutzeroberflächen-Plug-Ins verwendet wird. Lesen Sie diesen Abschnitt, um sich mit den allgemeinen Konzepten dieser Technologie vertraut zu machen. |
| [Programmierhandbuch für Benutzeroberfläche Plug-Ins](user-interface-plug-ins-programming-guide.md)         | Erläutert, was Sie tun müssen, um ein Benutzeroberflächen-Plug-In zu erstellen. Dieser Abschnitt enthält Beispielcode und schrittweise Prozeduren.                          |
| [Programmierreferenz zu Benutzeroberfläche Plug-Ins](user-interface-plug-ins-programming-reference.md) | Enthält eine ausführliche Referenz für die COM-Schnittstelle und -Methoden, die vom Windows Media Player SDK für Benutzeroberflächen-Plug-Ins unterstützt werden.                    |



 

> [!Note]  
> Windows Media Player 10 Mobile bietet das gleiche Plug-In-Modell wie die Desktop-Windows Media Player. Mit diesem Plug-In-Modell können Sie Hintergrund-Plug-Ins verwenden, um Ereignisse zu überwachen, den Status von Eigenschaften anzuzeigen usw. Im Programmierhandbuch in diesem Abschnitt wird beschrieben, wie Sie mithilfe des Assistenten für Desktop-Windows Media Player-Plug-Ins ein Ui-Plug-In für Windows Media Player 10 Mobile erstellen.
