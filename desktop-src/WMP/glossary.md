# <a name="glossary-windows-media-player-sdk"></a>Glossar (Windows Media Player SDK)

<dl> <dt>

<span id="projectid515.ambient"></span><span id="PROJECTID515.AMBIENT"></span>**Ambient**
</dt> <dd>

Wird von mehr als einem Skin-Element unterstützt.

</dd> <dt>

<span id="projectid515.analog"></span><span id="PROJECTID515.ANALOG"></span>**Analog**
</dt> <dd>

Das herkömmliche Format, in dem Audiodaten und Videos mithilfe eines Wave-oder analogen Signals übertragen werden. Ein analoges Signal funktioniert möglicherweise nicht mit digitalen Referenten; Computer verwenden digitale Signale.

</dd> <dt>

<span id="projectid515.anchor_window"></span><span id="PROJECTID515.ANCHOR_WINDOW"></span>**Anker Fenster**
</dt> <dd>

Ein kleines Fenster, das in der unteren rechten Ecke des Bildschirms angezeigt werden kann, wenn sich Windows Media Player im Skin-Modus befindet. Sie können das Fenster verwenden, um zum vollständigen Modus zurückzukehren und auf andere Befehle zuzugreifen.

</dd> <dt>

<span id="projectid515.announcement"></span><span id="PROJECTID515.ANNOUNCEMENT"></span>**Ankündigung**
</dt> <dd>

Eine Windows Media-Metadatendatei, die einem Player die erforderlichen Informationen zum Empfangen von Inhalten übergibt. Ankündigungs Dateien enthalten Extensible Markup Language (XML)-Skripts.

</dd> <dt>

<span id="projectid515.aspect_ratio"></span><span id="PROJECTID515.ASPECT_RATIO"></span>**Seitenverhältnis**
</dt> <dd>

Das Verhältnis der Breite eines Bilds zu seiner Höhe.

</dd> <dt>

<span id="projectid515.attribute"></span><span id="PROJECTID515.ATTRIBUTE"></span>**Attribut**
</dt> <dd>

Ein Name-Wert-Daten Paar.

</dd> <dt>

<span id="projectid515.bandwidth"></span><span id="PROJECTID515.BANDWIDTH"></span>**Bandbreite**
</dt> <dd>

Die Datenübertragungs Kapazität eines Übertragungsmediums.

</dd> <dt>

<span id="projectid515.bit_rate"></span><span id="PROJECTID515.BIT_RATE"></span>**Bitrate**
</dt> <dd>

Die Anzahl von Bits, die pro Zeiteinheit übertragen werden (in der Regel in Bits pro Sekunde ausgedrückt).

</dd> <dt>

<span id="projectid515.broadcast"></span><span id="PROJECTID515.BROADCAST"></span>**Broadcast**
</dt> <dd>

Eine Methode, mit der ein Client einen Datenstrom empfängt. Während einer Übertragungs Verbindung können Clients den Datenstrom nicht steuern. Dies ist das Gegenteil einer bedarfsgesteuerten Präsentation.

</dd> <dt>

<span id="projectid515.broadcast_multicast"></span><span id="PROJECTID515.BROADCAST_MULTICAST"></span>**Broadcast-Multicast**
</dt> <dd>

Übermittlung eines Datenstroms durch einen Windows Media-Server an viele Clients. Aus Sicht des Clients ist eine Übertragung eine verbindungslose Darstellung, da der Client keine Verbindung mit einem Windows Media-Server herstellt.

</dd> <dt>

<span id="projectid515.buffer"></span><span id="PROJECTID515.BUFFER"></span>**Puffer**
</dt> <dd>

Ein Arbeitsspeicher Bereich, der für die temporäre Speicherung von Daten reserviert ist, bevor diese Daten von einem empfangenden Gerät oder einer Anwendung verwendet werden. Die Pufferung schützt vor der Unterbrechung des Datenflusses.

</dd> <dt>

<span id="projectid515.caption"></span><span id="PROJECTID515.CAPTION"></span>**Beschriftung**
</dt> <dd>

Text, der Bilder oder Videos begleitet, entweder als ergänzende Beschreibung oder als Aufzeichnung von gesprochenen Wörtern.

</dd> <dt>

<span id="projectid515.cd"></span><span id="PROJECTID515.CD"></span>**CD**
</dt> <dd>

Siehe Definition für: Compact Disk (CD)

</dd> <dt>

<span id="projectid515.client"></span><span id="PROJECTID515.CLIENT"></span>**Client**
</dt> <dd>

Jeder Computer oder jedes Programm, das eine Verbindung mit einem anderen Computer oder Programm herstellt oder diese anfordert. Der Client kann auch auf die Software verweisen, mit der der Computer oder das Programm die Verbindung herstellen kann.

</dd> <dt>

<span id="projectid515.clipping_image"></span><span id="PROJECTID515.CLIPPING_IMAGE"></span>**Clipping-Bild**
</dt> <dd>

Ein Bild, das nicht angezeigt wird, sondern definiert stattdessen den sichtbaren Bereich eines anderen Bilds.

</dd> <dt>

<span id="projectid515.closed_disc"></span><span id="PROJECTID515.CLOSED_DISC"></span>**geschlossener Disc**
</dt> <dd>

Eine CD oder DVD, die während des Brennvorgangs einen Prozess durchlaufen hat, der den Inhalt der Festplatte für Wiedergabe Programme und Geräte sichtbar macht. Nachdem ein Datenträger geschlossen wurde, können keine weiteren Daten mehr an ihn gebrannt werden. Einige Wiedergabe Geräte, wie z. b. Portier Bare, Home-und Auto-CD-und DVD-Player, können keine nicht geschlossenen Scheiben abspielen.

</dd> <dt>

<span id="projectid515.codec"></span><span id="PROJECTID515.CODEC"></span>**Codec**
</dt> <dd>

Eine Abkürzung für "Kompressor/Debug". Software oder Hardware zum Komprimieren und decoden von digitalen Medien.

</dd> <dt>

<span id="projectid515.compact_disc__cd_"></span><span id="PROJECTID515.COMPACT_DISC__CD_"></span>**CD (CD)**
</dt> <dd>

Ein optisches Speichermedium für digitale Daten.

</dd> <dt>

<span id="projectid515.compression"></span><span id="PROJECTID515.COMPRESSION"></span>**Komprimierung**
</dt> <dd>

Ein Prozess zum Entfernen redundanter Daten aus einer digitalen Mediendatei oder einem Stream, um die Größe oder die verwendete Bandbreite zu reduzieren.

</dd> <dt>

<span id="projectid515.content"></span><span id="PROJECTID515.CONTENT"></span>**Inhalt**
</dt> <dd>

Audioinformationen, Videos, Bilder, Text oder andere Informationen, die in einer digitalen Mediendatei oder einem Stream enthalten sind.

</dd> <dt>

<span id="projectid515.down_image"></span><span id="PROJECTID515.DOWN_IMAGE"></span>**Bild nach unten**
</dt> <dd>

Ein Bild, das angezeigt wird, wenn sich die Maustaste an der nach-unten-Position über einem-Steuerelement befindet.

</dd> <dt>

<span id="projectid515.download_def"></span><span id="PROJECTID515.DOWNLOAD_DEF"></span>**herunterladen**
</dt> <dd>

So kopieren Sie eine Datei mithilfe eines Modem oder Netzwerks von einem Computer auf einen anderen

</dd> <dt>

<span id="projectid515.element"></span><span id="PROJECTID515.ELEMENT"></span>- **Element**
</dt> <dd>

Eine grundlegende syntaktische Einheit in Markup Sprachen, z. b. html oder XML. Als Trennzeichen für Elemente dient Start Tags und Endtags. Leere Elemente werden mithilfe eines Empty-Element-Tags definiert.

</dd> <dt>

<span id="projectid515.encode"></span><span id="PROJECTID515.ENCODE"></span>**Codieren**
</dt> <dd>

, Um Audioinhalte und Videoinhalte in ein bestimmtes digitales Format zu konvertieren, in der Regel mit Komprimierungs Technologie oder Verschlüsselungstechnologie.

</dd> <dt>

<span id="projectid515.error_correction"></span><span id="PROJECTID515.ERROR_CORRECTION"></span>**Fehlerkorrektur**
</dt> <dd>

In Windows Media Player ein Prozess, mit dem sichergestellt wird, dass digitale Audiodaten während der Wiedergabe oder des Kopierens genau vom CD-ROM-Laufwerk gelesen werden. Durch die Verwendung der Fehlerkorrektur können unerwünschte Geräusche verhindert werden, die nicht Teil des ursprünglichen Materials sind.

</dd> <dt>

<span id="projectid515.extensible_markup_language__xml_"></span><span id="PROJECTID515.EXTENSIBLE_MARKUP_LANGUAGE__XML_"></span>**Extensible Markup Language (XML)**
</dt> <dd>

Eine Markup Sprache, die ein Format zum beschreiben strukturierter Daten bereitstellt. XML ist eine World Wide Web Consortium Spezifikation (W3C) und eine Teilmenge Standard Generalized Markup Language (SGML).

</dd> <dt>

<span id="projectid515.file_format"></span><span id="PROJECTID515.FILE_FORMAT"></span>**Dateiformat**
</dt> <dd>

Die Struktur oder Organisation der Daten in einer Datei. Das Dateiformat wird normalerweise durch die Dateinamenerweiterung angegeben.

</dd> <dt>

<span id="projectid515.file_name_extension"></span><span id="PROJECTID515.FILE_NAME_EXTENSION"></span>**Dateinamenerweiterung**
</dt> <dd>

Eine Reihe von Zeichen, die am Ende eines Datei namens hinzugefügt werden, der den Dateityp oder das Format angibt.

</dd> <dt>

<span id="projectid515.file_type"></span><span id="PROJECTID515.FILE_TYPE"></span>**Dateityp**
</dt> <dd>

Das Format einer Datei, die in der Regel durch die Dateinamenerweiterung angegeben wird. Der Dateityp gibt an, in welchem Programm die Datei erstellt wurde, und kann mit geöffnet werden.

</dd> <dt>

<span id="projectid515.frame_rate"></span><span id="PROJECTID515.FRAME_RATE"></span>**Framerate**
</dt> <dd>

Die Anzahl der pro Sekunde angezeigten Video Frames. Höhere Frameraten führen in der Regel zu einer glatteren Bewegung des Bilds.

</dd> <dt>

<span id="projectid515.full_mode"></span><span id="PROJECTID515.FULL_MODE"></span>**Vollständiger Modus**
</dt> <dd>

Der Standard Betriebsstatus von Windows Media Player, von dem alle Features verfügbar sind. Der Player kann auch im Skin-, Mini Player-oder Vollbildmodus angezeigt werden.

</dd> <dt>

<span id="projectid515.global_attribute"></span><span id="PROJECTID515.GLOBAL_ATTRIBUTE"></span>**globales Attribut**
</dt> <dd>

Ein Attribut, das einer Skin-Definitionsdatei als Ganzes zugeordnet ist, auf die von jedem Element aus zugegriffen werden kann.

</dd> <dt>

<span id="projectid515.hover_image"></span><span id="PROJECTID515.HOVER_IMAGE"></span>**Bild mit Hover**
</dt> <dd>

Ein Bild, das für ein Steuerelement angezeigt wird, wenn mit dem Mauszeiger darauf gezeigt wird.

</dd> <dt>

<span id="projectid515.http"></span><span id="PROJECTID515.HTTP"></span>**Http**
</dt> <dd>

Siehe Definition für: Hypertext Transfer Protocol (http)

</dd> <dt>

<span id="projectid515.hypertext_transfer_protocol__http_"></span><span id="PROJECTID515.HYPERTEXT_TRANSFER_PROTOCOL__HTTP_"></span>**Hypertext Transfer Protocol (http)**
</dt> <dd>

Ein Standardsatz von Regeln, die von Computern zum Übertragen von Informationen über das Internet verwendet werden.

</dd> <dt>

<span id="projectid515.intelligent_streaming"></span><span id="PROJECTID515.INTELLIGENT_STREAMING"></span>**intelligentes Streaming**
</dt> <dd>

Ein Streamingtyp, der Netzwerkbedingungen erkennt und die Eigenschaften eines Videos oder Audiostreams anpasst, um die Qualität zu maximieren.

</dd> <dt>

<span id="projectid515.library"></span><span id="PROJECTID515.LIBRARY"></span>**Bibliothek**
</dt> <dd>

In Windows Media Player eine Datenbank, die Informationen zu digitalen Mediendateien auf einem Computer, einem Netzlaufwerk oder dem Internet enthält.

</dd> <dt>

<span id="projectid515.licensed_file"></span><span id="PROJECTID515.LICENSED_FILE"></span>**lizenzierte Datei**
</dt> <dd>

Eine Windows Media-Datei mit einer zugeordneten Lizenz, die definiert, wie die Datei wiedergegeben werden kann. Die in der Lizenz aufgeführten Einschränkungen variieren je nach Lizenz Ersteller. Wenn ein CD-Track mithilfe von Windows Media Player kopiert wird, kann der neu erstellten Datei eine Lizenz zugewiesen werden. Unter dieser Lizenz kann die Datei nur auf dem Computer abgespielt werden, auf dem die Datei erstellt wurde.

</dd> <dt>

<span id="projectid515.lossy_compression"></span><span id="PROJECTID515.LOSSY_COMPRESSION"></span>**Verlust von Verlust Verlust**
</dt> <dd>

Ein Prozess zum Komprimieren von Daten, bei denen Informationen, die als unnötig erachtet werden, entfernt und bei der Dekomprimierung nicht wieder hergestellt werden können Wird in der Regel mit Audiodaten und visuellen Daten verwendet, bei denen eine geringfügige Verschlechterung der Qualität akzeptabel ist.

</dd> <dt>

<span id="projectid515.mapping_image"></span><span id="PROJECTID515.MAPPING_IMAGE"></span>**Bild Zuordnung**
</dt> <dd>

Ein Bild, das nicht angezeigt wird, sondern definiert durch die Verwendung von Farben stattdessen bedeutende Bereiche in einem anderen Bild.

</dd> <dt>

<span id="projectid515.media_usage_rights"></span><span id="PROJECTID515.MEDIA_USAGE_RIGHTS"></span>**Nutzungsrechte für Medien**
</dt> <dd>

Berechtigungen zum Verwenden einer geschützten Windows Media-basierten Datei auf eine bestimmte Weise. Beispielsweise kann ein Inhaltsanbieter (z. b. ein Online-Musikspeicher) Ihnen das Recht geben, einen Song auf Ihrem Computer wiederzugeben (ein Play right), den Song auf eine Audio-CD (ein Burn right) zu brennen oder den Song mit einem tragbaren Gerät (Sync right) zu synchronisieren. Der Inhaltsanbieter bestimmt die Bedingungen der Rechte (z. b. welche Rechte er Ihnen und wie lange gewährt), und die Windows Media Digital Rights Management (DRM)-Komponente erzwingt diese Rechte. Wird auch als Lizenz bezeichnet.

</dd> <dt>

<span id="projectid515.metadata"></span><span id="PROJECTID515.METADATA"></span>**Metadaten**
</dt> <dd>

Daten zu Daten. "Title", "Subject", "Author" und "size" sind Beispiele für die Metadaten einer Datei.

</dd> <dt>

<span id="projectid515.metafile_playlist"></span><span id="PROJECTID515.METAFILE_PLAYLIST"></span>**Metadatei-Wiedergabeliste**
</dt> <dd>

</dd> <dt>

<span id="projectid515.method"></span><span id="PROJECTID515.METHOD"></span>**Methode**

Methoden können aus Informationen bestehen, die als Parameter bezeichnet werden, um das resultierende Verhalten zu ändern. Häufig ist das Ergebnis eines Methoden Aufrufes die Rückgabe eines anderen Informations Teils, der dann von einem anderen Teil des Programms verwendet werden kann.

</dd> <dt>

<span id="projectid515.mms_protocol"></span><span id="PROJECTID515.MMS_PROTOCOL"></span>**MMS-Protokoll**

</dd> <dt>

</dt> <dd>

Der Ausschuss, der internationale Standards für das Codieren audiovisueller Informationen in ein digitales, komprimiertes Format erstellt. Das Akronym MPEG wird an den Anfang der einzelnen Spezifikationen angehängt, die vom Ausschuss entwickelt wurden. MPEG-2 bezieht sich z. b. auf den standardmäßigen ISO/IEC-11172.

</dd> <dt>

<span id="projectid515.mpeg"></span><span id="PROJECTID515.MPEG"></span>**MPEG**
</dt> <dd>

Siehe Definition für: Moving Picture Experts Group (MPEG)

</dd> <dt>

<span id="projectid515.multicast"></span><span id="PROJECTID515.MULTICAST"></span>**Multicast**
</dt> <dd>

Eine Methode zur Übermittlung von Inhalten, bei der ein einzelner Stream von einem Medienserver an mehrere Clients übertragen wird. Die Clients haben keine Verbindung mit dem Server. Stattdessen sendet der Server eine einzelne Kopie des Streams über das Netzwerk an Multicast-fähige Router, die die Daten replizieren. Clients können dann den Datenstrom empfangen, indem Sie eine bestimmte Multicast-IP-Adresse und einen Port überwachen.

</dd> <dt>

<span id="projectid515.multipurpose_internet_mail_extension__mime_"></span><span id="PROJECTID515.MULTIPURPOSE_INTERNET_MAIL_EXTENSION__MIME_"></span>**Mehrzweck-Internet Mail Erweiterung (MIME)**
</dt> <dd>

</dd> <dt>

<span id="projectid515.musical_instrument_digital_interface__midi_"></span><span id="PROJECTID515.MUSICAL_INSTRUMENT_DIGITAL_INTERFACE__MIDI_"></span>**Digital Instrumentation Digital Interface (MIDI)**
</dt> <dd>

Eine Spezifikation der Zuordnung von MIDI-Herstellern (MMA). Die Spezifikation definiert ein Protokoll zum Beschreiben von Musikdaten, wie z. b. Hinweise zu und Notiz aus Nachrichten. ein Dateiformat zum Speichern von Musikdaten, die als Standard-MIDI bezeichnet werden. und eine Standard Hardwareschnittstelle.

</dd> <dt>

<span id="projectid515.object"></span><span id="PROJECTID515.OBJECT"></span>**Objekt**

</dd> <dt>

<span id="projectid515.parameter"></span><span id="PROJECTID515.PARAMETER"></span>**Parameter**

</dd> <dt>

<span id="projectid515.preset"></span><span id="PROJECTID515.PRESET"></span>**Voreinstellung**

</dd> <dt>

<span id="projectid515.progress_bar"></span><span id="PROJECTID515.PROGRESS_BAR"></span>**Status** Anzeige

</dd> <dt>

<span id="projectid515.property"></span><span id="PROJECTID515.PROPERTY"></span>**Eigenschaft**

Ein Informationselement, das ein bestimmtes Detail eines Objekts angibt.

</dd> <dt>

<span id="projectid515.protocol"></span><span id="PROJECTID515.PROTOCOL"></span>**Protokoll**

</dd> <dt>

</dd> <dt>

</dd> <dt>

<span id="projectid515.sami"></span><span id="PROJECTID515.SAMI"></span>**Samisch**

Siehe Definition für: synchroner verfügbarer Medienaustausch (Sami)
