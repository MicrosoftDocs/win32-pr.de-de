# <a name="ambientattributestop"></a>AmbientAttributes.top

Das **Top** -Attribut gibt die obere Koordinate des Steuer Elements an oder ruft diese ab.

``` syntax
        elementID.top
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/schreibzahl (**Long**), die den Abstand zwischen dem Steuerelement und dem oberen Rand der übergeordneten **Ansicht** oder der **untergeordneten Ansicht** in Pixel darstellt.  Der Standardwert ist 0 (null). Negative Zahlen sind zulässig. in diesem Fall schneidet der obere Rahmen der **Sicht** oder der **unter Ansicht** das Steuerelement ab.
