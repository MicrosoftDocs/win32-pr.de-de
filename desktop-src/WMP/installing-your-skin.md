# <a name="installing-your-skin"></a>Installieren ihrer Skin

Verwenden Sie das folgende Verfahren, um Ihre Skin zu installieren:

1.  Kopieren Sie alle Ihre Skindateien in einen neuen Ordner.
2.  Kopieren Sie den Ordner, der Ihre Skindateien enthält, in den Ordner Windows Media Player auf Ihrem portablen Gerät.

    Starten Sie Windows Media Player Mobile. Dadurch wird die **Skin Chooser-Datei** gestartet, die über zwei Navigationsschaltflächen verfügt, mit denen Sie durch die verfügbaren Skins scrollen können, und es wird jeweils eine Vorschau angezeigt. Der für Ihre Skin angezeigte Name ist der Name Ihrer Skindefinitionsdatei.

3.  Tippen Sie auf **OK,** um Ihre Skin auszuwählen.
