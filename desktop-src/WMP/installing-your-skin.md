# <a name="installing-your-skin"></a>Installieren der Skin

Verwenden Sie das folgende Verfahren, um Ihre Skin zu installieren:

1.  Kopieren Sie alle Ihre Skin-Dateien in einen neuen Ordner.
2.  Kopieren Sie den Ordner, der Ihre Skin-Dateien enthält, in den Ordner Windows Media Player auf Ihrem tragbaren Gerät.

    Starten Sie Windows Media Player Mobile. Dadurch wird die **Skin-Auswahl** geöffnet, die über zwei Navigations Schaltflächen verfügt, mit denen Sie einen Bildlauf durch die verfügbaren Skins durchführen können, und es wird jeweils eine Vorschau angezeigt. Der für Ihre Skin angezeigte Name ist der Name Ihrer Skin-Definitionsdatei.

3.  Tippen Sie auf **OK** , um Ihre Skin auszuwählen.
