# <a name="primary-images"></a>Primäre Images

Sie möchten für Ihre Skin primäre Images erstellen. Das primäre Bild besteht aus einem Hintergrundbild, das durch das **Ansichts** Element definiert wird, und allen anderen Bildern, die mit bestimmten Elementen kombiniert werden. Wenn Sie z. b. ein **Button** -Element erstellen, müssen Sie die Kunst für diese Schaltfläche einschließen und sicherstellen, dass Sie eine höhere z-Reihenfolge als das Hintergrundbild aufweist.
