# <a name="primary-images"></a>Primäre Images

Sie möchten primäre Images für Ihre Skin erstellen. Das primäre Bild besteht aus einem Hintergrundbild, das durch das **VIEW-Element** definiert wird, und allen anderen Bildern, die bestimmte Elemente enthalten. Wenn Sie z. B. ein **BUTTON-Element** erstellen, müssen Sie die Grafik für diese Schaltfläche einschließen und sicherstellen, dass sie eine höhere Z-Reihenfolge als das Hintergrundbild auf hat.
