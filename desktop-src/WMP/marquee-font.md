# <a name="marquee-font"></a>Marquee-Schriftart

Sie müssen die Schriftart definieren, die für das zu verwendende Marquee-Anzeige Feld verwendet werden soll. Die Schriftart wird durch Kommas getrennt durch Kommas definiert, die das Gesicht, die Größe und die Gewichtung der Schriftart darstellen.

**Schriftart Werte**

Jeder Schriftart Name kann verwendet werden, wenn er wahrscheinlich auf dem Computer des Benutzers installiert ist. Wenn keine Schriftart auf dem Computer gefunden wird, wird eine Alternative vom Betriebssystem ausgewählt. In der folgenden Tabelle sind typegesichter aufgeführt, die normalerweise auf Windows Mobile 2003-basierten Geräten gefunden werden.



| Schriftart       | BESCHREIBUNG              |
|----------------|--------------------------|
| Tahoma         | Eine Sans-Serif-Schriftart.   |
| Lucida Console | Eine quadratische Serif-Schriftart. |



 

**Größen Werte**

Dies ist die Größe der Schriftart in Punkten. Jeder positive ganzzahlige Wert ist gültig, obwohl Zahlen zwischen 10 und 18 empfohlen werden. Größen, die kleiner als 10 sind, sind möglicherweise schwierig zu lesen, und die Größen über 18 stellen Ihnen möglicherweise nicht genügend Platz zur Verfügung, um mehr als ein paar Buchstaben gleichzeitig anzuzeigen.

**Gewichtungswerte**

Die einzigen zulässigen Werte sind in der folgenden Tabelle aufgeführt.



| Wert | BESCHREIBUNG |
|-------|-------------|
| B     | Fett        |
| N     | Normal      |
