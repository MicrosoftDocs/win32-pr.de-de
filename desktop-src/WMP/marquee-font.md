# <a name="marquee-font"></a>Schriftart "Marquee"

Sie müssen die Schriftart definieren, die von dem Anzeigefeld für das Festzelt verwendet wird, das Sie verwenden möchten. Die Schriftart wird durch drei durch Kommas getrennte Werte definiert, die das Gesicht, die Größe und die Gewichtung der Schriftart darstellen.

**Schriftartwerte**

Ein beliebiger Schriftartname kann verwendet werden, wenn er wahrscheinlich auf dem Computer des Benutzers installiert wird. Wenn auf dem Computer keine Schriftart gefunden wird, wird vom Betriebssystem ein alternativer Typ ausgewählt. Die folgende Tabelle zeigt Schriftarten, die sich in der Regel auf Windows Mobile 2003-basierten Geräten befinden.



| Schriftart       | BESCHREIBUNG              |
|----------------|--------------------------|
| Tahoma         | Eine sans-serif-Schriftart.   |
| Lucida Console | Eine Quadrat-Serif-Schriftart. |



 

**Größenwerte**

Dies ist die Größe der Schriftart in Punkten. Jeder positive ganzzahlige Wert ist gültig, es werden jedoch Zahlen zwischen 10 und 18 empfohlen. Größen, die kleiner als 10 sind, sind möglicherweise schwierig zu lesen, und größen über 18 bieten Ihnen möglicherweise nicht genügend Platz, um mehr als einige Buchstaben gleichzeitig anzuzeigen.

**Gewichtungswerte**

Die einzigen zulässigen Werte sind in der folgenden Tabelle aufgeführt.



| Wert | BESCHREIBUNG |
|-------|-------------|
| B     | Fett        |
| N     | Normal      |
