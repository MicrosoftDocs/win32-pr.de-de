# <a name="duration-attribute"></a>Duration-Attribut

Das **Duration** -Attribut ist die Wiedergabedauer des Elements (in Sekunden).

## <a name="applies-to"></a>Gilt für

-   [Audioelemente](audio-item-attributes.md)
-   [Häufig verwendete Windows Media-Dateien](commonly-used-windows-media-file-attributes.md)
-   [Andere Elemente](other-item-attributes.md)
-   [Video Elemente](video-item-attributes.md)

## <a name="remarks"></a>Bemerkungen

Dieses Attribut ist sowohl in der Bibliothek als auch in der digitalen Mediendatei gespeichert.

Die SDK-Konstante für das Windows Media-Format für dieses Attribut ist g \_ wszwmduration.

Um zu ermitteln, ob Sie den Wert dieses Attributs ändern können, verwenden Sie die [Media. isread onlyitem](media-isreadonlyitem.md) -Methode.

|--------------------|---------------------------------------------------|
