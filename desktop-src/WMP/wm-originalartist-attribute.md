# <a name="wmoriginalartist-attribute"></a>WM/originalartist-Attribut

Das **WM/originalartist-** Attribut ist der Name des Künstlers, der den Inhalt ursprünglich erstellt hat.

## <a name="applies-to"></a>Gilt für

-   [Musikdateien](music-file-attributes.md)

## <a name="remarks"></a>Bemerkungen

Dieses Attribut wird nur in einer Musikdatei gespeichert, die nicht in der Bibliothek gespeichert ist.

Die SDK-Konstante für das Windows Media-Format für dieses Attribut ist g \_ wszwmoriginalartist.

Um zu ermitteln, ob Sie den Wert dieses Attributs ändern können, verwenden Sie die [Media. isread onlyitem](media-isreadonlyitem.md) -Methode.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9-Serie oder höher<br/> |
