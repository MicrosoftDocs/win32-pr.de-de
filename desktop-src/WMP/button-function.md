# <a name="button-function"></a>Button-Funktion

In der folgenden Tabelle sind die Werte aufgeführt, die für die Schaltflächen Funktion gültig sind. Sie müssen keine Werte für Funktionen definieren, die nicht in der Skin verwendet werden.



| Wert           | BESCHREIBUNG                                                                                                                                                  |
|-----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Abspielen            | Gibt die aktuelle Auswahl in der Wiedergabeliste wieder.                                                                                                                 |
| Anhalten           | Hält die Wiedergabe des aktuellen Medien Elements an.                                                                                                                   |
| Stop            | Beendet die Wiedergabe des aktuellen Medien Elements.                                                                                                                    |
| Prev (Vorheriger)            | Wechselt zum vorherigen Element in der Wiedergabeliste.                                                                                                                  |
| Nächste            | Wechselt zum nächsten Element in der Wiedergabeliste.                                                                                                                      |
| Shuffle         | Schließt die Wiedergabeliste nach dem Zufallsprinzip.                                                                                                                              |
| Repeatwiedergabe Liste  | Wiederholt die Wiedergabeliste unbegrenzt.                                                                                                                           |
| Mute            | Gibt die Audioausgabe an.                                                                                                                                   |
| PLAYPAUSE       | Schaltet zwischen dem Wiedergabe Zustand und dem anhaltezustand um.                                                                                                          |
| Info            | Stellt Informationen über das wiedergegebene Medien Element bereit. Für Windows Media Player 10 Mobile-oder spätere Skins wird diese Funktion nicht verwendet.                     |
| FullScreen      | Ändert die Größe des Videos an die maximale Größe der Geräte Anzeige.                                                                                                 |
| Windows Media\*  | Öffnet Pocket Internet Explorer und lädt die Website WindowsMedia.com.                                                                                       |
| Playpauseelenhalte\* | Zusätzlich zum Umschalten zwischen dem Wiedergabe Status und dem anhaltezustand können Benutzer auch eine Liveübertragung beenden, wenn keine Pause-Funktionalität verfügbar ist. |
| Volumeup\*      | Erhöht die Audioausgabe.                                                                                                                                  |
| Volumedown\*    | Verringert die Audioausgabe.                                                                                                                                  |



 

\*Verfügbar für Windows Media Player 10 Mobile oder höher.
