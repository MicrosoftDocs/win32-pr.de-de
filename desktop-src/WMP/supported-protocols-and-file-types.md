# <a name="supported-protocols-and-file-types"></a>Unterstützte Protokolle und Dateitypen

Windows Media Player unterstützt mehrere Protokolle und viele Dateitypen, die alle zum Angeben von URL-Werten im Player-Objektmodell für Eigenschaften wie *Player* verwendet werden können. **URL** oder in Windows Media Metadatei-Wiedergabelisten. Darüber hinaus können die unterstützten Protokolle verwendet werden, wenn Protokoll Werte mit den Proxy bezogenen Methoden des **Netzwerk** Objekts angegeben werden.

Die folgenden Digital Media-Dateiformate werden derzeit für die Wiedergabe durch Windows Media Player unterstützt:

-   Advanced Systems Format (ASF)
-   AIF
-   AIFC
-   AIFF
-   AU
-   AVI
-   MID
-   MPE
-   MPEG
-   MPG
-   MPv2
-   MP2
-   MP3
-   M1V
-   SND
-   WAV
-   Windows Media-Dateien mit der Dateinamenerweiterung ". WM"
-   Windows Media Audio (WMA)
-   Windows Media Video (WMV)

Die folgenden Protokolle werden derzeit von Windows Media Player unterstützt.



| Protokoll | BESCHREIBUNG                                                                        |
|----------|------------------------------------------------------------------------------------|
| HTTP     | Hypertext Transfer-Protokoll. Enthält http mit schnellem Cache und Multicast.          |
| RTSP     | Echt Zeit Streaming-Protokoll. Umfasst RTSP mit schnellem Cache.                       |
| RTSPU    | RTSP mit UDP (User Datagram Protocol) verwendet. Umfasst RTSPU mit schnellem Cache        |
| RTSPT    | RTSP wird mit TCP (Transmission Control Protocol) verwendet. Enthält RTSPT mit schnellem Cache |
| mms      | Microsoft Media Server-Protokoll.                                                   |
| MMSU     | Mit UDP verwendete MMS.                                                                 |
| MMST     | Mit TCP verwendete MMS                                                                 |
| Wmpcd    | Ein Protokoll, das von Windows Media Player verwendet wird, um den Zugriff auf CDs zu ermöglichen.                  |
| Wmpdvd   | Ein Protokoll, das von Windows Media Player verwendet wird, um den Zugriff auf DVDs zu ermöglichen.                 |
