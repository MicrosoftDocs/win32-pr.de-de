# <a name="supported-protocols-and-file-types"></a>Unterstützte Protokolle und Dateitypen

Windows Media Player unterstützt mehrere Protokolle und viele Dateitypen, die alle beim Angeben von URL-Werten im Player-Objektmodell für Eigenschaften wie *Player* verwendet werden können. **URL** oder in Windows Media-Metadateiwiedergabelisten. Darüber hinaus können die unterstützten Protokolle verwendet werden, wenn Protokollwerte mit den proxybezogenen Methoden des **Network-Objekts** angegeben werden.

Die folgenden Dateiformate für digitale Medien werden derzeit für die Wiedergabe durch Windows Media Player unterstützt:

-   Advanced Systems Format (ASF)
-   AIF
-   Aifc
-   Aiff
-   AU
-   AVI
-   MID
-   Mpe
-   Mpeg
-   MPG
-   MPv2
-   MP2
-   MP3
-   M1V
-   Snd
-   WAV
-   Windows Mediendateien mit der Dateinamenerweiterung ".wm"
-   Windows Media Audio (WMA)
-   Windows Media Video (WMV)

Die folgenden Protokolle werden derzeit von Windows Media Player unterstützt.



| Protokoll | BESCHREIBUNG                                                                        |
|----------|------------------------------------------------------------------------------------|
| HTTP     | Hypertext Transfer Protocol. Enthält HTTP mit schnellem Cache und Multicast.          |
| RTSP     | Echtzeitstreamingprotokoll. Enthält RTSP mit schnellem Cache.                       |
| Rtspu    | RTSP, der mit dem User Datagram-Protokoll (UDP) verwendet wird. Enthält RTSPU mit schnellem Cache        |
| RTSPT    | RTSP, der mit TCP (Transmission Control Protocol) verwendet wird. Enthält RTSPT mit schnellem Cache |
| mms      | Microsoft Media Server-Protokoll.                                                   |
| MMSU     | MMS, die mit UDP verwendet werden.                                                                 |
| MMST     | MMS, die mit TCP verwendet werden.                                                                 |
| WMPCD    | Ein Protokoll, das von Windows Media Player verwendet wird, um Zugriff auf CDs bereitzustellen.                  |
| WMPDVD   | Ein Protokoll, das von Windows Media Player verwendet wird, um Zugriff auf DVDs bereitzustellen.                 |
