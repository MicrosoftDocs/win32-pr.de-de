# <a name="playlistcheckboxesvisible"></a>Wiedergabeliste. checkboxesvisible

Das **checkboxesvisible** -Attribut gibt einen Wert an bzw. Ruft einen Wert ab, der angibt, ob Kontrollkästchen sichtbar sind.

``` syntax
        elementID.checkboxesVisible
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist ein **boolescher** Wert mit Lese-/Schreibzugriff.



| Wert | BESCHREIBUNG                           |
|-------|---------------------------------------|
| true  | Kontrollkästchen sind sichtbar.              |
| false | Standard. Kontrollkästchen sind nicht sichtbar. |



 

## <a name="remarks"></a>Bemerkungen

In der Spalte ganz links werden Kontrollkästchen angezeigt.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player, Version 7,0 oder höher<br/> |
