# <a name="currentpositiontext"></a>CURRENTPOSITIONTEXT

Dies ist ein vordefiniertes **TEXT-Element** mit den folgenden Standardwerten.

``` syntax
value="wmpprop:player.controls.currentPositionString"
tabstop="true"
justification="right"
```

## <a name="remarks"></a>Hinweise

Dadurch wird ein **TEXT-Element** erstellt, das die aktuelle Zeit der wiedergegebenen Medien anzeigt. Alle Eigenschaften dieses **TEXT-Elements** können überschrieben werden, indem sie explizit angegeben werden.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|----------------------------------------------|
| Version<br/> | Windows Media Player 7.0 oder höher<br/> |
