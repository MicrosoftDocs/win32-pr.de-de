# <a name="currentpositiontext"></a>Currentpositiontext

Dies ist ein vordefiniertes **Text** Element mit den folgenden Standardwerten.

``` syntax
value="wmpprop:player.controls.currentPositionString"
tabstop="true"
justification="right"
```

## <a name="remarks"></a>Bemerkungen

Dadurch wird ein **Text** Element erstellt, das die aktuelle Uhrzeit der wiedergegebenen Medien anzeigt. Alle Eigenschaften dieses **Text** Elements können überschrieben werden, indem Sie explizit angegeben werden.
