# <a name="iwmperroritem-vb-and-c-interface"></a>Iwmperroritem (VB und c#)-Schnittstelle

Bietet Zugriff auf Fehlerinformationen.

Die **iwmperroritem** -Schnittstelle macht die folgenden Eigenschaften verfügbar.



| Eigenschaft                                                                             | BESCHREIBUNG                                                                                |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [CustomURL](wmplibiwmperroritem-iwmperroritem-customurl--vb-and-c.md)               | Ruft die URL einer Website ab, auf der bestimmte Informationen zum Download Fehler beim Codec angezeigt werden. |
| [ErrorCode](wmplibiwmperroritem-iwmperroritem-errorcode--vb-and-c.md)               | Ruft den aktuellen Fehlercode ab.                                                               |
| [errorcontext](wmplibiwmperroritem-iwmperroritem-errorcontext--vb-and-c.md)         | Ruft einen Wert ab, der den Kontext des Fehlers angibt.                                          |
| [errorDescription](wmplibiwmperroritem-iwmperroritem-errordescription--vb-and-c.md) | Ruft eine Beschreibung des Fehlers ab.                                                           |
| [Lösung](wmplibiwmperroritem-iwmperroritem-remedy--vb-and-c.md)                     | Für die zukünftige Verwendung reserviert.                                                                   |



 

Verwenden Sie die folgende Eigenschaft, um eine **iwmperroritem** -Schnittstelle abzurufen.



| Schnittstelle                            | Eigenschaft                                                                   |
|--------------------------------------|----------------------------------------------------------------------------|
| [Iwmperror](iwmperror--vb-and-c.md) | [Item](iwmperror-item--vb-and-c.md) (in c# verwenden **Sie die get \_ Item** -Methode) |



 

## <a name="members"></a>Member

Die **iwmperroritem (VB und c#)** -Schnittstelle definiert keine Member.
