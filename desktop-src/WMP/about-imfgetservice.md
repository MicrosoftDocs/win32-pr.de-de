# <a name="about-imfgetservice"></a>Informationen über "IMF GetService"

Die **imsgetservice** -Schnittstelle ist eine der Schnittstellen, die von einem DSP-Plug-in mit zwei Modi implementiert werden muss. Es verfügt nur über eine Methode, **GetService**. Ein DSP-Plug-in implementiert die **GetService** -Methode, sodass Clients Zeiger auf die vom Plug-in unterstützten Schnittstellen abrufen können, wenn das Plug-in System eigen in der Media Foundation Pipeline ausgeführt wird.
