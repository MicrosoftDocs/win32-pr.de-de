# <a name="navigating-from-windows-media-player-features"></a>Navigieren von Windows Media Player-Features

Wenn Sie einen Musik Online-Store bereitstellen, können Sie die Navigation von der Info Center-Ansicht, von Albuminformationen und von HtmlView-Seiten zu den Aufgabenbereichen Ihres Dienstanbieter ausführen. Der Prozess funktioniert genau so wie unter [Navigieren zwischen Dienstaufgaben](navigating-between-service-task-panes.md)Bereichen beschrieben. Der Unterschied besteht darin, dass Sie von einer Funktion zu einem Dienstaufgaben Bereich navigieren können, jedoch nicht von einem Dienstaufgaben Bereich zu einer Funktion.
