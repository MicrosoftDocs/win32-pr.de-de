# <a name="recordingtimeyearmonth-attribute"></a>RecordingTimeYearMonth-Attribut

Das **RecordingTimeYearMonth-Attribut** ist der Jahres- und Monatsteil des Datums der ursprünglichen Aufzeichnung für Elemente, bei denen sich dieses Datum vom Veröffentlichungsdatum unterscheidet.

## <a name="applies-to"></a>Gilt für

-   [Audioelemente](audio-item-attributes.md)
-   [Fotoelemente](photo-item-attributes.md)
-   [Videoelemente](video-item-attributes.md)

## <a name="remarks"></a>Hinweise

Dieses Attribut wird nur in der Bibliothek gespeichert.

Verwenden Sie die [Media.isReadOnlyItem-Methode,](media-isreadonlyitem.md) um zu bestimmen, ob Sie den Wert dieses Attributs ändern können.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------|
| Version<br/> | Windows Media Player 11<br/> |
