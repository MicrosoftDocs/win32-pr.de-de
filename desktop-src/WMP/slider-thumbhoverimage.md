# <a name="sliderthumbhoverimage"></a>Slider. thumbhoverimage

Das **thumbhoverimage** -Attribut gibt das Bild des Zieh Punkts an oder ruft es ab, das angezeigt wird, wenn mit der Maus darauf gezeigt wird.

``` syntax
        elementID.thumbHoverImage
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese- **/schreibzeichenfolge** , die den Namen einer Bilddatei enthält.

## <a name="remarks"></a>Bemerkungen

Das **thumbhoverimage** ist optional. Wenn Sie nicht bereitgestellt wird, ändert sich das Schieberegler-Steuerelement nicht, wenn mit dem Mauszeiger darauf gezeigt wird.

Die unterstützten Formate sind BMP, JPG, PNG und GIF (keine animierten GIFs).
