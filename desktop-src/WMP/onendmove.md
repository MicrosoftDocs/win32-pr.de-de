# <a name="onendmove"></a>onendmove

Der **onendmove-Ereignishandler** behandelt ein Ereignis, das auftritt, wenn ein Element einen moveTo-Vorgang abschließt.

``` syntax
onendmove
```

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|-----------------------------------------------------|
| Version<br/> | Windows Media Player Version 70 oder höher<br/> |
