# <a name="iwmpplaylistname-property"></a>Iwmpwiedergabe:: Name-Eigenschaft

Die **Name** -Eigenschaft ruft den Namen der Wiedergabeliste ab oder legt ihn fest.

## <a name="syntax"></a>Syntax


```CSharp
public System.String name {get; set;}
```


```VB

Public Property name As System.String
```





## <a name="property-value"></a>Eigenschaftswert

Ein **System. String** -Wert, der der Name der Wiedergabeliste ist.

## <a name="remarks"></a>Bemerkungen

Vor der Verwendung dieser Eigenschaft müssen Sie über Lesezugriff auf die Bibliothek verfügen. Weitere Informationen finden Sie unter [Bibliotheks Zugriff](library-access.md).
