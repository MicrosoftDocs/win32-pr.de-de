# <a name="device-extensions-for-accelerated-metadata-transfer"></a>Geräte Erweiterungen für die beschleunigte metadatenübertragung

In Windows Media Player 10 wurden neue und erweiterte Funktionen für die Synchronisierung digitaler Mediendateien mit tragbaren Geräten eingeführt. Benutzer können ein Gerät mit einem Computer verbinden, Inhalte auf das Gerät übertragen und dann die Verbindung des Geräts trennen, um Inhalte vom Computer zu erhalten.

Wenn der Benutzer das Gerät erneut mit dem Computer verbindet, ist es möglich, dass sich der auf dem Gerät gespeicherte Inhalt seit der vorherigen Verbindung geändert hat. Wenn Sie z. b. eine bestimmte digitale Mediendatei abspielen, wird die Wiedergabe Anzahl für dieses Element geändert. Da aktuelle portable Geräte große Mengen an digitalen Medieninhalten speichern können, wäre der Prozess der Ermittlung von Änderungen zu zeitaufwändig, wenn Windows Media Player die einzelnen digitalen Medienelemente auflisten und überprüfen müssten. Stattdessen können tragbare Gerätehersteller besondere Funktionen implementieren, damit Windows Media Player 10 oder höher Informationen zu Änderungen, die an den auf einem Gerät gespeicherten Inhalt vorgenommen wurden, effizient abrufen können.

In den folgenden Abschnitten werden die Konventionen beschrieben, die zum Implementieren dieser Funktionalität verwendet werden.

-   [Windows Media Device Manager-Geräte Erweiterungen für die metadatenübertragung](windows-media-device-manager-device-extensions-for-metadata-transfer.md)
