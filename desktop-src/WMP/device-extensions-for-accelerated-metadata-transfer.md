# <a name="device-extensions-for-accelerated-metadata-transfer"></a>Geräteerweiterungen für die beschleunigte Metadatenübertragung

Windows Media Player 10 wurden neue und erweiterte Funktionen zum Synchronisieren digitaler Mediendateien mit portablen Geräten eingeführt. Benutzer können ein Gerät mit einem Computer verbinden, Inhalte auf das Gerät übertragen und dann das Gerät trennen, um Inhalte vom Computer zu nutzen.

Wenn der Benutzer das Gerät erneut mit dem Computer verbindet, ist es möglich, dass sich der auf dem Gerät gespeicherte Inhalt seit der vorherigen Verbindung geändert hat. Wenn Sie beispielsweise einfach eine bestimmte digitale Mediendatei wieder geben, ändert sich die Wiedergabeanzahl für dieses Element. Da aktuelle portable Geräte große Mengen digitaler Medieninhalte speichern können, wäre der Prozess der Änderungsermittelung zu zeitaufwändig, wenn Windows Media Player jedes digitale Medienelement aufzählen und überprüfen müssten. Stattdessen können Hersteller portabler Geräte spezielle Funktionen implementieren, um Windows Media Player 10 oder höher das effiziente Abrufen von Informationen über Änderungen an den auf einem Gerät gespeicherten Inhalten zu ermöglichen.

In den folgenden Abschnitten werden die Konventionen beschrieben, die zum Implementieren dieser Funktionalität verwendet werden.

-   [Informationen zu den Metadaten](about-the-metadata.md)
-   [MTP-Geräteerweiterungen für die Metadatenübertragung](mtp-device-extensions-for-metadata-transfer.md)
-   [Windows Media Geräte-Manager-Geräteerweiterungen für die Metadatenübertragung](windows-media-device-manager-device-extensions-for-metadata-transfer.md)
