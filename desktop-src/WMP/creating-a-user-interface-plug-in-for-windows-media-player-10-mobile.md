# <a name="creating-a-user-interface-plug-in-for-windows-media-player-10-mobile"></a>Erstellen eines Benutzeroberfläche-Plug-Ins für Windows Media Player 10 Mobile

Windows Media Player 10 Mobile verwendet das gleiche Benutzeroberflächen-Plug-In-Modell wie die Desktop-Windows Media Player. allerdings kann Windows Media Player 10 Mobile nur mit Plug-Ins der Hintergrundbenutzeroberfläche interagieren. Aufgrund der ähnlichen Plug-In-Modelle gelten die gleichen API-Aufrufe, die für Plug-Ins der Hintergrundbenutzeroberfläche auf dem Desktop gelten, auch für Plug-Ins der Hintergrundbenutzeroberfläche auf einem Windows Mobile-Gerät.

In den folgenden Abschnitten wird beschrieben, wie Sie ein Ui-Plug-In im Hintergrund erstellen, indem Sie vom Assistenten generierten Code aus dem Assistenten für Windows Media Player-Plug-Ins verwenden.



| `Section`                                                                | BESCHREIBUNG                                                                                                                                            |
|------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Erste Schritte](getting-started.md)                                 | Beschreibt, was Sie installieren müssen und wie Sie den Windows Media Player-Plug-In-Assistenten verwenden, um die Erstellung eines Plug-Ins für die Hintergrundbenutzeroberfläche zu automatisieren.    |
| [Ändern von vom Assistenten generierten Code](modifying-wizard-generated-code.md) | Beschreibt, was Sie aus dem vom Assistenten generierten Code ändern müssen, der im vorherigen Abschnitt erstellt wurde, damit er mit Windows Media Player 10 Mobile funktioniert. |
