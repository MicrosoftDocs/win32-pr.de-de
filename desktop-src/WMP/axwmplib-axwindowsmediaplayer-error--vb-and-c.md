# <a name="axwindowsmediaplayererror-property"></a>AxWindowsMediaPlayer. Error (Eigenschaft)

Die Error-Eigenschaft ruft die iwmperror-Schnittstelle ab, die Zugriff auf eine Auflistung von iwmperroritem-Schnittstellen bereitstellt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```CSharp
public IWMPError Error {get;}
```


```VB

Public ReadOnly Property Error As IWMPError
```





## <a name="property-value"></a>Eigenschaftswert

Die WMPLib. iwmperror-Schnittstelle.
