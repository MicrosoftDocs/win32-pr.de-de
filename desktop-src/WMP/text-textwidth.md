# <a name="texttextwidth"></a>Text. TextWidth

Das **TextWidth** -Attribut ruft die Breite des Texts in Pixel ab, der im **value** -Attribut enthalten ist.

``` syntax
        elementID.textWidth
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine schreibgeschützte **Zahl** (**int**).

## <a name="remarks"></a>Bemerkungen

Der zurückgegebene Wert basiert auf dem **fontface**-Attribut, dem **FontSize**-Attribut und dem **FontStyle** -Attribut sowie auf den tatsächlichen Zeichen in der **value** -Attribut Zeichenfolge.

Im [value](text-value.md) -Attribut finden Sie ein Beispiel, das veranschaulicht, wie die Attribute des **Text** -Elements verwendet werden.
