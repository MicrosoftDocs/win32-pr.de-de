# <a name="viewbackgroundtiled"></a>Ansicht. backgroundkacheln

Mit dem **Background** -Attribut wird ein Wert angegeben oder abgerufen, der angibt, ob das Hintergrundbild der **Ansicht** oder der **unter Ansicht** gekachelt ist.

``` syntax
        elementID.backgroundTiled
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist ein **boolescher** Wert mit Lese-/Schreibzugriff.



| Wert | BESCHREIBUNG                                        |
|-------|----------------------------------------------------|
| true  | Das Bild wird horizontal und vertikal wiederholt. |
| false | Standard. Das Bild wird nicht wiederholt.                |



 

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player, Version 7,0 oder höher<br/> |
