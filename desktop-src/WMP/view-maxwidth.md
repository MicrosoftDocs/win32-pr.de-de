# <a name="viewmaxwidth"></a>VIEW.maxWidth

Das **maxWidth-Attribut** gibt die maximale Breite der **VIEW** in Pixel an oder ruft sie ab, wenn die Größe geändert wird.

``` syntax
        elementID.maxWidth
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/Schreibnummer (**long**) mit einem Wert von 0 (null) oder größer.  Sie hat den Standardwert 0 (null), was bedeutet, dass die maximale Breite von VIEW nicht eingeschränkt **ist.**

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player Version 7.0 oder höher<br/> |
