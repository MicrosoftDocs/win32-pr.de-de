# <a name="viewmaxwidth"></a>View. MaxWidth

Das Attribut **MaxWidth** gibt die maximale Breite in Pixel der **Ansicht** an, wenn die Größe geändert wird.

``` syntax
        elementID.maxWidth
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/schreibzahl (**Long**) mit einem Wert von 0 (null) oder größer.  Der Standardwert ist 0 (null), was bedeutet, dass keine Einschränkung für die maximale Breite der **Ansicht** vorliegt.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player, Version 7,0 oder höher<br/> |
