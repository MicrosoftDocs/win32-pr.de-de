# <a name="about-the-metadatatext-object"></a>Informationen zum MetadataText-Objekt

Einige Metadatenattribute, wie z. b. **WM/Liedtext \_**, stellen Text bereit, der von einer Beschreibung des Texts begleitet wird. Das **MetadataText** -Objekt bietet eine Möglichkeit, den Text und die Beschreibung separat abzurufen. Wenn Sie *Medien* verwenden. **getItemInfoByType** zum Abrufen von Informationen zu komplexen Textattributen rufen Sie ein **MetadataText** -Objekt ab, das über Eigenschaften verfügt, die die einzelnen Daten enthalten.
