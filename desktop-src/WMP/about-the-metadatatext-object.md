# <a name="about-the-metadatatext-object"></a>Informationen zum MetadataText-Objekt

Einige Metadatenattribute, z. **B. WM/Synchronisierungs \_ synchronisiert,** stellen Text zusammen mit einer Beschreibung des Texts bereit. Das **MetadataText-Objekt** bietet eine Möglichkeit, den Text und die Beschreibung separat abzurufen. Wenn Sie Media *verwenden.* **getItemInfoByType** zum Abrufen von Informationen zu komplexen Textattributen rufen Sie ein **MetadataText-Objekt** mit Eigenschaften ab, die die einzelnen Daten enthalten.
