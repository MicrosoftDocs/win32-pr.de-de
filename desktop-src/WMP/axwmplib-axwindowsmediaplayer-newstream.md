# <a name="newstream-event-of-the-axwindowsmediaplayer-object"></a>NewStream-Ereignis des AxWindowsMediaPlayer-Objekts

Das newStream-Ereignis ist für die zukünftige Verwendung reserviert.

``` syntax
[C#]
private void player_NewStream(
  object sender,
  System.EventArgs e
)

[Visual Basic]
Private Sub player_NewStream(  
  sender As Object,
  e As System.EventArgs
) Handles player.NewStream
```

## <a name="event-data"></a>Ereignisdaten

Dieses Ereignis enthält keine Daten.

## <a name="remarks"></a>Bemerkungen

Dieses Ereignis ist für die zukünftige Verwendung reserviert.
