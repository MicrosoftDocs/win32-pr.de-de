# <a name="image-sizes"></a>Bildgrößen

Beim Erstellen von Grafiken für Skins müssen Sie die Größe des Anzeige Bereichs berücksichtigen. In der folgenden Tabelle werden die Größe des Anzeige Bereichs für verschiedene Bildschirmkonfigurationen angezeigt.



| DPI (Modus)                    | Breite in Pixel | Höhe in Pixel |
|-------------------------------|-----------------|------------------|
| 96 (Hochformat)                 | 240             | 320              |
| 96 (Querformat)                | 320             | 240              |
| 192 (Hochformat)                | 480             | 640              |
| 192 (Querformat)               | 640             | 220              |
| 96 (quadratisch)                   | 240             | 240              |
| 192 (quadratisch)                  | 480             | 480              |
| 96 (Hochformat für Smartphone)  | 176             | 220              |
| 131 (Hochformat für Smartphone) | 240             | 320              |



 

> [!Note]  
> Ältere Versionen als Windows Media Player 9-Serie für Windows Mobile 2003 unterstützen nur 96 dpi-Hochformat-Skins. Windows Media Player 10 Mobile oder höher unterstützt 131 dpi-Hochformat Skins auf Smartphone-Geräten, und diese Unterstützung wird in zukünftigen Versionen von Windows Media Player Mobile voraussichtlich fortgesetzt.
