# <a name="image-sizes"></a>Bildgrößen

Beim Erstellen von Grafiken für Skins müssen Sie die Größe des Anzeigebereichs berücksichtigen. Die folgende Tabelle zeigt die Anzeigebereichsgrößen für verschiedene Bildschirmkonfigurationen.



| DPI (Modus)                    | Breite in Pixel | Höhe in Pixel |
|-------------------------------|-----------------|------------------|
| 96 (Hochformat)                 | 240             | 320              |
| 96 (Querformat)                | 320             | 240              |
| 192 (Hochformat)                | 480             | 640              |
| 192 (Querformat)               | 640             | 220              |
| 96 (quadratisch)                   | 240             | 240              |
| 192 (quadratisch)                  | 480             | 480              |
| 96 (Hochformat für Smartphone)  | 176             | 220              |
| 131 (Hochformat für Smartphone) | 240             | 320              |



 

> [!Note]  
> Versionen vor Windows Media Player 9-Serie für Windows Mobile 2003 unterstützen nur 96 DPI-Skins im Hochformat. Windows Media Player 10 Mobile oder höher unterstützt 131 DPI-Skins im Hochformat auf Smartphonegeräten. Diese Unterstützung wird in zukünftigen Versionen von Windows Media Player Mobile voraussichtlich fortgesetzt.
