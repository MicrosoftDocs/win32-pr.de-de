# <a name="building-a-visualization"></a>Erstellen einer Visualisierung

Sie können eine benutzerdefinierte Visualisierung erstellen, indem Visual Studio zusammen mit dem Windows Media Player-Plug-In-Assistenten verwenden. Informationen zum Einrichten des Assistenten und zum Erstellen eines Plug-Ins finden Sie unter [Erstellen eines Plug-Ins.](building-a-plug-in.md)

Wenn Sie den Assistenten ausführen, können Sie verschiedene Arten von Plug-Ins auswählen. Wählen Sie Visualisierung aus.

Nachdem Sie den Assistenten abgeschlossen und Ihr Plug-In erstellt haben, starten Sie einen Titel in Windows Media Player und klicken Sie sich durch die Visualisierungen. Die Voreinstellungen, die automatisch für Sie erstellt werden, werden als *Projektname* Balken und *Projektname* Wave bezeichnet, wobei *Projektname* der Name Ihres Projekts ist.
