# <a name="building-a-visualization"></a>Aufbau einer Visualisierung

Sie können eine benutzerdefinierte Visualisierung erstellen, indem Sie Visual Studio zusammen mit dem Windows Media Player-Plug-in-Assistenten verwenden. Informationen zum Einrichten des Assistenten und zum Einrichten eines Plug-Ins finden Sie unter Einrichten [eines Plug-ins](building-a-plug-in.md).

Wenn Sie den Assistenten ausführen, können Sie verschiedene Arten von Plug-ins auswählen. Wählen Sie Visualisierung aus.

Nachdem Sie den Assistenten fertiggestellt und das Plug-in erstellt haben, starten Sie einen Song in Windows Media Player und klicken Sie durch die Visualisierungen. Die Voreinstellungen, die automatisch für Sie erstellt werden, werden als *ProjectName* -Balken und *ProjectName* Wave bezeichnet, wobei *ProjectName* der Name Ihres Projekts ist.
