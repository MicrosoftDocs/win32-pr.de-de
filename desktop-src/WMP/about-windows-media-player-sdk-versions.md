# <a name="about-windows-media-player-sdk-versions"></a>Informationen zu Windows Media Player SDK-Versionen

Die aktuelle Version des Windows Media Player SDK ist Version 12. Das Windows Media Player 12 SDK baut auf den Features auf, die in früheren Versionen des Windows Media Player SDK ab Version 7 eingeführt wurden. Die Informationen in dieser Dokumentation enthalten Funktionen der folgenden Windows Media Player SDK-Versionen:

-   Windows Media Player 7 SDK
-   Windows Media Player 7.1 SDK
-   SDK der Windows Media Player 9-Serie
-   Windows Media Player 10 SDK
-   Windows Media Player 11 SDK
-   Windows Media Player 12 SDK

Jede SDK-Version ist einem Release von Windows Media Player zugeordnet. Mit dem Windows Media Player 10 SDK wurden beispielsweise neue Features eingeführt, die von Windows Media Player 10 unterstützt werden. Beachten Sie, dass kein Update für das Windows Media Player SDK mit Windows Media Player für Windows XP veröffentlicht wurde.

In der gesamten Dokumentation werden ggf. Versionsanforderungen für jedes Feature bereitgestellt. Wenn keine bestimmten Versionsinformationen bereitgestellt werden, wurde das Feature im Windows Media Player 7 SDK eingeführt und wird in allen nachfolgenden Versionen unterstützt.

Informationen zum Aktualisieren von Projekten, die auf dem Windows Media Player 6.4 SDK basieren, um das Objektmodell für Windows Media Player 7 oder höher zu verwenden, finden Sie im [Leitfaden zur Migration von Objektmodellen.](object-model-migration-guide.md)
