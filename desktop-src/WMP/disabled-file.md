# <a name="disabled-file"></a>Deaktivierte Datei

Die deaktivierte Datei enthält die Bilder, die angezeigt werden, wenn eine bestimmte Schaltflächen Funktion nicht verwendet werden kann oder wenn ein Schaltflächen Zustand deaktiviert ist. Wenn z. b. eine leere Wiedergabeliste definiert ist, werden die Schaltflächen **weiter** und **Prev** angezeigt, und Sie sollten mithilfe eines deaktivierten Bilds angezeigt werden. Außerdem wird für UMSCHALT Flächen das deaktivierte Bild verwendet, um anzugeben, dass der Zustand deaktiviert ist.

Das folgende Bild ist eine typische deaktivierte Datei.

![deaktivierte Datei](images/cesdkdis.png)

Dadurch werden die Bilder für die Schaltflächen für den Treffer-Typ gespeichert, die deaktiviert sind. Die Bilder ähneln der Hintergrund Datei, aber die Farben sind heller. Mit dem Offset, der im Abschnitt Bitmaps der Skin-Definitionsdatei definiert ist, werden die Schaltflächen Bilder mit dem Hintergrund Datei Bild angezeigt.

Beachten Sie, dass der Hintergrund des Schaltflächen Bilds genau mit dem entsprechenden Bereich in der Hintergrund Datei übereinstimmt. Dies ist wichtig, da das gesamte Rechteck, das für das deaktivierte Bild definiert ist, den entsprechenden Bereich in der Hintergrund Datei ersetzt, wenn die Schaltfläche "Treffer" nicht verfügbar ist. Halten Sie die Grafik mit dem Hintergrundbild konsistent, um sicherzustellen, dass sich nur die Teile der Schaltfläche, die unterschiedlich angezeigt werden sollen, tatsächlich ändern.
