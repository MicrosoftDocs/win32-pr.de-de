# <a name="playerapplicationplayerdocked"></a>PlayerApplication.playerDocked

Die **playerDocked-Eigenschaft** ruft einen Wert ab, der angibt, ob sich Windows Media Player in einem andockten Zustand befindet.

## <a name="syntax"></a>Syntax

*playerApplication*. **playerDocked**

## <a name="possible-values"></a>Mögliche Werte

Diese Eigenschaft ist ein **schreibgeschützter boolescher** Wert.

## <a name="remarks"></a>Hinweise

Diese Eigenschaft wird nur beim Remoting des Windows Media Player-Steuerelements verwendet.

**Windows Media Player 10 Mobile:** Diese Eigenschaft gibt immer **false** zurück.
