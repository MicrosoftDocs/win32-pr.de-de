<span data-ttu-id="d1e6f-121">Das Einbetten des Windows Media Player-Steuer Elements in ein Office-Dokument ist eine einfache Möglichkeit, dynamischen, interaktiven digitalen Medieninhalt einem anderweitig statischen Dokument hinzuzufügen.</span><span class="sxs-lookup"><span data-stu-id="d1e6f-121">Embedding the Windows Media Player control in an Office document is an easy way to add dynamic, interactive digital media content to an otherwise static document.</span></span> <span data-ttu-id="d1e6f-122">Beispielsweise können Sie eine Kalkulations Tabelle in Microsoft Excel erstellen und ein "sprechendes Head"-Video einfügen, in dem ein Bericht zusammengefasst wird, oder Sie können ein Microsoft Word-Dokument erstellen und eine kurze Animation einfügen, die einen Punkt im Text veranschaulicht.</span><span class="sxs-lookup"><span data-stu-id="d1e6f-122">For example, you can create a spreadsheet in Microsoft Excel and insert a "talking head" video summarizing a report, or you can create a Microsoft Word document and insert a short animation illustrating a point made in the text.</span></span> <span data-ttu-id="d1e6f-123">Wenn die vom Steuerelement bereitgestellte Benutzeroberfläche nicht gefällt, können Sie Microsoft Visual Basic for Applications (VBA) verwenden, um eine benutzerdefinierte Benutzeroberfläche bereitzustellen.</span><span class="sxs-lookup"><span data-stu-id="d1e6f-123">If you don't like the user interface provided by the control, you can use Microsoft Visual Basic for Applications (VBA) to provide a custom user interface.</span></span>
