# <a name="embedding-the-windows-media-player-control-in-an-office-document"></a>Einbetten des Windows Media Player-Steuerelements in ein Office Dokument

Das Einbetten des Windows Media Player-Steuerelements in ein Office-Dokument ist eine einfache Möglichkeit, dynamischen, interaktiven digitalen Medieninhalt zu einem ansonsten statischen Dokument hinzuzufügen. Sie können z. B. in Microsoft Excel ein Arbeitsblatt erstellen und ein Video einfügen, in dem ein Bericht zusammengefasst wird, oder Sie können ein Microsoft Word-Dokument erstellen und eine kurze Animation einfügen, die einen im Text gemachten Punkt veranschaulicht. Wenn Ihnen die vom Steuerelement bereitgestellte Benutzeroberfläche nicht gefällt, können Sie microsoft Visual Basic for Applications (VBA) verwenden, um eine benutzerdefinierte Benutzeroberfläche zur Verfügung zu stellen.

Durch das Einbetten Windows Media Player -Steuerelements in eine Microsoft PowerPoint®-Bildschirmpräsentation können Sie die dynamischen Effekte ergänzen, die bereits von diesem Programm bereitgestellt werden. Wenn Sie die Präsentation als Webseite speichern, wird das Windows Media Player-Steuerelement in den HTML-Code eingebettet, wie unter Verwenden des [Windows Media Player-Steuerelements in einer Webseite beschrieben.](using-the-windows-media-player-control-in-a-web-page.md) Dadurch können Sie zusätzlichen Skriptcode hinzufügen, den Sie möglicherweise für andere Webseiten entwickelt haben.

Eine weitere Möglichkeit zum einfachen Generieren von Webseiten, die das Windows Media Player-Steuerelement einbetten, ist mit Microsoft FrontPage®. Mit FrontPage können Sie schnell eine Seite erstellen, das Windows Media Player-Steuerelement aus einem Menü hinzufügen und in einem Eigenschaftendialogfeld konfigurieren. Auch hier bettet der resultierende HTML-Code das Windows Media Player wie zuvor beschrieben ein.
