# <a name="embedding-the-windows-media-player-control-in-an-office-document"></a>Einbetten des Windows Media Player-Steuer Elements in ein Office-Dokument

Das Einbetten des Windows Media Player-Steuer Elements in ein Office-Dokument ist eine einfache Möglichkeit, dynamischen, interaktiven digitalen Medieninhalt einem anderweitig statischen Dokument hinzuzufügen. Beispielsweise können Sie eine Kalkulations Tabelle in Microsoft Excel erstellen und ein "sprechendes Head"-Video einfügen, in dem ein Bericht zusammengefasst wird, oder Sie können ein Microsoft Word-Dokument erstellen und eine kurze Animation einfügen, die einen Punkt im Text veranschaulicht. Wenn die vom Steuerelement bereitgestellte Benutzeroberfläche nicht gefällt, können Sie Microsoft Visual Basic for Applications (VBA) verwenden, um eine benutzerdefinierte Benutzeroberfläche bereitzustellen.

Durch Einbetten des Windows Media Player-Steuer Elements in eine Microsoft PowerPoint® Folie Show können Sie die dynamischen Effekte ergänzen, die bereits von diesem Programm bereitgestellt werden. Wenn Sie die Präsentation als Webseite speichern, wird das Windows Media Player-Steuerelement wie unter [Verwenden des Windows Media Player-Steuer Elements auf einer Webseite](using-the-windows-media-player-control-in-a-web-page.md)beschrieben in den HTML-Code eingebettet. Auf diese Weise können Sie zusätzlichen Skriptcode hinzufügen, den Sie möglicherweise für andere Webseiten entwickelt haben.

Eine weitere Möglichkeit zum einfachen Generieren von Webseiten, die das Windows Media Player-Steuerelement einbetten, ist die Microsoft FrontPage-®. FrontPage ermöglicht es Ihnen, schnell eine Seite zu erstellen, das Windows Media Player-Steuerelement über ein Menü hinzuzufügen und es in einem Eigenschaften Dialogfeld zu konfigurieren. Die resultierende HTML bettet das Windows Media Player-Steuerelement wie zuvor beschrieben ein.
