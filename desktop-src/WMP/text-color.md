# <a name="text-color"></a>Textfarbe

Der Farbwert des Typs muss angegeben werden. Der Wert muss als drei positive ganze Zahlen angegeben werden, die durch Kommas getrennt sind. Die Werte stellen die Menge von rot, grün und blau für eine bitmapfarbe dar und liegen zwischen 0 und 255.
