# <a name="effectspreviouspreset"></a>Effects. previoustreuset

Die Methode **previouredset** zeigt die vorherige Voreinstellung der aktuellen Visualisierung an.

``` syntax
        elementID.previousPreset()
```

## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.

## <a name="remarks"></a>Bemerkungen

Wenn die aktuelle Voreinstellung die erste Voreinstellung in der Liste für die aktuelle Visualisierung ist, wird die letzte Voreinstellung aktuell gemacht.

| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player, Version 7,0 oder höher<br/> |
