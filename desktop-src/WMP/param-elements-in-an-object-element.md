<span data-ttu-id="fa11f-116">Wenn Sie z. b. definieren möchten, ob die **Autostart** -Eigenschaft auf true festgelegt ist, würden Sie das param-Element in das Object-Element einbetten.</span><span class="sxs-lookup"><span data-stu-id="fa11f-116">For example, if you want to define whether the **autoStart** property is True, you would embed the PARAM element inside the OBJECT element.</span></span>

```HTML
  CLASSID="CLSID:6BF52A52-394A-11d3-B153-00C04F79FAA6">
    <PARAM name="autoStart" value="True">
</OBJECT>
```
