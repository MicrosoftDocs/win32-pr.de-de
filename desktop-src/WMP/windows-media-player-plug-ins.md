# <a name="windows-media-player-plug-ins"></a>Windows Media Player Plug-Ins

Microsoft Windows Media Player macht Schnittstellen verfügbar, mit denen Sie die Funktionalität auf verschiedene Weise erweitern können. In den folgenden Abschnitten werden die unterstützten Plug-In-Architekturen und der Prozess der Erstellung eines Plug-Ins beschrieben:



| `Section`                                                                                                         | BESCHREIBUNG                                                                                                                                                                                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Erstellen eines Plug-Ins](building-a-plug-in.md)                                                                    | Sie können mehrere Arten von Plug-Ins erstellen, indem Sie Visual Studio und den Assistenten für Windows Media Player Plug-Ins verwenden.                                                                                                                                                                                             |
| [Windows Media Player Benutzerdefinierte Visualisierungen](windows-media-player-custom-visualizations.md)                    | Windows Media Player Visualisierungen sind Component Object Model (COM)-Objekte, die zum Anzeigen visueller Bilder verwendet werden, die mit dem Audioteil der Medienwiedergabe des Players synchronisiert werden. Benutzerdefinierte Visualisierungen können mit Microsoft Visual C++ erstellt werden.                                             |
| [Windows Media Player Benutzeroberfläche-Plug-Ins](windows-media-player-user-interface-plug-ins.md)                | Windows Media Player-Plug-Ins für die Benutzeroberfläche fügen dem Bereich **Jetzt wiedergeben** des Vollmodusplayers neue Funktionen hinzu. Sie können Plug-Ins erstellen, die den Visualisierungsbereich, ein separates Fenster, den Einstellungsbereich, den Metadatenbereich oder Hintergrund-Plug-Ins verwenden, die keine sichtbare Benutzeroberfläche verfügbar machen. |
| [Windows Media Player DSP-Plug-Ins](windows-media-player-dsp-plug-ins.md)                                      | Windows Media Player DSP-Plug-Ins ändern oder verarbeiten Audio- und Videodaten, bevor sie vom Player gerendert werden. DSP-Plug-Ins sind DirectX Media Objects (DMO), die über COM-Schnittstellen eine Verbindung mit dem Player herstellen.                                                                                           |
| [Windows Media Player Rendering-Plug-Ins (veraltet)](/previous-versions/windows/desktop/legacy/dd564683(v=vs.85)) | Windows Media Player Rendering-Plug-Ins decodieren (falls erforderlich) und rendern sie benutzerdefinierte Daten, die in einem Windows Medienformatdatenstrom enthalten sind. Rendering-Plug-Ins sind DirectX Media Objects (DMO), die über COM-Schnittstellen eine Verbindung mit dem Player herstellen.                                                                  |
| [Windows Media Player Konvertierungs-Plug-Ins](windows-media-player-conversion-plug-ins.md)                        | Windows Media Player Konvertierungs-Plug-Ins konvertieren digitale Mediendateien, die mit technologien erstellt werden, die nicht von Microsoft bereitgestellt werden, in advanced Systems Format (ASF).                                                                                                                                 |
