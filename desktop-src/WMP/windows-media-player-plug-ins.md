# <a name="windows-media-player-plug-ins"></a>Windows Media Player-Plug-ins

Microsoft Windows Media Player macht Schnittstellen verfügbar, die es Ihnen ermöglichen, die Funktionalität auf verschiedene Weise zu erweitern. In den folgenden Abschnitten werden die unterstützten Plug-in-Architekturen und der Prozess zum Aufbau eines Plug-Ins beschrieben:



| `Section`                                                                                                         | BESCHREIBUNG                                                                                                                                                                                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Aufbauen eines Plug-ins](building-a-plug-in.md)                                                                    | Mithilfe von Visual Studio und dem Assistenten für Windows Media Player-Plug-Ins können Sie verschiedene Plug-in-Typen erstellen.                                                                                                                                                                                             |
| [Benutzerdefinierte Visualisierungen in Windows Media Player](windows-media-player-custom-visualizations.md)                    | Windows Media Player-Visualisierungen sind Component Object Model (com)-Objekte, die zum Anzeigen von visuellen Bildern verwendet werden, die mit dem Audioteil der Medienwiedergabe des Players synchronisiert werden. Benutzerdefinierte Visualisierungen können mit Microsoft Visual C++ erstellt werden.                                             |
| [Plug-Ins für Windows Media Player-Benutzeroberfläche](windows-media-player-user-interface-plug-ins.md)                | Windows Media Player-Plug-Ins für die Benutzeroberfläche fügen dem Bereich " **jetzt Wiedergabe** " des vollmodusplayers neue Funktionen hinzu. Sie können Plug-ins erstellen, die den Visualisierungs Bereich, ein separates Fenster, den Einstellungsbereich, den Metadatenbereich oder Background-Plug-Ins verwenden, die keine sichtbare Benutzeroberfläche verfügbar machen. |
| [Windows Media Player DSP-Plug-ins](windows-media-player-dsp-plug-ins.md)                                      | Windows Media Player DSP-Plug-ins ändern oder verarbeiten Audiodaten und Videodaten, bevor Sie vom Player gerendert werden. DSP-Plug-ins sind DirectX Media Objects (DMO), die über COM-Schnittstellen eine Verbindung mit dem Player herstellen.                                                                                           |
| [Windows Media Player Rendering-Plug-ins (veraltet)](/previous-versions/windows/desktop/legacy/dd564683(v=vs.85)) | Windows Media Player Rendering-Plug-ins decodieren (falls erforderlich) und Rendern benutzerdefinierter Daten, die in einem Windows Media-formatstream enthalten sind. Rendering-Plug-ins sind DirectX Media Objects (DMO), die über COM-Schnittstellen eine Verbindung mit dem Player herstellen.                                                                  |
| [Windows Media Player-Konvertierungs-Plug-ins](windows-media-player-conversion-plug-ins.md)                        | Windows Media Player-Konvertierungs-Plug-ins konvertieren digitale Mediendateien, die mithilfe von nicht von Microsoft bereitgestellten Technologien erstellt wurden, in das Advanced Systems Format (ASF).                                                                                                                                 |
