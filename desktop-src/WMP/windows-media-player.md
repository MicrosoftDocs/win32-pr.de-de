# <a name="windows-media-player"></a>Windows Media Player

Das Windows Media Player SDK bietet Features, die sich auf das Verhalten von Windows Media Player und Windows Media Player für Windows Mobile auswirken.

In den folgenden Abschnitten werden die SDK-Features ausführlich beschrieben, die für Windows Media Player im Allgemeinen gelten.



| `Section`                                                                                                        | BESCHREIBUNG                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Registrierungseinstellungen](registry-settings.md)                                                                     | Details zu Werten, die Sie in der Registrierung des Benutzers ändern können, damit Windows Media Player benutzerdefinierte Dateinamenerweiterungen erkennen können.                                    |
| [Befehlszeilenparameter](command-line-parameters.md)                                                         | Beschreibt den Satz von Befehlszeilenparametern, die angeben, wie sich Windows Media Player beim Start verhält.                                                              |
| [Audioausgaben](audio-outputs.md)                                                                             | Beschreibt, wie Windows Media Player und das Windows Media Player ActiveX-Steuerelement Standardaudioausgabegeräte auswählen.                                                  |
| [Verteilen von Windows Media Player Software](redistributing-windows-media-player-software.md)               | Stellt Informationen zum Verteilen von Windows Media Player bereit.                                                                                                       |
| [Veröffentlichen von Codecs für Windows Media Player](publishing-codecs-for-windows-media-player.md)                   | Das Veröffentlichen Ihres Codecs auf der WMPlugins-Website wird nicht mehr unterstützt.                                                                                                |
| [Ändern der Größe des Dialogfelds "Lizenzerwerb"](resizing-the-license-acquisition-dialog-box.md)                 | Beschreibt, wie ein Windows Media-Dateiattribut geändert wird, um eine Größe für das Dialogfeld Windows Media Player 10 oder höher des Lizenzerwerbs anzugeben.                     |
| [Geräteerweiterungen für die beschleunigte Metadatenübertragung](device-extensions-for-accelerated-metadata-transfer.md) | Beschreibt, wie Windows Media Player Informationen zu Änderungen an bestimmten Inhaltselementen zwischen Synchronisierungssitzungen von portablen Geräten abruft. |
| [Geräteerweiterungen für die Berichterstellung für erworbene Inhalte](device-extensions-for-reporting-acquired-content.md)       | Beschreibt, wie Windows Media Player eine Liste neuer Inhalte abrufen können, die von einem portablen Gerät zwischen Synchronisierungssitzungen abgerufen werden.                                 |
| [Geräteerweiterungen für Wiedergabelistenobjekteinstellungen](device-extensions-for-playlist-object-preferences.md)     | Beschreibt, wie ein portables Gerät angeben kann, welche Wiedergabelistenobjekttypen Windows Media Player während der automatischen Synchronisierung auf das Gerät kopieren müssen.                |
| [Geräte-E/A-Steuerungscodes](device-i-o-control-codes.md)                                                       | Listet Geräte-E/A-Steuerungscodes auf, die von Windows Media Player 10 oder höher für die Kommunikation mit portablen Geräten über das Windows Media Geräte-Manager SDK verwendet werden.            |
| [Unterstützung benutzerdefinierter Images für Geräte](custom-image-support-for-devices.md)                                       | Beschreibt zwei Bilddateien, die Hersteller portabler Geräte erstellen können, um das Branding in Windows Media Player 10 oder höher anzupassen.                                    |
