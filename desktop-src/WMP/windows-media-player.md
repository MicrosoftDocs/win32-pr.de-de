# <a name="windows-media-player"></a>Windows Media Player

Das Windows Media Player SDK bietet Features, die sich auf das Verhalten von Windows Media Player und Windows Media Player für Windows Mobile auswirken.

In den folgenden Abschnitten werden die SDK-Features ausführlich erläutert, die für Windows-Media Player gelten.



| `Section`                                                                                                        | BESCHREIBUNG                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Registrierungs Einstellungen](registry-settings.md)                                                                     | Detail Werte, die Sie in der Registrierung des Benutzers ändern können, damit Windows Media Player benutzerdefinierte Dateinamen Erweiterungen erkennen kann.                                    |
| [Befehlszeilenparameter](command-line-parameters.md)                                                         | Erläutert den Satz von Befehlszeilen Parametern, die angeben, wie sich Windows Media Player beim Start verhält.                                                              |
| [Audioausgaben](audio-outputs.md)                                                                             | Beschreibt, wie Windows Media Player und das Windows Media Player ActiveX-Steuerelement Standard-Audioausgabegeräte auswählen.                                                  |
| [Neuverteilen von Windows Media Player-Software](redistributing-windows-media-player-software.md)               | Enthält Informationen zur Neuverteilung von Windows Media Player.                                                                                                       |
| [Veröffentlichen von Codecs für Windows Media Player](publishing-codecs-for-windows-media-player.md)                   | Das Veröffentlichen Ihres Codecs auf der wmplugins-Website wird nicht mehr unterstützt.                                                                                                |
| [Ändern der Größe des Dialog Felds zum Erwerb von Lizenzen](resizing-the-license-acquisition-dialog-box.md)                 | Hier wird beschrieben, wie ein Windows Media File-Attribut geändert wird, um eine Größe für das Windows Media Player 10-oder höhere Lizenz Erwerbs Dialogfeld anzugeben.                     |
| [Geräte Erweiterungen für die beschleunigte metadatenübertragung](device-extensions-for-accelerated-metadata-transfer.md) | Beschreibt, wie Windows Media Player Informationen zu Änderungen, die für bestimmte Inhaltselemente zwischen Synchronisierungs Sitzungen auftreten, von tragbaren Geräten abruft. |
| [Geräte Erweiterungen für die Berichterstellung über den Inhalt](device-extensions-for-reporting-acquired-content.md)       | Beschreibt, wie Windows Media Player eine Liste neuer Inhalte abrufen kann, die von einem tragbaren Gerät zwischen Synchronisierungs Sitzungen abgerufen wurden.                                 |
| [Geräte Erweiterungen für Wiedergabelisten Objekt Einstellungen](device-extensions-for-playlist-object-preferences.md)     | Beschreibt, wie ein tragbares Gerät angeben kann, welche Wiedergabelisten Objekttypen Windows Media Player während der automatischen Synchronisierung auf das Gerät kopieren muss.                |
| [Geräte-e/a-Steuerungs Codes](device-i-o-control-codes.md)                                                       | Listet Geräte-e/a-Steuerungs Codes auf, die von Windows Media Player 10 oder höher verwendet werden, um über das Windows Media Device Manager SDK mit tragbaren Geräten zu kommunizieren.            |
| [Unterstützung für benutzerdefinierte Images für Geräte](custom-image-support-for-devices.md)                                       | Beschreibt zwei Bilddateien, die von tragbaren Geräteherstellern zum Anpassen des Brandings in Windows Media Player 10 oder höher erstellt werden können.                                    |
