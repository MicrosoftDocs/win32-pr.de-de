# <a name="modifiedby-attribute"></a>ModifiedBy-Attribut

\[Dieses Attribut wird nicht mehr unterstützt. Der Wert ist immer leer oder "user". \]

Das **ModifiedBy-Attribut** ist eine Zeichenfolge, die angibt, ob die Attribute des Elements vom Benutzer geändert wurden.

## <a name="applies-to"></a>Gilt für

-   [DVDs](dvd-attributes.md)

## <a name="remarks"></a>Hinweise

Dieses Attribut wird nur in der digitalen Mediendatei gespeichert.

Verwenden Sie die [**Media.isReadOnlyItem-Methode,**](media-isreadonlyitem.md) um zu bestimmen, ob Sie den Wert dieses Attributs ändern können.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9er Serie oder höher<br/> |
