# <a name="video-image-source"></a>Video Bildquelle

Für den Videoanzeige Bereich ist kein separates Bild erforderlich. Möglicherweise möchten Sie einen Bereich in Ihrem Hintergrundbild erstellen, der standardmäßig angezeigt wird, um als Platzhalter zu fungieren. Legen Sie den Bereich rechteckig in Form mit derselben Breite und Höhe fest, den Sie in der Skin-Definitionsdatei angeben möchten. Wenn Video abgespielt wird, wird das Video in der Hintergrund Datei über diesem Bereich überlagert.
