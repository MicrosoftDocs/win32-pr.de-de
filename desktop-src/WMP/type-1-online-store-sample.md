# <a name="type-1-online-store-sample"></a>Typ 1 Online Store-Beispiel

Das Windows Media Player SDK enthält ein Beispiel für einen Online Shop vom Typ 1, der von Windows Media Player 11 unterstützt wird.

Zum Anzeigen des Beispiel Typs 1 Online Store konfigurieren Sie Ihre Registrierung wie in [Registrierungs Schlüsseln und Einträgen für den Online Store vom Typ 1](registry-keys-and-entries-for-a-type-1-online-store.md)beschrieben. Der Schlüssel für den Samplingtyp 1 ist 2573.
