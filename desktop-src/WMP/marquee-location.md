# <a name="marquee-location"></a>Marquee-Speicherort

Sie müssen den Speicherort des Marquee-Anzeige Felds definieren, das Sie verwenden möchten. Sie müssen vier positive ganze Zahlen (durch Kommas getrennt) eingeben, die Links, oben, Breite und Höhe des Textanzeige Felds in Pixel relativ zum Hintergrundbild definieren.

Ein typischer Wert ist:


```C++
180,46,50,30 

```
