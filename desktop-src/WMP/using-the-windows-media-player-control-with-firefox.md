# <a name="using-the-windows-media-player-control-with-firefox"></a>Verwenden des Windows Media Player-Steuerelements mit Firefox

Microsoft stellt ein Plug-In zur Verfügung, mit dem Sie das Windows Media Player ActiveX-Steuerelement in eine Webseite einbetten können, damit es von einem Firefox-Browser ordnungsgemäß angezeigt wird. Das Plug-In, das auf dem Clientcomputer installiert werden muss, kann von der [Microsoft Port25-Website heruntergeladen werden.](https://www.mozilla.org/)

Informationen zum Firefox-Browser finden Sie auf der [Firefox-Website.](https://www.mozilla.org/)

Weitere Informationen zur Verwendung von Windows Media Player mit Firefox finden Sie im abschnitt Windows Media Player [der mozilla Wiesn-Website](http://kb.mozillazine.org/Windows_Media_Player).

In den folgenden Themen wird beschrieben, wie Sie das Windows Media Player-Steuerelement in eine Webseite einbetten, die sowohl von Internet Explorer firefox ordnungsgemäß angezeigt werden kann.
