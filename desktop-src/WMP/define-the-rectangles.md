# <a name="define-the-rectangles"></a>Definieren der Rechtecke

Da keine Elemente der Benutzeroberfläche überlappend sein können, ist es hilfreich, die Rechtecke zu definieren, in denen jedes Element gezeichnet wird. Außerdem sind das Erstellen von Rechtecken hilfreich, wenn Sie die Koordinaten schreiben müssen.

Schalten Sie das Photoshop-Raster mit Rasterlinien alle zehn Pixel ein. Aktivieren Sie das Snap-in-Raster, um die rechteckige Zeichnung zu vereinfachen.

Erstellen Sie eine neue Ebene, und zeichnen Sie Rechtecke. Wenn Sie über die untergeordnete Skizzen Ebene verfügen, kennen Sie den allgemeinen Speicherort, an dem das Benutzeroberflächen Element stehen soll, aber Sie können den genauen Speicherort jetzt optimieren. Sie können die Dinge später jederzeit ändern, aber es wird Ihnen dabei helfen, so nah wie möglich zu werden. Die folgende Abbildung zeigt mehrere Rechtecke in einer Ebene.

![Rechteck Ebene](images/ceswmrek.png)

Nachdem Sie nun die Rechtecke definiert haben, können Sie die Skizzen Ebene ausschalten.
