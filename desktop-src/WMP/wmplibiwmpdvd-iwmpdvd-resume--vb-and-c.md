# <a name="iwmpdvdresume-method"></a>Iwmpdvd:: Resume-Methode

Die **Resume** -Methode kehrt im Menü Modus wieder in den Wiedergabemodus zurück und nimmt an derselben Position wieder auf, als wenn das Menü aufgerufen wurde.

## <a name="syntax"></a>Syntax


```CSharp
public void resume();
```


```VB

Public Sub resume()
Implements IWMPDVD.resume
```





## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.
