# <a name="background-file"></a>Hintergrund Datei

Jedes Skin benötigt eine Hintergrund Datei mit dem Bild, das dem Benutzer angezeigt wird, wenn der Skin zum ersten Mal angezeigt wird. Diese Datei wird als Hintergrund Datei in der Skin-Definitionsdatei definiert.

Das folgende Bild ist eine typische Hintergrund Datei.

![Hintergrund Datei](images/cesdkbak.png)

Dies ist das Hintergrundbild, das den Benutzern zuerst angezeigt wird. Sie sehen jedoch auch die beiden Ziehpunkt Bilder an den entsprechenden Positionen auf den trackbars für Seek (oberhalb) und Volume (unten). Außerdem werden alle Schaltflächen, die deaktiviert sind, mithilfe der deaktivierten Images angezeigt.
