# <a name="submitting-your-visualization"></a>Übermitteln Ihrer Visualisierung

Das Übermitteln Ihrer Visualisierung über die WMPlugins-Website wird nicht mehr unterstützt.

## <a name="test-your-visualization"></a>Testen der Visualisierung

Sie sollten versuchen, Ihre Visualisierung auf mehreren Computern zu installieren, um sicherzustellen, dass alles wie sie funktioniert. Probieren Sie jede Voreinstellung aus, um sicherzustellen, dass Ihr Code ordnungsgemäß funktioniert.

## <a name="sign-your-visualization"></a>Signieren Ihrer Visualisierung

Sie müssen Ihre Visualisierung für die Codesignatur übermitteln, damit der Benutzer entscheiden kann, ob er Ihrer Visualisierung vertrauen soll, bevor er sie installiert. Informationen zum Signieren von Code finden Sie [unter Einführung in die Codesignatur.](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/ms537361(v=vs.85))

## <a name="distributing-your-visualization"></a>Verteilen der Visualisierung

Obwohl die WMPlugins-Website die Verteilung Ihrer Visualisierung nicht mehr unterstützt, können Sie Ihre Visualisierung trotzdem von einer Website Ihrer Wahl verteilen.
