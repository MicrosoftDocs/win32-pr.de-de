# <a name="submitting-your-visualization"></a>Übermitteln der Visualisierung

Das übermitteln der Visualisierung über die wmplugins-Website wird nicht mehr unterstützt.

## <a name="test-your-visualization"></a>Testen der Visualisierung

Möglicherweise möchten Sie versuchen, die Visualisierung auf mehreren Computern zu installieren, um sicherzustellen, dass alles wie gewünscht funktioniert. Testen Sie jede Voreinstellung, um sicherzustellen, dass der Code ordnungsgemäß funktioniert.

## <a name="sign-your-visualization"></a>Signieren der Visualisierung

Sie müssen ihre Visualisierung für die Code Signatur übermitteln, damit der Benutzer entscheiden kann, ob der Visualisierung vertraut werden soll, bevor Sie Sie installieren. Weitere Informationen zum Signieren von Code finden [Sie unter Einführung in die Code Signatur](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/ms537361(v=vs.85)).

## <a name="distributing-your-visualization"></a>Verteilen der Visualisierung

Obwohl die wmplugins-Website die Verteilung ihrer Visualisierung nicht mehr unterstützt, können Sie Ihre Visualisierung weiterhin von einer Website Ihrer Wahl verteilen.
