# <a name="customslider-element"></a>Customslider-Element

Das **customslider** -Element bietet eine Möglichkeit zum Erstellen und Bearbeiten eines Schieberegler-Steuer Elements beliebiger Form, z. b. eines kreisförmigen Knopfs. Die folgenden Tabellen enthalten die Attribute und Ereignishandler, die von diesem Element unterstützt werden.

Das **customslider** -Element unterstützt die folgenden Attribute.



| Attribut                                               | BESCHREIBUNG                                                                                                                 |
|---------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [Cursor](customslider-cursor.md)                       | Gibt den Wert des Schieberegler-Steuerelement Cursors an oder ruft ihn ab, der angezeigt wird, wenn sich die Maus über dem Schieberegler befindet               |
| [disabledImage](customslider-disabledimage.md)         | Gibt das Bild des Schiebereglers an, das verwendet wird, wenn der Schieberegler deaktiviert ist.                                            |
| [Windows Mage](customslider-downimage.md)                 | Gibt das Bild des benutzerdefinierten Schiebereglers an oder ruft dieses ab.                                                           |
| [hoverimage](customslider-hoverimage.md)               | Gibt das Bild an oder ruft es ab, das angezeigt wird, wenn sich die Maus über dem benutzerdefinierten Schieberegler befindet                                    |
| [image](customslider-image.md)                         | Gibt den Namen der Datei an, die die den verschiedenen Zuständen des benutzerdefinierten Schiebereglers entsprechenden Bilder enthält, oder ruft ihn ab. |
| [max](customslider-max.md)                             | Gibt den maximalen Wert des Bereichs an, der durch den benutzerdefinierten Schieberegler definiert ist, oder ruft diesen ab.                                         |
| [min](customslider-min.md)                             | Gibt den minimalen Wert des Bereichs an, der durch den benutzerdefinierten Schieberegler definiert ist, oder ruft diesen ab.                                         |
| [positionImage](customslider-positionimage.md)         | Gibt die ImageMap an oder ruft Sie ab, um zu bestimmen, welches Positions Bild aus der **Bilddatei** angezeigt werden soll.             |
| [QuickInfo](customslider-tooltip.md)                     | Gibt den QuickInfo-Text für den Schieberegler an oder ruft ihn ab.                                                                     |
| [transparendcolor](customslider-transparencycolor.md) | Gibt die Transparenz Farbe der benutzerdefinierten Schieberegler-Bilder an oder ruft diese ab.                                                  |
| [value](customslider-value.md)                         | Gibt die aktuelle Position des Schiebereglers an oder ruft diese ab.                                                                  |



 

Das **customslider** -Element kann die folgenden Ereignishandler implementieren.



| Ereignishandler                                         | BESCHREIBUNG                                                                                                          |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [ondragbegin](customslider-ondragbegin.md)           | Behandelt ein Ereignis, das auftritt, wenn der Benutzer auf die linke Maustaste klickt und mit der Maus bewegt wird. |
| [ondragend](customslider-ondragend.md)               | Behandelt ein Ereignis, das auftritt, wenn die linke Maustaste nach einem Zieh Vorgang losgelassen wird.                      |
| [onpositionchange](customslider-onpositionchange.md) | Behandelt ein Ereignis, das auftritt, wenn sich die Position des Schiebereglers ändert, wenn der Benutzer auf klicken oder ziehen klickt.   |



 

Das **customslider** -Element unterstützt die Ambient-Attribute und kann die Umgebungs Ereignishandler implementieren. Weitere Informationen finden Sie unter [Ambient-Attribute](ambient-attributes.md) und [Ambient-Ereignishandler](ambient-event-handlers.md).
