# <a name="customslider-element"></a>ELEMENTSLIDER-Element

Mit dem **ELEMENT DERLIDER** können Sie ein Schieberegler-Steuerelement beliebiger Form erstellen und bearbeiten, z. B. einen kreisförmigen Knöpf. In den folgenden Tabellen sind die Attribute und Ereignishandler aufgeführt, die von diesem Element unterstützt werden.

Das **ELEMENTS DERLIDER** unterstützt die folgenden Attribute.



| attribute                                               | Beschreibung                                                                                                                 |
|---------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [Cursor](customslider-cursor.md)                       | Gibt den Wert des Schieberegler-Steuerelementcursors an, der angezeigt wird, wenn sich der Mauszeiger über dem Schieberegler befindet, oder ruft diesen ab.               |
| [disabledImage](customslider-disabledimage.md)         | Gibt das Bild des Schiebereglers an, der verwendet wird, wenn der Schieberegler deaktiviert ist, oder ruft es ab.                                            |
| [downImage](customslider-downimage.md)                 | Gibt das Bild mit dem Abwärtszustand des benutzerdefinierten Schiebereglers an oder ruft es ab.                                                           |
| [hoverImage](customslider-hoverimage.md)               | Gibt das Bild an, das angezeigt wird, wenn sich der Mauszeiger über dem benutzerdefinierten Schieberegler befindet, oder ruft es ab.                                    |
| [image](customslider-image.md)                         | Gibt den Namen der Datei mit den Bildern an, die den verschiedenen Zuständen des benutzerdefinierten Schiebereglers entsprechen, oder ruft diesen ab. |
| [max](customslider-max.md)                             | Gibt den maximalen Wert des durch den benutzerdefinierten Schieberegler definierten Bereichs an oder ruft den Wert ab.                                         |
| [min](customslider-min.md)                             | Gibt den minimalen Wert des durch den benutzerdefinierten Schieberegler definierten Bereichs an oder ruft den Wert ab.                                         |
| [positionImage](customslider-positionimage.md)         | Gibt die Bildzuordnung an, die verwendet wird, um zu bestimmen, welches Positionsbild aus der **anzuzeigenden Bilddatei** verwendet wird, oder ruft sie ab.             |
| [Quickinfo](customslider-tooltip.md)                     | Gibt den QuickInfo-Text für den Schieberegler an oder ruft den Text ab.                                                                     |
| [transparencyColor](customslider-transparencycolor.md) | Gibt die Transparenzfarbe der benutzerdefinierten Schiebereglerbilder an oder ruft sie ab.                                                  |
| [value](customslider-value.md)                         | Gibt die aktuelle Position des Schiebereglers an oder ruft sie ab.                                                                  |



 

Das **ELEMENTS VONLIDER** kann die folgenden Ereignishandler implementieren.



| Ereignishandler                                         | Beschreibung                                                                                                          |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [onDragBegin](customslider-ondragbegin.md)           | Behandelt ein Ereignis, das auftritt, wenn der Benutzer klickt und die linke Maustaste gedrückt hält und beginnt, die Maus zu ziehen. |
| [onDragEnd](customslider-ondragend.md)               | Behandelt ein Ereignis, das auftritt, wenn die linke Maustaste nach einem Ziehvorgang losgelassen wird.                      |
| [onPositionChange](customslider-onpositionchange.md) | Behandelt ein Ereignis, das auftritt, wenn sich die Position des Schiebereglers ändert, wenn der Benutzer klickt oder zieht.   |



 

Das **ELEMENTS DERLIDER** unterstützt die Ambient-Attribute und kann die Ambient-Ereignishandler implementieren. Weitere Informationen finden Sie unter [Ambient Attributes](ambient-attributes.md) und Ambient [Event Handlers](ambient-event-handlers.md).
