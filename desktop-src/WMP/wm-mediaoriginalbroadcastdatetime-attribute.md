# <a name="wmmediaoriginalbroadcastdatetime-attribute"></a>WM/mediaoriginalbroadcastdatetime-Attribut

Das Attribut **WM/mediaoriginalbroadcastdatetime** gibt das Datum und die Uhrzeit an, zu denen ein angegebenes anzeigen zuerst gesendet wurde.

## <a name="applies-to"></a>Gilt für

-   [Video Elemente](video-item-attributes.md)

## <a name="remarks"></a>Bemerkungen

Die SDK-Konstante für das Windows Media-Format für dieses Attribut ist g \_ wszwmmediaoriginalbroadcastdatetime.

Um zu ermitteln, ob Sie den Wert dieses Attributs ändern können, verwenden Sie die [Media. isread onlyitem](media-isreadonlyitem.md) -Methode.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------|
| Version<br/> | Windows Media Player 10 oder höher<br/> |
