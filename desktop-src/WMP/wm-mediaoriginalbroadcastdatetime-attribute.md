# <a name="wmmediaoriginalbroadcastdatetime-attribute"></a>WM/MediaOriginalBroadcastDateTime-Attribut

Das **WM/MediaOriginalBroadcastDateTime-Attribut** gibt das Datum und die Uhrzeit der ersten Übertragung einer bestimmten Show an.

## <a name="applies-to"></a>Gilt für

-   [Videoelemente](video-item-attributes.md)

## <a name="remarks"></a>Hinweise

Die Windows Media Format SDK-Konstante für dieses Attribut lautet g \_ wszWMMediaOriginalBroadcastDateTime.

Verwenden Sie die [Media.isReadOnlyItem-Methode,](media-isreadonlyitem.md) um zu bestimmen, ob Sie den Wert dieses Attributs ändern können.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|---------------------------------------------|
| Version<br/> | Windows Media Player 10 oder höher<br/> |
