# <a name="drmkeyid-attribute"></a>Drmkeyid-Attribut

Das **drmkeyid** -Attribut identifiziert die Medien Nutzungsrechte für Inhalte, die mit Digital Rights Management (DRM) geschützt sind.

## <a name="applies-to"></a>Gilt für

-   [Audioelemente](audio-item-attributes.md)
-   [Video Elemente](video-item-attributes.md)

## <a name="remarks"></a>Bemerkungen

In der Regel wird dieses Attribut zum Abfragen und Anzeigen von Medien Nutzungsrechten verwendet.

Um zu ermitteln, ob Sie den Wert dieses Attributs ändern können, verwenden Sie die [Media. isread onlyitem](media-isreadonlyitem.md) -Methode.
