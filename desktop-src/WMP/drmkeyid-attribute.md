# <a name="drmkeyid-attribute"></a>DRMKeyID-Attribut

Das **DRMKeyID-Attribut** identifiziert die Mediennutzungsrechte für Inhalte, die mithilfe von Digital Rights Management (DRM) geschützt werden.

## <a name="applies-to"></a>Gilt für

-   [Audioelemente](audio-item-attributes.md)
-   [Videoelemente](video-item-attributes.md)

## <a name="remarks"></a>Hinweise

In der Regel wird dieses Attribut zum Abfragen und Anzeigen von Mediennutzungsrechten verwendet.

Um zu bestimmen, ob Sie den Wert dieses Attributs ändern können, verwenden Sie die [Media.isReadOnlyItem-Methode.](media-isreadonlyitem.md)

|--------------------|------------------------------------|
