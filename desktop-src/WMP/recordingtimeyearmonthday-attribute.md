# <a name="recordingtimeyearmonthday-attribute"></a>Recordingtimeyearmonthday-Attribut

Das Attribut **recordingtimeyearmonthday** ist das Datum der ursprünglichen Aufzeichnung (ohne die Tageszeit), für Elemente, bei denen dieses Datum nicht mit dem Veröffentlichungsdatum identisch ist.

## <a name="applies-to"></a>Gilt für

-   [Audioelemente](audio-item-attributes.md)
-   [Foto Elemente](photo-item-attributes.md)
-   [Video Elemente](video-item-attributes.md)

## <a name="remarks"></a>Bemerkungen

Dieses Attribut ist nur in der-Bibliothek gespeichert.

Um zu ermitteln, ob Sie den Wert dieses Attributs ändern können, verwenden Sie die [Media. isread onlyitem](media-isreadonlyitem.md) -Methode.

|--------------------|------------------------------------|
| Version<br/> | Windows Media Player 11<br/> |
