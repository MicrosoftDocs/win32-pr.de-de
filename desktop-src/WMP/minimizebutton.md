# <a name="minimizebutton"></a>Minimizebutton

Dies ist eine vordefinierte Schaltfläche mit den folgenden Standardwerten.

``` syntax
onclick="jscript:view.minimize();"
upToolTip="Minimize"
```

## <a name="remarks"></a>Bemerkungen

Dadurch wird ein **Schalt** Flächen-Steuerelement erstellt, das den Player minimiert. Die Quick Infos sind lokalisiert. Alle Eigenschaften dieser **Schaltfläche** können überschrieben werden, indem Sie explizit angegeben werden.

|--------------------|----------------------------------------------|
