# <a name="status-color"></a>Status Farbe

Der Farbwert für den Statusanzeige Text muss angegeben werden. Der Wert muss aus drei positiven ganzen Zahlen bestehen, die durch Kommas getrennt sind. Die Werte stellen die Größe von rot, grün und blau für eine Textfarbe dar und reichen von 0 bis 255.
