# <a name="status-color"></a>Statusfarbe

Der Farbwert des Statusanzeigetexts muss angegeben werden. Der Wert muss drei positive ganze Zahlen sein, die durch Kommas getrennt sind. Die Werte stellen die Menge von Rot, Grün und Blau für eine Textfarbe dar und liegen zwischen 0 und 255.
