# <a name="about-the-player-object"></a>Informationen zum Player-Objekt

Das **Player** -Objekt ist das Kern Objekt für das Windows Media Player-Steuerelement. Alle anderen verknüpften Objekte werden über bestimmte Eigenschaften, die das Objekt zurückgeben, mit diesem Objekt verbunden. Beispielsweise erfolgt der Zugriff auf das **Einstellungs** Objekt über die **Settings** -Eigenschaft. Das **Player** -Objekt stellt Methoden, Eigenschaften und Ereignisse bereit, die sich auf die Kernfunktionen von Windows Media Player beziehen.

Da dieser Verweis auch für die Skins-Programmierung verwendet werden soll, ist die ID des **Player** -Objekts für Syntax Beispiele "Player".

Die Verwendung **des Global-** Attributs in einer Skin-Definitionsdatei ermöglicht den Zugriff auf das **Player** -Objekt für die Verwendung bei der Skripterstellung. Über das **Player** -Objekt werden alle anderen Objekte im Windows Media Player-Steuerelement auch für Skripts zugänglich. Die Ereignisse des **Player** -Objekts und der **URL** -Eigenschaft können auch zur Entwurfszeit mit dem [Player](player-element.md) Skin-Element angegeben werden. Weitere Informationen finden Sie unter [Windows Media Player Skins](windows-media-player-skins.md).
