# <a name="about-the-player-object"></a>Informationen zum Player-Objekt

Das **Player-Objekt** ist das Kernobjekt für das Windows Media Player-Steuerelement. Alle anderen verknüpften Objekte werden über bestimmte Eigenschaften, die das Objekt zurückgeben, mit diesem Objekt verbunden. Auf das **Einstellungen-Objekt** wird beispielsweise über die **Einstellungseigenschaft** zugegriffen. Das **Player-Objekt** stellt Methoden, Eigenschaften und Ereignisse bereit, die sich auf die Kernfunktionalität von Windows Media Player beziehen.

Da dieser Verweis auch für die Skinprogrammierung verwendet werden soll, ist die ID des **Player-Objekts** für Syntaxbeispiele "player".

Die  Verwendung des globalen Player-Attributs in einer Skindefinitionsdatei ermöglicht den Zugriff auf das **Player-Objekt** für die Skripterstellung. Über das **Player-Objekt** sind auch alle anderen Objekte im Windows Media Player-Steuerelement für Skripts zugänglich. Die Ereignisse des **Player-Objekts** und der **URL-Eigenschaft** können auch zur Entwurfszeit mit dem [PLAYER-Skinelement](player-element.md) angegeben werden. Weitere Informationen finden Sie unter [Windows Media Player Skins](windows-media-player-skins.md).
