# <a name="playlistisidentical-method"></a>Wiedergabe. isidentical-Methode

Die **isidentical** -Methode ruft einen Wert ab, der angibt, ob das angegebene **Wiedergabe** Listen Objekt mit dem aktuellen-Objekt identisch ist.

## <a name="syntax"></a>Syntax


```JScript
bRetVal = Playlist.isIdentical(
  playlist
)
```



## <a name="parameters"></a>Parameter

<dl> <dt>

*Wiedergabeliste* \[ in\]
</dt> <dd>

**Wiedergabe** Listen Objekt, das mit dem aktuellen-Objekt verglichen werden soll.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt einen **booleschen** Wert zurück.

| Version<br/> | Windows Media Player Version 7,0 oder höher.<br/>                              |
