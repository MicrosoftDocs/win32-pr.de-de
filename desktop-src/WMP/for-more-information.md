# <a name="for-more-information-wmp-sdk"></a>Weitere Informationen (WMP SDK)

-   Weitere Informationen zum Entwickeln digitaler Medienlösungen finden Sie im [Microsoft Windows Media Developer Center.](https://msdn.microsoft.com/windowsmedia/default.aspx)
-   Weitere Informationen zu Windows-Medien finden Sie auf der [Windows Media-Website.](https://support.microsoft.com/help/18612/windows-media-player)
-   Weitere Informationen zu Windows Media Player finden Sie auf der Windows Media Player [auf](https://support.microsoft.com/help/18612/windows-media-player) der Windows Media-Website.
-   Um das Windows SDK herunterzuladen, das das Windows Media Player SDK enthält, wechseln Sie zur Seite [Windows SDK](https://msdn.microsoft.com/windows/bb980924.aspx) im Windows Developer Center.
-   Weitere Informationen zum Programmieren mit Microsoft-Technologien, einschließlich Windows Medientechnologien und Software Development [Kits,]()finden Sie in der MSDN Library .
-   Die aktuelle Version des Windows Media Player SDK ist Version 12. Informationen zu Problemen im Zusammenhang mit dem Windows Media Player 11 SDK finden Sie in der Windows Media Player [11 SDK-Readme](/previous-versions/aa905044(v=msdn.10)).
