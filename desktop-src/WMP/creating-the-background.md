# <a name="creating-the-background"></a>Erstellen des Hintergrunds

Nun können Sie das Hintergrundbild erstellen. Halten Sie die Rechteck Ebene sichtbar, behalten Sie Sie jedoch hinter der Ebene, die Sie zeichnen, und verwenden Sie Sie als Leitfaden zum Erstellen der Bilder für Ihre Skin. Sie möchten jedes Element in einer oder mehreren Ebenen zusammenstellen, damit es später problemlos in eine andere Datei kopiert werden kann. Beispielsweise muss die Schaltfläche "PLAYPAUSE" in die angehaltene Datei, die deaktivierte Datei und die Regions Datei kopiert und in diese kopiert werden. Wenn Sie Kunst in die Regions Datei verschieben, achten Sie darauf, dass die von Ihnen kopierende Kopie keine komplexen edgeeffekte wie z. b. Schatten oder Bilder enthält.

Wenn Sie die Erstellung des gewünschten Images abgeschlossen haben, können Sie alle Ebenen sichtbar machen, die Sie verwenden möchten, und die anderen ausblenden (z. b. Sketch und Rechteck). Verwenden Sie dann den Befehl Kopie speichern, um die Grafik in einer BMP-Datei zu speichern.

Das folgende Diagramm ist das Hintergrundbild.

![Hintergrund Datei](images/ceswmbak.png)

Mit einem Bild, einer Prägung und einem Schatten wurden die visuellen Effekte für die verschiedenen Elemente erzeugt. Beachten Sie, dass es keine Schaltfläche für das volumethumb-Bild gibt. Er wird später erstellt.

Photoshop hat mehrere visuelle Effekte, die Sie auf Bilder anwenden können, sodass Sie das Erstellen interessanterer Grafiken erleichtern können. Beispielsweise wurde die Schaltfläche PLAYPAUSE mit einem einfachen Kreis erstellt. Anschließend wurde eine einschrägung und ein Prägung mit Schatteneffekt erstellt. Der durch vollste Pfeil war ein einfaches Dreieck, das mit drei Klicks des Stift Tools erstellt wurde. Der Pfeil wurde dann mithilfe eines Effekts mit einem geprägten aussehen versehen. Wenn Sie ein Programm wie Photoshop verwenden, müssen Sie nicht zeichnen können. Sie können einfach einfache Images erstellen und Effekte anwenden. Sie können auch eigene Fotos oder gescannte Grafiken importieren.
