# <a name="creating-the-background"></a>Erstellen des Hintergrunds

Nun können Sie das Hintergrundbild erstellen. Halten Sie die Rechteckebene sichtbar, behalten Sie sie jedoch hinter der Ebene bei, auf der Sie zeichnen, und verwenden Sie sie als Leitfaden, um die Bilder für Ihre Skin zu erstellen. Sie sollten jedes Element in einer oder mehreren Ebenen zusammenstellen, um das spätere Kopieren in eine andere Datei zu ermöglichen. Beispielsweise muss die PlayPause-Schaltfläche in die angehaltene Datei, die Disabled-Datei und die Region-Datei kopiert und neu einfärbt werden. Wenn Sie die Art in die Region-Datei verschieben, stellen Sie sicher, dass die von Ihnen kopierte Art keine komplexen Kanteneffekte wie Schatten oder Abschrägungen hat.

Wenn Sie die Erstellung des Bilds abgeschlossen haben, machen Sie alle Ebenen sichtbar, die Sie verwenden möchten, und blenden Sie die anderen Ebenen (z. B. Skizzieren und Rechteck) aus. Verwenden Sie dann den Befehl Kopie speichern, um die Grafik in einer .bmp speichern.

Das folgende Diagramm zeigt das Hintergrundbild.

![Hintergrunddatei](images/ceswmbak.png)

Abschräge, Hüllen und Schatten wurden verwendet, um die visuellen Effekte für die verschiedenen Elemente zu erstellen. Beachten Sie, dass es keine Schaltfläche für das Volumefingerabbild gibt. Sie wird später erstellt.

Es gibt mehrere visuelle Effekte, die Sie auf Bilder anwenden können, um die Erstellung interessanterer Grafiken zu gestalten. Beispielsweise wurde die Schaltfläche PlayPause mit einem einfachen Kreis erstellt. Anschließend wurde eine Abschräge und Einbettung mit Schatteneffekt erstellt. Der durchgeklungene Pfeil war ein einfaches Dreieck, das mit drei Klicks des Stifttools erstellt wurde. Der Pfeil wurde dann mithilfe eines Effekts in ein eingebettes Aussehen eingebettet. Wenn Sie ein Programm wie Antivirus verwenden, müssen Sie nicht zeichnen können. Sie können einfach einfache Bilder erstellen und Effekte anwenden. Sie können auch Eigene Fotos oder gescannte Grafiken importieren.
