# <a name="creating-the-super-file"></a>Erstellen der Super-Datei

Die Super-Datei enthält das deaktivierte Bild für trackbars.

> [!Note]  
> Super Dateien werden in Skins für Windows Media Player 10 Mobile oder höher nicht verwendet, da sich die deaktivierten Images für Such trackbars in der seekthumb-Datei befinden.

 

Die folgende Abbildung stellt die Super-Datei dar.

![Super-Datei](images/ceswmsup.png)

Sie enthält das Bild für eine deaktivierte Volume TrackBar.
