# <a name="creating-the-super-file"></a>Erstellen der Superdatei

Die Super-Datei enthält das deaktivierte Bild für Trackleisten.

> [!Note]  
> Superdateien werden in Skins für Windows Media Player 10 Mobile oder höher nicht verwendet, da sich die deaktivierten Bilder für Such-Trackleisten in der Seekthumb-Datei befinden.

 

Die folgende Abbildung zeigt die Super-Datei.

![Superdatei](images/ceswmsup.png)

Sie enthält das Bild für eine deaktivierte Volume-Trackleiste.
