# <a name="reference-for-type-2-online-stores"></a>Referenz für Typ 2-Onlineshops

> [!Note]  
> In diesem Abschnitt werden die-Funktionen beschrieben, die für die Verwendung durch Online Stores Die Verwendung dieser Funktion außerhalb des Kontexts eines Online Stores wird nicht unterstützt.

 

Windows Media Player 10 oder höher stellt Objekte, Schnittstellen, Eigenschaften und Methoden bereit, die den Typ 2 Online-Speicher unterstützen. In den folgenden Abschnitten werden diese ausführlich beschrieben.



| `Section`                                                                                                        | BESCHREIBUNG                                                                                                                                                                 |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Iwmpabonneptionservice-Schnittstelle](/previous-versions/windows/desktop/api/subscriptionservices/nn-subscriptionservices-iwmpsubscriptionservice)                                               | Stellt Methoden bereit, um Digital Rights Management (DRM) zu erweitern und Hintergrundprozesse zu initiieren.                                                                              |
| [IWMPSubscriptionService2-Schnittstelle](/previous-versions/windows/desktop/api/subscriptionservices/nn-subscriptionservices-iwmpsubscriptionservice2)                                             | Stellt Methoden zum Initiieren von Hintergrundprozessen, zum Bereitstellen von Informationen über die Online Store-Aktivität und zum Bereitstellen eines Zeigers auf die Windows Media Player Core-Schnittstelle bereit. |
| [Iwmpabonneptionservicecallback-Schnittstelle](/previous-versions/windows/desktop/api/subscriptionservices/nn-subscriptionservices-iwmpsubscriptionservicecallback)                               | Definiert eine Methode, mit der Onlinespeicher Windows Media Player Benachrichtigen, wenn ein Hintergrundprozess abgeschlossen ist.                                                              |
| [Wmpnotifyabonptionpluginadressmove](/previous-versions/windows/desktop/api/contentpartner/nf-contentpartner-wmpnotifysubscriptionpluginaddremove)                               | Eine Funktion, mit der Windows Media Player benachrichtigt wird, dass ein Plug-in installiert oder deinstalliert wurde.                                                                            |
| [Download Collection-Objekt](downloadcollection-object.md)                                                     | Stellt Methoden und Eigenschaften zum Verwalten von Auflistungen von Download Elementen bereit.                                                                                                    |
| [Download Item-Objekt](downloaditem-object.md)                                                                 | Stellt Methoden und Eigenschaften für Download Anforderungen bereit.                                                                                                              |
| [Download Manager-Objekt](downloadmanager-object.md)                                                           | Stellt Methoden zum Verwalten von Download Auflistungen bereit.                                                                                                                            |
| [Externes Objekt für den Typ 2-Online Speicher](external-object-for-type-2-online-stores.md)                       | Stellt Eigenschaften, Methoden und ein Ereignis bereit, auf die über Webseiten im Online Store zugegriffen werden kann.                                                                                           |
| [Enumerationen für den Typ 2-Online Speicher](enumerations-for-type-2-online-stores.md)                             | Beschreibt Enumerationen, die von Online Stores verwendet werden.                                                                                                                               |
| [Windows Media Player Bits-Auftrags Konvention](windows-media-player-bits-job-convention.md)                       | Beschreibt eine Konvention für die Verwendung des Background Intelligent Transfer Service (Bits).                                                                                        |
| [Registrierungsschlüssel und-Einträge für einen Typ 2-Online Speicher](registry-keys-and-entries-for-a-type-2-online-store.md) | Beschreibt Registrierungseinträge, die von einem Online Shop in der Registrierung auf dem Computer des Benutzers erstellt werden müssen.                                                                         |
