# <a name="embedding-the-windows-media-player-control"></a>Einbetten des Windows Media Player-Steuerelements

Zusätzlich zum Einbetten des Windows Media Player-Steuerelements in eine Webseite, wie im vorherigen Abschnitt beschrieben, können Sie es in Microsoft Office Dokumente und benutzerdefinierte Programme einbetten, die Sie mit einer von mehreren Programmiersprachen erstellen, einschließlich Microsoft Visual Basic, C++ und C#.

Die folgenden Abschnitte enthalten Informationen zur Verwendung des -Steuerelements in verschiedenen Programmierumgebungen.



| `Section`                                                                                                                                  | BESCHREIBUNG                                                                                             |
|------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [Einbetten des Windows Media Player-Steuerelements in eine Webseite](embedding-the-windows-media-player-control-in-a-web-page.md)                 | In diesem Abschnitt wird die Verwendung des -Steuerelements auf einer Webseite beschrieben.                                                  |
| [Einbetten des Windows Media Player-Steuerelements in ein Office-Dokument](embedding-the-windows-media-player-control-in-an-office-document.md) | In diesem Abschnitt wird die Verwendung des -Steuerelements in einem Microsoft Office-Dokument beschrieben.                                |
| [Einbetten des Windows Media Player-Steuerelements in ein benutzerdefiniertes Programm](embedding-the-windows-media-player-control-in-a-custom-program.md)     | In diesem Abschnitt wird die Verwendung des -Steuerelements in einer Windows-basierten Anwendung beschrieben, die in Visual Basic oder C++ geschrieben ist. |
