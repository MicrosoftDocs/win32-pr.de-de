# <a name="button-location"></a>Schaltflächenspeicherort

Sie müssen den Speicherort für jedes Schaltflächenbild definieren. Dazu geben Sie vier positive ganze Zahlen an, die durch Kommas getrennt sind. Die Zahlen definieren die Koordinaten der linken oberen Ecke, die Breite und Höhe einer Schaltfläche in Pixel.

Wenn Sie Regionsschaltflächen (PushHit, ToggleHit und 2PushHit) verwenden, müssen Sie trotzdem diese vier Zahlen eingeben. Die Werte werden verwendet, um die Speicherorte für die sekundären Images wie Pushed und Disabled zu bestimmen.

> [!Note]  
> Schaltflächentypen sind in Skins für Windows Media Player 10 Mobile oder höher veraltet.
