# <a name="button-location"></a>Schaltflächen Position

Sie müssen den Speicherort für jedes Schaltflächen Bild definieren. Zu diesem Zweck stellen Sie vier positive ganze Zahlen bereit, die durch Kommas getrennt sind. Die Zahlen definieren die Koordinaten der linken oberen Ecke, die Breite und die Höhe einer Schaltfläche in Pixel.

Wenn Sie die Bereichs Schaltflächen (pushhit, "ongglehit" und "2pushhit") verwenden, müssen Sie dennoch diese vier Zahlen eingeben. Die Werte werden verwendet, um die Speicherorte für die sekundären Images zu ermitteln, z. b. per pushübertragung

> [!Note]  
> Schaltflächen Typen sind in Skins für Windows Media Player 10 Mobile oder höher veraltet.
