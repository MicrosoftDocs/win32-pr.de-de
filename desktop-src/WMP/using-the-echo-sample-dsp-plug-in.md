# <a name="using-the-echo-sample-dsp-plug-in"></a>Verwenden des DSP-Plug-Ins "Echo Sample"

Wenn Sie einen fehlerfreien Build Ihres Windows Media Player Echo-DSP-Plug-Ins erstellen, können Sie ihn in Windows Media Player verwenden, um den Effekt zu hören. Wenn Sie Windows Media Player starten, sollte das Plug-In standardmäßig aktiviert sein. Wenn Sie Audio oder Video mit Audio wiedergeben, sollten Sie ein Ein-Sekunden-Echo im Sound hören, der auf derselben Lautstärke wie der ursprüngliche Sound wiedergegeben wird.

Sie können mit unterschiedlichen Verzögerungszeiten und Effektebenen experimentieren, indem Sie die Werte auf der Eigenschaftenseite des Echo-Plug-Ins ändern. Testen Sie den Effekt auf verschiedene Arten von Quellmaterial, z. B. Musik oder Sprache. Eine Verzögerung von 200 ms mit einer Effektstufe von 30 führt zu einem eng verbreiterten Echo, das sich wie der Effekt anhört, der bei frühen Rock- und Rollaufzeichnungen verwendet wird. Effekte, die größer als 50 sind, führen dazu, dass das ursprüngliche Signal weniger volumenmäßig ist als das Echo, das als Vor-Echo-Effekt bezeichnet wird.
