# <a name="using-the-echo-sample-dsp-plug-in"></a>Verwenden des Echo Sample DSP-Plug-ins

Wenn Sie einen fehlerfreien Build Ihres Windows Media Player Echo DSP-Plug-ins erstellen, können Sie es in Windows Media Player verwenden, um die Auswirkung zu hören. Wenn Sie Windows Media Player starten, sollte das Plug-in standardmäßig aktiviert werden. Wenn Sie Audio oder Video mit Audiodaten wiedergeben, sollten Sie im Sound, der auf dem gleichen Volume wie der ursprüngliche Sound abgespielt wird, eine einsekunde-Echo hören.

Sie können mit unterschiedlichen Verzögerungszeiten und Effekten experimentieren, indem Sie die Werte auf der Eigenschaften Seite des Echo-Plug-ins ändern. Probieren Sie die Auswirkungen auf verschiedene Arten von Quellmaterial aus, z. b. Musik oder Sprache. Eine Verzögerungszeit von 200 MS mit der Auswirkung-Stufe 30 führt zu einem genau getrennten Echo, das wie die Auswirkung auf frühe Rock-und Roll-Aufzeichnungen klingt. Effekte, die größer als 50 sind, führen dazu, dass das ursprüngliche Signal kleiner als das Echo ist, was als präecho Effekt bezeichnet wird.
