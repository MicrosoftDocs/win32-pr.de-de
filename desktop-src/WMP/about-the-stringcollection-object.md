# <a name="about-the-stringcollection-object"></a>Informationen zum StringCollection-Objekt

Das **StringCollection** -Objekt regelt Zeichen folgen Auflistungen, die verwendet werden, um Funktionen zur Zeichen folgen Behandlung für das **mediacollection** -Objekt bereitzustellen. Das **StringCollection** -Objekt kann über die **getAttributeStringCollection** -Methode des **mediacollection** -Objekts abgerufen werden. Damit können Sie mit Zeichen folgen Auflistungen arbeiten, die die Werte verschiedener Medien Element Attribute darstellen.
