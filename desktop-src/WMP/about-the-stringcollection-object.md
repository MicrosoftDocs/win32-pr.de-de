# <a name="about-the-stringcollection-object"></a>Informationen zum StringCollection-Objekt

Das **StringCollection-Objekt** steuert Zeichenfolgenauflistungen, die zum Bereitstellen von Funktionen zur Behandlung von Zeichenfolgen für das **MediaCollection-Objekt** verwendet werden. Das **StringCollection-Objekt** kann über die **getAttributeStringCollection-Methode** des **MediaCollection-Objekts** abgerufen werden. Sie können dies verwenden, um mit Zeichenfolgenauflistungen zu arbeiten, die die Werte verschiedener Medienelementattribute darstellen.
