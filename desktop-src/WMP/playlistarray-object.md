# <a name="playlistarray-object"></a>Playlistarray-Objekt

Das **playlistarray** -Objekt bietet eine Möglichkeit für den Zugriff auf Wiedergabelisten nach Indexnummer.

Das **playlistarray** -Objekt unterstützt die folgende Eigenschaft.



| Eigenschaft                         | BESCHREIBUNG                                              |
|----------------------------------|----------------------------------------------------------|
| [count](playlistarray-count.md) | Ruft die Anzahl der Wiedergabelisten im Wiedergabelisten Array ab. |



 

Das **playlistarray** -Objekt unterstützt die folgende Methode.



| Methode                         | BESCHREIBUNG                                                              |
|--------------------------------|--------------------------------------------------------------------------|
| [item](playlistarray-item.md) | Ruft das [Wiedergabe](playlist-object.md) Listen Objekt am angegebenen Index ab. |



 

Der Zugriff auf das **playlistarray** -Objekt erfolgt über die folgenden Methoden.



| Object                                              | Methode                                                                                 |
|-----------------------------------------------------|----------------------------------------------------------------------------------------|
| [Playlistcollection](playlistcollection-object.md) | [GetAll](playlistcollection-getall.md), [getByName](playlistcollection-getbyname.md) |



 

Dient zur Veranschaulichung des *Players*. *playlistcollection*. **GetAll** wird in den Abschnitten der Verweis Syntax verwendet.
