# <a name="modifying-the-echo-sample-property-page"></a>Ändern der Eigenschaften Seite Echo Sample

Die Standard Implementierung der Eigenschaften Seite, die vom Windows Media Player Plug-in-Assistenten DSP-Plug-in-Beispiel bereitgestellt wird, enthält ein einzelnes Bearbeitungsfeld-Steuerelement, das den Skalierungsfaktor des Benutzers empfängt. Sie müssen die Eigenschaften Seite ändern, damit die beiden vom Echo-Beispiel verwendeten Eigenschaftswerte empfangen werden.

Eine Übersicht über die Eigenschaften Seiten des DSP-Plug-Ins finden Sie unter [Implementieren eines audiodsp-Plug-ins](implementing-an-audio-dsp-plug-in.md).

In den folgenden Abschnitten wird Schritt für Schritt erläutert, wie Sie die Beispiel Eigenschaften Seite ändern:

-   [Hinzufügen und Ändern der Ereignisse](adding-and-modifying-the-events.md)
-   [So speichert das Echo Beispiel Daten](how-the-echo-sample-persists-data.md)
-   [Implementieren von cechoproppage:: OnInitDialog](implementing-cechoproppage--oninitdialog.md)
-   [Implementieren von cechoproppage:: apply](implementing-cechoproppage--apply.md)
