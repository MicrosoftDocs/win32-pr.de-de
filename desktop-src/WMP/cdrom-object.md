# <a name="cdrom-object"></a>Cdrom-Objekt

Das **CDROM** -Objekt bietet eine Möglichkeit für den Zugriff auf eine CD oder DVD auf dem Laufwerk.

Das **CDROM** -Objekt unterstützt die folgenden Eigenschaften.



| Eigenschaft                                   | BESCHREIBUNG                                                                                                                                             |
|--------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [drivespecifier](cdrom-drivespecifier.md) | Ruft den CD-oder DVD-Laufwerk Buchstaben ab.                                                                                                                   |
| [Abspielen](cdrom-playlist.md)             | Ruft ein [Wiedergabe](playlist-object.md) Listen Objekt ab, das die Spuren auf der CD darstellt, die sich derzeit auf dem CD-Laufwerk oder auf der Stamm Ebene der Titel Einträge für DVD befinden. |



 

Das **CDROM** -Objekt unterstützt die folgende Methode.



| Methode                   | BESCHREIBUNG                          |
|--------------------------|--------------------------------------|
| [Auswerfen](cdrom-eject.md) | Fügt die CD oder DVD vom Laufwerk ein. |



 

Der Zugriff auf das **CDROM** -Objekt erfolgt über die folgende Methode.



| Object                                        | Methode                           |
|-----------------------------------------------|----------------------------------|
| [Cdromcollection](cdromcollection-object.md) | [item](cdromcollection-item.md) |



 

Zur Veranschaulichung wird Player. cdromcollection. Item (*Index*) in den Abschnitten der Verweis Syntax verwendet.
