# <a name="sample-bitmap-section"></a><span data-ttu-id="d089f-108">Beispiel für einen bitmapabschnitt</span><span class="sxs-lookup"><span data-stu-id="d089f-108">Sample Bitmap Section</span></span>

```C++
[ Bitmaps ]

//  <Type>      <File name>     <X,Y>
//  ------      -----------     -----
    Background  Background.bmp  0,0
    Disabled    Disabled.bmp    0,0
    Pushed      Pushed.bmp      0,0
    Region      Region.bmp      0,0
    Super       Super.bmp       0,0
    

```
