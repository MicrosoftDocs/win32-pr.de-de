# <a name="sample-bitmap-section"></a>Abschnitt "Beispielbitmap"

Die folgenden Zeilen zeigen einen typischen Bitmaps-Abschnitt einer Skindefinitionsdatei:


```C++
[ Bitmaps ]

//  <Type>      <File name>     <X,Y>
//  ------      -----------     -----
    Background  Background.bmp  0,0
    Disabled    Disabled.bmp    0,0
    Pushed      Pushed.bmp      0,0
    Region      Region.bmp      0,0
    Super       Super.bmp       0,0
    

```



Dadurch werden fünf Bitmaps definiert, die zum Erstellen eines Hintergrundbilds verwendet werden, Bilder für deaktivierte und pushed-Schaltflächen, ein Region-Bild für Diess-Schaltflächen und ein Super-Bild für Trackbars.

> [!Note]  
> Die Bitmaps Region und Super sind in Skins für Windows Media Player 10 Mobile oder höher veraltet.
