# <a name="button-functions"></a>Schaltflächenfunktionen

Sie sollten dem Benutzer Schaltflächen zur Verfügung stellen, um die Auswahl und Wiedergabe von Medienelementen zu steuern. Mithilfe der entsprechenden Schaltflächen können Sie das Medienelement wieder abspielen, anhalten oder beenden, zu verschiedenen Elementen in der Wiedergabeliste wechseln, den Sound stummschalten oder Informationen über das Medienelement bereitstellen, das abspielt.

Sie müssen nicht alle Schaltflächenfunktionen in Ihrer Skin bereitstellen, aber die beste Benutzererfahrung umfasst wahrscheinlich mindestens die Schaltflächenfunktionen PlayPause, Stop oder PlayPauseStop, Prev und Next.

Die folgenden Schaltflächenfunktionen sind verfügbar:

-   Abspielen
-   Anhalten
-   Beenden
-   Prev (Vorheriger)
-   Nächste
-   Shuffle
-   Info
-   RepeatPlaylist
-   Mute
-   PlayPause
-   FullScreen
-   Windowsmedia\*
-   PlayPauseStop\*
-   VolumeUp\*
-   VolumeDown\*

\*Verfügbar für Windows Media Player 10 Mobile oder höher.

Weitere Informationen zu Schaltflächenfunktionen finden Sie im Abschnitt [Schaltflächen](buttons.md) der Skin-Referenz.
