# <a name="button-functions"></a>Schaltflächen Funktionen

Sie sollten dem Benutzer Schaltflächen zum Steuern der Auswahl und Wiedergabe von Medien Elementen bereitstellen. Mithilfe der entsprechenden Schaltflächen können Sie das Medien Element wiedergeben, anhalten oder anhalten, zu unterschiedlichen Elementen in der Wiedergabeliste wechseln, den Sound stumm schalten oder Informationen über das wiedergegebene Medien Element angeben.

Sie müssen nicht alle Schaltflächen Funktionen in der Skin bereitstellen, aber die beste Benutzer Leistung umfasst wahrscheinlich mindestens die Funktionen "PLAYPAUSE", "anhalten", "playpausestoppt", "Prev" und "weiter".

Die folgenden Schaltflächen Funktionen sind verfügbar:

-   Abspielen
-   Anhalten
-   Stop
-   Prev (Vorheriger)
-   Nächste
-   Shuffle
-   Info
-   Repeatwiedergabe Liste
-   Mute
-   PLAYPAUSE
-   FullScreen
-   Windows Media\*
-   Playpauseelenhalte\*
-   Volumeup\*
-   Volumedown\*

\*Verfügbar für Windows Media Player 10 Mobile oder höher.

Weitere Informationen zu Schaltflächen Funktionen finden Sie im Abschnitt [Buttons](buttons.md) der Skin-Referenz.
