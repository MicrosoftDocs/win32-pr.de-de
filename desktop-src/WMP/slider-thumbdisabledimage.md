# <a name="sliderthumbdisabledimage"></a>Slider. thumbdisabledimage

Das **thumbdisabledimage** -Attribut gibt das Thumb-Bild des Schieberegler-Steuer Elements an oder ruft es ab, das verwendet wird, wenn das Steuerelement deaktiviert wird.

``` syntax
        elementID.thumbDisabledImage
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese- **/schreibzeichenfolge** , die den Namen einer Bilddatei enthält.

## <a name="remarks"></a>Bemerkungen

" **Thumbdisabledimage** " ist optional. Wenn Sie nicht angegeben ist, wird stattdessen " **thumbimage** " verwendet.

Die unterstützten Formate sind BMP, JPG, PNG und GIF (keine animierten GIFs).
