# <a name="ui-plug-in-overview"></a>Übersicht über das Benutzeroberflächen-Plug-In

Benutzeroberflächen-Plug-Ins können über die Registerkarte Plug-Ins des Dialogfelds Optionen in der Windows Media Player.

Es gibt fünf Arten von Benutzeroberflächen-Plug-Ins, die von Windows Media Player. Drei dieser Typen entsprechen verschiedenen Bereichen im Bereich **Jetzt** wieder verwendet des vollständigen Modus des Players. Die anderen beiden Typen sind für Plug-Ins, die in einem separaten Fenster angezeigt werden, und für Plug-Ins ohne Anzeige (außer vielleicht einer Eigenschaftenseite).

Benutzeroberflächen-Plug-Ins werden entladen, wenn sie geschlossen oder deaktiviert werden. Benutzeroberflächen-Plug-Ins,  die im Bereich Jetzt wieder verwendet angezeigt werden, werden ebenfalls entladen, wenn der Benutzer ein anderes Plug-In desselben Typs auswählt oder aus dem Bereich Jetzt wieder verwendet **wechselt.** Wenn der Benutzer in den Skinmodus wechselt, bleiben alle Plug-Ins geladen, werden aber nicht angezeigt. Wenn separate Fenster- oder Hintergrund-Plug-Ins ausgeführt werden, Windows Media Player, werden sie automatisch neu geladen, wenn der Player das nächste Mal gestartet wird.
