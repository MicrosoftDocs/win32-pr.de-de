# <a name="ui-plug-in-overview"></a>Übersicht über das UI-Plug-in

UI-Plug-Ins können auf der Registerkarte Plug-ins des Dialog Felds Optionen in Windows Media Player installiert, deinstalliert und konfiguriert werden.

Es gibt fünf Arten von UI-Plug-ins, die von Windows Media Player unterstützt werden. Drei dieser Typen entsprechen unterschiedlichen Bereichen des Fensters " **jetzt** wiedergegeben" des vollständigen Modus des Players. Die anderen beiden Typen sind für Plug-ins, die in einem separaten Fenster angezeigt werden, und für Plug-ins, die über keine Anzeige verfügen (außer vielleicht eine Eigenschaften Seite).

UI-Plug-ins werden entladen, wenn Sie geschlossen oder deaktiviert werden. Benutzeroberflächen-Plug-ins, die im Bereich " **jetzt Wiedergabe** " angezeigt werden, werden auch entladen, wenn der Benutzer ein anderes Plug-in desselben Typs auswählt oder aus dem Bereich " **jetzt Wiedergabe** " wechselt. Wenn der Benutzer in den Skin-Modus wechselt, bleiben alle Plug-Ins geladen, werden jedoch nicht angezeigt. Wenn beim nächsten Start von Windows Media Player separate Fenster-oder Hintergrund-Plug-Ins ausgeführt werden, werden Sie automatisch erneut geladen.
