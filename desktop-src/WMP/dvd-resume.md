# <a name="dvdresume-method"></a>DVD. Resume-Methode

Die **Resume** -Methode kehrt in den Wiedergabemodus aus dem Menü Modus an derselben Titel Position zurück, als wenn das Menü aufgerufen wurde.

## <a name="syntax"></a>Syntax


```JScript
DVD.resume()
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.

## <a name="remarks"></a>Bemerkungen

**Windows Media Player 10 Mobile:** Diese Methode ist immer erfolgreich, führt jedoch nicht den beabsichtigten Vorgang aus.
