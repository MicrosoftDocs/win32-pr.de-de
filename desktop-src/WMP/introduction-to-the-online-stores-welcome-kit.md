# <a name="introduction-to-the-online-stores-welcome-kit"></a>Einführung in das Welcome Kit für Onlineshops

Das Welcome Kit für Onlineshops beschreibt den Prozess, bei dem ein Online-Digital Media Store für Windows Media Player ins Boot geschaltet wird. Windows Media Player unterstützt zwei Arten von Onlineshops: Typ 1 und Typ 2. Informationen zum Unterschied zwischen Denktypen 1 und Typ 2 finden Sie unter [Windows Media Player Onlineshops.](windows-media-player-online-stores.md)

Der in den folgenden Abschnitten beschriebene Onboardingprozess gilt sowohl für Filialen vom Typ 1 als auch für den Typ 2, mit Ausnahme einiger Unterschiede, die notiert sind.

## <a name="eligibility"></a>Berechtigung

Um berechtigt zu sein, einen Online-Musikspeicher in Windows Media Player integriert zu haben, muss Ihr Unternehmen die Onboarding- und Wartungsanforderungen erfüllen, die im Onlineshopvertrag angegeben sind. Die Arten von Anforderungen in der Vereinbarung umfassen Folgendes, sind aber nicht darauf beschränkt.

-   Ihr Unternehmen muss über die Rechte zum Verteilen von Onlineinhalten über Windows Media Player verfügen.
-   Ihr Unternehmen muss zustimmen, benutzern von Windows Media Player eine bestimmte Anzahl von Spuren zur Verfügung zu stellen.
-   Ihr Unternehmen muss ein bestimmtes Maß an unterbrechungsfreiem Dienst bereitstellen.

Einige spezifische Anforderungen aus der Vereinbarung sind unter [Anforderungen für Online Musik Stores](requirements-for-online-music-stores.md)hervorgehoben.

## <a name="contacting-microsoft"></a>Kontaktieren von Microsoft

Wenn Sie Fragen zu Onlineshops und zum Onboardingprozess haben, wenden Sie sich unter an das Windows Media Player Services Virtual mpsvctm@microsoft.com Team.
