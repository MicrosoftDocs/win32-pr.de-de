# <a name="friendlyname-element"></a>FriendlyName-Element

> [!Note]  
> In diesem Abschnitt werden funktionen beschrieben, die für die Verwendung durch Onlineshops entwickelt wurden. Die Verwendung dieser Funktionalität außerhalb des Kontexts eines Onlineshops wird nicht unterstützt.

 

Das **FriendlyName-Element** gibt die Textzeichenfolge an, die dem Benutzer angezeigt Windows Media Player, um den Onlineshop zu identifizieren.

``` syntax
<FriendlyName>
    text string
</FriendlyName>
```

## <a name="attributes"></a>Attribute

Dieses Element weist keine Attribute auf.

## <a name="parentchild-elements"></a>Übergeordnete/untergeordnete Elemente



| Hierarchy       | Element         |
|-----------------|-----------------|
| Übergeordnete Elemente | **ServiceInfo** |
| Untergeordnete Elemente  | Keine            |
