# <a name="playermediacollectionchange-event"></a>Player. mediacollectionchange-Ereignis

Das **mediacollectionchange** -Ereignis tritt auf, wenn die Mediensammlung geändert wird.

## <a name="syntax"></a>Syntax


```JScript
Player.MediaCollectionChange()
```



## <a name="parameters"></a>Parameter

Dieses Ereignis weist keine Parameter auf.

## <a name="return-value"></a>Rückgabewert

Dieses Ereignis gibt keinen Wert zurück.

## <a name="remarks"></a>Bemerkungen

**Windows Media Player 10 Mobile:** Dieses Ereignis wird nicht unterstützt.

|--------------------|------------------------------------------------------------------------------------|
