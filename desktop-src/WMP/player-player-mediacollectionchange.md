# <a name="playermediacollectionchange-event"></a>Player.MediaCollectionChange-Ereignis

Das **MediaCollectionChange-Ereignis** tritt auf, wenn sich die Mediensammlung ändert.

## <a name="syntax"></a>Syntax


```JScript
Player.MediaCollectionChange()
```



## <a name="parameters"></a>Parameter

Dieses Ereignis verfügt über keine Parameter.

## <a name="return-value"></a>Rückgabewert

Dieses Ereignis gibt keinen Wert zurück.

## <a name="remarks"></a>Hinweise

**Windows Media Player 10 Mobile:** Dieses Ereignis wird nicht unterstützt.
