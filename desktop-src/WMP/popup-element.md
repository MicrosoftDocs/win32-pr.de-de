# <a name="popup-element"></a>Popup-Element

Das **Popup** -Element bietet Benutzern die Möglichkeit, Elemente aus einer Liste auszuwählen. Diese Elemente können mithilfe von **Item** -Elementen als untergeordnete Elemente des **Popup** -Elements angegeben werden. Das **Popup** Element ist mit dem **ListBox** -Element identisch, mit dem Unterschied, dass der Standardwert des **Popup** Attributs true ist. Wenn **popUp** auf true festgelegt ist, wird das Steuerelement nur angezeigt, wenn die **Show** -Methode aufgerufen wird und seine Breite auf die Breite des Inhalts festgelegt ist (das Ambient **Width** -Attribut wird ignoriert). Das-Steuerelement wird automatisch ausgeblendet, wenn der Benutzer ein Element aus der Liste auswählt oder wenn die **verwerfen** -Methode aufgerufen wird.

> [!Note]  
> Für dieses Element ist Windows Media Player für Windows XP oder höher erforderlich.
