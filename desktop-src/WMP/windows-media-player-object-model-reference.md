# <a name="windows-media-player-object-model-reference"></a>Windows Media Player Objektmodellreferenz

Die Abschnitte Windows Media Player Objektmodellreferenz enthalten ausführliche Informationen zum Windows Media Player ActiveX Steuerelementobjektmodell. Das Objektmodell stellt Schnittstellen bereit, mit denen Entwickler Webseiten, C++-Programmen und .NET-Anwendungen Windows Media Player Funktionalität hinzufügen können.

Die Objektmodellreferenz für die Skripterstellung wird in einem Stil für die Verwendung mit Skriptsprachen wie Microsoft JScript dargestellt und dokumentiert das Objektmodell in Bezug auf Objekte mit den zugehörigen Eigenschaften, Methoden und Ereignissen.

Die Objektmodellreferenz für C++ enthält Dokumentation zu COM-Schnittstellen, die C++-Programmierer für die Kommunikation mit dem Windows Media Player ActiveX-Steuerelement oder dem Windows Media Player Mobile ActiveX-Steuerelement verwenden können. Die meisten dieser Schnittstellen werden durch Windows Media Player-Steuerelement implementiert und von der C++-Anwendung aufgerufen. Einige der Schnittstellen (z. B. **IWMPEvents** und **IWMPRemoteMediaServices)** werden jedoch von der C++-Anwendung implementiert und vom Windows Media Player-Steuerelement aufgerufen.

Die Objektmodellreferenz für Visual Basic .NET und C# enthält Dokumentationen zu den Eigenschaften, Methoden und Ereignissen des **AxWindowsMediaPlayer-Objekts** sowie für Schnittstellen zum Windows Media Player ActiveX-Steuerelement, die über COM-Interoperabilität in einer projektmappe verfügbar sind, die in Visual Studio .NET 2002 oder höher codiert ist.

Die Windows Media Player Objektmodellreferenz enthält die folgenden Abschnitte.
