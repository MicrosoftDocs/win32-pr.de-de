# <a name="iwmpsettingsenableerrordialogs-property"></a>Iwmpsettings:: enableerrordialogs (Eigenschaft)

Mit der **enableerrordialogs** -Eigenschaft wird ein Wert abgerufen oder festgelegt, der angibt, ob automatisch Fehler Dialogfelder angezeigt werden.

## <a name="syntax"></a>Syntax


```CSharp
public System.Boolean enableErrorDialogs {get; set;}
```


```VB

Public Property enableErrorDialogs As System.Boolean
```





## <a name="property-value"></a>Eigenschaftswert

Ein **System. Boolean** -Wert, der angibt, ob automatisch Fehler Dialogfelder angezeigt werden. Der Standardwert ist **true**.
