# <a name="about-custom-visualizations"></a>Informationen zu benutzerdefinierten Visualisierungen

Windows Media Player bietet eine Schnittstelle zum Erstellen von benutzerdefinierten Visualisierungen. Dieser Abschnitt bietet eine Übersicht über die Architektur von Visualisierungen, Informationen zur Beziehung zwischen den Fenstern und Windows Media Player sowie technische Details, die Sie benötigen, um benutzerdefinierte Visualisierungen zu erstellen.

Beachten Sie, dass Sie an mehreren Stellen im Code das Wort "Effect" sehen. Visualisierungen sind Effekte.

Benutzerdefinierte Visualisierungen werden in den folgenden Themen ausführlicher beschrieben.



| Thema                                                              | BESCHREIBUNG                                                                                |
|--------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [Benutzer Übersicht](user-overview.md)                                 | Beschreibt Visualisierungen aus Benutzer Sicht.                                          |
| [Entwickler Übersicht](developer-overview.md)                       | Beschreibt Visualisierungen aus der Perspektive eines Entwicklers.                                     |
| [Aufbau einer Visualisierung](building-a-visualization.md)           | Beschreibt, wie ein Visualisierungsprojekt erstellt wird.                                           |
| [Implementieren des Codes](implementing-your-code.md)               | Beschreibt, wie der Beispielcode geändert wird, der vom Assistenten für das Visualisierungs-Plug-in generiert wird.     |
| [Installieren von Visualisierungen](installing-visualizations.md)         | Hier wird beschrieben, wie eine Visualisierung auf dem Computer eines Benutzers installiert wird.                             |
| [Übermitteln der Visualisierung](submitting-your-visualization.md) | Die Verteilung der abgeschlossenen Visualisierung auf der wmplugins-Website wird nicht mehr unterstützt. |
