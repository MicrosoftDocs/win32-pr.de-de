# <a name="status-location"></a>Status Speicherort

Sie müssen den Speicherort der Statusanzeige definieren, die Sie verwenden möchten. Sie müssen vier positive ganze Zahlen, getrennt durch Kommas, bereitstellen, die die linke, obere, Breite und Höhe der Statusanzeige in Pixel relativ zum Hintergrundbild definieren.


```C++
45,193,120,13 

```
