# <a name="cameramanufacturer-attribute"></a>CameraManufacturer-Attribut

Das **CameraManufacturer-Attribut** ist eine Zeichenfolge, die den Hersteller der Kamera darstellt.

## <a name="applies-to"></a>Gilt für

-   Fotoelemente

## <a name="remarks"></a>Hinweise

Um zu bestimmen, ob Sie den Wert dieses Attributs ändern können, verwenden Sie die [Media.isReadOnlyItem-Methode.](media-isreadonlyitem.md)

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------|
| Version<br/> | Windows Media Player 11<br/> |
