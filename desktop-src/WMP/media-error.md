# <a name="mediaerror"></a>Media. Error

Die **Error** -Eigenschaft ruft ein **ErrorItem** -Objekt ab, wenn das Medien Element einen Fehlerzustand aufweist.

## <a name="syntax"></a>Syntax

*Player*. *currentMedia*. **Fehler**

## <a name="possible-values"></a>Mögliche Werte

Diese Eigenschaft ist ein Schreib geschütztes **ErrorItem** -Objekt.

## <a name="remarks"></a>Bemerkungen

Wenn das Medien Element nicht wiedergegeben werden kann, ruft diese Eigenschaft ein **ErrorItem** -Objekt ab, das Informationen über das aufgetretene Problem enthält.
