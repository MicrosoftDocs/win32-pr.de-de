# <a name="text-editors"></a>Text-Editoren

Sie müssen einen Text-Editor verwenden, der keine Formatierungszeichen enthält. Ein einfacher Text-Editor wie Editor wird empfohlen, obwohl Entwicklungsumgebungen wie Microsoft Visual Studio erweiterte Features bieten. Sie sollten keinen Editor wie Microsoft Word verwenden, da das Word-Standarddateiformat zusätzliche Formatierungscodes speichert, die nicht verwendet werden können. Wenn Sie einen Textprozessor wie Word verwenden, müssen Sie daran denken, Ihre Arbeit jedes Mal als unformatierten Text zu speichern, wenn Sie Ihre Datei speichern.

Sie sollten den Zeilenumbruch wahrscheinlich deaktivieren, wenn ihr Text-Editor dies zulässt, und sie sollten die Möglichkeit aktivieren, alle Zeichen anzuzeigen, um sicherzustellen, dass Sie nicht versehentlich Registerkarten oder andere Steuerzeichen hinzufügen. Denken Sie daran, dass das Einschließen von Tabstoppzeichen in ihre Skindefinitionsdatei dazu führt, dass die Skin fehlschlägt.
