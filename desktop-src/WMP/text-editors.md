# <a name="text-editors"></a>Text-Editoren

Sie müssen einen Text-Editor verwenden, der keine Formatierungszeichen enthält. Ein einfacher Text-Editor wie z. b. Notepad wird empfohlen, aber Entwicklungsumgebungen wie Microsoft Visual Studio bieten erweiterte Features. Sie sollten keinen Editor wie z. b. Microsoft Word verwenden, da das Standardformat für Word-Dateien zusätzliche Formatierungscodes speichert, die nicht verwendbar sind. Wenn Sie ein Textverarbeitungs Tool wie Word verwenden, müssen Sie daran denken, die Arbeit beim Speichern der Datei als unformatierten Text zu speichern.

Sie sollten den Zeilenumbruch wahrscheinlich deaktivieren, wenn der Text-Editor dies zulässt, und Sie können die Fähigkeit zum Anzeigen aller Zeichen aktivieren, um sicherzustellen, dass Sie nicht versehentlich irgendwelche Registerkarten oder andere Steuerzeichen hinzufügen. Beachten Sie, dass das Einschließen von Tabstopp Zeichen in der Skin-Definitionsdatei zu einem Fehler bei der Skin führt.
