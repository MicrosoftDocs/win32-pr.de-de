# <a name="buttonelementdowntooltip"></a>ButtonElement. downtooltip

Das **downtooltip** -Attribut gibt den QuickInfo-Text an oder ruft ihn ab, der angezeigt wird, wenn sich der Mauszeiger über dem **Button-Element** befindet und sich das **Button Element** im Zustand "gedrückt" befindet.

``` syntax
        elementID.downToolTip
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist eine Lese-/schreibzeichenfolge mit dem Standardwert "" (leere Zeichenfolge) mit einer maximalen Länge von 1024 Zeichen.

## <a name="remarks"></a>Bemerkungen

Wenn dieses Attribut auf "" (leere Zeichenfolge) festgelegt ist, wird keine QuickInfo angezeigt.
