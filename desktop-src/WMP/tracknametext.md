# <a name="tracknametext"></a>Tracknametext

Dies ist ein vordefiniertes **Text** Element mit den folgenden Standardwerten.

``` syntax
value="wmpprop:player.currentMedia.name"
tabstop="true"
```

## <a name="remarks"></a>Bemerkungen

Dadurch wird ein **Text** Element erstellt, in dem der Name des aktuellen Mediums angezeigt wird. Alle Eigenschaften dieses **Text** Elements können überschrieben werden, indem Sie explizit angegeben werden.
