# <a name="tracknametext"></a>TRACKNAMETEXT

Dies ist ein vordefiniertes **TEXT-Element** mit den folgenden Standardwerten.

``` syntax
value="wmpprop:player.currentMedia.name"
tabstop="true"
```

## <a name="remarks"></a>Hinweise

Dadurch wird ein **TEXT-Element** erstellt, das den Namen des aktuellen Mediums anzeigt. Alle Eigenschaften dieses **TEXT-Elements** können überschrieben werden, indem sie explizit angegeben werden.
