# <a name="onfocus"></a>onfocus

Der **onfocus** -Ereignishandler behandelt ein Ereignis, das auftritt, wenn das Element den Tastaturfokus erhält.

``` syntax
onfocus
```

## <a name="remarks"></a>Bemerkungen

Der **onfocus** -Ereignishandler kann nicht auf das **Effects** -Element oder das **Popup** -Element angewendet werden.

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|-----------------------------------------------------|
| Version<br/> | Windows Media Player, Version 70 oder höher<br/> |
