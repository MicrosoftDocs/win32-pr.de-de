# <a name="building-a-dsp-plug-in"></a>Aufbau eines DSP-Plug-ins

Sie können ein DSP-Plug-in mithilfe von Visual Studio und dem Assistenten für Windows-Media Player-Plug-in erstellen. Informationen zum Einrichten des Assistenten und zum Einrichten eines Plug-Ins finden Sie unter Einrichten [eines Plug-ins](building-a-plug-in.md).

Wenn Sie den Assistenten ausführen, können Sie verschiedene Arten von Plug-ins auswählen. Wählen Sie **DSP-Plug-in aus**.

In den folgenden Themen werden die Features der Beispiel-Plug-Ins für Audiodaten und Video-DSP beschrieben, die Sie mithilfe des Assistenten erstellen können.

-   [Informationen zum Beispiel-audiodsp-Plug-in](about-the-sample-audio-dsp-plug-in.md)
-   [Informationen zum Beispiel-Video-DSP-Plug-in](about-the-sample-video-dsp-plug-in.md)
