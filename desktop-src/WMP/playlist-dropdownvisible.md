# <a name="playlistdropdownvisible"></a>Wiedergabeliste. dropdownvisible

Mit dem **dropdownvisible** -Attribut wird ein Wert angegeben oder abgerufen, der angibt, ob die Wiedergabelisten-Dropdown Liste sichtbar ist.

``` syntax
        elementID.dropDownVisible
```

## <a name="possible-values"></a>Mögliche Werte

Dieses Attribut ist ein **boolescher** Wert mit Lese-/Schreibzugriff.



| Wert | BESCHREIBUNG                             |
|-------|-----------------------------------------|
| true  | Standard. Die Dropdown Liste ist sichtbar. |
| false | Die Dropdown Liste ist nicht sichtbar.      |



 

## <a name="requirements"></a>Anforderungen



| Anforderung | Wert |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player, Version 7,0 oder höher<br/> |
