# <a name="managing-playlists"></a>Verwalten von Wiedergabelisten

Ein **Wiedergabe** Listen Objekt stellt einen Satz von Medien Elementen dar. Verwenden Sie die Eigenschaften und Methoden des-Objekts, um die darin enthaltenen Medienelemente zu bearbeiten und um Metadaten über die Wiedergabeliste abzurufen. Zum Verwalten von Wiedergabelisten können Sie das **playlistcollection** -Objekt und das **mediacollection** -Objekt verwenden.

In den folgenden Themen wird beschrieben, wie Wiedergabelisten mit dem Objektmodell für Windows Media Player-Steuerelement verwaltet werden:

-   [Wiedergabelisten Attribute](playlist-attributes.md)
-   [Wiedergabelisten und Medienelemente](playlists-and-media-items.md)
-   [Wiedergabelisten und das playlistcollection-Objekt](playlists-and-the-playlistcollection-object.md)
-   [Statische und automatische Wiedergabelisten](static-and-auto-playlists.md)
-   [Wiedergabelisten und das mediacollection-Objekt](playlists-and-the-mediacollection-object.md)
