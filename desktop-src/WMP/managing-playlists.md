# <a name="managing-playlists"></a>Verwalten von Wiedergabelisten

Ein **Wiedergabelistenobjekt** stellt einen Satz von Medienelementen dar. Sie verwenden Eigenschaften und Methoden des -Objekts, um die darin enthaltenen Medienelemente zu bearbeiten und Metadaten zur Wiedergabeliste abzurufen. Um Wiedergabelisten selbst zu verwalten, verwenden Sie das **PlaylistCollection-Objekt** und das **MediaCollection-Objekt.**

In den folgenden Themen wird beschrieben, wie Wiedergabelisten mithilfe des Windows Media Player Steuerelementobjektmodells verwaltet werden:

-   [Wiedergabelistenattribute](playlist-attributes.md)
-   [Wiedergabelisten und Medienelemente](playlists-and-media-items.md)
-   [Wiedergabelisten und das PlaylistCollection-Objekt](playlists-and-the-playlistcollection-object.md)
-   [Statische und automatische Wiedergabelisten](static-and-auto-playlists.md)
-   [Wiedergabelisten und das MediaCollection-Objekt](playlists-and-the-mediacollection-object.md)
