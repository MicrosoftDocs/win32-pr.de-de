# <a name="iwmpstringcollection2-vb-and-c-interface"></a>IWMPStringCollection2-Schnittstelle (VB und c#)

Stellt Methoden bereit, die die **iwmpstringcollection** -Schnittstelle ergänzen.

## <a name="members"></a>Member

Die IWMPStringCollection2-Schnittstelle **(VB und c#)** verfügt über diese Typen von Membern:

-   [Methoden](#methods)

### <a name="methods"></a>Methoden

Die IWMPStringCollection2-Schnittstelle **(VB und c#)** verfügt über diese Methoden.



| Methode                                                                                                                 | BESCHREIBUNG                                                                                                                           |
|:-----------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------|
| [**getattributecountrytbytype**](wmplibiwmpstringcollection2-iwmpstringcollection2-getattributecountbytype--vb-and-c.md) | Gibt die Anzahl der Attribute zurück, die dem angegebenen Index des Zeichen folgen Auflistungs Elements, dem Attributnamen und der Sprache zugeordnet sind.<br/> |
| [**getItemInfo**](wmplibiwmpstringcollection2-iwmpstringcollection2-getiteminfo--vb-and-c.md)                         | Gibt die Zeichenfolge zurück, die dem angegebenen Index und Namen des Zeichen folgen-Sammlungs Elements entspricht.<br/>                                   |
| [**getItemInfoByType**](wmplibiwmpstringcollection2-iwmpstringcollection2-getiteminfobytype--vb-and-c.md)             | Gibt den Wert zurück, der dem angegebenen Index, Namen, der Sprache und dem Attribut Index für Zeichen folgen Auflistungs Elemente entspricht.<br/>        |
| [**isidentical**](wmplibiwmpstringcollection2-iwmpstringcollection2-isidentical--vb-and-c.md)                         | Gibt einen Wert zurück, der angibt, ob das angegebene Objekt mit dem aktuellen Objekt identisch ist.<br/>                                     |



 

Sie erhalten eine **IWMPStringCollection2** -Schnittstelle durch Umwandeln des Werts, der von der [**iwmpmediacollection. getAttributeStringCollection**](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmpmediacollection-getattributestringcollection) -Methode zurückgegeben wird.
