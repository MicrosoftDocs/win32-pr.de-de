# <a name="the-constructor"></a>Der Konstruktor

Der Konstruktor, den der Assistent bereitstellt, speichert einen Zeiger auf die Haupt-Plug-In-Klasse in einer Membervariablen. Der Assistent verwendet diesen Verweis, um Zugriff auf Membervariablen zu erhalten, die in der CSearch-Klasse deklariert sind. Es ist keine Implementierungsänderung erforderlich.
