# <a name="the-constructor"></a>Der Konstruktor

Der Konstruktor, den der Assistent bereitstellt, speichert einen Zeiger auf die Plug-in-Hauptklasse in einer Member-Variablen. Der Assistent verwendet diese Referenz, um Zugriff auf in der csearch-Klasse deklarierte Element Variablen zu erhalten. Eine Implementierungs Änderung ist nicht erforderlich.
