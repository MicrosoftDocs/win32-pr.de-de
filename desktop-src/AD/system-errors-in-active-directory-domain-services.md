# <a name="system-errors-in-active-directory-domain-services"></a>Systemfehler in Active Directory Domain Services

Dieses Thema enthält Rückgabewerte für Systemfehler in Funktionen Active Directory Domain Services.



| Wert                             | BESCHREIBUNG                                                           |
|-----------------------------------|-----------------------------------------------------------------------|
| **hrFileClose**<br/>        | Die DOS-Datei kann nicht geschlossen werden.<br/>                              |
| **hrOutOfThreads**<br/>     | Ein Thread kann nicht gestartet werden, da kein Thread verfügbar ist.<br/> |
| **hrTooManyIO**<br/>        | Das System ist ausgelastet, da zu viele E/A-Daten verfügbar sind.<br/>        |
| **hrBFNotSynchronous**<br/> | Die Pufferseite wurde entiert.<br/>                          |
| **hrBFPageNotFound**<br/>   | Die Seite wurde nicht finden.<br/>                                   |
| **hrBFInUse**<br/>          | Der Puffer kann nicht verlassen werden.<br/>                              |
