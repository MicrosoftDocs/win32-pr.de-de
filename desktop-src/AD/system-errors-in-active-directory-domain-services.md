# <a name="system-errors-in-active-directory-domain-services"></a>System Fehler in Active Directory Domain Services

Dieses Thema enthält Systemfehler Rückgabewerte in Funktionen von Active Directory Domain Services.



| Wert                             | BESCHREIBUNG                                                           |
|-----------------------------------|-----------------------------------------------------------------------|
| **hrfileclose**<br/>        | Die DOS-Datei kann nicht geschlossen werden.<br/>                              |
| **hrouto-Threads**<br/>     | Ein Thread kann nicht gestartet werden, da keine vorhanden sind.<br/> |
| **hrin omanyio**<br/>        | Das System ist ausgelastet, weil zu viele e/a-Vorgängen vorhanden sind.<br/>        |
| **hrbbnotsynchrone**<br/> | Die Puffer Seite wurde entfernt.<br/>                          |
| **hrbfpagenotfound**<br/>   | Die Seite wurde nicht gefunden.<br/>                                   |
| **hrbfinuse**<br/>          | Der Puffer kann nicht abgebrochen werden.<br/>                              |
