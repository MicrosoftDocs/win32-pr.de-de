# <a name="creating-dynamic-objects"></a>Erstellen dynamischer Objekte

Geben Sie das **DynamicObject** der Erweiterungs Klasse als Wert für das **objectClass** -Attribut zum Zeitpunkt der Instanziierung eines Objekts an. In diesem Fall wird die spezielle Hilfsklasse in der Liste der Klassen angezeigt, die den Wert des Attributs **objectClass** für das Objekt darstellen. Wenn beim Hinzufügen der besonderen Erweiterungs Klasse kein Wert für das **entryttl** -Attribut angegeben wird, wird ein Standardwert, wie später angegeben, von Active Directory Domain Services bereitgestellt.

Weitere Informationen zum Hinzufügen einer Hilfsklasse zu einem Objekt finden Sie unter [Dynamic Additions Classes](dynamic-auxiliary-classes.md).
