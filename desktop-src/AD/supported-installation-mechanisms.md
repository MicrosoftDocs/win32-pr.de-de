# <a name="supported-installation-mechanisms"></a>Unterstützte Installations Mechanismen

Active Directory Domain Services unterstützen vier Mechanismen zur Erweiterung des Active Directory Schemas. Sie können das Active Directory Schema erweitern, indem Sie verwenden.

-   [LDIF-Skripts](ldif-scripts.md) (bevorzugt)
-   [CSV-Skripts (Comma-Separated Value)](comma-separated-value-csv-scripts.md)
-   [Programmgesteuerte Erweiterung](programmatic-extension.md)
-   und durch [Erweitern der Benutzeroberfläche für Verzeichnisobjekte](extending-the-user-interface-for-directory-objects.md).
