# <a name="characteristics-of-attributes"></a>Merkmale von Attributen

Jedes Attribut in einem Active Directory Domain Services Objekt wird von einem [**attributeSchema**](/windows/desktop/ADSchema/c-attributeschema) -Objekt im Schema Container definiert. Die Eigenschaften eines **attributeSchema** -Objekts geben die Merkmale des Attributs an, z. b.:

-   Attribut Bezeichner. Attribute verfügen über mehrere Bezeichner, von denen die interessanteste aus einer Programmier Perspektive der [**ldapDisplayName**](/windows/desktop/ADSchema/a-ldapdisplayname)ist, der von LDAP-Clients zum Lesen und Schreiben des Attributs verwendet wird, und die [**schemaIdGUID**](/windows/desktop/ADSchema/a-schemaidguid), die in Sicherheits Deskriptoren zum Steuern des Zugriffs auf das Attribut verwendet wird.
-   Der Typ der Daten, die in Instanzen des Attributs enthalten sind. Die Syntax Eigenschaften eines Attributs bestimmen den Datentyp, z. b. "Integer", "String" oder "Binary". Zusätzliche Eigenschaften können den Wertebereich angeben, der für das Attribut zulässig ist, und angeben, ob eine Instanz des Attributs mehrere Werte aufweisen kann.
-   Einschließen des-Attributs in Gruppen. Andere Eigenschaften markieren ein Attribut, das in einen Eigenschaften Satz eingeschlossen werden soll, der eine Gruppe verwandter Eigenschaften ist oder in den Satz von Attributen eingeschlossen werden soll, die im globalen Katalog repliziert oder indiziert werden, um die Suchleistung zu optimieren.

Weitere Informationen und eine komplette Liste der obligatorischen und optionalen Eigenschaften eines **attributeSchema** -Objekts finden Sie unter [**Attribut-Schema-Klasse**](/windows/desktop/ADSchema/c-attributeschema).
