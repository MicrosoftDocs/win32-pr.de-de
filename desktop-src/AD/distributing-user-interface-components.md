# <a name="distributing-user-interface-components"></a>Verteilen von Benutzeroberflächen Komponenten

Die COM-Komponenten, Anwendungen und/oder Dateien, die von einer Benutzeroberflächen Erweiterung verwendet werden, können verteilt werden, indem ein Windows Installer Paket erstellt und eine Gruppenrichtlinie zum Verteilen des Pakets verwendet wird. Weitere Informationen zum Erstellen des Installationspakets finden Sie unter [Windows Installer](/windows/desktop/Msi/windows-installer-portal). Weitere Informationen zu Gruppenrichtlinien finden Sie unter [Gruppenrichtlinie](/previous-versions/windows/desktop/Policy/group-policy-start-page).
