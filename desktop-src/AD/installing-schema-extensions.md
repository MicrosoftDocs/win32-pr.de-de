# <a name="installing-schema-extensions"></a>Installieren von Schemaerweiterungen

Schemaupdates werden an den meisten Kundenstandorten streng kontrolliert . Dies ist die empfohlene Vorgehensweise. Wenn ein Dienst Schemaerweiterungen erfordert, müssen die Schemaerweiterungen separat installiert werden können. Dadurch können Kunden das Schemaupdate separat und vor der restlichen Installation durchführen.
