# <a name="installing-schema-extensions"></a>Installieren von Schema Erweiterungen

Schema Aktualisierungen werden auf den meisten Kundenstandorten streng gesteuert. Dies ist die empfohlene Vorgehensweise. Wenn ein Dienst Schema Erweiterungen erfordert, müssen die Schema Erweiterungen separat installiert werden. Dies ermöglicht es Kunden, die Schema Aktualisierung separat und im Voraus auf den Rest der Installation durchzuführen.
