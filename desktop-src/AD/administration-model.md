# <a name="administration-model"></a>Verwaltungsmodell

Autorisierte Benutzer führen die Verwaltung in Active Directory Domain Services aus. Ein Benutzer wird von einer höheren Zertifizierungsstelle autorisiert, um einen bestimmten Satz von Aktionen für eine angegebene Gruppe von Objekten und Objektklassen in einer identifizierten Teilstruktur des Verzeichnisses auszuführen. Dies wird als *Delegierte Administration* bezeichnet. Mit der delegierten Administration können Sie genau steuern, wer welche Aktionen durchführen kann, und die Delegierung der Autorisierungs Autorität aktivieren, ohne erhöhte Rechte zu gewähren.
