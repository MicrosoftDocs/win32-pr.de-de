# <a name="mmc-property-page-structures-in-active-directory-domain-services"></a>MMC-Eigenschaften Seitenstrukturen in Active Directory Domain Services

Das MMC-Snap-in "Active Directory-Manager" verwendet Eigenschaften Blätter, um die Attribute von Objekten in Active Directory Domain Services anzuzeigen. Die folgenden Strukturen helfen bei der Bereitstellung der-Infrastruktur, um die Vorgänge zu steuern, die auf den Seiten dieser Eigenschaften Blätter ausgeführt werden:

-   [**Adsproperror**](/windows/desktop/api/Adsprop/ns-adsprop-adsproperror)
-   [**Adspropinitparameams**](/windows/desktop/api/Adsprop/ns-adsprop-adspropinitparams)
