# <a name="mmc-property-page-structures-in-active-directory-domain-services"></a>MMC-Eigenschaftenseitenstrukturen in Active Directory Domain Services

Das MMC-Snap-In des Active Directory-Managers verwendet Eigenschaftenblätter, um die Attribute von Objekten in Active Directory Domain Services anzuzeigen. Die folgenden Strukturen stellen die Infrastruktur bereit, um die Vorgänge zu steuern, die auf den Seiten dieser Eigenschaftenblätter ausgeführt werden:

-   [**ADSPROPERROR**](/windows/desktop/api/Adsprop/ns-adsprop-adsproperror)
-   [**ADSPROPINITPARAMS**](/windows/desktop/api/Adsprop/ns-adsprop-adspropinitparams)
