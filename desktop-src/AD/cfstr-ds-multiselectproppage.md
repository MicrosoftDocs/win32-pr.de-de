# <a name="cfstr_ds_multiselectproppage"></a>cfstr \_ DS \_ multiselectproppage

<dl> <dt>

<span id="CFSTR_DS_MULTISELECTPROPPAGE"></span><span id="cfstr_ds_multiselectproppage"></span>**cfstr \_ DS \_ multiselectproppage**
</dt> <dd> <dl> <dt>

"Dsadminmultiselectclipformat"
</dt> <dt>



Das " **cfstr \_ DS \_ multiselectproppage** "-Zwischenablage Format bietet ein globales Speicher handle (**HGLOBAL**), das eine auf NULL endende Unicode-Zeichenfolge enthält, die einen eindeutigen Namen enthält, der zum Erstellen des Benachrichtigungs Objekts für eine Eigenschaften Blatt Erweiterung mit Mehrfachauswahl verwendet wird.

> [!Note]  
> Dieser Wert für die Zwischenablage ist in einer veröffentlichten Header Datei nicht definiert. Wenn Sie diesen Wert für die Zwischenablage Format verwenden möchten, müssen Sie ihn im exakten Format definieren.
