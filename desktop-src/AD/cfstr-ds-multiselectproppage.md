# <a name="cfstr_ds_multiselectproppage"></a>CFSTR \_ DS \_ MULTISELECTPROPPAGE

<dl> <dt>

<span id="CFSTR_DS_MULTISELECTPROPPAGE"></span><span id="cfstr_ds_multiselectproppage"></span>**CFSTR \_ DS \_ MULTISELECTPROPPAGE**
</dt> <dd> <dl> <dt>

"DsAdminMultiSelectClipFormat"
</dt> <dt>



Das **CFSTR \_ DS \_ MULTISELECTPROPPAGE-Zwischenablageformat** stellt ein globales Speicherhandle **(HGLOBAL) bereit,** das eine auf NULL endende Unicode-Zeichenfolge enthält, die einen eindeutigen Namen enthält, der zum Erstellen des Benachrichtigungsobjekts für eine Eigenschaftenblatterweiterung mit mehrfacher Auswahl verwendet wird.

> [!Note]  
> Dieser Zwischenablageformatwert ist in einer veröffentlichten Headerdatei nicht definiert. Um diesen Zwischenablageformatwert zu verwenden, müssen Sie ihn selbst im genau angezeigten Format definieren.

 


</dt> </dl> </dd> </dl>
