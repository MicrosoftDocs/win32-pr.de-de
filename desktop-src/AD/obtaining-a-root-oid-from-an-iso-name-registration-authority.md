# <a name="obtaining-a-root-oid-from-an-iso-name-registration-authority"></a>Abrufen einer Stamm-OID von einer ISO-namens Registrierungsstelle

Die bevorzugte Methode zum Abrufen eines Stamm Objekt Bezeichners (OID) ist die Anforderung eines von einer ISO-namens Registrierungsstelle (International Standards Organization). Dies ist eine einmalige Aktion. Wenn Sie eine Stamm-OID erhalten haben, ist der OID-Speicherplatz, den Sie definiert, von Ihnen, und Sie können Sie selbst verwalten.

Es gibt in der Regel eine Gebühr, die dem Registrieren eines Organisations namens und dem Empfang einer Stamm-OID zugeordnet ist. Die Gebühren und Registrierungsrichtlinien variieren je nach Land/Region. In der USA ist die ISO-NRA der American National Standards Institute (ANSI). Weitere Informationen über die ANSI-Registrierungs Prozedur und den Gebühren Zeitplan finden Sie unter https://web.ansi.org . Der ISO-Abbild verwaltet eine Liste der Mitgliedsorganisationen unter https://www.iso.ch . Wenn Sie sich außerhalb des USA befinden, wenden Sie sich an die ISO-Mitgliedsorganisation für Ihr Land/Ihre Region, um Informationen zur namens Registrierung zu erhalten.

Unabhängig von der Quelle, die zum erhalten der OID verwendet wird, müssen Sie Ihre OID bei Microsoft registrieren, wenn Sie das Active Directory Schema erweitern und für das Logo "Certified for Windows" anwenden möchten. Weitere Informationen zum Registrieren Ihrer OID bei Microsoft finden Sie unter Abrufen [eines Objekt Bezeichners von Microsoft](obtaining-an-object-identifier-from-microsoft.md).
