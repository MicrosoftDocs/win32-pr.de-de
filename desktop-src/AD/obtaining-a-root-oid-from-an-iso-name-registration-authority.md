# <a name="obtaining-a-root-oid-from-an-iso-name-registration-authority"></a>Abrufen einer Stamm-OID von einer ISO-Namenregistrierungsstelle

Die bevorzugte Methode zum Abrufen eines Stammobjektbezeichners (Root Object Identifier, OID) ist das Anfordern eines Bezeichners von einer Iso-Zertifizierungsstelle (International Standards Organization). Dies ist eine einmalige Aktion. Wenn Sie eine Stamm-OID erhalten haben, liegt der von ihr definierte OID-Raum bei Ihnen, und Sie können sie selbst verwalten.

Für die Registrierung eines Organisationsnamens und den Empfang einer Stamm-OID wird in der Regel eine Gebühr berechnet. Die Gebühren und Registrierungsrichtlinien variieren je nach Land/Region. In der USA ist die ISO NRA die American National Standards Institute (ANSI). Weitere Informationen zum ANSI-Registrierungsverfahren und zum Gebührenzeitplan finden Sie unter https://web.ansi.org . Die ISO verwaltet eine Liste von Mitgliedsorganisationen unter https://www.iso.ch . Wenn Sie sich außerhalb der USA, wenden Sie sich an die ISO-Mitgliedsorganisation für Ihr Land/Ihre Region, um Informationen zur Namensregistrierung zu erhalten.

Unabhängig von der Quelle, die zum Erhalten der OID verwendet wird, müssen Sie Ihre OID bei Microsoft registrieren, wenn Sie beabsichtigen, das Active Directory-Schema zu erweitern und sich für das Certified for Windows-Logo zu bewerben. Weitere Informationen zum Registrieren Ihrer OID bei Microsoft finden Sie unter [Abrufen eines Objektbezeichners von Microsoft.](obtaining-an-object-identifier-from-microsoft.md)
