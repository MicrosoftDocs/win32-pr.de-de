# <a name="storing-dynamic-data"></a>Speichern von dynamische Daten

Die meisten daten, die mit Active Directory Domain Services gespeichert werden, sind statisch. Beispielsweise werden Verzeichnisobjekte, die Benutzer, Gruppen, Drucker, Netzwerkdienste usw. darstellen, in der Regel langlebig und selten geändert.

In Windows Server 2003 und höher von Windows bieten Active Directory Domain Services Unterstützung für die folgenden Features, mit denen Anwendungen das Verzeichnis zum Speichern dynamischer Daten verwenden können:

-   [Dynamische Objekte](dynamic-objects.md)
-   [Dynamische Hilfsklassen](dynamic-auxiliary-classes.md)
