# <a name="storing-dynamic-data"></a>Speichern von dynamische Daten

Die meisten mit Active Directory Domain Services gespeicherten Daten sind statisch. Beispielsweise sind Verzeichnisobjekte, die Benutzer, Gruppen, Drucker, Netzwerkdienste usw. darstellen, in der Regel langlebig und werden selten geändert.

In Windows Server 2003 und neueren Versionen von Windows Active Directory Domain Services Unterstützung für die folgenden Features bereitstellen, die es Anwendungen ermöglichen, das Verzeichnis zum Speichern dynamischer Daten zu verwenden:

-   [Dynamische Objekte](dynamic-objects.md)
-   [Dynamische Hilfsklassen](dynamic-auxiliary-classes.md)
