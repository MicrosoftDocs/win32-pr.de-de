# <a name="deciding-which-attributes-to-retrieve-for-each-object-found"></a>Entscheidung, welche Attribute für jedes gefundene Objekt abgerufen werden sollen

Wenn Sie nach Objekten einer bestimmten Klasse suchen, erscheint es logisch, dass Sie Attribute abrufen, die tatsächlich für die Objekte dieser Klasse vorhanden sind. Befolgen Sie die gleichen Regeln, die unter [Suchen einer Liste von abzufragende Attribute](finding-a-list-of-attributes-to-query.md)erläutert werden. Beachten Sie, dass konstruierte Attribute in der Abfrage abgerufen werden können, aber konstruierte Attribute können nicht in Filtern verwendet werden.
