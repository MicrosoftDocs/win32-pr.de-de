# <a name="deciding-which-attributes-to-retrieve-for-each-object-found"></a>Entscheiden, welche Attribute für die einzelnen gefundenen Objekte abgerufen werden sollen

Wenn Sie nach Objekten einer bestimmten Klasse suchen, scheint es logisch, dass Sie Attribute abrufen, die tatsächlich für die Objekte dieser Klasse vorhanden sind. Befolgen Sie die gleichen Regeln, die untersuchen [einer Liste der abzufragende Attribute](finding-a-list-of-attributes-to-query.md)erläutert werden. Beachten Sie, dass konstruierte Attribute in der Abfrage abgerufen werden können, konstruierte Attribute jedoch nicht in Filtern verwendet werden können.
