# <a name="where-to-create-a-service-connection-point"></a><span data-ttu-id="5ba3d-108">Erstellen eines Dienst Verbindungs Punkts</span><span class="sxs-lookup"><span data-stu-id="5ba3d-108">Where to Create a Service Connection Point</span></span>

<span data-ttu-id="5ba3d-109">Wenn eine Instanz von Active Directory Domain Services installiert ist, erstellt das Dienst Installationsprogramm Dienst Verbindungspunkt-Objekte (SCP) in Active Directory Domain Services.</span><span class="sxs-lookup"><span data-stu-id="5ba3d-109">When an instance of Active Directory Domain Services is installed, the service installer creates service connection point objects (SCP) in Active Directory Domain Services.</span></span> <span data-ttu-id="5ba3d-110">Das Hauptziel sollte darin bestehen, den Replikations Datenverkehr zu minimieren und eine effiziente Verwaltung und Wartung von Objekten zu ermöglichen.</span><span class="sxs-lookup"><span data-stu-id="5ba3d-110">The primary objective should be to minimize replication traffic and to enable efficient administration and maintenance of objects.</span></span>

<span data-ttu-id="5ba3d-111">Beachten Sie, dass Client Anwendungen SCPs suchen, indem Sie im Verzeichnis nach Schlüsselwörtern im SCP suchen.</span><span class="sxs-lookup"><span data-stu-id="5ba3d-111">Be aware that client applications find SCPs by searching the directory for keywords in the SCP.</span></span> <span data-ttu-id="5ba3d-112">Das **Schlüssel** Wort Attribut eines SCP ist im globalen Katalog enthalten. ## <a name="minimize-replication-traffic"></a><span data-ttu-id="5ba3d-114">Minimierung des Replikations</span><span class="sxs-lookup"><span data-stu-id="5ba3d-114">Minimize Replication Traffic</span></span>

Es wird nur mehr Replikations Datenverkehr generiert.</span><span class="sxs-lookup"><span data-stu-id="5ba3d-120">As noted above, clients throughout the forest can query the Global Catalog to find SCPs anywhere in the forest, so creating SCPs in the Configuration partition does not make them more visible to clients; it only generates more replication traffic.</span></span>
