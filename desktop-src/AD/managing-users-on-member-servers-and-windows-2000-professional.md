# <a name="managing-users-on-member-servers-and-windows-2000-professional"></a>Verwalten von Benutzern auf Mitglieds Servern und Windows 2000 Professional

Auf Mitglieds Servern und Windows 2000 Professional gibt es eine lokale Sicherheitsdatenbank. Diese lokale Sicherheitsdatenbank kann eigene lokale Benutzer enthalten, deren Bereich nur der bestimmte Computer ist, auf dem Sie erstellt werden. Wenn Sie diese Benutzer Typen auf Mitglieds Servern und Arbeitsstationen verwalten, verwenden Sie den WinNT-Anbieter.

Wenn Sie Benutzer in einer Windows 2000-Domäne mithilfe von ADSI verwalten, verwenden Sie den LDAP-Anbieter. Wenn Sie Benutzer auf Mitglieds Servern und Arbeitsstationen verwalten, verwenden Sie den WinNT-Anbieter.
