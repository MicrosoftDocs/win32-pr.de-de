# <a name="nesting-a-group-in-another-group"></a>Schachteln einer Gruppe in einer anderen Gruppe

Das Hinzufügen einer Gruppe als Mitglied einer anderen Gruppe wird *als Schachtelung bezeichnet.* Bei Verteiler Gruppen wird die Schachtelung sowohl im gemischten als auch im einheitlichen Modus unterstützt. Für Sicherheitsgruppen wird die Schachtelung nur für Domänen unterstützt, die im einheitlichen Modus ausgeführt werden.

Um eine Gruppe in einer anderen Gruppe zu schachteln, verwenden Sie die Verfahren, die unter [Hinzufügen von Mitgliedern zu Gruppen in einer Domäne](adding-members-to-groups-in-a-domain.md)beschrieben werden. Beachten Sie, dass die Gruppe abhängig vom Bereich der Gruppe nur bestimmte Typen und Bereiche anderer Gruppen enthalten kann.

Die Schachtelungs Optionen hängen auch davon ab, ob sich die Domäne im gemischten oder im einheitlichen Modus befindet. Weitere Informationen zu Gruppen in unterschiedlichen Domänen Modi finden Sie unter Schachtelung [im einheitlichen Modus](nesting-in-native-mode.md) und Schachtelung [im gemischten Modus](nesting-in-mixed-mode.md).

Weitere Informationen zum Schachteln von Gruppen finden Sie unter [häufige Fehler](common-errors.md).
