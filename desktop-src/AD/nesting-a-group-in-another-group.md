# <a name="nesting-a-group-in-another-group"></a>Schachteln einer Gruppe in einer anderen Gruppe

Das Hinzufügen einer Gruppe als Mitglied einer anderen Gruppe wird als *Schachtelung* bezeichnet. Für Verteilergruppen wird die Schachtelung sowohl im gemischten als auch im einheitlichen Modus unterstützt. Bei Sicherheitsgruppen wird die Schachtelung nur für Domänen unterstützt, die im einheitlichen Modus ausgeführt werden.

Um eine Gruppe in einer anderen Gruppe [zu](adding-members-to-groups-in-a-domain.md)schachteln, verwenden Sie die gleichen Verfahren wie unter Hinzufügen von Mitgliedern zu Gruppen in einer Domäne beschrieben. Beachten Sie, dass die Gruppe abhängig vom Bereich der Gruppe nur bestimmte Typen und Bereiche anderer Gruppen enthalten kann.

Die Schachtelungsoptionen hängen auch davon ab, ob sich die Domäne im gemischten oder einheitlichen Modus befindet. Weitere Informationen zu Gruppen in verschiedenen Domänenmodi finden Sie unter [Schachteln im einheitlichen Modus](nesting-in-native-mode.md) und [Schachteln im gemischten Modus.](nesting-in-mixed-mode.md)

Weitere Informationen zum Schachteln von Gruppen finden Sie unter [Häufige Fehler.](common-errors.md)
