<span data-ttu-id="14b20-105">Der Schema Container enthält alle **classSchema** -und **attributeSchema** -Objekte, die die Klassen und Attribute definieren, die in einer Verzeichnis Gesamtstruktur vorhanden sein können.</span><span class="sxs-lookup"><span data-stu-id="14b20-105">The schema container contains all of the **classSchema** and **attributeSchema** objects that define the classes and attributes that can exist in a directory forest.</span></span> <span data-ttu-id="14b20-106">Der Schema Container enthält auch ein Objekt mit dem Namen Aggregate of Class **subschema**.</span><span class="sxs-lookup"><span data-stu-id="14b20-106">The schema container also contains an object named Aggregate of class **subSchema**.</span></span> <span data-ttu-id="14b20-107">Dieses **unter Schema** Objekt wird als abstraktes Schema bezeichnet.</span><span class="sxs-lookup"><span data-stu-id="14b20-107">This **subSchema** object is known as the abstract schema.</span></span>
