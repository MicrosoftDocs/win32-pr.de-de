In diesem Thema werden die Eigenschaften Blätter für Domänen Objekte im-Snap-in "Active Directory Benutzer und Computer" erläutert.

## <a name="general-property-sheet"></a>Eigenschaften Blatt "Allgemein"

In der folgenden Tabelle werden die Benutzeroberflächen Bezeichnungen auf der Eigenschaften Seite " **Allgemein** " angezeigt.



| UI-Bezeichnung                       | Active Directory-Attribut |
|--------------------------------|----------------------------|
| Domänen Name (Pre-Windows 2000) | SL\*                       |
| BESCHREIBUNG                    | description                |



 

\*Das DC-Attribut bietet Zugriff auf die erste der Domänen Komponenten, die nicht notwendigerweise mit dem NetBIOS-DN identisch ist.
