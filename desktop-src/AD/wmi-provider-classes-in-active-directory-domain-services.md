# <a name="wmi-provider-classes-in-active-directory-domain-services"></a>WMI-Anbieterklassen in Active Directory Domain Services

Active Directory Domain Services verfügen über die folgenden Klassen für die Verwendung mit WMI-Skripts:

-   [**MSAD \_ DomainController**](msad-domaincontroller.md)
-   [**MSAD \_ NamingContext**](msad-namingcontext.md)
-   [**MSAD \_ ReplCursor**](msad-replcursor.md)
-   [**MSAD \_ ReplNeighbor**](msad-replneighbor.md)
-   [**MSAD \_ ReplPendingOp**](msad-replpendingop.md)

Der Anbieter selbst wird durch die [**ReplicationProvider1-Klasse**](replicationprovider1.md) dargestellt, die mit dem Namen "ReplProv1" instanziert wird.
