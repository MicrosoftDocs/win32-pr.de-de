# <a name="wmi-provider-classes-in-active-directory-domain-services"></a>WMI-Anbieter Klassen in Active Directory Domain Services

Active Directory Domain Services die folgenden Klassen für die Verwendung mit WMI-Skripts verwenden:

-   [**MSAD \_ Domain Controller**](msad-domaincontroller.md)
-   [**MSAD- \_ NamingContext**](msad-namingcontext.md)
-   [**MSAD \_ replcursor**](msad-replcursor.md)
-   [**MSAD \_ ReplNeighbor**](msad-replneighbor.md)
-   [**MSAD \_ replpdingop**](msad-replpendingop.md)

Der Anbieter selbst wird durch die [**ReplicationProvider1**](replicationprovider1.md) -Klasse dargestellt, die mit dem Namen "ReplProv1" instanziierten ist.
