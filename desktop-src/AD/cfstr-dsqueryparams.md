# <a name="cfstr_dsqueryparams"></a>CFSTR \_ DSQUERYPARAMS

<dl> <dt>

<span id="CFSTR_DSQUERYPARAMS"></span><span id="cfstr_dsqueryparams"></span>**CFSTR \_ DSQUERYPARAMS**
</dt> <dd> <dl> <dt>

"DsQueryParameters"
</dt> <dt>



Das **CFSTR \_ DSQUERYPARAMS-Zwischenablageformat** wird von [**dem IDataObject**](/windows/win32/api/objidl/nn-objidl-idataobject) unterstützt, das von der [**ICommonQuery::OpenQueryWindow-Methode bereitgestellt**](/windows/win32/api/cmnquery/nf-cmnquery-icommonquery-openquerywindow) wird. Das **CFSTR \_ DSQUERYPARAMS-Zwischenablageformat** stellt ein globales Speicherhandli (**HGLOBAL**) zur Verfügung, das eine [**DSQUERYPARAMS-Struktur**](/windows/desktop/api/Dsquery/ns-dsquery-dsqueryparams) enthält.


</dt> </dl> </dd> </dl>
