# <a name="guidelines-for-binding-to-the-schema"></a><span data-ttu-id="ca60d-105">Richtlinien für die Bindung an das Schema</span><span class="sxs-lookup"><span data-stu-id="ca60d-105">Guidelines for Binding to the Schema</span></span>
