# <a name="reading-and-writing-attributes-of-objects-in-active-directory-domain-services"></a>Lesen und Schreiben von Attributen von Objekten in Active Directory Domain Services

Objekte in Active Directory Domain Services über Attribute verfügen, die Daten über das jeweilige Objekt enthalten. Wie die Attribute aufgerufen und geändert werden, hängt von der verwendeten Programmier Technologie ab. Weitere Informationen zum Lesen und Ändern von Attributen in Active Directory Domain Services mit einer bestimmten Programmier Technologie finden Sie in den folgenden aufgeführten Themen.



| Programmier Technologie                                                                       | Weitere Informationen finden Sie unter                                                                        |
|----------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| [Active Directory Service Interfaces](/windows/desktop/ADSI/active-directory-service-interfaces-adsi)         | [Zugreifen auf und Bearbeiten von Daten mit ADSI](/windows/desktop/ADSI/accessing-and-manipulating-data-with-adsi) |
| [Lightweight Directory Access Protocol](/previous-versions/windows/desktop/ldap/lightweight-directory-access-protocol-ldap-api) | [Ändern eines Verzeichniseintrags](/previous-versions/windows/desktop/ldap/modifying-a-directory-entry)                             |
| [System.DirectoryServices](/dotnet/api/system.directoryservices) | [Verzeichnis Objekteigenschaften](https://msdn.microsoft.com/library/ms180858(v=VS.80).aspx)                         |



 

Weitere Informationen zu Attributen finden Sie unter [bestimmen eines Attribut Typs](determining-an-attribute-type.md).
