# <a name="cqpm_initialize-message"></a>Cqpm- \_ Initialisierungs Meldung

Die **cqpm- \_ Initialisierungs** Meldung wird an die [**cqpageproc**](/windows/desktop/api/Cmnquery/nc-cmnquery-lpcqpageproc) -Rückruffunktion einer Abfrageformular-Erweiterungs Seite gesendet, wenn die Seite einem Formular hinzugefügt wird.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Nicht verwendet.

</dd> <dt>

*lParam* 
</dt> <dd>

Nicht verwendet.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Der Rückgabewert für diese Nachricht wird ignoriert.
