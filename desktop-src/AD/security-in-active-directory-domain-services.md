# <a name="security-in-active-directory-domain-services"></a>Sicherheit in Active Directory Domain Services

Active Directory Domain Services sind Teil der Windows 2000 Trusted Computing-Basis und stellen einen vollständigen Teilnehmer an der Windows 2000-Sicherheitsinfrastruktur dar. Jedes Objekt in einem Active Directory Server wird durch eine eigene Sicherheits Beschreibung geschützt. Das System überprüft jeden Versuch, auf ein Objekt oder Attribut in einem Active Directory Server zuzugreifen, indem er die Zugriffsrechte überprüft, die von der Objekt Sicherheits Beschreibung erteilt werden.

In den folgenden Themen werden wichtige Elemente der Sicherheit in Active Directory Domain Services erläutert:

-   [Objekt-und Attribut Schutz](object-and-attribute-protection.md)
-   [Delegierung](delegation.md)
-   [Vererbung](inheritance.md)

Weitere Informationen und Codebeispiele finden Sie unter [Steuern des Zugriffs auf Objekte in Active Directory Domain Services](controlling-access-to-objects-in-active-directory-domain-services.md).
