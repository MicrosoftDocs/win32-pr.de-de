# <a name="security-in-active-directory-domain-services"></a>Sicherheit in Active Directory Domain Services

Active Directory Domain Services sind Teil der Windows 2000-vertrauenswürdigen Computingbasis und ein vollständiger Teilnehmer an der Windows 2000-Sicherheitsinfrastruktur. Jedes Objekt auf einem Active Directory-Server wird durch einen eigenen Sicherheitsdeskriptor geschützt. Das System überprüft alle Versuche, auf ein Objekt oder Attribut in einem Active Directory-Server zu zugreifen, indem die zugriffsrechte überprüft werden, die vom Objektsicherheitsdeskriptor gewährt werden.

In den folgenden Themen werden wichtige Sicherheitselemente in Active Directory Domain Services:

-   [Objekt- und Attributschutz](object-and-attribute-protection.md)
-   [Delegierung](delegation.md)
-   [Vererbung](inheritance.md)

Weitere Informationen und Codebeispiele finden Sie unter [Steuern des Zugriffs auf Objekte in Active Directory Domain Services.](controlling-access-to-objects-in-active-directory-domain-services.md)
