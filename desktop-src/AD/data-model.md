# <a name="data-model"></a>Datenmodell

Das Datenmodell für Active Directory Domain Services wird aus dem X. 500-Datenmodell abgeleitet. Das Verzeichnis enthält Objekte, die Elemente verschiedener Sortierungen darstellen, die von Attributen beschrieben werden. Das Universum von Objekten, die im Verzeichnis gespeichert werden können, wird im Schema definiert. Für jede Objektklasse definiert das Schema, welche Attribute eine Instanz der Klasse aufweisen muss, welche zusätzlichen Attribute Sie haben kann und welche Objektklasse als übergeordnetes Element der aktuellen Objektklasse festgelegt werden kann.
