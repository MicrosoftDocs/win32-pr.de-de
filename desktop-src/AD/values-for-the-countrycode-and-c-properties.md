# <a name="values-for-the-countrycode-and-c-properties"></a>Werte für die Eigenschaften countryCode und c

Die Werte für die Eigenschaften **countryCode** und **c** werden aus dem ISO-Standard 3166 abgerufen. Dieser Standard ist auf der [Internationale Organisation für Normung-Website](https://www.iso.org/) verfügbar. Der **countryCode-Eigenschaftswert** ist der numerische Ländercode aus ISO 3166, und der **c-Eigenschaftswert** ist die zweibuchstabe Länder-/Regionsbezeichnung aus ISO 3166.
