# <a name="values-for-the-countrycode-and-c-properties"></a>Werte für die Eigenschaften "CountryCode" und "c"

Die Werte für die Eigenschaften " **CountryCode** " und " **c** " werden aus dem ISO-Standard 3166 abgerufen. Dieser Standard ist auf der [internationale Organisation für Normung](https://www.iso.org/) -Website verfügbar. Der **CountryCode** -Eigenschafts Wert ist der numerische Ländercode aus ISO 3166, und der **c** -Eigenschafts Wert ist die zwei buchstabige Bezeichnung für Land/Region aus ISO 3166.
