# <a name="cfstr_ds_propsheetconfig"></a>cfstr \_ DS \_ propsheetconfig

<dl> <dt>

<span id="CFSTR_DS_PROPSHEETCONFIG"></span><span id="cfstr_ds_propsheetconfig"></span>**cfstr \_ DS \_ propsheetconfig**
</dt> <dd> <dl> <dt>

"Dspropsheetcfgclipformat"
</dt> <dt>



Das **cfstr \_ DS \_ propsheetconfig** -Zwischenablage Format stellt ein globales Speicher handle (**HGLOBAL**) bereit, das eine [**propsheetcfg**](propsheetcfg.md) -Struktur enthält.

> [!Note]  
> Dieser Wert für die Zwischenablage ist in einer veröffentlichten Header Datei nicht definiert. Wenn Sie diesen Wert für die Zwischenablage Format verwenden möchten, müssen Sie ihn im exakten Format definieren.
