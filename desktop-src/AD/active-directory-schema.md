# <a name="active-directory-schema-ad-ds"></a>Active Directory Schema (AD DS)

Das Active Directory Schema enthält formale Definitionen für jede Objektklasse, die in einer Active Directory Gesamtstruktur erstellt werden kann. Das Schema enthält außerdem formale Definitionen für jedes Attribut, das in einem Active Directory Objekt vorhanden sein kann.

Die folgenden Themen enthalten eine Übersicht über das Active Directory Schema:

-   [Schema Implementierung](schema-implementation.md)
-   [Eigenschaften von Objektklassen](characteristics-of-object-classes.md)
-   [Merkmale von Attributen](characteristics-of-attributes.md)
-   [Das abstrakte Schema](the-abstract-schema.md)

Weitere Informationen zur Schema Programmierung, einschließlich Lesen des Schemas und Definieren neuer Klassen und Attribute im Schema, finden Sie unter [Erweitern des Schemas](extending-the-schema.md).

Weitere Informationen finden Sie in der [Active Directory Domain Services-Referenz](active-directory-domain-services-reference.md). Informationen zu den Referenzseiten für die vordefinierten Schema Klassen, Attribute und Attribut Syntax finden Sie in der [Active Directory-Schema Referenz](/windows/desktop/ADSchema/active-directory-schema).
