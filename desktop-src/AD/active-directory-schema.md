# <a name="active-directory-schema-ad-ds"></a>Active Directory-Schema (AD DS)

Das Schema enthält formale Definitionen für jede Objektklasse, die in einer Active Directory-Gesamtstruktur erstellt werden kann. Das Schema enthält auch formale Definitionen jedes Attributs, das in einem Active Directory-Objekt vorhanden sein kann.

Zu den Themen, die eine Übersicht über das Active Directory-Schema bieten, gehören:

-   [Schemaimplementierungen](schema-implementation.md)
-   [Merkmale von Objektklassen](characteristics-of-object-classes.md)
-   [Merkmale von Attributen](characteristics-of-attributes.md)
-   [Das abstrakte Schema](the-abstract-schema.md)

Weitere Informationen zur Schemaprogrammierung, einschließlich lesen des Schemas und Definieren neuer Klassen und Attribute im Schema, finden Sie unter [Erweitern des Schemas.](extending-the-schema.md)

Weitere Informationen finden Sie in der [Active Directory Domain Services Reference](active-directory-domain-services-reference.md). Die Referenzseiten für die vordefinierten Schemaklassen, Attribute und Attributsyntaxen finden Sie in der [Active Directory-Schemareferenz.](/windows/desktop/ADSchema/active-directory-schema)
