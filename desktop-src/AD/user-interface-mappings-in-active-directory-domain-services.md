# <a name="user-interface-mappings-in-active-directory-domain-services"></a>Zuordnungen von Benutzeroberflächen in Active Directory Domain Services

Die Benutzeroberfläche für Active Directory Domain Services enthält Eigenschaften Blätter und andere Benutzeroberflächen zum Festlegen und Abrufen der Attribute der häufig verwendeten Klassen. Die Themen in diesem Abschnitt enthalten Tabellen, in denen Bezeichnungen, die in den Benutzeroberflächen angezeigt werden, mit den entsprechenden Attributen im Active Directory Schema zugeordnet werden.

-   [Zuordnungen für das Snap-in "Active Directory-Benutzer und-Computer"](mappings-for-the-active-directory-users-and-computers-snap-in.md)
