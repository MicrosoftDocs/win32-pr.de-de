# <a name="documenting-schema-extensions"></a>Dokumentieren von Schema Erweiterungen

Zusätzlich zur Bereitstellung eines separaten Installations Verfahrens für Schema Erweiterungen muss die Art der Schema Erweiterungen eindeutig dokumentiert werden. Die Dokumentation sollte Folgendes enthalten:

-   Eine Liste aller neuen Klassen und Attribute mit einer Erläuterung der einzelnen Klassen und Attribute
-   Eine Liste aller Standardklassen, die mit neuen Attributen und einer Liste der Attribute erweitert werden.
-   Eine Liste mit allen neuen Eigenschaften Seiten, die eine Erläuterung zur Verwendung der einzelnen Eigenschaften Seiten enthalten.
-   Eine Erläuterung der Verwendung der Schema Erweiterungen durch den Dienst
