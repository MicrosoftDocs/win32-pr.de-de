# <a name="extending-the-user-interface-for-directory-objects"></a>Erweitern der Benutzeroberfläche für Verzeichnisobjekte

Mit Microsoft Windows 2000 und höher sind Microsoft Management Console-Snap-Ins (MMC), z. B. das Active Directory-Benutzer und -Computer-Snap-In, für die Verwaltung von Active Directory Domain Services verfügbar. Darüber hinaus enthält die Windows 2000-Shell Benutzeroberflächen (UIs) zum Suchen von Objekten, die sich im Verzeichnis befinden, sowie zum Lesen und Schreiben von Eigenschaften. In diesem Abschnitt wird erläutert, wie Sie die Benutzeroberfläche zum Anzeigen und Verwalten von Active Directory Domain Services-Objekten in der Windows-Shell und den Active Directory-Administrator-Snap-Ins erweitern. In diesem Abschnitt wird auch beschrieben, was erforderlich ist, um die Benutzeroberflächenerweiterungen auf den Desktops des Benutzers bereitzustellen.
