# <a name="using-standard-dialog-boxes-to-handle-objects-in-active-directory-domain-services"></a>Verwenden von Standard Dialogfeldern zum Behandeln von Objekten in Active Directory Domain Services

Windows stellt eine allgemeine Dialog Feld Bibliothek für häufige Vorgänge bereit, wie z. b. Datei öffnen, Datei suchen und so weiter. Ab Windows 2000 stellt das System Dialogfelder bereit, die für allgemeine Benutzeroberflächen Vorgänge in Active Directory Domain Services verwendet werden können.

Die folgenden Dialogfelder sind für die Verwendung mit Active Directory Domain Services verfügbar:

-   [Verzeichnis Objektauswahl](directory-object-picker.md)
-   [Domänen Browser](domain-browser.md)
-   [Container Browser](container-browser.md)
-   [Eigenschaften Blätter für das Active Directory von Benutzern und Computern](active-directory-users-and-computers-property-sheets.md)

Außerdem können Active Directory Domain Services Erstellungs Assistenten auch aus einer Anwendung aufgerufen werden. Weitere Informationen finden Sie unter [Aufrufen von Erstellungs-Assistenten aus Ihrer Anwendung](invoking-creation-wizards-from-your-application.md).
