# <a name="directory-manager-errors"></a>Verzeichnis-Manager-Fehler

Dieses Thema enthält die Rückgabewerte für Verzeichnis-Manager-Fehler von Funktionen in Active Directory Domain Services.



| Wert                    | BESCHREIBUNG                                              |
|--------------------------|----------------------------------------------------------|
| **hrpmrecdeleted**       | Der Datensatz wurde gelöscht.                                  |
| **hrrestingversions**  | Es sind verbleibende Arbeit im Leerlauf vorhanden.                            |
| **hrfldkeyteobig**       | Der Schlüssel wurde abgeschnitten, weil er mehr als 255 Bytes groß ist. |
| **hrfldflomanysegments** | Es sind zu viele Schlüsselsegmente vorhanden.                         |
| **hrfldnullkey**         | Der Schlüssel ist **null**.                                     |
