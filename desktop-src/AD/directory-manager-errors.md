# <a name="directory-manager-errors"></a>Verzeichnis-Manager-Fehler

Dieses Thema enthält die Rückgabewerte für Directory Manager-Fehler von Funktionen in Active Directory Domain Services.



| Wert                    | Beschreibung                                              |
|--------------------------|----------------------------------------------------------|
| **hrPMRecDeleted**       | Der Datensatz wurde gelöscht.                                  |
| **hrRemainingVersions**  | Es verbleiben Leerlaufarbeiten.                            |
| **hrFLDKeyTooBig**       | Der Schlüssel wurde abgeschnitten, da er mehr als 255 Bytes beträgt. |
| **hrFLDTooManySegments** | Es gibt zu viele Schlüsselsegmente.                         |
| **hrFLDNullKey**         | Der Schlüssel ist **NULL.**                                     |
