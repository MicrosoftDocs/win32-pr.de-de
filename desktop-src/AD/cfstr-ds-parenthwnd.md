# <a name="cfstr_ds_parenthwnd"></a>cfstr \_ DS- \_ Klammern

<dl> <dt>

<span id="CFSTR_DS_PARENTHWND"></span><span id="cfstr_ds_parenthwnd"></span>**cfstr \_ DS- \_ Klammern**
</dt> <dd> <dl> <dt>

"Dsadminsamenthwndclipformat"
</dt> <dt>



Das Format **cfstr \_ DS- \_ Klammern** für die Zwischenablage bietet ein globales Speicher handle (**HGLOBAL**), das das Fenster Handle (**HWND**) des Haupt-MMC-Fensters enthält.

> [!Note]  
> Dieser Wert für die Zwischenablage ist in einer veröffentlichten Header Datei nicht definiert. Um diesen Wert für die Zwischenablage Format zu verwenden, definieren Sie ihn im exakten Format.
