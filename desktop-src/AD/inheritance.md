# <a name="inheritance-ad-ds"></a>Vererbung (AD DS)

Die [*Vererbung*](/previous-versions/windows/desktop/legacy/ms681910(v=vs.85)) ermöglicht, dass ein bestimmter ACE aus dem Container weitergegeben wird, in dem er auf alle untergeordneten Elemente des Containers angewendet wurde. Die Vererbung kann mit Delegierungen kombiniert werden, um in einem einzigen Vorgang Administratorrechte für eine gesamte Unterstruktur des Verzeichnisses zu erteilen.
