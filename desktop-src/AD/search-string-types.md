# <a name="initial-medial-and-final-search-strings"></a>Anfängliche, mediale und endgültige Suchzeichenfolgen

Es gibt drei Arten von Platzhaltersuchen, auf die in der gesamten Dokumentation zu Suchzeichenfolgenabfragen für Konstruktionen verwiesen wird. Bei diesen Platzhaltersuchen handelt es sich um die Zeichenfolgen initial-, medial- und final-search. In diesem Thema werden diese Suchvorgänge beschrieben.

## <a name="initial-search-strings"></a>Anfängliche Suchzeichenfolgen

Zeichenfolgen für die anfängliche Suche übereinstimmungen mit einem angegebenen Satz von Zeichen am Anfang einer Zeichenfolge, gefolgt von einem Platzhalter. Beispielsweise würde die Zeichenfolge für die anfängliche Suche `Act*` mit "Active Directory" übereinstimmen.

## <a name="medial-search-strings"></a>Medial-Search Zeichenfolgen

Medial-Suchzeichenfolgen übereinstimmungen mit einem angegebenen Satz von Zeichen in der Mitte einer Zeichenfolge, gefolgt von und/oder gefolgt von einem Platzhalter. Die medial-search-Zeichenfolgen `*Dir*` , und würden alle mit `*ive*Dir*` `*ve*Dir*tor*` "Active Directory" übereinstimmen.

## <a name="final-search-strings"></a>Zeichenfolgen für die endgültige Suche

Final-Search-Zeichenfolgen übereinstimmungen mit einem angegebenen Satz von Zeichen am Ende einer Zeichenfolge, der ein Platzhalter vorangegangen ist. Beispielsweise würde die endgültige Suchzeichenfolge `*ory` mit "Active Directory" übereinstimmen.
