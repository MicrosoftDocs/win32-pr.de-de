# <a name="initial-medial-and-final-search-strings"></a>Anfängliche, mediale und abschließende Such Zeichenfolgen

Es gibt drei Arten von Platzhalter suchen, auf die in der gesamten Dokumentation zum Erstellen von Such Zeichenfolgen-Abfragen verwiesen wird. Diese Platzhalter suchen sind: Initial-, medial-und Final-Search-Zeichen folgen. In diesem Thema werden diese Suchvorgänge beschrieben.

## <a name="initial-search-strings"></a>Anfängliche Such Zeichenfolgen

Such Zeichenfolgen entsprechen einem angegebenen Zeichensatz am Anfang einer Zeichenfolge, gefolgt von einem Platzhalter. Beispielsweise entspricht die Zeichenfolge für die anfängliche Suche `Act*` "Active Directory".

## <a name="medial-search-strings"></a>Medial-Search Zeichenfolgen

Medial-Such Zeichenfolgen entsprechen einem gegebenen Zeichensatz in der Mitte einer Zeichenfolge, dem ein Platzhalter vorangestellt ist und/oder darauf folgt. Die medialen Such Zeichenfolgen `*Dir*` , `*ive*Dir*` und `*ve*Dir*tor*` würden alle mit "Active Directory" vergleichen.

## <a name="final-search-strings"></a>Abschließende Such Zeichenfolgen

Final-Search-Zeichen folgen entsprechen einem angegebenen Zeichensatz am Ende einer Zeichenfolge, dem ein Platzhalter vorangestellt ist. Beispielsweise entspricht die Zeichenfolge für die abschließende Suche `*ory` "Active Directory".
