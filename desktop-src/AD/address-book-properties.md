# <a name="user-address-book-attributes"></a>Benutzer Adressbuch Attribute

Die Adressbuch Attribute werden verwendet, um zusätzliche Identifizierung und Informationen für einen Benutzer bereitzustellen. Der Inhalt wird durch den Benutzer und die Organisation des Benutzers definiert. Diese Attribute können über das Active Directory Benutzer-und Computer-Snap-in oder das Windows-Adressbuch (WAB) angezeigt und verwaltet werden.



| Attribut                         | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|-----------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **c**                             | Das Land/die Region in der Adresse des Benutzers. Das Land/die Region wird als der aus zwei Zeichen bestehende Länder-/Regionscode dargestellt, der auf ISO-3166 basiert. Gültige Codes finden Sie unter [Werte für CountryCode](values-for-the-countrycode-and-c-properties.md).<br/>                                                                                                                                                                                        |
| **erung**                            | Das Land oder die Region, in dem bzw. der der Benutzer ansässig ist.                                                                                                                                                                                                                                                                                                                                                                                              |
| **Anmerkungen**                         | Kommentar. Diese Zeichenfolge kann eine NULL-Zeichenfolge sein.                                                                                                                                                                                                                                                                                                                                                                                                  |
| **department**                    | Der Name der Abteilung, in der der Benutzer arbeitet.                                                                                                                                                                                                                                                                                                                                                                                          |
| **Beschreibung**                   | Die Beschreibung, die für den Benutzer angezeigt werden soll.                                                                                                                                                                                                                                                                                                                                                                                                      |
| **displayName**                   | Der im Adressbuch für einen bestimmten Benutzer angezeigte Name. Dies ist normalerweise die Kombination aus dem Vornamen, dem mittleren Anfangs-und Nachnamen des Benutzers.                                                                                                                                                                                                                                                                                        |
| **directReports**                 | Die Liste der Benutzer, die dem Benutzer direkt einen Bericht erstatten. Die als Berichte aufgeführten Benutzer sind diejenigen, für die das **Manager** -Attribut auf diesen Benutzer festgelegt ist. Jedes Element in der Liste ist ein verknüpfter Verweis auf das Objekt, das den Benutzer darstellt. Daher aktualisiert der Active Directory Server dieses Attribut automatisch, wenn das **Manager** -Attribut eines Benutzers diesen Benutzer als Manager hinzufügt oder entfernt. Die Elemente werden als Distinguished Names dargestellt. |
| **facsimileTelephoneNumber**      | Die geschäftliche Faxnummer des Benutzers.                                                                                                                                                                                                                                                                                                                                                                                      |
| **givenName**                     | Der Vorname des Benutzers.                                                                                                                                                                                                                                                                                                                                                                                                      |
| **homePhone**                     | Die Haupt Telefonnummer des Benutzers für den Benutzer.                                                                                                                                                                                                                                                                                                                                                                                               |
| **Initialen**                      | Die Initialen für Teile des vollständigen Namens des Benutzers. Diese kann als naher Ausgangswert im Windows-Adressbuch verwendet werden.                                                                                                                                                                                                                                                                                                                           |
| **ipPhone**                       | Wird von Telefoniediensten verwendet.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **l**                             | Der Ort, z. b. der Ort oder die Stadt, in der Adresse des Benutzers.                                                                                                                                                                                                                                                                                                                                                                                |
| **managedObjects**                | Die Liste der Objekte, die vom Benutzer verwaltet werden. Bei den aufgelisteten Objekten handelt es sich um das Attribut **ManagedBy** , das auf diesen Benutzer festgelegt ist. Jedes Element in der Liste ist ein verknüpfter Verweis auf das verwaltete Objekt. Daher aktualisiert der Active Directory Server das **managedobjects** -Attribut automatisch, wenn das **ManagedBy** -Attribut eines Objekts diesen Benutzer als Manager hinzufügt oder entfernt. Die Elemente werden als Distinguished Names dargestellt.  |
| **manager**                       | Der Benutzer, der der Manager des Benutzers ist. Das Benutzerobjekt des Managers enthält ein **DirectReports** -Attribut, das Verweise auf alle Benutzer Objekte enthält, deren **Manager** -Attribut auf das Benutzerobjekt des Managers festgelegt ist.                                                                                                                                                                                                                     |
| **Funk**                        | Die primäre Mobiltelefonnummer des Benutzers.                                                                                                                                                                                                                                                                                                                                                                                           |
| **otherfaksimiletelefonienumber** | Die Liste der Telefonnummern alternativer faxcomputer für den Benutzer.                                                                                                                                                                                                                                                                                                                                                                         |
| **otherIpPhone**                  | Wird von Telefoniediensten verwendet.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **otherMobile**                   | Die Liste der alternativen Mobiltelefonnummern für den Benutzer.                                                                                                                                                                                                                                                                                                                                                                                |
| **otherPager**                    | Die Liste der alternativen Pager-Telefonnummern für den Benutzer.                                                                                                                                                                                                                                                                                                                                                                                   |
| **otherTelephone**                | Die Liste der alternativen Geschäfts Telefonnummern für den Benutzer.                                                                                                                                                                                                                                                                                                                                                                                |
| **Pager**                         | Die primäre Pager-Telefonnummer für den Benutzer.                                                                                                                                                                                                                                                                                                                                                                                              |
| **physicalDeliveryOfficeName**    | Position des Büros am Unternehmensstandort des Benutzers.                                                                                                                                                                                                                                                                                                                                                                                          |
| **PostalAddress**                 | Die Postadresse des Benutzers.                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **postalCode**                    | Die Postleitzahl der Anschrift des Benutzers. Die Postleitzahl ist spezifisch für das Land oder die Region des Benutzers. In den USA enthält dieses Attribut die Postleitzahl.                                                                                                                                                                                                                                                               |
| **postOfficeBox**                 | Die Anzahl oder der Bezeichner für das Feld Postbüro des Benutzers.                                                                                                                                                                                                                                                                                                                                                                                       |
| **sn**                            | Der Nachname (oder Familienname) des Benutzers.                                                                                                                                                                                                                                                                                                                                                                                                |
| **St**                            | Das Bundesland oder der Kanton in der Adresse des Benutzers.                                                                                                                                                                                                                                                                                                                                                                                                  |
| **streetAddress**                 | Die Straße in der geschäftlichen Anschrift des Benutzers.                                                                                                                                                                                                                                                                                                                                                                                           |
| **telephoneNumber**               | Die primäre geschäftliche Telefonnummer des Benutzers.                                                                                                                                                                                                                                                                                                                                                                                 |
| **title**                         | Die Position des Benutzers. Dieses Attribut wird häufig verwendet, um die formale Berufsbezeichnung anzugeben, wie z. b. Senior Programmer, anstelle einer berufsklasse, z. b. Programmierer. Sie wird in der Regel nicht für "Suffix"-Titel wie z. b. Esq verwendet. oder DDS. Beispiele: Managing Director, Programmierer II, Associate Professor und Development Lead.<br/>                                                                                                    |
| **url**                           | Die Liste der URLs für die alternativen Webseiten des Benutzers.                                                                                                                                                                                                                                                                                                                                                                                           |
| **wWWHomePage**                   | Die URL für die primäre Webseite des Benutzers.                                                                                                                                                                                                                                                                                                                                                                                                       |
