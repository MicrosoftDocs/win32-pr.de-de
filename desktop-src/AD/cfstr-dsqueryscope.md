# <a name="cfstr_dsqueryscope"></a>cfstr \_ dsqueryscope

<dl> <dt>

<span id="CFSTR_DSQUERYSCOPE"></span><span id="cfstr_dsqueryscope"></span>**cfstr \_ dsqueryscope**
</dt> <dd> <dl> <dt>

"Dsqueryscope"
</dt> <dt>



Das Zwischenablage Format von **cfstr \_ dsqueryscope** wird von dem [**IDataObject**](/windows/win32/api/objidl/nn-objidl-idataobject) unterstützt, das von der [**icommonquery:: openquerywindow**](/windows/win32/api/cmnquery/nf-cmnquery-icommonquery-openquerywindow) -Methode bereitgestellt wird. Das Zwischenablage Format von **cfstr \_ dsqueryscope** stellt ein globales Speicher handle (**HGLOBAL**) bereit, das eine Zeichenfolge enthält, die den Abfrage Bereich angibt.
