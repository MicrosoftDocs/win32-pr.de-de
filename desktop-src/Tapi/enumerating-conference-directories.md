# <a name="enumerating-conference-directories"></a>Auflisten der Konferenz Verzeichnisse

\[ Rendezvous-Steuerelemente und Schnittstellen für die IP-telefoniekonferenz sind nicht für die Verwendung in Windows Vista, Windows Server 2008 und nachfolgenden Versionen des Betriebssystems verfügbar. Die RTC-Client-API bietet eine ähnliche Funktionalität.\]

Das folgende Code Fragment veranschaulicht die Enumeration von Konferenzen auf einem angegebenen ILS-Server. Dieses Fragment geht davon aus, dass bereits [eine Verbindung mit einem ILS-Server](connecting-to-an-ils-server.md) hergestellt wurde.


```C++
SysFreeString( bNameToSearch );
```
