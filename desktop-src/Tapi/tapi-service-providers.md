# <a name="tapi-service-providers"></a>TAPI-Dienstanbieter

TAPI-Dienstanbieter sind eine Abstraktionsschicht zwischen TAPI-Anwendungen und den zugrunde liegenden Hardware- und Transportprotokollen. Ein [Telefoniedienstanbieter (Telefoniedienstanbieter,](telephony-service-providers-start-page.md) TSP) stellt Anrufsteuerungen bereit, und ein [Mediendienstanbieter (Media Service Provider,](./media-service-providers-start-page.md) MSP) stellt eine detaillierte Kontrolle über die Medien eines Anrufs bereit. Ein TSP-/MSP-Paar kann die erforderlichen Funktionen für Anwendungen implementieren, die von einfacher Telefonie bis hin zu IP-Multicast-Multimediakonferenzen reichen.
