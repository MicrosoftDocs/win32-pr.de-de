# <a name="tapi-service-providers"></a>TAPI-Dienstanbieter

TAPI-Dienstanbieter sind eine Abstraktions Ebene zwischen TAPI-Anwendungen und den zugrunde liegenden Hardware-und Transportprotokollen. Ein [Telefoniedienstanbieter](telephony-service-providers-start-page.md) (TSP) stellt Callcenter-Steuerelemente bereit, und ein [Medien Dienstanbieter](./media-service-providers-start-page.md) (MSP) bietet eine ausführliche Kontrolle über die Medien eines Aufrufes. Ein TSP-/MSP-Paar kann die Funktionen implementieren, die für Anwendungen benötigt werden, die von der einfachen Telefonie bis hin zu IP-Multicast-Multimedia
