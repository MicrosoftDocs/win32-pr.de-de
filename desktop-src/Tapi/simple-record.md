# <a name="simple-record"></a>Einfacher Datensatz

Gehen Sie folgendermaßen vor, um eine einfache Aufzeichnung von Streams auszuführen:

-   Wählen Sie auf der Rufebene das Terminal für den Datei Daten Satz aus
-   Erstellen Sie das Terminal für den Datei Daten Satz.
-   Legen Sie die Eigenschaften – Dateiname und Typ des Speichers fest.
-   Wählen Sie im-Befehl das Terminal für den Datei Daten Satz aus.
-   Ruft die [**Start**](/windows/desktop/api/tapi3if/nf-tapi3if-itmediacontrol-start) -Methode für das Terminal Datei Daten Satz ab, um die Aufzeichnung von Streams zu starten.
