# <a name="acd-agent-client"></a>ACD-Agent-Client

Ein Callcenter-Agent ist für das empfangen und Verarbeiten eingehender Anrufe und das gleichzeitig ausgehende Aufrufen von Kunden oder Chancen zuständig. Eine ACD-Agent-Client Anwendung stellt die Kommunikation mit der [ACD-Proxy](acd-proxy.md) Anwendung bereit und stellt Informationen bereit, wie z. b. ob der Agent für Aufrufe verfügbar ist und welche Arten von Aufrufen der Agent verarbeiten kann.
