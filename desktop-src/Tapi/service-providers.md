<span data-ttu-id="52517-116">Die Benutzeroberflächen-DLL des Telefoniedienstanbieters wird von TAPI in den Prozess einer Anwendung geladen, die eine der Dienstanbieter Funktionen aufruft, die ein Dialogfeld anzeigen können (z.
