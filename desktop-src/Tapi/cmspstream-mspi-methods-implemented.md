# <a name="cmspstream-mspi-methods-implemented"></a>Implementierte cmspstream-MSPi-Methoden



| Cmspstream-MSPi-Methoden                                   | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|-----------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_mediaType erhalten**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_mediatype)          | Ruft den [**Medientyp**](tapimediatype--constants.md)des Streams ab.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**\_Richtung**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_direction)          | Ruft die [**Terminal Richtung**](/windows/desktop/api/Tapi3if/ne-tapi3if-terminal_direction)des Streams ab.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**Selectterminal**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-selectterminal)         | Wählt ein [**itterminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) -Objekt auf dem Stream aus. Erhält die Sperre in der Funktion. Diese Basisklassen Methode weiß nur, dass die Liste der Terminals beibehalten werden soll. Es wird erwartet, dass die abgeleitete Klasse diese Methode überschreibt, um zusätzliche Funktionen bereitzustellen (z. b. das Beenden und Neustarten des Streams, wenn ein Terminal ausgewählt wird, während der Stream ausgeführt wird). Die Basisklassen Implementierung überprüft, ob der Terminal Zeiger auf einen gültigen Speicher zeigt, ruft die [**itterminalcontrol**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalcontrol) -Schnittstelle aus dem Terminal ab, ruft das Adress Handle aus dem Terminal ab und überprüft, ob es mit dem Adress Handle des MSP übereinstimmt, überprüft, ob das Terminal bereits in der Liste des Streams vorhanden ist Wenn einer der Tests fehlschlägt, wird TAPI \_ E \_ invalidterminal zurückgegeben. |
| [**Unselectterminal**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-unselectterminal)     | Hebt die Auswahl des Terminals aus dem Stream auf. Erhält die Sperre in der Funktion. Sucht das Terminal in der Liste der Terminals des Streams, entfernt es aus der Liste und gibt es frei. Wenn das Terminal nicht gefunden wird, wird TAPI \_ E \_ invalidterminal zurückgegeben.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**Enumerateterminals**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-enumerateterminals) | Listet die für den Stream ausgewählten Terminals auf. Es handelt sich um eine Momentaufnahme der Terminals an dieser Stelle. Erhält die Sperre in der Funktion.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**\_Terminals erhalten**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_terminals)          | Erstellt eine Auflistung der dem aktuellen Stream zugeordneten Terminals. Wird für Automatisierungs Client Anwendungen bereitgestellt, wie z. b. in Microsoft Visual Basic. Es handelt sich um eine Momentaufnahme der Terminals an dieser Stelle. Erhält die Sperre in der Funktion.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**Startstream**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-startstream)               | Startet den Stream. Ruft **IMediaControl:: Run** im Filter Diagramm auf.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**Pausegstream**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-pausestream)               | Hält den Datenstrom an. Ruft **IMediaControl::P ause** im Filter Diagramm auf.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**Stopstream**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-stopstream)                 | Beendet den Stream. Ruft **IMediaControl:: Stopps** im Filter Diagramm auf.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
