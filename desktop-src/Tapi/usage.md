# <a name="usage"></a>Verwendung

Es gibt zwei allgemeine Verwendungs Fälle für die Schnittstelle der TSP-UI-dll: Wenn die Anwendung eine Funktion aufruft, die zum Generieren der Benutzeroberfläche entworfen wurde, und wenn der Dienstanbieter während der Verarbeitung einer Funktion, die nicht als generierende Benutzeroberfläche definiert ist, die Benutzeroberfläche spontan generieren muss.
