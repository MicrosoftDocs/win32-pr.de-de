# <a name="usage"></a>Verwendung

Es gibt zwei allgemeine Fälle der Verwendung der TSP-UI-DLL-Schnittstelle: wenn die Anwendung eine Funktion aufruft, die zum Generieren der Benutzeroberfläche entworfen wurde, und wenn der Dienstanbieter während der Verarbeitung einer Funktion, die nicht als generierende Benutzeroberfläche definiert ist, eine Benutzeroberfläche generieren muss.
