# <a name="terminal"></a>Terminal

Ein Terminal ist eine Quelle oder ein Renderer für eine bestimmte Adresse und einen Medientyp. Das Terminal Konzept wird primär in TAPI 3 verwendet und ist nur für TAPI 2-Anwendungen von Bedeutung. Weitere Informationen finden Sie in der Übersicht über das [Terminal Objekt](terminal-object.md) von TAPI 3.
