<span id="tapi2.vcomm_tapgloss"></span><span id="TAPI2.VCOMM_TAPGLOSS"></span>**VCOMM**
</dt> <dd>

Ein Kommunikationsgerätetreiber in TAPI. VCOMM kann TAPI-Diensten den geschützten Modus zuweisen und ermöglicht Windows Anwendungen und Treibern die Verwendung von Ports und Modems. Weitere Informationen finden Sie unter [*UniModem*](u-tapgloss.md).

</dd> <dt>

<span id="tapi2.voice_signal_tapgloss"></span><span id="TAPI2.VOICE_SIGNAL_TAPGLOSS"></span>**Sprachsignal**
</dt> <dd>

Gilt als jedes Signal, das über einen 3,1-kHz-Bandbreitenkanal übertragen werden kann.
