# <a name="linerequestmode_-constants"></a>Linerequestmode- \_ Konstanten

Die Bitflag-Konstanten von **linerequestmode** beschreiben verschiedene Arten von Telefonieanforderungen, die von einer Anwendung an eine andere gestellt werden können.

<dl> <dt>

<span id="LINEREQUESTMODE_DROP"></span><span id="linerequestmode_drop"></span>**linerequestmode- \_ Drop**
</dt> <dd> <dl> <dt>



Eine [tapirequestdrop](/windows/desktop/api/Tapi/nf-tapi-tapirequestdrop) -Anforderung.


</dt> </dl> </dd> <dt>

<span id="LINEREQUESTMODE_MAKECALL"></span><span id="linerequestmode_makecall"></span>**linerequestmode \_ MakeCall**
</dt> <dd> <dl> <dt>



Eine [tapirequestmakecall](/windows/desktop/api/Tapi/nf-tapi-tapirequestmakecall) -Anforderung.


</dt> </dl> </dd> <dt>

<span id="LINEREQUESTMODE_MEDIACALL"></span><span id="linerequestmode_mediacall"></span>**linerequestmode- \_ mediacall**
</dt> <dd> <dl> <dt>



Diese Konstante ist veraltet. Darf nicht verwendet werden.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Bemerkungen

Keine Erweiterbarkeit. Alle 32 Bits sind reserviert.
