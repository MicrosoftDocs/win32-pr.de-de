# <a name="linerequestmode_-constants"></a>\_LINEREQUESTMODE-Konstanten

Die  LINEREQUESTMODE-Bitflagkonstanten beschreiben verschiedene Arten von Telefonieanforderungen, die von einer Anwendung in eine andere gestellt werden können.

<dl> <dt>

<span id="LINEREQUESTMODE_DROP"></span><span id="linerequestmode_drop"></span>**LINEREQUESTMODE \_ DROP**
</dt> <dd> <dl> <dt>



Eine [tapiRequestDrop-Anforderung.](/windows/desktop/api/Tapi/nf-tapi-tapirequestdrop)


</dt> </dl> </dd> <dt>

<span id="LINEREQUESTMODE_MAKECALL"></span><span id="linerequestmode_makecall"></span>**LINEREQUESTMODE \_ MAKECALL**
</dt> <dd> <dl> <dt>



Eine [tapiRequestMakeCall-Anforderung.](/windows/desktop/api/Tapi/nf-tapi-tapirequestmakecall)


</dt> </dl> </dd> <dt>

<span id="LINEREQUESTMODE_MEDIACALL"></span><span id="linerequestmode_mediacall"></span>**LINEREQUESTMODE \_ MEDIACALL**
</dt> <dd> <dl> <dt>



Diese Konstante ist veraltet. Darf nicht verwendet werden.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Hinweise

Keine Erweiterbarkeit. Alle 32 Bits sind reserviert.
