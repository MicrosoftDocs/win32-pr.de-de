# <a name="remote-tsp"></a>Remote-TSP

Der Remote-TSP (Remotesp.tsp) bietet Zugriff auf Kommunikationsressourcen, die nicht lokal auf dem aktuellen Computer verfügbar sind, z. B. pbx. Wenn der Remote-TSP so konfiguriert ist, dass er auf einen bestimmten Server verweisen kann, werden Ressourcen in einer lokalen Anwendung so angezeigt, als ob sie auf dem Computer des Benutzers vorhanden waren.

Tcmsetup ist ein Befehlszeilenprogramm, das die Konfiguration eines Remote-TSP ermöglicht. Weitere Informationen zu diesem Hilfsprogramm finden Sie Windows Hilfe oder im Windows Resource Kit.

Der Remote-TSP wird mit den Betriebssystemen Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Me, Windows 98 und Windows 95 installiert.
