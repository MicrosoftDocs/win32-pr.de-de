# <a name="remote-tsp"></a>Remote-TSP

Der Remote-TSP (remotesp. TSP) ermöglicht den Zugriff auf Kommunikationsressourcen, die für den aktuellen Computer nicht lokal sind, z. b. eine PBX. Wenn der Remote-TSP konfiguriert ist, um auf einen bestimmten Server zu verweisen, werden Ressourcen in einer lokalen Anwendung angezeigt, als ob Sie auf dem Computer des Benutzers vorhanden wären.

Tcmsetup ist ein Befehlszeilen-Hilfsprogramm, das die Konfiguration eines Remote-TSP ermöglicht. Weitere Informationen zu diesem Hilfsprogramm finden Sie in der Windows-Hilfe oder im Windows Resource Kit.

Der Remote-TSP wird mit den Betriebssystemen Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Me, Windows 98 und Windows 95 installiert.
