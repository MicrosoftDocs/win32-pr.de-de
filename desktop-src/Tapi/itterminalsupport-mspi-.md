# <a name="itterminalsupport-mspi"></a>Itterminalsupport (MSPi)

Die [**itterminalsupport**](/windows/win32/api/tapi3if/nn-tapi3if-itterminalsupport) -Schnittstelle wird für ein [Adress Objekt](address-object.md) verfügbar gemacht, wenn ein MSP vorhanden ist. Mit den Methoden dieser Schnittstelle kann eine Anwendung verfügbare Terminals ermitteln und/oder eine erstellen und Zeiger auf erforderliche [Terminal Objekte](terminal-object.md)erhalten.

Die [**itterminalsupport**](/windows/win32/api/tapi3if/nn-tapi3if-itterminalsupport) -Schnittstelle wird von einem MSP implementiert und ist nicht verfügbar, wenn der Adresse kein Medien Dienstanbieter zugeordnet ist. Ausführliche Informationen zu dieser Schnittstelle finden Sie im Abschnitt **itterminalsupport** im Abschnitt MSP-Schnittstelle.
