# <a name="call-center-controls"></a>Callcentersteuerelemente

Ein Callcenter ist ein Ort, an dem Agents oder Operatoren an Telefon Telefonen entweder ausgehende Telefonanrufe tätigen oder eingehende Anrufe tätigen. Beispielsweise verwendet eine Bank oder ein Kreditkartenunternehmen ein Callcenter, um Konto Anfragen zu verarbeiten.

TAPI 3. x implementiert Objekte, Schnittstellen und Methoden, die dem Programmierer beim Erstellen von callcenteranwendungen helfen. Aus Sicht der Programmierung werden callcentervorgänge mithilfe von TAPI im folgenden Diagramm dargestellt.

![Callcenter-Vorgänge mit TAPI](images/callcent.png)

Ein Programmierer, der zum Erstellen von callcenteranwendungen interessiert ist, möchte möglicherweise auch die TAPI 2,2-Funktionen kennenlernen, die Teile einer callcentersuite aktivieren können. Siehe TAPI 2,2 [callcentersteuerelement](./call-center-control.md).
