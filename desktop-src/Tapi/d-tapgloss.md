<dl> <dt>

<span id="tapi2.d_channel_tapgloss"></span><span id="TAPI2.D_CHANNEL_TAPGLOSS"></span>**D-Kanal**
</dt> <dd>

Ein Datenkanal der ISDN-Schnittstelle. Es ist einer der drei Standardkanäle auf einer ISDN-Basislinie. In einer BRI-ISDN-Linie kann ein D-Kanal Signalinformationen und Paketdaten mit geringer Geschwindigkeit mit einer Rate von 16.000 Bit/s ausführen. In einer PRI-ISDN-Zeile wird der D-Kanal mit 64.000 Bit/s ausgeführt. Weitere Informationen finden Sie unter [*Basic Rate Interface (BRI-ISDN)*](b-tapgloss.md), [*B Channel*](b-tapgloss.md), [*Primary Rate Interface (PRI-ISDN)*](p-tapgloss.md).

</dd> <dt>

<span id="tapi2.device_class_tapgloss"></span><span id="TAPI2.DEVICE_CLASS_TAPGLOSS"></span>**Geräteklasse**
</dt> <dd>

Eine Gruppe verwandter physischer Geräte oder Gerätetreiber, über die Anwendungen die Informationen oder Daten, die einen-Rückruf bilden, senden und empfangen. Eine Liste finden Sie unter [TAPI-Geräteklassen](tapi-device-classes.md).

</dd> <dt>

<span id="tapi2.did_tapgloss"></span><span id="TAPI2.DID_TAPGLOSS"></span>**DID**
</dt> <dd>

Weitere Informationen finden Sie unter *direkte Wahl der inneren Wahl (DID)*.

</dd> <dt>

<span id="tapi2.direct_inward_dialing_did__tapgloss"></span><span id="TAPI2.DIRECT_INWARD_DIALING_DID__TAPGLOSS"></span>**direkte Wähl Wahl (DID)**
</dt> <dd>

Eine Möglichkeit, die praktisch alle modernen PBXs bereitstellt, mit denen ein externer Aufrufer eine interne Erweiterung erreichen kann, indem er eine standardmäßige siebenstellige Telefonnummer einwählt, ohne einen Operator oder eine Aufsicht durchlaufen zu müssen. Siehe [*Private Branch Exchange (PBX)*](p-tapgloss.md).

</dd> </dl>
