## <a name="tapi-newsgroups"></a>TAPI-Newsgroups

Usenet-Newsgroups können gute Foren für den Austausch von Informationen und unterstützung beim Debuggen von Code sein. Wenn Sie mit Usenet nicht vertraut sind, geben Sie "Usenet Newsgroups" in Ihre bevorzugte Suchmaschine ein.

Microsoft hostet eine Reihe von Newsgroups, darunter eine, die auf TAPI spezialisiert ist.

[https://msdn.microsoft.com/newsgroups](https://www.microsoft.com/communities/newsgroups/)

Navigieren Sie im linken Bereich zu Windows Entwicklung, klicken Sie auf Windows SDK, und wählen Sie win32.programmer.tapi aus.
