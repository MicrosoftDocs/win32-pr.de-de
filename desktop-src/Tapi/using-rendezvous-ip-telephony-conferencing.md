# <a name="using-rendezvous-ip-telephony-conferencing"></a>Verwenden von Rendezvous IP-Telefoniekonferenzen

\[Rendezvous-IP-Telefoniekonferenz-Steuerelemente und -Schnittstellen sind nicht für die Verwendung in Windows Vista, Windows Server 2008 und nachfolgenden Versionen des Betriebssystems verfügbar. Die RTC-Client-API bietet ähnliche Funktionen.\]

Rendezvous besteht aus drei grundlegenden Schnittstellen:

-   [Verzeichnissteuerelemente](directory-controls.md) bieten eine Möglichkeit zum Ermitteln, Erstellen oder Ändern der Konferenz- und Benutzerinformationen auf einem ILS- oder NTDS-Server.
-   [Konferenzblobsteuerelemente](conference-blob-controls.md) ermöglichen anbieterspezifische Änderungen an einer Konferenz.
-   [Multicast-COM-Schnittstellen](multicast-com-interfaces.md) bieten eine Möglichkeit, Adressen für die Verwendung durch Multicastkonferenzen zu reservieren.
