# <a name="using-rendezvous-ip-telephony-conferencing"></a>Verwenden Rendezvous-IP-telefoniekonferenzen

\[ Rendezvous-Steuerelemente und Schnittstellen für die IP-telefoniekonferenz sind nicht für die Verwendung in Windows Vista, Windows Server 2008 und nachfolgenden Versionen des Betriebssystems verfügbar. Die RTC-Client-API bietet eine ähnliche Funktionalität.\]

Rendezvous besteht aus drei grundlegenden Sätzen von Schnittstellen:

-   [Verzeichnis Steuerelemente](directory-controls.md) bieten eine Möglichkeit, die Konferenz-und Benutzerinformationen auf einem ILS-oder NTDS-Server zu ermitteln, zu erstellen oder zu ändern.
-   Mit [Konferenz-BLOB](conference-blob-controls.md) -Steuerelementen können anbieterspezifische Änderungen an einer Konferenz vorgenommen werden.
-   [Multicast-com-Schnittstellen](multicast-com-interfaces.md) bieten eine Möglichkeit zum Reservieren von Adressen für Multicast Konferenzen.
