# <a name="tapi-22-overview"></a>TAPI 2,2 (Übersicht)

Die Microsoft Telefonie Application Programming Interface (TAPI) Version 2,2 ist eine C-basierte Schnittstelle, mit der Entwickler eine Reihe von Kommunikationsanwendungen implementieren können. Das folgende Material stellt Übersichts Informationen für die Verwendung von TAPI 2,2 dar. Das primäre Übersichts Material ist im [Programmiermodell von Microsoft Telefonie](./microsoft-telephony-programming-model.md)enthalten. Es wird davon ausgegangen, dass dieses Material vertraut ist.

-   Die [TAPI-Versions](tapi-versioning.md) Verwaltung deckt die verfügbaren Versionen von TAPI ab, wie Sie verteilt werden und wie die Anwendung die zu verwendende Version von TAPI aushandeln kann.
-   Der [Sitzungs Zugriff](session-access.md) deckt einige für TAPI 2,2 (TAPI/C) spezifische aufrufssitzungsmaterialien ab. Weitere Informationen finden Sie unter allgemeine Übersicht über die [Sitzungssteuerung](./session-control.md) für den primären Verweis auf Aufrufe von Sitzungs Vorgängen und Informationen.
-   Der [Gerätezugriff](device-access.md) deckt die spezifischen Funktionen von TAPI 2,2 (TAPI/C) ab. Weitere Informationen zu Geräteinformationen und-Vorgängen finden Sie in der allgemeinen Übersicht über die [Gerätesteuerung](./device-control.md) .
-   Der [Medien Zugriff](media-access.md) deckt Material zu den Medieninformationen und Steuerelementen ab, die für eine TAPI 2,2-Anwendung verfügbar sind. Die Media Service-Anbieter erhalten Zugriff auf ausführlichere Steuerelemente, aber nur TAPI/com-Anwendungen können ihre Funktionen verwenden. Weitere Informationen finden Sie unter Informationen zum [Media Service-Anbieter](./about-the-media-service-provider-msp-.md) .
-   Das [Callcenter-Steuer](call-center-control.md) Element deckt das Material zum Aufbauen von ACD-Anwendungen (Automatic calldistribution) mithilfe von TAPI 2,2 ab. Informationen zu TAPI 3. x finden Sie unter [callcentersteuerelemente](./about-call-center-controls.md).
-   [Telefongeräte](phone-devices.md) umfassen die zusätzlichen Telefongeräte-Steuerelemente, die von TAPI 2,2 (TAPI/C) bereitgestellt werden.
