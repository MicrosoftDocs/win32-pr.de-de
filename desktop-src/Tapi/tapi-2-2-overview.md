# <a name="tapi-22-overview"></a>ÜBERSICHT ÜBER TAPI 2.2

Die TapI-Version 2.2 (Microsoft Telefonieanwendungsprogrammierschnittstelle) ist eine C-basierte Schnittstelle, mit der Entwickler eine Reihe von Kommunikationsanwendungen implementieren können. Das folgende Material enthält Übersichtsinformationen zur Verwendung von TAPI 2.2. Primäres Übersichtsmaterial ist im [Microsoft-Telefonieprogrammiermodell](./microsoft-telephony-programming-model.md)enthalten. Es wird vorausgesetzt, dass Sie mit diesem Material vertraut sind.

-   Die [TAPI-Versionierung](tapi-versioning.md) deckt die verfügbaren Versionen von TAPI ab, wie sie verteilt werden und wie Ihre Anwendung die version von TAPI aushandeln kann, die verwendet werden soll.
-   [Der Sitzungszugriff](session-access.md) deckt einige für TAPI 2.2 (TAPI/C) spezifisches Sitzungsmaterial für Anrufe ab. Eine primäre Referenz zu Aufrufen von Sitzungsvorgängen und Informationen finden Sie in der allgemeinen Übersicht zur [Sitzungssteuerung.](./session-control.md)
-   [Der Gerätezugriff](device-access.md) deckt funktionen ab, die für TAPI 2.2 (TAPI/C) spezifisch sind. Eine primäre Referenz zu Geräteinformationen und -vorgängen finden Sie in der allgemeinen Übersicht zur [Gerätesteuerung.](./device-control.md)
-   [Der Medienzugriff](media-access.md) deckt Material zu den Medieninformationen und Steuerelementen ab, die für eine TAPI 2.2-Anwendung verfügbar sind. Die Mediendienstanbieter bieten Ihnen Zugriff auf detailliertere Steuerelemente, aber nur TAPI 3-Anwendungen (TAPI/COM) können ihre Funktionen nutzen. Weitere Informationen finden Sie [unter Informationen zum Media Service Provider.](./about-the-media-service-provider-msp-.md)
-   [Die Callcentersteuerung](call-center-control.md) deckt Material zum Erstellen von ACD-Anwendungen (Automatic Call Distribution) mit TAPI 2.2 ab. Informationen zu TAPI 3.x finden Sie unter [Callcenter-Steuerelemente.](./about-call-center-controls.md)
-   [Telefon Geräte](phone-devices.md) deckt die zusätzlichen Steuerelemente für Telefongeräte ab, die von TAPI 2.2 (TAPI/C) bereitgestellt werden.
