# <a name="tapi-21"></a>TAPI 2.1

TAPI 2.1 bietet eine Ergänzung zur vorherigen TAPI 2.0-Spezifikation: Client-/Serverfunktionalität. Mit TAPI 2.1 und höher kann Telefoniehardware auf einem Telefonieserver installiert werden, auf den von jedem Computer in einem Netzwerk aus zugegriffen werden kann. Mit einem neuen Satz von Funktionen können Verwaltungs-DLLs geschrieben werden, um zu steuern, wer Zugriff auf Remoteressourcen hat.

**Windows Me/98/95:** TAPI 2.1 bietet Unterstützung für 32-Bit-TSPs. Die TAPI 2.1-Systembinärdateien unterstützen bei der Installation Folgendes:

-   32-Bit-TAPI 1.3-, TAPI 1.4- und TAPI 2.x-Anwendungen
-   32-Bit TAPI 2.x TSPs
-   16-Bit TAPI 1.3- und TAPI 1.4-Anwendungen
-   16-Bit-TAPI 1.3- und TAPI 1.4-TSPs
