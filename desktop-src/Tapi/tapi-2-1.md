# <a name="tapi-21"></a>TAPI 2,1

TAPI 2,1 bietet eine Ergänzung zur vorherigen TAPI 2,0-Spezifikation: Client/Server-Funktionalität. Mit TAPI 2,1 und höher kann die Telefoniehardware auf einem Telefonieserver installiert werden, auf den von einem beliebigen Computer in einem Netzwerk zugegriffen werden kann. Ein neuer Funktions Satz ermöglicht das Schreiben von Verwaltungs-DLLs, um zu steuern, wer Zugriff auf Remote Ressourcen hat.

**Windows Me/98/95:** TAPI 2,1 bietet Unterstützung für 32-Bit-TSPS. Die TAPI 2,1-System Binärdateien, bei installierter Unterstützung:

-   32-Bit TAPI 1,3, TAPI 1,4 und TAPI 2. x-Anwendungen
-   32-Bit TAPI 2. x-TSPS
-   16-Bit-TAPI 1,3-und TAPI 1,4-Anwendungen
-   16-Bit-TAPI 1,3-und TAPI 1,4-TSPS
