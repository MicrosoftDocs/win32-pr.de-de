# <a name="cvideocaptureterminal"></a>Cvideocaptureterminal

Das **cvideocaptureterminal** -Terminal wird von [csinglefilterterminal](csinglefilterterminal.md) abgeleitet und implementiert ein statisches Video Aufzeichnungs Terminal mit einem einzelnen DirectShow-Filter. Bestimmte erweiterte Video Erfassungs Schnittstellen, die nur für die WDM-Video Erfassung anwendbar sind, werden nicht unterstützt. Die meisten MSPs müssen die Implementierung dieses Terminals nicht außer Kraft setzen.

Definiert in "msptmvc. h".
