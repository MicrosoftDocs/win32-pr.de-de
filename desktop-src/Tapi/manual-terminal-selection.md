# <a name="manual-terminal-selection"></a>Manuelle Terminal Auswahl

Wenn die Regeln der Standardauswahl nicht den Anforderungen der Anwendung entsprechen, hat die Anwendung die Möglichkeit, das Terminal wie gewohnt auszuwählen: das heißt, es kann [**itstream**](/windows/win32/api/tapi3if/nn-tapi3if-itstream) zum Auflisten der im-Befehl vorhandenen Datenströme verwenden und das Terminal in den entsprechenden Datenströmen auswählen (oder bei Multitrack-Terminals, erstellen Sie die Datenströme in den Datenströmen).
