# <a name="linetermdev_-constants"></a>Linetermdev- \_ Konstanten

Die **linetermdev \_** -Bitflag-Konstanten beschreiben verschiedene Typen von Terminal Geräten.

<dl> <dt>

<span id="LINETERMDEV_HEADSET"></span><span id="linetermdev_headset"></span>**linetermdev- \_ Headset**
</dt> <dd> <dl> <dt>



Das Terminal ist ein Headset.


</dt> </dl> </dd> <dt>

<span id="LINETERMDEV_PHONE"></span><span id="linetermdev_phone"></span>**linetermdev- \_ Telefon**
</dt> <dd> <dl> <dt>



Das Terminal ist ein Telefon Satz.


</dt> </dl> </dd> <dt>

<span id="LINETERMDEV_SPEAKER"></span><span id="linetermdev_speaker"></span>**linetermdev- \_ Sprecher**
</dt> <dd> <dl> <dt>



Das Terminal ist ein externer Redner und Mikrofon.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Bemerkungen

Keine Erweiterbarkeit. Alle 32 Bits sind reserviert.

Diese Konstanten werden verwendet, um das Terminal Gerät einer Linie zu charakterisieren und eine Anwendung zu unterstützen, die Art eines Terminal Geräts zu bestimmen.
