Die **tapierr- \_ Konstanten** stellen Informationen zu Funktions Ausführungsfehlern bereit.

<dl> <dt>

<span id="TAPIERR_CONNECTED"></span><span id="tapierr_connected"></span>**Verbindung mit tapierr \_**

</dt> </dl> </dd> <dt>

<span id="TAPIERR_DESTBUSY"></span><span id="tapierr_destbusy"></span>**tapierr \_ destbusy**
</dt> <dd> <dl> <dt>

</dt> </dl> </dd> <dt>

<span id="TAPIERR_DESTNOANSWER"></span><span id="tapierr_destnoanswer"></span>**tapierr \_ destnoanswer**
</dt> <dd> <dl> <dt>



Das Anruf Ziel hat nicht geantwortet.


</dt> </dl> </dd> <dt>

<span id="TAPIERR_DESTUNAVAIL"></span><span id="tapierr_destunavail"></span>**tapierr \_ destunnütz**
</dt> <dd> <dl> <dt>



Das calldestination ist nicht verfügbar.


</dt> </dl> </dd> <dt>

<span id="TAPIERR_DEVICECLASSUNAVAIL"></span><span id="tapierr_deviceclassunavail"></span>**tapierr-Geräte \_ Freigabe**

</dt> </dl> </dd> <dt>

<span id="TAPIERR_DEVICEIDUNAVAIL"></span><span id="tapierr_deviceidunavail"></span>**tapierr-Geräte Dienst \_**

<span id="TAPIERR_DEVICEINUSE"></span><span id="tapierr_deviceinuse"></span>**tapierr-Geräte- \_ /zielverwendung**

<span id="TAPIERR_DROPPED"></span><span id="tapierr_dropped"></span>**tapierr \_ gelöscht**
</dt> <dd> <dl> <dt>



Der-Befehl wurde gelöscht.


</dt> </dl> </dd> <dt>

<span id="TAPIERR_INVALDESTADDRESS"></span><span id="tapierr_invaldestaddress"></span>**tapierr \_ invaldebug**

<span id="TAPIERR_INVALDEVICECLASS"></span><span id="tapierr_invaldeviceclass"></span>**tapierr \_ invalabviceclass**

<span id="TAPIERR_INVALDEVICEID"></span><span id="tapierr_invaldeviceid"></span>**tapierr \_ invalabviceid**

</dt> </dl> </dd> <dt>

<span id="TAPIERR_INVALPOINTER"></span><span id="tapierr_invalpointer"></span>**tapierr- \_ invalpointer**
</dt> <dd> <dl> <dt>



Der Zeiger verweist nicht auf einen gültigen Speicher Speicherort. Mindestens einer der Zeiger " *lpszdestaddress*", " *lpszappname*", " *lpszcalledparty*" oder " *lpszcomment* " wurde angegeben, aber ist ungültig.


</dt> </dl> </dd> <dt>

<span id="TAPIERR_INVALWINDOWHANDLE"></span><span id="tapierr_invalwindowhandle"></span>**tapierr \_ invalwindowhandle**

</dt> </dl> </dd> <dt>

<span id="TAPIERR_NOTADMIN"></span><span id="tapierr_notadmin"></span>**tapierr \_ notadmin**

<span id="TAPIERR_REQUESTCANCELLED"></span><span id="tapierr_requestcancelled"></span>**tapierr \_ requestabgeb Rochen**

<span id="TAPIERR_REQUESTFAILED"></span><span id="tapierr_requestfailed"></span>**"tapierr \_ RequestFailed"**

<span id="TAPIERR_REQUESTQUEUEFULL"></span><span id="tapierr_requestqueuefull"></span>**tapierr \_ requestqueuefull**

</dt> </dl> </dd> <dt>

<span id="TAPIERR_UNKNOWNREQUESTID"></span><span id="tapierr_unknownrequestid"></span>**tapierr \_ unknownrequestid**

</dt> </dl> </dd> <dt>

<span id="TAPIERR_UNKNOWNWINHANDLE"></span><span id="tapierr_unknownwinhandle"></span>**tapierr \_ unknownwinhandle**
</dt> <dd> <dl> <dt>

</dt> </dl> </dd> </dl>
