# <a name="tapi-31-requirements"></a>TAPI 3,1-Anforderungen

Im folgenden sind die Betriebssystemanforderungen für TAPI 3. x aufgeführt:

-   TAPI Version 3,0 erfordert Windows 2000 oder höher.
-   TAPI Version 3,1 erfordert Windows Server 2003-Betriebssysteme oder Windows XP.

Beachten Sie, dass die [IP-Konferenz Steuerung](about-rendezvous-ip-telephony-conferencing.md) , die Rendezvous verwendet, Zugriff auf mindestens einen ILS-oder NTDS-Server benötigt. Wenn ein NTDS-Server vorhanden ist, kann er zum Auffinden eines ILS-Servers verwendet werden. Wenn kein NTDS-Server vorhanden ist, muss der Endbenutzer den Namen des ILS-Servers kennen.
