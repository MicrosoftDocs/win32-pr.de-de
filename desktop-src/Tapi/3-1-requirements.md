# <a name="tapi-31-requirements"></a>TAPI 3.1-Anforderungen

Im Folgenden finden Sie die Betriebssystemanforderungen für TAPI 3.x:

-   TAPI Version 3.0 erfordert Windows 2000 oder höher.
-   Die TAPI-Version 3.1 erfordert Windows Server 2003-Betriebssysteme oder Windows XP.

Beachten Sie, [dass das IP-Konferenzsteuersystem](about-rendezvous-ip-telephony-conferencing.md) mit Rendezvous Zugriff auf mindestens einen ILS- oder NTDS-Server erfordert. Wenn ein NTDS-Server installiert ist, kann er verwendet werden, um einen ILS-Server zu suchen. Wenn kein NTDS-Server vorhanden ist, muss der Endbenutzer den Namen des ILS-Servers kennen.
