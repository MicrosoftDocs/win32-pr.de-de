# <a name="h323-msp"></a>H323 MSP

\[H323 MSP ist nicht für die Verwendung in Windows Vista, Windows Server 2008 und nachfolgenden Versionen des Betriebssystems verfügbar. Die RTC-Client-API bietet ähnliche Funktionen.\]

Der H323 MSP (H323msp.dll) ermöglicht einer Anwendung das Tätigen und Empfangen von Video- und Audioanrufen mithilfe des H.323-Protokolls für IP-Telefonie, wenn er mit dem TSP (H323.tsp) gekoppelt ist.

Der MSP H323 wurde in Windows 2000 SP1 verbessert, um die Kommunikation zwischen H323- und SDP-Clients zu ermöglichen. Jetzt werden die Schnittstellen [**IH323LineEx**](ih323lineex.md) und [**IKeyFrameControl**](ikeyframecontrol.md) verfügbar gemacht.

Der MSP H323 wird mit Windows 2000 installiert.
