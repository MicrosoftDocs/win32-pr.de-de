# <a name="h323-msp"></a>H323-msp

\[H323 MSP ist nicht für die Verwendung in Windows Vista, Windows Server 2008 und nachfolgenden Versionen des Betriebssystems verfügbar. Die RTC-Client-API bietet eine ähnliche Funktionalität.\]

Der h323 msp (H323msp.dll) ermöglicht es einer Anwendung, Video-und Audioanrufe mithilfe des H. 323-Protokolls für IP-Telefonie zu tätigen und zu empfangen, wenn Sie mit dem TSP (h323. TSP) gekoppelt ist.

H323 MSP wurde in Windows 2000 SP1 verbessert, um die Kommunikation zwischen h323-und SDP-Clients zu ermöglichen. Nun werden die Schnittstellen [**IH323LineEx**](ih323lineex.md) und [**ikeyframecontrol**](ikeyframecontrol.md) verfügbar gemacht.

H323 MSP wird mit Windows 2000 installiert.
