# <a name="call-and-media-controls-reference"></a>Verweise auf Aufrufe und mediensteuer Elemente

Das Referenzmaterial für das Referenz-und mediensteuer Element enthält Informationen zu Schnittstellen, Methoden, Enumeratoren und Typen, die in den Setup-und Steuerelement Einstellungen verwendet werden

Möglicherweise möchten Sie auch das Platform Software Development Kit (SDK) durchsuchen, um Informationen zu DirectShow und den Component Object Model (com) zu finden. TAPI 3. x verwendet DirectShow-Filter und Filter-Manager, um den Mediendaten Strom nach dem Einrichten der Verbindung zu steuern. COM ist das grundlegende Codierungs Modell für TAPI 3. x, und Vertrautheit mit diesem Modell wird in diesem Dokument angenommen.
