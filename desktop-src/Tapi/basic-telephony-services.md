# <a name="basic-telephony-services"></a>Grundlegende Telefoniedienste

Grundlegende Telefoniedienste sind eine minimale Teilmenge der telefoniespezifikation. Da alle Dienstanbieter die Funktionen von grundlegenden Telefoniediensten unterstützen müssen, funktionieren Anwendungen, die nur diese Funktionen verwenden, mit jedem TAPI-Dienstanbieter. Die in der grundlegenden Telefoniefunktion enthaltene Funktionalität entspricht ungefähr den Features von Töpfen.

Viele Programmierer verwenden nur die Dienste, die von Basic-Telefoniediensten bereitstellt werden. Andere, z. b. solche, die Code für PBX-Telefonsysteme schreiben, benötigen die Funktionen zusätzlicher Telefoniefunktionen.

Eine Liste der Funktionen der Standard telefonietelefoniefunktion finden Sie unter [TAPI-schnell Funktionsreferenz](./tapi-quick-function-reference.md).

Da davon ausgegangen wird, dass die Steuerung von Telefongeräten nicht von allen Dienstanbietern angeboten wird, werden die Telefongeräte Dienste als optional eingestuft. Das heißt, Sie sind kein Bestandteil der grundlegenden telefonietelefonie. Eine Liste der Telefongeräte Dienste finden Sie unter [ergänzende Telefoniedienste](supplementary-telephony-services.md); Weitere Informationen zu Telefongeräten finden Sie unter [TAPI-Geräteklassen](./tapi-device-classes.md).
