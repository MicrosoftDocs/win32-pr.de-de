# <a name="tapi-22"></a>TAPI 2,2

TAPI 2,2 bietet spezialisierte Callcenter-Verwaltungsfunktionen. Unter TAPI 2,1 und früher erforderte die Callcentersoftware eine sorgfältige Zuordnung von Linien und Adressen zur Callcenter-Funktion. Die TAPI 2,1-Funktionen koordinieren mit TAPI 3. x-Callcenter-Funktionen und vereinfachen die Erstellung von ACD-Systemen erheblich.

**TAPI 2. x:** Weitere Informationen finden Sie unter [Callcenter-Steuer](call-center-control.md)Element.

**TAPI 3. x:** Weitere Informationen finden Sie unter [Callcenter](./about-call-center-controls.md)-Steuerelemente.
