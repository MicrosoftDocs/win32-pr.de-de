# <a name="tapi-22"></a>TAPI 2.2

TAPI 2.2 bietet spezielle Callcenterverwaltungsfunktionen. Unter TAPI 2.1 und früher erforderte die Callcentersoftware eine sorgfältige Zuordnung von Zeilen und Adressen zur Callcenterfunktion. Die TAPI 2.1-Funktionen koordinieren mit TAPI 3.x-Callcenterfunktionen und vereinfachen die Erstellung von ACD-Systemen erheblich.

**TAPI 2.x:** Weitere Informationen finden Sie unter [Callcenter-Steuerelement](call-center-control.md).

**TAPI 3.x:** Weitere Informationen finden Sie unter [CallCenter-Steuerelemente.](./about-call-center-controls.md)
