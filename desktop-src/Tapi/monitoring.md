# <a name="monitoring"></a>Überwachung

Die Überwachung ist ein Dienstanbieter definiertes Feature, das Signale erkennt, die auf andere Medien hinweisen. So kann z. b. eine Anwendung eine ausgehende Meldung "Nachricht überlassen" wiedergeben, während der eingehende Anruf mit dem Senden eines Faxanrufs beginnt und auf einen Handshake wartet. Um den Faxanruf nicht zu verlieren, überwacht die lokale Anwendung bei der Wiedergabe der Sprachnachricht diesen Ton.

Die Überwachung umfasst [Medienüberwachung](/previous-versions/windows/desktop/legacy/ms725242(v=vs.85)), [Ziffern Überwachung](/previous-versions/windows/desktop/legacy/ms725185(v=vs.85))und [Ton Überwachung](/previous-versions/windows/desktop/legacy/ms725520(v=vs.85)).
