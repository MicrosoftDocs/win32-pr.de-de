# <a name="microsoft-telephony-overview"></a>Übersicht über Microsoft-Telefonie

Telefonie integriert Computer mit Kommunikationsgeräten und Netzwerken. Unter klassischer Telefoniedienst war das Gerät ein Telefon, und das Netzwerk war das Public Switched Telefon Network (PSTN). Moderne Telefonie erweitert kontinuierlich den Bereich von Geräten und Netzwerken und deckt derzeit Geräte wie Kameras und Netzwerke (z. b. das Internet) ab.

Mögliche Telefonieanwendungen:

-   Multicast-Multimedia-IP-Konferenzen
-   Sprachanrufe über das Internet (VoIP)
-   Client-und Server Anwendungen für die automatische aufrufsverteilung (ACD)
-   Einfache sprach Aufrufe für den PSTN
-   PBX-ähnliche Steuerelemente, wie z. b. "Callcenter" und "selektive Weiterleitung" in einem Unternehmens Telefonnetzwerk, ohne in spezielle Hardware investieren zu müssen
-   Interaktive sprach antwortsysteme (IVR)
-   Echtzeitzusammenarbeit
