# <a name="tapi-3-terminal-manager-reference"></a>TAPI 3 Terminal-Manager-Referenz



| Terminal-Manager-Schnittstellen                      | Beschreibung                                                                                                                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ITTerminalControl**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalcontrol)   | Stellt Methoden zum Verbinden und Trennen eines Terminals mit einem Filterdiagramm sowie zum Ausführen oder Beenden eines Renderers zur Verfügung.                                                                                 |
| [**ITTerminalManager**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager)   | Stellt Methoden zum Erstellen dynamischer Terminals bereit.                                                                                                                                          |
| [**ITTerminalManager2**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager2) | Stellt Methoden zum Abrufen von Informationen zu pluggable-Terminalklassen zur Verfügung, die im aktuellen System registriert sind. wird von der [**ITTerminalManager-Schnittstelle**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager) ableiten. |
