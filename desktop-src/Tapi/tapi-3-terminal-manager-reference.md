# <a name="tapi-3-terminal-manager-reference"></a>TAPI 3-Terminal-Manager-Referenz



| Terminal Manager-Schnittstellen                      | BESCHREIBUNG                                                                                                                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Itterminalcontrol**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalcontrol)   | Stellt Methoden bereit, um eine Verbindung mit einem Terminal von einem Filter Diagramm herzustellen und die Verbindung zu trennen und einen Renderer auszuführen oder zu beenden.                                                                                 |
| [**Itterminalmanager**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager)   | Stellt Methoden zum Erstellen dynamischer Terminals bereit.                                                                                                                                          |
| [**ITTerminalManager2**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager2) | Stellt Methoden zum Abrufen von Informationen über austauschbare Terminal Klassen bereit, die im aktuellen System registriert sind. wird von der [**itterminalmanager**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager) -Schnittstelle abgeleitet. |
