# <a name="linetolllistoption_-constants"></a>Linetolllistoption- \_ Konstanten

Die bitflagkonstanten von **linetolllistoption \_** beschreiben Optionen zum Bearbeiten einer Maut Liste.

<dl> <dt>

<span id="LINETOLLLISTOPTION_ADD"></span><span id="linetolllistoption_add"></span>**linetolllistoption \_ Hinzufügen**
</dt> <dd> <dl> <dt>



Der Maut Liste muss ein Präfix hinzugefügt werden.


</dt> </dl> </dd> <dt>

<span id="LINETOLLLISTOPTION_REMOVE"></span><span id="linetolllistoption_remove"></span>**linetolllistoption \_ Entfernen**
</dt> <dd> <dl> <dt>



Ein Präfix wird aus der Maut Liste entfernt.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Bemerkungen

Keine Erweiterbarkeit. Alle 32 Bits sind reserviert.
