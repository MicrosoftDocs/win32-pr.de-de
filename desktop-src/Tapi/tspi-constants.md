# <a name="tspi-constants"></a>TSPI-Konstanten

In diesem Abschnitt werden die Linien Geräte Konstanten beschrieben, die von der telefoniespi definiert werden. Die meisten Konstanten, die in der TSPI-Programmierung verwendet werden, entsprechen den in [TAPI-Konstanten](tapi-constants.md)definierten Konstanten.
