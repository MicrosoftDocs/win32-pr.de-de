# <a name="lineremovefromconf_-constants"></a>LINEREMOVEFROMCONF-Konstanten \_

Die **\_ LINEREMOVEFROMCONF-Skalarkonstanten** beschreiben, wie Teilnehmer an einer Telefonkonferenz aus einer Telefonkonferenz entfernt werden können.

<dl> <dt>

<span id="LINEREMOVEFROMCONF_ANY"></span><span id="lineremovefromconf_any"></span>**LINEREMOVEFROMCONF \_ ANY**
</dt> <dd> <dl> <dt>



Jede teilnehmende Partei kann aus der Telefonkonferenz entfernt werden.


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_LAST"></span><span id="lineremovefromconf_last"></span>**LINEREMOVEFROMCONF \_ LAST**
</dt> <dd> <dl> <dt>



Nur die zuletzt hinzugefügte Partei kann aus der Telefonkonferenz entfernt werden.


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_NONE"></span><span id="lineremovefromconf_none"></span>**LINEREMOVEFROMCONF \_ NONE**
</dt> <dd> <dl> <dt>



Parteien können nicht aus der Telefonkonferenz entfernt werden.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Hinweise

Keine Erweiterbarkeit. Alle 32 Bits sind reserviert.
