# <a name="lineremovefromconf_-constants"></a>Lineremovefromconf- \_ Konstanten

Die skalarkonstanten **lineremovefromconf \_** beschreiben, wie Parteien, die an einem Konferenzgespräch teilnehmen, von einem Konferenzgespräch entfernt werden können.

<dl> <dt>

<span id="LINEREMOVEFROMCONF_ANY"></span><span id="lineremovefromconf_any"></span>**lineremovefromconf \_ Any**
</dt> <dd> <dl> <dt>



Alle beteiligten Parteien können aus dem Konferenzgespräch entfernt werden.


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_LAST"></span><span id="lineremovefromconf_last"></span>**lineremovefromconf \_ zuletzt**
</dt> <dd> <dl> <dt>



Nur die zuletzt hinzugefügte Partei kann aus dem Konferenzgespräch entfernt werden.


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_NONE"></span><span id="lineremovefromconf_none"></span>**lineremovefromconf \_ None**
</dt> <dd> <dl> <dt>



Parteien können nicht aus dem Konferenzgespräch entfernt werden.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Bemerkungen

Keine Erweiterbarkeit. Alle 32 Bits sind reserviert.
