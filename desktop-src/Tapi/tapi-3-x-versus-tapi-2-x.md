# <a name="tapi-3x-vs-tapi-2x"></a>TAPI 3. x im Vergleich zu TAPI 2. x

Die COM-basierte telefonieapi TAPI 3. x ist ab Windows XP verfügbar. TAPI 3. x bietet erheblich verbesserte Entwicklungs Tools für die moderne Welt der Kommunikations Programmierung, bei der ein "Anruf" ein Videostream in einem IP-basierten Netzwerk sein kann und kein Telefon Satz an der Sitzung beteiligt ist. Im Allgemeinen finden die Anwendungs Schreiber TAPI 3. x bietet eine bessere Unterstützung für die Implementierung von flexiblen Kommunikations Steuerungen.

TAPI 2. x ist in einigen Situationen jedoch möglicherweise die richtige Wahl. Wenn für eine Legacy Anwendung kleinere Änderungen erforderlich sind oder Legacy Plattformen unterstützt werden müssen, ist die Konvertierung in die com-API möglicherweise nicht kostengünstig.
