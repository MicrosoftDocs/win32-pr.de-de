# <a name="itterminalevent-mspi"></a>Itterminalevent (MSPi)

Wenn die [**ittapieventnotification:: Event**](/windows/desktop/api/Tapi3if/nf-tapi3if-ittapieventnotification-event) -Methode das [**TAPI- \_ Ereignis**](/windows/desktop/api/Tapi3if/ne-tapi3if-tapi_event) gleich dem **Terminal- \_ Terminal** zurückgibt, können die Methoden, die von der **itterminalevent** -Schnittstelle verfügbar gemacht werden, zum Abrufen von Informationen über das aufgetretene Terminal Ereignis verwendet werden.

Die **itterminalevent** -Schnittstelle wird von einem MSP implementiert und ist nicht verfügbar, wenn der Adresse kein Medien Dienstanbieter zugeordnet ist. Ausführliche Informationen zu dieser Schnittstelle finden Sie im Abschnitt **itterminalevent** im Abschnitt MSP-Schnittstelle.
