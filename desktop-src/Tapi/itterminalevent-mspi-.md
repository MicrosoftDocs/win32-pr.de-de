# <a name="itterminalevent-mspi"></a>ITTerminalEvent (MSPI)

Wenn die [**ITTAPIEventNotification::Event-Methode**](/windows/desktop/api/Tapi3if/nf-tapi3if-ittapieventnotification-event) [**TAPI \_ EVENT**](/windows/desktop/api/Tapi3if/ne-tapi3if-tapi_event) gleich **TE \_ TERMINAL** zurückgibt, können die von der **ITTerminalEvent-Schnittstelle** verfügbar gemachten Methoden verwendet werden, um Informationen über das aufgetretene Terminalereignis abzurufen, wenn ein MSP vorhanden ist.

Die **ITTerminalEvent-Schnittstelle** wird von einem MSP implementiert und ist nicht verfügbar, wenn der Adresse kein Mediendienstanbieter zugeordnet ist. Weitere Informationen zu dieser Schnittstelle finden Sie im Abschnitt "MSP-Schnittstelle" unter **ITTerminalEvent.**
