# <a name="track-controlled-playback"></a>Track-Controlled Wiedergabe

Gehen Sie folgendermaßen vor, um eine Nachverfolgung gesteuerte Wiedergabe auszuführen:

-   Wählen Sie die Option Terminals in Streams nachverfolgen aus.
-   Erstellen Sie das Terminal Dateiwiedergabe.
-   Legen Sie die Eigenschaften – Dateiname und Typ des Speichers fest.
-   Auflisten Sie mithilfe einer Methode im Terminal für die Dateiwiedergabe die dateitrackendterminals.
-   Enumerieren von Streams für den TAPI-Befehl.
-   Wählen Sie im Stream das dateitrackterminal aus.
-   Ruft die [**Start**](/windows/desktop/api/tapi3if/nf-tapi3if-itmediacontrol-start) -Methode im Terminal der Dateiwiedergabe auf, um die Wiedergabe der aufgezeichneten Daten in den TAPI-Streams zu starten.
