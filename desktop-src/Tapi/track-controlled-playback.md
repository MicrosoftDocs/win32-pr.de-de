# <a name="track-controlled-playback"></a>Track-Controlled Wiedergeben

Gehen Sie wie folgt vor, um eine nachverfolgungsgesteuerte Wiedergabe durchzuführen:

-   Wählen Sie die Option Track Terminals to streams (Terminals auf Streams nachverfolgen) aus.
-   Erstellen Sie das Terminal für die Dateiwiedergabe.
-   Legen Sie Eigenschaften fest– Dateiname und Speichertyp.
-   Mithilfe einer -Methode im Terminal für die Dateiwiedergabe können Sie Dateiverfolgungsterminals aufzählen.
-   Aufzählen von Streams im TAPI-Aufruf.
-   Wählen Sie das Dateiverfolgungsterminal im Stream aus.
-   Rufen Sie die [**Start-Methode**](/windows/desktop/api/tapi3if/nf-tapi3if-itmediacontrol-start) im Terminal für die Dateiwiedergabe auf, um die Wiedergabe aufgezeichneter Daten in den TAPI-Streams zu starten.
