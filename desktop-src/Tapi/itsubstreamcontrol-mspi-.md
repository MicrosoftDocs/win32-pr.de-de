# <a name="itsubstreamcontrol-mspi"></a>Itsubstreamcontrol (MSPi)

Die [**itsubstreamcontrol**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstreamcontrol) -Schnittstelle wird für das Stream-Objekt verfügbar gemacht, wenn ein zugeordnetes MSP vorhanden ist und untergeordnete Datenströme implementiert. Die Methoden dieser Schnittstelle ermöglichen es einer Anwendung, Substreams zu erstellen und zu entfernen.

Die [**itsubstreamcontrol**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstreamcontrol) -Schnittstelle wird von einem MSP implementiert. Ausführliche Informationen zu dieser Schnittstelle finden Sie im Abschnitt **itsubstreamcontrol** im Abschnitt MSP-Schnittstelle.
