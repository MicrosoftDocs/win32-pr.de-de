# <a name="lineproxystatus_-constants"></a>Lineproxystatus- \_ Konstanten

Die **lineproxystatus- \_ Konstanten** geben den Status des Proxys in einer Zeile an, die die Anwendung zurzeit geöffnet hat.

Eine Liste und eine Beschreibung aller möglichen Proxy Anforderungs Werte finden Sie unter [**lineproxyrequest- \_ Konstanten**](lineproxyrequest--constants.md) .

<dl> <dt>

<span id="LINEPROXYSTATUS_ALLOPENFORACD"></span><span id="lineproxystatus_allopenforacd"></span>**lineproxystatus \_ allopenforacd**
</dt> <dd> <dl> <dt>



In der Zeile sind nun Proxys für alle Proxy Anforderungs Typen geöffnet, die für ACD-Vorgänge von TAPI, Version 3,0 und höher erforderlich sind.


</dt> </dl> </dd> <dt>

<span id="LINEPROXYSTATUS_CLOSE"></span><span id="lineproxystatus_close"></span>**lineproxystatus \_ Schließen**
</dt> <dd> <dl> <dt>



Ein Proxy wurde geschlossen.


</dt> </dl> </dd> <dt>

<span id="LINEPROXYSTATUS_OPEN"></span><span id="lineproxystatus_open"></span>**lineproxystatus \_ geöffnet**
</dt> <dd> <dl> <dt>



Ein neuer Proxy wurde geöffnet.


</dt> </dl> </dd> </dl>
