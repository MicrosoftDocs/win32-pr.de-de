# <a name="lineproxystatus_-constants"></a>\_LINEPROXYSTATUS-Konstanten

Die **\_ LINEPROXYSTATUS-Konstanten** geben den Status des Proxys in einer Zeile an, die die Anwendung derzeit geöffnet hat.

Eine Liste und Eine Beschreibung aller möglichen Proxyanforderungswerte finden Sie unter [**\_ LINEPROXYREQUEST-Konstanten.**](lineproxyrequest--constants.md)

<dl> <dt>

<span id="LINEPROXYSTATUS_ALLOPENFORACD"></span><span id="lineproxystatus_allopenforacd"></span>**LINEPROXYSTATUS \_ ALLOPENFORACD**
</dt> <dd> <dl> <dt>



Die Zeile verfügt jetzt über Proxys, die für alle Proxyanforderungstypen geöffnet sind, die für ACD-Vorgänge ab TAPI Version 3.0 erforderlich sind.


</dt> </dl> </dd> <dt>

<span id="LINEPROXYSTATUS_CLOSE"></span><span id="lineproxystatus_close"></span>**LINEPROXYSTATUS \_ CLOSE**
</dt> <dd> <dl> <dt>



Ein Proxy wurde geschlossen.


</dt> </dl> </dd> <dt>

<span id="LINEPROXYSTATUS_OPEN"></span><span id="lineproxystatus_open"></span>**LINEPROXYSTATUS \_ OPEN**
</dt> <dd> <dl> <dt>



Ein neuer Proxy wurde geöffnet.


</dt> </dl> </dd> </dl>
