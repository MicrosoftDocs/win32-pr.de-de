# <a name="hdrvphone"></a>Hdrvphone

Der **hdrvphone** -Datentyp stellt den nicht transparenten Handle eines Dienstanbieters für eine Telefondaten Struktur dar. Der Dienstanbieter muss einen Wert dieses Typs in einen Verweis auf die entsprechende Datenstruktur Instanz auflösen.
