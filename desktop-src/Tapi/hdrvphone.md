# <a name="hdrvphone"></a>HDRVPHONE

Der **HDRVPHONE-Datentyp** stellt das nicht transparente Handle eines Dienstanbieters für eine Telefondatenstruktur dar. Der Dienstanbieter muss einen Wert dieses Typs in einen Verweis auf die entsprechende Datenstrukturinstanz auflösen.
