# <a name="file-terminals"></a>Datei Terminals

In TAPI 3,1 wird das Terminal Objekt so erweitert, dass es ein Datei Terminal enthält. Datei Terminals bieten eine einfache Möglichkeit, häufige Vorgänge auszuführen, z. b. einen Audiostream oder Datenströme in eine Datei oder eine nachfolgende Wiedergabe der aufgezeichneten Audiodaten in einen Stream zu erfassen.

Zum Bearbeiten von Datei Terminals werden fünf grundlegende Objekte verwendet:

-   Dateiwiedergabe Terminal
-   Datei Aufzeichnungs Terminal
-   Dateiwiedergabe Track
-   Datei Aufzeichnungs Track
-   Datei Ereignis

Die Datei Terminal Objekte können als Abstraktion für den Speicher betrachtet werden, bei dem es sich in der Regel um eine Datei handelt. Die Spuren entsprechen den einzelnen audiodatenstreams in diesem Speicher, z. b. wenn Voice aufgezeichnet wird.

Weitere Informationen zu Datei Terminals finden Sie unter:

-   [Terminal Dateiwiedergabe Terminal und Datei Aufzeichnungs Terminal](file-playback-terminal-and-file-recording-terminal.md)
-   [Nachverfolgen von Terminals](track-terminals.md)

Weitere Informationen und Codebeispiele und Aufgabenlisten im Zusammenhang mit Datei Terminals finden [Sie unter Verwenden von Datei Terminals](using-file-terminals.md).
