<dl> <dt>

<span id="tapi2.integrated_services_digital_network_isdn__tapgloss"></span><span id="TAPI2.INTEGRATED_SERVICES_DIGITAL_NETWORK_ISDN__TAPGLOSS"></span>**Integrated Services Digital Network (ISDN)**
</dt> <dd>

Ein vollständig digitales Telefon-/Telekommunikationsnetzwerk für die Übertragung von Sprache, Daten, Bildern und Videos mit hoher Geschwindigkeit, indem digital codierte Signale mit Geschwindigkeiten von bis zu 144.000 Bps an BRI-ISDN-Linien (Basic Rate Interface) und viel höher an PRI-ISDN-Leitungen (Primary Rate Interface) gesendet werden. ISDN besteht aus mindestens drei Kanälen und bis zu 32 Kanälen für die gleichzeitige, unabhängig betriebene Übertragung von Sprache und Daten. Siehe [*Basic Rate Interface (BRI-ISDN)*](b-tapgloss.md), Primary Rate Interface [*(PRI-ISDN)*](p-tapgloss.md).

</dd> <dt>

<span id="tapi2.interactive_voice_response_ivr_application_tapgloss"></span><span id="TAPI2.INTERACTIVE_VOICE_RESPONSE_IVR_APPLICATION_TAPGLOSS"></span>**Anwendung für interaktive Sprachantworten (Interactive Voice Response, IVR)**
</dt> <dd>

Eine Telefonieanwendung, die einen Telefonanrufer durch eine Hierarchie von Menüs führt, Sprachantworten übermittelt, Sprach- und Dateneingaben sammelt und andere Vorgänge im Auftrag des Anrufers oder programmgesteuerten Sponsors ausführt. Weitere Informationen finden Sie unter [*Webtelefonie.*](w-tapgloss.md)

</dd> <dt>

<span id="tapi2.inverse_multiplexing_tapgloss"></span><span id="TAPI2.INVERSE_MULTIPLEXING_TAPGLOSS"></span>**Inverses Multiplexing**
</dt> <dd>

Der Prozess, bei dem die Bandbreite einem Aufruf ermöglicht, mehr Informationen gleichzeitig in demselben Zeitraum zu übertragen als Informationen von einem einzelnen Signal, das über einen einzelnen Kanal übertragen wird. Dies wird erreicht, indem ein einzelner Hochgeschwindigkeitskanal in mehrere Signale aufgeteilt wird, die mehrere Signale über mehrere Einrichtungen übertragen werden, die mit einer niedrigeren Geschwindigkeit als das ursprüngliche Signal arbeiten, und dann die separat übertragenen Teile mit der ursprünglichen Rate in das ursprüngliche Signal rekombinieren. Weitere Informationen finden Sie unter [*Bandbreite.*](b-tapgloss.md)

</dd> <dt>

<span id="tapi2.ip_multicast_tapgloss"></span><span id="TAPI2.IP_MULTICAST_TAPGLOSS"></span>**IP-Multicast**
</dt> <dd>

IP-Multicast ist ein Schema zum gleichzeitigen Senden der gleichen Daten an mehrere Computer. Ein Netzwerk mit aktivierten Multicastroutern kann solche Informationen viel schneller übermitteln, indem Pakete nur einmal zwischen Routern gesendet werden. Weitere Informationen finden Sie unter [Informationen zu Rendezvous IP-Telefoniekonferenzen.](./about-rendezvous-ip-telephony-conferencing.md)

</dd> <dt>

<span id="tapi2.isdn_tapgloss"></span><span id="TAPI2.ISDN_TAPGLOSS"></span>**Isdn**
</dt> <dd>

Weitere Informationen finden Sie unter *Integrated Services Digital Network (ISDN).*

</dd> </dl>
