# <a name="ipconf-msp"></a>IPConf MSP

\[IPConf MSP ist nicht für die Verwendung in Windows Vista, Windows Server 2008 und nachfolgenden Versionen des Betriebssystems verfügbar. Die RTC-Client-API bietet ähnliche Funktionen.\]

Der IPConf MSP (Confmsp.dll) ermöglicht, wenn er mit seinem TSP (IPconf.tsp) gekoppelt ist, Konferenzzugriff auf IP-Netzwerke (Internet Protocol), z. B. das Internet und viele Unternehmensintranets. Zusätzlich zu den MSPI-Standardmethoden macht dieser MSP Schnittstellen verfügbar, die die Kontrolle über einzelne Teilnehmer einer Konferenz ermöglichen.

Eine Zusammenfassung der anbieterspezifischen Schnittstellen, die von diesem MSP verfügbar gemacht werden, finden Sie unter [IPConf-MSP-Schnittstellen.](ipconf-msp-interfaces.md)

Der IPConf-MSP wurde in Windows 2000 SP1 verbessert, um die Kommunikation zwischen H323- und SDP-Clients zu ermöglichen. Jetzt wird zusätzlich die [**IMulticastControl-Schnittstelle**](imulticastcontrol.md) verfügbar gemacht.

Der IPConf-MSP wird mit Windows 2000 installiert.
