# <a name="tspi_linedropnoowner"></a>TSPI \_ linedropnoowner

Die **TSPI-Funktion " \_ linedropnoowner** " ist veraltet. TAPI löscht einen Aufruf nicht mehr automatisch, wenn kein anfänglicher Besitzer identifiziert werden kann, um die verteilte Kontrolle von Aufrufen über mehrere Client Computer in einer Client-/Server-Umgebung zu ermöglichen.
