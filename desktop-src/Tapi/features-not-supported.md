# <a name="features-not-supported"></a>Nicht unterstützte Features

-   Die dynamische Graphbehandlung wird vom abgeleiteten MSP implementiert, wenn er dies auswählt.
-   SubStreams werden vom abgeleiteten MSP implementiert, wenn er sich dafür entscheidet.
-   Private Schnittstellen wie Teilnehmer, Codec-Steuerung und so weiter werden von den Basisklassen nicht implementiert.
-   Die TSP-MSP-Kommunikation wird vom abgeleiteten MSP implementiert.
