<dl> <dt>

<span id="tapi2.b_channel_tapgloss"></span><span id="TAPI2.B_CHANNEL_TAPGLOSS"></span>**B-Kanal**
</dt> <dd>

Ein *bearerchannel, der eine Grund* Legende Komponente von ISDN-Schnittstellen ist. Sie beinhaltet 64.000 Bits pro Sekunde (Bit/s) in beide Richtungen, ist Leitungs gewechselt und kann entweder Voice oder Daten enthalten. Weitere Informationen finden Sie unter [bearermodus](./bearer-mode-ovr.md). Weitere Informationen finden Sie unter [*D Channel*](d-tapgloss.md), [*Integrated Services Digital Network (ISDN)*](i-tapgloss.md).

</dd> <dt>

<span id="tapi2.band_tapgloss"></span><span id="TAPI2.BAND_TAPGLOSS"></span>**Kreuz**
</dt> <dd>

Der Bereich der Frequenzen zwischen zwei definierten Punkten.

</dd> <dt>

<span id="tapi2.bandwidth_tapgloss"></span><span id="TAPI2.BANDWIDTH_TAPGLOSS"></span>**Bandbreite**
</dt> <dd>

Bei der Kommunikation der Unterschied zwischen der höchsten und der niedrigsten Frequenz in einem bestimmten Bereich. Beispielsweise wird in einem Telefon eine Bandbreite von 3000 Hz, der Unterschied zwischen der niedrigsten (300 Hz) und der höchsten (3300 Hz) Frequenz, die Sie tragen kann, berücksichtigt.

</dd> <dt>

<span id="tapi2.bandwidth_allocation_control_protocol_bacp__tapgloss"></span><span id="TAPI2.BANDWIDTH_ALLOCATION_CONTROL_PROTOCOL_BACP__TAPGLOSS"></span>**Bandbreiten Zuordnungs Steuerungs-Protokoll (BACP)**
</dt> <dd>

Der Regelsatz, der die Bandbreite über dynamische PPP-Verbindungen mit mehreren Links verwaltet. Weitere Informationen finden Sie unter [*Multi-Link PPP*](m-tapgloss.md), [*Point-to-Point-Protokoll (PPP)*](p-tapgloss.md).

</dd> <dt>

<span id="tapi2.basic_rate_interface_bri_isdn__tapgloss"></span><span id="TAPI2.BASIC_RATE_INTERFACE_BRI_ISDN__TAPGLOSS"></span>**Basic-Raten Schnittstelle (BRI-ISDN)**
</dt> <dd>

Die standardmäßige ISDN-Linie, bestehend aus zwei B-Kanälen und einem D-Kanal. Weitere Informationen finden Sie unter [*Integrated Services Digital Network (ISDN)*](i-tapgloss.md), [*Primary Rate Interface (PRI-ISDN)*](p-tapgloss.md).

</dd> <dt>

<span id="tapi2.basic_telephony_tapgloss"></span><span id="TAPI2.BASIC_TELEPHONY_TAPGLOSS"></span>**Basic-Telefonie**
</dt> <dd>

Eine elementare Ebene des Telefoniediensts mit einer minimalen Teilmenge von Funktionen, die einen minimalen Satz von Funktionen bereitstellt, die dem Plain Old Telefonservice (POTS) entsprechen. TAPI-Dienstanbieter sind erforderlich, um alle grundlegenden Telefoniefunktionen zu unterstützen. Siehe unter [stützte Telefonieinformationen](./assisted-telephony-overview.md), [*Erweiterte Telefonieinformationen*](e-tapgloss.md), [*ergänzende Telefonie*](s-tapgloss.md).

</dd> <dt>

<span id="tapi2.bri_isdn_tapgloss"></span><span id="TAPI2.BRI_ISDN_TAPGLOSS"></span>**BRI-ISDN**
</dt> <dd>

Siehe *Basic-Raten Schnittstelle (BRI-ISDN)*.

</dd> <dt>

<span id="tapi2.bps_tapgloss"></span><span id="TAPI2.BPS_TAPGLOSS"></span>**2,4**
</dt> <dd>

Siehe *Bits pro Sekunde (* Bit/s).

</dd> <dt>

<span id="tapi2.bits_per_second_bps__tapgloss"></span><span id="TAPI2.BITS_PER_SECOND_BPS__TAPGLOSS"></span>**Bits pro Sekunde (Bit/s)**
</dt> <dd>

Eine Übertragungsraten Messung, die angibt, wie viele Bits von Computer Daten innerhalb eines Sekunde durch ein Modem über eine Telefonleitung gesendet werden.

</dd> </dl>
