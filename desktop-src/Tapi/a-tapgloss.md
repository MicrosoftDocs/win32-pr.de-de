<dl> <dt>

<span id="tapi2.analog_display_services_interface_adsi__tapgloss"></span><span id="TAPI2.ANALOG_DISPLAY_SERVICES_INTERFACE_ADSI__TAPGLOSS"></span>**Analoge Anzeige Dienste-Schnittstelle (ADSI)**
</dt> <dd>

Ein "Bellcore"-Standard, der ein Protokoll für den Informationsfluss zwischen einem Switch, einem Server, einem Sprachnachrichten System oder einem Servicebüro definiert, z. b. einem Telefon, einem PC, einem Datenterminal oder einem anderen Kommunikationsgerät eines Abonnenten mit einem Bildschirm. Mit ADSI kann eine Textversion der Konversation während der Telefonverbindung auf dem Bildschirm angezeigt werden. Beispielsweise wird der Benutzer in ein Voicemail-e-Mail-Konto eingecheckt und erhält ein Audiomenü für die Sprachnachrichten Funktionen, während der Bildschirm das Menü in Textform anzeigt.

</dd> <dt>

<span id="tapi2.assisted_telephony_tapgloss"></span><span id="TAPI2.ASSISTED_TELEPHONY_TAPGLOSS"></span>**Unterstützte Telefonie**
</dt> <dd>

Der Satz von Funktionen, die nicht nur für die Telefonfunktion vorgesehen sind, sondern dafür konzipiert sind, Sprachanrufe und Medien Aufrufe für jede Anwendung verfügbar zu machen. Weitere Informationen finden Sie unter unter [stützte Telefonie](./assisted-telephony-overview.md) in der Übersicht über die Telefonie. Weitere Informationen finden Sie unter [*Basic Telefonie*](b-tapgloss.md), [*Erweiterte Telefonie*](e-tapgloss.md), [*ergänzende Telefonie*](s-tapgloss.md).

</dd> <dt>

<span id="tapi2.asynchronous_tapgloss"></span><span id="TAPI2.ASYNCHRONOUS_TAPGLOSS"></span>**asynchronen**
</dt> <dd>

Daten Übertragungsmethode, mit der Zeichen in unregelmäßigen Intervallen über eine Zeile gesendet werden können, indem jedes Zeichen einem Startbit vorangestellt und mit einem Stoppbit darauf folgt. Siehe [*synchron*](s-tapgloss.md).

</dd> </dl>
