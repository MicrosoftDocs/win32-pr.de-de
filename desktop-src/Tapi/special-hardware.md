# <a name="special-hardware"></a>Spezielle Hardware

Einige der erweiterten Funktionen von TAPI erfordern, dass eine Anwendung z. b. Daten von Telefonen abrufen kann. Die meisten Telefone können nicht direkt mit Computern verbunden werden, um sprach Aufrufe zu steuern. Daher sind Sie derzeit nicht in der Lage, Telefoniefunktionen zu unterstützen, die über die passive Rolle hinausgehen In der Zukunft werden die Benutzer Telefon Sätze wie andere Peripheriegeräte installieren und konfigurieren. Die Sätze werden von Karten begleitet, die den Informationsfluss zwischen dem Computer und dem Telefon steuern. Client-/Serverkonfigurationen können auch verwendet werden, um den Benutzern die Nutzung von Telefondiensten zu ermöglichen, indem Sie eine Verbindung über ein LAN mit einem Server herstellen, auf dem ein solches Board und eine zugehörige Software installiert sind.
