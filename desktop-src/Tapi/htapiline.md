# <a name="htapiline"></a>HTAPILINE

Der **HTAPILINE-Datentyp** stellt das nicht transparente TAPI-Handle für eine Liniendatenstruktur dar. TAPI muss einen Wert dieses Typs in einen Verweis auf die entsprechende Datenstrukturinstanz auflösen. Der Dienstanbieter darf nicht versuchen, über diesen Verweis zu verweisen, als ob es sich um einen Zeiger handelte, Annahmen über seine Werte zu treffen oder seine Darstellung auf eine andere Weise zu interpretieren, als seinen Wert zu geeigneten Zeiten an TAPI zu übergeben.
