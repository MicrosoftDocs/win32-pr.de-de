# <a name="tspi_linedroponclose"></a>TSPI \_ linedroponclose

Die **TSPI-Funktion \_ linedroponclose** ist veraltet. TAPI löscht einen Aufruf nicht mehr automatisch, wenn der alleinige Besitzer das Handle für das liniengerät schließt, um die verteilte Kontrolle über Aufrufe über mehrere Client Computer in einer Client-/Server-Umgebung zu ermöglichen.
