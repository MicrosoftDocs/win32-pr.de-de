# <a name="tspi_linedroponclose"></a>TSPI \_ lineDropOnClose

Die **\_ TSPI-Funktion lineDropOnClose** ist veraltet. TAPI löscht einen Aufruf nicht mehr automatisch, wenn der einzige Besitzer sein Handle mit dem Zeilengerät schließt, um die verteilte Steuerung von Aufrufen auf mehrere Clientcomputer in einer Client-/Serverumgebung zu gewährleisten.
