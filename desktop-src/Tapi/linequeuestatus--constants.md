# <a name="linequeuestatus_-constants"></a>Linequeuestatus- \_ Konstanten

Die **linequeuestatus- \_ Konstanten** geben die Änderung des Status einer ACD-Warteschlange in einem Agent-Handler an.

<dl> <dt>

<span id="LINEQUEUESTATUS_NEWQUEUE"></span><span id="linequeuestatus_newqueue"></span>**linequeuestatus \_ NewQueue**
</dt> <dd> <dl> <dt>



Der verfügbaren Warteschlange wurde eine Warteschlange hinzugefügt.


</dt> </dl> </dd> <dt>

<span id="LINEQUEUESTATUS_QUEUEREMOVED"></span><span id="linequeuestatus_queueremoved"></span>**linequeuestatus \_ queueremoved**
</dt> <dd> <dl> <dt>



Diese Warteschlange wurde aus den verfügbaren entfernt.


</dt> </dl> </dd> <dt>

<span id="LINEQUEUESTATUS_UPDATEINFO"></span><span id="linequeuestatus_updateinfo"></span>**linequeuestatus \_ updateingefo**
</dt> <dd> <dl> <dt>



Aktualisierung der Informationen über die ACD-Warteschlange.


</dt> </dl> </dd> </dl>
