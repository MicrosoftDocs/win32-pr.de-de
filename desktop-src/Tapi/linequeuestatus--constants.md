# <a name="linequeuestatus_-constants"></a>LINEQUEUESTATUS-Konstanten \_

Die **LINEQUEUESTATUS-Konstanten \_** geben die Änderung des Status einer ACD-Warteschlange für einen Agenthandler an.

<dl> <dt>

<span id="LINEQUEUESTATUS_NEWQUEUE"></span><span id="linequeuestatus_newqueue"></span>**LINEQUEUESTATUS \_ NEWQUEUE**
</dt> <dd> <dl> <dt>



Den verfügbaren wurde eine Warteschlange hinzugefügt.


</dt> </dl> </dd> <dt>

<span id="LINEQUEUESTATUS_QUEUEREMOVED"></span><span id="linequeuestatus_queueremoved"></span>**LINEQUEUESTATUS \_ QUEUEREMOVED**
</dt> <dd> <dl> <dt>



Diese Warteschlange wurde aus den verfügbaren Warteschlangen entfernt.


</dt> </dl> </dd> <dt>

<span id="LINEQUEUESTATUS_UPDATEINFO"></span><span id="linequeuestatus_updateinfo"></span>**LINEQUEUESTATUS \_ UPDATEINFO**
</dt> <dd> <dl> <dt>



Aktualisierung der Informationen zur ACD-Warteschlange.


</dt> </dl> </dd> </dl>
