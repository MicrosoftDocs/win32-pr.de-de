# <a name="h323-tsp"></a>H323-TSP

\[H323 TSP ist nicht für die Verwendung in Windows Vista, Windows Server 2008 und nachfolgenden Versionen des Betriebssystems verfügbar. Die RTC-Client-API bietet eine ähnliche Funktionalität.\]

Der h323 TSP (h323. TSP) ermöglicht es einer Anwendung, Video-und Audioanrufe mithilfe des H. 323-Protokolls für IP-Telefonie zu tätigen und zu empfangen, wenn Sie mit dem [h323 MSP](./h323-msp.md) (H323msp.dll) gekoppelt ist.

Dieser TSP unterstützt [**Adresstyp**](./lineaddresstype--constants.md) Werte von lineadresssstype \_ PhoneNumber, lineadresstype \_ IPAddress und lineadresstype \_ Domain Name. Der Domänen Name "lineadresstype" \_ bezieht sich auf den Domänen Namen des Computers.

H323 MSP wurde in Windows 2000 SP1 verbessert, um die Kommunikation zwischen h323-und SDP-Clients zu ermöglichen.

Der h323 TSP wird mit Windows 2000 und höher installiert.
