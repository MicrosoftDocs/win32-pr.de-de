# <a name="tapi-3-terminal-manager"></a>TAPI 3 Terminal-Manager

Der TAPI 3-Terminal-Manager wird von Tapi3.dll auf Anforderung eines MSP erstellt und vom MSP für Aufgaben wie das Erstellen und Löschen dynamischer Terminals verwendet.

Die genauen Implementierungsdetails für die dynamischen Terminals eines MSP gehen über den Rahmen dieser Dokumentation hinaus. Viele Autoren können sich jedoch dafür entscheiden, einige der Funktionen des Media Streaming Terminal (MST) zu verwenden. Dieses Terminal basiert auf DirectShow, und Filter, die für ein bestimmtes Protokoll spezifisch sind, können in Verbindung mit dem MST geschrieben und verwendet werden.
