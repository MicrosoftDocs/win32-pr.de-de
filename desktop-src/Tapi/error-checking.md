# <a name="error-checking"></a>Fehlerüberprüfung

Auf der TAPI-Ebene kann eine Anwendung eine Vielzahl verschiedener Parameter übergeben, von denen viele möglicherweise ungültig sind. TAPI überprüft Parameter und gibt Fehler an die Anwendung zurück, ohne den Dienstanbieter aufrufen zu müssen. Jede Funktionsbeschreibung auf der TSPI-Ebene beschreibt die Parameter Fehler, die bereits getestet wurden. Der Dienstanbieter muss diese Tests nicht wiederholen, obwohl er alle zusätzlichen Validierungstests ausführen muss, die für die Funktion geeignet sind. In der folgenden Tabelle sind Titel und Beschreibungen der allgemeinen Parameter Gültigkeits Tests aufgeführt, die in vielen Funktionen angezeigt werden.



| Gültigkeits Test       | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|---------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Zeiger Gültigkeit    | TAPI hat bereits Zeiger auf den Datenspeicher getestet, um sicherzustellen, dass Sie auf einen lesbaren oder beschreibbaren Arbeitsspeicher für die für den Vorgang geeignete Größe zeigen. Außerdem wurde die Datenstruktur für Datenstrukturen mit variabler Größe, beginnend mit einem **dwtotalsize** -Member, überprüft, um sicherzustellen, dass die angeforderte Gesamtgröße verfügbar ist.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| Gültigkeitsdauer der Größe | Bei Datenstrukturen mit variabler Größe wurde die Datenstruktur so überprüft, dass genügend Speicherplatz für den Teil der Datenstruktur mit fester Größe und **dwtotalsize** für den festgelegten Teil ausreichend ist.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Offset/Größe Null  | Bei Datenstrukturen mit variabler Größen "... **Offset**"und"... **Size**"Felder, die den Teilen entsprechen, für die der Dienstanbieter festgelegt wurde, wurden vor dem Aufrufen des Dienstanbieters mit Nullwerten vor dem Aufrufen des Dienstanbieters                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| Gültigkeit des Handles     | TAPI stellt sicher, dass die Zeilen-, Telefon-und Anruf Handles (der definierten Typen [hdrvline](hdrvline.md), [hdrvphone](hdrvphone.md)und hdrvcall) gültig sind. Das heißt, es handelt sich um Werte, die fehlerfrei zurückgegeben wurden, als Handles in [**TSPI \_ lineOpen**](/windows/win32/api/tspi/nf-tspi-tspi_lineopen), [**TSPI \_ phoneopen**](/windows/win32/api/tspi/nf-tspi-tspi_phoneopen)oder eine der folgenden, die die Lebensdauer eines Anruf Handles startet: [**TSPI \_ linemakecall**](/windows/win32/api/tspi/nf-tspi-tspi_linemakecall)<br/> [**TSPI \_ linecompletetransfer**](/windows/win32/api/tspi/nf-tspi-tspi_linecompletetransfer)<br/> [**TSPI ( \_ lineforward)**](/windows/win32/api/tspi/nf-tspi-tspi_lineforward)<br/> [**TSPI- \_ linepickup**](/windows/win32/api/tspi/nf-tspi-tspi_linepickup)<br/> [**TSPI \_ lineprepareaddumconference**](/windows/win32/api/tspi/nf-tspi-tspi_lineprepareaddtoconference)<br/> [**TSPI \_ linesetupconference**](/windows/win32/api/tspi/nf-tspi-tspi_linesetupconference)<br/> [**TSPI \_ lineSetupTransfer**](/windows/win32/api/tspi/nf-tspi-tspi_linesetuptransfer)<br/> [**TSPI \_ lineunpark**](/windows/win32/api/tspi/nf-tspi-tspi_lineunpark)<br/> [**Zeile \_ Neunachrichten Aufrufe**](line-newcall.md)<br/> |
