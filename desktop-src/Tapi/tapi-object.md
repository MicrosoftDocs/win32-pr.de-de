# <a name="tapi-object"></a>TAPI-Objekt

Das TAPI-Objekt stellt alle Telefonieressourcen dar, auf die der lokale Computer zugreifen kann, sodass eine Anwendung alle lokalen und Remote Adressen aufzählen kann. Eine TAPI 3-Anwendung muss eine Instanz des TAPI-Objekts erstellen und anschließend initialisieren.

Das Beispiel zum [Initialisieren von TAPI](initialize-tapi.md) -Code zeigt ein Beispiel für die Erstellung von TAPI-Objekten.

Eine Liste aller dem TAPI-Objekt zugeordneten Schnittstellen finden Sie unter [TAPI-Objekt Schnittstellen](tapi-object-interfaces.md) .
