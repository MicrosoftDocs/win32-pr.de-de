# <a name="tapi-quick-function-reference"></a>TAPI-schnell Funktionsreferenz

Die folgende hyperverknüpfte Zusammenfassung der TAPI-Funktionen ist nach Kategorie des Vorgangs organisiert. Weitere Informationen finden Sie in den Übersichten zu unter [stützten Telefoniediensten](./assisted-telephony-overview.md) und [Service Levels](./tapi-levels-of-service.md).



| Referenz Abschnitt                                                                            | BESCHREIBUNG                                                                                         |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [Grundlegende telefoniedienstreferenz](basic-telephony-services-reference.md)                 | Funktionen, die mit Telefonieaufgaben auf der skalarebene verknüpft sind, z. b. [**linemakecall**](/windows/desktop/api/Tapi/nf-tapi-linemakecall) |
| [Referenz zu ergänzenden Telefoniediensten](supplementary-telephony-services-reference.md) | Funktionen mit erweiterten Funktionen, wie z. b. der Umleitung von anrufen.                                       |
| [Referenz zu erweiterten Telefoniediensten](extended-telephony-services-reference.md)           | Funktionen, die den Zugriff auf Dienstanbieter spezifische Funktionen ermöglichen, die nicht direkt von TAPI definiert werden.   |
| [Referenz zu unterstützten Telefoniediensten](assisted-telephony-services-reference.md)           | Funktionen, die Sprachanrufe und Medien Aufrufe einrichten, wie z. b. in mit einem Telefon Dialer.              |
