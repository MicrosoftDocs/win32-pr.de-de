# <a name="completion-id"></a>Vervollständigungs-ID

Der Vervollständigungs Bezeichner für den eingehenden-Befehl ist das Ergebnis einer Beendigungs Anforderung, die beendet wird.

**TAPI 2. x:** Weitere Informationen finden Sie unter [**linegetcallinfo**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo) (**dwcompletionid** -Member der [**LINECALLINFO**](/windows/win32/api/tapi/ns-tapi-linecallinfo) -Struktur, auf die von *lpcallinfo* verwiesen wird).

**TAPI 3. x:** Siehe [**itcallinfo:: get \_ callinfolong**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-get_callinfolong) (**cil \_ completionid** -Member von [**CallInfo \_ Long**](/windows/desktop/api/Tapi3if/ne-tapi3if-callinfo_long)).
