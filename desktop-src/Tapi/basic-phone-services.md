# <a name="basic-phone-services"></a>Basic Telefon Services

Grundlegende Telefondienste sind die Kerndienste, die von allen Telefongeräten unterstützt werden. Da grundlegende Telefondienste vom Dienstanbieter definiert sind, gibt es eine große Vielfalt im Kernsatz.
