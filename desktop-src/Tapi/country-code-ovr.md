# <a name="country-or-region-code"></a>Länder-oder Regions Code

Der Länder-oder Regions Code ist nur relevant, wenn der Adresstyp "lineadresstype \_ PhoneNumber" ist. Es identifiziert einen Bereich der Welt.

Alle Dienstanbieter, die die Verwendung von Telefonnummern unterstützen, müssen die Verwendung dieser Informationen unterstützen.

**TAPI 2. x:** Weitere Informationen finden Sie unter [**linegetcallinfo**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo) (**dwcountrycode** -Member von *lpcallinfo*).

**TAPI 3. x:** Siehe [**itcallinfo:: get \_ callinfolong**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-get_callinfolong) (**cil \_ CountryCode** -Member von [**CallInfo \_ Long**](/windows/desktop/api/Tapi3if/ne-tapi3if-callinfo_long)).
