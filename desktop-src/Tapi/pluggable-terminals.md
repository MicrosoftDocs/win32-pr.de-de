# <a name="pluggable-terminals"></a>Anschlussbare Terminals

Mit anschlussbaren Terminals kann eine Anwendung Während der Laufzeit Terminals erstellen. Terminals werden normalerweise von einem TSP-/MSP-Paar (Telefoniedienstanbieter/Mediendienstanbieter) erstellt und gesteuert, und dies bleibt das vielseitigste und stabilste Mittel zur Bearbeitung von Terminals. Einige Anwendungen müssen jedoch einfache Terminals in Situationen verwenden, in denen kein geeignetes TSP/MSP-Paar verfügbar ist. Nur Anwendungsautoren, die mit TSP-/MSP-Vorgängen gründlich vertraut sind, sollten zusammensetzbare Terminals verwenden.

In TAPI Version 3.1 werden dem TAPI-Modell anschlussbare Terminals hinzugefügt. Mithilfe von anschlussbaren Terminals kann ein Drittanbieter ein neues Terminalobjekt hinzufügen, das von jedem MSP verwendet werden kann.

Weitere Informationen finden Sie unter Pluggable Terminal Registration and Writing a Pluggable Terminal [(Pluggable-Terminalregistrierung](pluggable-terminal-registration.md) und Schreiben eines [pluggable-Terminals).](writing-a-pluggable-terminal.md)
