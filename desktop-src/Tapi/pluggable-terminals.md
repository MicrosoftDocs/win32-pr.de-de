# <a name="pluggable-terminals"></a>Austauschbare Terminals

Austauschbare Terminals ermöglichen es einer Anwendung, Terminals zur Laufzeit zu erstellen. Terminals werden normalerweise von einem Telefoniedienstanbieter/Media Service Provider-Paar (TSP/msp) erstellt und gesteuert. Dies ist die vielseitigere und stabilere Methode zum Bearbeiten von Terminals. Allerdings müssen einige Anwendungen einfache Terminals in Situationen verwenden, in denen ein entsprechendes TSP/MSP-paar nicht verfügbar ist. Nur Anwendungs Schreiber, die mit TSP-/MSP-Vorgängen gründlich vertraut sind, sollten austauschbare Terminals verwenden.

In TAPI, Version 3,1, werden austauschbare Terminals zum TAPI-Modell hinzugefügt. Austauschbare Terminals ermöglichen einem Drittanbieter das Hinzufügen eines neuen Terminal Objekts, das von allen MSP verwendet werden kann.

Weitere Informationen finden Sie unter [austauschbare Terminal Registrierung](pluggable-terminal-registration.md) und [Schreiben eines austauschbaren Terminals](writing-a-pluggable-terminal.md) .
