# <a name="stringformat_-constants"></a>StringFormat- \_ Konstanten

Die **StringFormat- \_ Konstanten** beschreiben verschiedene Zeichen folgen Formate.

<dl> <dt>

<span id="STRINGFORMAT_ASCII"></span><span id="stringformat_ascii"></span>**StringFormat- \_ ASCII**
</dt> <dd> <dl> <dt>



Gibt ein ASCII-Standard Zeichenformat mit einem Byte pro Zeichen an.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_BINARY"></span><span id="stringformat_binary"></span>**StringFormat- \_ Binärdatei**
</dt> <dd> <dl> <dt>



Dies ist ein Array von Zeichen ohne Vorzeichen. kann für numerische Werte verwendet werden.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_DBCS"></span><span id="stringformat_dbcs"></span>**StringFormat- \_ DBCS**
</dt> <dd> <dl> <dt>



Gibt das standardmäßige DBCS-Zeichenformat mit einem oder zwei Bytes pro Zeichen an.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_UNICODE"></span><span id="stringformat_unicode"></span>**StringFormat- \_ Unicode**
</dt> <dd> <dl> <dt>



Gibt das Unicode-Standard Zeichenformat mit zwei Bytes pro Zeichen an.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Bemerkungen

Keine Erweiterbarkeit. Alle 32 Bits sind reserviert.
