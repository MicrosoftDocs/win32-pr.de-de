# <a name="itsubstream-mspi"></a>Itsubstream (MSPi)

Die [**itsubstream**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstream) -Schnittstelle ist verfügbar, wenn ein MSP dem-Befehl zugeordnet ist und untergeordnete Datenströme implementiert. Mit den Methoden dieser Schnittstelle kann eine Anwendung Informationen in einem substream abrufen. , wenn der substream gestartet, angehalten oder beendet werden soll. und, um Terminals in einem substream auszuwählen oder deren Auswahl zu deaktivieren.

Die [**itsubstream**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstream) -Schnittstelle wird von einem MSP implementiert. Ausführliche Informationen zu dieser Schnittstelle finden Sie im Abschnitt **itsubstream** im Abschnitt MSP-Schnittstelle.
