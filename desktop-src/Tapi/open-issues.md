# <a name="open-issues"></a>Offene Probleme

-   Vor dem Windows 2000 Final Release kann sich der Pfad zum Herunterfahren des Aufrufs geringfügig ändern, um das Herunterfahren eines Aufrufs zu einem asynchronen Vorgang zu machen (schützt vor Deadlocks für MSPs, die Videos unterstützen).
-   Methoden und Member der Arbeitsthreadklasse, Terminalbasisklassen und abgeleiteten statischen Terminals werden nur von den Headerdateien selbst dokumentiert.
