# <a name="open-issues"></a>Offene Probleme

-   Vor der endgültigen Version von Windows 2000 kann sich der Pfad zum Herunterfahren des Aufrufes leicht ändern, um das Herunterfahren eines asynchronen Vorgangs zu vereinfachen (schützt vor Deadlocks für MSPs, die Video unterstützen).
-   Methoden und Member der Arbeits Thread Klasse, Terminal Basisklassen und abgeleiteten statischen Terminals werden nur von den Header Dateien selbst dokumentiert.
