# <a name="terminal-object"></a><span data-ttu-id="6c367-103">Terminal Objekt</span><span class="sxs-lookup"><span data-stu-id="6c367-103">Terminal Object</span></span>

<span data-ttu-id="6c367-104">In TAPI-Version 3,0 und höher verwendet das TAPI-Objektmodell Terminal Objekte zur Darstellung der Quelle oder Senke eines Mediendaten Stroms, der einer Telefon-oder Kommunikationssitzung zugeordnet ist.</span><span class="sxs-lookup"><span data-stu-id="6c367-104">In TAPI version 3.0 and later, the TAPI object model uses terminal objects to represent the source or sink of a media stream associated with a call or communications session.</span></span> <span data-ttu-id="6c367-105">Dieses Objektmodell ermöglicht es einer Anwendung, auf einer detaillierten Ebene anzugeben, wie Medien bei einem-Befehl verarbeitet werden.</span><span class="sxs-lookup"><span data-stu-id="6c367-105">This object model enables an application to specify, at a detailed level, how media are processed on a call.</span></span> <span data-ttu-id="6c367-106">Dieses Modell ermöglicht außerdem die gleichzeitige Auswahl mehrerer Terminals, sodass z. b. ein-Rückruf an einen audioredner ausgegeben und gleichzeitig aufgezeichnet werden kann.</span><span class="sxs-lookup"><span data-stu-id="6c367-106">This model also enables multiple terminals to be selected simultaneously, so for example, a call can be output to an audio speaker and recorded simultaneously.</span></span>

<span data-ttu-id="6c367-107">Das Terminal Objekt stellt eine Quelle oder einen Renderer dar, z. b. ein Mikrofon oder ein Redner.</span><span class="sxs-lookup"><span data-stu-id="6c367-107">The Terminal object represents a source or renderer, such as a microphone or a speaker.</span></span> <span data-ttu-id="6c367-108">Eine Anwendung wählt basierend auf der Medien Richtung und dem Typ oder den Typen, die an einer Kommunikationssitzung beteiligt sind, unter Verfügbare Terminals aus.</span><span class="sxs-lookup"><span data-stu-id="6c367-108">An application chooses among available terminals based on the media direction and type or types involved in a communications session.</span></span> <span data-ttu-id="6c367-109">Jeder zugeordnete Mediendaten Strom wird dann auf dem entsprechenden Terminal ausgewählt, um das Streaming zu starten.</span><span class="sxs-lookup"><span data-stu-id="6c367-109">Each associated media stream is then selected onto the appropriate terminal in order to start streaming.</span></span>

<span data-ttu-id="6c367-110">Terminals werden normalerweise von einem Medien Dienstanbieter (MSP) implementiert, und Terminal Objekte sind nicht verfügbar, wenn einer Kommunikationssitzung kein MSP zugeordnet ist.</span><span class="sxs-lookup"><span data-stu-id="6c367-110">Terminals are normally implemented by a media service provider (MSP) and terminal objects will not be available if there is no MSP associated with a communications session.</span></span> <span data-ttu-id="6c367-111">Eine Ausnahme besteht darin, dass eine Anwendung mit Windows 2000 SP1 und höher eine Form des [austauschbaren Terminal](pluggable-terminals.md)implementieren kann.</span><span class="sxs-lookup"><span data-stu-id="6c367-111">One exception is that with Windows 2000 SP1 and later, an application can implement a form of [pluggable terminal](pluggable-terminals.md).</span></span> <span data-ttu-id="6c367-112">Dies ermöglicht es einem Konferenz Server, Bridging-Terminals zu erstellen, sodass nicht-Windows 2000 SP1-oder nicht-Multicast-h323-Clients zu TAPI 3 Multi-Party-SDP/IP-Multicast Konferenzen hinzugefügt werden können.</span><span class="sxs-lookup"><span data-stu-id="6c367-112">This enables a conference server to create bridging terminals so that non-Windows 2000 SP1 or non-multicast H323 clients can be added to TAPI 3 multi-party SDP/IP multicast conferences.</span></span>

<span data-ttu-id="6c367-113">Jedes Terminal gehört zu einer [Terminal Klasse](terminal-class.md).</span><span class="sxs-lookup"><span data-stu-id="6c367-113">Each terminal belongs to a [terminal class](terminal-class.md).</span></span> <span data-ttu-id="6c367-114">Eine Terminal Klasse stellt einen Satz von Quell-oder Rendering-Funktionen dar.</span><span class="sxs-lookup"><span data-stu-id="6c367-114">A terminal class represents a set of source or render features.</span></span> <span data-ttu-id="6c367-115">Beispielsweise würde ein Terminal, das einem Satz von audioreferenten zugeordnet ist, als CLSID \_ speakersterminal identifiziert werden, und der Dienstanbieter wird die Implementierung der volumesteuerung erwarten.</span><span class="sxs-lookup"><span data-stu-id="6c367-115">For example, a terminal that maps to a set of audio speakers would be identified as CLSID\_SpeakersTerminal, and the service provider would be expected to implement volume control.</span></span> <span data-ttu-id="6c367-116">TAPI 3 definiert einen Satz von Terminal Klassen, ein MSP kann zusätzliche Klassen definieren, und eine Anwendung kann neue Terminal Klassen registrieren.</span><span class="sxs-lookup"><span data-stu-id="6c367-116">TAPI 3 defines a set of terminal classes, an MSP can define additional classes, and an application can register new terminal classes.</span></span> <span data-ttu-id="6c367-117">Jeder Terminal Klasse wird eine Globally Unique Identifier (GUID) zugewiesen.</span><span class="sxs-lookup"><span data-stu-id="6c367-117">Each terminal class is assigned a globally unique identifier (GUID).</span></span>

<span data-ttu-id="6c367-118">Aus der Sicht einer Anwendung wird ein Terminal durch den [Terminaltyp](/windows/desktop/api/Tapi3if/ne-tapi3if-terminal_type) und die [Richtung](/windows/desktop/api/Tapi3if/ne-tapi3if-terminal_direction)beschrieben.</span><span class="sxs-lookup"><span data-stu-id="6c367-118">From the point of view of an application, a terminal is described by its [terminal type](/windows/desktop/api/Tapi3if/ne-tapi3if-terminal_type) and [direction](/windows/desktop/api/Tapi3if/ne-tapi3if-terminal_direction).</span></span> <span data-ttu-id="6c367-119">Der Typ kann entweder statisch oder dynamisch sein.</span><span class="sxs-lookup"><span data-stu-id="6c367-119">The type may be either static or dynamic.</span></span> <span data-ttu-id="6c367-120">Ein statisches Terminal ist Hardware zugeordnet, z. b. ein Telefon oder Mikrofon.</span><span class="sxs-lookup"><span data-stu-id="6c367-120">A static terminal maps to hardware, such as a telephone or microphone.</span></span> <span data-ttu-id="6c367-121">Ein dynamisches Terminal ist einem vorübergehenden Objekt zugeordnet, z. b. einer Datei oder einem Videofenster.</span><span class="sxs-lookup"><span data-stu-id="6c367-121">A dynamic terminal maps to a transient object, such as a file or a video window.</span></span> <span data-ttu-id="6c367-122">Richtung beschreibt, ob ein bestimmtes Terminal eine Quelle oder ein Renderer ist.</span><span class="sxs-lookup"><span data-stu-id="6c367-122">Direction describes whether a given terminal is a source or a renderer.</span></span>

<span data-ttu-id="6c367-123">Die Funktionen eines bestimmten Terminal Objekts können sich je nach verwendetem aktuellen Dienstanbieter paar erheblich unterscheiden.</span><span class="sxs-lookup"><span data-stu-id="6c367-123">The capabilities of a given terminal object can vary considerably depending on the current service provider pair in use.</span></span> <span data-ttu-id="6c367-124">Der MSP für ein spezielles Gerät kann eine Schnittstelle mit Methoden implementieren, die für dieses Gerät geeignet sind.</span><span class="sxs-lookup"><span data-stu-id="6c367-124">The MSP for a specialized device might implement an interface with methods appropriate to that device.</span></span> <span data-ttu-id="6c367-125">Diese Schnittstelle kann auf dem Terminal Objekt und den Methoden, die einer Anwendung zur Verfügung gestellt werden, aggregiert werden.</span><span class="sxs-lookup"><span data-stu-id="6c367-125">That interface could be aggregated onto the terminal object and the methods made available to an application.</span></span> <span data-ttu-id="6c367-126">Weitere Informationen und Referenzmaterial finden Sie in der Media Service Provider-Dokumentation.</span><span class="sxs-lookup"><span data-stu-id="6c367-126">For more information and reference material, see the media service provider documentation.</span></span>
