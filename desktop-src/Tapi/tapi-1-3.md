# <a name="tapi-13"></a>TAPI 1,3

TAPI 1,3 (manchmal auch als TAPI 1,03 bezeichnet) war die erste veröffentlichte Version der TAPI-Spezifikation. Die TAPI 1,3-System Binärdateien (Tapi.dll und andere) befinden sich im TAPI 1,0 SDK, das sich nur auf dem MSDN-Abonnement (Microsoft Developer Network) befindet. Die System Binärdateien (im Verzeichnis Redist) sind Verteil Bar, aber nur für die unter Microsoft Windows 3,1 und Windows für Workgroups 3,11 vorgesehen. Sie werden nicht unterstützt und funktionieren in keiner Version von 32-Bit-Windows.

Die Header und Bibliotheken, die zum Entwickeln einer TAPI 1,3-Anwendung verwendet werden, sind nur im TAPI 1,0 SDK verfügbar und unterstützen nur die 16-Bit-Entwicklung. Tatsächlich sind die TAPI 1,3-Bibliotheken die einzigen 16-Bit-TAPI-Bibliotheken, die für alle 16-Bit-Anwendungen verfügbar sind und unabhängig von der Zielplattform oder Zielversion verwendet werden müssen.

Die TAPI 1,3-System Binärdateien unterstützen nur Win16, TAPI 1,3 TSPS und Anwendungen.
