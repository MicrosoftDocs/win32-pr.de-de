# <a name="tapi-13"></a>TAPI 1.3

TAPI 1.3 (manchmal auch als TAPI 1.03 bezeichnet) war die erste veröffentlichte Version der TAPI-Spezifikation. Die TAPI 1.3-Systembinärdateien (Tapi.dll und andere) befinden sich im TAPI 1.0 SDK, das sich nur im Abonnement der Ebene II des Microsoft-Entwickler Network (MSDN) befindet. Die Systembinärdateien (im VERZEICHNIS REDIST) sind weiterverteilbar, aber nur für die Ausführung in Microsoft Windows 3.1 und Windows für Arbeitsgruppen 3.11 konzipiert. Sie werden nicht unterstützt und funktionieren in keiner Version von 32-Bit-Windows.

Die Header und Bibliotheken, die zum Entwickeln einer TAPI 1.3-Anwendung verwendet werden, sind nur im TAPI 1.0 SDK verfügbar und unterstützen nur die 16-Bit-Entwicklung. Tatsächlich sind die TAPI 1.3-Bibliotheken die einzigen verfügbaren 16-Bit-TAPI-Bibliotheken und müssen für alle 16-Bit-Anwendungen verwendet werden, unabhängig von der Zielplattform oder Zielversion.

Die TAPI 1.3-Systembinärdateien unterstützen nur Win16-, TAPI 1.3-TSPs und Anwendungen.
