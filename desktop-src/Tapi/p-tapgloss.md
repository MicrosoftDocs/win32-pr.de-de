<dl> <dt>

<span id="tapi2.phone_centric_connection_tapgloss"></span><span id="TAPI2.PHONE_CENTRIC_CONNECTION_TAPGLOSS"></span>**Telefon zentrierte Verbindung**
</dt> <dd>

Eine Verbindung, die aus einer einzelnen Töpfe-Zeile besteht, in der der Computer über den Desktop telefonsatz mit dem Switch verbunden ist. Vergleichen Sie die [*Computer zentrierte Verbindung*](c-tapgloss.md).

</dd> <dt>

<span id="tapi2.plain_old_telephone_service_pots__tapgloss"></span><span id="TAPI2.PLAIN_OLD_TELEPHONE_SERVICE_POTS__TAPGLOSS"></span>**Plain Old Telefon Service (POTS)**
</dt> <dd>

Grundlegender Telefoniedienst, der standardmäßige einzeilige Telefone, Telefonleitungen und Zugriff auf das öffentliche, Umgeschaltete Netzwerk bereitstellt.

</dd> <dt>

<span id="tapi2.point_to_point_protocol_ppp__tapgloss"></span><span id="TAPI2.POINT_TO_POINT_PROTOCOL_PPP__TAPGLOSS"></span>**Point-to-Point-Protokoll (PPP)**
</dt> <dd>

Ein Kommunikationsprotokoll, das es einem Computer ermöglicht, mithilfe eines hoch Geschwindigkeits Modem über eine Standard-DFÜ-Telefonleitung eine Verbindung mit anderen Computern herzustellen.

</dd> <dt>

<span id="tapi2.pots_tapgloss"></span><span id="TAPI2.POTS_TAPGLOSS"></span>**Steinerne**
</dt> <dd>

Siehe *Plain Old Telefon Service (POTS)*.

</dd> <dt>

<span id="tapi2.pri_isdn_tapgloss"></span><span id="TAPI2.PRI_ISDN_TAPGLOSS"></span>**PRI-ISDN**
</dt> <dd>

Siehe *Primary Rate Interface (PRI-ISDN)*.

</dd> <dt>

<span id="tapi2.primary_rate_interface_pri_isdn__tapgloss"></span><span id="TAPI2.PRIMARY_RATE_INTERFACE_PRI_ISDN__TAPGLOSS"></span>**Primary Rate Interface (PRI-ISDN)**
</dt> <dd>

Ein Typ von ISDN-Dienst, der 23 b-Kanäle mit 64 Kbit/s und einem D-Kanal bei 64 Kbit/s (23b + D) bietet. In der Europäischen Union bietet PRI 30 b-Kanäle und zwei D-Kanäle (30b + 2D). Siehe [*Basic-Raten Schnittstelle (BRI-ISDN)*](b-tapgloss.md).

</dd> <dt>

<span id="tapi2.private_branch_exchange_pbx__tapgloss"></span><span id="TAPI2.PRIVATE_BRANCH_EXCHANGE_PBX__TAPGLOSS"></span>**privater Zweig Austausch (PBX)**
</dt> <dd>

Ein privates Telefonwechsel System, das sich normalerweise auf dem Standort eines Kunden mit einer Aufsichts Konsole befindet. Die PBX ist mit einer gemeinsamen Gruppe von Zeilen von einem oder mehreren zentral Niederlassungen verbunden, um den Dienst für eine Reihe einzelner Telefone, wie z. b. in einem Büro-, Geschäfts-oder Regierungsbüro, bereitzustellen. PBXs verwendet oftmals proprietäre Digital-Line-Protokolle, einige sind aber analog.

</dd> <dt>

<span id="tapi2.provisioning_tapgloss"></span><span id="TAPI2.PROVISIONING_TAPGLOSS"></span>**Bereitstellung**
</dt> <dd>

Die Kombination von Geräte-und Dienst Optionen, die eine ISDN-Linie bilden. Wenn Sie Ihre ISDN-Leitung anordnen, stellt das Telefonunternehmen die Zeile bereit, d. h., Sie konfiguriert Ihren ISDN-Dienst gemäß den physischen Funktionen des Schalters und den von Ihnen ausgewählten Optionen.

</dd> <dt>

<span id="tapi2.public_switched_telephone_network_pstn__tapgloss"></span><span id="TAPI2.PUBLIC_SWITCHED_TELEPHONE_NETWORK_PSTN__TAPGLOSS"></span>**öffentliches umschtetes Telefon Netzwerk (PSTN)**
</dt> <dd>

Das lokale analoge Telefon Netzwerk, das die Qualität des Dienstanbieter der 3,1 kHz-Sprachqualität (bearermodus) bereitstellt.

</dd> </dl>
