# <a name="cmspcallmultigraph-public-methods-called-by-mspaddress-object"></a>Öffentliche CMSPCallMultiGraph-Methoden, die vom MSPAddress-Objekt aufgerufen werden



| Öffentliche CMSPCallMultiGraph-Methoden               | Beschreibung                                                                                                                                |
|-------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [**Init**](/windows/desktop/api/Mspcall/nf-mspcall-cmspcallmultigraph-init)         | Wird vom MSP-Adressobjekt (in der [**Methode CreateMSPCall**](/windows/desktop/api/msp/nf-msp-itmspaddress-createmspcall)) aufgerufen, um das MSP-Aufrufobjekt zu initialisieren.    |
| [**Herunterfahren**](/windows/desktop/api/Mspcall/nf-mspcall-cmspcallmultigraph-shutdown) | Wird vom MSP-Adressobjekt (in der [**Methode ShutdownMSPCall**](/windows/desktop/api/msp/nf-msp-itmspaddress-shutdownmspcall)) aufgerufen, um das MSP-Aufrufobjekt herunterfahren. |
