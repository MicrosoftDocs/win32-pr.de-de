# <a name="kernel-mode-device-driver-tsp"></a>Kernel-Mode Device Driver TSP

Der Kernelmodus-TSP (Kmddsp.tsp) ermöglicht es einer Anwendung, ältere WAN-Miniports (z.B. NDIS 4.0-Treiber) als TAPI-Geräte zu verarbeiten.

Der Kernelmodus-TSP wird mit Windows Server 2003-Betriebssystemen, Windows XP, Windows 2000, Windows NT, Windows Edition, Windows 98 und Windows 95 installiert.
