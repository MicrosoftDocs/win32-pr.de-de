# <a name="kernel-mode-device-driver-tsp"></a>Gerätetreiber-TSP Kernel-Mode

Der kernelmodussp (kmddsp. TSP) ermöglicht es einer Anwendung, ältere WAN-Miniports (z. b. NDIS 4,0-Treiber) als TAPI-Geräte zu verarbeiten.

Der Kernelmodus-TSP wird mit den Betriebssystemen Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Millennium Edition, Windows 98 und Windows 95 installiert.
