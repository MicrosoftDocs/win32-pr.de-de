# <a name="wave-msp"></a>Wave-MSP

Der Wave MSP (Wavemsp.dll) passt ältere TSPs an, die vor der Einführung von MSPs in die MSP-basierte Streamingarchitektur von TAPI 3 geschrieben wurden. Einige der Legacy-TSPs machen Wavegerätebezeichner verfügbar. Dies war eine Möglichkeit für die Anwendung, das Streaming unter TAPI 2.1 zu steuern. Tapi3.dll verwendet den Wave-MSP für jeden TSP, der nicht über einen eigenen MSP verfügt, aber über ein Wellengerät verfügt.

Wave MSP wird mit Windows 2000 installiert.
