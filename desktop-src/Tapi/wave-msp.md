# <a name="wave-msp"></a>Wave-MSP

Der Wave MSP (Wavemsp.dll) eignet sich für ältere TSPS, die vor der Einführung von MSPs in die MSP-basierte streamingarchitektur von TAPI 3 geschrieben wurden. Einige der Legacy-TSPS machen Wellen Geräte Bezeichner verfügbar. Dies war eine Möglichkeit für die Anwendung, das Streaming unter TAPI 2,1 zu steuern. Tapi3.dll verwendet den Wave MSP für alle TSP, die nicht über einen eigenen MSP verfügen, aber über ein Wave-Gerät verfügen.

Der Wave MSP wird mit Windows 2000 installiert.
