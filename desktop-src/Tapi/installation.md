# <a name="installation-telephony-api"></a>Installation (telefonieapi)

Die primären Quellen für Installations Prozeduren sind das Ressourcenkit für das Ziel Betriebssystem, die Anwendungsdokumentation und die Anweisungen für den Dienstanbieter. Für Microsoft-Dienstanbieter enthält das entsprechende Ressourcenkit Anweisungen.

Der Installationsvorgang muss das Auflisten des Telefoniedienstanbietern als Abhängigkeit enthalten.

Wenn eine Anwendung oder ein Dienstanbieter die Registrierung verwendet, um persistente private Daten zu speichern, dürfen die Informationen nicht in den TAPI-Abschnitt der Registrierung eingefügt werden. Es ist nicht garantiert, dass das Format dieses Abschnitts in zukünftigen Versionen beibehalten wird.
