# <a name="installation-telephony-api"></a>Installation (Telefonie-API)

Die primären Quellen für Installationsverfahren sind das Resource Kit für das Zielbetriebssystem, die Anwendungsdokumentation und Dienstanbieteranweisungen. Für Microsoft-Dienstanbieter enthält das entsprechende Resource Kit Anweisungen.

Der Installationsvorgang muss das Auflisten von "Telefoniedienst" als Abhängigkeit beinhalten.

Wenn eine Anwendung oder ein Dienstanbieter die Registrierung zum Speichern persistenter privater Daten verwendet, dürfen die Informationen nicht im TAPI-Abschnitt der Registrierung gespeichert werden. Das Format dieses Abschnitts wird in zukünftigen Versionen nicht garantiert beibehalten.
