<span data-ttu-id="9670b-113">Diese APIs enthalten die Waveform-API, die comm-API und die Media Control Interface (MCI).</span><span class="sxs-lookup"><span data-stu-id="9670b-113">These APIs include the Waveform API, the Comm API, and the Media Control Interface (MCI).</span></span> <span data-ttu-id="9670b-114">Die Waveform-API wird für die Multimedia-Programmierung verwendet. die comm-API ist der Satz von Kommunikationsfunktionen, die vom Platform Software Development Kit (SDK) bereitgestellt werden, und MCI bietet eine allgemeine generalisierte Schnittstelle zum Steuern von Mediengeräten.</span><span class="sxs-lookup"><span data-stu-id="9670b-114">The Waveform API is used for multimedia programming, the Comm API is the set of communications functions provided by the Platform Software Development Kit (SDK), and the MCI provides a high-level generalized interface for controlling media devices.</span></span>

<span data-ttu-id="9670b-115">Beispielsweise kann eine Anwendung für Linien Geräte TAPI 2,2 verwenden, um eine Verbindung mit einer anderen Station herzustellen.</span><span class="sxs-lookup"><span data-stu-id="9670b-115">For example, for line devices, an application can use TAPI 2.2 to establish a connection to another station.</span></span> <span data-ttu-id="9670b-116">Nachdem die Verbindung hergestellt wurde, kann die Anwendung die Waveform-API (oder die MCI-waveaudioapi) auf dem zugeordneten Gerät verwenden, um Audiodaten über die Verbindung wiederzugeben (zu senden) und aufzuzeichnen (zu empfangen).</span><span class="sxs-lookup"><span data-stu-id="9670b-116">Once the connection is established, the application can then use the Waveform API (or the MCI Waveaudio API) on the associated device to play back (send) and record (receive) audio data over the connection.</span></span> <span data-ttu-id="9670b-117">Analog dazu verwendet eine Anwendung die Modem Konfigurations Erweiterungen der Communications-API, um den Mediendaten Strom zu steuern, wenn der Verbindungs Mediendaten Strom von einem Modem aus verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="9670b-117">Similarly, if the connection media stream is from a modem, an application would use the modem configuration extensions of the Communications API to control the media stream.</span></span>

<span data-ttu-id="9670b-118">Um TAPI 2,2 den Medienstrom Zugriff auf ein Telefon oder einen Anruf an einem Geräte Gerät zu gewährleisten, muss der Dienstanbieter sowohl den telefoniespi als auch den entsprechenden Media Stream SPI oder die Gerätetreiber Schnittstelle (DDI) implementieren.</span><span class="sxs-lookup"><span data-stu-id="9670b-118">To provide TAPI 2.2 with media-stream access to either a phone or a call on a line device, the service provider must implement both the Telephony SPI and the appropriate media stream SPI or device-driver interface (DDI).</span></span> <span data-ttu-id="9670b-119">Der Dienstanbieter kann Zeilen und Telefone gleichzeitig unterstützen.</span><span class="sxs-lookup"><span data-stu-id="9670b-119">The service provider can support lines and phones simultaneously.</span></span>

<span data-ttu-id="9670b-120">Da diese Geräteklassen und Medienstrom Vorgänge unabhängig voneinander funktionieren, muss die Koordination ihrer Verwendung auf Anwendungsebene erfolgen.</span><span class="sxs-lookup"><span data-stu-id="9670b-120">Because these device classes and media stream operations function independently of one another, coordination of their usage must occur at the application level.</span></span> <span data-ttu-id="9670b-121">Mehrere Anwendungen, die Aufrufe und Mediendaten Ströme gemeinsam nutzen, erfordern wahrscheinlich eine Koordination ihrer Aktivitäten auf Anwendungsebene, um eine widersprüchliche Verwendung von TAPI und der Media Stream-API zu verhindern.</span><span class="sxs-lookup"><span data-stu-id="9670b-121">Multiple applications that share calls and media streams will likely require coordination of their activities at the application level to prevent conflicting usage of TAPI and the media stream API.</span></span>
