# <a name="itstreamcontrol-mspi"></a>Itstreamcontrol (MSPi)

Die [**itstreamcontrol**](/windows/win32/api/tapi3if/nn-tapi3if-itstreamcontrol) -Schnittstelle wird für das-Objekt aufgerufen, wenn ein zugeordnetes MSP vorhanden ist. Die Methoden dieser Schnittstelle ermöglichen es einer Anwendung, Streams zu erstellen und zu entfernen.

Die [**itstreamcontrol**](/windows/win32/api/tapi3if/nn-tapi3if-itstreamcontrol) -Schnittstelle wird von einem MSP implementiert. Ausführliche Informationen zu dieser Schnittstelle finden Sie im Abschnitt **itstreamcontrol** im Abschnitt MSP-Schnittstelle.
