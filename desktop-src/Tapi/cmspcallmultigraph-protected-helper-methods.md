# <a name="cmspcallmultigraph-protected-helper-methods"></a>Cmspcallmultigraph, geschützte Hilfsmethoden



| Cmspcallmultigraph, geschützte Hilfsmethoden                             | BESCHREIBUNG                                                                               |
|-------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**Registerwaitevent**](/windows/desktop/api/Mspcall/nf-mspcall-cmspcallmultigraph-registerwaitevent)       | Fügt dem Thread Pool ein Wait-Ereignis hinzu.                                                     |
| [**Unregisterwaitevent**](/windows/desktop/api/Mspcall/nf-mspcall-cmspcallmultigraph-unregisterwaitevent)   | Entfernt ein Wait-Ereignis aus dem Thread Pool.                                                |
| [**Internalkreatestream**](/windows/desktop/api/Mspcall/nf-mspcall-cmspcallmultigraph-internalcreatestream) | Wird von " [**kreatestream**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-createstream) " aufgerufen, um ein Streamobjekt zu erstellen. |
