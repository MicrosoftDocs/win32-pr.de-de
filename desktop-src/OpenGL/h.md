<dl> <dt>

<span id="opengl_half_space"></span><span id="OPENGL_HALF_SPACE"></span>**halber Leerraum**
</dt> <dd>

Das Ergebnis einer Ebene, die den Leerraum trennt. Eine Ebene dividiert den Raum in zwei halbe Leerzeichen.

</dd> <dt>

<span id="opengl_homogenous_coordinates"></span><span id="OPENGL_HOMOGENOUS_COORDINATES"></span>**homogene Koordinaten**
</dt> <dd>

Ein Satz von n + 1-Koordinaten, die verwendet werden, um Punkte im n-dimensionalen, projectiven Raum darzustellen. Punkte im projektives Raum können als Punkte im euklidischen Raum und einige Punkte auf unendlich betrachtet werden. Die Koordinaten sind homogen, da eine Skalierung der einzelnen Koordinaten durch dieselbe Konstante ungleich NULL nicht den Punkt ändert, auf den die Koordinaten verweisen. Homogene Koordinaten sind in den Berechnungen der projektiven Geometrie und somit in Computergrafiken nützlich, bei denen Szenen in ein Fenster projiziert werden müssen.

</dd> </dl>
