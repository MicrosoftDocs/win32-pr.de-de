<dl> <dt>

<span id="opengl_half_space"></span><span id="OPENGL_HALF_SPACE"></span>**Halber Platz**
</dt> <dd>

Das Ergebnis einer Ebenenteilung des Raumes. Eine Ebene teilt den Raum in zwei Halbräume auf.

</dd> <dt>

<span id="opengl_homogenous_coordinates"></span><span id="OPENGL_HOMOGENOUS_COORDINATES"></span>**Homogene Koordinaten**
</dt> <dd>

Ein Satz von n+1-Koordinaten, die zur Darstellung von Punkten im n-dimensionalen projektiven Raum verwendet werden. Punkte im projektiven Raum können als Punkte im euklidischen Raum zusammen mit einigen Unendlichkeitspunkten betrachtet werden. Die Koordinaten sind homogen, da eine Skalierung der einzelnen Koordinaten durch dieselbe Konstante ungleich 0 (null) den Punkt, auf den die Koordinaten verweisen, nicht ändert. Homogene Koordinaten sind nützlich bei Berechnungen der projektiven Geometrie und somit in Computergrafiken, bei denen Szenen auf ein Fenster projiziert werden müssen.

</dd> </dl>
