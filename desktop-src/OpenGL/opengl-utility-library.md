# <a name="opengl-utility-library"></a>OpenGL-Hilfsprogrammbibliothek

Die-Bibliothek des OpenGL-Hilfsprogramms (GLU) enthält mehrere Gruppen von Funktionen, die die Haupt-OpenGL-Schnittstelle ergänzen, indem Sie Unterstützung für zusätzliche Funktionen Diese Hilfsprogrammfunktionen nutzen die wichtigsten OpenGL-Funktionen, sodass jede OpenGL-Implementierung die Hilfsfunktionen unterstützt.

> [!Note]  
> Das Präfix für die Funktionen der Hilfsprogrammbibliothek ist "glu" und nicht "GL".

 

Viele dieser Funktionen werden in den vorangehenden Abschnitten der Dokumentation beschrieben, wenn Ihre Themen auftreten. Diese Funktionen werden hier aus Gründen der Vollständigkeit aufgeführt. In den vorstehenden Abschnitten werden die Funktionen von Glu nicht erläutert. Ausführlichere Beschreibungen dieser Funktionen finden Sie im OpenGL- *Referenzhandbuch*.
