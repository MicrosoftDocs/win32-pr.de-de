# <a name="opengl-utility-library"></a>OpenGL-Hilfsprogrammbibliothek

Die GLU-Bibliothek (OpenGL Utility) enthält mehrere Gruppen von Funktionen, die die OpenGL-Kernschnittstelle ergänzen, indem zusätzliche Features unterstützt werden. Diese Hilfsfunktionen nutzen OpenGL-Kernfunktionen, sodass jede OpenGL-Implementierung garantiert die Hilfsfunktionen unterstützt.

> [!Note]  
> Das Präfix für Hilfsprogrammbibliotheksfunktionen ist "glu" statt "gl".

 

Viele dieser Funktionen werden in den vorherigen Abschnitten der Documenation beschrieben, wenn ihre Themen auftreten. Diese Funktionen sind aus Vollständigkeits- und Vollständigkeits-Heiterkeits-Richtigkeit hier aufgeführt. GLU-Funktionen, die in den vorherigen Abschnitten nicht erläutert werden, werden hier beschrieben. -   [Verwenden von Rückruffunktionen](using-callback-functions.md)
