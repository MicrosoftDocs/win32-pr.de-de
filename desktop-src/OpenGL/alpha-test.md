# <a name="alpha-test"></a>Alphatest

Der Alphatest (nur im RGBA-Modus durchgeführt) verwirft ein Fragment, abhängig vom Ergebnis eines Vergleichs zwischen dem Alphawert des Fragments und einem konstanten Verweiswert. Sie geben die Vergleichsfunktion und den Verweiswert mit [**glAlphaFunc an.**](glalphafunc.md)
