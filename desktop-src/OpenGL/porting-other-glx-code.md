# <a name="porting-other-glx-code"></a>Portieren von anderem glx-Code

Zusätzlich zu den in den vorherigen Abschnitten beschriebenen Xlib-und glx-Funktionen enthält Ihr Programm wahrscheinlich einige der anderen in der über [setzung der glx-Bibliothek](translating-the-glx-library.md)aufgeführten Funktionen von glx oder xlib. Schreiben Sie Ihren X-Fenster System Code in Windows um, und ersetzen Sie dabei die entsprechenden Funktionen.
