# <a name="generating-texture-coordinates"></a>Texturkoordinaten werden erzeugt.

Anstatt die Texturkoordinaten explizit anzugeben, können Sie von OpenGL als Funktion anderer Scheitelpunkt Daten mithilfe von [gltexgen \* ](gltexgen-functions.md)generiert werden. Nachdem die Texturkoordinaten angegeben oder generiert wurden, werden Sie von der Textur Matrix transformiert. Diese Matrix wird mit denselben Funktionen gesteuert, die für Matrix Transformationen verwendet werden (siehe [Matrix Transformationen](matrix-transformations.md)).
