# <a name="opengl"></a>Opengl

## <a name="purpose"></a>Zweck

Als Softwareschnittstelle für Grafikhardware rendert OpenGL mehrdimensionale Objekte in einen Framepuffer. Die Microsoft-Implementierung von OpenGL für das Windows-Betriebssystem ist branchenüblich grafiksoftware, mit der Programmierer qualitativ hochwertige, noch animierte dreidimensionale Farbbilder erstellen können. Die in diesem Abschnitt beschriebene OpenGL-Version ist 1.1.

Informationen zur Ausführung von OpenGL ES auf Windows finden Sie unter [ANGLE für Windows Store](https://github.com/microsoft/angle/wiki).

## <a name="where-applicable"></a>Anwendungsbereich

OpenGL ist auf Kompatibilität zwischen Hardware und Betriebssystemen aufgebaut. Diese Architektur erleichtert das Portieren von OpenGL-Programmen von einem System in ein anderes. Obwohl jedes Betriebssystem besondere Anforderungen hat, kann der OpenGL-Code in vielen Programmen wie gewohnt verwendet werden.

## <a name="developer-audience"></a>Entwicklergruppe

OpenGL wurde für die Verwendung durch C/C++-Programmierer entwickelt und erfordert Vertrautheit mit der Windows grafischen Benutzeroberfläche sowie der nachrichtengesteuerten Architektur.
