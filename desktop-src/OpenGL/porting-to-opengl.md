# <a name="porting-to-opengl"></a>Portieren auf OpenGL

OpenGL ist für die Kompatibilität von Hardware und Betriebssystemen hinweg konzipiert. Dieser Entwurf erleichtert Programmierern das Portieren von OpenGL-Programmen von einem System zu einem anderen. Obwohl jedes Betriebssystem besondere Anforderungen erfüllt, kann ein Großteil des OpenGL-Codes in ihren aktuellen Programmen unverändert verwendet werden. Zum Portieren ihrer OpenGL-Anwendung auf Microsoft Windows müssen Sie Ihre Programme so ändern, dass Sie mit den Windows-windowingsystemen funktionieren.

In allgemeinen werden Anwendungen von einer von zwei Plattformen zu OpenGL für Windows portiert:

-   Für das x-Fenster System entwickelte OpenGL-Anwendungen und die x-Bibliothek (Xlib)
-   IRIS GL-Anwendungen
