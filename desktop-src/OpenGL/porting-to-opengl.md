# <a name="porting-to-opengl"></a>Portieren zu OpenGL

OpenGL ist auf Kompatibilität zwischen Hardware und Betriebssystemen ausgelegt. Dieses Design erleichtert Programmierern das Portierung von OpenGL-Programmen von einem System auf ein anderes. Während jedes Betriebssystem über eindeutige Anforderungen verfügt, kann ein Großteil des OpenGL-Codes in Ihren aktuellen Programmen wie benützt verwendet werden. Um Ihre OpenGL-Anwendung zu Microsoft Windows zu portieren, müssen Sie Ihre Programme so ändern, dass sie mit den Windows Windowing-Systemen funktionieren.

Im Allgemeinen werden Anwendungen für Windows von einer von zwei Plattformen zu OpenGL portiert:

-   OpenGL-Anwendungen, die für das X-Fenstersystem und die X-Bibliothek (Xlib) entwickelt wurden
-   IRIS GL-Anwendungen
