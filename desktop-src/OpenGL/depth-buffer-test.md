# <a name="depth-buffer-test"></a>Tiefen Puffer Test

Der tiefen Puffer Test verwirft ein Fragment, wenn ein tiefen Vergleich fehlschlägt. " [**gldepthfunc**](gldepthfunc.md) " gibt die Vergleichsfunktion an. Wenn die Schablone aktiviert ist, wirkt sich das Ergebnis des tiefen Vergleichs auch auf den Aktualisierungs Wert des Schablonen Puffers aus.
