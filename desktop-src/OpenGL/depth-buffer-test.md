# <a name="depth-buffer-test"></a>Tiefenpuffertest

Der Tiefenpuffertest verwirft ein Fragment, wenn ein Tiefenvergleich fehlschlägt. [**glDepthFunc**](gldepthfunc.md) gibt die Vergleichsfunktion an. Wenn Schablonen aktiviert sind, wirkt sich das Ergebnis des Tiefenvergleichs auch auf den Aktualisierungswert des Schablonenpuffers aus.
