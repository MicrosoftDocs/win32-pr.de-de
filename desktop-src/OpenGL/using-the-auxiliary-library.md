# <a name="using-the-auxiliary-library"></a>Verwenden der Hilfsbibliothek

Mithilfe von OpenGL hat Silicon Graphics (SGI) die Hilfsbibliothek erstellt, um einfache Beispiel Programme für den *OpenGL-Programmier Leit Faden* zu schreiben. Der Quellcode für die Hilfsbibliothek wird zusammen mit den OpenGL-Beispielen mit dem Microsoft Platform SDK bereitgestellt. Untersuchen Sie den Quellcode, um zu verstehen, wie die Hilfsbibliothek aus OpenGL-Funktionen und-Routinen entwickelt wurde. Sie können hilfsbibliotheks Funktionen in ihren eigenen Programmen verwenden. Eine Beschreibung der Hilfsbibliothek finden Sie im OpenGL- *Programmier Handbuch*.
