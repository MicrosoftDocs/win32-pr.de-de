# <a name="using-the-auxiliary-library"></a>Verwenden der Hilfsbibliothek

Mit OpenGL hat Silicon Graphics (SGI) die Hilfsbibliothek erstellt, um einfache Beispielprogramme für das *OpenGL-Programmierhandbuch zu schreiben.* Der Quellcode für die Hilfsbibliothek wird zusammen mit den OpenGL-Beispielen im Microsoft Platform SDK bereitgestellt. Um zu verstehen, wie die Hilfsbibliothek aus OpenGL-Funktionen und -Routinen entwickelt wurde, untersuchen Sie den Quellcode. Sie können Hilfsbibliotheksfunktionen in Ihren eigenen Programmen verwenden. Eine Beschreibung der Hilfsbibliothek finden Sie im *OpenGL-Programmierhandbuch.*
