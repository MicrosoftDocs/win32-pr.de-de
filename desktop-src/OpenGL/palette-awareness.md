# <a name="palette-awareness"></a>Palettenbewusstsein

Die Anwendung muss auf die Nachrichten " [WM \_ palettechanged](/windows/desktop/gdi/wm-palettechanged)", " [WM \_ querynewpalette](/windows/desktop/gdi/wm-querynewpalette)" und " [WM \_ aktivieren](../inputdev/wm-activate.md) " Antworten, um die Paletten zu beachten und zu verwenden. Entwerfen Sie Ihre Anwendung so, dass Sie Paletten als Reaktion auf diese Nachrichten auswählen und umsetzen können.

Weitere Informationen zu Paletten und Paletteninformationen finden Sie in den Artikeln "Palette Awareness" und "The Palette Manager: How and Why" auf der Microsoft Developer Network Development Library Compact Disks.
