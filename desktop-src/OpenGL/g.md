# <a name="g-opengl"></a>G (OpenGL)

[A](a.md) [B](b.md) [C](c.md) [D](d.md) [E](e.md) [F](f.md) G [H](h.md) [I](i.md) J [K](jk.md) [L](l.md) [M](m.md) [N](n.md) [O](o.md) [P](p.md) [Q](q.md) [R](r.md) [S](s.md) [T](t.md) U [V](u-v.md) [W](w.md) X [Y Z](x-y-z.md)

<dl> <dt>

<span id="opengl_gamma_correction"></span><span id="OPENGL_GAMMA_CORRECTION"></span>**Gammakorrektur**
</dt> <dd>

Eine Funktion, die auf im Framepuffer gespeicherte Farben angewendet wird, um die nichtlineare Reaktion des Auges (und manchmal des Monitors) auf lineare Änderungen der Farbdichtewerte zu korrigieren.

</dd> <dt>

<span id="opengl_geometric_model"></span><span id="OPENGL_GEOMETRIC_MODEL"></span>**geometrisches Modell**
</dt> <dd>

Die Objektkoordinatenvertices und Parameter, die ein Objekt beschreiben. Beachten Sie, dass OpenGL keine Syntax für geometrische Modelle definiert, sondern eine Syntax und Semantik für das Rendering geometrischer Modelle.

</dd> <dt>

<span id="opengl_geometric_object"></span><span id="OPENGL_GEOMETRIC_OBJECT"></span>**geometrisches Objekt**
</dt> <dd>

Ein geometrisches Modell.

</dd> <dt>

<span id="opengl_geometric_primitive"></span><span id="OPENGL_GEOMETRIC_PRIMITIVE"></span>**geometrischer Primitiver**
</dt> <dd>

Ein Punkt, eine Linie oder ein Polygon.

</dd> <dt>

<span id="opengl_gouraud_shading"></span><span id="OPENGL_GOURAUD_SHADING"></span>**Gouraud-Schattierung**
</dt> <dd>

Nahtlose Interpolation von Farben über ein Polygon- oder Liniensegment. Farben werden an Scheitelpunkten zugewiesen und linear über das Primitiv interpoliert, um eine relativ gleichmäßige Farbvariation zu erzeugen. Wird auch als Smooth Shading bezeichnet.

</dd> <dt>

<span id="opengl_group"></span><span id="OPENGL_GROUP"></span>**Gruppe**
</dt> <dd>

Eine Gruppe von ein, zwei, drei oder vier Elementen, die jedes Pixel eines Bilds im Clientspeicher darstellt. Daher sind im Kontext eines Clientspeicherbilds eine Gruppe und ein Pixel identisch.

</dd> </dl>
