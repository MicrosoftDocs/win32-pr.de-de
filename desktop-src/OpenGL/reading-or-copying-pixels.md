# <a name="reading-or-copying-pixels"></a>Lesen oder Kopieren von Pixeln

Sie können Pixel aus dem Frame Puffer in den Arbeitsspeicher lesen, auf verschiedene Weise codieren und das codierte Ergebnis im Arbeitsspeicher mit " [**gllesepixels**](glreadpixels.md)" speichern. Darüber hinaus können Sie ein Rechteck mit Pixelwerten von einem Bereich des Frame Puffers mit [**glcopypixels**](glcopypixels.md)in ein anderes kopieren. Die Funktion " [**gllebuffer**](glreadbuffer.md) " steuert, von welchem Farb Puffer die Pixel gelesen oder kopiert werden.
