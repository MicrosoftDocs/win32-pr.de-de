# <a name="primitives-opengl"></a>Primitive (OpenGL)

Während der nächsten Verarbeitungsphase werden Primitive in den folgenden Schritten in Pixelfragmente konvertiert:

-   Primitive werden entsprechend abgeschnitten.
-   Die erforderlichen entsprechenden Anpassungen an den Farb- und Texturdaten werden vorgenommen, und die relevanten Koordinaten werden in Fensterkoordinaten transformiert.
-   Bei der Rasterung werden die beschnittenen Primitive in Pixelfragmente konvertiert.

Weitere Informationen finden Sie in den folgenden Themen:

-   [Freistellen](clipping.md)
-   [Transformieren in Fensterkoordinaten](transforming-to-window-coordinates.md)
-   [Rasterung](rasterizing.md)
-   [Referenz zu Primitiven](primitives-reference.md)
