# <a name="dithering"></a>Dithering

Wenn Dithering aktiviert ist, wird ein Ditheringalgorithmus auf den Farb- oder Farbindexwert des Fragments angewendet. Dieser Algorithmus hängt nur vom Wert des Fragments und seinen x- und y-Fensterkoordinaten ab.
