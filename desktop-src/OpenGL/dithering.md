# <a name="dithering"></a>Dithering

Wenn Dithering aktiviert ist, wird ein Dithering-Algorithmus auf die Farbe oder den Farb Index Wert des Fragments angewendet. Dieser Algorithmus hängt nur vom Wert des Fragments und seinen x-und y-Fenster Koordinaten ab.
