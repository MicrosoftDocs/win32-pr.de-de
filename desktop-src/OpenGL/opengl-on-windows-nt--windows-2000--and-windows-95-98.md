# <a name="opengl-on-windows"></a>OpenGL unter Windows

Bei der Microsoft-Implementierung von OpenGL im Microsoft Windows-Betriebssystem handelt es sich um eine Implementierung der dreidimensionalen, dreidimensionalen (3D-) Grafiksoftware Schnittstelle, mit der Programmierer qualitativ hochwertige und animierte 3D-Farbbilder erstellen. In dieser Übersicht wird die Windows-Implementierung von OpenGL beschrieben.
