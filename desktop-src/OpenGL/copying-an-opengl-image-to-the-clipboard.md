# <a name="copying-an-opengl-image-to-the-clipboard"></a>Kopieren eines OpenGL-Images in die Zwischenablage

Obwohl die aktuelle Version der Microsoft-Implementierung von OpenGL in Windows die Zwischenablage nicht direkt unterstützt, können Sie ein Windows OpenGL-Image in die Zwischenablage kopieren.

**So kopieren Sie ein OpenGL-Image in die Zwischenablage**

1.  Zeichnen Sie das Bild in eine Speicherbitmap oder eine erweiterte Metadatei.
2.  Kopieren Sie diese Bitmap in die Zwischenablage.
