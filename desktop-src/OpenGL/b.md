<dl> <dt>

<span id="opengl_back_face"></span><span id="OPENGL_BACK_FACE"></span>**Back face (Gesicht zurück)**
</dt> <dd>

Siehe [Gesicht](f.md).

</dd> <dt>

<span id="opengl_bit"></span><span id="OPENGL_BIT"></span>**Bit**
</dt> <dd>

Binäre Ziffer. Eine Zustandsvariable, die nur zwei mögliche Werte hat: 0 oder 1. Binäre Zahlen sind Konstruktionen von einem oder mehr Bits.

</dd> <dt>

<span id="opengl_bitmap"></span><span id="OPENGL_BITMAP"></span>**Bitmap**
</dt> <dd>

Ein rechteckiges Array von Bits. Außerdem wird der primitive vom [**glBitmap-Befehl**](glbitmap.md) gerendert, der seinen Bitmapparameter als Maske verwendet.

</dd> <dt>

<span id="opengl_bitplane"></span><span id="OPENGL_BITPLANE"></span>**Bitebene**
</dt> <dd>

Ein rechteckiges Array von Bits, die 1:1 mit Pixeln zugeordnet sind.

</dd> <dt>

<span id="opengl_blending"></span><span id="OPENGL_BLENDING"></span>**Mischung**
</dt> <dd>

Reduzieren von zwei Farbkomponenten auf eine Komponente, in der Regel als lineare Interpolation zwischen den beiden Komponenten.

</dd> <dt>

<span id="opengl_buffer"></span><span id="OPENGL_BUFFER"></span>**Puffer**
</dt> <dd>

Eine Gruppe von Bitebenen, die eine einzelne Komponente (z. B. Tiefe oder Grün) oder einen einzelnen Index (z. B. den Farbindex oder den Schablonenindex) speichert. Manchmal werden die roten, grünen, blauen und Alphapuffer zusammen als Farbpuffer (und nicht als Farbpuffer) bezeichnet.

</dd> </dl>
