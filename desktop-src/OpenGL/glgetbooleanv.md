# <a name="glgetbooleanv-function"></a>glgetbooleanv-Funktion

Die Funktion " **glgetbooleanv** " gibt den Wert oder die Werte eines ausgewählten Parameters zurück.

## <a name="syntax"></a>Syntax


```C++
void WINAPI glGetBooleanv(
   GLenum    pname,
   GLboolean *params
);
```

## <a name="remarks"></a>Bemerkungen

Diese Funktion gibt Werte für einfache Zustandsvariablen in OpenGL zurück. Der *PName* -Parameter ist eine symbolische Konstante, die angibt, welche Zustands Variable zurückgegeben werden soll, und *params* ist ein Zeiger auf ein Array vom Typ, in dem die zurückgegebenen Daten abgelegt werden sollen.

Die Typkonvertierung wird ausgeführt *, wenn Parameter* einen anderen Typ als den angeforderten Wert der Zustandsvariablen aufweisen. Wenn Sie **glgetbooleanv** aufrufen, wird ein Gleit Komma-oder ganzzahliger Wert nur dann in GL false konvertiert, wenn \_ er 0 (null) ist. Andernfalls wird Sie in GL true konvertiert \_ .

Wenn Sie " [**glgetintegerv**](glgetintegerv.md)" aufrufen, werden boolesche Werte als "GL true" oder "GL false" zurückgegeben \_ \_ , und die meisten Gleit Komma Werte werden auf den nächstgelegenen ganzzahligen Wert gerundet. Gleit Komma Farben und normale werden jedoch mit einer linearen Zuordnung zurückgegeben, die 1,0 dem positivsten darstellbaren ganzzahligen Wert und 1,0 dem negativsten darstellbaren ganzzahligen Wert zuordnet.

Wenn Sie " [**glgetfloatv**](glgetfloatv.md) " oder " [**glgetdoublev**](glgetdoublev.md)" aufrufen, werden boolesche Werte als "GL true" oder "GL false" zurückgegeben \_ \_ , und ganzzahlige Werte werden in Gleit Komma Werte konvertiert.

Sie können viele der booleschen Parameter leichter mit " [**glisenabled**](glisenabled.md)" Abfragen.
