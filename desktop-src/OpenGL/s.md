<dl> <dt>

<span id="opengl_server"></span><span id="OPENGL_SERVER"></span>**Servers**
</dt> <dd>

Der Computer, auf dem OpenGL-Befehle ausgeführt werden. Dies kann sich von dem Computer unterscheiden, von dem aus Befehle ausgegeben werden. Siehe auch Client.

</dd> <dt>

<span id="opengl_shading"></span><span id="OPENGL_SHADING"></span>**Schattierung**
</dt> <dd>

Der Prozess der Interpolation von Farben im Inneren eines Polygons oder zwischen den Scheitel Punkten einer Linie während der rasterisierung.

</dd> <dt>

<span id="opengl_single_buffering"></span><span id="OPENGL_SINGLE_BUFFERING"></span>**einfache Pufferung**
</dt> <dd>

Ein OpenGL-Kontext ohne Hintergrund Farb Puffer.

</dd> <dt>

<span id="opengl_stipple"></span><span id="OPENGL_STIPPLE"></span>**Stippel**
</dt> <dd>

Ein ein-oder zweidimensionales binäres Muster, das die Generierung von Fragmenten unterscheidet, wobei der Wert 0 (null) ist. Zeilen Stipeln sind eindimensional und werden relativ zum Anfang einer Linie angewendet. Polygon-Stipeln sind zweidimensional und werden mit einer festgelegten Ausrichtung auf das Fenster angewendet.

</dd> </dl>
