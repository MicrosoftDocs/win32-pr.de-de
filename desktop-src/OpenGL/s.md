<dl> <dt>

<span id="opengl_server"></span><span id="OPENGL_SERVER"></span>**Server**
</dt> <dd>

Der Computer, auf dem OpenGL-Befehle ausgeführt werden. Dies kann sich von dem Computer unterscheiden, von dem Befehle ausgegeben werden. Siehe auch Client.

</dd> <dt>

<span id="opengl_shading"></span><span id="OPENGL_SHADING"></span>**Schattierung**
</dt> <dd>

Der Prozess der Interpolation der Farbe innerhalb des Inneren eines Polygons oder zwischen den Scheitelpunkten einer Linie während der Rasterung.

</dd> <dt>

<span id="opengl_single_buffering"></span><span id="OPENGL_SINGLE_BUFFERING"></span>**Einzelne Pufferung**
</dt> <dd>

Ein OpenGL-Kontext ohne Hintergrundfarbpuffer.

</dd> <dt>

<span id="opengl_stipple"></span><span id="OPENGL_STIPPLE"></span>**Stipple**
</dt> <dd>

Ein ein- oder zweidimensionales binäres Muster, das die Generierung von Fragmenten unterschließt, deren Wert 0 (null) ist. Zeilenstipples sind eindimensional und werden relativ zum Anfang einer Zeile angewendet. Polygonstipples sind zweidimensional und werden mit einer festen Ausrichtung auf das Fenster angewendet.

</dd> </dl>
