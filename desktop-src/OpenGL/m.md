</dd> <dt>

<span id="opengl_matrix"></span><span id="OPENGL_MATRIX"></span>**Ko**
</dt> <dd>

Ein zweidimensionales Array von-Werten. OpenGL-Matrizen sind alle 4 x 4. Wenn Sie jedoch im Client Speicher gespeichert werden, werden Sie als 1x16-Arrays mit einer Dimension behandelt.

</dd> <dt>

<span id="opengl_modelview_matrix"></span><span id="OPENGL_MODELVIEW_MATRIX"></span>**Modelview-Matrix**
</dt> <dd>

Die 4X4-Matrix, die Punkte, Linien, Polygone und Raster Positionen von Objekt Koordinaten zu Augen Koordinaten transformiert.

</dd> <dt>

<span id="opengl_monitor"></span><span id="OPENGL_MONITOR"></span>**über**
</dt> <dd>

Das Gerät, das das Bild im Framebuffer anzeigt.

</dd> <dt>

<span id="opengl_motion_blurring"></span><span id="OPENGL_MOTION_BLURRING"></span>**Bewegungsunschärfe**
</dt> <dd>

Ein Verfahren, das die Art und Weise simuliert, in der ein Teil des Films angezeigt wird, wenn Sie ein Bild eines beweglicher Objekts betrachten oder wenn Sie die Kamera verschieben, wenn Sie ein Bild eines stationären Objekts betrachten. In Animationen ohne Bewegungsunschärfe kann Objektbewegung ruckartig erscheinen.

</dd> </dl>
