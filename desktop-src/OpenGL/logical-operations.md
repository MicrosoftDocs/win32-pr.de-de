# <a name="logical-operations"></a>Logische Vorgänge

Eine logische Operation kann zwischen dem Fragment und dem Wert, der an der entsprechenden Position im Framebuffer gespeichert ist, angewendet werden. Das Ergebnis ersetzt den aktuellen Frame Puffer-Wert. Sie wählen den gewünschten logischen Vorgang mit [**gllogicop**](gllogicop.md)aus. Logische Vorgänge werden nur für Farbindizes ausgeführt, nie für RGBA-Werte.
