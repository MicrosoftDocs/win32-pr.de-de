# <a name="logical-operations"></a>Logische Vorgänge

Ein logischer Vorgang kann zwischen dem Fragment und dem Wert angewendet werden, der an der entsprechenden Position im Framepuffer gespeichert ist. das Ergebnis ersetzt den aktuellen Framepufferwert. Sie wählen den gewünschten logischen Vorgang mit [**glLogicOp aus.**](gllogicop.md) Logische Vorgänge werden nur für Farbindizes und nie für RGBA-Werte ausgeführt.
