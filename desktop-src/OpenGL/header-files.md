# <a name="header-files"></a>Headerdateien

Anwendungen, die Folgendes verwenden:

-   Die wichtigsten OpenGL-Funktionen müssen die Header Datei <GL \\ GL. h> enthalten.
-   Die OpenGL-Hilfsprogrammbibliothek muss die Header Datei <GL \\ glu. h> enthalten.
-   Die Hilfsbibliothek des OpenGL-Programmier Handbuchs muss die Header Datei <GL \\ Glaux. h> enthalten.
-   Die WGL-Funktionen müssen die Header Datei "Windows. h" enthalten.
-   Die Windows-Funktionen, die die Microsoft-Implementierung von OpenGL in Windows unterstützen, müssen die Header Datei "Windows. h" enthalten.
