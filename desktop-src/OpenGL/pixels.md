# <a name="pixels"></a>Okkludierte

Fragmente werden im Framepuffer in Pixel konvertiert. Der Framepuffer ist in einem Satz logischer Puffer organisiert, die die Farb-, Tiefen-, Schablonen- und Akkumulationspuffer sind. Der Farbpuffer selbst besteht aus einer Vorder-links-, Front-Right-, Back-Left-, Back-Right- und einer Reihe von Hilfspuffern. Sie können Funktionen zum Steuern dieser Puffer und zum direkten Lesen oder Kopieren von Pixeln aus diesen Puffern ausstellen. (Beachten Sie, dass der von Ihnen verwendete OpenGL-Kontext möglicherweise nicht alle puffert.)

-   [Framebuffer-Vorgänge](framebuffer-operations.md)
-   [Lesen oder Kopieren von Pixeln](reading-or-copying-pixels.md)
-   [Pixelverweis](pixels-reference.md)
