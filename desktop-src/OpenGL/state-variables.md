-   [**Texturierung**](texturing-state-variables.md)
-   [**Framebuffer-Steuerelement**](framebuffer-control-state-variables.md)
-   [**Pixel**](pixel-state-variables.md)
-   [**Auswerter**](evaluator-state-variables.md)
-   [**Hinweis**](hint-state-variables.md)
-   [**Implementierungsabhängig**](implementation-dependent-state-variables.md)
-   [**Implementierungsabhängige Pixeltiefe**](implementation-dependent-pixel-depth-state-variables.md)
