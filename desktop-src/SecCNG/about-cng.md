# <a name="about-cng"></a>Informationen zu CNG

Kryptografie-API: Next Generation (CNG) ist der langfristige Ersatz für [*CryptoAPI.*](/windows/desktop/SecGloss/c-gly) CNG ist so konzipiert, dass es auf vielen Ebenen erweiterbar und kryptografieunabhängig im Verhalten ist.

## <a name="features"></a>Features

Informationen zu den Features von CNG finden Sie unter [CNG-Features.](cng-features.md)

## <a name="cryptographic-primitives"></a>Kryptografische Grundelemente

Weitere Informationen zu den kryptografischen Primitiven in CNG finden Sie unter [Kryptografische Grundtypen.](cryptographic-primitives.md)

## <a name="key-storage-and-retrieval"></a>Storage und Abrufen von Schlüsseln

Weitere Informationen zum Speichern und Abrufen von Schlüsseln in CNG finden Sie unter [Key Storage and Retrieval](key-storage-and-retrieval.md).

## <a name="key-import-and-export"></a>Schlüsselimport und -export

Weitere Informationen zum Importieren und Exportieren von Schlüsseln finden Sie unter [Importieren und Exportieren](key-import-and-export.md)von Schlüsseln.

## <a name="data-protection-api-next-generation"></a>Datenschutz-API: Nächste Generation

CNG bietet eine Reihe von APIs, die verwendet werden können, um Inhalte für Autorisierungsprinzipale auf mehreren Computern einfach zu verschlüsseln und zu entschlüsseln. Weitere Informationen finden Sie unter [CNG DPAPI](cng-dpapi.md).
