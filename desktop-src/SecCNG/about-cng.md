# <a name="about-cng"></a>Informationen über CNG

Cryptography API: Next Generation (CNG) ist die langfristige Ersetzung der [*CryptoAPI*](/windows/desktop/SecGloss/c-gly). CNG ist so konzipiert, dass es auf vielen Ebenen erweiterbar und im Verhalten kryptografieagnostisch ist.

## <a name="features"></a>Features

Weitere Informationen zu den Features von CNG finden Sie unter [CNG-Features](cng-features.md).

## <a name="cryptographic-primitives"></a>Kryptografische Grundelemente

Weitere Informationen zu den kryptografischen primitiven in CNG finden Sie unter [kryptografischen Primitives](cryptographic-primitives.md).

## <a name="key-storage-and-retrieval"></a>Schlüssel Speicherung und-Abruf

Weitere Informationen zum Speichern und Abrufen von Schlüsseln in CNG finden Sie unter [Schlüssel Speicherung und-Abruf](key-storage-and-retrieval.md).

## <a name="key-import-and-export"></a>Schlüssel Import und-Export

Weitere Informationen zum Importieren und Exportieren von Schlüsseln finden Sie unter [Schlüssel Import und-Export](key-import-and-export.md).

## <a name="data-protection-api-next-generation"></a>Datenschutz-API: nächste Generation

CNG stellt einen Satz von APIs bereit, die verwendet werden können, um Inhalte für Autorisierungs Prinzipale auf mehreren Computern problemlos zu verschlüsseln und zu entschlüsseln. Weitere Informationen finden Sie unter [CNG DPAPI](cng-dpapi.md).
