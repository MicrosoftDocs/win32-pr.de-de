## <a name="remarks"></a>Bemerkungen

Um eine benannte Kurve zu verwenden, führen Sie [**BCryptOpenAlgorithmProvider**](/windows/win32/api/Bcrypt/nf-bcrypt-bcryptopenalgorithmprovider) entweder mithilfe des **bcrypt \_ ECDSA- \_ Algorithmus** oder des **bcrypt \_ ECDH \_** -Algorithmus als Algorithmuskennung aus. Anschließend können Sie [**BCryptSetProperty**](/windows/win32/api/Bcrypt/nf-bcrypt-bcryptsetproperty) aufrufen und die Eigenschaft **bcrypt \_ ECC- \_ Kurven \_ Name** auf eine der oben genannten Kurven oder auf benannte Kurven festlegen, die auf dem Computer registriert sind, wie im `certutil -displayEccCurve` Befehl gezeigt.
