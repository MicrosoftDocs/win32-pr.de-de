## <a name="remarks"></a>Bemerkungen

Ab Windows 10 bietet CNG vordefinierte algorithmushandles für viele Algorithmen. Sie können diese Handles in jeder Situation verwenden, die ein algorithmushandle erfordert. Jeder [**BCryptSetProperty**](/windows/desktop/api/Bcrypt/nf-bcrypt-bcryptsetproperty) -Befehl schlägt jedoch fehl, da der algorithmushandle freigegeben wird und nicht geändert werden kann. Außerdem können diese Handles nicht im Kernel Modus bei "unql = Dispatch" verwendet werden.
