## <a name="remarks"></a>Hinweise

Ab diesem Windows 10 CNG vordefinierte Algorithmushandles für viele Algorithmen. Sie können diese Handles in jeder Situation verwenden, die ein Algorithmushandles erfordert. Allerdings schlägt jeder Aufruf von [**BCryptSetProperty**](/windows/desktop/api/Bcrypt/nf-bcrypt-bcryptsetproperty) fehl, da das Algorithmushandling freigegeben wird und nicht geändert werden kann. Darüber hinaus können diese Handles nicht bei IRQL=DISPATCH im Kernelmodus verwendet werden.
