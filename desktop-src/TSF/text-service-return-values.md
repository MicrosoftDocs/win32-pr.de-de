# <a name="text-service-return-values"></a>Rückgabewerte für den Text Dienst

Ein Text Dienst erzeugt einen Rückgabewert von 0x *HHHH* 0600. In der folgenden Tabelle wird der Rückgabewert für den Text Dienst aufgeführt.



| Rückgabecode/-wert                                                                                                                                                                                                                          | BESCHREIBUNG                                                           |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| <span id="TF_E_NOCONVERSION"></span><span id="tf_e_noconversion"></span><dl> <dt>**Tf \_ E \_ noconversion**</dt> <dt>0x80040600</dt> </dl> | Für den ausgewählten Textbereich ist keine Konvertierung verfügbar.<br/> |
