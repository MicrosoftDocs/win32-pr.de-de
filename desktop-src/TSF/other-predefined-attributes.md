# <a name="other-predefined-attributes"></a>Andere vordefinierte Attribute

Die folgenden Werte identifizieren verschiedene Attribute, die mit der [ITF context:: getappproperty](/windows/desktop/api/msctf/nf-msctf-itfcontext-getappproperty) -Methode abgerufen werden. Das Datenformat und der Inhalt der einzelnen Eigenschaftentypen sind eingeschlossen.

## <a name="properties"></a>Eigenschaften



| Eigenschaft                             | BESCHREIBUNG                                                                       |
|--------------------------------------|-----------------------------------------------------------------------------------|
| **Anwendung "Tsat" \_**                    | Nicht verwendet.                                                                         |
| **Die Anwendung "der Anwendung" ist \_ \_ inkorretgrammar.**  | Enthält einen Wert ungleich 0 (null), wenn der Text einen Grammatikfehler oder andernfalls 0 (null) enthält.  |
| **Anwendungs Rechtschreibprüfung \_ \_** | Enthält einen Wert ungleich 0 (null), wenn der Text einen Rechtschreibfehler oder andernfalls 0 (null) enthält. |
| **\_andere Benutzer**                 | Nicht verwendet.                                                                         |
