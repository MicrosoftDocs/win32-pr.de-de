# <a name="how-to-set-up-text-services-framework"></a>Einrichten des Text Dienst-Frameworks

Zum Einrichten des Text Dienste-Frameworks (TSF), das vom Desktop verwendet werden soll, müssen Sie Text Dienste installieren. In den folgenden Prozeduren wird beschrieben, wie Sie Text Dienste und die TSF-sprach Steuerungs Leiste unter Windows XP installieren.

## <a name="how-to-install-text-services"></a>Vorgehensweise beim Installieren von Text Diensten

1.  Öffnen Sie in der Systemsteuerung Regions **-und Sprachoptionen**.
2.  Klicken Sie auf der Registerkarte **Sprachen** auf **Details**.
3.  Klicken Sie im Fenster **Text Dienste und Eingabe Sprachen** auf der Registerkarte **Einstellungen** auf **Hinzufügen**. Wählen Sie in der angezeigten Liste den gewünschten Text Dienst aus. Der ausgewählte Text Dienst wird im folgenden Fenster unter der zugehörigen Sprache angezeigt.

    ![Fenster "Text Dienst Auswahl"](images/text-service-selection.gif)

    Um einen Text Dienst zu entfernen, wählen Sie ihn in der Liste in diesem Fenster aus, und klicken Sie auf **Entfernen**.

    Wenn die sprach Leiste sichtbar ist, können Sie einen Text Dienst wie folgt schnell hinzufügen oder entfernen. Klicken Sie mit der rechten Maustaste auf die sprach Leiste, und wählen Sie **Einstellungen** aus, oder klicken Sie auf die Schaltfläche **Optionen** Optionen rechts neben der Sprach ![ Leiste, ](images/options-btn.gif) und wählen Sie **Einstellungen** aus. Fügen Sie dann einen Text Dienst hinzu, oder entfernen Sie ihn, wie in Schritt 3 beschrieben.

## <a name="how-to-show-the-language-bar"></a>Anzeigen der sprach Leiste

Die sprach Leiste ist ein Steuerelement auf dem Bildschirm, mit dem der Benutzer einen Text Dienst konfigurieren kann. Damit die sprach Leiste angezeigt wird, muss mindestens ein Text Dienst installiert sein.

1.  Öffnen Sie in der Systemsteuerung Regions **-und Sprachoptionen**.
2.  Klicken Sie auf der Registerkarte **Sprachen** auf **Details**.
3.  Klicken Sie im Fenster **Text Dienste und Eingabe Sprachen** auf **sprach Leiste**.
4.  Aktivieren Sie das Kontrollkästchen **sprach Leiste auf dem Desktop anzeigen** . Deaktivieren Sie ggf. zuerst das Kontrollkästchen **Erweiterte Text Dienste deaktivieren** .
5.  Klicken Sie auf **OK**.
6.  Klicken Sie auf der Registerkarte **Einstellungen** **auf über** nehmen und dann auf **OK**.
7.  Klicken Sie auf **OK** , um die Systemsteuerung zu beenden. Eine unverankerte sprach Leiste wird angezeigt.

> [!Note]
>
> Wenn Sie weitere Informationen zum Verschieben, ausblenden, anzeigen und Ändern der sprach Leiste und zum Entfernen oder Installieren eines Text Dienstanbieter benötigen, klicken Sie  ![ auf die Schaltfläche Hilfe Hilfe Auswahl in sprach leisten Optionen in ](images/help-balloon.gif) der sprach Leiste, und wählen Sie im Dropdown Menü die Option **sprach leisten Hilfe** aus. Wenn die Schaltfläche Hilfe nicht sichtbar ist, fügen Sie Sie der sprach Leiste hinzu, indem Sie auf die Options Schaltfläche **Optionen** auf der sprach leisten Schaltfläche rechts neben der sprach Leiste klicken ![ ](images/options-btn.gif) und im Dropdown Menü **Hilfe** ![ Hilfe Auswahl in den Optionen der sprach Leiste auswählen ](images/help-balloon.gif) .

 

Um zu überprüfen, ob TSF ordnungsgemäß installiert ist und ausgeführt wird, drücken Sie die Taste Crtl + ALT + ENTF, und klicken Sie auf **Task** Auf der Registerkarte **Prozesse** sollte der ctfmon.exe Prozess ausgeführt werden.
