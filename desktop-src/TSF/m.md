# <a name="m-text-services-framework"></a>M (Text Dienst-Framework)

<dl> <dt>

<span id="_tsf_modal_gly"></span><span id="_TSF_MODAL_GLY"></span>**aufkommen**
</dt> <dd>

Ein Begriff, der ein sekundäres Fenster beschreibt, das die Interaktion von Benutzern mit anderen Fenstern einschränkt. Ein modales Fenster, z. b. ein Dialogfeld, muss vom Benutzer geschlossen werden, bevor die Anwendung fortgesetzt werden kann.
