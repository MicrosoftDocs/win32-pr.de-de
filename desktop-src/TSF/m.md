# <a name="m-text-services-framework"></a>M (Textdienstframework)

<dl> <dt>

<span id="_tsf_modal_gly"></span><span id="_TSF_MODAL_GLY"></span>**Modale**
</dt> <dd>

Ein Begriff, der ein sekundäres Fenster beschreibt, das die Benutzerinteraktion mit anderen Fenstern einschränkt. Ein modales Fenster, z. B. ein Dialogfeld, muss vom Benutzer geschlossen werden, bevor die Anwendung fortgesetzt werden kann.

</dd> </dl>
