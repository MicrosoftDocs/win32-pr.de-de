# <a name="tf_lbmenuf_-constants"></a>TF- \_ lbmenuf- \_ \* Konstanten

Die **tf \_ lbmenuf \_ \** _-Konstanten werden in der [itfmenu:: addmenuitem](/windows/desktop/api/Ctfutb/nf-ctfutb-itfmenu-addmenuitem) -Methode verwendet, um die Eigenschaften eines Menü Elements in der sprach Leiste anzugeben.



| Konstante/Wert                                                                                                                                                                                                                                         | BESCHREIBUNG                                     |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------|
| <span id="TF_LBMENUF_CHECKED"></span><span id="tf_lbmenuf_checked"></span><dl> <dt>_ * TF \_ Lbmenuf \_ aktiviert * *</dt> <dt>0x01</dt> </dl>                | Das Menü Element ist aktiviert.<br/>            |
| <span id="TF_LBMENUF_SUBMENU"></span><span id="tf_lbmenuf_submenu"></span><dl> <dt>**Tf \_ Lbmenuf- \_ Untermenü**</dt> <dt>0x02</dt> </dl>                | Das Menü Element ist ein Untermenü.<br/>          |
| <span id="TF_LBMENUF_SEPARATOR"></span><span id="tf_lbmenuf_separator"></span><dl> <dt>**Tf \_ Lbmenuf- \_ Trenn**</dt> Zeichen <dt>0x04</dt> </dl>          | Das Menü Element ist ein Trennzeichen.<br/>        |
| <span id="TF_LBMENUF_RADIOCHECKED"></span><span id="tf_lbmenuf_radiochecked"></span><dl> <dt>**Tf \_ Lbmenuf \_ Radio Check**</dt> <dt>0x08</dt> </dl> | Das Menü Element ist ein Optionsfeld.<br/> |
| <span id="TF_LBMENUF_GRAYED"></span><span id="tf_lbmenuf_grayed"></span><dl> <dt>**Tf \_ Lbmenuf, \_ grau**</dt> , <dt>0x10</dt> </dl>                   | Das Menü Element ist deaktiviert.<br/>           |
