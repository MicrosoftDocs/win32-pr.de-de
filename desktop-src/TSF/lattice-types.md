# <a name="lattice-types"></a>Gitter Typen

Die folgenden Konstanten identifizieren bestimmte Typen von Gitter Eigenschaften. Diese Werte werden mit den Methoden [itflmlattice:: QueryType](/windows/desktop/api/Ctffunc/nf-ctffunc-itflmlattice-querytype) und [itflmlattice:: enumlatticeelements](/windows/desktop/api/Ctffunc/nf-ctffunc-itflmlattice-enumlatticeelements) verwendet.

<dl> <dt>

<span id="GUID_LMLATTICE_VER1_0"></span><span id="guid_lmlattice_ver1_0"></span>**GUID \_ Liebe meine lifeattice \_ ver1 \_ 0**
</dt> <dd>

Gibt an, dass die Gitter Eigenschaft Version 1,0 ist.

</dd> </dl>
