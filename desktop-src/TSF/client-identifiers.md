# <a name="client-identifiers"></a>Client Bezeichner

## <a name="applications"></a>Anwendungen

Eine Anwendung ruft ihren Client Bezeichner durch Aufrufen von [ITfThreadMgr:: Aktivierung](/windows/desktop/api/Msctf/nf-msctf-itfthreadmgr-activate)ab.

## <a name="text-services"></a>Text Dienste

Ein Text Dienst erhält seine Client-ID, wenn die Text-Dienst- [itftextinputprocessor:: Aktivierungs](/windows/desktop/api/Msctf/nf-msctf-itftextinputprocessor-activate) -Methode aufgerufen wird.
