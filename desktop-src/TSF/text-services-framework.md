# <a name="text-services-framework-text-services-framework"></a>Text Dienst Framework (Text Dienste-Framework)

## <a name="purpose"></a>Zweck

Das Microsoft Windows-Text Dienste-Framework (TSF) ist ein Systemdienst, der als verteilbare Komponente für Windows verfügbar ist. TSF bietet ein einfaches und skalierbares Framework für die Bereitstellung erweiterter Texteingabe-und natürlicher Sprachtechnologien. TSF kann in Anwendungen oder als TSF-Text Dienst aktiviert werden. Ein TSF-Text Dienst bietet mehrsprachige Unterstützung und bietet Text Dienste wie Tastatur Prozessoren, Handschrifterkennung und Spracherkennung.

## <a name="where-applicable"></a>Anwendungsbereich

Das Text Dienst Framework gilt für Windows-basierte Computer, die Text Dienste und Windows XP oder höhere Versionen des Betriebssystems verwenden.

## <a name="developer-audience"></a>Entwicklergruppe

Das Text Services-Framework ist für die Verwendung durch Component Object Model-Programmierer (com) konzipiert, die die C/C++-Programmiersprachen verwenden. Programmierer sollten mit Text Diensten für Windows-basierte Computer vertraut sein. Es wird empfohlen, die Handschrifterkennung, Spracherkennung und Programmierung für mehrsprachige Unterstützung zu kennen.
