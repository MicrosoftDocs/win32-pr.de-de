# <a name="a-text-services-framework"></a>A (Text Dienst-Framework)

<dl> <dt>

<span id="_tsf_anchor_gly"></span><span id="_TSF_ANCHOR_GLY"></span>**Ker**
</dt> <dd>

Einer der Elemente, die den Anfang und das Ende eines Text Bereichs markieren. Siehe [Anker](ranges.md).

</dd> <dt>

<span id="_tsf_application_character_position_gly"></span><span id="_TSF_APPLICATION_CHARACTER_POSITION_GLY"></span>**Zeichenposition der Anwendung (ACP)**
</dt> <dd>

Eine Null basierte Zeichenposition innerhalb eines kontinuierlichen Textstreams, die relativ zum Anfang des Streams ist. Eine ACP wird mit einem Long Integer-Wert angegeben. Siehe [Text Stores](text-stores.md).

</dd> </dl>
