# <a name="tf_-constants"></a>TF- \_ \* Konstanten

TSF Atom-Funktionsaufrufe verwenden diese Konstanten, um Thread Eingabe Prozesse zu verwalten.



| Konstante/Wert                                                                                                                                                                                                                                                                       | BESCHREIBUNG                                                                                          |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="TF_PROCESS_ATOM"></span><span id="tf_process_atom"></span><dl> <dt>**Tf \_ \_Atom**</dt> - <dt>Text verarbeiten ( \_ CTF- \_ Prozess \_ Atom \_ )</dt> </dl>                               | Wird zum Benachrichtigen von Atom-Funktionen verwendet, die von einem Thread Eingabeprozess initiiert wurden.<br/>                  |
| <span id="TF_ENABLE_PROCESS_ATOM"></span><span id="tf_enable_process_atom"></span><dl> <dt>**Tf \_ Enable \_ Process \_ Atom**</dt> <dt>Text ( \_ CTF \_ enable \_ Process \_ Atom \_ )</dt> </dl> | Wird zum Benachrichtigen von TSF über Atom-Funktionen verwendet, dass ein Thread Eingabeprozess aktiviert wurde.<br/> |
