# <a name="tf_floatinglangbar_-constants"></a>TF \_ floatinglangbar- \_ \* Konstanten

Definiert eine TF \_ floatinglangbar- \_ \* Konstanten Zeichenfolge für den Titel des sprach leisten Fensters.



| Konstante/Wert                                                                                                                                                                                                                                                                                  | BESCHREIBUNG                                                                                                                                                                  |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TF_FLOATINGLANGBAR_WNDTITLEW"></span><span id="tf_floatinglangbar_wndtitlew"></span><dl> <dt>**Tf \_ Floatinglangbar \_ wndtitlew**</dt> <dt>LTF \_ floatinglangbar \_ wndtitle</dt> </dl> | Die Zeichenfolge der Unicode-Zeichen L "TF \_ floatinglangbar \_ wndtitle" für den Titel.<br/>                                                                                  |
| <span id="TF_FLOATINGLANGBAR_WNDTITLEA"></span><span id="tf_floatinglangbar_wndtitlea"></span><dl> <dt>**Tf \_ Floatinglangbar \_ wndtitlea**</dt> <dt>tf \_ floatinglangbar \_ wndtitle</dt> </dl>  | Die Zeichenfolge der ANSI-Zeichen "TF \_ floatinglangbar \_ wndtitle" für den Titel.<br/>                                                                                      |
| <span id="TF_FLOATINGLANGBAR_WNDTITLE"></span><span id="tf_floatinglangbar_wndtitle"></span><dl> <dt>**Tf \_ floatinglangbar \_ wndtitle**</dt> <dt>tf \_ floatinglangbar \_ wndtitlea</dt> </dl>    | Wenn Unicode definiert ist, übernimmt den Wert der "TF \_ floatinglangbar \_ wndtitlew"-Konstante. Andernfalls übernimmt den Wert der "TF \_ floatinglangbar \_ wndtitlea"-Konstante.<br/> |



## <a name="remarks"></a>Bemerkungen

Obwohl in dieser Gruppe drei Konstanten vorhanden sind, ist die primäre Version TF \_ floatinglangbar \_ wndtitle, die je nach Bedarf entweder den ANSI-oder Unicode-Wert annimmt.
