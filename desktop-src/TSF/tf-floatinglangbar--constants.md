# <a name="tf_floatinglangbar_-constants"></a>TF \_ \_ FLOATINGLANGBAR-Konstanten \*

Definiert eine TF \_ FLOATINGLANGBAR-Konstantenzeichenfolge \_ \* für den Titel des Sprachleistenfensters.



| Konstante/Wert                                                                                                                                                                                                                                                                                  | BESCHREIBUNG                                                                                                                                                                  |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TF_FLOATINGLANGBAR_WNDTITLEW"></span><span id="tf_floatinglangbar_wndtitlew"></span><dl> <dt>**TF \_ FLOATINGLANGBAR \_ WNDTITLEW**</dt> <dt>LTF \_ FloatingLangBar \_ WndTitle</dt> </dl> | Die Zeichenfolge der Unicode-Zeichen L"TF \_ FloatingLangBar \_ WndTitle" für den Titel.<br/>                                                                                  |
| <span id="TF_FLOATINGLANGBAR_WNDTITLEA"></span><span id="tf_floatinglangbar_wndtitlea"></span><dl> <dt>**TF \_ FLOATINGLANGBAR \_ WNDTITLEA**</dt> <dt>TF \_ FloatingLangBar \_ WndTitle</dt> </dl>  | Die Zeichenfolge der ANSI-Zeichen "TF \_ FloatingLangBar \_ WndTitle" für den Titel.<br/>                                                                                      |
| <span id="TF_FLOATINGLANGBAR_WNDTITLE"></span><span id="tf_floatinglangbar_wndtitle"></span><dl> <dt>**TF \_ FLOATINGLANGBAR \_ WNDTITLE**</dt> <dt>TF \_ FLOATINGLANGBAR \_ WNDTITLEA</dt> </dl>    | Wenn Unicode definiert ist, nimmt den Wert der \_ TF FLOATINGLANGBAR \_ WNDTITLEW-Konstante an. Andernfalls nimmt den Wert der \_ TF FLOATINGLANGBAR \_ WNDTITLEA-Konstante an.<br/> |



## <a name="remarks"></a>Hinweise

Obwohl in dieser Gruppe drei Konstanten vorhanden sind, ist die primäre konstant TF \_ FLOATINGLANGBAR \_ WNDTITLE, die je nach Bedarf entweder den ANSI- oder Unicode-Wert annimmt.
