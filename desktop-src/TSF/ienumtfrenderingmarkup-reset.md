# <a name="ienumtfrenderingmarkupreset-method"></a>Ienumtfrenderingmarkup:: Reset-Methode

Die **ienumtfrenderingmarkup:: Reset** -Methode setzt das Enumeratorobjekt zurück, indem die aktuelle Position an den Anfang der enumerationssequenz verschoben wird.

## <a name="syntax"></a>Syntax


```C++
HRESULT Reset();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

| Wert                                                                                | BESCHREIBUNG                           |
|--------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**S \_ OK**</dt> </dl> | Die Methode war erfolgreich.<br/> |
