# <a name="text-services-framework-return-values"></a>Rückgabewerte des Text Dienst-Frameworks

Schnittstellen Methoden des Text Dienste-Frameworks ergeben Rückgabewerte im Bereich von 0x *HHHH* 0200 bis 0x *HHHH* 0600, wobei *HHHH* ein Hexadezimalwert ist, der vom Schweregrad Code und dem Einrichtungs Code eines bestimmten Rückgabewerts abhängt.

Die Gruppe der Rückgabewerte wird in die folgenden Kategorien eingeteilt.

-   [Rückgabewerte für den Text Dienst](text-service-return-values.md)
-   [Manager-Rückgabewerte](manager-return-values.md)
-   [Rückgabewerte für Text Speicher](text-store-return-values.md)
