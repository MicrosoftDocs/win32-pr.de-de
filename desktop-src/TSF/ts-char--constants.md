# <a name="ts_char_-constants"></a>TS \_ Char- \_ \* Konstanten

Die TS- \_ Char- \_ \* Konstanten beschreiben den Typ des Zeichens.



| Konstante/Wert                                                                                                                                                                                                                                   | BESCHREIBUNG                                                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------|
| <span id="TS_CHAR_EMBEDDED"></span><span id="ts_char_embedded"></span><dl> <dt>**TS \_ Char \_ Embedded**</dt> <dt>(0xfffc)</dt> </dl>          | Das Zeichen ist ein Unicode 2,1-Objekt Austausch Zeichen.<br/>                             |
| <span id="TS_CHAR_REGION"></span><span id="ts_char_region"></span><dl> <dt>**TS \_ Char- \_ Region**</dt> <dt>(0)</dt> </dl>                     | Das Zeichen ist eine Regions Grenze.<br/>                                                      |
| <span id="TS_CHAR_REPLACEMENT"></span><span id="ts_char_replacement"></span><dl> <dt>**TS \_ Char- \_ Ersetzung**</dt> <dt>(0xFFFD)</dt> </dl> | Das Zeichen ist ein Platzhalter Zeichen oder ein Unicode-Ersatz Zeichen.<br/> |
