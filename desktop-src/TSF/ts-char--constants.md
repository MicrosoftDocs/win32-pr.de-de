# <a name="ts_char_-constants"></a>TS \_ \_ \* CHAR-Konstanten

Die TS \_ \_ \* CHAR-Konstanten beschreiben den Typ des Zeichens.



| Konstante/Wert                                                                                                                                                                                                                                   | BESCHREIBUNG                                                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------|
| <span id="TS_CHAR_EMBEDDED"></span><span id="ts_char_embedded"></span><dl> <dt>**TS \_ CHAR \_ EMBEDDED**</dt> <dt>( 0xfffc )</dt> </dl>          | Das Zeichen ist ein Unicode 2.1-Objektersetzungszeichen.<br/>                             |
| <span id="TS_CHAR_REGION"></span><span id="ts_char_region"></span><dl> <dt>**TS \_ CHAR \_ REGION**</dt> <dt>( 0 )</dt> </dl>                     | Das Zeichen ist eine Bereichsgrenze.<br/>                                                      |
| <span id="TS_CHAR_REPLACEMENT"></span><span id="ts_char_replacement"></span><dl> <dt>**TS \_ CHAR \_ REPLACEMENT**</dt> <dt>( 0xfffd )</dt> </dl> | Das Zeichen ist ein Platzhalterzeichen für ausgeblendeten Text oder ein Unicode-Ersatzzeichen.<br/> |
