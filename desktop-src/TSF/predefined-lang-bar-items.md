# <a name="predefined-language-bar-items"></a>Vordefinierte sprach leisten Elemente

Die folgenden [**GUID**](/windows/desktop/TSF/predefined-lang-bar-items) -Werte identifizieren die vordefinierten sprach leisten Elemente. Diese Werte werden mit [**GetItem**](/windows/desktop/api/Ctfutb/nf-ctfutb-itflangbaritemmgr-getitem) zum Abrufen eines bestimmten Sprach leisten Elements verwendet.

<span id="_______________GUID_LBI_SAPILAYR_CFGMENUBUTTON_____________"></span><span id="_______________guid_lbi_sapilayr_cfgmenubutton_____________"></span> GUID \_ LBI \_ sapilayr \_ cfgmenubutton 
</dt> <dd>

Sprach leisten-Menü Element für Sprach Tools.

</dd> <dt>

<span id="_______________GUID_TFCAT_TIP_HANDWRITING_____________"></span><span id="_______________guid_tfcat_tip_handwriting_____________"></span> GUID \_ tfcat \_ Tip- \_ Handschrift 
</dt> <dd>

Handschrift Elementtyp Element.

</dd> <dt>

<span id="_______________GUID_TFCAT_TIP_KEYBOARD_____________"></span><span id="_______________guid_tfcat_tip_keyboard_____________"></span> GUID \_ tfcat \_ Tip \_ Tastatur 
</dt> <dd>

Element des Tastatur Typs.

</dd> </dl>
