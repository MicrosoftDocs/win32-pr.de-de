# <a name="ts_ch_-constants"></a>TS- \_ ch- \_ \* Konstanten

Die TS- \_ ch- \_ \* Konstanten werden von TSF intern verwendet, um anzugeben, dass Text Löschungen aufgetreten sind.



| Konstante/Wert                                                                                                                                                                                                                               | BESCHREIBUNG                                                        |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------|
| <span id="TS_CH_PRECEDING_DEL"></span><span id="ts_ch_preceding_del"></span><dl> <dt>**TS \_ Vor \_ \_**</dt> gestelltem del <dt>(1)</dt> </dl> | Text, der der Zeichenposition vorangestellt ist, wurde gelöscht.<br/> |
| <span id="TS_CH_FOLLOWING_DEL"></span><span id="ts_ch_following_del"></span><dl> <dt>**TS \_ \_Nach \_**</dt> der ENTF. <dt>(2)</dt> </dl> | Der Text, der auf die Zeichenposition folgt, wurde gelöscht.<br/> |
