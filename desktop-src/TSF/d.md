# <a name="d-text-services-framework"></a>D (Text Dienst-Framework)

<dl> <dt>

<span id="_tsf_document_lock_gly"></span><span id="_TSF_DOCUMENT_LOCK_GLY"></span>**Dokument Sperre**
</dt> <dd>

Ein Dokument Zustand, in dem der TSF-Manager Text lesen oder ändern kann, ohne dass der Text auf andere Weise geändert wird. Eine Dokument Sperre kann schreibgeschützt sein oder Lese-/Schreibzugriff aufweisen.

</dd> </dl>
