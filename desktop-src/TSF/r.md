# <a name="r-text-services-framework"></a>R (Text Dienste-Framework)

<dl> <dt>

<span id="_tsf_range_gly"></span><span id="_TSF_RANGE_GLY"></span>**Bereich**
</dt> <dd>

Ein-Objekt, das einen Textabschnitt beschreibt, der in einem Textstream vorhanden ist. Ein Bereich wird verwendet, um Text zu lesen, zu schreiben und zu bearbeiten. Wird auch als *Textbereich* bezeichnet.

Siehe auch *Region*.

</dd> <dt>

<span id="_tsf_reconversion_gly"></span><span id="_TSF_RECONVERSION_GLY"></span>**erneuten Konvertierung gelöscht**
</dt> <dd>

Der Vorgang, mit dem der Benutzer alternativen Text für den eingefügten Text auswählen kann.

</dd> <dt>

<span id="_tsf_region_gly"></span><span id="_TSF_REGION_GLY"></span>**AUM**
</dt> <dd>

Ein Textabschnitt, der basierend auf der Formatierung als eindeutiger Bearbeitungsbereich behandelt wird.

</dd> </dl>
