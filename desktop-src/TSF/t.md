# <a name="t-text-services-framework"></a>T (Text Dienst-Framework)

<dl> <dt>

<span id="_tsf_text_service_gly"></span><span id="_TSF_TEXT_SERVICE_GLY"></span>**Text Dienst**
</dt> <dd>

Ein Anwendungstyp, der Texteingaben über Sprache, mehrsprachige Tastatur oder Stift Gerät ermöglicht. Ein Text Dienst kann auch Tools zur Textbearbeitung, z. b. Rechtschreibprüfung und Grammatik Überprüfung, aktivieren.

Siehe auch *Textprozessor*.

</dd> <dt>

<span id="_tsf_text_services_framework_gly"></span><span id="_TSF_TEXT_SERVICES_FRAMEWORK_GLY"></span>**Text Services-Framework (TSF)**
</dt> <dd>

Ein Geräte unabhängiges, sprach neutrales und erweiterbares System für Texteingabe-und natürliche Sprachtechnologien.

</dd> </dl>
