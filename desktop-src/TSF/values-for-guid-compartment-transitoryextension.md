# <a name="values-for-guid_compartment_transitoryextension"></a>Werte für das GUID-Depot \_ \_ transitor Extension

Die folgenden Werte des "GUID \_ Depot \_ transitoryextension"-Depots werden verwendet, um das Verhalten der transitorischen Erweiterung zu steuern.

<dl> <dt>

<span id="TF_TRANSITORYEXTENSION_NONE"></span><span id="tf_transitoryextension_none"></span>**TF \_ transitor Extension \_ None**
</dt> <dd>

Dieser Wert beendet die vorübergehendes-Erweiterung.

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_FLOATING"></span><span id="tf_transitoryextension_floating"></span>**TF \_ transitor Extension \_ Floating**
</dt> <dd>

Dieser Wert startet die vorübergehendes-Erweiterung mit der gleitenden Benutzeroberfläche.

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_ATSELECTION"></span><span id="tf_transitoryextension_atselection"></span>**TF \_ transitor Extension \_ atselection**
</dt> <dd>

Dieser Wert startet die vorübergehendes-Erweiterung mit der Popup-Benutzeroberfläche an der IP-Adresse oder der Auswahl des übergeordneten Dokument-Managers.

</dd> </dl>
