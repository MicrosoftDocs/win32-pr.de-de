# <a name="installing-an-expert-to-network-monitor-21"></a>Installieren eines Experten für Netzwerkmonitor 2.1

Um eine Experten-DLL zu installieren, kopieren Sie sie in das Unterverzeichnis Netzwerkmonitor \\ Experts (z. B. C: \\ Program Files \\ NetMon2 \\ Experts). Nach der Installation wird der Experte erkannt und beim nächsten Start Netzwerkmonitor verfügbar.

Die folgende Abbildung zeigt myExpert DLL, MyExpert, die aus dem Beispielcode erstellt und in das Unterverzeichnis Experts kopiert \\ wurde.
