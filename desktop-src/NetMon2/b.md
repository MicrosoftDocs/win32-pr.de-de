# <a name="b-network-monitor"></a>B (Netzwerkmonitor)

<dl> <dt>

<span id="_netmon_binary_large_object_gly"></span><span id="_NETMON_BINARY_LARGE_OBJECT_GLY"></span>**Binary Large Object (BLOB)**
</dt> <dd>

Die generische Datenstruktur von Netzwerkmonitor. Sie können ein BLOB verwenden, um Konfigurations- und Speicherortinformationen zu kommunizieren und zu speichern sowie anwendungsspezifische Informationen zu speichern, ohne dass sich dies auf andere Daten in einem BLOB auswirkt.

</dd> <dt>

<span id="_netmon_blob_gly"></span><span id="_NETMON_BLOB_GLY"></span>**Blob**
</dt> <dd>

Weitere Informationen finden Sie unter Binary Large Object.

</dd> <dt>

<span id="_netmon_buffer_gly"></span><span id="_NETMON_BUFFER_GLY"></span>**Puffer**
</dt> <dd>

Ein reservierter Teil des Arbeitsspeichers, in dem Daten vorübergehend auf den Abschluss einer Übertragung an ein oder von einem Speichergerät oder an einen anderen Speicherort im Arbeitsspeicher warten.

</dd> </dl>
