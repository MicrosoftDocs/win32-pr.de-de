# <a name="event-reference-page"></a>Ereignisreferenzseite

In diesem Abschnitt wird die Ereignisverweisseite (ERP) beschrieben, ein HTML-basiertes Dokument, das Sie schreiben können, um Ereignisse anzuzeigen, die während des Betriebs eines Netzwerkmonitor Experten auftreten. Nachdem Sie die Anforderungen für die Erstellung von Experten definiert haben, können Sie ERPs verwenden, um Netzwerkprobleme Netzwerkmonitor lösen zu können.
