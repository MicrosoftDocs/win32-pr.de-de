# <a name="event-reference-page"></a>Seite "Ereignis Verweis"

In diesem Abschnitt wird die Ereignis Referenzseite (ERP) beschrieben, ein HTML-basiertes Dokument, das Sie schreiben können, um Ereignisse anzuzeigen, die während des Vorgangs eines Netzwerkmonitor Experten auftreten. Nachdem Sie die Anforderungen für das Erstellen von Experten definiert haben, können Sie Erps verwenden, um die Netzwerkmonitor Behebung von Netzwerkproblemen zu unterstützen.
