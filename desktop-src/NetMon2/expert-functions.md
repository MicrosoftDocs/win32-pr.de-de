# <a name="expert-functions"></a>Expertenfunktionen

Die folgenden Hilfsfunktionen können nur von Experten aufgerufen werden, die die Funktion " [Run](run.md) " oder " [configure](configure.md) " exportieren.



| Funktion                                         | BESCHREIBUNG                                                                             |
|--------------------------------------------------|-----------------------------------------------------------------------------------------|
| [Expertgetframe](expertgetframe.md)             | Ruft einen angegebenen Frame aus der Erfassung ab.                                               |
| [Profil Status](expertindicatestatus.md) | Gibt den Prozentsatz des Abschlusses der Analyse der Erfassung durch den Experten an.             |
| [Expertgetstartupinfo](expertgetstartupinfo.md) | Ruft die Startinformationen für den Experten ab.                                       |
| [Expertmemorysize](expertmemorysize.md)         | Ruft die Größe des Arbeitsspeichers ab, der durch einen Aufrufe der Funktion " **expertenlocmemory** " zugeordnet wird. |
| [ExpertSubmitEvent](expertsubmitevent.md)       | Gibt ein Problem an und ruft Informationen über das Problem ab, falls vorhanden.          |
| ["Expertenlocmemory"](expertallocmemory.md)       | Weist dem Experten Arbeitsspeicher zu.                                                        |
| [Expertrebelegcmemory](expertreallocmemory.md)   | Ändert die Größe des Arbeitsspeichers, der von der Funktion " **expertenlocmemory** " zugeordnet wird.         |
| [Expertenarbeits Speicher](expertfreememory.md)         | Gibt vom Experten zugeordnete Arbeitsspeicher frei.                                                   |
