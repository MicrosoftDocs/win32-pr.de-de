# <a name="system-requirements"></a>Systemanforderungen

Die mindesten Computerkonfigurationen unterscheiden sich; Für Netzwerkmonitor Entwicklungszwecke wird empfohlen, dass ein Entwickler über einen Computer verfügt, der die folgenden Mindestspezifikationen enthält:

-   Pentium-Prozessor
-   32 MB RAM
-   30 MB verfügbarer Festplattenspeicher
-   Unterstützung von Videomonitor und Grafikkarte für 256 Farben mit einer Auflösung von 800 x 600
-   Netzwerkverbindung
-   Microsoft Internet Explorer 4.01
-   Microsoft Visual C++ Version 5.0

Zum Testen einer mit Netzwerkmonitor Anwendung ist ein Computer mit den folgenden Mindestspezifikationen erforderlich:

-   486-Prozessor mit mindestens 50 MHz
-   16 MB RAM
-   20 MB verfügbarer Festplattenspeicher
-   Netzwerkverbindung
