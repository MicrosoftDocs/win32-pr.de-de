# <a name="system-requirements"></a>Systemanforderungen

Die minimalen Computerkonfigurationen sind unterschiedlich. zu Netzwerkmonitor Entwicklungszwecken wird empfohlen, dass ein Entwickler über einen Computer mit den folgenden minimalen Spezifikationen verfügt:

-   Pentium-Prozessor
-   32 MB RAM
-   30 MB verfügbarer Festplattenspeicher
-   Videomonitor-und Grafikkarten Unterstützung für 256-Farben mit einer 800 X 600-Auflösung
-   Netzwerkverbindung
-   Microsoft Internet Explorer 4,01
-   Microsoft Visual C++ Version 5,0

Das Testen einer Anwendung, die mit Netzwerkmonitor entwickelt wurde, erfordert einen Computer mit den folgenden Mindestspezifikationen:

-   486 Prozessor mit mindestens 50 MHz
-   16 MB RAM
-   20 MB verfügbarer Festplattenspeicher
-   Netzwerkverbindung
