# <a name="installing-network-monitor-files"></a>Installieren von Netzwerkmonitor Dateien

Gehen Sie folgendermaßen vor, um den Netzwerkmonitor Teil des Platform Software Development Kit (SDK) zu installieren und auszuführen.

**So installieren Sie Netzwerkmonitor Platform SDK-Dateien und führen Sie aus**

1.  Um die Dateien zu installieren, wählen Sie während des Platform SDK-Setups die Netzwerkmonitor Dokumente und den Beispielcode aus.

    Nach Abschluss des Setups erstellt das Platform SDK-Setup Programm eine Batchdatei, Setenv.bat.

2.  Führen Sie Setenv.bat aus, um die erforderlichen Umgebungsvariablen zu initialisieren.
3.  Um die Beispiele zu kompilieren, führen Sie **NMAKE** aus, der in Microsoft Visual Studio enthalten ist.
