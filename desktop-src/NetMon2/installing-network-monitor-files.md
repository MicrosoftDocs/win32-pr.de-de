# <a name="installing-network-monitor-files"></a>Installieren von Netzwerkmonitor Dateien

Gehen Sie wie folgt vor, um Netzwerkmonitor Teil des Platform Software Development Kit (SDK) zu installieren und ausführen.

**So installieren und führen Sie Netzwerkmonitor Platform SDK-Dateien aus**

1.  Um die Dateien zu installieren, wählen Sie während der Einrichtung des Plattform-SDK die Netzwerkmonitor und Beispielcode aus.

    Nach Abschluss des Setups erstellt das Plattform-SDK-Setupprogramm eine Batchdatei Setenv.bat.

2.  Führen Sie zum Initialisieren der erforderlichen Umgebungsvariablen Setenv.bat.
3.  Führen Sie zum Kompilieren der Beispiele **nmake** aus, das in der Microsoft Visual Studio.
