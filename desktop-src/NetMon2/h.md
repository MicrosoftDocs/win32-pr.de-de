# <a name="h-network-monitor"></a>H (Netzwerkmonitor)

<dl> <dt>

<span id="_netmon_handoff_set_gly"></span><span id="_NETMON_HANDOFF_SET_GLY"></span>**Übergabe festgelegt**
</dt> <dd>

Ein Protokoll Satz, der einem Protokoll folgt. Ein Parser verwendet den Übergabe-Satz eines Protokolls, wenn der Parser das nächste Protokoll aus den Daten in der Erfassung erkennen kann. Siehe auch [ *folgen Satz*](f.md)

</dd> <dt>

<span id="_netmon_handoff_table_gly"></span><span id="_NETMON_HANDOFF_TABLE_GLY"></span>**Übergabe-Tabelle**
</dt> <dd>

Eine Tabelle, die den festgelegten parserhandoff enthält. Sie können eine Übergabe-Tabelle verwenden, um auf die Übergabe-Set-Informationen zuzugreifen – ohne auf die vom Parser definierte ini-Datei zuzugreifen. Die Informationen zum Übergabe-Satz werden in der INI-Datei der einzelnen Parser gespeichert. Siehe auch *Übergabe festlegen*

</dd> <dt>

<span id="_netmon_hex_pane_gly"></span><span id="_NETMON_HEX_PANE_GLY"></span>**Hex-Bereich**
</dt> <dd>

Der Bereich im Fenster Netzwerkmonitor Frame-Viewer, in dem die hexadezimal Bytes angezeigt werden, die in Frames übertragen werden, die in einer Erfassung enthalten sind. Standardmäßig wird der Bereich Hex am unteren Rand des Fensters angezeigt.

</dd> </dl>
