# <a name="installing-a-monitor-to-network-monitor"></a>Installieren eines Monitors für Netzwerkmonitor

Die Installation einer Monitor-DLL ist ein einfacher Prozess. Kopieren Sie zunächst die DLL und das zugehörige HTML-Konfigurationsformular in das Unterverzeichnis NPP Monitors (z. B. C: \\ Windows \\ System32 \\ Npp \\ Monitors). Wenn der Monitor in das Verzeichnis kopiert wird, wird er erkannt und ist verfügbar, wenn das Monitor control Tool oder der Monitor Control Service (MCSVC) das nächste Mal aktiviert wird.

Wenn die Umgebungsvariable %SystemRoot% z. B. C: \\ Windows ist, lautet das Unterverzeichnis des Monitors C: \\ Windows \\ System32-Npp-Monitore. \\ \\
