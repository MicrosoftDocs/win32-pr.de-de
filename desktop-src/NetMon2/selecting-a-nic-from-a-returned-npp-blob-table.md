# <a name="selecting-a-nic-from-a-returned-npp-blob-table"></a>Auswählen einer NIC aus einer zurückgegebenen NPP-BLOB-Tabelle

Um eine NIC aus einer NPP-BLOB-Tabelle auszuwählen, die von Netzwerkmonitor zurückgegeben wird, rufen Sie die [**GetNPPBlobTable-Funktion**](getnppblobtable.md) auf. Diese Funktion gibt eine NPP-BLOB-Tabelle zurück, die dann von Ihrer Anwendung aufgelistet werden kann, um die gewünschte NIC auszuwählen.

Wenn Sie diese Funktion aufrufen, können Sie entweder eine BLOB-Tabelle aller auf dem lokalen Computer registrierten NICs oder eine kleinere Gruppe gefilterter NICs zurückgeben. Um die zurückgegebenen NICs zu filtern, können Sie ein [*Filterblob*](f.md) angeben, wenn der Aufruf erfolgt.
