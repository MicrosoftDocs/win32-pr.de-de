# <a name="configuring-experts"></a>Konfigurieren von Experten

Verwenden Sie zum Konfigurieren von Experten die benutzerdefinierten Dialogfelder und die Funktion " [**Konfigurieren**](configure.md) ". Beim Schreiben von Network Monitor-Experten haben Sie die Wahl. Sie können einen Satz von Standard Konfigurationselementen erstellen, die verwendet werden können, wenn der Experte zum ersten Mal ausgeführt wird, oder Sie können Netzwerkmonitor Benutzern gestatten, den Experten zur Laufzeit zu konfigurieren. Konfigurationen für die einzelnen Experten werden von Netzwerkmonitor gespeichert. Als Option können Benutzer die Expertenkonfiguration auf die ursprünglichen Standardeinstellungen zurücksetzen. Beachten Sie, dass Sie die Verwendung von **configure** umgehen können, indem Sie einen Experten entwickeln, der eine festgelegte Konfiguration an Netzwerkmonitor übergibt.
