# <a name="configuring-experts"></a>Konfigurieren von Experten

Verwenden Sie zum Konfigurieren von Experten die benutzerdefinierten Dialogfelder und die [**Funktion Konfigurieren.**](configure.md) Wenn Sie Netzwerkmonitorexperten schreiben, haben Sie die Wahl. Sie können eine Reihe von Standardkonfigurationselementen erstellen, die bei der ersten Ausführung des Experten verwendet werden können, oder Sie können Netzwerkmonitor Benutzern erlauben, den Experten zur Laufzeit zu konfigurieren. Konfigurationen für jeden Experten werden von Netzwerkmonitor gespeichert. Als Option können Benutzer die Expertenkonfiguration auf die ursprünglichen Standardeinstellungen zurücksetzen. Beachten Sie, dass Sie die Verwendung von Konfigurieren umgehen können, indem **Sie** einen Experten entwickeln, der eine feste Konfiguration an Netzwerkmonitor übergibt.
