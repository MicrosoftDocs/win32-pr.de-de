# <a name="monitor-functions"></a>Überwachen von Funktionen

In diesem Abschnitt werden die Hilfsfunktionen beschrieben, die Sie zum Entwickeln von benutzerdefinierten Monitoren verwenden können.



| Funktion                                       | BESCHREIBUNG                                                                                                                                                      |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Dllgetmonitorobject](dllgetmonitorobject.md) | Erstellt eine Instanz des Monitors.                                                                                                                              |
| [Htmlvaluetounicode](htmlvaluetounicode.md)   | Konvertiert eine HTML CP \_ UTF8-Versions Zeichenfolge in Unicode.                                                                                                             |
| [Loaddword](loaddword.md)                     | Lädt ein **DWORD** aus einer HTML-Konfigurations Zeichenfolge.                                                                                                             |
| [Loadipadressen](loadipaddresses.md)         | Lädt eine IP-Adressliste aus einer HTML-TEXTAREA-Konfigurations Zeichenfolge.                                                                                             |
| [Loadipxadressen](loadipxaddresses.md)       | Lädt eine IPX-Adressliste aus einer HTML-TEXTAREA-Konfigurations Zeichenfolge.                                                                                            |
| [Loadmacadressen](loadmacaddresses.md)       | Lädt eine Mac-Adressliste aus einer HTML-TEXTAREA-Konfigurations Zeichenfolge.                                                                                             |
| [Loadstringaddr](loadstringaddr.md)           | Wandelt eine Zeichenfolge (z. b. "157.54.32.45") in eine **DWORD** -Adresse um.                                                                                             |
| [Loadtchar](loadtchar.md)                     | Sucht in der Konfigurations Zeichenfolge nach einem angeforderten Variablennamen und ordnet ausreichend Speicherplatz (mithilfe von **heapbelegcmemory**) zu, um ihn zu speichern, zu kopieren und zurückzugeben. |
| [Onloadingdll](onloadingdll.md)               | Gibt an, dass die DLL bereits geladen wurde. Die mcsvc ruft diese Funktion auf, wenn Sie zum ersten Mal die Monitor-DLL lädt.                                                     |
