# <a name="capture-filters"></a>Erfassungsfilter

Das Netzwerkmonitor SDK enthält Erfassungsfilter, mit denen Sie die Anzahl der Frames einschränken können, die eine NPP-Anwendung während des Betriebs empfängt. Indem Sie angeben, welche Frames des Netzwerkdatenverkehrs von einem benutzerdefinierten Monitor oder einer NPP-Anwendung untersucht werden sollen, können Sie die Verarbeitungseffizienz verbessern.

-   [**Die CAPTUREFILTER-Struktur**](the-capturefilter-structure.md)
