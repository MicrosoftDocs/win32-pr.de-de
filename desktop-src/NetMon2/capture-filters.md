# <a name="capture-filters"></a>Erfassungs Filter

Das Netzwerkmonitor SDK enthält Erfassungs Filter, mit denen Sie die Anzahl von Frames einschränken können, die eine NPP-Anwendung (Network Packet Provider) während des Vorgangs empfängt. Durch die Angabe, welche Frames von Netzwerk Datenverkehr von einem benutzerdefinierten Monitor oder einer NPP-Anwendung untersucht werden sollen, können Sie die Verarbeitungseffizienz verbessern.

-   [**Die capturefilter-Struktur**](the-capturefilter-structure.md)
