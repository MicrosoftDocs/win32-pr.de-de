# <a name="testing-a-monitor"></a>Testen eines Monitors

Nachdem ein Monitor installiert wurde, können Sie ihn mit Livedaten aus Ihren Netzwerkvorgängen testen, oder Sie können einen bekannten Satz von Ereignissen testen, indem Sie die Daten mit Netzwerkmonitor erfassen, die Erfassungsdatei speichern und die Daten dann lokal übertragen. (Weitere Informationen zum Übertragen einer Erfassungsdatei in ein Netzwerk finden Sie unter Netzwerkmonitor Onlinehilfe.)
