# <a name="testing-a-monitor"></a>Testen eines Monitors

Nachdem ein Monitor installiert wurde, können Sie ihn mit Livedaten aus Ihren Netzwerk Vorgängen testen, oder Sie können einen bekannten Ereignis Satz testen, indem Sie die Daten mit Netzwerkmonitor erfassen, die Erfassungs Datei speichern und die Daten dann lokal übertragen. (Ausführliche Informationen zum Übertragen einer Erfassungs Datei in einem Netzwerk finden Sie in Netzwerkmonitor Online Hilfe.)
