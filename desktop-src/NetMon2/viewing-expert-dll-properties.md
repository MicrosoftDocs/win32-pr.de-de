# <a name="viewing-expert-dll-properties"></a>Anzeigen von Eigenschaften der expertendll

Verwenden Sie das folgende Verfahren, um die Eigenschaften einer Experten-dll anzuzeigen.

**So zeigen Sie dll-Eigenschaften an**

1.  Öffnen Sie Windows-Explorer, und wechseln Sie zum \\ Unterverzeichnis "Experten" Netzwerkmonitor (z. b. C: \\ Program Files \\ NetMon2 \\ Experts).
2.  Wählen Sie die gewünschte dll aus, und klicken Sie mit der rechten Maustaste darauf.
3.  Klicken Sie auf **Eigenschaften** und **Version**. Im folgenden Myexpert.dll Fenster Eigenschaften Statusfenster werden die DLL-Eigenschaften angezeigt.
4.  Klicken Sie auf den gewünschten **Elementnamen** , um Informationen im Bereich **Wert** anzuzeigen.
5.  Klicken Sie auf **OK**, um das Dialogfeld **Eigenschaften** zu schließen.
