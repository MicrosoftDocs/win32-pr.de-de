# <a name="viewing-expert-dll-properties"></a>Anzeigen von Expert-DLL-Eigenschaften

Verwenden Sie das folgende Verfahren, um die DLL-Eigenschaften eines Experten anzeigen zu können.

**So zeigen Sie DLL-Eigenschaften an**

1.  Öffnen Windows Explorer, und wechseln Sie zum Unterverzeichnis Experten von \\ Netzwerkmonitor (Z. B. C: \\ Programme \\ NetMon2 \\ Experts).
2.  Wählen Sie die gewünschte DLL aus, und klicken Sie mit der rechten Maustaste auf die Datei.
3.  Klicken Sie **auf Eigenschaften** und **Version**. Im folgenden Myexpert.dll Eigenschaftenstatusfenster werden die DLL-Eigenschaften angezeigt.
4.  Klicken Sie auf den gewünschten **Elementnamen,** um Informationen im Bereich **Wert** anzuzeigen.
5.  Klicken Sie auf **OK**, um das Dialogfeld **Eigenschaften** zu schließen.
