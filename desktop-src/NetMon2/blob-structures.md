# <a name="blob-structures"></a>BLOB-Strukturen

Die BLOB-Hilfs-API enthält die folgenden Strukturen.



| Struktur                                                        | Beschreibung                                                            |
|------------------------------------------------------------------|------------------------------------------------------------------------|
| [**Adresse**](address.md)                                       | Enthält eine einzelne unterstützte Adresse.                                  |
| [**ADDRESSPAIR**](addresspair.md)                               | Erstellt einen Erfassungsfilter.                                           |
| [**ADDRESSTABLE**](addresstable.md)                             | Gibt eine Tabelle mit Adresspaaren an.                                    |
| [**ANDEXP**](andexp.md)                                         | Gibt ein Array von Muster übereinstimmungen an.                                 |
| [**\_BLOB-TABELLE**](blob-table.md)                                | Gibt ein Array von BLOBs an.                                           |
| [**\_CAPTUREFILE-HEADERWERTE \_**](capturefile-header-values.md) | Definiert die Struktur der Netzwerkmonitor Headerdatei.              |
| [**CAPTUREFILTER**](capturefilter.md)                           | Gibt Erfassungsfilterdaten an.                                         |
| [**Ausdruck**](expression.md)                                 | Gibt einen Ausdruck von Peers an, die in logischen AND-Anweisungen strukturiert sind. |
| [**Rahmen**](frame.md)                                           | Gibt einen Frame an.                                                     |
| [**\_FRAMEDESKRIPTOR**](frame-descriptor.md)                    | Gibt Details zu unformatierten Frames an.                                       |
| [**FRAMETABLE**](frametable.md)                                 | Gibt einen Satz von Frames an.                                             |
| [**GENERISCHER \_ PORT**](generic-port.md)                            | Gibt eine Portnummer an, die in IP- oder IPX-Protokollen verwendet wird.                   |
| [**\_IPX-ADRESSE**](ipx-address.md)                              | Gibt eine IPX-Adresse an.                                              |
| [**NETWORKINFO**](networkinfo.md)                               | Gibt eine NIC an.                                                       |
| [**PATTERNMATCH**](patternmatch.md)                             | Gibt Musterelemente an, die zum Auswerten von Frames verwendet werden.                    |
| [**Trigger**](trigger.md)                                       | Gibt an, dass ein Ereignis zu einem bestimmten Zeitpunkt auftritt.                      |
| [**\_UPDATEEREIGNIS**](update-event.md)                            | Aktualisiert Ereignisse.                                                        |
| [**\_VINES-IP-ADRESSE \_**](vines-ip-address.md)                   | Gibt eine Adresse in einem Vines-Netzwerk an.                               |
