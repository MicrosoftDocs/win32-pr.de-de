# <a name="blob-structures"></a>BLOB-Strukturen

Die BLOB Helper-API umfasst die folgenden Strukturen.



| Struktur                                                        | BESCHREIBUNG                                                            |
|------------------------------------------------------------------|------------------------------------------------------------------------|
| [**Adresse**](address.md)                                       | Enthält eine einzelne, unterstützte Adresse.                                  |
| [**Adresssspair**](addresspair.md)                               | Erstellt einen Erfassungs Filter.                                           |
| [**Adresssstable**](addresstable.md)                             | Gibt eine Tabelle mit Adress Paaren an.                                    |
| [**Andexp**](andexp.md)                                         | Gibt ein Array von Muster Übereinstimmungen an.                                 |
| [**BLOB- \_ Tabelle**](blob-table.md)                                | Gibt ein Array von blobwörtern an.                                           |
| [**capturefile- \_ Header \_ Werte**](capturefile-header-values.md) | Definiert die Struktur der Netzwerkmonitor Header Datei.              |
| [**Capturefilter**](capturefilter.md)                           | Gibt Erfassungs Filterdaten an.                                         |
| [**Begriff**](expression.md)                                 | Gibt einen Ausdruck von Peers an, der in logischen and-Anweisungen strukturiert ist. |
| [**Dr**](frame.md)                                           | Gibt einen Frame an.                                                     |
| [**Frame \_ Deskriptor**](frame-descriptor.md)                    | Gibt Details von rohframes an.                                       |
| [**Frametable**](frametable.md)                                 | Gibt einen Satz von Frames an.                                             |
| [**generischer \_ Port**](generic-port.md)                            | Gibt eine Portnummer an, die in IP-oder IPX-Protokollen verwendet wird.                   |
| [**IPX- \_ Adresse**](ipx-address.md)                              | Gibt eine IPX-Adresse an.                                              |
| [**Networkinfo**](networkinfo.md)                               | Gibt eine NIC an.                                                       |
| [**Patternmatch**](patternmatch.md)                             | Gibt Musterelemente an, mit denen Frames ausgewertet werden.                    |
| [**TRIGGER**](trigger.md)                                       | Gibt an, dass ein Ereignis zu einem bestimmten Zeitpunkt auftritt.                      |
| [**\_Ereignis aktualisieren**](update-event.md)                            | Aktualisiert Ereignisse.                                                        |
| [**\_IP- \_ Adresse der Reben**](vines-ip-address.md)                   | Gibt eine Adresse in einem Reben Netzwerk an.                               |
