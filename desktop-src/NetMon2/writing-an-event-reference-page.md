# <a name="writing-an-event-reference-page"></a>Schreiben einer Ereignis Referenzseite

Verwenden Sie Ereignis Referenzseiten (Erps), um zu beschreiben, wie Sie ein Problem lösen können, das vom Experten erkannt wird. Da Erps aus HTML-Text bestehen, können Sie einen beliebigen HTML-Editor oder Text-Editor verwenden, um einen zu erstellen.

Der Netzwerkmonitor-Ereignis Browserbereich, in dem das ERP-Fenster angezeigt wird, unterstützt die gesamte Funktionalität von Microsoft Internet Explorer 4,01 und höher. Betten Sie alle unterstützten Steuerelemente oder Skriptsprachen in Ihr ERP ein, um Informationen anzuzeigen oder andere Funktionen von Ihrer ERP zu starten. Beispielsweise können Sie ein Steuerelement erstellen, das e-Mail an Netzwerk Vorgänge sendet.

Weitere Informationen zum Erstellen von ERP finden Sie in den folgenden Themen:

-   [Erstellen von Referenzseiten für Experten und Überwachungs Ereignisse](creating-expert-and-monitor-event-reference-pages.md)
-   [Benennen einer Ereignis Referenzseite](naming-an-event-reference-page.md)
-   [Testen einer Ereignis Referenzseite](testing-an-event-reference-page.md)
