# <a name="writing-an-event-reference-page"></a>Schreiben einer Ereignisreferenzseite

Verwenden Sie Ereignisreferenzseiten (Event Reference Pages, ERPs), um zu beschreiben, wie sie ein vom Experten erkanntes Problem lösen. Da ERPs aus HTML-Text bestehen, können Sie einen beliebigen HTML-Editor oder Text-Editor verwenden.

Der Netzwerkmonitor Event Browser, in dem erp angezeigt wird, unterstützt alle Funktionen von Microsoft Internet Explorer 4.01 und höher. Betten Sie alle unterstützten Steuerelemente oder Skriptsprachen in Ihr ERP ein, um Informationen anzuzeigen oder andere Funktionen aus Ihrem ERP-System zu starten. Beispielsweise können Sie ein Steuerelement erstellen, das E-Mails an Netzwerkvorgänge sendet.

Weitere Informationen zum Erstellen eines ERP-Systems finden Sie in den folgenden Themen:

-   [Erstellen von Referenzseiten für Experten und Monitor-Ereignisse](creating-expert-and-monitor-event-reference-pages.md)
-   [Benennen einer Ereignisreferenzseite](naming-an-event-reference-page.md)
-   [Testen einer Ereignisreferenzseite](testing-an-event-reference-page.md)
