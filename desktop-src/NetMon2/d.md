# <a name="d-network-monitor"></a>D (Netzwerkmonitor)

<dl> <dt>

<span id="_netmon_delayed_capture_gly"></span><span id="_NETMON_DELAYED_CAPTURE_GLY"></span>**verzögerte Erfassung**
</dt> <dd>

Erfasster Netzwerk Datenverkehr, der in einer Erfassungs Datei gespeichert ist. Ein Netzwerk Paketanbieter (NPP) führt eine verzögerte Erfassung aus. Parser und Experten verwenden verzögerte Erfassungen. Siehe auch [*Erfassung*](c.md)und [*echt Zeiterfassung*](/windows)

</dd> <dt>

<span id="_netmon_destination_address_gly"></span><span id="_NETMON_DESTINATION_ADDRESS_GLY"></span>**Zieladresse**
</dt> <dd>

Die Adresse des Computers, an den Daten gesendet werden.

</dd> <dt>

<span id="_netmon_detail_pane_gly"></span><span id="_NETMON_DETAIL_PANE_GLY"></span>**Detailbereich**
</dt> <dd>

Der Bereich im Fenster Netzwerkmonitor Rahmen, in dem ausführliche Informationen zu den Protokoll Eigenschaften angezeigt werden. Standardmäßig wird der Bereich in der Mitte des Fensters angezeigt.

</dd> <dt>

<span id="_netmon_display_filter_gly"></span><span id="_NETMON_DISPLAY_FILTER_GLY"></span>**Anzeige Filter**
</dt> <dd>

Ein Filter, der verwendet wird, um die Menge der aufgezeichneten Daten einzuschränken, die im Fenster des Frame-Viewers angezeigt werden. Ein Anzeige Filter ist ein postfix Stapel von Objekten, bei denen es sich um einen Operator, einen Wert oder eine Eigenschaft handeln kann.

</dd> </dl>
