# <a name="selecting-a-nic-from-a-supplied-npp-blob-table"></a>Auswählen einer NIC aus einer bereitgestellten NPP-BLOB-Tabelle

Um eine Netzwerkschnittstellenkarte (Network Interface Card, NIC) aus einer angegebenen NPP-BLOB-Tabelle auszuwählen, rufen Sie die [**SelectNPPBlobFromTable-Funktion**](selectnppblobfromtable.md) auf. Diese Funktion verwendet die Netzwerkmonitor Benutzeroberfläche, um die in der BLOB-Tabelle dargestellten NICs anzuzeigen, und gibt das NPP-BLOB zurück, das die ausgewählte NIC darstellt.

Wenn Sie die [**Funktion SelectNPPBlobFromTable**](selectnppblobfromtable.md) aufrufen, wird das Dialogfeld **Netzwerk auswählen** angezeigt. Darin können Sie die Details der NPPs anzeigen, die in der NPP-BLOB-Tabelle dargestellt sind, eine bestimmte NIC auswählen und die Details der zugehörigen NPP-BLOB-Struktur anzeigen.

Die folgende Abbildung zeigt die NICs, die in einer angegebenen NPP-BLOB-Tabelle dargestellt werden.
