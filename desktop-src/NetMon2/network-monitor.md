# <a name="network-monitor"></a>Netzwerkmonitor

## <a name="purpose"></a>Zweck

Netzwerkmonitor erfasst Netzwerk Datenverkehr für die Anzeige und Analyse. Es ermöglicht Ihnen das Ausführen von Aufgaben wie z. b. das Analysieren zuvor erfasster Daten in benutzerdefinierten Methoden und das Extrahieren von Daten aus definierten Protokoll Parser.

## <a name="developer-audience"></a>Entwicklergruppe

Auf alle von Netzwerkmonitor bereitgestellten API-Sätze kann mithilfe von C/C++ zugegriffen werden. Entwickler, die mit [*Netzwerk Paket Anbietern*](n.md) arbeiten, müssen auch mit com vertraut sein.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Um die Netzwerkmonitor-API aufzurufen, müssen Sie unter Windows NT Server 4,0, Windows 2000 Server oder Windows Server 2003 oder Microsoft Systems Management Server installiert sein.

Der NPP-Treiber und die unterstützten Funktionen umfassen alle Versionen von Windows NT 4,0 und Windows 2000 Server.

> [!Note]  
> Netzwerkmonitor 2,1 und frühere Versionen werden unter Windows Vista und höher nicht unterstützt.
