# <a name="network-monitor"></a>Netzwerkmonitor

## <a name="purpose"></a>Zweck

Netzwerkmonitor erfasst Netzwerkdatenverkehr zur Anzeige und Analyse. Sie können Aufgaben wie das Analysieren zuvor erfasster Daten in benutzerdefinierten Methoden und das Extrahieren von Daten aus definierten Protokollparsern ausführen.

## <a name="developer-audience"></a>Entwicklergruppe

Auf alle api-Sätze, Netzwerkmonitor bereitgestellt werden, kann mithilfe von C/C++ zugegriffen werden. Diese Entwickler, die auch mit [*Netzwerkpaketanbietern arbeiten,*](n.md) müssen auch mit COM vertraut sein.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Zum Aufrufen über die Netzwerkmonitor-API müssen Sie auf Windows NT Server 4.0, Windows 2000 Server oder Windows Server 2003 ausgeführt werden oder Microsoft Systems Management Server installiert haben.

Der NPP-Treiber und die unterstützten Features umfassen alle Versionen von Windows NT 4.0 und Windows 2000 Server.

> [!Note]  
> Netzwerkmonitor 2.1 und früher werden unter Windows Vista und höher nicht unterstützt.
