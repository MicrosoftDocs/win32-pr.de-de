# <a name="monitor-event-reference-pages"></a>Überwachen von Ereignis Referenzseiten

Mithilfe von Ereignis Referenzseiten (Erps) können Sie den Monitor mit der Netzwerkmonitor-Benutzeroberfläche verbinden. Sie können verwendet werden, um Endbenutzer über erkannte Bedingungen und Probleme zu benachrichtigen, oder Sie können verwendet werden, um vorgeschlagene Korrekturen oder erklärende Daten bereitzustellen.
