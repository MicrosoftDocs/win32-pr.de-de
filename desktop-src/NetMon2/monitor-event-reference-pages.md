# <a name="monitor-event-reference-pages"></a>Überwachen von Ereignisverweisseiten

Ereignisverweisseiten (ERPs) können verwendet werden, um Ihren Monitor mit der Netzwerkmonitor Ui zu verbinden. Sie können verwendet werden, um Endbenutzer über erkannte Bedingungen und Probleme zu informieren, oder sie können verwendet werden, um vorgeschlagene Korrekturen oder erläuternde Daten bereitzustellen.
