# <a name="obtaining-network-monitor-21"></a>Abrufen von Netzwerkmonitor 2,1

Netzwerkmonitor 2,1 ist als separate Installation auf dem Microsoft Windows Systems Management Server (SMS) 2003-oder Windows Server 2003-Medium verfügbar.

> [!Note]  
> Derzeit ist Netzwerkmonitor 2,1 nicht als Teil der Windows SDK verfügbar. und ist nicht in Windows Vista verfügbar.

 

## <a name="microsoft-windows-systems-management-server-2003"></a>Microsoft Windows Systems Management Server 2003

Installieren Sie Netzwerkmonitor 2,1 durch Ausführen von "NetmonSetup.exe" aus dem \\ Ordner "netmon \\ i386" auf der SMS 2003-Produkt-CD.

Wenn Sie ein Upgrade von SMS 2,0 ausführen, müssen Sie zunächst Netzwerkmonitor 2,0 mithilfe der Option Software **in der System** Steuerung deinstallieren.

## <a name="microsoft-windows-server-2003"></a>Microsoft Windows Server 2003

Befolgen Sie die nachfolgenden Anweisungen, um Netzwerkmonitor 2,1 von den Windows Server 2003-Medien zu installieren.

-   Klicken Sie auf **Start**, zeigen Sie auf **Einstellungen** , und wählen Sie das Menü Element **Systemsteuerung** aus.
-   Doppel **Klicken Sie** auf Software.
-   Klicken Sie auf **Windows-Komponenten hinzufügen/entfernen**.
-   Klicken Sie auf **Verwaltungs-und Überwachungs Tools**, und klicken Sie dann auf **Details**.
-   Aktivieren Sie das Kontrollkästchen **Netzwerkmonitor Tools** , und klicken Sie dann auf **OK**.
-   Klicken Sie auf **Weiter**.
