# <a name="t-network-monitor"></a>T (Netzwerkmonitor)

<dl> <dt>

<span id="_netmon_total_statistics_gly"></span><span id="_NETMON_TOTAL_STATISTICS_GLY"></span>**Statistiken gesamt**
</dt> <dd>

Statistiken, die die Netzwerkschnittstellenkarte (Network Interface Card, NIC) meldet und Netzwerkmonitor generiert. Beispielsweise umfassen die Statistiken für die Netzwerkframes die Anzahl der Frames, die die NIC meldet, die Anzahl der Frames, die die NIC verarbeitet, die Anzahl der Frames, die durch den Erfassungsfilter übergeben werden, und die Anzahl der Frames, die in der Puffer- oder Erfassungsdatei gespeichert sind. Eine vollständige Liste dieser Statistiken finden Sie in der **STATISTICS-Struktur.**

</dd> </dl>
