# <a name="network-monitor-architecture"></a>Netzwerkmonitor-Architektur

Die Netzwerkmonitor-Architektur kann in zwei Sätze von Programmier Konzepten unterteilt werden, die sich auf die Entwicklung von [*Experten*](e.md) und [*Parser*](p.md) beziehen und die sich auf die [*Überwachung*](m.md) der Entwicklung beziehen.

Obwohl beide die gleichen Komponenten verwenden, um den Netzwerk Datenverkehr zu erfassen, verwenden Sie jeweils verschiedene Schnittstellen, um die Daten und verschiedene Komponenten zu erfassen und zu analysieren, was aufgezeichnet wurde.
