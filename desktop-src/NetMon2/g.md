# <a name="g-network-monitor"></a>G (Netzwerkmonitor)

<dl> <dt>

<span id="_netmon_generic_formatter_gly"></span><span id="_NETMON_GENERIC_FORMATTER_GLY"></span>**Generisches Formatierungsformat**
</dt> <dd>

Ein Eigenschaftendatenformater, der von Netzwerkmonitor wird. Der generische Formatierungser kann die Daten formatieren, die im Detailbereich der benutzeroberfläche Netzwerkmonitor werden. Wenn Sie Netzwerkmonitor generischen Formatierungssteuerfeld verwenden möchten, müssen Sie die **FormatPropertyInstance-Funktion** im **InstanceData-Member** der **PROPERTYINFO-Struktur** einer Eigenschaft angeben. Siehe auch [generische Formatierungsausgabe](generic-formatter-output.md)

</dd> </dl>
