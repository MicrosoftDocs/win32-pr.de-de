# <a name="texturecubegatherred-methods"></a>TextureCube::GatherRed-Methoden

Gibt die roten Komponenten der vier Texelwerte zurück, die in einem bilinealen Filtervorgang verwendet werden.

### <a name="overload-list"></a>Überladeliste



| Methode                                                           | BESCHREIBUNG                                                                                                                                     |
|:-----------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherRed(S,float,uint)**](tcube-gatherred-s-float-uint-.md)  | Gibt die roten Komponenten der vier Texelwerte zurück, die in einem bilinealen Filtervorgang zusammen mit dem Kachelzuordnungsstatus verwendet werden.<br/> |
