# <a name="texturecubegatherred-methods"></a>Texturecube:: gatherred-Methoden

Gibt die roten Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang verwendet werden.

### <a name="overload-list"></a>Überladeliste



| Methode                                                           | BESCHREIBUNG                                                                                                                                     |
|:-----------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Gatherred (S, float, uint)**](tcube-gatherred-s-float-uint-.md)  | Gibt die roten Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang zusammen mit dem Status der Kachel Zuordnung verwendet werden.<br/> |
