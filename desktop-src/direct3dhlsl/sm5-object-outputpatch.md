# <a name="outputpatch"></a>OutputPatch

Stellt ein Array von Ausgabesteuerungspunkten dar, die für die Patchkonst constant-Funktion des Hüllen-Shaders sowie den Domänen-Shader verfügbar sind.



| Methode                                                       | Beschreibung                         |
|--------------------------------------------------------------|-------------------------------------|
| [**Operator\[\]**](sm5-object-outputpatch-operatorindex.md) | Ruft eine schreibgeschützte Ressourcenvariable ab. |



 

Darüber hinaus unterstützt die InputPatch-Klasse die folgenden Eigenschaften:



| Eigenschaften | type | Beschreibung                   |
|------------|------|-------------------------------|
| Länge     | uint | Die Anzahl der Kontrollpunkte. |



 

## <a name="minimum-shader-model"></a>Minimales Shadermodell

Dieses Objekt wird in den folgenden Shadermodellen unterstützt.



| Shadermodell                                                                | Unterstützt |
|-----------------------------------------------------------------------------|-----------|
| [Shadermodell 5](d3d11-graphics-reference-sm5.md) und höher– Shadermodelle | Ja       |



 

Dieses Objekt wird für die folgenden Shadertypen unterstützt:



| Scheitelpunkt | Rumpf | Domain | Geometrie | Pixel | Compute |
|--------|------|--------|----------|-------|---------|
|        | x    | x      |          |       |         |
