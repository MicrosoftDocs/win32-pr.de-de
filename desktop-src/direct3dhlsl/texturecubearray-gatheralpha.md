# <a name="texturecubearraygatheralpha-methods"></a>TextureCubeArray::GatherAlpha-Methoden

Gibt die Alphakomponenten der vier Texelwerte zurück, die in einem bilinealen Filtervorgang verwendet werden.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                    | BESCHREIBUNG                                                                                                                                      |
|:--------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherAlpha(S,float,uint)**](tcubearray-gatheralpha-s-float-uint-.md) | Gibt die Alphakomponenten der vier Texelwerte zurück, die in einem bilinealen Filtervorgang zusammen mit dem Kachelzuordnungsstatus verwendet werden.<br/> |
