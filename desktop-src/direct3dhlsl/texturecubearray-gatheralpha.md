# <a name="texturecubearraygatheralpha-methods"></a>Texturecubearray:: gatheralpha-Methoden

Gibt die Alpha Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang verwendet werden.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                    | BESCHREIBUNG                                                                                                                                      |
|:--------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Gatheralpha (S, float, uint)**](tcubearray-gatheralpha-s-float-uint-.md) | Gibt die Alpha Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang zusammen mit dem Status der Kachel Zuordnung verwendet werden.<br/> |
