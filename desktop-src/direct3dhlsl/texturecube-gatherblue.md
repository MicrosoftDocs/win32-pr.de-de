# <a name="texturecubegatherblue-methods"></a>TextureCube::GatherBlue-Methoden

Gibt die blauen Komponenten der vier Texelwerte zurück, die in einem bilinearen Filtervorgang verwendet werden.

### <a name="overload-list"></a>Überladeliste



| Methode                                                             | BESCHREIBUNG                                                                                                                                      |
|:-------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherBlue(S,float,uint)**](tcube-gatherblue-s-float-uint-.md)  | Gibt die blauen Komponenten der vier Texelwerte zurück, die in einem bilinearen Filtervorgang zusammen mit dem Kachelzuordnungsstatus verwendet werden.<br/> |
