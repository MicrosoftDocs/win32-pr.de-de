# <a name="texturecubegatherblue-methods"></a>Texturecube:: gatherblue-Methoden

Gibt die blauen Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang verwendet werden.

### <a name="overload-list"></a>Überladeliste



| Methode                                                             | BESCHREIBUNG                                                                                                                                      |
|:-------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Gatherblue (S, float, uint)**](tcube-gatherblue-s-float-uint-.md)  | Gibt die blauen Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang zusammen mit dem Status der Kachel Zuordnung verwendet werden.<br/> |
