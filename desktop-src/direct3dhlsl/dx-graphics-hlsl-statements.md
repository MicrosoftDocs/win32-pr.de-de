# <a name="statements"></a>Anweisungen

HLSL-Anweisungen sind Kombinationen von Variablen und Funktionen, aus denen Ausdrücke bestehen. Wie ein Satz erfordert eine HLSL-Anweisung Operatoren, die bestimmen, wie ein Ausdruck ausgewertet wird. Sie können einzelne Anweisungen in einem Shader oder mehrere Anweisungen in einer Gruppe erstellen, die als Anweisungsblock bezeichnet wird. Zum Erstellen von HLSL-Anweisungen müssen Sie die Syntax zum Schreiben von Ausdrücken und die Operatoren kennen, die Sie zum Kombinieren von Ausdrücken verwenden.

-   [Ausdrücke](dx-graphics-hlsl-expressions.md)
-   [Operatoren](dx-graphics-hlsl-operators.md)
-   [Anweisungsblöcke](dx-graphics-hlsl-statement-blocks.md)
