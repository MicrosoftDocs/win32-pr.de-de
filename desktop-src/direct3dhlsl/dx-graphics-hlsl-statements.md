# <a name="statements"></a>Anweisungen

HLSL-Anweisungen sind Kombinationen aus Variablen und Funktionen, die Ausdrücke bilden. wie ein Satz erfordert eine HLSL-Anweisung Operatoren, die bestimmen, wie ein Ausdruck ausgewertet wird. Sie können einzelne Anweisungen in einem Shader erstellen oder mehrere Anweisungen in einer Gruppe erstellen, die als Anweisungsblock bezeichnet wird. Um HLSL-Anweisungen zu erstellen, müssen Sie die Syntax zum Schreiben von Ausdrücken kennen und die Operatoren, die Sie zum Kombinieren von Ausdrücken verwenden.

-   [Ausdrücke](dx-graphics-hlsl-expressions.md)
-   [Operatoren](dx-graphics-hlsl-operators.md)
-   [Anweisungsblöcke](dx-graphics-hlsl-statement-blocks.md)
