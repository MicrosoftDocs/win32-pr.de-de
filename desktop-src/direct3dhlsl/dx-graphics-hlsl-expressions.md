# <a name="expressions"></a>Ausdrücke

Ein Ausdruck ist eine Sequenz von [Variablen](dx-graphics-hlsl-variable-syntax.md) und Literalen, die von [Operatoren](dx-graphics-hlsl-statement-blocks.md)interpunliert werden. Ein Literalwert ist ein expliziter Datenwert, z. b. 1 für eine Ganzzahl oder 2,1 für eine Gleit Komma Zahl. Literale werden häufig verwendet, um einer Variablen einen Wert zuzuweisen.

Ein Ausdruck, gefolgt von einem Semikolon (;) wird als-Anweisung bezeichnet. -Anweisungen reichen von einfachen Ausdrücken bis hin zu Blöcken von-Anweisungen, die eine Sequenz von Aktionen ausführen. Fluss Steuerungs Anweisungen bestimmen, welche Order-Anweisungen ausgeführt werden.

Ein Anweisungsblock gibt auch einen Teilbereich an. Innerhalb eines Anweisungsblocks deklarierte Variablen werden nur innerhalb des Blocks erkannt. Mit den HLSL-Anweisungen wird die Reihenfolge bestimmt, in der Ausdrücke ausgewertet werden. Jeder Ausdruck kann eines der folgenden sein:

-   Ein Ausdruck
-   Ein [Anweisungsblock](dx-graphics-hlsl-statement-blocks.md)
-   Eine [Return-Anweisung](dx-graphics-hlsl-return.md)
-   Eine [Flow-Control-Anweisung](dx-graphics-hlsl-flow-control.md)
