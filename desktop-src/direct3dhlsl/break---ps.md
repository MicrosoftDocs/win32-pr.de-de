# <a name="break---ps"></a>Break-PS

Brechen Sie die aktuelle Schleife bei den nächstgelegenen [ENDLOOP-PS](endloop---ps.md) oder [ENDREP-PS](endrep---ps.md)ab.

## <a name="syntax"></a>Syntax



| break |
|-------|



 

## <a name="remarks"></a>Bemerkungen

Diese Anweisung wird in den folgenden Versionen unterstützt.



| Pixel-Shader-Versionen | 1\_1 | 1\_2 | 1 \_ 3 | 1\_4 | 2 \_ 0 | 2 \_ x | 2 \_ SW | 3 \_ 0 | 3 \_ SW |
|-----------------------|------|------|------|------|------|------|-------|------|-------|
| break                 |      |      |      |      |      | x    | x     | x    | x     |



 



|                          |            |
|--------------------------|------------|
| Mindestens Betriebssystem | Windows 98 |
