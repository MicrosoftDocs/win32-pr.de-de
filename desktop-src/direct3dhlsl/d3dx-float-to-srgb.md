# <a name="d3dx_float_to_srgb-function"></a>D3DX \_ float \_ zu \_ sRGB-Funktion

Konvertiert einen float-Wert in einen sRGB-Wert.

## <a name="syntax"></a>Syntax

``` syntax
FLOAT D3DX_FLOAT_to_SRGB(
   FLOAT val
);
```

## <a name="parameters"></a>Parameter

<dl> <dt>

*ster* 
</dt> <dd>

</dd> </dl>
