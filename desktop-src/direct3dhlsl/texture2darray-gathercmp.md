# <a name="texture2darraygathercmp-methods"></a>Texture2DArray::GatherCmp-Methoden

Weitere Informationen zur zugrunde [liegenden DXBC-Gather4_c](./gather4-c--sm5---asm-.md) finden Sie in der Dokumentation zur -Gather4_c.
