# <a name="texture2darraygathercmp-methods"></a>Texture2DArray:: gathercmp-Methoden

Für vier textexwerte eines [**Texture2DArray**](sm5-object-texture2darray.md) , die in einem bilinearen Filter Vorgang verwendet werden, wird der Vergleich mit einem Vergleichswert zurückgegeben.

Weitere Informationen zum Beschreiben der zugrunde liegenden dxbc-Anweisung finden Sie in der Dokumentation zu [gather4_c](./gather4-c--sm5---asm-.md) .

### <a name="overload-list"></a>Überladeliste



| Methode                                                                             | BESCHREIBUNG                                                                                                      |
|:-----------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------|
| [**Gathercmp (S, float, float, int)**](sm5-object-texture2darray-gathercmp.md)        | Vergleicht und vergleicht eine Textur und gibt alle vier Komponenten zurück.<br/>                                       |
| [**Gathercmp (S, float, float, int, uint)**](t2d-gathercmp-s-float-float-int-uint-.md) | Vergleicht und vergleicht eine Textur und gibt alle vier Komponenten zusammen mit dem Status des Vorgangs zurück.<br/> |
