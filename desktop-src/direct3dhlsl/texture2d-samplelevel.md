# <a name="texture2dsamplelevel-methods"></a>Texture2D::SampleLevel-Methoden

Stichproben von [**Texture2D auf**](sm5-object-texture2d.md) der angegebenen Mipmap-Ebene.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                             | BESCHREIBUNG                                                                                        |
|:-----------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**SampleLevel(S,float,float,int)**](dx-graphics-hlsl-to-samplelevel.md)          | Stichproben einer Textur auf der angegebenen Mipmapebene.<br/>                                        |
| [**SampleLevel(S,float,float,int,uint)**](samplelevel-s-float-float-int-uint-.md) | Samples a texture on the specified mipmap level and returns status about the operation. (Stichprobenentnahme einer Textur auf der angegebenen Mipmapebene und Rückgabe des Status des Vorgangs.<br/> |
