# <a name="cross"></a>übergreifende

Gibt das Kreuz Produkt von zwei Gleit Komma-3D-Vektoren zurück.



| *ret* Kreuz (*x*, *y*) |
|-----------------------|



 

## <a name="parameters"></a>Parameter



| Element                                                   | BESCHREIBUNG                                             |
|--------------------------------------------------------|---------------------------------------------------------|
| <span id="x"></span><span id="X"></span>*Stuben*<br/> | \[im \] ersten Gleit Komma-3D-Vektor.<br/>  |
| <span id="y"></span><span id="Y"></span>*Teenie*<br/> | \[im \] zweiten Gleit Komma-3D-Vektor.<br/> |



 

## <a name="return-value"></a>Rückgabewert

Das Kreuz Produkt des *x* -Parameters und des *y* -Parameters.
