# <a name="texture2dgathergreen-methods"></a>Texture2D::GatherGreen-Methoden

Probieren Sie eine [**Textur2D aus,**](sm5-object-texture2d.md) und gibt die grüne Komponente zurück.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                     | Beschreibung                                                                                                                                       |
|:-----------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherGreen(S,float,int)**](sm5-object-texture2d-gathergreen.md)                                        | Gibt die grünen Komponenten der vier Texelwerte zurück, die in einem bilinearen Filtervorgang verwendet werden.<br/>                                 |
| [**GatherGreen(S,float,int,uint)**](t2d-gathergreen-s-float-int-uint-.md)                                  | Gibt die grünen Komponenten der vier Texelwerte zurück, die in einem bilinearen Filtervorgang zusammen mit dem Kachelzuordnungsstatus verwendet werden.<br/> |
| [**GatherGreen(S,float,int2,int2,int2,int2)**](t2d-gathergreen-s-float-float-int2-int2-int2-int2-.md)      | Gibt die grünen Komponenten der vier Texelwerte zurück, die in einem bilinearen Filtervorgang verwendet werden.<br/>                                 |
| [**GatherGreen(S,float,int2,int2,int2,int2,uint)**](t2d-gathergreen-s-float-int2-int2-int2-int2-uint-.md)  | Gibt die grünen Komponenten der vier Texelwerte zurück, die in einem bilinearen Filtervorgang zusammen mit dem Kachelzuordnungsstatus verwendet werden.<br/> |
