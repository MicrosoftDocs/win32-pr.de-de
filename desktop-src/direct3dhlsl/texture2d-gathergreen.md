# <a name="texture2dgathergreen-methods"></a>Texture2D:: gathergreen-Methoden

Abtastungen eines [**Texture2D**](sm5-object-texture2d.md) und zurückgegeben der grünen Komponente.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                     | BESCHREIBUNG                                                                                                                                       |
|:-----------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Gathergrün (S, float, int)**](sm5-object-texture2d-gathergreen.md)                                        | Gibt die grünen Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang verwendet werden.<br/>                                 |
| [**Gathergrün (S, float, int, uint)**](t2d-gathergreen-s-float-int-uint-.md)                                  | Gibt die grünen Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang zusammen mit dem Status der Kachel Zuordnung verwendet werden.<br/> |
| [**Gathergrün (S, float, int2, int2, int2, int2)**](t2d-gathergreen-s-float-float-int2-int2-int2-int2-.md)      | Gibt die grünen Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang verwendet werden.<br/>                                 |
| [**Gathergrün (S, float, int2, int2, int2, int2, uint)**](t2d-gathergreen-s-float-int2-int2-int2-int2-uint-.md)  | Gibt die grünen Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang zusammen mit dem Status der Kachel Zuordnung verwendet werden.<br/> |
