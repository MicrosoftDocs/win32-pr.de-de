# <a name="d3dx_truncate_float-function"></a>D3DX \_ Truncate \_ FLOAT-Funktion

## <a name="syntax"></a>Syntax

``` syntax
FLOAT D3DX_Truncate_FLOAT(
   FLOAT _V
);
```

## <a name="parameters"></a>Parameter

<dl> <dt>

*\_V* 
</dt> <dd>

</dd> </dl>

## <a name="return-value"></a>Rückgabewert
