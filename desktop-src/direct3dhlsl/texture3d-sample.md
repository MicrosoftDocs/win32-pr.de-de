# <a name="texture3dsample-methods"></a>Texture3D::Sample-Methoden

Stichproben einer Textur.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                       | Beschreibung                                                                                                                               |
|:-----------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------|
| [**Sample(S,float,int)**](dx-graphics-hlsl-to-sample.md)                    | Stichproben einer Textur.<br/>                                                                                                             |
| [**Sample(S,float,int,float)**](t3d-sample-s-float-int-float-.md)           | Stichproben einer Textur mit einem optionalen Wert zum Anschließen von Sample Level-of-Detail-Werten (LOD).<br/>                                      |
| [**Sample(S,float,int,float,uint)**](t3d-sample-s-float-int-float-uint-.md) | Samples a texture with an optional value to clamp sample level-of-detail (LOD) values to, and returns status of the operation.<br/> |
