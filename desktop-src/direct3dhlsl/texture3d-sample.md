# <a name="texture3dsample-methods"></a>Texture3D:: Sample-Methoden

Stichproben eine Textur.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                       | BESCHREIBUNG                                                                                                                               |
|:-----------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------|
| [**Sample (S, float, int)**](dx-graphics-hlsl-to-sample.md)                    | Stichproben eine Textur.<br/>                                                                                                             |
| [**Sample (S, float, int, float)**](t3d-sample-s-float-int-float-.md)           | Gibt eine Textur mit einem optionalen Wert aus, mit dem die Werte der Sample-Ebene (LOD) an eine Klammer angezeigt werden.<br/>                                      |
| [**Sample (S, float, int, float, uint)**](t3d-sample-s-float-int-float-uint-.md) | Führt eine Stichprobe für eine Textur mit einem optionalen Wert aus, um die Samplingrate-Werte (LOD) zu binden, und gibt den Status des Vorgangs zurück.<br/> |
