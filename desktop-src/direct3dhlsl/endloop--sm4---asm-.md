# <a name="endloop-sm4---asm"></a>ENDLOOP (SM4-ASM)

Beendet eine Schleifen Anweisung.



| ENDLOOP |
|---------|



 

## <a name="remarks"></a>Bemerkungen

Das folgende Beispiel zeigt die Verwendung dieser Anweisung.

``` syntax
                loop
                    // example of termination condition
                    if_nz r0.x
                        break
                    endif
                    ...
                endloop
```

Das tokenformat enthält den Offset der entsprechenden Schleifen Anweisung im Shader.

Diese Anweisung gilt für die folgenden Shader-Phasen:



| Vertexshader | Geometrie-Shader | Pixelshader |
|---------------|-----------------|--------------|
| x             | x               | x            |
