# <a name="texturecubesamplebias-methods"></a>TextureCube::SampleBias-Methoden

Stichproben einer Textur, nachdem der Biaswert auf die Mipmapebene angewendet wurde.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                     | Beschreibung                                                                                                                                                                                  |
|:-------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleBias(S,float,float)**](dx-graphics-hlsl-to-samplebias.md)                        | Stichproben einer Textur, nachdem der Biaswert auf die Mipmapebene angewendet wurde.<br/>                                                                                                             |
| [**SampleBias(S,float,float,float)**](tcube-samplebias-s-float-float-float-.md)           | Stichproben einer Textur, nachdem der Biaswert auf die Mipmapebene angewendet wurde, mit einem optionalen Wert zum Zusammenbinden von LOD-Werten (Sample Sample Level of Detail).<br/>                                     |
| [**SampleBias(S,float,float,float,uint)**](tcube-samplebias-s-float-float-float-uint-.md) | Stichproben einer Textur, nachdem der Biaswert auf die Mipmapebene angewendet wurde, mit einem optionalen Wert zum Zusammenbinden von LOD-Werten (Sample Sample Level of Detail). Gibt den Status des Vorgangs zurück.<br/> |
