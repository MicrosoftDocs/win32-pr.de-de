# <a name="quadreadacrossdiagonal-function"></a>Quad-acrossdiagonal-Funktion

Gibt den angegebenen lokalen Wert zurück, der aus der Diagonal umgekehrten Spur in diesem Quad gelesen wird.

## <a name="syntax"></a>Syntax


``` syntax
<type> QuadReadAcrossDiagonal(
   <type> localValue
);
```



## <a name="parameters"></a>Parameter

<dl> <dt>

*localvalue* 
</dt> <dd>

Der angeforderte Typ.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Der angegebene lokale Wert, der aus der Diagonal umgekehrten Strecke in diesem Quad gelesen wird.

## <a name="remarks"></a>Bemerkungen

Weitere Informationen zu den Quads finden Sie unter [Übersicht über Shader Model 6](hlsl-shader-model-6-0-features-for-direct3d-12.md).

Diese Funktion wird vom Shader-Modell 6,0 nur in Pixel-und Compute-Shadern unterstützt.
