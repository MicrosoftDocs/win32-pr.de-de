# <a name="texture3dsamplebias-methods"></a>Texture3D::SampleBias-Methoden

Stichproben einer Textur, nachdem der Biaswert auf die Mipmapebene angewendet wurde.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                           | Beschreibung                                                                                                                                                                                  |
|:-------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleBias(S,float,float,int)**](dx-graphics-hlsl-to-samplebias.md)                          | Stichproben einer Textur, nachdem der Biaswert auf die Mipmapebene angewendet wurde.<br/>                                                                                                             |
| [**SampleBias(S,float,float,int,float)**](t3d-samplebias-s-float-float-int-float-.md)           | Stichproben einer Textur, nachdem der Biaswert auf die Mipmapebene angewendet wurde, mit einem optionalen Wert zum Zusammenbinden von LOD-Werten (Sample Sample Level of Detail).<br/>                                     |
| [**SampleBias(S,float,float,int,float,uint)**](t3d-samplebias-s-float-float-int-float-uint-.md) | Stichproben einer Textur, nachdem der Biaswert auf die Mipmapebene angewendet wurde, mit einem optionalen Wert zum Zusammenbinden von LOD-Werten (Sample Sample Level of Detail). Gibt den Status des Vorgangs zurück.<br/> |
