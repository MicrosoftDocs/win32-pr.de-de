# <a name="semantics"></a><span data-ttu-id="4b2ef-155">Semantik</span><span class="sxs-lookup"><span data-stu-id="4b2ef-155">Semantics</span></span>

<span data-ttu-id="4b2ef-156">Eine Semantik ist eine Zeichenfolge, die an eine Shadereingabe oder-Ausgabe angehängt ist, die Informationen über die beabsichtigte Verwendung eines Parameters überträgt.</span><span class="sxs-lookup"><span data-stu-id="4b2ef-156">A semantic is a string attached to a shader input or output that conveys information about the intended use of a parameter.</span></span> <span data-ttu-id="4b2ef-157">Die Semantik ist für alle Variablen erforderlich, die zwischen den shaderstufen weitergegeben werden.</span><span class="sxs-lookup"><span data-stu-id="4b2ef-157">Semantics are required on all variables passed between shader stages.</span></span> <span data-ttu-id="4b2ef-158">Die Syntax zum Hinzufügen einer Semantik zu einer shadervariablen wird hier dargestellt ([Variablen Syntax (DirectX HLSL)](dx-graphics-hlsl-variable-syntax.md)).</span><span class="sxs-lookup"><span data-stu-id="4b2ef-158">The syntax for adding a semantic to a shader variable is shown here ([Variable Syntax (DirectX HLSL)](dx-graphics-hlsl-variable-syntax.md)).</span></span>
