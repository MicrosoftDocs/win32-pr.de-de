# <a name="texture1dsamplecmp-methods"></a>Texture1D::SampleCmp-Methoden

|:-----------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmp(S,float,float,int)**](dx-graphics-hlsl-to-samplecmp.md)                          | Samples a texture, using a comparison value to reject samples. (Stichproben werden mithilfe eines Vergleichswerts abgewiesen.)<br/>                                                                                                             |
| [**SampleCmp(S,float,float,int,float)**](t1d-samplecmp-s-float-float-int-float-.md)           | Samples a texture, using a comparison value to reject samples, with an optional value to clamp sample level-of-detail (LOD) values to .<br/>                                     |
| [**SampleCmp(S,float,float,int,float,uint)**](t1d-samplecmp-s-float-float-int-float-uint-.md) | Samples a texture, using a comparison value to reject samples, with an optional value to clamp sample level-of-detail (LOD) values to . Gibt den Status des Vorgangs zurück.<br/> |
