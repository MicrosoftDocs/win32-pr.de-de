# <a name="texture1darraysamplecmplevelzero-methods"></a>Texture1DArray::SampleCmpLevelZero-Methoden

Probieren Sie eine Textur nur auf Mipmapebene 0 aus, und verwenden Sie einen Vergleichswert, um Stichproben abzulehnen.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                    | Beschreibung                                                                                                                            |
|:----------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero(S,float,float,int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)                   | Probieren Sie eine Textur nur auf Mipmapebene 0 aus und vergleicht das Ergebnis mit einem Vergleichswert.<br/>                                     |
| [**SampleCmpLevelZero(S,float,float,int,uint)**](t1darray-samplecmplevelzero-s-float-float-int-uint-.md) | Probieren Sie eine Textur nur auf Mipmapebene 0 aus und vergleicht das Ergebnis mit einem Vergleichswert. Gibt den Status des Vorgangs zurück.<br/> |
