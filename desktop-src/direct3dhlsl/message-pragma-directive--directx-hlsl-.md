# <a name="message-pragma-directive"></a>message pragma-Direktive

Pragma-Direktive, die Compilerzeitmeldungen erzeugt.



| \#pragma message("*token-string*") |
|-----------------------------------|



 

## <a name="parameters"></a>Parameter



| Element                                                                                    | Beschreibung                  |
|-----------------------------------------------------------------------------------------|------------------------------|
| <span id="token-string"></span><span id="TOKEN-STRING"></span>*Tokenzeichenfolge*<br/> | Compilermeldung.<br/> |



 

## <a name="examples"></a>Beispiele

Im folgenden Beispiel wird die Nachrichtenverarbeitung während der Vorverarbeitung veranschaulicht.


```
#pragma message "Debugging flag set."
```
