# <a name="texturecubegather-methods"></a>TextureCube::Gather-Methoden

Gibt die vier Texelwerte zurück, die in einem bilinealen Filtervorgang verwendet werden.

Weitere Informationen zur zugrunde liegenden DXBC-Anweisung finden Sie in der Dokumentation zu [gather4.](./gather4--sm5---asm-.md)

### <a name="overload-list"></a>Überladeliste



| Methode                                                     | BESCHREIBUNG                                                                                                              |
|:-----------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------|
| [**Gather(S,float)**](dx-graphics-hlsl-to-gather.md)      | Stichproben einer Textur und Gibt die vier Stichproben (nur rote Komponente) zurück, die für die bilineare Interpolation verwendet werden.<br/> |
| [**Gather(S,float,uint)**](tcube-gather-s-float-uint-.md) | Samples a texture and returns all four components along with status about the operation. (Stichprobenentnahme einer Textur und Rückgabe aller vier Komponenten zusammen mit dem Status des Vorgangs.<br/>                      |
