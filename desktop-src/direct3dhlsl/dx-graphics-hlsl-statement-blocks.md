# <a name="statement-blocks"></a>Anweisungsblöcke

Ein Anweisungsblock enthält eine oder mehrere-Anweisungen. jede-Anweisung besteht aus [Ausdrücken](dx-graphics-hlsl-expressions.md) und [Operatoren](dx-graphics-hlsl-operators.md).


```
{
   statement 1;
   statement 2;
   ...
   statement n;
}
```



Ein Anweisungsblock gibt auch einen Teilbereich an. Innerhalb eines Anweisungsblocks deklarierte Variablen werden nur innerhalb des Blocks erkannt.
