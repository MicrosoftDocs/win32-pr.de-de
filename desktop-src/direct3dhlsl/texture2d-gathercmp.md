# <a name="texture2dgathercmp-methods"></a>Texture2D::GatherCmp-Methoden

Für vier Texelwerte einer [**Texture2D,**](sm5-object-texture2d.md) die in einem bilinealen Filtervorgang verwendet werden, gibt ihren Vergleich mit einem Vergleichswert zurück.

Weitere Informationen zur zugrunde [liegenden DXBC-Gather4_c](./gather4-c--sm5---asm-.md) finden Sie in der Dokumentation zur -Gather4_c.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                             | Beschreibung                                                                                                      |
|:-----------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------|
| [**GatherCmp(S,float,float,int)**](sm5-object-texture2d-gathercmp.md)             | Samples and compares a texture and returns all four components. (Stichprobenentnahme und Vergleich einer Textur und Rückgabe aller vier Komponenten.<br/>                                       |
| [**GatherCmp(S,float,float,int,uint)**](t2d-gathercmp-s-float-float-int-uint-.md) | Samples and compares a texture and returns all four components along with status about the operation.<br/> |
