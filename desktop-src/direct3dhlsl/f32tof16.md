# <a name="f32tof16-function"></a>f32tof16-Funktion

Konvertiert eine Eingabe in einen float16-Typ.

## <a name="syntax"></a>Syntax

``` syntax
uint f32tof16(
  in float value
);
```

## <a name="parameters"></a>Parameter

<dl> <dt>

*wert* \[ In\]
</dt> <dd>

Typ: **float**

Der Eingabewert.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Typ: **uint**

Der konvertierte Wert, der in der unteren Hälfte des UINT-Werts gespeichert ist.

## <a name="remarks"></a>Hinweise

Die folgenden überladenen Versionen sind ebenfalls verfügbar:

``` syntax
uint2 f32tof16(float2 value);
uint3 f32tof16(float3 value);
uint4 f32tof16(float4 value);
```

### <a name="minimum-shader-model"></a>Shader-Mindestmodell

Diese Funktion wird in den folgenden Shadermodellen unterstützt.



| Shadermodell                                                        | Unterstützt |
|---------------------------------------------------------------------|-----------|
| [Shadermodell 4](dx-graphics-hlsl-sm4.md) und höhere Shadermodelle | Ja       |



 

Obwohl diese Funktion in [Shadermodell 4](dx-graphics-hlsl-sm4.md) und höheren Shadermodellen verfügbar ist, da sie in 4.0 und 4.1 emuliert ist, ist sie für diese Shadermodelle weniger leistungsstärker als für [Shadermodell 5.](d3d11-graphics-reference-sm5.md)

Diese Funktion wird in den folgenden Shadertypen unterstützt:



| Scheitelpunkt | Rumpf | Domain | Geometrie | Pixel | Compute |
|--------|------|--------|----------|-------|---------|
| x      | x    | x      | x        | x     | x       |
