# <a name="calculatelevelofdetailunclamped-directx-hlsl-texture-object"></a>CalculateLevelOfDetailUnclamped (DirectX HLSL-Texturobjekt)

Berechnet die LOD, ohne das Ergebnis zu klammern.



| ret Object.CalculateLevelOfDetailUnclamped( sampler \_ state S, float x ); |
|--------------------------------------------------------------------------|



 

Diese Funktion funktioniert identisch mit [CalculateLevelOfDetail (DirectX HLSL Texture Object),](dx-graphics-hlsl-to-calculate-lod.md)mit der Ausnahme, dass es keine Klammern für die berechnete LOD gibt.
