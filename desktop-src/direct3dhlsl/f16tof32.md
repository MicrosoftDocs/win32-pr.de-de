# <a name="f16tof32-function"></a>f16tof32-Funktion

Konvertiert das float16, das in der unteren Hälfte des uint gespeichert ist, in einen float-Wert.

## <a name="syntax"></a>Syntax

``` syntax
float f16tof32(
  in uint value
);
```

## <a name="parameters"></a>Parameter

<dl> <dt>

*Wert* \[ in\]
</dt> <dd>

Typ: **uint**

Der Eingabewert.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Typ: **float**

Der konvertierte Wert.

## <a name="remarks"></a>Bemerkungen

Außerdem sind die folgenden überladenen Versionen verfügbar:

``` syntax
float2 f16tof32(uint2 value);
float3 f16tof32(uint3 value);
float4 f16tof32(uint4 value);
```

### <a name="minimum-shader-model"></a>Minimaler Shader-Modell

Diese Funktion wird in den folgenden shadermodellen unterstützt.



| Shadermodell                                                        | Unterstützt |
|---------------------------------------------------------------------|-----------|
| [Shader Model 4](dx-graphics-hlsl-sm4.md) und höhere shadermodelle | ja       |



 

Obwohl diese Funktion in [Shader Model 4](dx-graphics-hlsl-sm4.md) und höheren Shader-Modellen verfügbar ist, da Sie in 4,0 und 4,1 emuliert wird, ist Sie in diesen shadermodellen weniger leistungsfähig als in [Shader Model 5](d3d11-graphics-reference-sm5.md).

Diese Funktion wird in den folgenden Typen von Shadern unterstützt:



| Scheitelpunkt | Hülle | Domain | Geometrie | Pixel | Compute |
|--------|------|--------|----------|-------|---------|
| x      | x    | x      | x        | x     | x       |
