# <a name="texture2dsample-methods"></a>Texture2D:: Sample-Methoden

Beispiele für eine [**Texture2D**](sm5-object-texture2d.md).

### <a name="overload-list"></a>Überladeliste



| Methode                                                              | BESCHREIBUNG                                                                                                                               |
|:--------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------|
| [**Sample (S, float, int)**](dx-graphics-hlsl-to-sample.md)           | Stichproben eine Textur.<br/>                                                                                                             |
| [**Sample (S, float, int, float)**](sample-s-float--int-float-.md)     | Gibt eine Textur mit einem optionalen Wert aus, mit dem die Werte der Sample-Ebene (LOD) an eine Klammer angezeigt werden.<br/>                                      |
| [**Sample (S, float, int, float, uint)**](sample-s-float--int-uint-.md) | Führt eine Stichprobe für eine Textur mit einem optionalen Wert aus, um die Samplingrate-Werte (LOD) zu binden, und gibt den Status des Vorgangs zurück.<br/> |
