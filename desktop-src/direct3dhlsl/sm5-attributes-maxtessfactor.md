# <a name="maxtessfactor"></a>maxtessfactor

Gibt den Maximalwert an, den der Hüllen-Shader für jeden Mosaikfaktor zurückgeben würde.


```
maxtessfactor(X)   
```



## <a name="remarks"></a>Hinweise

Dieses Attribut legt eine Obergrenze für die Menge des Mosaiks fest, die angefordert wird, damit ein Treiber die maximale Menge an Ressourcen bestimmen kann, die für das Mosaik erforderlich sind.

Dieses Attribut wird in den folgenden Shadertypen unterstützt:



| Scheitelpunkt | Rumpf | Domain | Geometrie | Pixel | Compute |
|--------|------|--------|----------|-------|---------|
|        | x    |        |          |       |         |
