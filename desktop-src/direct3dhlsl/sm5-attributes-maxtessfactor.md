# <a name="maxtessfactor"></a>maxtess Factor

Gibt den maximalen Wert an, den der Hull-Shader für jeden Mosaik Faktor zurückgeben würde.


```
maxtessfactor(X)   
```



## <a name="remarks"></a>Bemerkungen

Mit diesem Attribut wird eine obere Grenze für die angeforderte Mosaik Menge festgelegt, um einem Treiber zu helfen, die maximale Menge an Ressourcen festzulegen, die für das Mosaik benötigt wird.

Dieses Attribut wird in den folgenden Typen von Shadern unterstützt:



| Scheitelpunkt | Hülle | Domain | Geometrie | Pixel | Compute |
|--------|------|--------|----------|-------|---------|
|        | x    |        |          |       |         |
