# <a name="texture2dsamplecmplevelzero-methods"></a>Texture2D::SampleCmpLevelZero-Methoden

Probieren Sie eine [**Textur2D**](sm5-object-texture2d.md) nur auf Mipmapebene 0 aus, und verwenden Sie einen Vergleichswert, um Stichproben abzulehnen.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                           | BESCHREIBUNG                                                                                                                            |
|:-------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero(S,float,float,int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)          | Probieren Sie eine Textur nur auf Mipmapebene 0 aus und vergleicht das Ergebnis mit einem Vergleichswert.<br/>                                     |
| [**SampleCmpLevelZero(S,float,float,int,uint)**](samplecmplevelzero-s-float-float-int-uint-.md) | Probieren Sie eine Textur nur auf Mipmapebene 0 aus und vergleicht das Ergebnis mit einem Vergleichswert. Gibt den Status des Vorgangs zurück.<br/> |
