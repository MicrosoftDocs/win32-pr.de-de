# <a name="texture2dsamplecmplevelzero-methods"></a>Texture2D:: samplecmplevelzero-Methoden

Stichproben [**Texture2D**](sm5-object-texture2d.md) nur auf MipMap-Ebene 0, wobei ein Vergleichswert zum ablehnen von Beispielen verwendet wird.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                           | BESCHREIBUNG                                                                                                                            |
|:-------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**Samplecmplevelzero (S, float, float, int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)          | Vergleicht nur eine Textur auf MipMap-Ebene 0 und vergleicht das Ergebnis mit einem Vergleichswert.<br/>                                     |
| [**Samplecmplevelzero (S, float, float, int, uint)**](samplecmplevelzero-s-float-float-int-uint-.md) | Vergleicht nur eine Textur auf MipMap-Ebene 0 und vergleicht das Ergebnis mit einem Vergleichswert. Gibt den Status des Vorgangs zurück.<br/> |
