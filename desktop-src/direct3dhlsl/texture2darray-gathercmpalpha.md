# <a name="texture2darraygathercmpalpha-methods"></a>Texture2DArray:: gathercmpalpha-Methoden

|:-----------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------|
| [**Gathercmpalpha (S, float, float, int)**](sm5-object-texture2darray-gathercmpalpha.md)                                        | Vergleicht und vergleicht eine Textur und gibt die Alpha Komponente zurück.<br/>                                       |
| [**Gathercmpalpha (S, float, float, int, uint)**](t2darray-gathercmpalpha-s-float-float-int-uint-.md)                            | Vergleicht und vergleicht eine Textur und gibt die Alpha Komponente zusammen mit dem Status des Vorgangs zurück.<br/> |
| [**Gathercmpalpha (S, float, float, int2, int2, int2, int2)**](sm5-object-texture2darray-gathercmpalpha.md)                        | Vergleicht und vergleicht eine Textur und gibt die Alpha Komponente zurück.<br/>                                       |
