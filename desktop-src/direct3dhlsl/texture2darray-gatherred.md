# <a name="texture2darraygatherred-methods"></a>Texture2DArray:: gatherred-Methoden

Gibt die roten Komponenten der vier texelwerte eines [**Texture2DArray**](sm5-object-texture2darray.md)zurück, die in einem bilinearen Filter Vorgang verwendet werden.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                      | BESCHREIBUNG                                                                                                                                     |
|:------------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Gatherred (S, float, int)**](sm5-object-texture2darray-gatherred.md)                                        | Gibt die roten Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang verwendet werden.<br/>                                 |
| [**Gatherred (S, float, int, uint)**](t2darray-gatherred-s-float-int-uint-.md)                                  | Gibt die roten Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang zusammen mit dem Status der Kachel Zuordnung verwendet werden.<br/> |
| [**Gatherred (S, float, int2, int2, int2, int2)**](t2darray-gatherred-s-float-int2-int2-int2-int2-.md)            | Gibt die roten Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang verwendet werden.<br/>                                 |
| [**Gatherred (S, float, int2, int2, int2, int2, uint)**](t2darray-gatherred-s-float-int2-int2-int2-int2-uint-.md)  | Gibt die roten Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang zusammen mit dem Status der Kachel Zuordnung verwendet werden.<br/> |
