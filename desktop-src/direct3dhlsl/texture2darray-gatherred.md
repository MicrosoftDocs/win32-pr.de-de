# <a name="texture2darraygatherred-methods"></a>Texture2DArray::GatherRed-Methoden

Gibt die roten Komponenten der vier Texelwerte eines [**Texture2DArray**](sm5-object-texture2darray.md)zurück, die in einem bilinealen Filtervorgang verwendet werden.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                      | Beschreibung                                                                                                                                     |
|:------------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherRed(S,float,int)**](sm5-object-texture2darray-gatherred.md)                                        | Gibt die roten Komponenten der vier Texelwerte zurück, die in einem bilinealen Filtervorgang verwendet werden.<br/>                                 |
| [**GatherRed(S,float,int,uint)**](t2darray-gatherred-s-float-int-uint-.md)                                  | Gibt die roten Komponenten der vier Texelwerte zurück, die in einem bilinealen Filtervorgang zusammen mit dem Kachelzuordnungsstatus verwendet werden.<br/> |
| [**GatherRed(S,float,int2,int2,int2,int2)**](t2darray-gatherred-s-float-int2-int2-int2-int2-.md)            | Gibt die roten Komponenten der vier Texelwerte zurück, die in einem bilinealen Filtervorgang verwendet werden.<br/>                                 |
| [**GatherRed(S,float,int2,int2,int2,int2,uint)**](t2darray-gatherred-s-float-int2-int2-int2-int2-uint-.md)  | Gibt die roten Komponenten der vier Texelwerte zurück, die in einem bilinealen Filtervorgang zusammen mit dem Kachelzuordnungsstatus verwendet werden.<br/> |
