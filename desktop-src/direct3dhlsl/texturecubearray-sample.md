# <a name="texturecubearraysample-methods"></a>Texturecubearray:: Sample-Methoden

Stichproben eine Textur.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                      | BESCHREIBUNG                                                                                                                               |
|:----------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------|
| [**Sample (S, float)**](dx-graphics-hlsl-to-sample.md)                       | Stichproben eine Textur.<br/>                                                                                                             |
| [**Sample (S, float, float)**](tcubearray-sample-s-float-float-.md)           | Gibt eine Textur mit einem optionalen Wert aus, mit dem die Werte der Sample-Ebene (LOD) an eine Klammer angezeigt werden.<br/>                                      |
| [**Sample (S, float, float, uint)**](tcubearray-sample-s-float-float-uint-.md) | Führt eine Stichprobe für eine Textur mit einem optionalen Wert aus, um die Samplingrate-Werte (LOD) zu binden, und gibt den Status des Vorgangs zurück.<br/> |
