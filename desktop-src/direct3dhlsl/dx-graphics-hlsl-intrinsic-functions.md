|----------------------|-------------------------------------------------------|-----------------------------------------------------------------|
| Matrix               | bis zu 16 Komponenten, abhängig von der Deklaration      | [Grundlegende HLSL-Typen](dx-graphics-hlsl-data-types.md)             |
| Objekt (object)               | samplerobjekt                                        | *Sampler*, *sampler1D*, *sampler2D*, *sampler3D*, *samplercube* |
| Skalar               | 1 Komponente                                           | [Grundlegende HLSL-Typen](dx-graphics-hlsl-data-types.md)             |
| Vektor               | mindestens 1 Komponente, maximal 4 Komponenten (inklusiv) | [Grundlegende HLSL-Typen](dx-graphics-hlsl-data-types.md)             |
