# <a name="texture1darraysamplebias-methods"></a>Texture1DArray::SampleBias-Methoden

Stichproben einer Textur, nachdem der Biaswert auf die Mipmapebene angewendet wurde.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                           | Beschreibung                                                                                                                                                                                  |
|:-------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleBias(S,float,float,int)**](dx-graphics-hlsl-to-samplebias.md)                          | Stichproben einer Textur, nachdem der Biaswert auf die Mipmapebene angewendet wurde.<br/>                                                                                                             |
| [**SampleBias(S,float,float,int,float)**](t1darray-samplebias-s-float-float-int-float-.md)      | Stichproben einer Textur, nachdem der Biaswert auf die Mipmapebene angewendet wurde, mit einem optionalen Wert zum Zusammenbinden von LOD-Werten (Sample Sample Level of Detail).<br/>                                     |
| [**SampleBias(S,float,float,int,float,uint)**](t1d-samplebias-s-float-float-int-float-uint-.md) | Stichproben einer Textur, nachdem der Biaswert auf die Mipmapebene angewendet wurde, mit einem optionalen Wert zum Zusammenbinden von LOD-Werten (Sample Sample Level of Detail). Gibt den Status des Vorgangs zurück.<br/> |
