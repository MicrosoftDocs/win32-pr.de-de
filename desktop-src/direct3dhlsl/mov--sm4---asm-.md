# <a name="mov-sm4---asm"></a>MOV (SM4-ASM)

Komponenten weises verschieben.



| Anweisung: MOV \[ \_ Sat \] dest \[ . mask \] , \[ - \] src0 \[ \_ ABS \] \[ . Swizzle\] |
|-------------------------------------------------------------------------|



 



| Element                                                            | BESCHREIBUNG                                                                              |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/> | \[in \] der Adresse des Vorgangs Ergebnisses.<br/> *dest*  =  *src0*<br/> |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[in \] den zu Verschieb-Komponenten.<br/>                                                |



 

## <a name="remarks"></a>Bemerkungen

Bei den Modifizierern, mit Ausnahme von Swizzle, wird davon ausgegangen, dass die Daten Gleit Komma sind. Das Fehlen von Modifizierern verschiebt Daten nur, ohne Bits zu ändern.

Diese Anweisung gilt für die folgenden Shader-Phasen:



| Vertexshader | Geometrie-Shader | Pixelshader |
|---------------|-----------------|--------------|
| x             | x               | x            |
