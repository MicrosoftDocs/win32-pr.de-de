# <a name="texture2darraygathergreen-methods"></a>Texture2DArray:: gathergreen-Methoden

Gibt die grünen Komponenten der vier texelwerte eines [**Texture2DArray**](sm5-object-texture2darray.md)zurück, die in einem bilinearen Filter Vorgang verwendet werden.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                          | BESCHREIBUNG                                                                                                                                       |
|:----------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Gathergrün (S, float, int)**](sm5-object-texture2darray-gathergreen.md)                                        | Gibt die grünen Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang verwendet werden.<br/>                                 |
| [**Gathergrün (S, float, int, uint)**](t2darray-gathergreen-s-float-int-uint-.md)                                  | Gibt die grünen Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang zusammen mit dem Status der Kachel Zuordnung verwendet werden.<br/> |
| [**Gathergrün (S, float, int2, int2, int2, int2)**](t2darray-gathergreen-s-float-int2-int2-int2-int2-.md)            | Gibt die grünen Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang verwendet werden.<br/>                                 |
| [**Gathergrün (S, float, int2, int2, int2, int2, uint)**](t2darray-gathergreen-s-float-int2-int2-int2-int2-uint-.md)  | Gibt die grünen Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang zusammen mit dem Status der Kachel Zuordnung verwendet werden.<br/> |
