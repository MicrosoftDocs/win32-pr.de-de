# <a name="hlsl-specified-root-signature"></a>HLSL angegebene Stamm Signatur

In HLSL kann eine Stamm [Signatur](/windows/desktop/direct3d12/root-signatures) (eine Schlüssel Tabelle mit Ressourcen und andere Elemente für D3D12) als Zeichenfolge angegeben werden.

Ausführliche Informationen und Beispielcode finden Sie in der D3D12-Dokumentation zum Angeben von Stamm [Signaturen in HLSL](/windows/desktop/direct3d12/specifying-root-signatures-in-hlsl) .
