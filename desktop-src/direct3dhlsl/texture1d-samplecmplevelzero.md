# <a name="texture1dsamplecmplevelzero-methods"></a>Texture1D:: samplecmplevelzero-Methoden

Führt eine Stichprobe eines [**Texture1D**](sm5-object-texture1d.md) nur auf MipMap-Ebene 0 aus und vergleicht das Ergebnis mit einem Vergleichswert.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                               | BESCHREIBUNG                                                                                                                            |
|:-----------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**Samplecmplevelzero (S, float, float, int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)              | Vergleicht nur eine Textur auf MipMap-Ebene 0 und vergleicht das Ergebnis mit einem Vergleichswert.<br/>                                     |
| [**Samplecmplevelzero (S, float, float, int, uint)**](t1d-samplecmplevelzero-s-float-float-int-uint-.md) | Vergleicht nur eine Textur auf MipMap-Ebene 0 und vergleicht das Ergebnis mit einem Vergleichswert. Gibt den Status des Vorgangs zurück.<br/> |
