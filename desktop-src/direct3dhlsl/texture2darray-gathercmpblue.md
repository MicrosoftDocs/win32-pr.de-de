# <a name="texture2darraygathercmpblue-methods"></a>Texture2DArray::GatherCmpBlue-Methoden

| Methode                                                                                                                          | BESCHREIBUNG                                                                                                     |
|:--------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------|
| [**GatherCmpBlue(S,float,float,int)**](sm5-object-texture2darray-gathercmpblue.md)                                             | Sampling und Vergleich einer Textur und Rückgabe der blauen Komponente.<br/>                                       |
| [**GatherCmpBlue(S,float,float,int,uint)**](t2darray-gathercmpblue-s-float-float-int-uint-.md)                                 | Lädt eine Textur ein und vergleicht sie und gibt die blaue Komponente zusammen mit dem Status des Vorgangs zurück.<br/> |
| [**GatherCmpBlue(S,float,float,int2,int2,int2,int2)**](sm5-object-texture2darray-gathercmpblue.md)                             | Sampling und Vergleich einer Textur und Rückgabe der blauen Komponente.<br/>                                       |
| [**GatherCmpBlue(S,float,float,int2,int2,int2,int2,uint)**](t2darray-gathercmpblue-s-float-float-int2-int2-int2-int2-uint-.md) | Lädt eine Textur ein und vergleicht sie und gibt die blaue Komponente zusammen mit dem Status des Vorgangs zurück.<br/> |
