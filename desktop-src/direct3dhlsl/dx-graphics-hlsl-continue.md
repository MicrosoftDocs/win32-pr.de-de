# <a name="continue-statement"></a>continue-Anweisung

Beenden Sie die Ausführung der aktuellen Schleife ([für](dx-graphics-hlsl-do.md) [,](dx-graphics-hlsl-for.md) [während](dx-graphics-hlsl-while.md)), aktualisieren Sie die Schleifenbedingungen, und beginnen Sie mit der Ausführung von oben in der Schleife.

continue;
