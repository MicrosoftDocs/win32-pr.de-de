# <a name="shader-model-5-attributes"></a>Shader Model 5-Attribute

Shader Model 5 implementiert die Attribute aus Shader Model 4 und darunter (siehe DirectX SDK) sowie die folgenden neuen Attribute:

-   [Domäne](sm5-attributes-domain.md)
-   [earlydepthstencil](sm5-attributes-earlydepthstencil.md)
-   [Instanz](sm5-attributes-instance.md)
-   [maxtessfactor](sm5-attributes-maxtessfactor.md)
-   [numthreads](sm5-attributes-numthreads.md)
-   [outputcontrolpoints](sm5-attributes-outputcontrolpoints.md)
-   [Ausgabetopologie](sm5-attributes-outputtopology.md)
-   [Partitionierung](sm5-attributes-partitioning.md)
-   [patchconstantfunc](sm5-attributes-patchconstantfunc.md)

Alle diese Attribute sind erforderlich (mit Ausnahme des **Instanzattributs).** Wenn sie nicht verwendet werden, werden Kompilierungsfehler verursacht.
