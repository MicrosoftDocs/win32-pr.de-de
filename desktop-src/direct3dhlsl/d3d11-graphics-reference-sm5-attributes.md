# <a name="shader-model-5-attributes"></a>Shader Model 5-Attribute

Shadermodell 5 implementiert die Attribute aus dem Shader-Modell 4 und darunter (siehe DirectX SDK) sowie die folgenden neuen Attribute:

-   [-](sm5-attributes-domain.md)
-   [earlydepthstencil](sm5-attributes-earlydepthstencil.md)
-   [lichen](sm5-attributes-instance.md)
-   [maxtess Factor](sm5-attributes-maxtessfactor.md)
-   [numThreads](sm5-attributes-numthreads.md)
-   [outputcontrolpoints](sm5-attributes-outputcontrolpoints.md)
-   [outputtopology](sm5-attributes-outputtopology.md)
-   [Aufteilung](sm5-attributes-partitioning.md)
-   [patchconstantfunc](sm5-attributes-patchconstantfunc.md)

Alle diese Attribute sind erforderlich (mit Ausnahme des **instanzattributs** ). Wenn Sie diese nicht verwenden, werden Kompilierungsfehler verursacht.
