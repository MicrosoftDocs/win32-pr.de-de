# <a name="texture2dgatherred-methods"></a>Texture2D::GatherRed-Methoden

Gibt die roten Komponenten der vier Texelwerte einer [**Texture2D**](sm5-object-texture2d.md)zurück, die in einem bilinearen Filtervorgang verwendet werden würden.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                  | BESCHREIBUNG                                                                                                                                     |
|:--------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherRed(S,float,int)**](sm5-object-texture2d-gatherred.md)                                         | Gibt die roten Komponenten der vier Texelwerte zurück, die in einem bilinearen Filtervorgang verwendet werden.<br/>                                 |
| [**GatherRed(S,float,int,uint)**](t2d-gatherred-s-float-int-uint-.md)                                   | Gibt die roten Komponenten der vier Texelwerte zurück, die in einem bilinearen Filtervorgang zusammen mit dem Kachelzuordnungsstatus verwendet werden.<br/> |
| [**GatherRed(S,float,int2,int2,int2,int2)**](t2d-gatherred-s-float-int2-int2-int2-int2-.md)             | Gibt die roten Komponenten der vier Texelwerte zurück, die in einem bilinearen Filtervorgang verwendet werden.<br/>                                 |
| [**GatherRed(S,float,int2,int2,int2,int2,uint)**](t2d-gathergreen-s-float-int2-int2-int2-int2-uint-.md) | Gibt die roten Komponenten der vier Texelwerte zurück, die in einem bilinearen Filtervorgang zusammen mit dem Kachelzuordnungsstatus verwendet werden.<br/> |
