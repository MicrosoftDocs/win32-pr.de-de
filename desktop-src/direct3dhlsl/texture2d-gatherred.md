# <a name="texture2dgatherred-methods"></a>Texture2D:: gatherred-Methoden

Gibt die roten Komponenten der vier texelwerte eines [**Texture2D**](sm5-object-texture2d.md)zurück, die in einem bilinearen Filter Vorgang verwendet werden.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                  | BESCHREIBUNG                                                                                                                                     |
|:--------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Gatherred (S, float, int)**](sm5-object-texture2d-gatherred.md)                                         | Gibt die roten Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang verwendet werden.<br/>                                 |
| [**Gatherred (S, float, int, uint)**](t2d-gatherred-s-float-int-uint-.md)                                   | Gibt die roten Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang zusammen mit dem Status der Kachel Zuordnung verwendet werden.<br/> |
| [**Gatherred (S, float, int2, int2, int2, int2)**](t2d-gatherred-s-float-int2-int2-int2-int2-.md)             | Gibt die roten Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang verwendet werden.<br/>                                 |
| [**Gatherred (S, float, int2, int2, int2, int2, uint)**](t2d-gathergreen-s-float-int2-int2-int2-int2-uint-.md) | Gibt die roten Komponenten der vier texelwerte zurück, die in einem bilinearen Filter Vorgang zusammen mit dem Status der Kachel Zuordnung verwendet werden.<br/> |
