# <a name="texture2darraysamplecmplevelzero-methods"></a>Texture2DArray::SampleCmpLevelZero-Methoden

Stichproben eines [**Texture2DArrays**](sm5-object-texture2darray.md) nur auf Mipmap-Ebene 0, indem ein Vergleichswert verwendet wird, um Stichproben abzulehnen.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                                    | Beschreibung                                                                                                                            |
|:----------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero(S,float,float,int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)                   | Samples a texture on mipmap level 0 only and compares the result to a comparison value. (Stichprobenentnahme einer Textur nur auf Mipmap-Ebene 0 und Vergleich des Ergebnisses mit einem Vergleichswert.<br/>                                     |
| [**SampleCmpLevelZero(S,float,float,int,uint)**](t2darray-samplecmplevelzero-s-float-float-int-uint-.md) | Samples a texture on mipmap level 0 only and compares the result to a comparison value. (Stichprobenentnahme einer Textur nur auf Mipmap-Ebene 0 und Vergleich des Ergebnisses mit einem Vergleichswert. Gibt den Status des Vorgangs zurück.<br/> |
