# <a name="debugging-shaders-in-visual-studio"></a>Debuggen von Shadern in Visual Studio

Das aktuellste Tool zum Debuggen von Shadern ist nun als Feature in Microsoft Visual Studio enthalten, das als Visual Studio-Grafik Debugger bezeichnet wird. Dieses neue Tool ist ein Ersatz für das [pix](/windows/desktop/directx-sdk--august-2009-) für Windows-Tool. Der Visual Studio-Grafik Debugger hat die Benutzerfreundlichkeit, die Unterstützung für Windows 8 und Direct3D 11,1 und die Integration mit herkömmlichen Visual Studio-Features wie Aufruf Listen und debuggingfenstern für das [HLSL](dx-graphics-hlsl.md) -Debugging erheblich verbessert. Weitere Informationen zu dieser neuen Funktion finden Sie unter [Debuggen von DirectX-Grafiken](/visualstudio/debugger/visual-studio-graphics-diagnostics?view=vs-2015).
