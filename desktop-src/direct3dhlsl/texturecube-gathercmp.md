# <a name="texturecubegathercmp-methods"></a>TextureCube::GatherCmp-Methoden

Für vier Texelwerte eines [**TextureCube,**](texturecube.md) die in einem bilinealen Filtervorgang verwendet werden, gibt ihren Vergleich mit einem Vergleichswert zurück.

Weitere Informationen zur zugrunde [liegenden DXBC-Gather4_c](./gather4-c--sm5---asm-.md) finden Sie in der Dokumentation zur -Gather4_c.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                       | Beschreibung                                                                                                      |
|:-----------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------|
| [**GatherCmp(S,float,float,uint)**](tcube-gathercmp-s-float-float-uint-.md) | Samples and compares a texture and returns all four components along with status about the operation.<br/> |
