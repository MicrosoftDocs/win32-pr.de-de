# <a name="partitioning"></a>Partitionierung

Definiert das im Hull-Shader zu verwendende Tesselations Schema.


```

```

Kann **ganzzahliger** Wert, **\_ nur Bruch** Zahlen, **\_ Bruch** Zahlen oder **pow2** sein.
