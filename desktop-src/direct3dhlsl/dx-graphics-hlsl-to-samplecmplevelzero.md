# <a name="samplecmplevelzero-directx-hlsl-texture-object"></a>Samplecmplevelzero (DirectX HLSL Texture-Objekt)

Vergleicht eine Textur und vergleicht das Ergebnis mit einem Vergleichswert. Diese Funktion ist identisch mit dem Aufruf von [**samplecmp**](dx-graphics-hlsl-to-samplecmp.md) auf MipMap-Ebene 0.
