# <a name="variables"></a>Variables

HLSL-Variablen ähneln Variablen, die in der Programmiersprache C definiert sind. Ähnlich wie bei C haben Variablen einige Benennungs Einschränkungen, die Bereichs Eigenschaften, die vom Speicherort der Deklaration abhängig sind und an die Benutzer Metadaten angefügt werden können. Wie bei C gibt es mehrere Standard Datentypen. Im Gegensatz zu C gibt es auch zusätzliche Datentypen, die von HLSL definiert werden, um die Leistung von 4-Komponenten-Vektoren zu maximieren, die für die Verarbeitung von 3D-Grafikdaten eine Matrix mathematische verwenden.
