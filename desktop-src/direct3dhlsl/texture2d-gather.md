# <a name="texture2dgather-methods"></a>Texture2D::Gather-Methoden

Probieren Sie eine [**Texture2D**](sm5-object-texture2d.md) aus, und gibt alle vier Komponenten zurück.

Weitere Informationen zur zugrunde liegenden DXBC-Anweisung finden Sie in der Dokumentation zu [gather4.](./gather4--sm5---asm-.md)

### <a name="overload-list"></a>Überladeliste



| Methode                                                           | BESCHREIBUNG                                                                                         |
|:-----------------------------------------------------------------|:----------------------------------------------------------------------------------------------------|
| [**Gather(S,float,int)**](sm5-object-texture2d-gather.md)       | Probieren Sie eine Textur aus, und gibt alle vier Komponenten zurück.<br/>                                       |
| [**Gather(S,float,int,uint)**](t2d-gather-s-float-int-uint-.md) | Probieren Sie eine Textur aus, und gibt alle vier Komponenten zusammen mit dem Status des Vorgangs zurück.<br/> |
