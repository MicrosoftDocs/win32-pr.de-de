# <a name="windows-imaging-component"></a>Windows-Bilderstellungskomponente

## <a name="purpose"></a>Zweck

Bei der Windows Imaging Component (WIC) handelt es sich um eine erweiterbare Plattform, die eine Low-Level-API für digitale Images bereitstellt.  WIC unterstützt die Standardweb Bildformate, hohe dynamische Bereichs Bilder und unformatierte Kameradaten.  WIC unterstützt auch zusätzliche Funktionen wie z. b.:

-   Integrierte Unterstützung für standardmetadatenformate
-   Erweiterbares Framework für Bild Codecs, Pixel Formate und Metadatenformate.
-   Unterstützung für eine breite Palette an Pixel Formaten.
-   Unterstützung für hohe Farben; Dazu zählen der erweiterte 30-Bit-Bereich, die 30-Bit-Genauigkeit mit hoher Genauigkeit und 48-Bit-Pixel Formate mit hoher Genauigkeit und Breite.
-   Die progressive Bild Decodierung.

## <a name="developer-audience"></a>Entwicklerzielgruppe

WIC ist so konzipiert, dass die Anforderungen der folgenden Klassen von Entwicklern erfüllt werden:

-   Entwickler, die Bilddaten in einer Anwendung lesen und schreiben.
-   Entwickler, die Bild Metadaten in einer Anwendung lesen und schreiben.
-   Entwickler, die die Laufzeit-Codec-Ermittlung für neu entworfene oder implementierte Bild Codecs benötigen.
-   Entwickler, die neue Bild Codecs und Metadatenformate entwerfen oder implementieren.
