# <a name="windows-imaging-component"></a>Windows-Bilderstellungskomponente

## <a name="purpose"></a>Zweck

Die Windows Imaging Component (WIC) ist eine erweiterbare Plattform, die eine low-level-API für digitale Bilder bereitstellt.  WIC unterstützt die Standardformate für Webbilder, Bilder mit hohem dynamischen Bereich und Unformatierte Kameradaten.  WIC unterstützt auch zusätzliche Features, z. B.:

-   Integrierte Unterstützung für Standardmetadatenformate
-   Erweiterbares Framework für Bildcodecs, Pixelformate und Metadatenformate.
-   Breite Unterstützung des Pixelformats.
-   Unterstützung hoher Farben; einschließlich erweiterter 30-Bit-Bereich, 30-Bit-Hochgenauigkeit und 48-Bit-Formaten mit hoher Genauigkeit und breitem Gamutpixel.
-   Progressive Bilddecodierung.

## <a name="developer-audience"></a>Entwicklerzielgruppe

WIC wurde entwickelt, um die Anforderungen der folgenden Klassen von Entwicklern zu erfüllen:

-   Entwickler, die Bilddaten in einer Anwendung lesen und schreiben.
-   Entwickler, die Bildmetadaten in einer Anwendung lesen und schreiben.
-   Entwickler, die die Laufzeitcodecermittlung für neu entworfene oder implementierte Bildcodecs wünschen.
-   Entwickler, die neue Bildcodecs und Metadatenformate entwerfen oder implementieren.
