# <a name="introduction-how-to-write-a-wic-enabled-codec"></a>Einführung (Schreiben eines WIC-Enabled Codecs)

Die Windows Imaging Component (WIC) ist eine erweiterbare Plattform für die digitale Bildverarbeitung auf den Betriebssystemen Windows Vista und Windows 7. WIC ist auch unter Windows XP und Windows Server 2003 verfügbar, entweder als Teil von Microsoft .NET Framework 3.0 oder als separate Komponente, die Sie verteilen können. Jede Anwendung, die WIC verwendet, kann auf Bilder in jedem Bildformat zugreifen, diese anzeigen, verarbeiten und drucken, das einen WIC-fähigen Codec bietet. Dabei wird ein einzelner Satz konsistenter Schnittstellen verwendet, die spezielle Kenntnisse zu bestimmten Formaten vermeiden.

Die Windows Vista Explorer, Windows Vista Fotogalerie, Live Fotogalerie und Windows 7 Bildanzeige auf WIC. Nachdem ein WIC-fähiger Codec auf einem Windows Vista- oder Windows 7-System installiert wurde, bietet das Betriebssystem die gleiche Unterstützung für das zugeordnete Imageformat wie für die Standardimageformate, die mit der Plattform verbunden sind. Da Sie den Codec für Ihr eigenes Bildformat schreiben, können Sie überall dort, wo Ihr Bildformat verwendet wird, eine konsistente Qualität sicherstellen und von der vollständigen Plattformunterstützung profitieren.
