# <a name="wic-structures"></a>WIC-Strukturen

Dieser Abschnitt enthält Informationen zu den Strukturen der Windows Imaging Component (WIC).

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                          | BESCHREIBUNG                                                                                                                   |
|--------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**Wicbitmappattern**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmappattern)<br/>             | Enthält Elemente, die ein Muster in einer Bilddatei identifizieren, das zum Identifizieren eines bestimmten Formats verwendet werden kann.<br/>   |
| [**Wicbitmapplane**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmapplane)<br/>                            | Gibt das Pixel Format, den Puffer, den Schritt und die Größe einer Komponentenebene für ein planare-Pixel Format an.<br/>                |
| [**Wicbitmapplanedescription**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmapplanedescription)<br/>      | Gibt das Pixel Format und die Größe einer Komponentenebene an.<br/>                                                          |
| [**Wicddsformatinfo**](/windows/desktop/api/Wincodec/ns-wincodec-wicddsformatinfo)<br/>                        | Gibt das [**DXGI- \_ Format**](/windows/desktop/api/dxgiformat/ne-dxgiformat-dxgi_format) und die Block Informationen eines DDS-Formats an.<br/>                  |
| [**Wicddsparameters**](/windows/desktop/api/Wincodec/ns-wincodec-wicddsparameters)<br/>                        | Gibt die DDS-Image Dimension, das [**DXGI- \_ Format**](/windows/desktop/api/dxgiformat/ne-dxgiformat-dxgi_format) und den Alpha-Modus enthaltener Daten an.<br/>  |
| [**Wicimageparameters**](/windows/desktop/api/Wincodec/ns-wincodec-wicimageparameters)<br/>                    | Definiert Parameter, die Sie zum Überschreiben der Standardparameter verwenden können, die normalerweise beim Codieren eines Bilds verwendet werden. <br/> |
| [**Wicjpgframeheader**](/windows/desktop/api/wincodec/ns-wincodec-wicjpegframeheader)<br/>                    | Stellt einen JPEG-Frame Header dar.<br/>                                                                                    |
| [**Wicjpgscanheader**](/windows/desktop/api/wincodec/ns-wincodec-wicjpegscanheader)<br/>                      | Stellt einen JPEG-Frame Header dar.<br/>                                                                                    |
| [**WICMetadataHeader**](/windows/desktop/api/Wincodecsdk/ns-wincodecsdk-wicmetadataheader)<br/>           | Stellt den Metadatenheader dar.<br/>                                                                                        |
| [**WICMetadataPattern**](/windows/desktop/api/Wincodecsdk/ns-wincodecsdk-wicmetadatapattern)<br/>         | Stellt ein metadatenmuster dar.<br/>                                                                                     |
| [**Wicrawcapabilitiesinfo**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawcapabilitiesinfo)<br/> | Definiert unformatierte Codec-Unterschriften.<br/>                                                                                     |
| [**WICRawToneCurve**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawtonecurve)<br/>               | Stellt eine rohbildton Kurve dar.<br/>                                                                                 |
| [**WICRawToneCurvePoint**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawtonecurvepoint)<br/>     | Stellt einen rohbildton-Kurven Punkt dar.<br/>                                                                           |
| [**WICRect**](/windows/desktop/api/Wincodec/ns-wincodec-wicrect)<br/>                               | Stellt ein Rechteck für die WIC-API dar.<br/>                                                                                |
