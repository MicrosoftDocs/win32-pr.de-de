# <a name="wic-structures"></a>WIC-Strukturen

Dieser Abschnitt enthält Informationen zu Windows WIC-Strukturen (Imaging Component).

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                          | Beschreibung                                                                                                                   |
|--------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**WICBitmapPattern**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmappattern)<br/>             | Enthält Member, die ein Muster in einer Bilddatei identifizieren, das zum Identifizieren eines bestimmten Formats verwendet werden kann.<br/>   |
| [**WICBitmapPlane**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmapplane)<br/>                            | Gibt das Pixelformat, den Puffer, den Schritt und die Größe einer Komponentenebene für ein planares Pixelformat an.<br/>                |
| [**WICBitmapPlaneDescription**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmapplanedescription)<br/>      | Gibt das Pixelformat und die Größe einer Komponentenebene an.<br/>                                                          |
| [**WICDdsFormatInfo**](/windows/desktop/api/Wincodec/ns-wincodec-wicddsformatinfo)<br/>                        | Gibt die [**DXGI \_ FORMAT-**](/windows/desktop/api/dxgiformat/ne-dxgiformat-dxgi_format) und Blockinformationen eines DDS-Formats an.<br/>                  |
| [**WICDdsParameters**](/windows/desktop/api/Wincodec/ns-wincodec-wicddsparameters)<br/>                        | Gibt die DDS-Bilddimension, [**DXGI \_ FORMAT**](/windows/desktop/api/dxgiformat/ne-dxgiformat-dxgi_format) und den Alphamodus der enthaltenen Daten an.<br/>  |
| [**WICImageParameters**](/windows/desktop/api/Wincodec/ns-wincodec-wicimageparameters)<br/>                    | Dadurch werden Parameter definiert, mit denen Sie die Standardparameter überschreiben können, die normalerweise beim Codieren eines Bilds verwendet werden. <br/> |
| [**WICJpegFrameHeader**](/windows/desktop/api/wincodec/ns-wincodec-wicjpegframeheader)<br/>                    | Stellt einen JPEG-Frameheader dar.<br/>                                                                                    |
| [**WICJpegScanHeader**](/windows/desktop/api/wincodec/ns-wincodec-wicjpegscanheader)<br/>                      | Stellt einen JPEG-Frameheader dar.<br/>                                                                                    |
| [**WICMetadataHeader**](/windows/desktop/api/Wincodecsdk/ns-wincodecsdk-wicmetadataheader)<br/>           | Stellt den Metadatenheader dar.<br/>                                                                                        |
| [**WICMetadataPattern**](/windows/desktop/api/Wincodecsdk/ns-wincodecsdk-wicmetadatapattern)<br/>         | Stellt ein Metadatenmuster dar.<br/>                                                                                     |
| [**WICRawCapabilitiesInfo**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawcapabilitiesinfo)<br/> | Definiert unformatiert Codecfunktionen.<br/>                                                                                     |
| [**WICRawToneCurve**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawtonecurve)<br/>               | Stellt eine unformatte Bildtonkurve dar.<br/>                                                                                 |
| [**WICRawToneCurvePoint**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawtonecurvepoint)<br/>     | Stellt einen unformatten Bildtonkurvenpunkt dar.<br/>                                                                           |
| [**WICRect**](/windows/desktop/api/Wincodec/ns-wincodec-wicrect)<br/>                               | Stellt ein Rechteck für die WIC-API dar.<br/>                                                                                |
