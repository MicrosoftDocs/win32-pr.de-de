# <a name="systemimagehorizontalresolution-proxy"></a>System. Image. HorizontalResolution-Proxy

Der Eigenschafts Proxy für die "System. Image. HorizontalResolution"-fotometadatenrichtlinie.

### <a name="description"></a>BESCHREIBUNG

Der Eigenschafts Proxy gibt ein rationelles paar a durch das [PROPVARIANT](/windows/win32/api/propidlbase/ns-propidlbase-propvariant) -Element "uhval" zurück.

### <a name="containers"></a>Container

JPEG, TIFF

### <a name="read-only"></a>Schreibgeschützt

Ja

### <a name="output-propvariant-type"></a>Ausgabe-PROPVARIANT-Typ

VT \_ UI8

### <a name="denominator"></a>Vorzuschlagen

Highpart des uhval.

### <a name="numerator"></a>Zähler

LowPart des uhval.
