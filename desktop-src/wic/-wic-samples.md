# <a name="wic-samples"></a>WIC-Beispiele

Dieser Abschnitt enthält die beispiele Windows Imaging Component (WIC).

WIC bietet zwei Arten von Beispielen:

-   Beispiele: Codesegmente, die mehrere WIC-Features verwenden. Der Quellcode für das Beispiel wird im Windows SDK bereitgestellt.
-   Codebeispiele: Kürzere Beispiele, die ein bestimmtes Feature veranschaulichen. Der Quellcode für das Beispiel wird nicht mit dem SDK bereitgestellt.
