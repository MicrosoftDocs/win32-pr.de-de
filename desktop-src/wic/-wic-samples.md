# <a name="wic-samples"></a>WIC-Beispiele

Dieser Abschnitt enthält die Beispiele für Windows Imaging Component (WIC).

WIC bietet zwei Arten von Beispielen:

-   Beispiele: Code Segmente, in denen mehrere WIC-Funktionen verwendet werden. Der Quellcode für das Beispiel wird im Windows SDK übermittelt.
-   Code Beispiele: kürzere Beispiele, die eine bestimmte Funktion veranschaulichen. Der Quellcode für das Beispiel wird nicht mit dem SDK bereitgestellt.
