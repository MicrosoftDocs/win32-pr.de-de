# <a name="standard-clipboard-formats"></a>Standardformate der Zwischenablage

Die vom System definierten Zwischenablageformate werden als *Standardmäßige Zwischenablageformate* bezeichnet. Diese Zwischenablageformate werden in der folgenden Tabelle beschrieben.

## <a name="example"></a>Beispiel

```cpp
  case CF_BITMAP:
        hdcMem = CreateCompatibleDC(hdc);
        if (hdcMem != NULL)
        {
            if (OpenClipboard(hwnd))
            {
                hbm = (HBITMAP) 
                     GetClipboardData(uFormat);
                SelectObject(hdcMem, hbm);
                GetClientRect(hwnd, &rc);
 
                BitBlt(hdc, 0, 0, rc.right, rc.bottom,
                    hdcMem, 0, 0, SRCCOPY);
                CloseClipboard();
             }
             DeleteDC(hdcMem);
        }
    break;
```

Ein vollständiges Beispiel ist [die Verwendung von Zwischenablage.](using-the-clipboard.md)
