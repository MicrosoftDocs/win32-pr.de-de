# <a name="standard-clipboard-formats"></a>Standard Formate für Zwischenablage

Die vom System definierten Zwischenablage Formate werden als *Standardformate für die Zwischenablage* bezeichnet. Diese Zwischenablage Formate werden in der folgenden Tabelle beschrieben.

## <a name="example"></a>Beispiel

```cpp
  case CF_BITMAP:
        hdcMem = CreateCompatibleDC(hdc);
        if (hdcMem != NULL)
        {
            if (OpenClipboard(hwnd))
            {
                hbm = (HBITMAP) 
                     GetClipboardData(uFormat);
                SelectObject(hdcMem, hbm);
                GetClientRect(hwnd, &rc);
 
                BitBlt(hdc, 0, 0, rc.right, rc.bottom,
                    hdcMem, 0, 0, SRCCOPY);
                CloseClipboard();
             }
             DeleteDC(hdcMem);
        }
    break;
```

Vollständiges Beispiel ist die [Verwendung von Clipboards](using-the-clipboard.md).
