# <a name="client-context-reset"></a>Client Kontext Zurücksetzung

Wenn der Server nicht mehr verfügbar ist, kann die Client Anwendung seine Kontext Daten freigeben, indem die RPC-Funktion [**rpcssdestroyclientcontext**](/windows/desktop/api/Rpcndr/nf-rpcndr-rpcssdestroyclientcontext)aufgerufen wird.
