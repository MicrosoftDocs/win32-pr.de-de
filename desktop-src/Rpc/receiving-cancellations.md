# <a name="receiving-cancellations"></a>Empfangen von Abbrüchen

Die Serveranwendung kann [**rpcservertestcancel**](/windows/desktop/api/Rpcdce/nf-rpcdce-rpcservertestcancel) mit dem Bindungs Handle des betreffenden Aufrufes aufzurufen, um herauszufinden, ob der Client den Abbruch des asynchronen Aufrufes angefordert hat. RPC S OK wird zurückgegeben, \_ \_ Wenn der Client den Aufruf abgebrochen hat.
