# <a name="rpc_authz_handle"></a>RPC- \_ Authz- \_ handle

Der Datentyp des **RPC \_ Authz- \_ Handles** deklariert ein Autorisierungs handle. Dieses Handle verweist auf die Berechtigungsinformationen für die Client Anwendung, die den Remote Prozedur Aufrufvorgang durchgeführt hat.


```C++
typedef void* RPC_AUTHZ_HANDLE;
```
