# <a name="rpc_ep_inq_handle"></a>RPC \_ EP \_ INQ \_ HANDLE

Der **RPC \_ EP \_ INQ \_ HANDLE-Datentyp** deklariert ein Handle für einen Abfragekontext. RPC-Anwendungen verwenden sie zum Anzeigen von Serveradresseninformationen, die in der Endpunktzuordnung gespeichert sind.


```C++
typedef I_RPC_HANDLE* RPC_EP_INQ_HANDLE;
```
