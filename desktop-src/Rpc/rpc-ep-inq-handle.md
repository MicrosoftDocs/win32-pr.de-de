# <a name="rpc_ep_inq_handle"></a>RPC- \_ EP- \_ INQ- \_ handle

Der Datentyp des **RPC \_ EP- \_ INQ- \_ Handles** deklariert ein Handle für einen Abfrage Kontext. RPC-Anwendungen verwenden Sie, um die in der Endpunkt Zuordnung gespeicherten Server Adressinformationen anzuzeigen.


```C++
typedef I_RPC_HANDLE* RPC_EP_INQ_HANDLE;
```
