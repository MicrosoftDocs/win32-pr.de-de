# <a name="arrays-and-rpc"></a>Arrays und RPC

Die Programmiersprachen C und C++ Stellen im Wesentlichen zwei Arten von Arrays bereit: eindimensional und mehrdimensional. Mithilfe von RPC können Entwickler zusätzliche Array Typen mithilfe von Mittelwert Attributen angeben, um die Merkmale von Arrays in verteilten Anwendungen zu beschreiben.

In diesem Abschnitt werden die unter RPC verfügbaren Typen von Arrays beschrieben. Außerdem werden die Mittelwert Attribute erläutert, die Entwickler verwenden können, um eindimensionale und mehrdimensionale Arrays zu beschreiben. Dieses Material wird in den folgenden Themen vorgestellt:

-   [Arten von Arrays](kinds-of-arrays.md)
-   [Array Attribute](array-attributes.md)
-   [Mehrdimensionale Arrays](multidimensional-arrays.md)
