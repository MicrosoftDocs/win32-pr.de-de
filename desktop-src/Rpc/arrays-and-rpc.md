# <a name="arrays-and-rpc"></a>Arrays und RPC

Die Programmiersprachen C und C++ bieten im Wesentlichen zwei Typen von Arrays: eindimensional und mehrdimensional. RPC ermöglicht Entwicklern das Angeben zusätzlicher Arraytypen mithilfe von MIDL-Attributen, um die Merkmale von Arrays in verteilten Anwendungen zu beschreiben.

In diesem Abschnitt werden die Typen von Arrays beschrieben, die unter RPC verfügbar sind. Außerdem werden die MIDL-Attribute erläutert, mit denen Entwickler eindimensionale und mehrdimensionale Arrays beschreiben können. Dieses Material wird in den folgenden Themen vorgestellt:

-   [Arten von Arrays](kinds-of-arrays.md)
-   [Arrayattribute](array-attributes.md)
-   [Mehrdimensionale Arrays](multidimensional-arrays.md)
