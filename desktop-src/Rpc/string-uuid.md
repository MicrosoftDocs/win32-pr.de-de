# <a name="string-uuid"></a>String UUID

Eine Zeichenfolgen-UUID enthält die Zeichenarraydarstellung einer UUID. Eine Zeichenfolgen-UUID besteht aus mehreren Feldern mit Hexadezimalzeichen. Jedes Element hat eine feste Länge, und Felder werden durch den Bindestrich getrennt. Beispiel:

`989C6E5C-2CC1-11CA-A044-08002B1BB4F5`

Wenn Sie eine Zeichenfolgen-UUID als Eingabeparameter für eine RPC-Laufzeitfunktion bereitstellen, geben Sie die alphabetischen Hexadezimalzeichen entweder als Groß- oder Kleinbuchstaben ein. Wenn eine Laufzeitfunktion jedoch eine Zeichenfolgen-UUID zurückgibt, sind die Hexadezimalzeichen immer klein geschrieben.
