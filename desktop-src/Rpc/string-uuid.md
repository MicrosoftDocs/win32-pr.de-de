# <a name="string-uuid"></a>Zeichenfolge UUID

Eine Zeichen folgen-uuid enthält die Zeichen Array Darstellung einer UUID. Eine Zeichenfolge UUID besteht aus mehreren Feldern von hexadezimalen Zeichen. Jeder Member hat eine Länge mit fester Länge, und Felder werden durch das Bindestrich Zeichen getrennt. Beispiel:

`989C6E5C-2CC1-11CA-A044-08002B1BB4F5`

Wenn Sie eine Zeichenfolge UUID als Eingabeparameter für eine RPC-Lauf Zeitfunktion bereitstellen, geben Sie die alphabetischen hexadezimalen Zeichen entweder als groß-oder Kleinbuchstaben ein. Wenn eine Lauf Zeitfunktion jedoch eine Zeichenfolge UUID zurückgibt, werden die hexadezimalen Zeichen immer in Kleinbuchstaben geschrieben.
