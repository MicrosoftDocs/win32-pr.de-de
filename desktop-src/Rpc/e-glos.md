# <a name="e-rpc"></a>E (RPC)

[A](a-glos.md) [B](b-glos.md) [C](c-glos.md) [D](d-glos.md) E [F](f-glos.md) G H [I](i-glos.md) J K [L](l-glos.md) [M](m-glos.md) [N](n-glos.md) [O](o-glos.md) [P](p-glos.md) [Q](q.md) [R](r-glos.md) [S](s-glos.md) [T](t-glos.md) [U](u-glos.md) [V](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_embedded_pointer_glos"></span><span id="_RPC_EMBEDDED_POINTER_GLOS"></span>**Eingebetteter Zeiger**
</dt> <dd>

Zeiger, der in einen Parameter eingebettet ist, der eine Datenstruktur ist, z. B. ein Array, eine Struktur oder eine Union. Siehe auch [*Zeiger der obersten Ebene.*](t-glos.md)

</dd> <dt>

<span id="_rpc_encoding_services_glos"></span><span id="_RPC_ENCODING_SERVICES_GLOS"></span>**Codierungsdienste**
</dt> <dd>

MIDL-generierte Stubroutinen, die Unterstützung für die Datencodierung und -decodierung bieten (auch als *Pickling* oder *Serialisierung* bezeichnet). Ermöglicht Programmierern das Steuern von Puffern, die Daten enthalten, die gemarshallt und nicht gemarshallt werden sollen. Siehe auch [*Typserialisierung,*](t-glos.md) [*Prozedurserialisierung.*](p-glos.md)

</dd> <dt>

<span id="_rpc_endpoint_glos"></span><span id="_RPC_ENDPOINT_GLOS"></span>**Endpunkt**
</dt> <dd>

Netzwerkspezifische Adresse eines Serverprozesses für Remoteprozeduraufrufe. Der tatsächliche Name des Endpunkts hängt von der verwendeten Protokollsequenz ab. Siehe auch [*dynamischer Endpunkt*](d-glos.md) und [*bekannter Endpunkt*](w-glos.md).

</dd> <dt>

<span id="_rpc_endpoint_mapper_glos"></span><span id="_RPC_ENDPOINT_MAPPER_GLOS"></span>**Endpunktzuordnung**
</dt> <dd>

Teil des RPC-Subsystems (RPCSS), das es der Laufzeitbibliothek ermöglicht, Endpunkte dynamisch zuzuweisen und aufzulösen. Siehe auch [Endpunkt](/windows).

</dd> <dt>

<span id="_rpc_encapsulated_union_glos"></span><span id="_RPC_ENCAPSULATED_UNION_GLOS"></span>**Gekapselte Union**
</dt> <dd>

MIDL-Konstrukt, mit dem Unions als Teil eines Remoteprozeduraufrufs übergeben werden können, indem die Union in eine Struktur eingebettet wird, in der der Diskriminant das erste Feld der Struktur und die Union das zweite (und letzte) Feld der Struktur ist. Der **IDL-Schlüsselwortschalter** gibt an, dass eine Union gekapselt ist. [](i-glos.md) Siehe auch [*nicht gekapselte Union.*](n-glos.md)

</dd> <dt>

<span id="_rpc_epv_glos"></span><span id="_RPC_EPV_GLOS"></span>**Einstiegspunktvektor (EPV)**
</dt> <dd>

Array von Zeigern auf Funktionen, die die in der -Schnittstelle angegebenen Vorgänge implementieren. Jedes Element im Array entspricht einer funktion, die in der IDL-Datei definiert ist. Einstiegspunktvektoren ermöglichen verteilten Anwendungen die Unterstützung mehrerer Implementierungen der in der IDL-Datei definierten Funktionen.

</dd> </dl>
