# <a name="e-rpc"></a>E (RPC)

[a](a-glos.md) [B](b-glos.md) [C](c-glos.md) [D](d-glos.md) E [F](f-glos.md) G H [I](i-glos.md) J K [L](l-glos.md) [M](m-glos.md) [N](n-glos.md) [O](o-glos.md) [P](p-glos.md) [Q](q.md) [R](r-glos.md) [S](s-glos.md) [T](t-glos.md) [U](u-glos.md) [V](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_embedded_pointer_glos"></span><span id="_RPC_EMBEDDED_POINTER_GLOS"></span>**eingebetteter Zeiger**
</dt> <dd>

Ein Zeiger, der in einen Parameter eingebettet ist, bei dem es sich um eine Datenstruktur wie Array, Struktur oder Union handelt. Siehe auch [*Zeiger der obersten Ebene*](t-glos.md).

</dd> <dt>

<span id="_rpc_encoding_services_glos"></span><span id="_RPC_ENCODING_SERVICES_GLOS"></span>**Codierungs Dienste**
</dt> <dd>

In der Mitte generierte Stub-Routinen, die Unterstützung für die Daten Codierung und-Decodierung bieten (auch als *Pickup* oder *Serialisierung* bezeichnet). Ermöglicht es Programmierern, Puffer mit Daten zu steuern, die gemarshallt und gemarshallt werden sollen. Siehe auch [*Type Serialisierung*](t-glos.md), [*Procedure Serialisierung*](p-glos.md).

</dd> <dt>

<span id="_rpc_endpoint_glos"></span><span id="_RPC_ENDPOINT_GLOS"></span>**Dreher**
</dt> <dd>

Netzwerk spezifische Adresse eines Server Prozesses für Remote Prozedur Aufrufe. Der tatsächliche Name des Endpunkts hängt von der verwendeten Protokoll Sequenz ab. Siehe auch [*dynamischer Endpunkt*](d-glos.md) und [*bekannter Endpunkt*](w-glos.md).

</dd> <dt>

<span id="_rpc_endpoint_mapper_glos"></span><span id="_RPC_ENDPOINT_MAPPER_GLOS"></span>**Endpunkt Zuordnung**
</dt> <dd>

Teil des RPC-Subsystems (RPCSS), das der Lauf Zeit Bibliothek das dynamische zuweisen und Auflösen von Endpunkten ermöglicht. Siehe auch [Endpunkt](/windows).

</dd> <dt>

<span id="_rpc_encapsulated_union_glos"></span><span id="_RPC_ENCAPSULATED_UNION_GLOS"></span>**kapselte Union**
</dt> <dd>

Das Mittel l-Konstrukt, das das Weitergeben von Unions als Teil eines Remote Prozedur Aufrufes ermöglicht, indem die Union in eine Struktur eingebettet wird, in der der diskriminant das erste Feld der Struktur ist, und die Union das zweite (und abschließende) Feld der Struktur ist. Das [*IDL*](i-glos.md) -Schlüsselwort **Switch** gibt an, dass eine Union gekapselt ist. Siehe auch [*nicht gekapselte Union*](n-glos.md).

</dd> <dt>

<span id="_rpc_epv_glos"></span><span id="_RPC_EPV_GLOS"></span>**Einstiegspunkt Vektor (EPV)**
</dt> <dd>

Array von Zeigern auf Funktionen, die die in der-Schnittstelle angegebenen Vorgänge implementieren. Jedes Element im Array entspricht einer Funktion, die in der IDL-Datei definiert ist. Mit Einstiegspunkt Vektoren können verteilte Anwendungen mehr als eine Implementierung der Funktionen unterstützen, die in der IDL-Datei definiert sind.

</dd> </dl>
