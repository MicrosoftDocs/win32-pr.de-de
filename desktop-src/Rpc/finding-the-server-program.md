# <a name="finding-the-server-program"></a>Suchen des Server Programms

Nachdem die Client seitige RPC-Laufzeit eine Verbindung mit einem im Bindungs handle angeforderten Server Host System hergestellt hat, wird der Server Prozess von der RPC-Lauf Zeit Bibliothek des Clients gefunden. Zu diesem Zweck wird die Endpunkt Zuordnung auf dem Server Host System abgefragt. Die Endpunkt Zuordnung enthält Informationen zu dem Endpunkt, an dem der Server lauscht.
