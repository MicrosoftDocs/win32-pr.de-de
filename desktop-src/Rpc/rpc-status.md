# <a name="rpc_status"></a>\_RPC-STATUS

Der Datentyp **RPC \_ STATUS** stellt einen plattformspezifischen Statuscodetyp dar.


```C++
typedef long RPC_STATUS;
typedef unsigned short RPC_STATUS;
```



## <a name="remarks"></a>Hinweise

Der **RPC \_ STATUS-Typ** wird von den meisten RPC-Funktionen zurückgegeben und ist Teil der [**\_ FN-Funktionstypdefinition RPC OBJECT \_ \_ INQ.**](/windows/desktop/api/Rpcdce/nc-rpcdce-rpc_object_inq_fn)
