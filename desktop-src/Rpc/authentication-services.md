# <a name="authentication-services"></a>Authentifizierungsdienste

Die Security Support Provider-Schnittstelle (SSPI) stellt die zugrunde liegenden Authentifizierungsdienste für RPC bereit. Wenn Ihre Anwendung einen Authentifizierungsdienst angibt, wird daher ein SSP ausgewählt. Eine Liste der SSPs, die derzeit von SPPI unterstützt werden, finden Sie unter [Authentication-Service-Konstanten](authentication-service-constants.md). Weitere Informationen finden Sie [unter Security Support Provider Interface (SSPI)](security-support-provider-interface-sspi-.md).

> [!Note]  
> Die Begriffe Sicherheitsanbieter und Authentifizierungsdienst sind austauschbar.
