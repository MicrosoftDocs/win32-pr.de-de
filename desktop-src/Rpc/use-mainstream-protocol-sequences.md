# <a name="use-mainstream-protocol-sequences"></a>Verwenden von gängigen Protokoll Sequenzen

Verwenden Sie [**ncacn \_ IP \_ TCP**](/windows/desktop/Midl/ncacn-ip-tcp), [**ncacn \_ http**](/windows/desktop/Midl/ncacn-http) oder [**ncacn \_ NP**](/windows/desktop/Midl/ncacn-np) für Remote-RPC-Aufrufe. Verwenden Sie [**Ncalrpc**](/windows/desktop/Midl/ncalrpc) für lokale RPC-Aufrufe. Diese Protokolle werden am häufigsten verwendet und erhalten die meisten Überprüfungen und Tests.
