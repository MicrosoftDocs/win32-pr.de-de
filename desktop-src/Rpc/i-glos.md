<dl> <dt>

<span id="_rpc_idempotent_glos"></span><span id="_RPC_IDEMPOTENT_GLOS"></span>**idempotent**
</dt> <dd>

Ein Remote Prozedur Aufruf, der einen Zustand nicht ändert und die gleichen Informationen zurückgibt, wenn er mit denselben Eingabe Parametern aufgerufen wird.

</dd> <dt>

<span id="_rpc_input_parameter_glos"></span><span id="_RPC_INPUT_PARAMETER_GLOS"></span>**Eingabeparameter**
</dt> <dd>

-Parameter, der Daten enthält, die von der Client Anwendung an die Serveranwendung übertragen werden. Das **in** -Attribut bezeichnet einen Eingabeparameter.

</dd> <dt>

<span id="_rpc_idl_glos"></span><span id="_RPC_IDL_GLOS"></span>**Interface Definition Language (IDL)**
</dt> <dd>

OSF: DCE-Standardsprache zum Angeben der-Schnittstelle für Remote Prozedur Aufrufe. Siehe auch " [*Mittel l*](m-glos.md)".

</dd> <dt>

<span id="_rpc_ip_glos"></span><span id="_RPC_IP_GLOS"></span>**Internet Protokoll (IP)**
</dt> <dd>

Verbindungsloses Netzwerkschicht-Kommunikationsprotokoll. Siehe auch [*Datagramm*](d-glos.md).

</dd> <dt>

<span id="_rpc_ipx_glos"></span><span id="_RPC_IPX_GLOS"></span>**Internetwork-Paket Austausch (IPX)**
</dt> <dd>

Novell NetWare Communication Protocol, das [*Datagramm*](d-glos.md) -Sockets zum Weiterleiten von Informationspaketen über lokale Netzwerke und Wide Area Networks verwendet.

</dd> <dt>

<span id="_rpc_intranet_glos"></span><span id="_RPC_INTRANET_GLOS"></span>**Intranet**
</dt> <dd>

Privates Netzwerk, das Internet Produkte und-Technologien (z. b. Webserver) verwendet, aber nicht für externe Internet Benutzer verfügbar ist.

</dd> </dl>
