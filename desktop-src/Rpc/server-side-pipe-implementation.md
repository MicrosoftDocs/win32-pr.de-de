# <a name="server-side-pipe-implementation"></a>Server-Side Pipeimplementierung

Serverprogramme für verteilte Anwendungen, die Pipes verwenden, müssen keine Push-, Pull- oder Alloc-Funktionen implementieren. Sie müssen Prozeduren enthalten, die Clients remote aufrufen können, um Datenübertragungen zu initiieren. In den folgenden Themen wird in diesem Abschnitt erläutert, wie die Remoteverfahren des Servers implementiert werden können.

-   [Implementieren von Eingabepipes auf dem Server](implementing-input-pipes-on-the-server.md)
-   [Implementieren von Ausgabepipes auf dem Server](implementing-output-pipes-on-the-server.md)
