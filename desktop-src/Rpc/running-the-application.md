# <a name="running-the-application"></a>Ausführen der Anwendung

Öffnen Sie zwei Konsolenfenster, um die Anwendung auf einem einzelnen Computer ausführen zu können. Geben Sie im ersten Fenster ein.

**Hellos**

und geben Sie im zweiten Fenster ein.

**helloc**

Um mit verschiedenen Protokollsequenzen, Endpunkten und anderen Optionen zu experimentieren, erstellen Sie die Hello-Beispielanwendung aus den Quelldateien im RPC Hello-Verzeichnis im \\ Platform Software Development Kit (SDK).

> [!Note]  
> **Die RPC-Schnittstelleneinschränkung** kann dazu  führen, dass für diese Anwendung auf Windows XP mit Service Pack 2 (SP2) und neueren Versionen von Windows. Weitere Informationen finden Sie unter [RPC-Schnittstelleneinschränkung.](/previous-versions/windows/it-pro/windows-xp/bb457156(v=technet.10)#EGAA)
