# <a name="running-the-application"></a>Ausführen der Anwendung

Öffnen Sie zwei Konsolenfenster, um die Anwendung auf einem einzelnen Computer auszuführen. Geben Sie im ersten Fenster

**Hellos**

und geben Sie im zweiten Fenster

**helloc**

Um mit verschiedenen Protokoll Sequenzen, Endpunkten und anderen Optionen zu experimentieren, erstellen Sie die Hello-Beispielanwendung aus den Quelldateien im RPC- \\ Verzeichnis "Hello" im Platform Software Development Kit (SDK).

> [!Note]  
> Die **Einschränkung der RPC-Schnittstelle** kann dazu führen, dass diese Anwendung mit dem Fehler " **Zugriff verweigert** " unter Windows XP mit Service Pack 2 (SP2) und höheren Versionen von Windows fehlschlägt. Weitere Informationen finden Sie unter [Einschränkung der RPC-Schnittstelle](/previous-versions/windows/it-pro/windows-xp/bb457156(v=technet.10)#EGAA).
