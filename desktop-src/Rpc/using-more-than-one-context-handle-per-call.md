# <a name="using-more-than-one-context-handle-per-call"></a>Verwenden von mehr als einem Kontexthandle pro Aufruf

Die Möglichkeit, Arrays von Kontexthandles als Parameter zu verwenden, wird in Windows Vista und späteren Betriebssystemen verfügbar gemacht. Dieses Feature sollte jedoch in einer Anwendung sorgfältig verwendet werden. In situationen, in denen ein Kontexthandle in mehreren Aufrufen verwendet wird, wird es immer schwieriger, die Verwendung nachzuverfolgen.
