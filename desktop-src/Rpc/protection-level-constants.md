# <a name="protection-level-constants"></a>Konstanten auf Schutz Ebene

Die OSF/DCE-RPC-Spezifikation hat die Konstanten auf Authentifizierungs Ebene umbenannt. Sie werden nun als schutzebenendkonstanten bezeichnet. Microsoft RPC unterstützt sowohl den alten als auch den neuen Namen. Weitere Informationen finden Sie unter [Konstanten auf Authentifizierungs Ebene](authentication-level-constants.md).
