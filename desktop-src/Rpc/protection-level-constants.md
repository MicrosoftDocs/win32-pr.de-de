# <a name="protection-level-constants"></a>Konstanten auf Schutzebene

Die OSF-/DCE-RPC-Spezifikation hat die Konstanten auf Authentifizierungsebene umbenannt. Sie werden jetzt als Konstanten auf Schutzebene bezeichnet. Microsoft RPC unterstützt sowohl den alten als auch den neuen Namen. Weitere Informationen finden Sie unter [Konstanten auf Authentifizierungsebene.](authentication-level-constants.md)
