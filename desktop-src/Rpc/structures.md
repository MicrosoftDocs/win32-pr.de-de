# <a name="structs-rpc"></a>Strukturen (RPC)

Normale C-Semantik gilt für die Felder von Basistypen. Felder komplexerer Typen, z. B. Zeiger, Arrays und andere konstruierte Typen, können durch Typ- oder [**\_ Feldattribute**](field-attributes.md)geändert werden. Weitere Informationen finden Sie unter [**struktur**](/windows/desktop/Midl/struct).
