# <a name="structs-rpc"></a>Strukturen (RPC)

Die normale C-Semantik gilt für die Felder von Basis Typen. Felder komplexer Typen, z. b. Zeiger, Arrays und andere erstellte Typen, können nach Typ-oder [**Feld \_ Attributen**](field-attributes.md)geändert werden. Weitere Informationen finden Sie unter [**struct**](/windows/desktop/Midl/struct).
