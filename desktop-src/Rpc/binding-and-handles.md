# <a name="binding-and-handles"></a>Bindung und Handles

In diesem Abschnitt wird das Erstellen und Verwenden von Bindungen und Bindungshandles zwischen Client- und Serverprogrammen erläutert. Außerdem werden Client-/Serverkontexte und Kontexthandles erläutert. Die Informationen sind in die folgenden Themen unterteilt:

-   [Bindungshandles](binding-handles.md)
-   [Kontexthandles](context-handles.md)

Zusätzlich zu Bindungs- und Kontexthandles unterstützt Microsoft RPC auch Serialisierungshandles, die zum Codieren oder Decodieren von Daten verwendet werden. Diese werden für die Serialisierung auf einem lokalen Computer verwendet und umfassen keine Remotebindung. Weitere Informationen finden Sie unter [Serialisierungsdienste](serialization-services.md).
