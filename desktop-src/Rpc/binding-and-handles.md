# <a name="binding-and-handles"></a>Bindung und Handles

In diesem Abschnitt wird das Erstellen und Verwenden von Bindungen und Bindungs Handles zwischen Client-und Serverprogrammen erläutert. Außerdem werden Client-/serverkontexte und Kontext Handles erläutert. Die Informationen sind in folgende Themen unterteilt:

-   [Bindungs Handles](binding-handles.md)
-   [Kontext Handles](context-handles.md)

Zusätzlich zu Bindungs-und Kontext Handles unterstützt Microsoft RPC auch serialisierungshandles, die zum Codieren oder Decodieren von Daten verwendet werden. Diese werden für die Serialisierung auf einem lokalen Computer verwendet und beinhalten keine Remote Bindung. Weitere Informationen finden Sie unter [Serialization Services](serialization-services.md).
