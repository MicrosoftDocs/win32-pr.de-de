# <a name="rpctryfinally"></a>Rpctryschließlich

Die **rpctryschluss** -Anweisung stellt eine strukturierte Abbruch Behandlung bereit. Es tritt eine Ausnahme während der Ausführungs Anweisungen des Codeblocks auf, der der **rpctryendlich** -Anweisung zugeordnet ist. die Anweisungen im Codeblock, die der [**rpcletzenanweisung**](/previous-versions/aa375699(v=vs.80)) zugeordnet sind, werden ausgeführt. Alle **rpctryend** -Anweisungen müssen durch eine [**rpcendendlich**](/previous-versions/aa375634(v=vs.80)) -Anweisung beendet werden.

Weitere Informationen finden Sie unter [**rpcschließlich**](/previous-versions/aa375699(v=vs.80)).
