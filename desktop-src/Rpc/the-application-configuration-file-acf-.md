# <a name="the-application-configuration-file-acf"></a>Die Anwendungs Konfigurationsdatei (ACF)

Die Anwendungs Konfigurationsdatei (ACF) besteht aus zwei Teilen: einem [Schnittstellen Header](the-acf-header.md), ähnlich dem Schnittstellen Header in der IDL-Datei und einem [Text](the-acf-body.md), der Konfigurations Attribute enthält, die für Typen und Funktionen gelten, die im Schnittstellen Text der IDL-Datei definiert sind. Diese werden in den folgenden Themen erläutert:

-   [Der ACF-Header](the-acf-header.md)
-   [Der ACF-Text](the-acf-body.md)
