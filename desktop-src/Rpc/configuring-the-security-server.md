# <a name="configuring-the-security-server"></a>Konfigurieren des Sicherheits Servers

Im folgenden werden die Schritte zum Konfigurieren des Sicherheits Servers für Microsoft RPC erläutert.

**So konfigurieren Sie den Sicherheitsserver für RPC**

1.  Starten Sie die Systemsteuerung.
2.  Klicken Sie in der Systemsteuerung auf das Symbol **Netzwerke** . Das Dialogfeld **Netzwerkeinstellungen** wird angezeigt.
3.  Wählen Sie in der Liste **installierte Netzwerk Software** die Option **RPC-Konfiguration** aus. Das Dialogfeld **RPC-Konfiguration** wird angezeigt.
4.  Wählen Sie in der Liste **Security Service Provider (Security Service Provider** ) einen oder mehrere Sicherheitsanbieter aus.
5.  Klicken Sie auf **OK**.
