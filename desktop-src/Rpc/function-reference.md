# <a name="function-reference"></a>Funktionsreferenz

In diesem Abschnitt werden die rpc-Laufzeitfunktionen (Remote Procedure Call, Remoteprozeduraufruf) beschrieben, die in Microsoft RPC unterstützt werden. In diesem Abschnitt werden der Zweck, die Syntax, die Eingabeparameter und die Rückgabewerte der einzelnen Funktionen beschrieben. Außerdem werden zusätzliche Informationen zur Verwendung von RPC-Funktionen in einer Anwendung zur Verfügung stehen.

Alle Zeiger, die an RPC-Funktionen übergeben werden, müssen das **\_ \_ RPC \_ FAR-Attribut** enthalten.
