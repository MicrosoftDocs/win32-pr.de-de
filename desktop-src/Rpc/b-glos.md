<dl> <dt>

<span id="_rpc_bind_glos"></span><span id="_RPC_BIND_GLOS"></span>**Zwick**
</dt> <dd>

Prozess, über den eine logische Verbindung zwischen einem Client und einem Server hergestellt wird, um einen Remote Prozedur Aufrufe an diesen Server weiterzuleiten.

</dd> <dt>

<span id="_rpc_binding_glos"></span><span id="_RPC_BINDING_GLOS"></span>**lichere**
</dt> <dd>

Beziehung zwischen einem Client und einem Server, der über eine bestimmte Protokoll Sequenz zu einem bestimmten Host System und Endpunkt hergestellt wird. Wird auch als Kurzform des [Bindungs Handles](/windows)verwendet.

</dd> <dt>

<span id="_rpc_binding_handle_glos"></span><span id="_RPC_BINDING_HANDLE_GLOS"></span>**Bindungs handle**
</dt> <dd>

Datenstruktur, die die logische Verbindung zwischen einem Client und einem Server darstellt.

</dd> <dt>

<span id="_rpc_binding_vector_glos"></span><span id="_RPC_BINDING_VECTOR_GLOS"></span>**Bindungs Vektor**
</dt> <dd>

Eine RPC-Datenstruktur, die eine Liste von [Bindungs Handles](/windows) enthält, über die eine Serveranwendung Remote Prozedur Aufrufe empfangen kann.

</dd> </dl>
