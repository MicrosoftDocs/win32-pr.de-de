<dl> <dt>

<span id="_rpc_bind_glos"></span><span id="_RPC_BIND_GLOS"></span>**Binden**
</dt> <dd>

Prozess, über den eine logische Verbindung zwischen einem Client und einem Server hergestellt wird, um einen Remoteprozeduraufruf an diesen Server weiterzuleiten.

</dd> <dt>

<span id="_rpc_binding_glos"></span><span id="_RPC_BINDING_GLOS"></span>**Bindung**
</dt> <dd>

Beziehung zwischen einem Client und einem Server, die über eine bestimmte Protokollsequenz zu einem bestimmten Hostsystem und Endpunkt eingerichtet wurde. Wird auch als Kurzform des [Bindungshandle](/windows)verwendet.

</dd> <dt>

<span id="_rpc_binding_handle_glos"></span><span id="_RPC_BINDING_HANDLE_GLOS"></span>**Bindungshandle**
</dt> <dd>

Datenstruktur, die die logische Verbindung zwischen einem Client und einem Server darstellt.

</dd> <dt>

<span id="_rpc_binding_vector_glos"></span><span id="_RPC_BINDING_VECTOR_GLOS"></span>**Bindungsvektor**
</dt> <dd>

RPC-Datenstruktur, die eine Liste von [Bindungshandles](/windows) enthält, über die eine Serveranwendung Remoteprozeduraufrufe empfangen kann.

</dd> </dl>
