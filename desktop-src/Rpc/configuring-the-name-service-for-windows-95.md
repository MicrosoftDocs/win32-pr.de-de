# <a name="configuring-the-name-service-for-windows-95"></a>Konfigurieren des Namensdiensts für Windows 95

Windows 95 verwendet keinen Microsoft-Locator. Um einen Namensdienst in einer Windows 95-Anwendung zu verwenden, muss der Computer mit Windows 95 eine der folgenden Angaben haben:

-   Sie müssen Teil einer Arbeitsgruppe oder Domäne sein, die einen Computer mit Windows 2000 oder Windows NT als Proxynamen-Dienstanbieter enthält.
-   Sie müssen mit einem Hostcomputer verbunden sein, auf dem der NSI-Daemon (nsid) ausgeführt wird, der als Gateway für den Digital Equipment Corporation DCE Cell Directory-Dienst dient.
