<dl> <dt>

<span id="_rpc_odl_glos"></span><span id="_RPC_ODL_GLOS"></span>**Object Description Language (ODL)**
</dt> <dd>

Teilmenge der [*MIDL-Attribute,*](m-glos.md) -Schlüsselwörter, -Anweisungen und -Anweisungen, die zum Definieren von Typbibliotheken für OLE Automation-Anwendungen verwendet werden.

</dd> <dt>

<span id="_rpc_open_array_glos"></span><span id="_RPC_OPEN_ARRAY_GLOS"></span>**Open Array**
</dt> <dd>

Array, das sowohl konform als auch variierend ist; Das heißt, sowohl die Größe als auch der Bereich der übertragenen Elemente werden zur Laufzeit durch andere Parameter, Strukturen oder Ausdrücke bestimmt.

</dd> <dt>

<span id="_rpc_osf_glos"></span><span id="_RPC_OSF_GLOS"></span>**Open Software Foundation (OSF)**
</dt> <dd>

Konsortium von Unternehmen, das zum Definieren der verteilten Computingumgebung (Distributed Computing Environment, DCE) gebildet wurde.

</dd> <dt>

<span id="_rpc_orphaned_memory_glos"></span><span id="_RPC_ORPHANED_MEMORY_GLOS"></span>**Verwaister Arbeitsspeicher**
</dt> <dd>

Arbeitsspeicher auf einem Client, auf den zuvor von einem Zeigerparameter verwiesen wurde, der vom Server auf NULL zurückgesetzt wurde.

</dd> <dt>

<span id="_rpc_output_parameter_glos"></span><span id="_RPC_OUTPUT_PARAMETER_GLOS"></span>**Ausgabeparameter**
</dt> <dd>

Parameter, der Daten enthält, die von der Serveranwendung an die Clientanwendung übertragen werden. Das \[ [out-Attribut](/windows/desktop/Midl/out-idl) \] legt einen Ausgabeparameter fest.

</dd> </dl>
