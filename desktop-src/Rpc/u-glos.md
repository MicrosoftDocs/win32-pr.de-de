# <a name="u-rpc"></a>U (RPC)

[a](a-glos.md) [B](b-glos.md) [C](c-glos.md) [D](d-glos.md) [E](e-glos.md) [F](f-glos.md) G H [I](i-glos.md) J K [L](l-glos.md) [M](m-glos.md) [N](n-glos.md) [O](o-glos.md) [P](p-glos.md) [Q](q.md) [R](r-glos.md) [S](s-glos.md) [T](t-glos.md) U [V](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_udp_glos"></span><span id="_RPC_UDP_GLOS"></span>**User Datagram-Protokoll (UDP)**
</dt> <dd>

Netzwerk Transport, der verbindungslose [*Datagramm*](d-glos.md) -Sockets verwendet und oberhalb des Internet Protokolls ([*IP)*](i-glos.md)angeordnet ist.

</dd> <dt>

<span id="_rpc_unbind_glos"></span><span id="_RPC_UNBIND_GLOS"></span>**Bindung**
</dt> <dd>

Prozess zum Beenden der logischen Verbindung zwischen einem Client und einem Server.

</dd> <dt>

<span id="_rpc_unique_pointer_glos"></span><span id="_RPC_UNIQUE_POINTER_GLOS"></span>**eindeutiger Zeiger**
</dt> <dd>

Ein Zeiger, der NULL sein kann oder auf vorhandene Daten verweist, deren Wert während eines Remote Prozedur Aufrufes geändert werden kann. Einem eindeutigen Zeiger kann kein Alias zugeordnet werden. Das \[ [Unique](/windows/desktop/Midl/unique) - \] Attribut legt einen eindeutigen Zeiger fest. Siehe auch [*Vollständiger Zeiger*](f-glos.md), [*Verweis Zeiger*](r-glos.md).

</dd> <dt>

<span id="_rpc_uuid_glos"></span><span id="_RPC_UUID_GLOS"></span>**Universal Unique Identifier (UUID)**
</dt> <dd>

(oder [**GUID**](/windows/win32/api/guiddef/ns-guiddef-guid)) 128-Bit-Wert, der in Prozess übergreifender Kommunikation verwendet wird, um Entitäten wie Client-und Server Schnittstellen, Manager-Einstiegspunkt-Vektoren und RPC-Objekte zu identifizieren. Siehe auch ["uuidgen](/windows).

</dd> <dt>

<span id="_rpc_unmarshaling_glos"></span><span id="_RPC_UNMARSHALING_GLOS"></span>**Unmarshalling**
</dt> <dd>

Prozess der Entpacken von Parametern, die über Prozess Grenzen hinweg gesendet wurden.

</dd> <dt>

<span id="_rpc_uuidgen_glos"></span><span id="_RPC_UUIDGEN_GLOS"></span>**"uuidgen**
</dt> <dd>

Hilfsprogrammprogramm, das mit dem Platform Software Development Kit (SDK) bereitgestellt wird, das einen Zeitwert und die Netzwerkkarten-ID Ihres Computers verwendet, um [UUIDs](/windows) zu generieren, die garantiert eindeutig sind.

</dd> </dl>
