# <a name="u-rpc"></a>U (RPC)

<dl> <dt>

<span id="_rpc_udp_glos"></span><span id="_RPC_UDP_GLOS"></span>**User Datagram Protocol (UDP)**
</dt> <dd>

Netzwerktransport, der [](d-glos.md) verbindungslose Datagrammsockets verwendet und über dem Internetprotokoll [*(IP) liegt.*](i-glos.md)

</dd> <dt>

<span id="_rpc_unbind_glos"></span><span id="_RPC_UNBIND_GLOS"></span>**Bindung**
</dt> <dd>

Prozess zum Beenden der logischen Verbindung zwischen einem Client und einem Server.

</dd> <dt>

<span id="_rpc_unique_pointer_glos"></span><span id="_RPC_UNIQUE_POINTER_GLOS"></span>**Eindeutiger Zeiger**
</dt> <dd>

Zeiger, der NULL sein kann oder auf vorhandene Daten zeigen kann, deren Wert sich während eines Remoteprozeduraufrufs ändern kann. Für einen eindeutigen Zeiger kann kein Alias verwendet werden. Das \[ [eindeutige](/windows/desktop/Midl/unique) \] Attribut bezeichnet einen eindeutigen Zeiger. Siehe auch [*vollständiger Zeiger ,*](f-glos.md) [*Verweiszeiger*](r-glos.md).

</dd> <dt>

<span id="_rpc_uuid_glos"></span><span id="_RPC_UUID_GLOS"></span>**Universal Unique Identifier (UUID)**
</dt> <dd>

(oder [**GUID)**](/windows/win32/api/guiddef/ns-guiddef-guid)128-Bit-Wert, der bei der prozessübergreifenden Kommunikation verwendet wird, um Entitäten wie Client- und Serverschnittstellen, Manager-Einstiegspunktvektoren und RPC-Objekte zu identifizieren. Siehe auch [uuidgen](/windows).

</dd> <dt>

<span id="_rpc_unmarshaling_glos"></span><span id="_RPC_UNMARSHALING_GLOS"></span>**Unmarshaling**
</dt> <dd>

Prozess des Entpackens von Parametern, die über Prozessgrenzen hinweg gesendet wurden.

</dd> <dt>

<span id="_rpc_uuidgen_glos"></span><span id="_RPC_UUIDGEN_GLOS"></span>**uuidgen**
</dt> <dd>

Hilfsprogramm, das mit dem Platform Software Development Kit (SDK) bereitgestellt wird, das einen Zeitwert und die Netzwerkkarten-ID Ihres Computers verwendet, um [UUIDs](/windows) zu generieren, die garantiert eindeutig sind.

</dd> </dl>
