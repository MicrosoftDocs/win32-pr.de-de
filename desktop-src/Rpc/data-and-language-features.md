# <a name="data-and-language-features"></a>Daten-und Sprachfunktionen

Der Microsoft Interface Definition Language (Mittelwert) stellt die Funktionen bereit, die die Programmiersprache C erweitern, um Remote Prozedur Aufrufe zu unterstützen. "Mittel l" ist keine Variation von "C;". Dabei handelt es sich um eine stark typisierte formale Sprache, über die Sie die über ein Netzwerk übertragenen Daten steuern können. Die Mittel l ist so konzipiert, dass Entwickler, die mit C vertraut sind, diese schnell erlernen können.

In diesem Thema werden drei sprach Features erläutert:

-   [Starke Typisierung](strong-typing.md)
-   [Direktionale Attribute (Parameter)](directional-parameter-attributes.md)
-   [Datendarstellung](data-representation.md)
