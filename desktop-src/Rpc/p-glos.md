</dd> <dt>

<span id="_rpc_pickling_glos"></span><span id="_RPC_PICKLING_GLOS"></span>**Pickeln**
</dt> <dd>

Siehe [*Serialisierung*](s-glos.md).

</dd> <dt>

<span id="_rpc_pipe_glos"></span><span id="_RPC_PIPE_GLOS"></span>**Kanal**
</dt> <dd>

Der [*IDL*](i-glos.md)-Type-Konstruktor, der die Übertragung eines geöffneten Datenstroms zwischen Client-und Server Anwendungen unterstützt.

</dd> <dt>

<span id="_rpc_primary_enterprise_controller_glos"></span><span id="_RPC_PRIMARY_ENTERPRISE_CONTROLLER_GLOS"></span>**primärer Enterprise Controller**
</dt> <dd>

Die MQIS-Datenbank (Master Message Queue Information Service) für ein Netzwerk. Siehe auch [*MQIS (Message Queue Information Service)*](m-glos.md).

</dd> <dt>

<span id="_rpc_primary_site_controller_glos"></span><span id="_RPC_PRIMARY_SITE_CONTROLLER_GLOS"></span>**primärer Standort Controller**
</dt> <dd>

MQIS-Datenbank für einen bestimmten Standort innerhalb eines Unternehmens. Siehe auch [*MQIS (Message Queue Information Service)*](m-glos.md).

</dd> <dt>

<span id="_rpc_procedure_serialization_glos"></span><span id="_RPC_PROCEDURE_SERIALIZATION_GLOS"></span>**prozessserialisierung**
</dt> <dd>

[*Datenserialisierung*](s-glos.md) , die einen von der mittleren l generierten serialisierungsstub verwendet, um die Codierung und Decodierung von einem oder mehreren Typen mit einem einzigen Prozedur aufzurufen. Die Serialisierung von Prozeduren erfolgt durch Anwenden der \[ Attribute [Codieren](/windows/desktop/Midl/encode) \] und \[ [decodieren](/windows/desktop/Midl/decode) \] auf einen Funktionsprototyp in der ACF-Datei. Siehe auch [*Typserialisierung*](t-glos.md).

</dd> <dt>

<span id="_rpc_protocol_sequence_glos"></span><span id="_RPC_PROTOCOL_SEQUENCE_GLOS"></span>**Protokoll Sequenz**
</dt> <dd>

Zeichenfolge, die eine gültige Kombination aus einem RPC-Protokoll, einem Protokoll der Netzwerkschicht und einem Transportschicht Protokoll darstellt. Beispielsweise beschreibt die Protokoll Sequenz ncacn \_ IP \_ TCP eine [*NCA*](n-glos.md) -Verbindung über ein Internet Protokoll (IP) mit einem TCP (Transmission Control Protocol) als Transport.

</dd> <dt>

<span id="_rpc_proxy_stub_glos"></span><span id="_RPC_PROXY_STUB_GLOS"></span>**ProxyStub**
</dt> <dd>

Mittel l-generierter C-oder C++-Sprache – Quellcode, der alle Funktionen enthält, die für eine benutzerdefinierte OLE-Schnittstelle erforderlich sind.

</dd> </dl>
