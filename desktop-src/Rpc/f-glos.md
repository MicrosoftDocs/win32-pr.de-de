<dl> <dt>

<span id="_rpc_firewall_glos"></span><span id="_RPC_FIREWALL_GLOS"></span>**Firewall**
</dt> <dd>

Steuerungssystem, das verhindert, dass nicht autorisierte Benutzer Zugriff auf ein lokales Netzwerk erhalten, das mit dem Internet verbunden ist.

</dd> <dt>

<span id="_rpc_full_pointer_glos"></span><span id="_RPC_FULL_POINTER_GLOS"></span>**Vollständiger Zeiger**
</dt> <dd>

Zeiger, der über alle Funktionen verfügt, die normalerweise Zeigern in C/C++ zugeordnet sind. Insbesondere kann ein vollständiger Zeiger NULL sein und als Alias für einen anderen Zeigerparameter verwendet werden. Das \[ [ptr-Attribut](/windows/desktop/Midl/ptr) \] kennzeichnet einen vollständigen Zeiger. Siehe auch [*eindeutiger Zeiger*](u-glos.md) und [*Verweiszeiger.*](r-glos.md)

</dd> <dt>

<span id="_rpc_fullsic_glos"></span><span id="_RPC_FULLSIC_GLOS"></span>**fullsic**
</dt> <dd>

Prinzipalname im Vollständigen Format der Antragstellerausstellerkette, wie in RFC1779 definiert.

</dd> <dt>

<span id="_rpc_fully_bound_handle_glos"></span><span id="_RPC_FULLY_BOUND_HANDLE_GLOS"></span>**Vollständig gebundenes Handle**
</dt> <dd>

[*Bindungshandle,*](b-glos.md) das [*Endpunktinformationen*](e-glos.md) enthält. Siehe auch [*teilweise gebundenes Handle.*](p-glos.md)

</dd> </dl>
