# <a name="r-rpc"></a>R (RPC)

<dl> <dt>

<span id="_rpc_rpc_object_glos"></span><span id="_RPC_RPC_OBJECT_GLOS"></span>**RPC-Objekt**
</dt> <dd>

Server Instanzen oder andere Ressourcen, z. b. Geräte, Datenbanken und Warteschlangen, die von RPC-Server Anwendungen betrieben und verwaltet werden. Jedes Objekt wird durch eine oder mehrere Objekt-UUIDs eindeutig identifiziert.

</dd> <dt>

<span id="_rpc_rpcss_glos"></span><span id="_RPC_RPCSS_GLOS"></span>**RPC-Subsystem (RPCSS)**
</dt> <dd>

Windows-Subsystem, das eine Vielzahl von RPC-und OLE-Diensten einschließlich der Endpunkt Zuordnung, des OLE Service Control Manager (SCM) und des com-objektresolvers enthält. Verwechseln Sie dies nicht mit dem RPC-spezifischen Speicher Belegungs Paket RPCSS.

</dd> <dt>

<span id="_rpc_reference_pointer_glos"></span><span id="_RPC_REFERENCE_POINTER_GLOS"></span>**Verweis Zeiger**
</dt> <dd>

Der einfachste Zeigertyp. Ein Verweis Zeiger verweist immer auf einen gültigen Speicher, und der Speicher wird nicht geändert (obwohl sich der Inhalt ändern kann). Ein Verweis Zeiger kann nicht mit einem Alias versehen werden. Das \[ [ref](/windows/desktop/Midl/ref) - \] Attribut bezeichnet einen Verweis Zeiger. Siehe auch [*Unique Pointer*](u-glos.md) und [*Full Pointer*](f-glos.md).

</dd> </dl>
