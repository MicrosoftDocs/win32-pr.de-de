# <a name="r-rpc"></a>R (RPC)

<dl> <dt>

<span id="_rpc_rpc_object_glos"></span><span id="_RPC_RPC_OBJECT_GLOS"></span>**RPC-Objekt**
</dt> <dd>

Serverinstanzen oder andere Ressourcen, z. B. Geräte, Datenbanken und Warteschlangen, die von RPC-Serveranwendungen betrieben und verwaltet werden. Jedes Objekt wird durch eine oder mehrere Objekt-UUIDs eindeutig identifiziert.

</dd> <dt>

<span id="_rpc_rpcss_glos"></span><span id="_RPC_RPCSS_GLOS"></span>**RPC-Subsystem (RPCSS)**
</dt> <dd>

Windows Subsystem, das eine Vielzahl von RPC- und OLE-Diensten umfasst, einschließlich Endpunktzuordnung, OLE Service Control Manager (SCM) und COM-Objektlöser. Verwechseln Sie dies nicht mit dem RPC-spezifischen Speicherzuweisungspaket RpcSs.

</dd> <dt>

<span id="_rpc_reference_pointer_glos"></span><span id="_RPC_REFERENCE_POINTER_GLOS"></span>**Verweiszeiger**
</dt> <dd>

Der einfachste Zeigertyp. Ein Verweiszeiger verweist immer auf gültigen Speicher, und dieser Speicher ändert sich nicht (obwohl sich der Inhalt ändern kann). Ein Verweiszeiger kann nicht mit einem Alias verknüpft werden. Das \[ [ref-Attribut](/windows/desktop/Midl/ref) \] bestimmt einen Verweiszeiger. Siehe auch [*eindeutiger Zeiger*](u-glos.md) und [*vollständiger Zeiger.*](f-glos.md)

</dd> </dl>
