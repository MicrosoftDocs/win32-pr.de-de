# <a name="use-context-handles-for-keeping-state-on-the-server"></a>Verwenden von Kontext Handles zum Beibehalten des Zustands auf dem Server

**Bewährte Methoden:** Verwenden Sie Kontext Handles, wenn der Zustand zwischen Aufrufen auf dem Server beibehalten wird.

Kontext Handles sind häufig für neue RPC-Programmierer einschüchternd, und der Aufwand für Lernkontext Handles wird möglicherweise nicht als Wert für den Aufwand angezeigt. Es lohnt sich, da Kontext Handles über integrierte Lösungen für viele gängige Fehler bei der Netzwerk Programmierung verfügen, die andernfalls von Grund auf neu implementiert werden müssten. Das verstehen von Kontext Handles zahlt später die Dividenden.
