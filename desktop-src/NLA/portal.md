# <a name="network-list-manager"></a>Netzwerklisten-Manager

## <a name="purpose"></a>Zweck

Mit der Network List Manager-API können Anwendungen eine Liste der verfügbaren Netzwerkverbindungen abrufen. Anwendungen können Netzwerke basierend auf Attributen und Signaturen filtern und die Netzwerke auswählen, die für ihre Aufgabe am besten geeignet sind. Die Netzwerklisten-Manager-Infrastruktur benachrichtigt Anwendungen über Änderungen in der Netzwerkumgebung, sodass Anwendungen Netzwerkverbindungen dynamisch aktualisieren können.

## <a name="developer-audience"></a>Entwicklergruppe

Der Netzwerklisten-Manager ist für die Verwendung durch COM-Programmierer konzipiert.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Die Network List Manager-API wird ab Windows Vista unterstützt.
