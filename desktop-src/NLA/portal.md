# <a name="network-list-manager"></a>Netzwerk Listen-Manager

## <a name="purpose"></a>Zweck

Die Netzwerk Listen-Manager-API ermöglicht Anwendungen das Abrufen einer Liste der verfügbaren Netzwerkverbindungen. Anwendungen können Netzwerke basierend auf Attributen und Signaturen Filtern und die Netzwerke auswählen, die für Ihre Aufgabe am besten geeignet sind. Die Infrastruktur des Netzwerk Listen-Managers benachrichtigt Anwendungen über Änderungen in der Netzwerkumgebung und ermöglicht so die dynamische Aktualisierung von Netzwerkverbindungen durch Anwendungen.

## <a name="developer-audience"></a>Entwicklergruppe

Der Netzwerk Listen-Manager ist für die Verwendung durch com-Programmierer konzipiert.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Die Netzwerk Listen-Manager-API wird ab Windows Vista unterstützt.
