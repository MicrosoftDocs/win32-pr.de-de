# <a name="on-demand-connection-routing-helper-api-reference"></a>Referenz zur Hilfs-API für on-Demand-Verbindungsrouting

In den folgenden Themen finden Sie Details zum Hilf hilf hilft für das routing bei Bedarf.



| Referenz                                                                          | Beschreibung                                                                                                                                                                 |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**OnDemandGetRoutingHint**](/windows/desktop/api/OnDemandConnRouteHelper/nf-ondemandconnroutehelper-ondemandgetroutinghint)                           | Suchen Sie nach einem Ziel im Routenanforderungscache, und gibt die entsprechende Schnittstellen-ID zurück, wenn eine Übereinstimmung gefunden wird.<br/>                                               |
| [**OnDemandRegisterNotification**](/windows/desktop/api/OnDemandConnRouteHelper/nf-ondemandconnroutehelper-ondemandregisternotification)               | Registrieren Sie sich, um benachrichtigt zu werden, wenn der Cache für Routenanforderungen geändert wird.<br/>                                                                                               |
| [**OnDemandUnregisterNotification**](/windows/desktop/api/OnDemandConnRouteHelper/nf-ondemandconnroutehelper-ondemandunregisternotification)           | Aufheben der Registrierung für Benachrichtigungen und Bereinigen von Ressourcen.<br/>                                                                                                             |
| [**NET \_ INTERFACE \_ CONTEXT**](/windows/desktop/api/OnDemandConnRouteHelper/ns-ondemandconnroutehelper-net_interface_context)                           | Der Schnittstellenkontext, der Teil der [**NET \_ INTERFACE CONTEXT \_ \_ TABLE-Struktur**](/windows/desktop/api/OnDemandConnRouteHelper/ns-ondemandconnroutehelper-net_interface_context_table) ist.<br/>                                       |
| [**NET \_ INTERFACE \_ CONTEXT \_ TABLE**](/windows/desktop/api/OnDemandConnRouteHelper/ns-ondemandconnroutehelper-net_interface_context_table)              | Die Tabelle der [**NET \_ INTERFACE \_ CONTEXT-Strukturen.**](/windows/desktop/api/OnDemandConnRouteHelper/ns-ondemandconnroutehelper-net_interface_context)<br/>                                                                                |
| [**GetInterfaceContextTableForHostName**](/windows/desktop/api/OnDemandConnRouteHelper/nf-ondemandconnroutehelper-getinterfacecontexttableforhostname) | Diese Funktion ruft eine Schnittstellenkontexttabelle für den angegebenen Hostnamen und Verbindungsprofilfilter ab.<br/>                                                         |
| [**FreeInterfaceContextTable**](/windows/desktop/api/OnDemandConnRouteHelper/nf-ondemandconnroutehelper-freeinterfacecontexttable)                     | Diese Funktion gibt die Mithilfe der [**GetInterfaceContextTableForHostName-Funktion**](/windows/desktop/api/OnDemandConnRouteHelper/nf-ondemandconnroutehelper-getinterfacecontexttableforhostname) abgerufene Schnittstellenkontexttabelle frei.<br/> |
