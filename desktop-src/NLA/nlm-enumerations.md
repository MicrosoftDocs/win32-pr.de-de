# <a name="nlm-enumerations"></a>NLM-Enumerationen

Im folgenden finden Sie die NLM-Enumerationen.

| Enumeration                                                                 | Beschreibung                                                                                |
|-----------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**NLM- \_ Verbindungs \_ Kosten**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_cost)                        | Gibt eine Gruppe von Kosten Stufen und Kosten Flags an, die in Windows 8-Kosten-APIs unterstützt werden.            |
| [**Änderung der NLM- \_ Verbindungs \_ Eigenschaft \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_property_change) | Definiert Änderungen, die an den Eigenschaften einer Netzwerkverbindung vorgenommen wurden.                            |
| [**NLM- \_ Konnektivität**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connectivity)                               | Stellt eine Benachrichtigung bereit, wenn verbindungsbezogene Parameter geändert wurden.               |
| [**NLM \_ - \_ Domänentyp**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_domain_type)                                | Gibt den Domänentyp eines Netzwerks an.                                                    |
| [**NLM- \_ Enum- \_ Netzwerk**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_enum_network)                              | Gibt an, welche Typen von Netzwerken aufgelistet werden.                                           |
| [**NLM- \_ Internet \_ Konnektivität**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_internet_connectivity)            | Definiert einen Satz von Flags, die zusätzliche Daten für die IPv4-oder IPv6-Netzwerk Konnektivität bereitstellen. |
| [**NLM- \_ Netzwerk \_ Klasse**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_class)                            | Definiert einen Satz von Flags, die angeben, ob ein Netzwerk identifiziert wurde.                      |
| [**\_Netzwerk Kategorie NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_category)                      | Gibt den Kategorietyp eines Netzwerks an.                                                  |
| [**NLM- \_ Netzwerk \_ Eigenschafts \_ Änderung**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_property_change)       | Definiert Änderungen, die an den Eigenschaften eines Netzwerks vorgenommen werden.                                       |
