# <a name="nlm-enumerations"></a>NLM-Enumerationen

Im Folgenden sind die NLM-Enumerationen aufgeführt.

| Enumeration                                                                 | Beschreibung                                                                                |
|-----------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**\_NLM-VERBINDUNGSKOSTEN \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_cost)                        | Gibt eine Reihe von Kostenebenen und Kostenflags an, die in Windows 8 Cost-APIs unterstützt werden.            |
| [**ÄNDERUNG DER \_ NLM-VERBINDUNGSEIGENSCHAFT \_ \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_property_change) | Definiert Änderungen an den Eigenschaften einer Netzwerkverbindung.                            |
| [**\_NLM-KONNEKTIVITÄT**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connectivity)                               | Stellt Benachrichtigungen bereit, wenn sich konnektivitätsbezogene Parameter geändert haben.               |
| [**\_NLM-DOMÄNENTYP \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_domain_type)                                | Gibt den Domänentyp eines Netzwerks an.                                                    |
| [**NLM \_ ENUM \_ NETWORK**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_enum_network)                              | Gibt an, welche Netzwerktypen aufzählt werden.                                           |
| [**\_ \_ NLM-INTERNETKONNEKTIVITÄT**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_internet_connectivity)            | Definiert einen Satz von Flags, die zusätzliche Daten für die IPv4- oder IPv6-Netzwerkkonnektivität bereitstellen. |
| [**\_NLM-NETZWERKKLASSE \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_class)                            | Definiert einen Satz von Flags, die angeben, ob ein Netzwerk identifiziert wurde.                      |
| [**\_NLM-NETZWERKKATEGORIE \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_category)                      | Gibt den Kategorietyp eines Netzwerks an.                                                  |
| [**ÄNDERUNG DER \_ NLM-NETZWERKEIGENSCHAFT \_ \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_property_change)       | Definiert Änderungen an den Eigenschaften eines Netzwerks.                                       |
