# <a name="nlm-structures"></a>NLM-Strukturen

Die Netzwerk Listen-Manager-API wird von den folgenden Strukturen unterstützt:



| Struktur                                                           | BESCHREIBUNG                                                          |
|---------------------------------------------------------------------|----------------------------------------------------------------------|
| [**Informationen zu \_ simulierten NLM- \_ Profilen \_**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_simulated_profile_info) | Stellt Profilinformationen dar, die während der Kosten Simulation verwendet werden sollen.    |
| [**NLM \_ sockaddr**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_sockaddr)                               | Stellt die IPv4/IPv6-Zieladresse dar.                        |
| [**NLM- \_ Verwendungs \_ Daten**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_usage_data)                          | Stellt Informationen dar, die die Datennutzung eines Daten Plans angeben. |
| [**NLM- \_ Daten \_ Plan \_ Status**](https://www.bing.com/search?q=**NLM\_DATA\_PLAN\_STATUS**)             | Stellt den aktuellen Status eines Daten Plans dar.                        |
