# <a name="macros"></a>Makros

Die Themen in diesem Abschnitt enthalten die Referenz Spezifikationen für [Zeiger Eingabe Meldungen und Benachrichtigungs](messages-and-notifications-portal.md) Makros zum Abrufen von Informationen aus Zeiger Eingabe-Nachrichten Parametern.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                  | BESCHREIBUNG                                                                                                                         |
|----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**GET_POINTERID_WPARAM**](/previous-versions/windows/desktop/api)<br/>                      | Ruft die Zeiger-ID unter Verwendung des angegebenen-Werts ab. <br/>                                                                     |
| [**HAS_POINTER_CONFIDENCE_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Überprüft, ob die angegebene Zeiger Nachricht als absichtlich und nicht als versehentlich erachtet wird.<br/>                           |
| [**IS_POINTER_CANCELED_WPARAM**](/previous-versions/windows/desktop/api)<br/>         | Überprüft, ob die angegebene Zeiger Eingabe abrupt beendet wurde oder ungültig war, was darauf hinweist, dass die Interaktion nicht abgeschlossen wurde.<br/> |
| [**IS_POINTER_FIFTHBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Überprüft, ob der angegebene Zeiger eine fünfte Aktion ausgeführt hat. <br/>                                                                 |
| [**IS_POINTER_FIRSTBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Überprüft, ob der angegebene Zeiger zuerst eine Aktion ausgeführt hat.<br/>                                                                  |
| [**IS_POINTER_FLAG_SET_WPARAM**](/previous-versions/windows/desktop/api)<br/>        | Überprüft, ob ein Zeiger Makro das angegebene Flag festlegt. <br/>                                                                 |
| [**IS_POINTER_FOURTHBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/> | Überprüft, ob der angegebene Zeiger die vierte Aktion ausgeführt hat. <br/>                                                                |
| [**IS_POINTER_INCONTACT_WPARAM**](/previous-versions/windows/desktop/api)<br/>       | Überprüft, ob der angegebene Zeiger in Kontakt ist. <br/>                                                                     |
| [**IS_POINTER_INRANGE_WPARAM**](/previous-versions/windows/desktop/api)<br/>           | Überprüft, ob sich der angegebene Zeiger im Bereich befindet. <br/>                                                                       |
| [**IS_POINTER_NEW_WPARAM**](/previous-versions/windows/desktop/api)<br/>                   | Überprüft, ob der angegebene Zeiger ein neuer Zeiger ist. <br/>                                                                  |
| [**IS_POINTER_PRIMARY_WPARAM**](/previous-versions/windows/desktop/api)<br/>           | Überprüft, ob der angegebene Zeiger der primäre Zeiger ist. <br/>                                                            |
| [**IS_POINTER_SECONDBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/> | Überprüft, ob der angegebene Zeiger eine zweite Aktion ausgeführt hat. <br/>                                                                |
| [**IS_POINTER_THIRDBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Überprüft, ob der angegebene Zeiger eine dritte Aktion ausgeführt hat. <br/>                                                                 |
