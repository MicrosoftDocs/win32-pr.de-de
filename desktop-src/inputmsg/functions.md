# <a name="pointer-input-messages-and-notifications-functions"></a>Zeigereingabemeldungen und Benachrichtigungsfunktionen

Die Themen in diesem Abschnitt enthalten die Referenzspezifikationen für [Zeigereingabenachrichten und Benachrichtigungsfunktionen.](messages-and-notifications-portal.md)

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                 | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**EnableMouseInPointer**](/previous-versions/windows/desktop/api)<br/>                       | Ermöglicht es der Maus, als Zeigereingabegerät zu fungieren und [WM_POINTER](messages.md) senden.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**GetPointerCursorId**](/previous-versions/windows/desktop/api)<br/>                           | Ruft den Cursorbezeichner ab, der dem angegebenen Zeiger zugeordnet ist.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**GetPointerFrameInfo**](/previous-versions/windows/desktop/api)<br/>                         | Ruft den gesamten Informationsrahmen für die angegebenen Zeiger ab, die der aktuellen Meldung zugeordnet sind. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**GetPointerFrameInfoHistory**](/previous-versions/windows/desktop/api)<br/>           | Ruft den gesamten Informationsrahmen (einschließlich zusammen 1000 Eingabeframes) für die angegebenen Zeiger ab, die der aktuellen Meldung zugeordnet sind. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**GetPointerFramePenInfo**](/previous-versions/windows/desktop/api)<br/>                   | Ruft den gesamten Rahmen stiftbasierter Informationen für die angegebenen Zeiger (vom Typ PT_PEN [**)**](/previous-versions/windows/desktop/api)ab, die der aktuellen Meldung zugeordnet sind. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**GetPointerFramePenInfoHistory**](/previous-versions/windows/desktop/api)<br/>     | Ruft den gesamten Rahmen stiftbasierter Informationen (einschließlich zusammengeketteter Eingaberahmen) für die angegebenen Zeiger (vom Typ [**PT_PEN**](/previous-versions/windows/desktop/api)) ab, die der aktuellen Meldung zugeordnet sind. <br/>                                                                                                                                                                                                                                                                                                                                                                                        |
| [**GetPointerFrameTouchInfo**](/previous-versions/windows/desktop/api)<br/>               | Ruft den gesamten Frame der touchbasierten Informationen für die angegebenen Zeiger (vom Typ PT_TOUCH [**)**](/previous-versions/windows/desktop/api)ab, die der aktuellen Meldung zugeordnet sind. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**GetPointerFrameTouchInfoHistory**](/previous-versions/windows/desktop/api)<br/> | Ruft den gesamten Frame touchbasierter Informationen (einschließlich zusammengekniffener Eingaberahmen) für die angegebenen Zeiger (vom Typ [**PT_TOUCH**](/previous-versions/windows/desktop/api)) ab, die der aktuellen Meldung zugeordnet sind. <br/>                                                                                                                                                                                                                                                                                                                                                                                    |
| [**GetPointerInfo**](/previous-versions/windows/desktop/api)<br/>                                   | Ruft die Informationen für den angegebenen Zeiger ab, der der aktuellen Meldung zugeordnet ist.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**GetPointerInfoHistory**](/previous-versions/windows/desktop/api)<br/>                     | Ruft die Informationen ab, die den einzelnen Eingaben zugeordnet sind, sofern diese in die aktuelle Meldung für den angegebenen Zeiger zusammenfingen. Die neueste Eingabe ist im zurückgegebenen Verlauf enthalten und identisch mit der letzten Eingabe, die von der [**GetPointerInfo-Funktion zurückgegeben**](/previous-versions/windows/desktop/api) wird.<br/>                                                                                                                                                                                                                                                                              |
| [**GetPointerInputTransform**](/previous-versions/windows/desktop/api)<br/>               | Ruft eine oder mehrere Transformationen für die Zeigerinformationskoordinaten ab, die der aktuellen Meldung zugeordnet sind. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**GetPointerPenInfo**](/previous-versions/windows/desktop/api)<br/>                             | Ruft die stiftbasierten Informationen für den angegebenen Zeiger (vom Typ PT_PEN [**)**](/previous-versions/windows/desktop/api)ab, der der aktuellen Meldung zugeordnet ist. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**GetPointerPenInfoHistory**](/previous-versions/windows/desktop/api)<br/>               | Ruft die stiftbasierten Informationen ab, die den einzelnen Eingaben zugeordnet sind, sofern diese in die aktuelle Meldung für den angegebenen Zeiger (vom Typ [**PT_PEN) wurden.**](/previous-versions/windows/desktop/api) Die neueste Eingabe ist im zurückgegebenen Verlauf enthalten und identisch mit der letzten Eingabe, die von der [**GetPointerPenInfo-Funktion zurückgegeben**](/previous-versions/windows/desktop/api) wird.<br/>                                                                                                                                                                                                         |
| [**GetPointerTouchInfo**](/previous-versions/windows/desktop/api)<br/>                         | Ruft die touchbasierten Informationen für den angegebenen Zeiger (vom Typ PT_TOUCH [**)**](/previous-versions/windows/desktop/api)ab, der der aktuellen Meldung zugeordnet ist. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**GetPointerTouchInfoHistory**](/previous-versions/windows/desktop/api)<br/>           | Ruft die touchbasierten Informationen ab, die den einzelnen Eingaben zugeordnet sind , die in der aktuellen Nachricht für den angegebenen Zeiger (vom Typ [**PT_TOUCH) zusammengefasst wurden.**](/previous-versions/windows/desktop/api) Die neueste Eingabe ist im zurückgegebenen Verlauf enthalten und identisch mit der letzten Eingabe, die von der [**GetPointerTouchInfo-Funktion zurückgegeben**](/previous-versions/windows/desktop/api) wird.<br/>                                                                                                                                                                                                 |
| [**GetPointerType**](/previous-versions/windows/desktop/api)<br/>                                   | Ruft den Zeigertyp für einen angegebenen Zeiger ab.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**GetUnpredictedMessagePos**](/previous-versions/windows/desktop/api)<br/>               | Ruft Zeigerdaten ab, bevor sie die Touchvorhersageverarbeitung durchgegangen sind.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**IsMouseInPointerEnabled**](/previous-versions/windows/desktop/api)<br/>                 | Gibt [**an, ob EnableMouseInPointer**](/previous-versions/windows/desktop/api) so festgelegt ist, dass die Maus als Zeigereingabegerät verwendet und WM_POINTER [sendet.](messages.md)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                    |
| [**SkipPointerFrameMessages**](/previous-versions/windows/desktop/api)<br/>               | Bestimmt, welcher Zeigereingabeframe die zuletzt abgerufene Nachricht für den angegebenen Zeiger generiert hat, und verwirft alle in der Warteschlange (nicht wiederholten) Zeigereingabemeldungen, die aus demselben Zeigereingabeframe generiert wurden. Wenn eine Anwendung Informationen für einen gesamten Frame mithilfe der [**GetPointerFrameInfo-Funktion,**](/previous-versions/windows/desktop/api) der [**GetPointerFrameInfoHistory-Funktion**](/previous-versions/windows/desktop/api) oder einer ihrer typspezifischen Varianten abgerufen hat, kann sie diese Funktion verwenden, um das Abrufen und Verwerfen verbleibender Nachrichten aus diesem Frame nach und nach zu vermeiden.<br/> |
