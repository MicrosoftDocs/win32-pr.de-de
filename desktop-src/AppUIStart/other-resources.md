<dl> <dt>

<span id="_Windows_Interface_Guidelines_for_Software_Design"></span><span id="_windows_interface_guidelines_for_software_design"></span><span id="_WINDOWS_INTERFACE_GUIDELINES_FOR_SOFTWARE_DESIGN"></span>*Windows Interface Guidelines for Software Design*
</dt> <dd>

(Microsoft Press, 1995; ISBN 1-55615-679-0)

In diesem Buch werden alle zu befolgenden wichtigen Benutzeroberflächenregeln erläutert. Es gibt zahlreiche Beispiele für das Erstellen von Dialogfeldern, Assistenten, Symbolleisten und Kontextmenüs.

<span id="Usability_Engineering_by_Jakob_Nielsen_"></span><span id="usability_engineering_by_jakob_nielsen_"></span><span id="USABILITY_ENGINEERING_BY_JAKOB_NIELSEN_"></span>*Usability Engineering* von Usability Nielsen 
</dt> <dd>

(AP Professional, 1994; ISBN 0125184069)

Dieses Buch konzentriert sich auf die Techniken zum effektiven Messen der Nutzbarkeit einer Schnittstelle. Es werden viele verschiedene Testtechniken für den Vergleich verschiedener Entwurfsvorschläge erläutert, einschließlich Vorschläge, wie die Kosten für Benutzerfreundlichkeitstests reduziert werden können. In diesem Buch werden Analysetechniken behandelt, die auf Websites, Webseiten oder jede Art von Software angewendet werden können. Sie ist ideal für die ersten Phasen des Lernens über Benutzer und die mittleren Phasen von Iterations- und Benutzerfreundlichkeitstests.

</dd> <dt>

<span id="Design_of_Everyday_Things_by_Don_Norman"></span><span id="design_of_everyday_things_by_don_norman"></span><span id="DESIGN_OF_EVERYDAY_THINGS_BY_DON_NORMAN"></span>*Design of Everyday Things (Entwurf von alltäglichen* Dingen) von Don Soll
</dt> <dd>

(Currency/Doubleday, 1988; ISBN: 0385267746)

In diesem Buch werden die Prinzipien der Arbeit erläutert, die die Verwendung von Schnittstellen so schwierig machen. Empfohlen für alle Benutzer, die an Benutzeroberflächen interessiert sind und ein umfassendes Verständnis der damit verbundenen Probleme haben möchten. Beispiele hierfür sind Geräte, Bürotüren und Verkaufsautomaten.
