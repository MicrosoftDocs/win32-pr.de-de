# <a name="windows-error-reporting"></a>Windows-Fehlerberichterstattung

Die Fehler Berichterstattungs Funktion ermöglicht es Benutzern, Microsoft über Anwendungsfehler, Kernel Fehler, nicht reagierende Anwendungen und andere anwendungsspezifische Probleme zu benachrichtigen. Microsoft kann die Fehler Berichterstattungs Funktion verwenden, um Kundeninformationen zur Problembehandlung, Lösungen oder Updates für Ihre spezifischen Probleme bereitzustellen. Entwickler können diese Infrastruktur verwenden, um Informationen zu erhalten, die zur Verbesserung ihrer Anwendungen verwendet werden können.

Benutzer können die Fehlerberichterstattung über die Windows-Benutzeroberfläche aktivieren. Sie können auswählen, dass Fehler für bestimmte Anwendungen gemeldet werden. Administratoren können diese Einstellungen mithilfe von Gruppenrichtlinie außer Kraft setzen.

Entwickler können sich beim [Windows-Desktop Anwendungsprogramm](../appxpkg/windows-desktop-application-program.md) registrieren, um Informationen zu den Problemen zu erhalten, die Kunden mit Ihren Anwendungen haben, und Kunden dabei zu helfen, diese Probleme zu beheben Entwickler können auch die [Anwendungs Wiederherstellung und den Neustart](/windows/desktop/Recovery/application-recovery-and-restart-portal) verwenden, um sicherzustellen, dass die Kunden keine Daten verlieren, wenn die Anwendung abstürzt und Benutzern ermöglicht, schnell zu ihren Aufgaben zurückzukehren.
