# <a name="mailslot-functions"></a>Mailslot-Funktionen

Die folgenden Funktionen können mit mailslots verwendet werden.



| Funktion                                   | Beschreibung                                                                  |
|--------------------------------------------|------------------------------------------------------------------------------|
| [**CreateMailslot**](/windows/desktop/api/Winbase/nf-winbase-createmailslota)   | Erstellt einen Mailslot mit dem angegebenen Namen.                                  |
| [**GetMailslotInfo**](/windows/desktop/api/Winbase/nf-winbase-getmailslotinfo) | Ruft Informationen zum angegebenen Maillot ab.                          |
| [**SetMailslotInfo**](/windows/desktop/api/Winbase/nf-winbase-setmailslotinfo) | Legt den Time out-Wert fest, der vom angegebenen Maillot für einen Lesevorgang verwendet wird. |
