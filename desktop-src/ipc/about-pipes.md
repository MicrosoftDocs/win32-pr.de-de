# <a name="about-pipes"></a>Informationen zu Pipes

Es gibt zwei Arten von Pipes: [Anonyme Pipes](anonymous-pipes.md) und [Named Pipes](named-pipes.md). Anonyme Pipes erfordern weniger Aufwand als Named Pipes, bieten jedoch eingeschränkte Dienste.

Der in diesem Beispiel verwendete Begriff Pipe impliziert, dass eine Pipe als Informationskanal verwendet wird. Konzeptionell weist eine Pipe zwei Enden auf. Eine unidirektionale Pipe ermöglicht dem Prozess an einem Ende das Schreiben in die Pipe und ermöglicht das Lesen des Prozesses am anderen Ende aus der Pipe. Eine bidirektionale (oder Duplex-) Pipe ermöglicht es einem Prozess, von seinem Ende der Pipe zu lesen und zu schreiben.
