# <a name="pipes-interprocess-communications"></a>Pipes (prozessübergreifende Kommunikation)

Eine *Pipe* ist ein Abschnitt mit gemeinsam genutzter Speicher, der von Prozessen für die Kommunikation verwendet wird. Der Prozess, durch den eine Pipe erstellt wird, ist der *Pipeserver*. Ein Prozess, der eine Verbindung mit einer Pipe herstellt, ist ein *Pipe-Client*. Ein Prozess schreibt Informationen in die Pipe, während der andere Prozess die Informationen aus der Pipe liest. In dieser Übersicht wird beschrieben, wie Pipes erstellt, verwaltet und verwendet werden.
