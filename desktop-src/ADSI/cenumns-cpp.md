# <a name="cenumnscpp"></a>Cenum NS. CPP

In der Beispiel Anbieter Komponente verwendet die-Enumeration eines Namespace-Objekts die-Methoden von cenumns. cpp, die in der folgenden Tabelle aufgeführt sind.



| Methode                                              | BESCHREIBUNG                                                                                                                                               |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Csampledsnamespaceenum:: Create**                  | Erstellen Sie ein-Objekt, um die Enumeration eines ADS-Namespace Objekts zuzulassen.                                                                                         |
| **Csampledsnamespaceenum:: csampledsnamespaceenum**  | Standardkonstruktor.                                                                                                                                     |
| **Csampledsnamespaceenum:: ~ csampledsnamespaceenum** | Standardedekonstruktor.                                                                                                                                      |
| **Csampledsnamespaceenum:: Next**                    | Ruft die angegebene Anzahl von Elementen aus dem angegebenen Namespace Objekt ab.                                                                            |
| **Csampledsnamespaceenum:: enumubjects**             | Verwaltet das Abrufen der Schnittstellen Zeiger auf die-Objekte.                                                                                                  |
| **Csampledsnamespaceenum:: fetchobjects**            | Rufen Sie den Satz der [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) -Zeiger ab.                                                                          |
| **Csampledsnamespaceenum:: fetchnextobject**         | Rufen Sie das nächste-Objekt ab. Wenn Sie gefunden wird, erstellen Sie ein generisches Active Directory Objekt, und rufen Sie dessen [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) -Zeiger ab |
