# <a name="benefits-of-using-active-directory-service-interfaces"></a>Vorteile der Verwendung von Active Directory Dienst Schnittstellen

ADSI bietet Systemadministratoren viele Vorteile, wie in der folgenden Tabelle beschrieben.



| Funktion                   | Vorteil                                                                                                                                                                                                                                                                                                                             |
|---------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Öffnen                      | Ein beliebiger Verzeichnis Anbieter kann einen ADSI-Anbieter implementieren. Benutzer profitieren von der Wahl in Verzeichnisdiensten, ohne dass Sie die Verwaltbarkeit beeinträchtigen.                                                                                                                                                                                        |
| Unabhängige DS            | Verwaltungs Anwendungen sind nicht eng an den Verzeichnisdienst eines bestimmten Anbieters gebunden. Administrative und andere Verzeichnis aktivierte Anwendungen können entwickelt werden, ohne dass Sie anbieterspezifische Verzeichnis-APIs kennen müssen. Dieselbe Anwendung kann in mehreren Verzeichnissen verwendet werden. Die Entwicklungszeit und die Supportkosten werden verringert. |
| Unterstützung mehrerer Sprachen | ADSI-Objekte bieten einfachen Zugriff auf Verzeichnisdienste durch das Komponentenobjektmodell. COM-Anwendungen können in Sprachen wie Microsoft Visual Basic, Microsoft Visual Basic Scripting Edition (VBScript) und C/C++ geschrieben werden.                                                                                             |
| Einfaches Programmiermodell  | ADSI besteht aus einem kleinen und leicht zu erlernenden Satz von Schnittstellen.                                                                                                                                                                                                                                                                          |
| Skript fähige                | Alle Automatisierungs kompatiblen Sprachen (z. b. Visual Basic, VBScript, Perl, REXX usw.) können verwendet werden, um Verzeichnisdienst Anwendungen zu entwickeln. Administratoren und Entwickler können die Tools verwenden, die Sie bereits kennen. Die Produktivität wird verbessert – Entwicklungszeit und Supportkosten werden verringert.                               |
| Funktionale reichhaltige         | ISVs und ausgereifte Endbenutzer können mit den gleichen ADSI-Modellen, die für einfache Skript gesteuerte administrative Anwendungen verwendet werden, ernsthafte Anwendungen entwickeln.                                                                                                                                                                         |
| Dieser Ansatz ist erweiterbar.                | Verzeichnis Anbieter, ISVs und Endbenutzer können ADSI durch neue Objekte und Funktionen erweitern, um Werte hinzuzufügen oder besondere Anforderungen zu erfüllen.                                                                                                                                                                                                          |
| OLE DB              | ADSI stellt eine OLE DB Schnittstelle bereit, sodass Programmierer, die mit der Datenbankprogrammierung durch OLE DB vertraut sind, schnell produktiv arbeiten können.                                                                                                                                                                                                  |
