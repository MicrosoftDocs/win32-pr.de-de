# <a name="generic-com-error-codes"></a>Generische COM-Fehlercodes

Generische COM-Fehlercodes geben den Vorgangsstatus an, der von allen COM-Modulen erzeugt wird. Die folgende Tabelle enthält einige Beispiele für solche Fehlercodes auf win32-Plattformen.



| Fehlercode     | Hexadezimalwert  | BESCHREIBUNG              |
|----------------|------------|--------------------------|
| E \_ ABORT       | 0x80004004 | Der Vorgang wurde abgebrochen.       |
| E \_ FAIL        | 0x80004005 | Unbekannter Fehler.       |
| E \_ NOINTERFACE | 0x80004002 | Schnittstelle wird nicht unterstützt. |
| E \_ NOTIMPL     | 0x80004001 | Nicht implementiert.         |
| \_E-ZEIGER     | 0x80004003 | Ungültiger Zeiger.         |
| E \_ UNEXPECTED  | 0x8000FFFF | Schwerwiegender Fehler.    |
