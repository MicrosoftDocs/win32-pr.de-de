# <a name="adsi-extensions"></a>ADSI-Erweiterungen

Eine ADSI-Erweiterung ist ein spezielles COM-Objekt, das es einem Entwickler ermöglicht, ein ADSI-Objekt zu erweitern. Jede Erweiterung ist einer angegebenen Klasse im Verzeichnis zugeordnet. Mit diesem Erweiterungsmodell können Entwickler Methoden hinzufügen, um einem Objekt eine dynamischere Bedeutung zu geben. In einem herkömmlichen Verzeichnisprogrammiermodell wird auf ein Objekt zugegriffen, indem die Objektattribute abgerufen und festgelegt werden. Mit ADSI-Erweiterungen können Sie einem Verzeichnisobjekt weitere Features hinzufügen.
