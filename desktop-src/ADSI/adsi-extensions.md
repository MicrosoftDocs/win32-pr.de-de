# <a name="adsi-extensions"></a>ADSI-Erweiterungen

Eine ADSI-Erweiterung ist ein spezielles com-Objekt, mit dem ein Entwickler ein ADSI-Objekt erweitern kann. Jede Erweiterung ist einer angegebenen Klasse im Verzeichnis zugeordnet. Mit diesem Erweiterungs Modell können Entwickler Methoden hinzufügen, um einem Objekt eine dynamischere Bedeutung zu verleihen. In einem herkömmlichen Verzeichnis Programmiermodell erfolgt der Zugriff auf ein Objekt, indem die Objekt Attribute abgerufen und festgelegt werden. Mithilfe von ADSI Extensions können Sie einem Verzeichnis Objekt weitere Funktionen hinzufügen.
