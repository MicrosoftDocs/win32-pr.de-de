Weitere Informationen zum Einrichten einer Entwicklungsumgebung und zum Erstellen von grundlegenden ADSI-Anwendungen finden Sie unter:

-   [Einstieg in ASP für ADSI](getting-started-with-asp-for-adsi.md)
-   [Einrichten von Visual Basic 6,0 für die ADSI-Entwicklung](setting-up-visual-basic-for-adsi-development.md)
-   [Einrichten von Visual C++ 6,0 für die ADSI-Entwicklung](setting-up-c---for-adsi-development.md)
