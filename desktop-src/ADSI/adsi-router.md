# <a name="adsi-router"></a>ADSI-Router

Der ADSI-Router befindet sich zwischen einem Client und den ADSI-Anbietern. Es implementiert einen Kernsatz von Objekten, der grundlegende Features und Dienste bietet, die allen Anbietern gemeinsam sind. Daher wird jede Funktion, die im Router unterstützt wird, auch von allen Anbietern unterstützt, es sei denn, der Anbieter wählt andernfalls aus. Das ADSI-Routermodul entspricht Activeds.dll.
