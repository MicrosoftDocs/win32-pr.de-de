# <a name="scenario-the-fabrikam-corporation"></a>Szenario: fabrikam Corporation

Das Fabrikam-Unternehmen hat seine Domäne von Windows NT 4,0 auf Windows 2000 aktualisiert. Bei der Installation wurde Joe von, dem Netzwerkadministrator, aufgefordert, einen DNS-Domänen Namen anzugeben. Joe hat "fabrikam.com" eingegeben. Die vorhandenen Benutzer, Gruppen und Computer werden zu Active Directory in die neue Domäne migriert: fabrikam.com.
