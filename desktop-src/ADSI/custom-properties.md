# <a name="custom-properties"></a>Benutzerdefinierte Eigenschaften

Anbieter können zusätzliche Objekte, Schnittstellen und Eigenschaften verfügbar machen, die nicht durch das ADSI-Schema definiert sind.

Weitere Informationen zu Implementierungsrichtlinien für Anbieter finden Sie im Dokument [ADSI-Beispielanbieterkomponente.](adsi-example-provider-component.md)
