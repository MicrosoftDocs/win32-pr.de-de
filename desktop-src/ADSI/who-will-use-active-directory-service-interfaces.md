# <a name="who-will-use-active-directory-service-interfaces"></a>Wer wird Active Directory Dienst Schnittstellen verwenden?

Netzwerkadministratoren verwenden ADSI, um allgemeine administrative Aufgaben zu automatisieren, z. b. das Hinzufügen von Benutzern und Gruppen, das Verwalten von Druckern und das Festlegen von Berechtigungen für Netzwerkressourcen.

Unabhängige Softwareanbieter (ISVs) und Endbenutzer Entwickler verwenden ADSI in "Verzeichnis Aktivierung" für Ihre Produkte und Anwendungen. Dienste können sich selbst in einem Verzeichnis veröffentlichen, Clients können das Verzeichnis zum Suchen der Dienste verwenden, und beide können das Verzeichnis verwenden, um andere Objekte zu suchen und zu bearbeiten, die von Interesse sind. Da Active Directory Dienst Schnittstellen unabhängig von den zugrunde liegenden Verzeichnisdiensten sind, werden die Verzeichnis fähigen Produkte und Anwendungen in mehreren Netzwerk-und Verzeichnis Umgebungen erfolgreich ausgeführt.
