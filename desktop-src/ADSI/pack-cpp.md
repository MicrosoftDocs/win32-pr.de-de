# <a name="packcpp"></a>Ru. CPP

In der Beispiel Anbieter Komponente finden Sie ein Codebeispiel für das Packen und entpacken von Datentypen in Varianten in "Pack. cpp". Die folgenden Funktionen werden verwendet, wenn Eigenschaften in den Eigenschafts Cache geladen und daraus entfernt werden:

-   **Packstringinvariant**
-   **Unpackstringinvariant**
-   **Packlonginvariant**
-   **Unpacklongfromvariant**
-   **Packdateinvariant**
-   **Unpackdateinvariant**
-   **Packvariant- \_ boolinvariant**
-   **Nicht packvariant \_ boolfromvariant**
-   **Packvariantinvariant**
-   **Unpackvariantfromvariant**
