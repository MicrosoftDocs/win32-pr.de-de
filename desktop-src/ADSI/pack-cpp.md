# <a name="packcpp"></a>Pack. Cpp

In der Beispielanbieterkomponente finden Sie in pack.cpp ein Codebeispiel für das Packen und Entpacken von Datentypen in VTARTS. Die folgenden Funktionen werden verwendet, wenn Eigenschaften in den Eigenschaftencache geladen und daraus entfernt werden:

-   **PackStringinVariant**
-   **UnpackStringinVariant**
-   **PackLONGinVariant**
-   **UnpackLONGfromVariant**
-   **PackDATEinVariant**
-   **UnpackDATEinVariant**
-   **PackVARIANT \_ BOOLinVariant**
-   **UnpackVARIANT \_ BOOLfromVariant**
-   **PackVARIANTinVariant**
-   **UnpackVARIANTfromVariant**
