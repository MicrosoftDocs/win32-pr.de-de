# <a name="cprpobjcpp"></a>CPRPOBJ. Cpp

In der Beispielanbieterkomponente sind Eigenschaftenobjektmethoden in cprpobj.cpp in der folgenden Tabelle aufgeführt.



| Methode                                                 | Beschreibung                                                                                                                                    |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| **CSampleDSProperty::CSampleDSProperty**               | Standardkonstruktor.                                                                                                                          |
| **CSampleDSProperty::~CSampleDSProperty**              | Standard-Destruktor.                                                                                                                           |
| **CSampleDSProperty::CreateProperty**                  | Erstellen Sie ein ADS-Eigenschaftsobjekt, und suchen Sie nach den Attributdefinitionen, indem **Sie SampleDSGetPropertyDefinition** aufrufen.                              |
| **CSampleDSProperty::CreateProperty**                  | Erstellen Sie bei der Attributdefinition ein Eigenschaftenobjekt, und legen Sie die Entsprechung zwischen unterstützten ADS-Syntaxen und Anbietersyntaxen fest. |
| **CSampleDSProperty::AllocatePropertyObject**          | Erstellen Sie ein Eigenschaftenobjekt, und laden Sie dessen Typdaten.                                                                                               |
| **CSampleDSProperty::QueryInterface**                  | Gibt ggf. den angeforderten Schnittstellenzeiger zurück.                                                                                          |
| [**Standard-IADs-Methoden.**](/windows/desktop/api/Iads/nn-iads-iads)                 |                                                                                                                                                |
| Standardmäßige [**IADsProperty-Methoden.**](/windows/desktop/api/Iads/nn-iads-iadsproperty) |                                                                                                                                                |
| **MapSyntaxIdtoADsSyntax**                             | Legen Sie die Entsprechung zwischen der Syntax-ID und der ADS-Syntax fest.                                                                               |
| **MapSyntaxIdtoSampleDSSyntax**                        | Legen Sie die Entsprechung zwischen der Syntax-ID und der Anbietersyntax fest.                                                                          |
