# <a name="adsi-utility-interfaces"></a>ADSI Utility-Schnittstellen

ADSI bietet verschiedene hilfsschnittstellen für ADSI-Objekte, die beim Zugriff und der Verwaltung der Active Directory helfen können. Diese Schnittstellen lauten wie folgt:

-   [**Iadsdeleteops**](iadsdeleteops-interface.md)
-   [**IADsObjectOptions**](iadsobjectoptions-interface.md)
-   [**IADsPathname**](iadspathname-interface.md)
-   [**IADsNameTranslate**](iadsnametranslate-interface.md)
