# <a name="binding-to-an-adsi-object"></a>Binden an ein ADSI-Objekt

Das Herstellen einer Verbindung mit einem Objekt in einem Verzeichnisdienst wird als Bindung bezeichnet. Die Bindung an ein ADSI-Objekt ist der erste Schritt bei der Kommunikation mit dem zugrunde liegenden Verzeichnissystem. Ein Objekt muss an gebunden werden, um in seinem Namespace zu navigieren, nach Daten zu suchen, Daten zu ändern oder die Identität eines Benutzers zu annehmen. Es ist auch möglich, zusätzliche Bindungsmerkmale wie Benutzername, Kennwort, Servername, Verschlüsselungsmethoden und sichere Authentifizierung anzugeben. Welche zusätzlichen Bindungsmerkmale tatsächlich verwendet werden können, hängt vom Anbieter ab.
