# <a name="example-code-for-using-idirectorysearch"></a>Beispiel Code für die Verwendung von idirector ysearch

Das Platform Software Development Kit (SDK) enthält ein Codebeispiel Thema, dssrch, das eine einfache Eingabe Aufforderungs Anwendung bereitstellt, die die [**IDirectorySearch**](/windows/desktop/api/Iads/nn-iads-idirectorysearch) -Schnittstelle verwendet, um einen Verzeichnisdienst abzufragen. Das vollständige Codebeispiel ist im Platform SDK in den Beispielen \\ netds \\ ADSI \\ Samples \\ General \\ dssrch Directory enthalten.
