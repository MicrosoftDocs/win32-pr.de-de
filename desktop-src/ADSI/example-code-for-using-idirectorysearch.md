# <a name="example-code-for-using-idirectorysearch"></a>Beispielcode für die Verwendung von IDirectorySearch

Das Platform Software Development Kit (SDK) enthält das Codebeispielthema DsSrch, das eine einfache Eingabeaufforderungsanwendung bereitstellt, die die [**IDirectorySearch-Schnittstelle**](/windows/desktop/api/Iads/nn-iads-idirectorysearch) zum Abfragen eines Verzeichnisdiensts verwendet. Das vollständige Codebeispiel ist im Platform SDK im Verzeichnis Samples \\ NetDs \\ ADSI \\ Samples General \\ \\ DsSrch enthalten.
