# <a name="winnt-user-object"></a>WinNT-Benutzerobjekt

Das WinNT-Benutzerobjekt stellt ein Benutzerkonto in einer Windows NT 4.0-Domäne dar. Das -Objekt weist besondere Features auf. In einem Fall werden nicht alle Eigenschaftenmethoden der [**IADsUser-Schnittstelle**](/windows/desktop/api/Iads/nn-iads-iadsuser) unterstützt. In einer zweiten Instanz werden einige benutzerdefinierte Eigenschaften unterstützt, auf die nur mit der [**IADs.Get-**](/windows/desktop/api/Iads/nf-iads-iads-get) oder [**IADs.Put-Methode**](/windows/desktop/api/Iads/nf-iads-iads-put) zugegriffen werden kann.

Weitere Informationen zu WinNT-Benutzerobjekten finden Sie unter:

-   [Nicht unterstützte IADsUser-Eigenschaften](unsupported-iadsuser-properties.md)
-   [Benutzerdefinierte WinNT-Benutzereigenschaften](winnt-custom-user-properties.md)
-   [Beispiele für die WinNT-Benutzerkontenverwaltung](winnt-user-account-management-examples.md)
