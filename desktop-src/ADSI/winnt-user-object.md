# <a name="winnt-user-object"></a>WinNT-Benutzerobjekt

Das WinNT-Benutzerobjekt stellt ein Benutzerkonto in einer Windows NT 4,0-Domäne dar. Das-Objekt stellt besondere Features dar. In einer Instanz werden nicht alle Eigenschaften Methoden der [**IADsUser**](/windows/desktop/api/Iads/nn-iads-iadsuser) -Schnittstelle unterstützt. In einer zweiten Instanz werden einige benutzerdefinierte Eigenschaften unterstützt, auf die nur mit der [**IADs. Get**](/windows/desktop/api/Iads/nf-iads-iads-get) -oder [**IADs. Put**](/windows/desktop/api/Iads/nf-iads-iads-put) -Methode zugegriffen werden kann.

Weitere Informationen zu WinNT-Benutzer Objekten finden Sie unter:

-   [Nicht unterstützte IADsUser-Eigenschaften](unsupported-iadsuser-properties.md)
-   [Benutzerdefinierte WinNT-Benutzereigenschaften](winnt-custom-user-properties.md)
-   [Beispiele für die WinNT-Benutzerkonten Verwaltung](winnt-user-account-management-examples.md)
