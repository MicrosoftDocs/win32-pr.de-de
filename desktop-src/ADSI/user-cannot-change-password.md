# <a name="user-cannot-change-password-ldap-provider"></a>Benutzer kann Kennwort nicht ändern (LDAP-Anbieter)

Die Fähigkeit eines Benutzers, sein eigenes Kennwort zu ändern, ist eine Berechtigung, die erteilt oder verweigert werden kann. Weitere Informationen zum programmgesteuerten Lesen und Ändern dieser Berechtigung mithilfe des LDAP-Anbieters finden Sie unter:

-   [Lese Benutzer kann Kennwort nicht ändern (LDAP-Anbieter)](reading-user-cannot-change-password-ldap-provider.md)
-   [Ändern des Benutzers kann das Kennwort nicht ändern (LDAP-Anbieter)](modifying-user-cannot-change-password-ldap-provider.md)

Es ist einfacher zu lesen und zu ändern, dass die Kenn Wort Einstellung des Benutzers mit dem WinNT-Anbieter nicht geändert werden kann. Weitere Informationen finden Sie unter [Benutzer kann das Kennwort nicht ändern (WinNT-Anbieter)](winnt-user-cannot-change-password.md).
