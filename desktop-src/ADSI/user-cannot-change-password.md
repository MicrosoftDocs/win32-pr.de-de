# <a name="user-cannot-change-password-ldap-provider"></a>Benutzer kann Kennwort nicht ändern (LDAP-Anbieter)

Die Fähigkeit eines Benutzers, sein eigenes Kennwort zu ändern, ist eine Berechtigung, die erteilt oder verweigert werden kann. Weitere Informationen zum programmgesteuerten Lesen und Ändern dieser Berechtigung mithilfe des LDAP-Anbieters finden Sie unter:

-   [Lesebenutzer kann Kennwort nicht ändern (LDAP-Anbieter)](reading-user-cannot-change-password-ldap-provider.md)
-   [Ändern des Kennworts durch Den Benutzer kann nicht geändert werden (LDAP-Anbieter)](modifying-user-cannot-change-password-ldap-provider.md)

Es ist einfacher, die Einstellung Benutzer kann Kennwort nicht ändern mithilfe des WinNT-Anbieters zu lesen und zu ändern. Weitere Informationen finden Sie unter [User Cannot Change Password (WinNT Provider) (Benutzer kann Kennwort nicht ändern (WinNT-Anbieter)).](winnt-user-cannot-change-password.md)
