# <a name="user-cannot-change-password-winnt-provider"></a>Benutzer kann Kennwort nicht ändern (WinNT-Anbieter)

Die Fähigkeit eines Benutzers, sein Kennwort zu ändern, ist eine Berechtigung, die erteilt oder verweigert werden kann. Weitere Informationen zum programmgesteuerten Lesen und Ändern dieser Berechtigung mithilfe des WinNT-Anbieters finden Sie unter:

-   [Lesevorgang kann das Kennwort nicht ändern (WinNT-Anbieter)](reading-user-cannot-change-password-winnt-provider.md)
-   [Ändern des Benutzers kann das Kennwort nicht ändern (WinNT-Anbieter)](modifying-user-cannot-change-password-winnt-provider.md)

Das **UserFlags** -Attribut des User-Objekts ist nur verfügbar, wenn der WinNT-Anbieter verwendet wird.

Es ist auch möglich, diese Berechtigung mithilfe des LDAP-Anbieters zu lesen und zu ändern. Weitere Informationen finden Sie unter [Benutzer kann das Kennwort nicht ändern (LDAP-Anbieter)](user-cannot-change-password.md).
