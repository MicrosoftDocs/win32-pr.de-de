# <a name="ldap-syntax-object"></a>LDAP-Syntaxobjekt

Der LDAP-Anbieter verwendet die folgende Zuordnung zwischen der LDAP-Syntax und der ADSI-Syntax.



| LDAP-Syntax   | ADSI-Syntax (Automation)            |
|---------------|-------------------------------------|
| ADsPath       | VT \_ BSTR                            |
| Boolean       | VT \_ BOOL                            |
| Leistungsindikator       | VT \_ I4                              |
| EmailAddress  | VT \_ BSTR                            |
| FaxNumber     | VT \_ BSTR                            |
| Integer       | VT \_ I4                              |
| Intervall      | VT \_ I4                              |
| List          | VT \_ VARIANT (VT \_ BSTR \| VT \_ ARRAY) |
| NetAddress    | VT \_ BSTR                            |
| OctetString   | VT \_ VARIANT                         |
| Pfad          | VT \_ BSTR                            |
| PhoneNumber   | VT \_ BSTR                            |
| PostalAddress | VT \_ BSTR                            |
| SmallInterval | VT \_ I4                              |
| String        | VT \_ BSTR                            |
| Time          | VT \_ DATE                            |
