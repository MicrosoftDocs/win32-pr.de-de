# <a name="ldap-syntax-object"></a>LDAP-Syntax Objekt

Der LDAP-Anbieter verwendet die folgende Zuordnung zwischen der LDAP-Syntax und der ADSI-Syntax.



| LDAP-Syntax   | ADSI-Syntax (Automatisierung)            |
|---------------|-------------------------------------|
| ADsPath       | VT \_ BSTR                            |
| Boolean       | VT \_ bool                            |
| Zähler       | VT \_ I4                              |
| EmailAddress  | VT \_ BSTR                            |
| Faxnummer     | VT \_ BSTR                            |
| Integer       | VT \_ I4                              |
| Intervall      | VT \_ I4                              |
| List          | VT- \_ Variante (VT \_ BSTR \| VT- \_ Array) |
| Netaddress    | VT \_ BSTR                            |
| Octetstring   | VT- \_ Variante                         |
| Pfad          | VT \_ BSTR                            |
| PhoneNumber   | VT \_ BSTR                            |
| PostalAddress | VT \_ BSTR                            |
| Smallinterval | VT \_ I4                              |
| String        | VT \_ BSTR                            |
| Time          | VT- \_ Datum                            |
