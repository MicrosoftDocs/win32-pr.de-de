# <a name="active-directory-service-interfaces-provider"></a>Anbieter von Active Directory-Dienstschnittstellen

Ein ADSI-Anbieter enthält die Implementierung von ADSI-Objekten und abhängigen Objekten für einen bestimmten Namespace. Die folgende Abbildung zeigt, dass es Clients nur um das Abrufen und Verwenden von Schnittstellen für ein Objekt geht, nicht um die Details dazu, wo und wie die Software eines Objekts implementiert wird.
