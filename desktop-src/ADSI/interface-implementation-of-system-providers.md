# <a name="interface-implementation-of-system-providers"></a>Schnittstellen Implementierung von System Anbietern

ADSI umfasst mehrere Verzeichnis Dienstanbieter, nämlich LDAP und WinNT. Verschiedene Anbieter können eine Schnittstelle implementieren oder nicht implementieren. Verschiedene Verzeichnisdienst Anbieter unterstützen verschiedene Eigenschaften und Methoden. Weitere Informationen zur Unterstützung bestimmter Verzeichnisdienst Anbieter finden Sie unter [ADSI-System Anbieter](adsi-system-providers.md).
