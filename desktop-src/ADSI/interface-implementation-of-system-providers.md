# <a name="interface-implementation-of-system-providers"></a>Schnittstellenimplementierung von Systemanbietern

ADSI umfasst mehrere Verzeichnisdienstanbieter, nämlich LDAP und WinNT. Verschiedene Anbieter können eine Schnittstelle implementieren oder nicht implementieren. Verschiedene Verzeichnisdienstanbieter unterstützen unterschiedliche Eigenschaften und Methoden. Weitere Informationen zur Unterstützung bestimmter Verzeichnisdienstanbieter finden Sie unter [ADSI-Systemanbieter.](adsi-system-providers.md)
