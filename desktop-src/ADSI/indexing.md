# <a name="indexing"></a>Indizierung

Suchen Sie nach Möglichkeit nach indizierten Attributen oder nach mindestens einem indizierten Attribut. Wie im Allgemeinen bei Datenbanken ermöglicht die Suche nach einem indizierten Attribut dem Server eine effizientere Berechnung des Ergebnisses.
