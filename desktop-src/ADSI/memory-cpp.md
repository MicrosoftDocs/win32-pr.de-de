# <a name="memorycpp"></a>Gedenkens. CPP

In der Beispiel Anbieter Komponente ist ein Codebeispiel, das die Speicher Belegung und-Freigabe anzeigt, in "Memory. cpp". In der folgenden Tabelle sind die unterstützten Routinen aufgeführt.



| Element                | BESCHREIBUNG                                                           |
|---------------------|-----------------------------------------------------------------------|
| **"Zuweisung"**    | Zuordnen des angegebenen Speichers.                                            |
| **Freiprovmem**     | Der freie Speicherplatz wird angegeben.                                                |
| **Rezuweisung**  | Zuordnen von zusammenhängenden Arbeitsspeicher.                                           |
| **Zuordnung von "Zuweisung"**    | Zuordnen einer LPWSTR-Zeichenfolge                                            |
| **Freiprovstr**     | Freie Zeichenfolge, wenn Sie nicht bereits freigegeben wurde.                                     |
| **Rezuweisung**  | Zuordnen von zusammenhängenden Arbeitsspeicher.                                           |
| **Provallocstring** | Überprüft die Zeichenfolge und den ersten Parameter. Wenn OK, wird die Zuordnung durchführt. |
