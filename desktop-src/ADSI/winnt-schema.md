# <a name="winnt-schema"></a>WinNT-Schema

Das WinNT-Schema definiert den Satz von ADSI-Objekten und ihre Klassenhierarchie für ein Windows NT 4,0-Netzwerk. [Zu den ADSI-Objekten von Winnt](adsi-objects-of-winnt.md) zählen Benutzer, Computer, Domäne, Gruppe, Ressource, Sitzung, Dienst und andere. Der folgende Abschnitt der [Winnt-Objektklassen Hierarchie](winnt-object-class-hierarchy.md) erläutert die hierarchischen Beziehungen zwischen diesen Objekten.

Jedes Objekt implementiert eine Reihe von ADSI-Schnittstellen. Allerdings werden für eine implementierte Schnittstelle nicht alle Eigenschaften und Methoden unterstützt. Erforderliche [und optionale Eigenschaften des WinNT-Schemas](winnt-schemaampaposs-mandatory-and-optional-properties.md) listet die Eigenschaften auf, die von den WinNT-Objektklassen unterstützt werden.
