# <a name="winnt-schema"></a>WinNT-Schema

Das WinNT-Schema definiert den Satz von ADSI-Objekten und deren Klassenhierarchie für Windows NT 4.0-Netzwerk. Zu [den ADSI-Objekten von WinNT](adsi-objects-of-winnt.md) gehören Benutzer, Computer, Domäne, Gruppe, Ressource, Sitzung, Dienst und andere. Im [folgenden Abschnitt Hierarchie der WinNT-Objektklasse](winnt-object-class-hierarchy.md) werden die hierarchischen Beziehungen zwischen diesen Objekten erläutert.

Jedes -Objekt implementiert einen Satz von ADSI-Schnittstellen. Für eine implementierte Schnittstelle werden jedoch nicht alle Eigenschaften und Methoden unterstützt. [Die obligatorischen und optionalen](winnt-schemaampaposs-mandatory-and-optional-properties.md) Eigenschaften des WinNT-Schemas listet die Eigenschaften auf, die von den WinNT-Objektklassen unterstützt werden.
