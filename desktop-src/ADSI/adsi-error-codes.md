# <a name="adsi-error-codes"></a>ADSI-Fehler Codes

Alle ADSI-Fehlercodes werden als com **HRESULT** -Werte zurückgegeben. Sie können in die folgenden vier Typen gruppiert werden:

-   [Generische com-Fehlercodes](generic-com-error-codes.md)
-   [Generische ADSI-Fehlercodes](generic-adsi-error-codes.md)
-   [Win32-Fehlercodes für ADSI](win32-error-codes-for-adsi.md)
-   [LDAP-Fehlercodes für ADSI](ldap-error-codes-for-adsi.md)

Außerdem bieten bestimmte Schnittstellen zusätzliche Fehlerinformationen, die als erweiterte ADSI-Fehlermeldungen bezeichnet werden, die mithilfe der [**adsgetlasterror**](/windows/desktop/api/Adshlp/nf-adshlp-adsgetlasterror) -Funktion abgerufen werden können.

Der C++- [Beispielcode](code-example-for-working-with-adsi-error-messages.md) in diesem Abschnitt zeigt, wie ADSI-Fehlermeldungen verwendet werden.
