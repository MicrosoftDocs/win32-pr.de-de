# <a name="cschobjcpp"></a>Cschobj. CPP

In der Beispiel Anbieter Komponente ist ein Codebeispiel, das die Lebensdauer der Schema Objekte verwaltet, in cschobj. cpp. In der folgenden Tabelle sind die unterstützten Methoden aufgeführt.



| Methode                                                   | BESCHREIBUNG                                    |
|----------------------------------------------------------|------------------------------------------------|
| **Csampledsschema:: kreateschema**                        | Standardkonstruktor.                          |
| **Csampledsschema:: ~ csampledsschema**                    | Standardedekonstruktor.                           |
| Standard- [**IADs**](/windows/desktop/api/Iads/nn-iads-iads) -Methoden.                   |                                                |
| Standard mäßige [**IADsContainer**](/windows/desktop/api/Iads/nn-iads-iadscontainer) -Methoden. |                                                |
| **Csampledsschema:: QueryInterface**                      | Überprüfen Sie die angegebene Schnittstellen-ID für dieses Objekt.  |
| **Csampledsschema:: Zuordnungs-Manager**                      | Erstellen Sie ein Schema Objekt, und laden Sie dessen Typdaten. |
