# <a name="cnamespcpp"></a>Cnamesp. CPP

In der Beispiel Anbieter Komponente befindet sich Code, der die Lebensdauer des Namespace Objekts der Beispiel Anbieter Komponente verwaltet, in cnamesp. cpp. In der folgenden Tabelle sind die unterstützten Methoden aufgeführt.



| Methode                                                   | BESCHREIBUNG                                            |
|----------------------------------------------------------|--------------------------------------------------------|
| **Csampledsnamespace:: csampledsnamespace**               | Standardkonstruktor.                                  |
| **Csampledsnamespace:: ~ csampledsnamespace**              | Standardedekonstruktor.                                   |
| Standard- [**IADs**](/windows/desktop/api/Iads/nn-iads-iads) -Methoden.                   |                                                        |
| Standard mäßige [**IADsContainer**](/windows/desktop/api/Iads/nn-iads-iadscontainer) -Methoden. |                                                        |
| **Csampledsnamespace:: Zuordnungs-emailespaceobject**          | Erstellen Sie ein Namespace Objekt, und laden Sie dessen Typdaten.      |
| **Csampledsnamespace:: erstellt**                  | Zuordnen, initialisieren und Validieren eines Namespace Objekts. |
| **Csampledsnamespace:: QueryInterface**                   | Überprüfen Sie die angegebene IID für dieses Objekt.                   |
