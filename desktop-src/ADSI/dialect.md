# <a name="dialect"></a>Dialect

Zwei der ADSI-Schnittstellen zum Durchsuchen eines Verzeichnisses, ADO und OLE DB, verwenden eine Such Zeichenfolge, um die Parameter für die Suche anzugeben. Die Suchparameter enthalten die Basis der Suche, den Suchfilter, die abzurufenden Attribute und die Tiefe der Suche.

Sie können die Such Zeichenfolge in einem von zwei Dialekten angeben:

-   [LDAP-Dialekt](ldap-dialect.md)
-   [SQL-Dialekt](sql-dialect.md)
