# <a name="about-active-directory-service-interfaces"></a>Informationen zu Active Directory Dienst Schnittstellen

Active Directory Service Interfaces (ADSI) abstrahiert die Funktionen der Verzeichnisdienste von unterschiedlichen Netzwerkanbietern in einer verteilten Computerumgebung, um eine einzelne Gruppe von Verzeichnisdienst Schnittstellen für die Verwaltung von Netzwerkressourcen bereitzustellen. Administratoren und Entwickler können ADSI-Dienste verwenden, um die Ressourcen in einem Verzeichnisdienst aufzulisten und zu verwalten, unabhängig davon, welche Netzwerkumgebung die Ressource enthält.

ADSI vereinfacht das Ausführen allgemeiner Verwaltungsaufgaben, z. b. das Hinzufügen neuer Benutzer, das Verwalten von Druckern und das Auffinden von Ressourcen in der verteilten Computerumgebung.

ADSI erleichtert Entwicklern das "Verzeichnis aktivieren" Ihrer Anwendungen. Administratoren und Entwickler verarbeiten einen einzelnen Satz von Verzeichnisdienst Schnittstellen, unabhängig davon, welche Verzeichnisdienste installiert sind.
