# <a name="about-active-directory-service-interfaces"></a>Informationen zu Active Directory-Dienstschnittstellen

Active Directory Service Interfaces (ADSI) abstrahiert die Funktionen von Verzeichnisdiensten verschiedener Netzwerkanbieter in einer verteilten Computingumgebung, um einen einzigen Satz von Verzeichnisdienstschnittstellen für die Verwaltung von Netzwerkressourcen darzustellen. Administratoren und Entwickler können ADSI-Dienste verwenden, um die Ressourcen in einem Verzeichnisdienst aufzulisten und zu verwalten, unabhängig davon, welche Netzwerkumgebung die Ressource enthält.

ADSI vereinfacht die Durchführung allgemeiner Verwaltungsaufgaben, z. B. das Hinzufügen neuer Benutzer, das Verwalten von Druckern und das Auffinden von Ressourcen in der gesamten verteilten Computingumgebung.

ADSI erleichtert Entwicklern das "Verzeichnis aktivieren" ihrer Anwendungen. Administratoren und Entwickler verarbeiten einen einzelnen Satz von Verzeichnisdienstschnittstellen, unabhängig davon, welche Verzeichnisdienste installiert sind.
