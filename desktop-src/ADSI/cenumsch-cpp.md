# <a name="cenumschcpp"></a>Cenumschlag. CPP

In der Beispiel Anbieter Komponente verwendet die Enumeration eines Schema Objekts die Methoden aus "cenumsch. cpp", die in der folgenden Tabelle aufgeführt sind.



| Methode                                        | BESCHREIBUNG                                                                                                          |
|-----------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| **Csampledsschemaerum:: Create**               | Erstellen Sie ein-Objekt, um die Enumeration eines ADSI-Schema Klassen Objekts zuzulassen.                                                |
| **Csampledsschemaerum:: csampledsschemaerum**  | Standardkonstruktor.                                                                                                |
| **Csampledsschemaerum:: ~ csampledsschemaerum** | Standardedekonstruktor.                                                                                                 |
| **Csampledsschemaerum:: Next**                 | Ruft die angegebene Anzahl von Elementen aus dem angegebenen Schema Objekt ab.                                          |
| **Csampledsschemadenum:: umumubjects**          | Verwaltet das Abrufen der Schnittstellen Zeiger auf die Objekte des Objekttyps, die angegeben sind.                               |
| **Csampledsschemadenum:: umumubjects**          | Verwaltet das Abrufen der Schnittstellen Zeiger auf die Objekte des Standard Objekt Typs.                                  |
| **Csampledsschemaenumum:: enumclasses**          | Verwaltet das Abrufen der Schnittstellen Zeiger auf die in diesem-Objekt enthaltenen Schema Klassen Objekte.                  |
| **Csampledsschemadenum:: GetClassObject**       | Abrufen der nächsten Schema Klassendefinition Wenn Sie gefunden wird, erstellen Sie ein Schema Klassenobjekt, und geben Sie den Schnittstellen Zeiger zurück. |
| **Csampledsschemaenum:: EnumProperties**       | Verwaltet das Abrufen der Schnittstellen Zeiger auf die Eigenschafts Objekte, die nur in diesem-Objekt enthalten sind.                      |
| **Csampledsschemaenum:: getpropertyobject**    | Abrufen der nächsten Eigenschafts Definition Wenn Sie gefunden wird, erstellen Sie ein Schema Klassenobjekt, und geben Sie den Schnittstellen Zeiger zurück.     |
| **Csampledsschemaenumum:: enumsyntaxen**         | Verwaltet das Abrufen der Schnittstellen Zeiger auf die Syntax Objekte, die nur in diesem-Objekt enthalten sind.                        |
| **Csampledsschemaenum:: getsyntaxobject**      | Abrufen der nächsten Syntax Definition Wenn Sie gefunden wird, erstellen Sie ein Schema Klassenobjekt, und geben Sie den Schnittstellen Zeiger zurück.       |
