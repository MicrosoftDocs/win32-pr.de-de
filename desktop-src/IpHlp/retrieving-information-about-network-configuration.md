# <a name="retrieving-information-about-network-configuration"></a>Abrufen von Informationen zur Netzwerkkonfiguration

Das IP-Hilfsprogramm stellt Informationen zur Netzwerkkonfiguration des lokalen Computers zur Verfügung. Verwenden Sie zum Abrufen allgemeiner Konfigurationsinformationen die [**GetNetworkParams-Funktion.**](/windows/desktop/api/Iphlpapi/nf-iphlpapi-getnetworkparams) Diese Funktion gibt Informationen zurück, die nicht spezifisch für einen bestimmten Adapter oder eine bestimmte Schnittstelle sind. **GetNetworkParams** gibt beispielsweise eine Liste der DNS-Server zurück, die vom lokalen Computer verwendet werden.

-   Codebeispiele im Zusammenhang [**mit GetNetworkParams finden**](/windows/desktop/api/Iphlpapi/nf-iphlpapi-getnetworkparams) Sie unter [Abrufen von Informationen mit GetNetworkParams.](retrieving-information-using-getnetworkparams.md)
