# <a name="pausezone-method-of-the-microsoftdns_zone-class"></a>PauseZone-Methode der MicrosoftDNS \_ Zone-Klasse

Die **PauseZone-Methode** hält die DNS-Zone an.

## <a name="syntax"></a>Syntax


```mof
void PauseZone();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.
