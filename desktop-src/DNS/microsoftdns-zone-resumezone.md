# <a name="resumezone-method-of-the-microsoftdns_zone-class"></a>Resumezone-Methode der MicrosoftDNS- \_ Zonen Klasse

Die **resumezone** -Methode nimmt die DNS-Zone wieder auf.

## <a name="syntax"></a>Syntax


```mof
void ResumeZone();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.
