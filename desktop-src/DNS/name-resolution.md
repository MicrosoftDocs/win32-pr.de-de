# <a name="name-resolution"></a>Namensauflösung

Der Zweck von DNS besteht darin, eine Namensauflösung bereitzustellen: der Prozess, bei dem ein Hostname (z. B. ein Computername) mit einer Liste von RRs verglichen und die entsprechende IP-Adresse identifiziert wird. Bei der umgekehrten Namensauflösung wird eine IP-Adresse mit einer Liste von RRs verglichen und der entsprechende Hostname identifiziert. Diese Auflösungsprozesse werden initiiert, wenn ein DNS-Server eine Abfrage empfängt.
