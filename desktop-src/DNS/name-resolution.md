# <a name="name-resolution"></a>Namensauflösung

Der Zweck des DNS besteht darin, die Namensauflösung anzugeben: den Prozess des Vergleichs eines Host namens (z. b. eines Computer namens) mit einer Liste von RRS und die Identifizierung der entsprechenden IP-Adresse. Die umgekehrte Namensauflösung ist der Prozess, bei dem eine IP-Adresse mit einer Liste von RRS verglichen und der entsprechende Hostname identifiziert wird. Diese Auflösungsprozesse werden initiiert, wenn ein DNS-Server eine Abfrage empfängt.
