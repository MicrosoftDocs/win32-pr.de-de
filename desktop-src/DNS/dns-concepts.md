# <a name="dns-concepts"></a>DNS-Konzepte

Es gibt einige grundlegende Konzepte, mit denen die funktionale Struktur von DNS arbeitet. Diese grundlegenden Konzepte reichen von Definitionen des Namensraums bis hin zur Abgrenzung der Zuständigkeitsgrenzen für die Namensauflösung. Das Verständnis dieser Konzepte ist ein langer Weg, um fundierte Kenntnisse über DNS zu erhalten. Die folgenden DNS-Konzepte werden in diesem Abschnitt definiert und erläutert:

-   [Name Space](name-space.md)
-   [DNS-Domänen](dns-domains.md)
-   [DNS-Zonen](dns-zones.md)
-   [wurzel](root.md)
-   [Namensauflösung](name-resolution.md)
-   [DNS-Server](dns-servers.md)
-   [DNS-Resolver](dns-resolvers.md)
-   [Ressourcendatensätze](resource-records.md)
