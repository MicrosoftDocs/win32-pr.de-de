# <a name="dns-concepts"></a>DNS-Konzepte

Es gibt eine Reihe grundlegender Konzepte, mit denen die Funktionsstruktur von DNS funktioniert. Diese grundlegenden Konzepte reichen von Namespace Definitionen bis hin zur Abgrenzung von Aufgaben Grenzen der Namensauflösung. Das Verständnis dieser Konzepte ist ein langer Weg, um eine solide Funktionsweise von DNS zu erhalten. Die folgenden DNS-Konzepte werden definiert und in diesem Abschnitt erläutert:

-   [Namespace](name-space.md)
-   [DNS-Domänen](dns-domains.md)
-   [DNS-Zonen](dns-zones.md)
-   [Fasst](root.md)
-   [Namensauflösung](name-resolution.md)
-   [DNS-Server](dns-servers.md)
-   [DNS-Resolver](dns-resolvers.md)
-   [Ressourcen Einträge](resource-records.md)
