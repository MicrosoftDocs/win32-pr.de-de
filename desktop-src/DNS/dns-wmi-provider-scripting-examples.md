# <a name="dns-wmi-provider-scripting-examples"></a>Skriptbeispiele für DNS-WMI-Anbieter

Dieser Abschnitt enthält Skriptbeispiele, die veranschaulichen, wie Administrative Aufgaben mithilfe des DNS-WMI-Anbieters erstellt werden.

Die folgenden Typen von Skriptbeispielen werden bereitgestellt.



| `Section`                                                                                     | Bereitgestellte Beispiele                                                                                                                                                                          |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Verwalten eines DNS-Servers](dns-wmi-provider-samples-managing-a-dns-server.md)                 | Enthält Beispiele, die die Verwaltung eines DNS-Servers ermöglichen, einschließlich Beenden, Starten oder Neustarten eines Servers, Auflisten von Zonen, Hinzufügen oder Löschen von IP-Adressen und anderen.                |
| [Verwalten von DNS-Ressourceneinträgen](dns-wmi-provider-samples-managing-dns-resource-records.md) | Veranschaulicht, wie Sie eine Verbindung mit dem DNS-WMI-Anbieter herstellen und verschiedene Aufgaben im Zusammenhang mit der Verwaltung von Ressourceneinträgen ausführen, z. B. Hinzufügen, Löschen, Ändern oder Auflisten von Ressourceneinträgen. |
| [Verwalten von DNS-Zonen](dns-wmi-provider-samples-managing-dns-zones.md)                       | Beispiele hierfür sind das Erstellen, Ändern und Löschen einer Zone sowie das Anhalten, Fortsetzen, Aktualisieren und andere Zonenaufgaben für DNS-Administratoren.                                  |
