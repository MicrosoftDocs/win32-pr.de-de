# <a name="dns-wmi-provider-scripting-examples"></a>Skript Beispiele für DNS-WMI-Anbieter

Dieser Abschnitt enthält Skript Beispiele, die veranschaulichen, wie Sie mithilfe des DNS-WMI-Anbieters Skripts für administrative Aufgaben ausführen.

Die folgenden Arten von Skript Beispielen werden bereitgestellt.



| `Section`                                                                                     | Bereitgestellte Beispiele                                                                                                                                                                          |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Verwalten eines DNS-Servers](dns-wmi-provider-samples-managing-a-dns-server.md)                 | Enthält Beispiele, die die Verwaltung eines DNS-Servers ermöglichen. dazu gehören das Beenden, starten oder Neustarten eines Servers, das Auflisten von Zonen, das Hinzufügen oder Löschen von IP-Adressen und anderen.                |
| [Verwalten von DNS-Ressourcen Einträgen](dns-wmi-provider-samples-managing-dns-resource-records.md) | Veranschaulicht das Herstellen einer Verbindung mit dem DNS-WMI-Anbieter und das Ausführen verschiedener Tasks im Zusammenhang mit der Verwaltung von Ressourcen Einträgen – hinzufügen, löschen, ändern oder Auflisten von Ressourcen Einträgen. |
| [Verwalten von DNS-Zonen](dns-wmi-provider-samples-managing-dns-zones.md)                       | Beispiele sind das Erstellen, ändern und Löschen einer Zone sowie das Anhalten, fortsetzen, aktualisieren und andere Zonen Aufgaben, die für DNS-Administratoren ausgeführt werden.                                  |
