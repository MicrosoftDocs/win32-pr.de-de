# <a name="r-dns"></a>R (DNS)

A B [C](c-gly.md) D E [F](f-gly.md) G [H](h-gly.md) I J K [L](l-gly.md) M N O [P](p-gly.md) Q R [S](s-gly.md) T U V W X Y [Z](z-gly.md)

<dl> <dt>

<span id="_dns_relative_distinguished_name_gly"></span><span id="_DNS_RELATIVE_DISTINGUISHED_NAME_GLY"></span>**Relativer Distinguished Name**
</dt> <dd>

Name, der einen Host innerhalb seiner eigenen Domäne eindeutig identifiziert, jedoch nicht in der gesamten DNS-Hierarchie. Beispielsweise ist server1 ein relativer Distinguished Name, während server1.products.Microsoft.com ein vollqualifizierter Domänenname (Fully Qualified Domain Name, FQDN) ist.

</dd> <dt>

<span id="_dns_resolver_gly"></span><span id="_DNS_RESOLVER_GLY"></span>**Resolver**
</dt> <dd>

Eine im Windows TCP/IP-Stapel (Transmission Control Protocol/Internet Protocol) gebündelte Softwarekomponente, die mit DNS-Servern kommuniziert, um Namensauflösungsabfragen zu erstellen und aufzulösen.

</dd> <dt>

<span id="_dns_resource_record_rr__gly"></span><span id="_DNS_RESOURCE_RECORD_RR__GLY"></span>**Ressourceneintrag (RR)**
</dt> <dd>

Die Informationseinheit im DNS. Es gibt viele Arten von Ressourcendatensätzen.

</dd> <dt>

<span id="_dns_root_gly"></span><span id="_DNS_ROOT_GLY"></span>**wurzel**
</dt> <dd>

Die oberste Ebene in einer bestimmten Hierarchie.

</dd> </dl>
