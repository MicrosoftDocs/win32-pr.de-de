# <a name="dns-wmi-classes"></a>DNS-WMI-Klassen

Der DNS-WMI-Anbieter definiert die folgenden DNS-WMI-Klassen.



| WMI-Klasse für Microsoft DNS                                                                               | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|-------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MicrosoftDNS- \_ Server**](microsoftdns-server.md)                                                   | Beschreibt einen DNS-Server. Jede Instanz dieser Klasse kann einer Instanz der Klasse " [**MicrosoftDNS- \_ Cache**](microsoftdns-cache.md)", einer Instanz der Klasse " [**MicrosoftDNS \_ roothints**](microsoftdns-roothints.md)" und mehreren Instanzen der Klasse " [**MicrosoftDNS \_ Zone**](microsoftdns-zone.md)" zugeordnet werden.                                                                                                                                                                                                                                                                                                                                                                                       |
| [**MicrosoftDNS- \_ Domäne**](microsoftdns-domain.md)                                                   | Stellt eine Domäne in einer DNS-Hierarchiestruktur dar.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**MicrosoftDNS- \_ Zone**](microsoftdns-zone.md)                                                       | Beschreibt eine DNS-Zone. Jede Instanz der [**MicrosoftDNS \_**](microsoftdns-zone.md) -Klasse muss genau einem DNS-Server zugewiesen werden. Zonen können mehreren Instanzen der Klassen " [**MicrosoftDNS \_ Domain**](microsoftdns-domain.md) " und " [**MicrosoftDNS \_ resourcerecord**](microsoftdns-resourcerecord.md)" zugeordnet werden.                                                                                                                                                                                                                                                                                                                                                                    |
| [**MicrosoftDNS- \_ Cache**](microsoftdns-cache.md)                                                     | Beschreibt einen Cache, der auf einem DNS-Server vorhanden ist (verwechseln Sie dies nicht mit einer Cachedatei, die [*Stamm Hinweise enthält*](r-gly.md) ). Diese Klasse vereinfacht die Visualisierung der Kapselung von DNS-Objekten, anstatt ein reales Objekt darzustellen. Die-Klasse, [**MicrosoftDNS- \_ Cache**](microsoftdns-cache.md), ist ein Container für die [*Ressourcen Datensätze*](r-gly.md) , die vom DNS-Server zwischengespeichert werden. Jede Instanz der **MicrosoftDNS \_** -Klasse muss genau einem DNS-Server zugewiesen werden. Sie ist möglicherweise mehreren Instanzen der [**MicrosoftDNS- \_ Domäne**](microsoftdns-domain.md) und [**MicrosoftDNS \_ resourcerecord**](microsoftdns-resourcerecord.md)zugeordnet. |
| [**MicrosoftDNS- \_ roothints**](microsoftdns-roothints.md)                                             | Beschreibt die in einer Cachedatei auf einem DNS-Server gespeicherten roothints. Diese Klasse vereinfacht die Visualisierung der Kapselung von DNS-Objekten, anstatt ein reales Objekt darzustellen. Die Klasse [**MicrosoftDNS \_ roothints**](microsoftdns-roothints.md) ist ein Container für die Ressourcen Datensätze, die vom DNS-Server in einer Cachedatei gespeichert werden. Jede Instanz der **MicrosoftDNS \_** -Klasse muss genau einem DNS-Server zugewiesen werden. Sie ist möglicherweise mehreren Instanzen der [**MicrosoftDNS \_ resourcerecord**](microsoftdns-resourcerecord.md) -Klasse zugeordnet.                                                                                                                                               |
