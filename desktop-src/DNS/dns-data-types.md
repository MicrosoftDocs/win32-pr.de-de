# <a name="dns-data-types"></a>DNS-Datentypen

Die folgenden Datentypen sind für DNS definiert:


```C++
typedef DWORD IP4_ADDRESS;
```



<dl> <dt>

**\_IP4-ADRESSE**
</dt> <dd>

Ein -Wert, der eine Internetprotokoll, Version 4 -Adresse (IPv4) darstellt. Beispielsweise wird die gepunktete dezimale IPv4-Adresse **127.0.0.1** in der Host-Bytereihenfolge als **0x7F000001** dargestellt.

</dd> </dl>
