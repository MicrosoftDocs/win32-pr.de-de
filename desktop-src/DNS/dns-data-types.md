# <a name="dns-data-types"></a>DNS-Datentypen

Die folgenden Datentypen sind für DNS definiert:


```C++
typedef DWORD IP4_ADDRESS;
```



<dl> <dt>

**IP4- \_ Adresse**
</dt> <dd>

Ein-Wert, der eine IPv4-Adresse (Internet Protocol Version 4) darstellt. Die gepunktete Dezimal-IPv4-Adresse **127.0.0.1** wird z. b. in der Host-Byte-Reihenfolge als **0x7b000001** dargestellt.

</dd> </dl>
