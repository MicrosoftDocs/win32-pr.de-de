# <a name="getdistinguishedname-method-of-the-microsoftdns_server-class"></a>GetDistinguishedName-Methode der \_ MicrosoftDNS-Serverklasse

Die **GetDistinguishedName-Methode** ruft den DS-Distinguished Name für die Zone ab.

## <a name="syntax"></a>Syntax


```mof
string GetDistinguishedName();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Gibt den DS-Distinguished Name für die Zone zurück.
