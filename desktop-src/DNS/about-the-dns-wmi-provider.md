# <a name="about-the-dns-wmi-provider"></a>Informationen zum DNS-WMI-Anbieter

DNS-Server ordnen IP-Adressen Computernamen und Computernamen IP-Adressen zu. Auf diese Weise stellen Sie den Mechanismus zum Auffinden von Netzwerkressourcen bereit. Der DNS-WMI-Anbieter ermöglicht Anwendungen die Interaktion mit DNS-Servern über das Unified Management Framework von Windows-Verwaltungsinstrumentation (WMI).
