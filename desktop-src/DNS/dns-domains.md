# <a name="dns-domains"></a>DNS-Domänen

DNS-Domänen sind den meisten Computerexperten vertraut, die das Internet verwenden. Domänen sind Knoten im Namespace der DNS-Hierarchie, und Domänen können weiter in Unterdomänen unterteilt werden. Domänen in der fiktiven Gadgets.widgets.Microsoft.com-Hierarchie enthalten beispielsweise die Gadgets Gadgets, Widgets, Microsoft und com.
