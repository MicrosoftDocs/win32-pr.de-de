# <a name="dns-domains"></a>DNS-Domänen

DNS-Domänen sind den meisten Computerexperten vertraut, die das Internet verwenden. -Domänen sind Knoten im DNS-Hierarchienamensraum, und Domänen können weiter in Unterdomänen unterteilt werden. Domänen in der fiktiven gadgets.widgets.microsoft.com umfassen beispielsweise die Domänen widgets, widgets, microsoft und com.
