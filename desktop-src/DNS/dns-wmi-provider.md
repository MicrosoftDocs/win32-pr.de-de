# <a name="dns-wmi-provider"></a>DNS-WMI-Anbieter

Der DNS-WMI-Anbieter ermöglicht es Administratoren und Programmierern, Domain Name System (DNS)-Ressourcen Einträge (RRS) und DNS-Server mithilfe von Windows-Verwaltungsinstrumentation (WMI) zu konfigurieren.
