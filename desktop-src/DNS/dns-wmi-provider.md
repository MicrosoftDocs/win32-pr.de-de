# <a name="dns-wmi-provider"></a>DNS-WMI-Anbieter

Der DNS-WMI-Anbieter ermöglicht Es Administratoren und Programmierern, Domain Name System (DNS)-Ressourceneinträge (RRs) und DNS-Server mithilfe Windows Management Instrumentation (WMI) zu konfigurieren.
