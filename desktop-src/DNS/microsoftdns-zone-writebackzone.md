# <a name="writebackzone-method-of-the-microsoftdns_zone-class"></a>"Write Backzone"-Methode der MicrosoftDNS- \_ Zonen Klasse

Die Methode " **Write Report Backzone** " speichert Zonendaten in der zugehörigen Zonendatei.

## <a name="syntax"></a>Syntax


```mof
void WriteBackZone();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.
