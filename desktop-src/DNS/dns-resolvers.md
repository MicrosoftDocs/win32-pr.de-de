# <a name="dns-resolvers"></a>DNS-Resolver

Der [*Resolver*](r-gly.md) ist eine Softwarekomponente, die im TCP/IP-Stapel (Transmission Control Protocol/Internet Protocol) von Windows 2000 gebündelt ist und mit DNS-Servern kommuniziert, um Abfragen zur Namensauflösung zu erstellen und aufzulösen. Das Paket ist auch in jedem DNS-fähigen Nicht-Microsoft-Client verfügbar. Konfliktlöser sind auf DNS-Clients und DNS-Servern vorhanden.
