# <a name="dns-resolvers"></a>DNS-Resolver

Der [*Konflikt Löser*](r-gly.md) ist eine Softwarekomponente, die im Windows 2000-TCP/IP-Stapel (Transmission Control Protocol/Internet Protocol) gebündelt ist, der mit DNS-Servern kommuniziert, um namens Auflösungs Abfragen zu erstellen und aufzulösen. Das Bündel steht auch in jedem DNS-fähigen nicht-Microsoft-Client zur Verfügung. Auf DNS-Clients und DNS-Servern sind Resolver vorhanden.
