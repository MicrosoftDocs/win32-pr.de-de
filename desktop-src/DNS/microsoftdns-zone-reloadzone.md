# <a name="reloadzone-method-of-the-microsoftdns_zone-class"></a>Reloadzone-Methode der MicrosoftDNS- \_ Zonen Klasse

Mit der **reloadzone** -Methode wird die DNS-Zone aus der Datenbank erneut geladen.

## <a name="syntax"></a>Syntax


```mof
void ReloadZone();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.
