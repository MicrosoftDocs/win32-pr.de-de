# <a name="reloadzone-method-of-the-microsoftdns_zone-class"></a>ReloadZone-Methode der MicrosoftDNS \_ Zone-Klasse

Mit **der ReloadZone-Methode** wird die DNS-Zone aus ihrer Datenbank neu geladen.

## <a name="syntax"></a>Syntax


```mof
void ReloadZone();
```



## <a name="parameters"></a>Parameter

Diese Methode hat keine Parameter.

## <a name="return-value"></a>Rückgabewert

Diese Methode gibt keinen Wert zurück.
