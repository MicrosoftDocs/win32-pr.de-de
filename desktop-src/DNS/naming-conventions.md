## <a name="fully-qualified-domain-name"></a><span data-ttu-id="96ddd-126">Vollqualifizierter Domänenname</span><span class="sxs-lookup"><span data-stu-id="96ddd-126">Fully Qualified Domain Name</span></span>

<span data-ttu-id="96ddd-127">DNS ist eine bessere Lösung. mit dem hierarchischen Namespace werden eindeutige Computernamen in eine bestimmte Domäne isoliert, sodass ein Computername wie *Server1* an verschiedenen Domänen Standorten in derselben Hierarchie vorhanden sein muss.</span><span class="sxs-lookup"><span data-stu-id="96ddd-127">DNS is a better solution; with its hierarchical name space, the need for unique computer names is isolated to a given domain, enabling a computer name such as *server1* to exist in different domain locations in the same hierarchy.</span></span> <span data-ttu-id="96ddd-128">Mit der Möglichkeit, in verschiedenen Domänen denselben Hostnamen zu haben, ist die Notwendigkeit für einen Namen aufgetreten, der die DNS-Hierarchie ordnungsgemäß adressiert hat.</span><span class="sxs-lookup"><span data-stu-id="96ddd-128">With the capability to have the same host name in different domains, the necessity arose for a name that properly addressed the DNS hierarchy.</span></span> <span data-ttu-id="96ddd-129">Der Name muss nicht nur den Computernamen oder den Hostnamen enthalten, sondern auch einen Namen, mit dem dieser Computer in der gesamten DNS-Hierarchie eindeutig identifiziert oder voll qualifiziert werden kann.</span><span class="sxs-lookup"><span data-stu-id="96ddd-129">The name had to include not only the computer name or host name, but also a name that could unambiguously identify, or fully qualify, that computer within the entire DNS hierarchy.</span></span> <span data-ttu-id="96ddd-130">Dieser Name ist der [*voll qualifizierte Domänen Name (Fully Qualified Domain Name*](f-gly.md) , FQDN) – z. b. server1.widgets.Microsoft.com.</span><span class="sxs-lookup"><span data-stu-id="96ddd-130">That name is the [*fully qualified domain name*](f-gly.md) (FQDN) — for example, server1.widgets.microsoft.com.</span></span>
