# <a name="using-the-dns-wmi-provider"></a>Verwenden des DNS-WMI-Anbieters

Der DNS-WMI-Anbieter definiert WMI- [*CIM-Klassen*](c-gly.md) , mit denen die Anwendung einen DNS-Server steuern kann. Die folgenden Abschnitte enthalten Übersichts Informationen, die Ihnen beim Einstieg in die Verwendung des DNS-WMI-Anbieters helfen können. Es werden Links zu Beispiel Skripts bereitgestellt, die veranschaulichen, wie der DNS-WMI-Anbieter verwendet werden kann.

Weitere Informationen zu grundlegenden WMI-Vorgängen finden Sie unter Windows-Verwaltungsinstrumentation (WMI) im Platform Software Development Kit (SDK).
