# <a name="f-dns"></a>F (DNS)

A B [C](c-gly.md) D E F G [H](h-gly.md) I J K [L](l-gly.md) M N O [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) T U V W X Y [Z](z-gly.md)

<dl> <dt>

<span id="_dns_fault_tolerance_gly"></span><span id="_DNS_FAULT_TOLERANCE_GLY"></span>**Fehlertoleranz**
</dt> <dd>

Methode, um sicherzustellen, dass ein Netzwerk oder Computersystem gegen Software- oder Hardwarefehler widerstandsfähig ist.

</dd> <dt>

<span id="_dns_flat_name_space_gly"></span><span id="_DNS_FLAT_NAME_SPACE_GLY"></span>**Flacher Namensraum**
</dt> <dd>

Ein nichtpartitioniertes Namensraum, in dem jeder Eintrag eindeutig sein muss. Flache Namensräume werden aufgrund der Anforderung, dass jeder Eintrag eindeutig sein muss, nicht gut skaliert.

</dd> <dt>

<span id="_dns_forwarder_gly"></span><span id="_DNS_FORWARDER_GLY"></span>**Weiterleitung**
</dt> <dd>

Ein DNS-Server, der für die Kommunikation mit externen DNS-Servern bestimmt wurde.

</dd> <dt>

<span id="_dns_fully_qualified_domain_name_fqdn__gly"></span><span id="_DNS_FULLY_QUALIFIED_DOMAIN_NAME_FQDN__GLY"></span>**Vollqualifizierter Domänenname (FQDN)**
</dt> <dd>

Ein Name, der einen Host in der DNS-Hierarchie eindeutig identifiziert, damit ein Host mit dem Namen server1 in der Produkthierarchie bei Microsoft möglicherweise über einen FQDN von server1.products.Microsoft.com.

</dd> </dl>
