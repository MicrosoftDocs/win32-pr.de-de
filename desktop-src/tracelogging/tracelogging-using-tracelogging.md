# <a name="using-tracelogging"></a>Verwenden von tracelogging

In den folgenden Themen finden Sie einen Schnellstart für einen tracelogging-Schnellstart für nativen und verwalteten Code mit Beispielen.

## <a name="prerequisites"></a>Voraussetzungen

-   Das Windows 10 Software Development Kit (SDK) ist erforderlich, um einen benutzermodusanbieter zu schreiben.
-   Zum Schreiben eines kernelmodusanbieters ist das Windows-Treiberkit (WDK) erforderlich.
