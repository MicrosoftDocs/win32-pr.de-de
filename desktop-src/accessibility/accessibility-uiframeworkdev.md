# <a name="developing-accessible-ui-frameworks-for-windows"></a>Entwickeln von barrierefreien Benutzeroberflächenframeworks für Windows

Zur Verwendung durch beliebig viele Personen sollten die für die Windows-Plattform erstellten Benutzeroberflächen-Frameworks Barrierefreiheits Features unterstützen, die die Verwendung Ihrer Windows-Geräte mit Behinderungen, persönlichen Vorlieben oder bestimmten Arbeitsstilen vereinfachen.

In dieser Übersicht wird beschrieben, wie Sie ein UI-Framework erstellen, das Funktionen wie den programmgesteuerten Zugriff und die Automatisierung, Tastaturnavigation und Befehlsoptionen, Farb-und Designoptionen und Personalisierungen über Benutzereinstellungen unterstützt.
