# <a name="developing-accessible-applications-for-windows"></a>Entwickeln von barrierefreien Anwendungen für Windows

Jedes Windows-Anwendungs Framework unterstützt zahlreiche Barrierefreiheits Features, die es für Benutzer mit Behinderungen, persönlichen Vorlieben oder bestimmten Arbeitsstilen erleichtern, alle von Ihnen gewählten Windows-Geräte erfolgreich zu verwenden.

In dieser Übersicht werden die in den einzelnen Frameworks unterstützten Barrierefreiheits Funktionen beschrieben und erläutert, wie Sie Sie beim Erstellen Ihrer Anwendung integrieren können.
