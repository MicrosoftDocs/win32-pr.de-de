# <a name="legacy-accessibility-and-automation-technology---msaa-to-ui-automation"></a>Legacy-Barrierefreiheit und -Automatisierungstechnologie – von MSAA bis Benutzeroberflächenautomatisierung

Windows-Barrierefreiheit und-Automatisierung bestehen aus zwei Technologien – Microsoft Active Accessibility (MSAA) und Microsoft UI Automation. MSAA ist eine ältere Technologie, die mit Windows 95 eingeführt wurde, während die Benutzeroberflächen Automatisierung eine neuere, leistungsfähigere Technologie ist, die die Einschränkungen von MSAA behandelt.

In diesem Abschnitt werden die Hauptunterschiede zwischen der MSAA-und der Benutzeroberflächen Automatisierung (und der Umstellung auf die Benutzeroberflächen Automatisierung) beschrieben.
