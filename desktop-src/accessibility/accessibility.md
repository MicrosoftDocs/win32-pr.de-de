# <a name="windows-accessibility"></a>Windows-Barrierefreiheit

Dieser Abschnitt enthält Dokumentation für Windows Entwickler, die barrierefreie Anwendungen entwerfen, Hilfstechnologieentwickler, die Tools wie Sprach- und Bildschirmlupe erstellen, und Softwaretesttechniker, die automatisierte Skripts zum Testen Windows erstellen.

Verwenden Sie die hier bereitgestellten Ressourcen, um barrierefreie Windows zu erstellen, die von so vielen Personen wie möglich verwendet werden können, einschließlich Personen mit Behinderungen, persönlichen Vorlieben und bestimmten Arbeitsstilen.
