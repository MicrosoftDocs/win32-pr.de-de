# <a name="windows-accessibility"></a>Windows-Barrierefreiheit

Dieser Abschnitt enthält die Dokumentation für Windows-Entwickler, die barrierefreie Anwendungen entwerfen, hilfstechnologieentwickler, die Tools wie Sprachausgabe und Bildschirmlupe erstellen, und Softwaretest Technikern, die automatisierte Skripts zum Testen von Windows-Anwendungen erstellen.

Verwenden Sie die hier bereitgestellten Ressourcen, um barrierefreie Windows-Anwendungen zu erstellen, die von so vielen Personen wie möglich verwendet werden können, einschließlich derjenigen mit Behinderungen, persönlichen Einstellungen und bestimmten Arbeitsstilen.
