# <a name="windows-accessibility-and-automation-reference"></a>Referenz zur Windows-Barrierefreiheit und-Automatisierung

Die Windows-Barrierefreiheit unterstützt zwei Kategorien von Features, die es Windows-Entwicklern ermöglichen, barrierefreie Anwendungen zu entwerfen, hilfstechnologieentwickler Tools wie Sprachausgabe und Bildschirmlupe zu erstellen, und Softwaretest Techniker erstellen automatisierte Skripts zum Testen von Windows-Anwendungen: Benutzereinstellungen und umfassende Win32-APIs.

Weitere Informationen finden Sie unter [Windows-Barrierefreiheits Funktionen](../WinAuto/about-windows-accessibility-features.md).
