# <a name="pkey_efx_processingmodes_supported_for_streaming-property"></a>Pkey- \_ EFX- \_ processingmodes \_ unterstützt \_ für \_ Streaming-Eigenschaft

Listet die Signal Verarbeitungsmodi auf, die vom Endpunkt Effekt-APO unterstützt werden.

PROPVARIANT-Typ (VT): **VT \_ Vector** \| **VT \_ LPWSTR**

## <a name="remarks"></a>Bemerkungen

Diese Liste enthält nur Signal Verarbeitungsmodi, in denen der APO das Audiosignal beim Streaming tatsächlich verarbeitet.

Diese Liste enthält keine Signal Verarbeitungsmodi, die von der APO nur zu Ermittlungszwecken unterstützt werden.
