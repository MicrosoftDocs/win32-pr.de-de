# <a name="pkey_device_friendlyname"></a>Pkey- \_ Gerät \_ FriendlyName

Die **pkey-Eigenschaft " \_ \_ FriendlyName** " enthält den anzeigen amen des Endpunkt Geräts (z. b. "Sprecher (XYZ-Audioadapter)").

Der **VT** -Member der **PROPVARIANT** -Struktur ist auf VT \_ LPWSTR festgelegt.

Der **pwszval** -Member der **PROPVARIANT** -Struktur verweist auf eine mit NULL endenden Zeichenfolge mit breit Zeichen, die den anzeigen amen des Endpunkt Geräts enthält.
