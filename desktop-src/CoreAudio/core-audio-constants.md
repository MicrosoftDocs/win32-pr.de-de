# <a name="core-audio-constants"></a>Kernaudiokonstanten

In diesem Abschnitt werden die Konstanten beschrieben, die von den Kerncode-APIs in Windows Vista und höher verwendet werden.

| Konstante                                                                            | BESCHREIBUNG                                                                         |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**audclnt \_ sessionflags \_ xxx**](audclnt-sessionflags-xxx-constants.md)<br/> | Flags für die Audiositzung, die während der streaminitialisierung geöffnet wird.<br/> |
| [**audclnt \_ StreamFlags \_ xxx**](audclnt-streamflags-xxx-constants.md)              | Initialisierungsflags für einen Audiodatenstrom.                                           |
| [**Geräte \_ Status \_ xxx**](device-state-xxx-constants.md)                            | Statusflags für ein audioendpunktgerät.                                           |
| [**Endpoint- \_ Hardware \_ Unterstützung \_ xxx**](endpoint-hardware-support-xxx-constants.md) | Hardwareunterstützungs-Flags für ein audioendpunktgerät.                                |
