# <a name="core-audio-constants"></a>Kernaudiokonstanten

In diesem Abschnitt werden die Konstanten beschrieben, die von den Core Audio-APIs in Windows Vista und höher verwendet werden.

| Konstante                                                                            | Beschreibung                                                                         |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**AUDCLNT \_ SESSIONFLAGS \_ XXX**](audclnt-sessionflags-xxx-constants.md)<br/> | Flags für die Audiositzung, die während der Streaminitialisierung geöffnet wird.<br/> |
| [**AUDCLNT \_ STREAMFLAGS \_ XXX**](audclnt-streamflags-xxx-constants.md)              | Initialisierungsflags für einen Audiostream.                                           |
| [**GERÄTEZUSTAND \_ \_ XXX**](device-state-xxx-constants.md)                            | Statusflags für ein Audioendpunktgerät.                                           |
| [**\_ \_ ENDPUNKTHARDWAREUNTERSTÜTZUNG \_ XXX**](endpoint-hardware-support-xxx-constants.md) | Hardwareunterstützungsflags für ein Audioendpunktgerät.                                |
