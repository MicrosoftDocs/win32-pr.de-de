# <a name="core-audio-enumerations"></a>Kernaudioenumerationen

In diesem Abschnitt werden die Enumerationen beschrieben, die von den Core Audio-APIs in Windows Vista und höher verwendet werden.



| Enumeration                                                                   | Beschreibung                                                                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [**\_AUDCLNT \_ BUFFERFLAGS**](/windows/win32/api/audioclient/ne-audioclient-_audclnt_bufferflags)                        | Statusflags für einen Audioendpunktpuffer.                                                         |
| [**AUDCLNT \_ SHAREMODE**](/windows/desktop/api/Audiosessiontypes/ne-audiosessiontypes-audclnt_sharemode)                               | Der Freigabemodus für einen Stream.                                                                     |
| [**AUDCLNT \_ STREAMOPTIONS**](/windows/desktop/api/audioclient/ne-audioclient-audclnt_streamoptions)                       | Definiert Werte, die die Merkmale eines Audiostreams beschreiben.                               |
| [**\_KATEGORIE \_ "AUDIOSTREAM"**](/windows/desktop/api/audiosessiontypes/ne-audiosessiontypes-audio_stream_category)                      | Gibt die Kategorie eines Audiostreams an.                                                         |
| [**AudioSessionState**](/windows/desktop/api/Audiosessiontypes/ne-audiosessiontypes-audiosessionstate)                                | Der Zustand einer Audiositzung.                                                                     |
| [**Connectortype**](/windows/win32/api/devicetopology/ne-devicetopology-connectortype)                                        | Der Typ des Connectors in einer Gerätetopologie.                                                        |
| [**Datenfluss**](/windows/win32/api/devicetopology/ne-devicetopology-dataflow)                                                  | Die Datenflussrichtung eines Audiostreams.                                                        |
| [**EDataFlow**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-edataflow)                                                | Die Richtung, in der Audiodaten zwischen einem Audioendpunktgerät und einer Clientanwendung fließen. |
| [**EndpointFormFactor**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-endpointformfactor)                              | Die allgemeinen physischen Attribute eines Audioendpunktgeräts.                                       |
| [**ERole**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-erole)                                                        | Die Systemrolle eines Audioendpunktgeräts.                                                       |
| [**KSJACK \_ SINK \_ CONNECTIONTYPE**](/windows/win32/api/devicetopology/ne-devicetopology-ksjack_sink_connectiontype)<br/> | Der Verbindungstyp.<br/>                                                                 |
| [**PartType**](/windows/win32/api/devicetopology/ne-devicetopology-parttype)                                                  | Der Teiltyp eines Teils (Connector oder Untereinheit) in einer Gerätetopologie.                               |
