# <a name="pkey_device_devicedesc"></a>Pkey- \_ Gerät \_ devicedebug

Die Eigenschaft " **\_ \_ devicedebug" des pkey-Geräts** enthält die Geräte Beschreibung des Endpunkt Geräts (z. b. "Sprecher").

Der **VT** -Member der **PROPVARIANT** -Struktur ist auf VT \_ LPWSTR festgelegt.

Der **pwszval** -Member der **PROPVARIANT** -Struktur verweist auf eine mit NULL endenden Zeichenfolge mit breit Zeichen, die die Geräte Beschreibung enthält.
