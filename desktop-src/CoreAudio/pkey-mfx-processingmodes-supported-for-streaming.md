# <a name="pkey_mfx_processingmodes_supported_for_streaming-property"></a>PKEY \_ MFX \_ ProcessingModes \_ Supported For Streaming \_ \_ -Eigenschaft

Listet die Vom Moduseffekt APO unterstützten Signalverarbeitungsmodi auf.

PROPVARIANT-Typ (vt): **VT \_ VECTOR** \| **VT \_ LPWSTR**

## <a name="remarks"></a>Hinweise

Diese Liste enthält nur Signalverarbeitungsmodi, in denen der APO das Audiosignal während des Streamings tatsächlich verarbeitet.

Diese Liste enthält keine Signalverarbeitungsmodi, die vom APO nur zu Ermittlungszwecken unterstützt werden.
