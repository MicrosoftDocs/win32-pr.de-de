# <a name="controltyperequiredpatternnotsupported"></a>ControlTypeRequiredPatternNotSupported

## <a name="text"></a>Text

Element mit einem ControlType {0} von sollte {1} unterstützen, dieses Element jedoch nicht.

## <a name="type"></a>Typ

Fehler

## <a name="description"></a>Beschreibung

Die Unterstützung der Benutzeroberflächenautomatisierung des Elements entspricht nicht der Benutzeroberflächenautomatisierung-Spezifikation, da das Element die bereitgestellte Liste von Steuerelementmustern unterstützen sollte, aber nicht. Daher wird dieses Element möglicherweise nicht ordnungsgemäß für Hilfstechnologien verfügbar gemacht, was schwerwiegende Auswirkungen auf die Benutzerfreundlichkeit haben kann.

## <a name="possible-causes"></a>Mögliche Ursachen

-   Unvollständige Benutzeroberflächenautomatisierung Implementierung.
-   Die ControlType-Eigenschaft ist nicht ordnungsgemäß festgelegt.
