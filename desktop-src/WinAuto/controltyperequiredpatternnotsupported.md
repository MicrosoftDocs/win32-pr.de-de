# <a name="controltyperequiredpatternnotsupported"></a>Controltyperequiredpatternnotsupported

## <a name="text"></a>Text

Das Element mit dem ControlType-Element muss {0} unterstützen, {1} dieses Element jedoch nicht.

## <a name="type"></a>type

Fehler

## <a name="description"></a>BESCHREIBUNG

Die Benutzeroberflächenautomatisierungs-Unterstützung des Elements entspricht nicht der Benutzeroberflächenautomatisierungs-Spezifikation, da das-Element die bereitgestellte Liste der Steuerelement Muster unterstützen soll, dies jedoch nicht. Folglich ist dieses Element möglicherweise nicht ordnungsgemäß für Hilfstechnologien verfügbar, was sich schwerwiegende Auswirkungen auf die Benutzer Leistung haben könnte.

## <a name="possible-causes"></a>Mögliche Ursachen

-   Unvollständige Implementierung der Benutzeroberflächen Automatisierung.
-   Die ControlType-Eigenschaft ist nicht ordnungsgemäß festgelegt.
