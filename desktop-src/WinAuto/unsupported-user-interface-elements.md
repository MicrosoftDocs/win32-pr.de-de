# <a name="unsupported-user-interface-elements"></a>Nicht unterstützte Elemente der Benutzeroberfläche

Die in der folgenden Tabelle aufgeführten allgemeinen Steuerelemente werden von Microsoft Active Accessibility nicht unterstützt.



| Steuerelementname                  | Fenster Klassenname   |
|-------------------------------|---------------------|
| Steuerelemente für Datums-und Zeitauswahl | datetimepick- \_ Klasse |
| Pager-Steuerelemente                | WC- \_ pagescroller    |
| Grund leisten-Steuerelemente                | Rebarclassname      |



 

Die in der folgenden Tabelle aufgeführten vordefinierten Steuerelemente und Fensterelemente werden von Microsoft Active Accessibility nicht unterstützt.



| Fenster Klassenname | BESCHREIBUNG                                                 |
|-------------------|-------------------------------------------------------------|
| Ddemlevent        | Windows NT: die-Klasse für Ddeml-Ereignisse                      |
| `Message`           | Windows 2000 und höher: die-Klasse für ein nur-Nachricht-Fenster |
| \#32772           | Windows NT: die Klasse für Symbol Titel                       |
