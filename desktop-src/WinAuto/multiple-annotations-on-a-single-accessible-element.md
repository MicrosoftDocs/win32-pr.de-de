# <a name="multiple-annotations-on-a-single-accessible-element"></a>Mehrere Anmerkungen für ein einzelnes barrierefreies Element

Derzeit unterstützt ein barrierefreies Element nur eine Anmerkung pro Eigenschaft. Nachfolgende Anmerkungen zu einer Eigenschaft ersetzen und überschreiben frühere Anmerkungen dieser Eigenschaft.
