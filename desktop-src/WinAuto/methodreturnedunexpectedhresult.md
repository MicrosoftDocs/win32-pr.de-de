# <a name="methodreturnedunexpectedhresult"></a>MethodReturnedUnexpectedHResult

## <a name="text"></a>Text

{0} zurückgegeben, {1} aber der erwartete Rückgabewert sollte {2}

## <a name="type"></a>type

Warnung

## <a name="description"></a>BESCHREIBUNG

Beim Versuch, alle untergeordneten Elemente des Elements abzurufen, wurde ein unerwartetes HRESULT zurückgegeben.

## <a name="possible-causes"></a>Mögliche Ursachen

Eine falsche oder ungültige MSAA-Implementierung.
