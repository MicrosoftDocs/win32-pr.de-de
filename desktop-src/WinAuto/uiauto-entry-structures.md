# <a name="ui-automation-structures"></a>UI-Automatisierungs Strukturen

In diesem Abschnitt werden die von Benutzeroberflächenautomatisierungs-Client Anwendungen und Benutzeroberflächenautomatisierungs-Anbietern für Microsoft Win32 verwendeten Strukturen beschrieben.

## <a name="in-this-section"></a>In diesem Abschnitt



| Struktur                                                                            | BESCHREIBUNG                                                                                   |
|--------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**ExtendedProperty**](/windows/desktop/api/UIAutomationClient/ns-uiautomationclient-extendedproperty)<br/>                       | Enthält Informationen über eine erweiterte Eigenschaft. <br/>                                  |
| [**Uiautomationeventinfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationeventinfo)<br/>       | Enthält Informationen über ein benutzerdefiniertes Ereignis.<br/>                                         |
| [**Uiautomationmethodinfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationmethodinfo)<br/>     | Enthält Informationen über eine Methode, die von einem benutzerdefinierten Steuerelement Muster unterstützt wird.<br/> |
| [**Uiautomationparameter**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationparameter)<br/>       | Enthält Informationen zu einem Parameter eines benutzerdefinierten Steuerelement Musters.<br/>                |
| [**Uiautomationpatterninfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationpatterninfo)<br/>   | Enthält Informationen über ein benutzerdefiniertes Steuerelement Muster.<br/>                               |
| [**Uiautomationpropertyinfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationpropertyinfo)<br/> | Enthält Informationen zu einer benutzerdefinierten Eigenschaft.<br/>                                      |
| [**Uiachangeingefo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiachangeinfo)<br/>                                    | Enthält Daten zu einer Änderung der Benutzeroberflächenautomatisierungs-Änderung.<br/>                          |
| [**Uiapoint**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiapoint)<br/>                                 | Enthält die Koordinaten eines Punkts. <br/>                                              |
| [**Uiarect**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiarect)<br/>                                   | Enthält die Position und Größe eines Rechtecks in Bildschirm Koordinaten.<br/>              |
