# <a name="getting-started-microsoft-active-accessibility"></a>Getting Started (Microsoft Active Accessibility)

Bei Microsoft Active Accessibility handelt es sich um eine Reihe von Component Object Model (com)-Schnittstellen und API-Elementen, die eine zuverlässige Möglichkeit bieten, Informationen zu auf Microsoft Windows basierenden Benutzeroberflächen Elementen und Webinhalt verfügbar zu machen und zu erfassen Mithilfe dieser Informationen können hilfstechnologiehersteller die Benutzeroberfläche in alternativen Formaten darstellen, wie z. b. Sprache oder Shell, und sprach Befehls-und Steuerungsanwendungen können die Schnittstelle Remote bearbeiten. Microsoft Active Accessibility ist von der Windows-Technologie abhängig und kann nur in Verbindung mit Windows-basierten Steuerelementen und anderen Windows-Anwendungen verwendet werden.

Diese Dokumentation ist so organisiert, dass Sie die Anforderungen von Entwicklern, die mit Microsoft Active Accessibility vertraut sind, erfüllt werden. Die Hauptabschnitte der Dokumentation werden im folgenden beschrieben:

|--------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Technische Übersicht](technical-overview.md)           | Übersicht über Microsoft-Active Accessibility und allgemeine Richtlinien für Microsoft-Active Accessibility Client-und Server Entwickler.                                |
| [Entwicklerhandbuch zu C/C++](c-c---developer-s-guide.md) | Ausführliche Informationen zu den wichtigsten Elementen und Konzepten von Microsoft Active Accessibility Application API. Verwendet Begriffe und Beispiele, die mit C-oder C++-Entwicklern vertraut sind. |
| [C/C++-Referenz](c-c---reference.md)                 | Eine umfassende Referenz für alle Microsoft Active Accessibility-Schnittstellen,-Funktionen,-Datentypen,-Datenstrukturen und-Nachrichten.                           |
| [Anhänge](appendixes.md)                           | Zusätzliches Referenzmaterial für Microsoft Active Accessibility-Client-und Server Entwickler und Microsoft Visual basicdevelopers.                               |
