# <a name="getting-started-microsoft-active-accessibility"></a>Erste Schritte (Microsoft Active Accessibility)

Microsoft Active Accessibility ist ein Satz von Component Object Model-Schnittstellen (COM) und API-Elementen, die eine zuverlässige Möglichkeit bieten, Informationen zu Microsoft Windows-basierten Benutzeroberflächenelementen und Webinhalten verfügbar zu machen und zu sammeln. Anhand dieser Informationen können Hilfstechnologieanbieter die Benutzeroberfläche in alternativen Formaten wie Sprache oder Braille darstellen, und Sprachbefehls- und -steuerungsanwendungen können die Schnittstelle remote bearbeiten. Microsoft Active Accessibility basiert auf Windows-Technologie und kann nur in Verbindung mit Windows-basierten Steuerelementen und anderen Windows-Anwendungen verwendet werden.
