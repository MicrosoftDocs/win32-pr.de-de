# <a name="getting-started-microsoft-active-accessibility"></a>Erste Schritte (Microsoft Active Accessibility)

Microsoft Active Accessibility besteht aus einer Reihe von COM-Schnittstellen (Component Object Model) und API-Elementen, die eine zuverlässige Möglichkeit bieten, Informationen zu Microsoft Windows-basierten Benutzeroberflächenelementen und Webinhalten verfügbar zu machen und zu sammeln. Mithilfe dieser Informationen können Anbieter von Hilfstechnologien die Benutzeroberfläche in alternativen Formaten darstellen, z. B. Sprache oder Braanmeldung, und Sprachbefehls- und Steuerungsanwendungen können die Schnittstelle remote bearbeiten. Microsoft Active Accessibility basiert auf Windows Technologie und kann nur in Verbindung mit Windows-basierten Steuerelementen und anderen Windows Anwendungen verwendet werden.
