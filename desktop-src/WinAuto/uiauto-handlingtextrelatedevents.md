# <a name="handling-text-related-events"></a>Behandeln Text bezogener Ereignisse

In diesem Thema werden die Ereignisse beschrieben, die von textbasierten Steuerelementen ausgelöst werden, wenn der Inhalt geändert, ausgewählt oder deaktiviert wird.


Textbasierte Steuerelemente rufen ein [**UIA \_ Text \_ textchangedebug-**](uiauto-event-ids.md) Ereignis auf, wenn sich der Text Inhalt ändert. Sie rufen das [**UIA \_ Text \_ textselectionchangedebug-**](uiauto-event-ids.md) Ereignis auf, wenn Textinhalte ausgewählt oder deaktiviert werden. Weitere Informationen finden Sie unter [Übersicht über Benutzeroberflächenautomatisierungs-Ereignisse](uiauto-eventsoverview.md) und [Abonnieren von Benutzeroberflächenautomatisierungs-Ereignissen](uiauto-eventsforclients.md).
