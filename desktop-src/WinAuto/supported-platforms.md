# <a name="supported-platforms"></a>Unterstützte Plattformen

Microsoft Active Accessibility unterstützt die meisten Microsoft-Plattformen und ist ab Windows XP als Teil des Windows-Betriebssystems enthalten. Wenn Sie nicht wissen, welche Version von Microsoft Active Accessibility Sie ausführen, finden Sie heraus, [welche Version von Active Accessibility derzeit installiert ist?](which-version-of-active-accessibility-is-currently-installed.md)

Wenn Sie nicht über die neueste Version von Microsoft Active Accessibility verfügen, verwenden Sie das [Microsoft Active Accessibility 2,0 Redistributable Kit (RDK)](https://www.microsoft.com/downloads/details.aspx?familyid=9b14f6e1-888a-4f1d-b1a1-da08ee4077df&displaylang=en) zum Upgrade von. Dieses RDK enthält alle Kernsystem Dateien, die für die Einbindung von Microsoft Active Accessibility-Technologie in Client-und Server Anwendungen benötigt werden. Es unterstützt Windows 98, Windows NT 4,0 mit Service Pack 6 (SP6) und höher, Windows 2000, Windows Millennium Edition (Windows Me), Windows XP und Windows Server 2003.

Informationen zum Installieren des Microsoft Active Accessibility RDK finden Sie [im Microsoft Developer Center für Barrierefreiheit auf MSDN](/windows/apps/accessibility).
