# <a name="supported-platforms"></a>Unterstützte Plattformen

Microsoft Active Accessibility unterstützt die meisten Microsoft-Plattformen und ist als Teil des Windows Betriebssystems ab Windows XP enthalten. Wenn Sie nicht wissen, welche Version von Microsoft Active Accessibility Sie ausführen, finden Sie weitere Informationen unter Welche Version von Active Accessibility [ist derzeit installiert?](which-version-of-active-accessibility-is-currently-installed.md)

Wenn Sie nicht über die neueste Version von Microsoft Active Accessibility verfügen, verwenden Sie das [Microsoft Active Accessibility 2.0 Redistributable Kit (RDK)](https://www.microsoft.com/downloads/details.aspx?familyid=9b14f6e1-888a-4f1d-b1a1-da08ee4077df&displaylang=en) für das Upgrade. Dieses RDK enthält alle Kernsystemdateien, die zum Integrieren Microsoft Active Accessibility-Technologie in Client- und Serveranwendungen erforderlich sind. Sie unterstützt Windows 98, Windows NT 4.0 mit Service Pack 6 (SP6) und höher, Windows 2000, Windows Edition (Windows Me), Windows XP und Windows Server 2003.

Informationen zum Installieren des Microsoft Active Accessibility RDK finden Sie im [Microsoft Accessibility Developer Center auf MSDN.](/windows/apps/accessibility)
