# <a name="custom-properties-events-and-control-patterns"></a>Benutzerdefinierte Eigenschaften, Ereignisse und Steuerelement Muster

Die Microsoft UI Automation-API gibt einen vordefinierten Kernsatz von Eigenschaften, Steuerelement Mustern und Ereignissen an. Anwendungen sind jedoch nicht auf die Verwendung dieser vordefinierten Spezifikationen beschränkt. Mit den Erweiterbarkeits Features der Benutzeroberflächen Automatisierung können Dritte benutzerdefinierte, einvernehmlich vereinbarte Eigenschaften, Ereignisse und Steuerelement Muster einführen, um neue Benutzeroberflächen Elemente und Anwendungs Szenarios zu unterstützen. Benutzeroberflächenautomatisierungs-Anbieter und-Clients können die benutzerdefinierten Eigenschaften, Ereignisse und Steuerelement Muster sofort verwenden, ohne dass das Core UI Automation-Framework aktualisiert werden muss.
