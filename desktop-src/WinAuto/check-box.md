# <a name="check-box-msaa-ui-element-reference"></a>Kontrollkästchen (MSAA-Benutzeroberflächen-Element Referenz)

> [!Note]  
> In diesem Thema werden **Kontrollkästchen** Objekte für den MSAA-Benutzeroberflächen-Element Verweis beschrieben. Das Erstellen von **Kontrollkästchen** Objekten in verschiedenen Benutzeroberflächen-Frameworks wird hier nicht beschrieben. Weitere Informationen finden Sie in der API-Referenz Dokumentation für das von Ihnen verwendete UI-Framework.

 

Kontrollkästchen werden verwendet, um eine oder mehrere Features oder Optionen aus einer Gruppe, normalerweise in einem Dialogfeld, zu aktivieren oder zu deaktivieren. In der Regel enthält ein Kontrollkästchen ein kleines Feld mit einem angrenzenden Text. Wenn eine Option ausgewählt ist, wird im Feld ein Häkchen angezeigt.

Der Fenster Klassenname für ein Kontrollkästchen ist "Schaltfläche".
