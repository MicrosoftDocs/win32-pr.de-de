# <a name="tools"></a>Tools

Sie können die folgenden Tools zum Testen von Microsoft Active Accessibility verwenden. Weitere Informationen zu den Testtools finden Sie unter [Testtools](testing-tools.md).



| Tool                                                     | Beschreibung                                                                                                                                                                           |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Überwachung für barrierefreie Ereignisse](accessible-event-watcher.md) | Protokolliert [WinEvents,](winevents-infrastructure.md) während sie innerhalb des Systems auftreten.                                                                                                         |
| [Überprüfen](inspect-objects.md)                           | Identifiziert Probleme bei der Implementierung von Eigenschaften und Methoden in barrierefreien Objekten von Microsoft Active Accessibility und in Automatisierungselementen von Microsoft Benutzeroberflächenautomatisierung. |
