# <a name="treetoodeep"></a>Treetoodeep

## <a name="text"></a>Text

Struktur ist {0} Tiefe Ebenen.

## <a name="type"></a>type

Warnung

## <a name="description"></a>BESCHREIBUNG

Die Elementstruktur ist möglicherweise zu tief.

Dieses Problem führt dazu, dass Personen, die sich auf einen Bildschirm und eine Tastatur für die Navigation verlassen, dass das Durchlaufen von Elementen lange Zeit in Anspruch nimmt und möglicherweise zyklisch erscheint.

## <a name="possible-causes"></a>Mögliche Ursachen

-   Mehrere Elemente oder ihre übergeordneten Elemente sind benutzerdefinierte Steuerelemente, die die Tabstopps nicht ordnungsgemäß implementieren. Beispielsweise wird die Eigenschaft MSAA State nicht ordnungsgemäß aktualisiert.
-   Die Benutzeroberfläche der Anwendung ist übermäßig Komplex.
