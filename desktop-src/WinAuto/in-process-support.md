# <a name="in-process-support"></a>In-Process Unterstützung

Die aktuelle Implementierung der dynamischen Anmerkung ist vollständig in Verarbeitung und ermöglicht folglich nur das Kommentieren von Benutzeroberflächen Elementen durch den Prozess, der diese Elemente der Benutzeroberfläche besitzt. Es ist nicht möglich, dass ein Prozess die Benutzeroberflächen Elemente eines anderen Prozesses mit Anmerkungen versehen.

Beachten Sie, dass dies nur Auswirkungen auf das Festlegen einer Anmerkung hat. Dies beeinträchtigt nicht die Fähigkeit von Clients, auf die Eigenschaften von Benutzeroberflächen Elementen in anderen Prozessen zuzugreifen (kommentiert oder anderweitig).
