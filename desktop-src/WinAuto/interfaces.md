# <a name="interfaces-active-accessibility-user-interface-services"></a>Schnittstellen (Active Accessibility Benutzeroberfläche Services)

Dieser Abschnitt enthält Informationen zu Schnittstellen, die mit Microsoft Active Accessibility verwendet werden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                | Beschreibung                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Iaccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible)<br/>                                        | Macht Methoden und Eigenschaften verfügbar, die ein Benutzeroberflächenelement und seine untergeordneten Elemente für Clientanwendungen zugänglich machen.<br/>                                                                     |
| [**IAccessibleWindowlessSite**](/windows/desktop/api/oleacc/nn-oleacc-iaccessiblewindowlesssite)<br/> | Eine Microsoft ActiveX-Steuerelementwebsite implementiert diese Schnittstelle, um ein fensterloses ActiveX-Steuerelement zu aktivieren, das über eine Microsoft Active Accessibility Implementierung verfügt, um seine Barrierefreiheit auszudrücken. <br/> |
| [**IAccIdentity**](/windows/desktop/api/oleacc/nn-oleacc-iaccidentity)<br/>                                      | Macht eine Methode verfügbar, die einen eindeutigen Bezeichner für ein barrierefreies Element bereitstellt.<br/>                                                                                                             |
| [**Iaccpropserver**](/windows/desktop/api/oleacc/nn-oleacc-iaccpropserver)<br/>                                  | Macht eine Methode verfügbar, die einen Eigenschaftswert für ein barrierefreies Element abruft.<br/>                                                                                                               |
| [**IAccPropServices**](/windows/desktop/api/oleacc/nn-oleacc-iaccpropservices)<br/>                              | Macht Methoden zum Kommentieren barrierefreier Elemente und zum Bearbeiten von Identitätszeichenfolgen verfügbar. <br/>                                                                                                |
| [**Iaccessiblehandler**](/windows/desktop/api/oleacc/nn-oleacc-iaccessiblehandler)<br/>                          | Macht eine Methode verfügbar, die ein barrierefreies Element aus einer Objekt-ID abruft.<br/>                                                                                                                  |
