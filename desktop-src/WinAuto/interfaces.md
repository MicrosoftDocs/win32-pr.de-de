# <a name="interfaces-active-accessibility-user-interface-services"></a>Schnittstellen (Active Accessibility Benutzeroberflächen Dienste)

Dieser Abschnitt enthält Informationen zu den Schnittstellen, die mit Microsoft Active Accessibility verwendet werden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                | BESCHREIBUNG                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible)<br/>                                        | Macht Methoden und Eigenschaften verfügbar, durch die Client Anwendungen auf ein Benutzeroberflächen Element und seine untergeordneten Elemente zugreifen können.<br/>                                                                     |
| [**Iaccessiblewindowlesssite**](/windows/desktop/api/oleacc/nn-oleacc-iaccessiblewindowlesssite)<br/> | Eine Microsoft-ActiveX-Steuerungs Site implementiert diese Schnittstelle, um ein fensterloses ActiveX-Steuerelement zu aktivieren, das über eine Microsoft Active Accessibility-Implementierung verfügt, um seine Barrierefreiheit auszudrücken <br/> |
| [**Iaccidentity**](/windows/desktop/api/oleacc/nn-oleacc-iaccidentity)<br/>                                      | Macht eine Methode verfügbar, die einen eindeutigen Bezeichner für ein barrierefreies Element bereitstellt.<br/>                                                                                                             |
| [**IAccPropServer**](/windows/desktop/api/oleacc/nn-oleacc-iaccpropserver)<br/>                                  | Macht eine Methode verfügbar, die einen Eigenschafts Wert für ein barrierefreies Element abruft.<br/>                                                                                                               |
| [**IAccPropServices**](/windows/desktop/api/oleacc/nn-oleacc-iaccpropservices)<br/>                              | Macht Methoden zum kommentieren barrierefreier Elemente und zum Bearbeiten von Identitäts Zeichenfolgen verfügbar. <br/>                                                                                                |
| [**IAccessibleHandler**](/windows/desktop/api/oleacc/nn-oleacc-iaccessiblehandler)<br/>                          | Macht eine Methode verfügbar, die ein barrierefreies Element aus einer Objekt-ID abruft.<br/>                                                                                                                  |
