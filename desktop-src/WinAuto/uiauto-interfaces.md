# <a name="interfaces-for-providers"></a>Schnittstellen für Anbieter

Dieser Abschnitt beschreibt die grundlegenden Schnittstellen, die von Benutzeroberflächenautomatisierungs-Anbietern für Microsoft Win32-Anwendungen implementiert

## <a name="in-this-section"></a>In diesem Abschnitt



| Schnittstelle                                                                                                    | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|--------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAccessibleEx**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iaccessibleex)<br/>                                                     | Macht Methoden verfügbar, die von Microsoft UI Automation aufgerufen werden, um zusätzliche Informationen zu einem Steuerelement abzurufen, das Microsoft Active Accessibility unterstützt.<br/>                                                                                                                                                                                                                                                                                                                |
| [**Iaccessiblehostingelementproviders**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iaccessiblehostingelementproviders)<br/>                  | Ein Microsoft Active Accessibility-Objekt implementiert diese Schnittstelle, wenn das Objekt der Stamm einer Barrierefreiheits Struktur ist, die fensterlose Microsoft-ActiveX-Steuerelemente enthält, die die Benutzeroberflächen Automatisierung implementieren. Da die Automatisierung von Microsoft Active Accessibility und der Benutzeroberfläche verschiedene Schnittstellen verwendet, ermöglicht diese Schnittstelle einem Client, die Liste der gehosteten fensterlosen ActiveX-Steuerelemente zu ermitteln, die die Benutzeroberflächen Automatisierung unterstützen, falls der Client diese anders behandeln muss.<br/> |
| [**Iproxyproviderwineventhandler**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iproxyproviderwineventhandler)<br/>                     | Macht eine Methode verfügbar, die von Proxy Anbietern zum Verarbeiten von WinEvents implementiert wird.<br/>                                                                                                                                                                                                                                                                                                                                                                                          |
| [**Iproxyproviderwineventsink**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iproxyproviderwineventsink)<br/>                           | Macht Methoden verfügbar, mit denen Proxy Anbieter Ereignisse auswerfen.<br/>                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**Irawelementprovideradvil-Ereignisse**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovideradviseevents)<br/>                 | Macht Methoden verfügbar, die aufgerufen werden, um das Stamm Element eines Fragments zu benachrichtigen, wenn eine Benutzeroberflächenautomatisierungs-Client Anwendung mit dem lauschen auf Ereignisse in diesem Fragment beginnt oder endet.<br/>                                                                                                                                                                                                                                                                                                |
| [**IRawElementProviderFragment**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragment)<br/>                         | Macht Methoden und Eigenschaften für Benutzeroberflächen Elemente verfügbar, die Teil einer Struktur sind, die mehr als eine Ebene tief ist, z. b. ein Listenfeld oder Listenelement. Wird vom Benutzeroberflächenautomatisierungs-Anbieter implementiert.<br/>                                                                                                                                                                                                                                                                                          |
| [**IRawElementProviderFragmentRoot**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragmentroot)<br/>                 | Macht Methoden und Eigenschaften des Stammelements in einem Fragment verfügbar.<br/>                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**Irawelementproviderhostingaccessibles**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-irawelementproviderhostingaccessibles)<br/> | Diese Schnittstelle wird von einem Benutzeroberflächenautomatisierungs-Anbieter implementiert, wenn der Anbieter der Stamm einer Barrierefreiheits Struktur ist, die fensterlose Steuerelemente enthält, die Microsoft-Active Accessibility unterstützen. Da die Benutzeroberflächen Automatisierung und Microsoft Active Accessibility unterschiedliche Schnittstellen verwenden, ermöglicht diese Schnittstelle einem Client, die Liste der gehosteten Microsoft Active Accessibility-Steuerelemente zu ermitteln, falls diese anders behandelt werden müssen.<br/>                                 |
| [**IRawElementProviderHwndOverride**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderhwndoverride)<br/>                 | Macht eine Methode verfügbar, die das Neupositionieren von Fenster basierten Elementen innerhalb der Benutzeroberflächenautomatisierungs-Struktur des Fragments ermöglicht.<br/>                                                                                                                                                                                                                                                                                                                                                        |
| [**IRawElementProviderSimple**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple)<br/>                             | Definiert Methoden und Eigenschaften, die einfache Benutzeroberflächen Elemente verfügbar machen.<br/>                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**IRawElementProviderSimple2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple2)<br/>                           | Erweitert die [**IRawElementProviderSimple**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple) -Schnittstelle, um das programmgesteuerte Aufrufen von Kontextmenüs zu ermöglichen.<br/>                                                                                                                                                                                                                                                                                                                        |
| [**IRawElementProviderSimple3**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple3)<br/>                           | Erweitert die [**IRawElementProviderSimple2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple2) -Schnittstelle, um das Abrufen von Metadaten zu ermöglichen, wie barrierefreie Technologie den bevorzugten Inhaltstyp sagen soll.<br/>                                                                                                                                                                                                                                                                    |
| [**Irawelementproviderwindowlesssite**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-irawelementproviderwindowlesssite)<br/>         | Diese Schnittstelle wird von einer ActiveX-Steuerelement Site implementiert, damit ein ActiveX-Steuerelement für die Benutzeroberflächen Automatisierung seine Barrierefreiheit Ausdrücken kann. Diese Schnittstelle ermöglicht es dem Steuerelement Container, einen [**IRawElementProviderFragment**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragment) -Zeiger für das übergeordnete Element oder das fensterlose ActiveX-Steuerelement bereitzustellen und eine Lauf Zeit-ID bereitzustellen, die für die Steuerelement Website eindeutig ist.<br/>                                                           |
