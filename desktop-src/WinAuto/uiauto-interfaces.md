# <a name="interfaces-for-providers"></a>Schnittstellen für Anbieter

In diesem Abschnitt werden die grundlegenden Schnittstellen beschrieben, die von Benutzeroberflächenautomatisierung-Anbietern für Microsoft Win32-Anwendungen implementiert werden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Schnittstelle                                                                                                    | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|--------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAccessibleEx**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iaccessibleex)<br/>                                                     | Macht Methoden verfügbar, die von Microsoft Benutzeroberflächenautomatisierung aufgerufen werden, um zusätzliche Informationen zu einem Steuerelement abzurufen, das Microsoft Active Accessibility unterstützt.<br/>                                                                                                                                                                                                                                                                                                                |
| [**IAccessibleHostingElementProviders**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iaccessiblehostingelementproviders)<br/>                  | Ein Microsoft Active Accessibility-Objekt implementiert diese Schnittstelle, wenn das -Objekt der Stamm einer Barrierefreiheitsstruktur ist, die fensterlose Microsoft ActiveX-Steuerelemente enthält, die Benutzeroberflächenautomatisierung implementieren. Da Microsoft Active Accessibility und Benutzeroberflächenautomatisierung verschiedene Schnittstellen verwenden, ermöglicht diese Schnittstelle einem Client, die Liste der gehosteten fensterlosen ActiveX-Steuerelemente zu ermitteln, die Benutzeroberflächenautomatisierung unterstützen, falls der Client sie anders behandeln muss.<br/> |
| [**IProxyProviderWinEventHandler**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iproxyproviderwineventhandler)<br/>                     | Macht eine Methode verfügbar, die von Proxyanbietern implementiert wird, um WinEvents zu verarbeiten.<br/>                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IProxyProviderWinEventSink**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iproxyproviderwineventsink)<br/>                           | Macht Methoden verfügbar, die von Proxyanbietern zum Auslösen von Ereignissen verwendet werden.<br/>                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**IRawElementProviderAdviseEvents**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovideradviseevents)<br/>                 | Macht Methoden verfügbar, die aufgerufen werden, um das Stammelement eines Fragments zu benachrichtigen, wenn eine Benutzeroberflächenautomatisierung Clientanwendung beginnt oder beendet, die auf Ereignisse auf diesem Fragment lauscht.<br/>                                                                                                                                                                                                                                                                                                |
| [**IRawElementProviderFragment**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragment)<br/>                         | Macht Methoden und Eigenschaften für Benutzeroberflächenelemente verfügbar, die Teil einer Struktur sind, die mehr als eine Ebene tief ist, z. B. ein Listenfeld oder Listenelement. Wird von Benutzeroberflächenautomatisierung Anbieter implementiert.<br/>                                                                                                                                                                                                                                                                                          |
| [**IRawElementProviderFragmentRoot**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragmentroot)<br/>                 | Macht Methoden und Eigenschaften des Stammelements in einem Fragment verfügbar.<br/>                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**IRawElementProviderHostingAccessibles**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-irawelementproviderhostingaccessibles)<br/> | Diese Schnittstelle wird von einem Benutzeroberflächenautomatisierung Anbieter implementiert, wenn der Anbieter der Stamm einer Barrierefreiheitsstruktur ist, die fensterlose Steuerelemente enthält, die Microsoft Active Accessibility unterstützen. Da Benutzeroberflächenautomatisierung und Microsoft Active Accessibility unterschiedliche Schnittstellen verwenden, ermöglicht diese Schnittstelle einem Client, die Liste der gehosteten Microsoft Active Accessibility Steuerelemente zu ermitteln, falls sie anders behandelt werden müssen.<br/>                                 |
| [**IRawElementProviderHwndOverride**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderhwndoverride)<br/>                 | Macht eine Methode verfügbar, die die Neupositionierung fensterbasierter Elemente innerhalb der Benutzeroberflächenautomatisierung Struktur des Fragments ermöglicht.<br/>                                                                                                                                                                                                                                                                                                                                                        |
| [**Irawelementprovidersimple**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple)<br/>                             | Definiert Methoden und Eigenschaften, die einfache Benutzeroberflächenelemente verfügbar machen.<br/>                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**IRawElementProviderSimple2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple2)<br/>                           | Erweitert die [**IRawElementProviderSimple-Schnittstelle,**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple) um das programmgesteuerte Aufrufen von Kontextmenüs zu ermöglichen.<br/>                                                                                                                                                                                                                                                                                                                        |
| [**IRawElementProviderSimple3**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple3)<br/>                           | Erweitert die [**IRawElementProviderSimple2-Schnittstelle,**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple2) um das Abrufen von Metadaten darüber zu ermöglichen, wie barrierefreie Technologien den bevorzugten Inhaltstyp angeben sollten.<br/>                                                                                                                                                                                                                                                                    |
| [**IRawElementProviderWindowlessSite**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-irawelementproviderwindowlesssite)<br/>         | Ein ActiveX-Steuerelementstandort implementiert diese Schnittstelle, um einem Benutzeroberflächenautomatisierung-fähigen ActiveX-Steuerelement zu ermöglichen, seine Barrierefreiheit auszudrücken. Diese Schnittstelle ermöglicht es dem Steuerelementcontainer, einen [**IRawElementProviderFragment-Zeiger**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragment) für das übergeordnete oder gleichgeordnete Element des fensterlosen ActiveX-Steuerelements bereitzustellen und eine Laufzeit-ID bereitzustellen, die für die Steuerelementsite eindeutig ist.<br/>                                                           |
