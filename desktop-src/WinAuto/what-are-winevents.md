<span data-ttu-id="299b4-118">Um z. b. <span data-ttu-id="299b4-121">Server wenden [**NotifyWinEvent**](/windows/desktop/api/Winuser/nf-winuser-notifywinevent) an, um das Ereignis für das System anzukündigen, nachdem das Ereignis aufgetreten ist.</span><span class="sxs-lookup"><span data-stu-id="299b4-121">Servers call [**NotifyWinEvent**](/windows/desktop/api/Winuser/nf-winuser-notifywinevent) to announce the event to the system after the event has occurred.</span></span> <span data-ttu-id="299b4-122">Sie müssen das System eines Ereignisses niemals Benachrichtigen, bevor das Ereignis auftritt.</span><span class="sxs-lookup"><span data-stu-id="299b4-122">They must never notify the system of an event before the event occurs.</span></span>

<span data-ttu-id="299b4-123">Um über Ereignisse benachrichtigt zu werden, registrieren Clients Rückruf-Hook-Funktionen mithilfe von [**setwineventhook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook).</span><span class="sxs-lookup"><span data-stu-id="299b4-123">To be notified of events, clients register callback hook functions by using [**SetWinEventHook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook).</span></span> <span data-ttu-id="299b4-124">Clients legen eine einzelne Hook-Funktion für alle möglichen Ereignisse oder mehrere Hook-Funktionen für diskrete Bereiche von Ereignissen fest.</span><span class="sxs-lookup"><span data-stu-id="299b4-124">Clients set a single hook function for all possible events or multiple hook functions for discrete ranges of events.</span></span> <span data-ttu-id="299b4-125">Weitere Informationen finden Sie unter [Registrieren einer Hookfunktion](registering-a-hook-function.md).</span><span class="sxs-lookup"><span data-stu-id="299b4-125">For more information, see [Registering a Hook Function](registering-a-hook-function.md).</span></span>
