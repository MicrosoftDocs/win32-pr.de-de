# <a name="deprecated-guids"></a>Als veraltet markierte GUIDs

-   [PatternId-GUIDs](#patternid-guids)
-   [Controltypeid-GUIDs](#controltypeid-guids)
-   [EventID-GUIDs](#eventid-guids)
-   [PropertyId-GUIDs](#propertyid-guids)
