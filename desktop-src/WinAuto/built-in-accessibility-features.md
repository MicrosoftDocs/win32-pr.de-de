# <a name="built-in-accessibility-features"></a>Integrierte Barrierefreiheits Features

Die integrierten Barrierefreiheits Features sind Erweiterungen des Betriebssystems. Diese Features wirken sich auf das Verhalten des Systems aus, unabhängig davon, welche Anwendungen ausgeführt werden.

Win32 unterstützt die folgenden integrierten Barrierefreiheits Features.



| Funktion       | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|---------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Accesstimeout | Ermöglicht Benutzern das Angeben eines Timeout Intervalls, nach dem systemweite Barrierefreiheits Features automatisch deaktiviert werden. Die Funktion "accesstimeout" ist für Computer vorgesehen, die von mehreren Benutzern mit unterschiedlichen Einstellungen gemeinsam genutzt werden. Jede Person kann mithilfe von Abkürzungs Schlüsseln oder dem Easy Access Center in der Systemsteuerung bevorzugte Funktionen aktivieren. Der Timeout Zeitraum für die Barrierefreiheit ist die Zeitspanne, die ohne Tastatur-und Maus Eingaben bestanden werden muss, bevor das Betriebssystem die Barrierefreiheits Funktionen automatisch deaktiviert. Die von der Zeitüberschreitung betroffenen Barrierefreiheits Funktionen sind FilterKeys, MouseKeys, StickyKeys und "degglekeys" und ein [hoher Kontrast](high-contrast-parameter.md). Die [**accesstimeout**](/windows/win32/api/winuser/ns-winuser-accesstimeout) -Struktur definiert die Parameter für diese Funktion. |
| Filter Keys    | Ermöglicht das Steuern von Tastatur Eigenschaften, z. b. die Zeitspanne, bevor eine Tastatureingabe als Eingabe akzeptiert wird, und die Zeitspanne bis zum Beginn der Wiederholung einer Tastatureingabe. Die Funktion FilterKeys bietet auch Audiofeedback, wenn eine Taste gedrückt wird, und wenn Sie als Eingabe akzeptiert wird. Die [**FilterKeys**](/windows/win32/api/winuser/ns-winuser-filterkeys) -Struktur definiert die Parameter für diese Funktion.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Maustasten     | Ermöglicht es dem Benutzer, den Mauszeiger mit der numerischen Tastatur zu steuern. Die [**mouskeys**](/windows/win32/api/winuser/ns-winuser-mousekeys) -Struktur definiert die Parameter für diese Funktion.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Serialkeys    | Ermöglicht dem Benutzer das Bereitstellen von Eingaben mithilfe eines alternativen Eingabe Geräts, das mit dem seriellen Anschluss des Computers verbunden ist. Diese Funktion ist für Benutzer konzipiert, die die Standardtastatur und-Maus des Computers nicht verwenden können. Die [**serialkeys**](/windows/win32/api/winuser/ns-winuser-serialkeysa) -Struktur definiert die Parameter für diese Funktion.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| Hat Dienst   | Zeigt ein visuelles Signal an, wenn ein Sound von einer Microsoft Win32-basierten Anwendung oder einer Microsoft-MS-DOS-Anwendung generiert wird, die in einem Fenster ausgeführt wird. Die Struktur [**Entry**](/windows/win32/api/winuser/ns-winuser-soundsentrya) -Struktur definiert die Parameter für diese Funktion, einschließlich der Adresse der Funktion " [*soundsentryproc*](/previous-versions/windows/desktop/legacy/dd373647(v=vs.85)) Callback".                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| "StickyKeys"    | Ermöglicht es dem Benutzer, Tastenkombinationen, z. b. STRG + ALT + ENTF, nacheinander und nicht gleichzeitig einzugeben. Die " [**StickyKeys**](/windows/win32/api/winuser/ns-winuser-stickykeys) "-Struktur definiert die Parameter für diese Funktion.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| "-Schlüssel"    | Bietet Sound Feedback, wenn der Benutzer die Feststell Taste, die Num-Sperre oder die scrollsperrtaste schaltet. Die [**ToggleKeys**](/windows/win32/api/winuser/ns-winuser-togglekeys) -Struktur definiert die Parameter für diese Funktion.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |



 

Die [**SystemParametersInfo**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) -Funktion stellt die Programmierschnittstelle für die integrierten Barrierefreiheits Funktionen bereit. Diese Funktion ist in System Steuerungsanwendungen oder anderen Programmen nützlich, die es dem Benutzer ermöglichen, die Umgebung anzupassen. Die **SystemParametersInfo** -Funktion kann die folgenden Aufgaben ausführen:

-   Fragen Sie das System ab, um die Verfügbarkeit und den aktuellen Status eines angegebenen integrierten Barrierefreiheits Features zu ermitteln.
-   Aktivieren Sie eine Funktion, und geben Sie Parameter an, die das Verhalten steuern.
-   Deaktivieren Sie eine Funktion.
-   Aktivieren Sie einen Hot Key, mit dem der Benutzer eine Funktion aktivieren oder deaktivieren kann.

Für alle integrierten Features definiert Win32 eine Struktur, die von Anwendungen mit [**SystemParametersInfo**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) verwendet wird, um die Parameter des Features abzufragen oder festzulegen.

Eine Anwendung kann [**SystemParametersInfo**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) verwenden, um eine Funktion direkt zu aktivieren oder zu deaktivieren. Für einige Funktionen kann **SystemParametersInfo** eine Tastenkombination oder eine Tastenkombination aktivieren, damit der Benutzer den Status einer Funktion umschalten kann. Es sind keine Tastenkombinationen für die Funktionen "serialkeys" oder "SoundSentry" oder für den "ShowSounds"-Parameter vorhanden. Die folgenden Barrierefreiheits Features haben heiße Schlüssel.



| Funktion                                           | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|---------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Filter Keys                                        | Halten Sie die UMSCHALTTASTE gedrückt, um die Filter Keys-Funktion ein-oder auszuschalten. Wenn Sie die Rechte UMSCHALTTASTE für 12 Sekunden gedrückt halten, wird FilterKeys mit den meisten konservativen Werten für den **ibouncemsec** -Member der [**FilterKeys**](/windows/win32/api/winuser/ns-winuser-filterkeys) -Struktur verwendet. Wenn Sie die Rechte UMSCHALTTASTE für 16 Sekunden gedrückt halten, kann FilterKeys die meisten konservativen Werte für das **iwaitmsec** -Element der **FilterKeys** -Struktur verwenden. |
| Maustasten                                         | Drücken Sie die linke ALT + linke UMSCHALT + NUM-Sperre.                                                                                                                                                                                                                                                                                                                                                                                                                  |
| "StickyKeys"                                        | Drücken Sie die UMSCHALTTASTE fünfmal.                                                                                                                                                                                                                                                                                                                                                                                                                             |
| "-Schlüssel"                                        | Halten Sie die Num-Taste 5 Sekunden gedrückt.                                                                                                                                                                                                                                                                                                                                                                                                                |
| [Modus für hohen Kontrast](high-contrast-parameter.md) | Drücken Sie nach links alt + nach-links-Taste + Druck.                                                                                                                                                                                                                                                                                                                                                                                                              |
