# <a name="built-in-accessibility-features"></a>Integrierte Barrierefreiheitsfunktionen

Die integrierten Barrierefreiheitsfunktionen sind Erweiterungen des Betriebssystems. Diese Features beeinflussen das Verhalten des Systems, unabhängig davon, welche Anwendungen ausgeführt werden.

Win32 unterstützt die folgenden integrierten Barrierefreiheitsfunktionen.



| Feature       | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|---------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| AccessTimeout | Ermöglicht einem Benutzer das Angeben eines Time out-Intervalls, nach dem systemweite Barrierefreiheitsfunktionen automatisch deaktiviert werden. Das AccessTimeout-Feature ist für Computer vorgesehen, die von mehreren Benutzern mit unterschiedlichen Einstellungen gemeinsam genutzt werden. Jede Person kann hot keys oder die Center für erleicherte Bedienung in Systemsteuerung verwenden, um bevorzugte Features zu aktivieren. Der Time out-Zeitraum für die Barrierefreiheit ist die Zeitspanne, die ohne Tastatur- und Mauseingabe verstreichen muss, bevor das Betriebssystem die Barrierefreiheitsfunktionen automatisch deaktiviert. Die vom Time out betroffenen Barrierefreiheitsfunktionen sind FilterKeys, MouseKeys, StickyKeys und ToggleKeys sowie [hoher Kontrast.](high-contrast-parameter.md) Die [**ACCESSTIMEOUT-Struktur**](/windows/win32/api/winuser/ns-winuser-accesstimeout) definiert die Parameter für dieses Feature. |
| FilterKeys    | Ermöglicht die Steuerung von Tastatureigenschaften, z. B. die Zeitspanne, bevor eine Tastatureingabe als Eingabe akzeptiert wird, und die Zeitspanne, bevor eine Tastatureingabe wiederholt wird. Das FilterKeys-Feature bietet auch ein fundiertes Feedback, wenn eine Taste gedrückt wird und als Eingabe akzeptiert wird. Die [**FILTERKEYS-Struktur**](/windows/win32/api/winuser/ns-winuser-filterkeys) definiert die Parameter für dieses Feature.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Mousekeys     | Ermöglicht dem Benutzer, den Mauszeiger mithilfe der numerischen Tastatur zu steuern. Die [**MOUSEKEYS-Struktur**](/windows/win32/api/winuser/ns-winuser-mousekeys) definiert die Parameter für dieses Feature.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| SerialKeys    | Ermöglicht dem Benutzer das Bereitstellen von Eingaben mithilfe eines alternativen Eingabegeräts, das an den seriellen Anschluss des Computers angefügt ist. Dieses Feature ist für Benutzer konzipiert, die die Standardtastatur und -maus des Computers nicht verwenden können. Die [**SERIALKEYS-Struktur**](/windows/win32/api/winuser/ns-winuser-serialkeysa) definiert die Parameter für dieses Feature.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| SoundSentry   | Zeigt ein visuelles Signal an, wenn ein Sound von einer Microsoft Win32-basierten Anwendung oder einer Microsoft MS-DOS-Anwendung generiert wird, die in einem Fenster ausgeführt wird. Die [**SOUNDSENTRY-Struktur**](/windows/win32/api/winuser/ns-winuser-soundsentrya) definiert die Parameter für dieses Feature, einschließlich der Adresse der [*Rückruffunktion SoundSentryProc.*](/previous-versions/windows/desktop/legacy/dd373647(v=vs.85))                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| StickyKeys    | Ermöglicht es dem Benutzer, Tastenkombinationen wie STRG+ALT+ENTF nacheinander und nicht gleichzeitig einzugeben. Die [**STICKYKEYS-Struktur**](/windows/win32/api/winuser/ns-winuser-stickykeys) definiert die Parameter für dieses Feature.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| ToggleKeys    | Gibt ein fundiertes Feedback, wenn der Benutzer die CAPS-, NUM- oder SCROLL-SPERRTASTE ein- oder ausschaltet. Die [**TOGGLEKEYS-Struktur**](/windows/win32/api/winuser/ns-winuser-togglekeys) definiert die Parameter für dieses Feature.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |



 

Die [**SystemParametersInfo-Funktion**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) stellt die Programmierschnittstelle für die integrierten Barrierefreiheitsfunktionen bereit. Diese Funktion ist nützlich bei Systemsteuerung Anwendungen oder anderen Programmen, die es dem Benutzer ermöglichen, die Umgebung anzupassen. Die **SystemParametersInfo-Funktion** kann die folgenden Aufgaben ausführen:

-   Fragen Sie das System ab, um die Verfügbarkeit und den aktuellen Status eines angegebenen integrierten Barrierefreiheitsfeatures zu ermitteln.
-   Aktivieren Sie ein Feature, und geben Sie Parameter an, die das Verhalten steuern.
-   Deaktivieren sie ein Feature.
-   Aktivieren Sie einen Hot-Key, der es dem Benutzer ermöglicht, ein Feature zu aktivieren oder zu deaktivieren.

Für jedes der integrierten Features definiert Win32 eine Struktur, die Anwendungen mit [**SystemParametersInfo**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) verwenden, um die Parameter des Features abzufragen oder festzulegen.

Eine Anwendung kann [**SystemParametersInfo**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) verwenden, um ein Feature direkt zu aktivieren oder zu deaktivieren. Für einige der Features kann **SystemParametersInfo** eine Tastenkombination aktivieren, damit der Benutzer den Status eines Features umschalten kann. Für die SerialKeys- oder SoundSentry-Features oder für den ShowSounds-Parameter sind keine Hot Keys vorhanden. Die folgenden Barrierefreiheitsfunktionen verfügen über Hot-Keys.



| Feature                                           | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|---------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| FilterKeys                                        | Halten Sie die RIGHT SHIFT-Taste acht Sekunden lang gedrückt, um das FilterKeys-Feature ein- oder auszuschalten. Wenn Sie die RIGHT SHIFT-Taste 12 Sekunden lang gedrückt halten, können FilterKeys die konservativesten Werte für das **iBounceMSec-Element** der [**FILTERKEYS-Struktur**](/windows/win32/api/winuser/ns-winuser-filterkeys) verwenden. Wenn Sie die RIGHT SHIFT-Taste 16 Sekunden lang gedrückt halten, können FilterKeys die konservativesten Werte für das **iWaitMSec-Element** der **FILTERKEYS-Struktur** verwenden. |
| Mousekeys                                         | Drücken Sie LINKE ALT+NACH-LINKS-UMSCHALT+NUM-SPERRE.                                                                                                                                                                                                                                                                                                                                                                                                                  |
| StickyKeys                                        | Drücken Sie fünfmal die UMSCHALTTASTE.                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ToggleKeys                                        | Halten Sie den NUM-LOCK-Schlüssel 5 Sekunden lang gedrückt.                                                                                                                                                                                                                                                                                                                                                                                                                |
| [Modus für hohen Kontrast](high-contrast-parameter.md) | Drücken Sie LINKS ALT + LINKE UMSCHALTTASTE + DRUCKBILDSCHIRM.                                                                                                                                                                                                                                                                                                                                                                                                              |
