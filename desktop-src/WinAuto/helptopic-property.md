# <a name="helptopic-property"></a>HelpTopic-Eigenschaft

Die **HelpTopic** -Eigenschaft ruft Details ab, die mit der Microsoft Win32 [WinHelp](/windows/win32/api/winuser/nf-winuser-winhelpa) -Funktion verwendet werden, um ein Hilfethema anzuzeigen, das dem Objekt zugeordnet ist. Diese Eigenschaft wird nicht von allen Objekten unterstützt.

Die **HelpTopic** -Eigenschaft wird durch Aufrufen von [**IAccessible:: get \_ accHelpTopic**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_acchelptopic)abgerufen.
