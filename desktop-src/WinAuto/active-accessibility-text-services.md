# <a name="active-accessibility-text-services"></a>Active Accessibility Text Dienste

In diesem Abschnitt werden die Schnittstellen der Active Accessibility Text Dienste erläutert.

> [!Note]  
> Active Accessibility Text Dienste sind veraltet. Weitere Informationen zu erweiterten Text Eingaben und Technologien für natürliche Sprache finden Sie im [Microsoft Windows-Text Dienst-Framework](../tsf/text-services-framework.md) .

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema                                                     | BESCHREIBUNG                                                                                                                    |
|-----------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [**Iaccserverdocmgr**](/windows/desktop/api/msaatext/nn-msaatext-iaccserverdocmgr)   | Macht Methoden verfügbar, die für Client Anwendungen auf die Dokumente zugreifen können.                              |
| [**Iaccclientdocmgr**](/windows/desktop/api/msaatext/nn-msaatext-iaccclientdocmgr)   | Macht Methoden verfügbar, mit denen Client Anwendungen Dokumente abrufen können.                                      |
| [**Iaccdictionary**](/windows/desktop/api/msaatext/nn-msaatext-iaccdictionary)       | Macht Methoden für die Zeichen folgen Bearbeitung verfügbar.                                                            |
| [**Icokreatelocal**](/windows/desktop/api/msaatext/nn-msaatext-icocreatelocally)   | Macht eine Methode verfügbar, die es einer Client Anwendung ermöglicht, ein Hilfsobjekt im Server Kontext zu erstellen. |
| [**Icokreatedlokal**](/windows/desktop/api/msaatext/nn-msaatext-icocreatedlocally) | Macht eine Methode verfügbar, um Informationen über ein lokales Objekt zurückzugeben.                                        |
| [**Iversioninfo**](/windows/desktop/api/msaatext/nn-msaatext-iversioninfo)           | Macht Methoden verfügbar, die Versionsinformationen für barrierefreie Elemente bereitstellen.                            |
