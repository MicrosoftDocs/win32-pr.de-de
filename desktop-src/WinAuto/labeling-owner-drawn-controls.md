<span data-ttu-id="8fd79-104">Anwendungsentwickler sollten für alle Steuerelemente, die vom Besitzer gezeichnet werden, eine Bezeichnung definieren, die auch als Beschriftungs Text bezeichnet wird, auch wenn die Bezeichnung nicht auf dem Bildschirm sichtbar ist.</span><span class="sxs-lookup"><span data-stu-id="8fd79-104">Application developers should define a label, also known as caption text, for all owner-drawn controls, even if the label is not visible on the screen.</span></span> <span data-ttu-id="8fd79-105">Die Bezeichnung muss direkt vor dem Steuerelement stehen, das Sie kennzeichnen, und muss sich im nächsten Element in der Aktivier Reihenfolge befinden.</span><span class="sxs-lookup"><span data-stu-id="8fd79-105">The label must come immediately before the control that it is labeling, and must be in the next item in the tab order.</span></span>

<span data-ttu-id="8fd79-106">Wenn Sie z. b. eine Bezeichnung definieren, wenn Sie eine von einem Besitzer gezeichnete Schaltfläche erstellen, in der anstelle des normalen Beschriftungs Texts eine Grafik angezeigt wird, erhalten Client Anwendungen die Bezeichnung und verwenden Sie, um das Steuerelement zu identifizieren.</span><span class="sxs-lookup"><span data-stu-id="8fd79-106">For example, if you define a label when creating an owner-drawn button in which a graphic is displayed instead of the normal caption text, client applications get the label and use it to identify the control.</span></span>
