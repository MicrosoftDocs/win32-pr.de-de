# <a name="elementshouldbeoffscreen"></a>Elementschuldbeoffscreen

## <a name="text"></a>Text

Element muss Offscreen sein

## <a name="type"></a>type

Warnung

## <a name="description"></a>BESCHREIBUNG

Das umgebende Rechteck des Elements wird durch das umgebende Rechteck des übergeordneten Elements abgeschnitten. Das-Element sollte ignoriert und außerhalb der Grenzen des Desktops berücksichtigt werden.
