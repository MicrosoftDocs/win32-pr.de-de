# <a name="elementshouldbeoffscreen"></a>ElementShouldBeOffScreen

## <a name="text"></a>Text

Element sollte im Offscreenmodus sein

## <a name="type"></a>Typ

Warnung

## <a name="description"></a>Beschreibung

Das umgebundene Rechteck des Elements wird durch das umgebundene Rechteck des übergeordneten Elements abgeschnitten. Das -Element sollte ignoriert und außerhalb der Grenzen des Desktops berücksichtigt werden.
