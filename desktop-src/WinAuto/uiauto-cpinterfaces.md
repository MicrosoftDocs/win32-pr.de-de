# <a name="control-pattern-interfaces-for-providers"></a>Steuerelement Muster-Schnittstellen für Anbieter

In diesem Abschnitt werden *Steuerungs Muster* Schnittstellen beschrieben, die von Benutzeroberflächenautomatisierungs-Anbietern implementiert werden, um Informationen zu Steuerelementen in nicht verwalteten Windows-

| Schnittstelle                                                                          | BESCHREIBUNG                                                                                                                                                                                                                                                                                                         |
|------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAnnotationProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-iannotationprovider)<br/>           | Macht die Eigenschaften einer Anmerkung in einem Dokument verfügbar.<br/>                                                                                                                                                                                                                                                   |
| [**IDockProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idockprovider)<br/>                           | Ermöglicht den Zugriff auf ein Element in einem Docking Container. <br/>                                                                                                                                                                                                                                                   |
| [**IDragProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idragprovider)<br/>                           | Ermöglicht einem Microsoft UI Automation-Element, sich selbst als ein Element zu beschreiben, das als Teil eines Drag & Drop-Vorgangs gezogen werden kann.<br/>                                                                                                                                                                     |
| [**IDropTargetProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idroptargetprovider)<br/>               | Ermöglicht einem Benutzeroberflächenautomatisierungs-Element, sich selbst als ein Element zu beschreiben, das als Teil eines Drag & Drop-Vorgangs für die Benutzeroberflächen Automatisierung eine Ablage eines gezogenen Elements empfangen kann.<br/>                                                                                                                                        |
| [**IExpandCollapseProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iexpandcollapseprovider)<br/>       | Ermöglicht den Zugriff auf ein Steuerelement, das visuell erweitert wird, um Inhalt anzuzeigen, und reduziert, um Inhalt auszublenden. <br/>                                                                                                                                                                                                   |
| [**IGridItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igriditemprovider)<br/>                   | Ermöglicht den Zugriff auf einzelne untergeordnete Steuerelemente von Containern, die [**IGridProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igridprovider)implementieren.<br/>                                                                                                                                                                                 |
| [**IGridProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igridprovider)<br/>                           | Bietet Zugriff auf Steuerelemente, die als Container für eine Auflistung von untergeordneten Elementen fungieren, die in einem zweidimensionalen logischen Koordinatensystem angeordnet sind, das durchlaufen werden kann (d. h. ein Benutzeroberflächenautomatisierungs-Client kann zu benachbarten Steuerelementen wechseln), indem die Tastatur verwendet wird.<br/>                                              |
| [**IInvokeProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iinvokeprovider)<br/>                       | Bietet Zugriff auf Steuerelemente, die eine einzelne, eindeutige Aktion initiieren oder ausführen und den Zustand bei der Aktivierung nicht beibehalten.<br/>                                                                                                                                                                              |
| [**IItemContainerProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iitemcontainerprovider)<br/>         | Bietet Zugriff auf Steuerelemente, die als Container anderer Steuerelemente fungieren, z. b. eine virtuelle Listenansicht. <br/>                                                                                                                                                                                                      |
| [**Ilegacyiaccessibleprovider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ilegacyiaccessibleprovider)<br/> | Ermöglicht Benutzeroberflächenautomatisierungs-Clients den Zugriff auf die zugrunde liegende [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) -Implementierung von Microsoft Active Accessibility-Elementen.<br/>                                                                                                                                                     |
| [**IMultipleViewProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-imultipleviewprovider)<br/>           | Bietet Zugriff auf Steuerelemente, die mehrere Darstellungen desselben Informations Satzes oder untergeordneter Steuerelemente bereitstellen und zwischen diesen wechseln können.<br/>                                                                                                                                                     |
| [**IObjectModelProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-iobjectmodelprovider)<br/>         | Bietet Zugriff auf das zugrunde liegende Objektmodell, das von einem Steuerelement oder einer Anwendung implementiert wird. <br/>                                                                                                                                                                                                                 |
| [**IRangeValueProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irangevalueprovider)<br/>               | Ermöglicht den Zugriff auf Steuerelemente, die auf einen Wert innerhalb eines Bereichs festgelegt werden können. <br/>                                                                                                                                                                                                                                  |
| [**IScrollItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollitemprovider)<br/>               | Ermöglicht den Zugriff auf einzelne untergeordnete Steuerelemente von Containern, die [**IScrollProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollprovider)implementieren. <br/>                                                                                                                                                                            |
| [**IScrollProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollprovider)<br/>                       | Bietet Zugriff auf Steuerelemente, die als scrollbare Container für eine Auflistung von untergeordneten Objekten fungieren.<br/>                                                                                                                                                                                                         |
| [**ISelectionItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionitemprovider)<br/>         | Ermöglicht den Zugriff auf einzelne, auswählbare untergeordnete Steuerelemente von Containern, die [**ISelectionProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider)implementieren. <br/>                                                                                                                                                          |
| [**ISelectionProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider)<br/>                 | Bietet Zugriff auf Steuerelemente, die als Container für eine Auflistung einzelner, auswählbarer untergeordneter Elemente fungieren.<br/>                                                                                                                                                                                               |
| [**ISelectionProvider2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider2)<br/>               | Erweitert die [**ISelectionItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionitemprovider) -Schnittstelle so, dass Informationen zu ausgewählten Elementen bereitgestellt werden.<br/>                                                                                                                                                                       |
| [**ISpreadsheetItemProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-ispreadsheetitemprovider)<br/> | Ermöglicht den Zugriff auf Informationen zu einem Element (Zelle) in einer Kalkulations Tabelle. <br/>                                                                                                                                                                                                                                   |
| [**ISpreadsheetProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-ispreadsheetprovider)<br/>         | Ermöglicht den Zugriff auf Elemente (Zellen) in einer Kalkulations Tabelle.<br/>                                                                                                                                                                                                                                                       |
| [**IStylesProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-istylesprovider)<br/>                   | Bietet Zugriff auf die visuellen Stile, die dem Inhalt eines Dokuments zugeordnet sind.<br/>                                                                                                                                                                                                                          |
| [**ISynchronizedInputProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-isynchronizedinputprovider)<br/> | Ermöglicht Benutzeroberflächenautomatisierungs-Client Anwendungen, die Maus-oder Tastatureingaben an ein bestimmtes Benutzeroberflächen Element weiterzuleiten. <br/>                                                                                                                                                                                               |
| [**ITableItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableitemprovider)<br/>                 | Bietet Zugriff auf untergeordnete Steuerelemente von Containern, die [**ITableProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableprovider)implementieren. <br/>                                                                                                                                                                                         |
| [**ITableProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableprovider)<br/>                         | Bietet Zugriff auf Steuerelemente, die als Container für eine Auflistung von untergeordneten Elementen fungieren. Die untergeordneten Elemente dieses Elements müssen [**ITableItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableitemprovider) implementieren und in einem zweidimensionalen logischen Koordinatensystem angeordnet sein, das mithilfe der Tastatur durchlaufen werden kann. <br/> |
| [**ITextChildProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itextchildprovider)<br/>            | Ermöglicht den Zugriff auf ein textbasiertes Steuerelement (oder ein Objekt, das in Text eingebettet ist), das ein untergeordnetes Element oder ein untergeordnetes Element eines anderen textbasierten Steuer Elements ist. <br/>                                                                                                                                                                    |
| [**Itexteditprovider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itexteditprovider)<br/>               | Erweitert die [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) -Schnittstelle, sodass Benutzeroberflächenautomatisierungs-Anbieter programmgesteuerte Text Bearbeitungs Aktionen verfügbar machen können.<br/>                                                                                                                                                          |
| [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider)<br/>                           | Ermöglicht den Zugriff auf Steuerelemente, die Text enthalten. <br/>                                                                                                                                                                                                                                                          |
| [**ITextProvider2**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itextprovider2)<br/>                     | Erweitert die [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) -Schnittstelle, damit Benutzeroberflächenautomatisierungs-Anbieter Textinhalte verfügbar machen können, die das Ziel einer Anmerkung sind, sowie Informationen zu einem Caretzeichen, das zum Anbieter gehört.<br/>                                                                         |
| [**ITextRangeProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider)<br/>                 | Ermöglicht den Zugriff auf eine Spanne fortlaufenden Texts in einem Text Container, der [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) oder [**ITextProvider2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider2)implementiert.<br/>                                                                                                                       |
| [**ITextRangeProvider2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider2)<br/>               | Erweitert die [**ITextRangeProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider) -Schnittstelle, sodass Benutzeroberflächenautomatisierungs-Anbieter Kontextmenüs aufrufen können.<br/>                                                                                                                                                                 |
| [**IToggleProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itoggleprovider)<br/>                       | Bietet Zugriff auf Steuerelemente, die eine Reihe von Zuständen durchlaufen und einen Zustand beibehalten können, nachdem dieser festgelegt wurde.<br/>                                                                                                                                                                                                 |
| [**ITransformProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itransformprovider)<br/>                 | Ermöglicht den Zugriff auf Steuerelemente, die in einem zweidimensionalen Raum verschoben, verkleinert, geändert und/oder gedreht werden können. <br/>                                                                                                                                                                                                  |
| [**ITransformProvider2**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itransformprovider2)<br/>           | Erweitert die [**ITransformProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itransformprovider) -Schnittstelle, damit Benutzeroberflächenautomatisierungs-Anbieter Eigenschaften verfügbar machen können, um die Viewport-Zoomfunktion eines Steuer Elements zu unterstützen. <br/>                                                                                                        |
| [**IValueProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ivalueprovider)<br/>                         | Bietet Zugriff auf Steuerelemente, die einen systeminternen Wert haben, der sich nicht auf einen Bereich erstreckt und der als Zeichenfolge dargestellt werden kann. <br/>                                                                                                                                                                           |
| [**IVirtualizedItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ivirtualizeditemprovider)<br/>     | Bietet Zugriff auf virtualisierte Elemente, bei denen es sich um Elemente handelt, die durch Platzhalter-Automatisierungselemente in der Benutzeroberflächenautomatisierungs-Struktur dargestellt werden.<br/>                                                                                                                                                                 |
| [**IWindowProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iwindowprovider)<br/>                       | Ermöglicht den Zugriff auf die grundlegende fensterbasierte Funktionalität eines-Steuer Elements.<br/>                                                                                                                                                                                                                              |
