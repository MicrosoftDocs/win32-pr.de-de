# <a name="control-pattern-interfaces-for-providers"></a>Steuerelementmusterschnittstellen für Anbieter

In diesem Abschnitt werden *Steuerelementmusterschnittstellen* beschrieben, die von Benutzeroberflächenautomatisierung implementiert werden, um Informationen zu Steuerelementen in nicht verwalteten Windows verfügbar zu machen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Schnittstelle                                                                          | BESCHREIBUNG                                                                                                                                                                                                                                                                                                         |
|------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAnnotationProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-iannotationprovider)<br/>           | Macht die Eigenschaften einer Anmerkung in einem Dokument verfügbar.<br/>                                                                                                                                                                                                                                                   |
| [**IDockProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idockprovider)<br/>                           | Ermöglicht den Zugriff auf ein Element in einem Andockcontainer. <br/>                                                                                                                                                                                                                                                   |
| [**IDragProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idragprovider)<br/>                           | Ermöglicht einem Microsoft Benutzeroberflächenautomatisierung,sich selbst als Element zu beschreiben, das als Teil eines Drag & Drop-Vorgangs gezogen werden kann.<br/>                                                                                                                                                                     |
| [**IDropTargetProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idroptargetprovider)<br/>               | Ermöglicht einem Benutzeroberflächenautomatisierung, sich selbst als Element zu beschreiben, das einen Absturz eines gezogenen Elements als Teil eines Benutzeroberflächenautomatisierung Drag & Drop-Vorgangs empfangen kann.<br/>                                                                                                                                        |
| [**IExpandCollapseProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iexpandcollapseprovider)<br/>       | Ermöglicht den Zugriff auf ein Steuerelement, das visuell erweitert wird, um Inhalt anzuzeigen, und reduziert, um Inhalt auszublenden. <br/>                                                                                                                                                                                                   |
| [**IGridItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igriditemprovider)<br/>                   | Ermöglicht den Zugriff auf einzelne untergeordnete Steuerelemente von Containern, die [**IGridProvider implementieren.**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igridprovider)<br/>                                                                                                                                                                                 |
| [**IGridProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igridprovider)<br/>                           | Ermöglicht den Zugriff auf Steuerelemente, die als Container für eine Auflistung untergeordneter Elemente fungieren, die in einem zweidimensionalen logischen Koordinatensystem organisiert sind, das mithilfe der Tastatur durchlaufen werden kann (d. h. ein Benutzeroberflächenautomatisierung-Client kann zu angrenzenden Steuerelementen wechseln).<br/>                                              |
| [**IInvokeProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iinvokeprovider)<br/>                       | Ermöglicht den Zugriff auf Steuerelemente, die eine einzelne, eindeutige Aktion initiieren oder ausführen und bei aktivierung keinen Zustand verwalten.<br/>                                                                                                                                                                              |
| [**IItemContainerProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iitemcontainerprovider)<br/>         | Ermöglicht den Zugriff auf Steuerelemente, die als Container anderer Steuerelemente fungieren, z. B. eine virtuelle Listenansicht. <br/>                                                                                                                                                                                                      |
| [**ILegacyIAccessibleProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ilegacyiaccessibleprovider)<br/> | Ermöglicht Benutzeroberflächenautomatisierung Clients den Zugriff auf die zugrunde liegende [**IAccessible-Implementierung**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) Microsoft Active Accessibility Elementen.<br/>                                                                                                                                                     |
| [**IMultipleViewProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-imultipleviewprovider)<br/>           | Ermöglicht den Zugriff auf Steuerelemente, die mehrere Darstellungen desselben Informations- oder untergeordneten Steuerelements bereitstellen und zwischen diesen wechseln können.<br/>                                                                                                                                                     |
| [**IObjectModelProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-iobjectmodelprovider)<br/>         | Ermöglicht den Zugriff auf das zugrunde liegende Objektmodell, das von einem Steuerelement oder einer Anwendung implementiert wird. <br/>                                                                                                                                                                                                                 |
| [**IRangeValueProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irangevalueprovider)<br/>               | Ermöglicht den Zugriff auf Steuerelemente, die auf einen Wert innerhalb eines Bereichs festgelegt werden können. <br/>                                                                                                                                                                                                                                  |
| [**IScrollItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollitemprovider)<br/>               | Ermöglicht den Zugriff auf einzelne untergeordnete Steuerelemente von Containern, die [**IScrollProvider implementieren.**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollprovider) <br/>                                                                                                                                                                            |
| [**IScrollProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollprovider)<br/>                       | Ermöglicht den Zugriff auf Steuerelemente, die als scrollbare Container für eine Auflistung untergeordneter Objekte fungieren.<br/>                                                                                                                                                                                                         |
| [**ISelectionItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionitemprovider)<br/>         | Ermöglicht den Zugriff auf einzelne, auswählbare untergeordnete Steuerelemente von Containern, die [**ISelectionProvider implementieren.**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider) <br/>                                                                                                                                                          |
| [**ISelectionProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider)<br/>                 | Ermöglicht den Zugriff auf Steuerelemente, die als Container für eine Auflistung einzelner, auswählbarer untergeordneter Elemente fungieren.<br/>                                                                                                                                                                                               |
| [**ISelectionProvider2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider2)<br/>               | Erweitert die [**ISelectionItemProvider-Schnittstelle,**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionitemprovider) um Informationen zu ausgewählten Elementen zur Verfügung zu stellen.<br/>                                                                                                                                                                       |
| [**ISpreadsheetItemProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-ispreadsheetitemprovider)<br/> | Ermöglicht den Zugriff auf Informationen zu einem Element (Zelle) in einer Kalkulationstabelle. <br/>                                                                                                                                                                                                                                   |
| [**ISpreadsheetProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-ispreadsheetprovider)<br/>         | Ermöglicht den Zugriff auf Elemente (Zellen) in einem Arbeitsblatt.<br/>                                                                                                                                                                                                                                                       |
| [**IStylesProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-istylesprovider)<br/>                   | Ermöglicht den Zugriff auf die visuellen Stile, die dem Inhalt eines Dokuments zugeordnet sind.<br/>                                                                                                                                                                                                                          |
| [**ISynchronizedInputProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-isynchronizedinputprovider)<br/> | Ermöglicht Benutzeroberflächenautomatisierung Clientanwendungen, die Maus- oder Tastatureingabe an ein bestimmtes Benutzeroberflächenelement weiter zu richten. <br/>                                                                                                                                                                                               |
| [**ITableItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableitemprovider)<br/>                 | Ermöglicht den Zugriff auf untergeordnete Steuerelemente von Containern, die [**ITableProvider implementieren.**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableprovider) <br/>                                                                                                                                                                                         |
| [**ITableProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableprovider)<br/>                         | Ermöglicht den Zugriff auf Steuerelemente, die als Container für eine Auflistung untergeordneter Elemente fungieren. Die unteren Elemente dieses Elements müssen [**ITableItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableitemprovider) implementieren und in einem zweidimensionalen logischen Koordinatensystem organisiert sein, das mithilfe der Tastatur durchlaufen werden kann. <br/> |
| [**ITextChildProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itextchildprovider)<br/>            | Ermöglicht den Zugriff auf ein textbasiertes Steuerelement (oder ein in Text eingebettetes Objekt), das ein untergeordnetes Element oder ein Nachfolger eines anderen textbasierten Steuerelements ist. <br/>                                                                                                                                                                    |
| [**ITextEditProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itexteditprovider)<br/>               | Erweitert die [**ITextProvider-Schnittstelle,**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) um Benutzeroberflächenautomatisierung, programmgesteuerte Textbearbeitungsaktionen verfügbar zu machen.<br/>                                                                                                                                                          |
| [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider)<br/>                           | Ermöglicht den Zugriff auf Steuerelemente, die Text enthalten. <br/>                                                                                                                                                                                                                                                          |
| [**ITextProvider2**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itextprovider2)<br/>                     | Erweitert die [**ITextProvider-Schnittstelle,**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) um es Benutzeroberflächenautomatisierung-Anbietern zu ermöglichen, Textinhalte verfügbar zu machen, die das Ziel einer Anmerkung sind, sowie Informationen zu einem Caretelement, das zum Anbieter gehört.<br/>                                                                         |
| [**ITextRangeProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider)<br/>                 | Ermöglicht den Zugriff auf eine Spanne von fortlaufenden Text in einem Textcontainer, der [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) oder [**ITextProvider2 implementiert.**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider2)<br/>                                                                                                                       |
| [**ITextRangeProvider2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider2)<br/>               | Erweitert die [**ITextRangeProvider-Schnittstelle,**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider) damit Benutzeroberflächenautomatisierung-Anbieter Kontextmenüs aufrufen können.<br/>                                                                                                                                                                 |
| [**IToggleProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itoggleprovider)<br/>                       | Ermöglicht den Zugriff auf Steuerelemente, die einen Satz von Zuzuständen durchzyklen und einen Zustand nach dem Festlegen verwalten können.<br/>                                                                                                                                                                                                 |
| [**ITransformProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itransformprovider)<br/>                 | Ermöglicht den Zugriff auf Steuerelemente, die innerhalb eines zweidimensionalen Raums verschoben, geändert und/oder gedreht werden können. <br/>                                                                                                                                                                                                  |
| [**ITransformProvider2**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itransformprovider2)<br/>           | Erweitert die [**ITransformProvider-Schnittstelle,**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itransformprovider) um es Benutzeroberflächenautomatisierung-Anbietern zu ermöglichen, Eigenschaften verfügbar zu machen, um die Viewport-Zoomfunktion eines Steuerelements zu unterstützen. <br/>                                                                                                        |
| [**IValueProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ivalueprovider)<br/>                         | Ermöglicht den Zugriff auf Steuerelemente, die über einen systeminternen Wert verfügen, der keinen Bereich umfasst und als Zeichenfolge dargestellt werden kann. <br/>                                                                                                                                                                           |
| [**IVirtualizedItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ivirtualizeditemprovider)<br/>     | Ermöglicht den Zugriff auf virtualisierte Elemente, bei denen es sich um Elemente handelt, die durch Platzhalterautomatisierungselemente in der Benutzeroberflächenautomatisierung werden.<br/>                                                                                                                                                                 |
| [**IWindowProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iwindowprovider)<br/>                       | Ermöglicht den Zugriff auf die grundlegenden fensterbasierten Funktionen eines Steuerelements.<br/>                                                                                                                                                                                                                              |
