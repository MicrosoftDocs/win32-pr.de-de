# <a name="header-and-lib-files"></a>Header- und Lib-Dateien

Die folgenden Header- und Bibliotheksdateien unterstützen die Entwicklung Microsoft Active Accessibility Anwendungen.



| Datei         | BESCHREIBUNG                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Winuser.h    | C- und C++-Headerdatei für die Verwendung Windows Steuerelementen                                                                                                                             |
| Oleacc.h     | C- und C++-Headerdatei mit Definitionen, die von Microsoft Active Accessibility Clients und Servern benötigt werden                                                                    |
| Winver.h     | C- und C++-Headerdatei für Versionsinformationen                                                                                                                                |
| Oleacc.lib   | Importieren der Bibliothek für Oleacc.dll                                                                                                                                                |
| Oleacc.dll   | Dynamic Link Library, die die Microsoft Active Accessibility laufzeit, verwaltet Anforderungen von Microsoft Active Accessibility Clients und proxyt allgemeine Benutzeroberflächenelemente. |
| Oleaccrc.dll | Mehrsprachige Ressourcenbibliothek für dynamische Verknüpfungen, die von Oleacc.dll                                                                                                                |
