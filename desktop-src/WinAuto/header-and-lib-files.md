# <a name="header-and-lib-files"></a>Header-und lib-Dateien

Die folgenden Header-und Bibliotheksdateien unterstützen die Entwicklung von Microsoft Active Accessibility-Anwendungen.



| Datei         | BESCHREIBUNG                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Winuser. h    | C-und C++-Header Datei zur Verwendung von Windows-Steuerelementen                                                                                                                             |
| Oleacc. h     | C-und C++-Header Datei, die die von Microsoft Active Accessibility Clients und-Servern benötigten Definitionen enthält                                                                    |
| "Winver. h"     | C-und C++-Header Datei für Versionsinformationen                                                                                                                                |
| Oleacc. lib   | Importieren der Bibliothek für Oleacc.dll                                                                                                                                                |
| Oleacc.dll   | Dynamic Link Library, die die Microsoft Active Accessibility-Laufzeit bereitstellt, Anforderungen von Microsoft Active Accessibility Clients verwaltet und allgemeine Benutzeroberflächen Elemente von Proxys verwaltet. |
| Oleaccrc.dll | Von Oleacc.dll verwendete Dynamic Link Library für mehrsprachige Ressourcen                                                                                                                |
