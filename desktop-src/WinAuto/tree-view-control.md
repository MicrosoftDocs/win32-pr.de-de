# <a name="tree-view-control-msaa-ui-element-reference"></a>Strukturansichtssteuerelement (REFERENZ ZUM MSAA-UI-Element)

> [!Note]  
> In diesem Thema werden Tree **View Control-Objekte** für die MsAA-Benutzeroberflächenelementreferenz beschrieben. Das Erstellen von **Strukturansicht-Steuerelementobjekten** in verschiedenen Benutzeroberflächenframeworks wird hier nicht beschrieben. Weitere Informationen finden Sie in der API-Referenzdokumentation für das benutzeroberflächenframework, das Sie verwenden.

 

Ein Strukturansicht-Steuerelement zeigt eine hierarchische Liste von Elementen mithilfe von Zeilen an, um verknüpfte Elemente in einer Hierarchie zu verbinden. Jedes Element besteht aus einer Bezeichnung und einer optionalen Bitmap. Windows Explorer verwendet ein Strukturansicht-Steuerelement, um Verzeichnisse anzuzeigen.

Der Name der Fensterklasse für ein Strukturansicht-Steuerelement ist WC \_ TREEVIEW, das in Commctrl.h als "SysTreeView" definiert ist.

## <a name="notes"></a>Hinweise

Die **accLocation-Eigenschaft** von Strukturansichtselementen gibt nur die Position des Texts im Element zurück. Die von **accLocation** für Strukturansichtselemente zurückgegebenen Koordinaten enthalten nicht das Symbol links neben dem Text.
