# <a name="tree-view-control-msaa-ui-element-reference"></a>Strukturansicht-Steuer Element (MSAA-Benutzeroberflächen Element-Referenz)

> [!Note]  
> In diesem Thema werden Struktur **Ansicht-Steuer** Element Objekte zum Zweck der MSAA-Benutzeroberflächen-Element Referenz beschrieben. Das Erstellen von Struktur **Ansicht-Steuer** Element Objekten in verschiedenen Benutzeroberflächen-Frameworks wird hier nicht beschrieben. Weitere Informationen finden Sie in der API-Referenz Dokumentation für das von Ihnen verwendete UI-Framework.

 

Ein Strukturansicht-Steuerelement zeigt eine hierarchische Liste von Elementen an, die Zeilen zum Verbinden verwandter Elemente in einer Hierarchie verwenden. Jedes Element besteht aus einer Bezeichnung und einer optionalen Bitmap. Windows-Explorer verwendet ein Strukturansicht-Steuerelement, um Verzeichnisse anzuzeigen.

Der Fenster Klassenname für ein Strukturansicht-Steuerelement ist \_ "WC TreeView", das in "commctrl. h" als "systreeview" definiert ist.

Abhängig von der **State** -Eigenschaft eines Elements in der Strukturansicht, erweitert oder reduziert **accDoDefaultAction** das angegebene Element. |
