# <a name="ui-automation-fundamentals"></a>Grundlagen der Benutzeroberflächenautomatisierung

Microsoft Benutzeroberflächenautomatisierung ermöglicht hilfsfähigen Technologieanwendungen und automatisierten Testtools die Interaktion mit den Benutzeroberflächensteuerelementen anderer Anwendungen. In diesem Abschnitt werden die grundlegenden Konzepte erläutert, auf denen Benutzeroberflächenautomatisierung basiert.

Die Benutzeroberflächenautomatisierung-API besteht aus zwei Teilen. Ein Teil wird von *Benutzeroberflächenautomatisierung Anbieteranwendungen* verwendet, der andere von *Benutzeroberflächenautomatisierung Clientanwendungen.* Mit der Anbieter-API können Entwickler von benutzerdefinierten Microsoft Win32-Steuerelementen und anderen Steuerelementframeworks diese Steuerelemente für Benutzeroberflächenautomatisierung verfügbar machen und für Clientanwendungen sichtbar machen. Die Client-API ermöglicht Anwendungen die Interaktion mit Steuerelementen in anderen Anwendungen und das Abrufen von Informationen darüber.
