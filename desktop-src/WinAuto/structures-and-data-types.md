# <a name="structures-and-data-types"></a>Strukturen und Datentypen

Dieser Abschnitt enthält Informationen zu den Strukturen und Datentypen, die mit Microsoft Active Accessibility verwendet werden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                             | BESCHREIBUNG                                                                                                                                                              |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msaamenuinfo**](/windows/win32/api/oleacc/ns-oleacc-msaamenuinfo)<br/>   | Wird von Server Entwicklern verwendet, um die Namen der von einem Besitzer gezeichneten Menü Elemente verfügbar zu machen.<br/>                                                                                      |
| [**Hwineventhook**](hwineventhook.md)<br/> | Wird verwendet, um eine Windows Event Hook-Funktion zu deklarieren.<br/>                                                                                                                 |
| [**Wineventproc**](/previous-versions/windows/desktop/legacy/dd373882(v=vs.85))<br/>   | Wird verwendet, um einen Zeiger auf eine Anwendungs definierte Rückruffunktion zu deklarieren, die vom System als Antwort auf Ereignisse aufgerufen wird, die von einem barrierefreien Objekt generiert werden.<br/> |
