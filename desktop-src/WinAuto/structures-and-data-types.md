# <a name="structures-and-data-types"></a>Strukturen und Datentypen

Dieser Abschnitt enthält Informationen zu den Strukturen und Datentypen, die mit dem Microsoft Active Accessibility.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                             | Beschreibung                                                                                                                                                              |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSAAMENUINFO**](/windows/win32/api/oleacc/ns-oleacc-msaamenuinfo)<br/>   | Wird von Serverentwicklern verwendet, um die Namen von Vom Besitzer gezeichneten Menüelementen verfügbar zu machen.<br/>                                                                                      |
| [**HWINEVENTHOOK**](hwineventhook.md)<br/> | Wird verwendet, um eine Window-Ereignishookfunktion zu deklarieren.<br/>                                                                                                                 |
| [**WINEVENTPROC**](/previous-versions/windows/desktop/legacy/dd373882(v=vs.85))<br/>   | Wird verwendet, um einen Zeiger auf eine anwendungsdefinierte Rückruffunktion zu deklarieren, die das System als Reaktion auf Ereignisse aufruft, die von einem barrierefreien Objekt generiert werden.<br/> |
