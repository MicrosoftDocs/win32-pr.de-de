# <a name="windows-automation-api-overview"></a>Übersicht über die Windows Automation-API

Dieser Abschnitt enthält eine allgemeine Übersicht über Microsoft Windows Automation API 3,0, die die Legacy-Microsoft-Active Accessibility sowie die Windows-Implementierung der Microsoft UI Automation-Spezifikation enthält. In der Übersicht werden die Ähnlichkeiten und Unterschiede zwischen Microsoft Active Accessibility und der Automatisierung der Benutzeroberfläche hervorgehoben, die Komponenten und Funktionen beschrieben, mit denen die beiden Technologien zusammenarbeiten können, und Richtlinien für die Auswahl der zu implementierenden Technologie bereitstellen.
