# <a name="duplicateaccesskey"></a>DuplicateAccessKey

## <a name="text"></a>Text

Es gibt ein gleichgeordnetes Automation-Element, das über den gleichen Zugriffsschlüssel/Zugriffstastenschlüssel verfügt.

## <a name="type"></a>Typ

Fehler

## <a name="description"></a>Beschreibung

Der dem Element zugewiesene Zugriffs-/Zugriffstastenschlüssel wird auch einem anderen Element im Überprüfungsziel zugewiesen.

Dieses Problem verursacht Probleme für Personen, die sich für die Navigation auf die Tastatur verlassen, da mehrere Elemente aufgerufen oder der Zyklusfokus mit der gleichen Tastenkombination oder Zugriffstaste verwendet werden kann.
