# <a name="duplicateaccesskey"></a>Duplicateaccesskey

## <a name="text"></a>Text

Es gibt ein gleich geordnetes Automatisierungs Element, das über denselben Zugriffsschlüssel/Zugriffstasten Schlüssel verfügt.

## <a name="type"></a>type

Fehler

## <a name="description"></a>BESCHREIBUNG

Die Zugriffs-/acceler-Taste, die dem-Element zugewiesen ist, wird auch einem anderen Element im Überprüfungs Ziel zugewiesen.

Dieses Problem bewirkt, dass Personen, die sich für die Navigation auf der Tastatur befinden, Probleme verursachen, da mehrere Elemente möglicherweise aufgerufen werden oder den Fokus mithilfe derselben Tastenkombination oder Zugriffstaste durchsuchen.
