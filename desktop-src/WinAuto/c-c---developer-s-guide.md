# <a name="cc-developers-guide"></a>Entwicklerhandbuch zu C/C++

Dieser Abschnitt enthält Informationen darüber, wie Microsoft Active Accessibility aus der Perspektive eines C-oder C++-Anwendungs Entwicklers funktioniert. Es werden Probleme behandelt, die für Client-und Server Entwickler spezifisch sind.

Beachten Sie, dass ein *Client* eine Hilfe zur Barrierefreiheit oder eine andere Anwendung ist, die Microsoft Active Accessibility zum Abrufen von Informationen über Benutzeroberflächen Elemente verwendet. Ein *Server* ist eine Anwendung, die Microsoft Active Accessibility zum Bereitstellen von Informationen über Benutzeroberflächen Elemente verwendet.
