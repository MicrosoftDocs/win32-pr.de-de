# <a name="cc-developers-guide"></a>C/C++-Entwicklerhandbuch

Dieser Abschnitt enthält Informationen zur Funktionsweise von Microsoft Active Accessibility aus der Perspektive eines C- oder C++-Anwendungsentwicklers. Es werden spezifische Probleme für Client- und Serverentwickler behandelt.

Beachten Sie, dass ein *Client* eine Barrierefreiheitshilfe oder eine andere Anwendung ist, die Microsoft Active Accessibility verwendet, um Informationen über Benutzeroberflächenelemente abzurufen. Ein *Server* ist eine Anwendung, die Microsoft Active Accessibility verwendet, um Informationen über Benutzeroberflächenelemente bereitzustellen.
