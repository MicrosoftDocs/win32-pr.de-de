# <a name="content-of-descriptive-properties"></a>Inhalt von beschreibenden Eigenschaften

Die [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) -Schnittstelle stellt beschreibende Eigenschaften zur Verfügung, die verschiedene Aspekte eines Objekts beschreiben. Einige dieser Eigenschaften sind inhaltsspezifisch. andere Eigenschaften verfügen über Inhalte, die aus beschreibenden Text bestehen, der vom Server bereitgestellt wird. Der Typ der Informationen für jede Eigenschaft variiert abhängig vom Objekt.

In den folgenden Themen werden Informationen beschrieben, die von-Clients aus diesen Eigenschaften abgerufen werden. Außerdem bieten Sie den Servern Vorschläge zum Auswählen von Inhalt.

-   [Name-Eigenschaft](name-property.md)
-   [Role-Eigenschaft](role-property.md)
-   [State-Eigenschaft](state-property.md)
-   [Value-Eigenschaft](value-property.md)
-   [Description-Eigenschaft](description-property.md)
-   [Hilfe (Eigenschaft)](help-property.md)
-   [HelpTopic-Eigenschaft](helptopic-property.md)
-   [KeyboardShortcut (Eigenschaft)](keyboardshortcut-property.md)
-   [DEFAULTACTION (Eigenschaft)](defaultaction-property.md)

Beim Entwerfen von zugänglichen Objekten sollten Server Entwickler auch die folgenden Themen lesen:

-   [Auswählen der zu unterstützten Eigenschaften](choosing-which-properties-to-support.md)
-   [Auswählen des Inhalts für beschreibende Eigenschaften](choosing-the-content-for-descriptive-properties.md)

Informationen zu den Parametern und Rückgabe Werten dieser Eigenschaften finden Sie im Abschnitt [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) der Referenz zu Microsoft Active Accessibility [C/C++](c-c---reference.md).
