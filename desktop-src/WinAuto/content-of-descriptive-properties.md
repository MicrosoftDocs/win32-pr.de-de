# <a name="content-of-descriptive-properties"></a>Inhalt beschreibender Eigenschaften

Die [**IAccessible-Schnittstelle**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) stellt beschreibende Eigenschaften bereit, die verschiedene Aspekte eines Objekts beschreiben. Einige dieser Eigenschaften sind inhaltsspezifisch. andere Eigenschaften verfügen über Inhalt, der aus beschreibendem Text besteht, der vom Server bereitgestellt wird. Der Informationstyp für jede Eigenschaft variiert je nach Objekt.

In den folgenden Themen werden Informationen beschrieben, die Clients aus diesen Eigenschaften abrufen. Außerdem erhalten Server Vorschläge für die Auswahl von Inhalten.

-   [Name-Eigenschaft](name-property.md)
-   [Role-Eigenschaft](role-property.md)
-   [State-Eigenschaft](state-property.md)
-   [Value-Eigenschaft](value-property.md)
-   [Description-Eigenschaft](description-property.md)
-   [Help-Eigenschaft](help-property.md)
-   [HelpTopic-Eigenschaft](helptopic-property.md)
-   [KeyboardShortcut-Eigenschaft](keyboardshortcut-property.md)
-   [DefaultAction-Eigenschaft](defaultaction-property.md)

Beim Entwerfen barrierefreier Objekte sollten Sich Serverentwickler auch auf die folgenden Themen beziehen:

-   [Auswählen der zu unterstützenden Eigenschaften](choosing-which-properties-to-support.md)
-   [Auswählen des Inhalts für beschreibende Eigenschaften](choosing-the-content-for-descriptive-properties.md)

Informationen zu den Parametern und Rückgabewerten dieser Eigenschaften finden Sie im Abschnitt [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) der Microsoft Active Accessibility [C/C++-Referenz.](c-c---reference.md)
