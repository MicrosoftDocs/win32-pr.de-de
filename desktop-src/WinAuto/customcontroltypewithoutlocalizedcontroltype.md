# <a name="customcontroltypewithoutlocalizedcontroltype"></a>Customcontroltypewithoutlocalizedcontroltype

## <a name="text"></a>Text

Für das Element ist die LocalizedControlType-Eigenschaft nicht festgelegt.

## <a name="type"></a>type

Fehler

## <a name="description"></a>BESCHREIBUNG

Die [**LocalizedControlType**](/windows/desktop/api/UIAutomationClient/nf-uiautomationclient-iuiautomationelement-get_currentlocalizedcontroltype) -Eigenschaft des Elements ist nicht festgelegt.

Die meisten Sprachausgaben lesen den lokalisierten Steuer Elementtyp, um die Art des Elements zu beschreiben. Wenn der lokalisierte Steuerelement nicht festgelegt ist, greift eine Sprachausgabe auf den Haupt Steuerungstyp zurück, der sich stets in englischer Sprache befindet und für alle Benutzer möglicherweise nicht sinnvoll ist.
