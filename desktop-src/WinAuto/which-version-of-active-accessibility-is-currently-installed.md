# <a name="determine-the-installed-version-of-active-accessibility"></a>Ermitteln der installierten Version von Active Accessibility

Um herauszufinden, welche Version von Microsoft Active Accessibility (MSAA) auf Ihrem System installiert ist, sollten Sie die Version der Oleacc.dll Datei im Ordner System32 auf der Festplatte überprüfen. In der folgenden Tabelle ist die Version von Microsoft Active Accessibility der Oleacc.dll Version zugeordnet.

| Active Accessibility Version     | Oleacc.dll Version                                                                       |
|----------------------------------|------------------------------------------------------------------------------------------|
| 1.0                              | 4.01.0970.0                                                                              |
| 1.2                              | 4.02.2900.0                                                                              |
| 1.3                              | 4.02.3100.0                                                                              |
| 2.0                              | 5.0.0.0                                                                                  |
| 3,0 (Windows Automation-API 3,0) | 7.0. x. y, wobei *x* für die Buildnummer und *y* für die QFE-Nummer (Quick Fix Engineering) steht. |
