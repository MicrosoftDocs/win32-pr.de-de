# <a name="scope-and-longevity-of-identity-string"></a>Bereich und Langlebigkeit der Identitäts Zeichenfolge

Eine Identitäts Zeichenfolge identifiziert eine bestimmte Instanz eines UI-Elements. Mehrere Instanzen eines zugänglichen Elements verfügen über unterschiedliche Identitäts Zeichenfolgen. Daher ist es in der Regel nicht sinnvoll, eine Identitäts Zeichenfolge beizubehalten.
