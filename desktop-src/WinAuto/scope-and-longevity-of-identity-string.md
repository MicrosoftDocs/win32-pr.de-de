# <a name="scope-and-longevity-of-identity-string"></a>Bereich und Wirksamkeit der Identitätszeichenfolge

Eine Identitätszeichenfolge identifiziert eine bestimmte Instanz eines Benutzeroberflächenelements. Mehrere Instanzen eines barrierefreien Elements verfügen über unterschiedliche Identitätszeichenfolgen. daher ist es in der Regel nicht sinnvoll, eine Identitätszeichenfolge dauerhaft zu speichern.
