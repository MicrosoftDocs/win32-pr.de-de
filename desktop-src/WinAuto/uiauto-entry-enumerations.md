# <a name="ui-automation-enumerations"></a>Benutzeroberflächenautomatisierung Enumerationen

In diesem Abschnitt werden die Enumerationstypen beschrieben, die von Benutzeroberflächenautomatisierung Clientanwendungen und Benutzeroberflächenautomatisierung Anbieter für Microsoft Win32 verwendet werden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Enumeration                                                                        | Beschreibung                                                                                                                                                                                                                                                          |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ActiveEnd**](/windows/desktop/api/uiautomationcore/ne-uiautomationcore-activeend)<br/>                               | Enthält mögliche Werte für das Textattribut [**SelectionActiveEnd,**](uiauto-textattribute-ids.md) das die Position des Caretzeichens relativ zu einem Textbereich angibt, der den aktuell ausgewählten Text darstellt.<br/> |
| [**AnimationStyle**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-animationstyle)<br/>                     | Enthält Werte für das Textattribut AnimationStyle.<br/>                                                                                                                                                                                                    |
| [**AsyncContentLoadedState**](/windows/desktop/api/UIAutomationCoreApi/ne-uiautomationcoreapi-asynccontentloadedstate)<br/>   | Enthält Werte, die den Fortschritt des asynchronen Ladens von Inhalten beschreiben.<br/>                                                                                                                                                                            |
| [**AutomationElementMode**](/windows/desktop/api/UIAutomationClient/ne-uiautomationclient-automationelementmode)<br/>       | Enthält Werte, die den beim Zurückgeben von „UI Automation“-Elementen zu verwendenden Verweistyp angeben.<br/>                                                                                                                                                          |
| [**AutomationIdentifierType**](/windows/desktop/api/UIAutomationCoreApi/ne-uiautomationcoreapi-automationidentifiertype)<br/>         | Enthält Werte, die in der [**UiaLookupId-Funktion**](/windows/desktop/api/UIAutomationCoreApi/nf-uiautomationcoreapi-uialookupid) verwendet werden.<br/>                                                                                                                                                                   |
| [**Bulletstyle**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-bulletstyle)<br/>                           | Enthält Werte für das BulletStyle-Textattribut.<br/>                                                                                                                                                                                                       |
| [**CapStyle**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-capstyle)<br/>                                 | Enthält Werte, die den Wert des CapStyle-Textattributs angeben.<br/>                                                                                                                                                                                    |
| [**CaretBidiMode**](/windows/desktop/api/uiautomationcore/ne-uiautomationcore-caretbidimode)<br/>                       | Enthält mögliche Werte für das [**Textattribut CaretBidiMode,**](uiauto-textattribute-ids.md) das angibt, ob sich das Caretzeichen in Text befindet, der von links nach rechts oder von rechts nach links fließt.<br/>                        |
| [**CaretPosition**](/windows/desktop/api/uiautomationcore/ne-uiautomationcore-caretposition)<br/>                       | Enthält mögliche Werte für das [**Textattribut CaretPosition,**](uiauto-textattribute-ids.md) das die Position des Caretzeichens relativ zu einer Textzeile in einem Textbereich angibt.<br/>                                     |
| [**ConditionType**](/windows/desktop/api/UIAutomationCoreApi/ne-uiautomationcoreapi-conditiontype)<br/>                       | Enthält Werte, die den Typ [**UiaCondition**](/windows/desktop/api/UIAutomationCoreApi/ns-uiautomationcoreapi-uiacondition)angeben.<br/>                                                                                                                                                                 |
| [**Dockposition**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-dockposition)<br/>                            | Enthält Werte, die den Speicherort eines Andockfensters angeben, das durch das [Dock-Steuerelementmuster](uiauto-implementingdock.md) dargestellt wird.<br/>                                                                                                                   |
| [**EventArgsType**](/windows/desktop/api/UIAutomationCoreApi/ne-uiautomationcoreapi-eventargstype)<br/>                       | Enthält Werte, die den von einer [**UiaEventArgs-Struktur**](/windows/desktop/api/UIAutomationCoreApi/ns-uiautomationcoreapi-uiaeventargs) beschriebenen Ereignistyp angeben.<br/>                                                                                                                                   |
| [**Expandcollapsestate**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-expandcollapsestate)<br/>           | Enthält Werte, die den Zustand eines UI-Elements angeben, das erweitert und reduziert werden kann.<br/>                                                                                                                                                                |
| [**FlowDirections**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-flowdirections)<br/>                     | Enthält Werte für das TextflowDirections-Textattribut. <br/>                                                                                                                                                                                               |
| [**FillType**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-filltype)<br/>                                 | Enthält Werte für das FillType-Attribut.<br/>                                                                                                                                                                                                               |
| [**HorizontalTextAlignmentEnum**](/previous-versions/windows/desktop/legacy/ee671233(v=vs.85))<br/>   | Enthält Werte für das Textattribut HorizontalTextAlignment.<br/>                                                                                                                                                                                           |
| [**LiveSetting**](/windows/desktop/api/uiautomationcore/ne-uiautomationcore-livesetting)<br/>                           | Enthält mögliche Werte für die LiveSetting-Eigenschaft. Diese Eigenschaft wird von Anbieterelementen implementiert, die Teil eines Livebereichs sind.<br/>                                                                                                                  |
| [**NavigateDirection**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-navigatedirection)<br/>                          | Enthält Werte, mit denen die Navigationsrichtung innerhalb der Microsoft Benutzeroberflächenautomatisierung-Struktur angegeben wird.<br/>                                                                                                                                                      |
| [**NormalizeState**](/windows/desktop/api/UIAutomationCoreApi/ne-uiautomationcoreapi-normalizestate)<br/>                     | Enthält Werte, die das Verhalten von [**UiaGetUpdatedCache**](/windows/desktop/api/UIAutomationCoreApi/nf-uiautomationcoreapi-uiagetupdatedcache)angeben.<br/>                                                                                                                                             |
| [**NotificationKind**](/windows/win32/api/uiautomationcore/ne-uiautomationcore-notificationkind)<br/>                     | Definiert Werte, die den Typ eines Benachrichtigungsereignisses und einen Hinweis für den Listener zur Verarbeitung des Ereignisses angeben.<br/>                                                                                                                              |
| [**NotificationProcessing**](/windows/win32/api/uiautomationcore/ne-uiautomationcore-notificationprocessing)<br/>         | Definiert Werte, die angeben, wie eine Benachrichtigung verarbeitet werden soll.<br/>                                                                                                                                                                                      |
| [**OrientationType**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-orientationtype)<br/>                       | Enthält Werte, die die Ausrichtung eines Steuerelements angeben.<br/>                                                                                                                                                                                                |
| [**OutlineStyles**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-outlinestyles)<br/>                       | Enthält Werte für das OutlineStyle-Textattribut.<br/>                                                                                                                                                                                                      |
| [**Propertyconditionflags**](/windows/desktop/api/UIAutomationClient/ne-uiautomationclient-propertyconditionflags)<br/>     | Enthält Werte, die beim Erstellen von Eigenschaftsbedingungen verwendet werden.<br/>                                                                                                                                                                                                     |
| [**ProviderOptions**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-provideroptions)<br/>                       | Enthält Werte, die den Typ des Benutzeroberflächenautomatisierungs-Anbieters angeben. Die [**IRawElementProviderSimple::P rlockderOptions-Eigenschaft**](/windows/desktop/api/UIAutomationCore/nf-uiautomationcore-irawelementprovidersimple-get_provideroptions) verwendet diese Enumeration.<br/>                                               |

| [**RowOrColumnMajor**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-roworcolumnmajor)<br/>                 | Enthält Werte, die angeben, ob die Daten in einer Tabelle in erster Linie nach Zeile oder Spalte gelesen werden sollen.<br/>                                                                                                                                                        |

| [**StructureChangeType**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-structurechangetype)<br/>           | Enthält Werte, die den Änderungstyp in der Benutzeroberflächenautomatisierung Struktur angeben.<br/>                                                                                                                                                                      |

| [**TextDecorationLineStyle**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-textdecorationlinestyle)<br/>              | Enthält Werte, die die Textattribute OverlineStyle, StrikethroughStyle und UnderlineStyle angeben.<br/>                                                                                                                                                   |

| [**TextPatternRangeEndpoint**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-textpatternrangeendpoint)<br/> | Enthält Werte, die die Endpunkte eines Textbereichs angeben.<br/>                                                                                                                                                                                               |
| [**Textunit**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-textunit)<br/>                                 | Enthält Werte, die Texteinheiten für die Navigation angeben.<br/>                                                                                                                                                                                |
| [**Togglestate**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-togglestate)<br/>                               | Enthält Werte, die den Umschaltzustand eines Benutzeroberflächenautomatisierung Elements angeben, das das [Umschalten-Steuerelementmuster](uiauto-implementingtoggle.md) implementiert.<br/>                                                                                                   |

| [**Windowvisualstate**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-windowvisualstate)<br/>               | Enthält Werte, die den visuellen Zustand eines Fensters angeben.<br/>                                                                                                                                                                                                |
| [**ZoomUnit**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-zoomunit)<br/>                                     | Enthält mögliche Werte für die [**IUIAutomationTransformPattern2::ZoomByUnit-Methode,**](/windows/desktop/api/UIAutomationClient/nf-uiautomationclient-iuiautomationtransformpattern2-zoombyunit) die den Viewport eines Steuerelements durch die angegebene Einheit vergrößert.<br/>                                              |
