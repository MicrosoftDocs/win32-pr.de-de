# <a name="client-functions"></a>Clientfunktionen

Dieser Abschnitt enthält Informationen zu den Client Funktionen, die mit Microsoft Active Accessibility verwendet werden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                             | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|-----------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Accessiblechildren**](/windows/desktop/api/Oleacc/nf-oleacc-accessiblechildren)<br/>                       | Ruft die untergeordnete ID oder [**IDispatch**](idispatch-interface.md) der einzelnen untergeordneten Elemente in einem zugänglichen Container Objekt ab.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**Accessibleobjectfromevent**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfromevent)<br/>         | Ruft die Adresse der [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) -Schnittstelle für das Objekt ab, das das Ereignis generiert hat, das zurzeit von der Ereignis-Hook-Funktion des Clients verarbeitet wird.<br/>                                                                                                                                                                                                                                                                                                                                                                             |
| [**Accessibleobjectfrompoint**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfrompoint)<br/>         | Ruft die Adresse des [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) -Schnittstellen Zeigers für das-Objekt ab, das an einem angegebenen Punkt auf dem Bildschirm angezeigt wird.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**Accessibleobjectfromwindow**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfromwindow)<br/>       | Ruft die Adresse der angegebenen-Schnittstelle für das dem angegebenen Fenster zugeordnete-Objekt ab.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**Getoleaccversioninfo**](/windows/desktop/api/Oleacc/nf-oleacc-getoleaccversioninfo)<br/>                   | Ruft die Versionsnummer und die Buildnummer der Microsoft Active Accessibility-Datei Oleacc.dll ab.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**Getprocesshandlefromhwnd**](getprocesshandlefromhwnd.md)<br/>           | Ruft ein Prozess handle von einem Fenster Handle ab.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**Getroletext**](/windows/desktop/api/Oleacc/nf-oleacc-getroletexta)<br/>                                     | Ruft die lokalisierte Zeichenfolge ab, die die Rolle des-Objekts für den angegebenen Rollen Wert beschreibt.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**Getstatetext**](/windows/desktop/api/Oleacc/nf-oleacc-getstatetexta)<br/>                                   | Ruft eine lokalisierte Zeichenfolge ab, die den Zustand eines Objekts für ein einzelnes vordefiniertes statusbitflag beschreibt. Da Zustands Werte eine Kombination aus mindestens einem Bitflag sind, rufen Clients diese Funktion mehrmals auf, um alle Zustands Zeichenfolgen abzurufen.<br/>                                                                                                                                                                                                                                                                                                                      |
| [**Setwineventhook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook)<br/>                             | Legt eine ereignishookfunktion für einen Bereich von Ereignissen fest.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**Unhuokwinevent**](/windows/desktop/api/Winuser/nf-winuser-unhookwinevent)<br/>                               | Entfernt eine Event Hook-Funktion, die durch einen vorherigen-Befehl von [**setwineventhook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook)erstellt wurde.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**Windowfromaccessibleobject**](/windows/desktop/api/Oleacc/nf-oleacc-windowfromaccessibleobject)<br/>       | Ruft das Fenster Handle ab, das einer bestimmten Instanz einer [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) -Schnittstelle entspricht.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [*Wineventproc-Rückruffunktion*](/windows/desktop/api/Winuser/nc-winuser-wineventproc)<br/> | Eine Anwendungs definierte Rückruffunktion (oder Hook-Funktion), die vom System als Antwort auf Ereignisse aufgerufen wird, die von einem barrierefreien Objekt generiert werden. Die Hook-Funktion verarbeitet die Ereignis Benachrichtigungen nach Bedarf. Clients installieren die Hook-Funktion und fordern bestimmte Arten von Ereignis Benachrichtigungen durch Aufrufen von [**setwineventhook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook)an.<br/> Der [**wineventproc**](/previous-versions/windows/desktop/legacy/dd373882(v=vs.85)) -Typ definiert einen Zeiger auf diese Rückruffunktion. [*Wineventproc*](/windows/desktop/api/winuser/nc-winuser-wineventproc) ist ein Platzhalter für den Namen der Anwendungs definierten Funktion.<br/> |
