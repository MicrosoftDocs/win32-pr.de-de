# <a name="taborderinquestion"></a>Taborderinquestion

## <a name="text"></a>Text

Die Aktivier Reihenfolge kann nicht den Windows-Standardrichtlinien entsprechen

## <a name="type"></a>type

Informational

## <a name="description"></a>BESCHREIBUNG

Mithilfe der Standardtastatur Navigation (Tab oder UMSCHALT + TAB) wird eine nicht standardmäßige Aktivier Reihenfolge angezeigt. Beispielsweise wird in Abhängigkeit von den lokalen Systemeinstellungen in der Regel die Aktivier Reihenfolge imitiert.

Dieses Problem bewirkt, dass Personen, die sich auf einen Bildschirm-Reader und eine Tastatur für die Navigation verlassen, dass das Durchlaufen von Elementen möglicherweise unberechenbar und unvorhersagbar ist.
