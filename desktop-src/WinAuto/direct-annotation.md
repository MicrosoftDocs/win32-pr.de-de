# <a name="direct-annotation"></a>Direkte Anmerkung

Dieser Abschnitt enthält Informationen zur Verwendung der direkten Anmerkung.

## <a name="summary"></a>Zusammenfassung

Die direkte Anmerkung ist die einfachste Form der dynamischen Anmerkung. Sie wird verwendet, um den Wert einer oder einer begrenzten Anzahl von Eigenschaften eines barrierefreien Elements zu überschreiben. Es ist besonders nützlich für barrierefreie Elemente, deren mit Anmerkungen versehene Eigenschaften nicht vom Zustand des Steuer Elements abhängig sind und nicht die zusätzliche Unterstützung erfordern, die von der [Wert Map](value-map-annotation.md) -Anmerkung und der [Server Anmerkung](server-annotation.md)bereitgestellt wird.

## <a name="when-to-use-this-technique"></a>Verwendungszwecke dieser Technik

Verwenden Sie diese Methode, wenn Sie den Wert einer oder einer begrenzten Anzahl von Eigenschaften für ein barrierefreies Element überschreiben möchten.

Weitere Informationen finden Sie unter [Verwenden der direkten Anmerkung](using-direct-annotation.md).
