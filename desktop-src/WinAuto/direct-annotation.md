# <a name="direct-annotation"></a>Direkte Anmerkung

Dieser Abschnitt enthält Informationen zur Verwendung direkter Anmerkungen.

## <a name="summary"></a>Zusammenfassung

Die direkte Anmerkung ist die einfachste Form der dynamischen Anmerkung. Sie wird verwendet, um den Wert einer oder einer begrenzten Anzahl von Eigenschaften eines barrierefreien Elements zu überschreiben. Dies ist besonders nützlich für barrierefreie Elemente, deren mit Anmerkungen versehene Eigenschaften nicht vom Zustand des Steuerelements abhängen und die zusätzliche Unterstützung durch [die Wertzuordnungsanmerkung](value-map-annotation.md) und [die Serveranmerkung](server-annotation.md)nicht erfordern.

## <a name="when-to-use-this-technique"></a>Wann sollte diese Technik verwendet werden?

Verwenden Sie dieses Verfahren, wenn Sie den Wert einer oder einer begrenzten Anzahl von Eigenschaften für ein barrierefreies Element überschreiben möchten.

Weitere Informationen finden Sie unter [Verwenden der direkten Anmerkung.](using-direct-annotation.md)
