# <a name="ui-automation-element-interfaces-for-clients"></a>Benutzeroberflächenautomatisierung-Elementschnittstellen für Clients

In diesem Abschnitt werden die Schnittstellen beschrieben, die von Microsoft Benutzeroberflächenautomatisierung Client verwendet werden, um Benutzeroberflächenautomatisierung Elemente zu suchen, darauf zuzugreifen und sie abzufragen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Schnittstelle                                                                        | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                |
|----------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IUIAutomation**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomation)<br/>                         | Macht Methoden verfügbar, die es Benutzeroberflächenautomatisierung Clientanwendungen ermöglichen, Benutzeroberflächenautomatisierung Elemente zu ermitteln, darauf zuzugreifen und zu filtern. Benutzeroberflächenautomatisierung macht jedes Element der Benutzeroberflächenautomatisierung als objekt verfügbar, das durch die [**IUIAutomation-Schnittstelle**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomation) dargestellt wird. Die Member dieser Schnittstelle sind nicht spezifisch für ein bestimmtes Element.<br/> |
| [**IUIAutomation2**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomation2)<br/>                       | Erweitert die [**IUIAutomation-Schnittstelle,**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomation) um zusätzliche Methoden zum Steuern Benutzeroberflächenautomatisierung Funktionalität verfügbar zu machen.<br/>                                                                                                                                                                                                   |
| [**IUIAutomation3**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomation3)<br/>                       | Erweitert die [**IUIAutomation2-Schnittstelle,**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomation2) um zusätzliche Methoden zum Steuern Benutzeroberflächenautomatisierung Funktionalität verfügbar zu machen.<br/>                                                                                                                                                                                                 |
| [**IUIAutomation4**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomation4)<br/>                       | Erweitert die [**IUIAutomation3-Schnittstelle,**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomation3) um zusätzliche Methoden zum Steuern Benutzeroberflächenautomatisierung Funktionalität verfügbar zu machen.<br/>                                                                                                                                                                                                 |
| [**IUIAutomation5**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomation5)<br/>                       | Erweitert die [**IUIAutomation4-Schnittstelle,**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomation4) um zusätzliche Methoden zum Steuern Benutzeroberflächenautomatisierung Funktionalität verfügbar zu machen.<br/>                                                                                                                                                                                                 |
| [**IUIAutomation6**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomation6)<br/>                       | Erweitert die [**IUIAutomation5-Schnittstelle,**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomation5) um zusätzliche Methoden zum Steuern Benutzeroberflächenautomatisierung Funktionalität verfügbar zu machen.<br/>                                                                                                                                                                                                 |
| [**IUIAutomationCacheRequest**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationcacherequest)<br/> | Macht Eigenschaften und Methoden einer Cacheanforderung verfügbar. Clientanwendungen verwenden diese Schnittstelle, um die Eigenschaften und Steuerelementmuster anzugeben, die zwischengespeichert werden sollen, wenn ein Benutzeroberflächenautomatisierung Element abgerufen wird.<br/>                                                                                                                                                 |
| [**IUIAutomationElement**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement)<br/>           | Macht Methoden und Eigenschaften für ein Benutzeroberflächenautomatisierung-Element verfügbar, das ein Benutzeroberflächenelement darstellt. <br/>                                                                                                                                                                                                                                                        |
| [**IUIAutomationElement2**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement2)<br/>         | Erweitert die [**IUIAutomationElement-Schnittstelle.**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement) <br/>                                                                                                                                                                                                                                                             |
| [**IUIAutomationElement3**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement3)<br/>         | Erweitert die [**IUIAutomationElement2-Schnittstelle.**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement2) <br/>                                                                                                                                                                                                                                                           |
| [**IUIAutomationElement4**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement4)<br/>         | Erweitert die [**IUIAutomationElement3-Schnittstelle.**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement3) <br/>                                                                                                                                                                                                                                                           |
| [**IUIAutomationElement5**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement5)<br/>         | Erweitert die [**IUIAutomationElement4-Schnittstelle,**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement4) um Zugriff auf aktuelle und zwischengespeicherte Wahrzeichendaten zu ermöglichen.<br/>                                                                                                                                                                                                      |
| [**IUIAutomationElement6**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement6)<br/>         | Erweitert die [**IUIAutomationElement5-Schnittstelle,**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement5) um Zugriff auf aktuelle und zwischengespeicherte vollständige Beschreibungen zu ermöglichen.<br/>                                                                                                                                                                                                  |
| [**IUIAutomationElement7**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement7)<br/>         | Erweitert die [**IUIAutomationElement6-Schnittstelle.**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement6)<br/>                                                                                                                                                                                                                                                            |
| [**IUIAutomationElement8**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement8)<br/>         | Erweitert die [**IUIAutomationElement7-Schnittstelle.**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement7)<br/>                                                                                                                                                                                                                                                            |
| [**IUIAutomationElement9**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement9)<br/>         | Erweitert die [**IUIAutomationElement8-Schnittstelle.**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement8)<br/>                                                                                                                                                                                                                                                            |
| [**IUIAutomationElementArray**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelementarray)<br/> | Stellt eine Auflistung von Benutzeroberflächenautomatisierung Elementen dar.<br/>                                                                                                                                                                                                                                                                                              |
| [**IUIAutomationRegistrar**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iuiautomationregistrar)<br/>       | Macht Methoden zum Registrieren neuer Steuerelementmuster, -eigenschaften und -ereignisse verfügbar.<br/>                                                                                                                                                                                                                                                                   |
| [**IUIAutomationTreeWalker**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtreewalker)<br/>     | Macht Eigenschaften und Methoden verfügbar, die Benutzeroberflächenautomatisierung Clientanwendungen verwenden, um die Benutzeroberflächenautomatisierung Elemente auf dem Desktop anzuzeigen und zu navigieren. <br/>                                                                                                                                                                                                      |
