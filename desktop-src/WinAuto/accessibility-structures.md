# <a name="accessibility-structures"></a>Barrierefreiheits Strukturen

In diesem Abschnitt werden die Strukturen beschrieben, mit denen Barrierefreiheits Funktionen von Windows implementiert werden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Struktur                                         | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Accesstimeout**](/windows/win32/api/winuser/ns-winuser-accesstimeout)<br/> | Enthält Informationen zu dem Timeout Zeitraum, der den Microsoft Win32-Barrierefreiheits Funktionen zugeordnet ist. <br/> Der Timeout Zeitraum für die Barrierefreiheit ist die Zeitspanne, die ohne Tastatur-und Maus Eingaben bestanden werden muss, bevor das Betriebssystem die Barrierefreiheits Funktionen automatisch deaktiviert. Der Barrierefreiheits Timeout ist für Computer konzipiert, die von mehreren Benutzern gemeinsam genutzt werden, sodass die von einem Benutzer ausgewählten Optionen für einen nachfolgenden Benutzer nicht beeinträchtigt werden.<br/> Die von diesem Timeout betroffenen Barrierefreiheits Funktionen sind die FilterKeys-Funktionen (SlowKeys, BounceKeys und RepeatKeys), MouseKeys, "degglekeys" und "StickyKeys". Das Barrierefreiheits Timeout wirkt sich auch auf die Einstellung für den Modus für hohe Kontraste aus.<br/> |
| [**Filter Keys**](/windows/win32/api/winuser/ns-winuser-filterkeys)<br/>       | Enthält Informationen über das Filter Keys-Barrierefreiheits Feature, mit dem Benutzer mit Behinderungen die Tastatur Wiederholungsrate (RepeatKeys), Akzeptanz Verzögerung (SlowKeys) und boundingrate (BounceKeys) festlegen können. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**HIGHCONTRAST**](/windows/win32/api/winuser/ns-winuser-highcontrasta)<br/>   | Enthält Informationen zum Barrierefreiheits Feature für hohen Kontrast.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**Maustasten**](/windows/win32/api/winuser/ns-winuser-mousekeys)<br/>         | Enthält Informationen über das MouseKeys-Barrierefreiheits Feature. Wenn die MouseKeys-Funktion aktiv ist, kann der Benutzer mit der numerischen Tastatur den Mauszeiger Steuern und auf klicken, doppelklicken, ziehen und ablegen klicken. Durch das Drücken von NumLock kann der Benutzer die numerische Tastatur zwischen dem Maus Steuerungs Modus und dem normalen Vorgang umschalten. <br/>                                                                                                                                                                                                                                                                                                                                                                                               |
| [**Serialkeys**](/windows/win32/api/winuser/ns-winuser-serialkeysa)<br/>       | Enthält Informationen über das serialkeys-Barrierefreiheits Feature, das Daten aus einer Kommunikationshilfe interpretiert, die einem seriellen Anschluss als Befehle zugeordnet ist, die bewirken, dass das System Tastatur-und Maus Eingaben simuliert. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**Hat Dienst**](/windows/win32/api/winuser/ns-winuser-soundsentrya)<br/>     | Enthält Informationen über die Barrierefreiheits Funktion von SoundSentry. Wenn die Funktion "Sound Entry" aktiviert ist, zeigt der Computer nur beim Generieren eines Sounds einen visuellen Hinweis an.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**"StickyKeys"**](/windows/win32/api/winuser/ns-winuser-stickykeys)<br/>       | Enthält Informationen zum Barrierefreiheits Feature "StickyKeys". Wenn das Feature "StickyKeys" aktiviert ist, kann der Benutzer eine Modifizierertaste drücken (UMSCHALTTASTE, STRG oder alt) und dann einen anderen Schlüssel nacheinander anstatt gleichzeitig, um verschobene (geänderte) Zeichen und andere Tastenkombinationen einzugeben.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**"-Schlüssel"**](/windows/win32/api/winuser/ns-winuser-togglekeys)<br/>       | Enthält Informationen zum Barrierefreiheits Feature "degglekeys". Wenn das ToggleKeys-Feature aktiviert ist, gibt der Computer immer dann einen großen Ton aus, wenn der Benutzer die Feststell Taste, die Num-Taste oder die scrollsperrtaste schaltet, und wenn der Benutzer einen dieser Schlüssel deaktiviert. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                       |
