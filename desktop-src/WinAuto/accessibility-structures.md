# <a name="accessibility-structures"></a>Barrierefreiheitsstrukturen

In diesem Abschnitt werden die Strukturen beschrieben, die zum Implementieren Windows Barrierefreiheitsfunktionen verwendet werden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Struktur                                         | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ACCESSTIMEOUT**](/windows/win32/api/winuser/ns-winuser-accesstimeout)<br/> | Enthält Informationen zum Time out-Zeitraum, der den Microsoft Win32-Barrierefreiheitsfeatures zugeordnet ist. <br/> Der Time outzeitraum für die Barrierefreiheit ist die Zeitspanne, die ohne Tastatur- und Mauseingabe verlaufzeiten muss, bevor das Betriebssystem Barrierefreiheitsfunktionen automatisch deaktiviert. Das Barrierefreiheits-Time out ist für Computer konzipiert, die von mehreren Benutzern gemeinsam genutzt werden, sodass die von einem Benutzer ausgewählten Optionen einen nachfolgenden Benutzer nicht belästigt.<br/> Die vom Time out betroffenen Barrierefreiheitsfeatures sind die FilterKeys-Features (SlowKeys, BounceKeys und RepeatKeys), MouseKeys, ToggleKeys und StickyKeys. Das Time out für die Barrierefreiheit wirkt sich auch auf die Einstellung für den Modus mit hohem Kontrast aus.<br/> |
| [**FILTERKEYS**](/windows/win32/api/winuser/ns-winuser-filterkeys)<br/>       | Enthält Informationen zur FilterKeys-Barrierefreiheitsfunktion, mit der Benutzer mit Behinderungen die Wiederholungsrate der Tastatur (RepeatKeys), die Akzeptanzverzögerung (SlowKeys) und die Absprungrate (BounceKeys) festlegen können. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**Highcontrast**](/windows/win32/api/winuser/ns-winuser-highcontrasta)<br/>   | Enthält Informationen zur Barrierefreiheitsfunktion mit hohem Kontrast.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**Mousekeys**](/windows/win32/api/winuser/ns-winuser-mousekeys)<br/>         | Enthält Informationen zum Barrierefreiheitsfeature MouseKeys. Wenn das MouseKeys-Feature aktiv ist, kann der Benutzer die numerische Tastatur verwenden, um den Mauszeiger zu steuern und zu klicken, zu doppelklicken, zu ziehen und zu ablegen. Durch Drücken von NUMLOCK kann der Benutzer die numerische Tastatur zwischen dem Maussteuerungsmodus und dem normalen Betrieb umschalten. <br/>                                                                                                                                                                                                                                                                                                                                                                                               |
| [**SERIALKEYS**](/windows/win32/api/winuser/ns-winuser-serialkeysa)<br/>       | Enthält Informationen zur SerialKeys-Barrierefreiheitsfunktion, die Daten aus einer Kommunikationshilfe interpretiert, die an einen seriellen Anschluss angefügt ist, als Befehle, die das System dazu bringen, Tastatur- und Mauseingaben zu simulieren. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**SOUNDSENTRY**](/windows/win32/api/winuser/ns-winuser-soundsentrya)<br/>     | Enthält Informationen zur SoundSentry-Barrierefreiheitsfunktion. Wenn die SoundSentry-Funktion ein on ist, zeigt der Computer nur dann eine visuelle Anzeige an, wenn ein Sound generiert wird.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**STICKYKEYS**](/windows/win32/api/winuser/ns-winuser-stickykeys)<br/>       | Enthält Informationen zum StickyKeys-Barrierefreiheitsfeature. Wenn das StickyKeys-Feature ein on ist, kann der Benutzer eine Modifizierertaste (UMSCHALT, STRG oder ALT) und dann eine andere Taste nacheinander statt gleichzeitig drücken, um umschalte (geänderte) Zeichen und andere Tastenkombinationen ein- und ein.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**TOGGLEKEYS**](/windows/win32/api/winuser/ns-winuser-togglekeys)<br/>       | Enthält Informationen zum ToggleKeys-Barrierefreiheitsfeature. Wenn das ToggleKeys-Feature aktiviert ist, gibt der Computer immer dann einen tonierten Ton aus, wenn der Benutzer die CAPS-SPERRE, NUM-SPERRE oder SCROLL-SPERRE einschaltet, und ein tonarmer Ton, wenn der Benutzer einen dieser Schlüssel ausschaltet. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                       |
