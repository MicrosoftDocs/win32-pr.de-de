# <a name="object-roles"></a>Objekt Rollen

In diesem Thema werden die Konstanten Werte beschrieben, die zum Beschreiben der Rollen verschiedener UI-Objekte in einer Anwendung verwendet werden. Die Rollen Konstanten werden in der Datei "Oleacc. h" definiert.

Sie sollten nur die aufgeführten Rollen Konstanten verwenden, die hier aufgelistet sind. Fügen Sie keine benutzerdefinierten Rollen oder Rollen hinzu, die nicht vordefiniert sind.

Vor der Verwendung dieser Objekt Rollen müssen Entwickler von Client Anwendungen das Tool "über [prüfen](inspect-objects.md) " verwenden, um zu überprüfen, ob Objekt Rollen von Benutzeroberflächen Elementen verwendet werden.

Zum Abrufen der Rolle eines Objekts rufen Clients die Methode [**IAccessible:: get \_ accRole**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accrole) auf, die einen der in der Tabelle beschriebenen Werte zurückgeben muss. Um eine lokalisierte Zeichenfolge abzurufen, die die Rolle des Objekts beschreibt, rufen Clients die [**getroletext**](/windows/desktop/api/Oleacc/nf-oleacc-getroletexta) -Funktion mit dem Rollen Wert auf. Die lokalisierten Zeichen folgen für die Rollen Werte befinden sich in der oleaccrc.dll-Datei.
