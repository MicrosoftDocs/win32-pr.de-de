# <a name="object-roles"></a>Objektrollen

In diesem Thema werden die konstanten Werte beschrieben, mit denen die Rollen verschiedener Benutzeroberflächenobjekte in einer Anwendung beschrieben werden. Die Rollenkonstanten werden in der Datei Oleacc.h definiert.

Sie sollten nur die hier aufgeführten Rollenkonstanten verwenden. Fügen Sie keine benutzerdefinierten Rollen oder Rollen hinzu, die nicht vordefiniert sind.

Vor der Verwendung dieser Objektrollen müssen Entwickler von Clientanwendungen das [Tool Inspect](inspect-objects.md) verwenden, um zu überprüfen, ob Objektrollen von Benutzeroberflächenelementen verwendet werden.

Um die Rolle eines Objekts abzurufen, rufen Clients die [**IAccessible::get \_ accRole-Methode**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accrole) auf, die einen der in der Tabelle beschriebenen Werte zurückgeben muss. Clients rufen die [**GetRoleText-Funktion**](/windows/desktop/api/Oleacc/nf-oleacc-getroletexta) mit dem Rollenwert auf, um eine lokalisierte Zeichenfolge abzurufen, die die Rolle des Objekts beschreibt. Die lokalisierten Zeichenfolgen für die Rollenwerte befinden sich in der datei oleaccrc.dll.
