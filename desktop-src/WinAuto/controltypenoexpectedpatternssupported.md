# <a name="controltypenoexpectedpatternssupported"></a>Controltypoexpectedpatternssupported

## <a name="text"></a>Text

Das Element mit dem ControlType-Element {0} sollte mindestens eines der folgenden Muster unterstützen: {1} dieses Element wird jedoch nicht verwendet.

## <a name="type"></a>type

Fehler

## <a name="description"></a>BESCHREIBUNG

Die Benutzeroberflächenautomatisierungs-Unterstützung entspricht nicht der Benutzeroberflächenautomatisierungs-Spezifikation, da das Element mindestens eine der bereitgestellten Steuerelement Muster unterstützen soll, dies jedoch nicht. Folglich ist dieses Element möglicherweise nicht ordnungsgemäß für Hilfstechnologien verfügbar, was sich schwerwiegende Auswirkungen auf die Benutzer Leistung haben könnte.

## <a name="possible-causes"></a>Mögliche Ursachen

-   Unvollständige Implementierung der Benutzeroberflächen Automatisierung.
-   Die ControlType-Eigenschaft ist nicht ordnungsgemäß festgelegt.

## <a name="remarks"></a>Bemerkungen

Die Zugriffs Überprüfung protokolliert diese Fehlermeldung für eine unbestimmtes Statusanzeige. Sie können diese Meldung ignorieren und/oder Sie der Liste der Unterdrückungen hinzufügen.
