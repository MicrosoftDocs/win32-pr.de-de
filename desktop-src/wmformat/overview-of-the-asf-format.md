<span data-ttu-id="84411-107">Das Advanced Systems Format (ASF) ist ein erweiterbares Dateiformat, das in erster Linie für das Speichern und wiedergeben synchronisierter digitaler Medienströme und die Übertragung über Netzwerke entwickelt wurde.</span><span class="sxs-lookup"><span data-stu-id="84411-107">The Advanced Systems Format (ASF) is an extensible file format designed primarily for storing and playing synchronized digital media streams and transmitting them over networks.</span></span> <span data-ttu-id="84411-108">Bei der ASF-Datei handelt es sich um das Containerformat für Windows Media Audio und Windows Media Video basierten Inhalt.</span><span class="sxs-lookup"><span data-stu-id="84411-108">ASF is the container format for Windows Media Audio and Windows Media Video-based content.</span></span> <span data-ttu-id="84411-109">Die-Erweiterung WMA oder WMV wird verwendet, um eine ASF-Datei anzugeben, die Inhalte enthält, die mit den Windows Media Audio-und/oder Windows Media Video Codecs codiert sind.</span><span class="sxs-lookup"><span data-stu-id="84411-109">The extension wma or wmv is used to specify an ASF file that contains content encoded with the Windows Media Audio and/or Windows Media Video codecs.</span></span> <span data-ttu-id="84411-110">Das Windows Media-Format-SDK kann zum Erstellen und Lesen von Windows Media-Dateien sowie von ASF-Dateien verwendet werden, die andere Typen von komprimierten oder unkomprimierten Daten enthalten.</span><span class="sxs-lookup"><span data-stu-id="84411-110">The Windows Media Format SDK can be used to create and read Windows Media files, as well as ASF files that contain other types of compressed or uncompressed data.</span></span>

<span data-ttu-id="84411-114">Die Hauptziele des ASF-Formats lauten:</span><span class="sxs-lookup"><span data-stu-id="84411-114">The primary goals of the ASF format are:</span></span>

<span data-ttu-id="84411-120">Eine ASF-Datei kann mehrere unabhängige oder abhängige Streams enthalten, einschließlich mehrerer Audiodatenströme für Multichannel-Audiodaten oder der Videodaten Ströme mit mehreren Bitraten, die für die Übertragung über verschiedene Bandbreiten geeignet sind.</span><span class="sxs-lookup"><span data-stu-id="84411-120">An ASF file can contain multiple independent or dependent streams, including multiple audio streams for multichannel audio, or multiple bit rate video streams suitable for transmission over different bandwidths.</span></span> <span data-ttu-id="84411-121">Die Streams können in einem beliebigen komprimierten oder unkomprimierten Format vorliegen. die beste Komprimierung wird jedoch mit den Codecs der Serie Microsoft Windows Media Audio und Video 9 erzielt.</span><span class="sxs-lookup"><span data-stu-id="84411-121">The streams can be in any compressed or uncompressed format; <span data-ttu-id="84411-125">Eine ASF-Datei ist in Abschnitte organisiert, die als "Objekte" bezeichnet werden.</span><span class="sxs-lookup"><span data-stu-id="84411-125">An ASF file is organized into sections called "objects."</span></span> <span data-ttu-id="84411-126">Es gibt drei Objekte der obersten Ebene, ein Header Objekt und ein Datenobjekt (beides erforderlich) sowie ein optionales <span data-ttu-id="84411-133">Jedes Objekt der obersten Ebene oder einer niedrigeren Ebene beginnt mit einem Globally Unique Identifier (GUID) und einem Größen Wert.</span><span class="sxs-lookup"><span data-stu-id="84411-133">Each top-level or lower-level object begins with a globally unique identifier (GUID) and a size value.</span></span> <span data-ttu-id="84411-134">Diese Zahlen ermöglichen es dem Datei Leser, die Informationen an geeigneten Stellen in identifizierbare Objekte zu analysieren.</span><span class="sxs-lookup"><span data-stu-id="84411-134">These numbers allow the file reader to parse the information at appropriate places into identifiable objects.</span></span> <span data-ttu-id="84411-135">Aufgrund dieser GUIDs können Objekte auf niedrigerer Ebene in beliebiger Reihenfolge gesendet und dennoch erkannt werden.</span><span class="sxs-lookup"><span data-stu-id="84411-135">Because of these GUIDs, lower-level objects can be sent in any order and still be recognized.</span></span> <span data-ttu-id="84411-136">Das ASF-Format ist darauf ausgelegt, einen ungenauen Datenempfang zu überwinden</span><span class="sxs-lookup"><span data-stu-id="84411-136">The ASF format is designed to overcome inaccurate data reception.</span></span> <span data-ttu-id="84411-137">Eine teilweise heruntergeladene ASF-Datei kann noch gelesen werden, solange Sie das Header Objekt und mindestens ein Datenobjekt enthält.</span><span class="sxs-lookup"><span data-stu-id="84411-137">A partially downloaded ASF file can still be read, as long as it contains the Header object and at least one Data object.</span></span>
