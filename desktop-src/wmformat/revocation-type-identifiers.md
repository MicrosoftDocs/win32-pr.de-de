# <a name="revocation-type-identifiers"></a>Sperrtyp Bezeichner

Die in der folgenden Tabelle aufgeführten Konstanten definieren GUID-Werte, mit denen unterschiedliche Sperr Typen identifiziert werden.



| Konstante                         | BESCHREIBUNG                                                                                         |
|----------------------------------|-----------------------------------------------------------------------------------------------------|
| WMDRM \_ - \_ Sperrentyp- \_ App     | Identifiziert Sperrinformationen zu Anwendungs Zertifikaten.                           |
| WMDRM \_ - \_ Sperrentyp \_ Gerät  | Identifiziert Sperrinformationen in Bezug auf Gerätezertifikate.                                |
| WMDRM \_ - \_ Sperrentyp- \_ kardea  | Identifiziert Sperrinformationen in Bezug auf Windows Media DRM für Netzwerkgeräte Zertifikate. |
| WMDRM \_ - \_ Sperrentyp \_ revinfo | In dieser Version nicht dokumentiert.                                                                     |
| WMDRM \_ - \_ Sperrentyp ( \_ GRL)     | In dieser Version nicht dokumentiert.                                                                     |
| WMDRM \_ - \_ Sperrentyp \_ HDCP    | In dieser Version nicht dokumentiert.                                                                     |
