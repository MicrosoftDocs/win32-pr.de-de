# <a name="revocation-type-identifiers"></a>Bezeichner des Sperrtyps

Die in der folgenden Tabelle aufgeführten Konstanten definieren GUID-Werte, die verschiedene Sperrtypen identifizieren.



| Konstante                         | Beschreibung                                                                                         |
|----------------------------------|-----------------------------------------------------------------------------------------------------|
| \_ \_ \_ WMDRM-SPERRTYP-APP     | Identifiziert Sperrinformationen zu Anwendungszertifikaten.                           |
| \_WMDRM-SPERRTYP \_ \_ GERÄT  | Identifiziert Sperrinformationen zu Gerätezertifikaten.                                |
| WMDRM \_ REVOCATION \_ TYPE \_ CARDEA  | Identifiziert Sperrinformationen zu Windows Medien-DRM für Netzwerkgerätezertifikate. |
| WMDRM \_ REVOCATION \_ TYPE \_ REVINFO | In dieser Version nicht dokumentiert.                                                                     |
| \_WMDRM-SPERRTYP \_ \_ GRL     | In dieser Version nicht dokumentiert.                                                                     |
| \_WMDRM-SPERRTYP \_ \_ HDCP    | In dieser Version nicht dokumentiert.                                                                     |
