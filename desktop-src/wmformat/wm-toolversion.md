# <a name="wmtoolversion"></a>WM/Toolversion

Das Attribut **WM/Toolversion** enthält die Versionsnummer der Anwendung, die zum Erstellen der Datei verwendet wurde. Wenn Sie dieses Attribut verwenden, sollten Sie auch den Namen der Anwendung mithilfe von " [**WM/Toolname**](wm-toolname.md)" einschließen.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmtoolversion

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**
