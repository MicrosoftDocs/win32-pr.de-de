# <a name="wmtoolversion"></a>WM/ToolVersion

Das **WM/ToolVersion-Attribut** enthält die Versionsnummer der Anwendung, die zum Erstellen der Datei verwendet wird. Wenn Sie dieses Attribut verwenden, sollten Sie auch den Namen der Anwendung mit [**WM/ToolName angeben.**](wm-toolname.md)

## <a name="global-constant"></a>Globale Konstante

g \_ wszWMToolVersion

## <a name="data-type"></a>Datentyp

**\_WMT-TYPZEICHENFOLGE \_**
