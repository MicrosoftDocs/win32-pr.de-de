# <a name="disabling-digital-output-deprecated"></a>Deaktivieren der digitalen Ausgabe (veraltet)

Diese Seite dokumentiert eine Funktion, die in zukünftigen Versionen von Windows mit einer anderen technischen Lösung unterstützt wird.

Ein weiteres Feature von Microsoft Secure-Audiopfad ist die Möglichkeit, die digitale Ausgabe zu deaktivieren. Bestimmte Soundkarten bieten eine Möglichkeit, digitale Ausgaben von einem Hardware Gerät zu senden. Diese Geräte sind möglicherweise zertifiziert und übergeben die Autorisierung von der DRM-Kernel Komponente. Wenn also eine legitime Soundkarte mit aktivierter digitaler Ausgabe ausgestattet ist, können Menschen immer noch eine perfekte digitale Aufzeichnung geschützter Musik erfassen.

Mit dieser Funktion können Inhalts Besitzer oder Lizenz Aussteller die digitale Ausgabe deaktivieren, indem Sie einen Parameter in Lizenzen für Ihre Musik festlegen. Wenn dieser Parameter festgelegt ist, deaktiviert die Soundkarte seine digitale Ausgabefunktion, wenn geschützte Musik gespielt wird. Benutzer können entschlüsselte Musik lauschen, aber keine Kopien erstellen.
