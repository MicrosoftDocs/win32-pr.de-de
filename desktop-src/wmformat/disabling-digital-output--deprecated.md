# <a name="disabling-digital-output-deprecated"></a>Deaktivieren der digitalen Ausgabe (veraltet)

Auf dieser Seite wird ein Feature dokumentiert, das in zukünftigen Versionen von Windows mit einer anderen technischen Lösung unterstützt wird.

Ein weiteres Feature von Microsoft Secure Audio Path ist die Möglichkeit, die digitale Ausgabe zu deaktivieren. Bestimmte Soundkarten bieten eine Möglichkeit, digitale Ausgaben von einem Hardwaregerät zu senden. Diese Geräte sind möglicherweise zertifiziert und übergeben die Autorisierung von der DRM-Kernelkomponente. Wenn also für eine legitime Soundkarte die digitale Ausgabe aktiviert ist, können Benutzer weiterhin eine perfekte digitale Aufzeichnung geschützter Musik erfassen.

Mit diesem Feature können Inhaltsbesitzer oder Lizenzaussteller die digitale Ausgabe deaktivieren, indem sie einen Parameter in Lizenzen für ihre Musik festlegen. Wenn dieser Parameter festgelegt ist, deaktiviert die Soundkarte die digitale Ausgabefunktion beim Wiedergeben geschützter Musik. Benutzer können auf entschlüsselte Musik lauschen, aber keine Kopien erstellen.
