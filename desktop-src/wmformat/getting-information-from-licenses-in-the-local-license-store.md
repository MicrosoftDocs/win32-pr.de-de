# <a name="getting-information-from-licenses-in-the-local-license-store"></a>Informationen aus Lizenzen im lokalen Lizenz Speicher erhalten

Die erweiterten APIs des Windows Media DRM-Clients bieten mehrere Möglichkeiten, Informationen zu den rechten zu erhalten, die von Lizenzen im lokalen Lizenz Speicher gewährt werden. In den folgenden Themen wird beschrieben, wie Sie diese verschiedenen Arten von Informationen erhalten.



| Thema                                                                                                  | BESCHREIBUNG                                                                                                                                                                     |
|--------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Abfragen von einfachen Rechte Informationen](querying-for-simple-rights-information.md)                   | Hier wird beschrieben, wie Sie einfache Informationen dazu erhalten, ob bestimmte Aktionen für geschützte Inhalte zulässig sind. Beispielsweise kann eine Anwendung eine Mediendatei abspielen.                    |
| [Abfragen ausführlicher Rechte Informationen](querying-for-detailed-rights-information.md)               | Hier wird beschrieben, wie detaillierte aggregierte Informationen zu den rechten, die von allen Lizenzen im lokalen Lizenz Speicher gewährt werden, für ein geschütztes Inhalts Element mithilfe der Schlüssel-ID erhalten werden. |
| [Auflisten von Lizenzen im lokalen Lizenz Speicher](enumerating-licenses-in-the-local-license-store.md) | Hier wird beschrieben, wie Sie einzelne Lizenzen aus dem lokalen Lizenz Speicher erhalten.                                                                                                          |
