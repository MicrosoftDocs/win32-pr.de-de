# <a name="getting-information-from-licenses-in-the-local-license-store"></a>Abrufen von Informationen aus Lizenzen im Store

Die erweiterten APIs des Windows Media DRM-Clients bieten mehrere Möglichkeiten, Informationen zu den Rechten abzurufen, die von Lizenzen im lokalen Lizenzspeicher gewährt werden. In den folgenden Themen wird beschrieben, wie Sie diese verschiedenen Arten von Informationen abrufen.



| Thema                                                                                                  | Beschreibung                                                                                                                                                                     |
|--------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Abfragen einfacher Rechteinformationen](querying-for-simple-rights-information.md)                   | Beschreibt, wie Sie einfache Informationen darüber erhalten, ob bestimmte Aktionen für geschützte Inhalte zulässig sind. Kann eine Anwendung beispielsweise eine Mediendatei wiedergeben?                    |
| [Abfragen ausführlicher Rechteinformationen](querying-for-detailed-rights-information.md)               | Beschreibt, wie Sie mithilfe der Schlüssel-ID detaillierte aggregierte Informationen zu den Rechten abrufen, die von allen Lizenzen im lokalen Lizenzspeicher für einen geschützten Inhalt gewährt werden. |
| [Aufzählen von Lizenzen im Store](enumerating-licenses-in-the-local-license-store.md) | Beschreibt das Abrufen einzelner Lizenzen aus dem lokalen Lizenzspeicher.                                                                                                          |
