# <a name="wmmediaissubtitled"></a>WM/mediaissubtitel

Das Attribut **WM/mediaissubmit dem Titel** gibt an, ob der Inhalt Untertitel enthält.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmmediaissubtitel

## <a name="data-type"></a>Datentyp

**WMT- \_ Typ \_ bool**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut wird für Broadcast-TV verwendet.
