# <a name="export-application-certificate"></a>Exportieren eines Anwendungszertifikats

Um Windows durch Medien-DRM geschützten Inhalt zu exportieren, müssen Sie ein Windows Medien-DRM-Exportanwendungszertifikat abrufen, das Sie nach dem Abschluss der DRM-Exportvereinbarung mit Microsoft erhalten.
