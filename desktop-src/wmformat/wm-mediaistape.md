# <a name="wmmediaistape"></a>WM/mediaistape

Das **WM/mediaistape-** Attribut gibt an, ob der Inhalt von einem zuvor aufgezeichneten Band übertragen wurde.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmmediaistape

## <a name="data-type"></a>Datentyp

**WMT- \_ Typ \_ bool**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut wird für Broadcast-TV verwendet.
