# <a name="wmencodingsettings"></a>WM/encodingsettings

Das **WM/encodingsettings-** Attribut enthält die Einstellungen, die zum Codieren des Inhalts verwendet werden.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmencodingsettings

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**

## <a name="remarks"></a>Bemerkungen

Es gibt kein Standardformat zum Angeben der Codierungs Einstellungen. Es ist jedoch vorzuziehen, die Codierungs Einstellungen in englischer Sprache zu beschreiben, anstatt ein Dump der Einstellungen einzuschließen.
