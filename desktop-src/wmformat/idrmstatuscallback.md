# <a name="idrmstatuscallback-interface"></a>Idrmstatus Callback-Schnittstelle

Die **idrmstatuscallback** -Schnittstelle stellt eine Methode zum Empfangen von Status für asynchrone DRM-Aufrufe bereit.

## <a name="members"></a>Member

Die **idrmstatus Callback** -Schnittstelle erbt von der [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) -Schnittstelle. **Idrmstatus Callback** verfügt auch über die folgenden Typen von Membern:

-   [Methoden](#methods)

### <a name="methods"></a>Methoden

Die **idrmstatus Callback** -Schnittstelle verfügt über diese Methoden.



| Methode                                          | BESCHREIBUNG                                                      |
|:------------------------------------------------|:-----------------------------------------------------------------|
| [**OnStatus**](idrmstatuscallback-onstatus.md) | Empfängt Statusmeldungen von asynchronen Prozessen.<br/> |
