# <a name="building-filter-graphs-to-write-asf-files-qasf"></a>Erstellen von Filterdiagrammen zum Schreiben von ASF-Dateien (QASF)

Anwendungen, die auf DirectShow basieren, verwenden in der Regel einen von drei grundlegenden Arten von Filterdiagrammen, wenn Windows medienbasierten Inhalt erstellen:

-   Filtern Sie Diagramme zum Konvertieren oder Transcodieren von Inhalten aus einem anderen Format in Windows Medienformat.
-   Filterdiagramme zum Einfügen von Inhalten, die nicht Windows medienbasierten (nativen Streamformaten) in ASF-Dateien.
-   Filtern Sie Diagramme, um Livedaten zu erfassen und sofort in Windows Medienformat zu codieren, bevor Sie sie auf dem Datenträger speichern.

Jeder Filterdiagrammtyp wird in den folgenden Abschnitten ausführlicher beschrieben.



| `Section`                                                                                                             | BESCHREIBUNG                                                                                           |
|---------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [Transcodieren von Dateien (QASF)](transcoding-files--qasf.md)                                                             | Beschreibt, wie Filterdiagramme für die Dateitranscodierung erstellt werden.                                               |
| [Einfügen nativer Streamformate in ASF-Dateien (QASF)](inserting-native-stream-formats-into-asf-files--qasf.md)   | Beschreibt, wie sie komprimierte oder nicht komprimierte Audio- oder Videodaten in eine ASF-Datei platzieren. |
| [Direkte Erfassung von einem Gerät in einer ASF-Datei (QASF)](capturing-directly-from-a-device-to-an-asf-file--qasf.md) | Beschreibt, wie Erfassungsfilterdiagramme erstellt werden, die in eine ASF-Datei ausgegeben werden.                             |
