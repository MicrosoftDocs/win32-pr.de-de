# <a name="live-drm"></a>Live DRM

Ab Windows Media 9-Serie kann das Windows Media Format SDK zum Schreiben von DRM-geschützten Dateien verwendet werden. Dies ermöglicht Szenarien wie z. b. Live DRM, bei denen der DRM-Schutz auf eine Datei oder einen Stream angewendet wird, während Sie codiert wird, und nicht nach Abschluss der Codierung als nach Verarbeitungsschritt. Live DRM ermöglicht Inhalts Besitzern, Live Übertragungen zu schützen, z. b. bei Konzerten mit Zahlen pro Ansicht und Sportveranstaltungen. Live DRM stellt auch eine vereinfachte oder optimierte Möglichkeit zum Erstellen geschützter Dateien dar. Sie können mit Live DRM eine beliebige Version des DRM-Schutzes anwenden. Eine geschützte Datei, die mit Live DRM erstellt wurde, ist identisch mit einer Datei, die als nach Verarbeitungsschritt geschützt ist.

**Hinweis** DRM wird von der x64-basierten Version dieses SDK nicht unterstützt.
