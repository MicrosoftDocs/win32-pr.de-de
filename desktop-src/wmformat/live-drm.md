# <a name="live-drm"></a>Live-DRM

Ab Windows Media 9-Serie kann das Windows Media Format SDK zum Schreiben von DRM-geschützten Dateien verwendet werden. Dies ermöglicht Szenarien wie Live-DRM, in denen DRM-Schutz auf eine Datei oder einen Stream angewendet wird, während sie codiert wird, anstatt als Nachbearbeitungsschritt nach Abschluss der Codierung. Live-DRM ermöglicht Inhaltsbesitzern den Schutz von Liveübertragungen, z.B. Veranstaltungen mit der bezahlungsbasierten Bezahlung oder Sportereignissen. Live-DRM stellt auch eine vereinfachte oder optimierte Methode zum Erstellen geschützter Dateien dar. Sie können Live DRM verwenden, um eine beliebige Version des DRM-Schutzes anzuwenden. Eine geschützte Datei, die mit Live DRM erstellt wurde, entspricht genau der Datei, die als Nachbearbeitungsschritt geschützt ist.

**Hinweis** DRM wird von der x64-basierten Version dieses SDK nicht unterstützt.
