# <a name="wmmediaispremiere"></a>WM/mediaiische miere

Das Attribut **WM/mediaispmiere** gibt an, ob der Inhalt der erste einer Reihe ist.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmmediaispree miere

## <a name="data-type"></a>Datentyp

**WMT- \_ Typ \_ bool**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut wird für Broadcast-TV verwendet.
