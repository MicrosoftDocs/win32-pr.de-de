# <a name="bannerimagetype"></a>Bannerimagetype

Das Attribut " **bannerimagetype** " gibt den Typ des Bilds an, das im Attribut " [**bannerimagedata**](bannerimagedata.md) " gespeichert ist. Dieses Attribut ist ein Wert aus dem Enumerationstyp des [**WMT- \_ Image \_ Typs**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_image_type) .

**WMT- \_ Typ \_ DWORD**
