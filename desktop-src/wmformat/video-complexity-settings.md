# <a name="video-complexity-settings"></a>Videokomplexität Einstellungen

Einige Videocodecs unterstützen mehrere *Komplexitätsstufen.* Die Komplexitätsstufe, die zum Codieren eines Streams verwendet wird, wirkt sich nicht direkt auf die Bitrate dieses Streams aus, kann sich jedoch auf die Qualität auswirken. Wenn die Bitrate ein Maß für die Größe der komprimierten Stichproben in der ASF-Datei ist, ist die Komplexitätsebene ein Maß für die Verarbeitungsleistung, die zum Rekonstruieren der komprimierten Daten erforderlich ist. Die Komplexitätsstufe wird daher mehr von der Verarbeitungsleistung der Wiedergabeplattform als von der verfügbaren Bandbreite bestimmt.

Informationen zum Arbeiten mit Komplexitätsstufen finden Sie unter [Configuring Video Streams](configuring-video-streams.md).
