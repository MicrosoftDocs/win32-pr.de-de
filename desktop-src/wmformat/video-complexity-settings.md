# <a name="video-complexity-settings"></a>Video Komplexitäts Einstellungen

Einige Video Codecs unterstützen mehrere *Komplexitäts Ebenen*. Der Komplexitäts Grad, der zum Codieren eines Streams verwendet wird, wirkt sich nicht direkt auf die Bitrate dieses Streams aus, kann jedoch die Qualität beeinflussen. Wenn die Bitrate ein Maß für die Größe der komprimierten Stichproben in der ASF-Datei ist, ist die Komplexitäts Stufe ein Maß für die Verarbeitungsleistung, die zum Rekonstruieren der komprimierten Daten erforderlich ist. Der Komplexitäts Grad wird daher durch die Verarbeitungsleistung der Wiedergabe Plattform besser vorgegeben als durch die verfügbare Bandbreite.

Weitere Informationen zum Arbeiten mit Komplexitäts Ebenen finden Sie unter [Konfigurieren von Videostreams](configuring-video-streams.md).
