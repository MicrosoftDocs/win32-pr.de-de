# <a name="drm_actionallowed_playback"></a>DRM- \_ Aktions zulässige \_ Wiedergabe

Das DRM-Attribut für die **\_ Aktions zulässige \_ Wiedergabe** gibt an, ob die Wiedergabe des Inhalts zulässig ist.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmdrm- \_ Wiedergabe, Aktions zulässig \_

## <a name="data-type"></a>Datentyp

**WMT- \_ Typ \_ bool**

## <a name="remarks"></a>Bemerkungen

Dies ist eine schreibgeschützte Eigenschaft, die mithilfe von [**iwmdrmreader:: getdrmproperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty)abgerufen wird.
