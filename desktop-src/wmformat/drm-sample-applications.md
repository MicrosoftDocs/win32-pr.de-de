# <a name="microsoft-windows-media-drm-client-sample-applications"></a>Beispielanwendungen für Microsoft Windows Media DRM-Client

Die Beispielanwendungen, die die erweiterten APIs des Windows Media DRM-Clients verwenden, werden als Projekte für Microsoft Visual Studio 2005 bereitgestellt.

In der folgenden Tabelle werden die Beispiele zusammengefasst. Weitere Informationen zu den einzelnen Beispielen, einschließlich der Verwendungs Informationen, finden Sie in der readme.txt-Datei in jedem Beispiel Verzeichnis.



| Beispiel      | BESCHREIBUNG                                                                                                                                 |
|-------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| Queryrights | Gibt die Rechte an, die von Lizenzen auf dem Client Computer erteilt wurden, um eine Liste der Kinder zu erhalten. Veranschaulicht sowohl schnelle Abfragen als auch Lizenz Zustands Abfragen. |
| Drmupdate   | Führt eine Individualisierung durch und erstellt ein Protokoll aller Ereignisse, die während des Prozesses ausgelöst werden.                                                   |
