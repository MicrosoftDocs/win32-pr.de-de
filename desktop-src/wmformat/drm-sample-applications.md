# <a name="microsoft-windows-media-drm-client-sample-applications"></a>Microsoft Windows Media DRM-Clientbeispielanwendungen

Die Beispielanwendungen, die die Windows Media DRM Client Extended APIs verwenden, werden als Projekte für Microsoft Visual Studio 2005 bereitgestellt.

In der folgenden Tabelle sind die Beispiele zusammengefasst. Zusätzliche Informationen zu den einzelnen Beispielen, einschließlich Nutzungsinformationen, finden Sie in der readme.txt in jedem Beispielverzeichnis.



| Beispiel      | BESCHREIBUNG                                                                                                                                 |
|-------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| QueryRights | Meldet die Rechte, die durch Lizenzen auf dem Clientcomputer gewährt werden, für eine Liste von KIDs. Veranschaulicht sowohl schnelle Abfragen als auch Lizenzstatusabfragen. |
| DRMUpdate   | Führt eine Individualisierung durch und erstellt ein Protokoll aller Ereignisse, die während des Prozesses ausgelöst werden.                                                   |
