# <a name="reading-metadata-at-playback"></a>Lesen von Metadaten bei der Wiedergabe

Sowohl das asynchrone Readerobjekt als auch das synchrone Readerobjekt können die Metadaten aus dem Header einer geladenen ASF-Datei lesen. Beim Lesen von Dateien sind alle Metadatenattribute schreibgeschützt. Beide Readerobjekte können für die [**Schnittstellen IWMHeaderInfo**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmheaderinfo) und [**IWMHeaderInfo2 abgefragt**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmheaderinfo2) werden.

Weitere Informationen zum Zugriff auf Metadaten finden Sie unter [Arbeiten mit Metadaten.](working-with-metadata.md)
