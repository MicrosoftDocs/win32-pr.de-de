data-ttu-id="ccb92-114">Eingaben müssen in einem unterstützten Format übermittelt werden.</span><span class="sxs-lookup"><span data-stu-id="ccb92-114">Inputs must be delivered in a supported <span data-ttu-id="ccb92-119">In einigen Fällen können komprimierte Audiodaten und Videodaten in einen Eingabestream übermittelt werden.</span><span class="sxs-lookup"><span data-stu-id="ccb92-119">In <span data-ttu-id="ccb92-126">Der Begriff "Stream" in dieser Dokumentation bezieht sich auf Daten in einer ASF-Datei im Gegensatz zu (1) den Eingabe Quelldaten, bevor Sie vom Writer-Objekt verarbeitet data-ttu-id="ccb92-127">Ein ASF-Datenstrom enthält Daten, die aus einer einzelnen Eingabe für das Writer-Objekt stammen, obwohl mehr als ein Stream aus derselben Eingabe erstellt werden data-stu-id="ccb92-128">A stream has the same format and compression settings from beginning to end.</span></span> <span data-ttu-id="ccb92-129">Eine einfache ASF-Datei verfügt über zwei Streams: einen für Audiodaten und einen für Video.</span><span class="sxs-lookup"><span data-stu-id="ccb92-129">A simple ASF file has two streams, one for audio and one for video.</span></span> <span data-ttu-id="ccb92-130">Eine komplexere Datei kann über zwei Audiostreams und mehrere Videostreams verfügen.</span><span class="sxs-lookup"><span data-stu-id="ccb92-130">A more complex file might have class="sxs-lookup"><span data-stu-id="ccb92-133">The media format and compression settings that the writer object will apply to each stream are specified in the profile.</span></span>

<span data-ttu-id="ccb92-152">Es gibt einige Abweichungen in den zuvor beschriebenen Beziehungen.</span><span class="sxs-lookup"><span data-stu-id="ccb92-152">There is some variation in the relationships described previously.</span></span> <span data-ttu-id="ccb92-153">Eine Datei kann z. b. alle drei Beziehungen oder eine oder zwei Beziehungen enthalten.</span><span class="sxs-lookup"><span data-stu-id="ccb92-153">For example, a file can contain all three relationships, or one or two of them.</span></span> <span data-ttu-id="ccb92-154">Es ist auch möglich, dass einige Eingaben komprimiert werden. in diesem Fall führt der Writer keine zusätzliche Komprimierung durch.</span><span class="sxs-lookup"><span data-stu-id="ccb92-154">It is also possible for some inputs to be compressed, in which case the writer performs no additional compression.</span></span> <span data-ttu-id="ccb92-155">Der Reader kann auch komprimierte Beispiele bereitzustellen.</span><span class="sxs-lookup"><span data-stu-id="ccb92-155">The reader, also, can deliver compressed samples.</span></span> <span data-ttu-id="ccb92-156">Wenn dies der Fall ist, müssen Sie über die streamnummer und nicht über die Ausgabe Nummer darauf zugreifen.</span><span class="sxs-lookup"><span data-stu-id="ccb92-156">But when it does, you must access them by stream number, not by output number.</span></span>
