| Attributtyp                                                   | Beschreibung                                                           |
|------------------------------------------------------------------|-----------------------------------------------------------------------|
| [Attribute für Musik Dateien](attributes-for-music-files.md)     | Listet Vorschläge für Attribute auf, die mit Musikdateien verwendet werden können.             |
| [Attribute für Videodateien](attributes-for-video-files.md)     | Listet Vorschläge für Attribute auf, die mit Videodateien verwendet werden können.             |

| [Multicaststationsattribute](multicast-station-attributes.md) | Listet die Attribute auf, die der Datei von einer Multicaststation überlagert werden. |
