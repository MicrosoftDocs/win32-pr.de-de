# <a name="playlist-burning"></a>Wiedergabelisten brennen

Windows Media DRM 10-Lizenzen unterstützen das Kopieren von Wiedergabelisten geschützter Dateien auf DVD für rotbook. Lizenz Aussteller können die Häufigkeit begrenzen, mit der Inhalt in einer Wiedergabeliste auf die CD kopiert werden kann, und zwar so oft wie oft eine bestimmte Wiedergabeliste kopiert werden kann. Weitere Informationen zu den Wiedergabelisten-rechten in Lizenzen finden Sie in der Dokumentation zum Windows Media Rights Manager 10-SDK.

Das SDK für den Windows Media-Format bietet die Möglichkeit, alle Dateien in einer Wiedergabeliste schnell zu überprüfen, um sicherzustellen, dass Sie für das Brennen von Wiedergabelisten lizenziert sind.

**Hinweis** DRM wird von der x64-basierten Version dieses SDK nicht unterstützt.
