# <a name="isvbr"></a>IsVBR

Das **IsVBR-Attribut** gibt an, ob der Inhalt in einem Stream mit VBR-Codierung (Variable Bit Rate) codiert wurde.

## <a name="global-constant"></a>Globale Konstante

g \_ wszWMIsVBR

## <a name="data-type"></a>Datentyp

**\_WMT-TYP \_ BOOL**

## <a name="remarks"></a>Hinweise

Wenn Sie auf die **IWMHeaderInfo3-Schnittstelle** des Writer-Objekts zugreifen, können Sie diesen Wert hinzufügen oder ändern. In anderen Objekten (Metadaten-Editor, Reader und synchroner Reader) ist dieser Wert schreibgeschützt.
