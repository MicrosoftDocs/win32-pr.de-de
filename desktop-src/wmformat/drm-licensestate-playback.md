# <a name="drm_licensestate_playback"></a>DRM \_ LicenseState \_ Playback

Die **DRM \_ LicenseState \_ Playback-Eigenschaft** enthält eine [**WM LICENSE STATE \_ \_ \_ DATA-Struktur,**](/previous-versions/windows/desktop/legacy/dd757942(v=vs.85)) die Details dazu enthält, wie dieses Recht auf den Inhalt angewendet wurde.

## <a name="global-constant"></a>Globale Konstante

g \_ wszWMDRM \_ LicenseState \_ Playback

## <a name="data-type"></a>Datentyp

**\_WMT-TYP \_ BINARY**

## <a name="remarks"></a>Hinweise

Dies ist eine schreibgeschützte Eigenschaft, die mit [**IWMDRMReader::GetDRMProperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty)abgerufen wird.
