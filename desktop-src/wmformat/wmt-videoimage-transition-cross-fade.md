# <a name="wmt_videoimage_transition_cross_fade"></a>WMT \_ VIDEOIMAGE \_ TRANSITION \_ CROSS \_ FADE

Der Übergang zum Überblenden hat keine besonderen Auswirkungen. die Merkmale der Ausblendung (Entschärtung) werden durch die **Member fPrevBlendCoef** und **fCurrBlendCoef** der [**WMT \_ VIDEOIMAGE \_ SAMPLE2-Struktur**](/previous-versions/windows/desktop/api/Wmsdkidl/ns-wmsdkidl-wmt_videoimage_sample2) bestimmt.

## <a name="parameters"></a>Parameter

Dieser Übergang nimmt keine Parameter an.

## <a name="remarks"></a>Hinweise

Das Festlegen dieses Übergangs entspricht dem Festlegen des **dwEffectType-Elements** der **WMT \_ VIDEOIMAGE \_ SAMPLE2-Struktur** auf 0.
