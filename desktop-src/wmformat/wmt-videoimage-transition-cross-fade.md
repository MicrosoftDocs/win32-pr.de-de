# <a name="wmt_videoimage_transition_cross_fade"></a>Kreuz Ausblenden des WMT- \_ Videobilds \_ \_ \_

Der Cross-Fade-Übergang hat keine besonderen Auswirkungen. die Merkmale der Fade (auflösen) werden von den Membern **fprevblendcoef** und **fcurrblendcoef** der [**WMT \_ Videoimage \_ SAMPLE2**](/previous-versions/windows/desktop/api/Wmsdkidl/ns-wmsdkidl-wmt_videoimage_sample2) -Struktur bestimmt.

## <a name="parameters"></a>Parameter

Dieser Übergang nimmt keine Parameter an.

## <a name="remarks"></a>Bemerkungen

Das Festlegen dieses Übergangs ist das gleiche wie das Festlegen des **dweffecttype** -Members der **WMT \_ Videoimage \_ SAMPLE2** -Struktur auf 0.
