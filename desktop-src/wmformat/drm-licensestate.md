# <a name="drm_licensestate"></a>DRM- \_ Lizenzstatus

Die " **DRM \_ licenanstate** "-Eigenschaft wird nicht verwendet. Dabei handelt es sich eigentlich nicht um eine Eigenschaft, sondern vielmehr um die Stamm Zeichenfolge für die anderen Zeichen folgen Bezeichner für den Lizenz Zustand, z. b. die **DRM- \_ Lizenzstatus \_ Wiedergabe**.

## <a name="global-constant"></a>Globale Konstante
