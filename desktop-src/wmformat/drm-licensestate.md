# <a name="drm_licensestate"></a>DRM \_ LicenseState

Die **\_ DRM-LicenseState-Eigenschaft** wird nicht verwendet. Es handelt sich eigentlich überhaupt nicht um eine Eigenschaft, sondern lediglich um die Stammzeichenfolge für die anderen Zeichenfolgenbezeichner der Lizenzzustandseigenschaft, z. B. **DRM \_ LicenseState \_ Playback.**

## <a name="global-constant"></a>Globale Konstante

g \_ wszWMDRM \_ LicenseState

## <a name="data-type"></a>Datentyp

**\_WMT-TYP \_ BINARY**

## <a name="remarks"></a>Hinweise

Diese Eigenschaft gibt keine Informationen zurück.
