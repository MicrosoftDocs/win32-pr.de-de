# <a name="numberofframes"></a>NumberOfFrames

Das **NumberOfFrames-Attribut** enthält die Anzahl der Frames in einem Videostream.

## <a name="global-constant"></a>Globale Konstante

g \_ wszWMNumberOfFrames

## <a name="data-type"></a>Datentyp

**WMT \_ TYPE \_ QWORD**

## <a name="remarks"></a>Hinweise

Der Writer fügt diesen Wert für einen Videostream zum Zeitpunkt der Codierung hinzu. **NumberOfFrames** ist nicht schreibgeschützt. Sie sollten den Wert jedoch nur ändern, wenn Sie die Datei bearbeiten und die Anzahl der Frames in einem Videostream ändern.
