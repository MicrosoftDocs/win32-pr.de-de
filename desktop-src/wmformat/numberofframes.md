# <a name="numberofframes"></a>NumberOfFrames

Das Attribut " **numofframes** " enthält die Anzahl der Frames in einem Videostream.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmnumofframes

## <a name="data-type"></a>Datentyp

**WMT- \_ Typ \_ QWORD**

## <a name="remarks"></a>Bemerkungen

Der Writer fügt diesen Wert für einen Videostream zum Zeitpunkt der Codierung hinzu. " **Numofframes** " ist nicht schreibgeschützt. Sie sollten den Wert jedoch nur ändern, wenn Sie die Datei bearbeiten, indem Sie die Anzahl der Frames in einem Videostream ändern.
