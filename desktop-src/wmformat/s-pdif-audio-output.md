# <a name="spdif-audio-output"></a>S/PDIF-Audioausgabe

Einige Audiogeräte verwenden S/PDIF (Digital Interconnect Format) zum Transport von Audiodaten. Sie können nun Audiodaten aus Audiodaten, die mit dem Media Windows-Codec codiert wurden, Professional in S/PDIF decodieren.
