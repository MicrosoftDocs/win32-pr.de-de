# <a name="spdif-audio-output"></a>S/PDIF-Audioausgabe

Für einige Audiogeräte wird das digitale Interconnect-Format (S/PDIF) von Sony/Philips verwendet, um Audiodaten zu transportieren. Sie können jetzt audiocode in S/PDIF von einem audiocode decodieren, der mit dem Windows Media Professional-Codec codiert wurde.
