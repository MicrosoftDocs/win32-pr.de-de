# <a name="wmcontentgroupdescription"></a>WM/contentgroupdescription

Das Attribut **WM/contentgroupdescription** enthält eine Beschreibung der Inhalts Gruppe.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmcontentgroupdescription

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**

## <a name="remarks"></a>Bemerkungen

Inhalts Gruppen sind Sammlungen von Medien, zu denen einzelne Dateien gehören können. Eine Audiodatei kann z. b. ein Titel sein, bei dem es sich um eine Spur auf einer kompakten CD handelt, die Teil einer festgelegten Gruppe ist. Ebenso kann ein Videoclip Teil einer Reihe sein.
