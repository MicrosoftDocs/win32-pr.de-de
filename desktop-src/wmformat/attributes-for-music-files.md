# <a name="attributes-for-music-files"></a>Attribute für Musikdateien

In diesem Abschnitt werden die Attribute aufgelistet, die häufig für Audiodateien mit Musik verwendet werden. Es wird empfohlen, dass Sie die Attribute für Dateien entsprechend diesen Listen festlegen, um sicherzustellen, dass Ihre Dateien mit einer Vielzahl von Wiedergabe Anwendungen vollständig kompatibel sind. Die Attribute in diesem Abschnitt sind in drei Kategorien aufgeführt: primär, Sekundär und Tertiär.

Primäre Attribute vermitteln die grundlegendsten Informationen zu einer Datei. Wenn Sie Audiodateien für die Verteilung erstellen, ist dies der minimale Satz von Attributen, die Sie verwenden sollten.

Sekundäre Attribute enthalten allgemeine Metadaten, die für alle Audiodateien wichtig, aber nicht universell sind.

Tertiäre Attribute sollten bei Bedarf eingeschlossen werden, sind jedoch für die Beschreibung der Datei nicht unbedingt erforderlich.
