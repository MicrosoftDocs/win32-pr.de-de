# <a name="attributes-for-music-files"></a>Attribute für Musik Dateien

In diesem Abschnitt werden die Attribute aufgelistet, die häufig für Audiodateien mit Musik verwendet werden. Es wird empfohlen, Attribute für Dateien gemäß diesen Listen festzulegen, um sicherzustellen, dass Ihre Dateien vollständig mit einer Vielzahl von Wiedergabeanwendungen kompatibel sind. Die Attribute in diesem Abschnitt sind in drei Kategorien aufgeführt: primär, sekundär und sekundär.

Primäre Attribute vermitteln die grundlegendsten Informationen zu einer Datei. Wenn Sie Audiodateien für die Verteilung erstellen, ist dies der Mindestsatz von Attributen, die Sie verwenden sollten.

Sekundäre Attribute enthalten allgemeine Metadaten, die wichtig, aber nicht für alle Audiodateien universell sind.

Bei Bedarf sollten auch die attributen eingeschlossen werden, die für die Beschreibung der Datei jedoch nicht wesentlich sind.
