# <a name="wmpeakbitrate"></a>WM/PeakBitrate

Das **WM/PeakBitrate-Attribut** enthält die Spitzenbitrate eines VBR-Streams (Variable Bit Rate) in Bits pro Sekunde.

## <a name="global-constant"></a>Globale Konstante

g \_ wszWMPeakBitrate

## <a name="data-type"></a>Datentyp

**\_WMT-TYP \_ DWORD**
