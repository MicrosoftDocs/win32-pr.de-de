# <a name="sdk-versions"></a>SDK-Versionen

Diese Dokumentation behandelt das Windows Media Format 7 SDK über das Windows Media Format 11 SDK. Einzelne Methoden, Strukturen, Funktionen und Enumerationen enthalten Versionsanforderungen in ihren Referenzthemen.

Das Windows Media Format 11 SDK wird nur unter Windows Vista und Windows XP unterstützt.

Das Windows Media Format 9.5 SDK wird nur auf Windows XP unterstützt. Wenn Sie eine Anwendung basierend auf dem Windows Media Format 9.5 SDK schreiben, wird Ihre Anwendung nicht unter einem anderen Betriebssystem ausgeführt.

Einige Features des SDK der Windows Media Format 9-Serie erfordern auch Windows XP. In den einzelnen Referenzthemen zu diesen Features wird diese Anforderung genannt.
