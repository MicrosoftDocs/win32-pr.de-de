# <a name="sdk-versions"></a>SDK-Versionen

In dieser Dokumentation wird das SDK für Windows Media-Format 7 über das Windows Media Format 11 SDK behandelt. Einzelne Methoden, Strukturen, Funktionen und Enumerationen enthalten Versions Anforderungen in ihren Referenz Themen.

Das Windows Media Format 11 SDK wird nur unter Windows Vista und Windows XP unterstützt.

Das Windows Media-Format 9,5 SDK wird nur unter Windows XP unterstützt. Wenn Sie eine Anwendung schreiben, die auf dem Windows Media-Format 9,5 SDK basiert, kann die Anwendung nicht unter einem anderen Betriebssystem ausgeführt werden.

Für einige Features des SDK für Windows Media-Serie 9 ist auch Windows XP erforderlich. Die einzelnen Referenz Themen im Zusammenhang mit diesen Features stellen diese Anforderung dar.
