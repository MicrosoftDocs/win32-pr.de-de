# <a name="video-image"></a>Videobild

Mit dem Windows Media Video 9-Abbild v2-Codec können Sie Windows Media Video Dateien erstellen, deren Inhalt aus einer Reihe von Bildern wie z. b. digitalen Fotos besteht. Die Bilder im Video Bild Datenstrom können so bearbeitet werden, dass Sie sowohl Pfannen als auch Zoom und eine Reihe von Übergängen erzeugt. Endbenutzer können diese Funktion verwenden, um einfache Folien anzeigen zu erstellen und freizugeben, die auch einen Musik-Sound Sound oder eine Voice-over-Erzählung enthalten können. Aufgrund der übergeordneten Komprimierung des Windows Media Video Bildcodecs ist eine Datei mit mehreren Fotos immer noch klein genug, um per e-Mail über das Internet eine e-Mail zu senden.

Ein videobildstream verfügt über einen eigenen Medientyp und eine spezialisierte Struktur, die die verschiedenen Übergänge definiert, die bei der Umstellung von einem Bild auf das nächste möglich sind. Sie können die Ansicht der Bilder schwenken und vergrößern, während Sie kombiniert werden. Die resultierende Datei kann dann wie jede andere Windows Media-Videodatei abgespielt werden.
