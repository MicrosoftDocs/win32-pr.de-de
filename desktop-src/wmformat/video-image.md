# <a name="video-image"></a>Videobild

Mit dem Codec Windows Media Video 9 Image v2 können Sie Windows Medienvideodateien erstellen, deren Inhalt aus einer Reihe von Standbildern wie digitalen Fotos besteht. Die Bilder im Videobildstream können bearbeitet werden, um Schwenken und Zooms sowie eine Reihe von Übergängen zu erzeugen. Endbenutzer können diese Funktion verwenden, um einfache Diashows zu erstellen und freizugeben, die auch eine Musik- oder Sprachausgabe enthalten können. Aufgrund der höheren Komprimierung des Windows Media Video Image-Codecs ist eine Datei mit mehreren Fotos immer noch klein genug, um bequem E-Mails über das Internet zu senden.

Ein Videobildstream verfügt über einen eigenen Medientyp und eine spezielle Struktur, die die verschiedenen Übergänge definiert, die beim Wechsel von einem Bild zum nächsten möglich sind. Sie können die Ansicht der Bilder schwenken und zoomen, während Sie sie zusammenführen. Die resultierende Datei kann dann wie jede andere Windows Medienvideodatei wiedergegeben werden.
