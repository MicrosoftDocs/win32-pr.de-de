# <a name="wmlyrics"></a>WM/Liedtexte

Das **WM/Lyrics-** Attribut enthält die Texte als einfache Zeichenfolge.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmlyrics

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**

## <a name="remarks"></a>Bemerkungen

Verwenden Sie für eine strukturiertere Speicherung von Texten das [**\_ synchronisierte WM/Lied-**](wm-lyrics-synchronised.md) Attribut.
