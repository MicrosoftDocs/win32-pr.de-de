# <a name="viewing-drm-attributes-in-the-metadata-editor"></a>Anzeigen von DRM-Attributen im Metadaten-Editor

Der Metadaten-Editor macht die IWMDRMEditor-Schnittstelle verfügbar, die es Bearbeitungsanwendungen ermöglicht, bestimmte Attribute eines DRM-Headers in einer geschützten ASF-Datei und Attribute in der Inhaltslizenz zu untersuchen, sofern vorhanden, auch wenn die Anwendung nicht über die DRM-spezifische statische Bibliothek (wmstubdrm.lib) verfügen, die für vollständig aktivierte DRM-Player- und Writeranwendungen erforderlich ist.

> [!Note]  
> DRM wird von der x64-basierten Version dieses SDK nicht unterstützt.
