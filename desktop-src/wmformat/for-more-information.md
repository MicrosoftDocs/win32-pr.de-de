# <a name="for-more-information-windows-media-format-sdk"></a>Weitere Informationen (Windows Media Format SDK)

-   Aktuelle Informationen zu bekannten Problemen mit dem Windows Media-Format-SDK finden Sie in den Anmerkungen zu dieser Version auf der [MSDN Library-Website](/documentation/?url=%2flibrary%2fdnwmt%2fhtml%2fwmfsdk11_b2_release_notes_rzxb.asp%3fframe%3dtrue).
-   Weitere Informationen zu Features, Artikeln und Antworten zu häufig gestellten Fragen zum Streamen von Medien im allgemeinen und zu den Windows Media sdchen finden Sie im Windows Media Developer Center auf der [Microsoft-Website](https://msdn.microsoft.com/windowsmedia/default.aspx).
-   Weitere Informationen zum Erwerb einer Lizenz für das Windows Media-Format-SDK finden Sie auf der [Microsoft-Website](https://msdn.microsoft.com/library/ms867178.aspx)auf der Seite zur Windows Media-Lizenzierung.
-   Weitere Informationen zu Newsgroups finden Sie auf der [Microsoft](https://windows.microsoft.com/windows/products/windows-media)-Website auf der Seite Windows Media-Newsgroups und e-Mail-Aliase.
-   Ausführliche Beschreibungen der Datenobjekte, aus denen sich die ASF-Dateien bilden, finden Sie in der [Microsoft-Website](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).
-   Weitere Informationen zu Features, Artikeln, bewährten Methoden und Antworten auf häufig gestellte Fragen zu Digital Rights Management (DRM) und Windows Media Rights Manager finden Sie in den technischen Artikeln auf der Seite " [Digital Rights Management](/documentation/?url=%2flibrary%2fdnwmt%2fhtml%2fdigitalrightsmanagement.asp) " auf der Microsoft-Website.
