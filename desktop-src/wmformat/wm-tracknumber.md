# <a name="wmtracknumber"></a>WM/tracknumber

Das **WM/tracknumber-** Attribut enthält die Nachverfolgung der Anzahl der Inhalte. Dieses Attribut ist 1-basiert.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmtracknumber

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**

## <a name="remarks"></a>Bemerkungen

Viele vorhandene Anwendungen schreiben den Wert für **WM/tracknumber** als **DWORD**. Wenn Sie eine Anwendung erstellen, die Dateien aus unbekannten Quellen wieder gibt, sollten Sie Code einschließen, um sowohl Zeichen folgen-als auch **DWORD** -Werte zu verarbeiten.
