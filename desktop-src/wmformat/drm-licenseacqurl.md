# <a name="drm_licenseacqurl"></a>DRM- \_ Lizenz-AcqURL

Das Attribut " **DRM \_ licenseacqurl** " enthält die Adresse einer Webseite, zu der die Client Anwendung navigieren kann, um eine Inhalts Lizenz für den DRM-Inhalt der Version 7 zu erhalten.

## <a name="global-constant"></a>Globale Konstante

"g \_ wszwmdrm \_ licenanacqurl"

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut kann mithilfe von [**iwmdrmwriter:: setdrmattribute**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) festgelegt werden, und es kann mit " [**iwmdrmreader:: getdrmproperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty)" abgerufen werden. Das gleiche Datei Attribut kann mit [**DRM \_ drmHeader \_ licenanacqurl**](drm-drmheader-licenseacqurl.md)abgerufen werden.
