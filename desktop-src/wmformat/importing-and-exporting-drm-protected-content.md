# <a name="importing-and-exporting-drm-protected-content"></a>Importieren und Exportieren von DRM-geschütztem Inhalt

Eine der Hauptrollen der neuen erweiterten APIs für den Windows Media DRM-Client besteht darin, den Import und Export von DRM-geschützten Inhalten in Rechteverwaltungssysteme von Drittanbietern zu aktivieren.

Die Hauptaufgabe des Importierens und Exportierens geschützter Daten in Ihrer Anwendung besteht darin, die ASF-Daten Stichproben für das System und das DRM-Subsystem zu verwenden. Wenn Ihre Anwendung Daten Beispiele verarbeitet, muss sichergestellt werden, dass bestimmte Stabilitäts Standards erfüllt werden, damit der Schutzmechanismus nicht umgangen werden kann. Diese Standards werden in den Stabilitäts-und Konformitäts Regeln erläutert, denen Sie bei der Lizenzierung von DRM zustimmen.
