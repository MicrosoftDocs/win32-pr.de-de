# <a name="importing-and-exporting-drm-protected-content"></a>Importieren und Exportieren von DRM-geschützten Inhalten

Eine der Hauptrollen der neuen erweiterten APIs des Windows Media DRM-Clients ist das Aktivieren des Imports und Exports von DRM-geschützten Inhalten in Rechteverwaltungssysteme von Drittanbietern.

Die wichtigste Herausforderung beim Importieren und Exportieren geschützter Daten in Ihrer Anwendung besteht darin, ASF-Datenbeispiele stabil zwischen Ihrem System und dem DRM-Subsystem zu teilen. Wenn Ihre Anwendung Datenbeispiele verarbeitet, muss sichergestellt werden, dass bestimmte Stabilitätsstandards erfüllt werden, um zu vermeiden, dass der Schutzmechanismus umgangen werden kann. Diese Standards werden in den Robustheits- und Konformitätsregeln beschrieben, denen Sie zustimmen, wenn Sie DRM lizenzieren.
