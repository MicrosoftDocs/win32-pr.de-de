# <a name="drm_actionallowed"></a>DRM- \_ Aktions zulässig

Die **DRM \_ Action Allowed** -Eigenschaft ist die Basis Zeichenfolge für die anderen "action Allowed"-Eigenschafts Zeichenfolgen-Konstanten, wie z. b. DRM \_ Action Allowed \_ copydecd, da die Eigenschaft "action Allowed" nicht erweiterbar ist. diese Basis Zeichenfolge wird nicht für Anwendungen verwendet.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmdrm- \_ Aktions zulässig

## <a name="data-type"></a>Datentyp

Kein Datentyp.

## <a name="remarks"></a>Bemerkungen

"Aktions zugelassene" Eigenschaften sind zurzeit nicht erweiterbar.
