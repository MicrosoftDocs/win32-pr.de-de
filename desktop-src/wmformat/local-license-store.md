# <a name="local-license-store"></a>Lokaler Lizenz Speicher

Wenn eine DRM-Lizenz an den Client Computer übermittelt wird, wird Sie im lokalen Lizenz Speicher abgelegt. Dabei handelt es sich um eine geschützte Datei auf der Festplatte, die alle für den Benutzer ausgegebenen Lizenzen sowie andere DRM-Informationen enthält.

Sie können Daten im lokalen Lizenz Speicher auf einige beschränkte Weise mithilfe der erweiterten APIs des Windows Media DRM-Clients bearbeiten.

Zusätzlich zum lokalen Lizenz Speicher verwenden einige Objekte einen temporären Lizenz Speicher. Eine Lizenz in einem temporären Speicher ist nur für kurze Zeit vorhanden. Allerdings können einige Lizenzen, die in einem temporären Speicher beginnen, im regulären lokalen Lizenz Speicher gespeichert werden.
