# <a name="local-license-store"></a>Lokale Lizenz Store

Wenn eine DRM-Lizenz an den Clientcomputer übermittelt wird, wird sie im lokalen Lizenzspeicher abgelegt. Dies ist eine geschützte Datei auf der Festplatte, die alle lizenzen enthält, die dem Benutzer zusammen mit anderen DRM-Informationen ausgestellt wurden.

Sie können Daten im lokalen Lizenzspeicher mithilfe der erweiterten APIs des Windows Media DRM-Clients auf verschiedene, eingeschränkte Weise bearbeiten.

Zusätzlich zum lokalen Lizenzspeicher verwenden einige Objekte einen temporären Lizenzspeicher. Eine Lizenz in einem temporären Speicher ist nur für einen kurzen Zeitraum vorhanden. Einige Lizenzen, die in einem temporären Speicher beginnen, können jedoch im regulären lokalen Lizenzspeicher gespeichert werden.
