# <a name="drm_crypto_type-enumeration"></a>DRM \_ - \_ kryptografietypenumeration

Die **DRM \_ - \_ kryptografietyp** -Enumeration definiert die kryptografiealgorithmustypen, die zum Verschlüsseln von Inhalten verwendet werden können

## <a name="syntax"></a>Syntax


```C++
typedef enum DRM_CRYPTO_TYPE { 
  CRYPTO_TYPE_MCE  = 0
} ;
```



## <a name="constants"></a>Konstanten

<dl> <dt>

<span id="CRYPTO_TYPE_MCE"></span><span id="crypto_type_mce"></span>**kryptografietyp- \_ \_ MCE**
</dt> <dd>

Gibt die AES-Verschlüsselung (128-bit-Counter-Advanced Encryption Standard Modus) an.

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

Keine.
