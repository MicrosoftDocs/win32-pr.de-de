# <a name="wmmediaisdelay"></a>WM/mediaisdelay

Das **WM/mediaisdelay-** Attribut gibt an, ob der Inhalt als Band Verzögerung gesendet wurde.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmmediaisdelay

## <a name="data-type"></a>Datentyp

**WMT- \_ Typ \_ bool**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut wird für Broadcast-TV verwendet.
