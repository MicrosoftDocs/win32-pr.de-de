# <a name="windows-media-drm-10-for-network-devices"></a>Windows Media DRM 10 für Netzwerkgeräte

Ihre Anwendung kann geschützte Inhalte mithilfe von Windows Media DRM 10 für Netzwerkgeräte an digitale Medienempfänger in einem Netzwerk streamen. Bei diesem Feature handelt es sich um ein Protokoll für die sichere Kommunikation mit Netzwerkgeräten.

Das Windows Media-Format-SDK unterstützt Windows Media DRM 10 für Netzwerkgeräte, indem eine Datenbank von Netzwerkgeräten verwaltet, die Netzwerk Nähe dieser Geräte überprüft und DRM-geschützte Inhalte in ein sicheres Transport Format umgerechnet werden.

> [!Note]  
> DRM wird von der x64-basierten Version dieses SDK nicht unterstützt.
