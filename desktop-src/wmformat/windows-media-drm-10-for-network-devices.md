# <a name="windows-media-drm-10-for-network-devices"></a>Windows Medien-DRM 10 für Netzwerkgeräte

Ihre Anwendung kann geschützte Inhalte an Empfänger digitaler Medien in einem Netzwerk streamen, indem Windows Media DRM 10 für Netzwerkgeräte verwendet wird. Dieses Feature ist ein Protokoll für die sichere Kommunikation mit Netzwerkgeräten.

Das Windows Media Format SDK unterstützt Windows Media DRM 10 für Netzwerkgeräte, indem eine Datenbank mit Netzwerkgeräten beibehalten wird, die Netzwerknähe dieser Geräte überprüfen und DRM-geschützte Inhalte in ein geschütztes Transportformat konvertiert werden.

> [!Note]  
> DRM wird von der x64-basierten Version dieses SDK nicht unterstützt.
