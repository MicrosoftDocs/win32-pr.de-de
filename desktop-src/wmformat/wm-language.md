# <a name="wmlanguage"></a>WM/Sprache

Das **WM/Language-** Attribut enthält die Sprache des Datenstroms.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmlanguage

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**

## <a name="remarks"></a>Bemerkungen

Die verwendete sprach Zeichenfolge sollte mit RFC 1766 kompatibel sein. Weitere Informationen finden Sie unter [sprach](language-strings.md)Zeichenfolgen.

### <a name="example"></a>Beispiel



| Dateityp | Beispielwert |
|-----------|---------------|
| Alle       | "en-US"       |
