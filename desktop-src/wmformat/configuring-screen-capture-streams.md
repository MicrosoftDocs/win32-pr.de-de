# <a name="configuring-screen-capture-streams"></a>Konfigurieren von Bildschirmaufnahme-Streams

Streams, die den Windows Media® Video 9 Screen-Codec verwenden, werden von Anwendungen auf die gleiche Weise wie normale Videostreams konfiguriert. Wenn Sie die Videokomplexitätsebene jedoch auf 0 festlegen, ignoriert der Writer alle Videoqualitätswerte, die mit [**IWMVideoMediaProps::SetQuality festgelegt wurden.**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmvideomediaprops-setquality) Weitere Informationen finden Sie unter [Getting Good Results with the Windows Media Video 9 Screen Codec](getting-good-results-with-the-windows-media-video-9-screen-codec.md).
