# <a name="multicast-station-attributes"></a>Attribute für Multicast Station

Wenn eine Datei von einer Multicast Station gestreamt wird, kann die Station einige Attribute für die Datei erzwingen. Diese in der folgenden Tabelle aufgeführten Attribute werden nicht tatsächlich in der Datei gespeichert und sind nur verfügbar, wenn die Datei Streaming ist. Sie enthalten Informationen über die Station und sind in der Regel für den gesamten Inhalt der Station identisch.



| Multicast Station-Attribut                 | Globaler Bezeichner      | Datentyp             |
|---------------------------------------------|------------------------|-----------------------|
| [**NSC- \_ Adresse**](nsc-address.md)         | g \_ wszwmnscaddress     | **WMT \_ - \_ Typzeichenfolge** |
| [**NSC- \_ Beschreibung**](nsc-description.md) | g \_ wszwmnscdescription | **WMT \_ - \_ Typzeichenfolge** |
| [**NSC \_ -e-Mail**](nsc-email.md)             | g \_ wszwmnscemail       | **WMT \_ - \_ Typzeichenfolge** |
| [**NSC- \_ Name**](nsc-name.md)               | g \_ wszwmnscname        | **WMT \_ - \_ Typzeichenfolge** |
| [**NSC- \_ Telefon**](nsc-phone.md)             | g \_ wszwmnscphone       | **WMT \_ - \_ Typzeichenfolge** |
