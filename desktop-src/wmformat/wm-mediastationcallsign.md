# <a name="wmmediastationcallsign"></a>WM/mediastationcallsign

Das **WM/mediastationcallsign-** Attribut enthält das Aufruf Vorzeichen der Station, die den Inhalt überträgt.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmmediastationcallsign

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut wird für Broadcast Fernsehen und Radio verwendet.
