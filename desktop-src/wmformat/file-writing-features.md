# <a name="file-writing-features"></a>Funktionen zum Schreiben von Dateien

Eines der Haupt Features des Windows Media SDK ist die Möglichkeit, Dateien im Advanced Systems-Format (ASF) zu schreiben. Das Writer-Objekt wird zum Schreiben von ASF-Dateien verwendet. Weitere Informationen finden Sie unter [Writer-Objekt](writer-object.md).

Im grundlegendsten Szenario zum Schreiben von Dateien weisen Sie ein zu verwendende Profil und den Namen der zu erstellenden Datei zu. Sie übergeben Beispiele nacheinander an den Writer. Wenn Sie die Übergabe von Beispielen an den Writer abgeschlossen haben, werden die zugehörigen Vorgänge abgeschlossen und die ASF-Datei abgeschlossen. Weitere Informationen zum Schreiben von grundlegenden Dateien finden Sie unter [Schreiben von ASF-Dateien](writing-asf-files.md).
