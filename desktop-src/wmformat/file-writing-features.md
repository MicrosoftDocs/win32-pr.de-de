# <a name="file-writing-features"></a>Funktionen zum Schreiben von Dateien

Eines der hauptmerkmale des Windows Media Format SDK ist die Möglichkeit, Dateien im Advanced Systems Format (ASF) zu schreiben. Das Writer-Objekt wird zum Schreiben von ASF-Dateien verwendet. Weitere Informationen finden Sie unter [Writer Object](writer-object.md).

Im einfachsten Dateischreibszenario weisen Sie ein zu verwendendes Profil und einen Namen der zu erstellende Datei zu. Sie übergeben Stichproben einzeln an den Writer. Wenn Sie die Übergabe von Beispielen an den Writer abgeschlossen haben, werden die Vorgänge abgeschlossen und die ASF-Datei abgeschlossen. Weitere Informationen zum grundlegenden Schreiben von Dateien finden Sie unter [Schreiben von ASF-Dateien.](writing-asf-files.md)
