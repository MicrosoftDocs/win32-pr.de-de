# <a name="iwmsecurebuffer-interface"></a>Iwmsecurebuffer-Schnittstelle

Die **iwmsecurebuffer** -Schnittstelle verschlüsselt und entschlüsselt Datenzeiger.

## <a name="members"></a>Member

Die **iwmsecurebuffer** -Schnittstelle erbt von der [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) -Schnittstelle. **Iwmsecurebuffer** verfügt auch über die folgenden Typen von Membern:

-   [Methoden](#methods)

### <a name="methods"></a>Methoden

Die **iwmsecurebuffer** -Schnittstelle verfügt über diese Methoden.



| Methode                                     | BESCHREIBUNG                         |
|:-------------------------------------------|:------------------------------------|
| [**Entschlüsseln**](iwmsecurebuffer-decrypt.md) | Entschlüsselt einen Datenzeiger.<br/> |
| [**Verschlüsseln**](iwmsecurebuffer-encrypt.md) | Verschlüsselt einen Datenzeiger.<br/> |
