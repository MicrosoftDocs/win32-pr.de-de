# <a name="sending-asf-data-over-a-network"></a>Senden von ASF-Daten über ein Netzwerk

Das SDK für den Windows Media-Format unterstützt zwei verschiedene Methoden zum Senden von ASF-Daten über ein Netzwerk:

-   Die Anwendung kann die ASF-Daten über das HTTP-Protokoll übertragen.
-   Die Anwendung kann die ASF-Daten per Push an einen Publishing Point auf einem Windows Media-Server übersetzen. Der Server verteilt die Daten dann.

In den folgenden Themen werden diese Funktionen beschrieben:

-   [Übertragungen von ASF-Daten](broadcasting-asf-data.md)
-   [Senden von ASF-Daten an einen Publishing Point](sending-asf-data-to-a-publishing-point.md)
