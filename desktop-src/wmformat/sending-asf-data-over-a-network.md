# <a name="sending-asf-data-over-a-network"></a>Senden von ASF-Daten über ein Netzwerk

Das Windows Media Format SDK unterstützt zwei verschiedene Möglichkeiten zum Senden von ASF-Daten über ein Netzwerk:

-   Die Anwendung kann die ASF-Daten mithilfe des HTTP-Protokolls übertragen.
-   Die Anwendung kann die ASF-Daten an einen Veröffentlichungspunkt auf einem Windows Übertragen übertragen. Der Server verteilt dann die Daten.

In den folgenden Themen werden diese Features beschrieben:

-   [Übertragungen von ASF-Daten](broadcasting-asf-data.md)
-   [Senden von ASF-Daten an einen Veröffentlichungspunkt](sending-asf-data-to-a-publishing-point.md)
