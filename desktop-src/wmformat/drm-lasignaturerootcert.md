# <a name="drm_lasignaturerootcert"></a>DRM \_ lasignaturerootcert

Das **DRM- \_ lasignaturerootcert** -Attribut enthält das Stamm Zertifikat in der Zertifizierungs Kette, das verwendet wird, um das in [**DRM \_ lasignaturelicsrvcert**](drm-lasignaturelicsrvcert.md)enthaltene Zertifikat zu authentifizieren.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmdrm \_ lasignaturerootcert

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**

## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft kann mit der [**iwmdrmwriter:: setdrmattribute**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) -Methode festgelegt werden. Das Reader-Objekt ist nicht verfügbar.
