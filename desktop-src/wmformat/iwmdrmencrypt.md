# <a name="iwmdrmencrypt-interface"></a>Iwmdrmencrypt-Schnittstelle

Die **iwmdrmencrypt** -Schnittstelle bietet eine Methode zum Verschlüsseln von Daten an Ort und Stelle.

## <a name="members"></a>Member

Die **iwmdrmencrypt** -Schnittstelle erbt von der [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) -Schnittstelle. **Iwmdrmencrypt** verfügt auch über die folgenden Typen von Membern:

-   [Methoden](#methods)

### <a name="methods"></a>Methoden

Die **iwmdrmencrypt** -Schnittstelle verfügt über diese Methoden.



| Methode                                   | BESCHREIBUNG                                 |
|:-----------------------------------------|:--------------------------------------------|
| [**Verschlüsseln**](iwmdrmencrypt-encrypt.md) | Verschlüsselt einen Datenpuffer an Ort und Stelle.<br/> |
