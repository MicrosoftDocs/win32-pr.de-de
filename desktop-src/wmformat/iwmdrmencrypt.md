# <a name="iwmdrmencrypt-interface"></a>IWMDRMEncrypt-Schnittstelle

Die **IWMDRMEncrypt-Schnittstelle** stellt eine Methode zum Verschlüsseln von Daten bereit.

## <a name="members"></a>Member

Die **IWMDRMEncrypt-Schnittstelle** erbt von der [**IUnknown-Schnittstelle.**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) **IWMDRMEncrypt** verfügt auch über diese Typen von Membern:

-   [Methoden](#methods)

### <a name="methods"></a>Methoden

Die **IWMDRMEncrypt-Schnittstelle** verfügt über diese Methoden.



| Methode                                   | Beschreibung                                 |
|:-----------------------------------------|:--------------------------------------------|
| [**Encrypt**](iwmdrmencrypt-encrypt.md) | Verschlüsselt einen Datenpuffer an Ort und Stelle.<br/> |
