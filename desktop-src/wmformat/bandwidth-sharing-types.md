# <a name="bandwidth-sharing-types"></a>Bandbreiten Freigabe Typen

Sie können die Art der Freigabe von Bandbreiten Freigaben verwenden, um die Art eines Bandbreiten Freigabe Objekts in einem Profil zu ermitteln. Bandbreiten Freigabe Typen werden als Parameter für [**iwmbandwidthsharing:: GetType**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmbandwidthsharing-gettype) und [**iwmbandwidthsharing:: settype**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmbandwidthsharing-settype)verwendet.

In der folgenden Tabelle werden die Bezeichner für die Freigabe Typen für die Bandbreite aufgelistet.



| Typkonstante für Bandbreiten Freigabe      | GUID                                 |
|--------------------------------------|--------------------------------------|
| CLSID \_ wmbandwidthsharing \_ exklusiv | af6060aa-5197-11d2-b6af-00c04fd908e9 |
| CLSID \_ wmbandwidthsharing \_ partiell   | af6060ab-5197-11d2-b6af-00c04fd908e9 |
