# <a name="bandwidth-sharing-types"></a>Bandbreitenfreigabetypen

Sie können Bandbreitenfreigabetypen verwenden, um die Art eines Bandbreitenfreigabeobjekts in einem Profil zu identifizieren. Bandbreitenfreigabetypen werden als Parameter für [**IWMBandwidthSharing::GetType**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmbandwidthsharing-gettype) und [**IWMBandwidthSharing::SetType verwendet.**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmbandwidthsharing-settype)

In der folgenden Tabelle sind die Bezeichner für Bandbreitenfreigabetypen aufgeführt.



| Konstante des Bandbreitenfreigabetyps      | GUID                                 |
|--------------------------------------|--------------------------------------|
| CLSID \_ WMBandwidthSharing \_ Exclusive | af6060aa-5197-11d2-b6af-00c04fd908e9 |
| CLSID \_ WMBandwidthSharing \_ Partial   | af6060ab-5197-11d2-b6af-00c04fd908e9 |
