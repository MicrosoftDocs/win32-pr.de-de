# <a name="codec-features"></a>Codec-Features

Das Windows Media-Format-SDK wird mit verschiedenen Audio-und Video Codecs geliefert. Sie können die bereitgestellten Codecs zum Komprimieren und decoden von Inhalten verwenden, um eine Vielzahl von Anforderungen zu erfüllen. Der Codec, der vom Writer zum Komprimieren von Daten verwendet wird, wird durch Datenstrom-Konfigurationsinformationen im Profil angegeben. Informationen aus dem Profil werden dann in der Kopfzeile der vom Writer erstellten Datei gespeichert. Wenn die Datei dann vom Reader oder dem synchronen Reader geöffnet wird, identifiziert die Profilinformationen im Header den Codec, der zum Dekomprimieren der Daten erforderlich ist.
