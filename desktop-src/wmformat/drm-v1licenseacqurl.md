# <a name="drm_v1licenseacqurl"></a>DRM- \_ V1LicenseAcqURL

Das **DRM \_ V1LicenseAcqURL** -Attribut enthält die Adresse einer Website, auf der ein Benutzer eine DRM-Lizenz der Version 1 abrufen kann.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmdrm \_ V1LicenseAcqURL

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut kann mithilfe von [**iwmdrmwriter:: setdrmattribute**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) festgelegt werden, und es kann mit " [**iwmdrmreader:: getdrmproperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty)" abgerufen werden.
