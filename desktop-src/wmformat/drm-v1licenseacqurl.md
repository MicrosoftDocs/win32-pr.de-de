# <a name="drm_v1licenseacqurl"></a>DRM \_ V1LicenseAcqURL

Das **ATTRIBUT DRM \_ V1LicenseAcqURL** enthält die Adresse einer Website, auf der ein Benutzer eine DRM-Lizenz der Version 1 abrufen kann.

## <a name="global-constant"></a>Globale Konstante

g \_ wszWMDRM \_ V1LicenseAcqURL

## <a name="data-type"></a>Datentyp

**\_WMT-TYPZEICHENFOLGE \_**

## <a name="remarks"></a>Hinweise

Dieses Attribut kann [**mithilfe von IWMDRMWriter::SetDRMAttribute**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) festgelegt und mit [**IWMDRMReader::GetDRMProperty abgerufen werden.**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty)
