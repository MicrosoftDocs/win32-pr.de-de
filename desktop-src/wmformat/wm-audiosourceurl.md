# <a name="wmaudiosourceurl"></a>WM/audiosourceurl

Das **WM/audiosourceurl-** Attribut enthält die Adresse einer offiziellen Webseite des Quell Mediums. Beispielsweise kann ein Song aus einem Sound-Sound einen Link zur offiziellen Seite für den Film enthalten.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmaudiosourceurl

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**
