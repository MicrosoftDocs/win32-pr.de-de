# <a name="acquiring-licenses"></a>Erwerben von Lizenzen

Ein häufiges Szenario für den Erwerb von Lizenzen ist, wenn ein Benutzer über eine geschützte Windows Media-Datei verfügt und versucht, zum ersten Mal auf den Inhalt zuzugreifen. Da die erweiterten APIs des Windows Media DRM-Clients von der Datei Behandlungs Routinen von ASF getrennt sind, erfordert das grundlegende Lizenz Erwerbs Szenario, unterstützt, mehr API-Aufrufe als das Windows Media-Format-SDK und das Media Foundation SDK.

In diesem Abschnitt wird beschrieben, wie Sie die erweiterten APIs des Windows Media DRM-Clients zum Abrufen von Lizenzen verwenden, die im lokalen Lizenz Speicher auf einem Client Computer gespeichert sind. Es enthält die folgenden Themen.



| Thema                                                                | BESCHREIBUNG                                                                                                                                |
|----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [Automatische Lizenz Beschaffung](silent-license-acquisition.md)         | Hier wird beschrieben, wie Sie Lizenzen ohne Benutzereingriff erwerben.                                                                               |
| [Nicht automatische Lizenz Beschaffung](non-silent-license-acquisition.md) | Hier wird beschrieben, wie Sie Lizenzen erwerben, wenn ein Benutzereingriff erforderlich ist (z. b. das bezahlen für den Inhalt).                                     |
| [Vorab Bereitstellung von Lizenzen](license-pre-delivery.md)                     | Beschreibt das Abrufen von Lizenzen von einem Lizenzserver an einen Client Computer.                                                                 |
| [Lokales Erstellen von Lizenzen](creating-licenses-locally.md)           | Hier wird beschrieben, wie Sie Ihre eigenen Lizenzen erstellen, ohne Sie von einem Lizenzserver herunterzuladen und im lokalen Lizenz Speicher zu speichern. |
