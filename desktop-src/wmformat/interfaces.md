## <a name="remarks"></a>Bemerkungen

Der Schnittstellen Bezeichner für jede Schnittstelle in diesem SDK ist der Name der Schnittstelle, der "iid" vorangestellt ist \_ . Der Bezeichner für die **iwmreaderadvanced** -Schnittstelle lautet z. b. iid \_ iwmreaderadvanced. Um einen Zeiger auf die **iwmreaderadvanced** -Schnittstelle eines Reader-Objekts zu erhalten, verwenden Sie den folgenden Code, wobei der *preader* ein Zeiger auf eine **iwmreader** -Schnittstelle ist:


```C++
IWMReaderAdvanced *pReaderAdvanced;
hr = pReader->QueryInterface(IID_IWMReaderAdvanced,
                             (LPVOID*) &pReaderAdvanced);

```
