## <a name="remarks"></a>Hinweise

Der Schnittstellenbezeichner für jede der Schnittstellen in diesem SDK ist der Name der Schnittstelle, der "IID" \_ vorangegangen ist. Der Bezeichner für die **IWMReaderAdvanced-Schnittstelle** ist beispielsweise IID \_ IWMReaderAdvanced. Um einen Zeiger auf die **IWMReaderAdvanced-Schnittstelle** eines Readerobjekts zu erhalten, verwenden Sie den folgenden Code, wobei *pReader* ein Zeiger auf eine **IWMReader-Schnittstelle** ist:


```C++
IWMReaderAdvanced *pReaderAdvanced;
hr = pReader->QueryInterface(IID_IWMReaderAdvanced,
                             (LPVOID*) &pReaderAdvanced);

```
