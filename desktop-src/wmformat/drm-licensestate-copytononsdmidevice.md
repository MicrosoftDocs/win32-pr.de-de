# <a name="drm_licensestate_copytononsdmidevice"></a>DRM \_ licenanstate \_ copytononsdmidevice

Die Eigenschaft " **DRM \_ licenanstate \_ copytononsdmidevice** " enthält eine Datenstruktur mit dem [**Status "WM \_ License \_ State \_**](/previous-versions/windows/desktop/legacy/dd757942(v=vs.85)) ", die Details dazu enthält, wie dieses Recht auf den Inhalt angewendet wurde.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmdrm \_ licensstate \_ copytononsdmidevice

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ typbinär Datei**

## <a name="remarks"></a>Bemerkungen

Dies ist eine schreibgeschützte Eigenschaft, die mithilfe von [**iwmdrmreader:: getdrmproperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty)abgerufen wird.
