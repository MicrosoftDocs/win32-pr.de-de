# <a name="guid-values"></a>GUID-Werte

In den folgenden Abschnitten werden die GUID-Werte aufgelistet, die im Windows Media-Format-SDK verwendet werden, sowie die globalen Konstanten, die zur Darstellung verwendet werden.



| `Section`                                                  | BESCHREIBUNG                                                                                                               |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [Schnittstellen Bezeichner](interface-identifiers.md)       | Listet die GUID-Werte und zugeordneten Konstanten für die Schnittstellen auf, die vom Windows Media Format SDK unterstützt werden.              |
| [Medientyp-IDs](media-type-identifiers.md)     | Listet die GUID-Werte und zugeordneten Konstanten für die vom Windows Media-Format-SDK unterstützten Medientypen auf.             |
| [Gegenseitige Ausschluss Typen](mutual-exclusion-types.md)     | Listet die GUID-Werte und zugeordneten Konstanten für die vom Windows Media-Format-SDK unterstützten gegenseitigen Ausschluss Typen auf.  |
| [Bandbreiten Freigabe Typen](bandwidth-sharing-types.md)   | Listet die GUID-Werte und zugeordneten Konstanten für die vom Windows Media-Format-SDK unterstützten Bandbreiten Freigabe Typen auf. |
| [Typen von Wasserzeichen Kategorien](watermark-category-types.md) | Listet die GUID-Werte und zugeordneten Konstanten, die Wasserzeichen-DMO-Kategorien identifizieren.                                 |
