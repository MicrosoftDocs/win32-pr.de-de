# <a name="wmcategory"></a>WM/Kategorie

Das **WM/Category-** Attribut enthält die Kategorie des Inhalts. Es sind keine Spezifikationen für Kategorien vorhanden. Benutzern sollte empfohlen werden, ihre eigenen Kategorien einzugeben, um ihren Inhalt zu organisieren.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmcategory

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**

## <a name="example"></a>Beispiel



| Dateityp | Beispielwert   |
|-----------|-----------------|
| Audio     | "Musik wird angefahren" |
| Video     | "Home Videos"   |
