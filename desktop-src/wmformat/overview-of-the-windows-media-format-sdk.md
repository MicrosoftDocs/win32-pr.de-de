data-ttu-id="1f025-152">Das synchrone Reader-Objekt hat unter bestimmten Umständen mehrere Vorteile gegenüber dem Reader-Objekt, hauptsächlich für Anwendungen zur Inhalts Bearbeitung, die schnell auf
