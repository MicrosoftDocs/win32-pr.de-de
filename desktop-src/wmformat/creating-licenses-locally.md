# <a name="creating-licenses-locally"></a>Lokales Erstellen von Lizenzen

Unter bestimmten Umständen, z. b. beim [DRM-Import](drm-import.md), können Sie Ihre eigenen Lizenzen erstellen. Windows Media DRM-Lizenzen können auf verschiedene Arten geschrieben werden, aber um Ihre eigene Lizenz zu erstellen, müssen Sie das binäre XMR-Schema (Extensible Media Rights) verwenden. Weitere Informationen finden Sie unter [Building a XMR License](building-an-xmr-license.md).

Wenn Sie eine Lizenz erstellen, können Sie Sie dem lokalen Lizenz Speicher hinzufügen, indem Sie die [**iwmdrmlicensetmanagement:: storelicense**](iwmdrmlicensemanagement-storelicense.md) -Methode aufrufen.
