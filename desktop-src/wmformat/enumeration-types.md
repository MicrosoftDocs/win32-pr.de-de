# <a name="windows-media-format-sdk-enumeration-types"></a>Windows Media Format SDK-Enumerationstypen

Das Windows Media Format SDK implementiert die folgenden Enumerationstypen.



| Enumerationstyp                                                               | Beschreibung                                                                                                                      |
|--------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**\_DRM-HTTP-STATUS \_**](drm-http-status.md)                                   | Definiert einen Bereich von Zuzuständen für eine DRM-Anforderung.                                                                                     |
| [**\_ \_ DRM-LIZENZSTATUSKATEGORIE \_**](drm-license-state-category.md)            | Definiert die Kategorien für DRM-Lizenzzeichenfolgen.                                                                                  |
| [**\_DRM-INDIVIDUALISIERUNGSSTATUS \_**](drm-individualization-status.md)         | Definiert die gültigen Zustände für die DRM-Individualisierung.                                                                              |
| [**\_NETSOURCE-URLCREDPOLICY-EINSTELLUNGEN \_**](/previous-versions/windows/desktop/api/wmsinternaladminnetsource/ne-wmsinternaladminnetsource-netsource_urlcredpolicy_settings) | Definiert mögliche Sicherheitsrichtlinieneinstellungen, die auf einem Clientcomputer vorhanden sein können.                                                   |
| [**WM \_ AETYPE**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wm_aetype)                                                | Gibt die Berechtigungen für einen Eintrag in einer IP-Adresszugriffsliste an.                                                             |
| [**WMT \_ \_ ATTR-DATENTYP**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_attr_datatype)                               | Definiert einen Bereich von grundlegenden Datentypen. Diese Werte werden verwendet, um den Datentyp zu identifizieren, der von verschiedenen Methoden in diesem SDK zurückgegeben wird. |
| [**WMT \_ ATTR \_ IMAGETYPE**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_attr_imagetype)                             | Definiert einen Bereich von Imagetypen, die in einem ASF-Dateiheader gespeichert werden können.                                                         |
| [**\_ \_ WMT-CODEC-INFOTYP \_**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_codec_info_type)                          | Definiert einen Bereich von Datengrößen, die von einem Codec verwendet werden.                                                                                   |
| [**\_ \_ WMT-ANMELDEINFORMATIONSFLAGS**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_credential_flags)                         | Definiert die Flags, die beim Abrufen von Anmeldeinformationen verwendet werden können.                                                                   |
| [**WMT \_ DRMLA \_ TRUST**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_drmla_trust)                                   | Definiert den Vertrauensstatus einer DRM-Lizenzerwerbs-URL.                                                                        |
| [**WMT \_ \_ FILESINK-MODUS**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_filesink_mode)                               | Definiert die Eingabetypen, die von der Dateisenke akzeptiert werden.                                                                            |
| [**\_WMT-IMAGETYP \_**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_image_type)                                     | Definiert die möglichen Bildtypen, die für Banneranzeigen verwendet werden.                                                                            |
| [**\_WMT-INDEXTYP \_**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_index_type)                                     | Definiert das -Objekt, dem ein zeitbasierter Index zugeordnet werden kann.                                                              |
| [**WMT \_ \_ INDEXER-TYP**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_indexer_type)                                 | Definiert die vom Indexer unterstützten Indizierungstypen.                                                                          |
| [**WMT \_ \_ NET-PROTOKOLL**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_net_protocol)                                 | Definiert die Protokolltypen, die von der Netzwerksenke unterstützt werden.                                                                   |
| [**WMT \_ \_ MUSICSPEECH-KLASSENMODUS \_**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_musicspeech_class_mode)            | Definiert die Komprimierungsmodi, die für den Windows Media Audio 9 Voice-Codec verfügbar sind.                                                |
| [**WMT \_ OFFSET \_ FORMAT**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_offset_format)                               | Definiert die Typen von Offsets, die in diesem SDK verwendet werden.                                                                                   |
| [**\_WMT-WIEDERGABEMODUS \_**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_play_mode)                                       | Definiert die Wiedergabeoptionen des Readers.                                                                                      |
| [**\_WMT-PROXYEINSTELLUNGEN \_**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_proxy_settings)                             | Definiert die Netzwerkproxyeinstellungen für die Verwendung mit einem Readerobjekt.                                                                 |
| [**\_WMT-RECHTE**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_rights)                                              | Definiert die Rechte, die in einer DRM-Lizenz angegeben werden können.                                                                       |
| [**\_WMT-STATUS**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_status)                                              | Definiert einen Bereich von Statusflags.                                                                                                 |
| [**\_WMT-SPEICHERFORMAT \_**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_storage_format)                             | Definiert die Dateitypen, die mit diesem SDK bearbeitet werden können.                                                                    |
| [**WMT \_ STREAM \_ SELECTION**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_stream_selection)                         | Definiert den Status eines Streams.                                                                                                  |
| [**\_WMT-TRANSPORTTYP \_**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_transport_type)                             | Definiert die von diesem SDK unterstützten Transporttypen.                                                                               |
| [**\_WMT-VERSION**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_version)                                            | Definiert die Versionsnummern des Windows Media Format SDK.                                                                     |
| [**\_ \_ WMT-WASSERZEICHENEINTRAGSTYP \_**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_watermark_entry_type)                | Definiert die Typen der unterstützten Wasserzeichen.                                                                                       |
