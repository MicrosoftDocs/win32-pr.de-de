# <a name="windows-media-format-sdk-enumeration-types"></a>Enumerationstypen für Windows Media-Format-SDK

Das SDK für den Windows Media-Format implementiert die folgenden Enumerationstypen.



| Enumerationstyp                                                               | BESCHREIBUNG                                                                                                                      |
|--------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**DRM- \_ http- \_ Status**](drm-http-status.md)                                   | Definiert einen Bereich von Zuständen für eine DRM-Anforderung.                                                                                     |
| [**DRM- \_ Lizenz \_ Zustands \_ Kategorie**](drm-license-state-category.md)            | Definiert die Kategorien für DRM-Lizenz Zeichenfolgen.                                                                                  |
| [**DRM- \_ Individualisierungs \_ Status**](drm-individualization-status.md)         | Definiert die gültigen Zustände für die DRM-Individualisierung.                                                                              |
| [**NetSource- \_ urlkredpolicy- \_ Einstellungen**](/previous-versions/windows/desktop/api/wmsinternaladminnetsource/ne-wmsinternaladminnetsource-netsource_urlcredpolicy_settings) | Definiert mögliche Sicherheitsrichtlinien Einstellungen, die auf einem Client Computer vorhanden sein können.                                                   |
| [**WM- \_ aetype**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wm_aetype)                                                | Gibt die Berechtigungen für einen Eintrag in einer IP-Adress Zugriffsliste an.                                                             |
| [**WMT \_ attr- \_ DataType**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_attr_datatype)                               | Definiert einen Bereich von Basis Datentypen. Diese Werte werden verwendet, um den Typ der Daten zu identifizieren, die von verschiedenen Methoden in diesem SDK zurückgegeben werden. |
| [**WMT \_ attr- \_ ImageType**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_attr_imagetype)                             | Definiert einen Bereich von Bildtypen, der in einem ASF-Dateiheader gespeichert werden kann.                                                         |
| [**WMT- \_ Codec- \_ \_ Infotyp**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_codec_info_type)                          | Definiert einen Bereich von Datengrößen, die von einem Codec verwendet werden.                                                                                   |
| [**WMT- \_ Credential- \_ Flags**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_credential_flags)                         | Definiert die Flags, die beim Abrufen von Anmelde Informationen verwendet werden können.                                                                   |
| [**WMT \_ drmla- \_ Vertrauensstellung**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_drmla_trust)                                   | Definiert den Vertrauens Zustand einer DRM-Lizenz Erwerbs-URL.                                                                        |
| [**WMT- \_ filesink- \_ Modus**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_filesink_mode)                               | Definiert die Typen von Eingaben, die von der Datei Senke akzeptiert werden.                                                                            |
| [**WMT \_ - \_ Bildtyp**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_image_type)                                     | Definiert die möglichen Bild Typen, die für Banner Anzeigen verwendet werden.                                                                            |
| [**WMT \_ - \_ Indextyp**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_index_type)                                     | Definiert das-Objekt, mit dem ein zeitbasierter Index verknüpft werden kann.                                                              |
| [**WMT- \_ \_ Indexertyp**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_indexer_type)                                 | Definiert die Typen der Indizierung, die vom Indexer unterstützt werden.                                                                          |
| [**WMT \_ net- \_ Protokoll**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_net_protocol)                                 | Definiert die Typen von Protokollen, die von der Netzwerk Senke unterstützt werden.                                                                   |
| [**WMT- \_ musicspeech- \_ Klassen \_ Modus**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_musicspeech_class_mode)            | Definiert die für den Windows Media Audio 9-Sprach Codec verfügbaren Komprimierungs Modi.                                                |
| [**WMT- \_ Offset \_ Format**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_offset_format)                               | Definiert die Typen der Offsets, die in diesem SDK verwendet werden.                                                                                   |
| [**WMT- \_ Wiedergabe \_ Modus**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_play_mode)                                       | Definiert die Wiedergabe Optionen des Readers.                                                                                      |
| [**WMT- \_ Proxy \_ Einstellungen**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_proxy_settings)                             | Definiert die Netzwerk Proxy Einstellungen für die Verwendung mit einem Reader-Objekt.                                                                 |
| [**WMT- \_ Rechte**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_rights)                                              | Definiert die Rechte, die in einer DRM-Lizenz angegeben werden können.                                                                       |
| [**WMT- \_ Status**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_status)                                              | Definiert einen Bereich von Statusflags.                                                                                                 |
| [**WMT- \_ Speicher \_ Format**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_storage_format)                             | Definiert die Dateitypen, die mit diesem SDK bearbeitet werden können.                                                                    |
| [**WMT-Daten \_ Strom \_ Auswahl**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_stream_selection)                         | Definiert den Status eines Streams.                                                                                                  |
| [**WMT \_ - \_ Transporttyp**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_transport_type)                             | Definiert die Transport Typen, die von diesem SDK unterstützt werden.                                                                               |
| [**WMT- \_ Version**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_version)                                            | Definiert die Versionsnummern des SDK für den Windows Media-Format.                                                                     |
| [**Typ des WMT- \_ Wasserzeichens \_ Eintrags \_**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_watermark_entry_type)                | Definiert die Typen unterstützter Wasserzeichen.                                                                                       |
