# <a name="advanced-systems-format-asf-files"></a>ASF-Dateien (Advanced Systems Format)

Alle Objekte des Windows Media Format SDK funktionieren mit Dateien, die der ASF-Dateistruktur (Advanced Systems Format) entsprechen. Diese Dateien werden in dieser Dokumentation als ASF-Dateien bezeichnet. Weitere Informationen zur Struktur von ASF-Dateien finden Sie unter [Übersicht über das ASF-Format.](overview-of-the-asf-format.md)

Manchmal wird der Begriff *Windows Mediendatei* verwendet, um Dateien zu beschreiben, die mit diesem SDK oder anderen Windows Medientechnologien erstellt wurden. Es ist nicht genau, eine ASF-Datei als Windows Mediendatei aufzurufen, es sei denn, sie enthält nur Inhalte, die mithilfe Windows Mediencodecs codiert wurden. Weitere Informationen finden Sie in den Richtlinien für die Verwendung der Dateierweiterungen WMA und WMV im Abschnitt Richtlinien für [Dateinamenerweiterungen.](file-name-extension-guidelines.md)
