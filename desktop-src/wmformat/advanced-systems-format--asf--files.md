# <a name="advanced-systems-format-asf-files"></a>Dateien im Advanced Systems Format (ASF)

Alle Objekte des Windows Media Format SDK arbeiten mit Dateien, die der Dateistruktur von Advanced Systems Format (ASF) entsprechen. Diese Dateien werden in dieser Dokumentation als "ASF-Dateien" bezeichnet. Weitere Informationen zur Struktur von ASF-Dateien finden Sie unter [Übersicht über das ASF-Format](overview-of-the-asf-format.md).

Manchmal wird der Begriff *Windows Media File* verwendet, um Dateien zu beschreiben, die mit diesem SDK oder anderen Windows Media-Technologien erstellt wurden.
