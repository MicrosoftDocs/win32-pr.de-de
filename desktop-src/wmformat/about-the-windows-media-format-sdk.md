# <a name="about-the-windows-media-format-sdk"></a>Informationen zum Windows Media Format SDK

Mit dem Microsoft® Windows Media® Format Software Development Kit (SDK) können Entwickler Anwendungen erstellen, die digitale Medien bearbeiten, die in Dateien gespeichert sind, die der ASF-Dateistruktur (Advanced Systems Format) entsprechen. Die ASF-Struktur ist ein vielseitiges Containerformat für beliebige [*Streamingmedien.*](wmformat-glossary.md) Weitere Informationen zur Struktur von ASF-Dateien finden Sie unter [Übersicht über das ASF-Format](overview-of-the-asf-format.md).

Dieses SDK bietet auch Unterstützung für das Codieren und Decodieren von Audio- und Videoinhalten mithilfe Windows Mediencodecs. Jeder Codec (Bzw. Jeder Dekomprimierer) ist optimiert, um die beste Qualität für einen bestimmten Inhaltstyp zu erhalten. Weitere Informationen zu den unterstützten Codecs finden Sie unter [Codecfunktionen](codec-features.md).

Zusätzlich zur allgemeinen Dateiformatunterstützung und Codecintegration stellt das Windows Media Format SDK Objekte zum Bearbeiten beschreibender Informationen zu digitalen Medien bereit. Dateilese- und Metadatenunterstützung ist auch für MP3-Dateien enthalten, sodass Sie flexiblere Anwendungen erstellen können.
