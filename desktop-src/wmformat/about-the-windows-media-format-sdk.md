# <a name="about-the-windows-media-format-sdk"></a>Informationen zum Windows Media-Format-SDK

Das Microsoft® Windows Media® Format Software Development Kit (SDK) ermöglicht es Entwicklern, Anwendungen zu erstellen, die in Dateien mit der Dateistruktur Advanced Systems Format (ASF) gespeicherte digitale Medien bearbeiten. Die ASF-Struktur ist ein vielseitiges Containerformat für beliebige [*Streamingmedien*](wmformat-glossary.md) . Weitere Informationen zur Struktur von ASF-Dateien finden Sie unter [Übersicht über das ASF-Format](overview-of-the-asf-format.md).

Dieses SDK bietet auch Unterstützung für das Codieren und Decodieren von Audioinhalten und Videoinhalten mithilfe von Windows Media Codecs. Jeder Codec (Kompressor/Dekompressor) ist optimiert, um die beste Qualität für einen bestimmten Inhaltstyp zu erhalten. Weitere Informationen zu den unterstützten Codecs finden Sie unter [Codec-Features](codec-features.md).

Zusätzlich zur allgemeinen Unterstützung von Dateiformaten und Codec-Integration stellt das Windows Media-Format-SDK Objekte zum Bearbeiten von beschreibenden Informationen zu digitalen Medien bereit. Datei Lese-und Metadatenunterstützung ist auch für MP3-Dateien enthalten, sodass Sie flexiblere Anwendungen erstellen können.
