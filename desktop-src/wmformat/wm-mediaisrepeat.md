# <a name="wmmediaisrepeat"></a>WM/mediaisrepeat

Das **WM/mediaisrepeat-** Attribut gibt an, ob der Inhalt eine wiederholte Folge einer Reihe ist.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmmediaisrepeat

## <a name="data-type"></a>Datentyp

**WMT- \_ Typ \_ bool**

## <a name="remarks"></a>Bemerkungen

Dieses Attribut wird für Broadcast-TV verwendet.
