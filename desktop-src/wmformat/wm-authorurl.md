# <a name="wmauthorurl"></a>WM/authorurl

Das Attribut **WM/authorurl** enthält die Adresse der Website des Autors.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmauthorurl

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**

## <a name="remarks"></a>Bemerkungen

Die von diesem Attribut angegebene Adresse sollte dem Entwickler oder dem Autor in einem [**Author**](author.md) -Attribut in der Datei entsprechen.
