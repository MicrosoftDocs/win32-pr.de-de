# <a name="implementing-network-functionality"></a>Implementieren von Netzwerkfunktionen

In diesem Abschnitt wird beschrieben, wie die Netzwerk Features verwendet werden, die über das Windows Media Format SDK verfügbar sind. Eine Anwendung kann diese Funktionen zum Lesen von ASF-Streams aus einem Netzwerk, zum Übertragen von ASF-Datenströmen an ein Netzwerk oder zum Übertragen von ASF-Streams an einen Publishing Point auf einem Server mit Windows Media Services verwenden.

In den folgenden Abschnitten werden die Netzwerk Features dieses SDK beschrieben.



| `Section`                                                                    | BESCHREIBUNG                                                                                                          |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [Übersicht über Netzwerkschnittstellen](overview-of-networking-interfaces.md) | Benennt und beschreibt jede Schnittstelle, die Netzwerk Methoden unterstützt.                                                 |
| [Lesen von ASF-Daten über ein Netzwerk](reading-asf-data-over-a-network.md)     | Beschreibt das Abspielen von Dateien aus einer Netzwerkquelle und das Konfigurieren der Netzwerkeinstellungen für das Reader-Objekt.    |
| [Senden von ASF-Daten über ein Netzwerk](sending-asf-data-over-a-network.md)     | Beschreibt das Übertragen von ASF-Daten über ein Netzwerk oder das Senden von ASF-Daten an einen Publishing Point auf einem Windows Media-Server. |
| [Standard Netzwerkeinstellungen](default-networking-settings.md)             | Bietet eine Kurzübersicht über die Standardeinstellungen, die vom SDK verwendet werden.                                                 |
