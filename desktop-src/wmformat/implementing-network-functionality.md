# <a name="implementing-network-functionality"></a>Implementieren von Netzwerkfunktionen

In diesem Abschnitt wird beschrieben, wie Sie die netzwerkfeatures verwenden, die über das Windows Media Format SDK verfügbar sind. Eine Anwendung kann diese Features verwenden, um ASF-Streams aus einem Netzwerk zu lesen, ASF-Datenströme an ein Netzwerk zu übertragen oder ASF-Streams an einen Veröffentlichungspunkt auf einem Server zu übertragen, auf dem Windows Media-Dienste ausgeführt wird.

In den folgenden Abschnitten werden die Netzwerkfeatures dieses SDK beschrieben.



| `Section`                                                                    | BESCHREIBUNG                                                                                                          |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [Übersicht über Netzwerkschnittstellen](overview-of-networking-interfaces.md) | Benennt und beschreibt jede Schnittstelle, die Netzwerkmethoden unterstützt.                                                 |
| [Lesen von ASF-Daten über ein Netzwerk](reading-asf-data-over-a-network.md)     | Beschreibt das Wiedergeben von Dateien aus einer Netzwerkquelle und das Konfigurieren der Netzwerkeinstellungen für das Readerobjekt.    |
| [Senden von ASF-Daten über ein Netzwerk](sending-asf-data-over-a-network.md)     | Beschreibt, wie ASF-Daten über ein Netzwerk übertragen oder ASF-Daten an einen Veröffentlichungspunkt auf einem Windows Medienserver gesendet werden. |
| [Standardnetzwerk-Einstellungen](default-networking-settings.md)             | Enthält eine Kurzübersicht über die standardeinstellungen, die vom SDK verwendet werden.                                                 |
