# <a name="wmuniquefileidentifier"></a>WM/UniqueFileIdentifier

Das **WM/UniqueFileIdentifier-** Attribut enthält einen eindeutigen Datei Bezeichner für den Inhalt.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmuniquefileidentifier

## <a name="data-type"></a>Datentyp

**WM- \_ Typ \_ Zeichenfolge**

## <a name="remarks"></a>Bemerkungen

Der eindeutige Datei Bezeichner ist eine generische Zeichenfolge, die von Anwendungen und Diensten verwendet werden kann, um die Datei eindeutig zu identifizieren. Die Zeichenfolge enthält durch Semikolons getrennte beliebige Werte. Sie sollten dieses Attribut niemals löschen. Sie können Werte anfügen und ihre eigenen Werte entfernen, aber alle anderen sollten unverändert bleiben.
