# <a name="wmuniquefileidentifier"></a>WM/UniqueFileIdentifier

Das **WM/UniqueFileIdentifier-Attribut** enthält einen eindeutigen Dateibezeichner für den Inhalt.

## <a name="global-constant"></a>Globale Konstante

g \_ wszWMUniqueFileIdentifier

## <a name="data-type"></a>Datentyp

**\_ \_ WM-TYPZEICHENFOLGE**

## <a name="remarks"></a>Hinweise

Der eindeutige Dateibezeichner ist eine generische Zeichenfolge, die von Anwendungen und Diensten verwendet werden kann, um die Datei eindeutig zu identifizieren. Die Zeichenfolge enthält durch Semikolons getrennte beliebige Werte. Sie sollten dieses Attribut nie löschen. Sie können Werte anfügen und Ihre eigenen Werte entfernen, aber alle anderen sollten unverändert bleiben.
