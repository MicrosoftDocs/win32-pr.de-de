# <a name="text-streams"></a>Textstreams

Textstreams enthalten nur Text, der in Beispielen gespeichert ist. Die Anwendung kann Text aus einem Textstream beliebig verwenden. Beispielsweise möchten Sie möglicherweise eine sprach Dozenten Anwendung erstellen, die Wörter in einer Fremdsprache einführt. Sie können jedes Wort als Text Strom Beispiel zu einer Präsentationszeit einschließen, die dem gesprochenen Wort in einem Audiostream entspricht. Anschließend könnte Ihre Anwendung den Text beim übermitteln der Stichprobe Rendering.
