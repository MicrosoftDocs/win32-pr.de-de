# <a name="drm_sourceid"></a>DRM \_ -SourceID

Das **DRM \_ -SourceID-** Attribut enthält einen Bezeichner, der geschützte Inhalte einer Quelle zuordnet.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmdrm \_ SourceID

## <a name="data-type"></a>Datentyp

**WMT- \_ Typ \_ DWORD**

## <a name="remarks"></a>Bemerkungen

Quell Bezeichner werden verwendet, um Inhalte zu unterscheiden, die nicht standardmäßig von Inhalten geschützt werden, die mit standardmäßigen Windows Media DRM geschützt sind.
