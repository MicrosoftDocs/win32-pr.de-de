# <a name="drm_sourceid"></a>\_DRM-Quell-ID

Das **DRM \_ SourceID-Attribut** enthält einen Bezeichner, der geschützten Inhalt einer Quelle zu ordnet.

## <a name="global-constant"></a>Globale Konstante

g \_ wszWMDRM \_ SourceID

## <a name="data-type"></a>Datentyp

**WMT \_ TYPE \_ DWORD**

## <a name="remarks"></a>Hinweise

Quellbezeichner werden verwendet, um Inhalte, die nicht standardmäßig geschützt sind, von Inhalten zu unterscheiden, die mithilfe von Media DRM Windows standard geschützt werden.
