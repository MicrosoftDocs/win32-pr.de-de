# <a name="microsoft-windows-media-drm-client-enumeration-types"></a>Enumerationstypen für Microsoft Windows Media DRM-Clients

In der folgenden Tabelle werden die Enumerationen beschrieben, die von den erweiterten APIs des Microsoft Windows Media DRM-Clients unterstützt werden.



| Enumeration                                                                      | Beschreibung                                                                                                                                                   |
|----------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ \_ zulässige \_ \_ Ergebnisse der DRM-Aktion**](drm-action-allowed-query-results.md) | Wird von der [**iwmdrmlicensequery:: queryAction Allowed**](iwmdrmlicensequery-queryactionallowed.md) -Schnittstelle verwendet, um den Grund anzugeben, warum eine Aktion nicht zulässig ist. |
| [**DRM- \_ attr- \_ Datentyp**](drm-attr-datatype.md)                                 | Definiert die Datentypen, die für DRM-Attribute und-Eigenschaften verwendet werden.                                                                                                |
| [**DRM- \_ Crypto- \_ Typ**](drm-crypto-type.md)                                     | Definiert die kryptografiealgorithmustypen, die zum Verschlüsseln von Inhalten verwendet werden können.                                                                                |
| [**DRM- \_ http- \_ Status**](drmdrm-http-status.md)                                  | Definiert einen Bereich von http-Zuständen für eine DRM-Anforderung.                                                                                                             |
| [**DRM- \_ Individualisierungs \_ Status**](drmdrm-individualization-status.md)        | Definiert die gültigen Zustände für die DRM-Individualisierung.                                                                                                           |
| [**DRM- \_ Lizenz \_ Zustands \_ Kategorie**](drmdrm-license-state-category.md)           | Gibt den Typ der Lizenz Beschränkung an, der durch eine [**DRM- \_ Lizenz \_ Zustands \_ Daten**](drmdrm-license-state-data.md) Struktur beschrieben wird.                    |
| [**msdrm- \_ Status**](msdrm-status.md)                                            | Definiert Status Bedingungen für das DRM-Subsystem.                                                                                                              |
| [**wmdrmnet \_ - \_ Richtlinientyp**](wmdrmnet-policy-type.md)                           | Listet die Typen von Richtlinien auf, die für Windows Media DRM für Netzwerkgeräte Vorgänge verfügbar sind.                                                          |
| [**WMT- \_ Rechte**](drm-wmt-rights.md)                                            | Definiert die Rechte, die in einer DRM-Lizenz angegeben werden können.                                                                                                    |
