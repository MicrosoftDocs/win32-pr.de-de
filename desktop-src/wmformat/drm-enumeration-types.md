# <a name="microsoft-windows-media-drm-client-enumeration-types"></a>Microsoft Windows Media DRM-Clientenumerationstypen

In der folgenden Tabelle werden die Enumerationen beschrieben, die von den erweiterten APIs des Microsoft Windows Media DRM-Clients unterstützt werden.



| Enumeration                                                                      | Beschreibung                                                                                                                                                   |
|----------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_DRM ACTION \_ ALLOWED QUERY RESULTS (DRM-AKTION: ZULÄSSIGE \_ \_ ABFRAGEERGEBNISSE)**](drm-action-allowed-query-results.md) | Wird von der [**IWMDRMLicenseQuery::QueryActionAllowed-Schnittstelle**](iwmdrmlicensequery-queryactionallowed.md) verwendet, um den Grund anzugeben, warum eine Aktion nicht zulässig ist. |
| [**DRM \_ \_ ATTR-DATENTYP**](drm-attr-datatype.md)                                 | Definiert die Datentypen, die für DRM-Attribute und -Eigenschaften verwendet werden.                                                                                                |
| [**DRM \_ CRYPTO \_ TYPE**](drm-crypto-type.md)                                     | Definiert die kryptografischen Algorithmustypen, die zum Verschlüsseln von Inhalten verwendet werden können.                                                                                |
| [**\_DRM-HTTP-STATUS \_**](drmdrm-http-status.md)                                  | Definiert einen Bereich von HTTP-Zuständen für eine DRM-Anforderung.                                                                                                             |
| [**\_DRM-INDIVIDUALISIERUNGSSTATUS \_**](drmdrm-individualization-status.md)        | Definiert die gültigen Zustände für die DRM-Individualisierung.                                                                                                           |
| [**\_ \_ DRM-LIZENZSTATUSKATEGORIE \_**](drmdrm-license-state-category.md)           | Gibt den Typ der Lizenzeinschränkung an, der von einer [**DRM \_ LICENSE STATE \_ \_ DATA-Struktur**](drmdrm-license-state-data.md) beschrieben wird.                    |
| [**\_MSDRM-STATUS**](msdrm-status.md)                                            | Definiert Statusbedingungen für das DRM-Subsystem.                                                                                                              |
| [**\_WMDRMNET-RICHTLINIENTYP \_**](wmdrmnet-policy-type.md)                           | Listet die Typen von Richtlinien auf, die für Windows Medien-DRM für Netzwerkgerätevorgänge verfügbar sind.                                                          |
| [**\_WMT-RECHTE**](drm-wmt-rights.md)                                            | Definiert die Rechte, die in einer DRM-Lizenz angegeben werden können.                                                                                                    |
