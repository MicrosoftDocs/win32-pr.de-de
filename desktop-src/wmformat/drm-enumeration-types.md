# <a name="microsoft-windows-media-drm-client-enumeration-types"></a>Microsoft Windows Media DRM-Clientenumerationstypen

In der folgenden Tabelle werden die Enumerationen beschrieben, die von den erweiterten APIs des Microsoft Windows Media DRM-Clients unterstützt werden.



| Enumeration                                                                      | Beschreibung                                                                                                                                                   |
|----------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ZULÄSSIGE \_ ABFRAGEERGEBNISSE \_ DER \_ DRM-AKTION \_**](drm-action-allowed-query-results.md) | Wird von [**der IWMDRMLicenseQuery::QueryActionAllowed-Schnittstelle**](iwmdrmlicensequery-queryactionallowed.md) verwendet, um den Grund anzugeben, aus dem eine Aktion nicht zulässig ist. |
| [**DRM \_ \_ ATTR-DATENTYP**](drm-attr-datatype.md)                                 | Definiert die Datentypen, die für DRM-Attribute und -Eigenschaften verwendet werden.                                                                                                |
| [**DRM \_ CRYPTO \_ TYPE**](drm-crypto-type.md)                                     | Definiert die Kryptografiealgorithmustypen, die zum Verschlüsseln von Inhalten verwendet werden können.                                                                                |
| [**\_DRM-HTTP-STATUS \_**](drmdrm-http-status.md)                                  | Definiert einen Bereich von HTTP-Zuzuständen für eine DRM-Anforderung.                                                                                                             |
| [**\_DRM-INDIVIDUALISIERUNGSSTATUS \_**](drmdrm-individualization-status.md)        | Definiert die gültigen Zustände für die DRM-Individualisierung.                                                                                                           |
| [**\_ \_ DRM-LIZENZSTATUSKATEGORIE \_**](drmdrm-license-state-category.md)           | Gibt den Typ der Lizenzeinschränkung an, der von einer [**DRM \_ LICENSE STATE \_ \_ DATA-Struktur beschrieben**](drmdrm-license-state-data.md) wird.                    |
| [**MSDRM-STATUS \_**](msdrm-status.md)                                            | Definiert Statusbedingungen für das DRM-Subsystem.                                                                                                              |
| [**WMDRMNET-RICHTLINIENTYP \_ \_**](wmdrmnet-policy-type.md)                           | Listet die Arten von Richtlinien auf, die für Windows Media DRM für Netzwerkgerätevorgänge verfügbar sind.                                                          |
| [**\_WMT-RECHTE**](drm-wmt-rights.md)                                            | Definiert die Rechte, die in einer DRM-Lizenz angegeben werden können.                                                                                                    |
