# <a name="drm-export"></a>DRM-Export

Die Exportfunktion von Windows Media DRM ermöglicht es Ihnen, lizenzierte Anwendungen zu erstellen, die von Windows Media DRM geschützte Inhalte entschlüsseln und in einem von dem zugeordneten Lizenz Aussteller explizit festgelegten Inhalts Schutzschema (CPS) erneut verschlüsseln. Der Lizenz Aussteller autorisiert den Export von Inhalten explizit zu einem bestimmten CPS über eine Inklusions Liste.

> [!Note]  
> Für den Zugriff auf die Exportfunktion müssen Sie ein [Export Anwendungs Zertifikat](export-application-certificate.md) von Microsoft beantragen.

 

Die Exportfunktion von Windows Media DRM wird in den folgenden Abschnitten erläutert.



| `Section`                                                                                      | BESCHREIBUNG                                                                                         |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [Anwendungs Zertifikat exportieren](export-application-certificate.md)                         | Beschreibt ein Anwendungs Zertifikat exportieren.                                                        |
| [Explizite Autorisierungs-und Inklusions Listen](explicit-authorization-and-inclusion-lists.md) | Erläutert den Prozess der expliziten Autorisierung und die Art und Weise, wie Einschlüsse Aufgaben auflisten.                      |
| [Exportieren von komprimiertem Inhalt](exporting-compressed-content.md)                             | Hier wird beschrieben, wie Sie komprimierten Inhalt aus geschützten Audiodateien oder Videoinhalten von Windows Media DRM exportieren. |
