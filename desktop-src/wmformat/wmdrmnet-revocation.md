# <a name="wmdrmnet_revocation"></a>Wmdrmnet \_ -Sperrung

Die **wmdrmnet \_** -Sperr Eigenschaft ruft die Sperr Liste für Windows Media DRM 10 für Netzwerkgeräte ab.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmdrmnet-Sperrung \_

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**

## <a name="remarks"></a>Bemerkungen

Die Sperr Liste enthält die Zertifikate von Geräten, die gesperrt wurden.
