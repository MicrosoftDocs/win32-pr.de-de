# <a name="wmdrmnet_revocation"></a>\_WMDRMNET-Sperrung

Die **\_ WMDRMNET-Sperreigenschaft** ruft die Sperrliste für Windows Medien-DRM 10 für Netzwerkgeräte ab.

## <a name="global-constant"></a>Globale Konstante

g \_ wszWMDRMNET \_ Revocation

## <a name="data-type"></a>Datentyp

**\_ \_ WMT-TYPZEICHENFOLGE**

## <a name="remarks"></a>Hinweise

Die Sperrliste enthält die Zertifikate von Geräten, die widerrufen wurden.
