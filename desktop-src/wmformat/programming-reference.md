# <a name="windows-media-format-sdk-programming-reference"></a>Programmier Referenz zum Windows Media-Format-SDK

Das Microsoft® Windows Media® Format Software Development Kit (SDK) unterstützt eine Reihe von Objekten, Schnittstellen, unabhängigen Funktionen, Strukturen, Enumerationstypen, Attributen usw. In den folgenden Abschnitten werden diese ausführlich beschrieben.



| `Section`                                                                              | BESCHREIBUNG                                                                                                  |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [Objekte](objects.md)                                                               | Beschreibt die Objekte und Schnittstellen, die von jedem Objekt im Windows Media-Format-SDK unterstützt werden.           |
| [Funktionen](functions.md)                                                           | Beschreibt alle unabhängigen Funktionen, in der Regel diejenigen, die zum Erstellen und initialisieren verschiedener-Objekte verwendet werden.      |
| [Schnittstellen](interfaces.md)                                                         | Beschreibt alle Schnittstellen und Methoden in diesem SDK.                                                        |
| [Strukturen](structures.md)                                                         | Beschreibt die Strukturen, die von diesem SDK unterstützt werden.                                                              |
| [Enumerationstypen](enumeration-types.md)                                           | Beschreibt die von diesem SDK unterstützten Enumerationstypen.                                                       |
| [Attribute](attributes.md)                                                         | Beschreibt die Attribute, die in den Headern von ASF-Dateien angegeben werden können.                                  |
| [Medientypen](media-types.md)                                                       | Beschreibt die Medientyp-IDs, die von diesem SDK verwendet werden.                                                       |
| [Ausgabeeinstellungen](output-settings.md)                                               | Beschreibt die globalen Konstanten, die zum Identifizieren der Ausgabeeinstellungen von Lesern verwendet werden.                                      |

| [System profile](system-profiles.md)                                               | Listet alle unterstützten Systemprofile auf.                                                                         |
| [Lokalisierte System profile](localized-system-profiles.md)                           | Listet die lokalisierten Systemprofil Dateien, die im SDK enthalten sind, und die jeweils zugeordneten Sprachen auf.       |
| [GUID-Werte](guid-values.md)                                                       | Beschreibt die GUID-Werte, die von diesem SDK verwendet werden.                                                                  |
| [Fehlercodes](error-codes.md)                                                       | Beschreibt allgemeine Fehlercodes, die von Methoden und Funktionen im SDK zurückgegeben werden.                                   |
| [Quell-Plug-ins](source-plug-ins.md)                                               | Beschreibt die Anforderungen für Benutzer implementierte Quellen.                                                     |
