# <a name="windows-media-format-sdk-programming-reference"></a>Windows Programmierreferenz zum Media Format SDK

Das Microsoft® Windows Media® Format Software Development Kit (SDK) unterstützt eine Reihe von Objekten, Schnittstellen, unabhängigen Funktionen, Strukturen, Enumerationstypen, Attributen und so weiter. In den folgenden Abschnitten werden diese im Detail dokumentiert.



| `Section`                                                                              | BESCHREIBUNG                                                                                                  |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [Objekte](objects.md)                                                               | Beschreibt die Objekte und Schnittstellen, die von jedem Objekt im Windows Media Format SDK unterstützt werden.           |
| [Funktionen](functions.md)                                                           | Beschreibt alle unabhängigen Funktionen, in der Regel die, die zum Erstellen und Initialisieren verschiedener Objekte verwendet werden.      |
| [Schnittstellen](interfaces.md)                                                         | Beschreibt alle Schnittstellen und Methoden in diesem SDK.                                                        |
| [Strukturen](structures.md)                                                         | Beschreibt die von diesem SDK unterstützten Strukturen.                                                              |
| [Enumerationstypen](enumeration-types.md)                                           | Beschreibt die von diesem SDK unterstützten Enumerationstypen.                                                       |
| [Attribute](attributes.md)                                                         | Beschreibt die Attribute, die in den Headern von ASF-Dateien angegeben werden können.                                  |
| [Medientypen](media-types.md)                                                       | Beschreibt die von diesem SDK verwendeten Medientypbezeichner.                                                       |
| [Ausgabe Einstellungen](output-settings.md)                                               | Beschreibt die globalen Konstanten, die zum Identifizieren von Readerausgabeeinstellungen verwendet werden.                                      |
