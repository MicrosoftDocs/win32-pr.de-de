# <a name="description-windows-media-format-11-sdk"></a>Beschreibung (Windows Media-Format 11 SDK)

Das **Description** -Attribut enthält eine Beschreibung des Inhalts der Datei.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmdescription

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**

## <a name="remarks"></a>Bemerkungen

Dabei handelt es sich um ein Attribut auf Dateiebene.

Bei ID3v1-MP3-Dateien ist dieses Attribut auf 30 Zeichen beschränkt.
