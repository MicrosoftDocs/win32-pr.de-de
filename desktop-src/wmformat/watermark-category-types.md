# <a name="watermark-category-types"></a>Wasserzeichenkategorietypen

Sie können die DMO Enumerationsfunktion **DMOEnum** verwenden, um Informationen zu den wassermarkierenden DMOs abzurufen, die auf einem Computer installiert sind. Das Windows Media Format SDK definiert GUIDs, die die Wasserzeichen DMO Kategorien identifizieren. Weitere Informationen zu **DMOEnum** finden Sie in der DirectShow-Dokumentation auf MSDN.

In der folgenden Tabelle sind die Bezeichner für Wasserzeichenkategorien aufgeführt.



| Wasserzeichen DMO Kategoriekonstante | GUID                                 |
|------------------------------------|--------------------------------------|
| WMT \_ DMOCATEGORY \_ AUDIO \_ WATERMARK | 65221c5a-fa75-4b39-b50c-06c336b6a3ef |
| WMT \_ DMOCATEGORY \_ VIDEO \_ WATERMARK | 187cc922-8efc-4404-9daf-63f4830df1bc |
