# <a name="watermark-category-types"></a>Typen von Wasserzeichen Kategorien

Sie können die DMO-Enumerationsfunktion **dmoenum** verwenden, um Informationen zu den auf einem Computer installierten Wasserzeichen-DMOS zu erhalten. Das Windows Media Format SDK definiert GUIDs, die die Wasserzeichen-DMO-Kategorien identifizieren. Weitere Informationen zu **dmuenum** finden Sie in der DirectShow-Dokumentation auf MSDN.

In der folgenden Tabelle werden die Bezeichner für Wasserzeichen Kategorien aufgelistet.



| Wasserzeichen DMO-kategoriekonstante | GUID                                 |
|------------------------------------|--------------------------------------|
| WMT \_ dmucategory \_ - \_ audiowasser Zeichen | 65221c5a-fa75-4b39-b50c-06c336b6a3ef |
| WMT \_ dmucategory- \_ Video \_ Wasserzeichen | 187cc922-8efc-4404-9daf -63F 4830df1bc |
