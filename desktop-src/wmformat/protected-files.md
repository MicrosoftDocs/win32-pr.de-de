# <a name="protected-files"></a>Geschützte Dateien

Wenn eine Datei mithilfe von Windows Media DRM geschützt ist, wird der Daten Abschnitt der Datei verschlüsselt. Der Prozess des Schützens einer Datei wird auch als Verpacken der Datei bezeichnet. Mit den erweiterten APIs für den Windows Media DRM-Client können Sie keine Dateien verpacken. Sie können Dateien mithilfe der Objekte des Windows Media Rights Manager SDK oder des Windows Media SDK-SDKs verpacken.

Ein Benutzer kann nicht auf die Daten in einer geschützten Datei zugreifen, ohne zuvor eine Lizenz für diese Datei zu erhalten.
