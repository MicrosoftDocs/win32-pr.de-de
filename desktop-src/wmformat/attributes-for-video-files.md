# <a name="attributes-for-video-files"></a>Attribute für Videodateien

In diesem Abschnitt werden die Attribute aufgeführt, die häufig für Videodateien verwendet werden. Es wird empfohlen, Attribute für Dateien gemäß diesen Listen zu setzen, um sicherzustellen, dass Ihre Dateien vollständig mit einer Vielzahl von Wiedergabeanwendungen kompatibel sind. Die Attribute in diesem Abschnitt sind in drei Kategorien aufgeführt: primär, sekundär und tertiär.

Primäre Attribute vermitteln die grundlegendsten Informationen zu einer Datei. Wenn Sie Videodateien für die Verteilung erstellen, ist dies der Mindestsatz von Attributen, die Sie verwenden sollten.

Sekundäre Attribute enthalten allgemeine Metadaten, die wichtig, aber nicht für alle Videodateien universell sind.

Tertiäre Attribute sollten nach Bedarf eingeschlossen werden, sind aber nicht wichtig, um die Datei zu beschreiben.
