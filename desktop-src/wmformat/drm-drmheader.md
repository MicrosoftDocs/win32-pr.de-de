# <a name="drm_drmheader"></a>DRM \_ DRMHeader

**DRM \_ DRMHeader** ist die Basiszeichenfolge für die DRMHeader-Attribute wie DRM \_ HeaderKeyID.

## <a name="global-constant"></a>Globale Konstante

g \_ wszWMDRM \_ DRM \_ DRMHeader

## <a name="data-type"></a>Datentyp

**\_ \_ WMT-TYPZEICHENFOLGE**

## <a name="remarks"></a>Hinweise

Diese Zeichenfolge kann verwendet werden, um benutzerdefinierte DRMHeader-Eigenschaften zu erstellen. Es ist kein Attribut an sich.
