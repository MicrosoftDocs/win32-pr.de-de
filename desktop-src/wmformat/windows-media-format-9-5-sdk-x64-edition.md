# <a name="windows-media-format-95-sdk-x64-edition"></a>Windows Media Format 9.5 SDK x64 Edition

Diese Dokumentation gilt für das neue Windows Media Format 9.5 SDK x64 Edition sowie für die vorherigen 32-Bit-Versionen des Windows Media Format SDK.

In der folgenden Liste sind die Änderungen am SDK für die x64-basierte Version zusammengefasst:

-   DRM wird nicht unterstützt. DRM-bezogene Dateien (einschließlich Beispielen) wurden aus der SDK-Installation entfernt.
-   Windows Die Geräte-Manager ist nicht Teil der x64-basierten SDK-Installation.
-   Die Beispiele und Header, die die Windows Medienaudio- und Videocodecschnittstellen unterstützen, sind nicht Teil der x64-basierten SDK-Installation.
-   Die lokalisierten Systemprofile sind nicht Teil der x64-basierten SDK-Installation.

In den Hinweisen im Abschnitt "Programmierreferenz" dieser Dokumentation werden Features beschrieben, die in der Windows Media Format 9.5 SDK x64 Edition nicht unterstützt werden.
