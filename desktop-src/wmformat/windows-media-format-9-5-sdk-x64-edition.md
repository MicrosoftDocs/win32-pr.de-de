# <a name="windows-media-format-95-sdk-x64-edition"></a>Windows Media-Format 9,5 SDK x64 Edition

Diese Dokumentation bezieht sich auf das neue Windows Media-Format 9,5 SDK x64 Edition sowie auf die vorherigen 32-Bit-Versionen des SDK für den Windows Media-Format.

In der folgenden Liste werden die Änderungen am SDK für die x64-basierte Version zusammengefasst:

-   DRM wird nicht unterstützt. DRM-bezogene Dateien (einschließlich Beispielen) wurden aus der SDK-Installation entfernt.
-   Windows Media Device Manager ist nicht Teil der x64-basierten SDK-Installation.
-   Die Beispiele und Header, die die Windows Media Audio-und Video-Codec-Schnittstellen unterstützen, sind nicht Teil der x64-basierten SDK-Installation.
-   Die lokalisierten Systemprofile sind nicht Teil der x64-basierten SDK-Installation.

Hinweise im Abschnitt "Programmier Referenz" in dieser Dokumentation sind Features, die im Windows Media-Format 9,5 SDK x64 Edition nicht unterstützt werden.
