# <a name="rating"></a>Rating

Das Bewertungs Attribut enthält die Bewertung des Inhalts.

## <a name="global-constant"></a>Globale Konstante

g \_ wszwmrating

## <a name="data-type"></a>Datentyp

**WMT \_ - \_ Typzeichenfolge**

## <a name="remarks"></a>Bemerkungen

Dabei handelt es sich um ein Attribut auf Dateiebene.

Einzelne Implementierungen in vorhandenen Dateien verwenden unterschiedliche Skalen und Notationen. Aus diesem Grund sollten Sie die Verwendung dieses Attributs vermeiden. Es gibt keine Standardisierung, von der konsistente Benutzeroberflächen entwickelt werden.
