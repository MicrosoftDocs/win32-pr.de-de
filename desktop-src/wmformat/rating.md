# <a name="rating"></a>Rating

Das Rating-Attribut enthält die Bewertung des Inhalts.

## <a name="global-constant"></a>Globale Konstante

g \_ wszWMRating

## <a name="data-type"></a>Datentyp

**\_ \_ WMT-TYPZEICHENFOLGE**

## <a name="remarks"></a>Hinweise

Dies ist ein Attribut auf Dateiebene.

Einzelne Implementierungen in vorhandenen Dateien verwenden unterschiedliche Skalierungen und Notationen. Aus diesem Grund sollten Sie die Verwendung dieses Attributs vermeiden. Es gibt keine Standardisierung, aus der konsistente Benutzeroberflächen entwickelt werden können.
