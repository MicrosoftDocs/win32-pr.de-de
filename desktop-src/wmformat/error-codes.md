# <a name="error-codes-windows-media-format-11-sdk"></a>Fehlercodes (Windows Media Format 11 SDK)

Microsoft hat die folgenden Fehler- und Erfolgscodes für die Verwendung im Windows Media Format SDK definiert. Diese Codes sind eine Teilmenge einer größeren Sammlung, die in den Headerdateien asferr.h (ASF-Codes) und nserror.h (andere Codes) angezeigt werden können, die im Windows Media Format SDK enthalten sind. Zusätzliche allgemeine Fehlercodes können in Winerror.h angezeigt werden.
