# <a name="error-codes-windows-media-format-11-sdk"></a>Fehler Codes (Windows Media-Format 11 SDK)

Microsoft hat die folgenden Fehler-und Erfolgs Codes für die Verwendung im Windows Media-Format-SDK definiert. Diese Codes stellen eine Teilmenge einer größeren Auflistung dar, die in den Header Dateien asferr. h (ASF Codes) und nserror. h (sonstige Codes) angezeigt werden kann, die im SDK für den Windows Media-Format enthalten sind. Weitere häufige Fehlercodes können in WinError. h angezeigt werden.
