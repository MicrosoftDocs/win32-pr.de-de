# <a name="input-source-identification"></a>Identifizierung der Eingabe Quelle

## <a name="purpose"></a>Zweck

Die Themen in diesem Abschnitt bieten eine Übersicht über die Unterstützung der Eingabe Quellen Identifizierung in Windows 8. Die Identifizierung der Eingabe Quelle ermöglicht es Anwendungen, eine Eingabe Quelle genau und zuverlässig zu erkennen, den Eingabetyp zu identifizieren und eine Benutzer Darstellung bereitzustellen, die am besten für den Eingabetyp geeignet ist.

In der hier bereitgestellten Auflistung von Themen wird beschrieben, wie Ihre APP diese Informationen erhalten kann, und Sie können Sie optimal nutzen, um die Benutzerfreundlichkeit für Ihren Kunden zu optimieren.

Die APIs für die Eingabe Quellen Identifizierung sind für Entwickler konzipiert, die Benutzeroberflächen-Frameworks erstellen, die eine konsistente, Berührungs optimierte Benutzeroberfläche für alle Desktop Anwendungen bereitstellen.
