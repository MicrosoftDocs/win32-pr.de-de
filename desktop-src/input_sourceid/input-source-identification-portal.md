# <a name="input-source-identification"></a>Eingabequellenidentifikation

## <a name="purpose"></a>Zweck

Die Themen in diesem Abschnitt bieten eine Übersicht über die Unterstützung der Eingabequellenidentifikation in Windows 8. Mit der Eingabequellenidentifikation können Anwendungen eine Eingabequelle genau und zuverlässig erkennen, den Eingabetyp identifizieren und eine Benutzeroberfläche bereitstellen, die für den Eingabetyp am besten geeignet ist.

Die hier bereitgestellte Sammlung von Themen beschreibt, wie Ihre App diese Informationen abrufen und am besten verwenden kann, um die Benutzerfreundlichkeit für Ihren Kunden zu optimieren.

## <a name="developer-audience"></a>Entwicklergruppe

Die APIs zur Eingabequellenidentifikation sind für Entwickler konzipiert, die Benutzeroberflächenframeworks erstellen, die eine konsistente touchoptimierte Benutzeroberfläche für Desktopanwendungen bieten.
