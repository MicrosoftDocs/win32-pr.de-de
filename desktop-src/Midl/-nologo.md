# <a name="nologo-switch"></a>/nologo-Schalter

Der Schalter **/nologo** deaktiviert die Anzeige des Copyright Banners.

``` syntax
midl /nologo
```

## <a name="switch-options"></a>Optionen wechseln

Dieser Switch hat keine Parameter.
