# <a name="nologo-switch"></a>/nologo switch

Der Schalter **/nologo** deaktiviert die Anzeige des Copyrightbanners.

``` syntax
midl /nologo
```

## <a name="switch-options"></a>Optionen wechseln

Dieser Schalter verfügt über keine Parameter.
