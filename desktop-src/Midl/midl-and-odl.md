# <a name="midl-and-odl"></a>Mittel l und ODL

Der Microsoft Interface Definition Language (Mittel l) enthält jetzt die vollständige ODL-Syntax (Object Definition Language). Auf diese Weise können Sie mit dem 32-Bit-Mittelwert Compiler eine Typbibliothek und optionale Header Dateien für eine COM-Anwendung generieren.

> [!Note]  
> Wenn sich die Dokumentation auf eine ODL-Datei bezieht, verweist Sie auf eine Datei, die Mktyplib.exe analysieren kann. Wenn eine IDL-Datei auf eine IDL-Datei verweist, verweist Sie auf eine Datei, die der mittlerer l-Compiler analysieren kann. Dabei handelt es sich um eine Benennungs Konvention. Der mittlerer l-Compiler analysiert eine Eingabedatei unabhängig von ihrer Dateinamenerweiterung.
