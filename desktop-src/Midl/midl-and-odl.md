# <a name="midl-and-odl"></a>MIDL und ODL

Die Microsoft Interface Definition Language (MIDL) enthält jetzt die vollständige ODL-Syntax (Object Definition Language). Dadurch können Sie den 32-Bit-MIDL-Compiler verwenden, um eine Typbibliothek und optionale Headerdateien für eine COM-Anwendung zu generieren.

> [!Note]  
> Wenn sich die Dokumentation auf eine ODL-Datei bezieht, bezieht sie sich auf eine Datei, die Mktyplib.exe kann. Wenn er auf eine IDL-Datei verweist, bezieht er sich auf eine Datei, die der MIDL-Compiler analysieren kann. Dies ist ausschließlich eine Namenskonvention. Der MIDL-Compiler analysiert eine Eingabedatei unabhängig von der Dateierweiterung.
