# <a name="directional-attributes"></a>Direktionale Attribute

Anwenden direktionaler Attribute auf Parameter, um die Richtung anzugeben, in der die Daten übermittelt werden. Beide Attribute können auf einen einzelnen Parameter angewendet werden, dies erhöht jedoch den mit einem-Befehl verbundenen Aufwand.



| Attribut              | Verbrauch                                                                 |
|------------------------|-----------------------------------------------------------------------|
| [**in**](in.md)       | Der-Parameter wird vom Aufrufer an die aufgerufene Funktion übergeben. |
| [**vorgenommen**](out-idl.md) | Der-Parameter wird von der aufgerufenen Funktion an den Aufrufer übergeben. |
