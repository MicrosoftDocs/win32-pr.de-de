# <a name="cc-compiler-considerations"></a>C/C++-compilerüberlegungen

Der mittlerer l-Compiler muss mit dem c-Compiler und dem c-Präprozessor zusammenarbeiten. Die C++-Syntax für die Eingabe wird von der mittleren l nicht akzeptiert. Die generierte h-Datei verfügt sowohl über C-als auch C++-Definitionen für Schnittstellen. In den folgenden Themen werden die Anforderungen für den C-Compiler beschrieben.

-   [C-Compilerprobleme beim Packen](c-compiler-packing-issues.md)
-   [C-compilerdefinitionen für Proxy/Stub](c-compiler-definitions-for-proxy-stubs.md)
