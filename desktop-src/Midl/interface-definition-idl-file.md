# <a name="interface-definition-idl-file"></a>IDL-Datei (Interface Definition)

Standardmäßig wird die Datei, die Schnittstellen- und Typbibliotheksdefinitionen enthält, als IDL-Datei bezeichnet und hat die Dateierweiterung .idl. In Wirklichkeit analysiert der MIDL-Compiler eine Schnittstellendefinitionsdatei unabhängig von der Erweiterung. Eine Schnittstelle wird durch die Schlüsselwortschnittstelle [**identifiziert.**](interface.md)

Jede Schnittstelle besteht aus einem Header und einem Text. Der Schnittstellenheader enthält Attribute, die für die gesamte Schnittstelle gelten. Der Text der Schnittstelle enthält die verbleibenden Schnittstellendefinitionen. Eine Übersicht über Schnittstellen und IDL-Dateien finden Sie unter [Die IDL-Datei (Interface Definition Language).](/windows/desktop/Rpc/the-interface-definition-language-idl-file) Eine Zusammenfassung der Attribute, die in einem Schnittstellenheader verwendet werden können, finden Sie unter [IDL-Attribute.](idl-attributes.md)
