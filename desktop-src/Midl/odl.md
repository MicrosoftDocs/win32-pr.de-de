# <a name="odl-attribute"></a>odl-Attribut

MkTypLib erforderte das **\[ ODL \]** -Attribut für ODL-Schnittstellen. Der mittlerer l-Compiler benötigt das **\[ ODL \]** -Attribut nicht. er wird nur aus Gründen der Kompatibilität mit älteren **ODL** -Dateien erkannt.

> [!Note]  
> Das Mktyplib.exe Tool ist veraltet. Verwenden Sie stattdessen den Mittel l-Compiler.
