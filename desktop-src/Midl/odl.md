# <a name="odl-attribute"></a>odl-Attribut

MKTYPLIB erforderte das **\[ odl-Attribut \]** für ODL-Schnittstellen. Der MIDL-Compiler erfordert das **\[ \] odl-Attribut** nicht. Es wird nur aus Kompatibilitäts- und Kompatibilitätsproblemen mit älteren **ODL-Dateien** erkannt.

> [!Note]  
> Das Mktyplib.exe ist veraltet. Verwenden Sie stattdessen den MIDL-Compiler.
