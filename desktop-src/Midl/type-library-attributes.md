# <a name="type-library-attributes"></a>Typbibliotheksattribute

Die folgenden Schlüsselwörter geben Typbibliotheksinformationen [](library.md) im Kontext einer Bibliotheksaufstellung in einer IDL-Datei an.



| attribute                                    | Verbrauch                                                                                                                                                                                                                                                                                                                                                                                                  |
|----------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**aggregatable**](aggregatable.md)         | Definiert die [**Co-Klasse**](coclass.md) als unterstützende Objekte, die den Schnittstellenzeiger eines anderen Objekts direkt verfügbar machen können.                                                                                                                                                                                                                                                                              |
| [**appobject**](appobject.md)               | Identifiziert die [**Co-Klasse**](coclass.md) als Anwendungsobjekt, das einer vollständigen EXE-Anwendung zugeordnet ist.                                                                                                                                                                                                                                                                                   |
| [**bindable**](bindable.md)                 | Gibt an, dass die Eigenschaft die Datenbindung unterstützt. Dadurch kann der Client benachrichtigt werden, wenn sich der Wert einer Eigenschaft geändert hat.                                                                                                                                                                                                                                                                        |
| [**coclass**](coclass.md)                   | Stellt eine Liste der unterstützten Schnittstellen für ein Komponentenobjekt bereit.                                                                                                                                                                                                                                                                                                                                 |
| [**Steuerung**](control.md)                   | Identifiziert eine [**Co-Klasse**](coclass.md) oder [**Bibliothek**](library.md) als COM-Steuerelement, von dem ein Containerstandort zusätzliche Typbibliotheken oder Komponentenobjektklassen ableitt.                                                                                                                                                                                                                 |
| [**Benutzerdefinierte**](custom.md)                     | Definiert ein benutzerdefiniertes Attribut (eines, das nicht in MIDL definiert ist).                                                                                                                                                                                                                                                                                                                                                  |
| [**Standard**](default.md)                   | Gibt an, dass die schnittstelle oder dispinterface, die in einer [**Co-Klasse**](coclass.md)definiert ist, die Standardschnittstelle darstellt.                                                                                                                                                                                                                                                                          |
| [**defaultbind**](defaultbind.md)           | Gibt die einzelne bindbare Eigenschaft an, die das Objekt am besten darstellt.                                                                                                                                                                                                                                                                                                                               |
| [**defaultcollelem**](defaultcollelem.md)   | Kennzeichnet die Eigenschaft als Accessorfunktion für ein Element der Standardauflistung. Wird für die Codeoptimierung von Microsoft Visual Basic verwendet.                                                                                                                                                                                                                                                                |
| [**Defaultvalue**](defaultvalue.md)         | Ermöglicht die Angabe eines Standardwerts für einen typisierten optionalen Parameter.                                                                                                                                                                                                                                                                                                                                |
| [**defaultvtable**](defaultvtable.md)       | Ermöglicht einem Objekt zwei verschiedene Quellschnittstellen.                                                                                                                                                                                                                                                                                                                                              |
| [**Dispatchschnittstelle**](dispinterface.md)       | Definiert einen Satz von Eigenschaften und Methoden, für die Sie **IDispatch::Invoke** aufrufen können.                                                                                                                                                                                                                                                                                                                   |
| [**displaybind**](displaybind.md)           | Gibt eine Eigenschaft an, die dem Benutzer als bindbar angezeigt werden soll.                                                                                                                                                                                                                                                                                                                                 |
| [**dllname(str)**](dllname-str-.md)         | Definiert den Namen der DLL, die die Einstiegspunkte für ein Modul enthält.                                                                                                                                                                                                                                                                                                                               |
| [**Dual**](dual.md)                         | Identifiziert eine Schnittstelle, die Eigenschaften und Methoden sowohl über **IDispatch** als auch direkt über die VTable verfügbar macht.                                                                                                                                                                                                                                                                                |
| [**Eintrag**](entry.md)                       | Gibt eine exportierte Funktion oder Konstante in einem Modul an, indem der Einstiegspunkt in der DLL identifiziert wird.                                                                                                                                                                                                                                                                                                      |
| [**helpcontext**](helpcontext.md)           | Gibt einen Kontextbezeichner an, mit dem der Benutzer Informationen zu diesem Element in der Hilfedatei anzeigen kann.                                                                                                                                                                                                                                                                                                |
| [**helpfile**](helpfile.md)                 | Legt den Namen der Hilfedatei für eine Typbibliothek fest.                                                                                                                                                                                                                                                                                                                                                     |
| [**helpstring**](helpstring.md)             | Gibt eine Zeichenfolge an, die zum Beschreiben des Elements verwendet wird, auf das sie angewendet wird.                                                                                                                                                                                                                                                                                                                 |
| [**helpstringdll**](helpstringdll.md)       | Legt den Namen der DLL fest, die zum Ausführen der Dokumentzeichenfolgensuche verwendet werden soll.                                                                                                                                                                                                                                                                                                                                 |
| [**Versteckte**](hidden.md)                     | Gibt an, dass das Element vorhanden ist, aber nicht in einem benutzerorientierten Browser angezeigt werden soll.                                                                                                                                                                                                                                                                                                                 |
| [**Id**](id.md)                             | Gibt eine DISPID für eine Memberfunktion an (entweder eine Eigenschaft oder eine Methode in einer Schnittstelle oder Dispinterface).                                                                                                                                                                                                                                                                                            |
| [**immediatebind**](immediatebind.md)       | Gibt an, dass die Datenbank sofort über alle Änderungen an einer Eigenschaft eines datengebundenen Objekts benachrichtigt wird.                                                                                                                                                                                                                                                                                          |
| [**Lcid**](lcid.md)                         | Identifiziert bei [](library.md) Anwendung auf die Bibliotheksanweisung zusammen mit einem localeID-Argument das Gebietsschema für eine Typbibliothek oder für ein Funktionsargument und ermöglicht die Verwendung von internationalen Zeichen innerhalb des Bibliotheksblocks. In  einer Bibliotheksanweisung akzeptiert MIDL ab dem Punkt, an dem das [**lcid-Attribut**](lcid.md) verwendet wird, eingaben, die gemäß dem angegebenen Gebietsschema lokalisiert sind. |
| [**Bibliothek**](library.md)                   | Weist den MIDL-Compiler an, Typbibliotheksinformationen für die Schnittstellen und Klassen zu generieren, auf die in der Anweisung verwiesen wird.                                                                                                                                                                                                                                                                       |
| [**Lizenziert**](licensed.md)                 | Gibt an, dass die [**Co-Klasse,**](coclass.md) für die sie gilt, lizenziert ist und Instanzen mit **IClassFactory2** erstellt werden müssen.                                                                                                                                                                                                                                                                 |
| [**nonbrowsable**](nonbrowsable.md)         | Gibt an, dass die Eigenschaft in einem Objektbrowser (der keine Eigenschaftswerte anzeigt), aber nicht in einem Eigenschaftenbrowser (der Eigenschaftswerte anzeigt) angezeigt wird.                                                                                                                                                                                                                         |
| [**noncreatable**](noncreatable.md)         | Verhindert, dass ein Client die Standardklassenfactory verwendet, um Instanzen einer Objektschnittstelle zu erstellen.                                                                                                                                                                                                                                                                                                     |
| [**nonextensible**](nonextensible.md)       | Gibt an, dass die **IDispatch-Implementierung** nur die In der Schnittstellenbeschreibung aufgeführten Eigenschaften und Methoden enthält und zur Laufzeit nicht mit zusätzlichen Membern erweitert werden kann.                                                                                                                                                                                                                |
| [**oleautomation**](oleautomation.md)       | Gibt an, dass eine Schnittstelle mit der Automatisierung kompatibel ist.                                                                                                                                                                                                                                                                                                                                             |
| [**Optional**](optional.md)                 | Gibt einen optionalen Parameter für eine Memberfunktion an.                                                                                                                                                                                                                                                                                                                                                 |
| [**propget**](propget.md)                   | Gibt eine Eigenschaftenaccessorfunktion an.                                                                                                                                                                                                                                                                                                                                                                |
| [**Propput**](propput.md)                   | Gibt eine Eigenschaftseinstellungsfunktion an.                                                                                                                                                                                                                                                                                                                                                                 |
| [**Propputref**](propputref.md)             | Gibt eine Eigenschaftseinstellungsfunktion an, die einen Verweis anstelle eines Werts verwendet.                                                                                                                                                                                                                                                                                                                        |
| [**Öffentlich**](public.md)                     | Stellt sicher, dass ein mit [**typedef**](typedef.md) deklarierter Alias Teil der Typbibliothek wird.                                                                                                                                                                                                                                                                                                       |
| [**Readonly**](readonly.md)                 | Verhindert das Zuweisen eines neuen Werts zu einer Variablen.                                                                                                                                                                                                                                                                                                                                                         |
| [**requestedit**](requestedit.md)           | Gibt an, dass die Eigenschaft die **OnRequestEdit-Benachrichtigung** unterstützt.                                                                                                                                                                                                                                                                                                                               |
| [**Beschränkt**](restricted.md)             | Gibt an, dass eine Bibliothek oder ein Member eines Moduls, einer Schnittstelle oder einer Disp-Schnittstelle nicht beliebig aufgerufen werden kann.                                                                                                                                                                                                                                                                                             |
| [**Retval**](retval.md)                     | Legt den Parameter fest, der den Rückgabewert des Members empfängt.                                                                                                                                                                                                                                                                                                                                 |
| [**Quelle**](source.md)                     | Gibt an, dass ein Member einer [**Co-Klasse,**](coclass.md)-Eigenschaft oder -Methode eine Quelle von Ereignissen ist.                                                                                                                                                                                                                                                                                                    |
| [**Schnur**](string.md)                     | Gibt an, dass das eindimensionale [**Char-,**](char-idl.md) [**wchar \_ t-,**](wchar-t.md) [**Byte-**](byte.md) (oder äquivalente) Array oder der Zeiger auf ein solches Array als Zeichenfolge behandelt werden muss.                                                                                                                                                                                                      |
| [**uidefault**](uidefault.md)               | Gibt an, dass der Typinformationsmember der Standardmember für die Anzeige auf der Benutzeroberfläche ist.                                                                                                                                                                                                                                                                                                    |
| [**usesgetlasterror**](usesgetlasterror.md) | Gibt an, dass ein Moduleinstiegspunkt [**SetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-setlasterror) verwendet, um Fehlercodes zurückzugeben, und dass ein Aufrufer [**getLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror) aufrufen kann, um den Fehlercode abzurufen, wenn beim Eingeben einer Funktion ein Fehler auftritt.                                                                                                                                                     |
| [**Uuid**](uuid.md)                         | Legt einen eindeutigen Bezeichner für eine Typbibliothek, [**Co-Klasse**](coclass.md)oder Schnittstelle fest.                                                                                                                                                                                                                                                                                                           |
| [**vararg**](vararg.md)                     | Gibt an, dass die Funktion eine variable Argumentenanzahl akzeptiert.                                                                                                                                                                                                                                                                                                                                      |
| [**Version**](version.md)                   | Identifiziert eine bestimmte Version einer Typbibliothek.                                                                                                                                                                                                                                                                                                                                                     |
