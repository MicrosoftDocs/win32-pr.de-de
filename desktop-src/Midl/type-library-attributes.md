# <a name="type-library-attributes"></a>Typbibliotheks Attribute

Die folgenden Schlüsselwörter geben Typbibliotheks Informationen im Kontext einer [**Bibliotheks**](library.md) Anweisung in einer IDL-Datei an.



| Attribut                                    | Verbrauch                                                                                                                                                                                                                                                                                                                                                                                                  |
|----------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**aggregatable**](aggregatable.md)         | Definiert die [**Co-Klasse**](coclass.md) als unterstützende Objekte, die den Schnittstellen Zeiger eines anderen Objekts direkt verfügbar machen können.                                                                                                                                                                                                                                                                              |
| [**appobject**](appobject.md)               | Identifiziert die [**Co-Klasse**](coclass.md) als Anwendungs Objekt, das einer vollständigen exe-Anwendung zugeordnet ist.                                                                                                                                                                                                                                                                                   |
| [**bindable**](bindable.md)                 | Gibt an, dass die Eigenschaft die Datenbindung unterstützt. Dadurch kann der Client benachrichtigt werden, wenn eine Eigenschaft den Wert geändert hat.                                                                                                                                                                                                                                                                        |
| [**coclass**](coclass.md)                   | Stellt eine Liste der unterstützten Schnittstellen für ein Komponenten Objekt bereit.                                                                                                                                                                                                                                                                                                                                 |
| [**Steuerelement**](control.md)                   | Identifiziert eine [**Co-Klasse**](coclass.md) oder eine [**Bibliothek**](library.md) als com-Steuerelement, von dem eine Container Site zusätzliche Typbibliotheken oder Komponenten Objektklassen ableitet.                                                                                                                                                                                                                 |
| [**Zollunion**](custom.md)                     | Definiert ein benutzerdefiniertes Attribut (das nicht in der mittleren l definiert ist).                                                                                                                                                                                                                                                                                                                                                  |
| [**vorgegebene**](default.md)                   | Gibt an, dass die Schnittstelle oder die dispinterface, die innerhalb einer [**Co-Klasse**](coclass.md)definiert ist, die Standardschnittstelle darstellt.                                                                                                                                                                                                                                                                          |
| [**defaultbind**](defaultbind.md)           | Gibt die einzelne, bindbare Eigenschaft an, die das Objekt am besten darstellt.                                                                                                                                                                                                                                                                                                                               |
| [**defaultcollelem**](defaultcollelem.md)   | Markiert die Eigenschaft als eine Accessorfunktion für ein Element der Standard Auflistung. Wird für die Optimierung von Microsoft Visual Basic-Code verwendet.                                                                                                                                                                                                                                                                |
| [**DefaultValue**](defaultvalue.md)         | Ermöglicht die Angabe eines Standardwerts für einen eingegebenen optionalen Parameter.                                                                                                                                                                                                                                                                                                                                |
| [**defaultvtable**](defaultvtable.md)       | Ermöglicht es, dass ein Objekt über zwei unterschiedliche Quell Schnittstellen verfügt.                                                                                                                                                                                                                                                                                                                                              |
| [**dispinterface**](dispinterface.md)       | Definiert einen Satz von Eigenschaften und Methoden, auf die Sie **IDispatch:: Aufrufen** aufrufen können.                                                                                                                                                                                                                                                                                                                   |
| [**displaybind**](displaybind.md)           | Gibt eine Eigenschaft an, die dem Benutzer als Bindable angezeigt werden soll.                                                                                                                                                                                                                                                                                                                                 |
| [**dllname (Str)**](dllname-str-.md)         | Definiert den Namen der dll, die die Einstiegspunkte für ein Modul enthält.                                                                                                                                                                                                                                                                                                                               |
| [**Ales**](dual.md)                         | Identifiziert eine Schnittstelle, die Eigenschaften und Methoden sowohl über **IDispatch** als auch direkt über die vtable verfügbar macht.                                                                                                                                                                                                                                                                                |
| [**ein**](entry.md)                       | Gibt eine exportierte Funktion oder Konstante in einem Modul an, indem der Einstiegspunkt in der DLL identifiziert wird.                                                                                                                                                                                                                                                                                                      |
| [**helpcontext**](helpcontext.md)           | Gibt einen Kontext Bezeichner an, mit dem der Benutzerinformationen zu diesem Element in der Hilfedatei anzeigen kann.                                                                                                                                                                                                                                                                                                |
| [**helpfile**](helpfile.md)                 | Legt den Namen der Hilfedatei für eine Typbibliothek fest.                                                                                                                                                                                                                                                                                                                                                     |
| [**helpstring**](helpstring.md)             | Gibt eine Zeichenfolge an, die zum Beschreiben des Elements verwendet wird, auf das sie angewendet wird.                                                                                                                                                                                                                                                                                                                 |
| [**helpstringdll**](helpstringdll.md)       | Legt den Namen der dll fest, die verwendet wird, um die Suche nach Dokument Zeichenfolgen auszuführen.                                                                                                                                                                                                                                                                                                                                 |
| [**verbirgt**](hidden.md)                     | Gibt an, dass das Element vorhanden ist, aber nicht in einem benutzerorientierten Browser angezeigt werden soll.                                                                                                                                                                                                                                                                                                                 |
| [**Name**](id.md)                             | Gibt eine DISPID für eine Element Funktion an (entweder eine Eigenschaft oder eine Methode in einer Schnittstelle oder einer dispinterface).                                                                                                                                                                                                                                                                                            |
| [**immediatebind**](immediatebind.md)       | Gibt an, dass die Datenbank sofort über alle Änderungen an einer Eigenschaft eines Daten gebundenen Objekts benachrichtigt wird.                                                                                                                                                                                                                                                                                          |
| [**LCID**](lcid.md)                         | Bei Anwendung auf die [**Library**](library.md) -Anweisung wird zusammen mit einem LocaleID-Argument das Gebiets Schema für eine Typbibliothek oder für ein Funktions Argument identifiziert, und Sie können internationale Zeichen innerhalb des Bibliotheks Blocks verwenden. Innerhalb einer **Library** -Anweisung akzeptiert die Mittel Stelle ab dem Zeitpunkt, an dem das [**LCID**](lcid.md) -Attribut verwendet wird, die nach dem angegebenen Gebiets Schema lokalisierte Eingabe. |
| [**Bibliothek**](library.md)                   | Weist den Mittel l-Compiler an, Typbibliotheks Informationen für die Schnittstellen und Klassen zu generieren, auf die in der Anweisung verwiesen wird.                                                                                                                                                                                                                                                                       |
| [**lizenziert**](licensed.md)                 | Gibt an, dass die [**Co-Klasse**](coclass.md) , auf die Sie angewendet wird, lizenziert ist, und-Instanzen müssen mit **IClassFactory2** erstellt werden.                                                                                                                                                                                                                                                                 |
| [**nonbrowsable**](nonbrowsable.md)         | Gibt an, dass die Eigenschaft in einem Objektkatalog angezeigt wird (der keine Eigenschaftswerte anzeigt), aber nicht in einem Eigenschaften Browser (in dem Eigenschaftswerte angezeigt werden) angezeigt wird.                                                                                                                                                                                                                         |
| [**noncreatable**](noncreatable.md)         | Verhindert, dass ein Client die Standardklassenfactory verwendet, um Instanzen einer Objektschnittstelle zu erstellen.                                                                                                                                                                                                                                                                                                     |
| [**nonextensible**](nonextensible.md)       | Gibt an, dass die **IDispatch** -Implementierung nur die Eigenschaften und Methoden enthält, die in der Schnittstellen Beschreibung aufgeführt sind, und kann zur Laufzeit nicht mit zusätzlichen Membern erweitert werden.                                                                                                                                                                                                                |
| [**oleautomation**](oleautomation.md)       | Gibt an, dass eine Schnittstelle mit Automation kompatibel ist.                                                                                                                                                                                                                                                                                                                                             |
| [**optionale**](optional.md)                 | Gibt einen optionalen Parameter für eine Member-Funktion an.                                                                                                                                                                                                                                                                                                                                                 |
| [**propget**](propget.md)                   | Gibt eine eigenschaftenaccessorfunktion an.                                                                                                                                                                                                                                                                                                                                                                |
| [**propput**](propput.md)                   | Gibt eine Eigenschafts Einstellungs Funktion an.                                                                                                                                                                                                                                                                                                                                                                 |
| [**propputref**](propputref.md)             | Gibt eine Eigenschafts Einstellungs Funktion an, die einen Verweis anstelle eines Werts verwendet.                                                                                                                                                                                                                                                                                                                        |
| [**public**](public.md)                     | Stellt sicher, dass ein Alias, der mit [**typedef**](typedef.md) deklariert wird, Teil der Typbibliothek ist.                                                                                                                                                                                                                                                                                                       |
| [**ReadOnly**](readonly.md)                 | Untersagt das Zuweisen eines neuen Werts zu einer Variablen.                                                                                                                                                                                                                                                                                                                                                         |
| [**requestedit**](requestedit.md)           | Gibt an, dass die Eigenschaft die **OnRequestEdit** -Benachrichtigung unterstützt.                                                                                                                                                                                                                                                                                                                               |
| [**begrenz**](restricted.md)             | Gibt an, dass eine Bibliothek oder ein Member eines Moduls, einer Schnittstelle oder einer dispinterface nicht willkürlich aufgerufen werden kann.                                                                                                                                                                                                                                                                                             |
| [**retval**](retval.md)                     | Gibt den Parameter an, der den Rückgabewert des Members empfängt.                                                                                                                                                                                                                                                                                                                                 |
| [**Ausgangs**](source.md)                     | Gibt an, dass ein Member einer [**Co-Klasse**](coclass.md), einer Eigenschaft oder Methode eine Quelle von Ereignissen ist.                                                                                                                                                                                                                                                                                                    |
| [**Zeichenfolge**](string.md)                     | Gibt an, dass das eindimensionale [**char**](char-idl.md)-, [**WCHAR \_ t**](wchar-t.md)-, [**Bytearray**](byte.md) -Array oder der-Zeiger auf ein solches Array als Zeichenfolge behandelt werden muss.                                                                                                                                                                                                      |
| [**uidefault**](uidefault.md)               | Gibt an, dass der Typinformationsmember der Standard Member für die Anzeige in der Benutzeroberfläche ist.                                                                                                                                                                                                                                                                                                    |
| [**usesgetlasterror**](usesgetlasterror.md) | Gibt an, dass ein Modul Einstiegspunkt [**SetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-setlasterror) zum Zurückgeben von Fehlercodes verwendet und dass ein Aufrufer dann [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror) aufrufen kann, um den Fehlercode abzurufen, wenn beim Eingeben einer Funktion ein Fehler auftritt.                                                                                                                                                     |
| [**UUID**](uuid.md)                         | Legt einen eindeutigen Bezeichner für eine Typbibliothek, eine [**Co-Klasse**](coclass.md)oder eine Schnittstelle fest.                                                                                                                                                                                                                                                                                                           |
| [**vararg**](vararg.md)                     | Gibt an, dass die Funktion eine variable Argumentenanzahl akzeptiert.                                                                                                                                                                                                                                                                                                                                      |
| [**version**](version.md)                   | Identifiziert eine bestimmte Version einer Typbibliothek.                                                                                                                                                                                                                                                                                                                                                     |
