# <a name="additional-files-required-to-generate-a-type-library"></a>Zusätzliche Dateien, die zum Generieren einer Typbibliothek erforderlich sind

Um eine IDL-Datei zu kompilieren, die eine Library-Anweisung enthält, müssen sich die OLE-und OLE-Automatisierungs-dll-Dateien auf dem System befinden. Diese Dateien werden automatisch installiert.
