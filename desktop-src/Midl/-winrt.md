# <a name="winrt-switch"></a>/WinRT-Schalter

Der [**/WinRT**](-winmd.md) -Schalter aktiviert Windows-Runtime Semantik.

``` syntax
midlrt /winrt 
```

## <a name="switch-options"></a>Optionen wechseln

Dieser Switch hat keine Parameter.
