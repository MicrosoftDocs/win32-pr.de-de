# <a name="help--switch"></a>Schalter /help (/?)

Der **Schalter /help** (**/?**) weist den Compiler an, eine Nutzungsmeldung mit allen verfügbaren MIDL-Befehlszeilenschaltern und -Optionen anzuzeigen.

Der [**Schalter /confirm**](-confirm.md) zeigt die vom Benutzer ausgewählten MIDL-Compilerschaltereinstellungen an.

``` syntax
midl /help
midl /?
```

## <a name="switch-options"></a>Switch-Optionen

Dieser Schalter verfügt über keine Parameter.

## <a name="examples"></a>Beispiele

**midl /help**

**midl /?**
