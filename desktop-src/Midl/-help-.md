# <a name="help--switch"></a>/Help (/?)-Schalter

Der **/Help** (**/?)-** Schalter weist den Compiler an, eine Verwendungs Meldung anzuzeigen, in der alle verfügbaren Mittell-Befehls Zeilenschalter und-Optionen aufgeführt sind.

Der [**/Confirm**](-confirm.md) -Schalter zeigt die vom Benutzer ausgewählten Einstellungen für den mittleren Compilerschalter an.

``` syntax
midl /help
midl /?
```

## <a name="switch-options"></a>Optionen wechseln

Dieser Switch hat keine Parameter.

## <a name="examples"></a>Beispiele

**Mittel l/Help**

**Mittel l/?**
