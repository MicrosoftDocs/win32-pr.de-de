# <a name="structure-and-union-attributes"></a>Struktur-und Union-Attribute

Verwenden Sie **die \_ Switch** - \* Attribute, um das Merkmal einer Union in einem Remote Prozedur aufzurufen. Verwenden Sie das [**Ignore**](ignore.md) -Attribut, um bestimmte Struktur-oder Union-Member als lokales Element für die Client Anwendung festzulegen.



| Attribut                           | Verbrauch                                                                                                                         |
|-------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**Not**](switch.md)            | Wählt die Diskriminante für eine gekapselte Union aus.                                                                           |
| [**Switch \_ ist**](switch-is.md)     | Identifiziert den Diskriminanten für eine nicht gekapselte Union.                                                                      |
| [**\_Switchtyp**](switch-type.md) | Identifiziert den Typ des Diskriminanten für eine nicht gekapselte Union.                                                          |
| [**lassen**](ignore.md)            | Gibt an, dass ein in einer Struktur oder Union enthaltener Zeiger und das vom Zeiger festgestellte Objekt nicht übertragen werden soll. |



 

Sie können auch die [Zeiger Attribute Array und Format](array-and-sized-pointer-attributes.md) verwenden, um die Merkmale von Struktur-oder Union-Membern anzugeben.
