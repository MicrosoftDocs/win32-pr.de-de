# <a name="structure-and-union-attributes"></a>Struktur- und Union-Attribute

Verwenden Sie die **\_** \* Switchattribute, um das Merkmal einer Union in einem Remoteprozeduraufruf anzugeben. Verwenden [](ignore.md) Sie das ignore-Attribut, um bestimmte Struktur- oder Union-Member als lokal für die Clientanwendung festzulegen.



| attribute                           | Verwendung                                                                                                                         |
|-------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**Wechseln**](switch.md)            | Wählt die Diskriminanz für eine gekapselte Union aus.                                                                           |
| [**switch \_ ist**](switch-is.md)     | Identifiziert die Diskriminanz für eine nicht gekapselte Union.                                                                      |
| [**\_switch-Typ**](switch-type.md) | Gibt den Typ des Diskriminanten für eine nicht gekapselte Union an.                                                          |
| [**Ignorieren**](ignore.md)            | Legt fest, dass ein Zeiger, der in einer Struktur oder Union enthalten ist, und das vom Zeiger angegebene Objekt nicht übertragen werden soll. |



 

Sie können auch die [Array- und Größenzeigerattribute](array-and-sized-pointer-attributes.md) verwenden, um Merkmale von Struktur- oder Union-Membern anzugeben.
