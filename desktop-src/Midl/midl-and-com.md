# <a name="midl-and-com"></a>Mittel l und com

Alle COM-Objekte implementieren eine oder mehrere Schnittstellen. Wenn Sie ein benutzerdefiniertes COM-Objekt erstellen, müssen Sie die Schnittstelle oder Schnittstellen in einer IDL-Datei beschreiben. Die Themen in diesem Abschnitt enthalten Informationen zur Verwendung von "Mittel l" zum Definieren von COM-Schnittstellen:

-   [Für eine COM-Schnittstelle generierte Dateien](files-generated-for-a-com-interface.md)
-   [Marshalling von OLE-Datentypen](marshaling-ole-data-types.md)

Weitere Informationen zum Definieren von COM-Schnittstellen mit mittlerer l finden Sie unter [Definieren von COM-Schnittstellen](../com/defining-com-interfaces.md). Weitere Informationen zum entwickeln generierter Dateien für verschiedene Plattformen finden Sie unter [Zielplattformen für bestimmte 32-Bit-oder 64-Bit-Plattformen](targeting-stubs-for-specific-32-bit-or-64-bit-platforms.md).
