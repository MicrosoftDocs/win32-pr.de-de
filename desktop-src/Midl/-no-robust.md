# <a name="no_robust-switch"></a>/No \_ robuster Switch

Der **/No \_ robust** -Switch weist den Mittel l-Compiler an, die [**/robust**](-robust.md) -Befehlszeilen Funktion explizit zu deaktivieren. Der Befehls Zeilenschalter **/robust** und die zugehörigen Features sind die standardmäßige Compilereinstellung für die Mittel l-Version 6.0.359 und höher.

``` syntax
midl /no_robust
```

## <a name="switch-options"></a>Optionen wechseln

Dieser Switch hat keine Parameter.

## <a name="remarks"></a>Bemerkungen

Mit der mittleren l-Version 6.0.359 und höher legt der Mittelwert Compiler standardmäßig [**/robust**](-robust.md) fest. Der **/No \_ robust** -Befehls Zeilenschalter muss verwendet werden, um das **/robust** -Feature zu deaktivieren, wenn generierte Stubdateien unter Microsoft Windows NT, Windows-95/98 oder Windows-Windows ausgeführt werden müssen.

## <a name="examples"></a>Beispiele

**Mittel l/No \_ robuster Dateiname. idl**
