# <a name="o-switch"></a>/o-Schalter

Der **/o** -Schalter leitet die Ausgabe, wie z. b. Fehlermeldungen und Warnungen, von der Standardausgabe an die angegebene Datei um.

``` syntax
midl /o outputfile
```

## <a name="switch-options"></a>Optionen wechseln

<dl> <dt>

*OutputFile* 
</dt> <dd>

Gibt den Namen der Datei an, in die der mittlerer l-Compiler die Ausgabe umleitet, wie z. b. Fehlermeldungen und Warnungen.

</dd> </dl>
