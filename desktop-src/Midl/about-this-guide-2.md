# <a name="about-this-guide"></a>Informationen zu diesem Leitfaden

Dieser Abschnitt ist eine Referenz für die Microsoft Interface Definition Language (MIDL) und ein Benutzerhandbuch für den MIDL-Compiler, der C-Sprachdateien aus IDL-Dateien generiert.

Im Abschnitt [Using the MIDL Compiler (Verwenden des MIDL-Compilers)](using-the-midl-compiler-2.md) werden die Anforderungen für den C-Compiler und den Vorprozessor aufgeführt, mit denen MIDL zusammenarbeiten muss. In diesem Abschnitt werden auch die Dateien beschrieben, die der MIDL-Compiler für RPC-Stubs, DCOM-Stubs und OLE Automation-Typbibliotheken generiert.

Der Abschnitt [Schnittstellendefinitionen und Typbibliotheken](interface-definitions-and-type-libraries.md) enthält Informationen zum Erstellen von Schnittstellendefinitionsdateien (IDL) und ACF-Dateien (Application Configuration) für MIDL, einschließlich Referenzseiten für IDL- und ACF-Attribute. Es gibt auch Informationen zum Importieren von Dateien und Typbibliotheken sowie zum Importieren von Systemheaderdateien.

Der Abschnitt [MIDL Command-Line Reference](midl-command-line-reference.md) enthält Referenzinformationen für jeden Befehlszeilenschalter und jede Switchoption, die vom MIDL-Compiler erkannt werden.

Der Abschnitt [MIDL-Sprachreferenz](midl-language-reference.md) enthält einen Verweiseintrag für jedes Schlüsselwort im Microsoft Interface Definition Language.

Im Abschnitt [MIDL Compiler Errors and Warnings (MIDL-Compilerfehler und -Warnungen)](midl-compiler-errors-and-warnings.md) werden die Fehler- und Warnmeldungen aufgelistet, die der MIDL-Compiler generieren kann.
