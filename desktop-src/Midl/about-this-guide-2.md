# <a name="about-this-guide"></a>Informationen zu diesem Leitfaden

Dieser Abschnitt ist eine Referenz für die Microsoft Interface Definition Language (mittlerer l) und ein Benutzerhandbuch für den Mittelwert Compiler, der C-Sprachdateien aus IDL-Dateien generiert.

Der Abschnitt [using the Mittel l Compiler](using-the-midl-compiler-2.md) listet die Anforderungen für den C-Compiler und den Präprozessor auf, mit dem die Mitte interagieren muss. In diesem Abschnitt werden auch die Dateien beschrieben, die der-compilercompiler für RPC-Stub-, DCOM-Stub-und OLE-Automatisierungs Typbibliotheken generiert.

Der Abschnitt [Schnittstellendefinitionen und Typbibliotheken](interface-definitions-and-type-libraries.md) enthält Informationen zum Erstellen von Schnittstellendefinitionsdateien (IDL) und Anwendungs Konfigurationsdateien (ACF) für die mittlere l, einschließlich Referenzseiten für IDL-und ACF-Attribute. Außerdem finden Sie Informationen zum Importieren von Dateien und Typbibliotheken sowie zum Importieren von System Header Dateien.

Der Abschnitt " [mittlerer l-Command-Line Referenz](midl-command-line-reference.md) " enthält Referenzinformationen zu den einzelnen Befehlszeilen Switches und Switch-Optionen, die vom-Mittel-Compiler erkannt werden.

Der Abschnitt " [Mittel l-sprach Verweis](midl-language-reference.md) " enthält einen Verweis Eintrag für jedes Schlüsselwort in der Microsoft Interface Definition Language.

Im Abschnitt " [Compilerfehler und-Warnungen](midl-compiler-errors-and-warnings.md) " werden die Fehler-und Warnmeldungen aufgelistet, die der-compilercompiler generieren kann.
