# <a name="idl-attributes"></a>IDL-Attribute

Attribute sind Schlüsselwörter, die die Merkmale einer Schnittstelle und der Daten und Methoden in dieser Schnittstelle angeben. Der Satz von Attributen, der sich auf eine einzelne Schnittstelle bezieht, wird in **\[ \]** eckige Klammern () eingeschlossen. In den folgenden Themen werden die Kategorien von Attributen vorgestellt, die in einer IDL-Datei verwendet werden können.

-   [Feldattribute](/windows/desktop/Rpc/field-attributes)
-   [Aliasing-und Marshallingattribute](aliasing-and-marshaling-attributes.md)
-   [Asynchrone Attribute](asynchronous-attributes.md)
-   [Array-und Sized-Pointer Attribute](array-and-sized-pointer-attributes.md)
-   [Datentyp Attribute](data-type-attributes.md)
-   [Direktionale Attribute](directional-attributes.md)
-   [Funktions Aufrufattribute](function-call-attributes.md)
-   [Schnittstellen Header Attribute](interface-header-attributes.md)
-   [Leistungs Attribute](performance-attributes.md)
-   [Zeiger-Typattribute](pointer-type-attributes.md)
-   [Struktur-und Union-Attribute](structure-and-union-attributes.md)
-   [Typbibliotheks Attribute](type-library-attributes.md)
