# <a name="idl-attributes"></a>IDL-Attribute

Attribute sind Schlüsselwörter, die die Merkmale einer Schnittstelle sowie der Daten und Methoden innerhalb dieser Schnittstelle angeben. Der Satz von Attributen, der für eine einzelne **\[ \]** Schnittstelle gilt, wird in eckige Klammern () eingeschlossen. In den folgenden Themen werden die Kategorien von Attributen dargestellt, die in einer IDL-Datei verwendet werden können.

-   [Feldattribute](/windows/desktop/Rpc/field-attributes)
-   [Aliasing- und Marshallingattribute](aliasing-and-marshaling-attributes.md)
-   [Asynchrone Attribute](asynchronous-attributes.md)
-   [Array- und Sized-Pointer attribute](array-and-sized-pointer-attributes.md)
-   [Datentypattribute](data-type-attributes.md)
-   [Direktionale Attribute](directional-attributes.md)
-   [Funktionsaufrufattribute](function-call-attributes.md)
-   [Schnittstellenheaderattribute](interface-header-attributes.md)
-   [Leistungsattribute](performance-attributes.md)
-   [Zeigertypattribute](pointer-type-attributes.md)
-   [Struktur- und Union-Attribute](structure-and-union-attributes.md)
-   [Typbibliotheksattribute](type-library-attributes.md)
