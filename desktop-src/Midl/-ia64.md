# <a name="ia64-switch"></a>/ia64-Schalter

Der **/ia64** -Schalter leitet den Mittelwert Compiler zum Generieren von Stubdateien oder einer Typbibliotheks Datei für eine Intel Architecture 64-Bit-Umgebung.

``` syntax
midl /ia64
```

## <a name="remarks"></a>Bemerkungen

> [!Note]  
> Es ist nicht möglich, mit MIDL.exe einen 64-Bit-TLB unter Windows 2000 zu entwickeln.
