# <a name="ia64-switch"></a>/ia64-Schalter

Der **Schalter /ia64** leitet den MIDL-Compiler an, Stubdateien oder eine Typbibliotheksdatei für eine Intel Architecture 64-Bit-Umgebung zu generieren.

``` syntax
midl /ia64
```

## <a name="remarks"></a>Hinweise

> [!Note]  
> Es ist nicht möglich, MIDL.exe zu verwenden, um einen 64-Bit-TLB auf Windows 2000 zu erzeugen.
