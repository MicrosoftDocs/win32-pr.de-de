# <a name="winmd-switch"></a>/winmd-Schalter

Der Schalter [**/winmd**](-metadata-dir.md) gibt den Namen der Ausgabe Metadatendatei (. winmd) an.

``` syntax
midlrt /winmd metadata_filename
```

## <a name="switch-options"></a>Optionen wechseln

<dl> <dt>

*\_Metadatenname* 
</dt> <dd>

Gibt den Namen der Metadatendatei an.

</dd> </dl>
