# <a name="winmd-switch"></a>/winmd switch

Der [**Schalter /winmd**](-metadata-dir.md) gibt den Namen der Ausgabemetadatendatei (.winmd) an.

``` syntax
midlrt /winmd metadata_filename
```

## <a name="switch-options"></a>Switch-Optionen

<dl> <dt>

*\_Metadatendateiname* 
</dt> <dd>

Gibt den Namen der Metadatendatei an.

</dd> </dl>

## <a name="remarks"></a>Bemerkungen

## <a name="examples"></a>Beispiele

**midlrt /winmd file.winmd**
