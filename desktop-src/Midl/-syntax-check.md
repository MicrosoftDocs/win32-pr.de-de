# <a name="syntax_check-switch"></a>/Syntax \_ aktivieren

Der Schalter **/Syntax \_ Check** gibt an, dass der Compiler nur die Syntax prüft und keine Ausgabedateien generiert.

``` syntax
midl /syntax_check
midl /Zs
```

## <a name="switch-options"></a>Optionen wechseln

Dieser Switch hat keine Parameter.

## <a name="remarks"></a>Bemerkungen

Dieser Schalter überschreibt alle anderen Switches, die Informationen zu Ausgabedateien angeben.

Sie können auch den Syntax Überprüfungs Modus mit der [**/ZS**](-zs.md)-Compileroption angeben.

## <a name="examples"></a>Beispiele

**Mittel l/ZS Dateiname. idl**

**Mittel l/Syntax \_ Überprüfen von Dateiname. idl**
