# <a name="tlb-switch"></a>/TLB-Schalter

Der **/tlb** -Schalter gibt einen Dateinamen für die vom Mittel l-Compiler generierte Typbibliothek an.

``` syntax
midl /tlb filename
```

## <a name="switch-options"></a>Optionen wechseln

<dl> <dt>

*filename* 
</dt> <dd>

Gibt den Namen der Ausgabetyp-Bibliotheksdatei (TLB-Datei) an. Wenn der Schalter **/tlb** nicht verwendet wird, hat die TLB-Datei standardmäßig denselben Namen wie die IDL-Datei mit der Erweiterung. TLB.

</dd> </dl>

## <a name="examples"></a>Beispiele

**Mittel l/TLB newname. tlb**
