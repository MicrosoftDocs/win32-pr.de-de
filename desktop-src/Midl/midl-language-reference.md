| Thema | BESCHREIBUNG |
| - | - |
| [Reservierte Schlüsselwörter](reserved-keywords.md) | In diesem Thema werden die reservierten Schlüsselwörter in der mittleren l aufgeführt. Sie dürfen diese Schlüsselwörter nicht in den Namen ihrer Typen und Member verwenden. |
| [Serialisierung im gemischten Modus von Kontext Handles](mixed-mode-serialization-of-context-handles.md) | In Microsoft Windows XP kann eine einzelne Schnittstelle sowohl serialisierte als auch nicht serialisierte Kontext Handles aufnehmen, die als "im gemischten Modus" bezeichnete Serialisierung bezeichnet werden. |
