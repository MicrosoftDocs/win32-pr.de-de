| Thema | Beschreibung |
| - | - |
| [Reservierte Schlüsselwörter](reserved-keywords.md) | In diesem Thema werden die reservierten Schlüsselwörter in MIDL aufgeführt. Sie dürfen diese Schlüsselwörter nicht in den Namen Ihrer Typen und Member verwenden. |
| [Serialisierung von Kontexthandles im gemischten Modus](mixed-mode-serialization-of-context-handles.md) | In Microsoft Windows XP kann eine einzelne Schnittstelle sowohl serialisierte als auch nicht serialisierte Kontexthandles aufnehmen, die als Serialisierung im gemischten Modus bezeichnet wird. |
