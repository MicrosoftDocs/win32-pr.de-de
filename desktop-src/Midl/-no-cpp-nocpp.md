# <a name="no_cpp-nocpp"></a>/No \_ cpp,/nocpp

Der **/No \_ cpp** -Schalter gibt an, dass der mittlerer l-Compiler den C-Präprozessor nicht aufruft, um die IDL-Datei vorzuverarbeiten.

Der Schalter " **/No \_ cpp** " hat Vorrang vor den [**/cpp- \_ cmd**](-cpp-cmd.md) -und [**/cpp- \_ opt**](-cpp-opt.md) -Switches.

## <a name="switch-options"></a>Optionen wechseln

``` syntax
midl /no_cpp
midl /nocpp
```

## <a name="example"></a>Beispiel

**Mittel l/No \_ cpp filename. idl**
