# <a name="ns_prefix-switch"></a>/NS- \_ Präfix Schalter

Mit dem Schalter **/NS \_ prefix** wird das Präfix "abi" allen Typen vorangestellt.

``` syntax
midlrt /ns_prefix 
```

## <a name="switch-options"></a>Optionen wechseln

Dieser Switch hat keine Parameter.
