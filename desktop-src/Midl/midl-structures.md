# <a name="midl-structures"></a>MIDL-Strukturen

Normale C-Semantik gilt für die Felder von Basistypen. Felder komplexerer Typen, z. B. Zeiger, Arrays und andere konstruierte Typen, können durch Typ- oder [**Feldattribute \_ geändert werden.**](/windows/desktop/Rpc/field-attributes) Weitere Informationen finden Sie unter [**Struktur**](struct.md).
