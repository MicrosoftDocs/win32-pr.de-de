# <a name="midl-structures"></a>Mittel l-Strukturen

Die normale C-Semantik gilt für die Felder von Basis Typen. Felder komplexer Typen, z. b. Zeiger, Arrays und andere erstellte Typen, können nach Typ-oder [**Feld \_ Attributen**](/windows/desktop/Rpc/field-attributes)geändert werden. Weitere Informationen finden Sie unter [**struct**](struct.md).
