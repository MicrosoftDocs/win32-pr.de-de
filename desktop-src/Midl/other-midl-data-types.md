# <a name="other-midl-data-types"></a>Andere Datentypen in der Mitte

In der Mitte werden eine Reihe von com-und Automatisierungs Datentypen, z. b. **BSTR**, **Variant** und **SAFEARRAY**, sowie com-Handles, wie z. b. **HMETAFILE \_ PICT**, **HENHMETAFILE**, **HMETAFILE**, **hBitmap**, **hpalette** und **HGLOBAL**, erkannt. Weitere Informationen, einschließlich der Namen der Import Dateien, die für die Verwendung dieser Typen erforderlich sind, finden Sie unter Mars Hallen von [OLE-Datentypen](marshaling-ole-data-types.md).
