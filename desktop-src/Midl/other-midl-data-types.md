# <a name="other-midl-data-types"></a>Andere MIDL-Datentypen

MIDL erkennt eine Reihe von COM- und Automation-Datentypen wie **BSTR,** **VARIANT** und **SAFEARRAY** sowie COM-Handles wie **HMETAFILE \_ PICT,** **HENHMETAFILE,** **HMETAFILE,** **HBITMAP,** **HPALETTE** und **LKLOBAL**. Weitere Informationen, einschließlich der Namen der Importdateien, die für die Verwendung dieser Typen erforderlich sind, finden Sie unter [Marshallen von OLE-Datentypen.](marshaling-ole-data-types.md)
