# <a name="amd64-switch"></a>/amd64-Switch

Der **Schalter /amd64** weist den MIDL-Compiler an, Stubdateien oder eine Typbibliotheksdatei f체r eine 64-Bit-Umgebung mit erweiterten Mikroger채ten zu generieren.

``` syntax

```

## <a name="remarks"></a>Hinweise

> [!Note]  
> Der **/x64-Switch,** der von der MIDL.exe Befehlszeilenhilfe beschrieben wird, ist veraltet. Stattdessen sollte **/amd64** verwendet werden.
