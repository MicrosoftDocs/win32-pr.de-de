# <a name="strict-type-checking"></a>Strikte Typüberprüfung

Die Header Datei "Windows. h" enthält Definitionen, Makros und Strukturen, die Sie beim Schreiben von Quellcode unterstützen, der Zwischenversionen von Windows portabel ist. Einige dieser Features werden aktiviert, wenn Sie beim Entwickeln der Anwendung das strikte Symbol definieren. In den folgenden Themen werden die Vorteile von strengen Features und deren Verwendung auf das Schreiben von Code erläutert.

-   [Aktivieren von Strict](enabling-strict.md)
-   [Deaktivieren von Strict](disabling-strict.md)
-   [Strikte Konformität](strict-compliance.md)
