# <a name="strict-type-checking"></a>STRICT-Typüberprüfung

Die Windows.h-Headerdatei enthält Definitionen, Makros und Strukturen, mit denen Sie Quellcode schreiben können, der zwischen Versionen von Windows. Einige dieser Features werden aktiviert, wenn Sie das STRICT-Symbol beim Erstellen Ihrer Anwendung definieren. In den folgenden Themen werden die Vorteile von STRICT-Features erläutert, und es wird erläutert, wie sich deren Verwendung auf das Schreiben von Code auswirken kann.

-   [Aktivieren von STRICT](enabling-strict.md)
-   [Deaktivieren von STRICT](disabling-strict.md)
-   [STRICT-Konformität](strict-compliance.md)
