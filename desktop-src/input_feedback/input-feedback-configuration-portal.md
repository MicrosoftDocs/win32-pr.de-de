# <a name="input-feedback-configuration"></a>Eingabe-Feedback-Konfiguration

## <a name="purpose"></a>Zweck

Die Themen in diesem Abschnitt bieten eine Übersicht über die Konfiguration von Eingabe Feedback in Windows 8. Die Eingabe-Feedback-Konfiguration ermöglicht den Zugriff auf visuelle Feedback Einstellungen für Benutzerinteraktionen.

In der hier bereitgestellten Themensammlung wird beschrieben, wie Sie die Eingabe-Feedback-Konfiguration verwenden, um die Benutzer Leistung Ihrer APP zu optimieren. Beispielsweise sind standardmäßige visuelle Feedback Einstellungen in einer Spiele-App aufgrund von Störungen oder Konflikten mit dem Spiel selbst möglicherweise nicht erforderlich oder erwünscht.

Die Feedback-configurationapis für Eingaben sind für Entwickler konzipiert, die Benutzeroberflächen-Frameworks erstellen, die eine konsistente touchoptimierte Benutzeroberfläche für alle Desktop Anwendungen bereitstellen.
