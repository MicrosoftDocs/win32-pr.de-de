# <a name="input-feedback-configuration"></a>Konfiguration des Eingabefeedbacks

## <a name="purpose"></a>Zweck

Die Themen in diesem Abschnitt bieten eine Übersicht über die Konfiguration von Eingabefeedback in Windows 8. Die Eingabefeedbackkonfiguration ermöglicht den Zugriff auf Einstellungen für visuelles Feedback für Benutzerinteraktionen.

In der hier bereitgestellten Themensammlung wird beschrieben, wie Sie die Eingabefeedbackkonfiguration verwenden, um die Benutzerfreundlichkeit Ihrer App zu optimieren. Beispielsweise sind standardmäßige visuelle Feedbackeinstellungen in einer Spiele-App aufgrund von Störungen oder Konflikten mit dem Spiel selbst möglicherweise nicht erforderlich oder wünschenswert.

Die Input Feedback ConfigurationAPIs sind für Entwickler konzipiert, die Benutzeroberflächenframeworks erstellen, die eine konsistente touchoptimierte Benutzeroberfläche für Desktopanwendungen bereitstellen.
