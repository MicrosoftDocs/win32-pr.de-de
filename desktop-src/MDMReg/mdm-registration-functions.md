# <a name="mdm-registration-functions"></a>MDM-Registrierungsfunktionen

Die folgenden Funktionen werden von der MDM-Registrierung verwendet.

## <a name="in-this-section"></a>In diesem Abschnitt

<dl> <dt>

[**Discovermanagementservice**](/windows/desktop/api/MDMRegistration/nf-mdmregistration-discovermanagementservice)
</dt> <dd>

Ermittelt den MDM-Dienst.

</dd> <dt>

[**Discovermanagementserviceex**](/windows/desktop/api/MDMRegistration/nf-mdmregistration-discovermanagementserviceex)
</dt> <dd>

Ermittelt den MDM-Dienst mithilfe eines Kandidaten Servers.

</dd> <dt>

[**Getdeviceregistrationinfo**](/windows/desktop/api/MDMRegistration/nf-mdmregistration-getdeviceregistrationinfo)
</dt> <dd>

Ruft die Geräte Registrierungsinformationen ab.

</dd> <dt>

[**Getmanagementapphyperlink**](/windows/desktop/api/MDMRegistration/nf-mdmregistration-getmanagementapphyperlink)
</dt> <dd>

Ruft den dem MDM-Dienst zugeordneten Verwaltungs-App-Hyperlink ab.

</dd> <dt>

[**Isdeviceregisteredwithmanagement**](/windows/desktop/api/MDMRegistration/nf-mdmregistration-isdeviceregisteredwithmanagement)
</dt> <dd>

Hiermit wird überprüft, ob das Gerät bei einem MDM-Dienst registriert ist.

</dd> <dt>

[**Ismanagementregistrationallowed**](/windows/desktop/api/MDMRegistration/nf-mdmregistration-ismanagementregistrationallowed)
</dt> <dd>

Überprüft, ob die MDM-Registrierung durch die lokale Richtlinie zulässig ist.

</dd> <dt>

[**Registerdevicewithmanagement**](/windows/desktop/api/MDMRegistration/nf-mdmregistration-registerdevicewithmanagement)
</dt> <dd>

Registriert ein Gerät bei einem MDM-Dienst, wobei das Registrierungs [ \[ Protokoll MS-MDE \] : Mobile Geräte](/openspecs/windows_protocols/ms-mde/5c841535-042e-489e-913c-9d783d741267)verwendet wird.

</dd> <dt>

[**Registerdevicewithmanagementusingaadanmeldeinformationen**](/windows/desktop/api/MDMRegistration/nf-mdmregistration-registerdevicewithmanagementusingaadcredentials)
</dt> <dd>

Registriert ein Gerät bei einem MDM-Dienst unter Verwendung von Azure Active Directory Anmelde Informationen (AAD).

</dd> <dt>

[**Setmanagedexternally**](/windows/desktop/api/MDMRegistration/nf-mdmregistration-setmanagedexternally)
</dt> <dd>

Gibt dem MDM-Agent an, dass das Gerät extern verwaltet wird und nicht bei einem MDM-Dienst registriert werden soll.

</dd> <dt>

[**Unregisterdevicewithmanagement**](/windows/desktop/api/MDMRegistration/nf-mdmregistration-unregisterdevicewithmanagement)
</dt> <dd>

Hebt die Registrierung eines Geräts beim MDM-Dienst auf.

</dd> </dl>
