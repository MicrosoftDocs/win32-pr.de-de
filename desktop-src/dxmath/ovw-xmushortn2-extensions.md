# <a name="xmushortn2-extensions"></a>XMUSHORTN2-Erweiterungen

[**XMUSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn2) beschreibt einen 2D-Vektor zum Speichern nicht signierter, normalisierter Werte als 16-Bit-Ganzzahlen ohne Vorzeichen, (Type `int16_t` ).

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
