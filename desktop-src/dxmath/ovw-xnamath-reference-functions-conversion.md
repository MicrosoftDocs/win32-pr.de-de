# <a name="directxmath-library-conversion-functions"></a>Konvertierungsfunktionen der DirectXMath-Bibliothek

Listet die von DirectXMath bereitgestellten Konvertierungsfunktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                       | Beschreibung                                                                                                                                                          |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMConvertFloatToHalf**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconvertfloattohalf)<br/>             | Konvertiert einen Gleitkommawert mit einfacher Genauigkeit in einen Gleitkommawert mit halber Genauigkeit.<br/>                                                                |
| [**XMConvertFloatToHalfStream**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconvertfloattohalfstream)<br/> | Konvertiert einen Stream von Gleitkommawerten mit einfacher Genauigkeit in einen Stream mit Gleitkommawerten mit halber Genauigkeit.<br/>                                          |
| [**XMConvertHalfToFloat**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconverthalftofloat)<br/>             | Konvertiert einen Gleitkommawert mit halber Genauigkeit in einen Gleitkommawert mit einfacher Genauigkeit.<br/>                                                                |
| [**XMConvertHalfToFloatStream**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconverthalftofloatstream)<br/> | Konvertiert einen Stream von Gleitkommawerten mit halber Genauigkeit in einen Stream von Gleitkommawerten mit einfacher Genauigkeit.<br/>                                          |
| [**XMConvertToDegrees**](/windows/desktop/api/DirectXMath/nf-directxmath-xmconverttodegrees)<br/>                 | Konvertiert einen im Bogenmaß gemessenen Winkel in einen in Grad gemessenen Winkel.<br/>                                                                                       |
| [**XMConvertToRadians**](/windows/desktop/api/DirectXMath/nf-directxmath-xmconverttoradians)<br/>                 | Konvertiert einen Winkel, der in Grad gemessen wird, in einen im Bogenmaß gemessenen Winkel.<br/>                                                                                       |
| [**XMConvertVectorFloatToInt**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorfloattoint)<br/>   | Konvertiert einen [**XMVECTOR**](xmvector-data-type.md) mit **float-Komponenten** in einen **XMVECTOR** mit **int32 \_ t-Komponenten** und wendet einen einheitlichen Bias an.<br/>  |
| [**XMConvertVectorFloatToUInt**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorfloattouint)<br/> | Konvertiert einen [**XMVECTOR**](xmvector-data-type.md) mit **float-Komponenten** in einen **XMVECTOR** mit **uint32 \_ t-Komponenten** und wendet einen einheitlichen Bias an.<br/> |
| [**XMConvertVectorIntToFloat**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorinttofloat)<br/>   | Konvertiert einen [**XMVECTOR**](xmvector-data-type.md) mit **int32 \_ t-Komponenten in** einen **XMVECTOR** mit **float-Komponenten** und wendet einen einheitlichen Bias an.<br/>  |
| [**XMConvertVectorUIntToFloat**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectoruinttofloat)<br/> | Konvertiert einen [**XMVECTOR**](xmvector-data-type.md) mit **uint32 \_ t-Komponenten** in einen **XMVECTOR** mit **float-Komponenten** und wendet einen einheitlichen Bias an.<br/> |
