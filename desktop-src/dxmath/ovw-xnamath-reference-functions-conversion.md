# <a name="directxmath-library-conversion-functions"></a>Konvertierungs Funktionen der directxmath-Bibliothek

Listet die von directxmath bereitgestellten Konvertierungs Funktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                       | BESCHREIBUNG                                                                                                                                                          |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Xmconvertfloatophalf**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconvertfloattohalf)<br/>             | Konvertiert einen Gleit Komma Wert mit einfacher Genauigkeit in einen Gleit Komma Wert mit halber Genauigkeit.<br/>                                                                |
| [**Xmconvertfloatdehalfstream**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconvertfloattohalfstream)<br/> | Konvertiert einen Stream von Gleit Komma Werten mit einfacher Genauigkeit in einen Stream von Gleit Komma Werten mit halber Genauigkeit.<br/>                                          |
| [**Xmconverthalf-float**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconverthalftofloat)<br/>             | Konvertiert einen Gleit Komma Wert mit halber Genauigkeit in einen Gleit Komma Wert mit einfacher Genauigkeit.<br/>                                                                |
| [**Xmconverthalf-floatstream**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconverthalftofloatstream)<br/> | Konvertiert einen Stream von Gleit Komma Werten mit halber Genauigkeit in einen Stream von Gleit Komma Werten mit einfacher Genauigkeit.<br/>                                          |
| [**Xmconvertumgrad**](/windows/desktop/api/DirectXMath/nf-directxmath-xmconverttodegrees)<br/>                 | Konvertiert einen Winkel, gemessen im Bogenmaß, in einen Wert, der in Grad gemessen wird.<br/>                                                                                       |
| [**Xmconverttoradiane**](/windows/desktop/api/DirectXMath/nf-directxmath-xmconverttoradians)<br/>                 | Konvertiert einen Winkel in Grad in einen Wert, der im Bogenmaß gemessen wird.<br/>                                                                                       |
| [**Xmconvertvector floatumint**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorfloattoint)<br/>   | Konvertiert einen [**xmvector**](xmvector-data-type.md) mit **float** -Komponenten in einen **xmvector** -Wert mit **Int32 \_ t** -Komponenten und wendet eine einheitliche Abweichung an.<br/>  |
| [**Xmconvertvector floattouint**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorfloattouint)<br/> | Konvertiert einen [**xmvector**](xmvector-data-type.md) mit **float** -Komponenten in einen **xmvector** -Wert mit **UInt32 \_ t** -Komponenten und wendet eine einheitliche Abweichung an.<br/> |
| [**Xmconvertvector intumfloat**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorinttofloat)<br/>   | Konvertiert einen [**xmvector**](xmvector-data-type.md) -Wert mit **Int32 \_ t** -Komponenten in einen **xmvector** -Wert mit **float** -Komponenten und wendet einen einheitlichen Bias an.<br/>  |
| [**Xmconvertvector User**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectoruinttofloat)<br/> | Konvertiert einen [**xmvector**](xmvector-data-type.md) -Wert mit **UInt32 \_ t** -Komponenten in einen **xmvector** -Wert mit **float** -Komponenten und wendet einen einheitlichen Bias an.<br/> |
