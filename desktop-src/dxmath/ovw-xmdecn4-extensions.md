# <a name="xmdecn4-extensions"></a>XMDECN4-Erweiterungen

[**XMDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdecn4) beschreibt einen 4D-Vektor zum Speichern von signierten, normalisierten Werten als x-,y- und z-Komponenten mit 10-Bit-Vorzeichen und eine W-Komponente mit 2 Bit Vorzeichen.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
