# <a name="xmdecn4-extensions"></a>XMDECN4-Erweiterungen

[**XMDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdecn4) beschreibt einen 4D-Vektor zum Speichern signierter, normalisierter Werte als 10-Bit-x-, y-und z-Komponenten und eine 2-Bit-signierte w-Komponente.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
