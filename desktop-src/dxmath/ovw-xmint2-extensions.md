# <a name="xmint2-extensions"></a>XMINT2-Erweiterungen

[**XMINT2**](/windows/win32/api/directxmath/ns-directxmath-xmint2) beschreibt einen 2D-Vektor, der aus zwei ganzzahligen Werten besteht.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
