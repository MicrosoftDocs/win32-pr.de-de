# <a name="directxmath-library-2d-vector-transformation-functions"></a>2D-Vektortransformationsfunktionen der DirectXMath Library

Listet die 2D-Vektortransformationsfunktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                               | Beschreibung                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector2Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transform)<br/>                         | Transformiert einen 2D-Vektor durch eine Matrix.<br/>                                                                                                   |
| [**XMVector2TransformCoord**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformcoord)<br/>               | Transformiert einen 2D-Vektor durch eine angegebene Matrix und projiziert das Ergebnis zurück in w = 1.<br/>                                                      |
| [**XMVector2TransformCoordStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformcoordstream)<br/>   | Transformiert einen Stream von 2D-Vektoren durch eine bestimmte Matrix und projiziert die resultierenden Vektoren so, dass ihre w-Koordinaten gleich 1,0 sind.<br/> |
| [**XMVector2TransformNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformnormal)<br/>             | Transformiert den 2D-Vektor normal durch die angegebene Matrix.<br/>                                                                                  |
| [**XMVector2TransformNormalStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformnormalstream)<br/> | Transformiert einen Stream von 2D-Normalvektoren durch eine angegebene Matrix.<br/>                                                                           |
| [**XMVector2TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformstream)<br/>             | Transformiert einen Stream von 2D-Vektoren durch eine angegebene Matrix.<br/>                                                                                  |
