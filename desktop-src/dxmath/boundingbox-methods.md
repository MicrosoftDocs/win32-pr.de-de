# <a name="boundingbox-methods"></a>BoundingBox-Methoden



| Methode                                                              | BESCHREIBUNG                                                                                            |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**Inhalt**](boundingbox-contains.md)<br/>                 | Testet, ob das BoundingBox-Objekt ein angegebenes Objekt enthält.<br/>                                  |
| [**"Kreatefrompoints"**](boundingbox-createfrompoints.md)<br/> | Erstellt eine BoundingBox aus Punkten.<br/>                                                          |
| [**Überschneidet**](boundingbox-intersects.md)<br/>             | Testet das BoundingBox-Objekt auf eine Überschneidung mit einem anderen Objekt.<br/>                                 |
| [**Transform**](boundingbox-transform.md)<br/>               | Transformiert die BoundingBox.<br/>                                                                 |
| [**Containedby**](/windows/desktop/api/DirectXCollision/nf-directxcollision-boundingbox-containedby)<br/>           | Testet, ob das [**BoundingBox-Feld**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) im angegebenen Frustum enthalten ist.<br/> |
| [**"Kreatefromsphere"**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-createfromsphere)<br/> | Erstellt eine BoundingBox, die groß genug ist, um die angegebene boundingsphere zu enthalten.<br/>               |
| [**Aufzeichnete**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-createmerged)<br/>         | Erstellt eine BoundingBox, die groß genug ist, um zwei angegebene boundbox-intanzen zu enthalten.<br/>             |
| [**Getcorners**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-getcorners)<br/>             | Ruft die Ecken des BoundingBox-Felds ab.<br/>                                                   |
| [**OP- \_ Zuweisung**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-operator-assign)<br/>      | Kopiert Werte aus einer anderen BoundingBox.<br/>                                                     |
