# <a name="boundingbox-methods"></a>BoundingBox-Methoden



| Methode                                                              | Beschreibung                                                                                            |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**Enthält**](boundingbox-contains.md)<br/>                 | Testet, ob boundingBox ein angegebenes -Objekt enthält.<br/>                                  |
| [**CreateFromPoints**](boundingbox-createfrompoints.md)<br/> | Erstellt eine BoundingBox aus Punkten.<br/>                                                          |
| [**Schneidet**](boundingbox-intersects.md)<br/>             | Testet boundingBox auf Schnittmenge mit einem anderen Objekt.<br/>                                 |
| [**Transform**](boundingbox-transform.md)<br/>               | Transformiert boundingBox.<br/>                                                                 |
| [**ContainedBy**](/windows/desktop/api/DirectXCollision/nf-directxcollision-boundingbox-containedby)<br/>           | Testet, ob [**boundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) im angegebenen Frustum enthalten ist.<br/> |
| [**CreateFromSphere**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-createfromsphere)<br/> | Erstellt ein BoundingBox-Feld, das groß genug ist, um die angegebene BoundingSphere-Datei zu enthalten.<br/>               |
| [**CreateMerged**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-createmerged)<br/>         | Erstellt eine BoundingBox, die groß genug ist, um zwei angegebene BoundBox-Intances zu enthalten.<br/>             |
| [**GetCorners**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-getcorners)<br/>             | Ruft die Ecken des BoundingBox ab.<br/>                                                   |
| [**op \_ Assignment**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-operator-assign)<br/>      | Kopiert Werte aus einer anderen BoundingBox.<br/>                                                     |
