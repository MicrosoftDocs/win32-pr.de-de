# <a name="xmushort2-extensions"></a>XMUSHORT2-Erweiterungen

[**XMUSHORT2 beschreibt**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushort2) einen A2D-Vektor zum Speichern von normalisierten Werten ohne Vorzeichen als 16-Bit-Ganzzahlen ohne Vorzeichen (Typ `uint16_t` ).

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
