# <a name="xmushort2-extensions"></a>XMUSHORT2-Erweiterungen

[**XMUSHORT2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushort2) beschreibt a2D Vector zum Speichern nicht signierter, normalisierter Werte als 16-Bit-Ganzzahlen ohne Vorzeichen, (Type `uint16_t` ).

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
