# <a name="directxmath-library-3d-vector-geometric-functions"></a>2D-Vektor Funktionen der directxmath-Bibliothek

Listet die geometrischen 3D-Vektor Funktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                 | BESCHREIBUNG                                                                                                                       |
|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3AngleBetweenNormals**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweennormals)<br/>       | Berechnet den Bogenwinkel zwischen zwei normalisierten 3D-Vektoren.<br/>                                                           |
| [**XMVector3AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweennormalsest)<br/> | Schätzt den Bogenwinkel zwischen zwei normalisierten 3D-Vektoren.<br/>                                                          |
| [**XMVector3AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweenvectors)<br/>       | Berechnet den radiin-Winkel zwischen zwei 3D-Vektoren.<br/>                                                                      |
| [**XMVector3ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector3clamplength)<br/>                       | Bindet die Länge eines 3D-Vektors in einen angegebenen Bereich.<br/>                                                                     |
| [**XMVector3ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector3clamplengthv)<br/>                     | Bindet die Länge eines 3D-Vektors in einen angegebenen Bereich.<br/>                                                                     |
| [**XMVector3ComponentsFromNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3componentsfromnormal)<br/>     | Durch die Verwendung eines Verweis-normal-Vektors wird ein 3D-Vektor in Komponenten aufgeteilt, die parallel und senkrecht zum normal sind.<br/> |
| [**XMVector3Cross**](/windows/win32/api/directxmath/nf-directxmath-xmvector3cross)<br/>                                   | Berechnet das Kreuz Produkt zwischen zwei 3D-Vektoren.<br/>                                                                     |
| [**XMVector3Dot**](/windows/win32/api/directxmath/nf-directxmath-xmvector3dot)<br/>                                       | Berechnet das Punktprodukt zwischen 3D-Vektoren.<br/>                                                                           |
| [**XMVector3InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector3inbounds)<br/>                             | Testet, ob die Komponenten eines 3D-Vektors innerhalb von festgelegten Begrenzungen liegen.<br/>                                                     |
| [**XMVector3Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector3length)<br/>                                 | Berechnet die Länge eines 3D-Vektors.<br/>                                                                                    |
| [**XMVector3LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lengthest)<br/>                           | Schätzt die Länge eines 3D-Vektors.<br/>                                                                                   |
| [**XMVector3LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lengthsq)<br/>                             | Berechnet das Quadrat der Länge eines 3D-Vektors.<br/>                                                                      |
| [**XMVector3LinePointDistance**](/windows/win32/api/directxmath/nf-directxmath-xmvector3linepointdistance)<br/>           | Berechnet den minimalen Abstand zwischen einer Linie und einem Punkt.<br/>                                                              |
| [**XMVector3Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector3normalize)<br/>                           | Gibt die normalisierte Version eines 3D-Vektors zurück.<br/>                                                                         |
| [**XMVector3NormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3normalizeest)<br/>                     | Schätzt die normalisierte Version eines 3D-Vektors.<br/>                                                                       |
| [**XMVector3Orthogonal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3orthogonal)<br/>                         | Berechnet einen Vektor senkrecht zu einem 3D-Vektor.<br/>                                                                        |
| [**XMVector3ReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reciprocallength)<br/>             | Berechnet die Gegenüberstellung der Länge eines 3D-Vektors.<br/>                                                                  |
| [**XMVector3ReciprocalLengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reciprocallengthest)<br/>       | Schätzt die gegenseitige Länge eines 3D-Vektors.<br/>                                                                 |
| [**XMVector3Reflect**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reflect)<br/>                               | Reflektiert einen 3D-incidentvektor über einen 3D-normal Vektor.<br/>                                                              |
| [**XMVector3Refract**](/windows/win32/api/directxmath/nf-directxmath-xmvector3refract)<br/>                               | Gibt einen 3D-Vektor Vektor über einen 3D-normal Vektor hinweg aus.<br/>                                                              |
| [**XMVector3RefractV**](/windows/win32/api/directxmath/nf-directxmath-xmvector3refractv)<br/>                             | Gibt einen 3D-Vektor Vektor über einen 3D-normal Vektor hinweg aus.<br/>                                                              |
