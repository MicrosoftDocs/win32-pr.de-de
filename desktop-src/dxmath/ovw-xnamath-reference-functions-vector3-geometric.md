# <a name="directxmath-library-3d-vector-geometric-functions"></a>Geometrische DirectXMath Library-3D-Vektorfunktionen

Listet die geometrischen 3D-Vektorfunktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                                 | BESCHREIBUNG                                                                                                                       |
|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3AngleBetweenNormals**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweennormals)<br/>       | Berechnet den Bogenwinkel zwischen zwei normalisierten 3D-Vektoren.<br/>                                                           |
| [**XMVector3AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweennormalsest)<br/> | Schätzt den Bogenwinkel zwischen zwei normalisierten 3D-Vektoren.<br/>                                                          |
| [**XMVector3AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweenvectors)<br/>       | Berechnet den Bogenwinkel zwischen zwei 3D-Vektoren.<br/>                                                                      |
| [**XMVector3ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector3clamplength)<br/>                       | Klammert die Länge eines 3D-Vektors an einen angegebenen Bereich.<br/>                                                                     |
| [**XMVector3ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector3clamplengthv)<br/>                     | Klammert die Länge eines 3D-Vektors an einen angegebenen Bereich.<br/>                                                                     |
| [**XMVector3ComponentsFromNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3componentsfromnormal)<br/>     | Unter Verwendung eines normalen Verweisvektors teilt einen 3D-Vektor in Komponenten auf, die parallel und senkrecht zur Normalität sind.<br/> |
| [**XMVector3Cross**](/windows/win32/api/directxmath/nf-directxmath-xmvector3cross)<br/>                                   | Berechnet das Kreuzprodukt zwischen zwei 3D-Vektoren.<br/>                                                                     |
| [**XMVector3Dot**](/windows/win32/api/directxmath/nf-directxmath-xmvector3dot)<br/>                                       | Berechnet das Punktprodukt zwischen 3D-Vektoren.<br/>                                                                           |
| [**XMVector3InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector3inbounds)<br/>                             | Testet, ob die Komponenten eines 3D-Vektors innerhalb der festgelegten Grenzen liegen.<br/>                                                     |
| [**XMVector3Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector3length)<br/>                                 | Berechnet die Länge eines 3D-Vektors.<br/>                                                                                    |
| [**XMVector3LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lengthest)<br/>                           | Schätzt die Länge eines 3D-Vektors.<br/>                                                                                   |
| [**XMVector3LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lengthsq)<br/>                             | Berechnet das Quadrat der Länge eines 3D-Vektors.<br/>                                                                      |
| [**XMVector3LinePointDistance**](/windows/win32/api/directxmath/nf-directxmath-xmvector3linepointdistance)<br/>           | Berechnet den Mindestabstand zwischen einer Linie und einem Punkt.<br/>                                                              |
| [**XMVector3Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector3normalize)<br/>                           | Gibt die normalisierte Version eines 3D-Vektors zurück.<br/>                                                                         |
| [**XMVector3NormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3normalizeest)<br/>                     | Schätzt die normalisierte Version eines 3D-Vektors.<br/>                                                                       |
| [**XMVector3Orthogonal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3orthogonal)<br/>                         | Berechnet einen Vektor, der einem 3D-Vektor überträgt.<br/>                                                                        |
| [**XMVector3ReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reciprocallength)<br/>             | Berechnet den Kehrwert der Länge eines 3D-Vektors.<br/>                                                                  |
| [**XMVector3ReciprocalLengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reciprocallengthest)<br/>       | Schätzt den Reziproz der Länge eines 3D-Vektors.<br/>                                                                 |
| [**XMVector3Reflect**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reflect)<br/>                               | Spiegelt einen Incident-3D-Vektor über einen 3D-Normalvektor wider.<br/>                                                              |
| [**XMVector3Refract**](/windows/win32/api/directxmath/nf-directxmath-xmvector3refract)<br/>                               | Refriert einen Incident-3D-Vektor über einen 3D-Normalvektor.<br/>                                                              |
| [**XMVector3RefractV**](/windows/win32/api/directxmath/nf-directxmath-xmvector3refractv)<br/>                             | Refriert einen Incident-3D-Vektor über einen 3D-Normalvektor.<br/>                                                              |
