# <a name="xmuint4-extensions"></a>XMUINT4-Erweiterungen

[**XMUINT4 beschreibt**](/windows/win32/api/directxmath/ns-directxmath-xmuint4) eine -Struktur, die einen 4D-Vektor enthält, der aus vier ganzzahligen Werten ohne Vorzeichen besteht.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
