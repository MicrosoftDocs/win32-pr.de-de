# <a name="xmuint4-extensions"></a>XMUINT4-Erweiterungen

[**XMUINT4**](/windows/win32/api/directxmath/ns-directxmath-xmuint4) beschreibt eine Struktur mit einem 4D-Vektor, der aus vier ganzzahligen Werten ohne Vorzeichen besteht.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
