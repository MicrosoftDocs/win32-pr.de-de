# <a name="directxmath-library-2d-vector-comparison-functions"></a>DirectXMath Library– 2D-Vektorvergleichsfunktionen

Listet die 2D-Vektorvergleichsfunktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                   | Beschreibung                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector2Equal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equal)<br/>                     | Testet, ob zwei 2D-Vektoren gleich sind.<br/>                                                                                                                                                                                                        |
| [**XMVector2EqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalint)<br/>               | Testet, ob zwei 2D-Vektoren gleich sind, und behandelt jede Komponente als ganze Zahl ohne Vorzeichen.<br/>                                                                                                                                                        |
| [**XMVector2EqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalintr)<br/>             | Testet, ob zwei 2D-Vektoren gleich sind, und behandelt jede Komponente als ganze Zahl ohne Vorzeichen. Darüber hinaus gibt diese Funktion einen Vergleichswert zurück, der mithilfe von Funktionen wie [**XMComparisonAllTrue untersucht werden kann.**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)<br/> |
| [**XMVector2EqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalr)<br/>                   | Testet, ob zwei 2D-Vektoren gleich sind. Darüber hinaus gibt diese Funktion einen Vergleichswert zurück, der mithilfe von Funktionen wie [**XMComparisonAllTrue untersucht werden kann.**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)<br/>                                                 |
| [**XMVector2Greater**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greater)<br/>                 | Testet, ob ein 2D-Vektor größer als ein anderer 2D-Vektor ist.<br/>                                                                                                                                                                                 |
| [**XMVector2GreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterorequal)<br/>   | Testet, ob ein 2D-Vektor größer als oder gleich einem anderen 2D-Vektor ist.<br/>                                                                                                                                                                     |
| [**XMVector2GreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterorequalr)<br/> | Testet, ob ein 2D-Vektor größer als oder gleich einem anderen 2D-Vektor ist, und gibt einen Vergleichswert zurück, der mithilfe von Funktionen wie [**XMComparisonAllTrue untersucht werden kann.**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)<br/>                                      |
| [**XMVector2GreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterr)<br/>               | Testet, ob ein 2D-Vektor größer als ein anderer 2D-Vektor ist, und gibt einen Vergleichswert zurück, der mithilfe von Funktionen wie [**XMComparisonAllTrue untersucht werden kann.**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)<br/>                                                  |
| [**XMVector2IsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmvector2isinfinite)<br/>           | Testet, ob eine Komponente eines 2D-Vektors positiv oder negativ unendlich ist.<br/>                                                                                                                                                                   |
| [**XMVector2IsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvector2isnan)<br/>                     | Testet, ob eine Komponente eines 2D-Vektors ein NaN ist.<br/>                                                                                                                                                                                           |
| [**XMVector2Less**](/windows/win32/api/directxmath/nf-directxmath-xmvector2less)<br/>                       | Testet, ob ein 2D-Vektor kleiner als ein anderer 2D-Vektor ist.<br/>                                                                                                                                                                                    |
| [**XMVector2LessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lessorequal)<br/>         | Testet, ob ein 2D-Vektor kleiner als oder gleich einem anderen 2D-Vektor ist.<br/>                                                                                                                                                                        |
| [**XMVector2NearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2nearequal)<br/>             | Testet, ob sich ein 2D-Vektor in der Nähe eines anderen 2D-Vektors befindet.<br/>                                                                                                                                                                                         |
| [**XMVector2NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2notequal)<br/>               | Testet, ob zwei 2D-Vektoren nicht gleich sind.<br/>                                                                                                                                                                                                    |
| [**XMVector2NotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector2notequalint)<br/>         | Testen Sie, ob zwei Vektoren nicht gleich sind, und behandeln Sie jede Komponente als ganze Zahl ohne Vorzeichen.<br/>                                                                                                                                                        |
