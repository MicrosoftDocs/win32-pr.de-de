# <a name="directxmath-library-2d-vector-comparison-functions"></a>Directxmath-Bibliothek, 2D-Vektor Vergleichsfunktionen

Listet die 2D-Vektor Vergleichsfunktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                   | BESCHREIBUNG                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector2Equal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equal)<br/>                     | Testet, ob zwei 2D-Vektoren gleich sind.<br/>                                                                                                                                                                                                        |
| [**XMVector2EqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalint)<br/>               | Testet, ob zwei 2D-Vektoren gleich sind, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird.<br/>                                                                                                                                                        |
| [**XMVector2EqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalintr)<br/>             | Testet, ob zwei 2D-Vektoren gleich sind, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird. Außerdem gibt diese Funktion einen Vergleichswert zurück, der mithilfe von Funktionen wie [**xmcomparisonalltrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)überprüft werden kann.<br/> |
| [**XMVector2EqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalr)<br/>                   | Testet, ob zwei 2D-Vektoren gleich sind. Außerdem gibt diese Funktion einen Vergleichswert zurück, der mithilfe von Funktionen wie [**xmcomparisonalltrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)überprüft werden kann.<br/>                                                 |
| [**XMVector2Greater**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greater)<br/>                 | Testet, ob ein 2D-Vektor größer als ein anderer 2D-Vektor ist.<br/>                                                                                                                                                                                 |
| [**XMVector2GreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterorequal)<br/>   | Testet, ob ein 2D-Vektor größer oder gleich einem anderen 2D-Vektor ist.<br/>                                                                                                                                                                     |
| [**XMVector2GreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterorequalr)<br/> | Testet, ob ein 2D-Vektor größer oder gleich einem anderen 2D-Vektor ist, und gibt einen Vergleichswert zurück, der mithilfe von Funktionen wie [**xmcomparisonalltrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)überprüft werden kann.<br/>                                      |
| [**XMVector2GreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterr)<br/>               | Testet, ob ein 2D-Vektor größer als ein anderer 2D-Vektor ist, und gibt einen Vergleichswert zurück, der mithilfe von Funktionen wie [**xmcomparisonalltrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)überprüft werden kann.<br/>                                                  |
| [**XMVector2IsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmvector2isinfinite)<br/>           | Testet, ob eine Komponente eines 2D-Vektors positiv oder minus unendlich ist.<br/>                                                                                                                                                                   |
| [**XMVector2IsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvector2isnan)<br/>                     | Testet, ob eine Komponente eines 2D-Vektors ein NaN ist.<br/>                                                                                                                                                                                           |
| [**XMVector2Less**](/windows/win32/api/directxmath/nf-directxmath-xmvector2less)<br/>                       | Testet, ob ein 2D-Vektor kleiner als ein anderer 2D-Vektor ist.<br/>                                                                                                                                                                                    |
| [**XMVector2LessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lessorequal)<br/>         | Testet, ob ein 2D-Vektor kleiner oder gleich einem anderen 2D-Vektor ist.<br/>                                                                                                                                                                        |
| [**XMVector2NearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2nearequal)<br/>             | Testet, ob ein 2D-Vektor fast einem zweiten 2D-Vektor ähnelt.<br/>                                                                                                                                                                                         |
| [**XMVector2NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2notequal)<br/>               | Testet, ob zwei 2D-Vektoren nicht gleich sind.<br/>                                                                                                                                                                                                    |
| [**XMVector2NotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector2notequalint)<br/>         | Testet, ob zwei Vektoren nicht gleich sind, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird.<br/>                                                                                                                                                        |
