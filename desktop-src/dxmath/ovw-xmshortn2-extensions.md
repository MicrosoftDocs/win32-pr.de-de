# <a name="xmshortn2-extensions"></a>XMSHORTN2-Erweiterungen

[**XMSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn2) beschreibt einen 2D-Vektor zum Speichern von signierten normalisierten Werten als 16-Bit-Ganzzahlen mit Vorzeichen (Typ `int16_t` ).

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
