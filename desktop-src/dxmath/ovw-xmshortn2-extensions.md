# <a name="xmshortn2-extensions"></a>XMSHORTN2-Erweiterungen

[**XMSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn2) beschreibt einen 2D-Vektor zum Speichern signierter, normalisierter Werte als ganze 16-Bit-Ganzzahlen, (Type `int16_t` ).

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
