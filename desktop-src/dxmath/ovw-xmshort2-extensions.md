# <a name="xmshort2-extensions"></a>XMSHORT2-Erweiterungen

[**XMSHORT2 beschreibt**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshort2) einen 2D-Vektor, der aus 16-Bit-Komponenten mit Vorzeichen und normalisierten ganzzahligen Komponenten besteht.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
