| Methode                                                                                | BESCHREIBUNG                                                                                                        |

| [**Inhalt**](boundingorientedbox-contains.md)<br/>                           | Testet, ob [**boundingorientedbox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) ein anderes Objekt enthält.<br/>           |
| [**Überschneidet**](boundingorientedbox-intersects.md)<br/>                       | Testet das [**boundingorientedbox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) -Objekt auf eine Überschneidung mit einem anderen Objekt.<br/>      |
