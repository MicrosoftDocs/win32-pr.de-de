# <a name="xmunibble4-extensions"></a>XMUNIBBLE4-Erweiterungen

[**XMUNIBBLE4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmunibble4) beschreibt einen 4D-Vektor mit vier 4-Bit-Ganzzahlkomponenten ohne Vorzeichen.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
