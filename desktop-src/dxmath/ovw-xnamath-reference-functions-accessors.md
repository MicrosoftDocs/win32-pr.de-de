# <a name="directxmath-library-vector-accessor-functions"></a>Vektorzugriffsfunktionen der DirectXMath-Bibliothek

Listet die Vektor-Accessorfunktionen auf, die von der DirectXMath-Bibliothek bereitgestellt werden.

Mit den DirectXMath-Vektor-Accessoren können Entwickler Code schreiben, der einzelne Komponenten eines Vektors auf portable und optimale Weise erhält und legt.
