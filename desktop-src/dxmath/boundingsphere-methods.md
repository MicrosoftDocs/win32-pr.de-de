# <a name="boundingsphere-methods"></a>Boundingsphere-Methoden



| Methode                                                                           | BESCHREIBUNG                                                                                                  |
|----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**Inhalt**](boundingsphere-contains.md)<br/>                           | Testet, ob boundingsphere ein angegebenes Objekt enthält.<br/>                                     |
| [**Überschneidet**](boundingsphere-intersects.md)<br/>                       | Testet die boundingsphere-Schnittmenge mit einem-Objekt.<br/>                                         |
| [**Transform**](boundingsphere-transform.md)<br/>                         | Wandelt boundingsphere um.<br/>                                                                    |
| [**Containedby**](/windows/desktop/api/DirectXCollision/nf-directxcollision-boundingbox-containedby)<br/>                     | Testet, ob [**boundingsphere**](/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere) in der angegebenen Frustum enthalten ist.<br/> |
| [**"Kreatefromboundingbox"**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-createfromboundingbox(boundingsphere__constboundingorientedbox_))<br/> | Erstellt eine boundingsphere mit der angegebenen BoundingBox.<br/>                                    |
| [**"Kreatefrompoints"**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-createfrompoints)<br/>           | Erstellt eine neue boundingsphere aus einer Liste von Punkten.<br/>                                               |
| [**Aufzeichnete**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-createmerged)<br/>                   | Erstellt eine boundingsphere, die die beiden angegebenen boundingsphere-Objekte enthält.<br/>                  |
| [**OP- \_ Zuweisung**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-operator-assign(boundingsphere__))<br/>                | Initialisiert die boundingsphere mit Werten aus einer angegebenen boundingsphere.<br/>                       |
