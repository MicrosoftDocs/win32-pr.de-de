# <a name="boundingsphere-methods"></a>BoundingSphere-Methoden



| Methode                                                                           | BESCHREIBUNG                                                                                                  |
|----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**Enthält**](boundingsphere-contains.md)<br/>                           | Testet, ob BoundingSphere ein angegebenes -Objekt enthält.<br/>                                     |
| [**Schneidet**](boundingsphere-intersects.md)<br/>                       | Testet BoundingSphere auf Schnittmenge mit einem -Objekt.<br/>                                         |
| [**Transform**](boundingsphere-transform.md)<br/>                         | Transformiert BoundingSphere.<br/>                                                                    |
| [**ContainedBy**](/windows/desktop/api/DirectXCollision/nf-directxcollision-boundingbox-containedby)<br/>                     | Testet, ob [**BoundingSphere**](/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere) im angegebenen Frustum enthalten ist.<br/> |
| [**CreateFromBoundingBox**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-createfromboundingbox(boundingsphere__constboundingorientedbox_))<br/> | Erstellt einen BoundingSphere-Wert, der die angegebene BoundingBox enthält.<br/>                                    |
| [**CreateFromPoints**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-createfrompoints)<br/>           | Erstellt eine neue BoundingSphere aus einer Liste von Punkten.<br/>                                               |
| [**CreateMerged**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-createmerged)<br/>                   | Erstellt ein BoundingSphere-Objekt, das die beiden angegebenen BoundingSphere-Objekte enthält.<br/>                  |
| [**\_op-Zuweisung**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-operator-assign(boundingsphere__))<br/>                | Initialisiert BoundingSphere mit Werten aus einer angegebenen BoundingSphere.<br/>                       |
