# <a name="xmfloat3x3-operators"></a>XMFLOAT3X3-Operatoren

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                      | BESCHREIBUNG                                                                                                              |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**Operator =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat3x3-operator-assign)<br/>      | Weist die Vektor Komponenten Daten einer Instanz von `XMFLOAT3X3` der aktuellen Instanz von zu `XMFLOAT3X3` . <br/> |
| [**Operator ()**](xmfloat3x3-operator-parens.md)<br/> | Greift auf bestimmte Matrix Elemente zu, auf die von Zeilen und Spalten aus der aktuellen Instanz von verwiesen wird `XMFLOAT3X3` . <br/>    |
