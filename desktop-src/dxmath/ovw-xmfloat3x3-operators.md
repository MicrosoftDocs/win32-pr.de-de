# <a name="xmfloat3x3-operators"></a>XMFLOAT3X3-Operatoren

Die [**XMFLOAT3X3-Struktur**](/windows/win32/api/directxmath/ns-directxmath-xmfloat3x3) macht die folgenden Operatoren verfügbar.

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                      | BESCHREIBUNG                                                                                                              |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**operator =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat3x3-operator-assign)<br/>      | Weist die Vektorkomponentendaten aus einer Instanz von `XMFLOAT3X3` der aktuellen Instanz von `XMFLOAT3X3` zu. <br/> |
| [**Operator ()**](xmfloat3x3-operator-parens.md)<br/> | Greift auf bestimmte Matrixelemente zu, auf die von Zeile und Spalte aus der aktuellen Instanz von verwiesen `XMFLOAT3X3` wird. <br/>    |
