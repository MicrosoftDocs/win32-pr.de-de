# <a name="xmudecn4-operators"></a>XMUDECN4-Operatoren

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                             | BESCHREIBUNG                                                                                                            |
|---------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**Operator UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmudecn4-operatoruint32_t)<br/> | Gibt eine Instanz von zurück, `uint32_t` die die Komponenten der- `XMUDECN4` Instanz in einem gepackten Format enthält. <br/> |
| [**Operator =**](xmudecn4-operator-eq.md)<br/>               | Weist der aktuellen Instanz von Vektor Komponenten Daten zu `XMUDECN4` . <br/>                                       |
