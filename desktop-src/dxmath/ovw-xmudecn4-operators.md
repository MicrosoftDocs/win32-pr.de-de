# <a name="xmudecn4-operators"></a>XMUDECN4-Operatoren

Die [**XMUDECN4-Struktur**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmudecn4) macht die folgenden Operatoren verfügbar.

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                             | BESCHREIBUNG                                                                                                            |
|---------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**Operator uint32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmudecn4-operatoruint32_t)<br/> | Gibt eine Instanz von `uint32_t` zurück, die die Komponenten der `XMUDECN4` Instanz in einem gepackten Format enthält. <br/> |
| [**operator =**](xmudecn4-operator-eq.md)<br/>               | Weist der aktuellen Instanz von Vektorkomponentendaten `XMUDECN4` zu. <br/>                                       |
