# <a name="xmfloat3-extensions"></a>XMFLOAT3-Erweiterungen

[**XMFLOAT3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat3) beschreibt eine Struktur, die einen 3D-Vektor enthält, der aus drei Gleitkommawerten mit einfacher Genauigkeit besteht.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
