# <a name="xmfloat3pk-operators"></a>XMFLOAT3PK-Operatoren

Die [**XMFLOAT3PK-Struktur**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmfloat3pk) macht die folgenden Operatoren verfügbar.

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                               | BESCHREIBUNG                                                                                                             |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**operator uint32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmfloat3pk-operatoruint32_t)<br/> | Gibt eine Instanz von `uint32_t` zurück, die die Komponenten der `XMFLOAT3PK` Instanz in einem gepackten Format enthält.<br/> |
| [**operator =**](xmfloat3pk-operator-eq.md)<br/>               | Weist der aktuellen Instanz von Vektorkomponentendaten `XMFLOAT3PK` zu.<br/>                                       |
