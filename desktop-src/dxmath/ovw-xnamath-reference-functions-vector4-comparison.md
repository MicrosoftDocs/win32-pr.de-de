# <a name="directxmath-library-4d-vector-comparison-functions"></a>Directxmath-Bibliotheks-4D-Vektor Vergleichsfunktionen

Listet die 4D-Vektor Vergleichsfunktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                   | BESCHREIBUNG                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector4Equal**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equal)<br/>                     | Testet, ob zwei 4D-Vektoren gleich sind.<br/>                                                                                                                                                                                                        |
| [**XMVector4EqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalint)<br/>               | Testet, ob zwei 4D-Vektoren gleich sind, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird.<br/>                                                                                                                                                        |
| [**XMVector4EqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalintr)<br/>             | Testet, ob zwei 4D-Vektoren gleich sind, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird. Außerdem gibt diese Funktion einen Vergleichswert zurück, der mithilfe von Funktionen wie [**xmcomparisonalltrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)überprüft werden kann.<br/> |
| [**XMVector4EqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalr)<br/>                   | Testet, ob zwei 4D-Vektoren gleich sind. Außerdem gibt diese Funktion einen Vergleichswert zurück, der mithilfe von Funktionen wie [**xmcomparisonalltrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)überprüft werden kann.<br/>                                                 |
| [**XMVector4Greater**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greater)<br/>                 | Testet, ob ein 4D-Vektor größer als ein anderer 4D-Vektor ist.<br/>                                                                                                                                                                                 |
| [**XMVector4GreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterorequal)<br/>   | Testet, ob ein 4D-Vektor größer oder gleich einem anderen 4D-Vektor ist.<br/>                                                                                                                                                                     |
| [**XMVector4GreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterorequalr)<br/> | Testet, ob ein 4D-Vektor größer oder gleich einem anderen 4D-Vektor ist, und gibt einen Vergleichswert zurück, der mithilfe von Funktionen wie [**xmcomparisonalltrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)überprüft werden kann.<br/>                                      |
| [**XMVector4GreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterr)<br/>               | Testet, ob ein 4D-Vektor größer als ein anderer 4D-Vektor ist, und gibt einen Vergleichswert zurück, der mithilfe von Funktionen wie [**xmcomparisonalltrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)überprüft werden kann.<br/>                                                  |
| [**XMVector4IsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmvector4isinfinite)<br/>           | Testet, ob eine Komponente eines 4D-Vektors positiv oder minus unendlich ist.<br/>                                                                                                                                                                   |
| [**XMVector4IsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvector4isnan)<br/>                     | Testet, ob eine Komponente eines 4D-Vektors ein NaN ist.<br/>                                                                                                                                                                                           |
| [**XMVector4Less**](/windows/win32/api/directxmath/nf-directxmath-xmvector4less)<br/>                       | Testet, ob ein 4D-Vektor kleiner als ein anderer 4D-Vektor ist.<br/>                                                                                                                                                                                    |
| [**XMVector4LessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4lessorequal)<br/>         | Testet, ob ein 4D-Vektor kleiner oder gleich einem anderen 4D-Vektor ist.<br/>                                                                                                                                                                        |
| [**XMVector4NearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4nearequal)<br/>             | Testet, ob ein 4D-Vektor fast einem anderen 4D-Vektor ähnelt.<br/>                                                                                                                                                                                         |
| [**XMVector4NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4notequal)<br/>               | Testet, ob zwei 4D-Vektoren nicht gleich sind.<br/>                                                                                                                                                                                                    |
| [**XMVector4NotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector4notequalint)<br/>         | Testet, ob zwei 4D-Vektoren nicht gleich sind, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird.<br/>                                                                                                                                                     |
