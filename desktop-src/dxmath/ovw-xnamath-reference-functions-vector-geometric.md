# <a name="geometric-vector-functions"></a>Geometrische Vektor Funktionen

Listet die geometrischen Vektor Funktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                           | BESCHREIBUNG                                                                                                                                                                                                      |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Xmvector baryzentriert**](/windows/win32/api/directxmath/nf-directxmath-xmvectorbarycentric)<br/>   | Gibt mithilfe der angegebenen Positions Vektoren einen Punkt in der unter-/Schreib-Koordinaten zurück.<br/>                                                                                                                     |
| [**Xmvector barycentricv**](/windows/win32/api/directxmath/nf-directxmath-xmvectorbarycentricv)<br/> | Gibt mithilfe der angegebenen Positions Vektoren einen Punkt in der unter-/Schreib-Koordinaten zurück.<br/>                                                                                                                     |
| [**Xmvector-mullrom**](/windows/win32/api/directxmath/nf-directxmath-xmvectorcatmullrom)<br/>     | Führt eine Catmull-Rom interpolung mithilfe der angegebenen Positions Vektoren aus.<br/>                                                                                                                           |
| [**Xmvector-mullromv**](/windows/win32/api/directxmath/nf-directxmath-xmvectorcatmullromv)<br/>   | Führt eine Catmull-Rom interpolung mithilfe der angegebenen Positions Vektoren aus.<br/>                                                                                                                           |
| [**Xmvector Hermite**](/windows/win32/api/directxmath/nf-directxmath-xmvectorhermite)<br/>           | Führt eine aufgenommene Spline-interpolung mithilfe der angegebenen Vektoren aus.<br/>                                                                                                                                 |
| [**Xmvector hermitev**](/windows/win32/api/directxmath/nf-directxmath-xmvectorhermitev)<br/>         | Führt eine aufgenommene Spline-interpolung mithilfe der angegebenen Vektoren aus.<br/>                                                                                                                                 |
| [**Xmvector inbounds**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinbounds)<br/>         | Testet, ob die Komponenten eines angegebenen Vektors innerhalb von festgelegten Begrenzungen liegen.<br/>                                                                                                                                 |
| [**Xmvector inboundsr**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinboundsr)<br/>       | Testet, ob sich die Komponenten eines angegebenen Vektors innerhalb bestimmter Begrenzungen befinden, und legt einen Vergleichswert fest, der mithilfe von Funktionen wie [**xmcomparisonalltrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)überprüft werden kann.<br/> |
| [**Xmvecktorlerp**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlerp)<br/>                 | Führt eine lineare interpolung zwischen zwei Vektoren aus.<br/>                                                                                                                                                  |
| [**Xmvectorilerpv**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlerpv)<br/>               | Führt eine lineare interpolung zwischen zwei Vektoren aus.<br/>                                                                                                                                                  |
