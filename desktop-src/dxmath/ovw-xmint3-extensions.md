# <a name="xmint3-extensions"></a>XMINT3-Erweiterungen

[**XMINT3**](/windows/win32/api/directxmath/ns-directxmath-xmint3) beschreibt eine Struktur mit einem 3D-Vektor, der aus drei ganzzahligen Werten besteht.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
