# <a name="xmint3-extensions"></a>XMINT3-Erweiterungen

[**XMINT3**](/windows/win32/api/directxmath/ns-directxmath-xmint3) beschreibt eine Struktur, die einen 3D-Vektor enthält, der aus drei ganzzahligen Werten besteht.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
