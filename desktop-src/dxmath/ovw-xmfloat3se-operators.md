# <a name="xmfloat3se-operators"></a>XMFLOAT3SE-Operatoren

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                               | BESCHREIBUNG                                                                                                             |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**Operator UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmfloat3se-operatoruint32_t)<br/> | Gibt eine Instanz von zurück, `uint32_t` die die Komponenten der- `XMFLOAT3SE` Instanz in einem gepackten Format enthält.<br/> |
| [**Operator =**](xmfloat3se-operator-eq.md)<br/>               | Weist der aktuellen Instanz von Vektor Komponenten Daten zu `XMFLOAT3SE` .<br/>                                       |
