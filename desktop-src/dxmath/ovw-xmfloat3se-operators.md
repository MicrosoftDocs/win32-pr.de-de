# <a name="xmfloat3se-operators"></a>XMFLOAT3SE-Operatoren

Die [**XMFLOAT3SE-Struktur**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmfloat3se) macht die folgenden Operatoren verfügbar.

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                               | BESCHREIBUNG                                                                                                             |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**operator uint32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmfloat3se-operatoruint32_t)<br/> | Gibt eine Instanz von `uint32_t` zurück, die die Komponenten der `XMFLOAT3SE` Instanz in einem gepackten Format enthält.<br/> |
| [**operator =**](xmfloat3se-operator-eq.md)<br/>               | Weist der aktuellen Instanz von Vektorkomponentendaten `XMFLOAT3SE` zu.<br/>                                       |
