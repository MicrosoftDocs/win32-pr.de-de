# <a name="xmint4-extensions"></a>XMINT4-Erweiterungen

[**XMINT4**](/windows/win32/api/directxmath/ns-directxmath-xmint4) beschreibt eine Struktur mit einem 4D-Vektor, der aus vier ganzzahligen Werten besteht.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
