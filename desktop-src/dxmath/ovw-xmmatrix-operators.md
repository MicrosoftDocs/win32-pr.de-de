# <a name="xmmatrix-operators"></a>Xmmatrix-Operatoren

Die [**xmmatrix**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix) -Struktur macht die folgenden Operatoren verfügbar.

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                    | BESCHREIBUNG                                                                                                                                                                                   |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Operator \** _](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult)<br/>    | Führt eine Matrix Multiplikation der aktuellen Instanz von `XMMATRIX` mit einer anderen-Instanz aus `XMMATRIX` .<br/>                                                                          |
| [*Operator \* =* _*](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult-assign)<br/> | Führt eine Matrix Multiplikation der aktuellen Instanz von `XMMATRIX` mit einer anderen Instanz von aus `XMMATRIX` und gibt einen Verweis auf die aktuelle Instanz zurück, die aktualisiert wurde. <br/> |
| [**Operator =**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-assign)<br/>      | Weist die Matrix Daten einer Instanz von der `XMMATRIX` aktuellen Instanz von zu `XMMATRIX` und gibt ein `reference` an die aktuelle Instanz zurück. <br/>                                    |
| [**Operator ()**](xmmatrix-operator-parens.md)<br/> | Greift auf bestimmte Matrix Elemente zu, auf die von Zeilen und Spalten aus der aktuellen Instanz von verwiesen wird `XMMATRIX` . <br/>                                                                           |
