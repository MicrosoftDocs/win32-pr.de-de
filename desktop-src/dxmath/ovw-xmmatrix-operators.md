# <a name="xmmatrix-operators"></a>XMMATRIX-Operatoren

Die [**XMMATRIX-Struktur**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix) macht die folgenden Operatoren verfügbar.

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                    | BESCHREIBUNG                                                                                                                                                                                   |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Operator \***](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult)<br/>    | Führt eine Matrixmultiplikation der aktuellen Instanz von `XMMATRIX` durch eine andere Instanz von `XMMATRIX` aus.<br/>                                                                          |
| [**Operator \*=**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult-assign)<br/> | Führt eine Matrixmultiplikation der aktuellen Instanz von `XMMATRIX` durch eine andere Instanz von aus und gibt einen Verweis auf die aktuelle Instanz `XMMATRIX` zurück, die aktualisiert wurde. <br/> |
| [**operator =**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-assign)<br/>      | Weist die Matrixdaten einer Instanz von `XMMATRIX` der aktuellen Instanz von zu `XMMATRIX` und gibt an die aktuelle Instanz `reference` zurück. <br/>                                    |
| [**Operator ()**](xmmatrix-operator-parens.md)<br/> | Greift auf bestimmte Matrixelemente zu, auf die von Zeile und Spalte aus der aktuellen Instanz von verwiesen `XMMATRIX` wird. <br/>                                                                           |
