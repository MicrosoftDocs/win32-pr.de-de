# <a name="xmubyten4-extensions"></a>XMUBYTEN4-Erweiterungen

[**XMUBYTEN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmubyten4) beschreibt einen 3D-Vektor zum Speichern von normalisierten Werten ohne Vorzeichen als 8-Bit-Ganzzahlen (1 Byte).

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
