# <a name="boundingboxintersects-methods"></a>BoundingBox. intersekten-Methoden

Testet das BoundingBox-Objekt auf eine Überschneidung mit einem anderen Objekt.

| Methode                                                                                   | BESCHREIBUNG                                                                                                                        |
|:-----------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**BoundingBox:: intersekten (xmvector)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-intersects(fxmvector))                   | Testen Sie die BoundingBox für Schnittmenge mit einer Ebene.<br/>                                                                     |
| [**BoundingBox:: Schnittpunkte (const BoundingBox&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-intersects(constboundingbox_))         | Testet die BoundingBox auf Schnittmenge mit einer anderen BoundingBox.<br/>                                                        |
| [**BoundingBox:: intersekten (const boundingsphere&)**](/previous-versions/windows/desktop/legacy/hh437825(v=vs.85))      | Testet die BoundingBox auf Schnittmenge mit einer boundingsphere.<br/>                                                           |
| [**BoundingBox:: intersekten (const boundingfrustum-&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-intersects(constboundingfrustum_))     | Testen Sie die [**BoundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) für Schnittmenge mit einem [**boundingfrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).<br/>         |
| [**BoundingBox:: intersekten (xmvector, xmvector, float&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-intersects(fxmvector_fxmvector_float_))   | Testen Sie die BoundingBox für Schnittmenge mit einem Strahl.<br/>                                                                       |

| [**BoundingBox:: intersekten (const boundingorientedbox-&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-intersects(constboundingorientedbox_)) | Testen Sie die [**BoundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) -Schnittstelle mit einer [**boundingorientedbox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox).<br/> |
