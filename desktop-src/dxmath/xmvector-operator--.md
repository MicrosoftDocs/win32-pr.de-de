<td style="text-align: left;">Subtrahiert eine Instanz von <code>XMVECTOR</code> von einer zweiten-Instanz und gibt das Ergebnis in einer neuen Instanz von zurück <code>XMVECTOR</code> . <br/> Der <code>operator -</code> subtrahiert jede Komponente einer Instanz des <a href="xmvector-data-type.md"><strong>xmvector-Datentyps</strong></a> von jeder Komponente einer anderen Instanz von <code>XMVECTOR</code> und gibt eine neue-Instanz zurück, <code>XMVECTOR</code> die das Ergebnis enthält. <br/>
