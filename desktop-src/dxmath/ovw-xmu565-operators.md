# <a name="xmu565-operators"></a>XMU565-Operatoren

Die [**XMU565-Struktur**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu565) macht die folgenden Operatoren verfügbar.

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                           | BESCHREIBUNG                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operator uint16 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmu565-operatoruint16_t)<br/> | Gibt eine Instanz von `uint16_t` zurück, die die Komponenten der `XMU555` Instanz in einem gepackten Format enthält. <br/> |
| [**operator =**](xmu565-operator-eq.md)<br/>               | Weist der aktuellen Instanz von Vektorkomponentendaten `XMU565` zu. <br/>                                       |
