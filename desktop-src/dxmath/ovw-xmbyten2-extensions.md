# <a name="xmbyten2-extensions"></a>XMBYTEN2-Erweiterungen

[**XMBYTEN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmbyten2) beschreibt einen 2D-Vektor zum Speichern von normalisierten Werten mit Vorzeichen als ganze 8-Bit-Ganzzahlen (1 Byte).

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
