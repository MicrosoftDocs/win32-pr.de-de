# <a name="component-wise-vector-functions"></a>Komponenten Weise Vektor Funktionen

Listet die Komponenten weisen Vektor Funktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                             | BESCHREIBUNG                                                                                                                              |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**Xmvector INSERT**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinsert)<br/>               | Dreht einen Vektor nach links um eine bestimmte Anzahl von 32-Bit-Komponenten und fügt ausgewählte Elemente dieses Ergebnisses in einen anderen Vektor ein.<br/> |
| [**Xmvector mergexy**](/windows/win32/api/directxmath/nf-directxmath-xmvectormergexy)<br/>             | Erstellt einen neuen Vektor durch Kombinieren der x-und y-Komponenten von zwei Vektoren.<br/>                                                      |
| [**Xmvector mergezw**](/windows/win32/api/directxmath/nf-directxmath-xmvectormergezw)<br/>             | Erstellt einen neuen Vektor, indem die z-und w-Komponenten von zwei Vektoren kombiniert werden.<br/>                                                      |
| [**Xmvector perstumm Zeichen**](/windows/win32/api/directxmath/nf-directxmath-xmvectorpermute)<br/>             | Permutes die Komponenten von zwei Vektoren, um einen neuen Vektor zu erstellen.<br/>                                                                |
| [**Xmvector rotateleft**](/windows/win32/api/directxmath/nf-directxmath-xmvectorrotateleft)<br/>       | Dreht den Vektor um eine angegebene Anzahl von 32-Bit-Elementen nach links.<br/>                                                                 |
| [**Xmvector rotateright**](/windows/win32/api/directxmath/nf-directxmath-xmvectorrotateright)<br/>     | Dreht den Vektor nach rechts um eine angegebene Anzahl von 32-Bit-Elementen.<br/>                                                                |
| [**Xmvector SELECT**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselect)<br/>               | Führt eine Auswahl pro Komponente zwischen zwei Eingabe Vektoren aus und gibt den resultierenden Vektor zurück.<br/>                                |
| [**Xmvector selectcontrol**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselectcontrol)<br/> | Definiert einen Steuerelement Vektor zur Verwendung in [**xmvector Select**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselect).<br/>                                                 |
| [**Xmvector shiftleft**](/windows/win32/api/directxmath/nf-directxmath-xmvectorshiftleft)<br/>         | Verschiebt einen Vektor um eine bestimmte Anzahl von 32-Bit-Elementen nach links und füllt die freigegebenen Elemente mit Elementen aus einem zweiten Vektor.<br/>   |
| [**Xmvector splatw**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatw)<br/>               | Repliziert die w-Komponente eines Vektors in alle Komponenten.<br/>                                                              |
| [**Xmvector splatx**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatx)<br/>               | Repliziert die x-Komponente eines Vektors in alle Komponenten.<br/>                                                              |
| [**Xmvector splaty**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplaty)<br/>               | Repliziert die y-Komponente eines Vektors in alle Komponenten.<br/>                                                              |
| [**Xmvector-Platz**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatz)<br/>               | Repliziert die z-Komponente eines Vektors in alle Komponenten.<br/>                                                              |
| [**Xmvector Swizzle**](/windows/win32/api/directxmath/nf-directxmath-xmvectorswizzle)<br/>             | Schwimmt einen Vektor.<br/>                                                                                                            |
