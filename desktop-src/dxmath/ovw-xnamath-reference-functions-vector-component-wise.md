# <a name="component-wise-vector-functions"></a>Komponentenbezogene Vektorfunktionen

Listet die komponentenbezogenen Vektorfunktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                             | Beschreibung                                                                                                                              |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorInsert**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinsert)<br/>               | Rotiert einen Vektor nach links um eine bestimmte Anzahl von 32-Bit-Komponenten und fügt ausgewählte Elemente dieses Ergebnisses in einen anderen Vektor ein.<br/> |
| [**XMVectorMergeXY**](/windows/win32/api/directxmath/nf-directxmath-xmvectormergexy)<br/>             | Erstellt einen neuen Vektor, indem die x- und y-Komponenten zweier Vektoren kombiniert werden.<br/>                                                      |
| [**XMVectorMergeZW**](/windows/win32/api/directxmath/nf-directxmath-xmvectormergezw)<br/>             | Erstellt einen neuen Vektor, indem die z- und w-Komponenten zweier Vektoren kombiniert werden.<br/>                                                      |
| [**XMVectorPermute**](/windows/win32/api/directxmath/nf-directxmath-xmvectorpermute)<br/>             | Permutiert die Komponenten von zwei Vektoren, um einen neuen Vektor zu erstellen.<br/>                                                                |
| [**XMVectorRotateLeft**](/windows/win32/api/directxmath/nf-directxmath-xmvectorrotateleft)<br/>       | Rotiert den Vektor um eine bestimmte Anzahl von 32-Bit-Elementen nach links.<br/>                                                                 |
| [**XMVectorRotateRight**](/windows/win32/api/directxmath/nf-directxmath-xmvectorrotateright)<br/>     | Rotiert den Vektor um eine bestimmte Anzahl von 32-Bit-Elementen nach rechts.<br/>                                                                |
| [**XMVectorSelect**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselect)<br/>               | Führt eine Komponentenspezifische Auswahl zwischen zwei Eingabevektoren aus und gibt den resultierenden Vektor zurück.<br/>                                |
| [**XMVectorSelectControl**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselectcontrol)<br/> | Definiert einen Steuerelementvektor für die Verwendung in [**XMVectorSelect.**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselect)<br/>                                                 |
| [**XMVectorShiftLeft**](/windows/win32/api/directxmath/nf-directxmath-xmvectorshiftleft)<br/>         | Verschiebt einen Vektor um eine bestimmte Anzahl von 32-Bit-Elementen nach links, wobei die frei gewordenen Elemente mit Elementen aus einem zweiten Vektor gefüllt werden.<br/>   |
| [**XMVectorSplatW**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatw)<br/>               | Repliziert die w-Komponente eines Vektors in alle Komponenten.<br/>                                                              |
| [**XMVectorSplatX**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatx)<br/>               | Repliziert die x-Komponente eines Vektors auf alle Komponenten.<br/>                                                              |
| [**XMVectorSplatY**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplaty)<br/>               | Repliziert die y-Komponente eines Vektors in alle Komponenten.<br/>                                                              |
| [**XMVectorSplatZ**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatz)<br/>               | Repliziert die z-Komponente eines Vektors in alle Komponenten.<br/>                                                              |
| [**XMVectorSwizzle**](/windows/win32/api/directxmath/nf-directxmath-xmvectorswizzle)<br/>             | Wischt einen Vektor.<br/>                                                                                                            |
