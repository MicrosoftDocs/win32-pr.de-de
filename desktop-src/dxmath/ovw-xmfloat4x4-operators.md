# <a name="xmfloat4x4-operators"></a>XMFLOAT4X4-Operatoren

Die [**XMFLOAT4X4-Struktur**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4x4) macht die folgenden Operatoren verfügbar.

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                      | BESCHREIBUNG                                                                                                              |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**operator =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat4x4-operator-assign)<br/>      | Weist die Vektorkomponentendaten aus einer Instanz von `XMFLOAT4X4` der aktuellen Instanz von `XMFLOAT4X4` zu. <br/> |
| [**Operator ()**](xmfloat4x4-operator-parens.md)<br/> | Greift auf bestimmte Matrixelemente zu, auf die von Zeile und Spalte aus der aktuellen Instanz von verwiesen `XMFLOAT4X4` wird. <br/>    |
