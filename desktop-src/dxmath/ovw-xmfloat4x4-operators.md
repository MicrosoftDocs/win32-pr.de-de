# <a name="xmfloat4x4-operators"></a>XMFLOAT4X4-Operatoren

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                      | BESCHREIBUNG                                                                                                              |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**Operator =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat4x4-operator-assign)<br/>      | Weist die Vektor Komponenten Daten einer Instanz von `XMFLOAT4X4` der aktuellen Instanz von zu `XMFLOAT4X4` . <br/> |
| [**Operator ()**](xmfloat4x4-operator-parens.md)<br/> | Greift auf bestimmte Matrix Elemente zu, auf die von Zeilen und Spalten aus der aktuellen Instanz von verwiesen wird `XMFLOAT4X4` . <br/>    |
