# <a name="directxmath-3d-vector-transformation-functions"></a>Funktionen der 3D-Vektor Transformation in directxmath

Listet die 3D-Vektor Transformations Funktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                               | BESCHREIBUNG                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3InverseRotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3inverserotate)<br/>                 | Dreht einen 3D-Vektor mithilfe der Umkehrung einer Quaternion.<br/>                                                                                |
| [**XMVector3Project**](/windows/win32/api/directxmath/nf-directxmath-xmvector3project)<br/>                             | Projizieren Sie einen 3D-Vektor aus dem Objekt Raum in den Bildschirmbereich.<br/>                                                                              |
| [**XMVector3ProjectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3projectstream)<br/>                 | Projiziert einen Stream von 3D-Vektoren aus dem Objekt Raum in den Bildschirmbereich.<br/>                                                                  |
| [**XMVector3Rotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3rotate)<br/>                               | Dreht einen 3D-Vektor mithilfe einer Quaternion.<br/>                                                                                               |
| [**XMVector3Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transform)<br/>                         | Transformiert einen 3D-Vektor durch eine Matrix.<br/>                                                                                                   |
| [**XMVector3TransformCoord**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoord)<br/>               | Transformiert einen 3D-Vektor durch eine angegebene Matrix und projiziert das Ergebnis zurück in w = 1.<br/>                                                      |
| [**XMVector3TransformCoordStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoordstream)<br/>   | Transformiert einen Datenstrom von 3D-Vektoren durch eine angegebene Matrix und projiziert die resultierenden Vektoren so, dass Ihre w-Koordinaten gleich 1,0 sind.<br/> |
| [**XMVector3TransformNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormal)<br/>             | Wandelt den 3D-Vektor normal um die angegebene Matrix um.<br/>                                                                                  |
| [**XMVector3TransformNormalStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormalstream)<br/> | Transformiert einen Stream von 3D-normal Vektoren durch eine angegebene Matrix.<br/>                                                                           |
| [**XMVector3TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformstream)<br/>             | Transformiert einen Datenstrom von 3D-Vektoren durch eine angegebene Matrix.<br/>                                                                                  |
| [**XMVector3Unproject**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unproject)<br/>                         | Projiziert einen 3D-Vektor aus dem Bildschirmbereich in den Objektbereich.<br/>                                                                             |
| [**XMVector3UnprojectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unprojectstream)<br/>             | Transformiert einen Datenstrom von 3D-Vektoren aus dem Bildschirmbereich in den Objekt Raum.<br/>                                                                  |
