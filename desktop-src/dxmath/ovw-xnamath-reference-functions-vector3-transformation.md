# <a name="directxmath-3d-vector-transformation-functions"></a>DirectXMath-Transformationsfunktionen für 3D-Vektoren

Listet die 3D-Vektortransformationsfunktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                               | Beschreibung                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3InverseRotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3inverserotate)<br/>                 | Rotiert einen 3D-Vektor mithilfe der Umkehrung einer Quaternion.<br/>                                                                                |
| [**XMVector3Project**](/windows/win32/api/directxmath/nf-directxmath-xmvector3project)<br/>                             | Project einen 3D-Vektor aus dem Objektbereich in den Bildschirmbereich.<br/>                                                                              |
| [**XMVector3ProjectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3projectstream)<br/>                 | Projektet einen Stream von 3D-Vektoren aus dem Objektraum in den Bildschirmraum.<br/>                                                                  |
| [**XMVector3Rotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3rotate)<br/>                               | Rotiert einen 3D-Vektor mithilfe einer Quaternion.<br/>                                                                                               |
| [**XMVector3Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transform)<br/>                         | Transformiert einen 3D-Vektor durch eine Matrix.<br/>                                                                                                   |
| [**XMVector3TransformCoord**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoord)<br/>               | Transformiert einen 3D-Vektor durch eine bestimmte Matrix und projektiert das Ergebnis zurück in w = 1.<br/>                                                      |
| [**XMVector3TransformCoordStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoordstream)<br/>   | Transformiert einen Stream von 3D-Vektoren durch eine bestimmte Matrix und pro projectiert die resultierenden Vektoren so, dass ihre w-Koordinaten gleich 1,0 sind.<br/> |
| [**XMVector3TransformNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormal)<br/>             | Transformiert den 3D-Vektor normal durch die gegebene Matrix.<br/>                                                                                  |
| [**XMVector3TransformNormalStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormalstream)<br/> | Transformiert einen Stream von 3D-Normalvektoren durch eine bestimmte Matrix.<br/>                                                                           |
| [**XMVector3TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformstream)<br/>             | Transformiert einen Stream von 3D-Vektoren durch eine bestimmte Matrix.<br/>                                                                                  |
| [**XMVector3Unproject**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unproject)<br/>                         | Projektet einen 3D-Vektor aus dem Bildschirmbereich in den Objektraum.<br/>                                                                             |
| [**XMVector3UnprojectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unprojectstream)<br/>             | Transformiert einen Stream von 3D-Vektoren aus dem Bildschirmbereich in den Objektraum.<br/>                                                                  |
