# <a name="directxmath-library-color-functions"></a>DirectXMath Library-Farbfunktionen

Listet die von DirectXMath bereitgestellten Farbfunktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                 | Beschreibung                                                                                                                                   |
|-----------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMColorSRGBToRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorsrgbtorgb)<br/>              | Konvertiert einen sRGB-Farbvektor in RGB.<br/>                                                                                              |
| [**XMColorRGBToSRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtosrgb)<br/>              | Konvertiert einen RGB-Farbvektor in sRGB.<br/>                                                                                              |
| [**XMColorAdjustContrast**](/windows/win32/api/directxmath/nf-directxmath-xmcoloradjustcontrast)<br/>     | Passt den Kontrastwert einer Farbe an.<br/>                                                                                             |
| [**XMColorAdjustSaturation**](/windows/win32/api/directxmath/nf-directxmath-xmcoloradjustsaturation)<br/> | Passt den Sättigungswert einer Farbe an.<br/>                                                                                           |
| [**XMColorEqual**](/windows/win32/api/directxmath/nf-directxmath-xmcolorequal)<br/>                       | Überprüft die Gleichheit zweier Farben.<br/>                                                                                              |
| [**XMColorGreater**](/windows/win32/api/directxmath/nf-directxmath-xmcolorgreater)<br/>                   | Testet, ob alle Komponenten der ersten Farbe größer sind als die entsprechenden Komponenten in der zweiten Farbe.<br/>             |
| [**XMColorGreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmcolorgreaterorequal)<br/>     | Testet, ob alle Komponenten der ersten Farbe größer oder gleich den entsprechenden Komponenten der zweiten Farbe sind.<br/> |
| [**XMColorHSLToRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorhsltorgb)<br/>                 | Konvertiert HSL-Farbwerte in RGB-Farbwerte.<br/>                                                                                     |
| [**XMColorHSVToRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorhsvtorgb)<br/>                 | Konvertiert HSV-Farbwerte in RGB-Farbwerte.<br/>                                                                                     |
| [**XMColorIsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmcolorisinfinite)<br/>             | Testet, ob eine der Komponenten einer Farbe entweder positiv oder negativ unendlich ist.<br/>                                    |
| [**XMColorIsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmcolorisnan)<br/>                       | Testet, ob eine Komponente einer Farbe keine Zahl (NaN) ist.<br/>                                                               |
| [**XMColorLess**](/windows/win32/api/directxmath/nf-directxmath-xmcolorless)<br/>                         | Testet, ob alle Komponenten der ersten Farbe kleiner als die entsprechenden Komponenten der zweiten Farbe sind.<br/>                |
| [**XMColorLessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmcolorlessorequal)<br/>           | Testet, ob alle Komponenten der ersten Farbe kleiner oder gleich den entsprechenden Komponenten der zweiten Farbe sind.<br/>    |
| [**XMColorModulate**](/windows/win32/api/directxmath/nf-directxmath-xmcolormodulate)<br/>                 | Kombiniert zwei Farben, indem entsprechende Komponenten miteinander multipliziert werden.<br/>                                                                |
| [**XMColorNegative**](/windows/win32/api/directxmath/nf-directxmath-xmcolornegative)<br/>                 | Bestimmt den negativen RGB-Farbwert einer Farbe.<br/>                                                                                |
| [**XMColorNotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmcolornotequal)<br/>                 | Testet, ob zwei Farben ungleich sind.<br/>                                                                                       |
| [**XMColorRGBToHSL**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtohsl)<br/>                 | Konvertiert RGB-Farbwerte in HSL-Farbwerte.<br/>                                                                                     |
| [**XMColorRGBToHSV**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtohsv)<br/>                 | Konvertiert RGB-Farbwerte in HSV-Farbwerte.<br/>                                                                                     |
| [**XMColorRGBToXYZ**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtoxyz)<br/>                 | Konvertiert RGB-Farbwerte in XYZ-Farbwerte.<br/>                                                                                     |
| [**XMColorRGBToYUV**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtoyuv)<br/>                 | Konvertiert RGB-Farbwerte in YUV-Farbwerte.<br/>                                                                                     |
| [**XMColorRGBToYUV \_ HD**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtoyuv_hd)<br/>          | Konvertiert RGB-Farbwerte in YUV HD-Farbwerte.<br/>                                                                                  |
| [**XMColorRGBToYUV \_ RFA**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtoyuv_uhd)<br/>        | Konvertiert RGB-Farbwerte in YUV-MSID-Farbwerte.<br/>                                                                                  |
| [**XMColorSRGBToXYZ**](/windows/win32/api/directxmath/nf-directxmath-xmcolorsrgbtoxyz)<br/>               | Konvertiert SRGB-Farbwerte in XYZ-Farbwerte.<br/>                                                                                    |
| [**XMColorXYZToRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorxyztorgb)<br/>                 | Konvertiert XYZ-Farbwerte in RGB-Farbwerte.<br/>                                                                                     |
| [**XMColorXYZToSRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorxyztosrgb)<br/>               | Konvertiert XYZ-Farbwerte in SRGB-Farbwerte.<br/>                                                                                    |
| [**XMColorYUVToRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcoloryuvtorgb)<br/>                 | Konvertiert YUV-Farbwerte in RGB-Farbwerte.<br/>                                                                                     |
| [**XMColorYUVToRGB \_ HD**](/windows/win32/api/directxmath/nf-directxmath-xmcoloryuvtorgb_hd)<br/>          | Konvertiert YUV-Farbwerte in RGB HD-Farbwerte.<br/>                                                                                  |
| [**XMColorYUVToRGB- \_ HMD**](/windows/win32/api/directxmath/nf-directxmath-xmcoloryuvtorgb_uhd)<br/>        | Konvertiert YUV-Farbwerte in RGB-MSID-Farbwerte.<br/>                                                                                  |
