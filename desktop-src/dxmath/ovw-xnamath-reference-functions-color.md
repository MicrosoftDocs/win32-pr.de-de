# <a name="directxmath-library-color-functions"></a>Funktionen der directxmath-Bibliotheks Farbe

Listet die von directxmath bereitgestellten Farbfunktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                 | BESCHREIBUNG                                                                                                                                   |
|-----------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [**Xmcolorsrgbtor GB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorsrgbtorgb)<br/>              | Konvertiert einen sRGB-Farb Vektor in RGB.<br/>                                                                                              |
| [**Xmcolorrgbtsrgb**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtosrgb)<br/>              | Konvertiert einen RGB-Farb Vektor in sRGB.<br/>                                                                                              |
| [**Xmcolor-Kontrast**](/windows/win32/api/directxmath/nf-directxmath-xmcoloradjustcontrast)<br/>     | Passt den Kontrastwert einer Farbe an.<br/>                                                                                             |
| [**Xmcolor-Sättigung**](/windows/win32/api/directxmath/nf-directxmath-xmcoloradjustsaturation)<br/> | Passt den Sättigungswert einer Farbe an.<br/>                                                                                           |
| [**Xmcolorequal**](/windows/win32/api/directxmath/nf-directxmath-xmcolorequal)<br/>                       | Testet auf die Gleichheit von zwei Farben.<br/>                                                                                              |
| [**Xmcolorgreater**](/windows/win32/api/directxmath/nf-directxmath-xmcolorgreater)<br/>                   | Testet, ob alle Komponenten der ersten Farbe größer sind als die entsprechenden Komponenten in der zweiten Farbe.<br/>             |
| [**Xmcolorgreaterorequal**](/windows/win32/api/directxmath/nf-directxmath-xmcolorgreaterorequal)<br/>     | Testet, ob alle Komponenten der ersten Farbe größer oder gleich den entsprechenden Komponenten der zweiten Farbe sind.<br/> |
| [**Xmcolorhsltor GB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorhsltorgb)<br/>                 | Konvertiert HSL-Farbwerte in RGB-Farbwerte.<br/>                                                                                     |
| [**Xmcolorhsvtor GB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorhsvtorgb)<br/>                 | Konvertiert die Werte von HSV-Farben in RGB-Farbwerte.<br/>                                                                                     |
| [**Xmcolorsinfinite**](/windows/win32/api/directxmath/nf-directxmath-xmcolorisinfinite)<br/>             | Testet, ob eine der Komponenten einer Farbe entweder positiv oder minus unendlich ist.<br/>                                    |
| [**Xmcolorisnan**](/windows/win32/api/directxmath/nf-directxmath-xmcolorisnan)<br/>                       | Testet, ob eine Komponente einer Farbe keine Zahl ist (NaN).<br/>                                                               |
| [**Xmcolorless**](/windows/win32/api/directxmath/nf-directxmath-xmcolorless)<br/>                         | Testet, ob alle Komponenten der ersten Farbe kleiner sind als die entsprechenden Komponenten der zweiten Farbe.<br/>                |
| [**Xmcolorlessorequal**](/windows/win32/api/directxmath/nf-directxmath-xmcolorlessorequal)<br/>           | Testet, ob alle Komponenten der ersten Farbe kleiner oder gleich den entsprechenden Komponenten der zweiten Farbe sind.<br/>    |
| [**Xmcolormodulate**](/windows/win32/api/directxmath/nf-directxmath-xmcolormodulate)<br/>                 | Kombiniert zwei Farben, indem die entsprechenden Komponenten miteinander multipliziert werden.<br/>                                                                |
| [**Xmcolornegative**](/windows/win32/api/directxmath/nf-directxmath-xmcolornegative)<br/>                 | Bestimmt den negativen RGB-Farbwert einer Farbe.<br/>                                                                                |
| [**Xmcolornotequal**](/windows/win32/api/directxmath/nf-directxmath-xmcolornotequal)<br/>                 | Testet, ob zwei Farben ungleich sind.<br/>                                                                                       |
| [**Xmcolorrgbto HSL**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtohsl)<br/>                 | Konvertiert RGB-Farbwerte in HSL-Farbwerte.<br/>                                                                                     |
| [**Xmcolorrgbper HSV**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtohsv)<br/>                 | Konvertiert RGB-Farbwerte in HSV-Farbwerte.<br/>                                                                                     |
| [**Xmcolorrgbtoxyz**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtoxyz)<br/>                 | Konvertiert RGB-Farbwerte in XYZ-Farbwerte.<br/>                                                                                     |
| [**Xmcolorrgbzu YUV**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtoyuv)<br/>                 | Konvertiert RGB-Farbwerte in YUV-Farbwerte.<br/>                                                                                     |
| [**Xmcolorrgbzu YUV- \_ HD**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtoyuv_hd)<br/>          | Konvertiert RGB-Farbwerte in YUV-HD-Farbwerte.<br/>                                                                                  |
| [**Xmcolorsrgbtoxyz**](/windows/win32/api/directxmath/nf-directxmath-xmcolorsrgbtoxyz)<br/>               | Konvertiert sRGB-Farbwerte in XYZ-Farbwerte.<br/>                                                                                    |
| [**Xmcolorxyztor GB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorxyztorgb)<br/>                 | Konvertiert XYZ-Farbwerte in RGB-Farbwerte.<br/>                                                                                     |
| [**Xmcolorxyz-sRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorxyztosrgb)<br/>               | Konvertiert XYZ-Farbwerte in sRGB-Farbwerte.<br/>                                                                                    |
| [**Xmcoloryuvtor GB**](/windows/win32/api/directxmath/nf-directxmath-xmcoloryuvtorgb)<br/>                 | Konvertiert YUV-Farbwerte in RGB-Farbwerte.<br/>                                                                                     |
| [**Xmcoloryuvtor GB \_ HD**](/windows/win32/api/directxmath/nf-directxmath-xmcoloryuvtorgb_hd)<br/>          | Konvertiert YUV-Farbwerte in RGB-HD-Farbwerte.<br/>                                                                                  |
