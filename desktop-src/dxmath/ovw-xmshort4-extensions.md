# <a name="xmshort4-extensions"></a>XMSHORT4-Erweiterungen

[**XMSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshort4) beschreibt einen 4D-Vektor, der aus 16-Bit-ganzzahligen Komponenten mit Vorzeichen besteht.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
