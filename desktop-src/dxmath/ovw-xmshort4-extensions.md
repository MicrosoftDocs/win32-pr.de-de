# <a name="xmshort4-extensions"></a>XMSHORT4-Erweiterungen

[**XMSHORT4 beschreibt**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshort4) einen 4D-Vektor, der aus 16-Bit-Ganzzahlkomponenten mit Vorzeichen besteht.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
