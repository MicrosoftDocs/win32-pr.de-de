# <a name="xmudec4-extensions"></a>XMUDEC4-Erweiterungen

[**XMUDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmudec4)beschreibt einen 4D-Vektor mit x-, y-und z-Komponenten, die als 10-Bit-ganz Zahl Werte ohne Vorzeichen dargestellt werden, und die w-Komponente als 2-Bit-Ganzzahl ohne Vorzeichen.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
