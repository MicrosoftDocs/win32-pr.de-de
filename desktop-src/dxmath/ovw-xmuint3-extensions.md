# <a name="xmuint3-extensions"></a>XMUINT3-Erweiterungen

[**XMUINT3 beschreibt**](/windows/win32/api/directxmath/ns-directxmath-xmuint3) eine -Struktur, die einen 3D-Vektor enthält, der aus drei ganzzahligen Werten ohne Vorzeichen besteht.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
