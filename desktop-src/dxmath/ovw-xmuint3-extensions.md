# <a name="xmuint3-extensions"></a>XMUINT3-Erweiterungen

[**XMUINT3**](/windows/win32/api/directxmath/ns-directxmath-xmuint3) beschreibt eine Struktur mit einem 3D-Vektor, der aus drei ganzzahligen Werten ohne Vorzeichen besteht.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
