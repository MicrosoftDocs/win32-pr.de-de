# <a name="directxmath-library-template-functions"></a>Vorlagenfunktionen der DirectXMath-Bibliothek

Listet die Vorlagenfunktionen auf, die von DirectXMath bereitgestellt werden.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                  | Beschreibung                                                                                                                                                                                                                               |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMMax**](xmmax-template.md)<br/>                             | Vergleicht zwei numerische Datentypinstanzen oder zwei Instanzen eines Objekts, das eine Überladung von < unterstützt, und gibt die größere der beiden Instanzen zurück. Der Datentyp der Argumente und der Rückgabewert sind identisch.<br/>  |
| [**XMMin**](xmmin-template.md)<br/>                             | Vergleicht zwei numerische Datentypinstanzen oder zwei Instanzen eines Objekts, das eine Überladung von < unterstützt, und gibt die kleinere der beiden Instanzen zurück. Der Datentyp der Argumente und der Rückgabewert sind identisch.<br/> |
| [**XMVectorPermute**](xmvectorpermute-template.md)<br/>         | Permutiert die Komponenten von zwei Vektoren, um einen neuen Vektor zu erstellen.<br/>                                                                                                                                                                 |
| [**XMVectorRotateLeft**](xmvectorrotateleft-template.md)<br/>   | Rotiert den Vektor um eine bestimmte Anzahl von 32-Bit-Elementen nach links.<br/>                                                                                                                                                                  |
| [**XMVectorRotateRight**](xmvectorrotateright-template.md)<br/> | Rotiert den Vektor um eine bestimmte Anzahl von 32-Bit-Elementen nach rechts.<br/>                                                                                                                                                                 |
| [**XMVectorShiftLeft**](xmvectorshiftleft-template.md)<br/>     | Verschiebt einen Vektor um eine bestimmte Anzahl von 32-Bit-Elementen nach links, wobei die frei gewordenen Elemente mit Elementen aus einem zweiten Vektor gefüllt werden.<br/>                                                                                                    |
| [**XMVectorSwizzle**](xmvectorswizzle-template.md)<br/>         | Wischt einen Vektor.<br/>                                                                                                                                                                                                             |
| [**XMVectorInsert**](xmvectorinsert-template.md)<br/>           | Rotiert einen Vektor nach links um eine bestimmte Anzahl von 32-Bit-Komponenten und fügt ausgewählte Elemente dieses Ergebnisses in einen anderen Vektor ein.<br/>                                                                                                  |
