# <a name="xmmatrix-extensions"></a>Xmmatrix-Erweiterungen

[**Xmmatrix**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix) beschreibt eine 4 \* 4-Matrix, die auf einer 16-Byte-Grenze ausgerichtet ist, die vier Hardware Vektor Registern zugeordnet ist.

> [!Note]  
> Die hier aufgeführten Features sind nur unter C++ verfügbar.
