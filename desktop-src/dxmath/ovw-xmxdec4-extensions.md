# <a name="xmxdec4-extensions"></a>XMXDEC4-Erweiterungen

[**XMXDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdec4)beschreibt einen 4D-Vektor mit x-, y-und z-Komponenten, die als 10-Bit-ganz Zahl Werte mit Vorzeichen dargestellt werden, und die w-Komponente als 2-Bit-Ganzzahl ohne Vorzeichen.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
