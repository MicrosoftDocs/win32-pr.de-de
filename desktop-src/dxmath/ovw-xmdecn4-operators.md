# <a name="xmdecn4-operators"></a>XMDECN4-Operatoren

Die [**XMDECN4-Struktur**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdecn4) macht die folgenden Operatoren verfügbar.

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                            | BESCHREIBUNG                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**Operator uint32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmdecn4-operatoruint32_t)<br/> | Gibt eine Instanz von `uint32_t` zurück, die die Komponenten der `XMDECN4` Instanz in einem gepackten Format enthält. <br/> |
| [**operator =**](xmdecn4-operator-eq.md)<br/>               | Weist der aktuellen Instanz von Vektorkomponentendaten `XMDECN4` zu. <br/>                                       |
