# <a name="xmdecn4-operators"></a>XMDECN4-Operatoren

Die [**XMDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdecn4) -Struktur macht die folgenden Operatoren verfügbar.

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                            | BESCHREIBUNG                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**Operator UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmdecn4-operatoruint32_t)<br/> | Gibt eine Instanz von zurück, `uint32_t` die die Komponenten der- `XMDECN4` Instanz in einem gepackten Format enthält. <br/> |
| [**Operator =**](xmdecn4-operator-eq.md)<br/>               | Weist der aktuellen Instanz von Vektor Komponenten Daten zu `XMDECN4` . <br/>                                       |
