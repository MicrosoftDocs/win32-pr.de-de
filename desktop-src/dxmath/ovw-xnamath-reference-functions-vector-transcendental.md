## <a name="remarks"></a>Bemerkungen

Bei [**xmvector Exp**](/windows/win32/api/directxmath/nf-directxmath-xmvectorexp) handelt es sich um einen Kompatibilitäts Alias für [**XMVectorExp2**](/windows/win32/api/directxmath/nf-directxmath-xmvectorexp2) für vorhandenen Windows 8-Code. **Xmvector Exp** ist für Windows 8.1 veraltet. Verwenden Sie Sie nicht, sondern verwenden Sie stattdessen **XMVectorExp2** oder [**xmvector EXPE**](/windows/win32/api/directxmath/nf-directxmath-xmvectorexpe).

[**Xmvector Log**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlog) ist ein Kompatibilitäts Alias für [**XMVectorLog2**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlog2) für vorhandenen Windows 8-Code. **Xmvector Log** ist für Windows 8.1 veraltet. Verwenden Sie Sie nicht, sondern verwenden Sie stattdessen **XMVectorLog2** oder [**xmvector Loge**](/windows/win32/api/directxmath/nf-directxmath-xmvectorloge).
