## <a name="remarks"></a>Hinweise

[**XMVectorExp ist**](/windows/win32/api/directxmath/nf-directxmath-xmvectorexp) ein Kompatibilitätsalias für [**XMVectorExp2**](/windows/win32/api/directxmath/nf-directxmath-xmvectorexp2) für vorhandenen Windows 8 Code. **XMVectorExp** ist für die Windows 8.1. Verwenden Sie sie nicht, sondern **XMVectorExp2** oder [**XMVectorExpE.**](/windows/win32/api/directxmath/nf-directxmath-xmvectorexpe)

[**XMVectorLog ist**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlog) ein Kompatibilitätsalias für [**XMVectorLog2 für**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlog2) vorhandenen Windows 8 Code. **XMVectorLog** ist für die Windows 8.1. Verwenden Sie sie nicht, sondern **XMVectorLog2** oder [**XMVectorLogE.**](/windows/win32/api/directxmath/nf-directxmath-xmvectorloge)
