## <a name="remarks"></a>Hinweise

[**XMVectorExp**](/windows/win32/api/directxmath/nf-directxmath-xmvectorexp) ist ein Kompatibilitätsalias für [**XMVectorExp2**](/windows/win32/api/directxmath/nf-directxmath-xmvectorexp2) für vorhandenen Windows 8 Code. **XMVectorExp** ist für Windows 8.1 veraltet. Verwenden Sie sie nicht und stattdessen **XMVectorExp2** oder [**XMVectorExpE.**](/windows/win32/api/directxmath/nf-directxmath-xmvectorexpe)

[**XMVectorLog**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlog) ist ein Kompatibilitätsalias für [**XMVectorLog2**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlog2) für vorhandenen Windows 8 Code. **XMVectorLog** ist für Windows 8.1 veraltet. Verwenden Sie sie nicht und stattdessen **XMVectorLog2** oder [**XMVectorLogE.**](/windows/win32/api/directxmath/nf-directxmath-xmvectorloge)
