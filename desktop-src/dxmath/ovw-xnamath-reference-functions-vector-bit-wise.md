# <a name="bit-wise-vector-functions"></a>Bit-Wise Vektorfunktionen

Listet die bitweisen Vektorfunktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                         | Beschreibung                                                                                                                             |
|---------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorAndCInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorandcint)<br/>         | Berechnet das logische AND eines Vektors mit der Negation eines zweiten Vektors und behandelt jede Komponente als ganze Zahl ohne Vorzeichen.<br/> |
| [**XMVectorAndInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorandint)<br/>           | Berechnet das logische AND von zwei Vektoren und behandelt jede Komponente als ganze Zahl ohne Vorzeichen.<br/>                                     |
| [**XMVectorNorInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectornorint)<br/>           | Berechnet das logische NOR von zwei Vektoren und behandelt jede Komponente als ganze Zahl ohne Vorzeichen.<br/>                                     |
| [**XMVectorNotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvectornotequal)<br/>       | Führt einen Komponententest auf die Ungleichheit von zwei Vektoren aus.<br/>                                                             |
| [**XMVectorNotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectornotequalint)<br/> | Führt einen Komponententest auf die Ungleichheit von zwei Vektoren durch und behandelt jede Komponente als ganze Zahl ohne Vorzeichen.<br/>             |
| [**XMVectorOrInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectororint)<br/>             | Berechnet das logische OR von zwei Vektoren und behandelt jede Komponente als ganze Zahl ohne Vorzeichen.<br/>                                      |
| [**XMVectorXorInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorxorint)<br/>           | Berechnet den logischen XOR von zwei Vektoren und behandelt jede Komponente als ganze Zahl ohne Vorzeichen.<br/>                                     |
