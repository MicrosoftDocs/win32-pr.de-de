# <a name="bit-wise-vector-functions"></a>Bit-Wise Vector-Funktionen

Listet die bitweisen Vektor Funktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                         | BESCHREIBUNG                                                                                                                             |
|---------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**Xmvector andcint**](/windows/win32/api/directxmath/nf-directxmath-xmvectorandcint)<br/>         | Berechnet das logische and von einem Vektor mit der Negation eines zweiten Vektors, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird.<br/> |
| [**Xmvector andint**](/windows/win32/api/directxmath/nf-directxmath-xmvectorandint)<br/>           | Berechnet das logische and von zwei Vektoren, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird.<br/>                                     |
| [**Xmvector norint**](/windows/win32/api/directxmath/nf-directxmath-xmvectornorint)<br/>           | Berechnet das logische OR von zwei Vektoren, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird.<br/>                                     |
| [**Xmvector NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvectornotequal)<br/>       | Führt einen Komponenten spezifischen Test für die Ungleichheit von zwei Vektoren aus.<br/>                                                             |
| [**Xmvector notequalint**](/windows/win32/api/directxmath/nf-directxmath-xmvectornotequalint)<br/> | Führt einen komponentenbasierten Test für die Ungleichheit zweier Vektoren durch, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird.<br/>             |
| [**Xmvector orint**](/windows/win32/api/directxmath/nf-directxmath-xmvectororint)<br/>             | Berechnet das logische OR von zwei Vektoren, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird.<br/>                                      |
| [**Xmvector xorint**](/windows/win32/api/directxmath/nf-directxmath-xmvectorxorint)<br/>           | Berechnet das logische XOR von zwei Vektoren, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird.<br/>                                     |
