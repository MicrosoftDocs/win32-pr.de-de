# <a name="xmhalf2-extensions"></a>XMHALF2-Erweiterungen

[**XMHALF2 beschreibt**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmhalf2) einen 2D-Vektor, der aus zwei Gleitkommawerten mit halber Genauigkeit (16 Bit) besteht.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
