# <a name="xmunibble4-operators"></a>XMUNIBBLE4-Operatoren

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                               | BESCHREIBUNG                                                                                                              |
|-----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**Operator UInt16 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmunibble4-operatoruint16_t)<br/> | Gibt eine Instanz von zurück, `uint16_t` die die Komponenten der- `XMUNIBBLE4` Instanz in einem gepackten Format enthält. <br/> |
| [**Operator =**](xmunibble4-operator-eq.md)<br/>               | Weist der aktuellen Instanz von Vektor Komponenten Daten zu `XMUNIBBLE4` . <br/>                                       |
