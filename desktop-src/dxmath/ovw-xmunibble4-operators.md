# <a name="xmunibble4-operators"></a>XMUNIBBLE4-Operatoren

Die [**XMUNIBBLE4-Struktur**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmunibble4) macht die folgenden Operatoren verfügbar.

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                               | BESCHREIBUNG                                                                                                              |
|-----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**Operator uint16 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmunibble4-operatoruint16_t)<br/> | Gibt eine Instanz von `uint16_t` zurück, die die Komponenten der `XMUNIBBLE4` Instanz in einem gepackten Format enthält. <br/> |
| [**operator =**](xmunibble4-operator-eq.md)<br/>               | Weist der aktuellen Instanz von Vektorkomponentendaten `XMUNIBBLE4` zu. <br/>                                       |
