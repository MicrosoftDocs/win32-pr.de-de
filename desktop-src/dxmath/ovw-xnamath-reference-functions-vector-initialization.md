# <a name="vector-initialization-functions"></a>Vektorin initialisierungsfunktionen

Listet die Vektorin initialisierungsfunktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                     | BESCHREIBUNG                                                                                                                                      |
|---------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorFalseInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorfalseint)<br/>                   | Gibt den Nullvektor (false) zurück.<br/>                                                                                                      |
| [**XMVectorReplicate**](/windows/win32/api/directxmath/nf-directxmath-xmvectorreplicate)<br/>                 | Repliziert einen Gleitkommawert in alle vier Komponenten eines Vektors.<br/>                                                               |
| [**XMVectorReplicateInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorreplicateint)<br/>           | Repliziert einen ganzzahligen Wert in alle vier Komponenten eines Vektors.<br/>                                                                     |
| [**XMVectorReplicateIntPtr**](/windows/win32/api/directxmath/nf-directxmath-xmvectorreplicateintptr)<br/>     | Repliziert einen ganzzahligen Wert, auf den ein Zeiger verweist, in alle vier Komponenten eines Vektors.<br/>                                            |
| [**XMVectorReplicatePtr**](/windows/win32/api/directxmath/nf-directxmath-xmvectorreplicateptr)<br/>           | Repliziert einen Gleitkommawert, auf den ein Zeiger verweist, in alle vier Komponenten eines Vektors.<br/>                                         |
| [**XMVectorSet**](/windows/win32/api/directxmath/nf-directxmath-xmvectorset)<br/>                             | Erstellt einen Vektor mit vier Gleitkommawerten.<br/>                                                                                    |
| [**XMVectorSetBinaryConstant**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsetbinaryconstant)<br/> | Erstellt einen Vektor, dessen Komponenten entweder 0,0f oder 1,0f sind.<br/>                                                                    |
| [**XMVectorSetInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsetint)<br/>                       | Erstellt einen Vektor mit ganzzahligen Komponenten ohne Vorzeichen.<br/>                                                                                    |
| [**XMVectorSplatConstant**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatconstant)<br/>         | Erstellt einen Vektor mit identischen Gleitkommakomponenten. Jede Komponente ist eine Konstante, dividiert durch zwei, die zu einem ganzzahligen Exponenten ausgelöst wird.<br/> |
| [**XMVectorSplatConstantInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatconstantint)<br/>   | Erstellt einen Vektor mit identischen ganzzahligen Komponenten.<br/>                                                                                   |
| [**XMVectorSplatEpsilon**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatepsilon)<br/>           | Gibt einen Vektor zurück, dessen Komponenten jeweils epsilon sind (1,192092896e-7).<br/>                                                              |
| [**XMVectorSplatInfinity**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatinfinity)<br/>         | Gibt einen Vektor zurück, dessen Komponenten unendlich sind (0x7F800000).<br/>                                                                 |
| [**XMVectorSplatOne**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatone)<br/>                   | Gibt einen Vektor zurück, dessen Komponenten jeweils eins sind.<br/>                                                                                   |
| [**XMVectorSplatQNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatqnan)<br/>                 | Gibt einen Vektor zurück, dessen Komponenten QNaN (0x7CF00000) sind.<br/>                                                                     |
| [**XMVectorSplatSignMask**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatsignmask)<br/>         | Gibt einen Vektor zurück, dessen Komponenten jeweils die Zeichenmaske (0x80000000.<br/>                                                            |
| [**XMVectorTrueInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectortrueint)<br/>                     | Gibt einen Vektor zurück, dessen Komponenten true (0xFFFFFFFF) darstellt.<br/>                                                              |
| [**XMVectorZero**](/windows/win32/api/directxmath/nf-directxmath-xmvectorzero)<br/>                           | Erstellt den Nullvektor.<br/>                                                                                                              |
