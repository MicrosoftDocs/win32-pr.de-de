# <a name="vector-initialization-functions"></a>Vektor Initialisierungs Funktionen

Listet die Vektor Initialisierungs Funktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                     | BESCHREIBUNG                                                                                                                                      |
|---------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Xmvector falseint**](/windows/win32/api/directxmath/nf-directxmath-xmvectorfalseint)<br/>                   | Gibt den Vektor NULL (false) zurück.<br/>                                                                                                      |
| [**Xmvector replizieren**](/windows/win32/api/directxmath/nf-directxmath-xmvectorreplicate)<br/>                 | Repliziert einen Gleit Komma Wert in alle vier Komponenten eines Vektors.<br/>                                                               |
| [**Xmvector replialisieint**](/windows/win32/api/directxmath/nf-directxmath-xmvectorreplicateint)<br/>           | Repliziert einen ganzzahligen Wert in alle vier Komponenten eines Vektors.<br/>                                                                     |
| [**Xmvector replialisieintptr**](/windows/win32/api/directxmath/nf-directxmath-xmvectorreplicateintptr)<br/>     | Repliziert einen ganzzahligen Wert, auf den von einem Zeiger verwiesen wird, in alle vier Komponenten eines Vektors.<br/>                                            |
| [**Xmvector replialisieptr**](/windows/win32/api/directxmath/nf-directxmath-xmvectorreplicateptr)<br/>           | Repliziert einen Gleit Komma Wert, auf den vom Zeiger in alle vier Komponenten eines Vektors verwiesen wird.<br/>                                         |
| [**Xmvector Set**](/windows/win32/api/directxmath/nf-directxmath-xmvectorset)<br/>                             | Erstellt einen Vektor mithilfe von vier Gleit Komma Werten.<br/>                                                                                    |
| [**Xmvector setbinaryconstant**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsetbinaryconstant)<br/> | Erstellt einen Vektor, dessen Komponenten entweder 0,0 f oder 1.0 f sind.<br/>                                                                    |
| [**Xmvector-tint**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsetint)<br/>                       | Erstellt einen Vektor mit ganzzahligen Komponenten ohne Vorzeichen.<br/>                                                                                    |
| [**Xmvector splatconstant**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatconstant)<br/>         | Erstellt einen Vektor mit identischen Gleit Komma Komponenten. Jede Komponente ist eine Konstante, dividiert durch zwei aufgelötete Werte in einen ganzzahligen Exponenten.<br/> |
| [**Xmvector splatconstantint**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatconstantint)<br/>   | Erstellt einen Vektor mit identischen ganzzahligen Komponenten.<br/>                                                                                   |
| [**Xmvector splatepsilon**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatepsilon)<br/>           | Gibt einen Vektor zurück, dessen Komponenten Epsilon sind (1.192092896 e-7).<br/>                                                              |
| [**Xmvector splatinfinity**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatinfinity)<br/>         | Gibt einen Vektor zurück, dessen Komponenten unendlich (0x7f 800000) sind.<br/>                                                                 |
| [**Xmvector page**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatone)<br/>                   | Gibt einen Vektor zurück, dessen Komponenten eins sind.<br/>                                                                                   |
| [**Xmvector splatqnan**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatqnan)<br/>                 | Gibt einen Vektor zurück, dessen Komponenten QNAN (0x7cf00000) sind.<br/>                                                                     |
| [**Xmvector splatsignmask**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatsignmask)<br/>         | Gibt einen Vektor zurück, dessen Komponenten die Zeichen Maske (0x80000000) sind.<br/>                                                            |
| [**Xmvector trueint**](/windows/win32/api/directxmath/nf-directxmath-xmvectortrueint)<br/>                     | Gibt einen Vektor zurück, dessen Komponenten true (0xFFFFFFFF) darstellen.<br/>                                                              |
| [**Xmvector Zero**](/windows/win32/api/directxmath/nf-directxmath-xmvectorzero)<br/>                           | Erstellt den Vektor NULL.<br/>                                                                                                              |
