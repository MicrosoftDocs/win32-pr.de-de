# <a name="xmubyten2-extensions"></a>XMUBYTEN2-Erweiterungen

[**XMUBYTEN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmubyten2) beschreibt einen 2D-Vektor zum Speichern nicht signierter, normalisierter Werte als 8-Bit-Ganzzahlen (1 Byte) ohne Vorzeichen.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
