# <a name="xmubyte2-extensions"></a>XMUBYTE2-Erweiterungen

Beschreibt einen 2D-Vektor, der aus ganzen Zahlen ohne Vorzeichen und einer Länge von 8 Bits (1 Byte) besteht.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
