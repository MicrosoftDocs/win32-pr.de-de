# <a name="xmu555-operators"></a>XMU555-Operatoren

Die [**XMU555-Struktur**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu555) macht die folgenden Operatoren verfügbar.

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                           | BESCHREIBUNG                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operator uint16 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmu555-operatoruint16_t)<br/> | Gibt eine Instanz von `uint16_t` zurück, die die Komponenten der `XMU555` Instanz in einem gepackten Format enthält. <br/> |
| [**operator =**](xmu555-operator-eq.md)<br/>               | Weist der aktuellen Instanz von Vektorkomponentendaten `XMU555` zu. <br/>                                       |
