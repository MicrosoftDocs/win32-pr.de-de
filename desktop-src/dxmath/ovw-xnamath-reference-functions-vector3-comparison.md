# <a name="directxmath-library-3d-vector-comparison-functions"></a>3D-Vektor Vergleichsfunktionen der directxmath-Bibliothek

Listet die 3D-Vektor Vergleichsfunktionen auf.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                                                   | BESCHREIBUNG                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3Equal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equal)<br/>                     | Testet, ob zwei 3D-Vektoren gleich sind.<br/>                                                                                                                                                                                                        |
| [**XMVector3EqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equalint)<br/>               | Testet, ob zwei 3D-Vektoren gleich sind, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird.<br/>                                                                                                                                                        |
| [**XMVector3EqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equalintr)<br/>             | Testet, ob zwei 3D-Vektoren gleich sind, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird. Außerdem gibt diese Funktion einen Vergleichswert zurück, der mithilfe von Funktionen wie [**xmcomparisonalltrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)überprüft werden kann.<br/> |
| [**XMVector3EqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equalr)<br/>                   | Testet, ob zwei 3D-Vektoren gleich sind. Außerdem gibt diese Funktion einen Vergleichswert zurück, der mithilfe von Funktionen wie [**xmcomparisonalltrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)überprüft werden kann.<br/>                                                 |
| [**XMVector3Greater**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greater)<br/>                 | Testet, ob ein 3D-Vektor größer als ein anderer 3D-Vektor ist.<br/>                                                                                                                                                                                 |
| [**XMVector3GreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greaterorequal)<br/>   | Testet, ob ein 3D-Vektor größer oder gleich einem anderen 3D-Vektor ist.<br/>                                                                                                                                                                     |
| [**XMVector3GreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greaterorequalr)<br/> | Testet, ob ein 3D-Vektor größer oder gleich einem anderen 3D-Vektor ist, und gibt einen Vergleichswert zurück, der mithilfe von Funktionen wie [**xmcomparisonalltrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)überprüft werden kann.<br/>                                      |
| [**XMVector3GreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greaterr)<br/>               | Testet, ob ein 3D-Vektor größer als ein anderer 3D-Vektor ist, und gibt einen Vergleichswert zurück, der mithilfe von Funktionen wie [**xmcomparisonalltrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)überprüft werden kann.<br/>                                                  |
| [**XMVector3IsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmvector3isinfinite)<br/>           | Testet, ob eine Komponente eines 3D-Vektors positiv oder minus unendlich ist.<br/>                                                                                                                                                                   |
| [**XMVector3IsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvector3isnan)<br/>                     | Testet, ob eine Komponente eines 3D-Vektors ein NaN ist.<br/>                                                                                                                                                                                           |
| [**XMVector3Less**](/windows/win32/api/directxmath/nf-directxmath-xmvector3less)<br/>                       | Testet, ob ein 3D-Vektor kleiner als ein anderer 3D-Vektor ist.<br/>                                                                                                                                                                                    |
| [**XMVector3LessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lessorequal)<br/>         | Testet, ob ein 3D-Vektor kleiner oder gleich einem anderen 3D-Vektor ist.<br/>                                                                                                                                                                        |
| [**XMVector3NearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3nearequal)<br/>             | Testet, ob ein 3D-Vektor fast einem anderen 3D-Vektor ähnelt.<br/>                                                                                                                                                                                         |
| [**XMVector3NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3notequal)<br/>               | Testet, ob zwei 3D-Vektoren nicht gleich sind.<br/>                                                                                                                                                                                                    |
| [**XMVector3NotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector3notequalint)<br/>         | Testet, ob zwei 3D-Vektoren nicht gleich sind, wobei jede Komponente als ganze Zahl ohne Vorzeichen behandelt wird.<br/>                                                                                                                                                     |
