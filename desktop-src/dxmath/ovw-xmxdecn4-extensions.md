# <a name="xmxdecn4-extensions"></a>XMXDECN4-Erweiterungen

[**XMXDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdecn4) definieren Sie einen 4D-Vektor zum Speichern signierter, normalisierter Werte als 10-Bit-x-, y-und z-Komponenten und nicht signierte, normalisierte Werte als 2-Bit-w-Komponente ohne Vorzeichen.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
