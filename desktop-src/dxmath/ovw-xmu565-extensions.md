# <a name="xmu565-extensions"></a>XMU565-Erweiterungen

[**XMU565**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu565) beschreibt einen 4D-Vektor mit 5-Bit-x-und z-Komponenten ohne Vorzeichen und eine 6-Bit-y-Komponente ohne Vorzeichen.

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
