# <a name="xmbyten4-extensions"></a>XMBYTEN4-Erweiterungen

[**XMBYTEN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmbyten4) beschreibt einen 3D-Vektor zum Speichern von normalisierten Werten mit Vorzeichen als ganze 8-Bit-Zahlen (1 Byte).

> [!Note]  
> Die hier aufgeführten Features sind nur bei der Entwicklung mit C++ verfügbar.
