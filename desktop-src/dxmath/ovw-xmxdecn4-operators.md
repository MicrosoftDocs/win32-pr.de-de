# <a name="xmxdecn4-operators"></a>XMXDECN4-Operatoren

Die [**XMXDECN4-Struktur**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdecn4) macht die folgenden Operatoren verfügbar.

> [!Note]  
> Die hier aufgeführten Operatoren sind nur unter C++ verfügbar.

 

## <a name="in-this-section"></a>In diesem Abschnitt



| Methoden                                                             | BESCHREIBUNG                                                                                                            |
|---------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**Operator uint32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmxdecn4-operatoruint32_t)<br/> | Gibt eine Instanz von `uint32_t` zurück, die die Komponenten der `XMXDECN4` Instanz in einem gepackten Format enthält. <br/> |
| [**operator =**](xmxdecn4-operator-eq.md)<br/>               | Weist der aktuellen Instanz von Vektorkomponentendaten `XMXDECN4` zu. <br/>                                       |
