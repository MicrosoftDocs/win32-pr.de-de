# <a name="deviceinfoport-property"></a>DeviceInfo. Port (Eigenschaft)

Ruft den Port ab, mit dem das Hardware Gerät für die Windows-Abbild Beschaffung (WIA) verbunden ist.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = DeviceInfo.Port
```



## <a name="property-value"></a>Eigenschaftswert

Zeichenfolge, die den Port empfängt.
