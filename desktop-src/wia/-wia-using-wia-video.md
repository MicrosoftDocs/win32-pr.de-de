# <a name="using-wia-video"></a>Verwenden von WIA-Videos

WIA stellt die [**iwiavideo**](/windows/desktop/api/Wiavideo/nn-wiavideo-iwiavideo) -Schnittstelle bereit, damit Anwendungen Streaming-Videos in einem Fenster anzeigen, Bilder aus dem Videostream aufzeichnen und diese Bilder als JPEG-Dateien speichern können.

In den folgenden Abschnitten wird erläutert, wie Videofunktionen für die Windows-Abbild Beschaffung (WIA) verwendet werden:

-   [Anzeigen einer Vorschau für Videos](-wia-previewing-video.md)
-   [Erfassen eines Images aus einem Videostream](-wia-capturing-a-still-image-from-a-video-stream.md)

> [!Note]  
> WIA unterstützt keine Videogeräte in Windows Server 2003, Windows Vista oder höher. Verwenden Sie für diese Versionen von Windows [DirectShow](/previous-versions//ms783323(v=vs.85)) zum Abrufen von Bildern aus Videos.
