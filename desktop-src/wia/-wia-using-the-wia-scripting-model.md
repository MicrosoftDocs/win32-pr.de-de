# <a name="using-the-wia-scripting-model"></a>Verwenden des WIA-Skriptmodells

Über das [WIA-Skriptmodell](-wia-wia-scripting-model.md)werden Windows Image Acquisition-Funktionen (WIA) Skriptsprachen wie Microsoft JScript-Entwicklungssoftware und Microsoft Visual Basic Scripting Edition (VBScript) sowie übergeordneten Sprachen wie Microsoft Visual Basic-Entwicklungssystem zur Verfügung gestellt.

> [!Note]  
> Dieses Skriptsystem wurde durch Windows Image Acquisition (WIA) Automation Layer ersetzt. Weitere Informationen finden Sie [unter Windows Image Acquisition Automation Layer](/previous-versions/windows/desktop/wiaaut/-wiaaut-startpage).

 

Das WIA-Skriptmodell macht Objekte verfügbar, die die [IDispatch-Schnittstelle](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-idispatch) erben.

Das Skriptmodell bietet eine vereinfachte Version der WIAAPI.
