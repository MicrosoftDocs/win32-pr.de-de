# <a name="itempicturewidth-property"></a>Item. picturewidth-Eigenschaft

Die Breite von Bildern, die von dieser digitalen Kamera erzeugt werden, in Pixel. Gilt nur für digitale Kameras. Schreibgeschützt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Item.PictureWidth
```



## <a name="property-value"></a>Eigenschaftswert

Eine Variable, die die Breite empfängt.
