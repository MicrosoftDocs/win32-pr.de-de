# <a name="itemwidth-property"></a>Item. Width-Eigenschaft

Die Breite des Elements in Pixel. Schreibgeschützt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Item.Width
```



## <a name="property-value"></a>Eigenschaftswert

Eine Variable, die die Breite empfängt.

## <a name="remarks"></a>Bemerkungen

Wenn es sich bei dem Element um eine digitale Kamera handelt, stellt die **Width** -Eigenschaft die Breite der von der Kamera generierten Bilder dar. Wenn es sich bei dem Element um einen Scanner handelt, stellt diese Eigenschaft die Breite des Scannens dar. Wenn das Element ein Bild ist, stellt diese Eigenschaft die tatsächliche Breite des Bilds dar.
