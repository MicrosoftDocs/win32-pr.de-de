# <a name="itemthumbnail-property"></a>Item. Miniaturansicht (Eigenschaft)

Der Pfad und der Dateiname des Miniatur Bilds. Diese Eigenschaft ist **null** , wenn das Element keine Miniaturansichten unterstützt, oder wenn kein Pfad erstellt werden kann. Schreibgeschützt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Item.Thumbnail
```



## <a name="property-value"></a>Eigenschaftswert

Zeichenfolge, die den Pfad und den Dateinamen empfängt.
