# <a name="syntax-conventions"></a>Syntaxkonventionen

In der folgenden Tabelle sind die Begriffe aufgeführt, die in Syntax Anweisungen für die Skript Modell Referenz für die Windows-Image Erfassung (WIA) verwendet werden. 

| Begriff                      | BESCHREIBUNG                                                    |
|---------------------------|----------------------------------------------------------------|
| *Filename*                | Der Pfad und der Name einer Datei.                                   |
| *objdebug-Info*           | Ein [**Debug**](-wia-deviceinfo.md) -Objekt.                |
| *objde viceinfocollection* | Eine [**Auflistung von "**](-wia-deviceinfo.md) Debug"-Objekten. |
| *objItem*                 | Ein [**Item**](-wia-item.md) -Objekt.                           |
| *objitemcollection*       | Eine Auflistung von [**Item**](-wia-item.md) -Objekten.             |
| *objwia*                  | Ein [**WIA**](-wia-wia.md) -Objekt.                              |
| *PROPID*                  | Eine WIA-Eigenschafts Konstante.                                       |
| *var*                     | Ein **Variant** -Wert.                                           |
