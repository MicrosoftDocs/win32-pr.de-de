# <a name="itemfirmwareversion-property"></a>Item. Firmwareversion-Eigenschaft

Ruft die Firmwareversion des Geräts ab. Diese Eigenschaft gilt nur für Elemente vom Typ Device (root Items). Schreibgeschützt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Item.FirmwareVersion
```



## <a name="property-value"></a>Eigenschaftswert

Zeichenfolge, die die Firmwareversion empfängt.
