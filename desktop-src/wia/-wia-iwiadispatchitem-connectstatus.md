# <a name="itemconnectstatus-property"></a>Item. connectstatus (Eigenschaft)

Ruft den Verbindungsstatus des Geräts ab. Diese Eigenschaft gilt nur für Elemente vom Typ Device (root Items). Mögliche Werte sind "verbunden", "getrennt" oder **null** (wenn diese Eigenschaft nicht für das Element gilt). Schreibgeschützt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Item.ConnectStatus
```



## <a name="property-value"></a>Eigenschaftswert

Eine Zeichenfolge, die den Verbindungsstatus empfängt.
