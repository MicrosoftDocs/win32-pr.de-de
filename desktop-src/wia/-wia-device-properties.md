# <a name="device-properties-wia"></a>Geräteeigenschaften (WIA)

Geräteeigenschaften sind Eigenschaften, die die Funktion des gesamten Geräts steuern. Auf diese Eigenschaften kann nur über das Stammelement der Struktur zugegriffen werden, das ein Gerät darstellt. Es gibt drei Arten von Geräteeigenschaften: common, scanner und camera.

Referenzinformationen finden Sie unter:

-   [**Allgemeine Geräteeigenschaftskonst constants**](-wia-wiaitempropcommondevice.md)
-   [**Eigenschaftenkonst constants des Kamerageräts**](-wia-wiaitempropcameradevice.md)
-   [**Eigenschaftenkonst constants des Scannergeräts**](-wia-wiaitempropscannerdevice.md)
