# <a name="device-properties-wia"></a>Geräteeigenschaften (WIA)

Geräteeigenschaften sind Eigenschaften, die die Funktion des gesamten Geräts steuern. Auf diese Eigenschaften kann nur über das Stamm Element der Struktur zugegriffen werden, die ein Gerät darstellt. Es gibt drei Arten von Geräteeigenschaften: Allgemein, Scanner und Kamera.

Referenzinformationen finden Sie unter:

-   [**Allgemeine Geräte Eigenschafts Konstanten**](-wia-wiaitempropcommondevice.md)
-   [**Eigenschaften Konstanten für Kamerageräte**](-wia-wiaitempropcameradevice.md)
-   [**Eigenschaften Konstanten für Scanner-Geräte**](-wia-wiaitempropscannerdevice.md)
