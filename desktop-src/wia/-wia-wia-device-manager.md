# <a name="wia-device-manager"></a>WIA-Geräte-Manager

Der Windows Image Acquisition (WIA) Geräte-Manager ist eine Erweiterung des STI-Ereignismonitors (Still Image). Die WIA-Geräte-Manager stellt Objekte, Methoden und Schnittstellen für Folgendes bereit:

-   Installieren von Geräten
-   Enumerieren von Geräten
-   Abfragen von Eigenschaften installierter Geräte
-   Erstellen von Geräteobjekten
-   Überwachen von Geräteereignissen
-   Abrufen von Bildern
-   Registrieren von Zielanwendungen.
