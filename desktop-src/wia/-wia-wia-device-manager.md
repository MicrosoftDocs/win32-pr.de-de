# <a name="wia-device-manager"></a>WIA-Device Manager

Der Windows-Abbild Erfassungs Device Manager (WIA) ist eine Erweiterung des nach wie vor Abbild-Ereignis Monitors (STI). Der WIA-Device Manager stellt-Objekte,-Methoden und-Schnittstellen für Folgendes bereit:

-   Installieren von Geräten
-   Enumerieren von Geräten
-   Abfragen der Eigenschaften installierter Geräte
-   Erstellen von Geräte Objekten
-   Überwachen von Geräte Ereignissen
-   Abrufen von Images
-   Zielanwendungen werden registriert.
