# <a name="camera-wia-item-property-constants"></a>WIA-Elementeigenschaftskonstanten der Kamera

Die  folgenden Wortkonstanten bilden den Satz gültiger Windows Wia-Kameraelementeigenschaften (Image Acquisition).

> [!Note]  
> WIA unterstützt keine Kameras in Windows Vista oder höher. Verwenden Sie für diese Versionen der Windows die im Windows Driver Development Kit (DDK) beschriebene WPD-API (Windows Portable Device), um Bilder von Kameras zu erhalten.

 

Das Präfix "WIA \_ \_ IPC" gibt eine Elementeigenschaft für Kameras an und ist die in C/C++ verwendete Namenskonvention.
