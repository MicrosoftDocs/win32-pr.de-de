# <a name="collectioncount-property"></a>Collection.Count-Eigenschaft

Gibt die Anzahl der Elemente in der Auflistung zurück.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Collection.Count
```
