# <a name="itemfullname-property"></a>Item. FullName-Eigenschaft

Ruft den vollständigen Namen des Elements ab, das in der Benutzeroberfläche angezeigt wird. Schreibgeschützt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Item.FullName
```



## <a name="property-value"></a>Eigenschaftswert

Zeichenfolge, die den Namen empfängt.
