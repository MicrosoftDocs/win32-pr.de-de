# <a name="wia-device-type-specifiers"></a>WIA-Gerätetyp spezifiker

Windows-Abbild Erfassungsgeräte (WIA) sind Konstanten, die den Typ des Geräts angeben, das dem Computer eines Benutzers zugeordnet ist. Verwenden Sie das \_ Makro Get stidevice \_ Type, um diese Werte aus der Eigenschaft WIA-Gerätetyp abzurufen. Im folgenden sind gültige WIA-Gerätetyp spezifiatoren aufgeführt: 

| Gerätetyp                 | Bedeutung                                                                                                                     |
|-----------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| Stide vicetypeer default        | Generisches WIA-Gerät. Während der Geräte Enumerationen wird diese Konstante zum Auflisten aller WIA-Geräte verwendet.                         |
| Stide vicetyetdigitalcamera  | Das Gerät ist eine Kamera. *Dieser Spezifizierer wird von nicht unterstützt* . Windows Vista *und höher.*                                     |
| Stide vicetypescanner        | Das Gerät ist ein Scanner.                                                                                                    |
| Stientvicetyetstreamingvideo | Das Gerät enthält Streaming-Videos. *Dieser Spezifizierer wird von nicht unterstützt* . Windows Server 2003 *,* Windows Vista *oder höher.* |
