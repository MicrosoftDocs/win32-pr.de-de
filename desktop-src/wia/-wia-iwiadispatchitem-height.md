# <a name="itemheight-property"></a>Item. Height-Eigenschaft

Die Höhe des Elements in Pixel. Schreibgeschützt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Item.Height
```



## <a name="property-value"></a>Eigenschaftswert

Eine Variable, die die Höhe empfängt.

## <a name="remarks"></a>Bemerkungen

Wenn das Element eine digitale Kamera ist, stellt die **height** -Eigenschaft die Höhe der Bilder dar, die von dieser Kamera generiert werden. Wenn es sich bei dem Element um einen Scanner handelt, stellt diese Eigenschaft die Höhe des Scannens dar. Wenn das Element ein Bild ist, stellt diese Eigenschaft die tatsächliche Höhe des Bilds dar.
