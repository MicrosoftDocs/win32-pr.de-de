# <a name="itemheight-property"></a>Item.Height-Eigenschaft

Die Höhe des Elements in Pixel. Schreibgeschützt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Item.Height
```



## <a name="property-value"></a>Eigenschaftswert

Variable, die die Höhe empfängt.

## <a name="remarks"></a>Hinweise

Wenn es sich bei dem Element um eine Digitalkamera handelt, stellt die **Height-Eigenschaft** die Höhe der Bilder dar, die von dieser Kamera generiert werden. Wenn es sich bei dem Element um einen Scanner handelt, stellt diese Eigenschaft die Höhe des Scanbetts dar. Wenn das Element ein Bild ist, stellt diese Eigenschaft die tatsächliche Höhe des Bilds dar.
