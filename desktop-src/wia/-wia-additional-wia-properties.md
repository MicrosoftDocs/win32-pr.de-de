# <a name="additional-wia-properties"></a>Zusätzliche WIA-Eigenschaften

WIA-Geräte unterstützen abhängig von der Hardwarekonfiguration möglicherweise zusätzliche Elementeigenschaften. Nicht unterstützte Eigenschaften weisen den VARTYPE VT \_ EMPTY auf.
