# <a name="wia-property-constants"></a>WIA-Eigenschafts Konstanten

Die Eigenschaften Konstanten der Windows-Image Erfassung (WIA) enthalten die folgenden Gruppen von Konstanten, die von der Windows Millennium Edition (Windows Me), Windows XP und höher unterstützt werden.

-   [**Eigenschaften Konstanten für Geräteinformationen**](-wia-wiadeviceinfoprop.md)
-   [**Allgemeine Geräte Eigenschafts Konstanten**](-wia-wiaitempropcommondevice.md)
-   [**Eigenschaften Konstanten für Scanner-Geräte**](-wia-wiaitempropscannerdevice.md)
-   [**Allgemeine WIA-Element Eigenschaften Konstanten**](-wia-wiaitempropcommonitem.md)
-   [**Eigenschaften Konstanten für Scanner-WIA-Elemente**](-wia-wiaitempropscanneritem.md)

Die folgenden Gruppen von Konstanten werden von Windows Me, Windows XP und Windows Server 2003 unterstützt. Diese werden jedoch von Windows Vista und höher nicht unterstützt.

-   [**Eigenschaften Konstanten für Kamerageräte**](-wia-wiaitempropcameradevice.md)
-   [**Eigenschaften Konstanten für Kamera-WIA-Elemente**](-wia-wiaitempropcameraitem.md)

Die folgende Gruppe von Konstanten wird von Windows Me und Windows XP unterstützt. wird jedoch von Windows Server 2003 und höher nicht unterstützt.

-   [**Video-WIA-Geräte Eigenschafts Konstanten**](-wia-wiaitempropvideodevice.md)
