# <a name="item-properties"></a>Elementeigenschaften

Element Eigenschaften sind Eigenschaften, die die Funktion eines Windows-Abbild Erwerbs (WIA)-Elements steuern. Der Zugriff auf diese Eigenschaften erfolgt über ein beliebiges Element in einer Struktur, die ein Gerät darstellt. Es gibt drei Arten von Element Eigenschaften: Common, Scanner und Kamera.

Referenzinformationen zu diesen Eigenschaften finden Sie unter:

-   [**Allgemeine WIA-Element Eigenschaften Konstanten**](-wia-wiaitempropcommonitem.md)
-   [**Eigenschaften Konstanten für Kamera-WIA-Elemente**](-wia-wiaitempropcameraitem.md)
-   [**Eigenschaften Konstanten für Scanner-WIA-Elemente**](-wia-wiaitempropscanneritem.md)
