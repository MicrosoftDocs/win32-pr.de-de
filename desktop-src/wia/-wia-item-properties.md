# <a name="item-properties"></a>Elementeigenschaften

Elementeigenschaften sind Eigenschaften, die die Funktion eines WIA-Elements (Windows Image Acquisition) steuern. Auf diese Eigenschaften wird über jedes Element in einer Struktur zugegriffen, die ein Gerät darstellt. Es gibt drei Arten von Elementeigenschaften: common, scanner und camera.

Referenzinformationen zu diesen Eigenschaften finden Sie unter:

-   [**Allgemeine WIA-Elementeigenschaftskonst constants**](-wia-wiaitempropcommonitem.md)
-   [**WIA-Elementeigenschaftskonst constants der Kamera**](-wia-wiaitempropcameraitem.md)
-   [**WiA-Elementeigenschaftskonst constants (WIA-Elementeigenschaftskonst constants) für**](-wia-wiaitempropscanneritem.md)
