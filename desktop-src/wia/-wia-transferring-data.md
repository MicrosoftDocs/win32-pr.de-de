# <a name="transferring-data"></a>Übertragen von Daten

> [!Note]  
> Dieses Skriptsystem wurde durch Windows Image Acquisition (WIA) Automation Layer ersetzt. Weitere Informationen finden Sie [unter Windows Image Acquisition Automation Layer](/previous-versions/windows/desktop/wiaaut/-wiaaut-startpage).

 

Verwenden Sie die [**Transfer-Methode**](-wia-iwiadispatchitem-transfer.md) eines [**Item-Objekts,**](-wia-item.md) um Bild- oder Audiodaten von einem Gerät in eine Datei oder zwischenablage zu übertragen.

Übergeben Sie einen Pfad und dateinamen als ersten Parameter, um die Daten auf dem Datenträger zu speichern, oder übergeben Sie die Zeichenfolge "Zwischenablage", um das Element in die Zwischenablage zu übertragen.
