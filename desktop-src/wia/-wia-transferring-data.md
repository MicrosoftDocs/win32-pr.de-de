# <a name="transferring-data"></a>Übertragen von Daten

> [!Note]  
> Dieses Skript System wurde durch die Windows-Abbild Erwerbs-Automatisierungs Schicht (WIA) ersetzt. Weitere Informationen finden Sie unter [Automatisierungs Schicht für Windows-Abbild](/previous-versions/windows/desktop/wiaaut/-wiaaut-startpage)

 

Verwenden Sie die [**Übertragungs**](-wia-iwiadispatchitem-transfer.md) Methode eines [**Item**](-wia-item.md) -Objekts, um Bild-oder Audiodaten von einem Gerät in eine Datei oder die Zwischenablage zu übertragen.

Übergeben Sie einen Pfad und Dateinamen als ersten Parameter, um die Daten auf dem Datenträger zu speichern, oder übergeben Sie die Zeichenfolge "Clipboard", um das Element in die Zwischenablage zu übertragen
