# <a name="deviceinfoname-property"></a>DeviceInfo.Name-Eigenschaft

Der Name des Hardware Geräts für die Windows-Abbild Beschaffung (WIA), wie es in der Benutzeroberfläche angezeigt wird.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = DeviceInfo.Name
```



## <a name="property-value"></a>Eigenschaftswert

Zeichenfolge, die den Namen empfängt.
