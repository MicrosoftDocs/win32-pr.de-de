# <a name="deviceinfoname-property"></a>DeviceInfo.Name-Eigenschaft

Der Name des WIA-Hardwaregeräts (Windows Image Acquisition), wie es auf der Benutzeroberfläche angezeigt wird.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = DeviceInfo.Name
```



## <a name="property-value"></a>Eigenschaftswert

Zeichenfolge, die den Namen empfängt.
