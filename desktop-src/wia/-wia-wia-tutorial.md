# <a name="wia-tutorial"></a>WIA-Tutorial

Dieses Tutorial enthält Schritt-für-Schritt-Anleitungen für die Verwendung der Windows-Abbild Beschaffung (WIA) in realen Anwendungen.

Das Tutorial zeigt Code aus den Beispielanwendungen, die im SDK enthalten sind. Es soll veranschaulichen, wie Entwickler die Hauptfunktionen von WIA implementieren können und es sich nicht um eine Blaupause zum Erstellen einer kompletten Anwendung handelt.

Das Tutorial konzentriert sich auf die folgenden Aufgaben:

-   [Erstellen eines WIA-Device Manager](-wia-creating-a-wia-device-manager.md)
-   [Auflisten von System Geräten](-wia-enumerating-system-devices.md)
-   [Lesen von Geräteeigenschaften](-wia-reading-device-properties.md)
-   [Erstellen eines Geräts](-wia-creating-a-device.md)
-   [Auflisten von Elementen](-wia-enumerating-items.md)
-   [Übertragen von Bilddaten](-wia-transferring-image-data.md)
-   [Registrieren für Ereignisse](-wia-registering-for-events.md)
-   [Erfassen eines Images vom Streaming-Video](-wia-capturing-a-still-image-from-streaming-video.md)
