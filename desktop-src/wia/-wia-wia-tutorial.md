# <a name="wia-tutorial"></a>WIA-Tutorial

Dieses Tutorial enthält schritt-für-Schritt-Anweisungen zur Verwendung Windows Image Acquisition (WIA) in realen Anwendungen.

Das Tutorial enthält Code aus den Beispielanwendungen, die im SDK enthalten sind. Es soll veranschaulichen, wie Entwickler die Hauptfunktionen von WIA implementieren können, und ist keine Blaupause zum Erstellen einer vollständigen Anwendung.

-   [Lesen von Geräteeigenschaften](-wia-reading-device-properties.md)
-   [Erstellen eines Geräts](-wia-creating-a-device.md)
-   [Aufzählen von Elementen](-wia-enumerating-items.md)
-   [Übertragen von Bilddaten](-wia-transferring-image-data.md)
-   [Registrieren für Ereignisse](-wia-registering-for-events.md)
-   [Erfassen eines Still-Bilds aus dem Streamingvideo](-wia-capturing-a-still-image-from-streaming-video.md)
