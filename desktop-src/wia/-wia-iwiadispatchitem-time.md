# <a name="itemtime-property"></a>Item. Time-Eigenschaft

Die aktuelle Zeit. Gilt nur für Geräte. Schreibgeschützt.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = Item.Time
```
