# <a name="deviceinfomanufacturer-property"></a>Eigenschaft "de viceingefo. Manufacturer"

Ruft den Namen des Herstellers dieses Hardware Geräts ab.

Diese Eigenschaft ist schreibgeschützt.

## <a name="syntax"></a>Syntax


```JScript
propVal = DeviceInfo.Manufacturer
```



## <a name="property-value"></a>Eigenschaftswert

Zeichenfolge, die den Namen empfängt.
