# <a name="nps-extensions-structures"></a>NPS-Erweiterungsstrukturen

> [!Note]  
> Internet Authentication Service (IAS) wurde ab Windows Server 2008 in Network Policy Server (NPS) umbenannt. Der Inhalt dieses Themas gilt sowohl für IAS als auch für NPS.

 

Verwenden Sie [**die RADIUS \_ ATTRIBUTE-Strukturen,**](/windows/desktop/api/authif/ns-authif-radius_attribute) um beim Entwickeln von NPS-Erweiterungs-DLLs ein RADIUS-Attribut oder ein erweitertes Attribut zu darstellen.

-   [**\_RADIUS-ATTRIBUT**](/windows/desktop/api/authif/ns-authif-radius_attribute)
-   [**\_ \_ RADIUS-ATTRIBUTARRAY**](/windows/desktop/api/authif/ns-authif-radius_attribute_array)
-   [**RADIUS \_ EXTENSION \_ CONTROL \_ BLOCK**](/windows/desktop/api/authif/ns-authif-radius_extension_control_block)
-   [**\_RADIUS-VSA-FORMAT \_**](/windows/desktop/api/authif/ns-authif-radius_vsa_format)
