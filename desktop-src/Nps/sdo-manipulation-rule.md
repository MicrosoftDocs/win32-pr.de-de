# <a name="manipulation-rule"></a>Manipulation-Rule

> [!Note]  
> Der Internet Authentifizierungsdienst (IAS) wurde ab Windows Server 2008 in den Netzwerk Richtlinien Server (Network Policy Server, NPS) umbenannt.

 

Die Manipulations Regel, die auf die Daten angewendet werden soll, die durch das [Manipulations Ziel](/windows/desktop/Nps/sdo-manipulation-target) Attribut angegeben werden. Weitere Informationen finden Sie unter [Bereiche Verarbeitung und Attribut Bearbeitung](/windows/desktop/Nps/sdo-realms-processing-and-attribute-manipulation).

**LDAP-Name:** msmanipulationrule

**ID:** 4114 = [ **IAS- \_ Attribut \_ Bearbeitungs \_ Regel**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Syntax:** Schnür
