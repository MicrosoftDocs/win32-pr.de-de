# <a name="np-allowed-port-types"></a>NP-allowed-Port-Typen

Porttypen, die für eine Verbindung zulässig sind.

**LDAP-Name:** msnpallowedporttypes

**ID:** 4104 = [ **IAS \_ - \_ Attribut \_ NP \_ zulässige \_ Porttypen**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Syntax:** Enumerator

**Werte:  **

Definiert von der Internet Assigned Numbers Authority (IANA) im [RADIUS-Typen](https://www.iana.org/assignments/radius-types) Dokument, basierend auf [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt).

-   Async (Modem) (0)
-   Sync (T1-Zeile) (1)
-   ISDN-Synchronisierung (2)
-   ISDN Async V. 120 (3)
-   ISDN Async V. 110 (4)
-   Virtuell (VPN) (5)
-   Piafs (6)
-   HDLC-Clear-Channel (7)
-   X. 25 (8)
-   X. 75 (9)
-   G. 3 Fax (10)
-   SDSL-symmetrische DSL (11)
-   ADSL-Cap-asymmetrische DSL carrierless Amplitude-Phasen Modulation (12)
-   ADSL-DMT-asymmetrische DSL-Multiton (13)
-   IDSL-ISDN Digital Subscriber Line (14)
-   Ethernet (15)
-   xDSL-digitale Abonnenten Zeile unbekannter Typen (16)
-   Kabel (17)
-   Drahtlos drahtlos (18)
-   Wireless-IEEE 802,11 (19)
-   Tokenring (20)
-   F-di (21)
