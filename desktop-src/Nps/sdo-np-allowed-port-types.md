# <a name="np-allowed-port-types"></a>NP-Allowed-Port-Types

Porttypen, die für eine Verbindung zulässig sind.

**LDAP-Name:** msNPAllowedPortTypes

**ID:** 4104 = [ **\_ IAS-ATTRIBUT NP ZULÄSSIGE \_ \_ \_ \_ PORTTYPEN**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Syntax:** Enumerator

**Werte: **

Wird von der Internet Assigned Numbers Authority (IANA) im [RADIUS TYPES-Dokument](https://www.iana.org/assignments/radius-types) basierend auf [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt)definiert.

-   Async (Modem) (0)
-   Sync (T1 Line) (1)
-   ISDN-Synchronisierung (2)
-   ISDN Async V.120 (3)
-   ISDN Async V.110 (4)
-   Virtual (VPN) (5)
-   PIAFS (6)
-   HDLC Clear Channel (7)
-   X.25 (8)
-   X.75 (9)
-   G.3 Fax (10)
-   SDSL – Symmetrische DSL (11)
-   ADSL-CAP – Asymmetrische DSL– Stufenlose Amplitudenphase (12)
-   ADSL-DMT – Asymmetrische DSL Diskreter Multiton (13)
-   IDSL – ISDN Digital Subscriber Line (14)
-   Ethernet (15)
-   xDSL – Digitale Abonnentenzeile mit unbekanntem Typ (16)
-   Kabel (17)
-   Drahtlos – Sonstiges (18)
-   Drahtlos – IEEE 802.11 (19)
-   Tokenring (20)
-   FDDI (21)
