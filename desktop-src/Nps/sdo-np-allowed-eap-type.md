# <a name="np-allowed-eap-type"></a>NP-Allowed-EAP-Type

EAP-Verschlüsselungsmodi, die für eine Verbindung zulässig sind.

**LDAP-Name:** msNPAllowedEapType

**ID:** 4106 = [ **IAS ATTRIBUTE NP \_ ALLOWED \_ \_ \_ EAP \_ TYPE**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Syntax:** Ganzzahl

**Werte: **

Wird von der Internet Assigned Numbers Authority (IANA) im Dokument [Extensible Authentication Protocol (EAP) Registry](https://www.iana.org/assignments/eap-numbers) basierend auf [RFC 3748](https://www.ietf.org/rfc/rfc3748.txt)definiert.

-   **MD5-Challenge** (4)
-   **EAP \_ TLS** (13)
-   **PEAP** (25)
-   **MS-EAP-Authentifizierung** (26)
