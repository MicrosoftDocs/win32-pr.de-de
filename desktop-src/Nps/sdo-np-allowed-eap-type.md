# <a name="np-allowed-eap-type"></a>NP-Allowed-EAP-Typ

Für eine Verbindung zugelassene EAP-Verschlüsselungs Modi.

**LDAP-Name:** msnpallowedeaptype

**ID:** 4106 = [ **IAS- \_ Attribut \_ NP der \_ \_ EAP- \_ Typ ist zulässig** .](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Syntax:** Zah

**Werte:  **

Definiert von der Internet Assigned Numbers Authority (IANA) im [Registrierungsdokument Extensible Authentication Protocol (EAP)](https://www.iana.org/assignments/eap-numbers) , basierend auf [RFC 3748](https://www.ietf.org/rfc/rfc3748.txt).

-   **MD5-Challenge** (4)
-   **EAP \_ TLS** (13)
-   **Peer-App** (25)
-   **MS-EAP-Authentifizierung** (26)
