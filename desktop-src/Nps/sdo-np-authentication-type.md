# <a name="np-authentication-type"></a>NP-Authentication-Type

Für eine Verbindung zulässige Authentifizierungstypen.

**LDAP-Name:** msNPAuthenticationType2

**ID:** 4105 = [ **\_ NF-ATTRIBUT \_ \_ NP-AUTHENTIFIZIERUNGSTYP \_**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Syntax:** Ganzzahl

**Werte: **

Definiert im [**\_ AUTHENTIFIZIERUNGSTYP-Enumerationstyp,**](/windows/desktop/api/SdoIas/ne-sdoias-authentication_type) der in der öffentlichen Headerdatei "SdoIas.h" deklariert ist.

-   **IAS \_ AUTH \_ INVALID** (0)
-   **IAS \_ AUTH \_ PAP** (1)
-   **IAS \_ AUTH \_ MD5CHAP** (2)
-   **IAS \_ AUTH \_ MSCHAP** (3)
-   **IAS \_ AUTH \_ MSCHAP2** (4)
-   **IAS \_ AUTH \_ EAP** (5)
-   **IAS \_ AUTH \_ ARAP** (6)
-   **IAS \_ AUTH \_ NONE** (7)
-   **IAS \_ AUTH \_ CUSTOM** (8)
-   **IAS \_ AUTH \_ MSCHAP \_ CPW** (9)
-   **IAS \_ AUTH \_ MSCHAP2 \_ CPW** (10)
-   **IAS \_ AUTH \_ PEAP** (11)
