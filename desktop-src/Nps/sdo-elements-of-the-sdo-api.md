# <a name="elements-of-the-sdo-api"></a>Elemente der SDO-API

Die SDO-API ist auf ein Objektmodell ausgelegt, das aus Objekten und Auflistungen von Objekten besteht. Im SDO-Objektmodell werden die Objekte und Auflistungen in einer Hierarchie angeordnet, d. h., Objekte und Auflistungen enthalten andere Objekte und Auflistungen.

Objekte werden durch Abrufen und Festlegen von Objekteigenschaften bearbeitet.

In den folgenden Abschnitten wird beschrieben, wie die SDO-API Objekte, Objekteigenschaften und Sammlungen verwendet:

-   [Objekte und Eigenschaften](/windows/desktop/Nps/sdo-objects-and-properties)
-   [Objektmodellhierarchie](/windows/desktop/Nps/sdo-object-model-hierarchy)
-   [Attribute](/windows/desktop/Nps/sdo-sdo-supported-attributes)
