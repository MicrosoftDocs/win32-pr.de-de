# <a name="elements-of-the-sdo-api"></a>Elemente der SDO-API

Die SDO-API ist um ein Objektmodell ausgelegt, das aus Objekten und Auflistungen von Objekten besteht. Im SDO-Objektmodell werden die Objekte und Auflistungen in einer Hierarchie angeordnet, d. h., Objekte und Auflistungen enthalten andere Objekte und Auflistungen.

Objekte werden bearbeitet, indem Objekteigenschaften abgerufen und festgelegt werden.

In den folgenden Abschnitten wird beschrieben, wie die SDO-API Objekte, Objekteigenschaften und Auflistungen verwendet:

-   [Objekte und Eigenschaften](/windows/desktop/Nps/sdo-objects-and-properties)
-   [Objektmodell Hierarchie](/windows/desktop/Nps/sdo-object-model-hierarchy)
-   [Attribute](/windows/desktop/Nps/sdo-sdo-supported-attributes)
