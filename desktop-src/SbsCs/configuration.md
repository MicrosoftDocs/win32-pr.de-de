# <a name="configuration-isolated-applications-and-side-by-side-assemblies"></a>Konfiguration (isolierte Anwendungen und parallele Assemblys)

Mithilfe von [Verleger Konfigurationsdateien](publisher-configuration-files.md) und [Anwendungs Konfigurationsdateien](application-configuration-files.md) können assemblerverleger, Anwendungs Herausgeber und Administratoren parallele Assemblyabhängigkeiten nach der Bereitstellung ändern.

Weitere Informationen zum Verwenden von Manifesten und Konfigurationsdateien zum Definieren der Abhängigkeit der parallelen Assemblyversion finden Sie in den folgenden Themen:

-   [Standardkonfiguration](default-configuration.md)
-   [Herausgeber Konfiguration](publisher-configuration.md)
-   [Konfiguration pro Anwendung](per-application-configuration.md)
-   [Konfiguration pro Anwendung unter Windows XP](per-application-configuration-on-windows-xp.md)
