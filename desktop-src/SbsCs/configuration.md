# <a name="configuration-isolated-applications-and-side-by-side-assemblies"></a>Konfiguration (Isolierte Anwendungen und nebenseitige Assemblys)

[Publisher Konfigurationsdateien](publisher-configuration-files.md) und [](application-configuration-files.md) Anwendungskonfigurationsdateien ermöglichen es Assemblyherausgebern, Anwendungsherausgebern und Administratoren, nach der Bereitstellung die Abhängigkeiten von assemblyseitigen Assemblys zu ändern.

Weitere Informationen zur Verwendung von Manifesten und Konfigurationsdateien zum Definieren der abhängigkeitsbasierten Assemblyversion finden Sie in den folgenden Themen:

-   [Standardkonfiguration](default-configuration.md)
-   [Publisher Konfiguration](publisher-configuration.md)
-   [Anwendungsspezifische Konfiguration](per-application-configuration.md)
-   [Anwendungsspezifische Konfiguration auf Windows XP](per-application-configuration-on-windows-xp.md)
