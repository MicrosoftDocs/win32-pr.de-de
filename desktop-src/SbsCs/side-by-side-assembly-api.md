# <a name="side-by-side-assembly-api"></a>Parallele Assembly-API

Die parallele Assembly-API installiert, entfernt oder fragt parallele Assemblys aus dem parallelen Assemblyspeicher des Systems ab. Dies wird auch als WinSxS-Store unter Windows XP und im Komponenten Speicher der Komponenten Wartungs Infrastruktur (CSI), beginnend mit Windows Vista und Windows Server 2008, bezeichnet.

Die Teile der parallelen assemblyapi werden in den folgenden Abschnitten beschrieben:

-   [Parallele Assemblyenumerationen](enumerations.md)
-   [Parallele Assemblyfunktionen](side-by-side-assembly-functions.md)
-   [Parallele Assemblyschnittstellen](side-by-side-assembly-interfaces.md)
-   [Parallele assemblystrukturen](structures.md)
