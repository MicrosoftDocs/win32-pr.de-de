# <a name="side-by-side-assembly-interfaces"></a>Parallele Assemblyschnittstellen

Die Seite-an-Seite-assemblyapi verwendet die folgenden Schnittstellen.



| Schnittstelle                                        | BESCHREIBUNG                                                                                                                                            |
|--------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAssemblyCache**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycache)         | Wird verwendet, um eine parallele Assembly zu installieren, zu deinstallieren oder abzufragen.                                                                                          |
| [**IAssemblyCacheItem**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycacheitem) | Wird verwendet, um parallele Assemblys mithilfe einer streambasierten Installation im parallelen Speicher zu installieren.                                                 |
| [**IAssemblyName**](/windows/desktop/api/winsxs/nn-winsxs-iassemblyname)           | Stellt einen parallelen Assemblynamen dar. Der parallele AssemblyName besteht aus einem Satz von Name-Wert-Paaren, die die parallele Assembly beschreiben. |
