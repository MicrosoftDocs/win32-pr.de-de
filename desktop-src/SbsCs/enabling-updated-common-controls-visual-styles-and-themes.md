# <a name="enabling-updated-common-controls-visual-styles-and-themes"></a>Aktivieren von aktualisierten allgemeinen Steuerelementen, visuellen Stilen und Designs

Anwendungsentwickler, die unter Windows XP ausgeführt werden, können der Benutzeroberfläche Ihrer Anwendungen visuelle Stile und Designs hinzufügen, indem Sie Manifeste einschließen, in denen die Abhängigkeit der Anwendung von COMCTL32.DLL Version 6,0 beschrieben wird. Eine Anwendung erfordert die Funktionalität der aktualisierten allgemeinen Steuerelemente, um Designs und visuelle Stile zu verwenden.

Weitere Informationen zur Verwendung visueller Stile finden Sie unter Verwenden von visuellen Windows XP-Stilen im [shellprogrammiererhandbuch](/previous-versions/windows/desktop/legacy/bb776778(v=vs.85)). Die parallele Assembly, die COMCTL32.DLL Version 6,0 enthält, ist eine der [unterstützten parallelen Microsoft](supported-microsoft-side-by-side-assemblies.md)-Assemblys.
