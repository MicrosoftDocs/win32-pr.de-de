# <a name="enabling-updated-common-controls-visual-styles-and-themes"></a>Aktivieren von aktualisierten allgemeinen Steuerelementen, visuellen Stilen und Designs

Anwendungsentwickler, die unter Windows XP ausgeführt werden, können der Benutzeroberfläche (UI) ihrer Anwendungen visuelle Stile und Designs hinzufügen, indem sie Manifeste hinzufügen, die die Abhängigkeit der Anwendung von COMCTL32.DLL Version 6.0 beschreiben. Eine Anwendung erfordert die Funktionalität der aktualisierten allgemeinen Steuerelemente, um Designs und visuelle Stile zu verwenden.

Weitere Informationen zur Verwendung visueller Stile finden Sie unter Using Windows XP Visual Styles in the [Shell Programmer es Guide](/previous-versions/windows/desktop/legacy/bb776778(v=vs.85)). Die seiteseitige Assembly, die COMCTL32.DLL 6.0 enthält, ist eine der unterstützten [Microsoft-Assemblys.](supported-microsoft-side-by-side-assemblies.md)
