# <a name="side-by-side-assembly-functions"></a>Parallele Assemblyfunktionen

Die Seite-an-Seite-assemblyapi verwendet die folgenden Funktionen für den Zugriff auf die Seite-an-Seite-Assemblyschnittstellen.



| Funktion                                                     | BESCHREIBUNG                                                                    |
|--------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**CreateAssemblyCache**](/windows/desktop/api/Winsxs/nf-winsxs-createassemblycache)           | Ruft eine Instanz der [**IAssemblyCache**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycache) -Schnittstelle ab. |
| [**CreateAssemblyNameObject**](/windows/desktop/api/Winsxs/nf-winsxs-createassemblynameobject) | Ruft eine Instanz der [**IAssemblyName**](/windows/desktop/api/winsxs/nn-winsxs-iassemblyname) -Schnittstelle ab.   |
