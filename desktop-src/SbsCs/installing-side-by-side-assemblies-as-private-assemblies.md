# <a name="installing-side-by-side-assemblies-as-private-assemblies"></a>Parallele Assemblys als private Assemblys installieren

Um parallele Assemblys als [private](/windows/desktop/Msi/private-assemblies)Assemblys zu installieren, können Sie die Assembly als Komponente eines Installer- [Pakets installieren.](about-side-by-side-assemblies-.md) Dabei kann es sich um das Installationspaket handeln, mit dem die Anwendung installiert oder aktualisiert wird. Zum Installieren von Assemblys ist Windows Installer Version 2,0 oder höher erforderlich. Weitere Informationen finden Sie im [Windows Installer](../msi/windows-installer-portal.md) SDK und in den Abschnitten unter [Installieren von Win32](../msi/installation-of-win32-assemblies.md)-Assemblys.

Als Alternative können Sie das Installationsprogramm verwenden, um eine private Assembly und ein Manifest in denselben Ordner wie die ausführbare Datei der Anwendung zu kopieren.
