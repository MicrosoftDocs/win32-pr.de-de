# <a name="default-configuration"></a>Standardkonfiguration

Die in [Assemblymanifesten](assembly-manifests.md) und [Anwendungs Manifesten](application-manifests.md) angegebenen Assemblyversionen bilden die Standardkonfiguration, die von der Anwendung beim Ausführen der Anwendung verwendet werden soll. Anwendungen und Administratoren können die standardassemblykonfiguration überschreiben, indem Sie eine [Verleger Konfiguration](publisher-configuration.md) oder eine [anwendungsspezifische Konfiguration](per-application-configuration.md) auf Windows Server 2003 installieren.
