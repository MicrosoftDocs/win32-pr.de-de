# <a name="default-configuration"></a>Standardkonfiguration

Die in Assemblymanifesten [](application-manifests.md) und Anwendungsmanifesten angegebenen Assemblyversionen umfassen die Standardkonfiguration, die die Anwendung verwenden soll, wenn die Anwendung ausgeführt wird. [](assembly-manifests.md) Anwendungen und Administratoren können die Standardmäßige [](publisher-configuration.md) Assemblykonfiguration [](per-application-configuration.md) überschreiben, indem sie eine Herausgeberkonfiguration oder anwendungsspezifische Konfiguration auf Windows Server 2003 installieren.
