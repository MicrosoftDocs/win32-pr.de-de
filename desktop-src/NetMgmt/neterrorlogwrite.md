# <a name="neterrorlogwrite"></a>"Netterrorlogwrite"

Die Funktion " **netterrorlogwrite** " ist veraltet. Es ist aus Gründen der Kompatibilität mit 16-Bit-Versionen von Windows enthalten. Andere Anwendungen sollten die [Ereignisprotokollierung](/windows/desktop/EventLog/event-logging)verwenden.
