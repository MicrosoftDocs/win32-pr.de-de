# <a name="network-management"></a>Netzwerkverwaltung

## <a name="purpose"></a>Zweck

Die Netzwerk Verwaltungsfunktionen bieten die Möglichkeit, Benutzerkonten und Netzwerkressourcen zu verwalten. Viele der Funktionen, die von den Netzwerk Verwaltungsfunktionen bereitgestellt werden, werden nicht von anderen Netzwerkfunktionen bereitgestellt. Wenn die Funktionen jedoch von einem anderen Satz von Funktionen bereitgestellt werden, werden Sie in der Dokumentation für die Netzwerk Verwaltungsfunktionen auf andere Funktionen verwiesen, die Sie für dieselbe Aufgabe verwenden können.

## <a name="developer-audience"></a>Entwicklergruppe

Die Netzwerk Verwaltungsfunktionen sind für die Verwendung durch C/C++-Programmierer konzipiert. Programmierer sollten auch mit den Windows-Netzwerk Konzepten vertraut sein.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Die Netzwerk Verwaltungsfunktionen können auf allen Windows-Plattformen verwendet werden. Wenn bestimmte Implementierungen oder Funktionen der Netzwerk Verwaltungsplattform-Einschränkungen vorhanden sind, sind Sie in der Dokumentation eindeutig angegeben.
