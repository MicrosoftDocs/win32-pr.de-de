# <a name="network-management"></a>Netzwerkverwaltung

## <a name="purpose"></a>Zweck

Die Netzwerkverwaltungsfunktionen bieten die Möglichkeit, Benutzerkonten und Netzwerkressourcen zu verwalten. Viele der Funktionen, die von den Netzwerkverwaltungsfunktionen bereitgestellt werden, werden nicht von anderen Netzwerkfunktionen bereitgestellt. Wenn die Funktionen jedoch von einem anderen Satz von Funktionen bereitgestellt werden, verweist die Dokumentation für die Netzwerkverwaltungsfunktionen auf andere Funktionen, die Sie für dieselbe Aufgabe verwenden können.

## <a name="developer-audience"></a>Entwicklergruppe

Die Netzwerkverwaltungsfunktionen sind für die Verwendung durch C/C++-Programmierer konzipiert. Programmierer sollten auch mit Windows Netzwerkkonzepten vertraut sein.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Die Netzwerkverwaltungsfunktionen können auf allen Windows Plattformen verwendet werden. Wenn bestimmte Implementierungen oder Funktionen von Einschränkungen der Netzwerkverwaltungsplattform vorhanden sind, sind sie in der Dokumentation eindeutig angegeben.
