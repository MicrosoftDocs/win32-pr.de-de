# <a name="obsolete-information-fields"></a>Veraltete Informationsfelder

Viele der Informationsfelder in den zentralen Informationsstrukturen sind veraltet. Diese Felder verbleiben aus Gründen der Kompatibilität mit 16-Bit-Versionen von Windows in der Informationsstruktur und geben auf anderen Windows Systemen einen intelligenten Standardwert zurück.
