# <a name="obsolete-information-fields"></a>Veraltete Informationsfelder

Viele der Informationsfelder in den Kern Informationsstrukturen sind veraltet. Diese Felder verbleiben in der Informationsstruktur zur Kompatibilität mit 16-Bit-Versionen von Windows und geben einen intelligenten Standardwert auf anderen Windows-Systemen zurück.
