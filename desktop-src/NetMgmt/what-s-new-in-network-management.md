# <a name="whats-new-in-network-management"></a>Neuerungen bei der Netzwerkverwaltung

## <a name="windows-8-and-windows-server-2012"></a>Windows 8 und Windows Server 2012

Mit Microsoft Windows 8 werden neue Programmierfunktionen für die Netzwerkverwaltung eingeführt. Diese neuen Elemente erweitern die Funktionalität der Netzwerkverwaltung zum Erstellen von Bereitstellungs Paketen für Offline-Domänen Beitritts Vorgänge bei der Bereitstellung von Computern mit Windows 8.

Im folgenden sind die neuen Netzwerk Verwaltungsfunktionen aufgeführt:

-   [**Netkreateprovisioningpackage**](/windows/desktop/api/Lmjoin/nf-lmjoin-netprovisioncomputeraccount)
-   [**"Nettrequestprovisioningpackagein Stall"**](/windows/desktop/api/Lmjoin/nf-lmjoin-netrequestprovisioningpackageinstall)

Im folgenden sind neue Netzwerk Verwaltungsstrukturen aufgeführt:

-   [**netsetup- \_ Bereitstellungs Parameter \_**](/windows/desktop/api/Lmjoin/ns-lmjoin-netsetup_provisioning_params)
-   [**Benutzer \_ Informationen \_ 24**](/windows/desktop/api/Lmaccess/ns-lmaccess-user_info_24)

Die Funktion " [**nettusergetinfo**](/windows/desktop/api/Lmaccess/nf-lmaccess-netusergetinfo) " wurde auf Windows 8 erweitert, um eine neue Informationsebene zu unterstützen und eine [**Benutzer \_ Info \_ 24**](/windows/desktop/api/Lmaccess/ns-lmaccess-user_info_24) -Struktur mit Benutzerkontoinformationen für ein Konto zurückzugeben, das mit einer Internet Identität verbunden ist.

## <a name="windows-7-and-windows-server-2008-r2"></a>Windows 7 und Windows Server 2008 R2

Mit Microsoft Windows 7 werden neue Programmierfunktionen für die Netzwerkverwaltung eingeführt. Diese Elemente erweitern die Funktionalität der Netzwerkverwaltung, um Offline-Domänen Beitritts Vorgänge bei der Bereitstellung von Computern mit Windows 7 zuzulassen.

Im folgenden sind die neuen Netzwerk Verwaltungsfunktionen aufgeführt:

-   [**NetProvisionComputerAccount**](/windows/desktop/api/Lmjoin/nf-lmjoin-netprovisioncomputeraccount)
-   [**"Nettrequestofflinedomainjoin"**](/windows/desktop/api/Lmjoin/nf-lmjoin-netrequestofflinedomainjoin)

Die folgenden vorhandenen Netzwerk Verwaltungsfunktionen wurden verbessert, um zusätzliche Optionen zu unterstützen:

-   [**NetJoinDomain**](/windows/desktop/api/Lmjoin/nf-lmjoin-netjoindomain)

## <a name="windows-server-2003"></a>Windows Server 2003

Mit Microsoft Windows Server 2003 werden neue Programmierfunktionen für die Netzwerkverwaltung eingeführt. Diese Elemente erweitern die Funktionalität von Netzwerk Verwaltungs Vorgängen auf Windows Server 2003 und höher.
