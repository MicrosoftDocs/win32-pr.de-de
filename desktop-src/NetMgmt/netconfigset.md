# <a name="netconfigset"></a>Netconfigset

Die **netconfigset** -Funktion ist veraltet. Es ist aus Gründen der Kompatibilität mit 16-Bit-Versionen von Windows enthalten. Andere Anwendungen sollten die [Registrierung](/windows/desktop/SysInfo/registry)verwenden.
