# <a name="netserviceinstall"></a>Netserviceingestall

Die **netserviceingestall** -Funktion ist veraltet. Es ist aus Gründen der Kompatibilität mit 16-Bit-Versionen von Windows enthalten. Andere Anwendungen sollten die [Dienstfunktionen](/windows/desktop/Services/service-functions)verwenden.
