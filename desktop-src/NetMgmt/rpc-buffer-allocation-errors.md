# <a name="rpc-buffer-allocation-errors"></a>RPC-Puffer Zuordnungs Fehler

Da die RPC-Lauf Zeit Bibliothek Speicher für Sende-und Empfangs Puffer zuordnet, sollte die Funktion RPC-Zuordnungs Fehler erwarten. Im Fall eines RPC-Zuordnungs Fehlers wird ein fort Setz bares Handle für ungültig erklärt. Dies ist eine Voraussetzung, weil fort Setz Bare Funktionen nicht wieder hergestellt werden können.
