# <a name="network-management-function-security-requirements"></a>Sicherheitsanforderungen für die Netzwerk Verwaltungsfunktion

Zum Aufrufen einiger der Netzwerk Verwaltungsfunktionen ist keine spezielle Gruppenmitgliedschaft erforderlich. Andere Funktionen erfordern, dass Benutzer über eine bestimmte Berechtigungsstufe verfügen, um erfolgreich ausgeführt zu werden. Wenn zutreffend, zeigt der Abschnitt "Hinweise" auf der Referenzseite einer Funktion die Berechtigungsstufe an, die ein Benutzer zum Ausführen der jeweiligen Funktion benötigen muss.

Sicherheitsanforderungen, die für Active Directory Domänen Controller gelten, können sich von den für Server und Arbeitsstationen geltenden Sicherheitsanforderungen unterscheiden.

Weitere Informationen und eine Liste der betroffenen Funktionen finden Sie in den folgenden Themen:

-   [Anforderungen an die Netzwerk Verwaltungsfunktionen auf Active Directory Domänen Controllern](requirements-for-network-management-functions-on-active-directory-domain-controllers.md)
-   [Anforderungen an Netzwerk Verwaltungsfunktionen auf Servern und Arbeitsstationen](requirements-for-network-management-functions-on-servers-and-workstations.md)

Weitere Informationen zum Steuern des Zugriffs auf Sicherungs fähige Objekte finden Sie unter [Access Control](/windows/desktop/SecAuthZ/access-control), [Berechtigungen](/windows/desktop/SecAuthZ/privileges)und Sicherungs [fähige Objekte](/windows/desktop/SecAuthZ/securable-objects).

Weitere Informationen zu bestimmten Sicherheits Deskriptoren, die bei Zugriffs Überprüfungen verwendet werden, finden Sie auf der Referenzseite zu den einzelnen Funktionen. Weitere Informationen zum Aufrufen von Funktionen, für die Administratorrechte erforderlich sind, finden Sie unter [Ausführen mit speziellen Berechtigungen](/windows/desktop/SecBP/running-with-special-privileges).
