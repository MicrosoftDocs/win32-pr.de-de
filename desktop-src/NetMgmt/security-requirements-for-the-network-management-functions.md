# <a name="network-management-function-security-requirements"></a>Sicherheitsanforderungen für Netzwerkverwaltungsfunktionen

Das Aufrufen einiger Netzwerkverwaltungsfunktionen erfordert keine spezielle Gruppenmitgliedschaft. Andere Funktionen erfordern, dass Benutzer über eine bestimmte Berechtigungsebene verfügen, um erfolgreich ausgeführt zu werden. Falls zutreffend, gibt der Abschnitt "Hinweise" auf der Referenzseite einer Funktion die Berechtigungsstufe an, die ein Benutzer zum Ausführen der jeweiligen Funktion erforderlich ist.

Sicherheitsanforderungen, die für Active Directory-Domänencontroller gelten, können sich von denen unterscheiden, die für Server und Arbeitsstationen gelten.

Weitere Informationen und eine Liste der betroffenen Funktionen finden Sie in den folgenden Themen:

-   [Anforderungen für Netzwerkverwaltungsfunktionen auf Active Directory-Domänencontrollern](requirements-for-network-management-functions-on-active-directory-domain-controllers.md)
-   [Anforderungen für Netzwerkverwaltungsfunktionen auf Servern und Arbeitsstationen](requirements-for-network-management-functions-on-servers-and-workstations.md)

Weitere Informationen zum Steuern des Zugriffs auf sicherungsfähige Objekte finden Sie [unter](/windows/desktop/SecAuthZ/access-control)Access Control , [Privileges](/windows/desktop/SecAuthZ/privileges)und [Securable Objects](/windows/desktop/SecAuthZ/securable-objects).

Weitere Informationen zu bestimmten Sicherheitsbeschreibungen, die bei Zugriffsüberprüfungen verwendet werden, finden Sie auf der Referenzseite für einzelne Funktionen. Weitere Informationen zum Aufrufen von Funktionen, die Administratorrechte erfordern, finden Sie unter [Ausführen mit speziellen Berechtigungen.](/windows/desktop/SecBP/running-with-special-privileges)
