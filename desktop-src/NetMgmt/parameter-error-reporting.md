# <a name="parameter-error-reporting"></a>Parameter Fehlerberichterstattung

Die Netzwerk Verwaltungsfunktionen "Add" und "*" geben einen Index für einen Parameter zurück, der fehlerhaft ist. Der Aufrufer kann einen **null** -Zeiger für den Parameter " *zim \_ Err* " übergeben, der angibt, dass das Feld nicht von der Funktion festgelegt werden sollte.
