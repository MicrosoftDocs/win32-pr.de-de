# <a name="parameter-error-reporting"></a>Berichterstellung für Parameterfehler

Die Netzwerkverwaltungsfunktionen Add und SetInfo geben einen Index für einen Parameter zurück, der einen Fehler auflistet. Der Aufrufer kann einen **NULL-Zeiger** für den *parameter parm \_ err* übergeben, der angibt, dass das Feld nicht von der Funktion festgelegt werden soll.
