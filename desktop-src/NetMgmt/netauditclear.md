# <a name="netauditclear"></a>Netauditclear

Die **netauditclear** -Funktion ist veraltet. Es ist aus Gründen der Kompatibilität mit 16-Bit-Versionen von Windows enthalten. Andere Anwendungen sollten die [Ereignisprotokollierung](/windows/desktop/EventLog/event-logging)verwenden.
