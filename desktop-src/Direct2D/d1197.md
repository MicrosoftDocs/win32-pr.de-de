# <a name="d1197-wrong-d3d-device"></a>D1197: falsches D3D-Gerät

Die an diese API über gegebene DXGI-Oberfläche wird vom falschen Gerät entfernt.



|             |         |
|-------------|---------|
| Fehlerstufe | Warnung |



 

## <a name="possible-causes"></a>Mögliche Ursachen

Die an diese API über gegebene DXGI-Oberfläche wird vom falschen Gerät entfernt.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass alle DXGI-Oberflächen über dasselbe D3D-Gerät wie der D2D-Gerätekontext verfügen.
