# <a name="d1197-wrong-d3d-device"></a>D1197: Falsches D3D-Gerät

Die DXGI-Oberfläche, die an diese API übergeben wird, stammt vom falschen Gerät.





 

## <a name="possible-causes"></a>Mögliche Ursachen

Die DXGI-Oberfläche, die an diese API übergeben wird, stammt vom falschen Gerät.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Stellen Sie sicher, dass alle DXGI-Oberflächen das gleiche D3D-Gerät wie der D2D-Gerätekontext aufweisen.
