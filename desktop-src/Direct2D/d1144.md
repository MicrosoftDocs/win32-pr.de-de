# <a name="d1144-effect-is-not-a-target"></a>D1144: der Effekt ist kein Ziel.

Fehler bei SetTarget, weil ein Effekt als Ziel bestanden wurde.






 

## <a name="possible-causes"></a>Mögliche Ursachen

ID2D1DeviceContext:: SetTarget nimmt einen ID2D1Image- \* Parameter an. Direct2D-Bilder können Bitmaps, Befehlslisten oder Effekte sein. Direct2D berücksichtigt nur Bitmaps und Befehlslisten als gültige Ziele.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Ändern Sie die Anwendung so, dass Sie eine Bitmap oder Befehlsliste als Ziel des Geräte Kontexts verwendet. Direct2D führt keine Initialisierung von Effekten mithilfe von SetTarget aus. Um den Inhalt eines Effekts auszuwählen, erstellt eine Anwendung den Effekt mithilfe von ID2D1DeviceContext:: kreateeffect und verwendet eine entsprechende GUID für den Effekt.
