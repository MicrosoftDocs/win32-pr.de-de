# <a name="d1117-bitmap-is-not-a-target"></a>D1117: Bitmap ist kein Ziel

Fehler bei SetTarget, weil die Bitmap nicht die target-Option hat.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Damit ein ID2D1Bitmap als Ziel festgelegt werden kann, muss es mit dem D2D1 \_ Bitmap \_ options Target- \_ Flag erstellt werden. Dieses Flag ist als ein Feld der D2D1 \_ Bitmap PROPERTIES1 verfügbar, das \_ mit ID2D1DeviceContext:: deatebitmap verwendet wird.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass die Anwendung die Ziel Bitmap mithilfe des Ziels "D2D1 \_ Bitmap Options" erstellt \_ \_ . Wenn die Ziel Bitmap mithilfe von ID2D1RenderTarget:: Erstellungs Bitmap erstellt wird, ändern Sie Sie so, dass Sie ID2D1DeviceContext:: deatebitmap verwendet. Wenn das Zielflag angegeben ist, überprüft Direct2D auch, ob die Direct3D-Attribute der Textur mit der Verwendung als Ziel kompatibel sind.
