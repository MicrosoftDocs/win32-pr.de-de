# <a name="d1178-custom-dash-style-requires-dash-array"></a>D1178: Benutzerdefinierter Dash-Stil erfordert das Bindestricharray

Zum Erstellen eines Strichstils mit DASH \_ STYLE CUSTOM ist ein \_ Bindestricharray erforderlich.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Wenn die Anwendung versucht hat, eine ID2D1StrokeStyle zu erstellen, hat sie das D2D1 \_ DASH \_ STYLE \_ CUSTOM-Attribut verwendet, aber ein NULL-Bindestricharray bereitgestellt.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Wenn die Anwendung benutzerdefinierte Bindestriche erfordert, stellen Sie sicher, dass ein gültiges Bindestricharray angegeben ist. Ein Beispiel finden Sie unter CreateStrokeStyle. Wenn ein integrierter Bindestrich akzeptabel ist, belassen Sie das Bindestricharray auf NULL, und verwenden Sie einen Bezeichner wie D2D1 \_ DASH \_ STYLE \_ DASH.
