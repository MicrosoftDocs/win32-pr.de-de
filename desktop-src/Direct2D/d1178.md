# <a name="d1178-custom-dash-style-requires-dash-array"></a>D1178: benutzerdefinierter Dash-Stil erfordert Bindestrich-Array

Das Erstellen eines Strich Stils mit \_ \_ benutzerdefiniertem Dash-Stil erfordert ein Bindestrich-Array.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Wenn die Anwendung versucht hat, ein ID2D1StrokeStyle zu erstellen, wurde das \_ benutzerdefinierte Attribut des D2D1 Dash-Stils verwendet, aber es wurde \_ \_ ein NULL-Bindestrich-Array

## <a name="possible-fixes"></a>Mögliche Korrekturen

Wenn die Anwendung benutzerdefinierte Bindestriche erfordert, stellen Sie sicher, dass ein gültiges Bindestrich-Array angegeben wird. Ein Beispiel finden Sie unter "kreatestrokestyle". Wenn ein integrierter Bindestrich zulässig ist, lassen Sie das Bindestrich-Array als NULL, und verwenden Sie einen Spezifizierer, z \_ . b. D2D1 Dash \_ Style \_ Dash.
