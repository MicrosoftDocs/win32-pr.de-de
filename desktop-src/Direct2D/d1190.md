# <a name="d1190-cannotdraw-bitmap-flag-was-used-incorrectly"></a>D1190: Das Bitmapflag cannotDraw wurde falsch verwendet.

D2D1 \_ BITMAP \_ OPTIONS \_ CANNOT \_ DRAW must be combined with D2D1 \_ BITMAP \_ OPTIONS \_ CPU \_ READ or D2D1 \_ BITMAP \_ OPTIONS \_ TARGET





 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat eine inkompatible Kombination der D2D1 \_ BITMAP \_ OPTIONS-Flags angegeben.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Hinzufügen von D2D1 \_ BITMAP OPTIONS CPU READ oder \_ \_ \_ D2D1 \_ BITMAP OPTIONS TARGET zu den \_ \_ D2D1 \_ BITMAP \_ OPTIONS
