# <a name="d1190-cannotdraw-bitmap-flag-was-used-incorrectly"></a>D1190: das cannotdraw-Bitmap-Flag wurde falsch verwendet.

D2D1 \_ Bitmap- \_ Optionen \_ können nicht \_ gezeichnet werden mit den D2D1 \_ Bitmap- \_ Optionen CPU-Lese- \_ \_ oder D2D1 \_ Bitmap- \_ options \_ Ziel



|             |             |
|-------------|-------------|
| Fehlerstufe | Information |



 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat eine inkompatible Kombination der D2D1 \_ Bitmap \_ options-Flags angegeben.

## <a name="possible-fixes"></a>Mögliche Korrekturen

D2D1 \_ Bitmap- \_ Optionen \_ CPU- \_ Lese-oder D2D1 \_ Bitmap-Optionen als Ziel hinzufügen \_ \_ für die D2D1- \_ Bitmap- \_ Optionen
