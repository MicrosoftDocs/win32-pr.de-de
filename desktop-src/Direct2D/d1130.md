# <a name="d1130-dxgi-format-disallowed-by-caps"></a>D1130: DXGI Format Disallowed By Caps

Das Gerät verfügt nicht über die Möglichkeit, das gegebene DXGI-Format zu unterstützen.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Für Direct2D-Vorgänge, die auf Hardware ausgeführt werden und ein Oberflächenformat verwenden, muss das Oberflächenformat von der Featureebene der Hardware unterstützt werden. Wenn Hardware beispielsweise feature level 9 ist, unterstützt sie möglicherweise keine 16bpp-Formate oder SRGB-Formate wie DXGI \_ FORMAT \_ B8G8R8A8 \_ UNORM \_ SRGB.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Anleitungen zu den Unterschieden zwischen [Featureebenen](/previous-versions/windows/apps/hh994923(v=win.10)) und den unterstützten Formaten finden Sie unter Direct3D-Featureebenen. Verwenden Sie [**die ID3D11Device::CheckFormatSupport-Methode,**](/windows/desktop/api/d3d11/nf-d3d11-id3d11device-checkformatsupport) um zu überprüfen, ob das erforderliche Format von der Hardware unterstützt wird, bevor Sie es mit Direct2D verwenden.
