# <a name="d1130-dxgi-format-disallowed-by-caps"></a>D1130: DXGI-Format ist von Caps nicht zulässig.

Das Gerät verfügt nicht über die Möglichkeit, das angegebene DXGI-Format zu unterstützen.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Bei Direct2D-Vorgängen, die auf der Hardware ausgeführt werden und ein Oberflächen Format verwenden, muss das Oberflächen Format von der Funktionsebene der Hardware unterstützt werden. Wenn z. b. die Hardware Funktionsebene 9 ist, werden 16bpp renderzielformat-Formate oder sRGB-Formate, z. b. DXGI- \_ Format \_ B8G8R8A8 \_ unorm \_ sRGB, möglicherweise nicht unterstützt.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Informationen zu den Unterschieden zwischen featureebenen und den unterstützten Formaten finden Sie unter [Direct3D featureebenen](/previous-versions/windows/apps/hh994923(v=win.10)) . Verwenden Sie die [**ID3D11Device:: checkformatsupport-Methode**](/windows/desktop/api/d3d11/nf-d3d11-id3d11device-checkformatsupport) , um zu überprüfen, ob das erforderliche Format von der Hardware unterstützt wird, bevor Sie es mit Direct2D verwenden.
