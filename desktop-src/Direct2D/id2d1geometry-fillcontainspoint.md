# <a name="id2d1geometryfillcontainspoint-methods"></a>ID2D1Geometry::FillContainsPoint-Methoden

Gibt an, ob der von der Geometrie ausgefüllte Bereich den angegebenen Punkt enthalten würde.

| [**FillContainsPoint(D2D1 \_ POINT \_ 2F,D2D1 \_ MATRIX \_ 3X2 \_ F , \* BOOL \* )**](/windows/win32/api/d2d1/nf-d2d1-id2d1geometry-fillcontainspoint(d2d1_point_2f_constd2d1_matrix_3x2_f_bool))             | Gibt an, ob der von dieser Geometrie ausgefüllte Bereich den angegebenen Punkt enthalten würde.<br/>                                          |
