# <a name="d1196-a8-shared-bitmap"></a>D1196: A8 Shared Bitmap

Eine DXGI-Oberfläche im Format DXGI \_ FORMAT \_ A8 \_ UNORM wird in dieser API nicht unterstützt.





 

## <a name="possible-causes"></a>Mögliche Ursachen

DXGI \_ FORMAT \_ A8 \_ UNORM wird nicht als DXGI-Interopbitmapformat unterstützt, auch wenn es in anderen APIs unterstützt wird.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Erstellen Sie eine DXGI-Interopbitmap in einem anderen Format.
