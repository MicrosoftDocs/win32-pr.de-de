# <a name="d1196-a8-shared-bitmap"></a>D1196: freigegebene A8-Bitmap

Eine DXGI-Oberfläche des Formats DXGI- \_ Format \_ a8 \_ unorm wird in dieser API nicht unterstützt.



|             |         |
|-------------|---------|
| Fehlerstufe | Warnung |



 

## <a name="possible-causes"></a>Mögliche Ursachen

Das DXGI- \_ Format \_ a8 \_ unorm wird nicht als DXGI-Interop-Bitmap-Format unterstützt, auch wenn es in anderen APIs unterstützt wird.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Erstellen Sie eine DXGI-Interop-Bitmap mit einem anderen Format.
