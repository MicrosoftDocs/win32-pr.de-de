## <a name="examples"></a><span data-ttu-id="7ce5d-117">Beispiele</span><span class="sxs-lookup"><span data-stu-id="7ce5d-117">Examples</span></span>

<span data-ttu-id="7ce5d-118">Im folgenden Codebeispiel wird gezeigt, wie Sie mithilfe von "Mosaik" eine Reihe von Dreiecken im Uhrzeigersinn erstellen, die die Geometrie abdecken.</span><span class="sxs-lookup"><span data-stu-id="7ce5d-118">The following code example shows how to use Tessellate to create a set of clockwise-wound triangles that cover the geometry.</span></span>


```C++
 ID2D1GeometrySink *pGeometrySink = NULL;
 hr = pPathGeometry->Open(&pGeometrySink);
 if (SUCCEEDED(hr))
 {
     hr = pGeometry->Widen(
             strokeWidth,
             pIStrokeStyle,
             pWorldTransform,
             pGeometrySink
             );

     if (SUCCEEDED(hr))
     {
         hr = pGeometrySink->Close();
         if (SUCCEEDED(hr))
         {
             ID2D1Mesh *pMesh = NULL;
             hr = m_pRT->CreateMesh(&pMesh);
             if (SUCCEEDED(hr))
             {
                 ID2D1TessellationSink *pSink = NULL;
                 hr = pMesh->Open(&pSink);
                 if (SUCCEEDED(hr))
                 {
                     hr = pPathGeometry->Tessellate(
                             NULL, // world transform (already handled in Widen)
                             pSink
                             );
                     if (SUCCEEDED(hr))
                     {
                         hr = pSink->Close();
                         if (SUCCEEDED(hr))
                         {
                             SafeReplace(&m_pStrokeMesh, pMesh);
                         }
                     }
                     pSink->Release();
                 }
                 pMesh->Release();
             }
         }
     }
     pGeometrySink->Release();
 }
 pPathGeometry->Release();
```
