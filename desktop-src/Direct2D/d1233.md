# <a name="d1233-begindraw-invalid-from-previous-error"></a>D1233: beginDraw ist aus vorheriger Fehler ungültig.

BeginDraw wurde aufgerufen, wird jedoch ignoriert, da sich der Gerätekontext in einem ungültigen Zustand befindet.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung namens beginDraw, aber der Gerätekontext befand sich in einem ungültigen Zustand als ein vorheriger Vorgang.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass die Anwendung entsprechend den von Flush oder EndDraw zurückgegebenen Fehlern antwortet, und erstellen Sie das Gerät nach Bedarf neu. Wenn ein Befehl bewirkt hat, dass sich ein Gerätekontext in einem ungültigen Zustand befindet, gibt der entsprechende Flush oder EndDraw einen Fehler zurück.
