# <a name="d1233-begindraw-invalid-from-previous-error"></a>D1233: BeginDraw Invalid From Previous Error

BeginDraw wurde aufgerufen, wird jedoch ignoriert, da sich der Gerätekontext in einem ungültigen Zustand befindet.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat BeginDraw aufgerufen, aber der Gerätekontext befand sich in einem ungültigen Zustand eines vorherigen Vorgangs.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Stellen Sie sicher, dass die Anwendung ordnungsgemäß auf Fehler reagiert, die von Flush oder EndDraw zurückgegeben werden, und das Gerät nach Bedarf neu erstellung. Wenn ein Befehl dazu geführt hat, dass sich ein Gerätekontext in einem ungültigen Zustand befindet, gibt die entsprechende Flush- oder EndDraw-Anweisung einen Fehler zurück.
