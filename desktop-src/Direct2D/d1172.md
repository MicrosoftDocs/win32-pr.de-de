# <a name="d1172-invalid-to-unmap-nonmapped-bitmap"></a>D1172: Ungültiges Aufheben der Zuordnung der nicht zugeordneten Bitmap.

Es wurde versucht, eine nicht zugeordnete Bitmap zu entordnen.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung möglicherweise aus und meldet einen Fehler, wenn eine Anwendung versucht, die Zuordnung einer Bitmap zu aufheben, die noch nicht mit ID2D1Bitmap1:: Map zugeordnet wurde.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie auf dieselbe Weise, wie ID2D1RenderTarget:: EndDraw den Aufrufen von ID2D1RenderTarget:: beginDraw folgt, sicher, dass ID2D1Bitmap1:: unmap den Aufrufen von ID2D1Bitmap1:: Map folgt. Nennen Sie ID2D1Bitmap1:: aufheben mehrmals nicht, um die Zuordnung einer Bitmap zu aufheben.
