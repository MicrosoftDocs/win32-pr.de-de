# <a name="d1172-invalid-to-unmap-nonmapped-bitmap"></a>D1172: Ungültiges Nicht zugeordnetes Bitmap

Es wurde versucht, die Zuordnung einer Bitmap zu entzuordnungen, die nicht zugeordnet ist.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung möglicherweise aus und gibt einen Fehler aus, wenn eine Anwendung versucht, eine Bitmap, die noch nicht zugeordnet ist, mit ID2D1Bitmap1::Map zu entzuordnungen.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie genauso wie ID2D1RenderTarget::EndDraw Aufrufe von ID2D1RenderTarget::BeginDraw sicher, dass ID2D1Bitmap1::Unmap den Aufrufen von ID2D1Bitmap1::Map folgt. Rufen Sie ID2D1Bitmap1::Unmap nicht mehrmals auf, um eine Bitmap zu entzuordnungen.
