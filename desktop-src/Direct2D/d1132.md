# <a name="d1132-dxgi-format-not-hwnd-or-dc-compatible"></a>D1132: DXGI-Format nicht HWND oder DC-kompatibel

Das angegebene DXGI-Format ist für diese Renderziel-Erstellungs Methode ungültig.



|             |         |
|-------------|---------|
| Fehlerstufe | Warnung |



 

## <a name="possible-causes"></a>Mögliche Ursachen

Beim Erstellen von Rendertargets für HWND oder DC mithilfe von ID2D1Factory:: foatehwndrendertarget oder ID2D1Factory:: roatedcrendertarget werden zusätzliche Einschränkungen für das Oberflächen Format des Ziels eingeführt. Insbesondere sollten diese Ziele den Typ DXGI- \_ Format \_ R8G8B8A8 \_ unorm oder DXGI- \_ Format \_ B8G8R8A8 \_ unorm aufweisen.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Um dieses Problem zu beheben, überprüfen Sie, ob das für das HWND oder DC-renderTarget einer Anwendung angegebene Format eines der oben genannten Formate ist. Wenn die Anwendung nicht notwendigerweise diese Arten von Zielen erfordert, sollten Sie stattdessen das Rendering in eine Textur in Erwägung gezogen, da die Oberflächen Formatoptionen für die Kompatibilität als Ziel möglicherweise weniger restriktiv sind.
