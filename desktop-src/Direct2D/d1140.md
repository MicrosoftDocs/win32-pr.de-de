# <a name="d1140-stroke-transform-type-uses-default-dpi"></a>D1140: Typ der Strichtransformation verwendet Standard-DPI

Beim Widening einer ID2D1Geometry mit einem FIXED- oder HAIRLINE-Strich geht Direct2D von einem DPI-96-Strich aus. Beim Rendern der breiteren Geometrie kann sich die Strichbreite ändern.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Wenn Direct2D eine Geometrie auf ein Ziel zeichnet, wird die Geometrie geweitet. Eine Anwendung kann jedoch auch ID2D1Geometry-Objekte ohne Ziel erstellen und sie mithilfe von Methoden von ID2D1Geometry wie Widen, StrokeContainsPoint und GetWidenedBounds ausweiten. Während dieser Vorgänge kann eine Anwendung D2D1 \_ STROKE \_ TRANFORM \_ TYPE \_ HAIRLINE oder FIXED verwenden. Diese Attribute betrachten normalerweise einen DPI, aber ohne Ziel wird kein DPI angegeben. Diese Informationsmeldung warnt, dass Direct2D einen DPI-Standardwert von 96,0 in Betracht ziehen wird.

## <a name="possible-fixes"></a>Mögliche Korrekturen
