# <a name="d1198-copy-to-read-only-bitmap"></a>D1198: in Read-Only Bitmap kopieren

Es wurde versucht, Daten in eine schreibgeschützte oder unveränderliche Bitmap zu kopieren.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Es wurde versucht, Daten in eine schreibgeschützte oder unveränderliche Bitmap zu kopieren.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Vergewissern Sie sich, dass Bitmaps, die in geschrieben werden, nicht als unveränderlich erstellt werden.
