# <a name="matrix5x4f-class"></a>Matrix5x4F-Klasse

Die **Matrix5x4F-Klasse** stellt eine 5 x 4-Matrix dar und stellt Benutzerfreundlichkeitsmethoden zum Erstellen von Matrizen zur Verfügung.

## <a name="members"></a>Member

Die **Matrix5x4F-Klasse** erbt von [**D2D1 \_ MATRIX \_ 5X4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f). **Matrix5x4F** verfügt auch über diese Typen von Membern:
