# <a name="id2d1propertiesgetvalue-methods"></a>ID2D1Properties::GetValue-Methoden

Ruft den Wert der Eigenschaft nach Index ab.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                      | Beschreibung                                                                                |
|:--------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------|
| [**GetValue(U)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getvalue(u))                                            | Ruft den Wert der Eigenschaft nach Index ab. Dies ist eine Vorlagenüberladung.<br/>           |
| [**GetValue(U, T \* )**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getvalue(u_t))                                       | Ruft den Wert der Eigenschaft nach Index ab. Dies ist eine Vorlagenüberladung.<br/>           |
| [**GetValue(U, \* BYTE, UINT32)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getvalue(u_byte_uint32))                            | Ruft den Wert der angegebenen Eigenschaft nach Index ab. Dies ist eine Vorlagenüberladung.<br/> |
| [**GetValue(UINT32, \* BYTE, UINT32)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getvalue(uint32_byte_uint32))                        | Ruft den Wert der angegebenen Eigenschaft nach Index ab.<br/>                              |
| [**GetValue(UINT32, D2D1 \_ PROPERTY \_ TYPE, \* BYTE, UINT32)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getvalue(uint32_byte_uint32)) | Ruft den Wert der angegebenen Eigenschaft nach Index ab.<br/>                              |
