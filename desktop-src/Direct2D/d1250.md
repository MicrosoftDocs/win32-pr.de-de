# <a name="d1250-inaccessible-source"></a>D1250: Quelle, auf die nicht zugegriffen werden kann

Es wurde ein Versuch erkannt, aus einer Bitmap zu zeichnen, auf die nicht zugegriffen werden kann.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Es wurde ein Versuch erkannt, aus einer Quelle zu zeichnen, auf die nicht zugegriffen werden kann. Ein Grund, warum eine Quelle möglicherweise unersässig ist, ist, wenn sie aus einer freigegebenen Textur erstellt wurde und die Textur nicht erworben wurde.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass auf alle Bitmaps zugegriffen werden kann, bevor Sie sie mit Direct2D zeichnen.
