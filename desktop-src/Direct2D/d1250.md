# <a name="d1250-inaccessible-source"></a>D1250: Quelle nicht zugänglich

Es wurde versucht, aus einer nicht zugänglichen Bitmap zu zeichnen.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Es wurde versucht, von einer nicht zugänglichen Quelle zu zeichnen. Ein Grund, warum eine Quelle möglicherweise inkonsistent ist, ist, wenn Sie aus einer freigegebenen Textur erstellt wurde und die Textur nicht abgerufen wurde.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass auf alle Bitmaps zugegriffen werden kann, bevor Sie Sie mit Direct2D zeichnen
