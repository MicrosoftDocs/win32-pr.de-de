# <a name="d1160-cycle-in-graph"></a>D1160: Cycle In Graph

Es wurde versucht, ein Bilddiagramm mit einem Zyklus zu zeichnen.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D ermöglicht Anwendungen das Erstellen von Effektdiagrammen, um ein flexibles Verhalten zu erzielen. Anwendungen sollten jedoch kein Effektdiagramm erstellen, das einen Zyklus enthält. Betrachten Sie beispielsweise ein Effektdiagramm mit den Knoten A, B und C: Es ist ungültig, A mit B, dann B mit C und dann auch C mit A zu verknüpfen. Direct2D identifiziert zyklische Effektdiagramme und gibt einen Fehler aus. Die Debugebene gibt diese Meldung aus.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Um diese Meldung zu vermeiden, vermeiden Sie das Erstellen von Effektdiagrammen, die einen Zyklus haben. Entwurfseffektdiagramme effektiv, einschließlich mindestens eines Knotens, der nur als Ausgabe fungiert.
