# <a name="d1160-cycle-in-graph"></a>D1160: in Diagramm Schleifen

Es wurde versucht, ein Bild Diagramm mit einem solchen Cycle zu zeichnen.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D ermöglicht Anwendungen das Erstellen von Effekt Diagrammen, um das Verhalten von Flexibilität zu erhöhen. Anwendungen sollten jedoch kein Effekt Diagramm erstellen, das einen Kreis enthält. Nehmen wir beispielsweise an, ein Effekt Diagramm enthält die Knoten a, B und c: Es ist ungültig, A mit b zu verknüpfen, dann B zu c und dann c zu A. Direct2D identifiziert zyklische Effekt Diagramme und meldet einen Fehler. Diese Meldung wird von der Debugebene ausgegeben.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Um diese Meldung zu vermeiden, sollten Sie keine Auswirkung Graphen erstellen, die einen Kreis haben. Design Effekt Grafiken, einschließlich mindestens eines Knotens, der nur als Ausgabe fungiert.
