# <a name="d1239-source-transform-bitmap-invalid-call"></a>D1239: Quelltransformation Bitmap Ungültiger Aufruf

Die an ID2D1SourceTransform::D raw übergebene Bitmap unterstützt diese Methode nicht.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat eine nicht unterstützte Methode für eine Bitmap aufgerufen, die an ID2D1SourceTransform::D übergeben wird.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Verwenden Sie ID2D1Bitmap1::Map, um diese Art von Bitmap zu aktualisieren.
