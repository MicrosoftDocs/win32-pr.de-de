Die an ID2D1SourceTransform über gegebene Bitmap::D RAW unterstützt diese Methode nicht.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat eine nicht unterstützte Methode in einer Bitmap aufgerufen, die an ID2D1SourceTransform übergeben wurde::D RAW.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Um diesen Typ von Bitmap zu aktualisieren, verwenden Sie ID2D1Bitmap1:: map.
