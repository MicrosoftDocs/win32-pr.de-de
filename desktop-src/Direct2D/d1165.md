# <a name="d1165-read-and-discard-map-options-are-incompatible"></a>D1165: Kartenoptionen lesen und verwerfen sind inkompatibel

D2D1 \_ MAP OPTIONS READ ist nicht kompatibel mit \_ \_ D2D1 \_ MAP OPTIONS \_ \_ DISCARD





 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung möglicherweise aus, wenn eine Anwendung ID2D1Bitmap1::Map mit den Optionen D2D1 MAP OPTIONS READ und \_ \_ \_ D2D1 \_ MAP OPTIONS DISCARD \_ \_ aufruft.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Die Optionen READ und DISCARD stimmen nicht überein, da das Zulassen des Lesezugriffs auf nicht definierte Daten für eine Anwendung nur eingeschränkt nützlich ist. Eine Beschreibung der Kartenoptionen finden Sie unter D2D1 \_ MAP \_ OPTIONS (D2D1-KARTENOPTIONEN). Berücksichtigen Sie beim Aufrufen von ID2D1Bitmap1::Map, welche Art von Zugriff die Anwendung benötigt.
