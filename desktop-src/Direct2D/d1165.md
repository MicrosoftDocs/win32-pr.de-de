# <a name="d1165-read-and-discard-map-options-are-incompatible"></a>D1165: Optionen zum Lesen und Verwerfen von Zuordnungen sind nicht kompatibel.

D2D1 \_ map- \_ Optionen \_ Read ist nicht kompatibel mit D2D1 \_ map- \_ Optionen \_ verwerfen



|             |         |
|-------------|---------|
| Fehlerstufe | Warnung |



 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung ggf. aus, wenn eine Anwendung ID2D1Bitmap1:: Map mit den Optionen D2D1 \_ map \_ Optionen \_ Read und D2D1 \_ map options \_ \_ Verwerfungs Optionen aufruft.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Die Optionen lesen und verwerfen stimmen nicht überein, da das Zulassen von Lesezugriff auf nicht definierte Daten eine eingeschränkte Nützlichkeit für eine Anwendung hat. \_ \_ Eine Beschreibung der Karten Optionen finden Sie unter D2D1 Map Options. Wenn Sie ID2D1Bitmap1:: Map aufrufen, sollten Sie überprüfen, welcher Zugriffstyp für die Anwendung erforderlich ist.
