# <a name="d1173-texture-cube-interop-unsupported"></a>D1173: Textur Cube-Interop wird nicht unterstützt.

Texturen, die als D3D11RESOURCE \_ misc \_ texturecube gekennzeichnet sind, dürfen nicht von Direct2D verwendet werden.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat versucht, eine Direct3D Textur mit der angegebenen D3D11RESOURCE \_ misc \_ texturecube-Option zu Interop.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Unterscheidet sich von der Interaktion mit dem texturecube-Spezifizierer. Ordnen Sie stattdessen den renderingentwurf der Anwendung neu an, oder verwenden Sie eine zwischengeschaltete Interoperation.
