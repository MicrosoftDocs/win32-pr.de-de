# <a name="d1173-texture-cube-interop-unsupported"></a>D1173: Texturcube-Interop nicht unterstützt

Texturen, die als D3D11RESOURCE \_ MISC TEXTURECUBE gekennzeichnet \_ sind, dürfen von Direct2D nicht verwendet werden.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat versucht, eine Direct3D-Textur mit der angegebenen D3D11RESOURCE \_ MISC \_ TEXTURECUBE-Option zu überlagern.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Vermeiden Sie die Interoperabilität von Texturen mit dem TEXTURECUBE-Bezeichner. Organisieren Sie stattdessen den Renderingentwurf der Anwendung neu, oder erwägen Sie die Interoperabilität eines Zwischenprodukts.
