# <a name="d1128-dxgi-format-not-supported-in-any-api"></a>D1128: DXGI-Format wird in keiner API unterstützt

Das an diese API übergebene DXGI-Format wird von Direct2D nicht unterstützt.





 

## <a name="possible-causes"></a>Mögliche Ursachen

Einige Oberflächenformate sind unabhängig von ihrer Verwendung nicht mit Direct2D-Vorgängen kompatibel. Diese Meldung gibt an, dass die Anwendung ein anderes Oberflächenformat auswählen soll.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Informationen dazu, welche Formate unterstützt werden und welche nicht, finden Sie unter [Unterstützte Pixelformate und Alphamodi.](supported-pixel-formats-and-alpha-modes.md) Wenn die Anwendung in einem einzelnen Kanal gerendert werden muss und versucht, ein Nur-Rot-Kanal-Format zu verwenden, das nicht unterstützt wird, sollten Sie erwägen, es in ein rein alphankanalbasiertes Format zu rendern.
