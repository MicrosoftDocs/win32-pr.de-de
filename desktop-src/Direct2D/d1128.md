# <a name="d1128-dxgi-format-not-supported-in-any-api"></a>D1128: DXGI-Format wird in keiner API unterstützt.

Das an diese API eingeführte DXGI-Format wird von Direct2D nicht unterstützt.



|             |         |
|-------------|---------|
| Fehlerstufe | Warnung |



 

## <a name="possible-causes"></a>Mögliche Ursachen

Einige Oberflächen Formate sind unabhängig von ihrer Verwendung nicht mit Direct2D-Vorgängen kompatibel. Diese Meldung gibt an, dass die Anwendung ein anderes Oberflächen Format auswählen sollte.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Informationen dazu, welche Formate unterstützt werden und welche nicht, finden Sie [unter Unterstützte Pixel Formate und Alpha-Modi](supported-pixel-formats-and-alpha-modes.md) . Wenn die Anwendung zu einem einzelnen Kanal Renderern muss und versucht, ein nur-Rot-Kanal-Format zu verwenden, das nicht unterstützt wird, sollten Sie es ändern, um es in ein nur-Alpha-Kanal-Format zu Renderern.
