# <a name="d1228-not-original-target"></a>D1228: Kein ursprüngliches Ziel

Ein Aufruf von ID2D1HwndRenderTarget::Resize oder ID2D1DCRenderTarget::BindDC wurde vorgenommen, während ein anderes Ziel als das ursprüngliche im Gerätekontext festgelegt wurde.
