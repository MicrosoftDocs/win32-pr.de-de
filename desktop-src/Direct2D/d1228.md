# <a name="d1228-not-original-target"></a>D1228: nicht ursprüngliches Ziel

Es wurde ein Aufrufen von ID2D1HwndRenderTarget:: Resize oder ID2D1DCRenderTarget:: binddc vorgenommen, während ein anderes als das ursprüngliche Ziel für den Gerätekontext festgelegt wurde.
