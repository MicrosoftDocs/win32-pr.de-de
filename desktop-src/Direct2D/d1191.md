# <a name="d1191-gdi-compatible-bitmap-option-was-used-incorrectly"></a>D1191: Die GDI-kompatible Bitmapoption wurde falsch verwendet.

D2D1 \_ BITMAP \_ OPTIONS \_ GDI \_ COMPATIBLE must be combined with D2D1 \_ BITMAP \_ OPTIONS \_ TARGET






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat eine inkompatible Kombination der BITMAP \_ OPTIONS-Flags von D2D1 \_ angegeben.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Fügen Sie den \_ \_ \_ Flags D2D1 BITMAP OPTIONS TARGET (D2D1 BITMAP OPTIONS TARGET) \_ \_ hinzu.
