# <a name="d1169-discard-map-option-requires-dynamic-texture"></a>D1169: map-Option verwerfen erfordert dynamische Textur

D2D1 \_ map- \_ Optionen \_ verwerfen ist nur gültig, wenn eine Bitmap aus einer dynamischen Textur erstellt wurde.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung möglicherweise aus, wenn eine Anwendung ID2D1Bitmap1:: Map mit dem D2D1 \_ map \_ options \_ DISCARD-Attribut aufruft, die zugrunde liegende Direct3D-Ressource für die Bitmap jedoch nicht mit der dynamischen D3D11-Verwendung attributiert wird \_ \_ .

## <a name="possible-fixes"></a>Mögliche Korrekturen

Wenn eine Anwendung den Zugriff auf eine Bitmap mithilfe von ID2D1Bitmap1:: Map erfordert, stellen Sie sicher, dass die zugrunde liegende Direct3D-Ressource das \_ dynamische Attribut D3D11 Usage aufweist \_ .
