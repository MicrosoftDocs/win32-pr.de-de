# <a name="d1169-discard-map-option-requires-dynamic-texture"></a>D1169: Option "Karte verwerfen" erfordert dynamische Textur

D2D1 \_ MAP OPTIONS DISCARD ist nur mit einer Bitmap \_ \_ gültig, die aus einer dynamischen Textur erstellt wurde.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung möglicherweise aus, wenn eine Anwendung ID2D1Bitmap1::Map mit dem ATTRIBUT D2D1 MAP OPTIONS DISCARD aufruft, die zugrunde liegende Direct3D-Ressource für die Bitmap jedoch nicht \_ \_ \_ D3D11 USAGE DYNAMIC zugeordnet \_ \_ ist.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Wenn eine Anwendung zugriff auf eine Bitmap mithilfe von ID2D1Bitmap1::Map benötigt, stellen Sie sicher, dass die zugrunde liegende Direct3D-Ressource über das Attribut D3D11 \_ USAGE \_ DYNAMIC verfügt.
