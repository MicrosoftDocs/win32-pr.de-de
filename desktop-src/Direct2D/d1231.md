# <a name="d1231-unknown-dxgi-format-support"></a>D1231: Unterstützung des unbekannten DXGI-Formats

IsDxgiFormatSupported wurde mit DXGI \_ FORMAT \_ UNKNOWN aufgerufen. Diese API gibt false zurück, obwohl DXGI \_ FORMAT UNKNOWN möglicherweise als Eingabe für einige \_ APIs gültig ist.






 

## <a name="possible-causes"></a>Mögliche Ursachen

IsDxgiFormatSupported wurde mit DXGI \_ FORMAT \_ UNKNOWN aufgerufen.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Bestimmen Sie, ob DXGI \_ FORMAT UNKNOWN eine gültige \_ Puffergenauigkeit für eine bestimmte API ist, indem Sie die Dokumentation für diese API lesen, anstatt zu versuchen, diese programmgesteuert zu bestimmen.
