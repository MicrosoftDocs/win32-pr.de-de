# <a name="d1231-unknown-dxgi-format-support"></a>D1231: Unbekannte DXGI-Format Unterstützung

Ein isdxgiformatsupported-isdxgiformatsupported-Befehl wurde mit dem DXGI- \_ Format \_ Unknown Diese API gibt false zurück, obwohl das DXGI- \_ Format \_ Unknown möglicherweise als Eingabe für einige APIs gültig ist.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Ein isdxgiformatsupported-isdxgiformatsupported-Befehl wurde mit dem DXGI- \_ Format \_ Unknown

## <a name="possible-fixes"></a>Mögliche Korrekturen

Bestimmen Sie, ob das DXGI- \_ Format \_ Unknown eine gültige Puffer Genauigkeit für eine bestimmte API ist, indem Sie die Dokumentation für diese API lesen, anstatt zu versuchen, dies Programm gesteuert zu ermitteln.
