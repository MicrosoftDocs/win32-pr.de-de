# <a name="d1152-command-list-closed-twice"></a>D1152: Befehlsliste zweimal geschlossen

Es wurde versucht, eine Befehlsliste ein zweites Mal zu schließen.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Nach dem Auflisten einer Befehlsliste muss sie nur einmal geschlossen werden. Direct2D betrachtet nachfolgende Aufrufe von ID2D1CommandList::Close als Fehler.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Um diese Meldung zu vermeiden, stellen Sie sicher, dass eine Befehlsliste nur einmal geschlossen wird. Wenn eine Anwendung Rückrufe oder rekursive Codepfade für die Interaktion mit Befehlslisten verwendet, stellen Sie sicher, dass das Verhalten korrekt ist.
