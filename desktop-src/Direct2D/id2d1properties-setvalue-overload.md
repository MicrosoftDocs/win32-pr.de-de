# <a name="id2d1propertiessetvalue-methods"></a>ID2D1Properties::SetValue-Methoden

Legt die entsprechende Eigenschaft nach Index fest.

### <a name="overload-list"></a>Überladeliste



| Methode                                                                                            | Beschreibung                                                                       |
|:--------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------|
| [**SetValue(U, const T&)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-setvalue(u_constt_))                                        | Legt die entsprechende Eigenschaft nach Index fest. Dies ist eine Vorlagenüberladung.<br/> |
| [**SetValue(U, const \* BYTE, UINT32)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-setvalue(u_constbyte_uint32))                            | Legt die entsprechende Eigenschaft nach Index fest. Dies ist eine Vorlagenüberladung.<br/> |
| [**SetValue(UINT32, const \* BYTE, UINT32)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-setvalue(uint32_constbyte_uint32))                        | Legt die entsprechende Eigenschaft nach Index fest.<br/>                              |
| [**SetValue(UINT32, D2D1 \_ PROPERTY \_ TYPE, const \* BYTE, UINT32)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-setvalue(uint32_d2d1_property_type_constbyte_uint32)) | Legt die entsprechende Eigenschaft nach Index fest.<br/>                              |
