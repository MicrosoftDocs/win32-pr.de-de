# <a name="d1146-invalid-render-controls"></a>D1146: Ungültige Rendersteuerelemente

Die angegebenen Renderingsteuerelemente hatten ungültige Parameter und werden ignoriert.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Diese Meldung kann während ID2D1DeviceContext::SetRenderingControls auftreten, wenn die Anwendung einen ungültigen Puffergenauigkeitswert oder eine Kachelgröße von 0 angibt.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Stellen Sie sicher, dass die Anwendung einen gültigen Puffergenauigkeitswert verwendet. Weitere Informationen finden Sie unter [**D2D1 \_ BUFFER \_ PRECISION**](/windows/desktop/api/D2d1_1/ne-d2d1_1-d2d1_buffer_precision). Stellen Sie sicher, dass die Anwendung eine Kachelgröße ungleich 0 (null) angibt.
