# <a name="d1146-invalid-render-controls"></a>D1146: Ungültige Rendering-Steuerelemente

Die angegebenen renderingsteuerelemente enthielten ungültige Parameter und werden ignoriert.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Diese Meldung kann während ID2D1DeviceContext:: setrenderingcontrols auftreten, wenn die Anwendung einen ungültigen Puffer Genauigkeits Wert oder eine Kachel Größe von 0 (null) angibt.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass die Anwendung einen gültigen Puffer Genauigkeits Wert verwendet. Siehe [**D2D1 \_ buffer \_ Precision**](/windows/desktop/api/D2d1_1/ne-d2d1_1-d2d1_buffer_precision). Stellen Sie sicher, dass die Anwendung eine Kachel Größe ungleich NULL angibt.
