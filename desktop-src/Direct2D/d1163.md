# <a name="d1163-print-bitmap-too-big"></a>D1163: Bitmap zu groß drucken

Bitmap ist zu groß für das Drucken. Erwägen Sie, sie in kleinere Kacheln zu unterlegen.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D Print erzwingt eine Begrenzung der Größe druckbarer Bitmaps. Dieser Grenzwert beträgt 65535 geräteabhängige Einheiten. Wenn die Anwendung eine Bitmap enthält, die diese Größe in druckbarem Inhalt überschreitet, gibt Direct2D einen Fehler aus, und die Debugebene gibt diese Meldung aus.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass bitmaps, die bei Druckvorgängen verwendet werden, die Größenbeschränkung von Direct2D Print nicht überschreiten. Erwägen Sie, die Bitmap in mehrere Kacheln zu partitionieren.
