# <a name="d1163-print-bitmap-too-big"></a>D1163: die Druck Bitmap ist zu groß.

Bitmap ist zu groß für den Druck, und Sie sollten es in kleineren Kacheln aufteilen.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D Print erzwingt eine Begrenzung der Größe von druckbaren Bitmaps. Dieser Grenzwert beträgt 65535 Geräte abhängige Einheiten. Wenn die Anwendung eine Bitmap enthält, die diese Größe in Druck barem Inhalt überschreitet, meldet Direct2D einen Fehler, und die debugschicht gibt diese Meldung aus.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass die bei Druck Vorgängen verwendeten Bitmaps die Größenbeschränkung des Direct2D-Drucks nicht überschreiten. Sie sollten die Bitmap in mehrere Kacheln aufteilen.
