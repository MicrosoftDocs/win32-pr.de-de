# <a name="direct2d-glossary"></a>Direct2D-Glossar

<dl> <dt>

<span id="direct2d.direct2d_glossary_a8_target"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_A8_TARGET"></span>**A8-Ziel**
</dt> <dd>

Ein Renderziel, das ein Pixelformat verwendet, das einen Alphakanal mit acht Bits darstellt. A8-Ziele sind nützlich, um Geometrien als Masken zu zeichnen.

</dd> <dt>

<span id="direct2d.direct2d_glossary_aliasing"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_ALIASING"></span>**Aliasing**
</dt> <dd>

In Computergrafiken die verknauste Darstellung von Kurven oder diagonalen Linien. Aliasing ist bei niedrigeren Auflösungen am sichtbarsten.

</dd> <dt>

<span id="direct2d.direct2d_glossary_antialiasing"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_ANTIALIASING"></span>**Antialiasing**
</dt> <dd>

Eine Technik zum Glätten der verkrümmten Darstellung von gekrümmten oder diagonalen Linien. Methoden zum Antialiasing umfassen umgebende Pixel mit zwischengeschalteten Schattierungen und die Bearbeitung der Größe und horizontalen Ausrichtung der Pixel.

</dd> <dt>

<span id="direct2d.direct2d_glossary_batch"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BATCH"></span>**Batch**
</dt> <dd>

Anforderungen oder Transaktionen, die gruppiert wurden.

</dd> <dt>

<span id="direct2d.direct2d_glossary_bitmap"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BITMAP"></span>**Bitmap**
</dt> <dd>

Eine Darstellung von Zeichen oder Grafiken nach einzelnen Pixeln. Die Pixel können horizontal, in Zeilen oder vertikal in Spalten angeordnet werden. Jedes Pixel kann durch ein oder mehrere Bits dargestellt werden.

</dd> <dt>

<span id="direct2d.direct2d_glossary_bits_per_pixel__bpp_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BITS_PER_PIXEL__BPP_"></span>**Bits pro Pixel (bpp)**
</dt> <dd>

Die Anzahl der Bits, die zum Speichern und Anzeigen der Farbdaten für ein einzelnes Pixel verwendet werden. Dies ist die Standardmessungseinheit für die Bittiefe (auch Farbtiefe genannt). Häufige Bittiefen sind 8, 16 und 32.

</dd> <dt>

<span id="direct2d.direct2d_glossary_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BRUSH"></span>**bürsten**
</dt> <dd>

Eine Direct2D-Ressource, die eine unendliche Ebene mit einer Volltonfarbe, einem Farbverlauf oder einer Bitmap zeichnet. Ein Pinsel wird in der Regel verwendet, um eine Geometrie zu strichen und zu füllen.

</dd> <dt>

<span id="direct2d.direct2d_glossary_cleartype"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_CLEARTYPE"></span>**Cleartype**
</dt> <dd>

Eine Schriftanzeigetechnologie, die die Schriftanzeigeauflösung erheblich verbessert, damit Buchstaben auf dem Computerbildschirm reibungslos und nicht verknackt angezeigt werden. ClearType verbessert die Darstellung von Text auf MONITOREN, die über eine digitale Schnittstelle verfügen, z. B. Laptopmonitore und hochwertige Flatpanel-Desktopmonitore.

</dd> <dt>

<span id="direct2d.direct2d_glossary_dc"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DC"></span>**Dc**
</dt> <dd>

Siehe Definition für: Gerätekontext.

</dd> <dt>

<span id="direct2d.direct2d_glossary_device_context__dc_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DEVICE_CONTEXT__DC_"></span>**Gerätekontext (DC)**
</dt> <dd>

Ein GDI-Gerätekontext. Siehe auch: Graphics Device Interface.

</dd> <dt>

<span id="direct2d.direct2d_glossary_direct2d"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECT2D"></span>**Direct2d**
</dt> <dd>

Eine hardwarebeschleunigte 2D-Rendering-API im unmittelbaren Modus.

</dd> <dt>

<span id="direct2d.direct2d_glossary_direct3d"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECT3D"></span>**Direct3D**
</dt> <dd>

Die Hardwarebeschleunigungsplattform und -runtime für 3D-Grafiken in Windows.

</dd> <dt>

<span id="direct2d.direct2d_glossary_directwrite"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECTWRITE"></span>**DirectWrite**
</dt> <dd>

Eine Text-API, die Textrendering, Schriftartverwaltung und Textlayoutunterstützung bietet, die unabhängig von einem bestimmten Renderingsystem ist.

</dd> <dt>

<span id="direct2d.direct2d_glossary_directx_graphics_infrastructure__dxgi_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECTX_GRAPHICS_INFRASTRUCTURE__DXGI_"></span>**DirectX Graphic Infrastructure (DXGI)**
</dt> <dd>

Die Infrastruktur, die die grafikbezogenen Aufgaben auf niedriger Ebene verwaltet, die unabhängig von der DirectX-Grafiklaufzeit sind. DXGI bietet ein allgemeines Framework für Grafikkomponenten.

</dd> <dt>

<span id="direct2d.direct2d_glossary_dxgi"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DXGI"></span>**Dxgi**
</dt> <dd>

Siehe Definition für: DirectX Graphic Infrastructure (DXGI).

</dd> <dt>

<span id="direct2d.direct2d_glossary_figure"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_FIGURE"></span>**Abbildung**
</dt> <dd>

Eine offene oder geschlossene Form, die durch einen Startpunkt und eine Auflistung von Segmenten definiert wird.

</dd> <dt>

<span id="direct2d.direct2d_glossary_flatten"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_FLATTEN"></span>**Glätten**
</dt> <dd>

Um eine polygonale Näherung einer (möglicherweise gekrümmten) Geometrie zu generieren.

</dd> <dt>

<span id="direct2d.direct2d_glossary_gdi"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GDI"></span>**Gdi**
</dt> <dd>

Siehe Definition für: Graphics Device Interface (GDI).

</dd> <dt>

<span id="direct2d.direct2d_glossary_geometric_mask"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GEOMETRIC_MASK"></span>**geometrische Maske**
</dt> <dd>

Ein Clip oder ein Ausschnitt, der durch ein ID2D1Geometry-Objekt definiert wird, das eine Ebene maskiert, wenn sie von einem Renderziel gezeichnet wird. Eine geometrische Maske kann entweder als Alias verwendet werden oder Ränder mit unter pixelbasierten Rändern haben.

</dd> <dt>

<span id="direct2d.direct2d_glossary_geometry_transform"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GEOMETRY_TRANSFORM"></span>**geometrietransformation**
</dt> <dd>

Eine Transformation, die auf eine Geometrie angewendet wird, bevor sie gestrichelt oder gefüllt wird.

</dd> <dt>

<span id="direct2d.direct2d_glossary_glyph"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GLYPH"></span>**Glyph**
</dt> <dd>

Eine grafische Darstellung eines Zeichens, eines Teils eines Zeichens oder einer Zeichensequenz.

</dd> <dt>

<span id="direct2d.direct2d_glossary_glyph_run"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GLYPH_RUN"></span>**Glyphenlauf**
</dt> <dd>

Eine fortlaufende Ausführung von Glyphen, die ein gemeinsames Format verwenden.

</dd> <dt>

<span id="direct2d.direct2d_glossary_gradient_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRADIENT_BRUSH"></span>**Farbverlaufspinsel**
</dt> <dd>

Ein Pinsel, der einen Bereich in einem allmählichen Verlauf von einer Farbe zur anderen oder von einem Farbton in einen anderen Schattierung derselben Farbe zeichnet.

</dd> <dt>

<span id="direct2d.direct2d_glossary_gradient_stop"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRADIENT_STOP"></span>**Farbverlaufsstopp**
</dt> <dd>

Eine bestimmte Farbe, die für eine Position innerhalb des Farbverlaufsbereichs definiert ist. Farbverlaufsstopps werden verwendet, um die Farbe an bestimmten Stellen entlang des Farbverlaufspfads zu definieren.

</dd> <dt>

<span id="direct2d.direct2d_glossary_grahics_primitive"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRAHICS_PRIMITIVE"></span>**grahics primitive**
</dt> <dd>

In Computergrafiken eine Form, z. B. eine Linie, ein Kreis, eine Kurve oder ein Polygon, die ein Grafikadapter als diskrete Entität zeichnen, speichern und bearbeiten kann.

</dd> <dt>

<span id="direct2d.direct2d_glossary_graphics_device_interface__gdi_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRAPHICS_DEVICE_INTERFACE__GDI_"></span>**Graphics Device Interface (GDI)**
</dt> <dd>

Eine Grafik-API, mit der Anwendungen Grafikbefehle an ein Anzeige- oder Druckgerät senden können, in der Regel ohne Berücksichtigung der Spezifikationen oder Funktionen. Eine Microsoft Win32-GDI empfängt Renderinganforderungen von Anwendungen und übergibt diese Anforderungen an die Kernelmodus-GDI.

</dd> <dt>

<span id="direct2d.direct2d_glossary_handle_to_a_device_context__hdc_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_HANDLE_TO_A_DEVICE_CONTEXT__HDC_"></span>**Handle für einen Gerätekontext (HDC)**
</dt> <dd>

Ein Verweis auf einen GDI-Gerätekontext.

</dd> <dt>

<span id="direct2d.direct2d_glossary_hdc"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_HDC"></span>**Hdc**
</dt> <dd>

Siehe Definition für: Handle für einen Gerätekontext.

</dd> <dt>

<span id="direct2d.direct2d_glossary_mesh"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_MESH"></span>**Masche**
</dt> <dd>

Eine Auflistung von Scheitelstellen, die eine Reihe von Dreiecken definieren, die eine Form bilden.

</dd> <dt>

<span id="direct2d.direct2d_glossary_multi-sample_anti-aliasing__msaa_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_MULTI-SAMPLE_ANTI-ALIASING__MSAA_"></span>**Multi-Sample Anti-Aliasing (MSAA)**
</dt> <dd>

Ein Antialiasingverfahren, das eine ganze Szene verarbeitet.

</dd> <dt>

<span id="direct2d.direct2d_glossary_opacity_mask"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_OPACITY_MASK"></span>**Deckkraftmaske**
</dt> <dd>

Eine von einem Pinsel oder einer Bitmap beschriebene Maske, die auf ein anderes Objekt angewendet wird, um dieses Objekt teilweise oder vollständig transparent zu machen. Eine Deckkraftmaske verwendet Alphakanalinformationen, um anzugeben, wie die Quellpixel des Objekts in das endgültige Zielziel gemischt werden. Die transparenten Teile der Maske geben die Bereiche an, in denen das zugrunde liegende Bild ausgeblendet ist, während die nicht transparenten Teile der Maske angeben, wo das maskierte Objekt zulässig ist.

</dd> <dt>

<span id="direct2d.direct2d_glossary_per_primitive_anti-aliasing__ppaa_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_PER_PRIMITIVE_ANTI-ALIASING__PPAA_"></span>**Pro primitivem Antialiasing (PPAA)**
</dt> <dd>

Ein Antialiasingverfahren, das auf jede einzelne Grafikprimitive und nicht auf eine gesamte Szene angewendet wird.

</dd> <dt>

<span id="direct2d.direct2d_glossary_radial_gradient_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_RADIAL_GRADIENT_BRUSH"></span>**Radialer Farbverlaufspinsel**
</dt> <dd>

Ein Farbverlauf, bei dem der Startpunkt durch einen Fokuspunkt und der Endpunkt durch einen Farbverlauf definiert wird.

</dd> <dt>

<span id="direct2d.direct2d_glossary_render_target"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_RENDER_TARGET"></span>**Renderziel**
</dt> <dd>

Eine Direct2D-Ressource, die Zeichnungsbefehle aus gibt. Es gibt verschiedene Arten von Renderzielen, die jeweils an ein anderes Ziel gerendert werden. Beispielsweise wird ein ID2D1HwndRenderTarget-Objekt in einem Fenster gerendert, und ein ID2D1BitmapRenderTarget-Objekt wird in einer Bitmap gerendert.

</dd> <dt>

<span id="direct2d.direct2d_glossary_segment"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_SEGMENT"></span>**Segment**
</dt> <dd>

Ein Teil eines geometrischen Pfads, der durch den Start- und Dendpunkte definiert wird.

</dd> <dt>

<span id="direct2d.direct2d_glossary_tessellate"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TESSELLATE"></span>**Tessellate**
</dt> <dd>

Um eine Form in eine Auflistung von Dreiecken zu unterbrechen.

</dd> <dt>

<span id="direct2d.direct2d_glossary_transform"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TRANSFORM"></span>**Verwandeln**
</dt> <dd>

Eine Matrix, die eine lineare Zuordnung in einer Reihe von Dimensionen darstellt. Direct2D verwendet eine 2 by 3-Matrix, die die API auf die affinen Affins beschränkt.

</dd> <dt>

<span id="direct2d.direct2d_glossary_translation"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TRANSLATION"></span>**Übersetzung**
</dt> <dd>

Der Prozess der Verwendung einer Transformationsmatrix zum Verschieben eines Objekts.

</dd> <dt>

<span id="direct2d.direct2d_glossary_vertex"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_VERTEX"></span>**Scheitelpunkt**
</dt> <dd>

Der höchste Punkt einer Kurve, der Punkt, an dem eine Kurve endet, oder der Punkt, an dem sich zwei Segmente in einem Polygon oder einem Freiformpfad treffen.

</dd> <dt>

<span id="direct2d.direct2d_glossary_vertex_shader"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_VERTEX_SHADER"></span>**Vertex-Shader**
</dt> <dd>

Ein Typ von programmierbarem Shader, der eine Reihe von Assemblyanweisungen enthält, die pro Scheitelpunkt und Pixel für den aktuellen Zeichnen-Aufruf auf der GPU ausgeführt werden. Ein Vertex-Shader überlastt intensive Berechnungen von der CPU auf die GPU.

</dd> <dt>

<span id="direct2d.direct2d_glossary_windows_imaging_component__wic_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_WINDOWS_IMAGING_COMPONENT__WIC_"></span>**Windows Imaging Component (WIC)**
</dt> <dd>

Eine API, die es Anwendungen ermöglicht, (1) jedes Bildformat anzuzeigen und zu bearbeiten, für das ein WIC-kompatibler CODEC installiert ist, und (2) Metadaten oder Bilddateien zu lesen und zu schreiben.

</dd> <dt>

<span id="direct2d.direct2d_glossary_xml_paper_specification__xps_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_XML_PAPER_SPECIFICATION__XPS_"></span>**XML Paper Specification (XPS)**
</dt> <dd>

Ein Dokumentformat, das von der XML Paper Specification beschrieben wird und zum Speichern von Dokumenten, zum Verarbeiten zum Drucken und zum Drucken verwendet werden kann.

</dd> <dt>

<span id="direct2d.direct2d_glossary_xps"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_XPS"></span>**Xps**
</dt> <dd>

Siehe Definition für: XML Paper Specification (XPS).

</dd> </dl>
