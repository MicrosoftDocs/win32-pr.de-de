# <a name="direct2d-glossary"></a>Direct2D Glossar

<dl> <dt>

<span id="direct2d.direct2d_glossary_a8_target"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_A8_TARGET"></span>**A8-Ziel**
</dt> <dd>

Ein Renderziel, das ein Pixel Format verwendet, das einen Alphakanal mit acht Bits darstellt. A8-Ziele sind nützlich zum Zeichnen von Geometrien als Masken.

</dd> <dt>

<span id="direct2d.direct2d_glossary_aliasing"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_ALIASING"></span>**Aliasing**
</dt> <dd>

In Computergrafiken die verzweigte Darstellung von Kurven oder diagonalen Linien. Aliasing ist am meisten bei niedrigeren Auflösungen sichtbar.

</dd> <dt>

<span id="direct2d.direct2d_glossary_antialiasing"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_ANTIALIASING"></span>**Antialiasing**
</dt> <dd>

Eine Technik zum Glätten der verzweigten Darstellung von gekrümmten oder diagonalen Linien. Zu den Methoden des Antialiasing zählen die umgebenden Pixel mit zwischen Schattierungen und die Bearbeitung der Größe und der horizontalen Ausrichtung der Pixel.

</dd> <dt>

<span id="direct2d.direct2d_glossary_batch"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BATCH"></span>**Batches**
</dt> <dd>

Anforderungen oder Transaktionen, die gruppiert wurden.

</dd> <dt>

<span id="direct2d.direct2d_glossary_bitmap"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BITMAP"></span>**Bitmap**
</dt> <dd>

Eine Darstellung von Zeichen oder Grafiken nach einzelnen Pixeln. Die Pixel können horizontal, in Zeilen oder vertikal in Spalten angeordnet werden. Jedes Pixel kann durch ein oder mehrere Bits dargestellt werden.

</dd> <dt>

<span id="direct2d.direct2d_glossary_bits_per_pixel__bpp_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BITS_PER_PIXEL__BPP_"></span>**Bits pro Pixel (BPP)**
</dt> <dd>

Die Anzahl der Bits, die zum Speichern und Anzeigen der Farbdaten für ein einzelnes Pixel verwendet werden. Dies ist die Standard Maßeinheit für die Bittiefe (auch als Farbtiefe bezeichnet). Allgemeine Bitraten sind 8, 16 und 32.

</dd> <dt>

<span id="direct2d.direct2d_glossary_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BRUSH"></span>**Zahn**
</dt> <dd>

Eine Direct2D-Ressource, die eine unendliche Ebene mit einer voll Tonfarbe, einem Farbverlauf oder einer Bitmap zeichnet. Ein Pinsel wird normalerweise verwendet, um eine Geometrie zu zeichnen und auszufüllen.

</dd> <dt>

<span id="direct2d.direct2d_glossary_cleartype"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_CLEARTYPE"></span>**ClearType**
</dt> <dd>

Eine Schriftart Anzeige Technologie, die die Auflösung der Schriftart Anzeige erheblich verbessert, sodass die Buchstaben auf dem Computerbildschirm glatt, nicht verzweigt angezeigt werden. ClearType verbessert die Darstellung von Text auf LCD-Monitoren mit digitaler Benutzeroberfläche, z. b. Laptop Monitoren und hochwertigen Flatpanel-Desktop Monitoren.

</dd> <dt>

<span id="direct2d.direct2d_glossary_dc"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DC"></span>**DC**
</dt> <dd>

Siehe Definition für: Gerätekontext.

</dd> <dt>

<span id="direct2d.direct2d_glossary_device_context__dc_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DEVICE_CONTEXT__DC_"></span>**Gerätekontext (DC)**
</dt> <dd>

Ein GDI-Gerätekontext. Siehe auch: Graphics Device Interface.

</dd> <dt>

<span id="direct2d.direct2d_glossary_direct2d"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECT2D"></span>**Direct2D**
</dt> <dd>

Eine hardwarebeschleunigte, 2D-Rendering-API im unmittelbaren Modus.

</dd> <dt>

<span id="direct2d.direct2d_glossary_direct3d"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECT3D"></span>**Direct3D**
</dt> <dd>

Die Hardware Beschleunigungs Plattform und-Laufzeit für 3D-Grafiken in Windows.

</dd> <dt>

<span id="direct2d.direct2d_glossary_directwrite"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECTWRITE"></span>**DirectWrite**
</dt> <dd>

Eine Text-API, die Text Rendering, Schriftart Verwaltung und Textlayoutunterstützung bereitstellt, die unabhängig von einem bestimmten Renderingsystem ist.

</dd> <dt>

<span id="direct2d.direct2d_glossary_directx_graphics_infrastructure__dxgi_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECTX_GRAPHICS_INFRASTRUCTURE__DXGI_"></span>**DirectX-Grafik Infrastruktur (DXGI)**
</dt> <dd>

Die Infrastruktur, mit der Grafik bezogene Aufgaben auf niedriger Ebene verwaltet werden, die unabhängig von der DirectX-Grafik Laufzeit sind. DXGI stellt ein gängiges Framework für Grafik Komponenten bereit.

</dd> <dt>

<span id="direct2d.direct2d_glossary_dxgi"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DXGI"></span>**DXGI**
</dt> <dd>

Siehe Definition für: DirectX Graphics Infrastructure (DXGI).

</dd> <dt>

<span id="direct2d.direct2d_glossary_figure"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_FIGURE"></span>**Re**
</dt> <dd>

Eine geöffnete oder geschlossene Form, die durch einen Startpunkt und eine Auflistung von Segmenten definiert ist.

</dd> <dt>

<span id="direct2d.direct2d_glossary_flatten"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_FLATTEN"></span>**vereinfachen**
</dt> <dd>

Zum Generieren einer polygonalen Näherung einer (möglicherweise gekrümmten) Geometrie.

</dd> <dt>

<span id="direct2d.direct2d_glossary_gdi"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GDI"></span>**GDI**
</dt> <dd>

Siehe Definition für: Graphics Device Interface (GDI).

</dd> <dt>

<span id="direct2d.direct2d_glossary_geometric_mask"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GEOMETRIC_MASK"></span>**geometrische Maske**
</dt> <dd>

Ein durch ein ID2D1Geometry-Objekt definiertes Clip-oder Ausschneide Element, das eine Ebene maskiert, wenn es von einem Renderziel gezeichnet wird. Bei einer geometrischen Maske kann es sich entweder um einen Alias oder um unter Pixel Ränder handeln.

</dd> <dt>

<span id="direct2d.direct2d_glossary_geometry_transform"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GEOMETRY_TRANSFORM"></span>**Geometrie Transformation**
</dt> <dd>

Eine Transformation, die auf eine Geometrie angewendet wird, bevor Sie gezeichnet oder ausgefüllt wird.

</dd> <dt>

<span id="direct2d.direct2d_glossary_glyph"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GLYPH"></span>**Symbol**
</dt> <dd>

Eine grafische Darstellung eines Zeichens, eines Teils eines Zeichens oder einer Sequenz von Zeichen.

</dd> <dt>

<span id="direct2d.direct2d_glossary_glyph_run"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GLYPH_RUN"></span>**Glyphe ausführen**
</dt> <dd>

Eine kontinuierliche Durchführung von Symbolen, die ein gemeinsames Format aufweisen.

</dd> <dt>

<span id="direct2d.direct2d_glossary_gradient_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRADIENT_BRUSH"></span>**Farbverlaufs Pinsel**
</dt> <dd>

Ein Pinsel, der einen Bereich in einer schrittweisen Fortschritt von einer Farbe in eine andere oder von einem Farbton zu einem anderen Farbton der gleichen Farbe zeichnet.

</dd> <dt>

<span id="direct2d.direct2d_glossary_gradient_stop"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRADIENT_STOP"></span>**Farbverlaufs Stopps**
</dt> <dd>

Eine bestimmte Farbe, die für eine Position innerhalb des Farbverlaufs Bereichs definiert ist. Farbverlaufs Stopps werden verwendet, um die Farbe an bestimmten Positionen entlang des Farbverlaufs Pfades zu definieren.

</dd> <dt>

<span id="direct2d.direct2d_glossary_grahics_primitive"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRAHICS_PRIMITIVE"></span>**grahics-primitiv**
</dt> <dd>

In Computergrafiken eine Form, z. b. eine Linie, ein Kreis, eine Kurve oder ein Polygon, die ein Grafikadapter als diskrete Entität zeichnen, speichern und bearbeiten kann.

</dd> <dt>

<span id="direct2d.direct2d_glossary_graphics_device_interface__gdi_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRAPHICS_DEVICE_INTERFACE__GDI_"></span>**Graphics Device Interface (GDI)**
</dt> <dd>

Eine Grafik-API, die es Anwendungen ermöglicht, Grafikbefehle an ein Anzeige-oder Druck Gerät zu senden, in der Regel ohne Berücksichtigung der Spezifikationen oder Funktionen. Ein Microsoft Win32-GDI empfängt Renderinganforderungen von Anwendungen und übergibt diese Anforderungen an den kernelmodusgdi.

</dd> <dt>

<span id="direct2d.direct2d_glossary_handle_to_a_device_context__hdc_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_HANDLE_TO_A_DEVICE_CONTEXT__HDC_"></span>**Handle für einen Gerätekontext (HDC)**
</dt> <dd>

Ein Verweis auf einen GDI-Gerätekontext.

</dd> <dt>

<span id="direct2d.direct2d_glossary_hdc"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_HDC"></span>**HDC**
</dt> <dd>

Siehe Definition für: Handle für einen Gerätekontext.

</dd> <dt>

<span id="direct2d.direct2d_glossary_mesh"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_MESH"></span>**schigen**
</dt> <dd>

Eine Auflistung von Scheitel Punkten, die eine Reihe von Dreiecken definieren, die eine Form bilden.

</dd> <dt>

<span id="direct2d.direct2d_glossary_multi-sample_anti-aliasing__msaa_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_MULTI-SAMPLE_ANTI-ALIASING__MSAA_"></span>**Multi-Sample Anti-Aliasing (MSAA)**
</dt> <dd>

Eine Antialiasing-Technik, die eine ganze Szene verarbeitet.

</dd> <dt>

<span id="direct2d.direct2d_glossary_opacity_mask"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_OPACITY_MASK"></span>**Deckkraft Maske**
</dt> <dd>

Eine von einem Pinsel oder einer Bitmap beschriebene Maske, die auf ein anderes Objekt angewendet wird, um dieses Objekt teilweise oder vollständig transparent zu machen. Eine Deck Kraft Maske verwendet Alphakanal Informationen, um anzugeben, wie die Quell Pixel des Objekts in das endgültige Ziel Ziel gemischt werden. Die transparenten Teile der Maske geben die Bereiche an, in denen das zugrunde liegende Bild ausgeblendet ist, während die nicht transparenten Teile der Maske angeben, wo das maskierte Objekt zulässig ist.

</dd> <dt>

<span id="direct2d.direct2d_glossary_per_primitive_anti-aliasing__ppaa_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_PER_PRIMITIVE_ANTI-ALIASING__PPAA_"></span>**Pro primitiver Antialiasing (ppaa)**
</dt> <dd>

Eine Antialiasing-Technik, die auf die einzelnen Grafik primitive anstelle einer ganzen Szene angewendet wird.

</dd> <dt>

<span id="direct2d.direct2d_glossary_radial_gradient_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_RADIAL_GRADIENT_BRUSH"></span>**Radialer Farbverlauf Pinsel**
</dt> <dd>

Ein Farbverlauf, bei dem der Startpunkt durch einen Fokuspunkt definiert wird und der Endpunkt durch einen Farbverlauf definiert wird.

</dd> <dt>

<span id="direct2d.direct2d_glossary_render_target"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_RENDER_TARGET"></span>**Renderziel**
</dt> <dd>

Eine Direct2D-Ressource, die Zeichnungs Befehle ausgibt. Es gibt verschiedene Arten von renderzielen, von denen jedes an ein anderes Ziel gerendert wird. Beispielsweise rendert ein ID2D1HwndRenderTarget-Rendering in ein Fenster, und ein ID2D1BitmapRenderTarget wird in eine Bitmap gerendert.

</dd> <dt>

<span id="direct2d.direct2d_glossary_segment"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_SEGMENT"></span>**Segment**
</dt> <dd>

Ein Teil eines geometrischen Pfads, der durch den Start-und Endpunkt definiert wird.

</dd> <dt>

<span id="direct2d.direct2d_glossary_tessellate"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TESSELLATE"></span>**& nbsp;**
</dt> <dd>

, Um eine Form in eine Auflistung von Dreiecken zu unterbrechen.

</dd> <dt>

<span id="direct2d.direct2d_glossary_transform"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TRANSFORM"></span>**Sin**
</dt> <dd>

Eine Matrix, die eine lineare Zuordnung in einer Reihe von Dimensionen darstellt. Direct2D verwendet eine 2 x 3-Matrix, die die API auf die affinkoformationen beschränkt, die affine sind.

</dd> <dt>

<span id="direct2d.direct2d_glossary_translation"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TRANSLATION"></span>**übersetzt**
</dt> <dd>

Der Prozess der Verwendung einer Transformationsmatrix zum Verschieben eines Objekts.

</dd> <dt>

<span id="direct2d.direct2d_glossary_vertex"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_VERTEX"></span>**Vertex**
</dt> <dd>

Der höchste Punkt einer Kurve, der Punkt, an dem eine Kurve endet, oder der Punkt, an dem zwei Segmente in einem Polygon-oder frei Hand Form Pfad entsprechen.

</dd> <dt>

<span id="direct2d.direct2d_glossary_vertex_shader"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_VERTEX_SHADER"></span>**Vertex-Shader**
</dt> <dd>

Ein Typ von programmierbarer Shader, der einen Satz von Assemblyanweisungen enthält, die auf der GPU pro Scheitelpunkt und pro Pixel für den aktuellen zeichnen-Befehl ausgeführt werden. Ein Vertex-Shader verlagert intensive Berechnungen von der CPU zu GPU.

</dd> <dt>

<span id="direct2d.direct2d_glossary_windows_imaging_component__wic_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_WINDOWS_IMAGING_COMPONENT__WIC_"></span>**Windows Imaging Component (WIC)**
</dt> <dd>

Eine API, die es Anwendungen ermöglicht (1), beliebige Bildformate anzuzeigen und zu bearbeiten, für die ein WIC-konformer Codec installiert ist, und (2) Metadaten oder Bilddateien zu lesen und zu schreiben.

</dd> <dt>

<span id="direct2d.direct2d_glossary_xml_paper_specification__xps_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_XML_PAPER_SPECIFICATION__XPS_"></span>**XML Paper Specification (XPS)**
</dt> <dd>

Ein Dokumentformat, das in der XML Paper Specification beschrieben wird, das zum Speichern von Dokumenten, zum Verarbeiten der Dokumente zum Drucken und zum Drucken verwendet werden kann.

</dd> <dt>

<span id="direct2d.direct2d_glossary_xps"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_XPS"></span>**XPS**
</dt> <dd>

Siehe Definition für: XML Paper Specification (XPS).

</dd> </dl>
