# <a name="d1238-source-transform-single-channel-srgb"></a>D1238: Quellen Transformation für einen einzelnen Kanal sRGB

ID2D1SourceTransform lässt nicht \_ \_ \_ zu, dass die sRGB-Genauigkeit für D2D1-Puffer mit der D2D1- \_ \_ Kanaltiefe 1 kombiniert wird \_ .






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat D2D1 \_ Puffer \_ Genauigkeit \_ sRGB und D2D1 \_ Channel \_ Tiefe \_ 1 auf einem ID2D1SourceTransform angegeben.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stattdessen sollte die Tiefe "D2D1 \_ Channel" \_ oder " \_ D2D1 \_ Channel \_ Tiefe 4" \_ verwendet werden.
