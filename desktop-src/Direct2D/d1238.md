# <a name="d1238-source-transform-single-channel-srgb"></a>D1238: Source Transform Single Channel SRGB

ID2D1SourceTransform lässt nicht zu, dass D2D1 BUFFER PRECISION SRGB mit \_ \_ \_ D2D1 \_ CHANNEL DEPTH \_ \_ 1 kombiniert wird.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat D2D1 \_ BUFFER \_ PRECISION \_ SRGB und D2D1 \_ CHANNEL DEPTH \_ \_ 1 für eine ID2D1SourceTransform angegeben.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stattdessen sollten D2D1 \_ CHANNEL DEPTH DEFAULT oder \_ \_ D2D1 \_ CHANNEL DEPTH \_ \_ 4 verwendet werden.
