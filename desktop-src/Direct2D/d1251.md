# <a name="d1251-d3d-state-issue"></a>D1251: D3D-Statusproblem

Der Zustand des Direct3D-Geräts hat sich geändert, als Direct2D es noch verwendet hat. Dies kann auf eine Threadingverletzung hindeuten, da jeweils nur ein Thread das Gerät verwenden kann.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung kann eine D2D-Factory mit einem einzelnen Thread gleichzeitig in zwei verschiedenen Threads verwenden.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Stellen Sie sicher, dass die Anwendung die Threadingrichtlinien von Direct2D ordnungsgemäß beachtet.
