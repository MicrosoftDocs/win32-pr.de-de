# <a name="d1251-d3d-state-issue"></a>D1251: D3D State-Problem

Der Status des Direct3D-Geräts wurde geändert, als Direct2D noch verwendet wurde. Dies kann ein Hinweis auf einen Threading Verstoß sein, da das Gerät jeweils nur von einem Thread verwendet werden kann.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung verwendet möglicherweise eine einzelne Thread D2D Factory auf zwei verschiedenen Threads gleichzeitig.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass die Anwendung renderingkonventionen Threading-Richtlinien ordnungsgemäß beachtet.
