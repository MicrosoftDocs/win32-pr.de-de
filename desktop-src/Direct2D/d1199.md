# <a name="d1199-changed-command-list-resource"></a>D1199: geänderte Befehlslisten Ressource

Ein Bild, das in eine Befehlsliste gezeichnet wurde, wurde soeben geändert. Dies kann unbeabsichtigte Folgen haben. Die Befehlsliste enthält nun das geänderte Bild und nicht den Inhalt des Bilds zum Zeitpunkt der Erstellung.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat der Befehlsliste ein Bild hinzugefügt und das Image später geändert.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Überprüfen Sie, ob das resultierende Verhalten erwartungsgemäß ist. Das Verhalten, bei dem es sich bei einer Befehlsliste um das Ziel handelt, führt zu unterschiedlichen Ergebnissen, als wenn eine Bitmap als Ziel festgelegt wurde. Wenn das Verhalten nicht erwartungsgemäß ist, kann die Anwendung das Image Klonen, bevor es aktualisiert wird.
