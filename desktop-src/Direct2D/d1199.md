# <a name="d1199-changed-command-list-resource"></a>D1199: Geänderte Befehlslistenressource

Ein Bild, das in eine Befehlsliste gezeichnet wird, wurde gerade geändert. Dies kann zu unbeabsichtigten Folgen führen. Die Befehlsliste enthält nun das geänderte Bild und nicht den Inhalt des Bilds zum Zeitpunkt der Zeichnete.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat einer Befehlsliste ein Image hinzugefügt und das Image später geändert.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass das resultierende Verhalten wie erwartet ist. Das Verhalten in diesem Fall, bei dem eine Befehlsliste das Ziel ist, führt zu anderen Ergebnissen als wenn eine Bitmap als Ziel festgelegt wurde. Wenn das Verhalten nicht wie erwartet ist, möchte die Anwendung das Image möglicherweise klonen, bevor es aktualisiert wird.
