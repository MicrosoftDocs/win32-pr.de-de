# <a name="d1118-target-not-a-bitmap"></a>D1118: Ziel keine Bitmap

Bei einem Zeichnungsvorgang ist ein Fehler aufgetreten, weil das Ziel NULL oder eine Befehlsliste ist.





 

## <a name="possible-causes"></a>Mögliche Ursachen

Eine wahrscheinliche Ursache für diese Meldung ist, dass die Anwendung GetDC oder ReleaseDC aufgerufen hat, während das Ziel des Gerätekontexts auf NULL oder auf eine Befehlsliste festgelegt wurde.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Wenn eine Anwendung GetDC oder ReleaseDC verwendet, das von IGdiInteropRenderTarget verfügbar gemacht wird, stellen Sie sicher, dass für den entsprechenden ID2D1DeviceContext eine gültige Bitmap als Ziel festgelegt ist.
