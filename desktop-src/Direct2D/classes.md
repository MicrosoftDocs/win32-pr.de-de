# <a name="direct2d-helper-classes"></a>Direct2D-Hilfsklassen

Der D2D1-Namespace definiert die folgenden Klassen.

## <a name="in-this-section"></a>In diesem Abschnitt



| Thema                                       | Beschreibung                                                                                                                              |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**ColorF**](/windows/win32/api/d2d1helper/nl-d2d1helper-colorf)<br/>         | Definiert eine Reihe von Konstanten, die bekannte Farben darstellt, und stellt Benutzerfreundlichkeitsmethoden zum Instanziieren verschiedener Farben zur Verfügung.<br/>  |
| [**Matrix3x2F**](/windows/win32/api/d2d1helper/nl-d2d1helper-matrix3x2f)<br/> | Die [**Matrix3x2F-Klasse**](/windows/win32/api/d2d1helper/nl-d2d1helper-matrix3x2f) stellt eine 3 x 2-Matrix dar und stellt Benutzerfreundlichkeitsmethoden zum Erstellen von Matrizen zur Verfügung.<br/> |
| [**Matrix4x3F**](matrix4x3f.md)<br/> | Die [**Matrix4x3F-Klasse**](matrix4x3f.md) stellt eine 4 x 3-Matrix dar und stellt Benutzerfreundlichkeitsmethoden zum Erstellen von Matrizen zur Verfügung.<br/> |
| [**Matrix4x4F**](/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f)<br/> | Die [**Matrix4x4F-Klasse**](/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f) stellt eine 4 x 4-Matrix dar und stellt Benutzerfreundlichkeitsmethoden zum Erstellen von Matrizen zur Verfügung.<br/> |
| [**Matrix5x4F**](matrix5x4f.md)<br/> | Die [**Matrix5x4F-Klasse**](matrix5x4f.md) stellt eine 5 x 4-Matrix dar und stellt Benutzerfreundlichkeitsmethoden zum Erstellen von Matrizen zur Verfügung.<br/> |
