# <a name="d1227-print-rasterization-event"></a>D1227: Druckrasterereignis

Diese Meldung wird vom Direct2D-Druck ausgegeben. Es gibt an, dass einer der folgenden Vorgänge eine Rasterung erforderlich macht:

-   Verwendung einer D2D1 \_ LAYER \_ OPTIONS1 \_ INITIALIZE \_ FROM \_ BACKGROUND-Ebene.
-   Verwendung des primitiven Blendmodus SOURCE COPY oder eines anderen Blendmodus als SOURCE OVER.
-   Verwendung bestimmter zusammengesetzter Modi wie BOUNDED \_ SOURCE \_ COPY, DESTINATION \_ OUT, SOURCE \_ ATOP, XOR, PLUS oder INVERT.
-   Verwendung von [**DrawGdiMetafile**](id2d1devicecontext-drawgdimetafile-overload.md).
-   Verwendung einer 4x4-Transformation (oder Perspektive) in einem [**DrawBitmap-Aufruf.**](id2d1devicecontext-drawbitmap-overload.md)

Um eine optimale Leistung zu erzielen, sollten diese Vorgänge nach Möglichkeit vermieden werden.
