# <a name="d1227-print-rasterization-event"></a>D1227: druckrasterisierungsereignis

Diese Meldung wird von Direct2D Print ausgegeben. Dies deutet darauf hin, dass einer der folgenden Vorgänge einen Bedarf an einer rasterisierung verursacht hat:

-   Verwendung einer D2D1 \_ Ebene \_ OPTIONS1 \_ initialisieren \_ von der \_ Hintergrund Ebene.
-   Verwendung der primitiven Quell Kopie des Blend-Modus oder eines anderen Blend-Modus als der Quell Kopie.
-   Verwendung bestimmter zusammengesetzter Modi, wie z \_ . b. begrenzte Quell \_ Kopie, \_ zielout, Quelle oberhalb \_ , Xor, plus oder Invert.
-   Verwendung von [**drawgdimetafile**](id2d1devicecontext-drawgdimetafile-overload.md).
-   Verwendung einer 4X4-Transformation (oder einer Perspektive) in einem [**drawbitmap**](id2d1devicecontext-drawbitmap-overload.md) -Befehl.

Um eine optimale Leistung zu erzielen, sollten diese Vorgänge nach Möglichkeit vermieden werden.
