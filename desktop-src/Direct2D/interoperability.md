# <a name="interoperability"></a>Interoperabilität

In den Themen in diesem Abschnitt wird beschrieben, wie Direct2D mit anderen Systemen (z. b. GDI, Direct3D und WIC) interagiert.

> [!Note]  
> Ab Windows 8 stellt Direct2D die [**ID2D1DeviceContext**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1devicecontext) -Schnittstelle bereit, die eine einfache Möglichkeit zur Interoperabilität bietet. Weitere Informationen finden Sie im Thema [Geräte-und Geräte Kontexte](devices-and-device-contexts.md) .
