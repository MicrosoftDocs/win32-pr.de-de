# <a name="d1148-invalid-operation-on-closed-print-control"></a>D1148: Ungültiger Vorgang für das geschlossene Drucksteuerelement

Direct2D Print: [**AddPage**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-addpage) oder [**Close**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-close) wird aufgerufen, nachdem [**ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol) geschlossen wurde.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Diese Meldung kann auftreten, wenn eine Anwendung ID2D1PrintControl::AddPage auf einem Drucksteuerelement aufruft, das geschlossen wurde. Direct2D betrachtet Drucksteuerelementobjekte nach dem Schließen auf ähnliche Weise wie Pfadgeometrien und Befehlslisten als unveränderlich.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Stellen Sie sicher, dass die Anwendung nicht versucht, einem geschlossenen Drucksteuerelement Seiten hinzuzufügen. Ändern Sie die Anwendungslogik so, dass alle erforderlichen Seiten an ein Drucksteuerelement gesendet werden, bevor es geschlossen wird.
