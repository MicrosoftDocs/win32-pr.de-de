# <a name="d1148-invalid-operation-on-closed-print-control"></a>D1148: Ungültiger Vorgang für das geschlossene Druck Steuerelement

Direct2D Print: [**addPage**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-addpage) oder [**Close**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-close) wird aufgerufen, nachdem [**ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol) geschlossen wurde.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Diese Meldung kann auftreten, wenn eine Anwendung in einem Druck Steuerelement, das geschlossen wurde, ID2D1PrintControl:: addPage aufruft. Direct2D betrachtet Druck Steuerungs Objekte so, dass Sie nach dem Schließen nicht änderbar sind, und zwar auf ähnliche Weise wie Pfad Geometrien und Befehlslisten.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass die Anwendung nicht versucht, Seiten zu einem geschlossenen Druck Steuerelement hinzuzufügen. Ändern Sie die Anwendungslogik so, dass alle erforderlichen Seiten an ein Druck Steuerelement gesendet werden, bevor Sie geschlossen wird.
