# <a name="d1188-cpu-read-and-target-bitmap-options-are-incompatible"></a>D1188: CPU-Lese- und Zielbitmapoptionen sind inkompatibel

D2D1 \_ BITMAP \_ OPTIONS \_ CPU \_ READ is not compatible with D2D1 \_ BITMAP \_ OPTIONS \_ TARGET.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat eine inkompatible Kombination der BITMAP \_ OPTIONS-Flags von D2D1 \_ angegeben.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Entfernen Sie D2D1 \_ BITMAP OPTIONS CPU READ oder \_ \_ \_ D2D1 \_ BITMAP OPTIONS TARGET aus den \_ \_ Flags. Wenn die Anwendung beide Optionen benötigt, sollte sie zwei separate Bitmaps erstellen.
