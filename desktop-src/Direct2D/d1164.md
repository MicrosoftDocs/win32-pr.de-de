# <a name="d1164-invalid-map-option"></a>D1164: Ungültige Karten Option.

D2D1 \_ map- \_ Optionen "None" \_ ist kein gültiger Parameter für die Zuordnung.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung möglicherweise aus, wenn eine Anwendung ID2D1Bitmap1:: Map mit \_ der \_ Option D2D1 Map Options None aufruft \_ .

## <a name="possible-fixes"></a>Mögliche Korrekturen

Verwenden Sie beim Aufrufen von ID2D1Bitmap1:: map eine gültige map-Option. Unter D2D1 \_ map \_ options finden Sie eine Beschreibung der Zuordnungs Optionen. Stellen Sie sich vor, ob die Anwendung Lese-oder Schreibzugriff benötigt.
