# <a name="d1158-pixels-unit-mode-unsupported-with-print"></a>D1158: Pixels Unit Mode Unsupported with Print

Direct2D-Druck: Nicht unterstützter SetUnitMode(PIXELS)-Befehl war in der zum Drucken gesendeten Befehlsliste vorhanden.





 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D Print lässt den Einheitenmodus PIXELS nicht zu. Der Standardeinheitsmodus ist DIPS.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Um diese Nachricht zu adressen, sollten Sie den Gerätekontext-Einheitenmodus nicht auf PIXELS festlegen, während Sie druckbaren Inhalt erstellen. Wenn es erforderlich ist, eine globale Skalierungstransformation anzuwenden, die sich auf alle primitiven Zeichnungsvorgänge auswirken soll, sollten Sie überlegen, ob ein gleichwertiger Aufruf von ID2D1RenderTarget::SetDpi oder ID2D1RenderTarget::SetTransform ausreichend ist.
