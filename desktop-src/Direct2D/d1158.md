# <a name="d1158-pixels-unit-mode-unsupported-with-print"></a>D1158: Pixels Unit Mode Unsupported with Print

Direct2D Print: Der nicht unterstützte SetUnitMode(PIXELS)-Befehl war in der Befehlsliste vorhanden, die zum Drucken gesendet wurde.





 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D Print lässt keinen Einheitenmodus von PIXEL zu. Der Standardeinheitsmodus ist DIPS.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Um diese Meldung zu beheben, sollten Sie beim Erstellen druckbarer Inhalte nicht den Einheitenmodus des Gerätekontexts auf PIXELS festlegen. Wenn eine globale Skalierungstransformation angewendet werden muss, die sich auf alle primitiven Zeichnungsvorgänge auswirkt, sollten Sie überlegen, ob ein entsprechender Aufruf von ID2D1RenderTarget::SetDpi oder ID2D1RenderTarget::SetTransform ausreichend ist.
