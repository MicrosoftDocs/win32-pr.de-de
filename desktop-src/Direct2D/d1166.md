# <a name="d1166-discard-map-option-requires-write-map-option"></a>D1166: die Zuordnungs Option "verwerfen" erfordert die Option "Map

D2D1 \_ map- \_ Optionen \_ verwerfen müssen mit D2D1 \_ map- \_ Optionen \_ Schreiben kombiniert werden.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung möglicherweise aus, wenn eine Anwendung ID2D1Bitmap1:: Map mit D2D1 \_ map- \_ Optionen \_ verwerfen, aber nicht D2D1 \_ map- \_ Optionen schreiben aufruft \_ .

## <a name="possible-fixes"></a>Mögliche Korrekturen

Die Option verwerfen erfordert den Schreibvorgang, um einen nützlichen Vorgang für eine Anwendung zu erstellen. \_ \_ Eine Beschreibung der Karten Optionen finden Sie unter D2D1 Map Options. Wenn Sie ID2D1Bitmap1:: Map aufrufen, sollten Sie überprüfen, welcher Zugriffstyp für die Anwendung erforderlich ist.
