# <a name="d1166-discard-map-option-requires-write-map-option"></a>D1166: Die Option "Karte verwerfen" erfordert die Option "Zuordnung schreiben".

D2D1 \_ MAP OPTIONS DISCARD muss mit \_ \_ D2D1 \_ MAP OPTIONS WRITE kombiniert \_ \_ werden.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung möglicherweise aus, wenn eine Anwendung ID2D1Bitmap1::Map with D2D1 \_ MAP \_ OPTIONS \_ DISCARD, aber nicht D2D1 \_ MAP OPTIONS WRITE \_ \_ aufruft.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Die Discard-Option erfordert WRITE, um ein nützlicher Vorgang für eine Anwendung zu sein. Eine Beschreibung der Kartenoptionen finden Sie unter D2D1 \_ MAP \_ OPTIONS (D2D1-KARTENOPTIONEN). Berücksichtigen Sie beim Aufrufen von ID2D1Bitmap1::Map, welche Art von Zugriff die Anwendung benötigt.
