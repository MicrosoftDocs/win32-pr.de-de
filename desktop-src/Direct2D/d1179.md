# <a name="d1179-gradient-stop-collection-zero-stops"></a>D1179: Auflistung der Farbverlaufs Stopps NULL endet

Zum Erstellen eines ID2D1GradientStopCollection ist mindestens ein Farbverlaufs Ende erforderlich.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung mit dem Namen "ID2D1RenderTarget:: Auflistungs Stopp", aber hat die Anzahl der Farbverlaufs Stopps 0 angegeben.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Beim Erstellen eines ID2D1GradientStopCollection sollte in der Anwendung mindestens ein Farbverlaufs Ende mit Farbe und Position angegeben werden. Ein Beispiel finden Sie unter ID2D1RenderTarget:: foategradientstopcollection.
