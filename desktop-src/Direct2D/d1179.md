# <a name="d1179-gradient-stop-collection-zero-stops"></a>D1179: Gradient Stop Collection Zero Stops

Zum Erstellen einer ID2D1GradientStopCollection ist mindestens ein Farbverlaufsstopp erforderlich.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung mit dem Namen ID2D1RenderTarget::CreateGradientStopCollection hat jedoch eine Gradientenstoppanzahl von 0 angegeben.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Beim Erstellen einer ID2D1GradientStopCollection sollte die Anwendung mindestens einen Farbverlaufsstopp mit Farbe und Position angeben. Ein Beispiel finden Sie unter ID2D1RenderTarget::CreateGradientStopCollection.
