# <a name="matrix4x3f-class"></a>Matrix4x3F-Klasse

Die **Matrix4x3F-Klasse** stellt eine 4:3-Matrix dar und stellt praktische Methoden zum Erstellen von Matrizen bereit.

## <a name="members"></a>Member

Die **Matrix4x3F-Klasse** erbt von [**D2D1 \_ MATRIX \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f). **Matrix4x3F** verfügt auch über diese Typen von Membern:

-   [Konstruktoren](#constructors)

### <a name="constructors"></a>Konstruktoren

Die **Matrix4x3F-Klasse** verfügt über diese Konstruktoren.



| Konstruktor                                                                                                                                                                                           | BESCHREIBUNG                                                                                                                        |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**Matrix4x3F()**](matrix4x3f-matrix4x3f--.md)                                                                                                                                                       | Instanziiert eine neue Instanz einer nicht initialisierten **Matrix4x3F-Klasse.**<br/>                                                   |
| [**Matrix4x3F(FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT) (FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT)**](matrix4x3f-matrix4x3f-floats-.md) | Instanziiert eine neue Instanz einer **Matrix4x3F-Klasse,** die mit allen Gleitkommamatrixwerten initialisiert wird.<br/> |
