# <a name="matrix4x3f-class"></a>Matrix4x3F-Klasse

Die **Matrix4x3F** -Klasse stellt eine 4 x 3-Matrix dar und bietet Hilfsmethoden zum Erstellen von Matrizen.

## <a name="members"></a>Member

Die **Matrix4x3F** -Klasse erbt von [**D2D1 \_ Matrix \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f). **Matrix4x3F** verfügt auch über die folgenden Typen von Membern:

-   [Konstruktoren](#constructors)

| [**Matrix4x3F()**](matrix4x3f-matrix4x3f--.md)                                                                                                                                                       | Instanziiert eine neue Instanz einer nicht initialisierten **Matrix4x3F** -Klasse.<br/>                                                   |
