# <a name="d1153-closed-command-lists-cannot-be-set-as-targets"></a>D1153: Geschlossene Befehlslisten können nicht als Ziele festgelegt werden

Es wurde versucht, eine geschlossene Befehlsliste für ein Ziel zu setzen.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Nachdem eine Befehlsliste geschlossen wurde, sollte eine Anwendung nicht versuchen, sie als Ziel zu setzen. Direct2D betrachtet diesen Aufruf von ID2D1DeviceContext::SetTarget als Fehler, da sich der Inhalt einer Befehlsliste nach dem Schließen nicht ändern kann. Das Festlegen einer Befehlsliste als Ziel gibt an, dass sie gerendert werden soll.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Um diese Meldung zu adressieren, rufen Sie ID2D1DeviceContext::SetTarget nicht mit einer geschlossenen Befehlsliste auf. Wenn eine Befehlsliste mit neuem, unterschiedlichen Inhalt verwendet werden muss, erstellen Sie eine neue Befehlsliste.
