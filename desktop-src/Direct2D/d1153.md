# <a name="d1153-closed-command-lists-cannot-be-set-as-targets"></a>D1153: geschlossene Befehlslisten können nicht als Ziele festgelegt werden.

Es wurde versucht, eine geschlossene Befehlsliste für ein Ziel festzulegen.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Nachdem eine Befehlsliste geschlossen wurde, sollte eine Anwendung nicht versuchen, Sie als Ziel festzulegen. Direct2D betrachtet diesen Befehl an ID2D1DeviceContext:: SetTarget als Fehler, da der Inhalt einer Befehlsliste nicht geändert werden kann, nachdem er geschlossen wurde. Das Festlegen einer Befehlsliste als Ziel gibt eine Absicht an, Sie zu rendieren.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Um diese Meldung zu beheben, müssen Sie ID2D1DeviceContext:: SetTarget nicht mit einer Befehlsliste abrufen, die geschlossen wurde. Wenn es erforderlich ist, eine Befehlsliste mit neuen, unterschiedlichen Inhalten zu verwenden, erstellen Sie eine neue Befehlsliste.
