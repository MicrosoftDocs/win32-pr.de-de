# <a name="d1133-buffer-precision-unknown-not-allowed"></a>D1133: die Puffer Genauigkeit ist unbekannt, nicht zulässig.

Die D2D1- \_ Puffer \_ Genauigkeit \_ ist unbekannt und nicht als Parameter für diese API zulässig.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D ermöglicht es einer Anwendung, eine gewünschte Puffer Genauigkeit für einige Vorgänge anzugeben. Beispielsweise wird die Genauigkeit, die für die Imaging-Pipeline verwendet wird, mithilfe von ID2D1DeviceContext:: * Trend deringcontrols angegeben. Der \_ Spezifizierer "puffergenauigkeits unbekannter Wert" \_ ermöglicht einer Anwendung das Festlegen dieser Einstellung, und Direct2D verwendet einen Standardwert. dieser Spezifizierer ist jedoch für einige Vorgänge nicht zulässig. Dies ist insbesondere bei der Erstellung einer Auflistung von Farbverlaufs Stopps möglicherweise nicht zulässig.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass der \_ \_ unbekannter Spezifizierer für die Puffer Genauigkeit nicht mit ID2D1DeviceContext:: foategradientstopcollection verwendet wird. Verwenden Sie einen bekannten Genauigkeits Wert. Verwenden Sie ggf. die Genauigkeit des Ziel Oberflächen Formats für den Vorgang.
