# <a name="id2d1devicecreateprintcontrol-methods"></a>ID2D1Device::CreatePrintControl-Methoden

Erstellt ein [**ID2D1PrintControl-Objekt,**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol) das in [**ID2D1CommandList**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1commandlist) gespeicherte [Direct2D-Primitive](/windows/desktop/direct2d/direct2d-portal) in eine feste Seitendarstellung konvertiert. Das Druckuntersystem verwendet dann die Primitive.
