# <a name="id2d1devicecreateprintcontrol-methods"></a>ID2D1Device:: kreateprintcontrol-Methoden

Erstellt ein [**ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol) -Objekt, das in [**ID2D1CommandList**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1commandlist) gespeicherte [Direct2D](/windows/desktop/direct2d/direct2d-portal) -primitive in eine festgelegte Seitendarstellung konvertiert. Das Druck Subsystem verwendet dann die primitiven.
