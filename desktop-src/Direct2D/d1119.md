# <a name="d1119-bitmap-bound-as-target"></a>D1119: Bitmap als Ziel gebunden

Ein Zeichnungs Vorgang ist fehlgeschlagen, da die Ziel Bitmap und eine Quell Bitmap identisch waren.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Eine wahrscheinliche Ursache für diese Meldung ist, dass die Anwendung versucht hat, einen Zeichnungs Vorgang auszuführen, bei dem dieselbe Bitmap sowohl für die Quelle als auch für das Ziel verwendet wurde. Semantisch ist dies für interbetriebene Direct3D-Texturen mit Bindungs \_ -Shader \_ -Ressource und Bindungs \_ - \_ Renderziel möglich.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass die Anwendung nicht versucht, die Zielbitmap als Quelle eines Zeichnungs Vorgangs, einschließlich der Effekte, zu verwenden. In einigen Fällen weist diese Meldung möglicherweise darauf hin, dass die Anwendung das Ziel des Geräte Kontexts auf eine andere Bitmap als beabsichtigt festgelegt hat.
