# <a name="d1176-negative-dpi"></a>D1176: negativer dpi-Wert

Beim Erstellen eines renderTarget-Werts ist ein Fehler aufgetreten, wenn ein negativer dpi-Wert angegeben wird.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Eine Anwendung hat versucht, einen negativen dpi-Wert anzugeben, entweder durch Verwendung von ID2D1RenderTarget:: setdpi oder durch Erstellen eines Ziels mit einer negativen dpi-Eigenschaft.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Wählen Sie gültige, nicht negative Werte für dpi beim Erstellen oder Ändern von renderzielen aus. Wenn ein Standard-dpi-Wert zulässig ist, sollten Sie setdpi nicht aufrufen oder beim Erstellen eines Ziels eine dpi-Option angeben. Die Angabe eines dpi-Wert von 0 entspricht der Verwendung einer Standard-dpi.
