# <a name="d1176-negative-dpi"></a>D1176: NegativeR DPI

Es ist ein Fehler, beim Erstellen eines Renderziels einen negativen DPI anzugeben.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Eine Anwendung hat versucht, einen negativen DPI anzugeben, entweder durch Verwendung von ID2D1RenderTarget::SetDpi oder durch Erstellen eines Ziels mit einer negativen DPI-Eigenschaft.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Wählen Sie beim Erstellen oder Ändern von Renderzielen gültige, nicht negative Werte für DPI aus. Wenn ein Standard-DPI akzeptabel ist, können Sie nicht SetDpi aufrufen oder beim Erstellen eines Ziels eine DPI-Option angeben. Die Angabe eines DPI von 0 entspricht der Verwendung eines Standard-DPI.
