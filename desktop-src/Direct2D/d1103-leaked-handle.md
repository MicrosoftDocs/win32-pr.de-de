# <a name="d1103-leaked-handle"></a>D1103: Handle mit Verlust

Es wurde eine Schnittstellen \[ *Schnittstelle* \] erstellt, aber nicht freigegeben.

## <a name="placeholders"></a>Platzhalter

<dl> <dt>

<span id="interface"></span><span id="INTERFACE"></span>*berfläche*

Die Adresse der-Schnittstelle.

</dd> </dl> 




 

## <a name="possible-causes"></a>Mögliche Ursachen

Ungültige Ressourcenverwendung. Eine Anwendung kann eine Schnittstelle nicht freigeben, wenn Sie nicht mehr verwendet wird.
