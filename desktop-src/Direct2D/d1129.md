# <a name="d1129-dxgi-format-incompatible-with-alpha-mode"></a>D1129: DXGI-Format ist mit dem Alpha-Modus nicht kompatibel

Die Kombination aus dem Alpha Modus und dem angegebenen DXGI-Format ist nicht miteinander kompatibel.



|             |         |
|-------------|---------|
| Fehlerstufe | Warnung |



 

## <a name="possible-causes"></a>Mögliche Ursachen

Bei Direct2D-Vorgängen, die ein Oberflächen Format und einen Alpha-Modus verwenden, gibt es Fälle, in denen das Oberflächen Format und der Alpha Modus nicht übereinstimmen. Beispielsweise wäre ein Oberflächen Format wie das DXGI- \_ Format \_ a8 \_ unorm nicht für die Verwendung mit Ignore Alpha sinnvoll, da es nur in einen Alphakanal gerendert wird. Außerdem wird mit einigen DXGI-Formaten "Straight Alpha" nicht unterstützt \_ , aber ein prämultipliziertes Alpha ist.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Informationen dazu, welche Format Kombinationen unterstützt werden und welche nicht, finden Sie [unter Unterstützte Pixel Formate und Alpha-Modi](supported-pixel-formats-and-alpha-modes.md) . Ändern Sie die Anwendung so, dass Sie eine unterstützte Format Kombination verwendet.
