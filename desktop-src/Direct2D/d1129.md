# <a name="d1129-dxgi-format-incompatible-with-alpha-mode"></a>D1129: DXGI-Format inkompatibel mit Alphamodus

Die bereitgestellte Kombination aus Alphamodus und DXGI-Format ist nicht kompatibel.





 

## <a name="possible-causes"></a>Mögliche Ursachen

Bei Direct2D-Vorgängen, die ein Oberflächenformat und einen Alphamodus verwenden, gibt es Fälle, in denen das Oberflächenformat und der Alphamodus nicht zustimmen. Beispielsweise wäre ein Oberflächenformat wie DXGI FORMAT A8 UNORM nicht sinnvoll, mit IGNORE alpha verwendet zu werden, da es nur in einem \_ \_ \_ Alphakanal gerendert wird. Darüber hinaus wird bei einigen DXGI FORMATs kein gerades Alpha \_ unterstützt, bei prämultipliierten Alphas jedoch.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Anleitungen [dazu, welche Formatkombinationen](supported-pixel-formats-and-alpha-modes.md) unterstützt werden und welche nicht, finden Sie unter Unterstützte Pixelformate und Alphamodi. Ändern Sie die Anwendung so, dass eine unterstützte Formatkombination verwendet wird.
