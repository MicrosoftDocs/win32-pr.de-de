# <a name="d1113-device-needs-bgra-flag"></a>D1113: Gerät benötigt BGRA-Flag

Das Direct3D-Gerät wurde nicht mit d3d10 \_ Create \_ Device \_ BGRA \_ -Unterstützung erstellt und ist daher nicht mit Direct2D kompatibel.



|             |         |
|-------------|---------|
| Fehlerstufe | Warnung |



 

## <a name="possible-causes"></a>Mögliche Ursachen

Das Direct3D-Gerät wurde nicht mit d3d10 \_ \_ \_ BGRA- \_ Unterstützung für das Erstellen von Geräten erstellt.
