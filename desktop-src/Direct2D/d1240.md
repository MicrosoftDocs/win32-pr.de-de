# <a name="d1240-source-tranform-still-mapped"></a>D1240: Quelltranform noch zugeordnet

Die an ID2D1SourceTransform::D-Aw übergebene Bitmap wurde nie aufgehoben.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung namens ID2D1Bitmap1::Map hat ID2D1Bitmap1::Unmap nicht für eine Bitmap aufgerufen, die an ID2D1SourceTransform::D raw übergeben wurde.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass Unmap aufgerufen wird, wenn Map aufgerufen wird.
