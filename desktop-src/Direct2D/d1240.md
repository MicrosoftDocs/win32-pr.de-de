# <a name="d1240-source-tranform-still-mapped"></a>D1240: das Quell-tranformular ist noch zugeordnet.

Die an ID2D1SourceTransform::D RAW nicht zugeordnete Bitmap.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung mit dem Namen ID2D1Bitmap1:: Map hat ID2D1Bitmap1:: unmap nicht in einer Bitmap aufgerufen, die an ID2D1SourceTransform::D RAW übergeben wurde.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass unmap aufgerufen wird, wenn Map aufgerufen wird.
