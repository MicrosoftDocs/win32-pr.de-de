# <a name="d1171-redundant-call-to-map-bitmap"></a>D1171: redundanter Aufrufe der Karten Bitmap

Es wurde versucht, eine Bitmap zuzuordnen, die bereits zugeordnet ist.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung möglicherweise aus und meldet einen Fehler, wenn eine Anwendung versucht, eine Bitmap zuzuordnen, die bereits mit ID2D1Bitmap1:: Map zugeordnet wurde.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie auf dieselbe Weise, wie ID2D1RenderTarget:: EndDraw den Aufrufen von ID2D1RenderTarget:: beginDraw folgt, sicher, dass ID2D1Bitmap1:: unmap den Aufrufen von ID2D1Bitmap1:: Map folgt. Wenn eine Anwendung zum Zuordnen von Bitmaps Rückrufe oder rekursive Codepfade verwendet, stellen Sie sicher, dass das zugehörige Verhalten korrekt ist.
