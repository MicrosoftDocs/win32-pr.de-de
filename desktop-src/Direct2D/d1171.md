# <a name="d1171-redundant-call-to-map-bitmap"></a>D1171: Redundanter Aufruf von Map Bitmap

Es wurde versucht, eine Bitmap zuzuordnen, die bereits zugeordnet ist.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D kann diese Meldung ausgeben und einen Fehler melden, wenn eine Anwendung versucht, eine Bitmap zuzuordnen, die bereits mit ID2D1Bitmap1::Map zugeordnet wurde.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Stellen Sie auf die gleiche Weise wie ID2D1RenderTarget::EndDraw Aufrufe von ID2D1RenderTarget::BeginDraw sicher, dass ID2D1Bitmap1::Unmap Den Aufrufen von ID2D1Bitmap1::Map folgt. Wenn eine Anwendung Rückrufe oder rekursive Codepfade verwendet, um Bitmaps zuzuordnen, stellen Sie sicher, dass ihr Verhalten richtig ist.
