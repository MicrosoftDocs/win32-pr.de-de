# <a name="d1139-unused-stroke-width"></a>D1139: nicht verwendete Strichbreite

Es wurde ein Strich Transformations Typen von "haarline" angegeben, aber die angegebene Strichbreite war nicht "1". Die Strichbreite wird ignoriert.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat einen primitiven mithilfe eines ID2D1StrokeStyle mit D2D1 \_ Stroke \_ Transform \_ Type- \_ Haarlinie gezeichnet, die Strichbreite, die in diesem zeichnen-Befehl verwendet wurde, war jedoch nicht 1. Wenn das Attribut "hairline" angegeben wird, überschreibt es tatsächlich die Strichbreite und behebt es bei 1,0 geräteabhängigen Einheiten, unabhängig von der Skalierungs Transformation oder der dpi-Auswahl.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Wenn das Attribut "hairline" für ein ID2D1StrokeStyle festgelegt ist, sollte die Anwendung eine Strichbreite von 1,0 Einheiten verwenden.
