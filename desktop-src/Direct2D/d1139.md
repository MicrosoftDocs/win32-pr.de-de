# <a name="d1139-unused-stroke-width"></a>D1139: Nicht verwendete Strichbreite

Ein Strichtransformationstyp von HAIRLINE wurde angegeben, aber die angegebene Strichbreite war nicht 1. Die Strichbreite wird ignoriert.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung erstellt ein Primitives mit einem ID2D1StrokeStyle mit D2D1 \_ STROKE \_ TRANSFORM TYPE \_ \_ HAIRLINE, aber die in diesem Draw-Aufruf verwendete Strichbreite war nicht 1. Wenn das Attribut für die Strichlinie angegeben wird, überschreibt es die Strichbreite und korrigiert sie bei geräteabhängigen Einheiten von 1,0, unabhängig von der Skalierungstransformation oder der DPI-Auswahl.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Wenn das ATTRIBUT "HAIRLINE" für "ID2D1StrokeStyle" festgelegt ist, sollte die Anwendung eine Strichbreite von 1,0 Einheiten verwenden.
