# <a name="d1126-alpha-mode-unknown-not-allowed"></a>D1126: Unbekannter Alphamodus nicht zulässig

D2D1 \_ ALPHA MODE UNKNOWN ist für diese API nicht als Parameter \_ \_ zulässig.





 

## <a name="possible-causes"></a>Mögliche Ursachen

Der ALPHA \_ MODE \_ UNKNOWN-Bezeichner ist in einigen Kontexten zulässig, z. B. in einer Formatbeschreibung für ID2D1RenderTarget::CreateCompatibleRenderTarget. Ähnlich wie DXGI \_ FORMAT \_ UNKNOWN erfordert Direct2D jedoch in der Regel, dass Formate vollständig angegeben werden.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Berücksichtigen Sie bei der Entscheidung für einen Formatbezeichner die Behandlung von Alpha auf der Oberfläche, um einen \_ anderen ALPHA-MODUS als ALPHA MODE UNKNOWN zu \_ \_ verwenden. Verwenden Sie D2D1 \_ ALPHA \_ MODE \_ IGNORE, wenn die Oberfläche kein Alpha verwendet. Siehe Hinweise in [**D2D1 \_ ALPHA \_ MODE**](/windows/desktop/api/dcommon/ne-dcommon-d2d1_alpha_mode).
