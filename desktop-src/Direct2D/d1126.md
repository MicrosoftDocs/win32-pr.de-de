# <a name="d1126-alpha-mode-unknown-not-allowed"></a>D1126: Unbekannter Alpha Modus ist nicht zulässig.

Der D2D1- \_ alpha \_ Modus \_ Unknown ist nicht als Parameter für diese API zulässig.



|             |         |
|-------------|---------|
| Fehlerstufe | Warnung |



 

## <a name="possible-causes"></a>Mögliche Ursachen

Der \_ unbekannte Alpha Modus- \_ Spezifizierer ist in einigen Kontexten zulässig, wie z. b. eine Formatbeschreibung für ID2D1RenderTarget:: up-compatiblerendertarget. Ähnlich wie beim DXGI- \_ Format " \_ unknown" erfordert Direct2D im Allgemeinen, dass Formate vollständig angegeben werden.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Berücksichtigen Sie bei der Entscheidung für einen Format Bezeichner, dass Alpha in der Oberfläche für die Verwendung eines anderen Alpha Modus als "unbekannt" verwendet wird \_ \_ \_ . Verwenden Sie \_ den D2D1 Alpha \_ Modus \_ Ignore, wenn die Oberfläche nicht alpha verwendet. Weitere Informationen finden Sie unter Hinweise im [**D2D1 \_ alpha- \_ Modus**](/windows/desktop/api/dcommon/ne-dcommon-d2d1_alpha_mode).
