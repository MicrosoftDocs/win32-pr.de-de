# <a name="d1126-alpha-mode-unknown-not-allowed"></a>D1126: Alphamodus Unbekannt nicht zulässig

D2D1 \_ ALPHA MODE UNKNOWN ist als Parameter für diese API nicht \_ \_ zulässig.





 

## <a name="possible-causes"></a>Mögliche Ursachen

Der ALPHA MODE UNKNOWN-Bezeichner ist in einigen Kontexten zulässig, z. B. in einer Formatbeschreibung für \_ \_ ID2D1RenderTarget::CreateCompatibleRenderTarget. Ähnlich wie DXGI FORMAT UNKNOWN erfordert Direct2D jedoch im Allgemeinen, \_ dass Formate vollständig angegeben \_ werden.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Berücksichtigen Sie bei der Entscheidung für einen Formatbezeichner die Behandlung von Alpha auf der Oberfläche, um einen anderen \_ ALPHA-MODUS als \_ ALPHA-MODUS \_ UNKNOWN zu verwenden. Verwenden Sie D2D1 \_ ALPHA \_ MODE \_ IGNORE, wenn die Oberfläche kein Alpha verwendet. Weitere Informationen finden Sie unter Hinweise im [**D2D1 \_ \_ ALPHA-MODUS.**](/windows/desktop/api/dcommon/ne-dcommon-d2d1_alpha_mode)
