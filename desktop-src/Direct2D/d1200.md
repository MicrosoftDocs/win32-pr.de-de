# <a name="d1200-pushlayer-with-multisampled-target"></a>D1200: PushLayer mit multisampled Target

Pushlayer/Poplayer wurde mit einem Multisampled-Ziel verwendet. Der Aufruf wird weiterhin ausgegeben, aber das gerenderte Ergebnis stellt möglicherweise nicht das erwartete Multisamplemuster dar.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat eine Multisampled Direct3D-Textur als Direct2D-Ziel interoperiert und pushLayer auf dem Ziel aufgerufen.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Vermeiden Sie die Interoperabilität von Texturen mit mehrerenAmpeln, wenn die Ebenenfunktionalität wichtig ist. Die Verwendung von Ebenen oder Clips mit diesen Ressourcentypen kann zu Leistungseinbußen oder unerwarteten Renderingergebnissen führt. Verwenden Sie nach Möglichkeit eine alternative Möglichkeit zum Steuern der Deckkraft oder Maskierung anstelle von Ebenen. Alternativ können Sie eine Zwischenressource anstelle einer Multisampled-Ressource verwenden. Wenn die Multisamplingressource für das Antialiasing von 2D-Inhalten vorgesehen ist, sollten Sie berücksichtigen, ob das integrierte Pro-Primitive-Antialiasing von Direct2D ausreichend ist.
