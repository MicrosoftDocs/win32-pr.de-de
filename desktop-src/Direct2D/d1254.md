# <a name="d1254-operation-on-single-threaded-domain"></a>D1254: Vorgang in einer Singlethreaddomäne

Ein Aufruf von [**ID2D1Multithread::Enter**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-enter) oder [**ID2D1Multithread::Leave**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-leave) wurde in einer Singlethreadfactorydomäne durchgeführt.

Dies kann auftreten, wenn die Anwendung versehentlich die D2D-Factory mit [**D2D1 FACTORY TYPE MULTI THREADED erstellt \_ \_ \_ \_ hat.**](/windows/desktop/api/d2d1/ne-d2d1-d2d1_factory_type) In diesem Fall wird der Aufruf ignoriert.
