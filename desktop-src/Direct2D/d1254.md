# <a name="d1254-operation-on-single-threaded-domain"></a>D1254: Vorgang für Single Thread-Domäne

Ein [**ID2D1Multithread:: Enter**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-enter) oder [**ID2D1Multithread:: Leave**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-leave) wurde für eine Single Thread-Factory-Domäne aufgerufen.

Dies kann vorkommen, wenn die Anwendung die D2D-Factory versehentlich mit dem [**D2D1 \_ Factory- \_ Typ \_ \_ Multithread**](/windows/desktop/api/d2d1/ne-d2d1-d2d1_factory_type)erstellt hat. In diesem Fall wird der-Rückruf ignoriert.
