# <a name="d1155-fillopacitymask-requires-aliased-rendering"></a>D1155: fillopacitymask erfordert ein Alias Rendering

Fillopacitymask erfordert, dass der AntiAlias Modus auf "D2D1 \_ Antialias \_ Mode Aliasing" festgelegt ist \_ .



|             |             |
|-------------|-------------|
| Fehlerstufe | Information |



 

## <a name="possible-causes"></a>Mögliche Ursachen

Wenn eine Anwendung ID2D1RenderTarget:: fillopacitymask aufruft, sollte der antialiasmodus des Beteiligten renderTarget den D2D1- \_ Antialias \_ Modus-Alias aufweisen \_ . Das Standardverhalten eines Renderziels ist die Verwendung von Antialiasing. Daher muss ID2D1RenderTarget:: setantialiasmode aufgerufen werden, bevor eine Deck Kraft Maske aufgefüllt wird.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Verwenden Sie ID2D1RenderTarget:: setantialiasmode, um sicherzustellen, dass renderTarget das Rendern von Aliasing beim Auffüllen einer Deck Kraft Maske verwendet. Wenn nachfolgende Vorgänge Antialiasing erfordern, legen Sie den antialiasmodus des Ziels auf den D2D1- \_ Antialias \_ Modus \_ pro \_ primitiv fest. Stellen Sie sicher, dass die Aufrufe an ID2D1RenderTarget:: setantialiasmode und ID2D1RenderTarget:: fillopacitymask für dasselbe Ziel ausgegeben werden.
