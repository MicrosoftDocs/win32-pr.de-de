# <a name="d1203-copyfrombitmap-with-multisampled-source"></a>D1203: copyfrombitmap mit Multisampling-Quelle

Copyfrombitmap wurde mit einer Multisampling-Quell Bitmap verwendet. Der-Rückruf wird weiterhin ausgegeben, das kopierte Ergebnis stellt jedoch möglicherweise nicht das erwartete Multisampling-Muster dar.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat eine Multisampling-Direct3D-Textur als Direct2D-Bitmap interagiert und in der Bitmap "copyfrombitmap" genannt.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Vermeiden Sie das interoperieren von Multisampling-Texturen als Bitmaps, wenn die copyfrombitmap-Funktionalität wichtig ist. Das kopierte Ergebnis stellt möglicherweise nicht das erwartete Multisampling-Muster dar.
