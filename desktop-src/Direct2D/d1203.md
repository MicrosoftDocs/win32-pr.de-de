# <a name="d1203-copyfrombitmap-with-multisampled-source"></a>D1203: CopyFromBitmap With Multisampled Source

CopyFromBitmap wurde mit einer Quellbitmap mit mehrerenAmpeln verwendet. Der Aufruf wird weiterhin ausgegeben, aber das kopierte Ergebnis stellt möglicherweise nicht das erwartete Multisamplemuster dar.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung interoperierte eine Direct3D-Textur mit mehrerenAmpeln als Direct2D-Bitmap und hat copyFromBitmap für die Bitmap aufgerufen.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Vermeiden Sie die Interoperabilität von Mehrfachamplertexturen als Bitmaps, wenn die CopyFromBitmap-Funktionalität wichtig ist. Das kopierte Ergebnis stellt möglicherweise nicht das erwartete Multisamplemuster dar.
