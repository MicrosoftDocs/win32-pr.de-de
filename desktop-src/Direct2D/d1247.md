# <a name="d1247-dynamic-copy-from-memory"></a>D1247: Dynamisches Kopieren aus dem Arbeitsspeicher

CopyFromMemory kann keinen Arbeitsspeicher in eine Bitmap kopieren, die aus einer dynamischen Textur erstellt wurde.





 

## <a name="possible-causes"></a>Mögliche Ursachen

CopyFromMemory wurde für eine Bitmap aufgerufen, die als D3D11 USAGE DYNAMIC erstellt \_ \_ wurde.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Verwenden Sie stattdessen Map und Unmap für diesen Bitmaptyp.
