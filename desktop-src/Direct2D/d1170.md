# <a name="d1170-invalid-flags-with-dynamic-texture-bitmap"></a>D1170: ungültige Flags mit dynamischer Textur Bitmap.

D2D1 \_ map- \_ Optionen \_ Schreiben D2D1 Zuordnungs \| \_ \_ Optionen \_ verwerfen ist erforderlich, wenn die aus einer dynamischen Textur erstellte Bitmap zugeordnet wird.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung möglicherweise aus, wenn eine Anwendung ID2D1Bitmap1:: Map mit einer Textur aufruft, deren zugrunde liegende Direct3D-Ressource mit der dynamischen D3D11-Verwendung attributiert ist \_ \_ , aber nicht beide D2D1 \_ map \_ \_ -Optionen Schreib-und D2D1 \_ map-Optionen zum Verwerfen von \_ \_ Flags angegeben hat

## <a name="possible-fixes"></a>Mögliche Korrekturen

Wenn eine Anwendung den Zugriff auf eine Bitmap mithilfe von ID2D1Bitmap1:: Map erfordert und die zugrunde liegende Direct3D-Ressource mit der dynamischen D3D11-Verwendung attributiert ist \_ \_ , geben Sie die \_ Optionen D2D1 Map \_ options \_ Write und D2D1 \_ map \_ Optionen Verwerfen von Flags an \_ . \_ \_ Eine Beschreibung der Karten Optionen finden Sie unter D2D1 Map Options.
