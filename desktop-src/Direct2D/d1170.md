# <a name="d1170-invalid-flags-with-dynamic-texture-bitmap"></a>D1170: Ungültige Flags mit dynamischer Texturbitmap

D2D1 \_ MAP OPTIONS WRITE \_ \_ \| D2D1 MAP OPTIONS DISCARD ist erforderlich, wenn eine \_ aus einer dynamischen Textur erstellte Bitmap zugeordnet \_ \_ wird.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D kann diese Meldung ausgeben, wenn eine Anwendung ID2D1Bitmap1::Map mit einer Textur aufruft, deren zugrunde liegende Direct3D-Ressource mit D3D11 USAGE DYNAMIC attributiert \_ \_ ist, aber nicht sowohl D2D1 \_ MAP OPTIONS \_ \_ WRITE- als auch D2D1 \_ MAP OPTIONS \_ \_ DISCARD-Flags angegeben hat.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Wenn eine Anwendung zugriff auf eine Bitmap mit ID2D1Bitmap1::Map benötigt und ihre zugrunde liegende Direct3D-Ressource mit D3D11 USAGE DYNAMIC attributiert \_ \_ ist, geben Sie sowohl die DISCARD-Flags D2D1 \_ MAP OPTIONS WRITE und \_ \_ D2D1 \_ MAP OPTIONS \_ \_ an. Eine Beschreibung der Kartenoptionen finden Sie unter D2D1 \_ \_ MAP OPTIONS (D2D1-KARTENOPTIONEN).
