# <a name="d1124-dxgi-format-unknown-with-null-target-bitmap"></a>D1124: DXGI-Format unbekannt mit NULL-Ziel Bitmap

Das DXGI- \_ Format ' \_ unknown ' ist nicht als Parameter für diese API zulässig, wenn das Ziel auf NULL festgelegt ist oder es sich um eine Befehlsliste handelt.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Bei Vorgängen, wie z. b. ID2D1RenderTarget:: up-compatiblerendertarget, muss dem quellrenderziel eine Zielbitmap zugewiesen sein. Ohne eine Ziel Bitmap kann Direct2D nicht ermitteln, welches Oberflächen Format die neue renderzielressource enthalten wird.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Um dies zu beheben, stellen Sie sicher, dass dem entsprechenden ID2D1DeviceContext eine Ziel Bitmap mithilfe von ID2D1DeviceContext:: SetTarget zugewiesen ist. Wenn die Anwendung lediglich die Fähigkeit zum Renderingvorgang auf zwei verschiedenen Ziel Oberflächen benötigt, empfiehlt es sich, ein ID2D1Bitmap zu erstellen und es als Ziel des Geräte Kontexts festzulegen, anstatt ID2D1RenderTarget:: deatecompatiblerendertarget zu verwenden.
