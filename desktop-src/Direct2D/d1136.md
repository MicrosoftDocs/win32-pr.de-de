# <a name="d1136-text-rendering-mode-conflict"></a>D1136: Konflikt im Textrenderingmodus

Der DWRITE \_ RENDERING MODE und der \_ D2D1 \_ TEXT \_ ANTIALIAS MODE, die für dieses Renderziel festgelegt sind, sind \_ nicht miteinander kompatibel.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Bei textbasierten Vorgängen wie ID2D1RenderTarget::D rawGlyphRun und ID2D1DeviceContext::GetGlyphRunWorldBounds berücksichtigt Direct2D die aktuelle Text antialiasing-Option des Gerätekontexts sowie die DirectWrite-Antialiasingoption. Antialiasingsemantik unterscheidet sich zwischen Direct2D und DirectWrite. Diese Meldung tritt auf, wenn die beiden angegebenen Antialiasingoptionen nicht zustimmen.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Um sicherzustellen, dass der Direct2D-Text-Antialiasingmodus mit dem DirectWrite stimmt, stellen Sie sicher, dass sie die folgenden Anforderungen erfüllen: • Für D2D1 TEXT ANTIALIAS MODE CLEARTYPE ist der DirectWrite-Renderingmodus möglicherweise nicht OUTLINE oder \_ \_ \_ \_ ALIASED.• Für D2D1 TEXT ANTIALIAS MODE GRAYSCALE ist der DirectWrite-Renderingmodus möglicherweise nicht \_ \_ \_ \_ ALIASED.• Für D2D1 \_ TEXT ANTIALIAS MODE ALIASED \_ \_ ist der \_ DirectWrite-Renderingmodus möglicherweise keine der CLEARTYPE-Optionen. \[newline \] Um den aktuellen Direct2D-Text-Antialiasingmodus zu ändern, verwenden Sie ID2D1RenderTarget::SetTextAntialiasMode.
