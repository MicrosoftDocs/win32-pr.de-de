# <a name="d1168-write-map-option-requires-cpu-writable-bitmap"></a>D1168: die Schreib Zuordnungs Option erfordert eine CPU-beschreibbare Bitmap.

D2D1 \_ map- \_ Optionen \_ schreiben ist nur für eine CPU-beschreibbare Bitmap gültig.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung möglicherweise aus, wenn eine Anwendung ID2D1Bitmap1:: Map mit dem Write-Attribut der D2D1 \_ map \_ -Optionen aufruft \_ , aber die Bitmap nicht CPU-beschreibbar ist.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Wenn eine Anwendung Schreibzugriff mithilfe von ID2D1Bitmap1:: Map erfordert, stellen Sie sicher, dass die zugrunde liegende Direct3D-Ressource über das D3D11 \_ CPU \_ Access \_ Write-Attribut verfügt.
