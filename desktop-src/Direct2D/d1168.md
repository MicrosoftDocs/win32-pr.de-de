# <a name="d1168-write-map-option-requires-cpu-writable-bitmap"></a>D1168: Option "Zuordnung schreiben" erfordert cpu-schreibbare Bitmap

D2D1 \_ MAP OPTIONS WRITE ist nur für eine \_ \_ CPU-schreibbare Bitmap gültig.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung möglicherweise aus, wenn eine Anwendung ID2D1Bitmap1::Map mit dem WRITE-Attribut D2D1 MAP OPTIONS aufruft, die Bitmap jedoch nicht \_ \_ \_ CPU-schreibbar ist.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Wenn eine Anwendung Schreibzugriff mithilfe von ID2D1Bitmap1::Map erfordert, stellen Sie sicher, dass die zugrunde liegende Direct3D-Ressource über das ATTRIBUT D3D11 \_ CPU \_ ACCESS WRITE \_ verfügt.
