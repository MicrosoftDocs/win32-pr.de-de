# <a name="d1230-unknown-buffer-precision-support"></a>D1230: Unbekannte Puffer Genauigkeits Unterstützung

Ein isbufferprecisionsupported-isbufferprecisupported-Befehl wurde mit der D2D1- \_ Puffer \_ Genauigkeit \_ unbekannt gemacht. Diese API gibt false zurück, obwohl \_ die D2D1-Puffer \_ Genauigkeit \_ Unknown möglicherweise als Eingabe für einige APIs gültig ist.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Ein isbufferprecisionsupported-isbufferprecisupported-Befehl wurde mit der D2D1- \_ Puffer \_ Genauigkeit \_ unbekannt gemacht.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Bestimmen Sie, \_ ob \_ die Genauigkeit des D2D1-Puffers \_ Unknown eine gültige Puffer Genauigkeit für eine bestimmte API ist, indem Sie die Dokumentation für diese API lesen, anstatt zu versuchen, diese Programm gesteuert zu ermitteln.
