# <a name="d1230-unknown-buffer-precision-support"></a>D1230: Unterstützung für unbekannte Puffergenauigkeit

IsBufferPrecisionSupported wurde mit D2D1 \_ BUFFER \_ PRECISION UNKNOWN \_ aufgerufen. Diese API gibt false zurück, obwohl D2D1 \_ BUFFER PRECISION UNKNOWN möglicherweise als Eingabe für einige \_ \_ APIs gültig ist.






 

## <a name="possible-causes"></a>Mögliche Ursachen

IsBufferPrecisionSupported wurde mit D2D1 \_ BUFFER \_ PRECISION UNKNOWN \_ aufgerufen.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Bestimmen Sie, ob D2D1 \_ BUFFER PRECISION UNKNOWN eine gültige \_ \_ Puffergenauigkeit für eine bestimmte API ist, indem Sie die Dokumentation für diese API lesen, anstatt zu versuchen, diese programmgesteuert zu bestimmen.
