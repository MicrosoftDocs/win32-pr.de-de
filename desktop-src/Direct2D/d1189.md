# <a name="d1189-cpu-read-bitmap-option-was-used-incorrectly"></a>D1189: die CPU-Lese Bitmap-Option wurde falsch verwendet.

D2D1 \_ \_ bitmapoptionen der \_ CPU-Lesevorgang \_ muss kombiniert werden D2D1 \_ Bitmap- \_ Optionen \_ können nicht \_ gezeichnet werden.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat eine inkompatible Kombination der D2D1 \_ Bitmap \_ options-Flags angegeben.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Entfernen Sie eines der Flags, um einen gültigen Satz von D2D1 \_ Bitmap-Optionen zu erstellen \_ .
