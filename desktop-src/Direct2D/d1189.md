# <a name="d1189-cpu-read-bitmap-option-was-used-incorrectly"></a>D1189: CPU Read Bitmap Option Was Used Incorrectly

D2D1 \_ BITMAP \_ OPTIONS \_ CPU \_ READ must be combined with D2D1 \_ BITMAP \_ OPTIONS \_ CANNOT \_ DRAW






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat eine inkompatible Kombination der BITMAP \_ OPTIONS-Flags von D2D1 \_ angegeben.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Entfernen Sie eines der Flags, um einen gültigen Satz von D2D1 \_ BITMAP OPTIONS zu \_ erstellen.
