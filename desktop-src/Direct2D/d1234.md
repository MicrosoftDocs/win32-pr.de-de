# <a name="d1234-null-resource-brush"></a>D1234: NULL-Ressourcen Pinsel

Ein Bitmap-Pinsel oder Bild Pinsel kann nicht zum Zeichnen verwendet werden, wenn das zugehörige Bitmap/Bild auf NULL festgelegt ist.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat versucht, einen primitiven mithilfe eines ID2D1BitmapBrush oder ID2D1ImageBrush zu erzeugen, dessen Bitmap oder Image Null war.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass Bitmap-oder Image-Pinsel über eine gültige Bitmap oder ein gültiges Bild verfügen, bevor Sie Sie verwenden. Die Anwendung kann dies zur Erstellungszeit oder später mithilfe von ID2D1BitmapBrush:: SetBitmap oder ID2D1ImageBrush:: SetImage durchführen. Direct2D lässt NULL Pinsel Ressourcen zu, sodass Anwendungen möglicherweise die Freiheit haben, Pinsel Ressourcen freizugeben, nachdem Sie zum Rendering verwendet wurden. Anwendungen können dies tun, wenn die Bitmap-oder Image-Ressource besonders kostspielig ist, und das Szenario erfordert, dass Sie sofort freigegeben wird. Wenn eine Pinsel Ressource auf NULL festgelegt wird, wird verhindert, dass Direct2D einen ausstehenden Verweis darauf beibehält. Wenn eine Anwendung NULL eine Pinsel Ressource zuweist, sollte Sie nicht versuchen, den Pinsel zu verwenden, bis eine neue, gültige Ressource zugewiesen wurde.
