# <a name="d1234-null-resource-brush"></a>D1234: NULL-Ressourcenpinsel

Ein Bitmappinsel oder Bildpinsel kann nicht zum Zeichnen verwendet werden, wenn die zugeordnete Bitmap bzw. das zugehörige Bild auf NULL festgelegt ist.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat versucht, einen Primitiven mit einem ID2D1BitmapBrush oder ID2D1ImageBrush zu rendern, dessen Bitmap oder Bild NULL war.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Stellen Sie sicher, dass Bitmap- oder Bildpinsel ihnen eine gültige Bitmap oder ein gültiges Bild zugewiesen sind, bevor Sie sie verwenden. Die Anwendung kann dies zur Erstellungszeit oder später mit ID2D1BitmapBrush::SetBitmap oder ID2D1ImageBrush::SetImage tun. Direct2D lässt NULL-Pinselressourcen zu, sodass Anwendungen die Möglichkeit haben können, Pinselressourcen freizugeben, nachdem sie zum Rendern verwendet wurden. Anwendungen können dies tun, wenn die Bitmap- oder Bildressource besonders teuer ist und das Szenario erfordert, dass sie sofort freigegeben wird. Das Festlegen einer Pinselressource auf NULL verhindert, dass Direct2D einen ausstehenden Verweis darauf beibehält. Wenn eine Anwendung jedoch eine Pinselressource NULL zuweist, sollte sie erst dann versuchen, den Pinsel zu verwenden, wenn ihr eine neue gültige Ressource zugewiesen wurde.
