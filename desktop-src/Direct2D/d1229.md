# <a name="d1229-d1229-immersive-call-to-getdesktopdpi"></a>D1229: D1229: Immersiver Aufruf von GetDesktopDpi

ID2D1Factory::GetDesktopDpi wurde in einer Windows Store-App aufgerufen. Diese Vorgehensweise wird nicht empfohlen. Stattdessen sollte DisplayProperties::LogicalDpi verwendet werden.






 

## <a name="possible-causes"></a>Mögliche Ursachen

ID2D1Factory::GetDesktopDpi wurde in einer Windows Store-App aufgerufen.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

DisplayProperties::LogicalDpi sollte verwendet werden, um den DPI in einer Windows Store-App abzurufen.
