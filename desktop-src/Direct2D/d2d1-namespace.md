# <a name="d2d1-namespace"></a>D2D1-Namespace

Der D2D1-Namespace stellt Hilfsklassen, Funktionen und Strukturen bereit, die allgemeine Aufgaben vereinfachen, z. B. das Erstellen einer Identitätsmatrix oder das Erstellen einer Farbe.
