# <a name="d2d1_matrix_5x4_f"></a>D2D1 \_ MATRIX \_ 5X4 \_ F

Stellt eine 5-mal-4-Matrix dar.


```C++
typedef D2D_MATRIX_5X4_F D2D1_MATRIX_5X4_F;
```



## <a name="remarks"></a>Hinweise

**D2D1 \_ MATRIX \_ 5X4** ist ein neuer Name für die [**D2D \_ MATRIX \_ 5X4 \_ F-Struktur.**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f) Eine Liste der von der Matrix bereitgestellten Felder finden Sie unter [**D2D \_ MATRIX \_ 5X4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f).
