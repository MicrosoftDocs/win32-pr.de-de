# <a name="direct2d-samples"></a>Direct2D-Beispiele

Die folgenden Beispiele veranschaulichen die Direct2D-API.

## <a name="windows-10-and-microsoft-visual-studio-2015-samples"></a>Windows 10 und Microsoft Visual Studio 2015-Beispiele

| Beispiel | BESCHREIBUNG |
|-|-|
| [Beispiel für benutzerdefinierte Direct2D-Bildeffekte](https://github.com/Microsoft/Windows-universal-samples/tree/master/Samples/D2DCustomEffects) | In diesem Beispiel wird veranschaulicht, wie Benutzerdefinierte [Effekte mithilfe](custom-effects.md) von HLSL-Pixeln, Scheitelpunkt und Compute-Shadern implementiert werden. |
| [Beispiel für ein Direct2D-Farbverlaufsgitter](https://github.com/Microsoft/Windows-universal-samples/tree/master/Samples/D2DGradientMesh) | In diesem Beispiel wird veranschaulicht, wie ein Farbverlaufsgitter in Direct2D instanziiert und gerendert wird. In diesem Beispiel wird die von Direct2D bereitgestellte Hilfsmethode verwendet, um ein Gitternetz zu erstellen, das aus zwei Tensorpatches besteht, die eine Seite gemeinsam nutzen. |
| [Beispiel für eine Direct2D-Fotoanpassung](https://github.com/Microsoft/Windows-universal-samples/tree/master/Samples/D2DPhotoAdjustment) | In diesem Beispiel wird veranschaulicht, wie Sie mit Direct2D und Direct2D Effects einen Fotoanzeiger und -editor erstellen. |
