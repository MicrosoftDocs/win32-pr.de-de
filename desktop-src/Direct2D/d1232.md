# <a name="d1232-32bpp-rgba-causes-conversion"></a>D1232: 32bpp RGBA verursacht Konvertierung

Erwägen Sie die Verwendung von BGR anstelle von RGB für 32bpp-Formate, damit der BitmapSource-Effekt zusätzliche Formatkonvertierungen vermeiden kann.
