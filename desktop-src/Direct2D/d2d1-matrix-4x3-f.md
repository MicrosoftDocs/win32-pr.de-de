# <a name="d2d1_matrix_4x3_f"></a>D2D1 \_ Matrix \_ 4X3 \_ F

Stellt eine 4 x 3-Matrix dar.


```C++
typedef D2D_MATRIX_4X3_F D2D1_MATRIX_4X3_F;
```



## <a name="remarks"></a>Bemerkungen

**D2D1 \_ Matrix \_ 4X3** ist ein neuer Name für die [**D2D \_ Matrix \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f) -Struktur. Eine Liste der Felder, die von der Matrix bereitgestellt werden, finden Sie unter [**D2D \_ Matrix \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f).
