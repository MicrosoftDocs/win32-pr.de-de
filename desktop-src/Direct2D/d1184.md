# <a name="d1184-createwicbitmaprendertarget-incompatible-with-hw-target-type"></a>D1184: "anatewicbitmaprendertarget" ist nicht kompatibel mit dem HW-Zieltyp.

Es ist ungültig, das Flag D2D1 \_ \_ \_ renderzieltyp \_ Hardware an "kreatewicbitmaprendertarget" zu übergeben.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Wenn eine Anwendung mit dem Namen ID2D1Factory:: | ":: | tyewicbitmaprendertarget" lautet, war das Typargument des D2D1- \_ \_ Renderziel- \_ Eigenschaften Arguments D2D1 \_ D2D1 \_ \_ Renderziel \_ \_

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass der Zieltyp, der beim Erstellen eines WIC-interoppten Renderziels verwendet wird, Software oder default ist Direct2D reserviert in der Regel die Hardware Semantik, um Direct3D Interopability zu beschreiben.
