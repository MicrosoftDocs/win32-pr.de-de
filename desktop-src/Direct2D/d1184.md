# <a name="d1184-createwicbitmaprendertarget-incompatible-with-hw-target-type"></a>D1184: CreateWicBitmapRenderTarget Incompatible With HW Target Type

Es ist ungültig, das Flag D2D1 \_ RENDER TARGET TYPE HARDWARE an \_ \_ \_ CreateWicBitmapRenderTarget zu übergeben.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Bei einer Anwendung mit dem Namen ID2D1Factory::CreateWicBitmapRenderTarget war das Typfeld des D2D1 RENDER TARGET PROPERTIES -Arguments \_ \_ \_ D2D1 \_ D2D1 \_ RENDER TARGET TYPE \_ \_ \_ HARDWARE.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass der Zieltyp, der beim Erstellen eines mit WIC ineinander verknappten Renderziels verwendet wird, SOFTWARE oder DEFAULT ist. Direct2D reserviert in der Regel HARDWARE-Semantik, um die Direct3D-Interopierbarkeit zu beschreiben.
