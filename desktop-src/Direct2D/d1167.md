# <a name="d1167-read-map-option-requires-cpu-readable-bitmap"></a>D1167: die Option "Map lesen" erfordert CPU-lesbare Bitmap

D2D1 \_ map- \_ Optionen \_ lesen ist nur für eine CPU-lesbare Bitmap gültig.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung möglicherweise aus, wenn eine Anwendung ID2D1Bitmap1:: Map mit dem Read-Attribut der D2D1 \_ map \_ -Optionen aufruft \_ , aber die Bitmap ist nicht CPU-lesbar.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Wenn eine Anwendung einen Lesezugriff mithilfe von ID2D1Bitmap1:: Map erfordert, erstellen Sie die Bitmap mithilfe der \_ \_ \_ Option CPU-Lese Option für D2D1 Bitmap-Optionen \_ .
