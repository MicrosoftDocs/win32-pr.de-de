# <a name="d1167-read-map-option-requires-cpu-readable-bitmap"></a>D1167: Read Map-Option erfordert cpulesbare Bitmap

D2D1 \_ MAP OPTIONS READ ist nur für eine \_ \_ CPU-lesbare Bitmap gültig.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Direct2D gibt diese Meldung möglicherweise aus, wenn eine Anwendung ID2D1Bitmap1::Map mit dem READ-Attribut D2D1 MAP OPTIONS aufruft, die Bitmap jedoch nicht \_ \_ \_ CPU-lesbar ist.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Wenn eine Anwendung Lesezugriff mithilfe von ID2D1Bitmap1::Map erfordert, erstellen Sie die Bitmap mithilfe der CPU \_ \_ \_ READ-Option D2D1 BITMAP OPTIONS \_ .
