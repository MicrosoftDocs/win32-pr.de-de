# <a name="d1137-unclosed-command-list"></a>D1137: nicht geschlossene Befehlsliste

Es wurde auf eine Befehlsliste verwiesen, die im Gerätekontext entweder nicht aufgefüllt oder nicht festgelegt wurde.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Diese Meldung kann auftreten, wenn versucht wird, eine Befehlsliste zu zeichnen oder zu streamen, bevor der ID2D1CommandList:: Close-Vorgang aufgerufen wurde. Direct2D berücksichtigt nicht, dass eine Befehlsliste gültig ist, bis Sie aufgefüllt wurde. die Verwendung der Close-Methode gibt an, dass das Füllen einer Befehlsliste abgeschlossen ist. Wenn die Anwendung die Befehlsliste für einen Zeichnungs Vorgang verwendet, darf der verwendete Gerätekontext die Befehlsliste nicht als Ziel festlegen.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Bevor Sie versuchen, eine Befehlsliste zu verwenden, wenden Sie die ID2D1CommandList:: Close-Methode an. Wenn die Befehlsliste mithilfe von ID2D1DeviceContext::D rawImage oder als Quelle eines Bild Pinsels gezeichnet wird, stellen Sie sicher, dass die Befehlsliste nicht das aktuelle Ziel des entsprechenden Geräte Kontexts ist.
