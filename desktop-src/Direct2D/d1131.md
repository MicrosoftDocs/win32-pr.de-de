# <a name="d1131-dxgi-format-not-gdi-compatible"></a>D1131: DXGI-Format nicht GDI-kompatibel

An diese API wurde eine GDI-kompatible Option übermittelt, aber das DXGI-Format ist nicht GDI-kompatibel.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Mit dem \_ \_ \_ GDI-kompatiblen Flag D2D1 Bitmap options \_ bei der bitmaperstellung wird die Kompatibilität der Bitmap mit dem GDI-Rendering ermöglicht, es werden jedoch zusätzliche Einschränkungen für das Oberflächen Format der Bitmap eingeführt. Im Allgemeinen ist das GDI-Rendering mit dem DXGI \_ \_ -Format B8G8R8A8 \_ unorm und DXGI \_ Format \_ B8G8R8A8 \_ unorm \_ sRGB-Format kompatibel.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Um dieses Problem zu beheben, stellen Sie sicher, dass das für eine GDI-kompatible Bitmap angegebene Format ein BGRA-Format mit vier Kanälen ist, wie oben angegeben. Wenn die Bitmap keine Interoperabilität mit GDI erfordert, lassen Sie den GDI- \_ kompatiblen Spezifizierer aus.
