# <a name="d1256-invalid-analysis-effect-usage"></a>D1256: Ungültige Verwendung von Analyse Effekten

Ein Analyse Effekt wurde falsch verwendet. Dies kann vorkommen, wenn Sie einen Analyse Effekt als Eingabe für einen anderen Effekt oder einen Bild Pinsel verwenden, ihn in eine Befehlsliste zeichnen oder seine Ausgabe Zwischenspeichern.
