# <a name="d1182-operation-on-unclosed-path-geometry"></a>D1182: Vorgang bei nicht geschlossenen Pfad Geometrie

In einer Pfad Geometrie ist ein Vorgang aufgetreten, der nicht geschlossen wurde.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat mit dem Auffüllen eines ID2D1PathGeometry begonnen, aber nicht "ID2D1SimplifiedGeometrySink:: Close" für die auffüllende Senke. Oder die Anwendung heißt Close, aber es wurde ein Fehlerwert zurückgegeben.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass alle Pfadgeometrien ordnungsgemäß aufgefüllt werden, indem Sie Close auf der Auffüllen-Senke aufrufen und sicherstellen, dass close keinen Fehler zurückgegeben hat. In der Referenz für ID2D1PathGeometry finden Sie ein Beispiel für das Erstellen und Auffüllen eines Pfad Geometrie Objekts.
