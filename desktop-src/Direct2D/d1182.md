# <a name="d1182-operation-on-unclosed-path-geometry"></a>D1182: Operation On Unclosed Path Geometry

Für eine Pfadgeometrie, die nicht geschlossen wurde, ist ein Vorgang aufgetreten.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat damit begonnen, eine ID2D1PathGeometry-Id2D1pathGeometry zu aufpopulieren, aber id2D1SimplifiedGeometrySink::Close in der aufpopulierenden Senke nicht aufgerufen. Oder die Anwendung hat Close aufgerufen, aber es wurde ein Fehlerwert zurückgegeben.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass alle Pfadgeometrien ordnungsgemäß aufgefüllt werden, indem Sie Schließen für ihre auffüllende Senke aufrufen und sicherstellen, dass Close keinen Fehler zurückgegeben hat. Ein Beispiel zum Erstellen und Auffüllen eines Pfadgeometrieobjekts finden Sie in der Referenz zu ID2D1PathGeometry.
