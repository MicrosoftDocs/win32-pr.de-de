# <a name="d1145-print-control-dpi-incorrect"></a>D1145: Print Control DPI Incorrect

Direct2D Print: Der Wert PRINT \_ CONTROL \_ PROPERTIES.rasterDPI ist ungültig.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Während der Erstellung eines Drucksteuer steuerelements gibt diese Meldung an, dass die Anwendung einen ungültigen DPI angegeben hat. insbesondere ein DPI von weniger als 1,0. Direct2D betrachtet dies zwar nicht als Fehler, Direct2D Print verhält sich jedoch so, als ob die Anwendung einen DPI-Wert von 150.0 angegeben hätte.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Verwenden Sie zum Adressen dieser Meldung einen DPI-Wert von 1,0 oder höher, während Sie ein Drucksteuer steuerelement erstellen.
