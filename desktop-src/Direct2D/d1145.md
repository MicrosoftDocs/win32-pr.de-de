# <a name="d1145-print-control-dpi-incorrect"></a>D1145: Druck Steuerelement-dpi falsch

Direct2D Print: Eigenschaften des Druck \_ Steuer Elements \_ . der rasterdpi-Wert ist ungültig.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Während der Erstellung eines Druck Steuer Elements weist diese Meldung darauf hin, dass die Anwendung einen ungültigen dpi-Wert angegeben hat. insbesondere ein dpi-Parameter kleiner als 1,0. Obwohl Direct2D dies nicht als Fehler betrachtet, verhält sich Direct2D Print so, als ob die Anwendung eine dpi-Angabe von 150,0 angegeben hätte.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Verwenden Sie beim Erstellen eines Druck Steuer Elements einen dpi-Code von 1,0 oder höher, um diese Meldung zu behandeln.
