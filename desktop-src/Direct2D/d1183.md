# <a name="d1183-createwicbitmaprendertarget-with-incompatible-feature-level"></a>D1183: CreateWicBitmapRenderTarget mit inkompatibler Featureebene

Es ist ungültig, eine andere Direct2D-Featureebene als D2D1 \_ FEATURE LEVEL DEFAULT an \_ \_ CreateWicBitmapRenderTarget zu übergeben.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Wenn eine Anwendung namens ID2D1Factory::CreateWicBitmapRenderTarget das MinLevel-Feld des D2D1 \_ RENDER \_ TARGET \_ PROPERTIES-Arguments nicht D2D1 \_ FEATURE LEVEL DEFAULT \_ \_ war.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Vergewissern Sie sich, dass die Featureebene, die beim Erstellen eines WIC-übergreifenden Renderziels verwendet wird, DEFAULT ist. Die Interoperabilität mit WIC umfasst nur Softwarefeatures. Semantik auf Featureebene ist für das Hardwareverhalten relevant.
