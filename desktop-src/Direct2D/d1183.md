# <a name="d1183-createwicbitmaprendertarget-with-incompatible-feature-level"></a>D1183: "anatewicbitmaprendertarget" mit nicht kompatibler Funktionsebene

Es ist nicht möglich, eine Direct2D Featureebene außer der \_ Standardeinstellung der D2D1-Funktions \_ Ebene \_ an "kreatewicbitmaprendertarget" zu übergeben.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Wenn eine Anwendung mit dem Namen "ID2D1Factory:: | atewicbitmaprendertarget" aufgerufen wird, war das minlevel-Feld des D2D1- \_ \_ Renderziel- \_ Eigenschaften Arguments nicht D2D1 \_ Funktions \_ Ebenen- \_ Standard.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Vergewissern Sie sich, dass die beim Erstellen eines WIC-interoppten Renderziels verwendete Funktionsebene standardmäßig ist. Interoperabilität mit WIC umfasst reine Software Features. die Semantik der Funktionsebene ist für das Hardware Verhalten relevant.
