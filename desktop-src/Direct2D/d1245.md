# <a name="d1245-error-triggered-commandlist-invalidation"></a>D1245: Error Triggered Commandlist Invalidation

Aufgrund vorheriger Fehler sind alle Befehlslisten, die zuvor als Ziel festgelegt wurden, jetzt ungültig.
