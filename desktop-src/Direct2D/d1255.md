# <a name="d1255-tile-opacity-metadata-conflict"></a>D1255: Konflikte bei Kachel Deckkraft-Metadaten

Das für ein Effekt Diagramm berechnete Rechteck hat einen als transparent markierten Bereich überlappen. Dies kann aufgrund einer unsachgemäßen Verwendung eines Atlas-Effekts oder einer Anpassungs Transformation für Begrenzungen auftreten.
