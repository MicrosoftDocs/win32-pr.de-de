# <a name="d1255-tile-opacity-metadata-conflict"></a>D1255: Konflikt mit Kachel-Deckkraftmetadaten

Das für ein Effektdiagramm berechnete nicht transparente Rechteck überlappt einen als transparent markierten Bereich. Dies kann aufgrund einer unsachgemäßen Verwendung eines Atlas-Effekts oder einer Anpassungstransformation für Begrenzungen auftreten.
