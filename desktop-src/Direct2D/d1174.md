# <a name="d1174-pushlayer-to-command-list-will-ignore-layer-object"></a>D1174: Das Ebenenobjekt wird von der pushschicht an die Befehlsliste ignoriert

Beim Pushen einer (nicht-null)-Ebene in eine Befehlsliste wird das Ebenenobjekt ignoriert.



|             |         |
|-------------|---------|
| Fehlerstufe | Warnung |



 

## <a name="possible-causes"></a>Mögliche Ursachen

Wenn ein pushlayer-Befehl ein ID2D1Layer-Objekt verwendet und es sich bei dem Ziel um eine Befehlsliste handelt, verhält sich Direct2D so, als ob keine Ebene angegeben wurde, und es werden keine festgelegten Grenzen auf der Ebene erzwungen, wenn die Befehlsliste später gerendert wird.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Anwendungen, die das Streaming von Befehlen an eine Befehlsliste erwarten, sollten die Verwendung von NULL-ebenenobjekten in Erwägung gezogen werden. andernfalls sollten Sie beachten, dass die im ID2D1Layer-Objekt angegebenen Größen Einschränkungs Einstellungen nicht berücksichtigt werden
