# <a name="d1249-inaccessible-target"></a>D1249: Ziel, auf das nicht zugegriffen werden kann

Es wurde ein Versuch erkannt, auf ein ziel zu zeichnen, auf das nicht zugegriffen werden kann.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Es wurde ein Versuch erkannt, auf ein ziel zu zeichnen, auf das nicht zugegriffen werden kann. Ein Grund, warum auf ein Ziel nicht zugegriffen werden kann, ist, wenn es aus einer freigegebenen Textur erstellt und die Textur nicht erworben wurde.

## <a name="possible-fixes"></a>Mögliche Fehlerbehebungen

Stellen Sie sicher, dass auf alle Ziele zugegriffen werden kann, bevor Sie sie mit Direct2D rendern.
