# <a name="d1249-inaccessible-target"></a>D1249: Ziel nicht zugänglich

Es wurde versucht, auf ein nicht zugängliches Ziel zu zeichnen.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Es wurde versucht, auf ein nicht zugängliches Ziel zu zeichnen. Ein Grund, warum ein Ziel möglicherweise nicht zugänglich ist, ist, wenn es aus einer freigegebenen Textur erstellt wurde und die Textur nicht abgerufen wurde.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Stellen Sie sicher, dass auf alle Ziele zugegriffen werden kann, bevor Sie Sie mit Direct2D Rendering.
