# <a name="d1177-unsupported-wic-pixel-format"></a>D1177: nicht unterstütztes WIC-Pixel Format

Das bereitgestellte WIC-Pixel Format wird von Direct2D nicht unterstützt. Direct2D unterstützt insbesondere keine WIC-Bitmaps, die nur "nur-Alpha" verwenden, wenn die Tiefe 8bpp ist.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat versucht, eine WIC-Ressource zu interagieren, deren Format von Direct2D nicht unterstützt wird. Normalerweise handelt es sich bei diesen Formaten um solche Formate mit einem geraden Alpha, deren Tiefe nicht 8 Bits pro Ebene ist.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Erstellen Sie die WIC-Ressource mit einem alternativen Format für die Interoperabilität mit Direct2D, oder wandeln Sie Sie um.
