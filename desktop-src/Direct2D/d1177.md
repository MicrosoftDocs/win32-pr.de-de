# <a name="d1177-unsupported-wic-pixel-format"></a>D1177: Nicht unterstütztes WIC-Pixelformat

Das angegebene WIC-Pixelformat wird von Direct2D nicht unterstützt. Insbesondere unterstützt Direct2D keine WIC-Bitmaps, die gerades Alpha verwenden, es sei denn, die Tiefe beträgt 8bpp.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat versucht, eine WIC-Ressource zu interoperieren, deren Format von Direct2D nicht unterstützt wird. In der Regel handelt es sich bei diesen Formaten um Formate mit geradem Alpha, deren Tiefe nicht 8 Bits pro Ebene beträgt.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Erwägen Sie, die WIC-Ressource mit einem alternativen Format für die Interoperabilität mit Direct2D zu erstellen oder zu konvertieren.
