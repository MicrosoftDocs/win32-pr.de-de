# <a name="d1162-fillmesh-requires-aliased-rendering"></a>D1162: fillmesh erfordert ein Alias Rendering

Fillmesh erfordert, dass der antialiasmodus auf "D2D1 \_ Antialias \_ Mode Aliasing" festgelegt ist \_ .






 

## <a name="possible-causes"></a>Mögliche Ursachen

Wenn eine Anwendung ID2D1RenderTarget:: fillmesh aufruft, sollte der antialiasmodus des Beteiligten renderTarget den D2D1- \_ Antialias \_ Modus-Alias aufweisen \_ . Das Standardverhalten eines Renderziels ist die Verwendung von Antialiasing. Daher ist es erforderlich, ID2D1RenderTarget:: setantialiasmode vor dem Auffüllen von Meshes aufzurufen.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Verwenden Sie ID2D1RenderTarget:: setantialiasmode, um sicherzustellen, dass renderTarget das Rendern mit Alias beim Auffüllen von Meshes verwendet. Wenn nachfolgende Vorgänge Antialiasing erfordern, legen Sie den antialiasmodus des Ziels auf den D2D1- \_ Antialias \_ Modus \_ pro \_ primitiv fest. Stellen Sie sicher, dass die Aufrufe an ID2D1RenderTarget:: setantialiasmode und ID2D1RenderTarget:: fillmesh für dasselbe Ziel ausgegeben werden.
