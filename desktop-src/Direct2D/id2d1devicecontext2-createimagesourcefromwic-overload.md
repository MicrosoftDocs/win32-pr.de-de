# <a name="id2d1devicecontext2createimagesourcefromwic-methods"></a>ID2D1DeviceContext2:: kreateimagesourcefromwic-Methoden

Erstellt ein Bild Quell Objekt aus einer WIC-Bitmapquelle und füllt dabei den gesamten Pixel Speicher innerhalb der Bildquelle auf. Das Bild wird bei minimaler Speichermenge geladen und gespeichert.
