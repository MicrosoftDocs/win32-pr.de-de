# <a name="id2d1imagesourcefromwicensurecached-methods"></a>ID2D1ImageSourceFromWic:: ensurecme-Methoden

Lädt Bilddaten in Caches der Bildquellen, wenn diese Daten noch nicht zwischengespeichert wurden.
