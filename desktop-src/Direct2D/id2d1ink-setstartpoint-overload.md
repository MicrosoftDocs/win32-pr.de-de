# <a name="id2d1inksetstartpoint-methods"></a>ID2D1Ink:: setstartpoint-Methoden

Legt den Anfangspunkt für dieses Ink-Objekt fest. Dadurch wird festgelegt, wo dieses frei Hand Objekt das Rendering startet.
