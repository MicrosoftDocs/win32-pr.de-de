# <a name="alpha-mask-effect"></a>Alpha maskeneffekt

Dieser Effekt wendet eine Alpha Maske auf ein Bild an. Sie verfügt über zwei Eingaben, die als Ziel und Maske bezeichnet werden. Farbwerte im Zielbild werden mit dem Alphakanal des Masken Bilds multipliziert.

Die CLSID für diesen Effekt ist CLSID \_ D2D1AlphaMask.

## <a name="effect-properties"></a>Effekt Eigenschaften

Dieser Effekt hat keine Auswirkungen spezifischen Eigenschaften.
