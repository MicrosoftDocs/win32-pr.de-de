# <a name="alpha-mask-effect"></a>Alphamaskeneffekt

Dieser Effekt wendet eine Alphamaske auf ein Bild an. Es verfügt über zwei Eingaben namens Destination und Mask. Farbwerte im Zielbild werden mit dem Alphakanal des Maskierungsbilds multipliziert.

Die CLSID für diesen Effekt ist CLSID \_ D2D1AlphaMask.

## <a name="effect-properties"></a>Effect-Eigenschaften

Dieser Effekt hat keine auswirkungsspezifischen Eigenschaften.
