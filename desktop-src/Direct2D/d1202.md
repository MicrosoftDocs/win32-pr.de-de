# <a name="d1202-copyfromrendertarget-with-multisampled-source"></a>D1202: copyfromrendertarget mit Multisampling-Quelle

Copyfromrendertarget wurde mit einem Multisampling-renderTarget verwendet. Der-Rückruf wird weiterhin ausgegeben, das kopierte Ergebnis stellt jedoch möglicherweise nicht das erwartete Multisampling-Muster dar.






 

## <a name="possible-causes"></a>Mögliche Ursachen

Die Anwendung hat eine Multisampling-Direct3D-Textur als Direct2D-Ziel interagiert und copyfromrendertarget für das Ziel aufgerufen.

## <a name="possible-fixes"></a>Mögliche Korrekturen

Vermeiden Sie das interoperieren von Multisampling-Texturen als Ziele, wenn die copyfromrendertarget-Funktionalität wichtig ist. Das kopierte Ergebnis stellt möglicherweise nicht das erwartete Multisampling-Muster dar.
