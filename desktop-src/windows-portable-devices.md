# <a name="windows-portable-devices"></a>Tragbare Windows-Geräte

Windows Portable Devices (WPD) ermöglicht Computern die Kommunikation mit angeschlossenen Medien und Speichergeräten. WPD bietet eine flexible und robuste Möglichkeit für Computer, mit Musik Playern, Speichergeräten, Mobiltelefonen, Kameras und vielen anderen Arten von verbundenen Geräten zu kommunizieren. Dieses System ersetzt sowohl Windows Media Device Manager als auch Windows-Abbild Erwerb.

Anwendungen, die auf WPD basieren, können ein Gerät untersuchen, Inhalte senden und empfangen und sogar das Gerät steuern, z. b. ein Bild machen oder eine Textnachricht senden. Das System ist so konzipiert, dass es so flexibel ist, dass viele Arten von Geräten untersucht und erweitert werden können, sodass Treiber Entwickler benutzerdefinierte Eigenschaften und Befehle für benutzerdefinierte Geräte definieren können.

Sie können sowohl Windows-Anwendungen als auch Webanwendungen mit WPD schreiben. Zum Erstellen von Windows-Anwendungen verwenden Sie die WPD-Programmierschnittstelle. Diese Anwendungen können in C++, C \# .net oder Visual Basic .NET geschrieben werden. Sie verwenden das WPD-Automatisierungs Objektmodell zum Schreiben von Webanwendungen. Diese Anwendungen werden in JScript und HTML geschrieben.

Die WPD-Anwendungsprogrammierschnittstelle wird in den Betriebssystemen Windows 7, Windows Vista und Windows XP unterstützt. Das WPD-Automatisierungs Objektmodell wird nur in Windows 7 unterstützt.

Weitere Informationen zum Erstellen einer WPD-Windows-Anwendung finden Sie in der Dokumentation und den Beispielen der [WPD-Anwendungsprogrammierschnittstelle](./wpd_sdk/wpd-application-programming-interface.md) .

Weitere Informationen zum Erstellen einer WPD-Automatisierungs Anwendung finden Sie in der Dokumentation und den Beispielen zum [WPD-Automatisierungs Objektmodell](/previous-versions/windows/desktop/legacy/dd389295(v=vs.85)) .
