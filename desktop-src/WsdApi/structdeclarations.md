# <a name="structdeclarations-element"></a>structDeclarations-Element

Generiert C-Strukturdeklarationen für bekannte Typen.

## <a name="usage"></a>Verbrauch

``` syntax
<structDeclarations/>
```

## <a name="attributes"></a>Attribute

Es gibt keine Attribute.

## <a name="child-elements"></a>Untergeordnete Elemente

Es gibt keine untergeordneten Elemente.

## <a name="parent-elements"></a>Übergeordnete Elemente



| Element                         | BESCHREIBUNG                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**Datei**](file.md)<br/> | Gibt eine Datei aus dem Codegenerator aus.<br/> <br/> |



## <a name="remarks"></a>Hinweise

Auf Strukturen für bekannte Typen wird von einem Großteil des generierten Codes und von Anwendungscode verwiesen. Dieses Element wird verwendet, um Includedateien zu generieren.

|-------------------------------------|---------------|
