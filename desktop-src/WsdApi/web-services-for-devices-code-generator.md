# <a name="web-services-on-devices-code-generator"></a>Code-Generator für Webdienste auf Geräten

Der Code-Generator für Webdienste auf Geräten (wsdcodegen) ist ein Befehlszeilenprogramm, das Programmcode aus einer Dienst Beschreibung generiert. Die Dienst Beschreibung wird in WSDL-und/oder XSD-Dateien gespeichert. Wsdcodegen erstellt C++-und IDL-Dateien (Schnittstellen Definitions Sprache). Entwickler können mit wsdcodegen WSDAPI-Anwendungen erstellen, ohne sich Gedanken darüber machen zu müssen, wie die Daten gemarshallt und bei der Übertragung dargestellt werden.

Wsdcodegen kann verwendet werden, um Code für einen Dienst, einen Client oder beides zu generieren. Das Hilfsprogramm generiert Stub-Code für Dienste, Proxycode für Clients und Schnittstellen-und Typcode für beide.

Dieses Tool ist im Microsoft Windows Software Development Kit (SDK) und im Windows-Treiberkit (WDK) verfügbar. Die SDK-Tools befinden sich im folgenden Verzeichnis: <Windows SDK Install Folder> \\ bin.

Der Windows SDK enthält Beispielcode, der von wsdcodegen generiert wurde. Weitere Informationen finden Sie unter [WSDAPI Samples](wsdapi-samples.md).
