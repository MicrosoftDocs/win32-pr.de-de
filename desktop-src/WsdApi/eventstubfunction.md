# <a name="eventstubfunction-element"></a>eventstubfunction-Element

Gibt an, ob Stub-Funktions Verweise in den Vorgangs Strukturen in den Porttyp Definitionen für Benachrichtigungs Vorgänge enthalten sein sollen.

## <a name="usage"></a>Verbrauch

``` syntax
<eventStubFunction/>
```

## <a name="attributes"></a>Attribute

Es gibt keine Attribute.

## <a name="child-elements"></a>Untergeordnete Elemente

Es gibt keine untergeordneten Elemente.

## <a name="parent-elements"></a>Übergeordnete Elemente



| Element                                                       | BESCHREIBUNG                                                  |
|---------------------------------------------------------------|--------------------------------------------------------------|
| [**porttypeer Definitionen**](porttypedefinitions.md)<br/> | Generiert C-Konstanten für Porttypen.<br/> <br/> |



## <a name="remarks"></a>Bemerkungen

Stub-Funktions Verweise sind in Client Szenarien für Benachrichtigungs Vorgänge (Ereignisse) enthalten.

Gültige Werte für dieses Element sind 1 (true-/stubfunktionsverweise eingeschlossen) und 0 (false/keine Stub-Funktions Verweise eingeschlossen).
