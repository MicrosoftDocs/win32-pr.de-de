## <a name="verifying-that-messages-meet-traffic-requirements"></a><span data-ttu-id="5bce1-117">Überprüfen, ob Nachrichten die Datenverkehrsanforderungen erfüllen</span><span class="sxs-lookup"><span data-stu-id="5bce1-117">Verifying that messages meet traffic requirements</span></span>

-   <span data-ttu-id="5bce1-123">Eine [ProbeMatches-Nachricht](probematches-message.md) muss an den HTTP- oder HTTPS-Port gesendet werden, von dem der [Test](probe-message.md) gesendet wurde.</span><span class="sxs-lookup"><span data-stu-id="5bce1-123">A [ProbeMatches](probematches-message.md) message must be sent to the HTTP or HTTPS port from which the [Probe](probe-message.md) was sent.</span></span>
-   <span data-ttu-id="5bce1-124">Das **RelatesTo-Element** einer [ProbeMatches-Nachricht](probematches-message.md) muss vorhanden sein und darf nicht leer sein.</span><span class="sxs-lookup"><span data-stu-id="5bce1-124">The **RelatesTo** element of a [ProbeMatches](probematches-message.md) message must be present and must not be empty.</span></span> <span data-ttu-id="5bce1-125">Der Wert muss mit dem Wert des **MessageId-Elements** aus der entsprechenden [Probe-Nachricht](probe-message.md) übereinstimmen.</span><span class="sxs-lookup"><span data-stu-id="5bce1-125">Its value must match the value of the **MessageId** element from the corresponding [Probe](probe-message.md) message.</span></span>

-   <span data-ttu-id="5bce1-133">Eine [ResolveMatches-Nachricht](resolvematches-message.md) muss innerhalb von 4 Sekunden nach der entsprechenden [Resolve-Nachricht](resolve-message.md) gesendet werden.</span><span class="sxs-lookup"><span data-stu-id="5bce1-133">A [ResolveMatches](resolvematches-message.md) message must be sent within 4 seconds of the corresponding [Resolve](resolve-message.md) message.</span></span> <span data-ttu-id="5bce1-134">Die Windows-Firewall kann eine ResolveMatches-Nachricht löschen, die mehr als 4 Sekunden nach einer Resolve-Nachricht gesendet wurde.</span><span class="sxs-lookup"><span data-stu-id="5bce1-134">The Windows Firewall may drop a ResolveMatchesmessage sent more than 4 seconds after a Resolve message.</span></span>
