# <a name="eventsourcebuilderdeclarations-element"></a>eventsourcebuilderdeklarationselement

Generiert Deklarationen für Funktionen, die Ereignis Quell Klassen erstellen.

## <a name="usage"></a>Verbrauch

``` syntax
<eventSourceBuilderDeclarations>
  child elements
</eventSourceBuilderDeclarations>
```

## <a name="attributes"></a>Attribute

Es gibt keine Attribute.

## <a name="child-elements"></a>Untergeordnete Elemente



| Element                                     | BESCHREIBUNG                                                                           |
|---------------------------------------------|---------------------------------------------------------------------------------------|
| [**proxyClass**](proxyclass.md)<br/> | Klassenname, der aus der Ereignis Quellen-Generator-Funktion generiert werden soll.<br/> <br/> |



### <a name="child-element-sequence"></a>Sequenz untergeordneter Elemente

``` syntax
proxyClass
```

## <a name="parent-elements"></a>Übergeordnete Elemente



| Element                         | BESCHREIBUNG                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**Datei**](file.md)<br/> | Gibt eine Datei aus dem Code-Generator aus.<br/> <br/> |
