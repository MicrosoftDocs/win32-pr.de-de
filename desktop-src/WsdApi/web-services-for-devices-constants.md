# <a name="web-services-on-devices-constants"></a>Webdienste auf Geräten – Konstanten

Die folgenden Konstanten werden von Webdiensten auf Geräten verwendet.



| Konstante                                                                                                                                                                                                                        | Beschreibung                                                                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WSD_DEFAULT_HOSTING_ADDRESS"></span><span id="wsd_default_hosting_address"></span><dl> <dt>**\_WSD-STANDARDHOSTADRESSE \_ \_**</dt> </dl>                       | Die Standardadresse (im [UrlPrefix-Format),](/windows/desktop/Http/urlprefix-strings) die ein WSDAPI-Host zum Lauschen auf Anforderungen an Port 5357 verwendet. <br/>                                                 |
| <span id="WSD_DEFAULT_SECURE_HOSTING_ADDRESS"></span><span id="wsd_default_secure_hosting_address"></span><dl> <dt>**SICHERE \_ WSD-STANDARDHOSTADRESSE \_ \_ \_**</dt> </dl> | Die standardmäßige sichere Adresse (im [UrlPrefix-Format),](/windows/desktop/Http/urlprefix-strings) die ein WSDAPI-Host zum Lauschen auf Anforderungen an Port 5358 verwendet. <br/>                                          |
| <span id="WSD_DEFAULT_EVENTING_ADDRESS__"></span><span id="wsd_default_eventing_address__"></span><dl> <dt>**WSD \_ \_ \_ STANDARDEREIGNISADRESSE**</dt> </dl>               | Die Standardadresse (im [UrlPrefix-Format),](/windows/desktop/Http/urlprefix-strings) die ein WSDAPI-Host zum Lauschen auf Ereignisse an Port 5358 für sichere Hosts und an Port 5357 für andere Hosts verwendet. <br/> |
