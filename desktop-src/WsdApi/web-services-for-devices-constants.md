# <a name="web-services-on-devices-constants"></a>Konstanten für Webdienste auf Geräten

Die folgenden Konstanten werden von Webdiensten auf Geräten verwendet.



| Konstante                                                                                                                                                                                                                        | BESCHREIBUNG                                                                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WSD_DEFAULT_HOSTING_ADDRESS"></span><span id="wsd_default_hosting_address"></span><dl> <dt>**WSD \_ - \_ standardhostingadresse \_**</dt> </dl>                       | Die Standardadresse (im [URLPrefix](/windows/desktop/Http/urlprefix-strings) -Format), die ein WSDAPI-Host zum lauschen auf Anforderungen an Port 5357 verwendet. <br/>                                                 |
| <span id="WSD_DEFAULT_SECURE_HOSTING_ADDRESS"></span><span id="wsd_default_secure_hosting_address"></span><dl> <dt>**WSD \_ - \_ Standard \_ Adresse für sicheres Hosting \_**</dt> </dl> | Die standardmäßige sichere Adresse (im [URLPrefix](/windows/desktop/Http/urlprefix-strings) -Format), mit der ein WSDAPI-Host auf Anforderungen an Port 5358 lauscht. <br/>                                          |
| <span id="WSD_DEFAULT_EVENTING_ADDRESS__"></span><span id="wsd_default_eventing_address__"></span><dl> <dt>**WSD \_ Standard \_ \_ Adresse** für Ereignisse</dt> </dl>               | Die Standardadresse (im [URLPrefix](/windows/desktop/Http/urlprefix-strings) -Format), die ein WSDAPI-Host zum lauschen auf Ereignisse an Port 5358 für sichere Hosts und Port 5357 für andere Hosts verwendet. <br/> |
