# <a name="namespacedefinitions-element"></a>namespaceDefinitions-Element

Generiert C-Definitionen für Namespacetabellen.

## <a name="usage"></a>Verbrauch

``` syntax
<namespaceDefinitions/>
```

## <a name="attributes"></a>Attribute

Es gibt keine Attribute.

## <a name="child-elements"></a>Untergeordnete Elemente

Es gibt keine untergeordneten Elemente.

## <a name="parent-elements"></a>Übergeordnete Elemente



| Element                         | BESCHREIBUNG                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**Datei**](file.md)<br/> | Gibt eine Datei aus dem Codegenerator aus.<br/> <br/> |



## <a name="remarks"></a>Hinweise

Dieses Element wird im Allgemeinen in C-Quelldateien verwendet, um die Namespacetabellen zur Verfügung zu stellen, die von [**namespaceDeclarations deklariert wurden.**](namespacedeclarations.md)

| Bezeichnung | Wert |
|-------------------------------------|---------------|
| Unterstützte Mindestversion (System)<br/> | Windows Vista |
| Kann leer bleiben                        | Ja           |
