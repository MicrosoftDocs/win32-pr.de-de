# <a name="tools-for-troubleshooting-using-network-tracing-in-windows-7"></a>Tools zur Problembehandlung mithilfe der Netzwerk Ablauf Verfolgung in Windows 7

Es gibt eine Reihe von Tools, die verwendet werden können, um die Netzwerk Ablauf Verfolgung zu aktivieren und zu verwalten und die resultierenden Daten anzuzeigen. Einige der Tools, die hilfreich sein können, sind Netsh.exe, Netzwerkmonitor, Ereignisanzeige, **Netsh Trace Convert**, Tracerpt.exe und Xperf.exe.

In diesem Abschnitt wird erläutert, wie Netsh.exe und Netzwerkmonitor verwendet werden, um die Problembehandlung bei der Netzwerk Ablauf Verfolgung zu unterstützen.
