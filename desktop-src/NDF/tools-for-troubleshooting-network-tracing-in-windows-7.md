# <a name="tools-for-troubleshooting-using-network-tracing-in-windows-7"></a>Tools für die Problembehandlung mithilfe der Netzwerkablaufverfolgung in Windows 7

Es gibt eine Reihe von Tools, mit denen Sie die Netzwerkablaufverfolgung aktivieren und verwalten und die resultierenden Daten anzeigen können. Einige der Tools, die hilfreich sein können, sind Netsh.exe, Netzwerkmonitor, Ereignisanzeige, **netsh trace convert,** Tracerpt.exe und Xperf.exe.

In diesem Abschnitt erfahren Sie, wie Sie Netsh.exe und Netzwerkmonitor zur Problembehandlung bei der Netzwerkablaufverfolgung verwenden.
