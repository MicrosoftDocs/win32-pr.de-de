# <a name="ndf-helper-class-extension-examples"></a>Beispiele für NDF-Hilfsklassenerweiterungen

Diese Beispiele gelten nur, wenn Entwickler der Meinung sind, dass es notwendig ist, die Funktionalität einer Microsoft NDF Helper Class zu erweitern, die als erweiterbar deklariert ist. Dies ist im Allgemeinen nicht erforderlich, aber in einigen Fällen bieten sich aufgrund der spezifischen Hardware- oder Softwareanforderungen der Komponente besondere Diagnosemöglichkeiten.

Die beiden folgenden Beispiele sind verknüpft, und das erste sollte verstanden werden, bevor das zweite analysiert wird. Die erste stellt eine Implementierung einer [NDF-Hilfsklassenerweiterung mithilfe](ndf-helper-class-example.md) einer Klasse namens SimpleHelperFileClass zur Anwendung.

Die zweite, die NDF-Hilfsklassenerweiterung mit [Handoff,](ndf-helper-class-example-with-handoff.md)stellt ein Beispiel für eine Erweiterung zur Verfügung, die keine bestimmten Diagnoseaufgaben, sondern nur eine Niedrige-Integritätshypothese für die SimpleHelperFileClass im ersten Beispiel generiert.
