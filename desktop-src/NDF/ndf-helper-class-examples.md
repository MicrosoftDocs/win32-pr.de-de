# <a name="ndf-helper-class-extension-examples"></a>Beispiele für die NDF-Hilfsklassen Erweiterung

Diese Beispiele gelten nur, wenn Entwickler der Meinung sind, dass es erforderlich ist, die Funktionalität einer Microsoft NDF-Hilfsklasse zu erweitern, die als erweiterbar deklariert ist. Dies ist in der Regel nicht erforderlich, aber in einigen Fällen stellen sich besondere Diagnosemöglichkeiten aufgrund der spezifischen Hardware-oder Softwareanforderungen der Komponente selbst dar.

Die beiden folgenden Beispiele sind miteinander verknüpft, und die erste sollte vor der Analyse der zweiten verstanden werden. Der erste stellt eine Implementierung einer [NDF-Hilfsklassen Erweiterung](ndf-helper-class-example.md) mithilfe einer Klasse namens simplehelperfileclass bereit.

Die zweite [NDF-Hilfsklassen Erweiterung mit Handoff](ndf-helper-class-example-with-handoff.md)bietet ein Beispiel für eine Erweiterung, die keine bestimmten Diagnose Tasks ausführt, sondern nur eine niedrig Integritäts Hypothese für die simplehelperfileclass im ersten Beispiel generiert.
