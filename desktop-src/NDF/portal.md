# <a name="network-diagnostics-framework"></a>Netzwerk Diagnose-Framework

## <a name="purpose"></a>Zweck

Das Network Diagnostics Framework (ndf) bietet Komponenten-und Anwendungsentwicklern die Möglichkeit, die Netzwerkproblem Behandlung für Benutzer zu vereinfachen. Benutzer können mithilfe eines einzelnen Tools zur Problembehandlung versuchen, ein Netzwerkproblem zu diagnostizieren und zu beheben.

Microsoft stellt NDF-Hilfsklassen bereit, von denen einige erweiterbar sind, damit Entwickler Problem Behandlungseinheiten erstellen können, die als Hilfsklassen Erweiterungen bezeichnet werden, um detailliertere Diagnosen speziell für bestimmte Software-oder Hardwarekomponenten bereitzustellen.

## <a name="where-applicable"></a>Anwendungsbereich

NDF kann von jeder beliebigen Hersteller Software verwendet werden, die von der Netzwerk Konnektivität abhängig ist.

## <a name="developer-audience"></a>Entwicklergruppe

Die NDF-API ist für C/C++-Entwickler konzipiert.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

NDF ist für Computer verfügbar, auf denen Windows Vista, Windows Server 2008 oder höher ausgeführt wird.
