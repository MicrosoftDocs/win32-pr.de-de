# <a name="network-diagnostics-framework"></a>Netzwerkdiagnoseframework

## <a name="purpose"></a>Zweck

Das Netzwerkdiagnoseframework (Network Diagnostics Framework, NDF) bietet Komponenten- und Anwendungsentwicklern eine Möglichkeit, die Netzwerkproblembehandlung für Benutzer zu vereinfachen. Benutzer können versuchen, ein Netzwerkproblem mit einem einzigen Problembehandlungstool zu diagnostizieren und zu reparieren.

Microsoft stellt NDF-Hilfsklassen bereit, von denen einige erweiterbar sind, sodass Entwickler Problembehandlungseinheiten erstellen können, die als Hilfsklassenerweiterungen bezeichnet werden, um detailliertere Diagnosen für bestimmte Software- oder Hardwarekomponenten bereitzustellen.

## <a name="where-applicable"></a>Anwendungsbereich

NDF kann von der Software eines beliebigen Anbieters verwendet werden, die auf Netzwerkkonnektivität basiert.

## <a name="developer-audience"></a>Entwicklergruppe

Die NDF-API ist für C/C++-Entwickler konzipiert.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

NDF ist für Computer verfügbar, auf denen Windows Vista, Windows Server 2008 oder höher ausgeführt wird.
