# <a name="using-ndf"></a>Verwenden von NDF

Das Network Diagnostics Framework (ndf) wird als Teil von Windows Vista, Windows Server 2008 und höher ausgeliefert. NDF kann von Softwareentwicklern verwendet werden, um Netzwerkprobleme zu diagnostizieren und zu beheben, die möglicherweise dazu führen, dass Ihre Anwendungen fehlschlagen. Entwickler können von Microsoft bereitgestellte APIs verwenden, die Ihre Anwendungen mit NDF verbinden. In einigen Fällen können Entwickler auch NDF-Hilfsklassen Erweiterungen erstellen, um Probleme zu diagnostizieren, die für Ihre Netzwerk Software spezifisch sind.
