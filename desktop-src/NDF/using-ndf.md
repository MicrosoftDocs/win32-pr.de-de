# <a name="using-ndf"></a>Verwenden von NDF

Das Framework für die Netzwerkdiagnose (Network Diagnostics Framework, NDF) ist im Rahmen von Windows Vista, Windows Server 2008 und höher enthalten. NDF kann von Softwareentwicklern verwendet werden, um Netzwerkprobleme zu diagnostizieren und zu beheben, die möglicherweise dazu führen, dass ihre Anwendungen ausfallen. Entwickler können von Microsoft bereitgestellte APIs verwenden, die ihre Anwendungen mit NDF verbinden. In einigen Fällen können Entwickler auch NDF-Hilfsklassenerweiterungen erstellen, um Probleme zu diagnostizieren, die spezifisch für ihre Netzwerksoftware sind.
