-   <span data-ttu-id="3da02-120">**WinHTTP- \_ Status \_ Rückruf Funktionen müssen Wiedereinstiegs fähig sein.**</span><span class="sxs-lookup"><span data-stu-id="3da02-120">**WINHTTP\_STATUS\_CALLBACK functions must be reentrant.**</span></span> <span data-ttu-id="3da02-121">Vermeiden Sie beim Schreiben eines Rückrufs statische Variablen oder andere Konstrukte, die bei erneuter Aufnahme unsicher sind, und vermeiden Sie das Aufrufen anderer Funktionen, die nicht wieder einentrant sind.</span><span class="sxs-lookup"><span data-stu-id="3da02-121">When writing a callback, avoid static variables or other constructs that are unsafe under reentrance, and avoid calling other functions that are not reentrant.</span></span>
