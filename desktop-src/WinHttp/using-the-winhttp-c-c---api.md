# <a name="using-the-winhttp-cc-api"></a>Verwenden der WinHTTP-C/C++-API

Dieser Abschnitt enthält Informationen, die für die Verwendung der Schnittstelle WinHTTP C/C++ spezifisch sind.

-   [WinHTTP-Sitzungen](winhttp-sessions-overview.md) beschreiben den grundlegenden Ablauf einer WinHTTP-Anwendung, einschließlich der Reihenfolge, in der Funktionen aufgerufen werden.
-   [HINTERNET Handles in WinHTTP](hinternet-handles-in-winhttp.md) beschreiben die verschiedenen Typen von Handles, die Benennungs Konventionen für diese Handles und ihre hierarchische Struktur.
-   [URL (Uniform Resource Locators) in WinHTTP](uniform-resource-locators--urls--in-winhttp.md) beschreibt, wie WinHTTP URLs verwendet, und stellt ein Codebeispiel zur Veranschaulichung der Verwendung der bereitgestellten Funktionen bereit.
-   [Authentifizierung in WinHTTP](authentication-in-winhttp.md) beschreibt die WinHTTP-Unterstützung für die Server-und Proxy Authentifizierung von HTTP-Sitzungen.
-   Die [Passport-Authentifizierung in WinHTTP](passport-authentication-in-winhttp.md) beschreibt die WinHTTP-Unterstützung für die Passport 1,4-Authentifizierung
-   [SSL in WinHTTP](ssl-in-winhttp.md) beschreibt die WinHTTP-Unterstützung für das Secure Sockets Layer (SSL)-Protokoll.
-   Die [Verwendung von WinHTTP als parallele Assembly](using-winhttp-as-a-side-by-side-assembly.md) zeigt ein Beispiel Anwendungs Manifest, das zum Verknüpfen mit WinHTTP verwendet werden kann.
-   [Bei der Cookiebehandlung in WinHTTP](cookie-handling-in-winhttp.md) werden verschiedene Optionen beschrieben, die WinHTTP-Client Anwendungen zum Verarbeiten von Cookies verwenden können.
-   [Fehlerbehandlung in WinHTTP](error-handling-in-winhttp.md) beschreibt, wie WinHTTP-Optionen Programm gesteuert von Ihrer C/C++-Anwendung aus festgelegt werden.
-   [Unterstützung von WinHTTP AutoProxy](winhttp-autoproxy-support.md) beschreibt die unterschiedlichen Methoden, mit denen WinHTTP-API-Funktionen Fehler melden.
