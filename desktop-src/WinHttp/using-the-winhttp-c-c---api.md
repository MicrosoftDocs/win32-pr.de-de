# <a name="using-the-winhttp-cc-api"></a>Verwenden der WinHTTP-C/C++-API

Dieser Abschnitt enthält Informationen zur Verwendung der WinHTTP-C/C++-Schnittstelle.

-   [WinHTTP-Sitzungen](winhttp-sessions-overview.md) beschreibt den grundlegenden Ablauf einer WinHTTP-Anwendung, einschließlich der Reihenfolge, in der Funktionen aufgerufen werden.
-   [HINTERNET Handles in WinHTTP](hinternet-handles-in-winhttp.md) beschreibt die verschiedenen Typen von Handles, die Namenskonventionen für diese Handles und ihre hierarchische Struktur.
-   [Uniform Resource Locators (URLs) in WinHTTP](uniform-resource-locators--urls--in-winhttp.md) beschreibt, wie WinHTTP URLs verwendet, und stellt ein Codebeispiel zur Veranschaulichung der Verwendung der bereitgestellten Funktionen zur Verfügung.
-   [Die Authentifizierung in WinHTTP](authentication-in-winhttp.md) beschreibt die WinHTTP-Unterstützung für die Server- und Proxyauthentifizierung von HTTP-Sitzungen.
-   [Die Passport-Authentifizierung in WinHTTP](passport-authentication-in-winhttp.md) beschreibt die WinHTTP-Unterstützung für die Passport 1.4-Authentifizierung.
-   [SSL in WinHTTP beschreibt](ssl-in-winhttp.md) die WinHTTP-Unterstützung für das Secure Sockets Layer (SSL)-Protokoll.
-   [Die Verwendung von WinHTTP als side-by-side Assembly](using-winhttp-as-a-side-by-side-assembly.md) zeigt ein Beispielanwendungsmanifest, das zum Verknüpfen mit WinHTTP verwendet werden kann.
-   [Cookiebehandlung in WinHTTP](cookie-handling-in-winhttp.md) beschreibt mehrere Optionen, die WinHTTP-Clientanwendungen zum Verarbeiten von Cookies verwenden können.
-   [Fehlerbehandlung in WinHTTP](error-handling-in-winhttp.md) beschreibt, wie WinHTTP-Optionen programmgesteuert über Ihre C/C++-Anwendung festgelegt werden.
-   [WinHTTP AutoProxy Support beschreibt](winhttp-autoproxy-support.md) die verschiedenen Möglichkeiten, wie WinHTTP-API-Funktionen Fehler melden.
