# <a name="winhttp-versions"></a>WinHTTP-Versionen

Benutzer der Microsoft Windows HTTP-Dienste (WinHTTP) sollten die neueste Version der Technologie verwenden, Version 5,1. Version 5,0 wird nicht mehr unterstützt.

## <a name="version-51"></a>Version 5.1

WinHTTP 5,1 bietet Verbesserungen gegenüber Version 5,0; Weitere Informationen zu neuen Funktionen finden Sie unter [What es New in WinHTTP 5,1](what-s-new-in-winhttp-5-1.md).

Bei Version 5,1 ist WinHTTP eine Betriebssystem Komponente der folgenden Betriebssysteme:

-   Windows 2000, Service Pack 3 und höher (mit Ausnahme des Datacenter-Servers)
-   Windows XP mit Service Pack 1 (SP1) und höher
-   Windows Server 2003 mit Service Pack 1 (SP1) und höher

Bei Windows Server 2003 mit SP1 ist WinHTTP eine parallele Assembly des Systems. Weitere Informationen finden Sie unter [](/windows/desktop/SbsCs/about-side-by-side-assemblies-)parallele Assemblys.

Die Neuverteilung von WinHTTP 5,1 außerhalb dieser Betriebssysteme wird nicht unterstützt.

## <a name="version-50"></a>Version 5.0

Die Redistributable-Komponente "winhttp5.dll" von WinHTTP 5,0 kann unter Windows 2000 oder höher oder Windows NT 4,0 mit Microsoft Internet Explorer 5,01 oder höher ausgeführt werden.

> [!IMPORTANT]
> Der WinHTTP 5,0-Download ist nicht mehr verfügbar. Ab dem 1. Oktober 2004 hat Microsoft das WinHTTP 5,0 SDK-Download von MSDN entfernt und hat den Produktsupport für Version 5,0 beendet.
