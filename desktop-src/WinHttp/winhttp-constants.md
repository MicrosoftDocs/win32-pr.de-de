# <a name="winhttp-constants"></a>WinHTTP-Konstanten

WinHTTP verwendet die folgenden Konstanten:

<dl> <dt>

[**Fehlermeldungen**](error-messages.md)
</dt> <dd>

Bestimmte Fehlermeldungen für die WinHTTP-Funktionen. Diese Funktionen geben ggf. auch Windows-Fehlermeldungen zurück. Der Wert, der den einzelnen Konstanten entspricht, ist der Wert der Konstante für die API-Funktionen (Application Programming Interface) und die unteren 16 Bits der Fehlernummer für das [**WinHttpRequest**](winhttprequest.md) -Objekt.

</dd> <dt>

[**HTTP-Status Codes**](http-status-codes.md)
</dt> <dd>

Konstanten und entsprechende Werte, die HTTP-Statuscodes angeben, die von Servern im Internet zurückgegeben werden.

</dd> <dt>

[**Optionsflags**](option-flags.md)
</dt> <dd>

Optionsflags, die von [**WinHttpQueryOption**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpqueryoption) und [**WinHttpSetOption**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpsetoption)unterstützt werden. Alle gültigen Optionsflags haben einen Wert größer oder gleich der Option WinHTTP \_ First \_ und kleiner als oder gleich WinHTTP \_ Last \_ .

</dd> <dt>

[**\_Internetport**](internet-port.md)
</dt> <dd>

Ein Word-Wert, der den Port angibt.

</dd> <dt>

[**Internet \_ Schema**](internet-scheme.md)
</dt> <dd>

Internet Schemas, die von WinHTTP unterstützt werden.

</dd> <dt>

[**Abfrageinfoflags**](query-info-flags.md)
</dt> <dd>

Attribute und modifiziererer, die von [**WinHttpQueryHeaders**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpqueryheaders)verwendet werden.

</dd> </dl>
