<span data-ttu-id="096ec-148">Im Allgemeinen werden Ausellipsen in Benutzeroberflächen verwendet, um auf Vollständigkeit hindeuten zu können.</span><span class="sxs-lookup"><span data-stu-id="096ec-148">Generally, ellipses are used in user interfaces to indicate incompleteness.</span></span> <span data-ttu-id="096ec-149">Befehle, die andere Fenster anzeigen, sind nicht unvollständig, sie müssen ein anderes Fenster anzeigen, und es sind keine zusätzlichen Informationen erforderlich, um ihre Aktion auszuführen.</span><span class="sxs-lookup"><span data-stu-id="096ec-149">Commands that show other windows aren't incomplete they must display another window and additional information isn't needed to perform their action.</span></span> <span data-ttu-id="096ec-150">Dieser Ansatz beseitigt unübersichtlichen Bildschirm in Situationen, in denen Ellipsen wenig Nutzen haben.</span><span class="sxs-lookup"><span data-stu-id="096ec-150">This approach eliminates screen clutter in situations where ellipses have little value.</span></span>

<span data-ttu-id="096ec-151">**Hinweis:** Wenn Sie bestimmen, ob für eine [Befehlsschaltfläche](winenv-uac.md) Auslassungsellipsen erforderlich sind, verwenden Sie nicht die Notwendigkeit, Berechtigungen als Faktor zu erhöhen.</span><span class="sxs-lookup"><span data-stu-id="096ec-151">**Note:** When determining if a command button needs an ellipsis, don't use the need to [elevate privileges](winenv-uac.md) as a factor.</span></span> <span data-ttu-id="096ec-152">Erhöhte Rechte sind keine Informationen, die zum Ausführen eines Befehls erforderlich sind (sondern aus Berechtigungsgründen), und die Notwendigkeit einer Erhöhung wird mit dem Sicherheitsschutz angezeigt.</span><span class="sxs-lookup"><span data-stu-id="096ec-152">Elevation isn't information needed to perform a command (rather, it's for permission) and the need to elevate is indicated with the security shield.</span></span>

<span data-ttu-id="096ec-153">**Wenn Sie nur eine Sache tun...** Verwenden Sie eine präzise, spezifische, selbsterklärende Bezeichnung, die die Aktion, die die Befehlsschaltfläche ausführt, eindeutig beschreibt, und verwenden Sie gegebenenfalls auslassungszeichen.</span><span class="sxs-lookup"><span data-stu-id="096ec-153">**If you do only one thing...** Use a concise, specific, self-explanatory label that clearly describes the action that the command button performs, and use an ellipsis when appropriate.</span></span>
