# <a name="single-image-compression"></a>Single-Image Komprimierung

Sie können die [**icimagecompress**](/windows/desktop/api/Vfw/nf-vfw-icimagecompress) -Funktion verwenden, um ein einzelnes Bild zu komprimieren. Diese Funktion gibt ein Handle der komprimierten geräteunabhängigen Bitmap (DIB) zurück. Das komprimierte DIB wird mithilfe des CF- \_ DIB-Formats verpackt.
