# <a name="single-image-compression"></a>Single-Image-Komprimierung

Sie können die [**ICImageCompress-Funktion verwenden,**](/windows/desktop/api/Vfw/nf-vfw-icimagecompress) um ein einzelnes Bild zu komprimieren. Diese Funktion gibt ein Handle der komprimierten geräteunabhängigen Bitmap (DIB) zurück. Der komprimierte DIB wird mithilfe des \_ CF-DIB-Formats gepackt.
