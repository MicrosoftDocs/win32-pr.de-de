# <a name="creating-an-avicap-capture-window"></a>Erstellen eines AVICap-Erfassungsfensters

Sie können ein Erfassungsfenster der WINDOWCap-Fensterklasse erstellen, indem Sie die [**capCreateCaptureWindow-Funktion**](/windows/desktop/api/Vfw/nf-vfw-capcreatecapturewindowa) verwenden. Diese Funktion gibt ein Fensterhand handle zurück, das das Erfassungsfenster identifiziert und von einer Anwendung verwendet wird, um nachfolgende Nachrichten an das Fenster zu senden.

Sie können ein oder mehrere Erfassungsfenster in einer Anwendung erstellen und jedes Erfassungsfenster mit einem anderen Erfassungsgerät verbinden.
