# <a name="creating-an-avicap-capture-window"></a>Erstellen eines avicap-Erfassungs Fensters

Sie können ein Aufzeichnungs Fenster der avicap-Fenster Klasse erstellen, indem Sie die [**capanatecapturewindow**](/windows/desktop/api/Vfw/nf-vfw-capcreatecapturewindowa) -Funktion verwenden. Diese Funktion gibt ein Fenster Handle zurück, das das Aufzeichnungs Fenster identifiziert und von einer Anwendung verwendet wird, um nachfolgende Nachrichten an das Fenster zu senden.

Sie können ein oder mehrere Erfassungsfenster in einer Anwendung erstellen und jedes Aufzeichnungs Fenster mit einem anderen Erfassungsgerät verbinden.
