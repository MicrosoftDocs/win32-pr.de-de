# <a name="about-the-audio-compression-manager"></a>Informationen zum Audiokomprimierungs-Manager

Der Audiokomprimierungs-Manager bietet Unterstützung auf Systemebene für die folgenden Dienste:

-   Transparente Lauf Zeit Audiokomprimierung und-Komprimierung
-   Waveform-audiodatenformatauswahl
-   Waveform-audiodatenfilter-Auswahl
-   Waveform-Konvertierung des audiodatenformats
-   Waveform-audiodatenfilterung
