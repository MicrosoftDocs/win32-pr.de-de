# <a name="windows-multimedia"></a>Windows Multimedia

## <a name="purpose"></a>Zweck

Microsoft Windows Multimediaunterstützung ermöglicht Anwendungen die Verwendung von Sound und Video.

## <a name="where-applicable"></a>Anwendungsbereich

Windows Multimedia kann in allen Windows Anwendungen verwendet werden.

## <a name="developer-audience"></a>Entwicklergruppe

Windows Multimedia ist für die Verwendung durch C/C++-Programmierer konzipiert. Sie müssen mit der Windows grafischen Benutzeroberfläche und nachrichtengesteuerten Architektur vertraut sein.
