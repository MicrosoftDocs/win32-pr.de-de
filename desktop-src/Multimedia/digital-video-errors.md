# <a name="digital-video-errors"></a>Digital-Video Fehler

Die folgenden zusätzlichen Rückgabewerte werden für Digital Video-Geräte definiert:



| Wert                           | Bedeutung                                                         |
|---------------------------------|-----------------------------------------------------------------|
| MCIAVI \_ ProductName             | Video                                                           |
| mcierr \_ AVI \_ audioerror         | Unbekannter Fehler beim Versuch, Audioinhalte wiederzugeben.                   |
| mcierr \_ AVI \_ badpalette         | Wechsel zur neuen Palette nicht möglich.                                |
| mcierr \_ AVI \_ cantplayfullscreen | Diese AVI-Datei kann nicht im Vollbildmodus abgespielt werden.             |
| mcierr \_ AVI \_ displayerror       | Unbekannter Fehler beim Versuch, Video anzuzeigen.                |
| mcierr \_ AVI \_ nokompressor       | Der installierbare Kompressor, der zum Abspielen dieser Datei erforderlich ist, kann   |
| mcierr \_ AVI \_ nodispdib          | 256 Farben-VGA-Modus nicht verfügbar.                               |
| mcierr \_ AVI \_ notinterleaved     | Diese AVI-Datei ist nicht verschachtelt.                               |
| mcierr \_ AVI \_ oldaviformat       | Diese AVI-Datei hat ein veraltetes Format.                         |
| mcierr \_ AVI \_ toobigforvga       | Diese AVI-Datei ist zu groß, um im ausgewählten VGA-Modus wiedergegeben zu werden. |
