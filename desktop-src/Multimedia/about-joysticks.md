# <a name="about-joysticks"></a>Informationen zu Joysticks

Der Joystick ist ein hilfeingabegerät für Anwendungen, die Alternativen zur Verwendung von Tastatur und Maus bereitstellen. Der Joystick bietet Positionsinformationen in einem Koordinatensystem, das in jeder Achse der Bewegung absolute höchst-und Mindestwerte aufweist.

Joystick Dienste werden geladen, wenn das Betriebssystem gestartet wird. Die Joystick Dienste können gleichzeitig zwei Joysticks überwachen, die jeweils über eine zwei-oder drei Achsenbewegung verfügen. Jeder Joystick kann bis zu vier Schaltflächen enthalten. Mithilfe der Joystick Funktionen können Sie die Funktionen der Joystick und des Joystick Treibers ermitteln. Außerdem können Sie die Positions-und Schaltflächen Informationen eines Joysticks verarbeiten, indem Sie den Joystick direkt Abfragen oder indem Sie den Joystick erfassen und Nachrichten daraus verarbeiten. Die letztere Methode ist einfacher, da die Anwendung den Joystick nicht manuell Abfragen oder die Zeit zum Generieren von Abfragen in regelmäßigen Abständen nachverfolgen muss.
