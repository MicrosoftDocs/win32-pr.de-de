# <a name="single-image-decompression"></a>Single-Image-Komprimierung

Sie können die [**icimagedebug**](/windows/desktop/api/Vfw/nf-vfw-icimagedecompress) -Funktion verwenden, um ein einzelnes Bild zu komprimieren. Diese Funktion gibt ein Handle des komprimierten DIB zurück. Das dekomprimierte DIB wird im CF \_ DIB-Format gespeichert.
