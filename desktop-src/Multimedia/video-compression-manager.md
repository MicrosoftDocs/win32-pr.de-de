# <a name="video-compression-manager"></a>Videokomprimierungs-Manager

Der Videokomprimierungs-Manager (VCM) ermöglicht den Zugriff auf die Schnittstelle, die von installierbaren Kompressoren verwendet wird, um Echtzeitdaten zu verarbeiten. Anwendungen können installierbare Kompressoren verwenden, um die folgenden Aufgaben auszuführen:

-   Komprimieren und decokomprimieren von Videodaten.
-   Senden Sie einen Renderer komprimierte Videodaten, und lassen Sie Sie auf die Anzeige zeichnen.
-   Komprimieren, decokomprimieren oder Zeichnen von Daten mit Anwendungs definierten Renderer.
-   Verwenden Sie Renderer, um Text und benutzerdefinierte Daten zu verarbeiten.
