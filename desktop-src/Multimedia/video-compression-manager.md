# <a name="video-compression-manager"></a>Videokomprimierungs-Manager

Der Videokomprimierungs-Manager (Video Compression Manager, VCM) ermöglicht den Zugriff auf die Schnittstelle, die von installierbaren Komprimierungsdateien für die Verarbeitung von Echtzeitdaten verwendet wird. Anwendungen können installierbare Beschriftungen verwenden, um die folgenden Aufgaben auszuführen:

-   Komprimieren und Dekomprimieren von Videodaten.
-   Senden Sie komprimierte Videodaten eines Renderers, und zeichnen Sie sie in die Anzeige.
-   Komprimieren, dekomprimieren oder zeichnen Sie Daten mit anwendungsdefinierten Renderern.
-   Verwenden Sie Renderer, um Text und benutzerdefinierte Daten zu verarbeiten.
