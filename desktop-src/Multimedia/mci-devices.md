# <a name="mci-devices"></a>MCI-Geräte

Jedes Multimediagerät in MCI unterstützt einen Kernsatz von MCI-Befehlen auf eine Weise, die für dieses Gerät sinnvoll ist. Beispielsweise bewirkt der Befehl [**Play**](play.md) ([**MCI \_ Play**](mci-play.md)), dass das geöffnete Gerät eine Datei oder einen Titel wieder gibt, unabhängig davon, mit welcher Art von Daten das Gerät arbeitet. In den folgenden Themen werden MCI-Geräte und ihre Reaktion auf standardmäßige MCI-Befehle erörtert.

-   [Gerätesteuerung](device-control.md)
-   [Wiedergabe und Positionierung](playback-and-positioning.md)
-   [Gerätetypen](device-types.md)
-   [Gerätenamen](device-names.md)
-   [Treiberunterstützung für MCI-Befehle](driver-support-for-mci-commands.md)
-   [Standardverhalten von Treibern](default-behavior-of-drivers.md)
