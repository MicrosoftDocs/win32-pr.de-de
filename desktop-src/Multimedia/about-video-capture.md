# <a name="about-video-capture"></a>Informationen zur Video Erfassung

Avicap unterstützt Streaming-Video Erfassung und Einzel Frame Erfassung in Echtzeit. Außerdem bietet avicap die Steuerung von Videoquellen, bei denen es sich um MCI-Geräte (Media Control Interface) handelt, sodass der Benutzer die Start-und Positions Positionen einer Videoquelle Steuern und den Erfassungs Vorgang erweitern kann, um die Schritt Frame Erfassung einzubeziehen.

Die Fenster, die Sie mithilfe der avicap-Fenster Klasse erstellen, können die folgenden Aufgaben ausführen:

-   Erfassen Sie Audiodaten und Videodaten Ströme in einer überlappenden (AVI) Datei.
-   Verbinden und trennen Sie Video-und Audioeingabegeräte dynamisch.
-   Zeigen Sie ein Live eingehender Videosignal mithilfe der Overlay-oder Vorschau Methoden an.
-   Geben Sie eine Datei an, die beim Erfassen und Kopieren des Inhalts der Erfassungs Datei in eine andere Datei verwendet werden soll.
-   Legen Sie die Erfassungs Rate fest.
-   Anzeigen von Dialogfeldern, die die Videoquelle und das Format steuern.
-   Erstellen, speichern und Laden von Paletten.
-   Kopieren Sie Bilder und Paletten in die Zwischenablage.
-   Erfassen und speichern Sie ein einzelnes Image als geräteunabhängige Bitmap (DIB).
