# <a name="about-mci"></a>Informationen zu MCI

MCI bietet Anwendungen geräteunabhängige Funktionen zum Steuern von Audiodaten und visuellen Peripheriegeräten. Die Anwendung kann MCI verwenden, um alle unterstützten Multimedia-Geräte zu steuern, darunter Waveform-Audiogeräte, MIDI Sequencer, CD-Audiogeräte und Digital Videowiedergabegeräte (Videowiedergabe).
