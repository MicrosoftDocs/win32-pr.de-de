# <a name="setaudio-command"></a>Befehl "SetAudio"

Der setaudiobefehl legt Werte fest, die mit Audiowiedergabe und-Erfassung verknüpft sind. Dieser Befehl wird von Digital-Video-und VCR-Geräten erkannt.

Um diesen Befehl zu senden, wenden Sie die [**mciSendString**](/previous-versions//dd757161(v=vs.85)) -Funktion mit dem festgelegten *lpszcommand* -Parameter wie folgt an.

``` syntax
_stprintf_s(
  lpszCommand, 
  TEXT("setaudio %s %s %s"), 
  lpszDeviceID, 
  lpszAudio, 
  lpszFlags
); 
```

## <a name="return-value"></a>Rückgabewert

Gibt 0 (null) zurück, wenn erfolgreich, andernfalls einen Fehler.

## <a name="remarks"></a>Bemerkungen

Bei VCR-Geräten bewirkt die Verwendung von setaudioverwendung mit einem Flag, das eine einzelne Spur deaktiviert ("Track *Track \_ Number* Off"), eine Statusmeldung, die besagt, dass der Befehl nicht ausgeführt werden konnte. Einige VCRs können nur die Kombinationen von Spuren und nicht einzelne Spuren deaktivieren. beispielsweise die erste Audiospur und eine Videospur einer Video-Kassette. Verwenden Sie in diesem Fall einfach setaudiodatei und [setvideo](setvideo.md) , um die anderen Titel zu deaktivieren, die die Kombination bilden. Der Treiber schaltet die Spuren aus, wenn er den Befehl empfängt, um den letzten Titel in der Kombination zu deaktivieren.
