# <a name="embedding-information-chunks-in-an-avi-file"></a>Einbetten von Informations Blöcken in eine AVI-Datei

Sie können Informationsblöcke (z. b. Text oder benutzerdefinierte Daten) in eine AVI-Datei einfügen, indem Sie die [**\_ \_ \_ \_ infochunk**](wm-cap-file-set-infochunk.md) -Nachricht des WM-Cap-Datentyps (oder das-Makro [**capflesetinfochunk**](/windows/desktop/api/Vfw/nf-vfw-capfilesetinfochunk) ) verwenden. Sie können diese Meldung auch zum Löschen von Informations Blöcken aus einer AVI-Datei verwenden.
