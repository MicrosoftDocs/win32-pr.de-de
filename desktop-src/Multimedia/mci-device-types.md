# <a name="mci-device-types"></a>MCI-Gerätetypen

Die folgenden Werte identifizieren Geräte in MCI-Nachrichten und -Strukturen:



| Wert                         | Bedeutung                       |
|-------------------------------|-------------------------------|
| MCI \_ ALL \_ DEVICE \_ ID          | Jedes Gerät                    |
| MCI \_ DEVTYPE \_ ANIMATION       | Animationswiedergabegerät     |
| MCI \_ DEVTYPE \_ CD \_ AUDIO       | CD-Audiogerät               |
| MCI \_ DEVTYPE \_ DAT             | Digitales Audiobandgerät     |
| MCI \_ DEVTYPE \_ DIGITAL \_ VIDEO  | Wiedergabegerät für digitale Videos |
| MCI \_ DEVTYPE \_ OTHER           | Nicht definiertes Gerät              |
| MCI \_ DEVTYPE \_ OVERLAY         | Videoüberlagerungsgerät          |
| MCI \_ DEVTYPE \_ SCANNER         | Scannergerät                |
| MCI \_ DEVTYPE \_ SEQUENCER       | GERÄT DES SEQUENCER         |
| MCI \_ DEVTYPE \_ VCR             | Video-Cassette-Aufzeichnung       |
| MCI \_ DEVTYPE \_ VIDEODISC       | Videodisc-Player              |
| MCI \_ DEVTYPE \_ WAVEFORM \_ AUDIO | Waveform-Audiogerät         |
