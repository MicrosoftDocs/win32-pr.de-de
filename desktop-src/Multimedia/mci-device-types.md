# <a name="mci-device-types"></a>MCI-Gerätetypen

Die folgenden Werte identifizieren Geräte in MCI-Nachrichten und-Strukturen:



| Wert                         | Bedeutung                       |
|-------------------------------|-------------------------------|
| MCI \_ alle \_ Geräte- \_ ID          | Beliebiges Gerät                    |
| MCI \_ devtype- \_ Animation       | Animation-Wiedergabe-Gerät     |
| MCI \_ devtype \_ \_ -CD-Audiodatei       | CD-Audiogerät               |
| MCI \_ devtype- \_ DAT             | Digitales audiobandgerät     |
| digitales MCI \_ devtype- \_ \_ Video  | Digitales Videowiedergabe Gerät |
| MCI- \_ devtype \_ other           | Nicht definiertes Gerät              |
| MCI \_ devtype- \_ Overlay         | Video Überlagerungs Gerät          |
| MCI \_ devtype- \_ Scanner         | Scannergerät                |
| MCI \_ devtype \_ Sequencer       | MIDI-Sequencer-Gerät         |
| MCI \_ devtype \_ VCR             | Video-Kassetten Aufzeichnung       |
| MCI \_ devtype- \_ Videodisk       | Videodisk-Player              |
| MCI \_ devtype \_ Waveform \_ -Audiodatei | Waveform-Audiogerät         |
