# <a name="level-2-audio-requirements"></a>Audioanforderungen der Ebene 2

Das Audiosubsystem eines PCs, der die Spezifikation der Ebene 2 erfüllt, umfasst die folgenden Elemente:

-   Ein CD-ROM-Treiber mit CD-da-Ausgaben (Red Book-Audiodaten) und volumesteuerung
-   Eine 16-Bit-DAC mit folgenden Merkmalen:
    -   Lineare PCM (Pulse Code Modulation) Sampling
    -   DMA-oder FIFO-gepufferte Übertragungsfunktion mit Interruptzeit für Puffer leer
    -   Verbindliche Stichproben Raten von 44,1, 22,05 und 11,025 kHz
    -   Stereo Kanäle
    -   CPU-Bandbreitennutzung von 10% oder weniger, wenn Audiodaten der 22,05-oder 11,025 kHz-Stichprobenrate oder eine CPU-Bandbreite von 15 Prozent oder weniger beträgt, wenn die Audioausgabe der 44,1 kHz-Stichprobenrate übersteigt

<!-- -->

-   Ein 16-Bit-ADC mit folgenden Merkmalen:
    -   Lineare PCM-Sampling
    -   DMA-oder FIFO-gepufferte Übertragungsfunktion mit Interruptzeit für Puffer leer
    -   Verbindliche Stichproben Raten von 44,1, 22,05 und 11,025 kHz
    -   Mikrofoneingabe

<!-- -->

-   Interne Synthesizer-Funktionen mit MultiVoice, multitimbral, sechs gleichzeitigen Melodien Notizen und zwei gleichzeitigen Percussive-Notizen
-   Interne Mischung mit den folgenden Funktionen:
    -   Kann drei Audioquellen kombinieren und die Ausgabe als Stereosignal auf Zeilenebene im Hintergrund anzeigen.
    -   Das Mischen von Quellen sind CD-rotbook-Audiodaten, Synthesizer und DAC.
    -   Jede Mischungs Quelle hat eine 3-Bit-volumesteuerung mit einem logarithmischen anzupassen.
