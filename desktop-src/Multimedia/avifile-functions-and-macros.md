# <a name="avifile-functions-and-macros"></a>Avifile-Funktionen und-Makros

Avifile-Funktionen und-Makros ermöglichen den Zugriff auf zeitbasierte Dateien, die das Format der Ressourcen Informationsdatei (Riff) verwenden, wie z. b. Waveform-Audiodaten und Audiovideo-überlappende (AVI)-Dateien. Diese Funktionen und Makros verwalten die Riff-Dateien, sodass Sie die Riff-Architektur nicht mehr verwalten und durch navigieren können.
