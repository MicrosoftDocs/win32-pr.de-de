# <a name="mci"></a>Mci

Die Media Control Interface (MCI) bietet Standardbefehle zum Wiedergeben von Multimediageräten und Aufzeichnen von Multimediaressourcendateien. Diese Befehle sind eine generische Schnittstelle für nahezu jede Art von Multimediagerät.

-   [Verwenden von MCI-Befehlsmeldungen](using-mci-command-messages.md)
-   [MCI-Referenz](mci-reference.md)
