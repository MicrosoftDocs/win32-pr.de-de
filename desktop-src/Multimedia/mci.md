# <a name="mci"></a>MCI

Die Media Control Interface (MCI) stellt Standard Befehle für die Wiedergabe von Multimedia-Geräten und die Aufzeichnung von Multimedia-Ressourcen Dateien bereit. Diese Befehle sind eine generische Schnittstelle für nahezu jede Art von multimediateigerät.
