Sie können Daten mit einem Aufzeichnungs Fenster verknüpfen, indem Sie die [**\_ \_ \_ Benutzer \_ Daten**](wm-cap-set-user-data.md) [**-nach**](/windows/desktop/api/Vfw/nf-vfw-capgetuserdata) richten "WM-Abdeckung [**\_ \_ get \_ User \_ Data**](wm-cap-get-user-data.md) [**" und "**](/windows/desktop/api/Vfw/nf-vfw-capsetuserdata) WM Cap" festlegen. Sie können einen **Long** -Datenwert abrufen, indem Sie die Benutzerdaten Meldung "WM \_ -Abdeckung \_ get \_ User Data" und " \_ **Long** Data" mithilfe der \_ Meldung " \_ \_ Benutzer \_ Daten für die WM-Abdeckung festlegen" festlegen.
