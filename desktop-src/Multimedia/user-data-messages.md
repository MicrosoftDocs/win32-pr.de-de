# <a name="user-data-messages"></a>Benutzerdatenmeldungen

Sie können Einem Erfassungsfenster Daten zuordnen, indem Sie die [**MELDUNGEN WM CAP GET USER \_ \_ \_ \_ DATA**](wm-cap-get-user-data.md) und [**WM CAP SET USER \_ \_ \_ \_ DATA**](wm-cap-set-user-data.md) (oder die Makros [**capGetUserData**](/windows/desktop/api/Vfw/nf-vfw-capgetuserdata) und [**capSetUserData)**](/windows/desktop/api/Vfw/nf-vfw-capsetuserdata) verwenden. Sie können  einen LONG-Datenwert mithilfe der WM CAP GET USER DATA-Meldung abrufen und einen LONG-Datenwert mithilfe der \_ WM CAP SET USER \_ \_ \_  \_ \_ \_ \_ DATA-Meldung festlegen.
