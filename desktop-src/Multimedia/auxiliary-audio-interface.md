# <a name="auxiliary-audio-interface"></a>Hilfsaudioschnittstelle

Bei zusätzlichen Audiogeräten handelt es sich um Audiogeräte, deren Ausgabe mit den Ausgabegeräten von MIDI und Waveform-Audiodaten auf einem multimediakcomputer gemischt ist. Ein Beispiel für ein zusätzliches Audiogerät ist die CD-Audioausgabe eines CD-ROM-Laufwerks.

-   [Abfragen von zusätzlichen Audiogeräten](querying-auxiliary-audio-devices.md)
-   [Ändern des Umfangs von zusätzlichen Audiogeräten](changing-the-volume-of-auxiliary-audio-devices.md)
