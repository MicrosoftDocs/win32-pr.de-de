# <a name="auxiliary-audio-interface"></a>Zusätzliche Audioschnittstelle

Bei zusätzlichen Audiogeräten handelt es sich um Audiogeräte, deren Ausgabe mit den Audioausgabegeräten DES -Audioformats und waveform-audio auf einem Multimediacomputer gemischt wird. Ein Beispiel für ein zusätzliches Audiogerät ist die CD-Audioausgabe von einem CD-ROM-Laufwerk.

-   [Abfragen von zusätzlichen Audiogeräten](querying-auxiliary-audio-devices.md)
-   [Ändern der Lautstärke von zusätzlichen Audiogeräten](changing-the-volume-of-auxiliary-audio-devices.md)
