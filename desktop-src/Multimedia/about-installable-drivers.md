# <a name="about-installable-drivers"></a>Informationen zu installierbaren Treibern

Ein installier barer Treiber ist eine Dynamic Link Library (dll), die eine Standardschnittstelle bereitstellt, über die Windows-basierte Anwendungen und DLLs mit dem Treiber kommunizieren und ihn verwalten. Installierbare Treiber werden am häufigsten als Multimedia-Gerätetreiber verwendet, aber installierbare Treiber können auch für andere Zwecke verwendet werden und sind besonders nützlich in Situationen, in denen eine Standardschnittstelle und die Steuerung mehrerer Instanzen erforderlich sind.
