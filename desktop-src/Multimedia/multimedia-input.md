# <a name="multimedia-input"></a>Multimediaeingabe

Windows stellt Funktionen bereit, mit denen eine Anwendung Multimediaeingaben empfangen kann. Die Multimediaeingabefunktionen wurden in 16-Bit-Windows eingeführt. Viele ihrer Features wurden durch Features in DirectX ersetzt. Weitere Informationen finden Sie in der DirectX-Dokumentation.

Die folgenden Themen werden in der Dokumentation zu Multimediaeingaben erläutert.

-   [Multimediadatei-E/A](multimedia-file-i-o.md)
-   [Joysticks](joysticks.md)
-   [Multimedia-Timer](multimedia-timers.md)
