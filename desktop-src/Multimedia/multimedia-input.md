# <a name="multimedia-input"></a>Multimedia-Eingabe

Windows stellt Funktionen bereit, die es einer Anwendung ermöglichen, multimediaeingaben zu empfangen. Die Multimedia-Eingabefunktionen wurden in 16-Bit-Fenstern eingeführt. Viele ihrer Features wurden durch Features in DirectX abgelöst. Weitere Informationen finden Sie in der DirectX-Dokumentation.

Die folgenden Themen werden in der Multimedia-Eingabe Dokumentation erläutert.

-   [Multimedia-Datei-e/a](multimedia-file-i-o.md)
-   [Joystick](joysticks.md)
-   [Multimedia-Timer](multimedia-timers.md)
