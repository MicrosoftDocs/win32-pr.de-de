# <a name="using-installable-drivers"></a>Verwenden installier barer Treiber

Sie verwenden installierbare Treiber, um Anwendungen oder DLLs eine Standardmethode für den Zugriff auf ein Gerät oder eine Reihe nützlicher Routinen zu vergeben. In den folgenden Abschnitten wird gezeigt, wie ein installier barer Treiber mithilfe einer [driverproc](/windows/win32/api/mmiscapi/nc-mmiscapi-driverproc) -Funktion erstellt wird und wie ein installier barer Treiber geöffnet wird, um nützliche Aufgaben auszuführen.
