# <a name="frame-windows-multimedia"></a>Frame (Windows Multimedia)

Ein Erfassungsfenster verwendet Framerückrufbenachrichtigungsmeldungen, um Ihre Anwendung zu benachrichtigen, wenn ein neuer Videoframe verfügbar ist. Das Erfassungsfenster aktiviert diese Rückrufbenachrichtigungen nur, wenn die Vorschaurate ungleich 0 (null) ist und die Streamingerfassung nicht ausgeführt wird.
