# <a name="frame-windows-multimedia"></a>Frame (Windows Multimedia)

Ein Aufzeichnungs Fenster verwendet Frame Rückruf-Benachrichtigungs Meldungen, um Ihre Anwendung zu benachrichtigen, wenn ein neuer Videoframe verfügbar ist. Das Erfassungsfenster aktiviert diese Rückruf Benachrichtigungen nur dann, wenn die Vorschau Rate nicht NULL ist und die streamingerfassung nicht ausgeführt wird.
