# <a name="multimedia-pc-specifications"></a>Multimedia-PC-Spezifikationen

Der Marketing-Rat von Multimedia-PCs (MPC) hat zwei Spezifikationen entwickelt, um die Einführung von Multimedia-Funktionen zu fördern. Die Spezifikation der Ebene 1, entwickelt in 1990, bietet eine grundlegende Definition von Multimedia Computing in Funktionalität, Hardwarekomponenten und Softwarekomponenten. Die in 1993 ausgegebene Spezifikation der Ebene 2 baut auf der ersten Spezifikation auf und konzentriert sich auf erweiterte Multimediafunktionen, seit die erste Spezifikation ausgegeben wurde.

In diesem Anhang werden die Spezifikationen zusammengefasst.

-   [Spezifikation der Ebene 1](level-1-specification.md)
-   [Spezifikation der Ebene 2](level-2-specification.md)
