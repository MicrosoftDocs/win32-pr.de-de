# <a name="multimedia-playback"></a>Multimedia-Wiedergabe

Die [**mciwndcreate**](/windows/desktop/api/Vfw/nf-vfw-mciwndcreatea) -Funktion bietet die Möglichkeit, ein mciwnd-Fenster und das zugeordnete Gerät zu steuern. Im allgemeinen registriert diese Funktion die mciwnd-Fenster Klasse und erstellt ein mciwnd-Fenster für die Verwendung von MCI-Diensten. In diesem Abschnitt wird beschrieben, wie Sie **mciwndcreate** verwenden, um die folgenden Aufgaben auszuführen:

-   [Hinzufügen User-Controlled Wiedergabe](adding-user-controlled-playback.md)
-   [Automatisieren der Wiedergabe](automating-playback.md)
-   [Ändern des mciwnd-Fensters mithilfe von Fenster Stilen](using-window-styles-to-change-the-mciwnd-window.md)
-   [Zulassen der Angabe von Geräten und Dateien durch den Benutzer](allowing-the-user-to-specify-devices-and-files.md)
