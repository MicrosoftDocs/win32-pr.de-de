# <a name="level-2-optical-storage"></a>Level 2 Optical Storage

Das optische Speichergerät mit minimaler Leistung ist ein CD-ROM-Laufwerk mit doppelter Geschwindigkeit, das die folgenden Kriterien erfüllt:

-   Eine dauerhafte Datenübertragungsrate von 300 Kilobyte pro Sekunde
-   Eine CPU-Bandbreitenauslastung von 40 Prozent oder weniger, wenn eine dauerhafte Datenübertragungsrate von 150 Kilobyte pro Sekunde beibehalten wird, oder eine CPU-Bandbreite von 60 Prozent oder weniger, wenn eine dauerhafte Datenübertragungsrate von 300 Kilobyte pro Sekunde beibehalten wird
-   Eine durchschnittliche Suchzeit von 400 Millisekunden oder weniger
-   10.000 Stunden durchschnittliche Zeit zwischen Fehlern
-   CD-ROM XA-bereit (Modus 1-fähig, Modus 2 form 1-fähig, Modus form 2-fähig)
-   Multisessionfähig
-   MSCDEX-2.2-Treiber oder gleichwertig, der die erweiterten Audiofunktionen implementiert

Die empfohlene CPU-Bandbreite sollte erreicht werden, indem eine Leseblockgröße von mindestens 16.000 und eine Vorlaufzeit von maximal der Zeit verwendet wird, die zum Laden eines Leseblocks von Daten in den CD-ROM-Puffer benötigt wird.
