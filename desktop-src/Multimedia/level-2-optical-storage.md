# <a name="level-2-optical-storage"></a>Optischer Speicher auf Ebene 2

Bei dem optischen minimal Speichergerät handelt es sich um ein CD-ROM-Laufwerk mit doppelter Geschwindigkeit, das die folgenden Kriterien erfüllt:

-   Eine dauerhafte Datenübertragungsrate von 300 Kilobyte pro Sekunde
-   Eine CPU-Bandbreitenauslastung von 40 Prozent oder weniger, wenn eine dauerhafte Datenübertragungsrate von 150 Kilobyte pro Sekunde beibehalten wird, oder eine CPU-Bandbreite von 60 Prozent oder weniger, wenn eine dauerhafte Datenübertragungsrate von 300 Kilobyte pro Sekunde beibehalten wird.
-   Eine durchschnittliche Suchzeit von 400 Millisekunden oder weniger
-   Eine 10.000-Stunden-Bewertung mit einer Zwischenzeit zwischen Fehlern
-   CD-ROM XA Ready (Modus 1 fähig, Modus 2 Formular 1 fähig, Modus 2 fähig)
-   Multisitzungsfähig
-   MSCDEX-2,2-Treiber oder äquivalent, der erweiterte Audiofunktionen implementiert

Die empfohlene CPU-Bandbreite sollte erreicht werden, indem eine Lese Blockgröße von mindestens 16K und eine Vorlaufzeit nicht mehr als die benötigte Zeit zum Laden eines Lese Blocks in den CD-ROM-Puffer verwendet werden.
