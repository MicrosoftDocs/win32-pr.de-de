# <a name="level-1-specification"></a>Level 1-Spezifikation

Die Level 1-Spezifikation wurde entwickelt, um die Einführung grundlegender Multimediafunktionen auf einer Mindestleistungsebene zu fördern. Die gängigsten Multimediakomponenten, die im Marketplace verfügbar waren, als die Level 1-Spezifikation ausgestellt wurde, umfassten mehrere Compact Disks – schreibgeschützte Speicherlaufwerke (CD-ROM), die Daten mit dauerhaften Übertragungsraten zwischen 90 und 150 Kilobyte pro Sekunde, 8-Bit-Soundkarten und 16-Farb- und 256-Farbadaptern (SVGA) bereitgestellt haben.
