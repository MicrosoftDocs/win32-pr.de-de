# <a name="about-audio-mixers"></a>Informationen zu audiomixern

Audiomischdienste steuern das Routing von audiolinien zu einem Zielgerät zum Wiedergeben oder aufzeichnen. Diese Dienste können auch das Volume und andere Auswirkungen steuern. Viele der Techniken, die für die Verwendung dieser Dienste erforderlich sind, ähneln denen für Audiogeräte, die in anderen Multimedia-Übersichten erläutert werden.
