# <a name="drawdib"></a>DrawDib

Die DrawDib-Funktionen bieten leistungsstarke Bildzeichnungsfunktionen für geräteunabhängige Bitmaps (DIBs). DrawDib-Funktionen unterstützen DIBs mit 8-Bit-, 16-Bit-, 24-Bit- und 32-Bit-Bildtiefe.

DrawDib-Funktionen schreiben direkt in den Videospeicher. Sie verlassen sich nicht auf Funktionen der Grafikgeräteschnittstelle (GDI).

-   [Informationen zu DrawDib-Funktionen](about-the-drawdib-functions.md)
-   [Verwenden von DrawDib](using-drawdib.md)
-   [DrawDib-Referenz](drawdib-reference.md)
