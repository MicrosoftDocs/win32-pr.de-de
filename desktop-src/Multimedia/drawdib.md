# <a name="drawdib"></a>Drawdib

Die drawdib-Funktionen bieten hochleistungsfähige Bild Zeichnungsfunktionen für geräteunabhängige Bitmaps (DIBs). Die drawdib-Funktionen unterstützen DIBS von 8-Bit-, 16-Bit-, 24-Bit-und 32-Bit-Bild tiefen.

Drawdib-Funktionen schreiben direkt in den Videospeicher. Sie sind nicht auf Funktionen der Graphics Device Interface (GDI) angewiesen.

-   [Verwenden von drawdib](using-drawdib.md)
-   [Drawdib-Referenz](drawdib-reference.md)
