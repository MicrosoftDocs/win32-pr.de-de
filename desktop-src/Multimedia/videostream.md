# <a name="videostream"></a>Videostream

Anwendungen können videostream-Rückruffunktionen während der Streamingerfassung verwenden, um einen erfassten Videoframe zu verarbeiten. Das Erfassungsfenster ruft eine Videostream-Rückruffunktion auf, bevor jeder erfasste Frame auf den Datenträger geschrieben wird.
