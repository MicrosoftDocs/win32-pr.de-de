# <a name="videostream"></a>Videostream

Anwendungen können Videostream-Rückruf Funktionen während der streamingerfassung verwenden, um einen aufgezeichneten Videoframe zu verarbeiten. Das Erfassungsfenster Ruft eine Videostream-Rückruffunktion auf, kurz bevor jeder erfasste Frame auf den Datenträger geschrieben wird.
