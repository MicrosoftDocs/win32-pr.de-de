# <a name="level-1-audio-requirements"></a>Audioanforderungen der Ebene 1

Das Audiosubsystem eines PCs, der die Spezifikation der Ebene 1 erfüllt, umfasst die folgenden Elemente:

-   Ein 8-Bit-DAC (Digital-to-Analog Converter), mit dem Waveform-Audiodateien verarbeitet werden können, die bei 22,05-und 11,025 kHz-Stichproben Raten aufgezeichnet wurden
-   Ein 8-Bit-ADC-zu-Digital Converter (ADC), der Waveform-Audiodateien mit der Samplingrate von 11,025 kHz über eine externe Quelle (z. b. ein Mikrofon) aufzeichnen kann
-   Interne Synthesizer-Funktionen mit vier oder neun MultiVoice-, multitimbral-Kapazität und zwei gleichzeitigen Percussive-Notizen
