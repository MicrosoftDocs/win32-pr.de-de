# <a name="wavestream"></a>WaveStream

Anwendungen verwenden die WaveStream-Rückruf Funktionen während der streamingerfassung, um Audiopuffer zu verarbeiten. Das Erfassungsfenster Ruft eine WaveStream-Rückruffunktion auf, kurz bevor jeder Audiopuffer auf den Datenträger geschrieben wird.
