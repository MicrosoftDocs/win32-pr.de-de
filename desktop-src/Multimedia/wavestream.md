# <a name="wavestream"></a>Wavestream

Anwendungen verwenden die Wavestream-Rückruffunktionen während der Streamingerfassung, um Audiopuffer zu verarbeiten. Das Erfassungsfenster ruft eine Wavestream-Rückruffunktion kurz vor dem Schreiben jedes Audiopuffers auf den Datenträger auf.
