# <a name="basic-capture-options"></a>Grundlegende Erfassungs Optionen

Wenn Sie einen oder mehrere der in der [**captuprojektstruktur**](/windows/win32/api/vfw/ns-vfw-captureparms) definierten Aufzeichnungsparameter ändern, können Sie die folgenden Aufgaben ausführen:

-   Ändern Sie die Frame Erfassungs Rate.
-   Geben Sie das Tastatur-oder Maus Steuerelement zum Beenden einer Aufzeichnungs Sitzung an.
-   Geben Sie eine Dauer für eine Aufzeichnungs Sitzung an.
