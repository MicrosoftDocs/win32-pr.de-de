# <a name="capture-windows"></a>Erfassungsfenster

Erfassungsfenster ähneln Standard Steuerelementen (z. b. Schaltflächen, Listenfelder oder Scrollleisten) konzeptionell. In der Regel verwenden Erfassungsfenster die \_ Fenster Stile "WS Child" und "WS Visible" \_ .
