# <a name="about-waveform-audio"></a>Informationen zu Waveform-Audiodaten

Es gibt mehrere Methoden zum Hinzufügen von Sound zu Ihrer Anwendung mithilfe von Wellenform-Audio. Die einfachste Methode, die hier dokumentiert ist, ist die Verwendung der Funktion [**PlaySound**](/previous-versions//dd743680(v=vs.85)) . Die meisten anderen Waveform-audioapi-Elemente sind relativ niedrig.

Weitere Informationen finden Sie unter den folgenden Themen:

-   [Einfache Audiowiedergabe](simple-audio-playback.md)
-   [Waveform-Audioschnittstelle](waveform-audio-interface.md)
-   [Hilfsaudioschnittstelle](auxiliary-audio-interface.md)
-   [Audiodatenblöcke](audio-data-blocks.md)
-   [Behandeln von Fehlern mit Audiofunktionen](handling-errors-with-audio-functions.md)

Zusätzlich zu den hier erörterten APIs kann die Media Control-Schnittstelle verwendet werden, um Wellenform-Audio und andere Medien, wie z. b. "MIDI Sounds" und "Video", wiederzugeben. Weitere Informationen finden Sie unter [MCI](mci.md).
