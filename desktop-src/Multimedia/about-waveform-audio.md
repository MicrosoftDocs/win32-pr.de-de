# <a name="about-waveform-audio"></a>Informationen zu Waveform-Audio

Es gibt mehrere Methoden zum Hinzufügen von Sound zu Ihrer Anwendung mithilfe von Waveformaudio. Die einfachste hier dokumentierte Methode ist die Verwendung der [**PlaySound-Funktion.**](/previous-versions//dd743680(v=vs.85)) Die meisten anderen Waveform-Audio-API-Elemente sind relativ niedrig.

Weitere Informationen finden Sie in den folgenden Themen:

-   [Einfache Audiowiedergabe](simple-audio-playback.md)
-   [Waveform-Audio-Schnittstelle](waveform-audio-interface.md)
-   [Zusätzliche Audioschnittstelle](auxiliary-audio-interface.md)
-   [Audiodatenblöcke](audio-data-blocks.md)
-   [Behandeln von Fehlern mit Audiofunktionen](handling-errors-with-audio-functions.md)

Zusätzlich zu den hier beschriebenen APIs kann die Mediensteuerungsschnittstelle verwendet werden, um Waveformaudio und andere Medien wie SOUNDS und Video wiederzuspielen. Weitere Informationen finden Sie unter [MCI](mci.md).
