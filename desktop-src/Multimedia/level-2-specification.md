# <a name="level-2-specification"></a>Spezifikation der Ebene 2

Die Spezifikation der Ebene 2 wurde entwickelt, um die Einführung erweiterter Multimedia-Funktionen zu fördern. Diese Spezifikation basiert auf den Anforderungen, die in der Spezifikation der Ebene 1 festgelegt sind, und ist eine supermenge davon. Die Spezifikation der Ebene 2 definiert die minimalen Systemfunktionen für Erweiterte Multimedia-Funktionen. Es handelt sich nicht um eine Empfehlung für eine Systemkonfiguration.
