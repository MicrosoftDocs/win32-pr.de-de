# <a name="level-2-specification"></a>Level 2-Spezifikation

Die Level 2-Spezifikation wurde entwickelt, um die Einführung erweiterter Multimediafunktionen zu fördern. Diese Spezifikation baut auf den In der Level 1-Spezifikation festgelegten Anforderungen auf und ist eine Obermenge davon. Die Level 2-Spezifikation definiert die Mindestsystemfunktionalität für erweiterte Multimediafunktionen. Dies ist keine Empfehlung für eine Systemkonfiguration.
