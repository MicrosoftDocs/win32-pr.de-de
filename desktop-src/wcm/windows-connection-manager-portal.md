# <a name="windows-connection-manager"></a>Windows-Verbindungs-Manager

## <a name="purpose"></a>Zweck

Windows Verbindungs-Manager (WCM) ermöglicht die Erstellung und Konfiguration von Verbindungs-Manager-Software.

## <a name="developer-audience"></a>Entwicklergruppe

Windows Verbindungs-Manager ist für die Verwendung durch Entwickler konzipiert, die C/C++-Entwicklungssoftware verwenden.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Die Windows Verbindungs-Manager-API wird unter Windows 8 und Windows Server 2012 unterstützt.
