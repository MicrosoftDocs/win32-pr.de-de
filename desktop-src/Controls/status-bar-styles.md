# <a name="status-bar-styles"></a>Statusleistenstile

In diesem Abschnitt werden die Stile aufgeführt, zusätzlich zu den Standardfensterstilen, die von *Statusleisten-Steuerelementen* unterstützt werden.



| Konstante                                                                                                                                                         | Beschreibung                                                                                                                                                                                                                           |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="SBARS_SIZEGRIP"></span><span id="sbars_sizegrip"></span><dl> <dt>**SBARS \_ SIZEGREIFBARP**</dt> </dl> | Das Statusleisten-Steuerelement enthält einen Größenregler am rechten Ende der Statusleiste. Ein Größen-Klammer ähnelt einem Größenrahmen. Es handelt sich um einen rechteckigen Bereich, auf den der Benutzer klicken und ziehen kann, um die Größe des übergeordneten Fensters zu ändern. <br/> |
| <span id="SBT_TOOLTIPS"></span><span id="sbt_tooltips"></span><dl> <dt>**\_SBT-QUICKINFOS**</dt> </dl>       | [Version 4.71.](common-control-versions.md) Verwenden Sie diesen Stil, um QuickInfos zu aktivieren.<br/>                                                                                                                                             |
| <span id="SBARS_TOOLTIPS"></span><span id="sbars_tooltips"></span><dl> <dt>**\_QUICKINFOS für SBARS**</dt> </dl> | [Version 5.80.](common-control-versions.md) Identisch mit SBT \_ TOOLTIPS. Verwenden Sie dieses Flag für Version 5.00 oder höher. <br/>                                                                                                         |
