# <a name="format-values"></a>Format Werte

In diesem Abschnitt werden die Werte aufgelistet, die mit dem *dwtextflags* -Parameter der Funktionen [**drawdermetext**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext) und [**getdermetextextent**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemetextextent) verwendet werden. Ein oder mehrere dieser Werte werden verwendet, um anzugeben, wie eine Text Zeichenfolge formatiert werden soll. Die **drawtemetext** -Funktion zeichnet die Text Zeichenfolge innerhalb eines Anzeige Rechtecks, das durch den *prect* -Parameter angegeben wird. Die **gettemetextextent** -Funktion kann verwendet werden, um das erforderliche umschließende Rechteck für Text mit einer bestimmten Länge und einem bestimmten Format zu bestimmen.
