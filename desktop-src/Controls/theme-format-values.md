# <a name="format-values"></a>Formatwerte

In diesem Abschnitt werden die Werte aufgelistet, die mit dem *dwTextFlags-Parameter* der [**Funktionen DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext) und [**GetThemeTextExtent**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemetextextent) verwendet werden. Mindestens einer dieser Werte wird verwendet, um anzugeben, wie eine Textzeichenfolge formatiert werden soll. Die **DrawThemeText-Funktion** zeichnet die Textzeichenfolge in einem Anzeigerechteck, das durch den *pRect-Parameter* angegeben wird. Die **GetThemeTextExtent-Funktion** kann verwendet werden, um das erforderliche umschließende Rechteck für Text mit einer bestimmten Länge und einem bestimmten Format zu bestimmen.

## <a name="format-values"></a>Formatwerte

Dies kann beispielsweise verwendet werden, um symbolbeschriftungen, die in koreanischem Text geschrieben sind, lesbarer zu machen. Zum Beispiel: <br/> Eingabezeichenfolge: "A&bc&&d"<br/> normal: "A<span class="underline">b</span>c&d<br/> DT \_ PREFIXONLY: <span class="underline"></span> " "<br/> |

Die Standardrichtung für Text ist von links nach rechts.                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| DT \_ SINGLELINE           | Zeigt Text in einer einzelnen Zeile an. | DT \_ TOP                  | Rendert den Text am oberen Rand des Anzeigerechtecks.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |

Dieser Wert wird nur mit dem DT \_ SINGLELINE-Wert verwendet.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| DT \_ WORDBREAK            | Unterbricht Zeilen zwischen Wörtern, wenn sich ein Wort über den Rand des Anzeigerechtecks erstreckt. | DT \_ WORD \_ ELLIPSIS       | Schneidet alle Wörter ab, die nicht in das Anzeigerechteck passen, und fügt eine Auslassungszeichen hinzu.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
