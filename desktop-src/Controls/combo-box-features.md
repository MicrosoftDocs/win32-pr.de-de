## <a name="special-features"></a>Besondere Features

Es gibt spezielle Nachrichten und Funktionen, mit denen eine Anwendung eine Verzeichnisliste in einem Kombinations Feld anzeigen, Daten mit Listenelementen in einem Kombinations Feld zuordnen und die Tastaturschnittstelle für ein Dropdown-Kombinations Feld oder ein Dropdown-Listenfeld ändern kann.

### <a name="directory-lists"></a>Verzeichnislisten

Eine Anwendung kann die Namen von Dateien oder Unterverzeichnissen einem Kombinations Feld hinzufügen, indem die [**CB- \_ dir**](cb-dir.md) -Nachricht an Sie gesendet wird. Der *wParam* -Parameter für diese Nachricht gibt die Attribute der hinzu zufügenden Dateien an, und der *LPARAM* -Parameter ist ein Zeiger auf die Text Zeichenfolge, die die Datei Spezifikation definiert.

Sie können die [**dlgdirlistcombobox**](/windows/desktop/api/Winuser/nf-winuser-dlgdirlistcomboboxa) -Funktion verwenden, um den Inhalt eines Kombinations Felds in einem Dialogfeld zu ersetzen. Die-Funktion füllt das Kombinations Feld mit den Namen von Laufwerken, Verzeichnissen und Dateien, die mit einem angegebenen Kriteriensatz übereinstimmen. Die [**dlgdirselectcomboboxex**](/windows/desktop/api/Winuser/nf-winuser-dlgdirselectcomboboxexa) -Funktion Ruft die aktuelle Auswahl in einem Kombinations Feld ab, das von **dlgdirlistcombobox** initialisiert wurde. Diese Funktionen ermöglichen es dem Benutzer, ein Laufwerk, ein Verzeichnis oder eine Datei aus einem Kombinations Feld auszuwählen, ohne den Speicherort und den Namen der Datei einzugeben.

Die Funktionen " [**dlgdirlistcombobox**](/windows/desktop/api/Winuser/nf-winuser-dlgdirlistcomboboxa) " und " [**dlgdirselectcomboboxex**](/windows/desktop/api/Winuser/nf-winuser-dlgdirselectcomboboxexa) " und die [**CB- \_ dir**](cb-dir.md) -Nachricht ähneln den Funktionen " [**dlgdirlist**](/windows/desktop/api/Winuser/nf-winuser-dlgdirlista) " und " [**DlgDirSelectEx**](/windows/desktop/api/Winuser/nf-winuser-dlgdirselectexa) " sowie der in Listenfeldern verwendeten [**lb- \_ dir**](lb-dir.md) -Nachricht.

### <a name="data-associated-with-list-items"></a>Mit Listenelementen verknüpfte Daten

Eine Anwendung kann den Listenelementen in einem Kombinations Feld Daten zuordnen. Die CB-Eigenschaft " [**\_ logdata**](cb-setitemdata.md) " ordnet einem Listenelement einen **DWORD** -Wert zu, und der [**CB- \_ GetItemData**](cb-getitemdata.md) Ruft den Wert ab, der einem Listenelement zugeordnet ist.

Im Beispiel zum [Erstellen eines](using-combo-boxes.md) Kombinations Felds für den Besitzer gezeichnet werden Elementdaten verwendet, um jedem Element in einem Dropdown-Listenfeld eine Konstante zuzuordnen. Bei einem solchen eindeutigen Wert werden die einzelnen Elemente unabhängig von der sortierten Position identifiziert.

Andere Anwendungen können Elementdaten verwenden, um ein Handle oder einen Zeiger einem Listenelement zuzuordnen. Wenn dies der Fall ist, kann eine Anwendung eine [**WM \_ DeleteItem**](wm-deleteitem.md) -Nachricht verarbeiten, um das angegebene Objekt zu löschen oder freizugeben, wenn das Listenelement gelöscht wird.

### <a name="the-extended-user-interface"></a>Die erweiterte Benutzeroberfläche

Dropdown-Kombinations Felder und Dropdown-Listenfelder unterstützen eine Alternative Tastaturschnittstelle, die als *Erweiterte Benutzeroberfläche* bezeichnet wird. Standardmäßig wird die Liste mit dem F4-Schlüssel geöffnet oder geschlossen, und die aktuelle Auswahl wird durch den Pfeil nach unten geändert. In einem Kombinations Feld mit der erweiterten Benutzeroberfläche ist die F4-Taste jedoch deaktiviert. durch Drücken der nach-unten-Taste wird die Dropdown Liste geöffnet. Außerdem verfügt das Mausrad, das normalerweise durch die Elemente in der Liste scrollt, nicht über eine Funktion, wenn die erweiterte Benutzeroberfläche festgelegt wird.

Um die Benutzeroberfläche für ein Kombinations Feld auszuwählen, kann eine Anwendung die [**CB \_ setextendedui**](cb-setextendedui.md) -Nachricht an das Kombinations Feld senden. Ein **true** -Wert für den *wParam* -Parameter aktiviert die erweiterte Benutzeroberfläche. mit einem **false** -Wert wird die Standardbenutzer Oberfläche festgelegt. Um zu ermitteln, ob ein Kombinations Feld die erweiterte Benutzeroberfläche verwendet, kann eine Anwendung die [**CB \_ getextendedui**](cb-getextendedui.md) -Nachricht an das Kombinations Feld senden.

### <a name="cue-banners"></a>Hinweis Banner

Hinweis Banner sind ein neues Feature der Bearbeitungs-und Kombinations Felder. Der Zweck eines Hinweis Banners ist das Bereitstellen eines Hinweises für den Benutzer als Zweck eines Bearbeitungs Steuer Elements oder eines Kombinations Felds. Der folgende Screenshot zeigt ein Bearbeitungs Steuerelement mit dem Hinweis Text "Search".

![Screenshot eines Bearbeitungs Steuer Elements mit dem Hinweis Text "Search"](images/cue-banner.png)

Der Text eines Hinweis Banners wird angezeigt, wenn ein Bearbeitungs Steuerelement keinen Text hat oder ein Kombinations Feld keine Auswahl hat. Wenn der Benutzer Text in das Bearbeitungs Steuerelement eingibt oder eine Auswahl in einem Kombinations Feld vornimmt, verschwindet das Hinweis Banner. Standardmäßig verschwindet das Hinweis Banner auch, wenn das Bearbeitungs Steuerelement oder Kombinations Feld den Fokus erhält.

| [**WM \_ lbuttondown**](/windows/desktop/inputdev/wm-lbuttondown)                | Legt den Fokus auf das Kombinations Feld fest, und für Dropdown-Kombinations Felder und Dropdown Listen kann die Liste geöffnet oder geschlossen werden. Wenn die Liste geöffnet wird, erfasst die Fenster Prozedur die Maus, um die Auswahl durchziehen und Loslassen der Maustaste zu aktivieren.                                                                                                                                        |

Das Kombinations Feld Fenster empfängt diese Meldung aus dem Listenfenster.                                                                                                                                                  |
| [**WM- \_ mouseelmove**](/windows/desktop/inputdev/wm-mousemove)                    | Sendet die Nachricht an das Listenfenster, wenn die Maus die Liste geöffnet hat und die Maustaste immer noch nicht angezeigt wird. | [**WM- \_ nccreate**](/windows/desktop/winmsg/wm-nccreate)                        | Weist eine interne Datenstruktur zu, die von der Kombinations Feld-Fenster Prozedur verwendet wird.                                                                                                                                                                                                                                                                                                         |
| [**WM \_ ncdestroy**](/windows/desktop/winmsg/wm-ncdestroy)                      | Gibt Ressourcen frei, die als Antwort auf die [**WM- \_ nccreate**](/windows/desktop/winmsg/wm-nccreate) -Meldung zugeordnet sind.                                                                                                                                                                                                                                                                                         |

Die Fenster Prozedur verwendet den angegebenen DC anstelle von [**BeginPaint**](/windows/desktop/api/winuser/nf-winuser-beginpaint) und [**endpaint**](/windows/desktop/api/winuser/nf-winuser-endpaint).                                                                                          |
