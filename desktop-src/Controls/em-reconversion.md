# <a name="em_reconversion-message"></a>\_Nachricht zum erneuten Konvertieren von EM

Ruft das Dialogfeld Neukonvertierung des Eingabemethoden-Editors (IME) auf.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Nicht verwendet; muss 0 (null) sein.

</dd> <dt>

*lParam* 
</dt> <dd>

Nicht verwendet; muss 0 (null) sein.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Diese Meldung gibt immer 0 (null) zurück.
