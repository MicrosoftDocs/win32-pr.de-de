# <a name="list-box-styles"></a>Listenfeldstile

Um ein Listenfeld mithilfe der [**CreateWindow-**](/windows/desktop/api/winuser/nf-winuser-createwindowa) oder [**CreateWindowEx-Funktion**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) zu erstellen, verwenden Sie die LISTBOX-Klasse, die entsprechenden Fensterstilkonstanten und die folgenden Stilkonstanten, um das Listenfeld zu definieren. Nachdem das Steuerelement erstellt wurde, können diese Stile nur wie angegeben geändert werden.



| Konstante                                                                                                                                                                              | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="LBS_COMBOBOX"></span><span id="lbs_combobox"></span><dl> <dt>**LBS \_ COMBOBOX**</dt> </dl>                            | Benachrichtigt ein Listenfeld, dass es Teil eines Kombinationsfelds ist. Dies ermöglicht die Koordination zwischen den beiden Steuerelementen, sodass sie eine einheitliche Benutzeroberfläche darstellen. Das Kombinationsfeld selbst muss diesen Stil festlegen. Wenn der Stil nur durch das Kombinationsfeld festgelegt wird, betrachtet sich das Listenfeld fälschlicherweise als untergeordnetes Element eines Kombinationsfelds, und es tritt ein Fehler auf.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_DISABLENOSCROLL"></span><span id="lbs_disablenoscroll"></span><dl> <dt>**LBS \_ DISABLENOSCROLL**</dt> </dl>       | Zeigt eine deaktivierte horizontale oder vertikale Bildlaufleiste an, wenn das Listenfeld nicht genügend Elemente zum Scrollen enthält. Wenn Sie diesen Stil nicht angeben, wird die Scrollleiste ausgeblendet, wenn das Listenfeld nicht genügend Elemente enthält. Dieser Stil muss mit dem [**\_ WS-VSCROLL-**](/windows/desktop/winmsg/window-styles) oder [**WS \_ HSCROLL-Stil**](/windows/desktop/winmsg/window-styles) verwendet werden.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_EXTENDEDSEL"></span><span id="lbs_extendedsel"></span><dl> <dt>**LBS \_ EXTENDEDSEL**</dt> </dl>                   | Ermöglicht die Auswahl mehrerer Elemente mithilfe der UMSCHALTTASTE und der Maus oder spezieller Tastenkombinationen.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_HASSTRINGS"></span><span id="lbs_hasstrings"></span><dl> <dt>**LBS \_ HASSTRINGS**</dt> </dl>                      | Gibt an, dass ein Listenfeld Elemente enthält, die aus Zeichenfolgen bestehen. Das Listenfeld verwaltet den Arbeitsspeicher und die Adressen für die Zeichenfolgen, sodass die Anwendung die [**LB \_ GETTEXT-Nachricht**](lb-gettext.md) verwenden kann, um den Text für ein bestimmtes Element abzurufen. Standardmäßig weisen alle Listenfelder mit Ausnahme von vom Besitzer gezeichneten Listenfeldern diesen Stil auf. Sie können ein vom Besitzer gezeichnetes Listenfeld entweder mit oder ohne diesen Stil erstellen.<br/> Für vom Besitzer gezeichnete Listenfelder ohne diesen Stil ruft die [**LB \_ GETTEXT-Nachricht**](lb-gettext.md) den Wert ab, der einem Element (den Elementdaten) zugeordnet ist. <br/>                                                                                                                                                                                                                                                                                                                                    |
| <span id="LBS_MULTICOLUMN"></span><span id="lbs_multicolumn"></span><dl> <dt>**LBS \_ MULTICOLUMN**</dt> </dl>                   | Gibt ein mehrspaltiges Listenfeld an, in dem horizontal gescrollt wird. Das Listenfeld berechnet automatisch die Breite der Spalten, oder eine Anwendung kann die Breite mithilfe der [**LB \_ SETCOLUMNWIDTH-Nachricht**](lb-setcolumnwidth.md) festlegen. Wenn ein Listenfeld den **LBS \_ OWNERDRAWFIXED-Stil** aufweist, kann eine Anwendung die Breite festlegen, wenn das Listenfeld die [**WM \_ MEASUREITEM-Nachricht**](wm-measureitem.md) sendet. <br/> Ein Listenfeld mit dem **LBS \_ MULTICOLUMN-Stil** kann nicht vertikal scrollen und ignoriert alle [**empfangenen \_ WM-VSCROLL-Nachrichten.**](wm-vscroll.md) <br/> Die **LBS \_ MULTICOLUMN-** und **LBS \_ OWNERDRAWVARIABLE-Stile** können nicht kombiniert werden. Wenn beide angegeben sind, wird **LBS \_ OWNERDRAWVARIABLE** ignoriert.<br/>                                                                                                                                                                                |
| <span id="LBS_MULTIPLESEL"></span><span id="lbs_multiplesel"></span><dl> <dt>**LBS \_ MULTIPLESEL**</dt> </dl>                   | Aktiviert oder deaktiviert die Zeichenfolgenauswahl jedes Mal, wenn der Benutzer auf eine Zeichenfolge im Listenfeld klickt oder doppelklickt. Der Benutzer kann eine beliebige Anzahl von Zeichenfolgen auswählen.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NODATA"></span><span id="lbs_nodata"></span><dl> <dt>**LBS \_ NODATA**</dt> </dl>                                  | Gibt ein Listenfeld ohne Daten an. Geben Sie diesen Stil an, wenn die Anzahl der Elemente im Listenfeld 1.000 überschreitet. Ein Listenfeld ohne Daten muss auch den **LBS \_ OWNERDRAWFIXED-Stil** aufweisen, aber nicht den **LBS \_ SORT-** oder **LBS \_ HASSTRINGS-Stil.** <br/> Ein Listenfeld ohne Daten ähnelt einem vom Besitzer gezeichneten Listenfeld, außer dass es keine Zeichenfolgen- oder Bitmapdaten für ein Element enthält. Befehle zum Hinzufügen, Einfügen oder Löschen eines Elements ignorieren immer alle angegebenen Elementdaten. -Anforderungen, um eine Zeichenfolge im Listenfeld zu finden, schlagen immer fehl. Das System sendet die [**WM \_ DRAWITEM-Nachricht**](wm-drawitem.md) an das Besitzerfenster, wenn ein Element gezeichnet werden muss. Der **itemID-Member** der [**DRAWITEMSTRUCT-Struktur,**](/windows/win32/api/winuser/ns-winuser-drawitemstruct) der mit der **WM \_ DRAWITEM-Meldung** übergeben wird, gibt die Zeilennummer des zu zeichnenden Elements an. Ein Listenfeld ohne Daten sendet keine [**WM \_ DELETEITEM-Nachricht.**](wm-deleteitem.md)<br/> |
| <span id="LBS_NOINTEGRALHEIGHT"></span><span id="lbs_nointegralheight"></span><dl> <dt>**LBS \_ NOINTEGRALHEIGHT**</dt> </dl>    | Gibt an, dass die Größe des Listenfelds genau der Größe entspricht, die von der Anwendung beim Erstellen des Listenfelds angegeben wurde. Normalerweise wird ein Listenfeld vom System so groß, dass im Listenfeld keine Teilelemente angezeigt werden.<br/> Für Listenfelder mit dem **LBS \_ OWNERDRAWVARIABLE-Stil** wird der **LBS \_ NOINTEGRALHEIGHT-Stil** immer erzwungen.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="LBS_NOREDRAW"></span><span id="lbs_noredraw"></span><dl> <dt>**LBS \_ NOREDRAW**</dt> </dl>                            | Gibt an, dass die Darstellung des Listenfelds nicht aktualisiert wird, wenn Änderungen vorgenommen werden.<br/> Um den neu gezeichneten Zustand des Steuerelements zu ändern, verwenden Sie die [**WM \_ SETREDRAW-Meldung.**](/windows/desktop/gdi/wm-setredraw)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="LBS_NOSEL"></span><span id="lbs_nosel"></span><dl> <dt>**LBS \_ NOSEL**</dt> </dl>                                     | Gibt an, dass das Listenfeld Elemente enthält, die angezeigt, aber nicht ausgewählt werden können. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NOTIFY"></span><span id="lbs_notify"></span><dl> <dt>**LBS \_ NOTIFY**</dt> </dl>                                  | Bewirkt, dass das Listenfeld einen Benachrichtigungscode an das übergeordnete Fenster sendet, wenn der Benutzer auf ein Listenfeldelement klickt ([LBN \_ SELCHANGE](lbn-selchange.md)), auf ein Element doppelklickt ([LBN \_ DBLCLK](lbn-dblclk.md)) oder die Auswahl abbricht ([LBN \_ SELCANCEL](lbn-selcancel.md)). <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="LBS_OWNERDRAWFIXED"></span><span id="lbs_ownerdrawfixed"></span><dl> <dt>**LBS \_ OWNERDRAWFIXED**</dt> </dl>          | Gibt an, dass der Besitzer des Listenfelds für das Zeichnen seines Inhalts verantwortlich ist und dass die Elemente im Listenfeld die gleiche Höhe haben. Das Besitzerfenster empfängt eine [**WM \_ MEASUREITEM-Nachricht,**](wm-measureitem.md) wenn das Listenfeld erstellt wird, und eine [**WM \_ DRAWITEM-Nachricht,**](wm-drawitem.md) wenn sich ein visueller Aspekt des Listenfelds geändert hat.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="LBS_OWNERDRAWVARIABLE"></span><span id="lbs_ownerdrawvariable"></span><dl> <dt>**LBS \_ OWNERDRAWVARIABLE**</dt> </dl> | Gibt an, dass der Besitzer des Listenfelds für das Zeichnen seines Inhalts verantwortlich ist und dass die Elemente im Listenfeld in der Höhe variabel sind. Das Besitzerfenster empfängt eine [**WM \_ MEASUREITEM-Nachricht**](wm-measureitem.md) für jedes Element im Feld, wenn das Listenfeld erstellt wird, und eine [**WM \_ DRAWITEM-Meldung,**](wm-drawitem.md) wenn sich ein visueller Aspekt des Listenfelds geändert hat.<br/> Dieser Stil bewirkt, dass der **LBS \_ NOINTEGRALHEIGHT-Stil** aktiviert wird.<br/> Dieser Stil wird ignoriert, wenn der **LBS \_ MULTICOLUMN-Stil** angegeben wird. <br/>                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="LBS_SORT"></span><span id="lbs_sort"></span><dl> <dt>**LBS \_ SORT**</dt> </dl>                                        | Sortiert Zeichenfolgen im Listenfeld alphabetisch.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="LBS_STANDARD"></span><span id="lbs_standard"></span><dl> <dt>**LBS \_ STANDARD**</dt> </dl>                            | Sortiert Zeichenfolgen im Listenfeld alphabetisch. Das übergeordnete Fenster empfängt einen Benachrichtigungscode, wenn der Benutzer auf ein Listenfeldelement klickt, auf ein Element doppelklickt oder die Auswahl abbricht. Das Listenfeld verfügt über eine vertikale Bildlaufleiste und hat Rahmen auf allen Seiten. Dieser Stil kombiniert die Stile **LBS \_ NOTIFY,** **LBS \_ SORT,** [**WS \_ VSCROLL**](/windows/desktop/winmsg/window-styles)und [**WS \_ BORDER.**](/windows/desktop/winmsg/window-styles)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="LBS_USETABSTOPS"></span><span id="lbs_usetabstops"></span><dl> <dt>**LBS \_ USETABSTOPS**</dt> </dl>                   | Aktiviert ein Listenfeld, um Tabstoppzeichen beim Zeichnen seiner Zeichenfolgen zu erkennen und zu erweitern. Sie können die [**MELDUNG LB \_ SETTABSTOPS**](lb-settabstops.md) verwenden, um Tabstopppositionen anzugeben. Die Standardpositionen der Registerkarten sind 32 Vorlageneinheiten des Dialogfelds voneinander getrennt. Dialogvorlageneinheiten sind die geräteunabhängigen Einheiten, die in Dialogfeldvorlagen verwendet werden. Verwenden Sie die [**MapDialogRect-Funktion,**](/windows/desktop/api/winuser/nf-winuser-mapdialogrect) um Messungen von Dialogvorlageneinheiten in Bildschirmeinheiten (Pixel) zu konvertieren.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_WANTKEYBOARDINPUT"></span><span id="lbs_wantkeyboardinput"></span><dl> <dt>**LBS \_ WANTKEYBOARDINPUT**</dt> </dl> | Gibt an, dass der Besitzer des Listenfelds [**\_ WM-VKEYTOITEM-Meldungen**](wm-vkeytoitem.md) empfängt, wenn der Benutzer eine Taste drückt und das Listenfeld den Eingabefokus besitzt. Dadurch kann eine Anwendung eine spezielle Verarbeitung der Tastatureingabe durchführen.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
