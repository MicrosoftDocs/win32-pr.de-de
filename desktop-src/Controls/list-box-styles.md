# <a name="list-box-styles"></a>Listenfeld Stile

Um ein Listenfeld mit der Funktion "up [**Window**](/windows/desktop/api/winuser/nf-winuser-createwindowa) " oder " [**deatewindowex**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) " zu erstellen, verwenden Sie die ListBox-Klasse, die entsprechenden Fenster Stil Konstanten und die folgenden Stil Konstanten, um das Listenfeld zu definieren. Nachdem das Steuerelement erstellt wurde, können diese Stile nicht geändert werden, außer wie bereits erwähnt.



| Konstante                                                                                                                                                                              | BESCHREIBUNG                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="LBS_COMBOBOX"></span><span id="lbs_combobox"></span><dl> <dt>**LBS- \_ ComboBox**</dt> </dl>                            | Benachrichtigt ein Listenfeld, dass es Teil eines Kombinations Felds ist. Dies ermöglicht die Koordination zwischen den beiden Steuerelementen, sodass Sie eine einheitliche Benutzeroberfläche darstellen. Das Kombinations Feld selbst muss diesen Stil festlegen. Wenn der Stil von einem beliebigen Element, aber vom Kombinations Feld festgelegt wird, wird das Listenfeld als untergeordnetes Element eines Kombinations Felds falsch betrachtet, und ein Fehler tritt auf.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_DISABLENOSCROLL"></span><span id="lbs_disablenoscroll"></span><dl> <dt>**LBS- \_ DisableNoScroll**</dt> </dl>       | Zeigt eine deaktivierte horizontale oder vertikale Schiebe Leiste an, wenn das Listenfeld nicht genügend Elemente enthält, die durchlaufen werden können. Wenn Sie diesen Stil nicht angeben, wird die Scrollleiste ausgeblendet, wenn das Listenfeld nicht genügend Elemente enthält. Dieser Stil muss mit dem WS- [**\_ VScroll**](/windows/desktop/winmsg/window-styles) -oder dem [**WS- \_ HScroll**](/windows/desktop/winmsg/window-styles) -Stil verwendet werden.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_EXTENDEDSEL"></span><span id="lbs_extendedsel"></span><dl> <dt>**LBS \_ extendedsel**</dt> </dl>                   | Ermöglicht die Auswahl mehrerer Elemente mithilfe der Umschalttaste und der Tastenkombinationen für die Maus oder die Tastenkombination.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_HASSTRINGS"></span><span id="lbs_hasstrings"></span><dl> <dt>**LBS \_ hasstrings**</dt> </dl>                      | Gibt an, dass ein Listenfeld Elemente enthält, die aus Zeichen folgen bestehen. Das Listenfeld verwaltet den Arbeitsspeicher und die Adressen für die Zeichen folgen, damit die Anwendung die [**lb- \_ gettext**](lb-gettext.md) -Nachricht verwenden kann, um den Text für ein bestimmtes Element abzurufen. Standardmäßig haben alle Listenfelder außer den von Besitzern gezeichneten Listenfeldern diesen Stil. Sie können ein vom Besitzer gezeichnetes Listenfeld mit oder ohne diesen Stil erstellen.<br/> Bei vom Besitzer gezeichneten Listenfeldern ohne diesen Stil Ruft die [**lb- \_ gettext**](lb-gettext.md) -Nachricht den Wert ab, der einem Element (den Elementdaten) zugeordnet ist. <br/>                                                                                                                                                                                                                                                                                                                                    |
| <span id="LBS_MULTICOLUMN"></span><span id="lbs_multicolumn"></span><dl> <dt>**LBS \_ mehrspaltige Spalten**</dt> </dl>                   | Gibt ein mehrspaltige Listenfeld an, das horizontal scrollt. Das Listenfeld berechnet automatisch die Breite der Spalten, oder eine Anwendung kann die Breite mithilfe der [**lb- \_ setcolumnwidth**](lb-setcolumnwidth.md) -Nachricht festlegen. Wenn ein Listenfeld den **\_ lbs** -Besitzer hat, kann eine Anwendung die Breite festlegen, wenn das Listenfeld die WM- [**\_ MeasureItem**](wm-measureitem.md) -Nachricht sendet. <br/> Ein Listenfeld mit dem **lbs- \_ MultiColumn** -Stil kann keinen vertikalen Bildlauf durchführen. es ignoriert alle empfangenen [**WM- \_ vscrollnachrichten**](wm-vscroll.md) . <br/> Die **lbs-Stile \_ MultiColumn** und **\_ lbs** -Besitzer können nicht kombiniert werden. Wenn beide angegeben werden, wird die lbs-Besitzer **\_ drawvariable** ignoriert.<br/>                                                                                                                                                                                |
| <span id="LBS_MULTIPLESEL"></span><span id="lbs_multiplesel"></span><dl> <dt>**LBS \_ multiplesel**</dt> </dl>                   | Schaltet die Zeichen folgen Auswahl ein oder aus, wenn der Benutzer auf eine Zeichenfolge im Listenfeld klickt oder doppelklickt. Der Benutzer kann eine beliebige Anzahl von Zeichen folgen auswählen.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NODATA"></span><span id="lbs_nodata"></span><dl> <dt>**LBS \_ NODATA**</dt> </dl>                                  | Gibt ein Listenfeld ohne Daten an. Geben Sie diesen Stil an, wenn die Anzahl der Elemente im Listenfeld 1000 überschreitet. Das Listenfeld "No-Data" muss auch **den \_ lbs** -Typ "-Besitzer" aufweisen, darf jedoch nicht den Typ " **lbs \_ Sort** " oder " **lbs \_ hasstrings** " aufweisen. <br/> Ein Listenfeld ohne Daten ähnelt einem von einem Besitzer gezeichneten Listenfeld, mit dem Unterschied, dass es keine Zeichen folgen-oder Bitmapdaten für ein Element enthält. Durch Befehle zum Hinzufügen, einfügen oder Löschen eines Elements werden alle angegebenen Elementdaten immer ignoriert. bei Anforderungen zum Suchen nach einer Zeichenfolge innerhalb des Listen Felds tritt immer ein Fehler auf. Das System sendet die [**WM- \_ DrawItem**](wm-drawitem.md) -Nachricht an das Besitzer Fenster, wenn ein Element gezeichnet werden muss. Der **ItemID** -Member der [**drawitemstruct**](/windows/win32/api/winuser/ns-winuser-drawitemstruct) -Struktur, die mit der **WM \_ DrawItem** -Meldung übergeben wird, gibt die Zeilennummer des Elements an, das gezeichnet werden soll. Ein Listenfeld ohne Daten sendet keine [**WM- \_ DeleteItem**](wm-deleteitem.md) -Nachricht.<br/> |
| <span id="LBS_NOINTEGRALHEIGHT"></span><span id="lbs_nointegralheight"></span><dl> <dt>**LBS \_ nointegralheight**</dt> </dl>    | Gibt an, dass die Größe des Listen Felds genau der Größe entspricht, die von der Anwendung beim Erstellen des Listen Felds angegeben wurde. Normalerweise nimmt das System ein Listenfeld an, sodass keine partiellen Elemente im Listenfeld angezeigt werden.<br/> Für Listenfelder mit dem **lbs- \_ besitzerwert** -Stil wird der **lbs \_ nointegralheight** -Stil immer erzwungen.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="LBS_NOREDRAW"></span><span id="lbs_noredraw"></span><dl> <dt>**LBS \_ noredraw**</dt> </dl>                            | Gibt an, dass die Darstellung des Listen Felds nicht aktualisiert wird, wenn Änderungen vorgenommen werden.<br/> Verwenden Sie die WM-Nachricht, um den neu zeichnen- [**Zustand \_**](/windows/desktop/gdi/wm-setredraw) des Steuer Elements zu ändern.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="LBS_NOSEL"></span><span id="lbs_nosel"></span><dl> <dt>**LBS \_ nosel**</dt> </dl>                                     | Gibt an, dass das Listenfeld Elemente enthält, die angezeigt, aber nicht ausgewählt werden können. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NOTIFY"></span><span id="lbs_notify"></span><dl> <dt>**LBS- \_ Benachrichtigung**</dt> </dl>                                  | Bewirkt, dass das Listenfeld einen Benachrichtigungs Code an das übergeordnete Fenster sendet, wenn der Benutzer auf ein Listenfeld Element klickt ([LBN \_ selChange](lbn-selchange.md)), auf ein Element ([LBN \_ dblclk](lbn-dblclk.md)) doppelklickt oder die Auswahl ([LBN \_ selcancel](lbn-selcancel.md)) abbricht. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="LBS_OWNERDRAWFIXED"></span><span id="lbs_ownerdrawfixed"></span><dl> <dt>**LBS \_ -Besitzer**</dt> </dl>          | Gibt an, dass der Besitzer des Listen Felds dafür zuständig ist, seinen Inhalt zu zeichnen, und dass die Elemente im Listenfeld die gleiche Höhe haben. Das Besitzer Fenster empfängt eine [**WM \_ MeasureItem**](wm-measureitem.md) -Nachricht, wenn das Listenfeld erstellt wird, und eine [**WM \_ DrawItem**](wm-drawitem.md) -Meldung, wenn sich ein visueller Aspekt des Listen Felds geändert hat.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="LBS_OWNERDRAWVARIABLE"></span><span id="lbs_ownerdrawvariable"></span><dl> <dt>**LBS-Besitzer \_ drawvariable**</dt> </dl> | Gibt an, dass der Besitzer des Listen Felds dafür zuständig ist, seinen Inhalt zu zeichnen, und dass die Elemente im Listenfeld in Höhe von variabler Größe sind. Das Besitzer Fenster empfängt eine [**WM \_ MeasureItem**](wm-measureitem.md) -Meldung für jedes Element im Feld, wenn das Listenfeld erstellt wird, und eine [**WM \_ DrawItem**](wm-drawitem.md) -Meldung, wenn sich ein visueller Aspekt des Listen Felds geändert hat.<br/> Dieser Stil bewirkt, dass der **lbs \_ nointegralheight** -Stil aktiviert wird.<br/> Dieser Stil wird ignoriert, wenn das **lbs- \_ MultiColumn** -Format angegeben wird. <br/>                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="LBS_SORT"></span><span id="lbs_sort"></span><dl> <dt>**LBS- \_ Sortierung**</dt> </dl>                                        | Sortiert Zeichen folgen im Listenfeld alphabetisch.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="LBS_STANDARD"></span><span id="lbs_standard"></span><dl> <dt>**LBS- \_ Standard**</dt> </dl>                            | Sortiert Zeichen folgen im Listenfeld alphabetisch. Das übergeordnete Fenster empfängt einen Benachrichtigungs Code, wenn der Benutzer auf ein Listenfeld Element klickt, auf ein Element doppelklickt oder die Auswahl abbricht. Das Listenfeld enthält eine vertikale Schiebe Leiste, die Rahmen auf allen Seiten enthält. In diesem Stil werden die Rahmen Stile **lbs \_ Notify**, **lbs \_ Sort**, [**WS \_ VScroll**](/windows/desktop/winmsg/window-styles)und [**WS \_ Border**](/windows/desktop/winmsg/window-styles) kombiniert.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="LBS_USETABSTOPS"></span><span id="lbs_usetabstops"></span><dl> <dt>**LBS \_ usetabstopps**</dt> </dl>                   | Ermöglicht einem Listenfeld, beim Zeichnen der Zeichen folgen Tabstopps zu erkennen und zu erweitern. Sie können die " [**lb \_ settabstopps**](lb-settabstops.md) "-Meldung verwenden, um Tabstopp Positionen anzugeben. Die Standard Registerkarten Positionen sind 32-Dialogfeld Vorlagen Einheiten voneinander entfernt. Dialogfeld Vorlagen Einheiten sind die geräteunabhängigen Einheiten, die in Dialogfeld Vorlagen verwendet werden. Verwenden Sie die [**mapdialogrect**](/windows/desktop/api/winuser/nf-winuser-mapdialogrect) -Funktion, um Messungen von Dialogfeld Vorlagen-Einheiten in Bildschirm Einheiten (Pixel) zu konvertieren.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_WANTKEYBOARDINPUT"></span><span id="lbs_wantkeyboardinput"></span><dl> <dt>**LBS \_ wantkeyboardinput**</dt> </dl> | Gibt an, dass der Besitzer des Listen Felds [**WM- \_ vkeytoitem**](wm-vkeytoitem.md) -Meldungen empfängt, wenn der Benutzer eine Taste drückt und das Listenfeld den Eingabefokus besitzt. Dies ermöglicht es einer Anwendung, eine spezielle Verarbeitung der Tastatureingaben auszuführen.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
