# <a name="lvm_gethotitem-message"></a>LVM- \_ gethotitem-Nachricht

Ruft den Index des aktiven Elements ab. Sie können diese Nachricht explizit senden oder das [**ListView \_ gethotitem**](/windows/desktop/api/Commctrl/nf-commctrl-listview_gethotitem) -Makro verwenden.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt den Index des aktiven Elements zurück.
