# <a name="bm_setdontclick-message"></a>BM- \_ setdontclick-Meldung

Legt ein Flag für ein Optionsfeld fest, das die Generierung [von \_ Milliarden](bn-clicked.md) -Nachrichten steuert, wenn die Schaltfläche den Fokus erhält.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* \[ in\]
</dt> <dd>

Ein **boolescher** Wert, der den Zustand angibt. **True** , wenn das Flag festgelegt werden soll, andernfalls **false**.

</dd> <dt>

*lParam* 
</dt> <dd>

Nicht verwendet. Muss Null sein.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Kein Rückgabewert.
