# <a name="tb_setstyle-message"></a>TB- \_ SetStyle-Nachricht

Legt den Stil für ein Symbolleisten-Steuerelement fest.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>

-Wert, der die für das Steuerelement festzulegenden Stile angibt. Dieser Wert kann eine Kombination von [Symbol](toolbar-control-and-button-styles.md)leisten-Steuerelement Stilen sein.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Kein Rückgabewert.
