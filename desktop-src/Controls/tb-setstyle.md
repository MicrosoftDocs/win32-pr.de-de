# <a name="tb_setstyle-message"></a>TB \_ SETSTYLE-Meldung

Legt den Stil für ein Symbolleisten-Steuerelement fest.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>

Wert, der die Stile an gibt, die für das Steuerelement festgelegt werden sollen. Dieser Wert kann eine Kombination aus [Symbolleisten-Steuerelementstilen sein.](toolbar-control-and-button-styles.md)

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Kein Rückgabewert.
