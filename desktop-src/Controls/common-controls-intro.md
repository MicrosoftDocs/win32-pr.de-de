# <a name="about-common-controls"></a>Informationen zu allgemeinen Steuerelementen

In diesem Thema werden die allgemeinen Steuerelemente erläutert, eine Reihe von Fenstern, die von der allgemeinen Steuerelementbibliothek implementiert werden, Comctl32.dll. Dabei handelt es sich um eine DLL, die im Windows Betriebssystem enthalten ist. Wie bei anderen Steuerelementfenstern ist ein allgemeines Steuerelement ein untergeordnetes Fenster, das eine Anwendung in Verbindung mit einem anderen Fenster verwendet, um die Interaktion mit dem Benutzer zu ermöglichen.
