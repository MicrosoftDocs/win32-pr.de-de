# <a name="about-common-controls"></a>Informationen zu allgemeinen Steuerelementen

In diesem Thema werden die allgemeinen Steuerelemente, eine Reihe von Fenstern, die von der allgemeinen Steuerelement Bibliothek implementiert werden, beschrieben, Comctl32.dll. dabei handelt es sich um eine im Windows-Betriebssystem enthaltene dll. Wie bei anderen Steuerungs Fenstern ist ein gängiges Steuerelement ein untergeordnetes Fenster, das von einer Anwendung in Verbindung mit einem anderen Fenster verwendet wird, um die Interaktion mit dem Benutzer zu ermöglichen.
