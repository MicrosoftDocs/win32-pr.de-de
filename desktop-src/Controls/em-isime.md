# <a name="em_isime-message"></a>EM- \_ isime-Nachricht

Bestimmen, ob das aktuelle Eingabe Gebiets Schema für das Bearbeitungs Steuerelement ein ostasiatische Gebiets Schema ist.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Nicht verwendet; muss 0 (null) sein.

</dd> <dt>

*lParam* 
</dt> <dd>

Nicht verwendet; muss 0 (null) sein.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt **true** zurück, wenn es sich um ein ostasiatische Gebiets Schema handelt. Andernfalls wird **false** zurückgegeben.
