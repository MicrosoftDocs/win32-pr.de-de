# <a name="tvm_getindent-message"></a>TVM \_ GETINDENT-Nachricht

Ruft den Betrag in Pixel ab, in dem untergeordnete Elemente relativ zu ihren übergeordneten Elementen eingerückt werden. Sie können diese Nachricht explizit oder mithilfe des [**TreeView \_ GetIndent-Makros**](/windows/desktop/api/Commctrl/nf-commctrl-treeview_getindent) senden.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt den Einzugsbetrag zurück.
