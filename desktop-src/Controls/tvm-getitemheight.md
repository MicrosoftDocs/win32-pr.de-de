# <a name="tvm_getitemheight-message"></a>TVM \_ GetItemHeight-Meldung

Ruft die aktuelle Höhe der einzelnen Struktur Ansichts Elemente ab. Sie können diese Nachricht explizit oder mithilfe des [**TreeView \_ GetItemHeight**](/windows/desktop/api/Commctrl/nf-commctrl-treeview_getitemheight) -Makros senden.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt die Höhe der einzelnen Elemente in Pixel zurück.
