# <a name="static-control-styles"></a>Statische Steuerelementstile

Um ein statisches Steuerelement mithilfe der [**CreateWindow-**](/windows/desktop/api/winuser/nf-winuser-createwindowa) oder [**CreateWindowEx-Funktion**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) zu erstellen, geben Sie die STATIC-Klasse, die entsprechenden Fensterformatkonst constants und eine Kombination der folgenden statischen Steuerelementstile an.

|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="SS_BITMAP"></span><span id="ss_bitmap"></span><dl> <dt>**\_SS-BITMAP**</dt> </dl>                            | Eine Bitmap soll im statischen Steuerelement angezeigt werden. Der Text ist der Name einer Bitmap (kein Dateiname), die an anderer Stelle in der Ressourcendatei definiert ist. Der Stil ignoriert die *Parameter nWidth* und *nHeight.* Das -Steuerelement passt automatisch die Größe selbst an, um die Bitmap aufnehmen zu können.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| <span id="SS_BLACKFRAME"></span><span id="ss_blackframe"></span><dl> <dt>**SS \_ BLACKFRAME**</dt> </dl>                | Ein Feld mit einem Rahmen, der in der gleichen Farbe wie die Fensterrahmen gezeichnet wird. Diese Farbe ist schwarz im Standardfarbschema.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |

| <span id="SS_CENTER"></span><span id="ss_center"></span><dl> <dt>**SS \_ CENTER**</dt> </dl>                            | Ein einfaches Rechteck und centert den Text im Rechteck. Der Text wird formatiert, bevor er angezeigt wird. Wörter, die sich über das Ende einer Zeile erstrecken, werden automatisch bis zum Anfang der nächsten zentrierten Zeile umschlossen. Insbesondere wird die durchschnittliche Zeichenbreite auf die gleiche Weise wie bei einem Bearbeitungssteuerzeichen berechnet, und die Funktion zeigt keine teilweise sichtbare letzte Zeile an.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |

Wenn ein Wort, das sich nicht am Ende der Zeichenfolge befindet, die Grenzen des Rechtecks überschreitet, wird es ohne Ellipsen abgeschnitten. Wenn Sie diesen Stil verwenden, wird der Text des Steuerelements in einer Zeile ohne Zeilenumbruch erzwingen. Vergleichen Sie **mit SS \_ PATHELLIPSIS** und **SS \_ WORDELLIPSIS**.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |

| <span id="SS_ETCHEDVERT"></span><span id="ss_etchedvert"></span><dl> <dt>**SS \_ ETCHEDVERT**</dt> </dl>                | Zeichnet den linken und rechten Rand des statischen Steuerelements mithilfe des **\_ EDGE-ETCHED-Kantenstils.** Weitere Informationen finden Sie unter der [**DrawEdge-Funktion.**](/windows/desktop/api/winuser/nf-winuser-drawedge)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |

Da die [**LoadIcon-Funktion verwendet**](/windows/desktop/api/winuser/nf-winuser-loadicona) wird, kann der **SS \_ ICON-Stil** nur Symbole der Dimensionen **SM \_ CXICON** und **SM \_ CYICON laden.** Diese Einschränkung kann umgangen werden, indem zusätzlich zu SS ICON der **SS \_ REALSIZEIMAGE-Stil** **verwendet \_ wird.** <br/> Wenn ein Symbol nicht über [**LoadIcon**](/windows/desktop/api/winuser/nf-winuser-loadicona)geladen werden kann, wird versucht, die angegebene Ressource mit [**loadCursor als Cursor zu laden.**](/windows/desktop/api/winuser/nf-winuser-loadcursora) Wenn auch dies fehlschlägt, wird versucht, mit LoadImage aus dem [**Gerätetreiber zu laden.**](/windows/desktop/api/winuser/nf-winuser-loadimagea) <br/> |

## <a name="remarks"></a>Hinweise

Um ein Format zu verwenden, das Wörter oder Zeichenfolgen mit Auslassungszeichen für mehrzeiligen Text abschneiden, können [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext) oder [**DrawText**](/windows/desktop/api/winuser/nf-winuser-drawtext) auf einem Owner-Draw-Steuerelement mit den Formatierungsflags **DT \_ WORDBREAK** und **DT \_ ENDELLIPSIS** verwendet werden.
