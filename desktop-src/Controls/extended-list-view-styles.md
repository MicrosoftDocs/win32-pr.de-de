## <a name="remarks"></a>Bemerkungen

> [!Note]  
> Comctl32.dll Version 6 ist nicht Verteil Bar, aber Sie ist in Windows enthalten. Wenn Sie Comctl32.dll Version 6 verwenden möchten, geben Sie ihn in einem Manifest an. Weitere Informationen zu Manifesten finden Sie unter [Aktivieren von visuellen Stilen](cookbook-overview.md).

 

Ein Element wird *heiß* , wenn der Mauszeiger darüber bewegt wird. Wenn Hot-Tracking aktiviert ist, werden heiße Elemente hervorgehoben, der Benutzer muss jedoch noch mindestens einmal auf das Element klicken, um es auszuwählen. Das *aktivieren* eines Elements bewirkt, dass das Listenansicht-Steuerelement eine [LVN- \_ itemaktivierungs](lvn-itemactivate.md) -Benachrichtigung sendet. Ein Element kann ausgewählt werden, wenn es sich in einem Zustand befindet, in dem er von einem einzelnen Mausklick *ausgewählt* wird.

Wenn der **LVS-Typ " \_ \_ oneclickaktivierungs** " festgelegt ist, ist nur ein Klick erforderlich, um ein beliebiges Element auszuwählen. Daher können alle Elemente ausgewählt werden. Wenn das Format **LVS \_ Ex \_ twoclickaktivierungs** festgelegt ist, sind zwei Klicks erforderlich, um ein Element auszuwählen. Bei diesem Stil kann das Element erst ausgewählt werden, nachdem es einmal geklickt wurde.

Wenn Sie den LVS-Wert für das **\_ \_ unterlinehot** festlegen, wird der Text aller aktiven Elemente, die möglicherweise ausgewählt werden, unterstrichen. Das Festlegen des **LVS \_ Ex- \_ unterlinecold** -Stils führt dazu, dass der Text von Elementen, die nicht heiß sind, aber möglicherweise ausgewählt werden, unterstrichen ist.
