## <a name="remarks"></a>Hinweise

> [!Note]  
> Comctl32.dll Version 6 ist nicht verteilbar, aber in der Windows. Um Version Comctl32.dll 6 zu verwenden, geben Sie sie in einem Manifest an. Weitere Informationen zu Manifesten finden Sie unter [Aktivieren von visuellen Stilen.](cookbook-overview.md)

 

Ein Element wird *heiß,* wenn der Mauszeiger darüber zeigt. Wenn hot-tracking aktiviert ist, werden hot-Elemente hervorgehoben, aber der Benutzer muss immer noch mindestens einmal auf das Element klicken, um es auszuwählen. *Das* Aktivieren eines Elements bewirkt, dass das Listenansichtssteuerelement eine [LVN \_ ITEMACTIVATE-Benachrichtigung](lvn-itemactivate.md) sendet. Ein Element kann ausgewählt *werden,* wenn es sich in einem Zustand befindet, in dem ein einzelner Klick es auswählt.

Wenn der **LVS \_ EX \_ ONECLICKACTIVATE-Stil** festgelegt ist, ist nur ein Klick erforderlich, um ein Element auszuwählen, sodass alle Elemente ausgewählt werden können. Wenn der **LVS \_ EX \_ TWOCLICKACTIVATE-Stil** festgelegt ist, sind zwei Klicks erforderlich, um ein Element auszuwählen. Bei diesem Stil kann das Element erst ausgewählt werden, nachdem es einmal angeklickt wurde.

Das Festlegen **des LVS \_ EX \_ UNDERLINEHOT-Stils** verursacht, dass der Text eines beliebigen hot-Elements, das ausgewählt werden kann, unterstrichen wird. Das Festlegen **des LVS \_ EX \_ UNDERLINECOLD-Stils** verursacht, dass der Text von Elementen, die nicht heiß sind, aber möglicherweise ausgewählt sind, unterstrichen wird.
