# <a name="lvm_gettextcolor-message"></a>LVM \_ GETTEXTCOLOR-Nachricht

Ruft die Textfarbe eines Listenansicht-Steuerelements ab. Sie können diese Nachricht explizit oder mithilfe des [**ListView \_ GetTextColor-Makros**](/windows/desktop/api/Commctrl/nf-commctrl-listview_gettextcolor) senden.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt die Textfarbe zurück.
