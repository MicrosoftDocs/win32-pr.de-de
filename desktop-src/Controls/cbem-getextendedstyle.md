# <a name="cbem_getextendedstyle-message"></a>CBEM \_ GETEXTENDEDSTYLE-Nachricht

Ruft die erweiterten Stile ab, die für ein ComboBoxEx-Steuerelement verwendet werden.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt einen DWORD-Wert zurück, der die [erweiterten Stile des ComboBoxEx-Steuerelements](comboboxex-control-extended-styles.md) enthält, die für das Steuerelement verwendet werden.
