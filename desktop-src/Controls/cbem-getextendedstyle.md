# <a name="cbem_getextendedstyle-message"></a>CBEM \_ GetExtendedStyle-Meldung

Ruft die erweiterten Stile ab, die für ein ComboBoxEx-Steuerelement verwendet werden.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt einen DWORD-Wert zurück, der das für das Steuerelement verwendete [Erweiterte ComboBoxEx-Steuer](comboboxex-control-extended-styles.md) Element enthält.
