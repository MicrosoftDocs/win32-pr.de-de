# <a name="tb_getpadding-message"></a>TB \_ GETPADDING-Nachricht

Ruft die Auf padding für ein Symbolleisten-Steuerelement ab.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt einen **DWORD-Wert** zurück, der den horizontalen Auf padding im unteren Wort und den vertikalen Auf padding im hohen Wort in Pixel enthält.
