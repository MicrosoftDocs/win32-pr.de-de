# <a name="tb_getpadding-message"></a>TB \_ getpadding-Nachricht

Ruft die Auffüll Zeichen für ein Symbolleisten-Steuerelement ab.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt einen **DWORD** -Wert zurück, der den horizontalen Abstand im niedrigen Wort und den vertikalen Abstand im hohen Wort in Pixel enthält.
