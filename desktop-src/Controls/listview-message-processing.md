# <a name="default-list-view-message-processing"></a>Standard List-View Nachrichtenverarbeitung

In diesem Abschnitt wird die Fenster Nachrichtenverarbeitung aufgelistet, die von einem Listenansicht-Steuerelement ausgeführt wird.

## <a name="message-processing"></a>Nachrichtenverarbeitung

In der folgenden Tabelle wird die Fenster Nachrichtenverarbeitung beschrieben, die von einem Listenansicht-Steuerelement ausgeführt wird.



| `Message`                  | Verarbeitung wird ausgeführt                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
|--------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WM \_ -Zeichen**             | Sucht nach einem Listen Ansichts Element, das mit dem angegebenen Zeichen beginnt, und legt den Fokus auf das Element fest, wenn das Element gefunden wurde. Mehrere Zeichen, die innerhalb eines Timeout Intervalls empfangen werden, das derzeit um eine Sekunde fixiert ist, werden verkettet, und das Listenansicht-Steuerelement sucht nach einem Element, das mit der resultierenden Zeichenfolge beginnt.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **WM- \_ Befehl**          | Verarbeitet die Benachrichtigungs Codes " [en \_ Update](en-update.md) " und " [en \_ killfocus](en-killfocus.md) " und leitet alle anderen Bearbeitungs Steuerungs Benachrichtigungen an das übergeordnete Fenster weiter.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **WM \_ Erstellen**           | Führt die Initialisierung zusätzlich zu [**WM \_ nccreate**](/windows/desktop/winmsg/wm-nccreate)aus. Wenn das Fenster Stil des [**LVS- \_ shareimagelists**](list-view-window-styles.md) nicht angegeben ist, erstellt das Listenansicht-Steuerelement an dieser Stelle das Symbol und die kleinen Symbolbild Listen.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **WM \_ zerstören**          | Gibt während [**WM \_ Create**](/windows/desktop/winmsg/wm-create)zugeordnete Ressourcen frei.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **WM- \_ erasebkgnd**       | Löscht den Fenster Hintergrund mithilfe der aktuellen Hintergrundfarbe für das Listenansicht-Steuerelement. Wenn die Hintergrundfarbe der CLR \_ None-Wert ist, leitet das Listenansicht-Steuerelement die Nachricht an das übergeordnete Fenster weiter.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| **WM \_ getdlgcode**       | Gibt eine Kombination der dlgc \_ wanttab-und dlgc- \_ wantpfeile-Werte zurück.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **WM- \_ getFont**          | Gibt das Handle für die Schriftart der aktuellen Bezeichnung zurück.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **WM- \_ HScroll**          | Führt im Listenansicht-Steuerelement einen horizontalen Bildlauf durch                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| **WM- \_ KeyDown**          | Verarbeitet die Leertaste, die EINGABETASTE und die Pfeiltasten und sendet einen [LVN- \_ KeyDown](lvn-keydown.md) -Benachrichtigungs Code an das übergeordnete Fenster.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **WM- \_ killfokus**        | Zeichnet das fokussierte Listenelement ggf. neu und sendet einen [nm- \_ killfocus](nm-killfocus.md) -Benachrichtigungs Code an das übergeordnete Fenster.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **WM \_ lbuttondblclk**    | Sendet das übergeordnete Fenster einen [nm- \_ dblclk-Benachrichtigungs Code (Listenansicht)](nm-dblclk-list-view.md) .                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **WM \_ lbuttondown**      | Wird auf unterschiedliche Weise verarbeitet, je nachdem, ob ein Click-oder Drag-Vorgang initiiert wird. Um zu ermitteln, welcher Vorgang beteiligt ist, wechselt das Listenansicht-Steuerelement in eine modale Nachrichten Schleife, bis entweder die Schaltfläche losgelassen oder die Maus bewegt wird. Im Fall eines Click kann das Listenansicht-Steuerelement ändern, welches Element den Fokus besitzt und welche Elemente ausgewählt sind. dabei werden die Cursorposition, der Zustand der Umschalttaste und die STRG-Taste gedrückt. Anschließend sendet das Listenansicht-Steuerelement das übergeordnete Fenster mit dem Benachrichtigungs Code " [nm \_ Click (Listenansicht)](nm-click-list-view.md) ".<br/> Wenn das ziehen für ein Element beginnt, wählt das Listenansicht-Steuerelement den Fokus aus und legt den Fokus auf das Element fest. Anschließend wird ein [LVN \_ BeginDrag](lvn-begindrag.md) -Benachrichtigungs Code an das übergeordnete Fenster gesendet. Das übergeordnete Fenster ist dafür verantwortlich, den Zieh Vorgang tatsächlich auszuführen.<br/> Wenn das ziehen über den Hintergrund des Fensters beginnt, wechselt das Listenansicht-Steuerelement in eine andere modale Nachrichten Schleife, sodass der Benutzer ein Rechteck durchziehen der Maus bilden kann. Elemente innerhalb des Rechtecks werden ausgewählt.<br/> |
| **WM- \_ nccreate**         | Ordnet eine interne Datenstruktur zu und initialisiert sie und ruft dann die [**defwindowproc**](/windows/desktop/api/winuser/nf-winuser-defwindowproca) -Funktion auf.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| **WM \_ ncdestroy**        | Gibt Ressourcen frei, die vom Listenansicht-Steuerelement zugeordnet sind. Wenn der [**LVS- \_ shareimagelists**](list-view-window-styles.md) -Stil nicht verwendet wird, umfasst dies das Löschen der Listen für vollständige und kleine Bilder.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| **WM- \_ Benachrichtigung**           | Verarbeitet Benachrichtigungs Codes für Header Steuerelemente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **WM- \_ Paint**            | Zeigt alle Elemente im Aktualisierungs Bereich an. Bei Rückruf Elementen sendet das Steuerelement zuerst einen [LVN \_ getdispinfo](lvn-getdispinfo.md) -Benachrichtigungs Code an das Besitzer Fenster, um Anzeigeinformationen anzufordern. Wenn der *wParam* -Parameter ungleich NULL ist, geht das Steuerelement davon aus, dass es sich bei dem Wert um einen HDC handelt und zeichnet sich durch den Gerätekontext aus.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **WM- \_ rbuttondown**      | Wird auf die gleiche Weise wie die [**WM- \_ lbuttondown**](/windows/desktop/inputdev/wm-lbuttondown) -Nachricht verarbeitet, mit dem Unterschied, dass das Steuerelement einen nm-rclick-Benachrichtigungs Code [ \_ (Listenansicht)](nm-rclick-list-view.md) sendet (anstelle von [nm \_ Click (Listenansicht)](nm-click-list-view.md)) und einen [LVN \_ beginrdrag](lvn-beginrdrag.md) -Benachrichtigungs Code (anstelle von [LVN \_ BeginDrag](lvn-begindrag.md)). Beachten Sie, dass das-Steuerelement die entsprechende [**WM- \_ rbuttonup**](/windows/desktop/inputdev/wm-rbuttonup) -Nachricht verarbeitet und nicht versendet. Anwendungen können diese Meldung nicht sehen, auch wenn Sie das Steuerelement Unterklassen.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **WM- \_ SetFocus**         | Zeichnet das fokussierte Listenelement ggf. neu und sendet einen [nm \_ SetFocus](nm-setfocus.md) -Benachrichtigungs Code an das übergeordnete Fenster.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **WM- \_ setFont**          | Speichert den angegebenen Schriftart handle, leitet die Meldung ggf. an das Header Fenster weiter und zeichnet mithilfe der neuen Schriftart.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **WM \_ -Ziel**        | Aktiviert oder deaktiviert das erneute zeichnen.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **WM- \_ Timer**            | Startet die Bearbeitung einer Element Bezeichnung. Wenn der Benutzer auf die Bezeichnung des Fokus Elements klickt, legt das Listenansicht-Steuerelement einen Timer fest, anstatt sofort in den Bearbeitungsmodus zu wechseln. Der Timer ermöglicht es dem Listenansicht-Steuerelement, *nicht* in den Bearbeitungsmodus zu wechseln, wenn der Benutzer auf die Bezeichnung doppelklickt.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **WM- \_ VScroll**          | Führt im Listenansicht-Steuerelement einen vertikalen Bildlauf durch                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **WM-Windows-Server \_** | Aktualisiert die Fenster Scrollleisten. Wenn die aktuelle Ansicht eine Symbol-oder kleine Symbol Ansicht ist und der [**LVS \_ AutoArrange**](list-view-window-styles.md) -Stil angegeben ist, ordnet das Listenansicht-Steuerelement auch die Listenelemente an.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **WM- \_ wininichange**     | Verarbeitet Änderungen an Systemmetriken.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
