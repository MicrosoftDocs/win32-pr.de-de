# <a name="window-classes-commctrlh"></a>Window-Klassen (CommCtrl.h)

In diesem Abschnitt werden die Fensterklassennamen aufgelistet, die von der allgemeinen Steuerelementbibliothek bereitgestellt werden.



| Konstante                                                                                                                                                                     | Beschreibung                                                                                                                                                                                                                                                    |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="ANIMATE_CLASS"></span><span id="animate_class"></span><dl> <dt>**\_ANIMATE-KLASSE**</dt> </dl>                | Erstellt Animationssteuerelemente. Diese Steuerelemente zeigen unbeaufsichtigt einen überlappenden Audiovideoclip (AVI) an.<br/>                                                                                                                                                  |
| <span id="DATETIMEPICK_CLASS"></span><span id="datetimepick_class"></span><dl> <dt>**\_DATETIMEPICK-KLASSE**</dt> </dl> | Erstellt Steuerelemente für die Datums- und Uhrzeitauswahl. Diese Steuerelemente stellen eine einfache Schnittstelle zum Austauschen von Datums- und Uhrzeitinformationen mit einem Benutzer bereit. <br/>                                                                                                                |
| <span id="HOTKEY_CLASS"></span><span id="hotkey_class"></span><dl> <dt>**\_HOTKEY-KLASSE**</dt> </dl>                   | Erstellt Hot-Key-Steuerelemente. Diese Steuerelemente erleichtern dem Benutzer das Definieren von Hot Keys. <br/>                                                                                                                                                             |
| <span id="LINK_CLASS"></span><span id="link_class"></span><dl> <dt>**\_LINK-KLASSE**</dt> </dl>                         | Erstellt Syslink-Steuerelemente. Diese Steuerelemente werden mit Links verwendet. <br/>                                                                                                                                                                                      |
| <span id="MONTHCAL_CLASS"></span><span id="monthcal_class"></span><dl> <dt>**\_MONTHCAL-KLASSE**</dt> </dl>             | Erstellt Monatskalender-Steuerelemente. Diese Steuerelemente bieten einem Benutzer eine einfache Möglichkeit, ein Datum auf einer vertrauten Benutzeroberfläche auszuwählen. <br/>                                                                                                                        |
| <span id="NATIVEFNTCTL_CLASS"></span><span id="nativefntctl_class"></span><dl> <dt>**NATIVEFNTCTL-KLASSE \_**</dt> </dl> | Erstellt native Schriftartsteuerelemente. Diese Steuerelemente werden mit nativen Schriftarten verwendet. <br/>                                                                                                                                                                           |
| <span id="PROGRESS_CLASS"></span><span id="progress_class"></span><dl> <dt>**\_PROGRESS-KLASSE**</dt> </dl>             | Erstellt Statusleisten. Diese Steuerelemente geben den Fortschritt eines längeren Vorgangs an.<br/>                                                                                                                                                                 |
| <span id="REBARCLASSNAME"></span><span id="rebarclassname"></span><dl> <dt>**REBARCLASSNAME**</dt> </dl>              | Erstellt Neuleistensteuerelemente. Diese Steuerelemente fungieren als Container für untergeordnete Fenster. <br/>                                                                                                                                                                       |
| <span id="STANDARD_CLASSES"></span><span id="standard_classes"></span><dl> <dt>**\_STANDARDKLASSEN**</dt> </dl>       | Erstellt systeminterne User32-Steuerelemente wie Schaltfläche, Bearbeitung und Scrollleiste.<br/>                                                                                                                                                                            |
| <span id="STATUSCLASSNAME"></span><span id="statusclassname"></span><dl> <dt>**STATUSCLASSNAME**</dt> </dl>           | Erstellt Statusfenster. Diese Steuerelemente zeigen Statusinformationen in einem horizontalen Fenster an.<br/>                                                                                                                                                           |
| <span id="TOOLBARCLASSNAME"></span><span id="toolbarclassname"></span><dl> <dt>**TOOLBARCLASSNAME**</dt> </dl>        | Erstellt Symbolleisten. Diese Steuerelemente enthalten Schaltflächen, die Menübefehle ausführen. <br/>                                                                                                                                                                     |
| <span id="TOOLTIPS_CLASS"></span><span id="tooltips_class"></span><dl> <dt>**\_TOOLTIPS-KLASSE**</dt> </dl>             | Erstellt QuickInfo-Steuerelemente. Diese Steuerelemente zeigen ein kleines Popupfenster mit einer Textzeile an, die den Zweck eines Tools in einer Anwendung beschreibt.<br/>                                                                                            |
| <span id="TRACKBAR_CLASS"></span><span id="trackbar_class"></span><dl> <dt>**\_TRACKBAR-KLASSE**</dt> </dl>             | Erstellt Trackleisten. Mit diesen Steuerelementen kann der Benutzer durch Verschieben eines Schiebereglers aus einem Wertebereich auswählen. <br/>                                                                                                                                                   |
| <span id="UPDOWN_CLASS"></span><span id="updown_class"></span><dl> <dt>**\_UPDOWN-KLASSE**</dt> </dl>                   | Erstellt Auf-Ab-Steuerelemente. Diese Steuerelemente kombinieren ein Pfeilpaar mit einem Bearbeitungssteuerelement. Durch Klicken auf die Pfeile wird der Wert im Bearbeitungssteuerelement erhöht oder dekrementiert. <br/>                                                                                |
| <span id="WC_BUTTON"></span><span id="wc_button"></span><dl> <dt>**\_WC-SCHALTFLÄCHE**</dt> </dl>                            | Erstellt Schaltflächensteuerelemente. Mit diesen Steuerelementen kann ein Benutzer eine Aktion initiieren.<br/>                                                                                                                                                                        |
| <span id="WC_COMBOBOX"></span><span id="wc_combobox"></span><dl> <dt>**WC \_ COMBOBOX**</dt> </dl>                      | Erstellt Combobox-Steuerelemente. Diese Steuerelemente enthalten eine Dropdownliste, aus der ein einzelnes Element ausgewählt werden kann. <br/>                                                                                                                                       |
| <span id="WC_COMBOBOXEX"></span><span id="wc_comboboxex"></span><dl> <dt>**WC \_ COMBOBOXEX**</dt> </dl>                | Erstellt ComboBoxEx-Steuerelemente. Diese Steuerelemente stellen eine Erweiterung des Kombinationsfeld-Steuerelements bereit, das native Unterstützung für Elementbilder bereitstellt. <br/>                                                                                                            |
| <span id="WC_EDIT"></span><span id="wc_edit"></span><dl> <dt>**WC \_ EDIT**</dt> </dl>                                  | Erstellt Bearbeitungssteuerelemente. Diese Steuerelemente enthalten bearbeitbaren Text. <br/>                                                                                                                                                                                       |
| <span id="WC_HEADER"></span><span id="wc_header"></span><dl> <dt>**\_WC-HEADER**</dt> </dl>                            | Erstellt Headersteuerelemente. Diese Steuerelemente zeigen Überschriften am oberen Rand der Informationsspalten an und ermöglichen es dem Benutzer, die Informationen durch Klicken auf die Überschriften zu sortieren. <br/>                                                                                      |
| <span id="WC_LISTBOX"></span><span id="wc_listbox"></span><dl> <dt>**WC \_ LISTBOX**</dt> </dl>                         | Erstellt Listbox-Steuerelemente. Diese Steuerelemente enthalten Listen von Elementen. <br/>                                                                                                                                                                                   |
| <span id="WC_IPADDRESS"></span><span id="wc_ipaddress"></span><dl> <dt>**WC \_ IPADDRESS**</dt> </dl>                   | Erstellt IP-Adresssteuerelemente. Diese Steuerelemente ähneln einem Bearbeitungssteuerelement, ermöglichen es Ihnen jedoch, eine numerische Adresse im IP-Format (Internet protocol) einzugeben. <br/>                                                                                       |
| <span id="WC_LINK"></span><span id="wc_link"></span><dl> <dt>**WC \_ LINK**</dt> </dl>                                  | Erstellt SysLink-Steuerelemente. Diese Steuerelemente enthalten Hypertextlinks.<br/>                                                                                                                                                                                   |
| <span id="WC_LISTVIEW"></span><span id="wc_listview"></span><dl> <dt>**WC \_ LISTVIEW**</dt> </dl>                      | Erstellt Listenansichtssteuerelemente. Diese Steuerelemente zeigen eine Auflistung von Elementen an, die jeweils aus einem Symbol und einer Bezeichnung bestehen, und bieten mehrere Möglichkeiten, die Elemente anzuordnen.<br/>                                                                                    |
| <span id="WC_NATIVEFONTCTL"></span><span id="wc_nativefontctl"></span><dl> <dt>**WC \_ NATIVEFONTCTL**</dt> </dl>       | Erstellt native Schriftartsteuerelemente. Dies sind unsichtbare Steuerelemente, die im Hintergrund funktionieren, um die vordefinierten Steuerelemente eines Dialogfelds zum Anzeigen der aktuellen Systemsprache zu aktivieren.<br/>                                                                         |
| <span id="WC_PAGESCROLLER"></span><span id="wc_pagescroller"></span><dl> <dt>**WC \_ PAGESCROLLER**</dt> </dl>          | Erstellt Pagersteuerelemente. Diese Steuerelemente werden verwendet, um ein anderes Fenster einzublenden und zu scrollen. <br/>                                                                                                                                                              |
| <span id="WC_SCROLLBAR"></span><span id="wc_scrollbar"></span><dl> <dt>**WC \_ SCROLLBAR**</dt> </dl>                   | Erstellt Bildlaufleisten-Steuerelemente. Mit diesen Steuerelementen kann der Benutzer den Inhalt eines Fensters scrollen.<br/>                                                                                                                                                     |
| <span id="WC_STATIC"></span><span id="wc_static"></span><dl> <dt>**WC \_ STATIC**</dt> </dl>                            | Erstellt statische Steuerelemente. Diese Steuerelemente enthalten keinenditablen Text. <br/>                                                                                                                                                                                  |
| <span id="WC_TABCONTROL"></span><span id="wc_tabcontrol"></span><dl> <dt>**WC \_ TABCONTROL**</dt> </dl>                | Erstellt Registerkartensteuerelemente. Diese Steuerelemente definieren mehrere Seiten für denselben Bereich eines Fensters oder Dialogfelds. Jede Seite besteht aus einer Reihe von Informationen oder einer Gruppe von Steuerelementen, die eine Anwendung anzeigt, wenn der Benutzer die entsprechende Registerkarte auswählt.<br/> |
| <span id="WC_TREEVIEW"></span><span id="wc_treeview"></span><dl> <dt>**WC \_ TREEVIEW**</dt> </dl>                      | Erstellt Strukturansichtssteuerelemente. Diese Steuerelemente zeigen eine hierarchische Liste von Elementen an. Jedes Element besteht aus einer Bezeichnung und einer optionalen Bitmap.<br/>                                                                                                              |
