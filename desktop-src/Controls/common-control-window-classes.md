In diesem Abschnitt werden die von der allgemeinen Steuerelement Bibliothek bereitgestellten Fenster Klassennamen aufgelistet.



| Konstante                                                                                                                                                                     | BESCHREIBUNG                                                                                                                                                                                                                                                    |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="ANIMATE_CLASS"></span><span id="animate_class"></span><dl> <dt>**Animieren der \_ Klasse**</dt> </dl>                | Erstellt Animations Steuerelemente. | <span id="DATETIMEPICK_CLASS"></span><span id="datetimepick_class"></span><dl> <dt>**datetimepick- \_ Klasse**</dt> </dl> | Erstellt Steuerelemente für Datums-und Zeitauswahl. | <span id="HOTKEY_CLASS"></span><span id="hotkey_class"></span><dl> <dt>**Hotkey- \_ Klasse**</dt> </dl>                   | Erstellt Hot Key-Steuerelemente. Diese Steuerelemente erleichtern dem Benutzer das Definieren von Abkürzungs Schlüsseln. <br/>                                                                                                                                                             |
| <span id="LINK_CLASS"></span><span id="link_class"></span><dl> <dt>**Link \_ Klasse**</dt> </dl>                         | Erstellt Syslink-Steuerelemente. | <span id="MONTHCAL_CLASS"></span><span id="monthcal_class"></span><dl> <dt>**monthcal- \_ Klasse**</dt> </dl>             | Erstellt Monatskalender-Steuerelemente. | <span id="NATIVEFNTCTL_CLASS"></span><span id="nativefntctl_class"></span><dl> <dt>**nativesntctl- \_ Klasse**</dt> </dl> | Erstellt native Schriftart Steuerelemente. | <span id="PROGRESS_CLASS"></span><span id="progress_class"></span><dl> <dt>**Fortschritts \_ Klasse**</dt> </dl>             | Erstellt Status leisten. | <span id="REBARCLASSNAME"></span><span id="rebarclassname"></span><dl> <dt>**Rebarclassname**</dt> </dl>              | Erstellt Grund leisten-Steuerelemente. Diese Steuerelemente fungieren als Container für untergeordnete Fenster. <br/>                                                                                                                                                                       |
| <span id="STANDARD_CLASSES"></span><span id="standard_classes"></span><dl> <dt>**Standard \_ Klassen**</dt> </dl>       | Erstellt systeminterne User32-Steuerelemente, z. b. Schaltfläche, bearbeiten und Bild Lauf Leiste.<br/>                                                                                                                                                                            |
| <span id="STATUSCLASSNAME"></span><span id="statusclassname"></span><dl> <dt>**Status Klassenname**</dt> </dl>           | Erstellt Statusfenster. Diese Steuerelemente zeigen Statusinformationen in einem horizontalen Fenster an.<br/>                                                                                                                                                           |
| <span id="TOOLBARCLASSNAME"></span><span id="toolbarclassname"></span><dl> <dt>**Toolbarclassname**</dt> </dl>        | Erstellt Symbolleisten. | <span id="TOOLTIPS_CLASS"></span><span id="tooltips_class"></span><dl> <dt>**Tooltips- \_ Klasse**</dt> </dl>             | Erstellt QuickInfo-Steuerelemente. | <span id="TRACKBAR_CLASS"></span><span id="trackbar_class"></span><dl> <dt>**TrackBar- \_ Klasse**</dt> </dl>             | Erstellt trackbars. | <span id="UPDOWN_CLASS"></span><span id="updown_class"></span><dl> <dt>**UpDown- \_ Klasse**</dt> </dl>                   | Erstellt auf-ab-Steuerelemente. Durch Klicken auf die Pfeile wird der Wert im Bearbeitungs Steuerelement erhöht oder verringert. <br/>                                                                                |
| <span id="WC_BUTTON"></span><span id="wc_button"></span><dl> <dt>**WC- \_ Schaltfläche**</dt> </dl>                            | Erstellt Schaltflächen-Steuerelemente. Diese Steuerelemente ermöglichen es einem Benutzer, eine Aktion zu initiieren.<br/>                                                                                                                                                                        |
| <span id="WC_COMBOBOX"></span><span id="wc_combobox"></span><dl> <dt>**Kombinations Feld "WC" \_**</dt> </dl>                      | Erstellt ComboBox-Steuerelemente. | <span id="WC_COMBOBOXEX"></span><span id="wc_comboboxex"></span><dl> <dt>**WC \_ ComboBoxEx**</dt> </dl>                | Erstellt ComboBoxEx-Steuerelemente. | <span id="WC_EDIT"></span><span id="wc_edit"></span><dl> <dt>**WC- \_ Bearbeitung**</dt> </dl>                                  | Erstellt Bearbeitungs Steuerelemente. Diese Steuerelemente enthalten bearbeitbaren Text. <br/>                                                                                                                                                                                       |
| <span id="WC_HEADER"></span><span id="wc_header"></span><dl> <dt>**WC- \_ Header**</dt> </dl>                            | Erstellt Header Steuerelemente. Diese Steuerelemente zeigen Überschriften am oberen Rand der Spalten von Informationen an und ermöglichen dem Benutzer das Sortieren der Informationen durch Klicken auf die Überschriften. <br/>                                                                                      |
| <span id="WC_LISTBOX"></span><span id="wc_listbox"></span><dl> <dt>**WC- \_ ListBox**</dt> </dl>                         | Erstellt ListBox-Steuerelemente. Diese Steuerelemente enthalten Listen von Elementen. <br/>                                                                                                                                                                                   |
| <span id="WC_IPADDRESS"></span><span id="wc_ipaddress"></span><dl> <dt>**WC- \_ IP-Adresse**</dt> </dl>                   | Erstellt IP-Adress Steuerelemente. Diese Steuerelemente ähneln einem Bearbeitungs Steuerelement, aber Sie ermöglichen es Ihnen, eine numerische Adresse im IP-Format (Internet Protocol) einzugeben. <br/>                                                                                       |
| <span id="WC_LINK"></span><span id="wc_link"></span><dl> <dt>**Link "WC" \_**</dt> </dl>                                  | Erstellt Syslink-Steuerelemente. Diese Steuerelemente enthalten Hypertext Links.<br/>                                                                                                                                                                                   |
| <span id="WC_LISTVIEW"></span><span id="wc_listview"></span><dl> <dt>**WC- \_ ListView**</dt> </dl>                      | Erstellt Listenansicht-Steuerelemente. Diese Steuerelemente zeigen eine Auflistung von Elementen an, die jeweils aus einem Symbol und einer Bezeichnung bestehen und verschiedene Methoden zum Anordnen der Elemente bereitstellen.<br/>                                                                                    |
| <span id="WC_NATIVEFONTCTL"></span><span id="wc_nativefontctl"></span><dl> <dt>**WC \_ nativefontctl**</dt> </dl>       | Erstellt native Schriftart Steuerelemente. Dabei handelt es sich um unsichtbare Steuerelemente, die im Hintergrund funktionieren, um die vordefinierten Steuerelemente eines Dialog Felds zum Anzeigen der aktuellen Systemsprache zu aktivieren.<br/>                                                                         |
| <span id="WC_PAGESCROLLER"></span><span id="wc_pagescroller"></span><dl> <dt>**WC- \_ pagescroller**</dt> </dl>          | Erstellt Pager-Steuerelemente. Diese Steuerelemente werden verwendet, um ein anderes Fenster zu speichern und zu scrollen. <br/>                                                                                                                                                              |
| <span id="WC_SCROLLBAR"></span><span id="wc_scrollbar"></span><dl> <dt>**WC- \_ Scrollleiste**</dt> </dl>                   | Erstellt ScrollBar-Steuerelemente. Mithilfe dieser Steuerelemente kann der Benutzer einen Bildlauf im Inhalt eines Fensters durchführen.<br/>                                                                                                                                                     |
| <span id="WC_STATIC"></span><span id="wc_static"></span><dl> <dt>**WC ( \_ statisch)**</dt> </dl>                            | Erstellt statische Steuerelemente. Diese Steuerelemente enthalten nicht bearbeitbaren Text. <br/>                                                                                                                                                                                  |
| <span id="WC_TABCONTROL"></span><span id="wc_tabcontrol"></span><dl> <dt>**WC \_ TabControl**</dt> </dl>                | Erstellt Registerkarten-Steuerelemente. Diese Steuerelemente definieren mehrere Seiten für denselben Bereich eines Fensters oder Dialog Felds. Jede Seite besteht aus einem Satz von Informationen oder einer Gruppe von Steuerelementen, die von einer Anwendung angezeigt werden, wenn der Benutzer die entsprechende Registerkarte auswählt.<br/> |
| <span id="WC_TREEVIEW"></span><span id="wc_treeview"></span><dl> <dt>**WC- \_ TreeView**</dt> </dl>                      | Erstellt Strukturansicht-Steuerelemente. Diese Steuerelemente zeigen eine hierarchische Liste von Elementen an. Jedes Element besteht aus einer Bezeichnung und einer optionalen Bitmap.<br/>                                                                                                              |
