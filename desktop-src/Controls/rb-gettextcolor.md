# <a name="rb_gettextcolor-message"></a>RB \_ gettextcolor-Meldung

Ruft die Standard Textfarbe eines Grund leisten-Steuer Elements ab.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt einen [**COLORREF**](/windows/desktop/gdi/colorref) -Wert zurück, der die aktuelle Standard Textfarbe darstellt.
