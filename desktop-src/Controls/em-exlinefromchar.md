# <a name="em_exlinefromchar-message"></a>EM \_ exlinefromchar-Meldung

Bestimmt, welche Zeile das angegebene Zeichen in einem Rich-Edit-Steuerelement enthält.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Dieser Parameter wird nicht verwendet. Er muss NULL sein.

</dd> <dt>

*lParam* 
</dt> <dd>

NULL basierter Index des Zeichens.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Diese Meldung gibt den NULL basierten Index der Zeile zurück.
