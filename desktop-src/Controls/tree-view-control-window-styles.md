# <a name="tree-view-control-window-styles"></a>Tree-View-Steuerelementfensterstile

In diesem Abschnitt werden Fensterstile aufgelistet, die beim Erstellen von Strukturansichtssteuerelementen verwendet werden.

| <span id="TVS_CHECKBOXES"></span><span id="tvs_checkboxes"></span><dl> <dt>**\_TVS-KONTROLLKÄSTCHEN**</dt> </dl>                | [Version 4.70.](common-control-versions.md) Aktiviert Kontrollkästchen für Elemente in einem Strukturansichtssteuerelement. Ein Kontrollkästchen wird nur angezeigt, wenn dem Element ein Bild zugeordnet ist. Wenn dieses Format festgelegt ist, verwendet das Steuerelement [**drawFrameControl,**](/windows/desktop/api/winuser/nf-winuser-drawframecontrol) um eine Zustandsbildliste mit zwei Bildern zu erstellen und festzulegen. Statusbild 1 ist das nicht aktivierte Kontrollkästchen, und Statusbild 2 ist das Kontrollkästchen. Durch Festlegen des Zustandsbilds auf 0 (null) wird das Kontrollkästchen vollständig entfernt. Weitere Informationen finden Sie unter [Arbeiten mit Zustandsbildindizes.](tree-view-controls.md)<br/> [Version 5.80.](common-control-versions.md) Zeigt ein Kontrollkästchen an, auch wenn dem Element kein Bild zugeordnet ist.<br/> Sobald ein Strukturansichtssteuerelement mit diesem Stil erstellt wurde, kann der Stil nicht mehr entfernt werden. Stattdessen müssen Sie das Steuerelement zerstören und stattdessen ein neues erstellen. Durch das Zerstören des Strukturansichtssteuerelements wird die Bildliste des Kontrollkästchenzustands nicht zerstört. Sie müssen sie explizit zerstören. Abrufen des Handles für die Statusbildliste durch Senden einer [**TVM \_ GETIMAGELIST-Nachricht**](tvm-getimagelist.md) an das Strukturansichtssteuerelement. Zerstören Sie dann die Bildliste mit [**ImageList \_ Destroy**](/windows/desktop/api/Commctrl/nf-commctrl-imagelist_destroy).<br/> Wenn Sie diesen Stil verwenden möchten, müssen Sie den TVS \_ CHECKBOXES-Stil mit [**SetWindowLong**](/windows/desktop/api/winuser/nf-winuser-setwindowlonga) festlegen, nachdem Sie das Treeview-Steuerelement erstellt und die Struktur aufgefüllt haben. Andernfalls werden die Kontrollkästchen je nach Zeitsteuerungsproblemen möglicherweise deaktiviert angezeigt.<br/> |

Um Schaltflächen mit Elementen im Stamm der Strukturansicht einzufügen, \_ muss TVS LINESATROOT ebenfalls angegeben werden. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
