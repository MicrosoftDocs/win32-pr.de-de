# <a name="tdm_set_progress_bar_pos-message"></a>TDM-Statusanzeige- \_ \_ POS- \_ \_ Nachricht festlegen

Legt die Position der Statusanzeige in einem Aufgaben Dialogfeld fest.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* \[ in\]
</dt> <dd>

Ein **int** -Wert, der die neue Position angibt.

</dd> <dt>

*LPARAM* \[ in\]
</dt> <dd>

Muss Null sein.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt die vorherige Position zurück.
