# <a name="general-control-reference"></a>Allgemeine Steuerelementreferenz

Dieser Abschnitt enthält Referenzinformationen zu Programmierelementen, die für mehrere Steuerelemente und nicht nur für ein bestimmtes Steuerelement gelten. Es gibt Funktionen, Makros, Nachrichten, Benachrichtigungen und Strukturen, die von den meisten Steuerelementen unterstützt werden. Beispielsweise verwenden die meisten Steuerelemente die [NM \_ HOVER-Benachrichtigungen,](nm-hover.md) um Mausklicks zu verarbeiten.
