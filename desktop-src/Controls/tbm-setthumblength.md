# <a name="tbm_setthumblength-message"></a>TBM- \_ setthumblength-Meldung

Legt die Länge des Schiebereglers in einer TrackBar fest. Diese Meldung wird ignoriert, wenn die TrackBar nicht über den [**TSB- \_ FixedLength**](trackbar-control-styles.md) -Stil verfügt.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Länge des Schiebereglers in Pixel.

</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Kein Rückgabewert.
