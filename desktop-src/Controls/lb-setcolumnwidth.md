# <a name="lb_setcolumnwidth-message"></a>LB- \_ setcolumnwidth-Nachricht

Legt die Breite aller Spalten in einem Listenfeld mit mehreren Spalten in Pixel fest.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Gibt die Breite aller Spalten in Pixel an.

</dd> <dt>

*lParam* 
</dt> <dd>

Dieser Parameter wird nicht verwendet.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Diese Meldung gibt keinen Wert zurück.
