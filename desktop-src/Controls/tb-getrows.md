# <a name="tb_getrows-message"></a>TB \_ GETROWS-Nachricht

Ruft die Anzahl der Zeilen von Schaltflächen in einer Symbolleiste mit dem [**TBSTYLE \_ WRAPABLE-Stil**](toolbar-control-and-button-styles.md) ab.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt die Anzahl der Zeilen zurück.
