# <a name="dtm_getmcstyle-message"></a>DTM \_ getmcstyle-Nachricht

Ruft den Stil eines DTP-Steuer Elements ab. Senden Sie diese Nachricht explizit oder mithilfe des [**DateTime- \_ getmonthcalstyle**](/windows/desktop/api/Commctrl/nf-commctrl-datetime_getmonthcalstyle) -Makros.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Muss Null sein.

</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt den Stil Wert des-Steuer Elements zurück. Weitere Informationen finden Sie unter [Month Calendar Control Styles](month-calendar-control-styles.md).
