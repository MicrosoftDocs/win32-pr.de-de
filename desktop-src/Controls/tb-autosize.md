# <a name="tb_autosize-message"></a>\_TB-AUTOIZE-Nachricht

Bewirkt, dass die Größe einer Symbolleiste geändert wird.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Kein Rückgabewert.

## <a name="remarks"></a>Hinweise

Eine Anwendung sendet die **\_ TB-AUTOIZE-Nachricht,** nachdem die Größe einer Symbolleiste geändert wurde, indem sie entweder die Schaltfläche oder Bitmapgröße festlegt oder zeichenfolgen zum ersten Mal hinzufügt.
