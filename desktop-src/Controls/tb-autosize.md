# <a name="tb_autosize-message"></a>TB \_ AutoSize-Nachricht

Bewirkt, dass die Größe einer Symbolleiste geändert wird.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Kein Rückgabewert.

## <a name="remarks"></a>Bemerkungen

Eine Anwendung sendet die **TB \_ AutoSize** -Nachricht, nachdem die Größe einer Symbolleiste geändert wurde, indem entweder die Schaltfläche oder die Bitmapgröße festgelegt oder Zeichen folgen zum ersten Mal hinzugefügt werden.
