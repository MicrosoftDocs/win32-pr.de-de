# <a name="toolbar"></a>Symbolleiste

Dieser Abschnitt enthält Informationen zu den Programmierelementen, die mit Symbolleistensteuerelementen verwendet werden.

### <a name="overviews"></a>Übersichten



| Thema                                                   | Inhalte                                                                                                                                                                                                                                                                                                                                 |
|---------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Informationen zu Symbolleistensteuerelementen](toolbar-controls-overview.md) | Eine Symbolleiste ist ein Steuerelement, das mindestens eine Schaltfläche enthält. Jede Schaltfläche sendet beim Klicken durch einen Benutzer eine Befehlsmeldung an das übergeordnete Fenster. In der Regel entsprechen die Schaltflächen in einer Symbolleiste Elementen im Menü der Anwendung und bieten eine zusätzliche und direktere Möglichkeit für den Benutzer, auf die Befehle einer Anwendung zuzugreifen.<br/> |
| [Verwenden von Symbolleistensteuerelementen](using-toolbar-controls.md)    | Dieses Thema enthält Implementierungsdetails und Beispielcode für die Verwendung von Symbolleistensteuerelementen in Ihren Anwendungen.<br/>                                                                                                                                                                                                                  |
