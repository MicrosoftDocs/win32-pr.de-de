# <a name="about-buttons"></a>Informationen zu Schaltflächen

Eine Schaltfläche ist ein Steuerelement, auf das der Benutzer klicken kann, um Eingaben für eine Anwendung bereitzustellen.

Es gibt mehrere Arten von Schaltflächen und mindestens einen Schaltflächen Stil, um zwischen den Schaltflächen desselben Typs zu unterscheiden. Der Benutzer klickt mit der Maus oder der Tastatur auf eine Schaltfläche. Wenn Sie auf eine Schaltfläche klicken, wird in der Regel das visuelle Aussehen und der zugehörige Zustand geändert (z. b. von aktiviert Das System, die Schaltfläche und die Anwendung arbeiten zusammen, um die Darstellung und den Zustand der Schaltfläche zu ändern. Eine Schaltfläche kann Nachrichten an das übergeordnete Fenster senden, und ein übergeordnetes Fenster kann Nachrichten an eine Schaltfläche Senden. Einige Schaltflächen werden vom System und einige der Anwendungen gezeichnet. Schaltflächen können allein oder in Gruppen verwendet werden und können mit oder ohne Anwendungs definiertem Text (eine Bezeichnung) angezeigt werden. Sie gehören zur Schaltflächen Fenster Klasse.
