# <a name="using-edit-controls"></a>Verwenden von Bearbeitungs Steuerelementen

Dieser Abschnitt enthält Informationen und Beispielcode für die Implementierung von Bearbeitungs Steuerelementen.

Bearbeitungs Steuerelemente werden in der Regel in Dialogfeldern verwendet, Sie können aber auch im Client Bereich eines Standard Fensters verwendet werden. Einzeilige Bearbeitungs Steuerelemente sind hilfreich, um eine einzelne Zeichenfolge vom Benutzer abzurufen. Mehrzeilige Bearbeitungs Steuerelemente ermöglichen es Ihnen, die meisten Funktionen eines einfachen Textprozessors in der Anwendung zu implementieren.

Die folgenden Themen werden erörtert.
