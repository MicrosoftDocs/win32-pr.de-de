# <a name="using-edit-controls"></a>Verwenden von Steuerelementen zum Bearbeiten

Dieser Abschnitt enthält Informationen und Beispielcode für die Implementierung von Bearbeitungssteuerelementen.

Bearbeitungssteuerelemente werden in der Regel in Dialogfeldern verwendet, aber Sie können sie auch im Clientbereich eines Standardfensters verwenden. Einzeilen-Bearbeitungssteuerelemente sind nützlich, um eine einzelne Zeichenfolge vom Benutzer abrufen zu können. Mit mehrline-Bearbeitungssteuerelementen können Sie die meisten Features eines einfachen Textprozessors in Ihrer Anwendung implementieren.

Die folgenden Themen werden erläutert.
