# <a name="tb_isbuttonindeterminate-message"></a>TB \_ isbuttonindeend-Nachricht

Bestimmt, ob die angegebene Schaltfläche in einer Symbolleiste unbestimmt ist.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Der Befehls Bezeichner der Schaltfläche.

</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt einen Wert ungleich 0 (null) zurück, wenn die Schaltfläche unbestimmt ist, andernfalls NULL
