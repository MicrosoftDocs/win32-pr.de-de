# <a name="about-pager-controls"></a>Informationen zu Pager-Steuerelementen

Ein *Pager-Steuerelement* ist ein Fenstercontainer, der mit einem Fenster verwendet wird, das nicht über genügend Anzeigebereich verfügt, um den ganzen Inhalt anzuzeigen. Mit dem Pager-Steuerelement kann der Benutzer zu dem Bereich des Fensters scrollen, der derzeit nicht angezeigt wird.

-   [Informationen zu Pager-Steuerelementen](#about-pager-controls)

## <a name="about-pager-controls"></a>Informationen zu Pager-Steuerelementen

Microsoft Internet Explorer Version 4.0 (commctrl.dll Version 4.71) führt das Pager-Steuerelement ein. Dieses Steuerelement ist in Situationen nützlich, in denen ein Fenster nicht über genügend Bereich zum Anzeigen eines untergeordneten Fensters verfügt. Wenn Ihre Anwendung beispielsweise über eine Symbolleiste verfügt, die nicht breit genug ist, um alle Elemente anzuzeigen, können Sie die Symbolleiste einem Pager-Steuerelement zuweisen, und Benutzer können nach links oder rechts scrollen, um auf alle Elemente zu zugreifen. Sie können auch Pagersteuerelemente erstellen, die vertikal scrollen.

Ein Fenster, das dem Pager-Steuerelement zugewiesen ist, wird als *enthaltenes Fenster bezeichnet.*

Der folgende Screenshot zeigt eine Symbolleiste, die in einem Pager-Steuerelement enthalten ist. Das Pager-Steuerelement wird rot angezeigt, um anzuzeigen, welche Bereiche des Steuerelements sichtbar sind.
