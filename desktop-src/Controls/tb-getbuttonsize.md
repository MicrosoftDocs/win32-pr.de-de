# <a name="tb_getbuttonsize-message"></a>TB \_ getbuttonsize-Nachricht

Ruft die aktuelle Breite und Höhe von Symbolleisten Schaltflächen in Pixel ab.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt einen **DWORD** -Wert zurück, der die Width-und Height-Werte im unteren und großen Wort enthält.
