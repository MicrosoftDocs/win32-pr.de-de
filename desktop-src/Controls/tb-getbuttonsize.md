# <a name="tb_getbuttonsize-message"></a>TB \_ GETBUTTONSIZE-Nachricht

Ruft die aktuelle Breite und Höhe der Symbolleistenschaltflächen in Pixel ab.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt einen **DWORD-Wert** zurück, der die Werte für Breite und Höhe im unteren bzw. hohen Wort enthält.
