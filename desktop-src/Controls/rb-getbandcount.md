# <a name="rb_getbandcount-message"></a>RB \_ GetBandCount-Meldung

Ruft die Anzahl der Bänder ab, die sich derzeit im Grund leisten-Steuerelement befinden.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt einen **uint** -Wert zurück, der die Anzahl der dem Steuerelement zugewiesenen Bänder darstellt.
