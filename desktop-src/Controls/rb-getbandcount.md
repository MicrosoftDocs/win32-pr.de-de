# <a name="rb_getbandcount-message"></a>RB \_ GETBANDCOUNT-Nachricht

Ruft die Anzahl der Bänder ab, die sich derzeit im Rebar-Steuerelement befinden.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt einen **UINT-Wert** zurück, der die Anzahl der Bänder darstellt, die dem Steuerelement zugewiesen sind.
