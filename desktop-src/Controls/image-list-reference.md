Dieser Abschnitt enthält Informationen zu den Programmier Elementen, die mit Bildlisten verwendet werden.

| Thema                          | Inhalte                                                                                                            |

| [Bildlisten](image-lists.md) | Eine Bildliste ist eine Auflistung von Bildern derselben Größe, auf die jeweils durch ihren Index verwiesen werden kann.<br/> |
