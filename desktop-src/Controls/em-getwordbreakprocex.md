# <a name="em_getwordbreakprocex-message"></a>EM \_ GETWORDBREAKPROCEX-Nachricht

Ruft die Adresse der derzeit registrierten erweiterten Wörterpausenprozedur für ein Rich-Edit-Steuerelement ab.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Nicht verwendet; muss 0 (null) sein.

</dd> <dt>

*lParam* 
</dt> <dd>

Nicht verwendet; muss 0 (null) sein.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Die Nachricht gibt die Adresse der aktuellen Prozedur zurück.
