# <a name="em_getwordbreakprocex-message"></a>EM \_ getwordbreakprocex-Meldung

Ruft die Adresse der derzeit registrierten Prozedur für erweiterte Wort Umbruch für ein Rich-Edit-Steuerelement ab.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Nicht verwendet; muss 0 (null) sein.

</dd> <dt>

*lParam* 
</dt> <dd>

Nicht verwendet; muss 0 (null) sein.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Die Meldung gibt die Adresse der aktuellen Prozedur zurück.
