# <a name="tb_getimagelistcount-message"></a>TB \_ getimagelistcount-Nachricht

Ruft die Anzahl der Bildlisten ab, die der Symbolleiste zugeordnet sind.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt die Anzahl der Bildlisten zurück.
