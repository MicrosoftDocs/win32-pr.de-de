# <a name="acm_isplaying-message"></a>ACM- \_ isplay-Nachricht

Überprüft, ob ein Audio-Video Interleaved-Clip (AVI) wiedergegeben wird. Sie können diese Nachricht explizit senden oder das [**Animieren von \_ isplay**](/windows/desktop/api/Commctrl/nf-commctrl-animate_isplaying) -Makro verwenden.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Muss Null sein.

</dd> <dt>

*lParam* 
</dt> <dd>

Muss Null sein.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt bei Erfolg einen Wert ungleich 0 (null) zurück, andernfalls NULL.
