# <a name="tb_movebutton-message"></a>TB-Meldungs \_ Tasten Meldung

Verschiebt eine Schaltfläche von einem Index zu einem anderen.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

NULL basierter Index der Schaltfläche, die verschoben werden soll.

</dd> <dt>

*lParam* 
</dt> <dd>

Der null basierte Index, an dem die Schaltfläche verschoben wird.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt bei Erfolg einen Wert ungleich 0 (null) zurück, andernfalls NULL.
