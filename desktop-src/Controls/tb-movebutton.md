# <a name="tb_movebutton-message"></a>TB \_ MOVEBUTTON-Nachricht

Verschiebt eine Schaltfläche von einem Index in einen anderen.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Nullbasierter Index der schaltfläche, die verschoben werden soll.

</dd> <dt>

*lParam* 
</dt> <dd>

Nullbasierter Index, in den die Schaltfläche verschoben wird.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt bei Erfolg einen Wert ungleich 0 (null) oder andernfalls 0 (null) zurück.
