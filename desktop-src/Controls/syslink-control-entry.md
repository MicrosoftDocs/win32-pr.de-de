# <a name="syslink"></a>SysLink

Dieser Abschnitt enthält Informationen zu den Programmierelementen, die mit SysLink-Steuerelementen verwendet werden.

### <a name="overviews"></a>Übersichten



| Thema                                    | Inhalte                                                                                       |
|------------------------------------------|------------------------------------------------------------------------------------------------|
| [SysLink-Steuerelemente](syslink-overview.md) | Das SysLink-Steuerelement bietet eine praktische Möglichkeit zum Einbetten von Hypertextlinks in ein Fenster.<br/> |



 

### <a name="messages"></a>Nachrichten



| Thema                                           | Inhalte                                                                             |
|-------------------------------------------------|--------------------------------------------------------------------------------------|
| [**LM \_ GETIDEALHEIGHT**](lm-getidealheight.md) | Ruft die bevorzugte Höhe eines Links für die aktuelle Breite des Steuerelements ab.<br/> |
| [**LM \_ GETIDEALSIZE**](lm-getidealsize.md)     | Ruft die bevorzugte Höhe eines Links für die aktuelle Breite des Steuerelements ab.<br/> |
| [**LM \_ GETITEM**](lm-getitem.md)               | Ruft die Zustände und Attribute eines Elements ab.<br/>                           |
| [**LM \_ HITTEST**](lm-hittest.md)               | Bestimmt, ob der Benutzer auf den angegebenen Link geklickt hat.<br/>                   |
| [**LM \_ SETITEM**](lm-setitem.md)               | Legt die Zustände und Attribute eines Elements fest.<br/>                                |



 

### <a name="structures"></a>Strukturen



| Thema                                | Inhalte                                                                                                                                                                           |
|--------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GIGAITTESTINFO**](/windows/win32/api/commctrl/ns-commctrl-lhittestinfo) | Wird verwendet, um Informationen über den Link zu erhalten, der einem bestimmten Speicherort entspricht. <br/>                                                                                              |
| [**LITEM**](/windows/win32/api/commctrl/ns-commctrl-litem)               | Wird zum Festlegen und Abrufen von Informationen zu einem Linkelement verwendet.<br/>                                                                                                                 |
| [**NMLINK**](/windows/win32/api/commctrl/ns-commctrl-nmlink)             | NMLINK [**Enthält**](/windows/win32/api/commctrl/ns-commctrl-nmlink) Benachrichtigungsinformationen. Senden Sie diese Struktur mit den [NM \_ CLICK- oder](nm-click-syslink.md) NM [ \_ RETURN-Meldungen.](nm-return.md)<br/> |
