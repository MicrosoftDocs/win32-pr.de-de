# <a name="syslink"></a>SysLink

Dieser Abschnitt enthält Informationen zu den Programmier Elementen, die mit Syslink-Steuerelementen verwendet werden.

### <a name="overviews"></a>Übersichten



| Thema                                    | Inhalte                                                                                       |
|------------------------------------------|------------------------------------------------------------------------------------------------|
| [Syslink-Steuerelemente](syslink-overview.md) | Das Syslink-Steuerelement stellt eine bequeme Möglichkeit zum Einbetten von Hypertext-Links in einem Fenster dar.<br/> |



 

### <a name="messages"></a>Meldungen



| Thema                                           | Inhalte                                                                             |
|-------------------------------------------------|--------------------------------------------------------------------------------------|
| [**LM- \_ getidealheight**](lm-getidealheight.md) | Ruft die bevorzugte Höhe eines Links für die aktuelle Breite des Steuer Elements ab.<br/> |
| [**LM- \_ getidealsize**](lm-getidealsize.md)     | Ruft die bevorzugte Höhe eines Links für die aktuelle Breite des Steuer Elements ab.<br/> |
| [**LM- \_ GetItem**](lm-getitem.md)               | Ruft die Zustände und Attribute eines Elements ab.<br/>                           |
| [**LM- \_ HitTest**](lm-hittest.md)               | Bestimmt, ob der Benutzer auf den angegebenen Link geklickt hat.<br/>                   |
| [**LM \_ -Element**](lm-setitem.md)               | Legt die Zustände und Attribute eines Elements fest.<br/>                                |



 

### <a name="structures"></a>Strukturen



| Thema                                | Inhalte                                                                                                                                                                           |
|--------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Lhittestinfo**](/windows/win32/api/commctrl/ns-commctrl-lhittestinfo) | Wird verwendet, um Informationen über den Link zu erhalten, der einem bestimmten Speicherort entspricht. <br/>                                                                                              |
| [**LItem**](/windows/win32/api/commctrl/ns-commctrl-litem)               | Wird verwendet, um Informationen zu einem Link Element festzulegen und abzurufen.<br/>                                                                                                                 |
| [**Nmlink**](/windows/win32/api/commctrl/ns-commctrl-nmlink)             | Der [**nmlink**](/windows/win32/api/commctrl/ns-commctrl-nmlink) enthält Benachrichtigungs Informationen. Senden Sie diese Struktur mit den [nm- \_ Click](nm-click-syslink.md) -oder [nm- \_ Rückgabe](nm-return.md) Nachrichten.<br/> |
