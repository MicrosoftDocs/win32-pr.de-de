# <a name="em_getmargins-message"></a>EM \_ getMargin-Nachricht

Ruft die Breite des linken und rechten Rands für ein Bearbeitungs Steuerelement ab.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Nicht verwendet; muss 0 (null) sein.

</dd> <dt>

*lParam* 
</dt> <dd>

Nicht verwendet; muss 0 (null) sein.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt die Breite des linken Randes des loworts und die Breite des rechten Rands im HIWORD zurück.

## <a name="remarks"></a>Bemerkungen

Umfassende **Bearbeitung:** Die **EM \_ getMargin** -Meldung wird nicht unterstützt.
