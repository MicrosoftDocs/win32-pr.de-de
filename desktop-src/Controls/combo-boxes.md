# <a name="combo-box-windows-controls"></a>Kombinations Feld (Windows-Steuerelemente)

Dieser Abschnitt enthält Informationen zu den Programmier Elementen, die mit Kombinations Feldern verwendet werden. Ein Kombinations *Feld* ist ein eindeutiger Typ von Steuerelement, der durch die ComboBox-Klasse definiert wird, die einen Großteil der Funktionalität eines Listen Felds und eines Bearbeitungs Steuer Elements kombiniert.
