# <a name="lvm_getgroupcount-message"></a>LVM \_ getgroupcount-Nachricht

Ruft die Anzahl der Gruppen ab.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* \[ vorgenommen\]
</dt> <dd>

Nicht verwendet. Muss 0 sein.

</dd> <dt>

*lParam* 
</dt> <dd>

Nicht verwendet. Muss 0 sein.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt die Anzahl der Gruppen zurück.
