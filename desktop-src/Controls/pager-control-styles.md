# <a name="pager-control-styles"></a>Pager-Steuerelementstile

In diesem Abschnitt werden die Fensterstile aufgeführt, die beim Erstellen von Pagersteuerelementen verwendet werden.



| Konstante                                                                                                                                                         | Beschreibung                                                                                                                                                                                                                 |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="PGS_AUTOSCROLL"></span><span id="pgs_autoscroll"></span><dl> <dt>**PGS \_ AUTOCROLL**</dt> </dl> | Das Pager-Steuerelement führt einen Bildlauf durch, wenn der Benutzer mit der Maus auf eine der Bildlaufschaltflächen klickt.<br/>                                                                                                                     |
| <span id="PGS_DRAGNDROP"></span><span id="pgs_dragndrop"></span><dl> <dt>**PGS \_ DRAGNDROP**</dt> </dl>    | Das enthaltene Fenster kann ein Drag & Drop-Ziel sein. Das Pagersteuerelement führt automatisch einen Bildlauf durch, wenn ein Element von außerhalb des Pagers über eine der Bildlaufschaltflächen gezogen wird.<br/>                                     |
| <span id="PGS_HORZ"></span><span id="pgs_horz"></span><dl> <dt>**PGS \_ HORZ**</dt> </dl>                   | Erstellt ein Pager-Steuerelement, das horizontal gescrollt werden kann. Dieser Stil und der **PGS \_ VERT-Stil** schließen sich gegenseitig aus und können nicht kombiniert werden.<br/>                                                                 |
| <span id="PGS_VERT"></span><span id="pgs_vert"></span><dl> <dt>**PGS \_ VERT**</dt> </dl>                   | Erstellt ein Pager-Steuerelement, das vertikal gescrollt werden kann. Dies ist die Standardrichtung, wenn kein Richtungsstil angegeben wird. Dieser Stil und der **PGS \_ HORZ-Stil** schließen sich gegenseitig aus und können nicht kombiniert werden.<br/> |
