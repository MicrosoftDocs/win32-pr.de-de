# <a name="udm_setrange32-message"></a>UDM \_ SETRANGE32-Nachricht

Legt den 32-Bit-Bereich eines Auf-Ab-Steuerelements fest.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Ganzzahliger Wert mit Vorzeichen, der die neue Untergrenze des Auf-Ab-Steuerelementbereichs darstellt.

</dd> <dt>

*lParam* 
</dt> <dd>

Ganzzahliger Wert mit Vorzeichen, der die neue Obergrenze des Auf-Ab-Steuerelementbereichs darstellt.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Der Rückgabewert für diese Meldung wird nicht verwendet.
