# <a name="udm_setrange32-message"></a>UDM \_ SETRANGE32 Meldung

Legt den 32-Bit-Bereich eines auf-ab-Steuer Elements fest.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>

Ganzzahliger Wert mit Vorzeichen, der die neue untere Grenze des auf-ab-Steuerelement Bereichs darstellt.

</dd> <dt>

*lParam* 
</dt> <dd>

Ganzzahliger Wert mit Vorzeichen, der die neue Obergrenze des auf-ab-Steuerelement Bereichs darstellt.

</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Der Rückgabewert für diese Nachricht wird nicht verwendet.
