# <a name="mcm_getmaxtodaywidth-message"></a>MCM \_ getMax| Day Width-Meldung

Ruft die maximale Breite der "heute"-Zeichenfolge in einem Monatskalender-Steuerelement ab. Dies schließt den Beschriftungs Text und den Datums Text ein. Sie können diese Nachricht explizit oder mit dem [**monthcal \_ getmaxredaywidth**](/windows/desktop/api/Commctrl/nf-commctrl-monthcal_getmaxtodaywidth) -Makro senden.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt die Breite der "Today"-Zeichenfolge in Pixel zurück.
