# <a name="udm_getbase-message"></a>UDM \_ GetBase-Nachricht

Ruft die aktuelle Basis 10 (d. h. entweder Basis 10 oder 16) für ein auf-ab-Steuerelement ab.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Der Rückgabewert ist der aktuelle Basiswert.
