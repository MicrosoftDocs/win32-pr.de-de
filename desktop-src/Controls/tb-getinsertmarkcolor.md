# <a name="tb_getinsertmarkcolor-message"></a>TB \_ GETINSERTMARKCOLOR-Nachricht

Ruft die Farbe ab, die zum Zeichnen der Einfügemarke für die Symbolleiste verwendet wird.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Gibt einen [**COLORREF-Wert**](/windows/desktop/gdi/colorref) zurück, der die aktuelle Einfügemarke enthält.
