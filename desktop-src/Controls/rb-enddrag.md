# <a name="rb_enddrag-message"></a>RB- \_ EndDrag-Nachricht

Beendet den Drag & Drop-Vorgang des Grund leisten-Steuer Elements. Diese Meldung bewirkt nicht, dass eine [RBN- \_ EndDrag](rbn-enddrag.md) -Benachrichtigung gesendet wird.

## <a name="parameters"></a>Parameter

<dl> <dt>

*wParam* 
</dt> <dd>Muss Null sein.</dd> <dt>

*lParam* 
</dt> <dd>Muss Null sein.</dd> </dl>

## <a name="return-value"></a>Rückgabewert

Der Rückgabewert für diese Nachricht wird nicht verwendet.
