# <a name="visual-styles-reference"></a>Referenz zu visuellen Stilen

In diesem Abschnitt werden die folgenden API-Elemente beschrieben, die mit [visuellen Stilen](themes-overview.md)verwendet werden.

### <a name="functions"></a>Functions



| Thema                                                                                  | Inhalte                                                                                                                                                                                                                           |
|----------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BeginBufferedAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedanimation)                               | Startet einen gepufferten Animationsvorgang. Die Animation besteht aus einer Kreuzblendung zwischen den Inhalten von zwei Puffern über einen angegebenen Zeitraum. <br/>                                                                     |
| [**BeginBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint)                                       | Startet einen gepufferten Farbvorgang.<br/>                                                                                                                                                                                      |
| [**BeginPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginpanningfeedback)                                   | Benachrichtigt das System, Feedback zu einem Zielfenster zu senden, das von Schwenkgesten betroffen ist. <br/>                                                                                                                               |
| [**BufferedPaintClear**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintclear)                                       | Löscht ein angegebenes Rechteck im Puffer in ARGB = {0,0,0,0} .<br/>                                                                                                                                                         |
| [**BufferedPaintInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintinit)                                         | Initialisiert gepuffertes Zeichnen für den aktuellen Thread.<br/>                                                                                                                                                                    |
| [**BufferedPaintRenderAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintrenderanimation)                   | Zeichnet den nächsten Frame einer gepufferten Farbanimation.<br/>                                                                                                                                                                    |
| [**BufferedPaintSetAlpha**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintsetalpha)                                 | Legt das Alpha auf einen angegebenen Wert in einem angegebenen Rechteck fest. Das Alpha steuert die Transparenz, die beim Mischen mit dem Puffer auf den Zielgerätekontext angewendet wird. <br/>                         |
| [**BufferedPaintStopAllAnimations**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintstopallanimations)               | Beendet alle gepufferten Animationen für das angegebene Fenster.<br/>                                                                                                                                                                     |
| [**BufferedPaintUnInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintuninit)                                     | Schließt gepuffertes Zeichnen für den aktuellen Thread. Wird einmal für jeden Aufruf von [**BufferedPaintInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintinit) aufgerufen, nachdem Aufrufe von [**BeginBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint) nicht mehr benötigt werden.<br/> |
| [**CloseThemeData**](/windows/desktop/api/Uxtheme/nf-uxtheme-closethemedata)                                               | Schließt das Designdatenhandle.<br/>                                                                                                                                                                                           |
| [**DrawThemeBackground**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemebackground)                                     | Zeichnet den Rahmen und die Füllung, die vom visuellen Stil für den angegebenen Steuerelementteil definiert werden.<br/>                                                                                                                                   |
| [**DrawThemeBackgroundEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemebackgroundex)                                 | Zeichnet das Hintergrundbild, das vom visuellen Stil für den angegebenen Steuerelementteil definiert wird.<br/>                                                                                                                                  |
| [**DrawThemeEdge**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeedge)                                                 | Zeichnet einen oder mehrere Ränder, die durch den visuellen Stil eines Rechtecks definiert sind.<br/>                                                                                                                                                     |
| [**DrawThemeIcon**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeicon)                                                 | Zeichnet ein Bild aus einer Bildliste mit dem durch den visuellen Stil definierten Symboleffekt.<br/>                                                                                                                                     |
| [**DrawThemeParentBackground**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeparentbackground)                         | Zeichnet den Teil eines übergeordneten Steuerelements, der von einem teilweise transparenten oder alphageblendeten untergeordneten Steuerelement abgedeckt wird.<br/>                                                                                                           |
| [**DrawThemeParentBackgroundEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeparentbackgroundex)                     | Wird von teilweise transparenten oder alphageblendeten untergeordneten Steuerelementen verwendet, um den Teil des übergeordneten Elements zu zeichnen, vor dem sie angezeigt werden. Sendet eine WM \_ ERASEBKGND-Nachricht gefolgt von einem WM \_ PRINTCLIENT.<br/>                             |
| [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext)                                                 | Zeichnet Text mithilfe der Farbe und Schriftart, die durch den visuellen Stil definiert ist.<br/>                                                                                                                                                        |
| [**DrawThemeTextEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetextex)                                             | Zeichnet Text mithilfe der Farbe und Schriftart, die durch den visuellen Stil definiert ist. Erweitert [**DrawThemeText,**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext) indem zusätzliche Textformatoptionen zugelassen werden.<br/>                                                             |
| [**EnableThemeDialogTexture**](/windows/desktop/api/Uxtheme/nf-uxtheme-enablethemedialogtexture)                           | Aktiviert oder deaktiviert den visuellen Stil des Hintergrunds eines Dialogfeldfensters.<br/>                                                                                                                                                   |
| [**EnableTheming**](/windows/desktop/api/Uxtheme/nf-uxtheme-enabletheming)                                                 | Aktiviert oder deaktiviert visuelle Stile für den aktuellen Benutzer in den aktuellen und späteren Sitzungen.<br/>                                                                                                                               |
| [**EndBufferedAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-endbufferedanimation)                                   | Rendert den ersten Frame eines gepufferten Animationsvorgangs und startet den Animationstimer.<br/>                                                                                                                               |
| [**EndBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-endbufferedpaint)                                           | Schließt einen gepufferten Farbvorgang ab und gibt das zugeordnete gepufferte Farbhandle frei.<br/>                                                                                                                                    |
| [**EndPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-endpanningfeedback)                                       | Beendet alle vorhandenen Animationen, die von [**BeginPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginpanningfeedback) und [**UpdatePanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-updatepanningfeedback)verarbeitet oder eingerichtet wurden. <br/>                                    |
| [**GetBufferedPaintBits**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpaintbits)                                   | Ruft einen Zeiger auf die Pufferbitmap ab, wenn der Puffer eine geräteunabhängige Bitmap (DIB) ist.<br/>                                                                                                                            |
| [**GetBufferedPaintDC**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpaintdc)                                       | Ruft den Farbdomänencontroller ab. Dies ist derselbe Wert, der von [**BeginBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint)abgerufen wird.<br/>                                                                                                                |
| [**GetBufferedPaintTargetDC**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpainttargetdc)                           | Ruft den Zieldomänencontroller ab. <br/>                                                                                                                                                                                               |
| [**GetBufferedPaintTargetRect**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpainttargetrect)                       | Ruft das durch BeginBufferedPaint angegebene Zielrechteck ab.<br/>                                                                                                                                                         |
| [**GetCurrentThemeName**](/windows/desktop/api/Uxtheme/nf-uxtheme-getcurrentthemename)                                     | Ruft den Namen des aktuellen visuellen Stils ab und ruft optional den Namen des Farbschemas und den Größennamen ab.<br/>                                                                                                           |
| [**GetThemeAppProperties**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeappproperties)                                 | Ruft die Eigenschaftenflags ab, die steuern, wie visuelle Stile in der aktuellen Anwendung angewendet werden.<br/>                                                                                                                     |
| [**GetThemeBackgroundContentRect**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundcontentrect)                 | Ruft die Größe des Inhaltsbereichs für den vom visuellen Stil definierten Hintergrund ab.<br/>                                                                                                                                  |
| [**GetThemeBackgroundExtent**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundextent)                           | Berechnet die Größe und Position des Hintergrunds, die durch den visuellen Stil definiert sind, unter Anstatt des Inhaltsbereichs.<br/>                                                                                                                |
| [**GetThemeBackgroundRegion**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundregion)                           | Berechnet den Bereich für einen regulären oder teilweise transparenten Hintergrund, der durch ein angegebenes Rechteck begrenzt ist.<br/>                                                                                                         |
| [**GetThemeBitmap**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebitmap)                                               | Ruft die Bitmap ab, die einem bestimmten Design, Teil, Zustand und einer bestimmten Eigenschaft zugeordnet ist.<br/>                                                                                                                                     |
| [**GetThemeBool**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebool)                                                   | Ruft den Wert einer **BOOL-Eigenschaft** aus dem SysMetrics-Abschnitt der Designdaten ab.<br/>                                                                                                                                   |
| [**GetThemeColor**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemecolor)                                                 | Ruft den Wert einer Farbeigenschaft ab.<br/>                                                                                                                                                                                |
| [**GetThemeDocumentationProperty**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemedocumentationproperty)                 | Ruft den Wert für eine Designeigenschaft aus dem Dokumentationsabschnitt der angegebenen Designdatei ab.<br/>                                                                                                                    |
