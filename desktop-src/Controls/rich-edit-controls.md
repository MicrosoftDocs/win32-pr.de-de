# <a name="rich-edit"></a>Umfassende Bearbeitung

Dieser Abschnitt enthält Informationen zu den Programmier Elementen, die mit Rich Edit-Steuerelementen verwendet werden. Mithilfe eines Rich-Edit-Steuer Elements können Benutzer Text eingeben, bearbeiten, Drucken und speichern. Dem Text kann eine Zeichen-und Absatz Formatierung zugewiesen werden, und es kann eingebettete Component Object Model (com)-Objekte enthalten.

Da Rich Edit-Steuerelemente fast alle Meldungen und Benachrichtigungs Codes für mehrzeilige [Bearbeitungs](edit-controls.md)Steuerelemente unterstützen, können Anwendungen, die bereits Bearbeitungs Steuerelemente verwenden, problemlos so geändert werden, dass Sie Rich Edit-Steuerelemente verwenden.
