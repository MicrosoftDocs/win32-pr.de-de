# <a name="rich-edit"></a>Umfangreiche Bearbeitung

Dieser Abschnitt enthält Informationen zu den Programmierelementen, die mit Rich-Edit-Steuerelementen verwendet werden. Mit einem Rich-Edit-Steuerelement kann der Benutzer Text eingeben, bearbeiten, drucken und speichern. Dem Text kann eine Zeichen- und Absatzformatierung zugewiesen werden, und er kann eingebettete Component Object Model (COM) enthalten.

Da umfangreiche Bearbeitungssteuerelemente fast alle Nachrichten und Benachrichtigungscodes unterstützen, die mit mehrstufigen Bearbeitungssteuerelementen verwendet [werden,](edit-controls.md)können Anwendungen, die bereits Bearbeitungssteuerelemente verwenden, problemlos geändert werden, um umfangreiche Bearbeitungssteuerelemente zu verwenden.
