# <a name="using-the-iis-ui-to-configure-the-virtual-directory"></a>Verwenden der IIS-Benutzeroberfläche zum Konfigurieren des virtuellen Verzeichnisses

Wenn Sie das virtuelle Verzeichnis für BITS-Uploads mit der IIS-Benutzeroberfläche konfigurieren möchten, wählen Sie im Kontextmenü des virtuellen Verzeichnisses **Eigenschaften** aus. Klicken Sie auf die Registerkarte **Bits-Erweiterung** , um auf die Bits-Eigenschaften Seite zuzugreifen (die BITS-Server Erweiterung ISAPI muss installiert sein). Ändern Sie die Standardeigenschaftswerte nach Bedarf, und aktivieren Sie das Verzeichnis für Uploads. Dateien können erst hochgeladen werden, wenn das Verzeichnis aktiviert ist. Ausführliche Informationen zum Festlegen der Eigenschaftswerte finden Sie in der Online Hilfe.
