# <a name="using-the-iis-ui-to-configure-the-virtual-directory"></a>Verwenden der IIS-Benutzeroberfläche zum Konfigurieren des virtuellen Verzeichnisses

Um die IIS-Benutzeroberfläche zum Konfigurieren des virtuellen Verzeichnisses für BITS-Uploads zu verwenden, wählen Sie **eigenschaften** aus dem Kontextmenü des virtuellen Verzeichnisses aus. Drücken Sie die Registerkarte **BITS-Erweiterung,** um auf die Bits-Eigenschaftenseite zuzugreifen (die ISAPI der BITS-Servererweiterung muss installiert sein). Ändern Sie die Standardeigenschaftswerte nach Bedarf, und aktivieren Sie das Verzeichnis für Uploads. Sie können Dateien erst hochladen, wenn das Verzeichnis aktiviert ist. Ausführliche Informationen zum Festlegen der Eigenschaftswerte finden Sie in der Onlinehilfe.
