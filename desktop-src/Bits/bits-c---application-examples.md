# <a name="bits-c-application-examples"></a>Bits C++ Anwendungsbeispiele

Die Background Intelligent Transfer Service (Bits)-Anwendungsbeispiele in diesem Abschnitt sind in C++ geschrieben. Sie veranschaulichen eine Reihe von Aufgaben, die mit Bits-Features ausgeführt werden können. Jede Anwendung wird in eine Reihe von Schritten aufgeteilt.

In der folgenden Tabelle sind die C++-Beispiele in diesem Abschnitt aufgeführt.



| Beispiel                                                                                                                                                            | BESCHREIBUNG                                                                                                                                                                                                                                                                                |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Beispiel: allgemeine Klassen](common-classes.md)                                                                                                                      | Dieses Beispiel enthält allgemeine Klassen, die für die Beispiele in diesem Abschnitt verwendet werden. Diese allgemeinen Klassen umfassen eine Rückruf Klasse, eine generische Fehler Klasse und eine ressourcenerwerbs-Hilfsklasse für die com-Funktion [CoInitializeEx](/windows/win32/api/combaseapi/nf-combaseapi-coinitializeex) . |
| [Beispiel: Angeben von Anmelde Informationen für die Server Authentifizierung für einen Bits-Übertragungs Auftrag](example-specifying-server-authentication-credentials-for-a-bits-transfer-job-.md) | Dieses Beispiel ruft das Autorisierungs Schema und die Anmelde Informationen vom Benutzer ab, erstellt einen Bits-Übertragungs Auftrag, legt die Anmelde Informationen für den Auftrag fest und überwacht den Status des Auftrags.                                                                                                               |
| [Beispiel: Verwenden der SSPI-Authentifizierungs Codierung mit Bits.](example-using-sspi-authentication-encoding-with-bits.md)                                                 | Dieses Beispiel veranschaulicht die Verwendung der SSPI-Authentifizierung (Security Support Provider Interface) und Bits zum erhalten der Anmelde Informationen von einem Benutzer, zum Codieren der Anmelde Informationen und zum Festlegen der codierten Anmelde Informationen für einen Bits-Übertragungs Auftrag.                                                                |
| [Beispiel: Hinzufügen eines Hilfsobjekts zu einem Bits-Übertragungs Auftrag](example-adding-a-helper-token-to-a-bits-transfer-job-.md)                                                 | In diesem Beispiel wird ein Bits-Übertragungs Auftrag erstellt, und dem Auftrag wird ein zweiter Satz von Anmelde Informationen hinzugefügt.                                                                                                                                                                                                  |
