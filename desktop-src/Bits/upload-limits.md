# <a name="upload-limits"></a>Uploadlimits

Um die Größe der Uploads einzuschränken, legen Sie die IIS-Erweiterungs Eigenschaft [**bizmaximumuploadsize**](bits-iis-extension-properties.md) fest. In IIS 7 müssen Sie möglicherweise auch das Attribut " **maxzuweisung wedcontentlength** " ändern. Der Standardwert für das IIS 7-Uploadlimit beträgt 30 Millionen Bytes. Weitere Informationen und Informationen zum Ändern der IIS-Standardeinstellungen finden Sie unter [KB942074](https://support.microsoft.com//kb/942074).
