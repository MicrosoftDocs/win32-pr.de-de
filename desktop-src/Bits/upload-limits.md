# <a name="upload-limits"></a>Hochladen Grenzen

Um die Größe von Uploads zu begrenzen, legen Sie die IIS-Erweiterungseigenschaft [**BITSMaximumUploadSize**](bits-iis-extension-properties.md) fest. In IIS 7 müssen Sie möglicherweise auch das **maxAllowedContentLength-Attribut** ändern. Standardmäßig beträgt das IIS 7-Uploadlimit 30 Millionen Bytes. Details und Informationen zum Ändern der IIS-Standardeinstellung finden Sie unter [KB942074](https://support.microsoft.com//kb/942074).
