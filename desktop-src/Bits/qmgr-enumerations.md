# <a name="qmgr-enumerations"></a>Qmgr-Enumerationen

\[Queue Manager-Enumerationen (qmgr) sind für die Verwendung in den Betriebssystemen verfügbar, die im Abschnitt Anforderungen eines Themas angegeben sind. In nachfolgenden Versionen wurde sie möglicherweise geändert oder entfernt. Verwenden Sie stattdessen die [Bits-Enumerationen](bits-enumerations.md).\]

Die Warteschlangen-Manager- [Schnittstellen](qmgr-interfaces.md) (qmgr) verwenden die folgende-Enumeration.



| Enumeration                               | Beschreibung                                                                              |
|-------------------------------------------|------------------------------------------------------------------------------------------|
| [**Groupprop**](/windows/desktop/api/Qmgr/ne-qmgr-groupprop)<br/> | Definiert die Konstanten Werte zum Abrufen und Festlegen von Gruppen Eigenschafts Werten.<br/> |
