# <a name="bits-samples-and-tools"></a>BITS-Beispiele und -Tools

Der folgende Abschnitt enthält schrittweise C++-Beispiele für Background Intelligent Transfer Service (BITS).



| `Section`                                                            | Zweck                                                                                      |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [BITS C++-Anwendungsbeispiele](bits-c---application-examples.md) | C++-Beispiele, die eine Reihe von Aufgaben veranschaulichen, die mit bits-Features ausgeführt werden können. |



 

Darüber hinaus enthält BITS die folgenden Beispiele in Windows SDK. Die Beispiele finden Sie unter *InstallDirectory* \\ Samples \\ Web. Jedes Beispiel enthält eine Readme.txt-Datei, in der erläutert wird, wie das Beispiel installiert und ausgeführt wird. Sie können das Windows SDK von [Microsoft Windows Software Development Kit](https://msdn.microsoft.com/windowsserver/bb980924.aspx)herunterladen. BITS ist im Core SDK enthalten.

> [!Note]  
> Windows Vista: Die Beispiele auf dieser Seite sind im [Microsoft Windows SDK für Windows Vista](https://www.microsoft.com/download/details.aspx?id=30998)enthalten. Das ClientCert-Beispiel ist jedoch nicht enthalten.

 



| Beispiel       | Zweck                                                                                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_BITS-IE     | Zeigt, wie Sie [](bits-interfaces.md) die BITS-Schnittstellen verwenden, um Dateien von einem Server herunterzuladen.                                                                                                                                             |
| Steuerelemente     | Zeigt, wie die [BITS-Schnittstellen](bits-interfaces.md) verwendet werden, um BITS-Optionen zu steuern und zu konfigurieren, einschließlich der Verwendung einiger Peercache-Verwaltungsmethoden.<br/> **BITS 4.0:** Das Beispiel Controls ist veraltet.<br/> |
| Downloads    | Zeigt, wie Sie [](bits-interfaces.md) die BITS-Schnittstellen verwenden, um Dateien von einem Server herunterzuladen.                                                                                                                                             |
| HTTPHeader  | Zeigt, wie die [BITS-Schnittstellen](bits-interfaces.md) zum Ändern von HTTP-Headern verwendet werden.                                                                                                                                                      |
| PeerCaching  | Zeigt, wie die [BITS-Schnittstellen](bits-interfaces.md) verwendet werden, um einem Auftrag das Herunterladen von einem Peer zu ermöglichen.<br/> **BITS 4.0:** Das PeerCaching-Beispiel ist veraltet.<br/>                                                           |
| Uploads      | Zeigt, wie die [BITS-Schnittstellen](bits-interfaces.md) zum Hochladen einer Datei verwendet werden.                                                                                                                                                            |
| UploadSample | Zeigt, wie Sie [](bits-interfaces.md) die BITS-Schnittstellen verwenden, um eine Datei auf einen Server hochzuladen und eine Antwort zu erhalten.                                                                                                                            |



 

BITS stellt das folgende Tool bereit.



| Tool                            | Zweck                                                                         |
|---------------------------------|---------------------------------------------------------------------------------|
| [BITSAdmin](bitsadmin-tool.md) | Befehlszeilentool zum Erstellen von Download- oder Uploadaufträgen und Überwachen des Fortschritts. |
