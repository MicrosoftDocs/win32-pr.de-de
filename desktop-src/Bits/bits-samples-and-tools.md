# <a name="bits-samples-and-tools"></a>Bits-Beispiele und-Tools

Der folgende Abschnitt enthält schrittweise C++ Beispiele für Background Intelligent Transfer Service (Bits).



| `Section`                                                            | Zweck                                                                                      |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [Bits C++ Anwendungsbeispiele](bits-c---application-examples.md) | C++ Beispiele, die eine Reihe von Aufgaben veranschaulichen, die mithilfe von Bits-Features abgeschlossen werden können. |



 

Darüber hinaus enthält Bits die folgenden Beispiele in Windows SDK. Die Beispiele befinden sich unter *installverzeichnis* \\ Samples \\ Web. Jedes Beispiel enthält eine Readme.txt-Datei, die erläutert, wie das Beispiel installiert und ausgeführt wird. Sie können die Windows SDK von [Microsoft Windows Software Development Kit](https://msdn.microsoft.com/windowsserver/bb980924.aspx)herunterladen. Bits ist im Core SDK enthalten.

> [!Note]  
> Windows Vista: die Beispiele auf dieser Seite sind im [Microsoft Windows SDK für Windows Vista](https://www.microsoft.com/download/details.aspx?id=30998)enthalten. Das clientcert-Beispiel ist jedoch nicht enthalten.

 



| Beispiel       | Zweck                                                                                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Bits \_ IE     | Zeigt, wie die Bits- [Schnittstellen](bits-interfaces.md) verwendet werden, um Dateien von einem Server herunterzuladen.                                                                                                                                             |
| Steuerelemente     | Zeigt, wie die Bits- [Schnittstellen](bits-interfaces.md) zum Steuern und Konfigurieren von Bits-Optionen verwendet werden, einschließlich der Verwendung einiger Peer Cache-Verwaltungsmethoden.<br/> **Bits 4,0:** Das Steuerelement Beispiel ist veraltet.<br/> |
| Downloads    | Zeigt, wie die Bits- [Schnittstellen](bits-interfaces.md) verwendet werden, um Dateien von einem Server herunterzuladen.                                                                                                                                             |
| Httpheaders  | Zeigt, wie die Bits- [Schnittstellen](bits-interfaces.md) zum Ändern von HTTP-Headern verwendet werden.                                                                                                                                                      |
| -Peer Caching  | Zeigt, wie die Bits- [Schnittstellen](bits-interfaces.md) verwendet werden, um einen Auftrag zum Herunterladen von einem Peer zu aktivieren.<br/> **Bits 4,0:** Das Beispiel "Peer Caching" ist veraltet.<br/>                                                           |
| Paketen      | Zeigt, wie die Bits- [Schnittstellen](bits-interfaces.md) verwendet werden, um eine Datei hochzuladen.                                                                                                                                                            |
| Uploadsample | Zeigt, wie die Bits- [Schnittstellen](bits-interfaces.md) verwendet werden, um eine Datei auf einen Server hochzuladen und eine Antwort zu erhalten.                                                                                                                            |



 

Bits bietet das folgende Tool.



| Tool                            | Zweck                                                                         |
|---------------------------------|---------------------------------------------------------------------------------|
| [BITSAdmin](bitsadmin-tool.md) | Befehlszeilen Tool zum Erstellen von Aufträgen zum Herunterladen oder Hochladen von Aufträgen und zum Überwachen des Fortschritts. |
