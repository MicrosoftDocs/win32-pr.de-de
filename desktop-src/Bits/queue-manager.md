# <a name="queue-manager"></a>Warteschlangen-Manager

\[Queue Manager (qmgr) steht für die Verwendung in den Betriebssystemen zur Verfügung, die im Abschnitt Anforderungen eines Themas angegeben sind. Es kann in nachfolgenden Versionen geändert oder entfernt werden. Verwenden Sie stattdessen die [Background Intelligent Transfer Service (Bits)](background-intelligent-transfer-service-portal.md).\]

Der Abschnitt Queue Manager (qmgr) Reference enthält Informationen zu den folgenden API-Elementen:

-   [Qmgr-Enumerationen](qmgr-enumerations.md)
-   [Qmgr-Schnittstellen](qmgr-interfaces.md)
-   [Qmgr-Strukturen](qmgr-structures.md)
