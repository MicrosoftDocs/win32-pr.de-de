# <a name="qmgr-structures"></a>Qmgr-Strukturen

\[Queue Manager-Strukturen (qmgr) können in den Betriebssystemen verwendet werden, die im Abschnitt Anforderungen eines Themas angegeben sind. In nachfolgenden Versionen wurde sie möglicherweise geändert oder entfernt. Verwenden Sie stattdessen die [Bits-Strukturen und Unions](bits-c---structures-and-unions.md).\]

Die Warteschlangen-Manager- [Schnittstellen](qmgr-interfaces.md) (qmgr) verwenden die folgenden Strukturen.



| Struktur                                     | Zweck                                                                   |
|-----------------------------------------------|---------------------------------------------------------------------------|
| [**"Flesetinfo"**](/windows/desktop/api/Qmgr/ns-qmgr-filesetinfo)<br/> | Gibt den Remote Namen und den lokalen Namen der herunter zuladenden Datei an.<br/> |
| **Hintergrund \_ Kopier \_ Zeiten**<br/>        | Wird nicht unterstützt.<br/>                                                 |
