# <a name="bits-enumerations"></a>Bits-Enumerationen

Die [Background Intelligent Transfer Service (Bits)-Schnittstellen](bits-interfaces.md) verwenden die folgenden Enumerationen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [**BG_AUTH_SCHEME**](/windows/win32/api/bits1_5/ne-bits1_5-bg_auth_scheme) | Definiert Konstanten, die das zu verwendende Authentifizierungsschema angeben, wenn ein Proxy oder Server eine Benutzerauthentifizierung anfordert. |
| [**BG_AUTH_TARGET**](/windows/win32/api/bits1_5/ne-bits1_5-bg_auth_target) | Definiert Konstanten, die angeben, ob die Anmelde Informationen für Proxy-oder Server Benutzer-Authentifizierungsanforderungen verwendet werden. |
| [**BG_CERT_STORE_LOCATION**](/windows/win32/api/bits2_5/ne-bits2_5-bg_cert_store_location) | Definiert Konstanten, die den Speicherort des Zertifikat Speicher angeben. |
| [**BG_ERROR_CONTEXT**](/windows/win32/api/bits/ne-bits-bg_error_context) | Definiert Konstanten, die den Kontext angeben, in dem der Fehler aufgetreten ist. |
| [**BG_JOB_PRIORITY**](/windows/win32/api/bits/ne-bits-bg_job_priority) | Definiert Konstanten, die die Prioritätsstufe eines Auftrags angeben.  |
| [**BG_JOB_PROXY_USAGE**](/windows/win32/api/bits/ne-bits-bg_job_proxy_usage) | Definiert Konstanten, die angeben, welcher Proxy für Dateiübertragungen verwendet werden soll. Sie können für jeden Auftrag verschiedene Proxy Einstellungen definieren. |
| [**BG_JOB_STATE**](/windows/win32/api/bits/ne-bits-bg_job_state) | Definiert Konstanten, die die verschiedenen Zustände eines Auftrags angeben. |
| [**BG_JOB_TYPE**](/windows/win32/api/bits/ne-bits-bg_job_type) | Definiert Konstanten, die den Typ des Übertragungs Auftrags angeben, z. b. den Download. |
| [**BITS_COST_STATE**](./bits-cost-state.md) | Definiert Konstanten, die den Bits-Kosten Zustand angeben. |
| [**BITS_FILE_PROPERTY_ID**](/windows/win32/api/bits5_0/ne-bits5_0-bits_file_property_id) | Definiert Konstanten, die ID-Werte angeben, die den Eigenschaften der Hintergrund Kopier Datei entsprechen. Weitere Informationen finden Sie unter [IBackgroundCopyFile5:: GetProperty](../delivery_optimization/ibackgroundcopyfile5-getproperty.md) und [SetProperty](../delivery_optimization/ibackgroundcopyfile5-setproperty.md). |
| [**BITS_JOB_PROPERTY_ID**](/windows/win32/api/bits5_0/ne-bits5_0-bits_job_property_id) | Definiert Konstanten, die die ID der Eigenschaft für den BITS-Auftrag angeben. |
| [**BITS_JOB_TRANSFER_POLICY**](/windows/win32/api/bits5_0/ne-bits5_0-bits_job_transfer_policy) | Definiert Konstanten, die ID-Werte angeben, die den Bits-Eigenschaften entsprechen. |
