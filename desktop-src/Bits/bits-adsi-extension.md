# <a name="bits-adsi-extension"></a>Bits ADSI-Erweiterung

Der BITS-Server stellt die folgenden ADSI-Erweiterungs Schnittstellen zum Aktivieren von BITS-Uploads bereit.



| Schnittstelle                                                        | BESCHREIBUNG                                                                                                                                    |
|------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Ibitsextensionsetup**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetup)               | Verwenden Sie, um Bits-Uploads in ein virtuelles Verzeichnis zu aktivieren und zu deaktivieren.                                                                                 |
| [**Ibitsextensionsetupfactory**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetupfactory) | Verwenden Sie, um einen Zeiger auf eine [**ibitsextensionsetup**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetup) -Schnittstelle in einem Setup Programm abzurufen, mit dem der BITS-Server installiert wird. |
