# <a name="encoding-windows-web-services"></a>Codierung (Windows Webdienste)

Die Codierung ist eine Transformation zwischen XML und einem linearen Kabelformat. Die Decodierung ist die umgekehrte Transformation.


Eine Codierung wird mithilfe einer [**WS \_ ENCODING-Konstante**](/windows/desktop/api/WebServices/ne-webservices-ws_encoding) identifiziert.
