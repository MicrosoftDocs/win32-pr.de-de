# <a name="encoding-windows-web-services"></a>Codierung (Windows-Webdienste)

Bei der Codierung handelt es sich um eine Transformation zwischen XML und einem linearen Wire-Format. Decodierung ist die umgekehrte Transformation.


Eine Codierung wird mithilfe einer [**WS- \_ Codierungs**](/windows/desktop/api/WebServices/ne-webservices-ws_encoding) Konstante identifiziert.
