# <a name="prefetch-debug-apis-for-windows-store-apps"></a>Vorabrufen von Debug-APIs für Windows Store-Apps

Die folgenden APIs werden zum Debuggen und Testen des Vorabrufverhaltens von Inhalten in einer Windows Store-App verwendet, die die [**ContentPrefetcher-Klasse**](/uwp/api/Windows.Networking.BackgroundTransfer.ContentPrefetcher) implementiert.



| Schnittstelle                                                              | BESCHREIBUNG                                                                                                                                                                      |
|------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IContentPrefetcherTaskTrigger**](/windows/desktop/api/IContentPrefetcherTaskTrigger/nn-icontentprefetchertasktrigger-icontentprefetchertasktrigger) | Verwenden Sie diese Schnittstelle, um zu überprüfen, ob ein installiertes Windows Store App-Paket für diese Hintergrundaufgabe registriert ist, und lösen Sie die Inhaltsvorrufvorgänge manuell aus.<br/> |
