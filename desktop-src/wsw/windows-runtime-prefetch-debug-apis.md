# <a name="prefetch-debug-apis-for-windows-store-apps"></a>Vorabruf von Debug-APIs für Windows Store-Apps

Die folgenden APIs werden verwendet, um das Vorabruf Verhalten von Inhalten in einer Windows Store-App zu Debuggen und zu testen, die die [**contentprefetcher**](/uwp/api/Windows.Networking.BackgroundTransfer.ContentPrefetcher) -Klasse implementiert.



| Schnittstelle                                                              | BESCHREIBUNG                                                                                                                                                                      |
|------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Icontentprefetchertask-Auslösers**](/windows/desktop/api/IContentPrefetcherTaskTrigger/nn-icontentprefetchertasktrigger-icontentprefetchertasktrigger) | Verwenden Sie diese Schnittstelle, um zu überprüfen, ob ein installiertes Windows Store-App-Paket für diese Hintergrundaufgabe registriert ist, und führen Sie die Vorabruf Vorgänge<br/> |
