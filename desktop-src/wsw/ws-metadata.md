# <a name="ws_metadata"></a>\_WS-METADATEN

Ein nicht transparenter Typ, der verwendet wird, um auf einen Satz von [Metadatendokumenten](metadata-import.md) zu verweisen.


```C++
typedef struct _WS_METADATA WS_METADATA;
```



## <a name="remarks"></a>Hinweise

Dieses Objekt ist nicht threadsicher. Weitere Informationen finden Sie unter [Threadsicherheit.](thread-safety.md)
