# <a name="windows-web-services-functions"></a>Windows-Webdienst Funktionen

Die Funktionen der Windows-Webdienste unterstützen das Methoden basierte Programmiermodell der Windows-Webdienste-API. Die [Dienst Host](service-host.md) -Laufzeitumgebung der [Dienstmodell](service-model-layer-overview.md) -API-Ebene stellt das serverseitige Programmiermodell bereit. Eine serverseitige Anwendung wird für die Verarbeitung von Client Anforderungen nach Bedarf geöffnet, geschlossen und freigibt eine Dienst Host Ressource frei. Der Dienst Proxy stellt das Client seitige API-Programmiermodell bereit. Client seitige Anwendungen können Nachrichten über einen Kanal über den Dienst Proxy als Methodenaufrufe senden und empfangen.
