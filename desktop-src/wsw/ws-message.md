# <a name="ws_message"></a>WS- \_ Nachricht

Der nicht transparente Typ, der zum referenzieren eines [Nachrichten](message.md) Objekts verwendet wird.


```C++
typedef struct _WS_MESSAGE WS_MESSAGE;
```



## <a name="remarks"></a>Bemerkungen

Dieses Objekt ist nicht Thread sicher. Weitere Informationen finden Sie unter [Thread Sicherheit](thread-safety.md).
