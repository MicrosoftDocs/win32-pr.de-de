# <a name="endpoint-identity"></a>Endpunktidentität

Die in diesem Abschnitt definierten Typen werden verwendet, um die Sicherheitsidentität einer Endpunkt Adresse zu definieren.

Die folgenden API-Elemente sind Teil der Endpunkt Indentity:



| Enumeration                                                       | Beschreibung                                                                                                                   |
|-------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**Identitäts Typ des WS- \_ Endpunkts \_ \_**](/windows/desktop/api/WebServices/ne-webservices-ws_endpoint_identity_type) | Der Typ der Endpunkt Identität, der als Selektor für die Untertypen der [**WS- \_ Endpunkt \_ Identität**](/windows/desktop/api/WebServices/ns-webservices-ws_endpoint_identity)verwendet wird. |



 



| Struktur                                                               | BESCHREIBUNG                                                                                  |
|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**Identität des WS \_ CERT- \_ Endpunkts \_**](/windows/desktop/api/WebServices/ns-webservices-ws_cert_endpoint_identity)       | Der Typ für die Zertifikat Endpunkt Identität.                                                 |
| [**Identität des WS \_ DNS- \_ Endpunkts \_**](/windows/desktop/api/WebServices/ns-webservices-ws_dns_endpoint_identity)         | Der Typ zum Angeben einer Endpunkt Identität, die durch einen DNS-Namen dargestellt wird.                      |
| [**WS- \_ Endpunkt \_ Identität**](/windows/desktop/api/WebServices/ns-webservices-ws_endpoint_identity)                  | Der Basistyp für alle Endpunkt Identitäten.                                                   |
| [**Identität des WS \_ RSA- \_ Endpunkts \_**](/windows/desktop/api/WebServices/ns-webservices-ws_rsa_endpoint_identity)         | Geben Sie für die RSA-Endpunkt Identität ein.                                                              |
| [**WS- \_ SPN- \_ Endpunkt \_ Identität**](/windows/desktop/api/WebServices/ns-webservices-ws_spn_endpoint_identity)         | Der Typ zum Angeben einer Endpunkt Identität, die durch einen SPN (Dienst Prinzipal Name) dargestellt wird. |
| [**Identität des unbekannten WS- \_ \_ Endpunkts \_**](/windows/desktop/api/WebServices/ns-webservices-ws_unknown_endpoint_identity) | Der Typ für eine unbekannte Endpunkt Identität.                                                   |
| [**WS- \_ UPN- \_ Endpunkt \_ Identität**](/windows/desktop/api/WebServices/ns-webservices-ws_upn_endpoint_identity)         | Der Typ zum Angeben einer Endpunkt Identität, die durch einen UPN (Benutzer Prinzipal Name) dargestellt wird.     |
