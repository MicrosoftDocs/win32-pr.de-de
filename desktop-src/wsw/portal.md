# <a name="windows-web-services-api"></a>Windows-Webdienst-API

## <a name="purpose"></a>Zweck

WWSAPI ist eine Native Code-Implementierung von SOAP, die kernige Netzwerkkommunikationsfunktionen bereitstellt, indem eine Reihe von WS- und unterstützt \* wird. NET: \* Familie von Protokollen. WWSAPI ist für die Verwendung durch Komponenten/Anwendungen konzipiert, die in eine der folgenden Kategorien fallen:

-   Nativer Codeauftrag
-   Minimale Abhängigkeiten erforderlich
-   Minimale Startzeit erforderlich
-   Umgebungen mit eingeschränktem Arbeitsspeicher

## <a name="developer-audience"></a>Entwicklergruppe

Die Windows Web Services-API (WWSAPI) bietet eine C/C++-Api (Application Programming Interface) zum Erstellen soapbasierter Webdienste und Clients für diese. Für die Verwendung dieser API ist ein grundlegendes Verständnis der damit verbundenen Webdienste und Protokolle wichtig.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Die Windows Web Services-API (WWSAPI) ist eine Betriebssystemkomponente von Windows 7 und Windows Server 2008 R2 oder höher von Microsoft Windows.
