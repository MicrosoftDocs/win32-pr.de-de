# <a name="do-enumerations"></a>DO-Enumerationen

Die [Übermittlungsoptimierung (DO)-Schnittstellen](do-interfaces.md) verwenden die folgenden Enumerationen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [**BG_ERROR_CONTEXT**](bg-error-context.md) | Die **BG_ERROR_CONTEXT-Enumeration** definiert die konstanten Werte, die den Kontext angeben, in dem der Fehler aufgetreten ist. |
| [**BG_JOB_PRIORITY**](bg-job-priority-.md) | Die **BG_JOB_PRIORITY-Enumeration** definiert die konstanten Werte, die die Prioritätsebene eines Auftrags angeben.  |
| [**BG_JOB_STATE**](bg-job-state-.md) | Die **BG_JOB_STATE-Enumeration** definiert konstante Werte für die verschiedenen Status eines Auftrags. |
| [**BG_JOB_TYPE**](bg-job-type.md) | Die **BG_JOB_TYPE-Enumeration** definiert konstante Werte, die den Typ des Übertragungsauftrags angeben, z. B. Download. |
| [**BITS_FILE_PROPERTY_ID**](bits-file-property-id-.md) | Die **BITS_FILE_PROPERTY_ID-Enumeration** gibt Werte an, die ID-Werte definieren, die **den BackgroundCopyFile-Eigenschaften** entsprechen. |
| [**BITS_JOB_PROPERTY_ID**](bits-job-property-id.md) | Die **BITS_JOB_PROPERTY_ID-Enumeration** gibt die ID der Eigenschaft für den DO-Auftrag an. Diese Enumeration wird in der [**BITS_JOB_PROPERTY_VALUE**](bits-job-property-value-.md) Union verwendet, um den Typ des in der Union enthaltenen Werts zu bestimmen. |
| [**BITS_JOB_TRANSFER_POLICY**](bits-job-transfer-policy-.md) | Die **BITS_JOB_TRANSFER_POLICY-Enumeration** definiert ID-Werte, die DO-Eigenschaften entsprechen. |
| [**DeliveryOptimizationFileProperty**](deliveryoptimizationfileproperty.md) | Gibt die ID einer optionalen Eigenschaft für die DO-Datei an. |
| [**DODownloadCostPolicy**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadcostpolicy.md) | Gibt die ID der Kostenrichtlinienoptionen an, die der **eigenschaft DODownloadProperty_CostPolicy** zugeordnet sind. |
| [**DODownloadProperty**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadproperty.md) | Gibt die ID der Eigenschaften für den DO-Downloadvorgang an. |
| [**DODownloadPropertyEx**](./dodownloadinternal/ne-dodownloadinternal-dodownloadpropertyex.md) | Gibt die ID der erweiterten Eigenschaften für den DO-Downloadvorgang an. |
| [**DODownloadState**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadstate.md) | Gibt die ID des aktuellen Downloadzustands an, der Teil der **DO_DOWNLOAD_STATUS-Struktur** ist. |
| [**DownloadMode**](downloadmode.md) | Definiert die verschiedenen Downloadmodi, die Übermittlungsoptimierung verwendet.  |
| [**SwarmStatus**](swarmstatus.md) | Definiert den Status einer Datei innerhalb des Übermittlungsoptimierungsclients. |
