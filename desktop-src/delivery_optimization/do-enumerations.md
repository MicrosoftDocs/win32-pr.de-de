# <a name="do-enumerations"></a>Enumerationen ausführen

Die Schnittstellen für die Übermittlungs [Optimierung (Do)](do-interfaces.md) verwenden die folgenden Enumerationen.

## <a name="in-this-section"></a>In diesem Abschnitt

| Thema | BESCHREIBUNG |
|-|-|
| [**BG_ERROR_CONTEXT**](bg-error-context.md) | Die **BG_ERROR_CONTEXT** -Enumeration definiert die Konstanten Werte, die den Kontext angeben, in dem der Fehler aufgetreten ist. |
| [**BG_JOB_PRIORITY**](bg-job-priority-.md) | Die **BG_JOB_PRIORITY** -Enumeration definiert die Konstanten Werte, die die Prioritätsstufe eines Auftrags angeben.  |
| [**BG_JOB_STATE**](bg-job-state-.md) | Die **BG_JOB_STATE** -Enumeration definiert konstante Werte für die verschiedenen Zustände eines Auftrags. |
| [**BG_JOB_TYPE**](bg-job-type.md) | Die **BG_JOB_TYPE** -Enumeration definiert konstante Werte, die den Typ des Übertragungs Auftrags angeben, z. b. den Download. |
| [**BITS_FILE_PROPERTY_ID**](bits-file-property-id-.md) | Die **BITS_FILE_PROPERTY_ID** -Enumeration gibt Werte an, die ID-Werte definieren, die den **backgroundcopyfile** -Eigenschaften entsprechen. |
| [**BITS_JOB_PROPERTY_ID**](bits-job-property-id.md) | Die **BITS_JOB_PROPERTY_ID** -Enumeration gibt die ID der Eigenschaft für den Do-Auftrag an. Diese Enumeration wird in der [**BITS_JOB_PROPERTY_VALUE**](bits-job-property-value-.md) Union verwendet, um den Typ des Werts zu bestimmen, der in der Union enthalten ist. |
| [**BITS_JOB_TRANSFER_POLICY**](bits-job-transfer-policy-.md) | Die **BITS_JOB_TRANSFER_POLICY** -Enumeration definiert ID-Werte, die den Do-Eigenschaften entsprechen. |
| [**Deliveryoptimizationfileproperty**](deliveryoptimizationfileproperty.md) | Gibt die ID einer optionalen Eigenschaft für die do-Datei an. |
| [**Dodownloadcostpolicy**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadcostpolicy.md) | Gibt die ID der Kosten Richtlinien Optionen an, die mit der **DODownloadProperty_CostPolicy** Eigenschaft verknüpft sind. |
| [**Dodownloadproperty**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadproperty.md) | Gibt die ID der Eigenschaften für den Downloadvorgang an. |
| [**Dodownloadpropertyex**](./dodownloadinternal/ne-dodownloadinternal-dodownloadpropertyex.md) | Gibt die ID der erweiterten Eigenschaften für den Downloadvorgang an. |
| [**Dodownloadstate**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadstate.md) | Gibt die ID des aktuellen Download Status an, der Teil der **DO_DOWNLOAD_STATUS** Struktur ist. |
| [**Download Mode**](downloadmode.md) | Definiert die verschiedenen Download Modi, die von der Übermittlungs Optimierung verwendet werden.  |
| [**Swarmstatus**](swarmstatus.md) | Definiert den Status einer Datei innerhalb des Bereitstellungs Optimierungs Clients. |
