# <a name="delivery-optimization-do"></a>Übermittlungs Optimierung (Do)

Bei der Übermittlungs Optimierung handelt es sich um einen cloudverwalteten Peer-to-Peer-Client und einen Download Dienst, der Updates für Netzwerkgeräte einer Organisation bereitstellt. Die Übermittlungs Optimierung ermöglicht Geräten das Herunterladen von Updates aus alternativen Quellen (z. b. anderen Peers im Netzwerk), zusätzlich zu Microsoft-Servern. Die Übermittlungs Optimierung kombiniert partielle Bits von lokalen Geräten mit partiellen Bits von Microsoft-Servern zum Aktualisieren von Geräten in der Netzwerkumgebung. Das erwartete Ergebnis ist die geringere Bandbreitennutzung und ein schnellerer Aktualisierungsprozess.

Weitere Informationen finden Sie unter Konfigurieren der Übermittlungs [Optimierung für Windows 10-Updates](/windows/deployment/update/waas-delivery-optimization) .
