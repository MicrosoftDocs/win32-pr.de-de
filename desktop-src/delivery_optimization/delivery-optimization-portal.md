# <a name="delivery-optimization-do"></a>Übermittlungsoptimierung (DO)

Übermittlungsoptimierung ist ein in der Cloud verwalteter Peer-zu-Peer-Client und ein Downloaddienst, der Updates für die netzwerkgesteuerten Geräte einer Organisation übermittelt. Übermittlungsoptimierung ermöglicht es Geräten, Updates von alternativen Quellen (z. B. anderen Peers im Netzwerk) zusätzlich zu Microsoft-Servern herunterzuladen. Übermittlungsoptimierung werden Teilbits von lokalen Geräten mit Teilbits von Microsoft-Servern kombiniert, um Geräte in der Netzwerkumgebung zu aktualisieren. Das erwartete Ergebnis ist eine geringere Bandbreitenauslastung und ein schnellerer Aktualisierungsprozess.

Weitere Informationen finden Sie unter [Konfigurieren von Übermittlungsoptimierung für Windows 10 Updates.](/windows/deployment/update/waas-delivery-optimization)
