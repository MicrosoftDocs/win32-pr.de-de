# <a name="vml-callout-element"></a>VML-Legenden Element

In diesem Thema wird VML beschrieben, eine Funktion, die ab Windows Internet Explorer 9 veraltet ist. Webseiten und Anwendungen, die auf VML basieren, sollten zu SVG oder anderen allgemein unterstützten Standards migriert werden.

> [!Note]  
> Ab Dezember 2011 wurde dieses Thema archiviert. Daher wird er nicht mehr aktiv verwaltet. Weitere Informationen finden Sie unter [archivierte Inhalte](/previous-versions/windows/internet-explorer/ie-developer/). Informationen, Empfehlungen und Anleitungen zur aktuellen Version von Windows Internet Explorer finden Sie im [Internet Explorer Developer Center](https://msdn.microsoft.com/ie/).

 

Definiert eine Legende für eine Form.

Die folgenden Attribute ändern eine Legende.



| Attribut                                                        | BESCHREIBUNG                                                                             |
|------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [Accentbar](msdn-online-vml-accentbar-attribute.md)             | Bestimmt, ob eine Akzent Leiste mit der Legende verwendet wird.                         |
| [Ultra](angle-attribute--callout--vml.md)                       | Definiert den Winkel, der von der Legende in Bezug auf das umgebende Feld der Form festgelegt wird. |
| [Entfernung](msdn-online-vml-distance-attribute.md)               | Definiert den Ablage Abstand einer Legende.                                                 |
| [Dropdown](msdn-online-vml-drop-attribute.md)                       | Bestimmt, wo das Ablegen einer Legende platziert wird.                                  |
| [Dropauto](msdn-online-vml-dropauto-attribute.md)               | Bestimmt, ob die Legende einen automatischen Löschvorgang hat.                             |
| [Antrags](ext-attribute--callout--vml.md)                           | Definiert, wie eine Legende verarbeitet wird.                                                     |
| [E](msdn-online-vml-gap-attribute.md)                         | Definiert die Entfernung der Legende aus dem umgebenden Rechteck.                        |
| [Länge](msdn-online-vml-length-attribute.md)                   | Definiert die Länge der Legende.                                                      |
| [Längenangabe](msdn-online-vml-lengthspecified-attribute.md) | Bestimmt, ob das **length** -Attribut für die Legende verwendet wird.               |
| [Minusx](msdn-online-vml-minusx-attribute.md)                   | Bestimmt, ob die Legende entlang der x-Achse gedreht wird.                              |
| [Minusy](msdn-online-vml-minusy-attribute.md)                   | Bestimmt, ob die Legende entlang der y-Achse gedreht wird.                              |
| [Ein](on-attribute--callout--vml.md)                             | Bestimmt, ob eine Form eine Legende ist.                                                |
| [Textborder](msdn-online-vml-textborder-attribute.md)           | Bestimmt, ob eine Legende einen Textrahmen hat.                                   |
| [Type](type-attribute--callout--vml.md)                         | Definiert den Typ der Legende.                                                            |



 

**Anmerkungen**

Dieses Element muss innerhalb eines [Shape](shape-element--vml.md) -Elements definiert werden.

Das **Legenden** Element ist eine Microsoft Office Erweiterung von VML.
