# <a name="vml-callout-element"></a>VML Callout-Element

In diesem Thema wird VML beschrieben, ein Feature, das ab Version 9 Windows Internet Explorer ist. Webseiten und Anwendungen, die auf VML basieren, sollten zu SVG oder anderen weit verbreiteten Standards migriert werden.

> [!Note]  
> Seit Dezember 2011 wurde dieses Thema archiviert. Daher wird sie nicht mehr aktiv verwaltet. Weitere Informationen finden Sie unter [Archivierter Inhalt.](/previous-versions/windows/internet-explorer/ie-developer/) Informationen, Empfehlungen und Anleitungen zur aktuellen Version von Windows Internet Explorer finden Sie im [Internet Explorer Developer Center.](https://msdn.microsoft.com/ie/)

 

Definiert eine Callout für eine Form.

Die folgenden Attribute ändern eine -Rückrufe.



| attribute                                                        | BESCHREIBUNG                                                                             |
|------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [AccentBar](msdn-online-vml-accentbar-attribute.md)             | Bestimmt, ob eine Akzentleiste mit der Callout verwendet wird.                         |
| [Winkel](angle-attribute--callout--vml.md)                       | Definiert den Winkel, den die Callout in Bezug auf den Begrenzungsfeld der Form macht. |
| [Entfernung](msdn-online-vml-distance-attribute.md)               | Definiert den Absturzabstand einer Rückrufliste.                                                 |
| [Drop](msdn-online-vml-drop-attribute.md)                       | Bestimmt, wo das Ablegen einer Rückrufliste platziert wird.                                  |
| [DropAuto](msdn-online-vml-dropauto-attribute.md)               | Bestimmt, ob die Rückrufliste automatisch verdringt wird.                             |
| [Extern](ext-attribute--callout--vml.md)                           | Definiert, wie eine Rückrufe verarbeitet werden.                                                     |
| [Lücke](msdn-online-vml-gap-attribute.md)                         | Definiert den Abstand zwischen der Callout und dem umgebundenen Rechteck.                        |
| [Länge](msdn-online-vml-length-attribute.md)                   | Definiert die Länge der Rückrufe.                                                      |
| [LengthSpecified](msdn-online-vml-lengthspecified-attribute.md) | Bestimmt, ob **das Length-Attribut** für die Rückrufe verwendet wird.               |
| [MinusX](msdn-online-vml-minusx-attribute.md)                   | Bestimmt, ob die Callout entlang der X-Achse gekippt wird.                              |
| [MinusY](msdn-online-vml-minusy-attribute.md)                   | Bestimmt, ob die Callout entlang der y-Achse gekippt wird.                              |
| [Ein](on-attribute--callout--vml.md)                             | Bestimmt, ob eine Form eine Callout ist.                                                |
| [TextBorder](msdn-online-vml-textborder-attribute.md)           | Bestimmt, ob eine Callout einen Text-Rahmen hat.                                   |
| [Typ](type-attribute--callout--vml.md)                         | Definiert den Typ der Rückrufe.                                                            |



 

**Anmerkungen**

Dieses Element muss innerhalb eines [Shape-Elements definiert](shape-element--vml.md) werden.

Das **Callout-Element** ist eine Microsoft Office Erweiterung für VML.
