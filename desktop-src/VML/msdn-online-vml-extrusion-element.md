# <a name="vml-extrusion-element"></a>VML Extrusion-Element

In diesem Thema wird VML beschrieben, eine Funktion, die ab Windows Internet Explorer 9 veraltet ist. Webseiten und Anwendungen, die auf VML basieren, sollten zu SVG oder anderen allgemein unterstützten Standards migriert werden.

> [!Note]  
> Ab Dezember 2011 wurde dieses Thema archiviert. Daher wird er nicht mehr aktiv verwaltet. Weitere Informationen finden Sie unter [archivierte Inhalte](/previous-versions/windows/internet-explorer/ie-developer/). Informationen, Empfehlungen und Anleitungen zur aktuellen Version von Windows Internet Explorer finden Sie im [Internet Explorer Developer Center](https://msdn.microsoft.com/ie/).

 

Definiert eine Extrusion für eine Form.

Die folgenden Attribute ändern eine-Extrusion.



| Attribut                                                              | BESCHREIBUNG                                                                                             |
|------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [Autorotationcenter](msdn-online-vml-autorotationcenter-attribute.md) | Bestimmt, ob der Mittelpunkt der Drehung das geometrische Zentrum der-Extrusion ist.                |
| [Backtiefe](msdn-online-vml-backdepth-attribute.md)                   | Definiert die Menge der abwärts Extrusion.                                                               |
| [Helligkeit](msdn-online-vml-brightness-attribute.md)                 | Gibt die Helligkeit einer Szene an.                                                          |
| [Farbe](color-attribute--extrusion--vml.md)                           | Definiert die Farbe der extrusiongesichter.                                                               |
| [ColorMode](msdn-online-vml-colormode-attribute.md)                   | Bestimmt den Modus der Extrusionfarbe.                                                                 |
| [Diffusity](msdn-online-vml-diffusity-attribute.md)                   | Definiert die Menge der Verbreitung von reflektiertem Licht aus einer extrudierten Form.                              |
| [Edge](msdn-online-vml-edge-attribute.md)                             | Definiert die sichtbare Abschrägung der extrusionkanten.                                                      |
| [Antrags](ext-attribute--extrusion--vml.md)                               | Definiert das standardmäßige Extrusionsverhalten für grafische Editoren.                                           |
| [Facet](msdn-online-vml-facet-attribute.md)                           | Definiert die Anzahl von Facetten, die zum Beschreiben von Kurven Oberflächen einer-Extrusion verwendet werden.                          |
| [Foretiefe](msdn-online-vml-foredepth-attribute.md)                   | Definiert den Umfang der vorwärts-Extrusion.                                                                |
| [Lightface](msdn-online-vml-lightface-attribute.md)                   | Bestimmt, ob die Vorderseite der-Extrusion auf Änderungen in der Beleuchtung antwortet.             |
| [Lightharsh](msdn-online-vml-lightharsh-attribute.md)                 | Bestimmt, ob die primäre Lichtquelle hart ist.                                              |
| [LightHarsh2](msdn-online-vml-lightharsh2-attribute.md)               | Bestimmt, ob die sekundäre Lichtquelle hart ist.                                            |
| [Lightlevel](msdn-online-vml-lightlevel-attribute.md)                 | Definiert die Intensität der primären Lichtquelle für die Szene.                                        |
| [LightLevel2](msdn-online-vml-lightlevel2-attribute.md)               | Definiert die Intensität der sekundären Lichtquelle für die Szene.                                      |
| [Lightposition](msdn-online-vml-lightposition-attribute.md)           | Gibt die Position des primären Lichts in einer Szene an.                                                 |
| [LightPosition2](msdn-online-vml-lightposition2-attribute.md)         | Gibt die Position des sekundären Lichts in einer Szene an.                                               |
| [Lockrotationcenter](msdn-online-vml-lockrotationcenter-attribute.md) | Bestimmt, ob die Drehung des extrudierten Objekts durch das **RotationAngle** -Attribut angegeben wird. |
| [Metallisch](msdn-online-vml-metal-attribute.md)                           | Bestimmt, ob die Oberfläche der extrudierten Form Metal ähnelt.                               |
| [Ein](on-attribute--extrusion--vml.md)                                 | Bestimmt, ob eine-Extrusion angezeigt wird.                                                      |
| [Ausrichtung](msdn-online-vml-orientation-attribute.md)               | Gibt den Vektor an, um den eine Form gedreht wird.                                              |
| [Orientationangle](msdn-online-vml-orientationangle-attribute.md)     | Definiert den Winkel, den eine-Extrusion um die Ausrichtung dreht.                                     |
| [Wasser](msdn-online-vml-plane-attribute.md)                           | Gibt die Ebene an, die sich im rechten Winkel der-Extrusion befindet.                                           |
| [Render](msdn-online-vml-render-attribute.md)                         | Definiert den Renderingmodus der-Extrusion.                                                            |
| [RotationAngle](msdn-online-vml-rotationangle-attribute.md)           | Gibt die Drehung des Objekts über die x-und die y-Achse an.                                           |
| [RotationCenter](msdn-online-vml-rotationcenter-attribute.md)         | Gibt den Mittelpunkt der Drehung für eine Form an.                                                           |
| [Glanz](msdn-online-vml-shininess-attribute.md)                   | Definiert die Konzentration der reflektierten Beleuchtung einer-extrusionfläche.                                   |
| [Schief](msdn-online-vml-skewamt-attribute.md)                       | Definiert den Umfang der Schiefe einer-Extrusion.                                                             |
| [Schief legen](msdn-online-vml-skewangle-attribute.md)                   | Definiert den Winkel der Schiefe einer-Extrusion.                                                              |
| [Spiegelung](msdn-online-vml-specularity-attribute.md)               | Definiert die Spekulation einer extrudierten Form.                                                           |
| [Type](type-attribute--extrusion--vml.md)                             | Definiert die Art und Weise, in der die Form extrudiert wird.                                                             |
| [Standpunkt](msdn-online-vml-viewpoint-attribute.md)                   | Definiert den Standpunkt des Beobachters.                                                                  |
| [Viewpointorigin](msdn-online-vml-viewpointorigin-attribute.md)       | Definiert den Ursprung des Standpunkts innerhalb des umgebenden Felds der Form.                               |



 

**Anmerkungen**

Dieses Element muss innerhalb eines [Shape](shape-element--vml.md) -Elements definiert werden.

Außerdem muss das [on](on-attribute--extrusion--vml.md) -Attribut auf **true** festgelegt werden.

Im folgenden finden Sie den minimalen Code, der zum Entwickeln einer-Extrusion erforderlich ist.


```HTML
   <v:rect style="width:50px;height:50px">
   <v:extrusion on="True"/>
   </v:rect>
```
