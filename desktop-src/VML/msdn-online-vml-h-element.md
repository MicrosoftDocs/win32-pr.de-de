# <a name="vml-h-element"></a>VML H-Element

In diesem Thema wird VML beschrieben, eine Funktion, die ab Windows Internet Explorer 9 veraltet ist. Webseiten und Anwendungen, die auf VML basieren, sollten zu SVG oder anderen allgemein unterstützten Standards migriert werden.

> [!Note]  
> Ab Dezember 2011 wurde dieses Thema archiviert. Daher wird er nicht mehr aktiv verwaltet. Weitere Informationen finden Sie unter [archivierte Inhalte](/previous-versions/windows/internet-explorer/ie-developer/). Informationen, Empfehlungen und Anleitungen zur aktuellen Version von Windows Internet Explorer finden Sie im [Internet Explorer Developer Center](https://msdn.microsoft.com/ie/).

 

Definiert ein Handle für eine Form.

Die folgenden Attribute ändern ein handle.



| Attribut                                                | BESCHREIBUNG                                                    |
|----------------------------------------------------------|----------------------------------------------------------------|
| [Invx](msdn-online-vml-invx-attribute.md)               | Bestimmt, ob die *x* -Position des Handles invertiert ist. |
| [Nicht](msdn-online-vml-invy-attribute.md)               | Bestimmt, ob die *y* -Position des Handles invertiert ist. |
| [Map](msdn-online-vml-map-attribute.md)                 | Definiert den Zuordnung Bereich eines Handles.                         |
| [Polar](msdn-online-vml-polar-attribute.md)             | Definiert die Mittelpunkt Position für Polar Handles.                 |
| [Position](position-attribute--h--vml.md)               | Gibt die *x* -und *y* -Werte des Handles an.                |
| [Radiin Range](msdn-online-vml-radiusrange-attribute.md) | Definiert den Bereich eines Polar Handles.                           |
| [Switch](msdn-online-vml-switch-attribute.md)           | Bestimmt, ob die handlerdirections getauscht werden.          |
| [Xrange](msdn-online-vml-xrange-attribute.md)           | Definiert den *x* -Bereich eines Handles.                             |
| [YRange](msdn-online-vml-yrange-attribute.md)           | Definiert den *y* -Bereich eines Handles.                             |



 

**Anmerkungen**

Dieses Element muss innerhalb eines [Shape](shape-element--vml.md) -Elements definiert werden.
