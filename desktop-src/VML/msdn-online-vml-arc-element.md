Vordefinierte Bogenform.

**Arc** verwendet die Attribute [startangle](msdn-online-vml-startangle-attribute.md) und [endangle.](msdn-online-vml-endangle-attribute.md)

Der folgende Code ist der Mindestcode, der zum Erstellen eines Bilds erforderlich ist.


```HTML
   <v:arc
   style="top:10;left:10;width:200;height:200"
   startangle="90" endangle="270">
   </v:arc>
```
