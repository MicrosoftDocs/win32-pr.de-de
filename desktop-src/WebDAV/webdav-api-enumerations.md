# <a name="webdav-api-enumerations"></a>WebDAV-API-Enumerationen

Die folgende Enumeration wird in der WebDAV-API verwendet.



| Enumeration                          | Beschreibung                                                                                                                                          |
|--------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AUTHNEXTSTEP**](/windows/desktop/api/davclnt/ne-davclnt-authnextstep) | Gibt die nächste Aktion an, die der WebDAV-Client nach einem erfolgreichen Aufruf der [*DavAuthCallback-Rückruffunktion*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) ergreifen soll. |
