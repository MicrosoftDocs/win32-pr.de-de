# <a name="webdav"></a>WebDAV

## <a name="purpose"></a>Zweck

Web Distributed Authoring and Versioning (WebDAV) ist eine Erweiterung des Hypertext Transfer-Protokolls (http), das definiert, wie grundlegende Dateifunktionen wie kopieren, verschieben, löschen und erstellen mithilfe von http ausgeführt werden. Die WebDAV-API ist ein Satz von Win32-Funktionen zum Erstellen und Verwalten von Verbindungen mit WebDAV-Servern und zum Ausführen von Datei-e/a-Vorgängen für Remote Dateien auf WebDAV-Servern. WebDAV-API-Funktionen sind für die Verwendung mit anderen Win32-API-Funktionen, wie z. b. [Windows-Netzwerkfunktionen](/windows/desktop/WNet/windows-networking-functions) und [Dateiverwaltungsfunktionen](/windows/desktop/FileIO/file-management-functions), konzipiert.

## <a name="developer-audience"></a>Entwicklergruppe

Die WebDAV-API ist für die Verwendung durch C-und C++-Entwickler konzipiert.

## <a name="run-time-requirements"></a>Laufzeitanforderungen

Die WebDAV-API befindet sich in Windows Vista und höher. Informationen zu den Betriebssystemversionen, die für die Verwendung eines bestimmten Programmier Elements erforderlich sind, finden Sie im Abschnitt "Anforderungen" der Referenzseite für dieses Element.
