# <a name="webdav-api-structures"></a>WebDAV-API-Strukturen

Die folgenden Strukturen werden in der WebDAV-API verwendet.



| Struktur                                                   | BESCHREIBUNG                                                                                                                                                                                         |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DAV \_ Callback \_ auth \_ BLOB**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_blob) | Diese Struktur wird verwendet, um ein [*authentifizierungsblob*](/windows/desktop/SecGloss/b-gly) zu speichern, das von der [*davauthcallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) -Rückruffunktion abgerufen wurde. |
| [**DAV-Rückruf Authentifizierung ( \_ \_ \_ unp)**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_unp)   | Diese Struktur wird zum Speichern von Benutzernamen-und Kenn Wort Informationen verwendet, die von der [*davauthcallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) -Rückruffunktion abgerufen wurden.                                               |
| [**DAV \_ \_ -Rückruf-Kred**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_cred)            | Die [*davauthcallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) -Rückruffunktion verwendet diese Struktur, um Benutzer Anmelde Informationen zu speichern.                                                                               |
