# <a name="webdav-api-structures"></a>WebDAV-API-Strukturen

Die folgenden Strukturen werden in der WebDAV-API verwendet.



| Struktur                                                   | Beschreibung                                                                                                                                                                                         |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DAV \_ CALLBACK \_ AUTH \_ BLOB**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_blob) | Diese Struktur wird verwendet, um ein [*Authentifizierungs-BLOB*](/windows/desktop/SecGloss/b-gly) zu speichern, das von der [*DavAuthCallback-Rückruffunktion*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) abgerufen wurde. |
| [**DAV \_ CALLBACK \_ AUTH \_ UNP**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_unp)   | Diese Struktur wird zum Speichern von Benutzernamen- und Kennwortinformationen verwendet, die von der [*DavAuthCallback-Rückruffunktion*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) abgerufen wurden.                                               |
| [**DAV \_ CALLBACK \_ CRED**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_cred)            | Die [*DavAuthCallback-Rückruffunktion*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) verwendet diese Struktur, um Benutzeranmeldeinformationen zu speichern.                                                                               |
