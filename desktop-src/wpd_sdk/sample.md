# <a name="wpd-samples"></a>WPD-Beispiele

Die Windows SDK enthält zwei Beispiele: [wpdapisample](wpdapisample-sample-application.md) und [wpdserviceapisample](wpdapisample-sample-service-application.md).

Wpdapisample ist eine Befehlszeilen-Desktop Anwendung, die es dem Benutzer ermöglicht, verschiedene Funktionen auszuführen, wie z. b. das Auflisten verbundener Geräte, das Untersuchen der Objekte auf diesen Geräten und das Aufzählen von Objekteigenschaften.

Wpdserviceapisample ist eine Desktop Anwendung für die Befehlszeile, die einen Contacts-Geräte Dienst unterstützt und dem Benutzer das Ausführen von Funktionen ermöglicht, wie z. b. das Auflisten verbundener Geräte Dienste, das Abrufen von Eigenschaften und Formaten für einen bestimmten Dienst und das Aufrufen von Methoden für diesen Dienst.
