# <a name="wpd-samples"></a>WPD-Beispiele

Das Windows SDK enthält zwei Beispiele: [WpdApiSample](wpdapisample-sample-application.md) und [WpdServiceApiSample.](wpdapisample-sample-service-application.md)

WpdApiSample ist eine Befehlszeilendesktopanwendung, mit der der Benutzer verschiedene Funktionen ausführen kann, z. B.: Aufzählen verbundener Geräte, Untersuchen der Objekte auf diesen Geräten und Aufzählen von Objekteigenschaften.

WpdServiceApiSample ist eine Befehlszeilendesktopanwendung, die einen Contacts-Gerätedienst unterstützt und es dem Benutzer ermöglicht, Funktionen wie das Aufzählen verbundener Gerätedienste, das Abrufen von Eigenschaften und Formaten für einen bestimmten Dienst und das Aufrufen von Methoden für diesen Dienst auszuführen.
