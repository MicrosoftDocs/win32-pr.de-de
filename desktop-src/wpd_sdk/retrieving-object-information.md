# <a name="retrieving-object-information"></a>Abrufen von Objektinformationen

Zusätzlich zum Abrufen von Geräteinformationen stellt die WPD-API eine Reihe von Schnittstellen und Methoden bereit, mit denen Sie Informationen für ein bestimmtes Objekt abrufen können. Im Allgemeinen kann das Abrufen von Objektinformationen in zwei Kategorien unterteilt werden, wie in der folgenden Tabelle beschrieben.



| Gerätedaten                                                                                    | BESCHREIBUNG                                                                               |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [Objekt Bezeichner](retrieving-an-object-identifier-from-a-persistent-unique-identifier.md)   | Sie können den Objekt Bezeichner abrufen, der einem gegebenen permanenten eindeutigen Bezeichner zugeordnet ist. |
| [Funktionale Objekt Bezeichner](retrieving-the-functional-object-identifiers-for-a-device.md) | Jede funktionale Kategorie unterstützt mindestens ein funktionales Objekt.                         |
