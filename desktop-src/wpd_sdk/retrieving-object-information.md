# <a name="retrieving-object-information"></a>Abrufen von Objektinformationen

Zusätzlich zum Abrufen von Geräteinformationen stellt die WPD-API eine Reihe von Schnittstellen und Methoden bereit, mit denen Sie Informationen für ein bestimmtes Objekt abrufen können. Im Allgemeinen kann der Abruf von Objektinformationen in zwei Kategorien unterteilt werden, wie in der folgenden Tabelle beschrieben.



| Gerätedaten                                                                                    | Beschreibung                                                                               |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [Objektbezeichner](retrieving-an-object-identifier-from-a-persistent-unique-identifier.md)   | Sie können den Objektbezeichner abrufen, der einem angegebenen persistenten eindeutigen Bezeichner zugeordnet ist. |
| [Funktionale Objektbezeichner](retrieving-the-functional-object-identifiers-for-a-device.md) | Jede Funktionskategorie unterstützt mindestens ein funktionales Objekt.                         |
