# <a name="support-in-windows-xp"></a>Unterstützung in Windows XP

Damit Ihre Anwendung in Windows XP ausgeführt werden kann, benötigt der Benutzer eine Kopie der Windows Media Format-Runtime auf dem Computer. Sie können diese Runtime mit Ihrer WPD-Anwendung neu verteilen. (Die Runtime befindet sich in einem Ordner mit verteilbaren Komponenten im Windows Media Format SDK.)
