# <a name="support-in-windows-xp"></a>Unterstützung in Windows XP

Damit Ihre Anwendung unter Windows XP ausgeführt werden kann, benötigt der Benutzer eine Kopie der Windows Media-Laufzeit auf dem Computer. Sie können diese Laufzeit mit ihrer WPD-Anwendung weiterverteilen. (Die Laufzeit befindet sich in einem Ordner von verteilbaren Dateien im Windows Media-Format-SDK.)
