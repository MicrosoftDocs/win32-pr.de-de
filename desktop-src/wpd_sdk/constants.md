# <a name="wpd-constants"></a>WPD-Konstanten

Portable Windows-Geräte definieren die folgenden konstanten Typen.



| Konstantentyp                                              | Beschreibung                                                                                                                                                                                                                               |
|------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Allgemeine Konstanten](general-constants.md)                 | Verwenden Sie allgemeine Konstanten für Geräte und Anwendungen.                                                                                                                                                                                       |
| [Fehlerkonst constants (Fehlerkonst constants](error-constants.md)                     | Von Methoden zurückgegebene Fehlerwerte.                                                                                                                                                                                                         |
| [Ereigniskonst constants](event-constants.md)                     | Ereignistypen, die von portablen Windows-Geräten definiert werden. Zu den Ereignistypen gehören Objekt hinzugefügt, Objekt entfernt, Objekt aktualisiert, Objekt verschoben, Objekt kopiert, Angeforderte Objektübertragung, Gerätezurücksetzung, entferntes Gerät und aktualisierte Gerätefunktionen. |
| [Objekttyp-GUIDs](object-type-guids.md)                 | Von portablen Windows-Geräten definierte Objekttypen. Objekttypen umfassen Audio-, E-Mail- und Bildobjekte.                                                                                                                                  |
| [Objektformat-GUIDs](object-format-guids.md)             | Dateitypen, die von portablen Windows-Geräten definiert werden. Zu den Formattypen gehören WMA, WMV und JPEG.                                                                                                                                           |
| [Ressourcentyp-GUIDs](resource-type-guids.md)             | Ressourcentypen, die von portablen Windows-Geräten definiert werden. Ressourcentypen umfassen Audio- und Miniaturbilddaten.                                                                                                                                |
| [Eigenschaften und Attribute](properties-and-attributes.md) | Eigenschaften und Attribute, die von portablen Windows-Geräten definiert werden.                                                                                                                                                                            |
