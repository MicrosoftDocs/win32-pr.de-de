# <a name="wpd-constants"></a>WPD-Konstanten

Tragbare Windows-Geräte definieren die folgenden Konstanten Typen.



| Konstanter Typ                                              | BESCHREIBUNG                                                                                                                                                                                                                               |
|------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Allgemeine Konstanten](general-constants.md)                 | Allgemeine Verwendungs Konstanten für Geräte und Anwendungen.                                                                                                                                                                                       |
| [Fehler Konstanten](error-constants.md)                     | Fehler Werte, die von Methoden zurückgegeben werden.                                                                                                                                                                                                         |
| [Ereignis Konstanten](event-constants.md)                     | Ereignis Typen, die von tragbaren Windows-Geräten definiert werden. Zu den Ereignis Typen zählen Objekt hinzugefügt, Objekt entfernt, Objekt aktualisiert, Objekt verschoben, Objekt kopiert, Objekt Übertragung angefordert, Geräte Zurücksetzung, Gerät entfernt und Gerätefunktionen aktualisiert. |
| [Objekttyp-GUIDs](object-type-guids.md)                 | Objekttypen, die von tragbaren Windows-Geräten definiert werden. Zu den Objekttypen zählen Audio-, e-Mail-und Bildobjekte.                                                                                                                                  |
| [Objekt Format-GUIDs](object-format-guids.md)             | Dateiformat Typen, die von tragbaren Windows-Geräten definiert werden. Format Typen sind WMA, WMV und JPEG.                                                                                                                                           |
| [Ressourcentyp-GUIDs](resource-type-guids.md)             | Ressourcentypen, die von tragbaren Windows-Geräten definiert werden. Zu den Ressourcentypen zählen Audiodaten und Miniaturbilder.                                                                                                                                |
| [Eigenschaften und Attribute](properties-and-attributes.md) | Eigenschaften und Attribute, die von tragbaren Windows-Geräten definiert werden.                                                                                                                                                                            |
