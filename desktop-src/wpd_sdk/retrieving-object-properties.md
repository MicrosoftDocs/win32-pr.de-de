Im folgenden Thema wird der Abruf für ein einzelnes -Objekt beschrieben. Eine Beschreibung des [Abrufens](advanced-tasks.md) von Eigenschaften für mehrere Objekte finden Sie im Abschnitt Erweiterte Aufgaben.
