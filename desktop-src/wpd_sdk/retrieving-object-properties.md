# <a name="retrieving-object-properties-overview"></a>Übersicht über das Abrufen von Objekteigenschaften

Eine Anwendung kann Eigenschaften für ein einzelnes Objekt, für mehrere durch Objektbezeichner identifizierte Objekte oder für mehrere Objekte desselben Typs abrufen. Im folgenden Thema wird der Abruf für ein einzelnes -Objekt beschrieben. Eine Beschreibung des Abrufens von Eigenschaften für mehrere Objekte finden Sie im Abschnitt [Erweiterte](advanced-tasks.md) Aufgaben.
