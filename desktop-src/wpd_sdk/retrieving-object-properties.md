# <a name="retrieving-object-properties-overview"></a>Übersicht über das Abrufen von Objekteigenschaften

Eine Anwendung kann Eigenschaften für ein einzelnes Objekt, für mehrere Objekte, die von Objekt bezeichmern identifiziert werden, oder für mehrere Objekte desselben Typs abrufen. Im folgenden Thema wird das Abrufen eines einzelnen-Objekts beschrieben. Weitere Informationen zum Abrufen von Eigenschaften für mehrere Objekte finden Sie im Abschnitt " [Erweiterte Aufgaben](advanced-tasks.md) ".
