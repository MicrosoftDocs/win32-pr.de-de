# <a name="wpd-programming-guide"></a>WPD-Programmierhandbuch

Die primäre Quelle für dieses Programmierhandbuch ist ein Beispiel, das mit dem Windows Portable Devices (WPD) Software Development Kit (SDK) geliefert wird. Dieses Beispiel mit dem Namen WpdApiSample.exe besteht aus sieben CPP-Modulen. Sechs dieser Module enthalten den Code, der 26 Aufgaben veranschaulicht, die eine Anwendung mithilfe der WPD-API (Application Programming Interface) ausführen kann.

Die Ausnahmen von oben sind die Themen, in denen folgende Themen beschrieben werden: Aufgaben des Gerätediensts, Kontextmenü und Eigenschaftenblatterweiterungen. diese Themen wurden nicht aus WpdApiSample übernommen. Die Aufgaben des Gerätediensts basieren auf dem Beispiel wpdServiceApiSample. Die Kontextmenü- und Eigenschaftenblatterweiterungen sind Codeausschnitte aus Anwendungen, die nicht mit dem Windows SDK bereitgestellt werden.

Die folgenden Abschnitte konzentrieren sich auf die Aufgaben, die mit diesen Beispielen ausgeführt werden, und erläutern, wie die einzelnen Mithilfe der WPD-Schnittstellen und einer Reihe von Methoden in diesen Schnittstellen ausgeführt werden.
