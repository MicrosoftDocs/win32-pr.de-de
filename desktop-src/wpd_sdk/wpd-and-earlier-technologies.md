# <a name="wpd-and-earlier-technologies"></a>WPD und frühere Technologien

WPD enthält viele Verbesserungen für digitale Kameras und Portable Digital Media-Player und ist die bevorzugte Methode für den Zugriff auf diese Geräte in neuen Versionen von Windows. Die vorhandenen Windows-Abbild Erfassungs-APIs (WIA) und Windows Media Device Manager (WMDM)-APIs in Windows werden weiterhin unterstützt, und die Kompatibilität mit WPD wird von kompatibilitätsschichten bereitgestellt. Mit diesen kompatibilitätsschichten wird sichergestellt, dass WPD-Geräte für vorhandene WIA-und WMDM-Anwendungen transparent als WIA-Geräte oder WMDM-Dienstanbieter verfügbar gemacht werden können. Diese Anwendungen verfügen jedoch möglicherweise nicht über die gleichen Funktionen wie mit der neuen WPD-API und neuen MTP-basierten Geräten.

Der WPD-MTP-Treiber unterstützt auch PTP-Geräte (Picture Transfer Protocol). PTP-Geräte, die in WIA unterstützt werden, funktionieren in WIA-Anwendungen weiterhin über die Kompatibilitätsschicht. Microsoft hat weiterhin die Möglichkeit, die Kompatibilität von Anwendungen und digitalen Technologien in der nächsten Version von Windows sicherzustellen, und diese neuen Technologien sind so konzipiert, dass Sie das digitale Verhalten der Kamera in Windows ergänzen und verbessern können.
