# <a name="supported-tasks-in-wpd-applications"></a>Unterstützte Aufgaben in WPD-Anwendungen

Die WPD-Anwendungsschnittstellen unterstützen die folgenden Aufgaben:

-   Herstellen einer Verbindung mit einem Gerät
-   Angefügte Geräte aufzählen
-   Überprüfen der Funktionen eines Geräts
-   Untersuchen des Inhalts eines Geräts
-   Übertragen von Inhalten auf ein Gerät oder Erstellen von Inhalten
-   Inhalt auf dem Gerät bearbeiten
-   Ändern von Objekten auf einem Gerät
-   Steuern des Geräts
-   Überwachen von Geräte Ereignissen

Weitere Informationen zu den einzelnen Aufgaben sowie Details zum Programmieren einer bestimmten Aufgabe finden Sie im [Programmier Handbuch für tragbare Windows-Geräte](programming-guide.md).
