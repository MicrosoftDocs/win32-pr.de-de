# <a name="common-information-properties"></a>Allgemeine Informations Eigenschaften

Tragbare Windows-Geräte unterstützen die folgenden allgemeinen Informations Eigenschaften.



| Eigenschaft                                      | VarType        | BESCHREIBUNG                                                                                              |
|-----------------------------------------------|----------------|----------------------------------------------------------------------------------------------------------|
| **\_Allgemeine Informationen zu \_ WPD \_**           | **VT \_ LPWSTR** | Bei Terminen, Tasks und ähnlichen Objekten enthält diese Eigenschaft Notizen für das angegebene Objekt.     |
| **häufig verwendeter WPD- \_ \_ Informations \_ Betreff**         | **VT \_ LPWSTR** | Ein-Wert, der das Betrefffeld dieses-Objekts angibt.                                                 |
| **Text für den allgemeinen WPD- \_ \_ Informations \_ \_ Text**      | **VT \_ LPWSTR** | Diese Eigenschaft enthält den Textkörper eines Objekts im Klartext-oder HTML-Format.                          |
| **Allgemeine WPD- \_ \_ Informations \_ Priorität**        | **VT \_ UI4**    | Ein-Wert, der die Priorität dieses-Objekts angibt. 0 gibt die höchste Priorität an.                    |
| **WPD- \_ allgemeine \_ Informationen \_ Start \_ DateTime** | **VT- \_ Datum**   | Ein-Wert, der das Datum/die Uhrzeit angibt, zu der ein Termin, eine Aufgabe oder ein ähnliches Objekt gestartet werden soll. |
| **DateTime-Wert für WPD- \_ allgemeine \_ Informationen \_ \_**   | **VT- \_ Datum**   | Ein-Wert, der das Datum/die Uhrzeit angibt, zu der ein Termin, eine Aufgabe oder ein ähnliches Objekt beendet werden soll.   |
