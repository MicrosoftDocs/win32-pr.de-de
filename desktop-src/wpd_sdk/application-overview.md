# <a name="application-overview"></a>Anwendungsübersicht

Windows Portable Devices (WPD) bietet eine Infrastruktur, die den PC-Zugriff auf Multifunktionsgeräte ermöglicht, auf denen verschiedene Medien und nicht medienbasierte Inhalte gespeichert werden. Diese Infrastruktur ermöglicht konvergente Geräte und konzentriert sich zunächst auf intelligente Speichergeräte wie portable Medienplayer, Digitalkameras und Mobiltelefone.

WPD besteht aus einer Gerätetreiberschnittstelle (Device Driver Interface, DDI) und einem Satz von Anwendungsprogrammierschnittstellen (APIs). In dieser Übersicht wird beschrieben, wie Sie diese APIs in Ihrer Windows nutzen können.

Die WPD-Infrastruktur bietet hochgradig erweiterbare Gerätedarstellungs- und Steuerungsmechanismen. Dies ermöglicht es vertikalen Lösungsanbietern, die WPD-API zu verwenden, um erweiterte Benutzererfahrungen zu erstellen, die sich außerhalb des standardisierten WPD-Sets befinden. Beispiele hierfür sind vom Anbieter bereitgestellte Software, die Firmwareupdates oder Gerätediagnosen ausführt (für Remoteunterstützung).
