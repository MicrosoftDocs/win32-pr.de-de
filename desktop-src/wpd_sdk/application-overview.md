# <a name="application-overview"></a>Anwendungsübersicht

Windows Portable Devices (WPD) bietet eine Infrastruktur, die PC-Zugriff auf Multifunktionsgeräte ermöglicht, in denen verschiedene Medien und nicht-Medieninhalte gespeichert werden. Diese Infrastruktur ermöglicht konvergent Geräte und konzentriert sich anfänglich auf intelligente Speichergeräte, wie z. b. tragbare Medien, digitale Kameras und Mobiltelefone.

WPD besteht aus einer Gerätetreiber Schnittstelle (DDI) und einer Reihe von Anwendungs Programmierschnittstellen (Application Programming Interfaces, APIs). In dieser Übersicht wird beschrieben, wie Sie diese APIs in Ihrer Windows-Anwendung nutzen können.

Die WPD-Infrastruktur bietet hochgradig erweiterbare Geräte Darstellung und Steuerungsmechanismen. Dadurch können vertikale Lösungsanbieter die WPD-API verwenden, um verbesserte Benutzeroberflächen zu erstellen, die sich außerhalb der standardisierten WPD-Gruppe befinden. Hierzu zählen beispielsweise vom Hersteller bereitgestellte Software, mit der Firmwareupdates oder Gerätediagnose (für Remote Unterstützung) durchführt werden.
