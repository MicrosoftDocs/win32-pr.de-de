# <a name="operating-on-device-content"></a>Verwenden von Geräteinhalten

Die WPD-API unterstützt eine Reihe von Vorgängen auf Geräten. Dazu gehören die folgenden Operationen:

-   Übertragen von Inhalten von einem PC auf ein verbundenes Gerät.
-   Übertragen von Inhalten von einem verbundenen Gerät auf einen PC.
-   Verschieben von Inhalten auf einem verbundenen Gerät.
-   Löschen von Inhalten von einem verbundenen Gerät.
-   Hinzufügen einer Ressource zu einem Objekt auf einem verbundenen Gerät.

Diese Vorgänge werden in den folgenden Abschnitten beschrieben.
