# <a name="device-service-tasks"></a>Geräte Dienst Tasks

Die WPD-API unterstützt eine Reihe von Tasks für Geräte Dienste. Dabei handelt es sich um die folgenden Tasks:

-   Geräte Dienste werden aufgelistet.
-   Öffnen eines Geräte Dienstanbieter.
-   Auflisten von Inhalten auf einem Geräte Dienst.
-   Unterstützte Dienst Ereignisse werden abgerufen.
-   Unterstützte Dienst Formate werden abgerufen.
-   Unterstützte Dienst Methoden werden abgerufen.
-   Unterstützte abstrakte Dienste werden abgerufen.
-   Inhalts Objekteigenschaften werden abgerufen.
-   Inhalts Objekteigenschaften werden geschrieben.
-   Synchrones Aufrufen von Dienst Methoden.
-   Asynchrones Aufrufen von Dienst Methoden.

Diese Vorgänge werden in den folgenden Abschnitten beschrieben und in der Anwendung wpdservicesapisample veranschaulicht. Außerdem unterstützt die WPD-API auch das Übertragen von Inhalten, das Verschieben von Inhalten und das Löschen von Inhalten aus einem Geräte Dienst.
