# <a name="still-image-properties"></a>Eigenschaften von "Still Image"

Windows Portable Geräte unterstützen die folgenden Eigenschaften von Stillimages.



| Eigenschaft                                                                                                                                                       | VarType        | Beschreibung                                                                                                                                                                                                                                                                                                                  |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WPD \_ STILL \_ IMAGE \_ INTERPRET**                                                                                                                                  | **VT \_ LPWSTR** | Der Name des Besitzers/Benutzers des Geräts.                                                                                                                                                                                                                                                                                    |
| <span id="wpd_still_image_burst_interval"></span><span id="WPD_STILL_IMAGE_BURST_INTERVAL"></span>**WPD \_ STILL \_ IMAGE \_ BURST \_ INTERVAL**                       | **VT \_ UI4**    | Bei einer Bilderfassung im Burstmodus die Zeitverzögerung zwischen Bilderfassungen in Millisekunden.                                                                                                                                                                                                                                       |
| <span id="wpd_still_image_burst_number"></span><span id="WPD_STILL_IMAGE_BURST_NUMBER"></span>**WPD \_ STILL \_ IMAGE \_ BURST \_ NUMBER**                             | **VT \_ UI4**    | Bei einer Bilderfassung im Burstmodus die Anzahl der zu erfassenden Bilder.                                                                                                                                                                                                                                                              |
| **WPD \_ STILL \_ IMAGE \_ CAPTURE \_ DELAY**                                                                                                                          | **VT \_ UI4**    | Gibt die Zeitverzögerung zwischen dem Befehl zum Erfassen eines Bilds und der tatsächlichen Erfassung in Millisekunden an.                                                                                                                                                                                                                    |
| **WPD \_ STILL \_ IMAGE \_ CAPTURE \_ FORMAT**                                                                                                                         | **VT \_ CLSID**  | Das Format, das zum Erfassen des Bilds verwendet werden soll. Dies ist einer der [Format-GUID-Werte,](object-format-guids.md) der von portablen Windows definiert wird.                                                                                                                                                                        |
| <span id="wpd_still_image_capture_mode"></span><span id="WPD_STILL_IMAGE_CAPTURE_MODE"></span>**WPD \_ STILL \_ IMAGE \_ CAPTURE \_ MODE**                             | **VT \_ UI4**    | Ein [**WPD \_ CAPTURE MODES-Enumerator, \_**](wpd-capture-modes.md) der den Erfassungsmodus angibt, der zum Erfassen eines Bilds verwendet werden soll.                                                                                                                                                                                                    |
| **WPD \_ STILL \_ IMAGE \_ CAPTURE \_ RESOLUTION**                                                                                                                     | **VT \_ LPWSTR** | Eine Zeichenfolge, die die zu erfassende Bildgröße angibt. Die Bildgröße wird in Pixel im Format "*Breite* x *Höhe "* angegeben. Beispiel: "800x600".                                                                                                                                                                          |
| **WPD–EINSTELLUNG \_ \_ FÜR DIE \_ BILDKOMPRIMIERUNG \_**                                                                                                                    | **VT \_ UI8**    | Beschreibt die Bildkomprimierungseinstellungen, die beim Erfassen eines Bilds verwendet werden. Dies ist gerätespezifisch.                                                                                                                                                                                                                     |
| **WPD \_ STILL \_ IMAGE \_ CONTRAST**                                                                                                                                | **VT \_ UI4**    | Eine Zahl, die die Kontrasteinstellung angibt, die beim Erfassen eines Bilds angewendet werden soll. Null ist der geringste Kontrast, und größere Zahlen geben einen größeren Kontrast an.                                                                                                                                                            |
| **WPD \_ STILL \_ IMAGE \_ DIGITAL \_ ZOOM**                                                                                                                           | **VT \_ UI4**    | Eine Zahl, die die Menge des digitalen Zooms angibt, die beim Erfassen eines Bilds verwendet werden soll. Die Zahl ist 1 oder höher, um den Faktor 10 skaliert. Der Wert 1 (für 10) gibt keinen digitalen Zoom an. Der Wert 2 (20) gibt einen 2-fachen Zoom an (1/4 des Bilds, das von einem 1-fachen Zoom gesehen wird, wird erfasst).                  |
| **WPD \_ STILL \_ IMAGE \_ \_ EFFECT-MODUS**                                                                                                                            | **VT \_ UI4**    | Ein [**WPD \_ EFFECT MODES-Enumerator, \_**](wpd-effect-modes.md) der einen Bildeffekt angibt, der beim Erfassen eines Bilds verwendet werden soll.                                                                                                                                                                                                   |
| **WPD \_ STILL \_ IMAGE \_ EXPOSURE \_ BIAS \_ COMPENSATION**                                                                                                            | **VT \_ UI4**    | Eine Zahl, die die APEX-Einheiten angibt, um die Belichtung anzupassen. Einheiten sind Stopps x 1.000. Die Angabe von 0 (null) gibt die Standardeinstellung der Factory an.                                                                                                                                                                                       |
| **WPD \_ STILL \_ IMAGE \_ EXPOSURE \_ INDEX**                                                                                                                         | **VT \_ UI4**    | Eine Zahl, die die ISO-Filmgeschwindigkeit angibt, die beim Erfassen eines Bilds emuliert werden soll. Der Wert 0xFFFF gibt an, dass dieser Wert automatisch festgelegt werden soll.                                                                                                                                                                      |
| **WPD \_ STILL \_ IMAGE \_ EXPOSURE \_ METERING \_ MODE**                                                                                                                | **VT \_ UI4**    | Ein [**WPD \_ EXPOSURE \_ METERING MODES-Enumerator, \_**](wpd-exposure-metering-modes.md) der den Messungsmodus angibt, der beim Erfassen eines Bilds verwendet werden soll.                                                                                                                                                                          |
| <span id="wpd_still_image_exposure_program_mode"></span><span id="WPD_STILL_IMAGE_EXPOSURE_PROGRAM_MODE"></span>**WPD \_ STILL \_ IMAGE \_ EXPOSURE \_ PROGRAM \_ MODE** | **VT \_ UI4**    | Ein [**WPD \_ EXPOSURE PROGRAM \_ MODES-Enumerator, \_**](wpd-exposure-program-modes.md) der das beim Erfassen eines Bilds zu verwendende Belichtungsprogramm angibt.                                                                                                                                                                         |
| **WPD \_ STILL \_ IMAGE \_ EXPOSURE \_ TIME**                                                                                                                          | **VT \_ UI4**    | Die beim Erfassen eines Bilds zu verwendende Geschwindigkeit in Sekunden x 10.000.                                                                                                                                                                                                                                                       |
| **WPD \_ STILL \_ IMAGE \_ FLASH \_ MODE**                                                                                                                             | **VT \_ UI4**    | Ein [**WPD \_ FLASH MODES-Enumerator, \_**](wpd-flash-modes.md) der den Flashmodus angibt, der beim Erfassen eines Bilds verwendet werden soll.                                                                                                                                                                                                      |
| **WPD \_ STILL \_ IMAGE \_ FNUMBER**                                                                                                                                 | **VT \_ UI4**    | Eine Zahl, die die Öffnung der Brille beim Erfassen eines Bilds angibt. Die Zahl ist die f-Stop-Nummer 100 (14 würde also f/1.4 angeben). Diese Eigenschaft ist in der Regel nur gültig, wenn [WPD \_ STILL IMAGE EXPOSURE PROGRAM \_ \_ \_ \_ MODE](/windows) auf manual oder aperture priority festgelegt ist.<br/> |
| **WPD \_ STILL \_ IMAGE \_ FOCAL \_ LENGTH**                                                                                                                           | **VT \_ UI4**    | Die Fokuslänge in Millimeter x 100, die beim Erfassen eines Bilds verwendet werden soll. Beispielsweise wäre 35mm 3500.                                                                                                                                                                                                                 |
| **WPD \_ STILL \_ IMAGE \_ FOCUS \_ DISTANCE**                                                                                                                         | **VT \_ UI4**    | Der Fokusabstand, den ein Gerät verwenden sollte, in Millimeter. Der Wert 0xFFFF gibt den unendlichen Fokus an (mehr als 655 Meter).                                                                                                                                                                                              |
| **WPD \_ STILL \_ IMAGE \_ FOCUS \_ METERING \_ MODE**                                                                                                                   | **VT \_ UI4**    | Ein [**WPD \_ FOCUS \_ METERING MODES-Enumerator, \_**](wpd-focus-metering-modes.md) der angibt, welchen Fokusmodus ein Bilderfassungsgerät bei der Entscheidung verwenden soll, wie sich ein Thema konzentrieren soll.                                                                                                                                     |
| **WPD \_ STILL \_ IMAGE \_ FOCUS \_ MODE**                                                                                                                             | **VT \_ UI4**    | Ein [**WPD \_ FOCUS MODES-Enumerator, \_**](wpd-focus-modes.md) der den Fokusmodus angibt, der beim Erfassen eines Bilds verwendet werden soll.                                                                                                                                                                                                      |
| **WPD \_ STILL \_ IMAGE \_ RGB \_ GAIN**                                                                                                                               | **VT \_ LPWSTR** | Eine auf NULL beendete Zeichenfolge, die den RGB-Gewinn angibt, der beim Erfassen eines Bilds verwendet werden soll. Bei der Zeichenfolge handelt es sich um drei durch Doppelpunkte getrennte Zahlen ohne Zwischenräume im Format "*R*:*G*:*B*".                                                                                                                                    |
| **WPD \_ STILL \_ IMAGE \_ SHARPNESS**                                                                                                                               | **VT \_ UI4**    | Eine Zahl, die die Schärfe angibt, die beim Erfassen eines Bilds verwendet werden soll, wobei 0 die geringste Schärfe ist und größere Zahlen auf eine zunehmende Schärfe hinweisen.                                                                                                                                                                        |
| <span id="wpd_still_image_timelapse_interval"></span><span id="WPD_STILL_IMAGE_TIMELAPSE_INTERVAL"></span>**WPD \_ STILL \_ IMAGE \_ TIMELAPSE \_ INTERVAL**           | **VT \_ UI4**    | Die Zeitverzögerung zwischen Bilderfassungen bei einer Zeitraffer-Bilderfassung in Millisekunden.                                                                                                                                                                                                                                        |
| <span id="wpd_still_image_timelapse_number"></span><span id="WPD_STILL_IMAGE_TIMELAPSE_NUMBER"></span>**WPD \_ STILL \_ IMAGE \_ TIMELAPSE \_ NUMBER**                 | **VT \_ UI4**    | Die Anzahl der Bilder, die in einer Bilderfassung im Zeitraffermodus erfasst werden.                                                                                                                                                                                                                                                          |
| **WPD \_ STILL \_ IMAGE \_ UPLOAD \_ URL**                                                                                                                             | **VT \_ LPWSTR** | Gibt eine URL an, an die das Gerät das Bild senden soll, nachdem es erfasst wurde.                                                                                                                                                                                                                                                 |
| **WPD \_ STILL \_ IMAGE \_ WHITE \_ BALANCE**                                                                                                                          | **VT \_ UI4**    | Ein [**aufzählender WPD \_ WHITE BALANCE \_ \_ SETTINGS-Wert,**](wpd-white-balance-settings.md) der die White Balance-Einstellungen angibt, die beim Erfassen eines Bilds verwendet werden sollen.                                                                                                                                                             |
