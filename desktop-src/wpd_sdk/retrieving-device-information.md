# <a name="retrieving-device-information"></a>Abrufen von Geräteinformationen

Die WPD-API stellt einen umfangreichen Satz von Schnittstellen und Methoden bereit, mit denen Sie Informationen für ein bestimmtes Gerät abrufen können. Im Allgemeinen können Gerätedaten in drei Kategorien unterteilt werden, wie in der folgenden Tabelle beschrieben.



| Gerätedaten                                                                             | BESCHREIBUNG                                                                               |
|-----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [Funktionale Kategorien](retrieving-the-functional-categories-supported-by-a-device.md) | In den funktionalen Kategorien werden die Vorgänge beschrieben, die ein Gerät ausführen kann. |
| [Inhaltstypen](retrieving-the-content-types-supported-by-a-device.md)                 | Die Inhaltstypen bezeichnen die Inhaltstypen, die ein Gerät rendern kann.         |
| [Ereignisse](retrieving-the-events-supported-by-a-device.md)                               | Einige Geräte können Ereignisse auslösen, die von der Anwendung nachverfolgt und beantwortet werden können. |
