# <a name="supported-operating-systems"></a>Unterstützte Betriebssysteme

Tragbare Windows-Geräte werden in den Betriebssystemen Windows 7, Windows Vista und Windows XP unterstützt. Ihre Anwendung wird mit den bereitgestellten Headern und Bibliotheken kompiliert und mit diesen Betriebssystemen verknüpft.
