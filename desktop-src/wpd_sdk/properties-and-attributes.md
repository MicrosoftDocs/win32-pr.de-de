# <a name="properties-and-attributes"></a>Eigenschaften und Attribute

Eigenschaften beschreiben Objekte, Eigenschaften, Ressourcen und Geräte. Attribute sind eine besondere Art von Eigenschaft, die andere Eigenschaften oder Ressourcen beschreibt.

Eigenschaften und Attribute werden als **PropertyKey** -Strukturen mit zwei Teilen definiert: eine Kategorie-GUID und eine eindeutige ID für diese Kategorie. Eigenschaften, die verwandte Dinge beschreiben, haben eine gemeinsame GUID. Diese GUID ist ihre Eigenschaften Kategorie. Eine Kategorie kann eine oder mehrere Eigenschaften beschreiben, die jeweils über eine eindeutige ID verfügen. Microsoft definiert einen Satz von Eigenschaften Kategorien. Sie können diese für Ihre eigenen Eigenschaften verwenden oder selbst neue Kategorien erstellen.

In den folgenden Themen sind die Attribute und Eigenschaften aufgeführt, die von den tragbaren Windows-Geräten definiert werden.
