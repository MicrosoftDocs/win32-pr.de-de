# <a name="properties-and-attributes"></a>Eigenschaften und Attribute

Eigenschaften beschreiben Objekte, Eigenschaften, Ressourcen und Geräte. Attribute sind eine besondere Art von Eigenschaft, die andere Eigenschaften oder Ressourcen beschreibt.

Eigenschaften und Attribute werden als **PROPERTYKEY-Strukturen** mit zwei Teilen definiert: einer Kategorie-GUID und einer eindeutigen ID für diese Kategorie. Eigenschaften, die verwandte Dinge beschreiben, verwenden eine gemeinsame GUID. diese GUID ist ihre Eigenschaftenkategorie. Eine Kategorie kann eine oder mehrere Eigenschaften mit jeweils einer eindeutigen ID beschreiben. Microsoft definiert einen Satz von Eigenschaftenkategorien. Sie können diese für Ihre eigenen Eigenschaften verwenden oder selbst neue Kategorien erstellen.

In den folgenden Themen werden die Attribute und Eigenschaften aufgeführt, die von portablen Windows definiert werden.
