# <a name="media-properties"></a>Medieneigenschaften

Windows Portable Geräte unterstützen die folgenden Medieneigenschaften.



| Eigenschaft                                                                                                                 | VarType        | Beschreibung                                                                                                                                                                                                                                                                                                    |
|--------------------------------------------------------------------------------------------------------------------------|----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WPD \_ MEDIA \_ ALBUM \_ INTERPRET**                                                                                            | **VT \_ LPWSTR** | Eine Zeichenfolge, die den Interpreten identifiziert, der die Medien vorgibt.                                                                                                                                                                                                                                                      |
| **WPD \_ MEDIA \_ AUDIO \_ ENCODING \_ PROFILE**                                                                                 | VT \_ LPWSTR     | Eine Zeichenfolge, die ein Codierungsprofil angibt. Mediencodecs können in Übereinstimmung mit einem Profil codiert werden, das einen bestimmten Codierungsalgorithmus oder Optimierungsprozess definiert.                                                                                                                                  |
| **WPD \_ MEDIA \_ \_ BITRATE-TYP**                                                                                            | **VT \_ UI4**    | Eine [**WPD \_ BITRATE \_ TYPES-Enumeration,**](wpd-bitrate-types.md) die die Bitrate von Audio- oder Videodaten weiter qualifiziert.                                                                                                                                                                                     |
| **WPD \_ MEDIA \_ JETZT \_ KAUFEN**                                                                                                 | **VT \_ BOOL**   | Ein boolescher Wert, der angibt, ob der Benutzer den Inhalt kaufen möchte, wenn das nächste Mal eine Internetverbindung hergestellt wird.                                                                                                                                                                                         |
| **WPD \_ MEDIA \_ BYTE \_ BOOKMARK**                                                                                           | **VT \_ UI8**    | Gibt ein Lesezeichen (ein nullbasiertes Byteoffset) an, das der letzten Position entspricht, die auf dem angegebenen Medium abgespielt oder angezeigt wurde.                                                                                                                                                                                     |
| **WPD \_ MEDIA \_ COMPOSER**                                                                                                 | **VT \_ LPWSTR** | Der Composer, wenn sich der Composer und der Performer unterscheiden.                                                                                                                                                                                                                                                     |
| **\_WPD-MEDIEN \_ COPYRIGHT**                                                                                                | **VT \_ LPWSTR** | Eine Zeichenfolge, die Copyrightinformationen zum Objekt enthält.                                                                                                                                                                                                                                                 |
| <span id="wpd_media_description"></span><span id="WPD_MEDIA_DESCRIPTION"></span>**\_WPD-MEDIENBESCHREIBUNG \_**              | **VT \_ LPWSTR** | Eine Beschreibung für das gegebene Objekt. Wenn die [WPD \_ MEDIA SUB \_ \_ DESCRIPTION-Eigenschaft](/windows) einen einzelnen Inhalt beschreiben kann, beschreibt diese Eigenschaft eine Sammlung verwandter Medien.<br/>                                                                                 |
| **\_ \_ WPD-MEDIENZIEL-URL \_**                                                                                         | **VT \_ LPWSTR** | Eine auf NULL beendete, für Menschen lesbare Zeichenfolge, die die URL für ein Objekt angibt. Diese URL identifiziert den Inhalt, der angezeigt wird, wenn der Benutzer auf das Objekt klickt. **Mediacastobjekte** verwenden diese Eigenschaft häufig, um den Speicherort verwandter Informationen anzugeben.<br/>                                          |
| **\_WPD-MEDIENDAUER \_**                                                                                                 | **VT \_ UI8**    | Die Dauer des Inhalts in Millisekunden.                                                                                                                                                                                                                                                                  |
| **WPD \_ MEDIA \_ EFFECTIVE \_ RATING**                                                                                        | **VT \_ UI4**    | Eine Zahl zwischen 1 und 100 (einschließlich), die den Inhalt einwertet, wobei 1 der niedrigste Wert ist. Dieser Wert basiert auf Nutzungsstatistiken, die nicht explizit vom Benutzer festgelegt werden.                                                                                                                                   |
| **\_ \_ WPD-MEDIENCODIERUNGSPROFIL \_**                                                                                        | **VT \_ LPWSTR** | Ein Profilname, der durch den verwendeten Codec definiert wird. Wenn der Codec keinen bestimmten Profilnamen erfordert, sollten Sie stattdessen Audioinformationen in den entsprechenden Eigenschaften angeben.                                                                                                            |
| **WPD \_ MEDIA \_ GENRE**                                                                                                    | **VT \_ LPWSTR** | Eine auf NULL beendete, für Menschen lesbare Zeichenfolge, die das Genre angibt, zu dem dieses Medium gehört.                                                                                                                                                                                                                     |
| **\_ \_ WPD-MEDIEN-GUID**                                                                                                     | **VT \_ LPWSTR** | Gibt einen 128-Bit-GUID (Globally Unique Identifier) an.                                                                                                                                                                                                                                                         |
| <span id="wpd_media_height"></span><span id="WPD_MEDIA_HEIGHT"></span>**\_WPD-MEDIENHÖHE \_**                             | **VT \_ UI4**    | Ein -Wert, der die Höhe der Hauptressource (in der Regel ein Video oder Bild) in Pixel angibt. Weitere Informationen [finden Sie unter WPD MEDIA \_ \_ WIDTH](/windows).                                                                                                                                                                    |
| **WPD \_ MEDIA \_ LAST \_ BUILD \_ DATE**                                                                                        | **VT \_ DATE**   | Ein Datum, das den Zeitpunkt an gibt, zu dem eine Reihe in einem Mediacast zuletzt geändert oder bearbeitet wurde.                                                                                                                                                                                                                                 |
| **ZEITPUNKT DES \_ LETZTEN \_ ZUGRIFFES AUF WPD-MEDIEN \_ \_**                                                                                     | **VT \_ DATE**   | Ein -Wert, der das Datum und die Uhrzeit des letzten Zugriffes auf das Medium auf dem Gerät angibt.                                                                                                                                                                                                                       |
| **\_ \_ WPD-MEDIENVERWALTUNGS-EDITOR \_**                                                                                         | **VT \_ LPWSTR** | Eine auf NULL beendete, lesbare Zeichenfolge, die die E-Mail-Adresse des Verwaltungs-Editors des Mediums angibt. **Mediacastobjekte** verwenden diese Eigenschaft häufig, um den Verwaltungs-Editor eines RSS-Kanals oder einer RSS-Serie anzugeben.<br/>                                                                                 |
| **WPD \_ MEDIA \_ META \_ GENRE**                                                                                              | **VT \_ UI4**    | Eine [**WPD \_ META \_ ENUMERATION-Enumeration,**](wpd-meta-genres.md) die den Medientyp beschreibt.                                                                                                                                                                                                                   |
| **WPD \_ MEDIA \_ OBJECT \_ BOOKMARK**                                                                                         | **VT \_ LPWSTR** | Gibt die \_ WPD-OBJEKT-ID \_ für das zuletzt angezeigte oder abgespielte Objekt an. Diese Eigenschaft gilt für Wiedergabelisten und Mediacasts.<br/>                                                                                                                                                                               |
| **\_WPD-MEDIENBESITZER \_**                                                                                                    | **VT \_ LPWSTR** | Eine auf NULL beendete, lesbare Zeichenfolge, die die E-Mail-Adresse des Medienbesitzers angibt. **Mediacastobjekte** verwenden diese Eigenschaft häufig, um den Besitzer eines RSS-Kanals oder einer RSS-Serie anzugeben.<br/>                                                                                                     |
| **WPD \_ \_ MEDIA-JUGENDBEWERTUNG \_**                                                                                         | **VT \_ LPWSTR** | Eine auf NULL beendete, lesbare Zeichenfolge, die die Jugendbewertung der Mediendatei angibt. Dies ist in der Regel eine MPAA-Bewertung (Motion Picture Association of America).                                                                                                                                        |
| **\_ \_ WPD-MEDIENVERÖFFENTLICHUNGSDATUM \_**                                                                                            | **VT \_ DATE**   | Das Datum, an dem der Inhalt ursprünglich veröffentlicht wurde.                                                                                                                                                                                                                                                             |
| **WPD \_ MEDIA \_ SAMPLE \_ RATE**                                                                                             | **VT \_ UI4**    | Die Codierungsstichprobenrate in Stichproben pro Sekunde für den Inhalt.                                                                                                                                                                                                                                              |
| **ANZAHL DER \_ ÜBERSPRUNGENEN WPD-MEDIEN \_ \_**                                                                                              | **VT \_ UI4**    | Die Gesamtanzahl, mit der dieses Medium für die Wiedergabe in die Warteschlange gestellt, aber explizit vom Benutzer übersprungen wurde. Dies kann verwendet werden, um automatisch eine Benutzerbewertung zu berechnen oder Inhaltaktualisierungsregeln zu ändern.                                                                                                                  |
| **\_ \_ WPD-MEDIENQUELLEN-URL \_**                                                                                              | **VT \_ LPWSTR** | Eine auf NULL beendete, lesbare Zeichenfolge, die die Quell-URL für die angegebenen Medien angibt. **Mediacastobjekte** verwenden diese Eigenschaft häufig, um anzugeben, woher der Inhalt stammt.<br/>                                                                                                                   |
| **WPD \_ MEDIA \_ STAR \_ RATING**                                                                                             | **VT \_ UI4**    | Eine Zahl zwischen 0 und 5, die den Inhalt bewertet, wobei 0 nicht bewertet, 1 die niedrigste Bewertung und 5 die höchste Bewertung ist. Dieser Wert wird explizit vom Benutzer eingegeben.                                                                                                                                        |
| <span id="wpd_media_sub_description"></span><span id="WPD_MEDIA_SUB_DESCRIPTION"></span>**\_ \_ WPD-MEDIENUNTERBESCHREIBUNG \_** | **VT \_ LPWSTR** | Eine untergeordnete Beschreibung für das gegebene Objekt. Diese Eigenschaft wird häufig verwendet, um zusätzliche Informationen zu Inhalten zur Verfügung zu stellen. Wenn die [WPD \_ MEDIA \_ DESCRIPTION-Eigenschaft](/windows) eine Sammlung verwandter Medien beschreiben kann, beschreibt diese Eigenschaft einen einzelnen Inhalt.<br/> |
| <span id="wpd_media_sub_title"></span><span id="WPD_MEDIA_SUB_TITLE"></span>**\_ \_ WPD-MEDIENUNTERTITEL \_**                   | **VT \_ LPWSTR** | Ein Untertitel für das -Objekt. Wenn es sich bei dem Objekt um einen Episodentitel in einer Reihe handelt, wäre dies der Titel der Folge, und [WPD \_ MEDIA \_ TITLE](/windows) wäre der Serienname.                                                                                                                                 |
| **\_ \_ INHALTS-ID DES WPD-MEDIENABONNEMENTS \_ \_**                                                                                | **VT \_ LPWSTR** | Eine ID, die zum Identifizieren eines Inhaltsteils für einen Onlineabonnementdienst verwendet wird. In der Regel wird dies vom Client und nicht vom Treiber oder Gerät festgelegt.                                                                                                                                                    |
| **\_ \_ \_ WPD-MEDIENZEITLESEZEICHEN**                                                                                           | **VT \_ UI8**    | Gibt ein Lesezeichen (in Millisekunden) an, das der letzten auf dem angegebenen Medium wiedergegebenen oder angezeigten Position entspricht.                                                                                                                                                                                              |
| **WPD \_ MEDIA \_ TIME \_ TO \_ LIVE**                                                                                           | **VT \_ UI8**    | Gibt die Zeit (in Minuten) bis zum nächsten Inhaltsupdate an.                                                                                                                                                                                                                                                 |
| <span id="wpd_media_title"></span><span id="WPD_MEDIA_TITLE"></span>**\_WPD-MEDIENTITEL \_**                                | **VT \_ LPWSTR** | Der Titel des Inhalts. Wenn das Objekt eine Folge in einer Reihe ist, wäre dies der Titel der Reihe, und [WPD \_ MEDIA SUB \_ \_ TITLE](/windows) wäre der Name der Episode.                                                                                                                               |
| **WPD \_ MEDIA \_ TOTAL \_ BITRATE**                                                                                           | **VT \_ UI4**    | Die Bitrate des -Objekts.                                                                                                                                                                                                                                                                                    |
| **ANZAHL DER \_ \_ WPD-MEDIENNUTZUNG \_**                                                                                               | **VT \_ UI4**    | Die Gesamtanzahl, mit der dieses Medium auf dem Gerät wiedergegeben oder angezeigt wurde.                                                                                                                                                                                                                                  |
| **WPD \_ MEDIA \_ USER \_ EFFECTIVE \_ RATING**                                                                                  | **VT \_ UI4**    | Eine Zahl zwischen 1 und 100, die diesen Inhalt bewertet, wobei 1 die niedrigste ist. Dieser Wert wird explizit vom Benutzer festgelegt. Der Standardwert ist 0 und gibt an, dass dieser Wert nicht festgelegt wurde.                                                                                                                    |
| **WPD \_ \_ MEDIALES**                                                                                                | **VT \_ LPWSTR** | Eine auf NULL endende, lesbare Zeichenfolge, die die E-Mail-Adresse des Mediums angibt. **Mediacastobjekte** verwenden diese Eigenschaft häufig, um die Anzahl eines RSS-Kanals oder einer RSS-Serie anzugeben.<br/>                                                                                             |
| <span id="wpd_media_width"></span><span id="WPD_MEDIA_WIDTH"></span>**\_WPD-MEDIENBREITE \_**                                | **VT \_ UI4**    | Ein -Wert, der die Breite der Hauptressource (in der Regel ein Video oder Bild) in Pixel angibt. Weitere Informationen finden Sie unter [WPD \_ MEDIA \_ HEIGHT](/windows).                                                                                                                                                                   |
| **WPD \_ MEDIA \_ INTERPRET**                                                                                                   | **VT \_ LPWSTR** | Ein -Wert, der den Medienzeichner identifiziert.                                                                                                                                                                                                                                                                      |
