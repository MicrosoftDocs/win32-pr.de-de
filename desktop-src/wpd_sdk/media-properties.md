# <a name="media-properties"></a>Medien Eigenschaften

Tragbare Windows-Geräte unterstützen die folgenden Medien Eigenschaften.



| Eigenschaft                                                                                                                 | VarType        | BESCHREIBUNG                                                                                                                                                                                                                                                                                                    |
|--------------------------------------------------------------------------------------------------------------------------|----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WPD \_ Media \_ Album- \_ Künstler**                                                                                            | **VT \_ LPWSTR** | Eine Zeichenfolge, die den Künstler identifiziert, der die Medien ausführt.                                                                                                                                                                                                                                                      |
| **WPD \_ \_ -Medien- \_ audiocodierungs \_ Profil**                                                                                 | VT \_ LPWSTR     | Eine Zeichenfolge, die ein Codierungs Profil angibt. Medien Codecs können in Übereinstimmung mit einem Profil codiert werden, das einen bestimmten Codierungs Algorithmus oder Optimierungsprozess definiert.                                                                                                                                  |
| **Typ der WPD- \_ Medien \_ Bitrate \_**                                                                                            | **VT \_ UI4**    | Eine [**WPD- \_ Bitrate \_ types**](wpd-bitrate-types.md) -Enumeration, die die Bitrate von Audiodaten oder Videodaten weiter qualifiziert.                                                                                                                                                                                     |
| **WPD \_ - \_ Medien \_ Jetzt kaufen**                                                                                                 | **VT \_ bool**   | Ein boolescher Wert, der angibt, ob der Benutzer den Inhalt bei der nächsten Internet Verbindung kaufen möchte.                                                                                                                                                                                         |
| **WPD- \_ Medien \_ Byte- \_ Lesezeichen**                                                                                           | **VT \_ UI8**    | Gibt ein Lesezeichen (ein NULL basiertes Byte Offset) an, das der letzten Position entspricht, die auf dem angegebenen Medium abgespielt oder angezeigt wird.                                                                                                                                                                                     |
| **WPD \_ Media \_ Composer**                                                                                                 | **VT \_ LPWSTR** | Der Composer, wenn sich der Composer und der Performer unterscheiden.                                                                                                                                                                                                                                                     |
| **Copyright von WPD- \_ Medien \_**                                                                                                | **VT \_ LPWSTR** | Eine Zeichenfolge, die Copyright Informationen über das Objekt bereitstellt.                                                                                                                                                                                                                                                 |
| <span id="wpd_media_description"></span><span id="WPD_MEDIA_DESCRIPTION"></span>**WPD- \_ Medien \_ Beschreibung**              | **VT \_ LPWSTR** | Eine Beschreibung für das angegebene Objekt. Wenn die [ \_ \_ unter \_ Description](/windows) -Eigenschaft der WPD-Media-Eigenschaft einen einzelnen Inhalt beschreiben kann, beschreibt diese Eigenschaft eine Sammlung verwandter Medien.<br/>                                                                                 |
| **Ziel-URL für WPD- \_ Medien \_ \_**                                                                                         | **VT \_ LPWSTR** | Eine auf NULL endenden, lesbare Zeichenfolge, die die URL für ein Objekt angibt. Mit dieser URL wird der Inhalt identifiziert, der angezeigt wird, wenn der Benutzer auf das Objekt klickt. **Mediacast** -Objekte verwenden diese Eigenschaft häufig, um den Speicherort der zugehörigen Informationen anzugeben.<br/>                                          |
| **Dauer der WPD- \_ Medien \_**                                                                                                 | **VT \_ UI8**    | Die Dauer des Inhalts in Millisekunden.                                                                                                                                                                                                                                                                  |
| **\_ \_ effektive \_ Bewertung von WPD-Medien**                                                                                        | **VT \_ UI4**    | Eine Zahl zwischen 1 und 100, die den Inhalt bewertet, wobei 1 der niedrigste Wert ist. Dieser Wert basiert auf Verwendungs Statistiken, die nicht explizit vom Benutzer festgelegt werden.                                                                                                                                   |
| **WPD- \_ Medien \_ Codierungs \_ Profil**                                                                                        | **VT \_ LPWSTR** | Ein Profilname, der vom verwendeten Codec definiert wird. Wenn für den Codec kein bestimmter Profilname erforderlich ist, sollten Sie stattdessen Audioinformationen in den entsprechenden Eigenschaften angeben.                                                                                                            |
| **WPD- \_ Medien \_ Genre**                                                                                                    | **VT \_ LPWSTR** | Eine auf NULL endenden, lesbare Zeichenfolge, die das Genre angibt, zu dem dieses Medium gehört.                                                                                                                                                                                                                     |
| **WPD- \_ Medien- \_ GUID**                                                                                                     | **VT \_ LPWSTR** | Gibt eine 128-Bit-Globally Unique Identifier (GUID) an.                                                                                                                                                                                                                                                         |
| <span id="wpd_media_height"></span><span id="WPD_MEDIA_HEIGHT"></span>**WPD- \_ Medien \_ Höhe**                             | **VT \_ UI4**    | Ein-Wert, der die Höhe der Haupt Ressource (in der Regel ein Video oder Bild) in Pixel angibt. Siehe [WPD- \_ Medien \_ Breite](/windows).                                                                                                                                                                    |
| **Datum des \_ \_ letzten \_ Builds \_ für WPD-Medien**                                                                                        | **VT- \_ Datum**   | Ein Datum, das den Zeitpunkt angibt, zu dem eine Reihe in einer Mediacast zuletzt geändert oder bearbeitet wurde.                                                                                                                                                                                                                                 |
| **Zeitpunkt des \_ \_ letzten \_ Zugriffs \_ auf WPD-Medien**                                                                                     | **VT- \_ Datum**   | Ein-Wert, der das Datum und die Uhrzeit des letzten Zugriffs auf den Medien auf dem Gerät angibt.                                                                                                                                                                                                                       |
| **Editor für WPD- \_ Medien \_ Verwaltung \_**                                                                                         | **VT \_ LPWSTR** | Eine auf NULL endenden, lesbare Zeichenfolge, die die e-Mail-Adresse des verwalterungs-Editors des Mediums angibt. **Mediacast** -Objekte verwenden diese Eigenschaft häufig, um den Verwaltungs-Editor eines RSS-Kanals oder einer RSS-Reihe anzugeben.<br/>                                                                                 |
| **WPD \_ - \_ medienmeta- \_ Genre**                                                                                              | **VT \_ UI4**    | Eine [**WPD \_ Meta \_ Genres**](wpd-meta-genres.md) -Enumeration, die den Medientyp beschreibt.                                                                                                                                                                                                                   |
| **Lesezeichen für WPD- \_ Medien \_ Objekt \_**                                                                                         | **VT \_ LPWSTR** | Gibt die WPD- \_ Objekt- \_ ID für das letzte angezeigte oder wiedergegebene Objekt an. Diese Eigenschaft gilt für Wiedergabelisten und Mediacasts.<br/>                                                                                                                                                                               |
| **WPD- \_ Medien \_ Besitzer**                                                                                                    | **VT \_ LPWSTR** | Eine auf NULL endenden, lesbare Zeichenfolge, die die e-Mail-Adresse des Besitzers der Medien angibt. **Mediacast** -Objekte verwenden diese Eigenschaft häufig, um den Besitzer eines RSS-Kanals oder einer RSS-Reihe anzugeben.<br/>                                                                                                     |
| **\_ \_ Eltern \_ Bewertung für WPD-Medien**                                                                                         | **VT \_ LPWSTR** | Eine auf NULL endenden, lesbare Zeichenfolge, die die Eltern Bewertung der Mediendatei angibt. Dies ist in der Regel eine Bewegungsbild Zuordnung der America-Bewertung (MPa).                                                                                                                                        |
| **\_ \_ Veröffentlichungs \_ Datum für WPD-Medien**                                                                                            | **VT- \_ Datum**   | Das Datum, an dem der Inhalt ursprünglich freigegeben wurde.                                                                                                                                                                                                                                                             |
| **WPD- \_ Medien \_ Stichproben \_ Rate**                                                                                             | **VT \_ UI4**    | Die Codierungs Stichproben Rate in Samplings pro Sekunde für den Inhalt.                                                                                                                                                                                                                                              |
| **Anzahl der \_ Skip-Medien für WPD \_ \_**                                                                                              | **VT \_ UI4**    | Die Gesamtanzahl der in die Warteschlange eingereihten Medien für die Wiedergabe, die jedoch explizit vom Benutzer übersprungen wurden. Dies kann verwendet werden, um automatisch eine Benutzer Bewertung zu berechnen oder Inhalts Aktualisierungs Regeln zu ändern.                                                                                                                  |
| **WPD- \_ Medien \_ Quellen- \_ URL**                                                                                              | **VT \_ LPWSTR** | Eine auf NULL endenden, lesbare Zeichenfolge, die die Quell-URL für die angegebenen Medien angibt. **Mediacast** -Objekte verwenden diese Eigenschaft häufig, um anzugeben, woher der Inhalt stammt.<br/>                                                                                                                   |
| **Bewertung der WPD \_ \_ -Medien Sterne \_**                                                                                             | **VT \_ UI4**    | Eine Zahl zwischen 0 und 5, die den Inhalt bewertet, wobei 0 nicht bewertet, 1 die niedrigste Bewertung und 5 die höchste Bewertung ist. Dieser Wert wird explizit vom Benutzer eingegeben.                                                                                                                                        |
| <span id="wpd_media_sub_description"></span><span id="WPD_MEDIA_SUB_DESCRIPTION"></span>**\_ \_ unter \_ Beschreibung der WPD-Medien** | **VT \_ LPWSTR** | Eine untergeordnete Beschreibung für das angegebene Objekt. Diese Eigenschaft wird häufig verwendet, um zusätzliche Informationen über Inhalte bereitzustellen. Wenn die Eigenschaft [WPD- \_ Medien \_ Beschreibung](/windows) eine Auflistung verwandter Medien beschreiben kann, wird diese Eigenschaft ein einzelnes Inhalts Element beschreiben.<br/> |
| <span id="wpd_media_sub_title"></span><span id="WPD_MEDIA_SUB_TITLE"></span>**\_ \_ unter \_ Titel der WPD-Medien**                   | **VT \_ LPWSTR** | Ein Untertitel für das-Objekt. Wenn das Objekt ein Episode-Titel in einer Reihe ist, wäre dies der Titel der Episode, und der [WPD- \_ Medien \_ Titel](/windows) ist der Name der Reihe.                                                                                                                                 |
| **Inhalts-ID des WPD- \_ Medien \_ Abonnements \_ \_**                                                                                | **VT \_ LPWSTR** | Eine ID, die zum Identifizieren eines Inhalts eines Online Abonnement Dienstanbieter verwendet wird. Dies wird in der Regel vom Client anstelle des Treibers oder Geräts festgelegt.                                                                                                                                                    |
| **Lesezeichen für WPD- \_ Medien \_ Zeit \_**                                                                                           | **VT \_ UI8**    | Gibt ein Lesezeichen (in Millisekunden) an, das der letzten Position entspricht, die auf dem angegebenen Medium abgespielt oder angezeigt wird.                                                                                                                                                                                              |
| **\_ \_ \_ Lebensdauer von \_ WPD-Medien**                                                                                           | **VT \_ UI8**    | Gibt die Zeit (in Minuten) bis zur nächsten Inhalts Aktualisierung an.                                                                                                                                                                                                                                                 |
| <span id="wpd_media_title"></span><span id="WPD_MEDIA_TITLE"></span>**WPD- \_ Medien \_ Titel**                                | **VT \_ LPWSTR** | Der Titel des Inhalts. Wenn es sich bei dem Objekt um eine Folge in einer Reihe handelt, wäre dies der Reihen Titel, und der [ \_ \_ unter \_ Titel der WPD-Medien](/windows) wäre der Folge Name.                                                                                                                               |
| **WPD- \_ Medien \_ Gesamt- \_ Bitrate**                                                                                           | **VT \_ UI4**    | Die Bitrate des Objekts.                                                                                                                                                                                                                                                                                    |
| **Verwendung der WPD- \_ Medien \_ \_ Anzahl**                                                                                               | **VT \_ UI4**    | Die Gesamtanzahl der Wiedergabe oder Anzeige dieses Mediums auf dem Gerät.                                                                                                                                                                                                                                  |
| **effektive Bewertung der WPD- \_ Medien \_ Benutzer \_ \_**                                                                                  | **VT \_ UI4**    | Eine Zahl zwischen 1 und 100, die diesen Inhalt bewertet, wobei 1 der niedrigste Wert ist. Dieser Wert wird explizit vom Benutzer festgelegt. Der Standardwert ist 0 (null) und gibt an, dass dieser Wert nicht festgelegt wurde.                                                                                                                    |
| **WPD \_ Media- \_ Webdienste**                                                                                                | **VT \_ LPWSTR** | Eine auf NULL endenden, lesbare Zeichenfolge, die die e-Mail-Adresse des Mediums des Mediums angibt. **Mediacast** -Objekte verwenden diese Eigenschaft häufig, um den Webmaster eines RSS-Kanals oder einer RSS-Reihe anzugeben.<br/>                                                                                             |
| <span id="wpd_media_width"></span><span id="WPD_MEDIA_WIDTH"></span>**WPD- \_ Medien \_ Breite**                                | **VT \_ UI4**    | Ein-Wert, der die Breite der Haupt Ressource (in der Regel ein Video oder Bild) in Pixel angibt. Siehe [WPD- \_ Medien \_ Höhe](/windows).                                                                                                                                                                   |
| **WPD- \_ Medien \_ Künstler**                                                                                                   | **VT \_ LPWSTR** | Ein-Wert, der den Medienkünstler bezeichnet.                                                                                                                                                                                                                                                                      |
